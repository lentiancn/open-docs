# Guía de Instalación de Element Plus

Element Plus es una biblioteca de componentes basada en Vue 3 para desarrolladores, diseñadores y gerentes de productos.

## Versión General

| Versión | Versión de Vue | Estado |
|---------|-------------|--------|
| Element Plus 2.x | Vue 3.2+ | Actual - Recomendado |
| Element Plus 1.x | Vue 3.0+ | Herencia |
| Element UI 2.x | Vue 2.x | Herencia (Vue 2) |

**Recomendación:** Use Element Plus 2.x para todos los proyectos nuevos de Vue 3. Element UI es solo para proyectos Vue 2.

### Compatibilidad de Versiones

| Element Plus | Versión de Vue | Node.js | Fecha de Lanzamiento |
|--------------|-------------|---------|--------------|
| 2.5.x | Vue 3.3+ | 16+ | 2024 |
| 2.4.x | Vue 3.2+ | 14+ | 2023 |
| 2.3.x | Vue 3.2+ | 14+ | 2023 |
| 2.2.x | Vue 3.2+ | 14+ | 2023 |

## Requisitos Previos

### Requisitos del Sistema

| Requisito | Mínimo | Recomendado |
|-------------|---------|-------------|
| Node.js | 16.x | 18.x o 20.x |
| npm | 8.x | 9.x o 10.x |
| Vue | 3.2+ | 3.4+ |
| Vite | 4.x | 5.x |

### Compatibilidad de Navegadores

Element Plus admite navegadores que soportan las últimas 2 versiones:

| Navegador | Versión Mínima |
|--------|-----------------|
| Chrome | 64+ (85+ para 2.5.0+) |
| Edge | 79+ (85+ para 2.5.0+) |
| Firefox | 78+ (79+ para 2.5.0+) |
| Safari | 12+ (14.1+ para 2.5.0+) |

**Nota:** Element Plus NO soporta IE11. Si necesita soporte para IE11, use Element UI con Vue 2.

## Métodos de Instalación

### Método 1: Gestor de Paquetes (Recomendado)

#### Usando npm

```bash
npm install element-plus --save
```

#### Usando yarn

```bash
yarn add element-plus
```

#### Usando pnpm

```bash
pnpm install element-plus
```

#### Usando cnpm (Espejo de China)

```bash
# Configurar espejo
npm config set registry https://registry.npmmirror.com

# Instalar
npm install element-plus --save
```

### Método 2: Importación CDN

#### Usando unpkg

```html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Demo CDN de Element Plus</title>
  <!-- Importar Vue 3 -->
  <script src="//unpkg.com/vue@3"></script>
  <!-- Importar estilos de Element Plus -->
  <link rel="stylesheet" href="//unpkg.com/element-plus/dist/index.css">
  <!-- Importar Element Plus -->
  <script src="//unpkg.com/element-plus"></script>
</head>
<body>
  <div id="app">
    <el-button>{{ message }}</el-button>
  </div>

  <script>
    const { createApp, ref } = Vue
    const app = createApp({
      setup() {
        const message = ref('¡Hola Element Plus!')
        return { message }
      }
    })
    app.use(ElementPlus)
    app.mount('#app')
  </script>
</body>
</html>
```

#### Usando jsDelivr

```html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Demo CDN de Element Plus</title>
  <!-- Importar Vue 3 -->
  <script src="//cdn.jsdelivr.net/npm/vue@3"></script>
  <!-- Importar estilos de Element Plus -->
  <link rel="stylesheet" href="//cdn.jsdelivr.net/npm/element-plus/dist/index.css">
  <!-- Importar Element Plus -->
  <script src="//cdn.jsdelivr.net/npm/element-plus"></script>
</head>
```

#### Bloqueando Versión en CDN

```html
<!-- Bloquear a versión específica -->
<link rel="stylesheet" href="//unpkg.com/element-plus@2.5.0/dist/index.css">
<script src="//unpkg.com/element-plus@2.5.0/dist/index.full.js"></script>
```

### Método 3: Integración con Nuxt.js

```bash
# Instalar módulo Nuxt
npm install -D @element-plus/nuxt
# o
yarn add -D @element-plus/nuxt
```

```typescript

  modules: ['@element-plus/nuxt'],
  elementPlus: {
    /** Opciones */
  }
})
```

## Configuración del Proyecto

### Paso 1: Crear Proyecto Vue

#### Usando Vite (Recomendado)

```bash
# Crear proyecto
npm create vite@latest my-element-app -- --template vue-ts

# Navegar al proyecto
cd my-element-app

# Instalar dependencias
npm install
```

#### Usando Vue CLI

```bash
# Instalar Vue CLI
npm install -g @vue/cli

# Crear proyecto
vue create my-element-app

# Seleccionar Vue 3 cuando se solicite
```

### Paso 2: Instalar Element Plus

```bash
npm install element-plus
```

### Paso 3: Configurar en main.ts/main.js

#### Importación Completa (Simple pero mayor tamaño)

```typescript
// main.ts
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')
```

#### Importación Bajo Demanda (Recomendado para producción)

Primero, instale los plugins de auto-importación:

```bash
npm install -D unplugin-vue-components unplugin-auto-import
```

Configurar Vite:

```typescript
// vite.config.ts
import { defineConfig } from 'vite'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

Configurar TypeScript:

```json
// tsconfig.json
{
  "compilerOptions": {
    // ... otras opciones
    "types": ["element-plus/global"]
  }
}
```

Configurar Webpack:

```javascript
// webpack.config.js
const AutoImport = require('unplugin-auto-import/webpack')
const Components = require('unplugin-vue-components/webpack')
const { ElementPlusResolver } = require('unplugin-vue-components/resolvers')

module.exports = {
  plugins: [
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
}
```

### Paso 4: Usar Componentes

```vue
<!-- App.vue -->
<template>
  <div>
    <h1>{{ message }}</h1>
    <el-button type="primary" @click="handleClick">
      Haz Clic
    </el-button>
    
    <el-input v-model="input" placeholder="Ingrese texto"></el-input>
    
    <el-table :data="tableData" style="width: 100%">
      <el-table-column prop="name" label="Nombre"></el-table-column>
      <el-table-column prop="age" label="Edad"></el-table-column>
    </el-table>
  </div>
</template>

<script setup>
import { ref } from 'vue'

const message = ref('¡Hola Element Plus!')
const input = ref('')
const tableData = ref([
  { name: 'Juan Pérez', age: 25 },
  { name: 'María García', age: 30 },
])
</script>
```

## Configuración Global

### Establecer Tamaño y zIndex Predeterminados

```typescript
// Configuración de importación completa
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import App from './App.vue'

const app = createApp(App)
app.use(ElementPlus, { size: 'small', zIndex: 3000 })
```

### Usando el-config-provider

```vue
<template>
  <el-config-provider :size="size" :z-index="zIndex">
    <app />
  </el-config-provider>
</template>

<script setup>
import { ref } from 'vue'
import { ElConfigProvider } from 'element-plus'

const size = ref('default')
const zIndex = ref(3000)
</script>
```

## Solución de Problemas

### Componentes No Encontrados

**Problema:** Los componentes no se muestran o hay errores

**Soluciones:**
1. Asegúrese de que Element Plus esté instalado: `npm list element-plus`
2. Verifique la importación en main.ts
3. Para importación bajo demanda, verifique la configuración de plugins en vite.config.ts
4. Reinicie el servidor de desarrollo

### Estilos No Aplicados

**Problema:** Los componentes tienen estilos incorrectos

**Soluciones:**
1. Importe CSS: `import 'element-plus/dist/index.css'`
2. Busque conflictos de CSS en su proyecto
3. Verifique la configuración de la herramienta de build

### Errores de TypeScript

**Problema:** TypeScript se queja de los tipos de Element Plus

**Soluciones:**
1. Agregue en tsconfig.json: `"types": ["element-plus/global"]`
2. Ejecute `npm install -D @types/node`
3. Actualice TypeScript: `npm install -D typescript@latest`

### Problemas de Memoria

**Problema:** La compilación se queda sin memoria

**Soluciones:**
1. Incremente la memoria de Node.js: `NODE_OPTIONS=--max_old_space_size=4096 npm run build`
2. Use importación bajo demanda para reducir el tamaño del bundle
3. Actualice Node.js a la última versión LTS

## Migración desde Element UI (Vue 2)

### Diferencias Clave

| Element UI (Vue 2) | Element Plus (Vue 3) |
|-------------------|---------------------|
| `el-button` | Igual |
| `el-table` | Igual |
| `el-form` | Igual |
| `v-model` | Misma sintaxis |
| `slot` | Misma sintaxis |
| Eventos | Igual |

### Cambios en Nombres de Paquetes

```javascript
// Vue 2 + Element UI
import { Button, Table } from 'element-ui'

// Vue 3 + Element Plus
import { Button, Table } from 'element-plus'
```

### Cambios Importantes

1. **Ya no soporta IE11**
2. **Soporte para Vue 3 Composition API**
3. **Soporte mejorado para TypeScript**
4. **Algunas props de componentes han sido renombradas**

## Mejores Prácticas

1. **Use importación bajo demanda** - Reduce significativamente el tamaño del bundle
2. **Use Vite** - Tiempos más rápidos de desarrollo y compilación
3. **Bloquee versiones** - Use versiones específicas en producción
4. **Use TypeScript** - Mejor seguridad de tipos y soporte de IDE
5. **Verifique soporte de navegadores** - Asegúrese de que los navegadores objetivo estén soportados
