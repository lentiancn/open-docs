# Go

> Googleが開発したコンパイル型プログラミング言語

---

## 概要

Go（Golang）は2009年にGoogleで作成されました。シンプルさ、効率性、強力な並行処理サポートで知られています。

---

## 特徴

- 静的型付け
- ガベージコレクション
- ネイティブ並行処理（goroutine）
- 高速なコンパイル
- 豊富な標準ライブラリ
