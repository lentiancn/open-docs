# Kubernetes FAQ

## Was ist Kubernetes?
Open-Source Container-Orchestrierungsplattform.

## Was ist ein Pod?
Kleinste bereitstellbare Einheit in Kubernetes. Kann einen oder mehrere Container enthalten.

## Grundlegende Befehle

```bash
# Deployment erstellen
kubectl create deployment my-app --image=nginx

# Skalieren
kubectl scale deployment my-app --replicas=3
```
