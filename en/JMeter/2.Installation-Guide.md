# JMeter Installation Guide

## Prerequisites

- **Java Development Kit (JDK)**: JMeter requires Java 8 or higher
- **Operating System**: Windows, Linux, or macOS
- **Memory**: At least 4GB RAM recommended for load testing

## Installation Steps

### 1. Download JMeter

Visit the official Apache JMeter website: https://jmeter.apache.org/

Download the latest version (usually a .zip or .tar.gz file).

### 2. Extract the Archive

Extract the downloaded file to your desired location:

```bash
# Linux/Mac
tar -xvf apache-jmeter-x.x.x.tgz

# Windows
# Use 7-Zip or WinRAR to extract
```

### 3. Set Up Environment Variables (Optional)

For convenience, you can add JMeter to your system PATH:

```bash
# Linux/Mac (add to ~/.bashrc or ~/.zshrc)
export JMETER_HOME=/path/to/apache-jmeter-x.x.x
export PATH=$PATH:$JMETER_HOME/bin

# Windows
# Add to System Properties -> Environment Variables -> PATH
```

### 4. Verify Installation

Run the following command to verify JMeter is installed correctly:

```bash
# Linux/Mac
./jmeter.sh --version

# Windows
jmeter.bat --version
```

### 5. Launch JMeter GUI

```bash
# Linux/Mac
./jmeter.sh

# Windows
jmeter.bat
```

## Directory Structure

```
apache-jmeter/
├── bin/          # Executable scripts and JMeter executable
├── lib/          # JMeter libraries
├── lib/ext/      # Plugins and extensions
├── docs/         # Documentation
├── printable_docs/  # User manual
└── LICENSE       # License file
```

## Running in Different Modes

### GUI Mode (Interactive)
```bash
jmeter.sh          # Linux/Mac
jmeter.bat         # Windows
```

### CLI Mode (Headless)
```bash
jmeter -n -t testplan.jmx -l results.jtl -e -o html-report
```

Parameters:
- `-n`: Run in non-GUI mode
- `-t`: Test plan file
- `-l`: Results file
- `-e`: Generate HTML report after test
- `-o`: Output directory for HTML report

## Installation Troubleshooting

### Java Not Found
Ensure JAVA_HOME is set correctly:
```bash
export JAVA_HOME=/path/to/jdk
```

### Memory Issues
Edit the `jmeter` script to increase heap size:
```bash
HEAP="-Xms1g -Xmx4g"
```
