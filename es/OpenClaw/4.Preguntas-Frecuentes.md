# Preguntas Frecuentes de OpenClaw

## Básico

### ¿Qué es OpenClaw?

Puerta de enlace autohospedada que conecta apps de chat con asistentes AI.

### ¿Es gratuito?

El software es gratis. Necesitas clave API.

### Plataformas

- OS: macOS, Linux, Windows
- Chat: WhatsApp, Telegram, Discord, iMessage

## Instalación

### ¿Falló instalación?

- Verificar Node: `node --version` (necesita 22+)
- Permisos: sudo

### Puerto en uso

```bash
openclaw gateway --port 18889
```

## Configuración

### Clave API

```bash
openclaw config set providers.anthropic.key <clave>
```
