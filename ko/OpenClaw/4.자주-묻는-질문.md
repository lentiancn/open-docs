# 자주 묻는 질문

## 기본

### OpenClaw 뭐임?

메시징 앱을 AI 어시스턴트에 연결하는 도움.

### 무료?

소프트웨어 무료 (오픈소스). 비용:
- 서버/컴퓨터
- AI 서비스 API 키

### 어디서 작동?

- 데스크탑: macOS, Linux, Windows
- 모바일: iOS, Android
- 채팅: WhatsApp, Telegram, Discord, iMessage, Signal, Slack 등 20개+

## 설치

### 설치 안 됨?

1. Node.js 확인: `node --version` (22+ 필요)
2. 진단: `openclaw doctor`
3. 로그: `openclaw logs`

### Raspberry Pi에서 됨?

OK. Raspberry Pi 4 + Raspberry Pi OS / Ubuntu.

### Mac 필요?

아니요. iMessage만 필요.

## 설정

### 설정 파일 어디?

`~/.openclaw/openclaw.json`

### 접근 통제 방법?

`dmPolicy`: pairing, allowlist, open, disabled

## 사용

### 채널 추가?

```bash
openclaw channels login whatsapp
```

### 여러 채널?

OK, 하나 Gateway가全部.
