# Руководство по установке Go

> Полное руководство по установке и настройке Go

---

## Системные требования

| Ресурс | Требование |
|---------|-----------|
| ОС | Windows, macOS, Linux, FreeBSD |
| Архитектура | amd64, arm64, 386 и др. |
| Место на диске | Около 100 МБ |

---

## Способы установки

### Windows

#### Использование MSI
1. Скачать Go: https://go.dev/dl/
2. Запустить установщик
3. Следовать инструкциям

#### Chocolatey
```powershell
choco install golang
```

### macOS

#### Homebrew
```bash
brew install go
```

### Linux

#### Ubuntu/Debian
```bash
sudo apt-get update
sudo apt-get install golang-go
```

#### Вручную
```bash
wget https://go.dev/dl/go1.22.0.linux-amd64.tar.gz
sudo tar -C /usr/local -xzf go1.22.0.linux-amd64.tar.gz
export PATH=$PATH:/usr/local/go/bin
```

---

## Настройка

```bash
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin
go env -w GOPROXY=https://goproxy.cn,direct
```

---

## Основные команды

```bash
go mod init <module>
go mod tidy
go build
go run main.go
go test ./...
```
