# FantasyTalking Manuel

## Synthèse vocale

```javascript
const audio = await client.speak('Texte à synthétiser');
```

## Streaming

```javascript
const stream = await client.speakStream('Long texte');
stream.on('data', chunk => player.play(chunk));
```

## Bonnes pratiques

- Réutiliser les connexions
- Utiliser le traitement par lots
- Mettre en cache les résultats
