# Element UI 常見問題

## 安裝問題

### 安裝失敗

檢查 Node.js 版本，確保 >= 8.0.0

## 使用問題

### 樣式不生效

確保已正確引入 CSS：

```javascript
import 'element-ui/lib/theme-chalk/index.css'
```

### 圖標不顯示

引入圖標字體：

```javascript
import 'elementalk/fonts/element-ui/lib/theme-ch-icons.ttf'
```

## Vue 版本問題

Element UI 僅支持 Vue 2。
