# MultiTalk Руководство по установке

Руководство по установке и настройке MultiTalk.

## Требования

- Windows 10/11, macOS 10.15+ или Linux
- Микрофон и наушники/колонки
- Интернет-соединение

## Установка

### Windows

1. Скачайте установщик
2. Запустите установщик
3. Следуйте мастеру установки
4. Запустите MultiTalk

### macOS

1. Скачайте файл .dmg
2. Откройте установщик
3. Перетащите MultiTalk в Приложения
4. Запустите MultiTalk

## Настройка

### Настройка аудио

1. Откройте Настройки > Аудио
2. Выберите устройство ввода
3. Выберите устройство вывода
4. Проверьте уровни

### Push-to-talk

1. Перейдите в Настройки > Горячие клавиши
2. Включите push-to-talk
3. Нажмите нужную комбинацию клавиш

## Устранение проблем

### Нет аудиовхода

**Решение:**
- Проверьте разрешения микрофона
- Проверьте выбор устройства

### Проблемы с эхом

**Решение:**
- Используйте наушники
- Уменьшите громкость колонок
