# FantasyTalking インストールガイド

## 動作環境

FantasyTalkingをインストールする前に、開発環境が以下の要件を満たしていることを確認してください。OSについては、プロジェクトはWindows、macOS、Linuxをサポートし、最適なパフォーマンスを得るために64ビットシステムの使用をお勧めします。Node.jsのバージョンは14.0以上、Pythonのバージョンは3.7以上が必要です。Pythonバックエンドを使用する場合は、FFmpegオーディオ処理ツールもインストールする必要があります。

ハードウェアについては、スムーズな音声合成体験を得るために、システムには少なくとも4GBのRAMとデュアルコアプロセッサが必要です。リアルタイム音声合成を必要とするアプリケーションでは、I/O遅延を減らすためにSSDストレージの使用をお勧めします。モデルトレーニングやカスタム音声開発を計画している場合は、CUDA対応のグラフィックスカードが必要です。

## インストール方法

FantasyTalkingは複数のインストール方法をサポートしています。開発者はニーズとテクノロジース택に基づいて最も適切な方法を選択できます。

### npmを使用したインストール（Node.js）

```bash
# コアライブラリをインストール
npm install fantasy-talking

# オプションコンポーネントをインストール
npm install fantasy-talking-ui  # Web UIコンポーネント
npm install fantasy-talking-node  # Node.jsバックエンドサポート
```

### pipを使用したインストール（Python）

```bash
# コアライブラリをインストール
pip install fantasy-talking

# オプションコンポーネントをインストール
pip install fantasy-talking[ui]  # Web UIコンポーネント
pip install fantasy-talking[all]  # すべてのオプションコンポーネント
```

### ソースからのビルド

```bash
# リポジトリをクローン
git clone https://github.com/fantasytalking/fantasy-talking.git
cd fantasy-talking

# Node.js依存関係をインストール
npm install

# Python依存関係をインストール
pip install -r requirements.txt

# プロジェクトをビルド
npm run build
```

## クイックスタート

### JavaScript/TypeScript使用例

```javascript
const FantasyTalking = require('fantasy-talking');

// クライアントを初期化
const client = new FantasyTalking({
  apiKey: 'your-api-key',
  voice: {
    style: 'neutral',
    language: 'ja-JP',
    speed: 1.0,
    pitch: 1.0
  }
});

// 音声を合成
async function synthesize() {
  const audio = await client.speak('こんにちは、これはFantasyTalkingのテスト音声です');
  console.log('音声合成成功');
  
  // ファイルに保存
  const fs = require('fs');
  fs.writeFileSync('output.wav', audio);
  
  // または直接再生
  client.play(audio);
}

synthesize();
```

## 設定オプション

| パラメータ | タイプ | デフォルト | 説明 |
|-----------|--------|-----------|------|
| apiKey | string | - | APIキー（必須）|
| apiUrl | string | デフォルト | APIサーバーアドレス |
| timeout | number | 30000 | タイムアウト（ミリ秒）|
| voice.style | string | neutral | 声のスタイル |
| voice.language | string | ja-JP | 言語コード |
| voice.speed | number | 1.0 | 話速 |

## サービスデプロイ

### Dockerデプロイ

```bash
docker pull fantasytalking/server:latest

docker run -d \
  --name fantasy-talking \
  -p 8080:8080 \
  -e API_KEY=your-api-key \
  fantasytalking/server:latest
```
