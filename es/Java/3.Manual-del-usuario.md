# Manual del Usuario de Java

## Tu Primer Programa Java

### Creando Hello World

Cree un nuevo archivo de texto llamado HelloWorld.java. Note que el nombre del archivo debe coincidir exactamente con el nombre de la clase.

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

### Compilando el Programa

Abra la terminal o símbolo del sistema, navegue al directorio del archivo y ejecute el comando de compilación:

```bash
javac HelloWorld.java
```

Tras una compilación exitosa, se generará un archivo HelloWorld.class.

### Ejecutando el Programa

```bash
java HelloWorld
```

Salida:

```
Hello, World!
```

## Sintaxis Básica de Java

### Variables y Tipos de Datos

Java es un lenguaje de tipo estático. Las variables deben declararse antes de usarse.

```java
// Tipos de datos primitivos
int age = 25;
double salary = 5000.50;
boolean isActive = true;
char grade = 'A';
String name = "Juan";

// Tipos de referencia
Integer count = 10;
Double price = 99.99;
```

### Operadores

```java
// Operadores aritméticos
int a = 10, b = 3;
int sum = a + b;      // Suma
int diff = a - b;     // Resta
int prod = a * b;     // Multiplicación
int quot = a / b;     // División
int rem = a % b;      // Residuo

// Operadores de comparación
boolean result = (a > b);

// Operadores lógicos
boolean flag = (a > 5) && (b < 10);
```

### Sentencias Condicionales

```java
// Sentencia if-else
if (score >= 90) {
    System.out.println("Excelente");
} else if (score >= 60) {
    System.out.println("Aprobado");
} else {
    System.out.println("Reprobado");
}

// Sentencia switch
switch (day) {
    case 1:
        System.out.println("Lunes");
        break;
    case 2:
        System.out.println("Martes");
        break;
    default:
        System.out.println("Desconocido");
}
```

### Sentencias de Bucle

```java
// Bucle for
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}

// Bucle while
int i = 0;
while (i < 10) {
    System.out.println(i);
    i++;
}

// Bucle do-while
do {
    System.out.println(i);
    i++;
} while (i < 10);

// for-each para arrays
int[] numbers = {1, 2, 3, 4, 5};
for (int num : numbers) {
    System.out.println(num);
}
```

## Programación Orientada a Objetos

### Clases y Objetos

```java
// Definir una clase
public class Person {
    // Campos
    private String name;
    private int age;
    
    // Constructor
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // Getter y Setter
    public String getName() {
        return name;
    }
    
    public void setName(String name) {
        this.name = name;
    }
    
    // Método
    public void sayHello() {
        System.out.println("Hola, soy " + name);
    }
}

// Crear objeto
Person person = new Person("Juan", 25);
person.sayHello();
```

### Herencia

```java

// Clase hijo
```

## Stream API

```java
import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);

// Filtrar
List<Integer> evenNumbers = numbers.stream()
    .filter(n -> n % 2 == 0)
    .collect(Collectors.toList());

// Mapear
List<Integer> squared = numbers.stream()
    .map(n -> n * n)
    .collect(Collectors.toList());

// Sumar
int sum = numbers.stream()
    .mapToInt(Integer::intValue)
    .sum();
```

## Programación Multithread

### Crear Hilos

```java
// Extender clase Thread
class MyThread extends Thread {
    @Override
    public void run() {
        System.out.println("Hilo en ejecución");
    }
}

// Implementar interfaz Runnable
class MyRunnable implements Runnable {
    @Override
    public void run() {
        System.out.println("Hilo en ejecución");
    }
}

// Iniciar hilo
MyThread thread = new MyThread();
thread.start();

// Usando Runnable
Thread runnableThread = new Thread(new MyRunnable());
runnableThread.start();
```

### Sincronización de Hilos

```java
public class Counter {

    public synchronized void increment() {
        count++;
    }
    
    public synchronized int getCount() {
        return count;
    }
}
```

## Programación de Redes

### Comunicación TCP

```java
import java.net.*;
import java.io.*;

ServerSocket serverSocket = new ServerSocket(8080);
Socket clientSocket = serverSocket.accept();
BufferedReader in = new BufferedReader(
    new InputStreamReader(clientSocket.getInputStream()));
PrintWriter out = new PrintWriter(
    clientSocket.getOutputStream(), true);

String message = in.readLine();
out.println("Servidor recibió: " + message);
```

### Solicitudes HTTP

```java
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

HttpClient client = HttpClient.newHttpClient();
HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create("https://api.example.com/data"))
    .GET()
    .build();

HttpResponse<String> response = client.send(request, 
    HttpResponse.BodyHandlers.ofString());
System.out.println(response.body());
```

## Operaciones de Base de Datos

### Conexión JDBC

```java
import java.sql.*;

String url = "jdbc:mysql://localhost:3306/mydb";
String user = "root";
String password = "password";

try (Connection conn = DriverManager.getConnection(url, user, password)) {
    String sql = "SELECT * FROM users WHERE id = ?";
    PreparedStatement stmt = conn.prepareStatement(sql);
    stmt.setInt(1, 1);
    
    ResultSet rs = stmt.executeQuery();
    while (rs.next()) {
        System.out.println(rs.getString("name"));
    }
} catch (SQLException e) {
    e.printStackTrace();
}
```
