# OpenFeign Installation Guide

This guide covers how to install and set up OpenFeign for Java development.

## Overview

OpenFeign is a Java HTTP client binder that provides annotations to express mappings between request parameters and template variables. It is part of the Spring Cloud ecosystem.

## Prerequisites

- Java 17 or later
- Spring Boot 3.x or Spring Cloud
- Maven or Gradle

## Add Dependencies

### Maven

```xml
<!-- Spring Cloud OpenFeign -->
<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-openfeign</artifactId>
</dependency>

<!-- For circuit breaker support -->
<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-circuitbreaker-resilience4j</artifactId>
</dependency>
```

### Gradle

```groovy
implementation 'org.springframework.cloud:spring-cloud-starter-openfeign'
implementation 'org.springframework.cloud:spring-cloud-starter-circuitbreaker-resilience4j'
```

## Enable OpenFeign

```java
@SpringBootApplication
@EnableFeignClients
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
```

## Configuration

### application.properties

```properties
# Enable OpenFeign
feign.client.config.default.connectTimeout=5000
feign.client.config.default.readTimeout=5000
feign.client.config.default.loggerLevel=basic

# Enable circuit breaker
feign.circuitbreaker.enabled=true

# Enable Hystrix (optional, legacy)
feign.hystrix.enabled=true
```
