## 1. Einführung

DragonTalker ist ein auf Deep Learning basierendes Werkzeug zur Erzeugung von sprechenden Köpfen, das realistische Videos aus einem einzelnen Bild und einer Audiodatei erstellt. Diese Technologie wird häufig in virtuellen Moderatoren, digitalem Kundenservice, KI-Bildung und anderen Szenarien eingesetzt.

### 1.1 Kerntechnologie

DragonTalker verwendet eine mehrstufige Pipeline-Architektur: Zuerst werden Mel-Spektrogramm-Merkmale durch Audioanalyse extrahiert; dann werden 3D-Gesichtspunkte zur Erfassung von Bewegungsparametern verwendet; anschließend werden kontinuierliche Videobilder mit Bildrendering-Technologie erzeugt; schließlich verbessert ein Gesichtsrestaurierungsnetzwerk die Qualität. Der gesamte Prozess erfordert keine großen Trainingsdatensätze und ist einfach bereitzustellen.

### 1.2 Hauptmerkmale

- Audio-gesteuerte Animation: Erzeugt synchronisierte Sprechvideos aus einem Bild und einer Audiodatei
- 3D-Bewegungsschätzung: Natürliche Kopfhaltungen und Gesichtsausdrücke basierend auf 3D-Gesichtsrekonstruktion
- Multiple Posen: Über 45 vordefinierte Kopfposenvorlagen
- Gesichtsverbesserung: Integriert GFPGAN, RestoreFormer, CodeFormer für fortschrittliche Gesichtsrestaurierung
- Web-Interface: Benutzerfreundliche browserbasierte Bedienung
- Python API: Vollständige Programmieroberfläche für Entwicklung und Stapelverarbeitung

## 3. Installation

### 3.1 Ubuntu/Linux

```bash
sudo apt-get update
sudo apt-get install -y build-essential git wget curl
sudo apt-get install -y python3.8 python3-pip python3-venv
sudo apt-get install -y ffmpeg libsm6 libxext6 libxrender-dev libgl1-mesa-glx

cd DragonTalker
python3 -m venv venv
source venv/bin/activate

pip install --upgrade pip
pip install torch==1.12.1+cu113 torchvision==0.13.1+cu113 -f https://download.pytorch.org/whl/cu113
pip install -r requirements.txt

bash scripts/download_models.sh
```

### 3.2 Windows

1. **Python installieren**: Python 3.8-3.10 von python.org herunterladen
2. **Git installieren**: Von git-scm.com
3. **CUDA installieren**: CUDA Toolkit 11.7+ von NVIDIA
4. **ffmpeg installieren**:
```powershell
choco install ffmpeg
```

```powershell
git clone https://github.com/your-repo/DragonTalker.git
cd DragonTalker
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 3.3 macOS

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install ffmpeg python@3.10 git wget
git clone https://github.com/your-repo/DragonTalker.git
cd DragonTalker
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## 4. Installation überprüfen

```bash
source venv/bin/activate
python inference.py \
  --source_image examples/source_image.jpg \
  --driven_audio examples/driven_audio.wav \
  --result_dir ./results \
  --enhancer gfpgan
```

---

## 5. Docker (Empfohlen)

```bash
docker build -t dragon-talker:latest .
docker run --gpus all -v $(pwd)/examples:/app/examples \
  -v $(pwd)/results:/app/results \
  dragon-talker:latest \
  python inference.py \
    --source_image /app/examples/source_image.jpg \
    --driven_audio /app/examples/driven_audio.wav \
    --result_dir /app/results
```

---

## 6. Deinstallation

```bash
deactivate
rm -rf venv
docker rmi dragon-talker:latest
```
