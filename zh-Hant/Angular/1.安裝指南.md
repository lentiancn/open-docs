# Angular 安裝指南

本指南詳細介紹 Angular 在所有主要作業系統上的安裝方法。

---

## 簡介

Angular 是由 Google 開發維護的 TypeScript 開源 Web 應用框架。

### 版本

| 版本 | 狀態 |
|---------|--------|
| Angular 19 | 最新 |
| Angular 18 | 支援 |

---

## 系統需求

- Node.js 18.19+
- npm 9+

---

## 安裝方法

### 使用 Angular CLI

```bash
npm install -g @angular/cli
ng new my-app
cd my-app
ng serve
```

---

## 在 Windows 上安裝

```cmd
npm install -g @angular/cli
ng new my-app
cd my-app
ng serve
```

---

## 在 macOS 上安裝

```bash
brew install node@18
npm install -g @angular/cli
```

---

## 在 Linux 上安裝

```bash
sudo apt update
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs
npm install -g @angular/cli
```

---

## 驗證

```bash
ng version
```
