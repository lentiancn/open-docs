# Element UI 사용자 매뉴얼

> 컴포넌트 사용

---

## 버튼

```html
<el-button>기본</el-button>
<el-button type="primary">주요</el-button>
```

## 폼

```html
<el-form :model="form">
  <el-form-item>
    <el-input v-model="form.name"></el-input>
  </el-form-item>
</el-form>
```

## 테이블

```html
<el-table :data="tableData">
  <el-table-column prop="name" label="이름"></el-table-column>
</el-table>
```

## 다이얼로그

```html
<el-dialog :visible.sync="dialogVisible" title="제목">
  <span>내용</span>
</el-dialog>
```
