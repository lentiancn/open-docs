# Jenkins

Jenkins ist ein Open-Source-Automatisierungsserver für die Implementierung von Continuous Integration und Continuous Deployment (CI/CD).

## Einführung

Jenkins ist eines der beliebtesten Open-Source CI/CD-Tools, das Build-, Test- und Deploy-Prozesse automatisieren kann.

## Hauptfunktionen

- **Continuous Integration**: Automatisches Bauen und Testen von Code
- **Continuous Deployment**: Automatisches Deployment in die Produktion
- **Umfangreiche Plugins**: Unterstützung für verschiedene Tools und Plattformen
- **Verteilte Builds**: Unterstützung für Multi-Node-Builds
- **Pipeline**: Codifizierte Workflows

## Schnellstart

### Installation

```bash
# Docker
docker run -d -p 8080:8080 jenkins/jenkins:lts

# macOS
brew install jenkins

# Linux
sudo apt install jenkins
```

### Zugriff

Browser unter http://localhost:8080 öffnen
