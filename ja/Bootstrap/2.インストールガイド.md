# インストールガイド

このドキュメントではBootstrapの様々なインストール方法を紹介します。素早くプロジェクトでBootstrapを使い始められるようになります。

## 動作環境

Bootstrap 5には以下の環境が必要です：

- **HTML5** 対応のモダンブラウザ
- **CSS3** サポート
- **JavaScript** ES6以上（JavaScriptプラグインを使用する場合）

## インストール方法

### 方法1：CDNを読み込む（推奨）

一番簡単な方法は、CDNを使ってBootstrapのCSSとJavaScriptファイルを直接読み込むことです：

```html
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrapページ</title>
    <!-- Bootstrap CSSを読み込む -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>こんにちは、Bootstrap！</h1>

    <!-- Bootstrap JavaScriptを読み込む（Popper含む） -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### 方法2：npmでインストール

npmでプロジェクトのパッケージを管理している場合：

```bash
# Bootstrapをインストール
npm install bootstrap@5

# Popperをインストール（ドロップダウン、ツールチップなど用）
npm install @popperjs/core
```

プロジェクト内で読み込みます：

```javascript
// BootstrapのCSSを読み込む
import 'bootstrap/dist/css/bootstrap.min.css';

// BootstrapのJSを読み込む
import 'bootstrap/dist/js/bootstrap.bundle.min.js';
```

### 方法3：ソースファイルをダウンロード

公式サイトからコンパイル済みのファイルをダウンロード：

1. https://getbootstrap.com/ にアクセス
2. 「Download」ボタンをクリック
3. 「Compiled CSS and JS」を選択してダウンロード
4. 解凍後、cssとjsフォルダをプロジェクトにコピー

ディレクトリ構造：

```
your-project/
├── css/
│   └── bootstrap.min.css
├── js/
│   └── bootstrap.bundle.min.js
└── index.html
```

### 方法4：Sassソースファイル

Bootstrapのスタイルをカスタマイズする必要がある場合は、Sassソースファイルをダウンロード：

```bash
npm install bootstrap
```

Sassファイルで読み込みます：

```scss
// カスタム変数（Bootstrapを読み込む前に定義）
$primary: #007bff;
$body-bg: #f8f9fa;

// Bootstrapを読み込む
@import "~bootstrap/scss/bootstrap";
```

## 開発ツール

### VS Code拡張

入れておくと便利な拡張機能：

- **Bootstrap 5 Snippets**：Bootstrapコンポーネントのコードスニペットを提供
- **HTML CSS Support**：CSSクラス名の自動補完
- **Live Server**：リアルタイムでページをプレビュー

### オンラインエディタ

- **Bootstrap Builder**：ビジュアルでBootstrapページを構築
- **Bootstrap Studio**：プロフェッショナルなBootstrap設計ツール

## クイックスタート

### 最初のページを作る

```html
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrapクイックスタート</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <!-- ナビゲーションバー -->
    <nav class="navbar navbar-expand-lg navbar-dark bg-primary">
        <div class="container">
            <a class="navbar-brand" href="#">マイサイト</a>
        </div>
    </nav>

    <!-- メインコンテンツ -->
    <main class="container mt-5">
        <div class="row">
            <div class="col-md-6">
                <h1>Bootstrapへようこそ</h1>
                <p class="lead">これはBootstrap 5で作ったレスポンシブページです。</p>
                <button class="btn btn-primary">もっと詳しく</button>
            </div>
            <div class="col-md-6">
                <div class="card">
                    <div class="card-body">
                        <h5 class="card-title">特徴</h5>
                        <ul>
                            <li>レスポンシブレイアウト</li>
                            <li>豊富なコンポーネント</li>
                            <li>強力なJavaScriptプラグイン</li>
                        </ul>
                    </div>
                </div>
            </div>
        </div>
    </main>

    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## バージョンの選択

### 本番環境

安定版を使用：

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### 開発環境

最新の開発バージョン（ソース含む）を使用：

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.js"></script>
```

### 中国語ミラー

中国国内では日本語ミラーで高速化：

```html
<link href="https://cdn.bootcss.com/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.bootcss.com/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

## アンインストール

もしアンインストールが必要なら（関連するファイルを削除するだけ）：

- HTMLからCSSとJSの参照を削除
- node_modules内のbootstrapディレクトリを削除（npmを使用した場合）
- ローカルにダウンロードしたCSSとJSファイルを削除
