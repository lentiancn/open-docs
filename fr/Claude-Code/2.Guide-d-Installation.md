# Claude Code Guide d'installation

Instructions d'installation détaillées pour Claude Code.

## Installation sur Windows

### Méthode un: Extension VS Code

1. Ouvrez VS Code
2. Recherchez l'extension "Claude Code"
3. Cliquez sur Installer
4. Connectez-vous à votre compte Anthropic

### Méthode deux: Client autonome

1. Visitez https://claude.com/code
2. Téléchargez le client Windows
3. Exécutez l'installateur
4. Connectez-vous

## Installation sur macOS

### Méthode un: Extension VS Code

1. Ouvrez VS Code
2. Recherchez l'extension "Claude Code"
3. Cliquez sur Installer
4. Connectez-vous

### Méthode deux: Homebrew

```
brew install claude-code
```

### Méthode trois: Client autonome

1. Visitez https://claude.com/code
2. Téléchargez le client macOS
3. Exécutez l'installateur
4. Connectez-vous

## Installation sur Linux

### Méthode un: Extension VS Code

1. Ouvrez VS Code
2. Recherchez l'extension "Claude Code"
3. Cliquez sur Installer
4. Connectez-vous

### Méthode deux: Gestionnaire de paquets

**Ubuntu/Debian**:
```
curl -fsSL https://downloads.anthropic.com/claude-code/install.sh | sh
```

**Arch Linux**:
```
yay -S claude-code
```

## Vérifier l'installation

```
claude --version
```

## Configuration

### Clé API

1. Visitez https://console.anthropic.com
2. Créez une clé API
3. Configurez dans Claude Code

### Variables d'environnement

```
export ANTHROPIC_API_KEY=votre-clé-api
```

## Désinstaller

### macOS

```
brew uninstall claude-code
```

### Linux

```
sudo rm -rf /usr/local/claude-code
```
