# OpenFeign Guide d'Installation

## Prérequis

- Java Development Kit (JDK) 8 ou supérieur
- Outil de build Maven ou Gradle
- IDE (IntelliJ IDEA, Eclipse ou VS Code)

---

## Utiliser Maven

Ajoutez la dépendance OpenFeign à votre `pom.xml`:

```xml
<dependencies>
    <!-- OpenFeign Core -->
    <dependency>
        <groupId>io.github.openfeign</groupId>
        <artifactId>feign-core</artifactId>
        <version>13.2.1</version>
    </dependency>
    
    <!-- OkHttp Client (optionnel mais recommandé) -->
    <dependency>
        <groupId>io.github.openfeign</groupId>
        ign-okhttp<artifactId>fe</artifactId>
        <version>13.2.1</version>
    </dependency>
</dependencies>
```

---

## Utiliser Gradle

Ajoutez les dépendances à votre `build.gradle`:

```groovy
dependencies {
    implementation 'io.github.openfeign:feign-core:13.2.1'
    implementation 'io.github.openfeign:feign-okhttp:13.2.1'
}
```

---

## Intégration Spring Cloud

Pour les applications Spring Boot:

```xml
<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-openfeign</artifactId>
    <version>4.1.0</version>
</dependency>
```

Activez les clients Feign dans votre application:

```java
@SpringBootApplication
@EnableFeignClients
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
```

---

## Vérifier l'Installation

Créez un client Feign simple pour vérifier:

```java
@FeignClient(name = "example-service", url = "https://api.example.com")
public interface ExampleClient {
    
    @RequestMapping(method = RequestMethod.GET, value = "/users/{id}")
    User getUser(@PathVariable("id") Long id);
}
```
