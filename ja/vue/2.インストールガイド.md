# Vue.js インストールガイド

## Vue をオンラインで試す

- **プレイグラウンド**: [Vue プレイグラウンド](https://play.vuejs.org) で直接試す
- **JSFiddle**: プレーン HTML セットアップ用の [JSFiddle テンプレート](https://jsfiddle.net/yyx990803/2ke1ab0z/)
- **StackBlitz**: ブラウザ内で完全なビルドセットアップを [StackBlitz](https://vite.new/vue) で試す
- **インタラクティブチュートリアル**: 推奨セットアップのウォークスルーを [Scrimba チュートリアル](http://scrimba.com/links/vue-quickstart) で視聴

## Vue アプリケーションの作成（推奨）

### 前提条件
- コマンドラインに慣れていること
- Node.js バージョン ^20.19.0 または >=22.12.0 がインストールされていること

### 公式スキャフォールディングツールの使用
ターミナルで以下のコマンドのいずれかを実行します：

```bash
# npm
npm create vue@latest

# pnpm  
pnpm create vue@latest

# yarn (v1+)
yarn create vue

# yarn (Modern v2+)
yarn create vue@latest

# yarn (^v4.11)
yarn dlx create-vue@latest

# bun
bun create vue@latest
```

### プロジェクト設定オプション
以下のオプション機能についてプロンプトが表示されます：
- Project name: プロジェクト名
- Add TypeScript?: TypeScript サポートを追加するかどうか
- Add JSX Support?: JSX サポートを追加するかどうか
- Add Vue Router: シングルページアプリケーション開発用の Vue Router
- Add Pinia: 状態管理用の Pinia
- Add Vitest: 単体テスト用の Vitest
- Add End-to-End Testing: エンドツーエンドテスト（Cypress/Nightwatch/Playwright）
- Add ESLint: コード品質のための ESLint
- Add Prettier: コードフォーマットのための Prettier
- Add Vue DevTools 7: デバッグ用の Vue DevTools 7（実験的）

### 開発サーバーの起動
```bash
cd <your-project-name>
npm install
npm run dev
```

### 本番ビルド
```bash
npm run build
```

## CDN から Vue を使用

### グローバルビルド
```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<div id="app">{{ message }}</div>

<script>
const { createApp } = Vue

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

### ES モジュールビルド
```html
<div id="app">{{ message }}</div>

<script type="module">
import { createApp } from 'https://unpkg.com/vue@3/dist/vue.esm-browser.js'

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

### インポートマップの有効化
```html
<script type="importmap">
{
  "imports": {
    "vue": "https://unpkg.com/vue@3/dist/vue.esm-browser.js"
  }
}
</script>

<div id="app">{{ message }}</div>

<script type="module">
import { createApp } from 'vue'

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

## 開発環境の推奨事項

- **IDE**: Visual Studio Code + [Vue - Official 拡張機能](https://marketplace.visualstudio.com/items?itemName=Vue.volar)
- **ビルドツール**: Vite（公式スキャフォールディングに含まれています）
- **ローカルサーバー**: ES モジュールを使用する場合は、`npx serve` でローカル HTTP サーバーを起動
- **シンプルなケース向け**: 軽量なシナリオには [petite-vue](https://github.com/vuejs/petite-vue) を検討
