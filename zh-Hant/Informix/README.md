# IBM Informix

IBM Informix 是一個功能強大的嵌入式關聯式資料庫管理系統（RDBMS），由 IBM 公司開發和維護。

## 文檔概述

本目錄包含 IBM Informix 的多語言技術文檔：

- **簡介**：產品概述、特性、應用場景和版本資訊
- **安裝指南**：Linux 和 Windows 平台的安裝步驟
- **使用手冊**：基本操作命令、SQL 命令和設定說明
- **常見問題**：常見故障的處理方法和解決方案

## 版本資訊

- 最新版本：Informix 14.10
