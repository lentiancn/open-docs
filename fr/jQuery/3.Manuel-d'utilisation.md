# Manuel d'utilisation de jQuery

## Démarrage rapide

### Inclure jQuery

Incluez jQuery dans le `<head>` ou à la fin du `<body>` de votre fichier HTML :

```html
<!DOCTYPE html>
<html>
<head>
    <title>Demo jQuery</title>
    <script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
</head>
<body>
    <!-- Contenu de la page -->
</body>
</html>
```

### Attendre que le DOM soit prêt

Le code jQuery doit s'exécuter après que le DOM soit complètement chargé :

```javascript
$(document).ready(function() {
    // Écrivez le code jQuery ici
});

// Forme abrégée
$(function() {
    // Écrivez le code jQuery ici
});
```

## Sélecteurs

Les sélecteurs jQuery sont l'une des fonctionnalités principales de jQuery, avec une syntaxe presque identique aux sélecteurs CSS.

### Sélecteurs de base

```javascript
$("#myId")          // Sélecteur d'ID
$(".myClass")       // Sélecteur de classe
$("div")            // Sélecteur d'élément
$("*")              // Sélecteur universel
$("div, p, span")   // Sélecteurs multiples
```

### Sélecteurs d'attributs

```javascript
$('[href]')                    // Éléments avec attribut href
$('[href="value"]')           // href égal à la valeur spécifiée
$('[href!="value"]')          // href différent de la valeur spécifiée
$('[href^="http"]')           // href commençant par la valeur spécifiée
$('[href$=".pdf"]')           // href finissant par la valeur spécifiée
$('[href*="google"]')         // href contenant la valeur spécifiée
```

### Sélecteurs de filtre

```javascript
$("div:first")        // Premier élément div
$("div:last")         // Dernier élément div
$("div:even")        // Éléments div en position paire
$("div:odd")         // Éléments div en position impaire
$("div:eq(n)")       // Div à l'index n
$("div:gt(n)")       // Divs avec index supérieur à n
$("div:lt(n)")       // Divs avec index inférieur à n
$("div:not(.exclude") // Divs sans la classe spécifiée
```

## Manipulation du DOM

### Obtenir et définir le contenu

```javascript
// Obtenir/Définir le contenu texte
var text = $("#myElement").text();
$("#myElement").text("Nouveau contenu texte");

// Obtenir/Définir le contenu HTML
var html = $("#myElement").html();
$("#myElement").html("<p>Nouveau contenu HTML</p>");

// Obtenir/Définir la valeur du formulaire
var value = $("#inputField").val();
$("#inputField").val("Nouvelle valeur");
```

### Obtenir et définir les attributs

```javascript
// Obtenir un attribut
var src = $("#myImage").attr("src");

// Définir un attribut unique
$("#myImage").attr("src", "image.jpg");

// Définir plusieurs attributs
$("#myImage").attr({
    src: "image.jpg",
    alt: "Description de l'image",
    title: "Titre de l'image"
});

// Supprimer un attribut
$("#myImage").removeAttr("title");
```

### Obtenir et définir CSS

```javascript
// Obtenir une propriété CSS unique
var color = $("#myElement").css("color");

// Définir une propriété CSS unique
$("#myElement").css("color", "red");

// Définir plusieurs propriétés CSS
$("#myElement").css({
    color: "red",
    backgroundColor: "blue",
    fontSize: "16px"
});

// Ajouter/Supprimer des classes
$("#myElement").addClass("active highlight");
$("#myElement").removeClass("highlight");
$("#myElement").toggleClass("active");  // Basculer la classe
```

### Traversée du DOM

```javascript
// Éléments parents
$("#myElement").parent();
$("#myElement").parents();
$("#myElement").closest(".container");

// Éléments enfants
$("#myElement").children();
$("#myElement").find(".child");

// Éléments frères
$("#myElement").siblings();
$("#myElement").next();
$("#myElement").prev();
$("#myElement").nextAll();
```

### Créer et supprimer des éléments

```javascript
// Créer un nouvel élément
var newDiv = $("<div>", {
    class: "new-element",
    text: "Contenu du nouvel élément"
});

// Ajouter à la page
$("#container").append(newDiv);      // À l'intérieur, à la fin
$("#container").prepend(newDiv);   // À l'intérieur, au début
$("#container").before(newDiv);     // À l'extérieur, avant
$("#container").after(newDiv);      // À l'extérieur, après

// Supprimer des éléments
$("#myElement").remove();             // Supprimer l'élément et les enfants
$("#myElement").empty();            // Supprimer uniquement les enfants
```

## Gestion des événements

### Lier des événements

```javascript
// Lier l'événement clic
$("#myButton").click(function() {
    alert("Bouton cliqué");
});

// Lier en utilisant la méthode on
$("#myButton").on("click", function() {
    alert("Bouton cliqué");
});

// Lier plusieurs événements
$("#myElement").on({
    mouseenter: function() {
        $(this).css("background", "yellow");
    },
    mouseleave: function() {
        $(this).css("background", "");
    }
});
```

### Délégation d'événements

```javascript
// Liaison d'événements pour éléments dynamiques
$("#container").on("click", ".dynamic-item", function() {
    alert("Élément dynamique cliqué");
});
```

### Événements courants

| Événement | Description |
|-----------|-------------|
| click | Événement clic |
| dblclick | Événement double clic |
| mouseenter | Souris entre |
| mouseleave | Souris sort |
| mousemove | Souris bouge |
| keydown | Touche appuyée |
| keyup | Touche relâchée |
| focus | Reçoit le focus |
| blur | Perd le focus |
| submit | Formulaire soumis |
| change | Valeur changée |
| load | Chargé |

## Effets d'animation

### Animations de base

```javascript
// Afficher l'élément
$("#myElement").show();

// Masquer l'élément
$("#myElement").hide();

// Basculer la visibilité
$("#myElement").toggle();

// Fondu d'entrée
$("#myElement").fadeIn();

// Fondu de sortie
$("#myElement").fadeOut();

// Basculer le fondu
$("#myElement").fadeToggle();

// Glisser vers le bas
$("#myElement").slideDown();

// Glisser vers le haut
$("#myElement").slideUp();

// Basculer le glissement
$("#myElement").slideToggle();
```

### Animations personnalisées

```javascript
$("#myElement").animate({
    opacity: 0.5,
    left: "+=50",
    height: "200px"
}, 300, function() {
    // Callback après l'animation
    console.log("Animation terminée");
});
```

### Arrêter les animations

```javascript
// Arrêter l'animation actuelle
$("#myElement").stop();

// Arrêter toutes les animations et atteindre l'état final
$("#myElement").stop(true, true);
```

## Requêtes Ajax

### Utilisation de base

```javascript
$.ajax({
    url: "/api/data",
    method: "GET",
    data: { id: 1, name: "test" },
    success: function(response) {
        console.log("Succès:", response);
    },
    error: function(xhr, status, error) {
        console.log("Erreur:", error);
    }
});
```

### Méthodes abrégées

```javascript
// Requête GET
$.get("/api/data", { id: 1 }, function(response) {
    console.log(response);
});

// Requête POST
$.post("/api/submit", { name: "test" }, function(response) {
    console.log(response);
});

// Charger HTML
$("#content").load("fragment.html");
```

### Exemple complet

```javascript
$.ajax({
    url: "https://api.example.com/users",
    type: "GET",
    dataType: "json",
    timeout: 5000,
    headers: {
        "Authorization": "Bearer token"
    },
    beforeSend: function(xhr) {
        // Avant l'envoi de la requête
        console.log("Requête sur le point d'être envoyée");
    },
    success: function(data) {
        // Requête réussie
        $("#user-list").html("");
        data.forEach(function(user) {
            $("#user-list").append("<li>" + user.name + "</li>");
        });
    },
    error: function(xhr, status, error) {
        // Requête échouée
        alert("Échec du chargement: " + error);
    },
    complete: function(xhr, status) {
        // Requête terminée (succès ou échec)
        console.log("Requête terminée, statut: " + status);
    }
});
```

## Fonctions utilitaires

### Opérations sur tableaux et objets

```javascript
// Itérer sur un tableau
$.each([1, 2, 3], function(index, value) {
    console.log(index + ": " + value);
});

// Itérer sur un objet
$.each({ name: "John", age: 30 }, function(key, value) {
    console.log(key + ": " + value);
});

// Filtrer un tableau
var filtered = $.grep([1, 2, 3, 4, 5], function(value) {
    return value > 3;
});

// Mapper un tableau
var mapped = $.map([1, 2, 3], function(value) {
    return value * 2;
});

// Fusionner des objets
var merged = $.extend({}, obj1, obj2);
```

### Opérations sur chaînes

```javascript
// Supprimer les espaces
$.trim("  hello  ");  // "hello"

// Échapper les caractères spéciaux
$.escapeSelector(".myClass");  // "\.myClass"
```

### Vérification de type

```javascript
$.isArray([]);        // true
$.isFunction(fn);     // true
$.isEmptyObject({});  // true
$.isPlainObject({});  // true
```

## Chaînage

jQuery supporte le chaînage, permettant d'opérer continuellement sur le même élément :

```javascript
$("#myElement")
    .addClass("highlight")
    .css("color", "red")
    .fadeIn()
    .animate({ height: "100px" }, function() {
        console.log("Terminé");
    });
```

Utilisez la méthode `end()` pour revenir au sélecteur précédent :

```javascript
$("#myElement")
    .find(".child")
    .addClass("active")
    .end()    // Revenir à #myElement
    .removeClass("highlight");
```
