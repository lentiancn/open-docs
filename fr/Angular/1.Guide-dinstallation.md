## Configuration requise

### Matériel
- **Processeur**: Dual-core ou mieux
- **RAM**: Minimum 4GB (8GB recommandé)
- **Espace disque**: Au moins 3GB libre

### Logiciel

#### Windows
- Windows 10 ou supérieur
- Node.js 18.19+ (LTS recommandé)
- npm 9.x+
- Git

#### macOS
- macOS 10.15+
- Node.js 18.19+
- npm 9.x+
- Git

#### Linux
- Ubuntu 20.04 LTS+
- Node.js 18.19+
- npm 9.x+
- Git

## Prérequis

### 1. Installer Node.js

#### Windows
Téléchargez depuis https://nodejs.org/

#### macOS
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install node@18
```

#### Linux
```bash
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs
```

### 2. Installer Git

```bash
# Ubuntu/Debian
sudo apt install git

# macOS
brew install git
```

---

## Installer Angular CLI

```bash
npm install -g @angular/cli
```

### Vérifier

```bash
ng version
```

## Installation sur Windows

1. Installez Node.js depuis https://nodejs.org/
2. Installez Git depuis https://git-scm.com/
3. Installez Angular CLI: `npm install -g @angular/cli`
4. Créez un projet:
   ```bash
   ng new my-first-app
   cd my-first-app
   ng serve
   ```

---

## Installation sur macOS

1. Installez Homebrew
2. Installez Node.js: `brew install node@18`
3. Installez Git: `brew install git`
4. Installez Angular CLI: `npm install -g @angular/cli`

---

## Installation sur Linux

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install curl build-essential git
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs
sudo npm install -g @angular/cli
```

## Dépannage

### Erreurs de permissions npm
```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
```

### Port 4200 occupé
```bash
ng serve --port 4201
```

### Compatibilité des versions

| Angular | Node.js |
|---------|---------|
| Angular 17 | 18.x, 20.x |
| Angular 16 | 16.x, 18.x |
