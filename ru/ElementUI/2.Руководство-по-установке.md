# Element UI Руководство по установке

Полное руководство по установке и настройке Element UI.

## Требования к окружению

### Программные требования

- Node.js: 12.0.0+
- npm: 6.0.0+ или yarn: 1.22.0+
- Vue: 2.6.0+

## Шаги установки

### Способ 1: установка через npm (Рекомендуется)

```bash
# Используя npm
npm install element-ui --save

# Или используя yarn
yarn add element-ui
```

### Способ 2: CDN импорт

```html
<!-- Импорт стилей -->
<link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">

<!-- Импорт Vue -->
<script src="https://unpkg.com/vue"></script>

<!-- Импорт Element UI -->
<script src="https://unpkg.com/element-ui/lib/index.js"></script>
```

## Быстрый старт

### Полный импорт

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(ElementUI)

new Vue({
  el: '#app',
  render: h => h(App)
})
```

### Импорт по требованию (Рекомендуется)

Сначала установите babel-plugin-component:

```bash
npm install babel-plugin-component -D
```

Настройте babel.config.js:

```javascript
module.exports = {
  presets: [
    '@vue/cli-plugin-babel/preset'
  ],
  plugins: [
    [
      'component',
      {
        libraryName: 'element-ui',
        styleLibraryName: 'theme-chalk'
      }
    ]
  ]
}
```

Импортируйте компоненты по требованию:

```javascript
import { Button, Select, Option, Table, TableColumn } from 'element-ui'

Vue.use(Button)
Vue.use(Select)
Vue.use(Option)
Vue.use(Table)
Vue.use(TableColumn)
```

## Настройка темы

### Использование переменных SCSS

Создайте файл переменных:

```scss
// styles/element-variables.scss

$--color-primary: #409EFF;
$--color-success: #67C23A;
$--color-warning: #E6A23C;
$--color-danger: #F56C6C;

@import 'element-ui/packages/theme-chalk/src/common/var.scss';
```

Импортируйте в main.js:

```javascript
import './styles/element-variables.scss'
```

### Онлайн-редактор тем

Посетите: https://element.eleme.io/#/ru/theme

1. Выберите или настройте цвета темы
2. Нажмите «Скачать»
3. Замените файлы темы Element UI

## Интернационализация

### Настройка нескольких языков

Установите vue-i18n:

```bash
npm install vue-i18n -S
```

Настройте:

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import VueI18n from 'vue-i18n'
import ruLocale from 'element-ui/lib/locale/lang/ru'

Vue.use(ElementUI, {
  i18n: (key, value) => i18n.t(key, value)
})

const messages = {
  'ru': { message: 'привет' }
}

const i18n = new VueI18n({
  locale: 'ru',
  messages
})

new Vue({ i18n }).$mount('#app')
```

## Частые проблемы

### Компоненты не отображаются?

1. Подтвердите, что компоненты правильно импортированы
2. Проверьте, вызван ли Vue.use()
3. Проверьте консоль на наличие ошибок

### Стили не применяются?

1. Подтвердите, что файл стилей импортирован
2. Проверьте глобальные переопределения стилей
3. Проверьте уровни z-index

### Иконки не отображаются?

```bash
# Установите библиотеку иконок
npm install element-ui-icons
```

Импортируйте иконки:

```javascript
import 'element-ui-icons'
```

Использование:

```html
<i class="el-icon-search"></i>
```

## Удаление

```bash
npm uninstall element-ui
```
