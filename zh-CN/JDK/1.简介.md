# JDK - 简介

JDK（Java Development Kit）是 Java 开发工具包，包含 Java 运行时环境（JRE）和开发工具。

## 主要版本

- **JDK 8**： LTS 版本，广泛使用
- **JDK 11**：LTS 版本，推荐生产使用
- **JDK 17**：LTS 版本，当前主流
- **JDK 21**：LTS 版本，最新 LTS

## 组件

- **JRE**：Java 运行时环境
- **javac**：编译器
- **java**：运行工具
- **jdb**：调试工具
- **javadoc**：文档生成工具
- **jar**：打包工具

## JDK 实现

- **Oracle JDK**：Oracle 官方维护
- **OpenJDK**：开源实现
- **Eclipse Temurin**：formerly AdoptOpenJDK
- **Amazon Corretto**：亚马逊维护
- **Alibaba Dragonwell**：阿里巴巴优化
