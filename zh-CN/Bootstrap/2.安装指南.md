# 安装指南

本文档介绍 Bootstrap 的多种安装方式，帮助您快速开始在项目中使用 Bootstrap。

## 环境要求

Bootstrap 5 需要以下环境：

- **HTML5** 现代浏览器
- **CSS3** 支持
- **JavaScript** ES6+（如使用 JavaScript 插件）

## 安装方式

### 方式一：CDN 引入（推荐）

最简单的方式是通过 CDN 直接引入 Bootstrap 的 CSS 和 JavaScript 文件：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 页面</title>
    <!-- 引入 Bootstrap CSS -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>你好，Bootstrap！</h1>

    <!-- 引入 Bootstrap JavaScript（包含 Popper） -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### 方式二：npm 安装

如果使用 npm 管理项目依赖：

```bash
# 安装 Bootstrap
npm install bootstrap@5

# 安装 Popper（用于下拉菜单、工具提示等）
npm install @popperjs/core
```

然后在项目中引入：

```javascript
// 引入 Bootstrap 的 CSS
import 'bootstrap/dist/css/bootstrap.min.css';

// 引入 Bootstrap 的 JS
import 'bootstrap/dist/js/bootstrap.bundle.min.js';
```

### 方式三：下载源码

从官网下载预编译好的文件：

1. 访问 https://getbootstrap.com/
2. 点击"Dowload"按钮
3. 选择"Compiled CSS and JS"下载
4. 解压后将 css 和 js 文件夹复制到项目

目录结构：

```
your-project/
├── css/
│   └── bootstrap.min.css
├── js/
│   └── bootstrap.bundle.min.js
└── index.html
```

### 方式四：Sass 源码

如果需要自定义 Bootstrap 样式，可以下载 Sass 源码：

```bash
npm install bootstrap
```

然后在 Sass 文件中引入：

```scss
// 自定义变量（在引入 Bootstrap 之前定义）
$primary: #007bff;
$body-bg: #f8f9fa;

// 引入 Bootstrap
@import "~bootstrap/scss/bootstrap";
```

## 开发工具

### VS Code 扩展

推荐安装以下扩展：

- **Bootstrap 5 Snippets**：提供 Bootstrap 组件代码片段
- **HTML CSS Support**：CSS 类名自动补全
- **Live Server**：实时预览页面

### 在线编辑器

- **Bootstrap Builder**：可视化构建 Bootstrap 页面
- **Bootstrap Studio**：专业的 Bootstrap 设计工具

## 快速开始

### 创建第一个页面

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 快速开始</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <!-- 导航栏 -->
    <nav class="navbar navbar-expand-lg navbar-dark bg-primary">
        <div class="container">
            <a class="navbar-brand" href="#">我的网站</a>
        </div>
    </nav>

    <!-- 主要内容 -->
    <main class="container mt-5">
        <div class="row">
            <div class="col-md-6">
                <h1>欢迎使用 Bootstrap</h1>
                <p class="lead">这是一个使用 Bootstrap 5 构建的响应式页面。</p>
                <button class="btn btn-primary">了解更多</button>
            </div>
            <div class="col-md-6">
                <div class="card">
                    <div class="card-body">
                        <h5 class="card-title">特性</h5>
                        <ul>
                            <li>响应式布局</li>
                            <li>丰富的组件</li>
                            <li>强大的 JavaScript 插件</li>
                        </ul>
                    </div>
                </div>
            </div>
        </div>
    </main>

    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## 版本选择

### 生产环境

使用稳定版本：

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### 开发环境

使用最新开发版本（包含源码）：

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.js"></script>
```

### 中文镜像

国内可以使用中文镜像加速：

```html
<link href="https://cdn.bootcss.com/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.bootcss.com/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

## 卸载

如果需要卸载（删除相关文件即可）：

- 删除 HTML 中的 CSS 和 JS 引用
- 删除 node_modules 中的 bootstrap 目录（如果使用 npm）
- 删除本地下载的 CSS 和 JS 文件
