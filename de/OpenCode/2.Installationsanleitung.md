# OpenCode Installationsanleitung

OpenCode unterstützt mehrere Installationsmethoden.

## Linux/macOS Installation

### Empfohlen: Installationsskript

```bash
curl -fsSL https://opencode.ai/install | bash
```

### Node.js

```bash
npm install -g opencode-ai
```

### Bun

```bash
bun install -g opencode-ai
```

### pnpm

```bash
pnpm install -g opencode-ai
```

### Yarn

```bash
yarn global add opencode-ai
```

### Homebrew (macOS/Linux)

```bash
brew install anomalyco/tap/opencode
```

### Arch Linux

```bash
# Stabil
sudo pacman -S opencode

# Neueste (AUR)
paru -S opencode-bin
```

## Windows Installation

### Chocolatey

```bash
choco install opencode
```

### Scoop

```bash
scoop install opencode
```

### NPM

```bash
npm install -g opencode-ai
```

### Mise

```bash
mise use -g github:anomalyco/opencode
```

### Docker

```bash
docker run -it --rm ghcr.io/anomalyco/opencode
```

### Manuelle Download

Du kannst auch die Binärdatei von [GitHub Releases](https://github.com/anomalyco/opencode/releases) herunterladen.

## Installation überprüfen

```bash
opencode --version
```

## Konfiguration

### LLM-Anbieter verbinden

Führe folgendes aus, um dich zu verbinden:

```bash
opencode
/connect
```

Wähle `opencode`, dann besuche [opencode.ai/auth](https://opencode.ai/auth) um dich anzumelden und deinen API-Schlüssel zu erhalten.

### Andere Anbieter

Du kannst auch andere LLM-Anbieter auswählen. Siehe die [offiziellen Docs](https://opencode.ai/docs/providers#directory) für Details.

## Projekt initialisieren

```bash
cd /path/to/project
opencode
/init
```

Dies lässt OpenCode dein Projekt analysieren und eine AGENTS.md-Datei im Projektverzeichnis erstellen.
