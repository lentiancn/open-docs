# 安装指南

本文档介绍 Apifox 的多种安装方式，帮助您快速开始在本地使用 Apifox。

## 环境要求

Apifox 支持以下操作系统：

- Windows 7 及以上版本（64位）
- macOS 10.13 及以上版本
- Linux（Debian/Ubuntu 20.04+、Fedora 36+）

## 下载方式

### 官网下载

访问 Apifox 官网 https://apifox.com/，点击"下载"按钮，选择对应操作系统的版本进行下载。

### 版本选择

Apifox 提供两个版本：

- **Apifox**（标准版）：包含所有核心功能，适合大多数用户
- **Apifox Plus**（付费版）：包含更多高级功能，如团队管理、高级自动化等

基础功能在标准版中已经足够使用。

## 安装步骤

### Windows 安装

1. 下载 Windows 安装包（.exe 文件）
2. 双击运行安装程序
3. 选择安装路径（建议使用默认路径）
4. 等待安装完成
5. 启动 Apifox，登录或注册账号

### macOS 安装

1. 下载 macOS 安装包（.dmg 文件）
2. 打开安装包，将 Apifox 拖入 Applications 文件夹
3. 启动 Apifox，登录或注册账号

首次启动时，可能需要在系统偏好设置中允许运行来自未知开发者的应用。

### Linux 安装

**Debian/Ubuntu：**

```bash
# 下载 deb 包
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.deb

# 安装
sudo dpkg -i Apifox-latest-linux.x64.deb

# 如果依赖缺失，修复安装
sudo apt-get install -f
```

**Fedora：**

```bash
# 下载 rpm 包
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.rpm

# 安装
sudo rpm -i Apifox-latest-linux.x64.rpm
```

## 账号与登录

### 注册账号

首次使用需要注册账号：

1. 启动 Apifox
2. 点击"注册"按钮
3. 输入邮箱和密码完成注册
4. 邮箱验证（如果需要）

### 登录

支持多种登录方式：

- 邮箱密码登录
- 第三方账号登录（GitHub、Google 等）

### 云端同步

登录账号后，项目数据会自动同步到云端。可以在不同设备间无缝切换，数据实时保持一致。

## 初始化配置

首次启动后，建议进行以下配置：

### 设置主题

在"设置"中可以选择浅色或深色主题，根据个人喜好选择。

### 设置代理

如果需要通过代理访问网络，在"设置"->"网络"中配置代理服务器。

### 导入导出

- **导入**：支持导入 Postman、Swagger、Har 等格式的项目
- **导出**：支持导出为 OpenAPI、Markdown、HTML 等格式

## 快速开始

### 创建项目

1. 点击左侧"项目"旁边的"+"
2. 输入项目名称
3. 选择项目类型（API 项目或 Mock 项目）

### 创建接口

1. 在项目中点击"+"选择"新建接口"
2. 填写接口基本信息：
   - 接口名称
   - 请求路径
   - 请求方法（GET/POST/PUT/DELETE 等）
3. 填写请求参数、请求体、响应等信息
4. 保存接口

### 调试接口

1. 打开已创建的接口
2. 点击"发送"按钮
3. 查看响应结果

### 使用 Mock

1. 在接口中定义数据结构
2. 系统会自动生成 Mock 地址
3. 访问 Mock 地址即可获取模拟数据

## 卸载

### Windows 卸载

- 打开"控制面板"->"程序和功能"
- 找到 Apifox，点击"卸载"

### macOS 卸载

- 将 Applications 文件夹中的 Apifox 拖到废纸篓
- 清空废纸篓

### Linux 卸载

```bash
# Debian/Ubuntu
sudo apt-get remove apifox

# Fedora
sudo rpm -e apifox
```
