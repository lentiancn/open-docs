# 常见问题

本文档汇总了 Apifox 使用过程中常见的技术问题和解答。

## 基础问题

### 什么是 Apifox？

Apifox 是一款 API 协作平台，集成 API 设计、文档管理、调试、Mock、自动化测试等功能。它可以帮助开发团队更高效地进行 API 开发、测试和文档维护。

### Apifox 是免费的吗？

Apifox 提供免费版和付费版。免费版已经包含所有核心功能，足以满足大多数开发需求。付费版提供更多高级功能，如更大的团队规模、高级自动化等。

### Apifox 和 Postman 有什么区别？

Apifox 和 Postman 都是 API 工具，但 Apifox 更适合国内用户：更好的中文支持、更智能的 Mock 功能、更完善的团队协作体验。Apifox 完全免费基础功能，而 Postman 很多功能需要付费。

## 安装问题

### 下载速度慢怎么办？

Apifox 官方提供多个下载节点，可以尝试访问 https://apifox.com/ 选择离你最近的节点下载。

### 无法启动怎么办？

1. 检查系统版本是否满足要求
2. 尝试以管理员身份运行
3. 检查是否安装了必要的运行库
4. 查看错误日志获取更多信息

### 登录失败怎么办？

1. 检查网络连接是否正常
2. 确认账号密码是否正确
3. 尝试使用第三方账号登录
4. 联系官方客服获取帮助

## 使用问题

### 接口数据不同步？

1. 确认已登录账号
2. 检查网络连接
3. 点击"同步"按钮手动同步
4. 检查账号权限

### Mock 数据不生效？

1. 确认 Mock 功能已开启
2. 检查 Mock 地址是否正确
3. 查看接口是否有语法错误
4. 尝试刷新页面

### 请求发送失败？

1. 检查网络连接
2. 确认请求地址正确
3. 检查请求参数格式
4. 查看是否被代理拦截

### 响应显示乱码？

1. 尝试更改响应编码
2. 检查响应头中的编码设置
3. 使用"格式化"功能重新解析

## 功能问题

### 如何导入 Postman 数据？

1. 点击"导入"
2. 选择"Postman"
3. 选择要导入的文件
4. 确认导入

### 如何导出为 Swagger？

1. 进入项目设置
2. 选择"导出"
3. 选择"Swagger/OpenAPI"
4. 选择版本
5. 点击导出

### 如何使用环境变量？

1. 在项目设置中定义环境变量
2. 在请求中使用 {{变量名}} 引用
3. 切换顶部环境选择器切换环境

### 如何批量操作接口？

1. 按住 Ctrl/Cmd 选择多个接口
2. 右键选择批量操作
3. 选择操作类型（删除、导出、移动等）

## 测试问题

### 自动化测试如何运行？

1. 打开测试套件
2. 点击"运行"按钮
3. 查看测试结果

### 断言失败怎么办？

1. 查看失败原因
2. 检查预期值和实际值
3. 修改断言条件
4. 重新运行测试

### 测试报告如何导出？

1. 运行测试完成后
2. 点击"导出报告"
3. 选择格式（HTML、PDF、JSON）

## 团队协作问题

### 如何邀请团队成员？

1. 进入项目设置
2. 点击"成员管理"
3. 点击"邀请成员"
4. 输入成员邮箱或用户名

### 成员权限如何设置？

在邀请成员时，可以设置不同角色：

- 管理员：完全控制
- 编辑者：可以编辑和运行
- 查看者：只读权限

### 如何解决冲突？

当多人同时编辑同一接口时：

1. 系统会提示冲突
2. 选择保留本地版本或远程版本
3. 或者手动合并修改

## 数据问题

### 数据如何备份？

1. 定期导出项目为 Apifox 格式
2. 或者使用云端同步功能

### 数据如何恢复？

1. 如果有备份文件，点击"导入"
2. 选择备份文件进行恢复

### 删除的数据能恢复吗？

删除的数据会在回收站中保留 30 天，可以从回收站恢复。

## 性能问题

### 运行卡顿怎么办？

1. 关闭不需要的项目
2. 清理浏览器缓存
3. 升级硬件配置
4. 尝试最新版本

### 大量数据导入慢？

1. 分批导入
2. 关闭自动同步
3. 耐心等待导入完成
