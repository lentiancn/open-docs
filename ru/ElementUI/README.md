# Element UI

> Библиотека компонентов UI для рабочего стола на Vue 2

---

## Описание

Element UI — библиотека компонентов для рабочего стола на Vue 2.0 от команды Eleme.

**Примечание**: Для Vue 3 используйте [Element Plus](https://github.com/element-plus/element-plus)

---

## Версия

| Версия | Vue | Статус |
|--------|-----|--------|
| Element UI 2.x | Vue 2 | Стабильная |
| Element UI 1.x | Vue 2 | Устарела |
