# Mockito

## 개요

Mockito는 Java 생태계에서 가장 인기 있는 단위 테스트 모킹 프레임워크로, 테스트驱动 개발(TDD)과 동작驱动 개발(BDD)을 위해 설계되었습니다. 개발자가 실제 객체의 행동을 제어된 환경에서 시뮬레이션하는 모ック 객체를 생성할 수 있게 해주며, 독립적이고 신뢰할 수 있는 단위 테스트를 가능하게 합니다.

## 주요 기능

Mockito는 학습과 사용이 직관적인 깨끗하고 간단한 API 설계를 제공합니다. 강력한 기능은 메서드 호출 검증, 반환값 스텁, 예외 발생, 인자 매칭 등을 지원합니다. 우수한 가독성은 테스트 코드를 직관적으로 이해하기 쉽게 만들고, 검증 실패 시 명확한 오류 메시지를 제공합니다. 활발한 커뮤니티가 프레임워크의 지속적인 유지보수와 종합적인 문서를 보장합니다.

## 빠른 시작

### 모ック 객체 생성

```java
List<String> mockedList = mock(List.class);
```

### 동작 정의

```java
when(mockedList.get(0)).thenReturn("first");
```

### 상호작용 검증

```java
verify(mockedList).get(0);
```

## 자료

공식 웹사이트에서 전체 문서와 샘플 코드를 제공합니다. GitHub 저장소에 소스 코드와 릴리스 정보가 있습니다. 온라인 문서에서 API 참조와 사용 가이드를 확인할 수 있습니다.

## 요구사항

Mockito 5.x는 JDK 11 이상이 필요합니다. Mockito 4.x는 JDK 8을 지원합니다. 프로젝트는 Maven 또는 Gradle을 사용하여 의존성을 관리할 수 있습니다.
