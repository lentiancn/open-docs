# Docker Обзор

Самая популярная платформа контейнеризации в мире.

## Что такое Docker?

Docker — это платформа с открытым исходным кодом для разработки, развёртывания и запуска приложений. Технология контейнеризации позволяет упаковать приложения и их зависимости в лёгкие переносимые контейнеры.

### Основные функции

- Лёгкость — контейнеры используют общее ядро, быстрый запуск
- Переносимость — собрал один раз, запускай везде
- Изоляция — контейнеры изолированы друг от друга
- Эффективность — экономичнее виртуальных машин

## Ключевые концепции

- Образ — шаблон для создания контейнеров
- Контейнер — запущенный экземпляр образа
- Репозиторий — место для хранения образов
- Dockerfile — скрипт сборки образа
