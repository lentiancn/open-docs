# SadTalker

SadTalker — это проект ИИ, который генерирует реалистичные видео говорящих голов из одного изображения и аудиовхода.

## Обзор

SadTalker создаёт высококачественные видео говорящих голов путём:
- Извлечения лицевых ориентиров из аудио
- Генерации коэффициентов движения лица 3D
- Рендеринга реалистичной синхронизации губ и выражений лица
- Повышения качества видео с помощью восстановления лица

## Быстрый старт

### Установка

```bash
git clone https://github.com/OpenTalker/SadTalker.git
cd SadTalker

# Создание виртуальной среды
python -m venv venv
source venv/bin/activate  # Linux/macOS
# venv\Scripts\activate  # Windows

# Установка зависимостей
pip install -r requirements.txt

# Загрузка моделей
bash scripts/download_models.sh
```

### Генерация видео

```bash
python inference.py \
  --source_image examples/source_image.jpg \
  --driven_audio examples/driven_audio.wav \
  --result_dir ./results \
  --preprocess full \
  --enhancer gfpgan
```

## Функции

- **Аудио-управляемая анимация** - Генерация говорящей головы из аудио
- **Оценка движения 3D** - Реалистичные движения лица
- **Несколько стилей поз** - Более 45 вариантов поз
- **Улучшение лица** - Интеграция GFP-GAN, RestoreFormer, CodeFormer
- **Веб-интерфейс** - Простая в использовании демонстрация
- **Python API** - Программный доступ

## Требования к входным данным

### Изображение
- Формат: JPG, PNG
- Разрешение: 512x512 или больше
- Лицо: Фронтальное, чёткое

### Аудио
- Формат: WAV, MP3
- Длительность: 1-60 секунд
- Качество: Чёткая речь

## Вывод

- Формат: MP4 (H.264)
- Разрешение: 256x256 или 512x512
- FPS: 25
