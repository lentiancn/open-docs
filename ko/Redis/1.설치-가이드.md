## Redis란?

Redis는 데이터베이스, 캐시, 메시지 브로커로 사용되는 오픈소스 인메모리 데이터 구조 저장소입니다. 문자열, 해시, 목록, 집합, 정렬된 집합 등 다양한 데이터 구조를 지원합니다.

## Linux

### Ubuntu / Debian

#### 방법 1: APT(권장)

```bash
# 패키지 인덱스 업데이트
sudo apt update

# Redis 설치
sudo apt install redis-server

# Redis 시작
sudo systemctl start redis-server

# 부팅 시 시작
sudo systemctl enable redis-server
```

#### 방법 2: 소스 컴파일

```bash
# 의존성 설치
sudo apt install build-essential tcl

# Redis 다운로드
cd /tmp
wget https://github.com/redis/redis/archive/7.2.4.tar.gz
tar -zxvf 7.2.4.tar.gz
cd redis-7.2.4

# 빌드 및 설치
make
sudo make install
```

### CentOS / RHEL

#### YUM 사용

```bash
# EPEL 저장소 설치
sudo yum install epel-release

# Redis 설치
sudo yum install redis

# Redis 시작
sudo systemctl start redis

# 부팅 시 시작
sudo systemctl enable redis
```

### Fedora

```bash
# Redis 설치
sudo dnf install redis

# Redis 시작
sudo systemctl start redis
sudo systemctl enable redis
```

### Arch Linux

```bash
# Redis 설치
sudo pacman -S redis

# Redis 시작
sudo systemctl start redis
sudo systemctl enable redis
```

## macOS

### Homebrew(권장)

```bash
# Homebrew 설치(필요한 경우)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Redis 설치
brew install redis

# Redis 시작
brew services start redis

# 또는 수동 실행
redis-server
```

### MacPorts

```bash
sudo port install redis
```

## Windows

### WSL2(권장)

```bash
# WSL2에 Redis 설치(Ubuntu)
sudo apt update
sudo apt install redis-server

# Redis 시작
sudo systemctl start redis
```

### Memurai(권장 대안)

1. https://www.memurai.com/ 에서 Memurai 다운로드
2. 설치하고 Windows 서비스로 실행

### Docker

```bash
# Redis 컨테이너 실행
docker run -d --name redis -p 6379:6379 redis:latest

# 영구 저장소 사용
docker run -d --name redis -p 6379:6379 -v redis-data:/data redis:latest redis-server --appendonly yes

# 비밀번호 설정
docker run -d --name redis -p 6379:6379 -v redis-data:/data redis:latest redis-server --requirepass yourpassword
```

## 설치 확인

```bash
# Redis 버전 확인
redis-server --version

# Redis 상태 확인
sudo systemctl status redis

# Redis 연결
redis-cli

# 연결 테스트
redis-cli ping
# 반환: PONG
```

## 설정

### 주요 설정 파일

- **Ubuntu/Debian**: `/etc/redis/redis.conf`
- **CentOS/RHEL**: `/etc/redis.conf`
- **macOS**: `/usr/local/etc/redis.conf` (Homebrew)
- **Docker**: `/etc/redis/redis.conf`

### 기본 설정

```bash
# 로컬에 바인딩(보안)
bind 127.0.0.1

# 비밀번호 설정
requirepass yourpassword

# 영구 저장 활성화
appendonly yes

# 최대 메모리 설정
maxmemory 256mb

# 메모리 제거 정책 설정
maxmemory-policy allkeys-lru
```

### 일반적인 명령어

```bash
# Redis 시작
redis-server

# 사용자 정의 설정으로 시작
redis-server /etc/redis/redis.conf

# Redis 연결
redis-cli

# 비밀번호로 연결
redis-cli -a yourpassword

# Redis 중지
sudo systemctl stop redis
```

## Redis CLI 기본 명령어

```bash
# 키 설정
redis-cli SET mykey "Hello"

# 키 가져오기
redis-cli GET mykey

# 모든 키 확인
redis-cli KEYS "*"

# 키 삭제
redis-cli DEL mykey

# 만료 시간 설정(초)
redis-cli SETEX mykey 60 "value"

# 서버 정보 확인
redis-cli INFO
```

## 제거

### Ubuntu/Debian

```bash
sudo apt remove redis-server
```

### CentOS/RHEL

```bash
sudo yum remove redis
```

### macOS (Homebrew)

```bash
brew uninstall redis
```
