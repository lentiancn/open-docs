# Mockito 安装文档

在所有主流 Java 构建工具中安装 Mockito 的完整指南。

---

## 什么是 Mockito？

Mockito 是一个流行的 Java 单元测试模拟框架。它允许你创建和配置模拟对象来简化测试。

---

## Maven

### 添加依赖

```xml
<dependencies>
    <!-- Mockito Core -->
    <dependency>
        <groupId>org.mockito</groupId>
        <artifactId>mockito-core</artifactId>
        <version>5.8.0</version>
        <scope>test</scope>
    </dependency>
</dependencies>
```

### 配合 JUnit 4

```xml
<dependency>
    <groupId>org.mockito</groupId>
    <artifactId>mockito-inline</artifactId>
    <version>5.8.0</version>
    <scope>test</scope>
</dependency>
```

### 配合 JUnit 5

```xml
<dependency>
    <groupId>org.mockito</groupId>
    <artifactId>mockito-junit-jupiter</artifactId>
    <version>5.8.0</version>
    <scope>test</scope>
</dependency>
```

---

## Gradle

### 添加依赖

```groovy
// Groovy DSL
testImplementation 'org.mockito:mockito-core:5.8.0'
```

### 配合 JUnit 5

```groovy
testImplementation 'org.mockito:mockito-junit-jupiter:5.8.0'
```

### 使用 Mock Maker

```groovy
// 用于模拟 final 类和方法
testImplementation 'org.mockito:mockito-inline:5.8.0'
```

### Kotlin DSL

```kotlin
testImplementation("org.mockito:mockito-core:5.8.0")
testImplementation("org.mockito:mockito-junit-jupiter:5.8.0")
```

## IDE 支持

### IntelliJ IDEA

- 内置 Mockito 支持
- 可安装 Mockit-o 插件增强功能

### Eclipse

- 使用 Eclipse Marketplace 安装 Mockito 插件

### VS Code

- 安装 Java Extension Pack
