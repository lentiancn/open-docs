# Flutter 安裝指南

Flutter 是 Google 開發的開源 UI 工具包，用於構建美觀、快速的跨平台應用。本指南涵蓋在所有環境中的安裝方法。

## 版本資訊

| 版本 | 狀態 | 說明 |
|------|------|------|
| Flutter 3.x | ✅ 目前 | 最新穩定版本 |
| Flutter 2.x | ⚠️ 維護 | 安全更新 |

> **重要提示**：本指南主要針對 **Flutter 3.x**。

## 系統需求

### Windows

| 需求 | 最低配置 |
|------|----------|
| 作業系統 | Windows 10 或更高版本 |
| 磁碟空間 | 2.8 GB |
| 記憶體 | 4 GB |

### macOS

| 需求 | 配置 |
|------|------|
| 作業系統 | macOS 10.14+ |
| 磁碟空間 | 2.8 GB |
| 記憶體 | 4 GB |

### Linux

| 需求 | 配置 |
|------|------|
| 作業系統 | Ubuntu 18.04+ |
| 磁碟空間 | 2.8 GB |
| 記憶體 | 4 GB |

## 安裝方法

### 方法一：Windows 安裝

#### 1. 下載 Flutter SDK

1. 下載 Flutter SDK：https://docs.flutter.dev/get-started/install/windows
2. 解壓到指定目錄（如 C:\src\flutter）

#### 2. 設定環境變數

```powershell
# 新增到 PATH
$env:PATH += ";C:\src\flutter\bin"
```

#### 3. 執行 Flutter doctor

```powershell
flutter doctor
```

### 方法二：macOS 安裝

#### 1. 使用 Homebrew（推薦）

```bash
# 安裝 Flutter
brew install flutter

# 或手動下載
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable
```

#### 2. 設定環境變數

```bash
# ~/.bash_profile 或 ~/.zshrc
export PATH="$PATH:`pwd`/flutter/bin"
```

#### 3. 執行 Flutter doctor

```bash
flutter doctor
```

### 方法三：Linux 安裝

#### 1. 安裝相依套件

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa clang cmake ninja-build pkg-config libgtk-3-dev

# Fedora
sudo dnf install -y curl git unzip xz zip mesa-libGLU clang cmake ninja-build pkgconfig gtk3-devel
```

#### 2. 下載 Flutter SDK

```bash
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable
export PATH="$PATH:~/development/flutter/bin"
```

#### 3. 執行 Flutter doctor

```bash
flutter doctor
```

### 方法四：使用 VS Code

1. 下載並安裝 VS Code
2. 開啟 VS Code
3. 安裝 Flutter 擴充功能：
   - 搜尋 "Flutter" 並安裝
   - 搜尋 "Dart" 並安裝

### 方法五：使用 Android Studio

1. 下載並安裝 Android Studio
2. 安裝 Flutter 外掛程式：
   - 開啟 Android Studio
   - Settings → Plugins
   - 搜尋 "Flutter" 並安裝

## Android 設定

### 安裝 Android SDK

```bash
# 使用 Flutter 工具安裝
flutter doctor --android-licenses
flutter config --android-sdk /path/to/android/sdk
```

### 設定環境變數

```bash
# macOS/Linux
export ANDROID_SDK_ROOT=/Users/user/Library/Android/sdk
export PATH=$PATH:$ANDROID_SDK_ROOT/cmdline-tools/latest/bin:$ANDROID_SDK_ROOT/platform-tools
```

### 建立模擬器

```bash
# 列出可用裝置
flutter devices

# 建立模擬器
flutter emulator --create --name my-emulator

# 啟動模擬器
flutter emulators --launch my-emulator
```

## 常見問題

### Flutter doctor 錯誤？

確保所有相依套件已安裝：

```bash
# 執行 flutter doctor
flutter doctor -v
```

### Android SDK 找不到？

設定 SDK 路徑：

```bash
flutter config --android-sdk /path/to/android/sdk
```

### 建置失敗？

清理並重新建置：

```bash
flutter clean
flutter pub get
flutter build apk
```
