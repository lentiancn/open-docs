## 1. Grid System

### 1.1 Basic Principles

Bootstrap's grid system is responsive, mobile-first, and flexbox-based.

- 12-column grid
- 5 responsive breakpoints
- Auto layout

### 1.2 Grid Classes

| Device | Prefix | Min Width |
|--------|--------|-----------|
| Extra small | col- | None |
| Small | col-sm | 576px |
| Medium | col-md | 768px |
| Large | col-lg | 992px |
| Extra large | col-xl | 1200px |
| Extra extra large | col-xxl | 1400px |

### 1.3 Basic Usage

```html
<div class="container">
    <div class="row">
        <div class="col-12 col-md-6 col-lg-4">
            <p>Column 1</p>
        </div>
        <div class="col-12 col-md-6 col-lg-4">
            <p>Column 2</p>
        </div>
        <div class="col-12 col-md-6 col-lg-4">
            <p>Column 3</p>
        </div>
    </div>
</div>
```

### 1.4 Column Offset

```html
<div class="col-md-4 offset-md-4"></div>
<div class="col-md-6 offset-md-3"></div>
```
