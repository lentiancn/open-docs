# FantasyTalking Benutzerhandbuch

## Grundfunktionen

FantasyTalking bietet umfassende Sprachsynthese- und Dialogfunktionen. Dieses Handbuch stellt die verschiedenen Funktionen im Detail vor.

### Sprachsynthese

Sprachsynthese ist die Kernfunktion von FantasyTalking und konvertiert Text in natürliche und fließende Sprache:

```javascript
// Einfache Text-zu-Sprache
const audio = await client.speak('Hallo, willkommen bei FantasyTalking');

// Sprachstil angeben
const audio = await client.speak('Das Wetter ist heute sehr gut', {
  voice: {
    style: 'happy',
    language: 'de-DE'
  }
});

// Geschwindigkeit und Tonhöhe angeben
const audio = await client.speak('Bitte warten Sie einen Moment', {
  voice: {
    speed: 1.2,
    pitch: 0.9
  }
});
```

### SSML-Markup-Sprache

FantasyTalking unterstützt SSML-Markup für feinere Kontrolle über die Sprachausgabe:

```xml
<speak>
  <p>
    <s>Erster Satz.</s>
    <s>Zweiter Satz.</s>
  </p>
  <break time="500ms"/>
  <p>
    <emphasis level="moderate">Das ist wichtiger Inhalt</emphasis>
  </p>
</speak>
```

### Sprachereignisse

Während der Sprachwiedergabe können Sie verschiedene Ereignisse für Interaktion abhören:

```javascript
// Sprachbeginn abhören
client.on('speech-start', (event) => {
  console.log('Sprachwiedergabe gestartet');
});

// Wortgrenze abhören (für Untertitel-Synchronisation)
client.on('word-boundary', (event) => {
  console.log(`Aktuelles Wort: ${event.word}`);
});

// Sprachende abhören
client.on('speech-end', (event) => {
  console.log('Sprachwiedergabe abgeschlossen');
});

// Fehler abhören
client.on('error', (event) => {
  console.error('Fehler:', event.error);
});
```

### Stapelverarbeitung

Bei der Synthese großer Textmengen können Sie die Stapelverarbeitung für höhere Effizienz nutzen:

```javascript
const texts = [
  'Das ist die erste Stimme',
  'Das ist die zweite Stimme',
  'Das ist die dritte Stimme'
];

// Parallele Synthese
const results = await Promise.all(
  texts.map(text => client.speak(text))
);

// Sequentielle Synthese (Reihenfolge beibehalten)
const results = [];
for (const text of texts) {
  const audio = await client.speak(text);
  results.push(audio);
}
```

## Dialogfunktionen

### Grundlegender Dialog

FantasyTalking bietet leistungsstarke Dialogmanagementfunktionen mit Unterstützung für Multi-Turn-Gespräche und Kontextspeicherung:

```javascript
// Dialogsitzung erstellen
const session = client.createSession({
  systemPrompt: 'Sie sind ein freundlicher Assistent, der gut auf Fragen antwortet',
  language: 'de-DE'
});

// Erste Runde
const response1 = await session.chat('Wie ist das Wetter heute?');
console.log(response1.text);
console.log(response1.audio);

// Zweite Runde (enthält automatisch Kontext)
const response2 = await session.chat('Und morgen?');
console.log(response2.text);
```

### Dialogkonfiguration

Sie können für verschiedene Dialogszenarien konfigurieren:

```javascript
const session = client.createSession({
  dialogue: {
    temperature: 0.7,
    maxTokens: 500,
    tags: ['Kundenservice', 'Verkaufsberatung']
  },
  voice: {
    style: 'professional',
    speed: 1.0
  }
});
```

## Erweiterte Funktionen

### Streaming-Sprachsynthese

Für Szenarien, die Echtzeit-Feedback erfordern, können Sie Streaming-Sprachsynthese verwenden:

```javascript
// Streaming-Ausgabe aktivieren
const stream = await client.speakStream('Dies ist ein langer Text');

// Auf Datenblöcke hören
stream.on('data', (chunk) => {
  audioPlayer.play(chunk);
});

stream.on('end', () => {
  console.log('Sprachsynthese abgeschlossen');
});
```

### Benutzerdefinierte Sprachmodelle

Wenn Sie eine markenspezifische Stimme benötigen, können Sie benutzerdefinierte Modelle trainieren:

```javascript
const trainingData = {
  name: 'Markenstimme',
  description: 'Stimme für Unternehmensmarke',
  samples: [
    { text: 'Beispieltext 1', audio: 'base64-kodiertes Audio 1' },
    { text: 'Beispieltext 2', audio: 'base64-kodiertes Audio 2' }
  ]
};

const job = await client.trainVoice(trainingData);

while (job.status !== 'completed') {
  await sleep(5000);
  const status = await client.getJobStatus(job.id);
  console.log(`Trainingsfortschritt: ${status.progress}%`);
}
```

## Best Practices

### Leistungsoptimierung

Beachten Sie Folgendes für Leistungsoptimierung:

1. **Verbindungswiederverwendung**: Erstellen Sie keine Client-Instanzen häufig neu
2. **Stapelverarbeitung**: Nutzen Sie Stapelverarbeitung für große Textmengen
3. **Caching**: Verwenden Sie Cache, um doppelte Synthese zu vermeiden

### Fehlerbehandlung

Es wird empfohlen, eine umfassende Fehlerbehandlung zu implementieren:

```javascript
try {
  const audio = await client.speak('Testtext');
} catch (error) {
  if (error.code === 'RATE_LIMIT') {
    console.log('Anfragen zu häufig, bitte später erneut versuchen');
  } else if (error.code === 'INVALID_API_KEY') {
    console.log('Ungültiger API-Schlüssel');
  }
}
```
