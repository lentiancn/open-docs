# Redis FAQ

## Questions Générales

### Qu'est-ce que Redis?
Redis (Remote Dictionary Server) est un stockage de structures de données en mémoire open source utilisé comme base de données, cache et broker de messages.

### Redis est-il gratuit?
Oui, Redis est open source et gratuit. Il est publié sous licence BSD 3-Clause.

### Quels langages de programmation puis-je utiliser avec Redis?
Redis fournit des bibliothèques clientes pour presque tous les langages majeurs: Python, JavaScript, Java, Go, C, C++, Ruby, PHP, et plus.

## Questions d'Installation

### Comment installer Redis sur Windows?
Redis est conçu pour les systèmes tipo Unix. Vous pouvez utiliser:
- Windows Subsystem for Linux (WSL)
- Memurai (alternative compatible)
- Conteneurs Docker

### Comment installer Redis sur Linux/macOS?
```bash
# Ubuntu/Debian
sudo apt-get install redis-server

# macOS
brew install redis
```

### Comment démarrer le serveur Redis?
```bash
redis-server
```

Pour se connecter:
```bash
redis-cli
```

## Questions de Performance

### Quelle est la vitesse de Redis?
Redis peut effectuer des millions d'opérations par seconde avec une latence sub-milliseconde.

### Combien de données Redis peut-il stocker?
Redis peut stocker des centaines de millions de clés. La principale limite est la RAM disponible.

### Redis supporte-t-il le clustering?
Oui, Redis Cluster fournit le partitionnement automatique des données sur plusieurs nœuds avec haute disponibilité.

## Dépannage

### Pourquoi Redis consomme-t-elle trop de mémoire?
- Vérifier les grandes clés ou valeurs
- Vérifier l'utilisation mémoire avec `INFO memory`
- Utiliser des structures de données appropriées

### Comment persister les données dans Redis?
Configurer la persistance dans `redis.conf`:
- RDB: Snapshots périodiques
- AOF: Logging append-only file

### Comment sécuriser Redis?
- Binder uniquement à localhost
- Utiliser l'authentification par mot de passe
- Activer le chiffrement TLS
