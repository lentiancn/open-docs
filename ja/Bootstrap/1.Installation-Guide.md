## Bootstrap の概要

Bootstrapは、レスポンシブレイアウト、モバイルファーストのウェブサイトやWebアプリケーションを開発するための最も人気のあるHTML、CSS、JSフレームワークです。

### Bootstrap の特徴

- **レスポンシブデザイン**: 組み込みレスポンシブグリッドシステム
- **定義済みスタイル**: 豊富なコンポーネントとスタイルクラス
- **JavaScriptコンポーネント**: インタラクティブコンポーネント（モadal、カーソルなど）
- **カスタマイズ**: Sass変数によるテーマカスタマイズ
- **ブラウザ互換性**: すべての最新ブラウザをサポート

### Bootstrap バージョン履歴

| バージョン | 発売日 | 主な機能 |
|------|----------|----------|
| Bootstrap 5 | 2021年5月 | jQuery依存の削除、改善されたフォームコンポーネント、新しいUtilities API |
| Bootstrap 4 | 2018年1月 | Flexboxサポート、Sass（Lessの代わりに）、新しいコンポーネント |
| Bootstrap 3 | 2013年8月 | モバイルファーストデザイン |
| Bootstrap 2 | 2012年1月 | レスポンシブデザインの改善 |
| Bootstrap 1 | 2011年8月 | 初期バージョン |

> **注意**: Bootstrap 5は現在のメジャーバージョンであり、新プロジェクトに推奨されます。Bootstrap 4はまだメンテナンス更新を受け取っていますが、新機能の開発は停止しています。

## インストール方法

Bootstrapは複数のインストール方法を提供します：

| インストール方法 | 適用シナリオ |
|----------|----------|
| CDN | クイックプロトタイピング、シンプルプロジェクト |
| npm | Node.jsプロジェクト、Webpack |
| パッケージマネージャー | Ruby、Pythonなどのプロジェクト |

## CDN インストール

### CSS のみ

以下のコードをHTMLの `<head>` セクションに追加します：

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH" crossorigin="anonymous">
```

### CSS + JavaScript

CSSを追加した後、`</body>` の前にJSを追加します：

```html
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-YvpcrYf0tY3lHB60NNkmXc5s9fDVZLESaAA55NDzOxhy9GkcIdslK1eN7N6jIeHz" crossorigin="anonymous"></script>
```

> **注意**: Bootstrap 5は `bootstrap.bundle.min.js`（Popperを含む）を使用するため、Popperを別途インクルードする必要はありません。

### 国内CDNの使用（中国ユーザー推奨）

```html
<!-- CSS -->
<link href="https://cdn.staticfile.org/twitter-bootstrap/5.3.3/css/bootstrap.min.css" rel="stylesheet">

<!-- JS -->
<script src="https://cdn.staticfile.org/twitter-bootstrap/5.3.3/js/bootstrap.bundle.min.js"></script>
```

## npm インストール

### Bootstrap のインストール

```bash
# npmでインストール
npm install bootstrap

# 特定のバージョンをインストール
npm install bootstrap@5.3.3

# Bootstrap 4をインストール
npm install bootstrap@4.6.2
```

### プロジェクトでの参照

**CSS**:
```javascript
// webpack または bundler
import 'bootstrap/dist/css/bootstrap.min.css';
```

**JavaScript**:
```javascript
import * as bootstrap from 'bootstrap';

// または必要なモジュールのみをインポート
import { Modal } from 'bootstrap';
```

## パッケージマネージャーインストール

### yarn

```bash
yarn add bootstrap
```

### pnpm

```bash
pnpm add bootstrap
```

### Ruby on Rails

Gemfileに追加：

```ruby
gem 'bootstrap', '~> 5.3'
```

その後 `bundle install` を実行します。

### Python/Django

```bash
pip install bootstrap5
```

または requirements.txtに追加：

```
django-bootstrap5
```

## ダウンロードインストール

### プリコンパイルバージョンをダウンロード

1. [Bootstrap公式ウェブサイト](https://getbootstrap.com/)にアクセス
2. 「Download」ボタンをクリック
3. プリコンパイルされたCSSとJSファイルをダウンロード

### ソースコードを含むバージョンをダウンロード

Sassソースファイルを含むバージョンをダウンロード：

1. [Bootstrapソース](https://github.com/twbs/bootstrap/releases)にアクセス
2. ソースアーカイブをダウンロード
3. 解凍後含まれるもの：
   - `dist/css/` - コンパイル済みCSS
   - `dist/js/` - コンパイル済みJS
   - `scss/` - Sassソースファイル

### ファイル構造

```
bootstrap/
├── css/
│   ├── bootstrap.css
│   ├── bootstrap.css.map
│   ├── bootstrap.min.css
│   ├── bootstrap.min.css.map
│   ├── bootstrap.rtl.css
│   └── bootstrap.rtl.min.css
├── js/
│   ├── bootstrap.bundle.js
│   ├── bootstrap.bundle.min.js
│   ├── bootstrap.esm.js
│   └── bootstrap.esm.min.js
└── scss/
    └── ...
```

## インストール確認

### 基本HTMLテンプレート

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Bootstrap テスト</title>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <div class="container mt-5">
    <h1>Hello, Bootstrap!</h1>
    <button class="btn btn-primary">プライマリーボタン</button>
  </div>
  
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### 確認手順

1. 上記のコードを `.html` ファイルとして保存
2. ブラウザで開く
3. Bootstrapスタイルのページが表示されれば、インストール成功です

## よくある質問

### Q1: Bootstrap 5 と Bootstrap 4 の違いは？

主な違い：
- Bootstrap 5はjQuery依存を削除
- ネイティブJavaScriptを使用
- 改善されたフォームコンポーネント
- 新しいUtilities API
- IEブラウザのサポートを終了

### Q2: Bootstrapテーマをカスタマイズ方法は？

**方法1: Sassの使用**
```scss
// カスタム変数
$primary: #ff6600;
$secondary: #cccccc;

// Bootstrapをインポート
@import "bootstrap";
```

**方法2: Bootstrapカスタマイズツールの使用**
https://getbootstrap.com/docs/5.3/customize/options/ を訪問

### Q3: Bootstrapはレスポンシブですか？

はい、Bootstrapはモバイルファーストデザインです：
- `col-` 極小画面（<576px）
- `col-sm-` 小画面（≥576px）
- `col-md-` 中画面（≥768px）
- `col-lg-` 大画面（≥992px）
- `col-xl-` 特大画面（≥1200px）
- `col-xxl-` 超特大画面（≥1400px）

### Q4: Bootstrapアイコンの使用方法は？

```bash
npm install bootstrap-icons
```

またはCDNを使用：

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css">
```

使用：
```html
<i class="bi bi-github"></i>
```

### Q5: Bootstrap 5はIE11をサポートしますか？

しません。Bootstrap 5は最新ブラウザが必要です。IE11サポートが必要な場合は、Bootstrap 4.6を使用してください。

### Q6: Bootstrapスタイルのデバッグ方法は？

1. ブラウザ開発者ツールを使用（F12）
2. 要素のクラス名を確認
3. Computedパネルでスタイルのソースを表示

### Q7: Bootstrapが他のフレームワークと競合した場合は？

- 特定のセレクターを使用
- Bootstrapの名前空間を利用
- Bootstrapの提供APIを使用

### Q8: Bootstrapバージョンをアップグレード方法は？

1. 現在のプロジェクトをバックアップ
2. CDNリンクまたはnpmパッケージバージョンを更新
3. 破壊的変更を確認
4. すべての機能をテスト
