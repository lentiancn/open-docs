# MultiTalk

ゲームとコミュニケーションのための強力な多人数音声チャットアプリケーション。

## MultiTalkとは？

MultiTalkは、ゲームやコミュニティ向けの多人数音声チャットを可能にするデスクトップアプリケーションです。

### 主な機能

- **多人数音声** - 複数の参加者をサポート
- **低遅延** - 最小限の遅延でリアルタイム通信
- **オーディオソース** - 複数のオーディオ入出力デバイス
- **プッシュトゥトーク** - 設定可能なキーで送信
- **ノイズ低減** - 内蔵オーディオフィルタリング
