# Installationsanleitung

## Systemanforderungen

- Node.js: 16.0+

---

## Installation

### Corepack (Empfohlen)

```bash
corepack enable
corepack prepare yarn@stable --activate
```

### npm

```bash
npm install -g yarn
```

---

## Verifizieren

```bash
yarn --version
```
