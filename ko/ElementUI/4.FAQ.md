# Element UI FAQ

일반적인 질문과 답변.

## 설치 문제

### Q: 설치 실패?

**해결책:**
1. Node.js 버전 확인
```bash
node --version  # 12.0.0+ 필요
```
2. npm 캐시 비우기
```bash
npm cache clean --force
```
3. 타오바오 미러 사용
```bash
npm config set registry https://registry.npmmirror.com
```
4. node_modules 삭제 후 재설치
```bash
rm -rf node_modules
npm install
```

### Q: 모듈을 찾을 수 없음?

**해결책:**
1. element-ui가 설치되었는지 확인
```bash
npm list element-ui
```
2. package.json 의존성 확인

### Q: Vue 버전 호환 안 됨?

**해결책:**
Element UI 2.x는 Vue 2.6.0+ 필요:
```bash
npm install vue@2
```

## 사용 문제

### Q: 컴포넌트가 표시되지 않음?

**해결책:**
1. 컴포넌트가 올바르게 임포트되었는지 확인
```javascript
import { Button } from 'element-ui'
Vue.use(Button)
```
2. 콘솔 오류 확인
3. 컴포넌트 이름이 정확한지 확인

### Q: 스타일이 적용되지 않음?

**해결책:**
1. 스타일 파일이 임포트되었는지 확인
```javascript
import 'element-ui/lib/theme-chalk/index.css'
```
2. 전역 스타일 오버라이드 확인
3. z-index 레벨 확인

### Q: 테이블에 데이터가 표시되지 않음?

**해결책:**
1. 데이터 형식 확인
2. 열 정의 확인
3. 데이터가 반응형인지 확인

### Q: 폼 유효성 검사가 작동하지 않음?

**해결책:**
1. Form과 FormItem 임포트
2. prop 속성 설정
3. data에 rules 정의

### Q: 드롭다운 옵션이 표시되지 않음?

**해결책:**
1. options 데이터 확인
2. value와 label 매핑 확인

### Q: 대화상자를 닫은 후 데이터가 재설정되지 않음?

**해결책:**
1. v-if로 대화상자 제어
2. 닫을 때 수동으로 폼 재설정

## 스타일 문제

### Q: 스타일 충돌?

**해결책:**
1. 더 구체적인 선택자 사용
2. z-index 레벨 확인

### Q: 아이콘이 표시되지 않음?

**해결책:**
1. element-ui-icons가 설치되었는지 확인
```bash
npm install element-ui-icons
```
2. 올바르게 임포트

### Q: 테마 색상이 정확하지 않음?

**해결책:**
1. 전역 스타일 오버라이드 확인
2. SCSS 변수 임포트 순서 확인
3. 브라우저 캐시 지우기

## 국제화 문제

### Q: 언어 전환이 작동하지 않음?

**해결책:**
1. i18n 올바르게 설정
2. 언어 팩이 임포트되었는지 확인

## 성능 문제

### Q: 페이지 로딩이 느림?

**해결책:**
1. 온디맨드 임포트 사용
2. CDN 임포트 사용
3. Gzip 압축 활성화

### Q: 대용량 데이터 테이블이 느림?

**해결책:**
1. 가상 스크롤링 사용
2. 페이지별로 데이터 로드
3. 불필요한 기능 비활성화

## 빌드 문제

### Q: 빌드 후 스타일 손실?

**해결책:**
1. vue.config.js 설정 확인
2. 스타일 임포트 위치 확인

### Q:本番 환경에서 컴포넌트가 표시되지 않음?

**해결책:**
1. Vue.use()가 호출되었는지 확인
2. 온디맨드 임포트 완전성 확인

## 업그레이드 문제

### Q: 1.x에서 2.x로 업그레이드 방법은?

**해결책:**
1. 공식 마이그레이션 가이드 읽기
2. 비호환 API 점진적 교체
3. 테스트 실행

### Q: Vue 3 프로젝트의 경우?

Element UI는 Vue 2용. Vue 3 프로젝트는 Element Plus 사용:
```bash
npm install element-plus
```
