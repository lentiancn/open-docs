## 1. Übersicht

DragonTalker ist ein auf Deep Learning basierendes System zur Erzeugung von sprechenden Köpfen. Die Hauptfunktion besteht darin, ein statisches Bild mit einer Audiodatei zu kombinieren und ein dynamisches Video mit Sprechbewegungen zu erzeugen. Die Technologie verwendet fortschrittliche 3D-Gesichtsrekonstruktion und Bildsynthese-Algorithmen, um die ursprünglichen Gesichtszüge zu erhalten und gleichzeitig Lippensynchronisation und natürliche Gesichtsausdrücke zu erzeugen.

### 1.1 Anwendungsbereiche

- **Virtuelle Moderatoren**: Virtuelle Avatare mit realistischer Stimme und Ausdrücken
- **Digitaler Kundenservice**: Anthropomorphe interaktive Videos generieren
- **Bildung und Schulung**: Kursinhalte in Videoform präsentieren
- **Film-Dubbing**: Schnell Dubbing-Animationen für Charaktere erstellen

## 2. Schnellstart

### 2.1 Grundlegende Verwendung

```bash
source venv/bin/activate
python inference.py \
  --source_image examples/source_image.jpg \
  --driven_audio examples/driven_audio.wav \
  --result_dir ./results
```

### 2.2 Parameter

| Parameter | Beschreibung | Standard |
|-----------|--------------|----------|
| --source_image | Pfad zum Quellbild | Erforderlich |
| --driven_audio | Pfad zur Audiodatei | Erforderlich |
| --result_dir | Ausgabeverzeichnis | ./results |
| --enhancer | Verbesserungsalgorithmus | gfpgan |
| --outputfps | Video-Bildrate | 25 |

---

## 3. Spezifikationen

### 3.1 Quellbild

- Format: JPG, PNG
- Auflösung: 512×512 oder höher empfohlen
- Inhalt: Frontales Gesicht, klar, keine Verdeckung

### 3.2 Audio

- Format: WAV, MP3
- Dauer: 1-60 Sekunden
- Qualität: Klare Stimme

### 3.3 Ausgabevideo

- Format: MP4 (H.264)
- Auflösung: 256×256 oder 512×512
- Bildrate: 25 FPS

## 5. Python API

### Grundlegende Verwendung

```python
from dragon_talker import DragonTalker

model = DragonTalker(
    device='cuda',
    faceid_weight=0.5,
    enhancer='gfpgan'
)

model.load_source('source.jpg')
model.load_audio('audio.wav')

result_path = model.generate(
    output_path='result.mp4',
    preprocess='full',
    fps=25
)

print(f"Video gespeichert unter: {result_path}")
```

## 6. FAQ

### 6.1 Videoqualität

**Erzeugtes Gesicht ist unscharf**

Lösung: Gesichtsverbesserung aktivieren

```bash
python inference.py --enhancer gfpgan ...
```

---

**Lippen nicht mit Audio synchronisiert**

Ursache: Schlechte Audioqualität oder zu schnelles Sprechen

Lösung: Audio vorverarbeiten, Abtastrate erhöhen

---

### 6.2 Leistung

**GPU-Speicher nicht ausreichend**

Lösungen:
1. Ausgabeauflösung reduzieren
2. Verbesserung deaktivieren
3. CPU-Modus verwenden
