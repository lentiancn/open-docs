# Preguntas frecuentes

## Básico

### Qué es OpenClaw?

Herramienta que conecta apps de mensajería con IA. Puedes usar IA desde WhatsApp, Telegram, etc.

### Es gratis?

El software es gratis (open source). Pagas: servidor + API de IA.

### Dónde corre?

- Desktop: macOS, Linux, Windows
- Móvil: iOS, Android
- Chat: WhatsApp, Telegram, Discord, +20 más

## Instalación

### Falló instalación?

1. `node --version` (necesita 22+)
2. `openclaw doctor`
3. `openclaw logs`

## Configuración

### Dónde está config?

`~/.openclaw/openclaw.json`

### Controlar quién puede escribir?

`dmPolicy`: pairing, allowlist, open, disabled

## Uso

### Agregar canal?

```bash
openclaw channels login whatsapp
```

### Múltiples canales?

Sí, un Gateway sirve varios.
