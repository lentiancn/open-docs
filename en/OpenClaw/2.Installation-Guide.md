# OpenClaw Installation Guide

> Complete guide to setting up and installing OpenClaw

---

## System Requirements

| Requirement | Specification |
|-------------|---------------|
| Operating System | Windows 10+, macOS, Linux |
| Node.js | Version 18+ |
| Memory | At least 4 GB RAM |
| Storage | ~500 MB |

---

## Installation

### Prerequisites

Make sure Node.js is installed:

```bash
node --version
npm --version
```

### Step-by-Step Installation

#### 1. Install via npm

```bash
npm install -g openclaw
```

#### 2. Start Gateway Service

The Gateway service is the core of OpenClaw:

```bash
# Check status
openclaw gateway status

# Start Gateway
openclaw gateway start

# Restart Gateway
openclaw gateway restart

# Stop Gateway
openclaw gateway stop
```

#### 3. Initial Configuration

After installation, set up configuration files:

```bash
# Create config directory
mkdir -p ~/.openclaw

# Copy default configuration
cp /path/to/openclaw/config/* ~/.openclaw/
```

## Configuration

### Main Configuration Files

| File | Description |
|------|-------------|
| AGENTS.md | Workspace configuration |
| SOUL.md | Personality settings |
| TOOLS.md | Tool configuration |
| USER.md | User preferences |

### Environment Variables

```bash
# Optional: Custom workspace directory
OPENCLAW_WORKSPACE=/path/to/workspace

# Optional: Gateway port
OPENCLAW_PORT=3000
```

## Verification

### Test Installation

```bash
# Show version information
openclaw --version

# Check gateway status
openclaw gateway status

# Show available commands
openclaw --help
```

### Quick Start

```bash
# Start interactive session
openclaw chat

# Execute single task
openclaw exec "say hello"
```

## Troubleshooting

### Gateway won't start?

**Solution**:
1. Verify Node.js is correctly installed
2. Check permissions
3. Review logs: `openclaw gateway logs`

### Port already in use?

**Solution**:
```bash
# Change port
openclaw gateway start --port 3001
```

### Module not found?

**Solution**:
```bash
# Reinstall
npm uninstall -g openclaw
npm install -g openclaw
```

---

## Uninstallation

```bash
npm uninstall -g openclaw

# Remove gateway service
openclaw gateway stop
rm -rf ~/.openclaw
```
