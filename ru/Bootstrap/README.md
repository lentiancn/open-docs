# Bootstrap

Самая популярная CSS-фреймворк в мире.

## Обзор

Bootstrap — самый популярный CSS-фреймворк от Twitter. С адаптивной сеткой, готовыми компонентами и CSS утилитами.

### Основные функции

- **Адаптивная сетка** - 12-колоночная сетка
- **Готовые компоненты** - Навбары, карточки, модальные окна
- **CSS утилиты** - Быстрые настройки стилей

### CDN

