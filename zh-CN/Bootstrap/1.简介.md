# Bootstrap 简介

> 流行的前端 CSS 框架

---

## 项目简介

Bootstrap 是由 Twitter 开发和维护的开源 CSS 框架，于 2011 年首次发布。Bootstrap 提供了丰富的预制组件和响应式网格系统，能够快速构建现代、响应式的网站和 Web 应用。

---

## 版本信息

| 版本 | 发布日期 | 状态 |
|------|----------|------|
| Bootstrap 5.3 | 2023年5月 | 当前稳定版 |
| Bootstrap 5.2 | 2022年9月 | 稳定版 |
| Bootstrap 5.1 | 2021年8月 | 稳定版 |
| Bootstrap 4.x | 2018年1月 | 已停止维护 |

---

## 主要特性

- **响应式设计**：内置响应式布局
- **预制组件**：按钮、导航、表单、卡片等
- **强大的网格系统**：12列响应式网格
- **自定义选项**：Sass 变量易于定制
- **JavaScript 插件**：下拉菜单、模态框等
- **图标库**：内置 Bootstrap 图标

---

## 适用场景

- 响应式网站开发
- 后台管理系统
- 快速原型开发
- 移动优先的项目
