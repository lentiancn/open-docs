# Redis Häufig Gestellte Fragen

## Allgemeine Fragen

### Was ist Redis?
Redis (Remote Dictionary Server) ist ein Open-Source-In-Memory-Datenstrukturspeicher, der als Datenbank, Cache und Nachrichtenbroker verwendet wird.

### Ist Redis kostenlos?
Ja, Redis ist Open-Source und kostenlos nutzbar. Es wird unter der BSD 3-Clause-Lizenz veröffentlicht.

### Welche Programmiersprachen kann ich mit Redis verwenden?
Redis bietet Client-Bibliotheken für fast alle gängigen Programmiersprachen, einschließlich Python, JavaScript, Java, Go, C, C++, Ruby, PHP und viele mehr.

## Installationsfragen

### Wie installiere ich Redis unter Windows?
Redis ist hauptsächlich für Unix-ähnliche Systeme konzipiert. Sie können jedoch verwenden:
- Windows Subsystem for Linux (WSL)
- Memurai (Redis-kompatible Alternative)
- Docker-Container

### Wie installiere ich Redis unter Linux/macOS?
```bash
# Ubuntu/Debian
sudo apt-get install redis-server

# macOS (mit Homebrew)
brew install redis
```

### Wie starte ich den Redis-Server?
```bash
redis-server
```

Verbinden mit Redis:
```bash
redis-cli
```

## Leistungsfragen

### Wie schnell ist Redis?
Redis kann Millionen von Operationen pro Sekunde mit Sub-Millisekunden-Latenz durchführen.

### Wie viel Daten kann Redis speichern?
Redis kann Hunderte von Millionen von Schlüsseln speichern. Die Hauptbeschränkung ist der verfügbare RAM.

### Unterstützt Redis Clustering?
Ja, Redis Cluster bietet automatische Datenpartitionierung über mehrere Knoten mit hoher Verfügbarkeit.

## Fehlerbehebung

### Warum verbraucht Redis zu viel Speicher?
- Überprüfen Sie große Schlüssel oder Werte
- Nutzen Sie `INFO memory` zur Überprüfung
- Verwenden Sie geeignete Datenstrukturen
- Aktivieren Sie Löschrichtlinien

### Wie persistiere ich Daten in Redis?
Konfigurieren Sie Persistenz in `redis.conf`:
- RDB: Periodische Snapshots
- AOF: Append-Only-File-Logging

### Wie sichere ich Redis?
- Nur an Localhost binden
- Passwort-Authentifizierung aktivieren
- TLS-Verschlüsselung aktivieren
