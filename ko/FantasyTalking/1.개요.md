# FantasyTalking - 개요

AI 기반 Talking Head 비디오 생성 도구.

## Talking Head란?

Talking Head는 정적 이미지와 오디오에서 사실적인 대화 비디오를 생성하는 AI 기술입니다.

## 기능

- 단일 이미지 생성
- 다국어 지원
- 고품질 출력
- 얼굴 향상
- 사용하기 쉬움
- 오픈 소스
