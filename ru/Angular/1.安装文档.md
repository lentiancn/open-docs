# Angular Руководство по установке

Angular - современный веб-фреймворк от Google.

## Установка

```bash
npm install -g @angular/cli
ng new my-app
cd my-app
ng serve
```

## Настройка

```javascript
{
  "compilerOptions": {
    "target": "ES2022"
  }
}
```
