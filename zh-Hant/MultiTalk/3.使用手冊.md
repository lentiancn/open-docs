# MultiTalk 使用手冊

本手冊詳細說明 MultiTalk 的各項功能和使用方法。

## 開始使用

### 建立房間

1. 點擊主畫面的「建立房間」按鈕
2. 輸入房間名稱
3. 設定房間密碼（可選）
4. 選擇房間類型（公開/私人）
5. 點擊「建立」

### 加入房間

1. 點擊「加入房間」
2. 輸入房間ID或選擇公開房間
3. 輸入密碼（如有）
4. 點擊「加入」

## 音頻控制

### 按住說話 (PTT)

- 預設熱鍵：左邊 Shift 鍵
- 可在設定中自訂熱鍵
- 按住時會顯示綠色指示燈

### 語音啟動 (VOX)

- 偵測到聲音時自動傳輸
- 可調整靈敏度
- 適合不想按鍵的使用者

### 音量控制

- 個別調整每位參與者的音量
- 總音量控制
- 靜音功能

## 頻道管理

### 建立頻道

1. 在房間設定中點擊「新增頻道」
2. 輸入頻道名稱
3. 設定頻道權限
4. 完成建立

### 頻道類型

- **語音頻道** - 即時語音聊天
- **文字頻道** - 文字訊息
- **舞台頻道** - 演講模式

## 參與者管理

### 房主權限

- 踢出參與者
- 禁言參與者
- 轉讓房主權限
- 關閉房間

### 主持人權限

- 管理參與者
- 調整音頻設定
- 建立臨時頻道

## 進階功能

### 音頻濾波

- 降噪
- 回音消除
- 增益控制
- 均衡器

### 快捷鍵

| 功能 | 快捷鍵 |
|------|--------|
| 按住說話 | Shift |
| 靜音切換 | Ctrl+M |
|  deafen | Ctrl+D |
| 離開房間 | Ctrl+Q |

### 音效

- 支援自訂音效
-  entering/-leaving sounds
- 點擊音效

## 設定

### 音頻設定

1. 開啟「設定」
2. 選擇「音頻」
3. 選擇輸入/輸出裝置
4. 調整音量
5. 設定噪音抑制

### 快捷鍵設定

1. 開啟「設定」
2. 選擇「快捷鍵」
3. 點擊要修改的動作
4. 按下新的快捷鍵
5. 儲存變更

### 通知設定

1. 開啟「設定」
2. 選擇「通知」
3. 設定通知偏好
4. 選擇聲音

## 疑難排解

### 音訊延遲

- 檢查網路連線
- 關閉其他應用程式
- 更新音訊驅動程式

### 爆音

- 檢查麥克風位置
- 降低麥克風增益
- 使用耳機

### 連線問題

- 檢查網路
- 嘗試其他伺服器
- 檢查防火牆設定

## 快捷鍵總覽

- **Ctrl+N** - 新建房間
- **Ctrl+J** - 加入房間
- **Ctrl+M** - 靜音切換
- **Ctrl+D** -  deafen切換
- **Ctrl+Q** - 離開房間
- **Ctrl+,** - 開啟設定
- **Escape** - 關閉彈出視窗
