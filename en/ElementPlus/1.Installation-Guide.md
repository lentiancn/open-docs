# Element Plus Installation Guide

Comprehensive guide for installing Element Plus on all major platforms.

---

## What is Element Plus?

Element Plus is a Vue 3 based component library for developers, designers and product managers. It provides high quality UI components for building responsive websites.

**Note**: Element Plus is for Vue 3. For Vue 2, use Element UI.

---

## CDN

### Quick Start

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Element Plus CDN</title>
  <link rel="stylesheet" href="https://unpkg.com/element-plus/dist/index.css">
</head>
<body>
  <div id="app">
    <el-button>Button</el-button>
  </div>

  <script src="https://unpkg.com/vue@3"></script>
  <script src="https://unpkg.com/element-plus"></script>
  <script>
    const { createApp } = Vue
    const App = {
      components: { ElButton: ElementPlus.ElButton }
    }
    createApp(App).use(ElementPlus).mount('#app')
  </script>
</body>
</html>
```

### CDN Providers

| Provider | CSS | JS |
|----------|-----|-----|
| unpkg | https://unpkg.com/element-plus/dist/index.css | https://unpkg.com/element-plus |
| jsDelivr | https://cdn.jsdelivr.net/npm/element-plus/dist/index.css | https://cdn.jsdelivr.net/npm/element-plus |

## NPM

### Install Element Plus

```bash
npm install element-plus
```

### Install Icons

```bash
npm install @element-plus/icons-vue
```

---

## Vite

### Create Project

```bash
npm create vite@latest my-element-app -- --template vue
cd my-element-app
npm install
npm install element-plus
npm install @element-plus/icons-vue
```

### vite.config.js

```javascript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [vue()],
})
```

---

## main.js

```javascript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)
app.use(ElementPlus)
app.mount('#app')
```

### With Icons

```javascript
import * as icons from '@element-plus/icons-vue'

// Register all icons
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```
