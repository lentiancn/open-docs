# Guide d'installation Spring Boot

Ce guide couvre l'installation et la configuration de l'environnement de développement Spring Boot sur Windows, macOS et Linux.

## Configuration requise

### Version Java

| Version Spring Boot | Version Java minimale |
|-------------------|---------------------|
| 3.x               | Java 17+            |
| 2.7.x             | Java 8+             |

## Installer Java

### Windows

1. Télécharger JDK depuis [Oracle](https://www.oracle.com/java/technologies/downloads/) ou [Adoptium](https://adoptium.net/)
2. Exécuter l'installateur
3. Configurer les variables d'environnement:
   ```
   JAVA_HOME=C:\Program Files\Java\jdk-21
   PATH=%JAVA_HOME%\bin;%PATH%
   ```

### macOS

```bash
brew install openjdk@21
echo 'export JAVA_HOME=$(/usr/libexec/java_home -v 21)' >> ~/.zshrc
source ~/.zshrc
```

### Linux

```bash
sudo apt update
sudo apt install openjdk-21-jdk
export JAVA_HOME=/usr/lib/jvm/java-21-openjdk-amd64
export PATH=$JAVA_HOME/bin:$PATH
```

## Installer Maven

### Windows

1. Télécharger depuis [Apache Maven](https://maven.apache.org/download.cgi)
2. Extraire dans `C:\apache-maven`
3. Configurer les variables d'environnement

### macOS

```bash
brew install maven
```

### Linux

```bash
sudo apt install maven
```

## Configuration IDE

### IntelliJ IDEA (Recommandé)

1. Télécharger depuis [JetBrains](https://www.jetbrains.com/idea/download/)
2. Configurer JDK: `File` → `Project Structure` → `Project SDK`

### VS Code

- Installer Extension Pack for Java
- Installer Spring Boot Extension Pack

## Démarrage rapide

### Utiliser Spring Initializr

1. Visiter [start.spring.io](https://start.spring.io/)
2. Sélectionner:
   - Projet: Maven / Gradle
   - Langage: Java
   - Spring Boot: 3.2.x
   - Group: com.example
   - Artifact: myapp
3. Ajouter les dépendances:
   - Spring Web
   - Spring Data JPA
   - H2 Database

### Ligne de commande

```bash
curl https://start.spring.io/starter.zip \
  -d type=maven-project \
  -d language=java \
  -d bootVersion=3.2.5 \
  -d baseDir=myapp \
  -d dependencies=web,jpa,h2 \
  -o myapp.zip

unzip myapp.zip
cd myapp
./mvnw spring-boot:run
```

## Vérifier l'installation

```bash
java -version
mvn -version
```
