# FantasyTalking Usage Guide

This guide covers how to use FantasyTalking for generating talking head videos.

## Quick Start

### Basic Usage

```bash
python inference.py \
  --source_image images/face.jpg \
  --audio audio/speech.wav \
  --output results/output.mp4
```

## Input Requirements

### Image Requirements

- Format: JPG, PNG
- Resolution: 512x512 or higher
- Face: Front-facing, clear features
- Background: Simple background preferred

### Audio Requirements

- Format: WAV, MP3
- Sample Rate: 16000 Hz or higher
- Duration: No limit
- Language: Multi-language support

## Advanced Usage

### Face Enhancement

```bash
python inference.py \
  --source_image images/face.jpg \
  --audio audio/speech.wav \
  --output results/output.mp4 \
  --face_enhancer
```

### Batch Processing

```bash
python batch_inference.py \
  --image_dir images/ \
  --audio_dir audio/ \
  --output_dir results/
```

### Custom FPS

```bash
python inference.py \
  --source_image images/face.jpg \
  --audio audio/speech.wav \
  --output results/output.mp4 \
  --fps 30
```

## Configuration

### Config File

Edit `config.yaml`:

```yaml
inference:
  face_enhancer: true
  fps: 25
  output_resolution: 512
  
model:
  checkpoint: "checkpoints/fantasytalking.pth"
```

## Model Training

### Prepare Dataset

```bash
python prepare_dataset.py \
  --video_dir videos/ \
  --output_dir dataset/
```

### Training

```bash
python train.py \
  --config config/train.yaml \
  --epochs 100
```

## Best Practices

### 1. Use High-Quality Images

- Use clear, front-facing photos
- Good lighting conditions
- High resolution preferred

### 2. Match Audio Quality

- Clear audio without background noise
- Proper sample rate
- Matching language with face ethnicity

### 3. Post-Processing

- Use face enhancer for better quality
- Adjust FPS as needed

## Troubleshooting

### Video Too Dark?

Check lighting in source image.

### Audio Not Synced?

Ensure audio sample rate is correct.

### Poor Quality?

Try face enhancement option.
