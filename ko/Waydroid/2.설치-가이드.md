### Fedora

Waydroid는 공식 패키지 저장소에서 설치할 수 있습니다:

```bash
sudo dnf install waydroid
```

설치 후 애플리케이션 메뉴에서 Waydroid를 실행하고 OTA 필드에 다음 URL을 붙여넣어 초기화를 진행하세요:

시스템 OTA: `https://ota.waydro.id/system`

벤더 OTA: `https://ota.waydro.id/vendor`

#### Silverblue/Kinoite/...

Fedora 불변형 변형에도 동일한 지침이 적용되지만, `dnf` 대신 `rpm-ostree`를 사용해야 합니다:

```bash
rpm-ostree install waydroid
```

### Ubuntu/Debian 및 파생 제품

Wayland 세션이 활성화되어 있는지 확인하세요 (Ubuntu 22.04+)

* 필수 요건 설치

```bash
sudo apt install curl ca-certificates -y
```

* 공식 저장소 추가

```bash
curl -s https://repo.waydro.id | sudo bash
```

스크립트가 배포판을 감지하지 못하는 경우, `-s <DISTRO>`를 추가하여 유효한 옵션을 제공할 수 있습니다. 현재 지원되는 값은 다음과 같습니다: **mantic**, **focal**, **jammy**, **kinetic**, **lunar**, **noble**, **plucky**, **questing**, **bookworm**, **bullseye**, **trixie**, **sid**

* waydroid 설치

```bash
sudo apt install waydroid -y
```

그런 다음 애플리케이션 메뉴에서 Waydroid를 시작합니다.

## 문제 해결

### Waydroid 수동 시작

systemctl 없이 Waydroid를 시작하려면 몇 가지 간단한 단계를 따라야 합니다:

**먼저 컨테이너를 시작합니다:**

```bash
sudo waydroid container start
```

**그리고 새 터미널 탭에서 waydroid 세션을 시작합니다(***sudo*** 없이):**

```bash
waydroid session start
```

시작 후 "Android with user 0 is ready"라는 메시지를 보면 애플리케이션 메뉴에서 앱을 안전하게 실행할 수 있습니다.

### Waydroid 전체 화면 모드로 실행

```bash
waydroid show-full-ui
```

### Waydroid 멀티 윈도우 모드로 실행

먼저 Waydroid 세션이 실행 중일 때 속성을 설정해야 합니다:

```bash
waydroid prop set persist.waydroid.multi_windows true
```

그 후 세션을 재시작할 수 있습니다:

```bash
waydroid session stop
```

이제 앱을 실행할 준비가 되었으며, 멀티 윈도우 모드로 시작됩니다.

### Waydroid 재설치

완전히 재설치해야 하는 경우:

```bash
waydroid session stop
sudo waydroid container stop
sudo apt remove waydroid  # 또는 해당 배포판의 제거 명령어
# 재부팅 후 정리 작업 수행
sudo rm -rf /var/lib/waydroid /home/.waydroid ~/waydroid ~/.share/waydroid ~/.local/share/applications/*aydroid* ~/.local/share/waydroid
```
