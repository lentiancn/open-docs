# Flutter

Open-Source UI-Toolkit von Google.

## Übersicht

Flutter ist ein plattformübergreifendes UI-Toolkit mit Dart.

## Funktionen

- Plattformübergreifend
- Hot Reload
- Reiche Widgets
- Hohe Leistung
