# Flutter 安装指南

## 环境要求

在安装 Flutter 之前，需要确保开发环境满足以下要求。操作系统方面，Flutter 支持 Windows、macOS 和 Linux。Windows 需要 Windows 10 或更高版本，macOS 需要 macOS 10.14 或更高版本，Linux 需要支持 GNOME 或 KDE 桌面环境。

磁盘空间方面，Flutter SDK 本身占用约 1GB，但考虑到开发项目和编译缓存，建议预留至少 10GB 的可用空间。内存方面，8GB 或更多内存可以获得更流畅的开发体验。

对于移动开发，iOS 开发需要 macOS 和 Xcode，Android 开发需要安装 Android Studio 和 Android SDK。Web 和桌面开发没有特殊要求，只需安装 Flutter SDK 即可。

## 安装步骤

### Windows 安装

在 Windows 上安装 Flutter 有两种主要方式。第一种是从 Flutter 官网下载预编译的压缩包，解压到合适的位置，然后将 flutter/bin 目录添加到系统 PATH 环境变量中。第二种是通过 Git 克隆 Flutter 仓库。

下载方式：访问 Flutter 官网 flutter.dev，下载最新的 stable 版本压缩包。解压到 C:\src\flutter 或其他合适的位置。打开「系统属性」→「高级」→「环境变量」，在用户变量的 Path 中添加 Flutter 的 bin 目录路径。

完成后，打开命令提示符，运行 flutter doctor 命令检查安装是否成功。Flutter doctor 会检测 Flutter SDK 的状态，并给出配置建议。

### macOS 安装

在 macOS 上，推荐使用 Git 方式安装。首先确保已安装 Xcode，然后打开终端，执行以下命令克隆 Flutter：

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

将 Flutter 添加到 PATH：

```bash
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.bash_profile
```

或者使用 zsh 的用户：

```bash
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.zshrc
```

然后执行 source ~/.bash_profile 或 source ~/.zshrc 刷新环境变量。

### Linux 安装

Linux 上的安装与 macOS 类似。首先安装依赖：

```bash
sudo apt-get update
sudo apt-get install curl git unzip xz-utils zip libglu1-mesa clang cmake ninja-build pkg-config libgtk-3-dev
```

然后克隆 Flutter 仓库并配置 PATH，与 macOS 步骤相同。

## 验证安装

安装完成后，运行以下命令验证：

```bash
flutter --version
flutter doctor
```

flutter doctor 会显示安装状态和任何缺失的依赖。按照提示完成必要的配置，如安装 Android Studio、配置 Android SDK 等。

## IDE 配置

### VS Code

VS Code 是 Flutter 推荐的编辑器之一。安装 VS Code 后，打开扩展市场，搜索 Flutter 和 Dart 扩展并安装。安装完成后，重启 VS Code，即可使用 Flutter 相关功能。

### Android Studio

安装 Android Studio 后，打开 Plugins 页面，搜索 Flutter 插件并安装。安装完成后，配置 Flutter SDK 路径：打开 Preferences → Language & Frameworks → Flutter，设置 Flutter SDK 路径。

## 快速开始

创建第一个 Flutter 项目：

```bash
flutter create my_first_app
cd my_first_app
flutter run
```

或者在 VS Code 中使用命令面板（Ctrl+Shift+P），输入 Flutter: New Project 来创建新项目。

运行 flutter devices 可以查看已连接的设备，flutter run -d device_id 可以指定设备运行。
