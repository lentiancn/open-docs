# Element UI 설치 가이드

> 설정

---

## 필요 환경

| 환경 | 최소 버전 |
|------|----------|
| Node.js | 8.x |
| npm | 5.x |
| Vue | 2.6.0 |

---

## 설치

```bash
npm install element-ui --save
```

---

## 가져오기

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```
