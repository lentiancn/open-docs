## Neuronales Netzwerk

```python
import torch.nn as nn

class Netz(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc = nn.Linear(784, 10)
        
    def forward(self, x):
        x = x.view(-1, 784)
        return self.fc(x)
```
