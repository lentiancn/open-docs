# Element Plus

Vue 3용 UI 컴포넌트 라이브러리.

## 개요

Element Plus는 Ele.me 팀에서 개발한 Vue 3 컴포넌트 라이브러리로, 70개 이상의 고품질 컴포넌트를 제공합니다.

### 주요 기능

- **Vue 3**: Composition API
- **TypeScript**: 완전한 타입 지원
- **Tree Shaking**: 최적화
- **테마**: CSS 변수

### 통계

- GitHub 별표: 20,000+

## 빠른 시작

### 설치

```bash
npm install element-plus
```

### 사용

```vue
<el-button type="primary">버튼</el-button>
```
