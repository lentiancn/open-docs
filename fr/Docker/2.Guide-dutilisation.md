# Guide d'utilisation Docker

Commandes Docker de base.

---

## Concepts de base

| Concept | Description |
|---------|-------------|
| Image | Modèle pour créer des conteneurs |
| Conteneur | Instance en cours d'exécution d'une image |
| Registre | Emplacement de stockage des images |

---

## Commandes Images

```bash
docker pull nginx      # Télécharger
docker images          # Lister
docker rmi nginx       # Supprimer
docker build -t nom . # Construire
```

---

## Commandes Conteneurs

```bash
docker run -d nginx              # Arrière-plan
docker run -d -p 8080:80 nginx   # Port
docker run -d -v /data:/data nginx # Volume

docker ps             # Lister
docker stop <id>     # Arrêter
docker start <id>    # Démarrer
docker rm <id>       # Supprimer
docker logs <id>     # Journaux
docker exec -it <id> bash # Shell
```

---

## Docker Compose

```bash
docker compose up -d    # Démarrer
docker compose down    # Arrêter
```
