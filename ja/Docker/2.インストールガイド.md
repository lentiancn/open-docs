# Docker インストールガイド

このガイドでは、さまざまなオペレーティングシステムに Docker をインストールするための詳細な手順を提供します。

## インストール方法

Docker はいくつかの方法でインストールできます：

1. **Docker Desktop**: デスクトップアプリケーションに Docker Engine を含み，大多数のユーザーに推奨
2. **Docker Engine のみ**: Docker のコアコンポーネントのみをインストールし，Linux サーバーに適している
3. **Docker Engine（apt リポジトリ経由）**: Docker の公式 apt リポジトリからインストール
4. **手動インストール**: deb パッケージをダウンロードして手動でインストール

## Docker Desktop のインストール

Docker Desktop は，最も簡単で最速の方法です。完全な Docker 開発環境，包含 Docker Engine、Docker CLI、Docker Compose、Docker Build、Kubernetes が含まれます。

### システム要件

**macOS:**

- macOS 10.15 (Catalina) 以降
- 4GB 以上の RAM
- 4GB 以上の空きディスク容量

**Windows:**

- Windows 10 Pro，Enterprise，Education (64-bit)
- 4GB 以上の RAM
- Hyper-V または WSL 2 が有効

**Linux:**

- Ubuntu 20.04 以降
- Debian 11 以降
- Fedora 33 以降
- 4GB 以上の RAM

### macOS へのインストール

1. 公式ウェブサイトから Mac 用 Docker Desktop をダウンロード
2. .dmg ファイルをダブルクリック
3. Docker アイコンを「アプリケーション」フォルダにドラッグ
4. Docker Desktop を起動

### Windows へのインストール

1. Windows 用 Docker Desktop をダウンロード
2. インストーラーを実行
3. インストールウィザードの指示に従う
4. Docker Desktop を再起動

### Linux へのインストール (Ubuntu)

```bash
# パッケージリストを更新
sudo apt-get update

# 必要な依存関係をインストール
sudo apt-get install ca-certificates curl gnupg lsb-release

# Docker GPG キーを追加
sudo mkdir -p /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg

# リポジトリを設定
echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

# Docker をインストール
sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

## Docker Engine のインストール

デスクトップなしの Docker Engine のみをインストールしたい場合は，次の手順に従ってください。

### Ubuntu と Debian

```bash
sudo apt-get update
sudo apt-get install docker.io
sudo systemctl start docker
sudo systemctl enable docker
```

### Fedora

```bash
sudo dnf install docker-ce
sudo systemctl start docker
sudo systemctl enable docker
```

## インストールを確認する

インストール後，次のコマンドでインストールを確認できます：

```bash
docker --version
docker run hello-world
```

すべてが機能している場合，Docker からの歓迎メッセージが表示されます。
