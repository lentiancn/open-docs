# Bootstrap

Bootstrap 是全球最受欢迎的 HTML、CSS 和 JavaScript 开源前端开发框架，由 Twitter 公司开发维护。它提供了丰富的组件和强大的 JavaScript 插件，能够帮助开发者快速构建响应式 Web 应用程序。

## 快速开始

### CDN 引入

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### npm 安装

```bash
npm install bootstrap@5
```

## 核心功能

- **响应式布局**：12 列网格系统，适应各种屏幕
- **丰富组件**：导航栏、卡片、表单、模态框等
- **JavaScript 插件**：下拉菜单、轮播图、折叠等
- **Sass 支持**：轻松自定义主题
