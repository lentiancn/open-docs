# JUnit Introduction

JUnit est le framework de tests unitaires le plus populaire pour Java, utilisé pour écrire et exécuter des tests répétables. C'est la base du développement de code Java de haute qualité et il est largement utilisé par des millions de développeurs dans le monde.

## JUnit 4 contre JUnit 5

### Aperçu des versions

- **JUnit 4** : Publié en 2010, la dernière version majeure du framework de tests utilisé pendant longtemps par la communauté Java
- **JUnit 5** : Publié en 2017, une refonte complète du framework JUnit, composée de trois sous-projets : JUnit Platform, JUnit Jupiter et JUnit Vintage

### Différences architecturales

JUnit 4 utilise une architecture monolithique avec toutes les fonctionnalités dans une seule bibliothèque. JUnit 5 utilise une architecture modulaire avec trois projets distincts :

- **JUnit Platform** : La base pour lancer les frameworks de tests, fournissant l'API TestEngine et le lanceur console
- **JUnit Jupiter** : Le nouveau modèle de programmation et modèle d'extension pour écrire des tests et des extensions
- **JUnit Vintage** : Le moteur de compatibilité qui permet d'exécuter des tests JUnit 3 et JUnit 4 sur la plateforme JUnit 5

### Différences principales

| Fonctionnalité | JUnit 4 | JUnit 5 |
|-----------------|---------|---------|
| Version Java minimale | Java 5+ | Java 8+ |
| Package d'annotations | org.junit | org.junit.jupiter.api |
| Classe de test sans héritage | Non | Oui |
| Style d'assertions | Méthodes statiques Assert | Méthodes statiques Assertions |
| Tests paramétrés | @Parameters | @ParameterizedTest |
| Tests imbriqués | Non supporté | Supporté |
| Tests dynamiques | Non supporté | Supporté |
| Exécution parallèle | Support limité | Support natif |
| Modèle d'extension | @RunWith | @ExtendWith |

### Recommandations de version

- **Nouveaux projets** : Utilisez JUnit 5 pour les fonctionnalités modernes et une meilleure extensibilité
- **Projets existants** : Continuez à utiliser JUnit 4, ou utilisez JUnit Vintage pour une migration progressive
- **Maintenance à long terme** : Migrez progressivement vers JUnit 5 pour une meilleure expérience de test
