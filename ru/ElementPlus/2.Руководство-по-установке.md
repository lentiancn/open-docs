# Element Plus Руководство по установке

Несколько способов установки Element Plus.

## Требования к среде

### Аппаратные требования

- CPU: 2 ядра и более
- Оперативная память: 4 ГБ и более

### Программные требования

- Node.js: 18.0+
- npm: 9.0+ / yarn: 1.22+ / pnpm: 8.0+
- Vue: 3.2+
- Vite: 4.0+ (рекомендуется)

## Установка через менеджер пакетов (рекомендуется)

### npm

```bash
npm install element-plus
```

### yarn

```bash
yarn add element-plus
```

### pnpm

```bash
pnpm add element-plus
```

## Полный импорт

### Настройка main.ts

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')
```

## Импорт по требованию (рекомендуется)

### Автоматический импорт (рекомендуется)

**1. Установить зависимости**

```bash
npm install -D unplugin-vue-components unplugin-auto-import
```

**2. Настроить Vite**

```typescript
// vite.config.ts
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

**3. Использовать компоненты**

```vue
<template>
  <el-button type="primary">Привет</el-button>
</template>
```

### Ручной импорт по требованию

```typescript
import { ElButton, ElInput } from 'element-plus'

export default {
  components: {
    ElButton,
    ElInput,
  },
}
```

## Установка через Vue CLI

### Установить

```bash
npm install -D sass sass-loader
npm install element-plus
```

### Настроить

```javascript
// vue.config.js
module.exports = {
  configureWebpack: {
    plugins: [
      AutoImport({
        resolvers: [ElementPlusResolver()],
      }),
      Components({
        resolvers: [ElementPlusResolver()],
      }),
    ],
  },
}
```

## Импорт через CDN

### HTML

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <title>Element Plus CDN</title>
  <link rel="stylesheet" href="//unpkg.com/element-plus/dist/index.css" />
  <script src="//unpkg.com/vue@3"></script>
  <script src="//unpkg.com/element-plus"></script>
</head>
<body>
  <div id="app">
    <el-button type="primary">Основная кнопка</el-button>
  </div>

  <script>
    const { createApp, ref } = Vue
    const App = {
      setup() {
        const visible = ref(false)
        return { visible }
      },
      template: `
        <el-button @click="visible = true">Открыть диалог</el-button>
        <el-dialog v-model="visible" title="Заголовок">Содержимое</el-dialog>
      `
    }
    const app = createApp(App)
    app.use(ElementPlus)
    app.mount('#app')
  </script>
</body>
</html>
```

## Настройка темы

### Использование CSS-переменных

```css
:root {
  --el-color-primary: #409eff;
  --el-color-success: #67c23a;
  --el-border-radius-base: 4px;
}
```

### Глубокая настройка темы

**1. Установить генератор тем**

```bash
npm install -D element-theme-chalk
```

**2. Инициализировать файлы темы**

```bash
et -i
```

**3. Изменить переменные**

Отредактировать `element-variables.scss`, затем:

```bash
et
```

### Использование тёмной темы

```typescript
import 'element-plus/theme-chalk/dark/css-vars.css'
```

## Распространённые проблемы

### Иконки не отображаются?

Установить пакет иконок:

```bash
npm install @element-plus/icons-vue
```

Зарегистрировать:

```typescript
import * as icons from '@element-plus/icons-vue'

for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```

### Конфликты стилей?

Использовать CSS Modules или scoped:

```vue
<style scoped>
:deep(.el-button) {
  margin: 10px;
}
</style>
```

### Ошибки типов TypeScript?

Убедиться, что определения типов установлены:

```bash
npm install -D @types/node
```

Настроить в `tsconfig.json`:

```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

### Настройка интернационализации

```typescript
import ru from 'element-plus/es/locale/lang/ru'

app.use(ElementPlus, { locale: ru })
```

## Удаление

### Удалить через npm

```bash
npm uninstall element-plus
```

### Очистить настройки

Удалить:
- `element-variables.scss` (если использовалась)
- Связанные настройки Vite/Webpack
