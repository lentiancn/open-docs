# Angular 簡介

現代 Web 應用開發框架。

## 什麼是 Angular？

Angular 是由 Google 開發和維護的開源 Web 應用框架，是現代 Web 應用開發的首選方案之一。

### 核心特性

- **TypeScript 原生**：完全基於 TypeScript 開發
- **組件化架構**：基於組件的開發模式，便於維護和測試
- **依賴注入**：強大的依賴注入系統，提高可測試性
- **指令系統**：強大的指令來擴展 HTML 功能
- **路由系統**：完整的客戶端路由解決方案
- **表單處理**：響應式表單和模板驅動表單
- **HTTP 客戶端**：完整的 HTTP 客戶端服務
- **國際化**：內置 i18n 支持
- **動畫**：強大的動畫 API
- **測試**：完整的單元和 E2E 測試支持

### 技術架構

Angular 使用模組化架構，包含以下核心概念：

- **模組（Module）**：組織應用程序代碼的基本單位
- **組件（Component）**：視圖和業務邏輯的組合
- **服務（Service）**：封裝可複用的業務邏輯
- **指令（Directive）**：擴展 HTML 元素行為
- **管道（Pipe）**：數據轉換工具

## 適用場景

- **企業級應用**：大型複雜業務系統
- **單頁應用（SPA）**：需要流暢用戶體驗的應用
- **漸進式 Web 應用（PWA）**：需要離線功能的應用
- **混合移動應用**：結合 Ionic 開發移動應用
- **儀表盤**：數據可視化管理後台
