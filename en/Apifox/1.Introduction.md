# Introduction

Apifox is a leading API collaboration platform in China that solves API design, mocking, testing, and documentation all in one place. Developed and maintained by Hangzhou Xiongweiguo Network Technology Co., Ltd., it's one of the most popular API management tools in China.

Apifox brings together API design, API documentation management, API debugging, API mocking, and API automated testing. Team members can complete all API-related work on a single platform, which significantly boosts development efficiency.

## Core Features

### API Design

Apifox provides visual API design capabilities and supports RESTful API design standards. You can define endpoint paths, request methods, request parameters, and response parameters through a graphical interface—no need to manually write Markdown or YAML. The system automatically generates standardized API documentation and lets you share it with team members with one click.

### API Documentation Management

Apifox automatically generates API documentation that you can browse online or export. Once your API definitions are complete, the documentation syncs automatically, ensuring team members always see the latest interface specifications. The documentation supports exporting in multiple formats, including Markdown, HTML, and OpenAPI (Swagger), making it easy to integrate with external systems.

### API Debugging

The powerful online API debugging tool lets you send various types of HTTP requests, including GET, POST, PUT, DELETE, and PATCH. You can customize request headers, parameters, and body data, supporting multiple formats like forms, JSON, and XML. Response content is displayed with formatting, including JSON, XML, and HTML, complete with syntax highlighting and code folding.

### API Mocking

Smart mocking is one of Apifox's standout features. Once you define your data structure, the system automatically generates compliant mock data—there's no need to write complex mock rules manually. It can automatically generate data based on field types like strings, numbers, booleans, arrays, and objects. You can also create custom mock rules for more complex scenarios.

### Automated Testing

Apifox provides visual automated testing functionality. You can configure test cases through a graphical interface, set assertion conditions and loop counts. You can complete API automated testing without writing code, which greatly lowers the testing barrier. It supports batch execution of test cases and generates detailed test reports, helping teams quickly find and locate issues.

### Team Collaboration

Apifox has robust team collaboration features—you can invite team members to join projects and set different roles and permissions. Project content syncs in real-time, so team members always see the latest version. It supports comments and annotations, making it easy for team members to communicate directly on interfaces.

## Use Cases

- API Automated Testing
- Team Collaboration and Knowledge Sharing

## How It Compares to Other Tools

Compared to Postman, Apifox is more popular among Chinese users, with key advantages including: better Chinese language support, smarter mocking features, and a more polished team collaboration experience. The basic features are completely free. Compared to Swagger, Apifox offers a more intuitive visual interface that's easier to get started with—you can complete API management and testing without writing any code.

As a leading API tool in China, Apifox continues to iterate and update, providing developers with quality technical support and services.
