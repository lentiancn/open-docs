# Element Plus インストールガイド

Element Plusは、開発者、デザイナー、プロダクトマネージャーのためのVue 3コンポーネントライブラリです。このガイドでは、すべての環境でのインストール方法を説明します。

## バージョン情報

| バージョン | 状態 | 説明 |
|------------|------|------|
| Element Plus 2.x | ✅ 現行 | Vue 3 最新安定版 |
| Element Plus 1.x | ⚠️ メンテナンス | セキュリティ更新のみ |
| Element UI | ❌ 旧版 | Vue 2 のみサポート |

> **重要**：Element PlusはVue 3のみサポートしています。Vue 2を使用する場合は、Element UIを使用してください。

## インストール方法

### 方法1：npm（推奨）

```bash
# npm を使用
npm install element-plus

# yarn を使用
yarn add element-plus

# pnpm を使用
pnpm add element-plus
```

### 方法2：CDN

```html
<!-- Vue 3 を読み込み -->
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<!-- Element Plus を読み込み -->
<link rel="stylesheet" href="https://unpkg.com/element-plus/dist/index.css" />
<script src="https://unpkg.com/element-plus/dist/index.full.js"></script>

<!-- アイコンを読み込み（オプション） -->
<script src="https://unpkg.com/@element-plus/icons-vue"></script>
```

#### 完全なCDNテンプレート

```html
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Element Plus CDN</title>
    <link rel="stylesheet" href="https://unpkg.com/element-plus/dist/index.css">
</head>
<body>
    <div id="app">
        <el-button>ボタン</el-button>
        <el-date-picker></el-date-picker>
    </div>

    <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
    <script src="https://unpkg.com/element-plus/dist/index.full.js"></script>
    <script src="https://unpkg.com/@element-plus/icons-vue"></script>
    
    <script>
        const { createApp, ref } = Vue
        const app = createApp({
            setup() {
                return {}
            }
        })
        
        for (const [key, component] of Object.entries(ElementPlus)) {
            app.component(key, component)
        }
        
        for (const [key, component] of Object.entries(ElementPlusIconsVue)) {
            app.component(key, component)
        }
        
        app.use(ElementPlus)
        app.mount('#app')
    </script>
</body>
</html>
```

### 方法3：Viteを使用

```bash
# Viteプロジェクトを作成
npm create vite@latest my-app -- --template vue

# プロジェクトディレクトリに移動
cd my-app

# Element Plusをインストール
npm install element-plus

# アイコンをインストール
npm install @element-plus/icons-vue
```

#### main.jsで設定

```javascript
import { createApp } from 'vue'
import App from './App.vue'

// Element Plusのスタイルを読み込み
import 'element-plus/dist/index.css'

// Element Plusを読み込み
import ElementPlus from 'element-plus'

// アイコンを読み込み
import * as ElementPlusIconsVue from '@element-plus/icons-vue'

const app = createApp(App)

// すべてのアイコンをグローバル登録
for (const [key, component] of Object.entries(ElementPlusIconsVue)) {
    app.component(key, component)
}

app.use(ElementPlus)
app.mount('#app')
```

### 方法4：Vue CLIを使用

```bash
# Vue CLIをインストール
npm install -g @vue/cli

# プロジェクトを作成
vue create my-project

# Element Plusを追加
cd my-project
npm install element-plus
npm install @element-plus/icons-vue
```

#### main.jsで設定

```javascript
import { createApp } from 'vue'
import App from './App.vue'

import 'element-plus/dist/index.css'
import ElementPlus from 'element-plus'
import * as icons from '@element-plus/icons-vue'

const app = createApp(App)

// アイコンをグローバル登録
for (const [key, component] of Object.entries(icons)) {
    app.component(key, component)
}

app.use(ElementPlus)
app.mount('#app')
```

### 方法5：オンデマンドインポート（本番環境推奨）

#### unplugin-vue-componentsを使用

```bash
# 依存関係をインストール
npm install element-plus unplugin-vue-components unplugin-auto-import
```

#### Vite設定

```javascript
// vite.config.js
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

#### 使用方法

```vue
<template>
  <!-- コンポーネントをインポートせずに使用可能 -->
  <el-button type="primary">ボタン</el-button>
  <el-input v-model="input" placeholder="入力してください" />
</template>

<script setup>
import { ref } from 'vue'
const input = ref('')
</script>
```

## よくある問題

### アイコンが表示されない？

アイコンをグローバル登録していることを確認してください：

```javascript
import * as icons from '@element-plus/icons-vue'
for (const [key, icon] of Object.entries(icons)) {
    app.component(key, icon)
}
```

### スタイルが効かない？

Element Plusのスタイルを読み込んでいることを確認してください：

```javascript
import 'element-plus/dist/index.css'
```

### Vueバージョンが合わない？

Element PlusはVue 3が必要です。Vueバージョンを確認：

```bash
npm list vue
```
