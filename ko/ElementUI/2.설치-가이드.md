# Element UI 설치 가이드

이 가이드에서는 다양한 프로젝트에서 Element UI를 설치하고 사용하는 방법에 대한 자세한 지침을 제공합니다.

## 환경 요구사항

Element UI는 다음 요구사항이 있습니다:

- Vue 2.0.0+
- Node.js 8.0.0+
- 최신 브라우저 (Chrome, Firefox, Safari, Edge 10+)

## 설치 방법

### 패키지 관리자 사용

npm을 사용하여 설치하는 것이 권장됩니다:

```bash
npm install element-ui --save
```

또는 yarn 사용:

```bash
yarn add element-ui
```

## 전체 가져오기

main.js에서:

```javascript
import Vue from 'vue'
import Element from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(Element)

new Vue({
  el: '#app',
  render: h => h(App)
})
```

### 필요 시 가져오기

필요 시 가져오기는 프로젝트 크기를 상당히 줄일 수 있습니다.

먼저 babel-plugin-component를 설치합니다:

```bash
npm install babel-plugin-component -D
```

.babelrc:

```json
{
  "presets": [["env", {
    "modules": false,
    "targets": {
      "browsers": ["> 1%", "last 2 versions", "not ie <= 8"]
    }
  }]],
  "plugins": [
    ["component", {
      "libraryName": "element-ui",
      "styleLibraryName": "theme-chalk"
    }]
  ]
}
```

필요 시 컴포넌트 가져오기:

```javascript
import { Button, Select } from 'element-ui'

Vue.component(Button.name, Button)
Vue.component(Select.name, Select)
```

## 시작 템플릿

Element UI는 공식 프로젝트 템플릿을 제공합니다:

- **Vue CLI 템플릿**: https://github.com/ElementUI/element-starter
- **Laravel 템플릿**: https://github.com/ElementUI/element-in-laravel-starter

## 전역 구성

Element UI를 가져올 때 전역 구성 객체를 전달할 수 있습니다:

```javascript
import Vue from 'vue'
import Element from 'element-ui'

Vue.use(Element, {
  size: 'small',
  zIndex: 2000
})
```

구성 옵션:
- size: 폼 컴포넌트의 기본 크기 설정 (medium / small / mini)
- zIndex: 팝업 컴포넌트의 z-index 설정, 기본값은 2000

자세한 내용은 [Element UI 공식 웹사이트](https://element.eleme.io/)를 방문하세요.
