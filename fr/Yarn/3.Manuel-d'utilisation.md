# Manuel d'Utilisation

## Utilisation Quotidienne

### Initialiser Projet

```bash
yarn init -2
```

### Installer Dépendances

```bash
yarn install
```

### Ajouter

```bash
yarn add lodash
yarn add -D typescript
```

### Supprimer

```bash
yarn remove lodash
```

### Mettre à Jour

```bash
yarn up lodash
yarn up --all
```
