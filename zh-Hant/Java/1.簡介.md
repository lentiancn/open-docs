# Java 簡介

> Java 是一種物件導向程式語言

---

## 說明

Java 由 Sun Microsystems 於 1995 年創建。以跨平台特性著稱。

---

## 特點

- 物件導向
- 跨平台 (JVM)
- 自動記憶體管理
- 多執行緒
- 豐富生態系統
