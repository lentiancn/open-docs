# Oracle 安裝指南

## 前提條件

- **作業系統**：Oracle Linux、RHEL、Windows Server
- **記憶體**：最少 2GB RAM（建議 8GB+）
- **磁碟空間**：軟體最少 8GB

---

## Linux 安裝

### 1. 建立 Oracle 使用者

```bash
groupadd -g 54321 oinstall
groupadd -g 54322 dba
useradd -u 54321 -g oinstall -G dba oracle
passwd oracle
```

### 2. 設定核心參數

```bash
kernel.shmmax = 4294967296
kernel.shmall = 2097152
```

### 3. 執行安裝程式

```bash
./runInstaller
```

---

## 安裝後驗證

```sql
sqlplus / as sysdba
SELECT name, open_mode FROM v$database;
```
