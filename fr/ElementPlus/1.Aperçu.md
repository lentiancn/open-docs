# Element Plus Aperçu

Bibliothèque de composants UI pour Vue 3.

## Qu'est-ce que Element Plus?

Element Plus est une bibliothèque de composants Vue 3 développée par l'équipe Ele.me, offrant plus de 70 composants de haute qualité.

### Fonctionnalités principales

- **Basé sur Vue 3**: Construit avec l'API Composition
- **TypeScript**: Support complet des types
- **Tree Shaking**: Optimisation du bundle
- **Thèmes**: Personnalisation via variables CSS
- **Internationalisation**: 20+ langues
