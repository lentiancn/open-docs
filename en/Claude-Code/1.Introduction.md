# Claude Code Introduction

Claude Code is an intelligent programming assistant developed by Anthropic that can read codebases, edit files, execute commands, and integrate seamlessly with development tools. It is available in terminals, IDEs, desktop applications, and browsers.

## What is Claude Code

Claude Code is an AI-powered coding assistant that helps you build features, fix bugs, and automate development tasks. It understands your entire codebase and can work across multiple files and tools to complete various development tasks.

## Key Features

Claude Code provides the following core capabilities:

**Automating Development Tasks**
- Writing tests for untested code
- Fixing lint errors across the entire project
- Resolving merge conflicts
- Updating dependency versions
- Writing release notes

**Feature Development and Bug Fixes**
- Describe what you want to implement in natural language
- Claude Code will plan the approach, write code across files, and verify the implementation

**Git Integration**
- Perform Git operations directly in Claude Code
- Stage changes, write commit messages, create branches, open Pull Requests

**MCP Integration**
- Model Context Protocol (MCP) is an open standard for connecting AI tools to external data sources
- Through MCP, Claude Code can read design documents from Google Drive, update Jira tickets, pull data from Slack, or use custom tools

**Customization Capabilities**
- Use CLAUDE.md to set coding standards, architecture decisions, preferred libraries, and review checklists for your project
- Create custom commands to encapsulate repeatable workflows
- Use Hooks to auto-format after file edits or run lint before commits

**Multi-Agent Collaboration**
- Spawn multiple Claude Code agents to work in parallel on complex tasks
- A lead agent coordinates work, assigns subtasks, and merges results

**CLI Flexibility**
- Support for piping, scripting, and automation
- Pipe logs to it, run in CI, or chain with other tools

## Usage Scenarios

Claude Code can be used in various scenarios:

| Scenario | Recommended Method |
|----------|-------------------|
| Local development | Terminal CLI |
| In-IDE coding | VS Code or JetBrains |
| Continue across devices | Remote Control |
| Mobile operations | Web or iOS app |
| CI/CD automation | GitHub Actions or GitLab CI/CD |
| Team collaboration | Slack integration |
| Browser debugging | Chrome extension |

## Account Requirements

Claude Code requires one of the following accounts:
- Claude Pro, Max, Teams, or Enterprise subscription
- Anthropic Console account
- Supported third-party cloud providers (Amazon Bedrock, Google Vertex AI, Microsoft Foundry)

The free Claude.ai plan does not include Claude Code access.
