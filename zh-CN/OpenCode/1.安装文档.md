# OpenCode 安装指南

OpenCode 是一个现代代码编辑器。

## 系统要求

- Windows 10/11, macOS 10.15+, Linux Ubuntu 18.04+
- 4GB+ 内存
- 500MB+ 磁盘空间

## Windows 安装

### Winget
```powershell
winget install OpenCode
```

### Chocolatey
```powershell
choco install opencode
```

## macOS 安装

```bash
brew install opencode
```

## Linux 安装

```bash
sudo apt install opencode
```
