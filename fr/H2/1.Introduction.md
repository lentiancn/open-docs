# Introduction à H2

H2 est une base de données relationnelle open source écrite en Java.

## Caractéristiques

- Base de données intégrée (pas de serveur séparé)
- Mode client-serveur
- Base de données en mémoire
- Compatibilité multi-base de données (MySQL, PostgreSQL, Oracle)
- Chiffrement de base de données
- Recherche full-text intégrée

## Applications

- Développement Java
- Tests unitaires
- Prototypage rapide
