# Flutter - Übersicht

Flutter ist ein Open-Source UI-Toolkit von Google.

## Was ist Flutter?

Flutter ist ein UI-Toolkit zum Erstellen plattformübergreifender Apps für iOS, Android, Web und Desktop aus einer Codebasis. Verwendet Dart.

## Funktionen

- **Plattformübergreifend**: Eine Codebasis
- **Hot Reload**: Sofortige Vorschau
- **Reiche Widgets**: Material Design + Cupertino
- **Hohe Leistung**: Skia Grafik-Engine
- **Open Source**: Kostenlos
