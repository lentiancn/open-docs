# OpenClaw Introduction

## What is OpenClaw?

OpenClaw is a **self-hosted gateway** that connects your favorite chat apps — WhatsApp, Telegram, Discord, iMessage, and more — to AI coding assistants like Pi. You run a single Gateway process on your own machine, and it becomes the bridge between your messaging apps and an always-available AI assistant.

## Key Features

### 1. Multi-Channel Support

- WhatsApp, Telegram, Discord, iMessage
- Plugin channels (Mattermost, etc.)
- Single Gateway for all platforms

### 2. Web Control UI

- Browser dashboard at http://127.0.0.1:18789/
- Chat directly in browser
- Configuration management
- Session and node management

### 3. Multi-Agent Routing

- Isolated sessions per agent/sender
- Multiple AI agents support
- Smart routing based on conditions

### 4. Media Support

- Send/receive images
- Voice messages
- File transfers
- Mobile nodes (iOS/Android)

### 5. Security Controls

- Token management
- Whitelist support
- Group mention rules
- Multiple security policies

## Requirements

- Node.js 22+
- API key (Anthropic recommended)
- 5 minutes to set up

## Quick Start

```bash
# Install
npm install -g openclaw@latest

# Initialize
openclaw onboard --install-daemon

# Start
openclaw gateway --port 18789
```

Then open http://127.0.0.1:18789/

## Who is it for?

```
Chat Apps → Gateway → AI Agent / CLI / Web / Mobile Nodes
```

OpenClaw is a powerful, flexible self-hosted AI gateway. It connects various chat apps with AI assistants, enabling convenient AI access anywhere. If you value data privacy, self-hosting, or multi-channel support, OpenClaw is an excellent choice.
