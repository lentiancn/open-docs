# IBM Informix

IBM Informix es una familia de sistemas de gestión de bases de datos relacionales (RDBMS) y ofertas de bases de datos multimodelo desarrolladas por IBM.

## Descripción de la Documentación

Este directorio contiene la documentación técnica de IBM Informix:

- **Introducción**: Descripción general del producto, características y ediciones
- **Guía de Instalación**: Pasos de instalación para Linux y Windows
- **Manual del Usuario**: Comandos básicos, sentencias SQL y configuración
- **Preguntas Frecuentes**: Preguntas comunes y soluciones

## Información de Versión

- Versión Actual: IBM Informix 15.0.0.1 (Marzo 2025)
