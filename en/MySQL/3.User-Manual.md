# MySQL User Manual

## Basic Database Operations

### Connecting to MySQL

Local connection:
```bash
mysql -u root -p
```

Remote connection:
```bash
mysql -h hostname -u username -p database_name
```

### Creating a Database

```sql
CREATE DATABASE database_name;
```

Create with specific charset:
```sql
CREATE DATABASE database_name CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### Viewing Databases

```sql
SHOW DATABASES;
```

### Selecting a Database

```sql
USE database_name;
```

### Deleting a Database

```sql
DROP DATABASE database_name;
```

## Table Operations

### Creating a Table

```sql
CREATE TABLE table_name (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100) UNIQUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### Viewing Table Structure

```sql
DESCRIBE table_name;
-- or
SHOW COLUMNS FROM table_name;
```

### Viewing All Tables

```sql
SHOW TABLES;
```

### Modifying Table Structure

Add column:
```sql
ALTER TABLE table_name ADD COLUMN new_column VARCHAR(50);
```

Modify column:
```sql
ALTER TABLE table_name MODIFY column_name VARCHAR(200);
```

Drop column:
```sql
ALTER TABLE table_name DROP COLUMN column_name;
```

### Deleting a Table

```sql
DROP TABLE table_name;
```

## Data Operations

### Inserting Data

Single insert:
```sql
INSERT INTO table_name (column1, column2) VALUES ('value1', 'value2');
```

Multiple inserts:
```sql
INSERT INTO table_name (column1, column2) VALUES 
('value1', 'value2'),
('value3', 'value4');
```

### Querying Data

Basic query:
```sql
SELECT * FROM table_name;
```

Conditional query:
```sql
SELECT column1, column2 FROM table_name WHERE condition;
```

Ordering:
```sql
SELECT * FROM table_name ORDER BY column_name DESC;
```

Pagination:
```sql
SELECT * FROM table_name LIMIT 10 OFFSET 20;
```

### Updating Data

```sql
UPDATE table_name SET column1 = 'new_value' WHERE condition;
```

### Deleting Data

```sql
DELETE FROM table_name WHERE condition;
```

Truncate table:
```sql
TRUNCATE TABLE table_name;
```

## Index Operations

### Creating Indexes

Regular index:
```sql
CREATE INDEX index_name ON table_name (column_name);
```

Unique index:
```sql
CREATE UNIQUE INDEX index_name ON table_name (column_name);
```

Composite index:
```sql
CREATE INDEX index_name ON table_name (column1, column2);
```

### Viewing Indexes

```sql
SHOW INDEX FROM table_name;
```

### Deleting Indexes

```sql
DROP INDEX index_name ON table_name;
```

## User and Privilege Management

### Creating Users

```sql
CREATE USER 'username'@'localhost' IDENTIFIED BY 'password';
```

### Granting Privileges

```sql
GRANT SELECT, INSERT, UPDATE ON database_name.* TO 'username'@'localhost';
```

Grant all privileges:
```sql
GRANT ALL PRIVILEGES ON database_name.* TO 'username'@'localhost';
```

### Refreshing Privileges

```sql
FLUSH PRIVILEGES;
```

### Viewing User Privileges

```sql
SHOW GRANTS FOR 'username'@'localhost';
```

### Deleting Users

```sql
DROP USER 'username'@'localhost';
```

## Transaction Control

### Starting a Transaction

```sql
START TRANSACTION;
-- or
BEGIN;
```

### Committing a Transaction

```sql
COMMIT;
```

### Rolling Back a Transaction

```sql
ROLLBACK;
```

### Setting Savepoints

```sql
SAVEPOINT savepoint_name;
ROLLBACK TO SAVEPOINT savepoint_name;
```

## Common Functions

### String Functions

```sql
SELECT CONCAT('Hello', ' ', 'World');
SELECT UPPER('hello');
SELECT LOWER('HELLO');
SELECT SUBSTRING('Hello World', 1, 5);
SELECT LENGTH('Hello');
SELECT TRIM('  Hello  ');
```

### Numeric Functions

```sql
SELECT ABS(-5);
SELECT CEIL(3.14);
SELECT FLOOR(3.14);
SELECT ROUND(3.14159, 2);
SELECT MOD(10, 3);
```

### Date Functions

```sql
SELECT NOW();
SELECT CURDATE();
SELECT CURTIME();
SELECT DATE('2024-01-01 10:30:00');
SELECT YEAR('2024-01-01');
SELECT MONTH('2024-01-01');
SELECT DAY('2024-01-01');
SELECT DATE_ADD('2024-01-01', INTERVAL 1 DAY);
SELECT DATEDIFF('2024-01-01', '2023-12-01');
```

### Aggregate Functions

```sql
SELECT COUNT(*) FROM table_name;
SELECT SUM(column_name) FROM table_name;
SELECT AVG(column_name) FROM table_name;
SELECT MAX(column_name) FROM table_name;
SELECT MIN(column_name) FROM table_name;
```

## Backup and Recovery

### Backing Up a Database

```bash
mysqldump -u root -p database_name > backup.sql
```

Backup all databases:
```bash
mysqldump -u root -p --all-databases > all_databases.sql
```

### Restoring a Database

```bash
mysql -u root -p database_name < backup.sql
```

### Exporting Query Results

```bash
mysql -u root -p -e "SELECT * FROM table_name" database_name > output.tsv
```

## Performance Optimization

### Analyzing Queries with EXPLAIN

```sql
EXPLAIN SELECT * FROM table_name WHERE condition;
```

### Optimization Tips

1. Create indexes for common query conditions
2. Avoid SELECT *, only query needed columns
3. Use LIMIT to restrict result sets
4. Avoid using functions on indexed columns
5. Use JOINs properly, avoid Cartesian products
6. Regularly analyze and optimize tables

### Viewing Slow Query Log

```sql
SHOW VARIABLES LIKE 'slow_query_log%';
```

## Common Error Handling

### Connection Errors
Check if MySQL service is running, port is correct, and user privileges are properly configured.

### Character Set Errors
Ensure database, table, and connection all use the same character set (utf8mb4 recommended).

### Privilege Errors
Confirm the user has appropriate database and table operation privileges.
