# Element Plus 概要

Element PlusはVue 3ベースのUIコンポーネントライブラリです。

## 什么是

Element PlusはElement UIのVue 3バージョンで、饿了么チームによって開発・メンテナンスされています。

## 主な機能

- 70以上のコンポーネント
- Vue 3完全サポート
- TypeScriptサポート
- ツリーシェイキング対応
- 国際化サポート
- テーマカスタマイズ

## デザイン原則

一貫性、フィードバック、効率、操作性。

## 適用シーン

企业管理システム、ECバックエンド、データ可視化プラットフォームなど。
