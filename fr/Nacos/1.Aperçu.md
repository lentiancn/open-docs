# Aperçu de Nacos

Nacos (Dynamic Naming and Configuration Service) est une plateforme de découverte de services et de gestion de configuration dynamique développée par Alibaba.

## Caractéristiques

- Découverte de services
- Gestion de configuration centralisée
- DNS dynamique
- Vérifications de santé

## Installation

```bash
# Mode autonome
wget https://github.com/alibaba/nacos/releases/download/v2.3.0/nacos-server-2.3.0.tar.gz
tar -xvf nacos-server-2.3.0.tar.gz
cd nacos/bin
./startup.sh -m standalone

# Docker
docker run --name nacos -d -p 8848:8848 nacos/nacos-server:v2.3.0
```
