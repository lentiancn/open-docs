# OpenClaw Häufig Gestellte Fragen

Diese Seite beantwortet häufige Fragen zu OpenClaw.

## Grundlagen

### Was ist OpenClaw?

OpenClaw ist ein selbst gehostetes Gateway, das deine Chat-Apps (WhatsApp, Telegram, Discord, etc.) mit KI-Coding-Assistenten verbindet. Es ermöglicht dir, von überall aus mit einem KI-Assistenten zu kommunizieren, während du die volle Kontrolle über deine Daten behältst.

### Ist OpenClaw kostenlos?

Ja, OpenClaw ist ein Open-Source-Projekt unter der MIT-Lizenz. Du musst jedoch selbst tragen:
- Den Server oder Computer zum Ausführen
- Die API-Schlüsselkosten für KI-Dienste

### Brauche ich ein KI-Dienst-Abonnement?

OpenClaw unterstützt mehrere Modell-Anbieter, einschließlich:
- Anthropic (Claude)
- OpenAI (GPT)
- Google Gemini
- Und jeden anderen OpenAI-API-kompatiblen Dienst

Du kannt Pay-as-you-go API-Schlüssel oder Abonnement-Authentifizierung verwenden.

### Welche Plattformen unterstützt OpenClaw?

- **Desktop**: macOS, Linux, Windows
- **Mobil**: iOS, Android (als Knoten)
- **Chat-Apps**: WhatsApp, Telegram, Discord, iMessage, Signal, Slack und 20+ weitere Plattformen

## Installation

### Was tun, wenn die Installation fehlschlägt?

1. Node.js-Version prüfen: `node --version` (22 oder höher erforderlich)
2. Diagnose ausführen: `openclaw doctor`
3. Logs prüfen: `openclaw logs`

### Kann ich es auf einem Raspberry Pi ausführen?

Ja, OpenClaw funktioniert auf dem Raspberry Pi. Wir empfehlen Raspberry Pi 4 oder neuer mit Raspberry Pi OS oder Ubuntu.

### Brauche ich einen Mac?

Nein, OpenClaw läuft auf jedem System, das Node.js unterstützt. Nur der iMessage-Kanal erfordert einen Mac (aufgrund von Apple-Beschränkungen).

## Konfiguration

### Wo ist die Konfigurationsdatei?

Standardort: `~/.openclaw/openclaw.json`

### Wie kontrolliere ich, wer Nachrichten senden kann?

Konfiguriere über `dmPolicy`:

- `pairing`: Unbekannte Absender benötigen einen Pairing-Code
- `allowlist`: Nur Benutzer auf der Erlaubnisliste
- `open`: Alle Direktnachrichten erlauben
- `disabled`: Direktnachrichten deaktivieren

### Wie richte ich Gruppenchats ein?

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

## Nutzung

### Wie füge ich einen neuen Chat-Kanal hinzu?

```bash
openclaw channels login whatsapp
```

### Kann ich mehrere Kanäle gleichzeitig nutzen?

Ja, ein Gateway kann sich gleichzeitig mit mehreren Kanälen verbinden.

### Wie bringe ich der KI bei, sich an frühere Gespräche zu erinnern?

OpenClaw hat eingebautes Memory, das Session-Kontext automatisch speichert. Du kannst die Session-Aufbewahrungsrichtlinien durch Konfiguration anpassen.

### Kann ich mehrere Agenten ausführen?

Ja, konfiguriere mehrere Agenten mit Routing-Regeln:

```json5
{
  agents: {
    list: [
      { id: "home", default: true },
      { id: "work" }
    ]
  }
}
```

## Sicherheit

### Sind meine Daten sicher?

Ja. OpenClaw ist selbst gehostet – alle Daten bleiben auf deinem Gerät und werden nicht an Drittanbieter-Server gesendet (außer an KI-Anbieter-APIs).

### Wie schütze ich mein Gateway?

1. Setze `allowFrom`, um Zugriffsquellen einzuschränken
2. Verwende starke Passwörter und API-Schlüssel
3. Halte OpenClaw aktuell
4. Nutze Firewall, um Port-Zugriff einzuschränken

## Fehlerbehebung

### Gateway startet nicht

1. Prüfe, ob Port belegt ist: `openclaw gateway status`
2. Fehlerlogs prüfen: `openclaw logs`
3. Diagnose ausführen: `openclaw doctor`

### KI antwortet nicht

1. Prüfe, ob API-Schlüssel korrekt konfiguriert ist
2. Bestätige, dass dein Modell-Anbieter-Konto ausreichendes Guthaben hat
3. Suche nach Fehlermeldungen in den Logs

### Nachrichtensenden schlägt fehl

1. Prüfe, ob Kanal angemeldet ist
2. Bestätige, dass Zielbenutzer oder -gruppe existiert
3. Prüfe detaillierte Fehlerlogs
