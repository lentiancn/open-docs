# Claude Code Guía de Instalación

Esta guía proporciona instrucciones detalladas para instalar, configurar y actualizar Claude Code en varias plataformas.

## Requisitos del Sistema

Claude Code soporta los siguientes sistemas operativos y configuraciones:

| Componente | Requisitos |
|------------|------------|
| Sistemas Operativos | macOS 13.0+, Windows 10 1809+ o Windows Server 2019+, Ubuntu 20.04+, Debian 10+, Alpine Linux 3.19+ |
| Memoria | 4 GB o más |
| Red | Se requiere conexión a Internet |
| Shell | Bash, Zsh, PowerShell o CMD |
| Región | Países y regiones soportados por Anthropic |

### Dependencias Adicionales

- **ripgrep**: Generalmente incluido con Claude Code; si la búsqueda falla, consulta la sección de solución de problemas.

## Métodos de Instalación

### Método Uno: Instalación Nativa (Recomendada)

**macOS, Linux, WSL:**

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell:**

```powershell
irm https://claude.ai/install.ps1 | iex
```

**Windows CMD:**

```batch
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
```

**Nota: Windows requiere [Git for Windows](https://git-scm.com/downloads/win) instalado primero.**

Las instalaciones nativas se actualizan automáticamente en segundo plano.

### Método Dos: Homebrew (macOS/Linux)

```bash
brew install --cask claude-code
```

Las instalaciones de Homebrew no se actualizan automáticamente. Ejecuta `brew upgrade claude-code` periódicamente para obtener las últimas funciones y correcciones de seguridad.

### Método Tres: WinGet (Windows)

```powershell
winget install Anthropic.ClaudeCode
```

Las instalaciones de WinGet no se actualizan automáticamente. Ejecuta `winget upgrade Anthropic.ClaudeCode` periódicamente para actualizaciones manuales.

## Verificar Instalación

Después de instalar, confirma que Claude Code funciona:

```bash
claude --version
```

Para una verificación más detallada de la instalación y configuración, ejecuta:

```bash
claude doctor
```

## Autenticación de Cuenta

Claude Code requiere cuentas Pro, Max, Teams, Enterprise o Console. El plan gratuito de Claude.ai no incluye acceso a Claude Code.

Después de instalar, ejecuta el comando `claude` y sigue las instrucciones del navegador para iniciar sesión.

## Actualizar Claude Code

### Actualizaciones Automáticas

Las instalaciones nativas verifican actualizaciones al iniciar y periódicamente mientras se ejecutan. Las actualizaciones se descargan e instalan en segundo plano y se aplican la próxima vez que inicies Claude Code.

Puedes controlar el momento de las actualizaciones configurando el canal de lanzamiento:
- `latest` (predeterminado): Recibir nuevas funciones inmediatamente
- `stable`: Usar versiones de aproximadamente una semana de antigüedad, omitiendo lanzamientos con problemas importantes

Configura en tu archivo de configuración:

```json
{
  "autoUpdatesChannel": "stable"
}
```

### Desactivar Actualizaciones Automáticas

Establece la variable de entorno en tu archivo de configuración:

```json
{
  "env": {
    "DISABLE_AUTOUPDATER": "1"
  }
}
```

### Actualizaciones Manuales

Aplica una actualización inmediatamente sin esperar la verificación en segundo plano:

```bash
claude update
```

Las instalaciones de Homebrew y WinGet requieren actualizaciones manuales:
- Homebrew: `brew upgrade claude-code`
- WinGet: `winget upgrade Anthropic.ClaudeCode`

## Opciones de Instalación Avanzada

### Instalar Versión Específica

Instalar última versión:
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

Instalar versión estable:
```bash
curl -fsSL https://claude.ai/install.sh | bash -s stable
```

Instalar número de versión específico (ej. 1.0.58):
```bash
curl -fsSL https://claude.ai/install.sh | bash -s 1.0.58
```

### Migrar de npm a Instalación Nativa

Si instalaste previamente Claude Code con npm, migra a instalación nativa:

```bash
# Instalar binario nativo
curl -fsSL https://claude.ai/install.sh | bash

# Remover instalación antigua de npm
npm uninstall -g @anthropic-ai/claude-code
```

### Método de Instalación npm Obsoleto

El método de instalación npm está obsoleto. Si realmente necesitas instalación npm, debes tener Node.js 18+ instalado primero:

```bash
npm install -g @anthropic-ai/claude-code
```

**Nota: NO uses `sudo npm install -g` ya que puede generar problemas de permisos y riesgos de seguridad.**

## Desinstalar Claude Code

### Desinstalar Instalación Nativa

**macOS, Linux, WSL:**

```bash
rm -f ~/.local/bin/claude
rm -rf ~/.local/share/claude
```

**Windows PowerShell:**

```powershell
Remove-Item -Path "$env:USERPROFILE\.local\bin\claude.exe" -Force
Remove-Item -Path "$env:USERPROFILE\.local\share\claude" -Recurse -Force
```

### Desinstalar Homebrew

```bash
brew uninstall --cask claude-code
```

### Desinstalar WinGet

```powershell
winget uninstall Anthropic.ClaudeCode
```

### Desinstalar npm

```bash
npm uninstall -g @anthropic-ai/claude-code
```

### Remover Archivos de Configuración

**Advertencia: Remover archivos de configuración eliminará todas las configuraciones, herramientas permitidas, configuraciones de servidores MCP e historial de sesiones.**

**macOS, Linux, WSL:**

```bash
# Remover configuración y estado de usuario
rm -rf ~/.claude
rm ~/.claude.json

# Remover configuración específica del proyecto (ejecutar desde el directorio del proyecto)
rm -rf .claude
rm -f .mcp.json
```

**Windows PowerShell:**

```powershell
# Remover configuración y estado de usuario
Remove-Item -Path "$env:USERPROFILE\.claude" -Recurse -Force
Remove-Item -Path "$env:USERPROFILE\.claude.json" -Force

# Remover configuración específica del proyecto
Remove-Item -Path ".claude" -Recurse -Force
Remove-Item -Path ".mcp.json" -Force
```

## Configuración Especial de Windows

Claude Code en Windows requiere Git for Windows o WSL. Puedes lanzar `claude` desde PowerShell, CMD o Git Bash.

Si Claude Code no puede encontrar tu instalación de Git Bash, configura la ruta en tu archivo de configuración:

```json
{
  "env": {
    "CLAUDE_CODE_GIT_BASH_PATH": "C:\\Program Files\\Git\\bin\\bash.exe"
  }
}
```

## Configuración de Alpine Linux

En Alpine y otras distribuciones basadas en musl/uClibc, el instalador nativo requiere `libgcc`, `libstdc++` y `ripgrep`. Instala estas dependencias usando tu administrador de paquetes, luego establece `USE_BUILTIN_RIPGREP=0`.

Instala los paquetes requeridos en Alpine:

```bash
apk add libgcc libstdc++ ripgrep
```

Luego configura en tu archivo de configuración:

```json
{
  "env": {
    "USE_BUILTIN_RIPGREP": "0"
  }
}
```

Puedes verificar la integridad de los binarios de Claude Code usando sumas de verificación SHA256. Las sumas se publican en los manifiestos de lanzamiento en:
`https://storage.googleapis.com/claude-code-dist-86c565f3-f756-42ad-8dfa-d59b1c096819/claude-code-releases/{VERSION}/manifest.json`

Los binarios firmados se distribuyen para:
- **macOS**: Firmado por "Anthropic PBC" y notarizado por Apple
- **Windows**: Firmado por "Anthropic, PBC"
