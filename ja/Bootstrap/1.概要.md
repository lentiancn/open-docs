# 概要

Bootstrapは、世界で最も使われているHTML、CSS、JavaScriptのオープンソース前端開発フレームワークです。Twitter社のデザイナーMark OttoとJacob Thorntonが中心になって開発しました。Bootstrapには豊富なコンポーネントと強力なJavaScriptプラグインが含まれており、開発者が素早くレスポンシブなWebアプリケーションを構築することを助けます。

Bootstrapの核となる考え方は「モバイルファースト」です。設計段階から様々なデバイスへの対応を考慮しています。Bootstrapを使えば、デスクトップ、タブレット、スマートフォンなど異なるデバイスで精美に表示されるウェブサイトを簡単に作れます。

## 開発の歩み

Bootstrapは当初「Twitter Blueprint」という名前で、2011年にオープンソースとして公開されました。2012年にはBootstrap 2がリリースされ、レスポンシブレイアウトが導入されました。2013年のBootstrap 3では、全面的に書き直されてフラットデザインになりました。2015年のBootstrap 4では、SassをCSSプリプロセッサとして採用しました。2021年のBootstrap 5では、jQueryの依存を排除し、ネイティブJavaScriptに完全移行しました。

## 主な特徴

### レスポンシブレイアウト

Bootstrapのグリッドシステムは、最も強力な機能の一つです。12列のグリッドレイアウトを使うことで、開発者は異なる画面サイズに対応するページレイアウトを簡単に作れます。Bootstrap 5では、xs、sm、md、lg、xl、xxlの6つのブレークポイントが用意されており、それぞれ異なる幅のデバイスに対応しています。

### 豊富なコンポーネント

Bootstrapにはすぐに使えるUIコンポーネントが豊富に用意されています。ナビゲーションバー、ボタン、フォーム、モーダル、スライダー、ドロップダウンリストなどが含まれています。これらのコンポーネントはすべて丁寧にデザインされており、見た目と使い心地の両方で優れた体験を提供します。

### 強力なJavaScriptプラグイン

Bootstrapには20以上のJavaScriptプラグインが含まれています。モーダル、スライダー、ツールチップ、ポップオーバー、アコーディオンなどがあります。これらのプラグインはすべて任意で、開発者は必要なものだけを導入してプロジェクトの容量を減らせます。

### Sassベース

Bootstrap 5はSassをCSSプリプロセッサとして使っており、多くのSass変数とMixinsが用意されています。開発者はBootstrapのテーマカラーやフォントサイズを簡単にカスタマイズできます。

### テーマのカスタマイズ

Bootstrapのテーマシステムを使えば、開発者は独自のテーマスタイルを簡単に作れます。公式サイトではBootstrap Themesマーケットプレイスを用意しており、多数の精美なテーマから選べます。

### アクセシビリティ対応

Bootstrapのコンポーネントにはアクセシビリティ対応が組み込まれています。キーボードナビゲーション、スクリーンリーダー互換性など含まれており、開発者はより包括的なウェブサイトを作れます。

## Bootstrap 5の新機能

Bootstrap 4からBootstrap 5にかけて、大きな変更점이多数あります：

- jQuery依存を排除し、ネイティブJavaScriptのみを使用
- グリッドシステムが改良され、XXLブレークポイントを追加
- 新しいフォームコントロールがフォーム検証を置き換え
- カスタムプロパティのサポートが向上
- アイコンライブラリをFont AwesomeからBootstrap Iconsに変更
- RTL（右から左）サポートに対応

## 適用シーン

- クイックプロトタイプ開発
- 企業内管理システム
- ECウェブサイト
- ブログやコンテンツサイト
- 管理画面インターフェース
- レスポンシブウェブサイト開発

世界で最も使われているCSSフレームワーク之一的Bootstrapは、大きなコミュニティと豊富な学習リソースを持っています。前端開発者にとって必須のスキルの一つです。
