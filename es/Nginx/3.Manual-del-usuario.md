# Manual de usuario Nginx

## Operaciones básicas
```bash
nginx              # Iniciar
nginx -s reload   # Recargar configuración
nginx -t          # Probar configuración
```

## Proxy inverso
```nginx
location / {
    proxy_pass http://backend;
}
```

## Balanceo de carga
```nginx
upstream backend {
    server backend1;
    server backend2;
}
```
