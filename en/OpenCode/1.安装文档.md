# OpenCode Installation Guide

OpenCode is a modern code editor and development environment.

## System Requirements

### Windows
- Windows 10/11 (64-bit)
- 4GB RAM minimum (8GB recommended)
- 500MB disk space

### macOS
- macOS 10.15+
- 4GB RAM minimum

### Linux
- Ubuntu 18.04+, Debian 10+, Fedora 30+
- 4GB RAM minimum

## Install on Windows

### Option 1: Installer
1. Download from official website
2. Run installer
3. Follow setup wizard

### Option 2: Winget
```powershell
winget install OpenCode
```

### Option 3: Chocolatey
```powershell
choco install opencode
```

## Install on macOS

### Option 1: Homebrew
```bash
brew install opencode
```

### Option 2: DMG
1. Download DMG from official website
2. Drag to Applications folder

## Install on Linux

### Debian/Ubuntu
```bash
sudo apt update
sudo apt install opencode
```

### Or use Snap
```bash
sudo snap install opencode
```

## Features

- Syntax highlighting for 50+ languages
- Integrated terminal
- Git integration
- Debugging support
- Extensions marketplace
- Multiple tabs
- File explorer
- Search and replace
