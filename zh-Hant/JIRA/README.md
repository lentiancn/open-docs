# JIRA

JIRA 是 Atlassian 開發的問題追蹤和專案管理工具。

## 簡介

JIRA 是業界領先的專案和問題管理工具，廣泛用於敏捷開發和 IT 服務管理。

## 主要特性

- **問題追蹤**：完整的缺陷和任務管理
- **敏捷看板**：視覺化的任務管理
- **Scrum 支援**：衝刺規劃和追蹤
- **工作流程自動化**：自動化業務流程
- **報告分析**：豐富的報表功能
- **整合**：與多種工具整合

## 快速開始

### 安裝

```bash
# Docker
docker run -v jira-data:/var/atlassian -p 8080:8080 atlassian/jira-software
```

### 存取

開啟瀏覽器造訪 http://localhost:8080
