## Quick Start

### 1. Install Angular CLI

```bash
npm install -g @angular/cli
```

### 2. Create New Project

```bash
ng new my-app
cd my-app
```

### 3. Start Development Server

```bash
ng serve
```

Open browser and visit http://localhost:4200

## Learning Path

### Phase 1: Fundamentals

1. Install development environment
2. Create first Angular application
3. Understand component basic structure
4. Master template syntax (interpolation, property binding, event binding)
5. Understand data binding

### Phase 2: Intermediate

1. Services and dependency injection
2. Routing and navigation
3. Form handling (template-driven forms + reactive forms)
4. HTTP communication
5. Component communication (Input, Output, ViewChild)

### Phase 3: Advanced

1. State management (Signal, NgRx)
2. Performance optimization
3. Testing strategy
4. Build and deployment
5. Animations
