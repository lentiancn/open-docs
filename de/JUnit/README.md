# JUnit 5 Dokumentation

Willkommen zur JUnit 5 Dokumentation. Dieser Ordner enthält umfassende Dokumentation für JUnit 5, das programmierfreundliche Testframework für Java und die JVM.

## Was ist JUnit 5?

JUnit 5 ist die nächste Generation von JUnit, einem der beliebtesten Unit-Testing-Frameworks für Java. Es bietet eine moderne Architektur mit drei Hauptmodulen: JUnit Platform, JUnit Jupiter und JUnit Vintage.

## Schnellstart

1. **Maven**: Fügen Sie junit-jupiter Abhängigkeit hinzu
2. **Gradle**: useJUnitPlatform() konfigurieren
3. **Schreiben**: @Test Annotation verwenden
4. **Ausführen**: mvn test oder ./gradlew test

## Hauptmerkmale

- **Modulare Architektur**: Platform, Jupiter und Vintage
- **Moderne Tests**: Verschachtelt, parametrisiert, dynamisch
- **Erweiterungsmodell**: Flexible Extension API
- **Verbesserte Assertions**: Gruppierte Assertions, Lambdas
- **Java 8+**: Nutzt moderne Java-Funktionen

## Anforderungen

- Java 8 oder höher
- Maven 3.6+ oder Gradle 5.6+
