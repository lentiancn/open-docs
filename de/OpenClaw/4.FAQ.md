# OpenClaw FAQ

> Häufig gestellte Fragen zu OpenClaw

---

## Allgemein

### Was ist OpenClaw?

OpenClaw ist ein KI-Assistent-Framework, das fortschrittliche Agententechnologie nutzt, um Aufgaben zu automatisieren. Es kombiniert verschiedene Tools für Dateiverwaltung, Browser-Steuerung und mehr.

### Für wen ist OpenClaw geeignet?

OpenClaw ist für Entwickler, Power-User und alle geeignet, die ihre Produktivität durch KI-gestützte Automatisierung steigern möchten.

## Installation

### Welche Systemanforderungen gibt es?

| Anforderung | Minimum |
|-------------|---------|
| Betriebssystem | Windows 10+, macOS, Linux |
| Node.js | Version 18+ |
| RAM | 4 GB |
| Speicher | 500 MB |

### Installation schlägt fehl?

**Lösung**:
1. Node.js Version prüfen: `node --version`
2. npm-Cache leeren: `npm cache clean --force`
3. Mit Administratorrechten erneut versuchen
4. Fehlermeldung dokumentieren

### Gateway startet nicht?

**Lösung**:
1. Port überprüfen: `netstat -ano | findstr 3000`
2. Anderen Port verwenden: `openclaw gateway start --port 3001`
3. Logs prüfen: `openclaw gateway logs`

## Nutzung

### Wie starte ich eine Konversation?

```bash
openclaw chat
```

### Wie führe ich einen einzelnen Befehl aus?

```bash
openclaw exec "mein befehl"
```

### Kann ich OpenClaw headless nutzen?

Ja, OpenClaw kann im Headless-Modus arbeiten. Starten Sie den Gateway-Dienst und nutzen Sie die API oder CLI.

## Dateiverwaltung

### Wie lese ich eine Datei?

Verwenden Sie das read-Tool:
```
read /pfad/zur/datei.txt
```

### Wie erstelle ich eine neue Datei?

Verwenden Sie das write-Tool:
```
write --content "Inhalt" --pfad /neue/datei.txt
```

### Wie bearbeite ich eine Datei?

Verwenden Sie das edit-Tool für präzise Änderungen:
```
edit --pfad /datei.txt --oldText "alt" --newText "neu"
```

## Browser-Steuerung

### Welche Browser werden unterstützt?

- Chrome (über Erweiterung)
- Isolierter OpenClaw-Browser
- Andere über Browser-Automatisierung

### Wie steuere ich den Browser?

```bash
browser start --profile openclaw
browser navigate --url https://beispiel.de
browser screenshot --pfad /output.png
```

### Browser startet nicht?

**Lösung**:
1. Chrome-Erweiterung installieren
2. Firewall-Einstellungen prüfen
3. Browser-Cache leeren

## Subagenten

### Was sind Subagenten?

Subagenten sind eigenständige Instanzen, die spezifische Aufgaben parallel oder sequenziell ausführen können.

### Wie erstelle ich einen Subagenten?

```bash
subagents create --task "beschreibung"
```

### Wie beende ich einen Subagenten?

```bash
subagents kill --id agent-id
```

## Sicherheit

### Ist OpenClaw sicher?

OpenClaw implementiert mehrere Sicherheitsebenen:
- Dateiberechtigungen
- Bestätigung für externe Aktionen
- Sandbox-Optionen

### Wie schütze ich sensible Daten?

1. Verwenden Sie verschlüsselte Konfiguration
2. Geben Sie keine Passwörter in Konversationen weiter
3. Nutzen Sie Umgebungsvariablen für Geheimnisse

## Fehlerbehebung

### "Befehl nicht gefunden"?

Stellen Sie sicher, dass:
1. OpenClaw korrekt installiert ist
2. PATH-Variable korrekt gesetzt ist
3. Gateway läuft

### Dateioperationen fehlschlagen?

1. Pfad überprüfen
2. Berechtigungen prüfen
3. Dateiname auf Sonderzeichen prüfen

### Langsame Reaktionszeiten?

1. Gateway-Status prüfen
2. Netzwerkverbindung prüfen
3. Logs analysieren
