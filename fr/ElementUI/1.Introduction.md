# Introduction à Element UI

Element UI est une bibliothèque de composants d'interface utilisateur basée sur Vue 2.0 qui fournit aux développeurs des solutions complètes de composants pour les applications de bureau.

## Qu'est-ce que Element UI

Element UI est une bibliothèque de composants Vue 2.0 développée et maintenue par l'équipe Ele.me.

## Caractéristiques principales

### Système de composants complet

Element UI propose plus de 70 composants de haute qualité:

- Composants de base: Boutons, Icônes, Mise en page, Conteneurs
- Composants de formulaire: Entrée, Sélecteur, Sélecteur de date, Interrupteur
- Affichage de données: Tableau, Arbre, Pagination, Étiquettes
- Composants de retour: Dialogue, Message, Chargement
- Composants de navigation: Menu, Onglets, Fil d'Ariane

### Support Vue 2.0 complet

Element UI est entièrement construit sur Vue 2.0.

### Importation à la demande

En utilisant babel-plugin-component.

### Support de l'internationalisation

Support multilingue intégré.

### Personnalisation des thèmes

Support de la personnalisation globale des thèmes.

## Principes de conception

- Cohérence: Avec la vie réelle et l'interface
- Retour d'information: Contrôle et retour de page
- Efficacité: Processus simplifiés
- Contrôlabilité: Décisions utilisateur

## Scénarios d'application

- Systèmes de gestion d'entreprise
- Systèmes de backend e-commerce
- Tout projet Vue 2

Pour plus d'informations, consultez le site officiel Element UI.
