# Element UI インストールガイド

> Element UIはVue 2ベースのデスクトップUIコンポーネントライブラリ
> 
> 更新日：2026年2月
> 
> 現行バージョン：2.x（安定版）
> 
> 注意：Vue 3プロジェクトの場合は、Element Plusをご使用ください

---

## 1. プロジェクト紹介

Element UIはVue 2.0ベースのデスクトップUIコンポーネントライブラリで、Ele.meチームが開発・メンテナンスしています。フォーム、テーブル、ダイアログ、ナビゲーションメニュー、ボタン、入力ボックスなど、よく使うPC向けコンポーネントを豊富に提供しています。

Element UIはデザインはシンプルで美しく、ドキュメントも詳しいため、中国で最も人気のVue 2 UIコンポーネントライブラリの1つです。官方はメンテナンスを停止していますが（Element Plusへの移行を推奨）、まだ多くのプロジェクトで使用されており、学習・デ 参考になります。

### 1.1 バージョン情報

| バージョン | Vueバージョン | 状態 | 説明 |
|------|----------|------|------|
| Element UI 2.x | Vue 2 | 安定 | 現在のメインドキュメント |
| Element UI 1.x | Vue 2 | メンテナンス停止 | 旧バージョン |

### 1.2 Element Plusとの違

| 機能 | Element UI | Element Plus |
|------|------------|--------------|
| Vueバージョン | Vue 2 | Vue 3 |
| TypeScript | 一部サポート | 完全サポート |
| Composition API | 非サポート | サポート |
| メンテナンス状態 | 停止 | アクティブ |
| リリース年 | 2016-2021 | 2022-現在 |

Element UIは次のブラウザバージョンをサポート：

- Safari >= 10

## 3. インストール手順

### 3.1 npmを使用（推奨）

```bash
# Vue 2プロジェクトディレクトリに移動
cd your-vue-project

# Element UIをインストール
npm install element-ui --save
```

### 3.2 yarnを使用

```bash
yarn add element-ui
```

### 3.3 特定バージョンをインストール

```bash
# 最新2.xバージョンをインストール
npm install element-ui@latest

# 特定バージョンをインストール
npm install element-ui@2.15.13
```

## 4. Vue 2プロジェクトでの導入

### 4.1 完全導入（小規模プロジェクト向け）

エントリファイル（通常main.js）でElement UIとスタイルを導入：

```javascript
// main.js
import Vue from 'vue'
import App from './App.vue'

// Element UIを導入
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)

new Vue({
  render: h => h(App)
}).$mount('#app')
```

### 4.2 必要時導入（推奨、本番環境向け）

バンドルサイズを小さくするため、必要なコンポーネントのみを導入。

#### 4.2.1 babel-plugin-componentをインストール

```bash
npm install babel-plugin-component -D
```

#### 4.2.2 babelを設定

.babelrcまたはbabel.config.jsに設定を追加：

```json
{
  "presets": [
    ["env", {
      "modules": false,
      "targets": {
        "browsers": ["> 1%", "last 2 versions", "not ie <= 8"]
      }
    }]
  ],
  "plugins": [
    ["component", {
      "libraryName": "element-ui",
      "styleLibraryName": "theme-chalk"
    }]
  ]
}
```

#### 4.2.3 必要時コンポーネントを導入

```javascript
// main.js
import Vue from 'vue'
import { Button, Form, FormItem, Input, Table, TableColumn } from 'element-ui'
import App from './App.vue'

// 必要なコンポーネントを登録
Vue.use(Button)
Vue.use(Form)
Vue.use(FormItem)
Vue.use(Input)
Vue.use(Table)
Vue.use(TableColumn)

new Vue({
  render: h => h(App)
}).$mount('#app')
```

### 4.3 CDNで導入

ビルドツールを使用したくない場合は、CDNでElement UIを導入できます：

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Element UI CDN 例</title>
  <!-- Vue 2を導入 -->
  <script src="https://cdn.jsdelivr.net/npm/vue@2.6.14/dist/vue.js"></script>
  <!-- Element UIを導入 -->
  <link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
  <script src="https://unpkg.com/element-ui"></script>
</head>
<body>
  <div id="app">
    <el-button type="primary" @click="visible = true">ダイアログを開く</el-button>
    <el-dialog title="ヒント" :visible.sync="visible" width="30%">
      <span>これはメッセージです</span>
      <span slot="footer" class="dialog-footer">
        <el-button @click="visible = false">キャンセル</el-button>
        <el-button type="primary" @click="visible = false">OK</el-button>
      </span>
    </el-dialog>
  </div>

  <script>
    new Vue({
      el: '#app',
      data: {
        visible: false
      }
    })
  </script>
</body>
</html>
```

## 5. クイックスタート例

### 5.1 基本ボタン

```html

    <el-button type="info">情報</el-button>
    <el-button type="warning">警告</el-button>
    <el-button type="danger">危険</el-button>
  </div>
</template>
```

### 5.2 基本フォーム

```html
<template>
  <el-form :model="form" :rules="rules" ref="loginForm" label-width="80px">
    <el-form-item label="ユーザー名" prop="username">
      <el-input v-model="form.username" placeholder="ユーザー名を入力"></el-input>
    </el-form-item>
    <el-form-item label="パスワード" prop="password">
      <el-input v-model="form.password" type="password" placeholder="パスワードを入力"></el-input>
    </el-form-item>
    <el-form-item>
      <el-button type="primary" @click="submitForm">ログイン</el-button>
      <el-button @click="resetForm">リセット</el-button>
    </el-form-item>
  </el-form>
</template>

<script>
export default {
  data() {
    return {
      form: {
        username: '',
        password: ''
      },
      rules: {
        username: [
          { required: true, message: 'ユーザー名を入力してください', trigger: 'blur' }
        ],
        password: [
          { required: true, message: 'パスワードを入力してください', trigger: 'blur' }
        ]
      }
    }
  },
  methods: {
    submitForm() {
      this.$refs.loginForm.validate(valid => {
        if (valid) {
          console.log('ログインフォームを送信：', this.form)
        } else {
          this.$message.error('フォーム検証に失敗しました')
        }
      })
    },
    resetForm() {
      this.$refs.loginForm.resetFields()
    }
  }
}
</script>
```

### 5.3 基本テーブル

```html
<template>
  <el-table :data="tableData" stripe border>
    <el-table-column prop="date" label="日付" width="180"></el-table-column>
    <el-table-column prop="name" label="名前" width="180"></el-table-column>
    <el-table-column prop="address" label="住所"></el-table-column>
    <el-table-column label="操作" width="180">
      <template slot-scope="scope">
        <el-button size="mini" @click="handleEdit(scope.$index, scope.row)">編集</el-button>
        <el-button size="mini" type="danger" @click="handleDelete(scope.$index, scope.row)">削除</el-button>
      </template>
    </el-table-column>
  </el-table>
</template>

<script>
export default {
  data() {
    return {
      tableData: [{
        date: '2024-01-01',
        name: '山田太郎',
        address: '東京都'
      }, {
        date: '2024-01-02',
        name: '鈴木花子',
        address: '大阪府'
      }]
    }
  },
  methods: {
    handleEdit(index, row) {
      console.log('編集：', index, row)
    },
    handleDelete(index, row) {
      console.log('削除：', index, row)
    }
  }
}
</script>
```

### 5.4 ダイアログ

```html
<template>
  <el-button type="text" @click="dialogVisible = true">ダイアログを開く</el-button>

  <el-dialog
    title="編集"
    :visible.sync="dialogVisible"
    width="30%"
    :before-close="handleClose">
    <span>これはダイアログの内容です</span>
    <span slot="footer" class="dialog-footer">
      <el-button @click="dialogVisible = false">キャンセル</el-button>
      <el-button type="primary" @click="dialogVisible = false">OK</el-button>
    </span>
  </el-dialog>
</template>

<script>
export default {
  data() {
    return {
      dialogVisible: false
    }
  },
  methods: {
    handleClose(done) {
      this.$confirm('閉じますか？')
        .then(_ => {
          done()
        })
        .catch(_ => {})
    }
  }
}
</script>
```

## 6. 国際化設定

Element UIはデフォルトで英語を使用しています。日本語を使用するには設定が必要です。

### 6.1 グローバル日本語設定

```javascript
// main.js
import Vue from 'vue'
import ElementUI from 'element-ui'
import ja from 'element-ui/lib/locale/lang/ja'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI, {
  locale: ja
})
```

## 7. テーマカスタマイズ

### 7.1 オンラインプレゼンツール

Element UIプレゼンツールページにアクセス：https://elementui.github.io/theme-chalk/

テーマ色、ボーダー色、ボーダー半径などを選択して、カスタマイズしたスタイルファイルをダウンロードできます。

### 7.2 手動で変数を上書き

プロジェクトでカスタムスタイルファイルを作成：

```scss
// styles/element-variables.scss
$--color-primary: #409EFF;
$--color-success: #67C23A;
$--color-warning: #E6A23C;
$--color-danger: #F56C6C;
$--color-info: #909399;

@import 'element-ui/packages/theme-chalk/src/common/var.scss';
```

## 8. よくある問題

### 8.1 インストール失敗

**問題**：npm installが失敗します

**解決方法**：

1. npmキャッシュをクリア：`npm cache clean --force`
2. ネットワークを確認し、国内ミラー可以使用

### 8.2 スタイルが適用されない

**問題**：コンポーネントスタイルが異常表示される

**解決方法**：

1. スタイルファイルを導入していることを確認：`import 'element-ui/lib/theme-chalk/index.css'`
2. webpack設定がCSSを正しく処理しているか確認
3. CSS競合がないことを確認
