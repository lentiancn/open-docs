# Bootstrap FAQ

> Common questions

---

## Installation

### CDN not loading?

Use alternative CDN:

```html
<link href="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
```

---

## Styles

### Styles not working?

1. Check Bootstrap CSS is loaded
2. JavaScript required for some components

### Responsive not working?

Add viewport meta:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

---

## Components

### Dropdown not working?

Check:
1. Bootstrap JS is loaded
2. Correct HTML structure

### Modal won't open?

1. Bootstrap JS loaded
2. Correct data-bs-toggle/target attributes

---

## Customization

### Customize theme colors?

**Sass:**

```scss
$primary: #ff0000;
@import "bootstrap";
```

**CSS Variables:**

```css
:root {
  --bs-primary: #ff0000;
}
```
