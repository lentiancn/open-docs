# Claude Code ユーザーマニュアル

基本的な使い方とテクニック。

## 基本的な使い方

### 起動

- VS Code：ショートカット Cmd/Ctrl + Shift + P
- CLI：ターミナルで直接 `claude` を実行

### 会話

チャットウィンドウで直接ニーズを描述：
```
排序関数を作成して
このコードを説明して
このバグの直し方
```

## コード生成

### 新規コード生成

必要な機能を描述：
```
ユーザーログインAPIを作成
ページネーションクエリを実装
ユニットテストを書く
```

### 既存コードの変更

1. コードを選択
2. 変更ニーズを描述
3. Claudeが変更後のコードを生成

## デバッグ支援

### エラーの説明

エラーメッセージを直接貼り付け：
```
このエラーは何ですか？
```

### 修正提案

Claudeに問題を分析させる：
```
この関数有什么問題？
```

## コードレビュー

### コードレビュー依頼

```
このコードをレビューして
セキュリティの問題はある？
パフォーマンス最適化の提案は？
```

## ショートカット

### VS Code

| ショートカット | 機能 |
|--------------|------|
| Cmd/Ctrl + L | チャットを開く |
| Cmd/Ctrl + Enter | 入力を送信 |

## 設定オプション

### claude.json

プロジェクトルートに `claude.json` を作成：

```json
{
  "preamble": "あなたはPythonの専門家です",
  "maxTokens": 4000
}
```

## ベストプラクティス

1. ニーズを明確に描述
2. コンテキスト情報を提供
3. コードを反復改善
4. 人間のレビューと組み合わせ
