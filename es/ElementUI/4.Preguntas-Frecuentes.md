# Element UI Preguntas Frecuentes

## Instalación

### Q: ¿Instalación fallida?

**Soluciones:**
1. Verificar versión de Node.js
2. Limpiar caché npm
```bash
npm cache clean --force
```

### Q: ¿Versión de Vue incompatible?

**Soluciones:**
Element UI 2.x requiere Vue 2.6.0+

## Uso

### Q: ¿Componentes no se muestran?

**Soluciones:**
Verificar importación:
```javascript
import { Button } from 'element-ui'
Vue.use(Button)
```

### Q: ¿Estilos no aplicados?

**Soluciones:**
Importar archivo de estilos:
```javascript
import 'element-ui/lib/theme-chalk/index.css'
```
