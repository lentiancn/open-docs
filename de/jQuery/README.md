# jQuery

## Übersicht

jQuery ist eine schnelle, kleine und funktionsreiche JavaScript-Bibliothek. Sie vereinfacht die Navigation und Manipulation von HTML-Dokumenten, Event-Handling, Animationen und Ajax durch eine benutzerfreundliche API, die in einer Vielzahl von Browsern funktioniert.

## Funktionen

- **Prägnante Syntax**: Kettenfähige API, Code ist prägnant und lesbar
- **Leistungsstarke Selektoren**: Vollständige CSS-Selektor-Unterstützung
- **Browserübergreifende Kompatibilität**: Löst Browser-API-Unterschiede
- **Umfangreiches Plugin-Ökosystem**: Umfangreiche Plugin-Bibliothek
