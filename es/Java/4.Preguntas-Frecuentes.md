# Preguntas Frecuentes de Java

> Preguntas comunes

---

## Instalación

### ¿javac no encontrado?

**Solución**: Configurar JAVA_HOME y PATH.

### ¿Cómo cambiar versión?

```bash
export JAVA_HOME=/ruta/al/jdk
```

---

## Desarrollo

### JDK vs JRE?

- **JRE**: Entorno de ejecución
- **JDK**: Entorno de desarrollo
