# Java 使用手冊

> 基本語法

## Hello World

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

## 變數

```java
int age = 25;
String name = "Tom";
```

## 類別

```java
public class Person {
    private String name;
    
    public Person(String name) {
        this.name = name;
    }
}

Person p = new Person("Tom");
```
