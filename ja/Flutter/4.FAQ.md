# Flutter FAQ

よくある質問。

## インストール

### ダウンロード失敗？

```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
```

### flutter doctor エラー？

PATHを確認。

### Android SDK 見つからない？

```bash
flutter config --android-sdk /path/to/android-sdk
```

## 開発

### ホットリロード動作しない？

ファイルを保存して待つ。

### APKビルド失敗？

```bash
flutter clean
```

### ウィジェット表示されない？

build()メソッドを確認。

## パフォーマンス

### アプリ遅い？

- const使用
- ListView.builder使用
