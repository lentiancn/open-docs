# Yarn パッケージマネージャー

Yarn は、JavaScript プロジェクトの依存関係を管理するために使用される確立されたオープンソースのパッケージマネージャーです。

- [3.ユーザーマニュアル.md](3.ユーザーマニュアル.md) - 基本コマンド、ワークスペース、設定ファイルなどの使用方法
- [4.よくある質問.md](4.よくある質問.md) - インストール、使用、移行、トラブルシューティングに関するよくある質問と回答

## メインバージョン情報

### Yarn v4+ (モダンバージョン)
- **公式サイト**: https://yarnpkg.com/
- **ステータス**: アクティブ開発中
- **特徴**: プラグインベースのアーキテクチャ、ネイティブワークスペースサポート、PnP、オフラインキャッシュ

### Yarn v1 (Classic)
- **公式サイト**: https://classic.yarnpkg.com/
- **ステータス**: メンテナンス中
- **特徴**: 従来のアーキテクチャ、シンプルで分かりやすく、広く互換性がある
