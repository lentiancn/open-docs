# Element UI

Vue.js 2.0 컴포넌트 라이브러리.

## 개요

Element UI - Ele.me 팀이 개발한 라이브러리.

### 기능

- **60+ 컴포넌트**
- **아름다운 디자인**
- **TypeScript 지원**

### 통계

- GitHub 별표: 54,000+

## 빠른 시작

```bash
npm install element-ui --save
```

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
Vue.use(ElementUI)
```
