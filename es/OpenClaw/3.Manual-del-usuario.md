# Manual de Usuario de OpenClaw

## Inicio Rápido

```bash
openclaw gateway --port 18789
openclaw dashboard
```

## Gestión de Canales

```bash
openclaw channels list
openclaw channels add whatsapp
openclaw channels login
```

## Enviar Mensajes

```bash
openclaw message send --target +15555550123 --message "Hola"
```

## Gestión de Sesiones

```bash
openclaw sessions list
```

## Configuración

```bash
openclaw config get
openclaw config set channels.whatsapp.allowFrom "+15555550123"
```

## Comandos

| Comando | Descripción |
|---------|-------------|
| openclaw gateway start | Iniciar Gateway |
| openclaw dashboard | Abrir panel |
| openclaw channels list | Listar canales |
