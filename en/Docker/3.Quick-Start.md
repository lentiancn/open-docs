## What is Docker?

Docker is an open-source platform for containerization. Containers are lightweight and include everything needed to run software: code, runtime, system tools, and libraries.

## Step 1: Install Docker

## Step 3: Your First Container

### Run Nginx Web Server

```bash
docker run -d -p 8080:80 nginx
```

Visit http://localhost:8080 in your browser.

### Run Ubuntu

```bash
docker run -it ubuntu /bin/bash
```

Inside the container:
```bash
echo "Hello from container!"
exit
```

## Step 4: Basic Commands

### Images

```bash
# Pull image
docker pull nginx

# List images
docker images

# Remove image
docker rmi nginx
```

### Containers

```bash
# List running containers
docker ps

# List all containers
docker ps -a

# Stop container
docker stop container_id

# Remove container
docker rm container_id
```

### Logs

```bash
docker logs container_id
```

## Step 5: Create Your Own Image

### Create Project

```bash
mkdir my-app
cd my-app
```

### Create Dockerfile

```dockerfile
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
EXPOSE 3000
CMD ["node", "-e", "console.log('Hello from my app!')"]
```

### Create package.json

```json
{
  "name": "my-app",
  "version": "1.0.0",
  "dependencies": {}
}
```

### Build and Run

```bash
docker build -t my-app .
docker run my-app
```

## Step 6: Use Docker Compose

### Create docker-compose.yml

```yaml
version: '3.8'
services:
  web:
    image: nginx
    ports:
      - "8080:80"
  db:
    image: postgres:15
    environment:
      POSTGRES_PASSWORD: secret
```

### Start Services

```bash
docker-compose up -d
```

### Stop Services

```bash
docker-compose down
```

## Common Operations

### Port Mapping

```bash
docker run -p 8080:80 nginx
# Host port 8080 -> Container port 80
```

### Volume Mounting

```bash
docker run -v /host/path:/container/path nginx
```

### Environment Variables

```bash
docker run -e MY_VAR=value nginx
```
