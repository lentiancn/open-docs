# Руководство по использованию MySQL

Базовые операции MySQL.

---

## Подключение

```bash
mysql -u root -p
```

## Основные команды

```sql
SHOW DATABASES;
CREATE DATABASE mydb;
USE mydb;
```

## CRUD

```sql
INSERT INTO users (name) VALUES ('John');
SELECT * FROM users;
UPDATE users SET name='Jane' WHERE id=1;
DELETE FROM users WHERE id=1;
```
