# Guia de Uso de Mockito

Guia para usar Mockito efectivamente en pruebas unitarias.

---

## Conceptos basicos

Mockito es un framework de mocking para pruebas unitarias en Java.

---

## Crear Mock

```java
List<String> mockList = Mockito.mock(List.class);
```

---

## Stubbing

```java
when(mockList.get(0)).thenReturn("first");
```

---

## Verificacion

```java
verify(mockList).add("element");
```

---

## Anotaciones

```java
@Mock
private UserService userService;
```
