# Bootstrap 5 Guide d'installation

Ce guide détaille l'installation et la configuration de Bootstrap 5.

## Versions

| Version | Statut | Description |
|---------|--------|-------------|
| Bootstrap 5.3.x | ✅ Actuelle | Dernière version |
| Bootstrap 5.2.x | ✅ Supportée | Version stable |
| Bootstrap 4.x | ❌ Obsolète | Non recommandé |

## Configuration requise

### Navigateurs supportés

| Navigateur | Version minimum |
|------------|-----------------|
| Chrome | 90+ |
| Firefox | 90+ |
| Safari | 14+ |
| Edge | 90+ |

## Méthodes d'installation

### Méthode 1: CDN

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
```

### Méthode 2: npm

```bash
npm install bootstrap@5.3.3
```

### Méthode 3: Vite

```bash
npm create vite@latest my-app
npm install bootstrap@5.3.3
```

```javascript
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

### Méthode 4: Webpack

```bash
npm install bootstrap@5.3.3
npm install --save-dev webpack webpack-cli css-loader style-loader
```

### Méthode 5: Vue CLI / Nuxt

```bash
npm install -g @vue/cli
vue create my-app
npm install bootstrap@5.3.3
```

### Méthode 6: React

```bash
npx create-react-app my-app
npm install bootstrap@5.3.3
```

### Méthode 7: Code source

```bash
git clone https://github.com/twbs/bootstrap.git
cd bootstrap
npm install
npm run dist
```

## Configuration spéciale

### RTL

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.rtl.min.css" rel="stylesheet">
```

### Sass

```bash
npm install bootstrap@5.3.3
npm install --save-dev sass
```

## Questions fréquentes

### Popper?

Popper est le moteur de positionnement pour les tooltips et popovers. Inclus dans bootstrap.bundle.min.js.

### jQuery requis?

**Non!** Bootstrap 5 est complètement réécrit en JavaScript natif.

### Icônes?

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css">
```
