# Oracle Base de Datos

Sistema de gestión de bases de datos relacional de clase empresarial

## Descripción

Oracle Database es un sistema de gestión de bases de datos relacional integral, escalable y seguro.

## Inicio Rápido

```bash
sqlplus / as sysdba
```
