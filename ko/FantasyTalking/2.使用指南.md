# FantasyTalking 사용 가이드

이 가이드에서는 FantasyTalking를 사용하여 Talking Head 비디오를 생성하는 방법을 설명합니다.

## 빠른 시작

### 기본 사용법

```bash
python inference.py \
  --source_image images/face.jpg \
  --audio audio/speech.wav \
  --output results/output.mp4
```

## 입력 요구사항

### 이미지 요구사항

- 형식: JPG, PNG
- 해상도: 512x512 이상
- 얼굴: 정면, 명확한 특징
- 배경: 단순한 배경 선호

### 오디오 요구사항

- 형식: WAV, MP3
- 샘플레이트: 16000 Hz 이상
- 길이: 제한 없음
- 언어: 다국어 지원

## 고급 사용법

### 얼굴 개선

```bash
python inference.py \
  --source_image images/face.jpg \
  --audio audio/speech.wav \
  --output results/output.mp4 \
  --face_enhancer
```

### 배치 처리

```bash
python batch_inference.py \
  --image_dir images/ \
  --audio_dir audio/ \
  --output_dir results/
```

### 사용자 정의 FPS

```bash
python inference.py \
  --source_image images/face.jpg \
  --audio audio/speech.wav \
  --output results/output.mp4 \
  --fps 30
```

## 구성

### 구성 파일

`config.yaml` 편집:

```yaml
inference:
  face_enhancer: true
  fps: 25
  output_resolution: 512
  
model:
  checkpoint: "checkpoints/fantasytalking.pth"
```

## 모델 학습

### 데이터셋 준비

```bash
python prepare_dataset.py \
  --video_dir videos/ \
  --output_dir dataset/
```

### 학습

```bash
python train.py \
  --config config/train.yaml \
  --epochs 100
```

## 모범 사례

### 1. 고품질 이미지 사용

- 선명하고 정면 촬영된 사진 사용
- 좋은 조명 조건
- 높은 해상도 선호

### 2. 오디오 품질 일치

- 배경 소음이 없는 선명한 오디오
- 적절한 샘플레이트
- 얼굴 인종과 언어 일치

### 3. 후처리

- 더 나은 품질을 위해 얼굴 개선 사용
- 필요에 따라 FPS 조정

## 문제 해결

### 비디오가 너무 어두운 경우?

소스 이미지의 조명을 확인하세요.

### 오디오가 동기화되지 않은 경우?

오디오 샘플레이트가 올바른지 확인하세요.

### 품질이 나쁜 경우?

얼굴 개선 옵션을 사용해 보세요.
