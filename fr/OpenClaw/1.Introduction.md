# OpenClaw 🦞

OpenClaw est un outil qui s'exécute sur votre propre appareil et connecte les apps de messagerie aux assistants IA. WhatsApp, Telegram, Discord, iMessage... tout直通aux services IA.

## Pour qui?

Si vous voulez votre propre assistant IA sans confier vos données à des tiers. Idéal pour les développeurs.

## Caractéristiques

- **Vos données, votre machine**: tout reste sur votre hardware
- **Multi-plateforme**: un processus gère plusieurs apps de chat
- **Pour programmation**: supporte outils, historique, mémoire, multi-assistant
- **Open source**: licence MIT

## Ce qu'il faut

- Node.js 22+
- Clé API de votre fournisseur IA
- Quelques minutes pour configurer

## Démarrage rapide

```bash
npm install -g openclaw@latest
openclaw onboard --install-daemon
openclaw gateway --port 18789
```

Ouvrez http://127.0.0.1:18789/ dans votre navigateur.
