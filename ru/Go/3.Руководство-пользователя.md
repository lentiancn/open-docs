# Руководство пользователя Go

## Базовый синтаксис

### Объявление переменных

В Go есть несколько способов объявить переменные. Базовое объявление использует ключевое слово var:

```go
// Объявить переменную с типом
var name string = "Go"

// Объявление с выводом типа
var age = 10

// Групповое объявление
var (
    firstName string
    lastName  string
    score     float64
)

// Короткое объявление (внутри функции)
message := "Hello, World!"
```

Переменные в Go должны использоваться после объявления, иначе произойдёт ошибка компиляции. Если нужно объявить временно неиспользуемую переменную, можно использовать нижнее подчёркивание _ для её приёма:

```go
result, _ := someFunction()  // Игнорировать первое возвращаемое значение
```

### Объявление констант

Константы объявляются с помощью ключевого слова const и могут быть символами, строками, булевыми значениями или числами:

```go
const Pi = 3.14159
const (
    StatusOK = 200
    StatusNotFound = 404
)
```

### Основные типы данных

Основные типы данных в Go включают:

- Булевый: bool (true или false)
- Целые числа: int, int8, int16, int32, int64 и соответствующие беззнаковые типы uint и т. д.
- Числа с плавающей точкой: float32, float64
- Комплексные числа: complex64, complex128
- Строки: string

Преобразование типов должно быть явным:

```go
var i int = 42
var f float64 = float64(i)
var s string = strconv.Itoa(i)  // Целое в строку
```

## Функции

### Определение функций

Функции в Go определяются с помощью ключевого слова func:

```go
func greet(name string) string {
    return "Hello, " + name
}

// Несколько возвращаемых значений
func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}

// Именованные возвращаемые значения
func split(sum int) (x, y int) {
    x = sum * 4 / 9
    y = sum - x
    return  // Не нужно указывать возвращаемые значения
}
```

### Вариативные параметры

Функции могут принимать переменное количество параметров:

```go
func sum(nums ...int) int {
    total := 0
    for _, n := range nums {
        total += n
    }
    return total
}

// Использование
result := sum(1, 2, 3, 4, 5)
```

### Замыкания

Go поддерживает замыкания, где функции могут захватывать и изменять переменные из внешней области видимости:

```go
func counter() func() int {
    count := 0
    return func() int {
        count++
        return count
    }
}

// Использование
c := counter()
fmt.Println(c())  // 1
fmt.Println(c())  // 2
```

## Параллельное программирование

### Горутины

Горутины — это легковесные единицы выполнения в Go с чрезвычайно низкими затратами на создание:

```go
// Запустить горутину
go func() {
    fmt.Println("Hello from goroutine!")
}()

// Горутина с параметрами
go func(msg string) {
    fmt.Println(msg)
}("Hello!")
```

### Каналы

Каналы — это механизм связи между горутинами:

```go
// Создать канал
ch := make(chan string)

// Отправить данные
ch <- "message"

// Получить данные
msg := <-ch

// Буферизированный канал
bufferedCh := make(chan int, 10)
```

### Select

Оператор select используется для одновременного ожидания нескольких операций с каналами:

```go
select {
case msg := <-ch1:
    fmt.Println("Received:", msg)
case msg := <-ch2:
    fmt.Println("Received:", msg)
case <-time.After(time.Second):
    fmt.Println("Timeout")
}
```

### Безопасность параллелизма

Используйте пакет sync для управления параллелизмом:

```go
var (
    mu    sync.Mutex
    count int
)

func increment() {
    mu.Lock()
    defer mu.Unlock()
    count++
}
```

## Обработка ошибок

### Возврат ошибок

Go использует возвращаемые значения для обработки ошибок, что является идиоматическим способом в Go:

```go
result, err := doSomething()
if err != nil {
    return err
}
```

### Пользовательские ошибки

Создавайте пользовательские ошибки с помощью пакета errors:

```go
import "errors"

func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}
```

### Обёртывание ошибок

Go 1.13 представил обёртывание ошибок с использованием fmt.Errorf и глагола %w:

```go
if err != nil {
    return fmt.Errorf("failed to do something: %w", err)
}
```

## Интерфейсы

### Определение интерфейсов

Интерфейсы Go реализуются неявно, не требуя явного объявления:

```go
type Writer interface {
    Write([]byte) (int, error)
}

type Reader interface {
    Read([]byte) (int, error)
}

// Общий составной интерфейс
type ReadWriter interface {
    Reader
    Writer
}
```

### Пустой интерфейс

Пустой интерфейс interface{} может хранить значения любого типа:

```go
var i interface{} = 42
var s interface{} = "hello"
var m interface{} = map[string]int{"a": 1}
```

Go 1.18 представил дженерики, позволяющие использовать any вместо interface{}:

```go
var i any = 42
```

## Пакеты и модули

### Создание модуля

Используйте go mod init для создания нового модуля:

```bash
go mod init github.com/username/project
```

### Управление зависимостями

Добавить зависимости:

```bash
go get github.com/pkg/errors
```

Обновить зависимости:

```bash
go get -u all
```

Очистить неиспользуемые зависимости:

```bash
go mod tidy
```

## Тестирование

### Модульные тесты

Go имеет встроенную среду тестирования. Файлы тестов заканчиваются на _test.go:

```go
import "testing"

func TestAdd(t *testing.T) {
    result := Add(2, 3)
    if result != 5 {
        t.Errorf("Add(2, 3) = %d; want 5", result)
    }
}
```

Запуск тестов:

```bash
go test ./...
go test -v ./...
go test -cover ./...
```

### Тесты производительности

Функции тестов начинаются с Benchmark:

```go
func BenchmarkHello(b *testing.B) {
    for i := 0; i < b.N; i++ {
        Hello()
    }
}
```

Запуск тестов производительности:

```bash
go test -bench=.
```

## Стандартная библиотека

### fmt — форматированный ввод/вывод

```go
fmt.Println("Hello")
fmt.Printf("Name: %s, Age: %d\n", name, age)
fmt.Sprintf("Result: %d", result)
```

### strings — операции со строками

```go
strings.Contains(s, "sub")
strings.Split(s, ",")
strings.TrimSpace(s)
strings.ToUpper(s)
```

### net/http — HTTP-сервер

```go
http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
    fmt.Fprint(w, "Hello, HTTP!")
})
http.ListenAndServe(":8080", nil)
```

### json — обработка JSON

```go
// Кодирование
data, _ := json.Marshal(person)

// Декодирование
var p Person
json.Unmarshal(data, &p)
```

## Инструменты

### gofmt — форматирование кода

```bash
gofmt -w main.go  # Форматировать и записать в файл
gofmt -d main.go  # Показать различия форматирования
```

### go vet — статический анализ

```bash
go vet ./...  # Проверить проблемы в коде
```

### go doc — просмотр документации

```go
go doc fmt.Println  # Показать документацию функции
go doc fmt          # Показать документацию пакета
```
