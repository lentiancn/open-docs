# MySQL 安装指南

## Windows 系统安装

### 使用 MySQL Installer 安装

MySQL Installer 是 Windows 平台上最便捷的安装方式，提供基于向导的安装体验。

1. 下载 MySQL Installer
   访问 https://dev.mysql.com/downloads/installer/ 下载安装程序。

2. 运行安装程序
   双击下载的 msi 文件启动安装向导。

3. 选择安装类型
   - **Full**：安装所有 MySQL 产品和功能
   - **Developer Default**：适合开发环境
   - **Server only**：仅安装 MySQL Server
   - **Custom**：自定义安装组件

4. 配置 MySQL Server
   - 选择 MySQL Server 版本
   - 设置端口号（默认 3306）
   - 选择启动类型

5. 设置 root 密码
   为 MySQL 管理员账户设置强密码。

6. 完成安装
   按照向导完成安装过程。

### 使用 ZIP 包手动安装

1. 下载 MySQL ZIP 包
   访问 https://dev.mysql.com/downloads/mysql/ 下载 ZIP 格式的安装包。

2. 解压文件
   将 ZIP 包解压到指定目录，如 C:\mysql。

3. 初始化数据目录
   ```
   mysqld --initialize-insecure
   ```

4. 启动 MySQL 服务
   ```
   mysqld --install MySQL
   net start MySQL
   ```

5. 登录并修改密码
   ```
   mysql -u root -p
   ALTER USER 'root'@'localhost' IDENTIFIED BY '新密码';
   ```

## Linux 系统安装

### Ubuntu/Debian 系统

1. 更新软件源
   ```bash
   sudo apt update
   ```

2. 安装 MySQL Server
   ```bash
   sudo apt install mysql-server
   ```

3. 启动 MySQL 服务
   ```bash
   sudo systemctl start mysql
   sudo systemctl enable mysql
   ```

4. 安全配置
   ```bash
   sudo mysql_secure_installation
   ```

### CentOS/RHEL 系统

1. 添加 MySQL Yum 仓库
   ```bash
   sudo yum install https://dev.mysql.com/get/mysql80-community-release-el8-4.noarch.rpm
   ```

2. 安装 MySQL Server
   ```bash
   sudo yum install mysql-community-server
   ```

3. 启动服务
   ```bash
   sudo systemctl start mysqld
   sudo systemctl enable mysqld
   ```

4. 获取临时密码
   ```bash
   sudo grep 'temporary password' /var/log/mysqld.log
   ```

## macOS 系统安装

### 使用 Homebrew 安装

1. 安装 Homebrew（如果未安装）
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```

2. 安装 MySQL
   ```bash
   brew install mysql
   ```

3. 启动 MySQL 服务
   ```bash
   brew services start mysql
   ```

4. 安全配置
   ```bash
   mysql_secure_installation
   ```

### 使用 DMG 包安装

1. 下载 DMG 安装包
   访问 https://dev.mysql.com/downloads/mysql/ 下载 macOS DMG 包。

2. 运行安装程序
   双击 PKG 文件，按照向导完成安装。

3. 启动 MySQL
   通过系统偏好设置启动 MySQL 服务。

## Docker 安装 MySQL

### 拉取 MySQL 镜像
```bash
docker pull mysql:latest
```

### 运行 MySQL 容器
```bash
docker run -d \
  --name mysql-container \
  -e MYSQL_ROOT_PASSWORD=your_password \
  -e MYSQL_DATABASE=testdb \
  -p 3306:3306 \
  mysql:latest
```

### 常用 Docker 参数说明
- `-e MYSQL_ROOT_PASSWORD`：设置 root 密码
- `-e MYSQL_DATABASE`：创建初始数据库
- `-p 3306:3306`：端口映射
- `-v /data/mysql:/var/lib/mysql`：数据卷持久化

## 安装后验证

### 检查 MySQL 版本
```bash
mysql --version
```

### 检查服务状态
```bash
systemctl status mysql   # Linux
brew services list      # macOS
```

### 登录 MySQL
```bash
mysql -u root -p
```

## 常见安装问题

### 端口冲突
如果 3306 端口被占用，可在配置文件中修改端口，或停止占用端口的服务。

### 权限问题
Linux 系统下注意使用 sudo 提升权限，确保数据目录有正确的读写权限。

### 防火墙设置
确保防火墙允许 3306 端口的访问。
