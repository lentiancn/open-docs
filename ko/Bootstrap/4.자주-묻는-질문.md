# 자주 묻는 질문

이 문서는 Bootstrap 작업 시 발생하는 일반적인 기술적인 질문과 답변을 모았습니다.

## 일반적인 질문

### Bootstrap이란 무엇인가요?

Bootstrap은 Twitter에서 관리하는 오픈소스 프론트엔드 프레임워크입니다. 개발자가 빠르게 반응형 웹 애플리케이션을 구축할 수 있도록 돕는 다양한 CSS 스타일과 JavaScript 컴포넌트를 제공합니다.

### Bootstrap 5와 Bootstrap 4의 차이점은 무엇인가요?

주요 차이점:

- Bootstrap 5는 jQuery를 제거하고 순수 JavaScript 사용
- XXL 중단점 추가
- 개선된 폼 컨트롤
- Font Awesome을 Bootstrap Icons로 교체
- 더 이상 Internet Explorer 지원 안 함

### 어떤 브라우저 지원이 필요한가요?

최신 브라우저만 있으면 됩니다. Bootstrap 5는 ES6를 지원하는 브라우저가 필요합니다.

## 설치 문제

### CDN이 로드되지 않아요?

1. 인터넷 연결 확인
2. 다른 CDN 시도 (bootcdn 또는 staticfile 등)
3. 다운로드하여 로컬 사용

### npm install이 실패해요?

```bash
# 캐시 지우기
npm cache clean --force

# 미러 사용
npm config set registry https://registry.npmmirror.com

# 다시 설치 시도
npm install bootstrap@5
```

### 스타일이 작동하지 않아요?

1. CSS 파일이 제대로 연결되어 있는지 확인
2. 브라우저 캐시 확인 (Ctrl+Shift+R로 강제 새로고침)
3. CSS 충돌 확인

## 사용 문제

### 그리드 시스템이 작동하지 않아요?

1. 행이 컨테이너 안에 있는지 확인
2. 열이 행 안에 있는지 확인
3. 중단점 클래스를 올바르게 사용하고 있는지 확인

### 반응형 디자인이 작동하지 않아요?

viewport 메타 태그를 추가했는지 확인:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### 모바일에서 Navbar가 접히지 않아요?

1. bootstrap.bundle.min.js를 사용하고 있는지 확인 (Popper 포함)
2. data-bs-toggle 및 data-bs-target 속성 확인

### 모달이 표시되지 않아요?

1. Bootstrap JS가 포함되어 있는지 확인
2. 버튼의 data-bs-toggle 및 data-bs-target 속성 확인
3. 모달 ID가 정확한지 확인

### 드롭다운이 작동하지 않아요?

1. bootstrap.bundle.min.js 포함
2. data-bs-toggle="dropdown" 속성이 있는지 확인

## 스타일 질문

### 테마 색상을 어떻게 변경하나요?

Sass 사용:

```scss
$primary: #007bff;
@import "bootstrap/scss/bootstrap";
```

### 기본 스타일을 어떻게 재정의하나요?

더 구체적인 선택자 사용:

```css
.navbar {
    background-color: #f8f9fa !important;
}
```

### 글꼴을 어떻게 변경하나요?

```scss
$font-family-sans-serif: "Helvetica Neue", Helvetica, Arial, sans-serif;
@import "bootstrap/scss/bootstrap";
```

### 아이콘이 표시되지 않아요?

Bootstrap 5는 Bootstrap Icons를 사용합니다:

```html
<link rel="stylesheet" href="https://bpscdn.jsdelivr.net/npm/bootstrap-icons@1.10.0/font/bootstrap-icons.css">

<i class="bi bi-alarm"></i>
```

## 컴포넌트 질문

### 카드를 어떻게 중앙에 놓나요?

```html
<div class="card">
    <div class="card-body">
        콘텐츠
    </div>
</div>
```

또는 flexbox 사용:

```html
<div class="card">
    <div class="card-body d-flex justify-content-center align-items-center">
        중앙 정렬된 콘텐츠
    </div>
</div>
```

### 폼 유효성 검사가 작동하지 않아요?

Bootstrap 5는 새로운 폼 유효성 검사를 사용합니다:

```html
<form class="row g-3 needs-validation" novalidate>
    <div class="col-md-4">
        <label for="validationCustom01" class="form-label">이름</label>
        <input type="text" class="form-control" id="validationCustom01" required>
        <div class="valid-feedback">좋습니다!</div>
        <div class="invalid-feedback">이름을 입력해주세요.</div>
    </div>
    <div class="col-12">
        <button class="btn btn-primary" type="submit">제출</button>
    </div>
</form>

<script>
    (function () {
        'use strict'
        var forms = document.querySelectorAll('.needs-validation')
        Array.prototype.slice.call(forms).forEach(function (form) {
            form.addEventListener('submit', function (event) {
                if (!form.checkValidity()) {
                    event.preventDefault()
                    event.stopPropagation()
                }
                form.classList.add('was-validated')
            }, false)
        })
    })()
</script>
```

### 배경 클릭 시 모달을 닫으려면 어떻게 하나요?

배경 속성 설정:

```html
<div class="modal" data-bs-backdrop="static">
```

`static` — 배경 클릭해도 닫히지 않음
`true` — 배경 클릭하면 닫힘
`false` — 배경 비활성화

## 성능 질문

### 파일 크기를 줄이려면 어떻게 하나요?

1. 필요한 것만 가져오기 (Sass 사용)
2. 프로덕션 빌드 사용
3. Gzip 압축 활성화

### 선택적으로 스타일을 가져오려면 어떻게 하나요?

필요한 컴포넌트만 가져오기 위해 Sass 사용:

```scss
// 필요한 함수와 변수 가져오기
@import "bootstrap/scss/functions";
@import "bootstrap/scss/variables";

// 필요한 것만 가져오기
@import "bootstrap/scss/root";
@import "bootstrap/scss/reboot";
@import "bootstrap/scss/containers";
@import "bootstrap/scss/grid";
@import "bootstrap/scss/buttons";
@import "bootstrap/scss/forms";
```

## 호환성 질문

### IE 브라우저가 지원되지 않아요?

Bootstrap 5는 더 이상 Internet Explorer를 지원하지 않습니다. IE11 지원이 필요하면 Bootstrap 4를 사용하세요.

### 모바일 클릭 지연이 있나요?

Bootstrap 5는 touch-action: manipulation을 사용하여 모바일에서 300ms 클릭 지연을 제거합니다.
