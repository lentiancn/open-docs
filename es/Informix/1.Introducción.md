# Introducción a IBM Informix

## ¿Qué es IBM Informix?

IBM Informix es una familia de sistemas de gestión de bases de datos relacionales (RDBMS) y ofertas de bases de datos multimodelo desarrolladas originalmente por Informix Corporation y adquiridas por IBM en 2001. En 2017, IBM delegó el desarrollo y soporte activo a HCLSoftware, mientras IBM continúa comercializándolo como IBM Informix.

Informix está optimizado para entornos con administración de bases de datos muy baja o nula, incluyendo su uso como base de datos embebida. Tiene un largo historial de apoyo a tasas de transacción muy altas y proporciona las características de tiempo de actividad necesarias para aplicaciones críticas como líneas de fabricación y sistemas de reservas.

## Características Principales

- **Base de Datos Multimodelo**: Soporta capacidades relacionales y orientadas a objetos para modelar estructuras de datos complejas
- **Alto Rendimiento**: Optimizado para transacciones siempre activas y análisis en tiempo real
- **Huella Pequeña**: Puede ejecutarse en dispositivos tan compactos como 100 MB (ej. Raspberry Pi)
- **Computación perimetral**: Ideal para soluciones IoT con capacidades de base de datos embebida
- **Soporte de Series Temporales**: Soporte integrado para datos de series temporales
- **Compatibilidad JSON/MongoDB**: Integración con colecciones JSON y controladores JSON MongoDB
- **Alta Disponibilidad**: Características como HDR (Replicación de Datos de Alta Disponibilidad), RSS y SDS

## Ediciones del Producto

A partir de 2025-2026, IBM Informix ofrece las siguientes ediciones:

| Edición | Usuarios Objetivo | Límites | Características Clave |
|---------|------------------|---------|----------------------|
| **Developer Edition** | Desarrolladores individuales | 1 CPU VP/Core, 1GB RAM, 8GB almacenamiento | Sin costo para desarrollo/pruebas |
| **Innovator-C Edition** | Pequeñas empresas | 2 CPU Cores, 8GB RAM, 50GB almacenamiento | Gratis para desarrollo/producción pequeña |
| **Express Edition** | Desarrolladores de terceros | 4 CPU Cores, 8GB RAM | Base de datos embebible |
| **Workgroup Edition** | Empresas medianas | Max 24 CPU cores, 32GB RAM | Todas las opciones de replicación |
| **Enterprise Edition** | Grandes empresas | Unlimited cores/RAM | Todas las funciones excepto IWA |
| **Advanced Enterprise Edition** | Enterprise + BI | Unlimited | Suite completa incluyendo IWA |

## Historial de Versiones

- **Versión Actual**: IBM Informix 15.0.0.1 (Marzo 2025)
- **Mejora Clave en v15.0**: Eliminación de la mayoría de los límites internos incluyendo límites de filas por página de datos, páginas de datos por partición de tabla y almacenamiento total
