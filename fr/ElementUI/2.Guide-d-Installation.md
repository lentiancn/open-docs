# Element UI Guide d'Installation

Guide complet d'installation et de configuration d'Element UI.

## Exigences environnementales

### Exigences logicielles

- Node.js: 12.0.0+
- npm: 6.0.0+ ou yarn: 1.22.0+
- Vue: 2.6.0+

## Étapes d'installation

### Méthode 1: installation npm (Recommandé)

```bash
# Utiliser npm
npm install element-ui --save

# Ou utiliser yarn
yarn add element-ui
```

### Méthode 2: Importation CDN

```html
<!-- Importer les styles -->
<link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">

<!-- Importer Vue -->
<script src="https://unpkg.com/vue"></script>

<!-- Importer Element UI -->
<script src="https://unpkg.com/element-ui/lib/index.js"></script>
```

## Démarrage rapide

### Importation complète

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(ElementUI)

new Vue({
  el: '#app',
  render: h => h(App)
})
```

### Importation à la demande (Recommandé)

Installer d'abord babel-plugin-component:

```bash
npm install babel-plugin-component -D
```

Configurer babel.config.js:

```javascript
module.exports = {
  presets: [
    '@vue/cli-plugin-babel/preset'
  ],
  plugins: [
    [
      'component',
      {
        libraryName: 'element-ui',
        styleLibraryName: 'theme-chalk'
      }
    ]
  ]
}
```

Importer les composants à la demande:

```javascript
import { Button, Select, Option, Table, TableColumn } from 'element-ui'

Vue.use(Button)
Vue.use(Select)
Vue.use(Option)
Vue.use(Table)
Vue.use(TableColumn)
```

## Personnalisation du thème

### Utiliser les variables SCSS

Créer un fichier de variables:

```scss
// styles/element-variables.scss

$--color-primary: #409EFF;
$--color-success: #67C23A;
$--color-warning: #E6A23C;
$--color-danger: #F56C6C;

@import 'element-ui/packages/theme-chalk/src/common/var.scss';
```

Importer dans main.js:

```javascript
import './styles/element-variables.scss'
```

### Éditeur de thème en ligne

Visiter: https://element.eleme.io/#/fr/theme

1. Sélectionner ou personnaliser les couleurs
2. Cliquer sur "Télécharger"
3. Remplacer les fichiers du thème

## Internationalisation

### Configurer le multilingue

Installer vue-i18n:

```bash
npm install vue-i18n -S
```

Configurer:

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import VueI18n from 'vue-i18n'
import frLocale from 'element-ui/lib/locale/lang/fr'

Vue.use(ElementUI, {
  i18n: (key, value) => i18n.t(key, value)
})

const messages = {
  'fr': { message: 'bonjour' }
}

const i18n = new VueI18n({
  locale: 'fr',
  messages
})

new Vue({ i18n }).$mount('#app')
```

## Problèmes courants

### Les composants ne s'affichent pas?

1. Confirmer que les composants sont correctement importés
2. Vérifier que Vue.use() est appelé
3. Vérifier la console pour les erreurs

### Les styles ne s'appliquent pas?

1. Confirmer que le fichier de styles est importé
2. Vérifier les remplacements de styles globaux
3. Vérifier les niveaux z-index

### Les icônes ne s'affichent pas?

```bash
# Installer la bibliothèque d'icônes
npm install element-ui-icons
```

Importer les icônes:

```javascript
import 'element-ui-icons'
```

Utilisation:

```html
<i class="el-icon-search"></i>
```

```bash
npm uninstall element-ui
```
