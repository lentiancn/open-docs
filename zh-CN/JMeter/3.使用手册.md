# JMeter 使用手册

## 创建测试计划

1. 添加线程组：Test Plan -> Add -> Thread Group
2. 添加 HTTP 请求：Thread Group -> Add -> Sampler -> HTTP Request
3. 添加监听器：Thread Group -> Add -> Listener -> View Results Tree

## HTTP 请求配置

```
Server Name: example.com
Port: 80
Path: /api/users
Method: GET
```

## 参数化

### CSV 数据文件

```
Thread Group -> Add -> Config Element -> CSV Data Set Config
Filename: users.csv
Variable Names: username,password
```

### 用户自定义变量

```
Test Plan -> Add -> Config Element -> User Defined Variables
```

## 断言

```
HTTP Request -> Add -> Assertions -> Response Assertion
```

## 命令行运行

```bash
# 运行测试
jmeter -n -t test.jmx -l result.jtl -e -o report

# 参数
-n: 非 GUI 模式
-t: 测试文件
-l: 结果文件
-e: 生成报告
-o: 报告目录
```
