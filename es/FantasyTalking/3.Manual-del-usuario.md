# FantasyTalking Manual del Usuario

## Funciones Básicas

FantasyTalking proporciona funciones completas de síntesis de voz y diálogo. Este manual presenta en detalle cómo usar varias funciones.

### Síntesis de Voz

La síntesis de voz es la función principal de FantasyTalking, convirtiendo texto en voz natural y fluida:

```javascript
// Texto simple a voz
const audio = await client.speak('Hola, bienvenido a FantasyTalking');

// Especificar estilo de voz
const audio = await client.speak('El clima está muy bueno hoy', {
  voice: {
    style: 'happy',
    language: 'es-ES'
  }
});

// Especificar velocidad y tono
const audio = await client.speak('Por favor espere un momento', {
  voice: {
    speed: 1.2,
    pitch: 0.9
  }
});
```

### Lenguaje de Marcado SSML

FantasyTalking soporta lenguaje SSML para control más fino de la salida de voz:

```xml
<speak>
  <p>
    <s>Primera oración.</s>
    <s>Segunda oración.</s>
  </p>
  <break time="500ms"/>
  <p>
    <emphasis level="moderate">Este es contenido importante</emphasis>
  </p>
</speak>
```

### Eventos de Voz

Durante la reproducción de voz, puedes escuchar varios eventos:

```javascript
// Escuchar inicio de voz
client.on('speech-start', (event) => {
  console.log('Reproducción de voz iniciada');
});

// Escuchar límites de palabras
client.on('word-boundary', (event) => {
  console.log(`Palabra actual: ${event.word}`);
});

// Escuchar fin de voz
client.on('speech-end', (event) => {
  console.log('Reproducción de voz completada');
});

// Escuchar errores
client.on('error', (event) => {
  console.error('Error:', event.error);
});
```

### Síntesis por Lotes

Para sintetizar grandes cantidades de texto, puedes usar síntesis por lotes:

```javascript
const texts = [
  'Esta es la primera voz',
  'Esta es la segunda voz',
  'Esta es la tercera voz'
];

// Síntesis paralela
const results = await Promise.all(
  texts.map(text => client.speak(text))
);
```

## Funciones de Diálogo

### Diálogo Básico

FantasyTalking proporciona funciones potentes de gestión de diálogo:

```javascript
// Crear sesión de diálogo
const session = client.createSession({
  systemPrompt: 'Eres un asistente amigable, bueno respondiendo preguntas',
  language: 'es-ES'
});

// Primera ronda
const response1 = await session.chat('¿Cómo está el clima hoy?');
console.log(response1.text);
console.log(response1.audio);

// Segunda ronda (incluye contexto automáticamente)
const response2 = await session.chat('¿Y mañana?');
console.log(response2.text);
```

## Funciones Avanzadas

### Síntesis de Voz en Streaming

Para escenarios que requieren retroalimentación en tiempo real:

```javascript
// Habilitar salida de streaming
const stream = await client.speakStream('Este es un texto largo');

// Escuchar chunks de datos
stream.on('data', (chunk) => {
  audioPlayer.play(chunk);
});

stream.on('end', () => {
  console.log('Síntesis de voz completada');
});

stream.on('error', (error) => {
  console.error('Error:', error);
});
```

### Modelos de Voz Personalizados

Si necesitas una voz de marca exclusiva:

```javascript
const trainingData = {
  name: 'Voz de Marca',
  samples: [
    { text: 'Texto de muestra 1', audio: 'audio1 codificado en base64' },
    { text: 'Texto de muestra 2', audio: 'audio2 codificado en base64' }
  ]
};

const job = await client.trainVoice(trainingData);

while (job.status !== 'completed') {
  await sleep(5000);
  const status = await client.getJobStatus(job.id);
  console.log(`Progreso: ${status.progress}%`);
}
```

## Mejores Prácticas

### Optimización de Rendimiento

1. **Reutilización de conexiones**: Evita crear instancias de cliente frecuentemente
2. **Procesamiento por lotes**: Usa síntesis por lotes para grandes cantidades de texto
3. **Caché**: Usa caché para evitar síntesis duplicada
4. **Salida de streaming**: Usa streaming para textos largos

### Manejo de Errores

```javascript
try {
  const audio = await client.speak('Texto de prueba');
} catch (error) {
  if (error.code === 'RATE_LIMIT') {
    console.log('Demasiadas solicitudes, por favor intenta más tarde');
  } else if (error.code === 'INVALID_API_KEY') {
    console.log('Clave API inválida');
  }
}
```

### Experiencia de Usuario

1. **Precarga**: Pre-sintetiza voz antes de que el usuario la necesite
2. **Comentarios**: Proporciona retroalimentación visual durante la reproducción
3. **Manejo de interrupción**: Permite al usuario interrumpir la reproducción
4. **Soporte sin conexión**: Implementa soporte sin conexión para funciones básicas
