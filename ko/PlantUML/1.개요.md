# PlantUML 개요

> 텍스트 설명에서 UML 다이어그램을 그리는 오픈소스 도구

---

## 개요

PlantUML은 사용자가 간단한 텍스트 언어로 다이어그램을 만들 수 있는 오픈소스 도구입니다. UML 다이어그램, 네트워크 다이어그램, 플로우차트 등을 지원합니다.

---

## 주요 기능

- **다양한 다이어그램 유형**: UML, 액티비티, 시퀀스, 클래스
- **텍스트 기반**: 간단한 텍스트 구문으로 다이어그램 정의
- **IDE 지원**: VS Code, IntelliJ, Eclipse 플러그인
- **내보내기**: PNG, SVG, PDF
