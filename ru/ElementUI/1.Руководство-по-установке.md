# Element UI Руководство по установке

> Element UI — это библиотека компонентов для настольных компьютеров на базе Vue 2
> 
> Обновлено: Февраль 2026
> 
> Текущая версия: 2.x (Стабильная)
> 
> Примечание: Для проектов Vue 3 используйте Element Plus

---

## 1. Описание проекта

Element UI — это библиотека компонентов для настольных компьютеров на Vue 2.0, разработанная и поддерживаемая командой Ele.me. Она предоставляет богатый набор часто используемых компонентов для ПК, включая формы, таблицы, диалоги, навигационные меню, кнопки, поля ввода и другие элементы интерфейса.

Element UI имеет чистый и элегантный дизайн с подробной документацией. Это одна из самых популярных библиотек компонентов Vue 2 в Китае. Хотя официальная поддержка прекращена (рекомендуется переход на Element Plus для Vue 3), Element UI по-прежнему имеет высокую справочную ценность.

### 1.1 Информация о версиях

| Версия | Vue Версия | Статус | Описание |
|--------|------------|--------|----------|
| Element UI 2.x | Vue 2 | Стабильная | Основная версия документации |
| Element UI 1.x | Vue 2 | Прекращена | Ранняя версия |

### 1.2 Отличия от Element Plus

| Функция | Element UI | Element Plus |
|---------|------------|--------------|
| Vue Версия | Vue 2 | Vue 3 |
| TypeScript | Частичная поддержка | Полная поддержка |
| Composition API | Не поддерживается | Поддерживается |
| Статус поддержки | Прекращена | Активно поддерживается |

## 3. Шаги установки

### 3.1 Использование npm (Рекомендуется)

```bash
# Перейдите в каталог проекта Vue 2
cd ваш-проект-vue

# Установите Element UI
npm install element-ui --save
```

### 3.2 Использование yarn

```bash
yarn add element-ui
```

---

## 4. Подключение в проекте Vue 2

### 4.1 Полное подключение

Импортируйте Element UI и его стили в главном файле (обычно main.js):

```javascript
// main.js
import Vue from 'vue'
import App from './App.vue'

import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)

new Vue({
  render: h => h(App)
}).$mount('#app')
```

### 4.2 Частичное подключение (Рекомендуется)

Для уменьшения размера пакета импортируйте только нужные компоненты:

```javascript
import { Button, Form, FormItem, Input } from 'element-ui'

Vue.use(Button)
Vue.use(Form)
Vue.use(FormItem)
Vue.use(Input)
```

## 5. Примеры для быстрого старта

### 5.1 Базовая кнопка

```html
<el-button>По умолчанию</el-button>
<el-button type="primary">Основная</el-button>
<el-button type="success">Успех</el-button>
<el-button type="warning">Предупреждение</el-button>
<el-button type="danger">Опасность</el-button>
```

### 5.2 Базовая форма

```html
<el-form :model="form" label-width="80px">
  <el-form-item label="Имя">
    <el-input v-model="form.name"></el-input>
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="submit">Отправить</el-button>
  </el-form-item>
</el-form>
```

### 5.3 Базовая таблица

```html
<el-table :data="tableData" stripe border>
  <el-table-column prop="name" label="Имя"></el-table-column>
  <el-table-column prop="email" label="Email"></el-table-column>
</el-table>
```

### 5.4 Диалог

```html
<el-dialog title="Заголовок" :visible.sync="visible">
  <span>Содержание диалога</span>
</el-dialog>
```
