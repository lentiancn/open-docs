# Bootstrap 简介

最流行的前端响应式 CSS 框架。

## 什么是 Bootstrap？

Bootstrap 是由 Twitter 开发的最流行的前端 CSS 框架，用于构建响应式、移动优先的网站和 Web 应用。

### 主要特性

- **响应式网格系统** - 12 列响应式网格
- **预制组件** - 导航栏、卡片、模态框等
- **CSS 工具类** - 快速样式调整
- **JavaScript 组件** - 交互式组件
- **自定义选项** - SASS 变量定制
- **浏览器兼容** - 支持所有现代浏览器
