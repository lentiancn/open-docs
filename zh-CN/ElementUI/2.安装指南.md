# Element UI 安装指南

完整的 Element UI 安装和配置指南。

## 环境要求

### 软件要求

- Node.js：12.0.0+
- npm：6.0.0+ 或 yarn：1.22.0+
- Vue：2.6.0+

## 安装步骤

### 方式一：npm 安装（推荐）

```bash
# 使用 npm 安装
npm install element-ui --save

# 或使用 yarn
yarn add element-ui
```

### 方式二：CDN 引入

```html
<!-- 引入样式 -->
<link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">

<!-- 引入组件库 -->
<script src="https://unpkg.com/vue"></script>
<script src="https://unpkg.com/element-ui/lib/index.js"></script>
```

## 快速开始

### 全量引入

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(ElementUI)

new Vue({
  el: '#app',
  render: h => h(App)
})
```

### 按需引入（推荐）

首先安装 babel-plugin-component：

```bash
npm install babel-plugin-component -D
```

配置 babel.config.js：

```javascript
module.exports = {
  presets: [
    '@vue/cli-plugin-babel/preset'
  ],
  plugins: [
    [
      'component',
      {
        libraryName: 'element-ui',
        styleLibraryName: 'theme-chalk'
      }
    ]
  ]
}
```

按需引入组件：

```javascript
import { Button, Select, Option, Table, TableColumn } from 'element-ui'

Vue.use(Button)
Vue.use(Select)
Vue.use(Option)
Vue.use(Table)
Vue.use(TableColumn)
```

## 主题定制

### 使用 SCSS 变量

创建变量文件：

```scss
// styles/element-variables.scss

$--color-primary: #409EFF;
$--color-success: #67C23A;
$--color-warning: #E6A23C;
$--color-danger: #F56C6C;

@import 'element-ui/packages/theme-chalk/src/common/var.scss';
```

在 main.js 中引入：

```javascript
import './styles/element-variables.scss'
```

### 在线主题编辑

访问：https://element.eleme.io/#/zh-CN/theme

1. 选择或自定义主题颜色
2. 点击「下载」按钮
3. 将下载的文件替换 element-ui 的主题文件

## 国际化

### 配置多语言

安装 vue-i18n：

```bash
npm install vue-i18n -S
```

配置：

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import VueI18n from 'vue-i18n'
import zhLocale from 'element-ui/lib/locale/lang/zh-CN'
import enLocale from 'element-ui/lib/locale/lang/en'

Vue.use(ElementUI, {
  i18n: (key, value) => i18n.t(key, value)
})

const messages = {
  'zh-CN': { message: '你好' },
  'en': { message: 'hello' }
}

const i18n = new VueI18n({
  locale: 'zh-CN',
  messages
})

new Vue({ i18n }).$mount('#app')
```

## 常见问题

### 组件不显示？

1. 确认已正确引入组件
2. 检查是否调用了 Vue.use()
3. 查看控制台错误信息

### 样式冲突？

1. 检查是否有全局样式覆盖
2. 使用更具体的选择器
3. 检查 z-index 层级

### 图标不显示？

```bash
# 安装图标库
npm install element-ui-icons
```

引入图标：

```javascript
import 'element-ui-icons'
```

使用：

```html
<i class="el-icon-search"></i>
```
