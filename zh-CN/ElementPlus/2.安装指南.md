# Element Plus 安装指南

Element Plus 多种安装方式详解。

## 环境要求

### 硬件要求

- CPU：2 核及以上
- 内存：4GB 及以上

### 软件要求

- Node.js：18.0+
- npm：9.0+ / yarn：1.22+ / pnpm：8.0+
- Vue：3.2+
- Vite：4.0+（推荐）

## 包管理器安装（推荐）

### npm 安装

```bash
npm install element-plus
```

### yarn 安装

```bash
yarn add element-plus
```

### pnpm 安装

```bash
pnpm add element-plus
```

## 完整引入

### main.ts 配置

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')
```

### 按需引入（推荐）

#### 自动导入（推荐）

**1. 安装依赖**

```bash
npm install -D unplugin-vue-components unplugin-auto-import
```

**2. 配置 Vite**

```typescript
// vite.config.ts
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

**3. 使用组件**

```vue
<template>
  <el-button type="primary">Hello</el-button>
</template>
```

#### 手动按需引入

```typescript
import { ElButton, ElInput } from 'element-plus'

export default {
  components: {
    ElButton,
    ElInput,
  },
}
```

## Vue CLI 安装

### 安装

```bash
npm install -D sass sass-loader
npm install element-plus
```

### 配置

```javascript
// vue.config.js
module.exports = {
  configureWebpack: {
    plugins: [
      AutoImport({
        resolvers: [ElementPlusResolver()],
      }),
      Components({
        resolvers: [ElementPlusResolver()],
      }),
    ],
  },
}
```

## CDN 引入

### HTML 引入

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>Element Plus CDN</title>
  <link rel="stylesheet" href="//unpkg.com/element-plus/dist/index.css" />
  <script src="//unpkg.com/vue@3"></script>
  <script src="//unpkg.com/element-plus"></script>
</head>
<body>
  <div id="app">
    <el-button type="primary">主要按钮</el-button>
  </div>

  <script>
    const { createApp, ref } = Vue
    const App = {
      setup() {
        const visible = ref(false)
        return { visible }
      },
      template: `
        <el-button @click="visible = true">打开对话框</el-button>
        <el-dialog v-model="visible" title="提示">内容</el-dialog>
      `
    }
    const app = createApp(App)
    app.use(ElementPlus)
    app.mount('#app')
  </script>
</body>
</html>
```

## 主题定制

### 使用 CSS 变量

```css
:root {
  --el-color-primary: #409eff;
  --el-color-success: #67c23a;
  --el-border-radius-base: 4px;
}
```

### 深度定制主题

**1. 安装主题生成器**

```bash
npm install -D element-theme-chalk
```

**2. 初始化主题文件**

```bash
et -i
```

**3. 修改变量**

编辑 `element-variables.scss`，修改颜色变量后：

```bash
et
```

### 使用暗色主题

```typescript
import 'element-plus/theme-chalk/dark/css-vars.css'
```

## 常见问题

### 图标不显示？

安装图标包：

```bash
npm install @element-plus/icons-vue
```

注册：

```typescript
import * as icons from '@element-plus/icons-vue'

for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```

### 样式冲突？

使用 CSS Modules 或 scoped：

```vue
<style scoped>
:deep(.el-button) {
  margin: 10px;
}
</style>
```

### TypeScript 类型错误？

确保安装类型定义：

```bash
npm install -D @types/node
```

在 `tsconfig.json` 中配置：

```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

### 国际化配置

```typescript
import zhCn from 'element-plus/es/locale/lang/zh-cn'

app.use(ElementPlus, { locale: zhCn })
```

## 卸载

### npm 卸载

```bash
npm uninstall element-plus
```

### 清理配置

删除以下文件：
- `element-variables.scss`（如使用自定义主题）
- 相关 Vite/Webpack 配置
