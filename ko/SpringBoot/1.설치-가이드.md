# Spring Boot 설치 가이드

이 가이드에서는 Java, Maven 및 Gradle을 포함한 Spring Boot 개발 환경을 설정하는 방법을 설명합니다.

## 전제 조건

### Java 설치

Spring Boot 3.x는 Java 17 이상이 필요합니다. Spring Boot 2.x는 Java 8 이상이 필요합니다.

**Windows:**
1. [Oracle](https://www.oracle.com/java/technologies/downloads/) 또는 [Adoptium](https://adoptium.net/)에서 JDK를 다운로드하세요
2. 설치 프로그램을 실행하세요
3. 환경 변수를 설정하세요:
   ```
   JAVA_HOME=C:\Program Files\Java\jdk-17
   PATH=%JAVA_HOME%\bin;%PATH%
   ```

**macOS:**
```bash
# Homebrew 사용
brew install openjdk@17

# PATH에 추가
echo 'export JAVA_HOME=$(/usr/libexec/java_home)' >> ~/.zshrc
source ~/.zshrc
```

**Linux:**
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install openjdk-17-jdk

# JAVA_HOME 설정
echo 'export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64' >> ~/.bashrc
source ~/.bashrc
```

설치 확인:
```bash
java -version
```

### Maven 설치

**Windows:**
1. [Apache Maven](https://maven.apache.org/download.cgi)에서 다운로드
2. 폴더에 압축 해제 (예: C:\apache-maven)
3. 환경 변수를 설정하세요:
   ```
   MAVEN_HOME=C:\apache-maven
   PATH=%MAVEN_HOME%\bin;%PATH%
   ```

**macOS:**
```bash
brew install maven
```

**Linux:**
```bash
sudo apt install maven
```

설치 확인:
```bash
mvn -version
```

### Gradle 설치

**Windows:**
1. [Gradle](https://gradle.org/install/)에서 다운로드
2. 압축 해제 후 PATH에 추가

**macOS:**
```bash
brew install gradle
```

**Linux:**
```bash
sudo apt install gradle
```

설치 확인:
```bash
gradle -version
```

### VS Code
- Extension Pack for Java 설치
- Spring Boot Extension 설치

## Spring Initializr로 빠른 시작

1. [start.spring.io](https://start.spring.io/)를 방문하세요
2. 선택:
   - 프로젝트: Maven 또는 Gradle
   - 언어: Java
   - Spring Boot 버전
   - 프로젝트 메타데이터
3. 의존성 추가 (Spring Web, Spring Data JPA 등)
4. "Generate"를 클릭하여 프로젝트 다운로드

## 설정 확인

```bash
# Java 확인
java -version

# Maven 확인
mvn -version

# Gradle 확인 (사용하는 경우)
gradle -version

# 간단한 Spring Boot 프로젝트 생성 및 실행
curl https://start.spring.io/starter.zip -o demo.zip
unzip demo.zip -d demo
cd demo
./mvnw spring-boot:run  # 또는 ./gradlew bootRun
```
