# Flutter - Installationsanleitung

Vollständige Anleitung zur Einrichtung der Flutter-Entwicklungsumgebung.

## Anforderungen

### Windows

- **Betriebssystem**: Windows 10 oder höher (64-Bit)
- **Speicherplatz**: Mindestens 2,8 GB
- **Werkzeuge**: Windows PowerShell 5.0+, Git für Windows

### macOS

- **Betriebssystem**: macOS 10.14 (Mojave) oder höher
- **Speicherplatz**: Mindestens 2,8 GB
- **Werkzeuge**: Command Line Tools, Git

### Linux

- **Betriebssystem**: Ubuntu 18.04 oder höher
- **Speicherplatz**: Mindestens 2,8 GB
- **Abhängigkeiten**: bash, curl, git, unzip, xz-utils, zip, libglu1-mesa

## Installationsschritte

### Windows-Installation

1. **Flutter SDK herunterladen**

Besuchen Sie die Flutter-Website, um die neueste Version herunterzuladen:
```bash
# Oder per Git klonen
git clone https://github.com/flutter/flutter.git -b stable
```

2. **Umgebungsvariablen konfigurieren**

Fügen Sie Flutter zum System-PATH hinzu:
```powershell
# In PowerShell ausführen
[Environment]::SetEnvironmentVariable(
    "Path",
    $env:Path + ";C:\src\flutter\bin",
    "User"
)
```

3. **Installation überprüfen**

```bash
flutter doctor
```

### macOS-Installation

1. **Flutter SDK herunterladen**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

2. **Umgebungsvariablen konfigurieren**

```bash
# Zu ~/.bash_profile oder ~/.zshrc hinzufügen
export PATH="$PATH:/Users/Benutzername/flutter/bin"
```

3. **flutter doctor ausführen**

```bash
flutter doctor
```

### Linux-Installation

1. **Abhängigkeiten installieren**

```bash
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa
```

2. **Flutter SDK herunterladen**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

3. **PATH konfigurieren**

```bash
export PATH="$PATH:/home/Benutzername/flutter/bin"
```

## Android-Entwicklungsumgebung

### Android Studio installieren

1. Android Studio herunterladen: https://developer.android.com/studio
2. Installieren und starten
3. SDK konfigurieren:

```bash
flutter config --android-sdk /path/to/android/sdk
```

### Lizenzen akzeptieren

```bash
flutter doctor --android-licenses
```

### Emulator konfigurieren

```bash
# Emulator erstellen
flutter emulator --create --name my-emulator

# Emulator starten
flutter emulator --launch my-emulator
```

## iOS-Entwicklungsumgebung (macOS)

### Xcode installieren

```bash
# Über App Store installieren
xcode-select --install
```

### iOS-Simulator konfigurieren

```bash
# Verfügbare Simulatoren anzeigen
xcrun simctl list devices available

# Simulator starten
open -a Simulator
```

## IDE-Konfiguration

### VS Code

1. Flutter-Erweiterung installieren
2. Dart-Erweiterung installieren
3. VS Code neu starten

### Android Studio

1. Flutter-Plugin installieren
2. Dart-Plugin installieren
3. Android Studio neu starten

## Installation überprüfen

```bash
# Flutter-Version prüfen
flutter --version

# flutter doctor ausführen
flutter doctor

# Nach Updates suchen
flutter upgrade
```

## Häufige Probleme

### Download fehlgeschlagen?

Spiegel verwenden:
```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
export PUB_CACHE=/path/to/cache
```

### Android SDK nicht gefunden?

```bash
flutter config --android-sdk /path/to/android/sdk
```

### Befehl nicht gefunden?

Flutter im PATH:
```bash
echo $PATH
# flutter/bin sollte enthalten sein
```

## Deinstallation

```bash
# Flutter-Verzeichnis löschen
rm -rf /path/to/flutter

# Aus PATH entfernen
```
