# Claude Code 常见问题

常见问题解答。

## 安装问题

### Q: 安装失败？

**解决方案**：
1. 检查系统要求
2. 更新 VS Code 到最新版本
3. 检查网络连接
4. 尝试重新安装

### Q: 无法登录？

**解决方案**：
1. 检查 API 密钥是否有效
2. 检查网络是否正常
3. 尝试清除缓存

### Q: 扩展不显示？

**解决方案**：
1. 重启 VS Code
2. 检查扩展是否启用
3. 查看 VS Code 通知

## 使用问题

### Q: 响应慢？

**解决方案**：
1. 检查网络连接
2. 减少上下文长度
3. 等待服务器响应

### Q: 代码生成质量差？

**解决方案**：
1. 描述更具体
2. 提供更多上下文
3. 迭代改进

### Q: 如何切换模型？

在设置中选择模型版本。

### Q: 支持哪些语言？

支持主流编程语言：Python、JavaScript、TypeScript、Go、Rust、Java、C++ 等。

## 账户问题

### Q: 如何获取 API 密钥？

访问 https://console.anthropic.com 创建。

### Q: 额度用完了？

在控制台购买额外额度。

## 安全问题

### Q: 代码会泄露吗？

Claude Code 不会保存或分享你的代码。

### Q: 如何保护隐私？

- 不在对话中输入敏感信息
- 使用本地模式
- 定期清除会话
