# Introducción a H2

## ¿Qué es H2?

H2 es un sistema de gestión de base de datos relacional de código abierto escrito en Java. Se puede usar como base de datos integrada en aplicaciones Java o ejecutar en modo cliente-servidor. El software está disponible como código abierto bajo la Licencia Pública de Mozilla 2.0 (MPL 2.0) o la Licencia Pública de Eclipse 1.0 (EPL 1.0).

## Características principales de H2

### Base de datos integrada
H2 puede integrarse directamente en aplicaciones Java sin necesidad de un servidor de base de datos separado. Esto lo hace muy popular en entornos de desarrollo y prueba.

### Modo cliente-servidor
Además del modo integrado, H2 también soporta el modo cliente-servidor, ejecutándose como servidor de base de datos independiente accesible por múltiples aplicaciones simultáneamente.

### Base de datos en memoria
H2 soporta el modo de base de datos en memoria, donde los datos se almacenan en memoria y se perderán después de reiniciar. Esto es muy útil para pruebas rápidas y desarrollo de prototipos.

### Compatibilidad con múltiples bases de datos
H2 es compatible con varios estándares SQL y puede simular el comportamiento de MySQL, PostgreSQL, Oracle, DB2, MS SQL Server y otras bases de datos, facilitando la migración y prueba de bases de datos.

### Base de datos cifrada
H2 soporta el cifrado de bases de datos para proteger datos sensibles.

### Búsqueda de texto completo
H2 incluye funcionalidad de búsqueda de texto completo incorporada sin necesidad de configuración adicional.

### Cero administración
Como base de datos integrada, H2 no requiere un administrador de base de datos dedicado.

## Escenarios de aplicación de H2

H2 se usa ampliamente en:

- Base de datos integrada para aplicaciones Java
- Entornos de desarrollo y prueba
- Desarrollo rápido de prototipos
- Pruebas unitarias y de integración
- Aplicaciones de escritorio ligeras
- Aprendizaje de sintaxis SQL

## Historia de versiones de H2

El desarrollo del motor de base de datos H2 comenzó en mayo de 2004, publicado por primera vez en diciembre de 2005. El desarrollador original, Thomas Mueller, también desarrolló el motor de base de datos Java HSQLDB (originalmente llamado Hypersonic SQL).

- Entorno de ejecución Java (JRE) 11 o superior
- Puede ejecutarse en cualquier sistema operativo que soporte Java, incluyendo Windows, Linux, macOS y más
