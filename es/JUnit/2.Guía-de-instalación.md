# JUnit Guía de instalación

JUnit se puede agregar a tu proyecto de varias maneras, incluyendo Maven, Gradle y descarga directa de JAR. A continuación se detallan las instrucciones para cada versión.

## Instalación de JUnit 5

### Configuración de Maven

Agrega las siguientes dependencias en tu pom.xml:

```xml
<!-- Módulo agregador JUnit Jupiter -->
<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter</artifactId>
    <version>5.14.3</version>
    <scope>test</scope>
</dependency>
```

Si usas Maven Surefire Plugin para ejecutar pruebas, agrega la siguiente configuración:

```xml
<plugin>
    <groupId>org.apache.maven.plugins</groupId>
    <artifactId>maven-surefire-plugin</artifactId>
    <version>3.5.2</version>
</plugin>
```

### Configuración de Gradle

Agrega la dependencia en tu build.gradle:

```groovy
testImplementation 'org.junit.jupiter:junit-jupiter:5.14.3'
```

Para ejecutar pruebas:

```groovy
test {
    useJUnitPlatform()
}
```

### Descarga directa de JAR

Descarga los archivos JAR necesarios desde Maven Central:

- junit-jupiter-api: API de JUnit Jupiter
- junit-jupiter-engine: Motor de JUnit Jupiter
- junit-platform-console-standalone: Lanzador de consola independiente

## Instalación de JUnit 4

### Configuración de Maven

```xml
<dependency>
    <groupId>junit</groupId>
    <artifactId>junit</artifactId>
    <version>4.13.2</version>
    <scope>test</scope>
</dependency>
```

### Configuración de Gradle

```groovy
testImplementation 'junit:junit:4.13.2'
```

### Descarga directa de JAR

Descarga junit-4.13.2.jar desde Maven Central:

https://repo1.maven.org/maven2/junit/junit/4.13.2/junit-4.13.2.jar

## Instalación de JUnit Vintage

Para ejecutar pruebas JUnit 4 en la plataforma JUnit 5, agrega la dependencia del motor Vintage:

### Maven

```xml
<dependency>
    <groupId>org.junit.vintage</groupId>
    <artifactId>junit-vintage-engine</artifactId>
    <version>5.14.3</version>
    <scope>test</scope>
</dependency>
```

### Gradle

```groovy
testImplementation 'org.junit.vintage:junit-vintage-engine:5.14.3'
```

## Compatibilidad de versiones

- JUnit 5 requiere Java 8 o superior
- JUnit 4 requiere Java 5 o superior
- JUnit Vintage 5.14.3 requiere JUnit 4.12 o superior

## Verificar instalación

Crea una clase de prueba para verificar la instalación:

```java
import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;

class SampleTest {
    @Test
    void testExample() {
        assertEquals(2, 1 + 1);
    }
}
```

Ejecuta las pruebas:

```bash
mvn test
# o
gradle test
```
