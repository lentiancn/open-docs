# Bootstrap

Das beliebteste CSS-Framework der Welt.

## Übersicht

Bootstrap ist das beliebteste CSS-Framework von Twitter. Mit responsivem Gridsystem, vorgefertigten Komponenten und CSS-Hilfsklassen.

### Hauptfunktionen

- **Responsives Gridsystem** - 12-Spalten Grid
- **Vorgefertigte Komponenten** - Navbars, Karten, Modals
- **CSS-Hilfsklassen** - Schnelle Stilanpassungen

## Schnellstart

### CDN

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```
