# Guide d'installation d'Angular

Angular est un framework Web moderne maintenu par Google. Ce guide couvre les méthodes d'installation dans tous les environnements.

## Informations de version

| Version | Statut | Description |
|---------|--------|-------------|
| Angular 17+ | ✅ Actuelle | Dernière version avec toutes les nouvelles fonctionnalités |
| Angular 16 | ✅ Supportée | Version stable |
| Angular 14-15 | ⚠️ Maintenance | Mises à jour de sécurité |
| Angular 2-13 | ❌ Obsolète | Non recommandé pour les nouveaux projets |

> **Important** : Ce guide est principalement destiné à **Angular 17+**. Il est recommandé d'utiliser la dernière version pour les nouveaux projets afin d'obtenir les meilleures performances et les dernières fonctionnalités.

## Méthodes d'installation

### Méthode 1 : Utiliser Angular CLI (Recommandé)

Angular CLI est la méthode officiellement recommandée pour créer et gérer des projets Angular.

```bash
# Installer Angular CLI
npm install -g @angular/cli

# Créer un nouveau projet
ng new my-app

# Aller dans le répertoire du projet
cd my-app

# Démarrer le serveur de développement
ng serve
```

#### Options de création de projet

```bash
# Créer un projet avec routage
ng new my-app --routing

# Créer un projet avec SCSS
ng new my-app --style=scss

# Créer un projet avec SSR
ng new my-app --ssr

# Ignorer l'installation des dépendances
ng new my-app --skip-install
```

### Méthode 2 : Créer le projet manuellement

```bash
# Initialiser le projet npm
npm init -y

# Installer les dépendances core d'Angular
npm install @angular/core@17 @angular/common@17 @angular/compiler@17

# Installer les plateformes Angular
npm install @angular/platform-browser@17 @angular/platform-browser-dynamic@17

# Installer le compilateur Angular
npm install @angular/compiler-cli@17 typescript@~5.2.0 zone.js@~0.14.0
```

### Méthode 3 : Utiliser Vite

```bash
# Créer un projet Angular avec Vite
npm create vite@latest my-angular-app -- --template angular

# Aller dans le répertoire du projet
cd my-angular-app

# Installer les dépendances
npm install
```

### Méthode 4 : Utiliser Nx Workspace

Nx est un puissant outil monorepo adapté aux grands projets d'entreprise.

```bash
# Installer Nx
npm install -g nx

# Créer une application Angular
npx create-nx-workspace my-org --preset=angular

# Ajouter une application Angular
nx g @nx/angular:app my-app
```

## Commandes de développement

### Commandes courantes

```bash
# Démarrer le serveur de développement
ng serve

# Builder pour la production
ng build

# Exécuter les tests
ng test

# Exécuter les tests end-to-end
ng e2e

# Générer un composant
ng g c my-component

# Générer un service
ng g s my-service

# Générer un module
ng g m my-module

# Générer le routage
ng g m my-module --routing
```

### Options du serveur de développement

```bash
# Spécifier le port
ng serve --port 4200

# Ouvrir le navigateur
ng serve --open

# Hot Module Replacement
ng serve --hmr

# Build production
ng serve --configuration production
```

## Gestion des dépendances

### Installer les dépendances

```bash
# Installer les dépendances runtime
npm install @angular/animations @angular/common @angular/compiler @angular/core @angular/forms @angular/platform-browser @angular/platform-browser-dynamic @angular/router

# Installer les dépendances de développement
npm install --save-dev @angular-devkit/build-angular @angular/cli @angular/compiler-cli @types/jasmine @types/node typescript jasmine-core jasmine-spec-reporter karma karma-chrome-launcher karma-coverage karma-jasmine karma-jasmine-html-reporter zone.js
```

### Utiliser yarn

```bash
# Installer avec yarn
yarn install

# Ajouter une dépendance
yarn add @angular/core@17

# Ajouter une dépendance de développement
yarn add --dev @angular/cli@17
```

## Foire aux questions

### Les commandes Angular CLI ne fonctionnent pas ?

Assurez-vous qu'Angular CLI est installé globalement :

```bash
npm install -g @angular/cli
```

### La version de Node.js ne correspond pas ?

Utilisez nvm pour gérer les versions de Node.js :

```bash
# Installer nvm (Windows)
winget install CoreyButler.NVMforWindows

# Utiliser une version spécifique
nvm install 18
nvm use 18
```

### Le build échoue ?

Nettoyez le cache et réinstallez :

```bash
rm -rf node_modules dist
npm install
```
