# OpenFeign Manual de Usuario

## Balanceo de Carga

### Integración con Spring Cloud LoadBalancer

OpenFeign se integra con Spring Cloud LoadBalancer por defecto, proporcionando funcionalidad de balanceo de carga del lado del cliente.

```java
@FeignClient(name = "user-service")
public interface UserService {
    @GetMapping("/users/{id}")
    User getUser(@PathVariable("id") Long id);
}
```

Configure instancias de servicio en `application.yml`:

```yaml
spring:
  cloud:
    discovery:
      client:
        simple:
          instances:
            user-service:
              - uri: http://localhost:8081
              - uri: http://localhost:8082
              - uri: http://localhost:8083
```

### Estrategia de Balanceo de Carga Personalizada

```java
@Configuration
public class LoadBalancerConfig {
    
    @Bean
    public ServiceInstanceListSupplier serviceInstanceListSupplier(
            ConfigurableApplicationContext context) {
        return ServiceInstanceListSupplier.builder()
                .withDiscoveryClient()
                .withCaching()
                .build(context);
    }
}
```

### Transformar Solicitud HTTP con Balanceo de Carga

```java
@Bean
public LoadBalancerFeignRequestTransformer transformer() {
    return new LoadBalancerFeignRequestTransformer() {
        @Override
        public Request transformRequest(Request request, ServiceInstance instance) {
            Map<String, Collection<String>> headers = new HashMap<>(request.headers());
            headers.put("X-ServiceId", Collections.singletonList(instance.getServiceId()));
            headers.put("X-InstanceId", Collections.singletonList(instance.getInstanceId()));
            return Request.create(request.httpMethod(), request.url(), headers, 
                request.body(), request.charset(), request.requestTemplate());
        }
    };
}
```

## Soporte para Cortacircuitos

### Integración con Spring Cloud CircuitBreaker

Habilite el soporte para cortacircuitos:

```yaml
spring:
  cloud:
    openfeign:
      circuitbreaker:
        enabled: true
```

### Fallback de Cortacircuitos

```java
@FeignClient(name = "user-service", fallback = UserServiceFallback.class)
public interface UserService {
    @GetMapping("/users/{id}")
    User getUser(@PathVariable("id") Long id);
}

@Component
public class UserServiceFallback implements UserService {
    @Override
    public User getUser(Long id) {
        // Devolver usuario predeterminado o lanzar excepción específica
        return new User(id, "Default User");
    }
}
```

### Fábrica de Fallback con Información de Excepción

```java
@FeignClient(name = "user-service", fallbackFactory = UserServiceFallbackFactory.class)
public interface UserService {
    @GetMapping("/users/{id}")
    User getUser(@PathVariable("id") Long id);
}

@Component
public class UserServiceFallbackFactory implements FallbackFactory<UserService> {
    @Override
    public UserService create(Throwable cause) {
        return new UserService() {
            @Override
            public User getUser(Long id) {
                // Puede acceder a la excepción que desencadenó el fallback
                log.error("UserService fallback triggered", cause);
                return new User(id, "Fallback User");
            }
        };
    }
}
```

### Nombre de Cortacircuitos Personalizado

```java
@Configuration
public class CircuitBreakerConfig {
    @Bean
    public CircuitBreakerNameResolver circuitBreakerNameResolver() {
        return (String feignClientName, Target<?> target, Method method) -> 
            feignClientName + "_" + method.getName();
    }
}
```

## Registro

### Configurar Nivel de Registro

```java
@Configuration
public class FeignLoggingConfig {
    
    @Bean
    Logger.Level feignLoggerLevel() {
        return Logger.Level.FULL;
    }
}
```

O a través del archivo de configuración:

```yaml
logging:
  level:
    com.example.clients.UserService: DEBUG

spring:
  cloud:
    openfeign:
      client:
        config:
          user-service:
            logger-level: full
```

### Filtrado de Registro Personalizado

```java
public class CustomLogger extends Logger {
    
    @Override
    protected void logRequest(String configKey, Level logLevel, Request request) {
        // Filtrar información sensible
        if (!shouldLogRequestHeader(request)) {
            // Lógica de registro personalizada
        }
    }
    
    @Override
    protected Response logAndRebufferResponse(String configKey, Level logLevel, 
            Response response, long elapsedTime) throws IOException {
        // Filtrar información sensible de respuesta
        if (!shouldLogResponseHeader(response)) {
            // Lógica de registro personalizada
        }
        return super.logAndRebufferResponse(configKey, logLevel, response, elapsedTime);
    }
    
    private boolean shouldLogRequestHeader(Request request) {
        // Filtrar encabezado de Autorización y otra información sensible
        return !request.headers().containsKey("Authorization");
    }
    
    private boolean shouldLogResponseHeader(Response response) {
        return true; // O filtrar según sea necesario
    }
}
```

## Compresión de Solicitud/Respuesta

### Habilitar Compresión

```yaml
spring:
  cloud:
    openfeign:
      compression:
        request:
          enabled: true
          mime-types: text/xml,application/xml,application/json
          min-request-size: 2048
        response:
          enabled: true
```

### Notas Importantes

- Cuando se utiliza OkHttpClient, la compresión se deshabilita si los encabezados `content-encoding` o `accept-encoding` están presentes
- La compresión solo se aplica a los tipos MIME especificados
- El tamaño de la solicitud debe exceder `min-request-size` para ser comprimido

## Soporte OAuth2

### Habilitar OAuth2

```yaml
spring:
  cloud:
    openfeign:
      oauth2:
        enabled: true
        client-registration-id: my-client
```

### Configuración de Dependencias

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-oauth2-client</artifactId>
</dependency>
```

### Configuración de Cliente OAuth2

```yaml
spring:
  security:
    oauth2:
      client:
        registration:
          my-client:
            client-id: your-client-id
            client-secret: your-client-secret
            authorization-grant-type: client_credentials
            scope: read,write
        provider:
          my-provider:
            token-uri: https://auth-server.com/oauth/token
```

## Soporte de Caché

### Habilitar Caché

```java
@SpringBootApplication
@EnableCaching
@EnableFeignClients
public class Application {
    // ...
}
```

### Usar Caché en Cliente Feign

```java
@FeignClient(name = "data-service")
public interface DataService {
    
    @GetMapping("/data/{key}")
    @Cacheable(cacheNames = "data-cache", key = "#key")
    Data getData(@PathVariable String key);
    
    @PostMapping("/data")
    @CachePut(cacheNames = "data-cache", key = "#data.key")
    Data updateData(@RequestBody Data data);
    
    @DeleteMapping("/data/{key}")
    @CacheEvict(cacheNames = "data-cache", key = "#key")
    void deleteData(@PathVariable String key);
}
```

### Deshabilitar Caché

```yaml
spring:
  cloud:
    openfeign:
      cache:
        enabled: false
```

## Soporte Asíncrono

### Soporte CompletableFuture

```java
@FeignClient(name = "async-service")
public interface AsyncService {
    
    @GetMapping("/data/{id}")
    CompletableFuture<Data> getDataAsync(@PathVariable String id);
}
```

### Configurar Cliente Asíncrono

```java
@Configuration
public class AsyncFeignConfig {
    
    @Bean
    public AsyncClient asyncClient() {
        return new AsyncClient.Default();
    }
}
```

### Ejemplo de Uso

```java
@Service
public class AsyncProcessingService {
    
    private final AsyncService asyncService;
    
    public AsyncProcessingService(AsyncService asyncService) {
        this.asyncService = asyncService;
    }
    
    public void processData() {
        CompletableFuture<Data> future1 = asyncService.getDataAsync("1");
        CompletableFuture<Data> future2 = asyncService.getDataAsync("2");
        
        // Procesar en paralelo
        CompletableFuture.allOf(future1, future2).thenRun(() -> {
            try {
                Data data1 = future1.get();
                Data data2 = future2.get();
                // Procesar datos
            } catch (Exception e) {
                log.error("Error processing async data", e);
            }
        });
    }
}
```

## Soporte de Herencia

### Definición de Interfaz Base

```java
public interface BaseService<T, ID> {
    @GetMapping("/{id}")
    T findById(@PathVariable ID id);
    
    @GetMapping
    List<T> findAll();
    
    @PostMapping
    T save(@RequestBody T entity);
    
    @DeleteMapping("/{id}")
    void deleteById(@PathVariable ID id);
}
```

### Implementación de Servicio Concreto

```java
@FeignClient(name = "user-service")
public interface UserService extends BaseService<User, Long> {
    @GetMapping("/users/by-email/{email}")
    User findByEmail(@PathVariable String email);
}

@FeignClient(name = "order-service")
public interface OrderService extends BaseService<Order, Long> {
    @GetMapping("/orders/by-user/{userId}")
    List<Order> findByUserId(@PathVariable Long userId);
}
```

## Soporte de Ámbito de Actualización

### Habilitar Soporte de Actualización

```yaml
spring:
  cloud:
    openfeign:
      client:
        refresh-enabled: true
```

### Actualizar Configuración

Actualizar a través del endpoint de Actuator:

```bash
POST /actuator/refresh
```

### Notas Importantes

- No anote la interfaz `@FeignClient` con `@RefreshScope`
- La funcionalidad de actualización está deshabilitada por defecto, necesita habilitación explícita
- Soporta actualizar tiempo de espera de conexión, tiempo de espera de lectura y configuración de URL

## Soporte AOT e Imagen Nativa

### Requisitos de Configuración

```yaml
spring:
  cloud:
    refresh:
      enabled: false
    openfeign:
      client:
        refresh-enabled: false
      lazy-attributes-resolution: false
```

### Construir Imagen Nativa

Asegure la configuración correcta durante la construcción, y sobrescriba la URL en tiempo de ejecución mediante propiedades del sistema:

```bash
-Dspring.cloud.openfeign.client.config.[clientId].url=[url]
```

Estas funciones avanzadas hacen que OpenFeign no sea solo un cliente HTTP simple, sino una solución integral de comunicación de microservicios capaz de satisfacer diversos requisitos de aplicaciones empresariales.
