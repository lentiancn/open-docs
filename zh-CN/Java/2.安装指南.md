## 安装步骤

### Windows 安装

#### 方法一：Oracle JDK（推荐）

1. 下载 JDK：https://www.oracle.com/java/technologies/downloads/
2. 运行安装程序
3. 设置环境变量：
   - `JAVA_HOME` = JDK 安装路径（如 `C:\Program Files\Java\jdk-21`）
   - 添加 `%JAVA_HOME%\bin` 到 PATH
4. 验证安装：`java -version`

#### 方法二：OpenJDK

1. 下载 OpenJDK：https://adoptium.net/
2. 解压到指定目录
3. 配置环境变量（同上）

#### 方法三：Chocolatey

```powershell
choco install openjdk21
```

### macOS 安装

#### 方法一：brew（推荐）

```bash
brew install openjdk@21

# 添加到 PATH
echo 'export PATH="/usr/local/opt/openjdk@21/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

#### 方法二：DMG 安装

1. 下载 .dmg 文件
2. 拖拽到 Applications 文件夹

### Linux 安装

#### apt（Ubuntu/Debian）

```bash
sudo apt update
sudo apt install openjdk-21-jdk
```

#### yum（CentOS/RHEL）

```bash
sudo yum install java-21-openjdk java-21-openjdk-devel
```

#### 手动安装

```bash
# 下载
wget https://download.java.net/java/GA/jdk21.0.2/f2283984656d49d69e91c558476027ac/13/GPL/openjdk-21.0.2_linux-x64_bin.tar.gz

# 解压
sudo tar -C /usr/local -xzf openjdk-21.0.2_linux-x64_bin.tar.gz

# 设置环境变量
export JAVA_HOME=/usr/local/jdk-21.0.2
export PATH=$JAVA_HOME/bin:$PATH
```

## 环境配置

### Windows 环境变量

```
JAVA_HOME = C:\Program Files\Java\jdk-21
PATH = %JAVA_HOME%\bin;%PATH%
```

### 验证安装

```bash
# 查看版本
java -version

# 查看编译器版本
javac -version

# 查看 JAVA_HOME
echo %JAVA_HOME%
```

## 常见问题

### javac 命令找不到？

确保 JDK 的 bin 目录已添加到 PATH 环境变量。

### 多个 Java 版本如何切换？

Windows：修改 JAVA_HOME 环境变量
macOS/Linux：使用 update-alternatives
