# Guia de Uso de Vue 2

Guia para usar Vue 2 efectivamente.

---

## Conceptos basicos

Vue 2 es un framework JavaScript para construir interfaces de usuario.

---

## Instancia

```javascript
new Vue({
  el: '#app',
  data: {
    message: 'Hello Vue!'
  }
})
```

---

## Directivas

| Directiva | Descripcion |
|----------|-------------|
| `v-if` | Renderizado condicional |
| `v-for` | Renderizado de listas |
| `v-bind` | Enlace de atributos |
| `v-model` | Enlace bidireccional |

---

## Componentes

```vue
<template>
  <div>{{ title }}</div>
</template>

<script>
export default {
  props: {
    title: String
  }
}
</script>
```
