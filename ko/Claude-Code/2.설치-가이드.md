# Claude Code 설치 가이드

이 가이드에서는 다양한 플랫폼에서 Claude Code를 설치, 구성, 업데이트하는 자세한 지침을 제공합니다.

## 시스템 요구사항

Claude Code는 다음 운영 체제 및 구성을 지원합니다:

| 구성요소 | 요구사항 |
|---------|---------|
| 운영 체제 | macOS 13.0+, Windows 10 1809+ 또는 Windows Server 2019+, Ubuntu 20.04+, Debian 10+, Alpine Linux 3.19+ |
| 메모리 | 4GB 이상 |
| 네트워크 | 인터넷 연결 필요 |
| 셸 | Bash, Zsh, PowerShell 또는 CMD |
| 지역 | Anthropic이 지원하는 국가 및 지역 |

### 추가 종속성

- **ripgrep**: 일반적으로 Claude Code에 포함되어 있습니다; 검색 실패 시 문제 해결 섹션을 참조하세요.

## 설치 방법

### 방법 1: 네이티브 설치 (권장)

**macOS, Linux, WSL:**

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell:**

```powershell
irm https://claude.ai/install.ps1 | iex
```

**Windows CMD:**

```batch
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
```

**참고: Windows에서는 먼저 [Git for Windows](https://git-scm.com/downloads/win)를 설치해야 합니다.**

네이티브 설치는 백그라운드에서 자동으로 업데이트됩니다.

### 방법 2: Homebrew (macOS/Linux)

```bash
brew install --cask claude-code
```

Homebrew 설치는 자동 업데이트되지 않습니다. 최신 기능과 보안 수정사항을 얻으려면 주기적으로 `brew upgrade claude-code`를 실행하세요.

### 방법 3: WinGet (Windows)

```powershell
winget install Anthropic.ClaudeCode
```

WinGet 설치는 자동 업데이트되지 않습니다. 수동 업데이트를 위해 주기적으로 `winget upgrade Anthropic.ClaudeCode`를 실행하세요.

## 설치 확인

설치 후 Claude Code가 작동하는지 확인합니다:

```bash
claude --version
```

보다 자세한 설치 및 구성 확인을 위해 다음을 실행합니다:

```bash
claude doctor
```

## 계정 인증

Claude Code는 Pro, Max, Teams, Enterprise 또는 Console 계정이 필요합니다. 무료 Claude.ai 플랜에는 Claude Code 접근 권한이 포함되지 않습니다.

설치 후 `claude` 명령을 실행하고 브라우저 프롬프트에 따라 로그인하세요.

## Claude Code 업데이트

### 자동 업데이트

네이티브 설치는 시작 시 그리고 실행 중 주기적으로 업데이트를 확인합니다. 업데이트는 백그라운드에서 다운로드 및 설치되며, 다음에 Claude Code를 시작할 때 적용됩니다.

릴리스 채널을 구성하여 업데이트 타이밍을 제어할 수 있습니다:
- `latest` (기본값): 즉시 새 기능 수신
- `stable`: 일반적으로 약 1주된 버전을 사용하며 주요 문제가 있는 릴리스 건너뛰기

설정 파일에서 구성합니다:

```json
{
  "autoUpdatesChannel": "stable"
}
```

### 자동 업데이트 비활성화

설정 파일에서 환경 변수를 설정합니다:

```json
{
  "env": {
    "DISABLE_AUTOUPDATER": "1"
  }
}
```

### 수동 업데이트

백그라운드 검사를 기다리지 않고 즉시 업데이트를 적용합니다:

```bash
claude update
```

Homebrew 및 WinGet 설치는 수동 업데이트가 필요합니다:
- Homebrew: `brew upgrade claude-code`
- WinGet: `winget upgrade Anthropic.ClaudeCode`

## 고급 설치 옵션

### 특정 버전 설치

최신 버전 설치:
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

안정 버전 설치:
```bash
curl -fsSL https://claude.ai/install.sh | bash -s stable
```

특정 버전 설치 (예: 1.0.58):
```bash
curl -fsSL https://claude.ai/install.sh | bash -s 1.0.58
```

### npm에서 네이티브 설치로 마이그레이션

이전 npm으로 Claude Code를 설치했다면 네이티브 설치로 마이그레이션합니다:

```bash
# 네이티브 바이너리 설치
curl -fsSL https://claude.ai/install.sh | bash

# 이전 npm 설치 제거
npm uninstall -g @anthropic-ai/claude-code
```

### 더 이상 사용되지 않는 npm 설치 방법

npm 설치는 더 이상 사용되지 않습니다. npm 설치가 정말 필요한 경우, 먼저 Node.js 18+가 설치되어 있어야 합니다:

```bash
npm install -g @anthropic-ai/claude-code
```

**참고: `sudo npm install -g`를 사용하지 마세요. 권한 문제와 보안 위험이 발생할 수 있습니다.**

## Claude Code 제거

### 네이티브 설치 제거

**macOS, Linux, WSL:**

```bash
rm -f ~/.local/bin/claude
rm -rf ~/.local/share/claude
```

**Windows PowerShell:**

```powershell
Remove-Item -Path "$env:USERPROFILE\.local\bin\claude.exe" -Force
Remove-Item -Path "$env:USERPROFILE\.local\share\claude" -Recurse -Force
```

### Homebrew 제거

```bash
brew uninstall --cask claude-code
```

### WinGet 제거

```powershell
winget uninstall Anthropic.ClaudeCode
```

### npm 제거

```bash
npm uninstall -g @anthropic-ai/claude-code
```

### 구성 파일 제거

**경고: 구성 파일을 제거하면 모든 설정, 허용된 도구, MCP 서버 구성, 세션 기록이 삭제됩니다.**

**macOS, Linux, WSL:**

```bash
# 사용자 설정 및 상태 제거
rm -rf ~/.claude
rm ~/.claude.json

# 프로젝트별 설정 제거 (프로젝트 디렉토리에서 실행)
rm -rf .claude
rm -f .mcp.json
```

**Windows PowerShell:**

```powershell
# 사용자 설정 및 상태 제거
Remove-Item -Path "$env:USERPROFILE\.claude" -Recurse -Force
Remove-Item -Path "$env:USERPROFILE\.claude.json" -Force

# 프로젝트별 설정 제거
Remove-Item -Path ".claude" -Recurse -Force
Remove-Item -Path ".mcp.json" -Force
```

## Windows 특수 구성

Windows에서 Claude Code는 Git for Windows 또는 WSL이 필요합니다. PowerShell, CMD 또는 Git Bash에서 `claude`를 실행할 수 있습니다.

Claude Code가 Git Bash 설치를 찾을 수 없는 경우 설정 파일에서 경로를 구성합니다:

```json
{
  "env": {
    "CLAUDE_CODE_GIT_BASH_PATH": "C:\\Program Files\\Git\\bin\\bash.exe"
  }
}
```

## Alpine Linux 구성

Alpine 및 기타 musl/uClibc 배포판에서 네이티브 설치 프로그램에는 `libgcc`, `libstdc++`, `ripgrep`이 필요합니다. 패키지 관리자를 사용하여 이러한 종속성을 설치한 다음 `USE_BUILTIN_RIPGREP=0`을 설정합니다.

Alpine에서 필요한 패키지 설치:

```bash
apk add libgcc libstdc++ ripgrep
```

然后设置文件:

```json
{
  "env": {
    "USE_BUILTIN_RIPGREP": "0"
  }
}
```

## 바이너리 무결성 확인

SHA256 체크섬을 사용하여 Claude Code 바이너리의 무결성을 확인할 수 있습니다. 체크섬은 다음에서 공개되는 릴리스 매니페스트에 있습니다:
`https://storage.googleapis.com/claude-code-dist-86c565f3-f756-42ad-8dfa-d59b1c096819/claude-code-releases/{VERSION}/manifest.json`

서명된 바이너리는 다음에 배포됩니다:
- **macOS**: "Anthropic PBC"가 서명하고 Apple이 공증
- **Windows**: "Anthropic, PBC"가 서명
