# Introduction à IBM Informix

## Qu'est-ce qu'IBM Informix ?

IBM Informix est une famille de systèmes de gestion de bases de données relationnelles (SGBDR) et d'offres de bases de données multi-modèles développé à l'origine par Informix Corporation et acquis par IBM en 2001. En 2017, IBM a délégué le développement actif et le support à HCLSoftware, tout en continuant à le commercialiser sous le nom d'IBM Informix.

Informix est optimisé pour les environnements avec une administration de base de données très faible ou nulle, y compris son utilisation comme base de données embarquée. Il a une longue histoire de soutien à des taux de transaction très élevés et fournit les caractéristiques de disponibilité nécessaires aux applications critiques.

## Caractéristiques Principales

- **Base de données multi-modèles** : Prend en charge les capacités relationnelles et orientées objet pour modéliser des structures de données complexes
- **Haute performance** : Optimisé pour les transactions always-on et l'analytique en temps réel
- **Empreinte kleine** : Peut fonctionner sur des appareils aussi compacts que 100 Mo (ex. Raspberry Pi)
- **Edge Computing** : Idéal pour les solutions IoT avec des capacités de base de données embarquée
- **Support des séries temporelles** : Support intégré pour les données de séries temporelles
- **Compatibilité JSON/MongoDB** : Intégration avec les collections JSON et les pilotes JSON MongoDB
- **Haute disponibilité** : Fonctionnalités comme HDR (High Availability Data Replication), RSS et SDS

## Éditions du Produit

À partir de 2025-2026, IBM Informix propose les éditions suivantes :

| Édition | Utilisateurs Cibles | Limites | Fonctionnalités Clés |
|---------|-------------------|---------|---------------------|
| **Developer Edition** | Développeurs individuels | 1 CPU VP/Core, 1Go RAM, 8Go stockage | Gratuit pour développement/tests |
| **Innovator-C Edition** | Petites entreprises | 2 CPU Cores, 8Go RAM, 50Go stockage | Gratuit pour développement/production limitée |
| **Express Edition** | Développeurs tiers | 4 CPU Cores, 8Go RAM | Base de données intégrable |
| **Workgroup Edition** | Entreprises moyennes | Max 24 CPU Cores, 32Go RAM | Toutes les options de réplication |
| **Enterprise Edition** | Grandes entreprises | CPU/RAM illimités | Toutes les fonctionnalités sauf IWA |
| **Advanced Enterprise Edition** | Enterprise + BI | Illimité | Suite complète incluant IWA |
