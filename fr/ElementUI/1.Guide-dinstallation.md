# Element UI Guide d'installation

> Element UI est une bibliothèque de composants UI de bureau basée sur Vue 2
> 
> Mis à jour : Février 2026
> 
> Version actuelle : 2.x (Stable)
> 
> Note : Pour les projets Vue 3, utilisez Element Plus

---

## 1. Présentation du projet

Element UI est une bibliothèque de composants UI de bureau basée sur Vue 2.0, développée et maintenue par l'équipe Ele.me. Elle fournit un ensemble riche de composants couramment utilisés pour PC, notamment les formulaires, tableaux, dialogues, menus de navigation, boutons, champs de saisie et autres éléments d'interface.

Element UI a un style de conception élégant avec une documentation détaillée. C'est l'une des bibliothèques de composants Vue 2 les plus populaires en Chine.

### 1.1 Informations de version

| Version | Version Vue | Statut | Description |
|---------|-------------|--------|--------------|
| Element UI 2.x | Vue 2 | Stable | Version principale |
| Element UI 1.x | Vue 2 | Arrêtée | Ancienne version |

### 1.2 Différence avec Element Plus

| Fonction | Element UI | Element Plus |
|----------|------------|--------------|
| Version Vue | Vue 2 | Vue 3 |
| TypeScript | Support partiel | Support complet |
| Composition API | Non supporté | Supporté |
| Maintenance | Arrêtée | Active |

## 3. Étapes d'installation

### 3.1 Avec npm (Recommandé)

```bash
# Accédez au répertoire du projet Vue 2
cd votre-projet-vue

# Installez Element UI
npm install element-ui --save
```

### 3.2 Avec yarn

```bash
yarn add element-ui
```

---

## 4. Importation dans le projet Vue 2

### 4.1 Importation complète

Importez Element UI et ses styles dans le fichier principal (habituellement main.js) :

```javascript
import Vue from 'vue'
import App from './App.vue'

import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)

new Vue({
  render: h => h(App)
}).$mount('#app')
```

### 4.2 Importation à la demande (Recommandée)

Pour réduire la taille du bundle, importez uniquement les composants nécessaires :

```javascript
import { Button, Form, FormItem, Input } from 'element-ui'

Vue.use(Button)
Vue.use(Form)
Vue.use(FormItem)
Vue.use(Input)
```

## 5. Exemples de démarrage rapide

### 5.1 Bouton de base

```html
<el-button>Par défaut</el-button>
<el-button type="primary">Primaire</el-button>
<el-button type="success">Succès</el-button>
<el-button type="warning">Avertissement</el-button>
<el-button type="danger">Danger</el-button>
```

### 5.2 Formulaire de base

```html
<el-form :model="form" label-width="80px">
  <el-form-item label="Nom d'utilisateur">
    <el-input v-model="form.username"></el-input>
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="submit">Soumettre</el-button>
  </el-form-item>
</el-form>
```

### 5.3 Tableau de base

```html
<el-table :data="tableData" stripe border>
  <el-table-column prop="name" label="Nom"></el-table-column>
  <el-table-column prop="email" label="Email"></el-table-column>
</el-table>
```
