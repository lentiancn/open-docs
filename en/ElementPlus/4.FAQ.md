# Element Plus FAQ

## Installation

### Installation Failed

Ensure Node.js >= 14.0.0

### TypeScript Errors

Configure tsconfig.json:
```json
{ "compilerOptions": { "types": ["element-plus/global"] } }
```

## Usage

### Styles Not Applied

Import CSS:
```javascript
import 'element-plus/dist/index.css'
```

### Icons Not Displayed

Install icon library:
```bash
npm install @element-plus/icons-vue
```

### On-Demand Import Not Working

Configure plugins correctly in vite.config.ts

## Performance

### Bundle Size Too Large

Use on-demand import
