# Angular Guide d'installation

Guide détaillé pour configurer l'environnement Angular.

## Prérequis

### Node.js

- Recommandé : Node.js 18+
- Nécessite npm ou yarn

### Vérifier la version de Node.js

```bash
node --version
npm --version
```

## Installer Angular CLI

### Installer

```bash
npm install -g @angular/cli
```

### Vérifier l'installation

```bash
ng version
```

### Mettre à jour

```bash
npm install -g @angular/cli@latest
```

## Créer un nouveau projet

### Utiliser Angular CLI

```bash
ng new my-app
cd my-app
```

### Options interactives

- Ajouter le routage ? Oui/Non
- Quel format de feuille de styles ? (SCSS/CSS/Sass/Less)

## Exécuter l'application

### Serveur de développement

```bash
ng serve
# ou
npm start
```

Visiter : http://localhost:4200

### Rechargement à chaud

Angular CLI supporte le rechargement à chaud — les modifications sont automatiquement refresh.

## Builder pour la production

```bash
ng build --configuration production
```

Sortie dans le répertoire `dist/`.

## Commandes courantes

| Commande | Fonction |
|----------|----------|
| ng new | Créer un nouveau projet |
| ng serve | Démarrer le serveur de dev |
| ng build | Builder le projet |
| ng generate | Générer composant/service/module |
| ng test | Exécuter les tests unitaires |
| ng e2e | Exécuter les tests e2e |

## Structure du projet

```
my-app/
├── src/
│   ├── app/
│   │   ├── components/
│   │   ├── services/
│   │   ├── app.component.ts
│   │   └── app.module.ts
│   ├── assets/
│   └── styles.css
├── angular.json
└── package.json
```

## Désinstaller Angular CLI

```bash
npm uninstall -g @angular/cli
```
