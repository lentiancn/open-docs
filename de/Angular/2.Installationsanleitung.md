# Installationsanleitung

Dieses Dokument beschreibt die Installation und Einrichtung von Angular, damit du schnell mit der Entwicklung loslegen kannst.

## Voraussetzungen

Bevor du Angular installierst, stelle sicher, dass folgende Voraussetzungen erfüllt sind:

- **Node.js:** Empfohlen wird Version 18.10.0 oder höher
- **npm:** Wird mit Node.js geliefert, Version 6.x oder höher
- **Betriebssystem:** Windows, macOS oder Linux

## Angular CLI installieren

Angular CLI ist das offizielle Kommandozeilen-Tool von Angular zum Erstellen und Verwalten von Angular-Projekten.

### Globale Installation

```bash
npm install -g @angular/cli
```

### Installation überprüfen

```bash
ng version
```

Bei erfolgreicher Installation wird die Angular-CLI-Versionsinformation angezeigt.

### Bestimmte Version installieren

Falls du eine spezifische Version benötigst:

```bash
npm install -g @angular/cli@17
```

## Neues Projekt erstellen

### Mit CLI erstellen

```bash
ng new my-app
```

Dies erstellt ein neues Projekt namens my-app und installiert automatisch die Abhängigkeiten.

### Optionen beim Erstellen angeben

```bash
# Erstellen ohne Installation
ng new my-app --skip-install

# SCSS verwenden
ng new my-app --style=scss

# Routing aktivieren
ng new my-app --routing

# Standalone-Komponenten (Angular 14+)
ng new my-app --standalone
```

## Entwicklungsumgebung einrichten

### Chinesische Spiegelserver verwenden

Aufgrund von Netzwerkbeschränkungen empfiehlt es sich, einen chinesischen Spiegelserver zu konfigurieren:

```bash
# npm-Spiegel einrichten
npm config set registry https://registry.npmmirror.com

# Oder nrm zur Spiegelverwaltung nutzen
npm install -g nrm
nrm use taobao
```

### Node.js installieren

**Installation mit nvm (empfohlen):**

```bash
# nvm installieren
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Node.js LTS-Version installieren
nvm install --lts
nvm use --lts
```

**Windows: nvm-windows verwenden:**

Lade das Installationspaket von https://github.com/coreybutler/nvm-windows herunter.

## Entwicklungsserver starten

### Ins Projektverzeichnis wechseln

```bash
cd my-app
```

### Entwicklungsserver starten

```bash
ng serve
```

Standard-Adresse: http://localhost:4200

### Live-Reload

Angular CLI unterstützt Live-Reload – der Browser aktualisiert sich automatisch nach Codeänderungen.

## Produktions-Build erstellen

### Build-Befehl

```bash
ng build
```

Die Build-Ausgabe befindet sich im Verzeichnis dist.

### Produktionsoptimierung

```bash
ng build --configuration=production
```

Dies aktiviert Code-Minifizierung, Tree-Shaking und weitere Optimierungen.

## Häufige CLI-Befehle

| Befehl | Beschreibung |
|--------|--------------|
| ng generate component xxx | Komponente erstellen |
| ng generate service xxx | Service erstellen |
| ng generate module xxx | Modul erstellen |
| ng generate directive xxx | Direktive erstellen |
| ng generate pipe xxx | Pipe erstellen |
| ng generate guard xxx | Route-Guard erstellen |

## Entwickler-Tools

### VS-Code-Erweiterungen

- **Angular Language Service:** Bietet Typprüfung und Autovervollständigung in Templates
- **Angular Snippets:** Häufig verwendete Code-Schnipsel
- **Prettier:** Code-Formatierung

### Debugging

Angular DevTools ist eine Browser-Erweiterung zum Debuggen und Analysieren der Performance.

## Angular CLI deinstallieren

Falls erforderlich:

```bash
npm uninstall -g @angular/cli
```
