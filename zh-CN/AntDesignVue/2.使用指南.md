# Ant Design Vue 3 使用指南

高效使用 Ant Design Vue 3 的完整指南。

---

## 基础组件

### 按钮

```vue
<a-button>默认</a-button>
<a-button type="primary">主要</a-button>
<a-button type="dashed">虚线</a-button>
<a-button type="text">文本</a-button>

<!-- 图标 -->
<a-button icon="search">搜索</a-button>
```

### 布局

```vue
<a-layout>
  <a-layout-header>头部</a-layout-header>
  <a-layout-content>内容</a-layout-content>
  <a-layout-footer>底部</a-layout-footer>
</a-layout>

<!-- 栅格 -->
<a-row :gutter="16">
  <a-col :span="12">12列</a-col>
  <a-col :span="12">12列</a-col>
</a-row>
```

---

## 表单组件

### 输入框

```vue
<a-input v-model:value="value" placeholder="请输入" />
```

### 选择器

```vue
<a-select v-model:value="value" placeholder="请选择">
  <a-select-option value="jack">Jack</a-select-option>
  <a-select-option value="lucy">Lucy</a-select-option>
</a-select>
```

### 表单

```vue
<a-form :model="form" layout="vertical">
  <a-form-item label="名称">
    <a-input v-model:value="form.name" />
  </a-form-item>
  <a-form-item>
    <a-button type="primary" @click="onSubmit">提交</a-button>
  </a-form-item>
</a-form>
```

---

## 数据展示

### 表格

```vue
<a-table :columns="columns" :data-source="data" :pagination="false" />
```

### 标签

```vue
<a-tag>标签</a-tag>
<a-tag color="green">绿色</a-tag>
<a-tag color="blue">蓝色</a-tag>
<a-tag color="red">红色</a-tag>
```

### 卡片

```vue
<a-card title="卡片标题">
  <p>卡片内容</p>
</a-card>
```

---

## 导航

### 菜单

```vue
<a-menu mode="horizontal">
  <a-menu-item key="mail">导航一</a-menu-item>
  <a-menu-item key="app">导航二</a-menu-item>
</a-menu>
```

### 标签页

```vue
<a-tabs v-model:activeKey="activeKey">
  <a-tab-pane key="1" tab="标签1">内容1</a-tab-pane>
  <a-tab-pane key="2" tab="标签2">内容2</a-tab-pane>
</a-tabs>
```

---

## 反馈

### 消息

```vue
<script setup>
import { message } from 'ant-design-vue'

message.success('成功')
message.error('错误')
</script>
```

### 弹窗

```vue
<a-button @click="showModal = true">打开弹窗</a-button>

<a-modal v-model:open="showModal" title="标题">
  <p>内容</p>
</a-modal>
```
