# TensorFlow Häufig Gestellte Fragen

## Allgemeine Fragen

### Was ist TensorFlow?
TensorFlow ist ein Open-Source-Machine-Learning-Framework, das von Google entwickelt wurde. Es bietet Tools für den Aufbau und das Training von neuronalen Netzen.

### Welche Programmiersprachen unterstützt TensorFlow?
- Python (primär)
- JavaScript
- Java
- C++
- Swift
- Go

### Was ist der Unterschied zwischen PyTorch und TensorFlow?
| Aspekt | TensorFlow | PyTorch |
|--------|------------|---------|
| Dynamische Graphen | Begrenzt (TF 2.x) | Nativ |
| Debugging | Schwieriger | Einfacher |
| Produktion | TensorFlow Serving | TorchServe |
| Community | Größer | Schnell wachsend |

## Installationsfragen

### Wie installiere ich TensorFlow?
```bash
# CPU-Version
pip install tensorflow

# GPU-Version
pip install tensorflow-gpu
```

### Was sind die Systemanforderungen?
- **Python**: 3.7-3.11
- **OS**: Linux, macOS, Windows
- **GPU**: NVIDIA GPU mit CUDA (für GPU-Version)
- **RAM**: 8GB+ empfohlen

### Wie verifiziere ich die Installation?
```python
import tensorflow as tf
print(tf.__version__)
print("GPU Available:", tf.config.list_physical_devices('GPU'))
```

## Nutzungsfragen

### Wie erstelle ich ein einfaches neuronales Netz?
```python
import tensorflow as tf
from tensorflow import keras

# Modell erstellen
model = keras.Sequential([
    keras.layers.Flatten(input_shape=(28, 28)),
    keras.layers.Dense(128, activation='relu'),
    keras.layers.Dense(10, activation='softmax')
])

# Kompilieren
model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

# Trainieren
model.fit(x_train, y_train, epochs=5)
```

### Wie speichere und lade ich Modelle?
```python
# Speichern
model.save('my_model.keras')

# Laden
model = keras.models.load_model('my_model.keras')
```

## GPU-Fragen

### Wie aktiviere ich GPU-Unterstützung?
```python
gpus = tf.config.list_physical_devices('GPU')
if gpus:
    for gpu in gpus:
        tf.config.experimental.set_memory_growth(gpu, True)
```

### Was, wenn ich keine GPU habe?
Verwenden Sie die CPU-Version von TensorFlow. Das Training ist langsamer, funktioniert aber auf jedem Rechner.

## Fehlerbehebung

### Out-of-Memory-Fehler
- Verringern Sie die Batch-Größe
- Nutzen Sie `set_memory_growth`
- Löschen Sie Keras-Session

### Langsames Training
- GPU-Beschleunigung aktivieren
- Mixed-Precision-Training verwenden
- XLA-Kompilierung aktivieren
