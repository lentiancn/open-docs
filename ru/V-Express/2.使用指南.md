# V-Express Руководство по использованию

## Базовая команда

```bash
python inference.py \
  --reference_image_path "./ref.jpg" \
  --audio_path "./audio.mp3" \
  --output_path "./output.mp4" \
  --retarget_strategy "no_retarget"
```

## Параметры

| Параметр | Описание |
|-----------|-----------|
| `--reference_image_path` | Путь к изображению |
| `--audio_path` | Путь к аудио |
| `--output_path` | Путь к видео |
