# Руководство пользователя Nginx

## Основные операции

```bash
nginx              # Запуск
nginx -s stop     # Остановка
nginx -s reload    # Перезагрузка конфигурации
nginx -t           # Проверка конфигурации
```

## Базовая конфигурация

```nginx
worker_processes auto;
events {
    worker_connections 1024;
}
http {
    server {
        listen 80;
        server_name example.ru;
        root /var/www/html;
        location / {
            index index.html;
        }
    }
}
```

## Статические файлы

```nginx
server {
    listen 80;
    root /var/www/html;
    location / {
        try_files $uri $uri/ =404;
    }
}
```

## Обратный прокси

```nginx
location / {
    proxy_pass http://backend;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
}
```

## Балансировка нагрузки

```nginx
upstream backend {
    server backend1.example.ru;
    server backend2.example.ru;
}
```

## SSL/TLS

```nginx
server {
    listen 443 ssl;
    ssl_certificate cert.crt;
    ssl_certificate_key cert.key;
    ssl_protocols TLSv1.2 TLSv1.3;
}
```

## Часто используемые директивы

- `root`: Корневой каталог документов
- `index`: Файлы по умолчанию
- `proxy_pass`: Бэкенд прокси
- `upstream`: Группа серверов
- `gzip`: Включить сжатие
- `expires`: Кэш браузера
