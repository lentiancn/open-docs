## Быстрый старт

### Подключение к MySQL

```bash
mysql -u root -p
```

### Создание базы данных

```sql
CREATE DATABASE моёприложение;
USE моёприложение;
```

### Создание таблицы

```sql
CREATE TABLE пользователи (
    id INT AUTO_INCREMENT PRIMARY KEY,
    имя VARCHAR(100) NOT NULL,
    email VARCHAR(100) UNIQUE
);
```

### Вставка данных

```sql
INSERT INTO пользователи (имя, email) VALUES ('Иван', 'ivan@пример.рф');
```

### Запрос данных

```sql
SELECT * FROM пользователи;
```
