# Angular Руководство пользователя

Основные концепции и базовое использование Angular.

## Компоненты

### Создать компонент

```bash
ng generate component my-component
# или
ng g c my-component
```

### Структура компонента

```typescript
import { Component } from '@angular/core';

@Component({
  selector: 'app-my-component',
  templateUrl: './my-component.component.html',
  styleUrls: ['./my-component.component.css']
})
export class MyComponentComponent {
  title = 'Hello Angular';
}
```

## Привязка данных

### Привязка свойств

```html
<img [src]="imageUrl">
```

### Привязка событий

```html
<button (click)="onClick()">Клик</button>
```

### Двусторонняя привязка

```html
<input [(ngModel)]="name">
```

## Сервисы

### Создать сервис

```bash
ng generate service my-service
# или
ng g s my-service
```

### Использование сервисов

```typescript
import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root'
})
export class MyServiceService {
  getData() {
    return 'данные';
  }
}
```

## Роутинг

### Настройка маршрутов

```typescript
const routes: Routes = [
  { path: '', component: HomeComponent },
  { path: 'about', component: AboutComponent },
  { path: '**', component: NotFoundComponent }
];
```

### Навигация

```html
<a routerLink="/about">О нас</a>
<router-outlet></router-outlet>
```

## HTTP-запросы

### Импорт HttpClient

```typescript
import { HttpClientModule } from '@angular/common/http';
```

### Отправка запросов

```typescript
import { HttpClient } from '@angular/common/http';

constructor(private http: HttpClient) {}

getData() {
  return this.http.get('https://api.example.com/data');
}
```

## Формы

### Реактивные формы

```typescript
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

form: FormGroup;

constructor(private fb: FormBuilder) {
  this.form = this.fb.group({
    name: ['', Validators.required],
    email: ['', [Validators.required, Validators.email]]
  });
}
```

## Хуки жизненного цикла

| Хук | Описание |
|-----|----------|
| ngOnInit | Инициализация компонента |
| ngOnChanges | Изменение входных свойств |
| ngOnDestroy | Уничтожение компонента |

## Директивы

### Структурные директивы

```html
<div *ngIf="show">Показать</div>
<div *ngFor="let item of items">{{item}}</div>
```

### Атрибутные директивы

```html
<div [ngClass]="{'active': isActive}"></div>
```

## Лучшие практики

1. Используйте строгую типизацию TypeScript
2. Компоненты должны быть простыми
3. Используйте сервисы для общей логики
4. Отписывайтесь от Observable
5. Используйте стратегию обнаружения изменений OnPush
