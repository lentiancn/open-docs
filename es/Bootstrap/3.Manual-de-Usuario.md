# Bootstrap Manual de usuario

Funciones principales y consejos de uso.

## Sistema de rejilla

### Básico

```html
<div class="container">
  <div class="row">
    <div class="col">Columna</div>
  </div>
</div>
```

### Ancho de columna

```html
<div class="row">
  <div class="col-4">col-4</div>
  <div class="col-4">col-4</div>
  <div class="col-4">col-4</div>
</div>
```

## Componentes

### Navbar

```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <a class="navbar-brand" href="#">Marca</a>
  <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav">
    <span class="navbar-toggler-icon"></span>
  </button>
  <div class="collapse navbar-collapse" id="navbarNav">
    <ul class="navbar-nav">
      <li class="nav-item">
        <a class="nav-link" href="#">Inicio</a>
      </li>
    </ul>
  </div>
</nav>
```

### Tarjetas

```html
<div class="card" style="width: 18rem;">
  <div class="card-body">
    <h5 class="card-title">Título</h5>
    <p class="card-text">Contenido</p>
    <a href="#" class="btn btn-primary">Botón</a>
  </div>
</div>
```

### Modales

```html
<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#miModal">
  Abrir modal
</button>

<div class="modal fade" id="miModal">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title">Título</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
      </div>
      <div class="modal-body">Contenido</div>
    </div>
  </div>
</div>
```

## Clases utilitarias

### Espaciado

```html
<div class="m-4">Margen</div>
<div class="p-3">Relleno</div>
```

### Colores

```html
<div class="text-primary">Texto primario</div>
<div class="bg-dark text-white">Fondo oscuro</div>
```

## Formularios

```html
<form>
  <div class="mb-3">
    <label class="form-label">Email</label>
    <input type="email" class="form-control">
  </div>
  <button type="submit" class="btn btn-primary">Enviar</button>
</form>
```

## Mejores prácticas

1. Usar HTML semántico
2. Usar el sistema de rejilla correctamente
3. Personalizar con variables Sass
