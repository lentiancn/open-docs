# OpenCode Übersicht

Open-Source-KI-Coding-Agent.

## Was ist OpenCode?

OpenCode ist ein Open-Source-KI-Coding-Agent, der als Terminal-Interface, Desktop-App oder IDE-Erweiterung verfügbar ist.

### Hauptfunktionen

- **LSP aktiviert** - Lädt automatisch die richtigen LSPs
- **Multi-Sitzung** - Starte mehrere Agenten parallel im selben Projekt
- **Links teilen** - Teile jeden Sitzungslink
- **GitHub Copilot** - Melde dich mit GitHub an, um Copilot zu nutzen
- **ChatGPT Plus/Pro** - Melde dich mit OpenAI an, um ChatGPT zu nutzen
- **Beliebiges Modell** - 75+ LLM-Anbieter inklusive lokaler Modelle
- **Beliebiger Editor** - Terminal, Desktop-App und IDE-Erweiterung
- **Datenschutz zuerst** - Speichert keinen Code oder Kontextdaten
