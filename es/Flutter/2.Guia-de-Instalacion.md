# Flutter - Guía de Instalación

Guía de configuración del entorno Flutter.

## Requisitos

### Windows

- OS: Windows 10+ (64-bit)
- Espacio: 2.8GB+

### macOS

- OS: macOS 10.14+

### Linux

- OS: Ubuntu 18.04+

## Instalación

### Windows

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

Agregar al PATH.

### macOS

```bash
git clone https://github.com/flutter/flutter.git -b stable
export PATH="$PATH:/Users/usuario/flutter/bin"
```

### Linux

```bash
sudo apt-get install -y curl git unzip xz-utils
git clone https://github.com/flutter/flutter.git -b stable
export PATH="$PATH:$HOME/flutter/bin"
```

## Verificación

```bash
flutter doctor
flutter --version
```

## Android SDK

```bash
flutter config --android-sdk /ruta/a/android-sdk
```
