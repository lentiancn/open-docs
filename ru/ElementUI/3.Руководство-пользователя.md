# Element UI Руководство пользователя

## Базовые компоненты

### Button

```html
<el-button>Кнопка</el-button>
<el-button type="primary">Основная</el-button>
<el-button type="success">Успех</el-button>
<el-button type="warning">Предупреждение</el-button>
<el-button type="danger">Опасность</el-button>
```

### Layout

```html
<el-row>
  <el-col :span="12"><div></div></el-col>
  <el-col :span="12"><div></div></el-col>
</el-row>
```

## Формы

### Input

```html
<el-input v-model="input" placeholder="Ввод"></el-input>
```

### Select

```html
<el-select v-model="value" placeholder="Выбрать">
  <el-option label="Опция 1" value="1"></el-option>
</el-select>
```

### Form

```html
<el-form :model="form">
  <el-form-item label="Имя">
    <el-input v-model="form.name"></el-input>
  </el-form-item>
</el-form>
```

## Данные

### Table

```html
<el-table :data="tableData">
  <el-table-column prop="date" label="Дата"></el-table-column>
  <el-table-column prop="name" label="Имя"></el-table-column>
</el-table>
```

### Pagination

```html
<el-pagination :total="1000"></el-pagination>
```

## Уведомления

### Message

```javascript
this.$message({ message: 'Успех', type: 'success' })
```

### Alert

```html
<el-alert title="Успех" type="success"></el-alert>
```

## Навигация

### Menu

```html
<el-menu>
  <el-menu-item index="1">Меню 1</el-menu-item>
</el-menu>
```

### Tabs

```html
<el-tabs v-model="activeName">
  <el-tab-pane label="Вкладка 1" name="first">Содержимое</el-tab-pane>
</el-tabs>
```

## Другие

### Dialog

```html
<el-button @click="visible = true">Открыть</el-button>
<el-dialog :visible.sync="visible" title="Заголовок">
  <span>Содержимое</span>
</el-dialog>
```
