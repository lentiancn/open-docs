# Element UI 简介

Element UI 是饿了么团队开发的一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库。

## 什么是 Element UI

Element UI 是一套为 Vue 2 设计的桌面端 UI 组件库，提供了丰富的组件和交互，用于快速构建企业级应用后台管理系统。

作为 Vue 生态中最流行的 UI 框架之一，Element UI 以其一致性、完整性和易用性著称。

## 主要特性

### 完善的组件体系

Element UI 提供了 70 多个组件，涵盖桌面应用开发的各个方面：

- **基础组件**：按钮、图标、布局、容器
- **表单组件**：输入框、选择器、日期选择器、开关
- **数据展示**：表格、树形控件、分页、标签
- **反馈组件**：对话框、抽屉、消息提示、加载
- **导航组件**：菜单、标签页、面包屑、步骤条

### Vue 2 深度集成

- 完整支持 Vue 2
- 支持 Vue Router
- 支持 Vuex
- 良好的 TypeScript 支持

### 按需加载

支持按需引入组件，减少项目体积：

- 使用 babel-plugin-component
- 支持 Tree Shaking

### 国际化

内置 30 多种语言支持。

### 主题定制

支持全局主题定制，可以轻松调整组件风格。

## 适用场景

Element UI 适用于：

- 企业内部管理系统
- 电商后台
- 数据可视化平台
- CRM 系统
- Vue 2 项目

## Element Plus 的关系

Element UI 是 Element Plus（Vue 3 版本）的前身：

- Element UI：Vue 2 版本
- Element Plus：Vue 3 版本

如果使用 Vue 2，推荐使用 Element UI。如果升级到 Vue 3，建议迁移到 Element Plus。
