# Element Plus インストールガイド

> Element PlusはElementチームが開発したVue 3デスクトップUIコンポーネントライブラリ
> 
> 更新日：2026年2月
> 
> 現行バージョン：2.x（安定版）

---

## 1. プロジェクト紹介

Element Plusは、Vue 3ベースのデスクトップUIコンポーネントライブラリで、Elementチームが開発・メンテナンスを行っています。Element UIのVue 3アップグレードバージョンとして、Element PlusはVue 3のComposition APIスタイルに 完全準拠し、TypeScriptで完全に書き直し、タイプセーフなコンポーネント開発体験を提供します。

Element Plusの最初の安定版は2022年2月7日に発表され、長年の 反復によりAPIは非常に安定しています。現在では簡体字中国語、英語など多 语言をサポートしており、アクティブなコミュニティ生態系を持っています。

### 2.1 開発環境要件

| 環境 | 最小バージョン | 推奨バージョン |

| 技術 | 説明 |

| ビルドツール | Vite（推奨）または Webpack |

## 3. インストール手順

### 3.1 npmを使用（推奨）

npmはNode.jsのパッケージマネージャーであり、Element Plusをインストールする公式推奨方法です。

```bash
# Vue 3プロジェクトディレクトリに移動
cd your-vue-project

# Element Plusをインストール
npm install element-plus
```

### 3.2 yarnを使用

```bash
yarn add element-plus
```

### 3.3 pnpmを使用

```bash
pnpm add element-plus
```

## 4. Vue 3プロジェクトでの導入

### 4.1 完全導入（小規模プロジェクトに適合）

完全導入はすべてのコンポーネントをバンドルし、設定は簡単ですが、バンド ルサイズが大きくなります。

```typescript
// main.ts または main.js
import { createApp } from 'vue'
import App from './App.vue'

// Element Plusとそのスタイルをインポート
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'

const app = createApp(App)

// Element Plusを使用
app.use(ElementPlus)

app.mount('#app')
```

### 4.2 需給導入（本番環境で推奨）

需給導入は、実際に使用するコンポーネントのみをインポートすることで、 バンドルサイズを大幅に削減できます。

#### 4.2.1 自動インポートプラグインのインストール

最初にunplugin-vue-componentsとunplugin-auto-importをインストール：

```bash
npm install -D unplugin-vue-components unplugin-auto-import
```

#### 4.2.2 Viteの設定

vite.config.tsに設定を追加：

```typescript
// vite.config.ts
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

このように設定すると、コンポーネントは自動的にインポートされます。例：

```vue
<template>
  <el-button type="primary">こんにちは</el-button>
</template>
```

#### 4.2.3 手動需給導入（代替案）

自動インポートプラグインを使用したくない場合は、個別のコンポーネントを 手動でインポートできます：

```typescript
// main.ts
import { createApp } from 'vue'
import App from './App.vue'

// 必要なコンポーネントを手動でインポート
import { ElButton, ElInput, ElForm } from 'element-plus'
import 'element-plus/es/components/button/style'
import 'element-plus/es/components/input/style'
import 'element-plus/es/components/form/style'

const app = createApp(App)
app.use(ElButton)
app.use(ElInput)
app.use(ElForm)

app.mount('#app')
```

## 5. CDNで導入（ビルドツール不要）

npmやビルドツールを使用したくない場合は、CDN経由でElement Plusを 導入できます。

### 5.1 unpkg CDN経由で導入

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Element Plus CDN 示例</title>
  <!-- Vue 3をインポート -->
  <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
  <!-- Element Plusをインポート -->
  <link rel="stylesheet" href="https://unpkg.com/element-plus/dist/index.css">
  <script src="https://unpkg.com/element-plus"></script>
</head>
<body>
  <div id="app">
    <el-button type="primary">Element Plusこんにちは</el-button>
  </div>

  <script>
    const { createApp, ref } = Vue
    
    const App = {
      setup() {
        const visible = ref(false)
        return { visible }
      },
      template: `
        <el-button @click="visible = true">ダイアログを開く</el-button>
        <el-dialog v-model="visible" title="こんにちは">これはElement Plusです</el-dialog>
      `
    }
    
    const app = createApp(App)
    app.use(ElementPlus)
    app.mount('#app')
  </script>
</body>
</html>
```

### 5.2 jsDelivr CDN経由で導入

```html
<!-- unpkgをjsDelivrに置き換える -->
<script src="https://cdn.jsdelivr.net/npm/vue@3/dist/vue.global.js"></script>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/element-plus/dist/index.css">
<script src="https://cdn.jsdelivr.net/npm/element-plus"></script>
```

## 6. 国際化設定

Element Plusはデフォルトで英語を使用しています。日本語を使用するには 国際化を設定する必要があります。

### 6.1 グローバル日本語設定

```typescript
// main.ts
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import ja from 'element-plus/es/locale/lang/ja'
import 'element-plus/dist/index.css'

const app = createApp(App)

app.use(ElementPlus, {
  locale: ja,
})

app.mount('#app')
```

## 7. テーマカスタマイズ

### 7.1 SCSS変数オーバーライド

Element PlusはSCSSで記述されており、変数をオーバーライドすることで テーマをカスタマイズできます。

```scss
// styles/element-variables.scss
// テーマ色
$--color-primary: #409EFF;

// コンポーネント変数をオーバーライド
@use "element-plus/theme-chalk/src/common/var.scss" as *;
```

### 7.2 ダークモード

Element Plusはダークモードをビルトインサポート：

```typescript
// ダークモードを有効化
import 'element-plus/theme-chalk/dark/css-vars.css'
```

HTMLタグにclassを追加：

```html
<html class="dark">
  <!-- アプリケーション -->
</html>
```

## 8. よくある質問

### 8.1 インストール失敗

**問題**：npm installが失敗する

**解決策**：

1. npmキャッシュをクリア：`npm cache clean --force`
2. ネットワークを確認し、国内ミラーを使用：

```bash
# npmミラーを設定
npm config set registry https://registry.npmmirror.com
```

### 8.2 スタイルが適用されない

**問題**：コンポーネントスタイルが異常に表示される

**解決策**：

1. スタイルファイルをインポートしていることを確認：`import 'element-plus/dist/index.css'`
2. ビルド設定がCSSを正しく処理しているか確認
3. ブラウザキャッシュをクリアして再ビルド
