# Go Installation Guide

> Complete environment setup and installation tutorial

---

## Requirements

| Environment | Requirement |
|-------------|-------------|
| OS | Linux, macOS, Windows |
| Architecture | amd64, arm64, 386 |
| Disk Space | ~100 MB |

---

## Installation

### Windows

#### MSI Installer

1. Download: https://go.dev/dl/
2. Run the MSI installer
3. Verify: `go version`

#### Chocolatey

```powershell
choco install golang
```

#### Manual

1. Download Windows ZIP
2. Extract to `C:\Go`
3. Add `C:\Go\bin` to PATH
4. Verify: `go version`

### macOS

#### brew (Recommended)

```bash
brew install go
```

### Linux

#### apt (Ubuntu/Debian)

```bash
sudo apt update
sudo apt install golang-go
```

#### Manual

```bash
wget https://go.dev/dl/go1.22.0.linux-amd64.tar.gz
sudo tar -C /usr/local -xzf go1.22.0.linux-amd64.tar.gz
export PATH=$PATH:/usr/local/go/bin
```

## Environment Setup

### Proxy Configuration (China)

```bash
go env -w GOPROXY=https://goproxy.cn,direct
```

### Verify Installation

```bash
go version
go env
```

---

## Common Issues

### Windows: go command not found?

Ensure `C:\Go\bin` is in your PATH.
