# Vue Guía de Uso

Esta guía cubre cómo usar Vue.js de manera efectiva.

## Conceptos Básicos

### Instancia Vue

```javascript
import { createApp } from 'vue'

const app = createApp({
  data() {
    return {
      message: '¡Hola Vue!'
    }
  }
})

app.mount('#app')
```

### Sintaxis de Plantilla

```html
<!-- Interpolación de texto -->
<h1>{{ message }}</h1>

<!-- Directivas -->
<div v-if="show">Contenido</div>
<div v-for="item in items">{{ item }}</div>
<button @click="handleClick">Clic</button>
<input v-model="inputValue">
```

## Reactividad

### Vue 3 Composition API (Recomendado)

```javascript
import { ref, reactive, computed } from 'vue'

export default {
  setup() {
    // ref para valores primitivos
    const count = ref(0)
    
    // reactive para objetos
    const state = reactive({
      name: 'Vue',
      version: 3
    })
    
    // propiedad calculada
    const doubleCount = computed(() => count.value * 2)
    
    return { count, state, doubleCount }
  }
}
```

## Componentes

### Crear un Componente

```vue
<template>
  <div class="my-component">
    <h2>{{ title }}</h2>
    <button @click="emitEvent">Clic</button>
  </div>
</template>

<script setup>
import { defineProps, defineEmits } from 'vue'

const props = defineProps({
  title: String
})

const emit = defineEmits(['click'])

const emitEvent = () => {
  emit('click', '¡Hola del componente!')
}
</script>
```

## Renderizado Condicional

```html
<!-- v-if -->
<div v-if="show">Contenido</div>
<div v-else>Por defecto</div>

<!-- v-show -->
<div v-show="visible">Alternar visibilidad</div>
```

## Renderizado de Listas

```html
<!-- v-for con array -->
<ul>
  <li v-for="(item, index) in items" :key="item.id">
    {{ index }}. {{ item.name }}
  </li>
</ul>
```

## Manejo de Eventos

```html
<!-- Evento de clic -->
<button @click="handleClick">Clic</button>

<!-- Modificadores -->
<button @click.stop="handleClick">Detener propagación</button>
<button @click.prevent="handleSubmit">Prevenir defecto</button>

<!-- Modificadores de teclado -->
<input @keyup.enter="submit" />
```

## Enlace de Entrada de Formulario

```vue
<template>
  <form @submit.prevent="submitForm">
    <input v-model="form.name" type="text" />
    <el-checkbox v-model="form.checked">Etiqueta</el-checkbox>
    <button type="submit">Enviar</button>
  </form>
</template>

<script setup>
import { reactive } from 'vue'

const form = reactive({
  name: '',
  checked: false
})

const submitForm = () => {
  console.log(form)
}
</script>
```

## Propiedades Calculadas

```javascript
import { ref, computed } from 'vue'

const firstName = ref('John')
const lastName = ref('Doe')

// Computed - en caché, solo re-evalúa cuando cambian las dependencias
const fullName = computed(() => {
  return `${firstName.value} ${lastName.value}`
})
```

## Watchers

```javascript
import { ref, watch } from 'vue'

const count = ref(0)

// Watcher simple
watch(count, (newValue, oldValue) => {
  console.log(`Cambió de ${oldValue} a ${newValue}`)
})
```

## Ganchos de Ciclo de Vida

### Vue 3 Composition API

| Ganchos | Descripción |
|---------|-------------|
| `setup()` | Se llama antes que otros ganchos |
| `onMounted()` | Después de que el componente se monta |
| `onUpdated()` | Después de que el componente se actualiza |
| `onUnmounted()` | Después de que el componente se desmonta |

```javascript
import { onMounted, onUpdated, onUnmounted } from 'vue'

export default {
  setup() {
    onMounted(() => {
      console.log('¡Montado!')
    })
  }
}
```

## Vue Router

### Configuración Básica

```javascript
import { createRouter, createWebHistory } from 'vue-router'
import Home from '../views/Home.vue'

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```

### Navegación

```vue
<template>
  <button @click="$router.push('/about')">Acerca de</button>
  <router-link to="/">Inicio</router-link>
</template>
```

## Pinia Gestión de Estado

### Definir Store

```javascript
import { defineStore } from 'pinia'

export const useUserStore = defineStore('user', {
  state: () => ({
    name: 'John',
    email: 'john@ejemplo.com'
  }),
  getters: {
    displayName: (state) => `${state.name} (${state.email})`
  },
  actions: {
    updateName(newName) {
      this.name = newName
    }
  }
})
```
