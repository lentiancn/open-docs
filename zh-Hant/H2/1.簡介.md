# H2 資料庫簡介

## 什麼是 H2 資料庫

H2 是一個用 Java 編寫的開源關聯式資料庫管理系統。它可以用作 Java 應用程式的嵌入式資料庫，也可以執行在客戶端-伺服器模式下。該軟體以 Mozilla 公共許可證 2.0（MPL 2.0）或原始的 Eclipse 公共許可證（EPL 1.0）開源提供。

## H2 的主要特點

### 嵌入式資料庫
H2 可以嵌入到 Java 應用程式中直接執行，無需單獨的資料庫伺服器。這使得它在開發和測試環境中非常受歡迎。

### 客戶端-伺服器模式
除了嵌入式模式，H2 還支援客戶端-伺服器模式，可以作為獨立的資料庫伺服器執行，供多個應用程式同時訪問。

### 記憶體資料庫
H2 支援記憶體資料庫模式，資料存儲在記憶體中，重啟後資料會丟失。這對於快速測試和原型開發非常有用。

### 支援多種資料庫
H2 相容多種 SQL 標準和語法，可以模擬 MySQL、PostgreSQL、Oracle、DB2、MS SQL Server 等資料庫的行為，便於資料庫遷移和測試。

### 加密資料庫
H2 支援資料庫加密，可以保護敏感資料的安全。

### 全文檢索
H2 內置全文檢索功能，無需額外配置。

### 零管理
作為嵌入式資料庫，H2 不需要專門的資料庫管理員進行維護。

## H2 應用場景

H2 廣泛應用於以下場景：

- Java 應用程式的嵌入式資料庫
- 開發與測試環境
- 原型快速開發
- 單元測試和整合測試
- 輕量級桌面應用
- 學習 SQL 語法

## H2 版本歷史

H2 資料庫引擎的開發始於 2004 年 5 月，首次發佈於 2005 年 12 月。原始開發者 Thomas Mueller 也是 Java 資料庫引擎 HSQLDB（原名 Hypersonic SQL）的創始人。

## 許可證

H2 採用雙許可證模式：
- Mozilla 公共許可證 2.0（MPL 2.0）
- Eclipse 公共許可證 1.0（EPL 1.0）

## 系統要求

- Java 執行時環境（JRE）11 或更高版本
- 可以在任何支援 Java 的作業系統上執行，包括 Windows、Linux、macOS 等
