# Introduction à Apache APISIX

## Qu'est-ce qu'Apache APISIX ?

Apache APISIX est une passerelle API cloud-native et IA, développée par API7.ai et offerte à la Apache Software Foundation.

APISIX agit comme un "gestionnaire de trafic" entre les clients et les services backend.

## Caractéristiques Principales

### 1. Capacités Dynamiques

- Routing dynamique
- Upstreams dynamiques
- Certificats dynamiques
- Plugins hot-reload

### 2. Gestion du Trafic

- Équilibrage de charge (round-robin, pondéré, hash cohérent)
- Déploiement canary
- Tests A/B
- Division du trafic
- Circuit breaker

### 3. Sécurité

- Authentification par clé API
- Authentification JWT
- OAuth 2.0
- Liste blanche/noire IP
- Limitation de débit

### 4. Observabilité

- Métriques Prometheus
- Traces distribuées (Zipkin, SkyWalking)
- Export de logs (Kafka, Elasticsearch)

### 5. Multi-Protocole

- HTTP/HTTPS
- gRPC
- Dubbo
- MQTT
- WebSocket

## Architecture

- Plan de données: Nginx + Lua
- Plan de contrôle: etcd
- Gestion: REST API + Dashboard
