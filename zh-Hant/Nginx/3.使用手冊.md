# Nginx 使用手冊

## 基本操作
```bash
nginx              # 啟動
nginx -s reload   # 重載配置
nginx -t          # 測試配置
```

## 反向代理
```nginx
location / {
    proxy_pass http://backend;
}
```

## 負載均衡
```nginx
upstream backend {
    server backend1;
    server backend2;
}
```

## SSL/TLS
```nginx
listen 443 ssl;
ssl_certificate cert.crt;
ssl_certificate_key cert.key;
```
