# PostgreSQL 常见问题

## 连接问题

### Q: 无法连接

检查服务是否启动：
```bash
sudo systemctl status postgresql
```

## 权限问题

### Q: 权限不足

创建用户并授权：
```sql
CREATE USER username WITH PASSWORD 'password';
GRANT ALL PRIVILEGES ON DATABASE mydb TO username;
```

## 性能问题

### Q: 查询慢

使用 EXPLAIN 分析查询：
```sql
EXPLAIN SELECT * FROM users WHERE name = 'Tom';
```
