# Flutter 安装指南

完整的 Flutter 开发环境安装和配置指南。

## 环境要求

### Windows

- **操作系统**：Windows 10 或更高版本（64位）
- **磁盘空间**：至少 2.8 GB
- **工具**：
  - Windows PowerShell 5.0+
  - Git for Windows

### macOS

- **操作系统**：macOS 10.14 (Mojave) 或更高
- **磁盘空间**：至少 2.8 GB
- **工具**：
  - Command Line Tools
  - Git

### Linux

- **操作系统**：Ubuntu 18.04 或更高
- **磁盘空间**：至少 2.8 GB
- **依赖**：
  - bash
  - curl
  - git
  - unzip
  - xz-utils
  - zip
  - libglu1-mesa

## 安装步骤

### Windows 安装

1. **下载 Flutter SDK**

访问 Flutter 官网下载最新版本：
```bash
# 或使用 git 克隆
git clone https://github.com/flutter/flutter.git -b stable
```

2. **配置环境变量**

将 Flutter 添加到系统 PATH：
```powershell
# 在 PowerShell 中运行
[Environment]::SetEnvironmentVariable(
    "Path",
    $env:Path + ";C:\src\flutter\bin",
    "User"
)
```

3. **验证安装**

```bash
flutter doctor
```

### macOS 安装

1. **下载 Flutter SDK**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

2. **配置环境变量**

```bash
# 添加到 ~/.bash_profile 或 ~/.zshrc
export PATH="$PATH:/Users/用户名/flutter/bin"
```

3. **运行 flutter doctor**

```bash
flutter doctor
```

### Linux 安装

1. **安装依赖**

```bash
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa
```

2. **下载 Flutter SDK**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

3. **配置 PATH**

```bash
export PATH="$PATH:/home/用户名/flutter/bin"
```

## Android 开发环境配置

### 安装 Android Studio

1. 下载 Android Studio：https://developer.android.com/studio
2. 安装并启动
3. 配置 SDK：

```bash
flutter config --android-sdk /path/to/android/sdk
```

### 接受许可证

```bash
flutter doctor --android-licenses
```

### 配置虚拟机

```bash
# 创建虚拟机
flutter emulator --create --name my-emulator

# 启动虚拟机
flutter emulator --launch my-emulator
```

## iOS 开发环境配置（macOS）

### 安装 Xcode

```bash
# 通过 App Store 安装
xcode-select --install
```

### 配置 iOS 模拟器

```bash
# 查看可用模拟器
xcrun simctl list devices available

# 启动模拟器
open -a Simulator
```

## IDE 配置

### VS Code

1. 安装 Flutter 扩展
2. 安装 Dart 扩展
3. 重启 VS Code

### Android Studio

1. 安装 Flutter 插件
2. 安装 Dart 插件
3. 重启 Android Studio

## 验证安装

```bash
# 检查 Flutter 版本
flutter --version

# 运行 flutter doctor
flutter doctor

# 检查更新
flutter upgrade
```

## 常见问题

### 下载失败？

使用镜像：
```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
export PUB_CACHE=/path/to/cache
```

### Android SDK 找不到？

```bash
flutter config --android-sdk /path/to/android-sdk
```

### 命令找不到？

确保 Flutter 在 PATH 中：
```bash
echo $PATH
# 确认 flutter/bin 在其中
```

## 卸载

```bash
# 删除 Flutter 目录
rm -rf /path/to/flutter

# 从 PATH 中移除
```
