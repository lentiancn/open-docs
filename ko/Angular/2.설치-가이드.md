# Angular 설치 가이드

Angular 환경 설정 상세 가이드.

## 요구 사항

### Node.js

- 권장: Node.js 18+
- npm 또는 yarn 필요

### Node.js 버전 확인

```bash
node --version
npm --version
```

## Angular CLI 설치

### 설치

```bash
npm install -g @angular/cli
```

### 설치 확인

```bash
ng version
```

### 업그레이드

```bash
npm install -g @angular/cli@latest
```

## 새 프로젝트 생성

### Angular CLI 사용

```bash
ng new my-app
cd my-app
```

### 대화형 옵션

- 라우팅 추가? 예/아니오
- 어떤 스타일시트 형식? (SCSS/CSS/Sass/Less)

## 애플리케이션 실행

### 개발 서버

```bash
ng serve
# 또는
npm start
```

접속: http://localhost:4200

### 핫 리로딩

Angular CLI는 핫 리로딩 지원 - 변경 사항 자동 새로고침.

## 프로덕션 빌드

```bash
ng build --configuration production
```

출력은 `dist/` 디렉토리.

## 자주 사용하는 명령

| 명령 | 기능 |
|------|------|
| ng new | 새 프로젝트 생성 |
| ng serve | 개발 서버 시작 |
| ng build | 프로젝트 빌드 |
| ng generate | 컴포넌트/서비스/모듈 생성 |
| ng test | 단위 테스트 실행 |
| ng e2e | E2E 테스트 실행 |

## Angular CLI 제거

```bash
npm uninstall -g @angular/cli
```
