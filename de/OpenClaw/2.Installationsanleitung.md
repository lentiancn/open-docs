# OpenClaw Installationsanleitung

OpenClaw unterstützt mehrere Installationsmethoden, egal ob Sie macOS, Linux oder Windows verwenden. Dieser Leitfaden führt Sie durch jede Installationsoption und ihre Voraussetzungen.

---

## Systemanforderungen

Stellen Sie vor der Installation sicher, dass Ihr System diese Anforderungen erfüllt:

- **Node.js**: 22.0.0 oder höher
- **Betriebssystem**:
  - macOS 10.15 (Catalina) oder höher
  - Linux (Ubuntu 20.04+, Debian 11+, Fedora 36+, etc.)
  - Windows 10/11 (WSL2 stark empfohlen)
- **Netzwerk**: Zugang zum npm-Registry und OpenClaw-Diensten
- **Speicher**: Mindestens 500MB freier Speicherplatz

Überprüfen Sie Ihre Node.js-Version:

```bash
node --version
```

Wenn die Version unter 22.0.0 liegt, aktualisieren Sie bitte zuerst Node.js. Wir empfehlen die Verwendung von nvm (Node Version Manager) zur Verwaltung mehrerer Node.js-Versionen.

## Installationsmethoden

### Methode 1: Offizielles Installationsskript (Empfohlen)

Dies ist der einfachste und schnellste Weg. Das offizielle Installationsskript erkennt Ihr Betriebssystem automatisch und führt die gesamte Konfiguration durch.

**macOS / Linux:**

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

**Windows (PowerShell):**

```powershell
iwr -useb https://openclaw.ai/install.ps1 | iex
```

### Methode 2: npm Global Installation

Wenn Sie Node.js und npm bereits installiert haben:

```bash
npm install -g openclaw@latest
```

Sie können auch eine bestimmte Version angeben:

```bash
npm install -g openclaw@v2026.2.28
```

Verfügbare Version-Tags:
- `latest`: Neueste stabile Version
- `beta`: Beta-Version
- `dev`: Entwicklungsversion

### Methode 3: pnpm Installation

OpenClaw empfiehlt die Verwendung von pnpm:

```bash
pnpm add -g openclaw@latest
```

### Methode 4: Bun Installation

Wenn Sie Bun als JavaScript-Laufzeit verwenden:

```bash
bun add -g openclaw@latest
```

### Methode 5: Aus Quellcode bauen

Wenn Sie die neueste Entwicklungsversion verwenden möchten:

```bash
# Repository klonen
git clone https://github.com/openclaw/openclaw.git
cd openclaw

# Abhängigkeiten installieren
pnpm install

# Projekt bauen
pnpm ui:build
pnpm build

# Initialisieren und Dienst installieren
pnpm openclaw onboard --install-daemon

# Entwicklungsmodus (Auto-Reload)
pnpm gateway:watch
```

## Nach der Installation

### Onboarding-Assistent

Führen Sie nach der Installation den Assistenten aus:

```bash
openclaw onboard --install-daemon
```

### Installation verifizieren

```bash
# Version prüfen
openclaw --version

# Gateway-Status prüfen
openclaw gateway status

# Kontrolloberfläche öffnen
openclaw dashboard
```

## Häufige Installationsprobleme

### Problem 1: npm Berechtigungsfehler

Verwenden Sie nvm:

```bash
# nvm installieren
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Node.js mit nvm installieren
nvm install 22
nvm use 22

# Dann OpenClaw installieren
npm install -g openclaw@latest
```

### Problem 2: Gateway startet nicht

Prüfen Sie, ob der Port belegt ist:

```bash
lsof -i :18789
```

### Problem 3: Veraltete Version

Aktualisieren Sie:

```bash
npm update -g openclaw@latest
```

---

## OpenClaw aktualisieren

```bash
npm update -g openclaw@latest
openclaw doctor
```

---

## OpenClaw deinstallieren

```bash
# Dienst stoppen
openclaw gateway stop

# Paket deinstallieren
npm uninstall -g openclaw

# Konfigurationsverzeichnis entfernen (optional)
rm -rf ~/.openclaw
```
