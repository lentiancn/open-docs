# Element UI 概要

Vue.js 2.0 デスクトップUIコンポーネントライブラリ。

## Element UIとは？

Element UIは、Ele.meチームが開発したVue.js 2.0デスクトップUIコンポーネントライブラリです。

### 主な機能

- **豊富なコンポーネント**：60+の共通コンポーネント
- **美しいデザイン**：Material Designガイドライン
- **TypeScriptサポート**：完全な型定義
- **ツリーシェイキング**：オンデマンドインポート
- **テーマカスタマイズ**：SCSS変数サポート

### 統計

- GitHubスター：54,000+
- 週間ダウンロード：180,000+

## ユースケース

- **エンタープライズ管理システム**：ERP、CRM、OAシステム
- **管理ダッシュボード**：データ可視化
