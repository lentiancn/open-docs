## Inicio Rápido

### Instalar Angular CLI

```bash
npm install -g @angular/cli
```

### Crear Nuevo Proyecto

```bash
ng new my-app
cd my-app
ng serve
```

### Generar Componentes

```bash
ng generate component my-component
```
