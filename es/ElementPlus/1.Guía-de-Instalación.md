# Element Plus Guía de Instalación

Element Plus es una biblioteca de componentes basada en Vue 3. Esta guía cubre la instalación en todos los entornos.

## Información de Versión

| Versión | Estado | Descripción |
|---------|--------|-------------|
| Element Plus 2.x | ✅ Actual | Estable para Vue 3 |
| Element Plus 1.x | ⚠️ Mantenimiento | Solo actualizaciones de seguridad |
| Element UI | ❌ Legado | Solo para Vue 2 |

## Requisitos Previos

### Requisitos del Sistema

| Requisito | Versión |
|-----------|---------|
| Vue | 3.x |
| Node.js | 16.x+ |

## Métodos de Instalación

### Método 1: npm (Recomendado)

```bash
# Usando npm
npm install element-plus

# Usando yarn
yarn add element-plus
```

### Método 2: CDN

```html
<link rel="stylesheet" href="https://unpkg.com/element-plus/dist/index.css">
<script src="https://unpkg.com/element-plus/dist/index.full.js"></script>
```

### Método 3: Vite

```bash
npm create vite@latest my-app -- --template vue
cd my-app
npm install element-plus
```

Configure en main.js:

```javascript
import { createApp } from 'vue'
import App from './App.vue'
import 'element-plus/dist/index.css'
import ElementPlus from 'element-plus'

const app = createApp(App)
app.use(ElementPlus)
app.mount('#app')
```

### Método 4: Importación bajo demanda

```bash
npm install element-plus unplugin-vue-components unplugin-auto-import
```

Configure vite.config.js:

```javascript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({ resolvers: [ElementPlusResolver()] }),
    Components({ resolvers: [ElementPlusResolver()] }),
  ],
})
```

## Iconos

```javascript
import * as icons from '@element-plus/icons-vue'

for (const [key, icon] of Object.entries(icons)) {
    app.component(key, icon)
}
```

## Configuración de Idioma

```javascript
import ElementPlus from 'element-plus'
import es from 'element-plus/es/locale/lang/es'

app.use(ElementPlus, { locale: es })
```
