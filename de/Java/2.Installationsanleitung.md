# Java Installationsanleitung

> Vollständige Einrichtung

---

## Anforderungen

| Umgebung | Anforderung |
|----------|-------------|
| OS | Windows, macOS, Linux |
| RAM | Mindestens 2GB |

---

## Installation

### Windows

```powershell
choco install openjdk21
```

### macOS

```bash
brew install openjdk@21
```

### Linux

```bash
sudo apt install openjdk-21-jdk
```

---

## Umgebungsvariablen

```
JAVA_HOME = C:\Program Files\Java\jdk-21
```

---

## Überprüfen

```bash
java -version
```
