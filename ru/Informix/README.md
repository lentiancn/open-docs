# IBM Informix

IBM Informix — это семейство систем управления реляционными базами данных (СУРБД) и многомодельных баз данных от IBM.

## Обзор документации

Этот каталог содержит техническую документацию IBM Informix:

- **Введение**: Обзор продукта, характеристики и редакции
- **Руководство по установке**: Шаги установки для Linux и Windows
- **Руководство пользователя**: Основные команды, SQL-операторы и конфигурация
- **Часто задаваемые вопросы**: Распространённые вопросы и решения
