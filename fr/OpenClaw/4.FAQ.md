# FAQ OpenClaw

Questions et réponses courantes sur OpenClaw.

## Bases

### Qu'est-ce qu'OpenClaw?

OpenClaw est un outil auto-hébergé qui connecte les applications de messagerie (WhatsApp, Telegram, Discord, etc.) aux assistants IA. Il vous permet d'utiliser l'IA via les applications de chat que vous avez déjà installées, tout en gardant le contrôle total sur vos données.

### OpenClaw est-il gratuit?

Le logiciel lui-même est gratuit car il est open source. Cependant, vous devez payer:
- Un ordinateur ou serveur pour l'exécuter
- Les coûts de clé API pour les services IA

### Ai-je besoin d'un abonnement à un service IA?

Pas nécessairement. Vous pouvez utiliser des clés API au frais ou des comptes par abonnement. OpenClaw supporte Anthropic, OpenAI, Google Gemini et d'autres fournisseurs.

### Sur quelles plateformes ça fonctionne?

- **Bureau**: macOS, Linux, Windows
- **Mobile**: iOS, Android (comme nœuds companions)
- **Apps de chat**: WhatsApp, Telegram, Discord, iMessage, Signal, Slack et plus de 20 autres

## Installation

### L'installation a échoué. Que faire?

1. Vérifiez la version de Node.js: `node --version` (nécessite 22+)
2. Exécutez les diagnostics: `openclaw doctor`
3. Vérifiez les logs: `openclaw logs`

### Puis-je l'exécuter sur un Raspberry Pi?

Oui. Raspberry Pi 4 ou plus récent avec Raspberry Pi OS ou Ubuntu fonctionne parfaitement.

### Ai-je besoin d'un Mac?

Non. OpenClaw fonctionne sur tout système avec Node.js. Seul le canal iMessage nécessite un Mac en raison des restrictions d'Apple.

## Configuration

### Où est le fichier de configuration?

Emplacement par défaut: `~/.openclaw/openclaw.json`

### Comment contrôler qui peut m'écrire?

Utilisez le paramètre `dmPolicy`:

- `pairing`: Les inconnus ont besoin d'un code de couplage
- `allowlist`: Seuls les utilisateurs autorisés peuvent écrire
- `open`: Tout le monde peut écrire
- `disabled`: Désactiver les messages directs

### Comment configurer les groupes de chat?

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

## Utilisation

### Comment ajouter un nouveau canal de chat?

```bash
openclaw channels login whatsapp
```

### Puis-je utiliser plusieurs canaux à la fois?

Oui. Un processus Gateway se connecte à plusieurs plateformes de chat simultanément.

### Comment l'IA se souvient-elle des conversations précédentes?

OpenClaw sauvegarde automatiquement le contexte de conversation. Vous pouvez ajuster les politiques de rétention dans la configuration.

### Puis-je exécuter plusieurs agents IA?

Oui. Configurez plusieurs agents avec différentes règles de routage:

```json5
{
  agents: {
    list: [
      { id: "home", default: true },
      { id: "work" }
    ]
  }
}
```

## Sécurité

### Mes données sont-elles sécurisées?

Oui. OpenClaw s'exécute localement — toutes les données restent sur votre appareil et ne sont pas envoyées à des tiers (sauf pour les appels API des fournisseurs IA).

### Comment protéger mon installation?

1. Configurez `allowFrom` pour restreindre les sources d'accès
2. Utilisez des mots de passe forts et des clés API sécurisées
3. Gardez OpenClaw à jour
4. Utilisez un pare-feu pour limiter les ports ouverts

## Dépannage

### Le Gateway ne démarre pas

1. Vérifiez si le port est utilisé: `openclaw gateway status`
2. Vérifiez les logs d'erreur: `openclaw logs`
3. Exécutez les diagnostics: `openclaw doctor`

### L'IA ne répond pas

1. Vérifiez que la clé API est correcte
2. Vérifiez que votre compte fournisseur IA a du crédit
3. Cherchez les erreurs dans les logs

### Les messages ne s'envoient pas

1. Vérifiez que le canal est connecté
2. Confirmez que le destinataire existe
3. Vérifiez les logs détaillés
