# Flutter 설치 가이드

Flutter는 아름답고 빠른 크로스 플랫폼 애플리케이션 구축을 위해 Google이 개발한 오픈소스 UI 툴킷입니다. 이 가이드에서는 모든 환경에서의 설치 방법을 설명합니다.

## 버전 정보

| 버전 | 상태 | 설명 |
|------|------|------|
| Flutter 3.x | ✅ 현재 | 최신 안정 버전 |
| Flutter 2.x | ⚠️ 유지보수 | 보안 업데이트 |

> **중요**: 이 가이드는 주로 **Flutter 3.x**를 대상으로 합니다.

## 설치 방법

### 방법 1: Windows에 설치

#### 1. Flutter SDK 다운로드

1. Flutter SDK 다운로드: https://docs.flutter.dev/get-started/install/windows
2. 원하는 디렉토리에解压 (예: C:\src\flutter)

#### 2. 환경 변수 구성

```powershell
# PATH에 추가
$env:PATH += ";C:\src\flutter\bin"
```

#### 3. Flutter doctor 실행

```powershell
flutter doctor
```

### 방법 2: macOS에 설치

#### 1. Homebrew 사용 (권장)

```bash
# Flutter 설치
brew install flutter

# 또는 수동 다운로드
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable
```

#### 2. 환경 변수 구성

```bash
# ~/.bash_profile 또는 ~/.zshrc
export PATH="$PATH:`pwd`/flutter/bin"
```

#### 3. Flutter doctor 실행

```bash
flutter doctor
```

### 방법 3: Linux에 설치

#### 1. 종속 항목 설치

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa clang cmake ninja-build pkg-config libgtk-3-dev

# Fedora
sudo dnf install -y curl git unzip xz zip mesa-libGLU clang cmake ninja-build pkgconfig gtk3-devel
```

#### 2. Flutter SDK 다운로드

```bash
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable
export PATH="$PATH:~/development/flutter/bin"
```

#### 3. Flutter doctor 실행

```bash
flutter doctor
```

### 방법 4: VS Code 사용

1. VS Code 다운로드 및 설치
2. VS Code 열기
3. Flutter 확장 프로그램 설치:
   - "Flutter" 검색 후 설치
   - "Dart" 검색 후 설치

### 방법 5: Android Studio 사용

1. Android Studio 다운로드 및 설치
2. Flutter 플러그인 설치:
   - Android Studio 열기
   - Settings → Plugins
   - "Flutter" 검색 후 설치

## Android 구성

### Android SDK 설치

```bash
# Flutter 도구를 사용하여 설치
flutter doctor --android-licenses
flutter config --android-sdk /path/to/android/sdk
```

### 환경 변수 구성

```bash
# macOS/Linux
export ANDROID_SDK_ROOT=/Users/user/Library/Android/sdk
export PATH=$PATH:$ANDROID_SDK_ROOT/cmdline-tools/latest/bin:$ANDROID_SDK_ROOT/platform-tools
```

### 에뮬레이터 생성

```bash
# 사용 가능한 장치 목록
flutter devices

# 에뮬레이터 생성
flutter emulator --create --name my-emulator

# 에뮬레이터 시작
flutter emulators --launch my-emulator
```

## 일반적인 문제

### Flutter doctor 오류?

모든 종속 항목이 설치되어 있는지 확인:

```bash
# flutter doctor 실행
flutter doctor -v
```

### Android SDK를 찾을 수 없음?

SDK 경로 설정:

```bash
flutter config --android-sdk /path/to/android/sdk
```

### 빌드 실패?

클린 후 다시 빌드:

```bash
flutter clean
flutter pub get
flutter build apk
```
