# 소개

Angular는 Google에서 개발하고 유지관리하는 오픈소스 웹 애플리케이션 프레임워크로, 최신 웹 애플리케이션을 구축하는 데 사용됩니다. AngularJS(Angular 1.x)의 후속 버전으로, 완전히 새로 작성되었으며 TypeScript를 주요 개발 언어로 채택했습니다.

Angular는 완전한 프론트엔드 개발 솔루션을 제공합니다. 컴포넌트 기반 아키텍처, 양방향 데이터 바인딩, 의존성 주입, 라우팅, 폼 처리, HTTP 클라이언트等功能을 포함하고 있습니다. 이러한 특성을 통해 개발자는高性能적이고 유지관리하기 쉬운 단일 페이지 애플리케이션(SPA)을 빠르게 구축할 수 있습니다.

## 핵심 개념

### 컴포넌트(Component)

컴포넌트는 Angular 애플리케이션의 기본 구성 블록입니다. 각 컴포넌트는 세 부분으로 구성됩니다:
- 템플릿(Template): 뷰의 구조를 정의
- 스타일(Styles): 뷰의 외관을 정의
- 로직(Logic): 컴포넌트의 동작을 정의

### 모듈(Module)

모듈은 코드를 조직화하는 데 사용됩니다.関連する 컴포넌트, 서비스, 디렉티브 등을 함께 묶습니다. 모든 Angular 애플리케이션에는 최소 하나의 루트 모듈(AppModule)이 있습니다.

### 서비스(Service)

서비스는 재사용 가능한 비즈니스 로직을 캡슐화하며, 의존성 주입 메커니즘을 통해 컴포넌트에서 사용할 수 있도록 합니다.

### 디렉티브(Directive)

디렉티브는 HTML 요소의 기능을 확장하는 데 사용됩니다. 구조 디렉티브(*ngIf, *ngFor)와 속성 디렉티브(ngClass, ngStyle)로 구분됩니다.

## 핵심 특성

### 컴포넌트 기반 아키텍처

Angular는 컴포넌트 기반 설계를 채택했습니다. 각 컴포넌트는 독립적인 기능 단위로, 재사용하고 조합할 수 있습니다.

### TypeScript

Angular는 TypeScript를 주요 개발 언어로 사용합니다. 강력한 타입 지원, 객체 지향 프로그래밍, 최신 JavaScript 기능을 제공합니다.

### 의존성 주입

Angular는 내장된 의존성 주입 시스템을 갖추고 있어, 코드를 더 모듈화하고 테스트하기 쉽고 유지관리하기 쉽게 만들어줍니다.

### 양방향 데이터 바인딩

템플릿과 컴포넌트 간의 데이터가 자동으로 동기화되어, 수동으로 DOM을 조작하는 코드가 줄어듭니다.

### 라우팅

강력한 라우팅 시스템으로, 지연 로딩, 가드, 파라미터 전달等功能을 지원합니다.

### HTTP 클라이언트

내장된 HTTP 클라이언트로, 인터셉터, 요청/응답 변환 등 고급 기능을 지원합니다.

### 폼 처리

반응형 폼과 템플릿驱动 폼 두 가지 방식을 제공하여 복잡한 폼 검증을 처리합니다.

### 반응형 프로그래밍

RxJS를 완벽하게 지원하여 강력한 비동기 프로그래밍 기능을 제공합니다.

## 사용 시나리오

- 기업용 웹 애플리케이션
- 단일 페이지 애플리케이션(SPA)
- 프로그레시브 웹 애플리케이션(PWA)
- 하이브리드 모바일 애플리케이션(Ionic과 결합)
- 데스크톱 애플리케이션(Electron과 결합)

## 버전 히스토리

- AngularJS(1.x): 2010년 출시
- Angular 2: 2016년 출시, 완전히 새로 작성됨
- Angular 17: 2023년 출시, 최신 안정 버전

Angular는活跃한 커뮤니티와 완비된 문서를 갖추고 있어, 최신 웹 애플리케이션 구축의 주요 선택지 중 하나입니다.
