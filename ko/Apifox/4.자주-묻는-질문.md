# 자주 묻는 질문

이 문서는 Apifox 사용 중 발생하는 일반적인 기술적인 질문과 답변을 모았습니다.

## 기본 질문

### Apifox란?

Apifox는 API 설계, 문서 관리, 디버깅, 모의(mock) 데이터 생성, 자동화 테스트를 하나로 통합한 API 협업 플랫폼입니다. 개발 팀이 API 개발, 테스트, 문서 유지보수에서 더 효율적으로 작업할 수 있도록 도와줍니다.

### Apifox는 무료인가요?

Apifox는 무료 버전과 유료 버전을 모두 제공합니다. 무료 버전에는 모든 핵심 기능이 포함되어 있어 대부분의 개발 요구를 충족할 수 있습니다. 유료 버전은 더 큰 팀 규모 및 고급 자동화 같은 더 고급 기능을 제공합니다.

### Apifox와 Postman의 차이점은?

Apifox와 Postman은 모두 API 도구이지만, Apifox는 중국 사용자에게 더 적합합니다: 더 나은 중국어 지원, 더 똑똑한 모의 기능, 더 완성도 높은 팀 협업 경험. Apifox의 기본 기능은 완전히 무료인 반면, Postman의 많은 기능은 유료입니다.

## 설치 관련 문제

### 다운로드가 느린 경우?

Apifox는 여러 다운로드 서버를 제공합니다. https://apifox.com/을 방문하여 가장 가까운 서버를 선택해 보세요.

### 시작이 안 되는 경우?

1. 시스템 버전이 요구사항을 충족하는지 확인합니다
2. 관리자 권한으로 실행해 봅니다
3. 필요한 런타임 라이브러리가 설치되어 있는지 확인합니다
4. 자세한 정보를 위해 오류 로그를 확인합니다

### 로그인이 실패하는 경우?

1. 네트워크 연결이 잘 되는지 확인합니다
2. 계정 자격 증명이 정확한지 확인합니다
3. 서드파티 계정으로 로그인을 시도합니다
4. 공식 지원에 도움을 요청합니다

## 사용 관련 문제

### 인터페이스 데이터가 동기화되지 않는 경우?

1. 로그인되어 있는지 확인합니다
2. 네트워크 연결을 확인합니다
3. "동기화" 버튼을 클릭하여 수동으로 동기화합니다
4. 계정 권한을 확인합니다

### Mock 데이터가 작동하지 않는 경우?

1. Mock이 활성화되어 있는지 확인합니다
2. Mock URL이 정확한지 확인합니다
3. 인터페이스에 구문 오류가 없는지 확인합니다
4. 페이지를 새로 고쳐 봅니다

### 요청 전송이 실패하는 경우?

1. 네트워크 연결을 확인합니다
2. 요청 URL이 정확한지 확인합니다
3. 요청 매개변수가 올바르게 형식화되어 있는지 확인합니다
4. 프록시에서 차단되고 있는지 확인합니다

### 응답이 깨져서 보이는 경우?

1. 응답 인코딩을 변경해 봅니다
2. 응답 헤더의 인코딩 설정을 확인합니다
3. "서식 지정" 기능을 사용하여 다시 파싱합니다

## 기능 관련 질문

### Postman 데이터는 어떻게 가져오나요?

1. "가져오기"를 클릭합니다
2. "Postman"을 선택합니다
3. 가져올 파일을 선택합니다
4. 가져오기를 확인합니다

### Swagger로 내보내기는 어떻게 하나요?

1. 프로젝트 설정으로 이동합니다
2. "내보내기"를 선택합니다
3. "Swagger/OpenAPI"를 선택합니다
4. 버전을 선택합니다
5. 내보내기를 클릭합니다

### 환경 변수는 어떻게 사용하나요?

1. 프로젝트 설정에서 환경 변수를 정의합니다
2. 요청에서 {{variableName}}으로 참조합니다
3. 상단 환경 선택기를 사용하여 환경을 전환합니다

### 인터페이스 일괄 작업은 어떻게 하나요?

1. Ctrl/Cmd를 눌러 여러 인터페이스를 선택합니다
2. 우클릭하여 일괄 작업 선택
3. 작업 유형을 선택합니다 (삭제, 내보내기, 이동 등)

## 테스트 관련 문제

### 자동화 테스트는 어떻게 실행하나요?

1. 테스트 스위트를 엽니다
2. "실행" 버튼을 클릭합니다
3. 테스트 결과를 확인합니다

### 어설션 실패 시?

1. 실패 이유를 확인합니다
2. 예상 값과 실제 값을 비교합니다
3. 어설션 조건을 수정합니다
4. 테스트를 다시 실행합니다

### 테스트 보고서는 어떻게 내보내나요?

1. 테스트 실행이 완료된 후
2. "보고서 내보내기"를 클릭합니다
3. 형식을 선택합니다 (HTML, PDF, JSON)

## 팀 협업 관련 문제

### 팀원은 어떻게 초대하나요?

1. 프로젝트 설정으로 이동합니다
2. "멤버 관리"를 클릭합니다
3. "팀원 초대"를 클릭합니다
4. 팀원의 이메일 또는 사용자 이름을 입력합니다

### 멤버 권한은 어떻게 설정하나요?

팀원을 초대할 때 다양한 역할을 설정할 수 있습니다:

- 관리자: 전체 제어
- 편집자: 편집 및 실행 가능
- 뷰어: 읽기 전용

### 충돌은 어떻게 해결하나요?

여러 사람이 동시에 동일한 인터페이스를 편집할 때:

1. 시스템이 충돌을 경고합니다
2. 로컬 또는 원격 버전을 유지할 선택
3. 또는 변경 사항을 수동으로 병합합니다

## 데이터 관련 문제

### 데이터는 어떻게 백업하나요?

1. 정기적으로 Apifox 형식으로 프로젝트를 내보내기
2. 또는 클라우드 동기화 기능을 사용합니다

### 데이터는 어떻게 복원하나요?

1. 백업 파일이 있는 경우 "가져오기"를 클릭합니다
2. 복원할 백업 파일을 선택합니다

### 삭제된 데이터는 복구할 수 있나요?

삭제된 데이터는 휴지통에 30일 동안 유지되며, 휴지통에서 복원할 수 있습니다.

## 성능 관련 문제

### 너무 느리게 실행되는 경우?

1. 필요 없는 프로젝트를 닫습니다
2. 브라우저 캐시를 지웁니다
3. 하드웨어를 업그레이드합니다
4. 최신 버전을 시도합니다

### 대량 데이터 가져오기가 느린 경우?

1. 배치로 가져옵니다
2. 자동 동기화를 끕니다
3. 인내심을 가지고 가져오기가 완료될 때까지 기다립니다

Apifox는 데스크톱 위젯을 지원하여 데스크톱에서 자주 사용하는 인터페이스에 빠르게 접근할 수 있습니다.
