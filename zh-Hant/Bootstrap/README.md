# Bootstrap 5

Bootstrap 是世界上最流行的前端開源工具包，用於構建響應式、移動優先的網站。

## 簡介

Bootstrap 5 是最新版本，完全使用原生 JavaScript 重寫，不再依賴 jQuery。它提供了豐富的 CSS 和 JavaScript 組件，幫助開發者快速構建現代網頁。

## 主要特性

- **響應式設計**：內置響應式柵格系統
- **豐富的組件**：按鈕、導航、表單、卡片、模態框等
- **強大的工俱類**：間距、顏色、文本、顯示等實用工俱
- **Sass 支持**：支持使用 Sass 自定義主題
- **圖標庫**：集成 Bootstrap Icons
- **無需 jQuery**：完全使用原生 JavaScript

## 快速開始

### CDN 引入

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
```

### npm 安裝

```bash
npm install bootstrap@5.3.3
```
