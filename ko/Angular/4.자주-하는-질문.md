# Angular FAQ

## Q: ng 명령어를 인식하지 못합니다?
**A:** Node.js 확인: `node --version`. 터미널 재시작.

## Q: 설치가 너무 느립니다?
**A:** 미러 사용: `npm config set registry https://registry.npmmirror.com`

## Q: CORS 오류?
**A:** 프록시 구성 또는 서버에 CORS 헤더 추가.

## Q: Angular 업데이트?
**A:** `npm install -g @angular/cli@latest`
