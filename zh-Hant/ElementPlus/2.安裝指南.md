# Element Plus 安裝指南

Element Plus 多種安裝方式詳解。

## 環境要求

### 硬體要求

- CPU：2 核及以上
- 記憶體：4GB 及以上

### 軟體要求

- Node.js：18.0+
- npm：9.0+ / yarn：1.22+ / pnpm：8.0+
- Vue：3.2+
- Vite：4.0+（推薦）

## 包管理器安裝（推薦）

### npm 安裝

```bash
npm install element-plus
```

### yarn 安裝

```bash
yarn add element-plus
```

### pnpm 安裝

```bash
pnpm add element-plus
```

## 完整引入

### main.ts 配置

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')
```

## 按需引入（推薦）

### 自動導入（推薦）

**1. 安裝依賴**

```bash
npm install -D unplugin-vue-components unplugin-auto-import
```

**2. 配置 Vite**

```typescript
// vite.config.ts
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

**3. 使用元件**

```vue
<template>
  <el-button type="primary">你好</el-button>
</template>
```

### 手動按需引入

```typescript
import { ElButton, ElInput } from 'element-plus'

export default {
  components: {
    ElButton,
    ElInput,
  },
}
```

## Vue CLI 安裝

### 安裝

```bash
npm install -D sass sass-loader
npm install element-plus
```

### 配置

```javascript
// vue.config.js
module.exports = {
  configureWebpack: {
    plugins: [
      AutoImport({
        resolvers: [ElementPlusResolver()],
      }),
      Components({
        resolvers: [ElementPlusResolver()],
      }),
    ],
  },
}
```

## CDN 引入

### HTML 引入

```html
<!DOCTYPE html>
<html lang="zh-TW">
<head>
  <meta charset="UTF-8">
  <title>Element Plus CDN</title>
  <link rel="stylesheet" href="//unpkg.com/element-plus/dist/index.css" />
  <script src="//unpkg.com/vue@3"></script>
  <script src="//unpkg.com/element-plus"></script>
</head>
<body>
  <div id="app">
    <el-button type="primary">主要按鈕</el-button>
  </div>

  <script>
    const { createApp, ref } = Vue
    const App = {
      setup() {
        const visible = ref(false)
        return { visible }
      },
      template: `
        <el-button @click="visible = true">打開對話框</el-button>
        <el-dialog v-model="visible" title="提示">內容</el-dialog>
      `
    }
    const app = createApp(App)
    app.use(ElementPlus)
    app.mount('#app')
  </script>
</body>
</html>
```

## 主題定製

### 使用 CSS 變量

```css
:root {
  --el-color-primary: #409eff;
  --el-color-success: #67c23a;
  --el-border-radius-base: 4px;
}
```

### 深度定製主題

**1. 安裝主題生成器**

```bash
npm install -D element-theme-chalk
```

**2. 初始化主題文件**

```bash
et -i
```

**3. 修改變量**

編輯 `element-variables.scss`，修改顏色變量後：

```bash
et
```

### 使用暗色主題

```typescript
import 'element-plus/theme-chalk/dark/css-vars.css'
```

## 常見問題

### 圖標不顯示？

安裝圖標包：

```bash
npm install @element-plus/icons-vue
```

註冊：

```typescript
import * as icons from '@element-plus/icons-vue'

for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```

### 樣式衝突？

使用 CSS Modules 或 scoped：

```vue
<style scoped>
:deep(.el-button) {
  margin: 10px;
}
</style>
```

### TypeScript 類型錯誤？

確保安裝類型定義：

```bash
npm install -D @types/node
```

在 `tsconfig.json` 中配置：

```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

### 國際化配置

```typescript
import zhTw from 'element-plus/es/locale/lang/zh-tw'

app.use(ElementPlus, { locale: zhTw })
```

## 卸載

### npm 卸載

```bash
npm uninstall element-plus
```

### 清理配置

刪除以下文件：
- `element-variables.scss`（如使用自定義主題）
- 相關 Vite/Webpack 配置
