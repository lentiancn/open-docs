# Element Plus 개요

Vue 3용 UI 컴포넌트 라이브러리.

## Element Plus란?

Element Plus는 Ele.me 팀에서 개발한 Vue 3 컴포넌트 라이브러리로, 70개 이상의 고품질 컴포넌트를 제공합니다.

### 주요 기능

- **Vue 3 기반**: Composition API로 구축
- **TypeScript 지원**: 네이티브 타입 정의
- **Tree Shaking**: 번들 크기 최적화
- **테마 커스터마이징**: CSS 변수로 테마 설정
- **국제화**: 20개 이상 언어 지원
