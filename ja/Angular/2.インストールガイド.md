# Angular インストールガイド

完全なAngular開発環境のセットアップ。

## 環境要件

### ハードウェア要件

- CPU: 2コア以上
- メモリ: 4GB以上
- ディスク: 10GB以上

### ソフトウェア要件

- OS: Windows/macOS/Linux
- Node.js: 18.10.0+ または 20.0.0+
- npm: 9.0.0+（Node.jsに付属）
- Git: 任意バージョン

## インストール手順

### 1. Node.jsをインストール

**Windows/macOS:**

https://nodejs.org からダウンロード

またはnvmを使用:

```bash
# nvmをインストール（macOS/Linux）
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Node.js LTSをインストール
nvm install --lts
nvm use --lts
```

**インストール確認:**

```bash
node --version  # v18.x.x または v20.x.x と表示されるはず
npm --version    # 9.x.x 以上と表示されるはず
```

### 2. Angular CLIをインストール

```bash
# npmでグローバルインストール
npm install -g @angular/cli

# インストール確認
ng version
```

### 3. 新規プロジェクト作成

```bash
# 新しいAngularアプリケーションを作成
ng new my-app

# プロジェクトに移動
cd my-app

# 開発サーバーを起動
ng serve
```

### 4. アプリケーションにアクセス

ブラウザで開く: http://localhost:4200

## 一般的なコマンド

### プロジェクトコマンド

```bash
# ルーティングとSCSSで新規プロジェクト作成
ng new my-app --routing --style=scss

# スタンドアロンアプリを作成（Angular 14+）
ng new my-app --standalone

# 開発サーバーを起動
ng serve

# 本番ビルド
ng build

# ユニットテスト実行
ng test

# E2Eテスト実行
ng e2e
```

### 生成コマンド

```bash
# コンポーネント生成
ng generate component my-component
ng g c my-component

# サービス生成
ng generate service my-service
ng g s my-service

# ディレクティブ生成
ng generate directive my-directive
ng g d my-directive

# パイプ生成
ng generate pipe my-pipe
ng g p my-pipe

# モジュール生成
ng generate module my-module
ng g m my-module

# クラス生成
ng generate class my-class
ng g cl my-class

# インターフェース生成
ng generate interface my-interface
ng g i my-interface

# ガード生成
ng generate guard my-guard
ng g g my-guard

# インタ셉タ生成
ng generate interceptor my-interceptor
ng g in my-interceptor
```

## 環境設定

### 開発環境

```typescript
// src/environments/environment.ts
export const environment = {
  production: false,
  apiUrl: 'http://localhost:3000/api'
};
```

### 本番環境

```typescript
// src/environments/environment.prod.ts
export const environment = {
  production: true,
  apiUrl: 'https://api.example.com'
};
```

### 環境を使用

```typescript
import { Component } from '@angular/core';
import { environment } from '../environments/environment';

@Component({
  selector: 'app-root',
  template: `<p>API: {{ apiUrl }}</p>`
})
export class AppComponent {
  apiUrl = environment.apiUrl;
}
```

## 一般的な問題

### npmが遅い？

```bash
# Taobaoミラーを使用
npm config set registry https://registry.npmmirror.com

# またはnrmでミラー管理
npm install -g nrm
nrm use taobao
```

### Nodeバージョンが合わない？

```bash
# nvmでバージョン切り替え
nvm install 20
nvm use 20
```

### 権限問題（Linux/macOS）？

```bash
# npm権限を修正
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc
```

## アンインストール

```bash
# Angular CLIをアンインストール
npm uninstall -g @angular/cli

# プロジェクトを削除
rm -rf my-app
```
