# Element UI 使用指南

本指南介绍如何在 Vue 2 应用中使用 Element UI 组件。

## 基本用法

### Hello World

```vue
<template>
  <div>
    <el-button>默认按钮</el-button>
    <el-button type="primary">主要按钮</el-button>
  </div>
</template>

<script>
export default {
  name: 'App'
}
</script>
```

## 组件分类

Element UI 提供以下组件分类：

| 分类 | 组件 |
|------|------|
| 基础 | Layout, Button, Icon, Link |
| 表单 | Input, Select, Radio, Checkbox, Switch, Slider |
| 数据 | Table, Tree, Pagination, Tag |
| 通知 | Alert, Message, Notification, Progress |
| 导航 | Menu, Tabs, Breadcrumb, Steps, Dropdown |
| 其他 | Dialog, Drawer, Card, Collapse |

### 输入框 Input

```vue
<template>
  <div>
    <!-- 基础 -->
    <el-input v-model="input" placeholder="输入文本"></el-input>
    
    <!-- 带图标 -->
    <el-input
      v-model="input1"
      placeholder="搜索"
      prefix-icon="el-icon-search">
    </el-input>
    
    <!-- 密码 -->
    <el-input
      v-model="input2"
      placeholder="密码"
      show-password>
    </el-input>
    
    <!-- 文本域 -->
    <el-input
      v-model="textarea"
      type="textarea"
      :rows="3"
      placeholder="输入描述">
    </el-input>
  </div>
</template>

<script>
export default {
  data() {
    return {
      input: '',
      input1: '',
      input2: '',
      textarea: ''
    }
  }
}
</script>
```

### 表格 Table

```vue
<template>
  <el-table :data="tableData" stripe border>
    <el-table-column prop="date" label="日期" width="180"></el-table-column>
    <el-table-column prop="name" label="姓名" width="180"></el-table-column>
    <el-table-column prop="address" label="地址"></el-table-column>
    <el-table-column label="操作">
      <template slot-scope="scope">
        <el-button
          size="mini"
          @click="handleEdit(scope.$index, scope.row)">编辑</el-button>
        <el-button
          size="mini"
          type="danger"
          @click="handleDelete(scope.$index, scope.row)">删除</el-button>
      </template>
    </el-table-column>
  </el-table>
</template>

<script>
export default {
  data() {
    return {
      tableData: [{
        date: '2016-05-02',
        name: '张三',
        address: '北京市朝阳区'
      }]
    }
  },
  methods: {
    handleEdit(index, row) {
      console.log(index, row);
    },
    handleDelete(index, row) {
      console.log(index, row);
    }
  }
}
</script>
```

### 表单 Form

```vue
<template>
  <el-form
    :model="ruleForm"
    :rules="rules"
    ref="ruleForm"
    label-width="120px"
  >
    <el-form-item label="姓名" prop="name">
      <el-input v-model="ruleForm.name"></el-input>
    </el-form-item>
    
    <el-form-item label="邮箱" prop="email">
      <el-input v-model="ruleForm.email"></el-input>
    </el-form-item>
    
    <el-form-item label="性别" prop="gender">
      <el-radio-group v-model="ruleForm.gender">
        <el-radio label="male">男</el-radio>
        <el-radio label="female">女</el-radio>
      </el-radio-group>
    </el-form-item>
    
    <el-form-item label="爱好" prop="hobby">
      <el-checkbox-group v-model="ruleForm.hobby">
        <el-checkbox label="阅读" name="hobby"></el-checkbox>
        <el-checkbox label="运动" name="hobby"></el-checkbox>
        <el-checkbox label="音乐" name="hobby"></el-checkbox>
      </el-checkbox-group>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm('ruleForm')">提交</el-button>
      <el-button @click="resetForm('ruleForm')">重置</el-button>
    </el-form-item>
  </el-form>
</template>

<script>
export default {
  data() {
    return {
      ruleForm: {
        name: '',
        email: '',
        gender: '',
        hobby: []
      },
      rules: {
        name: [
          { required: true, message: '请输入姓名', trigger: 'blur' }
        ],
        email: [
          { required: true, message: '请输入邮箱', trigger: 'blur' },
          { type: 'email', message: '邮箱格式不正确', trigger: 'blur' }
        ]
      }
    };
  },
  methods: {
    submitForm(formName) {
      this.$refs[formName].validate((valid) => {
        if (valid) {
          alert('提交成功!');
        }
      });
    },
    resetForm(formName) {
      this.$refs[formName].resetFields();
    }
  }
}
</script>
```

## 事件

```vue
<template>
  <el-button @click="handleClick">点击我</el-button>
  
  <el-input
    @focus="handleFocus"
    @blur="handleBlur"
    @change="handleChange">
  </el-input>
</template>

<script>
export default {
  methods: {
    handleClick() {
      console.log('点击!')
    },
    handleFocus() {
      console.log('获得焦点!')
    },
    handleBlur() {
      console.log('失去焦点!')
    },
    handleChange(value) {
      console.log('改变:', value)
    }
  }
}
</script>
```

## 最佳实践

### 1. 组件局部引入

```javascript
export default {
  components: {
    [Button.name]: Button,
    [Input.name]: Input
  }
}
```

### 2. 表单验证

```javascript
const validatePass = (rule, value, callback) => {
  if (value === '') {
    callback(new Error('请输入密码'));
  } else {
    callback();
  }
};
```
