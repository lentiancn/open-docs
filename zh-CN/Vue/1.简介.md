# Vue.js - 简介

Vue.js（通常简称为 Vue）是一个用于构建用户界面的渐进式 JavaScript 框架。由 Evan You 于 2014 年创建，Vue 最初是为了解决 AngularJS 开发中的复杂性而设计的。Vue 采用了 Angular 的模板语法和 React 的组件化思想，形成了自己独特的风格。

Vue 的核心库专注于视图层，易于学习和集成。同时，Vue 也拥有强大的生态系统，支持构建单页应用（SPA）、移动应用（通过 Vue Native）、桌面应用（通过 Electron）等各种应用。

## Vue 的设计理念

### 渐进式

Vue 是一个渐进式框架，这意味着你可以逐步采用它的特性：

1. **基础使用**：在 HTML 中直接引入 Vue.js，构建简单交互
2. **组件化**：使用单文件组件组织代码
3. **路由**：添加 Vue Router 实现页面导航
4. **状态管理**：引入 Vuex/Pinia 管理应用状态
5. **构建工具**：使用 Vite 或 Vue CLI 构建项目

### 响应式数据

Vue 的核心特性之一是响应式系统。当你修改数据时，视图会自动更新：

```javascript
const app = Vue.createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
})

app.mount('#app')
```

```html
<div id="app">
  {{ message }}
</div>
```

当 `message` 改变时，页面会自动更新。

### 组件化

Vue 推荐使用组件来构建用户界面。组件是可复用的 Vue 实例：

```javascript
app.component('my-button', {
  props: ['label'],
  template: `<button>{{ label }}</button>`
})
```

## Vue 的核心特性

### 1. 虚拟 DOM

Vue 使用虚拟 DOM 来优化 DOM 操作。当数据变化时，Vue 会创建一个虚拟 DOM 树，然后与旧的虚拟 DOM 树进行比较（diff 算法），只更新发生变化的部分。

### 2. 组件系统

Vue 的组件系统允许你将 UI 拆分为可重用的组件：

```vue
<template>
  <div class="user-profile">
    <img :src="user.avatar" :alt="user.name">
    <h2>{{ user.name }}</h2>
    <p>{{ user.bio }}</p>
  </div>
</template>

<script>
export default {
  props: {
    user: {
      type: Object,
      required: true
    }
  }
}
</script>

<style scoped>
.user-profile {
  text-align: center;
}
</style>
```

### 3. 指令系统

Vue 提供了丰富的指令来处理 DOM 操作：

| 指令 | 说明 |
|------|------|
| v-bind | 动态绑定属性 |
| v-if | 条件渲染 |
| v-for | 列表渲染 |
| v-on | 事件监听 |
| v-model | 双向数据绑定 |
| v-show | 显示/隐藏 |
| v-text | 文本插值 |

```html
<!-- 条件渲染 -->
<div v-if="isLoggedIn">欢迎回来</div>
<div v-else>请登录</div>

<!-- 列表渲染 -->
<ul>
  <li v-for="item in items" :key="item.id">
    {{ item.name }}
  </li>
</ul>

<!-- 事件处理 -->
<button @click="handleClick">点击</button>

<!-- 双向绑定 -->
<input v-model="username">
```

### 4. 单文件组件

Vue 单文件组件（.vue 文件）将模板、脚本和样式封装在一个文件中：

```vue
<template>
  <div class="component">
    <h1>{{ title }}</h1>
    <button @click="increment">{{ count }}</button>
  </div>
</template>

<script setup>
import { ref } from 'vue'

const count = ref(0)
const title = '计数器'

function increment() {
  count.value++
}
</script>

<style scoped>
.component {
  padding: 20px;
}
</style>
```

### 5. 组合式 API

Vue 3 引入了组合式 API，提供更灵活的代码组织方式：

```vue
<script setup>
import { ref, computed, onMounted } from 'vue'

// 响应式状态
const count = ref(0)

// 计算属性
const doubled = computed(() => count.value * 2)

// 方法
function increment() {
  count.value++
}

// 生命周期钩子
onMounted(() => {
  console.log('组件已挂载')
})
</script>
```

## Vue 的版本

- 2016 年发布
- Options API
- 响应式系统基于 Object.defineProperty
- 社区资源丰富

### Vue 3

- 2020 年发布
- 组合式 API
- 响应式系统基于 Proxy
- 更好的 TypeScript 支持
- 更好的性能

### Vue 3.3+

- 2023 年发布
- 进一步改进开发体验
- 更好的类型推导

## Vue 的应用场景

### 1. 单页应用（SPA）

Vue + Vue Router 构建单页应用，提供流畅的用户体验：

```javascript
import { createRouter, createWebHistory } from 'vue-router'
import Home from './views/Home.vue'
import About from './views/About.vue'

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})
```

### 2. 企业级应用

Vue 的稳定性和生态系统使其适合构建大型企业应用：

- Ant Design Vue
- Element Plus
- Vuetify

### 3. 渐进式增强

可以在现有项目中逐步引入 Vue：

```html
<!-- 在现有页面中添加交互 -->
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<div id="app">
  {{ message }}
</div>

<script>
  const { createApp } = Vue
  createApp({
    data() {
      return { message: 'Hello!' }
    }
  }).mount('#app')
</script>
```

### 4. 移动应用

通过 Vue Native 或 Weex 构建跨平台移动应用。

### 5. 桌面应用

结合 Electron 构建桌面应用：

- VS Code 的主要部分使用 Electron
- 可以使用 Vue 作为 UI 框架

## Vue 与其他框架对比

### Vue vs React

| 特性 | Vue | React |
|------|-----|-------|
| 学习曲线 | 平缓 | 较陡 |
| 模板 | HTML 模板 | JSX |
| 状态管理 | Pinia/Vuex | Redux/MobX |
| 响应式 | 自动 | 手动 |
| 社区 | 活跃 | 更活跃 |

### Vue vs Angular

| 特性 | Vue | Angular |
|------|-----|---------|
| 体积 | 小 | 大 |
| 灵活性 | 高 | 低 |
| TypeScript | 可选 | 必须 |
| 学习难度 | 低 | 高 |
