# Manual de Usuario de Go

## Sintaxis Básica

### Declaración de Variables

Go tiene múltiples formas de declarar variables. La declaración básica usa la palabra clave var:

```go
// Declarar variable con tipo
var name string = "Go"

// Declaración con inferencia de tipo
var age = 10

// Declaración por lotes
var (
    firstName string
    lastName  string
    score     float64
)

// Declaración corta (dentro de función)
message := "Hello, World!"
```

Las variables de Go deben usarse después de declararse, de lo contrario habrá error de compilación. Si necesita declarar una variable temporalmente no usada, puede usar el guión bajo _ para recibirla:

```go
result, _ := someFunction()  // Ignorar el primer valor de retorno
```

### Declaración de Constantes

Las constantes se declaran usando la palabra clave const y pueden ser caracteres, cadenas, booleanos o números:

```go
const Pi = 3.14159
const (
    StatusOK = 200
    StatusNotFound = 404
)
```

### Tipos de Datos Básicos

Los tipos de datos básicos de Go incluyen:

- Booleano: bool (true o false)
- Enteros: int, int8, int16, int32, int64, y tipos sin signo correspondientes uint, etc.
- Punto flotante: float32, float64
- Complejos: complex64, complex128
- Cadenas: string

La conversión de tipos debe ser explícita:

```go
var i int = 42
var f float64 = float64(i)
var s string = strconv.Itoa(i)  // Entero a cadena
```

## Funciones

### Definición de Funciones

Las funciones de Go se definen usando la palabra clave func:

```go
func greet(name string) string {
    return "Hello, " + name
}

// Múltiples valores de retorno
func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}

// Valores de retorno con nombre
func split(sum int) (x, y int) {
    x = sum * 4 / 9
    y = sum - x
    return  // No necesita especificar valores de retorno
}
```

### Parámetros Variádicos

Las funciones pueden aceptar un número variable de parámetros:

```go
func sum(nums ...int) int {
    total := 0
    for _, n := range nums {
        total += n
    }
    return total
}

// Uso
result := sum(1, 2, 3, 4, 5)
```

### Closures

Go soporta closures, donde las funciones pueden capturar y modificar variables del ámbito exterior:

```go
func counter() func() int {
    count := 0
    return func() int {
        count++
        return count
    }
}

// Uso
c := counter()
fmt.Println(c())  // 1
fmt.Println(c())  // 2
```

## Programación Concurrente

### Goroutines

Las goroutines son unidades de ejecución ligera en Go con un costo de creación extremadamente bajo:

```go
// Iniciar goroutine
go func() {
    fmt.Println("Hello from goroutine!")
}()

// Goroutine con parámetros
go func(msg string) {
    fmt.Println(msg)
}("Hello!")
```

### Canales

Los canales son el mecanismo de comunicación entre goroutines:

```go
// Crear canal
ch := make(chan string)

// Enviar datos
ch <- "message"

// Recibir datos
msg := <-ch

// Canal con buffer
bufferedCh := make(chan int, 10)
```

### Select

La sentencia select se usa para esperar múltiples operaciones de canal simultáneamente:

```go
select {
case msg := <-ch1:
    fmt.Println("Received:", msg)
case msg := <-ch2:
    fmt.Println("Received:", msg)
case <-time.After(time.Second):
    fmt.Println("Timeout")
}
```

### Seguridad de Concurrencia

Use el paquete sync para control de concurrencia:

```go
var (
    mu    sync.Mutex
    count int
)

func increment() {
    mu.Lock()
    defer mu.Unlock()
    count++
}
```

## Manejo de Errores

### Retornos de Error

Go usa valores de retorno para manejo de errores, que es la forma idiomática en Go:

```go
result, err := doSomething()
if err != nil {
    return err
}
```

### Errores Personalizados

Cree errores personalizados usando el paquete errors:

```go
import "errors"

func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}
```

### Envoltura de Errores

Go 1.13 introdujo la envoltura de errores usando fmt.Errorf y el verbo %w:

```go
if err != nil {
    return fmt.Errorf("failed to do something: %w", err)
}
```

## Interfaces

### Definición de Interfaces

Las interfaces de Go se implementan implícitamente, sin necesidad de declaración explícita:

```go
type Writer interface {
    Write([]byte) (int, error)
}

type Reader interface {
    Read([]byte) (int, error)
}

// Interfaz compuesta común
type ReadWriter interface {
    Reader
    Writer
}
```

### Interfaz Vacía

La interfaz vacía interface{} puede almacenar valores de cualquier tipo:

```go
var i interface{} = 42
var s interface{} = "hello"
var m interface{} = map[string]int{"a": 1}
```

Go 1.18 introdujo genéricos, permitiendo usar any en lugar de interface{}:

```go
var i any = 42
```

## Paquetes y Módulos

### Crear un Módulo

Use go mod init para crear un nuevo módulo:

```bash
go mod init github.com/username/project
```

### Gestión de Dependencias

Agregar dependencias:

```bash
go get github.com/pkg/errors
```

Actualizar dependencias:

```bash
go get -u all
```

Limpiar dependencias no usadas:

```bash
go mod tidy
```

## Pruebas

### Pruebas Unitarias

Go tiene un marco de pruebas incorporado. Los archivos de prueba terminan en _test.go:

```go
import "testing"

func TestAdd(t *testing.T) {
    result := Add(2, 3)
    if result != 5 {
        t.Errorf("Add(2, 3) = %d; want 5", result)
    }
}
```

Ejecutar pruebas:

```bash
go test ./...
go test -v ./...
go test -cover ./...
```

### Pruebas de Rendimiento

Las funciones de prueba comienzan con Benchmark:

```go
func BenchmarkHello(b *testing.B) {
    for i := 0; i < b.N; i++ {
        Hello()
    }
}
```

Ejecutar pruebas de rendimiento:

```bash
go test -bench=.
```

## Biblioteca Estándar Común

### fmt - E/S Formateado

```go
fmt.Println("Hello")
fmt.Printf("Name: %s, Age: %d\n", name, age)
fmt.Sprintf("Result: %d", result)
```

### strings - Operaciones de Cadena

```go
strings.Contains(s, "sub")
strings.Split(s, ",")
strings.TrimSpace(s)
strings.ToUpper(s)
```

### net/http - Servidor HTTP

```go
http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
    fmt.Fprint(w, "Hello, HTTP!")
})
http.ListenAndServe(":8080", nil)
```

### json - Procesamiento JSON

```go
// Codificar
data, _ := json.Marshal(person)

// Decodificar
var p Person
json.Unmarshal(data, &p)
```

## Herramientas Comunes

### gofmt - Formateo de Código

```bash
gofmt -w main.go  # Formatear y escribir en archivo
gofmt -d main.go  # Mostrar diferencias de formateo
```

### go vet - Análisis Estático

```bash
go vet ./...  # Verificar problemas de código
```

### go doc - Visor de Documentación

```go
go doc fmt.Println  # Ver documentación de función
go doc fmt          # Ver documentación de paquete
```
