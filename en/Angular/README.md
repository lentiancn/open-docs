# Angular

Angular 是由 Google 维护的现代 Web 应用框架。

## 简介

Angular 是一个 TypeScript 开源 Web 应用框架，专为构建高效、复杂的单页应用而设计。

## 主要特性

- **组件化架构**：基于组件的设计模式
- **TypeScript 支持**：原生 TypeScript 支持
- **双向数据绑定**：自动同步模型和视图
- **依赖注入**：强大的依赖注入系统
- **路由系统**：完整的路由解决方案
- **HTTP 客户端**：完整的 HTTP 客户端
- **表单处理**：响应式表单和模板驱动表单
- **测试支持**：完整的单元测试和端到端测试

## 快速开始

### 安装 Angular CLI

```bash
npm install -g @angular/cli
```

### 创建新项目

```bash
ng new my-app
cd my-app
ng serve
```
