# frp 安装指南

## 下载

```bash
# 下载 release 包
wget https://github.com/fatedier/frp/releases/download/v0.51.3/frp_0.51.3_linux_amd64.tar.gz

# 解压
tar -xzf frp_0.51.3_linux_amd64.tar.gz
cd frp_0.51.3_linux_amd64
```

## 目录结构

```
frps          # 服务端可执行文件
frpc          # 客户端可执行文件
frps.ini      # 服务端配置
frpc.ini      # 客户端配置
systemd/      # systemd 服务配置
```

## 服务端安装

### 1. 配置 frps.ini

```ini
[common]
bind_port = 7000
dashboard_port = 7500
token = your_token_here
```

### 2. 启动服务端

```bash
# 前台运行
./frps -c frps.ini

# 后台运行
nohup ./frps -c frps.ini &
```

### 3. systemd 服务

```bash
# 复制服务文件
sudo cp systemd/frps.service /etc/systemd/system/

# 启动
sudo systemctl start frps
sudo systemctl enable frps
```

## 客户端安装

### 1. 配置 frpc.ini

```ini
[common]
server_addr = your_server_ip
server_port = 7000
token = your_token_here

[ssh]
type = tcp
local_ip = 127.0.0.1
local_port = 22
remote_port = 6000
```

### 2. 启动客户端

```bash
# 前台运行
./frpc -c frpc.ini

# 后台运行
nohup ./frpc -c frpc.ini &
```
