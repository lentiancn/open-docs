# GitLab 使用手冊

> 使用 GitLab 的完整指南

---

## 開始使用

### 建立帳戶

1. 前往您的 GitLab 執行個體
2. 點擊「註冊」建立帳戶
3. 驗證電子郵件地址
4. 完成使用者個人資料

### 建立專案

1. 點擊「新增專案」
2. 選擇專案範本或空白專案
3. 設定專案名稱和描述
4. 設定可見性級別
5. 點擊「建立專案」

---

## Git 操作

### 複製儲存庫

```bash
# 使用 HTTPS 複製
git clone https://gitlab.example.com/使用者/專案.git

# 使用 SSH 複製
git clone git@gitlab.example.com:使用者/專案.git
```

### 基本 Git 工作流程

```bash
# 建立新分支
git checkout -b feature/新功能

# 進行更改並暫存
git add .
git commit -m "新增新功能"

# 推送到遠端
git push origin feature/新功能
```

---

## 問題追蹤

### 建立問題

1. 前往您的專案
2. 前往「問題」>「新增問題」
3. 新增標題和描述
4. 指派標籤、里程碑和負責人
5. 提交問題

---

## 合併請求

### 建立合併請求

1. 將功能分支推送到 GitLab
2. 點擊「建立合併請求」
3. 填寫標題和描述
4. 選擇來源和目標分支
5. 新增標籤和審查者
6. 提交合併請求

---

## CI/CD

### 基本 .gitlab-ci.yml

```yaml
stages:
  - build
  - test
  - deploy

build:
  stage: build
  script:
    - npm install
    - npm run build

test:
  stage: test
  script:
    - npm test

deploy:
  stage: deploy
  script:
    - npm run deploy
  only:
    - main
```
