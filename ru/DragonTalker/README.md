# DragonTalker

> Создавайте реалистичные видео говорящих голов из одного изображения и аудио

---

## Описание проекта

DragonTalker — это система генерации говорящих голов на основе глубокого обучения, которая создаёт реалистичные видео из одного статического изображения человека и аудиофайла. Используя передовые алгоритмы 3D-реконструкции лица и синтеза изображений, DragonTalker идеально сохраняет оригинальные черты лица, достигая точной синхронизации губ и естественных изменений выражения.

---

## Основные функции

- Аудио-управляемая анимация: создаёт синхронизированные говорящие видео из изображения и аудио
- 3D-оценка движения: естественные позы головы и выражения лица на основе 3D-реконструкции
- Множество поз: более 45 предопределённых шаблонов поз головы
- Улучшение лица: интегрированы GFPGAN, RestoreFormer, CodeFormer
- Веб-интерфейс: удобный браузерный интерфейс
- Python API: полный программный интерфейс для разработки

## Быстрый старт

### Установка

```bash
git clone https://github.com/your-repo/DragonTalker.git
cd DragonTalker
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
bash scripts/download_models.sh
```

### Создание видео

```bash
python inference.py \
  --source_image examples/source_image.jpg \
  --driven_audio examples/driven_audio.wav \
  --result_dir ./results \
  --enhancer gfpgan
```

### Веб-интерфейс

```bash
python app.py
```

Браузер: http://localhost:7860

## Требования к входным данным

### Изображение

- Формат: JPG, PNG
- Разрешение: 512×512 или выше
- Содержимое: лицо анфас, чёткое, без遮挡

### Аудио

- Формат: WAV, MP3
- Длительность: 1-60 секунд
- Качество: чёткий голос

---

## Выходные данные

- Формат: MP4 (H.264)
- Разрешение: 256×256 или 512×512
- Частота кадров: 25 FPS
