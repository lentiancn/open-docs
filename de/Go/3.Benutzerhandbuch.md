# Go Benutzerhandbuch

## Grundlegende Syntax

### Variablendeklaration

Go hat mehrere Möglichkeiten, Variablen zu deklarieren. Die grundlegende Deklaration verwendet das var-Schlüsselwort:

```go
// Variable mit Typ deklarieren
var name string = "Go"

// Typ-Inferenz-Deklaration
var age = 10

// Stapel-Deklaration
var (
    firstName string
    lastName  string
    score     float64
)

// Kurz-Deklaration (in Funktion)
message := "Hello, World!"
```

Go-Variablen müssen nach der Deklaration verwendet werden, sonst gibt es einen Kompilierungsfehler. Wenn Sie eine Variable vorübergehend deklarieren müssen, die nicht verwendet wird, können Sie den Unterstrich _ verwenden, um sie zu empfangen:

```go
result, _ := someFunction()  // Ersten Rückgabewert ignorieren
```

### Konstantendeklaration

Konstanten werden mit dem const-Schlüsselwort deklariert und können Zeichen, Strings, boolesche Werte oder Zahlen sein:

```go
const Pi = 3.14159
const (
    StatusOK = 200
    StatusNotFound = 404
)
```

### Grundlegende Datentypen

Go's grundlegende Datentypen umfassen:

- Boolean: bool (true oder false)
- Ganzzahlen: int, int8, int16, int32, int64, und entsprechende vorzeichenlose Typen uint usw.
- Gleitkommazahlen: float32, float64
- Komplexe Zahlen: complex64, complex128
- Strings: string

Typumwandlung muss explizit erfolgen:

```go
var i int = 42
var f float64 = float64(i)
var s string = strconv.Itoa(i)  // Integer zu String
```

## Funktionen

### Funktionsdefinition

Go-Funktionen werden mit dem func-Schlüsselwort definiert:

```go
func greet(name string) string {
    return "Hello, " + name
}

// Mehrere Rückgabewerte
func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}

// Benannte Rückgabewerte
func split(sum int) (x, y int) {
    x = sum * 4 / 9
    y = sum - x
    return  // Keine Rückgabewerte angeben
}
```

### Variadische Parameter

Funktionen können eine variable Anzahl von Parametern akzeptieren:

```go
func sum(nums ...int) int {
    total := 0
    for _, n := range nums {
        total += n
    }
    return total
}

// Verwendung
result := sum(1, 2, 3, 4, 5)
```

### Abschlüsse

Go unterstützt Abschlüsse, bei denen Funktionen Variablen aus dem äußeren Bereich erfassen und ändern können:

```go
func counter() func() int {
    count := 0
    return func() int {
        count++
        return count
    }
}

// Verwendung
c := counter()
fmt.Println(c())  // 1
fmt.Println(c())  // 2
```

## Parallele Programmierung

### Goroutinen

Goroutinen sind leichtgewichtige Ausführungseinheiten in Go mit extrem niedrigen Erstellungskosten:

```go
// Goroutine starten
go func() {
    fmt.Println("Hello from goroutine!")
}()

// Goroutine mit Parametern
go func(msg string) {
    fmt.Println(msg)
}("Hello!")
```

### Channels

Channels sind der Kommunikationsmechanismus zwischen Goroutinen:

```go
// Channel erstellen
ch := make(chan string)

// Daten senden
ch <- "message"

// Daten empfangen
msg := <-ch

// Gepufferter Channel
bufferedCh := make(chan int, 10)
```

### Select

Die select-Anweisung wird verwendet, um auf mehrere Channel-Operationen gleichzeitig zu warten:

```go
select {
case msg := <-ch1:
    fmt.Println("Received:", msg)
case msg := <-ch2:
    fmt.Println("Received:", msg)
case <-time.After(time.Second):
    fmt.Println("Timeout")
}
```

### Parallelitätssicherheit

Verwenden Sie das sync-Paket für Parallelitätssteuerung:

```go
var (
    mu    sync.Mutex
    count int
)

func increment() {
    mu.Lock()
    defer mu.Unlock()
    count++
}
```

## Fehlerbehandlung

### Fehlerrückgabe

Go verwendet Rückgabewerte für die Fehlerbehandlung, was die idiomatische Methode in Go ist:

```go
result, err := doSomething()
if err != nil {
    return err
}
```

### Benutzerdefinierte Fehler

Erstellen Sie benutzerdefinierte Fehler mit dem errors-Paket:

```go
import "errors"

func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}
```

### Fehlerverpackung

Go 1.13 führte die Fehlerverpackung mit fmt.Errorf und dem %w-Verb ein:

```go
if err != nil {
    return fmt.Errorf("failed to do something: %w", err)
}
```

## Schnittstellen

### Schnittstellendefinition

Go-Schnittstellen werden implizit implementiert, ohne explizite Deklaration:

```go
type Writer interface {
    Write([]byte) (int, error)
}

type Reader interface {
    Read([]byte) (int, error)
}

// Übliche zusammengesetzte Schnittstelle
type ReadWriter interface {
    Reader
    Writer
}
```

### Leere Schnittstelle

Die leere Schnittstelle interface{} kann Werte jedes Typs speichern:

```go
var i interface{} = 42
var s interface{} = "hello"
var m interface{} = map[string]int{"a": 1}
```

Go 1.18 führte Generics ein, die die Verwendung von any anstelle von interface{} ermöglichen:

```go
var i any = 42
```

## Pakete und Module

### Ein Modul erstellen

Verwenden Sie go mod init, um ein neues Modul zu erstellen:

```bash
go mod init github.com/username/project
```

### Abhängigkeitsverwaltung

Abhängigkeiten hinzufügen:

```bash
go get github.com/pkg/errors
```

Abhängigkeiten aktualisieren:

```bash
go get -u all
```

Nicht verwendete Abhängigkeiten bereinigen:

```bash
go mod tidy
```

## Tests

### Unit-Tests

Go hat ein eingebautes Test-Framework. Testdateien enden mit _test.go:

```go
import "testing"

func TestAdd(t *testing.T) {
    result := Add(2, 3)
    if result != 5 {
        t.Errorf("Add(2, 3) = %d; want 5", result)
    }
}
```

Tests ausführen:

```bash
go test ./...
go test -v ./...
go test -cover ./...
```

### Benchmark-Tests

Benchmark-Funktionen beginnen mit Benchmark:

```go
func BenchmarkHello(b *testing.B) {
    for i := 0; i < b.N; i++ {
        Hello()
    }
}
```

Benchmark-Tests ausführen:

```bash
go test -bench=.
```

## Häufige Standardbibliothek

### fmt - Formatierte E/A

```go
fmt.Println("Hello")
fmt.Printf("Name: %s, Age: %d\n", name, age)
fmt.Sprintf("Result: %d", result)
```

### strings - String-Operationen

```go
strings.Contains(s, "sub")
strings.Split(s, ",")
strings.TrimSpace(s)
strings.ToUpper(s)
```

### net/http - HTTP-Server

```go
http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
    fmt.Fprint(w, "Hello, HTTP!")
})
http.ListenAndServe(":8080", nil)
```

### json - JSON-Verarbeitung

```go
// Kodieren
data, _ := json.Marshal(person)

// Dekodieren
var p Person
json.Unmarshal(data, &p)
```

### gofmt - Code-Formatierung

```bash
gofmt -w main.go  # Formatieren und in Datei schreiben
gofmt -d main.go  # Formatierungsunterschiede anzeigen
```

### go vet - Statische Analyse

```bash
go vet ./...  # Auf Code-Probleme prüfen
```

### go doc - Dokumentationsanzeige

```go
go doc fmt.Println  # Funktionsdokumentation anzeigen
go doc fmt          # Paketdokumentation anzeigen
```
