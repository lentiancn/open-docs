# Apifox Overview

API all-in-one collaboration platform.

## What is Apifox?

Apifox is an API all-in-one collaboration platform that integrates API documentation, API debugging, API Mock, and API automated testing.

### Key Features

- **API Documentation** - Auto-generate API docs with team collaboration
- **API Debugging** - Complete HTTP request simulation tool
- **Mock Services** - Quickly create Mock servers
- **Automated Testing** - API automated testing functionality
- **Team Collaboration** - Project management and access control
- **Data Sync** - Real-time API change synchronization
- **Import/Export** - Support multiple formats
