# Docker

La plateforme de conteneurisation la plus populaire au monde.

## Introduction

Docker est une plateforme open source pour développer, déployer et exécuter des applications. La technologie de conteneurisation permet d'emballer les applications et leurs dépendances dans des conteneurs légers et portables.

### Caractéristiques principales

- Léger - Les conteneurs partagent le noyau, démarrage rapide
- Portable - Construire une fois, exécuter partout
- Isolation - Les conteneurs sont isolés les uns des autres
- Efficace - Plus économe en ressources que les machines virtuelles

## Démarrage rapide

### Installation

**Windows/macOS**:
Télécharger Docker Desktop: https://www.docker.com/products/docker-desktop

**Linux (Ubuntu)**:
```
curl -fsSL https://get.docker.com | sh
```

### Exécuter votre premier conteneur

```
docker run hello-world
```
