# MultiTalk 常見問題

本頁面回答關於 MultiTalk 的常見問題。

## 一般問題

### 什麼是 MultiTalk？

MultiTalk 是一款專為遊戲玩家和社群設計的多方語音聊天應用程式。它提供低延遲、高品質的語音通訊功能。

### MultiTalk 是免費的嗎？

是的，MultiTalk 提供免費版本，基本功能完全免費。也有付費版本提供更多功能。

### MultiTalk 支援哪些平台？

MultiTalk 支援 Windows、macOS 和 Linux 作業系統。

## 安裝問題

### 安裝失敗怎麼辦？

1. 確保您的系統符合最低需求
2. 暫時停用防毒軟體
3. 以系統管理員身份執行安裝程式
4. 重新下載安裝檔案

### 為什麼無法啟動 MultiTalk？

1. 檢查是否已完整安裝
2. 確認系統有圖形介面支援
3. 檢查顯示驅動程式
4. 嘗試重新安裝

## 音頻問題

### 麥克風沒有聲音怎麼辦？

1. 確認麥克風已正確連接
2. 檢查系統權限設定
3. 在 MultiTalk 設定中選擇正確的輸入裝置
4. 測試麥克風是否正常運作

### 聲音斷斷續續怎麼辦？

1. 檢查網路連線品質
2. 關閉其他佔用頻寬的應用程式
3. 嘗試更換音頻裝置
4. 檢查路由器設定

### 如何減少背景噪音？

1. 啟用噪音抑制功能
2. 使用品質較好的麥克風
3. 調整麥克風增益
4. 使用耳機以避免回音

### 聽不到其他人的聲音怎麼辦？

1. 確認喇叭或耳機音量
2. 檢查輸出裝置設定
3. 確認未被靜音
4. 檢查房間音量設定

## 連線問題

### 連線到房間失敗怎麼辦？

1. 檢查網路連線
2. 確認房間ID正確
3. 檢查密碼是否正確
4. 嘗試連線到其他伺服器

### 延遲很高怎麼辦？

1. 選擇距離較近的伺服器
2. 檢查網路速度
3. 關閉 VPN
4. 聯繫網路服務提供商

### 被踢出房間怎麼辦？

1. 聯繫房主了解原因
2. 等待一段時間後重新加入
3. 檢查是否違反房間規則

## 帳戶問題

### 如何建立帳戶？

1. 開啟 MultiTalk
2. 點擊「註冊」
3. 輸入電子郵件和密碼
4. 驗證電子郵件
5. 完成註冊

### 忘記密碼怎麼辦？

1. 點擊「忘記密碼」
2. 輸入註冊的電子郵件
3. 檢查郵件中的重設連結
4. 設定新密碼

### 如何修改個人資料？

1. 開啟「設定」
2. 選擇「個人資料」
3. 修改名稱、頭像等資訊
4. 儲存變更

## 功能和設定

### 如何設定按住說話？

1. 開啟「設定」
2. 選擇「音頻」
3. 啟用「按住說話」
4. 選擇熱鍵
5. 儲存設定

### 如何建立私人房間？

1. 點擊「建立房間」
2. 勾選「私人房間」
3. 設定密碼
4. 完成建立

### 如何管理房間成員？

1. 以房主身份進入房間
2. 點擊成員列表
3. 選擇要管理的成員
4. 執行相關操作

## 技術問題

### 如何回報錯誤？

1. 前往 GitHub 問題頁面
2. 搜尋是否已有相同問題
3. 創建新的問題
4. 提供詳細的錯誤資訊

### 如何取得技術支援？

- 官方網站：https://multitalk.app
- GitHub：https://github.com/multitalk
- 社群論壇

### 系統需求為何？

- Windows 10+ / macOS 10.15+ / Ubuntu 20.04+
- 4GB RAM
- 100MB 可用空間
- 音頻輸入/輸出裝置

## 隱私和安全

### MultiTalk 會收集哪些資料？

MultiTalk 收集基本帳戶資訊和使用資料以改善服務。我們不會出售個人資訊。

### 我的語音資料會被儲存嗎？

預設情況下，語音資料不會被儲存。如有需要，房主可以啟用錄音功能。

### 如何保護我的帳戶？

1. 使用強密碼
2. 啟用兩步驟驗證
3. 不要與他人分享帳戶資訊
