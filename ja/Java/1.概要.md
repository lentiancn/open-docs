# Java 概要

> Javaはオブジェクト指向プログラミング言語

---

## 説明

Javaは1995年にSun Microsystemsによって作成されました。プラットフォーム独立性で知られる。

---

## 特徴

- オブジェクト指向
- プラットフォーム独立（JVM）
- 自動メモリ管理
- マルチスレッド
- 豊富なエコシステム
