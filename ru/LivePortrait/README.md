# LivePortrait

LivePortrait — это эффективный инструмент анимации портретов с открытым исходным кодом, который преобразует статические портретные изображения в динамические видео, позволяя объектам на фотографиях воспроизводить выражения и движения головы на основе управляющих видео.

## Функции

- Анимация статических изображений: Дает жизнь фотографиям с динамическими выступлениями
- Поддержка животных: Поддержка анимации портретов кошек, собак и других животных
- Редактирование видео: Поддержка модификации выражений и движений в портретных видео
- Точный контроль: Предоставляет механизмы управления сшивкой и перенаправлением
- Кроссплатформенная поддержка: Поддерживает Linux, Windows и macOS

## Быстрый старт

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/KlingTeam/LivePortrait
cd LivePortrait

# Создание среды
conda create -n LivePortrait python=3.10
conda activate LivePortrait

# Установка зависимостей
pip install -r requirements.txt

# Загрузка предварительно обученных моделей
huggingface-cli download KlingTeam/LivePortrait --local-dir pretrained_weights
```

### Запуск вывода

```bash
# Базовый вывод
python inference.py

# Пользовательские входные данные
python inference.py -s исходное_изображение.jpg -d управляющее_видео.mp4
```

### Использование веб-интерфейса

```bash
python app.py
```

Затем откройте отображаемый адрес в браузере.

## Требования

- Python 3.10
- NVIDIA GPU (RTX 4090 рекомендуется)
- Минимум 8 ГБ видеопамяти
- 20 ГБ доступного места на диске
