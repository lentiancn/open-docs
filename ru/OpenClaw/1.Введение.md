# OpenClaw 🦞

Инструмент, который вы устанавливаете на своё устройство и который соединяет мессенджеры с ИИ-ассистентами. WhatsApp, Telegram, Discord, iMessage — всё работает напрямую с ИИ-сервисами.

## Для кого?

Если хотите своего ИИ-ассистента, но не хотите доверять данные третьим лицам. Особенно подходит для разработчиков.

## Основные возможности

- **Ваши данные на вашем устройстве**: ничего не уходит на внешние серверы
- **Несколько платформ**: один процесс управляет кучей чатов
- **Для программирования**: инструменты, история, память, несколько ассистентов
- **Открытый код**: MIT лицензия

## Что нужно

- Node.js 22+
- API-ключ от ИИ-провайдера
- несколько минут на настройку

## Быстрый старт

```bash
npm install -g openclaw@latest
openclaw onboard --install-daemon
openclaw gateway --port 18789
```

Откройте http://127.0.0.1:18789/ в браузере.
