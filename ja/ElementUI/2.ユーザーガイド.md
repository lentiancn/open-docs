# Element UI ユーザーガイド

このガイドでは、Vue 2アプリケーションでElement UIコンポーネントを使用方法について説明します。

## 基本的な使用方法

### Hello World

```vue
<template>
  <div>
    <el-button>デフォルトボタン</el-button>
    <el-button type="primaryプライマリーボタン</el-button>
  </div>
</template>

<script>
export default {
  name: 'App'
}
</script>
```

## コンポーネントカテゴリ

Element UIは以下のコンポーネントカテゴリを提供します：

| カテゴリ | コンポーネント |
|----------|----------------|
| -basic | Layout, Button, Icon, Link |
| フォーム | Input, Select, Radio, Checkbox, Switch, Slider |
| データ | Table, Tree, Pagination, Tag |
| 通知 | Alert, Message, Notification, Progress |
| ナビゲーション | Menu, Tabs, Breadcrumb, Steps, Dropdown |
| その他 | Dialog, Drawer, Card, Collapse |

### 入力フィールド Input

```vue
<template>
  <div>
    <!-- -basic -->
    <el-input v-model="input" placeholder="テキストを入力"></el-input>
    
    <!-- アイコン付き -->
    <el-input
      v-model="input1"
      placeholder="検索"
      prefix-icon="el-icon-search">
    </el-input>
    
    <!-- パスワード -->
    <el-input
      v-model="input2"
      placeholder="パスワード"
      show-password>
    </el-input>
    
    <!-- テキストエリア -->
    <el-input
      v-model="textarea"
      type="textarea"
      :rows="3"
      placeholder="説明を入力">
    </el-input>
  </div>
</template>

<script>
export default {
  data() {
    return {
      input: '',
      input1: '',
      input2: '',
      textarea: ''
    }
  }
}
</script>
```

### テーブル Table

```vue
<template>
  <el-table :data="tableData" stripe border>
    <el-table-column prop="date" label="日付" width="180"></el-table-column>
    <el-table-column prop="name" label="名前" width="180"></el-table-column>
    <el-table-column prop="address" label="住所"></el-table-column>
    <el-table-column label="操作">
      <template slot-scope="scope">
        <el-button
          size="mini"
          @click="handleEdit(scope.$index, scope.row)">編集</el-button>
        <el-button
          size="mini"
          type="danger"
          @click="handleDelete(scope.$index, scope.row)">削除</el-button>
      </template>
    </el-table-column>
  </el-table>
</template>

<script>
export default {
  data() {
    return {
      tableData: [{
        date: '2016-05-02',
        name: '田中太郎',
        address: '東京都渋谷区'
      }]
    }
  },
  methods: {
    handleEdit(index, row) {
      console.log(index, row);
    },
    handleDelete(index, row) {
      console.log(index, row);
    }
  }
}
</script>
```

### フォーム Form

```vue
<template>
  <el-form
    :model="ruleForm"
    :rules="rules"
    ref="ruleForm"
    label-width="120px"
  >
    <el-form-item label="名前" prop="name">
      <el-input v-model="ruleForm.name"></el-input>
    </el-form-item>
    
    <el-form-item label="メール" prop="email">
      <el-input v-model="ruleForm.email"></el-input>
    </el-form-item>
    
    <el-form-item label="性別" prop="gender">
      <el-radio-group v-model="ruleForm.gender">
        <el-radio label="male">男</el-radio>
        <el-radio label="female">女</el-radio>
      </el-radio-group>
    </el-form-item>
    
    <el-form-item label="趣味" prop="hobby">
      <el-checkbox-group v-model="ruleForm.hobby">
        <el-checkbox label="読書" name="hobby"></el-checkbox>
        <el-checkbox label="スポーツ" name="hobby"></el-checkbox>
        <el-checkbox label="音楽" name="hobby"></el-checkbox>
      </el-checkbox-group>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm('ruleForm')">送信</el-button>
      <el-button @click="resetForm('ruleForm')">リセット</el-button>
    </el-form-item>
  </el-form>
</template>

<script>
export default {
  data() {
    return {
      ruleForm: {
        name: '',
        email: '',
        gender: '',
        hobby: []
      },
      rules: {
        name: [
          { required: true, message: '名前を入力してください', trigger: 'blur' }
        ],
        email: [
          { required: true, message: 'メールを入力してください', trigger: 'blur' },
          { type: 'email', message: 'メール形式が正しくありません', trigger: 'blur' }
        ]
      }
    };
  },
  methods: {
    submitForm(formName) {
      this.$refs[formName].validate((valid) => {
        if (valid) {
          alert('送信成功!');
        }
      });
    },
    resetForm(formName) {
      this.$refs[formName].resetFields();
    }
  }
}
</script>
```

## イベント

```vue
<template>
  <el-button @click="handleClick">クリック</el-button>
  
  <el-input
    @focus="handleFocus"
    @blur="handleBlur"
    @change="handleChange">
  </el-input>
</template>

<script>
export default {
  methods: {
    handleClick() {
      console.log('クリック!')
    },
    handleFocus() {
      console.log('フォーカス!')
    },
    handleBlur() {
      console.log('ブラー!')
    },
    handleChange(value) {
      console.log('変更:', value)
    }
  }
}
</script>
```

## ベストプラクティス

### 1. コンポーネントのローカルインポート

```javascript
export default {
  components: {
    [Button.name]: Button,
    [Input.name]: Input
  }
}
```

### 2. フォーム検証

```javascript
const validatePass = (rule, value, callback) => {
  if (value === '') {
    callback(new Error('パスワードを入力してください'));
  } else {
    callback();
  }
};
```
