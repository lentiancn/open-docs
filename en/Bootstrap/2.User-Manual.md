## Grid System

### Basic Principles

Bootstrap's grid system is built using Flexbox and supports up to 12 columns.

### Containers

```html
<!-- Fixed width container -->
<div class="container">
  Content
</div>

<!-- Full width container -->
<div class="container-fluid">
  Content
</div>

<!-- Responsive breakpoint containers (Bootstrap 5.2+) -->
<div class="container-sm">100% width < 576px</div>
<div class="container-md">100% width >= 768px</div>
<div class="container-lg">100% width >= 992px</div>
<div class="container-xl">100% width >= 1200px</div>
<div class="container-xxl">100% width >= 1400px</div>
```

### Rows and Columns

```html
<div class="container">
  <div class="row">
    <div class="col">Column</div>
    <div class="col">Column</div>
    <div class="col">Column</div>
  </div>
</div>
```

### Column Width Setting

```html
<div class="row">
  <div class="col-4">4 columns</div>
  <div class="col-8">8 columns</div>
</div>
```

### Responsive Breakpoints

```html
<!-- Responsive column width -->
<div class="row">
  <div class="col-12 col-md-6 col-lg-4">
    Full width on mobile, 6 columns on tablet, 4 columns on desktop
  </div>
</div>
```

### Column Offset

```html
<!-- Using offset -->
<div class="row">
  <div class="col-md-4">4 columns</div>
  <div class="col-md-4 offset-md-4">Offset 4 columns</div>
</div>

<!-- Using margin auto offset -->
<div class="row">
  <div class="col-md-3 ms-auto">Right offset</div>
</div>
```

### Grid Nesting

```html
<div class="row">
  <div class="col-sm-8">
    Outer 8 columns
    <div class="row">
      <div class="col-6">Inner 6 columns</div>
      <div class="col-6">Inner 6 columns</div>
    </div>
  </div>
</div>
```

### Flex Layout

```html
<!-- Horizontal alignment -->
<div class="row justify-content-start">...</div>
<div class="row justify-content-center">...</div>
<div class="row justify-content-end">...</div>
<div class="row justify-content-around">...</div>
<div class="row justify-content-between">...</div>
<div class="row justify-content-evenly">...</div>

<!-- Vertical alignment -->
<div class="row align-items-start">...</div>
<div class="row align-items-center">...</div>
<div class="row align-items-end">...</div>
```

```html
<h3>Heading 3</h3>
<h4>Heading 4</h4>

<h1 class="display-3">Display 3</h1>
<h1 class="display-4">Display 4</h1>

```

### Paragraphs

```html
<!-- Regular paragraph -->
<p>This is a paragraph.</p>

<!-- Lead paragraph -->
<p class="lead">This is a lead paragraph.</p>

<!-- Text alignment -->
<p class="text-start">Left aligned</p>
<p class="text-center">Center aligned</p>
<p class="text-end">Right aligned</p>
<p class="text-justify">Justified</p>

<!-- Responsive alignment -->
<p class="text-sm-start">Left on small</p>
<p class="text-md-center">Center on medium</p>
<p class="text-lg-end">Right on large</p>
```

## Best Practices

### 1. Use Semantic HTML

```html
<!-- Good -->
<nav class="navbar">...</nav>
<main class="container">...</main>
<article class="card">...</article>

<!-- Avoid -->
<div class="navbar">...</div>
<div class="container">...</div>
<div class="card">...</div>
```

### 2. Keep Code Clean

```html
<!-- Good -->
<div class="container">
  <div class="row">
    <div class="col-md-6">...</div>
  </div>
</div>

<!-- Avoid excessive nesting -->
<div class="container">
  <div class="row">
    <div class="col-md-6">
      <div class="card">
        <div class="card-body">...</div>
      </div>
    </div>
  </div>
</div>
```

### 3. Responsive Design

```html
<!-- Use correct breakpoint classes -->
<div class="col-12 col-md-6 col-lg-4">
  Content
</div>
```

### 4. Performance Optimization

- Use CDN for acceleration
- Enable CSS and JS compression
- Load JavaScript on demand
