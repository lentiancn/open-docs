# Angular インストールガイド

> セットアップ

---

## 必要環境

| 環境 | 要件 |
|------|------|
| Node.js | 18.10.0+ |
| npm | 9.0.0+ |

---

## インストール

```bash
npm install -g @angular/cli
ng new my-app
ng serve
```

---

## コマンド

| コマンド | 説明 |
|--------|------|
| ng new | 新規プロジェクト |
| ng serve | 開発サーバー |
| ng build | ビルド |
