# JDK

Java 开发工具包

## 简介

JDK（Java Development Kit）是 Java 语言的软件开发工具包，是整个 Java 开发的核心。JDK 包含了 Java 运行环境（JRE）、Java 编译器（javac）、Java 虚拟机（JVM）以及大量的 Java 类库和工具。

## 主要版本

- **JDK 25** - 最新版本，带来新特性和性能改进
- **JDK 21** - 最新的长期支持版本（LTS）
- **JDK 17** - 成熟的 LTS 版本，广泛应用
- **JDK 11** - LTS 版本，标志 Java 付费特性的分水岭

## 核心组件

- **javac** - Java 编译器，将源代码编译为字节码
- **java** - Java 程序运行命令
- **jvm** - Java 虚拟机，执行字节码
- **jar** - Java 归档工具
- **javadoc** - API 文档生成工具
- **jdb** - Java 调试器
