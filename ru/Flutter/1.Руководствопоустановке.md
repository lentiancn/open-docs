# Руководство по установке Flutter

Flutter — это набор инструментов пользовательского интерфейса с открытым исходным кодом, разработанный Google для создания красивых и быстрых кроссплатформенных приложений. В этом руководстве описаны методы установки во всех средах.

## Информация о версии

| Версия | Статус | Описание |
|--------|--------|----------|
| Flutter 3.x | ✅ Текущая | Последняя стабильная версия |
| Flutter 2.x | ⚠️ Поддержка | Обновления безопасности |

> **Важно**: Это руководство в основном ориентировано на **Flutter 3.x**.

## Методы установки

### Метод 1: Установка в Windows

#### 1. Загрузка Flutter SDK

1. Загрузите Flutter SDK: https://docs.flutter.dev/get-started/install/windows
2. Распакуйте в нужный каталог (например, C:\src\flutter)

#### 2. Настройка переменных среды

```powershell
# Добавить в PATH
$env:PATH += ";C:\src\flutter\bin"
```

#### 3. Запуск Flutter doctor

```powershell
flutter doctor
```

### Метод 2: Установка в macOS

#### 1. Использование Homebrew (рекомендуется)

```bash
# Установить Flutter
brew install flutter

# Или загрузить вручную
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable
```

#### 2. Настройка переменных среды

```bash
# ~/.bash_profile или ~/.zshrc
export PATH="$PATH:`pwd`/flutter/bin"
```

#### 3. Запуск Flutter doctor

```bash
flutter doctor
```

### Метод 3: Установка в Linux

#### 1. Установка зависимостей

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa clang cmake ninja-build pkg-config libgtk-3-dev

# Fedora
sudo dnf install -y curl git unzip xz zip mesa-libGLU clang cmake ninja-build pkgconfig gtk3-devel
```

#### 2. Загрузка Flutter SDK

```bash
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable
export PATH="$PATH:~/development/flutter/bin"
```

#### 3. Запуск Flutter doctor

```bash
flutter doctor
```

### Метод 4: Использование VS Code

1. Скачайте и установите VS Code
2. Откройте VS Code
3. Установите расширения Flutter:
   - Найдите "Flutter" и установите
   - Найдите "Dart" и установите

### Метод 5: Использование Android Studio

1. Скачайте и установите Android Studio
2. Установите плагин Flutter:
   - Откройте Android Studio
   - Settings → Plugins
   - Найдите "Flutter" и установите

## Настройка Android

### Установка Android SDK

```bash
# Установка через инструменты Flutter
flutter doctor --android-licenses
flutter config --android-sdk /path/to/android/sdk
```

### Настройка переменных среды

```bash
# macOS/Linux
export ANDROID_SDK_ROOT=/Users/user/Library/Android/sdk
export PATH=$PATH:$ANDROID_SDK_ROOT/cmdline-tools/latest/bin:$ANDROID_SDK_ROOT/platform-tools
```

### Создание эмулятора

```bash
# Список доступных устройств
flutter devices

# Создание эмулятора
flutter emulator --create --name my-emulator

# Запуск эмулятора
flutter emulators --launch my-emulator
```

## Частые проблемы

### Ошибка Flutter doctor?

Убедитесь, что все зависимости установлены:

```bash
# Запустить flutter doctor
flutter doctor -v
### Android SDK не найден?

Укажите путь к SDK:

```bash
flutter config --android-sdk /path/to/android/sdk
```

### Ошибка сборки?

Очистите и пересоберите:

```bash
flutter clean
flutter pub get
flutter build apk
```
