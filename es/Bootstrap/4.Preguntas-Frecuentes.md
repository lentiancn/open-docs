# Bootstrap Preguntas frecuentes

Preguntas y respuestas comunes.

## Instalación

### Q: ¿CDN no carga?

**Soluciones**:
1. Verificar conexión de red
2. Confirmar URL correcta
3. Usar CDN alternativo

### Q: ¿npm instalación falló?

**Soluciones**:
1. Limpiar caché npm
2. Verificar versión de Node.js

### Q: ¿Aún necesito jQuery?

**Soluciones**:
No. Bootstrap 5 eliminó la dependencia de jQuery.

## Uso

### Q: ¿Estilos no funcionan?

**Soluciones**:
1. Confirmar CSS importado correctamente
2. Limpiar caché del navegador

### Q: ¿Responsive no funciona?

**Soluciones**:
Añadir meta标签 de viewport:
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### Q: ¿Componentes no funcionan?

**Soluciones**:
1. Confirmar JavaScript importado
2. Verificar Popper.js

## Compatibilidad

### Q: ¿IE no soportado?

Bootstrap 5 ya no soporta IE 10 y inferior.

## Rendimiento

### Q: ¿Archivos muy grandes?

**Soluciones**:
1. Usar Sass para importar solo lo necesario
2. Usar versiones de producción (.min)

## Recursos

### Q: ¿Dónde aprender Bootstrap?
