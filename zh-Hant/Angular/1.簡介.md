# 簡介

Angular 是由 Google 開發維護的開源 Web 應用框架，用於建構現代 Web 應用程式。它是 AngularJS（Angular 1.x）的後繼者，進行了完全重寫，並採用 TypeScript 作為主要開發語言。

Angular 提供了一套完整的前端開發解決方案，包括元件化架構、雙向資料綁定、依賴注入、路由、表單處理、HTTP 客戶端等功能。透過這些特性，開發者可以快速建構高效能、易於維護的單頁應用（SPA）。

## 核心概念

### 元件（Component）

元件是 Angular 應用的基本建構區塊。每個元件由三個部分組成：
- 模板（Template）：定義視圖的結構
- 樣式（Styles）：定義視圖的外觀
- 邏輯（Logic）：定義元件的行為

### 模組（Module）

模組用於組織程式碼，把相關的元件、服務、指令等整合在一起。每個 Angular 應用至少會有一個根模組（AppModule）。

### 服務（Service）

服務用於封裝可重複使用的業務邏輯，透過依賴注入機制提供給元件使用。

### 指令（Directive）

指令用來擴展 HTML 元素的功能，分為結構指令（如 *ngIf、*ngFor）和屬性指令（如 ngClass、ngStyle）。

## 核心特性

### 元件化架構

Angular 採用元件化設計，每個元件都是獨立的功能單元，可以重複使用和組合。

### TypeScript

Angular 使用 TypeScript 作為主要開發語言，提供強型別支援、物件導向程式設計和現代 JavaScript 特性。

### 依賴注入

Angular 內建依賴注入系統，讓程式碼更加模組化、易於測試和維護。

### 雙向資料綁定

模板與元件之間的資料會自動同步，減少了手動操作 DOM 的程式碼。

### 路由

強大的路由系統，支援懶載入、守衛、參數傳遞等功能。

### HTTP 客戶端

內建 HTTP 客戶端，支援攔截器、請求/回應轉換等進階功能。

### 表單處理

提供響應式表單和模板驅動表單兩種方式，處理複雜的表單驗證。

### 響應式程式設計

全面支援 RxJS，提供強大的非同步程式設計能力。

## 適用場景

- 企業級 Web 應用
- 單頁應用（SPA）
- 漸進式 Web 應用（PWA）
- 混合行動應用（結合 Ionic）
- 桌面應用（結合 Electron）

## 版本歷史

- AngularJS（1.x）：2010 年發布
- Angular 2：2016 年發布，完全重寫
- Angular 17：2023 年發布最新穩定版本

Angular 擁有活躍的社群和完善的文件，是建構現代 Web 應用的主流選擇之一。
