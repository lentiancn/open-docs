# TensorFlow 常见问题

## 安装问题

### 问题：ImportError: No module named tensorflow

**解决方案**：
```bash
# 确认安装
pip show tensorflow

# 重新安装
pip install --upgrade tensorflow
```

### 问题：GPU 不可用

**排查步骤**：
1. 确认 CUDA、cuDNN 安装正确
2. 检查版本兼容性
3. 确认 GPU 驱动版本

```python
import tensorflow as tf
print(tf.config.list_physical_devices('GPU'))
```

## 性能问题

### 问题：训练速度慢

**优化建议**：
1. 使用 GPU 加速
2. 调整 batch_size
3. 使用 tf.data 优化数据加载
4. 启用混合精度训练

```python
# 启用混合精度
from tensorflow.keras import mixed_precision
mixed_precision.set_global_policy('mixed_float16')
```

### 问题：内存不足（OOM）

**解决方案**：
```python
# 限制 GPU 内存增长
gpus = tf.config.list_physical_devices('GPU')
if gpus:
    for gpu in gpus:
        tf.config.experimental.set_memory_growth(gpu, True)
```

## 模型问题

### 问题：模型不收敛

**排查步骤**：
1. 检查学习率
2. 检查数据预处理
3. 检查损失函数选择
4. 添加正则化防止过拟合

### 问题：预测结果全为 0

检查数据是否正确归一化，标签是否正确。
