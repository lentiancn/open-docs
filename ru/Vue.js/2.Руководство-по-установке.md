# Vue.js Руководство по установке

## Попробуйте Vue онлайн

- **Песочница**: Попробуйте напрямую в [Vue Playground](https://play.vuejs.org)
- **JSFiddle**: Используйте этот [шаблон JSFiddle](https://jsfiddle.net/yyx990803/2ke1ab0z/) для простой HTML-настройки
- **StackBlitz**: Попробуйте полную настройку сборки прямо в браузере на [StackBlitz](https://vite.new/vue)
- **Интерактивное руководство**: Посмотрите это интерактивное [руководство Scrimba](http://scrimba.com/links/vue-quickstart), которое покажет вам, как запустить, редактировать и развернуть ваше первое приложение Vue

## Создание приложения Vue (рекомендуется)

### Предварительные требования
- Знакомство с командной строкой
- Установите Node.js версии ^20.19.0 || >=22.12.0

### Использование официального инструмента каркасной генерации
Выполните одну из следующих команд в терминале:

```bash
# npm
npm create vue@latest

# pnpm  
pnpm create vue@latest

# yarn (v1+)
yarn create vue

# yarn (Modern v2+)
yarn create vue@latest

# yarn (^v4.11)
yarn dlx create-vue@latest

# bun
bun create vue@latest
```

### Опции конфигурации проекта
Вам будут предложены несколько дополнительных функций:
- Project name: Название вашего проекта
- Add TypeScript?: Добавить поддержку TypeScript
- Add JSX Support?: Добавить поддержку JSX
- Add Vue Router: Для разработки одностраничных приложений
- Add Pinia: Для управления состоянием
- Add Vitest: Для модульного тестирования
- Add End-to-End Testing: Cypress/Nightwatch/Playwright
- Add ESLint: Для качества кода
- Add Prettier: Для форматирования кода
- Add Vue DevTools 7: Для отладки (экспериментально)

### Запуск сервера разработки
```bash
cd <your-project-name>
npm install
npm run dev
```

### Сборка для производства
```bash
npm run build
```

## Использование Vue из CDN

### Глобальная сборка
```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<div id="app">{{ message }}</div>

<script>
const { createApp } = Vue

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

### Сборка ES-модулей
```html
<div id="app">{{ message }}</div>

<script type="module">
import { createApp } from 'https://unpkg.com/vue@3/dist/vue.esm-browser.js'

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

### Включение карт импорта
```html
<script type="importmap">
{
  "imports": {
    "vue": "https://unpkg.com/vue@3/dist/vue.esm-browser.js"
  }
}
</script>

<div id="app">{{ message }}</div>

<script type="module">
import { createApp } from 'vue'

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

## Рекомендации по среде разработки

- **IDE**: Visual Studio Code + [официальное расширение Vue](https://marketplace.visualstudio.com/items?itemName=Vue.volar)
- **Инструмент сборки**: Vite (включен в официальный каркас)
- **Локальный сервер**: Для ES-модулей используйте `npx serve` для запуска локального HTTP-сервера
- **Альтернатива для простых случаев**: Рассмотрите [petite-vue](https://github.com/vuejs/petite-vue) для легковесных сценариев
