# Apifox User Manual

Apifox core features and usage tips.

## API Management

### Create Interface

1. Select project
2. Click "New Interface"
3. Fill in interface information:
   - Interface name
   - Request method (GET/POST/PUT/DELETE)
   - Request path
   - Request parameters
   - Request body

### Debug Interface

1. Select interface
2. Fill in request parameters
3. Click send
4. View response

## Request Configuration

### Request Headers

```http
Content-Type: application/json
Authorization: Bearer token
```

### Request Parameters

- **Query Parameters**: URL query parameters
- **Header Parameters**: Request header parameters
- **Body Parameters**: Request body parameters

### Authentication

- **Bearer Token**: JWT authentication
- **Basic Auth**: Basic authentication
- **OAuth 2.0**: OAuth authentication

## Mock Service

### Create Mock

1. Click "New Mock"
2. Define response rules
3. Get Mock URL

### Mock Rules

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "id": 1,
    "name": "{{name}}"
  }
}
```
