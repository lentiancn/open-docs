# Guía de Uso de Vue

Esta guía cubre cómo usar Vue.js efectivamente en tus aplicaciones.

## Conceptos Básicos

### Instancia Vue

```javascript
import { createApp } from 'vue'

const app = createApp({
  data() {
    return {
      message: '¡Hola Vue!'
    }
  }
})

app.mount('#app')
```

### Sintaxis de Plantillas

```html
<!-- Interpolación de texto -->
<h1>{{ message }}</h1>

<!-- Directivas -->
<div v-if="show">Contenido</div>
<div v-for="item in items">{{ item }}</div>
<button @click="handleClick">Clicar</button>
<input v-model="inputValue">
```

## Reactividad

### Vue 3 Composition API (Recomendado)

```javascript
import { ref, reactive, computed } from 'vue'

export default {
  setup() {
    // Ref para valores primitivos
    const count = ref(0)
    
    // Reactive para objetos
    const state = reactive({
      name: 'Vue',
      version: 3
    })
    
    // Computado
    const doubleCount = computed(() => count.value * 2)
    
    // Métodos
    const increment = () => {
      count.value++
    }
    
    return { count, state, doubleCount, increment }
  }
}
```

### Vue 2 Options API

```javascript
export default {
  data() {
    return {
      count: 0,
      name: 'Vue',
      version: 2
    }
  },
  computed: {
    doubleCount() {
      return this.count * 2
    }
  },
  methods: {
    increment() {
      this.count++
    }
  }
}
```

## Componentes

### Crear un Componente

```vue
<!-- MyComponent.vue -->
<template>
  <div class="my-component">
    <h2>{{ title }}</h2>
    <p>{{ content }}</p>
    <button @click="emitEvent">Clicar</button>
  </div>
</template>

<script setup>
import { defineProps, defineEmits } from 'vue'

const props = defineProps({
  title: {
    type: String,
    required: true
  },
  content: {
    type: String,
    default: 'Contenido por defecto'
  }
})

const emit = defineEmits(['click'])

const emitEvent = () => {
  emit('click', '¡Hola desde el componente!')
}
</script>

<style scoped>
.my-component {
  padding: 20px;
}
</style>
```

### Usar Componentes

```vue
<!-- ParentComponent.vue -->
<template>
  <MyComponent 
    title="Mi Título"
    content="Contenido personalizado"
    @click="handleClick"
  />
</template>

<script setup>
import MyComponent from './MyComponent.vue'

const handleClick = (message) => {
  console.log(message)
}
</script>
```

## Comunicación entre Componentes

### Props (Padre a Hijo)

```vue
<!-- Child.vue -->
<script setup>
defineProps({
  message: String,
  count: Number
})
</script>

<template>
  <p>{{ message }} - {{ count }}</p>
</template>

<!-- Parent.vue -->
<template>
  <Child message="Hola" :count="5" />
</template>
```

### Eventos (Hijo a Padre)

```vue
<!-- Child.vue -->
<script setup>
const emit = defineEmits(['update'])

const sendData = () => {
  emit('update', { name: 'Vue' })
}
</script>

<template>
  <button @click="sendData">Enviar</button>
</template>

<!-- Parent.vue -->
<template>
  <Child @update="handleUpdate" />
</template>

<script setup>
const handleUpdate = (data) => {
  console.log(data)
}
</script>
```

### Provide/Inject

```vue
<!-- Parent.vue -->
<script setup>
import { provide } from 'vue'

provide('theme', 'dark')
provide('user', { name: 'John' })
</script>

<!-- Child.vue -->
<script setup>
import { inject } from 'vue'

const theme = inject('theme')
const user = inject('user')
</script>
```

### Refs (Refs de Plantilla)

```vue
<script setup>
import { ref, onMounted } from 'vue'

const inputRef = ref(null)

onMounted(() => {
  inputRef.value.focus()
})
</script>

<template>
  <input ref="inputRef" type="text" />
</template>
```

## Manejo de Eventos

```html
<!-- Evento de clic -->
<button @click="handleClick">Clicar</button>

<!-- Objeto evento -->
<button @click="handleClick($event)">Clicar</button>

<!-- Múltiples manejadores -->
<button @click="handler1(), handler2()">Clicar</button>

<!-- Modificadores de eventos -->
<button @click.stop="handleClick">Detener propagación</button>
<button @click.prevent="handleSubmit">Prevenir defecto</button>
<button @click.stop.prevent="handleClick">Ambos</button>

<!-- Modificadores de teclado -->
<input @keyup.enter="submit" />
<input @keyup.ctrl.enter="submit" />

<!-- Modificadores de mouse -->
<button @click.left="leftClick">Izquierdo</button>
<button @click.right="rightClick">Derecho</button>
```

## Vinculación de Entrada de Formulario

```vue
<template>
  <form @submit.prevent="submitForm">
    <!-- Entrada de texto -->
    <input v-model="form.name" type="text" />
    
    <!-- Textarea -->
    <textarea v-model="form.message"></textarea>
    
    <!-- Casilla de verificación -->
    <input v-model="form.checked" type="checkbox" />
    
    <!-- Múltiples casillas -->
    <input v-model="form.colors" value="red" type="checkbox">Rojo
    <input v-model="form.colors" value="blue" type="checkbox">Azul
    
    <!-- Radio -->
    <input v-model="form.gender" value="male" type="radio">Masculino
    <input v-model="form.gender" value="female" type="radio">Femenino
    
    <!-- Select -->
    <select v-model="form.country">
      <option value="cn">China</option>
      <option value="us">EE.UU.</option>
    </select>
    
    <!-- Modificadores -->
    <input v-model.lazy="form.name" />
    <input v-model.number="form.age" />
    <input v-model.trim="form.nickname" />
    
    <button type="submit">Enviar</button>
  </form>
</template>

<script setup>
import { reactive } from 'vue'

const form = reactive({
  name: '',
  message: '',
  checked: false,
  colors: [],
  gender: '',
  country: '',
  age: 0,
  nickname: ''
})

const submitForm = () => {
  console.log(form)
}
</script>
```

## Propiedades Computadas

```javascript
import { ref, computed } from 'vue'

const firstName = ref('John')
const lastName = ref('Doe')

// Computado - cacheado, solo reevalúa cuando cambian las dependencias
const fullName = computed(() => {
  return `${firstName.value} ${lastName.value}`
})

// Método - reevalúa cada vez
const getFullName = () => {
  return `${firstName.value} ${lastName.value}`
}
```

## Observadores

```javascript
import { ref, watch } from 'vue'

const count = ref(0)

// Observador simple
watch(count, (newValue, oldValue) => {
  console.log(`Cambió de ${oldValue} a ${newValue}`)
})

// Observador profundo
const state = ref({ name: 'Vue' })
watch(state, (newValue) => {
  console.log('Estado cambiado:', newValue)
}, { deep: true })

// Observar múltiples fuentes
const firstName = ref('John')
const lastName = ref('Doe')
watch([firstName, lastName], ([newFirst, newLast]) => {
  console.log(`Nombre: ${newFirst} ${newLast}`)
})
```

## Ganchos de Ciclo de Vida

### Vue 3 Composition API

| Ganchos | Descripción |
|------|-------------|
| `setup()` | Llamado antes que otros ganchos, para configuración |
| `onMounted()` | Después de que el componente se monta |
| `onUpdated()` | Después de que el componente se actualiza |
| `onUnmounted()` | Después de que el componente se desmonta |
| `onErrorCaptured()` | Cuando se captura un error |

```javascript
import { 
  onMounted, 
  onUpdated, 
  onUnmounted 
} from 'vue'

export default {
  setup() {
    onMounted(() => {
      console.log('¡Montado!')
    })
    onUpdated(() => {
      console.log('¡Actualizado!')
    })
    onUnmounted(() => {
      console.log('¡Desmontado!')
    })
  }
}
```

### Vue 2 Options API

| Ganchos | Descripción |
|------|-------------|
| `beforeCreate` | Antes de que se cree la instancia |
| `created` | Después de que se crea la instancia |
| `beforeMount` | Antes del montaje |
| `mounted` | Después del montaje |
| `beforeUpdate` | Antes de la actualización |
| `updated` | Después de la actualización |
| `beforeDestroy` | Antes de la destrucción |
| `destroyed` | Después de la destrucción |

## Vue Router

### Configuración Básica

```javascript
// router/index.js
import { createRouter, createWebHistory } from 'vue-router'
import Home from '../views/Home.vue'
import About from '../views/About.vue'
import User from '../views/User.vue'

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About },
  { path: '/user/:id', component: User, props: true }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```

### Navegación

```vue
<template>
  <!-- Navegación programática -->
  <button @click="$router.push('/about')">Acerca de</button>
  <button @click="$router.back()">Atrás</button>
  <button @click="$router.forward()">Adelante</button>
  
  <!-- Enlace de router -->
  <router-link to="/">Inicio</router-link>
  <router-link to="/about">Acerca de</router-link>
  <router-link :to="`/user/${userId}`">Usuario</router-link>
</template>

<script setup>
import { useRouter } from 'vue-router'

const router = useRouter()
const userId = 123

const navigate = () => {
  router.push('/about')
}
</script>
```

### Información de Ruta

## Gestión de Estado con Pinia

### Definición de Store

```javascript
// stores/user.js
import { defineStore } from 'pinia'

export const useUserStore = defineStore('user', {
  state: () => ({
    name: 'John',
    email: 'john@example.com',
    age: 25
  }),
  
  getters: {
    isAdult: (state) => state.age >= 18,
    displayName: (state) => `${state.name} (${state.age})`
  },
  
  actions: {
    updateName(newName) {
      this.name = newName
    },
    async fetchUser() {
      const data = await fetch('/api/user')
      const user = await data.json()
      this.$patch(user)
    }
  }
})
```

### Usar Store

```vue
<script setup>
import { storeToRefs } from 'pinia'
import { useUserStore } from '../stores/user'

const userStore = useUserStore()

// Acceder al estado directamente
console.log(userStore.name)

// Usar storeToRefs para reactividad
const { name, email } = storeToRefs(userStore)

// Llamar acciones
userStore.updateName('Jane')

// Acción asíncrona
userStore.fetchUser()
</script>
```

```javascript

export default {

    const count = ref(0)
    return { count }
  }
}
```

### 2. Usar Script Setup

```vue
<script setup>
<!-- Sintaxis más concisa -->
</script>
```

### 3. Definir Props con Validación

```javascript
defineProps({
  title: {
    type: String,
    required: true,
    default: 'Título por Defecto'
  },
  count: {
    type: Number,
    default: 0
  }
})
```

### 4. Usar Computado para Estado Derivado

```javascript
const firstName = ref('John')
const lastName = ref('Doe')

// Bueno - computado
const fullName = computed(() => `${firstName.value} ${lastName.value}`)

// Malo - método
const getFullName = () => `${firstName.value} ${lastName.value}`
```

### 5. Mantener Componentes Pequeños y Enfocados


```vue

  import('./HeavyComponent.vue')

    <AsyncComponent />
    <template #fallback>
```
