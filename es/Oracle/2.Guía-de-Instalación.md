# Oracle Guía de Instalación

## Requisitos Previos

- **Sistema Operativo**: Oracle Linux, RHEL, Windows Server
- **Memoria**: Mínimo 2GB RAM (recomendado 8GB+)
- **Espacio en Disco**: Mínimo 8GB para software

---

## Instalación en Linux

### 1. Crear usuario oracle

```bash
groupadd -g 54321 oinstall
groupadd -g 54322 dba
useradd -u 54321 -g oinstall -G dba oracle
passwd oracle
```

### 2. Configurar parámetros del kernel

```bash
kernel.shmmax = 4294967296
kernel.shmall = 2097152
```

### 3. Ejecutar instalador

```bash
./runInstaller
```

---

## Después de la instalación

```sql
sqlplus / as sysdba
SELECT name, open_mode FROM v$database;
```
