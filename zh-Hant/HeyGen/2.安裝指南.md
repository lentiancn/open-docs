# HeyGen 安裝指南

## 帳戶註冊與登入

### 訪問官方網站

訪問 HeyGen 官方網站 https://heygen.com ，點擊頁面右上角的「Get Started For Free」或「註冊」按鈕開始建立帳戶。您可以使用電子郵箱直接註冊，也可以選擇使用 Google 帳戶快速登入。註冊過程簡單快捷，只需填寫基本資訊並驗證郵箱即可完成。

首次登入時，系統會引導您完成基礎設置，包括選擇使用語言、熟悉介面佈局和了解核心功能。建議新用戶先完成引導教程，了解平台的基本操作流程，為後續的視頻創作打下基礎。

### 帳戶類型選擇

HeyGen 提供四種主要的帳戶類型：免費版（Free）、創作者版（Creator）、專業版（Pro）和企業版（Business）。免費版每月可生成 3 個視頻，最長 3 分鐘，適合初步體驗平台功能。創作者版每月 $29，無視頻時長限制，支援更多高級功能。專業版每月 $99，提供 10 倍的高級功能使用量和 4K 視頻導出。企業版每月 $149，每增加一個席位額外 $20，適合團隊協作使用。

## 介面導航

### 主介面概覽

登入後，您會看到 HeyGen 的主介面。左側是導航菜單，包括創建視頻、數字人管理、場景選擇、項目管理和設置等主要功能模組。中央區域是內容展示區，顯示您的視頻項目列表和模板推薦。右側面板提供快捷設置選項，包括語音選擇、語言設置和預覽功能。

頂部導航欄包含帳戶資訊、使用配額顯示和幫助入口。底部狀態欄顯示當前任務進度和系統通知。熟悉介面佈局後，您可以快速導航到所需功能，提高工作效率。

### 功能模組介紹

創建視頻是核心功能，點擊後進入視頻創作工作區。數字人管理允許您瀏覽、創建和編輯 AI 數字人形象。場景選擇提供豐富的背景模板和自定義選項。項目管理幫助您組織和管理視頻內容，支援檔案夾分類和標籤管理。

## 快速創建第一個視頻

### 選擇數字人形象

點擊「創建視頻」後，首先需要選擇數字人形象。您可以在預設形象庫中選擇，涵蓋不同性別、年齡、膚色和風格的 500 多個公共數字人。如果您需要個性化形象，可以點擊「自定義數字人」上傳自己的照片，系統會基於照片生成專屬的數字人形象。

選擇數字人時，可以預覽其不同角度和表情的示例視頻，確保選擇最適合您內容需求的形象。選定後，該數字人將成為當前視頻的主角。

### 輸入視頻內容

在腳本編輯區域輸入您希望數字人播報的內容。您可以直接輸入文字，或上傳預先準備好的音頻檔案。文字輸入時，系統會即時顯示預計的視頻時長。專業術語和人名可以通過平台的发音糾正功能手動調整，確保 AI 配音準確無誤。

HeyGen 支援多種輸入方式：純文字腳本、音頻檔案（MP3、WAV）、PPT 或 PDF 文檔導入。對於長篇內容，系統會自動將其分割成適合視頻呈現的段落。

### 選擇語音和語言

在右側面板選擇語音選項。HeyGen 提供超過 1,000 種 AI 語音，涵蓋 175 種以上語言和方言。您可以預覽每個語音的示例，選擇最符合內容語調的音色。語音參數包括語速、音調和停頓間隔，都可以根據需要進行微調。

多語言支援是 HeyGen 的核心優勢。如果您需要創建多語言版本，只需在設置中選擇目標語言，系統會自動匹配相應的語音並進行翻譯。翻譯功能保留原始說話者的聲音特徵和表達風格。

### 添加背景和場景

選擇合適的背景可以顯著提升視頻的專業感。HeyGen 提供超過 75 種預設場景，涵蓋辦公室、演播室、城市風光，自然環境等多種風格。您也可以上傳自定義背景圖片作為視頻背景。

動態元素如文字標題、圖標動畫、數據圖表等可以增強資訊傳達效果。所有元素都支援位置、大小、顏色和動畫效果的自定義調整。

### 生成和導出視頻

完成所有設置後，點擊「生成」按鈕提交視頻創建任務。視頻生成時間取決於視頻長度、複雜度和伺服器負載，一般在幾分鐘內完成。您可以在任務列表中查看進度。

視頻生成完成後，您可以預覽最終效果。如需調整，可以返回編輯界面修改腳本或設置。滿意後，選擇合適的格式和解析度導出。HeyGen 支援 720p、1080p 和 4K 多種解析度，以及 MP4、MOV 等常用視頻格式。

## 高級功能使用

### 自定義數字人形象

點擊導航欄的「數字人」選項進入形象管理頁面。點擊「創建自定義形象」上傳人物照片，建議使用正面高清照片以獲得最佳效果。系統會自動分析照片特徵，生成對應的數字人形象。

自定義形象支援豐富的參數調整，包括發型、面部特徵、服裝風格、配飾等。您可以保存多個形象設置，方便在不同視頻中快速切換。企業用戶可以定製專屬的品牌形象，用於企業宣傳和客戶服務場景。

### 團隊協作功能

企業版用戶可以建立團隊帳戶，邀請團隊成員加入。管理員可以設置不同成員的使用權限，分配視頻配額和管理功能。團隊成員建立的資產可以設置為共享或私有，促進知識積累和復用。

企業帳戶還可以定製品牌元素，包括 Logo、專屬顏色和片頭片尾。所有使用企業品牌創建的視頻都會自動添加品牌標識，確保品牌一致性。

### API 整合

對於需要將 HeyGen 整合到現有工作流程的企業用戶，平台提供完整的 API 介面。 API 支援按需付費模式，包括視頻代理、視頻翻譯、文字轉語音等功能。

通過 API，您可以自動化視頻創建流程，實現大規模內容生產。詳細的 API 文檔和整合指南可以在開發者文檔中找到。

## 帳戶管理

### 訂閱計劃管理

在帳戶設置頁面可以查看當前訂閱狀態和可用配額。您可以隨時升級或降級訂閱計劃。升級立即生效，降級將在當前計費週期結束後生效。

取消訂閱後，您仍然可以在當前計費週期結束前使用訂閱功能。計費週期結束後，帳戶將自動轉為免費版。

### 安全性設置

建議開啟雙重驗證（MFA）增強帳戶安全性。企業用戶可以配置 SAML/SSO 實現單一登入，滿足企業安全策略要求。帳戶設置中還可以管理活動會話，查看登入歷史記錄。
