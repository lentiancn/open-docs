# Jira

Jira는 Atlassian이 개발한 프로젝트 및 이슈 추적 소프트웨어로, 전 세계 300,000개 이상의 기업에 프로젝트 관리 솔루션을 제공합니다.

## 빠른 시작

Atlassian 웹사이트를 방문하여 가입하고 팀에 적합한 요금제를 선택하세요. Jira 사이트를 빠르게 생성할 수 있습니다.

## 주요 기능

프로젝트 관리 및 이슈 추적, 애자일 개발 지원(Scrum, Kanban), 팀 협업 및 통신, Rovo AI 어시스턴트 통합, 다양한 타사 통합, 강력한 자동화 규칙.
