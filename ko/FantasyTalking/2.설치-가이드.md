# FantasyTalking 설치 가이드

## 요구 사항

FantasyTalking을 설치하기 전에 개발 환경이 다음 요구 사항을 충족하는지 확인하세요. 운영 체제의 경우 프로젝트는 Windows, macOS, Linux를 지원합니다. 최상의 성능을 위해 64비트 시스템 사용을 권장합니다. Node.js 버전 14.0 이상이 필요하며 Python 버전 3.7 이상이 필요합니다. Python 백엔드를 사용하는 경우 FFmpeg 오디오 처리 도구도 설치해야 합니다.

하드웨어의 경우 원활한 음성 합성 경험을 위해 시스템에 최소 4GB RAM과 듀얼 코어 프로세서가 권장됩니다. 실시간 음성 합성이 필요한 애플리케이션의 경우 I/O 지연을 줄이기 위해 SSD 스토리지 사용을 권장합니다.

## 설치 방법

### npm 사용 (Node.js)

```bash
npm install fantasy-talking
npm install fantasy-talking-ui
```

### pip 사용 (Python)

```bash
pip install fantasy-talking
pip install fantasy-talking[all]
```

## 빠른 시작

```javascript
const client = new FantasyTalking({
  apiKey: 'your-api-key',
  voice: { style: 'neutral', language: 'ko-KR' }
});

const audio = await client.speak('안녕하세요, 테스트입니다');
```

## 설정

| 매개변수 | 유형 | 기본값 | 설명 |
|---------|------|--------|------|
| apiKey | string | - | API 키 (필수) |
| voice.style | string | neutral | 음성 스타일 |
| voice.language | string | ko-KR | 언어 코드 |
| voice.speed | number | 1.0 | 속도 |

## 배포

```bash
docker pull fantasytalking/server:latest
docker run -d -p 8080:8080 -e API_KEY=your-api-key fantasytalking/server:latest
```
