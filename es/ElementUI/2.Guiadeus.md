# Guía de uso de Element UI

Esta guía cubre cómo usar los componentes de Element UI en tu aplicación Vue 2.

## Uso básico

### Hello World

```vue
<template>
  <div>
    <el-button>Botón predeterminado</el-button>
    <el-button type="primary">Botón primario</el-button>
  </div>
</template>

<script>
export default {
  name: 'App'
}
</script>
```

## Categorías de componentes

Element UI proporciona estas categorías de componentes:

| Categoría | Componentes |
|-----------|-------------|
| Básico | Layout, Button, Icon, Link |
| Formulario | Input, Select, Radio, Checkbox, Switch, Slider, TimePicker, DatePicker |
| Datos | Table, Tree, Pagination, Tag |
| Notificaciones | Alert, Message, Notification, Progress |
| Navegación | Menu, Tabs, Breadcrumb, Steps, Dropdown |
| Otros | Dialog, Drawer, Card, Collapse, Timeline |

### Entrada Input

```vue
<template>
  <div>
    <!-- Básico -->
    <el-input v-model="input" placeholder="Ingresar texto"></el-input>
    
    <!-- Con icono -->
    <el-input
      v-model="input1"
      placeholder="Buscar"
      prefix-icon="el-icon-search">
    </el-input>
    
    <!-- Contraseña -->
    <el-input
      v-model="input2"
      placeholder="Contraseña"
      show-password>
    </el-input>
    
    <!-- Área de texto -->
    <el-input
      v-model="textarea"
      type="textarea"
      :rows="3"
      placeholder="Ingresar descripción">
    </el-input>
    
    <!-- Deshabilitado -->
    <el-input disabled placeholder="Deshabilitado"></el-input>
  </div>
</template>

<script>
export default {
  data() {
    return {
      input: '',
      input1: '',
      input2: '',
      textarea: ''
    }
  }
}
</script>
```

```vue
<template>

    handleDelete(index, row) {
      console.log(index, row);
    }
  }
}
</script>
```

### Formulario Form

```vue
<template>
  <el-form
    :model="ruleForm"
    :rules="rules"
    ref="ruleForm"
    label-width="120px"
  >
    <el-form-item label="Nombre" prop="name">
      <el-input v-model="ruleForm.name"></el-input>
    </el-form-item>
    
    <el-form-item label="Correo" prop="email">
      <el-input v-model="ruleForm.email"></el-input>
    </el-form-item>
    
    <el-form-item label="Género" prop="gender">
      <el-radio-group v-model="ruleForm.gender">
        <el-radio label="male">Masculino</el-radio>
        <el-radio label="female">Femenino</el-radio>
      </el-radio-group>
    </el-form-item>
    
    <el-form-item label="Afición" prop="hobby">
      <el-checkbox-group v-model="ruleForm.hobby">
        <el-checkbox label="Lectura" name="hobby"></el-checkbox>
        <el-checkbox label="Deportes" name="hobby"></el-checkbox>
        <el-checkbox label="Música" name="hobby"></el-checkbox>
      </el-checkbox-group>
    </el-form-item>
    
    <el-form-item label="País" prop="country">
      <el-select v-model="ruleForm.country" placeholder="Seleccionar país">
        <el-option label="España" value="spain"></el-option>
        <el-option label="México" value="mexico"></el-option>
        <el-option label="Argentina" value="argentina"></el-option>
      </el-select>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm('ruleForm')">Enviar</el-button>
      <el-button @click="resetForm('ruleForm')">Restablecer</el-button>
    </el-form-item>
  </el-form>
</template>

<script>
export default {
  data() {
    return {
      ruleForm: {
        name: '',
        email: '',
        gender: '',
        hobby: [],
        country: ''
      },
      rules: {
        name: [
          { required: true, message: 'Por favor ingrese el nombre', trigger: 'blur' }
        ],
        email: [
          { required: true, message: 'Por favor ingrese el correo', trigger: 'blur' },
          { type: 'email', message: 'Formato de correo inválido', trigger: 'blur' }
        ],
        gender: [
          { required: true, message: 'Por favor seleccione el género', trigger: 'change' }
        ],
        country: [
          { required: true, message: 'Por favor seleccione el país', trigger: 'change' }
        ]
      }
    };
  },
  methods: {
    submitForm(formName) {
      this.$refs[formName].validate((valid) => {
        if (valid) {
          alert('¡Enviado!');
        } else {
          console.log('¡Error al enviar!!');
          return false;
        }
      });
    },
    resetForm(formName) {
      this.$refs[formName].resetFields();
    }
  }
}
</script>
```

## Eventos

```vue
<template>
  <el-button @click="handleClick">Haz clic</el-button>
  
  <el-input
    @focus="handleFocus"
    @blur="handleBlur"
    @change="handleChange"
    @input="handleInput">
  </el-input>
</template>

<script>
export default {
  methods: {
    handleClick() {
      console.log('¡Clickeado!')
    },
    handleFocus() {
      console.log('¡Enfocado!')
    },
    handleBlur() {
      console.log('¡Desenfocado!')
    },
    handleChange(value) {
      console.log('Cambiado:', value)
    },
    handleInput(value) {
      console.log('Input:', value)
    }
  }
}
</script>
```

## Internacionalización

### Establecer idioma

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import locale from 'element-ui/lib/locale/lang/es'

Vue.use(ElementUI, { locale })
```

## Mejores prácticas

### 1. Usar componentes localmente

```javascript
// Bueno - usar localmente
export default {
  components: {
    [Button.name]: Button,
    [Input.name]: Input
  }
}
```

### 2. Validación de formularios

```javascript
const validatePass = (rule, value, callback) => {
  if (value === '') {
    callback(new Error('Por favor ingrese la contraseña'));
  } else {
    if (this.ruleForm.checkPass !== '') {
      this.$refs.ruleForm.validateField('checkPass');
    }
    callback();
  }
};
```

### 3. Rendimiento de tablas

```vue
<el-table
  :data="tableData"
  :default-sort = "{prop: 'date', order: 'descending'}"
  >
</el-table>
```

## Problemas comunes

### Modificadores de eventos

```vue
<!-- Usar .native para eventos nativos -->
<el-input @keyup.enter.native="handleEnter"></el-input>
```

### Carga diferida

```vue
<el-select
  v-model="value"
  filterable
  remote
  :remote-method="remoteMethod"
  :loading="loading">
  <el-option
    v-for="item in options"
    :key="item.value"
    :label="item.label"
    :value="item.value">
  </el-option>
</el-select>
```
