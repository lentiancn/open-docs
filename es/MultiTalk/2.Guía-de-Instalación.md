# MultiTalk Guía de Instalación

Guía de instalación y configuración de MultiTalk.

## Requisitos Previos

- Windows 10/11, macOS 10.15+ o Linux
- Micrófono y auriculares/altavoz
- Conexión a Internet

## Instalación

### Windows

1. Descargue el instalador
2. Ejecute el instalador
3. Siga el asistente
4. Inicie MultiTalk

### macOS

1. Descargue el archivo .dmg
2. Abra el instalador
3. Arrastre MultiTalk a Aplicaciones
4. Inicie MultiTalk

## Configuración

### Configuración de Audio

1. Abra Configuración > Audio
2. Seleccione su dispositivo de entrada
3. Seleccione su dispositivo de salida
4. Pruebe los niveles

### Push-to-talk

1. Vaya a Configuración > Atajos
2. Active push-to-talk
3. Presione la combinación de teclas

## Solución de Problemas

### Sin entrada de audio

**Solución:**
- Verifique permisos del micrófono
- Verifique selección de dispositivo

### Problemas de eco

**Solución:**
- Use auriculares
- Baje el volumen
