## 1. 栅格系统

### 1.1 基本原理

Bootstrap 的栅格系统是响应式、移动设备优先的 flexbox 栅格系统。

- 12 列栅格
- 5 个响应式断点
- 自动布局

### 1.2 栅格类

| 设备 | 类名前缀 | 最小宽度 |
|------|----------|----------|
| 超小设备 | col- | 无 |
| 小设备 | col-sm | 576px |
| 中等设备 | col-md | 768px |
| 大设备 | col-lg | 992px |
| 超大设备 | col-xl | 1200px |
| 超超大设备 | col-xxl | 1400px |

### 1.3 基本用法

```html
<div class="container">
    <div class="row">
        <div class="col-12 col-md-6 col-lg-4">
            <p>列 1</p>
        </div>
        <div class="col-12 col-md-6 col-lg-4">
            <p>列 2</p>
        </div>
        <div class="col-12 col-md-6 col-lg-4">
            <p>列 3</p>
        </div>
    </div>
</div>
```

### 1.4 列偏移

```html
<!-- 偏移 4 列 -->
<div class="col-md-4 offset-md-4"></div>

<!-- 居中布局 -->
<div class="col-md-6 offset-md-3"></div>
```

### 1.5 栅格间距

```html
<!-- 无间距 -->
<div class="row g-0">
    <div class="col-6">无间距</div>
    <div class="col-6">无间距</div>
</div>

<!-- 小间距 g-1, g-2, g-3, g-4, g-5 -->
<div class="row g-3">
    <div class="col-4">间距</div>
    <div class="col-4">间距</div>
    <div class="col-4">间距</div>
</div>
```

### 1.6 对齐方式

#### 垂直对齐

```html
<!-- 顶部对齐 -->
<div class="row align-items-start">

<!-- 居中对齐 -->
<div class="row align-items-center">

<!-- 底部对齐 -->
<div class="row align-items-end">

<!-- 单列对齐 -->
<div class="row">
    <div class="col align-self-start">顶部</div>
    <div class="col align-self-center">居中</div>
    <div class="col align-self-end">底部</div>
</div>
```

#### 水平对齐

```html
<div class="row justify-content-start">    <!-- 左对齐 -->
<div class="row justify-content-center">  <!-- 居中 -->
<div class="row justify-content-end">    <!-- 右对齐 -->
<div class="row justify-content-around">  <!-- 两端对齐 -->
<div class="row justify-content-between"> <!-- 等间距 -->
```

## 2. 容器和布局

### 2.1 容器类

| 类名 | 特点 |
|------|------|
| `.container` | 固定宽度，响应式最大宽度 |
| `.container-fluid` | 100% 宽度 |
| `.container-{breakpoint}` | 特定断点起 100% 宽度 |

### 2.2 容器最大宽度

| 断点 | 容器最大宽度 |
|------|--------------|
| 超小 | 100% |
| 小 (≥576px) | 540px |
| 中 (≥768px) | 720px |
| 大 (≥992px) | 960px |
| 超大 (≥1200px) | 1140px |
| 超超大 (≥1400px) | 1320px |

### 2.3 布局示例

```html
<!-- 全宽布局 -->
<div class="container-fluid">
    <h1>全宽布局</h1>
</div>

<!-- 居中内容区 -->
<div class="container">
    <div class="row">
        <div class="col-md-8">主内容</div>
        <div class="col-md-4">侧边栏</div>
    </div>
</div>
```

## 4. 颜色系统

### 4.1 主题颜色

Bootstrap 使用 Sass 变量定义主题颜色：

```scss
$primary:   #0d6efd;
$secondary: #6c757d;
$success:   #198754;
$danger:    #dc3545;
$warning:   #ffc107;
$info:      #0dcaf0;
$light:     #f8f9fa;
$dark:      #212529;
```

### 4.2 颜色使用

```html
<!-- 背景颜色 -->
<div class="bg-primary text-white">主要背景</div>
<div class="bg-success text-white">成功背景</div>

<!-- 文本颜色 -->
<p class="text-primary">主要文本</p>
<p class="text-danger">危险文本</p>

<!-- 边框颜色 -->
<div class="border border-primary">主要边框</div>

<!-- 按钮颜色 -->
<button class="btn btn-primary">主要按钮</button>
<button class="btn btn-outline-secondary">轮廓按钮</button>
```

### 4.3 渐变背景

```html
<div class="bg-primary bg-gradient">渐变背景</div>
<div class="bg-danger bg-gradient">危险渐变</div>
```

## 6. 表单组件

### 6.1 基本表单

```html
<form>
    <div class="mb-3">
        <label for="exampleInputEmail1" class="form-label">邮箱地址</label>
        <input type="email" class="form-control" id="exampleInputEmail1" placeholder="name@example.com">
        <div class="form-text">我们不会分享您的邮箱。</div>
    </div>
    <div class="mb-3">
        <label for="exampleInputPassword1" class="form-label">密码</label>
        <input type="password" class="form-control" id="exampleInputPassword1">
    </div>
    <div class="mb-3 form-check">
        <input type="checkbox" class="form-check-input" id="exampleCheck1">
        <label class="form-check-label" for="exampleCheck1">记住我</label>
    </div>
    <button type="submit" class="btn btn-primary">提交</button>
</form>
```

### 6.2 表单尺寸

```html
<!-- 大尺寸 -->
<input class="form-control form-control-lg" type="text" placeholder=".form-control-lg">

<!-- 默认尺寸 -->
<input class="form-control" type="text" placeholder="默认">

<!-- 小尺寸 -->
<input class="form-control form-control-sm" type="text" placeholder=".form-control-sm">
```

### 6.3 禁用和只读

```html
<input class="form-control" type="text" placeholder="禁用" disabled>
<input class="form-control" type="text" value="只读" readonly>
```

### 6.4 选择框

```html
<select class="form-select" aria-label="Default select example">
    <option selected>打开此选择菜单</option>
    <option value="1">选项 1</option>
    <option value="2">选项 2</option>
    <option value="3">选项 3</option>
</select>
```

### 6.5 多选和文件输入

```html
<!-- 多选 -->
<div class="form-check">
    <input class="form-check-input" type="checkbox" value="" id="flexCheckDefault">
    <label class="form-check-label" for="flexCheckDefault">

<!-- 单选 -->
```
