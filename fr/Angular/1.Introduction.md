# Angular Aperçu

Plateforme de développement web moderne.

## Qu'est-ce qu'Angular ?

Angular est une plateforme de développement pour créer des applications web mobiles et de bureau, développée par Google. Utilise TypeScript/JavaScript et d'autres langages.

### Fonctionnalités principales

- **Basé sur les composants** - Architecture pilotée par les composants
- **TypeScript** - Support du typage fort
- **Liaison bidirectionnelle** - Synchronisation des données simplifiée
- **Injection de dépendances** - Système puissant de gestion des dépendances
- **Routage** - Solution de routage complète
- **Formulaires** - Formulaires réactifs et pilotés par les modèles
- **Client HTTP** - Services HTTP complets
- **Rendu côté serveur** - Support SSR
