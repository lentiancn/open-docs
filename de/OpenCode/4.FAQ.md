# OpenCode FAQ

Häufig gestellte Fragen.

## Installation

### Q: Installation fehlgeschlagen?

**Lösungen**:
1. Systemanforderungen prüfen (moderner Terminal-Emulator erforderlich)
2. Netzwerkverbindung prüfen
3. Installationsskript versuchen: `curl -fsSL https://opencode.ai/install | bash`
4. Fehlermeldung prüfen und erneut versuchen

### Q: Installation auf Windows nicht möglich?

**Lösungen**:
1. Chocolatey versuchen: `choco install opencode`
2. Oder Scoop verwenden: `scoop install opencode`
3. Oder NPM verwenden: `npm install -g opencode-ai`

### Q: Überprüfung der Installation fehlgeschlagen?

**Lösungen**:
1. Prüfen ob opencode im PATH ist
2. Terminal neu starten
3. Prüfen ob Installation erfolgreich war

## Nutzung

### Q: Verbindung zu LLM nicht möglich?

**Lösungen**:
1. Befehl `/connect` ausführen
2. opencode.ai/auth besuchen um sich anzumelden
3. Rechnungsinformationen hinzufügen und API-Schlüssel erhalten
4. API-Schlüssel in Terminal einfügen

### Q: Antwort ist langsam?

**Lösungen**:
1. Netzwerkverbindung prüfen
2. Versuchen zu einem anderen LLM-Anbieter zu wechseln
3. Kontextlänge der Konversation reduzieren

### Q: Wie Modelle wechseln?

Befehl `/connect` verwenden um anderen LLM-Anbieter auszuwählen.

### Q: Welche Modelle werden unterstützt?

Unterstützt 75+ LLM-Anbieter:
- OpenAI (GPT-4, Claude, etc.)
- Anthropic (Claude)
- Google (Gemini)
- Lokale Modelle
- Siehe [Models.dev](https://models.dev)

## Funktionen

### Q: Was ist AGENTS.md?

AGENTS.md ist eine Datei, die bei der Initialisierung von OpenCode erstellt wird und die Projektstruktur und Codierungskonventionen enthält, um OpenCode zu helfen das Projekt zu verstehen.

### Q: Unterschied zwischen Plan- und Build-Modus?

- **Plan-Modus**: Erstellt nur Pläne, ändert keinen Code
- **Build-Modus**: Führt Änderungen direkt aus

**Tab** drücken um zwischen Modi zu wechseln.

### Q: Wie Sitzungen teilen?

Befehl `/share` verwenden um Teilen-Link zu erstellen.

## Datenschutz

### Q: Speichert OpenCode meinen Code?

Nein. OpenCode speichert keinen Code oder Kontextdaten, sicher für sensible Umgebungen.

### Q: Wie Datenschutz schützen?

- Keine sensiblen Informationen in Konversationen eingeben
- Lokale Modelle verwenden
- Sitzungen regelmäßig löschen
