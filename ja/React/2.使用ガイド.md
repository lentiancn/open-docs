# React 使用ガイド

---

## コンポーネント

```jsx
function Welcome() {
  return <h1>Hello World</h1>;
}
```

---

## ステート (useState)

```jsx
import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);
  return <button onClick={() => setCount(count + 1)}>{count}</button>;
}
```

---

## エフェクト (useEffect)

