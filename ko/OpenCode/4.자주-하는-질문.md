# OpenCode 자주 묻는 질문

자주 묻는 질문과 답변.

## 설치

### Q: 설치 실패?

**해결책**:
1. 시스템 요구 사항 확인 (최신 터미널 에뮬레이터 필요)
2. 네트워크 연결 확인
3. 설치 스크립트 시도: `curl -fsSL https://opencode.ai/install | bash`
4. 오류 메시지 확인하고 다시 시도

### Q: Windows에 설치 불가?

**해결책**:
1. Chocolatey 시도: `choco install opencode`
2. 또는 Scoop 사용: `scoop install opencode`
3. 또는 NPM 사용: `npm install -g opencode-ai`

### Q: 설치 확인 실패?

**해결책**:
1. opencode가 PATH에 추가되었는지 확인
2. 터미널 재시작
3. 설치가 성공했는지 확인

## 사용

### Q: LLM에 연결 불가?

**해결책**:
1. `/connect` 명령 실행
2. opencode.ai/auth 방문하여 로그인
3. 결제 정보 추가하고 API 키 얻기
4. 터미널에 API 키 붙여넣기

### Q: 응답 느림?

**해결책**:
1. 네트워크 연결 확인
2. 다른 LLM 제공자로 전환 시도
3. 대화 컨텍스트 길이 줄이기

### Q: 모델切换 방법?

`/connect` 명령을 사용하여 다른 LLM 제공자 선택.

### Q: 어떤 언어를 지원합니까?

75+ LLM 제공자 지원:
- OpenAI (GPT-4, Claude 등)
- Anthropic (Claude)
- Google (Gemini)
- 로컬 모델
- [Models.dev](https://models.dev) 참조

## 기능

### Q: AGENTS.md란?

AGENTS.md는 OpenCode 초기화 시 생성되는 파일로, 프로젝트 구조와 코딩 규칙을 포함하여 OpenCode가 프로젝트를 이해하는 데 도움이 됩니다.

### Q: Plan 모드와 Build 모드의 차이?

- **Plan 모드**: 계획만 생성, 코드 변경 안 함
- **Build 모드**: 직접 변경 실행

**Tab**을 눌러 모드 전환.

### Q: 세션 공유 방법?

`/share` 명령으로 공유 링크 생성.

## 개인정보 보호

### Q: OpenCode가 내 코드를 저장합니까?

아니요. OpenCode는 코드나 컨텍스트 데이터를 저장하지 않아 민감한 환경에서도 안전하게 사용할 수 있습니다.

### Q: 개인정보를 보호하는 방법은?

- 대화에서 민감한 정보 입력 안 함
- 로컬 모델 사용
- 세션 정기적으로 지우기
