# Bootstrap FAQ

> Häufig gestellte Fragen

---

## Installationsprobleme

### CDN lädt nicht?

**Lösungen**:

1. Netzwerkverbindung prüfen
2. Alternativen CDN verwenden:
```html
<link href="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

### Stile funktionieren nicht nach npm Installation?

Stelle sicher, dass korrekt importiert wird:

```javascript
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap';
```

---

## Stilprobleme

### Komponenten-Stile funktionieren nicht?

**Häufige Ursachen**:

1. Bootstrap CSS nicht importiert
2. JavaScript nicht korrekt geladen (benötigt für einige Komponenten)
3. CSS-Priorität überschrieben

### Responsive funktioniert nicht?

Stelle sicher, dass Viewport-Meta-Tag hinzugefügt wurde:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

---

## Komponentenprobleme

### Dropdown funktioniert nicht?

**Checkliste**:

1. Bootstrap JS importieren (nicht nur CSS)
2. Korrektes HTML-Struktur verwenden
3. Auf JavaScript-Fehler prüfen

### Modal öffnet sich nicht?

Stelle sicher:
1. Bootstrap JS ist geladen
2. data-bs-toggle und data-bs-target Attribute sind korrekt
3. Modal existiert beim Laden der Seite

---

## Anpassungsprobleme

### Wie Theme-Farben anpassen?

**Methode 1: Sass verwenden**

```scss
$primary: #ff6600;
@import "bootstrap";
```

**Methode 2: CSS-Variablen verwenden**

```css
:root {
  --bs-primary: #ff6600;
}
```

---

## Kompatibilitätsprobleme

### IE 11 nicht unterstützt?

Bootstrap 5 unterstützt IE 11 nicht mehr. Wenn du IE 11-Unterstützung benötigst, verwende Bootstrap 4.

---

## Leistungsprobleme

### Wie Dateigröße reduzieren?

1. Nur benötigte Komponenten importieren (Sass)
2. Produktionsversionen verwenden (.min.css/.min.js)
3. Gzip-Komprimierung aktivieren

---

## Konflikte mit anderen Frameworks

### Mit Vue/React verwenden?

Bootstrap kann mit jedem JavaScript-Framework verwendet werden, aber beachte:

1. Einige Komponenten hängen von jQuery ab (Bootstrap 5 nicht mehr)
2. Framework-spezifische Bootstrap-Wrapper-Bibliotheken verwenden (z.B. react-bootstrap)
