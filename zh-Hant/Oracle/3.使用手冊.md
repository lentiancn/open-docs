# Oracle 使用手冊

## 連線到資料庫

### 使用 SQL*Plus

```bash
# 以 SYSDBA 連線
sqlplus / as sysdba

# 使用帳號密碼連線
sqlplus 使用者/密碼@資料庫
```

---

## 基本 SQL 操作

### 建立表格

```sql
CREATE TABLE 員工 (
    員工編號 NUMBER(10) PRIMARY KEY,
    姓名 VARCHAR2(50) NOT NULL,
    部門 VARCHAR2(50) NOT NULL,
    電子郵件 VARCHAR2(100),
    薪資 NUMBER(10, 2)
);
```

### 插入資料

```sql
INSERT INTO 員工 (員工編號, 姓名, 部門, 電子郵件, 薪資)
VALUES (1001, '王小明', '業務部', 'wang@company.com', 75000);
```

### 查詢資料

```sql
SELECT * FROM 員工;
SELECT 姓名, 部門, 薪資 FROM 員工 WHERE 部門編號 = 10;
```

## PL/SQL

### 儲存程序

```sql
CREATE OR REPLACE PROCEDURE 更新薪資 (
    p_員工編號 IN NUMBER,
    p_新薪資 IN NUMBER
) AS
BEGIN
    UPDATE 員工
    SET 薪資 = p_新薪資
    WHERE 員工編號 = p_員工編號;
    COMMIT;
END;
/
```
