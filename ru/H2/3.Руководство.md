# Руководство пользователя H2

## Базовые операции

### Создание базы данных

Встроенный режим (база данных в памяти):
```java
String url = "jdbc:h2:mem:testdb";
Connection conn = DriverManager.getConnection(url, "sa", "");
```

Файл базы данных:
```java
String url = "jdbc:h2:file:./data/testdb";
Connection conn = DriverManager.getConnection(url, "sa", "password");
```

### Создание таблицы

```sql
CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100) UNIQUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### Вставка данных

```sql
INSERT INTO users (name, email) VALUES ('Иван', 'ivan@пример.рф');
INSERT INTO users (name, email) VALUES ('Мария', 'maria@пример.рф');
```

### Запрос данных

```sql
SELECT * FROM users;
SELECT * FROM users WHERE name = 'Иван';
SELECT COUNT(*) FROM users;
```

### Обновление данных

```sql
UPDATE users SET email = 'новыйemail@пример.рф' WHERE id = 1;
```

### Удаление данных

```sql
DELETE FROM users WHERE id = 2;
```

## Примеры Java-кода

### Полный пример JDBC

```java
import java.sql.*;

public class H2Example {
    public static void main(String[] args) throws Exception {
        // Загрузить драйвер
        Class.forName("org.h2.Driver");
        
        // Подключиться к базе данных
        String url = "jdbc:h2:mem:testdb";
        Connection conn = DriverManager.getConnection(url, "sa", "");
        
        // Создать таблицу
        Statement stmt = conn.createStatement();
        stmt.execute("CREATE TABLE users (id INT PRIMARY KEY AUTO_INCREMENT, name VARCHAR(100))");
        
        // Вставить данные
        stmt.execute("INSERT INTO users (name) VALUES ('Иван')");
        stmt.execute("INSERT INTO users (name) VALUES ('Мария')");
        
        // Запросить данные
        ResultSet rs = stmt.executeQuery("SELECT * FROM users");
        while (rs.next()) {
            System.out.println(rs.getInt("id") + ": " + rs.getString("name"));
        }
        
        // Закрыть подключение
        rs.close();
        stmt.close();
        conn.close();
    }
}
```

### Использование PreparedStatement

```java
PreparedStatement pstmt = conn.prepareStatement("INSERT INTO users (name, email) VALUES (?, ?)");
pstmt.setString(1, "Петр");
pstmt.setString(2, "petr@пример.рф");
pstmt.executeUpdate();
pstmt.close();
```

### Использование транзакций

```java
conn.setAutoCommit(false);
try {
    Statement stmt = conn.createStatement();
    stmt.execute("INSERT INTO users (name) VALUES ('Алексей')");
    stmt.execute("INSERT INTO users (name) VALUES ('Елена')");
    conn.commit();
} catch (Exception e) {
    conn.rollback();
} finally {
    conn.setAutoCommit(true);
}
```

## Расширенные функции

### Переключение режима базы данных

H2 может имитировать SQL-синтаксис других баз данных:

```java
// Режим MySQL
String url = "jdbc:h2:mem:testdb;MODE=MySQL";

// Режим PostgreSQL
String url = "jdbc:h2:mem:testdb;MODE=PostgreSQL";

// Режим Oracle
String url = "jdbc:h2:mem:testdb;MODE=Oracle";
```

### Шифрование базы данных

```java
// Использовать зашифрованную базу данных
String url = "jdbc:h2:file:./data/encrypted;CIPHER=AES";
Connection conn = DriverManager.getConnection(url, "sa", "password");
```

### Функция трассировки

```java
// Включить SQL-трассировку
String url = "jdbc:h2:mem:testdb;TRACE_LEVEL_FILE=3";
```

### Запуск TCP-сервера

```java
import org.h2.tools.Server;

Server server = Server.createTcpServer().start();
System.out.println("H2 TCP Server started: " + server.getURL());
```

### Использование консоли H2

Запустить веб-консоль H2:

```bash
java -jar h2.jar -web
```

Затем откройте в браузере: http://localhost:8082

## Частые SQL-операторы

### Типы данных

H2 поддерживает множество типов данных, включая:
- Целочисленные типы: INT, INTEGER, BIGINT, SMALLINT, TINYINT
- Типы с плавающей точкой: FLOAT, REAL, DOUBLE, DECIMAL
- Строковые типы: VARCHAR, CHAR, CLOB
- Типы даты и времени: DATE, TIME, TIMESTAMP
- Логические типы: BOOLEAN
- Бинарные типы: BLOB, BINARY

### Операции с индексами

Создать индекс:
```sql
CREATE INDEX idx_name ON users(name);
```

Удалить индекс:
```sql
DROP INDEX idx_name;
```

### Операции с ограничениями

Добавить первичный ключ:
```sql
ALTER TABLE users ADD PRIMARY KEY (id);
```

Добавить внешний ключ:
```sql
ALTER TABLE orders ADD FOREIGN KEY (user_id) REFERENCES users(id);
```

### Просмотр структуры таблицы

```sql
DESCRIBE users;
-- или
SHOW COLUMNS FROM users;
```

### Просмотр всех таблиц

```sql
SHOW TABLES;
```

## Оптимизация производительности

### Пакетные операции

Использовать пакетные операции для повышения производительности:

```java
conn.setAutoCommit(false);
PreparedStatement pstmt = conn.prepareStatement("INSERT INTO users (name) VALUES (?)");
for (int i = 0; i < 1000; i++) {
    pstmt.setString(1, "Пользователь" + i);
    pstmt.addBatch();
}
pstmt.executeBatch();
conn.commit();
```

### Пул подключений

В производственных средах рекомендуется использовать пул подключений, например HikariCP:

```java
HikariConfig config = new HikariConfig();
config.setJdbcUrl("jdbc:h2:file:./data/testdb");
config.setUsername("sa");
config.setPassword("");
config.setMaximumPoolSize(10);

HikariDataSource ds = new HikariDataSource(config);
```


## Обработка распространенных ошибок

### Файл базы данных заблокирован
Убедитесь, что нет других подключений, использующих файл базы данных. H2 поддерживает параллельное чтение, но запись требует монопольного доступа.

### Потеря данных в памяти
База данных в памяти теряет данные после закрытия подключения. Для сохранности используйте файловый режим.

### Класс не найден
Убедитесь, что класс org.h2.Driver находится в classpath.
