# Flutter Installation Guide

Complete Flutter development environment setup guide.

## Requirements

### Windows

- **OS**: Windows 10 or higher (64-bit)
- **Disk space**: Minimum 2.8 GB
- **Tools**: Windows PowerShell 5.0+, Git for Windows

### macOS

- **OS**: macOS 10.14 (Mojave) or higher
- **Disk space**: Minimum 2.8 GB
- **Tools**: Command Line Tools, Git

### Linux

- **OS**: Ubuntu 18.04 or higher
- **Disk space**: Minimum 2.8 GB
- **Dependencies**: bash, curl, git, unzip, xz-utils, zip, libglu1-mesa

## Installation Steps

### Windows Installation

1. **Download Flutter SDK**

Visit the Flutter website to download the latest version:
```bash
# Or clone via git
git clone https://github.com/flutter/flutter.git -b stable
```

2. **Configure environment variables**

Add Flutter to system PATH:
```powershell
# Run in PowerShell
[Environment]::SetEnvironmentVariable(
    "Path",
    $env:Path + ";C:\src\flutter\bin",
    "User"
)
```

3. **Verify installation**

```bash
flutter doctor
```

### macOS Installation

1. **Download Flutter SDK**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

2. **Configure environment variables**

```bash
# Add to ~/.bash_profile or ~/.zshrc
export PATH="$PATH:/Users/username/flutter/bin"
```

3. **Run flutter doctor**

```bash
flutter doctor
```

### Linux Installation

1. **Install dependencies**

```bash
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa
```

2. **Download Flutter SDK**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

3. **Configure PATH**

```bash
export PATH="$PATH:/home/username/flutter/bin"
```

## Android Development Environment Setup

### Install Android Studio

1. Download Android Studio: https://developer.android.com/studio
2. Install and launch
3. Configure SDK:

```bash
flutter config --android-sdk /path/to/android/sdk
```

### Accept licenses

```bash
flutter doctor --android-licenses
```

### Configure emulator

```bash
# Create emulator
flutter emulator --create --name my-emulator

# Launch emulator
flutter emulator --launch my-emulator
```

## iOS Development Environment Setup (macOS)

### Install Xcode

```bash
# Install via App Store
xcode-select --install
```

### Configure iOS Simulator

```bash
# List available simulators
xcrun simctl list devices available

# Launch simulator
open -a Simulator
```

```bash
# Check Flutter version
flutter --version

# Run flutter doctor
flutter doctor

# Check for updates
flutter upgrade
```

## Common Problems

### Download failed?

Use mirror:
```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
export PUB_CACHE=/path/to/cache
```

### Android SDK not found?

```bash
flutter config --android-sdk /path/to/android-sdk

# Or set environment variable
export ANDROID_SDK_ROOT=/path/to/android-sdk
```

### Command not found?

Ensure Flutter is in PATH:
```bash
echo $PATH
# Should contain flutter/bin
```

## Uninstallation

```bash
# Delete Flutter directory
rm -rf /path/to/flutter

# Remove from PATH
```
