# jQuery 使用指南

高效使用 jQuery 的完整指南。

---

## 基本概念

### 什么是 jQuery？

jQuery 是一个 JavaScript 库，简化了 DOM 操作、事件处理和 Ajax。

### $ 函数

```javascript
// 选择元素
$('#myId')
$('.myClass')
$('div')

// 加载完成后执行
$(document).ready(function() {
  // 代码
})

// 简写
$(function() {
  // 代码
})
```

---

## 选择器

### 基本选择器

```javascript
// ID 选择器
$('#myId')

// 类选择器
$('.myClass')

// 元素选择器
$('div')

// 属性选择器
$('[href]')
$('a[href="https://example.com"]')

// 伪类选择器
$('li:first')
$('li:last')
$('li:even')
$('li:odd')
```

---

## DOM 操作

### 获取/设置内容

```javascript
// 获取文本
var text = $('#myElement').text()

// 设置文本
$('#myElement').text('新文本')

// 获取 HTML
var html = $('#myElement').html()

// 设置 HTML
$('#myElement').html('<span>新内容</span>')

// 获取/设置值
var value = $('#myInput').val()
$('#myInput').val('新值')
```

### CSS 操作

```javascript
// 获取 CSS
var color = $('#myElement').css('color')

// 设置 CSS
$('#myElement').css('color', 'red')

// 添加类
$('#myElement').addClass('active')

// 移除类
$('#myElement').removeClass('active')

// 切换类
$('#myElement').toggleClass('active')
```

---

## 事件

### 事件绑定

```javascript
// 点击事件
$('#myButton').click(function() {
  console.log('点击!')
})

// 绑定多个事件
$('#myElement').on('click hover', function() {})

// 事件委托
$('#parent').on('click', '.child', function() {})
```

### 常用事件

```javascript
// 鼠标事件
$('#myElement').click(function() {})
$('#myElement').hover(function() {}, function() {})

// 键盘事件
$('#myInput').keydown(function(e) {})
$('#myInput').keyup(function(e) {})

// 表单事件
$('#myForm').submit(function(e) {})
$('#myInput').focus(function() {})
$('#myInput').blur(function() {})
```

## 效果和动画

### 基本效果

```javascript
// 显示/隐藏
$('#myElement').hide()
$('#myElement').show()
$('#myElement').toggle()

// 淡入淡出
$('#myElement').fadeIn()
$('#myElement').fadeOut()
$('#myElement').fadeToggle()

// 滑动
$('#myElement').slideDown()
$('#myElement').slideUp()
$('#myElement').slideToggle()
```

### 自定义动画

```javascript
$('#myElement').animate({
  'left': '100px',
  'opacity': 0.5
}, 300)  // 300ms
```

## AJAX

### GET 请求

```javascript
$.get('https://api.example.com/data', function(data) {
  console.log(data)
})
```

### POST 请求

```javascript
$.post('https://api.example.com/submit', {
  name: 'John'
}, function(data) {
  console.log(data)
})
```

### AJAX

```javascript
$.ajax({
  url: 'https://api.example.com/data',
  method: 'GET',
  success: function(data) {
    console.log(data)
  },
  error: function(xhr, status, error) {
    console.error(error)
  }
})
```
