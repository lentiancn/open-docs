# Bootstrap

最受歡迎的前端響應式 CSS 框架。

## 簡介

Bootstrap 是由 Twitter 開發的最受歡迎的前端 CSS 框架，用於構建響應式、移動優先的網站和 Web 應用。

### 主要特性

- **響應式網格系統** - 12 列響應式網格
- **預製組件** - 導航欄、卡片、模態框等
- **CSS 工具類** - 快速樣式調整
- **JavaScript 組件** - 互動式組件

### 統計數據

- GitHub 星標：160,000+
- 網站使用量：數百萬

## 快速開始

### CDN 引入

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```
