# MySQL 使用手冊

## 資料庫基礎操作

### 連線 MySQL

本機連線：
```bash
mysql -u root -p
```

遠端連線：
```bash
mysql -h hostname -u username -p database_name
```

### 建立資料庫

```sql
CREATE DATABASE database_name;
```

指定字符集建立：
```sql
CREATE DATABASE database_name CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 查看資料庫

```sql
SHOW DATABASES;
```

### 選擇資料庫

```sql
USE database_name;
```

### 刪除資料庫

```sql
DROP DATABASE database_name;
```

## 資料表操作

### 建立資料表

```sql
CREATE TABLE table_name (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100) UNIQUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### 查看資料表結構

```sql
DESCRIBE table_name;
-- 或
SHOW COLUMNS FROM table_name;
```

### 查看所有資料表

```sql
SHOW TABLES;
```

### 修改資料表結構

新增資料行：
```sql
ALTER TABLE table_name ADD COLUMN new_column VARCHAR(50);
```

修改資料行：
```sql
ALTER TABLE table_name MODIFY column_name VARCHAR(200);
```

刪除資料行：
```sql
ALTER TABLE table_name DROP COLUMN column_name;
```

### 刪除資料表

```sql
DROP TABLE table_name;
```

## 資料操作

### 插入資料

單筆插入：
```sql
INSERT INTO table_name (column1, column2) VALUES ('value1', 'value2');
```

多筆插入：
```sql
INSERT INTO table_name (column1, column2) VALUES 
('value1', 'value2'),
('value3', 'value4');
```

### 查詢資料

基本查詢：
```sql
SELECT * FROM table_name;
```

條件查詢：
```sql
SELECT column1, column2 FROM table_name WHERE condition;
```

排序：
```sql
SELECT * FROM table_name ORDER BY column_name DESC;
```

分頁：
```sql
SELECT * FROM table_name LIMIT 10 OFFSET 20;
```

### 更新資料

```sql
UPDATE table_name SET column1 = 'new_value' WHERE condition;
```

### 刪除資料

```sql
DELETE FROM table_name WHERE condition;
```

清空資料表：
```sql
TRUNCATE TABLE table_name;
```

## 索引操作

### 建立索引

一般索引：
```sql
CREATE INDEX index_name ON table_name (column_name);
```

唯一索引：
```sql
CREATE UNIQUE INDEX index_name ON table_name (column_name);
```

複合索引：
```sql
CREATE INDEX index_name ON table_name (column1, column2);
```

### 查看索引

```sql
SHOW INDEX FROM table_name;
```

### 刪除索引

```sql
DROP INDEX index_name ON table_name;
```

## 使用者權限管理

### 建立使用者

```sql
CREATE USER 'username'@'localhost' IDENTIFIED BY 'password';
```

### 授予權限

```sql
GRANT SELECT, INSERT, UPDATE ON database_name.* TO 'username'@'localhost';
```

授予所有權限：
```sql
GRANT ALL PRIVILEGES ON database_name.* TO 'username'@'localhost';
```

### 重新載入權限

```sql
FLUSH PRIVILEGES;
```

### 查看使用者權限

```sql
SHOW GRANTS FOR 'username'@'localhost';
```

### 刪除使用者

```sql
DROP USER 'username'@'localhost';
```

## 交易控制

### 開始交易

```sql
START TRANSACTION;
-- 或
BEGIN;
```

### 提交交易

```sql
COMMIT;
```

### 復原交易

```sql
ROLLBACK;
```

### 設定儲存點

```sql
SAVEPOINT savepoint_name;
ROLLBACK TO SAVEPOINT savepoint_name;
```

## 常用函數

### 字串函數

```sql
SELECT CONCAT('Hello', ' ', 'World');
SELECT UPPER('hello');
SELECT LOWER('HELLO');
SELECT SUBSTRING('Hello World', 1, 5);
SELECT LENGTH('Hello');
SELECT TRIM('  Hello  ');
```

### 數值函數

```sql
SELECT ABS(-5);
SELECT CEIL(3.14);
SELECT FLOOR(3.14);
SELECT ROUND(3.14159, 2);
SELECT MOD(10, 3);
```

### 日期函數

```sql
SELECT NOW();
SELECT CURDATE();
SELECT CURTIME();
SELECT DATE('2024-01-01 10:30:00');
SELECT YEAR('2024-01-01');
SELECT MONTH('2024-01-01');
SELECT DAY('2024-01-01');
SELECT DATE_ADD('2024-01-01', INTERVAL 1 DAY);
SELECT DATEDIFF('2024-01-01', '2023-12-01');
```

### 聚合函數

```sql
SELECT COUNT(*) FROM table_name;
SELECT SUM(column_name) FROM table_name;
SELECT AVG(column_name) FROM table_name;
SELECT MAX(column_name) FROM table_name;
SELECT MIN(column_name) FROM table_name;
```

## 備份與復原

### 備份資料庫

```bash
mysqldump -u root -p database_name > backup.sql
```

備份所有資料庫：
```bash
mysqldump -u root -p --all-databases > all_databases.sql
```

### 復原資料庫

```bash
mysql -u root -p database_name < backup.sql
```

### 匯出查詢結果

```bash
mysql -u root -p -e "SELECT * FROM table_name" database_name > output.tsv
```

## 效能最佳化

### 使用 EXPLAIN 分析查詢

```sql
EXPLAIN SELECT * FROM table_name WHERE condition;
```

### 最佳化建議

1. 為常用查詢條件建立索引
2. 避免 SELECT *，只查詢需要的資料行
3. 使用 LIMIT 限制結果數量
4. 避免在索引資料行上使用函數
5. 合理使用 JOIN，避免笛卡爾積
6. 定期分析和最佳化資料表

### 查看慢查詢日誌

```sql
SHOW VARIABLES LIKE 'slow_query_log%';
```

## 常見錯誤處理

### 連線錯誤
檢查 MySQL 服務是否啟動，連接埠是否正確，使用者權限是否正確設定。

### 字符集錯誤
確保資料庫、資料表、連線都使用相同的字符集（建議 utf8mb4）。

### 權限錯誤
確認使用者具有相應的資料庫和資料表操作權限。
