# Nginx

Nginx (произносится как "engine x") — это высокопроизводительный HTTP-веб-сервер, обратный прокси, балансировщик нагрузки и почтовый прокси-сервер.

## Что такое Nginx

Nginx известен своей гибкостью, высокой производительностью и низким потреблением ресурсов. Это один из самых популярных веб-серверов в мире.

## Основные функции

- **HTTP-веб-сервер**: Обслуживание статических файлов
- **Обратный прокси**: В качестве обратного прокси для backend-серверов
- **Балансировка нагрузки**: Поддержка нескольких алгоритмов балансировки
- **Служба кэширования**: Поддержка HTTP-кэширования
- **SSL/TLS**: Поддержка HTTPS
- **Почтовый прокси**: Поддержка POP3-, IMAP- и SMTP-прокси

## Быстрый старт

### Требования к окружению

- Поддерживается Linux, FreeBSD, Solaris, macOS, Windows
- Требуется компилятор C (при сборке из исходного кода)

### Установка

#### Ubuntu/Debian

```bash
sudo apt update
sudo apt install nginx
```

#### CentOS/RHEL

```bash
sudo yum install epel-release
sudo yum install nginx
```

#### Docker

```bash
docker pull nginx
docker run -d -p 80:80 nginx
```

### Запуск

```bash
nginx
```

### Тестирование доступа

Открыть браузер: http://localhost
