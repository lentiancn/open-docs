# PyTorch Installation Guide

## Prerequisites

- **Python**: 3.8, 3.9, 3.10, 3.11, 3.12
- **Operating System**: Windows, macOS, Linux
- **GPU**: NVIDIA GPU with CUDA (optional for GPU acceleration)
- **Memory**: Minimum 8GB RAM (16GB+ recommended for deep learning)

---

## Installation with pip

### CPU Only

```bash
pip install torch torchvision torchaudio
```

### With CUDA (GPU Support)

```bash
# CUDA 11.8
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

# CUDA 11.7
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu117

# CUDA 10.2
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu102
```

---

## Installation with conda

### CPU Only

```bash
conda install pytorch torchvision torchaudio cpuonly -c pytorch
```

### With CUDA

```bash
conda install pytorch torchvision torchaudio pytorch-cuda=11.8 -c pytorch -c nvidia
```

## Verify Installation

```python
import torch

# Check PyTorch version
print(f"PyTorch version: {torch.__version__}")

# Check CUDA availability
print(f"CUDA available: {torch.cuda.is_available()}")

# Check CUDA version
print(f"CUDA version: {torch.version.cuda}")

# Check number of GPUs
if torch.cuda.is_available():
    print(f"Number of GPUs: {torch.cuda.device_count()}")
    print(f"GPU name: {torch.cuda.get_device_name(0)}")

# Test tensor operations
x = torch.randn(3, 3)
print(f"Tensor test:\n{x}")
```

## Installation for Development

```bash
# Clone repository
git clone --recursive https://github.com/pytorch/pytorch
cd pytorch

# Install dependencies
pip install -r requirements.txt

# Install PyTorch
python setup.py install
```

## Common Issues

### "No module named 'torch'"

- Ensure PyTorch is installed: `pip install torch`
- Check Python environment (are you using the correct virtualenv?)

### CUDA not available

- Verify NVIDIA GPU: `nvidia-smi`
- Ensure CUDA version matches PyTorch version
- Reinstall with correct CUDA index URL

### Out of memory errors

- Reduce batch size
- Use gradient accumulation
- Use mixed precision training (FP16)
