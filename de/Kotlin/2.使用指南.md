# Kotlin Benutzerhandbuch

Diese Anleitung führt in die grundlegende Syntax und fortgeschrittene Funktionen von Kotlin ein.

## Schnellstart

### Hello World

```kotlin
fun main() {
    println("Hello, World!")
}
```

### Variablen

```kotlin
// Unveränderlich
val name: String = "Kotlin"

// Veränderlich
var age: Int = 10
age = 11
```

## Grundlegende Typen

### Zahlen

```kotlin
val int: Int = 100
val long: Long = 100L
val double: Double = 100.0
val float: Float = 100.0f
```

### Zeichenketten

```kotlin
val str: String = "Hello"
val template = "Name: $name, Age: $age"
```

### Arrays

```kotlin
val arr = arrayOf(1, 2, 3)
val intArray = intArrayOf(1, 2, 3)
```

## Kontrollfluss

### Bedingungen

```kotlin
val max = if (a > b) a else b

// When-Ausdruck
val result = when (x) {
    1 -> "one"
    2 -> "two"
    else -> "other"
}
```

### Schleifen

```kotlin
// For-Schleife
for (i in 1..10) {
    println(i)
}

// While
while (x > 0) {
    x--
}
```

## Funktionen

### Grundlegende Funktionen

```kotlin
fun sum(a: Int, b: Int): Int {
    return a + b
}

// Kurzform
fun sum(a: Int, b: Int) = a + b
```

### Standardparameter

```kotlin
fun greet(name: String = "World") {
    println("Hello, $name!")
}
```

### Vararg-Parameter

```kotlin
fun sum(vararg numbers: Int): Int {
    return numbers.sum()
}
```

## Objektorientierung

### Klassen

```kotlin
class Person(val name: String, var age: Int) {
    
    fun speak() {
        println("My name is $name")
    }
}
```

### Vererbung

```kotlin
open class Animal(val name: String) {
    open fun sound() = ""
}

class Dog(name: String) : Animal(name) {
    override fun sound() = "Woof!"
}
```

### Datenklassen

```kotlin
data class User(val name: String, val email: String)
```

### Singleton

```kotlin
object Database {
    fun connect() { }
}
```

## Sammlungen

### Listen

```kotlin
val list = listOf(1, 2, 3)
val mutableList = mutableListOf(1, 2, 3)
```

### Maps

```kotlin
val map = mapOf("a" to 1, "b" to 2)
```

### Sammlungsoperationen

```kotlin
val list = listOf(1, 2, 3, 4, 5)

list.filter { it > 2 }           // [3, 4, 5]
list.map { it * 2 }              // [2, 4, 6, 8, 10]
list.reduce { acc, i -> acc + i } // 15
```

## Null-Sicherheit

### Nullable-Typen

```kotlin
var name: String? = null
name = "Kotlin"
```

### Sicherheitsoperatoren

```kotlin
val length = name?.length
val len = name?.length ?: 0
```

### Non-Null-Assertion

```kotlin
val len = name!!.length
```

## Koroutinen

### Grundlegende Verwendung

```kotlin
import kotlinx.coroutines.*

suspend fun fetchData(): String {
    delay(1000)
    return "Data"
}

fun main() = runBlocking {
    val result = fetchData()
    println(result)
}
```

### Parallelität

```kotlin
suspend fun fetchAll() = coroutineScope {
    val a = async { fetchData1() }
    val b = async { fetchData2() }
    "${a.await()} ${b.await()}"
}
```

## Erweiterungen

### Erweiterungsfunktionen

```kotlin
fun String.addExclamation() = this + "!"

val str = "Hello".addExclamation() // "Hello!"
```

### Erweiterungseigenschaften

```kotlin
val String.lastChar: Char
    get() = this[length - 1]
```

## Generics

```kotlin
class Box<T>(val value: T)

fun <T> printValue(value: T) {
    println(value)
}
```

## Best Practices

### 1. Bevorzugt val verwenden

```kotlin
// Empfohlen
val immutable = "value"

// Vermeiden
var mutable = "value"
```

### 2. Datenklassen verwenden

```kotlin
// Empfohlen
data class User(val name: String, val age: Int)
```

### 3. Null vermeiden

```kotlin
// Standardwerte verwenden
val name = nullableName ?: "default"
```

### 4. Koroutinen für Async verwenden

```kotlin
suspend fun loadData() = withContext(Dispatchers.IO) {
    // Asynchrone Operationen
}
```
