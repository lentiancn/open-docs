## グリッドシステム

### 基本原理

BootstrapのグリッドシステムはFlexboxを使用して構築され、最大12列をサポートします。

### コンテナ

```html
<!-- 固定幅コンテナ -->
<div class="container">
  コンテンツ
</div>

<!-- 全幅コンテナ -->
<div class="container-fluid">
  コンテンツ
</div>

<!-- レスポンシブブレークポイントコンテナ (Bootstrap 5.2+) -->
<div class="container-sm">100% 幅 < 576px</div>
<div class="container-md">100% 幅 >= 768px</div>
<div class="container-lg">100% 幅 >= 992px</div>
<div class="container-xl">100% 幅 >= 1200px</div>
<div class="container-xxl">100% 幅 >= 1400px</div>
```

### 行と列

```html
<div class="container">
  <div class="row">
    <div class="col">列</div>
    <div class="col">列</div>
    <div class="col">列</div>
  </div>
</div>
```

### 列幅の設定

```html
<div class="row">
  <div class="col-4">4列</div>
  <div class="col-8">8列</div>
</div>
```

### レスポンシブブレークポイント

```html
<!-- レスポンシブ列幅 -->
<div class="row">
  <div class="col-12 col-md-6 col-lg-4">
    モバイルでは全幅、タブレット6列、デスクトップ4列
  </div>
</div>
```

### 列のオフセット

```html
<!-- offsetを使用 -->
<div class="row">
  <div class="col-md-4">4列</div>
  <div class="col-md-4 offset-md-4">4列オフセット</div>
</div>

<!-- マージン自動オフセット -->
<div class="row">
  <div class="col-md-3 ms-auto">右側オフセット</div>
</div>
```

### グリッドのネスト

```html
<div class="row">
  <div class="col-sm-8">
    外側8列
    <div class="row">
      <div class="col-6">内側6列</div>
      <div class="col-6">内側6列</div>
    </div>
  </div>
</div>
```

### Flexレイアウト

```html
<!-- 水平配置 -->
<div class="row justify-content-start">...</div>
<div class="row justify-content-center">...</div>
<div class="row justify-content-end">...</div>
<div class="row justify-content-around">...</div>
<div class="row justify-content-between">...</div>
<div class="row justify-content-evenly">...</div>

<!-- 垂直配置 -->
<div class="row align-items-start">...</div>
<div class="row align-items-center">...</div>
<div class="row align-items-end">...</div>
```

## タイポグラフィ

### 見出し

```html
<h1>見出し1</h1>
<h2>見出し2</h2>
<h3>見出し3</h3>
<h4>見出し4</h4>
<h5>見出し5</h5>
<h6>見出し6</h6>

<!-- ディスプレイ見出し -->
<h1 class="display-1">ディスプレイ1</h1>
<h1 class="display-2">ディスプレイ2</h1>
<h1 class="display-3">ディスプレイ3</h1>
<h1 class="display-4">ディスプレイ4</h1>
<h1 class="display-5">ディスプレイ5</h1>
<h1 class="display-6">ディスプレイ6</h1>
```

### 段落

```html
<!-- 通常段落 -->
<p>これは段落です。</p>

<!-- リード段落 -->
<p class="lead">これはリード段落です。</p>

<!-- テキスト配置 -->
<p class="text-start">左揃え</p>
<p class="text-center">中央揃え</p>
<p class="text-end">右揃え</p>
<p class="text-justify">両端揃え</p>

<!-- レスポンシブ配置 -->
<p class="text-sm-start">小画面では左揃え</p>
<p class="text-md-center">中画面では中央揃え</p>
<p class="text-lg-end">大画面では右揃え</p>
```

### テキストスタイル

```html
<!-- フォントウェイト -->
<p class="fw-bold">ボールド</p>
<p class="fw-bolder">より太い</p>
<p class="fw-normal">通常</p>
<p class="fw-light">ライト</p>
<p class="fw-lighter">より軽い</p>

<!-- フォントスタイル -->
<p class="fst-italic">イタリック</p>
<p class="fst-normal">通常スタイル</p>

<!-- テキスト変換 -->
<p class="text-lowercase">小文字</p>
<p class="text-uppercase">大文字</p>
<p class="text-capitalize">Capitalized</p>
```

### リスト

```html
<!-- スタイルなしリスト -->
<ul class="list-unstyled">
  <li>アイテム1</li>
  <li>アイテム2</li>
</ul>

<!-- インラインリスト -->
<ul class="list-inline">
  <li class="list-inline-item">アイテム1</li>
  <li class="list-inline-item">アイテム2</li>
</ul>
```

## ベストプラクティス

### 1. 意味的なHTMLの使用

```html
<!-- 良い例 -->
<nav class="navbar">...</nav>
<main class="container">...</main>
<article class="card">...</article>

<!-- 避けるべき例 -->
<div class="navbar">...</div>
<div class="container">...</div>
<div class="card">...</div>
```

### 2. コードをシンプルに

```html
<!-- 良い例 -->
<div class="container">
  <div class="row">
    <div class="col-md-6">...</div>
  </div>
</div>

<!-- 過度のネストを避ける -->
<div class="container">
  <div class="row">
    <div class="col-md-6">
      <div class="card">
        <div class="card-body">...</div>
```
