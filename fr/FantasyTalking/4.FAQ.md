# FantasyTalking FAQ

## Problèmes d'Installation

### L'installation échoue?

Vérifiez la version Node.js (>= 14.0). Videz le cache npm ou installez depuis les sources.

## Utilisation

### Synthèse vocale lente?

Vérifiez la connexion réseau. Utilisez l'auto-hébergement.

### Obtenir une clé API?

Inscrivez-vous sur le site officiel et créez dans la console.

## Autres

### Plates-formes supportées?

- Web: tous navigateurs modernes
- Mobile: iOS et Android
- Bureau: Windows, macOS, Linux
- Serveur: Node.js et Python
