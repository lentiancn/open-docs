# FAQ Nacos

## Qu'est-ce que Nacos?

Nacos est une plateforme open source d'Alibaba pour la découverte de services et la gestion de configuration installer N dynamique.

## Commentacos?

```bash
# Mode autonome
./startup.sh -m standalone

# Docker
docker run --name nacos -d -p 8848:8848 nacos/nacos-server:v2.3.0
```

## Comment accéder à la console Nacos?

- URL: http://localhost:8848/nacos
- Identifiant: nacos
- Mot de passe: nacos
