Angular is a powerful frontend framework developed by Google. This guide covers installation methods for various environments.

## Installation Methods

### Method 1: Angular CLI (Recommended)

#### Install Angular CLI

```bash
# Install globally
npm install -g @angular/cli

# Verify installation
ng version
```

#### Create New Project

```bash
# Create new project
ng new my-app

# Create with options
ng new my-app --routing --style=scss --skip-tests
```

#### Options

```bash
# Skip installation
ng new my-app --skip-install

# Standalone components (Angular 14+)
ng new my-app --standalone

# SSR enabled
ng new my-app --ssr
```

### Method 2: Using npx

```bash
# Create project without installing CLI
npx @angular/cli new my-app
```

### Method 3: Manual Setup

```bash
# Create directory
mkdir my-app
cd my-app

# Initialize npm
npm init -y

# Install Angular
npm install @angular/core @angular/common @angular/platform-browser @angular/platform-browser-dynamic @angular/compiler @angular/forms @angular/router @angular/animations rxjs zone.js

# Install dev dependencies
npm install --save-dev @angular/compiler-cli @angular-devkit/build-angular typescript
```

### Method 4: Docker

```dockerfile
# Dockerfile
FROM node:20-alpine

WORKDIR /app

# Install Angular CLI
RUN npm install -g @angular/cli

# Create project
RUN ng new my-app --directory .

# Start dev server
CMD ["ng", "serve", "--host", "0.0.0.0"]
```

## Environment Configuration

### Node.js Version Management

#### Using nvm (Recommended)

```bash
# Install nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Install Node.js
nvm install 20
nvm use 20

# Set default
nvm alias default 20
```

### npm Configuration

```bash
# Set registry
npm config set registry https://registry.npmjs.org/

# Enable strict SSL
npm config set strict-ssl true

# Clear cache
npm cache clean --force
```

## Verification

```bash
# Check Angular CLI
ng version

# Check Node.js
node --version

# Check npm
npm --version
```

## Common Issues

### Permission Denied?

```bash
# Fix npm permissions
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
export PATH=~/.npm-global/bin:$PATH
```

### Slow Installation?

```bash
# Use npm ci for faster installation
npm ci

# Or use yarn
npm install -g yarn
```

## Updating Angular

```bash
# Update Angular CLI
npm install -g @angular/cli@latest

# Update your project
ng update @angular/core @angular/cli
```
