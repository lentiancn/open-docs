# Java - 简介

Java 是一种由 Sun Microsystems（现 Oracle）于 1995 年发布的面向对象编程语言。Java 设计目标是"一次编写，到处运行"（Write Once, Run Anywhere），通过 Java 虚拟机（JVM）实现跨平台运行。

## Java 的核心特性

### 1. 面向对象

Java 是一种纯面向对象语言，一切皆为对象：

```java
// 定义类
public class Person {
    private String name;
    private int age;
    
    // 构造函数
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // 方法
    public void sayHello() {
        System.out.println("Hello, I'm " + name);
    }
    
    // Getter 和 Setter
    public String getName() {
        return name;
    }
    
    public void setName(String name) {
        this.name = name;
    }
}

// 使用类
public class Main {
    public static void main(String[] args) {
        Person person = new Person("Tom", 25);
        person.sayHello();
    }
}
```

### 2. 平台无关性

Java 通过 JVM 实现"一次编译，到处运行"：

```
Java 源代码 (.java)
      ↓ 编译
Java 字节码 (.class)
      ↓ JVM 解释/编译
本地机器码
```

### 3. 自动内存管理

Java 提供垃圾回收（Garbage Collection）机制，自动管理内存：

```java
public class MemoryExample {
    public static void main(String[] args) {
        // 对象创建
        String str = new String("Hello");
        
        // 不再需要手动释放
        str = null; // 垃圾回收器会自动回收
        
        // 尝试触发垃圾回收
        System.gc();
    }
}
```

### 4. 强类型系统

Java 是强类型语言，编译时检查类型：

```java
// 必须声明类型
int number = 10;
String text = "Hello";
boolean flag = true;

// 类型不匹配会编译错误
// number = "text"; // 编译错误
```

### 5. 多线程

Java 内置多线程支持：

```java
public class ThreadExample extends Thread {
    @Override
    public void run() {
        for (int i = 0; i < 5; i++) {
            System.out.println("线程: " + getName() + " - " + i);
        }
    }
    
    public static void main(String[] args) {
        ThreadExample t1 = new ThreadExample();
        ThreadExample t2 = new ThreadExample();
        
        t1.start();
        t2.start();
    }
}
```

## Java 与其他语言对比

### Java vs Python

| 特性 | Java | Python |
|------|------|--------|
| 类型 | 静态类型 | 动态类型 |
| 编译 | 先编译后运行 | 解释型 |
| 速度 | 较快 | 较慢 |
| 适用 | 企业级应用 | 快速开发 |

### Java vs C++

| 特性 | Java | C++ |
|------|------|-----|
| 内存管理 | 自动 | 手动 |
| 指针 | 无 | 有 |
| 多继承 | 不支持 | 支持 |
| 可移植性 | 高 | 较低 |
