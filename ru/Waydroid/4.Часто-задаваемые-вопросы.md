# Часто задаваемые вопросы

## Сертификация Google Play

Waydroid использует настроенный образ Android на основе LineageOS, который по умолчанию не включает службы Google Play. Если вам нужна сертификация Google Play, вы можете установить образ GAPPS.

## Отключение экранных клавиатур

В некоторых случаях вы можете захотеть отключить экранные клавиатуры. Это можно сделать через настройки системы или конфигурацию Waydroid.

## Запуск Waydroid через виртуальную машину

Чтобы запустить Waydroid через виртуальную машину, убедитесь, что VM поддерживает вложенную виртуализацию и правильно настроено аппаратное ускорение.

## Использование ADB с Waydroid

Waydroid поддерживает отладку ADB. По умолчанию включен безопасный режим ADB, который не подключается автоматически. Используйте следующую команду для подключения:

```bash
waydroid adb connect
```

Затем предоставьте авторизацию на стороне Android.

## Настройка сессий только Waydroid

Вы можете настроить выделенные сессии, которые запускают только Waydroid, подходящие для конкретных случаев использования.

## Настройка общей папки

Waydroid поддерживает настройку общих папок для удобной передачи файлов между хостом Linux и контейнером Android.

## Коррекция цвета в Mutter

При использовании оконного менеджера Mutter могут возникать проблемы с отображением цвета, которые можно исправить с помощью определенных настроек.

## Резервное копирование и восстановление приложений и данных

Waydroid поддерживает резервное копирование и восстановление приложений и данных, чтобы гарантировать, что важные данные не будут потеряны.

## Использование пользовательских образов Waydroid

Для использования пользовательских образов:

1. Скачайте или соберите пользовательские `system.img` и `vendor.img`
2. Скопируйте их в каталог `/etc/waydroid-extra/images/`
3. Реинициализируйте: `sudo waydroid init -f`

## Установка системных приложений

Системные приложения можно устанавливать, но для этого требуются специальные разрешения и шаги конфигурации.

## Совместимость оборудования

Waydroid имеет определенные требования к оборудованию:
- Требуется поддержка сервера отображения Wayland
- Требуется поддержка ядра Linux для пространств имен и функций контейнера
- Графические драйверы должны поддерживать OpenGL ES 3.2

## Оптимизация производительности

Для достижения оптимальной производительности:
- Убедитесь, что включено аппаратное ускорение
- Используйте подходящие графические драйверы
- Запускайте на поддерживаемых процессорах (избегайте чистой эмуляции ARM)
