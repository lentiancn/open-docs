# Guide d'installation de Spring Boot

Ce guide couvre la configuration de l'environnement de développement pour Spring Boot, y compris Java, Maven et Gradle.

## Prérequis

### Installation de Java

Spring Boot 3.x nécessite Java 17 ou version ultérieure. Spring Boot 2.x nécessite Java 8 ou version ultérieure.

**Windows:**
1. Téléchargez JDK depuis [Oracle](https://www.oracle.com/java/technologies/downloads/) ou [Adoptium](https://adoptium.net/)
2. Exécutez l'installateur
3. Configurez les variables d'environnement:
   ```
   JAVA_HOME=C:\Program Files\Java\jdk-17
   PATH=%JAVA_HOME%\bin;%PATH%
   ```

**macOS:**
```bash
# Avec Homebrew
brew install openjdk@17

# Ajouter au PATH
echo 'export JAVA_HOME=$(/usr/libexec/java_home)' >> ~/.zshrc
source ~/.zshrc
```

**Linux:**
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install openjdk-17-jdk

# Configurer JAVA_HOME
echo 'export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64' >> ~/.bashrc
source ~/.bashrc
```

Vérifier l'installation:
```bash
java -version
```

### Installation de Maven

**Windows:**
1. Téléchargez depuis [Apache Maven](https://maven.apache.org/download.cgi)
2. Extrayez dans un dossier (ex: C:\apache-maven)
3. Configurez les variables d'environnement:
   ```
   MAVEN_HOME=C:\apache-maven
   PATH=%MAVEN_HOME%\bin;%PATH%
   ```

**macOS:**
```bash
brew install maven
```

**Linux:**
```bash
sudo apt install maven
```

Vérifier l'installation:
```bash
mvn -version
```

### Installation de Gradle

**Windows:**
1. Téléchargez depuis [Gradle](https://gradle.org/install/)
2. Extrayez et ajoutez au PATH

**macOS:**
```bash
brew install gradle
```

**Linux:**
```bash
sudo apt install gradle
```

Vérifier l'installation:
```bash
gradle -version
```

## Configuration IDE

### IntelliJ IDEA (Recommandé)
- Téléchargez depuis [JetBrains](https://www.jetbrains.com/idea/download/)
- Assurez-vous que JDK est configuré dans la structure du projet
- Installez le plugin Spring Boot Assistant

### VS Code
- Installez Extension Pack for Java
- Installez Spring Boot Extension

## Démarrage rapide avec Spring Initializr

1. Visitez [start.spring.io](https://start.spring.io/)
2. Sélectionnez:
   - Projet: Maven ou Gradle
   - Langage: Java
   - Version de Spring Boot
   - Métadonnées du projet
3. Ajoutez des dépendances (Spring Web, Spring Data JPA, etc.)
4. Cliquez sur "Generate" pour télécharger le projet

## Vérifier votre configuration

```bash
# Vérifier Java
java -version

# Vérifier Maven
mvn -version

# Vérifier Gradle (si utilisé)
gradle -version

# Créer un projet Spring Boot simple et l'exécuter
curl https://start.spring.io/starter.zip -o demo.zip
unzip demo.zip -d demo
cd demo
./mvnw spring-boot:run  # ou ./gradlew bootRun
```
