# APISIX 簡介

雲端原生 API 閘道。

## 什麼是 APISIX？

Apache APISIX 是 Apache 軟體基金會旗下的雲端原生 API 閘道，由 API7.ai 開發並捐贈。它是一個具有動態、即时、高效能等特點的 API 閘道，可作為所有業務的流量入口。

APISIX 提供了動態路由、動態上游、動態憑證、A/B 測試、灰度發布（金絲雀發布）、藍綠部署、限速、防攻擊、收集指標、監控報警、可觀測性、服務治理等功能。

### 核心特性

- **高效能**：基於 Nginx + Lua 實現，單核可達 10k+ QPS
- **動態路由**：無需重啟即可動態更新路由規則
- **插件系統**：豐富的插件生態，支援熱加載
- **服務發現**：支援 Consul、Eureka、Nacos 等多種服務發現機制
- **負載均衡**：輪詢、雜湊、一致性雜湊等多種策略
- **安全防護**：限流、認證、JWT、防火牆
- **可視化儀表板**：直觀的管理介面
- **多協議支援**：HTTP、HTTPS、HTTP/2、Dubbo、gRPC、WebSocket
- **AI 閘道**：支援 LLM 和 AI 工作負載

### 技術架構

APISIX 採用 Nginx 作為底層的 HTTP 反向代理伺服器，結合 Lua 腳本實現動態編排。整個架構分為資料面和控制面：

- **資料面**：處理實際請求轉發，零依賴
- **控制面**：透過 Admin API 進行配置管理

## 適用場景

- **API 統一管理**：集中管理企業內部所有 API
- **微服務閘道**：作為微服務入口，處理認證、限流、路由
- **服務網格入口**：對接 Istio、Linkerd 等服務網格
- **API 流量控制**：限流、熔斷、降級
- **混合雲架構**：統一管理多雲、混合雲 API
- **API 統計分析**：即時監控 API 調用情況
- **AI 閘道**：LLM 和 AI 應用的流量入口
