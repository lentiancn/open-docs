# Nginx Installationsanleitung

Anleitung zur Installation von Nginx auf allen wichtigen Plattformen.

---

## Linux

### Ubuntu / Debian

```bash
# Pakete aktualisieren
sudo apt update

# Nginx installieren
sudo apt install nginx

# Nginx starten
sudo systemctl start nginx

# Beim Boot starten
sudo systemctl enable nginx
```

### CentOS / RHEL

```bash
# EPEL installieren
sudo yum install epel-release

# Nginx installieren
sudo yum install nginx

# Starten
sudo systemctl start nginx
```

### Fedora

```bash
sudo dnf install nginx
sudo systemctl start nginx
```

---

## macOS

```bash
brew install nginx
brew services start nginx
```

---

## Windows

1. Von [nginx.org](http://nginx.org/en/download.html) herunterladen
2. Entpacken nach `C:\nginx`

```powershell
cd C:\nginx
start nginx
```

---

## Docker

```bash
docker run -d -p 80:80 --name nginx nginx:latest
```

---

## Uberprufen

```bash
nginx -v
nginx -t
curl http://localhost
```

---

## Konfiguration

| OS | Pfad |
|----|------|
| Ubuntu/Debian | `/etc/nginx/nginx.conf` |
| CentOS/RHEL | `/etc/nginx/nginx.conf` |

---

## Deinstallieren

```bash
sudo apt remove nginx
```
