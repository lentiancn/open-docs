# Angular Manuel utilisateur

Concepts de base et utilisation d'Angular.

## Composants

### Créer un composant

```bash
ng generate component my-component
# ou
ng g c my-component
```

### Structure du composant

```typescript
import { Component } from '@angular/core';

@Component({
  selector: 'app-my-component',
  templateUrl: './my-component.component.html',
  styleUrls: ['./my-component.component.css']
})
export class MyComponentComponent {
  title = 'Hello Angular';
}
```

## Liaison de données

### Liaison de propriété

```html
<img [src]="imageUrl">
```

### Liaison d'événement

```html
<button (click)="onClick()">Cliquer</button>
```

### Liaison bidirectionnelle

```html
<input [(ngModel)]="name">
```

## Services

### Créer un service

```bash
ng generate service my-service
# ou
ng g s my-service
```

### Utiliser les services

```typescript
import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root'
})
export class MyServiceService {
  getData() {
    return 'données';
  }
}
```

## Routage

### Configurer les routes

```typescript
const routes: Routes = [
  { path: '', component: HomeComponent },
  { path: 'about', component: AboutComponent },
  { path: '**', component: NotFoundComponent }
];
```

### Navigation

```html
<a routerLink="/about">À propos</a>
<router-outlet></router-outlet>
```

## Requêtes HTTP

### Importer HttpClient

```typescript
import { HttpClientModule } from '@angular/common/http';
```

### Faire des requêtes

```typescript
import { HttpClient } from '@angular/common/http';

constructor(private http: HttpClient) {}

getData() {
  return this.http.get('https://api.example.com/data');
}
```

## Formulaires

### Formulaires réactifs

```typescript
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

form: FormGroup;

constructor(private fb: FormBuilder) {
  this.form = this.fb.group({
    name: ['', Validators.required],
    email: ['', [Validators.required, Validators.email]]
  });
}
```

## Hooks de cycle de vie

| Hook | Description |
|------|-------------|
| ngOnInit | Initialisation du composant |
| ngOnChanges | Changement des propriétés d'entrée |
| ngOnDestroy | Destruction du composant |

## Directives

### Directives structurelles

```html
<div *ngIf="show">Afficher</div>
<div *ngFor="let item of items">{{item}}</div>
```

### Directives d'attribut

```html
<div [ngClass]="{'active': isActive}"></div>
```

## Meilleures pratiques

1. Utiliser le typage fort TypeScript
2. Garder les composants simples
3. Utiliser les services pour la logique partagée
4. Se désabonner des observables
5. Utiliser la stratégie de détection de changement OnPush
