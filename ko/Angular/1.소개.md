# 소개

Angular는 Google에서 개발하고 관리하는 최신 웹 애플리케이션 구축을 위한 오픈소스 웹 애플리케이션 프레임워크입니다. AngularJS(Angular 1.x)의 후속 버전으로, TypeScript를 주요 개발 언어로 사용하여 처음부터 완전히 다시 작성되었습니다.

Angular는 컴포넌트 기반 아키텍처, 양방향 데이터 바인딩, 의존성 주입, 라우팅, 폼 처리, HTTP 클라이언트 등 완전한 프론트엔드 개발 솔루션을 제공합니다. 이러한 기능을 통해 개발자는 고성능이며 유지보수가 용이한 싱글페이지 애플리케이션(SPA)을 신속하게 구축할 수 있습니다.

## 핵심 개념

### 컴포넌트

컴포넌트는 Angular 애플리케이션의 기본 구성 요소입니다. 각 컴포넌트는 세 부분으로 구성됩니다:
- **템플릿**: 뷰 구조를 정의합니다
- **스타일**: 뷰의appearance를 정의합니다
- **로직**: 컴포넌트의 동작을 정의합니다

### 모듈

모듈은 관련 컴포넌트, 서비스, 디렉티브를 함께 그룹화하여 코드를 구성합니다. 모든 Angular 애플리케이션에는 최소 하나의 루트 모듈(AppModule)이 있습니다.

### 서비스

서비스는 재사용 가능한 비즈니스 로직을 캡슐화하며, 의존성 주입을 통해 컴포넌트에서 사용할 수 있도록 합니다.

### 디렉티브

디렉티브는 HTML 요소의 기능을 확장하며, 구조적 디렉티브(*ngIf, *ngFor)와 속성 디렉티브(ngClass, ngStyle)로 나뉩니다.

## 주요 기능

### 컴포넌트 기반 아키텍처

Angular는 각 컴포넌트가 독립적이고 재사용 가능한 단위인 컴포넌트 기반 설계를 사용합니다.

### TypeScript

Angular는 TypeScript를 주요 언어로 사용하여 강력한 타입 시스템, 객체지향 프로그래밍, 현대적인 JavaScript 기능을 제공합니다.

### 의존성 주입

Angular에는 내장된 의존성 주입 시스템이 있어 코드를 더 모듈화하고 테스트 가능하며 유지보수하기 쉽게 만듭니다.

### 양방향 데이터 바인딩

템플릿과 컴포넌트 간의 데이터가 자동으로 동기화되어 수동 DOM 조작이 줄어듭니다.

### 라우팅

지연 로딩, 가드, 파라미터 전달 등을 지원하는 강력한 라우팅 시스템입니다.

### HTTP 클라이언트

인터셉터, 요청/응답 변환 등 고급 기능을 지원하는 내장 HTTP 클라이언트입니다.

### 폼 처리

복잡한 폼 유효성 검사를 위한 반응형 폼과 템플릿 기반 폼을 모두 제공합니다.

### 반응형 프로그래밍

강력한 비동기 프로그래밍 기능을 제공하는 전체 RxJS 지원을 포함합니다.

## 사용 사례

- 기업용 웹 애플리케이션
- 싱글페이지 애플리케이션(SPA)
- 프로그레시브 웹 앱(PWA)
- 하이브리드 모바일 앱(Ionic과 결합)
- 데스크톱 애플리케이션(Electron과 결합)

## 버전 역사

- **AngularJS (1.x)**: 2010년 출시
- **Angular 2**: 2016년 출시, 완전히 재작성됨
- **Angular 17**: 2023년 출시된 최신 안정 버전

Angular는 활발한 커뮤니티와 종합적인 문서를 갖추고 있어 최신 웹 애플리케이션 구축을 위한顶级 선택지 중 하나입니다.
