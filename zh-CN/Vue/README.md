# Vue.js

渐进式 JavaScript 框架。

## 简介

Vue.js 是一个用于构建用户界面的渐进式框架。

### 主要特性

- **响应式数据绑定**：数据变化自动更新视图
- **组件系统**：模块化的组件开发
- **虚拟 DOM**：高效的 DOM 更新
- **Composition API**：组合式 API
- **单文件组件**：模板、样式、逻辑在一起

## 快速开始

### 创建项目

```bash
npm create vue@latest
cd my-project
npm install
npm run dev
```

### 访问应用

打开浏览器：http://localhost:5173

## 常用命令

```bash
npm run dev      # 开发服务器
npm run build    # 构建生产版本
npm run preview  # 预览构建结果
```

## 核心概念

- **组件**：可复用的 Vue 实例
- **响应式**：数据驱动视图
- **指令**：增强模板能力
- **组合式 API**：逻辑复用
