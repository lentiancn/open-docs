# Resumen de Nacos

Nacos (Dynamic Naming and Configuration Service) es una plataforma fácil de usar para descubrimiento de servicios y gestión de configuración dinámica. Desarrollado por Alibaba.

## Características principales

- **Descubrimiento de servicios** - Registro y descubrimiento dinámico
- **Gestión de configuración** - Configuración centralizada
- **DNS dinámico** - Descubrimiento basado en DNS
- **Verificaciones de estado** - Monitoreo de salud del servicio

## Instalación

```bash
# Modo independiente
wget https://github.com/alibaba/nacos/releases/download/v2.3.0/nacos-server-2.3.0.tar.gz
tar -xvf nacos-server-2.3.0.tar.gz
cd nacos/bin
./startup.sh -m standalone

# Docker
docker run --name nacos -d -p 8848:8848 nacos/nacos-server:v2.3.0
```
