# Guía de Instalación de Spring Cloud

Esta guía cubre cómo instalar y configurar el entorno de desarrollo de Spring Cloud.

## Requisitos del Sistema

| Versión | Spring Boot |
|---------|-------------|
| 2025.1.x | 4.0.x |
| 2024.0.x | 3.4.x |
| 2023.0.x | 3.2.x, 3.3.x |

## Crear Proyecto

### Spring Initializr

1. Visita [start.spring.io](https://start.spring.io/)
2. Selecciona:
   - Proyecto: Maven
   - Lenguaje: Java
   - Spring Boot: 3.3.x
3. Añade dependencias:
   - Spring Cloud Config Client
   - Spring Cloud Eureka
   - Spring Cloud Gateway

### Línea de Comandos

```bash
curl https://start.spring.io/starter.zip \
  -d type=maven-project \
  -d language=java \
  -d bootVersion=3.3.0 \
  -d baseDir=myapp \
  -d dependencies=cloud-config-client,cloud-eureka,cloud-gateway \
  -o myapp.zip
```

## Configuración Maven

```xml
<properties>
    <spring-cloud.version>2023.0.3</spring-cloud.version>
</properties>

<dependencyManagement>
    <dependencies>
        <dependency>
            <groupId>org.springframework.cloud</groupId>
            <artifactId>spring-cloud-dependencies</artifactId>
            <version>${spring-cloud.version}</version>
            <type>pom</type>
            <scope>import</scope>
        </dependency>
    </dependencies>
</dependencyManagement>
```

## Proyectos Principales

| Proyecto | Descripción |
|----------|-------------|
| Spring Cloud Config | Configuración centralizada |

| Spring Cloud Eureka | Descubrimiento de servicios |
