# JMeter 安裝指南

## 前提條件

- **Java Development Kit (JDK)**：JMeter 需要 Java 8 或更高版本
- **作業系統**：Windows、Linux 或 macOS
- **記憶體**：建議至少 4GB RAM 用於負載測試

## 安裝步驟

### 1. 下載 JMeter

造訪官方 Apache JMeter 網站：https://jmeter.apache.org/

下載最新版本（通常為 .zip 或 .tar.gz 檔案）。

### 2. 解壓縮存檔

將下載的檔案解壓縮到您想要的位置：

```bash
# Linux/Mac
tar -xvf apache-jmeter-x.x.x.tgz

# Windows
# 使用 7-Zip 或 WinRAR 解壓縮
```

### 3. 設定環境變數（可選）

為方便起見，您可以將 JMeter 新增到系統 PATH：

```bash
# Linux/Mac（新增到 ~/.bashrc 或 ~/.zshrc）
export JMETER_HOME=/path/to/apache-jmeter-x.x.x
export PATH=$PATH:$JMETER_HOME/bin

# Windows
# 新增到系統屬性 -> 環境變數 -> PATH
```

### 4. 驗證安裝

執行以下命令以驗證 JMeter 已正確安裝：

```bash
# Linux/Mac
./jmeter.sh --version

# Windows
jmeter.bat --version
```

### 5. 啟動 JMeter GUI

```bash
# Linux/Mac
./jmeter.sh

# Windows
jmeter.bat
```

## 目錄結構

```
apache-jmeter/
├── bin/          # 執行指令碼和 JMeter 執行檔
├── lib/          # JMeter 函式庫
├── lib/ext/      # 外掛和擴充功能
├── docs/         # 文件
├── printable_docs/  # 使用者手冊
└── LICENSE       # 授權檔案
```

## 以不同模式執行

### GUI 模式（互動式）
```bash
jmeter.sh          # Linux/Mac
jmeter.bat         # Windows
```

### CLI 模式（無頭）
```bash
jmeter -n -t testplan.jmx -l results.jtl -e -o html-report
```

參數：
- `-n`：以非 GUI 模式執行
- `-t`：測試計劃檔案
- `-l`：結果檔案
- `-e`：測試後產生 HTML 報告
- `-o`：HTML 報告的輸出目錄

## 安裝疑難排解

### 找不到 Java
確保 JAVA_HOME 已正確設定：
```bash
export JAVA_HOME=/path/to/jdk
```

### 記憶體問題
編輯 jmeter 啟動指令碼以增加堆積大小：
```bash
HEAP="-Xms1g -Xmx4g"
```

## 後續步驟

安裝後，請繼續閱讀使用者手冊以了解如何建立和執行測試計劃。
