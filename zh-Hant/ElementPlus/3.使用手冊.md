# Element Plus 使用手冊

Element Plus 核心功能和使用技巧。

## 快速開始

### 第一個示例

```vue
<template>
  <el-button type="primary" @click="handleClick">
    點擊我
  </el-button>
</template>

<script setup>
import { ref } from 'vue'

const handleClick = () => {
  ElMessage.success('Hello Element Plus!')
}
</script>
```

## 組件使用

### Button 按鈕

```vue
<template>
  <el-button>默認</el-button>
  <el-button type="primary">主要</el-button>
  <el-button type="success">成功</el-button>
  <el-button type="warning">警告</el-button>
  <el-button type="danger">危險</el-button>
  <el-button disabled>禁用</el-button>
  <el-button loading>加載中</el-button>
</template>
```

### Input 輸入框

```vue
<template>
  <el-input v-model="value" placeholder="請輸入內容" />
  <el-input v-model="value" disabled placeholder="禁用狀態" />
  <el-input v-model="value" clearable placeholder="可清空" />
</template>

<script setup>
import { ref } from 'vue'
const value = ref('')
</script>
```

### Table 表格

```vue
<template>
  <el-table :data="tableData" stripe border>
    <el-table-column prop="date" label="日期" width="180" />
    <el-table-column prop="name" label="姓名" width="180" />
    <el-table-column prop="address" label="地址" />
  </el-table>
</template>

<script setup>
import { ref } from 'vue'

const tableData = ref([
  { date: '2024-01-01', name: '張三', address: '北京市' },
])
</script>
```

### Form 表單

```vue
<template>
  <el-form :model="form" :rules="rules" ref="formRef" label-width="100px">
    <el-form-item label="用戶名" prop="username">
      <el-input v-model="form.username" />
    </el-form-item>
    <el-form-item>
      <el-button type="primary" @click="submitForm">提交</el-button>
      <el-button @click="resetForm">重置</el-button>
    </el-form-item>
  </el-form>
</template>

<script setup>
import { ref, reactive } from 'vue'

const formRef = ref()
const form = reactive({ username: '' })

const rules = {
  username: [{ required: true, message: '請輸入用戶名', trigger: 'blur' }],
}

const submitForm = async () => {
  const valid = await formRef.value.validate()
  if (valid) console.log('提交成功', form)
}

const resetForm = () => formRef.value.resetFields()
</script>
```

### Dialog 對話框

```vue
<template>
  <el-button type="primary" @click="visible = true">
    打開對話框
  </el-button>
  
  <el-dialog v-model="visible" title="提示" width="500">
    <span>這是一段內容</span>
  </el-dialog>
</template>

<script setup>
import { ref } from 'vue'
const visible = ref(false)
</script>
```

### Message 消息提示

```typescript
import { ElMessage } from 'element-plus'

ElMessage.success('操作成功')
ElMessage.error('操作失敗')
```
