# Go インストールガイド

> 環境の完全セットアップ

---

## 必要環境

| 環境 | 要件 |
|------|------|
| OS | Linux, macOS, Windows |
| アーキテクチャ | amd64, arm64, 386 |

---

## インストール

### Windows

```powershell
choco install golang
```

### macOS

```bash
brew install go
```

### Linux

```bash
sudo apt install golang-go
```

---

## 設定

### プロキシ（中国）

```bash
go env -w GOPROXY=https://goproxy.cn,direct
```

### 確認

```bash
go version
```
