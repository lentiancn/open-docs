# Bootstrap 5 安装文档

在所有主流平台上安装 Bootstrap 5 的完整指南。

---

## 什么是 Bootstrap 5？

Bootstrap 5 是构建响应式、移动优先网站的流行 CSS 框架的最新版本。它具有新组件、更快的 JavaScript，并且不再依赖 jQuery。

---

## CDN

### 快速开始

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Bootstrap 5 页面</title>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <h1>你好，Bootstrap 5！</h1>
  
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### CDN 提供商

| 提供商 | CSS | JS |
|--------|-----|-----|
| jsDelivr | https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css | https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js |
| unpkg | https://unpkg.com/bootstrap@5.3.2/dist/css/bootstrap.min.css | https://unpkg.com/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js |

### 图标 CDN

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.1/font/bootstrap-icons.css">
```

---

## NPM

### 安装 Bootstrap 5

```bash
# 安装 Bootstrap
npm install bootstrap

# 安装 Bootstrap 图标
npm install bootstrap-icons
```

---

## 下载

### 手动下载

1. 访问 https://getbootstrap.com/docs/5.3/getting-started/download/
2. 下载编译好的 CSS 和 JS
3. 解压并引入项目
