# Guide dutilisation Vue 2

Guide pour utiliser Vue 2 efficacement.

---

## Concepts de base

Vue 2 est un framework JavaScript pour construire des interfaces utilisateur.

---

## Instance

```javascript
new Vue({
  el: '#app',
  data: {
    message: 'Hello Vue!'
  }
})
```

---

## Directives

| Directive | Description |
|------------|-------------|
| `v-if` | Rendu conditionnel |
| `v-for` | Rendu de liste |
| `v-bind` | Liaison dattributs |
| `v-model` | Liaison bidirectionnelle |

---

## Composants

```vue
<template>
  <div>{{ title }}</div>
</template>

<script>
export default {
  props: {
    title: String
  }
}
</script>
```
