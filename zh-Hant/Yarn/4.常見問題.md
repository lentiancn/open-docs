# Yarn 常見問題

## 基本問題

### 什麼是 Yarn？

Yarn 是一個 JavaScript 套件管理器，由 Facebook（現 Meta）開發並維護。它幫助你安裝、更新、配置和刪除項目中的依賴包。

### Yarn 和 npm 有什麼區別？

- 速度：Yarn 使用並行下載，通常比 npm 更快
- 確定性：Yarn 的 yarn.lock 確保每次安裝結果一致
- 離線支持：Yarn 有強大的離線緩存功能
- 工作區：Yarn 原生支持 monorepo 工作區

### Yarn 免費嗎？

是的，Yarn 完全免費，採用 MIT 許可證。

## 安裝問題

### 安裝失敗怎麼辦？

1. 檢查 Node.js 版本
2. 清理 npm 緩存
3. 使用 Corepack 安裝

### 如何升級 Yarn？

```bash
corepack prepare yarn@stable --activate
```

---

## 使用問題

### 如何添加依賴？

```bash
yarn add lodash
yarn add -D typescript
```

### 如何更新依賴？

```bash
yarn up lodash
yarn up --all
```

### 依賴安裝失敗？

1. 清理緩存
2. 刪除 node_modules 和 yarn.lock
3. 重新安裝

## 工作區問題

### 什麼是工作區？

工作區允許你在單個程式碼儲存庫中管理多個相關的套件。

### 如何配置工作區？

```json
{
  "workspaces": ["packages/*"]
}
```
