# Bootstrap 簡介

最受歡迎的前端響應式 CSS 框架。

## 什麼是 Bootstrap？

Bootstrap 是由 Twitter 開發的最受歡迎的前端 CSS 框架，用於構建響應式、移動優先的網站和 Web 應用。

### 主要特性

- **響應式網格系統** - 12 列響應式網格
- **預製組件** - 導航欄、卡片、模態框等
- **CSS 工具類** - 快速樣式調整
- **JavaScript 組件** - 互動式組件
- **自定義選項** - SASS 變量自定義
- **瀏覽器兼容** - 支持所有現代瀏覽器
