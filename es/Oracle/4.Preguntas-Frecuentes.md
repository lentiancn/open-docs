# Oracle Preguntas Frecuentes

## Preguntas Generales

### ¿Qué es Oracle Database?

Oracle Database es un sistema de gestión de bases de datos relacional patentado desarrollado por Oracle Corporation.

### ¿Qué es PL/SQL?

PL/SQL es la extensión del lenguaje procedimental de Oracle para SQL.

---

## Instalación

### ¿Requisitos del sistema?

- Mínimo 2GB de RAM
- Mínimo 8GB de espacio en disco

### ORA-12541: TNS:no listener?

Inicie el listener:
```bash
lsnrctl start
```

---

## Rendimiento

### ¿Cómo verificar rendimiento de consultas?

```sql
EXPLAIN PLAN FOR your_query;
SELECT * FROM TABLE(DBMS_XPLAN.DISPLAY);
```
