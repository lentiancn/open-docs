# Rukovodstvo po ustanovke Vue 3

Rukovodstvo po ustanovke Vue 3.

---

## CDN

```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
```

---

## NPM

```bash
npm install vue@3
```

---

## Vite (Rekomenduyetsya)

```bash
npm create vite@latest my-vue3-app -- --template vue
cd my-vue3-app
npm install
npm run dev
```
