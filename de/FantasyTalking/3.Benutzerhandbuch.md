# FantasyTalking Benutzerhandbuch

## Sprachsynthese

```javascript
const audio = await client.speak('Text hier');
```

## Streaming

```javascript
const stream = await client.speakStream('langer Text');
stream.on('data', chunk => player.play(chunk));
```

## Best Practices

- Verbindungen wiederverwenden
- Stapelverarbeitung nutzen
- Caching verwenden
