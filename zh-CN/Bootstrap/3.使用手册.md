# 使用手册

本手册详细介绍 Bootstrap 5 的核心功能和使用方法，包括网格系统、组件、CSS 工具类等。

## 网格系统

网格系统是 Bootstrap 的核心功能，提供 12 列布局。

### 基本用法

```html
<div class="container">
    <div class="row">
        <div class="col">列 1</div>
        <div class="col">列 2</div>
        <div class="col">列 3</div>
    </div>
</div>
```

### 响应式断点

| 类名 | 屏幕宽度 | 描述 |
|------|---------|------|
| `.col-xs` | < 576px | 超小屏幕 |
| `.col-sm` | ≥ 576px | 小屏幕 |
| `.col-md` | ≥ 768px | 中等屏幕 |
| `.col-lg` | ≥ 992px | 大屏幕 |
| `.col-xl` | ≥ 1200px | 超大屏幕 |
| `.col-xxl` | ≥ 1400px | 特大屏幕 |

### 列偏移

```html
<div class="row">
    <div class="col-md-4">.col-md-4</div>
    <div class="col-md-4 offset-md-4">.col-md-4 .offset-md-4</div>
</div>
```

### 响应式列

```html
<div class="row">
    <div class="col-12 col-md-6 col-lg-4">
        在手机上占满宽度，平板上半宽，大屏上1/3宽
    </div>
</div>
```

## 最佳实践

1. **使用语义化 HTML**：合理使用 header、nav、main、footer 等标签
2. **保持简洁**：避免过度嵌套
3. **响应式优先**：从移动端开始设计
4. **利用工具类**：快速调整样式
5. **自定义主题**：使用 Sass 变量保持一致性
