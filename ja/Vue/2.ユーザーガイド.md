このガイドでは、アプリケーションでVue.jsを効率的に使用方法について説明します。

## リアクティブ

### Vue 3 Composition API（推奨）

```javascript
import { ref, reactive, computed } from 'vue'

export default {
  setup() {
    // refはプリミティブ値用
    const count = ref(0)
    
    // reactiveはオブジェクト用
    const state = reactive({
      name: 'Vue',
      version: 3
    })
    
    // コンピューテッド
    const doubleCount = computed(() => count.value * 2)
    
    // メソッド
    const increment = () => {
      count.value++
    }
    
    return { count, state, doubleCount, increment }
  }
}
```

### Vue 2 Options API

```javascript
export default {
  data() {
    return {
      count: 0,
      name: 'Vue',
      version: 2
    }
  },
  computed: {
    doubleCount() {
      return this.count * 2
    }
  },
  methods: {
    increment() {
      this.count++
    }
  }
}
```

## コンポーネント

### コンポーネントの作成

```vue
<!-- MyComponent.vue -->
<template>
  <div class="my-component">
    <h2>{{ title }}</h2>
    <p>{{ content }}</p>
    <button @click="emitEvent">クリック</button>
  </div>
</template>

<script setup>
import { defineProps, defineEmits } from 'vue'

const props = defineProps({
  title: {
    type: String,
    required: true
  },
  content: {
    type: String,
    default: 'デフォルトコンテンツ'
  }
})

const emit = defineEmits(['click'])

const emitEvent = () => {
  emit('click', 'Hello from component!')
}
</script>

<style scoped>
.my-component {
  padding: 20px;
}
</style>
```

### コンポーネントの使用

```vue
<!-- ParentComponent.vue -->
<template>
  <MyComponent 
    title="マイタイトル"
    content="カスタムコンテンツ"
    @click="handleClick"
  />
</template>

<script setup>
import MyComponent from './MyComponent.vue'

const handleClick = (message) => {
  console.log(message)
}
</script>
```

## コンポーネント通信

### Props（親から子）

```vue
<!-- Child.vue -->
<script setup>
defineProps({
  message: String,
  count: Number
})
</script>

<template>
  <p>{{ message }} - {{ count }}</p>
</template>

<!-- Parent.vue -->
<template>
  <Child message="Hello" :count="5" />
</template>
```

### イベント（子から親）

```vue
<!-- Child.vue -->
<script setup>
const emit = defineEmits(['update'])

const sendData = () => {
  emit('update', { name: 'Vue' })
}
</script>

<template>
  <button @click="sendData">送信</button>
</template>

<!-- Parent.vue -->
<template>
  <Child @update="handleUpdate" />
</template>

<script setup>
const handleUpdate = (data) => {
  console.log(data)
}
</script>
```

### Provide/Inject

```vue
<!-- Parent.vue -->
<script setup>
import { provide } from 'vue'

provide('theme', 'dark')
provide('user', { name: 'John' })
</script>

<!-- Child.vue -->
<script setup>
import { inject } from 'vue'

const theme = inject('theme')
const user = inject('user')
</script>
```

## イベント処理

```html
<!-- クリックイベント -->
<button @click="handleClick">クリック</button>

<!-- イベントオブジェクト -->
<button @click="handleClick($event)">クリック</button>

<!-- イベント修飾子 -->
<button @click.stop="handleClick">バブリング停止</button>
<button @click.prevent="handleSubmit">デフォルト防止</button>

<!-- キーボード修飾子 -->
<input @keyup.enter="submit" />
<input @keyup.ctrl.enter="submit" />
```

## フォーム入力バインディング

```vue
<template>
  <form @submit.prevent="submitForm">
    <!-- テキスト入力 -->
    <input v-model="form.name" type="text" />
    
    <!-- チェックボックス -->
    <input v-model="form.checked" type="checkbox" />
    
    <!-- ラジオボタン -->
    <input v-model="form.gender" value="male" type="radio">男
    <input v-model="form.gender" value="female" type="radio">女
    
    <!-- セレクター -->
    <select v-model="form.country">
      <option value="cn">中国</option>
      <option value="us">アメリカ</option>
    </select>
    
    <button type="submit">送信</button>
  </form>
</template>

<script setup>
import { reactive } from 'vue'

const form = reactive({
  name: '',
  checked: false,
  gender: '',
  country: ''
})

const submitForm = () => {
  console.log(form)
}
</script>
```

## コンピューテッドプロパティ

```javascript
import { ref, computed } from 'vue'

const firstName = ref('John')
const lastName = ref('Doe')

// コンピューテッド - キャッシュ、依存が変更されたときのみ再評価
const fullName = computed(() => {
  return `${firstName.value} ${lastName.value}`
})
```

## ウォッチャー

```javascript
import { ref, watch } from 'vue'

const count = ref(0)

// シンプルなウォッチャー
watch(count, (newValue, oldValue) => {
  console.log(`${oldValue}から${newValue}に変更`)
})

// ディープウォッチャー
const state = ref({ name: 'Vue' })
watch(state, (newValue) => {
  console.log('状態が変更されました:', newValue)
}, { deep: true })
```

## ライフサイクルフック

### Vue 3 Composition API

| フック | 説明 |
|--------|------|
| `setup()` | 他のフックの前に呼ばれる |
| `onMounted()` | コンポーネントがマウントされた後 |
| `onUpdated()` | コンポーネントが更新された後 |
| `onUnmounted()` | コンポーネントがアンマウントされた後 |

```javascript
import { onMounted, onUpdated, onUnmounted } from 'vue'

export default {
  setup() {
    onMounted(() => {
      console.log('マウントされました!')
    })
    onUpdated(() => {
      console.log('更新されました!')
    })
    onUnmounted(() => {
      console.log('アンマウントされました!')
    })
  }
}
```

### Vue 2 Options API

| フック | 説明 |
|--------|------|
| `beforeCreate` | インスタンス作成前 |
| `created` | インスタンス作成後 |
| `beforeMount` | マウント前 |
| `mounted` | マウント後 |
| `beforeUpdate` | 更新前 |
| `updated` | 更新後 |
| `beforeDestroy` | 破棄前 |
| `destroyed` | 破棄後 |

## Vue Router

### 基本設定

```javascript
// router/index.js
import { createRouter, createWebHistory } from 'vue-router'
import Home from '../views/Home.vue'
import About from '../views/About.vue'

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```

### ナビゲーション

```vue
<template>
  <!-- プログラムによるナビゲーション -->
  <button @click="$router.push('/about')">概要</button>
  
  <!-- Router Link -->
  <router-link to="/">ホーム</router-link>
  <router-link to="/about">概要</router-link>
</template>
```

## Pinia 状態管理

### ストア定義

```javascript
// stores/user.js
import { defineStore } from 'pinia'

export const useUserStore = defineStore('user', {
  state: () => ({
    name: 'John',
    email: 'john@example.com'
  }),
  
  getters: {
    displayName: (state) => `${state.name} (${state.email})`
  },
  
  actions: {
    updateName(newName) {
      this.name = newName
    }
  }
})
```

### ストアの使用

```vue
<script setup>
import { useUserStore } from '../stores/user'

const userStore = useUserStore()

// 状態に直接アクセス
console.log(userStore.name)

// アクションを呼ぶ
userStore.updateName('Jane')
</script>
```
