# Element Plus インストールガイド

Element Plusの複数のインストール方法。

## 環境要件

### ハードウェア要件

- CPU: 2コア以上
- メモリ: 4GB以上

### ソフトウェア要件

- Node.js: 18.0以上
- npm: 9.0以上 / yarn: 1.22以上 / pnpm: 8.0以上
- Vue: 3.2以上
- Vite: 4.0以上（推奨）

## パッケージマネージャーでのインストール（推奨）

### npm

```bash
npm install element-plus
```

### yarn

```bash
yarn add element-plus
```

### pnpm

```bash
pnpm add element-plus
```

## 完全インポート

### main.tsの設定

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')
```

## 必要に応じたインポート（推奨）

### 自動インポート

**1. 依存関係をインストール**

```bash
npm install -D unplugin-vue-components unplugin-auto-import
```

**2. Viteを設定**

```typescript
// vite.config.ts
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

**3. コンポーネントを使用**

```vue
<template>
  <el-button type="primary">こんにちは</el-button>
</template>
```

### 手動で必要に応じてインポート

```typescript
import { ElButton, ElInput } from 'element-plus'

export default {
  components: {
    ElButton,
    ElInput,
  },
}
```

## Vue CLIでのインストール

### インストール

```bash
npm install -D sass sass-loader
npm install element-plus
```

### 設定

```javascript
// vue.config.js
module.exports = {
  configureWebpack: {
    plugins: [
      AutoImport({
        resolvers: [ElementPlusResolver()],
      }),
      Components({
        resolvers: [ElementPlusResolver()],
      }),
    ],
  },
}
```

## CDNからインポート

### HTML

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>Element Plus CDN</title>
  <link rel="stylesheet" href="//unpkg.com/element-plus/dist/index.css" />
  <script src="//unpkg.com/vue@3"></script>
  <script src="//unpkg.com/element-plus"></script>
</head>
<body>
  <div id="app">
    <el-button type="primary">プライマリーボタン</el-button>
  </div>

  <script>
    const { createApp, ref } = Vue
    const App = {
      setup() {
        const visible = ref(false)
        return { visible }
      },
      template: `
        <el-button @click="visible = true">ダイアログを開く</el-button>
        <el-dialog v-model="visible" title="タイトル">内容</el-dialog>
      `
    }
    const app = createApp(App)
    app.use(ElementPlus)
    app.mount('#app')
  </script>
</body>
</html>
```


## 一般的な問題

### アイコンが表示されない？

アイコンパッケージをインストール：

```bash
npm install @element-plus/icons-vue
```

登録：

```typescript
import * as icons from '@element-plus/icons-vue'

for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```

### スタイルが競合する？

CSS Modulesまたはscopedを使用：

```vue
<style scoped>
:deep(.el-button) {
  margin: 10px;
}
</style>
```

### TypeScriptの型エラー？

型定義がインストールされていることを確認：

```bash
npm install -D @types/node
```

tsconfig.jsonに設定：

```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

### 国際化

```typescript
import ja from 'element-plus/es/locale/lang/ja'

app.use(ElementPlus, { locale: ja })
```

## アンインストール

### npmでアンインストール

```bash
npm uninstall element-plus
```

### 設定をクリーンアップ

削除するもの：
- element-variables.scss（カスタムテーマを使用した場合）
- 関連するVite/Webpack設定
