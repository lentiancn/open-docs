# Element UI 安裝指南

本指南詳細介紹在各種項目中安裝和使用 Element UI 的方法。

## 環境要求

Element UI 對運行環境有以下要求：

- Vue 2.0.0+
- Node.js 8.0.0+
- 現代瀏覽器（Chrome、Firefox、Safari、Edge 10+）

## 安裝方式

### 使用包管理器

推薦使用 npm 安裝：

```bash
npm install element-ui --save
```

或者使用 yarn：

```bash
yarn add element-ui
```

## 完整引入

在 main.js 中引入：

```javascript
import Vue from 'vue'
import Element from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(Element)

new Vue({
  el: '#app',
  render: h => h(App)
})
```

### 按需導入

按需導入可以顯著減少項目體積。

首先安裝 babel-plugin-component：

```bash
npm install babel-plugin-component -D
```

然後在 .babelrc 中配置：

```json
{
  "presets": [["env", {
    "modules": false,
    "targets": {
      "browsers": ["> 1%", "last 2 versions", "not ie <= 8"]
    }
  }]],
  "plugins": [
    ["component", {
      "libraryName": "element-ui",
      "styleLibraryName": "theme-chalk"
    }]
  ]
}
```

按需引入組件：

```javascript
import { Button, Select } from 'element-ui'

Vue.component(Button.name, Button)
Vue.component(Select.name, Select)
```

## 快捷搭建項目模板

Element UI 提供了官方項目模板：

- Vue CLI 模板：https://github.com/ElementUI/element-starter
- Laravel 模板：https://github.com/ElementUI/element-in-laravel-starter

## 全局配置

在引入 Element UI 時，可以傳入全局配置對象：

```javascript
import Vue from 'vue'
import Element from 'element-ui'

Vue.use(Element, {
  size: 'small',
  zIndex: 2000
})
```

配置項說明：
- size：設置表單組件的默認尺寸，可選值：medium / small / mini
- zIndex：設置彈出組件的層級，默認值為 2000

更多信息請訪問 Element UI 官網。
