# OpenClaw 使用手册

本文档介绍 OpenClaw 的日常使用方法和各项功能。

## 核心概念

### Gateway

Gateway 是 OpenClaw 的核心组件，所有的消息转发、AI 调用和渠道管理都通过它完成。你只需要运行这一个进程，它会自动处理来自各个聊天平台 的请求。

### 渠道

渠道指的是你用来和 AI 交流的聊天应用。OpenClaw 支持同时连接多个渠道，包括微信国际版、Telegram、Discord 等。

### 代理

代理是 AI 助手的独立实例。你可以配置多个代理，分别处理不同的工作任务，每个代理有独立的对话历史和工作区。

## 日常操作

### 启动和停止服务

```bash
# 启动服务（后台运行）
openclaw gateway start

# 停止服务
openclaw gateway stop

# 重启服务
openclaw gateway restart

# 查看运行状态
openclaw gateway status

# 前台运行（调试时使用）
openclaw gateway --port 18789
```

### 发送消息

```bash
# 发送私信
openclaw message send --target +15555550123 --message "测试消息"

# 发送群组消息
openclaw message send --target "群名称" --message "大家好"
```

### 配置管理

OpenClaw 使用 JSON5 格式的配置文件，位于 `~/.openclaw/openclaw.json`。

修改配置有以下几种方式：

1. **向导模式**：
   ```bash
   openclaw onboard       # 完整设置向导
   openclaw configure    # 配置向导
   ```

2. **命令行模式**：
   ```bash
   openclaw config get agents.defaults.workspace
   openclaw config set agents.defaults.heartbeat.every "2h"
   openclaw config unset tools.web.search.apiKey
   ```

3. **网页界面**：
   打开 http://127.0.0.1:18789/ ，在 Config 标签页中修改

4. **直接编辑**：
   ```bash
   nano ~/.openclaw/openclaw.json
   ```

### 热重载

修改配置文件后无需手动重启服务，大部分配置会自动生效。

## 常用配置示例

### 限制可访问的用户

```json5
{
  channels: {
    whatsapp: {
      allowFrom: ["+15555550123"]
    }
  }
}
```

### 设置群组消息必须@

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

### 配置多个独立代理

```json5
{
  agents: {
    list: [
      { id: "home", default: true, workspace: "~/.openclaw/workspace-home" },
      { id: "work", workspace: "~/.openclaw/workspace-work" }
    ]
  }
}
```

### 定时任务

```json5
{
  cron: {
    enabled: true,
    maxConcurrentRuns: 2
  }
}
```

### 定期心跳检测

```json5
{
  agents: {
    defaults: {
      heartbeat: {
        every: "30m",
        target: "last"
      }
    }
  }
}
```

### 沙箱隔离

如果需要在隔离环境中运行 AI 操作：

```json5
{
  agents: {
    defaults: {
      sandbox: {
        mode: "non-main",
        scope: "agent"
      }
    }
  }
}
```

### Webhook 设置

```json5
{
  hooks: {
    enabled: true,
    token: "你的密钥",
    path: "/hooks"
  }
}
```

## 渠道管理

### 添加新渠道

```bash
# 登录微信
openclaw channels login whatsapp

# 登录 Telegram
openclaw channels login telegram
```

### 查看已连接的渠道

```bash
openclaw channels list
```

## 问题排查

### 查看日志

```bash
# 查看服务日志
openclaw logs

# 实时查看日志
openclaw logs -f
```

### 健康检查

```bash
openclaw health
```

### 自动诊断

```bash
openclaw doctor
```

## 网页控制台

除了命令行，你也可以通过网页界面管理 OpenClaw：

- 发送和接收消息
- 修改各项配置
- 查看对话历史
- 管理手机节点
- 监控服务状态

访问地址：http://127.0.0.1:18789/
