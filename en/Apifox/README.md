# Apifox

Apifox is a leading API collaboration platform in China that solves API design, mocking, testing, and documentation all in one place. It brings together API design, API documentation management, API debugging, API mocking, and API automated testing—making it one of the most popular API management tools in China.

## Quick Start

### Download and Install

Visit https://apifox.com/ to download the installer for your system.

### Create a Project

1. Launch Apifox and log in to your account
2. Click "New Project"
3. Enter a project name

### Create an Interface

1. In your project, click "New Interface"
2. Fill in the interface information
3. Save

### Debug an Interface

1. Open the interface
2. Click "Send"
3. View the response

## Core Features

- **API Design**: Visually define API interfaces
- **API Documentation**: Auto-generates documentation with multiple export formats
- **API Debugging**: Powerful online debugging features
- **API Mocking**: Smart mocking that automatically generates simulated data
- **Automated Testing**: Complete API testing without writing code
- **Team Collaboration**: Real-time sync and permission management
