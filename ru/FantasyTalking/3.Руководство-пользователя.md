# FantasyTalking - Руководство пользователя

Подробное руководство.

## Основное использование

### Генерация видео

```bash
python inference.py --source_image face.jpg --audio speech.wav --output output.mp4
```

### Параметры

| Параметр | Описание |
|----------|----------|
| --source_image | Входное изображение |
| --audio | Аудиофайл |
| --output | Путь к выводу |
| --face_enhancer | Улучшение лица |
| --fps | Кадры в секунду |

## Python API

```python
from fantasytalking import FantasyTalking

ft = FantasyTalking()
ft.generate(
    source_image="face.jpg",
    audio="speech.wav",
    output="output.mp4"
)
```

## Пакетная обработка

```python
ft.batch_generate(image_list, audio_list, output_dir)
```

## Требования к вводу

### Изображение

- Формат: JPG, PNG
- Разрешение: 512x512+
- Лицо: Фронтальное, чёткое

### Аудио

- Формат: WAV, MP3
- Длительность: 1-60 секунд
