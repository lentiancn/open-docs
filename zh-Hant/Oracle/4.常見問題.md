# Oracle 常見問題

## 一般問題

### 什麼是 Oracle Database？

Oracle Database 是由 Oracle 公司開發的專屬關聯式資料庫管理系統。

### 什麼是 PL/SQL？

PL/SQL 是 Oracle 的 SQL 程式語言擴展，允許開發者撰寫複雜的商業邏輯。

---

## 安裝

### 系統需求為何？

- 最少 2GB RAM
- 最少 8GB 磁碟空間

### ORA-12541: TNS:no listener？

啟動監聽器：
```bash
lsnrctl start
```

## 效能

### 如何檢查查詢效能？

```sql
EXPLAIN PLAN FOR 您的查詢;
SELECT * FROM TABLE(DBMS_XPLAN.DISPLAY);
```
