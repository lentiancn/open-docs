# JUnit 4 安装文档

在所有主流平台上安装 JUnit 4 的完整指南。

---

## 什么是 JUnit 4？

JUnit 4 是 Java 单元测试框架，是 Java 生态系统中使用最广泛的测试框架。

---

## Maven

### 添加依赖

```xml
<dependency>
    <groupId>junit</groupId>
    <artifactId>junit</artifactId>
    <version>4.13.2</version>
    <scope>test</scope>
</dependency>
```

---

## Gradle

### 添加依赖

```groovy
testImplementation 'junit:junit:4.13.2'
```

---

## Eclipse

### 使用 Eclipse Marketplace

1. 进入 **Help > Eclipse Marketplace**
2. 搜索 **JUnit**
3. 安装 **JUnit**

### 手动设置

1. 从 https://github.com/junit-team/junit4/releases 下载 JUnit jar
2. 右键项目 → **Build Path > Add External Archives**
3. 选择 **junit-4.13.2.jar** 和 **hamcrest-core-1.3.jar**

---

## IntelliJ IDEA

### 使用 Maven/Gradle

添加 Maven 或 Gradle 依赖后，JUnit 自动可用。
