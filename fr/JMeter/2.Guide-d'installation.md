# Guide d'installation de JMeter

Ce guide fournit des instructions détaillées pour installer et configurer Apache JMeter sur différents systèmes d'exploitation.

## Configuration requise

- **Environnement d'exécution Java** : JMeter nécessite Java 8 ou supérieur
- **Système d'exploitation** : Supporte tout système d'exploitation compatible Java, y compris Linux, Windows et Mac OSX
- **Mémoire** : Au moins 4 Go de mémoire disponible recommandés (plus peut être nécessaire selon l'échelle des tests)
- **Espace disque** : Au moins 1 Go d'espace disponible recommandé

## Téléchargement de JMeter

1. Visitez la page de téléchargement officielle de JMeter : https://jmeter.apache.org/download_jmeter.cgi
2. Téléchargez la dernière version binaire de JMeter (Binaries)
3. Il est recommandé de télécharger également les fichiers de signature PGP ou SHA-512 pour vérifier l'intégrité du téléchargement

## Installation sur Windows

### Extraction

1. Localisez l'archive JMeter Apache téléchargée (apache-jmeter-x.x.zip)
2. Choisissez un répertoire d'installation approprié (évitez les chemins avec des espaces et des caractères spéciaux)
3. Extrayez l'archive dans le répertoire sélectionné
4. JMeter est maintenant installé

### Vérification

1. Accédez au dossier bin dans le répertoire d'installation de JMeter
2. Double-cliquez sur jmeter.bat (fichier batch Windows)
3. Si tout se passe bien, l'interface graphique JMeter se lancera

## Installation sur Linux et Mac OSX

### Extraction

1. Localisez l'archive JMeter Apache téléchargée (apache-jmeter-x.x.tar.gz)
2. Extrayez en utilisant la commande suivante :
   ```bash
   tar -xf apache-jmeter-x.x.tar.gz
   ```
3. JMeter est maintenant installé

### Variables d'environnement (Optionnel)

1. Ouvrez votre fichier de configuration terminal (~/.bashrc ou ~/.profile)
2. Ajoutez les variables d'environnement suivantes :
   ```bash
   export JMETER_HOME=/path/to/apache-jmeter-x.x
   export PATH=$PATH:$JMETER_HOME/bin:$JMETER_HOME/lib
   ```
3. Sauvegardez et actualisez la configuration :
   ```bash
   source ~/.bashrc
   ```

### Vérification

1. Exécutez ce qui suit dans le terminal :
   ```bash
   jmeter --version
   ```
2. Si les informations de version s'affichent, l'installation a réussi

## Démarrage de JMeter

### Mode GUI

- **Windows** : Double-cliquez sur jmeter.bat dans le répertoire bin
- **Linux/Mac** : Exécutez la commande jmeter dans le terminal

### Mode ligne de commande

Pour les tests de charge, le mode ligne de commande est recommandé pour réduire l'utilisation des ressources :

```bash
jmeter -n -t test_plan.jmx -l results.jtl -e -o /path/to/output
```

Paramètres :
- -n : Exécuter en mode non-GUI
- -t : Spécifier le fichier de plan de test
- -l : Spécifier le fichier de résultats
- -e : Générer un rapport après la fin du test
- -o : Spécifier le répertoire de sortie du rapport

## Structure des répertoires

- **bin** : Contient les fichiers exécutables et les scripts de démarrage
- **lib** : Contient les bibliothèques de dépendances JMeter
- **docs** : Contient la documentation officielle JMeter
- **extras** : Contient des outils auxiliaires supplémentaires
- **printable_docs** : Contient le manuel utilisateur imprimable

## Gestion des dépendances

Le répertoire lib de JMeter contient les dépendances core. Pour ajouter des plugins ou bibliothèques supplémentaires :

1. Copiez les fichiers JAR dans le répertoire lib
2. Certains plugins doivent être placed dans le répertoire lib/ext
3. Redémarrez JMeter pour que les nouveaux plugins prennent effet

## Problèmes d'installation courants

- **Java non trouvé** : Assurez-vous que Java est correctement installé et que la variable d'environnement JAVA_HOME est définie
- **Mémoire insuffisante** : Modifiez les paramètres de taille du tas dans bin/jmeter.sh ou jmeter.bat
- **Échec du démarrage** : Vérifiez si la bonne version de Java (Java 8 ou supérieur) est installée
