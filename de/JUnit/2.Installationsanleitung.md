# JUnit 5 Installationsanleitung

## Voraussetzungen

- **Java Development Kit (JDK)**: JUnit 5 erfordert Java 8 oder höher
- **Build-Tool**: Maven oder Gradle (empfohlen)

## Installation mit Maven

### Schritt 1: Abhängigkeit hinzufügen

Fügen Sie die folgende Abhängigkeit zu Ihrer pom.xml hinzu:

```xml
<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter</artifactId>
    <version>5.10.0</version>
    <scope>test</scope>
</dependency>
```

### Schritt 2: Surefire-Plugin konfigurieren

Stellen Sie sicher, dass das maven-surefire-plugin auf dem neuesten Stand ist:

```xml
<plugin>
    <groupId>org.apache.maven.plugins</groupId>
    <artifactId>maven-surefire-plugin</artifactId>
    <version>3.2.2</version>
</plugin>
```

### Schritt 3: Tests ausführen

```bash
mvn test
```

## Installation mit Gradle

### Schritt 1: Abhängigkeit hinzufügen

Fügen Sie die folgende Abhängigkeit zu build.gradle hinzu:

```groovy
dependencies {
    testImplementation 'org.junit.jupiter:junit-jupiter:5.10.0'
}
```

### Schritt 2: JUnit Platform konfigurieren

```groovy
test {
    useJUnitPlatform()
}
```

### Schritt 3: Tests ausführen

```bash
./gradlew test
```

## Installation mit anderen Build-Tools

### Ant
Laden Sie die JAR-Dateien von Maven Central herunter und fügen Sie sie zum Klassenpfad hinzu.

### Ivy
Verwenden Sie die Ivy-Abhängigkeitsdeklaration für JUnit Jupiter.

## Überprüfung der Installation

Erstellen Sie eine einfache Testklasse:

```java
package com.example;

import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;

class SimpleTest {
    
    @Test
    void testAddition() {
        assertEquals(4, 2 + 2);
    }
}
```

Führen Sie die Tests aus. Wenn die Installation erfolgreich war, sollte der Test bestehen.

## JUnit Vintage für JUnit 4 Tests

Wenn Sie JUnit 4 Tests haben, die Sie mit JUnit 5 ausführen möchten:

### Maven

```xml
<dependency>
    <groupId>org.junit.vintage</groupId>
    <artifactId>junit-vintage-engine</artifactId>
    <version>5.10.0</version>
    <scope>test</scope>
</dependency>
```

### Gradle

```groovy
testImplementation 'org.junit.vintage:junit-vintage-engine:5.10.0'
```

## Häufige Probleme

### Keine Tests gefunden
- Stellen Sie sicher, dass useJUnitPlatform() in Gradle konfiguriert ist
- Überprüfen Sie, ob die Testklassen mit @Test von JUnit 5 annotiert sind

### Kompilierungsfehler
- Stellen Sie sicher, dass Java 8 oder höher verwendet wird
- Überprüfen Sie die Abhängigkeitsversionen

### Klassen nicht gefunden
- Überprüfen Sie die Abhängigkeiten in Ihrer pom.xml oder build.gradle
- Führen Sie mvn clean oder gradlew clean aus
