# Element Plus FAQ

## 설치 문제

### 설치 실패

Node.js 버전 확인 (>= 14.0.0)
npm 캐시 지우기: npm cache clean --force

### TypeScript 오류

tsconfig.json에 추가:

{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}

## 사용 문제

### 스타일이 적용되지 않음

CSS 임포트:

import 'element-plus/dist/index.css'

### 아이콘이 표시되지 않음

아이콘 라이브러리 설치:

npm install @element-plus/icons-vue

## 성능 문제

대량 데이터의 경우:
- 페이지네이션 사용
- el-table-v2 (가상 테이블) 사용

## Vue 버전 문제

Element Plus는 Vue 3만 지원합니다.
