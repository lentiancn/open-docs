# Руководство по установке Spring Boot

В этом руководстве описана настройка среды разработки для Spring Boot, включая Java, Maven и Gradle.

## Предварительные требования

### Установка Java

Spring Boot 3.x требует Java 17 или выше. Spring Boot 2.x требует Java 8 или выше.

**Windows:**
1. Скачайте JDK с [Oracle](https://www.oracle.com/java/technologies/downloads/) или [Adoptium](https://adoptium.net/)
2. Запустите установщик
3. Настройте переменные окружения:
   ```
   JAVA_HOME=C:\Program Files\Java\jdk-17
   PATH=%JAVA_HOME%\bin;%PATH%
   ```

**macOS:**
```bash
# Используя Homebrew
brew install openjdk@17

# Добавить в PATH
echo 'export JAVA_HOME=$(/usr/libexec/java_home)' >> ~/.zshrc
source ~/.zshrc
```

**Linux:**
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install openjdk-17-jdk

# Настроить JAVA_HOME
echo 'export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64' >> ~/.bashrc
source ~/.bashrc
```

Проверить установку:
```bash
java -version
```

### Установка Maven

**Windows:**
1. Скачайте с [Apache Maven](https://maven.apache.org/download.cgi)
2. Распакуйте в папку (например, C:\apache-maven)
3. Настройте переменные окружения:
   ```
   MAVEN_HOME=C:\apache-maven
   PATH=%MAVEN_HOME%\bin;%PATH%
   ```

**macOS:**
```bash
brew install maven
```

**Linux:**
```bash
sudo apt install maven
```

Проверить установку:
```bash
mvn -version
```

### Установка Gradle

**Windows:**
1. Скачайте с [Gradle](https://gradle.org/install/)
2. Распакуйте и добавьте в PATH

**macOS:**
```bash
brew install gradle
```

**Linux:**
```bash
sudo apt install gradle
```

Проверить установку:
```bash
gradle -version
```

## Настройка IDE

### IntelliJ IDEA (Рекомендуется)
- Скачайте с [JetBrains](https://www.jetbrains.com/idea/download/)
- Убедитесь, что JDK настроен в структуре проекта
- Установите плагин Spring Boot Assistant

### VS Code
- Установите Extension Pack for Java
- Установите Spring Boot Extension

## Быстрый старт с Spring Initializr

1. Посетите [start.spring.io](https://start.spring.io/)
2. Выберите:
   - Проект: Maven или Gradle
   - Язык: Java
   - Версия Spring Boot
   - Метаданные проекта
3. Добавьте зависимости (Spring Web, Spring Data JPA и т.д.)
4. Нажмите "Generate" для скачивания проекта

## Проверка настройки

```bash
# Проверить Java
java -version

# Проверить Maven
mvn -version

# Проверить Gradle (если используется)
gradle -version

# Создать простой проект Spring Boot и запустить
curl https://start.spring.io/starter.zip -o demo.zip
unzip demo.zip -d demo
cd demo
./mvnw spring-boot:run  # или ./gradlew bootRun
```
