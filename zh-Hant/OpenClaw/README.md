# OpenClaw

OpenClaw - 您的個人 AI 助手 🦞

## 概述

OpenClaw 是一個運行在您自有裝置上的個人 AI 助手。它可以透過您已經在使用的通訊渠道與您對話，包括 WhatsApp、Telegram、Slack、Discord、Google Chat 等 20 多種平台。OpenClaw 可以處理實際工作、管理日程、發送郵件、控制智慧裝置，真正成為您的數位夥伴。

## 特點

- **多渠道接入**：支援 20+ 通訊平台
- **本地優先**：資料儲存在本地，完全可控
- **全天候運行**：隨時為您服務
- **語音互動**：支援語音喚醒和對話
- **技能系統**：可擴展的自動化能力
- **開源免費**：代碼完全開放

## 快速開始

### 安裝

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

或使用 npm：

```bash
npm install -g openclaw@latest
```

### 配置

```bash
openclaw onboard --install-daemon
```

### 使用

```bash
# 開啟 Control UI
openclaw dashboard

# 或發送訊息
openclaw agent --message "Hello"
```
