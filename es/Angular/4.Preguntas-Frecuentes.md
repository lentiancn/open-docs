# Angular Preguntas frecuentes

Preguntas y respuestas comunes.

## Instalación

### Q: ¿La instalación de Angular CLI falló?

**Soluciones**:
1. Limpiar caché npm: `npm cache clean --force`
2. Usar privilegios de administrador
3. Verificar versión de Node.js

### Q: ¿Comando ng no encontrado?

**Soluciones**:
1. Confirmar que Angular CLI está instalado globalmente
2. Verificar variable de entorno PATH
3. Intentar reinstallar

### Q: ¿Versión de Node.js incompatible?

**Soluciones**:
1. Usar nvm para gestionar versiones de Node.js
2. Actualizar Node.js a versión recomendada

## Uso

### Q: ¿Estilos del componente no funcionan?

**Soluciones**:
1. Verificar que ruta de styleUrls es correcta
2. Confirmar selectores CSS correctos
3. Verificar configuración de ViewEncapsulation

### Q: ¿Navegación de rutas falla?

**Soluciones**:
1. Verificar configuración de rutas
2. Confirmar RouterModule importado
3. Verificar sintaxis de routerLink

### Q: ¿Solicitudes HTTP fallan?

**Soluciones**:
1. Confirmar HttpClientModule importado
2. Verificar configuración CORS
3. Verificar URL de API correcta

### Q: ¿Validación de formularios no funciona?

**Soluciones**:
1. Confirmar ReactiveFormsModule importado
2. Verificar configuración de Validators
3. Usar formControlName correctamente

## Rendimiento

### Q: ¿Aplicación carga lento?

**Soluciones**:
1. Habilitar carga perezosa
2. Usar detección de cambios OnPush
3. Comprimir imágenes y recursos
4. Compilar en modo producción

### Q: ¿Fugas de memoria?

**Soluciones**:
1. Cancelar suscripciones
2. Limpiar intervalos
3. Destruir escuchas de eventos

## Versión

### Q: ¿Cómo actualizar Angular?

```bash
ng update @angular/core @angular/cli
```

### Q: ¿Migrar de AngularJS a Angular?

Usar herramienta angular-upgrade para migración gradual.
