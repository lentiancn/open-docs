# Oracle FAQ

## 一般的な質問

### Oracle Databaseとは？

Oracle Databaseは、Oracle社が開発したプロプライエタリなリレーショナルデータベース管理システムです。

### PL/SQLとは？

PL/SQLは、OracleのSQLに対する手続き型言語拡張です。

---

## インストール

### システム要件は？

- 最小2GB RAM
- 最小8GB ディスク容量

### ORA-12541: TNS:no listener？

リスナーを起動：
```bash
lsnrctl start
```

---

## パフォーマンス

### クエリパフォーマンスの確認方法は？

```sql
EXPLAIN PLAN FOR your_query;
SELECT * FROM TABLE(DBMS_XPLAN.DISPLAY);
```
