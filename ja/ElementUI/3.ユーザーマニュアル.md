# Element UI ユーザーマニュアル

## 基本コンポーネント

### Button

```html
<el-button>デフォルト</el-button>
<el-button type="primary">プライマリ</el-button>
<el-button type="success">成功</el-button>
<el-button type="warning">警告</el-button>
<el-button type="danger">危険</el-button>
```

### Layout

```html
<el-row>
  <el-col :span="12"><div></div></el-col>
  <el-col :span="12"><div></div></el-col>
</el-row>
```

## フォームコンポーネント

### Input

```html
<el-input v-model="input" placeholder="入力"></el-input>
```

### Select

```html
<el-select v-model="value" placeholder="選択">
  <el-option label="オプション1" value="1"></el-option>
</el-select>
```

### Form

```html
<el-form :model="form" label-width="100px">
  <el-form-item label="名前">
    <el-input v-model="form.name"></el-input>
  </el-form-item>
</el-form>
```

## データコンポーネント

### Table

```html
<el-table :data="tableData">
  <el-table-column prop="date" label="日付"></el-table-column>
  <el-table-column prop="name" label="名前"></el-table-column>
</el-table>
```

### Pagination

```html
<el-pagination :total="1000" layout="prev, pager, next"></el-pagination>
```

## 通知コンポーネント

### Message

```javascript
this.$message({ message: '成功', type: 'success' })
```

### Alert

```html
<el-alert title="成功" type="success"></el-alert>
```

## ナビゲーションコンポーネント

### Menu

```html
<el-menu>
  <el-menu-item index="1">ナビ1</el-menu-item>
</el-menu>
```

### Tabs

```html
<el-tabs v-model="activeName">
  <el-tab-pane label="タブ1" name="first">内容</el-tab-pane>
</el-tabs>
```

## その他のコンポーネント

### Dialog

```html
<el-button @click="visible = true">開く</el-button>
<el-dialog :visible.sync="visible" title="タイトル">
  <span>内容</span>
</el-dialog>
```
