# FAQ Nacos

## Что такое Nacos?

Nacos — это платформа от Alibaba для обнаружения сервисов и управления конфигурацией.

## Как установить?

```bash
./startup.sh -m standalone

docker run --name nacos -d -p 8848:8848 nacos/nacos-server:v2.3.0
```

## Доступ к консоли

- URL: http://localhost:8848/nacos
- Логин: nacos
- Пароль: nacos
