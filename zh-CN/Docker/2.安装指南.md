# Docker 安装指南

本指南详细介绍在各种操作系统上安装 Docker 的方法。

## 安装方式概述

你可以通过多种方式安装 Docker：

1. **Docker Desktop**：包含 Docker Engine 的桌面应用程序，推荐大多数用户使用
2. **Docker Engine**：仅安装 Docker 核心组件，适合 Linux 服务器或不需要桌面集成的用户
3. **Docker Engine via apt 仓库**：从 Docker 官方 apt 仓库安装
4. **手动安装**：下载 deb 包手动安装

## Docker Desktop 安装

Docker Desktop 是最简单、最快速的入门方式。它包含完整的 Docker 开发环境，包括 Docker Engine、Docker CLI、Docker Compose、Docker Build 和 Kubernetes。

### 系统要求

**macOS：**
- macOS 10.15（Catalina）或更高版本
- 4 GB 以上 RAM
- 虚拟化支持（VT-x 或 AMD-V）

**Windows：**
- Windows 10 64 位专业版、企业版或教育版
- 4 GB 以上 RAM
- WSL 2（推荐）或 Hyper-V

**Linux：**
- Ubuntu 22.04 或更高版本
- Debian 11 或更高版本
- Fedora 39 或更高版本

### Windows 安装注意事项

在 Windows 上安装 Docker Desktop 之前，需要注意以下限制：

- 如果你使用 ufw 或 firewalld 管理防火墙设置，当使用 Docker 暴露容器端口时，这些端口会绕过防火墙规则
- Docker 仅兼容 `iptables-nft` 和 `iptables-legacy`，使用 `nft` 创建的防火墙规则在安装 Docker 的系统上不受支持

### macOS 安装

**方法一：下载安装包**

1. 访问 [Docker Desktop 下载页面](https://www.docker.com/products/docker-desktop/)
2. 下载适用于 Mac 的安装包（Intel 或 Apple Silicon）
3. 双击 .dmg 文件
4. 将 Docker.app 拖入 Applications 文件夹
5. 启动 Docker Desktop

**方法二：Homebrew**

```bash
brew install --cask docker
```

### Linux 安装（Ubuntu 示例）

**方法一：apt 仓库安装**

1. 设置 Docker 的 apt 仓库：

```bash
# 添加 Docker 的官方 GPG 密钥
sudo apt update
sudo apt install ca-certificates curl
sudo install -m 0755 -d /etc/apt/keyrings
sudo curl -fsSL https://download.docker.com/linux/ubuntu/gpg -o /etc/apt/keyrings/docker.asc
sudo chmod a+r /etc/apt/keyrings/docker.asc

# 添加仓库到 Apt 源
echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.asc] https://download.docker.com/linux/ubuntu $(. /etc/os-release && echo "$VERSION_CODENAME") stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

sudo apt update
```

2. 安装 Docker 包：

```bash
sudo apt install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

3. 验证安装：

```bash
sudo docker run hello-world
```

**方法二：便捷脚本安装**

Docker 提供便捷脚本用于非交互式安装：

```bash
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh
```

**注意**：便捷脚本仅推荐用于测试和开发环境，不推荐生产环境使用。

### 卸载旧版本

在安装 Docker 之前，需要卸载任何冲突的包：

```bash
sudo apt remove docker.io docker-compose docker-compose-v2 docker-doc podman-docker containerd runc
```

## Docker Engine 安装（无桌面版）

如果你不需要 Docker Desktop，可以仅安装 Docker Engine。

### Ubuntu 安装

**前提条件**

- Ubuntu 24.04 (Noble) 或更高版本
- 支持的架构：x86_64、armhf、arm64、s390x、ppc64le

**安装步骤**

1. 卸载旧版本：

```bash
sudo apt remove docker.io docker-compose docker-compose-v2 docker-doc podman-docker containerd runc
```

2. 设置 apt 仓库（见上文）

3. 安装 Docker：

```bash
sudo apt install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

4. 启动 Docker 服务：

```bash
sudo systemctl start docker
sudo systemctl enable docker
```

5. 验证安装：

```bash
sudo docker run hello-world
```

### CentOS/RHEL 安装

1. 安装依赖：

```bash
sudo yum install -y yum-utils
```

2. 添加 Docker 仓库：

```bash
sudo yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
```

3. 安装 Docker：

```bash
sudo yum install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

4. 启动 Docker：

```bash
sudo systemctl start docker
sudo systemctl enable docker
```

### Fedora 安装

1. 安装依赖：

```bash
sudo dnf -y install dnf-plugins-core
```

2. 添加 Docker 仓库：

```bash
sudo dnf config-manager --add-repo https://download.docker.com/linux/fedora/docker-ce.repo
```

3. 安装 Docker：

```bash
sudo dnf install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

### Debian 安装

1. 安装依赖：

```bash
sudo apt update
sudo apt install ca-certificates curl gnupg
```

2. 添加 GPG 密钥和仓库：

```bash
sudo install -m 0755 -d /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/debian/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
sudo chmod a+r /etc/apt/keyrings/docker.gpg

echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/debian $(. /etc/os-release && echo "$VERSION_CODENAME") stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
```

3. 安装 Docker：

```bash
sudo apt update
sudo apt install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

## 以非 root 用户运行 Docker

默认情况下，需要使用 sudo 运行 Docker 命令。要允许非特权用户运行 Docker：

```bash
sudo usermod -aG docker $USER
```

然后重新登录或运行：

```bash
newgrp docker
```

## 升级 Docker

### Ubuntu/Debian

```bash
sudo apt update
sudo apt upgrade docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

### CentOS/RHEL/Fedora

```bash
sudo yum update docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
# 或
sudo dnf update docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

## 卸载 Docker

### 卸载 Docker Desktop

- **macOS**：将 Docker.app 拖出 Applications 文件夹
- **Windows**：通过"设置"→"应用"卸载 Docker Desktop

### 卸载 Docker Engine

**Ubuntu/Debian：**

```bash
sudo apt purge docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
sudo rm -rf /var/lib/docker
sudo rm -rf /var/lib/containerd
sudo rm /etc/apt/sources.list.d/docker.list
sudo rm /etc/apt/keyrings/docker.asc
```

**CentOS/RHEL/Fedora：**

```bash
sudo yum remove docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
sudo rm -rf /var/lib/docker
sudo rm -rf /var/lib/containerd
```

## 验证安装

安装完成后，运行以下命令验证：

```bash
docker --version
docker-compose --version
docker run hello-world
```

如果都能正常运行，说明 Docker 安装成功。
