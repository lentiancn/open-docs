# FantasyTalking 사용 매뉴얼

## 기본 기능

FantasyTalking은 포괄적인 음성 합성 및 대화 기능을 제공합니다. 이 매뉴얼에서는 다양한 기능을 사용하는 방법에 대해 자세히 설명합니다.

### 음성 합성

음성 합성은 FantasyTalking의 핵심 기능으로, 텍스트를 자연스럽고 원활한 음성으로 변환합니다. 기본 사용 방법은 다음과 같습니다:

```javascript
// 간단한 텍스트 음성 변환
const audio = await client.speak('Hello, welcome to FantasyTalking');

// 음성 스타일 지정
const audio = await client.speak('The weather is great today', {
  voice: {
    style: 'happy',
    language: 'en-US'
  }
});

// 속도 및 음높이 지정
const audio = await client.speak('Please wait a moment', {
  voice: {
    speed: 1.2,
    pitch: 0.9
  }
});
```

### SSML 마크업 언어

FantasyTalking은 음성 출력에 대한 더 세밀한 제어를 위해 SSML 마크업 언어를 지원합니다:

```xml
<speak>
  <p>
    <s>첫 번째 문장.</s>
    <s>두 번째 문장.</s>
  </p>
  <break time="500ms"/>
  <p>
    <emphasis level="moderate">이것은 중요한 내용입니다</emphasis>
  </p>
</speak>
```

해당 JavaScript 코드:

```javascript
const ssml = `
  <speak>
    <p>
      <s>첫 번째 문장.</s>
      <s>두 번째 문장.</s>
    </p>
    <break time="500ms"/>
    <p>
      <emphasis level="moderate">이것은 중요한 내용입니다</emphasis>
    </p>
  </speak>
`;

const audio = await client.speak(ssml, { ssml: true });
```

### 음성 이벤트

음성 재생 중에 상호작용을 위해 다양한 이벤트를 수신할 수 있습니다:

```javascript
// 음성 시작 수신
client.on('speech-start', (event) => {
  console.log('음성 재생 시작');
});

// 단어 경계 수신 (자막 동기화용)
client.on('word-boundary', (event) => {
  console.log(`현재 단어: ${event.word}`);
  console.log(`시작 시간: ${event.start}ms`);
  console.log(`종료 시간: ${event.end}ms`);
});

// 음성 종료 수신
client.on('speech-end', (event) => {
  console.log('음성 재생 완료');
});

// 오류 수신
client.on('error', (event) => {
  console.error('오류:', event.error);
});
```

### 배치 합성

대량의 텍스트를 합성해야 할 때 더 나은 효율을 위해 배치 합성을 사용할 수 있습니다:

```javascript
const texts = [
  '첫 번째 음성',
  '두 번째 음성',
  '세 번째 음성'
];

// 병렬 합성
const results = await Promise.all(
  texts.map(text => client.speak(text))
);

// 순차 합성 (순서 유지)
const results = [];
for (const text of texts) {
  const audio = await client.speak(text);
  results.push(audio);
}
```

## 대화 기능

### 기본 대화

FantasyTalking은 다중 턴 대화 및 컨텍스트 메모리를 지원하는 강력한 대화 관리 기능을 제공합니다:

```javascript
// 대화 세션 생성
const session = client.createSession({
  // 선택사항: 시스템 프롬프트 설정
  systemPrompt: '당신은 질문에 답변하기에 뛰어난 친절한 어시스턴트입니다',
  // 선택사항: 언어 설정
  language: 'en-US'
});

// 첫 번째 턴
const response1 = await session.chat('오늘 날씨가 어때요?');
console.log(response1.text);  // AI 응답 텍스트
console.log(response1.audio);  // AI 응답 음성

// 두 번째 턴 (자동으로 컨텍스트 포함)
const response2 = await session.chat('내일은 어때요?');
console.log(response2.text);
```

### 대화 구성

다양한 대화 시나리오에 대해 구성할 수 있습니다:

```javascript
const session = client.createSession({
  // 대화 구성
  dialogue: {
    // Temperature, 응답 창의성 제어
    temperature: 0.7,
    // 최대 응답 토큰 수
    maxTokens: 500,
    // 주제 태그, 시나리오 구분용
    tags: ['customer-service', 'pre-sales']
  },
  
  // 음성 구성
  voice: {
    // AI 응답 음성 스타일
    style: 'professional',
    // 음성 속도
    speed: 1.0
  }
});
```

### 대화 기록 관리

```javascript
// 대화 기록 가져오기
const history = session.getHistory();
console.log(history);

// 대화 기록 저장
const savedHistory = session.save();
fs.writeFileSync('conversation.json', JSON.stringify(savedHistory));

// 대화 기록 로드
const loadedSession = client.loadSession(savedHistory);
```

## 고급 기능

### 스트리밍 음성 합성

실시간 피드백 시나리오의 경우 스트리밍 음성 합성을 사용할 수 있습니다:

```javascript
// 스트리밍 출력 활성화
const stream = await client.speakStream('이것은 긴 텍스트입니다');

// 데이터 청크 수신
stream.on('data', (chunk) => {
  // 오디오 청크 재생
  audioPlayer.play(chunk);
});

stream.on('end', () => {
  console.log('음성 합성 완료');
});

stream.on('error', (error) => {
  console.error('오류:', error);
});
```

### 사용자 정의 음성 모델

브랜드 전용 음성이 필요한 경우 사용자 정의 모델을 훈련할 수 있습니다:

```javascript
// 훈련 데이터 준비
const trainingData = {
  name: 'Brand Voice',
  description: '기업 브랜드용 음성',
  samples: [
    { text: '샘플 텍스트 1', audio: 'base64로 인코딩된 오디오 1' },
    { text: '샘플 텍스트 2', audio: 'base64로 인코딩된 오디오 2' },
    // 더 많은 샘플...
  ]
};

// 훈련 작업 제출
const job = await client.trainVoice(trainingData);

// 훈련 상태 폴링
while (job.status !== 'completed') {
  await sleep(5000);
  const status = await client.getJobStatus(job.id);
  console.log(`훈련 진행률: ${status.progress}%`);
}

// 훈련 후 사용자 정의 음성 사용
const audio = await client.speak('사용자 정의 음성 사용', {
  voice: {
    modelId: job.modelId
  }
});
```

### 오디오 후처리

FantasyTalking은 다양한 오디오 후처리 기능을 제공합니다:

```javascript
const audio = await client.speak('원본 음성');

// 볼륨 조정
const normalized = client.process(audio, {
  normalize: true,
  targetLevel: -20  // dB
});

// 노이즈 감소
const denoised = client.process(audio, {
  denoise: true,
  noiseThreshold: 0.5
});

// 속도 조정
const spedUp = client.process(audio, {
  speed: 1.5  // 1.5배 속도
});
```

## 모범 사례

### 성능 최적화

실제 애플리케이션에서 성능 최적화를 위해 다음 사항에 주의하세요:

1. **연결 재사용**: 클라이언트 인스턴스를 자주 생성하지 마세요; 연결을 재사용하면 지연 시간을 크게 줄일 수 있습니다
2. **배치 처리**: 대량 텍스트를 처리할 때 배치 합성을 사용하세요
3. **캐싱**: 동일한 텍스트의 경우 중복 합성을 피하기 위해 캐싱을 사용하세요
4. **스트리밍 출력**: 긴 텍스트의 경우 스트리밍 출력을 사용하여 더 일찍 재생을 시작하고 사용자 경험을 개선하세요

### 오류 처리

포괄적인 오류 처리를 구현하는 것이 권장됩니다:

```javascript
try {
  const audio = await client.speak('테스트 텍스트');
} catch (error) {
  if (error.code === 'RATE_LIMIT') {
    // 속도 제한 처리
    console.log('요청이 너무 많습니다, 나중에 다시 시도해 주세요');
  } else if (error.code === 'INVALID_API_KEY') {
    // 인증 오류 처리
    console.log('잘못된 API 키');
  } else {
    // 기타 오류
    console.error('알 수 없는 오류:', error);
  }
}
```
