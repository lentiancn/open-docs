# HeyGen Guía de Uso

Esta guía cubre cómo usar HeyGen para generación de video AI.

## Inicio

### Usar interfaz web

1. Inicie sesión en https://www.heygen.com
2. Seleccione un avatar
3. Genere video

### Usar API

```python
from heygen import HeyGen

heygen = HeyGen(api_key="your_api_key")

video = heygen.generate_video(
    avatar_id="avatar_id",
    script="¡Hola, este es mi primer video AI!",
    voice_id="voice_id"
)

print(f"URL del video: {video.url}")
```

## Funciones principales

### 1. Selección de avatar

```python
avatars = heygen.list_avatars()
for avatar in avatars:
    print(f"{avatar.id}: {avatar.name}")
```

### 2. Selección de voz

```python
voices = heygen.list_voices()
for voice in voices:
    print(f"{voice.id}: {voice.name}")
```

### 3. Generación de video

```python
# Desde texto
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="Tu texto",
    voice_id="voice_id"
)

# Desde audio
video = heygen.generate_video_from_audio(
    avatar_id="avatar_id",
    audio_url="https://example.com/audio.wav"
)
```

### 4. Generación por lotes

```python
scripts = ["Primer video", "Segundo video"]
jobs = heygen.batch_generate(
    avatar_id="avatar_id",
    scripts=scripts,
    voice_id="voice_id"
)
```

## Parámetros

| Parámetro | Tipo | Descripción |
|-----------|------|-------------|
| avatar_id | string | ID del avatar |
| script | string | Texto a hablar |
| voice_id | string | ID de voz |
| background | string | Escena de fondo |
| speed | float | Velocidad (0.5-2.0) |
| emotion | string | Estilo emocional |

## Emociones disponibles

| Emoción | Descripción |
|---------|-------------|
| Neutral | Expresión por defecto |

| Triste | Triste |

| Sorprendido | Sorprendido |
