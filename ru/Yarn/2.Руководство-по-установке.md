# Руководство по установке

## Системные требования

- Node.js: 16.0+

---

## Установка

### Corepack (Рекомендуется)

```bash
corepack enable
corepack prepare yarn@stable --activate
```

### npm

```bash
npm install -g yarn
```

---

## Проверка

```bash
yarn --version
```
