# Руководство по установке Nginx

## Требования к окружению

Nginx поддерживает множество операционных систем:

- Linux (Debian, Ubuntu, CentOS, RHEL, Alpine и др.)
- FreeBSD
- Solaris
- AIX
- HP-UX
- macOS
- Windows

## Способы установки

### Способ 1: Использование пакетного менеджера (Рекомендуется)

#### Ubuntu/Debian

```bash
sudo apt update
sudo apt install nginx
sudo systemctl start nginx
sudo systemctl enable nginx
```

#### CentOS/RHEL

```bash
sudo yum install epel-release
sudo yum install nginx
sudo systemctl start nginx
sudo systemctl enable nginx
```

#### Alpine

```bash
apk add nginx
```

#### macOS

```bash
brew install nginx
brew services start nginx
```

### Способ 2: Компиляция из исходного кода

```bash
wget https://nginx.org/download/nginx-1.26.0.tar.gz
tar -zxf nginx-1.26.0.tar.gz
cd nginx-1.26.0
./configure
make
sudo make install
```

### Способ 3: Использование Docker

```bash
docker pull nginx
docker run -d -p 80:80 nginx
```

### Способ 4: Windows

Скачать с https://nginx.org/en/download.html и запустить nginx.exe

## Проверка

```bash
nginx -v
curl http://localhost
```

## Пути по умолчанию

- Конфигурация: `/etc/nginx/nginx.conf`
- Логи: `/var/log/nginx/`
- Документы: `/var/www/html/`
