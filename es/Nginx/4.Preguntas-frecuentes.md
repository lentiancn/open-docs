# Preguntas frecuentes sobre Nginx

## Instalación

### Cómo verificar la versión
```bash
nginx -v
```

### Dónde está el archivo de configuración
- Debian/Ubuntu: `/etc/nginx/nginx.conf`
- CentOS: `/etc/nginx/nginx.conf`

## Configuración

### Cómo probar la configuración
```bash
nginx -t
```

### Cómo recargar la configuración
```bash
nginx -s reload
```

## Rendimiento

### Optimización
```nginx
worker_processes auto;
worker_connections 4096;
gzip on;
```

## Seguridad

### Habilitar HTTPS
```nginx
server {
    listen 443 ssl;
    ssl_certificate cert.crt;
    ssl_certificate_key cert.key;
}
```

### Limitación de tasa
```nginx
limit_req_zone $binary_remote_addr zone=one:10m rate=10r/s;
```

## Errores comunes

- **400**: Verificar tamaño de solicitud
- **403**: Verificar permisos
- **404**: Verificar ruta
- **502**: Servicio backend no ejecutándose
- **504**: Timeout, aumentar proxy_read_timeout

## Proxy

### Obtener IP real
```nginx
proxy_set_header X-Real-IP $remote_addr;
```
