# Nacos ユーザーマニュアル

## サービス登録

```java
NamingService namingService = NacosFactory.createNamingService("127.0.0.1:8848");
namingService.registerInstance("my-service", "192.168.1.10", 8080);
```

## 構成管理

```bash
curl -X POST 'http://127.0.0.1:8848/nacos/v1/cs/configs' \
  -d 'dataId=app-config' \
  -d 'content=key=value'
```
