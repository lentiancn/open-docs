# Руководство по установке Go

> Полная настройка окружения

---

## Требования

| Окружение | Требование |
|-----------|------------|
| ОС | Linux, macOS, Windows |
| Архитектура | amd64, arm64, 386 |

---

## Установка

### Windows

```powershell
choco install golang
```

### macOS

```bash
brew install go
```

### Linux

```bash
sudo apt install golang-go
```

---

## Настройка

### Прокси (Китай)

```bash
go env -w GOPROXY=https://goproxy.cn,direct
```

### Проверка

```bash
go version
```
