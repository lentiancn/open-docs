# Spring Boot FAQ

## Questions Générales

### Qu'est-ce que Spring Boot?
Spring Boot est un framework Java open source qui simplifie le développement d'applications d'entreprise avec auto-configuration, serveurs embarqués et fonctionnalités prêtes pour la production.

### Pourquoi utiliser Spring Boot?
Spring Boot réduit le code boilerplate, simplifie la configuration et accélère le développement.

### Quelle est la différence entre Spring et Spring Boot?
| Aspect | Spring | Spring Boot |
|--------|--------|-------------|
| Configuration | XML/Annotation | Auto-configuration |
| Serveur | Externe | Embarqué |
| Dépendances | Gestion manuelle | Starter POMs |

## Questions d'Installation

### Comment créer un projet Spring Boot?
1. Utilisez Spring Initializr (https://start.spring.io)
2. Utilisez un IDE avec plugin Spring Boot

### Quelle version de Java est requise?
- Spring Boot 2.x: Java 8-17
- Spring Boot 3.x: Java 17+

## Questions d'Utilisation

### Comment créer un REST controller?
```java
@RestController
@RequestMapping("/api")
public class HelloController {
    
    @GetMapping("/hello")
    public String hello() {
        return "Hello, World!";
    }
}
```

### Comment se connecter à une base de données?
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/mydb
    username: root
    password: secret
```
