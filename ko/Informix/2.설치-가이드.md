# IBM Informix 설치 가이드

## 시스템 요구 사항

**지원 플랫폼:**
- Linux (Red Hat, Ubuntu, SUSE 등)
- Windows Server
- AIX
- HP-UX
- Solaris
- macOS
- ARM (엣지 장치용)

**최소 요구 사항:**
- RAM: 최소 1GB (권장 8GB)
- 디스크: 설치용 최소 2GB
- CPU: x86, PowerPC, ARM 아키텍처 호환

## Linux 설치 단계

### 1단계: 설치 프로그램 다운로드

IBM Passport Advantage 또는 IBM Fix Central를 방문하여 적절한 Informix 설치 프로그램을 다운로드합니다.

### 2단계: 시스템 준비

```bash
# informix 사용자 및 그룹 생성
groupadd informix
useradd -g informix -d /home/informix informix

# 설치 디렉토리 생성
mkdir -p /opt/IBM/informix
chown informix:informix /opt/IBM/informix
```

### 3단계: 설치 프로그램 추출

```bash
tar -xf informix.tar
cd informix
```

### 4단계: 설치 실행

```bash
# root 또는 sudo로 실행
./install
```

최신 버전(14.10+)의 경우 IBM Installation Manager 사용:

```bash
chmod +x ids_install
./ids_install
```

프롬프트 따르기:
- 라이선스 계약 수락
- 설치 경로 선택 (/opt/IBM/informix)
- 설치할 구성요소 선택

### 5단계: 환경 변수 구성

`~/.bashrc` 또는 `/etc/profile`에 추가:

```bash
export INFORMIXDIR=/opt/IBM/informix
export PATH=$INFORMIXDIR/bin:$PATH
export ONCONFIG=onconfig
export INFORMIXSERVER=서버이름
```

변경 사항 적용:

```bash
source ~/.bashrc
```

### 6단계: sqlhosts 파일 구성

```bash
cd $INFORMIXDIR/etc
cp sqlhosts.std sqlhosts
vi sqlhosts
```

서버 항목 추가:

```
ol_informix onsoctcp localhost ol_informix
```

### 7단계: 데이터베이스 서버 초기화

```bash
# 기본 설정으로 Informix 초기화
oninit -ivy
```

### 8단계: 설치 확인

```bash
# 서버 상태 확인
onstat -

# 초기화 확인
ls -la $INFORMIXDIR/storage/
```

## Windows 설치 단계

### 1단계: Windows 설치 프로그램 다운로드

IBM Passport Advantage에서 다운로드합니다.

### 2단계: 설치 프로그램 실행

- `.exe` 파일 더블 클릭
- 설치 마법사 따르기

### 3단계: 구성

- Informix 구성 마법사 사용
- Windows 서비스로 설정

### 4단계: 환경 변수 설정

시스템 속성에서 `INFORMIXDIR` 구성:
- 내 컴퓨터 우클릭 → 속성
- 고급 시스템 설정 → 환경 변수
- `INFORMIXDIR` 및 `INFORMIXSERVER` 추가

## 설치 후 확인

```bash
# 서버 상태 확인
onstat -

# 서버 시작 (실행 중이 아닌 경우)
oninit

# 서버 중지
onmode -ky

# 데이터베이스 연결
dbaccess
```

## 설치 후 일반 작업

### 추가 저장 공간 생성

```bash
onspaces -c -d datadbs -p /경로/데이터청크 -s 500000
```

### 논리적 로그 구성

```bash
onparams -a -d datadbs -s 10000
```

## 문제 해결

### 서버가 시작되지 않음

- 환경 변수가 올바르게 설정되었는지 확인
- `onconfig` 및 `sqlhosts` 파일 검토
- `$INFORMIXDIR/tmp/`의 오류 로그 확인
- 충분한 디스크 공간 및 권한 확인

### 연결 문제

- 방화벽 설정 확인 (기본 포트 9088)
- `INFORMIXSERVER`가 sqlhosts 구성과 일치하는지 확인
- sqlhosts에 올바른 네트워크 설정이 있는지 확인
