# Oracle Database

Enterprise-Grade Relational Database Management System

## Overview

Oracle Database is a comprehensive, scalable, and secure relational database management system used by enterprises worldwide.

## Quick Start

### Connect to Database

```bash
sqlplus / as sysdba
```

### Basic Query

```sql
SELECT * FROM employees;
```

## Features

- Enterprise-grade security
- High availability (RAC, Data Guard)
- Advanced PL/SQL support
- Partitioning and data warehousing
- JSON support
