# PlantUML Guide d'Installation

## Prérequis

- Java 8 ou supérieur requis
- 512MB RAM minimum

---

## Installation

### Windows
Télécharger plantuml.jar

### macOS
```bash
brew install plantuml
```

### Linux
```bash
sudo apt-get install plantuml
```

---

## Vérification
```bash
java -jar plantuml.jar -version
```
