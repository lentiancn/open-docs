# Guía de Instalación de Docker

Instrucciones para instalar Docker en todas las plataformas principales.

---

## Docker Desktop (Recomendado)

La forma más sencilla para macOS, Windows y Linux.

### Requisitos

**macOS:**
- Apple Silicon: macOS 11.0+, 4GB RAM
- Intel: macOS 10.15+, 4GB RAM

**Windows:**
- Windows 10/11 64-bit
- WSL2 (recomendado) o Hyper-V

**Linux:**
- Ubuntu, Debian, Fedora, Arch Linux

### Descargar

- [Docker Desktop for Mac](https://desktop.docker.com/mac/main/amd64/Docker.dmg)
- [Docker Desktop for Windows](https://desktop.docker.com/win/main/amd64/Docker%20Desktop%20Installer.exe)

## Linux

### Ubuntu

```bash
sudo apt update
sudo apt install -y ca-certificates curl gnupg lsb-release

sudo install -m 0755 -d /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
sudo chmod a+r /etc/apt/keyrings/docker.gpg

echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

sudo apt update
sudo apt install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin

sudo systemctl start docker
sudo systemctl enable docker
sudo usermod -aG docker $USER
```

### CentOS/RHEL

```bash
sudo yum install -y yum-utils
sudo yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
sudo yum install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
sudo systemctl start docker
sudo systemctl enable docker
```

## Windows

### WSL2 (Recomendado)

```powershell
wsl --install
# Reiniciar
sudo apt update
sudo apt install -y docker.io
```

### Docker Desktop

1. Descargar [Docker Desktop for Windows](https://desktop.docker.com/win/main/amd64/Docker%20Desktop%20Installer.exe)
2. Ejecutar el instalador
3. Seleccionar WSL2 (recomendado)
4. Iniciar Docker Desktop
