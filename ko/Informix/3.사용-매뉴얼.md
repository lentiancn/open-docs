# IBM Informix 사용자 매뉴얼

## 기본 명령어

### 서버 관리

| 명령어 | 설명 |
|--------|------|
| `oninit` | Informix 서버 초기화 및 시작 |
| `onmode` | 서버 모드 변경 (quiescent, online 등) |
| `onstat` | 서버 상태 및 통계 표시 |
| `ontape` | 백업 및 복원 유틸리티 |
| `onbar` | IBM Storage Manager를 사용한 백업 및 복원 |
| `onspaces` | 저장 공간 관리 |
| `onparams` | 서버 매개변수 수정 |

### 일반 서버 명령어

```bash
# 서버 초기화 및 시작
oninit

# 전체 프로덕션 모드로 시작
onmode -vy

# 서버 종료
onstat -

# 디스크 공간 보기
onstat -d

# 논리적 로그 보기
onstat -l

# 활성 사용자 보기
onstat -u

# 잠금 정보 보기
onstat -k

# 트랜잭션 정보 보기
onstat -x
```

## 데이터베이스 연결

### dbaccess 사용

```bash
# 대화형 모드
dbaccess

# SQL 스크립트 실행
dbaccess 데이터베이스이름 < sql스크립트.sql

# 특정 사용자로 연결
dbaccess -user 내사용자 내데이터베이스
```

### 연결 매개변수

```
호스트: localhost (또는 서버 IP)
포트: 9088 (기본값)
데이터베이스: 데이터베이스이름
서버 이름: Informix서버이름
```

## SQL 문

### 데이터베이스 생성

```sql
-- 간단한 데이터베이스 생성
CREATE DATABASE 내데이터베이스;

-- 버퍼링된 로그와 함께 생성
CREATE DATABASE 내데이터베이스 WITH BUFFERED LOG;

-- 로그와 함께 생성
CREATE DATABASE 내데이터베이스 WITH LOG;

-- ANSI 호환 데이터베이스 생성
CREATE DATABASE 내데이터베이스 WITH LOG MODE ANSI;
```

### 테이블 생성

```sql
-- 기본 테이블 생성
CREATE TABLE 고객 (
    고객_ID SERIAL PRIMARY KEY,
    이름 CHAR(50),
    이메일 CHAR(100),
    생성일자 DATE
);

-- 제약 조건이 있는 테이블
CREATE TABLE 주문 (
    주문_ID SERIAL PRIMARY KEY,
    고객_ID INTEGER REFERENCES 고객(고객_ID),
    주문일자 DATE,
    총금액 DECIMAL(10,2)
);

-- 임시 테이블
CREATE TEMP TABLE 세션데이터 (
    id INTEGER,
    데이터 VARCHAR(255)
);
```

### 데이터 삽입

```sql
-- 단일 행 삽입
INSERT INTO 고객 (이름, 이메일, 생성일자)
VALUES ('홍길동', 'hong@example.com', TODAY);

-- 여러 행 삽입
INSERT INTO 고객 (이름, 이메일, 생성일자)
VALUES 
    ('김철수', 'kim@example.com', TODAY),
    ('이영희', 'lee@example.com', TODAY);
```

### 데이터 조회

```sql
-- 모든 열 선택
SELECT * FROM 고객;

-- 특정 열 선택
SELECT 이름, 이메일 FROM 고객;

-- WHERE 절
SELECT * FROM 고객 WHERE 고객_ID > 100;

-- 정렬
SELECT 이름, 이메일 FROM 고객 ORDER BY 이름;

-- 집계 함수로 그룹화
SELECT 고객_ID, COUNT(*) AS 주문수
FROM 주문 GROUP BY 고객_ID;

-- 테이블 조인
SELECT o.주문_ID, c.이름, o.주문일자
FROM 주문 o, 고객 c
WHERE o.고객_ID = c.고객_ID;
```

### 데이터 업데이트

```sql
-- 단일 행 업데이트
UPDATE 고객 
SET 이메일 = 'new@example.com' 
WHERE 고객_ID = 101;
```

### 데이터 삭제

```sql
-- 단일 행 삭제
DELETE FROM 고객 WHERE 고객_ID = 101;

-- 조건에 따라 삭제
DELETE FROM 주문 WHERE 주문일자 < '2024-01-01';
```

## 구성 파일

### 주요 구성 파일

| 파일 | 용도 |
|------|------|
| `$INFORMIXDIR/etc/onconfig` | 기본 서버 구성 |
| `$INFORMIXDIR/etc/sqlhosts` | 네트워크 구성 |
| `$INFORMIXDIR/etc/$ONCONFIG` | 루트 dbspace 구성 |

### onconfig 예제 매개변수

```
ROOTNAME        rootdbs
ROOTPATH        /opt/IBM/informix/online.root
ROOTOFFSET      0
ROOTSIZE        200000
DBSERVERNAME    ol_informix
```

## 백업 및 복원

### ontape 사용

```bash
# 전체 백업
ontape -s -L 0

# 복원
ontape -r
```

### onbar 사용

```bash
# 백업
onbar -b -L 0

# 복원
onbar -r
```

## 성능 튜닝

### 주요 메모리 매개변수

- `SHMVIRTSIZE` - 가상 공유 메모리 크기
- `SHMADD` - 추가 공유 메모리 세그먼트
- `BUFFERPOOL` - 데이터 캐시 구성

```

```
