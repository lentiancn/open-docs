# FantasyTalking - Installationsanleitung

Vollständige Installationsanleitung.

## Anforderungen

### Hardware

- GPU: NVIDIA 8GB+
- Speicher: 20GB+
- RAM: 16GB+

### Software

- OS: Linux, Windows 10+, macOS
- Python: 3.8 - 3.10
- CUDA: 11.8+

## Installation

### Methode 1: Aus Quelle

```bash
git clone https://github.com/FantasyTalking/FantasyTalking.git
cd FantasyTalking
pip install -r requirements.txt
python download_models.py
```

### Methode 2: Docker

```bash
docker pull fantasytalking/fantasytalking:latest
docker run -it --gpus all -v /your/data:/data fantasytalking/fantasytalking:latest
```

### Methode 3: pip

```bash
pip install fantasytalking
```

## Überprüfung

```bash
python -c "import fantasytalking; print(fantasytalking.__version__)"
```

## Probleme

### CUDA nicht verfügbar?

Kompatibles PyTorch installieren.

### Zu wenig Speicher?

Kleinere Bilder verwenden.

## Deinstallieren

```bash
pip uninstall fantasytalking
```
