# 常見問題

本文檔匯總了 Bootstrap 使用過程中常見的技術問題和解答。

## 基礎問題

### 什麼是 Bootstrap？

Bootstrap 是一個開源的前端開發框架，由 Twitter 公司開發維護。它提供了豐富的 CSS 樣式和 JavaScript 元件，幫助開發者快速建構響應式 Web 應用程式。

### Bootstrap 5 和 Bootstrap 4 有什麼區別？

主要區別：

- Bootstrap 5 移除了 jQuery 依賴，使用原生 JavaScript
- 增加了 XXL 斷點
- 改進了表單控制項
- 移除了 Font Awesome，改為 Bootstrap Icons
- 不再支援 Internet Explorer

### Bootstrap 需要什麼環境？

只需要現代瀏覽器即可。Bootstrap 5 需要支援 ES6 的瀏覽器。

## 安裝問題

### CDN 載入失敗怎麼辦？

1. 檢查網路連線
2. 嘗試其他 CDN（如 bootcdn、staticfile）
3. 下載本地使用

### npm 安裝失敗？

```bash
# 清除快取
npm cache clean --force

# 使用淘寶鏡像
npm config set registry https://registry.npmmirror.com

# 重新安裝
npm install bootstrap@5
```

### 樣式沒作用？

1. 確認 CSS 檔案正確引入
2. 檢查瀏覽器快取（強制重新整理 Ctrl+Shift+R）
3. 確認沒有 CSS 衝突

## 使用問題

### 網格系統不運作？

1. 確保 row 在 container 內
2. 確保 col 在 row 內
3. 檢查是否正確使用斷點類別

### 響應式沒作用？

確保已加入視口標籤：

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### 導航欄在行動端不摺疊？

1. 確保引入了 bootstrap.bundle.min.js（包含 Popper）
2. 檢查 data-bs-toggle 和 data-bs-target 屬性

### 彈出視窗出不來？

1. 確保引入了 Bootstrap JS
2. 檢查按鈕的 data-bs-toggle 和 data-bs-target 屬性
3. 確認彈出視窗 ID 正確

### 下拉選單不運作？

1. 引入 bootstrap.bundle.min.js
2. 檢查 data-bs-toggle="dropdown" 屬性

## 樣式問題

### 如何修改主題顏色？

使用 Sass：

```scss
$primary: #007bff;
@import "bootstrap/scss/bootstrap";
```

### 如何覆蓋預設樣式？

使用更高優先順序的選擇器：

```css
.navbar {
    background-color: #f8f9fa !important;
}
```

### 字型如何修改？

```scss
$font-family-sans-serif: "Helvetica Neue", Helvetica, Arial, sans-serif;
@import "bootstrap/scss/bootstrap";
```

### 圖示不顯示？

Bootstrap 5 使用 Bootstrap Icons：

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.10.0/font/bootstrap-icons.css">

<i class="bi bi-alarm"></i>
```

## 元件問題

### 卡片如何居中？

```html
<div class="card">
    <div class="card-body">
        內容
    </div>
</div>
```

使用 flex：

```html
<div class="card">
    <div class="card-body d-flex justify-content-center align-items-center">
        居中內容
    </div>
</div>
```

### 表單驗證不運作？

Bootstrap 5 使用新的表單驗證：

```html
<form class="row g-3 needs-validation" novalidate>
    <div class="col-md-4">
        <label for="validationCustom01" class="form-label">名字</label>
        <input type="text" class="form-control" id="validationCustom01" required>
        <div class="valid-feedback">正確！</div>
        <div class="invalid-feedback">請輸入名字</div>
    </div>
    <div class="col-12">
        <button class="btn btn-primary" type="submit">提交</button>
    </div>
</form>

<script>
    (function () {
        'use strict'
        var forms = document.querySelectorAll('.needs-validation')
        Array.prototype.slice.call(forms).forEach(function (form) {
            form.addEventListener('submit', function (event) {
                if (!form.checkValidity()) {
                    event.preventDefault()
                    event.stopPropagation()
                }
                form.classList.add('was-validated')
            }, false)
        })
    })()
</script>
```

### 彈出視窗背景可點擊如何關閉？

設定屬性：

```html
<div class="modal" data-bs-backdrop="static">
```

`static` - 點擊背景不關閉
`true` - 點擊關閉
`false` - 點擊不關閉

## 效能問題

### 如何減少檔案體積？

1. 按需引入元件（使用 Sass）
2. 使用生產版本
3. 啟用 Gzip 壓縮

### 如何按需引入樣式？

使用 Sass 引入需要的元件：

```scss
// 引入必要的函數和變數
@import "bootstrap/scss/functions";
@import "bootstrap/scss/variables";

// 只引入需要的元件
@import "bootstrap/scss/root";
@import "bootstrap/scss/reboot";
@import "bootstrap/scss/containers";
@import "bootstrap/scss/grid";
@import "bootstrap/scss/buttons";
@import "bootstrap/scss/forms";
```

## 相容性問題

### 不支援 IE 瀏覽器？

Bootstrap 5 不再支援 Internet Explorer。如需支援 IE11，使用 Bootstrap 4。

### 行動端點擊延遲？

Bootstrap 5 使用 touch-action: manipulation 消除了 300ms 點擊延遲。
