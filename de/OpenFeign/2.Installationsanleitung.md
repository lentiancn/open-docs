# OpenFeign Installationsanleitung

## Voraussetzungen

- Java Development Kit (JDK) 8 oder höher
- Maven oder Gradle Build-Tool
- IDE (IntelliJ IDEA, Eclipse oder VS Code)

---

## Mit Maven

Fügen Sie die OpenFeign-Abhängigkeit zu Ihrer `pom.xml` hinzu:

```xml
<dependencies>
    <!-- OpenFeign Core -->
    <dependency>
        <groupId>io.github.openfeign</groupId>
        <artifactId>feign-core</artifactId>
        <version>13.2.1</version>
    </dependency>
    
    <!-- OkHttp Client (optional aber empfohlen) -->
    <dependency>
        <groupId>io.github.openfeign</groupId>
        <artifactId>feign-okhttp</artifactId>
        <version>13.2.1</version>
    </dependency>
    
    <!-- Jackson für JSON (optional) -->
    <dependency>
        <groupId>io.github.openfeign</groupId>
        <artifactId>feign-jackson</artifactId>
        <version>13.2.1</version>
    </dependency>
</dependencies>
```

---

## Mit Gradle

Fügen Sie Abhängigkeiten zu Ihrer `build.gradle` hinzu:

```groovy
dependencies {
    implementation 'io.github.openfeign:feign-core:13.2.1'
    implementation 'io.github.openfeign:feign-okhttp:13.2.1'
    implementation 'io.github.openfeign:feign-jackson:13.2.1'
}
```

---

## Spring Cloud Integration

Für Spring Boot-Anwendungen fügen Sie Spring Cloud OpenFeign hinzu:

```xml
<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-openfeign</artifactId>
    <version>4.1.0</version>
</dependency>
```

Aktivieren Sie Feign-Clients in Ihrer Spring Boot-Anwendung:

```java
@SpringBootApplication
@EnableFeignClients
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
```

---

## Installation überprüfen

Erstellen Sie einen einfachen Feign-Client, um die Installation zu überprüfen:

```java
@FeignClient(name = "example-service", url = "https://api.example.com")
public interface ExampleClient {
    
    @RequestMapping(method = RequestMethod.GET, value = "/users/{id}")
    User getUser(@PathVariable("id") Long id);
}
```

Kompilieren Sie Ihr Projekt:

```bash
mvn clean compile
```
