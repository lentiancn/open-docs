# Guide d'installation

nginx peut être installé différemment selon le système d'exploitation.

#### Installation sur Linux

Pour Linux, vous pouvez utiliser les packages nginx de nginx.org. La méthode recommandée est la suivante :

**Sur Ubuntu et Debian :**

```bash
sudo apt update
sudo apt install nginx
```

**Sur CentOS et RHEL :**

```bash
sudo yum install nginx
```

**Sur Fedora :**

```bash
sudo dnf install nginx
```

#### Installation depuis les sources

Si des fonctionnalités spéciales sont requises, non disponibles dans les packages, nginx peut également être compilé depuis les sources.

Prérequis pour la compilation :

```bash
# Ubuntu/Debian
sudo apt install build-essential libpcre3 libpcre3-dev zlib1g zlib1g-dev libssl-dev

# CentOS/RHEL
sudo yum install gcc pcre-devel zlib-devel openssl-devel
```

Téléchargement des sources :

```bash
wget https://nginx.org/download/nginx-1.26.0.tar.gz
tar -xzvf nginx-1.26.0.tar.gz
cd nginx-1.26.0
```

Configuration et compilation :

```bash
./configure --prefix=/usr/local/nginx --with-http_ssl_module
make
sudo make install
```

#### Installation avec Docker

La méthode la plus rapide pour démarrer avec nginx est d'utiliser Docker :

```bash
# Serveur web de base
docker run -d -p 80:80 --name nginx nginx:latest

# Avec un volume local
docker run -d -p 80:80 -v /chemin/vers/html:/usr/share/nginx/html --name nginx nginx:latest

# Nginx avec configuration personnalisée
docker run -d -p 80:80 -v /chemin/vers/nginx.conf:/etc/nginx/nginx.conf:ro --name nginx nginx:latest
```

#### Vérification de l'installation

Après l'installation, vérifiez que nginx est correctement installé :

```bash
nginx -v
```

Cela affichera la version de nginx installée.

Pour démarrer le serveur :

```bash
# Linux (systemd)
sudo systemctl start nginx

# ou directement
nginx
```

Pour vérifier que le serveur fonctionne, accédez à http://localhost dans votre navigateur.
