# Introducción a Java

> Java es un lenguaje orientado a objetos

---

## Descripción

Java fue lanzado en 1995 por Sun Microsystems. Conocido por su portabilidad.

---

## Características

- Orientado a objetos
- Multiplataforma (JVM)
- Gestión automática de memoria
- Multithreading
- Rico ecosistema
