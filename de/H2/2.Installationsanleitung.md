# Installationsanleitung für H2

## Anforderungen

- Java 11 oder höher
- Beliebiges Betriebssystem (Windows, Linux, macOS)

## Installation

### Maven-Abhängigkeit
```xml
<dependency>
    <groupId>com.h2database</groupId>
    <artifactId>h2</artifactId>
    <version>2.2.224</version>
</dependency>
```

### ZIP herunterladen
Von https://www.h2database.com/html/download.html

## Überprüfung

```bash
java -jar h2.jar
```

Startet die Web-Konsole unter http://localhost:8082

## Verbindungs-URLs

| Modus | URL |
|-------|-----|
| In-Memory | jdbc:h2:mem:testdb |
| Datei | jdbc:h2:file:./data/testdb |
| TCP | jdbc:h2:tcp://localhost:9092/testdb |

## Standard-Benutzer

- Benutzer: sa
