# Element Plus ユーザーマニュアル

## クイックスタート

### 基本的な使用

```vue
<el-button>デフォルト</el-button>
<el-button type="primary">プライマリ</el-button>
```

## 一般的なコンポーネント

### ボタン

```vue
<el-button>ボタン</el-button>
<el-button type="primary">プライマリ</el-button>
```

### 入力

```vue
<el-input v-model="input" placeholder="入力してください" />
```

### テーブル

```vue
<el-table :data="tableData">
  <el-table-column prop="name" label="名前" />
</el-table>
```

### フォーム

```vue
<el-form :model="form">
  <el-form-item label="名前">
    <el-input v-model="form.name" />
  </el-form-item>
</el-form>
```

## テーマ

```css
:root {
  --el-color-primary: #409eff;
}
```
