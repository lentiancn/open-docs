# V-Express 사용 가이드

## 기본 사용

```bash
python inference.py \
  --reference_image_path "./ref.jpg" \
  --audio_path "./audio.mp3" \
  --output_path "./output.mp4" \
  --retarget_strategy "no_retarget"
```

## 매개변수

| 매개변수 | 설명 |
|---------|------|
| `--reference_image_path` | 참조 이미지 경로 |
| `--audio_path` | 오디오 파일 경로 |
| `--output_path` | 출력 비디오 경로 |
