# Element UI FAQ

## Installation

### Q: Installation échouée?

**Solutions:**
1. Vérifier version Node.js
2. Vider cache npm
```bash
npm cache clean --force
```

### Q: Version Vue incompatible?

**Solutions:**
Element UI 2.x requiert Vue 2.6.0+

## Utilisation

### Q: Composants pas affichés?

**Solutions:**
Vérifier l'import:
```javascript
import { Button } from 'element-ui'
Vue.use(Button)
```

### Q: Styles pas appliqués?

**Solutions:**
Importer fichier styles:
```javascript
import 'element-ui/lib/theme-chalk/index.css'
```
