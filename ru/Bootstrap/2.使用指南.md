# Bootstrap 5 Руководство по использованию

Это руководство описывает, как создавать адаптивные веб-сайты с Bootstrap 5.

## Быстрый старт

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 5 Страница</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <div class="container mt-5">
        <h1>Привет, Bootstrap 5!</h1>
        <p class="lead">Это адаптивная страница.</p>
    </div>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## Система макета

### Контейнеры

```html
<div class="container">Фиксированная ширина</div>
<div class="container-fluid">Полная ширина</div>
```

### Сетка

Bootstrap использует 12-колоночную систему:

```html
<div class="container">
    <div class="row">
        <div class="col-12 col-md-6 col-lg-4">Колонка</div>
        <div class="col-12 col-md-6 col-lg-4">Колонка</div>
        <div class="col-12 col-md-6 col-lg-4">Колонка</div>
    </div>
</div>
```

## Основные компоненты

### Кнопки

```html
<button class="btn btn-primary">Основная</button>
<button class="btn btn-secondary">Вторичная</button>
<button class="btn btn-success">Успех</button>
<button class="btn btn-danger">Опасность</button>
<button class="btn btn-outline-primary">Контурная</button>
```

### Навигация

```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">Бренд</a>
        <button class="navbar-toggler" data-bs-toggle="collapse" data-bs-target="#nav">
            <span class="navbar-toggler-icon"></span>
        </button>
        <div class="collapse navbar-collapse" id="nav">
            <ul class="navbar-nav">
                <li class="nav-item"><a class="nav-link" href="#">Главная</a></li>
            </ul>
        </div>
    </div>
</nav>
```

### Карточки

```html
<div class="card" style="width: 18rem;">
    <img src="..." class="card-img-top">
    <div class="card-body">
        <h5 class="card-title">Заголовок</h5>
        <p class="card-text">Текст</p>
        <a href="#" class="btn btn-primary">Подробнее</a>
    </div>
</div>
```

### Формы

```html
<form>
    <div class="mb-3">
        <label class="form-label">Email</label>
        <input type="email" class="form-control">
    </div>
    <div class="mb-3">
        <label class="form-label">Пароль</label>
        <input type="password" class="form-control">
    </div>
    <button type="submit" class="btn btn-primary">Отправить</button>
</form>
```

### Модальные окна

```html
<button class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#myModal">
    Открыть
</button>

<div class="modal fade" id="myModal">
    <div class="modal-dialog">
        <div class="modal-content">
            <div class="modal-header">
                <h5 class="modal-title">Заголовок</h5>
                <button class="btn-close" data-bs-dismiss="modal"></button>
            </div>
            <div class="modal-body">Контент</div>
        </div>
    </div>
</div>
```

## Утилиты

### Отступы

```html
<div class="m-3">Margin</div>
<div class="mt-3">Margin-top</div>
<div class="p-3">Padding</div>
```

### Текст

```html
<p class="text-center">По центру</p>
<p class="text-start">Слева</p>
<p class="text-end">Справа</p>
<p class="fw-bold">Жирный</p>
```

### Цвета

```html
<div class="bg-primary text-white">Основной</div>
<div class="bg-success text-white">Успех</div>
<div class="bg-danger text-white">Опасность</div>
```

## Настройка темы

```css
:root {
    --bs-primary: #ff6b35;
    --bs-secondary: #6c757d;
}
```
