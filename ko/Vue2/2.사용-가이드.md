# Vue 2 사용 가이드

Vue 2를 효과적으로 사용하는 가이드입니다.

---

## 기본 개념

Vue 2는 사용자 인터페이스를 구축하기 위한 JavaScript 프레임워크입니다.

---

## 인스턴스

```javascript
new Vue({
  el: '#app',
  data: {
    message: 'Hello Vue!'
  }
})
```

---

## 디렉티브

| 디렉티브 | 설명 |
|----------|------|
| `v-if` | 조건부 렌더링 |
| `v-for` | 목록 렌더링 |
| `v-bind` | 속성 바인딩 |
| `v-model` | 양방향 바인딩 |

---

## 컴포넌트

```vue
<template>
  <div>{{ title }}</div>
</template>

<script>
export default {
  props: {
    title: String
  }
}
</script>
```
