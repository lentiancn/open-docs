# Руководство по установке Kotlin

Kotlin — это современный язык программирования, разработанный компанией JetBrains. В этом руководстве описаны методы установки в различных средах.

## Информация о версиях

| Версия | Статус | Описание |
|--------|--------|----------|
| Kotlin 2.x | ✅ Текущая | Последняя версия |
| Kotlin 1.9.x | ✅ Поддерживается | Стабильная версия |

## Системные требования

### Аппаратные требования

| Требование | Конфигурация |
|------------|--------------|
| CPU | 1+ ядро |
| Оперативная память | 2+ ГБ |
| Диск | 500+ МБ |

### Программные требования

| Программа | Версия |
|-----------|--------|
| Java | JDK 8 или выше |

## Способы установки

### Способ 1: IntelliJ IDEA (Рекомендуется)

1. Скачать IntelliJ IDEA: https://www.jetbrains.com/idea/download/
2. Установить версию Ultimate или Community
3. Kotlin уже встроен

### Способ 2: Командная строка

#### Windows

```powershell
# С помощью Chocolatey
choco install kotlin

# Или скачать вручную
# Скачать с https://github.com/JetBrains/kotlin/releases
```

#### macOS

```bash
# С помощью Homebrew
brew install kotlin

# Или установить вручную
curl -s https://get.sdkman.io | bash
sdk install kotlin
```

#### Linux

```bash
# Ubuntu/Debian
sudo apt install kotlin

# Или через SDKMAN
curl -s https://get.sdkman.io | bash
sdk install kotlin
```

### Способ 3: SDKMAN

```bash
# Установить SDKMAN
curl -s "https://get.sdkman.io" | bash

# Установить Kotlin
sdk install kotlin

# Проверить
kotlin -version
```

### Способ 4: Maven

```xml
<dependencies>
    <dependency>
        <groupId>org.jetbrains.kotlin</groupId>
        <artifactId>kotlin-stdlib</artifactId>
        <version>2.0.0</version>
    </dependency>
</dependencies>
```

### Способ 5: Gradle

```groovy
plugins {
    id 'org.jetbrains.kotlin.jvm' version '2.0.0'
}

repositories {
    mavenCentral()
}

dependencies {
    implementation "org.jetbrains.kotlin:kotlin-stdlib"
}
```

### Способ 6: Docker

```dockerfile
# Использовать образ компилятора Kotlin
FROM gradle:8.5-jdk-21 AS builder
WORKDIR /app
COPY . .
RUN gradle build

# Образ для запуска
FROM openjdk:21
COPY --from=builder /app/build/libs/*.jar /app.jar
ENTRYPOINT ["java", "-jar", "/app.jar"]
```

## Настройка окружения

### Настройка Java

```bash
# Проверить Java
java -version

# Установить JAVA_HOME
export JAVA_HOME=/path/to/java
```

### Компиляция и запуск

```bash
# Компиляция
kotlinc hello.kt -include-runtime -d hello.jar

# Запуск
java -jar hello.jar

# Использовать скрипт kotlin
kotlin hello.kts
```

## Настройка IDE

### IntelliJ IDEA

1. Установить плагин Kotlin (обычно предустановлен)
2. Создать проект Kotlin
3. Настроить Kotlin SDK

### VS Code

1. Установить расширение Kotlin Language Server
2. Установить компилятор Kotlin

### Eclipse

1. Установить плагин Kotlin (Kotlin IDE for Eclipse)
2. Создать проект Kotlin

## Частые проблемы

### Ошибки компиляции?

Проверьте совместимость версий Kotlin и Java.

### Недостаточно памяти?

Увеличьте размер кучи JVM:

```bash
kotlinc -J-Xmx2g
```
