# Руководство пользователя MultiTalk

В этом руководстве описано, как использовать MultiTalk для создания многоголосого аудиоконтента.

## Быстрый старт

### Базовая эксплуатация

```bash
python inference.py \
  --text "Привет, добро пожаловать в MultiTalk." \
  --speakers speaker1,speaker2 \
  --output result.wav
```

## Требования к входным данным

### Требования к тексту

- Язык: Поддержка нескольких языков
- Формат: Обычный текст
- Длина: Без строгих ограничений

### Настройка дикторов

Определите дикторов в `speakers.yaml`:

```yaml
speakers:
  speaker1:
    voice_id: "voice_001"
    language: "ru"
  speaker2:
    voice_id: "voice_002"
    language: "ru"
```

## Расширенное использование

### Пользовательские ID голосов

```bash
python inference.py \
  --text "Это демонстрация пользовательского голоса." \
  --voice_ids voice001,voice002 \
  --output output.wav
```

### Регулировка скорости

```bash
python inference.py \
  --text "Пример медленной речи." \
  --speakers speaker1 \
  --speed 0.8 \
  --output slow.wav
```

### Пакетная обработка

```bash
python batch_inference.py \
  --text_file texts.txt \
  --output_dir output/
```

## Настройка

### Конфигурационный файл

Отредактируйте `config.yaml`:

```yaml
inference:
  sample_rate: 22050
  speed: 1.0
  
model:
  checkpoint: "checkpoints/multitalk.pth"
```

## Рекомендации

### 1. Используйте подходящие ID голосов

Выбирайте ID голосов, соответствующие характеристикам диктора.

### 2. Соответствие языков

Убедитесь, что язык текста соответствует языку голоса.

### 3. Оптимизация скорости

Настраивайте скорость для получения естественно звучащего результата.

## Устранение неполадок

### Плохое качество audio?

Попробуйте другие ID голосов.

### Проблемы со скоростью?

Постепенно настраивайте параметр скорости.
