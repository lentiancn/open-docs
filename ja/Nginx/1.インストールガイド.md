# Nginx インストールガイド

すべての主要プラットフォームに Nginx をインストールするためのガイド。

---

## Linux

### Ubuntu / Debian

```bash
# パッケージ更新
sudo apt update

# Nginx インストール
sudo apt install nginx

# 起動
sudo systemctl start nginx

# 自動起動
sudo systemctl enable nginx
```

### CentOS / RHEL

```bash
# EPEL リポジトリインストール
sudo yum install epel-release

# Nginx インストール
sudo yum install nginx

# 起動
sudo systemctl start nginx
```

---

## macOS

### Homebrew

```bash
brew install nginx
brew services start nginx
```

---

## Windows

1. [nginx.org](http://nginx.org/en/download.html) からダウンロード
2. 解凍して `C:\nginx` に配置
3. コマンドプロンプトで実行:
```
cd C:\nginx
start nginx
```

---

## 検証

```bash
nginx -v
nginx -t
curl http://localhost
```
