## 1. Installation Issues

### Q1: What to do if `ng` command is not found?

**Problem**: After installing Angular CLI, the `ng` command cannot be found.

**Solutions**:

1. Confirm npm global bin directory is in PATH:
   ```bash
   # Windows
   npm config get prefix
   
   # Add the returned path to system PATH
   ```

2. Or use npx directly:
   ```bash
   npx @angular/cli version
   ```

3. Close and reopen your terminal

---

### Q2: Node.js Version Incompatible

**Problem**: Installation fails due to low Node.js version.

**Solutions**:

Use nvm (Node Version Manager) to switch versions:

```bash
# Install nvm (Linux/macOS)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Install Node.js 20
nvm install 20
nvm use 20
```

```bash
arch -arm64 node --version

nvm install 20 --arm64
```

## 3. Component Issues

**Problem**: Created component doesn't display on the page.

**Solutions**:

```typescript
  imports: [CommonModule],
```

### Q8: Component Communication Fails

**Problem**: Parent component cannot pass values to child component.

**Solutions**:

Check if child component correctly uses `@Input()`:

```typescript
// Child component

@Input() count: number = 0;
```

Parent template:

```html
<app-child [data]="'hello'" [count]="5"></app-child>
```

## 5. HTTP and API Issues

### Q13: HTTP Request Fails CORS

**Problem**: Cross-origin requests are blocked.

**Solutions**:

1. Add CORS headers on backend
2. Use proxy:
   ```json
   // proxy.conf.json
   {
     "/api": {
       "target": "http://localhost:3000",
       "secure": false
     }
   }
   ```
   
   Run:
   ```bash
   ng serve --proxy-config proxy.conf.json
   ```

---

### Q14: HTTP Request Returns 401

**Problem**: Requests requiring authentication return unauthorized.

**Solutions**:

Add authentication interceptor:

```typescript
import { HttpInterceptorFn } from '@angular/common/http';

export const authInterceptor: HttpInterceptorFn = (req, next) => {
  const token = localStorage.getItem('token');
  if (token) {
    const cloned = req.clone({
      setHeaders: { Authorization: `Bearer ${token}` }
    });
    return next(cloned);
  }
  return next(req);
};
```

Register in `app.config.ts`:

```typescript
providers: [
  provideHttpClient(withInterceptors([authInterceptor]))
]
```
