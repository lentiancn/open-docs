# Angular Resumen

Plataforma de desarrollo web moderna.

## ¿Qué es Angular?

Angular es una plataforma de desarrollo para crear aplicaciones web móviles y de escritorio, desarrollada por Google. Utiliza TypeScript/JavaScript y otros lenguajes.

### Características principales

- **Basado en componentes** - Arquitectura dirigida por componentes
- **TypeScript** - Soporte de tipado fuerte
- **Enlace bidireccional** - Sincronización de datos simplificada
- **Inyección de dependencias** - Potente sistema de gestión de dependencias
- **Enrutamiento** - Solución completa de enrutamiento
- **Formularios** - Formularios reactivos y basados en plantillas
- **Cliente HTTP** - Servicios HTTP completos
- **Renderizado del lado del servidor** - Soporte SSR
