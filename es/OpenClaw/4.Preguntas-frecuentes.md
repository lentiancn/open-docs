# Preguntas frecuentes de OpenClaw

Preguntas y respuestas comunes sobre OpenClaw.

## Básico

### ¿Qué es OpenClaw?

OpenClaw es una herramienta autoalojada que conecta aplicaciones de mensajería (WhatsApp, Telegram, Discord, etc.) con asistentes de IA. Te permite usar IA a través de las aplicaciones de chat que ya tienes instaladas, mientras mantienes el control total sobre tus datos.

### ¿Es OpenClaw gratis?

El software en sí es gratis porque es de código abierto. Sin embargo, necesitas cubrir:
- Una computadora o servidor para ejecutarlo
- Los costos de la clave API para servicios de IA

### ¿Necesito una suscripción a un servicio de IA?

No necesariamente. Puedes usar claves API de pago por uso o cuentas con suscripción. OpenClaw soporta Anthropic, OpenAI, Google Gemini y otros proveedores.

### ¿En qué plataformas funciona?

- **Escritorio**: macOS, Linux, Windows
- **Móvil**: iOS, Android (como nodos companions)
- **Apps de chat**: WhatsApp, Telegram, Discord, iMessage, Signal, Slack y más de 20 más

## Instalación

### La instalación falló. ¿Qué hago?

1. Verifica la versión de Node.js: `node --version` (necesita 22+)
2. Ejecuta diagnósticos: `openclaw doctor`
3. Revisa los registros: `openclaw logs`

### ¿Puedo ejecutarlo en una Raspberry Pi?

Sí. Raspberry Pi 4 o más reciente con Raspberry Pi OS o Ubuntu funciona perfectamente.

### ¿Necesito una Mac?

No. OpenClaw funciona en cualquier sistema con Node.js. Solo el canal de iMessage requiere una Mac debido a las restricciones de Apple.

## Configuración

### ¿Dónde está el archivo de configuración?

Ubicación predeterminada: `~/.openclaw/openclaw.json`

### ¿Cómo controlo quién puede escribirme?

Usa la configuración `dmPolicy`:

- `pairing`: Los desconocidos necesitan un código de emparejamiento
- `allowlist`: Solo usuarios en la lista blanca pueden escribir
- `open`: Cualquiera puede escribir
- `disabled`: Desactivar mensajes directos

### ¿Cómo configuro grupos de chat?

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

## Uso

### ¿Cómo añado un nuevo canal de chat?

```bash
openclaw channels login whatsapp
```

### ¿Puedo usar varios canales a la vez?

Sí. Un proceso Gateway se conecta a múltiples plataformas de chat simultáneamente.

### ¿Cómo hace la IA para recordar conversaciones anteriores?

OpenClaw guarda automáticamente el contexto de la conversación. Puedes ajustar las políticas de retención en la configuración.

### ¿Puedo ejecutar múltiples agentes de IA?

Sí. Configura múltiples agentes con diferentes reglas de enrutamiento:

```json5
{
  agents: {
    list: [
      { id: "home", default: true },
      { id: "work" }
    ]
  }
}
```

## Seguridad

### ¿Mis datos están seguros?

Sí. OpenClaw se ejecuta localmente — todos los datos permanecen en tu dispositivo y no se envían a terceros (excepto para llamadas a API de proveedores de IA).

### ¿Cómo protejo mi instalación?

1. Configura `allowFrom` para restringir fuentes de acceso
2. Usa contraseñas fuertes y claves API seguras
3. Mantén OpenClaw actualizado
4. Usa firewall para limitar los puertos abiertos

## Solución de problemas

### El Gateway no inicia

1. Verifica si el puerto está en uso: `openclaw gateway status`
2. Revisa los registros de error: `openclaw logs`
3. Ejecuta diagnósticos: `openclaw doctor`

### La IA no responde

1. Verifica que la clave API sea correcta
2. Comprueba que tu cuenta del proveedor de IA tenga crédito
3. Busca errores en los registros

### Los mensajes no se envían

1. Verifica que el canal haya iniciado sesión
2. Confirma que el destinatario existe
3. Revisa los registros detallados
