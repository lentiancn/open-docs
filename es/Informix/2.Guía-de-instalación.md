# Guía de Instalación de IBM Informix

## Requisitos del Sistema

**Plataformas Soportadas:**
- Linux (Red Hat, Ubuntu, SUSE, etc.)
- Windows Server
- AIX
- HP-UX
- Solaris
- macOS
- ARM (para dispositivos perimetrales)

**Requisitos Mínimos:**
- RAM: 1GB mínimo (8GB recomendado)
- Disco: 2GB mínimo para instalación
- CPU: Compatible con arquitecturas x86, PowerPC, ARM

## Pasos de Instalación en Linux

### Paso 1: Descargar el Instalador

Visite IBM Passport Advantage o IBM Fix Central y descargue el paquete de instalador apropiado para Informix.

### Paso 2: Preparar el Sistema

```bash
# Crear usuario y grupo informix
groupadd informix
useradd -g informix -d /home/informix informix

# Crear directorio de instalación
mkdir -p /opt/IBM/informix
chown informix:informix /opt/IBM/informix
```

### Paso 3: Extraer el Instalador

```bash
tar -xf informix.tar
cd informix
```

### Paso 4: Ejecutar la Instalación

```bash
# Como root o con sudo
./install
```

Para versiones más nuevas (14.10+), use IBM Installation Manager:

```bash
chmod +x ids_install
./ids_install
```

Siga las indicaciones:
- Acepte el acuerdo de licencia
- Elija la ruta de instalación (/opt/IBM/informix)
- Seleccione los componentes a instalar

### Paso 5: Configurar Variables de Entorno

Agregue a `~/.bashrc` o `/etc/profile`:

```bash
export INFORMIXDIR=/opt/IBM/informix
export PATH=$INFORMIXDIR/bin:$PATH
export ONCONFIG=onconfig
export INFORMIXSERVER=su_nombre_servidor
```

Aplique los cambios:

```bash
source ~/.bashrc
```

### Paso 6: Configurar Archivo sqlhosts

```bash
cd $INFORMIXDIR/etc
cp sqlhosts.std sqlhosts
vi sqlhosts
```

Agregue su entrada de servidor:

```
ol_informix onsoctcp localhost ol_informix
```

### Paso 7: Inicializar el Servidor de Base de Datos

```bash
# Inicializar Informix con configuración predeterminada
oninit -ivy
```

### Paso 8: Verificar Instalación

```bash
# Verificar estado del servidor
onstat -

# Verificar inicialización
ls -la $INFORMIXDIR/storage/
```

## Pasos de Instalación en Windows

### Paso 1: Descargar el Instalador para Windows

Descargue desde IBM Passport Advantage.

### Paso 2: Ejecutar el Instalador

- Haga doble clic en el archivo `.exe`
- Siga el asistente de instalación

### Paso 3: Configuración

- Use el Asistente de Configuración de Informix
- Configure el servicio de Windows

### Paso 4: Configuración de Variables de Entorno

Configure `INFORMIXDIR` en Propiedades del Sistema:
- Haga clic derecho en Mi PC → Propiedades
- Configuración Avanzada del Sistema → Variables de Entorno
- Agregue `INFORMIXDIR` e `INFORMIXSERVER`

## Verificación Post-Instalación

```bash
# Verificar estado del servidor
onstat -

# Iniciar el servidor (si no está ejecutándose)
oninit

# Detener el servidor
onmode -ky

# Conectar a la base de datos
dbaccess
```

## Tareas Comunes Post-Instalación

### Crear Espacios de Almacenamiento Adicionales

```bash
onspaces -c -d datadbs -p /ruta/a/data_chunk -s 500000
```

### Configurar Logs Lógicos

```bash
onparams -a -d datadbs -s 10000
```

## Solución de Problemas

### El Servidor No Inicia

- Verifique que las variables de entorno estén configuradas correctamente
- Revise los archivos `onconfig` y `sqlhosts`
- Revise los registros de error en `$INFORMIXDIR/tmp/`
- Asegúrese de tener suficiente espacio en disco y permisos

### Problemas de Conexión

- Verifique la configuración del firewall (puerto predeterminado 9088)
- Verifique que `INFORMIXSERVER` coincida con la configuración de sqlhosts
- Asegúrese de que `sqlhosts` tenga la configuración de red correcta
