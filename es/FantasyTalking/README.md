# FantasyTalking

FantasyTalking - Sistema de Síntesis de Voz y Diálogo de Código Abierto

## Resumen

FantasyTalking es un proyecto técnico de código abierto centrado en la síntesis de voz y sistemas de diálogo.

## Características

- **Síntesis de Voz de Alta Calidad**: Tecnología TTS basada en aprendizaje profundo
- **Soporte Multiidioma**: Soporta chino, inglés, japonés y otros idiomas
- **Sistema de Diálogo Flexible**: Soporta conversaciones multiturno
- **Soporte Multiplataforma**: Web, móvil y escritorio
- **Código Abierto y Gratis**: Licencia MIT

## Inicio Rápido

### Instalación

```bash
npm install fantasy-talking
# o
pip install fantasy-talking
```

### Uso

```javascript
const client = new FantasyTalking({ apiKey: 'your-api-key' });
const audio = await client.speak('Hola');
```
