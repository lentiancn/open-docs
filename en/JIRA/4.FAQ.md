# Frequently Asked Questions

This section compiles common questions and answers during Jira usage, helping you quickly resolve difficulties encountered.

## Account and Permissions

### How to Reset Password

If you forgot your password, click the "Forgot password" link on the login page. The system will send a reset link to your registered email. If you cannot receive the email, check your spam folder or contact your administrator.

### How to Add or Remove Users

Only administrators can add or remove users. After logging into Jira, go to admin settings and find "User Management". Here you can add new users, delete existing users, or manage user permissions. When adding users, you need to enter their email address, and the system will send an invitation email.

### Is There a User Limit for the Free Version

Yes, the Free plan supports a maximum of 10 users. If you need to add more users, you need to upgrade to Standard, Premium, or Enterprise plans. When the user limit is exceeded, the system will prompt you to upgrade.

## Projects and Workflows

### How to Create Custom Workflows

Administrators can create and modify project workflows. Go to project settings and find "Workflows". You can add new workflow schemes and define various statuses and transition rules. Each transition can set conditions, triggers, and post-functions.

### Can Issues Be Moved Between Projects

Yes. You can move issues from one project to another. When moving issues, you can choose whether to retain history, comments, and attachments. Some custom fields may not be movable between different projects.

### How to Bulk Update Issues

Jira supports bulk editing. After selecting multiple issues, you can bulk modify assignees, priorities, labels, and other fields. In list view, check the issues you need to edit and use bulk operations to complete updates.

## Integrations and Automation

### What Third-Party Integrations Are Supported

Jira supports integration with a large number of third-party tools. Main categories include: code management tools like GitHub, GitLab, Bitbucket; communication tools like Slack, Microsoft Teams; design tools like Figma, Adobe XD; continuous integration tools like Jenkins, CircleCI. The complete integration list is available on the Atlassian Marketplace.

### How to Set Up Automation Rules

Find "Automation" in project settings and click "Create rule". You can set triggers (when to trigger), conditions (what conditions to meet), and actions (what to do). For example, you can set up a rule to automatically send notifications to the issue creator when an issue is marked as resolved.

### How to Configure Webhooks

Webhooks allow Jira to communicate with external systems in real time. Find Webhooks in admin settings under "Developer tools" and add new webhooks with trigger conditions and target URLs. When trigger conditions are met, Jira will send POST requests to the specified URL.

## Billing and Subscriptions

### How to Upgrade or Downgrade Plans

Find "Billing" or "Subscription" in admin settings to view your current plan and choose to upgrade or downgrade. Upgrades take effect immediately, while downgrades take effect at the end of the current billing cycle. Some features may require manual configuration to adapt to new plan limits.

### Can I Pay Monthly

Yes, Jira supports monthly subscriptions. Monthly subscriptions allow more flexible user count adjustments, but the unit price is usually higher than annual subscriptions. Choosing annual subscriptions can save up to 17% in discounts.

### How to Cancel Subscription

You can cancel your Jira subscription at any time. After cancellation, you will lose access to paid features but can continue using the Free plan or export data. Complete the cancellation in the "Billing" section of admin settings.

## Data and Security

### How to Export Data

Jira allows administrators to export project data. Find "Import and Export" in project settings to export as CSV or XML format. Exported data includes issues, comments, attachments, and more.

### Where Is Data Stored

Data for the cloud version is stored on Atlassian's cloud servers, with the specific geographic location depending on your chosen region. When purchasing Enterprise plans, you can select specific data residency regions to meet compliance requirements.

### How to Ensure Data Security

Jira provides multi-layered security: data encryption in transit and at rest; SSO single sign-on and SCIM user management support; IP allowlists to restrict access sources; comprehensive audit logs recording all critical operations. Enterprise plans also include Atlassian Guard advanced security features.

## Performance and Usage

### What to Do If Issues Load Slowly

If Jira loads slowly, try the following: clear browser cache and cookies; check if network connection is stable; close unnecessary browser plugins; try using a different browser. If the problem persists, the project may have too much data. Consider archiving old data or optimizing view filters.

### How to Search for Specific Issues

Jira provides powerful search capabilities. You can use JQL (Jira Query Language) for advanced searches, such as finding issues by assignee, status, or creation date. Simple searches can use the quick search box, entering issue keywords.
