## 快速開始

### 安裝

```bash
npm install element-ui --save
```

### 引入

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```

## 常用元件

- **基礎**：Button、Layout、Container

- **資料**：Table、Tree、Pagination
