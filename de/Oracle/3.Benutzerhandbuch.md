# Oracle Datenbank Benutzerhandbuch

## Verbindung zur Datenbank

### Mit SQL*Plus

```bash
# Als SYSDBA verbinden
sqlplus / as sysdba

# Mit Benutzername/Passwort
sqlplus benutzer/passwort@datenbank
```

---

## Grundlegende SQL-Operationen

### Tabelle erstellen

```sql
CREATE TABLE mitarbeiter (
    mitarbeiter_id NUMBER(10) PRIMARY KEY,
    vorname VARCHAR2(50) NOT NULL,
    nachname VARCHAR2(50) NOT NULL,
    email VARCHAR2(100),
    gehalt NUMBER(10, 2)
);
```

## PL/SQL

### Gespeicherte Prozedur

```sql
CREATE OR REPLACE PROCEDURE update_gehalt (
    p_mitarbeiter_id IN NUMBER,
    p_neues_gehalt IN NUMBER
) AS
BEGIN
    UPDATE mitarbeiter
    SET gehalt = p_neues_gehalt
    WHERE mitarbeiter_id = p_mitarbeiter_id;
    COMMIT;
END;
/
```
