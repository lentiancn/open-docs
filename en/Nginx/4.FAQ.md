# Nginx FAQ

## Installation

**How to check Nginx version?**
```bash
nginx -v
```

**Where is the config file?**
- Debian/Ubuntu: `/etc/nginx/nginx.conf`
- CentOS: `/etc/nginx/nginx.conf`

## Configuration

**How to test config?**
```bash
nginx -t
```

**How to reload config?**
```bash
nginx -s reload
```

## Performance

**How to optimize?**
```nginx
worker_processes auto;
worker_connections 4096;
gzip on;
```

## Security

**Enable HTTPS:**
```nginx
server {
    listen 443 ssl;
    ssl_certificate cert.crt;
    ssl_certificate_key cert.key;
}
```

**Rate limiting:**
```nginx
limit_req_zone $binary_remote_addr zone=one:10m rate=10r/s;
```

## Errors

- **400**: Check request size
- **403**: Check file permissions
- **404**: Check file path
- **502**: Backend service not running
- **504**: Timeout, increase proxy_read_timeout

## Proxy

**Get real IP:**
```nginx
proxy_set_header X-Real-IP $remote_addr;
```
