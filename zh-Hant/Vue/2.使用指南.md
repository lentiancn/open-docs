# Vue 使用指南

## 基本概念

### Vue 實例

```javascript
import { createApp } from 'vue'

const app = createApp({
  data() {
    return {
      message: '你好 Vue!'
    }
  }
})

app.mount('#app')
```

## 響應式

### Vue 3 Composition API（推薦）

```javascript
import { ref, reactive, computed } from 'vue'

export default {
  setup() {
    // ref 用於原始類型
    const count = ref(0)
    
    // reactive 用於對象
    const state = reactive({
      name: 'Vue',
      version: 3
    })
    
    // 計算屬性
    const doubleCount = computed(() => count.value * 2)
    
    return { count, state, doubleCount }
  }
}
```

## 組件

### 創建組件

```vue
<template>
  <div class="my-component">
    <h2>{{ title }}</h2>
    <button @click="emitEvent">點擊</button>
  </div>
</template>

<script setup>
import { defineProps, defineEmits } from 'vue'

const props = defineProps({
  title: String
})

const emit = defineEmits(['click'])

const emitEvent = () => {
  emit('click', '你好！')
}
</script>
```

## 條件渲染

```html
<!-- v-if -->
<div v-if="show">內容</div>
<div v-else>默認</div>

<!-- v-show -->
<div v-show="visible">切換可見性</div>
```

## 列表渲染

```html
<!-- v-for 數組 -->
<ul>
  <li v-for="(item, index) in items" :key="item.id">
    {{ index }}. {{ item.name }}
  </li>
</ul>
```

## 事件處理

```html
<!-- 點擊事件 -->
<button @click="handleClick">點擊</button>

<!-- 修飾符 -->
<button @click.stop="handleClick">停止冒泡</button>
<button @click.prevent="handleSubmit">阻止默認</button>

<!-- 鍵盤修飾符 -->
<input @keyup.enter="submit" />
```

## 表單輸入綁定

```vue

    <el-checkbox v-model="form.checked">標籤</el-checkbox>
    <button type="submit">提交</button>
  </form>

<script setup>
import { reactive } from 'vue'

const form = reactive({
  name: '',
  checked: false
})

const submitForm = () => {
  console.log(form)
}
</script>
```

## 計算屬性

```javascript
import { ref, computed } from 'vue'

const firstName = ref('John')
const lastName = ref('Doe')

// 計算屬性 - 緩存，只有依賴改變時才重新計算
const fullName = computed(() => {
  return `${firstName.value} ${lastName.value}`
})
```

## 監聽器

```javascript
import { ref, watch } from 'vue'

const count = ref(0)

// 簡單監聽器
watch(count, (newValue, oldValue) => {
  console.log(`從 ${oldValue} 變為 ${newValue}`)
})
```

## 生命週期鈎子

### Vue 3 Composition API

| 鈎子 | 說明 |
|------|------|
| `setup()` | 在其他鈎子之前調用 |
| `onMounted()` | 組件掛載後 |
| `onUpdated()` | 組件更新後 |
| `onUnmounted()` | 組件卸載後 |

```javascript
import { onMounted, onUpdated, onUnmounted } from 'vue'

export default {
  setup() {
    onMounted(() => {
      console.log('掛載完成!')
    })
  }
}
```

## Vue Router

### 基本設置

```javascript
import { createRouter, createWebHistory } from 'vue-router'
import Home from '../views/Home.vue'

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```

### 導航

```vue
<template>
  <button @click="$router.push('/about')">關於</button>
  <router-link to="/">首頁</router-link>
</template>
```

## Pinia 狀態管理

### 定義 Store

```javascript
import { defineStore } from 'pinia'

export const useUserStore = defineStore('user', {
  state: () => ({
    name: 'John',
    email: 'john@example.com'
  }),
  getters: {
    displayName: (state) => `${state.name} (${state.email})`
  },
  actions: {
    updateName(newName) {
      this.name = newName
    }
  }
})
```
