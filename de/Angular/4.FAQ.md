# Angular FAQ

## F: ng wird nicht erkannt?
**A:** Node.js prüfen: `node --version`. Terminal neu starten.

## F: Installation zu langsam?
**A:** Spiegel: `npm config set registry https://registry.npmmirror.com`

## F: CORS Fehler?
**A:** Proxy konfigurieren oder Backend CORS-Headers hinzufügen.

## F: Angular aktualisieren?
**A:** `npm install -g @angular/cli@latest`
