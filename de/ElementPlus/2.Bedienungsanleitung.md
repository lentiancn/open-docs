# Element Plus Bedienungsanleitung

Diese Anleitung erklärt, wie Sie Element Plus-Komponenten in Ihrer Vue 3-Anwendung verwenden.

## Schnellstart

### Grundlegende Vorlage

```vue
<template>
  <div>
    <el-button type="primary">Primärer Button</el-button>
    <el-input v-model="input" placeholder="Text eingeben" />
    <el-date-picker v-model="date" type="date" placeholder="Datum auswählen" />
  </div>
</template>

<script setup>
import { ref } from 'vue'

const input = ref('')
const date = ref('')
</script>
```

### Formular

```vue
<template>
  <el-form :model="form" :rules="rules" ref="formRef" label-width="120px">
    <el-form-item label="Aktivitätsname" prop="name">
      <el-input v-model="form.name"></el-input>
    </el-form-item>
    
    <el-form-item label="Aktivitätszone" prop="region">
      <el-select v-model="form.region" placeholder="Auswählen">
        <el-option label="Zone eins" value="shanghai"></el-option>
        <el-option label="Zone zwei" value="beijing"></el-option>
      </el-select>
    </el-form-item>
    
    <el-form-item label="Aktivitätszeit">
      <el-date-picker
        v-model="form.date"
        type="datetimerange"
        range-separator="Bis"
        start-placeholder="Startdatum"
        end-placeholder="Enddatum"
      ></el-date-picker>
    </el-form-item>
    
    <el-form-item label="Sofortige Lieferung">
      <el-switch v-model="form.delivery"></el-switch>
    </el-form-item>
    
    <el-form-item label="Ressourcen">
      <el-radio-group v-model="form.resource">
        <el-radio label="Sponsor">Sponsor</el-radio>
        <el-radio label="Venue">Veranstaltungsort</el-radio>
      </el-radio-group>
    </el-form-item>
    
    <el-form-item label="Beschreibung">
      <el-input v-model="form.desc" type="textarea"></el-input>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="onSubmit">Erstellen</el-button>
      <el-button @click="onReset">Zurücksetzen</el-button>
    </el-form-item>
  </el-form>
</template>

<script setup>
import { reactive, ref } from 'vue'

const formRef = ref(null)

const form = reactive({
  name: '',
  region: '',
  date: [],
  delivery: false,
  resource: '',
  desc: ''
})

const rules = {
  name: [
    { required: true, message: 'Bitte Aktivitätsname eingeben', trigger: 'blur' },
    { min: 3, max: 5, message: 'Länge sollte 3 bis 5 sein', trigger: 'blur' }
  ],
  region: [
    { required: true, message: 'Bitte Aktivitätszone auswählen', trigger: 'change' }
  ]
}

const onSubmit = () => {
  formRef.value.validate((valid) => {
    if (valid) {
      console.log('absenden!')
    }
  })
}

const onReset = () => {
  formRef.value.resetFields()
}
</script>
```

## Best Practices

### 1. Composition API verwenden

```vue
<script setup>
import { ref, reactive } from 'vue'

// ref für primitive Werte verwenden
const input = ref('')
const visible = ref(false)

// reactive für Objekte verwenden
const form = reactive({
  name: '',
  age: 0
})
</script>
```

### 2. Formularvalidierung

```vue
<script setup>
const rules = {
  name: [
    { required: true, message: 'Bitte Name eingeben', trigger: 'blur' },
    { min: 3, max: 10, message: 'Länge 3-10', trigger: 'blur' }
  ],
  email: [
    { required: true, message: 'Bitte E-Mail eingeben', trigger: 'blur' },
    { type: 'email', message: 'Ungültige E-Mail', trigger: 'blur' }
  ],
  age: [
    { required: true, message: 'Bitte Alter eingeben', trigger: 'blur' },
    { type: 'number', min: 18, message: 'Muss 18+ sein', trigger: 'blur' }
  ]
}
</script>
```

### 3. Ladezustände


## Häufige Probleme

### Komponente wird nicht gerendert?

Stellen Sie sicher, dass Komponenten korrekt importiert oder global registriert sind.

### Stile werden nicht angewendet?

Überprüfen Sie, ob Element Plus-Stile importiert sind:

```javascript
import 'element-plus/dist/index.css'
```

### Icons werden nicht angezeigt?

Icons global registrieren:

```javascript
import * as icons from '@element-plus/icons-vue'
for (const [key, icon] of Object.entries(icons)) {
    app.component(key, icon)
}
```
