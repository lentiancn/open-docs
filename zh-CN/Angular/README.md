## 关于 Angular

Angular 是一个基于 TypeScript 的开源 Web 应用框架，提供了完整的解决方案来构建单页应用程序（SPA）。

### 主要特性

- **组件化架构**：基于组件的设计，易于维护和测试
- **TypeScript 支持**：强类型支持，提供更好的开发体验
- **依赖注入**：强大的依赖注入系统，提高代码模块化
- **双向数据绑定**：自动同步模型和视图
- **路由系统**：完整的客户端路由支持
- **HTTP 客户端**：简洁的 HTTP 通信 API
- **表单处理**：响应式表单和模板驱动表单
- **测试支持**：内置单元测试和端到端测试支持

### Angular 版本

| 版本 | 发布时间 | 特性 |
|------|----------|------|
| Angular 17 | 2023年11月 | 全新控制流、信号、独立组件 |
| Angular 16 | 2023年5月 | 信号(Signals) API |
| Angular 15 | 2022年11月 | 独立组件改进 |
| Angular 14 | 2022年6月 | 强类型表单 |

## 快速开始

### 安装 Angular CLI

```bash
npm install -g @angular/cli
```

### 创建新项目

```bash
ng new my-app
cd my-app
```

### 启动开发服务器

```bash
ng serve
```

在浏览器中打开 http://localhost:4200

### 创建组件

```bash
ng generate component my-component
# 或简写
ng g c my-component
```
