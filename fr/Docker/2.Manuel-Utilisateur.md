# Docker Manuel utilisateur

## 1. Concepts de base

### Images

```bash
docker images
docker pull ubuntu:22.04
docker build -t my-image .
```

### Conteneurs

```bash
docker run -it ubuntu /bin/bash
docker run -d nginx
docker ps
docker stop container_id
docker rm container_id
docker exec -it container_id /bin/bash
```

---

## 2. Commandes

```bash
docker pull nginx
docker run -d -p 8080:80 nginx
docker logs container_id
```

## 3. Dockerfile

```dockerfile
FROM ubuntu:22.04
WORKDIR /app
COPY . .
EXPOSE 3000
CMD ["npm", "start"]
```

## 4. Docker Compose

```yaml
version: '3.8'
services:
  web:
    build: .
    ports:
      - "3000:3000"
```

```bash
docker-compose up -d
docker-compose down
```
