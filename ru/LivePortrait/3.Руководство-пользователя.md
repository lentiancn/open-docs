# Руководство пользователя LivePortrait

В этом документе представлены подробные инструкции по использованию LivePortrait, включая базовый вывод, интерфейс Gradio, режим животных, редактирование видео и другие функции.

## Базовый вывод

### Управление изображением

Наиболее простое использование включает использование статического изображения в качестве источника и управляющего видео в качестве эталона анимации:

```bash
python inference.py
```

По умолчанию скрипт использует примеры изображений из каталога assets/examples/source и примеры управляющих видео из каталога assets/examples/driving. При успешном выполнении результаты сохраняются в каталоге animations с именем "исходный_файл--управляющий_файл_concat.mp4".

Выходное видео содержит три части: управляющее видео, исходное изображение/видео и сгенерированный результат анимации, соединенные последовательно.

### Пользовательские входные данные

Вы можете указать свои собственные файлы источника и управления с помощью параметров командной строки:

```bash
# Файл источника является изображением
python inference.py -s assets/examples/source/s9.jpg -d assets/examples/driving/d0.mp4

# Файл источника является видео
python inference.py -s assets/examples/source/s13.mp4 -d assets/examples/driving/d0.mp4
```

Параметр -s указывает путь к исходному изображению или видео, а параметр -d указывает путь к управляющему видео.

### Просмотр дополнительных параметров

Просмотрите все доступные параметры командной строки:

```bash
python inference.py -h
```

Параметры включают путь к выходному файлу, размер изображения, скорость анимации, метод сшивки и другие детальные настройки.

## Использование веб-интерфейса Gradio

Для пользователей, не знакомых с интерфейсами командной строки, LivePortrait предлагает графический интерфейс на основе Gradio.

### Запуск интерфейса человеческого режима

```bash
python app.py
```

После запуска откройте браузер по указанному URL-адресу (обычно http://localhost:7860). В интерфейсе загрузите исходное изображение или видео, затем выберите управляющее видео и нажмите кнопку "сгенерировать", чтобы получить результат анимации.

Вы можете указать порт с помощью параметра --server_port, создать публичную ссылку с помощью параметра --share и указать адрес сервера с помощью параметра --server_name.

### Запуск интерфейса режима животных

В системах Linux с GPU NVIDIA вы можете запустить интерфейс режима животных:

```bash
python app_animals.py
```

## Спецификации управляющего видео

Для оптимальных результатов управляющие видео должны соответствовать следующим требованиям:

**Соотношение сторон**: Рекомендуется обрезать до квадрата 1:1 (например, 512x512 или 256x256 пикселей). Вы также можете включить автоматическую обрезку (--flag_crop_driving_video).

**Требования к содержимому**: Управляющее видео должно фокусироваться на области головы, аналогично примерным видео. Движения плеч должны быть минимальны.

**Требование к первому кадру**: Первый кадр управляющего видео должен быть лицом в анфас с нейтральным выражением. Это важно для обеспечения точных эффектов анимации.

### Автоматическая обрезка

Если ваше управляющее видео не соответствует вышеуказанным требованиям, вы можете использовать функцию автоматической обрезки:

```bash
python inference.py -s assets/examples/source/s9.jpg -d assets/examples/driving/d13.mp4 --flag_crop_driving_video
```

Если результаты автоматической обрезки неудовлетворительны, вы можете настроить параметры --scale_crop_driving_video и --vy_ratio_crop_driving_video для точной настройки масштаба и смещения.

## Использование шаблонов движения

Для ускорения вывода и защиты конфиденциальности LivePortrait поддерживает использование предварительно сгенерированных файлов шаблонов движения (.pkl). Эти файлы шаблонов записывают информацию о движении из управляющих видео и могут быть повторно использованы без обработки исходного видео каждый раз.

### Использование

```bash
# Анимация портрета
python inference.py -s assets/examples/source/s9.jpg -d assets/examples/driving/d5.pkl

# Редактирование портретного видео
python inference.py -s assets/examples/source/s13.mp4 -d assets/examples/driving/d5.pkl
```

Шаблоны не только ускоряют вывод, но и защищают конфиденциальность — использование шаблонов устраняет необходимость загрузки исходных управляющих видео.

## Редактирование портретного видео (Video-to-Video)

LivePortrait поддерживает редактирование и изменение выражений и движений головы субъектов в существующих видео.

### Баазовое использование

```bash
# Источник — изображение, управление — видео
python inference.py -s assets/examples/source/s9.jpg -d assets/examples/driving/d0.mp4
```

### Настройка параметров анимации

**Интенсивность выражения**: Настройте силу изменений выражения с помощью параметра --driving_multiplier. Значения больше 1 усиливают изменения выражения, а значения меньше 1 уменьшают их.

**Контроль сшивки**: Используйте параметр --flag_stitching для управления включением сшивки. Отключение сшивки может изменить способ представления лицевых областей.

## Ускорение вывода

### Ускорение Torch Compile

LivePortrait поддерживает использование функции torch.compile от PyTorch для ускорения вывода. Первый вывод запускает процесс оптимизации (примерно одну минуту), после чего скорость вывода улучшается на 20-30%. Эффекты ускорения могут различаться в зависимости от версии CUDA.

```bash
python app.py --flag_do_torch_compile
```

Примечание: torch.compile не поддерживается в Windows и macOS.

### Тест скорости

Проект предоставляет скрипт тестирования скорости вывода:

```bash
python speed.py
```

Результаты теста показывают время обработки каждого модуля, помогая вам определить узкие места производительности.

## Режим животных

Режим животных поддерживает анимацию портретов домашних животных, таких как кошки и собаки.

### Условия использования

Режим животных тестировался только в следующих средах: система Linux с GPU NVIDIA, система Windows с GPU NVIDIA.

### Требования к установке

Перед использованием режима животных установите зависимость X-Pose, как описано в руководстве по установке, и загрузите веса модели животных.

### Запуск вывода в режиме животных

```bash
python inference_animals.py -s assets/examples/source/s39.jpg -d assets/examples/driving/wink.pkl --driving_multiplier 1.75 --no_flag_stitching
```

### Интерфейс режима животных

```bash
python app_animals.py
```

## Региональный контроль

LivePortrait поддерживает контроль анимации для определенных областей изображений, обеспечивая более точную настройку эффектов.

### Включение регионального контроля

В интерфейсе Gradio вы можете найти параметры регионального контроля для выбора анимируемых лицевых областей (например, глаза, рот, положение головы и т. д.).

## Редактирование портрета

В интерфейсе Gradio пользователи могут точно редактировать различные атрибуты исходного портрета, включая выражение, направление взгляда, угол наклона головы и многое другое.

## Защита конфиденциальности

LivePortrait предоставляет несколько функций защиты конфиденциальности:

**Шаблоны движения**: Используйте файлы шаблонов .pkl вместо исходных управляющих видео, чтобы избежать загрузки конфиденциального содержимого

**Автоматическая обрезка**: Автоматически обрезайте управляющие видео, фокусируясь на голове, уменьшая фоновую информацию

**Защита видео**: Функция создания шаблонов преобразует видео в шаблоны движения перед использованием

## Формат вывода

Формат вывода по умолчанию — видеофайл MP4, содержащий три части: управляющее видео, исходное содержимое и сгенерированный результат. Используйте параметр --no_flag_concat для вывода только сгенерированной части анимации.

## Интеграция инструментов сообщества

LivePortrait интегрирован несколькими проектами сообщества:

**ComfyUI**: Несколько узлов ComfyUI поддерживают LivePortrait с функциями предварительного просмотра в реальном времени

**Stable Diffusion WebUI**: Расширение sd-webui-live-portrait интегрирует LivePortrait в SD WebUI

**FaceFusion**: Версия 3.0 интегрирует LivePortrait как процессор восстановления выражений и редактора лица

**HuggingFace Spaces**: Попробуйте LivePortrait онлайн без локальной установки

## Рекомендации

Для оптимальных эффектов анимации следуйте этим рекомендациям:

Выбирайте высококачественные исходные изображения: Четкие фронтальные портреты дают наилучшие результаты

Используйте профессиональные управляющие видео: Видео с фронтальными и естественными выражениями дают лучшие эффекты

Контролируйте длину управляющего видео: Слишком длинные управляющие видео увеличивают время обработки и могут привести к ухудшению эффекта

Устанавливайте параметры разумно: Настраивайте интенсивность выражения и другие параметры на основе фактических результатов

Поддерживайте согласованное освещение: Старайтесь поддерживать условия освещения согласованными между исходными изображениями и управляющими видео
