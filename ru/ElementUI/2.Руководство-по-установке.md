# Element UI Руководство по установке

## Требования

- Node.js: 12.0.0+
- npm: 6.0.0+
- Vue: 2.6.0+

## Установка

```bash
npm install element-ui --save
```

## Быстрый старт

### Полный импорт

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```

### Импорт по требованию

```bash
npm install babel-plugin-component -D
```

Настройка babel.config.js:

```javascript
module.exports = {
  plugins: [
    ['component', {
      libraryName: 'element-ui',
      styleLibraryName: 'theme-chalk'
    }]
  ]
}
```

Импорт компонентов:

```javascript
import { Button, Select } from 'element-ui'
Vue.use(Button)
```
