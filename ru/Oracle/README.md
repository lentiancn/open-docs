# Oracle

Oracle Database — система управления реляционными базами данных, разработанная корпорацией Oracle.

## Введение

Oracle Database — это система управления реляционными базами данных корпоративного уровня, известная своей надежностью, производительностью и масштабируемостью.

## Основные характеристики

- **Высокая надежность**: Соответствие ACID
- **Высокая производительность**: Оптимизатор и кэш
- **Масштабируемость**: Поддержка больших баз данных
- **Безопасность**: Мелкозернистый контроль безопасности
- **Репликация**: Аварийное восстановление Data Guard
- **Партиционирование**: Партиционирование больших таблиц
- **PL/SQL**: Процедурное расширение SQL

## Быстрый старт

### Установка

```bash
# Linux
./runInstaller

# Docker
docker run -d -e ORACLE_PASSWORD=secret container-registry.oracle.com/database/express:latest
```

### Подключение

```bash
sqlplus / as sysdba
```
