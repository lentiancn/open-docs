# Guide d'Installation de Docker

Ce guide fournit des instructions détaillées pour installer Docker sur différents systèmes d'exploitation.

## Méthodes d'Installation

Vous pouvez installer Docker de plusieurs manières :

1. **Docker Desktop** : Inclut Docker Engine dans une application de bureau, recommandé pour la plupart des utilisateurs
2. **Docker Engine uniquement** : Installer uniquement les composants principaux de Docker, adapté aux serveurs Linux
3. **Docker Engine via le dépôt apt** : Installer depuis le dépôt apt officiel de Docker
4. **Installation manuelle** : Télécharger les paquets deb et installer manuellement

## Installation de Docker Desktop

Docker Desktop est la manière la plus simple et la plus rapide de commencer. Il inclut un environnement de développement Docker complet, incluant Docker Engine, Docker CLI, Docker Compose, Docker Build et Kubernetes.

### Configuration Requise

**macOS :**

- macOS 10.15 (Catalina) ou supérieur
- Au moins 4 Go de RAM
- Au moins 4 Go d'espace disque disponible

**Windows :**

- Windows 10 Pro, Enterprise ou Education (64 bits)
- Au moins 4 Go de RAM
- Hyper-V ou WSL 2 activé

**Linux :**

- Ubuntu 20.04 ou supérieur
- Debian 11 ou supérieur
- Fedora 33 ou supérieur
- Au moins 4 Go de RAM

### Installation sur macOS

1. Téléchargez Docker Desktop pour Mac depuis le site officiel
2. Double-cliquez sur le fichier .dmg
3. Faites glisser l'icône Docker vers le dossier "Applications"
4. Lancez Docker Desktop

### Installation sur Windows

1. Téléchargez Docker Desktop pour Windows
2. Exécutez le programme d'installation
3. Suivez les instructions de l'assistant d'installation
4. Redémarrez Docker Desktop

### Installation sur Linux (Ubuntu)

```bash
# Mettez à jour la liste des paquets
sudo apt-get update

# Installez les dépendances requises
sudo apt-get install ca-certificates curl gnupg lsb-release

# Ajoutez la clé GPG de Docker
sudo mkdir -p /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg

# Configurez le dépôt
echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

# Installez Docker
sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

## Installation de Docker Engine

Si vous souhaitez installer uniquement Docker Engine sans le bureau, suivez ces étapes.

### Ubuntu et Debian

```bash
sudo apt-get update
sudo apt-get install docker.io
sudo systemctl start docker
sudo systemctl enable docker
```

### Fedora

```bash
sudo dnf install docker-ce
sudo systemctl start docker
sudo systemctl enable docker
```

## Vérifier l'Installation

Après l'installation, vous pouvez vérifier l'installation avec la commande suivante :

```bash
docker --version
docker run hello-world
```

Si tout fonctionne, vous verrez un message de bienvenue de Docker.
