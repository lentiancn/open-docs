# Introduction

MultiTalk is an audio-driven multi-person conversational video generation framework developed by the MeiGen-AI team. This project aims to address the limitations of existing digital human generation technologies that only support single-person animation. It can generate videos containing multi-person interactions based on multi-stream audio input, reference images, and text prompts.

## Background

Traditional audio-driven human animation methods, such as talking head and talking body generation technologies, have made significant progress in generating synchronized facial movements and high-quality videos. However, existing methods primarily focus on single-person animation and struggle with multi-stream audio inputs, facing incorrect binding problems between audio and persons. Additionally, these methods have limitations in instruction-following capabilities. MultiTalk was created to solve these challenges.

## Core Features

The core feature of MultiTalk is automated multi-person conversational video generation. Users only need to provide audio containing multi-person dialogue, a reference image with multiple people, and a text prompt describing the video scene. The system automatically generates corresponding conversational videos with highly synchronized lip movements and smooth, natural visuals.

## Technical Characteristics

MultiTalk employs several innovative technologies to solve the challenges of multi-person video generation. First, the project introduces the Label Rotary Position Embedding method to resolve audio-person binding errors during multi-stream audio injection. Second, through partial parameter training and multi-task training strategies, the instruction-following capability of the base model is effectively preserved. Additionally, the project supports adaptive person localization, accurately positioning corresponding persons in generated videos based on the features of person regions in reference images.

## Application Scenarios

MultiTalk has wide-ranging application scenarios. In content creation, creators can use this tool to quickly generate digital human conversational videos for educational explanations, creative expression, or entertainment content. In research, the project provides new research directions and technical benchmarks for multi-person video generation. In business, enterprises can apply it to virtual hosts, multi-person customer service, digital human meetings, and other scenarios.

## Performance

MultiTalk demonstrates excellent performance on multiple benchmarks. The project achieves leading generation results on talking head, talking body, and multi-person datasets, with lip synchronization accuracy reaching industry-leading levels. The project supports generating videos at 480P and 720P resolutions, with maximum video clips up to 15 seconds.

## Project Information

MultiTalk is an open-source project, with code and model weights available on GitHub and Hugging Face. The related paper has been published on arXiv, detailing technical methods and experimental results. The project uses the Apache 2.0 license, permitting academic and commercial use.
