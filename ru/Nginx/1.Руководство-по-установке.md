# Руководство по установке Nginx

Руководство по установке Nginx на всех основных платформах.

---

## Linux

### Ubuntu / Debian

```bash
# Обновить пакеты
sudo apt update

# Установить Nginx
sudo apt install nginx

# Запустить Nginx
sudo systemctl start nginx

# Включить при загрузке
sudo systemctl enable nginx
```

### CentOS / RHEL

```bash
# Установить EPEL
sudo yum install epel-release

# Установить Nginx
sudo yum install nginx

# Запустить
sudo systemctl start nginx
```

---

## macOS

```bash
brew install nginx
brew services start nginx
```

---

## Windows

1. Скачать с [nginx.org](http://nginx.org/en/download.html)
2. Распаковать в `C:\nginx`
3. Запустить: `nginx.exe`

---

## Проверка

```bash
nginx -v
nginx -t
curl http://localhost
```
