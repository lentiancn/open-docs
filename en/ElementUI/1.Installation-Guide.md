# Element UI Installation Guide

Element UI is a UI component library for Vue 2.x applications.

## Version Overview

| Version | Vue Version | Status | Release Date |
|---------|-------------|--------|--------------|
| Element UI 2.x | Vue 2.6+ | Current | 2020 |
| Element UI 1.x | Vue 2.x | Legacy | 2016 |

**Note:** Element UI is for Vue 2 only. For Vue 3, use Element Plus.

### Element UI vs Element Plus

| Feature | Element UI (Vue 2) | Element Plus (Vue 3) |
|---------|-------------------|---------------------|
| Vue Version | 2.x | 3.x |
| TypeScript | Limited | Full Support |
| Status | Maintenance Mode | Active Development |

## Installation Methods

### Method 1: npm (Recommended)

```bash
# Install Element UI
npm install element-ui --save
```

### Method 2: Using Vue CLI

```bash
# Install Vue CLI
npm install -g @vue/cli

# Create project with Element UI template
vue init webpack my-element-app
```

### Method 3: CDN

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Element UI CDN Demo</title>
  <!-- Import Vue 2 -->
  <script src="https://unpkg.com/vue@2.6.14/dist/vue.js"></script>
  <!-- Import Element UI styles -->
  <link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
  <!-- Import Element UI -->
  <script src="https://unpkg.com/element-ui/lib/index.js"></script>
</head>
<body>
  <div id="app">
    <el-button @click="visible = true">Open Dialog</el-button>
    <el-dialog :visible.sync="visible" title="Hello Element UI">
      <p>Welcome to Element UI!</p>
    </el-dialog>
  </div>

  <script>
    new Vue({
      el: '#app',
      data: {
        visible: false
      }
    })
  </script>
</body>
</html>
```

## Project Setup

### Step 1: Create Vue Project

```bash
# Using Vue CLI
npm install -g @vue/cli
vue create my-element-app
cd my-element-app
```

### Step 2: Install Element UI

```bash
npm install element-ui --save
```

### Step 3: Import in main.js

#### Full Import

```javascript
import Vue from 'vue'
import App from './App.vue'

// Import Element UI
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

// Use Element UI
Vue.use(ElementUI)

Vue.config.productionTip = false

new Vue({
  render: h => h(App)
}).$mount('#app')
```

#### On-Demand Import

First, install plugins:

```bash
npm install babel-plugin-component -D
```

Configure babel.config.js:

```javascript
module.exports = {
  presets: ['@vue/cli-plugin-babel/preset'],
  plugins: [
    [
      'component',
      {
        libraryName: 'element-ui',
        styleLibraryName: 'theme-chalk'
      }
    ]
  ]
}
```

Then import components:

```javascript
import { Button, Dialog, Input } from 'element-ui'

Vue.component(Button.name, Button)
Vue.component(Dialog.name, Dialog)
Vue.component(Input.name, Input)

// Or use locally in components
export default {
  components: {
    [Button.name]: Button
  }
}
```

## Troubleshooting

### Component Not Found

**Problem:** Components not rendering

**Solutions:**
1. Ensure Element UI is installed: `npm list element-ui`
2. Verify import in main.js
3. Check Vue.use() is called

### Style Not Applied

**Problem:** Styles missing

**Solutions:**
1. Import CSS: `import 'element-ui/lib/theme-chalk/index.css'`
2. Check webpack CSS loader configuration

### IE10 Compatibility

```bash
# Install polyfills
npm install babel-polyfill --save
```

```javascript
// main.js
import 'babel-polyfill'
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```

## Migration to Element Plus (Vue 3)

If migrating to Vue 3, use Element Plus instead:

```bash
npm install element-plus --save
```

Key changes:
1. Package name: `element-ui` → `element-plus`
2. Import path: `element-ui/lib/index` → `element-plus`
3. Some component props may have changed

## Best Practices

1. **Use On-Demand Import** - Reduces bundle size
2. **Customize Theme** - Use SCSS variables
3. **Use Vue 2.6+** - Better compatibility
4. **Configure babel** - For production builds
