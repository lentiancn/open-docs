# PostgreSQL - 简介

PostgreSQL 是一个功能强大的开源对象关系型数据库系统，由加州大学伯克利分校开发，现已成为全球最流行的开源数据库之一。

## 核心特性

### 1. ACID 支持

PostgreSQL 完整支持 ACID 事务特性：
- 原子性（Atomicity）
- 一致性（Consistency）
- 隔离性（Isolation）
- 持久性（Durability）

### 2. 丰富的数据类型

支持多种数据类型：
- 基础类型：整数、浮点、字符串
- JSON/JSONB
- 数组
- UUID
- 网络地址
- 几何类型
- 自定义类型

### 3. 扩展性

强大的扩展系统：
- PostGIS（地理信息）
- pgvector（向量）
- TimescaleDB（时序数据）
- Citus（分布式）

### 4. 高级特性

- 外键约束
- 触发器
- 存储过程
- 视图
- 物化视图
- 窗口函数
- CTE（公用表表达式）

## 应用场景

- 企业级应用
- 地理信息系统（PostGIS）
- 数据仓库
- Web 应用
