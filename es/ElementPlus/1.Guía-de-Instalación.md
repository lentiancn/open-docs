# Element Plus Guía de Instalación

> Element Plus es una biblioteca de componentes UI de escritorio para Vue 3 desarrollada por el equipo de Element
> 
> Fecha de actualización: Febrero 2026
> 
> Versión actual: 2.x (Estable)

---

## 1. Introducción al Proyecto

Element Plus es una biblioteca de componentes UI de escritorio basada en Vue 3, desarrollada y mantenida por el equipo de Element. Como la versión de actualización de Element UI para Vue 3, Element Plus está completamente escrito en el estilo Composition API de Vue 3 y completamente reescrito en TypeScript, proporcionando una experiencia de desarrollo de componentes con seguridad de tipos.

La primera versión estable de Element Plus fue lanzada el 7 de febrero de 2022. Después de años de iteración, la API es muy estable. Actualmente soporta múltiples idiomas incluyendo chino simplificado, inglés y tiene un ecosistema comunitario activo.

## 2. Requisitos del Entorno

### 2.1 Requisitos del Entorno de Desarrollo

| Entorno | Versión mínima | Versión recomendada |
|------|----------|----------|
| Node.js | 16.x | 18.x o 20.x |
| npm | 8.x | 9.x o 10.x |
| Vue | 3.2.0 | 3.4.x |
| Vite | 4.x | 5.x |

### 2.2 Compatibilidad con Navegadores

Element Plus soporta todos los navegadores modernos, incluyendo:

- Chrome >= 80
- Firefox >= 78
- Safari >= 14
- Edge >= 80

No soporta Internet Explorer 11.

### 2.3 Requisitos de la Pila Tecnológica

| Tecnología | Descripción |
|------|------|
| Vue 3 | Debe usar Vue 3.2.0 o superior |
| TypeScript | Versión 4.5 o superior recomendada |
| Herramienta de construcción | Vite (recomendado) o Webpack |

## 3. Pasos de Instalación

### 3.1 Usando npm (Recomendado)

npm es el administrador de paquetes de Node.js y es la forma oficialmente recomendada de instalar Element Plus.

```bash
# Entra en tu directorio de proyecto Vue 3
cd tu-proyecto-vue

# Instala Element Plus
npm install element-plus
```

### 3.2 Usando yarn

```bash
yarn add element-plus
```

### 3.3 Usando pnpm

```bash
pnpm add element-plus
```

## 4. Importar en el Proyecto Vue 3

### 4.1 Importación Completa (Adecuada para Proyectos Pequeños)

La importación completa empaquetará todos los componentes, la configuración es simple pero el tamaño del paquete es mayor.

```typescript
// main.ts o main.js
import { createApp } from 'vue'
import App from './App.vue'

// Importar Element Plus y sus estilos
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'

const app = createApp(App)

// Usar Element Plus
app.use(ElementPlus)

app.mount('#app')
```

### 4.2 Importación bajo Demanda (Recomendada para Producción)

La importación bajo demanda puede reducir significativamente el tamaño del paquete, solo importando los componentes que realmente se usan.

#### 4.2.1 Instalar el Plugin de Auto-importación

Primero, instala unplugin-vue-components y unplugin-auto-import:

```bash
npm install -D unplugin-vue-components unplugin-auto-import
```

#### 4.2.2 Configurar Vite

Agrega la configuración en vite.config.ts:

```typescript
// vite.config.ts
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

Con esta configuración, los componentes se pueden importar automáticamente sin necesidad de importarlos manualmente. Por ejemplo:

```vue
<template>
  <el-button type="primary">Hola</el-button>
</template>
```

#### 4.2.3 Importación Bajo Demanda Manual (Alternativa)

Si no quieres usar el plugin de auto-importación, también puedes importar manualmente componentes individuales:

```typescript
// main.ts
import { createApp } from 'vue'
import App from './App.vue'

// Importar manualmente los componentes necesarios
import { ElButton, ElInput, ElForm } from 'element-plus'
import 'element-plus/es/components/button/style'
import 'element-plus/es/components/input/style'
import 'element-plus/es/components/form/style'

const app = createApp(App)
app.use(ElButton)
app.use(ElInput)
app.use(ElForm)

app.mount('#app')
```

## 5. Importar via CDN (Sin Herramienta de Construcción)

Si no quieres usar npm o herramientas de construcción, puedes importar Element Plus a través de CDN.

### 5.1 Importar via unpkg CDN

```html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Ejemplo Element Plus CDN</title>
  <!-- Importar Vue 3 -->
  <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
  <!-- Importar Element Plus -->
  <link rel="stylesheet" href="https://unpkg.com/element-plus/dist/index.css">
  <script src="https://unpkg.com/element-plus"></script>
</head>
<body>
  <div id="app">
    <el-button type="primary">Hola Element Plus</el-button>
  </div>

  <script>
    const { createApp, ref } = Vue
    
    const App = {
      setup() {
        const visible = ref(false)
        return { visible }
      },
      template: `
        <el-button @click="visible = true">Abrir Diálogo</el-button>
        <el-dialog v-model="visible" title="Hola">Esto es Element Plus</el-dialog>
      `
    }
    
    const app = createApp(App)
    app.use(ElementPlus)
    app.mount('#app')
  </script>
</body>
</html>
```

### 5.2 Importar via jsDelivr CDN

```html
<!-- Reemplaza unpkg con jsDelivr -->
<script src="https://cdn.jsdelivr.net/npm/vue@3/dist/vue.global.js"></script>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/element-plus/dist/index.css">
<script src="https://cdn.jsdelivr.net/npm/element-plus"></script>
```

## 6. Configuración de Internacionalización

Element Plus usa inglés por defecto. Para usar español, necesitas configurar la internacionalización.

### 6.1 Configuración Global en Español

```typescript
// main.ts
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import es from 'element-plus/es/locale/lang/es'
import 'element-plus/dist/index.css'

const app = createApp(App)

app.use(ElementPlus, {
  locale: es,
})

app.mount('#app')
```

## 7. Personalización del Tema

### 7.1 Usando Variables SCSS

Element Plus está escrito en SCSS, puedes personalizar el tema sobrescribiendo variables.

```scss
// styles/element-variables.scss
// Color primario
$--color-primary: #409EFF;

// Sobrescribir variables de componentes
@use "element-plus/theme-chalk/src/common/var.scss" as *;
```

### 7.2 Modo Oscuro

Element Plus tiene soporte incorporado para el modo oscuro:

```typescript
// Habilitar modo oscuro
import 'element-plus/theme-chalk/dark/css-vars.css'
```

Agrega la clase a la etiqueta HTML:

```html
<html class="dark">
  <!-- Tu aplicación -->
</html>
```

## 8. Problemas Comunes

### 8.1 Error de Instalación

**Problema**: npm install falla

**Soluciones**:

1. Limpiar la caché de npm: `npm cache clean --force`
2. Verificar la red, puedes usar mirrors nacionales:

```bash
# Configurar mirror de npm
npm config set registry https://registry.npmmirror.com
```

### 8.2 Estilos No Aplicados

**Problema**: Los estilos de los componentes se muestran anormalmente

**Soluciones**:

1. Asegúrate de importar el archivo de estilos: `import 'element-plus/dist/index.css'`
2. Verificar si la configuración de construcción maneja correctamente el CSS
3. Limpiar la caché del navegador y reconstruir
