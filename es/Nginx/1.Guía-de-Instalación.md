# Guía de Instalación de Nginx

Guía para instalar Nginx en todas las plataformas principales.

---

## Linux

### Ubuntu / Debian

```bash
# Actualizar paquetes
sudo apt update

# Instalar Nginx
sudo apt install nginx

# Iniciar Nginx
sudo systemctl start nginx

# Habilitar al iniciar
sudo systemctl enable nginx
```

### CentOS / RHEL

```bash
# Instalar EPEL
sudo yum install epel-release

# Instalar Nginx
sudo yum install nginx

# Iniciar
sudo systemctl start nginx
```

---

## macOS

```bash
brew install nginx
brew services start nginx
```

---

## Windows

1. Descargar de [nginx.org](http://nginx.org/en/download.html)
2. Extraer a `C:\nginx`
3. Ejecutar: `nginx.exe`

---

## Verificar

```bash
nginx -v
nginx -t
curl http://localhost
```
