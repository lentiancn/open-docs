# PostgreSQL Resumen

> La base de datos de código abierto más avanzada del mundo

---

## Descripción

PostgreSQL es un potente sistema de base de datos relacional de código abierto.

---

## Características

- Conformidad ACID
- Soporte JSON
- Búsqueda de texto completo
