# Jira

Jira — это программное обеспечение для управления проектами и отслеживания проблем, разработанное Atlassian, предоставляющее решения по управлению проектами более чем для 300 000 компаний по всему миру.

## Быстрый старт

Посетите веб-сайт Atlassian для регистрации и выберите план, подходящий для вашей команды. Вы можете быстро создать сайт Jira.

## Основные функции

Управление проектами и отслеживание проблем, Поддержка гибкой разработки (Scrum, Kanban), Командное сотрудничество и коммуникация, Интеграция с AI-ассистентом Rovo, Богатые сторонние интеграции, Мощные правила автоматизации.

## Ценообразование

Free: 10 пользователей бесплатно навсегда

Standard: 7,91 доллара США за пользователя в месяц

Premium: 14,54 доллара США за пользователя в месяц

Enterprise: свяжитесь с отделом продаж для получения цен
