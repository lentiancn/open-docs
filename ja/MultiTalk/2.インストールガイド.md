# MultiTalk インストールガイド

MultiTalkのインストールと設定ガイド。

## 前提条件

- Windows 10/11、macOS 10.15+、またはLinux
- マイクとスピーカー/ヘッドセット
- インターネット接続

## インストール

### Windows

1. インストーラーをダウンロード
2. インストーラーを実行
3. インストールウィザードに従う
4. MultiTalkを起動

### macOS

1. .dmgファイルをダウンロード
2. インストーラーを開く
3. MultiTalkをアプリケーションにドラッグ
4. MultiTalkを起動

## 設定

### オーディオ設定

1. 設定 > オーディオを開く
2. 入力デバイスを選択
3. 出力デバイスを選択
4. レベルをテスト

### プッシュトゥトーク

1. 設定 > ホットキーに行く
2. プッシュトゥトークを有効化
3. 使用するキーの組み合わせを押す

## トラブルシューティング

### オーディオ入力なし

**解決策:**
- マイクの権限を確認
- デバイス選択を確認

### エコー問題

**解決策:**
- ヘッドセットを使用
- スピーカーの音量を下げる
