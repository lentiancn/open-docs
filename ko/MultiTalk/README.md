# MultiTalk

게임 및 커뮤니케이션을 위한 강력한 다중 파티 음성 채팅 애플리케이션.

## MultiTalk이란?

MultiTalk은 게이머와 커뮤니티를 위한 다중 파티 음성 채팅을 가능하게 하는 데스크톱 애플리케이션입니다.

### 주요 기능

- **다중 파티 음성** - 여러 참가자 지원
- **낮은 지연** - 최소 지연으로 실시간 통신
- **오디오 소스** - 여러 오디오 입출력 장치
- **푸시 투 토크** - 구성 가능한 전송 키
- **노이즈 감소** - 내장 오디오 필터링
