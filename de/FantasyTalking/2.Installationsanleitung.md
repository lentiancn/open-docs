# FantasyTalking Installationsanleitung

## Anforderungen

- Node.js 14.0+ oder Python 3.7+
- FFmpeg für Audio-Verarbeitung
- 4GB RAM empfohlen

## Installation

```bash
# Node.js
npm install fantasy-talking

# Python
pip install fantasy-talking
```

## Schnellstart

```javascript
const client = new FantasyTalking({
  apiKey: 'your-api-key',
  voice: { style: 'neutral', language: 'de-DE' }
});

const audio = await client.speak('Hallo, das ist ein Test');
```

## Verifizierung

```bash
fantasy-talking --text "Test" --output test.wav
```
