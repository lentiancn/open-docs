# Element UI 安裝指南

> Element UI 是基於 Vue 2 的桌面端組件庫
> 
> 更新日期：2026年2月
> 
> 當前版本：2.x（穩定版）
> 
> 注意：如果是 Vue 3 項目，請使用 Element Plus

---

## 一、項目簡介

Element UI 是一套基於 Vue 2.0 的桌面端組件庫，由餓了麼團隊開發和維護。它提供了豐富的 PC 端常用組件，涵蓋了表單、表格、對話框、導航菜單、按鈕、輸入框等常用 UI 元素。

Element UI 的設計風格簡潔大方，文檔詳細完善，是國內最流行的 Vue 2 UI 組件庫之一。雖然官方已經停止更新（推薦遷移到 Element Plus for Vue 3），但由於國內大量存量項目仍在使用，Element UI 仍然具有很高的參考價值和學習意義。

### 1.1 版本說明

| 版本 | Vue 版本 | 狀態 | 說明 |
|------|----------|------|------|
| Element UI 2.x | Vue 2 | 穩定 | 當前文檔主要版本 |
| Element UI 1.x | Vue 2 | 停止維護 | 已不再推薦使用 |

### 1.2 與 Element Plus 的區別

| 特性 | Element UI | Element Plus |
|------|------------|--------------|
| Vue 版本 | Vue 2 | Vue 3 |
| TypeScript | 部分支持 | 完全支持 |
| Composition API | 不支持 | 支持 |
| 維護狀態 | 已停止 | 活躍維護 |
| 發布年份 | 2016-2021 | 2022-至今 |

### 2.1 開發環境要求

| 環境 | 最低版本 | 推薦版本 |
|------|----------|----------|
| Node.js | 8.x | 10.x 或 12.x |

| 構建工具 | Webpack 4.x | Webpack 4.x 或 5.x |

Element UI 支持以下瀏覽器版本：

- Chrome >= 49
- Firefox >= 45
- Safari >= 10
- Edge >= 14
- IE 11（有限支持，部分功能可能不可用）

### 2.3 技術棧要求

| 技術 | 說明 |
|------|------|
| Vue 2 | 必須使用 Vue 2.6.0 及以上版本 |
| Vue CLI | 推薦使用 3.x 或 4.x |
| npm / yarn | 包管理工具 |

## 三、安裝步驟

### 3.1 使用 npm 安裝（推薦）

```bash
# 進入 Vue 2 項目目錄
cd your-vue-project

# 安裝 Element UI
npm install element-ui --save
```

### 3.2 使用 yarn 安裝

```bash
yarn add element-ui
```

### 3.3 指定版本安裝

```bash
# 安裝最新 2.x 版本
npm install element-ui@latest

# 安裝特定版本
npm install element-ui@2.15.13
```

## 四、在 Vue 2 項目中引入

### 4.1 完整引入（適合小型項目）

在項目的入口文件（通常是 main.js）中引入 Element UI 和它的樣式文件：

```javascript
// main.js
import Vue from 'vue'
import App from './App.vue'

// 引入 Element UI
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)

new Vue({
  render: h => h(App)
}).$mount('#app')
```

### 4.2 按需引入（推薦，適合生產環境）

為了減少項目體積，建議按需引入只需要組件。

#### 4.2.1 安裝 babel-plugin-component

```bash
npm install babel-plugin-component -D
```

#### 4.2.2 配置 babel

在 .babelrc 或 babel.config.js 中添加配置：

```json
{
  "presets": [
    ["env", {
      "modules": false,
      "targets": {
        "browsers": ["> 1%", "last 2 versions", "not ie <= 8"]
      }
    }]
  ],
  "plugins": [
    ["component", {
      "libraryName": "element-ui",
      "styleLibraryName": "theme-chalk"
    }]
  ]
}
```

#### 4.2.3 按需引入組件

```javascript
// main.js
import Vue from 'vue'
import { Button, Form, FormItem, Input, Table, TableColumn } from 'element-ui'
import App from './App.vue'

// 註冊需要的組件
Vue.use(Button)
Vue.use(Form)
Vue.use(FormItem)
Vue.use(Input)
Vue.use(Table)
Vue.use(TableColumn)

new Vue({
  render: h => h(App)
}).$mount('#app')
```

### 4.3 使用 CDN 引入

如果你不想使用構建工具，可以通過 CDN 方式引入 Element UI：

```html
<!DOCTYPE html>
<html lang="zh-TW">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Element UI CDN 示例</title>
  <!-- 引入 Vue 2 -->
  <script src="https://cdn.jsdelivr.net/npm/vue@2.6.14/dist/vue.js"></script>
  <!-- 引入 Element UI -->
  <link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
  <script src="https://unpkg.com/element-ui"></script>
</head>
<body>
  <div id="app">
    <el-button type="primary" @click="visible = true">打開對話框</el-button>
    <el-dialog title="提示" :visible.sync="visible" width="30%">
      <span>這是一段信息</span>
      <span slot="footer" class="dialog-footer">
        <el-button @click="visible = false">取消</el-button>
        <el-button type="primary" @click="visible = false">確定</el-button>
      </span>
    </el-dialog>
  </div>

  <script>
    new Vue({
      el: '#app',
      data: {
        visible: false
      }
    })
  </script>
</body>
</html>
```

## 五、快速開始示例

### 5.1 基礎按鈕

```html
<template>
  <div>
    <el-button>默認按鈕</el-button>
    <el-button type="primary">主要按鈕</el-button>
    <el-button type="success">成功按鈕</el-button>
    <el-button type="info">信息按鈕</el-button>
    <el-button type="warning">警告按鈕</el-button>
    <el-button type="danger">危險按鈕</el-button>
  </div>
</template>
```

### 5.2 基礎表單

```html
<template>
  <el-form :model="form" :rules="rules" ref="loginForm" label-width="80px">
    <el-form-item label="用戶名" prop="username">
      <el-input v-model="form.username" placeholder="請輸入用戶名"></el-input>
    </el-form-item>
    <el-form-item label="密碼" prop="password">
      <el-input v-model="form.password" type="password" placeholder="請輸入密碼"></el-input>
    </el-form-item>
    <el-form-item>
      <el-button type="primary" @click="submitForm">登錄</el-button>
      <el-button @click="resetForm">重置</el-button>
    </el-form-item>
  </el-form>
</template>

<script>
export default {
  data() {
    return {
      form: {
        username: '',
        password: ''
      },
      rules: {
        username: [
          { required: true, message: '請輸入用戶名', trigger: 'blur' }
        ],
        password: [
          { required: true, message: '請輸入密碼', trigger: 'blur' }
        ]
      }
    }
  },
  methods: {
    submitForm() {
      this.$refs.loginForm.validate(valid => {
        if (valid) {
          console.log('提交登錄表單：', this.form)
        } else {
          this.$message.error('表單驗證失敗')
        }
      })
    },
    resetForm() {
      this.$refs.loginForm.resetFields()
    }
  }
}
</script>
```

### 5.3 基礎表格

```html
<template>
  <el-table :data="tableData" stripe border>
    <el-table-column prop="date" label="日期" width="180"></el-table-column>
    <el-table-column prop="name" label="姓名" width="180"></el-table-column>
    <el-table-column prop="address" label="地址"></el-table-column>
    <el-table-column label="操作" width="180">
      <template slot-scope="scope">
        <el-button size="mini" @click="handleEdit(scope.$index, scope.row)">編輯</el-button>
        <el-button size="mini" type="danger" @click="handleDelete(scope.$index, scope.row)">刪除</el-button>
      </template>
    </el-table-column>
  </el-table>
</template>

<script>
export default {
  data() {
    return {
      tableData: [{
        date: '2024-01-01',
        name: '張三',
        address: '北京市朝陽區'
      }, {
        date: '2024-01-02',
        name: '李四',
        address: '上海市浦東新區'
      }]
    }
  },
  methods: {
    handleEdit(index, row) {
      console.log('編輯：', index, row)
    },
    handleDelete(index, row) {
      console.log('刪除：', index, row)
    }
  }
}
</script>
```

### 5.4 對話框

```html
<template>
  <el-button type="text" @click="dialogVisible = true">打開對話框</el-button>

  <el-dialog
    title="編輯信息"
    :visible.sync="dialogVisible"
    width="30%"
    :before-close="handleClose">
    <span>這是一段對話框內容</span>
    <span slot="footer" class="dialog-footer">
      <el-button @click="dialogVisible = false">取 消</el-button>
      <el-button type="primary" @click="dialogVisible = false">確 定</el-button>
    </span>
  </el-dialog>
</template>

<script>
export default {
  data() {
    return {
      dialogVisible: false
    }
  },
  methods: {
    handleClose(done) {
      this.$confirm('確認關閉？')
        .then(_ => {
          done()
        })
        .catch(_ => {})
    }
  }
}
</script>
```

## 六、國際化配置

Element UI 默認使用英文，要使用中文需要配置國際化。

### 6.1 全局中文配置

```javascript
// main.js
import Vue from 'vue'
import ElementUI from 'element-ui'
import locale from 'element-ui/lib/locale/lang/zh-CN'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI, {
  locale
})
```

| 語言 | 值 |

## 七、主題定製

### 7.1 使用在線主題工具

訪問 Element UI 主題定製頁面：https://elementui.github.io/theme-chalk/

可以選擇主題色、邊框顏色、圓角等參數，下載定製後的樣式文件。

### 7.2 手動覆蓋變量

在項目中創建自定義樣式文件：

```scss
// styles/element-variables.scss
$--color-primary: #409EFF;
$--color-success: #67C23A;
$--color-warning: #E6A23C;
$--color-danger: #F56C6C;
$--color-info: #909399;

// 覆蓋組件變量
@import 'element-ui/packages/theme-chalk/src/common/var.scss';
```

然後在 main.js 中引入：

```javascript
import 'element-ui/lib/theme-chalk/index.css'
import './styles/element-variables.scss'
```

## 八、常見問題

### 8.1 安裝失敗

**問題**：npm install 安裝失敗

**解決方案**：

1. 清除 npm 緩存：`npm cache clean --force`
2. 檢查網絡，可以使用國內鏡像源

### 8.2 樣式不生效

**問題**：組件樣式顯示異常

**解決方案**：

1. 確保已引入樣式文件：`import 'element-ui/lib/theme-chalk/index.css'`
2. 檢查 webpack 配置是否正確處理了 CSS
3. 確保沒有 CSS 衝突

### 8.3 組件不顯示

**問題**：組件引入後不顯示

**解決方案**：

1. 檢查是否正確使用了 Vue.use(ElementUI)
2. 確認 Vue 版本是否在 2.6.0 以上
3. 檢查瀏覽器控制台是否有錯誤信息

## 十一、注意事項

1. **安全建議**：Element UI 已停止維護，存在已知安全漏洞，建議盡快遷移到 Element Plus

3. **長期支持**：如果是新項目，請直接使用 Element Plus (Vue 3)
