# Guía de Uso de MySQL

Operaciones básicas de MySQL.

---

## Conectar

```bash
mysql -u root -p
```

## Comandos Básicos

```sql
SHOW DATABASES;
CREATE DATABASE mydb;
USE mydb;
SHOW TABLES;
```

## Tablas

```sql
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100)
);
```

## CRUD

```sql
INSERT INTO users (name) VALUES ('John');
SELECT * FROM users;
UPDATE users SET name='Jane' WHERE id=1;
DELETE FROM users WHERE id=1;
```
