# Preguntas Frecuentes sobre MySQL

Esta página responde preguntas frecuentes sobre MySQL.

---

## Instalación

### ¿Qué versión de MySQL debo usar?

Para entornos de producción, se recomienda MySQL 8.0.x o 8.4.x.

### ¿Cómo instalo MySQL en Windows?

1. Descarga MySQL Installer de https://dev.mysql.com/downloads/installer/
2. Ejecuta el instalador
3. Elige instalación "Full" o "Custom"
4. Sigue el asistente

### ¿Cómo instalo MySQL con Docker?

```bash
docker run -d --name mysql \
  -e MYSQL_ROOT_PASSWORD=tu_password \
  -v mysql-data:/var/lib/mysql \
  -p 3306:3306 \
  mysql:8
```

## Conexión

### ¿Cómo me conecto a MySQL desde la línea de comandos?

```bash
mysql -u root -p
```

### ¿Olvidé la contraseña de root de MySQL?

**Linux:**
1. Detén MySQL: `sudo systemctl stop mysql`
2. Inicia en modo seguro: `sudo mysqld_safe --skip-grant-tables`
3. Conecta: `mysql -u root`
4. Restablece contraseña:
```sql
FLUSH PRIVILEGES;
ALTER USER 'root'@'localhost' IDENTIFIED BY 'nueva_contraseña';
```
5. Reinicia MySQL

## Bases de datos y tablas

### ¿Cómo creo una base de datos?

```sql
CREATE DATABASE miapp;
```

### ¿Cómo creo una tabla?

```sql
CREATE TABLE usuarios (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nombre VARCHAR(50) NOT NULL,
    email VARCHAR(100) NOT NULL UNIQUE
);
```

### ¿Cómo veo todas las tablas?

```sql
SHOW TABLES;
```

---

## Operaciones de datos

### ¿Cómo inserto datos?

```sql
INSERT INTO usuarios (nombre, email) VALUES ('juan', 'juan@ejemplo.com');
```

### ¿Cómo actualizo datos?

```sql
UPDATE usuarios SET email = 'nuevo@ejemplo.com' WHERE id = 1;
```

### ¿Cómo elimino datos?

```sql
DELETE FROM usuarios WHERE id = 1;
```

## Consultas

### ¿Cómo联合 tablas?

```sql
SELECT * FROM pedidos 
INNER JOIN usuarios ON pedidos.usuario_id = usuarios.id;
```

### ¿Cómo uso LIMIT y OFFSET?

```sql
SELECT * FROM usuarios LIMIT 10 OFFSET 20;
```

### ¿Cómo cuento filas?

```sql
SELECT COUNT(*) FROM usuarios;
```

---

## Índices

### ¿Cómo creo un índice?

```sql
CREATE INDEX idx_email ON usuarios(email);
```

### ¿Cuándo debo usar índices?

- Columnas en cláusulas WHERE
- Columnas en condiciones JOIN
- Columnas en ORDER BY
- Columnas con alta cardinalidad

## Rendimiento

### MySQL funciona lentamente

1. Verificar consultas lentas: `SHOW FULL PROCESSLIST;`
2. Analizar consultas: `EXPLAIN SELECT ...`
3. Agregar índices a columnas frecuentemente consultadas
4. Optimizar configuración

### ¿Cómo habilito el registro de consultas lentas?

En my.cnf:
```ini
slow_query_log = 1
slow_query_log_file = /var/log/mysql/slow.log
long_query_time = 2
```

## Backup y restauración

### ¿Cómo hago backup de una base de datos?

```bash
mysqldump -u root -p miapp > backup.sql
```

### ¿Cómo restauro una base de datos?

```bash
mysql -u root -p miapp < backup.sql
```

## Gestión de usuarios

### ¿Cómo creo un nuevo usuario?

```sql
CREATE USER 'nombre_usuario'@'localhost' IDENTIFIED BY 'password';
```

### ¿Cómo otorgo privilegios?

```sql
GRANT SELECT, INSERT, UPDATE ON miapp.* TO 'nombre_usuario'@'localhost';
```

## Errores y solución de problemas

### "Too many connections"

Aumenta max_connections en my.cnf:
```ini
max_connections = 300
```

### "Table is full"

- Verificar espacio en disco
- Aumentar tamaño de tablespace para InnoDB

### "Out of memory"

- Aumentar tamaño de buffer pool
- Optimizar consultas
- Agregar más RAM
