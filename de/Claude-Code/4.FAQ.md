# Claude Code FAQ

Häufig gestellte Fragen.

## Installation

### Q: Installation fehlgeschlagen?

**Lösungen**:
1. Systemanforderungen prüfen
2. VS Code auf neueste Version aktualisieren
3. Netzwerkverbindung prüfen
4. Neuinstallation versuchen

### Q: Anmeldung nicht möglich?

**Lösungen**:
1. API-Schlüssel gültig?
2. Netzwerkverbindung prüfen
3. Cache leeren versuchen

### Q: Erweiterung wird nicht angezeigt?

**Lösungen**:
1. VS Code neu starten
2. Erweiterung aktiviert?
3. VS Code Benachrichtigungen prüfen

## Nutzung

### Q: Langsame Antwort?

**Lösungen**:
1. Netzwerkverbindung prüfen
2. Kontextlänge reduzieren
3. Auf Serverantwort warten

### Q: Schlechte Codequalität?

**Lösungen**:
1. Beschreibungen konkreter
2. Mehr Kontext bereitstellen
3. Iterativ verbessern

### Q: Wie Modelle wechseln?

Modellversion in Einstellungen wählen.

### Q: Welche Sprachen werden unterstützt?

Unterstützt gängige Sprachen: Python, JavaScript, TypeScript, Go, Rust, Java, C++ und mehr.

## Konto

### Q: API-Schlüssel erhalten?

https://console.anthropic.com besuchen.

### Q: Kontingent aufgebraucht?

Zusätzliches Kontingent in Console kaufen.

## Sicherheit

### Q: Code wird offengelegt?

Claude Code speichert oder teilt deinen Code nicht.

### Q: Wie Privatsphäre schützen?

- Keine sensiblen Informationen in Gespräche eingeben
- Lokalen Modus verwenden
- Sitzungen regelmäßig löschen
