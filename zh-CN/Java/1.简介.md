# Java 简介

> 面向对象的编程语言

---

## 项目简介

Java 是由 Sun Microsystems 于 1995 年发布的面向对象编程语言，现由 Oracle 维护。Java 以其"一次编写，到处运行"的跨平台特性著称，广泛用于企业级应用、Android 开发、大数据处理等领域。

---

## 语言特性

- **面向对象**：完全面向对象设计
- **跨平台**：JVM 实现"一次编写，到处运行"
- **自动内存管理**：垃圾回收机制
- **多线程**：内置多线程支持
- **安全性**：安全沙箱机制
- **丰富的生态**：庞大的开源库和框架

---

## 适用场景

- 企业级应用开发
- Android 应用开发
- Web 应用开发
- 大数据处理（Hadoop、Spark）
- 分布式系统
- 金融系统
