# Bootstrap 자주 묻는 질문

자주 묻는 질문과 답변.

## 설치

### Q: CDN이 로드되지 않음?

**해결책**:
1. 네트워크 연결 확인
2. URL 확인
3 대안 CDN 사용

### Q: npm 설치 실패?

**해결책**:
1. npm 캐시 지우기
2. Node.js 버전 확인

### Q: jQuery가 아직 필요함?

아니요. Bootstrap 5는 jQuery를 더 이상 사용하지 않습니다.

## 사용

### Q: 스타일이 작동하지 않음?

**해결책**:
1. CSS 올바르게 임포트되었는지 확인
2. 브라우저 캐시 지우기

### Q: 반응형이 작동하지 않음?

viewport 메타 태그 추가:
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

## 호환성

### Q: IE 지원 안 함?

Bootstrap 5는 IE를 더 이상 지원하지 않습니다.

## 성능

### Q: 파일이 너무 큼?

**해결책**:
1. Sass 사용
2. .min 버전 사용
