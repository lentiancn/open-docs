# 설치 가이드

이 문서에서는 Apifox의 다양한 설치 방법을 안내하며, 로컬에서 Apifox를 빠르게 시작할 수 있도록 도와줍니다.

## 시스템 요구사항

Apifox는 다음 운영 체제를 지원합니다:

- Windows 7 이상 (64비트)
- macOS 10.13 이상
- Linux (Debian/Ubuntu 20.04+, Fedora 36+)

## 다운로드 방법

### 공식 웹사이트에서 다운로드

Apifox 공식 웹사이트 https://apifox.com/를 방문하여 "다운로드" 버튼을 클릭하고, 사용하는 운영 체제에 맞는 버전을 다운로드하세요.

### 버전 선택

Apifox는 두 가지 버전을 제공합니다:

- **Apifox** (표준판): 모든 핵심 기능이 포함되어 있어 대부분의 사용자에게 적합합니다
- **Apifox** (유료판): 팀 관리, 고급 자동화 등 더 많은 고급 기능이 포함되어 있습니다

기본 기능은 표준판에서도 충분히 사용할 수 있습니다.

## 설치 단계

### Windows 설치

1. Windows 설치 패키지(.exe 파일) 다운로드
2. 설치 프로그램을 더블 클릭하여 실행
3. 설치 경로 선택 (기본 경로 사용 권장)
4. 설치 완료 대기
5. Apifox 실행 후 로그인 또는 계정 등록

### macOS 설치

1. macOS 설치 패키지(.dmg 파일) 다운로드
2. 설치 패키지를 열고 Apifox를 Applications 폴더로 드래그
3. Apifox 실행 후 로그인 또는 계정 등록

최초 실행 시 시스템 환경 설정에서 "알 수 없는 개발자의 앱 허용"을 설정해야 할 수 있습니다.

### Linux 설치

**Debian/Ubuntu:**

```bash
# deb 패키지 다운로드
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.deb

# 설치
sudo dpkg -i Apifox-latest-linux.x64.deb

# 의존성이 없을 경우 설치 수정
sudo apt-get install -f
```

**Fedora:**

```bash
# rpm 패키지 다운로드
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.rpm

# 설치
sudo rpm -i Apifox-latest-linux.x64.rpm
```

## 계정 및 로그인

### 계정 등록

최초 사용 시 계정을 등록해야 합니다:

1. Apifox 실행
2. "注册" 버튼 클릭
3. 이메일과 비밀번호 입력 후 등록 완료
4. 이메일 인증 (필요한 경우)

### 로그인

다양한 로그인 방법을 지원합니다:

- 이메일/비밀번호 로그인
-第三方 계정 로그인 (GitHub, Google 등)

### 클라우드 동기화

로그인하면 프로젝트 데이터가 자동으로 클라우드에 동기화됩니다. 서로 다른 기기 간에 원활하게 전환할 수 있으며 데이터가 실시간으로 동기화됩니다.

## 초기 설정

최초 실행 후 다음 설정을 진행하는 것을 권장합니다:

### 테마 설정

"설정"에서 라이트 또는 다크 테마를 선택할 수 있습니다. 취향에 맞는 테마를 선택하세요.

### 프록시 설정

프록시를 통해 네트워크에 접근해야 하는 경우, "설정" -> "네트워크"에서 프록시 서버를 설정하세요.

### 가져오기/내보내기

- **가져오기**: Postman, Swagger, Har 등 다양한 형식의 프로젝트를 가져올 수 있습니다
- **내보내기**: OpenAPI, Markdown, HTML 등 다양한 형식으로 내보낼 수 있습니다

## 빠른 시작

### 프로젝트 생성

1. 왼쪽 "프로젝트" 옆의 "+" 클릭
2. 프로젝트 이름 입력
3. 프로젝트 유형 선택 (API 프로젝트 또는 Mock 프로젝트)

### 인터페이스 생성

1. 프로젝트에서 "+" 클릭 후 "새 인터페이스" 선택
2. 인터페이스 기본 정보 입력:
   - 인터페이스 이름
   - 요청 경로
   - 요청 메서드 (GET/POST/PUT/DELETE 등)
3. 요청 매개변수, 요청 본문, 응답等信息 입력
4. 인터페이스 저장

### 인터페이스 디버깅

1. 생성한 인터페이스 열기
2. "보내기" 버튼 클릭
3. 응답 결과 확인

### Mock 사용

1. 인터페이스에서 데이터 구조 정의
2. 시스템이 자동으로 Mock 주소 생성
3. Mock 주소에 접근하여 모의 데이터 획득

## 제거

### Windows 제거

- "제어판" -> "프로그램 및 기능" 열기
- Apifox를 찾아 "제거" 클릭

### macOS 제거

- Applications 폴더의 Apifox를 휴지통으로 드래그
- 휴지통 비우기

### Linux 제거

```bash
# Debian/Ubuntu
sudo apt-get remove apifox

# Fedora
sudo rpm -e apifox
```
