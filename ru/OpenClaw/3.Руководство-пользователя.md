# Руководство пользователя OpenClaw

Руководство по ежедневному использованию и функциям OpenClaw.

## Основные понятия

### Gateway

Gateway — это сердце OpenClaw. Через него проходит маршрутизация сообщений, запросов к ИИ и управление каналами. Запустив один Gateway, он автоматически обрабатывает подключения со всех ваших платформ.

### Каналы

Каналы — это приложения для общения с ИИ-ассистентом. OpenClaw позволяет подключать несколько каналов одновременно: WhatsApp, Telegram, Discord, iMessage и другие.

### Агенты

Агенты — это независимые экземпляры ИИ-ассистента. Каждый агент имеет собственное рабочее пространство и историю разговоров. Можно запустить несколько агентов для разных целей.

## Ежедневные операции

### Запуск и остановка

```bash
# Запуск (в фоне)
openclaw gateway start

# Остановка
openclaw gateway stop

# Перезапуск
openclaw gateway restart

# Проверить статус
openclaw gateway status

# Запустить в консоли (для отладки)
openclaw gateway --port 18789
```

### Отправка сообщений

```bash
# Отправить личное сообщение
openclaw message send --target +15555550123 --message "Тестовое сообщение"

# Отправить в группу
openclaw message send --target "Название группы" --message "Привет всем"
```

### Управление конфигурацией

OpenClaw использует JSON5-файлы конфигурации в `~/.openclaw/openclaw.json`.

Способы изменения:

1. **Режим мастера**:
   ```bash
   openclaw onboard
   openclaw configure
   ```

2. **Командная строка**:
   ```bash
   openclaw config get agents.defaults.workspace
   openclaw config set agents.defaults.heartbeat.every "2h"
   openclaw config unset tools.web.search.apiKey
   ```

3. **Веб-интерфейс**:
   Откройте http://127.0.0.1:18789/, перейдите на вкладку Config

4. **Прямое редактирование**:
   ```bash
   nano ~/.openclaw/openclaw.json
   ```

### Горячая перезагрузка

Большинство изменений конфигурации применяются автоматически без перезапуска.

## Примеры настроек

### Ограничение доступа

```json5
{
  channels: {
    whatsapp: {
      allowFrom: ["+15555550123"]
    }
  }
}
```

### Настройка групповых чатов

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

### Несколько агентов

```json5
{
  agents: {
    list: [
      { id: "home", default: true, workspace: "~/.openclaw/workspace-home" },
      { id: "work", workspace: "~/.openclaw/workspace-work" }
    ]
  }
}
```

### Запланированные задачи

```json5
{
  cron: {
    enabled: true,
    maxConcurrentRuns: 2
  }
}
```

### Периодическая проверка

```json5
{
  agents: {
    defaults: {
      heartbeat: {
        every: "30m",
        target: "last"
      }
    }
  }
}
```

### Изоляция в песочнице

Для запуска ИИ-операций в изолированных контейнерах:

```json5
{
  agents: {
    defaults: {
      sandbox: {
        mode: "non-main",
        scope: "agent"
      }
    }
  }
}
```

### Webhooks

```json5
{
  hooks: {
    enabled: true,
    token: "ваш-секретный-токен",
    path: "/hooks"
  }
}
```

## Управление каналами

### Добавление каналов

```bash
# Подключить WhatsApp
openclaw channels login whatsapp

# Подключить Telegram
openclaw channels login telegram
```

### Просмотр подключенных каналов

```bash
openclaw channels list
```

## Решение проблем

### Просмотр логов

```bash
# Логи сервиса
openclaw logs

# Логи в реальном времени
openclaw logs -f
```

### Проверка состояния

```bash
openclaw health
```

### Автодиагностика

```bash
openclaw doctor
```

## Веб-консоль

Управлять OpenClaw можно через веб-интерфейс:

- Отправка и получение сообщений
- Изменение настроек
- Просмотр истории разговоров
- Управление мобильными узлами
- Мониторинг состояния сервиса

Откройте: http://127.0.0.1:18789/
