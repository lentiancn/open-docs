# LivePortrait

LivePortrait — эффективная платформа для анимации портретов, разработанная Kwai VGI. Она анимирует статические портреты с использованием движения из управляющих видео.

## Обзор

LivePortrait создает реалистичные видео-портреты путем:
- Извлечения движения из управляющих видео
- Применения движения к исходному портрету
- Поддержки управления сшиванием и перенацеливанием
- Генерации со скоростью 12.8 мс на кадр на RTX 4090

## Быстрый старт

### Установка

```bash
git clone https://github.com/KwaiVGI/LivePortrait.git
cd LivePortrait

# Создание виртуальной среды
python -m venv venv
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt

# Загрузка моделей
bash scripts/download_weights.sh
```

### Генерация видео

```bash
python inference.py \
  --source_image examples/source_image.jpg \
  --driving_video examples/driving_video.mp4 \
  --result_dir ./results
```

## Функции

- **Анимация на основе видео** - Анимация портретов из управляющих видео
- **Сшивание** - Бесшовное соединение головы и тела
- **Перенацеливание глаз** - Управление открытием глаз
- **Перенацеливание губ** - Управление движением губ
- **Быстрый вывод** - 12.8 мс на кадр на RTX 4090
- **Веб-интерфейс** - Простая в использовании демонстрация

- ffmpeg

## Поддерживаемые платформы

- Ubuntu 18.04/20.04/22.04
- Windows 10/11
- macOS 12+

## Требования к входным данным

### Исходное изображение
- Формат: JPG, PNG
- Разрешение: 512x512 или выше

### Управляющее видео
- Формат: MP4, AVI
- Длительность: 1-60 секунд

## Выходные данные

- Формат: MP4 (H.264)
- Разрешение: 512x512
- Частота кадров: 30
