# Flutter 安裝指南

完整的 Flutter 開發環境設置指南。

## 環境要求

### Windows

- **操作系統**：Windows 10 或更高版本（64位）
- **磁盤空間**：最少 2.8 GB
- **工具**：PowerShell 5.0+、Git for Windows

### macOS

- **操作系統**：macOS 10.14 (Mojave) 或更高
- **磁盤空間**：最少 2.8 GB
- **工具**：Command Line Tools、Git

### Linux

- **操作系統**：Ubuntu 18.04 或更高
- **磁盤空間**：最少 2.8 GB
- **依賴**：bash、curl、git、unzip、xz-utils、zip、libglu1-mesa

## 安裝步驟

### Windows 安裝

1. **下載 Flutter SDK**

訪問 Flutter 官網下載最新版本：
```bash
# 或通過 git 克隆
git clone https://github.com/flutter/flutter.git -b stable
```

2. **配置環境變量**

將 Flutter 添加到系統 PATH：
```powershell
# 在 PowerShell 中運行
[Environment]::SetEnvironmentVariable(
    "Path",
    $env:Path + ";C:\src\flutter\bin",
    "User"
)
```

3. **驗證安裝**

```bash
flutter doctor
```

### macOS 安裝

1. **下載 Flutter SDK**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

2. **配置環境變量**

```bash
# 添加到 ~/.bash_profile 或 ~/.zshrc
export PATH="$PATH:/Users/用戶名/flutter/bin"
```

3. **運行 flutter doctor**

```bash
flutter doctor
```

### Linux 安裝

1. **安裝依賴**

```bash
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa
```

2. **下載 Flutter SDK**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

3. **配置 PATH**

```bash
export PATH="$PATH:/home/用戶名/flutter/bin"
```

## Android 開發環境配置

### 安裝 Android Studio

1. 下載 Android Studio：https://developer.android.com/studio
2. 安裝並啟動
3. 配置 SDK：

```bash
flutter config --android-sdk /path/to/android/sdk
```

### 接受許可證

```bash
flutter doctor --android-licenses
```

### 配置模擬器

```bash
# 創建模擬器
flutter emulator --create --name my-emulator

# 啟動模擬器
flutter emulator --launch my-emulator
```

## iOS 開發環境配置（macOS）

### 安裝 Xcode

```bash
# 通過 App Store 安裝
xcode-select --install
```

### 配置 iOS 模擬器

```bash
# 列出可用模擬器
xcrun simctl list devices available

# 啟動模擬器
open -a Simulator
```

## IDE 配置

### VS Code

1. 安裝 Flutter 擴展
2. 安裝 Dart 擴展
3. 重啟 VS Code

### Android Studio

1. 安裝 Flutter 插件
2. 安裝 Dart 插件
3. 重啟 Android Studio

## 驗證安裝

```bash
# 檢查 Flutter 版本
flutter --version

# 運行 flutter doctor
flutter doctor

# 檢查更新
flutter upgrade
```

## 常見問題

### 下載失敗？

使用鏡像：
```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
export PUB_CACHE=/path/to/cache
```

### Android SDK 找不到？

```bash
flutter config --android-sdk /path/to/android-sdk

# 或設置環境變量
export ANDROID_SDK_ROOT=/path/to/android-sdk
```

### 命令找不到？

確保 Flutter 在 PATH 中：
```bash
echo $PATH
# 應該包含 flutter/bin
```

## 卸載

```bash
# 刪除 Flutter 目錄
rm -rf /path/to/flutter

# 從 PATH 中移除
```
