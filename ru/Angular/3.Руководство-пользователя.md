# Руководство пользователя Angular

> Функции

---

## Компоненты

```typescript
@Component({
  selector: 'app-my',
  templateUrl: './my.component.html'
})
export class MyComponent {
  title = 'Hello';
}
```

## Привязка данных

```html
<img [src]="imageUrl">
<button (click)="onClick()">Click</button>
<input [(ngModel)]="name">
```

## Директивы

```html
<div *ngIf="show">Content</div>
<li *ngFor="let item of items">{{item}}</li>
```

## Сервисы

```typescript
@Injectable({ providedIn: 'root' })
export class MyService {
  getData() { return 'Data'; }
}
```

## Роутинг

```typescript
const routes: Routes = [
  { path: '', component: HomeComponent }
];
```
