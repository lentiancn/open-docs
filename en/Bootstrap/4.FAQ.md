# Bootstrap FAQ

> Frequently Asked Questions

---

## Installation Issues

### CDN Fails to Load?

**Solutions**:

1. Check network connection
2. Use alternative CDN:
```html
<link href="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

### Styles Not Working After npm Install?

Make sure to import correctly:

```javascript
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap';
```

## Style Issues

### Component Styles Not Working?

**Common Causes**:

1. Bootstrap CSS not imported
2. JavaScript not loaded correctly (required for some components)
3. CSS specificity being overridden

### Responsive Not Working?

Make sure to add viewport meta tag:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

---

## Component Issues

### Dropdown Not Working?

**Checklist**:

1. Import Bootstrap JS (not just CSS)
2. Use correct HTML structure
3. Check for JavaScript errors

### Modal Won't Open?

Ensure:
1. Bootstrap JS is loaded
2. data-bs-toggle and data-bs-target attributes are correct
3. Modal exists when page loads

## Customization Issues

### How to Customize Theme Colors?

**Method 1: Using Sass**

```scss
$primary: #ff6600;
@import "bootstrap";
```

**Method 2: Using CSS Variables**

```css
:root {
  --bs-primary: #ff6600;
}
```

---

## Compatibility Issues

### IE 11 Not Supported?

Bootstrap 5 no longer supports IE 11. If you need IE 11 support, use Bootstrap 4.

---

## Performance Issues

### How to Reduce File Size?

1. Only import needed components (Sass)
2. Use production versions (.min.css/.min.js)
3. Enable gzip compression

---

## Conflicts with Other Frameworks

### Using with Vue/React?

Bootstrap can be used with any JavaScript framework, but note:

1. Some components depend on jQuery (Bootstrap 5 no longer depends on jQuery)
2. Use framework-specific Bootstrap wrapper libraries (e.g., react-bootstrap)
