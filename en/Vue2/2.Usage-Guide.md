## Basic Concepts

### What is Vue 2?

Vue 2 is a progressive JavaScript framework for building user interfaces. It uses a declarative approach to build UI components.

### Instance

```javascript
new Vue({
  el: '#app',
  data: {
    message: 'Hello Vue!'
  }
})
```

### Options

| Option | Description |
|--------|-------------|
| `el` | Mount point |
| `data` | Reactive data |
| `methods` | Methods |
| `computed` | Computed properties |
| `watch` | Watchers |
| `components` | Child components |

## Templates

### Template Syntax

```html
<div id="app">
  <h1>{{ title }}</h1>
  <p v-html="rawHtml"></p>
  <button v-bind:disabled="isButtonDisabled">Button</button>
  <img v-bind:src="imageUrl">
</div>
```

### Template Directives

| Directive | Description |
|----------|-------------|
| `v-text` | Update element text content |
| `v-html` | Update element HTML |
| `v-show` | Toggle element visibility |
| `v-if` | Conditional rendering |
| `v-else` | Alternative to v-if |
| `v-for` | List rendering |
| `v-on` | Event handling |
| `v-bind` | Attribute binding |
| `v-model` | Two-way binding |

## Data Binding

### Text Interpolation

```html
<p>Message: {{ message }}</p>
```

### Attribute Binding

```html
<a v-bind:href="url">Link</a>
<!-- Shorthand -->
<a :href="url">Link</a>
```

### Class Binding

```html
<div :class="{ active: isActive, 'text-danger': hasError }"></div>
```

### Style Binding

```html
<div :style="{ color: activeColor, fontSize: fontSize + 'px' }"></div>
```

## Methods

### Defining Methods

```javascript
new Vue({
  el: '#app',
  data: {
    name: 'Vue'
  },
  methods: {
    greet() {
      return `Hello, ${this.name}!`
    },
    say(message) {
      alert(message)
    }
  }
})
```

### Usage in Template

```html
<button v-on:click="greet">Greet</button>
<!-- Shorthand -->
<button @click="greet">Greet</button>
```

## Computed Properties

### Basic Computed

```javascript
new Vue({
  data: {
    firstName: 'John',
    lastName: 'Doe'
  },
  computed: {
    fullName() {
      return `${this.firstName} ${this.lastName}`
    }
  }
})
```

### Computed vs Methods

| Feature | Computed | Method |
|---------|----------|--------|
| Cached | Yes | No |
| Called | When dependencies change | Every render |

## Watchers

### Basic Watcher

```javascript
new Vue({
  data: {
    question: ''
  },
  watch: {
    question(newVal, oldVal) {
      console.log(`Changed from ${oldVal} to ${newVal}`)
    }
  }
})
```

### Deep Watcher

```javascript
watch: {
  obj: {
    handler(newVal) {
      console.log('Object changed')
    },
    deep: true
  }
}
```

## Components

### Register Component

```javascript
// Global registration
Vue.component('my-component', {
  template: '<div>A custom component!</div>'
})

// Local registration
new Vue({
  components: {
    'my-component': {
      template: '<div>A custom component!</div>'
    }
  }
})
```

### Component Template

```vue
<template>
  <div class="component">
    <h2>{{ title }}</h2>
  </div>
</template>

<script>
export default {
  name: 'MyComponent',
  props: {
    title: String
  }
}
</script>

<style scoped>
.component {
  padding: 20px;
}
</style>
```

## Props and Events

### Props

```javascript
// Parent component
<child-component :message="parentMessage"></child-component>

// Child component
export default {
  props: {
    message: String,
    count: {
      type: Number,
      default: 0,
      required: true
    }
  }
}
```

### Events

```javascript
// Child
this.$emit('update', value)

// Parent
<child-component @update="handleUpdate"></child-component>
```

## Router

### Installation

```bash
npm install vue-router@3
```

### Basic Setup

```javascript
import Vue from 'vue'
import VueRouter from 'vue-router'
import Home from './views/Home.vue'
import About from './views/About.vue'

Vue.use(VueRouter)

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
]

const router = new VueRouter({
  mode: 'history',
  routes
})

new Vue({
  router
}).$mount('#app')
```

### Router Links

```html
<router-link to="/">Home</router-link>
<router-link to="/about">About</router-link>
<router-view></router-view>
```

## Vuex

### Installation

```bash
npm install vuex@3
```

### Basic Store

```javascript
import Vue from 'vue'
import Vuex from 'vuex'

Vue.use(Vuex)

export default new Vuex.Store({
  state: {
    count: 0
  },
  mutations: {
    increment(state) {
      state.count++
    }
  },
  actions: {
    increment({ commit }) {
      commit('increment')
    }
  },
  getters: {
    doubleCount: state => state.count * 2
  }
})
```

### Using in Component

```javascript
export default {
  computed: {
    count() {
      return this.$store.state.count
    }
  },
  methods: {
    increment() {
      this.$store.commit('increment')
    }
  }
}
```

## Lifecycle Hooks

| Hook | Description |
|------|-------------|
| `beforeCreate` | Before instance creation |
| `created` | Instance created |
| `beforeMount` | Before mounting |
| `mounted` | Mounted to DOM |
| `beforeUpdate` | Before data update |
| `updated` | Data updated |
| `beforeDestroy` | Before destruction |
| `destroyed` | Destroyed |

### Usage

```javascript
new Vue({
  created() {
    console.log('Component created')
  },
  mounted() {
    console.log('Component mounted')
  }
})
```

## Directives

### Custom Directive

```javascript
Vue.directive('focus', {
  inserted(el) {
    el.focus()
  }
})

// Usage
<input v-focus>
```

### Directive Hooks

- `bind`: Once
- `inserted`: Inserted
- `update`: Updated
- `componentUpdated`: Component updated
- `unbind`: Unbound

## Mixins

### Global Mixin

```javascript
Vue.mixin({
  created() {
    console.log('Global mixin')
  }
})
```

### Component Mixin

```javascript
const myMixin = {
  methods: {
    hello() {
      console.log('Hello from mixin')
    }
  }
}

export default {
  mixins: [myMixin]
}
```

## Filters

### Global Filter

```javascript
Vue.filter('uppercase', value => value.toUpperCase())

// Usage
<p>{{ message | uppercase }}</p>
```

### Component Filter

```javascript
filters: {
  uppercase(value) {
    return value.toUpperCase()
  }
}
```
