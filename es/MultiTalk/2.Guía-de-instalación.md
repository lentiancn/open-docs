# Guia de instalacion

El proceso de instalacion de MultiTalk requiere configurar el entorno de Python y descargar los pesos del modelo. A continuacion se presentan instrucciones detalladas de instalacion.

## Requisitos del entorno

### Requisitos de hardware

MultiTalk es un modelo grande de generacion de video con altos requisitos de hardware. Se recomienda usar GPUs NVIDIA para ejecutarlo. Los requisitos de VRAM varian segun la resolucion de salida. Generar videos de 480P requiere al menos 16GB de VRAM, mientras que generar videos de 720P requiere configuracion de multiples GPUs. El proyecto tambien soporta ejecucion en entornos de VRAM bajo, permitiendo la generacion de videos 480P en una sola RTX 4090 a traves de optimizaciones especificas.

### Requisitos de software

Se admite el sistema operativo Linux. Se requiere Python version 3.10 o superior. Tambien se requiere soporte para CUDA 12.1 o superior. El proyecto depende de marcos de aprendizaje profundo como PyTorch 2.4.1 y xformers.

## Creacion del entorno virtual

Primero, cree un nuevo entorno virtual de Python para aislar las dependencias del proyecto. Abra la terminal y ejecute los siguientes comandos:

```bash
conda create -n multitalk python=3.10
conda activate multitalk
```

## Instalacion de PyTorch y xformers

Instale PyTorch y paquetes relacionados en el entorno activado. Ejecute el siguiente comando para instalar PyTorch 2.4.1:

```bash
pip install torch==2.4.1 torchvision==0.19.1 torchaudio==2.4.1 --index-url https://download.pytorch.org/whl/cu121
```

A continuacion, instale xformers:

```bash
pip install -U xformers==0.0.28 --index-url https://download.pytorch.org/whl/cu121
```

## Instalacion de dependencias del proyecto

Instale otros paquetes de dependencia necesarios:

```bash
pip install misaki[en]
pip install ninja
pip install psutil
pip install packaging
pip install flash_attn==2.7.4.post1
```

Luego, clone el repositorio del proyecto e instale las dependencias:

```bash
git clone https://github.com/MeiGen-AI/MultiTalk.git
cd MultiTalk
pip install -r requirements.txt
```

## Instalacion de bibliotecas de procesamiento de audio

El proyecto requiere bibliotecas relacionadas con el procesamiento de audio. Ejecute los siguientes comandos para instalar:

```bash
conda install -c conda-forge librosa
conda install -c conda-forge ffmpeg
```

Para sistemas CentOS o RHEL, use:

```bash
sudo yum install ffmpeg ffmpeg-devel
```

## Descarga de pesos del modelo

MultiTalk requiere descargar varios archivos de modelo para funcionar correctamente. Los principales modelos incluyen:

Wan2.1-I2V-14B-480P es el modelo base de difusion, descargable desde Hugging Face. chinese-wav2vec2-base es el codificador de audio para procesar audio de entrada. Kokoro-82M son los pesos de TTS para funcionalidad de texto a voz. MeiGen-MultiTalk son los pesos de condicion de audio proporcionados por el proyecto.

Descargue modelos usando huggingface-cli:

```bash
huggingface-cli download Wan-AI/Wan2.1-I2V-14B-480P --local-dir ./weights/Wan2.1-I2V-14B-480P
huggingface-cli download TencentGameMate/chinese-wav2vec2-base --local-dir ./weights/chinese-wav2vec2-base
huggingface-cli download hexgrad/Kokoro-82M --local-dir ./weights/Kokoro-82M
huggingface-cli download MeiGen-AI/MeiGen-MultiTalk --local-dir ./weights/MeiGen-MultiTalk
```

## Configuracion de enlaces de modelos

Despues de descargar los modelos, necesita configurar los enlaces. Hay dos opciones disponibles.

La opcion uno es usar enlaces simbolicos. Ingrese al directorio weights y ejecute:

```bash
cd weights
mv Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json_old
ln -s /path/to/weights/MeiGen-MultiTalk/diffusion_pytorch_model.safetensors.index.json Wan2.1-I2V-14B-480P/
ln -s /path/to/weights/MeiGen-MultiTalk/multitalk.safetensors Wan2.1-I2V-14B-480P/
```

La opcion dos es copiar archivos directamente:

```bash
cd weights
mv Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json_old
cp MeiGen-MultiTalk/diffusion_pytorch_model.safetensors_index.json Wan2.1-I2V-14B-480P/
cp MeiGen-MultiTalk/multitalk.safetensors Wan2.1-I2V-14B-480P/
```

## Verificacion de la instalacion

Despues de completar la instalacion, puede ejecutar scripts de ejemplo para verificar la instalacion. Los archivos de ejemplo se encuentran en el directorio examples.
