# OpenCode 安装指南

OpenCode 支持多种安装方式。

## Linux/macOS 安装

### 推荐：安装脚本

```bash
curl -fsSL https://opencode.ai/install | bash
```

### Node.js

```bash
npm install -g opencode-ai
```

### Bun

```bash
bun install -g opencode-ai
```

```bash
pn### pnpm

pm install -g opencode-ai
```

### Yarn

```bash
yarn global add opencode-ai
```

### Homebrew (macOS/Linux)

```bash
brew install anomalyco/tap/opencode
```

### Arch Linux

```bash
# 稳定版
sudo pacman -S opencode

# 最新版（AUR）
paru -S opencode-bin
```

## Windows 安装

### Chocolatey

```bash
choco install opencode
```

### Scoop

```bash
scoop install opencode
```

### NPM

```bash
npm install -g opencode-ai
```

### Mise

```bash
mise use -g github:anomalyco/opencode
```

### Docker

```bash
docker run -it --rm ghcr.io/anomalyco/opencode
```

### 手动下载

也可以从 [GitHub Releases](https://github.com/anomalyco/opencode/releases) 下载二进制文件。

## 验证安装

```bash
opencode --version
```

## 配置

### 连接 LLM 提供商

运行以下命令连接 OpenCode：

```bash
opencode
/connect
```

选择 `opencode`，然后访问 [opencode.ai/auth](https://opencode.ai/auth) 登录并获取 API 密钥。

### 其他提供商

也可以选择其他 LLM 提供商，详细配置请参阅[官方文档](https://opencode.ai/docs/providers#directory)。

## 初始化项目

```bash
cd /path/to/project
opencode
/init
```

这会让 OpenCode 分析项目并在根目录创建 AGENTS.md 文件，帮助它理解项目结构和编码规范。
