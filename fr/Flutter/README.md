# Flutter

Flutter - Framework d'Interface Utilisateur Multiplateforme

## Présentation

Flutter est un kit de développement logiciel open source créé par Google, utilisé pour créer de belles applications mobiles, Web et de bureau compilées de manière native à partir d'une seule base de code.

## Caractéristiques

- **Multiplateforme** : Une base de code pour iOS, Android, Web, Windows, macOS, Linux
- **Rechargement à Chaud** : Itération de développement rapide, préserve l'état de l'application
- **Composants Riches** : Bibliothèques de composants Material Design et Cupertino
- **Performances Natives** : Compile en code ARM natif
- **UI Déclarative** : Paradigme moderne de développement d'interface utilisateur
- **Open Source** : Licence BSD

## Démarrage Rapide

### Installation

```bash
# Cloner Flutter
git clone https://github.com/flutter/flutter.git -b stable

# Ajouter à PATH
export PATH="$PATH:$HOME/flutter/bin"

# Vérifier l'installation
flutter doctor
```

### Créer un Projet

```bash
flutter create my_app
cd my_app
flutter run
```
