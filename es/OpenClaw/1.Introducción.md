# Introducción a OpenClaw

## ¿Qué es OpenClaw?

OpenClaw es una puerta de enlace autohospedada que conecta aplicaciones de chat (WhatsApp, Telegram, Discord, iMessage) con asistentes de codificación AI.

## Características

- Multi-canal (WhatsApp, Telegram, Discord, iMessage)
- Panel web
- Enrutamiento multi-agente
- Nodos móviles
- Controles de seguridad

## Requisitos

- Node.js 22+
- Clave API (Anthropic recomendado)

## Inicio Rápido

```bash
npm install -g openclaw@latest
openclaw onboard --install-daemon
openclaw gateway --port 18789
```

Visitar http://127.0.0.1:18789/
