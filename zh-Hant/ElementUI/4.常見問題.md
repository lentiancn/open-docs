# Element UI 常見問題

常見問題解答。

## 安裝問題

### Q: 安裝失敗？

**解決方案：**
1. 檢查 Node.js 版本
```bash
node --version  # 需要 12.0.0+
```
2. 清理 npm 快取
```bash
npm cache clean --force
```
3. 使用淘寶鏡像
```bash
npm config set registry https://registry.npmmirror.com
```
4. 刪除 node_modules 重新安裝

### Q: 找不到模組？

**解決方案：**
確認已安裝 element-ui

### Q: Vue 版本不相容？

**解決方案：**
Element UI 2.x 需要 Vue 2.6.0+

## 使用問題

### Q: 元件不顯示？

**解決方案：**
確認已正確引入元件：
```javascript
import { Button } from 'element-ui'
Vue.use(Button)
```

### Q: 樣式不生效？

**解決方案：**
確認已引入樣式檔案：
```javascript
import 'element-ui/lib/theme-chalk/index.css'
```

### Q: 表格不顯示資料？

**解決方案：**
檢查資料格式和列定義

### Q: 表單驗證不生效？

**解決方案：**
引入 Form 和 FormItem，設置 prop 屬性
