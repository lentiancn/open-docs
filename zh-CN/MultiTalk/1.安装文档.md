# MultiTalk 安装指南

MultiTalk 是一款多说话人音频生成工具，可创建自然流畅的多人语音内容。

## 版本信息

| 版本 | 状态 | 说明 |
|---------|--------|-------------|
| MultiTalk v1.0 | ✅ 当前版本 | 初始版本发布 |

## 系统要求

### 硬件要求

| 要求 | 最低配置 | 推荐配置 |
|-------------|---------|-------------|
| GPU | NVIDIA 4GB 显存 | NVIDIA 8GB+ 显存 |
| RAM | 8GB | 16GB+ |
| 存储空间 | 5GB | 10GB+ SSD |

### 软件要求

| 软件 | 版本 |
|----------|---------|
| Python | 3.8+ |
| CUDA | 11.0+ |

## 安装方法

### 方法一：从源码安装

#### 1. 克隆仓库

```bash
git clone https://github.com/MultiTalk/MultiTalk.git
cd MultiTalk
```

#### 2. 创建虚拟环境

```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
# 或
venv\Scripts\activate  # Windows
```

#### 3. 安装依赖

```bash
pip install -r requirements.txt
```

#### 4. 下载模型

```bash
bash scripts/download_models.sh
```

### 方法二：使用 Docker

```bash
docker pull multitalk/multitalk:latest
docker run --gpus all -v /path/to/data:/data multitalk/multitalk:latest
```

## 配置

### 环境变量

```bash
export CUDA_VISIBLE_DEVICES=0
export PYTHONPATH=/path/to/MultiTalk:$PYTHONPATH
```

### 模型路径

编辑 `config.yaml`:

```yaml
model:
  checkpoint_path: "checkpoints/multitalk.pth"
```

## 快速开始

### 生成多人语音

```bash
python inference.py \
  --text "你好，这是一个多说话人演示。" \
  --speakers speaker1,speaker2 \
  --output output.wav
```

### 参数说明

```bash
--text         # 输入文本
--speakers     # 说话人ID（用逗号分隔）
--output       # 输出文件路径
--speed        # 语音速度（默认：1.0）
```
