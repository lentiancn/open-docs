# DragonTalker Guía de Instalación

DragonTalker es un proyecto de IA para generar videos realistas de cabeza parlante desde una imagen y audio.

## Requisitos del Sistema

### Requisitos de Hardware

| Componente | Mínimo | Recomendado |
|------------|--------|-------------|
| GPU | NVIDIA 6GB VRAM | NVIDIA 16GB VRAM |
| RAM | 8GB | 32GB |
| Almacenamiento | 20GB | 50GB |
| CUDA | 11.7+ | 11.8+ |

### Requisitos de Software

| Software | Versión | Notas |
|----------|---------|-------|
| Python | 3.8 - 3.10 | Recomendado 3.9 |
| CUDA | 11.7+ | Aceleración GPU requerida |
| ffmpeg | Latest | Procesamiento de video requerido |

## Instalación en Linux/Ubuntu

### Paso 1: Instalar dependencias del sistema

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install python3.9 python3-pip python3-venv git wget curl
sudo apt install libgl1-mesa-glx libglib2.0-0 libsm6 libxext6 libxrender-dev libgomp1
sudo apt install ffmpeg
```

### Paso 2: Clonar proyecto

```bash
git clone https://github.com/your-repo/DragonTalker.git
cd DragonTalker
```

### Paso 3: Crear entorno virtual

```bash
python3.9 -m venv venv
source venv/bin/activate
```

### Paso 4: Instalar PyTorch

```bash
pip install torch==2.0.1 torchvision==0.15.2 --index-url https://download.pytorch.org/whl/cu118
```

### Paso 5: Instalar dependencias

```bash
pip install -r requirements.txt
```

## Instalación en Windows

### Paso 1: Instalar Python

1. Descargar Python 3.9 desde python.org
2. Añadir Python al PATH
3. Verificar: `python --version`

### Paso 2: Instalar CUDA

1. Descargar CUDA 11.8 desde NVIDIA
2. Instalar con configuración predeterminada

### Paso 3: Configurar

```powershell
git clone https://github.com/your-repo/DragonTalker.git
cd DragonTalker
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Verificar Instalación

```bash
python inference.py --help
```

## Problemas Comunes

### Memoria CUDA insuficiente

**Soluciones:**
- Reducir tamaño de lote: `--batch_size 1`
- Usar menor resolución: `--size 256`
