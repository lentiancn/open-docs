# Element UI 常见问题

常见问题解答。

## 安装问题

### Q: 安装失败？

**解决方案：**
1. 检查 Node.js 版本
```bash
node --version  # 需要 12.0.0+
```
2. 清理 npm 缓存
```bash
npm cache clean --force
```
3. 使用淘宝镜像
```bash
npm config set registry https://registry.npmmirror.com
```
4. 删除 node_modules 重新安装
```bash
rm -rf node_modules
npm install
```

### Q: 找不到模块？

**解决方案：**
1. 确认已安装 element-ui
```bash
npm list element-ui
```
2. 检查 package.json 是否包含依赖

### Q: Vue 版本不兼容？

**解决方案：**
Element UI 2.x 需要 Vue 2.6.0+：
```bash
npm install vue@2
```

## 使用问题

### Q: 组件不显示？

**解决方案：**
1. 确认已正确引入组件
```javascript
import { Button } from 'element-ui'
Vue.use(Button)
```
2. 检查控制台错误
3. 确认组件名称正确

### Q: 样式不生效？

**解决方案：**
1. 确认已引入样式文件
```javascript
import 'element-ui/lib/theme-chalk/index.css'
```
2. 检查是否有样式覆盖
3. 使用开发者工具检查元素样式

### Q: 表格不显示数据？

**解决方案：**
1. 检查 data 数据格式
2. 确认列定义正确
```html
<el-table-column prop="name" label="姓名"></el-table-column>
```
3. 检查数据是否响应式更新

### Q: 表单验证不生效？

**解决方案：**
1. 确认引入 Form 和 FormItem
2. 设置 prop 属性
```html
<el-form-item label="名称" prop="name">
  <el-input v-model="form.name"></el-input>
</el-form-item>
```
3. 在 data 中定义 rules

### Q: 下拉框选项不显示？

**解决方案：**
1. 检查 options 数据
2. 确认 value 和 label 正确映射
```html
<el-option
  :value="item.value"
  :label="item.label">
</el-option>
```

### Q: 对话框关闭后数据未重置？

**解决方案：**
1. 使用 v-if 控制对话框
```html
<el-dialog :visible.sync="dialogVisible" v-if="dialogVisible">
```
2. 或在关闭时手动重置表单
```javascript
this.$refs.form.resetFields()
```

## 样式问题

### Q: 样式冲突？

**解决方案：**
1. 使用更具体的选择器
2. 检查 z-index 层级
3. 使用 !important 覆盖（不推荐）

### Q: 图标不显示？

**解决方案：**
1. 确认已安装图标库
```bash
npm install element-ui-icons
```
2. 正确引入
```javascript
import 'element-ui-icons'
```

### Q: 主题色不正确？

**解决方案：**
1. 检查是否有全局样式覆盖
2. 确认 SCSS 变量引入顺序正确
3. 清除浏览器缓存

## 国际化问题

### Q: 语言切换不生效？

**解决方案：**
1. 正确配置 i18n
```javascript
Vue.use(ElementUI, {
  i18n: (key, value) => i18n.t(key, value)
})
```
2. 确认语言包已引入

### Q: 部分文本未翻译？

**解决方案：**
1. 检查完整语言包
2. 使用 ElementUI 的 locales 配置

## 性能问题

### Q: 页面加载慢？

**解决方案：**
1. 使用按需引入
2. 使用 CDN 引入
3. 开启 Gzip 压缩

### Q: 大数据表格卡顿？

**解决方案：**
1. 使用虚拟滚动
2. 分页加载数据
3. 关闭不必要的特性

## 打包问题

### Q: 打包后样式丢失？

**解决方案：**
1. 检查 vue.config.js 配置
2. 确认样式引入位置正确

### Q: 生产环境组件不显示？

**解决方案：**
1. 检查 Vue.use() 调用
2. 确认按需引入完整

## 升级问题

### Q: 如何从 1.x 升级到 2.x？

**解决方案：**
1. 阅读官方迁移指南
2. 逐步替换不兼容 API
3. 运行测试确保功能正常

### Q: Vue 3 项目如何选择？

**解决方案：**
Element UI 对应 Vue 2，Vue 3 项目请使用 Element Plus：
```bash
npm install element-plus
```
