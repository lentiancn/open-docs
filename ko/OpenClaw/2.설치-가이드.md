# OpenClaw 설치 가이드

## 요구사항

- Node.js 22+
- macOS / Linux / Windows
- Docker (선택,隔离용)

## 설치

### macOS / Linux

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

### Windows (PowerShell)

```powershell
iwr -useb https://openclaw.ai/install.ps1 | iex
```

### npm

```bash
npm install -g openclaw@latest
```

## 설치 후

```bash
openclaw onboard --install-daemon
openclaw gateway status
openclaw dashboard
```

열기: http://127.0.0.1:18789/

## 확인

```bash
openclaw --version
openclaw doctor
```

## 자주ある 문제

- `node --version` — Node.js 버전 확인
- `openclaw logs` — 로그 보기
- `openclaw doctor` — 진단
