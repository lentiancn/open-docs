# Bootstrap 常見問題

## Q: Bootstrap 5 還需要 jQuery 嗎？

**不需要**。Bootstrap 5 完全使用原生 JavaScript。

---

## Q: CDN 載入失敗？

使用備用 CDN：

```html
<link href="https://cdnjs.cloudflare.com/ajax/libs/twitter-bootstrap/5.3.3/css/bootstrap.min.css">
```

---

## Q: 柵格不生效？

確保 row 在 container 內，col 在 row 內。

---

## Q: 如何自定義顏色？

```css
:root {
    --bs-primary: #your-color;
}
```
