# HeyGen

AI 디지털 아바타 영상 생성 플랫폼

## 소개

HeyGen은 고급 인공지능 기술을 사용하여 현실적인 디지털 아바타 영상을 만드는 혁신적인 AI 영상 생성 플랫폼입니다. 사용자는 텍스트를 입력하거나 이미지 또는 오디오를 업로드하여 자연스러운 표정과 립싱크를 갖춘 AI 내레이션 영상 콘텐츠를 빠르게 생성할 수 있습니다.

## 주요 기능

- **텍스트 → 영상**: 텍스트 스크립트를 전문 영상으로 변환
- **사진 → 영상**: 정적 이미지를 동적 영상으로 변환
- **AI 디지털 아바타**: 현실적인 가상 캐릭터 생성
- **AI 번역기**: 175개 이상 언어 지원
- **AI Studio**: 통합된 영상 제작 작업 공간
- **멀티플랫폼 내보내기**: 720p, 1080p, 4K

## 빠른 시작

1. HeyGen 공식 웹사이트를 방문하고 가입하세요
2. 디지털 아바타를 선택하거나 사용자 정의 아바타를 만드세요
3. 영상 스크립트를 입력하거나 오디오를 업로드하세요
4. 음성과 배경을 선택하세요
5. 영상을 생성하고 다운로드하세요
