# Nacos 概要

Nacos（Dynamic Naming and Configuration Service）は、Alibabaが開発したサービス発見と動的構成管理のためのプラットフォームです。

## 主な機能

- サービスディスカバリー
- 構成管理
- 動的DNS
- ヘルスチェック

## インストール

```bash
# スタンドアロンモード
./startup.sh -m standalone

# Docker
docker run --name nacos -d -p 8848:8848 nacos/nacos-server:v2.3.0
```
