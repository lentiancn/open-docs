# HeyGen 安裝指南

HeyGen 是一個 AI 影片生成平台，允許用戶建立逼真的 AI 頭像並從文字或音訊輸入生成說話頭像影片。

## 平台概述

HeyGen 主要是一個基於雲的 SaaS 平台。與開源專案不同，HeyGen 不需要本地安裝 - 用戶透過網頁介面或 API 訪問。

### 訪問方式

| 方式 | 描述 | 要求 |
|------|------|------|
| 網頁介面 | 基於瀏覽器的影片創作 | 帳戶 + 網路 |
| API | 程式化訪問 | API 金鑰 |
| SDK | 本地開發 | 開發者帳戶 |

## 版本概述

HeyGen 是雲服務，非本地安裝：

| 元件 | 版本 | 說明 |
|------|------|------|
| HeyGen 網頁 | 最新 | 始終保持最新 |
| HeyGen API | v1/v2 | REST API |
| HeyGen SDK | 最新 | 開發者使用 |

**注意：** HeyGen 是商業產品，某些功能需要付費訂閱。

## 系統需求

### 網頁訪問需求

| 要求 | 最低 | 推薦 |
|------|------|------|
| 瀏覽器 | Chrome 90+, Firefox 88+, Safari 14+, Edge 90+ | 最新版 Chrome |
| 網路 | 10 Mbps | 25 Mbps+ |
| 記憶體 | 4GB | 8GB+ |

### API/SDK 開發需求

| 要求 | 版本 |
|------|------|
| Node.js | 16+ |
| Python | 3.8+ |
| API 金鑰 | 來自 HeyGen 儀表板 |

## 訪問 HeyGen

### 步驟 1：建立帳戶

1. 訪問 https://www.heygen.com
2. 點擊「註冊」或「開始使用」
3. 驗證電子郵件地址

### 步驟 2：選擇方案

| 方案 | 功能 | 價格 |
|------|------|------|
| 免費 | 有限分鐘數，有浮水印 | 免費 |
| 專業版 | 更多分鐘數，無浮水印 | $29/月 |
| 企業版 | 自定義限額，API 訪問 | 聯繫銷售 |

### 步驟 3：取得 API 金鑰（開發者）

1. 登入 HeyGen 儀表板
2. 導航到設定 → API
3. 產生 API 金鑰

## API 安裝（開發者）

### Python SDK

```bash
# 安裝 HeyGen SDK
pip install heygen-python

# 或從原始碼安裝
pip install git+https://github.com/heygen-ai/heygen-python.git
```

### Node.js SDK

```bash
# 安裝 HeyGen SDK
npm install heygen-node

# 或從原始碼安裝
npm install github:heygen-ai/heygen-node
```

### 驗證安裝

```python
# Python
import heygen
print(heygen.__version__)
```

```javascript
// Node.js
const heygen = require('heygen-node');
console.log(heygen.version);
```

## API 配置

### Python 範例

```python
from heygen import HeyGen

# 使用 API 金鑰初始化
heygen = HeyGen(api_key="your_api_key")

# 檢查帳戶資訊
account = heygen.get_account_info()
print(account)
```

### Node.js 範例

```javascript
const { HeyGen } = require('heygen-node');

const heygen = new HeyGen({ apiKey: 'your_api_key' });

// 檢查帳戶資訊
const account = await heygen.getAccountInfo();
console.log(account);
```

## 環境變數

### 設定 API 金鑰

```bash
# Linux/macOS
export HEYGEN_API_KEY="your_api_key"

# Windows (PowerShell)
$env:HEYGEN_API_KEY="your_api_key"

# 在 .env 檔案中
HEYGEN_API_KEY=your_api_key
```

## 常見問題

### 認證錯誤

**問題：**「API 金鑰無效」

**解決方案：**
1. 檢查 API 金鑰是否正確
2. 確保 API 金鑰處於活動狀態
3. 驗證帳戶是否有 API 訪問權限

### 速率限制

**問題：**「請求過多」

**解決方案：**
1. 檢查 API 使用限制
2. 升級方案以獲得更高限制
3. 實施請求佇列

### 影片生成失敗

**問題：**「生成失敗」

**解決方案：**
1. 檢查輸入參數
2. 驗證積分餘額
3. 稍後重試

## 安全最佳實踐

1. **永不暴露 API 金鑰** - 使用環境變數
2. **定期輪換金鑰** - 定期產生新金鑰
3. **使用最小權限** - 只請求需要的範圍
4. **監控使用情況** - 檢查儀表板異常
