# Go Installationsanleitung

> Vollständige Umgebungskonfiguration

---

## Anforderungen

| Umgebung | Anforderung |
|----------|-------------|
| Betriebssystem | Linux, macOS, Windows |
| Architektur | amd64, arm64, 386 |

---

## Installation

### Windows

```powershell
choco install golang
```

### macOS

```bash
brew install go
```

### Linux

```bash
sudo apt install golang-go
```

---

## Umgebung

### Proxy (China)

```bash
go env -w GOPROXY=https://goproxy.cn,direct
```

### Überprüfen

```bash
go version
```
