# Введение в Kubernetes

## Что такое Kubernetes?

Kubernetes (K8s) — это открытое программное обеспечение для автоматизации развёртывания, масштабирования и управления контейнеризированными приложениями. Kubernetes группирует контейнеры, составляющие приложение, в логические единицы для более простого управления и обнаружения. При создании Kubernetes использован 15-летний опыт эксплуатации производственных нагрузок Google, который был совмещён с лучшими идеями и практиками сообщества.

## Основные функции

### Автоматические развёртывания и откаты
Kubernetes последовательно применяет изменения к приложению или его конфигурации, одновременно отслеживая состояние приложения, чтобы гарантировать, что все экземпляры не будут уничтожены одновременно. Если что-то пойдёт не так, Kubernetes откатит изменения за вас.

### Обнаружение сервисов и балансировка нагрузки
Вам не нужно изменять приложение, чтобы использовать незнакомый механизм обнаружения сервисов. Kubernetes назначает подам собственные IP-адреса и одно DNS-имя для набора подов, а также может балансировать нагрузку между ними.

### Оркестрация хранилища
Автоматически монтируйте выбранную систему хранения, будь то локальное хранилище, облачный провайдер или сетевая система хранения, такая как NFS или iSCSI.

### Управление Secret и конфигурацией
Развёртывайте и обновляйте Secret и конфигурацию приложения без пересборки образа и без раскрытия конфиденциальной информации в конфигурации.

### Автоматическая упаковка
Автоматически размещайте контейнеры на основе их требований к ресурсам и других ограничений, не жертвуя доступностью. Объединяйте критические рабочие нагрузки и нагрузки с низким приоритетом для повышения использования ресурсов.

### Пакетное выполнение
Помимо сервисов, Kubernetes может управлять пакетными рабочими нагрузками и нагрузками CI, заменяя вышедшие из строя контейнеры.

### Самоисцеление
Kubernetes перезапускает вышедшие из строя контейнеры, при необходимости заменяет целые поды, повторно подключает хранилище в ответ на более масштабные сбои и может интегрироваться с автомасштабированием узлов для самоисцеления на уровне узла.

### Горизонтальное масштабирование
Масштабируйте приложение вверх и вниз с помощью простой команды, через пользовательский интерфейс или автоматически на основе использования процессора.

### Расширяемость
Добавляйте функции в кластер Kubernetes без изменения исходного кода.

## Архитектура

### Плоскость управления
Плоскость управления отвечает за поддержание общего состояния кластера:
- API Server: точка входа для всех операций RESTful API
- etcd: распределённое хранилище ключ-значение для данных кластера
- Scheduler: назначает поды соответствующим узлам
- Controller Manager: запускает различные контроллеры для поддержания состояния кластера

### Рабочие узлы
Рабочие узлы — это места, где фактически выполняются контейнеры. Каждый узел содержит:
- Kubelet: взаимодействует с плоскостью управления, управляет контейнерами на узле
- Container Runtime: отвечает за выполнение контейнеров (Docker, containerd и т. д.)
- Kube Proxy: обрабатывает сетевое проксирование и балансировку нагрузки

## Варианты использования

### Облачные вычисления
Kubernetes — это проект с открытым исходным кодом, который даёт вам полную свободу воспользоваться преимуществами локальной, гибридной или публичной облачной инфраструктуры.

### Архитектура микросервисов
Kubernetes — идеальная платформа для запуска микросервисов, эффективно управляя большим количеством независимых экземпляров сервисов.

### Машинное обучение
Kubernetes — отличная платформа для машинного обучения, поскольку она предоставляет все необходимые функции планирования.

### Непрерывное развёртывание
Поддерживает быструю итерацию, позволяя несколько раз в день развёртывать обновления приложений.

## Преимущества

### Масштабируемость любого уровня
Разработанный на тех же принципах, которые позволяют Google запускать миллиарды контейнеров в неделю, Kubernetes может масштабироваться без увеличения вашей технической команды.

### Бесконечная гибкость
Гибкость Kubernetes растёт вместе с вами, обеспечивая бесперебойную и простую доставку приложений независимо от сложности ваших потребностей.

### Сильная поддержка сообщества
Kubernetes имеет большую и быстрорастущую экосистему с широким спектром сервисов, поддержки и инструментов.
