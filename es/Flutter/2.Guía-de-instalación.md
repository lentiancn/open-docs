# Guía de Instalación de Flutter

## Requisitos

Antes de instalar Flutter, asegúrate de que tu entorno de desarrollo cumpla los siguientes requisitos. En cuanto a sistemas operativos, Flutter soporta Windows, macOS y Linux. Windows requiere Windows 10 o superior, macOS requiere macOS 10.14 o superior, y Linux requiere un entorno de escritorio que soporte GNOME o KDE.

En cuanto al espacio en disco, el SDK de Flutter ocupa aproximadamente 1GB, pero considerando los proyectos de desarrollo y la caché de compilación, se recomienda reservar al menos 10GB de espacio disponible. En cuanto a memoria, 8GB o más proporciona una experiencia de desarrollo más fluida.

Para desarrollo móvil, el desarrollo de iOS requiere macOS y Xcode, el desarrollo de Android requiere Android Studio y Android SDK. El desarrollo web y de escritorio no tiene requisitos especiales, solo necesitas instalar el SDK de Flutter.

## Pasos de Instalación

### Instalación en Windows

Hay dos formas principales de instalar Flutter en Windows. La primera es descargar el archivo precompilado del sitio web de Flutter, extraerlo en una ubicación apropiada y luego agregar el directorio flutter/bin a la variable de entorno del sistema PATH.

Método de descarga: Visita el sitio web de Flutter en flutter.dev y descarga la última versión estable. Extrae en C:\src\flutter u otra ubicación adecuada. Abre Propiedades del sistema → Avanzado → Variables de entorno, y en las variables de usuario Path, agrega la ruta al directorio bin de Flutter.

Después de completar esto, abre el símbolo del sistema y ejecuta el comando flutter doctor para verificar si la instalación fue exitosa. Flutter doctor verificará el estado del SDK de Flutter y dará sugerencias de configuración.

### Instalación en macOS

En macOS, se recomienda la instalación basada en Git. Primero asegúrate de que Xcode esté instalado, luego abre Terminal y ejecuta el siguiente comando para clonar Flutter:

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

Agrega Flutter al PATH:

```bash
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.bash_profile
```

Para usuarios de zsh:

```bash
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.zshrc
```

Luego ejecuta source ~/.bash_profile o source ~/.zshrc para actualizar las variables de entorno.

### Instalación en Linux

La instalación en Linux es similar a macOS. Primero instala las dependencias:

```bash
sudo apt-get update
sudo apt-get install curl git unzip xz-utils zip libglu1-mesa clang cmake ninja-build pkg-config libgtk-3-dev
```

Luego clona el repositorio de Flutter y configura el PATH, igual que los pasos de macOS.

## Verificar Instalación

Después de la instalación, ejecuta los siguientes comandos para verificar:

```bash
flutter --version
flutter doctor
```

Flutter doctor mostrará el estado de la instalación y cualquier dependencia faltante. Completa las configuraciones necesarias según las indicaciones, como instalar Android Studio y configurar Android SDK.

## Configuración del IDE

### VS Code

VS Code es uno de los editores recomendados para Flutter. Después de instalar VS Code, abre el mercado de extensiones, busca las extensiones de Flutter y Dart e instálalas. Después de la instalación, reinicia VS Code para usar las funciones relacionadas con Flutter.

### Android Studio

Después de instalar Android Studio, abre la página de Plugins, busca el complemento de Flutter e instálalo. Después de la instalación, configura la ruta del SDK de Flutter: Abre Preferences → Language & Frameworks → Flutter y establece la ruta del SDK de Flutter.

## Inicio Rápido

Crea tu primer proyecto de Flutter:

```bash
flutter create my_first_app
cd my_first_app
flutter run
```

O usa la paleta de comandos (Ctrl+Shift+P) en VS Code e ingresa Flutter: New Project para crear un nuevo proyecto.

Ejecuta flutter devices para ver los dispositivos conectados, y flutter run -d device_id para ejecutar en un dispositivo específico.
