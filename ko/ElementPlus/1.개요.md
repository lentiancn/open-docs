# Element Plus 개요

Vue 3을 위한 데스크톱 UI 컴포넌트 라이브러리.

## Element Plus란?

Element Plus는 Ele.me 팀에서 개발한 Vue 3 컴포넌트 라이브러리로, 최신 웹 애플리케이션을 위한 완전한 데스크톱 UI 컴포넌트 세트를 제공합니다. Element UI의 Vue 3 버전으로서, Element Plus는 이전 버전의優れた 특성을 상속하면서 Vue 3에 완전히 최적화되었습니다.

### 주요 기능

- **Vue 3 기반**: Vue 3 Composition API로 구축
- **TypeScript 지원**: 네이티브 TypeScript 타입 정의
- **풍부한 컴포넌트**: 70개 이상의 고품질 컴포넌트
- **트리 쉐이킹**: 번들 크기 최적화를 위한 필요시 로드
- **테마 커스터마이징**: CSS 변수를 통한 커스텀 테마 지원
- **국제화**: 20개 이상의 내장 언어 팩
- **접근성**: 완전한 ARIA 지원
- **반응형 디자인**: 다양한 화면 크기 지원
