# Element UI FAQ

## 설치

### Q: 설치 실패?

**해결책:**
1. Node.js 버전 확인
2. npm 캐시 비우기
```bash
npm cache clean --force
```

### Q: Vue 버전 호환 안됨?

**해결책:**
Element UI 2.x는 Vue 2.6.0+ 필요

## 사용

### Q: 컴포넌트 표시 안됨?

**해결책:**
가져오기 확인:
```javascript
import { Button } from 'element-ui'
Vue.use(Button)
```

### Q: 스타일 적용 안됨?

**해결책:**
스타일 파일 가져오기:
```javascript
import 'element-ui/lib/theme-chalk/index.css'
```
