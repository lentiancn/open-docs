# Element UI

Vue.js 2.0 Desktop-UI-Komponentenbibliothek.

## Übersicht

Element UI ist eine von Ele.me entwickelte Komponentenbibliothek.

### Funktionen

- **60+ Komponenten**
- **Schönes Design**
- **TypeScript Support**

### Statistik

- GitHub Sterne: 54,000+

## Schnellstart

```bash
npm install element-ui --save
```

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
Vue.use(ElementUI)
```
