# MultiTalk

MultiTalk — это инструмент для генерации аудио с несколькими дикторами.

## Введение

MultiTalk создаёт естественно звучащий многоголосый аудиоконтент с использованием технологий глубокого обучения.

## Функции

- **Несколько дикторов**: Поддержка нескольких дикторов
- **Высокое качество**: Естественно звучащий вывод
- **Многоязычность**: Поддержка нескольких языков
- **Простота использования**: Простой интерфейс командной строки

## Быстрый старт

### Установка

```bash
git clone https://github.com/MultiTalk/MultiTalk.git
cd MultiTalk
pip install -r requirements.txt
```

### Генерация аудио

```bash
python inference.py --text "Привет" --speakers speaker1,speaker2 --output output.wav
```
