# Element UI 使用指南

本指南介紹如何在 Vue 2 應用程序中使用 Element UI 組件。

## 基本用法

### Hello World

```vue
<template>
  <div>
    <el-button>默認按鈕</el-button>
    <el-button type="primary">主要按鈕</el-button>
  </div>
</template>

<script>
export default {
  name: 'App'
}
</script>
```

## 常用組件

### 按鈕

```vue
<template>
  <div>
    <el-button>默認</el-button>
    <el-button type="primary">主要</el-button>
    <el-button type="success">成功</el-button>
    <el-button type="warning">警告</el-button>
    <el-button type="danger">危險</el-button>
    <el-button plain>樸素</el-button>
    <el-button round>圓角</el-button>
    <el-button disabled>禁用</el-button>
  </div>
</template>
```

### 輸入框

```vue
<template>
  <el-input v-model="input" placeholder="輸入文本"></el-input>
</template>

<script>
export default {
  data() {
    return { input: '' }
  }
}
</script>
```

### 表格

```vue
<template>
  <el-table :data="tableData" stripe border>
    <el-table-column prop="name" label="姓名"></el-table-column>
    <el-table-column prop="age" label="年齡"></el-table-column>
  </el-table>
</template>

<script>
export default {
  data() {
    return {
      tableData: [{ name: '張三', age: 25 }]
    }
  }
}
</script>
```

### 表單

```vue
<template>
  <el-form :model="form" label-width="80px">
    <el-form-item label="姓名">
      <el-input v-model="form.name"></el-input>
    </el-form-item>
    <el-form-item>
      <el-button type="primary" @click="submit">提交</el-button>
    </el-form-item>
  </el-form>
</template>

<script>
export default {
  data() {
    return { form: { name: '' } }
  },
  methods: {
    submit() {
      console.log(this.form)
    }
  }
}
</script>
```

### 對話框

```vue
<template>
  <el-button @click="visible = true">打開對話框</el-button>
  <el-dialog :visible.sync="visible" title="標題">
    <span>內容</span>
  </el-dialog>
</template>

<script>
export default {
  data() {
    return { visible: false }
  }
}
</script>
```
