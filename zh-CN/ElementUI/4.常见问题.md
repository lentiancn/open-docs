# Element UI 常见问题

> 常见问题解答

---

## 安装问题

### npm install 安装失败怎么办？

**问题**：npm install 安装失败

**解决方案**：

1. 清除 npm 缓存：`npm cache clean --force`
2. 检查网络，可以使用国内镜像源
3. 尝试使用 yarn：`yarn add element-ui`

---

## 样式问题

### 组件样式不生效怎么办？

**问题**：组件样式显示异常

**解决方案**：

1. 确保已引入样式文件：`import 'element-ui/lib/theme-chalk/index.css'`
2. 检查 webpack 配置是否正确处理了 CSS
3. 确保没有 CSS 冲突

---

## 组件问题

### 组件引入后不显示怎么办？

**问题**：组件引入后不显示

**解决方案**：

1. 检查是否正确使用了 `Vue.use(ElementUI)`
2. 确认 Vue 版本是否在 2.6.0 以上
3. 检查浏览器控制台是否有错误信息

---

## 环境要求

### Node.js 版本要求是多少？

| 环境 | 最低版本 | 推荐版本 |
|------|----------|----------|
| Node.js | 8.x | 10.x 或 12.x |
| npm | 5.x | 6.x |
| Vue | 2.6.0 | 2.6.14 |

### 浏览器兼容性如何？

Element UI 支持以下浏览器版本：

- Chrome >= 49
- Firefox >= 45
- Safari >= 10
- Edge >= 14
- IE 11（有限支持，部分功能可能不可用）

## 国际化

### 如何使用中文界面？

```javascript
// main.js
import ElementUI from 'element-ui'
import locale from 'element-ui/lib/locale/lang/zh-CN'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI, {
  locale
})
```

可以选择主题色、边框颜色、圆角等参数，下载定制后的样式文件。

## 迁移问题

### 如何从 Element UI 迁移到 Element Plus？

Element UI 已停止维护，强烈建议新项目使用 Element Plus (Vue 3)。迁移步骤：

1. 升级 Vue 2 → Vue 3
2. 升级 Element UI → Element Plus
3. 更新事件绑定语法（v-model）
4. 更新组件注册方式

官方提供了迁移工具 gogocode：

```bash
# 全局安装
npm install -g gogocode

# 执行迁移
gogocode -s your-project-path -t element-plus -o
```

## 安全建议

### Element UI 还能继续使用吗？

**注意**：

1. **安全风险**：Element UI 已停止维护，存在已知安全漏洞，建议尽快迁移到 Element Plus
2. **Vue 2 EOL**：Vue 2 已于 2023 年底停止维护，建议同步升级到 Vue 3
3. **长期支持**：如果是新项目，请直接使用 Element Plus (Vue 3)
