# JUnit 4 使用指南

---

## 測試類

```java
@Test
public void test() {
    assertEquals(5, 2 + 3);
}
```

---

## 斷言

```java
assertEquals(expected, actual);
assertTrue(condition);
assertNull(object);
```

---

## 註解

```java
@Before
public void setUp() { }

@After
public void tearDown() { }

@Ignore
@Test
public void testSkip() { }
```
