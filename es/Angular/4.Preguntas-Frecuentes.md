# Preguntas Frecuentes de Angular

> Preguntas comunes

## Instalación

### ¿CLI no se instala?

```bash
npm cache clean --force
npm install -g @angular/cli
```

### ¿ng no encontrado?

```bash
npm install -g @angular/cli
```

## Desarrollo

### ¿Proyecto no inicia?

```bash
rm -rf node_modules
npm install
```
