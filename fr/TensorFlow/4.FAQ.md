# TensorFlow FAQ

## Questions Générales

### Qu'est-ce que TensorFlow?
TensorFlow est un framework d'apprentissage automatique open source développé par Google. Il fournit des outils pour construire et entraîner des réseaux neuronaux.

### Quels langages de programmation TensorFlow supporte-t-il?
- Python (principal)
- JavaScript
- Java
- C++
- Swift
- Go

### Quelle est la différence entre PyTorch et TensorFlow?
| Aspect | TensorFlow | PyTorch |
|--------|------------|---------|
| Graphes dynamiques | Limité (TF 2.x) | Natif |
| Débogage | Plus difficile | Plus facile |

## Questions d'Installation

### Comment installer TensorFlow?
```bash
# Version CPU
pip install tensorflow

# Version GPU
pip install tensorflow-gpu
```

### Quels sont les prérequis système?
- **Python**: 3.7-3.11
- **GPU**: NVIDIA GPU avec CUDA (pour version GPU)

### Comment vérifier l'installation?
```python
import tensorflow as tf
print(tf.__version__)
print("GPU Available:", tf.config.list_physical_devices('GPU'))
```

## Questions d'Utilisation

### Comment construire un réseau neuronal simple?
```python
import tensorflow as tf
from tensorflow import keras

model = keras.Sequential([
    keras.layers.Flatten(input_shape=(28, 28)),
    keras.layers.Dense(128, activation='relu'),
    keras.layers.Dense(10)
])

model.compile(optimizer='adam',
              loss=tf.keras.losses.SparseCategoricalCrossentropy(from_logits=True),
              metrics=['accuracy'])

model.fit(x_train, y_train, epochs=5)
```

### Comment utiliser TensorBoard?
```python
tensorboard_callback = keras.callbacks.TensorBoard(log_dir='./logs')
model.fit(x_train, y_train, epochs=5, callbacks=[tensorboard_callback])
```

### Comment sauvegarder et charger des modèles?
```python
# Sauvegarder
model.save('mon_modele.keras')

# Charger
model = keras.models.load_model('mon_modele.keras')
```

## Questions GPU

### Comment activer le support GPU?
```python
gpus = tf.config.list_physical_devices('GPU')
if gpus:
    for gpu in gpus:
        tf.config.experimental.set_memory_growth(gpu, True)
```

### Que faire si je n'ai pas de GPU?
Utilisez la version CPU de TensorFlow. L'entraînement sera plus lent mais fonctionnera sur n'importe quelle machine.

## Dépannage

### Erreurs de mémoire
- Réduire la taille du batch
- Utiliser `set_memory_growth`
- Effacer la session Keras

### Entraînement lent
- Activer l'accélération GPU
- Utiliser l'entraînement en précision mixte
