# Claude Code 简介

下一代编程助手。

## 什么是 Claude Code？

Claude Code 是 Anthropic 公司开发的 AI 编程助手，基于 Claude 大语言模型，专为代码生成、调试和开发任务优化。

### 主要特性

- 代码生成 - 根据描述生成高质量代码
- 智能补全 - 上下文感知的代码补全
- 调试辅助 - 识别并修复代码问题
- 解释代码 - 帮助理解复杂代码逻辑
- 多语言支持 - 支持 Python、JavaScript、TypeScript、Go、Rust 等
