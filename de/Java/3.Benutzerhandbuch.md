# Java Benutzerhandbuch

## Ihr erstes Java-Programm

### Hello World erstellen

Erstellen Sie eine neue Textdatei namens HelloWorld.java. Beachten Sie, dass der Dateiname genau mit dem Klassennamen übereinstimmen muss.

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

### Programm kompilieren

Öffnen Sie das Terminal oder die Eingabeaufforderung, navigieren Sie zum Dateiverzeichnis und führen Sie den Kompilierungsbefehl aus:

```bash
javac HelloWorld.java
```

Bei erfolgreicher Kompilierung wird eine HelloWorld.class-Datei generiert.

### Programm ausführen

```bash
java HelloWorld
```

Ausgabe:

```
Hello, World!
```

## Java-Grundsyntax

### Variablen und Datentypen

Java ist eine statisch typisierte Sprache. Variablen müssen vor der Verwendung deklariert werden.

```java
// Primitive Datentypen
int age = 25;
double salary = 5000.50;
boolean isActive = true;
char grade = 'A';
String name = "Max";

// Referenztypen
Integer count = 10;
Double price = 99.99;
```

### Operatoren

```java
// Arithmetische Operatoren
int a = 10, b = 3;
int sum = a + b;      // Addition
int diff = a - b;     // Subtraktion
int prod = a * b;     // Multiplikation
int quot = a / b;     // Division
int rem = a % b;      // Rest

// Vergleichsoperatoren
boolean result = (a > b);

// Logische Operatoren
boolean flag = (a > 5) && (b < 10);
```

### Bedingte Anweisungen

```java
// if-else-Anweisung
if (score >= 90) {
    System.out.println("Ausgezeichnet");
} else if (score >= 60) {
    System.out.println("Bestanden");
} else {
    System.out.println("Nicht bestanden");
}

// switch-Anweisung
switch (day) {
    case 1:
        System.out.println("Montag");
        break;
    case 2:
        System.out.println("Dienstag");
        break;
    default:
        System.out.println("Unbekannt");
}
```

### Schleifen

```java
// for-Schleife
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}

// while-Schleife
int i = 0;
while (i < 10) {
    System.out.println(i);
    i++;
}

// do-while-Schleife
do {
    System.out.println(i);
    i++;
} while (i < 10);

// for-each für Arrays
int[] numbers = {1, 2, 3, 4, 5};
for (int num : numbers) {
    System.out.println(num);
}
```

## Objektorientierte Programmierung

### Klassen und Objekte

```java
// Klasse definieren
public class Person {
    // Felder
    private String name;
    private int age;
    
    // Konstruktor
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // Getter und Setter
    public String getName() {
        return name;
    }
    
    public void setName(String name) {
        this.name = name;
    }
    
    // Methode
    public void sayHello() {
        System.out.println("Hallo, ich bin " + name);
    }
}

// Objekt erstellen
Person person = new Person("Max", 25);
person.sayHello();
```

## Stream API

```java
import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);

// Filtern
List<Integer> evenNumbers = numbers.stream()
    .filter(n -> n % 2 == 0)
    .collect(Collectors.toList());

// Abbilden
List<Integer> squared = numbers.stream()
    .map(n -> n * n)
    .collect(Collectors.toList());

// Summieren
int sum = numbers.stream()
    .mapToInt(Integer::intValue)
    .sum();
```

## Multithreading-Programmierung

### Threads erstellen

```java
// Thread-Klasse erweitern
class MyThread extends Thread {
    @Override
    public void run() {
        System.out.println("Thread läuft");
    }
}

// Runnable-Schnittstelle implementieren
class MyRunnable implements Runnable {
    @Override
    public void run() {
        System.out.println("Thread läuft");
    }
}

// Thread starten
MyThread thread = new MyThread();
thread.start();

// Runnable verwenden
Thread runnableThread = new Thread(new MyRunnable());
runnableThread.start();
```

## Datenbankoperationen

### JDBC-Verbindung

```java
import java.sql.*;

String url = "jdbc:mysql://localhost:3306/mydb";
String user = "root";
String password = "password";

try (Connection conn = DriverManager.getConnection(url, user, password)) {
    String sql = "SELECT * FROM users WHERE id = ?";
    PreparedStatement stmt = conn.prepareStatement(sql);
    stmt.setInt(1, 1);
    
    ResultSet rs = stmt.executeQuery();
    while (rs.next()) {
        System.out.println(rs.getString("name"));
    }
} catch (SQLException e) {
    e.printStackTrace();
}
```
