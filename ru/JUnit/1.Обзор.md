# JUnit 5 - Обзор

## Обзор

JUnit 5 — следующее поколение JUnit, удобный фреймворк для тестирования для Java и JVM.

## Что такое JUnit?

JUnit — это фреймворк с открытым исходным кодом для модульного тестирования. Это стандарт для тестирования приложений Java.

## Основные функции

- Модульная архитектура
- Параметризованные тесты
- Вложенные тесты
- Расширяемый API
