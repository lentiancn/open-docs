# 簡介

Apifox 是國內領先的 API 協作平台，一站式解決 API 設計、Mock、測試、文件等問題。它由杭州熊衛國網路科技有限公司開發維護，是國內最受歡迎的 API 管理工具之一。

Apifox 整合了 API 設計、API 文件管理、API 調試、API Mock、API 自動化測試等功能，團隊成員可以在同一個平台上完成所有 API 相關工作，大幅提升開發效率。

## 核心功能

### API 設計

Apifox 提供視覺化的 API 設計功能，支援 RESTful API 設計規範。使用者可以透過圖形介面定義介面路徑、請求方法、請求參數、回傳參數等，不需要手動編寫 Markdown 或 YAML 文件。系統會自動產生規範的 API 文件，並支援一鍵分享給團隊成員。

### API 文件管理

自動產生 API 文件，支援線上瀏覽和匯出。當 API 定義完成後，文件會自動同步更新，確保團隊成員看到的永遠是最新的介面說明。文件支援多種格式匯出，包括 Markdown、HTML、OpenAPI (Swagger) 等，方便與外部系統整合。

### API 調試

強大的線上 API 調試功能，支援發送各種 HTTP 請求類型，包括 GET、POST、PUT、DELETE、PATCH 等。可以自訂請求頭、請求參數、請求主體，支援表單、JSON、XML 等多種資料格式。回應內容支援格式化顯示，包括 JSON、XML、HTML 等，並提供語法高亮和摺疊功能。

### API Mock

智慧 Mock 功能是 Apifox 的核心亮點之一。只要先定義好資料結構，系統就會自動產生符合規範的 Mock 資料，不需要手動編寫複雜的 Mock 規則。支援根據欄位類型自動產生資料，如字串、數字、布林值、陣列、物件等，還可以自訂 Mock 規則來實現更複雜的場景。

### 自動化測試

提供視覺化的自動化測試功能，使用者可以透過圖形介面設定測試案例，設定判斷條件和迴圈次數。不需要編寫程式碼就能完成 API 自動化測試，大幅降低測試門檻。支援批次執行測試案例，產生詳細的測試報告，幫助團隊快速發現和定位問題。

### 團隊協作

完善的團隊協作功能，支援邀請團隊成員加入專案，設定不同的角色和權限。可以即時同步專案內容，團隊成員看到的永遠是最新版本。支援評論和標註功能，方便團隊成員直接在介面上溝通交流。

## 適用場景

- API 設計與文件管理
- API 調試與問題排查
- 前端開發 Mock 資料
- API 自動化測試
- 團隊協作與知識共享
- 與 CI/CD 流水線整合

## 與其他工具比較

相比 Postman，Apifox 在國內使用者群體中更受歡迎，主要優勢包括：更完善的中文支援、更智慧的 Mock 功能、更順暢的團隊協作體驗、以及完全免費的基礎功能。相比 Swagger，Apifox 提供了更直觀的視覺化介面，上手更容易，不需要編寫程式碼就能完成 API 管理和測試。

作為國內領先的 API 工具，Apifox 持續迭代更新，為開發者提供優質的技術支援和服務。
