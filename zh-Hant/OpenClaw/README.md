## 快速開始

```bash
# 安裝
npm install -g openclaw@latest

# 初始化
openclaw onboard --install-daemon

# 啟動
openclaw gateway --port 18789
```

訪問 http://127.0.0.1:18789/

## 功能

- 多通道（WhatsApp、Telegram、Discord、iMessage）
- Web 控制台
- 多代理路由
- 移動節點
- 安全控制
