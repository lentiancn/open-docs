# Oracle インストールガイド

## 前提条件

- **オペレーティングシステム**: Oracle Linux、RHEL、Windows Server
- **メモリ**: 最小2GB RAM（推奨8GB以上）
- **ディスク容量**: ソフトウェア用に最小8GB

---

## Linuxへのインストール

### 1. Oracleユーザーの作成

```bash
groupadd -g 54321 oinstall
groupadd -g 54322 dba
useradd -u 54321 -g oinstall -G dba oracle
passwd oracle
```

### 2. カーネルパラメータの設定

```bash
kernel.shmmax = 4294967296
kernel.shmall = 2097152
```

### 3. インストーラーの実行

```bash
./runInstaller
```

---

## インストール後

```sql
sqlplus / as sysdba
SELECT name, open_mode FROM v$database;
```
