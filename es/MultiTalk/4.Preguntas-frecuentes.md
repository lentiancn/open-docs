# Preguntas frecuentes

Esta pagina compila preguntas y respuestas comunes de los usuarios de MultiTalk.

## Preguntas basicas

### Que es MultiTalk?

MultiTalk es un marco de generacion de video conversacional multipersonal impulsado por audio desarrollado por el equipo de MeiGen-AI. Puede generar videos que contienen interacciones multipersonales basandose en entradas de audio multiflujo, imagenes de referencia y prompts de texto, soportando sincronizacion de labios con audio.

### Que funciones soporta MultiTalk?

MultiTalk soporta varias funciones: generacion de video conversacional de una sola persona y multipersonal, generacion de video de personajes caricaturescos, generacion de video de canto, y generacion de video de seguimiento de instrucciones. Soporta salida en resoluciones 480P y 720P, con clips de video de hasta 15 segundos.

### Es MultiTalk gratis?

MultiTalk es un proyecto de codigo abierto y puede usarse gratuitamente. El proyecto utiliza la licencia Apache 2.0, permitiendo uso academico y comercial.

## Problemas de instalacion

### Que hacer si hay incompatibilidad de version de CUDA durante la instalacion?

MultiTalk requiere CUDA 12.1 o superior. Primero verifique la version de CUDA del sistema: nvcc --version. Si la version es muy baja, necesita actualizar el controlador de CUDA. Tambien puede intentar usar la version correspondiente de PyTorch CUDA.

### Como manejar VRAM insuficiente?

Si el VRAM es insuficiente, puede intentar los siguientes metodos: Usar el parametro --num_persistent_param_in_dit 0 para reducir el uso de memoria; Usar aceleracion TeaCache para reducir el uso de memoria; Usar aceleracion LoRA para reducir pasos; Bajar la resolucion de salida.

### Que hacer si la instalacion de flash_attn falla?

La instalacion de flash_attn requiere un entorno de compilacion especifico. Asegurese de tener ninja y el toolkit de CUDA correcto instalados. Si continua fallando, intente instalar una version precompilada o omita esta dependencia.

### Que hacer si la descarga de modelos es muy lenta?

Puede usar espejos u otras herramientas de descarga para pre-descargar archivos de modelo y luego moverlos al directorio especificado. Tambien puede intentar usar fuentes de espejo domesticas.

## Problemas de uso

### Que hacer si los labios del video generado no estan sincronizados?

Puede intentar aumentar el valor del parametro Audio CFG, la configuracion recomendada esta entre 3 y 5. Tambien puede aumentar los pasos de muestreo para mejorar la sincronizacion.

### Que hacer si la generacion de video es muy lenta?

Puede habilitar los siguientes metodos de aceleracion: Usar aceleracion TeaCache, que puede mejorar la velocidad de 2 a 3 veces; Usar aceleracion LoRA, FusioniX requiere solo 8 pasos, lightx2v solo 4 pasos; Usar version cuantizada para reducir computacion.

### Como generar videos conversacionales de personas mas largos?

Use el modo streaming para generar videos largos. Asegurese de que Audio CFG este configurado en 3 para mantener la consistencia del tono. Los videos generados pueden tener hasta 15 segundos de duracion.

### Que hacer si el enlace de personas es incorrecto en videos multipersonales?

Este es un problema comun en la generacion de videos multipersonales. MultiTalk resuelve esto a traves de la tecnologia L-RoPE. Asegurese de que el JSON de entrada especifique correctamente las coordenadas de cada region de persona.

### Como usar la funcionalidad TTS de texto a voz?

Prepare un archivo JSON de entrada que contenga texto, configure el parametro --audio_mode tts para convertir automaticamente texto a voz y generar video.

### Se pueden generar videos de personajes caricaturescos?

Si, MultiTalk soporta generacion de personajes caricaturescos. Use imagenes de personajes caricaturescos en la imagen de referencia con texto descriptivo correspondiente.

## Problemas tecnicos

### Que formatos de audio de entrada se soportan?

Se soportan formatos de audio comunes como WAV y MP3. El audio debe contener voz humana clara.

### Como elegir la resolucion del video?

La version actual soporta 480P por defecto. 720P requiere configuracion de multiples GPUs. 480P es suficiente para la mayoria de escenarios de uso.

### Como ejecutar en dispositivos con VRAM bajo?

Usar el parametro --num_persistent_param_in_dit 0 puede reducir significativamente los requisitos de VRAM, permitiendo operacion en una sola RTX 4090.

### Como integrar con otras plataformas?

MultiTalk soporta multiples metodos de uso: Se puede llamar directamente a traves de la plataforma en la nube Replicate; Se puede integrar en flujos de trabajo de ComfyUI; Tambien puede usar la interfaz Gradio proporcionada por el proyecto.

## Problemas de licencia

### Se pueden usar los videos generados comercialmente?

El proyecto utiliza la licencia Apache 2.0, y el contenido de video generado puede usarse para propositos comerciales. Sin embargo, asegurese de que el uso cumpla con los terminos de la licencia y las leyes y regulaciones relevantes.

### Que debo citar al usar?

Si MultiTalk es util para su investigacion, cite el documento relevante. La informacion del documento esta disponible en la pagina de GitHub del proyecto.
