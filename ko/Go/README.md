# Go

> Google이 개발한 컴파일형 프로그래밍 언어

---

## 개요

Go(Golang)는 2009년 Google에서 만들어졌습니다.シンプルさ、효율성、강력한 동시성 지원으로 유명합니다.

---

## 특징

- 정적 타이핑
- 가비지 컬렉션
- 네이티브 동시성 (goroutine)
- 빠른 컴파일
- 풍부한 표준 라이브러리
