# OpenCode 使用指南

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+S | 保存 |
| Ctrl+C | 复制 |
| Ctrl+V | 粘贴 |
| Ctrl+Z | 撤销 |
| Ctrl+/ | 切换注释 |
| Ctrl+K Ctrl+O | 打开文件夹 |

## 终端

- View → Terminal 或 Ctrl+`
- 集成终端支持

## 扩展

- Ctrl+Shift+X 打开扩展市场
- 安装 Prettier、ESLint 等

## Git

- 左侧源代码管理面板
- 提交、推送、拉取
