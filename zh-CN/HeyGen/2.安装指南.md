# HeyGen 安装指南

## 访问方式

HeyGen 是基于云的 SaaS 平台，无需本地安装。

## 使用步骤

1. 访问 https://heygen.com 注册账号
2. 登录控制台
3. 选择模板或创建数字人
4. 输入文本并生成视频

## API 接入

```bash
# API 接入示例
curl -X POST https://api.heygen.com/v1/video/generate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"script": "Hello World", "avatar_id": "avatar_1"}'
```
