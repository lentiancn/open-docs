# Element Plus 常見問題

常見問題解答。

## 安裝問題

### Q: 安裝失敗？

**解決方案**：
1. 檢查 Node.js 版本
```bash
node --version  # 需要 18.0+
```
2. 清理 npm 快取
```bash
npm cache clean --force
```
3. 刪除 node_modules 和 package-lock.json，重新安裝
```bash
rm -rf node_modules package-lock.json
npm install
```

### Q: 提示模組找不到？

**解決方案**：
1. 確認包已安裝
```bash
npm list element-plus
```
2. 檢查 npm/yarn/pnpm 配置
3. 嘗試刪除 lock 文件重新安裝

### Q: TypeScript 類型錯誤？

**解決方案**：
1. 安裝類型定義
```bash
npm install -D @types/node
```
2. 在 tsconfig.json 中添加
```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

## 使用問題

### Q: 元件不顯示？

**解決方案**：
1. 確認已正確引入
2. 檢查控制台錯誤信息
3. 確認 Vue 版本（需要 3.2+）

### Q: 樣式不生效？

**解決方案**：
1. 確認已引入 CSS
```typescript
import 'element-plus/dist/index.css'
```
2. 檢查 CSS 優先級，使用 `:deep()` 或 `!important`
3. 檢查是否有其他樣式覆蓋

### Q: 圖標不顯示？

**解決方案**：
1. 安裝圖標包
```bash
npm install @element-plus/icons-vue
```
2. 全域註冊
```typescript
import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```
3. 或者按需引入

### Q: Table 表格不渲染？

**解決方案**：
1. 確認 data 屬性正確綁定
2. 檢查 prop 是否正確
3. 確認 table 在 el-table-column 外面

### Q: Form 表單驗證不生效？

**解決方案**：
1. 確保 prop 與 v-model 對應
2. 確保有 ref 引用
3. 正確調用 validate 方法
```typescript
const formRef = ref()
await formRef.value.validate()
```

### Q: Dialog 對話框不顯示？

**解決方案**：
1. 檢查 v-model 綁定
2. 確認 visible 為 true
3. 檢查 z-index 是否有衝突

## 主題問題

### Q: 如何修改主題顏色？

**解決方案**：
使用 CSS 變量：
```css
:root {
  --el-color-primary: #409eff;
}
```

### Q: 如何使用暗色主題？

**解決方案**：
引入暗色 CSS：
```typescript
import 'element-plus/theme-chalk/dark/css-vars.css'
```

### Q: 元件樣式怎麼覆蓋？

**解決方案**：
使用 CSS 選擇器：
```css
.el-button--primary {
  background-color: #your-color;
}
```

## 國際化問題

### Q: 如何切換語言？

**解決方案**：
```typescript
import zhTw from 'element-plus/es/locale/lang/zh-tw'

app.use(ElementPlus, { locale: zhTw })
```

### Q: 自定義文案？

**解決方案**：
```typescript
import {ElMessage} from 'element-plus'
ElMessage('自定義信息')
```

## 效能問題

### Q: 首屏載入慢？

**解決方案**：
1. 使用按需引入
2. 使用自動導入插件
3. 配置 Vite 預構建

### Q: 打包體積太大？

**解決方案**：
1. 確保使用按需引入
2. 使用 tree shaking
3. 檢查是否引入了完整包

## 相容性問題

### Q: 不支持 IE 瀏覽器？

Element Plus 不支持 IE 瀏覽器，建議使用以下瀏覽器：
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### Q: 與其他庫衝突？

**解決方案**：
1. 檢查 z-index 層級
2. 使用 CSS Modules 避免樣式衝突
3. 檢查 Vue 版本相容性
