# Go 常見問題

> Go 常見問題解答

## 安裝

### Q: 安裝後指令找不到？
**A**: 檢查 PATH 是否包含 Go

### Q: 如何確認安裝？
```bash
go version
go env GOPATH
```

---

## 語法

### Q: Go 有例外處理嗎？
**A**: 沒有，錯誤透過返回值處理：
```go
result, err := doSomething()
if err != nil {
    // 處理錯誤
}
```

### Q: Go 支援泛型嗎？
**A**: Go 1.18+ 支援泛型：
```go
func Print[T any](s []T) {
    for _, v := range s {
        fmt.Println(v)
    }
}
```

## 並發

### Q: Goroutine 和線程的區別？
- Goroutine 由 Go 執行時期管理
- 比線程更輕量

### Q: 如何優雅等待 Goroutine 結束？
```go
var wg sync.WaitGroup
wg.Add(1)
go func() {
    defer wg.Done()
}()
wg.Wait()
```
