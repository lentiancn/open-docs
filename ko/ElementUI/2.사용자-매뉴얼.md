## 1. 기본 컴포넌트

### 1.1 버튼 컴포넌트

버튼은 가장 많이 사용되는 컴포넌트 중 하나입니다.

#### 기본 사용

```html
<el-button>기본</el-button>
<el-button type="primary">주요</el-button>
<el-button type="success">성공</el-button>
<el-button type="warning">경고</el-button>
<el-button type="danger">위험</el-button>
```

#### 버튼 속성

| 속성 | 설명 | 유형 | 기본값 |
|------|------|------|--------|
| type | 버튼 유형 | string | — |
| plain | 단순 스타일 | boolean | false |
| round | 둥근 스타일 | boolean | false |
| disabled | 비활성화 | boolean | false |

## 2. 폼 컴포넌트

### 2.1 입력 필드

```html
<el-input v-model="input" placeholder="내용 입력"></el-input>
```

### 2.2 선택 목록

```html
<el-select v-model="selected" placeholder="선택하세요">
  <el-option label="옵션 1" value="1"></el-option>
  <el-option label="옵션 2" value="2"></el-option>
</el-select>
```

### 2.3 폼 검증

```javascript
this.$refs.form.validate(valid => {
  if (valid) {
    console.log('검증 성공')
  }
})
```

## 3. 테이블 컴포넌트

### 3.1 기본 테이블

```html
<el-table :data="tableData" stripe border>
  <el-table-column prop="name" label="이름"></el-table-column>
  <el-table-column prop="email" label="이메일"></el-table-column>
</el-table>
```

### 3.2 페이지네이션이 있는 테이블

```html
<el-pagination
  @current-change="handlePageChange"
  :current-page="currentPage"
  :total="total">
</el-pagination>
```

## 4. 다이얼로그와 알림

### 4.1 다이얼로그

```html
<el-dialog title="제목" :visible.sync="visible">
  <span>다이얼로그 내용</span>
</el-dialog>
```

### 4.2 메시지

```javascript
this.$message.success('성공')
this.$message.error('오류')
this.$message.warning('경고')
```
