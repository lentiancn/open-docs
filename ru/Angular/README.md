# Angular

Современный фреймворк для веб-разработки.

## Обзор

Angular — это открытый веб-фреймворк, разработанный и поддерживаемый Google, предоставляющий полное решение для современной веб-разработки.

### Основные функции

- **Нативный TypeScript**: Построен на TypeScript
- **Компонентный**: Простое сопровождение и тестирование
- **Внедрение зависимостей**: Мощная система DI
- **Маршрутизация**: Полная клиентская маршрутизация
- **Формы**: Поддержка реактивных форм
- **HTTP-клиент**: Полные HTTP-сервисы

## Быстрый старт

### Установить Angular CLI

```bash
npm install -g @angular/cli
```

### Создать новый проект

```bash
ng new my-app
cd my-app
ng serve
```

### Открыть приложение

Откройте браузер: http://localhost:4200

## Общие команды

```bash
# Создать компонент
ng g c my-component

# Создать сервис
ng g s my-service

# Собрать для продакшена
ng build

# Запустить тесты
ng test
```

## Ключевые концепции

- **Компонент**: Сочетание представления и логики
- **Модуль**: Организация кода приложения
- **Сервис**: Инкапсуляция бизнес-логики
- **Маршрутизация**: Навигация по страницам
- **Директива**: Расширение HTML
- **Пайп**: Преобразование данных
