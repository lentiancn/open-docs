## インストール

### Windows: goコマンドが見つからない？

**解決策**: `C:\Go\bin` をPATHに追加。

### プロキシ設定？

```bash
go env -w GOPROXY=https://goproxy.cn,direct
```

---

## 開発

### "cannot find package" エラー？

**解決策**:
1. `go mod init` を実行
2. `go mod tidy` を実行

### := と var の違いは？

- `var`: 明示的、関数外で使える
- `:=`: 简短、関数内のみ

## 並行処理

### Goroutineリーク？

```go
func worker(ctx context.Context) {
    for {
        select {
        case <-ctx.Done():
            return
        }
    }
}
```
