# OpenClaw Benutzerhandbuch

> Umfassende Anleitung zur Verwendung von OpenClaw

---

## Erste Schritte

### Grundlegende Nutzung

OpenClaw kann auf verschiedene Arten verwendet werden:

#### Interaktive Sitzung

```bash
# Starten Sie eine interaktive Konversation
openclaw chat
```

#### Befehlsausführung

```bash
# Führen Sie einen einzelnen Befehl aus
openclaw exec "Hilf mir, eine Datei zu erstellen"

# Führen Sie komplexere Aufgaben aus
openclaw exec --task "Recherchiere aktuelle Nachrichten"
```

### Gateway-Verwaltung

Der Gateway-Dienst muss für die meisten Operationen aktiv sein:

```bash
# Gateway starten
openclaw gateway start

# Status prüfen
openclaw gateway status

# Gateway stoppen
openclaw gateway stop
```

## Dateiverwaltung

### Dateien lesen

```bash
# Verwenden Sie das read-Tool
read /pfad/zur/datei.txt
```

### Dateien schreiben

```bash
# Neue Datei erstellen oder überschreiben
write --content "Inhalt der Datei" --pfad /pfad/zu/datei.txt
```

### Dateien bearbeiten

```bash
# Genaue Textersetzung in einer Datei
edit --pfad /pfad/zu/datei.txt --oldText "alter Text" --newText "neuer Text"
```

## Browser-Steuerung

OpenClaw kann Webbrowser automatisiert steuern:

### Browser starten

```bash
# Isolierten Browser starten
browser start --profile openclaw

# Chrome-Erweiterung verwenden
browser start --profile chrome
```

### Browser-Aktionen

```bash
# Zu URL navigieren
browser navigate --url https://beispiel.de

# Screenshot erstellen
browser screenshot --pfad /output/screenshot.png

# Element klicken
browser click --selector "#button-id"
```

## Web-Zugriff

### Websuche

```bash
# Im Internet suchen
web_search --query "aktuelle Nachrichten" --count 5
```

### Webseite abrufen

```bash
# Inhalt einer Webseite abrufen
web_fetch --url https://beispiel.de --maxChars 5000
```

## Subagenten

### Subagenten erstellen

```bash
# Neuen Subagenten für spezifische Aufgaben starten
subagents create --task "Dokumentationssammlung"

# Subagenten steuern
subagents steer --id agent-123 --message "Weitere Aufgabe"

# Subagenten beenden
subagents kill --id agent-123
```

## Konfiguration und Personalisierung

### Persönlichkeit anpassen

Bearbeiten Sie `SOUL.md`, um die Persönlichkeit des Assistenten anzupassen:

```markdown
# SOUL.md - Wer du bist

## Kernwahrheiten

- Sei authentisch hilfreich
- Habe Meinungen
- Sei einfallsreich
```

### Werkzeuge konfigurieren

Bearbeiten Sie `TOOLS.md` für spezifische Werkzeugeinstellungen:

```markdown
# TOOLS.md - Lokale Notizen

## SSH-Verbindungen
- server1 -> 192.168.1.100
```

---

## Best Practices

1. **Arbeitsverzeichnis**: Halten Sie Ihre Dateien im definierten Workspace
2. **Sicherheit**: Seien Sie vorsichtig bei externen Aktionen
3. **Speicher**: Nutzen Sie memory-Dateien für wichtige Informationen
4. **Gateway**: Halten Sie den Gateway-Dienst für schnelle Reaktionszeiten aktiv

---

## Erweiterte Funktionen

### Automatisierung

Erstellen Sie Skripte zur Automatisierung wiederkehrender Aufgaben:

```bash
# Automatisierte Datensammlung
openclaw exec --task "Sammle Informationen zu Topic X"
```

### Cron-Jobs

Richten Sie periodische Aufgaben ein:

```bash
# Zeitgesteuerte Aufgabe erstellen
openclaw cron --schedule "0 9 * * *" --task "Morning briefing"
```
