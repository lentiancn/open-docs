# Angular Installationsanleitung

Angular ist ein leistungsstarkes Frontend-Framework, das von Google entwickelt wurde. Diese Anleitung behandelt Installationsmethoden für verschiedene Umgebungen.

## Installationsmethoden

### Methode 1: Angular CLI (Empfohlen)

#### Angular CLI installieren

```bash
# Global installieren
npm install -g @angular/cli

# Installation überprüfen
ng version
```

#### Neues Projekt erstellen

```bash
# Neues Projekt erstellen
ng new my-app

# Mit Optionen erstellen
ng new my-app --routing --style=scss --skip-tests
```

#### Optionen

```bash
# Installation überspringen
ng new my-app --skip-install

# Standalone-Komponenten (Angular 14+)
ng new my-app --standalone

# SSR aktivieren
ng new my-app --ssr
```

### Methode 2: npx verwenden

```bash
# Projekt ohne CLI-Installation erstellen
npx @angular/cli new my-app
```

### Methode 3: Manuelle Einrichtung

```bash
# Verzeichnis erstellen
mkdir my-app
cd my-app

# npm initialisieren
npm init -y

# Angular installieren
npm install @angular/core @angular/common @angular/platform-browser @angular/platform-browser-dynamic @angular/compiler @angular/forms @angular/router @angular/animations rxjs zone.js

# Entwicklungsabhängigkeiten installieren
npm install --save-dev @angular/compiler-cli @angular-devkit/build-angular typescript
```

### Methode 4: Docker

```dockerfile
# Dockerfile
FROM node:20-alpine

WORKDIR /app

# Angular CLI installieren
RUN npm install -g @angular/cli

# Projekt erstellen
RUN ng new my-app --directory .

# Entwicklungsserver starten
CMD ["ng", "serve", "--host", "0.0.0.0"]
```

## Umgebungskonfiguration

### Node.js-Versionsverwaltung

#### nvm verwenden (Empfohlen)

```bash
# nvm installieren
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Node.js installieren
nvm install 20
nvm use 20

# Standard festlegen
nvm alias default 20
```

### npm-Konfiguration

```bash
# Registry festlegen
npm config set registry https://registry.npmjs.org/

# Strenges SSL aktivieren
npm config set strict-ssl true

# Cache leeren
npm cache clean --force
```

## Überprüfung

```bash
# Angular CLI überprüfen
ng version

# Node.js überprüfen
node --version

# npm überprüfen
npm --version
```

## Häufige Probleme

### Berechtigung verweigert?

```bash
# npm-Berechtigungen reparieren
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
export PATH=~/.npm-global/bin:$PATH
```

### Langsame Installation?

```bash
# npm ci für schnellere Installation verwenden
npm ci

# Oder yarn verwenden
npm install -g yarn
```

## Angular aktualisieren

```bash
# Angular CLI aktualisieren
npm install -g @angular/cli@latest

# Projekt aktualisieren
ng update @angular/core @angular/cli
```
