# Guía de instalación de MySQL

## Instalación en Windows

### Instalación usando MySQL Installer

MySQL Installer es la forma más fácil de instalar MySQL en Windows, proporcionando una experiencia de instalación basada en asistentes.

1. Descargar MySQL Installer
   Visite https://dev.mysql.com/downloads/installer/ para descargar el instalador.

2. Ejecutar el instalador
   Haga doble clic en el archivo msi descargado para iniciar el asistente de instalación.

3. Elegir tipo de instalación
   - **Full**: Instalar todos los productos y características de MySQL
   - **Developer Default**: Apropiado para entorno de desarrollo
   - **Server Only**: Instalar solo MySQL Server
   - **Custom**: Elegir componentes específicos para instalar

4. Configurar MySQL Server
   - Seleccionar versión de MySQL Server
   - Establecer número de puerto (predeterminado 3306)
   - Elegir tipo de inicio

5. Establecer contraseña root
   Establezca una contraseña segura para la cuenta de administrador de MySQL.

6. Completar instalación
   Siga el asistente para terminar el proceso de instalación.

### Instalación manual usando ZIP

1. Descargar paquete ZIP de MySQL
   Visite https://dev.mysql.com/downloads/mysql/ para descargar el archivo ZIP.

2. Extraer archivos
   Extraiga el archivo ZIP en el directorio deseado, como C:\mysql.

3. Inicializar directorio de datos
   ```
   mysqld --initialize-insecure
   ```

4. Iniciar servicio MySQL
   ```
   mysqld --install MySQL
   net start MySQL
   ```

5. Iniciar sesión y cambiar contraseña
   ```
   mysql -u root -p
   ALTER USER 'root'@'localhost' IDENTIFIED BY 'nueva_contraseña';
   ```

## Instalación en Linux

### Sistemas Ubuntu/Debian

1. Actualizar índice de paquetes
   ```bash
   sudo apt update
   ```

2. Instalar MySQL Server
   ```bash
   sudo apt install mysql-server
   ```

3. Iniciar servicio MySQL
   ```bash
   sudo systemctl start mysql
   sudo systemctl enable mysql
   ```

4. Configuración segura
   ```bash
   sudo mysql_secure_installation
   ```

### Sistemas CentOS/RHEL

1. Agregar repositorio MySQL Yum
   ```bash
   sudo yum install https://dev.mysql.com/get/mysql80-community-release-el8-4.noarch.rpm
   ```

2. Instalar MySQL Server
   ```bash
   sudo yum install mysql-community-server
   ```

3. Iniciar servicio
   ```bash
   sudo systemctl start mysqld
   sudo systemctl enable mysqld
   ```

4. Obtener contraseña temporal
   ```bash
   sudo grep 'temporary password' /var/log/mysqld.log
   ```

## Instalación en macOS

### Usando Homebrew

1. Instalar Homebrew (si no está instalado)
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```

2. Instalar MySQL
   ```bash
   brew install mysql
   ```

3. Iniciar servicio MySQL
   ```bash
   brew services start mysql
   ```

4. Configuración segura
   ```bash
   mysql_secure_installation
   ```

### Usando paquete DMG

1. Descargar paquete DMG
   Visite https://dev.mysql.com/downloads/mysql/ para descargar el paquete DMG de macOS.

2. Ejecutar instalador
   Haga doble clic en el archivo PKG y siga el asistente para completar la instalación.

3. Iniciar MySQL
   Inicie el servicio MySQL a través de Preferencias del Sistema.

## Instalación de MySQL con Docker

### Descargar imagen de MySQL
```bash
docker pull mysql:latest
```

### Ejecutar contenedor MySQL
```bash
docker run -d \
  --name mysql-container \
  -e MYSQL_ROOT_PASSWORD=tu_contraseña \
  -e MYSQL_DATABASE=testdb \
  -p 3306:3306 \
  mysql:latest
```

### Parámetros comunes de Docker
- `-e MYSQL_ROOT_PASSWORD`: Establecer contraseña root
- `-e MYSQL_DATABASE`: Crear base de datos inicial
- `-p 3306:3306`: Mapeo de puertos
- `-v /data/mysql:/var/lib/mysql`: Volumen para persistencia de datos

## Verificación después de la instalación

### Verificar versión de MySQL
```bash
mysql --version
```

### Verificar estado del servicio
```bash
systemctl status mysql   # Linux
brew services list      # macOS
```

### Iniciar sesión en MySQL
```bash
mysql -u root -p
```

## Problemas comunes de instalación

### Conflictos de puerto
Si el puerto 3306 está en uso, modifique el puerto en el archivo de configuración o detenga el servicio que usa ese puerto.

### Problemas de permisos
En Linux, use sudo para elevar permisos y asegúrese de que el directorio de datos tenga permisos de lectura/escritura correctos.

### Configuración de firewall
Asegúrese de que el firewall permita el acceso al puerto 3306.
