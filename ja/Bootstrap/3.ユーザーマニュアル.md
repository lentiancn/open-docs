# ユーザーマニュアル

このマニュアルではBootstrap 5の主な機能と使い方を詳しく説明します。グリッドシステム、コンポーネント、CSSユーティリティクラスなどをカバーしています。

## グリッドシステム

グリッドシステムはBootstrapのコア機能で、12列レイアウトを提供します。

### 基本的な使い方

```html
<div class="container">
    <div class="row">
        <div class="col">列 1</div>
        <div class="col">列 2</div>
        <div class="col">列 3</div>
    </div>
</div>
```

### レスポンシブブレークポイント

| クラス名 | 画面幅 | 説明 |
|------|---------|------|
| `.col-xs` | < 576px | 極小画面 |
| `.col-sm` | ≥ 576px | 小画面 |
| `.col-md` | ≥ 768px | 中画面 |
| `.col-lg` | ≥ 992px | 大画面 |
| `.col-xl` | ≥ 1200px | 特大画面 |
| `.col-xxl` | ≥ 1400px | 超大画面 |

### 列のオフセット

```html
<div class="row">
    <div class="col-md-4">.col-md-4</div>
    <div class="col-md-4 offset-md-4">.col-md-4 .offset-md-4</div>
</div>
```

### レスポンシブ列

```html
<div class="row">
    <div class="col-12 col-md-6 col-lg-4">
        スマホでは全幅、タブレットでは半分、画面が大きいときは3分の1
    </div>
</div>
```

## ベストプラクティス

1. **意味的なHTMLを使う**：header、nav、main、footerなどのタグを適切に使う
2. **シンプルに保つ**：入れ子になりすぎないように
3. **レスポンシブを優先**：モバイルから設計を始める
4. **ユーティリティクラスを活用**：素早くスタイルを調整
5. **テーマをカスタマイズ**：Sass変数を使って一貫性を保つ
