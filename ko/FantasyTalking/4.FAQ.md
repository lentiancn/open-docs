# FantasyTalking - FAQ

자주 묻는 질문과 답변.

## 설치 문제

### Q: 의존성 설치 실패?

**해결:**
1. pip 업그레이드: `pip install --upgrade pip`
2. 중국 미러 사용:
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```
3. 실패한 패키지를 개별적으로 설치

### Q: CUDA 사용 불가?

**해결:**
1. NVIDIA 드라이버가 설치되어 있는지 확인: `nvidia-smi`
2. 호환되는 PyTorch 버전 설치:
```bash
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
```
3. CUDA 가용성 확인:
```python
import torch
print(torch.cuda.is_available())
```

### Q: 메모리 부족?

**해결:**
1. 더 작은 입력 이미지 사용
2. 배치 크기 줄이기
3. CPU 모드 사용 (권장하지 않음, 느림)

## 사용 문제

### Q: 생성 실패, 얼굴을 찾을 수 없음?

**해결:**
1. 선명한 얼굴이 포함된 이미지 사용
2. 얼굴이 전면이고 가려지지 않았는지 확인
3. 이미지 형식 확인

### Q: 비디오 생성 너무 느림?

**해결:**
1. GPU 가속 사용
2. 출력 해상도 낮추기
3. 출력 프레임 수 줄이기

### Q: 입술이 동기화되지 않음?

**해결:**
1. 오디오가 선명하고 노이즈가 없는지 확인
2. 오디오 샘플 레이트 확인
3. 다양한 동기화 매개변수 시도

### Q: 출력 비디오 흐림?

**해결:**
1. 얼굴 향상 사용: `--face_enhancer realusf`
2. 출력 해상도 높이
3. 더 선명한 소스 이미지 사용

### Q: 얼굴 변형/왜곡?

**해결:**
1. 가려지지 않은 전면 사진 사용
2. 이미지에 얼굴이 하나만 있는지 확인
3. 얼굴 감지 매개변수 조정

## 모델 문제

### Q: 모델 다운로드 실패?

**해결:**
1. 프록시 또는 미러 사용
2. 모델 파일을 수동으로 다운로드
3. 네트워크 연결 확인

### Q: 모델 로드 실패?

**해결:**
1. 모델 파일이 완전한지 확인
2. 모델 경로가 올바른지 확인
3. 모델 다시 다운로드

## 성능 문제

### Q: 생성 속도를 높이려면 어떻게 해야 하나?

**해결:**
1. GPU 사용
2. 혼합 정밀도 활성화: `--precision fp16`
3. 더 작은 해상도 사용

### Q: 메모리 사용량이 너무 높음?

**해결:**
1. 배치 크기 줄이기
2. 더 작은 모델 사용
3. 불필요한 기능 비활성화

## 출력 문제

### Q: 출력 비디오를 재생할 수 없음?

**해결:**
1. 디코더 설치: `pip install opencv-python`
2. 지원되는 형식 사용: MP4

### Q: 비디오 파일이 너무 큼?

**해결:**
1. 해상도 낮추기
2. 프레임 레이트 낮추기
3. 더 높은 압축 사용
