# Руководство пользователя JUnit 5

## Базовый тест

```java
@Test
void test() {
    assertEquals(4, 2 + 2);
}
```

## Assertions

```java
assertEquals(4, 2 + 2);
assertTrue(5 > 3);
assertThrows(Exception.class, () -> { });
```

## Параметризованные тесты

```java
@ParameterizedTest
@ValueSource(ints = {1, 2, 3})
void test(int n) {
    assertTrue(n > 0);
}
```

## Вложенные тесты

```java
@Nested
class InnerTests {
    @Test
    void test() { }
}
```
