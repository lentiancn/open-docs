## 柵格系統

### 基本原理

Bootstrap 的柵格系統使用 Flexbox 建構，最多支援 12 列。

### 容器

```html
<div class="container">內容</div>
<div class="container-fluid">100% 寬度</div>
```

### 列

```html
<div class="row">
  <div class="col-md-4">4 列</div>
  <div class="col-md-8">8 列</div>
</div>
```

## 元件

### 按鈕

```html
<button class="btn btn-primary">主要</button>
<button class="btn btn-secondary">次要</button>
<button class="btn btn-success">成功</button>
<button class="btn btn-danger">危險</button>
```

### 卡片

```html
<div class="card" style="width: 18rem;">
  <div class="card-body">
    <h5 class="card-title">標題</h5>
    <p class="card-text">內容</p>
  </div>
</div>
```

### 導航欄

```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <a class="navbar-brand" href="#">品牌</a>
</nav>
```

## 表單

```html
<form>
  <div class="mb-3">
    <label class="form-label">郵箱</label>
    <input type="email" class="form-control">
  </div>
  <button type="submit" class="btn btn-primary">提交</button>
</form>
```

---

## JavaScript 元件

### 彈出視窗

```html
<button data-bs-toggle="modal" data-bs-target="#myModal">開啟</button>
<div class="modal" id="myModal">
  <div class="modal-dialog">
    <div class="modal-content">內容</div>
  </div>
</div>
```
