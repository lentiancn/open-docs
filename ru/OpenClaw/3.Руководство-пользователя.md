# OpenClaw Руководство пользователя

> Подробное руководство по использованию OpenClaw

---

## Начало работы

### Основное использование

OpenClaw можно использовать несколькими способами:

#### Интерактивный сеанс

```bash
# Запустить интерактивный разговор
openclaw chat
```

#### Выполнение команд

```bash
# Выполнить одну команду
openclaw exec "помоги создать файл"

# Выполнить более сложные задачи
openclaw exec --task "исследовать последние новости"
```

### Управление Gateway

Служба Gateway должна быть активна для большинства операций:

```bash
# Запустить Gateway
openclaw gateway start

# Проверить статус
openclaw gateway status

# Остановить Gateway
openclaw gateway stop
```

## Управление файлами

### Чтение файлов

```bash
# Использовать инструмент чтения
read /path/to/file.txt
```

### Запись файлов

```bash
# Создать новый файл или перезаписать
write --content "Содержимое файла" --path /path/to/file.txt
```

### Редактирование файлов

```bash
# Точная замена текста в файле
edit --path /path/to/file.txt --oldText "старый текст" --newText "новый текст"
```

## Управление браузером

OpenClaw может автоматизировать веб-браузеры:

### Запуск браузера

```bash
# Запустить изолированный браузер
browser start --profile openclaw

# Использовать расширение Chrome
browser start --profile chrome
```

### Действия браузера

```bash
# Перейти по URL
browser navigate --url https://example.com

# Сделать скриншот
browser screenshot --path /output/screenshot.png

# Кликнуть по элементу
browser click --selector "#button-id"
```

## Веб-доступ

### Веб-поиск

```bash
# Искать в интернете
web_search --query "последние новости" --count 5
```

### Получение веб-страницы

```bash
# Получить содержимое веб-страницы
web_fetch --url https://example.com --maxChars 5000
```

message send --target user --message "Привет!"

## Под-агенты

### Создание под-агентов

```bash
# Запустить новый под-агент для конкретных задач
subagents create --task "сбор документации"

# Управлять под-агентом
subagents steer --id agent-123 --message "следующая задача"

# Завершить под-агента
subagents kill --id agent-123
```

## Настройка и персонализация

### Настройка личности

Отредактируйте `SOUL.md`, чтобы настроить личность ассистента:

```markdown
# SOUL.md - Кто вы

## Основные истины

- Будьте по-настоящему полезным
- Имейте мнение
- Будьте находчивым
```

### Настройка инструментов

Отредактируйте `TOOLS.md` для конкретных настроек инструментов:

```markdown
# TOOLS.md - Локальные заметки

## SSH-соединения
- server1 -> 192.168.1.100
```

---

## Рекомендации

1. **Рабочее пространство**: Храните файлы в определенном рабочем пространстве
2. **Безопасность**: Будьте осторожны с внешними действиями
3. **Память**: Используйте файлы памяти для важной информации
4. **Gateway**: Держите службу Gateway активной для быстрого отклика

## Расширенные функции

### Автоматизация

Создавайте скрипты для автоматизации повторяющихся задач:

```bash
# Автоматизированный сбор данных
openclaw exec --task "собрать информацию о теме X"
```

### Cron-задачи

Настройте периодические задачи:

```bash
# Создать запланированную задачу
openclaw cron --schedule "0 9 * * *" --task "утренний брифинг"
```
