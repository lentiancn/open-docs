# OpenClaw Installationsanleitung

Diese Anleitung führt dich Schritt für Schritt durch den Installationsprozess auf verschiedenen Betriebssystemen.

## Anforderungen

- **Node.js**: Version 22 oder höher
- **Betriebssystem**: macOS, Linux oder Windows
- **Optional**: Docker (für Sandbox-Isolierungsfunktionen)

## Installationsmethoden

### macOS und Linux (Empfohlen)

Der einfachste Weg ist das offizielle Installationsskript:

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

Dieses Skript erkennt automatisch deine Systemumgebung, installiert die notwendigen Abhängigkeiten und richtet die Grundeinstellungen ein.

### Windows

Führe dies in PowerShell aus:

```powershell
iwr -useb https://openclaw.ai/install.ps1 | iex
```

### Direkte npm-Installation

Wenn du Node.js bereits eingerichtet hast:

```bash
npm install -g openclaw@latest
```

### Aus dem Quellcode

Um die neueste Entwicklungsversion zu nutzen:

```bash
git clone https://github.com/openclaw/openclaw.git
cd openclaw
npm install
npm link
```

## Nach der Installation

### Einrichtungsassistent ausführen

Nach der Installation solltest du den Assistenten für die Grundeinrichtung ausführen:

```bash
openclaw onboard --install-daemon
```

Der Assistent führt dich durch:
- API-Schlüssel-Konfiguration
- Dienst-Port-Einstellungen
- Kanal-Anmeldung (WhatsApp, Telegram, Discord, etc.)
- Sicherheits- und Zugriffskontroll-Einstellungen

### Dienststatus prüfen

Wenn du wie oben beschrieben installiert hast, sollte der Dienst bereits laufen:

```bash
openclaw gateway status
```

### Im Vordergrund ausführen

Um die Logs direkt in deinem Terminal zu sehen (nützlich zum Debuggen):

```bash
openclaw gateway --port 18789
```

### Web-Interface öffnen

```bash
openclaw dashboard
```

Oder öffne direkt im Browser: `http://127.0.0.1:18789/`

## Umgebungsvariablen

Manchmal musst du die Speicherorte anpassen:

- `OPENCLAW_HOME`: Legt das Hauptverzeichnis für OpenClaw fest
- `OPENCLAW_STATE_DIR`: Überschreibt das Status-Datenverzeichnis
- `OPENCLAW_CONFIG_PATH`: Gibt einen benutzerdefinierten Pfad für die Konfigurationsdatei an

## Installation verifizieren

Nach der Installation kannst du:

1. `openclaw --version` ausführen um zu bestätigen, dass es funktioniert
2. `openclaw doctor` ausführen um den Systemzustand zu prüfen
3. Das Web-Interface öffnen und eine Testnachricht senden

## Häufige Probleme

### Node.js-Version prüfen

```bash
node --version
```

Stelle sicher, dass Version 22 oder höher angezeigt wird. Falls nicht, aktualisiere Node.js zuerst.

### Git auf Windows nicht gefunden

Stelle sicher, dass Git installiert und in deinem System-PATH enthalten ist. Du kannst es von git-scm.com herunterladen.

### Berechtigungsfehler

Bei Berechtigungsproblemen versuche:

```bash
sudo npm install -g openclaw@latest
```
