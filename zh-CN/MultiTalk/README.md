# MultiTalk

MultiTalk 是由梅津人工智能团队开发的音频驱动多人对话视频生成框架。该项目能够根据多流音频输入、参考图像和文本提示，生成包含多人交互的高质量视频内容。

## 主要特性

- 音频驱动的多人物对话视频生成
- 支持单人物和多人物场景
- 口型同步准确率业界领先
- 支持卡通角色和唱歌视频生成
- 输出分辨率支持 480P 和 720P
- 最长可生成 15 秒视频片段
