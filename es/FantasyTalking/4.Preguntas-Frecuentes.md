# FantasyTalking - Preguntas Frecuentes

Preguntas frecuentes y respuestas.

## Problemas de instalación

### Q: ¿Error al instalar dependencias?

**Solución:**
1. Actualizar pip: `pip install --upgrade pip`
2. Usar espejo chino:
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```
3. Instalar paquetes fallidos individualmente

### Q: ¿CUDA no disponible?

**Solución:**
1. Confirmar que el controlador NVIDIA está instalado: `nvidia-smi`
2. Instalar versión compatible de PyTorch:
```bash
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
```
3. Verificar disponibilidad de CUDA:
```python
import torch
print(torch.cuda.is_available())
```

### Q: ¿Memoria insuficiente?

**Solución:**
1. Usar imágenes de entrada más pequeñas
2. Reducir tamaño de lote
3. Usar modo CPU (no recomendado, lento)

## Problemas de uso

### Q: ¿Generación fallida, rostro no encontrado?

**Solución:**
1. Usar imagen con rostro claro
2. Asegurar que el rostro esté frontal, sin oclusión
3. Verificar formato de imagen

### Q: ¿Generación de video muy lenta?

**Solución:**
1. Usar aceleración GPU
2. Reducir resolución de salida
3. Reducir frames de salida

### Q: ¿Labios no sincronizados?

**Solución:**
1. Asegurar que el audio sea claro y sin ruido
2. Verificar frecuencia de muestreo de audio
3. Probar diferentes parámetros de sincronización

### Q: ¿Video de salida borroso?

**Solución:**
1. Usar mejora de rostro: `--face_enhancer realusf`
2. Aumentar resolución de salida
3. Usar imagen fuente más clara

### Q: ¿Rostro deformado/distorsionado?

**Solución:**
1. Usar foto frontal sin oclusión
2. Asegurar que solo haya un rostro en la imagen
3. Ajustar parámetros de detección de rostro

## Problemas de modelo

### Q: ¿Descarga de modelo fallida?

**Solución:**
1. Usar proxy o espejo
2. Descargar archivos del modelo manualmente
3. Verificar conexión de red

### Q: ¿Modelo no se cargó?

**Solución:**
1. Confirmar que el archivo del modelo está completo
2. Verificar que la ruta del modelo sea correcta
3. Descargar modelo nuevamente

## Problemas de rendimiento

### Q: ¿Cómo mejorar la velocidad de generación?

**Solución:**
1. Usar GPU
2. Habilitar precisión mixta: `--precision fp16`
3. Usar menor resolución

### Q: ¿Uso de memoria demasiado alto?

**Solución:**
1. Reducir tamaño de lote
2. Usar modelo más pequeño
3. Deshabilitar funciones innecesarias

## Problemas de salida

### Q: ¿Video de salida no se puede reproducir?

**Solución:**
1. Instalar decodificador: `pip install opencv-python`
2. Usar formato compatible: MP4

### Q: ¿Archivo de video demasiado grande?

**Solución:**
1. Reducir resolución
2. Reducir tasa de cuadros
3. Usar mayor compresión
