# Angular 常见问题

常见问题解答。

## 安装问题

### Q: Angular CLI 安装失败？

**解决方案**：
1. 清除 npm 缓存：`npm cache clean --force`
2. 使用管理员权限安装
3. 检查 Node.js 版本是否满足要求

### Q: ng 命令找不到？

**解决方案**：
1. 确认 Angular CLI 已全局安装
2. 检查 PATH 环境变量
3. 尝试重新安装

### Q: Node.js 版本不兼容？

**解决方案**：
1. 使用 nvm 管理 Node.js 版本
2. 或升级 Node.js 到推荐版本

## 使用问题

### Q: 组件样式不生效？

**解决方案**：
1. 检查 styleUrls 路径是否正确
2. 确认 CSS 选择器正确
3. 检查 ViewEncapsulation 设置

### Q: 路由跳转失败？

**解决方案**：
1. 检查路由配置是否正确
2. 确认 RouterModule 已正确导入
3. 检查 routerLink 属性写法

### Q: HTTP 请求失败？

**解决方案**：
1. 确认 HttpClientModule 已导入
2. 检查 CORS 配置
3. 验证 API URL 是否正确

### Q: 表单验证不工作？

**解决方案**：
1. 确认导入 ReactiveFormsModule
2. 检查 Validators 配置
3. 正确使用 formControlName

## 性能问题

### Q: 应用加载慢？

**解决方案**：
1. 启用懒加载
2. 使用 OnPush 变更检测
3. 压缩图片和资源
4. 启用生产模式构建

### Q: 内存泄漏？

**解决方案**：
1. 及时取消订阅
2. 清理定时器
3. 销毁事件监听器

## 版本问题

### Q: 如何升级 Angular？

```bash
ng update @angular/core @angular/cli
```

### Q: AngularJS 迁移到 Angular？

使用 angular-upgrade 工具逐步迁移。
