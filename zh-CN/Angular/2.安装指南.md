# Angular 安装指南

> 完整的环境配置和安装教程

---

## 环境要求

| 环境 | 要求 |
|------|------|
| Node.js | 18.10.0+ |
| npm | 9.0.0+ |
| TypeScript | 5.0+ |

---

## 安装步骤

### 1. 安装 Node.js

从官网下载：https://nodejs.org/

或使用 nvm 管理多版本：

```bash
# 安装 nvm (Linux/macOS)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 安装 Node.js
nvm install 18
nvm use 18
```

### 2. 安装 Angular CLI

```bash
npm install -g @angular/cli
```

### 3. 创建新项目

```bash
ng new my-app
cd my-app
ng serve
```

---

## 开发命令

| 命令 | 说明 |
|------|------|
| ng new | 创建新项目 |
| ng serve | 启动开发服务器 |
| ng build | 构建项目 |
| ng generate | 生成组件/服务/模块等 |
| ng test | 运行单元测试 |
| ng e2e | 运行端到端测试 |
