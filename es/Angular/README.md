# Angular

Plataforma de desarrollo web moderna.

## Resumen

Angular es una plataforma de desarrollo para crear aplicaciones web móviles y de escritorio, desarrollada por Google. Usa TypeScript/JavaScript con arquitectura basada en componentes, enlace bidireccional e inyección de dependencias.

### Características principales

- **Basado en componentes** - Arquitectura dirigida por componentes
- **TypeScript** - Soporte de tipado fuerte
- **Enlace bidireccional** - Sincronización de datos simplificada
- **Inyección de dependencias** - Potente sistema de gestión de dependencias
- **Enrutamiento** - Solución completa de enrutamiento

### Estadísticas

- GitHub Stars: 90,000+
- Millones de desarrolladores en todo el mundo

## Inicio rápido

### Instalar

```bash
npm install -g @angular/cli
```

### Crear proyecto

```bash
ng new my-app
cd my-app
ng serve
```

Visitar: http://localhost:4200
