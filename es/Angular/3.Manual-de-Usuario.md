# Angular Manual de usuario

Conceptos básicos y uso de Angular.

## Componentes

### Crear componente

```bash
ng generate component my-component
# o
ng g c my-component
```

### Estructura del componente

```typescript
import { Component } from '@angular/core';

@Component({
  selector: 'app-my-component',
  templateUrl: './my-component.component.html',
  styleUrls: ['./my-component.component.css']
})
export class MyComponentComponent {
  title = 'Hello Angular';
}
```

## Enlace de datos

### Enlace de propiedad

```html
<img [src]="imageUrl">
```

### Enlace de evento

```html
<button (click)="onClick()">Clic</button>
```

### Enlace bidireccional

```html
<input [(ngModel)]="name">
```

## Servicios

### Crear servicio

```bash
ng generate service my-service
# o
ng g s my-service
```

### Usar servicios

```typescript
import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root'
})
export class MyServiceService {
  getData() {
    return 'datos';
  }
}
```

## Enrutamiento

### Configurar rutas

```typescript
const routes: Routes = [
  { path: '', component: HomeComponent },
  { path: 'about', component: AboutComponent },
  { path: '**', component: NotFoundComponent }
];
```

### Navegación

```html
<a routerLink="/about">Acerca de</a>
<router-outlet></router-outlet>
```

## Solicitudes HTTP

### Importar HttpClient

```typescript
import { HttpClientModule } from '@angular/common/http';
```

### Hacer solicitudes

```typescript
import { HttpClient } from '@angular/common/http';

constructor(private http: HttpClient) {}

getData() {
  return this.http.get('https://api.example.com/data');
}
```

## Formularios

### Formularios reactivos

```typescript
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

form: FormGroup;

constructor(private fb: FormBuilder) {
  this.form = this.fb.group({
    name: ['', Validators.required],
    email: ['', [Validators.required, Validators.email]]
  });
}
```

## Ganchos de ciclo de vida

| Ganchos | Descripción |
|---------|-------------|
| ngOnInit | Inicialización del componente |
| ngOnChanges | Cambio de propiedades de entrada |
| ngOnDestroy | Destrucción del componente |

## Directivas

### Directivas estructurales

```html
<div *ngIf="show">Mostrar</div>
<div *ngFor="let item of items">{{item}}</div>
```

### Directivas de atributo

```html
<div [ngClass]="{'active': isActive}"></div>
```

## Mejores prácticas

1. Usar tipado fuerte de TypeScript
2. Mantener componentes simples
3. Usar servicios para lógica compartida
4. Cancelar suscripciones
5. Usar estrategia de detección de cambios OnPush
