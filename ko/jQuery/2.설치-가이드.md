# jQuery 설치 가이드

## jQuery 얻기

### 최신 버전

현재 최신 버전은 **jQuery 4.0.0**입니다.

#### 프로덕션용 압축 버전 (권장)
- 다운로드: [jquery-4.0.0.min.js](https://code.jquery.com/jquery-4.0.0.min.js)
- 프로덕션에 적합, 작은 파일 크기, 빠른 로딩

#### 개발 디버그 버전
- 다운로드: [jquery-4.0.0.js](https://code.jquery.com/jquery-4.0.0.js)
- 전체 코드 포함, 읽기 및 디버깅 용이

#### Source Map 파일
- 다운로드: [jquery-4.0.0.min.map](https://code.jquery.com/jquery-4.0.0.min.map)
- 브라우저 개발자 도구에서 원본 소스 코드를 표시하는 데 사용

#### Slim 버전
Slim 버전은 Ajax 및 이펙트 모듈을 제외한较小的 버전입니다:
- [jquery-4.0.0.slim.min.js](https://code.jquery.com/jquery-4.0.0.slim.min.js) - 압축
- [jquery-4.0.0.slim.js](https://code.jquery.com/jquery-4.0.0.slim.js) - 비압축

Ajax 및 애니메이션 기능이 필요 없는 경우 Slim 버전을 사용하면 파일 크기를 약 30% 줄일 수 있습니다.

## 설치 방법

### 방법 1: CDN 포함 (권장)

jQuery CDN을 사용하는 것이 가장 간단한 방법입니다:

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
```

Subresource Integrity (SRI) 검증이 있는 버전 (권장, CDN 변조 방지):

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js" 
        integrity="sha384-KJ3o2DKtIkvYIK3UENzmM7KCkRr/rE9/Qpg6aAZGJwFDMVNA/GpGFF93hXpG5KkN" 
        crossorigin="anonymous"></script>
```

전체 버전 목록과 SRI 해시를 얻으려면 [https://releases.jquery.com](https://releases.jquery.com)을 방문하세요.

### 방법 2: npm 설치

Node.js 프로젝트의 경우 npm으로 설치:

```bash
npm install jquery

설치 후 node_modules/jquery/dist/에서 다음을 찾을 수 있습니다:
- jquery-4.0.0.min.js - 압축 버전
- jquery-4.0.0.js - 비압축 버전
- jquery-4.0.0.min.map - Source Map
```

### 방법 3: Yarn 설치

```bash
yarn add jquery
```

### 방법 4: 로컬 다운로드

[jQuery 다운로드 페이지](https://jquery.com/download/)를 방문하여 필요한 버전을 마우스 오른쪽 단추로 클릭하고 "다른 이름으로 저장..."을 선택하여 로컬 프로젝트 디렉토리에 파일을 저장하세요.

## 다른 CDN 서비스

공식 CDN 외에도 다음 CDN도 jQuery를 제공합니다:

### Google CDN
```
https://ajax.googleapis.com/ajax/libs/jquery/4.0.0/jquery.min.js
```

### Microsoft CDN
```
https://ajax.aspnetcdn.com/ajax/jquery/jquery-4.0.0.min.js
```

### CDNJS
```
https://cdnjs.cloudflare.com/ajax/libs/jquery/4.0.0/jquery.min.js
```

### jsDelivr
```
https://cdn.jsdelivr.net/npm/jquery@4.0.0/dist/jquery.min.js
```

참고: 이 타사 CDN은 약간의 지연이 있을 수 있으며 새 릴리스가 동기화되는 데 시간이 걸릴 수 있습니다.

## jQuery 업그레이드

### jQuery Migrate 플러그인 사용

jQuery를 업그레이드할 때 호환성 문제를 식별하기 위해 jQuery Migrate 플러그인을 사용하는 것이 좋습니다:

1. 이전 버전에서 1.9+ 또는 3.x로 업그레이드: [jQuery Migrate 1.4.1](https://code.jquery.com/jquery-migrate-1.4.1.min.js) 사용

2. jQuery 3.x에서 최신 버전으로 업그레이드: [jQuery Migrate 3.6.0](https://code.jquery.com/jquery-migrate-3.6.0.min.js) 사용

사용 방법: jQuery 다음에 Migrate 플러그인 포함

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
<script src="https://code.jquery.com/jquery-migrate-3.6.0.min.js"></script>
```

Migrate는 문제를 단계별로 해결할 수 있도록 콘솔에 호환성 경고를 표시합니다.
