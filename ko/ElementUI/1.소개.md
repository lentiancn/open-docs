# Element UI 소개

Element UI는 개발자에게 풍부한 데스크톱 컴포넌트 솔루션을 제공하는 Vue 2.0 기반 UI 컴포넌트 라이브러리입니다.

## Element UI란?

Element UI는 Ele.me 팀에서 개발하고 유지 관리하는 Vue 2.0 컴포넌트 라이브러리입니다.

## 주요 기능

### 완전한 컴포넌트 시스템

Element UI는 70개 이상의 고품질 컴포넌트를 제공합니다:

- 기본 컴포넌트: 버튼, 아이콘, 레이아웃, 컨테이너
- 폼 컴포넌트: 입력, 선택기, 날짜 선택기, 스위치
- 데이터 표시: 테이블, 트리, 페이지네이션, 태그
- 피드백 컴포넌트: 다이얼로그, 메시지, 로딩
- 네비게이션 컴포넌트: 메뉴, 탭, 브래드크럼

### 완전한 Vue 2.0 지원

Element UI는 Vue 2.0 기반으로 완전히 구축되었습니다.

### 온디맨드 임포트

babel-plugin-component 플러그인 사용.

### 국제화 지원

다국어 지원 기본 제공.

### 테마 사용자 정의

전역 테마 사용자 정의 지원.

## 디자인 원칙

- 일관성: 실제생활과인터페이스와 일관성
- 피드백: 제어 페이지 피드백 피드백과
- 효율성: 단순화된 프로세스
- 제어 가능성: 사용자 결정

## 적용 시나리오

- 기업 내부 관리 시스템
- 전자상거래 백엔드 시스템
- 모든 Vue 2 프로젝트
