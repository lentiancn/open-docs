# Bootstrap 5 使用指南

本指南詳細介紹如何使用 Bootstrap 5 建構響應式網頁。

## 快速開始

### 基本模板

```html
<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 5 頁面</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <!-- 內容區域 -->
    <div class="container mt-5">
        <h1>你好，Bootstrap 5！</h1>
        <p class="lead">這是一個響應式頁面。</p>
    </div>

    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## 佈局系統

### 容器

Bootstrap 5 提供兩種容器類：

| 類名 | 描述 |
|------|------|
| `.container` | 固定寬度容器 |
| `.container-fluid` | 全寬度容器 |

```html
<!-- 固定寬度容器 -->
<div class="container">
    <p>內容居中，兩側有留白</p>
</div>

<!-- 全寬度容器 -->
<div class="container-fluid">
    <p>佔滿整個寬度</p>
</div>
```

### 柵格系統

Bootstrap 使用 12 列柵格系統：

```html
<div class="container">
    <div class="row">
        <div class="col-12 col-md-6 col-lg-4">
            <p>響應式列</p>
        </div>
        <div class="col-12 col-md-6 col-lg-4">
            <p>響應式列</p>
        </div>
        <div class="col-12 col-md-6 col-lg-4">
            <p>響應式列</p>
        </div>
    </div>
</div>
```

#### 柵格斷點

| 斷點 | 類名前綴 | 寬度 |
|------|----------|------|
| 超小 | col- | < 576px |
| 小 | col-sm- | ≥ 576px |
| 中 | col-md- | ≥ 768px |
| 大 | col-lg- | ≥ 992px |
| 特大 | col-xl- | ≥ 1200px |
| 超大 | col-xxl- | ≥ 1400px |

### Flex 佈局

```html
<div class="d-flex justify-content-between">
    <div>左側</div>
    <div>右側</div>
</div>

<div class="d-flex align-items-center">
    <div>垂直居中</div>
</div>
```

### 工具提示 (Tooltip)

```html
<button type="button" class="btn btn-secondary" data-bs-toggle="tooltip" data-bs-title="工具提示內容">
    懸停查看
</button>

<!-- 需要初始化 -->
<script>
    var tooltipTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="tooltip"]'))
    var tooltipList = tooltipTriggerList.map(function (tooltipTriggerEl) {
        return new bootstrap.Tooltip(tooltipTriggerEl)
    })
</script>
```

## JavaScript 組件

### 初始化

```javascript
// 啟用所有工具提示
var tooltipList = [].slice.call(document.querySelectorAll('[data-bs-toggle="tooltip"]'))
var tooltipList = tooltipList.map(function (tooltipTriggerEl) {
    return new bootstrap.Tooltip(tooltipTriggerEl)
})

// 啟用所有彈出框
var popoverTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="popover"]'))
var popoverList = popoverTriggerList.map(function (popoverTriggerEl) {
    return new bootstrap.Popover(popoverTriggerEl)
})

// 模態框控制
var myModal = new bootstrap.Modal(document.getElementById('myModal'))
myModal.show()     // 顯示
myModal.hide()     // 隱藏

// 輪播控制
var carousel = new bootstrap.Carousel(document.getElementById('myCarousel'))
carousel.next()     // 下一張
carousel.prev()    // 上一張
carousel.cycle()   // 開始自動播放
carousel.pause()   // 暫停
```

### 事件

```javascript
// 模態框事件
var myModalEl = document.getElementById('myModal')
myModalEl.addEventListener('show.bs.modal', function (event) {
    console.log('模態框即將顯示')
})
myModalEl.addEventListener('shown.bs.modal', function (event) {
    console.log('模態框已顯示')
})
myModalEl.addEventListener('hide.bs.modal', function (event) {
    console.log('模態框即將隱藏')
})
myModalEl.addEventListener('hidden.bs.modal', function (event) {
    console.log('模態框已隱藏')
})
```

## 最佳實踐

### 1. 使用語義化 HTML

```html

<div class="container">
    <div class="row">
        <div class="col">
            <div class="btn btn-primary">按鈕</div>
        </div>
    </div>

    <section class="row">

    </section>

```


## 常見問題

### 如何居中元素？

```html
<!-- 水平居中 -->
<div class="text-center">文本居中</div>
<div class="mx-auto">塊級元素居中 (需要 width)</div>

<!-- 垂直居中 -->
<div class="d-flex align-items-center" style="height: 100px;">
    <div>垂直居中</div>
</div>

<!-- 完全居中 -->
<div class="d-flex justify-content-center align-items-center" style="height: 100px;">
    <div>完全居中</div>
</div>
```

### 如何製作響應式表格？

```html

                <td>內容</td>
                <td>內容</td>
                <td>內容</td>

```
