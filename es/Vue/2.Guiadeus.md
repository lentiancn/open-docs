# Vue Guía de Uso

Esta guía cubre cómo usar Vue.js de manera efectiva en sus aplicaciones.

## Conceptos básicos

### Instancia Vue

```javascript
import { createApp } from 'vue'

const app = createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
})

app.mount('#app')
```

### Sintaxis de plantilla

```html
<!-- Interpolación de texto -->
<h1>{{ message }}</h1>

<!-- Directivas -->
<div v-if="show">Contenido</div>
<div v-for="item in items">{{ item }}</div>
<button @click="handleClick">Clic</button>
<input v-model="inputValue">
```

## Reactividad

### Vue 3 Composition API (Recomendado)

```javascript
import { ref, reactive, computed } from 'vue'

export default {
  setup() {
    // ref para valores primitivos
    const count = ref(0)
    
    // reactive para objetos
    const state = reactive({
      name: 'Vue',
      version: 3
    })
    
    // Propiedades calculadas
    const doubleCount = computed(() => count.value * 2)
    
    // Métodos
    const increment = () => {
      count.value++
    }
    
    return { count, state, doubleCount, increment }
  }
}
```

### Vue 2 Options API

```javascript
export default {
  data() {
    return {
      count: 0,
      name: 'Vue',
      version: 2
    }
  },
  computed: {
    doubleCount() {
      return this.count * 2
    }
  },
  methods: {
    increment() {
      this.count++
    }
  }
}
```

## Componentes

### Crear un componente

```vue
<!-- MyComponent.vue -->
<template>
  <div class="my-component">
    <h2>{{ title }}</h2>
    <p>{{ content }}</p>
    <button @click="emitEvent">Haz clic</button>
  </div>
</template>

<script setup>
import { defineProps, defineEmits } from 'vue'

const props = defineProps({
  title: {
    type: String,
    required: true
  },
  content: {
    type: String,
    default: 'Contenido predeterminado'
  }
})

const emit = defineEmits(['click'])

const emitEvent = () => {
  emit('click', '¡Hola desde el componente!')
}
</script>

<style scoped>
.my-component {
  padding: 20px;
}
</style>
```

### Usar componentes

```vue
<!-- ParentComponent.vue -->
<template>
  <MyComponent 
    title="Mi título"
    content="Contenido personalizado"
    @click="handleClick"
  />
</template>

<script setup>
import MyComponent from './MyComponent.vue'

const handleClick = (message) => {
  console.log(message)
}
</script>
```

## Comunicación entre componentes

### Props (Padre a hijo)

```vue
<!-- Child.vue -->
<script setup>
defineProps({
  message: String,
  count: Number
})
</script>

<template>
  <p>{{ message }} - {{ count }}</p>
</template>

<!-- Parent.vue -->
<template>
  <Child message="Hola" :count="5" />
</template>
```

### Eventos (Hijo a padre)

```vue
<!-- Child.vue -->
<script setup>
const emit = defineEmits(['update'])

const sendData = () => {
  emit('update', { name: 'Vue' })
}
</script>

<template>
  <button @click="sendData">Enviar</button>
</template>

<!-- Parent.vue -->
<template>
  <Child @update="handleUpdate" />
</template>

<script setup>
const handleUpdate = (data) => {
  console.log(data)
}
</script>
```

### Provide/Inject

```vue
<!-- Parent.vue -->
<script setup>
import { provide } from 'vue'

provide('theme', 'dark')
provide('user', { name: 'John' })
</script>

<!-- Child.vue -->
<script setup>
import { inject } from 'vue'

const theme = inject('theme')
const user = inject('user')
</script>
```

```html
<div v-else>Predeterminado</div>
```

## Renderizado de listas

```html
<!-- v-for con array -->
<ul>
  <li v-for="(item, index) in items" :key="item.id">
    {{ index }}. {{ item.name }}
  </li>
</ul>

<!-- v-for con objeto -->
<div v-for="(value, key) in object" :key="key">
  {{ key }}: {{ value }}
</div>
```

## Manejo de eventos

```html
<!-- Evento de clic -->
<button @click="handleClick">Clic</button>

<!-- Objeto de evento -->
<button @click="handleClick($event)">Clic</button>

<!-- Múltiples manejadores -->
<button @click="handler1(), handler2()">Clic</button>

<!-- Modificadores de eventos -->
<button @click.stop="handleClick">Detener propagación</button>
<button @click.prevent="handleSubmit">Prevenir acción por defecto</button>

<!-- Modificadores de teclado -->
<input @keyup.enter="submit" />
<input @keyup.ctrl.enter="submit" />
```

## Enlace de entrada de formulario

```vue
<template>
  <form @submit.prevent="submitForm">
    <!-- Entrada de texto -->
    <input v-model="form.name" type="text" />
    
    <!-- Casilla de verificación -->
    <input v-model="form.checked" type="checkbox" />
    
    <!-- Radio -->
    <input v-model="form.gender" value="male" type="radio">Masculino
    <input v-model="form.gender" value="female" type="radio">Femenino
    
    <!-- Select -->
    <select v-model="form.country">
      <option value="cn">China</option>
      <option value="us">EE.UU.</option>
    </select>
    
    <button type="submit">Enviar</button>
  </form>
</template>

<script setup>
import { reactive } from 'vue'

const form = reactive({
  name: '',
  checked: false,
  gender: '',
  country: ''
})

const submitForm = () => {
  console.log(form)
}
</script>
```

## Propiedades calculadas

```javascript
import { ref, computed } from 'vue'

const firstName = ref('John')
const lastName = ref('Doe')

// Computed - en caché, solo reevalúa cuando cambian las dependencias
const fullName = computed(() => {
  return `${firstName.value} ${lastName.value}`
})
```

## Observadores

```javascript
import { ref, watch } from 'vue'

const count = ref(0)

// Observador simple
watch(count, (newValue, oldValue) => {
  console.log(`Cambió de ${oldValue} a ${newValue}`)
})

// Observador profundo
const state = ref({ name: 'Vue' })
watch(state, (newValue) => {
  console.log('Estado cambiado:', newValue)
}, { deep: true })
```

## Ganchos de ciclo de vida

### Vue 3 Composition API

| Gанчо | Descripción |
|--------|-------------|
| `setup()` | Llamado antes que otros ganchos |
| `onMounted()` | Después de que el componente se monta |
| `onUpdated()` | Después de que el componente se actualiza |
| `onUnmounted()` | Después de que el componente se desmonta |

```javascript
import { onMounted, onUpdated, onUnmounted } from 'vue'

export default {
  setup() {
    onMounted(() => {
      console.log('¡Montado!')
    })
    onUpdated(() => {
      console.log('¡Actualizado!')
    })
    onUnmounted(() => {
      console.log('¡Desmontado!')
    })
  }
}
```

### Vue 2 Options API

| Ganchos | Descripción |
|---------|-------------|
| `beforeCreate` | Antes de crear la instancia |
| `created` | Después de crear la instancia |
| `beforeMount` | Antes de montar |
| `mounted` | Después de montar |
| `beforeUpdate` | Antes de actualizar |
| `updated` | Después de actualizar |
| `beforeDestroy` | Antes de destruir |
| `destroyed` | Después de destruir |

## Vue Router

### Configuración básica

```javascript
// router/index.js
import { createRouter, createWebHistory } from 'vue-router'
import Home from '../views/Home.vue'
import About from '../views/About.vue'

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```

### Navegación

```vue
<template>
  <!-- Navegación programática -->
  <button @click="$router.push('/about')">Acerca de</button>
  
  <!-- Router link -->
  <router-link to="/">Inicio</router-link>
  <router-link to="/about">Acerca de</router-link>
</template>
```

## Pinia Gestión de estado

### Definir store

```javascript
// stores/user.js
import { defineStore } from 'pinia'

export const useUserStore = defineStore('user', {
  state: () => ({
    name: 'John',
    email: 'john@example.com'
  }),
  
  getters: {
    displayName: (state) => `${state.name} (${state.email})`
  },
  
  actions: {
    updateName(newName) {
      this.name = newName
    }
  }
})
```

### Usar store

```vue
<script setup>
import { useUserStore } from '../stores/user'

const userStore = useUserStore()

// Acceder al estado directamente
console.log(userStore.name)

// Llamar acciones
userStore.updateName('Jane')
</script>
```

```vue
  }
```

```vue

// Sintaxis más concisa
```

```javascript

// Bueno - computed
```
