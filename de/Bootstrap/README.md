# Bootstrap

Bootstrap ist das weltweit beliebteste Open-Source-Framework für die Frontend-Entwicklung mit HTML, CSS und JavaScript, entwickelt und gepflegt von Twitter. Es bietet eine Fülle von Komponenten und leistungsstarken JavaScript-Plugins, mit denen Entwickler schnell responsive Web-Anwendungen erstellen können.

## Schnellstart

### CDN einbinden

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### npm Installation

```bash
npm install bootstrap@5
```

## Kernfunktionen

- **Responsives Layout** – 12-Spalten-Grid-System für alle Bildschirme
- **Umfangreiche Komponenten** – Navbar, Cards, Formulare, Modals und mehr
- **JavaScript-Plugins** – Dropdowns, Karussells, Accordions und mehr
- **Sass-Unterstützung** – Einfache Theme-Anpassung
