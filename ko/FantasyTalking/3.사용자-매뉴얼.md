# FantasyTalking - 사용자 매뉴얼

상세 사용 가이드.

## 기본 사용법

### 비디오 생성

```bash
python inference.py --source_image face.jpg --audio speech.wav --output output.mp4
```

### 매개변수

| 매개변수 | 설명 |
|----------|------|
| --source_image | 입력 이미지 |
| --audio | 오디오 파일 |
| --output | 출력 경로 |
| --face_enhancer | 얼굴 향상 |
| --fps | 초당 프레임 |

## Python API

```python
from fantasytalking import FantasyTalking

ft = FantasyTalking()
ft.generate(
    source_image="face.jpg",
    audio="speech.wav",
    output="output.mp4"
)
```

## 일괄 처리

```python
ft.batch_generate(image_list, audio_list, output_dir)
```

## 입력 요구사항

### 이미지

- 형식: JPG, PNG
- 해상도: 512x512+
- 얼굴: 정면, 선명

### 오디오

- 형식: WAV, MP3
- 길이: 1-60초
