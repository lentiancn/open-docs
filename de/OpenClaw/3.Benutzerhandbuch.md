# OpenClaw Benutzerhandbuch

## Grundkonzepte

### Gateway

Das Herzstück. Verwaltet Nachrichten, KI-Aufrufe und Kanalverbindungen.

### Kanäle

Chat-Apps: WhatsApp, Telegram, Discord, etc.

### Agenten

Unabhängige KI-Assistenten-Instanzen. Mehrere möglich.

## Operationen

### Start/Stopp

```bash
openclaw gateway start
openclaw gateway stop
openclaw gateway status
```

### Nachrichten senden

```bash
openclaw message send --target +15555550123 --message "Hallo"
```

### Konfiguration

Datei: `~/.openclaw/openclaw.json`

Methoden:
- `openclaw configure`
- `openclaw config set xxx`
- Direkt bearbeiten
- Web UI: http://127.0.0.1:18789/

## Probleme

```bash
openclaw logs
openclaw doctor
```
