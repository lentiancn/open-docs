# Claude Code

[Claude Code](https://code.claude.com/) 是由 Anthropic 公司开发的智能编程助手，能够读取代码库、编辑文件、执行命令，并与开发工具无缝集成。

## 概述

Claude Code 是一款 AI 驱动的编程助手，支持在终端、IDE、桌面应用和浏览器等多种环境中使用。它能够帮助你构建功能、修复 bug 和自动化开发任务。

## 功能特性

- **自动化任务**：编写测试、修复 lint 错误、解决合并冲突、更新依赖
- **功能开发**：用自然语言描述需求，Claude Code 会规划方案并实现
- **Git 集成**：暂存更改、编写提交信息、创建分支、打开 Pull Request
- **MCP 集成**：连接外部数据源和工具
- **自定义能力**：CLAUDE.md、自定义命令、Hooks
- **多代理协作**：并行处理复杂任务

## 快速开始

### 安装

**macOS、Linux、WSL：**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell：**
```powershell
irm https://claude.ai/install.ps1 | iex
```

### 启动

```bash
cd your-project
claude
```
