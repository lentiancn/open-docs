# Hadoop Introduction

> Hadoop est une plateforme de calcul distribué open source

---

## Vue d'ensemble

Hadoop fournit stockage distribué (HDFS) et calcul distribué (MapReduce).

## Composants

- **HDFS**: Système de fichiers distribué
- **MapReduce**: Framework de calcul
- **YARN**: Gestionnaire de ressources

## Caractéristiques

- Haute fiabilité
- Haute évolutivité
- Haute efficacité
- Faible coût
