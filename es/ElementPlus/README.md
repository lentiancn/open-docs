# Element Plus

Element Plus es una biblioteca de componentes para Vue 3.

## Características

- Componentes completos
- Soporte TypeScript
- Temas personalizables
- Internacionalización

## Versiones

| Versión | Estado |
|---------|--------|
| 2.x | ✅ Actual |
| Element UI | ❌ Legacy |

## Inicio Rápido

```bash
npm install element-plus
```
