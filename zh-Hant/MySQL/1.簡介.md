# MySQL 簡介

## 什麼是 MySQL

MySQL 是全球最受歡迎的開源資料庫。憑藉其經過驗證的效能、可靠性和易用性，MySQL 已成為 Web 應用程式的首選資料庫解決方案，被 Facebook、Twitter、YouTube、Yahoo 等眾多知名網站廣泛採用。

Oracle 推動 MySQL 的持續創新，為下一代 Web、雲端、行動和嵌入式應用程式提供強大的新功能。

## MySQL 的主要特點

### 開源免費
MySQL Community Edition 是免費的開源版本，採用 GNU 通用公共許可證（GPL）授權，由龐大的開源開發者社群提供支援。

### 跨平台支援
MySQL 支援 Windows、Linux、macOS、FreeBSD、Solaris 等多種作業系統，能夠在各種硬體和軟體環境中執行。

### 高效能
MySQL 採用最佳化的儲存引擎架構，支援多執行緒執行，能夠處理大規模資料和高並行請求。

### 可靠性
MySQL 提供完善的事務支援（ACID 相容），包括提交、復原、崩潰復原和資料列級鎖定等功能，確保資料完整性。

### 易於使用
MySQL 擁有簡潔的管理工具和豐富的檔獻，学习曲线平缓，適合各類開發者使用。

## MySQL 產品版本

### MySQL Community Edition
免費的開源版本，適合開發環境和生產環境，由活躍的開源社群提供支援。

### MySQL Enterprise Edition
企業級版本，提供進階功能、管理工具和技術支援，適用於業務關鍵型應用。

### MySQL HeatWave
整合生成式 AI 和機器學習功能的雲端服務，同時支援事務處理和湖倉級分析。

### MySQL NDB Cluster
即時開源事務資料庫，專為高吞吐量條件下的持續可用性存取而設計。

## MySQL 應用領域

MySQL 廣泛應用於以下場景：

- 網站和 Web 應用程式後端資料庫
- 電子商務平台
- 資料倉儲和分析
- 日誌系統
- 內容管理系統
- 行動應用程式後端
- 嵌入式系統

## MySQL 生態系統

MySQL 生態系統包含豐富的工具和產品：

- **MySQL Server**：核心資料庫伺服器
- **MySQL Workbench**：視覺化資料庫設計和管理工具
- **MySQL Shell**：命令列互動工具
- **MySQL Router**：資料庫路由中介軟體
- **MySQL Connectors**：各種程式語言的資料庫驅動程式

## 許可證說明

MySQL Community Edition 採用 GPL 許可證開源免費使用。如需將 MySQL 整合到專有應用程式中，可選擇 MySQL Enterprise Edition 商業許可。
