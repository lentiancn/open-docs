# よくある質問

このドキュメントでは、Apifox を使っている途中で出会うことが多い技術的な質問とその回答をまとめています。

## 基本的な質問

### Apifox とは何ですか？

Apifox は API  協業プラットフォームです。API 設計、ドキュメント管理、デバッグ、Mock、自動化テストなどの機能を一本化しています。開発チームが API の開発、テスト、ドキュメント作成をより効率的に行えるようにサポートします。

### Apifox は無料ですか？

Apifox には無料版があります。無料版でも全てのコア機能が使えて、一般的な開発ニーズには十分に対応しています。有料版では、より大きなチーム规模や高度な自動化機能などが使えます。

### Apifox と Postman の違いは？

どちらも API ツールですが、Apifox の方が国内ユーザーに向いています：中文サポートがより充実していること、Mock 機能がよりスマートであること、チームコラボレーション体験がより洗練されていることです。Apifox の基本機能は完全無料ですが、Postman は多くの機能が有料が必要です。

## インストールに関する質問

### ダウンロードが遅いです…

Apifox 官网では多个のダウロードノードを用意しています。https://apifox.com/ にアクセスして、あなたに一番近いノードを選んでダウンロードしてみてください。

### 起動できません…

1. お使いの OS のバージョンが要件を満たしているか確認
2. 管理者権限で実行してみる
3. 必要なランタイムライブラリ是否インストールされているか確認
4. エラーログを見て詳細を確認

### ログインに失敗します…

1. インターネット接続是否正常か確認
2. アカウントとパスワード是否正しいか確認
3. 第三方アカウントでログインを試みる
4. 官方サポートに連絡して助けを求める

## 使い方に関する質問

### 接口データが同期されません…

1. アカウントにログインしているか確認
2. インターネット接続是否正常か確認
3. 「同期」ボタンをクリックして手動で同期
4. アカウントの権限を確認

### Mock データが反映されません…

1. Mock 機能がオンになっているか確認
2. Mock のアドレス是否正しいか確認
3. 接口に语法エラーがないか確認
4. ページを更新してみる

### リクエスト送信に失敗します…

1. インターネット接続是否正常か確認
2. リクエストアドレス 是否正しいか確認
3. リクエストパラメータの形式 是否正しいか確認
4. プロキシに拦截されていないか確認

### レスポンスが文字化けしています…

1. レスポンスのエンコーディングを変えてみる
2. レスポンスヘッダー内のエンコーディング設定を確認
3. 「整形」機能を使って再解析

## 機能に関する質問

### Postman のデータをインポートするには？

1. 「インポート」をクリック
2. 「Postman」を選択
3. インポートするファイルを選択
4. インポートを実行

### Swagger 形式にエクスポートするには？

1. プロジェクト設定に入る
2. 「エクスポート」を選択
3. 「Swagger/OpenAPI」を選択
4. バージョンを選択
5. エクスポートを実行

### 環境変数の使い方は？

1. プロジェクト設定で環境変数を定義
2. リクエストでは {{変数名}} で参照
3. 上部の環境セレクターで環境を切り替え

### 接口を一括操作するには？

1. Ctrl/Cmd を押しながら複数の接口を選択
2. 右クリックして一括操作を選択
3. 操作の種類を選択（削除、エクスポート、移动など）

## テストに関する質問

### 自動化テストの実行方法は？

1. テストスイートを開く
2. 「実行」ボタンをクリック
3. テスト結果を確認

### アサーションが失敗しました…

1. 失敗の原因を確認
2. 期待値と实际値を比較
3. アサーションの条件を変更
4. テストを再実行

### テストレポートをエクスポートするには？

1. テスト実行完了後
2. 「レポートをエクスポート」をクリック
3. 形式を選択（HTML、PDF、JSON）

## チームコラボレーションに関する質問

### チームメンバーを招待するには？

1. プロジェクト設定に入る
2. 「メンバー管理」をクリック
3. 「メンバー招待」をクリック
4. メンバーのメールアドレスまたはユーザー名を入力

### メンバーの権限設定は？

メンバーを招待する際に、異なる役割を設定できます：

- 管理者：完全なコントロール
- 編集者：編集と実行が可能
- 閲覧者：閲覧のみ

### 競合が発生した場合は？

複数の人が同時に同じ接口を編集している場合：

1. システムが競合を通知
2. ローカル版かリモート版するかを選択
3. または手動で変更をマージ

## データに関する質問

### データのバックアップ方法は？

1. 定期的にプロジェクトを Apifox 形式でエクスポート
2. またはクラウド同期機能を使用

### データを復元するには？

1. バックアップファイルがある場合、「インポート」をクリック
2. バックアップファイルを選択して復元

### 削除したデータは復元できますか？

削除したデータはゴミ箱に30日間 保存されます。ゴミ箱から復元できます。

## パフォーマンスに関する質問

### 動作が重いです…

1. 使っていないプロジェクトを閉じる
2. ブラウザのキャッシュをクリア
3. ハードウェア構成をアップグレード
4. 最新バージョンを試す

### 大量データのインポートが遅いです…

1. 分割してインポート
2. 自動同期をオフ
3. インポートが完了するのを辛抱強く待つ

## その他の質問

### 問題を報告するには？

1. 公式フォーラム：https://community.apifox.cn/
2. 公式サポート
3. GitHub Issue

### 更新ログはどこで確認できますか？

Apifox 官网にアクセスするか、公式公众号をフォローして最新の更新情報をチェックしてください。

### デスクトップウィジェットはありますか？

Apifox はデスクトップウィジェット功能に対応しています。デスクトップからよく使う接口に素早くアクセスできます。
