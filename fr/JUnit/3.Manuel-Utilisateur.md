# Manuel utilisateur JUnit 5

## Test basique

```java
@Test
void testSimple() {
    assertEquals(4, 2 + 2);
}
```

## Assertions

```java
assertEquals(4, 2 + 2);
assertTrue(5 > 3);
assertThrows(Exception.class, () -> { });
```

## Tests paramétrés

```java
@ParameterizedTest
@ValueSource(ints = {1, 2, 3})
void estPositif(int n) {
    assertTrue(n > 0);
}
```

## Tests imbriqués

```java
@Nested
class TestsInternes {
    @Test
    void test() { }
}
```

## Bonnes pratiques

1. Utiliser @DisplayName pour noms descriptifs
2. Une assertion par test
3. Tests indépendants
