# Nacos 개요

Nacos(동적 명명 및 구성 서비스)는 Alibaba에서 개발한 서비스 발견 및 동적 구성 관리 플랫폼입니다.

## 주요 기능

- 서비스 디스커버리
- 구성 관리
- 동적 DNS
- 상태 확인

## 설치

```bash
# 독립 모드
./startup.sh -m standalone

# Docker
docker run --name nacos -d -p 8848:8848 nacos/nacos-server:v2.3.0
```
