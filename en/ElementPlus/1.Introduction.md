# Element Plus Introduction

Element Plus is a Vue 3-based UI component library that provides developers with rich desktop component solutions.

## What is Element Plus

Element Plus is the Vue 3 version of Element UI, developed and maintained by the Ele.me team. It provides a complete set of design specifications and component libraries to help developers quickly build desktop applications.

As the Vue 3 companion UI framework, Element Plus fully utilizes Vue 3's Composition API, providing better TypeScript support and more flexible component encapsulation.

## Main Features

### Complete Component System

Element Plus provides over 70 high-quality components covering various aspects of desktop application development:

- **Basic Components**: Buttons, Icons, Layout, Containers
- **Form Components**: Input, Select, Date Picker, Switch, Slider
- **Data Display**: Table, Tree, Pagination, Tags
- **Feedback Components**: Dialog, Drawer, Message, Loading
- **Navigation Components**: Menu, Tabs, Breadcrumb, Steps

### Full Vue 3 Support

Element Plus is built entirely on Vue 3, fully utilizing its new features:

- Components built with Composition API
- Complete TypeScript type support
- Better performance optimization
- Support for Vue 3's reactive system

### On-Demand Import

Element Plus supports Tree Shaking, allowing you to import only the components you use, effectively reducing project size:

- Auto-import plugin support
- Manual on-demand import
- Complete ES Module support

### Internationalization Support

Built-in support for over 40 language packages, meeting the needs of internationalized application development.

### Theme Customization

Supports global theme customization, easily adjusting the overall style of components:

- Theme color configuration
- Component style override
- CSS variable support

## Design Principles

Element Plus follows four major design principles:

### Consistency

- Consistent with real life: Keep consistent with real-life processes and logic, following user habits in language and concepts
- Consistent in the interface: All elements and structures should be consistent, including design styles, icons and text, element positions, etc.

### Feedback

- Control feedback: Let users clearly perceive their operations through interface styles and interactive effects
- Page feedback: After operations, clearly show the current state through changes in page elements

### Efficiency

- Simplify processes: Design simple and intuitive operation processes
- Clear and explicit: Clear language expressions that let users quickly understand and make decisions
- Help users recognize: Simple and straightforward interfaces let users quickly recognize rather than remember, reducing memory burden

### Controllability

- User decisions: Give users operation suggestions or safety prompts based on scenarios, but don't make decisions for them
- Controllable results: Users can freely perform operations, including undo, rollback, and terminating current operations

## Applicable Scenarios

Element Plus is suitable for the following scenarios:

- Enterprise internal management systems
- E-commerce backend management systems
- Data visualization platforms
- CRM customer management systems
- Any Vue 3 project requiring desktop UI

## Relationship with Element UI

Element Plus is the spiritual successor to Element UI (Vue 2 version):

- Element UI: Designed for Vue 2
- Element Plus: Redesigned for Vue 3

If your project is still using Vue 2, consider upgrading to Vue 3 and migrating to Element Plus.
