# Benutzerhandbuch

Dieses Handbuch beschreibt die Kernfunktionen und die Verwendung von Angular im Detail – einschließlich Komponenten, Services, Routing, Direktiven, Pipes und Formularen.

## Komponentenentwicklung

### Komponente erstellen

```bash
ng generate component user-list
```

Oder manuell erstellen:

```typescript
import { Component } from '@angular/core';

@Component({
  selector: 'app-user-list',
  templateUrl: './user-list.component.html',
  styleUrls: ['./user-list.component.scss']
})
export class UserListComponent {
  users = ['Max', 'Anna', 'Tom'];
}
```

### Template-Syntax

**Interpolation:**

```html
<h1>Hallo, {{ name }}</h1>
```

**Attributbindung:**

```html
<img [src]="imageUrl">
<button [disabled]="isDisabled">Klicken</button>
```

**Ereignisbindung:**

```html
<button (click)="onSave()">Speichern</button>
```

**Two-Way-Binding:**

```html
<input [(ngModel)]="username">
```

### Lebenszyklus-Hooks

## Services und Dependency Injection

### Service erstellen

```bash
ng generate service user
```

Service-Code:

```typescript
import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class UserService {
  constructor(private http: HttpClient) {}

  getUsers(): Observable<User[]> {
    return this.http.get<User[]>('/api/users');
  }
}
```

### Service verwenden

```typescript
import { Component, OnInit } from '@angular/core';
import { UserService } from './user.service';

@Component({
  selector: 'app-user-list',
  templateUrl: './user-list.component.html'
})
export class UserListComponent implements OnInit {
  users: User[] = [];

  constructor(private userService: UserService) {}

  ngOnInit() {
    this.userService.getUsers().subscribe(data => {
      this.users = data;
    });
  }
}
```

## Pipes

### Integrierte Pipes

```html
<!-- Datums-Pipe -->
<p>{{ today | date:'yyyy-MM-dd' }}</p>

<!-- Groß-/Kleinschreibung -->
<p>{{ name | uppercase }}</p>
<p>{{ name | lowercase }}</p>

<!-- Währungs-Pipe -->
<p>{{ price | currency:'EUR' }}</p>

<!-- Dezimal-Pipe -->
<p>{{ pi | number:'1.2-2' }}</p>

<!-- JSON-Pipe -->
<pre>{{ obj | json }}</pre>
```

### Eigene Pipes erstellen

```bash
ng generate pipe capitalize
```

```typescript
import { Pipe, PipeTransform } from '@angular/core';

@Pipe({
  name: 'capitalize'
})
export class CapitalizePipe implements PipeTransform {
  transform(value: string): string {
    if (!value) return value;
    return value.charAt(0).toUpperCase() + value.slice(1);
  }
}
```

## Formularverarbeitung

### Reaktive Formulare

```typescript
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

export class LoginComponent implements OnInit {
  loginForm: FormGroup;

  constructor(private fb: FormBuilder) {}

  ngOnInit() {
    this.loginForm = this.fb.group({
      username: ['', Validators.required],
      password: ['', [Validators.required, Validators.minLength(6)]],
      remember: [false]
    });
  }

  onSubmit() {
    if (this.loginForm.valid) {
      console.log(this.loginForm.value);
    }
  }
}
```

Template:

```html
<form [formGroup]="loginForm" (ngSubmit)="onSubmit()">
  <input formControlName="username" placeholder="Benutzername">
  <div *ngIf="loginForm.get('username').touched && loginForm.get('username').errors?.required">
    Benutzername ist erforderlich
  </div>
  
  <input type="password" formControlName="password" placeholder="Passwort">
  
  <label>
    <input type="checkbox" formControlName="remember"> Angemeldet bleiben
  </label>
  
  <button type="submit" [disabled]="loginForm.invalid">Anmelden</button>
</form>
```

### Templategetriebene Formulare

```html
<form #form="ngForm" (ngSubmit)="onSubmit(form)">
  <input name="username" [(ngModel)]="username" required #name="ngModel" placeholder="Benutzername">
  <div *ngIf="name.touched && name.errors?.required">Benutzername ist erforderlich</div>
  
  <button type="submit" [disabled]="form.invalid">Absenden</button>
</form>
```

## HTTP-Client

### Konfiguration

In app.module.ts importieren:

```typescript
import { HttpClientModule } from '@angular/common/http';

@NgModule({
  imports: [HttpClientModule]
})
export class AppModule {}
```

### Anfragen senden

```typescript
import { HttpClient } from '@angular/common/http';

constructor(private http: HttpClient) {}

getData() {
  this.http.get('/api/data').subscribe(data => {
    console.log(data);
  });
}

postData() {
  this.http.post('/api/data', { name: 'test' }).subscribe();
}
```

### Interceptors

```typescript
import { HttpInterceptorFn } from '@angular/common/http';

export const authInterceptor: HttpInterceptorFn = (req, next) => {
  const token = localStorage.getItem('token');
  const cloned = req.clone({
    setHeaders: { Authorization: `Bearer ${token}` }
  });
  return next(cloned);
};
```

## Best Practices

1. **TypeScript Strict Mode verwenden:** Strict-Compiling-Optionen aktivieren
2. **Komponenten klein und fokussiert:** Jede Komponente nur für eine Sache zuständig
3. **OnPush-Strategie nutzen:** Bessere Change-Detection-Performance
4. **Pipes sinnvoll einsetzen:** Keine komplexen Berechnungen in Pipes
5. **Lazy Loading verwenden:** Schnellere initiale Ladezeit
6. **Tests schreiben:** Code-Qualität sicherstellen
