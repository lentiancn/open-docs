# Vue 2 使用指南

高效使用 Vue 2 的指南。

---

## 基本概念

Vue 2 是一個用於構建用戶界面的 JavaScript 框架。

---

## 實例

```javascript
new Vue({
  el: '#app',
  data: {
    message: 'Hello Vue!'
  }
})
```

---

## 模板指令

| 指令 | 說明 |
|------|------|
| `v-if` | 條件渲染 |
| `v-for` | 列表渲染 |
| `v-bind` | 屬性綁定 |
| `v-model` | 雙向綁定 |

---

## 組件

```vue
<template>
  <div>{{ title }}</div>
</template>

<script>
export default {
  props: {
    title: String
  }
}
</script>
```
