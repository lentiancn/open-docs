# Bootstrap Benutzerhandbuch

Bootstrap-Grundfunktionen.

## Gridsystem

```html
<div class="container">
  <div class="row">
    <div class="col">Spalte</div>
  </div>
</div>
```

## Komponenten

### Navbar

```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <a class="navbar-brand" href="#">Marke</a>
  <div class="collapse navbar-collapse">
    <ul class="navbar-nav">
      <li class="nav-item">
        <a class="nav-link" href="#">Start</a>
      </li>
    </ul>
  </div>
</nav>
```

### Karten

```html
<div class="card" style="width: 18rem;">
  <div class="card-body">
    <h5 class="card-title">Titel</h5>
    <p class="card-text">Inhalt</p>
    <a href="#" class="btn btn-primary">Button</a>
  </div>
</div>
```

### Modals

```html
<button class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#meinModal">
  Modal öffnen
</button>

<div class="modal fade" id="meinModal">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title">Titel</h5>
        <button class="btn-close" data-bs-dismiss="modal"></button>
      </div>
      <div class="modal-body">Inhalt</div>
    </div>
  </div>
</div>
```

## Hilfsklassen

### Abstände

```html
<div class="m-4">Margin</div>
<div class="p-3">Padding</div>
```

### Farben

```html
<div class="text-primary">Primärer Text</div>
<div class="bg-dark text-white">Dunkler Hintergrund</div>
```

## Formulare

```html
<form>
  <div class="mb-3">
    <label class="form-label">E-Mail</label>
    <input type="email" class="form-control">
  </div>
  <button type="submit" class="btn btn-primary">Senden</button>
</form>
```
