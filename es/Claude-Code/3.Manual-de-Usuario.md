# Claude Code Manual de usuario

Uso básico y técnicas.

## Uso básico

### Iniciar

- VS Code: Usa el atajo Cmd/Ctrl + Shift + P
- CLI: Ejecuta `claude` directamente en terminal

### Conversación

Describe tus necesidades directamente en la ventana de chat:
```
Escribe una función de ordenamiento para mí
Explica este código
Cómo corregir este bug
```

## Generación de código

### Generar nuevo código

Describe la funcionalidad que quieres:
```
Crear una API de inicio de sesión de usuario
Implementar consulta con paginación
Escribir pruebas unitarias
```

### Modificar código existente

1. Selecciona el código
2. Describe las necesidades de modificación
3. Claude generará el código modificado

## Asistencia de depuración

### Explicar errores

Pega el mensaje de error directamente:
```
¿Qué significa este error?
```

### Sugerencias de corrección

Pide a Claude que analice el problema:
```
¿Qué tiene de malo esta función?
```

## Revisión de código

### Solicitudes de revisión de código

```
Revisa este código
¿Hay problemas de seguridad?
Sugerencias de optimización de rendimiento
```

## Atajos de teclado

### VS Code

| Atajo | Función |
|-------|----------|
| Cmd/Ctrl + L | Abrir chat |
| Cmd/Ctrl + Enter | Enviar entrada |

## Opciones de configuración

### claude.json

Crea `claude.json` en la raíz del proyecto:

```json
{
  "preamble": "Eres un experto en Python",
  "maxTokens": 4000
}
```

## Mejores prácticas

1. Describe los requisitos claramente
2. Proporciona información de contexto
3. Itera y mejora el código
4. Combina con revisión humana
