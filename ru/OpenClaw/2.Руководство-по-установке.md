# Руководство по установке OpenClaw

Пошаговое руководство по установке на разные операционные системы.

## Требования

- **Node.js**: версия 22 или выше
- **ОС**: macOS, Linux или Windows
- **Опционально**: Docker (для функции изоляции)

## Установка

### macOS и Linux (Рекомендуется)

Самый простой способ — использовать официальный скрипт установки:

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

Этот скрипт автоматически определяет вашу системную среду, устанавливает необходимые зависимости и настраивает OpenClaw.

### Windows

Выполните в PowerShell:

```powershell
iwr -useb https://openclaw.ai/install.ps1 | iex
```

### Прямая установка через npm

Если Node.js уже настроен:

```bash
npm install -g openclaw@latest
```

### Из исходного кода

Для использования последней версии:

```bash
git clone https://github.com/openclaw/openclaw.git
cd openclaw
npm install
npm link
```

## После установки

### Запуск мастера настройки

После установки рекомендуется запустить мастер для базовой настройки:

```bash
openclaw onboard --install-daemon
```

Мастер поможет с:
- Настройкой API-ключа
- Настройкой порта сервиса
- Подключением каналов (WhatsApp, Telegram, Discord и др.)
- Настройками безопасности

### Проверка состояния сервиса

Если вы установили как показано выше, сервис уже должен работать:

```bash
openclaw gateway status
```

### Запуск в фоновом режиме

Чтобы видеть логи напрямую (полезно для отладки):

```bash
openclaw gateway --port 18789
```

### Открытие веб-интерфейса

```bash
openclaw dashboard
```

Или просто откройте в браузере: `http://127.0.0.1:18789/`

## Переменные окружения

Для настройки расположения данных:

- `OPENCLAW_HOME`: домашний каталог OpenClaw
- `OPENCLAW_STATE_DIR`: каталог для данных состояния
- `OPENCLAW_CONFIG_PATH`: путь к файлу конфигурации

## Проверка установки

После установки вы можете:

1. Запустить `openclaw --version` для подтверждения
2. Запустить `openclaw doctor` для проверки системы
3. Открыть веб-интерфейс и отправить тестовое сообщение

## Частые проблемы

### Проверка версии Node.js

```bash
node --version
```

Убедитесь что версия 22 или выше.

### Git не найден в Windows

Убедитесь что Git установлен и добавлен в PATH.

### Ошибки прав доступа

При проблемах с правами попробуйте:

```bash
sudo npm install -g openclaw@latest
```
