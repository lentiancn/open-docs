# Bootstrap

세계에서 가장 인기 있는 프론트엔드 CSS 프레임워크.

## 개요

Bootstrap은 Twitter가 개발한 가장 인기 있는 CSS 프레임워크입니다. 반응형 그리드 시스템, 프리빌드 컴포넌트, CSS 유틸리티 클래스等特点이 있습니다.

### 주요 기능

- **반응형 그리드 시스템** - 12열 그리드
- **프리빌드 컴포넌트** - 네비게이션 바, 카드, 모달
- **CSS 유틸리티 클래스** - 빠른 스타일 조정

## 빠른 시작

### CDN

```html

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```
