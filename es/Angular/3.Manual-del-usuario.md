# Manual del Usuario

Este manual describe en detalle las funcionalidades principales de Angular y cómo usarlas, incluyendo componentes, servicios, enrutamiento, directivas, pipes, formularios y más.

## Desarrollo de Componentes

### Crear un Componente

```bash
ng generate component user-list
```

O de forma manual:

```typescript
import { Component } from '@angular/core';

@Component({
  selector: 'app-user-list',
  templateUrl: './user-list.component.html',
  styleUrls: ['./user-list.component.scss']
})
export class UserListComponent {
  users = ['Juan', 'María', 'Pedro'];
}
```

## Servicios e Inyección de Dependencias

### Crear un Servicio

```bash
ng generate service user
```

Código del servicio:

```typescript
import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class UserService {
  constructor(private http: HttpClient) {}

  getUsers(): Observable<User[]> {
    return this.http.get<User[]>('/api/users');
  }
}
```

### Usar el Servicio

```typescript
import { Component, OnInit } from '@angular/core';
import { UserService } from './user.service';

@Component({
  selector: 'app-user-list',
  templateUrl: './user-list.component.html'
})
export class UserListComponent implements OnInit {
  users: User[] = [];

  constructor(private userService: UserService) {}

  ngOnInit() {
    this.userService.getUsers().subscribe(data => {
      this.users = data;
    });
  }
}
```

## Pipes

### Pipes Integrados

```html
<!-- Pipe de fecha -->
<p>{{ today | date:'yyyy-MM-dd' }}</p>

<!-- Pipes de mayúsculas/minúsculas -->
<p>{{ name | uppercase }}</p>
<p>{{ name | lowercase }}</p>

<!-- Pipe de moneda -->
<p>{{ price | currency:'EUR' }}</p>

<!-- Pipe de números -->
<p>{{ pi | number:'1.2-2' }}</p>

<!-- Pipe JSON -->
<pre>{{ obj | json }}</pre>
```

### Crear un Pipe Personalizado

```bash
ng generate pipe capitalize
```

```typescript
import { Pipe, PipeTransform } from '@angular/core';

@Pipe({
  name: 'capitalize'
})
export class CapitalizePipe implements PipeTransform {
  transform(value: string): string {
    if (!value) return value;
    return value.charAt(0).toUpperCase() + value.slice(1);
  }
}
```

## Manejo de Formularios

### Formularios Reactivos

```typescript
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

export class LoginComponent implements OnInit {
  loginForm: FormGroup;

  constructor(private fb: FormBuilder) {}

  ngOnInit() {
    this.loginForm = this.fb.group({
      username: ['', Validators.required],
      password: ['', [Validators.required, Validators.minLength(6)]],
      remember: [false]
    });
  }

  onSubmit() {
    if (this.loginForm.valid) {
      console.log(this.loginForm.value);
    }
  }
}
```

Plantilla:

```html
<form [formGroup]="loginForm" (ngSubmit)="onSubmit()">
  <input formControlName="username" placeholder="Usuario">
  <div *ngIf="loginForm.get('username').touched && loginForm.get('username').errors?.required">
    El usuario es obligatorio
  </div>
  
  <input type="password" formControlName="password" placeholder="Contraseña">
  
  <label>
    <input type="checkbox" formControlName="remember"> Recordarme
  </label>
  
  <button type="submit" [disabled]="loginForm.invalid">Iniciar Sesión</button>
</form>
```

### Formularios Basados en Plantillas

```html
<form #form="ngForm" (ngSubmit)="onSubmit(form)">
  <input name="username" [(ngModel)]="username" required #name="ngModel" placeholder="Usuario">
  <div *ngIf="name.touched && name.errors?.required">El usuario es obligatorio</div>
  
  <button type="submit" [disabled]="form.invalid">Enviar</button>
</form>
```

## Cliente HTTP

### Configuración

En app.module.ts, importar:

```typescript
import { HttpClientModule } from '@angular/common/http';

@NgModule({
  imports: [HttpClientModule]
})
export class AppModule {}
```

### Enviar Solicitudes

```typescript
import { HttpClient } from '@angular/common/http';

constructor(private http: HttpClient) {}

getData() {
  this.http.get('/api/data').subscribe(data => {
    console.log(data);
  });
}

postData() {
  this.http.post('/api/data', { name: 'test' }).subscribe();
}
```

### Interceptores

```typescript
import { HttpInterceptorFn } from '@angular/common/http';

export const authInterceptor: HttpInterceptorFn = (req, next) => {
  const token = localStorage.getItem('token');
  const cloned = req.clone({
    setHeaders: { Authorization: `Bearer ${token}` }
  });
  return next(cloned);
};
```

## Mejores Prácticas

1. **Usar modo estricto de TypeScript**: Habilitar la opción de compilación strict
2. **Componentes pequeños y enfocados**: Cada componente debe hacer una sola cosa
3. **Usar estrategia OnPush**: Mejorar el rendimiento de detección de cambios
4. **Usar pipes de forma inteligente**: Evitar cálculos complejos dentro de pipes
5. **Usar carga diferida**: Reducir el tiempo de carga inicial
6. **Escribir pruebas**: Garantizar la calidad del código
