# Claude Code

[Claude Code](https://code.claude.com/) 是由 Anthropic 公司開發的智慧程式設計助手，能夠讀取程式碼庫、編輯檔案、執行命令，並與開發工具無縫整合。

## 概述

Claude Code 是一款 AI 驅動的程式設計助手，支援在終端、IDE、桌面應用程式和瀏覽器等多種環境中使用。它能夠幫助你建構功能、修復錯誤和自動化開發任務。

## 功能特性

- **自動化任務**：編寫測試、修復 lint 錯誤、解決合併衝突、更新依賴
- **功能開發**：用自然語言描述需求，Claude Code 會規劃方案並實現
- **Git 整合**：暫存更改、編寫提交資訊、建立分支、開啟 Pull Request
- **MCP 整合**：連接外部資料來源和工具
- **自訂能力**：CLAUDE.md、自訂命令、Hooks
- **多代理協作**：並行處理複雜任務

## 快速開始

### 安裝

**macOS、Linux、WSL：**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell：**
```powershell
irm https://claude.ai/install.ps1 | iex
```

### 啟動

```bash
cd your-project
claude
```
