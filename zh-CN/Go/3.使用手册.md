## 快速开始

### 第一个 Go 程序

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

运行程序：

```bash
go run main.go
```

## 基础语法

### 变量声明

```go
// 方式一：类型推断
var name = "Go"

// 方式二：显式类型
var age int = 10

// 方式三：短变量声明（函数内）
message := "Hello"
```

### 数据类型

```go
// 整数
var i int = 10
var ui uint = 10

// 浮点数
var f float64 = 3.14

// 字符串
var s string = "Hello"

// 布尔值
var b bool = true

// 数组
var arr [5]int = [5]int{1, 2, 3, 4, 5}

// 切片
slice := []int{1, 2, 3}

// 映射
m := map[string]int{"a": 1, "b": 2}
```

### 控制流

```go
// if 条件
if x > 10 {
    fmt.Println("大于10")
} else {
    fmt.Println("小于等于10")
}

// for 循环
for i := 0; i < 10; i++ {
    fmt.Println(i)
}

// switch
switch day {
case 1:
    fmt.Println("周一")
default:
    fmt.Println("其他")
}
```

## 函数

### 函数定义

```go
// 基础函数
func add(a, b int) int {
    return a + b
}

// 多返回值
func divide(a, b int) (int, error) {
    if b == 0 {
        return 0, errors.New("除数不能为零")
    }
    return a / b, nil
}

// 命名返回值
func split(sum int) (x, y int) {
    x = sum * 4 / 9
    y = sum - x
    return
}
```

### 变长参数

```go
func sum(nums ...int) int {
    total := 0
    for _, n := range nums {
        total += n
    }
    return total
}

// 使用
sum(1, 2, 3, 4, 5)
```

## 并发编程

### Goroutine

```go
// 启动协程
go func() {
    fmt.Println("Hello from goroutine")
}()

// 带参数
go func(msg string) {
    fmt.Println(msg)
}("Hello")
```

### Channel

```go
// 创建通道
ch := make(chan int)

// 发送数据
ch <- 10

// 接收数据
value := <-ch

// 带缓冲的通道
buffered := make(chan int, 10)
```

### Select

```go
select {
case msg := <-ch1:
    fmt.Println("收到:", msg)
case msg := <-ch2:
    fmt.Println("收到:", msg)
case <-time.After(time.Second):
    fmt.Println("超时")
}
```

## 错误处理

### 错误定义

```go
import "errors"

func divide(a, b int) (int, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}
```

### 错误检查

```go
result, err := divide(10, 0)
if err != nil {
    fmt.Println("错误:", err)
    return
}
fmt.Println("结果:", result)
```

## 包管理

### 初始化模块

```bash
go mod init github.com/username/project
```

### 添加依赖

```bash
go get github.com/package/name
```

### 整理依赖

```bash
go mod tidy
```

## 测试

### 单元测试

```go
package main

import "testing"

func TestAdd(t *testing.T) {
    result := add(2, 3)
    if result != 5 {
        t.Errorf("期望 5, 实际 %d", result)
    }
}
```

运行测试：

```bash
go test ./...
```

### 基准测试

```go
func BenchmarkHello(b *testing.B) {
    for i := 0; i < b.N; i++ {
        Hello()
    }
}
```

运行基准测试：

```bash
go test -bench=.
```

## 最佳实践

4. 避免全局变量
5. 编写单元测试
6. 错误及时处理
7. 使用 `context` 管理请求生命周期
