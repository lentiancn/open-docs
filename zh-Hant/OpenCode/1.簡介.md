# OpenCode 簡介

開源 AI 編程助手。

## 什麼是 OpenCode？

OpenCode 是一個開源 AI 編程助手，可作為終端介面、桌面應用或 IDE 擴展使用。

### 主要特性

- **LSP 自動啟用** - 自動載入正確的 LSP
- **多對話** - 可在同一個項目中並行啟動多個代理
- **分享連結** - 可分享任何對話連結供他人參考或調試
- **GitHub Copilot** - 登入 GitHub 使用 Copilot 帳戶
- **ChatGPT Plus/Pro** - 登入 OpenAI 使用 ChatGPT 帳戶
- **任意模型** - 支持 75+ LLM 提供商，包括本地模型
- **任意編輯器** - 支持終端、桌面應用和 IDE 擴展
- **隱私優先** - 不存儲任何代碼或上下文數據
