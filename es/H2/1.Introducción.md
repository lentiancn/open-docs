# Base de datos H2 - Introducción

## ¿Qué es H2?

H2 es un sistema de gestión de bases de datos relacionales de código abierto escrito en Java. Puede utilizarse como base de datos embebida dentro de aplicaciones Java o ejecutarse en modo cliente-servidor. El software se distribuye bajo la Licencia Pública de Mozilla 2.0 (MPL 2.0) o la Licencia Pública de Eclipse original (EPL 1.0).

## Características principales de H2

### Base de datos embebida
H2 puede integrarse directamente en aplicaciones Java para ejecutarse sin necesidad de un servidor de base de datos separado. Esto lo hace muy popular en entornos de desarrollo y pruebas.

### Modo cliente-servidor
Además del modo embebido, H2 también soporta el modo cliente-servidor, funcionando como un servidor de base de datos independiente al que múltiples aplicaciones pueden acceder simultáneamente.

### Base de datos en memoria
H2 soporta el modo de base de datos en memoria, donde los datos se almacenan en la memoria RAM y se pierden tras un reinicio. Esto es muy útil para pruebas rápidas y desarrollo de prototipos.

### Compatibilidad con múltiples bases de datos
H2 es compatible con múltiples estándares y sintaxis SQL, pudiendo simular el comportamiento de MySQL, PostgreSQL, Oracle, DB2, MS SQL Server, facilitando la migración y pruebas de bases de datos.

### Base de datos cifrada
H2 soporta el cifrado de bases de datos para proteger la seguridad de datos sensibles.

### Búsqueda de texto completo
H2 incluye funcionalidad de búsqueda de texto completo sin necesidad de configuración adicional.

### Cero administración
Como base de datos embebida, H2 no requiere un administrador de bases de datos dedicado para su mantenimiento.

## Escenarios de aplicación de H2

H2 se ampliamente utilizado en los siguientes escenarios:

- Base de datos embebida para aplicaciones Java
- Entornos de desarrollo y pruebas
- Desarrollo rápido de prototipos
- Pruebas unitarias y de integración
- Aplicaciones de escritorio ligeras
- Aprendizaje de sintaxis SQL

## Historial de versiones de H2

El desarrollo del motor de base de datos H2 comenzó en mayo de 2004, con el primer lanzamiento en diciembre de 2005. El desarrollador original Thomas Mueller también fue el fundador del motor de base de datos Java HSQLDB (originalmente llamado Hypersonic SQL).
