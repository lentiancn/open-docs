# OpenClaw

[English](./README.md) | [简体中文](./README.md) | [日本語](./README.md)

🦞 ИИ-шлюз для любой ОС. Поддерживает WhatsApp, Telegram, Discord, iMessage и другие.

## Возможности

- 🤖 ИИ-ассистент шлюз
- 📱 Мультиплатформа
- 🔒 Локальное развёртывание
- 🧠 Несколько агентов
- 🌐 Веб-управление
- 📷 Мобильные узлы

## Быстрый старт

```bash
npm install -g openclaw@latest
openclaw onboard --install-daemon
openclaw gateway --port 18789
```

Откройте http://127.0.0.1:18789/
