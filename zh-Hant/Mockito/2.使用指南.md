# Mockito 使用指南

高效使用 Mockito 進行單元測試的指南。

---

## 基本概念

Mockito 是一個 Java 單元測試模擬框架。

---

## 建立 Mock

```java
List<String> mockList = Mockito.mock(List.class);
```

---

## Stubbing

```java
when(mockList.get(0)).thenReturn("first");
when(mockList.get(0)).thenThrow(new RuntimeException());
```

---

## 驗證

```java
verify(mockList).add("element");
verify(mockList, times(2)).add("element");
verify(mockList, never()).clear();
```

---

## @Mock 注解

```java
@Mock
private UserService userService;

@BeforeEach
void setUp() {
    MockitoAnnotations.openMocks(this);
}
```

---

## @Spy

```java
@Spy
private List<String> spyList = new ArrayList<>();
```

---

## BDD 風格

```java
given(mockList.get(0)).willReturn("first");
then(mockList).should(times(1)).get(0);
```
