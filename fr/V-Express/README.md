# V-Express

V-Express est une méthode de génération de vidéos de portraits développée par Tencent AI Lab.

## Démarrage rapide

```bash
git clone https://github.com/tencent-ailab/V-Express.git
pip install -r requirements.txt
```

## Fonctionnalités

- Animation pilotée par audio
- Contrôle d'image de référence
