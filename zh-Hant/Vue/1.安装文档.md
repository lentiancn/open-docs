# Vue 安裝指南

Vue 是一個用於構建用戶界面的漸進式 JavaScript 框架。

## 版本

| 版本 | 發布日期 | 狀態 |
|------|----------|------|
| Vue 3 | 2020年9月 | 推薦 |
| Vue 2 | 2016年10月 | 舊版 |

**推薦：** 所有新項目使用 Vue 3。

## 前置條件

| 要求 | Vue 2 | Vue 3 |
|------|-------|-------|
| Node.js | 8.9+ | 16+ (18+推薦) |
| npm | 5.6+ | 8+ |

## 安裝方法

### 方法 1：npm（推薦）

```bash
# 創建新 Vue 項目
npm create vue@latest
```

### 方法 2：Vite（推薦用於 Vue 3）

```bash
npm create vite@latest my-vue-app -- --template vue
cd my-vue-app
npm install
npm run dev
```

### 方法 3：CDN

```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
```

## 項目設置

```bash
npm install vue-router@4
npm install pinia
```

## 開發服務器

```bash
# 啟動服務器
npm run dev

# 構建生產版本
npm run build
```

## 問題解決

### npm install 失敗

```bash
npm cache clean --force
rm -rf node_modules
npm install
```

## 最佳實踐

1. **使用 Vue 3** - 新項目
2. **使用 Vite** - 更快的開發
3. **使用 Composition API** - 更好的 TypeScript 支持
