# Claude Code 安裝指南

詳細介紹 Claude Code 的安裝方法。

## Windows 安裝

### 方法一：VS Code 擴展

1. 打開 VS Code
2. 搜尋 "Claude Code" 擴展
3. 點擊安裝
4. 登錄 Anthropic 賬戶

### 方法二：獨立客戶端

1. 訪問 https://claude.com/code
2. 下載 Windows 客戶端
3. 運行安裝程序
4. 登錄賬戶

## macOS 安裝

### 方法一：VS Code 擴展

1. 打開 VS Code
2. 搜尋 "Claude Code" 擴展
3. 點擊安裝
4. 登錄賬戶

### 方法二：Homebrew

```
brew install claude-code
```

### 方法三：獨立客戶端

1. 訪問 https://claude.com/code
2. 下載 macOS 客戶端
3. 運行安裝程序
4. 登錄賬戶

## Linux 安裝

### 方法一：VS Code 擴展

1. 打開 VS Code
2. 搜尋 "Claude Code" 擴展
3. 點擊安裝
4. 登錄賬戶

### 方法二：包管理器

**Ubuntu/Debian**:
```
curl -fsSL https://downloads.anthropic.com/claude-code/install.sh | sh
```

**Arch Linux**:
```
yay -S claude-code
```

## 驗證安裝

```
claude --version
```

## 配置

### API 密鑰

1. 訪問 https://console.anthropic.com
2. 創建 API 密鑰
3. 在 Claude Code 中配置

### 環境變量

```
export ANTHROPIC_API_KEY=your-api-key
```

## 卸載

### macOS

```
brew uninstall claude-code
```

### Linux

```
sudo rm -rf /usr/local/claude-code
```
