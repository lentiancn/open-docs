# jQuery

## 概述

jQuery 是一个快速、小巧、功能丰富的 JavaScript 库。它让 HTML 文档遍历和操作、事件处理、动画以及 Ajax 等操作变得更加简单，通过一个易于使用的 API 在多种浏览器中都能正常工作。

## 特性

- **简洁的语法**：链式调用，代码简洁易读
- **强大的选择器**：支持 CSS 选择器的全部功能
- **跨浏览器兼容**：解决不同浏览器之间的 API 差异
- **丰富的插件生态**：庞大的插件库支持
