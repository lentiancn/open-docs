# Element Plus FAQ

자주 묻는 질문과 답변.

## 설치 문제

### Q: 설치 실패?

**해결책:**
1. Node.js 버전 확인
```bash
node --version  # 18.0+ 필요
```
2. npm 캐시 지우기
```bash
npm cache clean --force
```
3. node_modules와 package-lock.json 삭제 후 재설치
```bash
rm -rf node_modules package-lock.json
npm install
```

### Q: 모듈을 찾을 수 없음?

**해결책:**
1. 패키지가 설치되어 있는지 확인
```bash
npm list element-plus
```
2. npm/yarn/pnpm 설정 확인
3. lock 파일 삭제 후 재설치 시도

### Q: TypeScript 타입 오류?

**해결책:**
1. 타입 정의 설치
```bash
npm install -D @types/node
```
2. tsconfig.json에 추가
```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

## 사용 문제

### Q: 컴포넌트가 표시되지 않음?

**해결책:**
1. 올바른 임포트 확인
2. 콘솔 오류 메시지 확인
3. Vue 버전 확인 (3.2+ 필요)

### Q: 스타일이 적용되지 않음?

**해결책:**
1. CSS가 임포트되어 있는지 확인
```typescript
import 'element-plus/dist/index.css'
```
2. CSS 우선순위 확인, `:deep()` 또는 `!important` 사용
3. 다른 스타일 덮어쓰기 확인

### Q: 아이콘이 표시되지 않음?

**해결책:**
1. 아이콘 패키지 설치
```bash
npm install @element-plus/icons-vue
```
2. 전역 등록
```typescript
import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```
3. 또는 필요시 임포트

### Q: 테이블이 렌더링되지 않음?

**해결책:**
1. data 속성이 올바르게 바인딩되어 있는지 확인
2. prop이 올바른지 확인
3. 테이블이 el-table-column 외부에 있는지 확인

### Q: 폼 유효성 검사가 작동하지 않음?

**해결책:**
1. prop이 v-model과 일치하는지 확인
2. ref 참조가 있는지 확인
3. validate 메서드를 올바르게 호출
```typescript
const formRef = ref()
await formRef.value.validate()
```

### Q: 다이얼로그가 표시되지 않음?

**해결책:**
1. v-model 바인딩 확인
2. visible이 true인지 확인
3. z-index 충돌 확인

## 테마 문제

### Q: 테마 색상을 변경하려면?

**해결책:**
CSS 변수 사용:
```css
:root {
  --el-color-primary: #409eff;
}
```

### Q: 다크 테마를 사용하려면?

**해결책:**
다크 CSS 임포트:
```typescript
import 'element-plus/theme-chalk/dark/css-vars.css'
```

### Q: 컴포넌트 스타일을 오버라이드하려면?

**해결책:**
CSS 선택자 사용:
```css
.el-button--primary {
  background-color: #원하는색상;
}
```

##国际化 문제

### Q: 언어를 전환하려면?

**해결책:**
```typescript
import ko from 'element-plus/es/locale/lang/ko'
import en from 'element-plus/es/locale/lang/en'

app.use(ElementPlus, { locale: ko })
```

### Q: 사용자 정의 텍스트?

**해결책:**
```typescript
import {ElMessage} from 'element-plus'
ElMessage('사용자 정의 메시지')
```

## 성능 문제

### Q: 초기 로드가 느림?

**해결책:**
1. 필요시 임포트 사용
2. 자동 임포트 플러그인 사용
3. Vite 프리빌드 설정

### Q: 번들 크기가 너무 큼?

**해결책:**
1. 필요시 임포트가 사용되는지 확인
2. 트리 쉐이킹 사용
3. 전체 패키지를 임포트하지 않는지 확인

## 호환성 문제

### Q: IE 브라우저는 지원되지 않음?

Element Plus는 IE 브라우저를 지원하지 않습니다. 다음 브라우저 사용:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### Q: 다른 라이브러리와 충돌?

**해결책:**
1. z-index 수준 확인
2. CSS Modules를 사용하여 스타일 충돌 방지
3. Vue 버전 호환성 확인
