# MultiTalk 安裝指南

本指南將幫助您在電腦上安裝和配置 MultiTalk。

## 系統需求

### Windows
- Windows 10 或更高版本
- 4GB RAM
- 100MB 可用磁碟空間
- 音頻輸入/輸出裝置

### macOS
- macOS 10.15 或更高版本
- 4GB RAM
- 100MB 可用磁碟空間
- 音頻輸入/輸出裝置

### Linux
- Ubuntu 20.04 或更高版本
- 4GB RAM
- 100MB 可用磁碟空間
- 音頻輸入/輸出裝置

## 安裝步驟

### Windows 安裝

1. 從官方網站下載 MultiTalk 安裝程式
2. 執行下載的 `.exe` 檔案
3. 選擇安裝位置
4. 完成安裝精靈
5. 啟動 MultiTalk

### macOS 安裝

1. 從官方網站下載 DMG 檔案
2. 開啟 DMG 檔案
3. 將 MultiTalk 拖移到應用程式資料夾
4. 啟動 MultiTalk

### Linux 安裝

#### 使用 Snap（推薦）
```bash
sudo snap install multitalk
```

#### 使用 DEB 套件
```bash
sudo dpkg -i multitalk_5.x_amd64.deb
```

#### 使用 RPM 套件
```bash
sudo rpm -i multitalk-5.x.x86_64.rpm
```

## 首次設定

首次啟動時，您需要：

1. 選擇您的語言
2. 授權麥克風權限
3. 選擇您的音頻輸入裝置
4. 選擇您的音頻輸出裝置
5. 測試音頻裝置
6. 建立帳戶或登入

## 驗證安裝

安裝完成後，請確認：

- [ ] 應用程式成功啟動
- [ ] 麥克風可以正常運作
- [ ] 喇叭可以正常運作
- [ ] 可以建立或加入房間

## 疑難排解

### 麥克風無法運作

1. 檢查系統權限設定
2. 確認麥克風已正確連接
3. 嘗試重新選擇音頻輸入裝置

### 聽不到其他人的聲音

1. 檢查喇叭音量
2. 確認輸出裝置設定正確
3. 檢查房間的音量設定

### 應用程式當機

1. 更新到最新版本
2. 檢查系統驅動程式
3. 重新安裝應用程式

## 卸載

### Windows
使用控制台的「新增或移除程式」功能卸載

### macOS
將 MultiTalk 從應用程式資料夾拖移到垃圾桶

### Linux
```bash
sudo snap remove multitalk
# 或
sudo dpkg -r multitalk
# 或
sudo rpm -e multitalk
```
