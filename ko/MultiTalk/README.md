# MultiTalk

MultiTalk는 MeiGen-AI 팀에서 개발한 오디오 기반 다인물 대화 영상 생성 프레임워크입니다. 이 프로젝트는 다중 스트림 오디오 입력, 참조 이미지 및 텍스트 프롬프트에 따라 여러 인물이 상호작용하는 고품질 영상 콘텐츠를 생성할 수 있습니다.

## 주요 기능

- 오디오 기반 다인물 대화 영상 생성
- 단일 인물 및 다인물 시나리오 지원
- 업계 최고 수준의 입술 동기화 정확도
- 만화 캐릭터 및 노래 영상 생성 지원
- 출력 해상도 480P 및 720P 지원
- 최대 15초 비디오 클립 생성 가능
