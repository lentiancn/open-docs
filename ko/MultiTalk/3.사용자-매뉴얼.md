# MultiTalk 사용자 매뉴얼

MultiTalk 상세 가이드.

## 방 만들기

### 1단계: 새 방 만들기

1. "방 만들기" 클릭
2. 방 이름 입력
3. 비밀번호 설정 (선택사항)
4. 방 설정 구성

### 2단계: 참가자 초대

- 방 링크 공유
- 방 코드 공유

### 3단계: 방 관리

- 음소거/음소거 해제
- 참가자音量 조정
- 참가자 추방 또는 음소거

## 방 참여

### 방법1: 방 링크

1. 방 링크 클릭
2. MultiTalk가 자동으로 열림
3. 필요하면 비밀번호 입력

### 방법2: 방 코드

1. "방 참여" 클릭
2. 방 코드 입력

## 오디오 설정

### 음성 활성화

1. 설정 > 오디오로 이동
2. 음성 활성화 활성화
3. 임계값 수준 조정

## 키보드 단축키

| 단축키 | 작업 |
|--------|------|
| Ctrl+Shift+T | 푸시 투 토크 |
| Ctrl+M | 음소거 |
