# JMeter 安装指南

本指南将详细介绍如何在不同操作系统上安装和配置 Apache JMeter。

## 环境要求

- **Java 运行环境**：JMeter 需要 Java 8 或更高版本运行
- **操作系统**：支持 Linux、Windows、Mac OSX 等任何 Java 兼容操作系统
- **内存**：建议至少 4GB 可用内存（根据测试规模可能需要更多）
- **磁盘空间**：建议至少 1GB 可用空间

## 下载 JMeter

1. 访问 JMeter 官方下载页面：https://jmeter.apache.org/download_jmeter.cgi
2. 下载最新版本的 JMeter 二进制文件（Binaries）
3. 建议同时下载 PGP 或 SHA-512 签名文件用于验证下载完整性

## Windows 安装步骤

### 解压安装

1. 找到下载的 Apache JMeter 压缩包（apache-jmeter-x.x.zip）
2. 选择一个合适的安装目录，建议路径不包含空格和特殊字符
3. 解压压缩包到选定目录
4. 解压完成后，JMeter 已安装完毕

### 验证安装

1. 进入 JMeter 安装目录的 bin 文件夹
2. 双击 jmeter.bat（Windows 批处理文件）
3. 如果一切正常，JMeter GUI 界面将会启动

## Linux 和 Mac OSX 安装步骤

### 解压安装

1. 找到下载的 Apache JMeter 压缩包（apache-jmeter-x.x.tar.gz）
2. 使用以下命令解压：
   ```bash
   tar -xf apache-jmeter-x.x.tar.gz
   ```
3. 解压完成后，JMeter 已安装完毕

### 设置环境变量（可选）

1. 打开终端配置文件（~/.bashrc 或 ~/.profile）
2. 添加以下环境变量：
   ```bash
   export JMETER_HOME=/path/to/apache-jmeter-x.x
   export PATH=$PATH:$JMETER_HOME/bin:$JMETER_HOME/lib
   ```
3. 保存并刷新配置文件：
   ```bash
   source ~/.bashrc
   ```

### 验证安装

1. 在终端中运行：
   ```bash
   jmeter --version
   ```
2. 如果显示版本信息，说明安装成功

## 启动 JMeter

### 图形界面模式

- **Windows**：双击 bin 目录下的 jmeter.bat
- **Linux/Mac**：在终端运行 jmeter 命令

### 命令行模式

对于负载测试，建议使用命令行模式以减少资源占用：

```bash
jmeter -n -t test_plan.jmx -l results.jtl -e -o /path/to/output
```

参数说明：
- -n：以非 GUI 模式运行
- -t：指定测试计划文件
- -l：指定结果文件
- -e：在测试结束后生成报告
- -o：指定报告输出目录

## 目录结构说明

- **bin**：包含可执行文件和启动脚本
- **lib**：包含 JMeter 依赖的库文件
- **docs**：包含 JMeter 官方文档
- **extras**：包含额外的辅助工具
- **printable_docs**：包含可打印格式的用户手册

## 依赖管理

JMeter 的 lib 目录包含核心依赖，如需添加额外的插件或库：

1. 将 JAR 文件复制到 lib 目录
2. 部分插件需要放入 lib/ext 目录
3. 重启 JMeter 使新插件生效

## 常见安装问题

- **Java 未找到**：确保已正确安装 Java 并设置 JAVA_HOME 环境变量
- **内存不足**：可以通过修改 bin/jmeter.sh 或 jmeter.bat 中的 Heap size 参数来解决
- **启动失败**：检查是否安装了正确版本的 Java（Java 8 或更高版本）
