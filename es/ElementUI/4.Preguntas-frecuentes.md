# Preguntas Frecuentes de Element UI

## Problemas de Instalación

Verifique Node.js >= 8.0.0

npm cache clean --force

## Problemas de Uso

### Estilos no aplicados

Importar CSS:

import 'element-ui/lib/theme-chalk/index.css'

## Problemas de Versión Vue

Element UI solo soporta Vue 2.
