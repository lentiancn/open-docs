## 1. 소개

DragonTalker는 단일 이미지와 오디오 파일에서 사실적인 말하는 동영상을 생성하는 딥러닝 기반 도구입니다. 이 기술은 가상 앵커, 디지털 고객 서비스, AI 교육 등 다양한场景에广泛应用됩니다.

### 1.1 핵심 기술

DragonTalker는 다단계 파이프라인 아키텍처를 사용: 먼저 오디오 분석을 통해 멜 스펙트로그램 특성을 추출하고, 3D 얼굴 랜드마크 감지를 통해 운동 매개변수를 얻은 다음, 이미지 렌더링 기술을 통해 연속적인 비디오 프레임을 생성하며, 마지막으로 얼굴 복원 네트워크를 통해 화질을 향상시킵니다.

### 1.2 주요 기능

- 오디오 구동 애니메이션: 정적 이미지와 오디오 파일에서 동기화된 말하는 동영상 생성
- 3D 모션 추정: 3D 얼굴 재구성에 기반한 자연스러운 머리 포즈와 표정
- 다중 포즈 지원: 45개 이상의 사전 정의된 머리 포즈 템플릿
- 얼굴 향상: GFPGAN, RestoreFormer, CodeFormer 통합
- 웹 인터페이스: 브라우저 기반 간편操作
- Python API: 개발 및 배치 처리를 위한 완전한 프로그래밍 인터페이스

## 3. 설치

### 3.1 Ubuntu/Linux

```bash
sudo apt-get update
sudo apt-get install -y build-essential git wget curl
sudo apt-get install -y python3.8 python3-pip python3-venv
sudo apt-get install -y ffmpeg libsm6 libxext6 libxrender-dev libgl1-mesa-glx

cd DragonTalker
python3 -m venv venv
source venv/bin/activate

pip install --upgrade pip
pip install torch==1.12.1+cu113 torchvision==0.13.1+cu113 -f https://download.pytorch.org/whl/cu113
pip install -r requirements.txt

bash scripts/download_models.sh
```

### 3.2 Windows

1. **Python 설치**: python.org에서 3.8-3.10 다운로드
2. **Git 설치**
3. **CUDA 설치**: NVIDIA에서 다운로드
4. **ffmpeg 설치**:
```powershell
choco install ffmpeg
```

```powershell
git clone https://github.com/your-repo/DragonTalker.git
cd DragonTalker
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 3.3 macOS

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install ffmpeg python@3.10 git wget
git clone https://github.com/your-repo/DragonTalker.git
cd DragonTalker
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## 4. 설치 확인

```bash
source venv/bin/activate
python inference.py \
  --source_image examples/source_image.jpg \
  --driven_audio examples/driven_audio.wav \
  --result_dir ./results \
  --enhancer gfpgan
```

---

## 5. Docker (권장)

```bash
docker build -t dragon-talker:latest .
docker run --gpus all -v $(pwd)/examples:/app/examples \
  -v $(pwd)/results:/app/results \
  dragon-talker:latest \
  python inference.py \
    --source_image /app/examples/source_image.jpg \
    --driven_audio /app/examples/driven_audio.wav \
    --result_dir /app/results
```

---

## 6. 제거

```bash
deactivate
rm -rf venv
docker rmi dragon-talker:latest
```
