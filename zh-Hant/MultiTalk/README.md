# MultiTalk

MultiTalk 是由梅津人工智慧團隊開發的音訊驅動多人對話影片生成框架。該專案能夠根據多流音訊輸入、參考圖像和文字提示，生成包含多人互動的高品質影片內容。

## 主要特性

- 音訊驅動的多人物對話影片生成
- 支援單人物和多人物場景
- 口型同步準確率業界領先
- 支援卡通角色和唱歌影片生成
- 輸出解析度支援 480P 和 720P
- 最長可生成 15 秒影片片段
