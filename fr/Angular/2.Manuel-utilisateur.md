## Introduction

Angular est un framework web open-source basé sur TypeScript développé par Google pour créer des applications modernes monopages (SPA).

### Fonctionnalités principales

- Architecture basée sur les composants
- Liaison de données bidirectionnelle
- Injection de dépendances
- Routage
- Formulaires avec validation
- Client HTTP
- Tests intégrés
- Multiplateforme

## Composants

### Création

```bash
ng generate component components/user-list
# ou
ng g c components/user-list
```

### Exemple

```typescript
import { Component, Input, Output, EventEmitter } from '@angular/core';

@Component({
  selector: 'app-user-card',
  templateUrl: './user-card.component.html'
})
export class UserCardComponent {
  @Input() user: User | null = null;
  @Output() delete = new EventEmitter<number>();

  onDelete() {
    if (this.user) {
      this.delete.emit(this.user.id);
    }
  }
}
```

### Hooks de cycle de vie

```typescript
ngOnInit() { }
ngOnDestroy() { }
ngAfterViewInit() { }
```

### Composants autonomes (Angular 14+)

```typescript
@Component({
  selector: 'app-standalone',
  standalone: true,
  imports: [CommonModule],
  template: `<h1>{{ title }}</h1>`
})
export class StandaloneComponent {
  title = 'Autonome';
}
```

## Templates

### Syntaxe

```html
<!-- Interpolation -->
<h1>Bonjour, {{ name }}!</h1>

<!-- Property binding -->
<img [src]="imageUrl">

<!-- Event binding -->
<button (click)="onClick()">Cliquer</button>

<!-- Two-way binding -->
<input [(ngModel)]="username">
```

### Directives structurelles

```html
<div *ngIf="isLoggedIn">Bienvenue!</div>

<li *ngFor="let user of users">{{ user.name }}</li>
```

### Pipes

```html
<p>{{ today | date:'fullDate' }}</p>
<p>{{ price | currency:'EUR' }}</p>
```

## Services

### Création

```bash
ng generate service services/user
```

### Exemple

```typescript
import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';

@Injectable({ providedIn: 'root' })
export class UserService {
  constructor(private http: HttpClient) {}

  getUsers(): Observable<User[]> {
    return this.http.get<User[]>('https://api.example.com/users');
  }
}
```

## Routage

### Définir les routes

```typescript
import { Routes } from '@angular/router';

export const routes: Routes = [
  { path: '', redirectTo: '/home', pathMatch: 'full' },
  { path: 'home', loadComponent: () => import('./home/home.component').then(m => m.HomeComponent) },
  { path: '**', loadComponent: () => import('./not-found/not-found.component').then(m => m.NotFoundComponent) }
];
```

### Paramètres de route

```typescript
constructor(private route: ActivatedRoute) {}

ngOnInit() {
  this.route.paramMap.subscribe(params => {
    const id = params.get('id');
  });
}
```

### Guards

```typescript
@Injectable({ providedIn: 'root' })

    return true;
  }
}
```

## Formulaires

### Template-driven

```typescript
@Component({
  selector: 'app-login',
  template: `
    <form #form="ngForm" (ngSubmit)="onSubmit(form)">
      <input [(ngModel)]="user.email" name="email" required>
      <button type="submit" [disabled]="form.invalid">Connexion</button>
    </form>
  `
})
export: LoginComponent {
  user = { email: '' };
}
```

```typescript
this.form = this.fb.group({

```

## Gestion d'état

### BehaviorSubject

```typescript
@Injectable({ providedIn: 'root' })
export class CartService {
  private items$ = new BehaviorSubject<CartItem[]>([]);
  
  getItems() { return this.items$.asObservable(); }
  addItem(item: CartItem) {
    this.items$.next([...this.items$.value, item]);
  }
}
```

### Signals (Angular 16+)

```typescript
count = signal(0);
doubled = computed(() => this.count() * 2);
increment() { this.count.update(c => c + 1); }
```

## Build et déploiement

### Développement
```bash
ng build
ng serve
```

### Production
```bash
ng build --configuration production
```

### Déploiement

**Netlify:**
```bash
npm install -g netlify-cli
netlify deploy --prod --dir=dist/your-app
```

**Docker:**
```dockerfile
FROM node:18-alpine as build
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build -- --configuration production

FROM nginx:alpine
COPY --from=build /app/dist/your-app/browser /usr/share/nginx/html
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
```
