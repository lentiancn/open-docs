# Spring Cloud インストールガイド

Spring Cloud開発環境のインストールと設定について説明します。

## システム要件

### JavaとSpring Bootバージョン

| リリース列車 | Spring Boot |
|------------|------------|
| 2025.1.x | 4.0.x |
| 2024.0.x | 3.4.x |
| 2023.0.x | 3.2.x, 3.3.x |

## プロジェクトの作成

### Spring Initializr

1. [start.spring.io](https://start.spring.io/)にアクセス
2. 選択：
   - プロジェクト：Maven
   - 言語：Java
   - Spring Boot：3.3.x
3. 依存関係を追加：
   - Spring Cloud Config Client
   - Spring Cloud Eureka
   - Spring Cloud Gateway

### コマンドライン

```bash
curl https://start.spring.io/starter.zip \
  -d type=maven-project \
  -d language=java \
  -d bootVersion=3.3.0 \
  -d baseDir=myapp \
  -d dependencies=cloud-config-client,cloud-eureka,cloud-gateway \
  -o myapp.zip
```

## Maven設定

```xml
<properties>
    <spring-cloud.version>2023.0.3</spring-cloud.version>
</properties>

<dependencyManagement>
    <dependencies>
        <dependency>
            <groupId>org.springframework.cloud</groupId>
            <artifactId>spring-cloud-dependencies</artifactId>
            <version>${spring-cloud.version}</version>
            <type>pom</type>
            <scope>import</scope>
        </dependency>
    </dependencies>
</dependencyManagement>
```

## メインプロジェクト

| プロジェクト | 説明 |
