# TensorFlow Guía de Instalación

Esta guía te lleva a través de la instalación de TensorFlow.

## Requisitos

- Python 3.7-3.11
- pip 19.0+
- GPU NVIDIA con CUDA (para versión GPU)

## Instalación

### Versión CPU

```bash
pip install tensorflow
```

### Versión GPU

```bash
pip install tensorflow-gpu
```

### Versión Más Reciente

```bash
pip install tf-nightly
```

## Verificar Instalación

```python
import tensorflow as tf
print(tf.__version__)
print("GPU Available:", tf.config.list_physical_devices('GPU'))
```

## Entornos Virtuales

### Usando venv

```bash
# Crear entorno
python -m venv tf-env

# Activar (Windows)
tf-env\Scripts\activate

# Activar (Linux/macOS)
source tf-env/bin/activate

# Instalar TensorFlow
pip install tensorflow
```

### Usando Conda

```bash
# Crear entorno
conda create -n tf-env python=3.10
conda activate tf-env

# Instalar TensorFlow
conda install tensorflow
```

## Configuración GPU

### Verificar GPU

```python
import tensorflow as tf
gpus = tf.config.list_physical_devices('GPU')
print(gpus)
```

### Configurar Memoria GPU

```python
gpus = tf.config.list_physical_devices('GPU')
if gpus:
    try:
        for gpu in gpus:
            tf.config.experimental.set_memory_growth(gpu, True)
    except RuntimeError as e:
        print(e)
```

### Especificar GPU

```python
import os
os.environ['CUDA_VISIBLE_DEVICES'] = '0'
```

## CUDA y cuDNN

### Requisitos de Versión

| TensorFlow | CUDA | cuDNN |
|------------|------|-------|
| 2.13-2.15 | 12.2 | 8.9 |
| 2.10-2.12 | 11.8 | 8.6 |
| 2.9 | 11.7 | 8.5 |

### Instalar NVIDIA Drivers

Descarga drivers desde: https://www.nvidia.com/drivers

### Instalar CUDA Toolkit

Descarga desde: https://developer.nvidia.com/cuda-downloads

### Instalar cuDNN

1. Descarga desde: https://developer.nvidia.com/cudnn
2. Extrae y copia archivos a CUDA directory

## Docker

### Usar con GPU

```bash
docker pull tensorflow/tensorflow:latest-gpu
docker run -it --gpus all -v $(pwd):/workspace tensorflow/tensorflow:latest-gpu
```

### Usar CPU

```bash
docker pull tensorflow/tensorflow:latest
docker run -it -v $(pwd):/workspace tensorflow/tensorflow:latest
```
