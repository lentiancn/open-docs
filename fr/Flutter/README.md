# Flutter

Kit d'outils UI open source de Google.

## Aperçu

Flutter est un kit d'outils UI open source pour créer des applications multiplateformes à partir d'une seule base de code utilisant Dart.

## Fonctionnalités

- **Multiplateforme**: Une base de code pour toutes les plateformes
- **Hot Reload**: Aperçu instantané des modifications
- **Widgets Riches**: Material Design + Cupertino
- **Haute Performance**: Expérience de rendu natif
- **Open Source**: Gratuit à utiliser

## Démarrage Rapide

### Installation

Voir guide d'installation : ./2.Guide-d-Installation.md

### Créer un projet

```bash
flutter create mon_app
cd mon_app
flutter run
```

### Structure du projet

```
lib/
└── main.dart        # Point d'entrée
pubspec.yaml         # Fichier de configuration
```

## Concepts Fondamentaux

- **Dart**: Langage de programmation Flutter
- **Widget**: Bloc de construction UI
- **State**: Gestion d'état
- **Platform Channel**: Canaux de plateforme
