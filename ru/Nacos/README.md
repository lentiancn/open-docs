# Nacos

Nacos (Dynamic Naming and Configuration Service) — это платформа с открытым исходным кодом для динамического обнаружения сервисов, управления конфигурацией и управления сервисами, разработанная Alibaba.

## Что такое Nacos

Nacos стремится помочь вам обнаруживать, настраивать и управлять микросервисами и приложениями ИИ-агентов. Он предоставляет набор простых и полезных функций, позволяющих реализовать динамическое обнаружение сервисов, настройку сервисов, метаданные сервисов и управление трафиком.

## Основные функции

- **Обнаружение сервисов**: поддерживает обнаружение сервисов на основе DNS и RPC с проверками работоспособности в реальном времени
- **Управление конфигурацией**: динамическая конфигурация с отслеживанием версий, канареечным развертыванием и откатом одним щелчком
- **Динамический DNS**: поддерживает взвешенную маршрутизацию для реализации балансировки нагрузки промежуточного уровня
- **Управление метаданными сервисов**: управляет описанием сервисов, жизненным циклом, управлением трафиком и т. д.
- **Управление MCP сервисами**: поддерживает регистрацию и обнаружение MCP сервисов (Model Context Protocol)

## Быстрый запуск

### Требования к среде

- JDK 17+
- 64-битная операционная система (Linux/Unix/Mac/Windows)

### Запуск Nacos

Автоматическое развертывание одним щелчком с помощью nacos-setup:

```bash
curl -fsSL https://nacos.io/nacos-installer.sh | sudo bash
nacos-setup
```

Или ручное развертывание:

```bash
# Распаковать дистрибутив
unzip nacos-server-$version.zip
cd nacos/bin

# Запустить в автономном режиме
sh startup.sh -m standalone
```

### Доступ к консоли

- Адрес: http://127.0.0.1:8080/index.html
- Имя пользователя: nacos
- Пароль: nacos@demo
