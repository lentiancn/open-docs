# FantasyTalking

ИИ-инструмент для генерации видео Talking Head.

## Обзор

FantasyTalking создаёт реалистичные видео из изображения и аудио с помощью глубокого обучения.

## Функции

- Из одного изображения
- Мультиязычность
- Высокое качество
- Улучшение лица
- Открытый исходный код

## Быстрый старт

```bash
git clone https://github.com/FantasyTalking/FantasyTalking.git
cd FantasyTalking
pip install -r requirements.txt
python inference.py --source_image face.jpg --audio speech.wav --output output.mp4
```
