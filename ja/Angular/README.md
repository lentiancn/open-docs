# Angular

Angularは、Googleが開発したWebアプリケーション構築用の強力なフロントエンドフレームワークです。

## 概要

Angularは、双方向データバインディング、依存注入、強力なCLIツールなど、モダンなWebアプリケーション構築のための完全なソリューションを提供するTypeScriptベースのフレームワークです。

## 機能

- **コンポーネントベース**：再利用可能なUIコンポーネントを構築
- **TypeScript**：型安全な開発
- **CLI**：強力なコマンドラインツール
- **ルーティング**：組み込みルーティングシステム
- **フォーム**：テンプレート駆動とリアクティブフォーム
- **HTTP**：API呼び出し用のHTTPクライアント
- **テスト**：組み込みテストサポート

## クイックスタート

### インストール

```bash
npm install -g @angular/cli
```

### プロジェクト作成

```bash
ng new my-app
cd my-app
ng serve
```

http://localhost:4200 にアクセス
