# Oracle ユーザーマニュアル

## データベースへの接続

### SQL*Plusの使用

```bash
# SYSDBAとして接続
sqlplus / as sysdba

# ユーザー名とパスワードで接続
sqlplus ユーザー名/パスワード@データベース
```

---

## 基本的なSQL操作

### テーブルの作成

```sql
CREATE TABLE 社員 (
    社員ID NUMBER(10) PRIMARY KEY,
    名 VARCHAR2(50) NOT NULL,
    性 VARCHAR2(50) NOT NULL,
    メール VARCHAR2(100),
    給与 NUMBER(10, 2)
);
```

### データの挿入

```sql
INSERT INTO 社員 (社員ID, 名, 性, メール, 給与)
VALUES (1001, '山田', '太郎', 'taro@company.com', 75000);
```

### データの取得

```sql
SELECT * FROM 社員;
SELECT 名, 性, 給与 FROM 社員 WHERE 部署ID = 10;
```

## PL/SQL

### ストアドプロシージャ

```sql
CREATE OR REPLACE PROCEDURE 給与更新 (
    p_社員ID IN NUMBER,
    p_新給与 IN NUMBER
) AS
BEGIN
    UPDATE 社員
    SET 給与 = p_新給与
    WHERE 社員ID = p_社員ID;
    COMMIT;
END;
/
```
