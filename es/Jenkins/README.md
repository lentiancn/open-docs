# Jenkins

Jenkins es un servidor de automatización de código abierto para implementar Integración Continua y Despliegue Continuo (CI/CD).

## Introducción

Jenkins es una de las herramientas de CI/CD de código abierto más populares, que puede automatizar los procesos de construcción, prueba y despliegue.

## Características Principales

- **Integración Continua**: Construcción y prueba automática de código
- **Despliegue Continuo**: Despliegue automático a producción
- **Rich Plugins**: Soporte para diversas herramientas y plataformas
- **Construcción Distribuida**: Soporte para construcción multinodo
- **Pipeline**: Flujo de trabajo codificado

## Inicio Rápido

### Instalación

```bash
# Docker
docker run -d -p 8080:8080 jenkins/jenkins:lts

# macOS
brew install jenkins

# Linux
sudo apt install jenkins
```

### Acceso

Abrir el navegador en http://localhost:8080
