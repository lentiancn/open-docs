## 简介

### 什么是 Bootstrap？

Bootstrap 是世界上最流行的前端开源工具包，由 Twitter 的 Mark Otto 和 Jacob Thornton 开发并维护。它是一个用于快速开发响应式网站和 Web 应用程序的 CSS、JavaScript 框架。

### Bootstrap 版本历史

| 版本 | 发布日期 | 状态 |
|------|----------|------|
| Bootstrap 5.3 | 2023年5月 | 当前稳定版 |
| Bootstrap 5.2 | 2022年9月 | 支持 |
| Bootstrap 5.1 | 2021年8月 | 支持 |
| Bootstrap 5.0 | 2021年7月 | 支持 |
| Bootstrap 4.x | 2018年1月 | 维护中 |
| Bootstrap 3.x | 2013年8月 | 已停止维护 |

### Bootstrap 5 新特性

- **全新配色方案**：新增 CSS 变量驱动的配色系统
- **改进的表单控件**：全新设计的表单控件样式
- **新的工具 API**：使用 CSS 类更灵活地控制样式
- **改进的网格系统**：支持更多断点和容器尺寸
- **更好的自定义方式**：使用 Sass 变量深度定制
- **不再依赖 jQuery**：完全使用原生 JavaScript
- **放弃 Internet Explorer 支持**：不再支持 IE10 和 IE11
- **RTL 支持**：原生支持从右到左的语言
- **改进的实用工具类**：更多实用的 CSS 类

## 安装方法

### 方法一：使用 CDN（最简单、最推荐）

CDN（内容分发网络）是最快的开始方式，不需要下载任何文件，不占用本地存储空间。

#### 基本模板

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 入门</title>
    <!-- 引入 Bootstrap CSS -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-T3c6CoIi6uLrA9TneNEoa7RxnatzjcDSCmG1MXxSR1GAsXEV/Dwwykc2MPK8M2HN" crossorigin="anonymous">
</head>
<body>
    <h1>Hello, Bootstrap!</h1>
    
    <!-- 引入 Bootstrap JS（包含 Popper） -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js" integrity="sha384-C6RzsynM9kWDrMNeT87bh95OGNyZPhcTNXj1NW7RuBCsyN/o0jlpcV8Qyq46cDfL" crossorigin="anonymous"></script>
</body>
</html>
```

#### CDN 提供商

**jsDelivr（推荐）**
```html
<!-- CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- JS Bundle（包含 Popper） -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
```

**unpkg**
```html
<link href="https://unpkg.com/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://unpkg.com/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
```

**cdnjs**
```html
<link href="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.2/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.2/js/bootstrap.bundle.min.js"></script>
```

### 方法二：使用 npm 安装

#### 1. 安装 Node.js

访问 https://nodejs.org/ 下载并安装 LTS 版本。

#### 2. 创建项目并安装 Bootstrap

```bash
# 创建新项目目录
mkdir my-bootstrap-project
cd my-bootstrap-project

# 初始化 npm
npm init -y

# 安装 Bootstrap
npm install bootstrap

# 安装 Popper.js（用于下拉菜单、弹出框、工具提示等）
npm install @popperjs/core
```

#### 3. 在项目中使用

**在 JavaScript 中引入：**
```javascript
// 引入 Bootstrap CSS
import 'bootstrap/dist/css/bootstrap.min.css';

// 引入 Bootstrap JS
import 'bootstrap/dist/js/bootstrap.bundle.min.js';
```

**在 HTML 中引用：**
```html
<link rel="stylesheet" href="node_modules/bootstrap/dist/css/bootstrap.min.css">
<script src="node_modules/bootstrap/dist/js/bootstrap.bundle.min.js"></script>
```

### 方法三：下载预编译文件

#### 1. 下载 Bootstrap

访问 Bootstrap 官网 https://getbootstrap.com/ 点击 Download 按钮下载预编译的 CSS 和 JS 文件。

#### 2. 文件结构

下载的压缩包包含以下文件：

```
bootstrap-5.3.2/
├── css/
│   ├── bootstrap.css
│   ├── bootstrap.css.map
│   ├── bootstrap.min.css
│   ├── bootstrap.min.css.map
│   ├── bootstrap.rtl.css
│   ├── bootstrap.rtl.css.map
│   ├── bootstrap.rtl.min.css
│   ├── bootstrap.rtl.min.css.map
│   └── bootstrap-grid.css (仅网格系统)
├── js/
│   ├── bootstrap.bundle.js
│   ├── bootstrap.bundle.js.map
│   ├── bootstrap.bundle.min.js
│   └── bootstrap.bundle.min.js.map
└── LICENSE
```

#### 3. 在 HTML 中引用

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link href="css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>Hello, Bootstrap!</h1>
    <script src="js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### 方法四：使用 Sass 源码（高级用户）

```bash
# 安装 Bootstrap 源码
npm install bootstrap

# 或者使用 Sass 版本
npm install bootstrap-sass
```

## 在 Windows 上安装

### 方法一：使用 CDN（推荐）

1. 创建一个新的 HTML 文件
2. 在 `<head>` 中添加 Bootstrap CSS 链接
3. 在 `<body>` 末尾添加 Bootstrap JS 脚本

### 方法二：使用 npm

1. **安装 Node.js**
   - 访问 https://nodejs.org/
   - 下载 LTS 版本
   - 运行安装程序
   - 注意：勾选 "Add to PATH" 选项

2. **打开命令提示符或 PowerShell**

3. **创建项目**
   ```cmd
   mkdir bootstrap-project
   cd bootstrap-project
   npm init -y
   npm install bootstrap
   ```

4. **创建 HTML 文件**
   ```html
   <!DOCTYPE html>
   <html lang="zh-CN">
   <head>
       <meta charset="UTF-8">
       <meta name="viewport" content="width=device-width, initial-scale=1.0">
       <link href="node_modules/bootstrap/dist/css/bootstrap.min.css" rel="stylesheet">
   </head>
   <body>
       <div class="container mt-5">
           <h1>Hello, Bootstrap!</h1>
       </div>
       <script src="node_modules/bootstrap/dist/js/bootstrap.bundle.min.js"></script>
   </body>
   </html>
   ```

### 方法三：使用 Visual Studio Code

1. 下载并安装 VS Code：https://code.visualstudio.com/
2. 安装推荐扩展：
   - Bootstrap 5 Quick Snippets
   - HTML CSS Support
   - Live Server
3. 创建新 HTML 文件，输入 `bs5` 即可快速生成 Bootstrap 模板

## 安装后配置

### 1. 配置 Sass 自定义（可选）

创建自定义 Bootstrap 样式：

```scss
// custom.scss

// 1. 自定义变量（在引入 Bootstrap 之前定义）
$primary: #ff6b6b;
$secondary: #4ecdc4;
$font-family-sans-serif: 'Noto Sans SC', sans-serif;
$border-radius: 0.5rem;

// 2. 引入 Bootstrap
@import "~bootstrap/scss/bootstrap";

// 3. 自定义样式
.my-custom-class {
    // 自定义样式
}
```

### 2. 使用 CSS 变量

```css
:root {
    --bs-primary: #007bff;
    --bs-body-bg: #f5f5f5;
}

.custom-class {
    background-color: var(--bs-primary);
}
```

### 3. 配置 Popper（用于下拉菜单等）

Bootstrap 5 的 JavaScript 文件已包含 Popper，无需额外安装。

## 常见问题

### 1. 样式不生效

**检查清单：**
- [ ] CSS 文件链接是否正确
- [ ] CDN 链接是否正确
- [ ] 文件路径是否正确
- [ ] 浏览器缓存是否需要清除

**解决方案：**
```html
<!-- 添加版本号强制刷新 -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css?v=1" rel="stylesheet">
```

### 2. JavaScript 功能不工作

**检查清单：**
- [ ] 是否引入了 bootstrap.bundle.min.js（不是 bootstrap.min.js）
- [ ] JS 文件是否在 body 结束前引入
- [ ] 是否有 JavaScript 控制台错误
- [ ] jQuery 引入顺序是否正确（Bootstrap 5 不需要 jQuery）

### 3. 下拉菜单不工作

**解决方案：**
- 确保引入了 Popper.js（包含在 bootstrap.bundle.min.js 中）
- 检查 data-bs-toggle 属性

```html
<!-- 正确的下拉菜单 -->
<div class="dropdown">
    <button class="btn btn-secondary dropdown-toggle" type="button" data-bs-toggle="dropdown">
        下拉菜单
    </button>
    <ul class="dropdown-menu">
        <li><a class="dropdown-item" href="#">操作一</a></li>
        <li><a class="dropdown-item" href="#">操作二</a></li>
    </ul>
</div>
```

### 4. 模态框不工作

**检查清单：**
- [ ] 模态框是否在页面加载完成前初始化
- [ ] 是否有 z-index 冲突
- [ ] 是否有多个相同 ID 的模态框

### 5. 移动端响应式问题

**确保添加视口 meta 标签：**
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### 6. RTL（从右到左）支持

**启用 RTL：**
```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.rtl.min.css" rel="stylesheet">
```
