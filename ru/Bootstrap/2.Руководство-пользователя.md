# Руководство пользователя Bootstrap

Это руководство содержит подробную информацию о основных компонентах Bootstrap.

## Компоненты

### Кнопки

`html
<button class=""btn btn-primary"">Основная</button>
<button class=""btn btn-secondary"">Вторичная</button>
`

### Карточки

`html
<div class=""card"" style=""width: 18rem;"">
    <div class=""card-body"">
        <h5 class=""card-title"">Заголовок</h5>
    </div>
</div>
`
