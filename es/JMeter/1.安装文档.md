# Guía de Instalación de JMeter

JMeter es una herramienta de pruebas de rendimiento de código abierto desarrollada por Apache. Esta guía cubre los métodos de instalación en varios entornos.

## Información de Versión

| Versión | Estado | Descripción |
|------|------|------|
| JMeter 5.x | ✅ Actual | Última versión |

## Requisitos del Sistema

### Requisitos de Hardware

| Requisito | Configuración |
|------|------|
| CPU | 2+ núcleos |
| Memoria | 4+ GB |
| Disco | 5+ GB |

### Requisitos de Software

| Software | Versión |
|------|------|
| Java | JDK 8 o superior |

## Métodos de Instalación

### Método 1: Instalación en Windows

#### 1. Descarga

1. Visite https://jmeter.apache.org/download.cgi
2. Descargue la versión Binaries
3. Extraiga al directorio deseado

#### 2. Ejecución

```powershell
# Haga doble clic en bin\jmeter.bat
# O ejecute desde línea de comandos
jmeter.bat
```

### Método 2: Instalación en macOS

#### 1. Usando Homebrew

```bash
brew install jmeter
```

#### 2. Instalación manual

```bash
# Descargue y extraiga
wget https://archive.apache.org/dist/jmeter/binaries/apache-jmeter-5.6.3.tgz
tar -xzf apache-jmeter-5.6.3.tgz

# Ejecute
./apache-jmeter-5.6.3/bin/jmeter
```

### Método 3: Instalación en Linux

```bash
# Ubuntu/Debian
sudo apt-get install jmeter

# O descargar e instalar
wget https://archive.apache.org/dist/jmeter/binaries/apache-jmeter-5.6.3.tgz
tar -xzf apache-jmeter-5.6.3.tgz
cd apache-jmeter-5.6.3/bin
./jmeter
```

### Método 4: Docker

```bash
# Ejecutar contenedor JMeter
docker run -v $(pwd):/workspace justb4/jmeter -n -t test.jmx -l result.jtl
```

## Configuración del Entorno

### Configuración de Java

```bash
# Verificar Java
java -version

# Establecer variables de entorno
export JAVA_HOME=/path/to/java
export PATH=$JAVA_HOME/bin:$PATH
```

## Preguntas Frecuentes

### ¿Memoria insuficiente?

Modifique jmeter.bat o jmeter:

```bash
# Establecer tamaño de heap
HEAP="-Xms1g -Xmx2g"
```

### ¿Inicio lento de GUI?

Use el modo de línea de comandos o incremente la memoria.
