# Element UI 安裝指南

Element UI 是一個用於 Vue 2.x 應用程式的 UI 元件庫。

## 版本概述

| 版本 | Vue 版本 | 狀態 | 發布日期 |
|------|----------|------|----------|
| Element UI 2.x | Vue 2.6+ | 當前 | 2020 |
| Element UI 1.x | Vue 2.x | 舊版 | 2016 |

**注意：** Element UI 僅適用於 Vue 2。對於 Vue 3，請使用 Element Plus。

### Element UI vs Element Plus

| 特性 | Element UI (Vue 2) | Element Plus (Vue 3) |
|------|-------------------|---------------------|
| Vue 版本 | 2.x | 3.x |
| TypeScript | 有限 | 完全支援 |
| 狀態 | 維護模式 | 活躍開發 |

## 前置條件

### 系統需求

| 需求 | 版本 |
|------|------|
| Node.js | 8.0+ |
| npm | 5.0+ |
| Vue | 2.6+ |
| 瀏覽器 | IE 10+, Chrome, Firefox, Safari |

### 必需軟體

- **Node.js** - JavaScript 執行環境
- **npm** - 套件管理器
- **Vue CLI** - 專案腳手架（可選）

## 安裝方法

### 方法 1：npm（推薦）

```bash
# 安裝 Element UI
npm install element-ui --save
```

### 方法 2：CDN

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Element UI CDN 示例</title>
  <!-- 引入 Vue 2 -->
  <script src="https://unpkg.com/vue@2.6.14/dist/vue.js"></script>
  <!-- 引入 Element UI 樣式 -->
  <link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
  <!-- 引入 Element UI -->
  <script src="https://unpkg.com/element-ui/lib/index.js"></script>
</head>
<body>
  <div id="app">
    <el-button @click="visible = true">開啟對話框</el-button>
    <el-dialog :visible.sync="visible" title="Hello Element UI">
      <p>歡迎使用 Element UI！</p>
    </el-dialog>
  </div>

  <script>
    new Vue({
      el: '#app',
      data: {
        visible: false
      }
    })
  </script>
</body>
</html>
```

## 專案設定

### 步驟 1：建立 Vue 專案

```bash
# 使用 Vue CLI
npm install -g @vue/cli
vue create my-element-app
cd my-element-app
```

### 步驟 2：安裝 Element UI

```bash
npm install element-ui --save
```

### 步驟 3：在 main.js 中引入

#### 全量引入

```javascript
import Vue from 'vue'
import App from './App.vue'

// 引入 Element UI
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

// 使用 Element UI
Vue.use(ElementUI)

Vue.config.productionTip = false

new Vue({
  render: h => h(App)
}).$mount('#app')
```

#### 按需引入

首先，安裝插件：

```bash
npm install babel-plugin-component -D
```

配置 babel.config.js：

```javascript
module.exports = {
  presets: ['@vue/cli-plugin-babel/preset'],
  plugins: [
    [
      'component',
      {
        libraryName: 'element-ui',
        styleLibraryName: 'theme-chalk'
      }
    ]
  ]
}
```

然後引入元件：

```javascript
import { Button, Dialog, Input } from 'element-ui'

Vue.component(Button.name, Button)
Vue.component(Dialog.name, Dialog)
Vue.component(Input.name, Input)
```

## 故障排除

### 元件找不到

**問題：** 元件不顯示

**解決方案：**
1. 確保已安裝：`npm list element-ui`
2. 檢查 main.js 中的引入
3. 確認 Vue.use() 已呼叫

### 樣式不生效

**問題：** 樣式缺失

**解決方案：**
1. 引入 CSS：`import 'element-ui/lib/theme-chalk/index.css'`
2. 檢查 webpack CSS loader 配置

### IE10 相容性

```javascript
// main.js
import 'babel-polyfill'
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```

## 遷移到 Element Plus (Vue 3)

如果遷移到 Vue 3，請使用 Element Plus：

```bash
npm install element-plus --save
```

主要變化：
1. 套件名：`element-ui` → `element-plus`
2. 引入路徑：`element-ui/lib/index` → `element-plus`

## 最佳實踐

1. **使用按需引入** - 減小打包體積
2. **自訂主題** - 使用 SCSS 變數
3. **使用 Vue 2.6+** - 更好的相容性
