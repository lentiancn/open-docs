## Einführung

### Was ist Angular?

Angular ist ein TypeScript-basiertes Open-Source-Webanwendungsframework, das von Google entwickelt und gepflegt wird. Es wird zum Erstellen moderner Single-Page-Anwendungen (SPAs) verwendet.

### Angular Versionen

|---------|------------------------|--------|
| Angular 19 | 2024 | Aktuell |
| Angular 18 | 2024 | Unterstützt |
| Angular 17 | 2023 | Unterstützt |
| Angular 16 | 2023 | Unterstützt |

## Installationsmethoden

### Methode 1: Angular CLI verwenden (Empfohlen)

```bash
# Angular CLI installieren
npm install -g @angular/cli

# Neues Projekt erstellen
ng new my-app

# Entwicklungsserver starten
cd my-app
ng serve
```

### Methode 2: npm verwenden

```bash
mkdir angular-project
cd angular-project
npm init -y
npm install @angular/core @angular/common @angular/platform-browser
```

## Installation unter Windows

### Schritt 1: Node.js installieren

1. Besuchen Sie https://nodejs.org/
2. LTS-Version herunterladen
3. Installer ausführen

### Schritt 2: Angular CLI installieren

```cmd
npm install -g @angular/cli
```

### Schritt 3: Projekt erstellen

```cmd
ng new my-angular-app
cd my-angular-app
ng serve
```

Öffnen Sie http://localhost:4200

---

## Installation unter macOS

### Homebrew verwenden (Empfohlen)

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install node@18
npm install -g @angular/cli

ng new my-angular-app
cd my-angular-app
ng serve
```

## Installation unter Linux

### Ubuntu/Debian

```bash
sudo apt update && sudo apt upgrade -y
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs
sudo npm install -g @angular/cli

ng new my-angular-app
cd my-angular-app
ng serve
```

### Fedora

```bash
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -
sudo dnf install nodejs
sudo npm install -g @angular/cli
```

## Fehlerbehebung

### npm Berechtigungsfehler

```bash
# nvm verwenden
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 18
nvm use 18
```

### Port bereits belegt

```bash
ng serve --port 4201
```

### Node.js Versionskonflikt

| Angular Version | Node.js Version |
|-----------------|-------------------|
| Angular 19 | 18.19+, 20.x |
| Angular 18 | 18.19+, 20.x |
