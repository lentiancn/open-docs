# OpenClaw 사용 매뉴얼

이 매뉴얼은 OpenClaw의 기능과 사용법에 대한 자세한 안내를 제공하며, 이 강력한 AI 어시스턴트 게이트웨이를 최대한 활용하는 데 도움이 됩니다.

---

## 기본 개념

### Gateway

Gateway는 OpenClaw의 핵심 구성 요소입니다. 로컬 머신에서 모든 채널 연결, AI 에이전트 및 클라이언트를 위한 제어 평면으로 실행됩니다. 단일 Gateway 프로세스는 여러 채팅 채널과 여러 AI 에이전트를 동시에 처리할 수 있습니다.

### 채널

채널은 WhatsApp, Telegram, Discord, iMessage 등을 포함한 AI 어시스턴트와 상호작용하는 데 사용하는 채팅 플랫폼을 말합니다. 각 채널은 독립적으로 구성和管理할 수 있습니다.

### 에이전트

에이전트는 AI 어시스턴트의 인스턴스입니다. 각 에이전트는 자체 작업 공간, 세션 및 구성을 가집니다. 개인용, 업무용 등 다양한 목적으로 여러 에이전트를 설정할 수 있습니다.

### 세션

세션은 대화 컨텍스트를 관리합니다. OpenClaw는 여러 세션 모드를 지원합니다:
- **main**: 공유 세션
- **per-peer**: 각 연락처별 별도 세션
- **per-channel-peer**: 각 채널-연락처 조합별 별도 세션
- **per-account-channel-channel**: 가장 세분화됨

## 일상적인 사용

### Gateway 시작 및 중지

**시작:**

```bash
# 포그라운드에서 실행 (디버깅용)
openclaw gateway --port 18789

# 백그라운드에서 실행
openclaw gateway start
```

**중지:**

```bash
openclaw gateway stop
```

**상태 확인:**

```bash
openclaw gateway status
```

### 제어 UI 사용

Gateway를 시작한 후 브라우저에서 http://127.0.0.1:18789/를 방문하여 제어 UI를 엽니다.

### CLI 명령

**메시지 보내기:**

```bash
openclaw message send --target +15555550123 --message "안녕하세요, OpenClaw"
```

**AI와 채팅:**

```bash
openclaw agent --message "Hello World 프로그램 작성 도와줘" --thinking high

## 채널 관리

### WhatsApp 구성

```bash
openclaw channels login
```

### Telegram 구성

```json5
{
  channels: {
    telegram: {
      botToken: "당신의Bot토큰",
    },
  },
}
```

### Discord 구성

```json5
{
  channels: {
    discord: {
      token: "당신의Bot토큰",
    },
  },
}
```

## 고급 기능

### 멀티 에이전트 라우팅

```json5
{
  agents: {
    list: [
      { id: "home", default: true, workspace: "~/.openclaw/workspace-home" },
      { id: "work", workspace: "~/.openclaw/workspace-work" },
    ],
  },
}
```

### Cron 작업

```json5
{
  cron: {
    enabled: true,
    maxConcurrentRuns: 2,
  },
}
```

### 샌드박스 모드

```json5
{
  agents: {
    defaults: {
      sandbox: {
        mode: "non-main",
        scope: "agent",
      },
    },
  },
}
```

## 일반적인 명령

| 명령 | 설명 |
|------|------|
| `openclaw --version` | 버전 보기 |
| `openclaw status` | 상태 보기 |
| `openclaw gateway start` | Gateway 시작 |
| `openclaw gateway stop` | Gateway 중지 |
| `openclaw dashboard` | 제어 UI 열기 |
| `openclaw channels login` | 채널에 로그인 |
| `openclaw config get` | 구성 가져오기 |
| `openclaw config set` | 구성 설정 |
| `openclaw doctor` | 문제 진단 |

---

## 채팅 명령

- `/status` - 세션 상태 보기
- `/new` 또는 `/reset` - 세션 재설정
- `/compact` - 세션 컨텍스트 압축
- `/think` - 사고 수준 설정
- `/verbose on|off` - 상세 출력 전환
