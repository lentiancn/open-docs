# Element UI

> 基於 Vue 2 的桌面端元件庫

---

## 說明

Element UI 是基於 Vue 2.0 的桌面端元件庫，由餓了麼團隊開發和維護。

**注意**: 如果是 Vue 3 項目，請使用 [Element Plus](https://github.com/element-plus/element-plus)

---

## 版本

| 版本 | Vue | 狀態 |
|------|-----|------|
| Element UI 2.x | Vue 2 | 穩定 |
| Element UI 1.x | Vue 2 | 已廢棄 |
