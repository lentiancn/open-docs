# OpenClaw 簡介

## 什麼是 OpenClaw？

OpenClaw 是一個**自托管的閘道**，連接聊天應用（WhatsApp、Telegram、Discord、iMessage）到 AI 編程助手（如 Pi）。

## 主要功能

- **多通道支持**：WhatsApp、Telegram、Discord、iMessage
- **Web 控制台**：瀏覽器直接管理和聊天
- **多代理路由**：支持多個 AI 代理
- **移動節點**：iOS、Android 設備配對
- **安全控制**：白名單、權限管理

## 系統要求

- Node.js 22+
- API 密鑰（推薦 Anthropic）
- 5 分鐘配置時間

## 快速開始

```bash
# 安裝
npm install -g openclaw@latest

# 初始化
openclaw onboard --install-daemon

# 啟動
openclaw gateway --port 18789
```

訪問 http://127.0.0.1:18789/

## 適用人群

- 開發者想要個人 AI 助手
- 技術愛好者重視數據控制
- 團隊共享 AI 編程助手

## 工作原理

```
聊天應用 → Gateway → AI 代理
```

## 總結

OpenClaw 是功能強大的自托管 AI 閘道，連接各種聊天應用與 AI 助手。
