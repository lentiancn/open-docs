# Element UI FAQ

Частые вопросы и ответы.

## Проблемы установки

### Q: Установка не удалась?

**Решения:**
1. Проверьте версию Node.js
```bash
node --version  # Требуется 12.0.0+
```
2. Очистите кэш npm
```bash
npm cache clean --force
```
3. Используйте зеркало Taobao
```bash
npm config set registry https://registry.npmmirror.com
```
4. Удалите node_modules и переустановите
```bash
rm -rf node_modules
npm install
```

### Q: Модуль не найден?

**Решения:**
1. Подтвердите, что element-ui установлен
```bash
npm list element-ui
```
2. Проверьте package.json на наличие зависимостей

### Q: Версия Vue несовместима?

**Решения:**
Element UI 2.x требует Vue 2.6.0+:
```bash
npm install vue@2
```

## Проблемы использования

### Q: Компоненты не отображаются?

**Решения:**
1. Подтвердите, что компоненты правильно импортированы
```javascript
import { Button } from 'element-ui'
Vue.use(Button)
```
2. Проверьте консоль на ошибки
3. Подтвердите правильность имён компонентов

### Q: Стили не применяются?

**Решения:**
1. Подтвердите, что файл стилей импортирован
```javascript
import 'element-ui/lib/theme-chalk/index.css'
```
2. Проверьте глобальные переопределения стилей
3. Проверьте уровни z-index

### Q: Таблица не показывает данные?

**Решения:**
1. Проверьте формат данных
2. Подтвердите определения колонок
3. Проверьте реактивность данных

### Q: Валидация формы не работает?

**Решения:**
1. Импортируйте Form и FormItem
2. Установите атрибут prop
3. Определите правила в data

### Q: Параметры выпадающего списка не отображаются?

**Решения:**
1. Проверьте данные параметров
2. Подтвердите правильность сопоставления value и label

### Q: Данные диалога не сбрасываются после закрытия?

**Решения:**
1. Используйте v-if для управления диалогом
2. Вручную сбрасывайте форму при закрытии

## Проблемы стилей

### Q: Конфликты стилей?

**Решения:**
1. Используйте более конкретные селекторы
2. Проверьте уровни z-index
3. Используйте !important для переопределения (не рекомендуется)

### Q: Иконки не отображаются?

**Решения:**
1. Подтвердите, что element-ui-icons установлен
```bash
npm install element-ui-icons
```
2. Правильно импортируйте

### Q: Цвет темы неверный?

**Решения:**
1. Проверьте глобальные переопределения стилей
2. Подтвердите правильный порядок импорта переменных SCSS
3. Очистите кэш браузера

## Проблемы интернационализации

### Q: Переключение языка не работает?

**Решения:**
1. Правильно настройте i18n
2. Подтвердите импорт языкового пакета

## Проблемы производительности

### Q: Страница загружается медленно?

**Решения:**
1. Используйте импорт по требованию
2. Используйте CDN импорт
3. Включите сжатие Gzip

### Q: Таблица с большими данными тормозит?

**Решения:**
1. Используйте виртуальную прокрутку
2. Загружайте данные постранично
3. Отключите ненужные функции

## Проблемы сборки

### Q: Стили теряются после сборки?

**Решения:**
1. Проверьте настройку vue.config.js
2. Подтвердите правильное положение импорта стилей

### Q: Компоненты не отображаются в продакшене?

**Решения:**
1. Проверьте вызов Vue.use()
2. Подтвердите полноту импорта по требованию

## Проблемы обновления

### Q: Как обновить с 1.x до 2.x?

**Решения:**
1. Прочитайте официальное руководство по миграции
2. Постепенно заменяйте несовместимые API
3. Запустите тесты для проверки

### Q: Что использовать для Vue 3?

**Решения:**
Element UI для Vue 2. Для Vue 3 используйте Element Plus:
```bash
npm install element-plus
```
