# Guía de Instalación de Angular

> Configuración

---

## Requisitos

| Entorno | Requisito |
|---------|-----------|
| Node.js | 18.10.0+ |
| npm | 9.0.0+ |

---

## Instalación

```bash
npm install -g @angular/cli
ng new my-app
ng serve
```

---

## Comandos

| Comando | Descripción |
|--------|-------------|
| ng new | Nuevo proyecto |
| ng serve | Servidor dev |
| ng build | Compilar |
