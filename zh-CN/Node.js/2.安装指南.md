# Node.js 安装指南

## 下载 Node.js

Node.js 提供了适用于各种操作系统的官方安装包。访问 [Node.js 官方网站](https://nodejs.org/zh-cn/download) 下载最新版本。

### 版本选择

- **LTS (长期支持) 版本**：推荐用于生产环境，提供稳定的 API 和长期支持
- **Current (当前) 版本**：包含最新功能，适合开发和测试环境

### Windows 安装

1. 访问 [Node.js 下载页面](https://nodejs.org/zh-cn/download)
2. 下载 Windows 安装程序 (.msi)
3. 运行安装程序并按照向导完成安装
4. 验证安装：
   ```bash
   node --version
   npm --version
   ```

### macOS 安装

#### 方法一：使用官方安装包
1. 下载 macOS 安装包 (.pkg)
2. 双击运行安装程序
3. 按照向导完成安装

#### 方法二：使用 Homebrew
```bash
# 安装 LTS 版本
brew install node@lts

# 或安装最新版本
brew install node
```

### Linux 安装

#### Ubuntu/Debian
```bash
# 使用包管理器（可能不是最新版本）
sudo apt update
sudo apt install nodejs npm

# 或使用 NodeSource 仓库获取最新版本
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs
```

#### CentOS/RHEL/Fedora
```bash
# 使用包管理器
sudo dnf install nodejs npm

# 或使用 NodeSource 仓库
curl -fsSL https://rpm.nodesource.com/setup_lts.x | sudo bash -
sudo yum install nodejs
```

### 验证安装

安装完成后，验证 Node.js 和 npm 是否正确安装：

```bash
# 检查 Node.js 版本
node --version

# 检查 npm 版本  
npm --version

# 运行简单的测试
node -e "console.log('Hello Node.js!')"
```

### 管理多个 Node.js 版本

在开发中可能需要在不同项目中使用不同的 Node.js 版本。推荐使用以下工具：

#### nvm (Node Version Manager)
- **macOS/Linux**: 
  ```bash
  curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
  ```
- **Windows**: 使用 [nvm-windows](https://github.com/coreybutler/nvm-windows)

常用命令：
```bash
# 列出可用版本
nvm list available

# 安装特定版本
nvm install 18.17.0

# 切换版本
nvm use 18.17.0

# 设置默认版本
nvm alias default 18.17.0
```

### 环境变量配置

确保 Node.js 和 npm 的可执行文件路径已添加到系统 PATH 环境变量中。

- **Windows**: 通常自动添加到 PATH
- **macOS/Linux**: 如果使用包管理器安装，通常会自动配置

如果命令未找到，请手动添加到 PATH：
```bash
# 通常位于 /usr/local/bin 或 ~/.nvm/versions/node/vXX.XX.XX/bin
export PATH=$PATH:/usr/local/bin
```

### 故障排除

#### 权限问题
在 Linux/macOS 上，如果遇到 npm 全局安装的权限问题，建议：
- 使用 nvm 管理 Node.js
- 或配置 npm 的全局目录：
  ```bash
  mkdir ~/.npm-global
  npm config set prefix '~/.npm-global'
  export PATH=~/.npm-global/bin:$PATH
  ```

#### 网络问题
如果下载速度慢或无法连接，可以配置 npm 使用国内镜像：
```bash
# 设置淘宝镜像
npm config set registry https://registry.npmmirror.com

# 或临时使用
npm install --registry https://registry.npmmirror.com
```
