# 简介

Bootstrap 是全球最受欢迎的 HTML、CSS 和 JavaScript 开源前端开发框架，由 Twitter 公司的设计师 Mark Otto 和 Jacob Thornton 主导开发。Bootstrap 提供了丰富的组件和强大的 JavaScript 插件，能够帮助开发者快速构建响应式 Web 应用程序。

Bootstrap 的核心理念是"移动优先"（Mobile First），从设计之初就考虑到了各种设备的适配问题。通过 Bootstrap，开发者可以轻松创建在桌面、平板、手机等不同设备上都能完美显示的网站。

## 发展历程

Bootstrap 最初名为 Twitter Blueprint，于 2011 年开源发布。2012 年发布 Bootstrap 2，引入了响应式布局。2013 年发布 Bootstrap 3，完全重写为扁平化设计。2015 年发布 Bootstrap 4，开始使用 Sass 作为 CSS 预处理器。2021 年发布 Bootstrap 5，移除了 jQuery 依赖并全面拥抱原生 JavaScript。

## 核心特性

### 响应式布局

Bootstrap 的网格系统是其最强大的功能之一。通过 12 列网格布局，开发者可以轻松创建适应不同屏幕尺寸的页面布局。Bootstrap 5 提供了 xs、sm、md、lg、xl、xxl 六个断点，分别对应不同宽度的设备。

### 丰富的组件

Bootstrap 提供了大量可直接使用的 UI 组件，包括导航栏、卡片、按钮、表单、模态框、轮播图、下拉菜单等。这些组件都经过精心设计，具有良好的视觉效果和交互体验。

### 强大的 JavaScript 插件

Bootstrap 包含 20 多个 JavaScript 插件，如模态框、轮播图、工具提示、弹出框、折叠面板等。这些插件都是可选的，开发者可以按需引入，减小项目体积。

### 基于 Sass

Bootstrap 5 使用 Sass 作为 CSS 预处理器，提供了大量的 Sass 变量和混入（Mixins）。开发者可以轻松定制 Bootstrap 的主题颜色、字体大小等样式。

### 自定义主题

通过 Bootstrap 的主题系统，开发者可以轻松创建自己的主题风格。官方还提供了 Bootstrap Themes 市场，有大量精美的主题可供选择。

### 无障碍支持

Bootstrap 组件内置了无障碍支持，包括键盘导航、屏幕阅读器兼容等，帮助开发者创建更具包容性的网站。

## Bootstrap 5 新特性

相比 Bootstrap 4，Bootstrap 5 带来了许多重大更新：

- 移除了 jQuery 依赖，全面使用原生 JavaScript
- 改进了网格系统，增加了 XXL 断点
- 新的表单控件替代了表单验证
- 更好的自定义属性支持
- 图标库从 Font Awesome 改为 Bootstrap Icons
- RTL（从右到左）支持

## 适用场景

- 快速原型开发
- 企业内部管理系统
- 电子商务网站
- 博客和内容网站
- 管理后台界面
- 响应式网站开发

作为全球最受欢迎的 CSS 框架之一，Bootstrap 拥有庞大的社区和丰富的学习资源，是前端开发者必备的技能之一。
