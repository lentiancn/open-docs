# Yarn よくある質問

## 基本的な質問

### Yarnとは？

YarnはJavaScriptのパッケージマネージャーで、依存関係のインストール、更新、削除を Helpします。

### Yarnとnpmの違いは？

- **速度**：Yarnは並行インストールで更快
- **確実性**：yarn.lockが毎回同じインストールを保証
- **オフラインサポート**：優れたオフラインキャッシュ
- **ワークスペース**：ネイティブワークスペースサポート

### Yarnは免费ですか？

はい、完全に免费でオープンソース（MITライセンス）です。

---

## インストール

### インストールに失敗した場合は？

1. Node.jsバージョンを確認：`node --version`
2. corepackを使用：`corepack enable`

---

## 使用

### 依存関係を追加するには？

```bash
yarn add lodash
yarn add -D typescript
```

---

## ワークスペース

### ワークスペースとは？

単一のリポジトリで複数のパッケージを管理できます。
