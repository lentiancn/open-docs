# SadTalker インストールガイド

このガイドでは、システムにSadTalkerをインストールして設定する方法を説明します。

## システム要件

### ハードウェア
- **GPU**：8GB+ VRAM付きNVIDIA GPU（推奨）
- **RAM**：最小8GB、推奨16GB
- **ストレージ**：モデルと出力用に20GB+

### ソフトウェア
- Python 3.8+
- CUDA 11.3+
- cuDNN 8.0+
- FFmpeg

## インストール

### 1. リポジトリをクローン

```bash
git clone https://github.com/WinfredSadTalker/SadTalker.git
cd SadTalker
```

### 2. Conda環境を作成

```bash
conda create -n sadtalker python=3.8
conda activate sadtalker
```

### 3. 依存関係をインストール

```bash
# PyTorchとCUDAをインストール
pip install torch==1.12.1 torchvision==0.13.1 --extra-index-url https://download.pytorch.org/whl/cu113

# 基本的な依存関係をインストール
pip install matplotlib scipy tqdm
pip install opencv-python
pip install librosa soundfile

# face alignmentをインストール
pip install face-alignment

# その他の依存関係をインストール
pip install -r requirements.txt
```

### 4. 事前学習済みモデルをダウンロード

```bash
bash scripts/download_models.sh
```

### 5. インストールを確認

```bash
python inference.py --help
```
