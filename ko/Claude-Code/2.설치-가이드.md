# Claude Code 설치 가이드

Claude Code 설치 방법详细介绍.

## Windows 설치

### 방법 1: VS Code 확장

1. VS Code 열기
2. "Claude Code" 확장 검색
3. 설치 클릭
4. Anthropic 계정에 로그인

### 방법 2: 독립 클라이언트

1. https://claude.com/code 방문
2. Windows 클라이언트 다운로드
3. 설치 프로그램 실행
4. 로그인

## macOS 설치

### 방법 1: VS Code 확장

1. VS Code 열기
2. "Claude Code" 확장 검색
3. 설치 클릭
4. 로그인

### 방법 2: Homebrew

```
brew install claude-code
```

### 방법 3: 독립 클라이언트

1. https://claude.com/code 방문
2. macOS 클라이언트 다운로드
3. 설치 프로그램 실행
4. 로그인

## Linux 설치

### 방법 1: VS Code 확장

1. VS Code 열기
2. "Claude Code" 확장 검색
3. 설치 클릭
4. 로그인

### 방법 2: 패키지 관리자

**Ubuntu/Debian**:
```
curl -fsSL https://downloads.anthropic.com/claude-code/install.sh | sh
```

**Arch Linux**:
```
yay -S claude-code
```

## 설치 확인

```
claude --version
```

## 구성

### API 키

1. https://console.anthropic.com 방문
2. API 키 생성
3. Claude Code에서 구성

### 환경 변수

```
export ANTHROPIC_API_KEY=your-api-key
```

## 제거

### macOS

```
brew uninstall claude-code
```

### Linux

```
sudo rm -rf /usr/local/claude-code
```
