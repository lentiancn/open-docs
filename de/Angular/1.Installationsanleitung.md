## Systemanforderungen

### Hardwareanforderungen
- **Prozessor**: Dual-Core-Prozessor oder besser
- **RAM**: Mindestens 4GB (8GB empfohlen)
- **Festplattenplatz**: Mindestens 3GB freier Speicher

### Softwareanforderungen

#### Für Windows
- Windows 10 oder höher
- Node.js 18.19 oder höher (LTS-Version empfohlen)
- npm 9.x oder höher (wird mit Node.js geliefert)
- Git für Versionskontrolle

#### Für macOS
- macOS 10.15 (Catalina) oder höher
- Node.js 18.19 oder höher (LTS-Version empfohlen)
- npm 9.x oder höher
- Git (wird mit Xcode Command Line Tools geliefert)

#### Für Linux
- Ubuntu 20.04 LTS oder höher / Debian-basierte Distributionen
- Fedora 36 oder höher / RHEL-basierte Distributionen
- Node.js 18.19 oder höher (LTS-Version empfohlen)
- npm 9.x oder höher
- Git

## Voraussetzungen

### 1. Node.js installieren

Node.js ist die Laufzeitumgebung, die für die Ausführung von Angular erforderlich ist.

#### Installation unter Windows

**Methode A: Node.js-Installer verwenden**
1. Besuchen Sie https://nodejs.org/
2. Laden Sie die **LTS**-Version herunter
3. Führen Sie den Installer aus
4. Überprüfen Sie die Installation:
   ```
   node --version
   npm --version
   ```

**Methode B: Chocolatey verwenden**
```powershell
choco install nodejs-lts -y
```

**Methode C: nvm-windows verwenden (empfohlen)**
```powershell
nvm install 18.19.0
nvm use 18.19.0
```

#### Installation unter macOS

**Methode A: Homebrew verwenden (empfohlen)**
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install node@18
```

#### Installation unter Linux

**Ubuntu/Debian**
```bash
sudo apt update
sudo apt install curl build-essential
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs
```

**Fedora**
```bash
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -
sudo dnf install nodejs
```

### 2. Git installieren

#### Windows
Von https://git-scm.com/ herunterladen

#### macOS
```bash
brew install git
```

#### Linux
```bash
sudo apt install git
```

## Installation unter Windows

1. **Node.js installieren** - Von https://nodejs.org/ herunterladen
2. **Git installieren** - Von https://git-scm.com/ herunterladen
3. **Angular CLI installieren**: `npm install -g @angular/cli`
4. **Projekt erstellen**:
   ```bash
   ng new my-first-app
   cd my-first-app
   ng serve
   ```

---

## Installation unter macOS

1. Homebrew installieren
2. Node.js installieren: `brew install node@18`
3. Git installieren: `brew install git`
4. Angular CLI installieren: `npm install -g @angular/cli`

---

## Installation unter Linux

### Ubuntu/Debian

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install curl build-essential git
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs
sudo npm install -g @angular/cli
```

## Fehlerbehebung

### NPM-Berechtigungsfehler
```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
```

### Port 4200 belegt
```bash
ng serve --port 4201
```
