# V-Express Guía de Uso

Esta guía proporciona instrucciones detalladas sobre cómo usar V-Express.

## Uso Básico

### Generar Video Parlante

```bash
python inference.py \
    --reference_image path/to/image.jpg \
    --audio path/to/audio.wav \
    --output output.mp4
```

### Con Video de Referencia

```bash
python inference.py \
    --reference_image path/to/image.jpg \
    --audio path/to/audio.wav \
    --reference_video path/to/video.mp4 \
    --output output.mp4
```

## Parámetros

| Parámetro | Descripción | Valor por Defecto |
|-----------|-------------|-------------------|
| `--reference_image` | Imagen de referencia | Requerido |
| `--audio` | Archivo de audio | Requerido |
| `--reference_video` | Video de referencia | Opcional |
| `--output` | Archivo de salida | output.mp4 |
| `--expression_strength` | Fuerza de expresión | 1.0 |
| `--pose_strength` | Fuerza de pose | 1.0 |
| `--blink_frequency` | Frecuencia de parpadeo | 0.5 |

## Ejemplos

### Ejemplo Básico

```bash
python inference.py \
    --reference_image examples/face.png \
    --audio examples/speech.wav \
    --output result.mp4
```

### Con Control de Expresión

```bash
python inference.py \
    --reference_image examples/face.png \
    --audio examples/speech.wav \
    --expression_strength 0.7 \
    --pose_strength 0.5 \
    --output result.mp4
```

### Con Video de Referencia

```bash
python inference.py \
    --reference_image examples/face.png \
    --audio examples/speech.wav \
    --reference_video examples/reference.mp4 \
    --output result.mp4
```

## Mejores Prácticas

### Imágenes de Entrada
1. Usa imágenes de alta resolución (512x512 o más)
2. Asegura buena iluminación
3. La cara debe estar orientada hacia adelante
4. Evita fotos muy editadas o filtradas

### Audio de Entrada
1. Usa audio limpio sin ruido de fondo
2. La voz debe ser clara y audible
3. Formatos recomendados: WAV, MP3

## Resolución de Problemas

### Mala Sincronización de Labios
- Usa audio de alta calidad
- Asegura audio sin ruido
- Ajusta parámetros de extracción

### Distorsión de Identidad
- Usa imagen de alta resolución
- Evita fotos muy editadas
- Asegura buena iluminación

### Artefactos en Salida
- Reduce resolución de salida
- Verifica instalación correcta de modelos

### Errores de Memoria
- Reduce tamaño de batch
- Baja resolución de salida
- Cierra otras aplicaciones GPU
