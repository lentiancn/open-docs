## О Docker

Docker — это платформа с открытым исходным кодом, которая обеспечивает быстрое развёртывание, масштабирование и запуск приложений с помощью технологии контейнеризации.

### Основные функции

- **Лёгкость**: Контейнеры используют ядро операционной системы хоста без дополнительной нагрузки ОС
- **Переносимость**: Собрал один раз — запускай везде
- **Изоляция**: Каждый контейнер изолирован друг от друга
- **Масштабируемость**: Поддерживает горизонтальное масштабирование и автоматическое развёртывание

## Быстрый старт

### Установка Docker

Подробные инструкции по установке см. в [Руководстве по установке](./1.Руководство-по-установке.md).

### Запуск первого контейнера

```bash
# Загрузить и запустить образ hello-world
docker run hello-world
```

### Запуск Nginx

```bash
# Запустить контейнер Nginx
docker run -d -p 8080:80 nginx

# Открыть http://localhost:8080
```
