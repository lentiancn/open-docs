# Element Plus

Element Plus est une bibliothèque de composants UI basée sur Vue 3.

## Caractéristiques

- Support Vue 3 avec Composition API
- 70+ composants
- Support TypeScript
- Importation à la demande
- Internationalisation

## Démarrage rapide

npm install element-plus
