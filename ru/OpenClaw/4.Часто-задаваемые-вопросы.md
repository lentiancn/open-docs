# Часто задаваемые вопросы

Ответы на частые вопросы об OpenClaw.

## Основы

### Что такое OpenClaw?

OpenClaw — это инструмент для подключения мессенджеров к ИИ-ассистентам. Позволяет использовать ИИ через уже установленные приложения, сохраняя полный контроль над данными.

### OpenClaw бесплатен?

Программа бесплатна (открытый исходный код). Но нужно оплатить:
- Компьютер или сервер для работы
- API-ключи для ИИ-сервисов

### Нужен ли подписочный ИИ-сервис?

Не обязательно. Можно использовать API-ключи с оплатой по факту или подписку. OpenClaw поддерживает Anthropic, OpenAI, Google Gemini и других.

### На каких платформах работает?

- **Десктоп**: macOS, Linux, Windows
- **Мобильные**: iOS, Android (как узлы)
- **Мессенджеры**: WhatsApp, Telegram, Discord, iMessage, Signal, Slack и ещё 20+

## Установка

### Установка не удалась. Что делать?

1. Проверьте версию Node.js: `node --version` (нужна 22+)
2. Запустите диагностику: `openclaw doctor`
3. Проверьте логи: `openclaw logs`

### Работает на Raspberry Pi?

Да. Raspberry Pi 4 или новее с Raspberry Pi OS или Ubuntu.

### Нужен Mac?

Нет. OpenClaw работает на любой системе с Node.js. Только для iMessage нужен Mac.

## Настройка

### Где конфигурационный файл?

По умолчанию: `~/.openclaw/openclaw.json`

### Как ограничить кто может писать?

Используйте параметр `dmPolicy`:
- `pairing`: незнакомцы получают код подключения
- `allowlist`: только разрешённые пользователи
- `open`: все могут писать
- `disabled`: отключить личные сообщения

### Как настроить групповые чаты?

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

## Использование

### Как добавить новый канал?

```bash
openclaw channels login whatsapp
```

### Можно использовать несколько каналов одновременно?

Да. Один Gateway подключается к нескольким платформам.

### ИИ запоминает предыдущие разговоры?

OpenClaw автоматически сохраняет контекст. Можно настроить срок хранения.

### Можно запустить несколько ИИ-агентов?

Да. Настройте несколько агентов с разными правилами:

```json5
{
  agents: {
    list: [
      { id: "home", default: true },
      { id: "work" }
    ]
  }
}
```

## Безопасность

### Мои данные защищены?

Да. OpenClaw работает локально — данные остаются на вашем устройстве.

### Как защитить установку?

1. Настройте `allowFrom` для ограничения источников
2. Используйте сложные пароли и API-ключи
3. Обновляйте OpenClaw
4. Настройте файрвол

## Проблемы

### Gateway не запускается

1. Проверьте занят ли порт: `openclaw gateway status`
2. Проверьте логи: `openclaw logs`
3. Запустите диагностику: `openclaw doctor`

### ИИ не отвечает

1. Проверьте API-ключ
2. Проверьте баланс счёта
3. Проверьте логи
