# LivePortrait 사용 가이드

LivePortrait는驱动 비디오의 모션을 사용하여 정적 초상을 애니메이션화하는 효율적인 초상 애니메이션 프레임워크입니다. 스티칭 및 리타겟팅을 지원합니다.

## 빠른 시작

### 기본 명령

```bash
# 비디오驱动 애니메이션
python inference.py \
  --source_image examples/source_image.jpg \
  --driving_video examples/driving_video.mp4 \
  --result_dir ./results
```

### 스티칭 포함

```bash
python inference.py \
  --source_image examples/source_image.jpg \
  --driving_video examples/driving_video.mp4 \
  --result_dir ./results \
  --stitching
```

## 명령 매개변수

### 입력 매개변수

| 매개변수 | 설명 | 기본값 |
|---------|------|-------|
| `--source_image` | 입력 초상 이미지 경로 | 필수 |
| `--driving_video` |驱动 비디오 경로 | 선택 |
| `--result_dir` | 출력 디렉토리 | ./results |

### 처리 매개변수

| 매개변수 | 설명 | 기본값 |
|---------|------|-------|
| `--stitching` | 스티칭 활성화 | False |
| `--eye_retargeting` | 눈 리타겟팅 (0-1) | 0 |
| `--lip_retargeting` | 입술 리타겟팅 (0-1) | 0 |

## 사용 예

### 예제 1: 기본 비디오驱动 애니메이션

```bash
python inference.py \
  --source_image examples/source_image.jpg \
  --driving_video examples/driving_video.mp4 \
  --result_dir results/basic
```

### 예제 2: 스티칭 포함

```bash
python --source_image examples inference.py \
 /source_image.jpg \
  --driving_video examples/driving_video.mp4 \
  --result_dir results/stitching \
  --stitching
```

## 기능

### 1. 스티칭

스티칭 모듈은 생성된 머리를 몸체와 원활하게 연결합니다:

```bash
--stitching
```

### 2. 눈 리타겟팅

스칼라 값으로 눈 공개 정도 제어 (0-1):

```bash
--eye_retargeting 0.5
```

### 3. 입술 리타겟팅

입술 움직임 강도 제어:

```bash
--lip_retargeting 0.5
```

## 입력 요구 사항

### 소스 이미지

- 형식: JPG, PNG
- 해상도: 512x512 이상 권장
- 얼굴: 정면, 선명

###驱动 비디오

- 형식: MP4, AVI
- 재생 시간: 1-60초

## 출력

- 형식: MP4 (H.264)
- 해상도: 512x512
- 프레임 속도: 30
