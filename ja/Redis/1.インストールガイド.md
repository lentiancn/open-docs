# Redis インストールガイド

このガイドでは、システムにRedisをインストールして設定する方法を説明します。

## 前提条件

- Linux、macOS、またはWindows（WSL）
- GCCコンパイラ（ソースからビルドする場合）

## インストール方法

### Ubuntu/Debian

```bash
sudo apt-get update
sudo apt-get install redis-server
```

### macOS

```bash
brew install redis
```

### CentOS/RHEL

```bash
sudo yum install redis
```

## ソースからビルド

```bash
# ダウンロード
wget https://github.com/redis/redis/archive/7.2.0.tar.gz
tar -xzf 7.2.0.tar.gz
cd redis-7.2.0

# ビルド
make

# インストール
sudo make install
```

## Redisの起動

### サーバーを起動

```bash
redis-server
```

### クライアントに接続

```bash
redis-cli
```

## 設定

### 設定ファイル

Redis設定ファイルは通常 `/etc/redis/redis.conf` または `/usr/local/etc/redis.conf` にあります。

### 基本的な設定オプション

```conf
# ポート
port 6379

# バインドアドレス
bind 127.0.0.1

# パスワード認証
requirepass yourpassword

# 永続化
save 900 1
save 300 10
save 60 10000
```

## 基本的な操作

### キーの操作

```bash
# 値を設定
SET mykey "Hello Redis"

# 値を取得
GET mykey

# キーを削除
DEL mykey
```

### データ型の操作

```bash
# リスト
LPUSH mylist "item1"
LPUSH mylist "item2"
LRANGE mylist 0 -1

# セット
SADD myset "member1"
SADD myset "member2"
SMEMBERS myset
```
