# Docker 常見問題

## Q: Docker Desktop 啟動失敗？

確保 BIOS 中已啟用虛擬化，並正確安裝 WSL 2（Windows）。

---

## Q: 鏡像下載太慢？

配置國內鏡像加速器：

```json
{
  "registry-mirrors": [
    "https://docker.mirrors.ustc.edu.cn"
  ]
}
```

---

## Q: 權限不足？

```bash
sudo usermod -aG docker $USER
# 重新登入
```

---

## Q: 容器無法啟動？

```bash
docker logs container_id
docker inspect container_id
```

---

## Q: 如何清理未使用的資源？

```bash
docker system prune -a
```
