# Claude Code 소개

Claude Code는 코드베이스를 읽고, 파일을 편집하고, 명령을 실행하며 개발 도구와 원활하게 통합할 수 있는 Anthropic의 지능형 프로그래밍 도우미입니다. 터미널, IDE, 데스크톱 앱 및 브라우저에서 사용할 수 있습니다.

## Claude Code란?

Claude Code는 기능 구축, 버그 수정 및 개발 작업 자동화를 도와주는 AI 기반 프로그래밍 도우미입니다. 전체 코드베이스를 이해하고 여러 파일과 도구에서 작업할 수 있습니다.

## 주요 기능

- **작업 자동화**: 테스트 작성, 린트 오류 수정, 병합 충돌 해결, 의존성 업데이트
- **기능 개발**: 자연어로 요구사항을 설명하세요; Claude Code가 계획하고 구현합니다
- **Git 통합**: 커밋, 브랜치, 풀 리퀘스트 생성
- **MCP 통합**: 외부 도구 및 데이터 소스에 연결
- **사용자 정의**: CLAUDE.md, 사용자 정의 명령, 훅

## 사용 시나리오

| 시나리오 | 권장 방법 |
|----------|-----------|
| 로컬 개발 | 터미널 CLI |
| IDE 코딩 | VS Code 또는 JetBrains |
| 장치 간 계속하기 | 원격 제어 |
| 모바일 작업 | 웹 또는 iOS 앱 |
| CI/CD 자동화 | GitHub Actions 또는 GitLab CI/CD |

## 계정 요구 사항

Claude Code에는 다음 중 하나가 필요합니다:
- Claude Pro, Max, Teams 또는 Enterprise 구독
- Anthropic Console 계정
