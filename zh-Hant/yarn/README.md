# Yarn 套件管理器

Yarn 是一個成熟的開源套件管理器，用於管理 JavaScript 專案中的相依性。

## 文件結構

- [1.簡介.md](1.簡介.md) - Yarn 概述和主要版本對比
- [2.安裝指南.md](2.安裝指南.md) - Yarn v4+ 和 v1 的詳細安裝方法
- [3.使用手冊.md](3.使用手冊.md) - 基本命令、工作區、設定檔等使用說明
- [4.常見問題.md](4.常見問題.md) - 安裝、使用、遷移和故障排除的常見問題解答

## 主要版本資訊

### Yarn v4+ (現代版本)
- **官網**: https://yarnpkg.com/
- **狀態**: 活躍開發中
- **特點**: 外掛式架構、工作區原生支援、PnP、離線快取

### Yarn v1 (Classic)
- **官網**: https://classic.yarnpkg.com/
- **狀態**: 維護中
- **特點**: 傳統架構、簡單直接、廣泛相容
