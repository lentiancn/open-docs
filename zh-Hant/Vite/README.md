# Vite

Vite 是新一代前端建構工具。

## 簡介

Vite 是由 Vue.js 作者尤雨溪開發的新一代前端建構工具。它利用瀏覽器原生 ES 模組支援，提供極速的開發體驗。

## 主要特性

- **極速開發**：基於 ESM 的開發伺服器
- **熱模組替換**：即時熱更新
- **按需編譯**：只編譯需要的檔案
- **豐富的插件**：支援 Vue、React、Svelte 等
- **生產優化**：內建程式碼分割和壓縮

## 快速開始

### 安裝

```bash
npm create vite@latest my-app -- --template vue
cd my-app
npm install
npm run dev
```
