# Guide d'Installation de Flutter

## Exigences

Avant d'installer Flutter, assurez-vous que votre environnement de développement répond aux exigences suivantes. En termes de systèmes d'exploitation, Flutter prend en charge Windows, macOS et Linux. Windows nécessite Windows 10 ou supérieur, macOS nécessite macOS 10.14 ou supérieur, et Linux nécessite un environnement de bureau supportant GNOME ou KDE.

En termes d'espace disque, le SDK Flutter occupe environ 1 Go, mais en tenant compte des projets de développement et du cache de compilation, il est recommandé de réservez au moins 10 Go d'espace disponible. En termes de mémoire, 8 Go ou plus offrent une expérience de développement plus fluide.

Pour le développement mobile, le développement iOS nécessite macOS et Xcode, le développement Android nécessite l'installation d'Android Studio et Android SDK. Le développement Web et bureau n'a pas d'exigences particulières, il suffit d'installer le SDK Flutter.

## Étapes d'Installation

### Installation Windows

Il y a deux méthodes principales pour installer Flutter sous Windows. La première consiste à télécharger l'archive précompilée depuis le site Web de Flutter, l'extraire à un emplacement approprié, puis ajouter le répertoire flutter/bin à la variable d'environnement système PATH.

Méthode de téléchargement : Visitez le site Web Flutter sur flutter.dev et téléchargez la dernière version stable. Extrayez dans C:\src\flutter ou un autre emplacement approprié. Ouvrez Propriétés système → Avancé → Variables d'environnement, et dans les variables utilisateur Path, ajoutez le chemin du répertoire bin de Flutter.

Après la fin, ouvrez l'invite de commande et exécutez la commande flutter doctor pour vérifier si l'installation a réussi. Flutter doctor vérifiera l'état du SDK Flutter et donnera des suggestions de configuration.

### Installation macOS

Sous macOS, l'installation basée sur Git est recommandée. Assurez-vous d'abord qu'Xcode est installé, puis ouvrez Terminal et exécutez la commande suivante pour cloner Flutter :

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

Ajoutez Flutter au PATH :

```bash
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.bash_profile
```

Pour les utilisateurs zsh :

```bash
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.zshrc
```

Exécutez ensuite source ~/.bash_profile ou source ~/.zshrc pour actualiser les variables d'environnement.

### Installation Linux

L'installation sous Linux est similaire à macOS. Installez d'abord les dépendances :

```bash
sudo apt-get update
sudo apt-get install curl git unzip xz-utils zip libglu1-mesa clang cmake ninja-build pkg-config libgtk-3-dev
```

Clonez ensuite le dépôt Flutter et configurez le PATH, comme pour macOS.

## Vérifier l'Installation

Après l'installation, exécutez les commandes suivantes pour vérifier :

```bash
flutter --version
flutter doctor
```

Flutter doctor affichera l'état de l'installation et les dépendances manquantes. Complétez les configurations nécessaires selon les indications, comme l'installation d'Android Studio et la configuration d'Android SDK.

## Configuration IDE

### VS Code

VS Code est l'un des éditeurs recommandés pour Flutter. Après avoir installé VS Code, ouvrez la place de marché des extensions, recherchez et installez les extensions Flutter et Dart. Après l'installation, redémarrez VS Code pour utiliser les fonctionnalités liées à Flutter.

### Android Studio

Après avoir installé Android Studio, ouvrez la page Plugins, recherchez le plugin Flutter et installez-le. Après l'installation, configurez le chemin du SDK Flutter : Ouvrez Preferences → Language & Frameworks → Flutter et définissez le chemin du SDK Flutter.

## Démarrage Rapide

Créez votre premier projet Flutter :

```bash
flutter create my_first_app
cd my_first_app
flutter run
```

Ou utilisez la palette de commandes (Ctrl+Maj+P) dans VS Code et entrez Flutter: New Project pour créer un nouveau projet.

Exécutez flutter devices pour voir les appareils connectés, et flutter run -d device_id pour exécuter sur un appareil spécifique.
