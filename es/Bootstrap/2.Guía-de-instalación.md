# Guía de Instalación

Esta guía te muestra las diferentes formas de instalar Bootstrap para que puedas empezar a usarlo en tu proyecto rápidamente.

## Requisitos del Sistema

Bootstrap 5 necesita lo siguiente:

- **HTML5** y navegadores modernos
- **CSS3** con soporte
- **JavaScript** ES6+ (si usas plugins de JavaScript)

## Formas de Instalar

### Opción 1: CDN (Recomendado)

La forma más sencilla es usar un CDN para incluir directamente los archivos CSS y JavaScript de Bootstrap:

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Página con Bootstrap</title>
    <!-- Incluir el CSS de Bootstrap -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>¡Hola, Bootstrap!</h1>

    <!-- Incluir el JavaScript de Bootstrap (incluye Popper) -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### Opción 2: npm

Si usas npm para gestionar las dependencias de tu proyecto:

```bash
# Instalar Bootstrap
npm install bootstrap@5

# Instalar Popper (para menús desplegables, tooltips, etc.)
npm install @popperjs/core
```

Luego lo incluyes en tu proyecto:

```javascript
// Incluir el CSS de Bootstrap
import 'bootstrap/dist/css/bootstrap.min.css';

// Incluir el JS de Bootstrap
import 'bootstrap/dist/js/bootstrap.bundle.min.js';
```

### Opción 3: Descargar los Archivos

Descarga los archivos precompilados desde el sitio oficial:

1. Visita https://getbootstrap.com/
2. Haz clic en el botón "Download"
3. Elige "Compiled CSS and JS" para descargar
4. Descomprime y copia las carpetas css y js a tu proyecto

Estructura de directorios:

```
tu-proyecto/
├── css/
│   └── bootstrap.min.css
├── js/
│   └── bootstrap.bundle.min.js
└── index.html
```

### Opción 4: Código Fuente Sass

Si necesitas personalizar los estilos de Bootstrap, puedes descargar el código fuente Sass:

```bash
npm install bootstrap
```

Luego lo incluyes en tu archivo Sass:

```scss
// Tus variables personalizadas (defínelas antes de importar Bootstrap)
$primary: #007bff;
$body-bg: #f8f9fa;

// Importar Bootstrap
@import "~bootstrap/scss/bootstrap";
```

## Herramientas de Desarrollo

### Extensiones para VS Code

Te recomendamos instalar estas extensiones:

- **Bootstrap 5 Snippets**: Ofrece fragmentos de código para componentes Bootstrap
- **HTML CSS Support**: Autocompletado de nombres de clases CSS
- **Live Server**: Vista previa en tiempo real de tu página

### Editores Online

- **Bootstrap Builder**: Construye páginas Bootstrap visualmente
- **Bootstrap Studio**: Herramienta profesional de diseño para Bootstrap

## Primeros Pasos

### Crear tu Primera Página

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Inicio Rápido con Bootstrap</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <!-- Barra de navegación -->
    <nav class="navbar navbar-expand-lg navbar-dark bg-primary">
        <div class="container">
            <a class="navbar-brand" href="#">Mi Sitio Web</a>
        </div>
    </nav>

    <!-- Contenido principal -->
    <main class="container mt-5">
        <div class="row">
            <div class="col-md-6">
                <h1>Bienvenido a Bootstrap</h1>
                <p class="lead">Esta es una página responsiva construida con Bootstrap 5.</p>
                <button class="btn btn-primary">Saber más</button>
            </div>
            <div class="col-md-6">
                <div class="card">
                    <div class="card-body">
                        <h5 class="card-title">Características</h5>
                        <ul>
                            <li>Diseño responsivo</li>
                            <li>Componentes variados</li>
                            <li>Potentes plugins de JavaScript</li>
                        </ul>
                    </div>
                </div>
            </div>
        </div>
    </main>

    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## Elegir la Versión

### Para Producción

Usa una versión estable:

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### Para Desarrollo

Usa la última versión de desarrollo (incluye código fuente):

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.js"></script>
```

### Mirror Chino

En China puedes usar un mirror local para mayor velocidad:

```html
<link href="https://cdn.bootcss.com/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.bootcss.com/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

## Desinstalar

Si necesitas eliminarlo (solo borra los archivos relacionados):

- Elimina las referencias a CSS y JS en tu HTML
- Elimina la carpeta bootstrap de node_modules (si usaste npm)
- Borra los archivos CSS y JS descargados localmente
