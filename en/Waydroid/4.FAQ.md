# FAQ

## Google Play Certification

Waydroid uses a customized Android image based on LineageOS, which doesn't include Google Play Services by default. If Google Play certification is needed, you can install a GAPPS image.

## Disable On-Screen Keyboard

In some cases, you may want to disable the on-screen keyboard. This can be achieved through system settings or Waydroid configuration.

## Get Waydroid to Work Through a VM

To run Waydroid through a virtual machine, ensure that the VM supports nested virtualization and has hardware acceleration properly configured.

## Using ADB with Waydroid

Waydroid supports ADB debugging. By default, ADB secure mode is enabled and won't auto-connect. Use the following command to connect:

```bash
waydroid adb connect
```

Then provide authorization on the Android side.

## Setting Up Waydroid-Only Sessions

You can configure dedicated sessions that only run Waydroid, suitable for specific use cases.

## Setting Up a Shared Folder

Waydroid supports setting up shared folders for easy file transfer between the Linux host and Android container.

## Color Correction in Mutter

When using the Mutter window manager, you might encounter color display issues that can be corrected with specific configurations.

## Backup & Restore Apps and Data

Waydroid supports backing up and restoring applications and data to ensure important data isn't lost.

## Using Custom Waydroid Images

To use custom images:

1. Download or build custom `system.img` and `vendor.img`
2. Copy to `/etc/waydroid-extra/images/` directory
3. Re-initialize: `sudo waydroid init -f`

## Install System Apps

System-level applications can be installed but require special permissions and configuration steps.

## Hardware Compatibility

Waydroid has certain hardware requirements:
- Requires Wayland display server support
- Requires Linux kernel support for namespaces and container features
- Graphics drivers need to support OpenGL ES 3.2

## Performance Optimization

For optimal performance:
- Ensure hardware acceleration is enabled
- Use appropriate graphics drivers
- Run on supported CPUs (avoid pure ARM emulation)
