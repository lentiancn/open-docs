# Guía de Instalación de JDK

JDK (Java Development Kit) es el kit de desarrollo oficial de Java. Esta guía cubre los métodos de instalación para varios entornos.

## Información de Versión

| Versión | Estado | Descripción |
|---------|--------|-------------|
| JDK 21 (LTS) | ✅ Actual | Última versión LTS |
| JDK 17 (LTS) | ✅ Soportada | LTS anterior |
| JDK 11 (LTS) | ✅ Soportada | LTS legacy |
| JDK 8 | ✅ Soportada | Versión legacy |
| JDK 22 | ✅ Última | Última versión de funciones |

### Versiones LTS vs Versiones de Funciones

- **LTS (Soporte a Largo Plazo)**: 2-3 años de actualizaciones, recomendado para producción
- **Versiones de Funciones**: Nuevas funciones cada 6 meses, soporte más corto

## Métodos de Instalación

### Método 1: Oracle JDK

#### Windows

1. Descargar desde: https://www.oracle.com/java/technologies/downloads/
2. Ejecutar el instalador (.exe)
3. Seguir el asistente

#### macOS

```bash
# Descargar DMG desde el sitio de Oracle
# O usar Homebrew
brew install openjdk@21
```

#### Linux

```bash
# Descargar tar.gz
wget https://download.oracle.com/java/21/latest/jdk-21_linux-x64_bin.tar.gz
tar -xzf jdk-21_linux-x64_bin.tar.gz
sudo mv jdk-21 /opt/jdk-21
```

### Método 2: Eclipse Temurin (Adoptium)

#### Windows

1. Descargar desde: https://adoptium.net/temurin/releases/
2. Ejecutar el instalador

#### macOS

```bash
# Usando Homebrew
brew install --cask temurin

# O descargar pkg desde el sitio web
```

#### Linux

```bash
# Ubuntu/Debian
wget https://github.com/adoptium/temurin21-binaries/releases/download/jdk-21.0.2%2B13/OpenJDK21U-jdk_x64_linux_hotspot_21.0.2_13.tar.gz
tar -xzf OpenJDK21U-jdk_x64_linux_hotspot_21.0.2_13.tar.gz

# Configurar entorno
export JAVA_HOME=/path/to/jdk-21
export PATH=$JAVA_HOME/bin:$PATH
```

### Método 3: SDKMAN

```bash
# Instalar SDKMAN
curl -s "https://get.sdkman.io" | bash

# Instalar JDK
sdk install java 21.0.2-tem

# Listar versiones disponibles
sdk list java
```

### Método 4: Docker

```dockerfile
FROM eclipse-temurin:21-jdk
WORKDIR /app
COPY . .
RUN javac Main.java
CMD ["java", "Main"]
```

### Método 5: Gestores de Paquetes de Windows

#### Chocolatey

```powershell
choco install openjdk21
```

#### Scoop

```powershell
scoop install openjdk
```

## Configuración del Entorno

### Windows

```powershell
# Establecer JAVA_HOME
[System.Environment]::SetEnvironmentVariable("JAVA_HOME", "C:\Program Files\Java\jdk-21", "User")

# Agregar a PATH
[System.Environment]::SetEnvironmentVariable("PATH", "$env:PATH;C:\Program Files\Java\jdk-21\bin", "User")
```

### macOS/Linux

```bash
# Agregar a ~/.bashrc o ~/.zshrc
export JAVA_HOME=/opt/jdk-21
export PATH=$JAVA_HOME/bin:$PATH

# Aplicar
source ~/.bashrc
```

## Verificación

```bash
# Verificar versión de Java
java -version

# Verificar versión de Javac
javac -version

# Verificar JAVA_HOME
echo $JAVA_HOME
```

## Múltiples Versiones de JDK

### Usando SDKMAN

```bash
# Instalar múltiples versiones
sdk install java 17.0.8-tem
sdk install java 21.0.2-tem

# Cambiar versión
sdk use java 17.0.8-tem
```

## Problemas Comunes

### ¿"java" no encontrado?

Verifique que PATH incluya el directorio bin de Java.

### ¿Versión de Java incorrecta?

Verifique qué Java se está usando:

```bash
which java
java -version
```

### ¿JAVA_HOME no está establecido?

Establezca JAVA_HOME en el directorio de instalación del JDK.
