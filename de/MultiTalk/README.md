# MultiTalk

MultiTalk ist ein audio-gesteuertes Framework zur Erzeugung von Mehrpersonen-Gesprachsvideos, das vom MeiGen-AI-Team entwickelt wurde. Dieses Projekt kann hochwertige Videoinhalte mit Mehrpersoneninteraktionen basierend auf Multi-Stream-Audio-Eingaben, Referenzbildern und Textprompts generieren.

## Hauptmerkmale

- Audio-gesteuerte Mehrpersonen-Gesprachsvideogenerierung
- Unterstuetzung fuer Einzelpersonen- und Mehrpersonen-Szenarien
- Branchenfuehrende Lippensynchronisationsgenauigkeit
- Unterstuetzung fuer Zeichentrick- und Sing-Videogenerierung
- Ausgabeauflosung unterstuetzt 480P und 720P
- Maximale Videoclip-Laenge bis zu 15 Sekunden
