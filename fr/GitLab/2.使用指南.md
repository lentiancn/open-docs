# Guide d'utilisation GitLab

## Cloner

```bash
git clone https://gitlab.example.com/user/projet.git
```

## Opérations de base

```bash
git checkout -b feature/nouveau
git add .
git commit -m "Nouvelle fonctionnalité"
git push origin feature/nouveau
```
