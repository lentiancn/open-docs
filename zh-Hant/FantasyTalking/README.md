# FantasyTalking

FantasyTalking 是一个专注于语音合成和对话系统的技术项目。

## 概述

提供自然、流畅的语音交互体验，适用于智能客服、有声读物、语音助手等场景。

## 主要特性

- 语音合成
- 对话管理
- 多平台支持
- 可定制性

## 快速开始

### 安装

```bash
npm install fantasy-talking
```

### 使用

```javascript
const ft = new FantasyTalking({
  apiKey: 'your-api-key'
})

ft.speak('你好，世界')
```
