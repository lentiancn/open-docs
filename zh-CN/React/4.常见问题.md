# React 常见问题

常见问题解答。

## 安装问题

### Q: 创建项目失败？

**解决方案**：
1. 检查 Node.js 版本
2. 清理 npm 缓存

## 开发问题

### Q: 组件不更新？

**解决方案**：
1. 检查状态修改方式
2. 使用 useEffect 处理副作用

### Q: 无限循环？

**解决方案**：
1. 检查 useEffect 依赖数组
2. 避免在 useEffect 中直接修改状态

### Q: 内存泄漏？

**解决方案**：
1. 清理 useEffect 中的订阅
2. 取消 fetch 请求

## 性能问题

### Q: 应用太慢？

**解决方案**：
1. 使用 useMemo/useCallback
2. 使用 React.memo
3. 懒加载组件

## 状态问题

### Q: 状态不更新？

**解决方案**：
1. 使用 setState 的函数形式
2. 确保状态不可变
