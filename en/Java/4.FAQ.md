## Installation

### javac command not found?

**Solution**:
1. Ensure JDK is installed correctly
2. Check JAVA_HOME environment variable
3. Add `%JAVA_HOME%\bin` to PATH

### How to switch Java versions?

**Windows**: Change JAVA_HOME

**macOS**:
```bash
export JAVA_HOME=$(/usr/libexec/java_home -v 17)
```

**Linux**:
```bash
sudo update-alternatives --config java
```

## Development

### JDK vs JRE?

- **JRE**: Java Runtime Environment (run Java programs)
- **JDK**: Java Development Kit (JRE + development tools)

### == vs equals()?

- `==`: Compare references
- `equals()`: Compare object content

```java
String a = new String("hello");
String b = new String("hello");

a == b        // false
a.equals(b)   // true
```

## Collections

### ArrayList vs LinkedList?

- **ArrayList**: Array-based, fast random access, slow insert/delete
- **LinkedList**: List-based, fast insert/delete, slow random access

### HashMap vs Hashtable?

- **HashMap**: Not thread-safe, allows null key/value
- **Hashtable**: Thread-safe, doesn't allow null

## Concurrency

### How to create threads?

1. Extend Thread class
2. Implement Runnable interface (recommended)
3. Use thread pool

### synchronized vs Lock?

| Feature | synchronized | Lock |
|---------|-------------|------|
| Type | Keyword | Interface |
| Release | Automatic | Manual |
| Flexibility | Less | More |

## JVM

### Memory areas?

- **Heap**: Object instances
- **Stack**: Local variables, method parameters
- **Method area**: Class info, constants
- **Program counter**: Execution position
