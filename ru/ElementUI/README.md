# Element UI

Библиотека компонентов Vue.js 2.0 для десктопа.

## Обзор

Element UI — это библиотека компонентов с открытым исходным кодом для Vue.js 2.0, разработанная командой Ele.me, предоставляющая полный набор элегантных UI-компонентов.

### Основные функции

- **Богатые компоненты**: 60+ компонентов
- **Красивый дизайн**: Рекомендации Material Design
- **Поддержка TypeScript**: Полные определения
- **Импорт по требованию**: Поддержка tree shaking
- **Настройка тем**: Переменные SCSS
- **Интернационализация**: Поддержка нескольких языков

## Быстрый старт

### Установить

```bash
npm install element-ui --save
```

### Импортировать

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```

## Частые компоненты

- **Базовые**: Button, Layout, Container

- **Данные**: Table, Tree, Pagination
