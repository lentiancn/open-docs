# LivePortrait インストールガイド

LivePortraitは、Kwai VGIによって開発される効率的なポートレートアニメーションフレームワークです。駆動ビデオのモーションを使用して、静的なポートレートをアニメーション化します。

## システム要件

### ハードウェア要件

| コンポーネント | 最低 | 推奨 |
|----------------|--------|---------|
| GPU | NVIDIA 8GB VRAM | NVIDIA 16GB VRAM |
| RAM | 8GB | 16GB |
| ストレージ | 30GB | 50GB |
| CUDA | 11.8+ | 12.1+ |

### ソフトウェア要件

| ソフトウェア | バージョン | メモ |
|--------------|-----------|------|
| Python | 3.8 - 3.11 | 推奨3.10 |
| CUDA | 11.8+ | GPU加速が必要 |
| ffmpeg | 最新 | ビデオ処理に必要 |

## Linux/Ubuntu インストール

### ステップ1：依存関係をインストール

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install python3.10 python3-pip git wget curl
sudo apt install libgl1-mesa-glx libglib2.0-0 libsm6 libxext6 libxrender-dev libgomp1
sudo apt install ffmpeg
```

### ステップ2：プロジェクトをクローン

```bash
git clone https://github.com/KwaiVGI/LivePortrait.git
cd LivePortrait
```

### ステップ3：仮想環境を作成

```bash10 -m v
python3.env venv
source venv/bin/activate
```

### ステップ4：PyTorchをインストール

```bash
pip install torch==2.1.0 torchvision==0.16.0 --index-url https://download.pytorch.org/whl/cu121
```

### ステップ5：依存関係をインストール

```bash
pip install -r requirements.txt
```

### ステップ6：モデルをダウンロード

```bash
mkdir -p pretrained_models
bash scripts/download_weights.sh
```

## 確認

```bash
python inference.py --help
```

## 一般的な問題

### CUDA メモリ不足

**解決策：**
- バッチサイズを削減：`--batch_size 1`
- より低い解像度を使用
