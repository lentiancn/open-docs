# Element Plus User Manual

This manual will help you quickly get started with Element Plus and master commonly used components and features.

## Quick Start

Before using Element Plus, make sure installation is complete.

### Basic Usage

Use components directly in Vue components:

```vue
<template>
  <el-button>Default Button</el-button>
  <el-button type="primary">Primary Button</el-button>
  <el-button type="success">Success Button</el-button>
  <el-button type="warning">Warning Button</el-button>
  <el-button type="danger">Danger Button</el-button>
</template>
```

## Common Components

### Button

Basic usage:

```vue
<el-button>Default</el-button>
<el-button type="primary">Primary</el-button>
<el-button type="success">Success</el-button>
<el-button type="warning">Warning</el-button>
<el-button type="danger">Danger</el-button>
```

Different sizes:

```vue
<el-button size="large">Large</el-button>
<el-button>Default</el-button>
<el-button size="small">Small</el-button>
```

Disabled state:

```vue
<el-button disabled>Disabled</el-button>
```

Icon buttons:

```vue
<el-button icon="Search">Search</el-button>
<el-button icon="Edit">Edit</el-button>
<el-button icon="Delete">Delete</el-button>
```

### Input

Basic usage:

```vue
<el-input v-model="input" placeholder="Please enter content" />
```

With icon:

```vue
<el-input v-model="input" prefix-icon="Search" />
<el-input v-model="input" suffix-icon="Calendar" />
```

Clearable:

```vue
<el-input v-model="input" clearable />
```

Password input:

```vue
<el-input v-model="input" type="password" show-password />
```

Textarea:

```vue
<el-input v-model="textarea" type="textarea" :rows="3" />
```

### Select

Basic select:

```vue
<el-select v-model="value" placeholder="Please select">
  <el-option label="Option 1" value="1" />
  <el-option label="Option 2" value="2" />
</el-select>
```

Multiple select:

```vue
<el-select v-model="value" multiple placeholder="Please select">
  <el-option label="Option 1" value="1" />
  <el-option label="Option 2" value="2" />
</el-select>
```

### Table

Basic table:

```vue
<el-table :data="tableData" stripe border>
  <el-table-column prop="date" label="Date" width="180" />
  <el-table-column prop="name" label="Name" width="180" />
  <el-table-column prop="address" label="Address" />
</el-table>
```

With index:

```vue
<el-table-column type="index" label="No." width="60" />
```

Fixed columns:

```vue
<el-table-column prop="date" label="Date" width="150" fixed />
```

### Form

Basic form:

```vue
<el-form :model="form" label-width="80px">
  <el-form-item label="Activity Name">
    <el-input v-model="form.name" />
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="onSubmit">Submit</el-button>
  </el-form-item>
</el-form>
```

Form validation:

```vue
<el-form :model="ruleForm" :rules="rules" ref="ruleFormRef">
  <el-form-item label="Name" prop="name">
    <el-input v-model="ruleForm.name" />
  </el-form-item>
</el-form>
```

### Dialog

```vue
<el-dialog v-model="visible" title="Title" width="30%">
  <span>Content</span>
</el-dialog>
```

### Message

```javascript
import { ElMessage } from 'element-plus'

ElMessage.success('Success message')
ElMessage.warning('Warning message')
ElMessage.error('Error message')
```

### Menu

Horizontal menu:

```vue
<el-menu mode="horizontal">
  <el-menu-item index="1">Processing Center</el-menu-item>
  <el-menu-item index="2">Order Management</el-menu-item>
</el-menu>
```

Vertical menu:

```vue
<el-menu default-active="1">
  <el-menu-item index="1">Navigation One</el-menu-item>
  <el-menu-item index="2">Navigation Two</el-menu-item>
</el-menu>
```

### Tabs

```vue
<el-tabs v-model="activeName">
  <el-tab-pane label="User Management" name="first">User Management</el-tab-pane>
  <el-tab-pane label="Config Management" name="second">Config Management</el-tab-pane>
</el-tabs>
```

### Pagination

```vue
<el-pagination
  v-model:current-page="currentPage"
  :page-size="100"
  :total="1000"
  layout="total, prev, pager, next"
/>
```

## Layout System

### Grid System

Element Plus uses 24-column grid system:

```vue
<el-row :gutter="20">
  <el-col :span="12"><div>12 columns</div></el-col>
  <el-col :span="12"><div>12 columns</div></el-col>
</el-row>
```

### Container Layout

```vue
<el-container>
  <el-header>Header</el-header>
  <el-main>Main</el-main>
  <el-footer>Footer</el-footer>
</el-container>
```

## Common Issues

### Component Styles Not Applied

Make sure CSS is imported correctly:

```javascript
import 'element-plus/dist/index.css'
```

### Icons Not Displayed

Need to install icon library:

```bash
npm install @element-plus/icons-vue
```

Then register globally:

```javascript
import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```

### On-Demand Import Not Working

Check if plugin configuration is correct and ensure the correct resolver is used.
