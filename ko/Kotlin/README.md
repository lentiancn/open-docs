# Kotlin

Kotlin은 JVM에서 실행되는 JetBrains가 개발한 최신 프로그래밍 언어입니다.

## 소개

Kotlin은 Java 가상 머신에서 실행되거나 JavaScript 코드로 컴파일될 수 있는 정적 타입 프로그래밍 언어입니다.

## 주요 기능

- **간결함**: 보일러플레이트 코드 감소
- **안전성**: Null 안전 설계
- **상호운용성**: Java 100% 상호운용
- **툴 친화적**:優れた IDE 지원
- **코루틴**: 강력한 비동기 프로그래밍 지원

## 빠른 시작

### 설치

```bash
# macOS
brew install kotlin

# Linux
sudo apt install kotlin

# SDKMAN
sdk install kotlin
```

### 컴파일 및 실행

```bash
kotlinc hello.kt -include-runtime -d hello.jar
java -jar hello.jar
```
