# Flutter インストールガイド

FlutterはGoogleが開発したオープンソースUIツールキットで、美しく高速なクロスプラットフォームアプリケーションの構築に使用されます。このガイドでは、すべての環境でのインストール方法について説明します。

## バージョン情報

| バージョン | 状態 | 説明 |
|------------|------|------|
| Flutter 3.x | ✅ 現行 | 最新安定版 |
| Flutter 2.x | ⚠️ メンテナンス | セキュリティ更新 |

> **重要**：このガイドは主に **Flutter 3.x** を対象としています。

## インストール方法

### 方法1：Windowsへのインストール

#### 1. Flutter SDKのダウンロード

1. Flutter SDKをダウンロード：https://docs.flutter.dev/get-started/install/windows
2. 指定のディレクトリに展開（例：C:\src\flutter）

#### 2. 環境変数の設定

```powershell
# PATHに追加
$env:PATH += ";C:\src\flutter\bin"
```

#### 3. Flutter doctorの実行

```powershell
flutter doctor
```

### 方法2：macOSへのインストール

#### 1. Homebrewの使用（推奨）

```bash
# Flutterのインストール
brew install flutter

# または手動ダウンロード
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable
```

#### 2. 環境変数の設定

```bash
# ~/.bash_profile または ~/.zshrc
export PATH="$PATH:`pwd`/flutter/bin"
```

#### 3. Flutter doctorの実行

```bash
flutter doctor
```

### 方法3：Linuxへのインストール

#### 1. 依存関係のインストール

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa clang cmake ninja-build pkg-config libgtk-3-dev

# Fedora
sudo dnf install -y curl git unzip xz zip mesa-libGLU clang cmake ninja-build pkgconfig gtk3-devel
```

#### 2. Flutter SDKのダウンロード

```bash
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable
export PATH="$PATH:~/development/flutter/bin"
```

#### 3. Flutter doctorの実行

```bash
flutter doctor
```

### 方法4：VS Codeの使用

1. VS Codeをダウンロードしてインストール
2. VS Codeを開く
3. Flutter拡張機能をインストール：
   - 「Flutter」を検索してインストール
   - 「Dart」を検索してインストール

### 方法5：Android Studioの使用

1. Android Studioをダウンロードしてインストール
2. Flutterプラグインをインストール：
   - Android Studioを開く
   - Settings → Plugins
   - 「Flutter」を検索してインストール

## Android設定

### Android SDKのインストール

```bash
# Flutterツールを使用してインストール
flutter doctor --android-licenses
flutter config --android-sdk /path/to/android/sdk
```

### 環境変数の設定

```bash
# macOS/Linux
export ANDROID_SDK_ROOT=/Users/user/Library/Android/sdk
export PATH=$PATH:$ANDROID_SDK_ROOT/cmdline-tools/latest/bin:$ANDROID_SDK_ROOT/platform-tools
```

### エミュレータの作成

```bash
# 利用可能なデバイスの一覧表示
flutter devices

# エミュレータの作成
flutter emulator --create --name my-emulator

# エミュレータの起動
flutter emulators --launch my-emulator
```

## よくある問題

### Flutter doctorがエラーになる場合

すべての依存関係がインストールされていることを確認：

```bash
# flutter doctorの実行
flutter doctor -v
```

### Android SDKが見つからない場合

SDKパスを設定：

```bash
flutter config --android-sdk /path/to/android/sdk
```

### ビルドが失敗する場合

クリーンにして再ビルド：

```bash
flutter clean
flutter pub get
flutter build apk
```
