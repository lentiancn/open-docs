# Element UI Einführung

> Element UI ist eine Vue 2-basierte Desktop-Komponentenbibliothek

---

## Übersicht

Element UI ist eine auf Vue 2.0 basierende Komponentenbibliothek vom Eleme-Team. Bietet Formulare, Tabellen, Dialoge, Navigation.

---

## Version

| Version | Vue Version | Status |
|---------|-------------|--------|
| Element UI 2.x | Vue 2 | Stabil |
| Element UI 1.x | Vue 2 | Veraltet |

---

## Funktionen

- Vue 2-basiert
- Reiche Komponenten
- Import nach Bedarf
- i18n Unterstützung
- Theme-Anpassung
