# Flutter

Flutter - 跨平台 UI 框架

## 概述

Flutter 是 Google 开发的开源 UI 软件开发工具包，用于从单个代码库构建漂亮的、本地编译的移动、Web 和桌面应用程序。

## 特性

- **跨平台**：一套代码同时支持 iOS、Android、Web、Windows、macOS、Linux
- **热重载**：快速开发迭代，保留应用状态
- **丰富组件**：Material Design 和 Cupertino 组件库
- **原生性能**：编译为原生 ARM 代码，达到原生应用性能
- **声明式 UI**：现代 UI 开发范式
- **开源免费**：BSD 许可证

## 快速开始

### 安装

```bash
# 克隆 Flutter
git clone https://github.com/flutter/flutter.git -b stable

# 添加到 PATH
export PATH="$PATH:$HOME/flutter/bin"

# 检查安装
flutter doctor
```

### 创建项目

```bash
flutter create my_app
cd my_app
flutter run
```
