# IBM Informix

IBM Informix는 IBM에서 개발한 관계형 데이터베이스 관리 시스템(RDBMS) 제품군입니다.

## 문서 개요

이 디렉토리에는 IBM Informix 기술 문서가 포함되어 있습니다:

- **소개**: 제품 개요, 기능 및 에디션 정보
- **설치 가이드**: Linux 및 Windows 플랫폼 설치 단계
- **사용자 매뉴얼**: 기본 명령어, SQL 문 및 구성
- **자주 묻는 질문**: 일반적인 질문과 해결 방법

## 버전 정보

- 현재 버전: IBM Informix 15.0.0.1 (2025년 3월)
