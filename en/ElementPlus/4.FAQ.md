# Element Plus FAQ

Common questions and answers.

## Installation

### Q: Installation failed?

**Solutions:**
1. Check Node.js version
```bash
node --version  # Requires 18.0+
```
2. Clear npm cache
```bash
npm cache clean --force
```
3. Remove node_modules and package-lock.json, reinstall
```bash
rm -rf node_modules package-lock.json
npm install
```

### Q: Module not found?

**Solutions:**
1. Confirm package is installed
```bash
npm list element-plus
```
2. Check npm/yarn/pnpm configuration
3. Try removing lock file and reinstalling

### Q: TypeScript type errors?

**Solutions:**
1. Install type definitions
```bash
npm install -D @types/node
```
2. Add to tsconfig.json
```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

## Usage

### Q: Component not showing?

**Solutions:**
1. Confirm proper import
2. Check console for errors
3. Verify Vue version (requires 3.2+)

### Q: Styles not applying?

**Solutions:**
1. Confirm CSS is imported
```typescript
import 'element-plus/dist/index.css'
```
2. Check CSS priority, use `:deep()` or `!important`
3. Check for other style overrides

### Q: Icons not showing?

**Solutions:**
1. Install icon package
```bash
npm install @element-plus/icons-vue
```
2. Register globally
```typescript
import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```
3. Or import on-demand

### Q: Table not rendering?

**Solutions:**
1. Confirm data property is correctly bound
2. Check prop is correct
3. Confirm table is outside el-table-column

### Q: Form validation not working?

**Solutions:**
1. Ensure prop matches v-model
2. Ensure has ref reference
3. Correctly call validate method
```typescript
const formRef = ref()
await formRef.value.validate()
```

### Q: Dialog not showing?

**Solutions:**
1. Check v-model binding
2. Confirm visible is true
3. Check for z-index conflicts

## Theme

### Q: How to change theme color?

**Solutions:**
Use CSS variables:
```css
:root {
  --el-color-primary: #409eff;
}
```

### Q: How to use dark theme?

**Solutions:**
Import dark CSS:
```typescript
import 'element-plus/theme-chalk/dark/css-vars.css'
```

### Q: How to override component styles?

**Solutions:**
Use CSS selectors:
```css
.el-button--primary {
  background-color: #your-color;
}
```

## Internationalization

### Q: How to switch language?

**Solutions:**
```typescript
import en from 'element-plus/es/locale/lang/en'

app.use(ElementPlus, { locale: en })
```

## Performance

### Q: Slow initial load?

**Solutions:**
1. Use on-demand import
2. Use auto-import plugin
3. Configure Vite pre-build

### Q: Bundle size too large?

**Solutions:**
1. Ensure on-demand import is used
2. Use tree shaking
3. Check if full package is imported

## Compatibility

### Q: IE browser not supported?

Element Plus does not support IE. Use:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### Q: Conflicts with other libraries?

**Solutions:**
1. Check z-index levels
2. Use CSS Modules to avoid style conflicts
3. Check Vue version compatibility
