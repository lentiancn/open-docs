# Angular 安裝指南

Angular 是由 Google 開發的一款強大的前端框架。本指南涵蓋各種環境的安裝方法。

## 版本資訊

| 版本 | 狀態 | 說明 |
|---------|--------|-------------|
| Angular 18.x | ✅ 目前版本 | 最新版本 |
| Angular 17.x | ✅ 受支援 | 上一版本 |
| Angular 16.x | ✅ 受支援 | 上一版本 |
| Angular 15.x | ✅ 受支援 | 舊版 |

## 系統需求

### 硬體需求

| 需求 | 最低配置 | 推薦配置 |
|-------------|---------|-------------|
| CPU | 2 核心 | 4 核心或以上 |
| 記憶體 | 4 GB | 8 GB 或以上 |
| 磁碟 | 3 GB | 10 GB 或以上 SSD |

### 軟體需求

| 軟體 | 版本 |
|----------|---------|
| Node.js | 18.19.1+ 或 20.10.0+ |
| npm | 10.2.3+ |

## 安裝方法

### 方法一：Angular CLI（推薦）

#### 安裝 Angular CLI

```bash
# 全域安裝
npm install -g @angular/cli

# 驗證安裝
ng version
```

#### 建立新專案

```bash
# 建立新專案
ng new my-app

# 帶選項建立
ng new my-app --routing --style=scss --skip-tests
```

#### 選項

```bash
# 跳過安裝
ng new my-app --skip-install

# 獨立元件（Angular 14+）
ng new my-app --standalone

# 啟用 SSR
ng new my-app --ssr
```

### 方法二：使用 npx

```bash
#不安裝 CLI 直接建立專案
npx @angular/cli new my-app
```

### 方法三：手動設定

```bash
# 建立目錄
mkdir my-app
cd my-app

# 初始化 npm
npm init -y

# 安裝 Angular
npm install @angular/core @angular/common @angular/platform-browser @angular/platform-browser-dynamic @angular/compiler @angular/forms @angular/router @angular/animations rxjs zone.js

# 安裝開發依賴
npm install --save-dev @angular/compiler-cli @angular-devkit/build-angular typescript
```

### 方法四：Docker

```dockerfile
# Dockerfile
FROM node:20-alpine

WORKDIR /app

# 安裝 Angular CLI
RUN npm install -g @angular/cli

# 建立專案
RUN ng new my-app --directory .

# 啟動開發伺服器
CMD ["ng", "serve", "--host", "0.0.0.0"]
```

## 環境設定

### Node.js 版本管理

#### 使用 nvm（推薦）

```bash
# 安裝 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 安裝 Node.js
nvm install 20
nvm use 20

# 設定預設版本
nvm alias default 20
```

### npm 設定

```bash
# 設定 registry
npm config set registry https://registry.npmjs.org/

# 啟用嚴格 SSL
npm config set strict-ssl true

# 清除快取
npm cache clean --force
```

## 驗證

```bash
# 檢查 Angular CLI
ng version

# 檢查 Node.js
node --version

# 檢查 npm
npm --version
```

## 常見問題

### 權限被拒絕？

```bash
# 修復 npm 權限
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
export PATH=~/.npm-global/bin:$PATH
```

### 安裝慢？

```bash
# 使用 npm ci 加速安裝
npm ci

# 或者使用 yarn
npm install -g yarn
```

## 更新 Angular

```bash
# 更新 Angular CLI
npm install -g @angular/cli@latest

# 更新專案
ng update @angular/core @angular/cli
```
