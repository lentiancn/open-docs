> Element Plus — это библиотека компонентов Vue 3 для настольных компьютеров от команды Element
> 
> Дата обновления: Февраль 2026
> 
> Текущая версия: 2.x (Стабильная)

---

## 1. Обзор проекта

Element Plus — это библиотека UI-компонентов для настольных компьютеров на базе Vue 3, разработанная и поддерживаемая командой Element. Как версия Element UI для Vue 3, Element Plus полностью написана в стиле Composition API Vue 3 и полностью переписана на TypeScript, обеспечивая безопасную типизацию при разработке компонентов.

Первая стабильная версия Element Plus была выпущена 7 февраля 2022 года. После многих лет итераций API стал очень стабильным. В настоящее время поддерживаются упрощённый китайский, английский и другие языки, а также существует активное сообщество.

## 2. Требования к окружению

### 2.1 Требования к среде разработки

| Окружение | Минимальная версия | Рекомендуемая версия |
|------|----------|----------|
| Node.js | 16.x | 18.x или 20.x |
| npm | 8.x | 9.x или 10.x |
| Vue | 3.2.0 | 3.4.x |
| Vite | 4.x | 5.x |

### 2.2 Совместимость с браузерами

Element Plus поддерживает все современные браузеры, включая:

- Chrome >= 80
- Firefox >= 78
- Safari >= 14
- Edge >= 80

Internet Explorer 11 не поддерживается.

### 2.3 Требования к технологическому стеку

| Технология | Описание |
|------|------|
| Vue 3 | Необходимо использовать Vue 3.2.0 или выше |
| TypeScript | Рекомендуется версия 4.5 или выше |
| Инструмент сборки | Vite (рекомендуется) или Webpack |

## 3. Шаги установки

### 3.1 Использование npm (Рекомендуется)

npm — это менеджер пакетов Node.js и официально рекомендуемый способ установки Element Plus.

```bash
# Перейдите в каталог вашего проекта Vue 3
cd ваш-проект-vue

# Установите Element Plus
npm install element-plus
```

### 3.2 Использование yarn

```bash
yarn add element-plus
```

### 3.3 Использование pnpm

```bash
pnpm add element-plus
```

## 4. Импорт в проект Vue 3

### 4.1 Полный импорт (подходит для небольших проектов)

Полный импорт включит все компоненты, настройка простая, но размер пакета будет больше.

```typescript
// main.ts или main.js
import { createApp } from 'vue'
import App from './App.vue'

// Импорт Element Plus и его стилей
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'

const app = createApp(App)

// Использовать Element Plus
app.use(ElementPlus)

app.mount('#app')
```

### 4.2 Импорт по требованию (рекомендуется для продакшна)

Импорт по требованию может значительно уменьшить размер пакета, импортируя только фактически используемые компоненты.

#### 4.2.1 Установка плагина автоматического импорта

Сначала установите unplugin-vue-components и unplugin-auto-import:

```bash
npm install -D unplugin-vue-components unplugin-auto-import
```

#### 4.2.2 Настройка Vite

Добавьте конфигурацию в vite.config.ts:

```typescript
// vite.config.ts
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

При такой конфигурации компоненты будут импортироваться автоматически. Например:

```vue
<template>
  <el-button type="primary">Привет</el-button>
</template>
```

## 5. Импорт через CDN (без инструментов сборки)

Если вы не хотите использовать npm или инструменты сборки, вы можете импортировать Element Plus через CDN.

### 5.1 Импорт через unpkg CDN

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Пример Element Plus CDN</title>
  <!-- Импорт Vue 3 -->
  <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
  <!-- Импорт Element Plus -->
  <link rel="stylesheet" href="https://unpkg.com/element-plus/dist/index.css">
  <script src="https://unpkg.com/element-plus"></script>
</head>
<body>
  <div id="app">
    <el-button type="primary">Привет Element Plus</el-button>
  </div>

  <script>
    const { createApp, ref } = Vue
    
    const App = {
      setup() {
        const visible = ref(false)
        return { visible }
      },
      template: `
        <el-button @click="visible = true">Открыть диалог</el-button>
        <el-dialog v-model="visible" title="Привет">Это Element Plus</el-dialog>
      `
    }
    
    const app = createApp(App)
    app.use(ElementPlus)
    app.mount('#app')
  </script>
</body>
</html>
```

## 6. Настройка интернационализации

Element Plus по умолчанию использует английский. Для использования русского языка необходимо настроить интернационализацию.

### 6.1 Глобальная русская конфигурация

```typescript
// main.ts
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import ru from 'element-plus/es/locale/lang/ru'
import 'element-plus/dist/index.css'

const app = createApp(App)

app.use(ElementPlus, {
  locale: ru,
})

app.mount('#app')
```

## 7. Настройка темы

### 7.1 Переопределение переменных SCSS

Element Plus написан на SCSS, вы можете настроить тему, переопределив переменные.

```scss
// styles/element-variables.scss
// Основной цвет
$--color-primary: #409EFF;

// Переопределение переменных компонентов
@use "element-plus/theme-chalk/src/common/var.scss" as *;
```

### 7.2 Тёмный режим

Element Plus имеет встроенную поддержку тёмного режима:

```typescript
// Включить тёмный режим
import 'element-plus/theme-chalk/dark/css-vars.css'
```

Добавьте класс к HTML-тегу:

```html
<html class="dark">
  <!-- Ваше приложение -->
</html>
```

## 8. Частые проблемы

### 8.1 Ошибка установки

**Проблема**: npm install не удаётся

**Решения**:

1. Очистить кэш npm: `npm cache clean --force`
2. Проверить сеть, можно использовать зеркала:

```bash
# Настроить зеркало npm
npm config set registry https://registry.npmmirror.com
```

### 8.2 Стили не применяются

**Проблема**: Стили компонентов отображаются некорректно

**Решения**:

1. Убедитесь, что вы импортировали файл стилей: `import 'element-plus/dist/index.css'`
2. Проверьте правильность обработки CSS в конфигурации сборки
3. Очистить кэш браузера и пересобрать
