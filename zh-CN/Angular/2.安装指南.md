# Angular 安装指南

Angular 环境配置详细指南。

## 环境要求

### Node.js

- 推荐 Node.js 18+
- 需要 npm 或 yarn

### 检查 Node.js 版本

```bash
node --version
npm --version
```

## 安装 Angular CLI

### 安装

```bash
npm install -g @angular/cli
```

### 验证安装

```bash
ng version
```

### 升级

```bash
npm install -g @angular/cli@latest
```

## 创建新项目

### 使用 Angular CLI 创建

```bash
ng new my-app
cd my-app
```

### 交互式选项

- 是否添加路由？Yes/No
- 使用哪种样式表格式？（SCSS/CSS/Sass/Less）

## 运行应用

### 开发服务器

```bash
ng serve
# 或
npm start
```

访问：http://localhost:4200

### 热重载

Angular CLI 支持热重载，修改代码后自动刷新。

## 构建生产版本

```bash
ng build --configuration production
```

构建产物位于 `dist/` 目录。

## 常用命令

| 命令 | 功能 |
|------|------|
| ng new | 创建新项目 |
| ng serve | 启动开发服务器 |
| ng build | 构建项目 |
| ng generate | 生成组件/服务/模块等 |
| ng test | 运行单元测试 |
| ng e2e | 运行端到端测试 |

## 项目结构

```
my-app/
├── src/
│   ├── app/
│   │   ├── components/
│   │   ├── services/
│   │   ├── app.component.ts
│   │   └── app.module.ts
│   ├── assets/
│   └── styles.css
├── angular.json
└── package.json
```

## 卸载 Angular CLI

```bash
npm uninstall -g @angular/cli
```
