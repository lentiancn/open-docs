# Claude Code

[Claude Code](https://code.claude.com/) es un asistente de programación inteligente desarrollado por Anthropic que puede leer bases de código, editar archivos, ejecutar comandos e integrarse perfectamente con herramientas de desarrollo.

## Resumen

Claude Code es un asistente de programación impulsado por IA disponible en terminal, IDE, aplicación de escritorio y navegador. Te ayuda a construir funciones, corregir errores y automatizar tareas de desarrollo.

## Características

- **Automatizar Tareas**: Escribir pruebas, corregir errores de lint, resolver conflictos de fusión, actualizar dependencias
- **Desarrollo de Funciones**: Describe requisitos en lenguaje natural; Claude Code planifica e implementa soluciones
- **Integración Git**: Preparar cambios, escribir mensajes de commit, crear ramas, abrir Pull Requests
- **Integración MCP**: Conectar con fuentes de datos y herramientas externas
- **Personalización**: CLAUDE.md, comandos personalizados, Hooks
- **Colaboración Multi-Agente**: Procesamiento paralelo para tareas complejas

## Inicio Rápido

### Instalar

**macOS, Linux, WSL:**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell:**
```powershell
irm https://claude.ai/install.ps1 | iex
```

### Iniciar

```bash
cd your-project
claude
```

## Requisitos de Cuenta

Claude Code requiere una de las siguientes cuentas:
- Suscripción Claude Pro, Max, Teams o Enterprise
- Cuenta de Anthropic Console
- Proveedores de nube de terceros soportados (Amazon Bedrock, Google Vertex AI, Microsoft Foundry)
