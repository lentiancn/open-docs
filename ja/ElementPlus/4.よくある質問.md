# Element Plus FAQ

## インストール問題

### インストール失敗

Node.jsバージョンが14.0.0以上であることを確認してください。

### TypeScriptエラー

tsconfig.jsonに設定を追加：

```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

## 使用問題

### スタイルが適用されない

CSSをインポートしていることを確認してください：

```javascript
import 'element-plus/dist/index.css'
```

### アイコンが表示されない

アイコンライブラリをインストール：

```bash
npm install @element-plus/icons-vue
```

## パフォーマンス問題

大数据の場合は、ページネーションを使用してください。
