# Java

## Vue d'ensemble

Java est un langage de programmation orienté objet maintenu par Oracle Corporation. C'est l'un des langages de développement d'entreprise les plus populaires au monde. Java est connu pour sa caractéristique "Écrire une fois, exécuter partout" et dispose d'un écosystème mature avec de larges domaines d'application.

## Caractéristiques principales

- Orienté objet : Langage de programmation entièrement orienté objet
- Indépendance de la plateforme : Exécution multiplateforme via JVM
- Gestion automatique de la mémoire : Mécanisme de collecte automatique des ordures
- Bibliothèque de classes riche : API standard complète
- Support multithread : Support intégré pour la programmation concurrente
- Sécurité : Mécanismes de sécurité multicouches

## Démarrage rapide

### Installer le JDK

Visitez la page de téléchargement officielle d'Oracle pour obtenir le JDK : https://www.oracle.com/java/technologies/downloads/

### Écrire votre premier programme

Créez HelloWorld.java :

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

### Compiler et exécuter

```bash
# Compiler
javac HelloWorld.java

# Exécuter
java HelloWorld
```
