# Jenkins - Einführung

## Überblick

Jenkins ist ein Open-Source-Automatisierungsserver. Es hilft bei der Automatisierung von Softwareentwicklungsaufgaben wie Build, Test und Deployment und erleichtert Continuous Integration und Continuous Delivery (CI/CD).

## Was ist Jenkins?

Jenkins ist ein eigenständiger, Open-Source-Automatisierungsserver, der verwendet werden kann, um alle Arten von Aufgaben im Zusammenhang mit dem Erstellen, Testen und Bereitstellen von Software zu automatisieren.

## Hauptmerkmale

- **Kontinuierliche Integration und Lieferung**: Build, Test und Deployment automatisieren
- **Plugin-Ökosystem**: 1800+ Plugins für erweiterte Funktionalität
- **Verteilte Builds**: Master-Slave-Architektur für verteiltes Testen
- **Einfache Konfiguration**: Webbasierte GUI für Konfiguration
- **Pipeline-Unterstützung**: Build-Pipelines als Code definieren
- **Plattformunterstützung**: Funktioniert auf Windows, Linux und macOS
