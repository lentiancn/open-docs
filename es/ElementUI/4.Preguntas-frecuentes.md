# Preguntas Frecuentes de Element UI

Esta página enumera los problemas comunes y sus soluciones al usar Element UI.

## Problemas de Instalación

### Instalación Fallida

Síntomas: npm install falla

Soluciones:

1. Verifique la versión de Node.js, asegúrese de que sea mayor o igual a 8.0.0
2. Limpie la caché de npm:
   npm cache clean --force
3. Elimine node_modules y package-lock.json, luego reinstale

### Importación bajo Demanda No Funciona

Síntomas: El tamaño del paquete sigue siendo grande

Soluciones:

1. Asegúrese de que la configuración de .babelrc sea correcta
2. Verifique si babel-plugin-component está instalado correctamente

## Problemas de Uso

### Estilos de Componentes No se Aplican

Síntomas: Los estilos de componentes no coinciden con las expectativas

Soluciones:

1. Asegúrese de que el CSS esté correctamente importado:
   import 'element-ui/lib/theme-chalk/index.css'

### Iconos No se Muestran

Síntomas: Los iconos se muestran en blanco o como cuadrados

Soluciones:

1. Importe las fuentes de iconos:
   import 'element-ui/lib/theme-chalk/fonts/element-icons.ttf'

### Problemas de Diálogo

Cuando el contenido del diálogo es demasiado largo, asegúrese de agregar desplazamiento:

<el-dialog>
  <div style="height: 400px; overflow-y: scroll;">
    Contenido
  </div>
</el-dialog>

## Problemas de Versión de Vue

### No Compatible con Vue 3

Element UI solo es compatible con Vue 2. Si necesita Vue 3, use Element Plus.

### Problemas con babel6

Si usa babel6, asegúrese de configurar correctamente:

{
  "presets": ["es2015", "stage-2"],
  "plugins": ["transform-runtime"]
}

## Problemas de Estilos

### Conflictos de Estilos

Síntomas: Los estilos personalizados se sobrescriben

Soluciones:

1. Use selectores más específicos
2. Use ::deep()

### Color de Tema No se Aplica

Verifique si las variables SCSS están correctamente sobrescritas:

$--color-primary: #409eff;

## Problemas de Rendimiento

### Lentitud en la Página

Para grandes cantidades de datos:

1. Use paginación para cargar datos
2. Evite componentes complejos en las tablas

### Tamaño del Paquete Demasiado Grande

1. Use importación bajo demanda
2. Elimine componentes no usados
