# Guide d'Utilisation FantasyTalking

Ce guide couvre comment utiliser FantasyTalking pour générer des vidéos de talking head.

## Démarrage Rapide

### Utilisation de Base

```bash
python inference.py \
  --source_image images/face.jpg \
  --audio audio/speech.wav \
  --output results/output.mp4
```

## Exigences d'Entrée

### Exigences d'Image

- Format: JPG, PNG
- Résolution: 512x512 ou supérieure
- Visage: De face, traits clairs
- Arrière-plan: Arrière-plan simple préféré

### Exigences Audio

- Format: WAV, MP3
- Fréquence d'Échantillonnage: 16000 Hz ou supérieure
- Durée: Sans limite
- Langue: Support multilingue

## Utilisation Avancée

### Amélioration du Visage

```bash
python inference.py \
  --source_image images/face.jpg \
  --audio audio/speech.wav \
  --output results/output.mp4 \
  --face_enhancer
```

### Traitement par Lots

```bash
python batch_inference.py \
  --image_dir images/ \
  --audio_dir audio/ \
  --output_dir results/
```

### FPS Personnalisé

```bash
python inference.py \
  --source_image images/face.jpg \
  --audio audio/speech.wav \
  --output results/output.mp4 \
  --fps 30
```

## Configuration

### Fichier de Configuration

Modifier `config.yaml`:

```yaml
inference:
  face_enhancer: true
  fps: 25
  output_resolution: 512
  
model:
  checkpoint: "checkpoints/fantasytalking.pth"
```

## Entraînement du Modèle

### Préparer le Jeu de Données

```bash
python prepare_dataset.py \
  --video_dir videos/ \
  --output_dir dataset/
```

### Entraînement

```bash
python train.py \
  --config config/train.yaml \
  --epochs 100
```

## Meilleures Pratiques

### 1. Utiliser des Images de Haute Qualité

- Utiliser des photos claires et de face
- Bonnes conditions d'éclairage
- Haute résolution préférée

### 2. Faire Correspondre la Qualité Audio

- Audio clair sans bruit de fond
- Fréquence d'échantillonnage appropriée
- Langue correspondant à l'ethnicité du visage

### 3. Post-Traitement

- Utiliser l'amélioration du visage pour une meilleure qualité
- Ajuster les FPS au besoin

## Dépannage

### Vidéo Trop Sombre?

Vérifier l'éclairage dans l'image source.

### Audio Non Synchronisé?

S'assurer que la fréquence d'échantillonnage audio est correcte.

### Mauvaise Qualité?

Essayer l'option d'amélioration du visage.
