# JDK Introduction

> JDK (Java Development Kit) is the software development toolkit for Java programming language

## Overview

JDK is the essential development environment for Java programmers, containing JRE (Java Runtime Environment) and development tools.

## Core Components

- **javac**: Java compiler
- **java**: Java interpreter
- **jdb**: Java debugger
- **jar**: Java archive tool
- **javap**: Java disassembler
