# Mockito インストールガイド

Mockitoのインストールと設定ガイド。

## 前提条件

- Java Development Kit (JDK) 8以上
- MavenまたはGradle（推奨）

## インストール

### Maven

`pom.xml`に以下の依存関係を追加:

```xml
<dependency>
    <groupId>org.mockito</groupId>
    <artifactId>mockito-core</artifactId>
    <version>5.8.0</version>
    <scope>test</scope>
</dependency>
```

### Gradle

`build.gradle`に依存関係を追加:

```groovy
testImplementation 'org.mockito:mockito-core:5.8.0'
```

## 設定

### JUnit 5用Mockito拡張

```java
import org.junit.jupiter.api.extension.ExtendWith;
import org.mockito.junit.jupiter.MockitoExtension;

@ExtendWith(MockitoExtension.class)
class MyTest {
    // テストをここに記述
}
```

## トラブルシューティング

### 問題: NoClassDefFoundError

**解決策:** すべての依存関係が含まれていることを確認してください。

### 問題: モックが動作しない

**解決策:**
- クラスがfinalでないか確認
- finalクラスの場合は`mockito-inline`を使用
