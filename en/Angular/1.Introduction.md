# Angular Introduction

> Angular is a TypeScript-based framework by Google

---

## Overview

Angular is a TypeScript-based framework developed and maintained by Google. First released in 2010 (AngularJS), rewritten as Angular 2+ in 2016.

---

## Features

- **TypeScript**: Native TypeScript support
- **Component-based**: Component architecture
- **Dependency injection**: Powerful DI system
- **Routing**: Complete routing solution
- **Forms**: Reactive and template-driven forms
- **HTTP**: HTTP client service
