# JUnit 5 インストールガイド

## 必要環境

- JDK 8以上
- MavenまたはGradle

## Mavenの場合

pom.xmlに追加:
```xml
<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter</artifactId>
    <version>5.10.0</version>
    <scope>test</scope>
</dependency>
```

実行: `mvn test`

## Gradleの場合

build.gradle:
```groovy
dependencies {
    testImplementation 'org.junit.jupiter:junit-jupiter:5.10.0'
}
test { useJUnitPlatform() }
```

実行: `./gradlew test`
