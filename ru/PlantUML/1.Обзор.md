# PlantUML Обзор

> Инструмент с открытым исходным кодом для рисования UML-диаграмм из текстовых описаний

---

## Обзор

PlantUML — это инструмент с открытым исходным кодом, позволяющий пользователям создавать диаграммы из простого текстового языка. Он поддерживает UML-диаграммы, сетевые диаграммы и блок-схемы.

---

## Основные функции

- **Множество типов диаграмм**: UML, Активность, Последовательность, Класс
- **Текстовый формат**: Определение диаграмм простым текстом
- **Поддержка IDE**: Плагины для VS Code, IntelliJ, Eclipse
- **Экспорт**: PNG, SVG, PDF
