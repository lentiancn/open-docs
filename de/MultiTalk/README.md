# MultiTalk

MultiTalk ist ein audio-gesteuertes Framework zur Erzeugung von Mehrpersonen-Gesprächsvideos, das vom MeiGen-AI-Team entwickelt wurde. Dieses Projekt kann hochwertige Videoinhalte mit Mehrpersoneninteraktionen basierend auf Multi-Stream-Audio-Eingaben, Referenzbildern und Textprompts generieren.

## Hauptmerkmale

- Audio-gesteuerte Mehrpersonen-Gesprachsvideogenerierung
- Unterstützung für Einzelpersonen- und Mehrpersonen-Szenarien
- Branchenführende Lippensynchronisationsgenauigkeit
- Unterstützung für Zeichentrick- und Sing-Videogenerierung
- Ausgabeauflösung unterstützt 480P und 720P
- Maximale Videoclip-Länge bis zu 15 Sekunden
