# Jenkins インストールガイド

Jenkinsは、継続的インテグレーションと継続的デプロイ（CI/CD）を実装するためのオープンソース自動化サーバーです。このガイドでは、さまざまな環境でのインストール方法を説明します。

## バージョン情報

| バージョン | 状態 | 説明 |
|------------|------|------|
| Jenkins 2.x | ✅ 現行 | 最新安定バージョン |
| Jenkins 1.x | ❌ 旧版 | メンテナンス終了 |

## システム要件

### ハードウェア要件

| 要件 | 最小構成 | 推奨構成 |
|------|----------|----------|
| CPU | 1 コア | 2+ コア |
| メモリ | 1 GB | 4+ GB |
| ディスク | 10 GB | 50+ GB |

### ソフトウェア要件

| ソフトウェア | バージョン |
|--------------|------------|
| Java | JDK 8 以上 |
| OS | Windows/macOS/Linux |

## インストール方法

### 方法1：Windowsへのインストール

#### 1. インストーラーを使用

1. Jenkinsをダウンロード：https://www.jenkins.io/download/
2. jenkins.msi インストーラーを実行
3. インストールウィザードに従って完了

#### 2. Chocolateyを使用

```powershell
choco install jenkins
```

#### 3. 手動インストール

```powershell
# WARファイルをダウンロード
# 指定ディレクトリに展開
java -jar jenkins.war
```

### 方法2：macOSへのインストール

#### 1. Homebrewを使用

```bash
# Jenkinsをインストール
brew install jenkins

# サービスを起動
brew services start jenkins

# サービスを停止
brew services stop jenkins
```

#### 2. 手動インストール

```bash
# WARファイルをダウンロード
wget https://get.jenkins.io/jenkins.war

# 実行
java -jar jenkins.war
```

### 方法3：Linuxへのインストール

#### Ubuntu/Debian

```bash
# キーを追加
wget -q -O - https://pkg.jenkins.io/debian/jenkins.io.key | sudo apt-key add -

# リポジトリを追加
sudo sh -c 'echo deb https://pkg.jenkins.io/debian binary/ > /etc/apt/sources.list.d/jenkins.list'

# 更新してインストール
sudo apt-get update
sudo apt-get install jenkins

# サービスを起動
sudo systemctl start jenkins
sudo systemctl enable jenkins
```

#### CentOS/RHEL

```bash
# リポジトリを追加
sudo wget -O /etc/yum.repos.d/jenkins.repo https://pkg.jenkins.io/redhat/jenkins.repo

# キーをインポート
sudo rpm --import https://pkg.jenkins.io/redhat/jenkins.io.key

# インストール
sudo yum install jenkins

# サービスを起動
sudo systemctl start jenkins
sudo systemctl enable jenkins
```

### 方法4：Dockerでのインストール

```bash
# Jenkinsコンテナを実行
docker run -d \
  --name jenkins \
  -p 8080:8080 \
  -v jenkins_home:/var/jenkins_home \
  jenkins/jenkins:lts
```

#### Docker Compose

```yaml
version: '3'
services:
  jenkins:
    image: jenkins/jenkins:lts
    ports:
      - "8080:8080"
    volumes:
      - jenkins_home:/var/jenkins_home
```

### 方法5：WARファイルのデプロイ

```bash
# WARファイルをダウンロード
wget https://get.jenkins.io/war-stable/latest/jenkins.war

# 実行
java -jar jenkins.war --httpPort=8080
```

## インストール後の設定

### 初回アクセス

1. ブラウザで http://localhost:8080 にアクセス
2. 初期管理者パスワードを確認：
   ```bash
   # Linux/macOS
   sudo cat /var/lib/jenkins/secrets/initialAdminPassword
   
   # Docker
   docker exec jenkins cat /var/jenkins_home/secrets/initialAdminPassword
   ```
3. 推奨プラグインをインストール
4. 管理者ユーザーを作成

### 日本語界面の設定

1. Manage Jenkins → Manage Plugins に移動
2. "Localization: Japanese" をインストール
3. Jenkinsを再起動

## よくある質問

### 起動に失敗する？

Javaがインストールされているか確認：

```bash
java -version
```

### ポートが佔用されている？

ポートを変更：

```bash
java -jar jenkins.war --httpPort=8090
```

### プラグインのインストールに失敗する？

ネットワーク接続を確認するか、プロキシを使用してください。
