# JUnit 安装指南

JUnit 是 Java 最流行的单元测试框架。

## 版本

| 版本 | Java 版本 | 推荐 |
|------|-----------|------|
| JUnit 5 | Java 8+ | ✅ 推荐 |
| JUnit 4 | Java 5+ | 旧版本 |

## 添加依赖

### Maven (JUnit 5)

```xml
<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter</artifactId>
    <version>5.11.0</version>
    <scope>test</scope>
</dependency>
```

### Gradle

```groovy
testImplementation 'org.junit.jupiter:junit-jupiter:5.11.0'
```

## IDE 支持

- **IntelliJ IDEA**: 内置支持
- **VS Code**: 安装 Extension Pack for Java
- **Eclipse**: 安装 JUnit 插件

## Spring Boot 集成

Spring Boot 默认包含 JUnit 5：

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-test</artifactId>
    <scope>test</scope>
</dependency>
```
