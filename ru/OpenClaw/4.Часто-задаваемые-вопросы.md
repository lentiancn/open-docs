# Часто задаваемые вопросы о OpenClaw

Эта страница отвечает на распространённые вопросы о OpenClaw.

## Основы

### Что такое OpenClaw?

OpenClaw — это шлюз с самостоятельным хостингом, который соединяет ваши приложения для чата (WhatsApp, Telegram, Discord и т.д.) с ИИ-помощниками для программирования. Он позволяет общаться с ИИ-помощником из любого места, сохраняя полный контроль над своими данными.

### OpenClaw бесплатен?

Да, OpenClaw — проект с открытым исходным кодом под лицензией MIT. Однако вам нужно будет оплатить:
- Сервер или компьютер для запуска
- Стоимость API-ключей для ИИ-сервисов

### Нужен ли мне подписка на ИИ-сервис?

OpenClaw поддерживает несколько провайдеров моделей, включая:
- Anthropic (Claude)
- OpenAI (GPT)
- Google Gemini
- Любой другой сервис, совместимый с OpenAI API

Вы можете использовать API-ключи с оплатой по факту или аутентификацию по подписке.

### Какие платформы поддерживает OpenClaw?

- **Настольные**: macOS, Linux, Windows
- **Мобильные**: iOS, Android (в качестве узлов)
- **Приложения для чата**: WhatsApp, Telegram, Discord, iMessage, Signal, Slack и более 20 платформ

## Установка

### Что делать, если установка не удалась?

1. Проверьте версию Node.js: `node --version` (требуется 22 или выше)
2. Запустите диагностику: `openclaw doctor`
3. Проверьте логи: `openclaw logs`

### Можно ли запустить на Raspberry Pi?

Да, OpenClaw работает на Raspberry Pi. Рекомендуется Raspberry Pi 4 или новее с Raspberry Pi OS или Ubuntu.

### Нужен ли мне Mac?

Нет, OpenClaw работает на любой системе с поддержкой Node.js. Только канал iMessage требует Mac (из-за ограничений Apple).

## Настройка

### Где находится файл конфигурации?

По умолчанию: `~/.openclaw/openclaw.json`

### Как ограничить, кто может отправлять сообщения?

Настройте через `dmPolicy`:

- `pairing`: неизвестным отправителям нужен код подключения
- `allowlist`: только пользователи из списка разрешённых
- `open`: разрешить все личные сообщения
- `disabled`: отключить личные сообщения

### Как настроить групповые чаты?

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

## Использование

### Как добавить новый канал чата?

```bash
openclaw channels login whatsapp
```

### Можно ли использовать несколько каналов одновременно?

Да, один Gateway может подключаться к нескольким каналам одновременно.

### Как заставить ИИ запоминать предыдущие разговоры?

OpenClaw имеет встроенную память, которая автоматически сохраняет контекст сессии. Вы можете настроить политики хранения сессий через конфигурацию.

### Можно ли запустить несколько агентов?

Да, настройте несколько агентов с правилами маршрутизации:

```json5
{
  agents: {
    list: [
      { id: "home", default: true },
      { id: "work" }
    ]
  }
}
```

## Безопасность

### Мои данные в безопасности?

Да. OpenClaw с самостоятельным хостингом — все данные остаются на вашем устройстве и не отправляются на сторонние серверы (кроме API ИИ-провайдеров).

### Как защитить мой Gateway?

1. Настройте `allowFrom` для ограничения источников доступа
2. Используйте надёжные пароли и API-ключи
3. Регулярно обновляйте OpenClaw
4. Используйте файрвол для ограничения доступа к портам

## Устранение неполадок

### Gateway не запускается

1. Проверьте, не занят ли порт: `openclaw gateway status`
2. Проверьте логи ошибок: `openclaw logs`
3. Запустите диагностику: `openclaw doctor`

### ИИ не отвечает

1. Проверьте правильность настройки API-ключа
2. Убедитесь, что на счёте провайдера модели достаточно средств
3. Проверьте сообщения об ошибках в логах

### Отправка сообщений не работает

1. Проверьте, вошёл ли канал
2. Убедитесь, что целевой пользователь или группа существуют
3. Проверьте подробные логи ошибок
