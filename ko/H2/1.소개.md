# H2 데이터베이스 - 소개

## H2란?

H2는 Java로 작성된 오픈소스 관계형 데이터베이스 관리 시스템입니다. Java 애플리케이션의 임베디드 데이터베이스로 사용하거나 클라이언트-서버 모드로 실행할 수 있습니다. 이 소프트웨어는 Mozilla Public License 2.0(MPL 2.0) 또는 Eclipse Public License(EPL 1.0)로 오픈소스로 제공됩니다.

## H2의 주요 특징

### 임베디드 데이터베이스
H2는 별도의 데이터베이스 서버 없이 Java 애플리케이션에 직접 임베디드될 수 있습니다. 이는 개발 및 테스트 환경에서 매우 인기가 있습니다.

### 클라이언트-서버 모드
임베디드 모드 외에도 H2는 클라이언트-서버 모드를 지원하며, 여러 애플리케이션이 동시에 액세스할 수 있는 독립형 데이터베이스 서버로 실행됩니다.

### 인메모리 데이터베이스
H2는 인메모리 데이터베이스 모드를 지원하며, 데이터가 메모리에 저장되어 재시작 후 손실됩니다. 이는 빠른 테스트 및 프로토타입 개발에 매우 유용합니다.

### 다양한 데이터베이스 호환성
H2는 다양한 SQL 표준 및 구문과 호환되며 MySQL, PostgreSQL, Oracle, DB2, MS SQL Server의 동작을 시뮬레이션할 수 있어 데이터베이스 마이그레이션 및 테스트를 용이하게 합니다.

### 암호화된 데이터베이스
H2는 민감한 데이터 보호를 위한 데이터베이스 암호화를 지원합니다.

### 전체 텍스트 검색
H2는 추가 구성 없이도 전체 텍스트 검색 기능을 기본적으로 제공합니다.

### 제로 관리
임베디드 데이터베이스로서 H2는 유지보수를 위한 전문 데이터베이스 관리자가 필요하지 않습니다.

## H2 적용 시나리오

H2는 다음과 같은 시나리오에서 널리 사용됩니다:

- Java 애플리케이션의 임베디드 데이터베이스
- 개발 및 테스트 환경
- 빠른 프로토타입 개발
- 단위 테스트 및 통합 테스트
- 경량 데스크톱 애플리케이션
- SQL 구문 학습

## H2 버전 역사

H2 데이터베이스 엔진 개발은 2004년 5월에 시작되어 2005년 12월에 처음 출시되었습니다. 원래 개발자 Thomas Mueller는 Java 데이터베이스 엔진 HSQLDB(원래 이름 Hypersonic SQL)의 창립자이기도 합니다.
