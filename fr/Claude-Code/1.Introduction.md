# Claude Code Introduction

Claude Code est un assistant de programmation intelligent développé par Anthropic qui peut lire des bases de code, éditer des fichiers, exécuter des commandes et s'intégrer parfaitement aux outils de développement. Disponible dans votre terminal, IDE, application de bureau et navigateur.

## Qu'est-ce que Claude Code ?

Claude Code est un assistant de programmation alimenté par l'IA qui vous aide à construire des fonctionnalités, corriger des bugs et automatiser des tâches de développement. Il comprend l'ensemble de votre base de code et peut travailler sur plusieurs fichiers et outils.

## Caractéristiques Principales

- **Automatiser les tâches**: Écrire des tests, corriger les erreurs lint, résoudre les conflits de fusion, mettre à jour les dépendances
- **Développement de fonctionnalités**: Décrivez vos besoins en langage naturel; Claude Code planifie et implémente
- **Intégration Git**: Préparer les modifications, créer des commits, ouvrir des Pull Requests
- **Intégration MCP**: Connecter aux sources de données et outils externes
- **Personnalisation**: CLAUDE.md, commandes personnalisées, Hooks
- **Collaboration multi-agents**: Traitement parallèle des tâches complexes

## Scénarios d'Utilisation

| Scénario | Méthode Recommandée |
|----------|---------------------|
| Développement local | Terminal CLI |
| Codage dans l'IDE | VS Code ou JetBrains |
| Continuation entre appareils | Contrôle à distance |
| Opérations mobiles | Web ou application iOS |
| Automatisation CI/CD | GitHub Actions ou GitLab CI/CD |

## Exigences de Compte

Claude Code nécessite:
- Abonnement Claude Pro, Max, Teams ou Enterprise
- Compte Anthropic Console
- Fournisseurs cloud tiers (Amazon Bedrock, Google Vertex AI, Microsoft Foundry)
