# Bootstrap 5

Bootstrap은 반응형 모바일 우선 웹사이트를 구축하기 위한 세계 가장 인기 있는 오픈소스 프론트엔드 툴킷입니다.

## 소개

Bootstrap 5는 jQuery 의존 없이 순수 JavaScript로 완전히 재작성된 최신 버전입니다. 풍부한 CSS와 JavaScript 컴포넌트를 제공합니다.

## 주요 기능

- **반응형 디자인**: 내장된 그리드 시스템
- **풍부한 컴포넌트**: 버튼, 네비게이션, 폼, 카드, 모달 등
- **강력한 유틸리티**: 여백, 색상, 텍스트
- **Sass 지원**: Sass로 테마 커스터마이징
- **아이콘 라이브러리**: Bootstrap Icons 포함

## 빠른 시작

### CDN

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
```

### npm

```bash
npm install bootstrap@5.3.3
```
