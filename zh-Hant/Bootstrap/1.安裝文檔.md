# Bootstrap 5 安裝指南

本指南詳細介紹 Bootstrap 5 在各種環境下的安裝和配置方法。

## 版本說明

| 版本 | 狀態 | 說明 |
|------|------|------|
| Bootstrap 5.3.x | ✅ 當前 | 最新版本，包含所有新功能 |
| Bootstrap 5.2.x | ✅ 支持 | 穩定版本 |
| Bootstrap 5.1.x | ⚠️ 維護 | 安全更新 |
| Bootstrap 4.x | ❌ 停止維護 | 不推薦新項目使用 |

> **重要提示**：本指南主要針對 **Bootstrap 5.3.x**。Bootstrap 4 已停止維護，不建議用於新項目。

## 系統要求

### 瀏覽器支持

| 瀏覽器 | 最低版本 |
|--------|----------|
| Chrome | 90+ |
| Firefox | 90+ |
| Safari | 14+ |
| Edge | 90+ |
| Opera | 76+ |

### 依賴要求

| 依賴 | 版本要求 | 說明 |
|------|----------|------|
| JavaScript | ES2018+ | 現代瀏覽器支持 |
| CSS | CSS3 | 需要 CSS 變量支持 |
| HTML | HTML5 | 推薦使用語義化標籤 |

## 安裝方法

### 方法一：CDN 引入（最簡單）

最快速的方式是通過 CDN 引入 Bootstrap 的 CSS 和 JavaScript 文件。

#### CSS only

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
```

#### CSS + JavaScript

```html
<!-- Bootstrap CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- Bootstrap JS (包含 Popper) -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
```

**CDN 提供商**：
- [jsDelivr](https://www.jsdelivr.com/) - 推薦
- [cdnjs](https://cdnjs.com/libraries/bootstrap)
- [unpkg](https://unpkg.com/browse/bootstrap/)

#### 完整模板

```html
<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 5 頁面</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>你好，Bootstrap 5！</h1>
    
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### 方法二：npm 安裝

#### 使用 npm

```bash
# 創建新項目
npm init -y

# 安裝 Bootstrap
npm install bootstrap@5.3.3

# 安裝 Popper.js (如果需要)
npm install @popperjs/core
```

#### 在項目中引入

```javascript
// main.js 或 app.js
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

### 方法三：使用 Vite

#### 1. 創建 Vite 項目

```bash
npm create vite@latest my-bootstrap-app -- --template vanilla
cd my-bootstrap-app
npm install
```

#### 2. 安裝 Bootstrap

```bash
npm install bootstrap@5.3.3
```

#### 3. 引入 Bootstrap

```javascript
// main.js
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

### 方法四：使用 Webpack

#### 1. 安裝依賴

```bash
npm install bootstrap@5.3.3
npm install --save-dev webpack webpack-cli css-loader style-loader
```

#### 2. 配置 Webpack

```javascript
// webpack.config.js
module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js'
  },
  module: {
    rules: [
      {
        test: /\.css$/,
        use: ['style-loader', 'css-loader']
      }
    ]
  }
};
```

#### 3. 引入 Bootstrap

```javascript
// src/index.js
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

### 方法五：使用 Vue CLI / Nuxt

#### Vue CLI

```bash
# 創建 Vue 項目
npm install -g @vue/cli
vue create my-app

# 安裝 Bootstrap
cd my-app
npm install bootstrap@5.3.3

# 在 main.js 中引入
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

#### Nuxt 3

```bash
# 創建 Nuxt 項目
npx nuxi@latest init my-nuxt-app
cd my-nuxt-app
npm install

# 安裝 Bootstrap
npm install bootstrap@5.3.3

# 在 nuxt.config.ts 中配置
export default defineNuxtConfig({
  css: ['bootstrap/dist/css/bootstrap.min.css'],
  build: {
    transpile: ['bootstrap']
  }
})
```

### 方法七：下載源碼

#### 從 GitHub 下載

```bash
# 克隆倉庫
git clone https://github.com/twbs/bootstrap.git
cd bootstrap

# 安裝依賴
npm install

# 構建
npm run dist
```

#### 直接下載

1. 訪問 [Bootstrap 官方發布頁](https://github.com/twbs/bootstrap/releases)
2. 下載 Source code (zip)
3. 解壓後引入 `dist/css/bootstrap.min.css` 和 `dist/js/bootstrap.bundle.min.js`

## 常見問題

### Popper 是什麼？

Popper 是一個定位引擎，用於工具提示（Tooltip）和彈出框（Popover）的定位。`bootstrap.bundle.min.js` 已包含 Popper，無需單獨安裝。

### jQuery 是否還需要？

**不需要！** Bootstrap 5 完全使用原生 JavaScript 重寫，不再依賴 jQuery。

### 如何使用圖標？

Bootstrap 5 使用 Bootstrap Icons：

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css">

<i class="bi bi-github"></i>
<i class="bi bi-bootstrap-fill"></i>
```

### 主題顏色如何修改？

可以通過 CSS 變量覆蓋：

```css
:root {
  --bs-primary: #ff6b35;
  --bs-secondary: #6c757d;
  --bs-success: #198754;
}
```
