# LivePortrait 설치 가이드

LivePortrait는 Kwai VGI가 개발한 효율적인 초상 애니메이션 프레임워크입니다.驱动 비디오의 모션을 사용하여 정적 초상을 애니메이션화합니다.

## 시스템 요구 사항

### 하드웨어 요구 사항

| 구성 요소 | 최소 | 권장 |
|----------|------|------|
| GPU | NVIDIA 8GB VRAM | NVIDIA 16GB VRAM |
| RAM | 8GB | 16GB |
| 스토리지 | 30GB | 50GB |
| CUDA | 11.8+ | 12.1+ |

### 소프트웨어 요구 사항

| 소프트웨어 | 버전 | 참고 |
|------------|------|------|
| Python | 3.8 - 3.11 | 권장 3.10 |
| CUDA | 11.8+ | GPU 가속 필요 |
| ffmpeg | 최신 | 비디오 처리 필요 |

## Linux/Ubuntu 설치

### 1단계: 종속성 설치

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install python3.10 python3-pip git wget curl
sudo apt install libgl1-mesa-glx libglib2.0-0 libsm6 libxext6 libxrender-dev libgomp1
sudo apt install ffmpeg
```

### 2단계: 프로젝트 클론

```bash
git clone https://github.com/KwaiVGI/LivePortrait.git
cd LivePortrait
```

### 3단계: 가상 환경 생성

```bash
python3.10 -m venv venv
source venv/bin/activate
```

### 4단계: PyTorch 설치

```bash
pip install torch==2.1.0 torchvision==0.16.0 --index-url https://download.pytorch.org/whl/cu121
```

### 5단계: 종속성 설치

```bash
pip install -r requirements.txt
```

### 6단계: 모델 다운로드

```bash
mkdir -p pretrained_models
bash scripts/download_weights.sh
```

## 설치 확인

```bash
python inference.py --help
```

## 일반적인 문제

### CUDA 메모리 부족

**해결책:**
- 배치 크기 줄이기: `--batch_size 1`
- 더 낮은 해상도 사용
