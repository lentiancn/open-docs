# Flutter

Flutter 是 Google 开发的开源 UI 工具包，用于构建跨平台应用。

## 简介

Flutter 允许您使用一套代码库同时为 iOS、Android、Web 和桌面构建美观的应用。

## 主要特性

- **跨平台**：一套代码，多个平台
- **热重载**：快速开发迭代
- **丰富组件**：Material Design 和 Cupertino 组件
- **Dart 语言**：易于学习，性能优秀
- **声明式 UI**：构建精美界面

## 快速开始

### 安装

```bash
# macOS
brew install flutter

# Linux
git clone https://github.com/flutter/flutter.git -b stable
```

### 创建项目

```bash
flutter create my_app
cd my_app
flutter run
```
