# Angular Guía de Instalación

## Requisitos

- Node.js: 18.10.0+ o 20.0.0+
- npm: 9.0.0+

## Instalación

### 1. Instalar Node.js

```bash
nvm install --lts
nvm use --lts
```

### 2. Instalar Angular CLI

```bash
npm install -g @angular/cli
```

### 3. Crear Proyecto

```bash
ng new my-app
cd my-app
ng serve
```

## Comandos

```bash
ng g c my-component
ng g s my-service
ng build
```
