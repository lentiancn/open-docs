# 安裝指南

## 各發行版安裝說明

### Arch Linux

Arch 使用者可以參考 Arch Wiki 上的完整安裝說明：
[https://wiki.archlinux.org/title/Waydroid](https://wiki.archlinux.org/title/Waydroid)

### Postmarket OS

Postmarket 社群提供了安裝說明和故障排除提示：
[https://wiki.postmarketos.org/wiki/Waydroid](https://wiki.postmarketos.org/wiki/Waydroid)

### Fedora

Waydroid 可以從官方軟體包倉庫安裝：

```bash
sudo dnf install waydroid
```

安裝後，從應用程式選單啟動 Waydroid，並在 OTA 欄位中貼上以下 URL 進行初始化：

系統 OTA：`https://ota.waydro.id/system`

供應商 OTA：`https://ota.waydro.id/vendor`

#### Silverblue/Kinoite/...

對於 Fedora 不可變變體，使用 `rpm-ostree` 代替 `dnf`：

```bash
rpm-ostree install waydroid
```

### Ubuntu/Debian 及衍生版本

確保已啟用 Wayland 工作階段（Ubuntu 22.04+）

* 安裝前置依賴

```bash
sudo apt install curl ca-certificates -y
```

* 新增官方儲存庫

```bash
curl -s https://repo.waydro.id | sudo bash
```

如果指令碼無法偵測到您的發行版，可以透過附加 `-s <DISTRO>` 提供有效選項。目前支援的值包括：**mantic**、**focal**、**jammy**、**kinetic**、**lunar**、**noble**、**plucky**、**questing**、**bookworm**、**bullseye**、**trixie**、**sid**

* 安裝 waydroid

```bash
sudo apt install waydroid -y
```

然後從應用程式選單啟動 Waydroid。

## 故障排除

### 手動啟動 Waydroid

要不使用 systemctl 啟動 Waydroid，需要遵循以下簡單步驟：

**首先啟動容器：**

```bash
sudo waydroid container start
```

**然後在新的終端機分頁中啟動 waydroid 工作階段（無需 ***sudo*****）：

```bash
waydroid session start
```

啟動後看到 "Android with user 0 is ready" 時，就可以從應用程式選單啟動應用程式了。

### 全螢幕模式啟動 Waydroid

```bash
waydroid show-full-ui
```

### 多視窗模式啟動 Waydroid

首先在 Waydroid 工作階段執行時設定屬性：

```bash
waydroid prop set persist.waydroid.multi_windows true
```

然後重新啟動工作階段：

```bash
waydroid session stop
```

之後啟動的應用程式將以多視窗模式執行。

### 重新安裝 Waydroid

如果需要完全重新安裝：

```bash
waydroid session stop
sudo waydroid container stop
sudo apt remove waydroid  # 或對應發行版的解除安裝指令
# 重新啟動後執行清理
sudo rm -rf /var/lib/waydroid /home/.waydroid ~/waydroid ~/.share/waydroid ~/.local/share/applications/*aydroid* ~/.local/share/waydroid
```
