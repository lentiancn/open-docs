# Preguntas frecuentes

## Certificación de Google Play

Waydroid utiliza una imagen de Android personalizada basada en LineageOS, que no incluye los servicios de Google Play de forma predeterminada. Si necesita certificación de Google Play, puede instalar una imagen GAPPS.

## Deshabilitar el teclado en pantalla

En algunos casos, es posible que desee deshabilitar el teclado en pantalla. Esto se puede lograr a través de la configuración del sistema o la configuración de Waydroid.

## Hacer funcionar Waydroid a través de una VM

Para ejecutar Waydroid a través de una máquina virtual, asegúrese de que la VM admita virtualización anidada y tenga la aceleración de hardware configurada correctamente.

## Uso de ADB con Waydroid

Waydroid admite depuración ADB. De forma predeterminada, el modo seguro de ADB está habilitado y no se conectará automáticamente. Use el siguiente comando para conectarse:

```bash
waydroid adb connect
```

Luego proporcione autorización en el lado de Android.

## Configuración de sesiones exclusivas de Waydroid

Puede configurar sesiones dedicadas que solo ejecuten Waydroid, adecuadas para casos de uso específicos.

## Configuración de una carpeta compartida

Waydroid admite la configuración de carpetas compartidas para facilitar la transferencia de archivos entre el host Linux y el contenedor Android.

## Corrección de color en Mutter

Al usar el gestor de ventanas Mutter, es posible que encuentre problemas de visualización de color que se pueden corregir con configuraciones específicas.

## Copia de seguridad y restauración de aplicaciones y datos

Waydroid admite la copia de seguridad y restauración de aplicaciones y datos para garantizar que los datos importantes no se pierdan.

## Uso de imágenes personalizadas de Waydroid

Para usar imágenes personalizadas:

1. Descargue o compile `system.img` y `vendor.img` personalizados
2. Copie al directorio `/etc/waydroid-extra/images/`
3. Reinicialice: `sudo waydroid init -f`

## Instalación de aplicaciones del sistema

Se pueden instalar aplicaciones de nivel de sistema, pero requieren permisos especiales y pasos de configuración.

## Compatibilidad de hardware

Waydroid tiene ciertos requisitos de hardware:
- Requiere soporte del servidor de visualización Wayland
- Requiere soporte del kernel de Linux para espacios de nombres y funciones de contenedor
- Los controladores gráficos deben admitir OpenGL ES 3.2

## Optimización del rendimiento

Para obtener un rendimiento óptimo:
- Asegúrese de que la aceleración de hardware esté habilitada
- Use controladores gráficos adecuados
- Ejecute en CPU compatibles (evite la emulación ARM pura)
