## Introduction to Bootstrap

Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive layouts, mobile-first websites and web applications.

### Bootstrap Features

- **Responsive Design**: Built-in responsive grid system
- **Predefined Styles**: Rich component and style classes
- **JavaScript Components**: Interactive components (modals, carousels, etc.)
- **Customization**: Theme customization via Sass variables
- **Browser Compatibility**: Supports all modern browsers

### Bootstrap Version History

| Version | Release Date | Key Features |
|---------|--------------|--------------|
| Bootstrap 5 | May 2021 | Removed jQuery dependency, improved form components, new Utilities API |
| Bootstrap 4 | January 2018 | Flexbox support, Sass instead of Less, new components |
| Bootstrap 3 | August 2013 | Mobile-first design |
| Bootstrap 2 | January 2012 | Improved responsive design |
| Bootstrap 1 | August 2011 | Initial release |

> **Note**: Bootstrap 5 is the current major version and is recommended for new projects. Bootstrap 4 still receives maintenance updates, but new feature development has stopped.

## CDN Installation

### CSS Only

Add the following code to the `<head>` section of your HTML:

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH" crossorigin="anonymous">
```

### CSS + JavaScript

After adding CSS, add JS before `</body>`:

```html
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-YvpcrYf0tY3lHB60NNkmXc5s9fDVZLESaAA55NDzOxhy9GkcIdslK1eN7N6jIeHz" crossorigin="anonymous"></script>
```

> **Note**: Bootstrap 5 uses `bootstrap.bundle.min.js` (includes Popper), so you no longer need to include Popper separately.

### Using Domestic CDN (Recommended for China Users)

```html
<!-- CSS -->
<link href="https://cdn.staticfile.org/twitter-bootstrap/5.3.3/css/bootstrap.min.css" rel="stylesheet">

<!-- JS -->
<script src="https://cdn.staticfile.org/twitter-bootstrap/5.3.3/js/bootstrap.bundle.min.js"></script>
```

## npm Installation

### Install Bootstrap

```bash
# Install Bootstrap using npm
npm install bootstrap

# Install specific version
npm install bootstrap@5.3.3

# Install Bootstrap 4
npm install bootstrap@4.6.2
```

### Reference in Project

**CSS**:
```javascript
// webpack or bundler
import 'bootstrap/dist/css/bootstrap.min.css';
```

**JavaScript**:
```javascript
import * as bootstrap from 'bootstrap';

// Or import only needed modules
import { Modal } from 'bootstrap';
```

## Package Manager Installation

### yarn

```bash
yarn add bootstrap
```

### pnpm

```bash
pnpm add bootstrap
```

### Ruby on Rails

Add to Gemfile:

```ruby
gem 'bootstrap', '~> 5.3'
```

Then run `bundle install`.

### Python/Django

```bash
pip install bootstrap5
```

Or add to requirements.txt:

```
django-bootstrap5
```

## Download Installation

### Download Pre-compiled Version

1. Visit [Bootstrap Official Website](https://getbootstrap.com/)
2. Click the "Download" button
3. Download pre-compiled CSS and JS files

### Download Version with Source Code

Download version containing Sass source files:

1. Visit [Bootstrap Source](https://github.com/twbs/bootstrap/releases)
2. Download source archive
3. After extraction, contains:
   - `dist/css/` - Compiled CSS
   - `dist/js/` - Compiled JS
   - `scss/` - Sass source files

### File Structure

```
bootstrap/
├── css/
│   ├── bootstrap.css
│   ├── bootstrap.css.map
│   ├── bootstrap.min.css
│   ├── bootstrap.min.css.map
│   ├── bootstrap.rtl.css
│   └── bootstrap.rtl.min.css
├── js/
│   ├── bootstrap.bundle.js
│   ├── bootstrap.bundle.min.js
│   ├── bootstrap.esm.js
│   └── bootstrap.esm.min.js
└── scss/
    └── ...
```

## Verify Installation

### Basic HTML Template

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Bootstrap Test</title>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <div class="container mt-5">
    <h1>Hello, Bootstrap!</h1>
    <button class="btn btn-primary">Primary Button</button>
  </div>
  
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### Verification Steps

1. Save the above code as an `.html` file
2. Open in a browser
3. If you see a page with Bootstrap styles, installation is successful

## Quick Start

### Using Grid System

```html
<div class="container">
  <div class="row">
    <div class="col-md-4">Column 1</div>
    <div class="col-md-4">Column 2</div>
    <div class="col-md-4">Column 3</div>
  </div>
</div>
```

### Using Components

**Buttons**:
```html
<button class="btn btn-primary">Primary</button>
<button class="btn btn-secondary">Secondary</button>
<button class="btn btn-success">Success</button>
<button class="btn btn-danger">Danger</button>
<button class="btn btn-warning">Warning</button>
<button class="btn btn-info">Info</button>
```

**Cards**:
```html
<div class="card" style="width: 18rem;">
  <img src="..." class="card-img-top" alt="...">
  <div class="card-body">
    <h5 class="card-title">Card Title</h5>
    <p class="card-text">Card content</p>
    <a href="#" class="btn btn-primary">Go somewhere</a>
  </div>
</div>
```

**Modals**:
```html
<!-- Trigger Button -->
<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#exampleModal">
  Open Modal
</button>

<!-- Modal -->
<div class="modal fade" id="exampleModal" tabindex="-1">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title">Modal Title</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
      </div>
      <div class="modal-body">
        Modal content
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Close</button>
        <button type="button" class="btn btn-primary">Save</button>
      </div>
    </div>
  </div>
</div>
```

## Frequently Asked Questions

### Q1: What's the Difference Between Bootstrap 5 and Bootstrap 4?

Main differences:
- Bootstrap 5 removed jQuery dependency
- Uses native JavaScript
- Improved form components
- New Utilities API
- No longer supports IE browser

### Q2: How to Customize Bootstrap Theme?

**Method 1: Using Sass**
```scss
// Custom variables
$primary: #ff6600;
$secondary: #cccccc;

// Import Bootstrap
@import "bootstrap";
```

**Method 2: Using Bootstrap Customization Tool**
Visit https://getbootstrap.com/docs/5.3/customize/options/

### Q3: Is Bootstrap Responsive?

Yes, Bootstrap is mobile-first design:
- `col-` Extra small (<576px)
- `col-sm-` Small (≥576px)
- `col-md-` Medium (≥768px)
- `col-lg-` Large (≥992px)
- `col-xl-` Extra large (≥1200px)
- `col-xxl-` Extra extra large (≥1400px)

### Q4: How to Use Bootstrap Icons?

```bash
npm install bootstrap-icons
```

Or use CDN:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css">
```

Usage:
```html
<i class="bi bi-github"></i>
```

### Q5: Does Bootstrap 5 Support IE11?

No. Bootstrap 5 requires modern browsers. If you need IE11 support, please use Bootstrap 4.6.

### Q6: How to Debug Bootstrap Styles?

1. Use browser developer tools (F12)
2. Check element's class names
3. View style sources in Computed panel

### Q7: How to Handle Bootstrap Conflicts with Other Frameworks?

- Use specific selectors
- Use Bootstrap's naming space
- Use Bootstrap's provided APIs

### Q8: How to Upgrade Bootstrap Version?

1. Backup current project
2. Update CDN link or npm package version
3. Check for breaking changes
4. Test all functionality
