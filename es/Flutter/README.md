# Flutter

Kit de herramientas UI de código abierto de Google.

## Resumen

Flutter es un kit de herramientas UI de código abierto para crear aplicaciones multiplataforma a partir de una única base de código usando Dart.

## Características

- **Multiplataforma**: Una base de código para todas las plataformas
- **Hot Reload**: Vista previa instantánea de cambios
- **Widgets Ricos**: Material Design + Cupertino
- **Alto Rendimiento**: Experiencia de renderizado nativo
- **Código Abierto**: Gratis para usar

## Inicio Rápido

### Instalación

Ver guía de instalación: ./2.Guia-de-Instalacion.md

### Crear proyecto

```bash
flutter create mi_app
cd mi_app
flutter run
```

### Estructura del proyecto

```
lib/
└── main.dart        # Punto de entrada
pubspec.yaml         # Archivo de configuración
```

## Conceptos Fundamentales

- **Dart**: Lenguaje de programación de Flutter
- **Widget**: Bloque de construcción de UI
- **State**: Gestión de estado
- **Platform Channel**: Canales de plataforma
