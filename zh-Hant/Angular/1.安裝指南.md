## 系統需求

### 硬體需求
- **處理器**：雙核心處理器或更好
- **記憶體**：最低 4GB（建議 8GB）
- **磁碟空間**：至少 3GB 可用空間

### 軟體需求

#### Windows
- Windows 10 或更高版本
- Node.js 18.19 或更高版本（建議使用 LTS 版本）
- npm 9.x 或更高版本（隨 Node.js 一起提供）
- Git 版本控制工具

#### macOS
- macOS 10.15（Catalina）或更高版本
- Node.js 18.19 或更高版本（建議使用 LTS 版本）
- npm 9.x 或更高版本
- Git（隨 Xcode 命令列工具一起提供）

#### Linux
- Ubuntu 20.04 LTS 或更高版本 / Debian 系列發行版
- Fedora 36 或更高版本 / RHEL 系列發行版
- Node.js 18.19 或更高版本（建議使用 LTS 版本）
- npm 9.x 或更高版本
- Git

## 前置條件

### 1. 安裝 Node.js

Node.js 是執行 Angular 所需的執行環境。

#### Windows 安裝

**方法 A：使用 Node.js 安裝程式**
1. 造訪 https://nodejs.org/
2. 下載 **LTS** 版本
3. 執行安裝程式（.msi 檔案）
4. 按照安裝精靈操作
5. 開啟命令提示字元驗證：
   ```
   node --version
   npm --version
   ```

**方法 B：使用 Chocolatey**
```powershell
# 安裝 Node.js
choco install nodejs-lts -y
```

**方法 C：使用 nvm-windows（推薦）**
```powershell
# 安裝並使用 Node.js LTS
nvm install 18.19.0
nvm use 18.19.0
```

#### macOS 安裝

**方法 A：使用 Homebrew（推薦）**
```bash
# 安裝 Homebrew（如未安裝）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安裝 Node.js LTS
brew install node@18
```

**方法 B：使用 nvm**
```bash
# 安裝 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 安裝 Node.js LTS
nvm install 18.19.0
nvm use 18.19.0
```

#### Linux 安裝

**Ubuntu/Debian**
```bash
# 更新軟體包列表
sudo apt update

# 安裝 curl 和 build-essential
sudo apt install curl build-essential

# 新增 Node.js 倉庫
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -

# 安裝 Node.js
sudo apt install nodejs
```

**Fedora/RHEL**
```bash
# 新增 Node.js 倉庫
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -

# 安裝 Node.js
sudo dnf install nodejs
```

### 2. 安裝 Git

Git 是 Angular CLI 版本控制功能必需的。

#### Windows
- 從 https://git-scm.com/ 下載
- 安裝時選擇 "Use Git from the Windows Command Prompt"

#### macOS
```bash
# 使用 Homebrew
brew install git

# 或安裝 Xcode 命令列工具
xcode-select --install
```

#### Linux
```bash
# Ubuntu/Debian
sudo apt install git

# Fedora
sudo dnf install git
```

## 在 Windows 上安裝

### 步驟指南

1. **安裝 Node.js**
   - 從 https://nodejs.org/ 下載 Node.js LTS
   - 執行安裝程式

2. **安裝 Git**（如未安裝）
   - 從 https://git-scm.com/ 下載

3. **安裝 Angular CLI**
   ```bash
   npm install -g @angular/cli
   ```

4. **驗證安裝**
   ```bash
   ng version
   ```

5. **建立第一個專案**
   ```bash
   ng new my-first-app
   cd my-first-app
   ng serve
   ```

### 使用 Visual Studio Code（推薦）

1. 從 https://code.visualstudio.com/ 下載 VS Code
2. 安裝 Angular 擴充功能：
   - Angular Language Service
   - Angular Snippets

## 在 Linux 上安裝

### Ubuntu/Debian

```bash
# 更新系統
sudo apt update && sudo apt upgrade -y

# 安裝依賴
sudo apt install curl build-essential git

# 新增 Node.js 倉庫
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -

# 安裝 Node.js
sudo apt install nodejs

# 安裝 Angular CLI
sudo npm install -g @angular/cli
```

### Fedora

```bash
# 安裝依賴
sudo dnf install curl git

# 新增 Node.js 倉庫
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -

# 安裝 Node.js
sudo dnf install nodejs

# 安裝 Angular CLI
sudo npm install -g @angular/cli
```

## 疑難排解

### 常見問題及解決方案

#### 1. npm 權限錯誤 (EACCES)

**解決方案：**
```bash
# 使用 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
```

#### 2. Node.js 版本不相符

**解決方案：**
```bash
# 使用 nvm 切換版本
nvm install 18.19.0
nvm use 18.19.0
```

#### 3. 找不到 Angular CLI

**解決方案：**
```bash
# 重新安裝 Angular CLI
npm uninstall -g @angular/cli
npm install -g @angular/cli
```

#### 4. 連接埠 4200 被佔用

**解決方案：**
```bash
# 使用不同連接埠
ng serve --port 4201
```

#### 5. Angular 版本相容性

| Angular 版本 | Node.js 版本 | TypeScript 版本 |
|-------------|-------------|----------------|
| Angular 17  | 18.x, 20.x  | ~5.2.0         |
| Angular 16  | 16.x, 18.x  | ~5.0.0         |
| Angular 15  | 16.x, 18.x  | ~4.9.0         |
| Angular 14  | 14.x, 16.x  | ~4.7.0         |
