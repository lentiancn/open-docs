# Introduction

Bootstrap est le framework de développement frontend open source le plus populaire au monde pour HTML, CSS et JavaScript. Développé par les designers de Twitter, Mark Otto et Jacob Thornton, Bootstrap offre une multitude de composants et de plugins JavaScript performants qui permettent aux développeurs de créer rapidement des applications web responsives.

Le concept central de Bootstrap est "Mobile First" : dès la conception, le framework prend en compte l'adaptation à tous les types d'appareils. Avec Bootstrap, vous pouvez facilement créer des sites qui s'affichent parfaitement sur ordinateurs, tablettes et smartphones.

## Historique

Bootstrap a d'abord été nommé Twitter Blueprint et a été publié en open source en 2012. En 2012, Bootstrap 2 a introduit les mises en page responsives. En 2013, Bootstrap 3 a été complètement réécrit avec un design plat. En 2015, Bootstrap 4 a commencé à utiliser Sass comme préprocesseur CSS. En 2021, Bootstrap 5 a supprimé la dépendance à jQuery et adopté pleinement le JavaScript natif.

## Fonctionnalités principales

### Mise en page responsive

Le système de grille de Bootstrap est l'une de ses fonctionnalités les plus puissantes. Avec une grille de 12 colonnes, vous pouvez facilement créer des mises en page qui s'adaptent à différentes tailles d'écran. Bootstrap 5 offre six points de rupture : xs, sm, md, lg, xl et xxl, correspondant à différentes largeurs d'appareils.

### Composants riches

Bootstrap propose une grande quantité de composants UI prêts à l'emploi : barres de navigation, cartes, boutons, formulaires, fenêtres modales, carrousels, menus déroulants, etc. Tous ces composants sont soigneusement conçus pour offrir une bonne expérience visuelle et interactive.

### Plugins JavaScript puissants

Bootstrap inclut plus de 20 plugins JavaScript : fenêtres modales, carrousels, info-bulles, popovers, panneaux repliables, et bien plus encore. Ces plugins sont optionnels — vous pouvez les importer au besoin pour réduire la taille de votre projet.

### Basé sur Sass

Bootstrap 5 utilise Sass comme préprocesseur CSS, avec de nombreuses variables et mixins Sass. Vous pouvez facilement personnaliser les couleurs du thème, les tailles de police et autres styles.

### Personnalisation du thème

Avec le système de thèmes de Bootstrap, vous pouvez facilement créer votre propre style. Le marché officiel Bootstrap Themes propose également de nombreux thèmes prêts à utiliser.

### Accessibilité

Les composants Bootstrap intègrent la prise en charge de l'accessibilité, notamment la navigation au clavier et la compatibilité avec les lecteurs d'écran, ce qui aide à créer des sites plus inclusifs.

## Nouveautés de Bootstrap 5

Par rapport à Bootstrap 4, Bootstrap 5 apporte de nombreuses évolutions majeures :

- Suppression de la dépendance à jQuery, utilisation du JavaScript natif
- Amélioration du système de grille avec le point de rupture XXL
- Nouveaux contrôles de formulaire替代 la validation de formulaire
- Meilleure prise en charge des attributs personnalisés
- Passage de Font Awesome à Bootstrap Icons
- Support RTL (de droite à gauche)
