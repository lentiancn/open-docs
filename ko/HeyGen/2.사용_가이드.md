# HeyGen 사용 가이드

이 가이드에서는 HeyGen을 사용하여 AI 비디오를 생성하는 방법을 설명합니다.

## 시작하기

### 웹 인터페이스 사용

1. **HeyGen에 로그인**: https://www.heygen.com
2. **라이브러리에서 아바타 선택** 또는 커스텀 업로드
3. **입력 유형 선택**: 텍스트, 오디오 또는 비디오
4. **비디오 생성** 및 다운로드

### API 사용

```python
from heygen import HeyGen

heygen = HeyGen(api_key="your_api_key")

# 텍스트에서 비디오 생성
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="Hello, this is my first AI video!",
    voice_id="voice_id"
)

print(f"비디오 URL: {video.url}")
```

## 핵심 기능

### 1. 아바타 선택

```python
# 사용 가능한 아바타 나열
avatars = heygen.list_avatars()
for avatar in avatars:
    print(f"{avatar.id}: {avatar.name}")

# 특정 아바타 가져오기
avatar = heygen.get_avatar(avatar_id="avatar_id")
```

### 2. 음성 선택

```python
# 사용 가능한 음성 나열
voices = heygen.list_voices()
for voice in voices:
    print(f"{voice.id}: {voice.name} ({voice.language})")

# 음성 세부 정보 가져오기
voice = heygen.get_voice(voice_id="voice_id")
```

### 3. 비디오 생성

#### 텍스트에서

```python
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="스크립트 텍스트",
    voice_id="voice_id",
    background="office"  # 선택 사항: 배경 장면
)
```

#### 오디오에서

```python
video = heygen.generate_video_from_audio(
    avatar_id="avatar_id",
    audio_url="https://example.com/audio.wav"
)
```

#### 사진에서 (TalkingPhoto)

```python
video = heygen.generate_talking_photo(
    photo_url="https://example.com/photo.jpg",
    script="스크립트",
    voice_id="voice_id"
)
```

### 4. 배치 생성

```python
scripts = [
    "첫 번째 비디오 스크립트",
    "두 번째 비디오 스크립트",
    "세 번째 비디오 스크립트"
]

# 여러 비디오 생성
jobs = heygen.batch_generate(
    avatar_id="avatar_id",
    scripts=scripts,
    voice_id="voice_id"
)

# 작업 상태 확인
for job in jobs:
    status = heygen.get_job_status(job.id)
    print(f"작업 {job.id}: {status}")
```

## 고급 기능

### 커스텀 아바타

```python
# 커스텀 아바타 업로드
avatar = heygen.upload_avatar(
    name="내 커스텀 아바타",
    video_url="https://example.com/reference.mp4"
)
print(f"커스텀 아바타 ID: {avatar.id}")
```

### 즉시 음성 클론

```python
# 오디오 샘플에서 음성 클론
voice = heygen.clone_voice(
    name="내 음성 클론",
    audio_url="https://example.com/voice_sample.wav"
)
print(f"클론된 음성 ID: {voice.id}")
```

### 비디오 편집

```python
# 비디오 세부 정보 가져오기
video = heygen.get_video(video_id="video_id")

# 비디오 설정 업데이트
updated = heygen.update_video(
    video_id="video_id",
    title="내 업데이트된 비디오",
    description="업데이트된 설명"
)

# 비디오 삭제
heygen.delete_video(video_id="video_id")
```

## 매개변수 참조

### generate_video 매개변수

| 매개변수 | 유형 | 필수 | 설명 |
|----------|------|------|------|
| avatar_id | string | 예 | 사용할 아바타 ID |
| script | string | 예 | 말할 텍스트 |
| voice_id | string | 예 | 음성 ID |
| background | string | 아니요 | 배경 장면 |
| speed | float | 아니요 | 말하기 속도 (0.5-2.0) |
| emotion | string | 아니요 | 감정 스타일 |
| caption | boolean | 아니요 | 자막 활성화 |

### 사용 가능한 감정

| 감정 | 설명 |
|------|------|
| Neutral | 기본 표정 |
| Happy | 행복한 표정 |
| Sad | 슬픈 표정 |
| Angry | 분노한 표정 |
| Surprised | 놀란 표정 |

## 웹 인터페이스 기능

### 대시보드

- **프로젝트**: 비디오 프로젝트 관리
- **템플릿**: 사전 제작된 템플릿 사용
- **미디어 라이브러리**: 자산 저장
- **분석**: 성능 지표 보기

- **타임라인**: 비디오 세그먼트 편집
- **자막**: 자막 추가/편집
- **브랜딩**: 로고 및 워터마크 추가
- **내보내기**: 다양한 형식 옵션

## 모범 사례

### 1. 스크립트 작성

- 문장을 짧고 명확하게 유지
- 자연스러운 언어 사용
- 복잡한 용어 피하기

### 2. 아바타 선택

- 사용 사례에 아바타 맞춤
- 시청자 선호도 고려
- 여러 아바타 테스트

### 3. 음성 선택

- 브랜드에 음성 맞춤
- 언어 요구사항 고려
- 샘플 스크립트로 테스트

### 4. 비디오 최적화

- 적절한 해상도 사용
- 플랫폼에 최적화 (소셜 미디어, 웹사이트)
- 접근성을 위해 자막 추가

## 문제 해결

### 비디오 품질 문제

**문제:** 비디오가 픽셀화됨

**해결책:**
- 더 높은 해상도 설정 사용
- 소스 이미지가 고품질인지 확인
- 인터넷 연결 확인

### 오디오 문제

**문제:** 오디오가 비디오와 일치하지 않음

**해결책:**
- 제공된 음성 사용
- 오디오가 올바르게 포맷되었는지 확인
- 립싱크 설정 확인

### 처리 시간

**문제:** 비디오 처리 시간이 너무 김

**해결책:**
- 크레딧 잔액 확인
- 배치 작업에 대기열 사용
- 우선 처리 고려
