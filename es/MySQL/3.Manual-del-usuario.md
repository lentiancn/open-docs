# Manual de usuario de MySQL

## Operaciones básicas de base de datos

### Conectar a MySQL

Conexión local:
```bash
mysql -u root -p
```

Conexión remota:
```bash
mysql -h hostname -u username -p database_name
```

### Crear base de datos

```sql
CREATE DATABASE nombre_base_datos;
```

Crear con conjunto de caracteres específico:
```sql
CREATE DATABASE nombre_base_datos CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### Ver bases de datos

```sql
SHOW DATABASES;
```

### Seleccionar base de datos

```sql
USE nombre_base_datos;
```

### Eliminar base de datos

```sql
DROP DATABASE nombre_base_datos;
```

## Operaciones con tablas

### Crear tabla

```sql
CREATE TABLE nombre_tabla (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nombre VARCHAR(100) NOT NULL,
    correo VARCHAR(100) UNIQUE,
    creado_en TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    actualizado_en TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### Ver estructura de tabla

```sql
DESCRIBE nombre_tabla;
-- o
SHOW COLUMNS FROM nombre_tabla;
```

### Ver todas las tablas

```sql
SHOW TABLES;
```

### Modificar estructura de tabla

Agregar columna:
```sql
ALTER TABLE nombre_tabla ADD COLUMN nueva_columna VARCHAR(50);
```

Modificar columna:
```sql
ALTER TABLE nombre_tabla MODIFY nombre_columna VARCHAR(200);
```

Eliminar columna:
```sql
ALTER TABLE nombre_tabla DROP COLUMN nombre_columna;
```

### Eliminar tabla

```sql
DROP TABLE nombre_tabla;
```

## Operaciones con datos

### Insertar datos

Inserción simple:
```sql
INSERT INTO nombre_tabla (columna1, columna2) VALUES ('valor1', 'valor2');
```

Inserciones múltiples:
```sql
INSERT INTO nombre_tabla (columna1, columna2) VALUES 
('valor1', 'valor2'),
('valor3', 'valor4');
```

### Consultar datos

Consulta básica:
```sql
SELECT * FROM nombre_tabla;
```

Consulta condicional:
```sql
SELECT columna1, columna2 FROM nombre_tabla WHERE condición;
```

Ordenación:
```sql
SELECT * FROM nombre_tabla ORDER BY nombre_columna DESC;
```

Paginación:
```sql
SELECT * FROM nombre_tabla LIMIT 10 OFFSET 20;
```

### Actualizar datos

```sql
UPDATE nombre_tabla SET columna1 = 'nuevo_valor' WHERE condición;
```

### Eliminar datos

```sql
DELETE FROM nombre_tabla WHERE condición;
```

Truncar tabla:
```sql
TRUNCATE TABLE nombre_tabla;
```

## Operaciones con índices

### Crear índices

Índice normal:
```sql
CREATE INDEX nombre_índice ON nombre_tabla (nombre_columna);
```

Índice único:
```sql
CREATE UNIQUE INDEX nombre_índice ON nombre_tabla (nombre_columna);
```

Índice compuesto:
```sql
CREATE INDEX nombre_índice ON nombre_tabla (columna1, columna2);
```

### Ver índices

```sql
SHOW INDEX FROM nombre_tabla;
```

### Eliminar índices

```sql
DROP INDEX nombre_índice ON nombre_tabla;
```

## Gestión de usuarios y privilegios

### Crear usuario

```sql
CREATE USER 'nombre_usuario'@'localhost' IDENTIFIED BY 'contraseña';
```

### Otorgar privilegios

```sql
GRANT SELECT, INSERT, UPDATE ON nombre_base_datos.* TO 'nombre_usuario'@'localhost';
```

Otorgar todos los privilegios:
```sql
GRANT ALL PRIVILEGES ON nombre_base_datos.* TO 'nombre_usuario'@'localhost';
```

### Actualizar privilegios

```sql
FLUSH PRIVILEGES;
```

### Ver privilegios de usuario

```sql
SHOW GRANTS FOR 'nombre_usuario'@'localhost';
```

### Eliminar usuario

```sql
DROP USER 'nombre_usuario'@'localhost';
```

## Control de transacciones

### Iniciar transacción

```sql
START TRANSACTION;
-- o
BEGIN;
```

### Confirmar transacción

```sql
COMMIT;
```

### Revertir transacción

```sql
ROLLBACK;
```

### Establecer puntos de guardado

```sql
SAVEPOINT nombre_punto;
ROLLBACK TO SAVEPOINT nombre_punto;
```

## Funciones comunes

### Funciones de cadena

```sql
SELECT CONCAT('Hola', ' ', 'Mundo');
SELECT UPPER('hola');
SELECT LOWER('HOLA');
SELECT SUBSTRING('Hola Mundo', 1, 5);
SELECT LENGTH('Hola');
SELECT TRIM('  Hola  ');
```

### Funciones numéricas

```sql
SELECT ABS(-5);
SELECT CEIL(3.14);
SELECT FLOOR(3.14);
SELECT ROUND(3.14159, 2);
SELECT MOD(10, 3);
```

### Funciones de fecha

```sql
SELECT NOW();
SELECT CURDATE();
SELECT CURTIME();
SELECT DATE('2024-01-01 10:30:00');
SELECT YEAR('2024-01-01');
SELECT MONTH('2024-01-01');
SELECT DAY('2024-01-01');
SELECT DATE_ADD('2024-01-01', INTERVAL 1 DAY);
SELECT DATEDIFF('2024-01-01', '2023-12-01');
```

### Funciones de agregación

```sql
SELECT COUNT(*) FROM nombre_tabla;
SELECT SUM(nombre_columna) FROM nombre_tabla;
SELECT AVG(nombre_columna) FROM nombre_tabla;
SELECT MAX(nombre_columna) FROM nombre_tabla;
SELECT MIN(nombre_columna) FROM nombre_tabla;
```

## Copias de seguridad y recuperación

### Respaldar base de datos

```bash
mysqldump -u root -p nombre_base_datos > backup.sql
```

Respaldar todas las bases de datos:
```bash
mysqldump -u root -p --all-databases > todas_bases_datos.sql
```

### Restaurar base de datos

```bash
mysql -u root -p nombre_base_datos < backup.sql
```

### Exportar resultados de consulta

```bash
mysql -u root -p -e "SELECT * FROM nombre_tabla" nombre_base_datos > salida.tsv
```

## Optimización del rendimiento

### Analizar consultas con EXPLAIN

```sql
EXPLAIN SELECT * FROM nombre_tabla WHERE condición;
```

### Consejos de optimización

1. Crear índices para condiciones de consulta comunes
2. Evitar SELECT *, solo consultar columnas necesarias
3. Usar LIMIT para restringir conjuntos de resultados
4. Evitar usar funciones en columnas indexadas
5. Usar JOINs correctamente, evitar productos cartesianos
6. Analizar y optimizar tablas regularmente

### Ver registro de consultas lentas

```sql
SHOW VARIABLES LIKE 'slow_query_log%';
```

## Manejo de errores comunes

### Errores de conexión
Verificar si el servicio MySQL está ejecutándose, el puerto es correcto, y los privilegios del usuario están configurados correctamente.

### Errores de conjunto de caracteres
Asegurarse de que la base de datos, tabla y conexión usen el mismo conjunto de caracteres (se recomienda utf8mb4).

### Errores de privilegios
Confirmar que el usuario tenga los privilegios apropiados para operaciones de base de datos y tabla.
