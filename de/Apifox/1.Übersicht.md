# Apifox Übersicht

API All-in-One Kollaborationsplattform.

## Was ist Apifox?

Apifox ist eine Plattform für API-Dokumentation, Debugging, Mock und Tests.

### Funktionen

- **API Dokumentation** - Automatische Generierung
- **API Debugging** - HTTP Simulator
- **Mock Services** - Mock Server
- **Automatisierte Tests** - API Tests
- **Zusammenarbeit** - Projektmanagement
