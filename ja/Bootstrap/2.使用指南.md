# Bootstrap 5 使用ガイド

このガイドでは、如何にBootstrap 5を使用してレスポンシブウェブを構築するかを詳しく説明します。

## クイックスタート

### 基本テンプレート

```html
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 5 ページ</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <!-- コンテンツ領域 -->
    <div class="container mt-5">
        <h1>こんにちは、Bootstrap 5！</h1>
        <p class="lead">これはレスポンシブページです。</p>
    </div>

    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## レイアウトシステム

### コンテナ

Bootstrap 5は以下の2つのコンテナクラスを提供します：

| クラス名 | 説明 |
|----------|------|
| `.container` | 固定幅コンテナ |
| `.container-fluid` | 全幅コンテナ |

```html
<!-- 固定幅コンテナ -->
<div class="container">
    <p>コンテンツは中央配置、左右に余白あり</p>
</div>

<!-- 全幅コンテナ -->
<div class="container-fluid">
    <p>全幅を占める</p>
</div>
```

### グリッドシステム

Bootstrapは12列グリッドシステムを使用しています：

```html
<div class="container">
    <div class="row">
        <div class="col-12 col-md-6 col-lg-4">
            <p>レスポンシブ列</p>
        </div>
        <div class="col-12 col-md-6 col-lg-4">
            <p>レスポンシブ列</p>
        </div>
        <div class="col-12 col-md-6 col-lg-4">
            <p>レスポンシブ列</p>
        </div>
    </div>
</div>
```

#### グリッドブレークポイント

| ブレークポイント | クラスプレフィックス | 幅 |
|-----------------|---------------------|------|
| 超小 | col- | < 576px |
| 小 | col-sm- | ≥ 576px |
| 中 | col-md- | ≥ 768px |
| 大 | col-lg- | ≥ 992px |
| 特大 | col-xl- | ≥ 1200px |
| 超大 | col-xxl- | ≥ 1400px |

### Flex レイアウト

```html
<div class="d-flex justify-content-between">
    <div>左側</div>
    <div>右側</div>
</div>

<div class="d-flex align-items-center">
    <div>垂直中央</div>
</div>
```

### ツールチップ

```html

</button>

<!-- 初期化が必要 -->
<script>
    var tooltipTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="tooltip"]'))

        return new bootstrap.Tooltip(tooltipTriggerEl)
    })
</script>
```

## JavaScript コンポーネント

### 初期化

```javascript
// すべてのツールチップを有効化
var tooltipList = [].slice.call(document.querySelectorAll('[data-bs-toggle="tooltip"]'))
var tooltipList = tooltipList.map(function (tooltipTriggerEl) {
    return new bootstrap.Tooltip(tooltipTriggerEl)
})

// すべてのポップオーバーを有効化
var popoverTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="popover"]'))
var popoverList = popoverTriggerList.map(function (popoverTriggerEl) {
    return new bootstrap.Popover(popoverTriggerEl)
})

// モーダル制御
var myModal = new bootstrap.Modal(document.getElementById('myModal'))
myModal.show()     // 表示
myModal.hide()     // 非表示

// カルーセル制御
var carousel = new bootstrap.Carousel(document.getElementById('myCarousel'))
carousel.next()     // 次へ
carousel.prev()    // 前へ
carousel.cycle()   // 自動再生開始
carousel.pause()   // 一時停止
```

### イベント

```javascript
// モーダルイベント
var myModalEl = document.getElementById('myModal')
myModalEl.addEventListener('show.bs.modal', function (event) {
    console.log('モーダルが表示される前')
})
myModalEl.addEventListener('shown.bs.modal', function (event) {
    console.log('モーダルが表示された後')
})
myModalEl.addEventListener('hide.bs.modal', function (event) {
    console.log('モーダルが非表示になる前')
})
myModalEl.addEventListener('hidden.bs.modal', function (event) {
    console.log('モーダルが非表示になった後')
})
```

## ベストプラクティス

### 1. セマンティックHTMLを使用

```html
<!-- 非推奨 -->
<div class="container">
    <div class="row">
        <div class="col">
            <div class="btn btn-primary">ボタン</div>
        </div>
    </div>
</div>

<!-- 推奨 -->
<main class="container">
    <section class="row">
        <article class="col">
            <button class="btn btn-primary">ボタン</button>
        </article>
    </section>
</main>
```

### 2. コードをシンプルに

```html
<!-- 非推奨 -->
<div class="container-fluid">
    <div class="row">
        <div class="col-12 col-sm-6 col-md-4 col-lg-3 col-xl-2">
            <div class="p-3 mb-2 bg-primary text-white rounded shadow-lg">
                コンテンツ
            </div>
        </div>
    </div>
</div>

<!-- 推奨：ユーティリティクラス组合せ -->
<div class="container">
    <div class="row g-3">
        <!-- g-3 グリッドギャップを追加 -->
    </div>
</div>
```

### 3. パフォーマンス最適化

```html
<!-- CDNを使用 -->
```
