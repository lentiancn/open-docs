# Element Plus User Manual

This manual will help you quickly get started with Element Plus and master commonly used components and features.

## Quick Start

Before using Element Plus, ensure installation is complete.

### Basic Usage

Use components directly in Vue components:

```vue
<template>
  <el-button>Default Button</el-button>
  <el-button type="primary">Primary Button</el-button>
  <el-button type="success">Success Button</el-button>
  <el-button type="warning">Warning Button</el-button>
  <el-button type="danger">Danger Button</el-button>
  <el-button type="info">Info Button</el-button>
</template>
```

### Form

Basic form:

```vue
<el-form :model="form" label-width="80px">
  <el-form-item label="Activity Name">
    <el-input v-model="form.name" />
  </el-form-item>
  <el-form-item label="Activity Area">
    <el-select v-model="form.region" placeholder="Please select activity area">

      <el-option label="Area 2" value="beijing" />
    </el-select>
  </el-form-item>
  <el-form-item label="Activity Date">
    <el-date-picker
      v-model="form.date"
      type="date"
      placeholder="Select date"
    />
  </el-form-item>
  <el-form-item label="Instant Delivery">
    <el-switch v-model="form.delivery" />
  </el-form-item>
  <el-form-item label="Activity Type">
    <el-checkbox-group v-model="form.type">
      <el-checkbox label="Online activities" name="type" />
      <el-checkbox label="Offline activities" name="type" />
      <el-checkbox label="Brand activities" name="type" />
    </el-checkbox-group>
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="onSubmit">Create</el-button>
    <el-button>Cancel</el-button>

<script setup>
import { reactive } from 'vue'

const form = reactive({
  name: '',
  region: '',
  date: '',
  delivery: false,
  type: [],
})

const onSubmit = () => {
  console.log('submit!', form)
}
</script>
```

Form validation:

```vue
<el-form :model="ruleForm" :rules="rules" ref="ruleFormRef" label-width="100px">
  <el-form-item label="Activity Name" prop="name">
    <el-input v-model="ruleForm.name" />
  </el-form-item>
  <el-form-item label="Activity Area" prop="region">
    <el-select v-model="ruleForm.region" placeholder="Please select activity area">
      <el-option label="Area 1" value="shanghai" />
      <el-option label="Area 2" value="beijing" />
    </el-select>
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="submitForm(ruleFormRef)">Submit</el-button>
    <el-button @click="resetForm(ruleFormRef)">Reset</el-button>
  </el-form-item>
</el-form>

<script setup>
import { reactive, ref } from 'vue'

const ruleFormRef = ref()
const ruleForm = reactive({
  name: '',
  region: '',
})

const rules = reactive({
  name: [
    { required: true, message: 'Please enter activity name', trigger: 'blur' },
    { min: 3, max: 5, message: 'Length should be 3 to 5 characters', trigger: 'blur' },
  ],
  region: [
    { required: true, message: 'Please select activity area', trigger: 'change' },
  ],
})

const submitForm = async (formEl) => {
  if (!formEl) return
  await formEl.validate((valid, fields) => {
    if (valid) {
      console.log('submit!')
    } else {
      console.log('error submit!', fields)
    }
  })
}

const resetForm = (formEl) => {
  if (!formEl) return
  formEl.resetFields()
}
</script>
```

## Layout System

### Grid System

Element Plus uses a 24-column grid system:

```vue
<el-row :gutter="20">
  <el-col :span="12"><div class="grid-content">12</div></el-col>
  <el-col :span="12"><div class="grid-content">12</div></el-col>
</el-row>

<el-row :gutter="20">
  <el-col :span="8"><div class="grid-content">8</div></el-col>
  <el-col :span="8"><div class="grid-content">8</div></el-col>
  <el-col :span="8"><div class="grid-content">8</div></el-col>
</el-row>

<style>
.grid-content {
  border-radius: 4px;
  min-height: 36px;
}
</style>
```

## Best Practices

### Responsive Design

Combine grid system with responsive breakpoints:

```vue
<el-row :gutter="20">
  <el-col :xs="24" :sm="12" :md="8" :lg="6" :xl="4">
    <div class="grid-content"></div>
  </el-col>
</el-row>
```

### Form Validation

Always validate user input and provide clear error messages.

### Internationalization

Use Element Plus built-in i18n:

```javascript
import en from 'element-plus/es/locale/lang/en'

app.use(ElementPlus, { locale: en })
```

## Common Issues

### Component Styles Not Applied

Make sure CSS is imported:

```javascript
import 'element-plus/dist/index.css'
```

### Icons Not Displayed

Need to install icon library separately:

```bash
npm install @element-plus/icons-vue
```

Then register globally:

```javascript
import * as icons from '@element-plus/icons-vue'
app.use(ElementPlus)
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```

### Table Performance Issues

For large datasets, use virtual scrolling:

```vue
<el-table-v2
  :columns="columns"
  :data="data"
  :height="400"
/>
```
