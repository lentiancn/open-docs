# Bootstrap Introduction

> Bootstrap is a popular CSS framework

---

## Overview

Bootstrap is an open-source CSS framework developed by Twitter, released in 2011. Provides responsive grid system and pre-built components for modern websites.

---

## Features

- Responsive design
- Pre-built components
- Grid system
- Sass customization
- JavaScript plugins
