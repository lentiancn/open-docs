# Foire Aux Questions

Ce document recense les questions techniques fréquemment posées lors de l'utilisation d'Apifox et leurs réponses.

## Questions de base

### Qu'est-ce qu'Apifox ?

Apifox est une plateforme de collaboration API qui intègre la conception, la gestion de documentation, le débogage, le mock et les tests automatisés. Elle aide les équipes de développement à être plus efficaces dans le développement, les tests et la maintenance de la documentation des API.

### Apifox est-il gratuit ?

Apifox propose une version gratuite et une version payante. La version gratuite contient toutes les fonctionnalités principales, suffisantes pour répondre à la plupart des besoins de développement. La version payante offre des fonctionnalités avancées supplémentaires, comme des équipes plus grandes et une automatisation avancée.

### Quelle est la différence entre Apifox et Postman ?

Apifox et Postman sont tous deux des outils API, mais Apifox est plus adapté aux utilisateurs chinois : un meilleur support en français, des fonctions de mock plus intelligentes et une expérience de collaboration d'équipe plus complète. Les fonctionnalités de base d'Apifox sont entièrement gratuites, tandis que de nombreuses fonctionnalités de Postman sont payantes.

## Questions d'installation

### Que faire si le téléchargement est lent ?

Apifox propose plusieurs serveurs de téléchargement. Vous pouvez essayer de visiter https://apifox.com/ pour choisir le serveur le plus proche de chez vous.

### Que faire si le logiciel ne démarre pas ?

1. Vérifiez si votre version du système répond aux exigences
2. Essayez d'exécuter en tant qu'administrateur
3. Vérifiez si les bibliothèques nécessaires sont installées
4. Consultez les journaux d'erreur pour plus d'informations

### Que faire si la connexion échoue ?

1. Vérifiez si la connexion réseau est normale
2. Confirmez que l'identifiant et le mot de passe sont corrects
3. Essayez de vous connecter via un compte tiers
4. Contactez le support officiel pour obtenir de l'aide

## Questions d'utilisation

### Les données de l'interface ne se synchronisent pas ?

1. Confirmez que vous êtes connecté à votre compte
2. Vérifiez la connexion réseau
3. Cliquez sur le bouton "Synchroniser" pour synchroniser manuellement
4. Vérifiez les permissions de votre compte

### Les données Mock ne fonctionnent pas ?

1. Confirmez que la fonction Mock est activée
2. Vérifiez si l'adresse Mock est correcte
3. Vérifiez s'il y a des erreurs de syntaxe dans l'interface
4. Essayez de rafraîchir la page

### L'envoi de la requête échoue ?

1. Vérifiez la connexion réseau
2. Confirmez que l'adresse de la requête est correcte
3. Vérifiez le format des paramètres de requête
4. Vérifiez si la requête est interceptée par un proxy

### La réponse s'affiche avec des caractères illisibles ?

1. Essayez de modifier l'encodage de la réponse
2. Vérifiez les paramètres d'encodage dans les en-têtes de réponse
3. Utilisez la fonction "Formater" pour réanalyser

## Questions sur les fonctionnalités

### Comment importer des données Postman ?

1. Cliquez sur "Importer"
2. Sélectionnez "Postman"
3. Choisissez le fichier à importer
4. Confirmez l'importation

### Comment exporter en Swagger ?

1. Accédez aux paramètres du projet
2. Sélectionnez "Exporter"
3. Choisissez "Swagger/OpenAPI"
4. Sélectionnez la version
5. Cliquez sur exporter

### Comment utiliser les variables d'environnement ?

1. Définissez les variables d'environnement dans les paramètres du projet
2. Dans les requêtes, utilisez {{nom_variable}} pour les référencer
3. Utilisez le sélecteur d'environnement en haut pour changer d'environnement

### Comment effectuer des opérations en lot sur les interfaces ?

1. Maintenez Ctrl/Cmd et sélectionnez plusieurs interfaces
2. Faites un clic droit pour sélectionner une opération en lot
3. Choisissez le type d'opération (supprimer, exporter, déplacer, etc.)

## Questions sur les tests

### Comment exécuter les tests automatisés ?

1. Ouvrez la suite de tests
2. Cliquez sur le bouton "Exécuter"
3. Consultez les résultats des tests

### Que faire si une assertion échoue ?

1. Consultez la raison de l'échec
2. Vérifiez les valeurs attendues et réelles
3. Modifiez les conditions d'assertion
4. Réexécutez les tests

### Comment exporter le rapport de test ?

1. Une fois les tests terminés
2. Cliquez sur "Exporter le rapport"
3. Choisissez le format (HTML, PDF, JSON)

## Questions de collaboration d'équipe

### Comment inviter des membres de l'équipe ?

1. Accédez aux paramètres du projet
2. Cliquez sur "Gestion des membres"
3. Cliquez sur "Inviter un membre"
4. Entrez l'adresse e-mail ou le nom d'utilisateur du membre

### Comment configurer les permissions des membres ?

Lors de l'invitation d'un membre, vous pouvez définir différents rôles :

- Administrateur : contrôle total
- Éditeur : peut modifier et exécuter
- Lecteur : permissions en lecture seule

### Comment résoudre les conflits ?

 Lorsque plusieurs personnes modifient simultanément la même interface :

1. Le système signale le conflit
2. Choisissez de conserver la version locale ou la version distante
3. Ou fusionnez manuellement les modifications

## Questions sur les données

### Comment sauvegarder les données ?

1. Exportez régulièrement le projet au format Apifox
2. Ou utilisez la fonction de synchronisation cloud

### Comment restaurer les données ?

1. Si vous avez un fichier de sauvegarde, cliquez sur "Importer"
2. Sélectionnez le fichier de sauvegarde pour restaurer

### Les données supprimées peuvent-elles être récupérées ?

Les données supprimées sont conservées dans la corbeille pendant 30 jours et peuvent être récupérées depuis la corbeille.

## Questions de performance

### Que faire en cas de ralentissement ?

1. Fermez les projets non nécessaires
2. Effacez le cache du navigateur
3. Mettez à niveau la configuration matérielle
4. Essayez la dernière version

### L'importation de grandes quantités de données est-elle lente ?

1. Importez en lots
2. Désactivez la synchronisation automatique
3. Patientez jusqu'à la fin de l'importation
