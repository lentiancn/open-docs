# FantasyTalking User Manual

## Basic Usage

```javascript
ft.speak('Hello')
ft.stop()
ft.pause()
ft.resume()
```

## Events

```javascript
ft.on('start', () => {})
ft.on('end', () => {})
ft.on('error', (err) => {})
```

## Voice Settings

```javascript
ft.setSpeed(1.5)
ft.setPitch(1.2)
ft.setVolume(0.8)
```
