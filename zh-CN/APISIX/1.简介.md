# APISIX - 简介

APISIX 是 Apache 软件基金会旗下的云原生 API 网关，使用 Nginx + OpenResty 构建，提供高性能、低延迟的 API 管理服务。

## 核心特性

- **高性能**：基于 OpenResty，单核 QPS 可达 18k+
- **动态路由**：支持 URL、Header、Cookie 等多种路由方式
- **插件热加载**：无需重启即可更新插件配置
- **丰富插件**：认证、限流、熔断、日志等 80+ 插件
- **多协议支持**：HTTP、HTTPS、HTTP/2、gRPC、WebSocket、Dubbo

## 架构特点

- 控制面与数据面分离
- 支持 Etcd 作为配置中心
- 支持 Consul、Kubernetes Ingress 等多种发现方式

## 应用场景

- API 统一入口
- 微服务网关
- 流量管理与安全防护
