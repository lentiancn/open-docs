# Angular

> Google 開發的 Web 框架

---

## 說明

Angular 是由 Google 開發和維護的 TypeScript 框架。提供構建現代 Web 應用的完整解決方案。

## 版本

| 版本 | 發布日期 | 狀態 |
|------|----------|------|
| Angular 17 | 2023年11月 | 當前 |
| Angular 16 | 2023年5月 | 穩定版 |
