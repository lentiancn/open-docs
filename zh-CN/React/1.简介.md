# React 简介

用于构建用户界面的 JavaScript 库。

## 什么是 React？

React 是 Facebook 维护的用于构建用户界面的 JavaScript 库。React 采用组件化的开发方式，使用虚拟 DOM 来提高性能，通过声明式的方式来描述用户界面。

### 核心特性

- **组件化**：基于组件的开发
- **虚拟 DOM**：高效的 DOM 更新
- **声明式**：描述 UI 状态
- **单向数据流**：数据流向清晰
- **Hooks**：函数组件状态管理
- **服务端渲染**：SSR 支持
- **React Native**：移动端开发
