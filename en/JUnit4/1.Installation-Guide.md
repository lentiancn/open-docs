# JUnit 4 Installation Guide

Comprehensive guide for installing JUnit 4 on all major platforms.

---

## What is JUnit 4?

JUnit 4 is a unit testing framework for Java. It is the most widely used testing framework in the Java ecosystem.

---

## Maven

### Add Dependency

```xml
<dependency>
    <groupId>junit</groupId>
    <artifactId>junit</artifactId>
    <version>4.13.2</version>
    <scope>test</scope>
</dependency>
```

### Full pom.xml

```xml
<project xmlns="http://maven.apache.org/POM/4.0.0"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 http://maven.apache.org/xsd/maven-4.0.0.xsd">
    <modelVersion>4.0.0</modelVersion>

    <groupId>com.example</groupId>
    <artifactId>my-project</artifactId>
    <version>1.0-SNAPSHOT</version>

    <dependencies>
        <dependency>
            <groupId>junit</groupId>
            <artifactId>junit</artifactId>
            <version>4.13.2</version>
            <scope>test</scope>
        </dependency>
    </dependencies>
</project>
```

## Gradle

### Add Dependency

```groovy
testImplementation 'junit:junit:4.13.2'
```

### Full build.gradle

```groovy
plugins {
    id 'java'
}

group = 'com.example'
version = '1.0-SNAPSHOT'

repositories {
    mavenCentral()
}

dependencies {
    testImplementation 'junit:junit:4.13.2'
}

test {
    useJUnit()
}
```

## Eclipse

### Using Eclipse Marketplace

1. Go to **Help > Eclipse Marketplace**
2. Search for **JUnit**
3. Install **JUnit**

### Manual Setup

1. Download JUnit jar from https://github.com/junit-team/junit4/releases
2. Right-click project â†’ **Build Path > Add External Archives**
3. Select **junit-4.13.2.jar** and **hamcrest-core-1.3.jar**

## IntelliJ IDEA

### Using Maven/Gradle

JUnit is automatically available when you add Maven or Gradle dependency.

### Manual Setup

1. Download JUnit jar from https://github.com/junit-team/junit4/releases
2. Go to **File > Project Structure > Modules**
3. Add JUnit library to **Dependencies** tab

## Standalone

### Download JAR

```bash
# Download JUnit
curl -L -o junit-4.13.2.jar https://github.com/junit-team/junit4/releases/download/r4.13.2/junit-4.13.2.jar

# Download Hamcrest (required)
curl -L -o hamcrest-core-1.3.jar https://repo1.maven.org/maven2/org/hamcrest/hamcrest-core/1.3/hamcrest-core-1.3.jar
```

### Compile and Run

```bash
# Compile
javac -cp .:junit-4.13.2.jar:hamcrest-core-1.3.jar Test.java

# Run
java -cp .:junit-4.13.2.jar:hamcrest-core-1.3.jar org.junit.runner.JUnitCore Test
```
