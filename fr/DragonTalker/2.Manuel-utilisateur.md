## 1. Vue d'ensemble

DragonTalker est un système de génération de têtes parlantes basé sur l'apprentissage profond. Sa fonction principale est de combiner une image statique avec un fichier audio pour générer une vidéo dynamique avec des mouvements de parole. La technologie utilise des algorithmes avancés de reconstruction faciale 3D et de synthèse d'image pour préserver les traits originaux tout en réalisant une synchronisation labiale naturelle.

### 1.1 Domaines d'application

- **Présentateurs virtuels** : configurer des avatars avec voix et expressions réalistes
- **Service client numérique** : générer des vidéos interactives anthropomorphes
- **Éducation** : présenter le contenu du cours sous forme vidéo
- **Doublage** : créer rapidement des animations de doublage pour les personnages

## 2. Démarrage rapide

### 2.1 Utilisation de base

```bash
source venv/bin/activate
python inference.py \
  --source_image examples/source_image.jpg \
  --driven_audio examples/driven_audio.wav \
  --result_dir ./results
```

### 2.2 Paramètres

| Paramètre | Description | Par défaut |
|-----------|-------------|------------|
| --source_image | Chemin de l'image source | Requis |
| --driven_audio | Chemin de l'audio | Requis |
| --result_dir | Dossier de sortie | ./results |
| --enhancer | Algorithme d'amélioration | gfpgan |
| --outputfps | Images par seconde | 25 |

---

## 3. Spécifications

### 3.1 Image source

- Format : JPG, PNG
- Résolution : 512×512 ou plus recommandé
- Contenu : visage de face, net, sans obstruction

### 3.2 Audio

- Format : WAV, MP3
- Durée : 1-60 secondes
- Qualité : voix claire

### 3.3 Vidéo de sortie

- Format : MP4 (H.264)
- Résolution : 256×256 ou 512×512
- Images : 25 FPS

## 5. Python API

### Utilisation de base

```python
from dragon_talker import DragonTalker

model = DragonTalker(
    device='cuda',
    faceid_weight=0.5,
    enhancer='gfpgan'
)

model.load_source('source.jpg')
model.load_audio('audio.wav')

result_path = model.generate(
    output_path='result.mp4',
    preprocess='full',
    fps=25
)

print(f"Vidéo enregistrée : {result_path}")
```

## 6. FAQ

### 6.1 Qualité vidéo

**Visage généré flou**

Solution : activer l'amélioration faciale

```bash
python inference.py --enhancer gfpgan ...
```

---

**Lèvres non synchronisées avec l'audio**

Cause : qualité audio insuffisante ou discours trop rapide

Solution : pré-traiter l audio, augmenter le taux d'échantillonnage

---

### 6.2 Performance

**Mémoire GPU insuffisante**

Solutions :
1. Réduire la résolution de sortie
2. Désactiver l'amélioration
3. Utiliser le mode CPU
