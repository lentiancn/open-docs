# Element Plus Preguntas Frecuentes

Preguntas comunes.

## Instalación

### Q: ¿Error de instalación?

**Soluciones:**
1. Verificar versión de Node.js
```bash
node --version  # Requiere 18.0+
```
2. Limpiar cache
```bash
npm cache clean --force
```

### Q: ¿Errores de TypeScript?

**Soluciones:**
Instalar tipos:
```bash
npm install -D @types/node
```

## Uso

### Q: ¿Componente no aparece?

**Soluciones:**
1. Verificar importación
2. Verificar versión de Vue (3.2+)

### Q: ¿Estilos no se aplican?

**Soluciones:**
1. Verificar importación de CSS
2. Usar `:deep()`

### Q: ¿Iconos no aparecen?

**Soluciones:**
Instalar paquete de iconos:
```bash
npm install @element-plus/icons-vue
```

Registrar:
```typescript
import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```
