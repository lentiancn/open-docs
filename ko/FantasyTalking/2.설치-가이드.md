# FantasyTalking - 설치 가이드

완전한 설치 및 설정 가이드.

## 요구사항

### 하드웨어

- **GPU**: NVIDIA 8GB+ VRAM (16GB 권장)
- **저장소**: 20GB+ 여유 공간
- **RAM**: 16GB+

### 소프트웨어

- **OS**: Linux (Ubuntu 20.04+), Windows 10/11, macOS
- **Python**: 3.8 - 3.10
- **CUDA**: 11.8+
- **cuDNN**: 8.0+

## 설치

### 방법 1: 소스에서 설치 (권장)

```bash
# 1. 저장소 복제
git clone https://github.com/FantasyTalking/FantasyTalking.git
cd FantasyTalking

# 2. 가상 환경 생성 (권장)
python -m venv venv
source venv/bin/activate  # Linux/macOS
# 또는
venv\Scripts\activate  # Windows

# 3. 의존성 설치
pip install -r requirements.txt

# 4. 사전 학습된 모델 다운로드
python download_models.py
```

### 방법 2: Docker (권장)

```bash
# 1. 이미지 풀
docker pull fantasytalking/fantasytalking:latest

# 2. 컨테이너 실행
docker run -it --gpus all -v /your/data:/data fantasytalking/fantasytalking:latest
```

### 방법 3: pip

```bash
pip install fantasytalking
```

## 확인

설치 후 다음 명령을 실행하여 확인:

```bash
python -c "import fantasytalking; print(fantasytalking.__version__)"
```

## 설정

### 설정 파일

프로젝트 루트에 `config.yaml` 생성:

```yaml
model:
  checkpoint_path: "./models/checkpoint.pth"
  face_enhancer: "realusf"

output:
  resolution: "512x512"
  fps: 30

audio:
  sample_rate: 16000
```

### 환경 변수

```bash
export FT_MODEL_PATH="./models"
export FT_OUTPUT_PATH="./output"
```

## 문제 해결

### CUDA 사용 불가?

1. NVIDIA 드라이버가 설치되어 있는지 확인: `nvidia-smi`
2. CUDA 버전 확인
3. 호환되는 PyTorch 버전 설치

### 메모리 부족?

1. 더 작은 입력 이미지 사용
2. 출력 해상도 낮추기
3. 배치 처리 모드 사용

### 모델 다운로드 실패?

```bash
# 프록시 사용
export HTTP_PROXY=http://127.0.0.1:7890
export HTTPS_PROXY=http://127.0.0.1:7890

# 수동 다운로드
python download_models.py --mirror china
```

## 제거

```bash
pip uninstall fantasytalking
rm -rf FantasyTalking
```
