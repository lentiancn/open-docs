# OpenClaw 🦞

OpenClaw ist ein **selbst gehostetes Gateway**, das deine Lieblings-Chat-Apps — WhatsApp, Telegram, Discord, iMessage und mehr — mit KI-Coding-Assistenten wie Pi verbindet. Du führst einen einzelnen Gateway-Prozess auf deinem eigenen Rechner (oder Server) aus, und er wird zur Brücke zwischen deinen Messaging-Apps und einem stets verfügbaren KI-Assistenten.

## Für wen ist es gedacht?

OpenClaw ist für Entwickler und Power-User gedacht, die einen persönlichen KI-Assistenten haben möchten, den sie von überall aus anschreiben können — ohne die Kontrolle über ihre Daten aufzugeben oder auf einen gehosteten Dienst angewiesen zu sein.

## Hauptmerkmale

Was OpenClaw einzigartig macht:

- **Selbst gehostet**: läuft auf deiner Hardware, deine Regeln
- **Mehrkanalig**: ein Gateway bedient WhatsApp, Telegram, Discord und mehr gleichzeitig
- **Agent-nativ**: gebaut für Coding-Agenten mit Tool-Nutzung, Sessions, Memory und Multi-Agent-Routing
- **Open Source**: MIT-Lizenz, community-getrieben

## Was du brauchst

- Node.js 22 oder höher
- Einen API-Schlüssel von deinem gewählten KI-Anbieter
- 5 Minuten

Verwende für beste Qualität und Sicherheit das stärkste verfügbare Modell der neuesten Generation.

## Schnellstart

```bash
# OpenClaw installieren
npm install -g openclaw@latest

# Onboarding-Assistent ausführen
openclaw onboard --install-daemon

# Bei Kanälen anmelden und Gateway starten
openclaw channels login
openclaw gateway --port 18789
```

## Wie es funktioniert

Das Gateway ist die alleinige Quelle für Sessions, Routing und Kanalverbindungen. Chat-Apps verbinden sich darüber mit KI-Agenten, CLI, Web-Control-UI oder mobilen Knoten.

## Hauptfunktionen

- **Mehrkanal-Gateway**: WhatsApp, Telegram, Discord und iMessage mit einem einzigen Gateway-Prozess
- **Plugin-Kanäle**: Füge Mattermost und mehr mit Erweiterungspaketen hinzu
- **Multi-Agent-Routing**: isolierte Sessions pro Agent, Workspace oder Absender
- **Medienunterstützung**: Senden und Empfangen von Bildern, Audio und Dokumenten
- **Web-Control-UI**: Browser-Dashboard für Chat, Konfiguration, Sessions und Knoten
- **Mobile Knoten**: Verbinde iOS- und Android-Knoten für Canvas, Kamera/Bildschirm und sprachaktivierte Workflows
