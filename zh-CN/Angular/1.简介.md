# 简介

Angular 是由 Google 开发维护的开源 Web 应用框架，用于构建现代 Web 应用程序。它是 AngularJS（Angular 1.x）的后继者，完全重写，采用 TypeScript 作为主要开发语言。

Angular 提供了一个完整的前端开发解决方案，包括组件化架构、双向数据绑定、依赖注入、路由、表单处理、HTTP 客户端等功能。通过这些特性，开发者可以快速构建高性能、可维护的单页应用（SPA）。

## 核心概念

### 组件（Component）

组件是 Angular 应用的基本构建块。每个组件由三个部分组成：
- 模板（Template）：定义视图的结构
- 样式（Styles）：定义视图的外观
- 逻辑（Logic）：定义组件的行为

### 模块（Module）

模块用于组织代码，将相关的组件、服务、指令等组织在一起。每个 Angular 应用至少有一个根模块（AppModule）。

### 服务（Service）

服务用于封装可重用的业务逻辑，通过依赖注入机制提供给组件使用。

### 指令（Directive）

指令用于扩展 HTML 元素的功能，分为结构指令（如 *ngIf、*ngFor）和属性指令（如 ngClass、ngStyle）。

## 核心特性

### 组件化架构

Angular 采用组件化设计，每个组件都是独立的功能单元，可以复用和组合。

### TypeScript

Angular 使用 TypeScript 作为主要开发语言，提供强类型支持、面向对象编程和现代 JavaScript 特性。

### 依赖注入

Angular 内置依赖注入系统，使得代码更加模块化、可测试和可维护。

### 双向数据绑定

模板与组件之间的数据自动同步，减少了手动操作 DOM 的代码。

### 路由

强大的路由系统，支持懒加载、守卫、参数传递等功能。

### HTTP 客户端

内置 HTTP 客户端，支持拦截器、请求/响应转换等高级功能。

### 表单处理

提供响应式表单和模板驱动表单两种方式，处理复杂的表单验证。

### 响应式编程

全面支持 RxJS，提供强大的异步编程能力。

## 适用场景

- 企业级 Web 应用
- 单页应用（SPA）
- 渐进式 Web 应用（PWA）
- 混合移动应用（结合 Ionic）
- 桌面应用（结合 Electron）

## 版本历史

- AngularJS（1.x）：2010 年发布
- Angular 2：2016 年发布，完全重写
- Angular 17：2023 年发布最新稳定版本

Angular 拥有活跃的社区和完善的文档，是构建现代 Web 应用的主流选择之一。
