# Bootstrap

> 인기 있는 프론트엔드 CSS 프레임워크

---

## 프로젝트 개요

Bootstrap은 Twitter에서 개발하고 유지 관리하는 오픈 소스 CSS 프레임워크로, 2011년에 처음 출시되었습니다. Bootstrap은 풍부한 사전 구축된 컴포넌트와 반응형 그리드 시스템을 제공하여 현대적인 반응형 웹사이트와 웹 애플리케이션을 빠르게 구축할 수 있습니다.

## 시작하기

### CDN 포함

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### 페이지 생성

```html
<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Bootstrap 페이지</title>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <div class="container">
    <h1>안녕하세요 Bootstrap!</h1>
  </div>
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```
