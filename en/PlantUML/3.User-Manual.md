# PlantUML User Manual

## Basic Syntax

### Comments

```plantuml
' This is a comment
```

### Alice-Bob Diagram

```plantuml
@startuml
Alice -> Bob: Hello
Bob --> Alice: Hi there!
@enduml
```

---

## UML Diagrams

### Sequence Diagram

```plantuml
@startuml
actor User
participant "Frontend" as FE
database "Database" as DB

User -> FE: Click button
FE -> DB: Save data
DB --> FE: Success
FE --> User: Show confirmation
@enduml
```

### Class Diagram

```plantuml
@startuml
class User {
    +id: int
    +name: string
    +email: string
    +login()
    +logout()
}

class Admin extends User {
    +approveContent()
    +banUser()
}

User "1" -- "*" Post: writes
@enduml
```

### Use Case Diagram

```plantuml
@startuml
left to right direction
actor Customer
actor Manager

rectangle Shop {
    Customer --> (Browse Products)
    Customer --> (Add to Cart)
    Customer --> (Checkout)
    (Checkout) .> (Process Payment) : include
    Manager --> (Manage Products)
    Manager --> (View Reports)
}
@enduml
```

### Activity Diagram

```plantuml
@startuml
start
:Login to system;
if (Valid credentials?) then (yes)
    :Access dashboard;
    :Perform actions;
    :Logout;
    stop
else (no)
    :Show error;
    stop
endif
@enduml
```

### State Diagram

```plantuml
@startuml
[*] --> Idle
Idle --> Processing: Start
Processing --> Completed: Success
Processing --> Failed: Error
Completed --> [*]
Failed --> Idle: Retry
@enduml
```

### Component Diagram

```plantuml
@startuml
package "Frontend" {
    [Web App]
    [Mobile App]
}

package "Backend" {
    [API Gateway]
    [User Service]
    [Order Service]
}

[Web App] --> [API Gateway]
[Mobile App] --> [API Gateway]
[API Gateway] --> [User Service]
[API Gateway] --> [Order Service]
@enduml
```

## Advanced Features

### Skinparam

```plantuml
@startuml
skinparam backgroundColor #DDD
skinparam classBackgroundColor #lightblue
skinparam arrowColor #333

class Demo
@enduml
```

### Notes

```plantuml
@startuml
Alice -> Bob: Hello
note right: This is a note
@enduml
```

### Grouping

```plantuml
@startuml
group Group1
    Alice -> Bob: Hello1
    Bob --> Alice: Hi1
end group

group Group2
    Alice -> Charlie: Hello2
    Charlie --> Alice: Hi2
end group
@enduml
```

## Export Diagrams

### Command Line

```bash
# Export to PNG
java -jar plantuml.jar diagram.wsd

# Export to SVG
java -jar plantuml.jar -tsvg diagram.wsd

# Export to PDF
java -jar plantuml.jar -tpdf diagram.wsd
```

### VS Code

Right-click in the editor and select:
- "Export Current Diagram"
- Choose format (PNG/SVG/PDF)
