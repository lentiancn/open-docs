# OpenCode Руководство по установке

OpenCode поддерживает несколько способов установки.

## Установка на Linux/macOS

### Рекомендуемый: скрипт установки

```bash
curl -fsSL https://opencode.ai/install | bash
```

### Node.js

```bash
npm install -g opencode-ai
```

### Bun

```bash
bun install -g opencode-ai
```

### pnpm

```bash
pnpm install -g opencode-ai
```

### Yarn

```bash
yarn global add opencode-ai
```

### Homebrew (macOS/Linux)

```bash
brew install anomalyco/tap/opencode
```

### Arch Linux

```bash
# Стабильная
sudo pacman -S opencode

# Последняя (AUR)
paru -S opencode-bin
```

## Установка на Windows

### Chocolatey

```bash
choco install opencode
```

### Scoop

```bash
scoop install opencode
```

### NPM

```bash
npm install -g opencode-ai
```

### Mise

```bash
mise use -g github:anomalyco/opencode
```

### Docker

```bash
docker run -it --rm ghcr.io/anomalyco/opencode
```

### Ручная загрузка

Также можно скачать бинарный файл с [GitHub Releases](https://github.com/anomalyco/opencode/releases).

## Проверка установки

```bash
opencode --version
```

## Настройка

### Подключение LLM-провайдера

Выполните для подключения:

```bash
opencode
/connect
```

Выберите `opencode`, затем посетите [opencode.ai/auth](https://opencode.ai/auth) для входа и получения API-ключа.

### Другие провайдеры

Также можно выбрать других LLM-провайдеров. Подробнее в [официальных документах](https://opencode.ai/docs/providers#directory).

## Инициализация проекта

```bash
cd /path/to/project
opencode
/init
```

Это заставит OpenCode проанализировать проект и создать файл AGENTS.md в корне проекта.
