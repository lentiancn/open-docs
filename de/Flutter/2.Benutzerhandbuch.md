# Flutter Benutzerhandbuch

Dieses Handbuch erklärt, wie Sie Flutter zum Erstellen von plattformübergreifenden Anwendungen verwenden.

## Schnellstart

### Projekt erstellen

```bash
# Neues Projekt erstellen
flutter create my_app

# In das Projektverzeichnis wechseln
cd my_app

# Anwendung ausführen
flutter run
```

### Projektstruktur

```
my_app/
├── lib/
│   ├── main.dart
│   └── ...
├── test/
├── android/
├── ios/
├── web/
├── linux/
├── windows/
├── macos/
├── pubspec.yaml
└── README.md
```

## Statusverwaltung

### setState

```dart
class MyWidget extends StatefulWidget {
  @override
  _MyWidgetState createState() => _MyWidgetState();
}

class _MyWidgetState extends State<MyWidget> {
  int _counter = 0;

  void _increment() {
    setState(() {
      _counter++;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        Text('Zähler: $_counter'),
        ElevatedButton(
          onPressed: _increment,
          child: Text('Erhöhen'),
        ),
      ],
    );
  }
}
```

### Provider

```bash
# Abhängigkeit hinzufügen
flutter pub add provider
```

```dart
// Provider erstellen
class CounterModel extends ChangeNotifier {
  int _count = 0;
  int get count => _count;

  void increment() {
    _count++;
    notifyListeners();
  }
}

// Provider verwenden
Consumer<CounterModel>(
  builder: (context, counter, child) {
    return Text('${counter.count}');
  },
)
```

### GetX

```bash
# Abhängigkeit hinzufügen
flutter pub add get
```

```dart
// GetX verwenden
Get.put(CounterController());

// In Widget verwenden
Get.find<CounterController>().increment();
```

## Routen-Navigation

### Grundlegende Navigation

```dart
// Zu neuer Seite navigieren
Navigator.push(
  context,
  MaterialPageRoute(builder: (context) => NextPage()),
);

// Zurück
Navigator.pop(context);
```

### Benannte Routen

```dart
// Routen konfigurieren
MaterialApp(
  routes: {
    '/': (context) => HomePage(),
    '/about': (context) => AboutPage(),
  },
);

// Navigieren
Navigator.pushNamed(context, '/about');
```

### Parameter übergeben

```bash
# Parameter übergeben
Navigator.pushNamed(
  context,
  '/detail',
  arguments: {'id': 123},
);

// Parameter empfangen
final args = ModalRoute.of(context).settings.arguments;
```

## Netzwerkanfragen

### HTTP

```bash
flutter pub add http
```

```dart
import 'package:http/http.dart' as http;

Future<void> fetchData() async {
  final response = await http.get(Uri.parse('https://api.example.com/data'));
  if (response.statusCode == 200) {
    print(response.body);
  }
}
```

### Dio

```bash
flutter pub add dio
```

```dart
import 'package:dio/dio.dart';

final dio = Dio();

```

## Lokale Speicherung

### SharedPreferences

```bash
flutter pub add shared_preferences
```

```dart
import 'package:shared_preferences/shared_preferences.dart';

// Daten speichern
SharedPreferences prefs = await SharedPreferences.getInstance();
await prefs.setString('name', 'John');
await prefs.setInt('age', 30);

// Daten lesen
String name = prefs.getString('name') ?? '';
int age = prefs.getInt('age') ?? 0;
```

### SQLite

```bash
flutter pub add sqflite
```

```dart
import 'package:sqflite/sqflite.dart';

final db = await openDatabase('my.db');
await db.insert('users', {'name': 'John', 'age': 30});
final users = await db.query('users');
```
