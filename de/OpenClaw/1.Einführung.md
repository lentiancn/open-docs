# OpenClaw 🦞

OpenClaw ist ein Tool, das auf deinem eigenen Gerät läuft und Messaging-Apps mit KI-Assistenten verbindet. WhatsApp, Telegram, Discord, iMessage – alles direkt mit KI-Diensten nutzbar.

## Für wen?

Wenn du einen eigenen KI-Assistenten willst, aber deine Daten nicht Dritten anvertrauen möchtest. Besonders für Entwickler geeignet.

## Merkmale

- **Deine Daten, deine Maschine**: Alles bleibt auf deiner Hardware
- **Mehrere Plattformen**: Ein Prozess verwaltet mehrere Chat-Apps
- **Für Programmierung**: Unterstützt Tools, Verlauf, Memory, Multi-Assistent
- **Open Source**: MIT-Lizenz

## Was du brauchst

- Node.js 22+
- API-Key von deinem KI-Anbieter
- Ein paar Minuten zum Einrichten

## Schnellstart

```bash
npm install -g openclaw@latest
openclaw onboard --install-daemon
openclaw gateway --port 18789
```

Öffne http://127.0.0.1:18789/ im Browser.
