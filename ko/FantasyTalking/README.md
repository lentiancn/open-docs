# FantasyTalking

## 개요

FantasyTalking은 음성 합성 및 대화 시스템을 위한 오픈소스 프로젝트입니다.

## 기능

- 고품질 음성 합성
- 다국어 지원
- 유연한 대화 시스템
- 크로스플랫폼

## 설치

```bash
npm install fantasy-talking
# 또는
pip install fantasy-talking
```
