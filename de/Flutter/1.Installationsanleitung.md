# Flutter Installationsanleitung

Flutter ist ein Open-Source-UI-Toolkit, das von Google entwickelt wurde, um schöne, schnelle plattformübergreifende Anwendungen zu erstellen. Diese Anleitung behandelt die Installationsmethoden in allen Umgebungen.

## Versionsinformationen

| Version | Status | Beschreibung |
|---------|--------|---------------|
| Flutter 3.x | ✅ Aktuell | Neueste stabile Version |
| Flutter 2.x | ⚠️ Wartung | Sicherheitsupdates |

> **Wichtig**: Diese Anleitung richtet sich hauptsächlich an **Flutter 3.x**.

## Installationsmethoden

### Methode 1: Installation unter Windows

#### 1. Flutter SDK herunterladen

1. Flutter SDK herunterladen: https://docs.flutter.dev/get-started/install/windows
2. In das gewünschte Verzeichnis entpacken (z.B. C:\src\flutter)

#### 2. Umgebungsvariablen konfigurieren

```powershell
# Zu PATH hinzufügen
$env:PATH += ";C:\src\flutter\bin"
```

#### 3. Flutter doctor ausführen

```powershell
flutter doctor
```

### Methode 2: Installation unter macOS

#### 1. Homebrew verwenden (empfohlen)

```bash
# Flutter installieren
brew install flutter

# Oder manuell herunterladen
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable
```

#### 2. Umgebungsvariablen konfigurieren

```bash
# ~/.bash_profile oder ~/.zshrc
export PATH="$PATH:`pwd`/flutter/bin"
```

#### 3. Flutter doctor ausführen

```bash
flutter doctor
```

### Methode 3: Installation unter Linux

#### 1. Abhängigkeiten installieren

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa clang cmake ninja-build pkg-config libgtk-3-dev

# Fedora
sudo dnf install -y curl git unzip xz zip mesa-libGLU clang cmake ninja-build pkgconfig gtk3-devel
```

#### 2. Flutter SDK herunterladen

```bash
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable
export PATH="$PATH:~/development/flutter/bin"
```

#### 3. Flutter doctor ausführen

```bash
flutter doctor
```

### Methode 4: VS Code verwenden

1. VS Code herunterladen und installieren
2. VS Code öffnen
3. Flutter-Erweiterungen installieren:
   - "Flutter" suchen und installieren
   - "Dart" suchen und installieren

### Methode 5: Android Studio verwenden

1. Android Studio herunterladen und installieren
2. Flutter-Plugin installieren:
   - Android Studio öffnen
   - Settings → Plugins
   - "Flutter" suchen und installieren

## Android-Konfiguration

### Android SDK installieren

```bash
# Mit Flutter-Tools installieren
flutter doctor --android-licenses
flutter config --android-sdk /path/to/android/sdk
```

### Umgebungsvariablen konfigurieren

```bash
# macOS/Linux
export ANDROID_SDK_ROOT=/Users/user/Library/Android/sdk
export PATH=$PATH:$ANDROID_SDK_ROOT/cmdline-tools/latest/bin:$ANDROID_SDK_ROOT/platform-tools
```

### Emulator erstellen

```bash
# Verfügbare Geräte auflisten
flutter devices

# Emulator erstellen
flutter emulator --create --name my-emulator

# Emulator starten
flutter emulators --launch my-emulator
```

## Häufige Probleme

### Flutter doctor Fehler?

Stellen Sie sicher, dass alle Abhängigkeiten installiert sind:

```bash
# flutter doctor ausführen
flutter doctor -v
```

### Android SDK nicht gefunden?

SDK-Pfad festlegen:

```bash
flutter config --android-sdk /path/to/android/sdk
```

### Build fehlgeschlagen?

Bereinigen und neu erstellen:

```bash
flutter clean
flutter pub get
flutter build apk
```
