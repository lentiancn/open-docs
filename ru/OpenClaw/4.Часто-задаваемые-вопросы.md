# Часто задаваемые вопросы о OpenClaw

Эта страница содержит наиболее распространённые вопросы и ответы об использовании OpenClaw.

---

## Основные вопросы

### Что такое OpenClaw?

OpenClaw — это шлюз ИИ-ассистента с самостоятельным хостингом, который соединяет несколько платформ чата, таких как WhatsApp, Telegram, Discord и iMessage, с ИИ-помощником по программированию (например, Pi).

### OpenClaw бесплатен?

OpenClaw сам по себе имеет открытый исходный код и бесплатен (лицензия MIT). Однако использование ИИ-моделей может потребовать оплаты.

### Какое устройство нужно для запуска OpenClaw?

Минимальные требования:
- Node.js 22+
- 2 ГБ ОЗУ
- 500 МБ места на диске

Рекомендуется:
- 4 ГБ+ ОЗУ
- Стабильное интернет-соединение
- Постоянно включённый компьютер или сервер

## Вопросы об установке

### Установка не удалась — что делать?

1. Проверьте версию Node.js:
   ```bash
   node --version
   ```

2. Используйте официальный скрипт установки:
   ```bash
   curl -fsSL https://openclaw.ai/install.sh | bash
   ```

3. Запустите диагностику:
   ```bash
   openclaw doctor
   ```

### Проблемы в Windows?

**Настоятельно рекомендуется WSL2.** OpenClaw лучше всего работает в среде Linux.

### Как обновить OpenClaw?

```bash
npm update -g openclaw@latest
openclaw doctor
```

## Вопросы о конфигурации

### Где находится файл конфигурации?

По умолчанию: `~/.openclaw/openclaw.json`

### Как настроить несколько каналов?

```json5
{
  channels: {
    whatsapp: { enabled: true },
    telegram: { enabled: true, botToken: "xxx" },
    discord: { enabled: true, token: "xxx" },
  },
}
```

### Конфигурация не применяется?

1. Проверьте формат файла (JSON5)
2. Перезапустите Gateway:
   ```bash
   openclaw gateway restart
   ```

## Вопросы о подключении

### Gateway не запускается?

Возможные причины:
- Порт 18789 уже используется
- Ошибка формата файла конфигурации
- Недостаточно прав

Решение:
```bash
lsof -i :18789
openclaw doctor
```

### Подключение к каналу не удалось?

1. Проверьте правильность учётных данных
2. Проверьте статус канала:
   ```bash
   openclaw channels status --probe
   ```
3. Проверьте логи:
   ```bash
   openclaw logs --follow
   ```

## Вопросы об использовании

### ИИ не отвечает на сообщения?

Проверьте:
1. Подключён ли канал?
2. Находитесь ли вы в белом списке?
3. В группах требуется ли упоминание @?
4. Проверьте логи на наличие ошибок

### Как сбросить сессию?

```bash
openclaw session reset
```

### Как изменить ИИ-модель?

```json5
{
  agents: {
    defaults: {
      model: {
        primary: "anthropic/claude-sonnet-4-5",
      },
    },
  },
}
```

## Вопросы о безопасности

### OpenClaw безопасен?

OpenClaw уделяет приоритет безопасности:
- Все данные хранятся локально
- Учётные данные шифруются
- Поддерживаются механизмы белого списка и связывания
- Можно включить изоляцию песочницы

Рекомендации:
- Используйте белый список для контроля доступа
- Регулярно обновляйте версии
- Запускайте `openclaw doctor` для проверки безопасности

### Что такое связывание (Pairing)?

Связывание — это механизм безопасности, при котором неизвестные пользователи блокируются при первой отправке сообщения. Им нужно ваше одобрение для использования ИИ.

Одобрить пользователя:
```bash
openclaw pairing approve --channel whatsapp --sender +15555550123
```

## Вопросы о производительности

### Работает медленно?

1. Проверьте системные ресурсы:
   ```bash
   openclaw status
   ```

2. Сожмите сессии:
   ```bash
   /compact
   ```

3. Очистите старые логи:
   ```bash
   openclaw logs clean
   ```
