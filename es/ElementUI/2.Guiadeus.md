# Element UI Guía de Uso

Esta guía cubre cómo usar los componentes de Element UI en su aplicación Vue 2.

## Uso básico

```vue
<template>
  <div>
    <el-button>Botón predeterminado</el-button>
    <el-button type="primary">Botón primario</el-button>
  </div>
</template>
```

## Componentes comunes

### Botón

```vue
<el-button type="primary">Primario</el-button>
<el-button type="success">Éxito</el-button>
<el-button type="warning">Advertencia</el-button>
<el-button type="danger">Peligro</el-button>
```

### Entrada

```vue
<el-input v-model="input" placeholder="Ingrese texto"></el-input>
```

### Tabla

```vue
<el-table :data="tableData">
  <el-table-column prop="name" label="Nombre"></el-table-column>
  <el-table-column prop="email" label="Email"></el-table-column>
</el-table>
```

### Formulario

```vue
<el-form :model="form" :rules="rules">
  <el-form-item label="Nombre" prop="name">
    <el-input v-model="form.name"></el-input>
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="submit">Enviar</el-button>
  </el-form-item>
</el-form>
```

### Diálogo

```vue
<el-button @click="visible = true">Abrir</el-button>
<el-dialog :visible.sync="visible" title="Título">
  <span>Contenido del diálogo</span>
</el-dialog>
```

## Mensajes

```javascript
this.$message.success('Éxito')
this.$message.error('Error')
this.$message.warning('Advertencia')
this.$message.info('Información')
```

## Slots

```vue
<el-dialog>
  <span slot="title">Título personalizado</span>
  <span>Contenido</span>
  <span slot="footer">
    <el-button>Cancelar</el-button>
    <el-button type="primary">Aceptar</el-button>
  </span>
</el-dialog>
```

## v-model

```vue
<el-input v-model="value"></el-input>
<el-checkbox v-model="checked">Etiqueta</el-checkbox>
<el-switch v-model="switchValue"></el-switch>
```

## Mejores prácticas

### Importación local

```javascript
export default {
  components: {
    [Button.name]: Button,
    [Input.name]: Input
  }
}
```

### Validación de formulario

```javascript
const validatePass = (rule, value, callback) => {
  if (value === '') {
    callback(new Error('Ingrese contraseña'))
  } else {
    callback()
  }
}
```
