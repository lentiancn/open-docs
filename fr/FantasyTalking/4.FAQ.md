# FantasyTalking FAQ

## Installation

### L'installation échoue, que faire?

D'abord, confirmez que l'environnement répond aux exigences. Vérifiez la version Node.js avec `node --version` (doit être >= 14.0). Si vous utilisez Python, assurez-vous que c'est >= 3.7.

Si l'installation continue de échouer, essayez: Vider le cache npm avec `npm cache clean --force`; Utiliser un miroir national; Ou installer depuis les sources.

### Problèmes FFmpeg

Sur certains systèmes, FFmpeg peut ne pas être correctement installé. Vous pouvez télécharger les binaires précompilés depuis le site officiel FFmpeg.

## Utilisation

### La synthèse vocale est lente?

La vitesse dépend de plusieurs facteurs. Vérifiez d'abord la connexion réseau. Envisagez l'auto-hébergement pour réduire la latence.

### Comment choisir le bon style vocal?

FantasyTalking propose plusieurs styles: neutral, happy, sad, angry, excited. Choisissez selon le scénario: Use neutral ou professional pour le service client.

### Comment implémenter la synthèse anglaise?

Spécifiez l'anglais dans la configuration:

```javascript
const audio = await client.speak('Hello, world!', {
  voice: { language: 'en-US', style: 'neutral' }
});
```

## Déploiement

### Le déploiement Docker échoue?

Confirmez d'abord que Docker est correctement installé et en cours d'exécution. Vérifiez si le port 8080 est utilisé.

## Autres

### Quelles plateformes supportées?

FantasyTalking supporte: Web (navigateurs modernes), Mobile (iOS/Android), Bureau (Windows/macOS/Linux), Serveur (Node.js/Python).
