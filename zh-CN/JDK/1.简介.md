# JDK 简介

> JDK (Java Development Kit) 是 Java 语言的软件开发工具包

---

## 概述

JDK 是 Java 程序员必备的开发环境，包含 JRE（Java 运行时环境）和开发工具。

---

## 主要组件

- **javac**：Java 编译器
- **java**：Java 解释器
- **jdb**：Java 调试器
- **jar**：Java 归档工具
- **javap**：Java 反汇编器
