# DragonTalker Guía de Instalación

DragonTalker es un proyecto AI para generar videos realistas de cabeza parlante desde una imagen y audio.

## Versiones

| Versión | Estado |
|---------|--------|
| DragonTalker 1.x | Actual |

## Requisitos del sistema

### Hardware

| Componente | Mínimo | Recomendado |
|------------|--------|-------------|
| GPU | NVIDIA 6GB VRAM | NVIDIA 16GB VRAM |
| RAM | 8GB | 32GB |
| Almacenamiento | 20GB | 50GB |
| CUDA | 11.7+ | 11.8+ |

### Software

| Software | Versión |
|----------|---------|
| Python | 3.8 - 3.10 |
| CUDA | 11.7+ |
| ffmpeg | Latest |

## Instalación Linux/Ubuntu

```bash
# Actualizar sistema
sudo apt update && sudo apt upgrade -y

# Instalar Python y herramientas
sudo apt install python3.9 python3-pip git wget curl

# Instalar librerías del sistema
sudo apt install libgl1-mesa-glx libglib2.0-0 libsm6 libxext6 libxrender-dev libgomp1

# Instalar ffmpeg
sudo apt install ffmpeg

# Clonar proyecto
git clone https://github.com/your-repo/DragonTalker.git
cd DragonTalker

# Crear entorno virtual
python3.9 -m venv venv
source venv/bin/activate

# Instalar PyTorch
pip install torch==2.0.1 torchvision==0.15.2 --index-url https://download.pytorch.org/whl/cu118

# Instalar dependencias
pip install -r requirements.txt
pip install numpy opencv-python librosa soundfile

# Descargar modelos
mkdir -p checkpoints
bash scripts/download_models.sh
```

## Verificar instalación

```bash
python inference.py --help
```

## Problemas comunes

### CUDA sin memoria

**Solución:**
- Reducir batch_size: `--batch_size 1`
- Usar menor resolución: `--size 256`

### ffmpeg no encontrado

**Solución:**
- Linux: `sudo apt install ffmpeg`
- Windows: Agregar ffmpeg al PATH
