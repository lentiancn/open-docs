# Vue 使用指南

## 基本概念

### Vue 實例

```javascript
import { createApp } from 'vue'

const app = createApp({
  data() {
    return {
      message: '你好 Vue!'
    }
  }
})

app.mount('#app')
```

### 模板語法

```html
<!-- 文本插值 -->
<h1>{{ message }}</h1>

<!-- 指令 -->
<div v-if="show">內容</div>
<div v-for="item in items">{{ item }}</div>
<button @click="handleClick">點擊</button>
<input v-model="inputValue">
```

## 響應式

### Vue 3 Composition API（推薦）

```javascript
import { ref, reactive, computed } from 'vue'

export default {
  setup() {
    const count = ref(0)
    const state = reactive({ name: 'Vue' })
    
    const doubleCount = computed(() => count.value * 2)
    
    return { count, state, doubleCount }
  }
}
```

## 組件

### 創建組件

```vue
<template>
  <div>
    <h2>{{ title }}</h2>
    <button @click="emitEvent">點擊</button>
  </div>
</template>

<script setup>
import { defineProps, defineEmits } from 'vue'

const props = defineProps({ title: String })
const emit = defineEmits(['click'])
</script>
```

## 條件渲染

```html
<div v-if="show">內容</div>
<div v-else>默認</div>
```

## 事件處理

```html
<button @click="handleClick">點擊</button>
<button @click.stop="handleClick">停止冒泡</button>
```
