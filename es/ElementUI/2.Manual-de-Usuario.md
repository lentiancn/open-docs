## 1. Componentes Básicos

### 1.1 Componente de Botón

Los botones son uno de los componentes más utilizados.

#### Uso Básico

```html
<el-button>Predeterminado</el-button>
<el-button type="primary">Primario</el-button>
<el-button type="success">Éxito</el-button>
<el-button type="info">Información</el-button>
<el-button type="warning">Advertencia</el-button>
<el-button type="danger">Peligro</el-button>
```

#### Atributos del Botón

| Atributo | Descripción | Tipo | Valores | Default |
|----------|-------------|------|---------|----------|
| type | Tipo de botón | string | primary/success/warning/danger/info/text | — |
| plain | Botón simple | boolean | — | false |
| round | Botón redondeado | boolean | — | false |
| disabled | Deshabilitado | boolean | — | false |

## 2. Componentes de Formulario

### 2.1 Campo de Entrada

```html
<el-input v-model="input" placeholder="Ingrese contenido"></el-input>
```

### 2.2 Selector

```html
<el-select v-model="selected" placeholder="Seleccione">
  <el-option label="Opción 1" value="1"></el-option>
  <el-option label="Opción 2" value="2"></el-option>
</el-select>
```

### 2.3 Validación de Formulario

```javascript
this.$refs.form.validate(valid => {
  if (valid) {
    console.log('Validación exitosa')
  }
})
```

## 3. Componente de Tabla

### 3.1 Tabla Básica

```html
<el-table :data="tableData" stripe border>
  <el-table-column prop="name" label="Nombre"></el-table-column>
  <el-table-column prop="email" label="Email"></el-table-column>
</el-table>
```

### 3.2 Tabla con Paginación

```html
<el-pagination
  @current-change="handlePageChange"
  :current-page="currentPage"
  :total="total">
</el-pagination>
```

## 4. Diálogos y Notificaciones

### 4.1 Diálogo

```html
<el-dialog title="Título" :visible.sync="visible">
  <span>Contenido del diálogo</span>
</el-dialog>
```

### 4.2 Mensajes

```javascript
this.$message.success('Operación exitosa')
this.$message.error('Operación fallida')
this.$message.warning('Advertencia')
```
