# Element UI 使用指南

高效使用 Element UI 的完整指南。

---

## 基础组件

### 按钮

```vue
<el-button>默认</el-button>
<el-button type="primary">主要</el-button>
<el-button type="success">成功</el-button>
<el-button type="danger">危险</el-button>
<el-button type="warning">警告</el-button>

<!-- 图标 -->
<el-button icon="el-icon-edit">编辑</el-button>
```

### 布局

```vue
<el-row>
  <el-col :span="24"><div>24列</div></el-col>
</el-row>

<el-row :gutter="20">
  <el-col :span="12"><div>12列</div></el-col>
  <el-col :span="12"><div>12列</div></el-col>
</el-row>
```

---

## 表单组件

### 输入框

```vue
<el-input v-model="input" placeholder="请输入"></el-input>

<!-- 带前缀图标 -->
<el-input prefix-icon="el-icon-search" v-model="input2"></el-input>

<!-- 文本域 -->
<el-input type="textarea" :rows="3" v-model="textarea"></el-input>
```

### 选择器

```vue
<el-select v-model="value" placeholder="请选择">
  <el-option
    v-for="item in options"
    :key="item.value"
    :label="item.label"
    :value="item.value">
  </el-option>
</el-select>
```

### 表单

```vue
<el-form :model="form" label-width="120px">
  <el-form-item label="活动名称">
    <el-input v-model="form.name"></el-input>
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="onSubmit">提交</el-button>
  </el-form-item>
</el-form>
```

## 数据展示

### 表格

```vue
<el-table :data="tableData" style="width: 100%">
  <el-table-column prop="date" label="日期" width="180"></el-table-column>
  <el-table-column prop="name" label="姓名" width="180"></el-table-column>
  <el-table-column prop="address" label="地址"></el-table-column>
</el-table>
```

### 标签

```vue
<el-tag>标签</el-tag>
<el-tag type="success">成功</el-tag>
<el-tag type="info">信息</el-tag>
<el-tag type="warning">警告</el-tag>
<el-tag type="danger">危险</el-tag>
```

### 进度条

```vue
<el-progress :percentage="50"></el-progress>
<el-progress :percentage="100" status="success"></el-progress>
```

## 导航

### 菜单

```vue
<el-menu default-active="1">
  <el-menu-item index="1">
    <i class="el-icon-menu"></i>
    <span>导航一</span>
  </el-menu-item>
</el-menu>
```

### 标签页

```vue
<el-tabs v-model="activeName">
  <el-tab-pane label="用户" name="first">用户</el-tab-pane>
  <el-tab-pane label="配置" name="second">配置</el-tab-pane>
</el-tabs>
```

## 反馈

### 提示

```vue
<el-alert title="成功提示" type="success"></el-alert>
<el-alert title="警告提示" type="warning"></el-alert>
<el-alert title="错误提示" type="error"></el-alert>
```

### 消息

```vue
this.$message('消息提示')
this.$message.success('成功')
this.$message.error('错误')
```

---

## 对话框

### 弹窗

```vue
<el-button type="text" @click="dialogVisible = true">打开弹窗</el-button>

<el-dialog title="标题" :visible.sync="dialogVisible" width="30%">
  <span>内容</span>
  <span slot="footer">
    <el-button @click="dialogVisible = false">取消</el-button>
    <el-button type="primary" @click="dialogVisible = false">确定</el-button>
  </span>
</el-dialog>
```
