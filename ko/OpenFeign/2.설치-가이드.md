# OpenFeign 설치 가이드

## 전제 조건

- Java Development Kit (JDK) 8 이상
- Maven 또는 Gradle 빌드 도구
- IDE (IntelliJ IDEA, Eclipse 또는 VS Code)

---

## Maven 사용

`pom.xml`에 OpenFeign 종속성 추가:

```xml
<dependencies>
    <!-- OpenFeign Core -->
    <dependency>
        <groupId>io.github.openfeign</groupId>
        <artifactId>feign-core</artifactId>
        <version>13.2.1</version>
    </dependency>
    
    <!-- OkHttp Client (선택 사항이지만 권장) -->
    <dependency>
        <groupId>io.github.openfeign</groupId>
        <artifactId>feign-okhttp</artifactId>
        <version>13.2.1</version>
    </dependency>
</dependencies>
```

---

## Gradle 사용

`build.gradle`에 종속성 추가:

```groovy
dependencies {
    implementation 'io.github.openfeign:feign-core:13.2.1'
    implementation 'io.github.openfeign:feign-okhttp:13.2.1'
}
```

---

## Spring Cloud 통합

Spring Boot 애플리케이션의 경우:

```xml
<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-openfeign</artifactId>
    <version>4.1.0</version>
</dependency>
```

Spring Boot 애플리케이션에서 Feign 클라이언트 활성화:

```java
@SpringBootApplication
@EnableFeignClients
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
```

---

## 설치 확인

간단한 Feign 클라이언트를 만들어 확인:

```java
@FeignClient(name = "example-service", url = "https://api.example.com")
public interface ExampleClient {
    
    @RequestMapping(method = RequestMethod.GET, value = "/users/{id}")
    User getUser(@PathVariable("id") Long id);
}
```
