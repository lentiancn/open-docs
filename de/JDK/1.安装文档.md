# JDK Installationsanleitung

JDK (Java Development Kit) ist das offizielle Java-Entwicklungskit. Diese Anleitung behandelt Installationsmethoden für verschiedene Umgebungen.

## Versionsinformationen

| Version | Status | Beschreibung |
|---------|--------|-------------|
| JDK 21 (LTS) | ✅ Aktuell | Neueste LTS-Version |
| JDK 17 (LTS) | ✅ Unterstützt | Vorherige LTS-Version |
| JDK 11 (LTS) | ✅ Unterstützt | Legacy-LTS-Version |
| JDK 8 | ✅ Unterstützt | Legacy-Version |
| JDK 22 | ✅ Neueste | Neueste Feature-Version |

### LTS vs. Feature-Versionen

- **LTS (Long Term Support)**: 2-3 Jahre Updates, empfohlen für Produktion
- **Feature-Versionen**: Alle 6 Monate neue Funktionen, kürzerer Support

## Installationsmethoden

### Methode 1: Oracle JDK

#### Windows

1. Herunterladen von: https://www.oracle.com/java/technologies/downloads/
2. Installationsprogramm (.exe) ausführen
3. Dem Assistenten folgen

#### macOS

```bash
# DMG von Oracle-Website herunterladen
# Oder Homebrew verwenden
brew install openjdk@21
```

#### Linux

```bash
# tar.gz herunterladen
wget https://download.oracle.com/java/21/latest/jdk-21_linux-x64_bin.tar.gz
tar -xzf jdk-21_linux-x64_bin.tar.gz
sudo mv jdk-21 /opt/jdk-21
```

### Methode 2: Eclipse Temurin (Adoptium)

#### Windows

1. Herunterladen von: https://adoptium.net/temurin/releases/
2. Installationsprogramm ausführen

#### macOS

```bash
# Homebrew verwenden
brew install --cask temurin

# Oder pkg von Website herunterladen
```

#### Linux

```bash
# Ubuntu/Debian
wget https://github.com/adoptium/temurin21-binaries/releases/download/jdk-21.0.2%2B13/OpenJDK21U-jdk_x64_linux_hotspot_21.0.2_13.tar.gz
tar -xzf OpenJDK21U-jdk_x64_linux_hotspot_21.0.2_13.tar.gz

# Umgebung setzen
export JAVA_HOME=/path/to/jdk-21
export PATH=$JAVA_HOME/bin:$PATH
```

### Methode 3: SDKMAN

```bash
# SDKMAN installieren
curl -s "https://get.sdkman.io" | bash

# JDK installieren
sdk install java 21.0.2-tem

# Verfügbare Versionen auflisten
sdk list java
```

### Methode 4: Docker

```dockerfile
FROM eclipse-temurin:21-jdk
WORKDIR /app
COPY . .
RUN javac Main.java
CMD ["java", "Main"]
```

### Methode 5: Windows-Paketmanager

#### Chocolatey

```powershell
choco install openjdk21
```

#### Scoop

```powershell
scoop install openjdk
```

## Umgebungskonfiguration

### Windows

```powershell
# JAVA_HOME setzen
[System.Environment]::SetEnvironmentVariable("JAVA_HOME", "C:\Program Files\Java\jdk-21", "User")

# Zu PATH hinzufügen
[System.Environment]::SetEnvironmentVariable("PATH", "$env:PATH;C:\Program Files\Java\jdk-21\bin", "User")
```

### macOS/Linux

```bash
# Zu ~/.bashrc oder ~/.zshrc hinzufügen
export JAVA_HOME=/opt/jdk-21
export PATH=$JAVA_HOME/bin:$PATH

# Anwenden
source ~/.bashrc
```

## Überprüfung

```bash
# Java-Version prüfen
java -version

# Javac-Version prüfen
javac -version

# JAVA_HOME prüfen
echo $JAVA_HOME
```

### SDKMAN verwenden

```bash
# Version wechseln
sdk use java 17.0.8-tem
```

## Häufige Probleme

### "java" nicht gefunden?

Überprüfen Sie, ob PATH das Java-bin-Verzeichnis enthält.

### Falsche Java-Version?

Prüfen Sie, welches java verwendet wird:

```bash
which java
java -version
```

### JAVA_HOME nicht gesetzt?

Setzen Sie JAVA_HOME auf das JDK-Installationsverzeichnis.
