# OpenClaw

**Puerta de Enlace AI Autohospedada**

Conecta WhatsApp, Telegram, Discord, iMessage a asistentes AI.

## Inicio

```bash
npm install -g openclaw@latest
openclaw onboard --install-daemon
openclaw gateway --port 18789
```

http://127.0.0.1:18789/
