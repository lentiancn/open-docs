# OpenClaw

[English](./README.md) | [简体中文](./README.md) | [日本語](./README.md)

🦞 Gateway de asistente IA para cualquier OS. Soporta WhatsApp, Telegram, Discord, iMessage.

## Características

- 🤖 Conecta apps de chat con asistentes IA
- 📱 Multi-plataforma
- 🔒 Local/tú controlas tus datos
- 🧠 Múltiples agentes
- 🌐 Interfaz web

## Inicio rápido

```bash
npm install -g openclaw@latest
openclaw onboard --install-daemon
openclaw gateway --port 18789
```

Abre http://127.0.0.1:18789/
