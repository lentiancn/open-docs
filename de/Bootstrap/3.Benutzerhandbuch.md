## Grid-System

Das Grid-System ist das Herzstück von Bootstrap und bietet ein 12-Spalten-Layout.

### Grundlegende Nutzung

```html
<div class="container">
    <div class="row">
        <div class="col">Spalte 1</div>
        <div class="col">Spalte 2</div>
        <div class="col">Spalte 3</div>
    </div>
</div>
```

### Responsive Breakpoints

| Klasse | Bildschirmbreite | Beschreibung |
|--------|------------------|--------------|
| `.col-xs` | < 576px | Extra kleine Bildschirme |
| `.col-sm` | ≥ 576px | Kleine Bildschirme |
| `.col-md` | ≥ 768px | Mittlere Bildschirme |
| `.col-lg` | ≥ 992px | Große Bildschirme |
| `.col-xl` | ≥ 1200px | Extra große Bildschirme |
| `.col-xxl` | ≥ 1400px | Extra extra große Bildschirme |

### Spalten versetzen

```html
<div class="row">
    <div class="col-md-4">.col-md-4</div>
    <div class="col-md-4 offset-md-4">.col-md-4 .offset-md-4</div>
</div>
```

### Responsive Spalten

```html
<div class="row">
    <div class="col-12 col-md-6 col-lg-4">
        Volle Breite auf Handy, halb auf Tablet, ein Drittel auf großem Bildschirm
    </div>
</div>
```

## Best Practices

1. **Semantisches HTML nutzen** – Verwende Tags wie header, nav, main, footer sinnvoll
2. **Bleib einfach** – Vermeide zu starke Verschachtelung
3. **Responsive zuerst** – Entwerfe zuerst für mobile Geräte
4. **Hilfsklassen nutzen** – Schnell Stile anpassen
5. **Eigene Themes** – Nutze Sass-Variablen für Konsistenz
