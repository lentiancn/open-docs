# Angular Guía de instalación

Guía detallada para configurar el entorno de Angular.

## Requisitos

### Node.js

- Recomendado: Node.js 18+
- Requiere npm o yarn

### Verificar versión de Node.js

```bash
node --version
npm --version
```

## Instalar Angular CLI

### Instalar

```bash
npm install -g @angular/cli
```

### Verificar instalación

```bash
ng version
```

### Actualizar

```bash
npm install -g @angular/cli@latest
```

## Crear nuevo proyecto

### Usando Angular CLI

```bash
ng new my-app
cd my-app
```

### Opciones interactivas

- ¿Añadir enrutamiento? Sí/No
- ¿Qué formato de hoja de estilos? (SCSS/CSS/Sass/Less)

## Ejecutar aplicación

### Servidor de desarrollo

```bash
ng serve
# o
npm start
```

Visitar: http://localhost:4200

### Recarga en caliente

Angular CLI soporta recarga en caliente - cambios actualizan automáticamente.

## Compilar para producción

```bash
ng build --configuration production
```

Salida en directorio `dist/`.

## Comandos comunes

| Comando | Función |
|---------|----------|
| ng new | Crear nuevo proyecto |
| ng serve | Iniciar servidor dev |
| ng build | Compilar proyecto |
| ng generate | Generar componente/servicio/módulo |
| ng test | Ejecutar pruebas unitarias |
| ng e2e | Ejecutar pruebas e2e |

## Estructura del proyecto

```
my-app/
├── src/
│   ├── app/
│   │   ├── components/
│   │   ├── services/
│   │   ├── app.component.ts
│   │   └── app.module.ts
│   ├── assets/
│   └── styles.css
├── angular.json
└── package.json
```

## Desinstalar Angular CLI

```bash
npm uninstall -g @angular/cli
```
