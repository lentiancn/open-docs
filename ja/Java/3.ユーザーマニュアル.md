# Java ユーザーマニュアル

## 最初のJavaプログラム

### Hello Worldの作成

HelloWorld.javaという名前の新しいテキストファイルを作成します。ファイル名はクラス名と完全に一致する必要があります。

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

### プログラムのコンパイル

ターミナルまたはコマンドプロンプトを開き、ファイルがあるディレクトリに移動して、コンパイルコマンドを実行します：

```bash
javac HelloWorld.java
```

コンパイルが成功すると、HelloWorld.classファイルが生成されます。

### プログラムの実行

```bash
java HelloWorld
```

出力結果：

```
Hello, World!
```

## Javaの基本構文

### 変数とデータ型

Javaは静的型付け言語で、変数は使用前に宣言する必要があります。

```java
// 基本データ型
int age = 25;
double salary = 5000.50;
boolean isActive = true;
char grade = 'A';
String name = "田中";

// 参照型
Integer count = 10;
Double price = 99.99;
```

### 演算子

```java
// 算術演算子
int a = 10, b = 3;
int sum = a + b;      // 加算
int diff = a - b;     // 減算
int prod = a * b;     // 乗算
int quot = a / b;     // 割算
int rem = a % b;      // 餘り

// 比較演算子
boolean result = (a > b);

// 論理演算子
boolean flag = (a > 5) && (b < 10);
```

### 条件文

```java
// if-else文
if (score >= 90) {
    System.out.println("優秀");
} else if (score >= 60) {
    System.out.println("合格");
} else {
    System.out.println("不合格");
}

// switch文
switch (day) {
    case 1:
        System.out.println("月曜日");
        break;
    case 2:
        System.out.println("火曜日");
        break;
    default:
        System.out.println("不明");
}
```

### 反復文

```java
// for文
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}

// while文
int i = 0;
while (i < 10) {
    System.out.println(i);
    i++;
}

// do-while文
do {
    System.out.println(i);
    i++;
} while (i < 10);

// for-eachで配列を反復
int[] numbers = {1, 2, 3, 4, 5};
for (int num : numbers) {
    System.out.println(num);
}
```

## オブジェクト指向プログラミング

### クラスとオブジェクト

```java
// クラスの定義
public class Person {
    // フィールド
    private String name;
    private int age;
    
    // コンストラクタ
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // GetterとSetter
    public String getName() {
        return name;
    }
    
    public void setName(String name) {
        this.name = name;
    }
    
    // メソッド
    public void sayHello() {
        System.out.println("こんにちは、" + name + "です");
    }
}

// オブジェクトの作成
Person person = new Person("田中", 25);
person.sayHello();
```

## Stream API

```java
import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);

// フィルタ
List<Integer> evenNumbers = numbers.stream()
    .filter(n -> n % 2 == 0)
    .collect(Collectors.toList());

// マッピング
List<Integer> squared = numbers.stream()
    .map(n -> n * n)
    .collect(Collectors.toList());

// 合計
int sum = numbers.stream()
    .mapToInt(Integer::intValue)
    .sum();
```

## マルチスレッドプログラミング

### スレッドの作成

```java
// Threadクラスを継承
class MyThread extends Thread {
    @Override
    public void run() {
        System.out.println("スレッド実行中");
    }
}

// Runnableインターフェースを実装
class MyRunnable implements Runnable {
    @Override
    public void run() {
        System.out.println("スレッド実行中");
    }
}

// スレッドの開始
MyThread thread = new MyThread();
thread.start();

// Runnableを使用
Thread runnableThread = new Thread(new MyRunnable());
runnableThread.start();
```

## ネットワークプログラミング

### TCP通信

```java
import java.net.*;
import java.io.*;

ServerSocket serverSocket = new ServerSocket(8080);
Socket clientSocket = serverSocket.accept();

    new InputStreamReader(clientSocket.getInputStream()));
```

## データベース操作

### JDBC接続

```java
import java.sql.*;

String url = "jdbc:mysql://localhost:3306/mydb";
String user = "root";
String password = "password";

try (Connection conn = DriverManager.getConnection(url, user, password)) {
    String sql = "SELECT * FROM users WHERE id = ?";
    PreparedStatement stmt = conn.prepareStatement(sql);
    stmt.setInt(1, 1);
    
    ResultSet rs = stmt.executeQuery();
    while (rs.next()) {
        System.out.println(rs.getString("name"));
    }
} catch (SQLException e) {
    e.printStackTrace();
}
```
