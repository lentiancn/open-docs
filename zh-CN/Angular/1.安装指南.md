## 简介

### 什么是 Angular？

Angular 是由 Google 开发维护的 TypeScript 开源 Web 应用框架，用于构建现代单页应用程序（SPA）。

### Angular 版本

| 版本 | 发布日期 | 状态 |
|------|----------|------|
| Angular 19 | 2024年 | 当前稳定版 |
| Angular 18 | 2024年 | 支持 |
| Angular 17 | 2023年 | 支持 |
| Angular 16 | 2023年 | 支持 |

### Angular 19 新特性

- 改进的响应式编程
- 更快的编译速度
- 更好的 SSR 支持
- 改进的信号（Signals）API
- 更小的 bundle 体积

## 安装方法

### 方法一：使用 Angular CLI（推荐）

Angular CLI 是官方推荐的安装方式。

```bash
# 安装 Angular CLI
npm install -g @angular/cli

# 创建新项目
ng new my-app

# 启动开发服务器
cd my-app
ng serve
```

### 方法二：使用 npm 本地安装

```bash
# 创建项目目录
mkdir angular-project
cd angular-project

# 初始化
npm init -y

# 安装 Angular
npm install @angular/core @angular/common @angular/platform-browser
```

## 在 Windows 上安装

### 步骤一：安装 Node.js

1. 访问 https://nodejs.org/
2. 下载 LTS 版本
3. 运行安装程序

### 步骤二：安装 Angular CLI

```cmd
npm install -g @angular/cli
```

### 步骤三：创建项目

```cmd
ng new my-angular-app
cd my-angular-app
ng serve
```

访问 http://localhost:4200

## 在 macOS 上安装

### 使用 Homebrew（推荐）

```bash
# 安装 Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装 Node.js
brew install node@18

# 安装 Angular CLI
npm install -g @angular/cli

# 创建项目
ng new my-angular-app
cd my-angular-app
ng serve
```

## 在 Linux 上安装

### Ubuntu/Debian

```bash
# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装 Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs

# 安装 Angular CLI
sudo npm install -g @angular/cli

# 创建项目
ng new my-angular-app
cd my-angular-app
ng serve
```

### Fedora

```bash
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -
sudo dnf install nodejs
sudo npm install -g @angular/cli
```

## 验证安装

```bash
# 检查版本
ng version

# 创建测试项目
ng new test-app
cd test-app
ng serve
```

如果看到 "Application bundle generation complete"，说明安装成功。

## 常见问题

### 1. npm 权限错误

```bash
# 使用 nvm 安装 Node.js
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 18
nvm use 18
```

### 2. 端口被占用

```bash
# 使用其他端口
ng serve --port 4201
```

### 3. Node.js 版本不匹配

| Angular 版本 | Node.js 版本 |
|-------------|-------------|
| Angular 19 | 18.19+, 20.x |
| Angular 18 | 18.19+, 20.x |
| Angular 17 | 18.x, 20.x |
