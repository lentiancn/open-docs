# Bootstrap 5 Guía de Instalación

Esta guía detalla cómo instalar y configurar Bootstrap 5 en varios entornos.

## Información de Versión

| Versión | Estado | Descripción |
|---------|--------|-------------|
| Bootstrap 5.3.x | ✅ Actual | Última versión con todas las funciones nuevas |
| Bootstrap 5.2.x | ✅ Soportado | Versión estable |
| Bootstrap 5.1.x | ⚠️ Mantenimiento | Solo actualizaciones de seguridad |
| Bootstrap 4.x | ❌ Fin de vida | No recomendado para nuevos proyectos |

> **Importante**: Esta guía está orientada a **Bootstrap 5.3.x**. Bootstrap 4 ha terminado su mantenimiento y no se recomienda para nuevos proyectos.

## Métodos de Instalación

### Método 1: CDN (Más Simple)

La forma más rápida es incluir los archivos CSS y JavaScript de Bootstrap a través de CDN.

#### Solo CSS

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
```

#### CSS + JavaScript

```html
<!-- Bootstrap CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- Bootstrap JS (incluye Popper) -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
```

**Proveedores de CDN**:
- [jsDelivr](https://www.jsdelivr.com/) - Recomendado
- [cdnjs](https://cdnjs.com/libraries/bootstrap)
- [unpkg](https://unpkg.com/browse/bootstrap/)

#### Plantilla Completa

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Página Bootstrap 5</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>¡Hola, Bootstrap 5!</h1>
    
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### Método 2: npm

#### Usando npm

```bash
# Crear nuevo proyecto
npm init -y

# Instalar Bootstrap
npm install bootstrap@5.3.3

# Instalar Popper.js (si es necesario)
npm install @popperjs/core
```

#### Importar en el Proyecto

```javascript
// main.js o app.js
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

### Método 3: Vite

#### 1. Crear Proyecto Vite

```bash
npm create vite@latest my-bootstrap-app -- --template vanilla
cd my-bootstrap-app
npm install
```

#### 2. Instalar Bootstrap

```bash
npm install bootstrap@5.3.3
```

#### 3. Importar Bootstrap

```javascript
// main.js
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

```javascript
// vite.config.js
export default {
  plugins: [],
  resolve: {
    alias: {
      '@': '/src'
    }
  }
}
```

### Método 4: Webpack

#### 1. Instalar Dependencias

```bash
npm install bootstrap@5.3.3
npm install --save-dev webpack webpack-cli css-loader style-loader
```

#### 2. Configurar Webpack

```javascript
// webpack.config.js
module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js'
  },
  module: {
    rules: [
      {
        test: /\.css$/,
        use: ['style-loader', 'css-loader']
      }
    ]
  }
};
```

#### 3. Importar Bootstrap

```javascript
// src/index.js
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

### Método 5: Vue CLI / Nuxt

#### Vue CLI

```bash
# Crear Proyecto Vue
npm install -g @vue/cli
vue create my-app

# Instalar Bootstrap
cd my-app
npm install bootstrap@5.3.3

# Importar en main.js
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

#### Nuxt 3

```bash
# Crear Proyecto Nuxt
npx nuxi@latest init my-nuxt-app
cd my-nuxt-app
npm install

# Instalar Bootstrap
npm install bootstrap@5.3.3

# Configurar en nuxt.config.ts
export default defineNuxtConfig({
  css: ['bootstrap/dist/css/bootstrap.min.css'],
  build: {
    transpile: ['bootstrap']
  }
})
```

### Método 7: Descargar Código Fuente

#### Desde GitHub

```bash
# Clonar repositorio
git clone https://github.com/twbs/bootstrap.git
cd bootstrap

# Instalar dependencias
npm install

# Construir
npm run dist
```

#### Descarga Directa

1. Visite [Página de Lanzamiento de Bootstrap](https://github.com/twbs/bootstrap/releases)
2. Descargue Source code (zip)
3. Después de extraer, importe `dist/css/bootstrap.min.css` y `dist/js/bootstrap.bundle.min.js`

## Configuración Especial

### Habilitar RTL (Derecha a Izquierda)

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.rtl.min.css" rel="stylesheet">
```

### Personalizar con Sass

```bash
npm install bootstrap@5.3.3
npm install --save-dev sass
```

```scss
// custom.scss
// Importar funciones de Bootstrap
@import "bootstrap/scss/functions";

// Variables personalizadas
$primary: #ff6b35;
$font-family-sans-serif: "Inter", sans-serif;

// Importar otras partes de Bootstrap
@import "bootstrap/scss/variables";
@import "bootstrap/scss/mixins";
@import "bootstrap/scss/utilities";
@import "bootstrap/scss/root";
@import "bootstrap/scss/reboot";
@import "bootstrap/scss/type";
@import "bootstrap/scss/containers";
@import "bootstrap/scss/grid";
@import "bootstrap/scss/utilities/api";
// Agregar otros componentes necesarios...
```

```javascript
// vite.config.js
export default {
  css: {
    preprocessorOptions: {
      scss: {
        additionalData: `@import "bootstrap/scss/functions"; @import "custom-variables";`
      }
    }
  }
}
```

## Preguntas Frecuentes

### ¿Qué es Popper?

Popper es un motor de posicionamiento utilizado para posicionar tooltips y popovers. `bootstrap.bundle.min.js` ya incluye Popper, no necesita instalarlo por separado.

### ¿Necesito jQuery?

**¡No!** Bootstrap 5 está completamente reescrito en JavaScript nativo, sin dependencia de jQuery.

### ¿Cómo usar iconos?

Bootstrap 5 usa Bootstrap Icons:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css">

<i class="bi bi-github"></i>
<i class="bi bi-bootstrap-fill"></i>
```

### ¿Cómo cambiar colores del tema?

Puede sobrescribir con variables CSS:

```css
:root {
  --bs-primary: #ff6b35;
  --bs-secondary: #6c757d;
  --bs-success: #198754;
}
```
