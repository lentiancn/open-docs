# Bootstrap 概要

世界で最も人気のフロントエンド CSS フレームワーク。

## Bootstrapとは？

Bootstrapは、Twitter開発した最も人気のフロントエンドCSSフレームワークで、レスポンシブでモバイルファーストなWebサイトを構築するために使用されます。

### 主な機能

- **レスポンシブグリッドシステム** - 12列レスポンシブグリッド
- **プリビルドコンポーネント** - ナビゲーションバー、カード、モーダルなど
- **CSSユーティリティクラス** - クイックスタイル調整
- **JavaScriptコンポーネント** - インタラクティブコンポーネント

### 統計

- GitHub スター：160,000+
- Webサイト使用数：数百万
