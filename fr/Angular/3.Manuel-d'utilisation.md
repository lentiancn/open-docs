# Manuel d'utilisation

Ce manuel détaille les fonctionnalités principales d'Angular et leur utilisation, incluant les composants, services, routage, directives, pipes et formulaires.

## Développement de composants

### Créer un composant

```bash
ng generate component user-list
```

Ou création manuelle :

```typescript
import { Component } from '@angular/core';

@Component({
  selector: 'app-user-list',
  templateUrl: './user-list.component.html',
  styleUrls: ['./user-list.component.scss']
})
export class UserListComponent {
  users = ['Jean', 'Pierre', 'Marie'];
}
```

### Syntaxe des modèles

**Interpolation :**

```html
<h1>Bonjour, {{ name }}</h1>
```

**Liaison de propriétés :**

```html
<img [src]="imageUrl">
<button [disabled]="isDisabled">Cliquer</button>
```

**Liaison d'événements :**


## Services et injection de dépendances

### Créer un service

```bash
ng generate service user
```

Code du service :

```typescript
import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class UserService {
  constructor(private http: HttpClient) {}

  getUsers(): Observable<User[]> {
    return this.http.get<User[]>('/api/users');
  }
}
```

### Utiliser un service

```typescript
import { Component, OnInit } from '@angular/core';
import { UserService } from './user.service';

@Component({
  selector: 'app-user-list',
  templateUrl: './user-list.component.html'
})
export class UserListComponent implements OnInit {
  users: User[] = [];

  constructor(private userService: UserService) {}

  ngOnInit() {
    this.userService.getUsers().subscribe(data => {
      this.users = data;
    });
  }
}
```

## Routage

### Configurer les routes

```typescript
import { Routes } from '@angular/router';

const routes: Routes = [
  { path: '', redirectTo: '/home', pathMatch: 'full' },
  { path: 'home', component: HomeComponent },
  { path: 'users', component: UserListComponent },
  { path: 'user/:id', component: UserDetailComponent },
  { path: '**', component: NotFoundComponent }
```

### Pipes intégrés

```html

<!-- Pipe monnaie -->

<!-- Pipe décimal -->

<!-- Pipe JSON -->

```

### Pipe personnalisé

```bash
ng generate pipe capitalize
```

```typescript
import { Pipe, PipeTransform } from '@angular/core';

@Pipe({
  name: 'capitalize'
})
export class CapitalizePipe implements PipeTransform {
  transform(value: string): string {
    if (!value) return value;
    return value.charAt(0).toUpperCase() + value.slice(1);
  }
}
```

## Gestion des formulaires

### Formulaires réactifs

```typescript
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

export class LoginComponent implements OnInit {
  loginForm: FormGroup;

  constructor(private fb: FormBuilder) {}

  ngOnInit() {
    this.loginForm = this.fb.group({
      username: ['', Validators.required],
      password: ['', [Validators.required, Validators.minLength(6)]],
      remember: [false]
    });
  }

  onSubmit() {
    if (this.loginForm.valid) {
      console.log(this.loginForm.value);
    }
  }
}
```

Modèle :

```html
<form [formGroup]="loginForm" (ngSubmit)="onSubmit()">
  <input formControlName="username" placeholder="Nom d'utilisateur">
  <div *ngIf="loginForm.get('username').touched && loginForm.get('username').errors?.required">
    Le nom d'utilisateur est requis
  </div>
  
  <input type="password" formControlName="password" placeholder="Mot de passe">
  
  <label>
    <input type="checkbox" formControlName="remember"> Se souvenir de moi
  </label>
  
  <button type="submit" [disabled]="loginForm.invalid">Connexion</button>
</form>
```

### Formulaires pilotés par le modèle

```html
<form #form="ngForm" (ngSubmit)="onSubmit(form)">
  <input name="username" [(ngModel)]="username" required #name="ngModel" placeholder="Nom d'utilisateur">
  <div *ngIf="name.touched && name.errors?.required">Le nom d'utilisateur est requis</div>
  
  <button type="submit" [disabled]="form.invalid">Soumettre</button>
</form>
```

## Client HTTP

### Configuration

Importer dans app.module.ts :

```typescript
import { HttpClientModule } from '@angular/common/http';

@NgModule({
  imports: [HttpClientModule]
})
export class AppModule {}
```

### Envoyer des requêtes

```typescript
import { HttpClient } from '@angular/common/http';

constructor(private http: HttpClient) {}

getData() {
  this.http.get('/api/data').subscribe(data => {
    console.log(data);
  });
}

postData() {
  this.http.post('/api/data', { name: 'test' }).subscribe();
}
```

### Intercepteurs

```typescript
import { HttpInterceptorFn } from '@angular/common/http';

export const authInterceptor: HttpInterceptorFn = (req, next) => {
  const token = localStorage.getItem('token');
  const cloned = req.clone({
    setHeaders: { Authorization: `Bearer ${token}` }
  });
  return next(cloned);
};
```

## Bonnes pratiques

1. **Utiliser le mode strict TypeScript** : activer les options de compilation strict
2. **Des composants petits et focalisés** : chaque composant ne fait qu'une chose
3. **Utiliser la stratégie OnPush** : améliorer les performances de détection des changements
4. **Utiliser les pipes de manière judicieuse** : éviter les calculs complexes dans les pipes
5. **Utiliser le chargement paresseux** : réduire le temps de chargement initial
6. **Écrire des tests** : garantir la qualité du code
