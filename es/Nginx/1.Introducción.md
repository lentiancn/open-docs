# Introducción a Nginx

Nginx es un servidor web HTTP de alto rendimiento, proxy inverso, balanceador de carga y proxy de correo.

## Características principales
- Servidor web HTTP
- Proxy inverso
- Balanceador de carga
- Caché
- SSL/TLS

## Ventajas
- Alto rendimiento
- Bajo consumo de recursos
- Alta estabilidad
- Configuración simple
