# OpenCode Resumen

Agente de codificación AI de código abierto.

## ¿Qué es OpenCode?

OpenCode es un agente de codificación AI de código abierto disponible como interfaz de terminal, aplicación de escritorio o extensión de IDE.

### Características principales

- **LSP habilitado** - Carga automáticamente los LSP correctos
- **Multisesión** - Inicia múltiples agentes en paralelo en el mismo proyecto
- **Compartir enlaces** - Comparte cualquier enlace de sesión
- **GitHub Copilot** - Inicia sesión con GitHub para usar Copilot
- **ChatGPT Plus/Pro** - Inicia sesión con OpenAI para usar ChatGPT
- **Cualquier modelo** - 75+ proveedores LLM incluyendo modelos locales
- **Cualquier editor** - Terminal, aplicación de escritorio y extensión IDE
- **Privacidad primero** - No almacena código ni datos de contexto
