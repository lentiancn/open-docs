# Flutter 简介

> 当前版本：Flutter 3.41.2 | 更新日期：2026-01-28

---

## 什么是 Flutter？

Flutter 是 Google 开发的**开源 UI 工具包**，用于从单一代码库构建美观、原生编译的多平台应用。

### 核心定位

| 特性 | 说明 |
|------|------|
| 跨平台 | 一套代码，运行 iOS、Android、Web、Windows、macOS、Linux |
| 原生性能 | 编译为原生 ARM 代码，无 JS 桥接 |
| 现代框架 | 采用 React 风格的响应式框架 |

---

## 为什么选择 Flutter？

### 单一代码库

```dart
// 一套代码，多端运行
void main() => runApp(MyApp());
```

### 高效开发

- **热重载**：修改代码后毫秒级看到效果
- **丰富工具**：Flutter DevTools 调试分析
- **强大生态**：pub.dev 超 35,000 个包

### 精美 UI

- Material Design 3（Google 原生设计）
- Cupertino Widgets（iOS 风格）
- 完全自定义每一像素

### 高性能

- Skia 2D 渲染引擎
- Impeller（iOS 默认渲染引擎）
- 60fps 流畅动画

## 核心概念

### Widget（组件）

Flutter 中**一切皆为 Widget**：

```dart
void main() {
  runApp(
    const Center(
      child: Text(
        'Hello, Flutter!',
        textDirection: TextDirection.ltr,
      ),
    ),
  );
}
```

### 两种 Widget 类型

| 类型 | 说明 | 使用场景 |
|------|------|----------|
| StatelessWidget | 无状态 | 静态内容 |
| StatefulWidget | 有状态 | 需要更新 |

```dart
// 无状态
class HelloWidget extends StatelessWidget {
  const HelloWidget({super.key});
  @override
  Widget build(BuildContext context) => const Text('Hello');
}

// 有状态
class Counter extends StatefulWidget {
  const Counter({super.key});
  @override
  State<Counter> createState() => _CounterState();
}

class _CounterState extends State<Counter> {
  int _count = 0;
  void _increment() => setState(() => _count++);

  @override
  Widget build(BuildContext context) => Column(
    mainAxisSize: MainAxisSize.min,
    children: [
      Text('$_count'),
      ElevatedButton(onPressed: _increment, child: const Text('增加')),
    ],
  );
}
```
