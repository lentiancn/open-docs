# Introduction

## What is MySQL

MySQL is the world's most popular open source database. With its proven performance, reliability and ease-of-use, MySQL has become the leading database choice for web-based applications, used by high profile web properties including Facebook, Twitter, YouTube, Yahoo! and many more.

Oracle drives MySQL innovation, delivering new capabilities to power next generation web, cloud, mobile and embedded applications.

## Key Features of MySQL

### Open Source and Free
MySQL Community Edition is the free, open-source distribution of MySQL, licensed under the GNU General Public License (GPL). It is supported by a large and active community of open-source developers.

### Cross-Platform Support
MySQL runs on various operating systems including Windows, Linux, macOS, FreeBSD, Solaris, and more. It can operate on different hardware configurations and software environments.

### High Performance
MySQL uses an optimized storage engine architecture and supports multi-threaded execution, capable of handling large-scale data and high-concurrency requests.

### Reliability
MySQL provides complete transaction support (ACID compliant), including commit, rollback, crash recovery, and row-level locking, ensuring data integrity.

### Ease of Use
MySQL offers simple management tools and comprehensive documentation with a gentle learning curve, suitable for developers of all skill levels.

## MySQL Product Editions

### MySQL Community Edition
The free, open-source edition suitable for development and production environments, supported by an active open-source community.

### MySQL Enterprise Edition
The enterprise-grade edition with advanced features, management tools, and technical support for business-critical applications.

### MySQL HeatWave
A cloud service with integrated generative AI and machine learning capabilities, supporting both transaction processing and lakehouse-scale analytics.

### MySQL NDB Cluster
A real-time open source transactional database designed for high availability and fast, always-on access to data under high throughput conditions.

## MySQL Application Areas

MySQL is widely used in:

- Website and Web Application Backends
- E-commerce Platforms
- Data Warehousing and Analytics
- Logging Systems
- Content Management Systems
- Mobile Application Backends
- Embedded Systems

## MySQL Ecosystem

The MySQL ecosystem includes a rich set of tools and products:

- **MySQL Server**: The core database server
- **MySQL Workbench**: Visual database design and management tool
- **MySQL Shell**: Command-line interactive tool
- **MySQL Router**: Database routing middleware
- **MySQL Connectors**: Database drivers for various programming languages

## Licensing

MySQL Community Edition is open source and free under the GPL license. For integrating MySQL into proprietary applications, you can choose MySQL Enterprise Edition commercial licensing.
