# Nginx Installationsanleitung

Anleitung zur Installation von Nginx auf allen wichtigen Plattformen.

---

## Linux

### Ubuntu / Debian

```bash
# Pakete aktualisieren
sudo apt update

# Nginx installieren
sudo apt install nginx

# Nginx starten
sudo systemctl start nginx

# Beim Boot starten
sudo systemctl enable nginx
```

### CentOS / RHEL

```bash
# EPEL installieren
sudo yum install epel-release

# Nginx installieren
sudo yum install nginx

# Starten
sudo systemctl start nginx
```

---

## macOS

```bash
brew install nginx
brew services start nginx
```

---

## Windows

1. Von [nginx.org](http://nginx.org/en/download.html) herunterladen
2. Entpacken nach `C:\nginx`
3. Ausführen: `nginx.exe`

---

## Überprüfen

```bash
nginx -v
nginx -t
curl http://localhost
```
