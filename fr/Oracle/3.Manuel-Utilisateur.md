# Oracle Manuel Utilisateur

## Connexion à la Base de Données

### Utiliser SQL*Plus

```bash
# Connecter comme SYSDBA
sqlplus / as sysdba

# Avec utilisateur et mot de passe
sqlplus utilisateur/motdepasse@database
```

---

## Opérations SQL de Base

### Créer une Table

```sql
CREATE TABLE employes (
    employe_id NUMBER(10) PRIMARY KEY,
    nom VARCHAR2(50) NOT NULL,
    prenom VARCHAR2(50) NOT NULL,
    email VARCHAR2(100),
    salaire NUMBER(10, 2)
);
```

### Insérer des Données

```sql
INSERT INTO employes (employe_id, nom, prenom, email, salaire)
VALUES (1001, 'Dupont', 'Jean', 'jean@company.com', 75000);
```

### Requêter des Données

```sql
SELECT * FROM employes;
SELECT nom, prenom, salaire FROM employes WHERE departement_id = 10;
```

## PL/SQL

### Procédure Stockée

```sql
CREATE OR REPLACE PROCEDURE mettre_a_jour_salaire (
    p_employe_id IN NUMBER,
    p_nouveau_salaire IN NUMBER
) AS
BEGIN
    UPDATE employes
    SET salaire = p_nouveau_salaire
    WHERE employe_id = p_employe_id;
    COMMIT;
END;
/
```
