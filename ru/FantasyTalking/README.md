# FantasyTalking

FantasyTalking - Система синтеза речи с открытым исходным кодом

## Обзор

FantasyTalking — проект с открытым исходным кодом для синтеза речи и диалоговых систем.

## Функции

- **Высококачественный синтез речи**: TTS технология на основе глубокого обучения
- **Поддержка нескольких языков**: Китайский, английский, японский и другие
- **Гибкая диалоговая система**: Поддержка многоходовых разговоров
- **Кроссплатформенность**: Веб, мобильные и десктоп
- **Простая интеграция**: Богатые SDK и API
- **Бесплатно с открытым кодом**: Лицензия MIT

## Быстрый старт

### Установка

```bash
npm install fantasy-talking
# или
pip install fantasy-talking
```

### Использование

```javascript
const client = new FantasyTalking({ apiKey: 'your-api-key' });
const audio = await client.speak('Привет');
```
