# 使用手冊

本手冊詳細介紹 Bootstrap 5 的核心功能和使用方法，包括網格系統、元件、CSS 工具類別等。

## 網格系統

網格系統是 Bootstrap 的核心功能，提供 12 欄布局。

### 基本用法

```html
<div class="container">
    <div class="row">
        <div class="col">欄 1</div>
        <div class="col">欄 2</div>
        <div class="col">欄 3</div>
    </div>
</div>
```

### 響應式斷點

| 類別名稱 | 螢幕寬度 | 描述 |
|------|---------|------|
| `.col-xs` | < 576px | 超小螢幕 |
| `.col-sm` | ≥ 576px | 小螢幕 |
| `.col-md` | ≥ 768px | 中等螢幕 |
| `.col-lg` | ≥ 992px | 大螢幕 |
| `.col-xl` | ≥ 1200px | 超大螢幕 |
| `.col-xxl` | ≥ 1400px | 特大螢幕 |

### 欄偏移

```html
<div class="row">
    <div class="col-md-4">.col-md-4</div>
    <div class="col-md-4 offset-md-4">.col-md-4 .offset-md-4</div>
</div>
```

### 響應式欄

```html
<div class="row">
    <div class="col-12 col-md-6 col-lg-4">
        在手機上佔滿寬度，平板上半寬，大螢幕上 1/3 寬
    </div>
</div>
```

## 最佳實踐

1. **使用語意化 HTML**：合理使用 header、nav、main、footer 等標籤
2. **保持簡潔**：避免過度巢狀
3. **響應式優先**：從行動端開始設計
4. **利用工具類別**：快速調整樣式
5. **自訂主題**：使用 Sass 變數保持一致性
