# Angular

Angular es un framework de código abierto desarrollado y mantenido por Google para crear aplicaciones web modernas. Utiliza TypeScript como lenguaje principal de desarrollo y ofrece una arquitectura basada en componentes, enlace de datos bidireccional, inyección de dependencias y muchas otras funciones potentes.

## Inicio Rápido

### Instalar Angular CLI

```bash
npm install -g @angular/cli
```

### Crear un Nuevo Proyecto

```bash
ng new my-app
cd my-app
```

### Iniciar el Servidor de Desarrollo

```bash
ng serve
```

Accede a http://localhost:4200

## Características Principales

- **Arquitectura de Componentes**: Componentes UI modulares y reutilizables
- **TypeScript**: Soporte de tipos fuertes para mejorar la calidad del código
- **Inyección de Dependencias**: Acoplamiento débil y fácil de probar
- **Enlace de Datos Bidireccional**: Sincronización automática de datos
- **Enrutamiento**: Solución completa de enrutamiento
- **Cliente HTTP**: Potentes capacidades de solicitudes de red
- **Manejo de Formularios**: Formularios reactivos y basados en plantillas
- **RxJS**: Soporte para programación reactiva
