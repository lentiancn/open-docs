# Руководство по установке Flutter

## Требования

Перед установкой Flutter убедитесь, что ваша среда разработки соответствует следующим требованиям. В отношении операционных систем Flutter поддерживает Windows, macOS и Linux. Для Windows требуется Windows 10 или выше, для macOS требуется macOS 10.14 или выше, для Linux требуется поддержка рабочего стола GNOME или KDE.

В отношении дискового пространства сам Flutter SDK занимает около 1 ГБ, но с учетом проектов разработки и кэша сборки рекомендуется зарезервировать не менее 10 ГБ свободного места. В отношении памяти 8 ГБ или более обеспечивают более плавный опыт разработки.

Для мобильной разработки разработка для iOS требует macOS и Xcode, разработка для Android требует установки Android Studio и Android SDK. Веб- и настольная разработка не имеют особых требований, достаточно установить Flutter SDK.

## Шаги установки

### Установка в Windows

Есть два основных способа установки Flutter в Windows. Первый способ — загрузить предварительно скомпилированный архив с веб-сайта Flutter, извлечь его в подходящее место, а затем добавить каталог flutter/bin в переменную среды PATH системы.

Способ загрузки: Посетите веб-сайт Flutter на flutter.dev и загрузите последнюю стабильную версию. Извлеките в C:\src\flutter или другое подходящее место. Откройте «Свойства системы» → «Дополнительно» → «Переменные среды», и в пользовательских переменных Path добавьте путь к каталогу bin Flutter.

После завершения откройте командную строку и выполните команду flutter doctor, чтобы проверить, была ли установка успешной. Flutter doctor проверит состояние Flutter SDK и даст предложения по настройке.

### Установка в macOS

В macOS рекомендуется установка через Git. Сначала убедитесь, что Xcode установлен, затем откройте терминал и выполните следующую команду для клонирования Flutter:

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

Добавьте Flutter в PATH:

```bash
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.bash_profile
```

Для пользователей zsh:

```bash
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.zshrc
```

Затем выполните source ~/.bash_profile или source ~/.zshrc для обновления переменных среды.

### Установка в Linux

Установка в Linux аналогична macOS. Сначала установите зависимости:

```bash
sudo apt-get update
sudo apt-get install curl git unzip xz-utils zip libglu1-mesa clang cmake ninja-build pkg-config libgtk-3-dev
```

Затем клонируйте репозиторий Flutter и настройте PATH, как в macOS.

## Проверка установки

После установки выполните следующие команды для проверки:

```bash
flutter --version
flutter doctor
```

Flutter doctor покажет состояние установки и любые отсутствующие зависимости. Завершите необходимые настройки согласно подсказкам, например установите Android Studio и настройте Android SDK.

## Настройка IDE

### VS Code

VS Code — один из рекомендуемых редакторов для Flutter. После установки VS Code откройте магазин расширений, найдите и установите расширения Flutter и Dart. После установки перезапустите VS Code для использования функций Flutter.

### Android Studio

После установки Android Studio откройте страницу плагинов, найдите плагин Flutter и установите его. После установки настройте путь к Flutter SDK: Откройте Preferences → Language & Frameworks → Flutter и установите путь к Flutter SDK.

## Быстрый старт

Создайте первый проект Flutter:

```bash
flutter create my_first_app
cd my_first_app
flutter run
```

Или используйте палитру команд (Ctrl+Shift+P) в VS Code и введите Flutter: New Project для создания нового проекта.

Выполните flutter devices для просмотра подключенных устройств, и flutter run -d device_id для запуска на конкретном устройстве.
