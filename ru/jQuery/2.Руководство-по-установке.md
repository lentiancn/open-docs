# Руководство по установке jQuery

## Получение jQuery

### Последняя версия

Текущая последняя версия — **jQuery 4.0.0**.

#### Сжатая версия для продакшена (Рекомендуется)
- Скачать: [jquery-4.0.0.min.js](https://code.jquery.com/jquery-4.0.0.min.js)
- Подходит для продакшена, маленький размер файла, быстрая загрузка

#### Версия для разработки
- Скачать: [jquery-4.0.0.js](https://code.jquery.com/jquery-4.0.0.js)
- Содержит полный код, легко читать и отлаживать

#### Файл Source Map
- Скачать: [jquery-4.0.0.min.map](https://code.jquery.com/jquery-4.0.0.min.map)
- Используется для отображения исходного кода в инструментах разработчика браузера

#### Slim-версия
Slim-версия — это уменьшенная версия без модулей Ajax и эффектов:
- [jquery-4.0.0.slim.min.js](https://code.jquery.com/jquery-4.0.0.slim.min.js) - сжатая
- [jquery-4.0.0.slim.js](https://code.jquery.com/jquery-4.0.0.slim.js) - несжатая

Если вам не нужны функции Ajax и анимации, использование Slim-версии может уменьшить размер файла примерно на 30%.

## Способы установки

### Способ 1: Подключение через CDN (Рекомендуется)

Использование jQuery CDN — самый простой способ:

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
```

Версия с проверкой Subresource Integrity (SRI) (рекомендуется, предотвращает подмену CDN):

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js" 
        integrity="sha384-KJ3o2DKtIkvYIK3UENzmM7KCkRr/rE9/Qpg6aAZGJwFDMVNA/GpGFF93hXpG5KkN" 
        crossorigin="anonymous"></script>
```

Посетите [https://releases.jquery.com](https://releases.jquery.com) для получения полного списка версий и хэшей SRI.

### Способ 2: Установка через npm

Для проектов Node.js установите через npm:

```bash
npm install jquery

После установки в node_modules/jquery/dist/ вы найдете:
- jquery-4.0.0.min.js - сжатая версия
- jquery-4.0.0.js - несжатая версия
- jquery-4.0.0.min.map - source map
```

### Способ 3: Установка через Yarn

```bash
yarn add jquery
```

### Способ 4: Скачивание локально

Посетите [страницу загрузки jQuery](https://jquery.com/download/), щелкните правой кнопкой мыши на нужной версии и выберите «Сохранить как...».

## Другие CDN-сервисы

Помимо официального CDN, следующие CDN также предоставляют jQuery:

### Google CDN
```
https://ajax.googleapis.com/ajax/libs/jquery/4.0.0/jquery.min.js
```

### Microsoft CDN
```
https://ajax.aspnetcdn.com/ajax/jquery/jquery-4.0.0.min.js
```

### CDNJS
```
https://cdnjs.cloudflare.com/ajax/libs/jquery/4.0.0/jquery.min.js
```

### jsDelivr
```
https://cdn.jsdelivr.net/npm/jquery@4.0.0/dist/jquery.min.js
```

Примечание: Эти сторонние CDN могут иметь некоторую задержку, и новые версии могут появиться с опозданием.

## Обновление jQuery

### Использование плагина jQuery Migrate

При обновлении jQuery рекомендуется использовать плагин jQuery Migrate для выявления проблем совместимости:

1. Обновление с более старых версий на 1.9+ или 3.x: Используйте [jQuery Migrate 1.4.1](https://code.jquery.com/jquery-migrate-1.4.1.min.js)

2. Обновление с jQuery 3.x на более новую версию: Используйте [jQuery Migrate 3.6.0](https://code.jquery.com/jquery-migrate-3.6.0.min.js)

Использование: Подключите плагин Migrate после jQuery

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
<script src="https://code.jquery.com/jquery-migrate-3.6.0.min.js"></script>
```

Migrate будет выводить предупреждения о совместимости в консоль, чтобы помочь вам исправить проблемы пошагово.

### Руководства по обновлению

Для обновления между разными версиями ознакомьтесь с официальными руководствами:

## Старые версии

Для использования старых версий jQuery посетите [jQuery CDN](https://releases.jquery.com) для просмотра всех исторических версий.

## Предрелизные версии

Команда разработчиков jQuery постоянно работает над кодом. Каждый коммит в репозитории GitHub создает рабочую версию. Эти версии могут быть нестабильными и не рекомендуются для использования в продакшене.

Просмотр предрелизных версий: [https://releases.jquery.com/jquery/](https://releases.jquery.com/jquery/)
