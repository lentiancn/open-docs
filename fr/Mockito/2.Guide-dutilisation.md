# Guide dutilisation Mockito

Guide pour utiliser Mockito efficacement.

---

## Principes de base

Mockito est un framework de mock pour les tests unitaires en Java.

---

## Creer un Mock

```java
List<String> mockList = Mockito.mock(List.class);
```

---

## Stubbing

```java
when(mockList.get(0)).thenReturn("first");
```

---

## Verification

```java
verify(mockList).add("element");
```

---

## Annotations

```java
@Mock
private UserService userService;
```
