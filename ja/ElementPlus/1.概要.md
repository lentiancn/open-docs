# Element Plus 概要

Vue 3のためのデスクトップUIコンポーネントライブラリ。

## Element Plusとは？

Element Plusは、Vele.meチームによって開発されたVue 3コンポーネントライブラリで、70以上の高品質コンポーネントを提供します。Element UIのVue 3バージョンとして、前バージョンの優れた特性を継承しつつ、Vue 3用に完全に最適化されています。

### 主な機能

- **Vue 3ベース**: Vue 3 Composition APIで構築
- **TypeScriptサポート**: ネイティブの型定義
- **豊富なコンポーネント**: 70以上の高品質コンポーネント
- **ツリーシェイキング**: 必要に応じた読み込みでバンドルサイズを最適化
- **テーマカスタマイズ**: CSS変数でテーマをカスタマイズ
- **国際化**: 20以上の言語パックを 内蔵
- **アクセシビリティ**: 完全なARIAサポート
- **レスポンシブデザイン**: 様々な画面サイズに対応

### コンポーネントカテゴリ

**基本コンポーネント:**
- Button（ボタン）
- Icon（アイコン）
- Layout（レイアウト）
- Container（コンテナ）

**フォームコンポーネント:**
- Input（入力欄）
- Select（セレクター）
- Checkbox（チェックボックス）
- Radio（ラジオボタン）
- Switch（スイッチ）
- DatePicker（日付ピッカー）
- Form（フォーム）
- Upload（アップロード）

**データ表示:**
- Table（テーブル）
- Tag（タグ）
- Progress（進捗バー）
- Tree（ツリー）
- Pagination（ページネーション）

**フィードバック:**
- Message（メッセージ）
- Notification（通知）
- Dialog（ダイアログ）
- Drawer（ドロワー）

**ナビゲーション:**
- Menu（メニュー）
- Tabs（タブ）
- Breadcrumb（パンくず）
