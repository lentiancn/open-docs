# MongoDB 사용 매뉴얼

## 기본 개념

### 데이터베이스

데이터베이스는 MongoDB에서 데이터의 최상위 컨테이너입니다.

### 컬렉션

컬렉션은 관계형 데이터베이스의 테이블과 유사한 MongoDB 문서 그룹입니다.

### 문서

문서는 MongoDB의 기본 데이터 단위로, 필드-값 쌍으로 구성됩니다.

## MongoDB에 연결

### 로컬 데이터베이스에 연결

```bash
mongosh
```

## 데이터베이스 작업

### 데이터베이스 보기

```bash
show dbs
```

### 데이터베이스 전환/생성

```bash
use mydatabase
```

## 컬렉션 작업

### 컬렉션 보기

```bash
show collections
```

### 컬렉션 생성

```bash
db.createCollection("mycollection")
```

## 문서 삽입

### 단일 문서 삽입

```bash
db.users.insertOne({
  name: "홍길동",
  age: 30,
  email: "hong@example.kr"
})
```

### 여러 문서 삽입

```bash
db.users.insertMany([
  { name: "김철수", age: 25 },
  { name: "이영희", age: 35 }
])
```

## 문서 조회

### 모든 문서 조회

```bash
db.users.find()
```

### 조건부 조회

```bash
db.users.find({ age: 30 })
db.users.find({ age: { $gte: 25 } })
```

## 문서 업데이트

```bash
db.users.updateOne(
  { name: "홍길동" },
  { $set: { age: 31 } }
)
```

## 문서 삭제

```bash
db.users.deleteOne({ name: "홍길동" })
```

## 집계

```bash
db.users.aggregate([
  { $match: { age: { $gte: 25 } } },
  { $group: { _id: "$city", total: { $sum: 1 } } }
])
```

## 인덱스

### 인덱스 생성

```bash
db.users.createIndex({ name: 1 })
db.users.createIndex({ age: 1, name: 1 })
```

##Replica Set 작업

### 초기화

```bash
rs.initiate()
```

## Sharding 작업

### Sharding 활성화

```bash
sh.enableSharding("mydatabase")
```

## 데이터 가져오기 및 내보내기

### mongodump로 내보내기

```bash
mongodump --db=mydatabase --out=/backup
```

### mongorestore로 가져오기

```bash
mongorestore --db=mydatabase /backup/mydatabase
```

## 관리 명령

### 서버 상태

```bash
db.serverStatus()
```

## 트랜잭션

MongoDB 4.0+는 다중 문서 트랜잭션을 지원합니다.

## Change Streams

Change Streams를 사용하면 애플리케이션이 실시간으로 데이터 변경을 감시할 수 있습니다.

## GridFS

GridFS는 16MB 제한을 초과하는 큰 파일을 저장하는 데 사용됩니다.
