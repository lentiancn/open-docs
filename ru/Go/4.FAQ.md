# FAQ Go

> Часто задаваемые вопросы

---

## Установка

### Windows: команда go не найдена?

**Решение**: Добавить `C:\Go\bin` в PATH.

### Настройка прокси?

```bash
go env -w GOPROXY=https://goproxy.cn,direct
```

---

## Разработка

### Ошибка "cannot find package"?

**Решение**:
1. Выполнить `go mod init`
2. Выполнить `go mod tidy`

### В чём разница между := и var?

- `var`: Явное объявление, можно использовать вне функций
- `:=`: Короткое объявление, только внутри функций

---

## Параллелизм

### Утечка goroutine?

```go
func worker(ctx context.Context) {
    for {
        select {
        case <-ctx.Done():
            return
        }
    }
}
```
