# 소개

Bootstrap은 전 세계에서 가장 많이 사용되는 HTML, CSS, JavaScript 오픈소스 프론트엔드 개발 프레임워크로, Twitter사의 디자이너 Mark Otto와 Jacob Thornton이 주축이 되어 개발했습니다. Bootstrap은 다양한 컴포넌트와 강력한 JavaScript 플러그인을 제공하여 개발자가 빠르게 반응형 웹 애플리케이션을 구축할 수 있도록 도와줍니다.

Bootstrap의 핵심 철학은 "모바일 우선(Mobile First)"으로, 처음 설계할 때부터 다양한 디바이스适配 문제를 고려했습니다. Bootstrap을 사용하면 개발자가 데스크톱, 태블릿, 스마트폰 등 다른 기기에서 완벽하게 표시되는 웹사이트를 쉽게 만들 수 있습니다.

## 발전 과정

Bootstrap은 처음에 Twitter Blueprint라는 이름으로 2011년에 오픈소스로 공개되었습니다. 2012년에 출시된 Bootstrap 2에서는 반응형 레이아웃이 도입되었고, 2013년에 출시된 Bootstrap 3에서는 완전히 다시 작성되어 플랫 디자인을 채택했습니다. 2015년에 출시된 Bootstrap 4에서는 Sass를 CSS 전처리기로 사용하기 시작했고, 2021년에 출시된 Bootstrap 5에서는 jQuery 의존성을 제거하고 네이티브 JavaScript를 전면적으로 사용하게 되었습니다.

## 핵심 기능

### 반응형 레이아웃

Bootstrap의 그리드 시스템은 가장 강력한 기능 중 하나입니다. 12열 그리드 레이아웃을 통해 개발자는 다양한 화면 크기에 맞는 페이지 레이아웃을 쉽게 만들 수 있습니다. Bootstrap 5는 xs, sm, md, lg, xl, xxl 총 6개의 중단점을 제공하며, 각각 서로 다른 너비의 디바이스에 대응합니다.

### 다양한 컴포넌트

Bootstrap은 바로 사용할 수 있는 많은 UI 컴포넌트를 제공합니다. 네비게이션 바, 카드, 버튼, 폼, 모달 창, 캐러셀, 드롭다운 메뉴 등이 포함됩니다. 이러한 컴포넌트들은 모두 정교하게 설계되어 있어 우수한 시각적 효과와 상호작용用户体验을 제공합니다.

### 강력한 JavaScript 플러그인

Bootstrap에는 모달 창, 캐러셀, 툴팁, 팝업, 접이식 패널 등 20개 이상의 JavaScript 플러그인이 포함되어 있습니다. 이러한 플러그인들은 모두 선택 사항이며, 개발자는 필요에 따라 가져와서 프로젝트 크기를 줄일 수 있습니다.

### Sass 기반

Bootstrap 5는 Sass를 CSS 전처리로 사용하며, 많은 Sass 변수와 Mixin을 제공합니다. 개발자는 Bootstrap의 테마 색상, 폰트 크기 등의 스타일을 쉽게 커스터마이징할 수 있습니다.

### 커스텀 테마

Bootstrap의 테마 시스템을 통해 개발자는 자신만의 테마 스타일을 쉽게 만들 수 있습니다. 또한 공식적으로 Bootstrap Themes 마켓이 있어 다양한精美한 테마를 선택할 수 있습니다.

### 접근성 지원

Bootstrap 컴포넌트에는 키보드 내비게이션, 스크린 리더 호환성 등 접근성 지원이 내장되어 있어, 개발자가更具포괄적인 웹사이트를 만드는 데 도움이 됩니다.

## Bootstrap 5 새 기능

Bootstrap 4와 비교할 때 Bootstrap 5에는 많은 중요한 업데이트가 있습니다:

- jQuery 의존성 제거, 네이티브 JavaScript 전면 사용
- 그리드 시스템 개선, XXL 중단점 추가
- 새로운 폼 컨트롤로 폼 검증 대체
- 더 나은 커스텀 속성 지원
- 아이콘 라이브러리 Font Awesome에서 Bootstrap Icons로 변경
- RTL(오른쪽에서 왼쪽) 지원

## 적용 시나리오

- 빠른 프로토타입 개발
- 기업 내부 관리 시스템
- 전자상거래 웹사이트
- 블로그 및 콘텐츠 웹사이트
- 관리자 백엔드 인터페이스
- 반응형 웹사이트 개발

전 세계에서 가장 사랑받는 CSS 프레임워크 중 하나인 Bootstrap은 큰 커뮤니티와 풍부한 학습 자료를 갖추고 있어, 프론트엔드 개발자에게 필수적인 기술 중 하나입니다.
