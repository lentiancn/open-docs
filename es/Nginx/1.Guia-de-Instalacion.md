# Guia de Instalacion de Nginx

Guia para instalar Nginx en todas las plataformas principales.

---

## Linux

### Ubuntu / Debian

```bash
# Actualizar paquetes
sudo apt update

# Instalar Nginx
sudo apt install nginx

# Iniciar Nginx
sudo systemctl start nginx

# Habilitar al iniciar
sudo systemctl enable nginx
```

### CentOS / RHEL

```bash
# Instalar EPEL
sudo yum install epel-release

# Instalar Nginx
sudo yum install nginx

# Iniciar
sudo systemctl start nginx
```

### Fedora

```bash
sudo dnf install nginx
sudo systemctl start nginx
```

---

## macOS

```bash
brew install nginx
brew services start nginx
```

---

## Windows

1. Descargar de [nginx.org](http://nginx.org/en/download.html)
2. Extraer a `C:\nginx`

```powershell
cd C:\nginx
start nginx
```

---

## Docker

```bash
docker run -d -p 80:80 --name nginx nginx:latest
```

---

## Verificar

```bash
nginx -v
nginx -t
curl http://localhost
```

---

## Configuracion

### Archivo de configuracion principal

| OS | Ruta |
|----|------|
| Ubuntu/Debian | `/etc/nginx/nginx.conf` |
| CentOS/RHEL | `/etc/nginx/nginx.conf` |

### Comandos comunes

```bash
nginx -t
sudo systemctl reload nginx
```

---

## Desinstalar

```bash
sudo apt remove nginx
```
