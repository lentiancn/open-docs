# Element Plus Installation Guide

Element Plus is a Vue 3 based component library for developers, designers and product managers. This guide covers installation in all environments.

## Version Information

| Version | Status | Description |
|---------|--------|-------------|
| Element Plus 2.x | ✅ Current | Latest stable for Vue 3 |
| Element Plus 1.x | ⚠️ Maintenance | Security updates only |
| Element UI | ❌ Legacy | For Vue 2 only, not maintained |

> **Important**: Element Plus is designed for Vue 3. If you're using Vue 2, use Element UI instead.

## Installation Methods

### Method 1: npm (Recommended)

```bash
# Using npm
npm install element-plus

# Using yarn
yarn add element-plus

# Using pnpm
pnpm add element-plus
```

### Method 2: CDN

```html
<!-- Import Vue 3 -->
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<!-- Import Element Plus -->
<link rel="stylesheet" href="https://unpkg.com/element-plus/dist/index.css" />
<script src="https://unpkg.com/element-plus/dist/index.full.js"></script>

<!-- Import Element Plus icons (optional) -->
<script src="https://unpkg.com/@element-plus/icons-vue"></script>
```

#### Complete CDN Template

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Element Plus CDN</title>
    <link rel="stylesheet" href="https://unpkg.com/element-plus/dist/index.css">
</head>
<body>
    <div id="app">
        <el-button>Button</el-button>
        <el-date-picker></el-date-picker>
    </div>

    <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
    <script src="https://unpkg.com/element-plus/dist/index.full.js"></script>
    <script src="https://unpkg.com/@element-plus/icons-vue"></script>
    
    <script>
        const { createApp, ref } = Vue
        const app = createApp({
            setup() {
                return {}
            }
        })
        
        for (const [key, component] of Object.entries(ElementPlus)) {
            app.component(key, component)
        }
        
        for (const [key, component] of Object.entries(ElementPlusIconsVue)) {
            app.component(key, component)
        }
        
        app.use(ElementPlus)
        app.mount('#app')
    </script>
</body>
</html>
```

### Method 3: Using Vite

```bash
# Create Vite project
npm create vite@latest my-app -- --template vue

# Enter project directory
cd my-app

# Install Element Plus
npm install element-plus

# Install icons
npm install @element-plus/icons-vue
```

#### Configure in main.js

```javascript
import { createApp } from 'vue'
import App from './App.vue'

// Import Element Plus styles
import 'element-plus/dist/index.css'

// Import Element Plus
import ElementPlus from 'element-plus'

// Import icons
import * as ElementPlusIconsVue from '@element-plus/icons-vue'

const app = createApp(App)

// Register all icons
for (const [key, component] of Object.entries(ElementPlusIconsVue)) {
    app.component(key, component)
}

app.use(ElementPlus)
app.mount('#app')
```

### Method 4: Using Vue CLI

```bash
# Install Vue CLI
npm install -g @vue/cli

# Create project
vue create my-project

# Add Element Plus
cd my-project
npm install element-plus
npm install @element-plus/icons-vue
```

#### Configure in main.js

```javascript
import { createApp } from 'vue'
import App from './App.vue'

import 'element-plus/dist/index.css'
import ElementPlus from 'element-plus'
import * as icons from '@element-plus/icons-vue'

const app = createApp(App)

// Register icons globally
for (const [key, component] of Object.entries(icons)) {
    app.component(key, component)
}

app.use(ElementPlus)
app.mount('#app')
```

### Method 5: On-demand Import (Recommended for Production)

#### Using unplugin-vue-components

```bash
# Install dependencies
npm install element-plus unplugin-vue-components unplugin-auto-import
```

#### Vite Configuration

```javascript
// vite.config.js
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

#### Usage

```vue
<template>
  <!-- No need to import components -->
  <el-button type="primary">Button</el-button>
  <el-input v-model="input" placeholder="Please input" />
</template>

<script setup>
// Just use directly
import { ref } from 'vue'
const input = ref('')
</script>
```

## Common Issues

### Icons not showing?

Make sure to register icons globally:

```javascript
import * as icons from '@element-plus/icons-vue'
for (const [key, icon] of Object.entries(icons)) {
    app.component(key, icon)
}
```

### Style not applied?

Make sure to import Element Plus styles:

```javascript
import 'element-plus/dist/index.css'
```

### Vue version mismatch?

Element Plus requires Vue 3. Check your Vue version:

```bash
npm list vue
```
