# Guía de Instalación de JDK

## Versiones

| Versión | Tipo |
|---------|------|
| JDK 25 | Última |
| JDK 21 | LTS |
| JDK 17 | LTS |

## Windows

```cmd
JAVA_HOME = C:\Program Files\Java\jdk-21
PATH = %JAVA_HOME%\bin
```

## macOS

```bash
brew install openjdk@21
```

## Linux

```bash
sudo apt install openjdk-21-jdk
```

## Verificar

```bash
java -version
```
