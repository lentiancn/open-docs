# Element Plus 자주 묻는 질문

이 페이지는 Element Plus를 사용할 때 발생하는 일반적인 문제와 해결책을 나열합니다.

## 설치 문제

### 설치 실패

증상: npm install 또는 yarn add가 실패합니다

해결책:

1. Node.js 버전을 확인하고 14.0.0 이상인지 확인하세요
2. npm 캐시를 지우세요:
   npm cache clean --force
3. node_modules와 package-lock.json을 삭제한 후 다시 설치하세요

### TypeScript 타입 오류

증상: 컴포넌트를 가져올 때 타입 오류가 발생합니다

해결책:

1. Volar 플러그인이 설치되어 있는지 확인하세요
2. tsconfig.json에 추가하세요:
   {
     "compilerOptions": {
       "types": ["element-plus/global"]
     }
   }

## 사용 문제

### 컴포넌트 스타일이 적용되지 않음

증상: 컴포넌트 스타일이 예상과 일치하지 않습니다

해결책:

1. CSS가 올바르게 임포트되었는지 확인하세요:
   import 'element-plus/dist/index.css'
2. 전역 스타일 재정의가 있는지 확인하세요

### 아이콘이 표시되지 않음

증상: 아이콘 컴포넱트가 공백 또는 사각형으로 표시됩니다

해결책:

1. 아이콘 라이브러리를 설치하세요:
   npm install @element-plus/icons-vue

2. 아이콘을 전역으로 등록하세요:
   import * as icons from '@element-plus/icons-vue'
   for (const [key, component] of Object.entries(icons)) {
     app.component(key, component)
   }

### 온디맨드 임포트가 작동하지 않음

증상: 번들 크기가 여전히 큽니다

해결책:

1. 플러그인 구성이 올바른지 확인하세요
2. 올바른 resolver를 사용하고 있는지 확인하세요

### 국제화가 작동하지 않음

증상: 언어 변경 후 컴포넌트 텍스트가 변경되지 않습니다

해결책:

1. 언어 팩을 올바르게 임포트하세요:
   import ko from 'element-plus/es/locale/lang/ko'
2. 로케일을 구성하세요:
   app.use(ElementPlus, { locale: ko })

## 컴포넌트 문제

### 테이블 문제

#### 테이블 정렬이 작동하지 않음

prop 속성이 설정되어 있는지 확인하세요:

<el-table-column prop="date" sortable />

#### 테이블 고정 열이 맞지 않음

테이블에 고정 높이가 있는지 확인하세요:

<el-table :data="data" height="300" />

### 폼 문제

#### 폼 검증이 트리거되지 않음

el-form-item에 prop 속성이 있는지 확인하세요:

<el-form-item label="이름" prop="name">
  <el-input v-model="ruleForm.name" />
</el-form-item>

### 다이얼로그 문제

#### 다이얼로그 스크롤 문제

다이얼로그 내용이 너무 많을 때 스크롤을 추가해야 합니다:

<dialog>
  <div style="max-height: 60vh; overflow-y: auto;">
    내용
  </div>
</dialog>

## 스타일 문제

### 스타일 충돌

증상: 사용자 정의 스타일이 재정의됩니다

해결책:

1. 더 구체적인 선택자를 사용하세요
2. ::v-deep 또는 :deep()을 사용하세요

### 테마 색상이 적용되지 않음

CSS 변수가 올바르게 재정의되었는지 확인하세요:

:root {
  --el-color-primary: #409eff;
}

element-plus 스타일 이후에 로드되었는지 확인하세요.

## 성능 문제

### 페이지 느림

대량 데이터의 경우:

1. el-table-v2 (가상 테이블)를 사용하세요
2. 페이지네이션을 사용하여 데이터를 로드하세요
3. 테이블에서 복잡한 컴포넌트를 피하세요

### 번들 크기가 너무 큼

1. 온디맨드 임포트를 사용하세요
2. 사용하지 않는 플러그인을 제거하세요

## Vue 버전 문제

### Vue 2와 호환되지 않음

Element Plus는 Vue 3만 지원합니다. Vue 2가 필요한 경우 Element UI를 사용하세요.

### Composition API가 지원되지 않음

더 나은 Composition API 지원을 위해 Vue 3.2 이상을 사용하세요.
