## 6. Adding Interactive Features

### 6.1 Create Counter Functionality

Add to `app.component.ts`:

```typescript
import { Component } from '@angular/core';
import { CommonModule } from '@angular/common';

@Component({
  selector: 'app-root',
  standalone: true,
  imports: [CommonModule],
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent {
  title = 'hello-angular';
  count = 0;
  
  increment(): void {
    this.count++;
  }
  
  decrement(): void {
    this.count--;
  }
}
```

### 6.2 Use in Template

Update `app.component.html`:

```html
<div class="container">
  <h1>Welcome to Angular!</h1>
  
  <div class="card">
    <h2>Counter Example</h2>
    <p class="count">{{ count }}</p>
    <div class="buttons">
      <button (click)="decrement()">-</button>
      <button (click)="increment()">+</button>
    </div>
  </div>
</div>
```

### 6.3 Add Button Styles

Update `app.component.css`:

```css
.count {
  font-size: 3rem;
  font-weight: bold;
  text-align: center;
  margin: 1rem 0;
}

.buttons {
  display: flex;
  gap: 1rem;
  justify-content: center;
}

button {
  padding: 0.5rem 1.5rem;
  font-size: 1.2rem;
  border: none;
  border-radius: 4px;
  cursor: pointer;
  background: #dd0031;
  color: white;
  transition: background 0.2s;
}

button:hover {
  background: #b3002d;
}
```

### 6.4 View Results

After saving, the browser will automatically refresh. You can click the buttons to test the interactive functionality.
