# Flutter Installation Guide

Complete Flutter development environment setup guide.

## Requirements

### Windows

- **OS**: Windows 10+ (64-bit)
- **Space**: 2.8 GB minimum
- **Tools**: PowerShell 5.0+, Git

### macOS

- **OS**: macOS 10.14+
- **Space**: 2.8 GB minimum
- **Tools**: Command Line Tools, Git

### Linux

- **OS**: Ubuntu 18.04+
- **Space**: 2.8 GB minimum
- **Dependencies**: bash, curl, git, unzip, xz-utils, zip, libglu1-mesa

## Installation

### Windows

1. **Download Flutter SDK**

```bash
# Or clone via git
git clone https://github.com/flutter/flutter.git -b stable
```

2. **Add to PATH**

```powershell
[Environment]::SetEnvironmentVariable(
    "Path",
    $env:Path + ";C:\src\flutter\bin",
    "User"
)
```

3. **Verify**

```bash
flutter doctor
```

### macOS

1. **Clone Flutter**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

2. **Add to PATH**

```bash
# Add to ~/.bash_profile or ~/.zshrc
export PATH="$PATH:/Users/username/flutter/bin"
```

3. **Run flutter doctor**

```bash
flutter doctor
```

### Linux

1. **Install dependencies**

```bash
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa
```

2. **Clone Flutter**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

3. **Configure PATH**

```bash
export PATH="$PATH:/home/username/flutter/bin"
```

## Android Setup

### Install Android Studio

1. Download from: https://developer.android.com/studio
2. Install and launch
3. Configure SDK:

```bash
flutter config --android-sdk /path/to/android/sdk
```

### Accept licenses

```bash
flutter doctor --android-licenses
```

## iOS Setup (macOS)

### Install Xcode

```bash
xcode-select --install
```

## IDE Setup

### VS Code

1. Install Flutter extension
2. Install Dart extension
3. Restart VS Code

### Android Studio

1. Install Flutter plugin
2. Install Dart plugin
3. Restart Android Studio

## Verification

```bash
flutter --version
flutter doctor
flutter upgrade
```

## Troubleshooting

### Download failed?

Use mirror:
```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
```

### Android SDK not found?

```bash
flutter config --android-sdk /path/to/android-sdk
```
