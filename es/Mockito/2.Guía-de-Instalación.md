# Mockito Guía de Instalación

Guía para instalar y configurar Mockito.

## Requisitos Previos

- Java Development Kit (JDK) 8 o superior
- Maven o Gradle (recomendado) o archivo JAR simple

## Instalación

### Maven

Agregue la siguiente dependencia a su `pom.xml`:

```xml
<dependency>
    <groupId>org.mockito</groupId>
    <artifactId>mockito-core</artifactId>
    <version>5.8.0</version>
    <scope>test</scope>
</dependency>
```

### Gradle

Agregue la dependencia a su `build.gradle`:

```groovy
testImplementation 'org.mockito:mockito-core:5.8.0'
```

### Archivo JAR

Descargue el JAR más reciente de Maven Central:
https://repo1.maven.org/maven2/org/mockito/mockito-core/

## Integración con IntelliJ IDEA

Mockito funciona automáticamente con IntelliJ IDEA:

1. Abra su proyecto
2. Agregue la dependencia Maven
3. Mockito se reconocerá automáticamente

## Integración con Eclipse

1. Haga clic derecho en su proyecto
2. Seleccione "Build Path" > "Configure Build Path"
3. Agregue el JAR en "Libraries" > "Add External JARs"

## Configuración

### Extensión Mockito para JUnit 5

JUnit 5 (Jupiter) tiene soporte integrado:

```java
import org.junit.jupiter.api.extension.ExtendWith;
import org.mockito.junit.jupiter.MockitoExtension;

@ExtendWith(MockitoExtension.class)
class MyTest {
    // Sus pruebas aquí
}
```

## Solución de Problemas

### Problema: NoClassDefFoundError

**Solución:** Asegúrese de que todas las dependencias estén incluidas.

### Problema: Mock no funciona

**Solución:**
- Verifique si la clase es final
- Use `mockito-inline` para clases finales
