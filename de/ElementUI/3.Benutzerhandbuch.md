# Element UI Benutzerhandbuch

Dieses Handbuch hilft Ihnen, schnell mit Element UI zu beginnen und die am häufigsten verwendeten Komponenten und Funktionen zu meistern.

## Schnellstart

Stellen Sie vor der Verwendung von Element UI sicher, dass die Installation abgeschlossen ist.

### Grundlegende Verwendung

Verwenden Sie Komponenten direkt in Vue-Komponenten:

<template>
  <el-button>Standardschaltfläche</el-button>
  <el-button type="primary">Primäre Schaltfläche</el-button>
  <el-button type="success">Erfolgschaltfläche</el-button>
  <el-button type="warning">Warnschaltfläche</el-button>
  <el-button type="danger">Gefahrenschaltfläche</el-button>
</template>

## Häufige Komponenten

### Schaltfläche (Button)

Grundlegende Verwendung:

<el-button>Standard</el-button>
<el-button type="primary">Primär</el-button>
<el-button type="success">Erfolg</el-button>
<el-button type="warning">Warnung</el-button>
<el-button type="danger">Gefahr</el-button>

Unterschiedliche Größen:

<el-button size="medium">Mittel</el-button>
<el-button>Standard</el-button>
<el-button size="small">Klein</el-button>
<el-button size="mini">Mini</el-button>

Deaktivierter Zustand:

<el-button disabled>Deaktiviert</el-button>

Schaltflächen mit Symbol:

<el-button icon="el-icon-search">Suchen</el-button>
<el-button icon="el-icon-edit">Bearbeiten</el-button>
<el-button icon="el-icon-delete">Löschen</el-button>

### Eingabe (Input)

Grundlegende Verwendung:

<el-input v-model="input" placeholder="Bitte Inhalt eingeben" />

Mit Symbol:

<el-input v-model="input" prefix-icon="el-icon-search" />
<el-input v-model="input" suffix-icon="el-icon-date" />

Mit Löschschaltfläche:

<el-input v-model="input" clearable />

Passworteingabe:

<el-input v-model="input" type="password" show-password />

### Auswahl (Select)

Grundauswahl:

<el-select v-model="value" placeholder="Bitte auswählen">
  <el-option label="Option 1" value="1" />
  <el-option label="Option 2" value="2" />
</el-select>

Mehrfachauswahl:

<el-select v-model="value" multiple placeholder="Bitte auswählen">
  <el-option label="Option 1" value="1" />
  <el-option label="Option 2" value="2" />
</el-select>

### Tabelle (Table)

Grundlegende Tabelle:

<el-table :data="tableData" stripe border>
  <el-table-column prop="date" label="Datum" width="180" />
  <el-table-column prop="name" label="Name" width="180" />
  <el-table-column prop="address" label="Adresse" />
</el-table>

### Formular (Form)

Grundlegendes Formular:

<el-form :model="form" label-width="80px">
  <el-form-item label="Aktivitätsname">
    <el-input v-model="form.name" />
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="onSubmit">Absenden</el-button>
  </el-form-item>
</el-form>

### Dialog

<el-dialog :visible.sync="dialogVisible" title="Titel" width="30%">
  <span>Inhalt</span>
</el-dialog>

### Nachricht

import { Message } from 'element-ui'

Message.success('Erfolgsmeldung')
Message.warning('Warnmeldung')
Message.error('Fehlermeldung')

### Menü

Horizontales Menü:

<el-menu mode="horizontal">
  <el-menu-item index="1">Verarbeitungszentrum</el-menu-item>
  <el-menu-item index="2">Auftragsverwaltung</el-menu-item>
</el-menu>

### Tabs

<el-tabs v-model="activeName">
  <el-tab-pane label="Benutzerverwaltung" name="first">Benutzerverwaltung</el-tab-pane>
  <el-tab-pane label="Konfigurationsverwaltung" name="second">Konfigurationsverwaltung</el-tab-pane>
</el-tabs>

### Paginierung

<el-pagination
  @current-change="handleCurrentChange"
  :current-page="currentPage"
  :page-size="100"
  :total="1000"
  layout="total, prev, pager, next"
/>

## Layout-System

### Rastersystem

Element UI verwendet ein 24-Spalten-Rastersystem:

<el-row :gutter="20">
  <el-col :span="12"><div>12 Spalten</div></el-col>
  <el-col :span="12"><div>12 Spalten</div></el-col>
</el-row>

### Container-Layout

<el-container>
  <el-header>Kopfzeile</el-header>
  <el-main>Hauptbereich</el-main>
  <el-footer>Fußzeile</el-footer>
</el-container>

## Best Practices

### Responsives Design

<el-row>
  <el-col :xs="24" :sm="12" :md="8" :lg="6">Responsive Spalte</el-col>
</el-row>

### Formularvalidierung

Validieren Sie immer die Benutzereingabe und geben Sie klare Fehlermeldungen aus.

## Häufige Probleme

### Komponentenstile werden nicht angewendet

Stellen Sie sicher, dass CSS importiert ist:

import 'element-ui/lib/theme-chalk/index.css'

### Symbole werden nicht angezeigt

Sie müssen die Symbolschriftarten importieren:

import 'element-ui/lib/theme-chalk/fonts/element-icons.ttf'
