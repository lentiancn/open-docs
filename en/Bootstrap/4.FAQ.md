## 1. Installation Issues

### Q1: Does Bootstrap 5 still need jQuery?

**No.** Bootstrap 5 completely removed jQuery dependency and uses native JavaScript.

---

### Q2: What's the difference between bootstrap.min.css and bootstrap.css?

- `bootstrap.css` - Uncompressed, for development
- `bootstrap.min.css` - Compressed, for production

---

### Q3: What's the difference between bootstrap.bundle.js and bootstrap.js?

- `bootstrap.bundle.js` - Includes Popper.js (needed for tooltips, dropdowns)
- `bootstrap.js` - Does not include Popper.js

---

### Q4: CDN fails to load?

```html
<!-- Backup CDN -->
<link href="https://cdnjs.cloudflare.com/ajax/libs/twitter-bootstrap/5.3.3/css/bootstrap.min.css" rel="stylesheet">

<!-- Or local reference -->
<link href="css/bootstrap.min.css" rel="stylesheet">
```

## 2. Layout Issues

### Q5: Grid system not working?

Ensure:
1. Rows (row) are inside containers
2. Columns (col) are inside rows
3. Don't add padding directly to columns

```html
<!-- Correct -->
<div class="container">
    <div class="row">
        <div class="col-md-6">Content</div>
    </div>
</div>
```

---

### Q6: How to center columns vertically?

```html
<div class="row align-items-center">
    <div class="col">Centered content</div>
</div>
```

## 3. Component Issues

### Q7: Modal won't close?

Ensure:
1. Bootstrap JS is properly included
2. Use correct close button

```html
<button type="button" class="btn-close" data-bs-dismiss="modal"></button>
```

---

### Q8: Tooltips not showing?

Initialize:

```javascript
var tooltipTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="tooltip"]'))
var tooltipList = tooltipTriggerList.map(function (tooltipTriggerEl) {
    return new bootstrap.Tooltip(tooltipTriggerEl)
})
```

---

### Q9: Navbar not collapsing on mobile?

Ensure:
1. Bootstrap JS is included
2. Correct collapse button structure

```html
<button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav">
    <span class="navbar-toggler-icon"></span>
</button>
```

---

### Q10: Dropdown not working?

```html
<div class="dropdown">
    <button class="btn btn-secondary dropdown-toggle" data-bs-toggle="dropdown">
        Dropdown
    </button>
    <ul class="dropdown-menu">
        <li><a class="dropdown-item" href="#">Option</a></li>
    </ul>
</div>
```

## 4. Customization Issues

### Q11: How to change theme colors?

```css
:root {
    --bs-primary: #your-color;
}
```

---

### Q12: How to add custom styles?

```html
<link href="bootstrap.min.css" rel="stylesheet">
<link href="custom.css" rel="stylesheet">
```

### Q13: How to use Bootstrap Icons?

## 5. Compatibility Issues

### Q14: Which browsers are supported?

Bootstrap 5 supports:
- Chrome 90+
- Firefox 90+
- Safari 14+
- Edge 90+

**Internet Explorer is not supported**

---

### Q15: Mobile layout broken?

Ensure viewport meta tag is added:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

---

### Q16: Responsive images not working?

```html
<img src="..." class="img-fluid" alt="...">
```
