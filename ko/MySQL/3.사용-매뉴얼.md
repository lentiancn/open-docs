# MySQL 사용 매뉴얼

## 기본 데이터베이스 작업

### MySQL에 연결

로컬 연결:
```bash
mysql -u root -p
```

원격 연결:
```bash
mysql -h 호스트명 -u 사용자명 -p 데이터베이스명
```

### 데이터베이스 생성

```sql
CREATE DATABASE 데이터베이스명;
```

특정 문자 집합으로 생성:
```sql
CREATE DATABASE 데이터베이스명 CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 데이터베이스 보기

```sql
SHOW DATABASES;
```

### 데이터베이스 선택

```sql
USE 데이터베이스명;
```

### 데이터베이스 삭제

```sql
DROP DATABASE 데이터베이스명;
```

## 테이블 작업

### 테이블 생성

```sql
CREATE TABLE 테이블명 (
    id INT AUTO_INCREMENT PRIMARY KEY,
    이름 VARCHAR(100) NOT NULL,
    이메일 VARCHAR(100) UNIQUE,
    생성일 TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    수정일 TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### 테이블 구조 보기

```sql
DESCRIBE 테이블명;
-- 또는
SHOW COLUMNS FROM 테이블명;
```

### 모든 테이블 보기

```sql
SHOW TABLES;
```

### 테이블 구조 수정

열 추가:
```sql
ALTER TABLE 테이블명 ADD COLUMN 새열 VARCHAR(50);
```

열 수정:
```sql
ALTER TABLE 테이블명 MODIFY 열이름 VARCHAR(200);
```

열 삭제:
```sql
ALTER TABLE 테이블명 DROP COLUMN 열이름;
```

### 테이블 삭제

```sql
DROP TABLE 테이블명;
```

## 데이터 작업

### 데이터 삽입

단일 삽입:
```sql
INSERT INTO 테이블명 (열1, 열2) VALUES ('값1', '값2');
```

다중 삽입:
```sql
INSERT INTO 테이블명 (열1, 열2) VALUES 
('값1', '값2'),
('값3', '값4');
```

### 데이터 조회

기본 조회:
```sql
SELECT * FROM 테이블명;
```

조건 조회:
```sql
SELECT 열1, 열2 FROM 테이블명 WHERE 조건;
```

정렬:
```sql
SELECT * FROM 테이블명 ORDER BY 열이름 DESC;
```

페이지네이션:
```sql
SELECT * FROM 테이블명 LIMIT 10 OFFSET 20;
```

### 데이터 업데이트

```sql
UPDATE 테이블명 SET 열1 = '새값' WHERE 조건;
```

### 데이터 삭제

```sql
DELETE FROM 테이블명 WHERE 조건;
```

테이블 비우기:
```sql
TRUNCATE TABLE 테이블명;
```

## 인덱스 작업

### 인덱스 생성

일반 인덱스:
```sql
CREATE INDEX 인덱스명 ON 테이블명 (열이름);
```

고유 인덱스:
```sql
CREATE UNIQUE INDEX 인덱스명 ON 테이블명 (열이름);
```

복합 인덱스:
```sql
CREATE INDEX 인덱스명 ON 테이블명 (열1, 열2);
```

### 인덱스 보기

```sql
SHOW INDEX FROM 테이블명;
```

### 인덱스 삭제

```sql
DROP INDEX 인덱스명 ON 테이블명;
```

## 사용자 및 권한 관리

### 사용자 생성

```sql
CREATE USER '사용자명'@'localhost' IDENTIFIED BY '비밀번호';
```

### 권한 부여

```sql
GRANT SELECT, INSERT, UPDATE ON 데이터베이스명.* TO '사용자명'@'localhost';
```

모든 권한 부여:
```sql
GRANT ALL PRIVILEGES ON 데이터베이스명.* TO '사용자명'@'localhost';
```

### 권한 새로고침

```sql
FLUSH PRIVILEGES;
```

### 사용자 권한 보기

```sql
SHOW GRANTS FOR '사용자명'@'localhost';
```

### 사용자 삭제

```sql
DROP USER '사용자명'@'localhost';
```

## 일반 함수

### 문자열 함수

```sql
SELECT CONCAT('안녕하세요', ' ', '세상');
SELECT UPPER('안녕하세요');
SELECT LOWER('안녕하세요');
SELECT SUBSTRING('안녕하세요 세상', 1, 5);
SELECT LENGTH('안녕하세요');
SELECT TRIM('  안녕하세요  ');
```

### 숫자 함수

```sql
SELECT ABS(-5);
SELECT CEIL(3.14);
SELECT FLOOR(3.14);
SELECT ROUND(3.14159, 2);
SELECT MOD(10, 3);
```

### 날짜 함수

```sql
SELECT NOW();
SELECT CURDATE();
SELECT CURTIME();
SELECT DATE('2024-01-01 10:30:00');
SELECT YEAR('2024-01-01');
SELECT MONTH('2024-01-01');
SELECT DAY('2024-01-01');
SELECT DATE_ADD('2024-01-01', INTERVAL 1 DAY);
SELECT DATEDIFF('2024-01-01', '2023-12-01');
```

### 집계 함수

```sql

SELECT AVG(열이름) FROM 테이블명;
SELECT MAX(열이름) FROM 테이블명;
SELECT MIN(열이름) FROM 테이블명;
```

## 백업 및 복원

### 데이터베이스 백업

```bash
mysqldump -u root -p 데이터베이스명 > backup.sql
```

모든 데이터베이스 백업:
```bash
mysqldump -u root -p --all-databases > all_databases.sql
```

### 데이터베이스 복원

```bash
mysql -u root -p 데이터베이스명 < backup.sql
```

### 조회 결과 내보내기

```bash
mysql -u root -p -e "SELECT * FROM 테이블명" 데이터베이스명 > output.tsv
```

## 성능 최적화

### EXPLAIN으로 조회 분석

```sql
EXPLAIN SELECT * FROM 테이블명 WHERE 조건;
```

### 최적화 팁

1. 일반적인 조회 조건에 인덱스 생성
2. SELECT * 피하고 필요한 열만 조회
3. LIMIT을 사용하여 결과 세트 제한
4. 인덱스 열에서 함수 사용 피하기
5. JOIN을 올바르게 사용하고 데카르트 곱 피하기
6. 테이블을 정기적으로 분석하고 최적화

### 슬로우 쿼리 로그 보기

```sql
SHOW VARIABLES LIKE 'slow_query_log%';
```

## 일반적인 오류 처리

### 연결 오류
MySQL 서비스가 실행 중인지, 포트가 올바른지, 사용자 권한이 올바르게 구성되어 있는지 확인하세요.

### 문자 집합 오류
데이터베이스, 테이블 및 연결이 모두 동일한 문자 집합을 사용하는지 확인하세요(utf8mb4 권장).

### 권한 오류
사용자에게 적절한 데이터베이스 및 테이블 작업 권한이 있는지 확인하세요.
