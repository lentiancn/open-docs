# Руководство пользователя MySQL

## Основные операции с базой данных

### Подключение к MySQL

Локальное подключение:
```bash
mysql -u root -p
```

Удалённое подключение:
```bash
mysql -h имя_хоста -u имя_пользователя -p имя_базы_данных
```

### Создание базы данных

```sql
CREATE DATABASE имя_базы_данных;
```

Создать с определённой кодировкой:
```sql
CREATE DATABASE имя_базы_данных CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### Просмотр баз данных

```sql
SHOW DATABASES;
```

### Выбор базы данных

```sql
USE имя_базы_данных;
```

### Удаление базы данных

```sql
DROP DATABASE имя_базы_данных;
```

## Операции с таблицами

### Создание таблицы

```sql
CREATE TABLE имя_таблицы (
    id INT AUTO_INCREMENT PRIMARY KEY,
    имя VARCHAR(100) NOT NULL,
    email VARCHAR(100) UNIQUE,
    создан TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    обновлён TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### Просмотр структуры таблицы

```sql
DESCRIBE имя_таблицы;
-- или
SHOW COLUMNS FROM имя_таблицы;
```

### Просмотр всех таблиц

```sql
SHOW TABLES;
```

### Изменение структуры таблицы

Добавить столбец:
```sql
ALTER TABLE имя_таблицы ADD COLUMN новый_столбец VARCHAR(50);
```

Изменить столбец:
```sql
ALTER TABLE имя_таблицы MODIFY имя_столбца VARCHAR(200);
```

Удалить столбец:
```sql
ALTER TABLE имя_таблицы DROP COLUMN имя_столбца;
```

### Удаление таблицы

```sql
DROP TABLE имя_таблицы;
```

## Операции с данными

### Вставка данных

Одиночная вставка:
```sql
INSERT INTO имя_таблицы (столбец1, столбец2) VALUES ('значение1', 'значение2');
```

Множественная вставка:
```sql
INSERT INTO имя_таблицы (столбец1, столбец2) VALUES 
('значение1', 'значение2'),
('значение3', 'значение4');
```

### Запрос данных

Базовый запрос:
```sql
SELECT * FROM имя_таблицы;
```

Условный запрос:
```sql
SELECT столбец1, столбец2 FROM имя_таблицы WHERE условие;
```

Сортировка:
```sql
SELECT * FROM имя_таблицы ORDER BY имя_столбца DESC;
```

Пагинация:
```sql
SELECT * FROM имя_таблицы LIMIT 10 OFFSET 20;
```

### Обновление данных

```sql
UPDATE имя_таблицы SET столбец1 = 'новое_значение' WHERE условие;
```

### Удаление данных

```sql
DELETE FROM имя_таблицы WHERE условие;
```

Очистить таблицу:
```sql
TRUNCATE TABLE имя_таблицы;
```

## Операции с индексами

### Создание индексов

Обычный индекс:
```sql
CREATE INDEX имя_индекса ON имя_таблицы (имя_столбца);
```

Уникальный индекс:
```sql
CREATE UNIQUE INDEX имя_индекса ON имя_таблицы (имя_столбца);
```

Составной индекс:
```sql
CREATE INDEX имя_индекса ON имя_таблицы (столбец1, столбец2);
```

### Просмотр индексов

```sql
SHOW INDEX FROM имя_таблицы;
```

### Удаление индексов

```sql
DROP INDEX имя_индекса ON имя_таблицы;
```

## Управление пользователями и правами

### Создание пользователя

```sql
CREATE USER 'имя_пользователя'@'localhost' IDENTIFIED BY 'пароль';
```

### Предоставление прав

```sql
GRANT SELECT, INSERT, UPDATE ON имя_базы_данных.* TO 'имя_пользователя'@'localhost';
```

Предоставить все права:
```sql
GRANT ALL PRIVILEGES ON имя_базы_данных.* TO 'имя_пользователя'@'localhost';
```

### Обновление прав

```sql
FLUSH PRIVILEGES;
```

### Просмотр прав пользователя

```sql
SHOW GRANTS FOR 'имя_пользователя'@'localhost';
```

### Удаление пользователя

```sql
DROP USER 'имя_пользователя'@'localhost';
```

## Управление транзакциями

### Начало транзакции

```sql
START TRANSACTION;
-- или
BEGIN;
```

### Фиксация транзакции

```sql
COMMIT;
```

### Откат транзакции

```sql
ROLLBACK;
```

### Установка точек сохранения

```sql
SAVEPOINT имя_точки;
ROLLBACK TO SAVEPOINT имя_точки;
```

## Часто используемые функции

### Строковые функции

```sql
SELECT CONCAT('Привет', ' ', 'Мир');
SELECT UPPER('привет');
SELECT LOWER('ПРИВЕТ');
SELECT SUBSTRING('Привет Мир', 1, 5);
SELECT LENGTH('Привет');
SELECT TRIM('  Привет  ');
```

### Числовые функции

```sql
SELECT ABS(-5);
SELECT CEIL(3.14);
SELECT FLOOR(3.14);
SELECT ROUND(3.14159, 2);
SELECT MOD(10, 3);
```

### Функции даты

```sql
SELECT NOW();
SELECT CURDATE();
SELECT CURTIME();
SELECT DATE('2024-01-01 10:30:00');
SELECT YEAR('2024-01-01');
SELECT MONTH('2024-01-01');
SELECT DAY('2024-01-01');
SELECT DATE_ADD('2024-01-01', INTERVAL 1 DAY);
SELECT DATEDIFF('2024-01-01', '2023-12-01');
```

### Агрегатные функции

```sql
SELECT COUNT(*) FROM имя_таблицы;
SELECT SUM(имя_столбца) FROM имя_таблицы;
SELECT AVG(имя_столбца) FROM имя_таблицы;
SELECT MAX(имя_столбца) FROM имя_таблицы;
SELECT MIN(имя_столбца) FROM имя_таблицы;
```

## Резервное копирование и восстановление

### Резервное копирование базы данных

```bash
mysqldump -u root -p имя_базы_данных > backup.sql
```

Резервное копирование всех баз данных:
```bash
mysqldump -u root -p --all-databases > all_databases.sql
```

### Восстановление базы данных

```bash
mysql -u root -p имя_базы_данных < backup.sql
```

### Экспорт результатов запроса

```bash
mysql -u root -p -e "SELECT * FROM имя_таблицы" имя_базы_данных > output.tsv
```

## Оптимизация производительности

### Анализ запросов с EXPLAIN

```sql
EXPLAIN SELECT * FROM имя_таблицы WHERE условие;
```

### Советы по оптимизации

1. Создавайте индексы для распространённых условий запроса
2. Избегайте SELECT *, запрашивайте только нужные столбцы
3. Используйте LIMIT для ограничения набора результатов
4. Избегайте использования функций на индексированных столбцах
5. Правильно используйте JOIN, избегайте декартовых произведений
6. Регулярно анализируйте и оптимизируйте таблицы

### Просмотр журнала медленных запросов

```sql
SHOW VARIABLES LIKE 'slow_query_log%';
```

## Обработка распространённых ошибок

### Ошибки подключения
Проверьте, запущена ли служба MySQL, правильный ли порт, правильно ли настроены права пользователя.

### Ошибки кодировки
Убедитесь, что база данных, таблица и подключение используют одну и ту же кодировку (рекомендуется utf8mb4).

### Ошибки прав
Подтвердите, что пользователь имеет соответствующие права на операции с базой данных и таблицей.
