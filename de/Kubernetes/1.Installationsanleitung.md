# Kubernetes Installationsanleitung

Installation von Kubernetes.

## Methoden

| Methode | Beschreibung |
|---------|--------------|

## minikube

```bash
brew install minikube
minikube start
```

---

## kubectl

```bash
brew install kubectl
```

---

## Prüfen

```bash
kubectl get nodes
```
