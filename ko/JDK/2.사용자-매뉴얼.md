# JDK 사용자 매뉴얼

이 가이드에서는 Java 개발을 위해 JDK를 사용하는 방법을 설명합니다.

## 빠른 시작

### 컴파일 및 실행

```bash
# 컴파일
javac HelloWorld.java

# 실행
java HelloWorld
```

### 단일 파일 실행 (JDK 11+)

```bash
java HelloWorld.java
```

## 기본 명령어

### javac - Java 컴파일러

```bash
javac MyClass.java
javac *.java
javac -encoding UTF-8 MyClass.java
javac -d bin MyClass.java
javac -Xlint MyClass.java
```

### java - Java 런처

```bash
java MyClass
java MyClass arg1 arg2
java -classpath "lib/*:." MyClass
java -Xmx2g -Xms512m MyClass
```

### jar - Java 아카이브

```bash
jar cf myapp.jar -C bin .
jar cfm myapp.jar manifest.txt -C bin .
jar xf myapp.jar
jar tf myapp.jar
```

### javadoc - 문서 생성기

```bash
javadoc -d docs -sourcepath src *.java
```

### jdb - Java 디버거

```bash
jdb MyClass
stop at MyClass:10
run
next
step
```

## JVM 옵션

### 메모리 옵션

```bash
-Xmx2g        # 최대 힙
-Xms512m      # 초기 힙
-Xss256k      # 스택 크기

# 가비지 컬렉션
-XX:+UseG1GC
-XX:+UseParallelGC
```

### 성능 옵션

```bash
-XX:+TieredCompilation
-Xshare:on
-XX:+UseStringDeduplication
```

## IDE 통합

### IntelliJ IDEA

1. File → Project Structure → SDKs
2. JDK 설치 경로 추가

### VS Code

1. "Extension Pack for Java" 설치
2. 설정에서 JAVA_HOME 구성

### Eclipse

1. Window → Preferences → Java → Installed JREs

## 빌드 도구

### Maven

```xml
<properties>
    <maven.compiler.source>21</maven.compiler.source>
    <maven.compiler.target>21</maven.compiler.target>
</properties>
```

### Gradle

```groovy
sourceCompatibility = '21'
targetCompatibility = '21'
```

## 모듈 (Java 9+)

```bash
java --list-modules
```

## 보안

### Keytool

```bash
keytool -genkeypair -alias mykey -keyalg RSA -keystore keystore.jks
keytool -list -keystore keystore.jks
```

## 프로파일링

### jcmd

```bash
jcmd
jcmd <pid> VM.flags
jcmd <pid> GC.run
```

## 모범 사례

```bash
# 권장: JDK 17 또는 21
```

### 2. 적절한 메모리 설정

```bash
java -Xmx4g -Xms2g -XX:+UseG1GC MyApp
```

```bash
-Xlog:gc*:file=gc.log:time,uptime,level,tags
```
