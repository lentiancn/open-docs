# Guide d'installation JMeter

JMeter est un outil de test de performance open source développé par l'organisation Apache. Ce guide couvre les méthodes d'installation dans différents environnements.

## Informations de version

| Version | Statut | Description |
|------|------|------|
| JMeter 5.x | ✅ Actuelle | Dernière version |

## Configuration requise

### Matériel

| Exigence | Configuration |
|------|------|
| CPU | 2+ cœurs |
| Mémoire | 4+ Go |
| Disque | 5+ Go |

### Logiciel

| Logiciel | Version |
|------|------|
| Java | JDK 8 ou supérieur |

## Méthodes d'installation

### Méthode 1 : Installation Windows

#### 1. Téléchargement

1. Visitez https://jmeter.apache.org/download.cgi
2. Téléchargez la version Binaries
3. Extrayez dans le répertoire souhaité

#### 2. Exécution

```powershell
# Double-cliquez sur bin\jmeter.bat
# Ou exécutez en ligne de commande
jmeter.bat
```

### Méthode 2 : Installation macOS

#### 1. Utilisation de Homebrew

```bash
brew install jmeter
```

#### 2. Installation manuelle

```bash
# Téléchargez et extrayez
wget https://archive.apache.org/dist/jmeter/binaries/apache-jmeter-5.6.3.tgz
tar -xzf apache-jmeter-5.6.3.tgz

# Exécutez
./apache-jmeter-5.6.3/bin/jmeter
```

### Méthode 3 : Installation Linux

```bash
# Ubuntu/Debian
sudo apt-get install jmeter

# Ou télécharger et installer
wget https://archive.apache.org/dist/jmeter/binaries/apache-jmeter-5.6.3.tgz
tar -xzf apache-jmeter-5.6.3.tgz
cd apache-jmeter-5.6.3/bin
./jmeter
```

### Méthode 4 : Docker

```bash
# Exécuter le conteneur JMeter
docker run -v $(pwd):/workspace justb4/jmeter -n -t test.jmx -l result.jtl
```

## Configuration de l'environnement

### Configuration Java

```bash
# Vérifier Java
java -version

# Définir les variables d'environnement
export JAVA_HOME=/path/to/java
export PATH=$JAVA_HOME/bin:$PATH
```

## Questions fréquentes

### Mémoire insuffisante ?

Modifiez jmeter.bat ou jmeter :

```bash
# Définir la taille du tas
HEAP="-Xms1g -Xmx2g"
```

### Démarrage lent de l'GUI ?

Utilisez le mode ligne de commande ou augmentez la mémoire.
