# Yarn 包管理器

Yarn 是一个成熟的开源包管理器，用于管理 JavaScript 项目中的依赖关系。它协助处理安装、更新、配置和移除包依赖的过程，最终帮助您更快地达成目标，减少干扰。

Yarn 的重点领域是速度、正确性、安全性和开发者体验，通过利用一系列创新功能（如工作区、离线缓存、并行安装、强化模式、交互式命令等）在每个方面都得到改进。

鉴于其强大的功能和卓越的记录，Yarn 证明了它是 JavaScript 生态系统中不可或缺的资产，在任何规模的项目中都能推动效率、安全性和积极的开发者体验。

## 主版本对比

### Yarn v4+ (现代版本)
- **架构**: 插件化架构 (Berry)
- **特点**: 工作区原生支持、PnP (Plug'n'Play)、离线缓存、并行安装、语义化错误提示
- **安装方式**: 推荐使用 Corepack
- **状态**: 活跃开发中

### Yarn v1 (Classic)
- **架构**: 传统架构
- **特点**: 简单直接，广泛兼容，与 npm 完全兼容
- **安装方式**: 通过 npm 全局安装
- **状态**: 维护中

> 注意：与其他大多数包管理器不同，Yarn 重新实现了所有命令，以完全控制开发者体验和稳定性。

## 核心优势

Yarn 相比其他包管理器具有以下核心优势：

- **速度**: 并行安装和离线缓存显著提升安装速度
- **确定性**: yarn.lock 文件确保依赖的一致性
- **安全性**: 内置审计功能和依赖验证
- **开发者体验**: 丰富的命令集和清晰的错误信息
- **现代化特性**: 工作区、PnP、插件系统等先进功能
