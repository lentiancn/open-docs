# Руководство по установке

В этом документе описаны различные способы установки Apifox, чтобы вы могли быстро начать работу с инструментом на локальном компьютере.

## Системные требования

Apifox поддерживает следующие операционные системы:

- Windows 7 и выше (64-бит)
- macOS 10.13 и выше
- Linux (Debian/Ubuntu 20.04+, Fedora 36+)

## Скачивание

### Скачивание с официального сайта

Перейдите на сайт Apifox по адресу https://apifox.com/ и нажмите кнопку «Скачать», затем выберите версию для вашей операционной системы.

### Выбор версии

Apifox предлагает две версии:

- **Apifox** (стандартная версия): включает все основные функции, подходит для большинства пользователей
- **Apifox** Plus (платная версия): содержит дополнительные продвинутые функции, такие как управление командой и расширенная автоматизация

Для базовых задач вполне хватает стандартной версии.

## Установка

### Установка в Windows

1. Скачайте установочный файл для Windows (.exe)
2. Дважды кликните по файлу, чтобы запустить установку
3. Выберите путь для установки (рекомендуется оставить путь по умолчанию)
4. Дождитесь завершения установки
5. Запустите Apifox и войдите в аккаунт или зарегистрируйтесь

### Установка на macOS

1. Скачайте установочный файл для macOS (.dmg)
2. Откройте установочный пакет и перетащите Apifox в папку Applications
3. Запустите Apifox и войдите в аккаунт или зарегистрируйтесь

При первом запуске может потребоваться разрешить запуск приложений от неизвестных разработчиков в системных настройках.

### Установка на Linux

**Debian/Ubuntu:**

```bash
# Скачивание deb-пакета
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.deb

# Установка
sudo dpkg -i Apifox-latest-linux.x64.deb

# Если есть проблемы с зависимостями, исправляем
sudo apt-get install -f
```

**Fedora:**

```bash
# Скачивание rpm-пакета
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.rpm

# Установка
sudo rpm -i Apifox-latest-linux.x64.rpm
```

## Регистрация и вход

### Регистрация аккаунта

Для первого использования нужно зарегистрировать аккаунт:

1. Запустите Apifox
2. Нажмите кнопку «Регистрация»
3. Введите email и пароль
4. Подтвердите email (если требуется)

### Вход в систему

Доступны несколько способов входа:

- Вход по email и паролю
- Вход через сторонние сервисы (GitHub, Google и другие)

### Синхронизация с облаком

После входа в аккаунт данные проекта автоматически синхронизируются с облаком. Можно без проблем переключаться между устройствами — данные всегда актуальны.

## Первоначальная настройка

После первого запуска рекомендуется выполнить следующие настройки:

### Выбор темы

В «Настройках» можно выбрать светлую или темную тему — кому как удобнее.

### Настройка прокси

Если нужно подключаться через прокси, настройте прокси-сервер в разделе «Настройки» → «Сеть».

### Импорт и экспорт

- **Импорт**: поддерживается импорт проектов в форматах Postman, Swagger, Har и других
- **Экспорт**: можно экспортировать в форматах OpenAPI, Markdown, HTML и других

## Быстрый старт

### Создание проекта

1. Нажмите «+» рядом со списком «Проекты» в левой панели
2. Введите название проекта
3. Выберите тип проекта (API-проект или Mock-проект)

### Создание интерфейса

1. В проекте нажмите «+» и выберите «Новый интерфейс»
2. Заполните основную информацию:
   - Название интерфейса
   - Путь запроса
   - Метод запроса (GET/POST/PUT/DELETE и другие)
3. Заполните параметры запроса, тело запроса, ответ и сохраните интерфейс

### Отладка интерфейса

1. Откройте созданный интерфейс
2. Нажмите кнопку «Отправить»
3. Просмотрите ответ

### Использование Mock-данных

1. Определите структуру данных в интерфейсе
2. Система автоматически сгенерирует Mock-адрес
3. Перейдите по Mock-адресу, чтобы получить тестовые данные

## Удаление

### Удаление в Windows

- Откройте «Панель управления» → «Программы и компоненты»
- Найдите Apifox и нажмите «Удалить»

### Удаление на macOS

- Перетащите Apifox из папки Applications в Корзину
- Очистите Корзину

### Удаление на Linux

```bash
# Debian/Ubuntu
sudo apt-get remove apifox

# Fedora
sudo rpm -e apifox
```
