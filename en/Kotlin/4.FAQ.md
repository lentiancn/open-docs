# Kotlin FAQ

## General Questions

### What is Kotlin?
Kotlin is a modern, statically typed programming language developed by JetBrains. It runs on the JVM and can be compiled to JavaScript or native code.

### Is Kotlin free?
Yes, Kotlin is open source under the Apache 2 license.

### What is Kotlin used for?
- Android development (official language)
- Backend development with frameworks like Ktor, Spring
- Frontend development (compiles to JavaScript)
- Multiplatform mobile development (Kotlin Multiplatform)
- Data science

## Comparison Questions

### Kotlin vs Java?
- Kotlin is more concise
- Kotlin has null safety
- Kotlin has data classes, sealed classes
- Kotlin supports coroutines for async
- Kotlin has extension functions

### Can I use Java libraries in Kotlin?
Yes, 100% interoperability. You can call any Java code from Kotlin.

### Can I use Kotlin in existing Java projects?
Yes, Kotlin and Java can coexist in the same project.

## Syntax Questions

### How do I create a main function in Kotlin?

```kotlin
fun main() {
    println("Hello, Kotlin!")
}
```

### How do I declare variables?

```kotlin
// Immutable
val name: String = "Kotlin"

// Mutable
var count: Int = 0
```

### How do I create a data class?

```kotlin
data class User(val name: String, val age: Int)
```

### How do null safety work?

```kotlin
// Non-nullable
var name: String = "Kotlin"

// Nullable
var nullableName: String? = null

// Safe call
val length = nullableName?.length

// Elvis operator
val len = nullableName?.length ?: 0
```

### How do I use coroutines?

```kotlin
suspend fun fetchData() {
    val data = async { apiCall() }.await()
}
```

## Installation Questions

### How do I install Kotlin?
- **IntelliJ IDEA**: Built-in support
- **Eclipse**: Install Kotlin plugin
- **Command Line**: Download from kotlinlang.org

### How do I use Kotlin with Maven/Gradle?
Add the Kotlin plugin and dependencies to your build file.
