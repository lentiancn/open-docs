# Element UI 소개

Element UI는 개발자에게 풍부한 데스크톱 컴포넌트 솔루션을 제공하는 Vue 2.0 기반 UI 컴포넜ent 라이브러리입니다.

## Element UI란

Element UI는 Ele.me 팀에서 개발하고 관리하는 Vue 2.0 컴포넌트 라이브러리입니다. 개발자가 데스크톱 애플리케이션을 빠르게 구축할 수 있도록 완전한 디자인 사양과 컴포넌트 라이브러리를 제공합니다.

Vue 2.0 동반 UI 프레임워크로서, Element UI는 간단하고 사용하기 쉬운 컴포넌트 캡슐화와 좋은 개발 경험을 제공합니다.

## 주요 기능

### 완전한 컴포넌트 시스템

Element UI는 데스크톱 애플리케이션 개발의 모든 측면을 다루는 70개 이상의 고품질 컴포넌트를 제공합니다:

- **기본 컴포넌트**: 버튼, 아이콘, 레이아웃, 컨테이너
- **폼 컴포넌트**: 입력, 선택, 날짜 선택기, 스위치, 슬라이더
- **데이터 표시**: 테이블, 트리, 페이지네이션, 태그
- **피드백 컴포넌트**: 대화상자, 드로어, 메시지, 로딩
- **네비게이션 컴포넌트**: 메뉴, 탭, 브레드크럼, 스텝

### 완전한 Vue 2.0 지원

Element UI는 Vue 2.0 위에 완전히 구축되었습니다:

- 컴포넌트 기반 개발
- 반응형 데이터 바인딩
- 완전한 TypeScript 지원
- 좋은 호환성

### 필요 시 가져오기

Element UI는 사용하는 컴포넌트만 가져오기를 지원합니다:

- babel-plugin-component 사용
- 수동 필요 시 가져오기
- 번들 크기 감소

### 국제화 지원

다중 언어 패키지에 대한 내장 지원.

### 테마 사용자 정의

전역 테마 사용자 정의를 지원합니다:

## 디자인 원칙

Element UI는 네 가지 주요 디자인 원칙을 따릅니다:

### 일관성

- 실제生活和 일관성: 실제 프로세스 및 로직과 일관성을 유지
- 인터페이스에서 일관성: 모든 요소와 구조가 일관되어야 함

### 피드백

- 제어 피드백: 인터페이스 스타일을 통해 사용자가 작업을 명확하게 인식하도록 함
- 페이지 피드백: 작업 후 현재 상태를 명확하게 표시

### 효율성

- 프로세스 단순화: 간단하고 직관적인 작업 프로세스 설계
- 명확하고 명시적: 빠른 이해를 위한 명확한 언어 표현
- 사용자가 인식하도록 돕기: 간단한 인터페이스로 기억 부담 감소

### 통제 가능성

- 사용자 결정: 제안은 제공하지만 사용자를 대신하여 결정하지 않음
- 통제 가능한 결과: 사용자가 자유롭게 작업 수행

## 적용 시나리오

Element UI는 다음에 적합합니다:

- 기업 내부 관리 시스템
- 전자상거래 백엔드 관리 시스템
- 데이터 시각화 플랫폼
- CRM 고객 관리 시스템
- Vue 2 데스크톱 UI가 필요한 모든 프로젝트

## Element Plus와의 관계

Element UI는 Element Plus(Vue 3 버전)의 조상입니다:

- Element UI: Vue 2용으로 설계됨
- Element Plus: Vue 3용으로 다시 설계됨

프로젝트에서 Vue 2를 사용 중이면 Element UI를 계속 사용하세요. Vue 3로 업그레이드하면 Element Plus로 마이그레이션하세요.
