# H2 資料庫安裝指南

## 環境要求

- Java 11 或更高版本
- 任何支援 Java 的作業系統（Windows、Linux、macOS 等）

## 安裝方式

### 方式一：下載 ZIP 包

1. 訪問官方網站下載頁面：https://www.h2database.com/html/download.html
2. 下載「Platform-Independent Zip」版本
3. 解壓下載的 ZIP 文件到任意目錄，例如：C:\h2 或 /opt/h2

### 方式二：Maven 依賴

如果你是使用 Maven 或 Gradle 的 Java 項目，可以直接添加依賴：

**Maven：**
```xml
<dependency>
    <groupId>com.h2database</groupId>
    <artifactId>h2</artifactId>
    <version>2.2.224</version>
</dependency>
```

**Gradle：**
```groovy
implementation 'com.h2database:h2:2.2.224'
```

### 方式三：Windows 安裝程式

1. 下載「Windows Installer」版本
2. 執行安裝程式，按照精靈完成安裝

## 配置環境變數（可選）

如果你下載了 ZIP 包並希望在命令列中直接使用 H2：

**Windows：**
1. 右鍵點擊「此電腦」→「內容」
2. 點擊「進階系統設定」
3. 點擊「環境變數」
4. 在系統變數中找到 Path，點擊「編輯」
5. 新增 H2 的 bin 目錄路徑

**Linux/macOS：**
```bash
export PATH=$PATH:/opt/h2/bin
```

## 驗證安裝

### 使用命令列

解壓後，在命令列中執行：

```bash
java -jar h2.jar
```

這將啟動 H2 控制台 Web 介面。

### 使用 JDBC 連線

在 Java 代碼中連線 H2 資料庫：

```java
// 嵌入式模式
String url = "jdbc:h2:mem:testdb";
Connection conn = DriverManager.getConnection(url, "sa", "");

// 檔案資料庫
String url = "jdbc:h2:file:./data/testdb";
Connection conn = DriverManager.getConnection(url, "sa", "password");
```

### 使用 H2 控制台

1. 執行命令：`java -jar h2.jar`
2. 開啟瀏覽器訪問：http://localhost:8082
3. 預設使用者名稱：sa，密碼為空

## 連線 URL 格式

H2 支援多種連線 URL 格式：

| 模式 | URL 示例 | 說明 |
|------|---------|------|
| 記憶體模式 | jdbc:h2:mem:testdb | 資料存儲在記憶體中 |
| 嵌入式檔案 | jdbc:h2:file:./data/testdb | 資料存儲在檔案中 |
| TCP 伺服器 | jdbc:h2:tcp://localhost:9092/testdb | 客戶端-伺服器模式 |
| TCP 加密 | jdbc:h2:ssl://localhost:9092/testdb | SSL 加密連線 |
| PostgreSQL 相容 | jdbc:h2:postgres://localhost:5432/testdb | PostgreSQL 協定相容模式 |

## 常見問題

### 連接埠被佔用
H2 預設使用 9092 連接埠（TCP 模式）和 8082 連接埠（Web 控制台）。如果連接埠被佔用，可以在連線 URL 中指定其他連接埠。

### Java 版本問題
確保安裝了 Java 11 或更高版本。可以通過以下命令檢查：

```bash
java -version
```

### 驅動程式未找到
確保 H2 的 JAR 檔案在 classpath 中。如果是 Maven 項目，確保依賴已正確添加。
