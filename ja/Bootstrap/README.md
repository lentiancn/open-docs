# Bootstrap 5

Bootstrapは、世界で最も人気のあるフロントエンドオープンソースツールキットで、レスポンシブでモバイルファーストのウェブサイトを構築するために使用されます。

## 简介

Bootstrap 5は最新バージョンで、完全にネイティブJavaScriptで書き直されており、jQueryに依存しません。豊富なCSSとJavaScriptコンポーネントを提供し、開発者が迅速にモダンヴェブサイトを構築するのに役立ちます。

## 主な特徴

- **レスポンシブデザイン**：組み込みレスポンシブグリッドシステム
- **豊富なコンポーネント**：ボタン、ナビゲーション、フォーム、カード、モadalなど
- **強力なユーティリティクラス**：余白、色、テキスト、表示などの実用的なツール
- **Sassサポート**：Sassを使用したテーマのカスタマイズが可能
- **アイコンライブラリ**：Bootstrap Iconsを統合
- **jQuery不要**：完全にネイティブJavaScriptを使用

## クイックスタート

### CDN読み込み

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
```

### npmインストール

```bash
npm install bootstrap@5.3.3
```
