# FantasyTalking Installationsanleitung

FantasyTalking ist ein KI-gestütztes Tool zur Generierung von Talking Heads, das realistische Talking-Head-Videos aus einem einzelnen Bild und Audio erstellt.

## Versionsinformationen

| Version | Status | Beschreibung |
|---------|--------|-------------|
| FantasyTalking v1.0 | ✅ Aktuell | Erste Veröffentlichung |

## Systemanforderungen

## Installationsmethoden

### Methode 1: Aus dem Quellcode

#### 1. Repository klonen

```bash
git clone https://github.com/FantasyTalking/FantasyTalking.git
cd FantasyTalking
```

#### 2. Virtuelle Umgebung erstellen

```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
# oder
venv\Scripts\activate  # Windows
```

#### 3. Abhängigkeiten installieren

```bash
pip install torch torchvision
pip install -r requirements.txt
```

#### 4. Modelle herunterladen

```bash
# Vortrainierte Modelle herunterladen
bash scripts/download_models.sh
```

### Methode 2: Docker

```bash
# Image abrufen
docker pull fantasytalking/fantasytalking:latest

# Ausführen
docker run --gpus all -v /path/to/data:/data fantasytalking/fantasytalking:latest
```

## Konfiguration

### Umgebungsvariablen

```bash
export CUDA_VISIBLE_DEVICES=0
export PYTHONPATH=/path/to/FantasyTalking:$PYTHONPATH
```

### Modellpfade

Bearbeiten Sie `config.yaml`, um Modellpfade festzulegen:

```yaml
model:
  checkpoint_path: "checkpoints/fantasytalking.pth"
  face_parser: "models/face_parser.pth"
```

## Schnellstart

### Talking Head generieren

```bash
python inference.py \
  --source_image path/to/image.jpg \
  --audio path/to/audio.wav \
  --output output.mp4
```

### Parameter

```bash
--source_image    # Quellgesichtsbild
--audio          # Eingabe-Audiodatei
--output         # Ausgabevideopfad
--face enhancer  # Gesichtsverbesserung aktivieren
--fps            # Ausgabe-FPS (Standard: 25)
```

## Häufige Probleme

### GPU nicht verfügbar?

CUDA-Installation überprüfen:

```bash
nvidia-smi
python -c "import torch; print(torch.cuda.is_available())"
```

### Nicht genügend Speicher?

Batch-Größe reduzieren oder niedrigere Auflösung verwenden.
