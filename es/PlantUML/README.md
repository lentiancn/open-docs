# PlantUML

Herramienta de código abierto para dibujar diagramas UML

## Descripción

PlantUML permite crear varios diagramas a partir de texto simple.

## Inicio Rápido

```plantuml
@startuml
Alice -> Bob: Hola
@enduml
```
