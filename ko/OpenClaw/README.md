# OpenClaw

> 고급 자동화를 위한 AI 어시스턴트 프레임워크

---

## 개요

OpenClaw는 고급 에이전트 기술을 통해 작업을自動化하도록 설계된 다용도 AI 어시스턴트 프레임워크입니다.

## 빠른 시작

### 설치

```bash
npm install -g openclaw
```

### Gateway 시작

```bash
openclaw gateway start
```

### 시작하기

```bash
openclaw chat
```

## 기능

- 지능형 에이전트
- 브라우저 자동화
- 파일 관리
- 메시징
- 도구 확장성
