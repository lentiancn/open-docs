# FantasyTalking ユーザーマニュアル

## 基本的な機能

FantasyTalkingは包括的な音声合成と対話機能を提供します。このマニュアルでは、各機能の詳しい使用方法を説明します。

### 音声合成

音声合成はFantasyTalkingの中核機能で、テキストを自然で滑らかな音声に変換します。使用例：

```javascript
// シンプルなテキストから音声へ
const audio = await client.speak('こんにちは、FantasyTalkingへようこそ');

// 声のスタイルを指定
const audio = await client.speak('今日はいい天気ですね', {
  voice: {
    style: 'happy',
    language: 'ja-JP'
  }
});

// 速度とピッチを指定
const audio = await client.speak('しばらくお待ちください', {
  voice: {
    speed: 1.2,
    pitch: 0.9
  }
});
```

### SSMLマークアップ言語

FantasyTalkingはSSMLマークアップ言語をサポートし、音声出力をより精细に制御できます：

```xml
<speak>
  <p>
    <s>最初の文。</s>
    <s>2番目の文。</s>
  </p>
  <break time="500ms"/>
  <p>
    <emphasis level="moderate">これは重要なコンテンツです</emphasis>
  </p>
</speak>
```

### 音声イベント

音声再生中にさまざまなイベントをリistenしてインタラクションを実現できます：

```javascript
// 音声開始をリisten
client.on('speech-start', (event) => {
  console.log('音声の再生が開始されました');
});

// 単語境界をリlisten（字幕同期用）
client.on('word-boundary', (event) => {
  console.log(`現在の単語: ${event.word}`);
});

// 音声終了をリisten
client.on('speech-end', (event) => {
  console.log('音声の再生が完了しました');
});

// エラーをリisten
client.on('error', (event) => {
  console.error('エラー:', event.error);
});
```

### バッチ合成

大量のテキストを合成する必要がある場合は、バッチ合成機能を使用して効率を向上できます：

```javascript
const texts = [
  'これは最初の音声です',
  'これは2番目の音声です',
  'これは3番目の音声です'
];

// 並列合成
const results = await Promise.all(
  texts.map(text => client.speak(text))
);
```

## 対話機能

### 基本的な対話

FantasyTalkingは強力な対話管理機能を提供し、マルチターン対話とコンテキストメモリをサポートします：

```javascript
// 対話セッションを作成
const session = client.createSession({
  systemPrompt: 'あなたは質問答え合うriendlyなアシスタントです',
  language: 'ja-JP'
});

// 最初のターン
const response1 = await session.chat('今日の天気はどうですか？');
console.log(response1.text);
console.log(response1.audio);

// 2番目のターン（コンテキストを自動包含）
const response2 = await session.chat('明日又如何ですか？');
console.log(response2.text);
```

### 対話設定

異なる対話シナリオに合わせて設定できます：

```javascript
const session = client.createSession({
  dialogue: {
    temperature: 0.7,
    maxTokens: 500,
    tags: ['カスタマーサービス', '事前相談']
  },
  voice: {
    style: 'professional',
    speed: 1.0
  }
});
```

## 上級機能

### ストリーミング音声合成

リアルタイムフィードバックが必要なシナリオでは、ストリーミング音声合成を使用できます：

```javascript
const stream = await client.speakStream('これは長いテキストです');

stream.on('data', (chunk) => {
  audioPlayer.play(chunk);
});

stream.on('end', () => {
  console.log('音声合成完了');
});
```

### カスタム音声モデル

ブランド専用の声が必要な場合、カスタムモデルをトレーニングできます：

```javascript
const trainingData = {
  name: 'ブランドボイス',
  description: '企業ブランド用の声',
  samples: [
    { text: 'サンプルテキスト1', audio: 'base64エンコードされたオーディオ1' },
    { text: 'サンプルテキスト2', audio: 'base64エンコードされたオーディオ2' }
  ]
};

const job = await client.trainVoice(trainingData);

while (job.status !== 'completed') {
  await sleep(5000);
  const status = await client.getJobStatus(job.id);
  console.log(`トレーニング進捗: ${status.progress}%`);
}
```

## ベストプラクティス

### パフォーマンス最適化

実際のアプリケーションでは、パフォーマンスを最適化するために以下に注意してください：

1. **接続の再利用**: クライアントインスタンスを頻繁に作成しないでください
2. **バッチ処理**: 大量のテキストを処理する場合はバッチ合成を使用
3. **キャッシュ**: 同一のテキストにはキャッシュを使用して重複を回避

### エラー処理

包括的なエラー処理を実装することをお勧めします：

```javascript
try {
  const audio = await client.speak('テストテキスト');
} catch (error) {
  if (error.code === 'RATE_LIMIT') {
    console.log('リクエストが多すぎます、後で再試行してください');
  } else if (error.code === 'INVALID_API_KEY') {
    console.log('APIキーが無効です');
  }
}
```
