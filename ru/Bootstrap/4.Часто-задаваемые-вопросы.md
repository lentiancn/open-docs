# Часто задаваемые вопросы

В этом документе собраны распространённые технические вопросы и ответы по использованию Bootstrap.

## Основные вопросы

### Что такое Bootstrap?

Bootstrap — это открытый фреймворк для фронтенд-разработки, созданный и поддерживаемый компанией Twitter. Он предлагает богатый набор CSS-стилей и JavaScript-компонентов, которые помогают разработчикам быстро создавать адаптивные веб-приложения.

### Чем Bootstrap 5 отличается от Bootstrap 4?

Основные отличия:

- Bootstrap 5 больше не требует jQuery, использует чистый JavaScript
- Добавлена контрольная точка XXL
- Улучшены элементы форм
- Убрана поддержка Font Awesome, теперь используются Bootstrap Icons
- больше не поддерживается Internet Explorer

### Какие требования к окружению для Bootstrap?

Достаточно современного браузера. Bootstrap 5 требует браузер с поддержкой ES6.

## Проблемы с установкой

### Не загружается CDN?

1. Проверьте подключение к интернету
2. Попробуйте другой CDN (например, bootcdn, staticfile)
3. Скачайте файлы локально

### Ошибка при установке через npm?

```bash
# Очистить кэш
npm cache clean --force

# Использовать зеркало Taobao
npm config set registry https://registry.npmmirror.com

# Переустановить
npm install bootstrap@5
```

### Стили не применяются?

1. Убедитесь, что CSS-файл подключён правильно
2. Очистите кэш браузера (принудительное обновление Ctrl+Shift+R)
3. Проверьте, нет ли конфликтов CSS

## Проблемы с использованием

### Сетка не работает?

1. Убедитесь, что row находится внутри container
2. Убедитесь, что col находится внутри row
3. Проверьте правильность использования классов контрольных точек

### Адаптивность не работает?

Добавьте мета-тег viewport:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### Навигационная панель не сворачивается на мобильных?

1. Убедитесь, что подключён bootstrap.bundle.min.js (включает Popper)
2. Проверьте атрибуты data-bs-toggle и data-bs-target

### Модальное окно не открывается?

1. Убедитесь, что подключён Bootstrap JS
2. Проверьте атрибуты data-bs-toggle и data-bs-target на кнопке
3. Убедитесь, что ID модального окна указан правильно

### Выпадающий список не работает?

1. Подключите bootstrap.bundle.min.js
2. Проверьте атрибут data-bs-toggle="dropdown"

## Проблемы со стилями

### Как изменить цвета темы?

Используйте Sass:

```scss
$primary: #007bff;
@import "bootstrap/scss/bootstrap";
```

### Как переопределить стили?

Используйте более специфичный селектор:

```css
.navbar {
    background-color: #f8f9fa !important;
}
```

### Как изменить шрифт?

```scss
$font-family-sans-serif: "Helvetica Neue", Helvetica, Arial, sans-serif;
@import "bootstrap/scss/bootstrap";
```

### Иконки не отображаются?

В Bootstrap 5 используются Bootstrap Icons:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.10.0/font/bootstrap-icons.css">

<i class="bi bi-alarm"></i>
```

## Проблемы с компонентами

### Как отцентровать карточку?

```html
<div class="card">
    <div class="card-body">
        Содержимое
    </div>
</div>
```

Или используя flex:

```html
<div class="card">
    <div class="card-body d-flex justify-content-center align-items-center">
        Центрированное содержимое
    </div>
</div>
```

### Валидация формы не работает?

В Bootstrap 5 новая система валидации:

```html
<form class="row g-3 needs-validation" novalidate>
    <div class="col-md-4">
        <label for="validationCustom01" class="form-label">Имя</label>
        <input type="text" class="form-control" id="validationCustom01" required>
        <div class="valid-feedback">Верно!</div>
        <div class="invalid-feedback">Пожалуйста, введите имя</div>
    </div>
    <div class="col-12">
        <button class="btn btn-primary" type="submit">Отправить</button>
    </div>
</form>

<script>
    (function () {
        'use strict'
        var forms = document.querySelectorAll('.needs-validation')
        Array.prototype.slice.call(forms).forEach(function (form) {
            form.addEventListener('submit', function (event) {
                if (!form.checkValidity()) {
                    event.preventDefault()
                    event.stopPropagation()
                }
                form.classList.add('was-validated')
            }, false)
        })
    })()
</script>
```

### Как закрыть модальное окно при клике на фон?

Настройте атрибут:

```html
<div class="modal" data-bs-backdrop="static">
```

`static` — клик по фону не закрывает
`true` — клик закрывает
`false` — клик не закрывает

## Проблемы с производительностью

### Как уменьшить размер файлов?

1. Подключайте только нужные компоненты (через Sass)
2. Используйте продакшен-версию
3. Включите Gzip-сжатие

### Как подключать только нужные стили?

Используйте Sass для импорта только необходимых компонентов:

```scss
// Подключаем функции и переменные
@import "bootstrap/scss/functions";
@import "bootstrap/scss/variables";

// Подключаем только нужные компоненты
@import "bootstrap/scss/root";
@import "bootstrap/scss/reboot";
@import "bootstrap/scss/containers";
@import "bootstrap/scss/grid";
@import "bootstrap/scss/buttons";
@import "bootstrap/scss/forms";
```

## Проблемы совместимости

### Не работает в IE?

Bootstrap 5 больше не поддерживает Internet Explorer. Для поддержки IE11 используйте Bootstrap 4.

### Задержка клика на мобильных?

Bootstrap 5 использует touch-action: manipulation, что убирает задержку в 300мс.
