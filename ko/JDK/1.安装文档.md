# JDK 설치 가이드

JDK(Java Development Kit)는 공식 Java 개발 키트입니다. 이 가이드에서는 다양한 환경에 대한 설치 방법을 설명합니다.

## 버전 정보

| 버전 | 상태 | 설명 |
|---------|--------|-------------|
| JDK 21 (LTS) | ✅ 현재 | 최신 LTS 버전 |
| JDK 17 (LTS) | ✅ 지원됨 | 이전 LTS 버전 |
| JDK 11 (LTS) | ✅ 지원됨 | 레거시 LTS 버전 |
| JDK 8 | ✅ 지원됨 | 레거시 버전 |
| JDK 22 | ✅ 최신 | 최신 기능 버전 |

### LTS와 기능 버전의 차이

- **LTS(장기 지원)**: 2~3년 업데이트, 프로덕션 환경에 권장
- **기능 버전**: 6개월마다 새 기능, 지원 기간이 짧음

## 설치 방법

### 방법 1: Oracle JDK

#### Windows

1. 다운로드: https://www.oracle.com/java/technologies/downloads/
2. 설치 프로그램(.exe) 실행
3. 마법사 안내に従う

#### macOS

```bash
# Oracle 웹사이트에서 DMG 다운로드
# 또는 Homebrew 사용
brew install openjdk@21
```

#### Linux

```bash
# tar.gz 다운로드
wget https://download.oracle.com/java/21/latest/jdk-21_linux-x64_bin.tar.gz
tar -xzf jdk-21_linux-x64_bin.tar.gz
sudo mv jdk-21 /opt/jdk-21
```

### 방법 2: Eclipse Temurin (Adoptium)

#### Windows

1. 다운로드: https://adoptium.net/temurin/releases/
2. 설치 프로그램 실행

#### macOS

```bash
# Homebrew 사용
brew install --cask temurin

# 또는 웹사이트에서 pkg 다운로드
```

#### Linux

```bash
# Ubuntu/Debian
wget https://github.com/adoptium/temurin21-binaries/releases/download/jdk-21.0.2%2B13/OpenJDK21U-jdk_x64_linux_hotspot_21.0.2_13.tar.gz
tar -xzf OpenJDK21U-jdk_x64_linux_hotspot_21.0.2_13.tar.gz

# 환경 설정
export JAVA_HOME=/path/to/jdk-21
export PATH=$JAVA_HOME/bin:$PATH
```

### 방법 3: SDKMAN

```bash
# SDKMAN 설치
curl -s "https://get.sdkman.io" | bash

# JDK 설치
sdk install java 21.0.2-tem

# 사용 가능한 버전 목록
sdk list java
```

### 방법 4: Docker

```dockerfile
FROM eclipse-temurin:21-jdk
WORKDIR /app
COPY . .
RUN javac Main.java
CMD ["java", "Main"]
```

### 방법 5: Windows 패키지 관리자

#### Chocolatey

```powershell
choco install openjdk21
```

#### Scoop

```powershell
scoop install openjdk
```

## 환경 설정

### Windows

```powershell
# JAVA_HOME 설정
[System.Environment]::SetEnvironmentVariable("JAVA_HOME", "C:\Program Files\Java\jdk-21", "User")

# PATH에 추가
[System.Environment]::SetEnvironmentVariable("PATH", "$env:PATH;C:\Program Files\Java\jdk-21\bin", "User")
```

### macOS/Linux

```bash
# ~/.bashrc 또는 ~/.zshrc에 추가
export JAVA_HOME=/opt/jdk-21
export PATH=$JAVA_HOME/bin:$PATH

# 적용
source ~/.bashrc
```

## 설치 확인

```bash
# Java 버전 확인
java -version

# Javac 버전 확인
javac -version

# JAVA_HOME 확인
echo $JAVA_HOME
```

## 여러 JDK 버전 관리

### SDKMAN 사용

```bash
# 여러 버전 설치
sdk install java 17.0.8-tem
sdk install java 21.0.2-tem

# 버전 전환
sdk use java 17.0.8-tem
```

## 일반적인 문제

### "java"를 찾을 수 없음?

PATH에 Java bin 디렉터리가 포함되어 있는지 확인하세요.

### 잘못된 Java 버전?

사용 중인 java 확인:

```bash
which java
java -version
```

### JAVA_HOME이 설정되지 않음?

JAVA_HOME을 JDK 설치 디렉터리로 설정하세요.
