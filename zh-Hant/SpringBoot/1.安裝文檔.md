# Spring Boot 安裝指南

本指南涵蓋 Spring Boot 開發的環境設置，包括 Java、Maven 和 Gradle。

## 前提條件

### Java 安裝

Spring Boot 3.x 需要 Java 17 或更高版本。Spring Boot 2.x 需要 Java 8 或更高版本。

**Windows：**
1. 從 [Oracle](https://www.oracle.com/java/technologies/downloads/) 或 [Adoptium](https://adoptium.net/) 下載 JDK
2. 執行安裝程式
3. 設定環境變數：
   ```
   JAVA_HOME=C:\Program Files\Java\jdk-17
   PATH=%JAVA_HOME%\bin;%PATH%
   ```

**macOS：**
```bash
# 使用 Homebrew
brew install openjdk@17

# 加入 PATH
echo 'export JAVA_HOME=$(/usr/libexec/java_home)' >> ~/.zshrc
source ~/.zshrc
```

**Linux：**
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install openjdk-17-jdk

# 設定 JAVA_HOME
_HOME=/usr/libecho 'export JAVA/jvm/java-17-openjdk-amd64' >> ~/.bashrc
source ~/.bashrc
```

驗證安裝：
```bash
java -version
```

### Maven 安裝

**Windows：**
1. 從 [Apache Maven](https://maven.apache.org/download.cgi) 下載
2. 解壓到資料夾（例如 C:\apache-maven）
3. 設定環境變數：
   ```
   MAVEN_HOME=C:\apache-maven
   PATH=%MAVEN_HOME%\bin;%PATH%
   ```

**macOS：**
```bash
brew install maven
```

**Linux：**
```bash
sudo apt install maven
```

驗證安裝：
```bash
mvn -version
```

### Gradle 安裝

**Windows：**
1. 從 [Gradle](https://gradle.org/install/) 下載
2. 解壓並加入 PATH

**macOS：**
```bash
brew install gradle
```

**Linux：**
```bash
sudo apt install gradle
```

驗證安裝：
```bash
gradle -version
```

## IDE 設定

### IntelliJ IDEA（推薦）
- 從 [JetBrains](https://www.jetbrains.com/idea/download/) 下載
- 確保在專案結構中設定 JDK
- 安裝 Spring Boot Assistant 擴充功能

### VS Code
- 安裝 Java 擴充包
- 安裝 Spring Boot 擴充功能

## 使用 Spring Initializr 快速開始

1. 造訪 [start.spring.io](https://start.spring.io/)
2. 選擇：
   - 專案：Maven 或 Gradle
   - 語言：Java
   - Spring Boot 版本
   - 專案中繼資料
3. 新增依賴（Spring Web、Spring Data JPA 等）
4. 點擊「Generate」下載專案

## 驗證環境設定

```bash
# 檢查 Java
java -version

# 檢查 Maven
mvn -version

# 檢查 Gradle（如果使用 Gradle）
gradle -version

# 建立簡單的 Spring Boot 專案並執行
curl https://start.spring.io/starter.zip -o demo.zip
unzip demo.zip -d demo
cd demo
./mvnw spring-boot:run  # 或 ./gradlew bootRun
```
