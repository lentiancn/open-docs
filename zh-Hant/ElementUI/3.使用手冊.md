# Element UI 使用手冊

## 基礎元件

### Button 按鈕

```html
<el-button>預設</el-button>
<el-button type="primary">主要</el-button>
<el-button type="success">成功</el-button>
<el-button type="warning">警告</el-button>
<el-button type="danger">危險</el-button>
```

### Layout 佈局

```html
<el-row>
  <el-col :span="12"><div></div></el-col>
  <el-col :span="12"><div></div></el-col>
</el-row>
```

### Container 容器

```html
<el-container>
  <el-header>Header</el-header>
  <el-container>
    <el-aside width="200px">Aside</el-aside>
    <el-main>Main</el-main>
  </el-container>
</el-container>
```

## 表單元件

### Input 輸入框

```html
<el-input v-model="input" placeholder="請輸入"></el-input>
<el-input prefix-icon="el-icon-search" placeholder="搜尋"></el-input>
```

### Select 選擇器

```html
<el-select v-model="value" placeholder="請選擇">
  <el-option label="選項一" value="1"></el-option>
  <el-option label="選項二" value="2"></el-option>
</el-select>
```

### Form 表單

```html
<el-form :model="form" :rules="rules" label-width="100px">
  <el-form-item label="名稱" prop="name">
    <el-input v-model="form.name"></el-input>
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="submit">提交</el-button>
  </el-form-item>
</el-form>
```

## 數據元件

### Table 表格

```html
<el-table :data="tableData">
  <el-table-column prop="date" label="日期"></el-table-column>
  <el-table-column prop="name" label="姓名"></el-table-column>
</el-table>
```

### Pagination 分頁

```html
<el-pagination :total="1000" layout="prev, pager, next"></el-pagination>
```

## 通知元件

### Message 消息

```javascript
this.$message({ message: '成功', type: 'success' })
this.$message.error('錯誤')
```

### Alert 警告

```html
<el-alert title="成功提示" type="success"></el-alert>
<el-alert title="警告提示" type="warning"></el-alert>
<el-alert title="錯誤提示" type="error"></el-alert>
```

## 導航元件

### Menu 菜單

```html
<el-menu>
  <el-menu-item index="1">導航一</el-menu-item>
  <el-menu-item index="2">導航二</el-menu-item>
</el-menu>
```

### Tabs 標籤頁

```html
<el-tabs v-model="activeName">
  <el-tab-pane label="用戶管理" name="first">內容</el-tab-pane>
</el-tabs>
```

## 其他元件

### Dialog 對話框

```html
<el-button @click="visible = true">打開</el-button>
<el-dialog title="標題" :visible.sync="visible" width="30%">
  <span>內容</span>
</el-dialog>
```
