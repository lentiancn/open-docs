# HeyGen Guide d'installation

HeyGen est une plateforme de génération vidéo IA qui permet aux utilisateurs de créer des avatars IA réalissants et de générer des vidéos de tête parlante à partir de texte ou d'audio.

## Vue d'ensemble de la plateforme

HeyGen est principalement une plateforme SaaS basée sur le cloud. Contrairement aux projets open source, HeyGen ne nécessite pas d'installation locale - les utilisateurs y accèdent via l'interface web ou l'API.

### Méthodes d'accès

| Méthode | Description | Exigences |
|---------|-------------|-----------|
| Interface web | Création vidéo basée sur un navigateur | Compte + Internet |
| API | Accès programmatique | Clé API |
| SDK | Développement local | Compte développeur |

## Présentation des versions

HeyGen est un service cloud, pas une installation locale :

| Composant | Version | Notes |
|-----------|---------|-------|
| HeyGen Web | Dernière | Toujours à jour |

## Accéder à HeyGen

### Étape 1 : Créer un compte

1. Visitez https://www.heygen.com
2. Cliquez sur "S'inscrire" ou "Commencer"
3. Vérifiez votre adresse e-mail

### Étape 2 : Choisir un plan

| Plan | Fonctionnalités | Prix |
|------|-----------------|------|
| Gratuit | Minutes limitées, filigrane | Gratuit |
| Pro | Plus de minutes, sans filigrane | 29$/mois |
| Enterprise | Limites personnalisées, accès API | Contacter les ventes |

### Étape 3 : Obtenir une clé API (pour les développeurs)

1. Connectez-vous au tableau de bord HeyGen
2. Accédez à Paramètres → API
3. Générez une clé API

## Installation de l'API (Pour les développeurs)

### Python SDK

```bash
# Installer HeyGen SDK
pip install heygen-python

# Ou installer depuis la source
pip install git+https://github.com/heygen-ai/heygen-python.git
```

### Node.js SDK

```bash
# Installer HeyGen SDK
npm install heygen-node

# Ou installer depuis la source
npm install github:heygen-ai/heygen-node
```

### Vérifier l'installation

```python
# Python
import heygen
print(heygen.__version__)
```

```javascript
// Node.js
const heygen = require('heygen-node');
console.log(heygen.version);
```

## Configuration de l'API

### Exemple Python

```python
from heygen import HeyGen

# Initialiser avec la clé API
heygen = HeyGen(api_key="your_api_key")

# Obtenir les informations du compte
account = heygen.get_account_info()
print(account)
```

### Exemple Node.js

```javascript
const { HeyGen } = require('heygen-node');

const heygen = new HeyGen({ apiKey: 'your_api_key' });

// Obtenir les informations du compte
const account = await heygen.getAccountInfo();
console.log(account);
```

## Variables d'environnement

### Définir la clé API

```bash
# Linux/macOS
export HEYGEN_API_KEY="your_api_key"

# Windows (PowerShell)
$env:HEYGEN_API_KEY="your_api_key"

# Dans le fichier .env
HEYGEN_API_KEY=your_api_key
```

### Python avec dotenv

```python
from dotenv import load_dotenv
load_dotenv()

import heygen
heygen = HeyGen(api_key=os.getenv("HEYGEN_API_KEY"))
```

## Problèmes courants

### Erreurs d'authentification

**Problème :** "Clé API invalide"

**Solutions :**
1. Vérifiez que la clé API est correcte
2. Assurez-vous que la clé API est active
3. Vérifiez que le compte a accès à l'API

### Limitation de débit

**Problème :** "Trop de requêtes"

**Solutions :**
1. Vérifiez les limites d'utilisation de l'API
2. Passez à un plan supérieur pour des limites plus élevées
3. Implémentez une file d'attente de requêtes

### Échec de génération vidéo

**Problème :** "Échec de la génération"

**Solutions :**
1. Vérifiez les paramètres d'entrée
2. Vérifiez le solde de crédits
3. Réessayez plus tard

## Bonnes pratiques de sécurité

1. **Ne exposez jamais les clés API** - Utilisez des variables d'environnement
2. **Faites pivoter les clés régulièrement** - Générez régulièrement de nouvelles clés
3. **Utilisez les autorisations minimales** - Demandez uniquement les scopes nécessaires
4. **Surveillez l'utilisation** - Vérifiez les anomalies dans le tableau de bord
