# Element Plus 使用手册

本手册将帮助你快速上手 Element Plus，掌握日常使用中常用的组件和功能。

## 快速开始

在使用 Element Plus 之前，确保已完成安装配置。

### 基础使用

在 Vue 组件中直接使用组件：

```vue
<template>
  <el-button>默认按钮</el-button>
  <el-button type="primary">主要按钮</el-button>
  <el-button type="success">成功按钮</el-button>
  <el-button type="warning">警告按钮</el-button>
  <el-button type="danger">危险按钮</el-button>
  <el-button type="info">信息按钮</el-button>
</template>
```

## 常用组件

### 按钮（Button）

基础用法：

```vue
<el-button>默认</el-button>
<el-button type="primary">主要</el-button>
<el-button type="success">成功</el-button>
<el-button type="warning">警告</el-button>
<el-button type="danger">危险</el-button>
<el-button type="info">信息</el-button>
```

不同尺寸：

```vue
<el-button size="large">大型</el-button>
<el-button>默认</el-button>
<el-button size="small">小型</el-button>
```

禁用状态：

```vue
<el-button disabled>禁用</el-button>
```

图标按钮：

```vue
<el-button icon="Search">搜索</el-button>
<el-button icon="Edit">编辑</el-button>
<el-button icon="Delete">删除</el-button>
```

### 输入框（Input）

基础用法：

```vue
<el-input v-model="input" placeholder="请输入内容" />
```

带图标：

```vue
<el-input v-model="input" prefix-icon="Search" />
<el-input v-model="input" suffix-icon="Calendar" />
```

带清空按钮：

```vue
<el-input v-model="input" clearable />
```

密码输入框：

```vue
<el-input v-model="input" type="password" show-password />
```

多行文本：

```vue
<el-input v-model="textarea" type="textarea" :rows="3" />
```

### 选择器（Select）

基础选择器：

```vue
<el-select v-model="value" placeholder="请选择">
  <el-option label="选项1" value="1" />
  <el-option label="选项2" value="2" />
</el-select>
```

多选：

```vue
<el-select v-model="value" multiple placeholder="请选择">
  <el-option label="选项1" value="1" />
  <el-option label="选项2" value="2" />
</el-select>
```

### 表格（Table）

基础表格：

```vue
<el-table :data="tableData" stripe border>
  <el-table-column prop="date" label="日期" width="180" />
  <el-table-column prop="name" label="姓名" width="180" />
  <el-table-column prop="address" label="地址" />
</el-table>
```

带索引：

```vue
<el-table-column type="index" label="序号" width="60" />
```

固定列：

```vue
<el-table-column prop="date" label="日期" width="150" fixed />
```

### 表单（Form）

基础表单：

```vue
<el-form :model="form" label-width="80px">
  <el-form-item label="活动名称">
    <el-input v-model="form.name" />
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="onSubmit">提交</el-button>
  </el-form-item>
</el-form>
```

表单验证：

```vue
<el-form :model="ruleForm" :rules="rules" ref="ruleFormRef">
  <el-form-item label="名称" prop="name">
    <el-input v-model="ruleForm.name" />
  </el-form-item>
</el-form>
```

### 对话框（Dialog）

```vue
<el-dialog v-model="visible" title="提示" width="30%">
  <span>内容</span>
</el-dialog>
```

### 消息提示

```javascript
import { ElMessage } from 'element-plus'

ElMessage.success('成功消息')
ElMessage.warning('警告消息')
ElMessage.error('错误消息')
```

### 菜单（Menu）

水平菜单：

```vue
<el-menu mode="horizontal">
  <el-menu-item index="1">处理中心</el-menu-item>
  <el-menu-item index="2">订单管理</el-menu-item>
</el-menu>
```

垂直菜单：

```vue
<el-menu default-active="1">
  <el-menu-item index="1">导航一</el-menu-item>
  <el-menu-item index="2">导航二</el-menu-item>
</el-menu>
```

### 标签页（Tabs）

```vue
<el-tabs v-model="activeName">
  <el-tab-pane label="用户管理" name="first">用户管理</el-tab-pane>
  <el-tab-pane label="配置管理" name="second">配置管理</el-tab-pane>
</el-tabs>
```

### 分页（Pagination）

```vue
<el-pagination
  v-model:current-page="currentPage"
  :page-size="100"
  :total="1000"
  layout="total, prev, pager, next"
/>
```

## 布局系统

### 栅格系统

Element Plus 使用 24 栏栅格系统：

```vue
<el-row :gutter="20">
  <el-col :span="12"><div>12栏</div></el-col>
  <el-col :span="12"><div>12栏</div></el-col>
</el-row>
```

### 容器布局

```vue
<el-container>
  <el-header>Header</el-header>
  <el-main>Main</el-main>
  <el-footer>Footer</el-footer>
</el-container>
```


## 常见问题

### 组件样式不生效

确保已正确引入 CSS：

```javascript
import 'element-plus/dist/index.css'
```

### 图标不显示

需要额外安装图标库：

```bash
npm install @element-plus/icons-vue
```

然后全局注册：

```javascript
import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```
