# Bootstrap 5 Guide d'utilisation

Ce guide explique comment créer des sites responsives avec Bootstrap 5.

## Démarrage rapide

```html
<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Bootstrap 5</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <div class="container mt-5">
        <h1>Bonjour Bootstrap 5!</h1>
        <p class="lead">Page responsive.</p>
    </div>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## Système de grille

```html
<div class="container">
    <div class="row">
        <div class="col-12 col-md-6">Colonne</div>
        <div class="col-12 col-md-6">Colonne</div>
    </div>
</div>
```

## Composants

### Boutons

```html
<button class="btn btn-primary">Primaire</button>
<button class="btn btn-secondary">Secondaire</button>
<button class="btn btn-success">Succès</button>
<button class="btn btn-danger">Danger</button>
<button class="btn btn-outline-primary">Outline</button>
```

### Navigation

```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">Marque</a>
    </div>
</nav>
```

### Cartes

```html
<div class="card" style="width: 18rem;">
    <div class="card-body">
        <h5 class="card-title">Titre</h5>
        <p class="card-text">Contenu</p>
    </div>
</div>
```

### Formulaires

```html
<form>
    <div class="mb-3">
        <label class="form-label">Email</label>
        <input type="email" class="form-control">
    </div>
    <button type="submit" class="btn btn-primary">Envoyer</button>
</form>
```

### Modales

```html
<button class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#myModal">
    Ouvrir
</button>

<div class="modal" id="myModal">
    <div class="modal-dialog">
        <div class="modal-content">
            <div class="modal-header">
                <h5 class="modal-title">Titre</h5>
                <button class="btn-close" data-bs-dismiss="modal"></button>
            </div>
        </div>
    </div>
</div>
```

## Utilitaires

### Espacements

```html
<div class="m-3">Marge</div>
<div class="mt-3">Marge-haut</div>
<div class="p-3">Padding</div>
```

### Texte

```html
<p class="text-center">Centré</p>
<p class="fw-bold">Gras</p>
```

### Couleurs

```html
<div class="bg-primary text-white">Primaire</div>
<div class="bg-success text-white">Succès</div>
```

## Personnalisation

```css
:root {
    --bs-primary: #ff6b35;
}
```
