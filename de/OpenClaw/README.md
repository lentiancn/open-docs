# OpenClaw

[English](./README.md) | [简体中文](./README.md) | [日本語](./README.md)

🦞 KI-Assistent-Gateway für jedes OS. Unterstützt WhatsApp, Telegram, Discord, iMessage.

## Funktionen

- 🤖 Verbindet Chat-Apps mit KI-Assistenten
- 📱 Multi-Plattform
- 🔒 Lokal/Du kontrollierst deine Daten
- 🧠 Mehrere Agenten
- 🌐 Web-Interface

## Schnellstart

```bash
npm install -g openclaw@latest
openclaw onboard --install-daemon
openclaw gateway --port 18789
```

Öffne http://127.0.0.1:18789/
