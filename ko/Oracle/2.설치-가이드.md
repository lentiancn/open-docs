# Oracle 설치 가이드

## 전제 조건

- **운영 체제**: Oracle Linux, RHEL, Windows Server
- **메모리**: 최소 2GB RAM (권장 8GB+)
- **디스크 공간**: 소프트웨어에 최소 8GB

---

## Linux에 설치

### 1. Oracle 사용자 생성

```bash
groupadd -g 54321 oinstall
groupadd -g 54322 dba
useradd -u 54321 -g oinstall -G dba oracle
passwd oracle
```

### 2. 커널 매개변수 구성

```bash
kernel.shmmax = 4294967296
kernel.shmall = 2097152
```

### 3. 인스톨러 실행

```bash
./runInstaller
```
