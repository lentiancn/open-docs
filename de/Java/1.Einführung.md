# Java Einführung

> Java ist eine objektorientierte Programmiersprache

---

## Übersicht

Java wurde 1995 von Sun Microsystems veröffentlicht. Bekannt für plattformübergreifende Fähigkeiten.

---

## Funktionen

- Objektorientiert
- Plattformübergreifend (JVM)
- Automatische Speicherverwaltung
- Multithreading
- Reiches Ökosystem
