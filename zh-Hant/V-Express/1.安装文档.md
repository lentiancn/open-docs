# V-Express 安裝指南

V-Express 是騰訊 AI Lab 開發的人像視頻生成方法。它在參考圖像、音頻和 V-Kps 序列的控制下生成說話頭像視頻。

## 版本概述

| 版本 | 狀態 | 描述 |
|------|------|------|
| V-Express 1.x | 當前 | 主要版本 |

## 系統要求

### 硬件要求

| 組件 | 最低配置 | 推薦配置 |
|------|----------|----------|
| GPU | NVIDIA 16GB 顯存 | NVIDIA 24GB+ 顯存 |
| 內存 | 16GB | 32GB+ |
| 存儲 | 50GB | 100GB+ |
| CUDA | 11.8+ | 12.1+ |

### 軟件要求

| 軟件 | 版本 | 說明 |
|------|------|------|
| Python | 3.8 - 3.10 | 推薦 3.9 |
| CUDA | 11.8+ | GPU 加速必需 |
| ffmpeg | 最新版 | 視頻處理必需 |

## Linux/Ubuntu 安裝

### 步驟 1：安裝系統依賴

```bash
# 更新系統
sudo apt update && sudo apt upgrade -y

# 安裝 Python 和開發工具
sudo apt install python3.9 python3-pip git wget curl

# 安裝 ffmpeg
sudo apt install ffmpeg git-lfs
```

### 步驟 2：克隆項目

```bash
git clone https://github.com/tencent-ailab/V-Express.git
cd V-Express
```

### 步驟 3：創建虛擬環境

```bash
python3.9 -m venv venv
source venv/bin/activate
```

### 步驟 4：安裝依賴

```bash
pip install -r requirements.txt
```

### 步驟 5：下載模型

```bash
git lfs install
git clone https://huggingface.co/tk93/V-Express
mv V-Express/model_ckpts model_ckpts
```

## 驗證安裝

```bash
python inference.py --help
```

## 常見問題

### CUDA 內存不足

**解決方案：**
- 減少推理步數：`--num_inference_steps 15`
- 啟用內存節省：`--save_gpu_memory`
