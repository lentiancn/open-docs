# Element UI Installationsanleitung

Diese Anleitung beschreibt, wie Sie Element UI in verschiedenen Projekten installieren und verwenden.

## Umgebungsanforderungen

Element UI erfordert:

- Vue 2.0.0 oder höher
- Node.js 8.0.0 oder höher
- Moderne Browser (Chrome, Firefox, Safari, Edge 10+)

## Installationsmethoden

### Verwendung des Paketmanagers

Die Installation mit npm wird empfohlen:

npm install element-ui --save

Oder mit yarn:

yarn add element-ui

## Vollständiger Import

In main.js:

import Vue from 'vue'
import Element from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(Element)

new Vue({
  el: '#app',
  render: h => h(App)
})

### Import bei Bedarf

Der Import bei Bedarf kann die Projektgröße erheblich reduzieren.

Installieren Sie zuerst babel-plugin-component:

npm install babel-plugin-component -D

Dann in .babelrc konfigurieren:

{
  "presets": [["env", {
    "modules": false,
    "targets": {
      "browsers": ["> 1%", "last 2 versions", "not ie <= 8"]
    }
  }]],
  "plugins": [
    ["component", {
      "libraryName": "element-ui",
      "styleLibraryName": "theme-chalk"
    }]
  ]
}

Komponenten bei Bedarf importieren:

import { Button, Select } from 'element-ui'

Vue.component(Button.name, Button)
Vue.component(Select.name, Select)

## Globale Konfiguration

Beim Importieren von Element UI können Sie ein globales Konfigurationsobjekt übergeben:

import Vue from 'vue'
import Element from 'element-ui'

Vue.use(Element, {
  size: 'small',
  zIndex: 2000
})

Konfigurationsoptionen:

- size: Legt die Standardgröße für Formularkomponenten fest. Mögliche Werte: medium, small, mini
- z-index: Legt den z-index für Popup-Komponenten fest. Standardwert ist 2000

## Verwendung mit Nuxt.js

Für Nuxt-Projekte können Sie nuxt-element verwenden:

npm install nuxt-element

In nuxt.config.js konfigurieren:

module.exports = {
  modules: ['nuxt-element']
}
