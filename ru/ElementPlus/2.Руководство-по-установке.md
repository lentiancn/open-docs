# Руководство по установке Element Plus

В этом руководстве подробно описано, как установить и использовать Element Plus в различных проектах.

## Требования к окружению

Element Plus требует:

- Vue 3.0.0 или выше
- Node.js 14.0.0 или выше
- Современные браузеры (Chrome, Firefox, Safari, Edge)

## Способы установки

### Использование менеджера пакетов

Рекомендуется устанавливать с помощью npm, yarn или pnpm:

Для npm:
npm install element-plus

Для yarn:
yarn add element-plus

Для pnpm:
pnpm add element-plus

## Полный импорт

Если вас не беспокоит размер пакета, полный импорт будет удобнее.

В main.ts:

import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')

### Поддержка Volar

Если вы используете Volar, добавьте определение глобального типа компонента в compilerOptions.types в tsconfig.json:

{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}

## Импорт по требованию

Импорт по требованию может значительно уменьшить размер проекта. Вам нужно использовать дополнительные плагины для импорта нужных компонентов.

### Автоматический импорт (Рекомендуется)

Сначала установите плагины:

Для npm:
npm install -D unplugin-vue-components unplugin-auto-import

Для yarn:
yarn add -D unplugin-vue-components unplugin-auto-import

Для pnpm:
pnpm add -D unplugin-vue-components unplugin-auto-import

#### Конфигурация Vite

В vite.config.ts:

import { defineConfig } from 'vite'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    AutoImport({ resolvers: [ElementPlusResolver()] }),
    Components({ resolvers: [ElementPlusResolver()] }),
  ],
})

#### Конфигурация Webpack

В webpack.config.js:

const AutoImport = require('unplugin-auto-import/webpack')
const Components = require('unplugin-vue-components/webpack')
const { ElementPlusResolver } = require('unplugin-vue-components/resolvers')

module.exports = {
  plugins: [
    AutoImport({ resolvers: [ElementPlusResolver()] }),
    Components({ resolvers: [ElementPlusResolver()] }),
  ],
}

### Использование с Nuxt

Для пользователей Nuxt просто установите @element-plus/nuxt:

npm install -D @element-plus/nuxt

Затем добавьте конфигурацию в nuxt.config.ts:

export default defineNuxtConfig({
  modules: ['@element-plus/nuxt'],
})

### Ручной импорт

Element Plus предоставляет готовые функции Tree Shaking на основе ES Module.

Вам нужно установить unplugin-element-plus для импорта стилей:

npm install -D unplugin-element-plus

В компонентах:

<template>
  <el-button>Кнопка</el-button>
</template>

<script>
import { ElButton } from 'element-plus'

export default {
  components: { ElButton },
}
</script>

Конфигурация Vite:

import { defineConfig } from 'vite'
import ElementPlus from 'unplugin-element-plus/vite'

export default defineConfig({
  plugins: [ElementPlus()],
})

## Глобальная конфигурация

При регистрации Element Plus можно передать глобальный объект конфигурации с size и zIndex:

import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import App from './App.vue'

const app = createApp(App)
app.use(ElementPlus, { size: 'small', zIndex: 3000 })

Параметры конфигурации:

- size: Установить размер по умолчанию для компонентов формы. Возможные значения: large, default, small
- zIndex: Установить z-index для всплывающих компонентов. Значение по умолчанию: 2000

### Конфигурация для импорта по требованию

Используйте компонент el-config-provider:

<template>
  <el-config-provider :size="size" :z-index="zIndex">
    <app />
  </el-config-provider>
</template>

<script>
import { defineComponent } from 'vue'
import { ElConfigProvider } from 'element-plus'

export default defineComponent({
  components: { ElConfigProvider },
  setup() {
    return { zIndex: 3000, size: 'small' }
  },
})
</script>
