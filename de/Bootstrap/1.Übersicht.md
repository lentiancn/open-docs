# Bootstrap Übersicht

> Belieftes Frontend-CSS-Framework

---

## Projektbeschreibung

Bootstrap ist ein Open-Source-CSS-Framework, das von Twitter entwickelt und gepflegt wird und erstmals 2011 veröffentlicht wurde. Bootstrap bietet reichhaltige vorgefertigte Komponenten und ein responsives Grid-System für den schnellen Aufbau moderner, responsiver Websites und Webanwendungen.

---

## Versionsinformationen

| Version | Veröffentlichungsdatum | Status |
|---------|------------------------|--------|
| Bootstrap 5.3 | Mai 2023 | Aktuell Stabil |
| Bootstrap 5.2 | September 2022 | Stabil |
| Bootstrap 5.1 | August 2021 | Stabil |
| Bootstrap 4.x | Januar 2018 | Nicht unterstützt |

---

## Hauptfunktionen

- **Responsives Design**: Integriertes responsives Layout
- **Vorgefertigte Komponenten**: Buttons, Navigation, Formulare, Karten, etc.
- **Leistungsstarkes Grid-System**: 12-spaltiges responsives Grid
- **Anpassungsoptionen**: Sass-Variablen für einfache Anpassung
- **JavaScript-Plugins**: Dropdowns, Modals, etc.
- **Icon-Bibliothek**: Integrierte Bootstrap Icons

---

## Anwendungsfälle

- Responsive Website-Entwicklung
- Admin-Dashboards
- Schnelles Prototyping
- Mobile-First-Projekte
