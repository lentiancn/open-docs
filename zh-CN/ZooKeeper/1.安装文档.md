# ZooKeeper 安装文档

在所有平台上安装 ZooKeeper 的详细指南。

---

## 安装方式

| 方式 | 描述 |
|------|------|
| **单机** | 单服务器 |
| **Docker** | 容器化部署 |
| **集群** | 多服务器 |

---

## 单机安装

### 下载

```bash
wget https://dlcdn.apache.org/zookeeper/zookeeper-3.9.3/zookeeper-3.9.3.tar.gz
tar -xzf zookeeper-3.9.3.tar.gz
sudo mv zookeeper-3.9.3 /opt/zookeeper
```

### 配置

```properties
# conf/zoo.cfg
tickTime=2000
dataDir=/var/lib/zookeeper
clientPort=2181
```

---

## Docker 安装

```bash
docker run -d --name zookeeper -p 2181:2181 confluentinc/cp-zookeeper:7.5.0
```

---

## 集群安装

```properties
tickTime=2000
dataDir=/var/lib/zookeeper
clientPort=2181
initLimit=10
syncLimit=5
server.1=zoo1:2888:3888
server.2=zoo2:2888:3888
server.3=zoo3:2888:3888
```

---

## 启动

```bash
/opt/zookeeper/bin/zkServer.sh start
```

---

## 验证

```bash
# 连接
/opt/zookeeper/bin/zkCli.sh -server localhost:2181

# 基本命令
ls /
create /test "hello"
get /test
```
