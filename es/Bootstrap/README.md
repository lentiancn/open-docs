# Bootstrap

El marco CSS frontend más popular del mundo.

## Resumen

Bootstrap es el marco CSS frontend más popular desarrollado por Twitter. Cuenta con sistema de rejilla responsivo, componentes preconstruidos y clases utilitarias CSS.

### Características principales

- **Sistema de rejilla responsivo** - Rejilla de 12 columnas
- **Componentes preconstruidos** - Navbars, tarjetas, modales
- **Clases utilitarias CSS** - Ajustes rápidos de estilo
- **Componentes JavaScript** - Componentes interactivos

### Estadísticas

- GitHub Stars: 160,000+
- Sitios web utilizados: Millones
