## What is Redis?

Redis is an in-memory data structure store, used as a database, cache, and message broker. It supports various data structures like strings, hashes, lists, sets, and more.

## Linux

### Ubuntu / Debian

#### Method 1: APT (Recommended)

```bash
# Update package index
sudo apt update

# Install Redis
sudo apt install redis-server

# Start Redis
sudo systemctl start redis-server

# Enable on boot
sudo systemctl enable redis-server
```

#### Method 2: From Source

```bash
# Install dependencies
sudo apt install build-essential tcl

# Download Redis
cd /tmp
wget https://github.com/redis/redis/archive/7.2.4.tar.gz
tar -zxvf 7.2.4.tar.gz
cd redis-7.2.4

# Build and install
make
sudo make install
```

### CentOS / RHEL

#### Using YUM

```bash
# Install EPEL repository
sudo yum install epel-release

# Install Redis
sudo yum install redis

# Start Redis
sudo systemctl start redis

# Enable on boot
sudo systemctl enable redis
```

### Fedora

```bash
# Install Redis
sudo dnf install redis

# Start Redis
sudo systemctl start redis
sudo systemctl enable redis
```

### Arch Linux

```bash
# Install Redis
sudo pacman -S redis

# Start Redis
sudo systemctl start redis
sudo systemctl enable redis
```

## macOS

### Homebrew (Recommended)

```bash
# Install Homebrew (if needed)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install Redis
brew install redis

# Start Redis
brew services start redis

# Or run manually
redis-server
```

### MacPorts

```bash
sudo port install redis
```

## Windows

### WSL2 (Recommended)

```bash
# Install Redis on WSL2 (Ubuntu)
sudo apt update
sudo apt install redis-server

# Start Redis
sudo systemctl start redis
```

### Memurai (Recommended Alternative)

1. Download Memurai from https://www.memurai.com/
2. Install and run as Windows service

### Docker

```bash
# Run Redis container
docker run -d --name redis -p 6379:6379 redis:latest

# With persistent storage
docker run -d --name redis -p 6379:6379 -v redis-data:/data redis:latest redis-server --appendonly yes

# With password
docker run -d --name redis -p 6379:6379 -v redis-data:/data redis:latest redis-server --requirepass yourpassword
```

```bash
# Check Redis version

# Check Redis status
sudo systemctl status redis

# Connect to Redis
redis-cli

# Test connection
redis-cli ping
# Should return: PONG
```

## Configuration

### Main Config File

- **Ubuntu/Debian**: `/etc/redis/redis.conf`
- **CentOS/RHEL**: `/etc/redis.conf`
- **macOS**: `/usr/local/etc/redis.conf` (Homebrew)
- **Docker**: `/etc/redis/redis.conf`

### Basic Configuration

```bash
# Bind to localhost (security)
bind 127.0.0.1

# Set password
requirepass yourpassword

# Enable persistence
appendonly yes

# Set max memory
maxmemory 256mb

# Set maxmemory policy
maxmemory-policy allkeys-lru
```

### Common Commands

```bash
# Start Redis
redis-server

# Start with custom config
redis-server /etc/redis/redis.conf

# Connect to Redis
redis-cli

# Connect with password
redis-cli -a yourpassword

# Stop Redis
sudo systemctl stop redis
```

## Redis CLI Basics

```bash
# Set a key
redis-cli SET mykey "Hello"

# Get a key
redis-cli GET mykey

# Check all keys
redis-cli KEYS "*"

# Delete a key
redis-cli DEL mykey

# Set expiration (seconds)
redis-cli SETEX mykey 60 "value"

# Check server info
redis-cli INFO
```

