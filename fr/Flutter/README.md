# Flutter

Flutter est un kit de développement UI open source développé par Google pour créer des applications multiplateformes.

## Description

Flutter vous permet de créer de belles applications pour iOS, Android, Web et bureau en utilisant une seule base de code.

## Principales fonctionnalités

- **Multiplateforme** : un code, plusieurs plateformes
- **Rechargement à chaud** : itérations de développement rapides
- **Composants riches** : composants Material Design et Cupertino
- **Langage Dart** : facile à apprendre, excellentes performances
- **UI déclarative** : créez de belles interfaces

## Démarrage rapide

### Installation

```bash
# macOS
brew install flutter

# Linux
git clone https://github.com/flutter/flutter.git -b stable
```

### Créer un projet

```bash
flutter create my_app
cd my_app
flutter run
```
