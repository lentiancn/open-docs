# JUnit 5 使用ガイド

---

## テスト

```java
@Test
void test() {
    assertEquals(5, 2 + 3);
}
```

---

## アサーション

```java
assertEquals(expected, actual);
assertTrue(condition);
```

---

## アノテーション

```java
@BeforeEach
void setUp() { }

@Disabled
@Test
void testSkip() { }

@DisplayName("名前")
@Test
void testName() { }
```

---

## パラメータ化テスト

```java
@ParameterizedTest
@ValueSource(ints = { 1, 2, 3 })
void test(int number) {
    assertTrue(number > 0);
}
```
