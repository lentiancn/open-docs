# Go FAQ

> Часто задаваемые вопросы о Go

## Установка

### Q: Команда не найдена после установки?
**A**: Проверьте, добавлен ли Go в PATH

### Q: Как проверить установку?
```bash
go version
go env GOPATH
```

---

## Синтаксис

### Q: Есть ли в Go исключения?
**A**: Нет, ошибки обрабатываются через возвращаемые значения:
```go
result, err := doSomething()
if err != nil {
    // обработка ошибки
}
```

### Q: Поддерживаются ли дженерики?
**A**: Go 1.18+ поддерживает дженерики:
```go
func Print[T any](s []T) {
    for _, v := range s {
        fmt.Println(v)
    }
}
```

## Параллелизм

### Q: Разница между горутинами и потоками?
- Горутины управляются runtime Go
- Легче, чем потоки

### Q: Как ждать завершения горутин?
```go
var wg sync.WaitGroup
wg.Add(1)
go func() {
    defer wg.Done()
}()
wg.Wait()
```
