# Руководство по установке JMeter

JMeter — это инструмент для нагрузочного тестирования с открытым исходным кодом, разработанный организацией Apache. В этом руководстве описаны методы установки в различных средах.

## Информация о версии

| Версия | Статус | Описание |
|------|------|------|
| JMeter 5.x | ✅ Текущая | Последняя версия |

## Системные требования

### Аппаратные требования

| Требование | Конфигурация |
|------|------|

| Оперативная память | 4+ ГБ |

| Программное обеспечение | Версия |
|------|------|
| Java | JDK 8 или выше |

## Методы установки

### Метод 1: Установка в Windows

#### 1. Загрузка

1. Посетите https://jmeter.apache.org/download.cgi
2. Загрузите версию Binaries
3. Распакуйте в нужный каталог

#### 2. Запуск

```powershell
# Дважды щелкните bin\jmeter.bat
# Или запустите из командной строки
jmeter.bat
```

### Метод 2: Установка в macOS

#### 1. Использование Homebrew

```bash
brew install jmeter
```

#### 2. Ручная установка

```bash
# Загрузить и распаковать
wget https://archive.apache.org/dist/jmeter/binaries/apache-jmeter-5.6.3.tgz
tar -xzf apache-jmeter-5.6.3.tgz

# Запустить
./apache-jmeter-5.6.3/bin/jmeter
```

### Метод 3: Установка в Linux

```bash
# Ubuntu/Debian
sudo apt-get install jmeter

# Или загрузить и установить
wget https://archive.apache.org/dist/jmeter/binaries/apache-jmeter-5.6.3.tgz
tar -xzf apache-jmeter-5.6.3.tgz
cd apache-jmeter-5.6.3/bin
./jmeter
```

### Метод 4: Docker

```bash
# Запустить контейнер JMeter
docker run -v $(pwd):/workspace justb4/jmeter -n -t test.jmx -l result.jtl
```

## Настройка окружения

### Настройка Java

```bash
# Проверить Java
java -version

# Установить переменные окружения
export JAVA_HOME=/path/to/java
export PATH=$JAVA_HOME/bin:$PATH
```

## Часто задаваемые вопросы

### Недостаточно памяти?

Измените jmeter.bat или jmeter:

```bash
# Установить размер кучи
HEAP="-Xms1g -Xmx2g"
```

### Медленный запуск GUI?

Используйте режим командной строки или увеличьте память.
