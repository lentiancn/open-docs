# Element UI Introduction

Element UI is a Vue 2-based UI component library developed by Ele.me team.

## What is Element UI

Element UI is a desktop UI component library for Vue 2, providing rich components for building enterprise management systems.

## Main Features

- Complete component system (70+ components)
- Vue 2 deep integration
- On-demand import
- Internationalization
- Theme customization

## Applicable Scenarios

- Enterprise management systems
- E-commerce backend
- Data visualization platforms
- CRM systems
