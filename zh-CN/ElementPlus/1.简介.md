# Element Plus - 简介

Element Plus 是 Element UI 的 Vue 3 版本，由饿了么团队开发维护的桌面端 UI 组件库。

## 核心特性

- **Vue 3 优先**：完全基于 Vue 3 Composition API 构建
- **TypeScript**：完整的 TypeScript 类型定义
- **组件丰富**：70+ 基础组件
- **主题定制**：支持 CSS 变量和 SCSS 变量
- **国际化**：内置 20+ 语言包
- **按需加载**：支持自动导入

## 组件分类

- 基础组件：Button、Layout、Container
- 表单组件：Input、Select、Form
- 数据展示：Table、Tree、Card
- 反馈组件：Message、Dialog、Loading
- 导航组件：Menu、Tabs、Breadcrumb
