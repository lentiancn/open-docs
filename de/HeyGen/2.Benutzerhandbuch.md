# HeyGen Benutzerhandbuch

Dieses Handbuch beschreibt, wie Sie HeyGen zur KI-Videogenerierung verwenden.

## Erste Schritte

### Web-Oberfläche verwenden

1. **Melden Sie sich** bei HeyGen unter https://www.heygen.com an
2. **Wählen Sie einen Avatar** aus der Bibliothek oder laden Sie einen benutzerdefinierten hoch
3. **Wählen Sie den Eingabetyp**: Text, Audio oder Video
4. **Generieren Sie das Video** und laden Sie es herunter

### API verwenden

```python
from heygen import HeyGen

heygen = HeyGen(api_key="your_api_key")

# Video aus Text generieren
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="Hello, this is my first AI video!",
    voice_id="voice_id"
)

print(f"Video-URL: {video.url}")
```

## Hauptfunktionen

### 1. Avatar-Auswahl

```python
# Verfügbare Avatare auflisten
avatars = heygen.list_avatars()
for avatar in avatars:
    print(f"{avatar.id}: {avatar.name}")

# Bestimmten Avatar abrufen
avatar = heygen.get_avatar(avatar_id="avatar_id")
```

### 2. Stimmenauswahl

```python
# Verfügbare Stimmen auflisten
voices = heygen.list_voices()
for voice in voices:
    print(f"{voice.id}: {voice.name} ({voice.language})")

# Stimmendetails abrufen
voice = heygen.get_voice(voice_id="voice_id")
```

### 3. Videogenerierung

#### Aus Text

```python
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="Ihr Skripttext hier",
    voice_id="voice_id",
    background="office"  # Optional: Hintergrundszene
)
```

#### Aus Audio

```python
video = heygen.generate_video_from_audio(
    avatar_id="avatar_id",
    audio_url="https://example.com/audio.wav"
)
```

#### Aus Foto (TalkingPhoto)

```python
video = heygen.generate_talking_photo(
    photo_url="https://example.com/photo.jpg",
    script="Ihr Skript hier",
    voice_id="voice_id"
)
```

### 4. Stapelgenerierung

```python
scripts = [
    "Erstes Video-Skript",
    "Zweites Video-Skript",
    "Drittes Video-Skript"
]

# Mehrere Videos generieren
jobs = heygen.batch_generate(
    avatar_id="avatar_id",
    scripts=scripts,
    voice_id="voice_id"
)

# Job-Status überprüfen
for job in jobs:
    status = heygen.get_job_status(job.id)
    print(f"Job {job.id}: {status}")
```

## Erweiterte Funktionen

### Benutzerdefinierter Avatar

```python
# Benutzerdefinierten Avatar hochladen
avatar = heygen.upload_avatar(
    name="Mein benutzerdefinierter Avatar",
    video_url="https://example.com/reference.mp4"
)
print(f"Benutzerdefinierter Avatar-ID: {avatar.id}")
```

### Sofortige Stimmklonung

```python
# Stimme aus Audiomuster klonen
voice = heygen.clone_voice(
    name="Mein Stimmklon",
    audio_url="https://example.com/voice_sample.wav"
)
print(f"Geklonte Stimm-ID: {voice.id}")
```

### Video-Bearbeitung

```python
# Videodetails abrufen
video = heygen.get_video(video_id="video_id")

# Videoeinstellungen aktualisieren
updated = heygen.update_video(
    video_id="video_id",
    title="Mein aktualisiertes Video",
    description="Aktualisierte Beschreibung"
)

# Video löschen
heygen.delete_video(video_id="video_id")
```

## Parameter-Referenz

### generate_video-Parameter

| Parameter | Typ | Erforderlich | Beschreibung |
|-----------|-----|---------------|---------------|
| avatar_id | string | Ja | Zu verwendende Avatar-ID |
| script | string | Zu | Text zum Sagen |
| voice_id | string | Ja | Stimm-ID |
| background | string | Nein | Hintergrundszene |
| speed | float | Nein | Sprechgeschwindigkeit (0.5-2.0) |
| emotion | string | Nein | Emotionsstil |
| caption | boolean | Nein | Untertitel aktivieren |

### Verfügbare Emotionen

| Emotion | Beschreibung |
|---------|--------------|
| Neutral | Standardausdruck |
| Happy | Fröhlicher Ausdruck |
| Sad | Trauriger Ausdruck |
| Angry | Wütender Ausdruck |
| Surprised | Überraschter Ausdruck |

## Best Practices

### 1. Skript-Schreiben

- Halten Sie Sätze kurz und klar
- Verwenden Sie natürliche Sprache
- Vermeiden Sie komplexe Terminologie

### 2. Avatar-Auswahl

- Passen Sie den Avatar an den Anwendungsfall an
- Berücksichtigen Sie Publikumspräferenzen
- Testen Sie mehrere Avatare

### 3. Stimmenauswahl

- Passen Sie die Stimme an die Marke an
- Berücksichtigen Sie Sprachanforderungen
- Testen Sie mit Beispielskripten

### 4. Video-Optimierung

- Verwenden Sie die entsprechende Auflösung
- Optimieren Sie für die Plattform (Social Media, Website)
- Fügen Sie Untertitel für Barrierefreiheit hinzu

## Fehlerbehebung

### Videoqualitätsprobleme

**Problem:** Video sieht verpixelt aus

**Lösungen:**
- Höhere Auflösungseinstellung verwenden
- Stellen Sie sicher, dass das Quellbild von hoher Qualität ist
- Internetverbindung überprüfen

### Audio-Probleme

**Problem:** Audio passt nicht zum Video

**Lösungen:**
- Verwenden Sie die bereitgestellten Stimmen
- Stellen Sie sicher, dass Audio korrekt formatiert ist
- Überprüfen Sie die Lippen-sync-Einstellungen

### Verarbeitungszeit

**Problem:** Video dauert zu lange

**Lösungen:**
- Guthaben überprüfen
- Warteschlange für Stapelverarbeitung verwenden
- Prioritätsverarbeitung in Betracht ziehen
