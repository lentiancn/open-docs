# Preguntas Frecuentes de JMeter

## Preguntas Generales

### ¿Qué es Apache JMeter?
Apache JMeter es un software de código abierto diseñado para pruebas de carga y medición de rendimiento. Es una aplicación 100% pura Java que puede probar comportamiento funcional y medir rendimiento.

### ¿Qué protocolos soporta JMeter?
JMeter soporta múltiples protocolos:
- HTTP/HTTPS
- Servicios Web SOAP/REST
- FTP
- JDBC (Base de datos)
- LDAP
- JMS (Middleware orientado a mensajes)
- SMTP/POP3/IMAP (Correo)
- TCP
- Objetos Java

### ¿Es JMeter un navegador?
No, JMeter no es un navegador. Funciona a nivel de protocolo y no ejecuta JavaScript ni renderiza páginas HTML como un navegador real.

## Preguntas de Instalación

### ¿Cuáles son los requisitos del sistema?
- Java 8 o superior
- Se recomienda al menos 4GB de RAM
- SO compatible: Windows, Linux, macOS

### ¿Cómo instalo JMeter?
1. Descargar de https://jmeter.apache.org/
2. Extraer el archivo
3. Ejecutar jmeter.sh (Linux/Mac) o jmeter.bat (Windows)

### ¿Por qué no inicia JMeter?
- Verificar que Java esté instalado: `java -version`
- Verificar que JAVA_HOME esté configurado
- Verificar conflictos de puertos

### ¿Cómo aumento la memoria para JMeter?
Editar el script de inicio de jmeter y modificar la variable HEAP:
```bash
HEAP="-Xms1g -Xmx4g"
```

## Preguntas de Uso

### ¿Cómo creo un plan de prueba básico?
1. Agregar un Grupo de Hilos
2. Agregar Sampler de Solicitud HTTP
3. Agregar Listeners (Árbol de Resultados, Informe Resumido)
4. Ejecutar la prueba

### ¿Qué es un Grupo de Hilos?
Grupo de Hilos define un grupo de usuarios virtuales que ejecutarán el mismo test. Controla:
- Número de hilos (usuarios)
- Período de ramp-up
- Contador de bucles

### ¿Cómo extraigo datos de respuestas?
Usar extractores:
- Extractor JSON para respuestas JSON
- Extractor XPath para XML/HTML
- Extractor de Expresión Regular para cualquier texto

### ¿Cómo parametrizar pruebas?
Usar variables y funciones:
- `${variable}` para referencias de variables
- `${__Random(1,100)}` para valores aleatorios
- `${__time()}`

### ¿Puede JMeter manejar autenticación?
Sí, JMeter soporta:
- Autenticación Básica
- Autenticación NTLM
- Gestor de Cookies
- Gestor de Autorización

## Preguntas de Rendimiento

### ¿Cuántos usuarios puede simular JMeter?
El número depende de su hardware. Con ajustes adecuados:
- Modo GUI: 100-300 usuarios
- Modo CLI: 1000+ usuarios
- Pruebas distribuidas: 10,000+ usuarios

### ¿Qué son las pruebas distribuidas?
Las pruebas distribuidas usan múltiples máquinas para generar más carga. Un maestro coordina múltiples máquinas slaves.

### ¿Cómo ejecuto JMeter en modo headless?
```bash
jmeter -n -t testplan.jmx -l results.jtl
```

## Solución de Problemas

### OutOfMemoryError durante la prueba
- Aumentar tamaño del heap
- Reducir número de hilos
- Usar modo CLI en lugar de GUI
- Desactivar listeners innecesarios

### Resultados no mostrados
- Verificar si la prueba está ejecutándose
- Verificar que el listener esté vinculado al elemento correcto
- Verificar archivos de log para errores

### Tiempos de respuesta altos
- Verificar la red
- Verificar rendimiento del servidor
- Reducir carga si el servidor está saturado

### ¿Cómo depurar mi plan de prueba?
1. Usar listener Árbol de Resultados
2. Habilitar logging de depuración
3. Comenzar con un solo hilo
4. Verificar assertions

## Mejores Prácticas

1. Siempre usar modo CLI para pruebas de carga
2. Comenzar con carga baja y aumentar gradualmente
3. Usar assertions para validar respuestas
4. Limpiar después de pruebas (tearDown)
5. Mantener planes de prueba en control de versiones
6. Monitorear recursos del sistema
