# OpenClaw 🦞

OpenClaw는 좋아하는 채팅 앱——WhatsApp, Telegram, Discord, iMessage 등——을 AI 코딩 도우미(예: Pi)에 연결하는 **자체 호스팅 게이트웨이**입니다. 자체 컴퓨터(또는 서버)에서 단일 Gateway 프로세스를 실행하면 메시징 앱과 항상 사용 가능한 AI 도우미 사이의 다리가 됩니다.

## 대상 사용자

OpenClaw는 데이터를 제어권을 유지하면서 어디서든 메시지를 보낼 수 있는 개인 AI 도우미를 원하는 개발자와 파워 유저를 위해 설계되었습니다. 호스팅 서비스에 의존하지 않습니다.

## 주요 특징

OpenClaw가 다른 점:

- **자체 호스팅**: 내 하드웨어에서 실행, 내 규칙
- **멀티 채널**: 하나의 Gateway가 WhatsApp, Telegram, Discord 등을 동시에 지원
- **에이전트 네이티브**: 도구 사용, 세션, 메모리 및 멀티 에이전트 라우팅을 갖춘 코딩 에이전트용
- **오픈 소스**: MIT 라이선스, 커뮤니티 주도

## 필요한 것

- Node.js 22 이상
- 선택한 AI 제공자의 API 키
- 5분

최상의 품질과 보안을 위해 사용 가능한 최신 세대中最强大的 모델을 사용하세요.

## 빠른 시작

```bash
# OpenClaw 설치
npm install -g openclaw@latest

# 온보딩 마법사 실행
openclaw onboard --install-daemon

# 채널 로그인 및 Gateway 시작
openclaw channels login
openclaw gateway --port 18789
```

## 작동 방식

Gateway는 세션, 라우팅 및 채널 연결의 단일 진실 공급원입니다. 채팅 앱은 이를 통해 AI 에이전트, CLI, 웹 제어 UI 또는 모바일 노드에 연결됩니다.

## 핵심 기능

- **멀티 채널 게이트웨이**: 단일 Gateway 프로세스로 WhatsApp, Telegram, Discord, iMessage 지원
- **플러그인 채널**: 확장 패키지로 Mattermost 등 추가
- **멀티 에이전트 라우팅**: 에이전트, 작업 공간 또는 발신자별로 격리된 세션
- **미디어 지원**: 이미지, 오디오 및 문서 전송 및 수신
- **웹 제어 UI**: 채팅, 구성, 세션 및 노드를 위한 브라우저 대시보드
- **모바일 노드**: Canvas, 카메라/화면 및 음성 활성화 워크플로우를 위한 iOS/Android 노드 페어링
