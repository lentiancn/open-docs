# Docker Nutzungsanleitung

Grundlegende Docker-Befehle.

---

## Grundkonzepte

| Begriff | Beschreibung |
|---------|--------------|
| Image | Vorlage für Container |
| Container | Laufende Instanz eines Images |
| Registry | Speicherort für Images |

---

## Image-Befehle

```bash
docker pull nginx      # Herunterladen
docker images          # Auflisten
docker rmi nginx       # Löschen
docker build -t name . # Bauen
```

---

## Container-Befehle

```bash
docker run -d nginx              # Hintergrund
docker run -d -p 8080:80 nginx   # Port
docker run -d -v /data:/data nginx # Volume

docker ps             # Auflisten
docker stop <id>     # Stoppen
docker start <id>    # Starten
docker rm <id>       # Löschen
docker logs <id>     # Logs
docker exec -it <id> bash # Shell
```

---

## Docker Compose

```bash
docker compose up -d    # Starten
docker compose down    # Stoppen
```
