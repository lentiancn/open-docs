# Introducción a MySQL

MySQL es el sistema de gestión de bases de datos relacionales de código abierto más popular del mundo. Es ampliamente utilizado para aplicaciones web y es un componente central del stack LAMP (Linux, Apache, MySQL, PHP/Python/Perl).

## ¿Qué es MySQL?

MySQL es un sistema de base de datos potente, rápido y confiable que almacena y gestiona datos para aplicaciones. Utiliza SQL (Structured Query Language) como lenguaje de consulta.

### Características principales

- **Código abierto** - Uso gratuito bajo licencia GPL
- **Alto rendimiento** - Optimizado para velocidad y eficiencia
- **Escalabilidad** - Maneja grandes bases de datos con facilidad
- **Seguridad** - Robusta gestión de usuarios y cifrado
- **Replicación** - Soporte master-slave y master-master
- **Motores de almacenamiento** - InnoDB, MyISAM, Memory y más
- **Multiplataforma** - Funciona en Windows, Linux, macOS y más

1. Optimización de consultas
