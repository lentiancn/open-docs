# Guía de Uso de Kotlin

Esta guía presenta la sintaxis básica y las características avanzadas de Kotlin.

## Inicio Rápido

### Hello World

```kotlin
fun main() {
    println("Hello, World!")
}
```

### Variables

```kotlin
// Inmutable
val name: String = "Kotlin"

// Mutable
var age: Int = 10
age = 11
```

## Tipos Básicos

### Números

```kotlin
val int: Int = 100
val long: Long = 100L
val double: Double = 100.0
val float: Float = 100.0f
```

### Cadenas

```kotlin
val str: String = "Hello"
val template = "Name: $name, Age: $age"
```

### Arrays

```kotlin
val arr = arrayOf(1, 2, 3)
val intArray = intArrayOf(1, 2, 3)
```

## Control de Flujo

### Condiciones

```kotlin
val max = if (a > b) a else b

// Expresión When
val result = when (x) {
    1 -> "one"
    2 -> "two"
    else -> "other"
}
```

### Bucles

```kotlin
// Bucle for
for (i in 1..10) {
    println(i)
}

// While
while (x > 0) {
    x--
}
```

## Funciones

### Funciones Básicas

```kotlin
fun sum(a: Int, b: Int): Int {
    return a + b
}

// Forma abreviada
fun sum(a: Int, b: Int) = a + b
```

### Parámetros por Defecto

```kotlin
fun greet(name: String = "World") {
    println("Hello, $name!")
}
```

### Parámetros Vararg

```kotlin
fun sum(vararg numbers: Int): Int {
    return numbers.sum()
}
```

## Orientado a Objetos

### Clases

```kotlin
class Person(val name: String, var age: Int) {
    
    fun speak() {
        println("My name is $name")
    }
}
```

### Herencia

```kotlin
open class Animal(val name: String) {
    open fun sound() = ""
}

class Dog(name: String) : Animal(name) {
    override fun sound() = "Woof!"
}
```

### Clases de Datos

```kotlin
data class User(val name: String, val email: String)
```

### Singleton

```kotlin
object Database {
    fun connect() { }
}
```

## Colecciones

### Listas

```kotlin
val list = listOf(1, 2, 3)
val mutableList = mutableListOf(1, 2, 3)
```

### Mapas

```kotlin
val map = mapOf("a" to 1, "b" to 2)
```

### Operaciones de Colección

```kotlin
val list = listOf(1, 2, 3, 4, 5)

list.filter { it > 2 }           // [3, 4, 5]
list.map { it * 2 }              // [2, 4, 6, 8, 10]
list.reduce { acc, i -> acc + i } // 15
```

## Seguridad Nula

### Tipos Nullable

```kotlin
var name: String? = null
name = "Kotlin"
```

### Operadores de Seguridad

```kotlin
val length = name?.length
val len = name?.length ?: 0
```

### Aserción No Nula

```kotlin
val len = name!!.length
```

## Corrutinas

### Uso Básico

```kotlin
import kotlinx.coroutines.*

suspend fun fetchData(): String {
    delay(1000)
    return "Data"
}

fun main() = runBlocking {
    val result = fetchData()
    println(result)
}
```

### Paralelismo

```kotlin
suspend fun fetchAll() = coroutineScope {
    val a = async { fetchData1() }
    val b = async { fetchData2() }
    "${a.await()} ${b.await()}"
}
```

## Extensiones

### Funciones de Extensión

```kotlin
fun String.addExclamation() = this + "!"

val str = "Hello".addExclamation() // "Hello!"
```

### Propiedades de Extensión

```kotlin
val String.lastChar: Char
    get() = this[length - 1]
```

## Genéricos

```kotlin
class Box<T>(val value: T)

fun <T> printValue(value: T) {
    println(value)
}
```

## Mejores Prácticas

### 1. Preferir val

```kotlin
// Recomendado
val immutable = "value"

// Evitar
var mutable = "value"
```

### 2. Usar Clases de Datos

```kotlin
// Recomendado
data class User(val name: String, val age: Int)
```

### 3. Evitar Null

```kotlin
// Usar valores por defecto
val name = nullableName ?: "default"
```

### 4. Usar Corrutinas para Async

```kotlin
suspend fun loadData() = withContext(Dispatchers.IO) {
    // Operaciones asíncronas
}
```
