# Element Plus Introduction

Element Plus est une bibliothèque de composants UI basée sur Vue 3.

## Caractéristiques

- Composants Vue 3
- 70+ composants
- TypeScript
- Importation à la demande
- Internationalisation
- Thèmes personnalisables

## Scénarios

- Systèmes de gestion d'entreprise
- Plateformes de commerce électronique

Pour plus d'informations, consultez le [site officiel](https://element-plus.org/).
