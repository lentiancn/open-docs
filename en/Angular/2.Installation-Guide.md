# Angular Installation Guide

Complete Angular development environment setup.

## Environment Requirements

### Hardware Requirements

- CPU: 2 cores or more
- Memory: 4GB or more
- Disk: 10GB or more

### Software Requirements

- Operating System: Windows/macOS/Linux
- Node.js: 18.10.0+ or 20.0.0+
- npm: 9.0.0+ (included with Node.js)
- Git: Any version

## Installation Steps

### 1. Install Node.js

**Windows/macOS:**

Download from: https://nodejs.org

Or use nvm:

```bash
# Install nvm (macOS/Linux)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Install Node.js LTS
nvm install --lts
nvm use --lts
```

**Verify installation:**

```bash
node --version  # Should show v18.x.x or v20.x.x
npm --version   # Should show 9.x.x or higher
```

### 2. Install Angular CLI

```bash
# Install globally via npm
npm install -g @angular/cli

# Verify installation
ng version
```

### 3. Create New Project

```bash
# Create new Angular application
ng new my-app

# Navigate to project
cd my-app

# Start development server
ng serve
```

### 4. Access Application

Open browser: http://localhost:4200

## Common Commands

### Project Commands

```bash
# Create new project with routing and SCSS
ng new my-app --routing --style=scss

# Create standalone app (Angular 14+)
ng new my-app --standalone

# Start development server
ng serve

# Build for production
ng build

# Run unit tests
ng test

# Run E2E tests
ng e2e
```

### Generate Commands

```bash
# Generate component
ng generate component my-component
ng g c my-component

# Generate service
ng generate service my-service
ng g s my-service

# Generate directive
ng generate directive my-directive
ng g d my-directive

# Generate pipe
ng generate pipe my-pipe
ng g p my-pipe

# Generate module
ng generate module my-module
ng g m my-module

# Generate class
ng generate class my-class
ng g cl my-class

# Generate interface
ng generate interface my-interface
ng g i my-interface

# Generate guard
ng generate guard my-guard
ng g g my-guard

# Generate interceptor
ng generate interceptor my-interceptor
ng g in my-interceptor
```

## Environment Configuration

### Development Environment

```typescript
// src/environments/environment.ts
export const environment = {
  production: false,
  apiUrl: 'http://localhost:3000/api'
};
```

### Production Environment

```typescript
// src/environments/environment.prod.ts
export const environment = {
  production: true,
  apiUrl: 'https://api.example.com'
};
```

### Using Environment

```typescript
import { Component } from '@angular/core';
import { environment } from '../environments/environment';

@Component({
  selector: 'app-root',
  template: `<p>API: {{ apiUrl }}</p>`
})
export class AppComponent {
  apiUrl = environment.apiUrl;
}
```

## Common Issues

### npm Install Slow?

```bash
# Use Taobao mirror
npm config set registry https://registry.npmmirror.com

# Or use nrm to manage mirrors
npm install -g nrm
nrm use taobao
```

### Node Version Mismatch?

```bash
# Use nvm to switch version
nvm install 20
nvm use 20
```

### Permission Issues (Linux/macOS)

```bash
# Fix npm permissions
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc
```

## Uninstall

```bash
# Uninstall Angular CLI
npm uninstall -g @angular/cli

# Delete project
rm -rf my-app
```
