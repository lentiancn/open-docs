# Bootstrap 常見問題

常見問題解答。

## 安裝問題

### Q: CDN 無法加載？

**解決方案**：
1. 檢查網絡連接
2. 確認 URL 正確
3. 使用備用 CDN

### Q: npm 安裝失敗？

**解決方案**：
1. 清除 npm 緩存
2. 檢查 Node.js 版本
3. 使用管理員權限

### Q: 還需要 jQuery 嗎？

**解決方案**：
不需要。Bootstrap 5 已移除 jQuery 依賴。

## 使用問題

### Q: 樣式不生效？

**解決方案**：
1. 確認 CSS 文件正確引入
2. 檢查 CSS 引入順序
3. 清理瀏覽器緩存

### Q: 響應式不工作？

**解決方案**：
添加 viewport meta 標籤：
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### Q: 組件不工作？

**解決方案**：
1. 確認引入 Bootstrap JavaScript 文件
2. 檢查 Popper.js 是否正確引入

## 兼容性問題

### Q: 不支持 IE 瀏覽器？

Bootstrap 5 不再支持 IE 10 以下。

## 性能問題

### Q: 文件太大？

**解決方案**：
1. 使用 SASS 按需引入
2. 使用生產版本（.min 文件）
