# Jira

Jira es un software de gestión de proyectos y seguimiento de problemas desarrollado por Atlassian, proporcionando soluciones de gestión de proyectos para más de 300,000 empresas en todo el mundo.

## Inicio Rápido

Visita el sitio web de Atlassian para registrarte y elige un plan adecuado para tu equipo. Puedes crear un sitio de Jira rápidamente.

## Características Principales

Gestión de proyectos y seguimiento de problemas, Soporte para desarrollo ágil (Scrum, Kanban), Colaboración y comunicación en equipo, Integración con asistente de IA Rovo, Rich third-party integrations, Powerful automation rules.

## Precios

Free: 10 usuarios permanentemente gratuito

Standard: $7.50 por usuario al mes

Premium: $14.50 por usuario al mes

Enterprise: Contacta a ventas para precios
