# Flutter

Google 推出的開源 UI 工具包。

## 簡介

Flutter 是一個用於構建跨平台應用的 UI 工具包，使用 Dart 語言。

## 功能特點

- **跨平台**：單一代碼庫支持多平台
- **熱重載**：即時預覽更改
- **豐富組件**：Material Design + Cupertino
- **高性能**：原生渲染體驗
- **開源免費**

## 快速開始

### 安裝

參考安裝指南：./2.安裝指南.md

### 創建項目

```bash
flutter create my_app
cd my_app
flutter run
```

### 項目結構

```
lib/
└── main.dart        # 入口文件
pubspec.yaml         # 配置文件
```

## 核心概念

- **Dart**：Flutter 開發語言
- **Widget**：UI 構建基礎
- **State**：狀態管理
- **Platform Channel**：平台通道
