# Element Plus ユーザーマニュアル

Element Plusの主要機能と使用ヒント。

## クイックスタート

### 最初の例

```vue
<template>
  <el-button type="primary" @click="handleClick">
    クリックしてね
  </el-button>
</template>

<script setup>
import { ref } from 'vue'

const handleClick = () => {
  ElMessage.success('こんにちはElement Plus!')
}
</script>
```

### Form（フォーム）

```vue
<template>
  <el-form :model="form" :rules="rules" ref="formRef" label-width="100px">
    <el-form-item label="ユーザー名" prop="username">
      <el-input v-model="form.username" />
    </el-form-item>
    
    <el-form-item label="メール" prop="email">
      <el-input v-model="form.email" />
    </el-form-item>
    
    <el-form-item label="役割" prop="role">
      <el-select v-model="form.role" placeholder="役割を選択">
        <el-option label="管理者" value="admin" />
        <el-option label="一般ユーザー" value="user" />
      </el-select>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm">送信</el-button>
      <el-button @click="resetForm">リセット</el-button>
    </el-form-item>
  </el-form>
</template>

<script setup>
import { ref, reactive } from 'vue'

const formRef = ref()
const form = reactive({
  username: '',
  email: '',
  role: '',
})

const rules = {
  username: [{ required: true, message: 'ユーザー名を入力してください', trigger: 'blur' }],
  email: [
    { required: true, message: 'メールを入力してください', trigger: 'blur' },
    { type: 'email', message: '正しいメール形式を入力してください', trigger: 'blur' }
  ],
  role: [{ required: true, message: '役割を選択してください', trigger: 'change' }],
}

const submitForm = async () => {
  const valid = await formRef.value.validate()
  if (valid) {
    console.log('送信成功', form)
  }
}

const resetForm = () => {
  formRef.value.resetFields()
}
</script>
```

### Message（メッセージ）

```typescript
import { ElMessage } from 'element-plus'

// 成功メッセージ
ElMessage.success('成功しました')

// 警告メッセージ
ElMessage.warning('警告情報')

// エラーメッセージ
ElMessage.error('失敗しました')

// 通常メッセージ
ElMessage.info('情報')

// 長時間表示
ElMessage({
  message: 'カスタム時間',
  duration: 10000,
})
```

### テーブルページネーション

```vue
<template>
  <el-table :data="paginatedData">
    <el-table-column prop="name" label="名前" />
  </el-table>
  
  <el-pagination
    v-model:current-page="currentPage"
    v-model:page-size="pageSize"
    :total="total"
    :page-sizes="[10, 20, 50, 100]"
    layout="total, sizes, prev, pager, next, jumper"
    @size-change="handleSizeChange"
    @current-change="handleCurrentChange"
  />
</template>

<script setup>
import { ref, computed } from 'vue'

const currentPage = ref(1)
const pageSize = ref(10)
const tableData = ref([/* 大量データ */])
const total = computed(() => tableData.value.length)

const paginatedData = computed(() => {
  const start = (currentPage.value - 1) * pageSize.value
  return tableData.value.slice(start, start + pageSize.value)
})
</script>
```

## デバッグのヒント

### コンポーネントの問題を確認

```bash
npm list element-plus
```

### 一般的なエラー

**コンポーネントが登録されていない**: インポートまたは自動インポート設定を確認

**スタイルが適用されていない**: CSS競合を確認、`:deep()`を試す

**型エラー**: `element-plus/global`タイプをインストール
