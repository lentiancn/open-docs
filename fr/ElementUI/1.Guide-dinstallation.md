# Element UI Guide d'installation

Element UI est une bibliothèque de composants UI pour les applications Vue 2.x.

## Versions

| Version | Version Vue | Statut |
|---------|-------------|--------|
| Element UI 2.x | Vue 2.6+ | Actuel |
| Element UI 1.x | Vue 2.x | Hérité |

**Note:** Element UI est pour Vue 2 uniquement. Pour Vue 3, utilisez Element Plus.

## Conditions Préalables

### Exigences Système

| Exigence | Version |
|-----------|---------|
| Node.js | 8.0+ |
| npm | 5.0+ |
| Vue | 2.6+ |
| Navigateur | IE 10+, Chrome, Firefox, Safari |

## Méthodes d'Installation

### Méthode 1: npm (Recommandé)

```bash
# Installer Element UI
npm install element-ui --save
```

### Méthode 2: CDN

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Démo Element UI CDN</title>
  <!-- Importer Vue 2 -->
  <script src="https://unpkg.com/vue@2.6.14/dist/vue.js"></script>
  <!-- Importer les styles Element UI -->
  <link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
  <!-- Importer Element UI -->
  <script src="https://unpkg.com/element-ui/lib/index.js"></script>
</head>
```

## Configuration du Projet

### Étape 1: Créer un Projet Vue

```bash
npm install -g @vue/cli
vue create my-element-app
cd my-element-app
```

### Étape 2: Installer Element UI

```bash
npm install element-ui --save
```

### Étape 3: Importer dans main.js

```javascript
import Vue from 'vue'
import App from './App.vue'

import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)

Vue.config.productionTip = false

new Vue({
  render: h => h(App)
}).$mount('#app')
```

## Import à la Demande

D'abord, installer les plugins:

```bash
npm install babel-plugin-component -D
```

Configurer babel.config.js:

```javascript
module.exports = {
  presets: ['@vue/cli-plugin-babel/preset'],
  plugins: [
    [
      'component',
      {
        libraryName: 'element-ui',
        styleLibraryName: 'theme-chalk'
      }
    ]
  ]
}
```

## Dépannage

### Composant Non Trouvé

**Solutions:**
1. Vérifier l'installation: `npm list element-ui`
2. Vérifier l'import dans main.js
3. Vérifier que Vue.use() est appelé

### Styles Non Appliqués

**Solutions:**
1. Importer CSS: `import 'element-ui/lib/theme-chalk/index.css'`
