# jQuery 使用指南

使用 jQuery 的指南。

---

## 基本

```javascript
$('#myId')
$(document).ready(function() {})
```

---

## 選擇器

```javascript
$('#myId')
$('.myClass')
$('div')
```

---

## 事件

```javascript
$('#myButton').click(function() {})
```

---

## AJAX

```javascript
$.get('url', function(data) {})
$.post('url', data, function(data) {})
```
