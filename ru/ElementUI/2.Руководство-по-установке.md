# Руководство по установке Element UI

## Требования

- Vue 2.0.0+
- Node.js 8.0.0+

## Установка

npm install element-ui --save

## Полный импорт

import Vue from 'vue'
import Element from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(Element)

## Импорт по требованию

Сначала установите babel-plugin-component:

npm install babel-plugin-component -D

Настройте .babelrc:

{
  "plugins": [
    ["component", {
      "libraryName": "element-ui",
      "styleLibraryName": "theme-chalk"
    }]
  ]
}

## Глобальная конфигурация

Vue.use(Element, {
  size: 'small',
  zIndex: 2000
})

Для получения дополнительной информации посетите официальный веб-сайт Element UI.
