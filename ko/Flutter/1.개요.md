# Flutter - 개요

Flutter는 Google의 오픈소스 UI 툴킷.

## Flutter란?

Flutter는 단일 코드베이스로 iOS, Android, Web, Desktop용 크로스플랫폼 앱을 만들기 위한 UI 툴킷. Dart 사용.

## 기능

- **크로스플랫폼**: 하나의 코드베이스
- **핫 리로드**: 즉각 미리보기
- **풍부한 위젯**: Material Design + Cupertino
- **고성능**: Skia 그래픽 엔진
- **오픈소스**: 무료
