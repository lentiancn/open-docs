# Preguntas Frecuentes

Este documento recopila las preguntas técnicas más comunes y sus respuestas durante el uso de Apifox.

## Preguntas Básicas

### ¿Qué es Apifox?

Apifox es una plataforma de colaboración para APIs que integra diseño de APIs, gestión de documentación, depuración, Mock y pruebas automatizadas. Ayuda a los equipos de desarrollo a realizar el desarrollo, pruebas y mantenimiento de documentación de APIs de manera más eficiente.

### ¿Apifox es gratuito?

Apifox ofrece versiones gratuita y de pago. La versión gratuita ya incluye todas las funciones principales, lo cual es suficiente para la mayoría de las necesidades de desarrollo. La versión de pago ofrece funciones más avanzadas, como equipos más grandes y automatización avanzada.

### ¿Cuál es la diferencia entre Apifox y Postman?

Tanto Apifox como Postman son herramientas para APIs, pero Apifox es más adecuado para usuarios chinos: mejor soporte en chino, funciones de Mock más inteligentes y una experiencia de colaboración en equipo más completa. Las funciones básicas de Apifox son completamente gratuitas, mientras que muchas funciones de Postman requieren pago.

## Problemas de Instalación

### ¿Qué hago si la descarga es lenta?

Apifox ofrece múltiples servidores de descarga. Puedes intentar acceder a https://apifox.com/ y elegir el servidor más cercano a ti para descargar.

### ¿Qué hago si no puedo iniciar la aplicación?

1. Verifica si la versión de tu sistema cumple con los requisitos
2. Intenta ejecutar como administrador
3. Verifica si tienes las librerías necesarias instaladas
4. Consulta los registros de error para obtener más información

### ¿Qué hago si el inicio de sesión falla?

1. Verifica si la conexión a internet funciona correctamente
2. Confirma que el nombre de usuario y contraseña sean correctos
3. Intenta iniciar sesión con una cuenta de tercero
4. Contacta al soporte técnico oficial para obtener ayuda

## Problemas de Uso

### ¿Los datos de la interfaz no se sincronizan?

1. Confirma que hayas iniciado sesión
2. Verifica la conexión a internet
3. Haz clic en el botón "Sincronizar" para sincronizar manualmente
4. Verifica los permisos de tu cuenta

### ¿Los datos Mock no funcionan?

1. Confirma que la función Mock esté activada
2. Verifica si la dirección Mock es correcta
3. Revisa si hay errores de sintaxis en la interfaz
4. Intenta actualizar la página

### ¿El envío de solicitudes falla?

1. Verifica la conexión a internet
2. Confirma que la dirección de solicitud sea correcta
3. Verifica el formato de los parámetros de solicitud
4. Revisa si está siendo bloqueado por un proxy

### ¿La respuesta se muestra con caracteres raros?

1. Intenta cambiar la codificación de la respuesta
2. Verifica la configuración de codificación en los encabezados de respuesta
3. Usa la función "Formatear" para reanalizar

## Problemas de Funciones

### ¿Cómo importar datos de Postman?

1. Haz clic en "Importar"
2. Selecciona "Postman"
3. Selecciona el archivo a importar
4. Confirma la importación

### ¿Cómo exportar a Swagger?

1. Entra en la configuración del proyecto
2. Selecciona "Exportar"
3. Elige "Swagger/OpenAPI"
4. Selecciona la versión
5. Haz clic en exportar

### ¿Cómo usar variables de entorno?

1. Define las variables de entorno en la configuración del proyecto
2. Usa {{nombreVariable}} en las solicitudes para hacer referencia
3. Cambia de entorno usando el selector de entorno en la parte superior

### ¿Cómo realizar operaciones por lotes en interfaces?

1. Mantén presionado Ctrl/Cmd para seleccionar múltiples interfaces
2. Haz clic derecho para seleccionar operación por lotes
3. Elige el tipo de operación (eliminar, exportar, mover, etc.)

## Problemas de Pruebas

### ¿Cómo ejecutar pruebas automatizadas?

1. Abre el conjunto de pruebas
2. Haz clic en el botón "Ejecutar"
3. Verifica los resultados de las pruebas

### ¿Qué hago si una aserción falla?

1. Verifica el motivo del fallo
2. Compara el valor esperado con el valor real
3. Modifica las condiciones de la aserción
4. Vuelve a ejecutar las pruebas

### ¿Cómo exportar el informe de pruebas?

1. Después de ejecutar las pruebas
2. Haz clic en "Exportar Informe"
3. Elige el formato (HTML, PDF, JSON)

## Problemas de Colaboración en Equipo

### ¿Cómo invitar miembros al equipo?

1. Entra en la configuración del proyecto
2. Haz clic en "Gestión de Miembros"
3. Haz clic en "Invitar Miembro"
4. Ingresa el correo electrónico o nombre de usuario del miembro

### ¿Cómo configurar los permisos de los miembros?

Al invitar miembros, puedes configurar diferentes roles:

- Administrador: Control total
- Editor: Puede editar y ejecutar
- Visor: Solo lectura

### ¿Cómo resolver conflictos?

Cuando múltiples personas editan la misma interfaz simultáneamente:

1. El sistema mostrará una notificación de conflicto
2. Elige si mantener la versión local o la versión remota
3. O fusiona los cambios manualmente

## Problemas de Datos

### ¿Cómo hacer una copia de seguridad de los datos?

1. Exporta regularmente el proyecto en formato Apifox
2. O usa la función de sincronización en la nube

### ¿Cómo recuperar datos?

1. Si tienes un archivo de copia de seguridad, haz clic en "Importar"
2. Selecciona el archivo de copia de seguridad para recuperar

### ¿Puedo recuperar los datos eliminados?

Los datos eliminados se conservan en la papelera durante 30 días y se pueden recuperar desde allí.

## Problemas de Rendimiento

### ¿Qué hago si la aplicación está lenta?

1. Cierra los proyectos que no necesites
2. Limpia la caché del navegador
3. Actualiza la configuración de hardware
4. Intenta usar la última versión

### ¿La importación de grandes cantidades de datos es lenta?

1. Importa en lotes
2. Desactiva la sincronización automática
3. Ten paciencia y espera a que complete la importación
