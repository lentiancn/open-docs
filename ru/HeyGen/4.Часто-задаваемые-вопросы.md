# HeyGen Часто задаваемые вопросы

## Основные вопросы

### Что такое HeyGen

HeyGen — это платформа генерации видео с использованием искусственного интеллекта, ориентированная на создание реалистичных видео с цифровыми аватарами. Пользователи могут быстро генерировать видеоконтент с озвучкой от аватаров ИИ, вводя текст, загружая изображения или аудио. Платформа использует передовые технологии глубокого обучения для генерации видео с цифровыми аватарами с естественными выражениями и синхронизацией губ, подходящих для маркетинговых видео, корпоративного обучения, онлайн-образования, создания контента и других сценариев. Без опыта производства видео пользователи могут создавать видеоработы профессионального уровня.

Основная технология HeyGen основана на генеративных моделях искусственного интеллекта, включая Sora, Veo, Kling для генерации видео, Flux для генерации изображений и ElevenLabs для синтеза речи. Платформа поддерживает более 175 языков и диалектов, являясь одним из ведущих в мире инструментов генерации видео с помощью ИИ.

### HeyGen бесплатен

HeyGen предлагает бесплатную и платные версии. Бесплатная версия позволяет создавать 3 видео в месяц длиной до 3 минут каждое, подходит для ознакомления с базовыми функциями платформы. Платные версии включают Creator (29$/месяц), Pro (99$/месяц) и Business (149$/месяц), разблокирующие все расширенные функции и более высокие квоты использования.

Бесплатная пробная версия не требует кредитной карты, вы можете начать использовать сразу после регистрации. Платные версии обычно включают неограниченную генерацию видео, более высокие ограничения по длительности, настраиваемые аватары, функции командной работы и другие услуги с добавленной стоимостью.

### Какие языки поддерживаются

HeyGen поддерживает более 175 языков и диалектов, включая китайский, английский, японский, корейский, французский, немецкий, испанский, русский и другие. Функция клонирования голоса также поддерживает многоязычное обучение голосу, удовлетворяя потребностям создания международного контента.

Функция многоязычного перевода сохраняет голос, тон и стиль выражения оригинального спикера, обеспечивая естественный многоязычный видеоконтент.

### Каковы основные функции

Основные функции HeyGen включают: Текст в видео, Фото в видео, Реклама продуктов, Создание UGC-рекламы, Генерация аватаров ИИ (Фото-аватар, Публичный аватар, Цифровой двойник), Перевод видео ИИ, Редактор AI Studio и Интеграция API.

Пользователи также могут получить доступ к новейшим генеративным моделям ИИ, включая Sora, Veo, Kling для генерации видео, Flux для генерации изображений и ElevenLabs для синтеза речи.

## Технические вопросы

### Качество видео

Качество видео, генерируемого HeyGen, зависит от выбранных параметров. Платформа поддерживает несколько разрешений от 720p до 4K. В режиме HD детали лица аватара, изменения выражений и синхронизация губ очень реалистичны, что затрудняет для обычных пользователей отличить сгенерированное ИИ видео от реальной видеозаписи.

Разрешение 4K подходит для отображения на большом экране или профессионального использования. 1080p — лучший выбор для платформ социальных сетей, обеспечивая баланс между чёткостью и скоростью загрузки.

### Скорость генерации

Скорость генерации видео зависит от нескольких факторов, включая длину видео, нагрузку на сервер и сложность выбранных эффектов. Обычно видео длиной 1 минуту можно сгенерировать за несколько минут. В пиковые периоды может потребоваться более длительное ожидание.

Платные версии (Creator, Pro) обеспечивают более быструю обработку видео. Рекомендуется заранее планировать время производства видео, чтобы избежать спешки.

### Как обеспечивается безопасность данных

HeyGen уделяет приоритетное внимание безопасности данных пользователей. Все загруженные фотографии и аудио сохраняются после шифрования. Платформа прошла множество международных сертификатов безопасности, включая GDPR, SOC 2 Type II, CCPA, AI ACT и DPF, соответствующих международным стандартам защиты данных.

Корпоративные пользователи могут запросить решения для частного развёртывания для полного контроля над данными. Платформа также предоставляет функции безопасности корпоративного уровня, включая детальное управление правами, контроль квот использования и журналы аудита.

### Можно ли использовать видео в коммерческих целях

Видео, созданные с помощью платных версий, можно использовать в коммерческих целях. Корпоративные пользователи могут использовать видео для продвижения продуктов, маркетинга бренда, обслуживания клиентов и других коммерческих сценариев. Следует отметить, что HeyGen нельзя использовать для создания ложных новостей, вводящего в заблуждение контента или контента, нарушающего права других. Подробные условия использования см. в соглашении об обслуживании.

## Вопросы по использованию

### Как создать настраиваемый цифровой аватар

Шаги по созданию настраиваемого цифрового аватара: Перейдите на страницу управления «Аватар»; нажмите «Создать настраиваемый аватар»; загрузите портретное фото (рекомендуется использовать чёткое фронтальное фото высокого разрешения); дождитесь, пока система проанализирует и сгенерирует цифровой аватар; при необходимости отрегулируйте параметры черт лица; сохраните настраиваемый аватар для дальнейшего использования.

Настраиваемые аватары можно использовать многократно, выбирая их напрямую при каждом создании видео. Платформа также постоянно оптимизирует эффекты генерации настраиваемых аватаров. Функция Цифрового двойника требует загрузки видео для создания цифровой копии, очень похожей на человека.

### Можно ли создавать несколько видео одновременно

Да, HeyGen поддерживает пакетное создание видео. Вы можете создать несколько задач, и система обработает их по порядку. Платные версии поддерживают более высокие возможности параллельной обработки.

Корпоративные пользователи с большими потребностями в производстве видео могут обратиться в службу поддержки для получения индивидуальных решений.

### Какие форматы видео поддерживаются для экспорта

HeyGen поддерживает экспорт нескольких форматов видео, включая MP4 (наиболее распространённый, лучшая совместимость), MOV (подходит для экосистемы Apple) и AVI. Разрешения поддерживают 720p, 1080p и 4K.

При экспорте можно выбрать, добавлять ли субтитры, которые могут быть встроенными или отдельными файлами SRT. Версии Pro также поддерживают удаление водяного знака.

### Как улучшить качество видео

Способы улучшения качества видео: Выберите разрешение HD или 4K для вывода; используйте чёткие портретные фотографии в качестве настраиваемых аватаров; выбирайте профессиональные сцены, соответствующие контенту; разумно используйте текстовые и графические элементы для улучшения передачи информации; уделяйте внимание качеству и профессионализму сценария.

HeyGen постоянно оптимизирует алгоритмы для улучшения качества генерации. Рекомендуется поддерживать клиент в актуальном состоянии для получения новейших оптимизаций функций и улучшений производительности.

## Вопросы об аккаунте

### Что делать, если забыт пароль

Нажмите «Забыли пароль?» на странице входа и следуйте инструкциям, чтобы ввести адрес электронной почты, указанный при регистрации. Система отправит ссылку для сброса пароля на вашу электронную почту; нажмите на ссылку, чтобы установить новый пароль. Если восстановить не удаётся, рекомендуется обратиться в службу поддержки за помощью.

### Как повысить план подписки

Войдя в систему, перейдите на страницу настроек аккаунта и выберите «Управление подпиской», чтобы увидеть текущий статус подписки и доступные варианты повышения. Нажмите на нужный план и следуйте инструкциям для завершения оплаты. Повышение вступает в силу немедленно, а использованные квоты будут вычтены из нового плана.

Повышение вступает в силу немедленно, а понижение вступит в силу в конце текущего биллингового цикла.

### Возможен ли возврат средств

HeyGen предлагает бесплатные пробные версии; рекомендуется протестировать бесплатную версию или краткосрочные планы перед покупкой. После официальной подписки при возникновении серьёзных проблем с обслуживанием обратитесь в службу поддержки для обсуждения. Конкретную политику возврата см. в условиях обслуживания.

### Как отменить подписку

Вы можете перейти непосредственно на страницу аккаунта, чтобы найти варианты приостановки или отмены подписки. После отмены вы всё ещё можете использовать функции подписки до конца текущего биллингового цикла. По окончании биллингового цикла аккаунт автоматически перейдёт на бесплатную версию.

## Авторские права и соответствие требованиям

### Кому принадлежат авторские права на сгенерированный контент

Видео, созданные пользователями с помощью HeyGen, принадлежат пользователям. Пользователи могут свободно использовать, публиковать и коммерчески использовать эти видео. Однако они должны соблюдать соответствующие правила соответствия контента в соглашении об обслуживании и не должны создавать незаконный или несоответствующий контент.

### Ограничения на использование активов платформы

Предоставленные HeyGen готовые изображения цифровых аватаров, фоновые сцены и аудиоматериалы могут использоваться только на платформе и не могут быть загружены или извлечены для других целей. Бизнес-пользователям, которым требуется более широкая лицензия на использование, могут обратиться в бизнес-команду для получения индивидуальных решений.

### Обработка несоответствующего контента

HeyGen установил полный механизм проверки контента. Система автоматически обнаруживает и блокирует запросы на создание несоответствующего контента. Опубликованные несоответствующие видео могут быть удалены платформой, а связанные аккаунты заблокированы. Серьёзные нарушения могут привести к постоянной блокировке аккаунта и юридической ответственности.

Запрещённый контент включает: ложные новости и вводящая в заблуждение информация; насильственный и ненавистнический контент; порнографический и вульгарный контент; контент, нарушающий авторские права других; любое незаконное использование контента.

## Техническая поддержка

### Как связаться со службой поддержки

HeyGen предоставляет несколько каналов поддержки: онлайн-чат в правом нижнем углу; поддержка по электронной почте; выделенная телефонная поддержка для корпоративных пользователей.

Заявки, оставленные вне рабочего времени, будут обработаны дежурной командой. Корпоративные пользователи получают приоритетный ответ и услуги выделенного менеджера аккаунта.

### Есть ли документация по API

Да, HeyGen предоставляет полную документацию по API. Посетите страницу документации для разработчиков, чтобы ознакомиться с подробными справочными материалами по API, включая аутентификацию, описания конечных точек и примеры кода.

API поддерживает модели оплаты по использованию, включая Video Agent, Video Translation и функции преобразования текста в речь. Через API HeyGen можно глубоко интегрировать в существующие рабочие процессы.
