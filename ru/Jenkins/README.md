# Jenkins

Jenkins — самый популярный сервер автоматизации с открытым исходным кодом для реализации непрерывной интеграции (CI) и непрерывной доставки (CD).

## Обзор

Jenkins предоставляет платформу для автоматизации различных задач в процессе сборки, тестирования и развертывания программного обеспечения. С помощью Jenkins команды разработчиков могут достичь непрерывной доставки, обеспечивая быстрый и надежный выпуск программного обеспечения в производство.

## Основные функции

- **Непрерывная интеграция**: автоматическое обнаружение изменений кода и запуск сборок
- **Непрерывная доставка**: автоматизация процесса выпуска программного обеспечения
- **Конвейер как код**: определение процессов сборки с помощью Jenkinsfile
- **Экосистема плагинов**: более 1800 плагинов доступно
- **Поддержка распределенных сборок**: поддержка многоузловых распределенных сборок
- **Кроссплатформенность**: поддерживает Windows, Linux, macOS

## Быстрый старт

### Запуск с Docker

```bash
docker run -d -p 8080:8080 jenkinsci/blueocean
```

### Создание первого конвейера

1. Установите Jenkins
2. Установите плагин Blue Ocean
3. Откройте Blue Ocean
4. Создайте задачу конвейера
5. Напишите Jenkinsfile
