# Guía de Instalación de MySQL

Instrucciones para instalar MySQL en todas las plataformas.

---

## Docker (Recomendado)

```bash
docker run -d \
  --name mysql \
  -e MYSQL_ROOT_PASSWORD=secret \
  -e MYSQL_DATABASE=mydb \
  -p 3306:3306 \
  mysql:8
```

---

## Linux

### Ubuntu

```bash
sudo apt update
sudo apt install -y mysql-server
sudo systemctl start mysql
```

---

## macOS

```bash
brew install mysql
brew services start mysql
```

---

## Verificar

```bash
mysql --version
mysql -u root -p
```
