# OpenCode よくある質問

よくある質問と回答。

## インストール

### Q: インストールに失敗した？

**解決策**：
1. システム要件を確認（最新のターミナルエミュレータが必要）
2. ネットワーク接続を確認
3. インストールスクリプトを試す：`curl -fsSL https://opencode.ai/install | bash`
4. エラーメッセージを確認して再試行

### Q: Windows にインストールできない？

**解決策**：
1. Chocolatey を試す：`choco install opencode`
2. または Scoop を使用：`scoop install opencode`
3. または NPM を使用：`npm install -g opencode-ai`

### Q: インストール確認に失敗した？

**解決策**：
1. opencode が PATH に追加されているか確認
2. ターミナルを再起動
3. インストールが成功したか確認

## 使用

### Q: LLM に接続できない？

**解決策**：
1. `/connect` コマンドを実行
2. opencode.ai/auth にアクセスしてログイン
3. 請求情報を追加して API キーを取得
4. API キーをターミナルに貼り付け

### Q: 応答が遅い？

**解決策**：
1. ネットワーク接続を確認
2. 他の LLM プロバイダに切り替えを試す
3. 会話のコンテキスト長を短くする

### Q: モデルを切り替える方法は？

`/connect` コマンドで異なる LLM プロバイダを選択。

### Q: 哪些言語をサポート？

75+ LLM プロバイダをサポート：
- OpenAI (GPT-4, Claude など)
- Anthropic (Claude)
- Google (Gemini)
- ローカルモデル
- [Models.dev](https://models.dev) を参照

## 機能

### Q: AGENTS.md とは？

AGENTS.md は OpenCode が初期化時に作成するファイルで、プロジェクト構造とコーディング規則を含み、OpenCode がプロジェクトを理解するのを助けます。

### Q: Plan モードと Build モードの違いは？

- **Plan モード**：計画のみ生成、コードを変更しない
- **Build モード**：直接変更を実行

**Tab** キーでモードを切り替え。

### Q: 会話を共有する方法は？

`/share` コマンドで共有リンクを作成。

## プライバシー

### Q: OpenCode はコードを保存しますか？

いいえ。OpenCode はコードやコンテキストデータを保存しないため、機密環境でも安全に使用できます。

### Q: プライバシーを保護する方法は？

- 会話に機密情報を入力しない
- ローカルモデルを使用
- 会話を定期的にクリア
