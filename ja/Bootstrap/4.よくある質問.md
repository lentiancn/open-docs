# よくある質問

このドキュメントでは、Bootstrapを使う中でよくある技術的な質問とその回答をまとめています。

## 基本的な質問

### Bootstrapは何ですか？

BootstrapはTwitter社が開発・メンテナンスしている、オープンソースの前端開発フレームワークです。豊富なCSSスタイルとJavaScriptコンポーネントを提供しており、開発者が素早くレスポンシブなWebアプリを作れます。

### Bootstrap 5とBootstrap 4の違いは？

主な違い：

- Bootstrap 5はjQuery依存を排除し、ネイティブJavaScriptを使用
- XXLブレークポイントを追加
- フォームコントロールを改良
- Font Awesomeを排除し、Bootstrap Iconsに変更
- Internet Explorerのサポートを終了

### Bootstrapは何が必要？

モダンブラウザさえあればOK。Bootstrap 5はES6対応のブラウザが必要です。

## インストールに関する質問

### CDNの読み込みに失敗したら？

1. ネットワーク接続を確認
2. 他のCDNを試す（bootcdn、staticfileなど）
3. ダウンロードしてローカルで使用

### npmインストールが失敗するんですが...

```bash
# キャッシュをクリア
npm cache clean --force

# 中国ミラーを使用
npm config set registry https://registry.npmmirror.com

# 再インストール
npm install bootstrap@5
```

### スタイルが効かない...

1. CSSファイルが正しく読み込まれているか確認
2. ブラウザキャッシュをクリア（Ctrl+Shift+Rで強制更新）
3. CSSの競合がないか確認

## 使い方の質問

### グリッドシステムが動かない...

1. rowがcontainerの中に入っているか確認
2. colがrowの中に入っているか確認
3. ブレークポイントクラスが正しく使われているか確認

### レスポンシブが効かない...

ビューポートタグが正しく設定されているか確認：

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### ナビゲーションバーがモバイルで折りたたまれない...

1. bootstrap.bundle.min.js（Popper含む）が読み込まれているか確認
2. data-bs-toggleとdata-bs-target属性を確認

### モーダルが開かない...

1. Bootstrap JSが読み込まれているか確認
2. ボタンのdata-bs-toggleとdata-bs-target属性を確認
3. モーダルのIDが正しいか確認

### ドロップダウンが動かない...

1. bootstrap.bundle.min.jsを読み込む
2. data-bs-toggle="dropdown"属性を確認

## スタイルに関する質問

### テーマ色を変更したい！

Sassを使う：

```scss
$primary: #007bff;
@import "bootstrap/scss/bootstrap";
```

### デフォルトスタイルをオーバーライドしたい！

より優先度の高いセレクタを使う：

```css
.navbar {
    background-color: #f8f9fa !important;
}
```

### フォントを変更したい！

```scss
$font-family-sans-serif: "Helvetica Neue", Helvetica, Arial, sans-serif;
@import "bootstrap/scss/bootstrap";
```

### アイコンが表示されない...

Bootstrap 5はBootstrap Iconsを使用：

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.10.0/font/bootstrap-icons.css">

<i class="bi bi-alarm"></i>
```

## コンポーネントに関する質問

### カードを中央配置したい！

```html
<div class="card">
    <div class="card-body">
        内容
    </div>
</div>
```

flexを使う：

```html
<div class="card">
    <div class="card-body d-flex justify-content-center align-items-center">
        中央配置の内容
    </div>
</div>
```

### フォーム検証が動かない...

Bootstrap 5は新しいフォーム検証を使用：

```html
<form class="row g-3 needs-validation" novalidate>
    <div class="col-md-4">
        <label for="validationCustom01" class="form-label">名前</label>
        <input type="text" class="form-control" id="validationCustom01" required>
        <div class="valid-feedback">正しい！</div>
        <div class="invalid-feedback">名前を入力してください</div>
    </div>
    <div class="col-12">
        <button class="btn btn-primary" type="submit">送信</button>
    </div>
</form>

<script>
    (function () {
        'use strict'
        var forms = document.querySelectorAll('.needs-validation')
        Array.prototype.slice.call(forms).forEach(function (form) {
            form.addEventListener('submit', function (event) {
                if (!form.checkValidity()) {
                    event.preventDefault()
                    event.stopPropagation()
                }
                form.classList.add('was-validated')
            }, false)
        })
    })()
</script>
```

### モーダルの背景をクリックしても閉じない！

属性を設定：

```html
<div class="modal" data-bs-backdrop="static">
```

`static` - 背景をクリックしても閉じない
`true` - 背景をクリックして閉じる
`false` - 背景をクリックしても閉じない

## パフォーマンスに関する質問

### ファイルサイズを小さくしたい！

1. 必要なコンポーネントだけ読み込む（Sassを使用）
2. 本番用バージョンを使用
3. Gzip圧縮を有効化

### 必要なスタイルだけ読み込みたい！

Sassで必要なコンポーネントだけ読み込む：

```scss
// 必要な関数と変数を読み込み
@import "bootstrap/scss/functions";
@import "bootstrap/scss/variables";

// 必要なコンポーネントだけ読み込み
@import "bootstrap/scss/root";
@import "bootstrap/scss/reboot";
@import "bootstrap/scss/containers";
@import "bootstrap/scss/grid";
@import "bootstrap/scss/buttons";
@import "bootstrap/scss/forms";
```

## 互換性に関する質問

### IEブラウザに対応していない？

Bootstrap 5はInternet Explorerをサポートしていません。IE11に対応させる場合はBootstrap 4を使用してください。

### モバイルでのクリック遅延が！

Bootstrap 5はtouch-action: manipulationを使用して300msのクリック遅延を消除しています。
