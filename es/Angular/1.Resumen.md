# Angular Resumen

Framework moderno de desarrollo de aplicaciones web.

## ¿Qué es Angular?

Angular es un framework de aplicaciones web de código abierto desarrollado y mantenido por Google. Es una de las soluciones preferidas para el desarrollo moderno de aplicaciones web.

### Características principales

- **TypeScript nativo**: Completamente basado en TypeScript
- **Arquitectura basada en componentes**: Desarrollo modular para fácil mantenimiento
- **Inyección de dependencias**: Sistema poderoso para mejor testabilidad
- **Sistema de directivas**: Directivas para extender capacidades HTML
- **Enrutamiento**: Solución completa de enrutamiento del lado del cliente
- **Formularios**: Formularios reactivos y basados en plantillas
- **Cliente HTTP**: Servicio completo de cliente HTTP
- **Internacionalización**: Soporte i18n incorporado
- **Animaciones**: API poderosa de animaciones
- **Testing**: Soporte completo de pruebas unitarias y E2E

### Arquitectura técnica

Angular usa una arquitectura modular con conceptos centrales:

- **Módulo**: Unidad básica para organizar código
- **Componente**: Combinación de vista y lógica de negocio
- **Servicio**: Lógica de negocio encapsulada y reutilizable
- **Directiva**: Extiende el comportamiento de elementos HTML
- **Pipe**: Utilidad de transformación de datos

## Casos de uso

- **Aplicaciones empresariales**: Sistemas de negocios complejos
- **Aplicaciones de página única (SPA)**: Apps que requieren experiencia de usuario fluida
- **Progressive Web Apps (PWA)**: Apps con capacidades offline
- **Apps móviles híbridas**: Combinar con Ionic para apps móviles
- **Dashboards**: Visualización de datos y paneles de administración
