# Manual del usuario

## Instalar y ejecutar aplicaciones Android

Waydroid puede realizar varias operaciones que se encuentran usando el comando `waydroid app -h`:

```bash
usage: waydroid app [-h] {install,remove,launch,list} ...

optional arguments:
  -h, --help            mostrar este mensaje de ayuda y salir

subaction:
  {install,remove,launch,list}
    install             enviar un solo paquete al contenedor e instalarlo
    remove              eliminar un solo paquete de aplicación del contenedor
    launch              iniciar una sola aplicación
    list                listar aplicaciones instaladas
```

### Instalar aplicaciones Android

Puede instalar aplicaciones Android con:

```bash
waydroid app install xyz.apk
```

Luego puede lanzar la aplicación a través del menú de aplicaciones de su escritorio.

Si conoce el nombre del paquete Android de la aplicación (`com.xxx.yyy`), también puede usar la línea de comandos con:

```bash
waydroid app launch com.foo.bar
```

### Notas

Los archivos apk que a veces encontrará en Internet tienden a tener solo soporte para arm, por lo tanto, no funcionarán en x86_64.

Es posible que desee instalar [F-Droid](https://f-droid.org) para obtener aplicaciones gráficamente. O use Play Store si instaló una imagen GAPPS.

## Comandos comunes

### Opciones de línea de comandos de Waydroid

Waydroid proporciona ricas opciones de línea de comandos para controlar contenedores y sesiones:

- `waydroid show-full-ui`: Mostrar la interfaz de usuario completa de Android
- `waydroid session start`: Iniciar sesión de Waydroid
- `waydroid session stop`: Detener sesión de Waydroid
- `waydroid container start`: Iniciar contenedor de Waydroid
- `waydroid container stop`: Detener contenedor de Waydroid
- `waydroid logcat`: Ver registros de Android
- `waydroid shell`: Entrar en la shell de Android
- `waydroid prop set <property> <value>`: Establecer propiedades del sistema
- `waydroid bugreport`: Generar registros de informe de errores

### Configuración de propiedades del sistema

Se pueden configurar varias propiedades del sistema mediante el comando `waydroid prop`:

- `persist.waydroid.multi_windows`: Habilitar modo de múltiples ventanas
- Otras propiedades consulte la documentación oficial

## Características avanzadas

### Uso de imágenes personalizadas

Para usar imágenes personalizadas, siga estos pasos:

1. Descargue imágenes de [SourceForge](https://sourceforge.net/projects/waydroid/files/images/) o use `system.img` y `vendor.img` compilados personalizados
2. Copie las imágenes al directorio `/etc/waydroid-extra/images/`
3. Reinicialice Waydroid: `sudo waydroid init -f`

### Depuración ADB

Waydroid admite depuración ADB. De forma predeterminada, el modo seguro de ADB está habilitado y no se conectará automáticamente al iniciar la sesión. Use el siguiente comando para conectarse:

```bash
waydroid adb connect
```

Luego proporcione autorización en el lado de Android.

### Reenvío de notificaciones

Waydroid admite el reenvío de notificaciones de Android al entorno de escritorio, requiriendo una actualización de la imagen del sistema Android a una versión compatible.
