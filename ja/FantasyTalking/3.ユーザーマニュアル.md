# FantasyTalking ユーザーマニュアル

## 基本的な機能

### 音声合成

```javascript
const audio = await client.speak('こんにちは、FantasyTalkingへようこそ');
```

### イベント

```javascript
client.on('speech-start', () => console.log('再生開始'));
client.on('speech-end', () => console.log('再生終了'));
```

### 対話

```javascript
const session = client.createSession({ language: 'ja-JP' });
const response = await session.chat('今日の天気は？');
```

### ストリーミング

```javascript
const stream = await client.speakStream('長いテキスト');
stream.on('data', chunk => player.play(chunk));
```
