# Apache JMeter

Apache JMeterはApache Software Foundation旗下的オープンソース負荷テストツールで、100%純粋なJavaで作成されたアプリケーションです。負荷テスト機能とパフォーマンス測定のために設計されています。

## 概要

JMeterは元々はWebアプリケーションのテストのために設計されていましたが、その後他のテスト機能にも拡張されました。静的リソースと動的リソースのパフォーマンステストに使用でき、サーバー、ネットワーク、またはオブジェクトに対する高負荷をシミュレートして、その強度を確認したり、さまざまな負荷タイプでの全体的なパフォーマンスを分析したりできます。

## 主な機能

- **マルチプロトコルサポート**：HTTP、HTTPS、SOAP、REST、FTP、JDBC、LDAP、JMS、SMTP、POP3、IMAP、TCP、Javaオブジェクトなど
- **完全なテストIDE**：テスト計画の録音、ビルド、デバッグをサポートするGUIを提供
- **コマンドラインモード**：ヘッドレスモードをサポートし、任何Java対応OSで負荷テストを実行可能
- **動的レポート**：動的HTMLレポート生成機能を内置
- **データ抽出**：HTML、JSON、XML形式からデータを抽出をサポート
- **純粋なJava**：完全なポータブル性
- **マルチスレッドフレームワーク**：并发サンプリングと分散テストをサポート
- **高い拡張性**：プラグインとスクリプトの拡張をサポート

## クイックスタート

### 基本的な使用方法

1. JMeterを起動：
   ```bash
   jmeter
   ```

2. テスト計画を作成し、スレッドグループとHTTPリクエストを追加

3. テストを実行して結果を表示

### コマンドラインモード

```bash
jmeter -n -t test_plan.jmx -l results.jtl -e -o report
```
