# Vite Installationsanleitung

Diese Anleitung führt Sie durch die Installation und Einrichtung von Vite für Ihr Frontend-Projekt.

## Voraussetzungen

Stellen Sie sicher, dass Sie folgende Software installiert haben:
- Node.js (Version 14.18+ oder 16+)
- npm, yarn oder pnpm

## Installation

### Projekt erstellen

Verwenden Sie den folgenden Befehl, um ein neues Vite-Projekt zu erstellen:

```bash
# npm
npm create vite@latest my-project -- --template vue

# yarn
yarn create vite my-project -- --template vue

# pnpm
pnpm create vite my-project -- --template vue
```

Verfügbare Vorlagen:
- `vanilla` - Vanilla JavaScript
- `vanilla-ts` - Vanilla TypeScript
- `vue` - Vue
- `vue-ts` - Vue + TypeScript
- `react` - React
- `react-ts` - React + TypeScript
- `preact` - Preact
- `preact-ts` - Preact + TypeScript
- `svelte` - Svelte
- `svelte-ts` - Svelte + TypeScript

### Bestehendes Projekt

Wenn Sie Vite zu einem bestehenden Projekt hinzufügen möchten:

```bash
npm install -D vite
```

## Konfiguration

### Grundkonfiguration

Erstellen Sie eine `vite.config.js` Datei:

```javascript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [vue()],
  server: {
    port: 3000
  }
})
```

### TypeScript-Konfiguration

Vite unterstützt TypeScript out of the box. Fügen Sie eine `tsconfig.json` hinzu:

```json
{
  "compilerOptions": {
    "target": "ESNext",
    "useDefineForClassFields": true,
    "module": "ESNext",
    "lib": ["ESNext", "DOM"],
    "moduleResolution": "Node",
    "strict": true,
    "resolveJsonModule": true,
    "isolatedModules": true,
    "esModuleInterop": true,
    "noEmit": true,
    "noUnusedLocals": true,
    "noUnusedParameters": true,
    "noImplicitReturns": true,
    "skipLibCheck": true
  },
  "include": ["src"]
}
```

### Vue

```bash
npm install @vitejs/plugin-vue
```

```javascript
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [vue()]
})
```

### React

```bash
npm install @vitejs/plugin-react
```

```javascript
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [react()]
})
```

## Proxy-Konfiguration

Konfigurieren Sie einen Proxy für API-Anfragen:

```javascript
export default defineConfig({
  server: {
    proxy: {
      '/api': {
        target: 'http://localhost:8000',
        changeOrigin: true,
        rewrite: (path) => path.replace(/^\/api/, '')
      }
    }
  }
})
```

## Umgebungsvariablen

Erstellen Sie `.env` Dateien:

```
VITE_API_URL=http://localhost:8000
```

Zugriff im Code:

```javascript
const apiUrl = import.meta.env.VITE_API_URL
```
