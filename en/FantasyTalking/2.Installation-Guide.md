# FantasyTalking Installation Guide

## Environment Requirements

- Node.js 14.0.0+

## Installation

```bash
npm install fantasy-talking
```

## Quick Start

```javascript
const ft = new FantasyTalking({
  apiKey: 'your-api-key'
})

ft.speak('Hello world')
```

## Configuration

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| apiKey | string | - | API key |
| voice | string | default | Voice type |
| speed | number | 1.0 | Speech speed |
| pitch | number | 1.0 | Pitch |
| volume | number | 1.0 | Volume |
