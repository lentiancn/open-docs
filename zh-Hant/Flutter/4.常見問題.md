# Flutter 常見問題

常見問題解答。

## 安裝問題

### Q: Flutter 下載失敗？

**解決方案：**
使用鏡像：
```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
```

### Q: flutter doctor 報錯？

**解決方案：**
1. 檢查 PATH 配置
2. 確認 Git 已安裝

### Q: Android SDK 找不到？

**解決方案：**
```bash
flutter config --android-sdk /path/to/android-sdk
```

## 開發問題

### Q: 熱重載不生效？

**解決方案：**
1. 保存文件後等待熱重載
2. 檢查控制台錯誤

### Q: 構建 APK 失敗？

**解決方案：**
1. 運行 `flutter clean`
2. 更新 Gradle

### Q: Widget 不顯示？

**解決方案：**
1. 檢查 build() 方法返回值
2. 確認添加到了 MaterialApp

## 性能問題

### Q: 應用卡頓？

**解決方案：**
1. 使用 const 構造函數
2. 避免不必要的 setState
3. 使用 ListView.builder

### Q: APK 體積太大？

**解決方案：**
1. 使用混淆
2. 移除調試符號
