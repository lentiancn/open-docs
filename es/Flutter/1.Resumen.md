# Flutter - Resumen

Flutter es un kit de herramientas UI de código abierto por Google.

## ¿Qué es Flutter?

Flutter es un kit de herramientas UI para crear aplicaciones multiplataforma desde un solo código base para iOS, Android, Web y escritorio. Usa Dart.

## Características

- **Multiplataforma**: Una base de código para todo
- **Hot Reload**: Vista previa instantánea
- **Widgets ricos**: Material Design + Cupertino
- **Alto rendimiento**: Motor gráficos Skia
- **Código abierto**: Gratis

## Uso

- Apps móviles
- Apps web
- Apps escritorio
