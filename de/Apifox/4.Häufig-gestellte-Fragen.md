# Häufig gestellte Fragen

Dieses Dokument fasst häufige technische Fragen und Antworten während der Nutzung von Apifox zusammen.

## Grundlegende Fragen

### Was ist Apifox?

Apifox ist eine API-Kollaborationsplattform, die Funktionen für API-Design, Dokumentationsverwaltung, Debugging, Mocking und automatisierte Tests integriert. Es hilft Entwicklungsteams, API-Entwicklung, -Tests und -Dokumentation effizienter zu gestalten.

### Ist Apifox kostenlos?

Apifox bietet eine kostenlose und eine kostenpflichtige Version. Die kostenlose Version enthält bereits alle Kernfunktionen und reicht für die meisten Entwicklungsanforderungen aus. Die kostenpflichtige Version bietet zusätzliche erweiterte Funktionen wie größere Teams und erweiterte Automatisierung.

### Was ist der Unterschied zwischen Apifox und Postman?

Sowohl Apifox als auch Postman sind API-Tools, aber Apifox ist besser für chinesische Nutzer geeignet: bessere chinesische Unterstützung, intelligentere Mock-Funktionen und ein verbessertes Teamzusammenarbeitserlebnis. Die Grundfunktionen von Apifox sind vollständig kostenlos, während viele Funktionen von Postman kostenpflichtig sind.

## Installationsprobleme

### Was tun bei langsamer Download-Geschwindigkeit?

Apifox bietet mehrere Download-Server. Sie können versuchen, https://apifox.com/ zu besuchen und den nächstgelegenen Server für den Download zu wählen.

### Was tun, wenn die Anwendung nicht startet?

1. Überprüfen Sie, ob die Systemversion die Anforderungen erfüllt
2. Versuchen Sie, die Anwendung als Administrator auszuführen
3. Überprüfen Sie, ob erforderliche Laufzeitbibliotheken installiert sind
4. Schauen Sie in den Fehlerprotokollen nach weiteren Informationen

### Anmeldung fehlgeschlagen – was tun?

1. Überprüfen Sie, ob die Netzwerkverbindung funktioniert
2. Bestätigen Sie, ob Kontoname und Passwort korrekt sind
3. Versuchen Sie die Anmeldung über ein Drittanbieterkonto
4. Kontaktieren Sie den offiziellen Support für Hilfe

## Nutzungsprobleme

### Schnittstellendaten werden nicht synchronisiert?

1. Bestätigen Sie, dass Sie angemeldet sind
2. Überprüfen Sie die Netzwerkverbindung
3. Klicken Sie auf die Schaltfläche "Synchronisieren" für manuelle Synchronisierung
4. Überprüfen Sie die Kontoberechtigungen

### Mock-Daten werden nicht生效?

1. Bestätigen Sie, dass die Mock-Funktion aktiviert ist
2. Überprüfen Sie, ob die Mock-Adresse korrekt ist
3. Prüfen Sie, ob Syntaxfehler in der Schnittstelle vorliegen
4. Versuchen Sie, die Seite zu aktualisieren

### Anfrage kann nicht gesendet werden?

1. Überprüfen Sie die Netzwerkverbindung
2. Bestätigen Sie, dass die Anfrageadresse korrekt ist
3. Überprüfen Sie das Anfrageparameterformat
4. Prüfen Sie, ob die Anfrage von einem Proxy blockiert wird

### Antwort wird falsch angezeigt (乱码)?

1. Versuchen Sie, die Antwortkodierung zu ändern
2. Überprüfen Sie die Kodierungseinstellung in den Antwortheadern
3. Verwenden Sie die "Formatieren"-Funktion für erneute Analyse

## Funktionsfragen

### Wie importiere ich Postman-Daten?

1. Klicken Sie auf "Importieren"
2. Wählen Sie "Postman"
3. Wählen Sie die zu importierende Datei
4. Bestätigen Sie den Import

### Wie exportiere ich als Swagger?

1. Gehen Sie zu den Projekteinstellungen
2. Wählen Sie "Exportieren"
3. Wählen Sie "Swagger/OpenAPI"
4. Wählen Sie die Version
5. Klicken Sie auf Exportieren

### Wie verwende ich Umgebungsvariablen?

1. Definieren Sie Umgebungsvariablen in den Projekteinstellungen
2. Verwenden Sie {{Variablenname}} in Anfragen zur Referenzierung
3. Wechseln Sie die Umgebung über das Umgebungsauswahlfeld oben

### Wie führe ich Stapeloperationen für Schnittstellen durch?

1. Halten Sie Strg/Cmd gedrückt und wählen Sie mehrere Schnittstellen aus
2. Klicken Sie mit der rechten Maustaste und wählen Sie Stapeloperation
3. Wählen Sie den Operationstyp (Löschen, Exportieren, Verschieben usw.)

## Testfragen

### Wie führe ich automatisierte Tests aus?

1. Öffnen Sie die Test-Suite
2. Klicken Sie auf die Schaltfläche "Ausführen"
3. Sehen Sie sich die Testergebnisse an

### Was tun, wenn eine Assertion fehlschlägt?

1. Schauen Sie sich den Fehlergrund an
2. Überprüfen Sie erwartete und tatsächliche Werte
3. Ändern Sie die Assertionsbedingungen
4. Führen Sie die Tests erneut aus

### Wie exportiere ich Testberichte?

1. Nach Abschluss der Testausführung
2. Klicken Sie auf "Bericht exportieren"
3. Wählen Sie das Format (HTML, PDF, JSON)

## Fragen zur Teamzusammenarbeit

### Wie lade ich Teammitglieder ein?

1. Gehen Sie zu den Projekteinstellungen
2. Klicken Sie auf "Mitgliederverwaltung"
3. Klicken Sie auf "Mitglied einladen"
4. Geben Sie die E-Mail-Adresse oder den Benutzernamen des Mitglieds ein

### Wie werden Mitgliederberechtigungen festgelegt?

Beim Einladen von Mitgliedern können verschiedene Rollen festgelegt werden:

- Administrator: Volle Kontrolle
- Bearbeiter: Kann bearbeiten und ausführen
- Betrachter: Nur-Lese-Berechtigung

### Wie löse ich Konflikte?

Wenn mehrere Personen gleichzeitig dieselbe Schnittstelle bearbeiten:

1. Das System zeigt einen Konflikt an
2. Wählen Sie, ob Sie die lokale oder die Remote-Version behalten möchten
3. Oder führen Sie die Änderungen manuell zusammen

## Datenfragen

### Wie werden Daten gesichert?

1. Exportieren Sie regelmäßig das Projekt im Apifox-Format
2. Oder verwenden Sie die Cloud-Synchronisierungsfunktion

### Wie stelle ich Daten wieder her?

1. Falls Sie eine Sicherungsdatei haben, klicken Sie auf "Importieren"
2. Wählen Sie die Sicherungsdatei zur Wiederherstellung

### Gelöschte Daten wiederherstellen?

Gelöschte Daten werden 30 Tage lang im Papierkorb aufbewahrt und können aus dem Papierkorb wiederhergestellt werden.

## Leistungsprobleme

### Was tun bei Leistungsproblemen/Verzögerungen?

1. Schließen Sie nicht benötigte Projekte
2. Löschen Sie den Browser-Cache
3. Aktualisieren Sie die Hardwarekonfiguration
4. Versuchen Sie die neueste Version

### Langsamer Import großer Datenmengen?

1. Importieren Sie in Stapeln
2. Deaktivieren Sie die automatische Synchronisierung
3. Haben Sie Geduld, bis der Import abgeschlossen ist
