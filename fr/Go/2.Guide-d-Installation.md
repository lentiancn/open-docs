# Guide d'Installation de Go

> Guide complet pour installer et configurer Go

## Installation

### Windows
```powershell
choco install golang
```

### macOS
```bash
brew install go
```

### Linux
```bash
sudo apt-get install golang-go
```

## Configuration
```bash
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin
go env -w GOPROXY=https://goproxy.cn,direct
```

## Commandes de base
```bash
go mod init <module>
go mod tidy
go build
go run main.go
go test ./...
```
