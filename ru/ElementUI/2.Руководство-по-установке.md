# Руководство по установке Element UI

> Настройка

---

## Требования

| Среда | Минимум |
|--------|---------|
| Node.js | 8.x |
| npm | 5.x |
| Vue | 2.6.0 |

---

## Установка

```bash
npm install element-ui --save
```

---

## Импорт

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```
