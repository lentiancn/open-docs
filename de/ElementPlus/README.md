# Element Plus

[Element Plus](https://element-plus.org/) ist eine Vue 3-basierte UI-Komponentenbibliothek.

## Funktionen

- Vue 3 mit Composition API
- 70+ Komponenten
- TypeScript-Unterstützung
- Import bei Bedarf
- Internationalisierung

## Schnellstart

```bash
npm install element-plus
```
