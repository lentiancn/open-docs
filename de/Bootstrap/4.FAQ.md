# Bootstrap FAQ

Häufig gestellte Fragen.

## Installation

### Q: CDN lädt nicht?

**Lösungen**:
1. Netzwerkverbindung prüfen
2. URL bestätigen
3. Alternativen CDN verwenden

### Q: npm fehlgeschlagen?

**Lösungen**:
1. npm-Cache leeren
2. Node.js-Version prüfen

### Q: Noch jQuery nötig?

Nein. Bootstrap 5 hat jQuery entfernt.

## Nutzung

### Q: Styles funktionieren nicht?

**Lösungen**:
1. CSS korrekt eingebunden
2. Browser-Cache leeren

### Q: Responsive funktioniert nicht?

viewport Meta-Tag hinzufügen:
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

## Kompatibilität

### Q: IE nicht unterstützt?

Bootstrap 5 unterstützt IE nicht mehr.

## Leistung

### Q: Dateien zu groß?

**Lösungen**:
1. Sass verwenden
2. .min-Versionen verwenden
