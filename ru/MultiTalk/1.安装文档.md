# Руководство по установке MultiTalk

MultiTalk — это инструмент для генерации аудио с несколькими дикторами, создающий естественно звучащий многоголосый аудиоконтент.

## Информация о версии

| Версия | Статус | Описание |
|---------|--------|-------------|
| MultiTalk v1.0 | ✅ Текущая | Первоначальный выпуск |

## Системные требования

### Аппаратные требования

| Требование | Минимум | Рекомендуется |
|-------------|---------|-------------|
| GPU | NVIDIA 4GB VRAM | NVIDIA 8GB+ VRAM |
| RAM | 8GB | 16GB+ |
| Место на диске | 5GB | 10GB+ SSD |

### Программные требования

| Программа | Версия |
|----------|---------|
| Python | 3.8+ |
| CUDA | 11.0+ |

## Способы установки

### Способ 1: Из исходного кода

#### 1. Клонирование репозитория

```bash
git clone https://github.com/MultiTalk/MultiTalk.git
cd MultiTalk
```

#### 2. Создание виртуальной среды

```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
# или
venv\Scripts\activate  # Windows
```

#### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

#### 4. Загрузка моделей

```bash
bash scripts/download_models.sh
```

### Способ 2: Docker

```bash
docker pull multitalk/multitalk:latest
docker run --gpus all -v /path/to/data:/data multitalk/multitalk:latest
```

## Настройка

### Переменные окружения

```bash
export CUDA_VISIBLE_DEVICES=0
export PYTHONPATH=/path/to/MultiTalk:$PYTHONPATH
```

### Пути к моделям

Отредактируйте `config.yaml`:

```yaml
model:
  checkpoint_path: "checkpoints/multitalk.pth"
```

## Быстрый старт

### Генерация аудио с несколькими дикторами

```bash
python inference.py \
  --text "Привет, это демонстрация с несколькими дикторами." \
  --speakers speaker1,speaker2 \
  --output output.wav
```

### Параметры

```bash
--text         # Входной текст
--speakers     # ID дикторов (через запятую)
--output       # Путь к выходному файлу
--speed        # Скорость речи (по умолчанию: 1.0)
```

## Частые проблемы

### GPU недоступен?

```bash
nvidia-smi
python -c "import torch; print(torch.cuda.is_available())"
```

### Ошибка загрузки модели?

Проверьте подключение к интернету или загрузите модели вручную.
