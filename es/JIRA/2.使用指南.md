# Guía de Usuario de JIRA

Esta guía describe cómo usar JIRA para la gestión de proyectos y seguimiento de problemas.

## Inicio Rápido

### Crear un Proyecto

1. Haga clic en "Crear Proyecto"
2. Seleccione una plantilla de proyecto
3. Ingrese el nombre y la clave del proyecto
4. Haga clic en "Crear"

## Tipos de Proyecto

### Proyectos Ágiles

```markdown
- Proyectos Scrum
- Proyectos Kanban
- Ágil a Escala
```

### Proyectos Clásicos

```markdown
- Proyectos de Negocio
- Gestión de Proyectos
- Seguimiento de Tareas
```

## Tipos de Problemas

### Crear un Problema

```markdown
1. Haga clic en "Crear Problema"
2. Seleccione el proyecto
3. Seleccione el tipo de problema
4. Complete el título y la descripción
5. Configure la prioridad y los componentes
6. Asigne a un usuario
```

### Tipos de Problemas

- **Bug** - Defecto de software
- **Story** - Historia de usuario
- **Task** - Tarea
- **Epic** - Épica
- **Sub-task** - Subtarea

## Flujos de Trabajo

### Flujo de Trabajo Predeterminado

```
Crear → Asignar → Resolver → Cerrar
```

### Flujo de Trabajo Ágil

```
Pendiente → En Progreso → Completado → Revisado
```

3. Configure las opciones del campo

## Automatización

### Crear una Regla

1. Vaya a Configuración del Proyecto → Automatización
2. Cree una regla
3. Configure los disparadores
4. Configure las acciones

### Ejemplo de Automatización

```
Disparador: Creación de problema
Condición: Tipo = Bug
Acción: Asignar a = Responsable
```
