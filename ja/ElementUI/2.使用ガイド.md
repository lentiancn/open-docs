# Element UI 使用ガイド

このガイドでは、Vue 2アプリケーションでElement UIコンポーネントを使用する方法を説明します。

## 基本的な使用法

### Hello World

```vue
<template>
  <div>
    <el-button>デフォルトボタン</el-button>
    <el-button type="primary">プライマリボタン</el-button>
  </div>
</template>

<script>
export default {
  name: 'App'
}
</script>
```

## コンポーネントカテゴリ

Element UIはこれらのコンポーネントカテゴリを提供します：

| カテゴリ | コンポーネント |
|----------|----------------|
| ベース | Layout, Button, Icon, Link |
| フォーム | Input, Select, Radio, Checkbox, Switch, Slider, TimePicker, DatePicker |
| データ | Table, Tree, Pagination, Tag |
| 通知 | Alert, Message, Notification, Progress |
| ナビゲーション | Menu, Tabs, Breadcrumb, Steps, Dropdown |
| その他 | Dialog, Drawer, Card, Collapse, Timeline |

### 入力 Input

```vue
<template>
  <div>
    <!-- ベース -->
    <el-input v-model="input" placeholder="テキストを入力"></el-input>
    
    <!-- アイコン付き -->
    <el-input
      v-model="input1"
      placeholder="検索"
      prefix-icon="el-icon-search">
    </el-input>
    
    <!-- パスワード -->
    <el-input
      v-model="input2"
      placeholder="パスワード"
      show-password>
    </el-input>
    
    <!-- テキストエリア -->
    <el-input
      v-model="textarea"
      type="textarea"
      :rows="3"
      placeholder="説明を入力">
    </el-input>
    
    <!-- 無効 -->
    <el-input disabled placeholder="無効"></el-input>
  </div>
</template>

<script>
export default {
  data() {
    return {
      input: '',
      input1: '',
      input2: '',
      textarea: ''
    }
  }
}
</script>
```

### フォーム Form

```vue
<template>
  <el-form
    :model="ruleForm"
    :rules="rules"
    ref="ruleForm"
    label-width="120px"
  >
    <el-form-item label="名前" prop="name">
      <el-input v-model="ruleForm.name"></el-input>
    </el-form-item>
    
    <el-form-item label="メール" prop="email">
      <el-input v-model="ruleForm.email"></el-input>
    </el-form-item>
    
    <el-form-item label="性別" prop="gender">
      <el-radio-group v-model="ruleForm.gender">
        <el-radio label="male">男性</el-radio>
        <el-radio label="female">女性</el-radio>
      </el-radio-group>
    </el-form-item>
    
    <el-form-item label="趣味" prop="hobby">
      <el-checkbox-group v-model="ruleForm.hobby">
        <el-checkbox label="読書" name="hobby"></el-checkbox>
        <el-checkbox label="スポーツ" name="hobby"></el-checkbox>
        <el-checkbox label="音楽" name="hobby"></el-checkbox>
      </el-checkbox-group>
    </el-form-item>
    
    <el-form-item label="国" prop="country">
      <el-select v-model="ruleForm.country" placeholder="国を選択">
        <el-option label="日本" value="japan"></el-option>
        <el-option label="アメリカ" value="usa"></el-option>
        <el-option label="イギリス" value="uk"></el-option>
      </el-select>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm('ruleForm')">送信</el-button>
      <el-button @click="resetForm('ruleForm')">リセット</el-button>
    </el-form-item>
  </el-form>
</template>

<script>
export default {
  data() {
    return {
      ruleForm: {
        name: '',
        email: '',
        gender: '',
        hobby: [],
        country: ''
      },
      rules: {
        name: [
          { required: true, message: '名前を入力してください', trigger: 'blur' }
        ],
        email: [
          { required: true, message: 'メールアドレスを入力してください', trigger: 'blur' },
          { type: 'email', message: 'メールアドレスの形式が正しくありません', trigger: 'blur' }
        ],
        gender: [
          { required: true, message: '性別を選択してください', trigger: 'change' }
        ],
        country: [
          { required: true, message: '国を選択してください', trigger: 'change' }
        ]
      }
    };
  },
  methods: {
    submitForm(formName) {
      this.$refs[formName].validate((valid) => {
        if (valid) {
          alert('送信しました！');
        } else {
          console.log('エラー送信！！');
          return false;
        }
      });
    },
    resetForm(formName) {
      this.$refs[formName].resetFields();
    }
  }
}
</script>
```

## イベント

```vue
<template>
  <el-button @click="handleClick">クリックして</el-button>
  
  <el-input
    @focus="handleFocus"
    @blur="handleBlur"
    @change="handleChange"
    @input="handleInput">
  </el-input>
</template>

<script>
export default {
  methods: {
    handleClick() {
      console.log('クリックしました！')
    },
    handleFocus() {
      console.log('フォーカスしました！')
    },
    handleBlur() {
      console.log('ブラーしました！')
    },
    handleChange(value) {
      console.log('変更：', value)
    },
    handleInput(value) {
      console.log('入力：', value)
    }
  }
}
</script>
```

## ベストプラクティス

### 1. コンポーネントをローカルで使用

```javascript
// 良い例 - ローカル使用
export default {
  components: {
    [Button.name]: Button,
    [Input.name]: Input
  }
}
```

### 2. フォーム検証

```javascript
const validatePass = (rule, value, callback) => {
  if (value === '') {
    callback(new Error('パスワードを入力してください'));
  } else {
    if (this.ruleForm.checkPass !== '') {
      this.$refs.ruleForm.validateField('checkPass');
    }
    callback();
  }
};
```

### 3. テーブルパフォーマンス

```vue
<el-table
  :data="tableData"
  :default-sort = "{prop: 'date', order: 'descending'}"
  >
</el-table>
```
