# Flutter 安装指南

Flutter 是 Google 开发的开源 UI 工具包，用于构建美观、快速的跨平台应用。本指南涵盖在所有环境中的安装方法。

## 版本信息

| 版本 | 状态 | 说明 |
|------|------|------|
| Flutter 3.x | ✅ 当前 | 最新稳定版本 |
| Flutter 2.x | ⚠️ 维护 | 安全更新 |

> **重要提示**：本指南主要针对 **Flutter 3.x**。

## 系统要求

### Windows

| 要求 | 最低配置 |

| 操作系统 | Windows 10 或更高版本 |

| 要求 | 配置 |
|------|------|

### Linux

| 要求 | 配置 |
|------|------|
| 操作系统 | Ubuntu 18.04+ |
| 磁盘空间 | 2.8 GB |
| 内存 | 4 GB |

## 安装方法

### 方法一：Windows 安装

#### 1. 下载 Flutter SDK

1. 下载 Flutter SDK：https://docs.flutter.dev/get-started/install/windows
2. 解压到指定目录（如 C:\src\flutter）

#### 2. 配置环境变量

```powershell
# 添加到 PATH
$env:PATH += ";C:\src\flutter\bin"
```

#### 3. 运行 Flutter doctor

```powershell
flutter doctor
```

### 方法二：macOS 安装

#### 1. 使用 Homebrew（推荐）

```bash
# 安装 Flutter
brew install flutter

# 或手动下载
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable
```

#### 2. 配置环境变量

```bash
# ~/.bash_profile 或 ~/.zshrc
export PATH="$PATH:`pwd`/flutter/bin"
```

#### 3. 运行 Flutter doctor

```bash
flutter doctor
```

### 方法三：Linux 安装

#### 1. 安装依赖

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa clang cmake ninja-build pkg-config libgtk-3-dev

# Fedora
sudo dnf install -y curl git unzip xz zip mesa-libGLU clang cmake ninja-build pkgconfig gtk3-devel
```

#### 2. 下载 Flutter SDK

```bash
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable
export PATH="$PATH:~/development/flutter/bin"
```

#### 3. 运行 Flutter doctor

```bash
flutter doctor
```

### 方法四：使用 VS Code

1. 下载并安装 VS Code
2. 打开 VS Code
3. 安装 Flutter 扩展：
   - 搜索 "Flutter" 并安装
   - 搜索 "Dart" 并安装

### 方法五：使用 Android Studio

1. 下载并安装 Android Studio
2. 安装 Flutter 插件：
   - 打开 Android Studio
   - Settings → Plugins
   - 搜索 "Flutter" 并安装

## Android 配置

### 安装 Android SDK

```bash
# 使用 Flutter 工具安装
flutter doctor --android-licenses
flutter config --android-sdk /path/to/android/sdk
```

### 配置环境变量

```bash
# macOS/Linux
export ANDROID_SDK_ROOT=/Users/user/Library/Android/sdk
export PATH=$PATH:$ANDROID_SDK_ROOT/cmdline-tools/latest/bin:$ANDROID_SDK_ROOT/platform-tools
```

### 创建模拟器

```bash
# 列出可用设备
flutter devices

# 创建模拟器
flutter emulator --create --name my-emulator

# 启动模拟器
flutter emulators --launch my-emulator
```

## 常见问题

### Flutter doctor 报错？

确保所有依赖已安装：

```bash
# 运行 flutter doctor
flutter doctor -v
```

### Android SDK 找不到？

设置 SDK 路径：

```bash
flutter config --android-sdk /path/to/android/sdk
```

### 构建失败？

清理并重新构建：

```bash
flutter clean
flutter pub get
flutter build apk
```
