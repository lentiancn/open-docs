# Go 자주 묻는 질문

> Go에 대한 일반적인 질문 답변

## 설치

### Q: 설치 후 명령어를 찾을 수 없음?
**A**: PATH에 Go가 추가되었는지 확인

### Q: 설치 확인 방법은?
```bash
go version
go env GOPATH
```

## 구문

### Q: Go에 예외 처리가 있나요?
**A**: 아니오, 에러는 반환값으로 처리:
```go
result, err := doSomething()
if err != nil {
    // 에러 처리
}
```

### Q: 제네릭을 지원하나요?
**A**: Go 1.18+에서 지원:
```go
func Print[T any](s []T) {
    for _, v := range s {
        fmt.Println(v)
    }
}
```

## 동시성

### Q: 고루틴과 스레드의 차이는?
- 고루틴은 Go 런타임에서 관리
- 스레드보다 가벼움

### Q: 고루틴 종료를 기다리는 방법은?
```go
var wg sync.WaitGroup
wg.Add(1)
go func() {
    defer wg.Done()
}()
wg.Wait()
```
