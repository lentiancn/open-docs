# Bootstrap FAQ

Questions fréquentes.

## Installation

### Q: CDN ne charge pas?

**Solutions**:
1. Vérifier connexion réseau
2. Confirmer URL
3. Utiliser CDN alternatif

### Q: Installation npm échouée?

**Solutions**:
1. Vider cache npm
2. Vérifier version Node.js

### Q: jQuery toujours nécessaire?

Non. Bootstrap 5 n'utilise plus jQuery.

## Utilisation

### Q: Styles ne fonctionnent pas?

**Solutions**:
1. CSS correctement importé
2. Vider cache navigateur

### Q: Responsive ne fonctionne pas?

Ajouter meta viewport:
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

## Compatibilité

### Q: IE pas supporté?

Bootstrap 5 ne supporte plus IE.

## Performance

### Q: Fichiers trop gros?

**Solutions**:
1. Utiliser Sass
2. Utiliser versions .min
