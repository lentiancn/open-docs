# Bootstrap

最流行的前端响应式 CSS 框架。

## 简介

Bootstrap 是由 Twitter 开发的最流行的前端 CSS 框架，用于构建响应式、移动优先的网站和 Web 应用。具有响应式网格系统、预制组件、CSS 工具类等特点。

### 主要特性

- **响应式网格系统** - 12 列响应式网格
- **预制组件** - 导航栏、卡片、模态框等
- **CSS 工具类** - 快速样式调整
- **JavaScript 组件** - 交互式组件

### 统计数据

- GitHub 星标：160,000+
- 网站使用量：数百万

## 快速开始

### CDN 引入

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### npm 安装

```bash
npm install bootstrap
```
