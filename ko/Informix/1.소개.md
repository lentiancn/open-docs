# IBM Informix 소개

## IBM Informix란?

IBM Informix는 원래 Informix Corporation에서 개발하고 2001년 IBM이 인수한 관계형 데이터베이스 관리 시스템(RDBMS) 제품군입니다. 2017년 IBM은 개발 및 지원을 HCLSoftware에 위임하면서 계속 IBM Informix로 판매하고 있습니다.

Informix는 내장된 데이터베이스 사용을 포함하여 데이터베이스 관리 작업이 거의 없거나 없는 환경에 최적화되어 있습니다. 매우 높은 트랜잭션률을 지원하며 제조 라인과 예약 시스템과 같은 중요 비즈니스 애플리케이션에 필요한 가동 시간 특성을 제공합니다.

## 주요 기능

- **멀티모델 데이터베이스**: 복잡한 데이터 구조를 모델링하기 위한 관계형 및 객체 지향 기능 지원
- **높은 성능**: 항상 가동 중인 트랜잭션 및 실시간 분석에 최적화
- **작은 공간**: Raspberry Pi와 같은 100MB程度的 소형 장치에서 실행 가능
- **엣지 컴퓨팅**: 내장 데이터베이스 기능으로 IoT 솔루션에 이상적
- **시계열 지원**: 시계열 데이터에 대한 기본 제공 지원
- **JSON/MongoDB 호환성**: JSON 컬렉션 및 MongoDB JSON 드라이버 통합
- **고가용성**: HDR(고가용성 데이터 복제), RSS 및 SDS 기능

## 제품 에디션

2025-2026년 기준 IBM Informix는 다음과 같은 에디션을 제공합니다:

| 에디션 | 대상 사용자 | 제한 | 주요 기능 |
|--------|-----------|------|----------|
| **Developer Edition** | 개별 개발자 | 1 CPU VP/Core, 1GB RAM, 8GB 저장용량 | 개발/테스트 무료 |
| **Innovator-C Edition** | 소기업 | 2 CPU 코어, 8GB RAM, 50GB 저장용량 | 개발/소규모 프로덕션 무료 |
| **Express Edition** | 타사 개발자 | 4 CPU 코어, 8GB RAM | 내장 가능한 데이터베이스 |
| **Workgroup Edition** | 중견 기업 | 최대 24 CPU 코어, 32GB RAM | 모든 복제 옵션 |
| **Enterprise Edition** | 대기업 | 무제한 코어/RAM | IWA를 제외한 모든 기능 |
| **Advanced Enterprise Edition** | Enterprise + BI | 무제한 | IWA를 포함한 전체 스위트 |

## 버전 기록

- **현재 버전**: IBM Informix 15.0.0.1 (2025년 3월)
- **v15.0 주요 향상**: 데이터 페이지당 행 수, 테이블 파티션당 데이터 페이지, 총 저장 용량 등 대부분의 내부 제한 제거
