# Claude Code 사용자 매뉴얼

기본 사용법 및 기술.

## 기본 사용법

### 시작

- VS Code: 단축키 Cmd/Ctrl + Shift + P
- CLI: 터미널에서 직접 `claude` 실행

### 대화

채팅 창에서 직접 필요 사항 설명:
```
정렬 함수 만들어줘
이 코드 설명해줘
이 버그 어떻게 수정해?
```

## 코드 생성

### 새 코드 생성

원하는 기능 설명:
```
사용자 로그인 API 생성
페이지네이션 쿼리 구현
단위 테스트 작성
```

### 기존 코드 수정

1. 코드 선택
2. 수정 필요 사항 설명
3. Claude가 수정된 코드 생성

## 디버깅 지원

### 오류 설명

오류 메시지를 직접 붙여넣기:
```
이 오류 무슨 뜻야?
```

### 수정 제안

Claude에 문제 분석 요청:
```
이 함수什么问题?
```

## 코드 리뷰

### 코드 리뷰 요청

```
이 코드 리뷰해줘
보안 문제 있어?
성능 최적화 제안은?
```

## 단축키

### VS Code

| 단축키 | 기능 |
|--------|------|
| Cmd/Ctrl + L | 채팅 열기 |
| Cmd/Ctrl + Enter | 입력 제출 |

## 구성 옵션

### claude.json

프로젝트 루트에 `claude.json` 생성:

```json
{
  "preamble": "당신은 Python 전문가입니다",
  "maxTokens": 4000
}
```
