# Введение в Apache APISIX

## Что такое Apache APISIX?

Apache APISIX — это облачный API-шлюз и AI-шлюз, разработанный API7.ai и пожертвованный Apache Software Foundation.

APISIX выступает как "менеджер трафика" между клиентами и backend-сервисами.

## Основные характеристики

### 1. Динамические возможности

- Динамическая маршрутизация
- Динамические upstream
- Динамические сертификаты
- Hot-reload плагины

### 2. Управление трафиком

- Балансировка нагрузки (round-robin, взвешенная, хеширование)
- Canary-релизы
- A/B тестирование
- Разделение трафика
- Circuit breaker

### 3. Безопасность

- Аутентификация по API-ключу
- JWT-аутентификация
- OAuth 2.0
- IP whitelist/blacklist
- Ограничение частоты запросов

### 4. Наблюдаемость

- Метрики Prometheus
- Распределенная трассировка (Zipkin, SkyWalking)
- Экспорт логов (Kafka, Elasticsearch)

### 5. Мультипротокол

- HTTP/HTTPS
- gRPC
- Dubbo
- MQTT
- WebSocket

## Архитектура

- Плоскость данных: Nginx + Lua
- Плоскость управления: etcd
- Управление: REST API + Dashboard
