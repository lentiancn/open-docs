# Yarn インストールガイド

## システム要件

- Node.js: 16.0+ (推奨 18+)
- OS: macOS、Linux、Windows 10+

---

## インストール方法

### corepackを使用（推奨）

```bash
corepack enable
corepack prepare yarn@stable --activate
```

### npmを使用

```bash
npm install -g yarn
```

---

## 確認

```bash
yarn --version
```
