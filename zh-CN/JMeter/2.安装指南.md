# JMeter 安装指南

## 环境要求

- JDK 8+
- 内存 2GB+

## 安装

```bash
# 下载
wget https://archive.apache.org/dist/jmeter/binaries/apache-jmeter-5.6.tgz

# 解压
tar -xzf apache-jmeter-5.6.tgz

# 进入目录
cd apache-jmeter-5.6
```

## 目录结构

- bin/：启动脚本
- lib/：依赖库
- docs/：文档
- extras/：扩展

## Windows

下载 ZIP 解压，运行 bin/jmeter.bat

## 运行

```bash
# GUI 模式
./bin/jmeter

# 命令行模式
./bin/jmeter -n -t test.jmx -l result.jtl
```
