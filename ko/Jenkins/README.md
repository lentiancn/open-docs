# Jenkins

Jenkins는 지속적인 통합(CI) 및 지속적인 배포(CD)를 구현하기 위한 가장 인기 있는 오픈소스 자동화 서버입니다.

## 개요

Jenkins는 소프트웨어 빌드, 테스트 및 배포 프로세스에서 다양한 작업을 자동화하기 위한 플랫폼을 제공합니다. Jenkins를 통해 개발 팀은 지속적인 배포를 달성하여 소프트웨어를 빠르고 신뢰할 수 있는 방식으로 프로덕션에 출시할 수 있습니다.

## 주요 기능

- **지속적인 통합**: 코드 변경 자동 감지 및 빌드 트리거
- **지속적인 배포**: 소프트웨어 출시 프로세스 자동화
- **파이프라인 애즈 코드**: Jenkinsfile을 사용하여 빌드 프로세스 정의
- **플러그인 생태계**: 1,800개 이상의 플러그인 사용 가능
- **분산 빌드 지원**: 다중 노드 분산 빌드 지원
- **크로스 플랫폼**: Windows, Linux, macOS 지원
