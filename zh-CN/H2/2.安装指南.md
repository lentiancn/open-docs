# H2 数据库安装指南

## 环境要求

- Java 11 或更高版本
- 任何支持 Java 的操作系统（Windows、Linux、macOS 等）

## 安装方式

### 方式一：下载 ZIP 包

1. 访问官方网站下载页面：https://www.h2database.com/html/download.html
2. 下载 "Platform-Independent Zip" 版本
3. 解压下载的 ZIP 文件到任意目录，例如：C:\h2 或 /opt/h2

### 方式二：Maven 依赖

如果你是使用 Maven 或 Gradle 的 Java 项目，可以直接添加依赖：

**Maven：**
```xml
<dependency>
    <groupId>com.h2database</groupId>
    <artifactId>h2</artifactId>
    <version>2.2.224</version>
</dependency>
```

**Gradle：**
```groovy
implementation 'com.h2database:h2:2.2.224'
```

### 方式三：Windows 安装程序

1. 下载 "Windows Installer" 版本
2. 运行安装程序，按照向导完成安装

## 配置环境变量（可选）

如果你下载了 ZIP 包并希望在命令行中直接使用 H2：

**Windows：**
1. 右键点击"此电脑" → "属性"
2. 点击"高级系统设置"
3. 点击"环境变量"
4. 在系统变量中找到 Path，点击"编辑"
5. 添加 H2 的 bin 目录路径

**Linux/macOS：**
```bash
export PATH=$PATH:/opt/h2/bin
```

## 验证安装

### 使用命令行

解压后，在命令行中运行：

```bash
java -jar h2.jar
```

这将启动 H2 控制台 web 界面。

### 使用 JDBC 连接

在 Java 代码中连接 H2 数据库：

```java
// 嵌入式模式
String url = "jdbc:h2:mem:testdb";
Connection conn = DriverManager.getConnection(url, "sa", "");

// 文件数据库
String url = "jdbc:h2:file:./data/testdb";
Connection conn = DriverManager.getConnection(url, "sa", "password");
```

### 使用 H2 控制台

1. 运行命令：`java -jar h2.jar`
2. 打开浏览器访问：http://localhost:8082
3. 默认用户名：sa，密码为空

## 连接 URL 格式

H2 支持多种连接 URL 格式：

| 模式 | URL 示例 | 说明 |
|------|---------|------|
| 内存模式 | jdbc:h2:mem:testdb | 数据存储在内存中 |
| 嵌入式文件 | jdbc:h2:file:./data/testdb | 数据存储在文件中 |
| TCP 服务器 | jdbc:h2:tcp://localhost:9092/testdb | 客户端-服务器模式 |
| TCP 加密 | jdbc:h2:ssl://localhost:9092/testdb | SSL 加密连接 |
| PostgreSQL 兼容 | jdbc:h2:postgres://localhost:5432/testdb | PostgreSQL 协议兼容模式 |

## 常见问题

### 端口被占用
H2 默认使用 9092 端口（TCP 模式）和 8082 端口（Web 控制台）。如果端口被占用，可以在连接 URL 中指定其他端口。

### Java 版本问题
确保安装了 Java 11 或更高版本。可以通过以下命令检查：

```bash
java -version
```

### 驱动程序未找到
确保 H2 的 JAR 文件在 classpath 中。如果是 Maven 项目，确保依赖已正确添加。
