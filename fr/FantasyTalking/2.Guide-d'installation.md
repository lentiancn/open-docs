# FantasyTalking Guide d'installation

## Configuration Requise

Avant d'installer FantasyTalking, assurez-vous que votre environnement de développement répond aux exigences suivantes. En termes de systèmes d'exploitation, le projet supporte Windows, macOS et Linux. L'utilisation d'un système 64 bits est recommandée pour des performances optimales. Node.js version 14.0 ou supérieure requis, Python version 3.7 ou supérieure. Si vous utilisez le backend Python, FFmpeg doit également être installé.

Pour le matériel, il est recommandé d'avoir au moins 4 Go de RAM et un processeur bicœur pour une expérience de synthèse vocale fluide.

## Méthodes d'Installation

### Utiliser npm (Node.js)

```bash
# Installer la bibliothèque core
npm install fantasy-talking

# Installer les composants optionnels
npm install fantasy-talking-ui
npm install fantasy-talking-node
```

### Utiliser pip (Python)

```bash
# Installer la bibliothèque core
pip install fantasy-talking

# Installer les composants optionnels
pip install fantasy-talking[ui]
pip install fantasy-talking[all]
```

## Démarrage Rapide

### Exemple JavaScript/TypeScript

```javascript
const FantasyTalking = require('fantasy-talking');

// Initialiser le client
const client = new FantasyTalking({
  apiKey: 'your-api-key',
  voice: {
    style: 'neutral',
    language: 'fr-FR',
    speed: 1.0,
    pitch: 1.0
  }
});

// Synthétiser la parole
async function synthesize() {
  const audio = await client.speak('Bonjour, ceci est un test de FantasyTalking');
  console.log('Synthèse vocale réussie');
  
  // Sauvegarder dans un fichier
  const fs = require('fs');
  fs.writeFileSync('output.wav', audio);
}

synthesize();
```

## Configuration

| Paramètre | Type | Défaut | Description |
|-----------|------|--------|-------------|
| apiKey | string | - | Clé API, requis |
| apiUrl | string | Défaut | Adresse du serveur API |
| timeout | number | 30000 | Timeout (ms) |
| voice.style | string | neutral | Style vocal |
| voice.language | string | fr-FR | Code langue |
| voice.speed | number | 1.0 | Vitesse, 0.5-2.0 |

## Déploiement Docker

```bash
docker pull fantasytalking/server:latest

docker run -d \
  --name fantasy-talking \
  -p 8080:8080 \
  -e API_KEY=your-api-key \
  fantasytalking/server:latest
```

## Vérification

```bash
fantasy-talking --text "Texte de test" --output test.wav
```
