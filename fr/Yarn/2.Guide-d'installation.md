# Guide d'installation de Yarn

## Installation de Yarn v4+ (Version moderne)

### Méthode recommandée : Utilisation de Corepack

Corepack est un outil intermédiaire qui vous permettra de configurer la version de votre gestionnaire de paquets par projet.

```bash
# 1. Installer Corepack
npm install -g corepack

# 2. Initialiser un nouveau projet
yarn init -2
```

### Mise à jour de Yarn

Chaque fois que vous souhaitez mettre à jour Yarn vers la dernière version, exécutez simplement :

```bash
yarn set version stable
yarn install
```

Yarn configurera alors votre projet pour utiliser le binaire stable le plus récent.

### Utilisation des versions Canary

Yarn fournit également fréquemment des versions candidates à la publication (Release Candidate). Utilisez la commande suivante si vous avez besoin d'une fonctionnalité qui n'a pas encore été publiée sur le canal stable :

```bash
yarn set version canary
```

Ces versions sont très stables, la seule différence avec le canal régulier étant une migration plus échelonnée entre les versions majeures à mesure que nous implémentons de nouvelles modifications cassantes.

### Installation de la dernière version depuis master

Vous souhaiterez peut-être tester une version de Yarn si récente qu'elle n'a pas encore été publiée dans une version candidate, ou même pas encore fusionnée. La commande suivante clonera, construira et installera Yarn dans votre projet directement depuis notre dépôt :

```bash
yarn set version from sources
```

Elle accepte un drapeau `--branch` que vous pouvez utiliser pour tester des PR spécifiques :

```bash
yarn set version from sources --branch 1211
```

> Avertissement : Contrairement aux canaux stable et canary, la commande `yarn set version from sources` ne peut pas tirer parti de Corepack et devra stocker le binaire Yarn dans le dossier `.yarn/releases` et y faire référence depuis le fichier `.yarnrc.yml` de votre projet.

## Installation de Yarn v1 (Classic)

### Via npm (Recommandé)

Il est recommandé d'installer Yarn via le [gestionnaire de paquets npm](http://npmjs.org/), qui est inclus avec [Node.js](https://nodejs.org/) lorsque vous l'installez sur votre système.

Une fois que vous avez installé npm, vous pouvez exécuter ce qui suit à la fois pour installer et mettre à jour Yarn :

```bash
npm install --global yarn
```

### Vérification de l'installation

Vérifiez que Yarn est installé en exécutant :

```bash
yarn --version
```

## Recommandations de sélection de version

- **Nouveaux projets** : Il est recommandé d'utiliser Yarn v4+ pour profiter des fonctionnalités modernes et de meilleures performances
- **Projets existants** : Si vous utilisez déjà Yarn v1, vous pouvez continuer à l'utiliser ou envisager de migrer vers la version moderne
- **Environnements d'entreprise** : Yarn v1 fournit un support à long terme plus stable, tandis que v4+ offre des fonctionnalités plus innovantes
