# Vue 3 설치 가이드

Vue 3를 설치하는 가이드입니다.

---

## CDN

```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
```

---

## NPM

```bash
npm install vue@3
```

---

## Vite (권장)

```bash
npm create vite@latest my-vue3-app -- --template vue
cd my-vue3-app
npm install
npm run dev
```
