# Angular

 сучасний вебアプリケーション開発フレームワーク。

## 概要

Googleが開発・メンテナンスするオープンソースのWebアプリケーションフレームワーク。

### 主な機能

- **TypeScriptベース**
- **コンポーネント指向**
- **依存性注入**
- **ルーティング**
- **フォーム**

### 統計

- GitHubスター：93,000+

## クイックスタート

### インストール

```bash
npm install -g @angular/cli
```

### プロジェクト作成

```bash
ng new my-app
ng serve
```

## コマンド

```bash
ng g c my-component
ng g s my-service
ng build
```
