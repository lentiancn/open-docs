## 一、基礎組件

### 1.1 按鈕組件

按鈕是最常用的組件之一，Element UI 提供了多種類型的按鈕。

#### 基礎用法

```html
<template>
  <el-button>默認按鈕</el-button>
  <el-button type="primary">主要按鈕</el-button>
  <el-button type="success">成功按鈕</el-button>
  <el-button type="info">信息按鈕</el-button>
  <el-button type="warning">警告按鈕</el-button>
  <el-button type="danger">危險按鈕</el-button>
</template>
```

#### 按鈕屬性

| 屬性 | 說明 | 類型 | 可選值 | 默認值 |
|------|------|------|--------|--------|
| type | 按鈕類型 | string | primary / success / warning / danger / info / text | — |
| plain | 是否樸素按鈕 | boolean | — | false |
| round | 是否圓角按鈕 | boolean | — | false |
| circle | 是否圓形按鈕 | boolean | — | false |
| loading | 是否加載中 | boolean | — | false |
| disabled | 是否禁用 | boolean | — | false |
| icon | 圖標類名 | string | Element UI 圖標名稱 | — |
| size | 按鈕尺寸 | string | medium / small / mini | — |

#### 按鈕組

```html
<el-button-group>
  <el-button type="primary" icon="el-icon-arrow-left">上一頁</el-button>
  <el-button type="primary">下一頁<i class="el-icon-arrow-right el-icon--right"></i></el-button>
</el-button-group>
```

## 二、表單組件

### 2.1 輸入框

#### 基礎輸入框

```html
<el-input v-model="input" placeholder="請輸入內容"></el-input>
```

#### 輸入框屬性

| 屬性 | 說明 | 類型 | 可選值 | 默認值 |
|------|------|------|--------|--------|
| type | 輸入框類型 | string | text / textarea / password | text |
| v-model | 綁定值 | string / number | — | — |
| placeholder | 占位符 | string | — | — |
| disabled | 是否禁用 | boolean | — | false |
| clearable | 是否可清空 | boolean | — | false |
| show-password | 是否顯示密碼切換 | boolean | — | false |
| prefix-icon | 前綴圖標 | string | — | — |
| suffix-icon | 後綴圖標 | string | — | — |

#### 帶圖標的輸入框

```html
<el-input placeholder="請輸入用戶名" prefix-icon="el-icon-user" v-model="username"></el-input>
<el-input placeholder="請輸入密碼" type="password" suffix-icon="el-icon-view" v-model="password"></el-input>
```

#### 文本域

```html
<el-input type="textarea" :rows="3" placeholder="請輸入描述"></el-input>
```

### 2.8 表單驗證

Element UI 提供了強大的表單驗證功能。

#### 基礎驗證

```html
<el-form :model="form" :rules="rules" ref="ruleForm">
  <el-form-item label="用戶名" prop="username">
    <el-input v-model="form.username"></el-input>
  </el-form-item>
  <el-form-item label="密碼" prop="password">
    <el-input v-model="form.password" type="password"></el-input>
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="submitForm('ruleForm')">提交</el-button>
    <el-button @click="resetForm('ruleForm')">重置</el-button>
  </el-form-item>
</el-form>
```

```javascript
export default {
  data() {
    return {
      form: {
        username: '',
        password: ''
      },
      rules: {
        username: [
          { required: true, message: '請輸入用戶名', trigger: 'blur' },
          { min: 3, max: 20, message: '長度在 3 到 20 個字符', trigger: 'blur' }
        ],
        password: [
          { required: true, message: '請輸入密碼', trigger: 'blur' },
          { min: 6, message: '密碼長度至少 6 位', trigger: 'blur' }
        ]
      }
    }
  },
  methods: {
    submitForm(formName) {
      this.$refs[formName].validate(valid => {
        if (valid) {
          console.log('提交成功：', this.form)
        } else {
          this.$message.error('表單驗證失敗')
          return false
        }
      })
    },
    resetForm(formName) {
      this.$refs[formName].resetFields()
    }
  }
}
```

## 三、表格組件

### 3.1 基礎表格

```html
<el-table :data="tableData" stripe border>
  <el-table-column prop="date" label="日期" width="180"></el-table-column>
  <el-table-column prop="name" label="姓名" width="180"></el-table-column>
  <el-table-column prop="address" label="地址"></el-table-column>
</el-table>
```

```javascript
export default {
  data() {
    return {
      tableData: [
        { date: '2024-01-01', name: '張三', address: '北京市朝陽區' },
        { date: '2024-01-02', name: '李四', address: '上海市浦東新區' },
        { date: '2024-01-03', name: '王五', address: '廣州市天河區' }
      ]
    }
  }
}
```

### 3.2 帶操作列的表格

```html
<el-table :data="tableData">
  <el-table-column prop="name" label="姓名"></el-table-column>
  <el-table-column label="操作" width="200">
    <template slot-scope="scope">
      <el-button size="mini" @click="handleEdit(scope.$index, scope.row)">編輯</el-button>
      <el-button size="mini" type="danger" @click="handleDelete(scope.$index, scope.row)">刪除</el-button>
    </template>
  </el-table-column>
</el-table>
```

### 3.3 表格屬性

| 屬性 | 說明 | 類型 | 可選值 | 默認值 |
|------|------|------|--------|--------|
| data | 顯示數據 | array | — | — |
| stripe | 是否斑馬紋 | boolean | — | false |
| border | 是否帶邊框 | boolean | — | false |
| height | 表格高度 | string / number | — | — |
| v-loading | 加載狀態 | boolean | — | false |

### 3.4 帶分頁的表格

```html
<el-table :data="tableData" v-loading="loading">
  <el-table-column prop="title" label="標題"></el-table-column>
  <el-table-column prop="author" label="作者"></el-table-column>
  <el-table-column prop="date" label="日期"></el-table-column>
</el-table>

<el-pagination
  @size-change="handleSizeChange"
  @current-change="handleCurrentChange"
  :current-page="currentPage"
  :page-sizes="[10, 20, 50, 100]"
  :page-size="pageSize"
  layout="total, sizes, prev, pager, next, jumper"
  :total="total">
</el-pagination>
```

## 四、對話框和通知

### 4.1 對話框

#### 基礎對話框

```html
<el-button type="text" @click="dialogVisible = true">打開對話框</el-button>

<el-dialog
  title="編輯"
  :visible.sync="dialogVisible"
  width="30%"
  :before-close="handleClose">
  <span>這是一段信息</span>
  <span slot="footer">
    <el-button @click="dialogVisible = false">取消</el-button>
    <el-button type="primary" @click="dialogVisible = false">確定</el-button>
  </span>
</el-dialog>
```

### 4.2 消息提示

#### 成功提示

```javascript
this.$message.success('操作成功')
```

#### 警告提示

```javascript
this.$message.warning('警告信息')
```

#### 錯誤提示

```javascript
this.$message.error('操作失敗')
```

#### 信息提示

```javascript
this.$message.info('提示信息')
```

### 4.3 通知提醒

```javascript
// 成功通知
this.$notify({
  title: '成功',
  message: '操作已成功完成',
  type: 'success'
})

// 錯誤通知
this.$notify.error({
  title: '錯誤',
  message: '操作失敗'
})
```
