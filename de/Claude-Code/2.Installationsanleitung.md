# Claude Code Installationsanleitung

Detaillierte Installationsanweisungen für Claude Code.

## Installation unter Windows

### Methode Eins: VS Code Erweiterung

1. VS Code öffnen
2. Nach "Claude Code" Erweiterung suchen
3. Auf Installieren klicken
4. Bei Anthropic-Konto anmelden

### Methode Zwei: Eigenständiger Client

1. https://claude.com/code besuchen
2. Windows-Client herunterladen
3. Installer ausführen
4. Anmelden

## Installation unter macOS

### Methode Eins: VS Code Erweiterung

1. VS Code öffnen
2. Nach "Claude Code" Erweiterung suchen
3. Auf Installieren klicken
4. Anmelden

### Methode Zwei: Homebrew

```
brew install claude-code
```

### Methode Drei: Eigenständiger Client

1. https://claude.com/code besuchen
2. macOS-Client herunterladen
3. Installer ausführen
4. Anmelden

## Installation unter Linux

### Methode Eins: VS Code Erweiterung

1. VS Code öffnen
2. Nach "Claude Code" Erweiterung suchen
3. Auf Installieren klicken
4. Anmelden

### Methode Zwei: Paketmanager

**Ubuntu/Debian**:
```
curl -fsSL https://downloads.anthropic.com/claude-code/install.sh | sh
```

**Arch Linux**:
```
yay -S claude-code
```

## Installation überprüfen

```
claude --version
```

## Konfiguration

### API-Schlüssel

1. https://console.anthropic.com besuchen
2. API-Schlüssel erstellen
3. In Claude Code konfigurieren

### Umgebungsvariablen

```
export ANTHROPIC_API_KEY=dein-api-schlüssel
```

## Deinstallieren

### macOS

```
brew uninstall claude-code
```

### Linux

```
sudo rm -rf /usr/local/claude-code
```
