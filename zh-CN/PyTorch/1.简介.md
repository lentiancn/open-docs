# PyTorch - 简介

PyTorch 是 Facebook 开发的开源机器学习框架，以动态计算图和易用性著称。

## 核心特性

- **动态计算图**：即时执行，易于调试
- **Python 优先**：与 Python 深度集成
- **TorchScript**：静态图编译优化
- **分布式训练**：多 GPU、多节点
- **GPU 加速**：CUDA 支持
- **自动微分**：autograd 模块
- **丰富生态**：torchvision、torchtext

## 主要模块

- **torch**：核心张量库
- **torch.nn**：神经网络模块
- **torch.optim**：优化器
- **torch.data**：数据加载
- **torchvision**：计算机视觉

## 应用场景

- 深度学习研究
- 计算机视觉
- 自然语言处理
- 生成对抗网络
