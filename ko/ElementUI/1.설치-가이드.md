# Element UI 설치 가이드

Element UI는 Vue 2.x 애플리케이션용 UI 컴포넌트 라이브러리입니다.

## 버전 개요

| 버전 | Vue 버전 | 상태 |
|------|----------|------|
| Element UI 2.x | Vue 2.6+ | 현재 |
| Element UI 1.x | Vue 2.x | 레거시 |

**참고:** Element UI는 Vue 2 전용입니다. Vue 3은 Element Plus를 사용하세요.

## 전제 조건

### 시스템 요구 사항

| 요구사항 | 버전 |
|----------|------|
| Node.js | 8.0+ |
| npm | 5.0+ |
| Vue | 2.6+ |
| 브라우저 | IE 10+, Chrome, Firefox, Safari |

## 설치 방법

### 방법 1: npm (권장)

```bash
# Element UI 설치
npm install element-ui --save
```

### 방법 2: CDN

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Element UI CDN 데모</title>
  <!-- Vue 2 로드 -->
  <script src="https://unpkg.com/vue@2.6.14/dist/vue.js"></script>
  <!-- Element UI 스타일 로드 -->
  <link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
  <!-- Element UI 로드 -->
  <script src="https://unpkg.com/element-ui/lib/index.js"></script>
</head>
```

## 프로젝트 설정

### 1단계: Vue 프로젝트 생성

```bash
npm install -g @vue/cli
vue create my-element-app
cd my-element-app
```

### 2단계: Element UI 설치

```bash
npm install element-ui --save
```

### 3단계: main.js에서 가져오기

```javascript
import Vue from 'vue'
import App from './App.vue'

import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)

Vue.config.productionTip = false

new Vue({
  render: h => h(App)
}).$mount('#app')
```

## 주문형 가져오기

먼저 플러그인 설치:

```bash
npm install babel-plugin-component -D
```

babel.config.js 설정:

```javascript
module.exports = {
  presets: ['@vue/cli-plugin-babel/preset'],
  plugins: [
    [
      'component',
      {
        libraryName: 'element-ui',
        styleLibraryName: 'theme-chalk'
      }
    ]
  ]
}
```

## 문제 해결

### 컴포넌트를 찾을 수 없음

**해결책:**
1. 설치 확인: `npm list element-ui`
2. main.js 가져오기 확인
3. Vue.use() 호출 확인

### 스타일이 적용되지 않음

**해결책:**
1. CSS 가져오기: `import 'element-ui/lib/theme-chalk/index.css'`
