# PlantUML Preguntas Frecuentes

## Preguntas Generales

### ¿Qué es PlantUML?

PlantUML es una herramienta de código abierto para crear diagramas a partir de descripciones de texto.

### ¿Es gratuito PlantUML?

Sí, PlantUML es de código abierto y gratuito bajo licencia GPL.

---

## Instalación

### ¿Por qué se necesita Java?

PlantUML está escrito en Java y requiere JRE 8 o superior.

### ¿Cómo instalar PlantUML?

- **Windows**: Descargar archivo JAR
- **macOS**: `brew install plantuml`
- **Linux**: Usar gestor de paquetes

---

## Solución de Problemas

### "java" no se reconoce?

Agregar Java al PATH o configurar JAVA_HOME.

### Errores de memoria?

Aumentar tamaño heap de Java:

```bash
java -Xmx2048m -jar plantuml.jar diagrama.wsd
```
