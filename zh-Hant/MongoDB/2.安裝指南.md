# MongoDB 安裝指南

## 版本選擇與平台要求

### 版本選擇

MongoDB 目前有兩個主要版本系列：

- **MongoDB 8.2**：最新穩定版本，推薦新專案使用
- **MongoDB 7.0**：長期支援版本，適合需要穩定性的生產環境

從 MongoDB 8.2 開始，次要版本可用於特定使用案例的本地部署。

### 平台要求

- MongoDB 不再支援 32 位元平台，請確保您的作業系統是 64 位元
- MongoDB 6.0 已於 2025年7月31日停止支援

### 支援的平台

MongoDB 社群版支援以下平台和架構：

**Ubuntu**

- Ubuntu 24.04 (x86_64, arm64)
- Ubuntu 22.04 (x86_64, arm64)
- Ubuntu 20.04 (x86_64, arm64)

**Debian**

- Debian 12 (x86_64)
- Debian 11 (x86_64)

**RHEL/CentOS/Rocky Linux**

- RHEL/Rocky/Alma/Oracle Linux 9.0+ (x86_64, arm64)
- RHEL/Rocky/Alma/Oracle Linux 8.0+ (x86_64, arm64, ppc64le, s390x)
- RHEL/Oracle Linux 7.0+ (x86_64)

**macOS**

- macOS 14 (x86_64, arm64)
- macOS 13 (x86_64, arm64)
- macOS 12 (x86_64, arm64)
- macOS 11 (x86_64, arm64)

**Windows**

- Windows 11 (x86_64)
- Windows Server 2022 (x86_64)
- Windows Server 2019 (x86_64)

**其他 Linux**

- Amazon Linux 2023 (x86_64, arm64)
- Amazon Linux 2 (x86_64, arm64)
- SLES 15 (x86_64)
- SLES 12 (x86_64)

## 在 Ubuntu 上安裝 MongoDB 社群版

### 步驟一：匯入套件管理器的公鑰

```bash
curl -fsSL https://www.mongodb.org/static/pgp/server-8.0.asc | sudo gpg -o /usr/share/keyrings/mongodb-server-8.0.gpg --dearmor
```

### 步驟二：建立 MongoDB 來源列表檔案

根據您的 Ubuntu 版本，選擇合適的配置：

**Ubuntu 24.04 (Noble)**：

```bash
echo "deb [ signed-by=/usr/share/keyrings/mongodb-server-8.0.gpg ] http://repo.mongodb.org/apt/debian noble/mongodb-org/8.0 main" | sudo tee /etc/apt/sources.list.d/mongodb-org-8.0.list
```

**Ubuntu 22.04 (Jammy)**：

```bash
echo "deb [ signed-by=/usr/share/keyrings/mongodb-server-8.0.gpg ] http://repo.mongodb.org/apt/debian jammy/mongodb-org/8.0 main" | sudo tee /etc/apt/sources.list.d/mongodb-org-8.0.list
```

**Ubuntu 20.04 (Focal)**：

```bash
echo "deb [ signed-by=/usr/share/keyrings/mongodb-server-8.0.gpg ] http://repo.mongodb.org/apt/debian focal/mongodb-org/8.0 main" | sudo tee /etc/apt/sources.list.d/mongodb-org-8.0.list
```

### 步驟三：更新套件列表

```bash
sudo apt update
```

### 步驟四：安裝 MongoDB

安裝最新版本：

```bash
sudo apt install -y mongodb-org
```

或者安裝特定版本：

```bash
sudo apt install -y mongodb-org=8.0.0 mongodb-org-server=8.0.0 mongodb-org-shell=8.0.0 mongodb-org-mongos=8.0.0 mongodb-org-tools=8.0.0
```

### 步驟五：啟動 MongoDB

啟動 mongod 服務：

```bash
sudo systemctl start mongod
```

設定開機自動啟動：

```bash
sudo systemctl enable mongod
```

## 在 RHEL/CentOS/Rocky Linux 上安裝

### 使用 YUM/DNF 安裝

#### 步驟一：建立倉庫檔案

建立 `/etc/yum.repos.d/mongodb-org.repo` 檔案：

```bash
[mongodb-org-8.0]
name=MongoDB Repository
baseurl=https://repo.mongodb.org/yum/redhat/$releasever/mongodb-org/8.0/x86_64/
gpgcheck=1
enabled=1
gpgkey=https://www.mongodb.org/static/pgp/server-8.0.asc
```

#### 步驟二：安裝 MongoDB

使用 DNF（推薦）：

```bash
sudo dnf install -y mongodb-org
```

或使用 YUM：

```bash
sudo yum install -y mongodb-org
```

#### 步驟三：啟動 MongoDB

```bash
sudo systemctl start mongod
sudo systemctl enable mongod
```

## 在 macOS 上安裝

### 使用 Homebrew 安裝

#### 步驟一：更新 Homebrew

```bash
brew update
```

#### 步驟二：安裝 MongoDB

```bash
brew install mongodb-community
```

#### 步驟三：啟動 MongoDB

做為背景服務啟動（推薦）：

```bash
brew services start mongodb-community
```

或者手動執行：

```bash
mongod --config /usr/local/etc/mongod.conf
```

#### 步驟四：停止 MongoDB

```bash
brew services stop mongodb-community
```

### 手動安裝

#### 步驟一：下載 MongoDB

```bash
curl -O https://fastdl.mongodb.org/osx/mongodb-macos-x86_64-8.0.0.tgz
```

ARM64 架構：

```bash
curl -O https://fastdl.mongodb.org/osx/mongodb-macos-arm64-8.0.0.tgz
```

#### 步驟二：解壓縮

```bash
tar -zxvf mongodb-macos-*.tgz
```

#### 步驟三：移動二進位檔案

```bash
sudo mkdir -p /usr/local/bin
sudo cp mongodb-macos-*/bin/* /usr/local/bin/
```

#### 步驟四：建立資料目錄

```bash
sudo mkdir -p /data/db
sudo chown -R $(whoami) /data/db
```

#### 步驟五：啟動 MongoDB

```bash
mongod --dbpath /data/db
```

## 在 Windows 上安裝

### 使用 MSI 安裝程式

1. 從 MongoDB 下載中心下載 Windows 版本的 MSI 安裝程式
2. 執行安裝程式，按照精靈步驟操作
3. 選擇「Complete」安裝類型以安裝所有元件
4. 完成安裝後，MongoDB 會自動安裝為 Windows 服務

### 使用 ZIP 套件

#### 步驟一：下載 MongoDB

從 MongoDB 下載中心下載 Windows ZIP 套件。

#### 步驟二：解壓縮

解壓縮檔案到目標目錄，例如 `C:\Program Files\MongoDB`。

#### 步驟三：建立資料目錄

```cmd
mkdir C:\data\db
```

#### 步驟四：啟動 MongoDB

```cmd
"C:\Program Files\MongoDB\Server\8.0\bin\mongod.exe" --dbpath C:\data\db
```

## 使用 Docker 安裝

### 安裝 MongoDB 社群版

#### 步驟一：拉取映像檔

```bash
docker pull mongodb/mongodb-community-server
```

#### 步驟二：執行容器

```bash
docker run --name mongodb -d -p 27017:27017 mongodb/mongodb-community-server
```

#### 資料持久化

如果需要持久化資料，使用磁碟區掛載：

```bash
docker run --name mongodb -d -p 27017:27017 -v mongodb_data:/data/db mongodb/mongodb-community-server
```

### 使用 Docker Compose

建立 `docker-compose.yml` 檔案：

```yaml
version: '3.8'
services:
  mongodb:
    image: mongodb/mongodb-community-server
    container_name: mongodb
    ports:
      - "27017:27017"
    volumes:
      - mongodb_data:/data/db
    environment:
      - MONGO_INITDB_ROOT_USERNAME=admin
      - MONGO_INITDB_ROOT_PASSWORD=yourpassword

volumes:
  mongodb_data:
```

啟動：

```bash
docker-compose up -d
```

## 驗證安裝

### 連線到 MongoDB

安裝完成後，使用 mongosh（MongoDB Shell）連線：

```bash
mongosh
```

如果連線成功，您會看到 MongoDB 的互動式 shell 提示符。

### 檢查版本

在 mongosh 中檢查版本：

```bash
db.version()
```

### 基本命令

```bash
# 查看資料庫
show dbs

# 切換資料庫
use test

# 查看目前資料庫
db

# 查看說明
db.help()
```

## 後續設定

### 建立管理員使用者

在 mongosh 中執行：

```bash
use admin
db.createUser({
  user: "admin",
  pwd: "your_secure_password",
  roles: [
    { role: "userAdminAnyDatabase", db: "admin" },
    { role: "readWriteAnyDatabase", db: "admin" }
  ]
})
```

### 啟用身份驗證

1. 停止 MongoDB 服務
2. 修改 `/etc/mongod.conf` 啟用身份驗證：

```yaml
security:
  authorization: enabled
```

3. 重新啟動 MongoDB 服務：

```bash
sudo systemctl restart mongod
```

### 設定遠端連線

1. 修改 `/etc/mongod.conf` 中的網路設定：

```yaml
net:
  port: 27017
  bindIp: 0.0.0.0
```

2. 重新啟動 MongoDB 服務

3. 在防火牆中開啟連接埠：

```bash
sudo ufw allow 27017
```

## 解除安裝 MongoDB

### Ubuntu/Debian

```bash
sudo apt remove --purge mongodb-org
sudo rm -rf /var/lib/mongodb
sudo rm -rf /var/log/mongodb
sudo rm -rf /etc/mongod.conf
```

### macOS (Homebrew)

```bash
brew uninstall mongodb-community
brew services stop mongodb-community
rm -rf /usr/local/etc/mongod.conf
rm -rf /data/db
```

### RHEL/CentOS

```bash
sudo yum remove -y mongodb-org
sudo rm -rf /var/lib/mongo
sudo rm -rf /var/log/mongodb
```

### Windows

透過控制台的「程式和功能」解除安裝 MongoDB，然後手動刪除資料目錄。

## 常見問題

### MongoDB 預設連接埠是多少？

預設連接埠是 27017。也可以透過 `--port` 參數指定其他連接埠。

### MongoDB 資料儲存在哪裡？

預設情況下：
- Linux/macOS：`/data/db`
- Windows：`C:\data\db`

可以透過修改設定檔中的 `storage.dbPath` 來變更。

### 如何檢查 MongoDB 執行狀態？

```bash
sudo systemctl status mongod
```

或者在 mongosh 中：

```bash
db.adminCommand({ ping: 1 })
```
