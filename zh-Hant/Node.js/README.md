# Node.js

Node.js® 是一個免費、開源、跨平台的 JavaScript 執行環境，它讓開發人員能夠建立伺服器、Web 應用程式、命令列工具和指令碼。

## 快速開始

### 安裝 Node.js

造訪 [Node.js 官方網站](https://nodejs.org/zh-cn/download) 下載並安裝最新版本。

### 驗證安裝

```bash
node --version
npm --version
```

### 建立第一個應用

建立 `hello.js` 檔案：
```javascript
console.log('Hello World!');
```

執行應用：
```bash
node hello.js
```
