# Python 常见问题

> 常见问题解答

---

## 安装问题

### Python 命令找不到？

**Windows**：确保安装时勾选了 "Add Python to PATH"，或手动添加到环境变量

**macOS/Linux**：使用 `python3` 代替 `python`

### pip 安装失败？

**解决方案**：

1. 升级 pip：
```bash
python -m pip install --upgrade pip
```

2. 使用国内镜像源：
```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple 包名
```

## 环境问题

### Python 2 和 Python 3 共存？

**方案一**：使用 `python3` 运行 Python 3

**方案二**：使用 pyenv 管理多版本
```bash
pyenv install 3.12.0
pyenv global 3.12.0
```

### 虚拟环境有什么用？

虚拟环境可以隔离不同项目的依赖，避免包冲突。

```bash
# 创建
python -m venv myenv

# 激活
# Windows
myenv\Scripts\activate
# Linux/macOS
source myenv/bin/activate
```

## 语法问题

### list 和 tuple 的区别？

- **list**：可变列表，可增删改查
- **tuple**：不可变元组，定义后不能修改

### dict 和 set 的区别？

- **dict**：键值对，可通过键快速查找值
- **set**：无序不重复元素集合，用于去重

### == 和 is 的区别？

- `==`：比较值是否相等
- `is`：比较是否为同一对象（内存地址）

```python
a = [1, 2, 3]
b = [1, 2, 3]

a == b  # True，值相等
a is b  # False，不是同一对象
```

## 编码问题

### 中文显示乱码？

在文件开头添加编码声明：
```python
# -*- coding: utf-8 -*-
```

或（Python 3 推荐）：
```python
# -*- coding: utf-8 -*-
```

print 时指定编码：
```python
print("你好", encoding="utf-8")
```

## 性能问题

### Python 慢怎么办？

1. 使用内置函数和库（它们是 C 实现的）
2. 使用列表推导式代替循环
3. 使用 NumPy、Pandas 等优化库
4. 对于性能关键代码，可用 Cython 或 Numba 加速

### 如何分析代码性能？

```bash
# 使用 cProfile
python -m cProfile script.py

# 使用 timeit
import timeit
timeit.timeit("code", setup="code")
```

## 打包问题

### 如何打包 Python 程序？

使用 PyInstaller：

```bash
pip install pyinstaller

# 打包为单个可执行文件
pyinstaller --onefile script.py
```

### 如何发布自己的包？

1. 创建 setup.py
2. 打包：`python setup.py sdist`
3. 上传到 PyPI

## 常见错误

### IndentationError？

Python 使用缩进表示代码块，确保缩进一致（通常 4 个空格）。

### ModuleNotFoundError？

使用 pip 安装缺少的模块：
```bash
pip install 模块名
```

### NameError？

变量未定义或拼写错误。
