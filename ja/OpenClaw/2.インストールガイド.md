# OpenClaw インストールガイド

OS別インストール手順の説明です。

## 動作環境

- Node.js：22 以上
- OS：macOS、Linux、Windows
- オプション：Docker（サンドボックス機能を使う場合）

## インストール方法

### macOS / Linux（おすすめ）

公式インストールスクリプトを使うのが一番簡単：

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

### Windows

PowerShellで：

```powershell
iwr -useb https://openclaw.ai/install.ps1 | iex
```

### npmで直接

Node.jsが入ってたら：

```bash
npm install -g openclaw@latest
```

### ソースから

最新開発版を使うなら：

```bash
git clone https://github.com/openclaw/openclaw.git
cd openclaw
npm install
npm link
```

## インストール後の設定

### セットアップウィザード

最初はウィザードを実行おすすめ：

```bash
openclaw onboard --install-daemon
```

やること：
- APIキー設定
- ポート設定
- チャンネルログイン
- セキュリティ設定

### 動作確認

インストールしたら：

```bash
openclaw gateway status
```

### 管理画面を開く

```bash
openclaw dashboard
```

またはブラウザで：http://127.0.0.1:18789/

## 確認方法

1. `openclaw --version` でバージョン確認
2. `openclaw doctor` でシステムチェック

## よくある問題

### Node.jsバージョン

```bash
node --version
```

22以上あるか確認。

### 権限エラー

```bash
sudo npm install -g openclaw@latest
```
