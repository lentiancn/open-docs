# Element UI 설치 가이드

완전한 Element UI 설치 및 설정 가이드.

## 환경 요구사항

### 소프트웨어 요구사항

- Node.js: 12.0.0+
- npm: 6.0.0+ 또는 yarn: 1.22.0+
- Vue: 2.6.0+

## 설치 단계

### 방법 1: npm 설치 (권장)

```bash
# npm 사용
npm install element-ui --save

# 또는 yarn 사용
yarn add element-ui
```

### 방법 2: CDN 임포트

```html
<!-- 스타일 임포트 -->
<link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">

<!-- Vue 임포트 -->
<script src="https://unpkg.com/vue"></script>

<!-- Element UI 임포트 -->
<script src="https://unpkg.com/element-ui/lib/index.js"></script>
```

## 빠른 시작

### 전체 임포트

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(ElementUI)

new Vue({
  el: '#app',
  render: h => h(App)
})
```

### 온디맨드 임포트 (권장)

먼저 babel-plugin-component 설치:

```bash
npm install babel-plugin-component -D
```

babel.config.js 설정:

```javascript
module.exports = {
  presets: [
    '@vue/cli-plugin-babel/preset'
  ],
    [
      '  plugins: [
component',
      {
        libraryName: 'element-ui',
        styleLibraryName: 'theme-chalk'
      }
    ]
  ]
}
```

컴포넌트 온디맨드 임포트:

```javascript
import { Button, Select, Option, Table, TableColumn } from 'element-ui'

Vue.use(Button)
Vue.use(Select)
Vue.use(Option)
Vue.use(Table)
Vue.use(TableColumn)
```

## 테마 커스터마이징

### SCSS 변수 사용

변수 파일 생성:

```scss
// styles/element-variables.scss

$--color-primary: #409EFF;
$--color-success: #67C23A;
$--color-warning: #E6A23C;
$--color-danger: #F56C6C;

@import 'element-ui/packages/theme-chalk/src/common/var.scss';
```

main.js에서 임포트:

```javascript
import './styles/element-variables.scss'
```

## 국제화

### 다국어 설정

vue-i18n 설치:

```bash
npm install vue-i18n -S
```

설정:

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import VueI18n from 'vue-i18n'
import koLocale from 'element-ui/lib/locale/lang/ko'

Vue.use(ElementUI, {
  i18n: (key, value) => i18n.t(key, value)
})

const messages = {
  'ko': { message: '안녕하세요' }
}

const i18n = new VueI18n({
  locale: 'ko',
  messages
})

new Vue({ i18n }).$mount('#app')
```

## 일반적인 문제

### 컴포넌트가 표시되지 않음?

1. 컴포넌트가 올바르게 임포트되었는지 확인
2. Vue.use()가 호출되었는지 확인
3. 콘솔에서 오류 확인

### 스타일이 적용되지 않음?

1. 스타일 파일이 임포트되었는지 확인
2. 전역 스타일 오버라이드 확인
3. z-index 레벨 확인

### 아이콘이 표시되지 않음?

```bash
# 아이콘 라이브러리 설치
npm install element-ui-icons
```

아이콘 임포트:

```javascript
import 'element-ui-icons'
```

사용:

```html
<i class="el-icon-search"></i>
```

## 제거

```bash
npm uninstall element-ui
```
