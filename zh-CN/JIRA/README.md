# Jira

Jira 是由 Atlassian 公司开发的项目管理与事务跟踪软件，为全球超过30万家企业提供项目管理解决方案。

## 快速开始

访问 Atlassian 官网注册账号，选择适合您团队的计划，即可快速创建 Jira 站点。

## 主要功能

项目管理与事务跟踪、敏捷开发支持（Scrum、看板）、团队协作与沟通、人工智能助手 Rovo 集成、丰富的第三方集成、强大的自动化规则。

## 定价
