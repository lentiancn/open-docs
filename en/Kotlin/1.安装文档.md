# Kotlin 安装指南

Kotlin 是 JetBrains 开发的现代编程语言。本指南涵盖在各种环境中的安装方法。

## 版本信息

| 版本 | 状态 | 说明 |
|------|------|------|
| Kotlin 2.x | ✅ 当前 | 最新版本 |
| Kotlin 1.9.x | ✅ 支持 | 稳定版本 |

## 系统要求

### 硬件要求

| 要求 | 配置 |
|------|------|
| CPU | 1+ 核 |
| 内存 | 2+ GB |
| 磁盘 | 500+ MB |

### 软件要求

| 软件 | 版本 |
|------|------|
| Java | JDK 8 或更高 |

## 安装方法

### 方法一：IntelliJ IDEA (推荐)

1. 下载 IntelliJ IDEA：https://www.jetbrains.com/idea/download/
2. 安装 Ultimate 或 Community 版本
3. Kotlin 已内置支持

### 方法二：命令行

#### Windows

```powershell
# 使用 Chocolatey
choco install kotlin

# 或手动下载
# 从 https://github.com/JetBrains/kotlin/releases 下载
```

#### macOS

```bash
# 使用 Homebrew
brew install kotlin

# 或手动安装
curl -s https://get.sdkman.io | bash
sdk install kotlin
```

#### Linux

```bash
# Ubuntu/Debian
sudo apt install kotlin

# 或使用 SDKMAN
curl -s https://get.sdkman.io | bash
sdk install kotlin
```

### 方法三：SDKMAN

```bash
# 安装 SDKMAN
curl -s "https://get.sdkman.io" | bash

# 安装 Kotlin
sdk install kotlin

# 验证
kotlin -version
```

### 方法四：Maven

```xml
<dependencies>
    <dependency>
        <groupId>org.jetbrains.kotlin</groupId>
        <artifactId>kotlin-stdlib</artifactId>
        <version>2.0.0</version>
    </dependency>
</dependencies>
```

### 方法五：Gradle

```groovy
plugins {
    id 'org.jetbrains.kotlin.jvm' version '2.0.0'
}

repositories {
    mavenCentral()
}

dependencies {
    implementation "org.jetbrains.kotlin:kotlin-stdlib"
}
```

### 方法六：Docker

```dockerfile
# 使用 Kotlin 编译器镜像
FROM gradle:8.5-jdk-21 AS builder
WORKDIR /app
COPY . .
RUN gradle build

# 运行镜像
FROM openjdk:21
COPY --from=builder /app/build/libs/*.jar /app.jar
ENTRYPOINT ["java", "-jar", "/app.jar"]
```

## 环境配置

### Java 配置

```bash
# 检查 Java
java -version

# 设置 JAVA_HOME
export JAVA_HOME=/path/to/java
```

### 编译运行

```bash
# 编译
kotlinc hello.kt -include-runtime -d hello.jar

# 运行
java -jar hello.jar

# 使用 kotlin 脚本
kotlin hello.kts
```

## 常见问题

### 编译错误？

检查 Kotlin 和 Java 版本兼容性。

### 内存不足？

增加 JVM 堆内存：

```bash
kotlinc -J-Xmx2g
```
