# OpenClaw ユーザーマニュアル

## クイックスタート

```bash
openclaw gateway --port 18789
openclaw dashboard
```

## チャンネル管理

```bash
openclaw channels list
openclaw channels add whatsapp
openclaw channels login
```

## メッセージ送信

```bash
openclaw message send --target +15555550123 --message "Hello"
```

## セッション管理

```bash
openclaw sessions list
```

## 設定

```bash
openclaw config get
openclaw config set channels.whatsapp.allowFrom "+15555550123"
```

## トラブルシューティング

```bash
openclaw logs
openclaw status
```

## コマンド一覧

| コマンド | 説明 |
|----------|------|
| openclaw gateway start | Gateway起動 |
| openclaw dashboard | ダッシュボード |
| openclaw channels list | チャンネル一覧 |
| openclaw sessions list | セッション一覧 |
