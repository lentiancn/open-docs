# OpenCode Preguntas frecuentes

Preguntas y respuestas comunes.

## Instalación

### Q: ¿La instalación falló?

**Soluciones**:
1. Verifica los requisitos del sistema (necesita un emulador de terminal moderno)
2. Confirma la conexión de red
3. Intenta el script de instalación: `curl -fsSL https://opencode.ai/install | bash`
4. Revisa el mensaje de error e intenta de nuevo

### Q: ¿No se puede instalar en Windows?

**Soluciones**:
1. Intenta Chocolatey: `choco install opencode`
2. O usa Scoop: `scoop install opencode`
3. O usa NPM: `npm install -g opencode-ai`

### Q: ¿La verificación de instalación falló?

**Soluciones**:
1. Confirma que opencode está en el PATH
2. Reinicia la terminal
3. Verifica si la instalación fue exitosa

## Uso

### Q: ¿No puede conectar a LLM?

**Soluciones**:
1. Ejecuta el comando `/connect`
2. Visita opencode.ai/auth para iniciar sesión
3. Agrega datos de facturación y obtén la clave API
4. Pega la clave API en la terminal

### Q: ¿La respuesta es lenta?

**Soluciones**:
1. Verifica la conexión de red
2. Intenta cambiar a otro proveedor LLM
3. Reduce la longitud del contexto de conversación

### Q: ¿Cómo cambiar de modelo?

Usa el comando `/connect` para seleccionar un proveedor LLM diferente.

### Q: ¿Qué modelos se soportan?

Soporta 75+ proveedores LLM:
- OpenAI (GPT-4, Claude, etc.)
- Anthropic (Claude)
- Google (Gemini)
- Modelos locales
- Ver [Models.dev](https://models.dev)

## Funciones

### Q: ¿Qué es AGENTS.md?

AGENTS.md es un archivo creado cuando OpenCode se inicializa, que contiene la estructura del proyecto y las convenciones de código para ayudar a OpenCode a entender el proyecto.

### Q: ¿Cuál es la diferencia entre modo Plan y Build?

- **Modo Plan**: Solo genera planes, no modifica código
- **Modo Build**: Ejecuta cambios directamente

Presiona **Tab** para cambiar entre modos.

### Q: ¿Cómo compartir sesiones?

Usa el comando `/share` para crear un enlace para compartir.

## Privacidad

### Q: ¿OpenCode guarda mi código?

No. OpenCode no almacena ningún código o dato de contexto, es seguro usar en ambientes sensibles.

### Q: ¿Cómo proteger la privacidad?

- No ingreses información sensible en las conversaciones
- Usa modelos locales
- Limpia las sesiones regularmente
