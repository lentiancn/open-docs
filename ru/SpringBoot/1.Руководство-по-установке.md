# Руководство по установке Spring Boot

В этом руководстве описана установка и настройка среды разработки Spring Boot в Windows, macOS и Linux.

## Системные требования

### Требования к версии Java

| Версия Spring Boot | Минимальная версия Java |
|-------------------|------------------------|
| 3.x               | Java 17+               |
| 2.7.x             | Java 8+                |

## Установка Java

### Windows

1. Скачать JDK с [Oracle](https://www.oracle.com/java/technologies/downloads/) или [Adoptium](https://adoptium.net/)
2. Запустить установщик
3. Настроить переменные среды:
   ```
   JAVA_HOME=C:\Program Files\Java\jdk-21
   PATH=%JAVA_HOME%\bin;%PATH%
   ```

### macOS

```bash
brew install openjdk@21
echo 'export JAVA_HOME=$(/usr/libexec/java_home -v 21)' >> ~/.zshrc
source ~/.zshrc
```

### Linux

```bash
sudo apt update
sudo apt install openjdk-21-jdk
export JAVA_HOME=/usr/lib/jvm/java-21-openjdk-amd64
export PATH=$JAVA_HOME/bin:$PATH
```

## Установка Maven

### Windows

1. Скачать с [Apache Maven](https://maven.apache.org/download.cgi)
2. Распаковать в `C:\apache-maven`
3. Настроить переменные среды

### macOS

```bash
brew install maven
```

### Linux

```bash
sudo apt install maven
```

## Настройка IDE

### IntelliJ IDEA (Рекомендуется)

1. Скачать с [JetBrains](https://www.jetbrains.com/idea/download/)
2. Настроить JDK: `File` → `Project Structure` → `Project SDK`

### VS Code

- Установить Extension Pack for Java
- Установить Spring Boot Extension Pack

## Быстрый старт

### Использование Spring Initializr

1. Открыть [start.spring.io](https://start.spring.io/)
2. Выбрать:
   - Проект: Maven / Gradle
   - Язык: Java
   - Spring Boot: 3.2.x
   - Group: com.example
   - Artifact: myapp
3. Добавить зависимости:
   - Spring Web
   - Spring Data JPA
   - H2 Database

### Командная строка

```bash
curl https://start.spring.io/starter.zip \
  -d type=maven-project \
  -d language=java \
  -d bootVersion=3.2.5 \
  -d baseDir=myapp \
  -d dependencies=web,jpa,h2 \
  -o myapp.zip

unzip myapp.zip
cd myapp
./mvnw spring-boot:run
```

## Проверка установки

```bash
java -version
mvn -version
```
