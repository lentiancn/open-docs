# База данных H2 - Введение

## Что такое H2?

H2 — это система управления реляционными базами данных с открытым исходным кодом, написанная на Java. Она может использоваться как встроенная база данных в приложениях Java или работать в режиме клиент-сервер. Программное обеспечение распространяется под лицензией Mozilla Public License 2.0 (MPL 2.0) или исходной Eclipse Public License (EPL 1.0).

## Основные характеристики H2

### Встроенная база данных
H2 может быть встроена непосредственно в приложения Java для работы без отдельного сервера базы данных. Это делает её очень популярной в средах разработки и тестирования.

### Режим клиент-сервер
Помимо встроенного режима, H2 также поддерживает режим клиент-сервер, работая как автономный сервер базы данных, к которому могут одновременно обращаться несколько приложений.

### База данных в памяти
H2 поддерживает режим базы данных в памяти, где данные хранятся в оперативной памяти и теряются после перезагрузки. Это очень полезно для быстрого тестирования и разработки прототипов.

### Совместимость с несколькими базами данных
H2 совместима с различными стандартами и синтаксисом SQL и может имитировать поведение MySQL, PostgreSQL, Oracle, DB2, MS SQL Server, что упрощает миграцию и тестирование баз данных.

### Шифрованная база данных
H2 поддерживает шифрование базы данных для защиты конфиденциальных данных.

### Полнотекстовый поиск
H2 имеет встроенную функцию полнотекстового поиска без дополнительной настройки.

### Нулевое администрирование
В качестве встроенной базы данных H2 не требует выделенного администратора базы данных для обслуживания.

## Области применения H2

H2 широко используется в следующих сценариях:

- Встроенная база данных для Java-приложений
- Среды разработки и тестирования
- Быстрая разработка прототипов
- Модульное и интеграционное тестирование
- Легкие настольные приложения
- Изучение синтаксиса SQL

## История версий H2

Разработка движка базы данных H2 началась в мае 2004 года, первая публикация состоялась в декабре 2005 года. Оригинальный разработчик Томас Мюллер также является основателем движка базы данных Java HSQLDB ( первоначально называвшегося Hypersonic SQL).

## Системные требования

- Java Runtime Environment (JRE) 11 или выше
- Может работать на любой операционной системе с поддержкой Java, включая Windows, Linux, macOS и т.д.
