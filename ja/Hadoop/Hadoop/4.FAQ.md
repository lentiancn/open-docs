# Hadoop FAQ

## Installation

### Q: NameNode format fails?
**A**: 
- Check Java installation: `java -version`
- Verify JAVA_HOME is set correctly
- Delete temp directory and retry

### Q: DataNode won't start?
**A**:
- Check network connection
- Verify node addresses in config files
- Check logs: `tail -f $HADOOP_HOME/logs/hadoop-*-datanode-*.log`

## HDFS

### Q: File upload fails?
**A**:
- Check DataNode status
- Verify directory permissions
- Check disk space

### Q: How to adjust block size?
**A**:
In hdfs-site.xml:
```xml
<property>
    <name>dfs.blocksize</name>
    <value>134217728</value>
</property>
```

## MapReduce

### Q: Job stuck in ACCEPTED state?
**A**:
- Check ResourceManager status
- Verify cluster resources
- Check queue configuration
