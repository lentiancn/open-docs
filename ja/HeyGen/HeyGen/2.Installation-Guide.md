# HeyGen Installation Guide

## Access

HeyGen is a cloud-based SaaS platform, no local installation required.

## Getting Started

1. Visit https://heygen.com to register
2. Log in to the console
3. Choose a template or create a digital human
4. Input text and generate video

## API Access

```bash
curl -X POST https://api.heygen.com/v1/video/generate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"script": "Hello World", "avatar_id": "avatar_1"}'
```
