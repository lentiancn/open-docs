# Element UI

> Biblioteca de componentes UI de escritorio basada en Vue 2

---

## Descripción

Element UI es una biblioteca de componentes de escritorio basada en Vue 2.0 desarrollada por el equipo Eleme. Proporciona componentes ricos.

**Nota**: Para proyectos Vue 3, use [Element Plus](https://github.com/element-plus/element-plus)

---

## Versión

| Versión | Vue | Estado |
|---------|-----|--------|
| Element UI 2.x | Vue 2 | Estable |
| Element UI 1.x | Vue 2 | Obsoleto |
