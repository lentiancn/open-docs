# GitLab Руководство по установке

> Полное руководство по установке и настройке GitLab

---

## Системные требования

| Ресурс | Минимум | Рекомендуется |
|--------|---------|---------------|
| CPU | 2 ядра | 4+ ядра |
| ОЗУ | 4 ГБ | 8+ ГБ |
| Диск | 10 ГБ | 50+ ГБ |

---

## Установка

### Omnibus (Linux)

#### Ubuntu/Debian

```bash
# Установить зависимости
sudo apt-get update
sudo apt-get install -y curl openssh-server ca-certificates

# Добавить репозиторий GitLab
curl -fsSL https://packages.gitlab.com/install/repositories/gitlab/gitlab-ee/script.deb.sh | sudo bash

# Установить GitLab
sudo EXTERNAL_URL="https://gitlab.example.com" apt-get install gitlab-ee
```

---

## Начальная настройка

### Настройка внешнего URL

```bash
sudo vim /etc/gitlab/gitlab.rb
external_url 'https://gitlab.example.com'
sudo gitlab-ctl reconfigure
```

---

## Основные команды

```bash
# Проверить статус
sudo gitlab-ctl status

# Запустить
sudo gitlab-ctl start

# Остановить
sudo gitlab-ctl stop

# Перезапустить
sudo gitlab-ctl restart
```

---

## Устранение неполадок

### Установка не удается?

- Проверьте системные требования
- Установите все зависимости
- Проверьте настройки брандмауэра

### GitLab не запускается?

```bash
# Проверить порты
sudo netstat -tulpn | grep -E ':(80|443|22)\s'

# Проверить место на диске
df -h

# Проверить логи
sudo gitlab-ctl tail
```
