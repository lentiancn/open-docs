# JIRA 설치 가이드

JIRA는 Atlassian에서 개발한 문제 추적 및 프로젝트 관리 도구입니다. 이 가이드에서는 다양한 환경에서의 설치 방법을 설명합니다.

## 버전 정보

| 버전 | 상태 | 설명 |
|------|------|------|
| JIRA 9.x | ✅ 현재 | 최신 버전 |
| JIRA 8.x | ✅ 지원 | 안정 버전 |

## 시스템 요구사항

### 하드웨어 요구사항

| 요구사항 | 최소 | 권장 |
|----------|------|------|
| CPU | 4코어 | 8+코어 |
| 메모리 | 4GB | 16+GB |
| 디스크 | 10GB | 50+GB |

### 소프트웨어 요구사항

| 소프트웨어 | 버전 |
|-----------|------|
| Java | JDK 11 또는 17 |
| 운영체제 | Linux/Windows/macOS |
| 데이터베이스 | PostgreSQL/MySQL/Oracle |

## 설치 방법

### 방법 1: Windows 설치

#### 1. 설치 프로그램 다운로드

1. https://www.atlassian.com/software/jira/download 방문
2. JIRA Software 다운로드
3. 설치 프로그램 실행

#### 2. 설치 실행

설치 마법사に従って 설치 완료:
- 설치 유형 선택
- 포트 설정
- 데이터베이스 구성
- 관리자 계정 생성

### 방법 2: Linux 설치

#### 1. 설치 패키지 다운로드

```bash
# JIRA 다운로드
wget https://product-downloads.atlassian.com/software/jira/downloads/atlassian-jira-software-X.X.X-x64.bin
```

#### 2. 설치

```bash
# 실행 권한 추가
chmod +x atlassian-jira-software-X.X.X-x64.bin

# 설치 실행
sudo ./atlassian-jira-software-X.X.X-x64.bin
```

#### 3. 구성

```bash
# JIRA 시작
sudo /opt/atlassian/jira/bin/start-jira.sh

# JIRA 중지
sudo /opt/atlassian/jira/bin/stop-jira.sh
```

### 방법 3: Docker 설치

```bash
# 이미지 풀
docker pull atlassian/jira-software

# 컨테이너 실행
docker run -v jira-data:/var/atlassian/application-data/jira --name="jira" -d -p 8080:8080 atlassian/jira-software
```

### 방법 4: AWS 사용

1. AWS Marketplace에서 JIRA 검색
2. 인스턴스 유형 선택
3. 데이터베이스 구성
4. 인스턴스 시작

## 구성

### 데이터베이스 구성

```properties
# dbconfig.xml
<?xml version="1.0" encoding="UTF-8"?>
<jira-database-config>
  <name>defaultDS</name>
  <connection-type>default</connection-type>
  <url>jdbc:postgresql://localhost:5432/jira</url>
  <username>jira</username>
  <password>password</password>
  <pool-size>20</pool-size>
</jira-database-config>
```

### 리버스 프록시 구성

#### Nginx

```nginx
server {
    listen 80;
    server_name jira.example.com;
    
    location / {
        proxy_pass http://localhost:8080;
        proxy_set_header X-Forwarded-Host $host;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

## 자주 묻는 질문

### 시작 실패?

Java 버전 확인:

```bash
java -version
```

### 데이터베이스 연결 실패?

### 메모리 부족?

메모리 설정 수정:

```bash
# setenv.sh
export JVM_MINIMUM_MEMORY="2048m"
export JVM_MAXIMUM_MEMORY="4096m"
```
