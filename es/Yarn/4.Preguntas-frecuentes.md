# Preguntas Frecuentes de Yarn

## Preguntas Básicas

### ¿Qué es Yarn?

Yarn es un gestor de paquetes JavaScript que ayuda a instalar, actualizar y eliminar dependencias.

### ¿Yarn es gratis?

Sí, es código abierto y gratis (MIT).

### ¿Diferencia con npm?

- Velocidad: Yarn es más rápido
- Determinismo: yarn.lock asegura consistencia

---

## Instalación

### ¿Instalación fallida?

1. Verificar Node.js: `node --version`
2. Usar corepack: `corepack enable`

---

## Uso

### ¿Cómo agregar dependencias?

```bash
yarn add lodash
```

### ¿Cómo actualizar?

```bash
yarn up lodash
```
