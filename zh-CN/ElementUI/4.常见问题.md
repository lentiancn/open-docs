# Element UI 常见问题

本页面列出 Element UI 使用过程中最常见的问题及解决方案。

## 安装问题

### 安装失败

**症状**：npm install 失败

**解决方案**：

1. 检查 Node.js 版本，确保 >= 8.0.0
2. 清除 npm 缓存：
   ```bash
   npm cache clean --force
   ```
3. 删除 node_modules 和 package-lock.json 后重新安装

### 按需导入不生效

**症状**：打包后体积仍然很大

**解决方案**：

1. 确保 .babelrc 配置正确
2. 检查 babel-plugin-component 是否正确安装

## 使用问题

### 组件样式不生效

**症状**：组件样式与预期不符

**解决方案**：

1. 确保已正确引入 CSS：
   ```javascript
   import 'element-ui/lib/theme-chalk/index.css'
   ```

### 图标不显示

**症状**：图标组件显示为空白或方块

**解决方案**：

1. 引入图标字体：
   ```javascript
   import 'element-ui/lib/theme-chalk/fonts/ element-icons.ttf'
   ```

### 对话框问题

#### 对话框滚动问题

在对话框内容过多时，确保添加滚动：

```vue
<el-dialog>
  <div style="height: 400px; overflow-y: scroll;">
    <!-- 内容 -->
  </div>
</el-dialog>
```

## Vue 版本问题

### 与 Vue 3 不兼容

Element UI 仅支持 Vue 2。如果需要 Vue 3，请使用 Element Plus。

### 使用 babel6 问题

如果使用 babel6，确保正确配置：

```json
{
  "presets": ["es2015", "stage-2"],
  "plugins": ["transform-runtime"]
}
```

## 样式问题

### 样式冲突

**症状**：自定义样式被覆盖

**解决方案**：

1. 使用更具体的选择器
2. 使用 ::deep()

### 主题色不生效

检查 SCSS 变量是否被正确覆盖：

```scss
$--color-primary: #409eff;
```

## 性能问题

### 页面卡顿

对于大数据量：

1. 使用分页加载数据
2. 避免在表格中使用复杂组件

### 打包体积过大

1. 使用按需导入
2. 移除未使用的组件
