# Element Plus Manual de Usuario

Guía de uso.

## Inicio Rápido

```vue
<template>
  <el-button type="primary" @click="handleClick">
    Clic
  </el-button>
</template>

<script setup>
import { ref } from 'vue'

const handleClick = () => {
  ElMessage.success('¡Hola!')
}
</script>
```

## Componentes

### Button

```vue
<el-button>Predeterminado</el-button>
<el-button type="primary">Primario</el-button>
<el-button disabled>Deshabilitado</el-button>
<el-button loading>Cargando</el-button>
```

### Input

```vue
<el-input v-model="value" placeholder="Ingrese texto" />
<el-input v-model="value" disabled placeholder="Deshabilitado" />
<el-input v-model="value" clearable placeholder="Limpiable" />
```

### Table

```vue
<el-table :data="tableData" stripe border>
  <el-table-column prop="name" label="Nombre" />
  <el-table-column prop="email" label="Correo" />
</el-table>
```

### Form

```vue
<el-form :model="form" :rules="rules" ref="formRef">
  <el-form-item label="Nombre" prop="name">
    <el-input v-model="form.name" />
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="submit">Enviar</el-button>
  </el-form-item>
</el-form>
```

### Dialog

```vue
<el-button @click="visible = true">Abrir</el-button>
<el-dialog v-model="visible" title="Título">
  <span>Contenido</span>
</el-dialog>
```

### Message

```typescript
import { ElMessage } from 'element-plus'

ElMessage.success('Éxito')
ElMessage.error('Error')
```

## Diseño

```vue
<el-row :gutter="20">
  <el-col :span="12"><div>12</div></el-col>
  <el-col :span="12"><div>12</div></el-col>
</el-row>
```
