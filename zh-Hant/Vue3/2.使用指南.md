# Vue 3 使用指南

高效使用 Vue 3 的指南。

---

## Composition API

```javascript
import { ref } from 'vue'

const count = ref(0)
count.value++
```

---

## 模板

```html
<h1>{{ title }}</h1>
<button @click="handleClick">點擊</button>
```

---

## 組件

```vue
<script setup>
defineProps({
  title: String
})
</script>
```
