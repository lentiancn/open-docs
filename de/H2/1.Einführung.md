# Einführung in H2

## Was ist H2?

H2 ist ein in Java geschriebenes Open-Source-relationales Datenbankmanagementsystem. Es kann als eingebettete Datenbank in Java-Anwendungen verwendet oder im Client-Server-Modus betrieben werden. Die Software ist als Open Source unter der Mozilla Public License 2.0 (MPL 2.0) oder der Eclipse Public License 1.0 (EPL 1.0) verfügbar.

## Hauptmerkmale

- Eingebettete Datenbank: Kein separater Datenbankserver erforderlich
- Client-Server-Modus: Als eigenständiger Server ausführbar
- In-Memory-Datenbank: Schnelltests und Prototypentwicklung
- Multi-Datenbank-Kompatibilität: MySQL, PostgreSQL, Oracle, etc.
- Verschlüsselte Datenbanken: Datensicherheit
- Volltextsuche: Integriert
- Zero-Management: Kein DBA erforderlich

## Anwendungsbereiche

- Java-Anwendungen (eingebettet)
- Entwicklung und Testumgebungen
- Schnelle Prototypentwicklung
- Unit- und Integrationstests

## Lizenz

Dual-Lizenz: MPL 2.0 oder EPL 1.0

## Systemanforderungen

Java 11 oder höher
