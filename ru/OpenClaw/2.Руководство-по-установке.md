# Установка OpenClaw

Пошаговое руководство по установке на разные ОС.

## Требования

- Node.js 22+
- macOS / Linux / Windows
- Docker (опционально, для изоляции)

## Установка

### macOS / Linux

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

### Windows (PowerShell)

```powershell
iwr -useb https://openclaw.ai/install.ps1 | iex
```

### npm

```bash
npm install -g openclaw@latest
```

## После установки

```bash
openclaw onboard --install-daemon
openclaw gateway status
openclaw dashboard
```

Откройте http://127.0.0.1:18789/

## Проверка

```bash
openclaw --version
openclaw doctor
```

## Частые проблемы

- `node --version` — проверка версии Node.js
- `openclaw logs` — просмотр логов
- `openclaw doctor` — диагностика
