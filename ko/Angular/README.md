# Angular

> Google 개발 웹 프레임워크

---

## 개요

Angular는 Google이 개발한 TypeScript 기반 프레임워크입니다. 최신 웹 애플리케이션을 위한 완전한 솔루션을 제공합니다.

---

## 버전

| 버전 |发布日期| 상태 |
|------|--------|------|
| Angular 17 | 2023년 11월 | 현재 |
| Angular 16 | 2023년 5월 | 안정판 |
