# Angular

Angular 是由 Google 开发的一款强大的前端框架，用于构建 Web 应用程序。

## 简介

Angular 是一个基于 TypeScript 的框架，提供了构建现代 Web 应用程序的完整解决方案，包括双向数据绑定、依赖注入和强大的 CLI 工具。

## 特性

- **组件化**：构建可重用的 UI 组件
- **TypeScript**：类型安全的开发
- **CLI**：强大的命令行工具
- **路由**：内置路由系统
- **表单**：模板驱动和响应式表单
- **HTTP**：用于 API 调用的 HTTP 客户端
- **测试**：内置测试支持

## 快速开始

### 安装

```bash
npm install -g @angular/cli
```

### 创建项目

```bash
ng new my-app
cd my-app
ng serve
```

访问 http://localhost:4200
