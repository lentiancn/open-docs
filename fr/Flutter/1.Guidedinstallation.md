# Guide d'installation de Flutter

Flutter est un kit de développement UI open source développé par Google pour créer des applications multiplateformes belles et rapides. Ce guide couvre les méthodes d'installation dans tous les environnements.

## Informations de version

| Version | Statut | Description |
|---------|--------|--------------|
| Flutter 3.x | ✅ Actuelle | Dernière version stable |
| Flutter 2.x | ⚠️ Maintenance | Mises à jour de sécurité |

> **Important** : Ce guide est principalement destiné à **Flutter 3.x**.

## Méthodes d'installation

### Méthode 1 : Installation sur Windows

#### 1. Télécharger Flutter SDK

1. Téléchargez Flutter SDK : https://docs.flutter.dev/get-started/install/windows
2. Extrayez dans le répertoire souhaité (ex: C:\src\flutter)

#### 2. Configurer les variables d'environnement

```powershell
# Ajouter à PATH
$env:PATH += ";C:\src\flutter\bin"
```

#### 3. Exécuter Flutter doctor

```powershell
flutter doctor
```

### Méthode 2 : Installation sur macOS

#### 1. Utiliser Homebrew (recommandé)

```bash
# Installer Flutter
brew install flutter

# Ou télécharger manuellement
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable
```

#### 2. Configurer les variables d'environnement

```bash
# ~/.bash_profile ou ~/.zshrc
export PATH="$PATH:`pwd`/flutter/bin"
```

#### 3. Exécuter Flutter doctor

```bash
flutter doctor
```

### Méthode 3 : Installation sur Linux

#### 1. Installer les dépendances

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa clang cmake ninja-build pkg-config libgtk-3-dev

# Fedora
sudo dnf install -y curl git unzip xz zip mesa-libGLU clang cmake ninja-build pkgconfig gtk3-devel
```

#### 2. Télécharger Flutter SDK

```bash
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable
export PATH="$PATH:~/development/flutter/bin"
```

#### 3. Exécuter Flutter doctor

```bash
flutter doctor
```

### Méthode 4 : Utiliser VS Code

1. Télécharger et installer VS Code
2. Ouvrir VS Code
3. Installer les extensions Flutter :
   - Rechercher "Flutter" et installer
   - Rechercher "Dart" et installer

### Méthode 5 : Utiliser Android Studio

1. Télécharger et installer Android Studio
2. Installer le plugin Flutter :
   - Ouvrir Android Studio
   - Settings → Plugins
   - Rechercher "Flutter" et installer

## Configuration Android

### Installer Android SDK

```bash
# Installer via les outils Flutter
flutter doctor --android-licenses
flutter config --android-sdk /path/to/android/sdk
```

### Configurer les variables d'environnement

```bash
# macOS/Linux
export ANDROID_SDK_ROOT=/Users/user/Library/Android/sdk
export PATH=$PATH:$ANDROID_SDK_ROOT/cmdline-tools/latest/bin:$ANDROID_SDK_ROOT/platform-tools
```

### Créer un émulateur

```bash
# Lister les appareils disponibles
flutter devices

# Créer un émulateur
flutter emulator --create --name my-emulator

# Lancer l'émulateur
flutter emulators --launch my-emulator
```

## Problèmes courants

### Erreur Flutter doctor ?

Assurez-vous que toutes les dépendances sont installées :

```bash
# Exécuter flutter doctor
flutter doctor -v
```

### Android SDK introuvable ?

Définir le chemin du SDK :

```bash
flutter config --android-sdk /path/to/android/sdk
```

### Échec de compilation ?

Nettoyer et recompiler :

```bash
flutter clean
flutter pub get
flutter build apk
```
