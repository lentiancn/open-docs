# Webpack 使用指南

本指南介紹如何使用 Webpack 建構現代 JavaScript 應用程式。

## 快速開始

### 建立專案

```bash
mkdir my-project
cd my-project
npm init -y
npm install webpack webpack-cli --save-dev
```

### 建立入口檔案

```javascript
// src/index.js
import './style.css';
import { add } from './math.js';

console.log('Hello Webpack!');
console.log(add(2, 3));
```

```javascript
// src/math.js
export function add(a, b) {
  return a + b;
}
```

```css
/* src/style.css */
body {
  background: #f0f0f0;
  font-family: sans-serif;
}
```

```html
<!-- src/index.html -->
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Webpack App</title>
</head>
<body>
  <h1>Hello Webpack!</h1>
  <script src="index.js"></script>
</body>
</html>
```

### 建立設定檔

```javascript
// webpack.config.js
const path = require('path');

module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
  },
  module: {
    rules: [
      {
        test: /\.css$/i,
        use: ['style-loader', 'css-loader'],
      },
    ],
  },
};
```

### 執行建置

```bash
npx webpack
```

## 核心概念

### 入口（Entry）

```javascript
module.exports = {
  // 單一入口
  entry: './src/index.js',
  
  // 多入口
  entry: {
    home: './src/home.js',
    about: './src/about.js',
  },
};
```

### 輸出（Output）

```javascript
module.exports = {
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
    clean: true,
    publicPath: '/',
  },
};
```

### 載入器（Loaders）

```javascript
module.exports = {
  module: {
    rules: [
      {
        test: /\.css$/,
        use: ['style-loader', 'css-loader'],
      },
      {
        test: /\.js$/,
        exclude: /node_modules/,
        use: 'babel-loader',
      },
    ],
  },
};
```

### 外掛（Plugins）

```javascript
const HtmlWebpackPlugin = require('html-webpack-plugin');
const MiniCssExtractPlugin = require('mini-css-extract-plugin');

module.exports = {
  plugins: [
    new HtmlWebpackPlugin({
      template: './src/index.html',
    }),
    new MiniCssExtractPlugin({
      filename: '[name].css',
    }),
  ],
};
```

### 模式（Mode）

```javascript
// 開發模式
module.exports = {
  mode: 'development',
  devtool: 'source-map',
};

// 生產模式
module.exports = {
  mode: 'production',
  devtool: false,
};
```

## 常用載入器

### 處理 CSS

```bash
npm install --save-dev style-loader css-loader
```

```javascript
{
  test: /\.css$/,
  use: ['style-loader', 'css-loader'],
}
```

### 處理 Sass

```bash
npm install --save-dev sass sass-loader
```

```javascript
{
  test: /\.scss$/,
  use: ['style-loader', 'css-loader', 'sass-loader'],
}
```

### 處理圖片

```javascript
{
  test: /\.(png|jpg|gif)$/,
  type: 'asset/resource',
}
```

### 處理字型

```javascript
{
  test: /\.(woff|woff2|eot|ttf|otf)$/,
  type: 'asset/resource',
}
```

## 開發伺服器

### 安裝

```bash
npm install --save-dev webpack-dev-server
```

### 設定

```javascript
module.exports = {
  devServer: {
    static: './dist',
    hot: true,
    port: 3000,
    open: true,
    compress: true,
  },
};
```

### 啟動

```bash
webpack serve --open
```

或新增腳本到 package.json：

```json
{
  "scripts": {
    "start": "webpack serve --mode development",
    "build": "webpack --mode production"
  }
}
```

## 程式碼分割

### 動態匯入

```javascript
// 路由延遲載入
const Home = () => import('./Home.vue');
const About = () => import('./About.vue');

// 條件載入
if (condition) {
  import('./module').then(module => {
    module.doSomething();
  });
}
```

### SplitChunks

```javascript
module.exports = {
  optimization: {
    splitChunks: {
      chunks: 'all',
      cacheGroups: {
        vendor: {
          test: /[\\/]node_modules[\\/]/,
          name: 'vendors',
          chunks: 'all',
        },
      },
    },
  },
};
```

## 環境設定

### 開發環境

```javascript
module.exports = {
  mode: 'development',
  devtool: 'inline-source-map',
  devServer: {
    static: './dist',
    hot: true,
  },
};
```

### 生產環境

```javascript
module.exports = {
  mode: 'production',
  devtool: 'source-map',
  optimization: {
    minimize: true,
  },
};
```

### 多環境設定

```javascript
const devConfig = require('./webpack.dev');
const prodConfig = require('./webpack.prod');

module.exports = (env, argv) => {
  return argv.mode === 'production' ? prodConfig : devConfig;
};
```

## 效能優化

### 1. 使用 production 模式

```javascript
module.exports = {
  mode: 'production',
};
```

### 2. 啟用 Tree Shaking

```javascript
// package.json
{
  "sideEffects": false
}
```

### 3. 程式碼分割

```javascript
optimization: {
  splitChunks: {
    chunks: 'all',
  },
}
```

### 4. 快取優化

```javascript
output: {
  filename: '[name].[contenthash].js',

```

### 5. 壓縮

```javascript
const TerserPlugin = require('terser-webpack-plugin');

optimization: {
  minimizer: [new TerserPlugin()],
}
```

## 常見問題

### 模組解析失敗？

檢查 resolve 設定：

```javascript
resolve: {
  extensions: ['.js', '.json'],
  alias: {
    '@': path.resolve(__dirname, 'src'),
  },
}
```

### 建置太慢？

- 使用 cache
- 排除 node_modules
- 使用 thread-loader

```javascript
{
  test: /\.js$/,
  exclude: /node_modules/,
  use: {
    loader: 'thread-loader',
    options: {
      workers: 2,
    },
  },
}
```
