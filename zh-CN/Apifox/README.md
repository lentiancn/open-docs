# Apifox

Apifox 是国内领先的 API 协作平台，一体化解决 API 设计、Mock、测试、文档等问题。它集成了 API 设计、API 文档管理、API 调试、API Mock、API 自动化测试等功能，是国内最受欢迎的 API 管理工具之一。

## 快速开始

### 下载安装

访问 https://apifox.com/ 下载对应系统的安装包。

### 创建项目

1. 启动 Apifox，登录账号
2. 点击"新建项目"
3. 输入项目名称

### 创建接口

1. 在项目中点击"新建接口"
2. 填写接口信息
3. 保存

### 调试接口

1. 打开接口
2. 点击"发送"
3. 查看响应

## 核心功能

- **API 设计**：可视化定义 API 接口
- **API 文档**：自动生成，支持多种格式导出
- **API 调试**：强大的在线调试功能
- **API Mock**：智能 Mock，自动生成模拟数据
- **自动化测试**：无需编码即可完成 API 测试
- **团队协作**：实时同步，权限管理
