# Руководство по установке OpenClaw

OpenClaw поддерживает несколько способов установки, независимо от того, используете ли вы macOS, Linux или Windows. Это руководство проведёт вас через каждый вариант установки и их предварительные требования.

---

## Системные требования

Перед установкой убедитесь, что ваша система соответствует этим требованиям:

- **Node.js**: 22.0.0 или выше
- **Операционная система**:
  - macOS 10.15 (Catalina) или выше
  - Linux (Ubuntu 20.04+, Debian 11+, Fedora 36+ и т.д.)
  - Windows 10/11 (настоятельно рекомендуется WSL2)
- **Сеть**: Доступ к реестру npm и сервисам OpenClaw
- **Хранилище**: Не менее 500 МБ свободного места

Проверьте версию Node.js:

```bash
node --version
```

Если версия ниже 22.0.0, пожалуйста, сначала обновите Node.js. Мы рекомендуем использовать nvm (Node Version Manager) для управления несколькими версиями Node.js.

---

## Способы установки

### Способ 1: Официальный скрипт установки (рекомендуемый)

Это самый простой и быстрый способ. Официальный скрипт установки автоматически определяет вашу операционную систему и выполняет всю настройку.

**macOS / Linux:**

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

**Windows (PowerShell):**

```powershell
iwr -useb https://openclaw.ai/install.ps1 | iex
```

### Способ 2: Глобальная установка через npm

Если у вас уже установлены Node.js и npm:

```bash
npm install -g openclaw@latest
```

Вы также можете указать конкретную версию:

```bash
npm install -g openclaw@v2026.2.28
```

Доступные теги версий:
- `latest`: Последняя стабильная версия
- `beta`: Бета-версия
- `dev`: Версия для разработки

### Способ 3: Установка через pnpm

OpenClaw рекомендует использовать pnpm:

```bash
pnpm add -g openclaw@latest
```

### Способ 4: Установка через Bun

Если вы используете Bun как среду выполнения JavaScript:

```bash
bun add -g openclaw@latest
```

### Способ 5: Сборка из исходного кода

```bash
# Клонировать репозиторий
git clone https://github.com/openclaw/openclaw.git
cd openclaw

# Установить зависимости
pnpm install

# Собрать проект
pnpm ui:build
pnpm build

# Инициализировать и установить сервис
pnpm openclaw onboard --install-daemon

# Режим разработки (автоперезагрузка)
pnpm gateway:watch
```

## Настройка после установки

### Мастер настройки

После установки запустите мастер:

```bash
openclaw onboard --install-daemon
```

### Проверка установки

```bash
# Проверить версию
openclaw --version

# Проверить статус Gateway
openclaw gateway status

# Открыть интерфейс управления
openclaw dashboard
```

## Частые проблемы при установке

### Проблема 1: Ошибки разрешений npm

Используйте nvm:

```bash
# Установить nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Использовать nvm для установки Node.js
nvm install 22
nvm use 22

# Затем установить OpenClaw
npm install -g openclaw@latest
```

### Проблема 2: Gateway не запускается

Проверьте, используется ли порт:

```bash
lsof -i :18789
```

### Проблема 3: Устаревшая версия

Обновите:

```bash
npm update -g openclaw@latest
```

## Обновление OpenClaw

```bash
npm update -g openclaw@latest
openclaw doctor
```

---

## Удаление OpenClaw

```bash
# Остановить сервис
openclaw gateway stop

# Удалить глобальный пакет
npm uninstall -g openclaw

# Удалить каталог конфигурации (необязательно)
rm -rf ~/.openclaw
```
