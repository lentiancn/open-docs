# Element UI 安裝指南

## 環境要求

- Vue 2.0.0+
- Node.js 8.0.0+
- 現代瀏覽器

## 安裝方式

使用 npm 安裝：

```bash
npm install element-ui --save
```

## 完整引入

在 main.js 中引入：

```javascript
import Vue from 'vue'
import Element from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(Element)
```

## 按需導入

首先安裝 babel-plugin-component：

```bash
npm install babel-plugin-component -D
```

配置 .babelrc：

```json
{
  "plugins": [
    ["component", {
      "libraryName": "element-ui",
      "styleLibraryName": "theme-chalk"
    }]
  ]
}
```

## 全局配置

```javascript
Vue.use(Element, {
  size: 'small',
  zIndex: 2000
})
```

更多信息請訪問 Element UI 官網。
