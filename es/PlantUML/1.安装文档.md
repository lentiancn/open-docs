# Guía de Instalación de PlantUML

PlantUML es una herramienta de código abierto para crear diagramas UML rápidamente. Esta guía cubre los métodos de instalación en varios entornos.

## Información de Versión

| Versión | Estado | Descripción |
|------|------|------|
| PlantUML | ✅ Actual | Última versión estable |

## Métodos de Instalación

### Método Uno: Instalación en Windows

#### 1. Descargar el Archivo JAR

1. Descargar PlantUML JAR: https://sourceforge.net/projects/plantuml/files/plantuml.jar/download
2. Guardar en un directorio local

#### 2. Instalar Graphviz (Opcional pero Recomendado)

```powershell
# Usar Chocolatey
choco install graphviz
```

### Método Dos: Instalación en macOS

```bash
# Usar Homebrew
brew install plantuml

# O instalar Graphviz
brew install graphviz
```

### Método Tres: Instalación en Linux

```bash
# Ubuntu/Debian
sudo apt-get install -y plantuml graphviz

# CentOS/RHEL
sudo yum install -y plantuml graphviz
```

### Método Cuatro: Como Plugin de VS Code

1. Abrir VS Code
2. Buscar "PlantUML"
3. Instalar el plugin "PlantUML"
4. Instalar Graphviz

### Método Cinco: Como Plugin de IntelliJ

1. Abrir IntelliJ IDEA
2. Settings → Plugins
3. Buscar "PlantUML"
4. Instalar el plugin

### Método Seis: Usar Docker

```bash
# Ejecutar servidor PlantUML
docker run -d -p 8080:8080 plantuml/plantuml-server:jetty

# Acceder a http://localhost:8080
```

## Configuración del Entorno

### Requisitos de Java

```bash
# Verificar Java
java -version

# Se requiere Java 8 o superior
```

### Configuración de Graphviz

```bash
# Configurar variable de entorno (Linux/macOS)
export GRAPHVIZ_DOT=/usr/bin/dot

# Windows agregar a PATH
# C:\Program Files\Graphviz\bin
```

## Preguntas Frecuentes

### ¿Las imágenes no se muestran?

Asegúrese de tener Graphviz instalado:

```bash
dot -V
```

### ¿Los caracteres chinos aparecen corruptos?

Agregue al diagrama:

```plantuml
skinparam defaultFontName SimSun
```
