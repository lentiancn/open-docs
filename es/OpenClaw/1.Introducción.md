# OpenClaw 🦞

OpenClaw es una herramienta que ejecutas en tu propio dispositivo y conecta apps de mensajería con asistentes IA. WhatsApp, Telegram, Discord, iMessage... todo直通 a servicios de IA.

## Para quién?

Si quieres tu propio asistente IA sin confiar tus datos a terceros, OpenClaw es buena opción. Ideal para desarrolladores que usan línea de comandos.

## Características

- **Tus datos, tu máquina**: todo queda en tu hardware
- **Múltiples plataformas**: un proceso maneja varias apps de chat
- **Para programación**: soporta herramientas, historial, memoria, multi-asistente
- **Código abierto**: MIT license

## Qué necesitas

- Node.js 22+
- API key de tu proveedor IA
- Algunos minutos para configurar

## Inicio rápido

```bash
npm install -g openclaw@latest
openclaw onboard --install-daemon
openclaw gateway --port 18789
```

Abre http://127.0.0.1:18789/ en tu navegador.
