# Introduction

Bootstrap is the world's most popular open-source front-end framework for HTML, CSS, and JavaScript, originally developed by designers Mark Otto and Jacob Thornton at Twitter. It comes packed with ready-to-use components and powerful JavaScript plugins that help developers build responsive web applications in a snap.

The core philosophy behind Bootstrap is "Mobile First" — it was designed from the ground up to work beautifully across all devices. With Bootstrap, you can easily create websites that look great on desktops, tablets, and phones alike.

## A Quick History

Bootstrap started out called Twitter Blueprint and was open-sourced in 2011. Bootstrap 2 came along in 2012 with responsive layout support. In 2013, Bootstrap 3 launched with a complete redesign using a flat look. Bootstrap 4 arrived in 2015, switching to Sass for CSS preprocessing. Then Bootstrap 5 dropped in 2021, dropping jQuery dependency and fully embracing vanilla JavaScript.

## What Makes Bootstrap Tick

### Responsive Layouts

Bootstrap's grid system is its bread and butter. With a 12-column grid layout, you can easily create page designs that adapt to different screen sizes. Bootstrap 5 gives you six breakpoints — xs, sm, md, lg, xl, and xxl — each targeting different device widths.

### Ready-to-Use Components

Bootstrap ships with a ton of UI components out of the box: navbars, cards, buttons, forms, modals, carousels, dropdowns, and more. They're all thoughtfully designed with good looks and smooth interactions.

### Powerful JavaScript Plugins

Bootstrap includes over 20 JavaScript plugins — modals, carousels, tooltips, popovers, accordions, and so on. They're all optional, so you can pick and choose what you need to keep your project light.

### Built on Sass

Bootstrap 5 uses Sass as its CSS preprocessor, with tons of Sass variables and mixins built in. You can easily customize theme colors, font sizes, and other styles to match your project.

### Custom Themes

Bootstrap's theming system makes it easy to create your own look and feel. The official Bootstrap Themes marketplace has plenty of polished themes to choose from too.

### Accessibility Out of the Box

Bootstrap components come with accessibility features built in, including keyboard navigation and screen reader compatibility, helping you build more inclusive websites.

## What's New in Bootstrap 5

Bootstrap 5 brought some big changes from Bootstrap 4:

- Dropped jQuery dependency, now uses vanilla JavaScript
- Improved grid system with a new XXL breakpoint
- New form controls that replace the old form validation
- Better custom property support
- Switched icon library from Font Awesome to Bootstrap Icons
- RTL (right-to-left) language support

## When to Use Bootstrap

- Rapid prototyping
- Internal business management systems
- E-commerce websites
- Blogs and content sites
- Admin dashboards
- Any responsive website

As one of the most popular CSS frameworks in the world, Bootstrap has a huge community and plenty of learning resources. It's a must-have skill for any front-end developer.
