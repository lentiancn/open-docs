# 安裝指南

本文檔介紹 Angular 的安裝和環境配置，幫助您快速搭建開發環境。

## 環境要求

在安裝 Angular 之前，請確保滿足以下環境要求：

- **Node.js**：建議 18.10.0 或更高版本
- **npm**：Node.js 內建，版本 6.x 或更高
- **作業系統**：Windows、macOS、Linux

## 安裝 Angular CLI

Angular CLI 是 Angular 官方的命令列工具，用於建立和管理 Angular 專案。

### 全域安裝

```bash
npm install -g @angular/cli
```

### 驗證安裝

```bash
ng version
```

如果安裝成功，會顯示 Angular CLI 版本資訊。

### 安裝指定版本

如果需要安裝特定版本：

```bash
npm install -g @angular/cli@17
```

## 建立新專案

### 使用 CLI 建立

```bash
ng new my-app
```

這會建立一個名為 my-app 的新專案，並自動安裝相依套件。

### 建立時指定選項

```bash
# 建立並跳過安裝
ng new my-app --skip-install

# 使用 SCSS 樣式
ng new my-app --style=scss

# 啟用路由
ng new my-app --routing

# 建立獨立元件（Angular 14+）
ng new my-app --standalone
```

## 開發環境配置

### 使用國內鏡像

由於網路因素，建議配置國內鏡像加速：

```bash
# 設定 npm 鏡像
npm config set registry https://registry.npmmirror.com

# 或使用 nrm 工具管理鏡像
npm install -g nrm
nrm use taobao
```

### 安裝 Node.js

**使用 nvm 安裝（建議）：**

```bash
# 安裝 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 安裝 Node.js LTS 版本
nvm install --lts
nvm use --lts
```

**Windows 使用 nvm-windows：**

從 https://github.com/coreybutler/nvm-windows 下載安裝包。

## 啟動開發伺服器

### 進入專案目錄

```bash
cd my-app
```

### 啟動開發伺服器

```bash
ng serve
```

預設存取位址：http://localhost:4200

### 即時重載

Angular CLI 支援即時重載，修改程式碼後會自動重新整理瀏覽器。

## 建構生產版本

### 建構命令

```bash
ng build
```

建構產物會輸出到 dist 目錄。

### 生產環境優化

```bash
ng build --configuration=production
```

這會啟用程式碼壓縮、tree-shaking 等優化。

## 常用 CLI 命令

| 命令 | 說明 |
|------|------|
| ng generate component xxx | 建立元件 |
| ng generate service xxx | 建立服務 |
| ng generate module xxx | 建立模組 |
| ng generate directive xxx | 建立指令 |
| ng generate pipe xxx | 建立管道 |
| ng generate guard xxx | 建立路由守衛 |

## 開發工具

### VS Code 擴充

- Angular Language Service：提供模板型別檢查和自動完成
- Angular Snippets：提供常用程式碼片段
- Prettier：程式碼格式化

### 偵錯

Angular 提供 Angular DevTools 瀏覽器擴充功能，用於偵錯和效能分析。

## 解除安裝 Angular CLI

如果需要解除安裝：

```bash
npm uninstall -g @angular/cli
```
