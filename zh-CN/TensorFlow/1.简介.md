# TensorFlow - 简介

TensorFlow 是 Google 开发的开源机器学习框架，广泛用于深度学习研究和应用。

## 核心特性

- **多平台支持**：Linux、macOS、Windows、移动端
- **多语言**：Python、JavaScript、C++、Java
- **TF2.x**：Eager Execution、keras 集成
- **分布式训练**：多 GPU、多节点
- **TensorBoard**：可视化工具
- **TF Lite**：移动端部署
- **TF.js**：浏览器端运行

## 主要模块

- **tf.keras**：高级 API
- **tf.data**：数据管道
- **tf.function**：图编译
- **tf.distribute**：分布式策略

## 应用场景

- 图像分类
- 自然语言处理
- 推荐系统
- 时间序列预测
