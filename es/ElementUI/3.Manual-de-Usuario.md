# Element UI Manual de Usuario

## Componentes básicos

### Button

```html
<el-button>Predeterminado</el-button>
<el-button type="primary">Primario</el-button>
<el-button type="success">Éxito</el-button>
<el-button type="warning">Advertencia</el-button>
<el-button type="danger">Peligro</el-button>
```

### Layout

```html
<el-row>
  <el-col :span="12"><div></div></el-col>
  <el-col :span="12"><div></div></el-col>
</el-row>
```

## Componentes de formulario

### Input

```html
<el-input v-model="input" placeholder="Ingrese"></el-input>
```

### Select

```html
<el-select v-model="value" placeholder="Seleccione">
  <el-option label="Opción 1" value="1"></el-option>
</el-select>
```

### Form

```html
<el-form :model="form" label-width="100px">
  <el-form-item label="Nombre">
    <el-input v-model="form.name"></el-input>
  </el-form-item>
</el-form>
```

## Componentes de datos

### Table

```html
<el-table :data="tableData">
  <el-table-column prop="date" label="Fecha"></el-table-column>
  <el-table-column prop="name" label="Nombre"></el-table-column>
</el-table>
```

### Pagination

```html
<el-pagination :total="1000" layout="prev, pager, next"></el-pagination>
```

## Notificaciones

### Message

```javascript
this.$message({ message: 'Éxito', type: 'success' })
```

### Alert

```html
<el-alert title="Éxito" type="success"></el-alert>
```

## Navegación

### Menu

```html
<el-menu>
  <el-menu-item index="1">Nav 1</el-menu-item>
</el-menu>
```

### Tabs

```html
<el-tabs v-model="activeName">
  <el-tab-pane label="Pestaña 1" name="first">Contenido</el-tab-pane>
</el-tabs>
```

## Otros

### Dialog

```html
<el-button @click="visible = true">Abrir</el-button>
<el-dialog :visible.sync="visible" title="Título">
  <span>Contenido</span>
</el-dialog>
```
