## 柵格系統

### 基本原理

Bootstrap 的柵格系統使用 Flexbox 建構，最多支援 12 列。

### 容器

```html
<!-- 固定寬度容器 -->
<div class="container">
  內容
</div>

<!-- 100% 寬度容器 -->
<div class="container-fluid">
  內容
</div>

<!-- 響應式斷點容器 (Bootstrap 5.2+) -->
<div class="container-sm">100% 寬 < 576px</div>
<div class="container-md">100% 寬 >= 768px</div>
<div class="container-lg">100% 寬 >= 992px</div>
<div class="container-xl">100% 寬 >= 1200px</div>
<div class="container-xxl">100% 寬 >= 1400px</div>
```

### 列

```html
<div class="container">
  <div class="row">
    <div class="col">列</div>
    <div class="col">列</div>
    <div class="col">列</div>
  </div>
</div>
```

### 列寬設定

```html
<div class="row">
  <div class="col-4">4 列</div>
  <div class="col-8">8 列</div>
</div>
```

### 響應式斷點

```html
<!-- 響應式列寬 -->
<div class="row">
  <div class="col-12 col-md-6 col-lg-4">
    在手機上全寬，平板 6 列，桌面 4 列
  </div>
</div>
```

### 列偏移

```html
<!-- 使用 offset -->
<div class="row">
  <div class="col-md-4">4 列</div>
  <div class="col-md-4 offset-md-4">偏移 4 列</div>
</div>

<!-- 使用 margin 自動偏移 -->
<div class="row">
  <div class="col-md-3 ms-auto">右側偏移</div>
</div>
```

### 柵格嵌套

```html
<div class="row">
  <div class="col-sm-8">
    外層 8 列
    <div class="row">
      <div class="col-6">內層 6 列</div>
      <div class="col-6">內層 6 列</div>
    </div>
  </div>
</div>
```

### Flex 佈局

```html
<!-- 水平對齊 -->
<div class="row justify-content-start">...</div>
<div class="row justify-content-center">...</div>
<div class="row justify-content-end">...</div>
<div class="row justify-content-around">...</div>
<div class="row justify-content-between">...</div>
<div class="row justify-content-evenly">...</div>

<!-- 垂直對齊 -->
<div class="row align-items-start">...</div>
<div class="row align-items-center">...</div>
<div class="row align-items-end">...</div>
```

## 排版

### 標題

```html
<h1>標題 1</h1>
<h2>標題 2</h1>
<h3>標題 3</h1>
<h4>標題 4</h1>
<h5>標題 5</h1>
<h6>標題 6</h1>

<!-- 展示標題 -->
<h1 class="display-1">展示 1</h1>
<h1 class="display-2">展示 2</h1>
<h1 class="display-3">展示 3</h1>
<h1 class="display-4">展示 4</h1>
<h1 class="display-5">展示 5</h1>
<h1 class="display-6">展示 6</h1>
```

### 段落

```html
<!-- 一般段落 -->
<p>這是一個段落。</p>

<!-- 突出段落 -->
<p class="lead">這是一個突出的段落。</p>

<!-- 文字對齊 -->
<p class="text-start">左對齊</p>
<p class="text-center">置中</p>
<p class="text-end">右對齊</p>
<p class="text-justify">兩端對齊</p>

<!-- 響應式對齊 -->
<p class="text-sm-start">小螢幕左對齊</p>
<p class="text-md-center">中等螢幕置中</p>
<p class="text-lg-end">大螢幕右對齊</p>
```

### 文字樣式

```html
<!-- 字體粗細 -->
<p class="fw-bold">粗體</p>
<p class="fw-bolder">更粗</p>
<p class="fw-normal">正常</p>
<p class="fw-light">細體</p>
<p class="fw-lighter">更細</p>

<!-- 字體斜體 -->
<p class="fst-italic">斜體</p>
<p class="fst-normal">正常字體</p>

<!-- 文字轉換 -->
<p class="text-lowercase">全部小寫</p>
<p class="text-uppercase">全部大寫</p>
<p class="text-capitalize">首字母大寫</p>
```

### 清單

```html
<!-- 無樣式清單 -->

  <li>項目 1</li>
  <li>項目 2</li>
</ul>

<!-- 行內清單 -->

  <li class="list-inline-item">項目 1</li>
  <li class="list-inline-item">項目 2</li>
</ul>
```

## 最佳實踐

### 1. 使用語義化 HTML

```html
<!-- 好 -->
<nav class="navbar">...</nav>
<main class="container">...</main>
<article class="card">...</article>

<!-- 避免 -->
<div class="navbar">...</div>
<div class="container">...</div>
<div class="card">...</div>
```

### 2. 保持代碼簡潔

```html
<!-- 好 -->
<div class="container">
  <div class="row">
    <div class="col-md-6">...</div>
  </div>
</div>

<!-- 避免過度嵌套 -->
<div class="container">
  <div class="row">
    <div class="col-md-6">
      <div class="card">
        <div class="card-body">...</div>
      </div>
    </div>
  </div>
</div>
```

### 3. 響應式設計
