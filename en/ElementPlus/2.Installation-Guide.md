# Element Plus Installation Guide

Multiple installation methods for Element Plus.

## Environment Requirements

### Hardware Requirements

- CPU: 2 cores and above
- Memory: 4GB and above

### Software Requirements

- Node.js: 18.0+
- npm: 9.0+ / yarn: 1.22+ / pnpm: 8.0+
- Vue: 3.2+
- Vite: 4.0+ (recommended)

## Package Manager Installation (Recommended)

### npm

```bash
npm install element-plus
```

### yarn

```bash
yarn add element-plus
```

### pnpm

```bash
pnpm add element-plus
```

## Full Import

### main.ts Configuration

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')
```

## On-Demand Import (Recommended)

### Auto Import (Recommended)

**1. Install Dependencies**

```bash
npm install -D unplugin-vue-components unplugin-auto-import
```

**2. Configure Vite**

```typescript
// vite.config.ts
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

**3. Use Components**

```vue
<template>
  <el-button type="primary">Hello</el-button>
</template>
```

### Manual On-Demand Import

```typescript
import { ElButton, ElInput } from 'element-plus'

export default {
  components: {
    ElButton,
    ElInput,
  },
}
```

## Vue CLI Installation

### Install

```bash
npm install -D sass sass-loader
npm install element-plus
```

### Configure

```javascript
// vue.config.js
module.exports = {
  configureWebpack: {
    plugins: [
      AutoImport({
        resolvers: [ElementPlusResolver()],
      }),
      Components({
        resolvers: [ElementPlusResolver()],
      }),
    ],
  },
}
```

## CDN Import

### HTML

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Element Plus CDN</title>
  <link rel="stylesheet" href="//unpkg.com/element-plus/dist/index.css" />
  <script src="//unpkg.com/vue@3"></script>
  <script src="//unpkg.com/element-plus"></script>
</head>
<body>
  <div id="app">
    <el-button type="primary">Primary Button</el-button>
  </div>

  <script>
    const { createApp, ref } = Vue
    const App = {
      setup() {
        const visible = ref(false)
        return { visible }
      },
      template: `
        <el-button @click="visible = true">Open Dialog</el-button>
        <el-dialog v-model="visible" title="Title">Content</el-dialog>
      `
    }
    const app = createApp(App)
    app.use(ElementPlus)
    app.mount('#app')
  </script>
</body>
</html>
```

## Common Issues

### Icons Not Showing?

Install icon package:

```bash
npm install @element-plus/icons-vue
```

Register:

```typescript
import * as icons from '@element-plus/icons-vue'

for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```

### Style Conflicts?

Use CSS Modules or scoped:

```vue
<style scoped>
:deep(.el-button) {
  margin: 10px;
}
</style>
```

### TypeScript Type Errors?

Ensure type definitions are installed:

```bash
npm install -D @types/node
```

Configure in `tsconfig.json`:

```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

### Internationalization

```typescript
import en from 'element-plus/es/locale/lang/en'

app.use(ElementPlus, { locale: en })
```
