# OpenClaw 安装文档

在所有主流平台上安装 OpenClaw 的完整指南。

---

## 什么是 OpenClaw？

OpenClaw 是一个自托管网关，将聊天应用（WhatsApp、Telegram、Discord、iMessage）连接到 AI 编码代理。

**要求：**
- Node.js 22+
- API 密钥（推荐 Anthropic）
- 5 分钟

---

## npm 安装（推荐）

### 快速安装

```bash
npm install -g openclaw@latest
```

### 验证安装

```bash
openclaw --version
openclaw --help
```

---

## 初始化和设置

### 初始化 OpenClaw

```bash
# 初始化并安装服务
openclaw onboard --install-daemon
```

### 启动网关

```bash
# 启动网关
openclaw gateway --port 18789
```

```bash
# 使用 npm (PowerShell 或 CMD)
npm install -g openclaw@latest
```

---

## Docker 安装

### 拉取镜像

```bash
docker pull openclaw/openclaw:latest
```

### 运行容器

```bash
docker run -d \
  --name openclaw \
  -p 18789:18789 \
  -v ~/.openclaw:/root/.openclaw \
  openclaw/openclaw:latest
```

---

## 配置

### 基本配置

配置文件位置：`~/.openclaw/openclaw.json`

```json
{
  "channels": {
    "whatsapp": {
      "allowFrom": ["+15555550123"]
    }
  }
}
```

### 环境变量

```bash
# 设置 API 密钥
export ANTHROPIC_API_KEY=sk-ant-your-key
```

---

## 更新 OpenClaw

```bash
# 更新到最新版本
npm update -g openclaw

# 或重新安装
npm install -g openclaw@latest
```
