# Element Plus Einführung

Element Plus ist eine auf Vue 3 basierte UI-Komponentenbibliothek.

## Funktionen

- Vue 3 Komponenten
- 70+ Komponenten
- TypeScript
- Import nach Bedarf
- Internationalisierung
- Themes

## Szenarien

- Unternehmensverwaltungssysteme
- E-Commerce-Plattformen

Weitere Informationen finden Sie auf der [offiziellen Website](https://element-plus.org/).
