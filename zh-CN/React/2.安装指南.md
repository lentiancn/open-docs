# React 安装指南

本指南将帮助你搭建 React 开发环境。

## 环境要求

| 软件 | 要求 |
|------|------|
| Node.js | 18.0+ |
| npm | 9.0+ |
| 代码编辑器 | VS Code（推荐） |

## 安装步骤

### 方式一：使用 Vite（推荐）

Vite 是新一代构建工具，创建项目快速简单：

```bash
# 创建项目
npm create vite@latest my-react-app -- --template react

# 进入目录
cd my-react-app

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 方式二：使用 Create React App

传统的脚手架工具：

```bash
# 创建项目
npx create-react-app my-react-app

# 进入目录
cd my-react-app

# 启动
npm start
```

### 方式三：CDN 引入

适合简单页面：

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>React 入门</title>
  <script src="https://unpkg.com/react@18/umd/react.development.js"></script>
  <script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
  <script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>
</head>
<body>
  <div id="root"></div>

  <script type="text/babel">
    function App() {
      return <h1>Hello React!</h1>;
    }
    ReactDOM.render(<App />, document.getElementById('root'));
  </script>
</body>
</html>
```

## 配置 Tailwind CSS

### 安装

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

### 配置

```javascript
// tailwind.config.js
export default {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
```

### 使用

```css
/* index.css */
@tailwind base;
@tailwind components;
@tailwind utilities;
```

## 配置 ESLint

### 安装

```bash
npm install -D eslint eslint-plugin-react eslint-plugin-react-hooks eslint-plugin-react-refresh
```

### 配置

```javascript
// .eslintrc.cjs
module.exports = {
  root: true,
  env: { browser: true, es2020: true },
  extends: [
    'eslint:recommended',
    'plugin:react/recommended',
    'plugin:react/jsx-runtime',
    'plugin:react-hooks/recommended',
  ],
  parserOptions: { ecmaVersion: 'latest', sourceType: 'module' },
  settings: { react: { version: '18.2' } },
  plugins: ['react-refresh'],
  rules: {
    'react-refresh/only-export-components': [
      'warn',
      { allowConstantExport: true },
    ],
  },
}
```

## 常见问题

### Q1: 启动失败

**解决方案：**
```bash
# 删除 node_modules 和 package-lock.json
rm -rf node_modules package-lock.json

# 重新安装
npm install
```

### Q2: 端口被占用

**解决方案：**
```javascript
// vite.config.js
export default {
  server: {
    port: 3000
  }
}
```

### Q3: 无法解析 JSX

**解决方案：**
确保安装了 @vitejs/plugin-react：

```bash
npm install -D @vitejs/plugin-react
```

```javascript
// vite.config.js
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [react()],
})
```
