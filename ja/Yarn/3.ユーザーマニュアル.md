# Yarn ユーザーマニュアル

## 日常的な使用

### プロジェクト初期化

```bash
yarn init
```

### 依存関係インストール

```bash
yarn install
```

### 依存関係追加

```bash
yarn add lodash
yarn add -D typescript
```

### 依存関係削除

```bash
yarn remove lodash
```

### 依存関係更新

```bash
yarn up lodash
yarn up --all
```

### スクリプト実行

```bash
yarn build
yarn dev
yarn test
```

## 一般的なコマンド

| コマンド | 説明 |
|----------|------|
| `yarn init` | プロジェクト初期化 |
| `yarn install` | 依存関係インストール |
| `yarn add` | 依存関係追加 |
| `yarn remove` | 依存関係削除 |
| `yarn up` | 依存関係更新 |
| `yarn list` | 依存関係一覧 |
