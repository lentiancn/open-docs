# React Installation Guide

Comprehensive guide for installing React on all major platforms.

## What is React?

React is a JavaScript library for building user interfaces. It is maintained by Meta (formerly Facebook).

## Create React App

### Quick Start

```bash
# Create a new React app
npx create-react-app my-app

# Navigate to the app
cd my-app

# Start development server
npm start
```

### Using npm

```bash
npx create-react-app my-app
cd my-app
npm start
```

### Using yarn

```bash
npx create-react-app my-app
cd my-app
yarn start
```

## Manual Installation

### Using npm

```bash
# Create project directory
mkdir my-react-app
cd my-react-app

# Initialize package.json
npm init -y

# Install React and ReactDOM
npm install react react-dom

# Install development dependencies
npm install --save-dev vite @vitejs/plugin-react
```

### Create vite.config.js

```javascript
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [react()],
})
```

### Create index.html

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <title>React App</title>
  </head>
  <body>
    <div id="root"></div>
    <script type="module" src="/src/main.jsx"></script>
  </body>
</html>
```

### Create src/main.jsx

```jsx
import React from 'react'
import ReactDOM from 'react-dom/client'
import App from './App'

ReactDOM.createRoot(document.getElementById('root')).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
)
```

## CDN

### Using CDN

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>React CDN</title>
  <script crossorigin src="https://unpkg.com/react@18/umd/react.development.js"></script>
  <script crossorigin src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
  <script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>
</head>
<body>
  <div id="root"></div>

  <script type="text/babel">
    function App() {
      return <h1>Hello React</h1>;
    }
    ReactDOM.createRoot(document.getElementById('root')).render(<App />);
  </script>
</body>
</html>
```

### CDN Providers

| Provider | React | ReactDOM |
|----------|-------|----------|
| unpkg | https://unpkg.com/react@18/umd/react.development.js | https://unpkg.com/react-dom@18/umd/react-dom.development.js |
| jsDelivr | https://cdn.jsdelivr.net/npm/react@18/umd/react.development.js | https://cdn.jsdelivr.net/npm/react-dom@18/umd/react-dom.development.js |

## Version

### Install Specific Version

```bash
npm install react@18.2.0 react-dom@18.2.0
npm install react@17.0.2 react-dom@17.0.2
```

### Check Version

```javascript
console.log(React.version); // '18.2.0'
```

## Dependencies

### React Router

```bash
npm install react-router-dom
```

### State Management

```bash
# Redux
npm install @reduxjs/toolkit react-redux

# Zustand
npm install zustand

# MobX
npm install mobx mobx-react-lite
```

### UI Libraries

```bash
# Material UI
npm install @mui/material @emotion/react @emotion/styled

# Ant Design
npm install antd

# Tailwind CSS
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```
