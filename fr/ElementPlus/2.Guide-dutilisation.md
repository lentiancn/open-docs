# Element Plus Guide d'utilisation

## Démarrage rapide

```vue
<template>
  <el-button type="primary">Bouton</el-button>
  <el-input v-model="input" />
</template>

<script setup>
import { ref } from 'vue'
const input = ref('')
</script>
```

## Composants

### Boutons

```vue
<el-button>Par défaut</el-button>
<el-button type="primary">Primary</el-button>
```

### Formulaire

```vue
<el-form :model="form">
  <el-form-item label="Nom">
    <el-input v-model="form.nom" />
  </el-form-item>
</el-form>
```

### Table

```vue
<el-table :data="data">
  <el-table-column prop="nom" label="Nom" />
</el-table>
```
