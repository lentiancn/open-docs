# Element Plus FAQ

Частые вопросы.

## Установка

### Q: Ошибка установки?

**Решения:**
1. Проверить версию Node.js
```bash
node --version  # Требуется 18.0+
```
2. Очистить кэш
```bash
npm cache clean --force
```

### Q: Ошибки TypeScript?

**Решения:**
Установить типы:
```bash
npm install -D @types/node
```

## Использование

### Q: Компонент не отображается?

**Решения:**
1. Проверить импорт
2. Проверить версию Vue (3.2+)

### Q: Стили не применяются?

**Решения:**
1. Проверить импорт CSS
2. Использовать `:deep()`

### Q: Иконки не отображаются?

**Решения:**
Установить пакет иконок:
```bash
npm install @element-plus/icons-vue
```

Зарегистрировать:
```typescript
import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```
