# Flutter 簡介

Flutter 是 Google 推出的開源 UI 工具包。

## 什麼是 Flutter？

Flutter 是一個用於構建跨平台應用的 UI 工具包，可以讓開發者使用一套代碼同時構建 iOS、Android、Web 和桌面應用。Flutter 使用 Dart 語言編寫，以其豐富的組件庫和高性能渲染而聞名。

## 主要特性

- **跨平台開發**：一套代碼同時支持 iOS、Android、Web、桌面
- **熱重載**：修改代碼即時預覽，無需重啟應用
- **豐富組件**：Material Design 和 Cupertino 風格組件
- **高性能**：使用 Skia 圖形引擎，原生性能體驗
- ** expressive UI**：靈活的聲明式 UI 構建方式
- **開源免費**：完全開源，免費使用

## 核心技術

### Dart 語言

Flutter 使用 Dart 語言開發，Dart 是一種面向對象的編程語言：

- **強類型**：靜態類型檢查
- **JIT & AOT**：支持 JIT 即時編譯和 AOT 預編譯
- **垃圾回收**：自動內存管理
- **協程**：異步編程支持

### Widget 系統

Flutter 的核心是 Widget，所有 UI 元素都是 Widget：

- **StatelessWidget**：無狀態組件
- **StatefulWidget**：有狀態組件
- **InheritedWidget**：數據傳遞
- **RenderObjectWidget**：渲染對象

## 應用場景

- **移動應用**：iOS 和 Android 原生體驗
- **Web 應用**：響應式 Web 應用
- **桌面應用**：Windows、macOS、Linux 桌面應用
- **嵌入式**：IoT 設備界面
