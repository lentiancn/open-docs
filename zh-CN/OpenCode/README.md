# OpenCode

轻量级代码编辑器。

## 简介

OpenCode 是一个轻量级，功能强大的代码编辑器，专为现代开发工作流设计。

### 主要特性

- 轻量级 - 快速启动，占用资源少
- 语法高亮 - 支持多种编程语言
- 插件系统 - 丰富的插件生态
- 跨平台 - Windows、macOS、Linux 支持
