# Oracle Datenbank Installationsanleitung

## Voraussetzungen

- **Betriebssystem**: Oracle Linux, RHEL, Windows Server
- **Speicher**: Mindestens 2GB RAM (empfohlen 8GB+)
- **Festplatte**: Mindestens 8GB für Software

---

## Installation unter Linux

### 1. Oracle-Benutzer erstellen

```bash
groupadd -g 54321 oinstall
groupadd -g 54322 dba
useradd -u 54321 -g oinstall -G dba oracle
passwd oracle
```

### 2. Kernel-Parameter konfigurieren

```bash
kernel.shmmax = 4294967296
kernel.shmall = 2097152
```

### 3. Installer ausführen

```bash
./runInstaller
```

---

## Nach der Installation

```sql
sqlplus / as sysdba
SELECT name, open_mode FROM v$database;
```
