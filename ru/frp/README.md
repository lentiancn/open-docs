# frp

frp — это высокопроизводительный инструмент обратного прокси для穿透内网 (NAT-穿透).

## Введение

frp (Fast Reverse Proxy) позволяет внутренним сетевым службам быть доступными через публичный сервер, поддерживая протоколы TCP, UDP, HTTP и HTTPS.

## Основные функции

- **Высокая производительность**: Разработан на Go, высокая производительность
- **Мультипротокольность**: Поддержка TCP, UDP, HTTP, HTTPS
- **Безопасность**: Поддержка шифрования и сжатия
- **Dashboard**: Веб-панель управления
- **Плагин-система**: Обширная поддержка плагинов

## Быстрый старт

### Сервер

```bash
./frps -c frps.ini
```

### Клиент

```bash
./frpc -c frpc.ini
```
