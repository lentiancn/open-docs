# Yarn 🧶

## Yarnとは？

Yarnは強力な**パッケージマネージャー**であり、プロジェクトマネージャーです。シンプルなプロジェクトでも大規模なmonorepoでも、オープンソース開発者でも企業ユーザーでも、Yarnがサポートします。

YarnはJavaScriptプロジェクトのために設計された依存関係管理ツールです。**速度**、**正確性**、**セキュリティ**、**開発者体験**に焦点を当て、工作区、オフラインキャッシュ、並行インストール、ハードニングモード、インタラクティブコマンドなどの革新的な機能で常に改善しています。

### Yarnの主要機能

- **ワークスペース**：工作区専用に設計された最初のパッケージマネージャー
- **安定性**：今日の安装が将来も同じ方法で動作することを保証
- **ドキュメント**：各バージョンで常に改善
- **プラグインシステム**：問題解決のためのツールを提供
- **イノベーション**：常に最前線で現状を挑戦
- **完全独立性**：企業に依存しない完全なオープンソースプロジェクト

## クイックスタート

### インストール

```bash
# corepackを使用（推奨）
corepack enable
corepack prepare yarn@stable --activate

# npmを使用
npm install -g yarn
```

### 確認

```bash
yarn --version
```

### 初期化

```bash
yarn init
```

### 依存関係インストール

```bash
yarn install
```

---

## 一般的なコマンド

### 依存関係追加

```bash
yarn add lodash
yarn add -D typescript
yarn add lodash@4.17.21
```

### 依存関係削除

```bash
yarn remove lodash
```

### 依存関係更新

```bash
yarn up lodash
yarn up --all
```

### スクリプト実行

```bash
yarn build
yarn dev
yarn test
```

## ワークスペース

ワークスペースを使用すると、単一のリポジトリで複数のパッケージを管理できます。

### 設定

```json
{
  "workspaces": ["packages/*"]
}
```

### コマンド

```bash
yarn workspace @my-org/core add lodash
yarn workspaces foreach -pt run build
```

## 設定

### .yarnrc.yml

```yaml
npmRegistryServer: "https://registry.npmjs.org"
nodeLinker: node-modules
```
