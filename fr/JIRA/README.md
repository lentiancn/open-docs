# JIRA

JIRA est un outil de suivi des problèmes et de gestion de projet développé par Atlassian.

## Présentation

JIRA est un outil leader de gestion de projet et de problèmes, largement utilisé dans le développement agile et la gestion des services informatiques.

## Fonctionnalités principales

- **Suivi des problèmes** : Gestion complète des défauts et des tâches
- **Kanban agile** : Gestion visuelle des tâches
- **Support Scrum** : Planification et suivi des sprints
- **Automatisation des flux de travail** : Automatisation des processus métier
- **Rapports et analyses** : Fonctionnalités riches de création de rapports
- **Intégrations** : Intégration avec divers outils

## Démarrage rapide

### Installation

```bash
# Docker
docker run -v jira-data:/var/atlassian -p 8080:8080 atlassian/jira-software
```

### Accès

Ouvrez votre navigateur et accédez à http://localhost:8080
