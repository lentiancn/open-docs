# JDK Installation Guide

JDK (Java Development Kit) is the official Java development kit. This guide covers installation methods for various environments.

## Version Information

| Version | Status | Description |
|---------|--------|-------------|
| JDK 21 (LTS) | ✅ Current | Latest LTS version |
| JDK 17 (LTS) | ✅ Supported | Previous LTS |
| JDK 11 (LTS) | ✅ Supported | Legacy LTS |
| JDK 8 | ✅ Supported | Legacy version |
| JDK 22 | ✅ Latest | Latest feature release |

### LTS vs Feature Releases

- **LTS (Long Term Support)**: 2-3 years of updates, recommended for production
- **Feature Releases**: New features every 6 months, shorter support

## Installation Methods

### Method 1: Oracle JDK

#### Windows

1. Download from: https://www.oracle.com/java/technologies/downloads/
2. Run the installer (.exe)
3. Follow the wizard

#### macOS

```bash
# Download DMG from Oracle website
# Or use Homebrew
brew install openjdk@21
```

#### Linux

```bash
# Download tar.gz
wget https://download.oracle.com/java/21/latest/jdk-21_linux-x64_bin.tar.gz
tar -xzf jdk-21_linux-x64_bin.tar.gz
sudo mv jdk-21 /opt/jdk-21
```

### Method 2: Eclipse Temurin (Adoptium)

#### Windows

1. Download from: https://adoptium.net/temurin/releases/
2. Run the installer

#### macOS

```bash
# Using Homebrew
brew install --cask temurin

# Or download pkg from website
```

#### Linux

```bash
# Ubuntu/Debian
wget https://github.com/adoptium/temurin21-binaries/releases/download/jdk-21.0.2%2B13/OpenJDK21U-jdk_x64_linux_hotspot_21.0.2_13.tar.gz
tar -xzf OpenJDK21U-jdk_x64_linux_hotspot_21.0.2_13.tar.gz

# Set environment
export JAVA_HOME=/path/to/jdk-21
export PATH=$JAVA_HOME/bin:$PATH
```

### Method 3: SDKMAN

```bash
# Install SDKMAN
curl -s "https://get.sdkman.io" | bash

# Install JDK
sdk install java 21.0.2-tem

# List available versions
sdk list java
```

### Method 4: Docker

```dockerfile
FROM eclipse-temurin:21-jdk
WORKDIR /app
COPY . .
RUN javac Main.java
CMD ["java", "Main"]
```

### Method 5: Windows Package Managers

#### Chocolatey

```powershell
choco install openjdk21
```

#### Scoop

```powershell
scoop install openjdk
```

## Environment Configuration

### Windows

```powershell
# Set JAVA_HOME
[System.Environment]::SetEnvironmentVariable("JAVA_HOME", "C:\Program Files\Java\jdk-21", "User")

# Add to PATH
[System.Environment]::SetEnvironmentVariable("PATH", "$env:PATH;C:\Program Files\Java\jdk-21\bin", "User")
```

### macOS/Linux

```bash
# Add to ~/.bashrc or ~/.zshrc
export JAVA_HOME=/opt/jdk-21
export PATH=$JAVA_HOME/bin:$PATH

# Apply
source ~/.bashrc
```

## Verification

```bash
# Check Java version
java -version

# Check Javac version
javac -version

# Check JAVA_HOME
echo $JAVA_HOME
```

## Multiple JDK Versions

### Using SDKMAN

```bash
# Install multiple versions
sdk install java 17.0.8-tem
sdk install java 21.0.2-tem

# Switch version
sdk use java 17.0.8-tem
```

## Common Issues

### "java" not found?

Verify PATH includes Java bin directory.

### Wrong Java version?

Check which java is being used:

```bash
which java
java -version
```

### JAVA_HOME not set?

Set JAVA_HOME to JDK installation directory.
