# Claude Code

[Claude Code](https://code.claude.com/)는 Anthropic의 지능형 프로그래밍 도우미입니다.

## 개요

 Claude Code는 터미널, IDE, 데스크톱 앱 및 브라우저에서 사용할 수 있는 AI 기반 프로그래밍 도우미입니다.

## 기능

- **작업 자동화**: 테스트, 린트 오류 수정, 병합 충돌 해결
- **기능 개발**: 요구사항을 설명하면 Claude Code가 구현
- **Git 통합**: 커밋, 브랜치, 풀 리퀘스트
- **MCP 통합**: 외부 도구 및 데이터 소스에 연결
- **사용자 정의**: CLAUDE.md, 사용자 정의 명령

## 빠른 시작

### 설치

**macOS, Linux, WSL:**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell:**
```powershell
irm https://claude.ai/install.ps1 | iex
```

### 시작

```bash
cd your-project
claude
```
