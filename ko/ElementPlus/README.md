# Element Plus

Vue 3 기반 UI 컴포넌트 라이브러리

## 특징

- Composition API를 통한 Vue 3 지원
- 70+ 컴포넌트
- TypeScript 지원
- 온디맨드 임포트
- 국제화

## 빠른 시작

npm install element-plus
