# Webpack インストールガイド

WebpackはモダンなJavaScriptアプリケーション用の静的モジュールバンドラーです。このガイドでは、すべての環境でのインストール方法を説明します。

## バージョン情報

| バージョン | ステータス | 説明 |
|------------|------------|------|
| Webpack 5.x | ✅ 現行 | 最新安定版 |
| Webpack 4.x | ❌ 旧版 | 新規プロジェクトでは非推奨 |

> **重要**：このガイドは**Webpack 5.x**を対象としています。新規プロジェクトでは最新バージョンを使用してください。

## インストール方法

### 方法1：ローカルインストール（推奨）

```bash
# プロジェクトディレクトリを作成
mkdir my-webpack-project
cd my-webpack-project

# npmを初期化
npm init -y

# webpackをインストール
npm install webpack --save-dev

# webpack-cliをインストール
npm install webpack-cli --save-dev
```

### 方法2：グローバルインストール

```bash
npm install -g webpack webpack-cli
```

### 方法3：npxを使用

```bash
npx webpack --version
```

### 方法4：プロジェクトテンプレートを作成

```bash
# create-react-appを使用（webpackが組み込まれている）
npx create-react-app my-app

# Vue CLIを使用
npm install -g @vue/cli
vue create my-app

# Viteを使用
npm create vite@latest my-app -- --template vanilla
```

## webpack.config.jsの設定

### 基本的な設定

```javascript
const path = require('path');

module.exports = {
  mode: 'development', // または 'production'
  
  // エントリーポイント
  entry: './src/index.js',
  
  // 出力設定
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
    clean: true, // distディレクトリをクリア
  },
  
  // ローダー
  module: {
    rules: [
      {
        test: /\.css$/i,
        use: ['style-loader', 'css-loader'],
      },
      {
        test: /\.(png|svg|jpg|jpeg|gif)$/i,
        type: 'asset/resource',
      },
    ],
  },
};
```

### 本番環境用設定

```javascript
const path = require('path');
const HtmlWebpackPlugin = require('html-webpack-plugin');

module.exports = {
  mode: 'production',
  entry: './src/index.js',
  output: {
    filename: '[name].[contenthash].js',
    path: path.resolve(__dirname, 'dist'),
    clean: true,
  },
  module: {
    rules: [
      {
        test: /\.css$/i,
        use: ['style-loader', 'css-loader'],
      },
      {
        test: /\.js$/i,
        exclude: /node_modules/,
        use: {
          loader: 'babel-loader',
          options: {
            presets: ['@babel/preset-env'],
          },
        },
      },
    ],
  },
  plugins: [
    new HtmlWebpackPlugin({
      template: './src/index.html',
    }),
  ],
  optimization: {
    splitChunks: {
      chunks: 'all',
    },
  },
};
```

### 開発環境用設定

```javascript
const path = require('path');
const HtmlWebpackPlugin = require('html-webpack-plugin');

module.exports = {
  mode: 'development',
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
  },
  devServer: {
    static: './dist',
    hot: true,
    port: 3000,
    open: true,
  },
  plugins: [
    new HtmlWebpackPlugin({
      template: './src/index.html',
    }),
  ],
};
```

## プラグイン設定

### HtmlWebpackPlugin

```bash
npm install --save-dev html-webpack-plugin
```

```javascript
const HtmlWebpackPlugin = require('html-webpack-plugin');

plugins: [
  new HtmlWebpackPlugin({
    title: 'My App',
    template: './src/index.html',
    filename: 'index.html',
  }),
];
```

### MiniCssExtractPlugin

```bash
npm install --save-dev mini-css-extract-plugin
```

```javascript
const MiniCssExtractPlugin = require('mini-css-extract-plugin');

module.exports = {
  plugins: [
    new MiniCssExtractPlugin({
      filename: '[name].[contenthash].css',
    }),
  ],
  module: {
    rules: [
      {
        test: /\.css$/i,
        use: [MiniCssExtractPlugin.loader, 'css-loader'],
      },
    ],
  },
};
```

### DefinePlugin

```javascript
const webpack = require('webpack');

plugins: [
  new webpack.DefinePlugin({
    'process.env.NODE_ENV': JSON.stringify('production'),
  }),
];
```

## 開発サーバー

### インストール

```bash
npm install --save-dev webpack-dev-server
```

### 設定

```javascript
module.exports = {
  devServer: {
    static: {
      directory: path.join(__dirname, 'dist'),
    },
    compress: true,
    port: 3000,
    hot: true,
    open: true,
    historyApiFallback: true,
    proxy: {
      '/api': 'http://localhost:8080',
    },
  },
};
```

### 実行

```bash
# package.json
{
  "scripts": {
    "start": "webpack serve --mode development",
    "build": "webpack --mode production"
  }
}

npm start
```

## よくある問題

### ビルドが失敗する場合

エラーメッセージを確認し、すべての依存関係がインストールされていることを確認してください：

```bash
npm install
npm audit fix
```

### モジュールが見つからない場合

resolve.extensionsが正しく設定されていることを確認してください：

```javascript
resolve: {
  extensions: ['.js', '.jsx', '.ts', '.tsx'],
}
```

### ホットリロードが動作しない場合

devServerの設定が正しいことを確認してください：

```javascript
devServer: {
  hot: true,
  liveReload: true,
}
```
