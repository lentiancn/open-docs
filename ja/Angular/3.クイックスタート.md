```bash

# 期待される出力: v18.x.x 以上
```

## 2. プロジェクトの作成

ターミナルを開き、以下のコマンドを実行して新しいプロジェクトを作成します：

```bash
ng new hello-angular
```

### 2.1 インストールオプション

インストールプロセスで以下の質問があります：

| オプション | 推奨選択 | 説明 |
|----------|----------|------|
| Would you like to add Angular routing? | Yes | ルーティング機能を追加 |
| Which stylesheet format would you like to use? | CSS または SCSS | SCSSを推奨 |

インストールが完了するのを待ちます（約2〜5分）。

### 2.2 プロジェクトディレクトリに移動

```bash
cd hello-angular
```

## 4. プロジェクト構造の理解

### 4.1 ディレクトリ構造

```
hello-angular/
├── src/
│   ├── app/
│   │   ├── app.component.ts       # ルートコンポーネント
│   │   ├── app.component.html    # ルートコンポーネントテンプレート
│   │   ├── app.component.css      # ルートコンポーネントスタイル
│   │   ├── app.config.ts         # アプリケーション設定
│   │   └── app.routes.ts         # ルーティング設定
│   ├── assets/                   # 静的アセット
│   ├── main.ts                   # アプリケーションエントリ
│   └── index.html                # HTMLエントリ
├── angular.json                  # CLI設定
├── package.json                  # 依存関係設定
└── tsconfig.json                 # TS設定
```

### 4.2 ルートコンポーネントファイル

`src/app/app.component.ts` を開きます：

```typescript
import { Component } from '@angular/core';
import { CommonModule } from '@angular/common';
import { RouterOutlet, RouterLink } from '@angular/router';

@Component({
  selector: 'app-root',
  standalone: true,
  imports: [CommonModule, RouterOutlet, RouterLink],
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent {
  title = 'hello-angular';
}
```

## 6. インタラクティブ機能の追加

### 6.1 カウンター機能の作成

`app.component.ts` に以下を追加します：

```typescript
import { Component } from '@angular/core';
import { CommonModule } from '@angular/common';

@Component({
  selector: 'app-root',
  standalone: true,
  imports: [CommonModule],
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent {
  title = 'hello-angular';
  count = 0;
  
  increment(): void {
    this.count++;
  }
  
  decrement(): void {
    this.count--;
  }
}
```

### 6.2 テンプレートでの使用

`app.component.html` を更新します：

```html
<div class="container">
  <h1>Angularへようこそ！</h1>
  
  <div class="card">
    <h2>カウンター例</h2>
    <p class="count">{{ count }}</p>
    <div class="buttons">
      <button (click)="decrement()">-</button>
      <button (click)="increment()">+</button>
    </div>
  </div>
</div>
```

### 6.3 ボタンスキルの追加

`app.component.css` を更新します：

```css
.count {
  font-size: 3rem;
  font-weight: bold;
  text-align: center;
  margin: 1rem 0;
}

.buttons {
  display: flex;
  gap: 1rem;
  justify-content: center;
}

button {
  padding: 0.5rem 1.5rem;
  font-size: 1.2rem;
  border: none;
  border-radius: 4px;
  cursor: pointer;
  background: #dd0031;
  color: white;
  transition: background 0.2s;
}

button:hover {
  background: #b3002d;
}
```

### 6.4 結果の確認

保存すると、ブラウザが自動的に更新されます。ボタンをクリックしてインタラクティブ機能をテストできます。
