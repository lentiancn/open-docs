# MySQL Benutzerhandbuch

## Grundlegende Datenbankoperationen

### Verbindung zu MySQL

Lokale Verbindung:
```bash
mysql -u root -p
```

Remote-Verbindung:
```bash
mysql -h hostname -u username -p datenbankname
```

### Datenbank erstellen

```sql
CREATE DATABASE datenbankname;
```

Mit spezifischem Zeichensatz erstellen:
```sql
CREATE DATABASE datenbankname CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### Datenbanken anzeigen

```sql
SHOW DATABASES;
```

### Datenbank auswählen

```sql
USE datenbankname;
```

### Datenbank löschen

```sql
DROP DATABASE datenbankname;
```

## Tabellenoperationen

### Tabelle erstellen

```sql
CREATE TABLE tabellenname (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100) UNIQUE,
    erstellt_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    aktualisiert_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### Tabellenstruktur anzeigen

```sql
DESCRIBE tabellenname;
-- oder
SHOW COLUMNS FROM tabellenname;
```

### Alle Tabellen anzeigen

```sql
SHOW TABLES;
```

### Tabellenstruktur ändern

Spalte hinzufügen:
```sql
ALTER TABLE tabellenname ADD COLUMN neue_spalte VARCHAR(50);
```

Spalte ändern:
```sql
ALTER TABLE tabellenname MODIFY spaltenname VARCHAR(200);
```

Spalte löschen:
```sql
ALTER TABLE tabellenname DROP COLUMN spaltenname;
```

### Tabelle löschen

```sql
DROP TABLE tabellenname;
```

## Datenoperationen

### Daten einfügen

Einzelne Einfügung:
```sql
INSERT INTO tabellenname (spalte1, spalte2) VALUES ('wert1', 'wert2');
```

Mehrfache Einfügungen:
```sql
INSERT INTO tabellenname (spalte1, spalte2) VALUES 
('wert1', 'wert2'),
('wert3', 'wert4');
```

### Daten abfragen

Basisabfrage:
```sql
SELECT * FROM tabellenname;
```

Bedingte Abfrage:
```sql
SELECT spalte1, spalte2 FROM tabellenname WHERE bedingung;
```

Sortierung:
```sql
SELECT * FROM tabellenname ORDER BY spaltenname DESC;
```

Paginierung:
```sql
SELECT * FROM tabellenname LIMIT 10 OFFSET 20;
```

### Daten aktualisieren

```sql
UPDATE tabellenname SET spalte1 = 'neuer_wert' WHERE bedingung;
```

### Daten löschen

```sql
DELETE FROM tabellenname WHERE bedingung;
```

Tabelle leeren:
```sql
TRUNCATE TABLE tabellenname;
```

## Indexoperationen

### Indizes erstellen

Normaler Index:
```sql
CREATE INDEX indexname ON tabellenname (spaltenname);
```

Eindeutiger Index:
```sql
CREATE UNIQUE INDEX indexname ON tabellenname (spaltenname);
```

Zusammengesetzter Index:
```sql
CREATE INDEX indexname ON tabellenname (spalte1, spalte2);
```

### Indizes anzeigen

```sql
SHOW INDEX FROM tabellenname;
```

### Indizes löschen

```sql
DROP INDEX indexname ON tabellenname;
```

## Benutzer- und Berechtigungsverwaltung

### Benutzer erstellen

```sql
CREATE USER 'benutzername'@'localhost' IDENTIFIED BY 'passwort';
```

### Berechtigungen erteilen

```sql
GRANT SELECT, INSERT, UPDATE ON datenbankname.* TO 'benutzername'@'localhost';
```

Alle Berechtigungen erteilen:
```sql
GRANT ALL PRIVILEGES ON datenbankname.* TO 'benutzername'@'localhost';
```

### Berechtigungen aktualisieren

```sql
FLUSH PRIVILEGES;
```

### Benutzerberechtigungen anzeigen

```sql
SHOW GRANTS FOR 'benutzername'@'localhost';
```

### Benutzer löschen

```sql
DROP USER 'benutzername'@'localhost';
```

## Transaktionskontrolle

### Transaktion starten

```sql
START TRANSACTION;
-- oder
BEGIN;
```

### Transaktion bestätigen

```sql
COMMIT;
```

### Transaktion zurücksetzen

```sql
ROLLBACK;
```

### Sicherungspunkte setzen

```sql
SAVEPOINT sicherungspunktname;
ROLLBACK TO SAVEPOINT sicherungspunktname;
```

## Häufige Funktionen

### Zeichenkettenfunktionen

```sql
SELECT CONCAT('Hallo', ' ', 'Welt');
SELECT UPPER('hallo');
SELECT LOWER('HALLO');
SELECT SUBSTRING('Hallo Welt', 1, 5);
SELECT LENGTH('Hallo');
SELECT TRIM('  Hallo  ');
```

### Numerische Funktionen

```sql
SELECT ABS(-5);
SELECT CEIL(3.14);
SELECT FLOOR(3.14);
SELECT ROUND(3.14159, 2);
SELECT MOD(10, 3);
```

### Datumsfunktionen

```sql
SELECT NOW();
SELECT CURDATE();
SELECT CURTIME();
SELECT DATE('2024-01-01 10:30:00');
SELECT YEAR('2024-01-01');
SELECT MONTH('2024-01-01');
SELECT DAY('2024-01-01');
SELECT DATE_ADD('2024-01-01', INTERVAL 1 DAY);
SELECT DATEDIFF('2024-01-01', '2023-12-01');
```

### Aggregatfunktionen

```sql
SELECT COUNT(*) FROM tabellenname;
SELECT SUM(spaltenname) FROM tabellenname;
SELECT AVG(spaltenname) FROM tabellenname;
SELECT MAX(spaltenname) FROM tabellenname;
SELECT MIN(spaltenname) FROM tabellenname;
```

## Sicherung und Wiederherstellung

### Datenbank sichern

```bash
mysqldump -u root -p datenbankname > backup.sql
```

Alle Datenbanken sichern:
```bash
mysqldump -u root -p --all-databases > alle_datenbanken.sql
```

### Datenbank wiederherstellen

```bash
mysql -u root -p datenbankname < backup.sql
```

### Abfrageergebnisse exportieren

```bash
mysql -u root -p -e "SELECT * FROM tabellenname" datenbankname > ausgabe.tsv
```

## Leistungsoptimierung

### Abfragen mit EXPLAIN analysieren

```sql
EXPLAIN SELECT * FROM tabellenname WHERE bedingung;
```

### Optimierungstipps

1. Indizes für häufige Abfragebedingungen erstellen
2. SELECT * vermeiden, nur benötigte Spalten abfragen
3. LIMIT verwenden, um Ergebnismengen einzuschränken
4. Vermeiden Sie Funktionen auf indizierten Spalten
5. JOINs korrekt verwenden, Kartesische Produkte vermeiden
6. Tabellen regelmäßig analysieren und optimieren

### Langsame Abfragen anzeigen

```sql
SHOW VARIABLES LIKE 'slow_query_log%';
```

## Behandlung häufiger Fehler

### Verbindungsfehler
Überprüfen Sie, ob der MySQL-Dienst läuft, der Port korrekt ist und die Benutzerberechtigungen korrekt konfiguriert sind.

### Zeichensatzfehler
Stellen Sie sicher, dass Datenbank, Tabelle und Verbindung denselben Zeichensatz verwenden (utf8mb4 wird empfohlen).

### Berechtigungsfehler
Bestätigen Sie, dass der Benutzer über die entsprechenden Datenbank- und Tabellenoperationsberechtigungen verfügt.
