# Element Plus Installationsanleitung

Element Plus ist eine Vue 3 Komponentenbibliothek für Entwickler, Designer und Produktmanager. Diese Anleitung behandelt Installationsmethoden für alle Umgebungen.

## Versionsinformationen

| Version | Status | Beschreibung |
|---------|--------|--------------|
| Element Plus 2.x | ✅ Aktuell | Neueste stabile Vue 3 Version |
| Element Plus 1.x | ⚠️ Wartung | Nur Sicherheitsupdates |
| Element UI | ❌ Veraltet | Nur Vue 2 Unterstützung |

> **Wichtig**: Element Plus unterstützt nur Vue 3. Wenn Sie Vue 2 verwenden, bitte Element UI verwenden.

## Installationsmethoden

### Methode 1: npm (Empfohlen)

```bash
# npm verwenden
npm install element-plus

# yarn verwenden
yarn add element-plus

# pnpm verwenden
pnpm add element-plus
```

### Methode 2: CDN

```html
<!-- Vue 3 einbinden -->
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<!-- Element Plus einbinden -->
<link rel="stylesheet" href="https://unpkg.com/element-plus/dist/index.css" />
<script src="https://unpkg.com/element-plus/dist/index.full.js"></script>

<!-- Icons einbinden (optional) -->
<script src="https://unpkg.com/@element-plus/icons-vue"></script>
```

#### Vollständige CDN-Vorlage

```html
<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Element Plus CDN</title>
    <link rel="stylesheet" href="https://unpkg.com/element-plus/dist/index.css">
</head>
<body>
    <div id="app">
        <el-button>Button</el-button>
        <el-date-picker></el-date-picker>
    </div>

    <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
    <script src="https://unpkg.com/element-plus/dist/index.full.js"></script>
    <script src="https://unpkg.com/@element-plus/icons-vue"></script>
    
    <script>
        const { createApp, ref } = Vue
        const app = createApp({
            setup() {
                return {}
            }
        })
        
        for (const [key, component] of Object.entries(ElementPlus)) {
            app.component(key, component)
        }
        
        for (const [key, component] of Object.entries(ElementPlusIconsVue)) {
            app.component(key, component)
        }
        
        app.use(ElementPlus)
        app.mount('#app')
    </script>
</body>
</html>
```

### Methode 3: Vite verwenden

```bash
# Vite-Projekt erstellen
npm create vite@latest my-app -- --template vue

# Projektverzeichnis betreten
cd my-app

# Element Plus installieren
npm install element-plus

# Icons installieren
npm install @element-plus/icons-vue
```

#### In main.js konfigurieren

```javascript
import { createApp } from 'vue'
import App from './App.vue'

// Element Plus Styles einbinden
import 'element-plus/dist/index.css'

// Element Plus einbinden
import ElementPlus from 'element-plus'

// Icons einbinden
import * as ElementPlusIconsVue from '@element-plus/icons-vue'

const app = createApp(App)

// Alle Icons global registrieren
for (const [key, component] of Object.entries(ElementPlusIconsVue)) {
    app.component(key, component)
}

app.use(ElementPlus)
app.mount('#app')
```

### Methode 4: Vue CLI verwenden

```bash
# Vue CLI installieren
npm install -g @vue/cli

# Projekt erstellen
vue create my-project

# Element Plus hinzufügen
cd my-project
npm install element-plus
npm install @element-plus/icons-vue
```

#### In main.js konfigurieren

```javascript
import { createApp } from 'vue'
import App from './App.vue'

import 'element-plus/dist/index.css'
import ElementPlus from 'element-plus'
import * as icons from '@element-plus/icons-vue'

const app = createApp(App)

// Icons global registrieren
for (const [key, component] of Object.entries(icons)) {
    app.component(key, component)
}

app.use(ElementPlus)
app.mount('#app')
```

### Methode 5: Bedarfsweise Importierung (Empfohlen für Produktion)

#### unplugin-vue-components verwenden

```bash
# Abhängigkeiten installieren
npm install element-plus unplugin-vue-components unplugin-auto-import
```

#### Vite-Konfiguration

```javascript
// vite.config.js
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

#### Verwendung

```vue
<template>
  <!-- Komponenten ohne Import verwenden -->
  <el-button type="primary">Button</el-button>
  <el-input v-model="input" placeholder="Bitte eingeben" />
</template>

<script setup>
import { ref } from 'vue'
const input = ref('')
</script>
```

## Häufig gestellte Fragen

### Icons werden nicht angezeigt?

Stellen Sie sicher, dass Icons global registriert sind:

```javascript
import * as icons from '@element-plus/icons-vue'
for (const [key, icon] of Object.entries(icons)) {
    app.component(key, icon)
}
```

### Styles werden nicht angewendet?

Stellen Sie sicher, dass Element Plus Styles eingebunden sind:

```javascript
import 'element-plus/dist/index.css'
```

### Vue-Version stimmt nicht?

Element Plus erfordert Vue 3. Vue-Version überprüfen:

```bash
npm list vue
```
