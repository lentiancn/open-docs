# Angular FAQ

## Проблемы установки

### Q: Установка не удалась?

**Решения:**
1. Проверить версию Node.js
```bash
node --version
```
2. Очистить кэш npm
```bash
npm cache clean --force
```

## Производительность

### Q: Медленная загрузка?

**Решения:**
1. Lazy loading
2. Продакшен режим
```bash
ng build --configuration=production
```
