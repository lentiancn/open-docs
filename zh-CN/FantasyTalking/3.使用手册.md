# FantasyTalking 使用手册

## 基础功能

FantasyTalking 提供了完善的语音合成和对话功能，本手册将详细介绍各项功能的使用方法。

### 语音合成

语音合成是 FantasyTalking 的核心功能，将文本转换为自然流畅的语音。以下是基本的使用方法：

```javascript
// 简单文本转语音
const audio = await client.speak('你好，欢迎使用 FantasyTalking');

// 指定语音风格
const audio = await client.speak('今天天气真好', {
  voice: {
    style: 'happy',
    language: 'zh-CN'
  }
});

// 指定语速和音调
const audio = await client.speak('请稍等片刻', {
  voice: {
    speed: 1.2,
    pitch: 0.9
  }
});
```

### SSML 标记语言

FantasyTalking 支持 SSML 标记语言，可以对语音输出进行更精细的控制：

```xml
<speak>
  <p>
    <s>第一句话。</s>
    <s>第二句话。</s>
  </p>
  <break time="500ms"/>
  <p>
    <emphasis level="moderate">这是重点内容</emphasis>
  </p>
  <phoneme alphabet="py" ph="ni3 hao3">你好</phoneme>
</speak>
```

对应的 JavaScript 代码：

```javascript
const ssml = `
  <speak>
    <p>
      <s>第一句话。</s>
      <s>第二句话。</s>
    </p>
    <break time="500ms"/>
    <p>
      <emphasis level="moderate">这是重点内容</emphasis>
    </p>
  </speak>
`;

const audio = await client.speak(ssml, { ssml: true });
```

### 语音事件

在语音播放过程中，可以监听各种事件来实现交互：

```javascript
// 监听语音开始
client.on('speech-start', (event) => {
  console.log('语音开始播放');
});

// 监听单词边界（用于字幕同步）
client.on('word-boundary', (event) => {
  console.log(`当前单词: ${event.word}`);
  console.log(`开始时间: ${event.start}ms`);
  console.log(`结束时间: ${event.end}ms`);
});

// 监听语音结束
client.on('speech-end', (event) => {
  console.log('语音播放完成');
});

// 监听错误
client.on('error', (event) => {
  console.error('错误:', event.error);
});
```

### 批量合成

需要合成大量文本时，可以使用批量合成功能来提高效率：

```javascript
const texts = [
  '这是第一条语音',
  '这是第二条语音',
  '这是第三条语音'
];

// 并行合成
const results = await Promise.all(
  texts.map(text => client.speak(text))
);

// 串行合成（保留顺序）
const results = [];
for (const text of texts) {
  const audio = await client.speak(text);
  results.push(audio);
}
```

## 对话功能

### 基础对话

FantasyTalking 提供了强大的对话管理功能，支持多轮对话和上下文记忆：

```javascript
// 创建对话会话
const session = client.createSession({
  // 可选：设置系统提示
  systemPrompt: '你是一个友好的助手，擅长回答问题',
  // 可选：设置语言
  language: 'zh-CN'
});

// 第一轮对话
const response1 = await session.chat('今天天气怎么样？');
console.log(response1.text);  // AI 回复的文本
console.log(response1.audio);  // AI 回复的语音

// 第二轮对话（自动包含上下文）
const response2 = await session.chat('明天呢？');
console.log(response2.text);
```

### 对话配置

可以针对不同的对话场景进行配置：

```javascript
const session = client.createSession({
  // 对话配置
  dialogue: {
    // 温度参数，控制回复的创造性
    temperature: 0.7,
    // 最大回复 token 数
    maxTokens: 500,
    // 话题标签，用于区分不同场景
    tags: ['客服', '售前咨询']
  },
  
  // 语音配置
  voice: {
    // AI 回复时的语音风格
    style: 'professional',
    // 语速
    speed: 1.0
  }
});
```

### 对话历史管理

```javascript
// 获取对话历史
const history = session.getHistory();
console.log(history);

// 保存对话历史
const savedHistory = session.save();
fs.writeFileSync('conversation.json', JSON.stringify(savedHistory));

// 加载对话历史
const loadedSession = client.loadSession(savedHistory);
```

## 高级功能

### 流式语音合成

对于需要实时反馈的应用场景，可以使用流式语音合成：

```javascript
// 启用流式输出
const stream = await client.speakStream('这是一段很长的文本');

// 监听数据块
stream.on('data', (chunk) => {
  // 播放音频块
  audioPlayer.play(chunk);
});

stream.on('end', () => {
  console.log('语音合成完成');
});

stream.on('error', (error) => {
  console.error('错误:', error);
});
```

### 自定义声音模型

如果需要使用品牌专属的声音，可以训练自定义模型：

```javascript
// 准备训练数据
const trainingData = {
  name: '品牌声音',
  description: '用于企业品牌的声音',
  samples: [
    { text: '样本文本1', audio: 'base64编码的音频1' },
    { text: '样本文本2', audio: 'base64编码的音频2' },
    // 更多样本...
  ]
};

// 提交训练任务
const job = await client.trainVoice(trainingData);

// 轮询训练状态
while (job.status !== 'completed') {
  await sleep(5000);
  const status = await client.getJobStatus(job.id);
  console.log(`训练进度: ${status.progress}%`);
}

// 训练完成，使用自定义声音
const audio = await client.speak('使用自定义声音', {
  voice: {
    modelId: job.modelId
  }
});
```

### 音频后处理

FantasyTalking 提供了多种音频后处理功能：

```javascript
const audio = await client.speak('原始语音');

// 音量调整
const normalized = client.process(audio, {
  normalize: true,
  targetLevel: -20  // dB
});

// 降噪
const denoised = client.process(audio, {
  denoise: true,
  noiseThreshold: 0.5
});

// 语速调整
const spedUp = client.process(audio, {
  speed: 1.5  // 1.5倍速
});
```

## 最佳实践

### 性能优化

在实际应用中，需要注意以下几点来优化性能：

1. **连接复用**：避免频繁创建客户端实例，复用连接可以显著降低延迟
2. **批量处理**：需要处理大量文本时，使用批量合成功能
3. **缓存机制**：对于相同的文本，可以使用缓存避免重复合成
4. **流式输出**：长文本使用流式输出可以提前开始播放，改善用户体验

### 错误处理

建议实现完善的错误处理机制：

```javascript
try {
  const audio = await client.speak('测试文本');
} catch (error) {
  if (error.code === 'RATE_LIMIT') {
    // 处理速率限制
    console.log('请求过于频繁，请稍后重试');
  } else if (error.code === 'INVALID_API_KEY') {
    // 处理认证错误
    console.log('API 密钥无效');
  } else {
    // 其他错误
    console.error('未知错误:', error);
  }
}
```

### 用户体验

为了提供更好的用户体验，建议：

1. **预加载**：在用户可能需要语音前提前合成
2. **反馈提示**：语音播放时提供视觉反馈
3. **中断处理**：支持用户中断语音播放
4. **离线支持**：实现基础功能的离线支持
