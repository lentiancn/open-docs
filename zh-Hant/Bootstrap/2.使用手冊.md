## 簡介

### 什麼是 Bootstrap？

Bootstrap 是一個強大的前端框架，用於使用 HTML、CSS 和 JavaScript 快速開發響應式網站。

### 核心特性

- **響應式網格系統**：12 列響應式網格
- **預編譯 CSS**：豐富的實用工具類
- **組件庫**：20 多個 UI 組件
- **JavaScript 插件**：模態框、輪播、工具提示等

## 佈局

### 容器

```html
<div class="container">內容</div>
<div class="container-fluid">滿寬內容</div>
```

### 網格系統

```html
<div class="row">
    <div class="col">列</div>
    <div class="col">列</div>
</div>
```

---

## 組件

### 按鈕

```html
<button class="btn btn-primary">主要</button>
<button class="btn btn-secondary">次要</button>
<button class="btn btn-success">成功</button>
<button class="btn btn-danger">危險</button>
```

### 卡片

```html
<div class="card" style="width: 18rem;">
    <div class="card-body">
        <h5 class="card-title">標題</h5>
        <p class="card-text">內容</p>
    </div>
</div>
```

### 表單

```html
<form>
    <div class="mb-3">
        <label class="form-label">郵箱</label>
        <input type="email" class="form-control">
    </div>
    <button type="submit" class="btn btn-primary">提交</button>
</form>
```

## JavaScript 組件

### 模態框

```html
<button class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#myModal">
    打開
</button>

<div class="modal" id="myModal">
    <div class="modal-dialog">
        <div class="modal-content">
            <div class="modal-header">
                <h5 class="modal-title">標題</h5>
                <button class="btn-close" data-bs-dismiss="modal"></button>
            </div>
            <div class="modal-body">內容</div>
        </div>
    </div>
</div>
```

### 下拉菜單

```html
<div class="dropdown">
    <button class="btn btn-secondary dropdown-toggle" data-bs-toggle="dropdown">
        下拉
    </button>
    <ul class="dropdown-menu">
        <li><a class="dropdown-item" href="#">選項 1</a></li>
        <li><a class="dropdown-item" href="#">選項 2</a></li>
    </ul>
</div>
```
