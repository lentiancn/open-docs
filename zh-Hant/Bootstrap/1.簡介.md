# 簡介

Bootstrap 是全球最受歡迎的 HTML、CSS 和 JavaScript 開源前端開發框架，由 Twitter 公司的設計師 Mark Otto 和 Jacob Thornton 主導開發。Bootstrap 提供了豐富的元件和強大的 JavaScript 外掛程式，能夠幫助開發者快速建構響應式 Web 應用程式。

Bootstrap 的核心概念是「行動優先」（Mobile First），從設計之初就考慮到了各種裝置的相容性問題。透過 Bootstrap，開發者可以輕鬆建立無論在桌面、平板或手機上都能完美顯示的網站。

## 發展歷程

Bootstrap 最初名為 Twitter Blueprint，於 2011 年開源發布。2012 年發布 Bootstrap 2，引入了響應式布局。2013 年發布 Bootstrap 3，完全重寫為平面化設計。2015 年發布 Bootstrap 4，開始使用 Sass 作為 CSS 預處理器。2021 年發布 Bootstrap 5，移除了 jQuery 依賴並全面採用原生 JavaScript。

## 核心特色

### 響應式布局

Bootstrap 的網格系統是其最強大的功能之一。透過 12 欄網格布局，開發者可以輕鬆建立適應不同螢幕尺寸的頁面布局。Bootstrap 5 提供了 xs、sm、md、lg、xl、xxl 六個斷點，分別對應不同寬度的裝置。

### 豐富的元件

Bootstrap 提供了大量可直接使用的 UI 元件，包括導航欄、卡片、按鈕、表單、彈出視窗、輪播圖、下拉選單等。這些元件都經過精心設計，具有良好的視覺效果和互動體驗。

### 強大的 JavaScript 外掛程式

Bootstrap 包含 20 多個 JavaScript 外掛程式，例如彈出視窗、輪播圖、工具提示、彈出框、摺疊面板等。這些外掛程式都是可選的，開發者可以按需引入，減少專案體積。

### 採用 Sass

Bootstrap 5 使用 Sass 作為 CSS 預處理器，提供了大量的 Sass 變數和混入（Mixins）。開發者可以輕鬆自訂 Bootstrap 的主題顏色、字體大小等樣式。

### 自訂主題

透過 Bootstrap 的主題系統，開發者可以輕鬆建立自己的主題風格。官方還提供了 Bootstrap Themes 市集，有大量精美的主題可供選擇。

### 無障礙支援

Bootstrap 元件內建了無障礙支援，包括鍵盤導航、螢幕閱讀器相容等，幫助開發者建立更具包容性的網站。

## Bootstrap 5 新特色

相較於 Bootstrap 4，Bootstrap 5 帶來了許多重大更新：

- 移除了 jQuery 依賴，全面使用原生 JavaScript
- 改進了網格系統，增加了 XXL 斷點
- 新的表單控制項替代了表單驗證
- 更好的自訂屬性支援
- 圖示庫從 Font Awesome 改為 Bootstrap Icons
- RTL（從右到左）支援

## 適用場景

- 快速原型開發
- 企業內部管理系統
- 電子商務網站
- 部落格和內容網站
- 管理後台介面
- 響應式網站開發

作為全球最受歡迎的 CSS 框架之一，Bootstrap 擁有龐大的社群和豐富的學習資源，是前端開發者必備的技能之一。
