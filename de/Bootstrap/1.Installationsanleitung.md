# Bootstrap 5 Installationsanleitung

Diese Anleitung beschreibt detailliert die Installation und Konfiguration von Bootstrap 5 in verschiedenen Umgebungen.

## Versionshinweise

| Version | Status | Beschreibung |
|---------|--------|--------------|
| Bootstrap 5.3.x | ✅ Aktuell | Neueste Version mit allen neuen Funktionen |
| Bootstrap 5.2.x | ✅ Unterstützt | Stabile Version |
| Bootstrap 5.1.x | ⚠️ Wartung | Sicherheitsupdates |
| Bootstrap 4.x | ❌ Eingestellt | Nicht für neue Projekte empfohlen |

> **Wichtig**: Diese Anleitung bezieht sich hauptsächlich auf **Bootstrap 5.3.x**. Bootstrap 4 wird nicht mehr gewartet und wird für neue Projekte nicht empfohlen.

## Installationsmethoden

### Methode 1: CDN-Einbindung (Einfachste)

Der schnellste Weg ist die Einbindung von Bootstrap CSS und JavaScript über ein CDN.

#### Nur CSS

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
```

#### CSS + JavaScript

```html
<!-- Bootstrap CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- Bootstrap JS (enthält Popper) -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
```

**CDN-Anbieter**:
- [jsDelivr](https://www.jsdelivr.com/) - Empfohlen
- [cdnjs](https://cdnjs.com/libraries/bootstrap)
- [unpkg](https://unpkg.com/browse/bootstrap/)

#### Vollständiges Template

```html
<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 5 Seite</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>Hallo, Bootstrap 5!</h1>
    
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### Methode 2: npm-Installation

#### npm verwenden

```bash
# Neues Projekt erstellen
npm init -y

# Bootstrap installieren
npm install bootstrap@5.3.3

# Popper.js installieren (falls benötigt)
npm install @popperjs/core
```

#### Im Projekt einbinden

```javascript
// main.js oder app.js
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

### Methode 3: Vite verwenden

#### 1. Vite-Projekt erstellen

```bash
npm create vite@latest my-bootstrap-app -- --template vanilla
cd my-bootstrap-app
npm install
```

#### 2. Bootstrap installieren

```bash
npm install bootstrap@5.3.3
```

#### 3. Bootstrap einbinden

```javascript
// main.js
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

```javascript
// vite.config.js
export default {
  plugins: [],
  resolve: {
    alias: {
      '@': '/src'
    }
  }
}
```

### Methode 4: Webpack verwenden

#### 1. Abhängigkeiten installieren

```bash
npm install bootstrap@5.3.3
npm install --save-dev webpack webpack-cli css-loader style-loader
```

#### 2. Webpack konfigurieren

```javascript
// webpack.config.js
module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js'
  },
  module: {
    rules: [
      {
        test: /\.css$/,
        use: ['style-loader', 'css-loader']
      }
    ]
  }
};
```

#### 3. Bootstrap einbinden

```javascript
// src/index.js
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

### Methode 5: Vue CLI / Nuxt verwenden

#### Vue CLI

```bash
# Vue-Projekt erstellen
npm install -g @vue/cli
vue create my-app

# Bootstrap installieren
cd my-app
npm install bootstrap@5.3.3

# In main.js einbinden
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

#### Nuxt 3

```bash
# Nuxt-Projekt erstellen
npx nuxi@latest init my-nuxt-app
cd my-nuxt-app
npm install

# Bootstrap installieren
npm install bootstrap@5.3.3

# In nuxt.config.ts konfigurieren
export default defineNuxtConfig({
  css: ['bootstrap/dist/css/bootstrap.min.css'],
  build: {
    transpile: ['bootstrap']
  }
})
```

### Methode 7: Quellcode herunterladen

#### Von GitHub herunterladen

```bash
# Repository klonen
git clone https://github.com/twbs/bootstrap.git
cd bootstrap

# Abhängigkeiten installieren
npm install

# Build erstellen
npm run dist
```

#### Direkt herunterladen

1. Besuchen Sie die [offizielle Bootstrap-Release-Seite](https://github.com/twbs/bootstrap/releases)
2. Laden Sie Source code (zip) herunter
3. Nach dem Entpacken binden Sie `dist/css/bootstrap.min.css` und `dist/js/bootstrap.bundle.min.js` ein

## Spezielle Konfigurationen

### RTL (Rechts-nach-Links) aktivieren

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.rtl.min.css" rel="stylesheet">
```

### Sass-Anpassung verwenden

```bash
npm install bootstrap@5.3.3
npm install --save-dev sass
```

```scss
// custom.scss
// Bootstrap-Funktionen importieren
@import "bootstrap/scss/functions";

// Variablen anpassen
$primary: #ff6b35;
$font-family-sans-serif: "Noto Sans SC", sans-serif;

// Andere Bootstrap-Teile importieren
@import "bootstrap/scss/variables";
@import "bootstrap/scss/mixins";
@import "bootstrap/scss/utilities";
@import "bootstrap/scss/root";
@import "bootstrap/scss/reboot";
@import "bootstrap/scss/type";
@import "bootstrap/scss/containers";
@import "bootstrap/scss/grid";
@import "bootstrap/scss/utilities/api";
// Weitere benötigte Komponenten hinzufügen...
```

```javascript
// vite.config.js
export default {
  css: {
    preprocessorOptions: {
      scss: {
        additionalData: `@import "bootstrap/scss/functions"; @import "custom-variables";`
      }
    }
  }
}
```

## Häufig gestellte Fragen

### Was ist Popper?

Popper ist eine Positionierungs-Engine für die Positionierung von Tooltips und Popovers. `bootstrap.bundle.min.js` enthält bereits Popper, eine separate Installation ist nicht erforderlich.

### Wird jQuery noch benötigt?

**Nein!** Bootstrap 5 wurde vollständig in reinem JavaScript neu geschrieben und benötigt keine Abhängigkeit mehr von jQuery.

### Wie verwendet man Icons?

Bootstrap 5 verwendet Bootstrap Icons:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css">

<i class="bi bi-github"></i>
<i class="bi bi-bootstrap-fill"></i>
```

### Wie ändert man die Theme-Farben?

Dies kann durch Überschreiben der CSS-Variablen erfolgen:

```css
:root {
  --bs-primary: #ff6b35;
  --bs-secondary: #6c757d;
  --bs-success: #198754;
}
```
