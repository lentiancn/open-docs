# Guía de Instalación de Webpack

Webpack es un empaquetador de módulos estáticos para aplicaciones JavaScript modernas. Esta guía cubre los métodos de instalación en todos los entornos.

## Información de Versión

| Versión | Estado | Descripción |
|---------|--------|-------------|
| Webpack 5.x | ✅ Actual | Última versión estable |
| Webpack 4.x | ❌ Obsoleto | No recomendado para nuevos proyectos |

> **Importante**: Esta guía está dirigida a **Webpack 5.x**. Se recomienda usar la última versión para nuevos proyectos.

## Métodos de Instalación

### Método 1: Instalación Local (Recomendado)

```bash
# Crear directorio del proyecto
mkdir my-webpack-project
cd my-webpack-project

# Inicializar npm
npm init -y

# Instalar webpack
npm install webpack --save-dev

# Instalar webpack-cli
npm install webpack-cli --save-dev
```

### Método 2: Instalación Global

```bash
npm install -g webpack webpack-cli
```

### Método 3: Usar npx

```bash
npx webpack --version
```

### Método 4: Crear Plantillas de Proyecto

```bash
# Usar create-react-app (tiene webpack incorporado)
npx create-react-app my-app

# Usar Vue CLI
npm install -g @vue/cli
vue create my-app

# Usar Vite
npm create vite@latest my-app -- --template vanilla
```

## Configuración de webpack.config.js

### Configuración Básica

```javascript
const path = require('path');

module.exports = {
  mode: 'development', // o 'production'
  
  // Archivo de entrada
  entry: './src/index.js',
  
  // Configuración de salida
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
    clean: true, // Limpiar directorio dist
  },
  
  // Cargadores
  module: {
    rules: [
      {
        test: /\.css$/i,
        use: ['style-loader', 'css-loader'],
      },
      {
        test: /\.(png|svg|jpg|jpeg|gif)$/i,
        type: 'asset/resource',
      },
    ],
  },
};
```

### Configuración de Producción

```javascript
const path = require('path');
const HtmlWebpackPlugin = require('html-webpack-plugin');

module.exports = {
  mode: 'production',
  entry: './src/index.js',
  output: {
    filename: '[name].[contenthash].js',
    path: path.resolve(__dirname, 'dist'),
    clean: true,
  },
  module: {
    rules: [
      {
        test: /\.css$/i,
        use: ['style-loader', 'css-loader'],
      },
      {
        test: /\.js$/i,
        exclude: /node_modules/,
        use: {
          loader: 'babel-loader',
          options: {
            presets: ['@babel/preset-env'],
          },
        },
      },
    ],
  },
  plugins: [
    new HtmlWebpackPlugin({
      template: './src/index.html',
    }),
  ],
  optimization: {
    splitChunks: {
      chunks: 'all',
    },
  },
};
```

### Configuración de Desarrollo

```javascript
const path = require('path');
const HtmlWebpackPlugin = require('html-webpack-plugin');

module.exports = {
  mode: 'development',
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
  },
  devServer: {
    static: './dist',
    hot: true,
    port: 3000,
    open: true,
  },
  plugins: [
    new HtmlWebpackPlugin({
      template: './src/index.html',
    }),
  ],
};
```

## Configuración de Cargadores

### Cargador CSS

```bash
npm install --save-dev style-loader css-loader
```

```javascript
module: {
  rules: [
    {
      test: /\.css$/i,
      use: ['style-loader', 'css-loader'],
    },
  ],
}
```

### Cargador Sass

```bash
npm install --save-dev sass sass-loader style-loader css-loader
```

```javascript
{
  test: /\.scss$/i,
  use: ['style-loader', 'css-loader', 'sass-loader'],
}
```

### Configuración de Babel

```bash
npm install --save-dev @babel/core @babel/preset-env babel-loader
```

```javascript
{
  test: /\.js$/i,
  exclude: /node_modules/,
  use: {
    loader: 'babel-loader',
    options: {
      presets: ['@babel/preset-env'],
    },
  },
}
```

### Configuración de TypeScript

```bash
npm install --save-dev typescript ts-loader
```

```javascript
{
  test: /\.tsx?$/,
  use: 'ts-loader',
  exclude: /node_modules/,
}
```

## Configuración de Plugins

### HtmlWebpackPlugin

```bash
npm install --save-dev html-webpack-plugin
```

```javascript
const HtmlWebpackPlugin = require('html-webpack-plugin');

plugins: [
  new HtmlWebpackPlugin({
    title: 'My App',
    template: './src/index.html',
    filename: 'index.html',
  }),
];
```

### MiniCssExtractPlugin

```bash
npm install --save-dev mini-css-extract-plugin
```

```javascript
const MiniCssExtractPlugin = require('mini-css-extract-plugin');

module.exports = {
  plugins: [
    new MiniCssExtractPlugin({
      filename: '[name].[contenthash].css',
    }),
  ],
  module: {
    rules: [
      {
        test: /\.css$/i,
        use: [MiniCssExtractPlugin.loader, 'css-loader'],
      },
    ],
  },
};
```

### DefinePlugin

```javascript
const webpack = require('webpack');

plugins: [
  new webpack.DefinePlugin({
    'process.env.NODE_ENV': JSON.stringify('production'),
  }),
];
```

## Servidor de Desarrollo

### Instalación

```bash
npm install --save-dev webpack-dev-server
```

### Configuración

```javascript
module.exports = {
  devServer: {
    static: {
      directory: path.join(__dirname, 'dist'),
    },
    compress: true,
    port: 3000,
    hot: true,
    open: true,
    historyApiFallback: true,
    proxy: {
      '/api': 'http://localhost:8080',
    },
  },
};
```

### Ejecución

```bash
# package.json
{
  "scripts": {
    "start": "webpack serve --mode development",
    "build": "webpack --mode production"
  }
}

npm start
```

## Problemas Comunes

### ¿La construcción falla?

Verifica los mensajes de error y asegúrate de que todas las dependencias estén instaladas:

```bash
npm install
npm audit fix
```

### ¿Módulo no encontrado?

Asegúrate de que resolve.extensions esté configurado correctamente:

```javascript
resolve: {
  extensions: ['.js', '.jsx', '.ts', '.tsx'],
}
```

### ¿Hot reload no funciona?

Asegúrate de que la configuración de devServer sea correcta:

```javascript
devServer: {
  hot: true,
  liveReload: true,
}
```
