# Docker Installationsanleitung

## Systemanforderungen

- Windows 10/11 64-bit
- macOS 10.15+
- Ubuntu 20.04+

## Installation

### Windows/macOS
Von https://www.docker.com/products/docker-desktop/

### Linux (Ubuntu)

```bash
sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io docker-compose-plugin
sudo systemctl start docker
```

## Verifizieren

```bash
docker --version
docker run hello-world
```
