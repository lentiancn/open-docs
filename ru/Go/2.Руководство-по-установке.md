# Руководство по установке Go

> Полное руководство по установке и настройке Go

---

## Системные требования

| Ресурс | Требование |
|---------|-----------|
| Операционная система | Windows, macOS, Linux, FreeBSD |
| Архитектура | amd64, arm64, 386 и др. |

## Способы установки

### Установка в Windows

#### Использование MSI-установщика

1. Скачать Go: https://go.dev/dl/
2. Запустить MSI-установщик
3. Следовать инструкциям мастера установки
4. Открыть командную строку, ввести `go version`

#### Использование Chocolatey

```powershell
choco install golang
```

### Установка в macOS

#### Использование Homebrew

```bash
brew install go
```

#### Использование DMG-пакета

1. Скачать пакет для macOS: https://go.dev/dl/
2. Открыть файл пакета и запустить
3. Переместить каталог Go в /usr/local/go: `sudo mv go /usr/local/`

### Установка в Linux

#### Использование пакетного менеджера

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install golang-go

# CentOS/RHEL
sudo yum install golang
```

#### Ручная установка

```bash
# Скачать бинарный файл Go
wget https://go.dev/dl/go1.22.0.linux-amd64.tar.gz

# Распаковать в /usr/local
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go1.22.0.linux-amd64.tar.gz

# Добавить в PATH
export PATH=$PATH:/usr/local/go/bin

# Проверить установку
go version
```

## Настройка окружения

### Настройка GOPATH

```bash
# Linux/macOS
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin

# Windows (PowerShell)
$env:GOPATH = "$env:USERPROFILE\go"
$env:PATH += ";$env:GOPATH\bin"
```

### Настройка прокси для Go Modules

```bash
# Использовать китайское зеркало
go env -w GOPROXY=https://goproxy.cn,direct

# Настроить прокси
go env -w GOPROXY=https://proxy.golang.org,direct
```

### Проверка установки

```bash
# Проверить версию
go version

# Посмотреть информацию об окружении
go env

# Запустить тест
go version
```

## Основные команды

```bash
# Инициализировать модуль
go mod init <имя-модуля>

# Скачать зависимости
go mod tidy

# Собрать проект
go build

# Запустить программу
go run main.go

# Запустить тесты
go test ./...

# Установить инструменты
go install
```

## Устранение неполадок

### Ошибка установки?

- Проверить совместимость с системой
- Убедиться, что скачанная версия соответствует архитектуре
- Проверить права доступа

### Команда не найдена?

- Проверить, добавлен ли Go в переменную PATH
- Переоткрыть терминал
- Проверить путь установки

### Медленная загрузка зависимостей?

- Настроить китайское зеркало
- Использовать VPN
