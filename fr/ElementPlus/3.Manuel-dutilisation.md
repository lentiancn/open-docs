# Manuel d'utilisation Element Plus

Ce manuel vous aider à démarrer rapidement avec Element Plus et à maîtriser les composants et fonctionnalités les plus couramment utilisés.

## Démarrage rapide

Avant d'utiliser Element Plus, assurez-vous que l'installation est complète.

### Utilisation de base

Utilisez les composants directement dans les composants Vue:

<template>
  <el-button>Bouton par défaut</el-button>
  <el-button type="primary">Bouton principal</el-button>
  <el-button type="success">Bouton succès</el-button>
  <el-button type="warning">Bouton avertissement</el-button>
  <el-button type="danger">Bouton danger</el-button>
</template>

## Composants courants

### Bouton

Utilisation de base:

<el-button>Par défaut</el-button>
<el-button type="primary">Principal</el-button>
<el-button type="success">Succès</el-button>
<el-button type="warning">Avertissement</el-button>
<el-button type="danger">Danger</el-button>

Différentes tailles:

<el-button size="large">Grand</el-button>
<el-button>Par défaut</el-button>
<el-button size="small">Petit</el-button>

État désactivé:

<el-button disabled>Désactivé</el-button>

Boutons avec icône:

<el-button icon="Search">Rechercher</el-button>
<el-button icon="Edit">Modifier</el-button>
<el-button icon="Delete">Supprimer</el-button>

### Entrée

Utilisation de base:

<el-input v-model="input" placeholder="Veuillez saisir le contenu" />

Avec icône:

<el-input v-model="input" prefix-icon="Search" />
<el-input v-model="input" suffix-icon="Calendar" />

Avec bouton effacer:

<el-input v-model="input" clearable />

Entrée mot de passe:

<el-input v-model="input" type="password" show-password />

Texte multiligne:

<el-input v-model="textarea" type="textarea" :rows="3" />

### Sélecteur

Sélecteur de base:

<el-select v-model="value" placeholder="Veuillez sélectionner">
  <el-option label="Option 1" value="1" />
  <el-option label="Option 2" value="2" />
</el-select>

Sélection multiple:

<el-select v-model="value" multiple placeholder="Veuillez sélectionner">
  <el-option label="Option 1" value="1" />
  <el-option label="Option 2" value="2" />
</el-select>

### Tableau

Tableau de base:

<el-table :data="tableData" stripe border>
  <el-table-column prop="date" label="Date" width="180" />
  <el-table-column prop="name" label="Nom" width="180" />
  <el-table-column prop="address" label="Adresse" />
</el-table>

Avec index:

<el-table-column type="index" label="Numéro" width="60" />

Colonnes fixes:

<el-table-column prop="date" label="Date" width="150" fixed />

### Formulaire

Formulaire de base:

<el-form :model="form" label-width="80px">
  <el-form-item label="Nom de l'activité">
    <el-input v-model="form.name" />
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="onSubmit">Soumettre</el-button>
  </el-form-item>
</el-form>

Validation de formulaire:

<el-form :model="ruleForm" :rules="rules" ref="ruleFormRef">
  <el-form-item label="Nom" prop="name">
    <el-input v-model="ruleForm.name" />
  </el-form-item>
</el-form>

### Dialogue

<el-dialog v-model="visible" title="Titre" width="30%">
  <span>Contenu</span>
</el-dialog>

### Message

import { ElMessage } from 'element-plus'

ElMessage.success('Message de succès')
ElMessage.warning('Message d'avertissement')
ElMessage.error('Message d'erreur')

### Menu

Menu horizontal:

<el-menu mode="horizontal">
  <el-menu-item index="1">Centre de traitement</el-menu-item>
  <el-menu-item index="2">Gestion des commandes</el-menu-item>
</el-menu>

Menu vertical:

<el-menu default-active="1">
  <el-menu-item index="1">Navigation un</el-menu-item>
  <el-menu-item index="2">Navigation deux</el-menu-item>
</el-menu>

### Onglets

<el-tabs v-model="activeName">
  <el-tab-pane label="Gestion des utilisateurs" name="first">Gestion des utilisateurs</el-tab-pane>
  <el-tab-pane label="Gestion de la configuration" name="second">Gestion de la configuration</el-tab-pane>
</el-tabs>

### Pagination

<el-pagination
  v-model:current-page="currentPage"
  :page-size="100"
  :total="1000"
  layout="total, prev, pager, next"
/>

## Système de mise en page

### Système de grille

Element Plus utilise un système de grille à 24 colonnes:

<el-row :gutter="20">
  <el-col :span="12"><div>12 colonnes</div></el-col>
  <el-col :span="12"><div>12 colonnes</div></el-col>
</el-row>

### Mise en page en conteneur

<el-container>
  <el-header>En-tête</el-header>
  <el-main>Principal</el-main>
  <el-footer>Pied de page</el-footer>
</el-container>

### Validation de formulaire

Validez toujours les entrées utilisateur et fournissez des messages d'erreur clairs.

### Internationalisation

import fr from 'element-plus/es/locale/lang/fr'
app.use(ElementPlus, { locale: fr })

## Problèmes courants

### Styles de composants non appliqués

Assurez-vous que le CSS est importé:

import 'element-plus/dist/index.css'

### Icônes non affichées

Vous devez installer la bibliothèque d'icônes:

npm install @element-plus/icons-vue

Puis enregistrer globalement:

import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}

### Importation à la demande ne fonctionne pas

Vérifiez que la configuration du plugin est correcte et assurez-vous d'utiliser le bon resolver.
