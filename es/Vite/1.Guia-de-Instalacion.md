# Vite Guía de Instalación

Esta guía te lleva a través de la instalación y configuración de Vite.

## Requisitos Previos

- Node.js (versión 14.18+ o 16+)
- npm, yarn o pnpm

## Instalación

### Crear Proyecto

```bash
# npm
npm create vite@latest my-project -- --template vue

# yarn
yarn create vite my-project -- --template vue

# pnpm
pnpm create vite my-project -- --template vue
```

Plantillas disponibles:
- `vanilla` - Vanilla JavaScript
- `vanilla-ts` - Vanilla TypeScript
- `vue` - Vue
- `vue-ts` - Vue + TypeScript
- `react` - React
- `react-ts` - React + TypeScript

### Proyecto Existente

Si deseas agregar Vite a un proyecto existente:

```bash
npm install -D vite
```

## Configuración

### Configuración Básica

Crea un archivo `vite.config.js`:

```javascript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [vue()],
  server: {
    port: 3000
  }
})
```

### TypeScript

Vite soporta TypeScript out of the box. Agrega `tsconfig.json`:

```json
{
  "compilerOptions": {
    "target": "ESNext",
    "module": "ESNext",
    "moduleResolution": "Node",
    "strict": true,
    "jsx": "preserve"
  },
  "include": ["src"]
}
```

## Desarrollo

### Iniciar Servidor de Desarrollo

```bash
npm run dev
```

### Build de Producción

```bash
npm run build
```

### Previsualizar Build de Producción

```bash
npm run preview
```

## Configuración de Plugins

### Vue

```bash
npm install @vitejs/plugin-vue
```

```javascript
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [vue()]
})
```

### React

```bash
npm install @vitejs/plugin-react
```

```javascript
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [react()]
})
```

## Configuración de Proxy

```javascript
export default defineConfig({
  server: {
    proxy: {
      '/api': {
        target: 'http://localhost:8000',
        changeOrigin: true
      }
    }
  }
})
```

## Variables de Entorno

Crea archivos `.env`:

```
VITE_API_URL=http://localhost:8000
```

Acceder en código:

```javascript
const apiUrl = import.meta.env.VITE_API_URL
```
