# Go ユーザーマニュアル

## 基本構文

### 変数宣言

Goには複数の変数宣言があります。基本の宣言方法はvarキーワードを使用することです：

```go
// 型を指定して変数を宣言
var name string = "Go"

// 型推論による宣言
var age = 10

// 一括宣言
var (
    firstName string
    lastName  string
    score     float64
)

// 短い宣言（関数内）
message := "Hello, World!"
```

Goでは、変数は宣言後使用する必要があります。使用しない場合はコンパイルエラーになります。一時的に使用しない変数を宣言する必要がある場合は、アンダースコア_を使用して受信できます：

```go
result, _ := someFunction()  // 最初の戻り値を無視
```

### 定数宣言

定数はconstキーワードを使用して宣言し、文字列、文字列、ブール値または数値にできます：

```go
const Pi = 3.14159
const (
    StatusOK = 200
    StatusNotFound = 404
)
```

### 基本的なデータ型

Goの基本的なデータ型には以下が含まれます：

- ブール型：bool（trueまたはfalse）
- 整数：int、int8、int16、int32、int64、および対応する符号なし型uintなど
- 浮動小数点：float32、float64
- 複素数：complex64、complex128
- 文字列：string

型変換は明示的に行う必要があります：

```go
var i int = 42
var f float64 = float64(i)
var s string = strconv.Itoa(i)  // 整数を文字列に変換
```

## 関数

### 関数定義

Go関数はfuncキーワードを使用して定義します：

```go
func greet(name string) string {
    return "Hello, " + name
}

// 複数の戻り値
func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}

// 名前付き戻り値
func split(sum int) (x, y int) {
    x = sum * 4 / 9
    y = sum - x
    return  // 戻り値を指定する必要はありません
}
```

### 可変長引数

関数は可変数の引数を受け取ることができます：

```go
func sum(nums ...int) int {
    total := 0
    for _, n := range nums {
        total += n
    }
    return total
}

// 使用
result := sum(1, 2, 3, 4, 5)
```

### クロージャ

Goはクロージャをサポートしています。これは関数が外部スコープの変数をキャプチャして変更できることを意味します：

```go
func counter() func() int {
    count := 0
    return func() int {
        count++
        return count
    }
}

// 使用
c := counter()
fmt.Println(c())  // 1
fmt.Println(c())  // 2
```

## 並行プログラミング

### Goroutine

GoroutineはGoの軽量な実行単位で、作成コストが非常に低いです：

```go
// goroutineを開始
go func() {
    fmt.Println("Hello from goroutine!")
}()

// 引数付きgoroutine
go func(msg string) {
    fmt.Println(msg)
}("Hello!")
```

Channelはgoroutine間の通信メカニズムです：

```go
// channelを作成
ch := make(chan string)

// データを送信
ch <- "message"

// データを受信
msg := <-ch

// バッファ付きchannel
bufferedCh := 10)
```


### 並行安全性

syncパッケージを使用した並行制御：

```go
var (
    mu    sync.Mutex
    count int
)

func increment() {
    mu.Lock()
    defer mu.Unlock()
    count++
}
```

## エラー処理

### エラー戻り値

Goは戻り値を使用してエラー処理を行いますこれがGoのイディomaticな方法です：

```go
result, err := doSomething()
if err != nil {
    return err
}
```

### カスタムエラー

errorsパッケージを使用してカスタムエラーを作成できます：

```go
import "errors"

func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}
```

### エラー包装

Go 1.13では、fmt.Errorfと%w動詞を使用したエラー包装が導入されました：

```go
if err != nil {
    return fmt.Errorf("failed to do something: %w", err)
}
```

## インターフェース

### インターフェース定義

Goのインターフェースは暗黙的に実装され、明示的な宣言は不要です：

```go
type Writer interface {
    Write([]byte) (int, error)
}

type Reader interface {
    Read([]byte) (int, error)
}

// 一般的な複合インターフェース
type ReadWriter interface {
    Reader
    Writer
}
```

### 空インターフェース

空インターフェースinterface{}は任意の型の値を格納できます：

```go
var i interface{} = 42
var s interface{} = "hello"
var m interface{} = map[string]int{"a": 1}
```

Go 1.18ではジェネリクスが導入され、interface{}の代わりにanyを使用できます：

```go
var i any = 42
```

## テスト

### ユニットテスト

Goには組み込みのテストフレームワークがあります。テストファイルは_test.goで終わります：

```go
import "testing"

func TestAdd(t *testing.T) {
    result := Add(2, 3)
    if result != 5 {
        t.Errorf("Add(2, 3) = %d; want 5", result)
    }
}
```

テストの実行：

```bash
go test ./...
go test -v ./...
go test -cover ./...
```

### ベンチマークテスト

ベンチマーク関数はBenchmarkで始まります：

```go
func BenchmarkHello(b *testing.B) {
    for i := 0; i < b.N; i++ {
        Hello()
    }
}
```

ベンチマークテストの実行：

```bash
go test -bench=.
```
