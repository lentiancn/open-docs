# OpenCode 使用手册

OpenCode 基本使用方法和技巧。

## 快速开始

### 提问

你可以让 OpenCode 解释代码库：

```
How is authentication handled in @packages/functions/src/api/index.ts
```

### 添加功能

让 OpenCode 添加新功能前，建议先让它创建计划：

1. 按 **Tab** 切换到 Plan 模式
2. 描述你想要的功能
3. 查看计划后可提供反馈或补充细节
4. 满意后按 **Tab** 切换回 Build 模式执行

### 直接修改

对于更直接的修改，可以直接让 OpenCode 执行：

```
We need to add authentication to the /settings route. Take a look at how this is handled in the /notes route and implement the same logic.
```

### 撤销更改

如果对修改不满意，可以使用撤销命令：

```bash
/undo
```

OpenCode 会恢复更改并重新显示你的原始消息，你可以在此基础上调整提示。

### 重做

可以使用 `/redo` 命令重做更改。

## 分享会话

```bash
/share
```

这会创建一个指向当前会话的链接并复制到剪贴板。
