# Einführung

Bootstrap ist das weltweit beliebteste Open-Source-Framework für die frontent-Entwicklung mit HTML, CSS und JavaScript. Es wurde von den Twitter-Designern Mark Otto und Jacob Thornton entwickelt und bietet eine Fülle von Komponenten sowie leistungsstarke JavaScript-Plugins, mit denen Entwickler schnell responsive Web-Anwendungen erstellen können.

Das Kernkonzept von Bootstrap ist „Mobile First" – von Anfang an wurde die Anpassung an verschiedene Geräte berücksichtigt. Mit Bootstrap kannst du mühelos Websites erstellen, die auf dem Desktop, Tablet und Smartphone gleichermaßen gut aussehen.

## Entwicklungsgeschichte

Bootstrap trug ursprünglich den Namen Twitter Blueprint und wurde 2011 als Open Source veröffentlicht. 2012 kam Bootstrap 2 mit responsivem Layout heraus. 2013 wurde Bootstrap 3 komplett überarbeitet – mit einem flachen Design. 2015 folgte Bootstrap 4, das Sass als CSS-Präprozessor einführte. Und 2021 erschien Bootstrap 5 ohne jQuery-Abhängigkeit, das nun vollständig auf nativem JavaScript basiert.

## Kernfunktionen

### Responsives Layout

Das Grid-System ist eine der stärksten Funktionen von Bootstrap. Mit einem 12-Spalten-Layout kannst du ganz einfach Seitenlayouts erstellen, die sich an verschiedene Bildschirmgrößen anpassen. Bootstrap 5 bietet sechs Breakpoints: xs, sm, md, lg, xl und xxl – jeweils für unterschiedliche Gerätebreiten.

### Umfangreiche Komponenten

Bootstrap bringt jede Menge direkt einsetzbare UI-Komponenten mit: Navbars, Cards, Buttons, Formulare, Modals, Karussells, Dropdowns und mehr. Alle Komponenten sind sorgfältig gestaltet und bieten eine gute Optik sowie interaktive Nutzererfahrung.

### Leistungsstarke JavaScript-Plugins

Bootstrap enthält über 20 JavaScript-Plugins – etwa Modals, Karussells, Tooltips, Popovers und Accordions. Diese Plugins sind optional, du kannst sie bei Bedarf einbinden und so die Projektgröße gering halten.

### Sass-basiert

Bootstrap 5 nutzt Sass als CSS-Präprozessor und bietet zahlreiche Sass-Variablen und Mixins. Damit kannst du Bootstraps Farben, Schriftgrößen und andere Stile ganz einfach anpassen.

### Eigene Themes

Über Bootstraps Theme-System kannst du eigene Designstile erstellen. Der offizielle Bootstrap Themes Marketplace hält viele fertige Themes bereit.

### Barrierefreiheit

Die Bootstrap-Komponenten sind von Grund auf barrierefrei konzipiert – mit Tastaturnavigation, Screenreader-Kompatibilität und mehr. So kannst du inklusivere Websites bauen.

## Neue Funktionen in Bootstrap 5

Bootstrap 5 bringt gegenüber Bootstrap 4 viele wichtige Änderungen:

- jQuery ist raus, natives JavaScript ist drin
- Verbessertes Grid-System mit neuem XXL-Breakpoint
- Neue Formularsteuerelemente代替以往的表单验证
- Bessere Unterstützung für benutzerdefinierte Attribute
- Icons wechseln von Font Awesome zu Bootstrap Icons
- RTL-Unterstützung (Rechts-nach-Links)

## Anwendungsbereiche

- Schnelle Prototypen-Entwicklung
- Interne Unternehmensverwaltungssysteme
- E-Commerce-Websites
- Blogs und Content-Seiten
- Admin-Oberflächen
- Responsive Website-Entwicklung

Als eines der beliebtesten CSS-Frameworks weltweit hat Bootstrap eine riesige Community und reichhaltige Lernressourcen. Es gehört zu den unverzichtbaren Fähigkeiten für Frontend-Entwickler.
