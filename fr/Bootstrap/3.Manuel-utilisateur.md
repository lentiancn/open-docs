# Manuel utilisateur Bootstrap

> Composants principaux et utilisation

---

## Système de grille

### Structure de base

```html
<div class="container">
  <div class="row">
    <div class="col">Colonne 1</div>
    <div class="col">Colonne 2</div>
    <div class="col">Colonne 3</div>
  </div>
</div>
```

### Colonnes responsives

```html
<!-- Plein écran sur mobile, 6 colonnes sur écrans moyens -->
<div class="col-12 col-md-6">Contenu</div>

<!-- 4 colonnes sur grands écrans -->
<div class="col-lg-4">Contenu</div>
```

## Boutons

### Boutons de base

```html
<button class="btn btn-primary">Primaire</button>
<button class="btn btn-secondary">Secondaire</button>
<button class="btn btn-success">Succès</button>
<button class="btn btn-danger">Danger</button>
<button class="btn btn-warning">Avertissement</button>
<button class="btn btn-info">Info</button>
<button class="btn btn-light">Clair</button>
<button class="btn btn-dark">Sombre</button>
```

### Tailles de boutons

```html
<button class="btn btn-primary btn-lg">Grand</button>
<button class="btn btn-primary">Par défaut</button>
<button class="btn btn-primary btn-sm">Petit</button>
```

## Formulaires

### Formulaire de base

```html
<form>
  <div class="mb-3">
    <label class="form-label">Email</label>
    <input type="email" class="form-control" placeholder="nom@exemple.com">
  </div>
  <div class="mb-3">
    <label class="form-label">Mot de passe</label>
    <input type="password" class="form-control">
  </div>
  <button type="submit" class="btn btn-primary">Soumettre</button>
</form>
```

### Liste déroulante

```html
<select class="form-select">
  <option selected>Sélectionner...</option>
  <option value="1">Option 1</option>
  <option value="2">Option 2</option>
</select>
```

## Navigation

### Barre de navigation

```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">Marque</a>
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav">
      <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="navbarNav">
      <ul class="navbar-nav">
        <li class="nav-item">
          <a class="nav-link active" href="#">Accueil</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">À propos</a>
        </li>
      </ul>
    </div>
  </div>
</nav>
```

```html
    <h5 class="card-title">Titre de la carte</h5>

  </div>

```

## Modales

```html
<!-- Bouton déclencheur -->
<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#exampleModal">
  Ouvrir la modale
</button>

<!-- Modale -->
<div class="modal fade" id="exampleModal" tabindex="-1">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title">Titre</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
      </div>
      <div class="modal-body">
        Contenu
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Fermer</button>
        <button type="button" class="btn btn-primary">Enregistrer</button>
      </div>
    </div>
  </div>
</div>
```
