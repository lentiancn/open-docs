# IBM Informix Installation Guide

## System Requirements

**Supported Platforms:**
- Linux (Red Hat, Ubuntu, SUSE, etc.)
- Windows Server
- AIX
- HP-UX
- Solaris
- macOS
- ARM (for edge devices)

**Minimum Requirements:**
- RAM: 1GB minimum (8GB recommended)
- Disk: 2GB minimum for installation
- CPU: Compatible with x86, PowerPC, ARM architectures

## Linux Installation Steps

### Step 1: Download the Installer

Visit IBM Passport Advantage or IBM Fix Central and download the appropriate Informix installer package.

### Step 2: Prepare the System

```bash
# Create informix user and group
groupadd informix
useradd -g informix -d /home/informix informix

# Create installation directory
mkdir -p /opt/IBM/informix
chown informix:informix /opt/IBM/informix
```

### Step 3: Extract the Installer

```bash
tar -xf informix.tar
cd informix
```

### Step 4: Run the Installation

```bash
# As root or with sudo
./install
```

For newer versions (14.10+), use the IBM Installation Manager or:

```bash
chmod +x ids_install
./ids_install
```

Follow the prompts:
- Accept the license agreement
- Choose installation path (/opt/IBM/informix)
- Select components to install

### Step 5: Configure Environment Variables

Add to `~/.bashrc` or `/etc/profile`:

```bash
export INFORMIXDIR=/opt/IBM/informix
export PATH=$INFORMIXDIR/bin:$PATH
export ONCONFIG=onconfig
export INFORMIXSERVER=your_server_name
```

Apply the changes:

```bash
source ~/.bashrc
```

### Step 6: Configure sqlhosts File

```bash
cd $INFORMIXDIR/etc
cp sqlhosts.std sqlhosts
vi sqlhosts
```

Add your server entry:

```
ol_informix onsoctcp localhost ol_informix
```

### Step 7: Initialize the Database Server

```bash
# Initialize Informix with default settings
oninit -ivy
```

### Step 8: Verify Installation

```bash
# Check server status
onstat -

# Verify initialization
ls -la $INFORMIXDIR/storage/
```

## Windows Installation Steps

### Step 1: Download the Windows Installer

Download from IBM Passport Advantage.

### Step 2: Run the Installer

- Double-click the `.exe` file
- Follow the installation wizard

### Step 3: Configuration

- Use the Informix Configuration Wizard
- Set up the Windows service

### Step 4: Environment Setup

Set `INFORMIXDIR` in System Properties:
- Right-click My Computer → Properties
- Advanced System Settings → Environment Variables
- Add `INFORMIXDIR` and `INFORMIXSERVER`

## Post-Installation Verification

```bash
# Check server status
onstat -

# Start the server (if not already running)
oninit

# Stop the server
onmode -ky

# Connect to the database
dbaccess
```

## Common Post-Installation Tasks

### Create Additional Storage Spaces

```bash
onspaces -c -d datadbs -p /path/to/data_chunk -s 500000
```

### Configure Logical Logs

```bash
onparams -a -d datadbs -s 10000
```

## Troubleshooting

### Server Won't Start

- Verify environment variables are set correctly
- Check `onconfig` and `sqlhosts` files
- Review error logs in `$INFORMIXDIR/tmp/`
- Ensure sufficient disk space and permissions

### Connection Issues

- Check firewall settings (default port 9088)
- Verify `INFORMIXSERVER` matches sqlhosts configuration
- Ensure `sqlhosts` has correct network settings
