# Guía de Instalación de Element Plus

Esta guía detalla cómo instalar y usar Element Plus en varios proyectos.

## Requisitos del Entorno

Element Plus tiene los siguientes requisitos:

- Vue 3.0.0+
- Node.js 14.0.0+
- Navegadores modernos (Chrome, Firefox, Safari, Edge)

## Métodos de Instalación

### Usando el Gestor de Paquetes

Se recomienda instalar usando npm, yarn o pnpm:

```bash
# npm
npm install element-plus

# yarn
yarn add element-plus

# pnpm
pnpm add element-plus
```

## Importación Completa

Si no te importa el tamaño del paquete, usar la importación completa será más conveniente.

En main.ts:

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')
```

### Soporte para Volar

Si usas Volar, por favor añade la definición de tipo de componente global en compilerOptions.types en tsconfig.json:

```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

## Importación bajo Demanda

La importación bajo demanda puede reducir significativamente el tamaño del proyecto. Necesitas usar plugins adicionales para importar los componentes que uses.

### Importación Automática (Recomendado)

Primero instala los plugins:

```bash
npm install -D unplugin-vue-components unplugin-auto-import
# o
yarn add -D unplugin-vue-components unplugin-auto-import
# o
pnpm add -D unplugin-vue-components unplugin-auto-import
```

#### Configuración de Vite

En vite.config.ts:

```typescript
import { defineConfig } from 'vite'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    AutoImport({ resolvers: [ElementPlusResolver()] }),
    Components({ resolvers: [ElementPlusResolver()] }),
  ],
})
```

#### Configuración de Webpack

En webpack.config.js:

```javascript
const AutoImport = require('unplugin-auto-import/webpack')
const Components = require('unplugin-vue-components/webpack')
const { ElementPlusResolver } = require('unplugin-vue-components/resolvers')

module.exports = {
  plugins: [
    AutoImport({ resolvers: [ElementPlusResolver()] }),
    Components({ resolvers: [ElementPlusResolver()] }),
  ],
}
```

### Uso con Nuxt

Para usuarios de Nuxt, solo necesitas instalar @element-plus/nuxt:

```bash
npm install -D @element-plus/nuxt
```

Luego añade la configuración en nuxt.config.ts:

```typescript
export default defineNuxtConfig({
  modules: ['@element-plus/nuxt'],
})
```

### Importación Manual

Element Plus proporciona funcionalidades de Tree Shaking basadas en ES Module.

Necesitas instalar unplugin-element-plus para importar estilos:

```bash
npm install -D unplugin-element-plus
```

En componentes:

```vue
<template>
  <el-button>Botón</el-button>
</template>

<script>
import { ElButton } from 'element-plus'

export default {
  components: { ElButton },
}
</script>
```

Configuración de Vite:

```typescript
import { defineConfig } from 'vite'
import ElementPlus from 'unplugin-element-plus/vite'

export default defineConfig({
  plugins: [ElementPlus()],
})
```

## Configuración Global

Al registrar Element Plus, puedes pasar un objeto de configuración global con size y zIndex:

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import App from './App.vue'

const app = createApp(App)
app.use(ElementPlus, { size: 'small', zIndex: 3000 })
```

Opciones de configuración:
- size: Establecer tamaño predeterminado para componentes de formulario (large, default, small)
- zIndex: Establecer z-index para componentes emergentes, el valor predeterminado es 2000

Para más información, visita el [sitio web oficial de Element Plus](https://element-plus.org/).
