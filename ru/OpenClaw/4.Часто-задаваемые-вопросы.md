# Часто задаваемые вопросы

## Основное

### Что такое OpenClaw?

Инструмент для подключения мессенджеров к ИИ-ассистентам.

### Это бесплатно?

Софт бесплатный (открытый код). Платить нужно за:
- Сервер/компьютер
- API-ключи ИИ-сервисов

### Где работает?

- Десктоп: macOS, Linux, Windows
- Мобильные: iOS, Android
- Мессенджеры: WhatsApp, Telegram, Discord, iMessage, Signal, Slack и ещё 20+

## Установка

### Не устанавливается?

1. Проверьте Node.js: `node --version` (нужно 22+)
2. Запустите диагностику: `openclaw doctor`
3. Посмотрите логи: `openclaw logs`

### Работает на Raspberry Pi?

Да. Raspberry Pi 4 + Raspberry Pi OS / Ubuntu.

### Нужен Mac?

Нет. Только для iMessage нужен Mac.

## Настройка

### Где конфиг?

`~/.openclaw/openclaw.json`

### Как ограничить доступ?

Настройка `dmPolicy`: pairing, allowlist, open, disabled

## Использование

### Добавить канал?

```bash
openclaw channels login whatsapp
```

### Несколько каналов?

Да, один Gateway может.
