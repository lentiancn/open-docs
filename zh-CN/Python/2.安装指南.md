# Python 安装指南

> 完整的环境配置和安装教程

---

## 环境要求

| 环境 | 要求 |
|------|------|
| 操作系统 | Windows, macOS, Linux |
| Python 版本 | 3.8+ |
| 硬盘空间 | 约 100MB |

---

## 安装步骤

### Windows 安装

#### 方法一：官方安装包

1. 下载 Python：https://www.python.org/downloads/
2. 运行安装程序
3. **注意**：勾选 "Add Python to PATH"
4. 验证安装：`python --version`

#### 方法二：Microsoft Store（仅 Windows 10+）

```
在 Microsoft Store 搜索 "Python"
```

#### 方法三：Chocolatey

```powershell
choco install python
```

### macOS 安装

#### 方法一：brew（推荐）

```bash
brew install python@3.12
```

#### 方法二：官方安装包

从 https://www.python.org/downloads/ 下载 .pkg 安装包

### Linux 安装

#### apt（Ubuntu/Debian）

```bash
sudo apt update
sudo apt install python3 python3-pip
```

#### yum（CentOS/RHEL）

```bash
sudo yum install python3
```

#### pyenv 管理多版本（推荐）

```bash
# 安装 pyenv
curl https://pyenv.run | bash

# 安装 Python
pyenv install 3.12.0

# 设置全局版本
pyenv global 3.12.0
```

## 包管理

### pip 使用

```bash
# 升级 pip
python -m pip install --upgrade pip

# 安装包
pip install requests

# 安装requirements.txt中的包
pip install -r requirements.txt

# 卸载包
pip uninstall requests

# 查看已安装的包
pip list
```

### 虚拟环境

```bash
# 创建虚拟环境
python -m venv myenv

# 激活虚拟环境
# Windows
myenv\Scripts\activate
# macOS/Linux
source myenv/bin/activate

# 退出虚拟环境
deactivate
```

## 常见问题

### pip 安装失败？

1. 升级 pip：`python -m pip install --upgrade pip`
2. 使用国内镜像源：`pip install -i https://pypi.tuna.tsinghua.edu.cn/simple 包名`

### Python 命令找不到？

Windows：确保 Python 安装目录在 PATH 中
macOS/Linux：使用 `python3` 代替 `python`
