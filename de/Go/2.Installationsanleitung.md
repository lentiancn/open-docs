# Go Installationsanleitung

> Vollständige Anleitung zur Installation und Konfiguration von Go

---

| Ressource | Anforderung |
|-----------|-------------|
| Betriebssystem | Windows, macOS, Linux, FreeBSD |

| Speicherplatz | ca. 100 MB |

## Installationsmethoden

### Windows Installation

#### MSI-Installationspaket

1. Go herunterladen: https://go.dev/dl/
2. MSI-Installationspaket ausführen
3. Installationsassistenten folgen
4. Eingabeaufforderung öffnen, `go version` eingeben

#### Chocolatey

```powershell
choco install golang
```

### macOS Installation

#### Homebrew

```bash
brew install go
```

#### DMG-Installationspaket

1. macOS-Paket herunterladen: https://go.dev/dl/
2. Paket öffnen und ausführen
3. Go-Verzeichnis nach /usr/local/go verschieben: `sudo mv go /usr/local/`

### Linux Installation

#### Paketmanager

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install golang-go

# CentOS/RHEL
sudo yum install golang
```

#### Manuelle Installation

```bash
# Go-Binary herunterladen
wget https://go.dev/dl/go1.22.0.linux-amd64.tar.gz

# Nach /usr/local entpacken
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go1.22.0.linux-amd64.tar.gz

# Zu PATH hinzufügen
export PATH=$PATH:/usr/local/go/bin

# Installation überprüfen
go version
```

## Umgebungskonfiguration

### GOPATH einrichten

```bash
# Linux/macOS
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin

# Windows (PowerShell)
$env:GOPATH = "$env:USERPROFILE\go"
$env:PATH += ";$env:GOPATH\bin"
```

### Go Module Proxy konfigurieren

```bash
# Chinesischen Spiegel verwenden
go env -w GOPROXY=https://goproxy.cn,direct

# Proxy einrichten
go env -w GOPROXY=https://proxy.golang.org,direct
```

### Installation überprüfen

```bash
# Version anzeigen
go version

# Umgebungsinformationen anzeigen
go env

# Test ausführen
go version
```

## Häufige Befehle

```bash
# Modul initialisieren
go mod init <modulname>

# Abhängigkeiten herunterladen
go mod tidy

# Projekt bauen
go build

# Programm ausführen
go run main.go

# Tests ausführen
go test ./...

# Tools installieren
go install
```

## Fehlerbehebung

### Installation fehlgeschlagen?

- Systemkompatibilität prüfen
- Heruntergeladene Version muss zur Systemarchitektur passen
- Berechtigungen prüfen

### Befehl nicht gefunden?

- Go wurde zu PATH-Umgebungsvariable hinzugefügt?
- Terminal neu öffnen
- Installationspfad prüfen

### Langsame Abhängigkeitsdownloads?

- Chinesischen Spiegelproxy konfigurieren
- VPN verwenden
