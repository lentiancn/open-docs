# Руководство по установке

Jira предлагает несколько вариантов установки для удовлетворения потребностей команд разного размера и технических возможностей. Это руководство содержит подробные инструкции как для облачного развёртывания, так и для локального развёртывания Data Center.

## Установка облачной версии

Облачная версия — самый быстрый способ начать работу с Jira, не требующий управления инфраструктурой.

### Регистрация учётной записи

Сначала посетите веб-сайт Jira и нажмите кнопку бесплатной регистрации. Введите адрес рабочей электронной почты для создания учётной записи. Atlassian автоматически определит участников команды и упростит совместную работу.

### Выбор плана

Облачная версия предлагает четыре уровня цен:

**План Free** — бесплатно навсегда для доступа 10 пользователей. Включает неограниченное количество целей и проектов, списки задач и представления досок, 100 правил автоматизации в месяц и 2 ГБ хранилища.

**План Standard** — стоит 7,91 доллара США за пользователя в месяц. Включает роли пользователей и разрешения, внешнее сотрудничество, многорегиональное хранение данных, 1700 правил автоматизации в месяц и 250 ГБ хранилища.

**План Premium** — стоит 14,54 доллара США за пользователя в месяц. Включает планирование между командами и управление зависимостями, настраиваемые процессы утверждения, неограниченное хранилище, круглосуточную поддержку и Соглашение об уровне услуг с доступностью 99,9%.

**План Enterprise** — предназначен для крупных предприятий. Включает расширенную аналитику и безопасность корпоративного уровня, поддержку нескольких сайтов, неограниченное количество правил автоматизации и Соглашение об уровне услуг с доступностью 99,95%.

### Создание сайта

После регистрации система проведёт вас через процесс создания первого сайта Jira. Вы можете создать сайт для своей организации или личного использования и выбрать шаблоны проектов, соответствующие потребностям вашей команды.

### Добавление пользователей

После создания сайта вы можете добавлять участников команды через панель администратора. Пользователи могут быть бесплатными или платными в зависимости от выбранного плана.

## Локальное развёртывание Data Center

Для предприятий, требующих строгого контроля данных или имеющих особые требования к безопасности, доступна локальная версия Data Center.

### Системные требования

Для версии Data Center требуется следующее:

Поддержка операционных систем включает Linux (Red Hat Enterprise Linux, Ubuntu Server, CentOS и др.) и Windows Server.

Поддержка баз данных включает PostgreSQL, Oracle, Microsoft SQL Server и другие корпоративные базы данных.

Оперативная память: рекомендуется не менее 8 ГБ, для производственных сред — 16 ГБ и более.

Дисковое пространство: зависит от объёма данных, рекомендуется не менее 100 ГБ свободного места.

### Шаги установки

Сначала загрузите установщик Data Center с веб-сайта Atlassian. Запустите установщик и следуйте мастеру, чтобы выбрать каталог установки и настроить подключения к базам данных. После установки откройте административный интерфейс через браузер для начальной настройки.

### Миграция данных

При миграции с более ранней версии используйте инструменты импорта и экспорта Jira для переноса данных. Рекомендуется создать резервную копию всех данных перед миграцией и проверить результаты в тестовой среде.

## Мобильные приложения

Jira предлагает мобильные приложения для iOS и Android, позволяющие просматривать и обновлять проблемы в любом месте. Мобильные приложения поддерживают просмотр проблем, создание комментариев, обновления статуса и многое другое, подходят для участников команды, которым нужно работать в движении.

## Начальная настройка

Независимо от варианта развёртывания начальная настройка включает:

Настройка структуры проекта, создание проектов и досок для вашей команды. Настройка рабочих процессов, определение статусов проблем и правил перехода. Добавление участников команды и установка разрешений. Интеграция с другими инструментами, такими как Slack и GitHub. Создание правил автоматизации для сокращения ручных операций.
