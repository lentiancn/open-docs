# IBM Informix

IBM Informix 是一个功能强大的嵌入式关系型数据库管理系统（RDBMS），由 IBM 公司开发和维护。

## 文档概述

本目录包含 IBM Informix 的多语言技术文档：

- **简介**：产品概述、特性、应用场景和版本信息
- **安装指南**：Linux 和 Windows 平台的安装步骤
- **使用手册**：基本操作命令、SQL 命令和配置说明
- **常见问题**：常见故障的处理方法和解决方案

## 版本信息

- 最新版本：Informix 14.10
