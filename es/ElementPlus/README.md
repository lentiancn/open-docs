# Element Plus

[Element Plus](https://element-plus.org/) es una biblioteca de componentes UI basada en Vue 3.

## Características

- Soporte Vue 3 con Composition API
- 70+ componentes
- Soporte TypeScript
- Importación bajo demanda
- Internacionalización
- Temas personalizables

## Inicio Rápido

```bash
npm install element-plus
```
