# JDK インストールガイド

このガイドでは、Windows、macOS、LinuxにJDK（Java Development Kit）をインストール方法について説明します。

## バージョン概要

| バージョン | タイプ | リリース日 | サポート期限 |
|---------|------|--------------|---------------|
| JDK 25 | 最新 | 2026 | 6ヶ月 |
| JDK 21 | LTS | 2023年9月 | 2029年9月 |
| JDK 17 | LTS | 2021年9月 | 2029年9月 |
| JDK 11 | LTS | 2018年9月 | 2031年9月 |
| JDK 8 | レガシー | 2014年3月 | 2030年12月 |

**推奨：** 本番環境にはJDK 21（LTS）またはJDK 17（LTS）を使用してください。最新機能にはJDK 25を使用してください。

## Windowsへのインストール

### オプション1：インストーラー (.exe)

1. OracleまたはAdoptiumからJDKをダウンロード
2. インストーラーを実行
3. 「次へ」をクリックしてウィザードを完了
4. インストールパスをメモ（デフォルト：`C:\Program Files\Java\jdk-21`）

### オプション2：ZIPアーカイブ

1. OracleまたはAdoptiumからZIPをダウンロード
2. 任意の場所に展開（例：`C:\Java\jdk-21`）
3. インストール不要

### 環境変数の設定

```cmd
# システムプロパティ → 詳細設定 → 環境変数

# システム変数 → 新規
JAVA_HOME = C:\Program Files\Java\jdk-21

# Path変数を編集し、追加：
%JAVA_HOME%\bin
```

### インストールの確認

```cmd
java -version
javac -version
```

## macOSへのインストール

### オプション1：DMGインストーラー

1. OracleまたはAdoptiumからDMGをダウンロード
2. DMGファイルを開く
3. パッケージインストーラーをダブルクリック
4. ウィザードに従う

### オプション2：Homebrew

```bash
# Homebrewがインストールされていない場合
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# JDK 21をインストール
brew install openjdk@21

# システムJavaにリンク
sudo ln -sfn $(brew --prefix)/opt/openjdk@21/libexec/openjdk.jdk /Library/Java/JavaVirtualMachines/openjdk-21.jdk

# PATHに追加
echo 'export JAVA_HOME=$(/usr/libexec/java_home -v 21)' >> ~/.zshrc
source ~/.zshrc
```

### オプション3：SDKMAN

```bash
# SDKMANをインストール
curl -s "https://get.sdkman.io" | bash
source ~/.sdkman/bin/sdkman-init.sh

# JDK 21をインストール
sdk install java 21.0.5-tem
```

### インストールの確認

```bash
java -version
javac -version
```

## Linuxへのインストール

### Ubuntu/Debian

```bash
# パッケージリストを更新
sudo apt update

# OpenJDK 21をインストール
sudo apt install openjdk-21-jdk

# JAVA_HOMEを設定
echo 'export JAVA_HOME=/usr/lib/jvm/java-21-openjdk-amd64' >> ~/.bashrc
source ~/.bashrc
```

### RHEL/CentOS/Fedora

```bash
# JDK 21をインストール
sudo dnf install java-21-openjdk-devel

# JAVA_HOMEを設定
echo 'export JAVA_HOME=/usr/lib/jvm/java-21-openjdk' >> ~/.bashrc
source ~/.bashrc
```

### 手動インストール（すべてのLinux）

```bash
# JDKをダウンロード
wget https://download.oracle.com/java/21/latest/jdk-21_linux-x64_bin.tar.gz

# 解凍
sudo tar -xzf jdk-21_linux-x64_bin.tar.gz -C /usr/local/

# JAVA_HOMEを設定
echo 'export JAVA_HOME=/usr/local/jdk-21' >> ~/.bashrc
source ~/.bashrc
```

### インストールの確認

```bash
java -version
javac -version
```

## 複数のJDKバージョン

### macOSでのJDK切り替え

```bash
# インストールされたJDKを一覧表示
/usr/libexec/java_home -V

# 特定のバージョンに切り替え
export JAVA_HOME=$(/usr/libexec/java_home -v 17)
```

### LinuxでのJDK切り替え

```bash
# update-alternativesを使用（Ubuntu/Debian）
sudo update-alternatives --config java

# 手動切り替え
export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64
export PATH=$JAVA_HOME/bin:$PATH
```

## IDE設定

### IntelliJ IDEA

1. File → Project Structure → Project
2. SDKを選択：Add JDK → インストール済みJDKを選択
3. Apply → OK

### VS Code

1. 「Extension Pack for Java」をインストール
2. Ctrl+Shift+P → Java: Configure Java Runtime
3. インストール済みJDKを選択

### Eclipse

1. Window → Preferences → Java → Installed JREs
2. Add → Standard VM
3. JDKインストールフォルダを参照
4. デフォルトに設定

## 一般的な問題

### "java"コマンドが認識されない

- JAVA_HOMEが正しく設定されているか確認
- PATHに`%JAVA_HOME%\bin`が含まれているか確認
- ターミナル/コマンドプロンプトを再起動

### ポート8080が使用中

```bash
# Windows
netstat -ano | findstr :8080

# Linux/macOS
lsof -i :8080
```

### JAVA_HOMEがJREではなくJDKを指している

- JAVA_HOMEがJDKを指していることを確認（`bin/javac`を含む）
- JREではない（`bin/java`のみを含む）
