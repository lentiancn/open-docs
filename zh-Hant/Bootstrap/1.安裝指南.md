## 簡介

### 什麼是 Bootstrap？

Bootstrap 是全球最受歡迎的前端開源工具包，由 Twitter 開發和維護。它是一個用於快速開發響應式網站和 Web 應用程式的 CSS、JavaScript 框架。

### Bootstrap 版本

| 版本 | 發布日期 | 狀態 |
|---------|----------|------|
| Bootstrap 5.3 | 2023年5月 | 當前穩定版 |
| Bootstrap 5.2 | 2022年9月 | 支援 |
| Bootstrap 5.1 | 2021年8月 | 支援 |

## 安裝方法

### 方法一：使用 CDN（最簡單）

```html
<!DOCTYPE html>
<html lang="zh-Hant">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 入門</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>Hello, Bootstrap!</h1>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### 方法二：使用 npm

```bash
mkdir bootstrap-project
cd bootstrap-project
npm init -y
npm install bootstrap
```

### 方法三：下載原始碼

訪問 https://getbootstrap.com/ 下載預編譯的 CSS 和 JS 檔案。

## 在 Windows 上安裝

### 使用 CDN（推薦）

1. 建立新的 HTML 檔案
2. 在 `<head>` 中新增 Bootstrap CSS 連結
3. 在 `<body>` 結尾新增 Bootstrap JS 腳本

---

## 在 macOS 上安裝

1. 安裝 Homebrew
2. 安裝 Node.js
3. 建立專案並安裝 Bootstrap

---

## 在 Linux 上安裝

```bash
sudo apt update
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs
npm install bootstrap
```

## 驗證安裝

建立測試 HTML 檔案，如果看到藍色按鈕和綠色提示框，表示安裝成功。

## 建立第一個專案

```html
<!DOCTYPE html>
<html lang="zh-Hant">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <div class="container mt-5">
        <h1>我的 Bootstrap 網站</h1>
        <button class="btn btn-primary">主要按鈕</button>
    </div>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## 疑難排解

### 常見問題

1. **樣式不生效**
   - 檢查 CSS 連結是否正確
   - 清除瀏覽器快取

2. **JavaScript 功能不工作**
   - 確保引入 bootstrap.bundle.min.js
   - 檢查是否有 JavaScript 錯誤
