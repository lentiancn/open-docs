# Vite

Vite es la nueva generación de herramientas de construcción de front-end.

## Introducción

Vite es una herramienta de construcción de front-end de nueva generación desarrollada por Evan You, el autor de Vue.js. Utiliza el soporte nativo de módulos ES del navegador para proporcionar una experiencia de desarrollo ultrarrápida.

## Características principales

- **Desarrollo ultrarrápido**: Servidor de desarrollo basado en ESM
- **Reemplazo de módulos en caliente**: Actualización instantánea en caliente
- **Compilación bajo demanda**: Solo compila los archivos necesarios
- **Ricos plugins**: Soporta Vue, React, Svelte y más
- **Optimización de producción**: División de código y minificación incorporadas

## Inicio rápido

### Instalación

```bash
npm create vite@latest my-app -- --template vue
cd my-app
npm install
npm run dev
```
