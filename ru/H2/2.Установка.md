# Руководство по установке H2

## Установка через Maven

```xml
<dependency>
    <groupId>com.h2database</groupId>
    <artifactId>h2</artifactId>
    <version>2.2.224</version>
</dependency>
```

## Запуск

```bash
java -jar h2.jar
```

Веб-консоль: http://localhost:8082

## URL подключения

| Режим | URL |
|-------|-----|
| В памяти | jdbc:h2:mem:testdb |
| Файл | jdbc:h2:file:./data/testdb |
| TCP | jdbc:h2:tcp://localhost:9092/testdb |
