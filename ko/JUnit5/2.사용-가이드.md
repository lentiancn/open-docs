# JUnit 5 사용 가이드

---

## 테스트

```java
@Test
void test() {
    assertEquals(5, 2 + 3);
}
```

---

## 단언

```java
assertEquals(expected, actual);
assertTrue(condition);
```

---

## 어노테이션

```java
@BeforeEach
void setUp() { }

@Disabled
@Test
void testSkip() { }

@DisplayName("이름")
@Test
void testName() { }
```

---

## 매개변수화된 테스트

```java
@ParameterizedTest
@ValueSource(ints = { 1, 2, 3 })
void test(int number) {
    assertTrue(number > 0);
}
```
