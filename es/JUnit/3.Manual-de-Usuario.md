# Manual de usuario de JUnit 5

## Prueba básica

```java
import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;

class MiPrueba {
    
    @Test
    void pruebaSimple() {
        assertEquals(2, 1 + 1);
    }
}
```

## Ciclo de vida

```java
@BeforeAll
static void antesDeTodo() { }

@AfterAll
static void despuesDeTodo() { }

@BeforeEach
void antesDeCadaUno() { }

@AfterEach
void despuesDeCadaUno() { }
```

## Assertions

```java
assertEquals(4, 2 + 2);
assertTrue(5 > 3);
assertThrows(Exception.class, () -> { });
```

## Pruebas parametrizadas

```java
@ParameterizedTest
@ValueSource(ints = {1, 2, 3})
void esPositivo(int numero) {
    assertTrue(numero > 0);
}
```

## Pruebas anidadas

```java
@Nested
class PruebasInternas {
    @Test
    void prueba() { }
}
```

## Mejores prácticas

1. Nombres descriptivos con @DisplayName
2. Una aserción por prueba
3. Pruebas independientes
4. Mensajes de error significativos
