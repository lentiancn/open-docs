# Bootstrap

> Belieftes Frontend-CSS-Framework

---

## Projektübersicht

Bootstrap ist ein Open-Source-CSS-Framework, das von Twitter entwickelt und gepflegt wird und erstmals 2011 veröffentlicht wurde. Bootstrap bietet reichhaltige vorgefertigte Komponenten und ein responsives Grid-System für den schnellen Aufbau moderner, responsiver Websites und Webanwendungen.

## Schnellstart

### CDN einbinden

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### Seite erstellen

```html
<!DOCTYPE html>
<html lang="de">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Bootstrap Seite</title>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <div class="container">
    <h1>Hallo Bootstrap!</h1>
  </div>
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```
