# MySQL 설치 가이드

## Windows에 설치

### MySQL Installer를 사용한 설치

MySQL Installer는 마법사 기반 설치 환경을 제공하는 Windows에서 MySQL을 설치하는 가장 쉬운 방법입니다.

1. MySQL Installer 다운로드
   https://dev.mysql.com/downloads/installer/에서 설치 프로그램을 다운로드하세요.

2. 설치 프로그램 실행
   다운로드한 MSI 파일을 두 번 클릭하여 설치 마법사를 시작하세요.

3. 설치 유형 선택
   - **Full**: 모든 MySQL 제품 및 기능 설치
   - **Developer Default**: 개발 환경에 적합
   - **Server Only**: MySQL Server만 설치
   - **Custom**: 설치할 특정 구성 요소 선택

4. MySQL Server 구성
   - MySQL Server 버전 선택
   - 포트 번호 설정(기본값 3306)
   - 시작 유형 선택

5. root 비밀번호 설정
   MySQL 관리자 계정에 강력한 비밀번호를 설정하세요.

6. 설치 완료
   마법사에 따라 설치 프로세스를 완료하세요.

### ZIP 아카이브를 사용한 수동 설치

1. MySQL ZIP 패키지 다운로드
   https://dev.mysql.com/downloads/mysql/에서 ZIP 아카이브를 다운로드하세요.

2. 파일 추출
   C:\mysql과 같이 원하는 디렉터리에 ZIP 아카이브를 추출하세요.

3. 데이터 디렉터리 초기화
   ```
   mysqld --initialize-insecure
   ```

4. MySQL 서비스 시작
   ```
   mysqld --install MySQL
   net start MySQL
   ```

5. 로그인 및 비밀번호 변경
   ```
   mysql -u root -p
   ALTER USER 'root'@'localhost' IDENTIFIED BY '새비밀번호';
   ```

## Linux에 설치

### Ubuntu/Debian 시스템

1. 패키지 인덱스 업데이트
   ```bash
   sudo apt update
   ```

2. MySQL Server 설치
   ```bash
   sudo apt install mysql-server
   ```

3. MySQL 서비스 시작
   ```bash
   sudo systemctl start mysql
   sudo systemctl enable mysql
   ```

4. 보안 설정
   ```bash
   sudo mysql_secure_installation
   ```

### CentOS/RHEL 시스템

1. MySQL Yum 저장소 추가
   ```bash
   sudo yum install https://dev.mysql.com/get/mysql80-community-release-el8-4.noarch.rpm
   ```

2. MySQL Server 설치
   ```bash
   sudo yum install mysql-community-server
   ```

3. 서비스 시작
   ```bash
   sudo systemctl start mysqld
   sudo systemctl enable mysqld
   ```

4. 임시 비밀번호 가져오기
   ```bash
   sudo grep 'temporary password' /var/log/mysqld.log
   ```

## macOS에 설치

### Homebrew 사용

1. Homebrew 설치(설치되지 않은 경우)
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```

2. MySQL 설치
   ```bash
   brew install mysql
   ```

3. MySQL 서비스 시작
   ```bash
   brew services start mysql
   ```

4. 보안 설정
   ```bash
   mysql_secure_installation
   ```

### DMG 패키지 사용

1. DMG 패키지 다운로드
   https://dev.mysql.com/downloads/mysql/에서 macOS DMG 패키지를 다운로드하세요.

2. 설치 프로그램 실행
   PKG 파일을 두 번 클릭하고 마법사에 따라 설치를 완료하세요.

3. MySQL 시작
   시스템 환경설정을 통해 MySQL 서비스를 시작하세요.

## Docker에 MySQL 설치

### MySQL 이미지 풀
```bash
docker pull mysql:latest
```

### MySQL 컨테이너 실행
```bash
docker run -d \
  --name mysql-container \
  -e MYSQL_ROOT_PASSWORD=비밀번호 \
  -e MYSQL_DATABASE=testdb \
  -p 3306:3306 \
  mysql:latest
```

### 일반 Docker 매개변수
- `-e MYSQL_ROOT_PASSWORD`: root 비밀번호 설정
- `-e MYSQL_DATABASE`: 초기 데이터베이스 생성
- `-p 3306:3306`: 포트 매핑
- `-v /data/mysql:/var/lib/mysql`: 데이터 지속성을 위한 볼륨

## 설치 후 확인

### MySQL 버전 확인
```bash
mysql --version
```

### 서비스 상태 확인
```bash
systemctl status mysql   # Linux
brew services list      # macOS
```

### MySQL에 로그인
```bash
mysql -u root -p
```

## 일반적인 설치 문제

### 포트 충돌
포트 3306이 사용 중인 경우 구성 파일에서 포트를 변경하거나 해당 포트를 사용하는 서비스를 중지하세요.

### 권한 문제
Linux에서는 sudo를 사용하여 권한을 상승시키고 데이터 디렉터리에 올바른 읽기/쓰기 권한이 있는지 확인하세요.

### 방화벽 설정
방화벽이 포트 3306에 대한 액세스를 허용하는지 확인하세요.
