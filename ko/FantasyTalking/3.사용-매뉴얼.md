# FantasyTalking 사용자 매뉴얼

## 기본 기능

### 음성 합성

음성 합성은 FantasyTalking의 핵심 기능으로 텍스트를 자연스럽고 부드러운 음성으로 변환합니다:

```javascript
// 간단한 텍스트에서 음성으로
const audio = await client.speak('안녕하세요, FantasyTalking에 오신 것을 환영합니다');

// 음성 스타일 지정
const audio = await client.speak('오늘 날씨가 정말 좋네요', {
  voice: {
    style: 'happy',
    language: 'ko-KR'
  }
});

// 속도 및 피치 지정
const audio = await client.speak('잠시만 기다려 주세요', {
  voice: {
    speed: 1.2,
    pitch: 0.9
  }
});
```

### SSML 마크업 언어

FantasyTalking은 더精细한 음성 출력을 위해 SSML 마크업을 지원합니다:

```xml
<speak>
  <p>
    <s>첫 번째 문장.</s>
    <s>두 번째 문장.</s>
  </p>
  <break time="500ms"/>
  <p>
    <emphasis level="moderate">중요한 내용입니다</emphasis>
  </p>
</speak>
```

### 음성 이벤트

음성 재생 중 다양한 이벤트를 리슨할 수 있습니다:

```javascript
// 음성 시작 리슨
client.on('speech-start', (event) => {
  console.log('음성 재생 시작');
});

// 단어 경계 리슨
client.on('word-boundary', (event) => {
  console.log(`현재 단어: ${event.word}`);
});

// 음성 종료 리슨
client.on('speech-end', (event) => {
  console.log('음성 재생 완료');
});

// 오류 리슨
client.on('error', (event) => {
  console.error('오류:', event.error);
});
```

### 배치 합성

대량 텍스트를 합성해야 할 때 배치 합성 기능을 사용할 수 있습니다:

```javascript
const texts = [
  '첫 번째 음성입니다',
  '두 번째 음성입니다',
  '세 번째 음성입니다'
];

// 병렬 합성
const results = await Promise.all(
  texts.map(text => client.speak(text))
);
```

## 대화 기능

### 기본 대화

FantasyTalking은 다중 턴 대화와 컨텍스트 메모리를 지원하는 강력한 대화 관리 기능을 제공합니다:

```javascript
// 대화 세션 생성
const session = client.createSession({
  systemPrompt: '당신은 친절한 어시스턴트입니다',
  language: 'ko-KR'
});

// 첫 번째 턴
const response1 = await session.chat('오늘 날씨가 어때요?');
console.log(response1.text);
console.log(response1.audio);

// 두 번째 턴 (자동 컨텍스트 포함)
const response2 = await session.chat('내일은 어때요?');
console.log(response2.text);
```

## 고급 기능

### 스트리밍 음성 합성

실시간 피드백이 필요한 시나리오에서는 스트리밍 음성 합성을 사용할 수 있습니다:

```javascript
// 스트리밍 출력 활성화
const stream = await client.speakStream('이것은 긴 텍스트입니다');

// 데이터 청크 리슨
stream.on('data', (chunk) => {
  audioPlayer.play(chunk);
});

stream.on('end', () => {
  console.log('음성 합성 완료');
});
```

## 모범 사례

### 성능 최적화

1. **연결 재사용**: 클라이언트 인스턴스를 자주 생성하지 마세요
2. **배치 처리**: 대량 텍스트에는 배치 합성 사용
3. **캐싱**: 중복 합성을 피하기 위해 캐시 사용

### 오류 처리

```javascript
try {
  const audio = await client.speak('테스트 텍스트');
} catch (error) {
  if (error.code === 'RATE_LIMIT') {
    console.log('요청이 너무 많습니다, 나중에 다시 시도해 주세요');
  } else if (error.code === 'INVALID_API_KEY') {
    console.log('API 키가無効입니다');
  }
}
```
