# Введение в H2

H2 — это система управления реляционными базами данных с открытым исходным кодом, написанная на Java.

## Особенности

- Встроенная база данных (без отдельного сервера)
- Клиент-серверный режим
- База данных в памяти
- Совместимость с MySQL, PostgreSQL, Oracle
- Шифрование базы данных
- Полнотекстовый поиск

## Применение

- Разработка на Java
- Модульное тестирование
- Быстрое прототипирование

## Лицензия

Двойная лицензия: MPL 2.0 или EPL 1.0

## Требования

Java 11 или выше
