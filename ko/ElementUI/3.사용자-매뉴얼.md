# Element UI 사용자 매뉴얼

## 기본 컴포넌트

### Button

```html
<el-button>버튼</el-button>
<el-button type="primary">주요</el-button>
<el-button type="success">성공</el-button>
<el-button type="warning">경고</el-button>
<el-button type="danger">위험</el-button>
```

### Layout

```html
<el-row>
  <el-col :span="12"><div></div></el-col>
  <el-col :span="12"><div></div></el-col>
</el-row>
```

## 폼 컴포넌트

### Input

```html
<el-input v-model="input" placeholder="입력"></el-input>
```

### Select

```html
<el-select v-model="value" placeholder="선택">
  <el-option label="옵션 1" value="1"></el-option>
</el-select>
```

## 데이터 컴포넌트

### Table

```html
<el-table :data="tableData">
  <el-table-column prop="date" label="날짜"></el-table-column>
  <el-table-column prop="name" label="이름"></el-table-column>
</el-table>
```

### Pagination

```html
<el-pagination :total="1000"></el-pagination>
```

## 알림

### Message

```javascript
this.$message({ message: '성공', type: 'success' })
```

### Alert

```html
<el-alert title="성공" type="success"></el-alert>
```

## 내비게이션

### Menu

```html
<el-menu>
  <el-menu-item index="1">메뉴 1</el-menu-item>
</el-menu>
```

### Tabs

```html
<el-tabs v-model="activeName">
  <el-tab-pane label="탭 1" name="first">내용</el-tab-pane>
</el-tabs>
```

## 기타

### Dialog

```html
<el-button @click="visible = true">열기</el-button>
<el-dialog :visible.sync="visible" title="제목">
  <span>내용</span>
</el-dialog>
```
