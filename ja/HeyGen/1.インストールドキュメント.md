# HeyGen インストールガイド

HeyGenは、テキストやオーディオ入力からリアルなAIアバターを作成し、話し頭のビデオを生成できるAIビデオ生成プラットフォームです。

## プラットフォームの概要

HeyGenは主にクラウドベースのSaaSプラットフォームです。オープンソースプロジェクトとは異なり、HeyGenはローカルインストールは不要で、WebインターフェースまたはAPIを通じてアクセスします。

### アクセス方法

| 方法 | 説明 | 要件 |
|------|------|------|
| Webインターフェース | ブラウザベースのビデオ作成 | アカウント + インターネット |
| API | プログラムアクセス | APIキー |
| SDK | ローカル開発 | 開発者アカウント |

## バージョンの概要

HeyGenはクラウドサービスであり、ローカルインストールではありません：

| コンポーネント | バージョン | 備考 |
|----------------|------------|------|
| HeyGen Web | 最新版 | 常に最新 |
| HeyGen API | v1/v2 | REST API |
| HeyGen SDK | 最新版 | 開発者用 |

**注意：** HeyGenは商用製品です。一部の機能には有料サブスクリプションが必要です。

## システム要件

### Webアクセス要件

## HeyGenへのアクセス

### ステップ1：アカウント作成

1. https://www.heygen.comにアクセス
2. 「サインアップ」または「始める」をクリック
3. メールアドレスを認証

### ステップ2：プラン選択

| プラン | 機能 | 価格 |
|--------|------|------|
| 無料 | 制限あり、ウォーターマークあり | 無料 |
| Pro | 更多時間、ウォーターマークなし | $29/月 |
| エンタープライズ | カスタム制限、APIアクセス | お問い合わせ |

### ステップ3：APIキーの取得（開発者）

1. HeyGenダッシュボードにログイン
2. 設定 → APIに移動
3. APIキーを生成

## APIインストール（開発者）

### Python SDK

```bash
# HeyGen SDKをインストール
pip install heygen-python

# またはソースからインストール
pip install git+https://github.com/heygen-ai/heygen-python.git
```

### Node.js SDK

```bash
# HeyGen SDKをインストール
npm install heygen-node

# またはソースからインストール
npm install github:heygen-ai/heygen-node
```

### インストール確認

```python
# Python
import heygen
print(heygen.__version__)
```

```javascript
// Node.js
const heygen = require('heygen-node');
console.log(heygen.version);
```

## API設定

### Pythonの例

```python
from heygen import HeyGen

# APIキーで初期化
heygen = HeyGen(api_key="your_api_key")

# アカウント情報を取得
account = heygen.get_account_info()
print(account)
```

### Node.jsの例

```javascript
const { HeyGen } = require('heygen-node');

const heygen = new HeyGen({ apiKey: 'your_api_key' });

// アカウント情報を取得
const account = await heygen.getAccountInfo();
console.log(account);
```

## 一般的な問題

### 認証エラー

**問題：**「無効なAPIキー」

**解決策：**
1. APIキーが正しいか確認
2. APIキーがアクティブか確認
3. アカウントにAPIアクセス権限があるか確認

### レート制限

**問題：**「リクエスト过多」

**解決策：**
1. API使用制限を確認
2. 上位プランにアップグレード
3. リクエストキューイングを実装

### ビデオ生成失败

**問題：**「生成失败」

**解決策：**
1. 入力パラメータを確認
2. クレジット残高を確認
3. 後でもう一度試す

## セキュリティベストプラクティス

1. **APIキーを公開しない** - 環境変数を使用
2. **定期的にキーをローテーション** - 定期的に新しいキーを生成
3. **最小権限を使用** - 必要なスコープのみをリクエスト
4. **使用状況を監視** - ダッシュボードで異常を確認
