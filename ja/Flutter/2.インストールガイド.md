# Flutter インストールガイド

## 環境要件

Flutterをインストールする前に、開発環境が以下の要件を満たしていることを確認してください。OSについては、FlutterはWindows、macOS、Linuxをサポートしています。WindowsはWindows 10以降、macOSはmacOS 10.14以降、LinuxはGNOMEまたはKDEデスクトップ環境をサポートしている必要があります。

ディスク容量について、Flutter SDK自体は約1GBを占めますが、開発プロジェクトとビルドキャッシュを考慮すると、少なくとも10GBの空き容量を確保することをお勧めします。メモリについて、8GB以上のメモリがあればよりスムーズな開発体験ができます。

モバイル開発について、iOS開発にはmacOSとXcodeが必要で、Android開発にはAndroid StudioとAndroid SDKのインストールが必要です。Webとデスクトップ開発は特に要件がなく、Flutter SDKをインストールするだけです。

## インストール手順

### Windows インストール

WindowsにFlutterをインストールするには2つの主な方法があります。1つはFlutter公式サイトから事前コンパイルされたアーカイブをダウンロードし、適切な場所に展開し、flutter/binディレクトリをシステムPATH環境変数に追加することです。

ダウンロード方法：Flutter公式サイトflutter.devを訪問し、最新のstableバージョンをダウンロードします。C:\src\flutterまたは別の適切な場所に展開します。「システム」→「詳細設定」→「環境変数」を開き、ユーザー変数のPathにFlutterのbinatalogへのパスを追加します。

完了後、コマンドプロンプトを開き、flutter doctorコマンドを実行してインストールが成功したかどうかを確認します。Flutter doctorはFlutter SDKの状態を檢査し、構成の提案を行います。

### macOS インストール

macOSでは、Gitを使用したインストールをお勧めします。まずXcodeがインストールされていることを確認しターミナルを開いて以下のコマンドを実行してFlutterをクローンします：

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

FlutterをPATHに追加：

```bash
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.bash_profile
```

zshユーザーの場合：

```bash
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.zshrc
```

その後、source ~/.bash_profileまたはsource ~/.zshrcを実行して環境変数を更新します。

### Linux インストール

LinuxのインストールはmacOS similaresです。まず依存関係をインストール：

```bash
sudo apt-get update
sudo apt-get install curl git unzip xz-utils zip libglu1-mesa clang cmake ninja-build pkg-config libgtk-3-dev
```

その後、Flutterリポジトリをクローンし、PATHを設定します（macOSと同じ手順）。

## インストール確認

インストール後、以下のコマンドを実行して確認します：

```bash
flutter --version
flutter doctor
```

Flutter doctorはインストール状態と不足している依存関係を表示します。プロンプトに従って、Android StudioのインストールやAndroid SDKの設定などの必要な設定を完了します。

## IDE設定

### VS Code

VS CodeはFlutter推奨のエディタの一つです。VS Codeをインストールした後、拡張機能マーケットプレイスを開いて、FlutterとDartの拡張機能を検索してインストールします。インストール後、VS Codeを再起動すると、Flutter関連の機能が使用できます。

### Android Studio

Android Studioをインストールした後、Pluginsページを開いて、Flutterプラグインを検索してインストールします。インストール後、Flutter SDKパスを設定します：Preferences → Language & Frameworks → Flutterを開き、Flutter SDKパスを設定します。

## クイックスタート

最初のFlutterプロジェクトを作成：

```bash
flutter create my_first_app
cd my_first_app
flutter run
```

またはVS Codeでコマンドパレット（Ctrl+Shift+P）を開き、Flutter: New Projectと入力して新しいプロジェクトを作成します。

flutter devicesを実行すると接続されたデバイスが表示され、flutter run -d device_idで特定のデバイスで実行できます。
