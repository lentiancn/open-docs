# JUnit Installation Guide

JUnit is the most popular unit testing framework for Java and the default testing framework for Spring Boot applications.

## Version Overview

| Version | Java Version | Release Date |
|---------|-------------|-------------|
| JUnit 5 (Jupiter) | Java 8+ | 2016 |
| JUnit 4 | Java 5+ | 2005 |

**Recommendation:** Use JUnit 5 (Jupiter) for new projects. JUnit 4 is legacy.

## Add Dependencies

### Maven (JUnit 5)

```xml
<dependencies>
    <!-- JUnit 5 -->
    <dependency>
        <groupId>org.junit.jupiter</groupId>
        <artifactId>junit-jupiter</artifactId>
        <version>5.11.0</version>
        <scope>test</scope>
    </dependency>
</dependencies>
```

### Gradle (JUnit 5)

```groovy
testImplementation 'org.junit.jupiter:junit-jupiter:5.11.0'
```

### JUnit 4 (Legacy)

```xml
<dependency>
    <groupId>junit</groupId>
    <artifactId>junit</artifactId>
    <version>4.13.2</version>
    <scope>test</scope>
</dependency>
```

## IDE Support

### IntelliJ IDEA
- Built-in support for JUnit 5
- Version 2019.2+ required

### VS Code
- Install "Extension Pack for Java"
- Version 1.0+ required

### Eclipse
- Install "JUnit" from Eclipse Marketplace
- Eclipse IDE for Enterprise Java Developers recommended

## Spring Boot Integration

JUnit 5 is included by default in Spring Boot:

```xml
<parent>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-parent</artifactId>
    <version>3.3.0</version>
</parent>

<dependencies>
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-test</artifactId>
        <scope>test</scope>
    </dependency>
</dependencies>
```

This includes:
- JUnit Jupiter
- Mockito
- AssertJ
- Spring Test
