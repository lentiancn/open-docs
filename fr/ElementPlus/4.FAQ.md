# Element Plus FAQ

Questions fréquentes.

## Installation

### Q: Erreur d'installation?

**Solutions:**
1. Vérifier la version de Node.js
```bash
node --version  # Nécessite 18.0+
```
2. Vider le cache
```bash
npm cache clean --force
```

### Q: Erreurs TypeScript?

**Solutions:**
Installer les types:
```bash
npm install -D @types/node
```

## Utilisation

### Q: Composant non affiché?

**Solutions:**
1. Vérifier l'import
2. Vérifier la version de Vue (3.2+)

### Q: Styles non appliqués?

**Solutions:**
1. Vérifier l'import CSS
2. Utiliser `:deep()`

### Q: Icônes non affichées?

**Solutions:**
Installer le package d'icônes:
```bash
npm install @element-plus/icons-vue
```

Enregistrer:
```typescript
import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```
