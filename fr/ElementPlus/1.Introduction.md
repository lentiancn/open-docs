# ElementPlus Introduction

ElementPlus is a Vue 3 based UI component library.

## What is ElementPlus

ElementPlus is the Vue 3 version of Element UI.

## Main Features

- Vue 3 Support
- 70+ Components
- TypeScript Support
- Tree Shaking
- Internationalization
- Theme Customization

## Components

- Basic: Button, Icon, Layout
- Form: Input, Select, Date Picker
- Data Display: Table, Tree, Pagination
- Feedback: Dialog, Drawer, Message

## Installation

```bash
npm install element-plus
```
