# MySQL 安裝指南

## Windows 系統安裝

### 使用 MySQL Installer 安裝

MySQL Installer 是 Windows 平台上最便捷的安裝方式，提供基於精靈的安裝體驗。

1. 下載 MySQL Installer
   造訪 https://dev.mysql.com/downloads/installer/ 下載安裝程式。

2. 執行安裝程式
   雙擊下載的 msi 檔案啟動安裝精靈。

3. 選擇安裝類型
   - **Full**：安裝所有 MySQL 產品和功能
   - **Developer Default**：適合開發環境
   - **Server only**：僅安裝 MySQL Server
   - **Custom**：自訂安裝元件

4. 設定 MySQL Server
   - 選擇 MySQL Server 版本
   - 設定連接埠號（預設 3306）
   - 選擇啟動類型

5. 設定 root 密碼
   為 MySQL 系統管理員帳戶設定強密碼。

6. 完成安裝
   按照精靈完成安裝過程。

### 使用 ZIP 壓縮檔手動安裝

1. 下載 MySQL ZIP 壓縮檔
   造訪 https://dev.mysql.com/downloads/mysql/ 下載 ZIP 格式的安裝壓縮檔。

2. 解壓縮檔案
   將 ZIP 壓縮檔解壓縮到指定目錄，如 C:\mysql。

3. 初始化資料目錄
   ```
   mysqld --initialize-insecure
   ```

4. 啟動 MySQL 服務
   ```
   mysqld --install MySQL
   net start MySQL
   ```

5. 登入並修改密碼
   ```
   mysql -u root -p
   ALTER USER 'root'@'localhost' IDENTIFIED BY '新密碼';
   ```

## Linux 系統安裝

### Ubuntu/Debian 系統

1. 更新軟體來源
   ```bash
   sudo apt update
   ```

2. 安裝 MySQL Server
   ```bash
   sudo apt install mysql-server
   ```

3. 啟動 MySQL 服務
   ```bash
   sudo systemctl start mysql
   sudo systemctl enable mysql
   ```

4. 安全設定
   ```bash
   sudo mysql_secure_installation
   ```

### CentOS/RHEL 系統

1. 新增 MySQL Yum 儲存庫
   ```bash
   sudo yum install https://dev.mysql.com/get/mysql80-community-release-el8-4.noarch.rpm
   ```

2. 安裝 MySQL Server
   ```bash
   sudo yum install mysql-community-server
   ```

3. 啟動服務
   ```bash
   sudo systemctl start mysqld
   sudo systemctl enable mysqld
   ```

4. 取得暫時密碼
   ```bash
   sudo grep 'temporary password' /var/log/mysqld.log
   ```

## macOS 系統安裝

### 使用 Homebrew 安裝

1. 安裝 Homebrew（如果未安裝）
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```

2. 安裝 MySQL
   ```bash
   brew install mysql
   ```

3. 啟動 MySQL 服務
   ```bash
   brew services start mysql
   ```

4. 安全設定
   ```bash
   mysql_secure_installation
   ```

### 使用 DMG 套件安裝

1. 下載 DMG 套件
   造訪 https://dev.mysql.com/downloads/mysql/ 下載 macOS DMG 套件。

2. 執行安裝程式
   雙擊 PKG 檔案，按照精靈完成安裝。

3. 啟動 MySQL
   透過系統偏好設定啟動 MySQL 服務。

## Docker 安裝 MySQL

### 拉取 MySQL 映像檔
```bash
docker pull mysql:latest
```

### 執行 MySQL 容器
```bash
docker run -d \
  --name mysql-container \
  -e MYSQL_ROOT_PASSWORD=your_password \
  -e MYSQL_DATABASE=testdb \
  -p 3306:3306 \
  mysql:latest
```

### 常用 Docker 參數說明
- `-e MYSQL_ROOT_PASSWORD`：設定 root 密碼
- `-e MYSQL_DATABASE`：建立初始資料庫
- `-p 3306:3306`：連接埠對應
- `-v /data/mysql:/var/lib/mysql`：資料卷持久化

## 安裝後驗證

### 檢查 MySQL 版本
```bash
mysql --version
```

### 檢查服務狀態
```bash
systemctl status mysql   # Linux
brew services list      # macOS
```

### 登入 MySQL
```bash
mysql -u root -p
```

## 常見安裝問題

### 連接埠衝突
如果 3306 連接埠被佔用，可在設定檔中修改連接埠，或停止佔用連接埠的服務。

### 權限問題
Linux 系統下注意使用 sudo 提升權限，確保資料目錄有正確的讀寫權限。

### 防火牆設定
確保防火牆允許 3306 連接埠的存取。
