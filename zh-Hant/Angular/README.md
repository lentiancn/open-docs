# Angular

現代 Web 應用開發框架。

## 簡介

Angular 是由 Google 開發和維護的開源 Web 應用框架，為現代 Web 應用開發提供完整解決方案。

### 主要特性

- **TypeScript 原生**：完全基於 TypeScript 開發
- **組件化架構**：便於維護和測試
- **依賴注入**：強大的依賴注入系統
- **路由系統**：完整的客戶端路由
- **表單處理**：響應式表單支持
- **HTTP 客戶端**：完整的 HTTP 服務

### 統計數據

- GitHub 星標：93,000+
- 每週下載量：3,000,000+

## 快速開始

### 安裝 Angular CLI

```bash
npm install -g @angular/cli
```

### 創建新項目

```bash
ng new my-app
cd my-app
ng serve
```

### 訪問應用

打開瀏覽器：http://localhost:4200

## 常用命令

```bash
# 創建組件
ng g c my-component

# 創建服務
ng g s my-service

# 構建生產版本
ng build

# 運行測試
ng test
```

## 核心概念

- **組件**：視圖和邏輯的組合
- **模組**：組織應用程序代碼
- **服務**：封裝業務邏輯
- **路由**：頁面導航
- **指令**：擴展 HTML
- **管道**：數據轉換
