# Element UI 安装指南

> 完整的环境配置和安装教程

---

## 环境要求

### 开发环境要求

| 环境 | 最低版本 | 推荐版本 |
|------|----------|----------|
| Node.js | 8.x | 10.x 或 12.x |
| npm | 5.x | 6.x |
| Vue | 2.6.0 | 2.6.14 |
| 构建工具 | Webpack 4.x | Webpack 4.x 或 5.x |

### 浏览器兼容性

- Chrome >= 49
- Firefox >= 45
- Safari >= 10
- Edge >= 14
- IE 11（有限支持）

## 安装步骤

### 使用 npm 安装（推荐）

```bash
# 进入 Vue 2 项目目录
cd your-vue-project

# 安装 Element UI
npm install element-ui --save
```

### 使用 yarn 安装

```bash
yarn add element-ui
```

### 指定版本安装

```bash
# 安装最新 2.x 版本
npm install element-ui@latest

# 安装特定版本
npm install element-ui@2.15.13
```

## 在 Vue 2 项目中引入

### 完整引入（适合小型项目）

```javascript
// main.js
import Vue from 'vue'
import App from './App.vue'

// 引入 Element UI
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)

new Vue({
  render: h => h(App)
}).$mount('#app')
```

### 按需引入（推荐）

#### 1. 安装 babel-plugin-component

```bash
npm install babel-plugin-component -D
```

#### 2. 配置 babel

```json
{
  "presets": [
    ["env", {
      "modules": false,
      "targets": {
        "browsers": ["> 1%", "last 2 versions", "not ie <= 8"]
      }
    }]
  ],
  "plugins": [
    ["component", {
      "libraryName": "element-ui",
      "styleLibraryName": "theme-chalk"
    }]
  ]
}
```

#### 3. 按需引入组件

```javascript
// main.js
import Vue from 'vue'
import { Button, Form, FormItem, Input, Table, TableColumn } from 'element-ui'
import App from './App.vue'

// 注册需要的组件
Vue.use(Button)
Vue.use(Form)
Vue.use(FormItem)
Vue.use(Input)
Vue.use(Table)
Vue.use(TableColumn)

new Vue({
  render: h => h(App)
}).$mount('#app')
```

### 使用 CDN 引入

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Element UI CDN 示例</title>
  <script src="https://cdn.jsdelivr.net/npm/vue@2.6.14/dist/vue.js"></script>
  <link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
  <script src="https://unpkg.com/element-ui"></script>
</head>
<body>
  <div id="app">
    <el-button type="primary" @click="visible = true">打开对话框</el-button>
    <el-dialog title="提示" :visible.sync="visible" width="30%">
      <span>这是一段信息</span>
      <span slot="footer" class="dialog-footer">
        <el-button @click="visible = false">取消</el-button>
        <el-button type="primary" @click="visible = false">确定</el-button>
      </span>
    </el-dialog>
  </div>

  <script>
    new Vue({
      el: '#app',
      data: {
        visible: false
      }
    })
  </script>
</body>
</html>
```

## 国际化配置

### 全局中文配置

```javascript
// main.js
import Vue from 'vue'
import ElementUI from 'element-ui'
import locale from 'element-ui/lib/locale/lang/zh-CN'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI, {
  locale
})
```

## 主题定制

### 使用在线主题工具

```scss
// styles/element-variables.scss
$--color-primary: #409EFF;
$--color-success: #67C23A;
$--color-warning: #E6A23C;
$--color-danger: #F56C6C;
$--color-info: #909399;

@import 'element-ui/packages/theme-chalk/src/common/var.scss';
```
