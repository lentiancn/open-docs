# Apifox

Apifox는 중국에서 가장 많이 사용되는 API 협업 플랫폼으로, API 설계, 모의(mock) 데이터 생성, 테스트, 문서화를 하나의 플랫폼에서 해결합니다. API 설계, API 문서 관리, API 디버깅, API 모의(mock) 데이터 생성, API 자동화 테스트를 하나로 통합하여 중국에서 가장 인기 있는 API 관리 도구 중 하나입니다.

## 빠른 시작

### 다운로드 및 설치

https://apifox.com/를 방문하여 시스템용 설치 프로그램을 다운로드하세요.

### 프로젝트 생성

1. Apifox를 실행하고 계정에 로그인합니다
2. "새 프로젝트"를 클릭합니다
3. 프로젝트 이름을 입력합니다

### 인터페이스 생성

1. 프로젝트에서 "새 인터페이스"를 클릭합니다
2. 인터페이스 정보를 입력합니다
3. 저장합니다

### 인터페이스 디버깅

1. 인터페이스를 엽니다
2. "전송"을 클릭합니다
3. 응답을 확인합니다

## 주요 기능

- **API 설계**: 시각적으로 API 인터페이스 정의
- **API 문서**: 다양한 내보내기 형식으로 자동 문서 생성
- **API 디버깅**: 강력한 온라인 디버깅 기능
- **API 모의(mock)**: 시뮬레이션된 데이터를 자동으로 생성하는 스마트 모의
- **자동화 테스트**: 코드를 작성하지 않고 완벽한 API 테스트
- **팀 협업**: 실시간 동기화 및 권한 관리
