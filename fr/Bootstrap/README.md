# Bootstrap

Bootstrap est le framework de développement frontend open source le plus populaire au monde pour HTML, CSS et JavaScript. Développé et maintenu par Twitter, il offre une multitude de composants et de plugins JavaScript performants pour créer rapidement des applications web responsives.

## Démarrage rapide

### Inclusion via CDN

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### Installation npm

```bash
npm install bootstrap@5
```

## Fonctionnalités principales

- **Mise en page responsive** : Système de grille 12 colonnes, adapté à tous les écrans
- **Composants riches** : Barres de navigation, cartes, formulaires, fenêtres modales, etc.
- **Plugins JavaScript** : Menus déroulants, carrousels, collapse, et- **Support bien plus
 Sass** : Personnalisation facile du thème
