# OpenClaw

OpenClaw - Tu Asistente de IA Personal 🦞

## Resumen

OpenClaw es un asistente de IA personal que se ejecuta en tus propios dispositivos. Se comunica a través de canales que ya usas, incluyendo WhatsApp, Telegram, Slack, Discord, Google Chat y más de 20 otras plataformas. OpenClaw puede manejar trabajo real, gestionar horarios, enviar correos, controlar dispositivos inteligentes, convirtiéndose verdaderamente en tu compañero digital.

## Características

- **Acceso multi-canal**: Más de 20 plataformas de comunicación
- **Local-first**: Datos almacenados localmente, completamente bajo tu control
- **Operación 24/7**: Disponible cuando lo necesites
- **Interacción por voz**: Soporta activación por voz y conversación
- **Sistema de habilidades**: Capacidades de automatización extensibles
- **Código abierto**: Código completamente abierto y gratuito

## Inicio Rápido

### Instalar

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

O usando npm:

```bash
npm install -g openclaw@latest
```

### Configurar

```bash
openclaw onboard --install-daemon
```

### Usar

```bash
# Abrir Control UI
openclaw dashboard

# O enviar un mensaje
openclaw agent --message "Hello"
```
