# Руководство пользователя jQuery

## Быстрый старт

### Подключение jQuery

Подключите jQuery в `<head>` или в конце `<body>` вашего HTML-файла:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Демо jQuery</title>
    <script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
</head>
<body>
    <!-- Содержимое страницы -->
</body>
</html>
```

### Ожидание готовности DOM

Код jQuery должен выполняться после полной загрузки DOM:

```javascript
$(document).ready(function() {
    // Здесь пишите код jQuery
});

// Короткая форма
$(function() {
    // Здесь пишите код jQuery
});
```

## Селекторы

Селекторы jQuery — одна из основных функций jQuery, с синтаксисом почти идентичным CSS-селекторам.

### Базовые селекторы

```javascript
$("#myId")          // Селектор по ID
$(".myClass")       // Селектор по классу
$("div")            // Селектор по элементу
$("*")              // Универсальный селектор
$("div, p, span")   // Несколько селекторов
```

### Селекторы атрибутов

```javascript
$('[href]')                    // Элементы с атрибутом href
$('[href="value"]')           // href равен указанному значению
$('[href!="value"]')          // href не равен указанному значению
$('[href^="http"]')           // href начинается с указанного значения
$('[href$=".pdf"]')           // href заканчивается указанным значением
$('[href*="google"]')         // href содержит указанное значение
```

### Фильтрующие селекторы

```javascript
$("div:first")        // Первый элемент div
$("div:last")         // Последний элемент div
$("div:even")         // Элементы div с четным индексом
$("div:odd")          // Элементы div с нечетным индексом
$("div:eq(n)")        // Элемент div с индексом n
$("div:gt(n)")        // Элементы div с индексом больше n
$("div:lt(n)")        // Элементы div с индексом меньше n
$("div:not(.exclude") // Элементы div без указанного класса
```

## Манипулирование DOM

### Получение и установка содержимого

```javascript
// Получить/установить текстовое содержимое
var text = $("#myElement").text();
$("#myElement").text("Новое текстовое содержимое");

// Получить/установить HTML-содержимое
var html = $("#myElement").html();
$("#myElement").html("<p>Новое HTML-содержимое</p>");

// Получить/установить значение формы
var value = $("#inputField").val();
$("#inputField").val("Новое значение");
```

### Получение и установка атрибутов

```javascript
// Получить атрибут
var src = $("#myImage").attr("src");

// Установить один атрибут
$("#myImage").attr("src", "image.jpg");

// Установить несколько атрибутов
$("#myImage").attr({
    src: "image.jpg",
    alt: "Описание изображения",
    title: "Заголовок изображения"
});

// Удалить атрибут
$("#myImage").removeAttr("title");
```

### Получение и установка CSS

```javascript
// Получить одно CSS-свойство
var color = $("#myElement").css("color");

// Установить одно CSS-свойство
$("#myElement").css("color", "red");

// Установить несколько CSS-свойств
$("#myElement").css({
    color: "red",
    backgroundColor: "blue",
    fontSize: "16px"
});

// Добавить/удалить класс
$("#myElement").addClass("active highlight");
$("#myElement").removeClass("highlight");
$("#myElement").toggleClass("active");  // Переключить класс
```

### Обход DOM

```javascript
// Родительские элементы
$("#myElement").parent();
$("#myElement").parents();
$("#myElement").closest(".container");

// Дочерние элементы
$("#myElement").children();
$("#myElement").find(".child");

// Элементы-братья
$("#myElement").siblings();
$("#myElement").next();
$("#myElement").prev();
$("#myElement").nextAll();
```

### Создание и удаление элементов

```javascript
// Создать новый элемент
var newDiv = $("<div>", {
    class: "new-element",
    text: "Содержимое нового элемента"
});

// Добавить на страницу
$("#container").append(newDiv);      // Внутри, в конец
$("#container").prepend(newDiv);   // Внутри, в начало
$("#container").before(newDiv);      // Снаружи, перед
$("#container").after(newDiv);       // Снаружи, после

// Удалить элементы
$("#myElement").remove();             // Удалить элемент и детей
$("#myElement").empty();            // Удалить только детей
```

## Обработка событий

### Привязка событий

```javascript
// Привязать событие клика
$("#myButton").click(function() {
    alert("Кнопка нажата");
});

// Привязать используя метод on
$("#myButton").on("click", function() {
    alert("Кнопка нажата");
});

// Привязать несколько событий
$("#myElement").on({
    mouseenter: function() {
        $(this).css("background", "yellow");
    },
    mouseleave: function() {
        $(this).css("background", "");
    }
});
```

### Делегирование событий

```javascript
// Привязка событий для динамических элементов
$("#container").on("click", ".dynamic-item", function() {
    alert("Динамический элемент нажат");
});
```

### Частые события

| Событие | Описание |
|---------|----------|
| click | Событие клика |
| dblclick | Событие двойного клика |
| mouseenter | Мышь входит |
| mouseleave | Мышь выходит |
| mousemove | Мышь движется |
| keydown | Клавиша нажата |
| keyup | Клавиша отпущена |
| focus | Получает фокус |
| blur | Теряет фокус |
| submit | Форма отправлена |
| change | Значение изменено |
| load | Загружено |

## Анимационные эффекты

### Базовая анимация

```javascript
// Показать элемент
$("#myElement").show();

// Скрыть элемент
$("#myElement").hide();

// Переключить видимость
$("#myElement").toggle();

// Появление (fade in)
$("#myElement").fadeIn();

// Исчезновение (fade out)
$("#myElement").fadeOut();

// Переключение fade
$("#myElement").fadeToggle();

// Выезд сверху
$("#myElement").slideDown();

// Выезд снизу
$("#myElement").slideUp();

// Переключение slide
$("#myElement").slideToggle();
```

### Пользовательская анимация

```javascript
$("#myElement").animate({
    opacity: 0.5,
    left: "+=50",
    height: "200px"
}, 300, function() {
    // Callback после анимации
    console.log("Анимация завершена");
});
```

### Остановка анимации

```javascript
// Остановить текущую анимацию
$("#myElement").stop();

// Остановить все анимации и перейти в конечное состояние
$("#myElement").stop(true, true);
```

## Ajax-запросы

### Основное использование

```javascript
$.ajax({
    url: "/api/data",
    method: "GET",
    data: { id: 1, name: "test" },
    success: function(response) {
        console.log("Успех:", response);
    },
    error: function(xhr, status, error) {
        console.log("Ошибка:", error);
    }
});
```

### Сокращенные методы

```javascript
// GET-запрос
$.get("/api/data", { id: 1 }, function(response) {
    console.log(response);
});

// POST-запрос
$.post("/api/submit", { name: "test" }, function(response) {
    console.log(response);
});

// Загрузка HTML
$("#content").load("fragment.html");
```

### Полный пример

```javascript
$.ajax({
    url: "https://api.example.com/users",
    type: "GET",
    dataType: "json",
    timeout: 5000,
    headers: {
        "Authorization": "Bearer token"
    },
    beforeSend: function(xhr) {
        // Перед отправкой запроса
        console.log("Запрос будет отправлен");
    },
    success: function(data) {
        // Запрос успешен
        $("#user-list").html("");
        data.forEach(function(user) {
            $("#user-list").append("<li>" + user.name + "</li>");
        });
    },
    error: function(xhr, status, error) {
        // Запрос неуспешен
        alert("Ошибка загрузки: " + error);
    },
    complete: function(xhr, status) {
        // Запрос завершен (успех или неуспех)
        console.log("Запрос завершен, статус: " + status);
    }
});
```

## Утилитные функции

### Операции с массивами и объектами

```javascript
// Перебор массива
$.each([1, 2, 3], function(index, value) {
    console.log(index + ": " + value);
});

// Перебор объекта
$.each({ name: "John", age: 30 }, function(key, value) {
    console.log(key + ": " + value);
});

// Фильтрация массива
var filtered = $.grep([1, 2, 3, 4, 5], function(value) {
    return value > 3;
});

// Отображение массива
var mapped = $.map([1, 2, 3], function(value) {
    return value * 2;
});

// Слияние объектов
var merged = $.extend({}, obj1, obj2);
```

### Операции со строками

```javascript
// Удаление пробелов
$.trim("  hello  ");  // "hello"

// Экранирование спецсимволов
$.escapeSelector(".myClass");  // "\.myClass"
```

### Проверка типов

```javascript
$.isArray([]);        // true
$.isFunction(fn);     // true
$.isEmptyObject({});  // true
$.isPlainObject({});  // true
```

## Цепочка вызовов

jQuery поддерживает цепочку вызовов, позволяя непрерывно работать с одним элементом:

```javascript
$("#myElement")
    .addClass("highlight")
    .css("color", "red")
    .fadeIn()
    .animate({ height: "100px" }, function() {
        console.log("Готово");
    });
```

Используйте метод `end()` для возврата к предыдущему селектору:

```javascript
$("#myElement")
    .find(".child")
    .addClass("active")
    .end()    // Вернуться к #myElement
    .removeClass("highlight");
```
