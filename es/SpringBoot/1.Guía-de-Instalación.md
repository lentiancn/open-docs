# Guía de Instalación de Spring Boot

Esta guía cubre la configuración del entorno para el desarrollo de Spring Boot, incluyendo Java, Maven y Gradle.

## Requisitos Previos

### Instalación de Java

Spring Boot 3.x requiere Java 17 o superior. Spring Boot 2.x requiere Java 8 o superior.

**Windows:**
1. Descarga JDK de [Oracle](https://www.oracle.com/java/technologies/downloads/) o [Adoptium](https://adoptium.net/)
2. Ejecuta el instalador
3. Configura las variables de entorno:
   ```
   JAVA_HOME=C:\Program Files\Java\jdk-17
   PATH=%JAVA_HOME%\bin;%PATH%
   ```

**macOS:**
```bash
# Usando Homebrew
brew install openjdk@17

# Agregar al PATH
echo 'export JAVA_HOME=$(/usr/libexec/java_home)' >> ~/.zshrc
source ~/.zshrc
```

**Linux:**
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install openjdk-17-jdk

# Configurar JAVA_HOME
echo 'export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64' >> ~/.bashrc
source ~/.bashrc
```

Verificar instalación:
```bash
java -version
```

### Instalación de Maven

**Windows:**
1. Descarga de [Apache Maven](https://maven.apache.org/download.cgi)
2. Extrae a una carpeta (ej., C:\apache-maven)
3. Configura las variables de entorno:
   ```
   MAVEN_HOME=C:\apache-maven
   PATH=%MAVEN_HOME%\bin;%PATH%
   ```

**macOS:**
```bash
brew install maven
```

**Linux:**
```bash
sudo apt install maven
```

Verificar instalación:
```bash
mvn -version
```

### Instalación de Gradle

**Windows:**
1. Descarga de [Gradle](https://gradle.org/install/)
2. Extrae y agrega al PATH

**macOS:**
```bash
brew install gradle
```

**Linux:**
```bash
sudo apt install gradle
```

Verificar instalación:
```bash
gradle -version
```

## Configuración del IDE

### IntelliJ IDEA (Recomendado)
- Descarga de [JetBrains](https://www.jetbrains.com/idea/download/)
- Asegúrate de que JDK esté configurado en la estructura del proyecto
- Instala el plugin Spring Boot Assistant

### VS Code
- Instala Extension Pack for Java
- Instala Spring Boot Extension

## Inicio Rápido con Spring Initializr

1. Visita [start.spring.io](https://start.spring.io/)
2. Selecciona:
   - Proyecto: Maven o Gradle
   - Lenguaje: Java
   - Versión de Spring Boot
   - Metadatos del proyecto
3. Agrega dependencias (Spring Web, Spring Data JPA, etc.)
4. Haz clic en "Generate" para descargar el proyecto

## Verificar tu Configuración

```bash
# Verificar Java
java -version

# Verificar Maven
mvn -version

# Verificar Gradle (si usas Gradle)
gradle -version

# Crear un proyecto simple de Spring Boot y ejecutarlo
curl https://start.spring.io/starter.zip -o demo.zip
unzip demo.zip -d demo
cd demo
./mvnw spring-boot:run  # o ./gradlew bootRun
```
