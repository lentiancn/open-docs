# MultiTalk

MultiTalk is an audio-driven multi-person conversational video generation framework developed by the MeiGen-AI team. This project can generate high-quality video content containing multi-person interactions based on multi-stream audio input, reference images, and text prompts.

## Key Features

- Audio-driven multi-person conversational video generation
- Support for single-person and multi-person scenarios
- Industry-leading lip synchronization accuracy
- Support for cartoon character and singing video generation
- Output resolution supports 480P and 720P
- Maximum video clip length up to 15 seconds
