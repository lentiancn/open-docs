# OpenClaw

OpenClaw - 개인 AI 어시스턴트 🦞

## 개요

OpenClaw는 사용자의 자체 기기에서 실행되는 개인 AI 어시스턴트입니다. WhatsApp, Telegram, Slack, Discord, Google Chat 등 20개 이상의 플랫폼을 통해 커뮤니케이션합니다. OpenClaw는 실제 작업을 처리하고, 일정을 관리하고, 이메일을 보내고, 스마트 기기를 제어할 수 있어 진정한 디지털 파트너가 됩니다.

## 기능

- **멀티 채널 지원**: 20개 이상의 커뮤니케이션 플랫폼
- **로컬 우선**: 데이터가 로컬에 저장, 완전한 제어
- **24시간 운영**: 필요할 때 언제든지 사용 가능
- **음성 상호작용**: 음성 활성화 및 대화 지원
- **스킬 시스템**: 확장 가능한 자동화 기능
- **오픈소스**: 코드가 완전히 열려 있고 무료

## 빠른 시작

### 설치

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

또는 npm 사용:

```bash
npm install -g openclaw@latest
```

### 설정

```bash
openclaw onboard --install-daemon
```

### 사용

```bash
# Control UI 열기
openclaw dashboard

# 또는 메시지 보내기
openclaw agent --message "Hello"
```
