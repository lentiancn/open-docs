# Element UI 簡介

Element UI 是一個基於 Vue 2.0 的 UI 組件庫，為開發者提供了豐富的桌面端組件解決方案。

## 什麼是 Element UI

Element UI 是餓了麼團隊開發和維護的 Vue 2.0 組件庫。它提供了一套完整的設計規範和組件庫，幫助開發者快速構建桌面端應用程式。

作為 Vue 2.0 的配套 UI 框架，Element UI 提供了簡潔易用的組件封裝和良好的開發體驗。

## 主要特性

### 完善的組件體系

Element UI 提供了 70 多個高質量組件，涵蓋了桌面應用開發的各個方面：

- 基礎組件：按鈕、圖標、佈局、容器
- 表單組件：輸入框、選擇器、日期選擇器、開關、滑塊
- 數據展示：表格、樹形控件、分頁、標籤
- 反饋組件：對話框、抽屜、訊息提示、加載
- 導航組件：菜單、標籤頁、麵包屑、步驟條

### Vue 2.0 全面支援

Element UI 完全基於 Vue 2.0 開發：

- 組件化開發
- 響應式數據綁定
- 完善的 TypeScript 支援
- 良好的兼容性

### 按需加載

Element UI 支援按需導入使用的組件：

- 使用 babel-plugin-component 插件
- 手動按需導入
- 減少打包體積

### 國際化支援

內置多種語言包，滿足國際化應用開發需求。

### 主題定製

支援全局主題定製：

- 主題色配置
- 組件樣式覆蓋
- 在線主題生成器

## 設計原則

Element UI 遵循四大設計原則：

### 一致性

- 與現實生活一致：與現實生活的流程、邏輯保持一致，遵循用戶習慣的語言和概念
- 在界面中一致：所有的元素和結構需保持一致，包括設計樣式、圖標和文本、元素的位置等

### 反饋

- 控制反饋：通過界面樣式和交互动效讓用戶可以清晰地感知自己的操作
- 頁面反饋：操作後，通過頁面元素的變化清晰地展現當前狀態

### 效率

- 簡化流程：設計簡潔直觀的操作流程
- 清晰明確：語言表達清晰且表意明確，讓用戶快速理解進而作出決策
- 幫助用戶識別：界面簡單直白，讓用戶快速識別而非回憶，減少用戶記憶負擔

### 可控

- 用戶決策：根據場景可給予用戶操作建議或安全提示，但不能代替用戶進行決策
- 結果可控：用戶可以自由地進行操作，包括撤銷、回退和終止當前操作等

## 適用場景

Element UI 適用於以下場景：

- 企業內部管理系統
- 電商後台管理系統
- 數據可視化平台
- CRM 客戶管理系統
- 任何需要桌面端 UI 的 Vue 2 項目

## 與 Element Plus 的關係

Element UI 是 Element Plus（Vue 3 版本）的祖先：

- Element UI：為 Vue 2 設計
- Element Plus：為 Vue 3 重新設計

如果你的項目使用 Vue 2，建議繼續使用 Element UI。如果升級到 Vue 3，建議遷移到 Element Plus。
