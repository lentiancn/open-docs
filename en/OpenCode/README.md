# OpenCode

Open source AI coding agent.

## Overview

OpenCode is an open source AI coding agent available as a terminal interface, desktop app, or IDE extension. Features include LSP auto-enable, multi-session, and support for 75+ LLM providers.

### Key Features

- **LSP Enabled** - Automatically loads the right LSPs
- **Multi-session** - Start multiple agents in parallel
- **Any Model** - 75+ LLM providers
- **Privacy First** - No code or context data stored


```bash

/connect
```
