# OpenClaw Руководство по установке

> Полное руководство по настройке и установке OpenClaw

---

## Системные требования

| Требование | Спецификация |
|------------|--------------|
| Операционная система | Windows 10+, macOS, Linux |
| Node.js | Версия 18+ |
| Память | Не менее 4 ГБ ОЗУ |
| Место на диске | ~500 МБ |

---

## Установка

### Предварительные требования

Убедитесь, что Node.js установлен:

```bash
node --version
npm --version
```

### Пошаговая установка

#### 1. Установка через npm

```bash
npm install -g openclaw
```

#### 2. Запуск службы Gateway

Служба Gateway — это ядро OpenClaw:

```bash
# Проверить статус
openclaw gateway status

# Запустить Gateway
openclaw gateway start

# Перезапустить Gateway
openclaw gateway restart

# Остановить Gateway
openclaw gateway stop
```

#### 3. Начальная настройка

После установки настройте файлы конфигурации:

```bash
# Создать каталог конфигурации
mkdir -p ~/.openclaw

# Копировать конфигурацию по умолчанию
cp /path/to/openclaw/config/* ~/.openclaw/
```

## Конфигурация

### Основные файлы конфигурации

| Файл | Описание |
|------|----------|
| AGENTS.md | Конфигурация рабочего пространства |
| SOUL.md | Настройки личности |
| TOOLS.md | Конфигурация инструментов |
| USER.md | Настройки пользователя |

### Переменные окружения

```bash
# Опционально: пользовательский каталог рабочего пространства
OPENCLAW_WORKSPACE=/path/to/workspace

# Опционально: порт Gateway
OPENCLAW_PORT=3000
```

## Проверка

### Тестирование установки

```bash
# Показать информацию о версии
openclaw --version

# Проверить статус Gateway
openclaw gateway status

# Показать доступные команды
openclaw --help
```

### Быстрый старт

```bash
# Запустить интерактивный сеанс
openclaw chat

# Выполнить одиночную задачу
openclaw exec "сказать привет"
```

## Устранение неполадок

### Gateway не запускается?

**Решение**:
1. Проверьте правильность установки Node.js
2. Проверьте разрешения
3. Проверьте логи: `openclaw gateway logs`

### Порт уже используется?

**Решение**:
```bash
# Изменить порт
openclaw gateway start --port 3001
```

### Модуль не найден?

**Решение**:
```bash
# Переустановить
npm uninstall -g openclaw
npm install -g openclaw
```

## Удаление

```bash
npm uninstall -g openclaw

# Удалить службу Gateway
openclaw gateway stop
rm -rf ~/.openclaw
```
