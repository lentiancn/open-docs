# OpenClaw 🦞

OpenClaw ist ein Tool, das du auf deinem eigenen Gerät installierst und das deine Lieblings-Messaging-Apps mit KI-Assistenten verbindet. WhatsApp, Telegram, Discord, iMessage... alles funktioniert direkt mit KI-Diensten wie eine Brücke zwischen deinen Chats und dem künstlichen Gehirn.

## Für wen ist das?

Wenn du deinen eigenen KI-Assistenten haben möchtest, aber deine Daten nicht Drittanbietern anvertrauen willst, ist OpenClaw eine ausgezeichnete Wahl. Es ist besonders für Entwickler und Power-User geeignet, die Kommandozeilentools bevorzugen und volle Kontrolle über ihre Einrichtung wollen.

## Hauptmerkmale

- **Deine Daten, deine Maschine**: Alles bleibt auf deiner eigenen Hardware, nichts geht an externe Server
- **Mehrere Plattformen**: Ein einzelner Prozess verwaltet mehrere Chat-Apps gleichzeitig
- **Für Programmierung entwickelt**: Unterstützt Tools, Konversationsverlauf, Memory und mehrere Assistenten
- **Open Source**: MIT-Lizenz, aktive Community

## Was du brauchst

- Node.js Version 22 oder höher
- Einen API-Schlüssel von deinem bevorzugten KI-Anbieter
- Einige Minuten für die Ersteinrichtung

Für das beste Erlebnis nutze die leistungsstärksten Modelle der neuesten Generation.

## Schnellstart

```bash
# OpenClaw installieren
npm install -g openclaw@latest

# Einrichtungsassistent ausführen
openclaw onboard --install-daemon

# Dienst starten
openclaw gateway --port 18789
```

Sobald es läuft, öffne deinen Browser unter http://127.0.0.1:18789/ um loszulegen.

## Wie es funktioniert

OpenClaw arbeitet mit einem zentralen Prozess namens Gateway. Dieser Prozess ist verantwortlich für das Routing von Nachrichten, die Verwaltung der KI-Aufrufe und die Aufrechterhaltung der Verbindungen zu allen deinen Chat-Kanälen. Du musst nur diesen einen Prozess am Laufen halten und hast automatisch KI-Fähigkeiten auf allen deinen Messaging-Plattformen gleichzeitig.

## Hauptfunktionen

- **Zentrale Verwaltung**: Ein System für alle deine Chat-Kanäle
- **Plugin-Architektur**: Füge mehr Plattformen über Erweiterungspakete hinzu
- **Mehrere Assistenten**: Betreibe separate KI-Instanzen für verschiedene Zwecke
- **Dateiverwaltung**: Sende und empfange Bilder, Audio und Dokumente
- **Web-Interface**: Verwalte alles über ein Browser-Dashboard
- **Mobile Integration**: Verbinde iOS- oder Android-Geräte für Fernzugriff, Kamerazugriff und Screenshots
