# Guide d'installation d'IBM Informix

## Configuration Requise

**Plateformes Supportées :**
- Linux (Red Hat, Ubuntu, SUSE, etc.)
- Windows Server
- AIX
- HP-UX
- Solaris
- macOS
- ARM (pour les appareils edge)

**Exigences Minimales :**
- RAM : 1Go minimum (8Go recommandé)
- Disque : 2Go minimum pour l'installation
- CPU : Compatible avec les architectures x86, PowerPC, ARM

## Étapes d'Installation sous Linux

### Étape 1 : Télécharger l'Installateur

Visitez IBM Passport Advantage ou IBM Fix Central et téléchargez le package d'installation Informix approprié.

### Étape 2 : Préparer le Système

```bash
# Créer l'utilisateur et le groupe informix
groupadd informix
useradd -g informix -d /home/informix informix

# Créer le répertoire d'installation
mkdir -p /opt/IBM/informix
chown informix:informix /opt/IBM/informix
```

### Étape 3 : Extraire l'Installateur

```bash
tar -xf informix.tar
cd informix
```

### Étape 4 : Exécuter l'Installation

```bash
# En tant que root ou avec sudo
./install
```

Pour les versions plus récentes (14.10+), utilisez IBM Installation Manager :

```bash
chmod +x ids_install
./ids_install
```

Suivez les invites :
- Acceptez l'accord de licence
- Choisissez le chemin d'installation (/opt/IBM/informix)
- Sélectionnez les composants à installer

### Étape 5 : Configurer les Variables d'Environnement

Ajoutez à `~/.bashrc` ou `/etc/profile` :

```bash
export INFORMIXDIR=/opt/IBM/informix
export PATH=$INFORMIXDIR/bin:$PATH
export ONCONFIG=onconfig
export INFORMIXSERVER=votre_nom_serveur
```

Appliquez les modifications :

```bash
source ~/.bashrc
```

### Étape 6 : Configurer le Fichier sqlhosts

```bash
cd $INFORMIXDIR/etc
cp sqlhosts.std sqlhosts
vi sqlhosts
```

Ajoutez votre entrée de serveur :

```
ol_informix onsoctcp localhost ol_informix
```

### Étape 7 : Initialiser le Serveur de Base de Données

```bash
# Initialiser Informix avec les paramètres par défaut
oninit -ivy
```

### Étape 8 : Vérifier l'Installation

```bash
# Vérifier le statut du serveur
onstat -

# Vérifier l'initialisation
ls -la $INFORMIXDIR/storage/
```

## Étapes d'Installation sous Windows

### Étape 1 : Télécharger l'Installateur Windows

Téléchargez depuis IBM Passport Advantage.

### Étape 2 : Exécuter l'Installateur

- Double-cliquez sur le fichier `.exe`
- Suivez l'assistant d'installation

### Étape 3 : Configuration

- Utilisez l'Assistant de Configuration Informix
- Configurez le service Windows

### Étape 4 : Configuration des Variables d'Environnement

Configurez `INFORMIXDIR` dans les Propriétés Système :
- Clic droit sur Poste de travail → Propriétés
- Paramètres système avancés → Variables d'environnement
- Ajoutez `INFORMIXDIR` et `INFORMIXSERVER`

## Vérification Post-Installation

```bash
# Vérifier le statut du serveur
onstat -

# Démarrer le serveur (si pas déjà en cours)
oninit

# Arrêter le serveur
onmode -ky

# Se connecter à la base de données
dbaccess
```

## Tâches Courantes Post-Installation

### Créer des Espaces de Stockage Supplémentaires

```bash
onspaces -c -d datadbs -p /chemin/vers/data_chunk -s 500000
```

### Configurer les Journaux Logiques

```bash
onparams -a -d datadbs -s 10000
```

## Dépannage

### Le Serveur Ne Démarre Pas

- Vérifiez que les variables d'environnement sont correctement définies
- Examinez les fichiers `onconfig` et `sqlhosts`
- Consultez les journaux d'erreurs dans `$INFORMIXDIR/tmp/`
- Assurez-vous d'avoir suffisamment d'espace disque et les permissions

### Problèmes de Connexion

- Vérifiez les paramètres du pare-feu (port par défaut 9088)
- Vérifiez que `INFORMIXSERVER` correspond à la configuration sqlhosts
- Assurez-vous que sqlhosts a les paramètres réseau corrects
