## リアクティブシステム

### Vue 3 Composition API（推奨）

```javascript
import { ref, reactive, computed } from 'vue'

export default {
  setup() {
    // refはプリミティブ値用
    const count = ref(0)
    
    // reactiveはオブジェクト用
    const state = reactive({
      name: 'Vue',
      version: 3
    })
    
    // 算出プロパティ
    const doubleCount = computed(() => count.value * 2)
    
    // メソッド
    const increment = () => {
      count.value++
    }
    
    return { count, state, doubleCount, increment }
  }
}
```

### Vue 2 Options API

```javascript
export default {
  data() {
    return {
      count: 0,
      name: 'Vue',
      version: 2
    }
  },
  computed: {
    doubleCount() {
      return this.count * 2
    }
  },
  methods: {
    increment() {
      this.count++
    }
  }
}
```

## コンポーネント

### コンポーネントの作成

```vue
<!-- MyComponent.vue -->
<template>
  <div class="my-component">
    <h2>{{ title }}</h2>
    <p>{{ content }}</p>
    <button @click="emitEvent">クリックして</button>
  </div>
</template>

<script setup>
import { defineProps, defineEmits } from 'vue'

const props = defineProps({
  title: {
    type: String,
    required: true
  },
  content: {
    type: String,
    default: 'デフォルトコンテンツ'
  }
})

const emit = defineEmits(['click'])

const emitEvent = () => {
  emit('click', 'コンポーネントからこんにちは！')
}
</script>

<style scoped>
.my-component {
  padding: 20px;
}
</style>
```

## コンポーネント通信

### Props（親から子）

```vue

  message: String,
```

### イベント（子から親）

```vue

const emit = defineEmits(['update'])
```

## イベント処理

```html

<!-- 複数のハンドラ -->

<!-- イベント修飾子 -->
<button @click.stop="handleClick">伝播を停止</button>
<button @click.prevent="handleSubmit">デフォルト防止</button>
<button @click.stop.prevent="handleClick">両方</button>

<!-- キーボード修飾子 -->
<input @keyup.enter="submit" />
<input @keyup.ctrl.enter="submit" />

<!-- マウス修飾子 -->
<button @click.left="leftClick">左</button>
<button @click.right="rightClick">右</button>
```

## 算出プロパティ

```javascript
import { ref, computed } from 'vue'

const firstName = ref('John')
const lastName = ref('Doe')

// 算出プロパティ - キャッシュされ、依存関係が変更された時のみ再評価
const fullName = computed(() => {
  return `${firstName.value} ${lastName.value}`
})

// メソッド - 每次レンダリング時に再評価
const getFullName = () => {
  return `${firstName.value} ${lastName.value}`
}
```

## ウォッチャー

```javascript
import { ref, watch } from 'vue'

const count = ref(0)

// シンプルなウォッチャー
watch(count, (newValue, oldValue) => {
  console.log(`${oldValue}から${newValue}に変更`)
})

// ディープウォッチャー
const state = ref({ name: 'Vue' })
watch(state, (newValue) => {
  console.log('状態が変更されました：', newValue)
}, { deep: true })

// 複数のソースを監視
const firstName = ref('John')
const lastName = ref('Doe')
watch([firstName, lastName], ([newFirst, newLast]) => {
  console.log(`名前：${newFirst} ${newLast}`)
})
```

## ライフサイクルフック

### Vue 3 Composition API

| フック | 説明 |
|------|-------------|
| `setup()` | 他のフックの前に呼び出し、設定用 |
| `onMounted()` | コンポーネントがマウントされた後 |
| `onUpdated()` | コンポーネントが更新された後 |
| `onUnmounted()` | コンポーネントがアンマウントされた後 |
| `onErrorCaptured()` | エラーがキャプチャされた時 |

```javascript
import { 
  onMounted, 
  onUpdated, 
  onUnmounted 
} from 'vue'

export default {
  setup() {
    onMounted(() => {
      console.log('マウント完了！')
    })
    onUpdated(() => {
      console.log('更新完了！')
    })
    onUnmounted(() => {
      console.log('アンマウント完了！')
    })
  }
}
```

### Vue 2 Options API

| フック | 説明 |
|------|-------------|
| `beforeCreate` | インスタンス作成前 |
| `created` | インスタンス作成後 |
| `beforeMount` | マウント前 |
| `mounted` | マウント後 |
| `beforeUpdate` | 更新前 |
| `updated` | 更新後 |
| `beforeDestroy` | 破棄前 |
| `destroyed` | 破棄後 |

## Vue Router

### 基本セットアップ

```javascript
// router/index.js
import { createRouter, createWebHistory } from 'vue-router'
import Home from '../views/Home.vue'
import About from '../views/About.vue'
import User from '../views/User.vue'

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About },
  { path: '/user/:id', component: User, props: true }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```

### ナビゲーション

```vue
<template>
  <!-- プログラムによるナビゲーション -->

  <button @click="$router.back()">戻る</button>
  <button @click="$router.forward()">進む</button>
  
  <!-- ルーターリンク -->
  <router-link to="/">ホーム</router-link>
  <router-link to="/about">概要</router-link>
  <router-link :to="`/user/${userId}`">ユーザー</router-link>
</template>

<script setup>
import { useRouter } from 'vue-router'

const router = useRouter()
const userId = 123

const navigate = () => {
  router.push('/about')
}
</script>
```

## Pinia 状態管理

### ストア定義

```javascript
// stores/user.js
import { defineStore } from 'pinia'

export const useUserStore = defineStore('user', {
  state: () => ({
    name: 'John',
    email: 'john@example.com',
    age: 25
  }),
  
  getters: {
    isAdult: (state) => state.age >= 18,
    displayName: (state) => `${state.name} (${state.age})`
  },
  
  actions: {
    updateName(newName) {
      this.name = newName
    },
    async fetchUser() {
      const data = await fetch('/api/user')
      const user = await data.json()
      this.$patch(user)
    }
  }
})
```

### ストアの使用

```vue
<script setup>
import { storeToRefs } from 'pinia'
import { useUserStore } from '../stores/user'

const userStore = useUserStore()

// 状態に直接アクセス
console.log(userStore.name)

// storeToRefsでリアクティブ性を維持
const { name, email } = storeToRefs(userStore)

// アクションを 호출
userStore.updateName('Jane')

// 非同期アクション
userStore.fetchUser()
</script>
```
