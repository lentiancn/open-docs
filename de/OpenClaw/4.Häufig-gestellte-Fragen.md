# Häufig gestellte Fragen zu OpenClaw

Antworten auf häufige Fragen zu OpenClaw.

## Grundlagen

### Was ist OpenClaw?

OpenClaw ist ein selbst gehostetes Tool, das Messaging-Apps (WhatsApp, Telegram, Discord, etc.) mit KI-Assistenten verbindet. Es ermöglicht dir, KI durch Chat-Apps zu nutzen, die du bereits installiert hast, während du die volle Kontrolle über deine Daten behältst.

### Ist OpenClaw kostenlos?

Die Software selbst ist kostenlos, da sie Open Source ist. Du musst jedoch bezahlen:
- Einen Computer oder Server zum Betreiben
- API-Schlüssel-Kosten für KI-Dienste

### Brauche ich ein KI-Dienst-Abonnement?

Nicht unbedingt. Du kannst nutzungsbasierte API-Schlüssel oder Abonnement-Konten verwenden. OpenClaw unterstützt Anthropic, OpenAI, Google Gemini und andere Anbieter.

### Auf welchen Plattformen läuft es?

- **Desktop**: macOS, Linux, Windows
- **Mobil**: iOS, Android (als Companion-Knoten)
- **Chat-Apps**: WhatsApp, Telegram, Discord, iMessage, Signal, Slack und mehr als 20 weitere

## Installation

### Installation fehlgeschlagen. Was nun?

1. Node.js-Version prüfen: `node --version` (braucht 22+)
2. Diagnose ausführen: `openclaw doctor`
3. Logs prüfen: `openclaw logs`

### Kann ich es auf einem Raspberry Pi betreiben?

Ja. Raspberry Pi 4 oder neuer mit Raspberry Pi OS oder Ubuntu funktioniert einwandfrei.

### Brauche ich einen Mac?

Nein. OpenClaw läuft auf jedem System mit Node.js. Nur der iMessage-Kanal erfordert einen Mac aufgrund von Apple-Beschränkungen.

## Konfiguration

### Wo ist die Konfigurationsdatei?

Standardort: `~/.openclaw/openclaw.json`

### Wie kontrolliere ich, wer mir schreiben kann?

Nutze die `dmPolicy`-Einstellung:

- `pairing`: Unbekannte brauchen einen Kopplungscode
- `allowlist`: Nur Benutzer auf der Whitelist können schreiben
- `open`: Jeder kann schreiben
- `disabled`: Direktnachrichten deaktivieren

### Wie richte ich Gruppenchats ein?

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

## Nutzung

### Wie füge ich einen neuen Chat-Kanal hinzu?

```bash
openclaw channels login whatsapp
```

### Kann ich mehrere Kanäle gleichzeitig nutzen?

Ja. Ein Gateway-Prozess verbindet sich mit mehreren Chat-Plattformen gleichzeitig.

### Wie merkt sich die KI frühere Gespräche?

OpenClaw speichert automatisch den Gesprächskontext. Du kannst Aufbewahrungsrichtlinien in der Konfiguration anpassen.

### Kann ich mehrere KI-Agenten betreiben?

Ja. Konfiguriere mehrere Agenten mit verschiedenen Routing-Regeln:

```json5
{
  agents: {
    list: [
      { id: "home", default: true },
      { id: "work" }
    ]
  }
}
```

## Sicherheit

### Sind meine Daten sicher?

Ja. OpenClaw läuft lokal — alle Daten bleiben auf deinem Gerät und werden nicht an Dritte gesendet (außer für KI-Anbieter-API-Aufrufe).

### Wie schütze ich meine Installation?

1. Setze `allowFrom` um Zugriffsquellen einzuschränken
2. Verwende starke Passwörter und API-Schlüssel
3. Halte OpenClaw aktuell
4. Nutze Firewall um offene Ports zu begrenzen

## Fehlerbehebung

### Gateway startet nicht

1. Prüfe ob Port belegt: `openclaw gateway status`
2. Fehlerlogs prüfen: `openclaw logs`
3. Diagnose ausführen: `openclaw doctor`

### KI antwortet nicht

1. API-Schlüssel prüfen
2. Kontostand beim KI-Anbieter prüfen
3. Auf Fehler in den Logs achten

### Nachrichten werden nicht gesendet

1. Kanal-Anmeldung prüfen
2. Empfänger existiert?
3. Detaillierte Logs prüfen
