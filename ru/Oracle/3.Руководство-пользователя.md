# Oracle Руководство пользователя

## Подключение к базе данных

### Использование SQL*Plus

```bash
# Подключиться как SYSDBA
sqlplus / as sysdba

# С именем пользователя и паролем
sqlplus пользователь/пароль@база_данных
```

---

## Основные операции SQL

### Создание таблицы

```sql
CREATE TABLE сотрудники (
    сотрудник_id NUMBER(10) PRIMARY KEY,
    имя VARCHAR2(50) NOT NULL,
    фамилия VARCHAR2(50) NOT NULL,
    email VARCHAR2(100),
    зарплата NUMBER(10, 2)
);
```

### Вставка данных

```sql
INSERT INTO сотрудники (сотрудник_id, имя, фамилия, email, зарплата)
VALUES (1001, 'Иван', 'Петров', 'ivan@company.com', 75000);
```

### Запрос данных

```sql
SELECT * FROM сотрудники;
SELECT имя, фамилия, зарплата FROM сотрудники WHERE отдел_id = 10;
```

## PL/SQL

### Хранимая процедура

```sql
CREATE OR REPLACE PROCEDURE обновить_зарплату (
    p_сотрудник_id IN NUMBER,
    p_новая_зарплата IN NUMBER
) AS
BEGIN
    UPDATE сотрудники
    SET зарплата = p_новая_зарплата
    WHERE сотрудник_id = p_сотрудник_id;
    COMMIT;
END;
/
```
