# Docker 설치 가이드

이 가이드에서는 다양한 운영 체제에 Docker를 설치하는 방법에 대한 자세한 지침을 제공합니다.

## 설치 방법

여러 가지 방식으로 Docker를 설치할 수 있습니다:

1. **Docker Desktop**: 데스크톱 애플리케이션에 Docker Engine 포함, 대부분의 사용자에게 권장
2. **Docker Engine만**: Docker의 핵심 구성 요소만 설치, Linux 서버에 적합
3. **apt 저장소를 통한 Docker Engine**: Docker의 공식 apt 저장소에서 설치
4. **수동 설치**: deb 패키지를 다운로드하여 수동으로 설치

## Docker Desktop 설치

Docker Desktop은 시작하는 가장 쉽고 빠른 방법입니다. Docker Engine, Docker CLI, Docker Compose, Docker Build 및 Kubernetes를 포함한 완전한 Docker 개발 환경을 포함합니다.

### 시스템 요구 사항

**macOS:**

- macOS 10.15 (Catalina) 이상
- 최소 4GB RAM
- 최소 4GB 사용 가능한 디스크 공간

**Windows:**

- Windows 10 Pro, Enterprise 또는 Education (64비트)
- 최소 4GB RAM
- Hyper-V 또는 WSL 2 활성화

**Linux:**

- Ubuntu 20.04 이상
- Debian 11 이상
- Fedora 33 이상
- 최소 4GB RAM

### macOS에 설치

1. 공식 웹사이트에서 Mac용 Docker Desktop 다운로드
2. .dmg 파일 더블 클릭
3. Docker 아이콘을 "Applications" 폴더로 드래그
4. Docker Desktop 실행

### Windows에 설치

1. Windows용 Docker Desktop 다운로드
2. 인스톨러 실행
3. 설치 마법사의 지침 따르기
4. Docker Desktop 다시 시작

### Linux에 설치 (Ubuntu)

```bash
# 패키지 목록 업데이트
sudo apt-get update

# 필요한 의존성 설치
sudo apt-get install ca-certificates curl gnupg lsb-release

# Docker GPG 키 추가
sudo mkdir -p /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg

# 저장소 설정
echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

# Docker 설치
sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

## Docker Engine 설치

데스크톱 없이 Docker Engine만 설치하려면 다음 단계를 따르세요.

### Ubuntu 및 Debian

```bash
sudo apt-get update
sudo apt-get install docker.io
sudo systemctl start docker
sudo systemctl enable docker
```

### Fedora

```bash
sudo dnf install docker-ce
sudo systemctl start docker
sudo systemctl enable docker
```

## 설치 확인

설치 후 다음 명령으로 설치를 확인할 수 있습니다:

```bash
docker --version
docker run hello-world
```

모두 작동하면 Docker의 환영 메시지가 표시됩니다.
