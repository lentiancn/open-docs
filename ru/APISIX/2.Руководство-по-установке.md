# APISIX Руководство по установке

Несколько способов установки APISIX.

## Требования

### Аппаратные требования

- CPU: 2 ядра и более
- Оперативная память: 4GB и более
- Диск: 40GB и более

### Программные требования

- ОС: Linux (Ubuntu 20.04+, CentOS 7+)
- Nginx или OpenResty
- Lua 5.1+
- etcd 3.4+

## Установка через Docker (Рекомендуется)

### Быстрый старт

```bash
# Запустить etcd
docker run -d --name etcd -p 2379:2379 apache/apisix:latest etcd

# Запустить APISIX
docker run -d --name apisix -p 9080:9080 -p 9090:9090 apache/apisix:latest
```

### docker-compose

```yaml
version: '3'
services:
  etcd:
    image: apache/apisix:3.5.0-etcd
    ports:
      - "2379:2379"
  apisix:
    image: apache/apisix:3.5.0
    ports:
      - "9080:9080"
    depends_on:
      - etcd
```

```bash
docker-compose up -d
```

## Установка через Helm (Kubernetes)

```bash
helm repo add apisix https://apache.github.io/apisix-helm-chart
helm install apisix apisix/apisix
```

## Установка из исходного кода

### Установка зависимостей

**Ubuntu/Debian:**

```bash
wget -qO - https://openresty.org/package/pubkey.gpg | sudo apt-key add -
sudo apt-get update
sudo apt-get install -y openresty etcd
```

### Клонирование

```bash
git clone https://github.com/apache/apisix.git
cd apisix
git checkout 3.5.0
```

### Установка зависимостей

```bash
make deps
make init
```

### Запуск

```bash
make run
```

## Конфигурация

```yaml
apisix:
  node_listen: 9080
  enable_admin: true
  admin_key:
    - name: admin
      key: 123456

etcd:
  host:
    - "http://127.0.0.1:2379"
```

## Удаление

```bash
docker stop apisix etcd
docker rm apisix etcd
```
