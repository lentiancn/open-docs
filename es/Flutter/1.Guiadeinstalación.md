# Guía de instalación de Flutter

Flutter es un kit de herramientas de UI de código abierto desarrollado por Google para crear aplicaciones multiplataforma atractivas y rápidas. Esta guía cubre los métodos de instalación en todos los entornos.

## Información de versión

| Versión | Estado | Descripción |
|---------|--------|-------------|
| Flutter 3.x | ✅ Actual | Última versión estable |
| Flutter 2.x | ⚠️ Mantenimiento | Actualizaciones de seguridad |

> **Importante**: Esta guía está orientada principalmente a **Flutter 3.x**.

## Métodos de instalación

### Método 1: Instalación en Windows

#### 1. Descargar Flutter SDK

1. Descarga Flutter SDK: https://docs.flutter.dev/get-started/install/windows
2. Extrae en el directorio deseado (ej. C:\src\flutter)

#### 2. Configurar variables de entorno

```powershell
# Añadir a PATH
$env:PATH += ";C:\src\flutter\bin"
```

#### 3. Ejecutar Flutter doctor

```powershell
flutter doctor
```

### Método 2: Instalación en macOS

#### 1. Usar Homebrew (recomendado)

```bash
# Instalar Flutter
brew install flutter

# O descargar manualmente
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable
```

#### 2. Configurar variables de entorno

```bash
# ~/.bash_profile o ~/.zshrc
export PATH="$PATH:`pwd`/flutter/bin"
```

#### 3. Ejecutar Flutter doctor

```bash
flutter doctor
```

### Método 3: Instalación en Linux

#### 1. Instalar dependencias

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa clang cmake ninja-build pkg-config libgtk-3-dev

# Fedora
sudo dnf install -y curl git unzip xz zip mesa-libGLU clang cmake ninja-build pkgconfig gtk3-devel
```

#### 2. Descargar Flutter SDK

```bash
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable
export PATH="$PATH:~/development/flutter/bin"
```

#### 3. Ejecutar Flutter doctor

```bash
flutter doctor
```

### Método 4: Usar VS Code

1. Descargar e instalar VS Code
2. Abrir VS Code
3. Instalar extensiones de Flutter:
   - Buscar "Flutter" e instalar
   - Buscar "Dart" e instalar

### Método 5: Usar Android Studio

1. Descargar e instalar Android Studio
2. Instalar el plugin de Flutter:
   - Abrir Android Studio
   - Settings → Plugins
   - Buscar "Flutter" e instalar

## Configuración de Android

### Instalar Android SDK

```bash
# Instalar usando herramientas de Flutter
flutter doctor --android-licenses
flutter config --android-sdk /path/to/android/sdk
```

### Configurar variables de entorno

```bash
# macOS/Linux
export ANDROID_SDK_ROOT=/Users/user/Library/Android/sdk
export PATH=$PATH:$ANDROID_SDK_ROOT/cmdline-tools/latest/bin:$ANDROID_SDK_ROOT/platform-tools
```

### Crear emulador

```bash
# Listar dispositivos disponibles
flutter devices

# Crear emulador
flutter emulator --create --name my-emulator

# Iniciar emulador
flutter emulators --launch my-emulator
```

## Problemas comunes

### ¿Error en Flutter doctor?

Asegúrate de que todas las dependencias estén instaladas:

```bash
# Ejecutar flutter doctor
flutter doctor -v
```

### ¿No se encuentra Android SDK?

Establecer la ruta del SDK:

```bash
flutter config --android-sdk /path/to/android/sdk
```

### ¿Fallos de compilación?

Limpiar y recompilar:

```bash
flutter clean
flutter pub get
flutter build apk
```
