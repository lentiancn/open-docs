# Element UI

Element UI ist eine auf Vue 2.0 basierende UI-Komponentenbibliothek, die Entwicklern umfassende Desktop-Komponentenlösungen bietet.

## Übersicht

Element UI ist eine von Ele.me entwickelte und gepflegte Vue 2.0-Komponentenbibliothek. Sie bietet einen vollständigen Satz von Designspezifikationen und Komponentenbibliotheken, um Entwicklern zu helfen, schnell Desktop-Anwendungen zu erstellen.

## Hauptfunktionen

- Vollständige Vue 2.0-Unterstützung: Auf Vue 2.0 aufgebaut
- Über 70 Komponenten: Deckt alle Aspekte der Desktop-Entwicklung ab
- Import bei Bedarf: Mit babel-plugin-component
- Internationalisierung: Unterstützung mehrerer Sprachen
- Theme-Anpassung: Flexible Theming-Konfiguration

## Schnellstart

### Installation

npm install element-ui --save

### Verwendung

Vollständiger Import:

import Vue from 'vue'
import Element from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(Element)

Import bei Bedarf (empfohlen):

Zuerst babel-plugin-component installieren:

npm install -D babel-plugin-component
