# Apifox

API 一体型协作プラットフォーム。

## 概要

Apifoxは、APIドキュメント、デバッグ、Mock、自動化テスト于一体的プラットフォームです。

### 主な機能

- **APIドキュメント** - 自動生成
- **APIデバッグ** - HTTPリクエストシミュレーション
- **Mockサービス** - 快速Mockサーバー
- **自動化テスト** - API自動化テスト
- **チーム协作** - プロジェクト管理
