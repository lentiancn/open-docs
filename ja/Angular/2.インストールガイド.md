# Angular インストールガイド

Angular 環境設定の詳細ガイド。

## 必要環境

### Node.js

- 推奨：Node.js 18+
- npm または yarn が必要

### Node.js バージョンを確認

```bash
node --version
npm --version
```

## Angular CLI をインストール

### インストール

```bash
npm install -g @angular/cli
```

### インストール確認

```bash
ng version
```

### アップグレード

```bash
npm install -g @angular/cli@latest
```

## 新規プロジェクト作成

### Angular CLI を使用

```bash
ng new my-app
cd my-app
```

### インタラクティブオプション

- ルーティングを追加しますか？ Yes/No
- 哪种スタイルシート形式？（SCSS/CSS/Sass/Less）

## アプリケーション実行

### 開発サーバー

```bash
ng serve
# または
npm start
```

アクセス：http://localhost:4200

### ホットリロード

Angular CLI はホットリロードをサポート、変更すると自動更新。

## プロダクションビルド

```bash
ng build --configuration production
```

出力は `dist/` ディレクトリ。

## よく使うコマンド

| コマンド | 機能 |
|----------|------|
| ng new | 新規プロジェクト作成 |
| ng serve | 開発サーバー起動 |
| ng build | プロジェクトビルド |
| ng generate | コンポーネント/サービス/モジュール生成 |
| ng test | ユニットテスト実行 |
| ng e2e | E2Eテスト実行 |

## プロジェクト構造

```
my-app/
├── src/
│   ├── app/
│   │   ├── components/
│   │   ├── services/
│   │   ├── app.component.ts
│   │   └── app.module.ts
│   ├── assets/
│   └── styles.css
├── angular.json
└── package.json
```

## Angular CLI をアンインストール

```bash
npm uninstall -g @angular/cli
```
