# Guide d'installation d'Angular

Angular est un framework frontend puissant développé par Google. Ce guide couvre les méthodes d'installation pour différents environnements.

## Informations de version

| Version | Statut | Description |
|---------|--------|-------------|
| Angular 18.x | ✅ Actuelle | Dernière version |
| Angular 17.x | ✅ Supportée | Version précédente |
| Angular 16.x | ✅ Supportée | Version précédente |
| Angular 15.x | ✅ Supportée | Version legacy |

## Configuration requise

### Matériel

| Exigence | Minimum | Recommandé |

| Logiciel | Version |

## Méthodes d'installation

### Méthode 1 : Angular CLI (Recommandé)

#### Installer Angular CLI

```bash
# Installer globalement
npm install -g @angular/cli

# Vérifier l'installation
ng version
```

#### Créer un nouveau projet

```bash
# Créer un nouveau projet
ng new my-app

# Créer avec des options
ng new my-app --routing --style=scss --skip-tests
```

#### Options

```bash
# Ignorer l'installation
ng new my-app --skip-install

# Composants autonomes (Angular 14+)
ng new my-app --standalone

# Activer SSR
ng new my-app --ssr
```

### Méthode 2 : Utiliser npx

```bash
# Créer un projet sans installer CLI
npx @angular/cli new my-app
```

### Méthode 3 : Configuration manuelle

```bash
# Créer le répertoire
mkdir my-app
cd my-app

# Initialiser npm
npm init -y

# Installer Angular
npm install @angular/core @angular/common @angular/platform-browser @angular/platform-browser-dynamic @angular/compiler @angular/forms @angular/router @angular/animations rxjs zone.js

# Installer les dépendances de développement
npm install --save-dev @angular/compiler-cli @angular-devkit/build-angular typescript
```

### Méthode 4 : Docker

```dockerfile
# Dockerfile
FROM node:20-alpine

WORKDIR /app

# Installer Angular CLI
RUN npm install -g @angular/cli

# Créer le projet
RUN ng new my-app --directory .

# Démarrer le serveur de développement
CMD ["ng", "serve", "--host", "0.0.0.0"]
```

## Configuration de l'environnement

### Gestion des versions Node.js

#### Utiliser nvm (Recommandé)

```bash
# Installer nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Installer Node.js
nvm install 20
nvm use 20

# Définir par défaut
nvm alias default 20
```

### Configuration npm

```bash
# Définir le registre registry https://registry.npmjs.org/

# Activer SSL strict
npm config set
npm config set strict-ssl true

# Vider le cache
npm cache clean --force
```

## Vérification

```bash
# Vérifier Angular CLI
ng version

# Vérifier Node.js
node --version

# Vérifier npm
npm --version
```

## Problèmes courants

### Permission refusée ?

```bash
# Réparer les permissions npm
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
export PATH=~/.npm-global/bin:$PATH
```

### Installation lente ?

```bash
# Utiliser npm ci pour une installation plus rapide
npm ci

# Ou utiliser yarn
npm install -g yarn
```

## Mettre à jour Angular

```bash
# Mettre à jour Angular CLI
npm install -g @angular/cli@latest

# Mettre à jour votre projet
ng update @angular/core @angular/cli
```
