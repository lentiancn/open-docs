# Flutter - Manual de Usuario

Ejemplos básicos de desarrollo Flutter.

## Crear proyecto

```bash
flutter create mi_app
cd mi_app
flutter run
```

## Widgets básicos

### Texto

```dart
Text('Hola Flutter')
```

### Botones

```dart
ElevatedButton(onPressed: () {}, child: Text('Clicar'))
```

## Layout

```dart
Row(children: [Text('A'), Text('B')])
Column(children: [Text('A'), Text('B')])
```

## Estado

```dart
class Contador extends StatefulWidget {
  @override
  _ContadorState createState() => _ContadorState();
}

class _ContadorState extends State<Contador> {
  int _cuenta = 0;
  void _incrementar() => setState(() => _cuenta++);

  @override
  Widget build(BuildContext context) {
    return Column(children: [
      Text('Cuenta: $_cuenta'),
      ElevatedButton(onPressed: _incrementar, child: Text('Aumentar')),
    ]);
  }
}
```
