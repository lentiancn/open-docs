# Angular

Framework moderno de desarrollo de aplicaciones web.

## Resumen

Angular es un framework de aplicaciones web de código abierto desarrollado y mantenido por Google, proporcionando una solución completa para el desarrollo moderno de aplicaciones web.

### Características principales

- **TypeScript nativo**: Construido con TypeScript
- **Basado en componentes**: Fácil mantenimiento y pruebas
- **Inyección de dependencias**: Sistema poderoso de DI
- **Enrutamiento**: Enrutamiento completo del lado del cliente
- **Formularios**: Soporte de formularios reactivos
- **Cliente HTTP**: Servicios HTTP completos

## Inicio rápido

### Instalar Angular CLI

```bash
npm install -g @angular/cli
```

### Crear nuevo proyecto

```bash
ng new my-app
cd my-app
ng serve
```

### Acceder a la aplicación

Abrir navegador: http://localhost:4200

## Comandos comunes

```bash
# Crear componente
ng g c my-component

# Crear servicio
ng g s my-service

# Construir para producción
ng build

# Ejecutar pruebas
ng test
```

## Conceptos centrales

- **Componente**: Combinación de vista y lógica
- **Módulo**: Organizar código de aplicación
- **Service**: Encapsular lógica de negocio
- **Enrutamiento**: Navegación de páginas
- **Directiva**: Extender HTML
- **Pipe**: Transformación de datos
