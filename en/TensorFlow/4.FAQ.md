# TensorFlow Frequently Asked Questions

## General Questions

### What is TensorFlow?
TensorFlow is an open-source machine learning framework developed by Google. It provides tools for building and training neural networks, with support for both research and production deployment.

### What programming languages does TensorFlow support?
- Python (primary)
- JavaScript
- Java
- C++
- Swift
- Go

### What is the difference between PyTorch and TensorFlow?
| Aspect | TensorFlow | PyTorch |
|--------|------------|---------|
| Dynamic Graphs | Limited (TF 2.x) | Native |
| Debugging | Harder | Easier |
| Production | TensorFlow Serving | TorchServe |
| Community | Larger | Growing fast |
| Learning Curve | Steeper | Easier |

## Installation Questions

### How do I install TensorFlow?
```bash
# CPU version
pip install tensorflow

# GPU version
pip install tensorflow-gpu

# Latest version
pip install tf-nightly
```

### What are the system requirements?
- **Python**: 3.7-3.11
- **OS**: Linux, macOS, Windows
- **GPU**: NVIDIA GPU with CUDA (for GPU version)
- **RAM**: 8GB+ recommended

### How do I verify installation?
```python
import tensorflow as tf
print(tf.__version__)
print("GPU Available:", tf.config.list_physical_devices('GPU'))
```

## Usage Questions

### How do I build a simple neural network?
```python
import tensorflow as tf
from tensorflow import keras

# Load data
mnist = keras.datasets.mnist
(x_train, y_train), (x_test, y_test) = mnist.load_data()

# Normalize
x_train, x_test = x_train / 255.0, x_test / 255.0

# Build model
model = keras.Sequential([
    keras.layers.Flatten(input_shape=(28, 28)),
    keras.layers.Dense(128, activation='relu'),
    keras.layers.Dense(10)
])

# Compile and train
model.compile(optimizer='adam',
              loss=tf.keras.losses.SparseCategoricalCrossentropy(from_logits=True),
              metrics=['accuracy'])

model.fit(x_train, y_train, epochs=5)
```

### How do I use TensorBoard?
```python
tensorboard_callback = keras.callbacks.TensorBoard(log_dir='./logs')
model.fit(x_train, y_train, epochs=5, callbacks=[tensorboard_callback])
```

Run in terminal:
```bash
tensorboard --logdir=./logs
```

### How do I save and load models?
```python
# Save
model.save('my_model.keras')

# Load
model = keras.models.load_model('my_model.keras')
```

## GPU Questions

### How do I enable GPU support?
```python
gpus = tf.config.list_physical_devices('GPU')
if gpus:
    for gpu in gpus:
        tf.config.experimental.set_memory_growth(gpu, True)
```

### What if I don't have a GPU?
Use the CPU version of TensorFlow. Training will be slower but works on any machine.

### Which CUDA version do I need?
TensorFlow 2.x requires CUDA 11.0+ and cuDNN 8.0+.

## Troubleshooting

### Out of memory errors
- Reduce batch size
- Use `tf.config.experimental.set_memory_growth`
- Clear Keras session with `tf.keras.backend.clear_session()`

### Import errors
- Check Python version compatibility
- Reinstall TensorFlow: `pip uninstall tensorflow && pip install tensorflow`
- Check for conflicting packages

### Slow training
- Enable GPU acceleration
- Use mixed precision training
- Enable XLA compilation
- Optimize data pipeline

## Advanced Questions

### What is Keras?
Keras is TensorFlow's high-level API for building neural networks. It's included in TensorFlow 2.x as `tf.keras`.

### How do I use custom layers?
```python
class MyLayer(keras.layers.Layer):
    def __init__(self, units, **kwargs):
        super(MyLayer, self).__init__(**kwargs)
        self.units = units
    
    def build(self, input_shape):
        self.kernel = self.add_weight(
            name='kernel',
            shape=(input_shape[-1], self.units),
            initializer='glorot_uniform',
            trainable=True)
    
    def call(self, inputs):
        return tf.matmul(inputs, self.kernel)
```

### How do I use TensorFlow Extended (TFX)?
TFX provides a production ML pipeline:
- ExampleGen: Data ingestion
- StatisticsGen: Data statistics
- SchemaGen: Schema definition
- ExampleValidator: Data validation
- Trainer: Model training
- Evaluator: Model evaluation
- Pusher: Model deployment
