# FAQ по Bootstrap

> Часто задаваемые вопросы

---

## Проблемы с установкой

### CDN не загружается?

**Решения**:

1. Проверьте подключение к сети
2. Используйте альтернативный CDN:
```html
<link href="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

### Стили не работают после установки через npm?

Убедитесь, что импорт выполнен правильно:

```javascript
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap';
```

---

## Проблемы со стилями

### Стили компонентов не работают?

**Распространенные причины**:

1. Bootstrap CSS не импортирован
2. JavaScript загружен неправильно (необходим для некоторых компонентов)
3. Приоритет CSS переопределен

### Адаптивность не работает?

Убедитесь, что добавлен мета-тег viewport:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

---

## Проблемы с компонентами

### Выпадающий список не работает?

**Контрольный список**:

1. Импортировать Bootstrap JS (не только CSS)
2. Использовать правильную HTML-структуру
3. Проверить наличие JavaScript-ошибок

### Модальное окно не открывается?

Убедитесь, что:
1. Bootstrap JS загружен
2. Атрибуты data-bs-toggle и data-bs-target правильные
3. Модальное окно существует при загрузке страницы

---

## Проблемы с настройкой

### Как настроить цвета темы?

**Способ 1: Использовать Sass**

```scss
$primary: #ff6600;
@import "bootstrap";
```

**Способ 2: Использовать CSS-переменные**

```css
:root {
  --bs-primary: #ff6600;
}
```

---

## Проблемы совместимости

### IE 11 не поддерживается?

Bootstrap 5 больше не поддерживает IE 11. Если вам нужна поддержка IE 11, используйте Bootstrap 4.

---

## Проблемы с производительностью

### Как уменьшить размер файла?

1. Импортировать только нужные компоненты (Sass)
2. Использовать продакшен-версии (.min.css/.min.js)
3. Включить gzip-сжатие

---

## Конфликты с другими фреймворками

### Использовать с Vue/React?

Bootstrap можно использовать с любым JavaScript-фреймворком, но учтите:

1. Некоторые компоненты зависят от jQuery (Bootstrap 5 больше не зависит)
2. Используйте специфичные для фреймворка библиотеки-обертки Bootstrap (например, react-bootstrap)
