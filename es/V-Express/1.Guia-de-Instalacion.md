# V-Express Guía de Instalación

Esta guía te lleva a través de la instalación de V-Express.

## Requisitos del Sistema

- GPU NVIDIA con 8GB+ VRAM (recomendado)
- 16GB RAM mínimo
- 50GB+ espacio en disco
- Python 3.8+

## Instalación

### 1. Clonar Repositorio

```bash
git clone https://github.com/TencentARC/V-Express.git
cd V-Express
```

### 2. Crear Entorno

```bash
conda create -n vexpress python=3.8
conda activate vexpress
```

### 3. Instalar Dependencias

```bash
# Instalar PyTorch
pip install torch==1.12.1 torchvision==0.13.1 --extra-index-url https://download.pytorch.org/whl/cu113

# Instalar dependencias
pip install opencv-python
pip install opencv-python-headless
pip install librosa
pip install scipy
pip install tqdm
pip install pyyaml
pip install pillow

# Instalar face alignment
pip install face-alignment
```

### 4. Descargar Modelos

```bash
bash scripts/download_models.sh
```

### 5. Verificar Instalación

```bash
python inference.py --help
```

## Estructura de Archivos

```
V-Express/
├── configs/
├── scripts/
│   └── download_models.sh
├── inference.py
├── requirements.txt
└── README.md
```

## Configuración Adicional

### Variables de Entorno

```bash
# Para Windows
set PYTHONPATH=%PYTHONPATH%;C:\path\to\V-Express

# Para Linux/macOS
export PYTHONPATH=$PYTHONPATH:/path/to/V-Express
```

## Solución de Problemas

### Error de CUDA
Asegúrate de tener los drivers NVIDIA correctos y CUDA instalado.

### Error de Memoria
- Reducir tamaño de batch
- Usar modelo más pequeño

### Modelos No Descargados
Descarga manualmente desde las releases.
