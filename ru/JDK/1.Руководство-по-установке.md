# Руководство по установке JDK

JDK (Java Development Kit) — официальный комплект разработки на Java. В этом руководстве описаны способы установки в различных средах.

## Информация о версиях

| Версия | Статус | Описание |
|--------|--------|----------|
| JDK 21 (LTS) | ✅ Текущая | Последняя LTS-версия |
| JDK 17 (LTS) | ✅ Поддерживается | Предыдущая LTS |
| JDK 11 (LTS) | ✅ Поддерживается | Устаревшая LTS |
| JDK 8 | ✅ Поддерживается | Устаревшая версия |
| JDK 22 | ✅ Последняя | Последняя версия с новыми функциями |

### LTS и версии с новыми функциями

- **LTS (Long Term Support)**: 2-3 года обновлений, рекомендуется для продакшена
- **Версии с новыми функциями**: Новые функции каждые 6 месяцев, более короткая поддержка

## Способы установки

### Способ 1: Oracle JDK

#### Windows

1. Загрузите с: https://www.oracle.com/java/technologies/downloads/
2. Запустите установщик (.exe)
3. Следуйте инструкциям

#### macOS

```bash
# Загрузите DMG с сайта Oracle
# Или используйте Homebrew
brew install openjdk@21
```

#### Linux

```bash
# Загрузите tar.gz
wget https://download.oracle.com/java/21/latest/jdk-21_linux-x64_bin.tar.gz
tar -xzf jdk-21_linux-x64_bin.tar.gz
sudo mv jdk-21 /opt/jdk-21
```

### Способ 2: Eclipse Temurin (Adoptium)

#### Windows

1. Загрузите с: https://adoptium.net/temurin/releases/
2. Запустите установщик

#### macOS

```bash
# Используя Homebrew
brew install --cask temurin
```

#### Linux

```bash
# Ubuntu/Debian
wget https://github.com/adoptium/temurin21-binaries/releases/download/jdk-21.0.2%2B13/OpenJDK21U-jdk_x64_linux_hotspot_21.0.2_13.tar.gz
tar -xzf OpenJDK21U-jdk_x64_linux_hotspot_21.0.2_13.tar.gz

# Настройка переменных
export JAVA_HOME=/path/to/jdk-21
export PATH=$JAVA_HOME/bin:$PATH
```

### Способ 3: SDKMAN

```bash
# Установка SDKMAN
curl -s "https://get.sdkman.io" | bash

# Установка JDK
sdk install java 21.0.2-tem

# Список доступных версий
sdk list java
```

### Способ 4: Docker

```dockerfile
FROM eclipse-temurin:21-jdk
WORKDIR /app
COPY . .
RUN javac Main.java
CMD ["java", "Main"]
```

### Способ 5: Менеджеры пакетов Windows

#### Chocolatey

```powershell
choco install openjdk21
```

## Настройка переменных среды

### Windows

```powershell
# Установить JAVA_HOME
[System.Environment]::SetEnvironmentVariable("JAVA_HOME", "C:\Program Files\Java\jdk-21", "User")

# Добавить в PATH
[System.Environment]::SetEnvironmentVariable("PATH", "$env:PATH;C:\Program Files\Java\jdk-21\bin", "User")
```

### macOS/Linux

```bash
# Добавить в ~/.bashrc или ~/.zshrc
export JAVA_HOME=/opt/jdk-21
export PATH=$JAVA_HOME/bin:$PATH

# Применить
source ~/.bashrc
```

## Проверка установки

```bash
# Проверить версию Java
java -version

# Проверить версию Javac
javac -version

# Проверить JAVA_HOME
echo $JAVA_HOME
```

## Работа с несколькими версиями JDK

### Использование SDKMAN

```bash
# Установить несколько версий
sdk install java 17.0.8-tem
sdk install java 21.0.2-tem

# Переключить версию
sdk use java 17.0.8-tem
```

## Частые проблемы

### "java" не найдена?

Проверьте, что PATH включает каталог bin Java.

### Неправильная версия Java?

Проверьте, какая java используется:

```bash
which java
java -version
```

### JAVA_HOME не установлен?

Установите JAVA_HOME на каталог установки JDK.
