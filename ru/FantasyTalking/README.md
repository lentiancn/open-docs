# FantasyTalking

## Обзор

FantasyTalking — проект с открытым исходным кодом для синтеза речи и диалоговых систем.

## Особенности

- Высококачественный синтез речи
- Поддержка нескольких языков
- Гибкая диалоговая система
- Кроссплатформенность

## Установка

```bash
npm install fantasy-talking
# или
pip install fantasy-talking
```
