# Go 使用手冊

## 基本語法

### 變數宣告

Go 語言有多種變數宣告方式。基本的宣告方式使用 var 關鍵字：

```go
// 宣告變數，指定類型
var name string = "Go"

// 類型推斷宣告
var age = 10

// 批量宣告
var (
    firstName string
    lastName  string
    score     float64
)

// 簡短宣告（函數內部）
message := "Hello, World!"
```

Go 語言的變數宣告後必須使用，否則會編譯錯誤。如果需要宣告暫時不使用的變數，可以使用下底線 _ 接收：

```go
result, _ := someFunction()  // 忽略第一個回傳值
```

### 常數宣告

常數使用 const 關鍵字宣告，常數可以是字元、字串、布林值或數值：

```go
const Pi = 3.14159
const (
    StatusOK = 200
    StatusNotFound = 404
)
```

### 基本資料類型

Go 語言的基本資料類型包括：

- 布林類型：bool（true 或 false）
- 整數：int、int8、int16、int32、int64，以及對應的無符號類型 uint 等
- 浮點數：float32、float64
- 複數：complex64、complex128
- 字串：string

類型轉換需要顯式進行：

```go
var i int = 42
var f float64 = float64(i)
var s string = strconv.Itoa(i)  // 整數轉字串
```

## 函數

### 函數定義

Go 函數的定義使用 func 關鍵字：

```go
func greet(name string) string {
    return "Hello, " + name
}

// 多回傳值
func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}

// 命名回傳值
func split(sum int) (x, y int) {
    x = sum * 4 / 9
    y = sum - x
    return  // 不需要指定回傳值
}
```

### 可變參數

函數可以接受可變數量的參數：

```go
func sum(nums ...int) int {
    total := 0
    for _, n := range nums {
        total += n
    }
    return total
}

// 使用
result := sum(1, 2, 3, 4, 5)
```

### 閉包

Go 支援閉包，即函數可以捕獲並修改外部作用域的變數：

```go
func counter() func() int {
    count := 0
    return func() int {
        count++
        return count
    }
}

// 使用
c := counter()
fmt.Println(c())  // 1
fmt.Println(c())  // 2
```

## 並發程式設計

### Goroutine

Goroutine 是 Go 語言中的輕量級執行單元，創建成本極低：

```go
// 啟動 goroutine
go func() {
    fmt.Println("Hello from goroutine!")
}()

// 帶參數的 goroutine
go func(msg string) {
    fmt.Println(msg)
}("Hello!")
```

### Channel

Channel 是 goroutine 之間的通訊機制：

```go
// 建立 channel
ch := make(chan string)

// 發送資料
ch <- "message"

// 接收資料
msg := <-ch

// 帶緩衝的 channel
bufferedCh := make(chan int, 10)
```

### Select

Select 語句用於同時等待多個 channel 操作：

```go
select {
case msg := <-ch1:
    fmt.Println("Received:", msg)
case msg := <-ch2:
    fmt.Println("Received:", msg)
case <-time.After(time.Second):
    fmt.Println("Timeout")
}
```

### 並發安全

使用 sync 包進行並發控制：

```go
var (
    mu    sync.Mutex
    count int
)

func increment() {
    mu.Lock()
    defer mu.Unlock()
    count++
}
```

## 錯誤處理

### 錯誤回傳

Go 使用回傳值進行錯誤處理，這是 Go 的慣用方式：

```go
result, err := doSomething()
if err != nil {
    return err
}
```

### 自訂錯誤

可以使用 errors 包建立自訂錯誤：

```go
import "errors"

func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}
```

### 錯誤包裝

Go 1.13 引入了錯誤包裝，可以使用 fmt.Errorf 和 %w 動詞：

```go
if err != nil {
    return fmt.Errorf("failed to do something: %w", err)
}
```

## 介面

### 介面定義

Go 的介面是隱式實現的，不需要顯式宣告：

```go
type Writer interface {
    Write([]byte) (int, error)
}

type Reader interface {
    Read([]byte) (int, error)
}

// 常用組合介面
type ReadWriter interface {
    Reader
    Writer
}
```

### 空介面

空介面 interface{} 可以儲存任意類型的值：

```go
var i interface{} = 42
var s interface{} = "hello"
var m interface{} = map[string]int{"a": 1}
```

Go 1.18 引入了泛型，可以使用 any 替代 interface{}：

```go
var i any = 42
```

## 包和模組

### 建立模組

使用 go mod init 建立新模組：

```bash
go mod init github.com/username/project
```

### 依賴管理

新增依賴：

```bash
go get github.com/pkg/errors
```

更新依賴：

```bash
go get -u all
```

清理未使用的依賴：

```bash
go mod tidy
```

## 測試

### 單元測試

Go 內建了測試框架，測試檔案以 _test.go 結尾：

```go
import "testing"

func TestAdd(t *testing.T) {
    result := Add(2, 3)
    if result != 5 {
        t.Errorf("Add(2, 3) = %d; want 5", result)
    }
}
```

執行測試：

```bash
go test ./...
go test -v ./...
go test -cover ./...
```

### 基準測試

測試函數以 Benchmark 開頭：

```go
func BenchmarkHello(b *testing.B) {
    for i := 0; i < b.N; i++ {
        Hello()
    }
}
```

執行基準測試：

```bash
go test -bench=.
```

## 常用標準庫

### fmt - 格式化 I/O

```go
fmt.Println("Hello")
fmt.Printf("Name: %s, Age: %d\n", name, age)
fmt.Sprintf("Result: %d", result)
```

### strings - 字串操作

```go
strings.Contains(s, "sub")
strings.Split(s, ",")
strings.TrimSpace(s)
strings.ToUpper(s)
```

### net/http - HTTP 服務

```go
http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
    fmt.Fprint(w, "Hello, HTTP!")
})
http.ListenAndServe(":8080", nil)
```

### json - JSON 處理

```go
// 編碼
data, _ := json.Marshal(person)

// 解碼
var p Person
json.Unmarshal(data, &p)
```

## 常用工具

### gofmt - 程式碼格式化

```bash
gofmt -w main.go  # 格式化並寫入檔案
gofmt -d main.go  # 顯示格式化差異
```

### go vet - 靜態分析

```bash
go vet ./...  # 檢查程式碼問題
```

### go doc - 檔案查看

```go
go doc fmt.Println  # 查看函數檔案
go doc fmt          # 查看包檔案
```
