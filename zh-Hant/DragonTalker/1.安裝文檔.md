# DragonTalker 安裝指南

DragonTalker 是一個 AI 專案，用於從單張圖像和音頻生成逼真的說話頭像視頻，類似於 SadTalker。它使用深度學習透過音頻驅動的面部運動來將靜態圖像動畫化。

## 版本概述

| 版本 | 狀態 | 描述 |
|------|------|------|
| DragonTalker 1.x | 當前 | 主要版本 |

**注意：** DragonTalker 是一項新興技術。對於更成熟的解決方案，請考慮 SadTalker 或 Wav2Lip。

## 系統要求

### 硬體要求

| 組件 | 最低配置 | 推薦配置 |
|------|----------|----------|
| GPU | NVIDIA 6GB 顯存 | NVIDIA 16GB 顯存 |
| RAM | 8GB | 32GB |
| 儲存 | 20GB | 50GB |
| CUDA | 11.7+ | 11.8+ |

### 軟體要求

| 軟體 | 版本 | 說明 |
|------|------|------|
| Python | 3.8 - 3.10 | 推薦 3.9 |
| CUDA | 11.7+ | GPU 加速必需 |
| cuDNN | 8.5+ | CUDA 必需 |
| ffmpeg | 最新版 | 視頻處理必需 |

## Linux/Ubuntu 安裝

### 步驟 1：安裝系統依賴

```bash
# 更新系統
sudo apt update && sudo apt upgrade -y

# 安裝 Python 和開發工具
sudo apt install python3.9 python3-pip python3-venv git wget curl

# 安裝系統庫
sudo apt install libgl1-mesa-glx libglib2.0-0 libsm6 libxext6 libxrender-dev libgomp1

# 安裝 ffmpeg
sudo apt install ffmpeg
```

### 步驟 2：安裝 CUDA（如需要）

```bash
# 下載並安裝 CUDA 11.8
wget https://developer.download.nvidia.com/compute/cuda/11.8.0/local_installers/cuda_11.8.0_520.61.05_linux.run
sudo sh cuda_11.8.0_520.61.05_linux.run
```

### 步驟 3：克隆項目

```bash
git clone https://github.com/your-repo/DragonTalker.git
cd DragonTalker
```

### 步驟 4：創建虛擬環境

```bash
python3.9 -m venv venv
source venv/bin/activate

# 驗證 Python 版本
python --version
```

### 步驟 5：安裝 PyTorch

```bash
# CUDA 11.8 版本
pip install torch==2.0.1 torchvision==0.15.2 --index-url https://download.pytorch.org/whl/cu118

# 驗證安裝
python -c "import torch; print(torch.cuda.is_available())"
```

### 步驟 6：安裝依賴

```bash
pip install -r requirements.txt

# 安裝額外依賴
pip install numpy opencv-python librosa soundfile
```

### 步驟 7：下載模型

```bash
# 創建模型目錄
mkdir -p checkpoints

# 下載模型（請查看項目的具體說明）
# bash scripts/download_models.sh
```

## Windows 安裝

### 步驟 1：安裝 Python

1. 從 python.org 下載 Python 3.9
2. 安裝時勾選「Add Python to PATH」
3. 驗證：`python --version`

### 步驟 2：安裝 CUDA

1. 從 NVIDIA 網站下載 CUDA 11.8
2. 使用默認設置安裝
3. 下載並安裝 cuDNN 8.9

### 步驟 3：克隆和設置

```powershell
git clone https://github.com/your-repo/DragonTalker.git
cd DragonTalker

python -m venv venv
venv\Scripts\activate
```

### 步驟 4：安裝依賴

```powershell
# 安裝 PyTorch（CUDA 支持）
pip install torch==2.0.1 torchvision==0.15.2 --index-url https://download.pytorch.org/whl/cu118

pip install -r requirements.txt
```

### 步驟 5：下載模型

從項目 releases 頁面下載模型，放入 `checkpoints/` 目錄。

## macOS 安裝

### 步驟 1：安裝依賴

```bash
# 安裝 Homebrew（如未安裝）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安裝 Python 和 ffmpeg
brew install python@3.9 ffmpeg git
```

### 步驟 2：設置

```bash
git clone https://github.com/your-repo/DragonTalker.git
cd DragonTalker

python3.9 -m venv venv
source venv/bin/activate

# 安裝 PyTorch（macOS 僅 CPU）
pip install torch==2.0.1 torchvision==0.15.2

pip install -r requirements.txt
```

注意：macOS 僅支持 CPU 模式，無法使用 GPU 加速。

## Docker 安裝

### 使用 Docker

```bash
# 拉取鏡像
docker pull your-registry/dragontalker:latest

# 運行（需要 GPU）
docker run --gpus all -v $(pwd):/workspace -p 8888:8888 your-registry/dragontalker:latest
```

### 使用 Docker Compose

```yaml
version: '3.8'
services:
  dragontalker:
    image: your-registry/dragontalker:latest
    volumes:
      - ./workspace:/workspace
    ports:
      - "8888:8888"
    deploy:
      resources:
        reservations:
          devices:
            - driver: nvidia
              count: 1
              capabilities: [gpu]
```

## 常見問題

### CUDA 記憶體不足

**解決方案：**
- 減小批次大小：`--batch_size 1`
- 使用較低分辨率：`--size 256`
- 啟用模型卸載

### 缺少庫（Windows）

**解決方案：**
- 安裝 Visual C++ Redistributable 2015-2022
- 將 CUDA 添加到系統 PATH

### 模型下載失敗

**解決方案：**
- 使用 VPN 或代理
- 從 GitHub releases 手動下載
- 檢查網絡連接

### 找不到 ffmpeg

**解決方案：**
- Linux：`sudo apt install ffmpeg`
- Windows：將 ffmpeg 添加到 PATH
- macOS：`brew install ffmpeg`
