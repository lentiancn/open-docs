# Element Plus ユーザーマニュアル

このマニュアルでは、Element Plusを素早く理解し、日常的に使用するコンポーネントと機能をマスターするのに役立ちます。

## クイックスタート

Element Plusを使用する前に、インストールが完了していることを確認してください。

### 基本的な使用

Vueコンポーネントで直接コンポーネントを使用します：

```vue
<template>
  <el-button>デフォルトボタン</el-button>
  <el-button type="primary">プライマリボタン</el-button>
  <el-button type="success">成功ボタン</el-button>
  <el-button type="warning">警告ボタン</el-button>
  <el-button type="danger">危険ボタン</el-button>
</template>
```

## 一般的なコンポーネント

### ボタン（Button）

基本的な使用：

```vue
<el-button>デフォルト</el-button>
<el-button type="primary">プライマリ</el-button>
<el-button type="success">成功</el-button>
<el-button type="warning">警告</el-button>
<el-button type="danger">危険</el-button>
```

異なるサイズ：

```vue
<el-button size="large">大</el-button>
<el-button>デフォルト</el-button>
<el-button size="small">小</el-button>
```

無効状態：

```vue
<el-button disabled>無効</el-button>
```

アイコンボタン：

```vue
<el-button icon="Search">検索</el-button>
<el-button icon="Edit">編集</el-button>
<el-button icon="Delete">削除</el-button>
```

### インプット（Input）

基本的な使用：

```vue
<el-input v-model="input" placeholder="内容を入力してください" />
```

アイコン付き：

```vue
<el-input v-model="input" prefix-icon="Search" />
<el-input v-model="input" suffix-icon="Calendar" />
```

クリアボタン付き：

```vue
<el-input v-model="input" clearable />
```

パスワード入力：

```vue
<el-input v-model="input" type="password" show-password />
```

テキストエリア：

```vue
<el-input v-model="textarea" type="textarea" :rows="3" />
```

### セレクター（Select）

基本的なセレクター：

```vue
<el-select v-model="value" placeholder="選択してください">
  <el-option label="オプション1" value="1" />
  <el-option label="オプション2" value="2" />
</el-select>
```

複数選択：

```vue
<el-select v-model="value" multiple placeholder="選択してください">
  <el-option label="オプション1" value="1" />
  <el-option label="オプション2" value="2" />
</el-select>
```

### テーブル（Table）

基本的なテーブル：

```vue
<el-table :data="tableData" stripe border>
  <el-table-column prop="date" label="日付" width="180" />
  <el-table-column prop="name" label="名前" width="180" />
  <el-table-column prop="address" label="住所" />
</el-table>
```

インデックス付き：

```vue
<el-table-column type="index" label="番号" width="60" />
```

固定列：

```vue
<el-table-column prop="date" label="日付" width="150" fixed />
```

### フォーム（Form）

基本的なフォーム：

```vue
<el-form :model="form" label-width="80px">
  <el-form-item label="アクティビティ名">
    <el-input v-model="form.name" />
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="onSubmit">送信</el-button>
  </el-form-item>
</el-form>
```

フォーム検証：

```vue
<el-form :model="ruleForm" :rules="rules" ref="ruleFormRef">
  <el-form-item label="名前" prop="name">
    <el-input v-model="ruleForm.name" />
  </el-form-item>
</el-form>
```

### ダイアログ（Dialog）

```vue
<el-dialog v-model="visible" title="ヒント" width="30%">
  <span>内容</span>
</el-dialog>
```

### メッセージ

```javascript
import { ElMessage } from 'element-plus'

ElMessage.success('成功メッセージ')
ElMessage.warning('警告メッセージ')
ElMessage.error('エラーメッセージ')
```

### メニュー（Menu）

水平メニュー：

```vue

  <el-menu-item index="1">処理センター</el-menu-item>
  <el-menu-item index="2">注文管理</el-menu-item>
</el-menu>
```

垂直メニュー：

```vue
<el-menu default-active="1">
  <el-menu-item index="1">ナビゲーション1</el-menu-item>
  <el-menu-item index="2">ナビゲーション2</el-menu-item>
</el-menu>
```

### タブ（Tabs）

```vue
<el-tabs v-model="activeName">
  <el-tab-pane label="ユーザー管理" name="first">ユーザー管理</el-tab-pane>
  <el-tab-pane label="設定管理" name="second">設定管理</el-tab-pane>

```

### ページネーション（Pagination）

```vue
<el-pagination
  v-model:current-page="currentPage"
  :page-size="100"
  :total="1000"
  layout="total, prev, pager, next"
/>
```

## レイアウトシステム

### グリッドシステム

Element Plusは24列グリッドシステムを使用します：

```vue
<el-row :gutter="20">
  <el-col :span="12"><div>12列</div></el-col>
  <el-col :span="12"><div>12列</div></el-col>
</el-row>
```

### コンテナレイアウト

```vue
<el-container>
  <el-header>Header</el-header>
  <el-main>Main</el-main>
  <el-footer>Footer</el-footer>
</el-container>
```

## 一般的な問題

### コンポーネントスタイルが機能しない

CSSが正しくインポートされていることを確認してください：

```javascript
import 'element-plus/dist/index.css'
```

### アイコンが表示されない

アイコンラブラリをインストールする必要があります：

```bash
npm install @element-plus/icons-vue
```

次にグローバルに登録します：

```javascript
import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```

### オンデマンドインポートが機能しない

プラグイン設定が正しいかどうか確認し、正しいresolverを使用していることを確認してください。
