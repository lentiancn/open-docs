# Introduction

Angular est un framework open-source développé et maintenu par Google, conçu pour créer des applications web modernes. Il succède à AngularJS (Angular 1.x) et a été complètement réécrit, utilisant TypeScript comme langage de développement principal.

Angular offre une solution complète pour le développement front-end, incluant l'architecture par composants, la liaison de données bidirectionnelle, l'injection de dépendances, le routage, la gestion des formulaires, le client HTTP et bien d'autres fonctionnalités. Grâce à ces caractéristiques, les développeurs peuvent construire rapidement des applications monopages (SPA) performantes et maintenables.

## Concepts fondamentaux

### Le composant (Component)

Le composant est le bloc de construction de base d'une application Angular. Chaque composant se compose de trois parties :

- **Le modèle (Template)** : définit la structure de la vue
- **Les styles (Styles)** : définit l'apparence de la vue
- **La logique (Logic)** : définit le comportement du composant

### Le module (Module)

Les modules permettent d'organiser le code en regroupant les composants, services et directives apparentés. Chaque application Angular possède au moins un module racine (AppModule).

### Le service (Service)

Les services encapsulent la logique métier réutilisable et sont mis à disposition des composants via le mécanisme d'injection de dépendances.

### La directive (Directive)

Les directives étendent les fonctionnalités des éléments HTML. On distingue les directives structurelles (comme *ngIf, *ngFor) et les directives d'attribut (comme ngClass, ngStyle).

## Fonctionnalités principales

### Architecture par composants

Angular adopte une conception basée sur les composants. Chaque composant est une unité fonctionnelle indépendante, réutilisable et combinable.

### TypeScript

Angular utilise TypeScript comme langage principal, offrant le support du typage fort, la programmation orientée objet et les fonctionnalités modernes de JavaScript.

### Injection de dépendances

Angular intègre un système d'injection de dépendances qui rend le code plus modulaire, testable et maintenable.

### Liaison de données bidirectionnelle

Les données entre le modèle et le composant sont synchronisées automatiquement, ce qui réduit considérablement le code nécessaire pour manipuler le DOM.

### Routage

Un système de routage puissant supportant le chargement paresseux, les gardiens et la transmission de paramètres.

### Client HTTP

Un client HTTP intégré supportant les intercepteurs et les transformations de requêtes/réponses.

### Gestion des formulaires

Angular propose deux approches : les formulaires réactifs et les formulaires pilotés par le modèle, pour gérer des validations complexes.

### Programmation réactive

Support complet de RxJS, offrant de puissantes capacités de programmation asynchrone.

## Cas d'utilisation

- Applications web d'entreprise
- Applications monopages (SPA)
- Applications web progressives (PWA)
- Applications mobiles hybrides (avec Ionic)
- Applications de bureau (avec Electron)

## Historique des versions

- **AngularJS (1.x)** : publié en 2010
- **Angular 2** : publié en 2016, complètement réécrit
- **Angular 17** : dernière version stable publiée en 2023

Angular bénéficie d'une communauté active et d'une documentation complète, ce qui en fait l'un des choix les plus répandus pour créer des applications web modernes.
