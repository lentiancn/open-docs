# Vite

Vite 是新一代前端构建工具。

## 简介

Vite 是由 Vue.js 作者尤雨溪开发的新一代前端构建工具。它利用浏览器原生 ES 模块支持，提供极速的开发体验。

## 主要特性

- **极速开发**：基于 ESM 的开发服务器
- **热模块替换**：即时热更新
- **按需编译**：只编译需要的文件
- **丰富的插件**：支持 Vue、React、Svelte 等
- **生产优化**：内置代码分割和压缩

## 快速开始

### 安装

```bash
npm create vite@latest my-app -- --template vue
cd my-app
npm install
npm run dev
```
