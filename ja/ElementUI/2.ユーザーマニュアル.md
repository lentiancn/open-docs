## 1. 基本コンポーネント

### 1.1 ボタンコンポーネント

ボタンは最も使用頻度の高いコンポーネントの1つで、Element UIは多くの类型的ボタンを提供します。

#### 基本的な使い方

```html
<template>
  <el-button>デフォルト</el-button>
  <el-button type="primary">プライマリ</el-button>
  <el-button type="success">成功</el-button>
  <el-button type="info">情報</el-button>
  <el-button type="warning">警告</el-button>
  <el-button type="danger">危険</el-button>
</template>
```

#### ボタン属性

| 属性 | 説明 | タイプ | オプション値 | デフォルト |
|------|------|------|-----------|----------|
| type | ボタンの種類 | string | primary / success / warning / danger / info / text | — |
| plain | シンプルボタンか | boolean | — | false |
| round | 角丸ボタンか | boolean | — | false |
| circle | 円形ボタンか | boolean | — | false |
| loading | ローディング中か | boolean | — | false |
| disabled | 無効か | boolean | — | false |
| icon | アイコンクラス名 | string | Element UIアイコン名 | — |
| size | ボタンサイズ | string | medium / small / mini | — |

#### ボタングループ

```html
<el-button-group>
  <el-button type="primary" icon="el-icon-arrow-left">前へ</el-button>
  <el-button type="primary">次へ<i class="el-icon-arrow-right el-icon--right"></i></el-button>
</el-button-group>
```

## 2. フォームコンポーネント

### 2.1 入力フィールド

#### 基本的な入力

```html
<el-input v-model="input" placeholder="内容を入力"></el-input>
```

#### 入力フィールド属性

| 属性 | 説明 | タイプ | オプション値 | デフォルト |
|------|------|------|-----------|----------|
| type | 入力タイプ | string | text / textarea / password | text |
| v-model | バインディング値 | string / number | — | — |
| placeholder | プレースホルダー | string | — | — |
| disabled | 無効か | boolean | — | false |
| clearable | クリア可能か | boolean | — | false |
| show-password | パスワード表示切替 | boolean | — | false |
| prefix-icon | 接頭辞アイコン | string | — | — |
| suffix-icon | 接尾辞アイコン | string | — | — |

### 2.2 選択ボックス

#### 基本的な選択

```html
<el-select v-model="selected" placeholder="選択してください">
  <el-option
    v-for="item in options"
    :key="item.value"
    :label="item.label"
    :value="item.value">
  </el-option>
</el-select>
```

#### クリア可能な選択

```html
<el-select v-model="selected" clearable placeholder="選択してください">

    v-for="item in options"
    :key="item.value"
    :label="item.label"
    :value="item.value">
  </el-option>
</el-select>
```

```html
<el-select v-model="selectedMultiple" multiple placeholder="選択してください">
  <el-option
    v-for="item in options"
    :key="item.value"
    :label="item.label"
    :value="item.value">
  </el-option>
</el-select>
```

### 2.6 フォーム検証

Element UIは強力なフォーム検証機能を提供します。

#### 基本的な検証

```html
<el-form :model="form" :rules="rules" ref="ruleForm">
  <el-form-item label="ユーザー名" prop="username">
    <el-input v-model="form.username"></el-input>
  </el-form-item>
  <el-form-item label="パスワード" prop="password">
    <el-input v-model="form.password" type="password"></el-input>
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="submitForm('ruleForm')">送信</el-button>
    <el-button @click="resetForm('ruleForm')">リセット</el-button>
  </el-form-item>
</el-form>
```

```javascript
export default {
  data() {
    return {
      form: {
        username: '',
        password: ''
      },
      rules: {
        username: [
          { required: true, message: 'ユーザー名を入力してください', trigger: 'blur' }
        ],
        password: [
          { required: true, message: 'パスワードを入力してください', trigger: 'blur' }
        ]
      }
    }
  },
  methods: {
    submitForm(formName) {
      this.$refs[formName].validate(valid => {
        if (valid) {
          console.log('送信成功：', this.form)
        } else {
          this.$message.error('フォーム検証に失敗しました')
          return false
        }
      })
    },
    resetForm(formName) {
      this.$refs[formName].resetFields()
    }
  }
}
```

## 3. テーブルコンポーネント

### 3.1 基本的なテーブル

```html
<el-table :data="tableData" stripe border>
  <el-table-column prop="date" label="日付" width="180"></el-table-column>
  <el-table-column prop="name" label="名前" width="180"></el-table-column>
  <el-table-column prop="address" label="住所"></el-table-column>
</el-table>
```

### 3.2 操作付きテーブル

```html
<el-table :data="tableData">
  <el-table-column prop="name" label="名前"></el-table-column>
  <el-table-column label="操作" width="200">
    <template slot-scope="scope">
      <el-button size="mini" @click="handleEdit(scope.$index, scope.row)">編集</el-button>
      <el-button size="mini" type="danger" @click="handleDelete(scope.$index, scope.row)">削除</el-button>
    </template>
  </el-table-column>
</el-table>
```

### 3.3 ページネーション付きテーブル

```html
<el-table :data="tableData" v-loading="loading">
  <el-table-column prop="title" label="タイトル"></el-table-column>
  <el-table-column prop="author" label="著者"></el-table-column>
  <el-table-column prop="date" label="日付"></el-table-column>
</el-table>

<el-pagination
  @size-change="handleSizeChange"
  @current-change="handleCurrentChange"
  :current-page="currentPage"
  :page-sizes="[10, 20, 50, 100]"
  :page-size="pageSize"
  layout="total, sizes, prev, pager, next, jumper"
  :total="total">
```

## 4. ダイアログと通知

### 4.1 ダイアログ

```html
<el-button type="text" @click="dialogVisible = true">ダイアログを開く</el-button>

<el-dialog
  title="編集"
  :visible.sync="dialogVisible"
  width="30%"
  :before-close="handleClose">
  <span>これはダイアログの内容です</span>
  <span slot="footer">
    <el-button @click="dialogVisible = false">キャンセル</el-button>
    <el-button type="primary" @click="dialogVisible = false">OK</el-button>
  </span>
</el-dialog>
```

### 4.2 メッセージ通知

```javascript
// 成功メッセージ
this.$message.success('操作成功')

// 警告メッセージ
this.$message.warning('警告メッセージ')

// エラーメッセージ
this.$message.error('操作失敗')

// 情報メッセージ
this.$message.info('情報メッセージ')
```

### 4.3 通知

```javascript
// 成功通知
this.$notify({
  title: '成功',
  message: '操作が完了しました',
  type: 'success'
})

// エラー通知
this.$notify.error({
  title: 'エラー',
  message: '操作に失敗しました'
})
```
