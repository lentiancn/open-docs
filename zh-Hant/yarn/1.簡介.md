# Yarn 套件管理器

Yarn 是一個成熟的開源套件管理器，用於管理 JavaScript 專案中的相依性。它協助處理安裝、更新、配置和移除套件相依性的過程，最終幫助您更快地達成目標，減少干擾。

Yarn 的重點領域是速度、正確性、安全性和開發者體驗，通過利用一系列創新功能（如工作區、離線快取、平行安裝、強化模式、互動式命令等）在每個方面都得到改進。

鑑於其強大的功能和卓越的記錄，Yarn 證明了它是 JavaScript 生態系統中不可或缺的資產，在任何規模的專案中都能推動效率、安全性和積極的開發者體驗。

## 主要版本對比

### Yarn v4+ (現代版本)
- **架構**: 外掛式架構 (Berry)
- **特點**: 工作區原生支援、PnP (Plug'n'Play)、離線快取、平行安裝
- **安裝方式**: 推薦使用 Corepack
- **狀態**: 活躍開發中

### Yarn v1 (Classic)
- **架構**: 傳統架構
- **特點**: 簡單直接，廣泛相容
- **安裝方式**: 通過 npm 全域安裝
- **狀態**: 維護中

> 注意：與其他大多數套件管理器不同，Yarn 重新實作了所有命令，以完全控制開發者體驗和穩定性。
