# Element UI 개요

Vue.js 2.0 데스크톱 UI 컴포넌트 라이브러리.

## Element UI란?

Element UI는 Ele.me 팀이 개발한 오픈소스 컴포넌트 라이브러리입니다.

### 주요 기능

- **풍부한 컴포넌트**: 60+ 컴포넌트
- **아름다운 디자인**: Material Design
- **TypeScript 지원**

### 통계

- GitHub 별표: 54,000+
