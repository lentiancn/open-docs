# V-Express 설치 가이드

V-Express는 Tencent AI Lab이 개발한 초상 비디오 생성 방법입니다.

## 시스템 요구 사항

### 하드웨어

| 구성 요소 | 최소 | 권장 |
|----------|------|------|
| GPU | NVIDIA 16GB VRAM | NVIDIA 24GB+ VRAM |

## 설치

```bash
git clone https://github.com/tencent-ailab/V-Express.git
cd V-Express
pip install -r requirements.txt
```
