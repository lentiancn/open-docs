# OpenClaw 常見問題

> 有關 OpenClaw 的常見問題

---

## 一般

### 什麼是 OpenClaw？

OpenClaw 是一個 AI 助理框架，利用先進的代理程式技術自動執行任務。它結合了各種工具，包括檔案管理、瀏覽器控制等。

### OpenClaw 適合誰使用？

OpenClaw 適合開發者、進階使用者以及希望透過 AI 驅動的自動化提高生產力的任何人。

## 安裝

### 系統需求是什麼？

| 需求 | 最低 |
|------|------|
| 作業系統 | Windows 10+、macOS、Linux |
| Node.js | 版本 18+ |
| RAM | 4 GB |
| 儲存空間 | 500 MB |

### 安裝失敗？

**解決方案**：
1. 檢查 Node.js 版本：`node --version`
2. 清除 npm 快取：`npm cache clean --force`
3. 嘗試使用管理員/根權限
4. 記錄錯誤訊息

### Gateway 無法啟動？

**解決方案**：
1. 檢查連接埠：`netstat -ano | findstr 3000`
2. 使用不同連接埠：`openclaw gateway start --port 3001`
3. 查看日誌：`openclaw gateway logs`

## 使用方法

### 如何開始對話？

```bash
openclaw chat
```

### 如何執行單一命令？

```bash
openclaw exec "我的命令"
```

### 可以使用無周邊模式執行 OpenClaw 嗎？

可以，OpenClaw 可以無周邊模式運作。啟動 Gateway 服務並使用 API 或 CLI。

## 檔案管理

### 如何讀取檔案？

使用讀取工具：
```
read /path/to/file.txt
```

### 如何建立新檔案？

使用寫入工具：
```
write --content "內容" --path /新檔案/路徑.txt
```

### 如何編輯檔案？

使用編輯工具進行精確更改：
```
edit --path /檔案.txt --oldText "舊" --newText "新"
```

## 瀏覽器控制

### 支援哪些瀏覽器？

- Chrome（透過擴充功能）
- OpenClaw 隔離瀏覽器
- 其他透過瀏覽器自動化

### 如何控制瀏覽器？

```bash
browser start --profile openclaw
browser navigate --url https://example.com
browser screenshot --path /output.png
```

### 瀏覽器無法啟動？

**解決方案**：
1. 安裝 Chrome 擴充功能
2. 檢查防火牆設定
3. 清除瀏覽器快取

## 子代理程式

### 什麼是子代理程式？

子代理程式是獨立的實例，可以平行或順序執行特定任務。

### 如何建立子代理程式？

```bash
subagents create --task "描述"
```

### 如何終止子代理程式？

```bash
subagents kill --id agent-id
```

## 安全性

### OpenClaw 安全嗎？

OpenClaw 實作多層安全性：
- 檔案權限
- 外部操作確認
- 沙盒選項

### 如何保護敏感資料？

1. 使用加密配置
2. 不要在對話中分享密碼
3. 使用環境變數儲存密鑰

## 故障排除

### 「找不到命令」？

確認：
1. OpenClaw 已正確安裝
2. PATH 變數設定正確
3. Gateway 正在執行

### 檔案操作失敗？

1. 檢查路徑
2. 確認權限
3. 檢查檔案名稱是否有特殊字元

### 回應時間慢？

1. 檢查 Gateway 狀態
2. 確認網路連線
3. 分析日誌
