# Guía de instalación de Nginx

## Requisitos del entorno

Nginx es compatible con múltiples sistemas operativos:

- Linux (Debian, Ubuntu, CentOS, RHEL, Alpine, etc.)
- FreeBSD
- Solaris
- AIX
- HP-UX
- macOS
- Windows

## Métodos de instalación

### Método 1: Usando el gestor de paquetes (Recomendado)

#### Ubuntu/Debian

```bash
sudo apt update
sudo apt install nginx
sudo systemctl start nginx
sudo systemctl enable nginx
```

#### CentOS/RHEL

```bash
sudo yum install epel-release
sudo yum install nginx
sudo systemctl start nginx
sudo systemctl enable nginx
```

#### Alpine

```bash
apk add nginx
```

#### macOS

```bash
brew install nginx
brew services start nginx
```

### Método 2: Instalar desde código fuente

```bash
wget https://nginx.org/download/nginx-1.26.0.tar.gz
tar -zxf nginx-1.26.0.tar.gz
cd nginx-1.26.0
./configure
make
sudo make install
```

### Método 3: Usando Docker

```bash
docker pull nginx
docker run -d -p 80:80 nginx
```

### Método 4: Windows

Descargar desde https://nginx.org/en/download.html y ejecutar nginx.exe

## Verificación

```bash
nginx -v
curl http://localhost
```

## Rutas predeterminadas

- Configuración: `/etc/nginx/nginx.conf`
- Logs: `/var/log/nginx/`
- Documentos: `/var/www/html/`
