# HeyGen 安装指南

HeyGen 是一个 AI 视频生成平台，允许用户创建逼真的 AI 头像并从文本或音频输入生成说话头像视频。

## 平台概述

HeyGen 主要是一个基于云的 SaaS 平台。与开源项目不同，HeyGen 不需要本地安装 - 用户通过网络界面或 API 访问。

### 访问方式

| 方式 | 描述 | 要求 |
|------|------|------|
| 网页界面 | 基于浏览器的视频创作 | 账户 + 网络 |
| API | 编程访问 | API 密钥 |
| SDK | 本地开发 | 开发者账户 |

## 版本概述

HeyGen 是云服务，非本地安装：

| 组件 | 版本 | 说明 |
|------|------|------|
| HeyGen 网页 | 最新 | 始终保持最新 |
| HeyGen API | v1/v2 | REST API |
| HeyGen SDK | 最新 | 开发者使用 |

**注意：** HeyGen 是商业产品，某些功能需要付费订阅。

## 系统要求

### 网页访问要求

| 要求 | 最低 | 推荐 |
|------|------|------|
| 浏览器 | Chrome 90+, Firefox 88+, Safari 14+, Edge 90+ | 最新版 Chrome |
| 网络 | 10 Mbps | 25 Mbps+ |
| 内存 | 4GB | 8GB+ |

### API/SDK 开发要求

| 要求 | 版本 |
|------|------|
| Node.js | 16+ |
| Python | 3.8+ |
| API 密钥 | 来自 HeyGen 仪表板 |

## 访问 HeyGen

### 步骤 1：创建账户

1. 访问 https://www.heygen.com
2. 点击"注册"或"开始使用"
3. 验证邮箱地址

### 步骤 2：选择套餐

| 套餐 | 功能 | 价格 |
|------|------|------|
| 免费 | 有限分钟数，有水印 | 免费 |
| 专业版 | 更多分钟数，无水印 | $29/月 |
| 企业版 | 自定义限额，API 访问 | 联系销售 |

### 步骤 3：获取 API 密钥（开发者）

1. 登录 HeyGen 仪表板
2. 导航到设置 → API
3. 生成 API 密钥

## API 安装（开发者）

### Python SDK

```bash
# 安装 HeyGen SDK
pip install heygen-python

# 或从源码安装
pip install git+https://github.com/heygen-ai/heygen-python.git
```

### Node.js SDK

```bash
# 安装 HeyGen SDK
npm install heygen-node

# 或从源码安装
npm install github:heygen-ai/heygen-node
```

### 验证安装

```python
# Python
import heygen
print(heygen.__version__)
```

```javascript
// Node.js
const heygen = require('heygen-node');
console.log(heygen.version);
```

## API 配置

### Python 示例

```python
from heygen import HeyGen

# 使用 API 密钥初始化
heygen = HeyGen(api_key="your_api_key")

# 检查账户信息
account = heygen.get_account_info()
print(account)
```

### Node.js 示例

```javascript
const { HeyGen } = require('heygen-node');

const heygen = new HeyGen({ apiKey: 'your_api_key' });

// 检查账户信息
const account = await heygen.getAccountInfo();
console.log(account);
```

## 环境变量

### 设置 API 密钥

```bash
# Linux/macOS
export HEYGEN_API_KEY="your_api_key"

# Windows (PowerShell)
$env:HEYGEN_API_KEY="your_api_key"

# 在 .env 文件中
HEYGEN_API_KEY=your_api_key
```

## 常见问题

### 认证错误

**问题：** "API 密钥无效"

**解决方案：**
1. 检查 API 密钥是否正确
2. 确保 API 密钥处于活动状态
3. 验证账户是否有 API 访问权限

### 速率限制

**问题：** "请求过多"

**解决方案：**
1. 检查 API 使用限制
2. 升级套餐以获得更高限制
3. 实施请求队列

### 视频生成失败

**问题：** "生成失败"

**解决方案：**
1. 检查输入参数
2. 验证积分余额
3. 稍后重试

## 安全最佳实践

1. **永不暴露 API 密钥** - 使用环境变量
2. **定期轮换密钥** - 定期生成新密钥
3. **使用最小权限** - 只请求需要的范围
4. **监控使用情况** - 检查仪表板异常
