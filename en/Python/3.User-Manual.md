# Python User Manual

> Python core syntax and features

---

## Basic Syntax

### Hello World

```python
print("Hello, World!")
```

### Variables

```python
name = "Tom"
age = 25
salary = 5000.50
is_active = True

# List
fruits = ["apple", "banana"]

# Dict
person = {"name": "Tom", "age": 25}

# Set
numbers = {1, 2, 3}
```

---

## Control Flow

### Conditionals

```python
if age >= 18:
    print("Adult")
elif age >= 6:
    print("Minor")
else:
    print("Child")
```

### Loops

```python
# for
for i in range(10):
    print(i)

# while
count = 0
while count < 5:
    print(count)
    count += 1
```

---

## Functions

```python
def greet(name):
    return f"Hello, {name}!"

# Lambda
square = lambda x: x ** 2
```

---

## OOP

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        return f"Hello, I'm {self.name}"

person = Person("Tom", 25)
print(person.greet())
```

## Exception Handling

```python
try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero")
finally:
    print("Always executes")
```

---

## File Operations

```python
# Read
with open("file.txt", "r") as f:
    content = f.read()

# Write
with open("file.txt", "w") as f:
    f.write("Hello!")
```
