## 1. Basis-Komponenten

### 1.1 Button-Komponente

Buttons sind eine der am häufigsten verwendeten Komponenten.

#### Basis-Verwendung

```html
<el-button>Standard</el-button>
<el-button type="primary">Primär</el-button>
<el-button type="success">Erfolg</el-button>
<el-button type="warning">Warnung</el-button>
<el-button type="danger">Gefahr</el-button>
```

#### Button-Attribute

| Attribut | Beschreibung | Typ | Standard |
|----------|-------------|------|----------|
| type | Button-Typ | string | — |
| plain | Einfacher Stil | boolean | false |
| round | Abgerundet | boolean | false |
| disabled | Deaktiviert | boolean | false |

## 2. Formular-Komponenten

### 2.1 Eingabefeld

```html
<el-input v-model="input" placeholder="Eingabe"></el-input>
```

### 2.2 Auswahl

```html
<el-select v-model="selected" placeholder="Auswählen">
  <el-option label="Option 1" value="1"></el-option>
  <el-option label="Option 2" value="2"></el-option>
</el-select>
```

### 2.3 Formular-Validierung

```javascript
this.$refs.form.validate(valid => {
  if (valid) {
    console.log('Validierung erfolgreich')
  }
})
```

## 3. Tabelle-Komponente

### 3.1 Basis-Tabelle

```html
<el-table :data="tableData" stripe border>
  <el-table-column prop="name" label="Name"></el-table-column>
  <el-table-column prop="email" label="E-Mail"></el-table-column>
</el-table>
```

### 3.2 Tabelle mit Paginierung

```html
<el-pagination
  @current-change="handlePageChange"
  :current-page="currentPage"
  :total="total">
</el-pagination>
```

## 4. Dialoge und Benachrichtigungen

### 4.1 Dialog

```html
<el-dialog title="Titel" :visible.sync="visible">
  <span>Inhalt</span>
</el-dialog>
```

### 4.2 Nachrichten

```javascript
this.$message.success('Erfolgreich')
this.$message.error('Fehlgeschlagen')
this.$message.warning('Warnung')
```
