# Angular

> 현대적인 웹 애플리케이션 개발 프레임워크

---

## 소개

Angular는 Google의 SPA용 오픈소스 웹 프레임워크입니다.

### 기능
- 컴포넌트
- 양방향 바인딩
- 의존성 주입
- 라우팅
- 폼
- HTTP 클라이언트

## 빠른 시작

```bash
npm install -g @angular/cli
ng new my-app
cd my-app
ng serve
```

사이트: http://localhost:4200
