# JUnit 使用指南

## 基本测试

```java
import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;

class MyTest {
    
    @Test
    void simpleTest() {
        assertEquals(2, 1 + 1);
    }
}
```

## 常用断言

```java
// 相等
assertEquals(expected, actual);

// 布尔
assertTrue(condition);
assertFalse(condition);

// 空
assertNull(object);
assertNotNull(object);

// 异常
assertThrows(Exception.class, () -> {
    // code
});
```

## 生命周期

```java
@BeforeAll
static void beforeAll() {}

@AfterAll
static void afterAll() {}

@BeforeEach
void beforeEach() {}

@AfterEach
void afterEach() {}
```

## 参数化测试

```java
@ParameterizedTest
@ValueSource(ints = {1, 2, 3})
void isEven(int number) {
    assertTrue(number % 2 == 0);
}
```

## Spring Boot 测试

```java
@SpringBootTest
class MyServiceTest {
    
    @Autowired
    private MyService myService;
    
    @Test
    void testService() {
        assertNotNull(myService.doSomething());
    }
}
```

## 运行测试

```bash
mvn test
./gradlew test
```

## 最佳实践

1. 清晰命名测试方法
2. 每个测试一个断言
3. 使用 @BeforeEach 做通用设置
4. 使用 Mockito 模拟外部依赖
