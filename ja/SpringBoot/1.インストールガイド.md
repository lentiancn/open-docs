# Spring Boot インストールガイド

このガイドでは、Java、Maven、Gradle を含む Spring Boot 開発環境のセットアップ説明します。

## 前提条件

### Java のインストール

Spring Boot 3.x は Java 17 以降が必要です。Spring Boot 2.x は Java 8 以降が必要です。

**Windows:**
1. [Oracle](https://www.oracle.com/java/technologies/downloads/) または [Adoptium](https://adoptium.net/) から JDK をダウンロード
2. インストーラーを実行
3. 環境変数を設定:
   ```
   JAVA_HOME=C:\Program Files\Java\jdk-17
   PATH=%JAVA_HOME%\bin;%PATH%
   ```

**macOS:**
```bash
# Homebrew を使用
brew install openjdk@17

# PATH に追加
echo 'export JAVA_HOME=$(/usr/libexec/java_home)' >> ~/.zshrc
source ~/.zshrc
```

**Linux:**
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install openjdk-17-jdk

# JAVA_HOME を設定
echo 'export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64' >> ~/.bashrc
source ~/.bashrc
```

インストール確認:
```bash
java -version
```

### Maven のインストール

**Windows:**
1. [Apache Maven](https://maven.apache.org/download.cgi) からダウンロード
2. フォルダに展開（例: C:\apache-maven）
3. 環境変数を設定:
   ```
   MAVEN_HOME=C:\apache-maven
   PATH=%MAVEN_HOME%\bin;%PATH%
   ```

**macOS:**
```bash
brew install maven
```

**Linux:**
```bash
sudo apt install maven
```

インストール確認:
```bash
mvn -version
```

### Gradle のインストール

**Windows:**
1. [Gradle](https://gradle.org/install/) からダウンロード
2. 展開して PATH に追加

**macOS:**
```bash
brew install gradle
```

**Linux:**
```bash
sudo apt install gradle
```

インストール確認:
```bash
gradle -version
```

## IDE のセットアップ

### IntelliJ IDEA（推奨）
- [JetBrains](https://www.jetbrains.com/idea/download/) からダウンロード
- プロジェクト構造で JDK が設定されていることを確認
- Spring Boot Assistant プラグインをインストール

### VS Code
- Java 拡張パックをインストール
- Spring Boot 拡張機能をインストール

## Spring Initializr でクイックスタート

1. [start.spring.io](https://start.spring.io/) にアクセス
2. 選択:
   - プロジェクト: Maven または Gradle
   - 言語: Java
   - Spring Boot バージョン
   - プロジェクトメタデータ
3. 依存関係を追加（Spring Web、Spring Data JPA など）
4. 「Generate」をクリックしてプロジェクトをダウンロード

## セットアップの確認

```bash
# Java を確認
java -version

# Maven を確認
mvn -version

# Gradle を確認（Gradle を使用する場合）
gradle -version

# シンプルな Spring Boot プロジェクトを作成して実行
curl https://start.spring.io/starter.zip -o demo.zip
unzip demo.zip -d demo
cd demo
./mvnw spring-boot:run  # または ./gradlew bootRun
```
