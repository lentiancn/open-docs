# Manual del Usuario de Element UI

Este manual le ayudará a comenzar rápidamente con Element UI y dominar los componentes y funciones más utilizados.

## Inicio Rápido

Antes de usar Element UI, asegúrese de que la instalación esté completa.

### Uso Básico

Use los componentes directamente en componentes Vue:

<template>
  <el-button>Botón Predeterminado</el-button>
  <el-button type="primary">Botón Principal</el-button>
  <el-button type="success">Botón de Éxito</el-button>
  <el-button type="warning">Botón de Advertencia</el-button>
  <el-button type="danger">Botón de Peligro</el-button>
</template>

## Componentes Comunes

### Botón

Uso básico:

<el-button>Predeterminado</el-button>
<el-button type="primary">Principal</el-button>
<el-button type="success">Éxito</el-button>
<el-button type="warning">Advertencia</el-button>
<el-button type="danger">Peligro</el-button>

Diferentes tamaños:

<el-button size="medium">Mediano</el-button>
<el-button>Predeterminado</el-button>
<el-button size="small">Pequeño</el-button>
<el-button size="mini">Mini</el-button>

Estado deshabilitado:

<el-button disabled>Deshabilitado</el-button>

Botones con icono:

<el-button icon="el-icon-search">Buscar</el-button>
<el-button icon="el-icon-edit">Editar</el-button>
<el-button icon="el-icon-delete">Eliminar</el-button>

### Entrada

Uso básico:

<el-input v-model="input" placeholder="Por favor ingrese contenido" />

Con icono:

<el-input v-model="input" prefix-icon="el-icon-search" />
<el-input v-model="input" suffix-icon="el-icon-date" />

Con botón de limpiar:

<el-input v-model="input" clearable />

Entrada de contraseña:

<el-input v-model="input" type="password" show-password />

### Selector

Selector básico:

<el-select v-model="value" placeholder="Por favor seleccione">
  <el-option label="Opción 1" value="1" />
  <el-option label="Opción 2" value="2" />
</el-select>

Selección múltiple:

<el-select v-model="value" multiple placeholder="Por favor seleccione">
  <el-option label="Opción 1" value="1" />
  <el-option label="Opción 2" value="2" />
</el-select>

### Tabla

Tabla básica:

<el-table :data="tableData" stripe border>
  <el-table-column prop="date" label="Fecha" width="180" />
  <el-table-column prop="name" label="Nombre" width="180" />
  <el-table-column prop="address" label="Dirección" />
</el-table>

### Formulario

Formulario básico:

<el-form :model="form" label-width="80px">
  <el-form-item label="Nombre de Actividad">
    <el-input v-model="form.name" />
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="onSubmit">Enviar</el-button>
  </el-form-item>
</el-form>

### Diálogo

<el-dialog :visible.sync="visible" title="Título" width="30%">
  <span>Contenido</span>
</el-dialog>

### Mensaje

import { Message } from 'element-ui'

Message.success('Mensaje de éxito')
Message.warning('Mensaje de advertencia')
Message.error('Mensaje de error')

### Menú

Menú horizontal:

<el-menu mode="horizontal">
  <el-menu-item index="1">Centro de Procesamiento</el-menu-item>
  <el-menu-item index="2">Gestión de Pedidos</el-menu-item>
</el-menu>

### Pestañas

<el-tabs v-model="activeName">
  <el-tab-pane label="Gestión de Usuarios" name="first">Gestión de Usuarios</el-tab-pane>
  <el-tab-pane label="Gestión de Configuración" name="second">Gestión de Configuración</el-tab-pane>
</el-tabs>

### Paginación

<el-pagination
  @current-change="handleCurrentChange"
  :current-page="currentPage"
  :page-size="100"
  :total="1000"
  layout="total, prev, pager, next"
/>

## Sistema de Diseño

### Sistema de Cuadrícula

Element UI usa un sistema de cuadrícula de 24 columnas:

<el-row :gutter="20">
  <el-col :span="12"><div>12 columnas</div></el-col>
  <el-col :span="12"><div>12 columnas</div></el-col>
</el-row>

### Diseño de Contenedor

<el-container>
  <el-header>Encabezado</el-header>
  <el-main>Principal</el-main>
  <el-footer>Pie de página</el-footer>
</el-container>

## Mejores Prácticas

### Diseño Responsivo

  <el-col :xs="24" :sm="12" :md="8" :lg="6">Columna responsiva</el-col>
</el-row>

### Validación de Formularios

Siempre valide la entrada del usuario y proporcione mensajes de error claros.

## Problemas Comunes

### Estilos de Componentes No se Aplican

Asegúrese de importar el CSS:

import 'element-ui/lib/theme-chalk/index.css'

### Iconos No se Muestran

Necesita importar las fuentes de iconos:

import 'element-ui/lib/theme-chalk/fonts/element-icons.ttf'
