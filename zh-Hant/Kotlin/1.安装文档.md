# Kotlin 安裝指南

Kotlin 是 JetBrains 開發的現代程式語言。本指南涵蓋在各種環境中的安裝方法。

## 版本資訊

| 版本 | 狀態 | 說明 |
|------|------|------|
| Kotlin 2.x | ✅ 目前 | 最新版本 |
| Kotlin 1.9.x | ✅ 支援 | 穩定版本 |

## 系統需求

### 硬體需求

| 需求 | 配置 |
|------|------|
| CPU | 1+ 核心 |
| 記憶體 | 2+ GB |
| 磁碟 | 500+ MB |

### 軟體需求

| 軟體 | 版本 |
|------|------|
| Java | JDK 8 或更高 |

## 安裝方法

### 方法一：IntelliJ IDEA (推薦)

1. 下載 IntelliJ IDEA：https://www.jetbrains.com/idea/download/
2. 安裝 Ultimate 或 Community 版本
3. Kotlin 已內建支援

### 方法二：命令列

#### Windows

```powershell
# 使用 Chocolatey
choco install kotlin

# 或手動下載
# 從 https://github.com/JetBrains/kotlin/releases 下載
```

#### macOS

```bash
# 使用 Homebrew
brew install kotlin

# 或手動安裝
curl -s https://get.sdkman.io | bash
sdk install kotlin
```

#### Linux

```bash
# Ubuntu/Debian
sudo apt install kotlin

# 或使用 SDKMAN
curl -s https://get.sdkman.io | bash
sdk install kotlin
```

### 方法三：SDKMAN

```bash
# 安裝 SDKMAN
curl -s "https://get.sdkman.io" | bash

# 安裝 Kotlin
sdk install kotlin

# 驗證
kotlin -version
```

### 方法四：Maven

```xml
<dependencies>
    <dependency>
        <groupId>org.jetbrains.kotlin</groupId>
        <artifactId>kotlin-stdlib</artifactId>
        <version>2.0.0</version>
    </dependency>
</dependencies>
```

### 方法五：Gradle

```groovy
plugins {
    id 'org.jetbrains.kotlin.jvm' version '2.0.0'
}

repositories {
    mavenCentral()
}

dependencies {
    implementation "org.jetbrains.kotlin:kotlin-stdlib"
}
```

### 方法六：Docker

```dockerfile
# 使用 Kotlin 編譯器映像檔
FROM gradle:8.5-jdk-21 AS builder
WORKDIR /app
COPY . .
RUN gradle build

# 執行映像檔
FROM openjdk:21
COPY --from=builder /app/build/libs/*.jar /app.jar
ENTRYPOINT ["java",app.jar"]
```

 "-jar", "/## 環境配置

### Java 配置

```bash
# 檢查 Java
java -version

# 設定 JAVA_HOME
export JAVA_HOME=/path/to/java
```

### 編譯執行

```bash
# 編譯
kotlinc hello.kt -include-runtime -d hello.jar

# 執行
java -jar hello.jar

# 使用 kotlin 腳本
kotlin hello.kts
```

## 常見問題

### 編譯錯誤？

檢查 Kotlin 和 Java 版本相容性。

### 記憶體不足？

增加 JVM 堆記憶體：

```bash
kotlinc -J-Xmx2g
```
