# FantasyTalking Preguntas Frecuentes

## Problemas de Instalación

### ¿Qué hacer si la instalación falla?

Durante la instalación pueden ocurrir varios problemas. Primero, confirma que el entorno cumple los requisitos. Verifica la versión de Node.js con `node --version` (debe ser >= 14.0). Si usas Python, asegúrate de que sea >= 3.7.

Si la instalación sigue fallando, intenta: Limpiar caché npm con `npm cache clean --force`; Usar espejo nacional; O instalar desde código fuente.

### Problemas de FFmpeg

En algunos sistemas, FFmpeg puede no estar correctamente instalado. Puedes descargar binarios precompilados desde el sitio oficial de FFmpeg.

## Problemas de Uso

### ¿Qué hacer si la síntesis de voz es lenta?

La velocidad de síntesis depende de múltiples factores. Primero verifica la conexión de red. Considera usar autoalojamiento para reducir latencia. Para grandes tareas, usa la interfaz de síntesis por lotes.

### ¿Cómo elegir el estilo de voz adecuado?

FantasyTalking ofrece varios estilos: neutral, happy, sad, angry, excited. Elige según el escenario: Usa neutral o professional para servicio al cliente.

### ¿Cómo implementar síntesis en inglés?

Especifica inglés en la configuración:

```javascript
const audio = await client.speak('Hello, world!', {
  voice: { language: 'en-US', style: 'neutral' }
});
```

### ¿SSML no funciona?

Asegúrate de pasar el parámetro `ssml: true` al llamar a speak. Verifica que el markup SSML sea correcto.

## Funciones de Diálogo

### ¿El contexto del diálogo no es coherente?

Asegúrate de usar la misma instancia de sesión. Cada llamada a `session.chat()` incluye automáticamente el historial previo.

### ¿Cómo guardar y restaurar conversaciones?

Usa `session.getHistory()` para obtener el historial, y `client.loadSession(history)` para restaurarlo.

## Problemas de Audio

### ¿Ruido en el audio generado?

Los problemas de ruido generalmente están relacionados con configuración de audio o transmisión de red. Verifica el formato y tasa de muestreo.

### ¿Cómo ajustar velocidad y tono?

```javascript
const audio = await client.speak('Texto', {
  voice: { speed: 1.5, pitch: 0.8 }
});
```

## Cuenta y Facturación

### ¿Cómo obtener clave API?

Regístrate en el sitio web oficial de FantasyTalking, luego crea una clave API en la consola.

### ¿Cómo ver uso?

Después de iniciar sesión en la consola, puedes ver llamadas API y uso en el panel.

## Problemas de Despliegue

### ¿Despliegue Docker falla?

Primero confirma que Docker está correctamente instalado y ejecutándose. Verifica si el puerto 8080 está en uso.

### ¿Cómo lograr despliegue de alta disponibilidad?

Para producción, usa balanceador de carga con múltiples instancias. Usa almacenamiento externo (como Redis) para compartir datos entre instancias.

## Desarrollo y Depuración

### ¿Cómo habilitar modo debug?

```javascript
const client = new FantasyTalking({
  apiKey: 'your-api-key',
  debug: true
});
```

### ¿Cómo obtener información detallada de errores?

```javascript
try {
  await client.speak('text');
} catch (error) {
  console.log('Código de error:', error.code);
  console.log('Mensaje:', error.message);
  console.log('ID de solicitud:', error.requestId);
}
```
