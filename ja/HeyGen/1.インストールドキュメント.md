# HeyGen インストールガイド

HeyGenは、ユーザーがリアルなAIアバターを作成し、テキストやオーディオ入力から話し手的ビデオを生成できるAIビデオ生成プラットフォームです。

## プラットフォームの概要

HeyGenは、主にクラウドベースのSaaSプラットフォームです。オープンソースプロジェクトとは異なり、HeyGenはローカルインストールは不要です - ユーザーはWebインターフェースまたはAPIを通じてアクセスします。

### アクセス方法

| 方法 | 説明 | 必要条件 |
|------|------|----------|
| Webインターフェース | ブラウザベースのビデオ制作 | アカウント + インターネット |
| API | プログラムによるアクセス | APIキー |
| SDK | ローカル開発 | デベロッパーアカウント |

## バージョンの概要

HeyGenはクラウドサービスであり、ローカルインストールではありません：

| コンポーネント | バージョン | メモ |
|---------------|-----------|------|
| HeyGen Web | 最新版 | 常に最新 |
| HeyGen API | v1/v2 | REST API |
| HeyGen SDK | 最新版 | デベロッパー用 |

**注意：** HeyGenは商用製品です。一部の機能には有料サブスクリプションが必要です。

## システム要件

| 要件 | 最小 | 推奨 |

## HeyGenへのアクセス

### ステップ1：アカウント作成

1. https://www.heygen.com にアクセス
2. 「新規登録」または「始める」をクリック
3. メールアドレスを認証

### ステップ2：プラン選択

| プラン | 機能 | 価格 |
|--------|------|------|
| 無料 | 制限時間、水印あり | 無料 |
| プロ | より多くの時間、水印なし | $29/月 |
| エンタープライズ | カスタム制限、APIアクセス | 営業お問い合わせ |

### ステップ3：APIキーの取得（デベロッパー向け）

1. HeyGenダッシュボードにログイン
2. 設定 → APIに移動
3. APIキーを生成

## APIインストール（デベロッパー向け）

### Python SDK

```bash
# HeyGen SDKをインストール
pip install heygen-python

# またはソースからインストール
pip install git+https://github.com/heygen-ai/heygen-python.git
```

### Node.js SDK

```bash
# HeyGen SDKをインストール
npm install heygen-node

# またはソースからインストール
npm install github:heygen-ai/heygen-node
```

### インストール確認

```python
# Python
import heygen
print(heygen.__version__)
```

```javascript
// Node.js
const heygen = require('heygen-node');
console.log(heygen.version);
```

## API設定

### Pythonの例

```python
from heygen import HeyGen

# APIキーで初期化
heygen = HeyGen(api_key="your_api_key")

# アカウント情報を確認
account = heygen.get_account_info()
print(account)
```

### Node.jsの例

```javascript
const { HeyGen } = require('heygen-node');

const heygen = new HeyGen({ apiKey: 'your_api_key' });

// アカウント情報を確認
const account = await heygen.getAccountInfo();
console.log(account);
```

## 環境変数

### APIキーの設定

```bash
# Linux/macOS
export HEYGEN_API_KEY="your_api_key"

# Windows (PowerShell)
$env:HEYGEN_API_KEY="your_api_key"

# .envファイルの場合
HEYGEN_API_KEY=your_api_key
```

### dotenvを使用したPython

```python
from dotenv import load_dotenv
load_dotenv()

import heygen
heygen = HeyGen(api_key=os.getenv("HEYGEN_API_KEY"))
```

## 一般的な問題

### 認証エラー

**問題：** 「APIキーが無効です」

**解決策：**
1. APIキーが正しいか確認
2. APIキーが有効であることを確認
3. アカウントにAPIアクセス権があるか確認

### レート制限

**問題：** 「リクエストが多すぎます」

**解決策：**
1. API使用制限を確認
2. より高い制限のためにプランをアップグレード
3. リクエストキューを実装

### ビデオ生成の失敗

**問題：** 「生成に失敗しました」

**解決策：**
1. 入力パラメータを確認
2. クレジット残高を検証
3. 後でもう一度試してください

## セキュリティのベストプラクティス

1. **APIキーを絶対に公開しない** - 環境変数を使用
2. **定期的にキーをローテーション** - 定期的に新しいキーを生成
3. **最小権限を使用** - 必要なスコープのみをリクエスト
4. **使用状況を監視** - ダッシュボードで異常を確認
