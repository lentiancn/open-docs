# Element UI

Vue.js 2.0 桌面端元件庫。

## 簡介

Element UI 是由餓了麼團隊開源的 Vue.js 2.0 元件庫。

### 主要特性

- **豐富元件**：60+ 元件
- **精美設計**
- **TypeScript 支持**
- **按需引入**

### 統計

- GitHub 星標：54,000+

## 快速開始

```bash
npm install element-ui --save
```

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```
