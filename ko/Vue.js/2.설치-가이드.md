## Vue 애플리케이션 생성 (권장)

### 전제 조건
- 명령줄에 익숙해야 함
- Node.js 버전 ^20.19.0 || >=22.12.0 설치 필요

### 공식 스캐폴딩 도구 사용
터미널에서 다음 명령어 중 하나를 실행하세요:

```bash
# npm
npm create vue@latest

# pnpm  
pnpm create vue@latest

# yarn (v1+)
yarn create vue

# yarn (Modern v2+)
yarn create vue@latest

# yarn (^v4.11)
yarn dlx create-vue@latest

# bun
bun create vue@latest
```

### 프로젝트 구성 옵션
다음과 같은 선택적 기능에 대한 프롬프트가 표시됩니다:
- Project name: 프로젝트 이름
- Add TypeScript?: TypeScript 지원 추가 여부
- Add JSX Support?: JSX 지원 추가 여부
- Add Vue Router: 싱글 페이지 애플리케이션 개발용
- Add Pinia: 상태 관리용
- Add Vitest: 유닛 테스트용
- Add End-to-End Testing Solution?: Cypress/Nightwatch/Playwright
- Add ESLint: 코드 품질용
- Add Prettier: 코드 포맷팅용
- Add Vue DevTools 7 extension: 디버깅용 (실험적)

### 개발 서버 시작
```bash
cd <your-project-name>
npm install
npm run dev
```

### 프로덕션 빌드
```bash
npm run build
```

## CDN을 통해 Vue 사용

### 글로벌 빌드
```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<div id="app">{{ message }}</div>

<script>
const { createApp } = Vue

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

### ES 모듈 빌드
```html
<div id="app">{{ message }}</div>

<script type="module">
import { createApp } from 'https://unpkg.com/vue@3/dist/vue.esm-browser.js'

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

### 임포트 맵 활성화
```html
<script type="importmap">
{
  "imports": {
    "vue": "https://unpkg.com/vue@3/dist/vue.esm-browser.js"
  }
}
</script>

<div id="app">{{ message }}</div>

<script type="module">
import { createApp } from 'vue'

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

## 개발 환경 권장사항

- **IDE**: Visual Studio Code + [Vue - Official 확장](https://marketplace.visualstudio.com/items?itemName=Vue.volar)
- **빌드 도구**: Vite (공식 스캐폴딩에 포함됨)
- **로컬 서버**: ES 모듈을 사용하려면 `npx serve`로 로컬 HTTP 서버를 시작하세요
- **간단한 시나리오 대안**: 경량 시나리오에는 [petite-vue](https://github.com/vuejs/petite-vue) 고려
