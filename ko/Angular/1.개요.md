# Angular 개요

현대 웹 개발 플랫폼.

## Angular란?

Angular는 Google이 개발한 모바일 및 데스크톱 웹 애플리케이션 구축을 위한 개발 플랫폼입니다. TypeScript/JavaScript及其他 언어를 사용합니다.

### 주요 기능

- **컴포넌트 기반** - 컴포넌트 중심 아키텍처
- **TypeScript** - 강한 타입 지원
- **양방향 바인딩** - 데이터 동기화 간소화
- **의존성 주입** - 강력한 의존성 관리 시스템
- **라우팅** - 완전한 라우팅 솔루션
- **폼** - 반응형 및 템플릿 기반 폼
- **HTTP 클라이언트** - 완전한 HTTP 서비스
- **서버사이드 렌더링** - SSR 지원
