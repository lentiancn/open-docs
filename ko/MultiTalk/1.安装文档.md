# MultiTalk 설치 가이드

MultiTalk는 자연스러운 다중 음성 오디오 콘텐츠를 생성하는 다중 화자 오디오 생성 도구입니다.

## 버전 정보

| 버전 | 상태 | 설명 |
|---------|--------|-------------|
| MultiTalk v1.0 | ✅ 현재 | 초기 릴리스 |

## 시스템 요구사항

### 하드웨어 요구사항

| 요구사항 | 최소 | 권장 |
|-------------|---------|-------------|
| GPU | NVIDIA 4GB VRAM | NVIDIA 8GB+ VRAM |
| RAM | 8GB | 16GB+ |
| 저장공간 | 5GB | 10GB+ SSD |

### 소프트웨어 요구사항

| 소프트웨어 | 버전 |
|----------|---------|
| Python | 3.8+ |
| CUDA | 11.0+ |

## 설치 방법

### 방법 1: 소스에서 설치

#### 1. 저장소 복제

```bash
git clone https://github.com/MultiTalk/MultiTalk.git
cd MultiTalk
```

#### 2. 가상 환경 생성

```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
# 또는
venv\Scripts\activate  # Windows
```

#### 3. 종속 항목 설치

```bash
pip install -r requirements.txt
```

#### 4. 모델 다운로드

```bash
bash scripts/download_models.sh
```

### 방법 2: Docker 사용

```bash
docker pull multitalk/multitalk:latest
docker run --gpus all -v /path/to/data:/data multitalk/multitalk:latest
```

## 구성

### 환경 변수

```bash
export CUDA_VISIBLE_DEVICES=0
export PYTHONPATH=/path/to/MultiTalk:$PYTHONPATH
```

### 모델 경로

`config.yaml` 편집:

```yaml
model:
  checkpoint_path: "checkpoints/multitalk.pth"
```

## 빠른 시작

### 다중 화자 오디오 생성

```bash
python inference.py \
  --text "안녕하세요, 다중 화자 데모입니다." \
  --speakers speaker1,speaker2 \
  --output output.wav
```

### 매개변수

```bash
--text         # 입력 텍스트
--speakers     # 화자 ID (쉼표로 구분)
--output       # 출력 파일 경로
--speed        # 음성 속도 (기본값: 1.0)
```
