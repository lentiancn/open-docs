# Go 소개

> Go는 Google이 개발한 컴파일 프로그래밍 언어

---

## 개요

Go ( Golang)는 2009년 Google에서 Robert Griesemer, Rob Pike, Ken Thompson이 만들었습니다. 단순성, 효율성, 강력한 동시성 지원으로 유명하며, 서버 개발, 클라우드 네이티브 애플리케이션, 컨테이너화에 널리 사용됩니다.

---

## 기능

- **정적 타이핑**: 컴파일 타임 타입 안전성
- **가비지 컬렉션**: 자동 메모리 관리
- **네이티브 동시성**: 가벼운 고루틴과 채널
- **단순하고 효율적**: 깔끔한 문법, 빠른 컴파일
- **풍부한 표준 라이브러리**: 포괄적인 내장 라이브러리

---

## 사용 사례

- 서버 프로그래밍
- 클라우드 네이티브 애플리케이션 (Kubernetes, Docker)
- 네트워크 프로그래밍
- 명령줄 도구
- 분산 시스템
