## 安装方式

### Windows 安装

#### 使用 MSI 安装包

1. 下载 Go 安装包：https://go.dev/dl/
2. 运行 MSI 安装包
3. 按照安装向导完成安装
4. 打开命令提示符，输入 `go version` 验证安装

#### 使用 Chocolatey

```powershell
choco install golang
```

### macOS 安装

#### 使用 Homebrew

```bash
brew install go
```

#### 使用 DMG 安装包

1. 下载 macOS 安装包：https://go.dev/dl/
2. 打开包文件并运行
3. 将 Go 目录移动到 /usr/local/go：`sudo mv go /usr/local/`

### Linux 安装

#### 使用包管理器

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install golang-go

# CentOS/RHEL
sudo yum install golang
```

#### 手动安装

```bash
# 下载 Go 二进制包
wget https://go.dev/dl/go1.22.0.linux-amd64.tar.gz

# 解压到 /usr/local
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go1.22.0.linux-amd64.tar.gz

# 添加到 PATH
export PATH=$PATH:/usr/local/go/bin

# 验证安装
go version
```

## 环境配置

### GOPATH 设置

```bash
# Linux/macOS
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin

# Windows (PowerShell)
$env:GOPATH = "$env:USERPROFILE\go"
$env:PATH += ";$env:GOPATH\bin"
```

### Go Module 代理配置

```bash
# 使用国内镜像
go env -w GOPROXY=https://goproxy.cn,direct

# 设置代理
go env -w GOPROXY=https://proxy.golang.org,direct
```

### 验证安装

```bash
# 查看版本
go version

# 查看环境信息
go env

# 运行测试
go version
```

## 常用命令

```bash
# 初始化模块
go mod init <模块名>

# 下载依赖
go mod tidy

# 构建项目
go build

# 运行程序
go run main.go

# 运行测试
go test ./...

# 安装工具
go install
```

## 故障排除

### 安装失败？

- 检查系统兼容性
- 确认下载的版本与系统架构匹配
- 检查权限是否足够

### 命令找不到？

- 确认 Go 已添加到 PATH 环境变量
- 重新打开终端
- 检查安装路径

### 依赖下载慢？

- 配置国内镜像代理
- 使用 VPN
