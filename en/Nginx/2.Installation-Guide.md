# Nginx Installation Guide

## Requirements

Nginx supports multiple operating systems: Linux, FreeBSD, Solaris, macOS, Windows.

## Installation Methods

### Using Package Manager (Recommended)

**Ubuntu/Debian:**
```bash
sudo apt update
sudo apt install nginx
sudo systemctl start nginx
```

**CentOS/RHEL:**
```bash
sudo yum install epel-release
sudo yum install nginx
```

**macOS:**
```bash
brew install nginx
```

### Using Docker

```bash
docker pull nginx
docker run -d -p 80:80 nginx
```

### From Source

```bash
wget https://nginx.org/download/nginx-1.26.0.tar.gz
tar -zxf nginx-1.26.0.tar.gz
cd nginx-1.26.0
./configure
make
sudo make install
```

## Verify Installation

```bash
nginx -v
curl http://localhost
```

## Default Paths

- Config: `/etc/nginx/nginx.conf`
- Logs: `/var/log/nginx/`
- Document Root: `/var/www/html/`

## Common Commands

```bash
nginx -s reload    # Reload config
nginx -s stop      # Stop
nginx -t            # Test config
```
