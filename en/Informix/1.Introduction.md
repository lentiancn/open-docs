# IBM Informix Introduction

## What is IBM Informix?

IBM Informix is a family of relational database management systems (RDBMS) and multi-model database offerings developed originally by Informix Corporation and acquired by IBM in 2001. In 2017, IBM delegated active development and support to HCLSoftware, while IBM continues to market and sell it as IBM Informix.

Informix is optimized for environments with very low or no database administration, including use as an embedded database. It has a long track record of supporting very high transaction rates and providing uptime characteristics needed for mission-critical applications such as manufacturing lines and reservation systems.

## Key Features

- **Multi-model Database**: Supports relational and object-oriented capabilities to model complex data structures
- **High Performance**: Optimized for always-on transactions and real-time analytics
- **Small Footprint**: Can run on devices as compact as 100 MB (e.g., Raspberry Pi)
- **Edge Computing**: Ideal for IoT solutions with embedded database capabilities
- **TimeSeries Support**: Built-in support for time-series data
- **JSON/MongoDB Compatibility**: Integration with JSON collections and MongoDB JSON drivers
- **High Availability**: Features like HDR (High Availability Data Replication), RSS, and SDS

## Product Editions

As of 2025-2026, IBM Informix offers the following editions:

| Edition | Target Users | Limits | Key Features |
|---------|-------------|--------|---------------|
| **Developer Edition** | Individual developers | 1 CPU VP/Core, 1GB RAM, 8GB storage | No-cost for development/testing |
| **Innovator-C Edition** | Small businesses | 2 CPU Cores, 8GB RAM, 50GB storage | Free for development/small production |
| **Express Edition** | Third-party developers | 4 CPU Cores, 8GB RAM | Embeddable database |
| **Workgroup Edition** | Midsize companies | Max 24 CPU cores, 32GB RAM | All replication options |
| **Enterprise Edition** | Large enterprises | Unlimited cores/RAM | All features except IWA |
| **Advanced Enterprise Edition** | Enterprise + BI | Unlimited | Complete suite including IWA |

## Version History

- **Current Version**: IBM Informix 15.0.0.1 (March 2025)
