# Jenkins 설치 가이드

Jenkins는 지속적인 통합과 지속적인 배포(CI/CD)를 구현하기 위한 오픈소스 자동화 서버입니다. 이 가이드에서는 다양한 환경에서의 설치 방법을 설명합니다.

## 버전 정보

| 버전 | 상태 | 설명 |
|------|------|------|
| Jenkins 2.x | ✅ 현재 | 최신 안정 버전 |
| Jenkins 1.x | ❌ 폐기됨 | 유지보수 중단 |

## 시스템 요구사항

### 하드웨어 요구사항

| 요구사항 | 최소 | 권장 |
|----------|------|------|
| CPU | 1코어 | 2+코어 |
| 메모리 | 1GB | 4+GB |
| 디스크 | 10GB | 50+GB |

### 소프트웨어 요구사항

| 소프트웨어 | 버전 |
|------------|------|
| Java | JDK 8 이상 |
| 운영체제 | Windows/macOS/Linux |

## 설치 방법

### 방법 1: Windows 설치

#### 1. 설치 프로그램 사용

1. Jenkins 다운로드: https://www.jenkins.io/download/
2. jenkins.msi 설치 프로그램 실행
3. 설치 마법사 따라 진행

#### 2. Chocolatey 사용

```powershell
choco install jenkins
```

#### 3. 수동 설치

```powershell
# WAR 파일 다운로드
# 지정된 디렉토리에 압축 해제
java -jar jenkins.war
```

### 방법 2: macOS 설치

#### 1. Homebrew 사용

```bash
# Jenkins 설치
brew install jenkins

# 서비스 시작
brew services start jenkins

# 서비스 중지
brew services stop jenkins
```

#### 2. 수동 설치

```bash
# WAR 파일 다운로드
wget https://get.jenkins.io/jenkins.war

# 실행
java -jar jenkins.war
```

### 방법 3: Linux 설치

#### Ubuntu/Debian

```bash
# 키 추가
wget -q -O - https://pkg.jenkins.io/debian/jenkins.io.key | sudo apt-key add -

# 저장소 추가
sudo sh -c 'echo deb https://pkg.jenkins.io/debian binary/ > /etc/apt/sources.list.d/jenkins.list'

# 업데이트 및 설치
sudo apt-get update
sudo apt-get install jenkins

# 서비스 시작
sudo systemctl start jenkins
sudo systemctl enable jenkins
```

#### CentOS/RHEL

```bash
# 저장소 추가
sudo wget -O /etc/yum.repos.d/jenkins.repo https://pkg.jenkins.io/redhat/jenkins.repo

# 키 가져오기
sudo rpm --import https://pkg.jenkins.io/redhat/jenkins.io.key

# 설치
sudo yum install jenkins

# 서비스 시작
sudo systemctl start jenkins
sudo systemctl enable jenkins
```

### 방법 4: Docker 설치

```bash
# Jenkins 컨테이너 실행
docker run -d \
  --name jenkins \
  -p 8080:8080 \
  -v jenkins_home:/var/jenkins_home \
  jenkins/jenkins:lts
```

#### Docker Compose

```yaml
version: '3'
services:
  jenkins:
    image: jenkins/jenkins:lts
    ports:
      - "8080:8080"
    volumes:
      - jenkins_home:/var/jenkins_home
```

### 방법 5: WAR 파일 배포

```bash
# WAR 파일 다운로드
wget https://get.jenkins.io/war-stable/latest/jenkins.war

# 실행
java -jar jenkins.war --httpPort=8080
```

## 설치 후 설정

### 첫 번째 액세스

1. 브라우저에서 http://localhost:8080 열기
2. 초기 관리자 비밀번호 확인:
   ```bash
   # Linux/macOS
   sudo cat /var/lib/jenkins/secrets/initialAdminPassword
   
   # Docker
   docker exec jenkins cat /var/jenkins_home/secrets/initialAdminPassword
   ```
3. 권장 플러그인 설치
4. 관리자 사용자 생성

### 한국어 인터페이스 설정

1. Manage Jenkins → Manage Plugins로 이동
2. "Localization: Korean" 설치
3. Jenkins 재시작

## 자주 묻는 질문

### 시작 실패?

Java 설치 여부 확인:

```bash
java -version
```

### 포트 사용 중?

포트 변경:

```bash
java -jar jenkins.war --httpPort=8090
```

### 플러그인 설치 실패?

네트워크 연결 확인 또는 프록시 사용.
