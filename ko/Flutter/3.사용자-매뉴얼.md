# Flutter - 사용자 매뉴얼

기본 Flutter 개발 예제.

## 프로젝트 생성

```bash
flutter create 내_앱
cd 내_앱
flutter run
```

## 기본 위젯

```dart
Text('안녕하세요 Flutter')
```

## 상태

```dart
class 카운터 extends StatefulWidget {
  @override
  _카운터State createState() => _카운터State();
}

class _카운터State extends State<카운터> {
  int _카운트 = 0;
  void _증가() => setState(() => _카운트++);

  @override
  Widget build(BuildContext context) {
    return Column(children: [
      Text('카운트: $_카운트'),
      ElevatedButton(onPressed: _증가, child: Text('증가')),
    ]);
  }
}
```
