# 자주 묻는 질문

이 문서에서는 Apifox 사용 과정에서 흔히 마주치는 기술적인 질문과 답변을 정리했습니다.

## 기초적인 질문

### Apifox란 무엇인가요?

Apifox는 API 협업 플랫폼으로, API 설계, 문서 관리, 디버깅, Mock, 자동화 테스트 등의 기능을 통합했습니다. 개발 팀이 더 효율적으로 API를 개발, 테스트하고 문서를 관리할 수 있도록 도와줍니다.

### Apifox는 무료인가요?

Apifox는 무료 버전과 유료 버전을 제공합니다. 무료 버전에는 이미 모든 핵심 기능이 포함되어 있어 대부분의 개발 요구를 충족할 수 있습니다. 유료 버전은 더 큰 팀 규모, 고급 자동화 등 추가 고급 기능을 제공합니다.

### Apifox와 Postman의 차이점은 무엇인가요?

Apifox와 Postman 모두 API 도구이지만, Apifox가 중국 국내 사용자에게 더 적합합니다: 더 나은 중국어 지원, 더 똑똑한 Mock 기능, 더 완성된 팀 협업 경험. Apifox는 기본 기능이 완전히 무료인 반면, Postman은 많은 기능이 유료입니다.

## 설치 관련 질문

### 다운로드 속도가 느려요 어떻게 하나요?

Apifox 공식에서는 여러 다운로드 노드를 제공합니다. https://apifox.com/을 방문하여 가장 가까운 노드를 선택하여 다운로드해 보세요.

### 실행이 안 돼요 어떻게 하나요?

1. 시스템 버전이 요구사항을 만족하는지 확인
2. 관리자 권한으로 실행 시도
3. 필요한 런타임 라이브러리가 설치되어 있는지 확인
4. 자세한 정보는 오류 로그 확인

### 로그인이 안 돼요 어떻게 하나요?

1. 네트워크 연결이 정상인지 확인
2. 계정과 비밀번호가 정확한지 확인
3. Third-party 계정으로 로그인 시도
4. 공식 고객센터에 문의하여 도움 요청

## 사용 관련 질문

### 인터페이스 데이터가 동기화되지 않아요

1. 계정에 로그인되어 있는지 확인
2. 네트워크 연결 확인
3. "동기화" 버튼을 클릭하여 수동 동기화
4. 계정 권한 확인

### Mock 데이터가 적용되지 않아요

1. Mock 기능이 활성화되어 있는지 확인
2. Mock 주소가 정확한지 확인
3. 인터페이스에 문법 오류가 있는지 확인
4. 페이지 새로고침 시도

### 요청 보내기가 실패해요

1. 네트워크 연결 확인
2. 요청 주소가 정확한지 확인
3. 요청 매개변수 형식 확인
4. 프록시에서 차단되지 않았는지 확인

### 응답이 깨져 보여요

1. 응답 인코딩 변경 시도
2. 응답 헤더의 인코딩 설정 확인
3. "포맷" 기능을 사용하여 다시 파싱

## 기능 관련 질문

### Postman 데이터는 어떻게 가져오나요?

1. "가져오기" 클릭
2. "Postman" 선택
3. 가져올 파일 선택
4. 가져오기 확인

### Swagger로 어떻게 내보내나요?

1. 프로젝트 설정으로 이동
2. "내보내기" 선택
3. "Swagger/OpenAPI" 선택
4. 버전 선택
5. 내보내기 클릭

### 환경 변수는 어떻게 사용하나요?

1. 프로젝트 설정에서 환경 변수 정의
2. 요청에서 {{변수명}}으로 참조
3. 상단 환경 선택기로 환경 전환

### 인터페이스를 일괄로 어떻게 작업하나요?

1. Ctrl/Cmd를 누른 상태에서 여러 인터페이스 선택
2. 우클릭하여 일괄 작업 선택
3. 작업 유형 선택 (삭제, 내보내기, 이동 등)

## 테스트 관련 질문

### 자동화 테스트는 어떻게 실행하나요?

1. 테스트 스위트 열기
2. "실행" 버튼 클릭
3. 테스트 결과 확인

### assertion이 실패했어요 어떻게 하나요?

1. 실패 이유 확인
2. 예상 값과 실제 값 확인
3. assertion 조건 수정
4. 테스트 다시 실행

### 테스트 보고서는 어떻게 내보내나요?

1. 테스트 실행 완료 후
2. "보고서 내보내기" 클릭
3. 형식 선택 (HTML, PDF, JSON)

## 팀 협업 관련 질문

### 팀원은 어떻게 초대하나요?

1. 프로젝트 설정으로 이동
2. "멤버 관리" 클릭
3. "팀원 초대" 클릭
4. 팀원 이메일 또는 사용자 이름 입력

### 멤버 권한은 어떻게 설정하나요?

팀원을 초대할 때 다양한 역할을 설정할 수 있습니다:

- 관리자: 전체 제어
- 편집자: 편집 및 실행 가능
- 뷰어: 읽기 전용 권한

### 충돌은 어떻게 해결하나요?

여러 사람이 동시에 같은 인터페이스를 편집할 때:

1. 시스템이 충돌을 안내
2. 로컬 버전 또는 원격 버전 유지 선택
3. 또는 수동으로 수정 병합

## 데이터 관련 질문

### 데이터는 어떻게 백업하나요?

1. 정기적으로 Apifox 형식으로 프로젝트 내보내기
2. 또는 클라우드 동기화 기능 사용

### 데이터는 어떻게 복구하나요?

1. 백업 파일이 있다면 "가져오기" 클릭
2. 백업 파일 선택하여 복구

### 삭제한 데이터는 복구할 수 있나요?

삭제된 데이터는 휴지통에 30일 동안 보관되며, 휴지통에서 복구할 수 있습니다.

## 성능 관련 질문

### 실행이 버거워요 어떻게 하나요?

1. 필요 없는 프로젝트 닫기
2. 브라우저 캐시 정리
3. 하드웨어 사양 업그레이드
4. 최신 버전 사용해 보기

### 많은 데이터 가져오기가 느려요

1. 분할하여 가져오기
2. 자동 동기화 끄기
3. 가져오기 완료까지 인내

Apifox는 데스크톱 위젯 기능을 지원하여 데스크톱에서 자주 사용하는 인터페이스에 빠르게 접근할 수 있습니다.
