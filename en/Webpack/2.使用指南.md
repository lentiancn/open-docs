# Webpack 使用指南

本指南介绍如何使用 Webpack 构建现代 JavaScript 应用程序。

## 快速开始

### 创建项目

```bash
mkdir my-project
cd my-project
npm init -y
npm install webpack webpack-cli --save-dev
```

### 创建入口文件

```javascript
// src/index.js
import './style.css';
import { add } from './math.js';

console.log('Hello Webpack!');
console.log(add(2, 3));
```

```javascript
// src/math.js
export function add(a, b) {
  return a + b;
}
```

```css
/* src/style.css */
body {
  background: #f0f0f0;
  font-family: sans-serif;
}
```

```html
<!-- src/index.html -->
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Webpack App</title>
</head>
<body>
  <h1>Hello Webpack!</h1>
  <script src="index.js"></script>
</body>
</html>
```

### 创建配置文件

```javascript
// webpack.config.js
const path = require('path');

module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
  },
  module: {
    rules: [
      {
        test: /\.css$/i,
        use: ['style-loader', 'css-loader'],
      },
    ],
  },
};
```

### 运行构建

```bash
npx webpack
```

## 核心概念

### 入口（Entry）

```javascript
module.exports = {
  // 单个入口
  entry: './src/index.js',
  
  // 多入口
  entry: {
    home: './src/home.js',
    about: './src/about.js',
  },
};
```

### 输出（Output）

```javascript
module.exports = {
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
    clean: true,
    publicPath: '/',
  },
};
```

### 加载器（Loaders）

```javascript
module.exports = {
  module: {
    rules: [
      {
        test: /\.css$/,
        use: ['style-loader', 'css-loader'],
      },
      {
        test: /\.js$/,
        exclude: /node_modules/,
        use: 'babel-loader',
      },
    ],
  },
};
```

### 插件（Plugins）

```javascript
const HtmlWebpackPlugin = require('html-webpack-plugin');
const MiniCssExtractPlugin = require('mini-css-extract-plugin');

module.exports = {
  plugins: [
    new HtmlWebpackPlugin({
      template: './src/index.html',
    }),
    new MiniCssExtractPlugin({
      filename: '[name].css',
    }),
  ],
};
```

### 模式（Mode）

```javascript
// 开发模式
module.exports = {
  mode: 'development',
  devtool: 'source-map',
};

// 生产模式
module.exports = {
  mode: 'production',
  devtool: false,
};
```

## 常用加载器

### 处理 CSS

```bash
npm install --save-dev style-loader css-loader
```

```javascript
{
  test: /\.css$/,
  use: ['style-loader', 'css-loader'],
}
```

### 处理 Sass

```bash
npm install --save-dev sass sass-loader
```

```javascript
{
  test: /\.scss$/,
  use: ['style-loader', 'css-loader', 'sass-loader'],
}
```

### 处理图片

```javascript
{
  test: /\.(png|jpg|gif)$/,
  type: 'asset/resource',
}
```

### 处理字体

```javascript
{
  test: /\.(woff|woff2|eot|ttf|otf)$/,
  type: 'asset/resource',
}
```

## 开发服务器

### 安装

```bash
npm install --save-dev webpack-dev-server
```

### 配置

```javascript
module.exports = {
  devServer: {
    static: './dist',
    hot: true,
    port: 3000,
    open: true,
    compress: true,
  },
};
```

### 启动

```bash
webpack serve --open
```

或添加脚本到 package.json：

```json
{
  "scripts": {
    "start": "webpack serve --mode development",
    "build": "webpack --mode production"
  }
}
```

## 代码分割

### 动态导入

```javascript
// 路由懒加载
const Home = () => import('./Home.vue');
const About = () => import('./About.vue');

// 条件加载
if (condition) {
  import('./module').then(module => {
    module.doSomething();
  });
}
```

### SplitChunks

```javascript
module.exports = {
  optimization: {
    splitChunks: {
      chunks: 'all',
      cacheGroups: {
        vendor: {
          test: /[\\/]node_modules[\\/]/,
          name: 'vendors',
          chunks: 'all',
        },
      },
    },
  },
};
```

## 环境配置

### 开发环境

```javascript
module.exports = {
  mode: 'development',
  devtool: 'inline-source-map',
  devServer: {
    static: './dist',
    hot: true,
  },
};
```

### 生产环境

```javascript
module.exports = {
  mode: 'production',
  devtool: 'source-map',
  optimization: {
    minimize: true,
  },
};
```

### 多环境配置

```javascript
const devConfig = require('./webpack.dev');
const prodConfig = require('./webpack.prod');

module.exports = (env, argv) => {
  return argv.mode === 'production' ? prodConfig : devConfig;
};
```

## 性能优化

### 1. 使用 production 模式

```javascript
module.exports = {
  mode: 'production',
};
```

### 2. 启用 Tree Shaking

```javascript
// package.json
{
  "sideEffects": false
}
```

### 3. 代码分割

```javascript
optimization: {
  splitChunks: {
    chunks: 'all',
  },
}
```

### 4. 缓存优化

```javascript
output: {
  filename: '[name].[contenthash].js',
  chunkFilename: '[name].[contenthash].chunk.js',
}
```

### 5. 压缩

```javascript
const TerserPlugin = require('terser-webpack-plugin');

optimization: {
  minimizer: [new TerserPlugin()],
}
```

## 常见问题

### 模块解析失败？

检查 resolve 配置：

```javascript
resolve: {
  extensions: ['.js', '.json'],
  alias: {
    '@': path.resolve(__dirname, 'src'),
  },
}
```

### 构建太慢？

- 使用 cache
- 排除 node_modules
- 使用 thread-loader

```javascript
{
  test: /\.js$/,
  exclude: /node_modules/,
  use: {
    loader: 'thread-loader',
    options: {
      workers: 2,
    },
  },
}
```
