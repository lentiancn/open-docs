## インストール方法

### Windows インストール

#### MSIインスーラーを使う

1. Goをダウンロード：https://go.dev/dl/
2. MSIインスーラーを実行
3. インストールウィザードに従う
4. コマンドプロンプトを開き、`go version`と入力して確認

#### Chocolateyを使う

```powershell
choco install golang
```

### macOS インストール

#### Homebrewを使う

```bash
brew install go
```

#### DMGパッケージを使う

1. macOS用パッケージをダウンロード：https://go.dev/dl/
2. パッケージファイルを開いて実行
3. Goディレクトリを/usr/local/goに移動：`sudo mv go /usr/local/`

### Linux インストール

#### パッケージマネージャーを使う

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install golang-go

# CentOS/RHEL
sudo yum install golang
```

#### 手動インストール

```bash
# Goバイナリをダウンロード
wget https://go.dev/dl/go1.22.0.linux-amd64.tar.gz

# /usr/localに展開
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go1.22.0.linux-amd64.tar.gz

# PATHに追加
export PATH=$PATH:/usr/local/go/bin

# インストール確認
go version
```

## 環境設定

### GOPATHの設定

```bash
# Linux/macOS
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin

# Windows (PowerShell)
$env:GOPATH = "$env:USERPROFILE\go"
$env:PATH += ";$env:GOPATH\bin"
```

### Goモジュールプロキシの設定

```bash
# 中国ミラーを使う
go env -w GOPROXY=https://goproxy.cn,direct

# プロキシを設定
go env -w GOPROXY=https://proxy.golang.org,direct
```

### インストール確認

```bash
# バージョンを確認
go version

# 環境情報を表示
go env

# テストを実行
go version
```

## 一般的なコマンド

```bash
# モジュールを初期化
go mod init <モジュール名>

# 依存関係をダウンロード
go mod tidy

# プロジェクトをビルド
go build

# プログラムを実行
go run main.go

# テストを実行
go test ./...

# ツールをインストール
go install
```

## トラブルシューティング

### インストール失敗？

- システムの互換性を確認
- ダウンロードしたバージョンがアーキテクチャと一致するか確認
- 権限を確認

### コマンドが見つからない？

- GoがPATH環境変数に追加されているか確認
- ターミナルを再起動
- インストールパスを確認

### 依存関係のダウンロードが遅い？

- 中国ミラープロキシを設定
- VPNを使用
