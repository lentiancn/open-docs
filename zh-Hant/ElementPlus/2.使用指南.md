# Element Plus 使用指南

本指南介紹如何在 Vue 3 應用中使用 Element Plus 組件。

## 快速開始

### 基礎模板

```vue
<template>
  <div>
    <el-button type="primary">主要按鈕</el-button>
    <el-input v-model="input" placeholder="輸入文本" />
    <el-date-picker v-model="date" type="date" placeholder="選擇日期" />
  </div>
</template>

<script setup>
import { ref } from 'vue'

const input = ref('')
const date = ref('')
</script>
```

### 表單

```vue
<template>
  <el-form :model="form" :rules="rules" ref="formRef" label-width="120px">
    <el-form-item label="活動名稱" prop="name">
      <el-input v-model="form.name"></el-input>
    </el-form-item>
    
    <el-form-item label="活動區域" prop="region">
      <el-select v-model="form.region" placeholder="請選擇">
        <el-option label="區域一" value="shanghai"></el-option>
        <el-option label="區域二" value="beijing"></el-option>
      </el-select>
    </el-form-item>
    
    <el-form-item label="活動時間">
      <el-date-picker
        v-model="form.date"
        type="datetimerange"
        range-separator="至"
        start-placeholder="開始日期"
        end-placeholder="結束日期"
      ></el-date-picker>
    </el-form-item>
    
    <el-form-item label="即時配送">
      <el-switch v-model="form.delivery"></el-switch>
    </el-form-item>
    
    <el-form-item label="活動資源">
      <el-radio-group v-model="form.resource">
        <el-radio label="線上品牌商贊助">線上品牌商贊助</el-radio>
        <el-radio label="線下場地免費">線下場地免費</el-radio>
      </el-radio-group>
    </el-form-item>
    
    <el-form-item label="活動形式">
      <el-input v-model="form.desc" type="textarea"></el-input>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="onSubmit">創建</el-button>
      <el-button @click="onReset">重置</el-button>
    </el-form-item>
  </el-form>
</template>

<script setup>
import { reactive, ref } from 'vue'

const formRef = ref(null)

const form = reactive({
  name: '',
  region: '',
  date: [],
  delivery: false,
  resource: '',
  desc: ''
})

const rules = {
  name: [
    { required: true, message: '請輸入活動名稱', trigger: 'blur' },
    { min: 3, max: 5, message: '長度在 3 到 5 個字符', trigger: 'blur' }
  ],
  region: [
    { required: true, message: '請選擇活動區域', trigger: 'change' }
  ]
}

const onSubmit = () => {
  formRef.value.validate((valid) => {
    if (valid) {
      console.log('submit!')
    }
  })
}

const onReset = () => {
  formRef.value.resetFields()
}
</script>
```

```vue

  name: '',
  age: 0
```

```vue
<script setup>
const rules = {
```

## 常見問題

### 組件不渲染？

確保組件已正確導入或全域註冊。

### 樣式未生效？

檢查是否已引入 Element Plus 樣式：

```javascript
import 'element-plus/dist/index.css'
```

### 圖標不顯示？

全域註冊圖標：

```javascript
import * as icons from '@element-plus/icons-vue'
for (const [key, icon] of Object.entries(icons)) {
    app.component(key, icon)
}
```
