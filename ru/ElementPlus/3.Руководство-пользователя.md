# Руководство пользователя Element Plus

Это руководство поможет вам быстро освоить Element Plus и освоить наиболее часто используемые компоненты и функции.

## Быстрый старт

Перед использованием Element Plus убедитесь, что установка завершена.

### Основное использование

Используйте компоненты непосредственно в компонентах Vue:

<template>
  <el-button>Кнопка по умолчанию</el-button>
  <el-button type="primary">Основная кнопка</el-button>
  <el-button type="success">Кнопка успеха</el-button>
  <el-button type="warning">Кнопка предупреждения</el-button>
  <el-button type="danger">Кнопка опасности</el-button>
</template>

## Часто используемые компоненты

### Кнопка

Основное использование:

<el-button>По умолчанию</el-button>
<el-button type="primary">Основная</el-button>
<el-button type="success">Успех</el-button>
<el-button type="warning">Предупреждение</el-button>
<el-button type="danger">Опасность</el-button>

Разные размеры:

<el-button size="large">Большая</el-button>
<el-button>По умолчанию</el-button>
<el-button size="small">Маленькая</el-button>

Отключенное состояние:

<el-button disabled>Отключено</el-button>

Кнопки с иконкой:

<el-button icon="Search">Поиск</el-button>
<el-button icon="Edit">Редактировать</el-button>
<el-button icon="Delete">Удалить</el-button>

### Ввод

Основное использование:

<el-input v-model="input" placeholder="Пожалуйста, введите содержимое" />

С иконкой:

<el-input v-model="input" prefix-icon="Search" />
<el-input v-model="input" suffix-icon="Calendar" />

С кнопкой очистки:

<el-input v-model="input" clearable />

Ввод пароля:

<el-input v-model="input" type="password" show-password />

Многострочный текст:

<el-input v-model="textarea" type="textarea" :rows="3" />

### Выбор

Базовый выбор:

<el-select v-model="value" placeholder="Пожалуйста, выберите">
  <el-option label="Опция 1" value="1" />
  <el-option label="Опция 2" value="2" />
</el-select>

Множественный выбор:

<el-select v-model="value" multiple placeholder="Пожалуйста, выберите">
  <el-option label="Опция 1" value="1" />
  <el-option label="Опция 2" value="2" />
</el-select>

### Таблица

Базовая таблица:

<el-table :data="tableData" stripe border>
  <el-table-column prop="date" label="Дата" width="180" />
  <el-table-column prop="name" label="Имя" width="180" />
  <el-table-column prop="address" label="Адрес" />
</el-table>

С индексом:

<el-table-column type="index" label="Номер" width="60" />

Фиксированные столбцы:

<el-table-column prop="date" label="Дата" width="150" fixed />

### Форма

Базовая форма:

<el-form :model="form" label-width="80px">
  <el-form-item label="Название активности">
    <el-input v-model="form.name" />
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="onSubmit">Отправить</el-button>
  </el-form-item>
</el-form>

Валидация формы:

<el-form :model="ruleForm" :rules="rules" ref="ruleFormRef">
  <el-form-item label="Имя" prop="name">
    <el-input v-model="ruleForm.name" />
  </el-form-item>
</el-form>

### Диалог

<el-dialog v-model="visible" title="Заголовок" width="30%">
  <span>Содержимое</span>
</el-dialog>

### Сообщение

import { ElMessage } from 'element-plus'

ElMessage.success('Сообщение об успехе')
ElMessage.warning('Предупреждающее сообщение')
ElMessage.error('Сообщение об ошибке')

### Меню

Горизонтальное меню:

<el-menu mode="horizontal">
  <el-menu-item index="1">Центр обработки</el-menu-item>
  <el-menu-item index="2">Управление заказами</el-menu-item>
</el-menu>

Вертикальное меню:

<el-menu default-active="1">
  <el-menu-item index="1">Навигация один</el-menu-item>
  <el-menu-item index="2">Навигация два</el-menu-item>
</el-menu>

### Вкладки

<el-tabs v-model="activeName">

### Пагинация

<el-pagination

  :total="1000"

/>

## Система макета

### Сеточная система

Element Plus использует 24-колоночную сеточную систему:

<el-row :gutter="20">
  <el-col :span="12"><div>12 колонок</div></el-col>
  <el-col :span="12"><div>12 колонок</div></el-col>
</el-row>

### Макет контейнера

<el-container>
  <el-header>Заголовок</el-header>
  <el-main>Основная часть</el-main>
  <el-footer>Нижний колонтитул</el-footer>
</el-container>

## Распространенные проблемы

### Стили компонентов не применяются

Убедитесь, что CSS импортирован:

import 'element-plus/dist/index.css'

### Иконки не отображаются

Вам нужно установить библиотеку иконок:

npm install @element-plus/icons-vue

Затем зарегистрируйте глобально:

import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}

### Импорт по требованию не работает

Проверьте правильность конфигурации плагина и убедитесь, что используется правильный резолвер.
