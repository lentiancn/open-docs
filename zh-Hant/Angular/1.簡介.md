# Angular 簡介

現代 Web 開發平台。

## 什麼是 Angular？

Angular 是由 Google 開發的設計平台，用於構建移動端和桌面端 Web 應用。它使用 TypeScript/JavaScript 及其他語言。

### 主要特性

- **組件化** - 基於組件的架構
- **TypeScript** - 強類型支持
- **雙向綁定** - 簡化數據同步
- **依賴注入** - 強大的依賴管理系統
- **路由** - 完整的路由解決方案
- **表單處理** - 響應式表單和模板驅動表單
- **HTTP 客戶端** - 完整的 HTTP 服務
- **服務端渲染** - 支持 SSR
