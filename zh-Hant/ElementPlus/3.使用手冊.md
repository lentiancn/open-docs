# Element Plus 使用手冊

本手冊將幫助你快速上手 Element Plus，掌握日常使用中常用的組件和功能。

## 快速開始

在使用 Element Plus 之前，確保已完成安裝配置。

### 基礎使用

在 Vue 組件中直接使用組件：

```vue
<template>
  <el-button>默認按鈕</el-button>
  <el-button type="primary">主要按鈕</el-button>
  <el-button type="success">成功按鈕</el-button>
  <el-button type="warning">警告按鈕</el-button>
  <el-button type="danger">危險按鈕</el-button>
</template>
```

## 常用組件

### 按鈕（Button）

基礎用法：

```vue
<el-button>默認</el-button>
<el-button type="primary">主要</el-button>
<el-button type="success">成功</el-button>
<el-button type="warning">警告</el-button>
<el-button type="danger">危險</el-button>
```

不同尺寸：

```vue
<el-button size="large">大型</el-button>
<el-button>默認</el-button>
<el-button size="small">小型</el-button>
```

禁用狀態：

```vue
<el-button disabled>禁用</el-button>
```

圖標按鈕：

```vue
<el-button icon="Search">搜索</el-button>
<el-button icon="Edit">編輯</el-button>
<el-button icon="Delete">刪除</el-button>
```

### 輸入框（Input）

基礎用法：

```vue
<el-input v-model="input" placeholder="請輸入內容" />
```

帶圖標：

```vue
<el-input v-model="input" prefix-icon="Search" />
<el-input v-model="input" suffix-icon="Calendar" />
```

帶清除按鈕：

```vue
<el-input v-model="input" clearable />
```

密碼輸入框：

```vue
<el-input v-model="input" type="password" show-password />
```

多行文本：

```vue
<el-input v-model="textarea" type="textarea" :rows="3" />
```

### 選擇器（Select）

基礎選擇器：

```vue
<el-select v-model="value" placeholder="請選擇">
  <el-option label="選項1" value="1" />
  <el-option label="選項2" value="2" />
</el-select>
```

多選：

```vue
<el-select v-model="value" multiple placeholder="請選擇">
  <el-option label="選項1" value="1" />
  <el-option label="選項2" value="2" />
</el-select>
```

### 表格（Table）

基礎表格：

```vue
<el-table :data="tableData" stripe border>
  <el-table-column prop="date" label="日期" width="180" />
  <el-table-column prop="name" label="姓名" width="180" />
  <el-table-column prop="address" label="地址" />
</el-table>
```

帶索引：

```vue
<el-table-column type="index" label="序號" width="60" />
```

固定列：

```vue
<el-table-column prop="date" label="日期" width="150" fixed />
```

### 表單（Form）

基礎表單：

```vue
<el-form :model="form" label-width="80px">
  <el-form-item label="活動名稱">
    <el-input v-model="form.name" />
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="onSubmit">提交</el-button>
  </el-form-item>
</el-form>
```

表單驗證：

```vue
<el-form :model="ruleForm" :rules="rules" ref="ruleFormRef">
  <el-form-item label="名稱" prop="name">
    <el-input v-model="ruleForm.name" />
  </el-form-item>
</el-form>
```

### 對話框（Dialog）

```vue
<el-dialog v-model="visible" title="提示" width="30%">
  <span>內容</span>
</el-dialog>
```

### 消息提示

```javascript
import { ElMessage } from 'element-plus'

ElMessage.success('成功消息')
ElMessage.warning('警告消息')
ElMessage.error('錯誤消息')
```

### 菜單（Menu）

水平菜單：

```vue
<el-menu mode="horizontal">
  <el-menu-item index="1">處理中心</el-menu-item>
  <el-menu-item index="2">訂單管理</el-menu-item>
</el-menu>
```

垂直菜單：

```vue
<el-menu default-active="1">
  <el-menu-item index="1">導航一</el-menu-item>
  <el-menu-item index="2">導航二</el-menu-item>
</el-menu>
```

### 標籤頁（Tabs）

```vue
<el-tabs v-model="activeName">
  <el-tab-pane label="用戶管理" name="first">用戶管理</el-tab-pane>
  <el-tab-pane label="配置管理" name="second">配置管理</el-tab-pane>
</el-tabs>
```

### 分頁（Pagination）

```vue
<el-pagination
  v-model:current-page="currentPage"
  :page-size="100"
  :total="1000"
  layout="total, prev, pager, next"
/>
```

## 佈局系統

### 柵格系統

Element Plus 使用 24 欄柵格系統：

```vue
<el-row :gutter="20">
  <el-col :span="12"><div>12欄</div></el-col>
  <el-col :span="12"><div>12欄</div></el-col>
</el-row>
```

### 容器佈局

```vue
<el-container>
  <el-header>Header</el-header>
  <el-main>Main</el-main>
  <el-footer>Footer</el-footer>
</el-container>
```

```javascript

app.use(ElementPlus, { locale: en })
```

## 常見問題

### 組件樣式不生效

確保已正確引入 CSS：

```javascript
import 'element-plus/dist/index.css'
```

### 圖標不顯示

需要額外安裝圖標庫：

```bash
npm install @element-plus/icons-vue
```

然後全局註冊：

```javascript
import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```

### 按需導入不生效

檢查插件配置是否正確，確保使用了正確的 resolver。
