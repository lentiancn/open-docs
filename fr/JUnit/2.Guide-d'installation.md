# JUnit Guide d'installation

JUnit peut être ajouté à votre projet de plusieurs manières, notamment Maven, Gradle et téléchargement direct de JAR. Ci-dessous les instructions détaillées pour chaque version.

## Installation de JUnit 5

### Configuration Maven

Ajoutez les dépendances suivantes dans votre pom.xml :

```xml
<!-- Module agrégateur JUnit Jupiter -->
<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter</artifactId>
    <version>5.14.3</version>
    <scope>test</scope>
</dependency>
```

Si vous utilisez Maven Surefire Plugin pour exécuter les tests, ajoutez la configuration suivante :

```xml
<plugin>
    <groupId>org.apache.maven.plugins</groupId>
    <artifactId>maven-surefire-plugin</artifactId>
    <version>3.5.2</version>
</plugin>
```

### Configuration Gradle

Ajoutez la dépendance dans votre build.gradle :

```groovy
testImplementation 'org.junit.jupiter:junit-jupiter:5.14.3'
```

Pour exécuter les tests :

```groovy
test {
    useJUnitPlatform()
}
```

### Téléchargement direct de JAR

Téléchargez les fichiers JAR nécessaires depuis Maven Central :

- junit-jupiter-api : API JUnit Jupiter
- junit-jupiter-engine : Moteur JUnit Jupiter
- junit-platform-console-standalone : Lanceur console autonome

## Installation de JUnit 4

### Configuration Maven

```xml
<dependency>
    <groupId>junit</groupId>
    <artifactId>junit</artifactId>
    <version>4.13.2</version>
    <scope>test</scope>
</dependency>
```

### Configuration Gradle

```groovy
testImplementation 'junit:junit:4.13.2'
```

### Téléchargement direct de JAR

Téléchargez junit-4.13.2.jar depuis Maven Central :

https://repo1.maven.org/maven2/junit/junit/4.13.2/junit-4.13.2.jar

## Installation de JUnit Vintage

Pour exécuter des tests JUnit 4 sur la plateforme JUnit 5, ajoutez la dépendance du moteur Vintage :

### Maven

```xml
<dependency>
    <groupId>org.junit.vintage</groupId>
    <artifactId>junit-vintage-engine</artifactId>
    <version>5.14.3</version>
    <scope>test</scope>
</dependency>
```

### Gradle

```groovy
testImplementation 'org.junit.vintage:junit-vintage-engine:5.14.3'
```

## Compatibilité des versions

- JUnit 5 nécessite Java 8 ou supérieur
- JUnit 4 nécessite Java 5 ou supérieur
- JUnit Vintage 5.14.3 nécessite JUnit 4.12 ou supérieur

## Vérifier l'installation

Créez une classe de test pour vérifier l'installation :

```java
import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;

class SampleTest {
    @Test
    void testExample() {
        assertEquals(2, 1 + 1);
    }
}
```

Exécutez les tests :

```bash
mvn test
# ou
gradle test
```
