# Angular Installation Guide

Detailed guide for setting up Angular environment.

## Requirements

### Node.js

- Recommended: Node.js 18+
- Requires npm or yarn

### Check Node.js Version

```bash
node --version
npm --version
```

## Install Angular CLI

### Install

```bash
npm install -g @angular/cli
```

### Verify Installation

```bash
ng version
```

### Upgrade

```bash
npm install -g @angular/cli@latest
```

## Create New Project

### Using Angular CLI

```bash
ng new my-app
cd my-app
```

### Interactive Options

- Add routing? Yes/No
- Which stylesheet format? (SCSS/CSS/Sass/Less)

## Run Application

### Development Server

```bash
ng serve
# or
npm start
```

Visit: http://localhost:4200

### Hot Reload

Angular CLI supports hot reload - changes automatically refresh.

## Build for Production

```bash
ng build --configuration production
```

Build output in `dist/` directory.

## Common Commands

| Command | Function |
|---------|----------|
| ng new | Create new project |
| ng serve | Start dev server |
| ng build | Build project |
| ng generate | Generate component/service/module |
| ng test | Run unit tests |
| ng e2e | Run e2e tests |

## Uninstall Angular CLI

```bash
npm uninstall -g @angular/cli
```
