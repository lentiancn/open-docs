# jQuery

## Descripción general

jQuery es una biblioteca de JavaScript rápida, pequeña y rica en funciones. Facilita la manipulación y recorrido de documentos HTML, manejo de eventos, animaciones y Ajax con una API fácil de usar que funciona en múltiples navegadores.

## Características

- **Sintaxis concisa**: API encadenable, código conciso y legible
- **Selectores potentes**: Soporte completo de selectores CSS
- **Compatibilidad entre navegadores**: Resuelve diferencias de API entre navegadores
- **Amplio ecosistema de plugins**: Extensa biblioteca de plugins
