# MySQL Installationsanleitung

## Installation unter Windows

### Installation mit MySQL Installer

MySQL Installer ist der einfachste Weg, MySQL unter Windows zu installieren, und bietet eineAssistenten-basierte Installation.

1. MySQL Installer herunterladen
   Besuchen Sie https://dev.mysql.com/downloads/installer/ um den Installer herunterzuladen.

2. Installer ausführen
   Doppelklicken Sie auf die heruntergeladene MSI-Datei, um den Installationsassistenten zu starten.

3. Installationstyp wählen
   - **Full**: Alle MySQL-Produkte und Funktionen installieren
   - **Developer Default**: Geeignet für Entwicklungsumgebungen
   - **Server Only**: Nur MySQL Server installieren
   - **Custom**: Bestimmte Komponenten auswählen

4. MySQL Server konfigurieren
   - MySQL Server-Version auswählen
   - Portnummer festlegen (Standard 3306)
   - Starttyp wählen

5. Root-Passwort festlegen
   Legen Sie ein sicheres Passwort für das MySQL-Administratorkonto fest.

6. Installation abschließen
   Folgen Sie dem Assistenten, um den Installationsprozess abzuschließen.

### Manuelle Installation mit ZIP-Archiv

1. MySQL ZIP-Paket herunterladen
   Besuchen Sie https://dev.mysql.com/downloads/mysql/ um das ZIP-Archiv herunterzuladen.

2. Dateien entpacken
   Entpacken Sie das ZIP-Archiv in das gewünschte Verzeichnis, z.B. C:\mysql.

3. Datenverzeichnis initialisieren
   ```
   mysqld --initialize-insecure
   ```

4. MySQL-Dienst starten
   ```
   mysqld --install MySQL
   net start MySQL
   ```

5. Anmelden und Passwort ändern
   ```
   mysql -u root -p
   ALTER USER 'root'@'localhost' IDENTIFIED BY 'neues_passwort';
   ```

## Installation unter Linux

### Ubuntu/Debian-Systeme

1. Paketindex aktualisieren
   ```bash
   sudo apt update
   ```

2. MySQL Server installieren
   ```bash
   sudo apt install mysql-server
   ```

3. MySQL-Dienst starten
   ```bash
   sudo systemctl start mysql
   sudo systemctl enable mysql
   ```

4. Sichere Konfiguration
   ```bash
   sudo mysql_secure_installation
   ```

### CentOS/RHEL-Systeme

1. MySQL Yum-Repository hinzufügen
   ```bash
   sudo yum install https://dev.mysql.com/get/mysql80-community-release-el8-4.noarch.rpm
   ```

2. MySQL Server installieren
   ```bash
   sudo yum install mysql-community-server
   ```

3. Dienst starten
   ```bash
   sudo systemctl start mysqld
   sudo systemctl enable mysqld
   ```

4. Temporäres Passwort abrufen
   ```bash
   sudo grep 'temporary password' /var/log/mysqld.log
   ```

## Installation unter macOS

### Verwendung von Homebrew

1. Homebrew installieren (falls nicht installiert)
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```

2. MySQL installieren
   ```bash
   brew install mysql
   ```

3. MySQL-Dienst starten
   ```bash
   brew services start mysql
   ```

4. Sichere Konfiguration
   ```bash
   mysql_secure_installation
   ```

### Verwendung des DMG-Pakets

1. DMG-Paket herunterladen
   Besuchen Sie https://dev.mysql.com/downloads/mysql/ um das macOS DMG-Paket herunterzuladen.

2. Installer ausführen
   Doppelklicken Sie auf die PKG-Datei und folgen Sie dem Assistenten, um die Installation abzuschließen.

3. MySQL starten
   Starten Sie den MySQL-Dienst über die Systemeinstellungen.

## MySQL mit Docker installieren

### MySQL-Image abrufen
```bash
docker pull mysql:latest
```

### MySQL-Container ausführen
```bash
docker run -d \
  --name mysql-container \
  -e MYSQL_ROOT_PASSWORD=ihr_passwort \
  -e MYSQL_DATABASE=testdb \
  -p 3306:3306 \
  mysql:latest
```

### Häufige Docker-Parameter
- `-e MYSQL_ROOT_PASSWORD`: Root-Passwort festlegen
- `-e MYSQL_DATABASE`: Erste Datenbank erstellen
- `-p 3306:3306`: Port-Mapping
- `-v /data/mysql:/var/lib/mysql`: Volume für Datenpersistenz

## Überprüfung nach der Installation

### MySQL-Version überprüfen
```bash
mysql --version
```

### Dienststatus überprüfen
```bash
systemctl status mysql   # Linux
brew services list      # macOS
```

### Bei MySQL anmelden
```bash
mysql -u root -p
```

## Häufige Installationsprobleme

### Port-Konflikte
Wenn Port 3306 belegt ist, ändern Sie den Port in der Konfigurationsdatei oder stoppen Sie den Dienst, der diesen Port verwendet.

### Berechtigungsprobleme
Verwenden Sie unter Linux sudo, um Berechtigungen zu erhöhen, und stellen Sie sicher, dass das Datenverzeichnis über korrekte Lese-/Schreibberechtigungen verfügt.

### Firewall-Einstellungen
Stellen Sie sicher, dass die Firewall den Zugriff auf Port 3306 erlaubt.
