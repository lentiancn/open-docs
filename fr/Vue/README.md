# Vue

Vue est un framework JavaScript progressif pour construire des interfaces utilisateur.

## Présentation

Vue est conçu pour être adoptable de manière incrémentale. Il peut être utilisé pour des composants simples ou des applications monopages complexes.

## Documentation

- [Guide d'installation](./1.安装文档.md) - Instructions complètes
- [Guide d'utilisation](./2.使用指南.md) - Guide complet

## Démarrage rapide

```bash
npm create vue@latest
```

## Fonctionnalités

- **Réactif** - Suivi automatique des dépendances
- **Composants** - Blocs UI réutilisables
- **Directives** - Extensions de syntaxe de modèle
- **Transitions** - Système d'animation intégré
