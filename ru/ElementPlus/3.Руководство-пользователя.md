# Element Plus Руководство пользователя

Основные функции и советы.

## Быстрый старт

```vue
<template>
  <el-button type="primary" @click="handleClick">
    Нажми меня
  </el-button>
</template>

<script setup>
import { ref } from 'vue'

const handleClick = () => {
  ElMessage.success('Привет!')
}
</script>
```

## Компоненты

### Button

```vue
<el-button>По умолчанию</el-button>
<el-button type="primary">Основной</el-button>
<el-button disabled>Отключён</el-button>
<el-button loading>Загрузка</el-button>
```

### Input

```vue
<el-input v-model="value" placeholder="Введите текст" />
<el-input v-model="value" disabled placeholder="Отключён" />
<el-input v-model="value" clearable placeholder="Очищаемый" />
```

### Table

```vue
<el-table :data="tableData" stripe border>
  <el-table-column prop="name" label="Имя" />
  <el-table-column prop="email" label="Email" />
</el-table>
```

### Form

```vue
<el-form :model="form" :rules="rules" ref="formRef">
  <el-form-item label="Имя" prop="name">
    <el-input v-model="form.name" />
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="submit">Отправить</el-button>
  </el-form-item>
</el-form>
```

### Dialog

```vue
<el-button @click="visible = true">Открыть</el-button>
<el-dialog v-model="visible" title="Заголовок">
  <span>Содержимое</span>
</el-dialog>
```

### Message

```typescript
import { ElMessage } from 'element-plus'

ElMessage.success('Успех')
ElMessage.error('Ошибка')
```

## Макет

```vue
<el-row :gutter="20">
  <el-col :span="12"><div>12</div></el-col>
  <el-col :span="12"><div>12</div></el-col>
</el-row>
```
