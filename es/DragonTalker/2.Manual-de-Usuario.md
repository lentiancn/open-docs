## 1. Descripción General

DragonTalker es un sistema de generación de cabezas parlantes basado en aprendizaje profundo. Su función principal es combinar una imagen estática de una persona con un archivo de audio para generar videos dinámicos con movimientos de habla. Esta tecnología utiliza algoritmos avanzados de reconstrucción facial 3D y síntesis de imagen para preservar los rasgos faciales originales mientras logra sincronización labial natural y cambios de expresión.

### 1.1 Escenarios de Aplicación

- **Presentadores Virtuales**: Configurar avatares virtuales con voz y expresiones realistas
- **Servicio al Cliente Digital Humano**: Generar videos interactivos antropomórficos
- **Educación y Capacitación**: Presentar contenido del curso en forma de video dinámico
- **Doblaje Cinematográfico**: Generar rápidamente animaciones de doblaje para personajes

## 2. Inicio Rápido

### 2.1 Uso Básico

```bash
source venv/bin/activate
python inference.py \
  --source_image examples/source_image.jpg \
  --driven_audio examples/driven_audio.wav \
  --result_dir ./results
```

### 2.2 Parámetros

| Parámetro | Descripción | Valor por defecto |
|-----------|-------------|-------------------|
| --source_image | Ruta de imagen fuente | Obligatorio |
| --driven_audio | Ruta de audio | Obligatorio |
| --result_dir | Directorio de salida | ./results |
| --enhancer | Algoritmo de mejora | gfpgan |
| --outputfps | FPS del video | 25 |

---

## 3. Especificaciones

### 3.1 Imagen Fuente

- Formato: JPG, PNG
- Resolución: 512×512 o superior recomendada
- Contenido: Rostro frontal, claro, sin obstrucción

### 3.2 Audio

- Formato: WAV, MP3
- Duración: 1-60 segundos
- Calidad: Voz clara

### 3.3 Video de Salida

- Formato: MP4 (H.264)
- Resolución: 256×256 o 512×512
- FPS: 25

## 5. Python API

### Uso Básico

```python
from dragon_talker import DragonTalker

model = DragonTalker(
    device='cuda',
    faceid_weight=0.5,
    enhancer='gfpgan'
)

model.load_source('source.jpg')
model.load_audio('audio.wav')

result_path = model.generate(
    output_path='result.mp4',
    preprocess='full',
    fps=25
)

print(f"Video guardado en: {result_path}")
```

## 6. Preguntas Frecuentes

### 6.1 Calidad de Video

**El rostro generado está borroso**

Solución: Habilitar mejora facial

```bash
python inference.py --enhancer gfpgan ...
```

---

**Labios no sincronizados con audio**

Causa: Audio de baja calidad o habla muy rápida

Solución: Preprocesar audio, aumentar tasa de muestreo

---

### 6.2 Rendimiento

**Memoria GPU insuficiente**

Soluciones:
1. Reducir resolución de salida
2. Desactivar mejora
3. Usar modo CPU
