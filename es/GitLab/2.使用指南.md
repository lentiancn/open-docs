# Guía de Uso de GitLab

## Clonar

```bash
git clone https://gitlab.example.com/usuario/proyecto.git
```

## Operaciones Básicas

```bash
git checkout -b feature/nueva
git add .
git commit -m "Nueva función"
git push origin feature/nueva
```
