# OpenFeign 개요

> Java 선언적 HTTP 클라이언트

---

## 프로젝트 개요

OpenFeign은 선언적 REST 클라이언트 기능을 제공하는 Java 라이브러리입니다. 개발자가 어노테이션을 사용하여 HTTP 서비스 인터페이스를 정의할 수 있도록 하여 HTTP API 호출을 단순화합니다.

---

## 주요 기능

- **선언적 API**: 어노테이션으로 HTTP 인터페이스 정의
- **로드 밸런싱**: 클라이언트 측 로드 밸런싱 지원
- **서비스 발견**: Eureka와의 통합
- **압축**: 효율적인 데이터 전송을 위한 GZIP 지원
- **로깅**: HTTP 요청 및 응답에 대한 상세 로깅
- **오류 처리**: 폴백 메커니즘을 통한 사용자 정의 오류 처리
- **인터셉터**: 사용자 정의 요청/응답 인터셉터 지원

---

## 사용 사례

- 마이크로서비스 통신
- REST API 클라이언트 개발
- 외부 API 통합
- 분산 시스템에서의 서비스 간 호출
