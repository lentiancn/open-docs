# 자주 묻는 질문

이 문서에서는 Bootstrap 사용 과정에서 자주 마주하는 기술적 질문과 답변을 모았습니다.

## 기초적인 질문

### Bootstrap이란 무엇인가요?

Bootstrap은 Twitter사에서 개발하고 관리하는 오픈소스 프론트엔드 개발 프레임워크입니다. 풍부한 CSS 스타일과 JavaScript 컴포넌트를 제공하여 개발자가 빠르게 반응형 웹 애플리케이션을 구축할 수 있도록 도와줍니다.

### Bootstrap 5와 Bootstrap 4의 차이점은 무엇인가요?

주요 차이점:

- Bootstrap 5는 jQuery 의존성을 제거하고 네이티브 JavaScript 사용
- XXL 중단점 추가
- 폼 컨트롤 개선
- Font Awesome 제거, Bootstrap Icons로 변경
- Internet Explorer 지원 중단

### Bootstrap은 어떤 개발 환경이 필요한가요?

모던 브라우저만 있으면 됩니다. Bootstrap 5는 ES6를 지원하는 브라우저가 필요합니다.

## 설치 관련 질문

### CDN 로딩이 실패하면 어떻게 해야 하나요?

1. 네트워크 연결 확인
2. 다른 CDN 시도(예: bootcdn, staticfile)
3. 로컬에 다운로드하여 사용

### npm 설치가 실패하면?

```bash
# 캐시 삭제
npm cache clean --force

# 중국 타오바오 미러 사용
npm config set registry https://registry.npmmirror.com

# 다시 설치
npm install bootstrap@5
```

### 스타일이 적용되지 않아요?

1. CSS 파일이 제대로 불러와졌는지 확인
2. 브라우저 캐시 확인(강제 새로고침 Ctrl+Shift+R)
3. CSS 충돌이 없는지 확인

## 사용 관련 질문

### 그리드 시스템이 작동하지 않아요?

1. row가 container 안에 있는지 확인
2. col이 row 안에 있는지 확인
3. 중단점 클래스를 제대로 사용했는지 확인

### 반응형이 작동하지 않아요?

반드시 뷰포트 메타 태그를 추가했는지 확인:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### 네비게이션 바가 모바일에서 접히지 않아요?

1. bootstrap.bundle.min.js(Popper 포함)가 제대로 불러와졌는지 확인
2. data-bs-toggle과 data-bs-target 속성 확인

### 모달 창이 열리지 않아요?

1. Bootstrap JS가 제대로 불러와졌는지 확인
2. 버튼의 data-bs-toggle과 data-bs-target 속성 확인
3. 모달 ID가 정확한지 확인

### 드롭다운 메뉴가 작동하지 않아요?

1. bootstrap.bundle.min.js 불러오기
2. data-bs-toggle="dropdown" 속성 확인

## 스타일 관련 질문

### 테마 색상을 수정하려면 어떻게 하나요?

Sass 사용:

```scss
$primary: #007bff;
@import "bootstrap/scss/bootstrap";
```

### 기본 스타일을 덮어쓰려면 어떻게 하나요?

더 높은 우선순위 선택자 사용:

```css
.navbar {
    background-color: #f8f9fa !important;
}
```

### 폰트를 수정하려면 어떻게 하나요?

```scss
$font-family-sans-serif: "Helvetica Neue", Helvetica, Arial, sans-serif;
@import "bootstrap/scss/bootstrap";
```

### 아이콘이 표시되지 않아요?

Bootstrap 5는 Bootstrap Icons를 사용합니다:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.10.0/font/bootstrap-icons.css">

<i class="bi bi-alarm"></i>
```

## 컴포넌트 관련 질문

### 카드를 중앙 정렬하려면 어떻게 하나요?

```html
<div class="card">
    <div class="card-body">
        내용
    </div>
</div>
```

flex 사용:

```html
<div class="card">
    <div class="card-body d-flex justify-content-center align-items-center">
        중앙 정렬된 내용
    </div>
</div>
```

### 폼 검증이 작동하지 않아요?

Bootstrap 5는 새로운 폼 검증을 사용합니다:

```html
<form class="row g-3 needs-validation" novalidate>
    <div class="col-md-4">
        <label for="validationCustom01" class="form-label">이름</label>
        <input type="text" class="form-control" id="validationCustom01" required>
        <div class="valid-feedback">올바릅니다!</div>
        <div class="invalid-feedback">이름을 입력해 주세요</div>
    </div>
    <div class="col-12">
        <button class="btn btn-primary" type="submit">제출</button>
    </div>
</form>

<script>
    (function () {
        'use strict'
        var forms = document.querySelectorAll('.needs-validation')
        Array.prototype.slice.call(forms).forEach(function (form) {
            form.addEventListener('submit', function (event) {
                if (!form.checkValidity()) {
                    event.preventDefault()
                    event.stopPropagation()
                }
                form.classList.add('was-validated')
            }, false)
        })
    })()
</script>
```

### 모달 배경 클릭으로 닫게 하려면 어떻게 하나요?

속성 설정:

```html
<div class="modal" data-bs-backdrop="static">
```

`static` - 배경 클릭으로 닫기 않음
`true` - 클릭하면 닫기
`false` - 클릭해도 닫지 않음

## 성능 관련 질문

### 파일 크기를 줄이려면 어떻게 하나요?

1. 필요한 컴포넌트만 불러오기(Sass 사용)
2. 프로덕션 버전 사용
3. Gzip 압축 활성화

### 필요한 스타일만 불러오려면 어떻게 하나요?

불러와야 하는 컴포넌트를 Sass로 지정:

```scss
// 필요한 함수와 변수 불러오기
@import "bootstrap/scss/functions";
@import "bootstrap/scss/variables";

// 필요한 컴포넌트만 불러오기
@import "bootstrap/scss/root";
@import "bootstrap/scss/reboot";
@import "bootstrap/scss/containers";
@import "bootstrap/scss/grid";
@import "bootstrap/scss/buttons";
@import "bootstrap/scss/forms";
```

## 호환성 관련 질문

### IE 브라우저를 지원하지 않나요?

Bootstrap 5는 Internet Explorer를 더 이상 지원하지 않습니다. IE11을 지원해야 하면 Bootstrap 4를 사용하세요.

### 모바일에서 클릭 지연이 있나요?

Bootstrap 5는 touch-action: manipulation을 사용하여 300ms 클릭 지연을 제거했습니다.
