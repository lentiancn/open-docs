# IBM Informix

IBM Informix est une famille de systèmes de gestion de bases de données relationnelles (SGBDR) et d'offres de bases de données multi-modèles développée par IBM.

## Présentation de la Documentation

Ce répertoire contient la documentation technique d'IBM Informix :

- **Introduction** : Aperçu du produit, fonctionnalités et éditions
- **Guide d'installation** : Étapes d'installation pour Linux et Windows
- **Manuel d'utilisation** : Commandes de base, instructions SQL et configuration
- **FAQ** : Questions fréquentes et solutions

## Informations de Version
