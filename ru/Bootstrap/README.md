# Bootstrap

> Популярный CSS-фреймворк для фронтенда

---

## Описание проекта

Bootstrap — это CSS-фреймворк с открытым исходным кодом, разработанный и поддерживаемый Twitter, впервые выпущенный в 2011 году. Bootstrap предоставляет богатые готовые компоненты и адаптивную сетку для быстрого создания современных адаптивных веб-сайтов и веб-приложений.

## Быстрый старт

### Подключение через CDN

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### Создание страницы

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Страница Bootstrap</title>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <div class="container">
    <h1>Привет, Bootstrap!</h1>
  </div>
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```
