# FantasyTalking 安装指南

## 环境要求

- Node.js 14.0.0+
- 现代浏览器

## 安装

### 使用 npm

```bash
npm install fantasy-talking
```

### 使用 yarn

```bash
yarn add fantasy-talking
```

### CDN 引入

```html
<script src="https://unpkg.com/fantasy-talking/dist/fantasy-talking.min.js"></script>
```

## 快速开始

### 初始化

```javascript
import FantasyTalking from 'fantasy-talking'

const ft = new FantasyTalking({
  apiKey: 'your-api-key',
  voice: 'default'
})
```

### 基本使用

```javascript
// 语音合成
ft.speak('你好，这是一个测试')
```

## 配置选项

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| apiKey | string | - | API 密钥 |
| voice | string | default | 语音类型 |
| speed | number | 1.0 | 语速 |
| pitch | number | 1.0 | 音调 |
| volume | number | 1.0 | 音量 |
