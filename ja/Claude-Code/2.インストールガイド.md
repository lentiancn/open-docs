# Claude Code インストールガイド

このガイドでは、各プラットフォームへのClaude Codeのインストール、構成、更新方法について詳しく説明します。

## システム要件

Claude Codeは次のOSと設定をサポートしています：

| コンポーネント | 要件 |
|----------------|------|
| OS | macOS 13.0+、Windows 10 1809+ または Windows Server 2019+、Ubuntu 20.04+、Debian 10+、Alpine Linux 3.19+ |
| メモリ | 4GB以上 |
| ネットワーク | インターネット接続が必要 |
| Shell | Bash、Zsh、PowerShell、またはCMD |
| 地域 | Anthropicがサポートする国と地域 |

### 追加の依存関係

- **ripgrep**：通常Claude Codeに含まれています。検索に失敗した場合はトラブルシューティングを参照してください。

## インストール方法

### 方法1：ネイティブインストール（推奨）

**macOS、Linux、WSL：**

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell：**

```powershell
irm https://claude.ai/install.ps1 | iex
```

**Windows CMD：**

```batch
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
```

**注意：Windowsは先に[Git for Windows](https://git-scm.com/downloads/win)をインストールする必要があります。**

ネイティブインストールは自動的にバックグラウンドで最新バージョンに更新されます。

### 方法2：Homebrew（macOS/Linux）

```bash
brew install --cask claude-code
```

Homebrewインストールは自動更新されません。最新の機能とセキュリティ修正を取得するには定期的に`brew upgrade claude-code`を実行してください。

### 方法3：WinGet（Windows）

```powershell
winget install Anthropic.ClaudeCode
```

WinGetインストールは自動更新されません。定期的に`winget upgrade Anthropic.ClaudeCode`を実行して手動で更新してください。

## インストールの確認

インストール後、Claude Codeが動作していることを確認します：

```bash
claude --version
```

より詳細なインストールと構成の確認を行うには：

```bash
claude doctor
```

## アカウント認証

Claude CodeにはPro、Max、Teams、Enterprise、またはConsoleアカウントが必要です。無料のClaude.aiプランにはClaude Codeアクセス権は含まれていません。

インストール後、`claude`コマンドを実行し、ブラウザのプロンプトに従ってログインします。

## Claude Codeの更新

### 自動更新

ネイティブインストールしたClaude Codeは起動時と実行中に定期的に更新を確認します。更新はバックグラウンドでダウンロードインストールされ、次回Claude Codeを起動時に有効になります。

リリースチャンネル構成で更新タイミングを制御できます：
- `latest`（デフォルト）：新機能をすぐに受信
- `stable`：通常約1週間前のバージョンを使用 majorな問題があるリリースをスキップ

設定ファイルで構成：

```json
{
  "autoUpdatesChannel": "stable"
}
```

### 自動更新の無効化

設定ファイルで環境変数を設定：

```json
{
  "env": {
    "DISABLE_AUTOUPDATER": "1"
  }
}
```

### 手動更新

バックグラウンドチェックを待たずに今すぐ更新を適用：

```bash
claude update
```

HomebrewとWinGetインストールは手動更新が必要です：
- Homebrew：`brew upgrade claude-code`
- WinGet：`winget upgrade Anthropic.ClaudeCode`

## 高度なインストールオプション

### 特定のバージョンをインストール

最新バージョンをインストール：
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

安定バージョンをインストール：
```bash
curl -fsSL https://claude.ai/install.sh | bash -s stable
```

特定のバージョン番号をインストール（例：1.0.58）：
```bash
curl -fsSL https://claude.ai/install.sh | bash -s 1.0.58
```

### npmからネイティブへの移行

以前npmでClaude Codeをインストールしていた場合、ネイティブインストールに移行：

```bash
# ネイティブバイナリをインストール
curl -fsSL https://claude.ai/install.sh | bash

# 古いnpmインストールを削除
npm uninstall -g @anthropic-ai/claude-code
```

### 廃止されたnpmインストール方法

npmインストール方法は廃止されました。npmインストールする必要がある場合は、まずNode.js 18+をインストールする必要があります：

```bash
npm install -g @anthropic-ai/claude-code
```

**注意：`sudo npm install -g`は使用しないでください。権限の問題やセキュリティリスクが発生する可能性があります。**

## Claude Codeのアンインストール

### ネイティブインストールのアンスインストール

**macOS、Linux、WSL：**

```bash
rm -f ~/.local/bin/claude
rm -rf ~/.local/share/claude
```

**Windows PowerShell：**

```powershell
Remove-Item -Path "$env:USERPROFILE\.local\bin\claude.exe" -Force
Remove-Item -Path "$env:USERPROFILE\.local\share\claude" -Recurse -Force
```

### Homebrewのアンスインストール

```bash
brew uninstall --cask claude-code
```

### WinGetのアンスインストール

```powershell
winget uninstall Anthropic.ClaudeCode
```

### npmのアンスインストール

```bash
npm uninstall -g @anthropic-ai/claude-code
```

### 設定ファイルの削除

**警告：設定ファイルを削除すると、すべての設定、許可されたツール、MCPサーバー構成、およびセッション履歴が削除されます。**

**macOS、Linux、WSL：**

```bash
# ユーザー設定と状態を削除
rm -rf ~/.claude
rm ~/.claude.json

# プロジェクト固有の設定を削除（プロジェクトディレクトリから実行）
rm -rf .claude
rm -f .mcp.json
```

**Windows PowerShell：**

```powershell
# ユーザー設定と状態を削除
Remove-Item -Path "$env:USERPROFILE\.claude" -Recurse -Force
Remove-Item -Path "$env:USERPROFILE\.claude.json" -Force

# プロジェクト固有の設定を削除
Remove-Item -Path ".claude" -Recurse -Force
Remove-Item -Path ".mcp.json" -Force
```

## Windowsの特別な構成

Windows上のClaude CodeにはGit for WindowsまたはWSLが必要です。PowerShell、CMD、またはGit Bashから`claude`を起動できます。

Claude CodeがGit Bashのインストール場所を 찾られない場合、設定ファイルでパスを構成：

```json
{
  "env": {
    "CLAUDE_CODE_GIT_BASH_PATH": "C:\\Program Files\\Git\\bin\\bash.exe"
  }
}
```

## Alpine Linuxの構成

Alpineおよび他のmusl/uClibcベースディストリビューションでは、ネイティブインストーラーに`libgcc`、`libstdc++`、`ripgrep`が必要です。パッケージマネージャーでこれらの依存関係をインストールし、`USE_BUILTIN_RIPGREP=0`を設定します。

Alpineで必要なパッケージをインストール：

```bash
apk add libgcc libstdc++ ripgrep
```

次に設定ファイルで構成：

```json
{
  "env": {
    "USE_BUILTIN_RIPGREP": "0"
  }
}
```

## バイナリの整合性検証

SHA256チェックサムを使用してClaude Codeバイナリの整合性を検証できます。チェックサムはリリースマニフェストで公開されています：
`https://storage.googleapis.com/claude-code-dist-86c565f3-f756-42ad-8dfa-d59b1c096819/claude-code-releases/{VERSION}/manifest.json`

署名済みバイナリは次のプラットフォームに配布されます：
- **macOS**：「Anthropic PBC」により署名され、Appleにより公证
- **Windows**：「Anthropic, PBC」により署名
