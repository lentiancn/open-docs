### Fedora

Waydroid kann aus dem offiziellen Paket-Repository installiert werden:

```bash
sudo dnf install waydroid
```

Nach der Installation starten Sie Waydroid aus dem Anwendungsmenü und fahren mit der Initialisierung fort, indem Sie diese URLs in die OTA-Felder einfügen:

System-OTA: `https://ota.waydro.id/system`

Vendor-OTA: `https://ota.waydro.id/vendor`

#### Silverblue/Kinoite/...

Die gleichen Anweisungen gelten für die Fedora Immutable-Varianten, aber Sie sollten `rpm-ostree` anstelle von `dnf` verwenden:

```bash
rpm-ostree install waydroid
```

### Ubuntu/Debian und Derivate

Stellen Sie sicher, dass Sie Wayland Session aktiviert haben (Ubuntu 22.04+)

* Voraussetzungen installieren

```bash
sudo apt install curl ca-certificates -y
```

* Offizielles Repository hinzufügen

```bash
curl -s https://repo.waydro.id | sudo bash
```

Wenn das Skript Ihre Distribution nicht erkennt, können Sie eine gültige Option durch Anhängen von `-s <DISTRO>` angeben. Aktuell unterstützte Werte sind: **mantic**, **focal**, **jammy**, **kinetic**, **lunar**, **noble**, **plucky**, **questing**, **bookworm**, **bullseye**, **trixie**, **sid**

* Waydroid installieren

```bash
sudo apt install waydroid -y
```

Starten Sie dann Waydroid aus dem Anwendungsmenü.

## Fehlerbehebung

### Waydroid manuell starten

Um Waydroid ohne systemctl zu starten, müssen Sie einige einfache Schritte befolgen:

**Starten Sie zuerst den Container:**

```bash
sudo waydroid container start
```

**Und in einem neuen Terminal-Tab starten Sie die Waydroid-Sitzung (ohne ***sudo*****):**

```bash
waydroid session start
```

Nachdem es gestartet ist und Sie "Android with user 0 is ready" sehen, können Sie sicher eine App aus dem Anwendungsmenü starten.

### Waydroid im Vollbildmodus starten

```bash
waydroid show-full-ui
```

### Waydroid im Multi-Fenster-Modus starten

Zuerst müssen wir die Eigenschaft festlegen, während eine Waydroid-Sitzung läuft:

```bash
waydroid prop set persist.waydroid.multi_windows true
```

Danach können wir die Sitzung neu starten:

```bash
waydroid session stop
```

Dann sind wir bereit, eine App zu starten, und sie wird im Multi-Fenster-Modus gestartet.

### Waydroid neu installieren

Wenn Sie eine vollständige Neuinstallation benötigen:

```bash
waydroid session stop
sudo waydroid container stop
sudo apt remove waydroid  # oder entsprechender Distributions-Befehl zum Deinstallieren
# Nach dem Neustart Bereinigung durchführen
sudo rm -rf /var/lib/waydroid /home/.waydroid ~/waydroid ~/.share/waydroid ~/.local/share/applications/*aydroid* ~/.local/share/waydroid
```
