# OpenClaw

OpenClaw — Ваш персональный ИИ-ассистент 🦞

## Обзор

OpenClaw — это персональный ИИ-ассистент, который работает на ваших собственных устройствах. Он общается через каналы, которые вы уже используете, включая WhatsApp, Telegram, Slack, Discord, Google Chat и более 20 других платформ. OpenClaw может выполнять реальную работу, управлять расписанием, отправлять электронные письма, управлять умными устройствами — становится вашим настоящим цифровым партнёром.

## Функции

- **Мультиканальный доступ**: Более 20 коммуникационных платформ
- **Локальное хранение**: Данные хранятся локально, полностью под вашим контролем
- **Работа 24/7**: Доступен в любое время
- **Голосовое взаимодействие**: Поддержка голосовой активации и разговора
- **Система навыков**: Расширяемые возможности автоматизации
- **Открытый исходный код**: Код полностью открыт и бесплатен

## Быстрый старт

### Установка

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

Или с помощью npm:

```bash
npm install -g openclaw@latest
```

### Настройка

```bash
openclaw onboard --install-daemon
```

### Использование

```bash
# Открыть Control UI
openclaw dashboard

# Или отправить сообщение
openclaw agent --message "Hello"
```
