# Kubernetes 설치 가이드

Kubernetes 설치 방법.

---

## 설치 방법

| 방법 | 설명 |
|------|------|
| **minikube** | 로컬 클러스터 |
| **kind** | Docker에서 K8s |
| **kubeadm** | 프로덕션 |

---

## minikube

```bash
brew install minikube
minikube start
```

---

## kubectl

```bash
brew install kubectl
```

---

## 확인

```bash
kubectl get nodes
```
