# Flutter - Обзор

Flutter — инструмент UI с открытым исходным кодом от Google.

## Что такое Flutter?

Flutter — это инструмент UI для создания кроссплатформенных приложений для iOS, Android, Web и Desktop из одной кодовой базы. Использует Dart.

## Функции

- **Кроссплатформенность**: Одна кодовая база
- **Hot Reload**: Мгновенный предпросмотр
- **Богатые виджеты**: Material Design + Cupertino
- **Высокая производительность**: Графический движок Skia
- **Открытый исходный код**: Бесплатно
