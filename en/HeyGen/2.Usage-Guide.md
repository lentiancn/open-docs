# HeyGen Usage Guide

This guide covers how to use HeyGen for AI video generation.

## Getting Started

### Using Web Interface

1. **Login** to HeyGen at https://www.heygen.com
2. **Select Avatar** from the library or upload custom
3. **Choose Input Type**: Text, Audio, or Video
4. **Generate Video** and download

### Using API

```python
from heygen import HeyGen

heygen = HeyGen(api_key="your_api_key")

# Generate video from text
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="Hello, this is my first AI video!",
    voice_id="voice_id"
)

print(f"Video URL: {video.url}")
```

## Core Features

### 1. Avatar Selection

```python
# List available avatars
avatars = heygen.list_avatars()
for avatar in avatars:
    print(f"{avatar.id}: {avatar.name}")

# Get specific avatar
avatar = heygen.get_avatar(avatar_id="avatar_id")
```

### 2. Voice Selection

```python
# List available voices
voices = heygen.list_voices()
for voice in voices:
    print(f"{voice.id}: {voice.name} ({voice.language})")

# Get voice details
voice = heygen.get_voice(voice_id="voice_id")
```

### 3. Video Generation

#### From Text

```python
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="Your script text here",
    voice_id="voice_id",
    background="office"  # Optional: background scene
)
```

#### From Audio

```python
video = heygen.generate_video_from_audio(
    avatar_id="avatar_id",
    audio_url="https://example.com/audio.wav",
)
```

#### From Video (TalkingPhoto)

```python
video = heygen.generate_talking_photo(
    photo_url="https://example.com/photo.jpg",
    script="Your script here",
    voice_id="voice_id"
)
```

### 4. Batch Generation

```python
scripts = [
    "First video script",
    "Second video script",
    "Third video script"
]

# Generate multiple videos
jobs = heygen.batch_generate(
    avatar_id="avatar_id",
    scripts=scripts,
    voice_id="voice_id"
)

# Check job status
for job in jobs:
    status = heygen.get_job_status(job.id)
    print(f"Job {job.id}: {status}")
```

## Advanced Features

### Custom Avatar

```python
# Upload custom avatar
avatar = heygen.upload_avatar(
    name="My Custom Avatar",
    video_url="https://example.com/reference.mp4"
)
print(f"Custom Avatar ID: {avatar.id}")
```

### Instant Voice Clone

```python
# Clone voice from audio sample
voice = heygen.clone_voice(
    name="My Voice Clone",
    audio_url="https://example.com/voice_sample.wav"
)
print(f"Cloned Voice ID: {voice.id}")
```

### Video Editing

```python
# Get video details
video = heygen.get_video(video_id="video_id")

# Update video settings
updated = heygen.update_video(
    video_id="video_id",
    title="My Updated Video",
    description="Updated description"
)

# Delete video
heygen.delete_video(video_id="video_id")
```

## Parameters Reference

### generate_video Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| avatar_id | string | Yes | Avatar ID to use |
| script | string | Yes | Text to speak |
| voice_id | string | Yes | Voice ID |
| background | string | No | Background scene |
| speed | float | No | Speech speed (0.5-2.0) |
| emotion | string | No | Emotion style |
| caption | boolean | No | Enable captions |

### Available Emotions

| Emotion | Description |
|---------|-------------|
| Neutral | Default expression |
| Happy | Happy expression |
| Sad | Sad expression |
| Angry | Angry expression |
| Surprised | Surprised expression |

## Best Practices

### 1. Script Writing

- Keep sentences short and clear
- Use natural language
- Avoid complex terminology

### 2. Avatar Selection

- Match avatar to use case
- Consider audience preferences
- Test multiple avatars

### 3. Voice Selection

- Match voice to brand
- Consider language requirements
- Test with sample scripts

### 4. Video Optimization

- Use appropriate resolution
- Optimize for platform (social media, website)
- Add captions for accessibility

## Troubleshooting

### Video Quality Issues

**Problem:** Video looks pixelated

**Solutions:**
- Use higher resolution setting
- Ensure source image is high quality
- Check internet connection

### Audio Issues

**Problem:** Audio not matching video

**Solutions:**
- Use provided voices
- Ensure audio is properly formatted
- Check lip sync settings

### Processing Time

**Problem:** Video taking too long

**Solutions:**
- Check credit balance
- Use queue for batch jobs
- Consider priority processing
