# Claude Code FAQ

Часто задаваемые вопросы.

## Установка

### Q: Установка не удалась?

**Решения**:
1. Проверьте системные требования
2. Обновите VS Code до последней версии
3. Проверьте сетевое подключение
4. Попробуйте переустановить

### Q: Не могу войти?

**Решения**:
1. Проверьте действительность API-ключа
2. Проверьте сетевое подключение
3. Попробуйте очистить кэш

### Q: Расширение не отображается?

**Решения**:
1. Перезапустите VS Code
2. Проверьте, включено ли расширение
3. Проверьте уведомления VS Code

## Использование

### Q: Медленный ответ?

**Решения**:
1. Проверьте сетевое подключение
2. Уменьшите длину контекста
3. Ждите ответа сервера

### Q: Плохое качество генерации кода?

**Решения**:
1. Будьте более конкретны в описаниях
2. Предоставьте больше контекста
3. Итеративно улучшайте

### Q: Как переключить модель?

Выберите версию модели в настройках.

### Q: Какие языки поддерживаются?

Поддерживает основные языки: Python, JavaScript, TypeScript, Go, Rust, Java, C++ и другие.

## Аккаунт

### Q: Как получить API-ключ?

Посетите https://console.anthropic.com для создания.

### Q: Закончилась квота?

Купите дополнительную квоту в консоли.

## Безопасность

### Q: Код будет утек?

Claude Code не сохраняет и не делится вашим кодом.

### Q: Как защитить приватность?

- Не вводите чувствительную информацию в беседах
- Используйте локальный режим
- Регулярно очищайте сессии
