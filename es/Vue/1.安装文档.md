# Guía de Instalación de Vue

Vue es un framework progresivo de JavaScript para construir interfaces de usuario.

## Descripción de Versiones

Vue tiene dos versiones principales en uso activo:

| Versión | Fecha de Lanzamiento | Estado | Vue.js |
|---------|-------------|--------|--------|
| Vue 3 | Septiembre 2020 | Actual - Recomendado | 3.x |
| Vue 2 | Octubre 2016 | Legado (Fin de vida en 2023) | 2.x |

**Recomendación:** Usa Vue 3 para todos los proyectos nuevos. Vue 2 llegó al fin de su vida útil en 2023.

### Diferencias Clave: Vue 2 vs Vue 3

| Función | Vue 2 | Vue 3 |
|---------|-------|-------|
| Arquitectura | Options API | Composition API + Options API |
| Reactividad | Object.defineProperty | Proxy |
| TypeScript | Soporte parcial | Soporte completo TypeScript |
| Rendimiento | Bueno | Significativamente mejor |
| Tamaño del Bundle | ~20KB comprimido | ~10KB comprimido |
| Ciclo de vida | created, mounted | setup + onMounted |
| Teleport | No | Sí |
| Fragments | No | Sí |
| Múltiples Raíces | No | Sí |

## Requisitos Previos

### Requisitos del Sistema

| Requisito | Vue 2 | Vue 3 |
|-------------|-------|-------|
| Node.js | 8.9+ | 16+ (18+ recomendado) |
| npm | 5.6+ | 8+ |
| Soporte de Navegador | IE9+ | IE11 no soportado |

### Software Requerido

- **Node.js** - Entorno de ejecución JavaScript
- **npm** - Gestor de paquetes (incluido con Node.js)
- **Git** - Control de versiones (opcional pero recomendado)

## Métodos de Instalación

### Método 1: Usando npm (Recomendado)

#### Vue 3 (Recomendado)

```bash
# Crear un nuevo proyecto Vue
npm create vue@latest

# O usando Vue CLI (para compatibilidad con Vue 2)
npm install -g @vue/cli

# Crear proyecto
vue create my-vue-app
```

#### Vue 2 (Legado)

```bash
# Instalar Vue CLI
npm install -g @vue/cli

# Crear proyecto Vue 2
vue create my-vue2-app --default

# O especificar plantilla Vue 2
vue init webpack my-vue2-app
```

### Método 2: Usando Vite (Recomendado para Vue 3)

Vite es una herramienta de compilación moderna que proporciona un servidor de desarrollo extremadamente rápido.

```bash
# Crear proyecto Vue 3 con Vite
npm create vite@latest my-vue-app -- --template vue

# Navegar al proyecto
cd my-vue-app

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```

### Método 3: Usando CDN

#### Vue 3 CDN

```html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Vue 3 CDN Demo</title>
</head>
<body>
  <div id="app">
    <h1>{{ message }}</h1>
    <button @click="reverseMessage">Invertir</button>
  </div>

  <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
  <script>
    const { createApp, ref } = Vue
    
    createApp({
      setup() {
        const message = ref('¡Hola Vue 3!')
        
        const reverseMessage = () => {
          message.value = message.value.split('').reverse().join('')
        }
        
        return { message, reverseMessage }
      }
    }).mount('#app')
  </script>
</body>
</html>
```

#### Vue 2 CDN

```html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Vue 2 CDN Demo</title>
</head>
<body>
  <div id="app">
    <h1>{{ message }}</h1>
    <button @click="reverseMessage">Invertir</button>
  </div>

  <script src="https://unpkg.com/vue@2/dist/vue.js"></script>
  <script>
    new Vue({
      el: '#app',
      data: {
        message: '¡Hola Vue 2!'
      },
      methods: {
        reverseMessage() {
          this.message = this.message.split('').reverse().join('')
        }
      }
    })
  </script>
</body>
</html>
```

## Configuración del Entorno

### Instalar Node.js

#### Windows

1. Descarga Node.js desde https://nodejs.org
2. Ejecuta el instalador (versión LTS recomendada)
3. Verificar: `node --version`

#### macOS

```bash
# Usando Homebrew
brew install node

# O descargar desde nodejs.org
```

#### Linux (Ubuntu/Debian)

```bash
# Usando NodeSource
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# O usando nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 18
```

### Configurar npm

```bash
# Verificar versión de npm
npm --version

# Establecer registry de npm (opcional, para China)
npm config set registry https://registry.npmmirror.com

# Ver configuración
npm config list
```

## Configuración de Vue Router

### Instalación

```bash
npm install vue-router@4
```

### Configuración Básica (Vue 3)

```javascript
// src/router/index.js
import { createRouter, createWebHistory } from 'vue-router'
import Home from '../views/Home.vue'
import About from '../views/About.vue'

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```

```javascript
// src/main.js
import { createApp } from 'vue'
import App from './App.vue'
import router from './router'

createApp(App)
  .use(router)
  .mount('#app')
```

## Pinia (Gestión de Estado)

### Instalación

```bash
npm install pinia
```

### Configuración Básica (Vue 3)

```javascript
// src/stores/counter.js
import { defineStore } from 'pinia'

export const useCounterStore = defineStore('counter', {
  state: () => ({
    count: 0
  }),
  getters: {
    doubleCount: (state) => state.count * 2
  },
  actions: {
    increment() {
      this.count++
    }
  }
})
```

```javascript
// src/main.js
import { createApp } from 'vue'
import { createPinia } from 'pinia'
import App from './App.vue'

const app = createApp(App)
app.use(createPinia())
app.mount('#app')
```

## Vue 2 Options API (Legado)

```javascript
new Vue({
  el: '#app',
  data() {
    return {
      message: '¡Hola Vue 2!',
      items: []
    }
  },
  computed: {
    reversedMessage() {
      return this.message.split('').reverse().join('')
    }
  },
  methods: {
    handleClick() {
      console.log('¡Clickeado!')
    }
  },
  created() {
    console.log('Componente creado')
  },
  mounted() {
    console.log('Componente montado')
  }
})
```

## Vue 3 Composition API

```javascript
import { ref, computed, onMounted } from 'vue'

export default {
  setup() {
    // Estado reactivo
    const message = ref('¡Hola Vue 3!')
    const count = ref(0)
    
    // Computado
    const reversedMessage = computed(() => {
      return message.value.split('').reverse().join('')
    })
    
    // Métodos
    const increment = () => {
      count.value++
    }
    
    // Ciclo de vida
    onMounted(() => {
      console.log('Componente montado')
    })
    
    return {
      message,
      count,
      reversedMessage,
      increment
    }
  }
}
```

## Solución de Problemas

### Problemas Comunes

#### npm install falla

```bash
# Limpiar caché de npm
npm cache clean --force

# Eliminar node_modules y reinstalar
rm -rf node_modules
npm install
```

#### Puerto ya en uso

```bash
# Usar puerto diferente
npm run dev -- --port 3000

# O matar proceso en puerto
fkill -k 8080
```

#### Problemas de versión de Node

```bash
# Verificar versión de Node
node --version

# Usar nvm para cambiar versión
nvm use 18

# O instalar versión específica
nvm install 18
```

#### Problemas de memoria durante compilación

```bash
# Aumentar memoria de Node
NODE_OPTIONS=--max_old_space_size=4096 npm run build
```

## Mejores Prácticas

1. **Usar Vue 3** - A menos que tengas requisitos específicos de Vue 2
2. **Usar Vite** - Desarrollo y compilación más rápidos
3. **Usar Composition API** - Mejor soporte TypeScript y organización de código
4. **Usar TypeScript** - Mejor experiencia de desarrollador
5. **Usar Pinia** - Para gestión de estado (no Vuex para proyectos nuevos)
6. **Usar Vue Router** - Para navegación
