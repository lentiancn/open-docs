# Руководство пользователя Element UI

> Использование компонентов

---

## Кнопки

```html
<el-button>По умолчанию</el-button>
<el-button type="primary">Основная</el-button>
```

## Форма

```html
<el-form :model="form">
  <el-form-item>
    <el-input v-model="form.name"></el-input>
  </el-form-item>
</el-form>
```

## Таблица

```html
<el-table :data="tableData">
  <el-table-column prop="name" label="Имя"></el-table-column>
</el-table>
```

## Диалог

```html
<el-dialog :visible.sync="dialogVisible" title="Заголовок">
  <span>Содержимое</span>
</el-dialog>
```
