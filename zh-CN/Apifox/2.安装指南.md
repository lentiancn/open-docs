# Apifox 安装指南

本指南将帮助你在各种操作系统上安装 Apifox。

## 环境要求

| 软件 | 要求 |
|------|------|
| 操作系统 | Windows/macOS/Linux |
| 内存 | 最低 2GB |
| 磁盘 | 至少 500MB |

## Windows 安装

### 方法一：官方安装包

1. 访问 Apifox 官网：https://apifox.com/downloads/
2. 下载 Windows 版本
3. 运行安装程序
4. 完成后启动 Apifox

### 方法二：便携版

下载便携版，解压后直接运行。

## macOS 安装

### 方法一：DMG 安装包

1. 下载 macOS 版本：https://apifox.com/downloads/
2. 双击 .dmg 文件
3. 拖动到 Applications 文件夹

### 方法二：Homebrew

```bash
brew install --cask apifox
```

## Linux 安装

### 方法一：AppImage

1. 下载 AppImage 版本
2. 添加执行权限
   ```bash
   chmod +x Apifox-*.AppImage
   ```
3. 运行
   ```bash
   ./Apifox-*.AppImage
   ```

### 方法二：DEB/RPM

```bash
# Debian/Ubuntu
sudo dpkg -i apifox_*.deb

# CentOS/RHEL
sudo rpm -i apifox_*.rpm
```

## 初始化配置

### 1. 登录

- 使用账号登录
- 或使用本地模式

### 2. 同步设置

- 开启数据同步
- 配置团队协作

### 3. 环境配置

配置不同环境：
- 开发环境
- 测试环境
- 生产环境

## 常用配置

### 快捷键

```bash
# 发送请求
Ctrl + Enter

# 格式化代码
Ctrl + Shift + F

# 切换环境
Ctrl + E
```

### 代理设置

1. 点击设置
2. 选择网络代理
3. 配置 HTTP/SOCKS 代理

### SSL 证书

1. 关闭 SSL 验证（开发环境）
2. 或导入证书

## 常见问题

### Q1: 无法启动

**解决方案：**
- 检查系统要求
- 重新安装

### Q2: 无法发送请求

**解决方案：**
- 检查网络
- 检查代理设置
- 确认 SSL 证书

### Q3: 数据同步失败

**解决方案：**
- 检查网络
- 重新登录
