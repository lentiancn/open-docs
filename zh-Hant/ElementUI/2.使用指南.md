# Element UI 使用指南

本指南介紹如何在 Vue 2 應用程式中使用 Element UI 元件。

## 基本用法

### Hello World

```vue
<template>
  <div>
    <el-button>預設按鈕</el-button>
    <el-button type="primary">主要按鈕</el-button>
  </div>
</template>

<script>
export default {
  name: 'App'
}
</script>
```

## 元件分類

Element UI 提供以下元件分類：

| 分類 | 元件 |
|------|------|
| 基礎 | Layout, Button, Icon, Link |
| 表單 | Input, Select, Radio, Checkbox, Switch, Slider, TimePicker, DatePicker |
| 資料 | Table, Tree, Pagination, Tag |
| 通知 | Alert, Message, Notification, Progress |
| 導航 | Menu, Tabs, Breadcrumb, Steps, Dropdown |
| 其他 | Dialog, Drawer, Card, Collapse, Timeline |

### 輸入框 Input

```vue
<template>
  <div>
    <!-- 基礎 -->
    <el-input v-model="input" placeholder="輸入文字"></el-input>
    
    <!-- 帶圖示 -->
    <el-input
      v-model="input1"
      placeholder="搜尋"
      prefix-icon="el-icon-search">
    </el-input>
    
    <!-- 密碼 -->
    <el-input
      v-model="input2"
      placeholder="密碼"
      show-password>
    </el-input>
    
    <!-- 文字區域 -->
    <el-input
      v-model="textarea"
      type="textarea"
      :rows="3"
      placeholder="輸入描述">
    </el-input>
    
    <!-- 停用 -->
    <el-input disabled placeholder="停用"></el-input>
  </div>
</template>

<script>
export default {
  data() {
    return {
      input: '',
      input1: '',
      input2: '',
      textarea: ''
    }
  }
}
</script>
```

### 表單 Form

```vue
<template>
  <el-form
    :model="ruleForm"
    :rules="rules"
    ref="ruleForm"
    label-width="120px"
  >
    <el-form-item label="姓名" prop="name">
      <el-input v-model="ruleForm.name"></el-input>
    </el-form-item>
    
    <el-form-item label="電子郵件" prop="email">
      <el-input v-model="ruleForm.email"></el-input>
    </el-form-item>
    
    <el-form-item label="性別" prop="gender">
      <el-radio-group v-model="ruleForm.gender">
        <el-radio label="male">男性</el-radio>
        <el-radio label="female">女性</el-radio>
      </el-radio-group>
    </el-form-item>
    
    <el-form-item label="興趣" prop="hobby">
      <el-checkbox-group v-model="ruleForm.hobby">
        <el-checkbox label="閱讀" name="hobby"></el-checkbox>
        <el-checkbox label="運動" name="hobby"></el-checkbox>
        <el-checkbox label="音樂" name="hobby"></el-checkbox>
      </el-checkbox-group>
    </el-form-item>
    
    <el-form-item label="國家" prop="country">
      <el-select v-model="ruleForm.country" placeholder="選擇國家">
        <el-option label="台灣" value="taiwan"></el-option>
        <el-option label="美國" value="usa"></el-option>
        <el-option label="英國" value="uk"></el-option>
      </el-select>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm('ruleForm')">提交</el-button>
      <el-button @click="resetForm('ruleForm')">重設</el-button>
    </el-form-item>
  </el-form>
</template>

<script>
export default {
  data() {
    return {
      ruleForm: {
        name: '',
        email: '',
        gender: '',
        hobby: [],
        country: ''
      },
      rules: {
        name: [
          { required: true, message: '請輸入姓名', trigger: 'blur' }
        ],
        email: [
          { required: true, message: '請輸入電子郵件', trigger: 'blur' },
          { type: 'email', message: '電子郵件格式不正確', trigger: 'blur' }
        ],
        gender: [
          { required: true, message: '請選擇性別', trigger: 'change' }
        ],
        country: [
          { required: true, message: '請選擇國家', trigger: 'change' }
        ]
      }
    };
  },
  methods: {
    submitForm(formName) {
      this.$refs[formName].validate((valid) => {
        if (valid) {
          alert('提交成功！');
        } else {
          console.log('提交錯誤！！');
          return false;
        }
      });
    },
    resetForm(formName) {
      this.$refs[formName].resetFields();
    }
  }
}
</script>
```

## 事件

```vue
<template>
  <el-button @click="handleClick">點擊我</el-button>
  
  <el-input
    @focus="handleFocus"
    @blur="handleBlur"
    @change="handleChange"
    @input="handleInput">
  </el-input>
</template>

<script>
export default {
  methods: {
    handleClick() {
      console.log('已點擊！')
    },
    handleFocus() {
      console.log('獲得焦點！')
    },
    handleBlur() {
      console.log('失去焦點！')
    },
    handleChange(value) {
      console.log('已變更：', value)
    },
    handleInput(value) {
      console.log('輸入：', value)
    }
  }
}
</script>
```

## 最佳實踐

### 1. 元件區域性使用

```javascript
// 好的做法 - 區域性使用
export default {
  components: {
    [Button.name]: Button,
    [Input.name]: Input
  }
}
```

### 2. 表單驗證

```javascript
const validatePass = (rule, value, callback) => {
  if (value === '') {
    callback(new Error('請輸入密碼'));
  } else {
    if (this.ruleForm.checkPass !== '') {
      this.$refs.ruleForm.validateField('checkPass');
    }
    callback();
  }
};
```

### 3. 表格效能

```vue
  >
</el-table>
```
