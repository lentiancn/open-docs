# Vue Guide d'installation

Vue est un framework JavaScript progressif pour construire des interfaces utilisateur.

## Version

| Version | Date de sortie | Statut |
|---------|---------------|--------|
| Vue 3 | Septembre 2020 | Recommandé |
| Vue 2 | Octobre 2016 | Hérité |

**Recommandation:** Utilisez Vue 3 pour tous les nouveaux projets.

## Prérequis

| Exigence | Vue 2 | Vue 3 |
|----------|--------|-------|
| Node.js | 8.9+ | 16+ (18+ recommandé) |
| npm | 5.6+ | 8+ |

## Méthodes d'installation

### Méthode 1: npm (Recommandé)

```bash
# Créer un nouveau projet Vue
npm create vue@latest
```

### Méthode 2: Vite (Recommandé pour Vue 3)

```bash
npm create vite@latest my-vue-app -- --template vue
cd my-vue-app
npm install
npm run dev
```

### Méthode 3: CDN

```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
```

## Configuration du projet

```bash
npm install vue-router@4
npm install pinia
```

## Serveur de développement

```bash
# Démarrer le serveur
npm run dev

# Construire pour production
npm run build
```

## Résolution des problèmes

### Échec de npm install

```bash
npm cache clean --force
rm -rf node_modules
npm install
```

## Bonnes pratiques

1. **Utiliser Vue 3** - Pour les nouveaux projets
2. **Utiliser Vite** - Développement plus rapide
3. **Utiliser Composition API** - Meilleur support TypeScript
