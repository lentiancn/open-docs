# FantasyTalking FAQ

## Installation

### L'installation échoue?

- Vérifier version Node.js (>= 14.0)
- Vider le cache npm
- Installer depuis les sources

## Utilisation

### Synthèse vocale lente?

- Vérifier connexion réseau
- Utiliser un service auto-hébergé
- Utiliser le traitement par lots

### Obtenir une clé API?

S'inscrire sur le site officiel et créer dans la console.
