# Element Plus Benutzerhandbuch

Dieses Handbuch hilft Ihnen, schnell mit Element Plus zu beginnen.

## Schnellstart

### Grundlegende Verwendung

```vue
<el-button>Standard</el-button>
<el-button type="primary">Primär</el-button>
```

## Häufige Komponenten

### Schaltfläche

```vue
<el-button>Schaltfläche</el-button>
<el-button type="primary">Primär</el-button>
<el-button size="small">Klein</el-button>
```

### Eingabe

```vue
<el-input v-model="input" placeholder="Eingabe" />
```

### Tabelle

```vue
<el-table :data="tableData">
  <el-table-column prop="name" label="Name" />
</el-table>
```

### Formular

```vue
<el-form :model="form">
  <el-form-item label="Name">
    <el-input v-model="form.name" />
  </el-form-item>
</el-form>
```

### Dialog

```vue
<el-dialog v-model="visible" title="Titel">
  <span>Inhalt</span>
</el-dialog>
```

## Theme-Anpassung

```css
:root {
  --el-color-primary: #409eff;
}
```

Weitere Informationen finden Sie auf der [offiziellen Website](https://element-plus.org/).
