# Java インストールガイド

> 環境のセットアップ

---

## 必要環境

| 環境 | 要件 |
|------|------|
| OS | Windows, macOS, Linux |
| RAM | 2GB以上 |

---

## インストール

### Windows

```powershell
choco install openjdk21
```

### macOS

```bash
brew install openjdk@21
```

### Linux

```bash
sudo apt install openjdk-21-jdk
```

---

## 環境変数

```
JAVA_HOME = /usr/lib/jvm/java-21-openjdk
```

---

## 確認

```bash
java -version
```
