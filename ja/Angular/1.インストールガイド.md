# Angular インストールガイド

AngularはGoogleが開発した強力なフロントエンドフレームワークです。本ガイドでは、さまざまな環境のインストール方法について説明します。

## バージョン情報

| バージョン | ステータス | 説明 |
|---------|--------|-------------|
| Angular 18.x | ✅ 現行 | 最新バージョン |
| Angular 17.x | ✅ サポート | 前バージョン |
| Angular 16.x | ✅ サポート | 前バージョン |
| Angular 15.x | ✅ サポート | 旧バージョン |

## システム要件

## インストール方法

### 方法1：Angular CLI（推奨）

#### Angular CLIのインストール

```bash
# グローバルインストール
npm install -g @angular/cli

# インストール確認
ng version
```

#### 新規プロジェクトの作成

```bash
# 新規プロジェクト作成
ng new my-app

# オプション付き作成
ng new my-app --routing --style=scss --skip-tests
```

#### オプション

```bash
# インストールをスキップ
ng new my-app --skip-install

# スタンドアロンコンポーネント（Angular 14+）
ng new my-app --standalone

# SSR有効化
ng new my-app --ssr
```

### 方法2：npxの使用

```bash
# CLIをインストールせずにプロジェクト作成
npx @angular/cli new my-app
```

### 方法3：手動セットアップ

```bash
# ディレクトリ作成
mkdir my-app
cd my-app

# npm初期化
npm init -y

# Angularインストール
npm install @angular/core @angular/common @angular/platform-browser @angular/platform-browser-dynamic @angular/compiler @angular/forms @angular/router @angular/animations rxjs zone.js

# 開発依存関係のインストール
npm install --save-dev @angular/compiler-cli @angular-devkit/build-angular typescript
```

### 方法4：Docker

```dockerfile
# Dockerfile
FROM node:20-alpine

WORKDIR /app

# Angular CLIインストール
RUN npm install -g @angular/cli

# プロジェクト作成
RUN ng new my-app --directory .

# 開発サーバー起動
CMD ["ng", "serve", "--host", "0.0.0.0"]
```

## 環境設定

### Node.jsバージョン管理

#### nvmの使用（推奨）

```bash
# nvmインストール
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Node.jsインストール
nvm install 20
nvm use 20

# デフォルト設定
nvm alias default 20
```

### npm設定

```bash
# レジストリ設定
npm config set registry https://registry.npmjs.org/

# 厳格SSL有効化
npm config set strict-ssl true

# キャッシュクリア
npm cache clean --force
```

## 確認

```bash
# Angular CLI確認
ng version

# Node.js確認
node --version

# npm確認
npm --version
```

## よくある問題

### 権限エラー

```bash
# npm権限を修復
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
export PATH=~/.npm-global/bin:$PATH
```

### インストールが遅い

```bash
# npm ciで高速インストール
npm ci

# またはyarn使用
npm install -g yarn
```

## Angularのアップデート

```bash
# Angular CLIアップデート
npm install -g @angular/cli@latest

# プロジェクトアップデート
ng update @angular/core @angular/cli
```
