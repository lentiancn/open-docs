# FAQ OpenClaw

Cette page compile les questions et réponses les plus courantes sur l'utilisation d'OpenClaw.

---

## Questions de base

### Qu'est-ce qu'OpenClaw ?

OpenClaw est une passerelle d'assistant IA auto-hébergée qui connecte plusieurs plateformes de chat comme WhatsApp, Telegram, Discord et iMessage à un assistant de codage IA (comme Pi).

### OpenClaw est-il gratuit ?

OpenClaw lui-même est open source et gratuit (licence MIT). Cependant, l'utilisation des modèles IA peut nécessiter un paiement.

### Quel type d'appareil faut-il pour exécuter OpenClaw ?

Configuration minimale :
- Node.js 22+
- 2 Go de RAM
- 500 Mo d'espace disque

Recommandé :
- 4 Go+ de RAM
- Connexion réseau stable
- Ordinateur ou serveur toujours allumé

## Questions d'installation

### L'installation a échoué — que faire ?

1. Vérifiez la version de Node.js :
   ```bash
   node --version
   ```

2. Utilisez le script d'installation officiel :
   ```bash
   curl -fsSL https://openclaw.ai/install.sh | bash
   ```

3. Exécutez le diagnostic :
   ```bash
   openclaw doctor
   ```

### Problèmes sous Windows ?

**WSL2 est fortement recommandé.** OpenClaw fonctionne mieux dans un environnement Linux.

### Comment mettre à jour OpenClaw ?

```bash
npm update -g openclaw@latest
openclaw doctor
```

## Questions de configuration

### Où est le fichier de configuration ?

Emplacement par défaut : `~/.openclaw/openclaw.json`

### Comment configurer plusieurs canaux ?

```json5
{
  channels: {
    whatsapp: { enabled: true },
    telegram: { enabled: true, botToken: "xxx" },
    discord: { enabled: true, token: "xxx" },
  },
}
```

### La configuration ne s'applique pas ?

1. Vérifiez le format du fichier (JSON5)
2. Redémarrez le Gateway :
   ```bash
   openclaw gateway restart
   ```

## Questions de connexion

### Le Gateway ne démarre pas ?

Raisons possibles :
- Le port 18789 est déjà utilisé
- Erreur de format de fichier de configuration
- Permissions insuffisantes

Solution :
```bash
lsof -i :18789
openclaw doctor
```

### La connexion au canal a échoué ?

1. Vérifiez si les identifiants sont valides
2. Vérifiez le statut du canal :
   ```bash
   openclaw channels status --probe
   ```
3. Consultez les logs :
   ```bash
   openclaw logs --follow
   ```

## Questions d'utilisation

### L'IA ne répond pas aux messages ?

Vérifiez :
1. Le canal est-il connecté ?
2. Êtes-vous sur la liste blanche ?
3. Dans les groupes, la mention @ est-elle requise ?
4. Consultez les logs pour les messages d'erreur

### Comment réinitialiser une session ?

```bash
openclaw session reset
```

### Comment changer le modèle IA ?

```json5
{
  agents: {
    defaults: {
      model: {
        primary: "anthropic/claude-sonnet-4-5",
      },
    },
  },
}
```

## Questions de sécurité

### OpenClaw est-il sûr ?

OpenClaw privilégie la sécurité :
- Toutes les données stockées localement
- Identifiants chiffrés
- Prend en charge les mécanismes de liste blanche et de couplage
- Peut activer l'isolation sandbox

Recommandations :
- Utilisez la liste blanche pour contrôler l'accès
- Mettez à jour régulièrement les versions
- Exécutez `openclaw doctor` pour vérifier la sécurité

### Qu'est-ce que le couplage (Pairing) ?

Le couplage est un mécanisme de sécurité où les utilisateurs inconnus sont bloqués lors de leur premier message. Ils ont besoin de votre approbation pour utiliser l'IA.

Approuver un utilisateur :
```bash
openclaw pairing approve --channel whatsapp --sender +15555550123
```

## Questions de performance

### Ça fonctionne lentement ?

1. Vérifiez les ressources système :
   ```bash
   openclaw status
   ```

2. Compactez les sessions :
   ```bash
   /compact
   ```

3. Nettoyez les anciens logs :
   ```bash
   openclaw logs clean
   ```

```bash

pnpm build
pnpm openclaw gateway
```
