# Flutter - 설치 가이드

완전한 Flutter 개발 환경 설정 가이드.

## 요구 사항

### Windows

- **OS**: Windows 10 이상 (64비트)
- **디스크 공간**: 최소 2.8GB
- **도구**: Windows PowerShell 5.0+, Git for Windows

### macOS

- **OS**: macOS 10.14 (Mojave) 이상
- **디스크 공간**: 최소 2.8GB
- **도구**: Command Line Tools, Git

### Linux

- **OS**: Ubuntu 18.04 이상
- **디스크 공간**: 최소 2.8GB
- **종속성**: bash, curl, git, unzip, xz-utils, zip, libglu1-mesa

## 설치 단계

### Windows 설치

1. **Flutter SDK 다운로드**

최신 버전을 다운로드하려면 Flutter 웹사이트를 방문하세요:
```bash
# 또는 git으로 클론
git clone https://github.com/flutter/flutter.git -b stable
```

2. **환경 변수 구성**

시스템 PATH에 Flutter 추가:
```powershell
# PowerShell에서 실행
[Environment]::SetEnvironmentVariable(
    "Path",
    $env:Path + ";C:\src\flutter\bin",
    "User"
)
```

3. **설치 확인**

```bash
flutter doctor
```

### macOS 설치

1. **Flutter SDK 다운로드**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

2. **환경 변수 구성**

```bash
# ~/.bash_profile 또는 ~/.zshrc에 추가
export PATH="$PATH:/Users/사용자이름/flutter/bin"
```

3. **flutter doctor 실행**

```bash
flutter doctor
```

### Linux 설치

1. **종속성 설치**

```bash
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa
```

2. **Flutter SDK 다운로드**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

3. **PATH 구성**

```bash
export PATH="$PATH:/home/사용자이름/flutter/bin"
```

## Android 개발 환경 설정

### Android Studio 설치

1. Android Studio 다운로드: https://developer.android.com/studio
2. 설치 및 실행
3. SDK 구성:

```bash
flutter config --android-sdk /path/to/android/sdk
```

### 라이선스 수락

```bash
flutter doctor --android-licenses
```

### 에뮬레이터 구성

```bash
# 에뮬레이터 생성
flutter emulator --create --name my-emulator

# 에뮬레이터 시작
flutter emulator --launch my-emulator
```

## 일반적인 문제

### 다운로드 실패?

미러 사용:
```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
export PUB_CACHE=/path/to/cache
```

### Android SDK를 찾을 수 없음?

```bash
flutter config --android-sdk /path/to/android-sdk

# 또는 환경 변수 설정
export ANDROID_SDK_ROOT=/path/to/android-sdk
```

### 명령을 찾을 수 없음?

Flutter가 PATH에 있는지 확인:
```bash
echo $PATH
# flutter/bin이 포함되어야 함
```

## 제거

```bash
# Flutter 디렉토리 삭제
rm -rf /path/to/flutter

# PATH에서 제거
```
