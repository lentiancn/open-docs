# Angular Overview

Modern Web Application Development Framework.

## What is Angular?

Angular is an open-source web application framework developed and maintained by Google. As one of the preferred solutions for modern web application development, Angular provides a complete toolchain from development to deployment on the TypeScript platform.

### Core Features

- **Native TypeScript**: Built entirely with TypeScript for type safety
- **Component-Based Architecture**: Component-based development model for easy maintenance and testing
- **Dependency Injection**: Powerful DI system for better testability
- **Directive System**: Powerful directives to extend HTML capabilities
- **Routing**: Complete client-side routing solution
- **Forms**: Reactive forms and template-driven forms
- **HTTP Client**: Complete HTTP client service
- **Internationalization**: Built-in i18n support
- **Animation**: Powerful animation API
- **Testing**: Complete unit and E2E testing support

### Technical Architecture

Angular uses a modular architecture with core concepts:

- **Module**: Basic unit for organizing application code
- **Component**: Combination of view and business logic
- **Service**: Encapsulated reusable business logic
- **Directive**: Extends HTML element behavior
- **Pipe**: Data transformation utility

- **Enterprise Applications**: Large complex business systems
- **Single Page Applications (SPA)**: Apps requiring smooth user experience
