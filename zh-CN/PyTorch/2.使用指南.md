PyTorch 基本操作。

## 基础

```python
import torch

# 张量
x = torch.randn(3, 3)
print(x)

# GPU
if torch.cuda.is_available():
    x = x.cuda()
```

## 神经网络

```python
import torch.nn as nn

model = nn.Sequential(
    nn.Linear(784, 256),
    nn.ReLU(),
    nn.Linear(256, 10)
)

# 训练
import torch.optim as optim
optimizer = optim.Adam(model.parameters(), lr=0.001)
```
