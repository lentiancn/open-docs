# Guide d'installation PlantUML

PlantUML est un outil open source pour créer rapidement des diagrammes UML. Ce guide couvre les méthodes d'installation dans divers environnements.

## Informations de version

| Version | Statut | Description |
|------|------|------|
| PlantUML | ✅ Actuelle | Dernière version stable |

## Méthodes d'installation

### Méthode un : Installation sous Windows

#### 1. Télécharger le fichier JAR

1. Téléchargez PlantUML JAR : https://sourceforge.net/projects/plantuml/files/plantuml.jar/download
2. Enregistrez dans un répertoire local

#### 2. Installer Graphviz (Optionnel mais recommandé)

```powershell
# Utiliser Chocolatey
choco install graphviz
```

### Méthode deux : Installation sous macOS

```bash
# Utiliser Homebrew
brew install plantuml

# Ou installer Graphviz
brew install graphviz
```

### Méthode trois : Installation sous Linux

```bash
# Ubuntu/Debian
sudo apt-get install -y plantuml graphviz

# CentOS/RHEL
sudo yum install -y plantuml graphviz
```

### Méthode quatre : En tant que plugin VS Code

1. Ouvrir VS Code
2. Rechercher "PlantUML"
3. Installer le plugin "PlantUML"
4. Installer Graphviz

### Méthode cinq : En tant que plugin IntelliJ

1. Ouvrir IntelliJ IDEA
2. Settings → Plugins
3. Rechercher "PlantUML"
4. Installer le plugin

### Méthode six : Utiliser Docker

```bash
# Exécuter le serveur PlantUML
docker run -d -p 8080:8080 plantuml/plantuml-server:jetty

# Accéder à http://localhost:8080
```

## Configuration de l'environnement

### Exigences Java

```bash
# Vérifier Java
java -version

# Java 8 ou supérieur requis
```

### Configuration Graphviz

```bash
# Définir la variable d'environnement (Linux/macOS)
export GRAPHVIZ_DOT=/usr/bin/dot

# Windows ajouter au PATH
# C:\Program Files\Graphviz\bin
```

## Questions fréquentes

### Les images ne s'affichent pas ?

Assurez-vous que Graphviz est installé :

```bash
dot -V
```

### Les caractères chinois s'affichent mal ?

Ajoutez au diagramme :

```plantuml
skinparam defaultFontName SimSun
```
