# Vue.js 安裝指南

## 快速體驗

- **在線演練場**：直接訪問 [Vue 演練場](https://play.vuejs.org) 快速體驗
- **JSFiddle**：使用 [JSFiddle 入門模板](https://jsfiddle.net/yyx990803/2ke1ab0z/)
- **StackBlitz**：在瀏覽器中打開 [StackBlitz](https://vite.new/vue) 嘗試完整構建設置

## 創建 Vue 應用（推薦方式）

### 前提條件
- 熟悉命令行操作
- 已安裝 Node.js 版本 ^20.19.0 或 >=22.12.0

### 使用官方腳手架
在命令行中運行以下命令之一：

```bash
# npm
npm create vue@latest

# pnpm  
pnpm create vue@latest

# yarn (v1+)
yarn create vue

# yarn (Modern v2+)
yarn create vue@latest

# yarn (^v4.11)
yarn dlx create-vue@latest

# bun
bun create vue@latest
```

### 項目配置選項
創建項目時會提示以下可選功能：
- Project name: 項目名稱
- Add TypeScript?: 是否添加 TypeScript 支持
- Add JSX Support?: 是否添加 JSX 支持
- Add Vue Router: 是否添加 Vue Router（單頁應用開發）
- Add Pinia: 是否添加 Pinia（狀態管理）
- Add Vitest: 是否添加 Vitest（單元測試）
- Add End-to-End Testing: 是否添加端到端測試（Cypress/Nightwatch/Playwright）
- Add ESLint: 是否添加 ESLint（代碼質量）
- Add Prettier: 是否添加 Prettier（代碼格式化）
- Add Vue DevTools 7: 是否添加 Vue DevTools 7 擴展（調試，實驗性）

### 啟動開發服務器
```bash
cd <your-project-name>
npm install
npm run dev
```

### 生產環境構建
```bash
npm run build
```

## 通過 CDN 使用 Vue

### 全局構建版本
```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<div id="app">{{ message }}</div>

<script>
const { createApp } = Vue

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

### ES 模塊構建版本
```html
<div id="app">{{ message }}</div>

<script type="module">
import { createApp } from 'https://unpkg.com/vue@3/dist/vue.esm-browser.js'

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

### 啟用 Import Maps
```html
<script type="importmap">
{
  "imports": {
    "vue": "https://unpkg.com/vue@3/dist/vue.esm-browser.js"
  }
}
</script>

<div id="app">{{ message }}</div>

<script type="module">
import { createApp } from 'vue'

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

## 開發環境推薦

- **IDE**: Visual Studio Code + [Vue - Official 擴展](https://marketplace.visualstudio.com/items?itemName=Vue.volar)
- **構建工具**: Vite（已包含在官方腳手架中）
- **本地服務器**: 如需使用 ES 模塊，可使用 `npx serve` 啟動本地 HTTP 服務器
