# Flutter 安装指南

> 当前版本：Flutter 3.41.2

---

## 系统要求

### Windows

| 项目 | 要求 |
|------|------|
| 系统 | Windows 10 (64-bit) |
| 磁盘 | 2.8 GB |
| 内存 | 4 GB |

### macOS

| 项目 | 要求 |
|------|------|
| 系统 | macOS 10.14+ |
| 磁盘 | 2.8 GB |
| Xcode | 需要安装 |

### Linux

| 项目 | 要求 |
|------|------|
| 系统 | Ubuntu 18.04+ |
| 工具 | GCC、CMake、Ninja |

---

## Windows 安装

### 1. 下载 SDK

访问 https://docs.flutter.dev/get-started/install/windows

或使用 Git：

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

### 2. 配置 PATH

```
我的电脑 → 属性 → 高级 → 环境变量
```

添加 Flutter bin 目录到 PATH，重启终端。

### 3. 运行检查

```bash
flutter doctor
```

### 4. Android 环境

安装 Android Studio 或手动配置 Android SDK：

```bash
flutter doctor --android-licenses
```

---

## macOS 安装

### 1. 安装 Xcode

App Store 免费安装 Xcode

### 2. 安装 Flutter

**方式一：Homebrew（推荐）**

```bash
brew update
brew install flutter
```

**方式二：直接下载**

1. 下载 SDK
2. 解压到 `~/flutter`
3. 添加到 PATH：

```bash
# ~/.bash_profile 或 ~/.zshrc
export PATH="$PATH:$HOME/flutter/bin"
source ~/.bashrc
```

### 3. 验证

```bash
flutter doctor
```

---

## Linux 安装

### 1. 安装依赖

```bash
sudo apt-get update
sudo apt-get install -y \
    curl git unzip xz-utils \
    libglu1-mesa clang cmake ninja-build \
    pkg-config libgtk-3-dev
```

### 2. 安装 SDK

```bash
cd /opt
sudo git clone https://github.com/flutter/flutter.git -b stable

# 添加到 PATH
echo 'export PATH="$PATH:/opt/flutter/bin"' >> ~/.bashrc
source ~/.bashrc
```

## 验证安装

```bash
# 查看版本
flutter --version

# 检查环境
flutter doctor

# 创建项目
flutter create my_app
cd my_app
flutter run
```

## 升级

```bash
# 升级到最新
flutter upgrade

# 切换通道
flutter channel stable   # 稳定版
flutter channel beta    # 测试版
```
