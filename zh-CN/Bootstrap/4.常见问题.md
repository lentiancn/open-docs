# 常见问题

本文档汇总了 Bootstrap 使用过程中常见的技术问题和解答。

## 基础问题

### 什么是 Bootstrap？

Bootstrap 是一个开源的前端开发框架，由 Twitter 公司开发维护。它提供了丰富的 CSS 样式和 JavaScript 组件，帮助开发者快速构建响应式 Web 应用。

### Bootstrap 5 和 Bootstrap 4 有什么区别？

主要区别：

- Bootstrap 5 移除了 jQuery 依赖，使用原生 JavaScript
- 增加了 XXL 断点
- 改进了表单控件
- 移除了 Font Awesome，改为 Bootstrap Icons
- 不再支持 Internet Explorer

### Bootstrap 需要什么环境？

只需要现代浏览器即可。Bootstrap 5 需要支持 ES6 的浏览器。

## 安装问题

### CDN 加载失败怎么办？

1. 检查网络连接
2. 尝试其他 CDN（如 bootcdn、staticfile）
3. 下载本地使用

### npm 安装失败？

```bash
# 清除缓存
npm cache clean --force

# 使用淘宝镜像
npm config set registry https://registry.npmmirror.com

# 重新安装
npm install bootstrap@5
```

### 样式不生效？

1. 确认 CSS 文件正确引入
2. 检查浏览器缓存（强制刷新 Ctrl+Shift+R）
3. 确认没有 CSS 冲突

## 使用问题

### 网格系统不工作？

1. 确保 row 在 container 内
2. 确保 col 在 row 内
3. 检查是否正确使用断点类

### 响应式不生效？

确保添加了视口标签：

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### 导航栏在移动端不折叠？

1. 确保引入了 bootstrap.bundle.min.js（包含 Popper）
2. 检查 data-bs-toggle 和 data-bs-target 属性

### 模态框不弹出？

1. 确保引入了 Bootstrap JS
2. 检查按钮的 data-bs-toggle 和 data-bs-target 属性
3. 确认模态框 ID 正确

### 下拉菜单不工作？

1. 引入 bootstrap.bundle.min.js
2. 检查 data-bs-toggle="dropdown" 属性

## 样式问题

### 如何修改主题颜色？

使用 Sass：

```scss
$primary: #007bff;
@import "bootstrap/scss/bootstrap";
```

### 如何覆盖默认样式？

使用更高优先级的选择器：

```css
.navbar {
    background-color: #f8f9fa !important;
}
```

### 字体如何修改？

```scss
$font-family-sans-serif: "Helvetica Neue", Helvetica, Arial, sans-serif;
@import "bootstrap/scss/bootstrap";
```

### 图标不显示？

Bootstrap 5 使用 Bootstrap Icons：

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.10.0/font/bootstrap-icons.css">

<i class="bi bi-alarm"></i>
```

## 组件问题

### 卡片如何居中？

```html
<div class="card">
    <div class="card-body">
        内容
    </div>
</div>
```

使用 flex：

```html
<div class="card">
    <div class="card-body d-flex justify-content-center align-items-center">
        居中内容
    </div>
</div>
```

### 表单验证不工作？

Bootstrap 5 使用新的表单验证：

```html
<form class="row g-3 needs-validation" novalidate>
    <div class="col-md-4">
        <label for="validationCustom01" class="form-label">名字</label>
        <input type="text" class="form-control" id="validationCustom01" required>
        <div class="valid-feedback">正确！</div>
        <div class="invalid-feedback">请输入名字</div>
    </div>
    <div class="col-12">
        <button class="btn btn-primary" type="submit">提交</button>
    </div>
</form>

<script>
    (function () {
        'use strict'
        var forms = document.querySelectorAll('.needs-validation')
        Array.prototype.slice.call(forms).forEach(function (form) {
            form.addEventListener('submit', function (event) {
                if (!form.checkValidity()) {
                    event.preventDefault()
                    event.stopPropagation()
                }
                form.classList.add('was-validated')
            }, false)
        })
    })()
</script>
```

### 模态框背景可点击如何关闭？

设置属性：

```html
<div class="modal" data-bs-backdrop="static">
```

`static` - 点击背景不关闭
`true` - 点击关闭
`false` - 点击不关闭

## 性能问题

### 如何减小文件体积？

1. 按需引入组件（使用 Sass）
2. 使用生产版本
3. 启用 Gzip 压缩

### 如何按需引入样式？

使用 Sass 引入需要的组件：

```scss
// 引入必要的函数和变量
@import "bootstrap/scss/functions";
@import "bootstrap/scss/variables";

// 只引入需要的组件
@import "bootstrap/scss/root";
@import "bootstrap/scss/reboot";
@import "bootstrap/scss/containers";
@import "bootstrap/scss/grid";
@import "bootstrap/scss/buttons";
@import "bootstrap/scss/forms";
```

## 兼容性问题

### 不支持 IE 浏览器？

Bootstrap 5 不再支持 Internet Explorer。如需支持 IE11，使用 Bootstrap 4。

### 移动端点击延迟？

Bootstrap 5 使用 touch-action: manipulation 消除了 300ms 点击延迟。
