# Apifox

API 文档、调试、Mock、测试一体化平台

## 概述

Apifox 是 API 文档、API 调试、API Mock、API 自动化测试一体化协作平台。由腾讯公司推出，帮助团队提升 API 开发效率。

## 核心功能

- **API 文档**：自动生成、在线分享
- **API 调试**：可视化调试
- **Mock**：内置 Mock、自定义规则
- **自动化测试**：批量测试、报告
