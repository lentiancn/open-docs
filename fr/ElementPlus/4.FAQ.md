# FAQ Element Plus

Cette page liste les problèmes courants et leurs solutions.

## Problèmes d'installation

### Échec d'installation

Vérifiez la version de Node.js (>= 14.0.0)
Videz le cache npm: npm cache clean --force

### Erreurs TypeScript

Ajoutez dans tsconfig.json:

{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}

## Problèmes d'utilisation

### Styles non appliqués

Importez le CSS:

import 'element-plus/dist/index.css'

### Icônes non affichées

Installez la bibliothèque d'icônes:

npm install @element-plus/icons-vue

Puis enregistrez globalement:

import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}

## Problèmes de performances

Pour grandes quantités de données:
- Utilisez la pagination
- Utilisez el-table-v2 (table virtuelle)

## Problèmes de version Vue

Element Plus supporte uniquement Vue 3. Pour Vue 2, utilisez Element UI.
