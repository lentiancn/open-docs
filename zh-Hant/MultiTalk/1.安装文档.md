# MultiTalk 安裝指南

MultiTalk 是一款多說話人音訊生成工具，可建立自然流暢的多人語音內容。

## 版本資訊

| 版本 | 狀態 | 說明 |
|---------|--------|-------------|
| MultiTalk v1.0 | ✅ 目前版本 | 初始版本發布 |

## 系統需求

### 硬體需求

| 需求 | 最低配置 | 推薦配置 |
|-------------|---------|-------------|
| GPU | NVIDIA 4GB 記憶體 | NVIDIA 8GB+ 記憶體 |
| RAM | 8GB | 16GB+ |
| 儲存空間 | 5GB | 10GB+ SSD |

### 軟體需求

| 軟體 | 版本 |
|----------|---------|
| Python | 3.8+ |
| CUDA | 11.0+ |

## 安裝方法

### 方法一：從原始碼安裝

#### 1. 複製儲存庫

```bash
git clone https://github.com/MultiTalk/MultiTalk.git
cd MultiTalk
```

#### 2. 建立虛擬環境

```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
# 或
venv\Scripts\activate  # Windows
```

#### 3. 安裝依賴

```bash
pip install -r requirements.txt
```

#### 4. 下載模型

```bash
bash scripts/download_models.sh
```

### 方法二：使用 Docker

```bash
docker pull multitalk/multitalk:latest
docker run --gpus all -v /path/to/data:/data multitalk/multitalk:latest
```

## 設定

### 環境變數

```bash
export CUDA_VISIBLE_DEVICES=0
export PYTHONPATH=/path/to/MultiTalk:$PYTHONPATH
```

### 模型路徑

編輯 `config.yaml`:

```yaml
model:
  checkpoint_path: "checkpoints/multitalk.pth"
```

## 快速開始

### 生成多人語音

```bash
python inference.py \
  --text "你好，這是一個多說話人示範。" \
  --speakers speaker1,speaker2 \
  --output output.wav
```

### 參數說明

```bash
--text         # 輸入文字
--speakers     # 說話人ID（用逗號分隔）
--output       # 輸出檔案路徑
--speed        # 語音速度（預設：1.0）
```

## 常見問題

### GPU 不可用？

```bash
nvidia-smi

```

### 模型下載失敗？

檢查網路連線或手動下載模型。
