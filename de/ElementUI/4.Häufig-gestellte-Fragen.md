# Element UI FAQ

## Installationsprobleme

Stellen Sie sicher, dass Node.js >= 8.0.0

npm cache clean --force

## Nutzungsprobleme

### Stile werden nicht angewendet

CSS importieren:

import 'element-ui/lib/theme-chalk/index.css'

### Symbole werden nicht angezeigt

Symbol-Schriftarten importieren:

import 'element-ui/lib/theme-chalk/fonts/element-icons.ttf'

## Vue-Versionsprobleme

Element UI unterstützt nur Vue 2.
