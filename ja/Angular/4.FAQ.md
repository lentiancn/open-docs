# Angular FAQ

> よくある質問

---

## インストール

### CLI インストール失敗？

```bash
npm cache clean --force
npm install -g @angular/cli
```

### ng コマンドが見つからない？

```bash
npm install -g @angular/cli
```

---

## 開発

### プロジェクトが起動しない？

```bash
rm -rf node_modules
npm install
```
