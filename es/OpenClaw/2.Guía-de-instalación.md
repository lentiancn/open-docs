# Guía de instalación de OpenClaw

## Requisitos

- Node.js 22+
- macOS, Linux o Windows
- Docker (opcional, para sandbox)

## Instalación

### macOS/Linux

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

### Windows (PowerShell)

```powershell
iwr -useb https://openclaw.ai/install.ps1 | iex
```

### npm directo

```bash
npm install -g openclaw@latest
```

## Post-instalación

```bash
openclaw onboard --install-daemon
openclaw gateway status
openclaw dashboard
```

Abre http://127.0.0.1:18789/

## Verificar

```bash
openclaw --version
openclaw doctor
```
