# Mockito 설치 가이드

Mockito 설치 및 설정 가이드.

## 전제 조건

- Java Development Kit (JDK) 8 이상
- Maven 또는 Gradle (권장)

## 설치

### Maven

`pom.xml`에 다음 종속성 추가:

```xml
<dependency>
    <groupId>org.mockito</groupId>
    <artifactId>mockito-core</artifactId>
    <version>5.8.0</version>
    <scope>test</scope>
</dependency>
```

### Gradle

`build.gradle`에 종속성 추가:

```groovy
testImplementation 'org.mockito:mockito-core:5.8.0'
```

## 설정

### JUnit 5용 Mockito 확장

```java
import org.junit.jupiter.api.extension.ExtendWith;
import org.mockito.junit.jupiter.MockitoExtension;

@ExtendWith(MockitoExtension.class)
class MyTest {
    // 테스트 코드
}
```

## 문제 해결

### 문제: NoClassDefFoundError

**해결:** 모든 종속성이 포함되었는지 확인하세요.

### 문제: 모크가 작동하지 않음

**해결:**
- 클래스가 final이 아닌지 확인
- final 클래스의 경우 `mockito-inline` 사용
