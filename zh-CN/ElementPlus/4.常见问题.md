# Element Plus 常见问题

本页面列出 Element Plus 使用过程中最常见的问题及解决方案。

## 安装问题

### 安装失败

**症状**：npm install 或 yarn add 失败

**解决方案**：

1. 检查 Node.js 版本，确保 >= 14.0.0
2. 清除 npm 缓存：
   ```bash
   npm cache clean --force
   ```
3. 删除 node_modules 和 package-lock.json 后重新安装

### TypeScript 类型错误

**症状**：引入组件时提示类型错误

**解决方案**：

1. 确保安装了 Volar 插件
2. 在 tsconfig.json 中添加：
   ```json
   {
     "compilerOptions": {
       "types": ["element-plus/global"]
     }
   }
   ```

## 使用问题

### 组件样式不生效

**症状**：组件样式与预期不符

**解决方案**：

1. 确保已正确引入 CSS：
   ```javascript
   import 'element-plus/dist/index.css'
   ```
2. 检查是否有全局样式覆盖

### 图标不显示

**症状**：图标组件显示为空白或方块

**解决方案**：

1. 安装图标库：
   ```bash
   npm install @element-plus/icons-vue
   ```

2. 全局注册图标：
   ```javascript
   import * as icons from '@element-plus/icons-vue'
   for (const [key, component] of Object.entries(icons)) {
     app.component(key, component)
   }
   ```

### 按需导入不生效

**症状**：打包后体积仍然很大

**解决方案**：

1. 检查插件配置是否正确
2. 确保使用了正确的 resolver

### 国际化不生效

**症状**：语言切换后组件文字未变化

**解决方案**：

1. 正确引入语言包：
   ```javascript
   import zhCn from 'element-plus/es/locale/lang/zh-cn'
   ```
2. 配置 locale：
   ```javascript
   app.use(ElementPlus, { locale: zhCn })
   ```

## 组件问题

### 表格问题

#### 表格排序不生效

确保设置了 prop 属性：

```vue
<el-table-column prop="date" sortable />
```

#### 表格固定列错位

确保表格有固定高度：

```vue
<el-table :data="data" height="300" />
```

### 表单问题

#### 表单验证不触发

确保 el-form-item 有 prop 属性：

```vue
<el-form-item label="名称" prop="name">
  <el-input v-model="ruleForm.name" />
</el-form-item>
```

### 对话框问题

#### 对话框滚动问题

在对话框内容过多时，确保添加滚动：

```vue
<el-dialog>
  <div style="max-height: 60vh; overflow-y: auto;">
    <!-- 内容 -->
  </div>
</el-dialog>
```

## 样式问题

### 样式冲突

**症状**：自定义样式被覆盖

**解决方案**：

1. 使用更具体的选择器
2. 使用 ::v-deep 或 :deep()

### 主题色不生效

检查 CSS 变量是否被正确覆盖：

```css
:root {
  --el-color-primary: #409eff;
}
```

确保在 element-plus 样式之后加载。

## 性能问题

### 页面卡顿

对于大数据量：

1. 使用 el-table-v2（虚拟表格）
2. 使用分页加载数据
3. 避免在表格中使用复杂组件

### 打包体积过大

1. 使用按需导入
2. 移除未使用的插件

## Vue 版本问题

### 与 Vue 2 不兼容

Element Plus 仅支持 Vue 3。如果需要 Vue 2，请使用 Element UI。

### Composition API 不支持

确保使用 Vue 3.2+ 以获得更好的 Composition API 支持。
