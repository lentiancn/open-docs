# 소개

Bootstrap은 HTML, CSS, JavaScript를 위한 세계에서 가장 인기 있는 오픈소스 프론트엔드 프레임워크로,originally developed by designers Mark Otto and Jacob Thornton at Twitter. 개발자가 빠르게 반응형 웹 애플리케이션을 구축할 수 있도록 돕는 즉시 사용 가능한 컴포넌트와 강력한 JavaScript 플러그인이 포함되어 있습니다.

Bootstrap의 핵심 철학은 "모바일 우선"입니다 — 처음부터 모든 장치에서 아름답게 작동하도록 설계되었습니다. Bootstrap을 사용하면 데스크톱, 태블릿, 스마트폰 어디서나 훌륭하게 보이는 웹사이트를 쉽게 만들 수 있습니다.

## 간략한 역사

Bootstrap은 Twitter Blueprint라는 이름으로 시작하여 2011년에 오픈소스로 공개되었습니다. Bootstrap 2는 2012년에 반응형 레이아웃 지원과 함께 등장했습니다. 2013년 Bootstrap 3은 플랫 디자인을 사용한 완전히 재설계와 함께 출시되었습니다. Bootstrap 4는 2015년에 등장하여 CSS 전처리를 위해 Sass로 전환했습니다. 그런 다음 2021년 Bootstrap 5가 출시되어 jQuery 의존성을 제거하고 순수 Vanilla JavaScript를 완전히 채택했습니다.

## Bootstrap의 핵심

### 반응형 레이아웃

Bootstrap의 그리드 시스템은 그입니다. 12열 그리드 레이아웃을 사용하여 다양한 화면 크기에 적응하는 페이지 디자인을 쉽게 만들 수 있습니다. Bootstrap 5는 xs, sm, md, lg, xl, xxl 등 6개의 중단점을 제공합니다 — 각각 다양한 장치 너비를 대상으로 합니다.

### 즉시 사용 가능한 컴포넌트

Bootstrap에는 다양한 UI 컴포넌트가 기본적으로 포함되어 있습니다: 네비게이션 바, 카드, 버튼, 폼, 모달, 캐러셀, 드롭다운 등입니다. 모두 세련된 디자인과 부드러운 상호작용을 고려하여 설계되었습니다.

### 강력한 JavaScript 플러그인

Bootstrap은 20개 이상의 JavaScript 플러그인을 포함합니다 — 모달, 캐러셀, 툴팁, 팝오버, 아코디언 등입니다. 모두 선택 사항이므로 프로젝트가 가벼울 수 있도록 필요한 것만 선택할 수 있습니다.

### Sass 기반

Bootstrap 5는 Sass를 CSS 전처리로 사용하며, 수많은 Sass 변수와 mixin이되어 있습니다. 프로젝트에 맞게 테마 색상, 글꼴 크기, 기타 스타일을 쉽게 사용자 정의할 수 있습니다.

### 사용자 정의 테마

Bootstrap의 테마 시스템으로 자신만의 모양과 느낌을 쉽게 만들 수 있습니다. 공식 Bootstrap Themes 마켓플레이스에도 선택할 수 있는 수많은 완성도 높은 테마가 있습니다.

### 즉시 사용 가능한 접근성

Bootstrap 컴포넌트에는 접근성 기능이 기본적으로 포함되어 있으며, 키보드 내비게이션과 스크린 리더 호환성을 포함하여 보다 포용적인 웹사이트를 구축하는 데 도움이 됩니다.

## Bootstrap 5의 새로운 기능

Bootstrap 5는 Bootstrap 4에서 몇 가지 큰 변경 사항을 가져왔습니다:

- jQuery 의존성 제거, 이제 순수 JavaScript 사용
- 새로운 XXL 중단점으로 개선된 그리드 시스템
- 이전 폼 유효성 검사를 대체하는 새로운 폼 컨트롤
- 개선된 사용자 정의 속성 지원
- Font Awesome에서 Bootstrap Icons로 아이콘 라이브러리 변경
- RTL(오른쪽에서 왼쪽) 언어 지원

## 언제 Bootstrap을 사용해야 하나요

- 빠른 프로토타이핑
- 내부 비즈니스 관리 시스템
- 전자상거래 웹사이트
- 블로그 및 콘텐츠 사이트
- 관리자 대시보드
- 모든 반응형 웹사이트

세계에서 가장 인기 있는 CSS 프레임워크 중 하나인 Bootstrap은 방대한 커뮤니티와 수많은 학습 리소스를 가지고 있습니다. 프론트엔드 개발자에게 필수적인 기술입니다.
