# Webpack

Webpack — это статический модульный бандлер для современных JavaScript-приложений.

## Введение

Webpack — это мощный инструмент для объединения модулей, используемый в основном для современных JavaScript-приложений. Он анализирует зависимости проекта и объединяет несколько модулей в оптимизированные статические ресурсы.

## Основные функции

- **Объединение модулей**: Поддержка JavaScript, CSS, изображений, шрифтов и других ресурсов
- **Разделение кода**: Поддержка ленивой загрузки маршрутов и динамического импорта
- **Горячая перезагрузка**: Сервер разработки поддерживает горячую замену модулей
- **Tree Shaking**: Автоматическое удаление неиспользуемого кода
- **Система плагинов**: Обширная экосистема плагинов
- **Сервер разработки**: Встроенный сервер разработки

## Быстрый старт

### Установка

```bash
npm install webpack webpack-cli --save-dev
```

### Настройка

Создайте webpack.config.js:

```javascript
module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
  },
};
```

### Сборка

```bash
npx webpack
```
