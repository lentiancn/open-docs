# Element Plus User Manual

This manual will help you quickly get started with Element Plus and master commonly used components.

## Quick Start

Before using Element Plus, make sure installation is complete.

### Basic Usage

Use components directly in Vue components:

```vue
<template>
  <el-button>Default</el-button>
  <el-button type="primary">Primary</el-button>
  <el-button type="success">Success</el-button>
  <el-button type="warning">Warning</el-button>
  <el-button type="danger">Danger</el-button>
</template>
```

## Common Components

### Button

```vue
<el-button>Default</el-button>
<el-button type="primary">Primary</el-button>
<el-button size="large">Large</el-button>
<el-button disabled>Disabled</el-button>
<el-button icon="Search">Search</el-button>
```

### Input

```vue
<el-input v-model="input" placeholder="Please enter" />
<el-input v-model="input" prefix-icon="Search" />
<el-input v-model="input" clearable />
<el-input v-model="input" type="password" show-password />
```

### Select

```vue
<el-select v-model="value" placeholder="Select">
  <el-option label="Option 1" value="1" />
  <el-option label="Option 2" value="2" />
</el-select>
```

### Table

```vue
<el-table :data="tableData" stripe border>
  <el-table-column prop="name" label="Name" />
  <el-table-column prop="date" label="Date" />
</el-table>
```

### Form

```vue
<el-form :model="form">
  <el-form-item label="Name">
    <el-input v-model="form.name" />
  </el-form-item>
</el-form>
```

### Dialog

```vue
<el-dialog v-model="visible" title="Title" width="30%">
  <span>Content</span>
</el-dialog>
```

### Message

```javascript
import { ElMessage } from 'element-plus'

ElMessage.success('Success message')
ElMessage.warning('Warning message')
ElMessage.error('Error message')
```

### Menu

```vue
<el-menu mode="horizontal">
  <el-menu-item index="1">Processing Center</el-menu-item>
  <el-menu-item index="2">Order Management</el-menu-item>
</el-menu>
```

### Tabs

```vue
<el-tabs v-model="activeName">
  <el-tab-pane label="User" name="first">User Management</el-tab-pane>
  <el-tab-pane label="Config" name="second">Config Management</el-tab-pane>
</el-tabs>
```

### Pagination

```vue
<el-pagination
  v-model:current-page="currentPage"
  :page-size="100"
  :total="1000"
/>
```

## Layout

### Grid System

Element Plus uses 24-column grid system:

```vue
<el-row :gutter="20">
  <el-col :span="12">Column</el-col>
  <el-col :span="12">Column</el-col>
</el-row>
```

### Container

```vue
<el-container>
  <el-header>Header</el-header>
  <el-main>Main</el-main>
</el-container>
```

## Theme Customization

```css
:root {
  --el-color-primary: #409eff;
}
```

## Best Practices

### Responsive Design

```vue
<el-row>
  <el-col :xs="24" :sm="12" :md="8">Responsive</el-col>
</el-row>
```

### Form Validation

Always validate user input and provide clear error messages.

### Internationalization

```javascript
import en from 'element-plus/es/locale/lang/en'
app.use(ElementPlus, { locale: en })
```

## Common Issues

### Styles Not Applied

Make sure CSS is imported:

```javascript
import 'element-plus/dist/index.css'
```

### Icons Not Displayed

Install icon library:

```bash
npm install @element-plus/icons-vue
```

Then register globally:
```javascript
import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```
