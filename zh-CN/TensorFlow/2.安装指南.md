# TensorFlow 安装指南

## 环境要求

- Python 3.8-3.11
- CUDA（GPU 支持）
- cuDNN（GPU 加速）

## CPU 版本

```bash
# 创建虚拟环境（推荐）
python -m venv tf-env
source tf-env/bin/activate  # Linux/macOS
# 或
tf-env\Scripts\activate  # Windows

# 安装
pip install tensorflow
```

## GPU 版本

### 1. 安装 CUDA

下载安装 CUDA Toolkit 11.8+：
https://developer.nvidia.com/cuda-downloads

### 2. 安装 cuDNN

下载 cuDNN：
https://developer.nvidia.com/cudnn

配置环境变量：
```bash
export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/usr/local/cuda/lib64
```

### 3. 安装 TensorFlow GPU 版

```bash
pip install tensorflow[and-cuda]
```

## 验证安装

```python
import tensorflow as tf

print(tf.__version__)

# 检查 GPU
print("GPU Available:", tf.config.list_physical_devices('GPU'))
print("GPU Count:", len(tf.config.list_physical_devices('GPU')))

# 简单测试
a = tf.constant([[1, 2], [3, 4]])
b = tf.constant([[5, 6], [7, 8]])
print(tf.matmul(a, b))
```

## Docker（推荐）

```bash
# CPU 版本
docker pull tensorflow/tensorflow

# GPU 版本
docker pull tensorflow/tensorflow:latest-gpu

# 运行
docker run -it tensorflow/tensorflow
```
