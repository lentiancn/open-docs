# Yarn Installation Guide

Comprehensive guide for installing Yarn on all major platforms.

---

## What is Yarn?

Yarn is a fast, reliable, and secure package manager for JavaScript. There are two major versions:

| Version | Name | Description |
|---------|------|-------------|
| Yarn 1.x | Classic | Original Yarn, widely used |
| Yarn 2/3/4.x | Berry | Modern Yarn with better performance and features |

---

## Yarn 1 (Classic) Installation

### npm (Recommended)

```bash
npm install -g yarn
```

### Corepack (Node.js 16+)

```bash
corepack enable
corepack prepare yarn@stable --activate
```

### Direct Download

```bash
# macOS / Linux
curl -o- -L https://yarnpkg.com/install.sh | bash

# Windows (PowerShell)
scoop install yarn
# or
choco install yarn
```

### Verify Installation

```bash
yarn --version
```

---

## Yarn 2/3/4 (Berry) Installation

### Using Corepack (Recommended)

```bash
# Enable corepack
corepack enable

# Prepare specific version
corepack prepare yarn@stable --activate
corepack prepare yarn@4.1.1 --activate
corepack prepare yarn@3.6.4 --activate
corepack prepare yarn@2.4.3 --activate
```

### Using npm

```bash
npm install -g yarn
# Then enable berry
yarn set version stable
```

### Using Scoop (Windows)

```bash
scoop install yarn
```

### Using Chocolatey (Windows)

```bash
choco install yarn
```

## Platform-Specific Installation

### macOS

```bash
# Using Homebrew
brew install yarn

# Using npm
sudo npm install -g yarn
```

### Linux

```bash
# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
sudo npm install -g yarn

# Fedora
sudo dnf install nodejs
sudo npm install -g yarn
```

### Windows

```bash
# Using Winget (recommended)
winget install Yarn.Yarn

# Using Scoop
scoop install yarn

# Using Chocolatey
choco install yarn
```

## Version Management

### Switch Between Versions

```bash
# Using corepack
corepack prepare yarn@1.22.22 --activate
corepack prepare yarn@4.1.1 --activate

# Using yarn (after installed)
yarn set version 1.22.22
yarn set version 4.1.1
```

### Check Current Version

```bash
yarn --version
```

---

## Upgrading Yarn

### Upgrade Yarn 1

```bash
npm update -g yarn
# or
npm install -g yarn@latest
```

### Upgrade Yarn Berry

```bash
yarn set version stable
yarn set version latest
```
