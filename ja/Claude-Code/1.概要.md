# Claude Code 概要

Claude Codeは、Anthropic社が開発したインテリジェントなプログラミングアシスタントです。コードベースを読み取り、ファイルを編集し、コマンドを実行し、開発ツールとシームレスに統合できます。ターミナル、IDE、デスクトップアプリ、ブラウザなどさまざまな環境で利用可能です。

## Claude Codeとは

Claude Codeは、AI駆動のプログラミングアシスタントです。機能の構築、バグの修正、開発タスクの自動化を支援します。コードベース全体を理解し、複数のファイルやツールをまたいで動作して、さまざまな開発タスクを完了できます。

## 主な機能

Claude Codeは次のコア機能を提供します：

**開発タスクの自動化**
- テストされていないコードのテスト作成
- プロジェクト全体のlintエラー修正
- マージコンフリクトの解決
- 依存関係の更新
- リリースノート作成

**機能開発とバグ修正**
- 自然言語で実現したい機能を説明
- Claude Codeがアプローチを計画し、複数のファイルにコードを書いて検証

**Git統合**
- Claude Code内で直接Git操作を実行
- 変更のステージ、コミットメッセージの作成、マージリクエスト的开启

**MCP統合**
- Model Context Protocol（MCP）は、AIツールを外部データソースに接続するオープン標準
- MCPにより、Claude CodeはGoogle Driveの設計ドキュメントを読み取り、Jiraのチケットを更新、Slackからデータを取得、またはカスタムツールを使用可能

**カスタマイズ機能**
- CLAUDE.mdファイルでプロジェクトのコーディング標準、アーキテクチャ決定事項、推奨ライブラリ、レビューチェックリストを設定
- カスタムコマンドで反復可能なワークフローをパッケージ化
- Hooksでファイル編集後の自動フォーマットやコミット前のlint実行

**マルチエージェントコラボレーション**
- 複数のClaude Codeエージェントを同時に起動して並列作業
- リードエージェントが作業を調整、サブタスクを割り当て、結果をマージ

**CLIの柔軟性**
- パイプライン操作、スクリプト化、自動化をサポート
- ログをパイプラインで渡したり、CIで実行したり、他のツールとチェーン可能

## 使用シナリオ

Claude Codeはさまざまなシナリオで使用できます：

| シナリオ | 推奨方法 |
|----------|----------|
| ローカル開発 | ターミナルCLI |
| IDE内コーディング | VS CodeまたはJetBrains |
| デバイス間のContinuation | リモートコントロール機能 |
| モバイル操作 | WebまたはiOSアプリ |
| CI/CD自動化 | GitHub ActionsまたはGitLab CI/CD |
| チームコラボレーション | Slack統合 |
| ブラウザデバッグ | Chrome拡張機能 |

## アカウント要件

Claude Codeには次のいずれかのアカウントが必要です：
- Claude Pro、Max、Teams、またはEnterpriseサブスクリプション
- Anthropic Consoleアカウント
- サポートされているサードパーティクラウドプロバイダー（Amazon Bedrock、Google Vertex AI、Microsoft Foundry）

無料のClaude.aiプランにはClaude Codeアクセス権は含まれていません。

## 次のステップ

Claude Codeをインストールしたら、次のように探索を続けることができます：

- **クイックスタート**：コードベースの探索から修正のコミットまで、実際のタスクを通じてステップバイステップで学習
- **指示とメモリの保存**：CLAUDE.mdファイルでClaudeに永続的な指示を提供
- **一般的なワークフローとベストプラクティス**：Claude Codeの能力を最大限に引き出すパターン
- **カスタム設定**：ワークフローに応じてClaude Codeをカスタマイズ
- **トラブルシューティング**：一般的な問題の解決

詳細については、[code.claude.com](https://code.claude.com/)をご覧ください。
