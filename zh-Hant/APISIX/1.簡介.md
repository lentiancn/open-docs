# APISIX 簡介

雲端原生 API 閘道器。

## 什麼是 APISIX？

APISIX 是 Apache 軟體基金會旗下的雲端原生 API 閘道器，為企業提供高性能、高可用的 API 管理服務。作為動態、實時、可擴展的 API 閘道器，APISIX 已成為數百萬容器化部署的首選解決方案。

### 核心特性

- **高性能**：基於 Nginx + Lua 實現，單核可達 10k+ QPS
- **動態路由**：無需重啟即可動態更新路由規則
- **插件系統**：豐富的插件生態，支持熱加載
- **服務發現**：支持 Consul、Eureka、Nacos 等多種服務發現機制
- **負載均衡**：輪詢、哈希、一致性哈希等多種策略
- **安全防護**：限流、認證，JWT，防火牆
- **可視化儀表盤**：直觀的管理界面
- **多協議支持**：HTTP、HTTPS、HTTP/2、Dubbo、gRPC、WebSocket

### 技術架構

APISIX 採用 Nginx 作為底層的 HTTP 反向代理服務器，結合 Lua 腳本實現動態編排。整個架構分為：

- **數據面**：處理實際請求轉發，零依賴
- **控制面**：通過 Admin API 進行配置管理

## 適用場景

- **API 統一管理**：集中管理企業內部所有 API
- **微服務閘道器**：作為微服務入口，處理認證、限流、路由
- **服務網格入口**：對接 Istio、Linkerd 等服務網格
- **API 流量控制**：限流、熔斷、降級
- **混合雲架構**：統一管理多雲、混合雲 API
- **API 統計分析**：實時監控 API 調用情況
