# OpenClaw User Manual

This manual covers daily operations and features of OpenClaw.

## Core Concepts

### Gateway

Gateway is the heart of OpenClaw. All message routing, AI requests, and channel management flow through this single process. You only need to run one Gateway, and it handles connections from all your chat platforms automatically.

### Channels

Channels are the messaging apps you use to talk to your AI assistant. OpenClaw supports connecting multiple channels at once, including WhatsApp, Telegram, Discord, iMessage, and more.

### Agents

Agents are independent AI assistant instances. Each agent has its own workspace and conversation history. You can run multiple agents for different purposes.

## Daily Operations

### Starting and Stopping

```bash
# Start (runs in background)
openclaw gateway start

# Stop
openclaw gateway stop

# Restart
openclaw gateway restart

# Check status
openclaw gateway status

# Run in foreground (for debugging)
openclaw gateway --port 18789
```

### Sending Messages

```bash
# Send a direct message
openclaw message send --target +15555550123 --message "Test message"

# Send to a group
openclaw message send --target "Group Name" --message "Hello everyone"
```

### Managing Configuration

OpenClaw uses JSON5 format config files, located at `~/.openclaw/openclaw.json`.

Ways to modify config:

1. **Wizard mode**:
   ```bash
   openclaw onboard
   openclaw configure
   ```

2. **Command line**:
   ```bash
   openclaw config get agents.defaults.workspace
   openclaw config set agents.defaults.heartbeat.every "2h"
   openclaw config unset tools.web.search.apiKey
   ```

3. **Web interface**:
   Open http://127.0.0.1:18789/, go to the Config tab

4. **Direct edit**:
   ```bash
   nano ~/.openclaw/openclaw.json
   ```

### Hot Reload

Most config changes take effect automatically without restarting.

## Common Config Examples

### Restrict Access

```json5
{
  channels: {
    whatsapp: {
      allowFrom: ["+15555550123"]
    }
  }
}
```

### Group Chat Settings

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

### Multiple Agents

```json5
{
  agents: {
    list: [
      { id: "home", default: true, workspace: "~/.openclaw/workspace-home" },
      { id: "work", workspace: "~/.openclaw/workspace-work" }
    ]
  }
}
```

### Scheduled Tasks

```json5
{
  cron: {
    enabled: true,
    maxConcurrentRuns: 2
  }
}
```

### Periodic Check-ins

```json5
{
  agents: {
    defaults: {
      heartbeat: {
        every: "30m",
        target: "last"
      }
    }
  }
}
```

### Sandbox Isolation

For running AI operations in isolated containers:

```json5
{
  agents: {
    defaults: {
      sandbox: {
        mode: "non-main",
        scope: "agent"
      }
    }
  }
}
```

### Webhooks

```json5
{
  hooks: {
    enabled: true,
    token: "your-secret-token",
    path: "/hooks"
  }
}
```

## Channel Management

### Adding Channels

```bash
# Login to WhatsApp
openclaw channels login whatsapp

# Login to Telegram
openclaw channels login telegram
```

### List Connected Channels

```bash
openclaw channels list
```

## Troubleshooting

### View Logs

```bash
# Service logs
openclaw logs

# Follow logs in real-time
openclaw logs -f
```

### Health Check

```bash
openclaw health
```

### Auto-diagnose

```bash
openclaw doctor
```
