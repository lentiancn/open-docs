# Kotlin

Kotlin est un langage de programmation moderne développé par JetBrains qui s'exécute sur la JVM.

## Introduction

Kotlin est un langage de programmation à typage statique qui peut s'exécuter sur la machine virtuelle Java et peut également être compilé en code JavaScript.

## Caractéristiques principales

- **Concis**: Réduit le code boilerplate
- **Sûr**: Conception avec sécurité null
- **Interopérable**: 100% interopérable avec Java
- **Outils conviviaux**: Excellent support IDE
- **Coroutines**: Support puissant pour la programmation asynchrone

## Démarrage rapide

### Installation

```bash
# macOS
brew install kotlin

# Linux
sudo apt install kotlin

# SDKMAN
sdk install kotlin
```

### Compilation et exécution

```bash
kotlinc hello.kt -include-runtime -d hello.jar
java -jar hello.jar
```
