# Claude Code User Manual

Basic usage and techniques.

## Basic Usage

### Starting

- VS Code: Use shortcut Cmd/Ctrl + Shift + P
- CLI: Run `claude` directly in terminal

### Conversation

Describe your needs directly in chat window:
```
Write a sorting function for me
Explain this code
How to fix this bug
```

## Code Generation

### Generate New Code

Describe the functionality you want:
```
Create a user login API
Implement pagination query
Write unit tests
```

### Modify Existing Code

1. Select the code
2. Describe modification needs
3. Claude will generate modified code

## Debugging Assistance

### Explain Errors

Paste error message directly:
```
What does this error mean?
```

### Fix Suggestions

Ask Claude to analyze the problem:
```
What's wrong with this function?
```

## Code Review

### Code Review Requests

```
Review this code
Any security issues?
Performance optimization suggestions?
```

## Keyboard Shortcuts

### VS Code

| Shortcut | Function |
|---------|----------|
| Cmd/Ctrl + L | Open chat |
| Cmd/Ctrl + Enter | Submit input |

## Configuration Options

### claude.json

Create `claude.json` in project root:

```json
{
  "preamble": "You are a Python expert",
  "maxTokens": 4000
}
```

## Best Practices

1. Describe requirements clearly
2. Provide context information
3. Iterate and improve code
4. Combine with human review
