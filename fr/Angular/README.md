# Angular

Plateforme de développement web moderne.

## Aperçu

Angular est une plateforme de développement pour créer des applications web mobiles et de bureau, développée par Google. Utilise TypeScript/JavaScript avec architecture basée sur les composants, liaison bidirectionnelle et injection de dépendances.

### Fonctionnalités principales

- **Basé sur les composants** - Architecture pilotée par les composants
- **TypeScript** - Support du typage fort
- **Liaison bidirectionnelle** - Synchronisation des données simplifiée
- **Injection de dépendances** - Système puissant de gestion des dépendances
- **Routage** - Solution de routage complète

## Démarrage rapide

### Installer

```bash
npm install -g @angular/cli
```

### Créer un projet

```bash
ng new my-app
cd my-app
ng serve
```

Visiter : http://localhost:4200
