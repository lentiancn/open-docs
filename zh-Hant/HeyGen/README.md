# HeyGen

AI 數字人視頻生成平台

## 簡介

HeyGen 是一個創新的 AI 視頻生成平台，使用先進的人工智慧技術創建逼真的數字人視頻。用戶可以通過輸入文字、上傳圖片或音頻，快速生成具有自然表情和口型同步的 AI 主播視頻內容。

## 主要功能

- **文字轉視頻**：將文字腳本轉化為專業視頻
- **圖片轉視頻**：將靜態圖片轉化為動態視頻
- **AI 數字人**：創建逼真的虛擬人物形象
- **AI 翻譯**：支援 175 種以上語言
- **AI Studio**：統一的視頻創作工作空間
- **多平台導出**：720p、1080p、4K

## 快速開始

1. 訪問 HeyGen 官方網站並註冊帳戶
2. 選擇數字人或創建自定義形象
3. 輸入視頻腳本或上傳音頻
4. 選擇語音和背景
5. 生成並下載視頻
