# jQuery 安裝指南

## 取得 jQuery

### 最新版本

目前最新版本為 **jQuery 4.0.0**。

#### 生產環境壓縮版（推薦）
- 下載位址：[jquery-4.0.0.min.js](https://code.jquery.com/jquery-4.0.0.min.js)
- 適用於生產環境，檔案體積小，載入速度快

#### 開發偵錯版
- 下載位址：[jquery-4.0.0.js](https://code.jquery.com/jquery-4.0.0.js)
- 包含完整程式碼，便於閱讀和偵錯

#### Source Map 檔案
- 下載位址：[jquery-4.0.0.min.map](https://code.jquery.com/jquery-4.0.0.min.map)
- 用於在瀏覽器偵錯器中顯示原始程式碼

#### Slim 版本
Slim 版本是精簡版，移除了 Ajax 和動畫模組，檔案更小：
- [jquery-4.0.0.slim.min.js](https://code.jquery.com/jquery-4.0.0.slim.min.js) - 壓縮版
- [jquery-4.0.0.slim.js](https://code.jquery.com/jquery-4.0.0.slim.js) - 未壓縮版

如果不需要 Ajax 和動畫功能，使用 Slim 版本可以減少約 30% 的檔案大小。

## 安裝方式

### 方式一：CDN 引入（推薦）

使用 jQuery CDN 引入是最簡單的方式，無需下載檔案，直接在 HTML 中參照即可：

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
```

帶 Subresource Integrity（SRI）驗證的版本（推薦，可防止 CDN 被篡改）：

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js" 
        integrity="sha384-KJ3o2DKtIkvYIK3UENzmM7KCkRr/rE9/Qpg6aAZGJwFDMVNA/GpGFF93hXpG5KkN" 
        crossorigin="anonymous"></script>
```

可以透過存取 [https://releases.jquery.com](https://releases.jquery.com) 取得完整版本列表和 SRI 雜湊值。

### 方式二：npm 安裝

如果使用 Node.js 專案，可以透過 npm 安裝：

```bash
npm install jquery
```

安裝後，在 node_modules/jquery/dist/ 目錄下可以找到：
- jquery-4.0.0.min.js - 壓縮版
- jquery-4.0.0.js - 未壓縮版
- jquery-4.0.0.min.map - Source Map

### 方式三：Yarn 安裝

```bash
yarn add jquery
```

### 方式四：下載到本地

存取 [jQuery 官網下載頁面](https://jquery.com/download/)，右鍵點擊所需版本，選擇「另存新檔」將檔案儲存到本地專案目錄中。

## 其他 CDN 服務

除官方 CDN 外，以下 CDN 也提供 jQuery：

### Google CDN
```
https://ajax.googleapis.com/ajax/libs/jquery/4.0.0/jquery.min.js
```

### Microsoft CDN
```
https://ajax.aspnetcdn.com/ajax/jquery/jquery-4.0.0.min.js
```

### CDNJS
```
https://cdnjs.cloudflare.com/ajax/libs/jquery/4.0.0/jquery.min.js
```

### jsDelivr
```
https://cdn.jsdelivr.net/npm/jquery@4.0.0/dist/jquery.min.js
```

注意：這些第三方 CDN 可能存在一定的延遲，新版本發布後可能需要等待一段時間才能同步。

## 升級 jQuery

### 使用 jQuery Migrate 插件

升級 jQuery 時，建議使用 jQuery Migrate 插件來識別相容性問題：

1. 從舊版本升級到 1.9+ 或 3.x：使用 [jQuery Migrate 1.4.1](https://code.jquery.com/jquery-migrate-1.4.1.min.js)

2. 從 jQuery 3.x 升級到更新版本：使用 [jQuery Migrate 3.6.0](https://code.jquery.com/jquery-migrate-3.6.0.min.js)

使用方法：在 jQuery 之後引入 Migrate 插件

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
<script src="https://code.jquery.com/jquery-migrate-3.6.0.min.js"></script>
```

Migrate 會在主控台輸出相容性問題警告，幫助你逐步修正程式碼。

### 升級指南

針對不同版本升級，請查閱官方升級指南：
- [jQuery 4.0 升級指南](https://jquery.com/upgrade-guide/4.0/)
- [jQuery 3.0 升級指南](https://jquery.com/upgrade-guide/3.0/)
- [jQuery 1.9 升級指南](https://jquery.com/upgrade-guide/1.9/)

## 歷史版本

需要使用舊版本 jQuery 時，可以存取 [jQuery CDN](https://releases.jquery.com) 查看所有歷史版本。

## 預發布版本

jQuery 開發團隊會在 GitHub 上持續開發，每個提交都會生成預發布版本。這些版本可能不穩定，不建議在生產環境中使用。

查看預發布版本：[https://releases.jquery.com/jquery/](https://releases.jquery.com/jquery/)
