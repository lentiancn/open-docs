# FAQ Go

> Questions fréquemment posées

---

## Installation

### Windows: commande go non trouvée?

**Solution**: Ajouter `C:\Go\bin` au PATH.

### Proxy?

```bash
go env -w GOPROXY=https://goproxy.cn,direct
```

---

## Développement

### Erreur "cannot find package"?

**Solution**:
1. Exécuter `go mod init`
2. Exécuter `go mod tidy`

### := vs var?

- `var`: Explicite, hors fonctions
- `:=`: Court, seulement dans les fonctions

## Concurrence

### Goroutine leak?

```go
func worker(ctx context.Context) {
    for {
        select {
        case <-ctx.Done():
            return
        }
    }
}
```
