# Element UI

Biblioteca de componentes Vue.js 2.0.

## Resumen

Element UI es una biblioteca de componentes desarrollada por Ele.me.

### Características

- **60+ Componentes**
- **Diseño hermoso**
- **Soporte TypeScript**

### Estadísticas

- GitHub Stars: 54,000+

## Inicio rápido

```bash
npm install element-ui --save
```

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
Vue.use(ElementUI)
```
