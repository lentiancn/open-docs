# Claude Code Guide d'Installation

Ce guide détaille l'installation, la configuration et la mise à jour de Claude Code.

## Configuration Requise

Claude Code fonctionne sur:
- macOS 13.0+, Windows 10 1809+, Ubuntu 20.04+, Debian 10+, Alpine Linux 3.19+
- 4 GB+ RAM
- Connexion Internet requise

## Méthodes d'Installation

### Installation Native (Recommandée)

**macOS, Linux, WSL:**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell:**
```powershell
irm https://claude.ai/install.ps1 | iex
```

**Windows CMD:**
```batch
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
```

### Homebrew (macOS/Linux)
```bash
brew install --cask claude-code
```

### WinGet (Windows)
```powershell
winget install Anthropic.ClaudeCode
```

## Vérification

Après installation:
```bash
claude --version
```

Pour diagnostic:
```bash
claude doctor
```

## Mise à Jour

### Automatique (installation native)
Les mises à jour se font automatiquement en arrière-plan.

### Manuelle
```bash
claude update
```

## Configuration

### Windows
Configurez le chemin Git Bash si nécessaire:
```json
{
  "env": {
    "CLAUDE_CODE_GIT_BASH_PATH": "C:\\Program Files\\Git\\bin\\bash.exe"
  }
}
```

### Alpine Linux
```json
{
  "env": {
    "USE_BUILTIN_RIPGREP": "0"
  }
}
```

## Désinstallation

**Native:**
```bash
rm -f ~/.local/bin/claude
rm -rf ~/.local/share/claude
```

**Homebrew:**
```bash
brew uninstall --cask claude-code
```

**WinGet:**
```powershell
winget uninstall Anthropic.ClaudeCode
```
