# Element UI FAQ

## インストール

### Q: インストール失敗？

**解決策:**
1. Node.jsバージョンを確認
```bash
node --version
```
2. npmキャッシュをクリア
```bash
npm cache clean --force
```

### Q: Vueバージョン非互換？

**解決策:**
Element UI 2.xはVue 2.6.0+が必要

## 使用

### Q: コンポーネント表示されない？

**解決策:**
インポートを確認：
```javascript
import { Button } from 'element-ui'
Vue.use(Button)
```

### Q: スタイル適用されない？

**解決策:**
スタイルファイルをインポート：
```javascript
import 'element-ui/lib/theme-chalk/index.css'
```
