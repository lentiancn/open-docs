# APISIX Aperçu

Passerelle API cloud-native.

## Qu'est-ce que APISIX?

APISIX est une passerelle API cloud-native développée par la Apache Software Foundation, offrant des services de gestion d'API haute performance et haute disponibilité. En tant que passerelle API dynamique, en temps réel et évolutive, APISIX est devenue la solution privilégiée pour des millions de déploiements en conteneurs.

### Fonctionnalités Principales

- **Haute Performance**: Basé sur Nginx + Lua, 10k+ QPS par cœur
- **Routage Dynamique**: Mettre à jour les règles de routage sans redémarrage
- **Système de Plugins**: 70+ plugins avec support hot-loading
- **Découverte de Services**: Support pour Consul, Eureka, Nacos
- **Équilibrage de Charge**: Roundrobin, hash, hash cohérent
- **Sécurité**: Limitation de débit, authentification, JWT, pare-feu
