# FantasyTalking 安装指南

FantasyTalking 是一款由 AI 驱动的 talking head 生成工具，可根据单张图片和音频输入生成逼真的 talking head 视频。

## 版本信息

| 版本 | 状态 | 描述 |
|---------|--------|-------------|
| FantasyTalking v1.0 | ✅ 当前版本 | 初始发布 |

## 系统需求

### 硬件需求

| 需求 | 最低配置 | 推荐配置 |
|-------------|---------|-------------|
| GPU | NVIDIA 4GB VRAM | NVIDIA 8GB+ VRAM |
| RAM | 8GB | 16GB+ |
| 存储空间 | 10GB | 20GB+ SSD |

### 软件需求

| 软件 | 版本 |
|----------|---------|
| Python | 3.8+ |
| CUDA | 11.0+ |
| cuDNN | 8.0+ |

## 安装方法

### 方法一：从源码安装

#### 1. 克隆仓库

```bash
git clone https://github.com/FantasyTalking/FantasyTalking.git
cd FantasyTalking
```

#### 2. 创建虚拟环境

```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
# 或
venv\Scripts\activate  # Windows
```

#### 3. 安装依赖

```bash
pip install torch torchvision
pip install -r requirements.txt
```

#### 4. 下载模型

```bash
# 下载预训练模型
bash scripts/download_models.sh
```

### 方法二：Docker

```bash
# 拉取镜像
docker pull fantasytalking/fantasytalking:latest

# 运行
docker run --gpus all -v /path/to/data:/data fantasytalking/fantasytalking:latest
```

## 配置

### 环境变量

```bash
export CUDA_VISIBLE_DEVICES=0
export PYTHONPATH=/path/to/FantasyTalking:$PYTHONPATH
```

### 模型路径

编辑 `config.yaml` 来设置模型路径：

```yaml
model:
  checkpoint_path: "checkpoints/fantasytalking.pth"
  face_parser: "models/face_parser.pth"
```

## 快速开始

### 生成 Talking Head

```bash
python inference.py \
  --source_image path/to/image.jpg \
  --audio path/to/audio.wav \
  --output output.mp4
```

### 参数

```bash
--source_image    # 来源人脸图片
--audio          # 输入音频文件
--output         # 输出视频路径
--face enhancer  # 启用的人脸增强
--fps            # 输出帧率（默认：25）
```

## 常见问题

### GPU 不可用？

检查 CUDA 安装：

```bash
nvidia-smi
python -c "import torch; print(torch.cuda.is_available())"
```

### 内存不足？

减少批次大小或使用较低分辨率。
