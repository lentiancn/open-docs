# Flutter - FAQ

Preguntas frecuentes.

## Instalación

### ¿Descarga fallida?

```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
```

### ¿Error flutter doctor?

Verificar PATH.

### ¿Android SDK no encontrado?

```bash
flutter config --android-sdk /ruta/android-sdk
```

## Desarrollo

### ¿Hot reload no funciona?

Guardar y esperar.

### ¿Build APK fallido?

```bash
flutter clean
```
