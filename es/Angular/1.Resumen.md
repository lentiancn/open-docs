# Angular Resumen

Framework de desarrollo de aplicaciones web modernas.

## ¿Qué es Angular?

Angular es un framework de aplicaciones web de código abierto desarrollado y mantenido por Google.

### Características principales

- **TypeScript nativo**
- **Arquitectura basada en componentes**
- **Inyección de dependencias**
- **Sistema de enrutamiento**
- **Formularios reactivos**
