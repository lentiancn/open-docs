# Mockito

Ведущий фреймворк для мокирования Java-юнит-тестов.

## Что такое Mockito?

Mockito — ведущий фреймворк для мокирования Java-юнит-тестов. Он позволяет разработчикам создавать и проверять мок-объекты для написания изолированных тестов.

### Основные функции

- **Интуитивный API** - легко изучить и использовать
- **Гибкое мокирование** - создание моков для классов и интерфейсов
- **Проверка** - проверка вызовов методов
- **Стаббинг** - определение предопределённых возвращаемых значений

## Быстрый старт

```xml
<dependency>
    <groupId>org.mockito</groupId>
    <artifactId>mockito-core</artifactId>
    <version>5.8.0</version>
    <scope>test</scope>
</dependency>
```
