# H2 데이터베이스 사용 매뉴얼

## 기본 작업

### 데이터베이스 생성

임베디드 모드 (인메모리 데이터베이스):
```java
String url = "jdbc:h2:mem:testdb";
Connection conn = DriverManager.getConnection(url, "sa", "");
```

파일 데이터베이스:
```java
String url = "jdbc:h2:file:./data/testdb";
Connection conn = DriverManager.getConnection(url, "sa", "password");
```

### 테이블 생성

```sql
CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100) UNIQUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 데이터 삽입

```sql
INSERT INTO users (name, email) VALUES ('김철수', 'chulsoo@예제.com');
INSERT INTO users (name, email) VALUES ('이영희', 'younghee@예제.com');
```

### 데이터 조회

```sql
SELECT * FROM users;
SELECT * FROM users WHERE name = '김철수';
SELECT COUNT(*) FROM users;
```

### 데이터 업데이트

```sql
UPDATE users SET email = '새이메일@예제.com' WHERE id = 1;
```

### 데이터 삭제

```sql
DELETE FROM users WHERE id = 2;
```

## Java 코드 예제

### 완전한 JDBC 예제

```java
import java.sql.*;

public class H2Example {
    public static void main(String[] args) throws Exception {
        // 드라이버 로드
        Class.forName("org.h2.Driver");
        
        // 데이터베이스 연결
        String url = "jdbc:h2:mem:testdb";
        Connection conn = DriverManager.getConnection(url, "sa", "");
        
        // 테이블 생성
        Statement stmt = conn.createStatement();
        stmt.execute("CREATE TABLE users (id INT PRIMARY KEY AUTO_INCREMENT, name VARCHAR(100))");
        
        // 데이터 삽입
        stmt.execute("INSERT INTO users (name) VALUES ('김철수')");
        stmt.execute("INSERT INTO users (name) VALUES ('이영희')");
        
        // 데이터 조회
        ResultSet rs = stmt.executeQuery("SELECT * FROM users");
        while (rs.next()) {
            System.out.println(rs.getInt("id") + ": " + rs.getString("name"));
        }
        
        // 연결 닫기
        rs.close();
        stmt.close();
        conn.close();
    }
}
```

### PreparedStatement 사용

```java
PreparedStatement pstmt = conn.prepareStatement("INSERT INTO users (name, email) VALUES (?, ?)");
pstmt.setString(1, "박지영");
pstmt.setString(2, "jiyoung@예제.com");
pstmt.executeUpdate();
pstmt.close();
```

### 트랜잭션 사용

```java
conn.setAutoCommit(false);
try {
    Statement stmt = conn.createStatement();
    stmt.execute("INSERT INTO users (name) VALUES ('정수민')");
    stmt.execute("INSERT INTO users (name) VALUES ('최민호')");
    conn.commit();
} catch (Exception e) {
    conn.rollback();
} finally {
    conn.setAutoCommit(true);
}
```

## 고급 기능

### 데이터베이스 모드 전환

H2는 다른 데이터베이스의 SQL 구문을 시뮬레이션할 수 있습니다:

```java
// MySQL 모드
String url = "jdbc:h2:mem:testdb;MODE=MySQL";

// PostgreSQL 모드
String url = "jdbc:h2:mem:testdb;MODE=PostgreSQL";

// Oracle 모드
String url = "jdbc:h2:mem:testdb;MODE=Oracle";
```

### 데이터베이스 암호화

```java
// 암호화된 데이터베이스 사용
String url = "jdbc:h2:file:./data/encrypted;CIPHER=AES";
Connection conn = DriverManager.getConnection(url, "sa", "password");
```

### 추적 기능

```java
// SQL 추적 활성화
String url = "jdbc:h2:mem:testdb;TRACE_LEVEL_FILE=3";
```

### TCP 서버 시작

```java
import org.h2.tools.Server;

Server server = Server.createTcpServer().start();
System.out.println("H2 TCP Server started: " + server.getURL());
```

### H2 콘솔 사용

H2 웹 콘솔 시작:

```bash
java -jar h2.jar -web
```

그런 다음 브라우저에서 열기: http://localhost:8082

## 일반적인 SQL 문

### 데이터 타입

H2는 다양한 데이터 타입을 지원합니다:
- 정수 타입: INT, INTEGER, BIGINT, SMALLINT, TINYINT
- 부동소수점 타입: FLOAT, REAL, DOUBLE, DECIMAL
- 문자열 타입: VARCHAR, CHAR, CLOB
- 날짜시간 타입: DATE, TIME, TIMESTAMP
- 불리언 타입: BOOLEAN
- 이진 타입: BLOB, BINARY

### 인덱스 작업

인덱스 생성:
```sql
CREATE INDEX idx_name ON users(name);
```

인덱스 삭제:
```sql
DROP INDEX idx_name;
```

### 제약조건 작업

기본 키 추가:
```sql
ALTER TABLE users ADD PRIMARY KEY (id);
```

외래 키 추가:
```sql
ALTER TABLE orders ADD FOREIGN KEY (user_id) REFERENCES users(id);
```

### 테이블 구조 보기

```sql
DESCRIBE users;
-- 또는
SHOW COLUMNS FROM users;
```

### 모든 테이블 보기

```sql
SHOW TABLES;
```

## 성능 최적화

### 배치 작업

배치 작업을 사용하여 성능 향상:

```java
conn.setAutoCommit(false);
PreparedStatement pstmt = conn.prepareStatement("INSERT INTO users (name) VALUES (?)");
for (int i = 0; i < 1000; i++) {
    pstmt.setString(1, "사용자" + i);
    pstmt.addBatch();
}
pstmt.executeBatch();
conn.commit();
```

### 커넥션 풀

프로덕션 환경에서는 HikariCP와 같은 커넥션 풀 사용을 권장합니다:

```java
config.setJdbcUrl("jdbc:h2:file:./data/testdb");
config.setUsername("sa");
config.setPassword("");
config.setMaximumPoolSize(10);

HikariDataSource ds = new HikariDataSource(config);
```

## 일반적인 오류 처리

### 데이터베이스 파일이 잠겨 있음
다른 연결이 데이터베이스 파일을 사용하고 있지 않은지 확인합니다. H2는 동시 읽기를 지원하지만 쓰기에는独占 액세스가 필요합니다.

### 인메모리 데이터 손실
인메모리 데이터베이스는 연결이 닫힌 후 데이터를 손실합니다. 영구 저장이 필요하면 파일 데이터베이스 모드를 사용하세요.

### 클래스를 찾을 수 없음
org.h2.Driver 클래스가 classpath에 있는지 확인합니다.
