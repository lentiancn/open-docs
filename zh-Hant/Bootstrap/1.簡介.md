# Bootstrap 簡介

> 熱門的前端 CSS 框架

---

## 項目簡介

Bootstrap 是由 Twitter 開發和維護的開源 CSS 框架，於 2011 年首次發布。Bootstrap 提供了豐富的預製組件和響應式網格系統，能夠快速構建現代、響應式的網站和 Web 應用。

---

## 版本資訊

| 版本 | 發布日期 | 狀態 |
|------|----------|------|
| Bootstrap 5.3 | 2023年5月 | 當前穩定版 |
| Bootstrap 5.2 | 2022年9月 | 穩定版 |
| Bootstrap 5.1 | 2021年8月 | 穩定版 |
| Bootstrap 4.x | 2018年1月 | 已停止維護 |

---

## 主要特性

- **響應式設計**：內置響應式佈局
- **預製組件**：按鈕、導航、表單、卡片等
- **強大的網格系統**：12列響應式網格
- **自定義選項**：Sass 變量易於定製
- **JavaScript 插件**：下拉菜單、模態框等
- **圖標庫**：內置 Bootstrap 圖標

---

## 適用場景

- 響應式網站開發
- 後台管理系統
- 快速原型開發
- 移動優先的項目
