# Angular

현대 웹 개발 플랫폼.

## 개요

Angular는 Google이 개발한 모바일 및 데스크톱 웹 애플리케이션 구축을 위한 개발 플랫폼입니다. TypeScript/JavaScript를 사용하며, 컴포넌트 기반, 양방향 바인딩, 의존성 주입等特点이 있습니다.

### 주요 기능

- **컴포넌트 기반** - 컴포넌트 중심 아키텍처
- **TypeScript** - 강한 타입 지원
- **양방향 바인딩** - 데이터 동기화 간소화
- **의존성 주입** - 강력한 의존성 관리 시스템
- **라우팅** - 완전한 라우팅 솔루션

## 빠른 시작

### 설치

```bash
npm install -g @angular/cli
```

### 프로젝트 생성

```bash
ng new my-app
cd my-app
ng serve
```

접속: http://localhost:4200
