# Manual de usuario de Nginx

## Operaciones básicas

```bash
nginx              # Iniciar
nginx -s stop     # Detener
nginx -s reload    # Recargar configuración
nginx -t           # Probar configuración
```

## Configuración básica

```nginx
worker_processes auto;
events {
    worker_connections 1024;
}
http {
    server {
        listen 80;
        server_name ejemplo.com;
        root /var/www/html;
        location / {
            index index.html;
        }
    }
}
```

## Archivos estáticos

```nginx
server {
    listen 80;
    root /var/www/html;
    location / {
        try_files $uri $uri/ =404;
    }
}
```

## Proxy inverso

```nginx
location / {
    proxy_pass http://backend;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
}
```

## Balanceo de carga

```nginx
upstream backend {
    server backend1.example.com;
    server backend2.example.com;
}
```

## SSL/TLS

```nginx
server {
    listen 443 ssl;
    ssl_certificate cert.crt;
    ssl_certificate_key cert.key;
    ssl_protocols TLSv1.2 TLSv1.3;
}
```

## Directivas comunes

- `root`: Directorio raíz de documentos
- `index`: Archivos de índice predeterminados
- `proxy_pass`: Backend del proxy
- `upstream`: Grupo de servidores backend
- `gzip`: Habilitar compresión
- `expires`: Caché del navegador
