# Angular

> Google開発のWebフレームワーク

---

## 概要

AngularはGoogleが開発したTypeScriptベースのフレームワークです。モダンWebアプリケーションのための完全なソリューションを提供します。

## バージョン

| バージョン | 发布日期 | 状態 |
|-----------|--------|------|
| Angular 17 | 2023年11月 | 現行 |
| Angular 16 | 2023年5月 | 安定版 |
