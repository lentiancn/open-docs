# Element UI インストールガイド

完全なElement UIのインストールと設定ガイド。

## 環境要件

### ソフトウェア要件

- Node.js：12.0.0+
- npm：6.0.0+ または yarn：1.22.0+
- Vue：2.6.0+

## インストール手順

### 方法1：npmインストール（推奨）

```bash
# npmを使用
npm install element-ui --save

# またはyarnを使用
yarn add element-ui
```

### 方法2：CDNインポート

```html
<!-- スタイルをインポート -->
<link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">

<!-- Vueをインポート -->
<script src="https://unpkg.com/vue"></script>

<!-- Element UIをインポート -->
<script src="https://unpkg.com/element-ui/lib/index.js"></script>
```

## クイックスタート

### 完全インポート

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(ElementUI)

new Vue({
  el: '#app',
  render: h => h(App)
})
```

### オンデマンドインポート（推奨）

まず babel-plugin-component をインストール：

```bash
npm install babel-plugin-component -D
```

babel.config.jsを設定：

```javascript
module.exports = {
  presets: [
    '@vue/cli-plugin-babel/preset'
  ],
  plugins: [
    [
      'component',
      {
        libraryName: 'element-ui',
        styleLibraryName: 'theme-chalk'
      }
    ]
  ]
}
```

コンポーネントをオンデマンドでインポート：

```javascript
import { Button, Select, Option, Table, TableColumn } from 'element-ui'

Vue.use(Button)
Vue.use(Select)
Vue.use(Option)
Vue.use(Table)
Vue.use(TableColumn)
```

## テーマカスタマイズ

### SCSS変数を使用

変数ファイルを作成：

```scss
// styles/element-variables.scss

$--color-primary: #409EFF;
$--color-success: #67C23A;
$--color-warning: #E6A23C;
$--color-danger: #F56C6C;

@import 'element-ui/packages/theme-chalk/src/common/var.scss';
```

main.jsでインポート：

```javascript
import './styles/element-variables.scss'
```

### オンラインテーマエディタ

訪問：https://element.eleme.io/#/ja/theme

1. テーマカラーを選択またはカスタマイズ
2. 「ダウンロード」ボタンをクリック
3. Element UIテーマファイルを置き換え

## 国際化

### 多言語設定

vue-i18nをインストール：

```bash
npm install vue-i18n -S
```

設定：

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import VueI18n from 'vue-i18n'
import jaLocale from 'element-ui/lib/locale/lang/ja'

Vue.use(ElementUI, {
  i18n: (key, value) => i18n.t(key, value)
})

const messages = {
  'ja': { message: 'こんにちは' }
}

const i18n = new VueI18n({
  locale: 'ja',
  messages
})

new Vue({ i18n }).$mount('#app')
```

## 一般的な問題

### コンポーネントが表示されない？

1. コンポーネントが正しくインポートされているか確認
2. Vue.use()が呼ばれているか確認
3. コンソールでエラーを確認

### スタイルが適用されない？

1. スタイルファイルがインポートされているか確認
2. グローバルスタイルの上書きがないか確認
3. z-indexレベルを確認

### アイコンが表示されない？

```bash
# アイコンライブラリをインストール
npm install element-ui-icons
```

アイコンをインポート：

```javascript
import 'element-ui-icons'
```

使用：

```html
<i class="el-icon-search"></i>
```
