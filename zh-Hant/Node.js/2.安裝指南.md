# Node.js 安裝指南

## 下載 Node.js

Node.js 提供了適用於各種作業系統的官方安裝套件。造訪 [Node.js 官方網站](https://nodejs.org/zh-cn/download) 下載最新版本。

### 版本選擇

- **LTS (長期支援) 版本**：推薦用於生產環境，提供穩定的 API 和長期支援
- **Current (當前) 版本**：包含最新功能，適合開發和測試環境

### Windows 安裝

1. 造訪 [Node.js 下載頁面](https://nodejs.org/zh-cn/download)
2. 下載 Windows 安裝程式 (.msi)
3. 執行安裝程式並依照精靈完成安裝
4. 驗證安裝：
   ```bash
   node --version
   npm --version
   ```

### macOS 安裝

#### 方法一：使用官方安裝套件
1. 下載 macOS 安裝套件 (.pkg)
2. 雙擊執行安裝程式
3. 依照精靈完成安裝

#### 方法二：使用 Homebrew
```bash
# 安裝 LTS 版本
brew install node@lts

# 或安裝最新版本
brew install node
```

### Linux 安裝

#### Ubuntu/Debian
```bash
# 使用套件管理器（可能不是最新版本）
sudo apt update
sudo apt install nodejs npm

# 或使用 NodeSource 倉儲取得最新版本
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs
```

#### CentOS/RHEL/Fedora
```bash
# 使用套件管理器
sudo dnf install nodejs npm

# 或使用 NodeSource 倉儲
curl -fsSL https://rpm.nodesource.com/setup_lts.x | sudo bash -
sudo yum install nodejs
```

### 驗證安裝

安裝完成後，驗證 Node.js 和 npm 是否正確安裝：

```bash
# 檢查 Node.js 版本
node --version

# 檢查 npm 版本  
npm --version

# 執行簡單的測試
node -e "console.log('Hello Node.js!')"
```

### 管理多個 Node.js 版本

在開發中可能需要在不同專案中使用不同的 Node.js 版本。推薦使用以下工具：

#### nvm (Node Version Manager)
- **macOS/Linux**: 
  ```bash
  curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
  ```
- **Windows**: 使用 [nvm-windows](https://github.com/coreybutler/nvm-windows)

常用指令：
```bash
# 列出可用版本
nvm list available

# 安裝特定版本
nvm install 18.17.0

# 切換版本
nvm use 18.17.0

# 設定預設版本
nvm alias default 18.17.0
```

### 環境變數設定

確保 Node.js 和 npm 的可執行檔路徑已新增到系統 PATH 環境變數中。

- **Windows**: 通常自動新增到 PATH
- **macOS/Linux**: 如果使用套件管理器安裝，通常會自動設定

如果指令未找到，請手動新增到 PATH：
```bash
# 通常位於 /usr/local/bin 或 ~/.nvm/versions/node/vXX.XX.XX/bin
export PATH=$PATH:/usr/local/bin
```

### 故障排除

#### 權限問題
在 Linux/macOS 上，如果遇到 npm 全域安裝的權限問題，建議：
- 使用 nvm 管理 Node.js
- 或設定 npm 的全域目錄：
  ```bash
  mkdir ~/.npm-global
  npm config set prefix '~/.npm-global'
  export PATH=~/.npm-global/bin:$PATH
  ```

#### 網路問題
如果下載速度慢或無法連線，可以設定 npm 使用國內鏡像：
```bash
# 設定淘寶鏡像
npm config set registry https://registry.npmmirror.com

# 或臨時使用
npm install --registry https://registry.npmmirror.com
```
