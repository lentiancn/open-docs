# Element UI Introduction

Element UI is a desktop UI component library for Vue 2, developed by Ele.me team.

## What is Element UI

Element UI is a component library designed for Vue 2, providing rich components for building enterprise admin dashboards.

## Main Features

- Vue 2 Deep Integration
- 70+ Components
- On-demand Import
- Internationalization
- Theme Customization

## Components

- Basic: Button, Icon, Layout
- Form: Input, Select, Date Picker
- Data Display: Table, Tree, Pagination
- Feedback: Dialog, Message

## Installation

```bash
npm install element-ui -S
```

## Usage

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```

## Relationship with ElementPlus

- Element UI: Vue 2 version
- ElementPlus: Vue 3 version

For Vue 3 projects, use ElementPlus.
