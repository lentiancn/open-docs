# Docker 使用指南

## 常用命令

```bash
# 鏡像
docker pull <image>    # 拉取鏡像
docker images          # 查看鏡像

# 容器
docker run -d <image> # 後台運行
docker ps              # 運行中容器
docker ps -a           # 所有容器
docker stop <id>       # 停止容器
docker rm <id>         # 刪除容器
docker logs -f <id>    # 查看日誌

# Docker Compose
docker-compose up -d   # 啟動
docker-compose down    # 停止
```

## 示例

```bash
docker run -d -p 80:80 nginx
```
