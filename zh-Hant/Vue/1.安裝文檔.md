# Vue 安裝指南

Vue 是一個漸進式 JavaScript 框架，用於構建用戶界面。

## 版本概述

Vue 有兩個主要版本在使用中：

| 版本 | 發布日期 | 狀態 |
|------|----------|------|
| Vue 3 | 2020年9月 | 當前推薦 |
| Vue 2 | 2016年10月 | 已停止維護 (2023年) |

**建議：** 所有新項目使用 Vue 3。Vue 2 於 2023 年停止維護。

### Vue 2 vs Vue 3 主要區別

| 特性 | Vue 2 | Vue 3 |
|------|-------|-------|
| 架構 | Options API | Composition API + Options API |
| 響應式 | Object.defineProperty | Proxy |
| TypeScript | 部分支持 | 完全支持 |
| 效能 | 好 | 更好 |
| 體積 | ~20KB gzip | ~10KB gzip |
| 生命週期 | created, mounted | setup + onMounted |
| Teleport | 不支援 | 支援 |
| Fragments | 不支援 | 支援 |
| 多根節點 | 不支援 | 支援 |

## 前置條件

### 系統需求

| 需求 | Vue 2 | Vue 3 |
|------|-------|-------|
| Node.js | 8.9+ | 16+ (推薦 18+) |
| npm | 5.6+ | 8+ |
| 瀏覽器支援 | IE9+ | 不支援 IE11 |

### 必需軟體

- **Node.js** - JavaScript 執行環境
- **npm** - 套件管理器（隨 Node.js 一起安裝）
- **Git** - 版本控制（可選但推薦）

## 安裝方法

### 方法 1：使用 npm（推薦）

#### Vue 3（推薦）

```bash
# 建立新 Vue 專案
npm create vue@latest

# 或使用 Vue CLI
npm install -g @vue/cli

# 建立專案
vue create my-vue-app
```

#### Vue 2（舊版）

```bash
# 安裝 Vue CLI
npm install -g @vue/cli

# 建立 Vue 2 專案
vue create my-vue2-app --default
```

### 方法 2：使用 Vite（推薦用於 Vue 3）

Vite 是一個現代構建工具，提供極快的開發伺服器。

```bash
# 使用 Vite 建立 Vue 3 專案
npm create vite@latest my-vue-app -- --template vue

# 進入專案
cd my-vue-app

# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev
```

### 方法 3：使用 CDN

#### Vue 3 CDN

```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
```

#### Vue 2 CDN

```html
<script src="https://unpkg.com/vue@2/dist/vue.js"></script>
```

### 方法 4：使用 Vue CLI

```bash
# 全域安裝 Vue CLI
npm install -g @vue/cli

# 驗證安裝
vue --version

# 建立專案
vue create my-project

# 或使用 GUI
vue ui
```

### 方法 5：使用 Nuxt.js

Nuxt 是一個用於建立 Vue 應用程式的框架。

```bash
# 建立 Nuxt 3 專案
npx nuxi@latest init my-nuxt-app

# 進入專案
cd my-nuxt-app

# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev
```

## 環境配置

### 安裝 Node.js

#### Windows

1. 從 https://nodejs.org 下載 Node.js
2. 執行安裝程式（推薦 LTS 版本）
3. 驗證：`node --version`

#### macOS

```bash
# 使用 Homebrew
brew install node
```

#### Linux (Ubuntu/Debian)

```bash
# 使用 NodeSource
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# 或使用 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 18
```

### 配置 npm

```bash
# 查看 npm 版本
npm --version

# 設定 npm 鏡像（中國區可選）
npm config set registry https://registry.npmmirror.com
```

## Vue Router 設定

### 安裝

```bash
npm install vue-router@4
```

### 基本配置（Vue 3）

```javascript
// src/router/index.js
import { createRouter, createWebHistory } from 'vue-router'
import Home from '../views/Home.vue'
import About from '../views/About.vue'

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```

## Pinia（狀態管理）

### 安裝

```bash
npm install pinia
```

### 基本配置（Vue 3）

```javascript
// src/stores/counter.js
import { defineStore } from 'pinia'

export const useCounterStore = defineStore('counter', {
  state: () => ({
    count: 0
  }),
  getters: {
    doubleCount: (state) => state.count * 2
  },
  actions: {
    increment() {
      this.count++
    }
  }
})
```

## Vue 2 Options API（旧版）

```javascript
new Vue({
  el: '#app',
  data() {
    return {
      message: 'Hello Vue 2!',
      items: []
    }
  },
  computed: {
    reversedMessage() {
      return this.message.split('').reverse().join('')
    }
  },
  methods: {
    handleClick() {
      console.log('Clicked!')
    }
  },
  created() {
    console.log('Component created')
  }
})
```

## Vue 3 Composition API

```javascript
import { ref, computed, onMounted } from 'vue'

export default {
  setup() {
    // 響應式狀態
    const message = ref('Hello Vue 3!')
    const count = ref(0)
    
    // 計算屬性
    const reversedMessage = computed(() => {
      return message.value.split('').reverse().join('')
    })
    
    // 方法
    const increment = () => {
      count.value++
    }
    
    // 生命週期
    onMounted(() => {
      console.log('Component mounted')
    })
    
    return {
      message,
      count,
      reversedMessage,
      increment
    }
  }
}
```

## 故障排除

### 常見問題

#### npm install 失敗

```bash
# 清除 npm 快取
npm cache clean --force

# 刪除 node_modules 並重新安裝
rm -rf node_modules
npm install
```

#### 連接埠被佔用

```bash
# 使用不同連接埠
npm run dev -- --port 3000
```

#### Node 版本問題

```bash
# 查看 Node 版本
node --version

# 使用 nvm 切換版本
nvm use 18
```

## 最佳實踐

1. **使用 Vue 3** - 除非有特定的 Vue 2 要求
2. **使用 Vite** - 更快的開發和建構
3. **使用 Composition API** - 更好的 TypeScript 支援和程式碼組織
4. **使用 TypeScript** - 更好的開發體驗
5. **使用 Pinia** - 新專案使用 Pinia 而不是 Vuex
