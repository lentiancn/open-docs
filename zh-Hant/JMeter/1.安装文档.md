# JMeter 安裝指南

JMeter 是 Apache 組織開發的開源效能測試工具。本指南涵蓋在各種環境中的安裝方法。

## 版本資訊

| 版本 | 狀態 | 說明 |
|------|------|------|
| JMeter 5.x | ✅ 目前 | 最新版本 |

## 系統需求

### 硬體需求

| 需求 | 配置 |
|------|------|
| CPU | 2+ 核心 |
| 記憶體 | 4+ GB |
| 磁碟 | 5+ GB |

### 軟體需求

| 軟體 | 版本 |
|------|------|
| Java | JDK 8 或更高 |

## 安裝方法

### 方法一：Windows 安裝

#### 1. 下載

1. 訪問 https://jmeter.apache.org/download.cgi
2. 下載 Binaries 版本
3. 解壓到指定目錄

#### 2. 執行

```powershell
# 雙擊 bin\jmeter.bat
# 或命令列執行
jmeter.bat
```

### 方法二：macOS 安裝

#### 1. 使用 Homebrew

```bash
brew install jmeter
```

#### 2. 手動安裝

```bash
# 下載並解壓
wget https://archive.apache.org/dist/jmeter/binaries/apache-jmeter-5.6.3.tgz
tar -xzf apache-jmeter-5.6.3.tgz

# 執行
./apache-jmeter-5.6.3/bin/jmeter
```

### 方法三：Linux 安裝

```bash
# Ubuntu/Debian
sudo apt-get install jmeter

# 或下載安裝
wget https://archive.apache.org/dist/jmeter/binaries/apache-jmeter-5.6.3.tgz
tar -xzf apache-jmeter-5.6.3.tgz
cd apache-jmeter-5.6.3/bin
./jmeter
```

### 方法四：Docker

```bash
# 執行 JMeter 容器
docker run -v $(pwd):/workspace justb4/jmeter -n -t test.jmx -l result.jtl
```

## 環境配置

### Java 配置

```bash
# 檢查 Java
java -version

# 設定環境變數
export JAVA_HOME=/path/to/java
export PATH=$JAVA_HOME/bin:$PATH
```

## 外掛安裝

### 方法一：使用外掛管理器

1. 下載 Plugins Manager
2. 放入 lib/ext 目錄
3. 重新啟動 JMeter

### 方法二：手動安裝

```bash
# 下載外掛
# 放入 lib/ext 目錄
```

## 常見問題

### 記憶體不足？

修改 jmeter.bat 或 jmeter：

```bash
# 設定堆大小
HEAP="-Xms1g -Xmx2g"
```

### GUI 啟動慢？

使用命令列模式或增加記憶體。
