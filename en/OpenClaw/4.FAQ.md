# OpenClaw FAQ

This page answers common questions about OpenClaw.

## Basics

### What is OpenClaw?

OpenClaw is a self-hosted gateway that connects your chat apps (WhatsApp, Telegram, Discord, etc.) to AI coding assistants. It lets you communicate with an AI assistant from anywhere while maintaining full control over your data.

### Is OpenClaw free?

Yes, OpenClaw is an open-source project under the MIT license. However, you'll need to cover:
- The server or computer to run it
- API key costs for AI services

### Do I need an AI service subscription?

OpenClaw supports multiple model providers, including:
- Anthropic (Claude)
- OpenAI (GPT)
- Google Gemini
- Any other OpenAI API-compatible service

You can use pay-as-you-go API keys or subscription authentication.

### What platforms does OpenClaw support?

- **Desktop**: macOS, Linux, Windows
- **Mobile**: iOS, Android (as nodes)
- **Chat Apps**: WhatsApp, Telegram, Discord, iMessage, Signal, Slack, and 20+ more platforms

## Installation

### What if installation fails?

1. Check Node.js version: `node --version` (requires 22 or higher)
2. Run diagnostics: `openclaw doctor`
3. Check logs: `openclaw logs`

### Can I run it on a Raspberry Pi?

Yes, OpenClaw works on Raspberry Pi. We recommend Raspberry Pi 4 or newer, with Raspberry Pi OS or Ubuntu.

### Do I need a Mac?

No, OpenClaw runs on any system that supports Node.js. Only the iMessage channel requires a Mac (due to Apple's restrictions).

## Configuration

### Where is the config file?

Default location: `~/.openclaw/openclaw.json`

### How do I control who can message?

Configure via `dmPolicy`:

- `pairing`: unknown senders need a pairing code
- `allowlist`: only users on the allowlist
- `open`: allow all direct messages
- `disabled`: disable direct messages

### How do I set up group chats?

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

## Usage

### How do I add a new chat channel?

```bash
openclaw channels login whatsapp
```

### Can I use multiple channels at once?

Yes, one Gateway can connect to multiple channels simultaneously.

### How do I make the AI remember previous conversations?

OpenClaw has built-in memory that automatically saves session context. You can adjust session retention policies through configuration.

### Can I run multiple agents?

Yes, configure multiple agents with routing rules:

```json5
{
  agents: {
    list: [
      { id: "home", default: true },
      { id: "work" }
    ]
  }
}
```

## Security

### Is my data secure?

Yes. OpenClaw is self-hosted — all data stays on your device and isn't sent to third-party servers (except to AI provider APIs).

### How do I protect my Gateway?

1. Set `allowFrom` to restrict access sources
2. Use strong passwords and API keys
3. Keep OpenClaw updated
4. Use firewall to restrict port access

## Troubleshooting

### Gateway fails to start

1. Check if port is in use: `openclaw gateway status`
2. Check error logs: `openclaw logs`
3. Run diagnostics: `openclaw doctor`

### AI doesn't reply

1. Verify API key is configured correctly
2. Check your model provider account has sufficient balance
3. Look for error messages in logs

### Message sending fails

1. Check if channel is logged in
2. Verify target user or group exists
3. Check detailed error logs
