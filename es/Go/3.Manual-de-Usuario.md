## Funciones

```go
func sumar(a, b int) int {
    return a + b
}

func dividir(a, b int) (int, error) {
    if b == 0 {
        return 0, errors.New("no se puede dividir por cero")
    }
    return a / b, nil
}
```

## Concurrencia

### Goroutine

```go
func decir(s string) {
    for i := 0; i < 3; i++ {
        fmt.Println(s)
    }
}

func main() {
    go decir("Hola")
    decir("Mundo")
    time.Sleep(time.Second)
}
```

### Channel

```go
ch := make(chan string)
go func() { ch <- "mensaje" }()
msg := <-ch
```

## Manejo de errores

```go
resultado, err := dividir(10, 0)

    fmt.Println("Error:", err)
}
```
