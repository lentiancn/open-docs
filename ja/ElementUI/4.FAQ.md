# Element UI FAQ

一般的な質問と回答。

## インストール問題

### Q: インストールに失敗した？

**解決策：**
1. Node.jsバージョンを確認
```bash
node --version  # 12.0.0+が必要
```
2. npmキャッシュをクリア
```bash
npm cache clean --force
```
3. タオバオミラーを使う
```bash
npm config set registry https://registry.npmmirror.com
```
4. node_modulesを削除して再インストール
```bash
rm -rf node_modules
npm install
```

### Q: モジュールが見つからない？

**解決策：**
1. element-uiがインストールされているか確認
```bash
npm list element-ui
```
2. package.jsonの依存関係を確認

### Q: Vueバージョンが合わない？

**解決策：**
Element UI 2.xはVue 2.6.0+が必要：
```bash
npm install vue@2
```

## 使用問題

### Q: コンポーネントが表示されない？

**解決策：**
1. コンポーネントが正しくインポートされているか確認
```javascript
import { Button } from 'element-ui'
Vue.use(Button)
```
2. コンソールのエラーを確認
3. コンポーネント名が正しいか確認

### Q: スタイルが適用されない？

**解決策：**
1. スタイルファイルがインポートされているか確認
```javascript
import 'element-ui/lib/theme-chalk/index.css'
```
2. グローバルスタイルの上書きがないか確認
3. z-indexレベルを確認

### Q: テーブルにデータが表示されない？

**解決策：**
1. データフォーマットを確認
2. 列定義を確認
3. データがリアクティブか確認

### Q: フォームバリデーションが機能しない？

**解決策：**
1. FormとFormItemをインポート
2. prop属性を設定
3. dataにrulesを定義

### Q: ドロップダウンオプションが表示されない？

**解決策：**
1. optionsデータを確認
2. valueとlabelのマッピングを確認

### Q: ダイアログを閉じた後データがリセットされない？

**解決策：**
1. v-ifでダイアログを制御
2. 閉じる時に手動でフォームをリセット

## スタイル問題

### Q: スタイルが競合する？

**解決策：**
1. より具体的なセレクターを使用
2. z-indexレベルを確認

### Q: アイコンが表示されない？

**解決策：**
1. element-ui-iconsがインストールされているか確認
```bash
npm install element-ui-icons
```
2. 正しくインポート

### Q: テーマカラーが正しくない？

**解決策：**
1. グローバルスタイルの上書きを確認
2. SCSS変数のインポート順を確認
3. ブラウザキャッシュをクリア

## 国際化問題

### Q: 言語切り替えが機能しない？

**解決策：**
1. i18nを正しく設定
2. 言語パックがインポートされているか確認

## パフォーマンス問題

### Q: ページの読み込みが遅い？

**解決策：**
1. オンデマンドインポートを使用
2. CDNインポートを使用
3. Gzip圧縮を有効化

### Q: 大量データテーブルが重い？

**解決策：**
1. 仮想スクロールを使用
2. ページごとにデータを読み込み
3. 不要な機能を無効化

## ビルド問題

### Q: ビルド後スタイルが失われる？

**解決策：**
1. vue.config.js設定を確認
2. スタイルのインポート位置を確認

### Q: 本番環境でコンポーネントが表示されない？

**解決策：**
1. Vue.use()が呼ばれているか確認
2. オンデマンドインポートが完全か確認

## アップグレード問題

### Q: 1.xから2.xへアップグレード方法は？

**解決策：**
1. 公式マイグレーションガイドを読む
2. 非互換APIを段階的に置き換え
3. テストを実行

### Q: Vue 3プロジェクトの場合？

Element UIはVue 2向け。Vue 3プロジェクトはElement Plusを使用：
```bash
npm install element-plus
```
