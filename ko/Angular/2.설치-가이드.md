# Angular 설치 가이드

완전한 Angular 개발 환경 설정.

## 환경 요구사항

### 하드웨어 요구사항

- CPU: 2코어 이상
- 메모리: 4GB 이상
- 디스크: 10GB 이상

### 소프트웨어 요구사항

- 운영체제: Windows/macOS/Linux
- Node.js: 18.10.0+ 또는 20.0.0+
- npm: 9.0.0+ (Node.js에 포함)
- Git: 모든 버전

## 설치 단계

### 1. Node.js 설치

**Windows/macOS:**

https://nodejs.org 에서 다운로드

또는 nvm 사용:

```bash
# nvm 설치 (macOS/Linux)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Node.js LTS 설치
nvm install --lts
nvm use --lts
```

**설치 확인:**

```bash
node --version  # v18.x.x 또는 v20.x.x 표시
npm --version    # 9.x.x 이상 표시
```

### 2. Angular CLI 설치

```bash
# npm으로 전역 설치
npm install -g @angular/cli

# 설치 확인
ng version
```

### 3. 새 프로젝트 생성

```bash
# 새 Angular 애플리케이션 생성
ng new my-app

# 프로젝트로 이동
cd my-app

# 개발 서버 시작
ng serve
```

### 4. 애플리케이션 접근

브라우저 열기: http://localhost:4200

## 일반적인 명령

### 프로젝트 명령

```bash
# 라우팅과 SCSS로 새 프로젝트 생성
ng new my-app --routing --style=scss

# 스탠드얼론 앱 생성 (Angular 14+)
ng new my-app --standalone

# 개발 서버 시작
ng serve

# 프로덕션 빌드
ng build

# 단위 테스트 실행
ng test

# E2E 테스트 실행
ng e2e
```

### 생성 명령

```bash
# 컴포넌트 생성
ng generate component my-component
ng g c my-component

# 서비스 생성
ng generate service my-service
ng g s my-service

# 디렉티브 생성
ng generate directive my-directive
ng g d my-directive

# 파이프 생성
ng generate pipe my-pipe
ng g p my-pipe

# 모듈 생성
ng generate module my-module
ng g m my-module

# 클래스 생성
ng generate class my-class
ng g cl my-class

# 인터페이스 생성
ng generate interface my-interface
ng g i my-interface

# 가드 생성
ng generate guard my-guard
ng g g my-guard

# 인터셉터 생성
ng generate interceptor my-interceptor
ng g in my-interceptor
```

## 환경 설정

### 개발 환경

```typescript
// src/environments/environment.ts
export const environment = {
  production: false,
  apiUrl: 'http://localhost:3000/api'
};
```

### 프로덕션 환경

```typescript
// src/environments/environment.prod.ts
export const environment = {
  production: true,
  apiUrl: 'https://api.example.com'
};
```

### 환경 사용

```typescript
import { Component } from '@angular/core';
import { environment } from '../environments/environment';

@Component({
  selector: 'app-root',
  template: `<p>API: {{ apiUrl }}</p>`
})
export class AppComponent {
  apiUrl = environment.apiUrl;
}
```

## 일반적인 문제

### npm이 느리다?

```bash
# 타오바오 미러 사용
npm config set registry https://registry.npmmirror.com

# 또는 nrm으로 미러 관리
npm install -g nrm
nrm use taobao
```

### Node 버전이 안 맞는다?

```bash
# nvm으로 버전 전환
nvm install 20
nvm use 20
```

### 권한 문제 (Linux/macOS)?

```bash
# npm 권한 수정
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc
```

## 제거

```bash
# Angular CLI 제거
npm uninstall -g @angular/cli

# 프로젝트 삭제
rm -rf my-app
```
