# JDK User Manual

## Compile and Run

```bash
# Compile Java file
javac HelloWorld.java

# Run Java program
java HelloWorld
```

## Common Commands

```bash
# View class information
javap HelloWorld

# Generate documentation
javadoc HelloWorld.java

# Package JAR
jar cvf Hello.jar Hello.class

# Run in debug mode
java -agentlib:jdwp=transport=dt_socket,server=y,suspend=n Hello
```

## Environment Variables

```bash
# Windows
set JAVA_HOME=C:\Program Files\Java\jdk-21
set PATH=%JAVA_HOME%\bin;%PATH%

# Linux/macOS
export JAVA_HOME=/usr/lib/jvm/java-21-openjdk
export PATH=$JAVA_HOME/bin:$PATH
```

## Memory Parameters

```bash
# Set heap memory
java -Xmx2048m -Xms512m Hello

# Set young generation
java -Xmn256m -Xmx2048m Hello
```
