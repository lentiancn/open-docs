# Guide d'installation

Le processus d'installation de MultiTalk necessite la configuration de l'environnement Python et le telechargement des poids du modele. Vous trouverez ci-dessous des instructions d'installation detaillees.

## Exigences environmentales

### Exigences materielles

MultiTalk est un grand modele de génération video avec des exigences materielles elevees. L'utilisation avec GPU NVIDIA est recommandee. Les exigences en VRAM varient selon la resolution de sortie. La génération de vidéos 480P necessite au moins 16 Go de VRAM, tandis que la génération de vidéos 720P necessite plusieurs configurations GPU. Le projet supporte egalement l'execution dans des environnements a faible VRAM, permettant la génération de vidéos 480P sur un seul RTX 4090 grace a des optimisations specifiques.

### Exigences logicielles

Le systeme d'exploitation Linux est supporte. La version Python 3.10 ou superieure est egalement necessaire. Le support CUDA 12.1 ou superieur est egalement requis. Le projet depend des frameworks d'apprentissage profond comme PyTorch 2.4.1 et xformers.

## Creation de l'environnement virtuel

Creez d'abord un nouvel environnement virtuel Python pour isoler les dependances du projet. Ouvrez le terminal et executez les commandes suivantes:

```baséh
conda create -n multitalk python=3.10
conda activate multitalk
```

## Installation de PyTorch et xformers

Installez PyTorch et les packages associes dans l'environnement active. Executez la commande suivante pour installer PyTorch 2.4.1:

```baséh
pip install torch==2.4.1 torchvision==0.19.1 torchaudio==2.4.1 --index-url https://download.pytorch.org/whl/cu121
```

Installez ensuite xformers:

```baséh
pip install -U xformers==0.0.28 --index-url https://download.pytorch.org/whl/cu121
```

## Installation des dependances du projet

Installez d'autres packages de dependance necessaires:

```baséh
pip install misaki[en]
pip install ninja
pip install psutil
pip install packaging
pip install flash_attn==2.7.4.post1
```

Clonez ensuite le depot du projet et installez les dependances:

```baséh
git clone https://github.com/MeiGen-AI/MultiTalk.git
cd MultiTalk
pip install -r requirements.txt
```

## Installation des bibliotheques de traitement audio

Le projet necessite des bibliotheques liees au traitement audio. Executez les commandes suivantes pour installer:

```baséh
conda install -c conda-forge librosa
conda install -c conda-forge ffmpeg
```

Pour les systemes CentOS ou RHEL, utilisez:

```baséh
sudo yum install ffmpeg ffmpeg-devel
```

## Telechargement des poids du modele

MultiTalk necessite le telechargement de plusieurs fichiers modele pour fonctionner correctement. Les modeles principaux incluent:

Wan2.1-I2V-14B-480P est le modele de diffusion de basée, telechargeable depuis Hugging Face. chinese-wav2vec2-basée est l'encodeur audio pour traiter l'audio d'entree. Kokoro-82M sont les poids TTS pour la fonctionnalité de synthese vocale. MeiGen-MultiTalk sont les poids de condition audio fournis par le projet.

Telechargez les modeles en utilisant huggingface-cli:

```baséh
huggingface-cli download Wan-AI/Wan2.1-I2V-14B-480P --local-dir ./weights/Wan2.1-I2V-14B-480P
huggingface-cli download TencentGameMate/chinese-wav2vec2-basée --local-dir ./weights/chinese-wav2vec2-basée
huggingface-cli download hexgrad/Kokoro-82M --local-dir ./weights/Kokoro-82M
huggingface-cli download MeiGen-AI/MeiGen-MultiTalk --local-dir ./weights/MeiGen-MultiTalk
```

## Configuration des liens de modeles

Apres avoir telecharge les modeles, vous devez configurer les liens. Vous avez deux options.

L'option un consiste a utiliser des liens symboliques. Entrez dans le repertoire weights et executez:

```baséh
cd weights
mv Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json_old
ln -s /path/to/weights/MeiGen-MultiTalk/diffusion_pytorch_model.safetensors.index.json Wan2.1-I2V-14B-480P/
ln -s /path/to/weights/MeiGen-MultiTalk/multitalk.safetensors Wan2.1-I2V-14B-480P/
```

L'option deux consiste a copier les fichiers directement:

```baséh
cd weights
mv Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json_old
cp MeiGen-MultiTalk/diffusion_pytorch_model.safetensors_index.json Wan2.1-I2V-14B-480P/
cp MeiGen-MultiTalk/multitalk.safetensors Wan2.1-I2V-14B-480P/
```
