# jQuery Benutzerhandbuch

## Schnellstart

### jQuery einbinden

Binden Sie jQuery in den `<head>` oder am Ende des `<body>` in Ihrer HTML-Datei ein:

```html
<!DOCTYPE html>
<html>
<head>
    <title>jQuery Demo</title>
    <script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
</head>
<body>
    <!-- Seiteninhalt -->
</body>
</html>
```

### Auf DOM-Bereitschaft warten

jQuery-Code muss ausgeführt werden, nachdem das DOM vollständig geladen wurde:

```javascript
$(document).ready(function() {
    // jQuery-Code hier schreiben
});

// Kurzform
$(function() {
    // jQuery-Code hier schreiben
});
```

## Selektoren

jQuery-Selektoren sind eines der Kernmerkmale von jQuery, mit fast identischer Syntax zu CSS-Selektoren.

### Basis-Selektoren

```javascript
$("#myId")          // ID-Selektor
$(".myClass")       // Klassen-Selektor
$("div")            // Element-Selektor
$("*")              // Universeller Selektor
$("div, p, span")   // Mehrfach-Selektoren
```

### Attribut-Selektoren

```javascript
$('[href]')                    // Elemente mit href-Attribut
$('[href="value"]')           // href gleich dem angegebenen Wert
$('[href!="value"]')          // href nicht gleich dem angegebenen Wert
$('[href^="http"]')           // href beginnt mit dem angegebenen Wert
$('[href$=".pdf"]')           // href endet mit dem angegebenen Wert
$('[href*="google"]')         // href enthält den angegebenen Wert
```

### Filter-Selektoren

```javascript
$("div:first")        // Erstes div-Element
$("div:last")         // Letztes div-Element
$("div:even")         // Geradzahlige div-Elemente
$("div:odd")          // Ungeradzahlige div-Elemente
$("div:eq(n)")        // Div bei Index n
$("div:gt(n)")        // Divs mit Index größer als n
$("div:lt(n)")        // Divs mit Index kleiner als n
$("div:not(.exclude") // Divs ohne angegebene Klasse
```

## DOM-Manipulation

### Inhalt abrufen und setzen

```javascript
// Textinhalt abrufen/setzen
var text = $("#myElement").text();
$("#myElement").text("Neuer Textinhalt");

// HTML-Inhalt abrufen/setzen
var html = $("#myElement").html();
$("#myElement").html("<p>Neuer HTML-Inhalt</p>");

// Formularwert abrufen/setzen
var value = $("#inputField").val();
$("#inputField").val("Neuer Wert");
```

### Attribute abrufen und setzen

```javascript
// Attribut abrufen
var src = $("#myImage").attr("src");

// Einzelnes Attribut setzen
$("#myImage").attr("src", "image.jpg");

// Mehrere Attribute setzen
$("#myImage").attr({
    src: "image.jpg",
    alt: "Bildbeschreibung",
    title: "Bildtitel"
});

// Attribut entfernen
$("#myImage").removeAttr("title");
```

### CSS abrufen und setzen

```javascript
// Einzelne CSS-Eigenschaft abrufen
var color = $("#myElement").css("color");

// Einzelne CSS-Eigenschaft setzen
$("#myElement").css("color", "red");

// Mehrere CSS-Eigenschaften setzen
$("#myElement").css({
    color: "red",
    backgroundColor: "blue",
    fontSize: "16px"
});

// Klassen hinzufügen/entfernen
$("#myElement").addClass("active highlight");
$("#myElement").removeClass("highlight");
$("#myElement").toggleClass("active");  // Klasse umschalten
```

### DOM-Traversierung

```javascript
// Eltern-Elemente
$("#myElement").parent();
$("#myElement").parents();
$("#myElement").closest(".container");

// Kind-Elemente
$("#myElement").children();
$("#myElement").find(".child");

// Geschwister-Elemente
$("#myElement").siblings();
$("#myElement").next();
$("#myElement").prev();
$("#myElement").nextAll();
```

### Elemente erstellen und entfernen

```javascript
// Neues Element erstellen
var newDiv = $("<div>", {
    class: "new-element",
    text: "Neuer Elementinhalt"
});

// Zur Seite hinzufügen
$("#container").append(newDiv);      // Innerhalb, am Ende
$("#container").prepend(newDiv);    // Innerhalb, am Anfang
$("#container").before(newDiv);      // Außerhalb, davor
$("#container").after(newDiv);       // Außerhalb, danach

// Elemente entfernen
$("#myElement").remove();             // Element und Kinder entfernen
$("#myElement").empty();             // Nur Kinder entfernen
```

## Event-Handling

### Events binden

```javascript
// Klick-Event binden
$("#myButton").click(function() {
    alert("Button geklickt");
});

// Mit on-Methode binden
$("#myButton").on("click", function() {
    alert("Button geklickt");
});

// Mehrere Events binden
$("#myElement").on({
    mouseenter: function() {
        $(this).css("background", "yellow");
    },
    mouseleave: function() {
        $(this).css("background", "");
    }
});
```

### Event-Delegation

```javascript
// Event-Bindung für dynamische Elemente
$("#container").on("click", ".dynamic-item", function() {
    alert("Dynamisches Element geklickt");
});
```

### Häufige Events

| Event | Beschreibung |
|-------|--------------|
| click | Klick-Event |
| dblclick | Doppelklick-Event |
| mouseenter | Maus betritt |
| mouseleave | Maus verlässt |
| mousemove | Maus bewegt sich |
| keydown | Taste gedrückt |
| keyup | Taste losgelassen |
| focus | Fokus erhalten |
| blur | Fokus verloren |
| submit | Formular gesendet |
| change | Wert geändert |
| load | Geladen |

## Animationseffekte

### Basis-Animationen

```javascript
// Element anzeigen
$("#myElement").show();

// Element ausblenden
$("#myElement").hide();

// Sichtbarkeit umschalten
$("#myElement").toggle();

// Einblenden
$("#myElement").fadeIn();

// Ausblenden
$("#myElement").fadeOut();

// Fade umschalten
$("#myElement").fadeToggle();

// Einschieben
$("#myElement").slideDown();

// Ausschieben
$("#myElement").slideUp();

// Slide umschalten
$("#myElement").slideToggle();
```

### Benutzerdefinierte Animationen

```javascript
$("#myElement").animate({
    opacity: 0.5,
    left: "+=50",
    height: "200px"
}, 300, function() {
    // Callback nach Animation
    console.log("Animation abgeschlossen");
});
```

### Animationen stoppen

```javascript
// Aktuelle Animation stoppen
$("#myElement").stop();

// Alle Animationen stoppen und Endzustand erreichen
$("#myElement").stop(true, true);
```

## Ajax-Anfragen

### Grundlegende Verwendung

```javascript
$.ajax({
    url: "/api/data",
    method: "GET",
    data: { id: 1, name: "test" },
    success: function(response) {
        console.log("Erfolg:", response);
    },
    error: function(xhr, status, error) {
        console.log("Fehler:", error);
    }
});
```

### Kurzmethoden

```javascript
// GET-Anfrage
$.get("/api/data", { id: 1 }, function(response) {
    console.log(response);
});

// POST-Anfrage
$.post("/api/submit", { name: "test" }, function(response) {
    console.log(response);
});

// HTML laden
$("#content").load("fragment.html");
```

### Vollständiges Beispiel

```javascript
$.ajax({
    url: "https://api.example.com/users",
    type: "GET",
    dataType: "json",
    timeout: 5000,
    headers: {
        "Authorization": "Bearer token"
    },
    beforeSend: function(xhr) {
        // Vor dem Senden der Anfrage
        console.log("Anfrage wird gesendet");
    },
    success: function(data) {
        // Anfrage erfolgreich
        $("#user-list").html("");
        data.forEach(function(user) {
            $("#user-list").append("<li>" + user.name + "</li>");
        });
    },
    error: function(xhr, status, error) {
        // Anfrage fehlgeschlagen
        alert("Laden fehlgeschlagen: " + error);
    },
    complete: function(xhr, status) {
        // Anfrage abgeschlossen (Erfolg oder Fehler)
        console.log("Anfrage abgeschlossen, Status: " + status);
    }
});
```

## Hilfsfunktionen

### Array- und Objektoperationen

```javascript
// Array durchlaufen
$.each([1, 2, 3], function(index, value) {
    console.log(index + ": " + value);
});

// Objekt durchlaufen
$.each({ name: "John", age: 30 }, function(key, value) {
    console.log(key + ": " + value);
});

// Array filtern
var filtered = $.grep([1, 2, 3, 4, 5], function(value) {
    return value > 3;
});

// Array abbilden
var mapped = $.map([1, 2, 3], function(value) {
    return value * 2;
});

// Objekte zusammenführen
var merged = $.extend({}, obj1, obj2);
```

### Zeichenkettenoperationen

```javascript
// Leerzeichen entfernen
$.trim("  hello  ");  // "hello"

// Spezialzeichen escapen
$.escapeSelector(".myClass");  // "\.myClass"
```

### Typprüfung

```javascript
$.isArray([]);        // true
$.isFunction(fn);     // true
$.isEmptyObject({});  // true
$.isPlainObject({});  // true
```

## Verkettung

jQuery unterstützt Verkettung, sodass Sie kontinuierlich am selben Element arbeiten können:

```javascript
$("#myElement")
    .addClass("highlight")
    .css("color", "red")
    .fadeIn()
    .animate({ height: "100px" }, function() {
        console.log("Fertig");
    });
```

Verwenden Sie die `end()`-Methode, um zum vorherigen Selektor zurückzukehren:

```javascript
$("#myElement")
    .find(".child")
    .addClass("active")
    .end()    // Zurück zu #myElement
    .removeClass("highlight");
```
