# Element Plus

> Библиотека Vue 3 UI-компонентов для настольных компьютеров

---

## Описание проекта

Element Plus — это библиотека Vue 3 UI-компонентов для настольных компьютеров, разработанная командой Element. Она полностью написана в стиле Composition API Vue 3 и полностью переписана на TypeScript. Первая стабильная версия была выпущена 7 февраля 2022 года. После многих лет итераций API стал очень стабильным.

Element Plus предоставляет богатый набор базовых компонентов, включая кнопки, формы, таблицы, диалоги, навигационные меню и многое другое. Позволяет быстро создавать корпоративные приложения для бэкенда.

## Быстрый старт

### Установка

```bash
npm install element-plus
```

### Импорт

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)

app.mount('#app')
```
