# Bootstrap

Bootstrapは、世界で最も使われているHTML、CSS、JavaScriptのオープンソース前端開発フレームワークです。Twitter社が開発・メンテナンスしています。豊富なコンポーネントと強力なJavaScriptプラグインが揃っており、開発者が素早くレスポンシブなWebアプリケーションを構築することを助けます。

## クイックスタート

### CDNを読み込む

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### npmでインストール

```bash
npm install bootstrap@5
```

## 主な機能

- **レスポンシブレイアウト**：12列グリッドシステムで様々な画面に対応
- **豊富なコンポーネント**：ナビゲーションバー、ボタン、フォーム、モーダルなど
- **JavaScriptプラグイン**：ドロップダウン、スライダー、アコーディオンなど
- **Sassサポート**：テーマを簡単にカスタマイズ
