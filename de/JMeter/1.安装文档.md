# JMeter Installationsanleitung

JMeter ist ein Open-Source-Leistungstest-Tool, das von der Apache-Organisation entwickelt wurde. Diese Anleitung deckt die Installationsmethoden in verschiedenen Umgebungen ab.

## Versionsinformationen

| Version | Status | Beschreibung |
|------|------|------|
| JMeter 5.x | ✅ Aktuell | Neueste Version |

## Systemanforderungen

### Hardwareanforderungen

| Anforderung | Konfiguration |
|------|------|
| CPU | 2+ Kerne |
| Speicher | 4+ GB |
| Festplatte | 5+ GB |

### Softwareanforderungen

| Software | Version |
|------|------|
| Java | JDK 8 oder höher |

## Installationsmethoden

### Methode 1: Windows-Installation

#### 1. Herunterladen

1. Besuchen Sie https://jmeter.apache.org/download.cgi
2. Laden Sie die Binaries-Version herunter
3. Entpacken Sie in das gewünschte Verzeichnis

#### 2. Ausführung

```powershell
# Doppelklick auf bin\jmeter.bat
# Oder über Befehlszeile ausführen
jmeter.bat
```

### Methode 2: macOS-Installation

#### 1. Homebrew verwenden

```bash
brew install jmeter
```

#### 2. Manuelle Installation

```bash
# Herunterladen und entpacken
wget https://archive.apache.org/dist/jmeter/binaries/apache-jmeter-5.6.3.tgz
tar -xzf apache-jmeter-5.6.3.tgz

# Ausführen
./apache-jmeter-5.6.3/bin/jmeter
```

### Methode 3: Linux-Installation

```bash
# Ubuntu/Debian
sudo apt-get install jmeter

# Oder herunterladen und installieren
wget https://archive.apache.org/dist/jmeter/binaries/apache-jmeter-5.6.3.tgz
tar -xzf apache-jmeter-5.6.3.tgz
cd apache-jmeter-5.6.3/bin
./jmeter
```

### Methode 4: Docker

```bash
# JMeter-Container ausführen
docker run -v $(pwd):/workspace justb4/jmeter -n -t test.jmx -l result.jtl
```

## Umgebungskonfiguration

### Java-Konfiguration

```bash
# Java überprüfen
java -version

# Umgebungsvariablen setzen
export JAVA_HOME=/path/to/java
export PATH=$JAVA_HOME/bin:$PATH
```

## Plugin-Installation

### Methode 1: Plugin-Manager verwenden

1. Plugins Manager herunterladen
2. In das Verzeichnis lib/ext legen
3. JMeter neu starten

### Methode 2: Manuelle Installation

```bash
# Plugin herunterladen
# In das Verzeichnis lib/ext legen
```

## Häufig gestellte Fragen

### Nicht genügend Speicher?

Bearbeiten Sie jmeter.bat oder jmeter:

```bash
# Heap-Größe einstellen
HEAP="-Xms1g -Xmx2g"
```

### Langsamer GUI-Start?

Verwenden Sie den Befehlszeilenmodus oder erhöhen Sie den Speicher.
