# Java

## 概述

Java 是由 Oracle 公司维护的面向对象编程语言，是全球最流行的企业级开发语言之一。Java 以"一次编写，到处运行"的特性著称，拥有成熟的生态系统和广泛的应用领域。

## 主要特性

- 面向对象：完全面向对象的编程语言
- 平台无关：通过 JVM 实现跨平台运行
- 自动内存管理：垃圾回收机制
- 丰富的类库：庞大的标准 API
- 多线程支持：内置并发编程支持
- 安全性：多层安全机制

## 快速开始

### 安装 JDK

访问 Oracle 官方下载页面获取 JDK：https://www.oracle.com/java/technologies/downloads/

### 编写第一个程序

创建 HelloWorld.java：

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

### 编译与运行

```bash
# 编译
javac HelloWorld.java

# 运行
java HelloWorld
```
