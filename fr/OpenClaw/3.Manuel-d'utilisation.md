# Manuel utilisateur OpenClaw

## Concepts

### Gateway

Le cœur du système. Gère les messages, appels IA et connexions.

### Canaux

Apps de chat: WhatsApp, Telegram, Discord, etc.

### Agents

Instances indépendantes d'assistants IA. Possibles plusieurs.

## Opérations

```bash
openclaw gateway start
openclaw gateway stop
openclaw message send --target +15555550123 --message "Bonjour"
```

## Config

Fichier: `~/.openclaw/openclaw.json`

```json5
{
  channels: { whatsapp: { allowFrom: ["+1234567890"] } }
}
```

## Problèmes

```bash
openclaw logs
openclaw doctor
```
