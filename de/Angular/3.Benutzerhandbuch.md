# Angular Benutzerhandbuch

Angular Kernkonzepte und Grundnutzung.

## Komponenten

### Komponente erstellen

```bash
ng generate component my-component
# oder
ng g c my-component
```

### Komponentenstruktur

```typescript
import { Component } from '@angular/core';

@Component({
  selector: 'app-my-component',
  templateUrl: './my-component.component.html',
  styleUrls: ['./my-component.component.css']
})
export class MyComponentComponent {
  title = 'Hello Angular';
}
```

## Datenbindung

### Eigenschaftsbindung

```html
<img [src]="imageUrl">
```

### Ereignisbindung

```html
<button (click)="onClick()">Klick</button>
```

### Bidirektionale Bindung

```html
<input [(ngModel)]="name">
```

## Dienste

### Dienst erstellen

```bash
ng generate service my-service
# oder
ng g s my-service
```

### Dienste verwenden

```typescript
import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root'
})
export class MyServiceService {
  getData() {
    return 'Daten';
  }
}
```

## Routing

### Routen konfigurieren

```typescript
const routes: Routes = [
  { path: '', component: HomeComponent },
  { path: 'about', component: AboutComponent },
  { path: '**', component: NotFoundComponent }
];
```

### Routen-Navigation

```html
<a routerLink="/about">Über</a>
<router-outlet></router-outlet>
```

## HTTP-Anfragen

### HttpClient importieren

```typescript
import { HttpClientModule } from '@angular/common/http';
```

### Anfragen senden

```typescript
import { HttpClient } from '@angular/common/http';

constructor(private http: HttpClient) {}

getData() {
  return this.http.get('https://api.example.com/data');
}
```

## Formulare

### Reaktive Formulare

```typescript
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

form: FormGroup;

constructor(private fb: FormBuilder) {
  this.form = this.fb.group({
    name: ['', Validators.required],
    email: ['', [Validators.required, Validators.email]]
  });
}
```

## Lebenszyklus-Hooks

| Hook | Beschreibung |
|------|--------------|
| ngOnInit | Komponenten-Initialisierung |
| ngOnChanges | Eingabeeigenschaften ändern sich |
| ngOnDestroy | Komponenten-Zerstörung |

## Direktiven

### Strukturelle Direktiven

```html
<div *ngIf="show">Anzeigen</div>
<div *ngFor="let item of items">{{item}}</div>
```

### Attribut-Direktiven

```html
<div [ngClass]="{'active': isActive}"></div>
```

## Best Practices

1. TypeScript starke Typisierung verwenden
2. Komponenten einfach halten
3. Dienste für gemeinsame Logik verwenden
4. Subscriptions kündigen
5. OnPush-Änderungserkennungsstrategie verwenden
