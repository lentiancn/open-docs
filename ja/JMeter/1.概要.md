# Apache JMeter - 概要

## Apache JMeterとは？

Apache JMeter™はオープンソースソフトウェアであり、パフォーマンステストと負荷テストのために設計された100％Pure Javaアプリケーションです。 元々はWebアプリケーションのテスト用に設計されていましたが、他のテスト機能にも拡張されています。

## JMeterでできること

Apache JMeterは、静的リソースと動的リソース、Web動的アプリケーションのパフォーマンステストに使用できます。 サーバー、サーバーグループ、ネットワーク、またはオブジェクトに重い負荷をシミュレートして、強度 різні типиの負荷の下での全体的なパフォーマンスを分析できます。

### 主な機能

- **マルチプロトコルサポート**: Web（HTTP、HTTPS）、SOAP/REST Webサービス、FTP、JDBC経由のデータベース、LDAP、JMS、メール、TCP、Javaオブジェクト
- **完全なテストIDE**: ブラウザまたはネイティブアプリケーションからテスト plansを記録
- **CLIモード**: 任意のJava対応OSから負荷テストを実行
- **動的HTMLレポート**: 完全でプレゼンテーション可能な形式
- **データ抽出**: HTML、JSON、XML、正規表現のサポート
- **100％Java**: 完全な移植性
- **マルチスレッド**: 多くのスレッドによる同時サンプリング
- **高い拡張性**: プラグイン可能なサンプラー、スクリプト可能なサンプラー（Groovy、BeanShell）
- **CI統合**: Maven、Gradle、Jenkinsのサポート

## JMeterができないこと

JMeterはブラウザではありません-プロトコルレベルで動作します。 HTMLページで見つかるJavaScriptを実行せず、実際のブラウザのようにHTMLページをレンダリングしません。

## はじめに

1. 公式サイトからJMeterをダウンロード
2. JMeterスクリプトを実行（Windowsはjmeter.bat、Linux/Macはjmeter.sh）
3. GUIを使用してテスト planを作成
4. テストを実行して結果を分析
