# Element Plus FAQ

一般的な質問と回答。

## インストール

### Q: インストールに失敗する？

**解決策:**
1. Node.jsバージョンを確認
```bash
node --version  # 18.0+が必要
```
2. npmキャッシュをクリア
```bash
npm cache clean --force
```
3. node_modulesとpackage-lock.jsonを削除して再インストール
```bash
rm -rf node_modules package-lock.json
npm install
```

### Q: モジュールが見つからない？

**解決策:**
1. パッケージがインストールされているか確認
```bash
npm list element-plus
```
2. npm/yarn/pnpm設定を確認
3. lockファイルを削除して再インストールを試す

### Q: TypeScriptの型エラー？

**解決策:**
1. 型定義をインストール
```bash
npm install -D @types/node
```
2. tsconfig.jsonに追加
```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

## 使用

### Q: コンポーネントが表示されない？

**解決策:**
1. 正しくインポートされているか確認
2. コンソールのエラーメッセージを確認
3. Vueバージョン（3.2+が必要）を確認

### Q: スタイルが適用されない？

**解決策:**
1. CSSがインポートされているか確認
```typescript
import 'element-plus/dist/index.css'
```
2. CSS優先度を確認し、`:deep()`または`!important`を使用
3. 他のスタイルで上書きされていないか確認

### Q: アイコンが表示されない？

**解決策:**
1. アイコンパッケージをインストール
```bash
npm install @element-plus/icons-vue
```
2. グローバルに登録
```typescript
import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```
3. または必要に応じてインポート

### Q: Tableがレンダリングされない？

**解決策:**
1. dataプロパティが正しくバインドされているか確認
2. propが正しいか確認
3. tableがel-table-columnの外側にあるか確認

### Q: Formのバリデーションが機能しない？

**解決策:**
1. propがv-modelと対応しているか確認
2. ref参照があるか確認
3. validateメソッドを正しく呼び出す
```typescript
const formRef = ref()
await formRef.value.validate()
```

### Q: Dialogが表示されない？

**解決策:**
1. v-modelバインディングを確認
2. visibleがtrueか確認
3. z-indexの競合がないか確認

## テーマ

### Q: テーマの色を変更するには？

**解決策:**
CSS変数を使用:
```css
:root {
  --el-color-primary: #409eff;
}
```

### Q: ダークテーマを使用するには？

**解決策:**
ダークCSSをインポート:
```typescript
import 'element-plus/theme-chalk/dark/css-vars.css'
```

### Q: コンポーネントスタイルをオーバーライドするには？

**解決策:**
CSSセレクタを使用:
```css
.el-button--primary {
  background-color: #あなたの色;
}
```

## 国際化

### Q: 言語を切り替えるには？

**解決策:**
```typescript
import ja from 'element-plus/es/locale/lang/ja'
import en from 'element-plus/es/locale/lang/en'

app.use(ElementPlus, { locale: ja })
```

### Q: カスタムテキスト？

**解決策:**
```typescript
import {ElMessage} from 'element-plus'
ElMessage('カスタムメッセージ')
```

## パフォーマンス

### Q: 初期ロードが遅い？

**解決策:**
1. 必要に応じたインポートを使用
2. 自動インポートプラグインを使用
3. Viteのプリビルドを設定

### Q: バンドルサイズが大きい？

**解決策:**
1. 必要に応じたインポートが使用されているか確認
2. ツリーシェイキングを使用
3. 完全パッケージをインポートしていないか確認

## 互換性

### Q: IEブラウザはサポートされていない？

Element PlusはIEブラウザをサポートしていません。以下のブラウザを使用してください:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### Q: 他のライブラリとの競合？

**解決策:**
1. z-indexレベルを確認
2. CSS Modulesを使用してスタイルの競合を回避
3. Vueバージョンの互換性を確認
