# Manuel Utilisateur Go

> Guide complet de programmation Go

## Variables et Types
```go
var name = "Go"
age := 10

// Types
var i int = 10
var f float64 = 3.14
var s string = "Hello"
var b bool = true
```

## Fonctions
```go
func add(a, b int) int {
    return a + b
}

func divide(a, b int) (int, error) {
    if b == 0 {
        return 0, errors.New("division par zéro")
    }
    return a / b, nil
}
```

## Goroutines et Channels
```go
go func() {
    fmt.Println("Hello from goroutine")
}()

ch := make(chan int)
ch <- 10
value := <-ch
```

## Tests
```go
import "testing"

func TestAdd(t *testing.T) {
    result := add(2, 3)
    if result != 5 {
        t.Errorf("Expected 5, got %d", result)
    }
}
```
