# Apifox Installation Guide

Multiple installation methods for Apifox.

## Download and Install

### Desktop Application

1. Visit https://apifox.com
2. Download Windows/macOS/Linux version
3. Run the installer

### Windows

- Download .exe installer
- Run the installation wizard

### macOS

- Download .dmg package
- Drag to Applications

### Linux

- Download AppImage or deb package
- Add execute permission

## Web Version

Visit https://web.apifox.com directly to use.

## Account Registration

1. Open Apifox
2. Choose registration method (email/phone/GitHub)
3. Complete registration and log in

## Common Shortcuts

| Shortcut | Function |
|----------|----------|
| Ctrl + Enter | Send request |
| Ctrl + S | Save |
| Ctrl + / | Comment |
| Ctrl + Space | Auto complete |

## First Time Use

### Create Project

1. Click "New Project"
2. Enter project name
3. Choose project type

### Add API

1. Click "New Interface" in project
2. Fill in interface information
3. Send request to test

## Configure Proxy

### Set Proxy

1. Open Settings
2. Choose network proxy
3. Configure proxy address and port

## Plugin Extensions

Support installing plugin extensions.
