# Guía de instalación de JDK

Esta guía cubre cómo instalar JDK (Java Development Kit) en Windows, macOS y Linux.

## Visión general de versiones

| Versión | Tipo | Fecha de lanzamiento | Soporte hasta |
|---------|------|--------------|---------------|
| JDK 25 | Última | 2026 | 6 meses |
| JDK 21 | LTS | Septiembre 2023 | Septiembre 2029 |
| JDK 17 | LTS | Septiembre 2021 | Septiembre 2029 |
| JDK 11 | LTS | Septiembre 2018 | Septiembre 2031 |
| JDK 8 | Herencia | Marzo 2014 | Diciembre 2030 |

**Recomendación:** Use JDK 21 (LTS) o JDK 17 (LTS) para producción. JDK 25 para las últimas funciones.

## Descargar JDK

### Fuentes oficiales

- **Oracle JDK**: https://www.oracle.com/java/technologies/downloads/
- **OpenJDK (Eclipse Temurin)**: https://adoptium.net/
- **Amazon Corretto**: https://aws.amazon.com/corretto/

|----------|--------------|---------------|

| macOS x64 | .dmg | .dmg |
| macOS ARM64 (Apple Silicon) | .dmg | .dmg |
| Linux x64 | .tar.gz / .rpm / .deb | .tar.gz / .rpm / .deb |
| Linux ARM64 | .tar.gz / .rpm | .tar.gz / .rpm |

## Instalar en Windows

### Opción 1: Instalador (.exe)

1. Descargar JDK de Oracle o Adoptium
2. Ejecutar el instalador
3. Hacer clic en "Siguiente" en el asistente
4. Anotar la ruta de instalación (predeterminada: `C:\Program Files\Java\jdk-21`)

### Opción 2: Archivo ZIP

1. Descargar ZIP de Oracle o Adoptium
2. Extraer en la ubicación deseada (ej., `C:\Java\jdk-21`)
3. No requiere instalación

### Configurar variables de entorno

```cmd
# Abrir Propiedades del sistema → Avanzadas → Variables de entorno

# Variables del sistema → Nueva
JAVA_HOME = C:\Program Files\Java\jdk-21

# Editar variable Path, agregar:
%JAVA_HOME%\bin
```

### Verificar instalación

```cmd
java -version
javac -version
```

## Instalar en macOS

### Opción 1: Instalador DMG

1. Descargar DMG de Oracle o Adoptium
2. Abrir el archivo DMG
3. Hacer doble clic en el instalador de paquetes
4. Seguir el asistente

### Opción 2: Homebrew

```bash
# Instalar Homebrew si no está instalado
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Instalar JDK 21
brew install openjdk@21

# Enlazar a Java del sistema
sudo ln -sfn $(brew --prefix)/opt/openjdk@21/libexec/openjdk.jdk /Library/Java/JavaVirtualMachines/openjdk-21.jdk

# Agregar a PATH
echo 'export JAVA_HOME=$(/usr/libexec/java_home -v 21)' >> ~/.zshrc
source ~/.zshrc
```

### Opción 3: SDKMAN

```bash
# Instalar SDKMAN
curl -s "https://get.sdkman.io" | bash
source ~/.sdkman/bin/sdkman-init.sh

# Instalar JDK 21
sdk install java 21.0.5-tem
```

### Verificar instalación

```bash
java -version
javac -version
```

## Instalar en Linux

### Ubuntu/Debian

```bash
# Actualizar lista de paquetes
sudo apt update

# Instalar OpenJDK 21
sudo apt install openjdk-21-jdk

# Configurar JAVA_HOME
echo 'export JAVA_HOME=/usr/lib/jvm/java-21-openjdk-amd64' >> ~/.bashrc
source ~/.bashrc
```

### RHEL/CentOS/Fedora

```bash
# Instalar JDK 21
sudo dnf install java-21-openjdk-devel

# Configurar JAVA_HOME
echo 'export JAVA_HOME=/usr/lib/jvm/java-21-openjdk' >> ~/.bashrc
source ~/.bashrc
```

### Instalación manual (todas las Linux)

```bash
# Descargar JDK
wget https://download.oracle.com/java/21/latest/jdk-21_linux-x64_bin.tar.gz

# Extraer
sudo tar -xzf jdk-21_linux-x64_bin.tar.gz -C /usr/local/

# Configurar JAVA_HOME
echo 'export JAVA_HOME=/usr/local/jdk-21' >> ~/.bashrc
source ~/.bashrc
```

### Verificar instalación

```bash
java -version
javac -version
```

## Múltiples versiones de JDK

### Cambiar JDK en macOS

```bash
# Listar JDK instalados
/usr/libexec/java_home -V

# Cambiar a versión específica
export JAVA_HOME=$(/usr/libexec/java_home -v 17)
```

### Cambiar JDK en Linux

```bash
# Usar update-alternatives (Ubuntu/Debian)
sudo update-alternatives --config java

# Cambio manual
export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64
export PATH=$JAVA_HOME/bin:$PATH
```

## Configuración de IDE

### IntelliJ IDEA

1. File → Project Structure → Project
2. Seleccionar SDK: Add JDK → elegir JDK instalado
3. Apply → OK

### VS Code

1. Instalar "Extension Pack for Java"
2. Ctrl+Shift+P → Java: Configure Java Runtime
3. Seleccionar JDK instalado

### Eclipse

1. Window → Preferences → Java → Installed JREs
2. Add → Standard VM
3. Navegar a la carpeta de instalación de JDK
4. Establecer como predeterminado

## Problemas comunes

### "java" no reconocido

- Verificar que JAVA_HOME esté configurado correctamente
- Comprobar que PATH incluya `%JAVA_HOME%\bin`
- Reiniciar terminal/símbolo del sistema

### Puerto 8080 en uso

```bash
# Windows
netstat -ano | findstr :8080

# Linux/macOS
lsof -i :8080
```

### JAVA_HOME apunta a JRE en lugar de JDK

- Asegurarse de que JAVA_HOME apunte a JDK (contiene `bin/javac`)
- No JRE (solo contiene `bin/java`)
