# Kubernetes インストールガイド

Kubernetesのインストール方法。

---

## インストール方法

| 方法 | 説明 |
|------|------|
| **minikube** | シングルノードクラスタ |
| **kind** | DockerでK8s実行 |
| **kubeadm** | 本番環境向け |

---

## minikube

```bash
brew install minikube
minikube start
```

## kubectl インストール

```bash
brew install kubectl
```

## 確認

```bash
kubectl get nodes
```
