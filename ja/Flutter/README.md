# Flutter

GoogleのオープンソースUIツールキット。

## 概要

Flutterは、Dartを使用した単一コードベースからクロスプラットフォームアプリケーションを構築するためのオープンソースUIツールキット。

## 機能

- **クロスプラットフォーム**：すべてのプラットフォーム用の単一コードベース
- **ホットリロード**：変更のインスタントプレビュー
- **豊富なウィジェット**：Material Design + Cupertino
- **高性能**：ネイティブレンダリング体験
- **オープンソース**：無料で使用可能

### インストール

インストールガイドを参照：./2.インストールガイド.md

### プロジェクトを作成

```bash
flutter create my_app
cd my_app
flutter run
```

### プロジェクト構造

```
lib/
└── main.dart        # エントリーポイント
pubspec.yaml         # 設定ファイル
```
