> 
> Mis à jour : Février 2026

---

## 1. Composants de base

### 1.1 Composant bouton

Les boutons sont l'un des composants les plus utilisés.

#### Utilisation de base

```html
<el-button>Par défaut</el-button>
<el-button type="primary">Primaire</el-button>
<el-button type="success">Succès</el-button>
<el-button type="warning">Avertissement</el-button>
<el-button type="danger">Danger</el-button>
```

#### Attributs du bouton

| Attribut | Description | Type | Par défaut |
|----------|--------------|------|-------------|
| type | Type de bouton | string | — |
| plain | Style simple | boolean | false |
| round | Arrondi | boolean | false |
| disabled | Désactivé | boolean | false |

## 2. Composants de formulaire

### 2.1 Champ de saisie

```html
<el-input v-model="input" placeholder="Entrez le contenu"></el-input>
```

### 2.2 Liste déroulante

```html
<el-select v-model="selected" placeholder="Sélectionnez">
  <el-option label="Option 1" value="1"></el-option>
  <el-option label="Option 2" value="2"></el-option>
</el-select>
```

### 2.3 Validation de formulaire

```javascript
this.$refs.form.validate(valid => {
  if (valid) {
    console.log('Validation réussie')
  }
})
```

## 3. Composant tableau

### 3.1 Tableau de base

```html
<el-table :data="tableData" stripe border>
  <el-table-column prop="name" label="Nom"></el-table-column>
  <el-table-column prop="email" label="Email"></el-table-column>
</el-table>
```

### 3.2 Tableau avec pagination

```html
<el-pagination
  @current-change="handlePageChange"
  :current-page="currentPage"
  :total="total">
</el-pagination>
```

## 4. Dialogues et notifications

### 4.1 Dialogue

```html
<el-dialog title="Titre" :visible.sync="visible">
  <span>Contenu du dialogue</span>
</el-dialog>
```

### 4.2 Messages

```javascript
this.$message.success('Succès')
this.$message.error('Erreur')
this.$message.warning('Avertissement')
```

## 5. Composants de navigation

### 5.1 Menu de navigation

```html
<el-menu default-active="1">
  <el-menu-item index="1">Accueil</el-menu-item>
  <el-menu-item index="2">À propos</el-menu-item>
</el-menu>
```

### 5.2 Onglets

```html
<el-tabs>
  <el-tab-pane label="Onglet 1">Contenu 1</el-tab-pane>
  <el-tab-pane label="Onglet 2">Contenu 2</el-tab-pane>
</el-tabs>
```
