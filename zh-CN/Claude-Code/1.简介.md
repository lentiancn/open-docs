# Claude Code 简介

Claude Code 是由 Anthropic 公司开发的智能编程助手，能够读取代码库、编辑文件、执行命令，并与开发工具无缝集成。它支持在终端、IDE、桌面应用和浏览器等多种环境中使用。

## 什么是 Claude Code

Claude Code 是一款 AI 驱动的编程助手，能够帮助你构建功能、修复 bug 和自动化开发任务。它能够理解整个代码库，可以跨多个文件和工具协同工作来完成各种开发任务。

## 主要特性

Claude Code 提供以下核心能力：

**自动化开发任务**
- 为未测试的代码编写测试用例
- 修复整个项目的 lint 错误
- 解决合并冲突
- 更新依赖版本
- 编写发布说明

**功能开发与缺陷修复**
- 用自然语言描述你想要实现的功能
- Claude Code 会规划方案、跨文件编写代码并验证功能

**Git 集成**
- 直接在 Claude Code 中进行 Git 操作
- 暂存更改、编写提交信息、创建分支、打开 Pull Request

**MCP 集成**
- Model Context Protocol（MCP）是一个开放标准，用于将 AI 工具连接到外部数据源
- 通过 MCP，Claude Code 可以读取 Google Drive 中的设计文档、更新 Jira 工单、从 Slack 获取数据或使用自定义工具

**自定义能力**
- 使用 CLAUDE.md 文件为项目设置编码标准、架构决策、首选库和审查清单
- 创建自定义命令来封装可重复的工作流
- 使用 Hooks 在文件编辑后自动格式化或在提交前运行 lint

**多代理协作**
- 可以同时启动多个 Claude Code 代理并行工作
- 牵头代理协调工作、分配子任务并合并结果

**CLI 灵活性**
- 支持管道操作、脚本化和自动化
- 可以将日志通过管道传输给它、 在 CI 中运行或与其他工具链式组合

## 使用场景

Claude Code 可以在多种场景下使用：

| 使用场景 | 推荐方式 |
|---------|---------|
| 本地开发 | 终端 CLI |
| IDE 内编码 | VS Code 或 JetBrains |
| 跨设备继续工作 | 远程控制功能 |
| 移动端操作 | Web 或 iOS 应用 |
| CI/CD 自动化 | GitHub Actions 或 GitLab CI/CD |
| 团队协作 | Slack 集成 |
| 浏览器调试 | Chrome 扩展 |

## 账户要求

Claude Code 需要以下账户之一：
- Claude Pro、Max、Teams 或 Enterprise 订阅
- Anthropic Console 账户
- 支持的第三方云提供商（Amazon Bedrock、Google Vertex AI、Microsoft Foundry）

免费 Claude.ai 计划不包含 Claude Code 访问权限。
