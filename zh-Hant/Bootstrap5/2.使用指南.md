# Bootstrap 5 使用指南

使用 Bootstrap 5 的指南。

---

## 柵格系統

```html
<div class="container">
  <div class="row">
    <div class="col">列</div>
  </div>
</div>
```

---

## 按鈕

```html
<button class="btn btn-primary">主要</button>
<button class="btn btn-secondary">次要</button>
```

---

## 卡片

```html
<div class="card">
  <div class="card-body">內容</div>
</div>
```

---

## 表單

```html
<input class="form-control">
```

---

## 圖標

```html
<i class="bi bi-github"></i>
```
