# Spring Boot Frequently Asked Questions

## General Questions

### What is Spring Boot?
Spring Boot is an open-source Java framework that simplifies enterprise application development by providing auto-configuration, embedded servers, and production-ready features out of the box.

### Why should I use Spring Boot?
Spring Boot reduces boilerplate code, simplifies configuration, and speeds up development. It follows "convention over configuration" and lets you focus on business logic rather than infrastructure.

### What is the difference between Spring and Spring Boot?
| Aspect | Spring | Spring Boot |
|--------|--------|-------------|
| Configuration | XML/Annotation-based | Auto-configuration |
| Server | External (Tomcat, etc.) | Embedded |
| Dependencies | Manual management | Starter POMs |
| Development | Verbose | Concise |

## Installation Questions

### How do I create a Spring Boot project?
1. Use Spring Initializr (https://start.spring.io)
2. Use IDE (IntelliJ, Eclipse) with Spring Boot plugin
3. Use Spring Boot CLI: `spring init --name myapp`

### What Java version does Spring Boot require?
- Spring Boot 2.x: Java 8-17
- Spring Boot 3.x: Java 17+

### How do I add dependencies?
Add starters to your `pom.xml` or `build.gradle`:
```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-web</artifactId>
</dependency>
```

## Usage Questions

### How do I create a REST controller?
```java
@RestController
@RequestMapping("/api")
public class HelloController {
    
    @GetMapping("/hello")
    public String hello() {
        return "Hello, World!";
    }
}
```

### How do I connect to a database?
```java
// application.yml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/mydb
    username: root
    password: secret

// Repository
@Repository
public interface UserRepository extends JpaRepository<User, Long> {
}
```

### How do I enable security?
```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-security</artifactId>
</dependency>
```

### How do I configure application properties?
```yaml
# application.yml
server:
  port: 8080
spring:
  application:
    name: myapp
logging:
  level:
    com.demo: DEBUG
```

## Configuration Questions

### What is the difference between @SpringBootApplication and @EnableAutoConfiguration?
- `@SpringBootApplication`: Combines `@Configuration`, `@EnableAutoConfiguration`, and `@ComponentScan`
- `@EnableAutoConfiguration`: Enables Spring Boot's auto-configuration mechanism

### How do I use profiles?
```yaml
# application-dev.yml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/devdb

# application-prod.yml
spring:
  datasource:
    url: jdbc:mysql://prodserver/proddb
```

Activate with: `spring.profiles.active=dev`

### How do I create custom auto-configuration?
```java
@Configuration
@ConditionalOnClass(RedisTemplate.class)
public class RedisAutoConfiguration {
    @Bean
    public RedisTemplate<String, Object> redisTemplate() {
        // configuration
    }
}
```

## Deployment Questions

### How do I create an executable JAR?
```bash
./mvnw clean package
java -jar target/myapp-1.0.0.jar
```

### How do I deploy to Docker?
```dockerfile
FROM eclipse-temurin:17-jre
COPY target/myapp.jar app.jar
ENTRYPOINT ["java", "-jar", "app.jar"]
```

```bash
docker build -t myapp .
docker run -p 8080:8080 myapp
```

### How do I deploy to Kubernetes?
Use Spring Boot's Kubernetes support or use the Spring Cloud Kubernetes project.

## Troubleshooting

### Application fails to start
- Check for missing dependencies
- Review stack trace for configuration errors
- Verify database connections
- Check port availability (8080)

### Bean not found error
- Ensure component scanning is enabled
- Check if bean is in the correct package
- Verify class is annotated correctly

### OutOfMemory errors
- Increase JVM heap size: `-Xmx2g`
- Optimize database queries
- Check for memory leaks

## Testing Questions

### How do I write tests?
```java
@SpringBootTest
class MyControllerTest {
    
    @Autowired
    private MockMvc mockMvc;
    
    @Test
    void testHello() throws Exception {
        mockMvc.perform(get("/api/hello"))
            .andExpect(status().isOk())
            .andExpect(content().string("Hello, World!"));
    }
}
```

### What testing tools does Spring Boot support?
- JUnit 5
- Mockito
- Spring Test
- AssertJ
- RestAssured

## Best Practices

1. Use application.yml over application.properties
2. Keep configuration externalized
3. Use constructor injection over field injection
4. Write unit and integration tests
5. Use profiles for environment-specific settings
6. Enable production-ready features
