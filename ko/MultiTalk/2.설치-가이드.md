# MultiTalk 설치 가이드

MultiTalk 설치 및 설정 가이드.

## 전제 조건

- Windows 10/11, macOS 10.15+ 또는 Linux
- 마이크 및 스피커/헤드셋
- 인터넷 연결

## 설치

### Windows

1. 설치 프로그램을 다운로드
2. 설치 프로그램 실행
3. 설치 마법사 따르기
4. MultiTalk 시작

### macOS

1. .dmg 파일 다운로드
2. 설치 프로그램 열기
3. MultiTalk를 애플리케이션으로 드래그
4. MultiTalk 시작

## 설정

### 오디오 설정

1. 설정 > 오디오 열기
2. 입력 장치 선택
3. 출력 장치 선택
4. 수준 테스트

### 푸시 투 토크

1. 설정 > 단축키로 이동
2. 푸시 투 토크 활성화
3. 사용할 키 조합 누르기

## 문제 해결

### 오디오 입력 없음

**해결:**
- 마이크 권한 확인
- 장치 선택 확인

### 에코 문제

**해결:**
- 헤드셋 사용
- 스피커音量 낮추기
