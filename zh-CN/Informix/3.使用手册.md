# Informix 使用手册

## 基本操作命令

### 数据库管理常用命令

| 命令 | 说明 |
|------|------|
| `oninit` | 启动 Informix 实例 |
| `onmode -ky` | 停止 Informix 实例 |
| `onstat` | 查看实例状态 |
| `onstat -d` | 查看磁盘空间使用情况 |
| `onstat -l` | 查看逻辑日志状态 |
| `onstat -u` | 查看用户会话 |
| `onstat -k` | 查看锁信息 |
| `onstat -x` | 查看事务信息 |

## SQL 命令详解

### 创建数据库

```sql
-- 创建不记录日志的数据库
CREATE DATABASE testdb;

-- 创建带缓冲日志的数据库
CREATE DATABASE testdb WITH BUFFERED LOG;

-- 创建带日志的数据库
CREATE DATABASE testdb WITH LOG;

-- 创建 ANSI 模式的数据库
CREATE DATABASE testdb WITH LOG MODE ANSI;
```

### 创建数据表

```sql
-- 创建普通数据表
CREATE TABLE cti_vccinfo (
    vccid     CHAR(6)     NOT NULL,
    vccname   VARCHAR(255),
    effective INTEGER     DEFAULT 0 NOT NULL,
    agentmax  INTEGER     DEFAULT 0 NOT NULL,
    ivrmax    INTEGER     DEFAULT 0 NOT NULL,
    updatekey VARCHAR(30),
    PRIMARY KEY (vccid) CONSTRAINT PK_CTI_VI
);

-- 创建临时表
CREATE TEMP TABLE customer_temp (
    num        SERIAL    NOT NULL,
    name       CHAR(15),
    create_time DATETIME YEAR TO FRACTION(3)
);

-- 从查询结果创建临时表
SELECT num, name, create_time 
FROM customer 
INTO TEMP customer_temp;
```

### 主键与外键约束

```sql
-- 创建复合主键
CREATE UNIQUE INDEX Index_CTI_HTG ON cti_humantaskgroup(
    serviceid ASC,
    agentid ASC
);

-- 创建外键约束
ALTER TABLE cti_humantaskgroup
ADD CONSTRAINT FOREIGN KEY (serviceid)
    REFERENCES cti_humantask (serviceid) 
    ON DELETE CASCADE
    CONSTRAINT FK_CTI_HTG_HT;
```

### 检查约束

```sql
CREATE TABLE student (
    id        VARCHAR(10) NOT NULL,
    name      VARCHAR(10) NOT NULL,
    age       INTEGER     DEFAULT 0 NOT NULL CHECK (age BETWEEN 5 AND 35),
    birthday  VARCHAR(8)
);
```

### 视图操作

```sql
-- 创建视图
CREATE VIEW student_age (id, name, age, birthday)
AS SELECT id, name, age, birthday 
FROM student 
WHERE age >= 20 AND age <= 25;

-- 删除视图
DROP VIEW student_age;
```

### 查询语句

```sql
-- 基本查询
SELECT * FROM student;
SELECT name, age FROM student;

-- 条件查询
SELECT * FROM student WHERE name = 'amigo';
SELECT * FROM student WHERE age != 23;
SELECT * FROM student WHERE age > 23;

-- 逻辑运算
SELECT * FROM student WHERE name = 'amigo' AND id = '1000';
SELECT * FROM student WHERE name = 'amigo' OR name = 'xingxing';

-- BETWEEN/IN
SELECT id, name, age FROM student WHERE age BETWEEN 24 AND 30;
SELECT id, name FROM student WHERE name NOT IN ('amigo', 'xingxing');

-- NULL 判断
SELECT * FROM student WHERE birthday IS NOT NULL;

-- 模糊匹配
SELECT id, name FROM student WHERE name MATCHES '?migo';  -- ? 单字符
SELECT * FROM student WHERE name MATCHES '*go';          -- * 任意字符

-- 分组查询
SELECT detail, COUNT(*) AS ratio 
FROM CTI_CallStat 
WHERE taskid = '000001200002111864' 
GROUP BY detail;

-- 表连接
SELECT order_num, order_time, c.customer_num
FROM customer c, orders o
WHERE c.customer_num = o.customer_num;

-- 或使用 JOIN
SELECT order_num, order_time, c.customer_num
FROM customer c JOIN orders o
ON c.customer_num = o.customer_num;

-- 外连接
SELECT e.employee_num, employee_name, project_num, project_name
FROM employee e LEFT OUTER JOIN project p 
ON e.employee_num = p.employee_num;
```

### INSERT / UPDATE / DELETE

```sql
-- 插入数据
INSERT INTO student VALUES('1000', 'amigo', 27, '19821121');
INSERT INTO student (id, name, age) VALUES('1005', 'amigo', 27);

-- 从其他表插入
INSERT INTO student
SELECT id, name, age, birthday
FROM student_bak WHERE age > 25;

-- 更新数据
UPDATE student SET name = 'amigoxie', age = 28 WHERE id = '1000';

-- 删除数据
DELETE FROM student;
DELETE FROM student WHERE id = '1001';
DELETE FROM student WHERE name MATCHES '*go';
```

## 配置说明

### 关键配置文件

| 文件 | 说明 |
|------|------|
| `onconfig` | 主配置文件，包含实例级参数 |
| `sqlhosts` | 客户端连接配置文件 |
| `$INFORMIXDIR/etc/onconfig.std` | 配置文件模板 |
| `$INFORMIXDIR/etc/sqlhosts.std` | sqlhosts 模板 |

### 常用配置参数

```
ROOTNAME     rootdbs              # 根数据库空间名称
ROOTPATH     /home/informix/chunk/rootchk  # 根空间路径
ROOTSIZE     1024000              # 根空间大小（KB）
DBSERVERNAME demo_on              # 数据库服务器名
TAPEDEV      /dev/null            # 磁带设备
LTAPEDEV     /dev/null            # 磁带设备
```

### 字符集配置

在环境变量中添加：

```bash
export DB_LOCALE=zh_cn.GB18030-2000
export CLIENT_LOCALE=zh_cn.GB18030-2000
```

JDBC 连接字符串示例：

```
jdbc:informix-sqli://192.168.1.1:9088/test:INFORMIXSERVER=ifxserver;NEWCODESET=GB18030,GB18030-2000,5488;CLIENT_LOCALE=zh_cn.GB18030-2000;DB_LOCALE=zh_cn.GB18030-2000
```
