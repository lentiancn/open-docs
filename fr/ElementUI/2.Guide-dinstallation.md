# Guide d'installation d'Element UI

> Configuration

---

## Configuration requise

| Environnement | Minimum |
|---------------|---------|
| Node.js | 8.x |
| npm | 5.x |
| Vue | 2.6.0 |

---

## Installation

```bash
npm install element-ui --save
```

---

## Import

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```
