## 栅格系统

### 基本原理

Bootstrap 的栅格系统使用 Flexbox 构建，最多支持 12 列。

### 容器

```html
<!-- 固定宽度容器 -->
<div class="container">
  内容
</div>

<!-- 100% 宽度容器 -->
<div class="container-fluid">
  内容
</div>

<!-- 响应式断点容器 (Bootstrap 5.2+) -->
<div class="container-sm">100% 宽 < 576px</div>
<div class="container-md">100% 宽 >= 768px</div>
<div class="container-lg">100% 宽 >= 992px</div>
<div class="container-xl">100% 宽 >= 1200px</div>
<div class="container-xxl">100% 宽 >= 1400px</div>
```

### 行和列

```html
<div class="container">
  <div class="row">
    <div class="col">列</div>
    <div class="col">列</div>
    <div class="col">列</div>
  </div>
</div>
```

### 列宽设置

```html
<div class="row">
  <div class="col-4">4 列</div>
  <div class="col-8">8 列</div>
</div>
```

### 响应式断点

```html
<!-- 响应式列宽 -->
<div class="row">
  <div class="col-12 col-md-6 col-lg-4">
    在手机上全宽，平板 6 列，桌面 4 列
  </div>
</div>
```

### 列偏移

```html
<!-- 使用 offset -->
<div class="row">
  <div class="col-md-4">4 列</div>
  <div class="col-md-4 offset-md-4">偏移 4 列</div>
</div>

<!-- 使用 margin 自动偏移 -->
<div class="row">
  <div class="col-md-3 ms-auto">右侧偏移</div>
</div>
```

### 栅格嵌套

```html
<div class="row">
  <div class="col-sm-8">
    外层 8 列
    <div class="row">
      <div class="col-6">内层 6 列</div>
      <div class="col-6">内层 6 列</div>
    </div>
  </div>
</div>
```

### Flex 布局

```html
<!-- 水平对齐 -->
<div class="row justify-content-start">...</div>
<div class="row justify-content-center">...</div>
<div class="row justify-content-end">...</div>
<div class="row justify-content-around">...</div>
<div class="row justify-content-between">...</div>
<div class="row justify-content-evenly">...</div>

<!-- 垂直对齐 -->
<div class="row align-items-start">...</div>
<div class="row align-items-center">...</div>
<div class="row align-items-end">...</div>
```

## 最佳实践

### 1. 使用语义化 HTML

```html
<!-- 好 -->
<nav class="navbar">...</nav>
<main class="container">...</main>
<article class="card">...</article>

<!-- 避免 -->
<div class="navbar">...</div>
<div class="container">...</div>
<div class="card">...</div>
```

### 2. 保持代码简洁

```html
<!-- 好 -->
<div class="container">
  <div class="row">
    <div class="col-md-6">...</div>
  </div>
</div>

<!-- 避免过度嵌套 -->
<div class="container">
  <div class="row">
    <div class="col-md-6">
      <div class="card">
        <div class="card-body">...</div>
      </div>
    </div>
  </div>
</div>
```

### 3. 响应式设计

```html
<!-- 使用正确的断点类 -->
<div class="col-12 col-md-6 col-lg-4">
  内容
</div>
```

### 4. 性能优化

- 使用 CDN 加速
- 启用 CSS 和 JS 压缩
- 按需加载 JavaScript
