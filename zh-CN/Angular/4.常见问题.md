# Angular 常见问题

> 常见问题解答

---

## 安装问题

### Angular CLI 安装失败？

**解决方案**：

1. 清除 npm 缓存：
```bash
npm cache clean --force
```

2. 使用管理员权限：
```bash
sudo npm install -g @angular/cli
```

3. 检查 Node.js 版本：
```bash
node --version  # 需要 18.10.0+
```

---

## 开发问题

### ng 命令找不到？

确保 Angular CLI 已全局安装：

```bash
npm install -g @angular/cli
```

### 项目启动失败？

1. 删除 node_modules 重新安装：
```bash
rm -rf node_modules
npm install
```

2. 检查端口是否被占用：
```bash
ng serve --port 4201
```

---

## 组件问题

### 组件样式不生效？

确保样式文件路径正确：

```typescript
@Component({
  styleUrls: ['./my.component.css']
})
```

---

## 路由问题

### 路由跳转失败？

检查路由配置是否正确：

```typescript
const routes: Routes = [
  { path: '', component: HomeComponent },
  // 确保 path 不能有前导斜杠
];
```

---

## 表单问题

### 表单验证不生效？

确保引入 FormsModule：

```typescript
import { FormsModule } from '@angular/forms';

@NgModule({
  imports: [FormsModule]
})
export class AppModule { }
```

---

## HTTP 问题

### HTTP 请求失败？

确保引入 HttpClientModule：

```typescript
import { HttpClientModule } from '@angular/common/http';

@NgModule({
  imports: [HttpClientModule]
})
export class AppModule { }
```

---

## 性能问题

### 应用加载慢？

1. 开启生产模式：
```bash
ng build --configuration production
```

2. 使用懒加载：
```typescript
const routes: Routes = [
  { path: 'admin', loadChildren: () => import('./admin/admin.module').then(m => m.AdminModule) }
];
```

## 构建问题

### 构建失败？

1. 检查 TypeScript 配置
2. 确保所有依赖已安装
3. 查看错误信息具体定位问题
