## 快速开始

### 安装 Angular CLI

```bash
npm install -g @angular/cli
```

### 创建新项目

```bash
ng new my-app
cd my-app
ng serve
```

### 生成组件

```bash
ng generate component my-component
# 或
ng g c my-component
```
