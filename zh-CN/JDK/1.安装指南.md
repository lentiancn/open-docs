# JDK 安装指南

JDK（Java Development Kit）是官方的 Java 开发工具包。本指南涵盖各种环境的安装方法。

## 版本信息

| 版本 | 状态 | 说明 |
|---------|--------|-------------|
| JDK 21 (LTS) | ✅ 当前版本 | 最新的 LTS 版本 |
| JDK 17 (LTS) | ✅ 受支持 | 上一版 LTS |
| JDK 11 (LTS) | ✅ 受支持 | 传统 LTS |
| JDK 8 | ✅ 受支持 | 传统版本 |
| JDK 22 | ✅ 最新版本 | 最新的功能版本 |

### LTS 与功能版本

- **LTS（长期支持）**：2-3 年更新，推荐用于生产环境
- **功能版本**：每 6 个月发布新功能，支持期较短

## 系统要求

### 硬件要求

| 要求 | 最低配置 | 推荐配置 |
|-------------|---------|-------------|

## 安装方法

### 方法 1：Oracle JDK

#### Windows

1. 从以下地址下载：https://www.oracle.com/java/technologies/downloads/
2. 运行安装程序（.exe）
3. 按照向导操作

#### macOS

```bash
# 从 Oracle 网站下载 DMG
# 或使用 Homebrew
brew install openjdk@21
```

#### Linux

```bash
# 下载 tar.gz
wget https://download.oracle.com/java/21/latest/jdk-21_linux-x64_bin.tar.gz
tar -xzf jdk-21_linux-x64_bin.tar.gz
sudo mv jdk-21 /opt/jdk-21
```

### 方法 2：Eclipse Temurin（Adoptium）

#### Windows

1. 从以下地址下载：https://adoptium.net/temurin/releases/
2. 运行安装程序

#### macOS

```bash
# 使用 Homebrew
brew install --cask temurin

# 或从网站下载 pkg
```

#### Linux

```bash
# Ubuntu/Debian
wget https://github.com/adoptium/temurin21-binaries/releases/download/jdk-21.0.2%2B13/OpenJDK21U-jdk_x64_linux_hotspot_21.0.2_13.tar.gz
tar -xzf OpenJDK21U-jdk_x64_linux_hotspot_21.0.2_13.tar.gz

# 设置环境变量
export JAVA_HOME=/path/to/jdk-21
export PATH=$JAVA_HOME/bin:$PATH
```

### 方法 3：SDKMAN

```bash
# 安装 SDKMAN
curl -s "https://get.sdkman.io" | bash

# 安装 JDK
sdk install java 21.0.2-tem

# 列出可用版本
sdk list java
```

### 方法 4：Docker

```dockerfile
FROM eclipse-temurin:21-jdk
WORKDIR /app
COPY . .
RUN javac Main.java
CMD ["java", "Main"]
```

### 方法 5：Windows 包管理器

#### Chocolatey

```powershell
choco install openjdk21
```

#### Scoop

```powershell
scoop install openjdk
```

## 环境配置

### Windows

```powershell
# 设置 JAVA_HOME
[System.Environment]::SetEnvironmentVariable("JAVA_HOME", "C:\Program Files\Java\jdk-21", "User")

# 添加到 PATH
[System.Environment]::SetEnvironmentVariable("PATH", "$env:PATH;C:\Program Files\Java\jdk-21\bin", "User")
```

### macOS/Linux

```bash
# 添加到 ~/.bashrc 或 ~/.zshrc
export JAVA_HOME=/opt/jdk-21
export PATH=$JAVA_HOME/bin:$PATH

# 生效
source ~/.bashrc
```

## 验证

```bash
# 检查 Java 版本
java -version

# 检查 Javac 版本
javac -version

# 检查 JAVA_HOME
echo $JAVA_HOME
```

## 多个 JDK 版本

### 使用 SDKMAN

```bash
# 安装多个版本
sdk install java 17.0.8-tem
sdk install java 21.0.2-tem

# 切换版本
sdk use java 17.0.8-tem
```

## 常见问题

### 找不到 "java"？

请验证 PATH 是否包含 Java 的 bin 目录。

### Java 版本错误？

检查正在使用的 Java：

```bash
which java
java -version
```

### JAVA_HOME 未设置？

将 JAVA_HOME 设置为 JDK 安装目录。
