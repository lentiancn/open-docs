# Vite Guide d'Installation

Ce guide vous aide à installer et configurer Vite.

## Prérequis

- Node.js (version 14.18+ ou 16+)
- npm, yarn ou pnpm

## Installation

### Créer un Projet

```bash
# npm
npm create vite@latest my-project -- --template vue

# yarn
yarn create vite my-project -- --template vue

# pnpm
pnpm create vite my-project -- --template vue
```

Modèles disponibles:
- `vanilla` - Vanilla JavaScript
- `vue` - Vue
- `react` - React

### Projet Existant

```bash
npm install -D vite
```

## Configuration

### Configuration de Base

Créez un fichier `vite.config.js`:

```javascript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [vue()],
  server: {
    port: 3000
  }
})
```

## Développement

### Démarrer le Serveur

```bash
npm run dev
```

### Build de Production

```bash
npm run build
```

### Prévisualiser le Build

```bash
npm run preview
```

## Configuration des Plugins

### Vue

```bash
npm install @vitejs/plugin-vue
```

```javascript
import vue from '@vitejs/plugin-vue'
export default defineConfig({ plugins: [vue()] })
```

### React

```bash
npm install @vitejs/plugin-react
```

```javascript
import react from '@vitejs/plugin-react'
export default defineConfig({ plugins: [react()] })
```

## Variables d'Environnement

Créez des fichiers `.env`:

```
VITE_API_URL=http://localhost:8000
```

Accéder dans le code:

```javascript
const apiUrl = import.meta.env.VITE_API_URL
```
