# OpenClaw 概要

## OpenClawとは？

OpenClawは、チャットアプリ（WhatsApp、Telegram、Discord、iMessage）をAIコーディングアシスタントに接続する自己ホスト型ゲートウェイです。

## 主な機能

- マルチチャンネル（WhatsApp、Telegram、Discord、iMessage）
- Webダッシュボード
- マルチエージェントルーティング
- モバイルノード
- セキュリティ制御

## 必要環境

- Node.js 22+
- APIキー（Anthropic推奨）

## クイックスタート

```bash
# インストール
npm install -g openclaw@latest

# 初期化
openclaw onboard --install-daemon

# 起動
openclaw gateway --port 18789
```

http://127.0.0.1:18789/ にアクセス
