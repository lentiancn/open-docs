# Обзор MySQL

MySQL — самая популярная в мире система управления реляционными базами данных с открытым исходным кодом. Она широко используется для веб-приложений и является основным компонентом стека LAMP (Linux, Apache, MySQL, PHP/Python/Perl).

## Что такое MySQL?

MySQL — это мощная, быстрая и надёжная система баз данных, которая хранит и управляет данными для приложений. Она использует SQL (Structured Query Language) в качестве языка запросов.

### Ключевые функции

- **Открытый исходный код** — бесплатное использование по лицензии GPL
- **Высокая производительность** — оптимизирована для скорости и эффективности
- **Масштабируемость** — легко обрабатывает большие базы данных
- **Безопасность** — надёжное управление пользователями и шифрование
- **Репликация** — поддержка мастер-мастер и мастер-слэйв
- ** движки хранения** — InnoDB, MyISAM, Memory и другие
- **Кроссплатформенность** — работает на Windows, Linux, macOS и других
