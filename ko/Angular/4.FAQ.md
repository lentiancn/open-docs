## 설치

### CLI 설치 실패?

```bash
npm cache clean --force
npm install -g @angular/cli
```

### ng 명령어를 찾을 수 없음?

```bash
npm install -g @angular/cli
```

---

## 개발

### 프로젝트가 시작되지 않음?

```bash
rm -rf node_modules
npm install
```
