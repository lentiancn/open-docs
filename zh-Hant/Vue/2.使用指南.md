## 響應式系統

### Vue 3 Composition API（推薦）

```javascript
import { ref, reactive, computed } from 'vue'

export default {
  setup() {
    // ref 用於原始類型
    const count = ref(0)
    
    // reactive 用於物件
    const state = reactive({
      name: 'Vue',
      version: 3
    })
    
    // 計算屬性
    const doubleCount = computed(() => count.value * 2)
    
    // 方法
    const increment = () => {
      count.value++
    }
    
    return { count, state, doubleCount, increment }
  }
}
```

### Vue 2 Options API

```javascript
export default {
  data() {
    return {
      count: 0,
      name: 'Vue',
      version: 2
    }
  },
  computed: {
    doubleCount() {
      return this.count * 2
    }
  },
  methods: {
    increment() {
      this.count++
    }
  }
}
```

## 元件

### 建立元件

```vue
<!-- MyComponent.vue -->
<template>
  <div class="my-component">
    <h2>{{ title }}</h2>
    <p>{{ content }}</p>
    <button @click="emitEvent">點擊我</button>
  </div>
</template>

<script setup>
import { defineProps, defineEmits } from 'vue'

const props = defineProps({
  title: {
    type: String,
    required: true
  },
  content: {
    type: String,
    default: '預設內容'
  }
})

const emit = defineEmits(['click'])

const emitEvent = () => {
  emit('click', '來自元件的問候！')
}
</script>

<style scoped>
.my-component {
  padding: 20px;
}
</style>
```

### 使用元件

```vue
<!-- ParentComponent.vue -->
<template>
  <MyComponent 
    title="我的標題"
    content="自訂內容"
    @click="handleClick"
  />
</template>

<script setup>
import MyComponent from './MyComponent.vue'

const handleClick = (message) => {
  console.log(message)
}
</script>
```

## 元件通訊

### Props（父元件到子元件）

```vue
<!-- Child.vue -->
<script setup>
defineProps({
  message: String,
  count: Number
})
</script>

<template>
  <p>{{ message }} - {{ count }}</p>
</template>

<!-- Parent.vue -->
<template>
  <Child message="你好" :count="5" />
</template>
```

### 事件（子元件到父元件）

```vue
<!-- Child.vue -->
<script setup>
const emit = defineEmits(['update'])

const sendData = () => {
  emit('update', { name: 'Vue' })
}
</script>

<template>
  <button @click="sendData">傳送</button>
</template>

<!-- Parent.vue -->
<template>
  <Child @update="handleUpdate" />
</template>

<script setup>
const handleUpdate = (data) => {
  console.log(data)
}
</script>
```

### Provide/Inject

```vue
<!-- Parent.vue -->
<script setup>
import { provide } from 'vue'

provide('theme', 'dark')

</script>

<!-- Child.vue -->
```

## 事件處理

```html
<!-- 點擊事件 -->
<button @click="handleClick">點擊</button>

<!-- 事件物件 -->
<button @click="handleClick($event)">點擊</button>

<!-- 多個處理器 -->
<button @click="handler1(), handler2()">點擊</button>

<!-- 事件修飾符 -->
<button @click.stop="handleClick">停止傳播</button>
<button @click.prevent="handleSubmit">阻止預設</button>
<button @click.stop.prevent="handleClick">兩者皆要</button>

<!-- 鍵盤修飾符 -->
<input @keyup.enter="submit" />
<input @keyup.ctrl.enter="submit" />

<!-- 滑鼠修飾符 -->
<button @click.left="leftClick">左鍵</button>
<button @click.right="rightClick">右鍵</button>
```

## 表單輸入綁定

## 計算屬性

```javascript
import { ref, computed } from 'vue'

const firstName = ref('John')
const lastName = ref('Doe')

// 計算屬性 - 會快取，只有在相依資料改變時才會重新計算
const fullName = computed(() => {
  return `${firstName.value} ${lastName.value}`
})

// 方法 - 每次渲染都會重新執行
const getFullName = () => {
  return `${firstName.value} ${lastName.value}`
}
```

## 監聽器

```javascript
import { ref, watch } from 'vue'

const count = ref(0)

// 簡單監聽器
watch(count, (newValue, oldValue) => {
  console.log(`從 ${oldValue} 變更為 ${newValue}`)
})

// 深度監聽器
const state = ref({ name: 'Vue' })
watch(state, (newValue) => {
  console.log('狀態變更：', newValue)
}, { deep: true })

// 監聽多個來源
const firstName = ref('John')
const lastName = ref('Doe')
watch([firstName, lastName], ([newFirst, newLast]) => {
  console.log(`姓名：${newFirst} ${newLast}`)
})
```

## 生命週期鉤子

### Vue 3 Composition API

| 鉤子 | 說明 |
|------|-------------|
| `setup()` | 在其他鉤子之前呼叫，用於設定 |
| `onMounted()` | 元件掛載完成後 |
| `onUpdated()` | 元件更新後 |
| `onUnmounted()` | 元件卸載後 |
| `onErrorCaptured()` | 捕获錯誤時 |

```javascript
import { 
  onMounted, 
  onUpdated, 
  onUnmounted 
} from 'vue'

export default {
  setup() {
    onMounted(() => {
      console.log('掛載完成！')
    })
    onUpdated(() => {
      console.log('更新完成！')
    })
    onUnmounted(() => {
      console.log('卸載完成！')
    })
  }
}
```

### Vue 2 Options API

| 鉤子 | 說明 |
|------|-------------|
| `beforeCreate` | 實例建立前 |
| `created` | 實例建立後 |
| `beforeMount` | 掛載前 |
| `mounted` | 掛載後 |
| `beforeUpdate` | 更新前 |
| `updated` | 更新後 |
| `beforeDestroy` | 銷毀前 |
| `destroyed` | 銷毀後 |

## Vue Router

### 基本設定

```javascript
// router/index.js
import { createRouter, createWebHistory } from 'vue-router'
import Home from '../views/Home.vue'
import About from '../views/About.vue'
import User from '../views/User.vue'

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About },
  { path: '/user/:id', component: User, props: true }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```

### 導航

```vue
<template>
  <!-- 程式化導航 -->
  <button @click="$router.push('/about')">關於</button>
  <button @click="$router.back()">返回</button>
  <button @click="$router.forward()">前進</button>
  
  <!-- 路由連結 -->
  <router-link to="/">首頁</router-link>
  <router-link to="/about">關於</router-link>
  <router-link :to="`/user/${userId}`">用戶</router-link>
</template>

<script setup>
import { useRouter } from 'vue-router'

const router = useRouter()
const userId = 123

const navigate = () => {
  router.push('/about')
}
</script>
```

## Pinia 狀態管理

### 定義 Store

```javascript
// stores/user.js
import { defineStore } from 'pinia'

export const useUserStore = defineStore('user', {
  state: () => ({
    name: 'John',
    email: 'john@example.com',
    age: 25
  }),
  
  getters: {
    isAdult: (state) => state.age >= 18,
    displayName: (state) => `${state.name} (${state.age})`
  },
  
  actions: {
    updateName(newName) {
      this.name = newName
    },
    async fetchUser() {
      const data = await fetch('/api/user')
      const user = await data.json()
      this.$patch(user)
    }
  }
})
```

### 使用 Store

```vue
<script setup>
import { storeToRefs } from 'pinia'
import { useUserStore } from '../stores/user'

const userStore = useUserStore()

// 直接存取狀態
console.log(userStore.name)

// 使用 storeToRefs 保持響應式
const { name, email } = storeToRefs(userStore)

// 呼叫 action
userStore.updateName('Jane')

// 非同步 action
userStore.fetchUser()
</script>
```
