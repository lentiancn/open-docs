# Element Plus 安裝指南

> Element Plus 是由 Element 團隊打造的 Vue 3 桌面端元件庫
> 
> 更新日期：2026年2月
> 
> 當前版本：2.x（穩定版）

---

## 一、項目簡介

Element Plus 是一個基於 Vue 3 的桌面端 UI 元件庫，由 Element 團隊開發和維護。作為 Element UI 的 Vue 3 升級版本，Element Plus 完全遵循 Vue 3 的 Composition API 風格編寫，並全面採用 TypeScript 重構，提供了類型安全的元件開發體驗。

Element Plus 首次正式穩定版於 2022 年 2 月 7 日發布，經過多年迭代，API 已經非常穩定。目前支援簡體中文、英文等多種語言，並且擁有活躍的社群生態。

## 二、環境要求

### 2.1 開發環境要求

| 環境 | 最低版本 | 推薦版本 |
|------|----------|----------|
| Node.js | 16.x | 18.x 或 20.x |
| npm | 8.x | 9.x 或 10.x |
| Vue | 3.2.0 | 3.4.x |
| Vite | 4.x | 5.x |

### 2.2 瀏覽器相容性

Element Plus 支援所有現代瀏覽器，包括：

- Chrome >= 80
- Firefox >= 78
- Safari >= 14
- Edge >= 80

不支援 Internet Explorer 11。

### 2.3 技術堆疊要求

| 技術 | 說明 |
|------|------|
| Vue 3 | 必須使用 Vue 3.2.0 及以上版本 |
| TypeScript | 推薦使用 4.5 及以上版本 |
| 構建工具 | Vite（推薦）或 Webpack |

## 三、安裝步驟

### 3.1 使用 npm 安裝（推薦）

npm 是 Node.js 的套件管理工具，也是安裝 Element Plus 的官方推薦方式。

```bash
# 進入你的 Vue 3 專案目錄
cd your-vue-project

# 安裝 Element Plus
npm install element-plus
```

### 3.2 使用 yarn 安裝

```bash
# 安裝 Element Plus
yarn add element-plus
```

### 3.3 使用 pnpm 安裝

```bash
# 安裝 Element Plus
pnpm add element-plus
```

## 四、在 Vue 3 專案中引入

### 4.1 完整引入（適合小型專案）

完整引入會打包所有元件，配置簡單，但打包體積較大。

```typescript
// main.ts 或 main.js
import { createApp } from 'vue'
import App from './App.vue'

// 導入 Element Plus 和它的樣式
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'

const app = createApp(App)

// 使用 Element Plus
app.use(ElementPlus)

app.mount('#app')
```

### 4.2 按需引入（推薦，適合生產環境）

按需引入可以顯著減少打包體積，只引入實際使用的元件。

#### 4.2.1 安裝自動導入外掛程式

首先安裝 unplugin-vue-components 和 unplugin-auto-import：

```bash
npm install -D unplugin-vue-components unplugin-auto-import
```

#### 4.2.2 配置 Vite

在 vite.config.ts 中添加配置：

```typescript
// vite.config.ts
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

這樣配置後，元 件可以自動導入，無需手動 import。例如直接使用：

```vue
<template>
  <el-button type="primary">你好</el-button>
</template>
```

#### 4.2.3 手動按需引入（備選方案）

如果不使用自動導入外掛程式，也可以手動引入單個元件：

```typescript
// main.ts
import { createApp } from 'vue'
import App from './App.vue'

// 手動引入需要的元件
import { ElButton, ElInput, ElForm } from 'element-plus'
import 'element-plus/es/components/button/style'
import 'element-plus/es/components/input/style'
import 'element-plus/es/components/form/style'

const app = createApp(App)
app.use(ElButton)
app.use(ElInput)
app.use(ElForm)

app.mount('#app')
```

## 五、使用 CDN 引入（無需構建工具）

如果你不想使用 npm 或構建工具，可以通過 CDN 方式引入 Element Plus。

### 5.1 通過 unpkg CDN 引入

```html
<!DOCTYPE html>
<html lang="zh-TW">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Element Plus CDN 示例</title>
  <!-- 引入 Vue 3 -->
  <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
  <!-- 引入 Element Plus -->
  <link rel="stylesheet" href="https://unpkg.com/element-plus/dist/index.css">
  <script src="https://unpkg.com/element-plus"></script>
</head>
<body>
  <div id="app">
    <el-button type="primary">你好 Element Plus</el-button>
  </div>

  <script>
    const { createApp, ref } = Vue
    
    const App = {
      setup() {
        const visible = ref(false)
        return { visible }
      },
      template: `
        <el-button @click="visible = true">打開對話框</el-button>
        <el-dialog v-model="visible" title="你好">這是 Element Plus</el-dialog>
      `
    }
    
    const app = createApp(App)
    app.use(ElementPlus)
    app.mount('#app')
  </script>
</body>
</html>
```

### 5.2 通過 jsDelivr CDN 引入

```html
<!-- 替換上面的 unpkg 為 jsDelivr -->
<script src="https://cdn.jsdelivr.net/npm/vue@3/dist/vue.global.js"></script>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/element-plus/dist/index.css">
<script src="https://cdn.jsdelivr.net/npm/element-plus"></script>
```

## 六、國際化配置

Element Plus 預設使用英文，要使用中文需要配置國際化。

### 6.1 全域中文配置

```typescript
// main.ts
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import zhCn from 'element-plus/es/locale/lang/zh-cn'
import 'element-plus/dist/index.css'

const app = createApp(App)

app.use(ElementPlus, {
  locale: zhCn,
})

app.mount('#app')
```

### 6.2 動態切換語言

```typescript
// i18n.ts
import { createI18n } from 'vue-i18n'
import zhCn from 'element-plus/es/locale/lang/zh-cn'
import en from 'element-plus/es/locale/lang/en'

const i18n = createI18n({
  legacy: false,
  locale: 'zh-cn',
  messages: {
    'zh-cn': {
      ...zhCn,
    },
    en: {
      ...en,
    },
  },
})

export default i18n
```

然後在 main.ts 中使用：

```typescript
import { createApp } from 'vue'
import App from './App.vue'
import ElementPlus from 'element-plus'
import i18n from './i18n'
import 'element-plus/dist/index.css'

const app = createApp(App)

app.use(ElementPlus)
app.use(i18n)

app.mount('#app')
```

## 七、主題定製

### 7.1 使用 SCSS 變數覆蓋

Element Plus 使用 SCSS 編寫，可以通過覆蓋變數來定製主題。

```scss
// styles/element-variables.scss
// 主題色
$--color-primary: #409EFF;

// 覆蓋元件變數
@use "element-plus/theme-chalk/src/common/var.scss" as *;

// 方法一：使用 scss 變數覆蓋（需要 sass）
$el-button-primary-bg-color: #409EFF;
```

### 7.2 線上主題定製

訪問 Element Plus 官方主題定製頁面：https://element-plus.org/zh-TW/guide/theming.html

可以在線選擇主題色、邊框顏色、圓角等參數，下載定製後的 CSS 檔案。

### 7.3 暗黑模式

Element Plus 內建支援暗黑模式：

```typescript
// 開啟暗黑模式
import 'element-plus/theme-chalk/dark/css-vars.css'
```

在 HTML 標籤上添加 class：

```html
<html class="dark">
  <!-- 你的應用 -->
</html>
```

## 八、常見問題

### 8.1 安裝失敗

**問題**：npm install 安裝失敗

**解決方案**：

1. 清除 npm 緩存：`npm cache clean --force`
2. 檢查網路，可以使用國內鏡像源：

```bash
# 設定 npm 鏡像
npm config set registry https://registry.npmmirror.com

# 或者使用 yarn
yarn config set registry https://registry.npmmirror.com
```

### 8.2 樣式不生效

**問題**：元件樣式顯示異常

**解決方案**：

1. 確保已引入樣式檔案：`import 'element-plus/dist/index.css'`
2. 檢查建構配置是否正確處理了 CSS
3. 清除瀏覽器緩存並重新建構

### 8.3 TypeScript 類型錯誤

**問題**：引入元件時提示類型錯誤

**解決方案**：

1. 確保安裝了最新版本的 TypeScript：`npm install -D typescript`
2. 在 tsconfig.json 中添加配置：

```json
{
  "compilerOptions": {
    "moduleResolution": "node",
    "allowSyntheticDefaultImports": true,
    "esModuleInterop": true
  }
}
```

### 8.4 元件不顯示

**問題**：元件引入後不顯示

**解決方案**：

1. 檢查是否正確使用了 app.use(ElementPlus)
2. 確認 Vue 版本是否在 3.2.0 以上
3. 檢查瀏覽器控制台是否有錯誤訊息
