# 安装指南

## 各发行版安装说明

### Arch Linux

Arch 用户可以参考 Arch Wiki 上的完整安装说明：
[https://wiki.archlinux.org/title/Waydroid](https://wiki.archlinux.org/title/Waydroid)

### Postmarket OS

Postmarket 社区提供了安装说明和故障排除提示：
[https://wiki.postmarketos.org/wiki/Waydroid](https://wiki.postmarketos.org/wiki/Waydroid)

### Fedora

Waydroid 可以从官方软件包仓库安装：

```bash
sudo dnf install waydroid
```

安装后，从应用程序菜单启动 Waydroid，并在 OTA 字段中粘贴以下 URL 进行初始化：

系统 OTA：`https://ota.waydro.id/system`

供应商 OTA：`https://ota.waydro.id/vendor`

#### Silverblue/Kinoite/...

对于 Fedora 不可变变体，使用 `rpm-ostree` 代替 `dnf`：

```bash
rpm-ostree install waydroid
```

### Ubuntu/Debian 及衍生版本

确保已启用 Wayland 会话（Ubuntu 22.04+）

* 安装前置依赖

```bash
sudo apt install curl ca-certificates -y
```

* 添加官方仓库

```bash
curl -s https://repo.waydro.id | sudo bash
```

如果脚本无法检测到您的发行版，可以通过附加 `-s <DISTRO>` 提供有效选项。当前支持的值包括：**mantic**、**focal**、**jammy**、**kinetic**、**lunar**、**noble**、**plucky**、**questing**、**bookworm**、**bullseye**、**trixie**、**sid**

* 安装 waydroid

```bash
sudo apt install waydroid -y
```

然后从应用程序菜单启动 Waydroid。

## 故障排除

### 手动启动 Waydroid

要不使用 systemctl 启动 Waydroid，需要遵循以下简单步骤：

**首先启动容器：**

```bash
sudo waydroid container start
```

**然后在新的终端标签页中启动 waydroid 会话（无需 ***sudo*****）：

```bash
waydroid session start
```

启动后看到 "Android with user 0 is ready" 时，就可以从应用程序菜单启动应用了。

### 全屏模式启动 Waydroid

```bash
waydroid show-full-ui
```

### 多窗口模式启动 Waydroid

首先在 Waydroid 会话运行时设置属性：

```bash
waydroid prop set persist.waydroid.multi_windows true
```

然后重启会话：

```bash
waydroid session stop
```

之后启动的应用将以多窗口模式运行。

### 重新安装 Waydroid

如果需要完全重新安装：

```bash
waydroid session stop
sudo waydroid container stop
sudo apt remove waydroid  # 或对应发行版的卸载命令
# 重启后执行清理
sudo rm -rf /var/lib/waydroid /home/.waydroid ~/waydroid ~/.share/waydroid ~/.local/share/applications/*aydroid* ~/.local/share/waydroid
```
