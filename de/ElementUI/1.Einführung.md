# Element UI Einführung

Element UI ist eine auf Vue 2.0 basierende UI-Komponentenbibliothek, die Entwicklern umfassende Desktop-Komponentenlösungen bietet.

## Was ist Element UI

Element UI ist eine von Ele.me entwickelte und gepflegte Vue 2.0-Komponentenbibliothek. Sie bietet einen vollständigen Satz von Designspezifikationen und Komponentenbibliotheken, um Entwicklern zu helfen, schnell Desktop-Anwendungen zu erstellen.

Als Vue 2.0-Companion-UI-Framework bietet Element UI eine einfache und benutzerfreundliche Komponentenkapselung sowie eine gute Entwicklungserfahrung.

## Hauptfunktionen

### Vollständiges Komponentensystem

Element UI bietet über 70 hochwertige Komponenten für alle Aspekte der Desktop-Anwendungsentwicklung:

- Basiskomponenten: Schaltflächen, Symbole, Layout, Container
- Formularkomponenten: Eingabe, Auswahl, Datumsauswahl, Schalter, Slider
- Datenanzeige: Tabelle, Baum, Paginierung, Tags
- Feedback-Komponenten: Dialog, Schublade, Nachricht, Laden
- Navigationskomponenten: Menü, Tabs, Breadcrumb, Schritte

### Vollständige Vue 2.0-Unterstützung

Element UI ist vollständig auf Vue 2.0 aufgebaut:

- Komponentenbasierte Entwicklung
- Reaktive Datenbindung
- Vollständige TypeScript-Unterstützung
- Gute Kompatibilität

### Import bei Bedarf

Element UI unterstützt das Importieren nur der verwendeten Komponenten:

- Verwendung des babel-plugin-component-Plugins
- Manueller Import bei Bedarf
- Reduzierung der Paketgröße

### Internationalisierungsunterstützung

Eingebaute Unterstützung für mehrere Sprachpakete, um die Anforderungen internationaler Anwendungen zu erfüllen.

### Theme-Anpassung

Unterstützt globale Theme-Anpassung:

- Theme-Farbkonfiguration
- Komponentenstil-Überschreibung
- Online-Theme-Generator

## Designprinzipien

Element UI folgt vier Hauptdesignprinzipien:

### Konsistenz

- Konsistent mit dem echten Leben: Konsistenz mit Prozessen und Logik des echten Lebens, unter Berücksichtigung der Gewohnheiten der Benutzer bei Sprache und Konzepten
- Konsistent in der Oberfläche: Alle Elemente und Strukturen müssen konsistent sein, einschließlich Designstile, Symbole und Text, Elementpositionen usw.

### Feedback

- Kontroll-Feedback: Benutzer können ihre Operationen durch Oberflächenstile und interaktive Effekte klar wahrnehmen
- Seiten-Feedback: Nach Operationen den aktuellen Zustand durch Änderungen der Seitenelemente anzeigen

### Effizienz

- Prozesse vereinfachen: Intuitive Betriebsabläufe gestalten
- Klar und eindeutig: Klare Sprachausdrücke für schnelles Verständnis und Entscheidungsfindung
- Benutzern helfen zu erkennen: Einfache Oberflächen ermöglichen schnelles Erkennen statt Erinnern, reduzieren die Gedächtnisbelastung

### Kontrollierbarkeit

- Benutzerentscheidungen: Vorschläge oder Sicherheitshinweise geben, aber nicht anstelle des Benutzers entscheiden
- Kontrollierbare Ergebnisse: Benutzer können Operationen frei durchführen, einschließlich Rückgängig machen, Zurücksetzen und Beenden der aktuellen Operation

## Anwendungsbereiche

Element UI eignet sich für:

- Unternehmensinterne Managementsysteme
- E-Commerce-Backend-Managementsysteme
- Datenvisualisierungsplattformen
- CRM-Kundensysteme
- Jedes Vue 2-Projekt mit Desktop-UI

## Beziehung zu Element Plus

Element UI ist der Vorgänger von Element Plus (Vue 3-Version):

- Element UI: Für Vue 2 entwickelt
- Element Plus: Für Vue 3 neu entwickelt

Wenn Ihr Projekt Vue 2 verwendet, fahren Sie mit Element UI fort. Wenn Sie auf Vue 3 aktualisieren, migrieren Sie zu Element Plus.

## Nächste Schritte

Nach der Installation von Element UI können Sie:

- Komponentendokumentation anzeigen und die Verwendung jeder Komponente lernen
- Designrichtlinien referenzieren, um die Konsistenz der Oberfläche zu wahren
- Theme-Anpassung verwenden, um einen personalisierten Stil zu erstellen
- Internationalisierungsfunktionen erkunden, um mehrsprachige Anwendungen zu entwickeln

Weitere Informationen finden Sie auf der offiziellen Element UI-Website.
