# Rukovodstvo po ustanovke Nginx

Rukovodstvo po ustanovke Nginx na vsekh osnovnykh platformakh.

---

## Linux

### Ubuntu / Debian

```bash
# Obnovit pakety
sudo apt update

# Ustanovit Nginx
sudo apt install nginx

# Zapustit Nginx
sudo systemctl start nginx

# Vklupit pri zagruzke
sudo systemctl enable nginx
```

### CentOS / RHEL

```bash
# Ustanovit EPEL
sudo yum install epel-release

# Ustanovit Nginx
sudo yum install nginx

# Zapustit
sudo systemctl start nginx
```

### Fedora

```bash
sudo dnf install nginx
sudo systemctl start nginx
```

---

## macOS

```bash
brew install nginx
brew services start nginx
```

---

## Windows

1. Skachat s [nginx.org](http://nginx.org/en/download.html)
2. Raspakovat v `C:\nginx`

```powershell
cd C:\nginx
start nginx
```

---

## Docker

```bash
docker run -d -p 80:80 --name nginx nginx:latest
```

---

## Proverka

```bash
nginx -v
nginx -t
curl http://localhost
```

---

## Konfiguraciya

| OS | Put |
|----|-----|
| Ubuntu/Debian | `/etc/nginx/nginx.conf` |
| CentOS/RHEL | `/etc/nginx/nginx.conf` |

---

## Udalenie

```bash
sudo apt remove nginx
```
