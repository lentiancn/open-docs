# Mockito 簡介

領先的 Java 單元測試模擬框架。

## 什麼是 Mockito？

Mockito 是領先的 Java 單元測試模擬框架。它使開發人員能夠建立和驗證模擬物件，以編寫隔離的測試並消除對外部元件的依賴。

### 主要特點

- **直觀的 API** - 易於學習和使用
- **靈活的模擬** - 為類別和介面建立模擬
- **驗證** - 驗證方法調用
- **Stubbing** - 定義預定義的返回值
- **參數匹配器** - 靈活的參數驗證

## 核心概念

- **Mock** - 模擬真實物件行為的代理物件
- **Stub** - 對方法調用的預定義響應
- **驗證** - 檢查是否調用了特定方法
