# Flutter 簡介

Flutter 是 Google 推出的開源 UI 工具包。

## 什麼是 Flutter？

Flutter 是一個用於構建美觀、原生編譯的移動、Web、桌面和嵌入式設備應用的開源 UI 工具包，僅需單一代碼庫。Flutter 使用 Dart 編程語言，以其豐富的組件庫和高性能渲染而聞名。

## 主要功能

- **跨平台**：單一代碼庫支持 iOS、Android、Web、桌面
- **熱重載**：即時預覽更改，無需丟失狀態
- **豐富組件**：Material Design 和 Cupertino 組件
- **高性能**：使用 Skia 圖形引擎的原生性能
- **表達性 UI**：靈活的聲明式 UI 構建
- **開源免費**：完全免費使用

## 核心技術

### Dart 語言

Flutter 使用 Dart，這是一種面向對象的編程語言：

- **強類型**：靜態類型檢查
- **JIT 和 AOT**：即時編譯和預編譯
- **垃圾回收**：自動內存管理
- **協程**：異步編程支持

### 組件系統

Flutter 中一切都是組件：

- **StatelessWidget**：無狀態組件
- **StatefulWidget**：有狀態組件
- **InheritedWidget**：數據傳遞
- **RenderObjectWidget**：渲染對象

## 應用場景

- **移動應用**：iOS 和 Android 原生體驗
- **Web 應用**：響應式 Web 應用
- **桌面應用**：Windows、macOS、Linux 桌面應用
- **嵌入式**：物聯網設備界面
