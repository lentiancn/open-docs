# Oracle 資料庫

> 企業級關聯式資料庫管理系統

---

## 簡介

Oracle Database 是由 Oracle 公司開發的專屬多模型資料庫管理系統。它是全球最廣泛使用的企業資料庫系統之一，以其可靠性、可擴展性和全面的功能集著稱。

---

## 主要特點

- **高效能**：優化的查詢處理和索引
- **可擴展性**：支援大量資料和並發用戶
- **安全性**：進階安全功能，包括加密、審計
- **高可用性**：真正應用叢集 (RAC)、Data Guard
- **PL/SQL**：用於複雜邏輯的程式語言擴展
