# PlantUML Resumen

> Herramienta de código abierto para dibujar diagramas UML desde descripciones de texto

---

## Descripción del Proyecto

PlantUML es una herramienta de código abierto que permite a los usuarios crear diagramas a partir de un lenguaje de texto simple. Soporta varios tipos de diagramas incluyendo UML, diagramas de red y diagramas de flujo.

---

## Características Principales

- **Múltiples tipos de diagramas**: UML, Actividad, Secuencia, Clase
- **Basado en texto**: Definir diagramas usando sintaxis de texto simple
- **Soporte IDE**: Plugins para VS Code, IntelliJ, Eclipse
- **Opciones de exportación**: PNG, SVG, PDF
- **Código abierto**: Gratuito y impulsado por la comunidad
