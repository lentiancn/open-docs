# Oracle FAQ

## Questions Générales

### Qu'est-ce qu'Oracle Database?

Oracle Database est un système de gestion de base de données relationnelle propriétaire développé par Oracle Corporation.

### Qu'est-ce que PL/SQL?

PL/SQL est l'extension du langage procédural d'Oracle pour SQL.

---

## Installation

### Configuration requise?

- Minimum 2GB de RAM
- Minimum 8GB d'espace disque

### ORA-12541: TNS:no listener?

Démarrez le listener:
```bash
lsnrctl start
```

## Performance

### Comment vérifier la performance des requêtes?

```sql
EXPLAIN PLAN FOR your_query;
SELECT * FROM TABLE(DBMS_XPLAN.DISPLAY);
```
