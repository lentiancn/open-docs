# 설치 가이드

이 문서에서는 Angular의 설치와 환경 설정 방법을 설명합니다. 빠르게 개발 환경을 구축할 수 있도록 도와줍니다.

## 환경 요구사항

Angular를 설치하기 전에 다음 환경 요구사항을 확인하세요:

- **Node.js**: 18.10.0 이상 권장
- **npm**: Node.js에 포함되어 있으며, 6.x 이상
- **운영체제**: Windows, macOS, Linux

## Angular CLI 설치

Angular CLI는 Angular 공식 명령줄 도구로, Angular 프로젝트를 생성하고 관리하는 데 사용됩니다.

### 전역 설치

```bash
npm install -g @angular/cli
```

### 설치 확인

```bash
ng version
```

설치가 성공하면 Angular CLI 버전 정보가 표시됩니다.

### 특정 버전 설치

특정 버전을 설치해야 하는 경우:

```bash
npm install -g @angular/cli@17
```

## 새 프로젝트 생성

### CLI로 프로젝트 생성

```bash
ng new my-app
```

my-app이라는 이름의 새 프로젝트가 생성되고, 의존성이 자동으로 설치됩니다.

### 옵션指定으로 생성

```bash
# 생성하고 설치 건너뛰기
ng new my-app --skip-install

# SCSS 스타일 사용
ng new my-app --style=scss

# 라우팅 활성화
ng new my-app --routing

# 독립형 컴포넌트 생성 (Angular 14+)
ng new my-app --standalone
```

## 개발 환경 설정

### 국내 미러 사용

네트워크 문제로 인해 국내 미러 설정이 권장됩니다:

```bash
# npm 미러 설정
npm config set registry https://registry.npmmirror.com

# 또는 nrm 도구 사용
npm install -g nrm
nrm use taobao
```

### Node.js 설치

**nvm으로 설치 (권장):**

```bash
# nvm 설치
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Node.js LTS 버전 설치
nvm install --lts
nvm use --lts
```

**Windows에서 nvm-windows 사용:**

https://github.com/coreybutler/nvm-windows에서 설치 프로그램을 다운로드하세요.

## 개발 서버 실행

### 프로젝트 디렉토리로 이동

```bash
cd my-app
```

### 개발 서버 시작

```bash
ng serve
```

기본 접근 주소: http://localhost:4200

### 실시간 재로딩

Angular CLI는 실시간 재로딩을 지원합니다. 코드를 수정하면 브라우저가 자동으로 새로고침됩니다.

## 프로덕션 빌드

### 빌드 명령

```bash
ng build
```

빌드 산출물은 dist 디렉토리에 출력됩니다.

### 프로덕션 최적화

```bash
ng build --configuration=production
```

이렇게 하면 코드 압축, tree-shaking 등의 최적화가 활성화됩니다.

## 자주 사용하는 CLI 명령

| 명령 | 설명 |
|------|------|
| ng generate component xxx | 컴포넌트 생성 |
| ng generate service xxx | 서비스 생성 |
| ng generate module xxx | 모듈 생성 |
| ng generate directive xxx | 디렉티브 생성 |
| ng generate pipe xxx | 파이프 생성 |
| ng generate guard xxx | 라우팅 가드 생성 |

## 개발 도구

### VS Code 확장

- Angular Language Service: 템플릿 타입 검사 및 자동완성 제공
- Angular Snippets: 자주 사용하는 코드 스니펫 제공
- Prettier: 코드 포맷팅

### 디버깅

Angular는 Angular DevTools 브라우저 확장을 제공하여 디버깅과 성능 분석에 사용합니다.

## Angular CLI 제거

제거가 필요한 경우:

```bash
npm uninstall -g @angular/cli
```
