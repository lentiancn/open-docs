# Nacos 簡介

Nacos（Dynamic Naming and Configuration Service）是由阿里巴巴開發的服務發現和動態配置管理平台。

## 主要功能

- 服務發現
- 配置管理
- 動態 DNS
- 健康檢查

## 安裝

```bash
# 獨立模式
./startup.sh -m standalone

# Docker
docker run --name nacos -d -p 8848:8848 nacos/nacos-server:v2.3.0
```
