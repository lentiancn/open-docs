# JDK FAQ

## Installation

### Q: java command not found?
**A**: Verify JAVA_HOME and PATH are configured correctly, reopen terminal

### Q: How to switch between multiple JDK versions?
**A**: 
- Windows: Modify JAVA_HOME environment variable
- macOS: Use jenv to manage multiple versions

## Usage

### Q: How to find JDK path?
**A**: 
```bash
which java
echo $JAVA_HOME
```

### Q: Memory insufficient?
**A**: 
```bash
java -Xmx4g -Xms2g Hello
```

## Version

### Q: Difference between JDK and JRE?
**A**: 
- JDK = JRE + development tools
- JRE = JVM + class library

### Q: Which version for production?
**A**: Recommended LTS versions: JDK 21, JDK 17, JDK 11
