# Manual de usuario OpenClaw

## Conceptos básicos

### Gateway

El núcleo de OpenClaw. Maneja mensajes, llamadas IA y conexiones de canales.

### Canales

Apps de chat: WhatsApp, Telegram, Discord, etc.

### Agentes

Instancias independientes de asistentes IA. Puedes tener varios.

## Operaciones

### Iniciar/Detener

```bash
openclaw gateway start
openclaw gateway stop
openclaw gateway status
```

### Enviar mensajes

```bash
openclaw message send --target +15555550123 --message "Hola"
```

### Configuración

Archivo: `~/.openclaw/openclaw.json`

Métodos:
- `openclaw configure`
- `openclaw config set xxx`
- Editar directamente
- Web UI: http://127.0.0.1:18789/

## Configuración común

### Restringir acceso

```json5
{
  channels: {
    whatsapp: { allowFrom: ["+1234567890"] }
  }
}
```

### Múltiples agentes

```json5
{
  agents: {
    list: [
      { id: "home", default: true },
      { id: "work" }
    ]
  }
}
```

## Problemas

```bash
openclaw logs
openclaw doctor
```
