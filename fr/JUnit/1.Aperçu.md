# JUnit 5 - Aperçu

## Présentation

JUnit 5 est la prochaine génération de JUnit, un framework de test convivial pour Java et JVM.

## Qu'est-ce que JUnit?

JUnit est un framework open source pour les tests unitaires. C'est le standard de fait pour tester les applications Java.

## Caractéristiques principales

- Architecture modulaire (Platform, Jupiter, Vintage)
- Tests paramétrés intégrés
- Tests imbriqués
- Modèle d'extension flexible
