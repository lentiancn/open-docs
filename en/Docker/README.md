## Quick Start

### Installation

**Windows/macOS**:
Download Docker Desktop: https://www.docker.com/products/docker-desktop

**Linux (Ubuntu)**:
```bash
curl -fsSL https://get.docker.com | sh
```

### Run Your First Container

```bash
docker run hello-world
```

### Run Nginx

```bash
docker run -d -p 8080:80 nginx
```

### Images

```bash
docker pull nginx
```

### Containers

```bash
docker ps
```
