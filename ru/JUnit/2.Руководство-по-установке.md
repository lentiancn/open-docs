# JUnit Руководство по установке

JUnit можно добавить в проект несколькими способами: через Maven, Gradle или прямую загрузку JAR-файлов. Ниже приведены подробные инструкции для каждой версии.

## Установка JUnit 5

### Настройка Maven

Добавьте следующие зависимости в ваш pom.xml:

```xml
<!-- Модуль-агрегатор JUnit Jupiter -->
<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter</artifactId>
    <version>5.14.3</version>
    <scope>test</scope>
</dependency>
```

При использовании Maven Surefire Plugin для запуска тестов добавьте следующую конфигурацию:

```xml
<plugin>
    <groupId>org.apache.maven.plugins</groupId>
    <artifactId>maven-surefire-plugin</artifactId>
    <version>3.5.2</version>
</plugin>
```

### Настройка Gradle

Добавьте зависимость в ваш build.gradle:

```groovy
testImplementation 'org.junit.jupiter:junit-jupiter:5.14.3'
```

Для запуска тестов:

```groovy
test {
    useJUnitPlatform()
}
```

### Прямая загрузка JAR

Загрузите необходимые JAR-файлы из Maven Central:

- junit-jupiter-api: API JUnit Jupiter
- junit-jupiter-engine: Движок JUnit Jupiter
- junit-platform-console-standalone: автономный консольный запускатель

## Установка JUnit 4

### Настройка Maven

```xml
<dependency>
    <groupId>junit</groupId>
    <artifactId>junit</artifactId>
    <version>4.13.2</version>
    <scope>test</scope>
</dependency>
```

### Настройка Gradle

```groovy
testImplementation 'junit:junit:4.13.2'
```

### Прямая загрузка JAR

Загрузите junit-4.13.2.jar из Maven Central:

https://repo1.maven.org/maven2/junit/junit/4.13.2/junit-4.13.2.jar

## Установка JUnit Vintage

Для запуска тестов JUnit 4 на платформе JUnit 5 добавьте зависимость движка Vintage:

### Maven

```xml
<dependency>
    <groupId>org.junit.vintage</groupId>
    <artifactId>junit-vintage-engine</artifactId>
    <version>5.14.3</version>
    <scope>test</scope>
</dependency>
```

### Gradle

```groovy
testImplementation 'org.junit.vintage:junit-vintage-engine:5.14.3'
```

## Совместимость версий

- JUnit 5 требует Java 8 или выше
- JUnit 4 требует Java 5 или выше
- JUnit Vintage 5.14.3 требует JUnit 4.12 или выше

## Проверка установки

Создайте тестовый класс для проверки установки:

```java
import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;

class SampleTest {
    @Test
    void testExample() {
        assertEquals(2, 1 + 1);
    }
}
```

Запустите тесты:

```bash
mvn test
# или
gradle test
```
