## 布局

### 容器

```html
<!-- 固定宽度容器 -->
<div class="container">内容</div>

<!-- 满宽容器 -->
<div class="container-fluid">内容</div>
```

### 网格系统

```html
<div class="row">
    <div class="col">列</div>
    <div class="col">列</div>
</div>
```

### 响应式断点

| 断点 | 类名前缀 | 屏幕宽度 |
|------|----------|----------|
| 小 | col-sm- | ≥ 576px |
| 中 | col-md- | ≥ 768px |
| 大 | col-lg- | ≥ 992px |
| 超大 | col-xl- | ≥ 1200px |

## 内容

### 标题

```html
<h1>标题 1</h1>
<h1 class="display-1">Display 1</h1>
```

### 文本

```html
<p class="text-primary">主要颜色</p>
<p class="text-success">成功颜色</p>
<p class="text-center">居中</p>
```

### 背景

```html
<div class="bg-primary text-white">主要背景</div>
```

## 组件

### 按钮

```html
<button class="btn btn-primary">主要</button>
<button class="btn btn-secondary">次要</button>
<button class="btn btn-success">成功</button>
<button class="btn btn-danger">危险</button>
<button class="btn btn-warning">警告</button>
<button class="btn btn-info">信息</button>

<!-- 轮廓按钮 -->
<button class="btn btn-outline-primary">主要</button>

<!-- 按钮尺寸 -->
<button class="btn btn-primary btn-lg">大</button>
<button class="btn btn-primary btn-sm">小</button>

<!-- 按钮组 -->
<div class="btn-group">
    <button class="btn btn-primary">左</button>
    <button class="btn btn-primary">中</button>
    <button class="btn btn-primary">右</button>
</div>
```

### 卡片

```html
<div class="card" style="width: 18rem;">
    <img src="..." class="card-img-top" alt="...">
    <div class="card-body">
        <h5 class="card-title">卡片标题</h5>
        <p class="card-text">卡片内容。</p>
        <a href="#" class="btn btn-primary">按钮</a>
    </div>
</div>
```

### 徽章

```html
<h1>标题 <span class="badge bg-primary">新</span></h1>
<span class="badge bg-success">成功</span>
```

### 进度条

```html
<div class="progress">
    <div class="progress-bar" style="width: 50%">50%</div>
</div>
```

### 警告框

```html
<div class="alert alert-success">成功！</div>
<div class="alert alert-danger">危险！</div>

<!-- 可关闭的警告框 -->
<div class="alert alert-warning alert-dismissible fade show" role="alert">
    <strong>注意！</strong>
    <button type="button" class="btn-close" data-bs-dismiss="alert"></button>
</div>
```

## 表单

### 基本表单

```html
<form>
    <div class="mb-3">
        <label class="form-label">邮箱</label>
        <input type="email" class="form-control">
    </div>
    <div class="mb-3">
        <label class="form-label">密码</label>
        <input type="password" class="form-control">
    </div>
    <div class="mb-3 form-check">
        <input type="checkbox" class="form-check-input" id="check1">
        <label class="form-check-label" for="check1">记住我</label>
    </div>
    <button type="submit" class="btn btn-primary">提交</button>
</form>
```

### 选择框

```html
<select class="form-select">
    <option selected>请选择</option>
    <option value="1">选项 1</option>
    <option value="2">选项 2</option>
</select>
```

### 浮动标签

```html
<div class="form-floating">
    <input type="email" class="form-control" id="floatingInput" placeholder="name@example.com">
    <label for="floatingInput">邮箱地址</label>
</div>
```

## 导航

### 基本导航

```html
<ul class="nav">
    <li class="nav-item"><a class="nav-link active" href="#">首页</a></li>
    <li class="nav-item"><a class="nav-link" href="#">关于</a></li>
    <li class="nav-item"><a class="nav-link" href="#">联系</a></li>
</ul>
```

### 导航栏

```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">品牌</a>
        <button class="navbar-toggler" data-bs-toggle="collapse" data-bs-target="#navbarNav">
            <span class="navbar-toggler-icon"></span>
        </button>
        <div class="collapse navbar-collapse" id="navbarNav">
            <ul class="navbar-nav">
                <li class="nav-item"><a class="nav-link" href="#">首页</a></li>
            </ul>
        </div>
    </div>
</nav>
```

## JavaScript 组件

### 模态框

```html
<button class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#myModal">打开</button>

<div class="modal" id="myModal">
    <div class="modal-dialog">
        <div class="modal-content">
            <div class="modal-header">
                <h5 class="modal-title">标题</h5>
                <button class="btn-close" data-bs-dismiss="modal"></button>
            </div>
            <div class="modal-body">内容</div>
            <div class="modal-footer">
                <button class="btn btn-secondary" data-bs-dismiss="modal">关闭</button>
                <button class="btn btn-primary">保存</button>
            </div>
        </div>
    </div>
</div>
```

### 下拉菜单

```html
<div class="dropdown">
    <button class="btn btn-secondary dropdown-toggle" data-bs-toggle="dropdown">下拉</button>
    <ul class="dropdown-menu">
        <li><a class="dropdown-item" href="#">选项 1</a></li>
        <li><a class="dropdown-item" href="#">选项 2</a></li>
    </ul>
</div>
```

### 轮播

```html
<div class="carousel slide" data-bs-ride="carousel">
    <div class="carousel-inner">
        <div class="carousel-item active">
            <img src="..." class="d-block w-100" alt="...">
        </div>
    </div>
</div>
```

### 手风琴

```html
<div class="accordion" id="accordion">
    <div class="accordion-item">
        <h2 class="accordion-header">
            <button class="accordion-button" data-bs-toggle="collapse" data-bs-target="#collapse">
                可折叠项
            </button>
        </h2>
        <div id="collapse" class="accordion-collapse collapse show">
            <div class="accordion-body">内容</div>
        </div>
    </div>
</div>
```
