# OpenClaw User Manual

## Basic Concepts

### Gateway
Core process connecting chat apps to AI agents.

### Channel
Specific chat platform (WhatsApp, Telegram, Discord).

### Agent
AI assistant (Pi, Claude, GPT-4).

### Session
Conversation context with AI.

## Quick Start

```bash
# Start Gateway
openclaw gateway --port 18789

# Open dashboard
openclaw dashboard
```

## Channel Management

```bash
# List channels
openclaw channels list

# Add channel
openclaw channels add whatsapp

# Login
openclaw channels login

# Remove channel
openclaw channels remove whatsapp
```

## Message Management

```bash
# Send message
openclaw message send --target +15555550123 --message "Hello"

# List sessions
openclaw sessions list

# Delete session
openclaw sessions delete --session-id <id>
```

## Configuration

```bash
# Get config
openclaw config get

# Set config
openclaw config set channels.whatsapp.allowFrom "+15555550123"
```

## Session Management

```bash
# List sessions
openclaw sessions list

# Create session
openclaw sessions create --user <user>

# Delete session
openclaw sessions delete --session-id <id>
```

## Node Management

```bash
# Pair mobile
openclaw nodes pair

# List nodes
openclaw nodes list

# Remove node
openclaw nodes remove --node-id <id>
```

## Security

```bash
# Add to whitelist
openclaw security allow +15555550123

# List whitelist
openclaw security list
```

## Troubleshooting

```bash
# View logs
openclaw logs

# Check status
openclaw status
```

## Common Commands

| Command | Description |
|---------|-------------|
| `openclaw gateway start` | Start Gateway |
| `openclaw dashboard` | Open Control UI |
| `openclaw channels list` | List channels |
| `openclaw sessions list` | List sessions |
| `openclaw message send` | Send message |
| `openclaw config get` | Get config |
