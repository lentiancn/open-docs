# Flutter 是什么？

Flutter 是 Google 推出的一款**跨平台 UI 开发框架**，可以用**同一套代码**同时做出 iOS、Android、Web（网页）、Windows、macOS、Linux 的应用。

---

## 为什么要用 Flutter？

### 1. 省时省力

以前开发手机应用，iOS 和 Android 需要两套人马、两份代码。用 Flutter 只需要写一次，就能同时跑在多个平台上。

### 2. 开发快

Flutter 支持**热重载**（Hot Reload），代码修改后按一下保存，应用就会瞬间更新，不用等编译重启。

### 3. 界面漂亮

内置了两套设计语言：
- **Material Design** - Google 风格
- **Cupertino** - 苹果风格

还支持完全自定义 UI，想怎么画就怎么画。

### 4. 性能好

Flutter 不像其他框架那样需要 JavaScript 桥接，它是直接编译成原生 ARM 代码，运行起来和原生应用一样流畅。

---

## 核心概念

### Widget

Flutter 里所有东西都是 **Widget**（组件）。按钮是 Widget，文字是 Widget，布局容器也是 Widget。

```dart
// 这就是一个最简单的 Flutter 应用
import 'package:flutter/material.dart';

void main() {
  runApp(
    const Center(
      child: Text('你好，Flutter！'),
    ),
  );
}
```

### 两种 Widget

| 类型 | 说明 | 什么时候用 |
|------|------|-----------|
| StatelessWidget | 无状态 | 界面不会变化 |
| StatefulWidget | 有状态 | 界面会随数据变化 |

```dart
// StatefulWidget 示例：带计数器的按钮
class Counter extends StatefulWidget {
  const Counter({super.key});

  @override
  State<Counter> createState() => _CounterState();
}

class _CounterState extends State<Counter> {
  int count = 0;  // 数据

  @override
  Widget build(BuildContext context) {
    return Column(
      mainAxisSize: MainAxisSize.min,
      children: [
        Text('计数：$count'),
        ElevatedButton(
          onPressed: () => setState(() => count++),  // 点击后更新数据
          child: const Text('点我'),
        ),
      ],
    );
  }
}
```
