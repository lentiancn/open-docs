# Flutter

Flutter - 跨平台 UI 框架

## 概述

Flutter 是 Google 開發的開源 UI 軟體開發工具包，用於從單個程式碼庫構建精美的、原生編譯的行動、Web 和桌面應用程式。

## 特性

- **跨平台**：一套程式碼同時支援 iOS、Android、Web、Windows、macOS、Linux
- **熱重載**：快速開發迭代，保留應用狀態
- **豐富元件**：Material Design 和 Cupertino 元件庫
- **原生效能**：編譯為原生 ARM 程式碼
- **聲明式 UI**：現代 UI 開發範式
- **開源免費**：BSD 授權

## 快速開始

### 安裝

```bash
# 克隆 Flutter
git clone https://github.com/flutter/flutter.git -b stable

# 加入 PATH
export PATH="$PATH:$HOME/flutter/bin"

# 檢查安裝
flutter doctor
```

### 建立專案

```bash
flutter create my_app
cd my_app
flutter run
```
