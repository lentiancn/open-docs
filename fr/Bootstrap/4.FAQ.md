# FAQ Bootstrap

> Foire aux questions

---

## Problèmes d'installation

### Le CDN ne se charge pas ?

**Solutions**:

1. Vérifier la connexion réseau
2. Utiliser un CDN alternatif:
```html
<link href="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

### Les styles ne fonctionnent pas après l'installation npm ?

Assurez-vous d'importer correctement:

```javascript
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap';
```

---

## Problèmes de styles

### Les styles des composants ne fonctionnent pas ?

**Causes courantes**:

1. Bootstrap CSS non importé
2. JavaScript non chargé correctement (requis pour certains composants)
3. Priorité CSS écrasée

### Le responsive ne fonctionne pas ?

Assurez-vous d'ajouter la méta balise viewport:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

---

## Problèmes de composants

### Le menu déroulant ne fonctionne pas ?

**Liste de contrôle**:

1. Importer Bootstrap JS (pas seulement le CSS)
2. Utiliser la structure HTML correcte
3. Vérifier les erreurs JavaScript

### La modale ne s'ouvre pas ?

Assurez-vous que:
1. Bootstrap JS est chargé
2. Les attributs data-bs-toggle et data-bs-target sont corrects
3. La modale existe au chargement de la page

---

## Problèmes de personnalisation

### Comment personnaliser les couleurs du thème ?

**Méthode 1: Utiliser Sass**

```scss
$primary: #ff6600;
@import "bootstrap";
```

**Méthode 2: Utiliser les variables CSS**

```css
:root {
  --bs-primary: #ff6600;
}
```

---

## Problèmes de compatibilité

### IE 11 non supporté ?

Bootstrap 5 ne supporte plus IE 11. Si vous avez besoin du support IE 11, utilisez Bootstrap 4.

---

## Problèmes de performance

### Comment réduire la taille des fichiers ?

1. Importer uniquement les composants nécessaires (Sass)
2. Utiliser les versions de production (.min.css/.min.js)
3. Activer la compression gzip

---

## Conflits avec d'autres frameworks

### Utiliser avec Vue/React ?

Bootstrap peut être utilisé avec n'importe quel framework JavaScript, mais attention:

1. Certains composants dépendent de jQuery (Bootstrap 5 n'en dépend plus)
2. Utiliser des wrappers Bootstrap spécifiques au framework (ex: react-bootstrap)
