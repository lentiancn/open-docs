# Spring Boot Installationsanleitung

Diese Anleitung beschreibt, wie Sie die Spring Boot-Entwicklungsumgebung unter Windows, macOS und Linux installieren und einrichten.

## Systemanforderungen

### Java-Versionsanforderungen

| Spring Boot Version | Minimale Java-Version |
|---------------------|---------------------|
| 3.x                 | Java 17+            |
| 2.7.x               | Java 8+             |
| 2.5.x               | Java 8+             |

**Hinweis:** Spring Boot 3.x erfordert Java 17 oder höher. Es wird empfohlen, Java 21 (LTS) für beste Kompatibilität zu verwenden.

## Java installieren

### Windows

**Option 1: Installation von Oracle/OpenJDK**
1. JDK von [Oracle](https://www.oracle.com/java/technologies/downloads/) oder [Adoptium](https://adoptium.net/) herunterladen
2. Installer ausführen
3. Umgebungsvariablen festlegen:
   ```
   JAVA_HOME=C:\Program Files\Java\jdk-21
   PATH=%JAVA_HOME%\bin;%PATH%
   ```

**Option 2: Mit Chocolatey**
```powershell
choco install openjdk21
```

**Option 3: Mit Winget**
```powershell
winget install Oracle.JDK.21
```

Überprüfen:
```cmd
java -version
```

### macOS

**Option 1: Mit Homebrew**
```bash
brew install openjdk@21
sudo ln -sfn $(brew --prefix)/opt/openjdk@21/libexec/openjdk.jdk /Library/Java/JavaVirtualMachines/openjdk-21.jdk
echo 'export JAVA_HOME=$(/usr/libexec/java_home -v 21)' >> ~/.zshrc
source ~/.zshrc
```

**Option 2: Mit SDKMAN**
```bash
curl -s "https://get.sdkman.io" | bash
sdk install java 21.0.2-tem
```

Überprüfen:
```bash
java -version
```

### Linux

**Ubuntu/Debian:**
```bash
sudo apt update
sudo apt install openjdk-21-jdk
export JAVA_HOME=/usr/lib/jvm/java-21-openjdk-amd64
export PATH=$JAVA_HOME/bin:$PATH
```

**RHEL/CentOS/Fedora:**
```bash
sudo dnf install java-21-openjdk-devel
export JAVA_HOME=/usr/lib/jvm/java-21-openjdk
export PATH=$JAVA_HOME/bin:$PATH
```

**Mit SDKMAN:**
```bash
curl -s "https://get.sdkman.io" | bash
source ~/.sdkman/bin/sdkman-init.sh
sdk install java 21.0.2-tem
```

Überprüfen:
```bash
java -version
```

## Maven installieren

### Windows

**Option 1: Manuelle Installation**
1. Von [Apache Maven](https://maven.apache.org/download.cgi) herunterladen
2. Nach `C:\apache-maven` entpacken
3. Umgebungsvariablen festlegen:
   ```
   MAVEN_HOME=C:\apache-maven
   PATH=%MAVEN_HOME%\bin;%PATH%
   ```

**Option 2: Mit Chocolatey**

```powershell
choco install maven
```

## IDE-Einrichtung

### IntelliJ IDEA (Empfohlen)

1. Von [JetBrains](https://www.jetbrains.com/idea/download/) herunterladen
2. Installieren und öffnen
3. JDK konfigurieren: `File` → `Project Structure` → `Project SDK`
4. Spring Boot Assistant Plugin installieren (optional)

### VS Code

1. [VS Code](https://code.visualstudio.com/) installieren
2. Erweiterungen installieren:
   - Extension Pack for Java
   - Spring Boot Extension Pack
3. Java konfigurieren: `Strg+Umschalt+P` → `Java: Configure Java Runtime`

### Eclipse

1. [Spring Tools 4 (STS)](https://spring.io/tools) herunterladen
2. Entpacken und ausführen
3. File → New → Spring Starter Project

## Schnellstart

### Spring Initializr verwenden

1. [start.spring.io](https://start.spring.io/) besuchen
2. Optionen auswählen:
   - Project: Maven / Gradle
   - Language: Java / Kotlin / Groovy
   - Spring Boot Version: 3.2.x (latest)
   - Group: com.example
   - Artifact: myapp
   - Packaging: Jar / War
   - Java Version: 17 / 21
3. Abhängigkeiten hinzufügen:
   - Spring Web
   - Spring Data JPA
   - H2 Database (dev)
   - Spring Boot DevTools
4. Auf "Generate" klicken zum Herunterladen

### Befehlszeile verwenden

```bash
# Maven-Projekt erstellen
curl https://start.spring.io/starter.zip \
  -d type=maven-project \
  -d language=java \
  -d bootVersion=3.2.5 \
  -d baseDir=myapp \
  -d dependencies=web,jpa,h2,devtools \
  -o myapp.zip

unzip myapp.zip
cd myapp
./mvnw spring-boot:run
```

## Installation überprüfen

```bash
# Java prüfen
java -version

# Maven prüfen
mvn -version

# Gradle prüfen (falls installiert)
gradle -version

# Spring Boot CLI prüfen
spring --version

# Demo-Projekt ausführen
curl https://start.spring.io/starter.zip -o demo.zip
unzip demo.zip -d demo
cd demo
./mvnw spring-boot:run
```

## Häufige Probleme

### Java nicht gefunden
- Stellen Sie sicher, dass JAVA_HOME korrekt gesetzt ist
- Starten Sie das Terminal nach dem Festlegen der Umgebungsvariablen neu
- Überprüfen Sie mit `echo $JAVA_HOME` (Linux/macOS) oder `echo %JAVA_HOME%` (Windows)

### Port 8080 bereits belegt
```bash
# Windows
netstat -ano | findstr :8080
taskkill /PID <pid> /F

# Linux/macOS
lsof -i :8080
kill -9 <pid>
```

### Maven-Build fehlgeschlagen
- Maven-Cache leeren: `mvn clean`
- Abhängigkeiten aktualisieren: `mvn dependency:resolve`
- Verwenden Sie `./mvnw` (Maven Wrapper) anstelle des System-Maven
