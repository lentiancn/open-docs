# Руководство по установке Apache APISIX

## Требования

- **ОС**: Linux (Ubuntu, CentOS, Debian), macOS
- **Память**: минимум 2GB (4GB+ для продакшена)
- **CPU**: минимум 2 ядра

## Способы установки

### Способ 1: Docker Quickstart (Рекомендуется)

```bash
curl -sL https://run.api7.ai/apisix/quickstart | sh
```

После установки:
```
✔ APISIX is ready!
```

Порты по умолчанию:
- 9080: HTTP
- 9443: HTTPS
- 9180: Admin API

### Способ 2: Docker Compose

```yaml
version: '3'
services:
  apisix:
    image: apache/apisix:3.3.0
    ports:
      - "9080:9080"
      - "9443:9443"
      - "9180:9180"
  etcd:
    image: apache/apisix/etcdserve:v3.5.0
```

```bash
docker-compose up -d
```

### Способ 3: Helm (Kubernetes)

```bash
helm repo add apisix https://apache.github.io/apisix-helm-chart
helm install apisix apisix/apisix --namespace apisix
```

## Проверка установки

```bash
curl "http://127.0.0.1:9080" --head
# Ответ: Server: APISIX/3.3.0
```

## Dashboard

```
http://127.0.0.1:9180/ui
```

## Удаление

### Docker
```bash
docker stop apisix-quickstart etcd
docker rm apisix-quickstart etcd
```

### Kubernetes
```bash
helm uninstall apisix --namespace apisix
```
