# FantasyTalking - FAQ

Часто задаваемые вопросы и ответы.

## Проблемы установки

### Q: Ошибка установки зависимостей?

**Решение:**
1. Обновите pip: `pip install --upgrade pip`
2. Используйте китайское зеркало:
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```
3. Установите проблемные пакеты по отдельности

### Q: CUDA недоступен?

**Решение:**
1. Подтвердите установку драйвера NVIDIA: `nvidia-smi`
2. Установите совместимую версию PyTorch:
```bash
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
```
3. Проверьте доступность CUDA:
```python
import torch
print(torch.cuda.is_available())
```

### Q: Недостаточно памяти?

**Решение:**
1. Используйте меньшие входные изображения
2. Уменьшите размер пакета
3. Используйте режим CPU (не рекомендуется, медленно)

## Проблемы использования

### Q: Ошибка генерации, лицо не найдено?

**Решение:**
1. Используйте изображение с чётким лицом
2. Убедитесь, что лицо обращено вперёд, без occlusии
3. Проверьте формат изображения

### Q: Генерация видео очень медленная?

**Решение:**
1. Используйте ускорение GPU
2. Уменьшите разрешение вывода
3. Уменьшите количество кадров вывода

### Q: Губы не синхронизированы?

**Решение:**
1. Убедитесь, что аудио чёткое и без шума
2. Проверьте частоту дискретизации аудио
3. Попробуйте разные параметры синхронизации

### Q: Выходное видео размытое?

**Решение:**
1. Используйте улучшение лица: `--face_enhancer realusf`
2. Увеличьте разрешение вывода
3. Используйте более чёткое исходное изображение

### Q: Лицо деформировано/искажено?

**Решение:**
1. Используйте фронтальное фото без occlusии
2. Убедитесь, что на изображении только одно лицо
3. Настройте параметры обнаружения лица

## Проблемы с моделью

### Q: Ошибка загрузки модели?

**Решение:**
1. Используйте прокси или зеркало
2. Скачайте файлы модели вручную
3. Проверьте сетевое подключение

### Q: Модель не загружается?

**Решение:**
1. Подтвердите полноту файла модели
2. Проверьте правильность пути к модели
3. Скачайте модель повторно

## Проблемы производительности

### Q: Как улучшить скорость генерации?

**Решение:**
1. Используйте GPU
2. Включите смешанную точность: `--precision fp16`
3. Используйте меньшее разрешение

### Q: Слишком высокий расход памяти?

**Решение:**
1. Уменьшите размер пакета
2. Используйте меньшую модель
3. Отключите ненужные функции

## Проблемы с выводом

### Q: Выходное видео не воспроизводится?

**Решение:**
1. Установите декодер: `pip install opencv-python`
2. Используйте поддерживаемый формат: MP4

### Q: Видеофайл слишком большой?

**Решение:**
1. Уменьшите разрешение
2. Уменьшите частоту кадров
3. Используйте более высокую компрессию
