# OpenClaw 🦞

자신의 기기에서 실행하는 도구입니다. 메시징 앱을 AI 서비스에 연결합니다. WhatsApp, Telegram, Discord, iMessage 다 됩니다.

## 누가 쓰지?

데이터를 Third Party에 넘기고 싶지 않은 사람. 개발자에게 좋음.

## 특징

- 데이터는 내 기기에
- 여러 플랫폼: 하나로 처리
- 프로그래밍용: 도구, 히스토리, 메모리
- 오픈소스: MIT 라이선스

## 필요한 것

- Node.js 22+
- AI 제공자 API 키

## 시작

```bash
npm install -g openclaw@latest
openclaw onboard --install-daemon
openclaw gateway --port 18789
```

http://127.0.0.1:18789/ 열기
