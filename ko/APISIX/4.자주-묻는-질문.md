# Apache APISIX 자주 묻는 질문

## 기초

### 1. APISIX란?

APISIX는 동적 트래픽 관리를 제공하는 클라우드 네이티브 API 게이트웨이입니다.

### 2. APISIX vs Nginx

APISIX는 Nginx 기반으로 재시작 없이 핫 업데이트를 지원합니다.

### 3. 종속성

- etcd: 구성 저장
- Lua: 플러그인 로직
- Nginx: HTTP 처리

### 4. 성능

18,000 QPS/코어, 지연시간 <0.2ms

## 설치

### 5. Quickstart 오류
