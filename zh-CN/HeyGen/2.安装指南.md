# HeyGen 安装指南

## 账户注册与登录

### 访问官方网站

访问 HeyGen 官方网站 https://heygen.com ，点击页面右上角的"Get Started For Free"或"注册"按钮开始创建账户。您可以使用电子邮箱直接注册，也可以选择使用 Google 账户快速登录。注册过程简单快捷，只需填写基本信息并验证邮箱即可完成。

首次登录时，系统会引导您完成基础设置，包括选择使用语言、熟悉界面布局和了解核心功能。建议新用户先完成引导教程，了解平台的基本操作流程，为后续的视频创作打下基础。

### 账户类型选择

HeyGen 提供四种主要的账户类型：免费版（Free）、创作者版（Creator）、专业版（Pro）和企业版（Business）。免费版每月可生成 3 个视频，最长 3 分钟，适合初步体验平台功能。创作者版每月 $29，无视频时长限制，支持更多高级功能。专业版每月 $99，提供 10 倍的高级功能使用量和 4K 视频导出。企业版每月 $149，每增加一个席位额外 $20，适合团队协作使用。

## 界面导航

### 主界面概览

登录后，您会看到 HeyGen 的主界面。左侧是导航菜单，包括创建视频、数字人管理、场景选择、项目管理和设置等主要功能模块。中央区域是内容展示区，显示您的视频项目列表和模板推荐。右侧面板提供快捷设置选项，包括语音选择、语言设置和预览功能。

顶部导航栏包含账户信息、使用配额显示和帮助入口。底部状态栏显示当前任务进度和系统通知。熟悉界面布局后，您可以快速导航到所需功能，提高工作效率。

### 功能模块介绍

创建视频是核心功能，点击后进入视频创作工作区。数字人管理允许您浏览、创建和编辑 AI 数字人形象。场景选择提供丰富的背景模板和自定义选项。项目管理帮助您组织和管理视频内容，支持文件夹分类和标签管理。

## 快速创建第一个视频

### 选择数字人形象

点击"创建视频"后，首先需要选择数字人形象。您可以在预设形象库中选择，涵盖不同性别、年龄、肤色和风格的 500 多个公共数字人。如果您需要个性化形象，可以点击"自定义数字人"上传自己的照片，系统会基于照片生成专属的数字人形象。

选择数字人时，可以预览其不同角度和表情的示例视频，确保选择最适合您内容需求的形象。选定后，该数字人将成为当前视频的主角。

### 输入视频内容

在脚本编辑区域输入您希望数字人播报的内容。您可以直接输入文字，或上传预先准备好的音频文件。文字输入时，系统会实时显示预计的视频时长。专业术语和人名可以通过平台的发音纠正功能手动调整，确保 AI 配音准确无误。

HeyGen 支持多种输入方式：纯文字脚本、音频文件（MP3、WAV）、PPT 或 PDF 文档导入。对于长篇内容，系统会自动将其分割成适合视频呈现的段落。

### 选择语音和语言

在右侧面板选择语音选项。HeyGen 提供超过 1000 种 AI 语音，涵盖 175 种以上语言和方言。您可以预览每个语音的示例，选择最符合内容调性的音色。语音参数包括语速、音调和停顿间隔，都可以根据需要进行微调。

多语言支持是 HeyGen 的核心优势。如果您需要创建多语言版本，只需在设置中选择目标语言，系统会自动匹配相应的语音并进行翻译。翻译功能保留原始说话者的声音特征和表达风格。

### 添加背景和场景

选择合适的背景可以显著提升视频的专业感。HeyGen 提供超过 75 种预设场景，涵盖办公室、演播室、城市风光、自然环境等多种风格。您也可以上传自定义背景图片作为视频背景。

动态元素如文字标题、图标动画、数据图表等可以增强信息传达效果。所有元素都支持位置、大小、颜色和动画效果的个性化调整。

### 生成和导出视频

完成所有设置后，点击"生成"按钮提交视频创建任务。视频生成时间取决于视频长度、复杂度和服务器负载，一般在几分钟内完成。您可以在任务列表中查看进度。

视频生成完成后，您可以预览最终效果。如需调整，可以返回编辑界面修改文案或设置。满意后，选择合适的格式和分辨率导出。HeyGen 支持 720p、1080p 和 4K 多种分辨率，以及 MP4、MOV 等常用视频格式。

## 高级功能使用

### 自定义数字人形象

点击导航栏的"数字人"选项进入形象管理页面。点击"创建自定义形象"上传人物照片，建议使用正面高清照片以获得最佳效果。系统会自动分析照片特征，生成对应的数字人形象。

自定义形象支持丰富的参数调整，包括发型、面部特征、服装风格、配饰等。您可以保存多个形象设置，方便在不同视频中快速切换。企业用户可以定制专属的品牌形象，用于企业宣传和客户服务场景。

### 团队协作功能

企业版用户可以创建团队账户，邀请团队成员加入。管理员可以设置不同成员的使用权限，分配视频配额和管理功能。团队成员创建的资产可以设置为共享或私有，促进知识积累和复用。

企业账户还可以定制品牌元素，包括 Logo、专属颜色和片头片尾。所有使用企业品牌创建的视频都会自动添加品牌标识，确保品牌一致性。

### API 集成

对于需要将 HeyGen 集成到现有工作流程的企业用户，平台提供完整的 API 接口。API 支持按需付费模式，包括视频代理、视频翻译、文本转语音等功能。

通过 API，您可以自动化视频创建流程，实现大规模内容生产。详细的 API 文档和集成指南可以在开发者文档中找到。

## 账户管理

### 订阅计划管理

在账户设置页面可以查看当前订阅状态和可用配额。您可以随时升级或降级订阅计划。升级立即生效，降级将在当前计费周期结束后生效。

取消订阅后，您仍然可以在当前计费周期结束前使用订阅功能。计费周期结束后，账户将自动转为免费版。

### 安全性设置

建议开启双因素认证（ MFA ）增强账户安全性。企业用户可以配置 SAML/SSO 实现单点登录，满足企业安全策略要求。账户设置中还可以管理活动会话，查看登录历史记录。
