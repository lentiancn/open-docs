# Flutter

FlutterはGoogleが開発したオープンソースUIツールキットで、クロスプラットフォームアプリケーションの構築に使用されます。

## 概要

Flutterを使用すると、単一のコードベースでiOS、Android、Web、デスクトップ向けの美しいアプリケーションを構築できます。

## 主な機能

- **クロスプラットフォーム**：1つのコード、複数のプラットフォーム
- **ホットリロード**：快速な開発サイクル
- **豊富なコンポーネント**：Material DesignとCupertinoコンポーネント
- **Dart言語**：学びやすく、パフォーマンスに優れる
- **宣言型UI**：美しいUIを構築

## バージョン情報

| バージョン | 状態 |
|------------|------|
| Flutter 3.x | ✅ 現行 |

## クイックスタート

### インストール

```bash
# macOS
brew install flutter

# Linux
git clone https://github.com/flutter/flutter.git -b stable
```

### プロジェクトの作成

```bash
flutter create my_app
cd my_app
flutter run
```
