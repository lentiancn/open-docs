## What is jQuery?

jQuery is a fast, small, and feature-rich JavaScript library. It simplifies HTML document traversal, event handling, animation, and Ajax interactions.

## CDN

### CDN Providers

| Provider | URL |
|----------|-----|
| cdnjs | https://cdnjs.cloudflare.com/ajax/libs/jquery/3.7.1/jquery.min.js |
| jsDelivr | https://cdn.jsdelivr.net/npm/jquery@3.7.1/dist/jquery.min.js |
| Google | https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js |

### Basic CDN Usage

```html
<!-- jQuery Core (minified) -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script>

<!-- jQuery Core (uncompressed, for development) -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.7.1/jquery.js"></script>
```

### Complete Example

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>jQuery App</title>
</head>
<body>
  <div id="content">
    <p>Hello, jQuery!</p>
  </div>

  <script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script>
  <script>
    $(document).ready(function() {
      $('#content p').text('Hello, jQuery!')
    })
  </script>
</body>
</html>
```

## NPM

### Install jQuery

```bash
# Install latest version
npm install jquery

# Install specific version
npm install jquery@3.6.0

# Install as dev dependency
npm install --save-dev jquery
```

### Using with Module Bundlers

```javascript
// ES Modules
import $ from 'jquery'

// CommonJS
const $ = require('jquery')
```

---

## Bower

### Install jQuery

```bash
# Install jQuery
bower install jquery

# Install specific version
bower install jquery#3.7.1
```

## Basic Setup

### HTML Setup

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>My jQuery App</title>
</head>
<body>
  <h1 id="title">Welcome</h1>
  
  <script src="jquery.min.js"></script>
  <script>
    $(document).ready(function() {
      console.log('jQuery is ready!')
    })
  </script>
</body>
</html>
```

### Version Compatibility

| jQuery Version | IE Support | Notes |
|---------------|------------|-------|
| 3.x | IE 9+ | Current version |
| 2.x | IE 9+ | No IE 6-8 support |
| 1.x | IE 6-8 | Legacy support |
