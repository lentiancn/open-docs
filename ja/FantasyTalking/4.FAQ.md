# FantasyTalking - FAQ

よくある質問と回答。

## インストール問題

### Q：依存関係のインストールに失敗した？

**解決策：**
1. pipをアップグレード：`pip install --upgrade pip`
2. 中国ミーラーを使用：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```
3. 失敗したパッケージを個別にインストール

### Q：CUDAが利用できない？

**解決策：**
1. NVIDIAドライバーがインストールされているか確認：`nvidia-smi`
2. 適切なバージョンのPyTorchをインストール：
```bash
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
```
3. CUDAの可用性を確認：
```python
import torch
print(torch.cuda.is_available())
```

### Q：メモリ不足？

**解決策：**
1. より小さい入力画像を使用
2. バッチサイズを小さくする
3. CPUモードを使用（推奨，速度が遅い）

## 使用問題

### Q：生成に失敗、顔が見つからない？

**解決策：**
1. 明確な顔を含む画像を使用
2. 顔が正面、向いているか確認
3. 画像形式を確認

### Q：動画生成が遅い？

**解決策：**
1. GPUアクセラレーションを使用
2. 出力解像度を下げる
3. 出力フレーム数を減らす

### Q：唇的不同期？

**解決策：**
1. 音声が明確でノイズがないことを確認
2. 音声のサンプルレートを確認
3. 異なる同期パラメータを試す

### Q：出力動画がぼやける？

**解決策：**
1. 顔強調を使用：`--face_enhancer realusf`
2. 出力解像度を高める
3. より明確なソース画像を使用

### Q：顔が変形/歪み？

**解決策：**
1. 遮蔽のない正面写真を使用
2. 画像に顔が1つだけ含まれていることを確認
3. 顔検出パラメータを調整

## モデル問題

### Q：モデルダウンロードに失敗？

**解決策：**
1. プロキシまたはミラーを使用
2. モデルファイルを手動でダウンロード
3. ネットワーク接続を確認

### Q：モデルの読み込みに失敗？

**解決策：**
1. モデルファイルが完全か確認
2. モデルパスが正しいか確認
3. モデルを再ダウンロード

## パフォーマンス問題

### Q：生成速度を改善するには？

**解決策：**
1. GPUを使用
2. 混合精度を有効化：`--precision fp16`
3. より小さい解像度を使用

### Q：メモリ使用量が大きすぎる？

**解決策：**
1. バッチサイズを小さくする
2. より小さいモデルを使用
3. 不要な機能を無効化

## 出力問題

### Q：出力動画を再生できない？

**解決策：**
1. デコーダをインストール：`pip install opencv-python`
2. 対応形式を使用：MP4

### Q：動画ファイルが大きすぎる？

**解決策：**
1. 解像度を下げる
2. フレームレートを下げる
3. より高い圧縮率を使用
