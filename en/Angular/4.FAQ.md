# Angular FAQ

Common questions and answers.

## Installation

### Q: Angular CLI installation failed?

**Solutions**:
1. Clear npm cache: `npm cache clean --force`
2. Use admin/root privileges
3. Check Node.js version

### Q: ng command not found?

**Solutions**:
1. Confirm Angular CLI is installed globally
2. Check PATH environment variable
3. Try reinstalling

### Q: Node.js version incompatible?

**Solutions**:
1. Use nvm to manage Node.js versions
2. Upgrade Node.js to recommended version

## Usage

### Q: Component styles not working?

**Solutions**:
1. Check styleUrls path is correct
2. Confirm CSS selectors are correct
3. Check ViewEncapsulation settings

### Q: Route navigation failing?

**Solutions**:
1. Check route configuration is correct
2. Confirm RouterModule is imported
3. Check routerLink syntax

### Q: HTTP requests failing?

**Solutions**:
1. Confirm HttpClientModule is imported
2. Check CORS configuration
3. Verify API URL is correct

### Q: Form validation not working?

**Solutions**:
1. Confirm ReactiveFormsModule is imported
2. Check Validators configuration
3. Use formControlName correctly

## Performance

### Q: Application loading slow?

**Solutions**:
1. Enable lazy loading
2. Use OnPush change detection
3. Compress images and resources
4. Build in production mode

### Q: Memory leaks?

**Solutions**:
1. Unsubscribe from observables
2. Clear intervals
3. Destroy event listeners

## Version

### Q: How to upgrade Angular?

```bash
ng update @angular/core @angular/cli
```

### Q: Migrate from AngularJS to Angular?

Use angular-upgrade tool for gradual migration.
