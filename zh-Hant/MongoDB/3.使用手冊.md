# MongoDB 使用手冊

## 基本概念

### 資料庫

資料庫是 MongoDB 中最高層級的資料容器。每個資料庫都有自己的集合和權限設定。在 MongoDB 中可以建立多個資料庫，每個資料庫都是獨立的。

### 集合

集合就是一群 MongoDB 文件，相當於關聯式資料庫中的表。集合中的文件可以擁有不同的欄位，這使得集合非常靈活。與關聯式資料庫不同，集合不強制要求嚴格的預先定義模式。

### 文件

文件是 MongoDB 中的基本資料單元，由欄位和值對組成。文件類似於 JSON 物件，使用 BSON（二進位 JSON）格式儲存。

BSON 是 JSON 文件的二進位表示形式，有額外的資料類型支援。文件大小上限為 16 MB。

## 插入文件

MongoDB 提供以下方法將文件插入到集合中：

### 插入單個文件

```bash
db.users.insertOne({
  name: "張三",
  age: 30,
  email: "zhangsan@example.com",
  address: {
    city: "臺北",
    street: "忠孝東路"
  },
  hobbies: ["籃球", "游泳"]
})
```

### 插入多個文件

```bash
db.users.insertMany([
  { name: "李四", age: 25, city: "高雄" },
  { name: "王五", age: 35, city: "臺中" },
  { name: "趙六", age: 28, city: "新竹" }
])
```

### 插入文件（已過時但仍然可用）

```bash
db.users.insert({
  name: "錢七",
  age: 32
})
```

### 查看插入結果

```bash
# 查看插入的文件 ID
db.users.insertOne({ name: "測試" })
# 回傳：{ acknowledged: true, insertedId: ObjectId("...") }
```

## 查詢文件

### 查詢所有文件

```bash
db.users.find()
```

相當於 SQL 中的 `SELECT * FROM users`

### 查詢單條記錄

```bash
db.users.findOne()
```

### 條件查詢

```bash
# 等於查詢
db.users.find({ age: 30 })

# 不等於查詢
db.users.find({ age: { $ne: 30 } })

# 大於查詢
db.users.find({ age: { $gt: 25 } })

# 大於等於查詢
db.users.find({ age: { $gte: 25 } })

# 小於查詢
db.users.find({ age: { $lt: 30 } })

# 小於等於查詢
db.users.find({ age: { $lte: 30 } })

# 範圍查詢
db.users.find({ age: { $gte: 20, $lte: 30 } })
```

### 邏輯運算子

```bash
# AND（預設）
db.users.find({ age: { $gte: 25 }, name: "李四" })

# OR
db.users.find({ $or: [{ age: { $gte: 30 } }, { name: "王五" }] })

# NOR（既不也不）
db.users.find({ $nor: [{ age: { $lt: 20 } }, { city: "臺北" }] })

# NOT
db.users.find({ age: { $not: { $gte: 25 } } })
```

### 陣列查詢

```bash
# 陣列包含元素
db.users.find({ hobbies: "籃球" })

# 陣列包含所有元素
db.users.find({ hobbies: { $all: ["籃球", "游泳"] } })

# 陣列長度
db.users.find({ hobbies: { $size: 2 } })

# 陣列特定位置元素
db.users.find({ "hobbies.0": "籃球" })

# 陣列元素滿足條件
db.users.find({ hobbies: { $elemMatch: { $eq: "籃球" } } })
```

### 巢狀文件查詢

```bash
# 巢狀文件欄位查詢
db.users.find({ "address.city": "臺北" })

# 巢狀文件完全比對
db.users.find({ address: { city: "臺北", street: "忠孝東路" } })
```

### 投影查詢（只回傳指定欄位）

```bash
# 只回傳 name 和 email 欄位
db.users.find({ age: 30 }, { name: 1, email: 1 })

# 排除某個欄位
db.users.find({}, { age: 0 })

# 回傳所有欄位除了 _id
db.users.find({}, { _id: 0 })
```

### 排序

```bash
# 按年齡升序
db.users.find().sort({ age: 1 })

# 按年齡降序
db.users.find().sort({ age: -1 })

# 多欄位排序
db.users.find().sort({ age: -1, name: 1 })
```

### 限制和跳過

```bash
# 回傳前 5 條
db.users.find().limit(5)

# 跳過前 10 條，回傳接下來的 5 條（分頁）
db.users.find().skip(10).limit(5)

# 統計數量
db.users.countDocuments({ age: { $gte: 25 } })

# 估計集合中文檔數量（快但可能不精確）
db.users.estimatedDocumentCount()
```

### 游標操作

```bash
# 使用游標
var cursor = db.users.find()
while (cursor.hasNext()) {
  printjson(cursor.next())
}

# forEach 遍歷
db.users.find().forEach(function(doc) {
  print("Name: " + doc.name)
})
```

## 更新文件

### 更新單個文件

```bash
db.users.updateOne(
  { name: "張三" },
  { $set: { age: 31 } }
)
```

### 更新多個文件

```bash
db.users.updateMany(
  { age: { $lt: 30 } },
  { $set: { status: "young" } }
)
```

### 取代文件

```bash
db.users.replaceOne(
  { name: "張三" },
  { name: "張三", age: 32, email: "new@example.com", city: "臺北" }
)
```

### 更新運算子

```bash
# $set - 設定欄位值
db.users.updateOne({ name: "李四" }, { $set: { age: 26 } })

# $unset - 移除欄位
db.users.updateOne({ name: "李四" }, { $unset: { email: "" } })

# $inc - 遞增/遞減
db.users.updateOne({ name: "李四" }, { $inc: { age: 1 } })
db.users.updateOne({ name: "李四" }, { $inc: { age: -1 } })

# $rename - 重新命名欄位
db.users.updateOne({ name: "李四" }, { $rename: { "name": "fullname" } })

# $mul - 乘
db.users.updateOne({ name: "李四" }, { $mul: { age: 2 } })

# $min - 取最小值
db.users.updateOne({ name: "李四" }, { $min: { age: 20 } })

# $max - 取最大值
db.users.updateOne({ name: "李四" }, { $max: { age: 40 } })

# $push - 新增元素到陣列
db.users.updateOne({ name: "李四" }, { $push: { hobbies: "跑步" } })

# $pull - 移除陣列中比對的元素
db.users.updateOne({ name: "李四" }, { $pull: { hobbies: "游泳" } })

# $addToSet - 新增元素到陣列（不重複）
db.users.updateOne({ name: "李四" }, { $addToSet: { hobbies: "籃球" } })

# $pop - 移除陣列第一個或最後一個元素
db.users.updateOne({ name: "李四" }, { $pop: { hobbies: 1 } })
```

### upsert 操作

```bash
# 如果不存在則插入，存在則更新
db.users.updateOne(
  { name: "新使用者" },
  { $set: { age: 25, city: "臺北" } },
  { upsert: true }
)
```

## 聚合管道

聚合管道是 MongoDB 最強大的功能之一，用於處理資料流，執行複雜的資料轉換和分析。

### 基本語法

```bash
db.collection.aggregate([
  { $stage1: { ... } },
  { $stage2: { ... } },
  ...
])
```

### 常用聚合階段

#### $match - 過濾文件

```bash
db.users.aggregate([
  { $match: { age: { $gte: 25 } } }
])
```

#### $project - 投影

```bash
db.users.aggregate([
  { $project: { _id: 0, name: 1, age: 1 } }
])
```

#### $group - 分組

```bash
# 統計每個年齡的使用者數量
db.users.aggregate([
  { $group: { _id: "$age", count: { $sum: 1 } } }
])

# 統計平均年齡
db.users.aggregate([
  { $group: { _id: null, avgAge: { $avg: "$age" } } }
])

# 分組並收集名字
db.users.aggregate([
  { $group: { _id: "$city", users: { $push: "$name" } } }
])
```

#### $sort - 排序

```bash
db.users.aggregate([
  { $sort: { age: -1, name: 1 } }
])
```

#### $limit - 限制數量

```bash
db.users.aggregate([
  { $limit: 5 }
])
```

#### $skip - 跳過文件

```bash
db.users.aggregate([
  { $skip: 10 },
  { $limit: 5 }
])
```

#### $unwind - 展開陣列

```bash
db.users.aggregate([
  { $unwind: "$hobbies" }
])
```

#### $lookup - 聯表查詢

```bash
# 假設有兩個集合：orders 和 products
db.orders.aggregate([
  {
    $lookup: {
      from: "products",
      localField: "productId",
      foreignField: "_id",
      as: "productDetails"
    }
  }
])
```

### 聚合運算子

#### 算術運算子

```bash
db.sales.aggregate([
  {
    $project: {
      total: { $add: ["$price", "$tax"] },
      discount: { $multiply: ["$price", 0.1] }
    }
  }
])
```

#### 字串運算子

```bash
db.users.aggregate([
  {
    $project: {
      fullName: { $concat: ["$firstName", " ", "$lastName"] },
      upperName: { $toUpper: "$name" }
    }
  }
])
```

#### 日期運算子

```bash
db.users.aggregate([
  {
    $project: {
      year: { $year: "$birthdate" },
      month: { $month: "$birthdate" },
      day: { $dayOfMonth: "$birthdate" }
    }
  }
])
```

### 完整範例

```bash
# 找到從 PDX 機場出發的直飛航班最多的前三大家航空公司
db.routes.aggregate([
  {
    $match: {
      "src_airport": "PDX",
      "stops": 0
    }
  },
  {
    $group: {
      _id: "$airline.name",
      count: { $sum: 1 }
    }
  },
  {
    $sort: { count: -1 }
  },
  {
    $limit: 3
  }
])
```

## 索引

索引支援在 MongoDB 中高效執行查詢。如果沒有索引，MongoDB 就必須掃描集合中的每個文件以回傳查詢結果。

### 建立索引

```bash
# 單欄位索引
db.users.createIndex({ name: 1 })

# 降序索引
db.users.createIndex({ age: -1 })

# 複合索引
db.users.createIndex({ age: 1, name: 1 })

# 唯一索引
db.users.createIndex({ email: 1 }, { unique: true })

# 文字索引
db.users.createIndex({ description: "text" })

# 多鍵索引（陣列欄位）
db.users.createIndex({ hobbies: 1 })

# 地理空間索引
db.locations.createIndex({ location: "2dsphere" })

# 雜湊索引
db.users.createIndex({ _id: "hashed" })
```

### 查看索引

```bash
# 查看集合的所有索引
db.users.getIndexes()

# 查看集合的索引大小
db.users.totalIndexSize()
```

### 刪除索引

```bash
# 刪除指定索引
db.users.dropIndex("name_1")

# 刪除所有索引（保留 _id 索引）
db.users.dropIndexes()

# 刪除文字索引
db.users.dropIndex("description_text")
```

### 索引屬性

```bash
# 稀疏索引（只索引非空值）
db.users.createIndex({ email: 1 }, { sparse: true })

# 部分索引
db.users.createIndex(
  { email: 1 },
  { partialFilterExpression: { email: { $exists: true } } }
)

# TTL 索引（自動過期）
db.sessions.createIndex(
  { createdAt: 1 },
  { $set: { expireAfterSeconds: 3600 } }
)
```

### 索引策略建議

- 為經常用於查詢條件的欄位建立索引
- 複合索引的欄位順序很重要：精確比對的欄位放在前面，範圍查詢的欄位放在後面
- 避免在寫入頻率高的集合上建立過多索引
- 使用 explain() 分析查詢效能

### 查詢分析

```bash
# 分析查詢
db.users.find({ age: 25 }).explain()

# 分析執行計畫
db.users.find({ age: { $gte: 25 } }).sort({ name: 1 }).explain("executionStats")
```

## 資料匯出匯入

### 使用 mongodump 和 mongorestore

#### 匯出資料

```bash
# 匯出整個資料庫
mongodump --db=mydatabase --out=/backup

# 匯出指定集合
mongodump --db=mydatabase --collection=users --out=/backup

# 匯出並壓縮
mongodump --db=mydatabase --gzip --out=/backup
```

#### 匯入資料

```bash
# 復原整個資料庫
mongorestore --db=mydatabase /backup/mydatabase

# 復原指定集合
mongorestore --db=mydatabase --collection=users /backup/mydatabase/users.bson

# 復原並壓縮
mongorestore --db=mydatabase --gzip /backup/mongodb
```

### 使用 mongoexport 和 mongoimport

#### 匯出為 JSON

```bash
# 匯出為 JSON
mongoexport --db=mydatabase --collection=users --out=users.json

# 匯出為 JSON 陣列
mongoexport --db=mydatabase --collection=users --out=users.json --jsonArray

# 匯出 CSV
mongoexport --db=mydatabase --collection=users --out=users.csv --fields=name,age,email
```

#### 從 JSON 匯入

```bash
# 從 JSON 匯入
mongoimport --db=mydatabase --collection=users --file=users.json

# 從 JSON 陣列匯入
mongoimport --db=mydatabase --collection=users --file=users.json --jsonArray
```

## 交易操作

MongoDB 4.0+ 支援多文件交易。

### 基本交易

```bash
session = db.getMongo().startSession()
session.startTransaction({ readConcern: { level: "snapshot" }, writeConcern: { w: "majority" } })

try {
  session.getDatabase("mydb").accounts.updateOne(
    { accountId: 1 },
    { $inc: { balance: -100 } }
  )
  session.getDatabase("mydb").accounts.updateOne(
    { accountId: 2 },
    { $inc: { balance: 100 } }
  )
  session.commitTransaction()
} catch (error) {
  session.abortTransaction()
  throw error
} finally {
  session.endSession()
}
```
