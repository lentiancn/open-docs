# Oracle Guide d'Installation

## Prérequis

- **Système d'exploitation**: Oracle Linux, RHEL, Windows Server
- **Mémoire**: Minimum 2GB RAM (recommendé 8GB+)
- **Espace Disque**: Minimum 8GB pour logiciel

---

## Installation sur Linux

### 1. Créer utilisateur oracle

```bash
groupadd -g 54321 oinstall
groupadd -g 54322 dba
useradd -u 54321 -g oinstall -G dba oracle
passwd oracle
```

### 2. Configurer les paramètres du kernel

```bash
kernel.shmmax = 4294967296
kernel.shmall = 2097152
```

### 3. Exécuter l'installeur

```bash
./runInstaller
```

---

## Après l'installation

```sql
sqlplus / as sysdba
SELECT name, open_mode FROM v$database;
```
