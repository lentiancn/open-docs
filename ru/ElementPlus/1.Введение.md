# Element Plus Введение

Element Plus — это библиотека компонентов UI на Vue 3.

## Характеристики

- Компоненты Vue 3
- 70+ компонентов
- TypeScript
- Импорт по требованию
- Интернационализация
- Настраиваемые темы

## Сценарии

- Системы управления предприятием
- Платформы электронной коммерции

Дополнительные сведения см. на [официальном сайте](https://element-plus.org/).
