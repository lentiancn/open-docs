# Docker Manual de Usuario

Conceptos básicos, uso general y guía de desarrollo.

## Operaciones con Imágenes

### Extraer Imágenes

```
docker pull nginx
docker pull ubuntu:22.04
```

### Construir Imágenes

```
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
EXPOSE 3000
CMD ["npm", "start"]
```

```
docker build -t my-app:tag .
```

### Gestionar Imágenes

```
docker images
docker rmi nginx
```

## Operaciones con Contenedores

### Crear y Iniciar Contenedores

```
# Contenedor interactivo
docker run -it ubuntu /bin/bash

# Contenedor en segundo plano
docker run -d nginx

# Mapeo de puertos
docker run -d -p 8080:80 nginx

# Variables de entorno
docker run -e MYSQL_ROOT_PASSWORD=123456 mysql
```

### Gestionar Contenedores

```
docker ps              # Contenedores en ejecución
docker ps -a          # Todos los contenedores
docker start container_id
docker stop container_id
docker rm container_id
```

### Entrar en un Contenedor

```
docker exec -it container_id /bin/bash
```

### Ver Logs

```
docker logs container_id
docker logs -f container_id
```

## Escribir Dockerfiles

### Instrucciones Básicas

```
FROM node:18-alpine
WORKDIR /app
COPY . .
RUN npm install
EXPOSE 3000
CMD ["npm", "start"]
```

### Construcciones Multi-etapa

```
# Etapa de construcción
FROM node:18 AS builder
WORKDIR /app
COPY . .
RUN npm run build

# Etapa de producción
FROM nginx:alpine
COPY --from=builder /app/dist /usr/share/nginx/html
```

## Docker Compose

### docker-compose.yml

```
version: '3.8'

services:
  web:
    build: .
    ports:
      - "3000:3000"
  db:
    image: postgres:15
    environment:
      POSTGRES_PASSWORD: secret
```

### Comandos Comunes

```
docker-compose up -d
docker-compose down
docker-compose logs -f
```

## Gestión de Redes

### Mapeo de Puertos

```
docker run -p 8080:80 nginx
```

### Redes Personalizadas

```
docker network create my-network
docker run --network my-network nginx
```

## Gestión de Datos

### Volúmenes

```
docker volume create my-data
docker run -v my-data:/data nginx
```

### Montaje de Enlace

```
docker run -v /host/path:/container/path nginx
```

## Límites de Recursos

```
# Límite de memoria
docker run -m 512m nginx

# Límite de CPU
docker run --cpus=0.5 nginx
```

## Monitoreo y Depuración

```
docker stats
docker inspect container_id
docker logs container_id
```

## Limpieza

```
docker system prune -a
docker volume prune
```
