# IBM Informix

IBM Informix（インフォーマックス）は、HCLテクノロジーズが開発・提供するリレーショナルデータベース管理システム（RDBMS）です。

## ドキュメント概要

このディレクトリには IBM Informix の多言語技術ドキュメントが含まれています：

- **概要**：製品の概要、機能、エディション情報
- **インストールガイド**：Linux および Windows プラットフォームのインストール手順
- **ユーザーマニュアル**：基本操作コマンド、SQL コマンド、設定説明
- **よくある質問**：一般的な問題と解決方法

## バージョン情報

- 最新バージョン：Informix 14.10
