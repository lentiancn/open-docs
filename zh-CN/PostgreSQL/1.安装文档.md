# PostgreSQL 安装指南

PostgreSQL 是一个功能强大的开源对象关系型数据库系统。本指南涵盖在所有环境中的安装方法。

## 版本信息

| 版本 | 状态 | 说明 |
|------|------|------|
| PostgreSQL 16.x | ✅ 当前 | 最新稳定版本 |
| PostgreSQL 15.x | ✅ 支持 | 稳定版本 |
| PostgreSQL 14.x | ⚠️ 维护 | 安全更新 |
| PostgreSQL 13.x | ⚠️ 维护 | 长期支持 |
| PostgreSQL 12.x | ❌ 旧版 | 已停止维护 |

> **重要提示**：本指南主要针对 **PostgreSQL 16.x**。建议新项目使用最新版本。

## 系统要求

### 硬件要求

| 要求 | 最低配置 | 推荐配置 |
|------|----------|----------|
| CPU | 1 核 | 2+ 核 |
| 内存 | 1 GB | 4+ GB |
| 磁盘 | 10 GB | 50+ GB |

### 操作系统支持

- **Linux**: Ubuntu, Debian, CentOS, RHEL, Fedora, openSUSE
- **Windows**: Windows Server 2016+, Windows 10+
- **macOS**: 10.14+

## 安装方法

### 方法一：Windows 安装

#### 1. 使用安装程序（推荐）

1. 下载 PostgreSQL 安装程序：https://www.postgresql.org/download/windows/
2. 运行 postgresql-16.x-x-windows-x64.exe
3. 按照安装向导完成安装
4. 设置超级用户 postgres 的密码

#### 2. 使用 Chocolatey

```powershell
# 安装 PostgreSQL
choco install postgresql

# 安装特定版本
choco install postgresql --version=16.0
```

#### 3. 使用 ZIP 包

```powershell
# 下载 ZIP 包
# 解压到 C:\Program Files\PostgreSQL\16

# 初始化数据库
"C:\Program Files\PostgreSQL\16\bin\initdb" -D "C:\Program Files\PostgreSQL\16\data"

# 启动服务
"C:\Program Files\PostgreSQL\16\bin\pg_ctl" -D "C:\Program Files\PostgreSQL\16\data" -l logfile start
```

### 方法二：macOS 安装

#### 1. 使用 Homebrew（推荐）

```bash
# 安装 PostgreSQL
brew install postgresql@16

# 启动服务
brew services start postgresql@16

# 停止服务
brew services stop postgresql@16
```

#### 2. 使用 PostgreSQL.app

1. 下载 PostgreSQL.app：https://postgresapp.com/
2. 拖动到 Applications 文件夹
3. 点击启动

#### 3. 使用 MacPorts

```bash
sudo port install postgresql16
sudo port load postgresql16-server
```

### 方法三：Linux 安装

#### Ubuntu/Debian

```bash
# 添加 PostgreSQL 官方仓库
sudo sh -c 'echo "deb http://apt.postgresql.org/pub/repos/apt $(lsb_release -cs)-pgdg main" > /etc/apt/sources.list.d/pgdg.list'

# 导入密钥
wget --quiet -O - https://www.postgresql.org/media/keys/ACCC4CF8.asc | sudo apt-key add -

# 更新包列表
sudo apt-get update

# 安装 PostgreSQL
sudo apt-get install -y postgresql-16

# 启动服务
sudo systemctl start postgresql
sudo systemctl enable postgresql
```

#### CentOS/RHEL

```bash
# 安装 RPM 仓库
sudo yum install -y https://download.postgresql.org/pub/repos/yum/16/redhat/rhel-9-x86_64/pgdg-redhat-repo-latest.noarch.rpm

# 安装 PostgreSQL
sudo yum install -y postgresql16-server

# 初始化数据库
sudo /usr/pgsql-16/bin/postgresql-16-setup initdb

# 启动服务
sudo systemctl start postgresql-16
sudo systemctl enable postgresql-16
```

#### Fedora

```bash
# 安装 PostgreSQL
sudo dnf install -y postgresql-server postgresql-contrib

# 初始化数据库
sudo postgresql-setup --initdb

# 启动服务
sudo systemctl start postgresql
sudo systemctl enable postgresql
```

### 方法四：Docker 安装

```bash
# 拉取镜像
docker pull postgres:16

# 运行容器
docker run -d \
  --name postgres \
  -e POSTGRES_PASSWORD=mysecretpassword \
  -e POSTGRES_USER=postgres \
  -e POSTGRES_DB=mydb \
  -p 5432:5432 \
  -v postgres_data:/var/lib/postgresql/data \
  postgres:16
```

#### Docker Compose

```yaml
version: '3.8'

services:
  postgres:
    image: postgres:16
    container_name: postgres
    environment:
      POSTGRES_USER: postgres
      POSTGRES_PASSWORD: mysecretpassword
      POSTGRES_DB: mydb
    ports:
      - "5432:5432"
    volumes:
      - postgres_data:/var/lib/postgresql/data

volumes:
  postgres_data:
```

运行：

```bash
docker-compose up -d
```

## 安装后配置

### 连接 PostgreSQL

```bash
# 使用 psql
sudo -u postgres psql

# 或
psql -U postgres -h localhost -p 5432
```

### 创建数据库

```sql
CREATE DATABASE mydb;

-- 创建用户
CREATE USER myuser WITH PASSWORD 'mypassword';

-- 授权
GRANT ALL PRIVILEGES ON DATABASE mydb TO myuser;
```

### 配置文件

#### postgresql.conf

```properties
# 监听地址
listen_addresses = '*'

# 端口
port = 5432

# 最大连接数
max_connections = 100

# 共享缓冲区
shared_buffers = 128MB

# 工作内存
work_mem = 4MB
```

#### pg_hba.conf

```properties
# 允许本地连接
host    all             all             127.0.0.1/32            trust
host    all             all             ::1/128                 trust

# 允许远程连接
host    all             all             0.0.0.0/0              md5
host    all             all             ::/0                    md5
```

### 远程连接配置

```sql
-- 允许远程连接
ALTER SYSTEM SET listen_addresses = '*';

-- 重载配置
SELECT pg_reload_conf();
```

## 常见问题

### 连接被拒绝？

检查防火墙：

```bash
# Ubuntu
sudo ufw allow 5432/tcp

# CentOS
sudo firewall-cmd --add-port=5432/tcp --permanent
sudo firewall-cmd --reload
```

### 忘记 postgres 密码？

```bash
# 停止 PostgreSQL
sudo systemctl stop postgresql

# 以跳过权限模式启动
sudo -u postgres /usr/lib/postgresql/16/bin/pg_ctl -D /var/lib/postgresql/16/main -m fast stop

# 修改密码
sudo -u postgres psql -c "ALTER USER postgres PASSWORD 'newpassword';"
```

### 数据库启动失败？

检查日志：

```bash
# 查看日志
tail -f /var/log/postgresql/postgresql-16-main.log
```
