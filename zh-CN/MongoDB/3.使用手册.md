# MongoDB 使用手册

## 连接数据库

使用 mongosh 连接本地 MongoDB：

```bash
mongosh
```

连接远程数据库：

```bash
mongosh "mongodb://远程服务器地址:27017"
```

或者使用 URI 格式：

```bash
mongosh "mongodb://用户名:密码@主机地址:27017/数据库名"
```

## 基本操作

### 查看数据库

```javascript
// 查看所有数据库
show dbs

// 切换/创建数据库
use myDatabase

// 查看当前数据库
db
```

### 查看集合

```javascript
// 查看当前数据库中的所有集合
show collections

// 查看集合中的文档数量
db.集合名.countDocuments()
```

## 插入文档

### 单条插入

```javascript
db.users.insertOne({
  name: "张三",
  age: 28,
  email: "zhangsan@example.com",
  createdAt: new Date()
})
```

### 多条插入

```javascript
db.users.insertMany([
  { name: "李四", age: 25 },
  { name: "王五", age: 30 },
  { name: "赵六", age: 22 }
])
```

### 插入校验

MongoDB 默认不强制执行 schema，但可以通过文档验证器定义规则：

```javascript
db.runCommand({
  collMod: "users",
  validator: {
    $jsonSchema: {
      bsonType: "object",
      required: ["name", "email"],
      properties: {
        name: {
          bsonType: "string",
          description: "用户名必填"
        },
        email: {
          bsonType: "string",
          pattern: "^[^@]+@[^@]+$",
          description: "邮箱格式不正确"
        },
        age: {
          bsonType: "int",
          minimum: 0,
          maximum: 150
        }
      }
    }
  }
})
```

## 查询文档

### 基础查询

```javascript
// 查询集合中所有文档
db.users.find()

// 条件查询
db.users.find({ name: "张三" })

// 查询单条文档
db.users.findOne({ name: "张三" })
```

### 比较运算符

```javascript
// 等于
db.users.find({ age: 25 })

// 大于 $gt
db.users.find({ age: { $gt: 25 } })

// 大于等于 $gte
db.users.find({ age: { $gte: 25 } })

// 小于 $lt
db.users.find({ age: { $lt: 30 } })

// 小于等于 $lte
db.users.find({ age: { $lte: 30 } })

// 不等于 $ne
db.users.find({ age: { $ne: 25 } })

// 包含在数组中 $in
db.users.find({ age: { $in: [25, 30] } })
```

### 逻辑运算符

```javascript
// AND（默认）
db.users.find({ age: { $gt: 20 }, name: "张三" })

// OR
db.users.find({
  $or: [
    { name: "张三" },
    { age: 25 }
  ]
})

// NOT
db.users.find({ age: { $not: { $gt: 25 } } })
```

### 字段选择

```javascript
// 只返回指定字段
db.users.find({}, { name: 1, email: 1 })

// 排除指定字段
db.users.find({}, { age: 0 })

// 限制返回数量
db.users.find().limit(10)

// 跳过前 N 条
db.users.find().skip(20)

// 分页查询
db.users.find().skip((page - 1) * pageSize).limit(pageSize)
```

### 排序

```javascript
// 升序
db.users.find().sort({ age: 1 })

// 降序
db.users.find().sort({ age: -1 })

// 多字段排序
db.users.find().sort({ age: 1, name: -1 })
```

## 更新文档

### 更新单条

```javascript
db.users.updateOne(
  { name: "张三" },
  { $set: { age: 29 } }
)
```

### 更新多条

```javascript
db.users.updateMany(
  { age: { $lt: 25 } },
  { $set: { status: "young" } }
)
```

### 替换文档

```javascript
db.users.replaceOne(
  { name: "张三" },
  { name: "张三", age: 30, email: "new@example.com" }
)
```

### 更新运算符

```javascript
// $set - 设置字段值
db.users.updateOne({ name: "张三" }, { $set: { age: 30 } })

// $unset - 删除字段
db.users.updateOne({ name: "张三" }, { $unset: { email: "" } })

// $inc - 数值递增
db.users.updateOne({ name: "张三" }, { $inc: { age: 1 } })

// $mul - 数值乘积
db.users.updateOne({ name: "张三" }, { $mul: { age: 2 } })

// $push - 数组添加元素
db.users.updateOne({ name: "张三" }, { $push: { tags: "VIP" } })

// $pull - 数组删除元素
db.users.updateOne({ name: "张三" }, { $pull: { tags: "VIP" } })

// $addToSet - 数组添加唯一元素
db.users.updateOne({ name: "张三" }, { $addToSet: { tags: "new" } })

// $rename - 重命名字段
db.users.updateMany({}, { $rename: { "oldName": "newName" } })
```

## 删除文档

## 聚合管道

聚合管道是 MongoDB 强大的数据处理框架，用于对文档进行转换和计算。

### 基础示例

```javascript
db.orders.aggregate([
  // 匹配阶段
  { $match: { status: "completed" } },
  
  // 投影阶段
  { $project: { orderId: 1, total: 1, customer: 1 } },
  
  // 分组阶段
  { $group: {
    _id: "$customer",
    totalAmount: { $sum: "$total" },
    orderCount: { $sum: 1 }
  }},
  
  // 排序阶段
  { $sort: { totalAmount: -1 } },
  
  // 限制阶段
  { $limit: 10 }
])
```

### 常用聚合阶段

| 阶段 | 说明 |
|------|------|
| $match | 筛选文档，类似 find |
| $project | 选择字段、重命名、计算新字段 |
| $group | 分组聚合 |
| $sort | 排序 |
| $limit | 限制数量 |
| $skip | 跳过文档 |
| $unwind | 展开数组 |
| $lookup | 左连接其他集合 |
| $addFields | 添加新字段 |
| $count | 统计数量 |

### 常用聚合运算符

```javascript
// 求和
{ $sum: "$amount" }

// 平均值
{ $avg: "$score" }

// 最大值
{ $max: "$price" }

// 最小值
{ $min: "$price" }

// 数组操作
{ $push: "$item" }        // 收集值
{ $addToSet: "$item" }   // 收集唯一值
{ $first: "$field" }     // 取第一个值
{ $last: "$field" }      // 取最后一个值
```

## 索引操作

### 创建索引

```javascript
// 单字段索引
db.users.createIndex({ name: 1 })

// 复合索引
db.users.createIndex({ age: 1, name: -1 })

// 唯一索引
db.users.createIndex({ email: 1 }, { unique: true })

// 文本索引
db.users.createIndex({ description: "text" })

// 地理空间索引
db.locations.createIndex({ location: "2dsphere" })
```

### 查看索引

```javascript
// 查看集合的所有索引
db.users.getIndexes()

// 查看索引大小
db.users.totalIndexSize()
```

### 删除索引

```javascript
// 删除指定索引
db.users.dropIndex("name_1")

// 删除所有索引（不包括 _id）
db.users.dropIndexes()
```

## 事务处理

MongoDB 4.0+ 支持多文档事务，确保 ACID 特性。

```javascript
const session = db.getMongo().startSession();

session.startTransaction({
  readConcern: { level: "snapshot" },
  writeConcern: { w: "majority" }
});

try {
  const accounts = session.getDatabase("bank").accounts;
  
  accounts.updateOne(
    { accountId: "001", balance: { $gte: 100 } },
    { $inc: { balance: -100 } }
  );
  
  accounts.updateOne(
    { accountId: "002" },
    { $inc: { balance: 100 } }
  );
  
  session.commitTransaction();
} catch (error) {
  session.abortTransaction();
  throw error;
} finally {
  session.endSession();
}
```

## 数据导出导入

### 导出为 JSON

```bash
mongosh --quiet --eval "db.users.find().forEach(doc => printjson(doc))" myDatabase > users.json
```

### 导出为 CSV

```bash
mongosh --quiet --eval "db.users.find().toArray()" myDatabase | mongoexport --collection=users --type=csv --fields=name,age,email --out=users.csv
```

### 导入数据

```bash
mongorestore --db myDatabase --collection users dump/myDatabase/users.bson
```
