# Docker 使用手冊

## 基本概念

### 映像檔
`ash
docker images
docker pull ubuntu:20.04
`

### 容器
`ash
docker run ubuntu:20.04
docker ps
docker ps -a
`

## 常用命令

`ash
# 執行容器
docker run -it ubuntu /bin/bash
docker run -d nginx
docker run -p 8080:80 nginx

# 容器操作
docker start/stop/restart <container>
docker exec -it <container> /bin/bash
docker logs <container>
docker rm <container>
`

## Dockerfile

`dockerfile
FROM node:18-al /app
COPY . .
RUN npmpine
WORKDIR install
CMD ["node", "server.js"]
`

## Docker Compose

`yaml
version: '3.8'
services:
  web:
    build: .
    ports:
      - "8080:80"
`
