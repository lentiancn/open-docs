## Linux

### Ubuntu / Debian

#### 方法一：APT 倉庫（推薦）

```bash
# 更新軟體包索引
sudo apt update

# 安裝 MySQL 伺服器
sudo apt install mysql-server

# 啟動 MySQL 服務
sudo systemctl start mysql
sudo systemctl enable mysql

# 安全配置
sudo mysql_secure_installation
```

#### 方法二：Docker

```bash
# 拉取 MySQL 映像檔
docker pull mysql:latest

# 執行 MySQL 容器
docker run -d \
  --name mysql \
  -e MYSQL_ROOT_PASSWORD=your_password \
  -v mysql-data:/var/lib/mysql \
  -p 3306:3306 \
  mysql:latest
```

### CentOS / RHEL

#### 使用 YUM 倉庫

```bash
# 下載 MySQL YUM 倉庫
sudo yum install https://dev.mysql.com/get/mysql80-community-release-el8-4.noarch.rpm

# 安裝 MySQL 伺服器
sudo yum install mysql-community-server

# 啟動 MySQL 服務
sudo systemctl start mysqld
sudo systemctl enable mysqld

# 取得臨時密碼
sudo grep 'temporary password' /var/log/mysqld.log

# 安全配置
mysql_secure_installation
```

### Fedora

```bash
# 安裝 MySQL
sudo dnf install @mysql

# 啟動服務
sudo systemctl start mysqld
sudo systemctl enable mysqld
```

### Arch Linux

```bash
# 安裝 MySQL (MariaDB)
sudo pacman -S mariadb

# 初始化資料庫
sudo mysql_install_db --user=mysql --basedir=/usr --datadir=/var/lib/mysql

# 啟動服務
sudo systemctl start mariadb
sudo systemctl enable mariadb
```

## macOS

### Homebrew（推薦）

```bash
# 安裝 Homebrew（如未安裝）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安裝 MySQL
brew install mysql

# 啟動 MySQL
brew services start mysql
```

### DMG 套件

1. 從 [dev.mysql.com](https://dev.mysql.com/downloads/mysql/) 下載 MySQL
2. 安裝 `.dmg` 套件
3. 從系統偏好設定或命令列執行 MySQL

## Windows

### MSI 安裝程式（推薦）

1. 從 [dev.mysql.com](https://dev.mysql.com/downloads/installer/) 下載 MySQL 安裝程式
2. 執行安裝程式
3. 選擇「完整」或「自訂」安裝
4. 按精靈操作

### ZIP 套件

```powershell
# 下載 ZIP 套件
# 解壓到 C:\mysql

# 初始化 MySQL
cd C:\mysql
mysqld --initialize-insecure

# 安裝 MySQL 服務
mysqld --install MySQL

# 啟動 MySQL
net start MySQL

# 設定 root 密碼
mysql -u root
ALTER USER 'root'@'localhost' IDENTIFIED BY 'YourPassword';
```

### Docker

```powershell
# 拉取並執行 MySQL
docker run -d --name mysql -e MYSQL_ROOT_PASSWORD=secret -p 3306:3306 -v mysql-data:/var/lib/mysql mysql:latest
```

## 驗證安裝

```bash
# 檢查 MySQL 版本
mysql --version

# 檢查 MySQL 服務狀態
sudo systemctl status mysql

# 連線 MySQL
mysql -u root -p

# 查看資料庫
SHOW DATABASES;
```

## 初始配置

### 建立資料庫和使用者

```sql
-- 以 root 身份連線
mysql -u root -p

-- 建立資料庫
CREATE DATABASE app_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 建立使用者
CREATE USER 'app_user'@'localhost' IDENTIFIED BY 'user_password';

-- 授予權限
GRANT ALL PRIVILEGES ON app_db.* TO 'app_user'@'localhost';

-- 刷新權限
FLUSH PRIVILEGES;
```

## 解除安裝

### Ubuntu/Debian

```bash
# 停止 MySQL
sudo systemctl stop mysql

# 解除安裝軟體包
sudo apt purge mysql-server mysql-client mysql-common
sudo apt autoremove

# 刪除資料目錄
sudo rm -rf /var/lib/mysql
sudo rm -rf /etc/mysql
```

### macOS (Homebrew)

```bash
# 停止 MySQL
brew services stop mysql

# 解除安裝
brew uninstall mysql
```
