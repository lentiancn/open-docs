# FantasyTalking FAQ

## Установка

### Ошибка установки?

Проверить версию Node.js (>= 14.0). Очистить кэш npm или установить из исходного кода.

## Использование

### Медленный синтез?

Проверить сетевое подключение. Использовать самостоятельный хостинг.

### Получить API-ключ?

Зарегистрироваться на официальном сайте.

## Развертывание

### Docker не работает?

Проверить работает ли Docker.
