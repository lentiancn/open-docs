# HeyGen Guía de Instalación

HeyGen es una plataforma de generación de video con IA que permite a los usuarios crear avatares de IA realistas y generar videos de cabeza parlante a partir de entradas de texto o audio.

## Descripción general de la plataforma

HeyGen es principalmente una plataforma SaaS basada en la nube. A diferencia de los proyectos de código abierto, HeyGen no requiere instalación local: los usuarios acceden a través de la interfaz web o la API.

### Métodos de acceso

| Método | Descripción | Requisitos |
|--------|-------------|------------|
| Interfaz web | Creación de video basada en navegador | Cuenta + Internet |
| API | Acceso programático | Clave API |
| SDK | Desarrollo local | Cuenta de desarrollador |

## Descripción general de versiones

HeyGen es un servicio en la nube, no una instalación local:

| Componente | Versión | Notas |
|------------|---------|-------|
| HeyGen Web | Última | Siempre actualizado |
| HeyGen API | v1/v2 | API REST |
| HeyGen SDK | Última | Para desarrolladores |

**Nota:** HeyGen es un producto comercial. Algunas funciones requieren una suscripción de pago.

3. Verifica tu dirección de correo electrónico

### Paso 2: Elegir un plan

| Gratis | Minutos limitados, marca de agua | Gratis |

### Paso 3: Obtener clave API (para desarrolladores)

1. Inicia sesión en el panel de HeyGen
2. Navega a Configuración → API
3. Genera la clave API

## Instalación de API (Para desarrolladores)

### Python SDK

```bash
# Instalar HeyGen SDK
pip install heygen-python

# O instalar desde el código fuente
pip install git+https://github.com/heygen-ai/heygen-python.git
```

### Node.js SDK

```bash
# Instalar HeyGen SDK
npm install heygen-node

# O instalar desde el código fuente
npm install github:heygen-ai/heygen-node
```

### Verificar instalación

```python
# Python
import heygen
print(heygen.__version__)
```

```javascript
// Node.js
const heygen = require('heygen-node');
console.log(heygen.version);
```

## Configuración de API

### Ejemplo de Python

```python
from heygen import HeyGen

# Inicializar con clave API
heygen = HeyGen(api_key="your_api_key")

# Ver información de la cuenta
account = heygen.get_account_info()
print(account)
```

### Ejemplo de Node.js

```javascript
const { HeyGen } = require('heygen-node');

const heygen = new HeyGen({ apiKey: 'your_api_key' });

// Ver información de la cuenta
const account = await heygen.getAccountInfo();
console.log(account);
```

## Variables de entorno

### Configurar clave API

```bash
# Linux/macOS
export HEYGEN_API_KEY="your_api_key"

# Windows (PowerShell)
$env:HEYGEN_API_KEY="your_api_key"

# En archivo .env
HEYGEN_API_KEY=your_api_key
```

### Python con dotenv

```python
from dotenv import load_dotenv
load_dotenv()

import heygen
heygen = HeyGen(api_key=os.getenv("HEYGEN_API_KEY"))
```

## Problemas comunes

### Errores de autenticación

**Problema:** "Clave API inválida"

**Soluciones:**
1. Verifica que la clave API sea correcta
2. Asegúrate de que la clave API esté activa
3. Verifica que la cuenta tenga acceso a la API

### Límites de velocidad

**Problema:** "Demasiadas solicitudes"

**Soluciones:**
1. Verifica los límites de uso de la API
2. Actualiza el plan para límites más altos
3. Implementa colas de solicitud

### Error en generación de video

**Problema:** "Error en la generación"

**Soluciones:**
1. Verifica los parámetros de entrada
2. Verifica el saldo de créditos
3. Intenta más tarde

## Mejores prácticas de seguridad

1. **Nunca expongas las claves API** - Usa variables de entorno
2. **Rota las claves regularmente** - Genera nuevas claves periódicamente
3. **Usa permisos mínimos** - Solo solicita los permisos necesarios
4. **Monitorea el uso** - Revisa el panel por anomalías
