# ElementPlus Einführung

ElementPlus ist ein Vue 3-basiertes UI-Komponentenbibliothek.

## Was ist ElementPlus

ElementPlus ist die Vue 3-Version von Element UI.

## Hauptmerkmale

- Vue 3-Unterstützung
- 70+ Komponenten
- TypeScript-Unterstützung
- Baum-Shaking
- Internationalisierung
- Theme-Anpassung

## Komponenten

- Grundkomponenten: Button, Icon, Layout
- Formularkomponenten: Input, Select, Date Picker
- Datenanzeige: Table, Tree, Pagination
- Feedback: Dialog, Drawer, Message

## Verwendung

```bash
npm install element-plus
```
