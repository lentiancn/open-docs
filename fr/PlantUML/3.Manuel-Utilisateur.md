# PlantUML Manuel Utilisateur

## Syntaxe de Base

### Diagramme de Séquence

```plantuml
@startuml
Alice -> Bob: Bonjour
@enduml
```

### Diagramme de Classes

```plantuml
@startuml
class Utilisateur {
    +id: int
    +nom: string
}
@enduml
```

## Types de Diagrammes

- Diagrammes de séquence
- Diagrammes de classes
- Diagrammes d'activité
