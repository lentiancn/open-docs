# Flutter - 개요

Flutter는 Google에서 개발한 오픈소스 UI 툴킷입니다.

## Flutter란?

Flutter는 모바일, 웹, 데스크탑, 임베디드 디바이스를 위한 아름답게 네이티브로 컴파일된 애플리케이션을 단일 코드베이스에서 구축하기 위한 오픈소스 UI 툴킷입니다. Flutter는 Dart 프로그래밍 언어를 사용하며, 풍부한 컴포넌트 라이브러리와 고성능 렌더링으로 유명합니다.

## 주요 기능

- **크로스플랫폼**: iOS, Android, 웹, 데스크탑용 단일 코드베이스
- **핫 리로드**: 상태를 잃지 않고 즉시 변경 사항 확인
- **풍부한 위젯**: Material Design 및 Cupertino 컴포넌트
- **높은 성능**: Skia 그래픽 엔진의 네이티브 성능
- **표현력 있는 UI**: 유연한 선언형 UI 구축
- **오픈소스**: 완전히 무료로 사용

## 핵심 기술

### Dart 언어

Flutter는 객체지향 프로그래밍 언어인 Dart를 사용합니다:

- **강력한 타이핑**: 정적 타입 검사
- **JIT & AOT**: Just-in-Time 및 Ahead-of-Time 컴파일
- **가비지 컬렉션**: 자동 메모리 관리
- **코루틴**: 비동기 프로그래밍 지원

### 위젯 시스템

Flutter에서는 모든 것이 위젯입니다:

- **StatelessWidget**: 상태 없는 위젯
- **StatefulWidget**: 상태가 있는 위젯
- **InheritedWidget**: 데이터 전파
- **RenderObjectWidget**: 렌더링 객체

## 사용 사례

- **모바일 앱**: iOS 및 Android 네이티브 경험
- **웹 앱**: 반응형 웹 애플리케이션
- **데스크탑 앱**: Windows, macOS, Linux 데스크탑 애플리케이션
- **임베디드**: IoT 디바이스 인터페이스
