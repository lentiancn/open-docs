# Docker 使用ガイド

Docker の基本的な使用方法を説明します。

---

## 基本概念

| 概念 | 説明 |
|------|------|
| イメージ | コンテナ作成用のテンプレート |
| コンテナ | イメージの実行インスタンス |
| レジストリ | イメージを保存する場所 |

---

## イメージ操作

```bash
docker pull nginx              # プル
docker images                 # 一覧
docker rmi nginx              # 削除
docker build -t myapp:1.0 .  # ビルド
```

---

## コンテナ操作

```bash
docker run -d nginx                           # バックグラウンド
docker run -d -p 8080:80 nginx                # ポートマッピング
docker run -d -v /data:/data nginx            # ボリュウーム
docker run -d -e ENV=prod nginx                # 環境変数
docker run -it ubuntu bash                    # インタラクティブ

docker ps                     # 一覧
docker stop <id>             # 停止
docker start <id>            # 起動
docker rm <id>               # 削除
docker logs <id>             # ログ
docker exec -it <id> bash   # シェルに入る
```

---

## Docker Compose

```yaml
version: '3.8'

services:
  web:
    image: nginx:alpine
    ports:
      - "80:80"
```

```bash
docker compose up -d      # 起動
docker compose down      # 停止
docker compose logs -f   # ログ
```

---

## トラブルシューティング

```bash
docker logs <container>        # ログ確認
docker system prune -a         # クリーンアップ
```
