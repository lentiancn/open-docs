# OpenCode 常見問題

常見問題解答。

## 安裝問題

### Q: 安裝失敗？

**解決方案**：
1. 檢查系統要求（需要現代終端模擬器）
2. 確認網絡連接正常
3. 嘗試使用安裝腳本：`curl -fsSL https://opencode.ai/install | bash`
4. 查看錯誤信息並重試

### Q: Windows 上無法安裝？

**解決方案**：
1. 嘗試使用 Chocolatey：`choco install opencode`
2. 或使用 Scoop：`scoop install opencode`
3. 或使用 NPM：`npm install -g opencode-ai`

### Q: 驗證安裝失敗？

**解決方案**：
1. 確認 opencode 已添加到 PATH
2. 重啟終端
3. 檢查安裝是否成功

## 使用問題

### Q: 無法連接 LLM？

**解決方案**：
1. 運行 `/connect` 命令
2. 訪問 opencode.ai/auth 登入
3. 添加帳單信息並獲取 API 金鑰
4. 將 API 金鑰粘貼到終端

### Q: 響應很慢？

**解決方案**：
1. 檢查網絡連接
2. 嘗試切換到其他 LLM 提供商
3. 減少對話上下文長度

### Q: 如何切換模型？

使用 `/connect` 命令選擇不同的 LLM 提供商。

### Q: 支持哪些模型？

支持 75+ LLM 提供商：
- OpenAI (GPT-4, Claude 等)
- Anthropic (Claude)
- Google (Gemini)
- 本地模型
- 詳見 [Models.dev](https://models.dev)

## 功能問題

### Q: AGENTS.md 是什麼？

AGENTS.md 是 OpenCode 初始化時創建的文件，包含項目結構和編碼規範，幫助 OpenCode 更好地理解項目。

### Q: Plan 模式和 Build 模式區別？

- **Plan 模式**：只生成計劃，不實際修改代碼
- **Build 模式**：直接執行修改

按 **Tab** 鍵切換模式。

### Q: 如何分享對話？

使用 `/share` 命令創建分享連結。

## 隱私問題

### Q: OpenCode 會保存我的代碼嗎？

不會。OpenCode 不會存儲任何代碼或上下文數據，可以在敏感環境中安全使用。

### Q: 如何保護隱私？

- 不在對話中輸入敏感信息
- 使用本地模型
- 定期清理對話
