# Element Plus FAQ

よくある質問。

## インストール

### Q: インストール失敗？

**解決策:**
1. Node.js バージョン確認
```bash
node --version  # 18.0+ 必要
```
2. npm キャッシュクリア
```bash
npm cache clean --force
```

### Q: TypeScript エラー？

**解決策:**
型定義をインストール:
```bash
npm install -D @types/node
```

tsconfig.json に追加:
```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

## 使用

### Q: コンポーネント表示されない？

**解決策:**
1. インポート確認
2. コンソールエラー確認
3. Vue バージョン確認 (3.2+)

### Q: スタイル効かない？

**解決策:**
1. CSS インポート確認
2. `:deep()` 试试

### Q: アイコン表示されない？

**解決策:**
アイコンパッケージをインストール:
```bash
npm install @element-plus/icons-vue
```

登録:
```typescript
import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```

## テーマ

### Q: テーマ色を変更？

**解決策:**
CSS 変数を使用:
```css
:root {
  --el-color-primary: #409eff;
}
```
