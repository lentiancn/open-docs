# Element UI Guía de Instalación

Element UI es una biblioteca de componentes UI para aplicaciones Vue 2.x.

## Versiones

| Versión | Versión Vue | Estado | Fecha |
|---------|-------------|--------|-------|
| Element UI 2.x | Vue 2.6+ | Actual | 2020 |
| Element UI 1.x | Vue 2.x | Legado | 2016 |

**Nota:** Element UI es solo para Vue 2. Para Vue 3, use Element Plus.

## Requisitos previos

### Sistema

| Requisito | Versión |
|-----------|---------|
| Node.js | 8.0+ |
| npm | 5.0+ |
| Vue | 2.6+ |
| Navegador | IE 10+, Chrome, Firefox, Safari |

## Instalación

### Método 1: npm (Recomendado)

```bash
npm install element-ui --save
```

### Método 2: CDN

```html
<script src="https://unpkg.com/vue@2.6.14/dist/vue.js"></script>
<link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
<script src="https://unpkg.com/element-ui/lib/index.js"></script>
```

## Configuración

### Importación completa

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```

### Importación bajo demanda

```bash
npm install babel-plugin-component -D
```

```javascript
// babel.config.js
module.exports = {
  plugins: [
    ['component', {
      libraryName: 'element-ui',
      styleLibraryName: 'theme-chalk'
    }]
  ]
}
```

```javascript
import { Button, Dialog, Input } from 'element-ui'
Vue.component(Button.name, Button)
```

## Migración a Element Plus (Vue 3)

```bash
npm install element-plus --save
```

## Mejores prácticas

1. **Importación bajo demanda** - Reduce tamaño del bundle
2. **Personalizar tema** - Usar variables SCSS
3. **Usar Vue 2.6+** - Mejor compatibilidad
