# Claude Code Руководство Пользователя

Руководство по быстрому освоению Claude Code и выполнению ежедневных задач разработки.

## Подготовка

Убедитесь, что у вас есть:
- Открытый терминал
- Проект для работы
- Подписка Claude (Pro, Max, Teams или Enterprise)

## Запуск Сессии

Откройте терминал в папке проекта и запустите Claude Code:

```bash
cd /path/to/your/project
claude
```

## Основные Операции

### Понимание Кодовой Базы

Попросите Claude проанализировать проект:
```
Что делает этот проект?
```

```
Какие технологии использует этот проект?
```

### Первое Изменение Кода

Попробуйте простую задачу:
```
Добавьте функцию hello world в главный файл
```

Claude Code:
1. Найдёт подходящий файл
2. Покажет предлагаемые изменения
3. Запросит разрешение
4. Выполнит редактирование

### Операции Git

```
Какие файлы я изменил?
```

```
Создайте коммит с описательным сообщением
```

```
Создайте новую ветку feature/quickstart
```

### Исправление Ошибок и Добавление Функций

Опишите желаемое на обычном языке:
```
Добавьте валидацию в форму регистрации
```

## Основные Команды

| Команда | Функция |
|---------|---------|
| `claude` | Интерактивный режим |
| `claude "задача"` | Выполнить задачу |
| `claude -p "запрос"` | Одиночный запрос |
| `claude commit` | Создать коммит |
| `/help` | Доступные команды |

## Продвинутые Советы

### CLAUDE.md
Создайте файл CLAUDE.md в корне проекта для постоянного контекста.

### Пользовательские Команды
Создайте команды вроде `/review-pr` или `/deploy-staging`.

### Мультиагентная Работа
Для сложных задач запустите несколько агентов параллельно.

## Лучшие Практики
