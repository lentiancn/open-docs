# Apifox ユーザーマニュアル

Apifoxの基本機能。

## API管理

### インターフェース作成

1. プロジェクトを選択
2. 「新規インターフェース」をクリック
3. 情報を入力

### デバッグ

1. インターフェースを選択
2. パラメータを入力
3. 送信をクリック
4. レスポンスを確認

## リクエスト設定

### リクエストヘッダー

```http
Content-Type: application/json
Authorization: Bearer token
```

### 認証

- Bearer Token
- Basic Auth
- OAuth 2.0

## Mockサービス

### Mock作成

1. 「新規Mock」をクリック
2. レスポンスルールを定義
3. Mock URLを取得

## 自動化テスト

### テストケース作成

1. 「新規テスト」をクリック
2. テストステップを追加
3. アサーションを設定

## チーム协作

### メンバー招待

1. プロジェクト設定
2. メンバー管理
3. 招待

## 環境変数

### 環境作成

1. 環境管理
2. 新規環境
3. 変数を設定
