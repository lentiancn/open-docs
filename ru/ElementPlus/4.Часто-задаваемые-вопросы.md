# Часто задаваемые вопросы Element Plus

## Проблемы установки

### Ошибка установки

Проверьте версию Node.js (>= 14.0.0)
Очистите кэш npm: npm cache clean --force

### Ошибки TypeScript

Добавьте в tsconfig.json:

{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}

## Проблемы использования

### Стили не применяются

Импортируйте CSS:

import 'element-plus/dist/index.css'

### Иконки не отображаются

Установите библиотеку иконок:

npm install @element-plus/icons-vue

## Проблемы с производительностью

При больших объемах данных:
- Используйте пагинацию
- Используйте el-table-v2 (виртуальная таблица)

## Проблемы с версией Vue

Element Plus поддерживает только Vue 3.
