# FantasyTalking Manuel d'Utilisation

## Fonctionnalités de Base

### Synthèse Vocale

La synthèse vocale est la fonction principale de FantasyTalking, convertissant le texte en voix naturelle et fluide:

```javascript
// Texte simple vers voix
const audio = await client.speak('Bonjour, bienvenue chez FantasyTalking');

// Spécifier le style vocal
const audio = await client.speak('Le temps est magnifique aujourd\'hui', {
  voice: {
    style: 'happy',
    language: 'fr-FR'
  }
});

// Spécifier la vitesse et la hauteur
const audio = await client.speak('Veuillez patienter un instant', {
  voice: {
    speed: 1.2,
    pitch: 0.9
  }
});
```

### Langage de Balisage SSML

FantasyTalking supporte le langage SSML pour un contrôle plus fin de la sortie vocale:

```xml
<speak>
  <p>
    <s>Première phrase.</s>
    <s>Deuxième phrase.</s>
  </p>
  <break time="500ms"/>
  <p>
    <emphasis level="moderate">Ceci est un contenu important</emphasis>
  </p>
</speak>
```

### Événements Vocaux

Pendant la lecture vocale, vous pouvez écouter divers événements:

```javascript
client.on('speech-start', (event) => {
  console.log('Lecture vocale démarrée');
});

client.on('word-boundary', (event) => {
  console.log(`Mot actuel: ${event.word}`);
});

client.on('speech-end', (event) => {
  console.log('Lecture vocale terminée');
});

client.on('error', (event) => {
  console.error('Erreur:', event.error);
});
```

### Synthèse par Lots

Pour synthétiser de grandes quantités de texte:

```javascript
const texts = [
  'Ceci est la première voix',
  'Ceci est la deuxième voix',
  'Ceci est la troisième voix'
];

const results = await Promise.all(
  texts.map(text => client.speak(text))
);
```

## Fonctionnalités de Dialogue

### Dialogue de Base

FantasyTalking fournit des fonctions puissantes de gestion de dialogue:

```javascript
const session = client.createSession({
  systemPrompt: 'Vous êtes un assistant amical',
  language: 'fr-FR'
});

const response1 = await session.chat('Comment allez-vous?');
console.log(response1.text);
```

## Fonctionnalités Avancées

### Synthèse en Streaming

```javascript
const stream = await client.speakStream('Ceci est un long texte');
stream.on('data', (chunk) => {
  audioPlayer.play(chunk);
});
```

### Gestion des Erreurs

```javascript
try {
  const audio = await client.speak('Texte de test');
} catch (error) {
  if (error.code === 'RATE_LIMIT') {
    console.log('Trop de demandes, réessayez plus tard');
  }
}
```
