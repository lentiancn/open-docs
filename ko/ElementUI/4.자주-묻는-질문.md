# Element UI 자주 묻는 질문

## 설치 문제

Node.js >= 8.0.0 확인

npm cache clean --force

## 사용 문제

### 스타일이 적용되지 않음

CSS 임포트:

import 'element-ui/lib/theme-chalk/index.css'

### 아이콘이 표시되지 않음

아이콘 글꼴 임포트:

import 'element-ui/lib/theme-chalk/fonts/element-icons.ttf'

## Vue 버전 문제

Element UI는 Vue 2만 지원합니다.

## 도움말

GitHub Issues: https://github.com/ElemeFE/element/issues
