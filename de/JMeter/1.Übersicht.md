# Apache JMeter - Übersicht

## Was ist Apache JMeter?

Apache JMeter™ ist Open-Source-Software, eine zu 100% reine Java-Anwendung, die für Lasttests und Leistungsmessung entwickelt wurde. Sie wurde ursprünglich für das Testen von Web-Anwendungen entwickelt, hat sich aber seitdem auf andere Testfunktionen ausgeweitet.

## Was kann JMeter tun?

Apache JMeter kann verwendet werden, um die Leistung sowohl bei statischen als auch dynamischen Ressourcen und dynamischen Web-Anwendungen zu testen. Es kann eine schwere Last auf einen Server, eine Gruppe von Servern, ein Netzwerk oder ein Objekt simulieren, um seine Stärke zu testen oder die Gesamtleistung unter verschiedenen Lasttypen zu analysieren.

### Hauptfunktionen

- **Multi-Protokoll-Unterstützung**: Web (HTTP, HTTPS), SOAP/REST Webservices, FTP, Datenbank über JDBC, LDAP, JMS, Mail, TCP, Java-Objekte
- **Vollständige Test-IDE**: Testpläne von Browsern oder nativen Anwendungen aufnehmen
- **CLI-Modus**: Lasttests von jedem Java-kompatiblen Betriebssystem ausführen
- **Dynamische HTML-Berichte**: Vollständig und präsentationsbereit
- **Datenextraktion**: Unterstützung für HTML, JSON, XML und reguläre Ausdrücke
- **100% Java**: Vollständige Portabilität
- **Multi-Threading**: Gleichzeitige Stichproben durch viele Threads
- **Hoch erweiterbar**: Einsteckbare Sampler und skriptbare Sampler (Groovy, BeanShell)
- **CI-Integration**: Maven, Gradle und Jenkins-Unterstützung

## Was JMeter NICHT ist

JMeter ist kein Browser - es arbeitet auf Protokollebene. Es führt kein JavaScript aus, das in HTML-Seiten gefunden wird, und rendert HTML-Seiten nicht wie ein echter Browser.

## Erste Schritte

1. JMeter von der offiziellen Website herunterladen
2. Das JMeter-Skript ausführen (jmeter.bat für Windows, jmeter.sh für Linux/Mac)
3. Erstellen Sie Ihren Testplan mit der GUI
4. Tests ausführen und Ergebnisse analysieren
