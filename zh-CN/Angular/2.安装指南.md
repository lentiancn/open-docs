# 安装指南

本文档介绍 Angular 的安装和环境配置，帮助您快速搭建开发环境。

## 环境要求

在安装 Angular 之前，请确保满足以下环境要求：

- **Node.js**：推荐 18.10.0 或更高版本
- **npm**：Node.js 自带，版本 6.x 或更高
- **操作系统**：Windows、macOS、Linux

## 安装 Angular CLI

Angular CLI 是 Angular 官方的命令行工具，用于创建和管理 Angular 项目。

### 全局安装

```bash
npm install -g @angular/cli
```

### 验证安装

```bash
ng version
```

如果安装成功，会显示 Angular CLI 版本信息。

### 安装指定版本

如果需要安装特定版本：

```bash
npm install -g @angular/cli@17
```

## 创建新项目

### 使用 CLI 创建

```bash
ng new my-app
```

这会创建一个名为 my-app 的新项目，并自动安装依赖。

### 创建时指定选项

```bash
# 创建并跳过安装
ng new my-app --skip-install

# 使用 SCSS 样式
ng new my-app --style=scss

# 启用路由
ng new my-app --routing

# 创建独立组件（Angular 14+）
ng new my-app --standalone
```

## 开发环境配置

### 使用国内镜像

由于网络原因，建议配置国内镜像加速：

```bash
# 设置 npm 镜像
npm config set registry https://registry.npmmirror.com

# 或使用 nrm 工具管理镜像
npm install -g nrm
nrm use taobao
```

### 安装 Node.js

**使用 nvm 安装（推荐）：**

```bash
# 安装 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 安装 Node.js LTS 版本
nvm install --lts
nvm use --lts
```

**Windows 使用 nvm-windows：**

从 https://github.com/coreybutler/nvm-windows 下载安装包。

## 启动开发服务器

### 进入项目目录

```bash
cd my-app
```

### 启动开发服务器

```bash
ng serve
```

默认访问地址：http://localhost:4200

### 实时重载

Angular CLI 支持实时重载，修改代码后会自动刷新浏览器。

## 构建生产版本

### 构建命令

```bash
ng build
```

构建产物会输出到 dist 目录。

### 生产环境优化

```bash
ng build --configuration=production
```

这会启用代码压缩、 tree-shaking 等优化。

## 常用 CLI 命令

| 命令 | 说明 |
|------|------|
| ng generate component xxx | 生成组件 |
| ng generate service xxx | 生成服务 |
| ng generate module xxx | 生成模块 |
| ng generate directive xxx | 生成指令 |
| ng generate pipe xxx | 生成管道 |
| ng generate guard xxx | 生成路由守卫 |

## 开发工具

### VS Code 扩展

- Angular Language Service：提供模板类型检查和自动完成
- Angular Snippets：提供常用代码片段
- Prettier：代码格式化

### 调试

Angular 提供 Angular DevTools 浏览器扩展，用于调试和性能分析。

## 卸载 Angular CLI

如果需要卸载：

```bash
npm uninstall -g @angular/cli
```
