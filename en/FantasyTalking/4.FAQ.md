# FantasyTalking FAQ

Frequently asked questions and answers.

## Installation Issues

### Q: Dependency installation failed?

**Solution:**
1. Upgrade pip: `pip install --upgrade pip`
2. Use Chinese mirror:
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```
3. Install failing packages individually

### Q: CUDA not available?

**Solution:**
1. Confirm NVIDIA driver is installed: `nvidia-smi`
2. Install matching PyTorch version:
```bash
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
```
3. Verify CUDA availability:
```python
import torch
print(torch.cuda.is_available())
```

### Q: Out of memory?

**Solution:**
1. Use smaller input images
2. Reduce batch size
3. Use CPU mode (not recommended, slow)

## Usage Issues

### Q: Generation failed, face not found?

**Solution:**
1. Use image with clear face
2. Ensure face is front-facing, no occlusion
3. Check image format

### Q: Video generation very slow?

**Solution:**
1. Use GPU acceleration
2. Reduce output resolution
3. Reduce output frames

### Q: Lips not synced?

**Solution:**
1. Ensure audio is clear and noise-free
2. Check audio sample rate
3. Try different sync parameters

### Q: Output video blurry?

**Solution:**
1. Use face enhancer: `--face_enhancer realusf`
2. Increase output resolution
3. Use clearer source image

### Q: Face deformed/distorted?

**Solution:**
1. Use front-facing photo without occlusion
2. Ensure only one face in image
3. Adjust face detection parameters

## Model Issues

### Q: Model download failed?

**Solution:**
1. Use proxy or mirror
2. Download model files manually
3. Check network connection

### Q: Model failed to load?

**Solution:**
1. Confirm model file is complete
2. Check model path is correct
3. Re-download model

## Performance Issues

### Q: How to improve generation speed?

**Solution:**
1. Use GPU
2. Enable mixed precision: `--precision fp16`
3. Use smaller resolution

### Q: Memory usage too high?

**Solution:**
1. Reduce batch size
2. Use smaller model
3. Disable unnecessary features

## Output Issues

### Q: Output video cannot play?

**Solution:**
1. Install decoder: `pip install opencv-python`
2. Use supported format: MP4

### Q: Video file too large?

**Solution:**
1. Reduce resolution
2. Reduce frame rate
3. Use higher compression
