# Guía de Uso de Docker

Guía básica de uso de Docker.

---

## Conceptos Básicos

| Concepto | Descripción |
|----------|-------------|
| Imagen | Plantilla para crear contenedores |
| Contenedor | Instancia ejecutando una imagen |
| Registro | Donde se almacenan las imágenes |

---

## Operaciones con Imágenes

```bash
docker pull nginx              # Descargar
docker images                 # Listar
docker rmi nginx              # Eliminar
docker build -t myapp:1.0 .  # Construir
```

---

## Operaciones con Contenedores

```bash
docker run -d nginx                           # Fondo
docker run -d -p 8080:80 nginx                # Puerto
docker run -d -v /data:/data nginx            # Volumen
docker run -d -e ENV=prod nginx               # Variable

docker ps                     # Listar
docker stop <id>             # Detener
docker start <id>            # Iniciar
docker rm <id>               # Eliminar
docker logs <id>             # Ver registros
docker exec -it <id> bash   # Entrar al contenedor
```

---

## Docker Compose

```yaml
version: '3.8'

services:
  web:
    image: nginx:alpine
    ports:
      - "80:80"
```

```bash
docker compose up -d      # Iniciar
docker compose down      # Detener
docker compose logs -f   # Registros
```

---

## Solución de Problemas

```bash
docker logs <container>        # Ver registros
docker system prune -a         # Limpiar
```
