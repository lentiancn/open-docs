# MongoDB 使用手册

## 基本概念

### 数据库

数据库是 MongoDB 中最高级别的数据容器。每个数据库都有自己的集合和权限设置。在 MongoDB 中可以创建多个数据库，每个数据库都是独立的。

### 集合

集合就是一组 MongoDB 文档，相当于关系数据库中的表。集合中的文档可以拥有不同的字段，这使得集合非常灵活。与关系数据库不同，集合不实施严格的预定义模式。

### 文档

文档是 MongoDB 中的基本数据单元，由字段和值对组成。文档类似于 JSON 对象，使用 BSON（二进制 JSON）格式存储。

BSON 是 JSON 文档的二进制表示形式，有额外的数据类型支持。文档大小上限为 16 MB。

## 插入文档

MongoDB 提供以下方法将文档插入到集合中：

### 插入单个文档

```bash
db.users.insertOne({
  name: "张三",
  age: 30,
  email: "zhangsan@example.com",
  address: {
    city: "北京",
    street: "朝阳区"
  },
  hobbies: ["篮球", "游泳"]
})
```

### 插入多个文档

```bash
db.users.insertMany([
  { name: "李四", age: 25, city: "上海" },
  { name: "王五", age: 35, city: "广州" },
  { name: "赵六", age: 28, city: "深圳" }
])
```

### 插入文档（已过时但仍然可用）

```bash
db.users.insert({
  name: "钱七",
  age: 32
})
```

### 查看插入结果

```bash
# 查看插入的文档 ID
db.users.insertOne({ name: "测试" })
# 返回：{ acknowledged: true, insertedId: ObjectId("...") }
```

## 查询文档

### 查询所有文档

```bash
db.users.find()
```

相当于 SQL 中的 `SELECT * FROM users`

### 查询单条记录

```bash
db.users.findOne()
```

### 条件查询

```bash
# 等于查询
db.users.find({ age: 30 })

# 不等于查询
db.users.find({ age: { $ne: 30 } })

# 大于查询
db.users.find({ age: { $gt: 25 } })

# 大于等于查询
db.users.find({ age: { $gte: 25 } })

# 小于查询
db.users.find({ age: { $lt: 30 } })

# 小于等于查询
db.users.find({ age: { $lte: 30 } })

# 范围查询
db.users.find({ age: { $gte: 20, $lte: 30 } })
```

### 逻辑运算符

```bash
# AND（默认）
db.users.find({ age: { $gte: 25 }, name: "李四" })

# OR
db.users.find({ $or: [{ age: { $gte: 30 } }, { name: "王五" }] })

# NOR（既不也不）
db.users.find({ $nor: [{ age: { $lt: 20 } }, { city: "北京" }] })

# NOT
db.users.find({ age: { $not: { $gte: 25 } } })
```

### 数组查询

```bash
# 数组包含元素
db.users.find({ hobbies: "篮球" })

# 数组包含所有元素
db.users.find({ hobbies: { $all: ["篮球", "游泳"] } })

# 数组长度
db.users.find({ hobbies: { $size: 2 } })

# 数组特定位置元素
db.users.find({ "hobbies.0": "篮球" })

# 数组元素满足条件
db.users.find({ hobbies: { $elemMatch: { $eq: "篮球" } } })
```

### 嵌套文档查询

```bash
# 嵌套文档字段查询
db.users.find({ "address.city": "北京" })

# 嵌套文档完全匹配
db.users.find({ address: { city: "北京", street: "朝阳区" } })
```

### 投影查询（只返回指定字段）

```bash
# 只返回 name 和 email 字段
db.users.find({ age: 30 }, { name: 1, email: 1 })

# 排除某个字段
db.users.find({}, { age: 0 })

# 返回所有字段除了 _id
db.users.find({}, { _id: 0 })
```

### 排序

```bash
# 按年龄升序
db.users.find().sort({ age: 1 })

# 按年龄降序
db.users.find().sort({ age: -1 })

# 多字段排序
db.users.find().sort({ age: -1, name: 1 })
```

### 限制和跳过

```bash
# 返回前 5 条
db.users.find().limit(5)

# 跳过前 10 条，返回接下来的 5 条（分页）
db.users.find().skip(10).limit(5)

# 统计数量
db.users.countDocuments({ age: { $gte: 25 } })

# 估计集合中文档数量（快但可能不精确）
db.users.estimatedDocumentCount()
```

### 游标操作

```bash
# 使用游标
var cursor = db.users.find()
while (cursor.hasNext()) {
  printjson(cursor.next())
}

# forEach 遍历
db.users.find().forEach(function(doc) {
  print("Name: " + doc.name)
})
```

## 更新文档

### 更新单个文档

```bash
db.users.updateOne(
  { name: "张三" },
  { $set: { age: 31 } }
)
```

### 更新多个文档

```bash
db.users.updateMany(
  { age: { $lt: 30 } },
  { $set: { status: "young" } }
)
```

### 替换文档

```bash
db.users.replaceOne(
  { name: "张三" },
  { name: "张三", age: 32, email: "new@example.com", city: "北京" }
)
```

### 更新运算符

```bash
# $set - 设置字段值
db.users.updateOne({ name: "李四" }, { $set: { age: 26 } })

# $unset - 移除字段
db.users.updateOne({ name: "李四" }, { $unset: { email: "" } })

# $inc - 递增/递减
db.users.updateOne({ name: "李四" }, { $inc: { age: 1 } })
db.users.updateOne({ name: "李四" }, { $inc: { age: -1 } })

# $rename - 重命名字段
db.users.updateOne({ name: "李四" }, { $rename: { "name": "fullname" } })

# $mul - 乘
db.users.updateOne({ name: "李四" }, { $mul: { age: 2 } })

# $min - 取最小值
db.users.updateOne({ name: "李四" }, { $min: { age: 20 } })

# $max - 取最大值
db.users.updateOne({ name: "李四" }, { $max: { age: 40 } })

# $push - 添加元素到数组
db.users.updateOne({ name: "李四" }, { $push: { hobbies: "跑步" } })

# $pull - 移除数组中匹配的元素
db.users.updateOne({ name: "李四" }, { $pull: { hobbies: "游泳" } })

# $addToSet - 添加元素到数组（不重复）
db.users.updateOne({ name: "李四" }, { $addToSet: { hobbies: "篮球" } })

# $pop - 移除数组第一个或最后一个元素
db.users.updateOne({ name: "李四" }, { $pop: { hobbies: 1 } })
```

### upsert 操作

```bash
# 如果不存在则插入，存在则更新
db.users.updateOne(
  { name: "新用户" },
  { $set: { age: 25, city: "杭州" } },
  { upsert: true }
)
```

## 聚合管道

聚合管道是 MongoDB 最强大的功能之一，用于处理数据流，执行复杂的数据转换和分析。

### 基本语法

```bash
db.collection.aggregate([
  { $stage1: { ... } },
  { $stage2: { ... } },
  ...
])
```

### 常用聚合阶段

#### $match - 过滤文档

```bash
db.users.aggregate([
  { $match: { age: { $gte: 25 } } }
])
```

#### $project - 投影

```bash
db.users.aggregate([
  { $project: { _id: 0, name: 1, age: 1 } }
])
```

#### $group - 分组

```bash
# 统计每个年龄的用户数量
db.users.aggregate([
  { $group: { _id: "$age", count: { $sum: 1 } } }
])

# 统计平均年龄
db.users.aggregate([
  { $group: { _id: null, avgAge: { $avg: "$age" } } }
])

# 分组并收集名字
db.users.aggregate([
  { $group: { _id: "$city", users: { $push: "$name" } } }
])
```

#### $sort - 排序

```bash
db.users.aggregate([
  { $sort: { age: -1, name: 1 } }
])
```

#### $limit - 限制数量

```bash
db.users.aggregate([
  { $limit: 5 }
])
```

#### $skip - 跳过文档

```bash
db.users.aggregate([
  { $skip: 10 },
  { $limit: 5 }
])
```

#### $unwind - 展开数组

```bash
db.users.aggregate([
  { $unwind: "$hobbies" }
])
```

#### $lookup - 联表查询

```bash
# 假设有两个集合：orders 和 products
db.orders.aggregate([
  {
    $lookup: {
      from: "products",
      localField: "productId",
      foreignField: "_id",
      as: "productDetails"
    }
  }
])
```

### 聚合运算符

#### 算术运算符

```bash
db.sales.aggregate([
  {
    $project: {
      total: { $add: ["$price", "$tax"] },
      discount: { $multiply: ["$price", 0.1] }
    }
  }
])
```

#### 字符串运算符

```bash
db.users.aggregate([
  {
    $project: {
      fullName: { $concat: ["$firstName", " ", "$lastName"] },
      upperName: { $toUpper: "$name" }
    }
  }
])
```

#### 日期运算符

```bash
db.users.aggregate([
  {
    $project: {
      year: { $year: "$birthdate" },
      month: { $month: "$birthdate" },
      day: { $dayOfMonth: "$birthdate" }
    }
  }
])
```

### 完整示例

```bash
# 找到从 PDX 机场出发的直飞航班最多的前三大航空公司
db.routes.aggregate([
  {
    $match: {
      "src_airport": "PDX",
      "stops": 0
    }
  },
  {
    $group: {
      _id: "$airline.name",
      count: { $sum: 1 }
    }
  },
  {
    $sort: { count: -1 }
  },
  {
    $limit: 3
  }
])
```

## 索引

索引支持在 MongoDB 中高效执行查询。如果没有索引，MongoDB 就必须扫描集合中的每个文档以返回查询结果。

### 创建索引

```bash
# 单字段索引
db.users.createIndex({ name: 1 })

# 降序索引
db.users.createIndex({ age: -1 })

# 复合索引
db.users.createIndex({ age: 1, name: 1 })

# 唯一索引
db.users.createIndex({ email: 1 }, { unique: true })

# 文本索引
db.users.createIndex({ description: "text" })

# 多键索引（数组字段）
db.users.createIndex({ hobbies: 1 })

# 地理空间索引
db.locations.createIndex({ location: "2dsphere" })

# 哈希索引
db.users.createIndex({ _id: "hashed" })
```

### 查看索引

```bash
# 查看集合的所有索引
db.users.getIndexes()

# 查看集合的索引大小
db.users.totalIndexSize()
```

### 删除索引

```bash
# 删除指定索引
db.users.dropIndex("name_1")

# 删除所有索引（保留 _id 索引）
db.users.dropIndexes()

# 删除文本索引
db.users.dropIndex("description_text")
```

### 索引属性

```bash
# 稀疏索引（只索引非空值）
db.users.createIndex({ email: 1 }, { sparse: true })

# 部分索引
db.users.createIndex(
  { email: 1 },
  { partialFilterExpression: { email: { $exists: true } } }
)

# TTL 索引（自动过期）
db.sessions.createIndex(
  { createdAt: 1 },
  { expireAfterSeconds: 3600 }
)
```

### 索引策略建议

- 为经常用于查询条件的字段创建索引
- 复合索引的字段顺序很重要：精确匹配的字段放在前面，范围查询的字段放在后面
- 避免在写入频率高的集合上创建过多索引
- 使用 explain() 分析查询性能

### 查询分析

```bash
# 分析查询
db.users.find({ age: 25 }).explain()

# 分析执行计划
db.users.find({ age: { $gte: 25 } }).sort({ name: 1 }).explain("executionStats")
```

## 数据导出导入

### 使用 mongodump 和 mongorestore

#### 导出数据

```bash
# 导出整个数据库
mongodump --db=mydatabase --out=/backup

# 导出指定集合
mongodump --db=mydatabase --collection=users --out=/backup

# 导出并压缩
mongodump --db=mydatabase --gzip --out=/backup
```

#### 导入数据

```bash
# 恢复整个数据库
mongorestore --db=mydatabase /backup/mydatabase

# 恢复指定集合
mongorestore --db=mydatabase --collection=users /backup/mydatabase/users.bson

# 恢复并压缩
mongorestore --db=mydatabase --gzip /backup/mongodb
```

#### 导出为 JSON

```bash

# 导出为 JSON 数组
mongoexport --db=mydatabase --collection=users --out=users.json --jsonArray

mongoexport --db=mydatabase --collection=users --out=users.csv --fields=name,age,email
```


## 事务操作

MongoDB 4.0+ 支持多文档事务。

### 基本事务

```bash
session = db.getMongo().startSession()
session.startTransaction({ readConcern: { level: "snapshot" }, writeConcern: { w: "majority" } })

try {
  session.getDatabase("mydb").accounts.updateOne(
    { accountId: 1 },
    { $inc: { balance: -100 } }
  )
  session.getDatabase("mydb").accounts.updateOne(
    { accountId: 2 },
    { $inc: { balance: 100 } }
  )
  session.commitTransaction()
} catch (error) {
  session.abortTransaction()
  throw error
} finally {
  session.endSession()
}
```
