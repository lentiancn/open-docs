## 1. Базовые компоненты

### 1.1 Компонент кнопки

Кнопки — один из наиболее часто используемых компонентов.

#### Базовое использование

```html
<el-button>По умолчанию</el-button>
<el-button type="primary">Основная</el-button>
<el-button type="success">Успех</el-button>
<el-button type="warning">Предупреждение</el-button>
<el-button type="danger">Опасность</el-button>
```

#### Атрибуты кнопки

| Атрибут | Описание | Тип | По умолчанию |
|---------|-----------|-----|--------------|
| type | Тип кнопки | string | — |
| plain | Простой стиль | boolean | false |
| round | Закругленная | boolean | false |
| disabled | Отключена | boolean | false |

## 2. Компоненты формы

### 2.1 Поле ввода

```html
<el-input v-model="input" placeholder="Введите текст"></el-input>
```

### 2.2 Выпадающий список

```html
<el-select v-model="selected" placeholder="Выберите">
  <el-option label="Вариант 1" value="1"></el-option>
  <el-option label="Вариант 2" value="2"></el-option>
</el-select>
```

### 2.3 Валидация формы

```javascript
this.$refs.form.validate(valid => {
  if (valid) {
    console.log('Валидация успешна')
  }
})
```

## 3. Компонент таблицы

### 3.1 Базовая таблица

```html
<el-table :data="tableData" stripe border>
  <el-table-column prop="name" label="Имя"></el-table-column>
  <el-table-column prop="email" label="Email"></el-table-column>
</el-table>
```

### 3.2 Таблица с пагинацией

```html
<el-pagination
  @current-change="handlePageChange"
  :current-page="currentPage"
  :total="total">
</el-pagination>
```

## 4. Диалоги и уведомления

### 4.1 Диалог

```html
<el-dialog title="Заголовок" :visible.sync="visible">
  <span>Содержание диалога</span>
</el-dialog>
```

### 4.2 Сообщения

```javascript
this.$message.success('Успешно')
this.$message.error('Ошибка')
this.$message.warning('Предупреждение')
```
