# Apifox 概要

API 一体型协作プラットフォーム。

## Apifoxとは？

Apifoxは、APIドキュメント、APIデバッグ、API Mock、API自動化テスト于一体的API一体化协作プラットフォームです。

### 主な機能

- **APIドキュメント** - 自動生成、、チーム协作
- **APIデバッグ** - HTTPリクエストシミュレーションツール
- **Mockサービス** - 快速Mockサーバー作成
- **自動化テスト** - API自動化テスト機能
- **チーム协作** - プロジェクト管理、权限控制

### 統計

- ユーザー数：数百万开发者
