# Resumen de Bootstrap

> Marco CSS frontend popular

---

## Introducción al Proyecto

Bootstrap es un marco CSS de código abierto desarrollado y mantenido por Twitter, lanzado por primera vez en 2011. Bootstrap proporciona ricos componentes preconstruidos y un sistema de cuadrícula receptiva para construir rápidamente sitios web y aplicaciones web modernas y receptivas.

---

## Características Principales

- **Diseño Responsivo**: Diseños receptivos incorporados
- **Componentes Preconstruidos**: Botones, navegación, formularios, tarjetas, etc.
- **Potente Sistema de Cuadrícula**: Cuadrícula receptiva de 12 columnas
- **Opciones de Personalización**: Variables Sass para fácil personalización
- **Plugins JavaScript**: Menús desplegables, modales, etc.
- **Biblioteca de Iconos**: Bootstrap Icons incorporado

---

## Casos de Uso

- Desarrollo de sitios web responsivos
- Sistemas de administración
- Prototipado rápido
- Proyectos mobile-first
