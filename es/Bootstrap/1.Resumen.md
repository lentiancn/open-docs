# Bootstrap Resumen

El marco CSS frontend más popular del mundo.

## ¿Qué es Bootstrap?

Bootstrap es el marco CSS frontend más popular desarrollado por Twitter para construir sitios web responsivos y mobile-first.

### Características principales

- **Sistema de rejilla responsivo** - Rejilla de 12 columnas
- **Componentes preconstruidos** - Navbars, tarjetas, modales y más
- **Clases utilitarias CSS** - Ajustes rápidos de estilo
- **Componentes JavaScript** - Componentes interactivos

### Estadísticas

- GitHub Stars: 160,000+
- Sitios web utilizados: Millones
