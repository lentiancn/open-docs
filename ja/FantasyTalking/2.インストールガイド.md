# FantasyTalking インストールガイド

## 動作環境

FantasyTalkingをインストールする前に、開発環境が以下の要件を満たしていることを確認してください。OSについては、プロジェクトはWindows、macOS、Linuxをサポートし、最適なパフォーマンスを得るために64ビットシステムの使用をお勧めします。Node.jsのバージョンは14.0以上、Pythonのバージョンは3.7以上が必要です。Pythonバックエンドを使用する場合は、FFmpegオーディオ処理ツールもインストールする必要があります。

ハードウェアについては、スムーズな音声合成体験を得るために、システムには少なくとも4GBのRAMとデュアルコアプロセッサが必要です。

## インストール方法

### npmを使用したインストール（Node.js）

```bash
npm install fantasy-talking
npm install fantasy-talking-ui
npm install fantasy-talking-node
```

### pipを使用したインストール（Python）

```bash
pip install fantasy-talking
pip install fantasy-talking[all]
```

## クイックスタート

### JavaScript/TypeScript

```javascript
const client = new FantasyTalking({
  apiKey: 'your-api-key',
  voice: { style: 'neutral', language: 'ja-JP' }
});

const audio = await client.speak('こんにちは、これはテストです');
```

## 設定オプション

| パラメータ | タイプ | デフォルト | 説明 |
|-----------|--------|-----------|------|
| apiKey | string | - | APIキー（必須）|
| voice.style | string | neutral | 声のスタイル |
| voice.language | string | ja-JP | 言語コード |
| voice.speed | number | 1.0 | 話速 |

## サービスデプロイ

### Docker

```bash
docker pull fantasytalking/server:latest
docker run -d -p 8080:8080 -e API_KEY=your-api-key fantasytalking/server:latest
```
