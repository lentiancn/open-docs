# Angular

Angular — это открытый фреймворк для веб-приложений, разработанный и поддерживаемый компанией Google. Он используется для создания современных веб-приложений и применяет TypeScript как основной язык разработки. Angular предоставляет мощные возможности: компонентная архитектура, двустороннее связывание данных, внедрение зависимостей и многое другое.

## Быстрый старт

### Установка Angular CLI

```bash
npm install -g @angular/cli
```

### Создание нового проекта

```bash
ng new my-app
cd my-app
```

### Запуск сервера разработки

```bash
ng serve
```

Откройте в браузере http://localhost:4200

## Основные возможности

- **Компонентная архитектура** — модульные, переиспользуемые UI-компоненты
- **TypeScript** — строгая типизация для качественного кода
- **Внедрение зависимостей** — слабая связанность, удобное тестирование
- **Двустороннее связывание данных** — автоматическая синхронизация
- **Маршрутизация** — полноценное решение для навигации
- **HTTP-клиент** — мощные возможности для сетевых запросов
- **Обработка форм** — реактивные формы и формы на основе шаблонов
- **RxJS** — поддержка реактивного программирования

## Часто используемые команды

| Команда | Описание |
|---------|----------|
| ng new xxx | Создать новый проект |
| ng serve | Запустить сервер разработки |
| ng build | Собрать для продакшена |
| ng generate component xxx | Создать компонент |
| ng generate service xxx | Создать сервис |
| ng generate module xxx | Создать модуль |
