# V-Express Guide d'installation

V-Express est une méthode de génération de vidéos de portraits développée par Tencent AI Lab.

## Configuration requise

### Matériel

| Composant | Minimum | Recommandé |
|-----------|----------|---------------|
| GPU | NVIDIA 16GB VRAM | NVIDIA 24GB+ VRAM |

## Installation

```bash
git clone https://github.com/tencent-ailab/V-Express.git
cd V-Express
pip install -r requirements.txt
```
