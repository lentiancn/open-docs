# Claude Code Einführung

Claude Code ist ein intelligenter Programmierassistent von Anthropic, der Codebasen lesen, Dateien bearbeiten, Befehle ausführen und sich nahtlos in Entwicklungstools integrieren kann. Er ist in Terminal, IDE, Desktop-App und Browser verfügbar.

## Was ist Claude Code?

Claude Code ist ein KI-gesteuerter Programmierassistent, der Ihnen hilft, Funktionen zu entwickeln, Fehler zu beheben und Entwicklungsaufgaben zu automatisieren. Er versteht Ihre gesamte Codebasis und kann über mehrere Dateien und Tools hinweg arbeiten.

## Hauptfunktionen

Claude Code bietet die folgenden Kernfunktionen:

**Entwicklungsaufgaben automatisieren**
- Tests für nicht getesteten Code schreiben
- Lint-Fehler im gesamten Projekt beheben
- Merge-Konflikte lösen
- Abhängigkeiten aktualisieren
- Release-Notes schreiben

**Funktionsentwicklung und Fehlerbehebung**
- Beschreiben Sie, was Sie in natürlicher Sprache erreichen möchten
- Claude Code plant den Ansatz, schreibt Code über mehrere Dateien und verifiziert die Implementierung

**Git-Integration**
- Führen Sie Git-Operationen direkt in Claude Code durch
- Änderungen bereitstellen, Commit-Nachrichten schreiben, Branches erstellen, Pull Requests öffnen

**MCP-Integration**
- Model Context Protocol (MCP) ist ein offener Standard zum Verbinden von KI-Tools mit externen Datenquellen
- Mit MCP kann Claude Code Designdokumente aus Google Drive lesen, Jira-Tickets aktualisieren, Daten aus Slack abrufen oder eigene Tools verwenden

**Anpassungsmöglichkeiten**
- Verwenden Sie CLAUDE.md-Dateien, um Codierungsstandards, Architekturentscheidungen, bevorzugte Bibliotheken und Review-Checklisten festzulegen
- Erstellen Sie benutzerdefinierte Befehle für wiederholbare Workflows
- Verwenden Sie Hooks für automatisches Formatieren nach Dateiänderungen oder Lint vor Commits

**Multi-Agenten-Zusammenarbeit**
- Starten Sie mehrere Claude Code-Agenten für parallele Arbeit an komplexen Aufgaben
- Ein Lead-Agent koordiniert die Arbeit, weist Unteraufgaben zu und führt Ergebnisse zusammen

**CLI-Flexibilität**
- Unterstützung für Piping, Skripting und Automatisierung
- Pipen Sie Logs, führen Sie es in CI aus oder kombinieren Sie es mit anderen Tools

## Nutzungsszenarien

Claude Code kann in verschiedenen Szenarien verwendet werden:

| Szenario | Empfohlene Methode |
|----------|---------------------|
| Lokale Entwicklung | Terminal-CLI |
| IDE-Codierung | VS Code oder JetBrains |
| Geräteübergreifend weiterarbeiten | Fernsteuerung |
| Mobile Operationen | Web oder iOS-App |
| CI/CD-Automatisierung | GitHub Actions oder GitLab CI/CD |
| Team-Zusammenarbeit | Slack-Integration |
| Browser-Debugging | Chrome-Erweiterung |

## Kontoanforderungen

Claude Code erfordert eines der folgenden Konten:
- Claude Pro, Max, Teams oder- Anthropic Console Enterprise Abonnement
-Konto
- Unterstützte Drittanbieter-Cloud-Anbieter (Amazon Bedrock, Google Vertex AI, Microsoft Foundry)

Der kostenlose Claude.ai-Plan beinhaltet keinen Claude Code-Zugang.

## Nächste Schritte

Nach der Installation von Claude Code können Sie sich weiter informieren:

- **Schnellstart**: Lernen Sie Schritt für Schritt von der Codebasis-Erkundung bis zum Committen von Fixes
- **Anweisungen und Erinnerungen speichern**: Verwenden Sie CLAUDE.md-Dateien für persistente Anweisungen an Claude
- **Allgemeine Workflows und Best Practices**: Muster für die optimale Nutzung von Claude Code
- **Benutzerdefinierte Einstellungen**: Passen Sie Claude Code für Ihren Workflow an
- **Fehlerbehebung**: Lösen Sie häufige Probleme

Weitere Informationen finden Sie unter [code.claude.com](https://code.claude.com/).
