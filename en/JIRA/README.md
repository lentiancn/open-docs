# Jira

Jira is project and issue tracking software developed by Atlassian, providing project management solutions for over 300,000 companies worldwide.

## Quick Start

Visit the Atlassian website to sign up and choose a plan suitable for your team. You can quickly create a Jira site.

## Key Features

Project management and issue tracking, Agile development support (Scrum, Kanban), Team collaboration and communication, Rovo AI assistant integration, Rich third-party integrations, Powerful automation rules.

## Pricing

Free: 10 users permanently free

Standard: $7.91 per user per month

Premium: $14.54 per user per month

Enterprise: Contact sales for pricing
