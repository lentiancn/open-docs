# Vue Руководство по установке

Vue — прогрессивный JavaScript-фреймворк для создания пользовательских интерфейсов.

## Версия

| Версия | Дата выпуска | Статус |
|---------|---------------|--------|
| Vue 3 | Сентябрь 2020 | Рекомендуется |
| Vue 2 | Октябрь 2016 | Устарело |

**Рекомендация:** Используйте Vue 3 для всех новых проектов.

## Предварительные требования

| Требование | Vue 2 | Vue 3 |
|------------|--------|-------|
| Node.js | 8.9+ | 16+ (18+ рекомендуется) |
| npm | 5.6+ | 8+ |

## Способы установки

### Способ 1: npm (Рекомендуется)

```bash
Создать новый проект Vue
npm create vue@latest
```

### Способ 2: Vite (Рекомендуется для Vue 3)

```bash
npm create vite@latest my-vue-app -- --template vue
cd my-vue-app
npm install
npm run dev
```

### Способ 3: CDN

```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
```

## Настройка проекта

```bash
npm install vue-router@4
npm install pinia
```

## Сервер разработки

```bash
Запустить сервер
npm run dev

Собрать для продакшена
npm run build
```

## Решение проблем

### Ошибка npm install

```bash
npm cache clean --force
rm -rf node_modules
npm install
```

## Рекомендации

1. **Используйте Vue 3** — Для новых проектов
2. **Используйте Vite** — Более быстрая разработка
3. **Используйте Composition API** — Лучшая поддержка TypeScript
