# Guía de instalación de OpenClaw

Esta guía te lleva paso a paso através del proceso de instalación en diferentes sistemas operativos.

## Requisitos

- **Node.js**: versión 22 o superior
- **Sistema operativo**: macOS, Linux o Windows
- **Opcional**: Docker (para funciones de aislamiento tipo sandbox)

## Métodos de instalación

### macOS y Linux (Recomendado)

El método más fácil es usar el script de instalación oficial:

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

Este script detecta automáticamente tu entorno de sistema, instala las dependencias necesarias y configura los ajustes básicos.

### Windows

Ejecuta esto en PowerShell:

```powershell
iwr -useb https://openclaw.ai/install.ps1 | iex
```

### Instalación directa con npm

Si ya tienes Node.js configurado:

```bash
npm install -g openclaw@latest
```

### Desde el código fuente

Para usar la última versión en desarrollo o instalaciones personalizadas:

```bash
git clone https://github.com/openclaw/openclaw.git
cd openclaw
npm install
npm link
```

## Configuración después de instalar

### Ejecutar el asistente de configuración

Después de la instalación, es recomendable ejecutar el asistente para completar la configuración básica:

```bash
openclaw onboard --install-daemon
```

El asistente te guiará a través de:
- Configuración de la clave API
- Configuración del puerto del servicio
- Inicio de sesión en canales (WhatsApp, Telegram, Discord, etc.)
- Configuración de seguridad y control de acceso

### Verificar el estado del servicio

Si seguiste los pasos de instalación, el servicio debería estar ejecutándose:

```bash
openclaw gateway status
```

### Ejecutar en primer plano

Para ver los registros directamente en tu terminal (útil para depuración):

```bash
openclaw gateway --port 18789
```

### Abrir la interfaz web

```bash
openclaw dashboard
```

O simplemente abre en tu navegador: `http://127.0.0.1:18789/`

## Variables de entorno

En algunos casos necesitas personalizar las ubicaciones de almacenamiento de datos:

- `OPENCLAW_HOME`: Especifica el directorio principal para OpenClaw
- `OPENCLAW_STATE_DIR`: Sobreescribe el directorio de datos de estado
- `OPENCLAW_CONFIG_PATH`: Especifica una ruta personalizada para el archivo de configuración

## Verificar la instalación

Después de instalar, puedes:

1. Ejecutar `openclaw --version` para confirmar que funciona
2. Ejecutar `openclaw doctor` para verificar el estado del sistema
3. Abrir la interfaz web y enviar un mensaje de prueba

## Problemas comunes

### Verificar la versión de Node.js

```bash
node --version
```

Asegúrate de que muestre la versión 22 o superior. Si es menor, actualiza Node.js primero.

### Git no encontrado en Windows

Asegúrate de que Git esté instalado y en tu PATH del sistema. Puedes descargarlo desde git-scm.com.

### Errores de permisos

Si tienes problemas de permisos, intenta:

```bash
sudo npm install -g openclaw@latest
```
