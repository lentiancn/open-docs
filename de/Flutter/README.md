# Flutter

Flutter - Plattformübergreifendes UI-Framework

## Überblick

Flutter ist ein von Google entwickeltes Open-Source-UI-Softwareentwicklungskit, das verwendet wird, um schöne, nativ kompilierte mobile, Web- und Desktop-Anwendungen aus einer einzigen Codebasis zu erstellen.

## Funktionen

- **Plattformübergreifend**: Eine Codebasis für iOS, Android, Web, Windows, macOS, Linux
- **Heißes Nachladen**: Schnelle Entwicklung, behält Anwendungszustand bei
- **Reichhaltige Komponenten**: Material Design und Cupertino Komponentenbibliotheken
- **Native Leistung**: Kompiliert zu nativem ARM-Code
- **Deklaratives UI**: Modernes UI-Entwicklungsparadigma
- **Open Source**: BSD-Lizenz

## Schnellstart

### Installation

```bash
# Flutter klonen
git clone https://github.com/flutter/flutter.git -b stable

# Zu PATH hinzufügen
export PATH="$PATH:$HOME/flutter/bin"

# Installation überprüfen
flutter doctor
```

### Projekt erstellen

```bash
flutter create my_app
cd my_app
flutter run
```
