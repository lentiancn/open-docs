# Angular

Angular는 Google이 개발한 웹 애플리케이션 구축용 강력한 프론트엔드 프레임워크입니다.

## 소개

Angular는 양방향 데이터 바인딩, 의존성 주입, 강력한 CLI 도구 등 최신 웹 애플리케이션 구축을 위한 완전한 솔루션을 제공하는 TypeScript 기반 프레임워크입니다.

## 기능

- **컴포넌트 기반**: 재사용 가능한 UI 컴포넌트 구축
- **TypeScript**: 타입 안전 개발
- **CLI**: 강력한命令行 도구
- **라우팅**: 내장 라우팅 시스템
- **폼**: 템플릿 기반 및 반응형 폼
- **HTTP**: API 호출용 HTTP 클라이언트
- **테스트**: 내장 테스트 지원

## 빠른 시작

### 설치

```bash
npm install -g @angular/cli
```

### 프로젝트 생성

```bash
ng new my-app
cd my-app
ng serve
```

http://localhost:4200 열기
