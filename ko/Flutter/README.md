# Flutter

Flutter - 크로스플랫폼 UI 프레임워크

## 개요

Flutter는 단일 코드베이스에서 아름답게 네이티브로 컴파일된 모바일, 웹 및 데스크톱 애플리케이션을 구축하는 데 사용되는 Google이 개발한 오픈소스 UI 소프트웨어 개발 키트입니다.

## 기능

- **크로스플랫폼**: iOS, Android, 웹, Windows, macOS, Linux를 위한 단일 코드베이스
- **핫 리로드**: 빠른 개발 반복, 애플리케이션 상태 유지
- **풍부한 컴포넌트**: Material Design 및 Cupertino 컴포넌트 라이브러리
- **네이티브 성능**: 네이티브 ARM 코드로 컴파일
- **선언적 UI**: 현대적인 UI 개발 패러다임
- **오픈소스**: BSD 라이선스

## 빠른 시작

### 설치

```bash
# Flutter 클론
git clone https://github.com/flutter/flutter.git -b stable

# PATH에 추가
export PATH="$PATH:$HOME/flutter/bin"

# 설치 확인
flutter doctor
```

### 프로젝트 생성

```bash
flutter create my_app
cd my_app
flutter run
```
