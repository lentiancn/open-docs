# Element Plus

Biblioteca de componentes UI para Vue 3.

## Resumen

Element Plus es una biblioteca de componentes Vue 3 desarrollada por Ele.me, proporcionando 70+ componentes de alta calidad.

### Características

- **Vue 3**: Composition API
- **TypeScript**: Soporte completo
- **Tree Shaking**: Optimización
- **Temas**: Variables CSS

### Estadísticas

- GitHub Stars: 20,000+

## Inicio Rápido

### Instalación

```bash
npm install element-plus
```

### Uso

```vue
<el-button type="primary">Botón</el-button>
```
