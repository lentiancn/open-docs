# Angular FAQ

## Problemas de instalación

### Q: ¿Angular CLI falla?

**Soluciones:**
1. Verificar versión de Node.js
```bash
node --version
```
2. Limpiar caché npm
```bash
npm cache clean --force
```

## Problemas de desarrollo

### Q: ¿Datos no se actualizan?

**Soluciones:**
Para Signals:
```typescript
count.set(1);
count.update(c => c + 1);
```

## Rendimiento

### Q: ¿Carga lenta?

**Soluciones:**
1. Usar lazy loading
2. Modo producción
```bash
ng build --configuration=production
```
