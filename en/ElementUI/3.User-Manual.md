# Element UI User Manual

## Button

```html
<el-button>Default</el-button>
<el-button type="primary">Primary</el-button>
<el-button type="success">Success</el-button>
<el-button type="warning">Warning</el-button>
<el-button type="danger">Danger</el-button>
```

## Input

```html
<el-input v-model="input" placeholder="Please input"></el-input>
```

## Select

```html
<el-select v-model="value" placeholder="Please select">
  <el-option
    v-for="item in options"
    :key="item.value"
    :label="item.label"
    :value="item.value">
  </el-option>
</el-select>
```

## Table

```html
<el-table :data="tableData" stripe border>
  <el-table-column prop="date" label="Date" width="180"></el-table-column>
  <el-table-column prop="name" label="Name" width="180"></el-table-column>
  <el-table-column prop="address" label="Address"></el-table-column>
</el-table>
```

## Form

```html
<el-form :model="form" label-width="80px">
  <el-form-item label="Name">
    <el-input v-model="form.name"></el-input>
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="onSubmit">Submit</el-button>
  </el-form-item>
</el-form>
```

## Dialog

```html
<el-button type="text" @click="dialogVisible = true">Click to open Dialog</el-button>

<el-dialog
  title="Tips"
  :visible.sync="dialogVisible"
  width="30%">
  <span>This is a message</span>
  <span slot="footer" class="dialog-footer">
    <el-button @click="dialogVisible = false">Cancel</el-button>
    <el-button type="primary" @click="dialogVisible = false">Confirm</el-button>
  </span>
</el-dialog>
```

## Message

```javascript
this.$message({
  message: 'Success message',
  type: 'success'
});
```

## Menu

```html
<el-menu :default-active="activeIndex" mode="horizontal">
  <el-menu-item index="1">Center</el-menu-item>
  <el-menu-item index="2">Orders</el-menu-item>
</el-menu>
```

## Grid System

```html
<el-row :gutter="20">
  <el-col :span="12"><div class="grid-content"></div></el-col>
  <el-col :span="12"><div class="grid-content"></div></el-col>
</el-row>
```

## Container Layout

```html
<el-container>
  <el-header>Header</el-header>
  <el-container>
    <el-aside width="200px">Aside</el-aside>
    <el-main>Main</el-main>
  </el-container>
  <el-footer>Footer</el-footer>
</el-container>
```

## Theme Customization

Create custom theme in project:

```javascript
import { Input } from 'element-ui'
import '../theme/input.css'
```

Or use online theme editor.
