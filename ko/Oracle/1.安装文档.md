# Oracle 데이터베이스 설치 가이드

Oracle 데이터베이스는 Oracle에서 개발한 업계 최고 수준의 관계형 데이터베이스 관리 시스템입니다. 이 가이드에서는 다양한 환경에서의 설치 방법을 설명합니다.

## 버전 정보

| 버전 | 상태 | 설명 |
|------|------|------|
| Oracle 21c | ✅ 최신 | 최신 버전 |
| Oracle 19c | ✅ LTS | 장기 지원 버전 |
| Oracle 18c | ❌ 구버전 | 지원 중단 |

> **중요**: 이 가이드는 주로 **Oracle 19c/21c**를 대상으로 합니다.

## 시스템 요구사항

### 하드웨어 요구사항

| 요구사항 | 최소 구성 | 권장 구성 |
|------|----------|----------|
| CPU | 2코어 | 4+코어 |
| 메모리 | 2GB | 8+GB |
| 디스크 | 40GB | 100+GB |

### 지원 운영체제

- **Linux**: Oracle Linux, RHEL, CentOS, SLES, Ubuntu
- **Windows**: Windows Server 2016+

## 설치 방법

### 방법 1: Linux 설치

#### 1. 시스템 준비

```bash
# 사용자 그룹 생성
groupadd oinstall
groupadd dba
groupadd oper
groupadd backupdba
groupadd dgdba
groupadd kmdba
groupadd racdba
groupadd asmdba
groupadd asmoper
groupadd asmadmin

# Oracle 사용자 생성
useradd -g oinstall -G dba,oper,backupdba,dgdba,kmdba,racdba,asmdba,asmoper,asmadmin oracle

# 비밀번호 설정
passwd oracle

# 디렉토리 생성
mkdir -p /u01/app/oracle/product/19.3.0/dbhome_1
mkdir -p /u01/app/oraInventory
mkdir -p /u01/app/oracle

# 권한 설정
chown -R oracle:oinstall /u01
chmod -R 775 /u01
```

#### 2. 커널 매개변수 설정

```bash
# /etc/sysctl.conf
cat >> /etc/sysctl.conf << EOF
kernel.shmmax = 68719476736
kernel.shmmni = 4096
kernel.shmall = 2097152
kernel.sem = 250 32000 100 128
fs.file-max = 6815744
net.ipv4.ip_local_port_range = 9000 65500
net.core.rmem_default = 262144
net.core.rmem_max = 4194304
net.core.wmem_default = 262144
net.core.wmem_max = 1048576
fs.aio-max-nr = 1048576
EOF

# 적용
sysctl -p

# /etc/security/limits.conf
cat >> /etc/security/limits.conf << EOF
oracle soft nproc 2047
oracle hard nproc 16384
oracle soft nofile 1024
oracle hard nofile 65536
oracle soft stack 10240
oracle hard stack 32768
EOF
```

#### 3. 의존성 설치

```bash
# Oracle Linux
yum install -y libaio libaio-devel libXi libXtst libXext

# Ubuntu
apt-get install -y libaio1 libaio-dev libxi6 libxtst6 libxext6
```

#### 4. 인스톨러 실행

```bash
# 설치 파일 압축 해제
unzip -q LINUX.X64_193000_db_home.zip -d /u01/app/oracle/product/19.3.0/dbhome_1/

# oracle 사용자로 실행
cd /u01/app/oracle/product/19.3.0/dbhome_1
./runInstaller
```

### 방법 2: Windows 설치

1. Oracle 인스톨러 다운로드
2. setup.exe 실행
3. 설치 마법사 따르기:
   - "단일 인스턴스 데이터베이스 생성 및 구성" 선택
   - "데스크톱 클래스" 또는 "서버 클래스" 선택
   - sys 및 system 비밀번호 설정
   - 설치 완료

### 방법 3: Docker 사용

```bash
# 이미지 가져오기
docker pull container-registry.oracle.com/database/express:21.3.0-xe

# 컨테이너 실행
docker run -d \
  --name oracle \
  -p 1521:1521 \
  -e ORACLE_PWD=MyPassword123 \
  -v oracle_data:/opt/oracle/oradata \
  container-registry.oracle.com/database/express:21.3.0-xe
```

## 설치 후 구성

### 환경 변수

```bash
# ~/.bash_profile
export ORACLE_HOME=/u01/app/oracle/product/19.3.0/dbhome_1
export ORACLE_SID=orcl
export PATH=$ORACLE_HOME/bin:$PATH
export LD_LIBRARY_PATH=$ORACLE_HOME/lib:$LD_LIBRARY_PATH
```

### 데이터베이스 시작

```bash
# 리스너 시작
lsnrctl start

# 데이터베이스 시작
sqlplus / as sysdba
STARTUP;
```
