# Element UI 使用指南

本指南介紹如何在 Vue 2 應用程式中使用 Element UI 元件。

## 基本用法

### Hello World

```vue
<template>
  <div>
    <el-button>預設按鈕</el-button>
    <el-button type="primary">主要按鈕</el-button>
  </div>
</template>

<script>
export default {
  name: 'App'
}
</script>
```

## 元件分類

Element UI 提供以下元件分類：

| 分類 | 元件 |
|------|------|
| 基礎 | Layout, Button, Icon, Link |
| 表單 | Input, Select, Radio, Checkbox, Switch, Slider |
| 資料 | Table, Tree, Pagination, Tag |
| 通知 | Alert, Message, Notification, Progress |
| 導航 | Menu, Tabs, Breadcrumb, Steps, Dropdown |
| 其他 | Dialog, Drawer, Card, Collapse |

## 常用元件

```vue

    <!-- 類型 -->

    <el-button circle icon="el-icon-search"></el-button>

    <el-button disabled>停用</el-button>
```

### 輸入框 Input

```vue
<template>
  <div>
    <!-- 基礎 -->
    <el-input v-model="input" placeholder="輸入文字"></el-input>
    
    <!-- 帶圖示 -->
    <el-input
      v-model="input1"
      placeholder="搜尋"
      prefix-icon="el-icon-search">
    </el-input>
    
    <!-- 密碼 -->
    <el-input
      v-model="input2"
      placeholder="密碼"
      show-password>
    </el-input>
    
    <!-- 文字域 -->
    <el-input
      v-model="textarea"
      type="textarea"
      :rows="3"
      placeholder="輸入描述">
    </el-input>
  </div>
</template>

<script>
export default {
  data() {
    return {
      input: '',
      input1: '',
      input2: '',
      textarea: ''
    }
  }
}
</script>
```

### 表格 Table

```vue
<template>
  <el-table :data="tableData" stripe border>
    <el-table-column prop="date" label="日期" width="180"></el-table-column>
    <el-table-column prop="name" label="姓名" width="180"></el-table-column>
    <el-table-column prop="address" label="地址"></el-table-column>
    <el-table-column label="操作">
      <template slot-scope="scope">
        <el-button
          size="mini"
          @click="handleEdit(scope.$index, scope.row)">編輯</el-button>
        <el-button
          size="mini"
          type="danger"
          @click="handleDelete(scope.$index, scope.row)">刪除</el-button>
      </template>
    </el-table-column>
  </el-table>
</template>

<script>
export default {
  data() {
    return {
      tableData: [{
        date: '2016-05-02',
        name: '張三',
        address: '北京市朝陽區'
      }]
    }
  },
  methods: {
    handleEdit(index, row) {
      console.log(index, row);
    },
    handleDelete(index, row) {
      console.log(index, row);
    }
  }
}
</script>
```

### 表單 Form

```vue
<template>
  <el-form
    :model="ruleForm"
    :rules="rules"
    ref="ruleForm"
    label-width="120px"
  >
    <el-form-item label="姓名" prop="name">
      <el-input v-model="ruleForm.name"></el-input>
    </el-form-item>
    
    <el-form-item label="郵箱" prop="email">
      <el-input v-model="ruleForm.email"></el-input>
    </el-form-item>
    
    <el-form-item label="性別" prop="gender">
      <el-radio-group v-model="ruleForm.gender">
        <el-radio label="male">男</el-radio>
        <el-radio label="female">女</el-radio>
      </el-radio-group>
    </el-form-item>
    
    <el-form-item label="愛好" prop="hobby">
      <el-checkbox-group v-model="ruleForm.hobby">
        <el-checkbox label="閱讀" name="hobby"></el-checkbox>
        <el-checkbox label="運動" name="hobby"></el-checkbox>
        <el-checkbox label="音樂" name="hobby"></el-checkbox>
      </el-checkbox-group>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm('ruleForm')">提交</el-button>
      <el-button @click="resetForm('ruleForm')">重設</el-button>
    </el-form-item>
  </el-form>
</template>

<script>
export default {
  data() {
    return {
      ruleForm: {
        name: '',
        email: '',
        gender: '',
        hobby: []
      },
      rules: {
        name: [
          { required: true, message: '請輸入姓名', trigger: 'blur' }
        ],
        email: [
          { required: true, message: '請輸入郵箱', trigger: 'blur' },
          { type: 'email', message: '郵箱格式不正確', trigger: 'blur' }
        ]
      }
    };
  },
  methods: {
    submitForm(formName) {
      this.$refs[formName].validate((valid) => {
        if (valid) {
          alert('提交成功!');
        }
      });
    },
    resetForm(formName) {
      this.$refs[formName].resetFields();
    }
  }
}
</script>
```

```vue
    </el-dialog>
  </div>

<script>
export default {
  data() {
    return {
      dialogVisible: false
    }
  }
}
</script>
```

## 事件

```vue
<template>
  <el-button @click="handleClick">點擊我</el-button>
  
  <el-input
    @focus="handleFocus"
    @blur="handleBlur"
    @change="handleChange">
  </el-input>
</template>

<script>
export default {
  methods: {
    handleClick() {
      console.log('點擊!')
    },
    handleFocus() {
      console.log('獲得焦點!')
    },
    handleBlur() {
      console.log('失去焦點!')
    },
    handleChange(value) {
      console.log('改變:', value)
    }
  }
}
</script>
```

## 最佳實踐

### 1. 元件局部引入

```javascript
export default {
  components: {
    [Button.name]: Button,
    [Input.name]: Input
  }
}
```

### 2. 表單驗證

```javascript
const validatePass = (rule, value, callback) => {
  if (value === '') {
    callback(new Error('請輸入密碼'));
  } else {
    callback();
  }
};
```
