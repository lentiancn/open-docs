# Bootstrap Benutzerhandbuch

> Hauptkomponenten und Verwendung

---

## Grid-System

### Grundstruktur

```html
<div class="container">
  <div class="row">
    <div class="col">Spalte 1</div>
    <div class="col">Spalte 2</div>
    <div class="col">Spalte 3</div>
  </div>
</div>
```

### Responsives Spalten

```html
<!-- Voll auf Mobile, 6 Spalten auf mittleren Bildschirmen -->
<div class="col-12 col-md-6">Inhalt</div>

<!-- 4 Spalten auf großen Bildschirmen -->
<div class="col-lg-4">Inhalt</div>
```

## Buttons

### Grundlegende Buttons

```html
<button class="btn btn-primary">Primär</button>
<button class="btn btn-secondary">Sekundär</button>
<button class="btn btn-success">Erfolg</button>
<button class="btn btn-danger">Gefahr</button>

<button class="btn btn-info">Info</button>
<button class="btn btn-light">Hell</button>
```

### Button-Größen

```html
<button class="btn btn-primary btn-lg">Groß</button>
<button class="btn btn-primary">Standard</button>
<button class="btn btn-primary btn-sm">Klein</button>
```

## Formulare

### Grundlegendes Formular

```html
<form>
  <div class="mb-3">
    <label class="form-label">E-Mail</label>
    <input type="email" class="form-control" placeholder="name@beispiel.de">
  </div>
  <div class="mb-3">
    <label class="form-label">Passwort</label>
    <input type="password" class="form-control">
  </div>
  <button type="submit" class="btn btn-primary">Absenden</button>
</form>
```

### Auswahlfeld

```html
<select class="form-select">
  <option selected>Auswählen...</option>
  <option value="1">Option 1</option>
  <option value="2">Option 2</option>
</select>
```

## Navigation

### Navigationsleiste

```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">Marke</a>
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav">
      <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="navbarNav">
      <ul class="navbar-nav">
        <li class="nav-item">
          <a class="nav-link active" href="#">Startseite</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Über</a>
        </li>
      </ul>
    </div>
  </div>
</nav>
```

## Modals

```html
<!-- Auslöser-Button -->
<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#exampleModal">
  Modal öffnen
</button>

<!-- Modal -->
<div class="modal fade" id="exampleModal" tabindex="-1">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title">Titel</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
      </div>
      <div class="modal-body">
        Inhalt
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Schließen</button>
        <button type="button" class="btn btn-primary">Speichern</button>
      </div>
    </div>
  </div>
</div>
```
