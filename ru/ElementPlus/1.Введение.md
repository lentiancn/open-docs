# Введение в Element Plus

Element Plus — это библиотека компонентов пользовательского интерфейса на Vue 3.

## Особенности

- 70+ компонентов
- Полная поддержка TypeScript
- Импорт по требованию
- Интернационализация
- Настраиваемые темы

## Начало работы

```bash
npm install element-plus
```

Подробнее на [официальном сайте](https://element-plus.org/).
