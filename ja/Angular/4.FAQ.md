# Angular よくある質問

よくある質問と回答。

## インストール

### Q: Angular CLI インストール失敗？

**解決策**：
1. npm キャッシュをクリア：`npm cache clean --force`
2. 管理者権限でインストール
3. Node.js バージョンを確認

### Q: ng コマンドが見つからない？

**解決策**：
1. Angular CLI がグローバルインストール確認
2. PATH 環境変数を確認
3. 再インストールを試す

### Q: Node.js バージョン互換性なし？

**解決策**：
1. nvm を使用して Node.js バージョン管理
2. 推奨バージョンにアップグレード

## 使用

### Q: コンポーネントスタイルが効かない？

**解決策**：
1. styleUrls パスが正しいか確認
2. CSS セレクターが正しいか確認
3. ViewEncapsulation 設定を確認

### Q: ルーティング失敗？

**解決策**：
1. ルーター設定が正しいか確認
2. RouterModule がインポート確認
3. routerLink 構文を確認

### Q: HTTPリクエスト失敗？

**解決策**：
1. HttpClientModule がインポート確認
2. CORS 設定を確認
3. API URL が正しいか確認

### Q: フォーム検証が効かない？

**解決策**：
1. ReactiveFormsModule がインポート確認
2. Validators 設定を確認
3. formControlName を正しく使用

## パフォーマンス

### Q: アプリ読み込み遅い？

**解決策**：
1. レイジーローディングを有効化
2. OnPush 変更検出を使用
3. 画像とリソースを圧縮
4. プロダクションモードでビルド

### Q: メモリリーク？

**解決策**：
1. オブザーバブルを unsubscribe
2. タイマーをクリア
3. イベントリスナーを破棄

## バージョン

### Q: Angular をアップグレード方法は？

```bash
ng update @angular/core @angular/cli
```

### Q: AngularJS から Angular への移行？

angular-upgrade ツールを使用して段階的に移行。
