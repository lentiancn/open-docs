# JMeter

JMeter — это инструмент для нагрузочного тестирования с открытым исходным кодом, разработанный организацией Apache.

## Введение

JMeter используется для нагрузочного тестирования программного обеспечения и систем, поддерживает различные протоколы, такие как Web, базы данных, FTP и другие.

## Основные функции

- **Несколько протоколов**: HTTP, HTTPS, JDBC, FTP и другие
- **Параметризация**: Тесты на основе данных CSV
- **Распределенное тестирование**: Совместная работа нескольких машин
- **Богатые отчеты**: Агрегированные отчеты, графические результаты
- **Расширяемость**: Система плагинов

## Быстрый старт

### Установка

```bash
# Загрузить и распаковать
tar -xzf apache-jmeter-5.6.3.tgz

# Запустить
./apache-jmeter-5.6.3/bin/jmeter
```

### Использование

```bash
# Запуск из командной строки
jmeter -n -t test.jmx -l result.jtl
```
