# Go 설치 가이드

> 전체 환경 설정

---

## 요구 사항

| 환경 | 요구사항 |
|------|---------|
| OS | Linux, macOS, Windows |
| 아키텍처 | amd64, arm64, 386 |

---

## 설치

### Windows

```powershell
choco install golang
```

### macOS

```bash
brew install go
```

### Linux

```bash
sudo apt install golang-go
```

---

## 설정

### 프록시 (중국)

```bash
go env -w GOPROXY=https://goproxy.cn,direct
```

### 확인

```bash
go version
```
