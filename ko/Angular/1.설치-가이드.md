## 시스템 요구사항

### 하드웨어
- **프로세서**: 득얼 코어 이상
- **RAM**: 최소 4GB (8GB 권장)
- **디스크 공간**: 최소 3GB 여유 공간

### 소프트웨어

#### Windows
- Windows 10 이상
- Node.js 18.19 이상 (LTS 권장)
- npm 9.x 이상
- Git

#### macOS
- macOS 10.15 이상
- Node.js 18.19 이상
- npm 9.x 이상
- Git

#### Linux
- Ubuntu 20.04 LTS 이상
- Node.js 18.19 이상
- npm 9.x 이상
- Git

## 전제 조건

### 1. Node.js 설치

#### Windows
https://nodejs.org/에서 다운로드

#### macOS
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install node@18
```

#### Linux
```bash
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs
```

### 2. Git 설치

```bash
# Ubuntu/Debian
sudo apt install git

# macOS
brew install git
```

---

## Angular CLI 설치

```bash
npm install -g @angular/cli
```

### 확인

```bash
ng version
```

---

## Windows에 설치

1. https://nodejs.org/에서 Node.js 설치
2. https://git-scm.com/에서 Git 설치
3. Angular CLI 설치: `npm install -g @angular/cli`
4. 프로젝트 생성:
   ```bash
   ng new my-first-app
   cd my-first-app
   ng serve
   ```

---

## macOS에 설치

1. Homebrew 설치
2. Node.js 설치: `brew install node@18`
3. Git 설치: `brew install git`
4. Angular CLI 설치: `npm install -g @angular/cli`

---

## Linux에 설치

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install curl build-essential git
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs
sudo npm install -g @angular/cli
```

## 문제 해결

### npm 권한 오류
```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
```

### 포트 4200 사용 중
```bash
ng serve --port 4201
```

### 버전 호환성

| Angular | Node.js |
|---------|---------|
| Angular 17 | 18.x, 20.x |
| Angular 16 | 16.x, 18.x |
