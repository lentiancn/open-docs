# MySQL 簡介

MySQL 是世界上最流行的開源關係型資料庫管理系統（RDBMS）。它廣泛用於 Web 應用程式，是 LAMP 堆疊（Linux、Apache、MySQL、PHP/Python/Perl）的核心組件。

## 什麼是 MySQL？

MySQL 是一個強大、快速且可靠的資料庫系統，用於儲存和管理應用程式的資料。它使用 SQL（結構化查詢語言）作為查詢語言。

### 主要功能

- **開源** - 免費使用 GPL 授權
- **高效能** - 針對速度和效率進行優化
- **可擴展性** - 輕鬆處理大型資料庫
- **安全性** - 強大的使用者管理和加密
- **複寫** - 支援主從和主主架構
- **儲存引擎** - InnoDB、MyISAM、Memory 等
- **跨平台** - 執行於 Windows、Linux、macOS 等
