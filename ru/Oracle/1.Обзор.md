# Oracle База Данных

> Корпоративная система управления реляционными базами данных

---

## Обзор

Oracle Database — это собственная система управления базами данных, разработанная и продвигаемая Oracle Corporation. Это одна из наиболее широко используемых корпоративных систем баз данных в мире.

---

## Основные функции

- **Высокая производительность**: Оптимизированная обработка запросов
- **Масштабируемость**: Поддержка больших объёмов данных
- **Безопасность**: Расширенные функции безопасности
- **Высокая доступность**: Real Application Clusters (RAC)
- **PL/SQL**: Процедурное расширение языка
