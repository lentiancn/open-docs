# MySQL 使用ガイド

MySQL の基本的な操作。

---

## 接続

```bash
mysql -u root -p
```

## 基本的なコマンド

```sql
SHOW DATABASES;
CREATE DATABASE mydb;
USE mydb;
SHOW TABLES;
```

## テーブル作成

```sql
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(255)
);
```

## CRUD

```sql
INSERT INTO users (name) VALUES ('John');
SELECT * FROM users;
UPDATE users SET name='Jane' WHERE id=1;
DELETE FROM users WHERE id=1;
```

## バックアップ

```bash
mysqldump -u root -p mydb > mydb.sql
```
