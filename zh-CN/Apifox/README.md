# Apifox 技术文档

## 文档简介

本目录收录了 Apifox 的相关技术文档，旨在帮助开发者快速上手并熟练使用 Apifox 进行 API 开发、测试和文档管理。

Apifox 是一款功能强大的 API 协作平台，它将 API 设计、Mock、测试、文档等功能整合在一起，为开发者提供了一站式的 API 开发体验。无论你是前端开发、后端开发还是测试工程师，Apifox 都能帮助你更高效地完成 API 相关的工作。

## 文档目录

本技术文档包含以下内容：

**[1. 简介](1.简介.md)**：介绍 Apifox 是什么、核心功能、适用场景，以及它与其他 API 工具的对比。通过这部分内容，你可以快速了解 Apifox 的定位和价值。

**[2. 安装指南](2.安装指南.md)**：详细的安装教程，包括 Windows、macOS、Linux 三大平台的安装步骤，首次启动的配置，以及常见安装问题的解决方案。

**[3. 使用手册](3.使用手册.md)**：最全面的功能使用教程，覆盖项目管理、接口设计与调试、Mock 功能、接口测试、接口文档、团队协作等各个方面。每个功能都配有详细的操作说明。

**[4. 常见问题](4.常见问题.md)**：汇总了用户在使用过程中经常遇到的问题，按类别进行整理，提供了详细的解答方案。如果遇到问题，可以先查阅这部分内容。

## 快速开始

如果你想快速了解 Apifox，建议按照以下顺序阅读：

1. 首先阅读「简介」，了解 Apifox 是什么、能做什么
2. 按照「安装指南」完成软件安装和初始配置
3. 阅读「使用手册」的基本功能部分，开始你的第一次 API 设计
4. 在使用过程中遇到问题时，查阅「常见问题」
