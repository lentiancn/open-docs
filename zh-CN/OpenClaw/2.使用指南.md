# OpenClaw 使用指南

高效使用 OpenClaw 的完整指南。

---

## 基本命令

### 网关

```bash
# 启动网关
openclaw gateway

# 在特定端口启动
openclaw gateway --port 18789
```

### 渠道

```bash
# 登录渠道
openclaw channels login

# 列出可用渠道
openclaw channels list
```

### 会话

```bash
# 列出活动会话
openclaw sessions list

# 结束会话
openclaw sessions kill <session-id>
```

---

## 渠道设置

### WhatsApp

```bash
# 登录 WhatsApp
openclaw channels login whatsapp

# 用手机扫描二维码
```

### Telegram

```bash
# 登录 Telegram
openclaw channels login telegram

# 输入机器人令牌
```
