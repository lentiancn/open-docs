# Python 安裝指南

> 環境設定

---

## 安裝

### Windows

```powershell
choco install python
```

### macOS

```bash
brew install python@3.12
```

### Linux

```bash
sudo apt install python3 python3-pip
```

---

## 套件

```bash
pip install requests
python -m venv myenv
```

---

## 驗證

```bash
python --version
```
