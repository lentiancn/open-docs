# Vue Guide d'utilisation

## Concepts de base

### Instance Vue

```javascript
import { createApp } from 'vue'

const app = createApp({
  data() {
    return {
      message: 'Bonjour Vue!'
    }
  }
})

app.mount('#app')
```

### Syntaxe de modèle

```html
<!-- Interpolation de texte -->
<h1>{{ message }}</h1>

<!-- Directives -->
<div v-if="show">Contenu</div>
<div v-for="item in items">{{ item }}</div>
<button @click="handleClick">Cliquez</button>
<input v-model="inputValue">
```

## Réactivité

### Vue 3 Composition API (Recommandé)

```javascript
import { ref, reactive, computed } from 'vue'

export default {
  setup() {
    const count = ref(0)
    const state = reactive({ name: 'Vue' })
    
    const doubleCount = computed(() => count.value * 2)
    
    return { count, state, doubleCount }
  }
}
```

## Composants

### Créer un composant

```vue
<template>
  <div>
    <h2>{{ title }}</h2>
    <button @click="emitEvent">Cliquez</button>
  </div>
</template>

<script setup>
import { defineProps, defineEmits } from 'vue'

const props = defineProps({ title: String })
const emit = defineEmits(['click'])
</script>
```

## Rendu conditionnel

```html
<div v-if="show">Contenu</div>
<div v-else>Par défaut</div>
```

## Gestion des événements

```html
<button @click="handleClick">Cliquez</button>
<button @click.stop="handleClick">Stop propagation</button>
```
