# Guide d'installation Nginx

Guide pour installer Nginx sur toutes les plateformes principales.

---

## Linux

### Ubuntu / Debian

```bash
# Mettre à jour les paquets
sudo apt update

# Installer Nginx
sudo apt install nginx

# Démarrer Nginx
sudo systemctl start nginx

# Activer au démarrage
sudo systemctl enable nginx
```

### CentOS / RHEL

```bash
# Installer EPEL
sudo yum install epel-release

# Installer Nginx
sudo yum install nginx

# Démarrer
sudo systemctl start nginx
```

---

## macOS

```bash
brew install nginx
brew services start nginx
```

---

## Windows

1. Télécharger depuis [nginx.org](http://nginx.org/en/download.html)
2. Extraire dans `C:\nginx`
3. Exécuter: `nginx.exe`

---

## Vérification

```bash
nginx -v
nginx -t
curl http://localhost
```
