# PlantUML Installationsanleitung

## Voraussetzungen

- **Java Runtime**: Java 8 oder höher erforderlich
- **Speicher**: Mindestens 512MB RAM
- **Festplatte**: 100MB für Installation

---

## Standalone-Installation

### Windows

1. PlantUML JAR von sourceforge.net herunterladen
2. `plantuml.jar` an einem geeigneten Ort speichern
3. (Optional) Zum PATH hinzufügen

### macOS

Mit Homebrew:

```bash
brew install### Linux

Mit plantuml
```

 Paketmanager:

```bash
# Debian/Ubuntu
sudo apt-get install plantuml
```

---

## IDE-Integration

### VS Code

1. VS Code öffnen
2. Erweiterungen (Strg+Umschalt+X)
3. "PlantUML" suchen
4. Installieren
5. VS Code neu starten

---

## Installation überprüfen

```bash
java -jar plantuml.jar -version
```
