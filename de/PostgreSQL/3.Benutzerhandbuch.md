# PostgreSQL Benutzerhandbuch

## Verbindung

```bash
psql -U postgres
```

## Tabelle erstellen

```sql
CREATE TABLE benutzer (
    id SERIAL PRIMARY KEY,
    name VARCHAR(50),
    email VARCHAR(100) UNIQUE
);
```

```sql
INSERT INTO benutzer (name, email)

```

## Daten abfragen

```sql
SELECT * FROM benutzer;
```
