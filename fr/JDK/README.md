# JDK

Kit de développement Java

## Introduction

JDK (Java Development Kit) est le kit de développement logiciel pour le langage de programmation Java, formant le cœur de tout développement Java. JDK comprend l'environnement d'exécution Java (JRE), le compilateur Java (javac), la machine virtuelle Java (JVM) et de vastes bibliothèques de classes et outils Java.

## Versions principales

- **JDK 25** - Dernière version avec nouvelles fonctionnalités et améliorations de performances
- **JDK 21** - Dernière version de support à long terme (LTS)
- **JDK 17** - Version LTS mature, largement adoptée
- **JDK 11** - Version LTS, marquant la division des fonctionnalités payantes de Java

## Composants principaux

- **javac** - Compilateur Java, compile le code source en bytecode
- **java** - Commande d'exécution de programmes Java
- **jvm** - Machine virtuelle Java, exécute le bytecode
- **jar** - Outil d'archivage Java
- **javadoc** - Outil de génération de documentation API
- **jdb** - Débogueur Java

## Différences entre versions

Les différentes versions principales de JDK ont des différences significatives en installation, utilisation et fonctionnalités :

- **JDK 8** : Dernière version Oracle JDK entièrement gratuite pour usage commercial
- **JDK 11** : A introduit var, nouveau client HTTP, ZGC (expérimental)
- **JDK 17** : Dernière version Oracle JDK gratuite sous NFTC
- **JDK 21** : Threads virtuels, pattern matching, records (stables)
- **JDK 25** : Dernières fonctionnalités, cycle de support plus court
