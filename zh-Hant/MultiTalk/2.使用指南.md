# MultiTalk 使用指南

本指南說明如何使用 MultiTalk 產生多說話人音訊內容。

## 快速開始

### 基本用法

```bash
python inference.py \
  --text "大家好，歡迎使用 MultiTalk。" \
  --speakers speaker1,speaker2 \
  --output result.wav
```

## 輸入需求

### 文字需求

- 語言：多語言支援
- 格式：純文字
- 長度：無嚴格限制

### 說話人設定

在 `speakers.yaml` 中定義說話人：

```yaml
speakers:
  speaker1:
    voice_id: "voice_001"
    language: "zh"
  speaker2:
    voice_id: "voice_002"
    language: "zh"
```

## 進階用法

### 自訂語音 ID

```bash
python inference.py \
  --text "這是一個自訂語音示範。" \
  --voice_ids voice001,voice002 \
  --output output.wav
```

### 調整語速

```bash
python inference.py \
  --text "慢速語音示範。" \
  --speakers speaker1 \
  --speed 0.8 \
  --output slow.wav
```

### 批次處理

```bash
python batch_inference.py \
  --text_file texts.txt \
  --output_dir output/
```

## 設定

### 設定檔

編輯 `config.yaml`:

```yaml
inference:
  sample_rate: 22050
  speed: 1.0
  
model:
  checkpoint: "checkpoints/multitalk.pth"
```

## 最佳實踐

### 1. 使用合適的語音 ID

選擇與說話人特徵相符的語音 ID。

### 2. 語言匹配

確保文字語言與語音語言一致。

### 3. 優化語速

調整語速以獲得自然的輸出效果。

## 故障排除

### 音訊品質差？

嘗試不同的語音 ID。

### 語速問題？

逐步調整語速參數。
