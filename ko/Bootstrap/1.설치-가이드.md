# Bootstrap 5 설치 가이드

이 가이드에서는 다양한 환경에서 Bootstrap 5를 설치하고 설정하는 방법을详细介绍합니다.

## 버전 정보

| 버전 | 상태 | 설명 |
|------|------|------|
| Bootstrap 5.3.x | ✅ 현재 | 최신 버전 |
| Bootstrap 5.2.x | ✅ 지원 | 안정 버전 |
| Bootstrap 4.x | ❌ 지원 중단 | 신규 프로젝트 비추천 |

> **중요**: 이 가이드는 **Bootstrap 5.3.x**를 대상으로 합니다.

## 설치 방법

### 방법 1: CDN (가장 간단)

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
```

### 방법 2: npm

```bash
npm install bootstrap@5.3.3
```

### 방법 3: Vite

```bash
npm create vite@latest my-app
npm install bootstrap@5.3.3
```

```javascript
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

### 방법 4: Webpack

```bash
npm install bootstrap@5.3.3
npm install --save-dev webpack webpack-cli css-loader style-loader
```

### 방법 5: Vue CLI / Nuxt

```bash
npm install -g @vue/cli
vue create my-app
npm install bootstrap@5.3.3
```

### 방법 6: React

```bash
npx create-react-app my-app
npm install bootstrap@5.3.3
```

### 방법 7: 소스 코드 다운로드

```bash
git clone https://github.com/twbs/bootstrap.git
cd bootstrap
npm install
npm run dist
```

## 특수 설정

### RTL (오른쪽에서 왼쪽)

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.rtl.min.css" rel="stylesheet">
```

### Sass로 커스터마이징

```bash
npm install bootstrap@5.3.3
npm install --save-dev sass
```

## 자주 묻는 질문

### Popper란?

Popper은 툴팁과 팝오버의 위치를 지정하는 엔진입니다. bootstrap.bundle.min.js에 포함되어 있습니다.

### jQuery가 필요한가?

**아니요!** Bootstrap 5는 jQuery 의존 없이 순수 JavaScript로 다시 작성되었습니다.

### 아이콘 사용법

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css">
```
