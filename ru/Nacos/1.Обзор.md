# Обзор Nacos

Nacos (Dynamic Naming and Configuration Service) — платформа от Alibaba для обнаружения сервисов и динамического управления конфигурацией.

## Основные функции

- Обнаружение сервисов
- Управление конфигурацией
- Динамический DNS
- Проверка работоспособности

## Установка

```bash
# Автономный режим
./startup.sh -m standalone

# Docker
docker run --name nacos -d -p 8848:8848 nacos/nacos-server:v2.3.0
```
