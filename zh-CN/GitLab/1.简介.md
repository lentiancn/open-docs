# GitLab - 简介

GitLab 是一个基于 Git 的完整 DevOps 平台，提供从代码管理到持续集成、持续部署的完整工具链。由 GitLab Inc. 开发，于 2011 年发布开源版本，是最受欢迎的 Git 仓库管理平台之一。

## GitLab 的核心特性

### 1. Git 仓库管理

GitLab 提供强大的 Git 仓库管理功能：

```bash
# 克隆仓库
git clone git@gitlab.com:username/project.git

# 添加远程
git remote add gitlab git@gitlab.com:username/project.git
```

### 2. Web 界面

直观的 Web 管理界面：
- 代码浏览
- 提交历史
- 分支管理
- Merge Request
- 代码审查

### 3. 问题跟踪

完整的问题跟踪系统：

```markdown
## 问题标题

**描述**：详细描述

**标签**：bug, feature

**里程碑**：v1.0

**指派给**：@developer
```

### 4. CI/CD

内置持续集成和持续部署：

```yaml
# .gitlab-ci.yml
stages:
  - build
  - test
  - deploy

build:
  stage: build
  script:
    - npm install
    - npm run build

test:
  stage: test
  script:
    - npm test

deploy:
  stage: deploy
  script:
    - deploy.sh
  only:
    - master
```

### 5. 容器注册

内置 Docker 容器注册：

```bash
# 登录
docker login registry.gitlab.com

# 推送镜像
docker push registry.gitlab.com/username/project:latest
```

### 6. 安全扫描

内置安全扫描功能：
- 依赖扫描
- 容器扫描
- 代码扫描
- SAST/DAST

## GitLab 的应用场景

### 1. 代码管理

企业级代码托管：

```bash
# 创建项目
git init
git add .
git commit -m "Initial commit"
git remote add origin git@gitlab.com:group/project.git
git push -u origin master
```

### 2. 团队协作

代码审查和合并：

```bash
# 创建分支
git checkout -b feature/new-feature

# 提交更改
git add .
git commit -m "Add new feature"

# 推送
git push -u origin feature/new-feature

# 创建 Merge Request
```

### 3. 持续集成

自动化构建和测试：

```yaml
# .gitlab-ci.yml
image: node:18

cache:
  paths:
    - node_modules/

before_script:
  - npm install

build:
  stage: build
  script:
    - npm run build
  artifacts:
    paths:
      - dist/

test:
  stage: test
  script:
    - npm test
  coverage: '/Coverage: \d+\.\d+%/'
```

### 4. 持续部署

自动部署到生产环境：

```yaml
deploy:
  stage: deploy
  script:
    - echo "Deploying to production"
    - ./deploy.sh production
  environment:
    name: production
    url: https://example.com
  only:
    - master
```

## GitLab 与其他平台对比

### GitLab vs GitHub

| 特性 | GitLab | GitHub |
|------|--------|--------|
| 自托管 | 支持 | 有限 |
| CI/CD | 内置 | GitHub Actions |
| 问题跟踪 | 完整 | 基础 |
| Wiki | 有 | 有 |
| 包管理 | 多语言 | 多语言 |

### GitLab vs Gitee

| 特性 | GitLab | Gitee |
|------|--------|-------|
| 开源 | 是 | 是 |
| 自托管 | 支持 | 支持 |
| CI/CD | 强大 | 基础 |
| 国际化 | 全球 | 中国 |

3. **自托管**：数据完全自主可控

5. **安全扫描**：代码安全检测
6. **团队协作**：代码审查、问题跟踪
7. **国际化**：支持中文界面
