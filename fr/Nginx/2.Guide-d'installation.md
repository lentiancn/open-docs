# Guide d'installation de Nginx

## Exigences environnementales

Nginx est compatible avec plusieurs systèmes d'exploitation :

- Linux (Debian, Ubuntu, CentOS, RHEL, Alpine, etc.)
- FreeBSD
- Solaris
- AIX
- HP-UX
- macOS
- Windows

## Méthodes d'installation

### Méthode 1 : Utilisation du gestionnaire de packages (Recommandé)

#### Ubuntu/Debian

```bash
sudo apt update
sudo apt install nginx
sudo systemctl start nginx
sudo systemctl enable nginx
```

#### CentOS/RHEL

```bash
sudo yum install epel-release
sudo yum install nginx
sudo systemctl start nginx
sudo systemctl enable nginx
```

#### Alpine

```bash
apk add nginx
```

#### macOS

```bash
brew install nginx
brew services start nginx
```

### Méthode 2 : Compiler depuis les sources

```bash
wget https://nginx.org/download/nginx-1.26.0.tar.gz
tar -zxf nginx-1.26.0.tar.gz
cd nginx-1.26.0
./configure
make
sudo make install
```

### Méthode 3 : Utiliser Docker

```bash
docker pull nginx
docker run -d -p 80:80 nginx
```

### Méthode 4 : Windows

Télécharger depuis https://nginx.org/en/download.html et exécuter nginx.exe

## Vérification

```bash
nginx -v
curl http://localhost
```

## Chemins par défaut

- Configuration : `/etc/nginx/nginx.conf`
- Journaux : `/var/log/nginx/`
- Documents : `/var/www/html/`
