# PlantUML FAQ

## Allgemeine Fragen

### Was ist PlantUML?

PlantUML ist ein Open-Source-Tool zum Erstellen verschiedener Diagramme aus Textbeschreibungen.

### Ist PlantUML kostenlos?

Ja, PlantUML ist Open-Source und kostenlos unter GPL-Lizenz.

---

## Installation

### Warum wird Java benötigt?

PlantUML ist in Java geschrieben und benötigt JRE 8 oder höher.

### Wie installiere ich PlantUML?

- **Windows**: JAR-Datei herunterladen
- **macOS**: `brew install plantuml`
- **Linux**: Paketmanager verwenden

---

## Fehlerbehebung

### "java" wird nicht erkannt?

Java zum PATH hinzufügen oder JAVA_HOME setzen.

### Speicherfehler?

Java Heap-Size erhöhen:

```bash
java -Xmx2048m -jar plantuml.jar diagramm.wsd
```
