# Java 簡介

## 什麼是 Java

Java 是由 Sun Microsystems 公司於 1995 年推出的物件導向程式語言，經過 Oracle 公司收購後，目前由 Oracle 公司繼續維護和發展。Java 最初被稱為 Oak，1995 年正式更名為 Java。

Java 是一種廣泛應用於企業級開發的程式語言，其設計理念是「一次編寫，到處執行」（Write Once, Run Anywhere）。這意味著用 Java 編寫的程式可以在不同的作業系統上執行，無需重新編譯。

## Java 的核心特性

### 物件導向

Java 是一種純物件導向的語言，所有程式碼都必須寫在類別中。它支援封裝、繼承和多態三大特性，幫助開發者建構模組化、可維護的應用程式。

### 平台無關性

Java 程式透過 Java 虛擬機器（JVM）執行，不同作業系統有對應的 JVM 實現。同樣的 Java 位元組碼可以在任何安裝了 JVM 的系統上執行，這是 Java 最重要的特性之一。

### 自動記憶體管理

Java 提供了垃圾回收機制（Garbage Collection），自動管理程式執行時建立的記憶體物件，開發者無需手動釋放記憶體，大幅減少記憶體洩漏和懸空指標問題。

### 安全性

Java 在設計上提供了多層安全機制，包括位元組碼驗證、安全管理器、權限控制等，可以有效防止惡意程式碼的執行。

### 多執行緒支援

Java 內建了多執行緒支援，可以同時執行多個任務，提高程式的並發效能。

### 豐富的類別庫

Java 擁有龐大的標準類別庫（Java API），涵蓋了檔案操作、網路程式設計、資料庫連線、圖形介面、Web 開發等各個領域。

## Java 版本演進

### Long Term Support（LTS）版本

- **Java 8**：2014年發布，是最廣泛使用的 LTS 版本，引入了 Lambda 運算式和 Stream API
- **Java 11**：2018年發布，新增了局部變數類型推斷（var）和新的垃圾回收器
- **Java 17**：2021年發布，是最新的免費 LTS 版本，提供了更多的語言增強
- **Java 21**：2023年發布，引入了虛擬執行緒等重要特性
- **Java 25**：2025年發布，是目前的最新 LTS 版本

### 版本選擇建議

對於新專案，推薦使用 Java 21 或 Java 25，因為它們提供了最新的語言特性和效能改進。對於維護現有系統，應根據專案依賴和團隊熟悉程度選擇合適的版本。

## Java 應用領域

Java 廣泛應用於以下領域：

- **企業級應用**：大型分散式系統、ERP 系統、CRM 系統
- **Web 應用**：電子商務平台、內容管理系統、企業入口網站
- **行動應用**：Android 應用開發
- **大數據技術**：Hadoop、Spark 等大數據框架
- **雲端運算**：微服務架構、容器化應用
- **桌面應用**：跨平台的桌面軟體

## Java 生態系統

Java 擁有完整的生態系統，包括：

- **開發工具**：IntelliJ IDEA、Eclipse、VS Code
- **建構工具**：Maven、Gradle
- **框架**：Spring、Hibernate、MyBatis、Struts
- **應用伺服器**：Tomcat、Jetty、WildFly、WebLogic

## 總結

Java 作為最受歡迎的程式語言之一，憑藉其穩定性、可擴展性和豐富的生態系統，在企業級開發領域佔據重要地位。無論是初學者還是資深開發者，Java 都是值得學習和掌握的技能。
