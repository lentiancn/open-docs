# MySQL 개요

MySQL은 세계에서 가장 인기 있는 오픈 소스 관계형 데이터베이스 관리 시스템(RDBMS)입니다. 웹 애플리케이션에广泛적으로 사용되며 LAMP 스택( Linux, Apache, MySQL, PHP/Python/Perl)의 핵심 구성 요소입니다.

## MySQL이란?

MySQL은 애플리케이션을 위한 데이터를 저장하고 관리하는 강력하고 빠르며 신뢰할 수 있는 데이터베이스 시스템입니다. SQL(구조화 쿼리 언어)을 쿼리 언어로 사용합니다.

### 주요 기능

- **오픈 소스** - GPL 라이선스하 무료 사용
- **고성능** - 속도와 효율성에 최적화
- **확장성** - 대규모 데이터베이스 쉽게 처리
- **보안** - 강력한 사용자 관리 및 암호화
- **복제** - 마스터-슬레이브 및 마스터-마스터 지원
- **스토리지 엔진** - InnoDB, MyISAM, Memory 등
- **크로스 플랫폼** - Windows, Linux, macOS 등에서 동작

## 라이선스

MySQL은 두 가지 라이선스로 제공됩니다:
1. **GPL 라이선스** - 오픈 소스 프로젝트 무료
2. **상업용 라이선스** - 독점 애플리케이션용
