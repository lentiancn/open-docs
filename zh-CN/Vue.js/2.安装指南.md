# Vue.js 安装指南

## 快速体验

- **在线演练场**：直接访问 [Vue 演练场](https://play.vuejs.org) 快速体验
- **JSFiddle**：使用 [JSFiddle 入门模板](https://jsfiddle.net/yyx990803/2ke1ab0z/)
- **StackBlitz**：在浏览器中打开 [StackBlitz](https://vite.new/vue) 尝试完整构建设置

## 创建 Vue 应用（推荐方式）

### 前提条件
- 熟悉命令行操作
- 已安装 Node.js 版本 ^20.19.0 或 >=22.12.0

### 使用官方脚手架
在命令行中运行以下命令之一：

```bash
# npm
npm create vue@latest

# pnpm  
pnpm create vue@latest

# yarn (v1+)
yarn create vue

# yarn (Modern v2+)
yarn create vue@latest

# yarn (^v4.11)
yarn dlx create-vue@latest

# bun
bun create vue@latest
```

### 项目配置选项
创建项目时会提示以下可选功能：
- Project name: 项目名称
- Add TypeScript?: 是否添加 TypeScript 支持
- Add JSX Support?: 是否添加 JSX 支持
- Add Vue Router: 是否添加 Vue Router（单页应用开发）
- Add Pinia: 是否添加 Pinia（状态管理）
- Add Vitest: 是否添加 Vitest（单元测试）
- Add End-to-End Testing: 是否添加端到端测试（Cypress/Nightwatch/Playwright）
- Add ESLint: 是否添加 ESLint（代码质量）
- Add Prettier: 是否添加 Prettier（代码格式化）
- Add Vue DevTools 7: 是否添加 Vue DevTools 7 扩展（调试，实验性）

### 启动开发服务器
```bash
cd <your-project-name>
npm install
npm run dev
```

### 生产环境构建
```bash
npm run build
```

## 通过 CDN 使用 Vue

### 全局构建版本
```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<div id="app">{{ message }}</div>

<script>
const { createApp } = Vue

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

### ES 模块构建版本
```html
<div id="app">{{ message }}</div>

<script type="module">
import { createApp } from 'https://unpkg.com/vue@3/dist/vue.esm-browser.js'

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

### 启用 Import Maps
```html
<script type="importmap">
{
  "imports": {
    "vue": "https://unpkg.com/vue@3/dist/vue.esm-browser.js"
  }
}
</script>

<div id="app">{{ message }}</div>

<script type="module">
import { createApp } from 'vue'

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

## 开发环境推荐

- **IDE**: Visual Studio Code + [Vue - Official 扩展](https://marketplace.visualstudio.com/items?itemName=Vue.volar)
