# JDK

JDK (Java Development Kit) — это официальный комплект разработки на Java.

## Введение

JDK предоставляет инструменты и библиотеки для разработки на Java, включая компилятор (javac), среду выполнения (java) и различные утилиты.

## Функции

- **javac**: компилятор Java
- **java**: лаунчер среды выполнения Java
- **javadoc**: генератор документации
- **jar**: инструмент для создания архивов Java
- **jdb**: отладчик
- **Инструменты безопасности**: keytool, jarsigner

## Быстрый старт

### Установка

```bash
# Используя Homebrew
brew install openjdk@21

# Используя SDKMAN
sdk install java 21.0.2-tem
```

### Компиляция и запуск

```bash
javac HelloWorld.java
java HelloWorld
```
