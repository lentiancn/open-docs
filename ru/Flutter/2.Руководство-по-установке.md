# Flutter - Руководство по установке

Полное руководство по настройке среды разработки Flutter.

## Требования

### Windows

- **ОС**: Windows 10 или выше (64-бит)
- **Место на диске**: Минимум 2.8 ГБ
- **Инструменты**: PowerShell 5.0+, Git для Windows

### macOS

- **ОС**: macOS 10.14 (Mojave) или выше
- **Место на диске**: Минимум 2.8 ГБ
- **Инструменты**: Command Line Tools, Git

### Linux

- **ОС**: Ubuntu 18.04 или выше
- **Место на диске**: Минимум 2.8 ГБ
- **Зависимости**: bash, curl, git, unzip, xz-utils, zip, libglu1-mesa

## Шаги установки

### Установка на Windows

1. **Скачать Flutter SDK**

Посетите сайт Flutter, чтобы скачать последнюю версию:
```bash
# Или клонировать через git
git clone https://github.com/flutter/flutter.git -b stable
```

2. **Настроить переменные среды**

Добавить Flutter в системный PATH:
```powershell
# Запустить в PowerShell
[Environment]::SetEnvironmentVariable(
    "Path",
    $env:Path + ";C:\src\flutter\bin",
    "User"
)
```

3. **Проверить установку**

```bash
flutter doctor
```

### Установка на macOS

1. **Скачать Flutter SDK**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

2. **Настроить переменные среды**

```bash
# Добавить в ~/.bash_profile или ~/.zshrc
export PATH="$PATH:/Users/имя_пользователя/flutter/bin"
```

3. **Запустить flutter doctor**

```bash
flutter doctor
```

### Установка на Linux

1. **Установить зависимости**

```bash
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa
```

2. **Скачать Flutter SDK**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

3. **Настроить PATH**

```bash
export PATH="$PATH:/home/имя_пользователя/flutter/bin"
```

## Настройка среды разработки Android

### Установить Android Studio

1. Скачать Android Studio: https://developer.android.com/studio
2. Установить и запустить
3. Настроить SDK:

```bash
flutter config --android-sdk /путь/к/android/sdk
```

### Принять лицензии

```bash
flutter doctor --android-licenses
```

### Настроить эмулятор

```bash
# Создать эмулятор
flutter emulator --create --name my-emulator

# Запустить эмулятор
flutter emulator --launch my-emulator
```

## Настройка среды разработки iOS (macOS)

### Установить Xcode

```bash
# Установить через App Store
xcode-select --install
```

### Настроить iOS Simulator

```bash
# Список доступных симуляторов
xcrun simctl list devices available

# Запустить симулятор
open -a Simulator
```

## Настройка IDE

### VS Code

3. Перезапустить VS Code

### Android Studio

1. Установить плагин Flutter
2. Установить плагин Dart

## Проверка

```bash
# Проверить версию Flutter
flutter --version

# Запустить flutter doctor
flutter doctor

# Проверить обновления
flutter upgrade
```

## Распространённые проблемы

### Скачивание не удалось?

Использовать зеркало:
```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
export PUB_CACHE=/путь/к/кэшу
```

### Android SDK не найден?

```bash
flutter config --android-sdk /путь/к/android-sdk

# Или установить переменную среды
export ANDROID_SDK_ROOT=/путь/к/android-sdk
```

### Команда не найдена?

Убедиться, что Flutter в PATH:
```bash
echo $PATH
# должен содержать flutter/bin
```

## Удаление

```bash
# Удалить директорию Flutter
rm -rf /путь/к/flutter

# Удалить из PATH
```
