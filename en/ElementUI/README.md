# Element UI

[Element UI](https://element.eleme.io/) is a Vue 2.0-based UI component library that provides developers with rich desktop component solutions.

## Overview

Element UI is a Vue 2.0 component library developed and maintained by the Ele.me team.

## Main Features

- **Vue 2.0 Support**: Built on Vue 2.0
- **70+ Components**: Cover all aspects of desktop development
- **On-Demand Import**: Support babel-plugin-component
- **Internationalization**: Multiple language support
- **Theme Customization**: Flexible theming

## Quick Start

### Installation

```bash
npm install element-ui --save
```

### Usage

Full import:

```javascript
import Vue from 'vue'
import Element from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(Element)
```
