# GitLab 簡介

> GitLab 是一個完整的 DevOps 平台，用於軟體開發和營運

---

## 概述

GitLab 是一個基於 Git 的 Web 程式碼倉庫管理工具，提供 Wiki、問題追蹤和 CI/CD 流水線功能。它是一個完整的 DevOps 平台，使團隊能夠在一個介面中規劃、開發、測試、部署和監控應用程式。

---

## 核心功能

- **Git 程式碼倉庫托管**：功能完整的 Git 程式碼倉庫管理，支援分支、標籤和合併請求
- **問題追蹤**：完整的問題追蹤系統，支援標籤、里程碑和看板視圖
- **CI/CD 流水線**：內建持續整合和持續部署流水線
- **Wiki**：整合的文檔 Wiki
- **程式碼審查**：內建的程式碼審查和合併請求工作流
- **安全掃描**：企業級安全掃描和合規功能
- **容器鏡像倉庫**：內建 Docker 容器鏡像倉庫
- **GitOps**：Kubernetes 部署整合

---

## 架構

| 元件 | 說明 |
|------|------|
| GitLab Core | 程式碼倉庫管理和問題追蹤 |
| GitLab CI/CD | 持續整合和部署 |
| GitLab Registry | Docker 容器鏡像倉庫 |
| GitLab Pages | 靜態網站托管 |
| GitLab Mattermost | 內建聊天整合 |

---

## 應用場景

- 原始碼管理
- 持續整合/持續部署
- 專案管理和問題追蹤
- 使用 Wiki 進行文檔管理
- 安全掃描和合規審計
- 容器管理
