# Element UI

> Bibliothèque de composants UI de bureau basée sur Vue 2

---

## Description

Element UI est une bibliothèque de composants de bureau basée sur Vue 2.0 développée par l'équipe Eleme.

**Note**: Pour Vue 3, utilisez [Element Plus](https://github.com/element-plus/element-plus)

---

## Version

| Version | Vue | État |
|---------|-----|------|
| Element UI 2.x | Vue 2 | Stable |
| Element UI 1.x | Vue 2 | Obsolète |
