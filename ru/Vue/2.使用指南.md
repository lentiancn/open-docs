# Vue Руководство по использованию

## Основные концепции

### Экземпляр Vue

```javascript
import { createApp } from 'vue'

const app = createApp({
  data() {
    return {
      message: 'Привет Vue!'
    }
  }
})

app.mount('#app')
```

### Синтаксис шаблонов

```html
<!-- Интерполяция текста -->
<h1>{{ message }}</h1>

<!-- Директивы -->
<div v-if="show">Содержимое</div>
<div v-for="item in items">{{ item }}</div>
<button @click="handleClick">Клик</button>
<input v-model="inputValue">
```

## Реактивность

### Vue 3 Composition API (Рекомендуется)

```javascript
import { ref, reactive, computed } from 'vue'

export default {
  setup() {
    const count = ref(0)
    const state = reactive({ name: 'Vue' })
    
    const doubleCount = computed(() => count.value * 2)
    
    return { count, state, doubleCount }
  }
}
```

## Компоненты

### Создание компонента

```vue
<template>
  <div>
    <h2>{{ title }}</h2>
    <button @click="emitEvent">Клик</button>
  </div>
</template>

<script setup>
import { defineProps, defineEmits } from 'vue'

const props = defineProps({ title: String })
const emit = defineEmits(['click'])
</script>
```

## Условный рендеринг

```html
<div v-if="show">Содержимое</div>
<div v-else>По умолчанию</div>
```

## Обработка событий

```html
<button @click="handleClick">Клик</button>
<button @click.stop="handleClick">Остановить всплытие</button>
```
