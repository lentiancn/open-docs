# Angular 简介

现代 Web 应用开发框架。

## 什么是 Angular？

Angular 是一个由 Google 专业团队维护的 Web 框架，帮助开发者构建快速、可靠的应用程序。Angular 提供了一套广泛的工具、API 和库来简化和优化开发工作流。Angular 为构建快速、可靠的应用程序提供了坚实平台，能够随着团队和代码库规模的增长而扩展。Angular.dev 是 Angular 官方文档之家。

### 核心特性

- **TypeScript 原生**：完全基于 TypeScript 开发，享受类型安全
- **组件化架构**：基于组件的开发模式，易于维护和测试
- **依赖注入**：强大的依赖注入系统，提高代码可测试性
- **指令系统**：强大的指令系统，增强 HTML 表达能力
- **路由系统**：完整的客户端路由解决方案
- **表单处理**：响应式表单和模板驱动表单
- **HTTP 客户端**：完整的 HTTP 客户端服务
- **国际化**：内置国际化支持
- **动画系统**：强大的动画 API
- **测试支持**：完整的单元测试和端到端测试支持
- **信号（Signals）**：全新的响应式原语
- **Standalone API**：简化的模块化开发

### 技术架构

Angular 采用模块化架构，核心概念包括：

- **模块（Module）**：组织应用代码的基本单元
- **组件（Component）**：视图和业务逻辑的组合
- **服务（Service）**：封装可复用的业务逻辑
- **指令（Directive）**：扩展 HTML 元素行为
- **管道（Pipe）**：数据转换工具
- **信号（Signal）**：响应式状态管理

## 适用场景

- **企业级应用**：大型复杂业务系统
- **单页应用（SPA）**：需要流畅用户体验的应用
- **渐进式 Web 应用（PWA）**：需要离线能力的应用
- **混合移动应用**：结合 Ionic 开发移动应用
- **仪表盘**：数据可视化和管理后台
- **SSR 应用**：服务器端渲染应用
