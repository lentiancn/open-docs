# Bootstrap 5 사용 가이드

이 가이드는 Bootstrap 5로 반응형 웹사이트를 만드는 방법을 설명합니다.

## 빠른 시작

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 5 페이지</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <div class="container mt-5">
        <h1>안녕하세요, Bootstrap 5!</h1>
        <p class="lead">반응형 페이지입니다.</p>
    </div>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## 그리드 시스템

```html
<div class="container">
    <div class="row">
        <div class="col-12 col-md-6">열</div>
        <div class="col-12 col-md-6">열</div>
    </div>
</div>
```

## 주요 컴포넌트

### 버튼

```html
<button class="btn btn-primary">주요</button>
<button class="btn btn-secondary">보조</button>
<button class="btn btn-success">성공</button>
<button class="btn btn-danger">위험</button>
<button class="btn btn-outline-primary">외곽선</button>
```

### 네비게이션

```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">브랜드</a>
    </div>
</nav>
```

### 카드

```html
<div class="card" style="width: 18rem;">
    <div class="card-body">
        <h5 class="card-title">제목</h5>
        <p class="card-text">내용</p>
    </div>
</div>
```

### 폼

```html
<form>
    <div class="mb-3">
        <label class="form-label">이메일</label>
        <input type="email" class="form-control">
    </div>
    <button type="submit" class="btn btn-primary">전송</button>
</form>
```

### 모달

```html
<button class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#myModal">
    열기
</button>

<div class="modal" id="myModal">
    <div class="modal-dialog">
        <div class="modal-content">
            <div class="modal-header">
                <h5 class="modal-title">제목</h5>
                <button class="btn-close" data-bs-dismiss="modal"></button>
            </div>
        </div>
    </div>
</div>
```

## 유틸리티

### 여백

```html
<div class="m-3">마진</div>
<div class="mt-3">상단 마진</div>
<div class="p-3">패딩</div>
```

### 텍스트

```html
<p class="text-center">중앙 정렬</p>
<p class="fw-bold">굵게</p>
```

### 색상

```html
<div class="bg-primary text-white">주요</div>
<div class="bg-success text-white">성공</div>
```
