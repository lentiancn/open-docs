# Installationsanleitung für IBM Informix

## Systemanforderungen

**Unterstützte Plattformen:**
- Linux (Red Hat, Ubuntu, SUSE, etc.)
- Windows Server
- AIX
- HP-UX
- Solaris
- macOS
- ARM (für Edge-Geräte)

**Mindestanforderungen:**
- RAM: 1GB Minimum (8GB empfohlen)
- Festplatte: 2GB Minimum für Installation
- CPU: Kompatibel mit x86, PowerPC, ARM Architekturen

## Installationsschritte unter Linux

### Schritt 1: Installer herunterladen

Besuchen Sie IBM Passport Advantage oder IBM Fix Central und laden Sie das entsprechende Informix-Installationspaket herunter.

### Schritt 2: System vorbereiten

```bash
# Informix-Benutzer und -Gruppe erstellen
groupadd informix
useradd -g informix -d /home/informix informix

# Installationsverzeichnis erstellen
mkdir -p /opt/IBM/informix
chown informix:informix /opt/IBM/informix
```

### Schritt 3: Installer entpacken

```bash
tar -xf informix.tar
cd informix
```

### Schritt 4: Installation ausführen

```bash
# Als Root oder mit sudo
./install
```

Für neuere Versionen (14.10+), verwenden Sie den IBM Installation Manager:

```bash
chmod +x ids_install
./ids_install
```

Folgen Sie den Anweisungen:
- Akzeptieren Sie die Lizenzvereinbarung
- Wählen Sie den Installationspfad (/opt/IBM/informix)
- Wählen Sie die zu installierenden Komponenten

### Schritt 5: Umgebungsvariablen konfigurieren

Fügen Sie zu `~/.bashrc` oder `/etc/profile` hinzu:

```bash
export INFORMIXDIR=/opt/IBM/informix
export PATH=$INFORMIXDIR/bin:$PATH
export ONCONFIG=onconfig
export INFORMIXSERVER=ihr_servername
```

Änderungen anwenden:

```bash
source ~/.bashrc
```

### Schritt 6: sqlhosts-Datei konfigurieren

```bash
cd $INFORMIXDIR/etc
cp sqlhosts.std sqlhosts
vi sqlhosts
```

Fügen Sie Ihren Servereintrag hinzu:

```
ol_informix onsoctcp localhost ol_informix
```

### Schritt 7: Datenbankserver initialisieren

```bash
# Informix mit Standardeinstellungen initialisieren
oninit -ivy
```

### Schritt 8: Installation überprüfen

```bash
# Serverstatus prüfen
onstat -

# Initialisierung überprüfen
ls -la $INFORMIXDIR/storage/
```

## Installationsschritte unter Windows

### Schritt 1: Windows-Installer herunterladen

Herunterladen von IBM Passport Advantage.

### Schritt 2: Installer ausführen

- Doppelklicken Sie auf die `.exe` Datei
- Folgen Sie dem Installations-Assistenten

### Schritt 3: Konfiguration

- Verwenden Sie den Informix-Konfigurations-Assistenten
- Richten Sie den Windows-Dienst ein

### Schritt 4: Umgebungsvariablen einrichten

Konfigurieren Sie `INFORMIXDIR` in den Systemeigenschaften:
- Rechtsklick auf Arbeitsplatz → Eigenschaften
- Erweiterte Systemeinstellungen → Umgebungsvariablen
- Fügen Sie `INFORMIXDIR` und `INFORMIXSERVER` hinzu

## Überprüfung nach der Installation

```bash
# Serverstatus prüfen
onstat -

# Server starten (falls nicht bereits gestartet)
oninit

# Server stoppen
onmode -ky

# Mit Datenbank verbinden
dbaccess
```

## Häufige Aufgaben nach der Installation

### Zusätzliche Speicherbereiche erstellen

```bash
onspaces -c -d datadbs -p /pfad/zum/data_chunk -s 500000
```

### Logische Protokolle konfigurieren

```bash
onparams -a -d datadbs -s 10000
```

## Fehlerbehebung

### Server startet nicht

- Überprüfen Sie, ob Umgebungsvariablen korrekt gesetzt sind
- Prüfen Sie die `onconfig` und `sqlhosts` Dateien
- Überprüfen Sie Fehlerprotokolle in `$INFORMIXDIR/tmp/`
- Stellen Sie ausreichend Speicherplatz und Berechtigungen sicher

### Verbindungsprobleme

- Firewall-Einstellungen prüfen (Standardport 9088)
- Überprüfen Sie, ob `INFORMIXSERVER` mit der sqlhosts-Konfiguration übereinstimmt
- Stellen Sie sicher, dass sqlhosts die richtigen Netzwerkeinstellungen hat
