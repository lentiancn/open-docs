# H2 Datenbank - Einführung

## Was ist H2?

H2 ist ein in Java geschriebenes Open-Source relationales Datenbankmanagementsystem. Es kann als eingebettete Datenbank in Java-Anwendungen oder im Client-Server-Modus verwendet werden. Die Software wird unter der Mozilla Public License 2.0 (MPL 2.0) oder der ursprünglichen Eclipse Public License (EPL 1.0) Open-Source zur Verfügung gestellt.

## Hauptmerkmale von H2

### Eingebettete Datenbank
H2 kann direkt in Java-Anwendungen eingebettet werden, ohne einen separaten Datenbankserver. Dies macht es in Entwicklungs- und Testumgebungen sehr beliebt.

### Client-Server-Modus
Zusätzlich zum eingebetteten Modus unterstützt H2 auch den Client-Server-Modus und kann als eigenständiger Datenbankserver betrieben werden, auf den mehrere Anwendungen gleichzeitig zugreifen können.

### In-Memory-Datenbank
H2 unterstützt den In-Memory-Datenbankmodus, bei dem Daten im Speicher gespeichert werden und nach einem Neustart verloren gehen. Dies ist für schnelle Tests und Prototypenentwicklung sehr nützlich.

### Kompatibilität mit mehreren Datenbanken
H2 ist mit verschiedenen SQL-Standards und -Syntax kompatibel und kann das Verhalten von MySQL, PostgreSQL, Oracle, DB2, MS SQL Server simulieren, was Datenbankmigration und -tests erleichtert.

### Verschlüsselte Datenbank
H2 unterstützt die Datenbankverschlüsselung zum Schutz sensibler Daten.

### Volltextsuche
H2 verfügt über integrierte Volltextsuchfunktionen ohne zusätzliche Konfiguration.

### Zero-Management
Als eingebettete Datenbank erfordert H2 keinen dedizierten Datenbankadministrator für die Wartung.

## H2-Anwendungsszenarien

H2 wird häufig in folgenden Szenarien verwendet:

- Eingebettete Datenbank für Java-Anwendungen
- Entwicklungs- und Testumgebungen
- Schnelle Prototypenentwicklung
- Unit- und Integrationstests
- Leichte Desktop-Anwendungen
- Erlernen von SQL-Syntax

## Versionshistorie von H2

Die Entwicklung der H2-Datenbank-Engine begann im Mai 2004, mit der ersten Veröffentlichung im Dezember 2005. Der ursprüngliche Entwickler Thomas Mueller war auch der Gründer der Java-Datenbank-Engine HSQLDB (ursprünglich Hypersonic SQL).
