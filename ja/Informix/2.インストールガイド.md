# IBM Informix インストールガイド

## インストール前の前提条件

### 必要なシステム要件

- **オペレーティングシステム**：Linux（Red Hat、SUSE、Ubuntuなど）、Windows、macOS、AIX、Solarisなど
- **メモリ**：エディションにより異なる（Express Editionは8GBまで）
- **ディスク容量**：インストール媒体およびデータ用として十分な容量
- **依存関係**：Cコンパイラ、Java Development Kitなど

### 必要なソフトウェア

- Informix Connect または Client SDK
- データベース接続用のJDBC/ODBC driver

## Linux へのインストール手順

### ステップ1：ソフトウェアのダウンロード

IBM公式ウェブサイトまたは信頼性の高いソースから、Linuxオペレーティングシステム用のInformixデータベースソフトウェアパッケージをダウンロードします。

### ステップ2：依存関係のインストール

```bash
# Red Hat系
yum install gcc java-1.8.0-openjdk-devel

# SUSE系
zypper install gcc java-1_8_0-devel

# Ubuntu系
apt-get install gcc default-jdk
```

### ステップ3：圧縮パッケージの展開

```bash
tar -xzvf informix.tar.gz -C /opt/
cd /opt/informix
```

### ステップ4：インストールの実行

```bash
# インストールスクリプトを実行
./install
```

インストール中に：
- ライセンス契約に同意する
- インストールパスを選択する（/opt/informix）
- インストールするコンポーネントを選択する

### ステップ5：環境変数の設定

`.bashrc` または `.profile` に以下を追加：

```bash
export INFORMIXDIR=/opt/informix
export PATH=$INFORMIXDIR/bin:$PATH
export ONCONFIG=onconfig
export INFORMIXSERVER=server_name
```

設定を反映：

```bash
source ~/.bashrc
```

### ステップ6：sqlhosts ファイルの設定

```bash
cd $INFORMIXDIR/etc
cp sqlhosts.std sqlhosts
vi sqlhosts
```

以下を追加：

```
ol_informix onsoctcp localhost ol_informix
```

**注意**：リモートアクセスが必要な場合は、`localhost` を `*` に変更してください。

### ステップ7：onconfig ファイルの設定

```bash
cd $INFORMIXDIR/etc
cp onconfig.std onconfig.ol_informix
vi onconfig.ol_informix
```

主なパラメータを設定：

```
SERVERNUM 49
FULL_DISK_INIT 1
NETTYPE soctcp,1,50,CPU
```

### ステップ8：データベースサーバーの起動

初回起動は `-ivy` パラメータを使用：

```bash
oninit -ivy
```

### ステップ9：インストール確認

```bash
# プロセスの確認
ps -ef | grep oninit

# サーバー状態の確認
onstat -

# バージョン確認
onstat -V
```

## Windows へのインストール

### ステップ1：インストーラーのダウンロード

IBM Passport AdvantageからWindows用インストーラーをダウンロードします。

### ステップ2：インストーラーの実行

- ダウンロードした .exe ファイルをダブルクリック
- インストールウィザードの指示に従う

### ステップ3：設定

- Informix 設定ウィザードを使用
- Windows サービスとして設定

### ステップ4：環境変数の設定

システムプロパティで `INFORMIXDIR` を設定：
- 右クリック「マイコンピューター」→プロパティ
- システムの詳細設定→環境変数
- `INFORMIXDIR` と `INFORMIXSERVER` を追加

## インストール後の確認事項

- **プロセスの確認**：`onstat -` コマンドでサーバー状態を確認
- **ポートの確認**：`netstat -an | grep 9088` でリスニングポートを確認
- **ログの確認**：`$INFORMIXDIR/logs/` 配下のログファイルを確認

## 注意事項

- システムがハードウェア・ソフトウェア要件を満たしていることを確認
- ファイアウォールで Informix が使用するポート（デフォルト：9088）を開放
- Informix の実行には専用ユーザーの使用を推奨（root ユーザーは非推奨）
- 本番環境ではストレージレイアウトを事前に計画
