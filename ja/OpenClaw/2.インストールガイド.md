# OpenClaw インストールガイド

> OpenClawのセットアップとインストールに関する完全なガイド

---

## システム要件

| 要件 | 仕様 |
|-----|------|
| OS | Windows 10+、macOS、Linux |
| Node.js | バージョン18以上 |
| メモリ | 最低4GB RAM |
| ストレージ | 約500MB |

---

## インストール

### 前提条件

Node.jsがインストールされていることを確認します：

```bash
node --version
npm --version
```

### ステップバイステップインストール

#### 1. npmでインストール

```bash
npm install -g openclaw
```

#### 2. Gatewayサービスを起動

GatewayサービスはOpenClawの中核です：

```bash
# ステータスを確認
openclaw gateway status

# Gatewayを起動
openclaw gateway start

# Gatewayを再起動
openclaw gateway restart

# Gatewayを停止
openclaw gateway stop
```

#### 3. 初期設定

インストール後、設定ファイルをセットアップします：

```bash
# 設定ディレクトリを作成
mkdir -p ~/.openclaw

# デフォルト設定をコピー
cp /path/to/openclaw/config/* ~/.openclaw/
```

---

## 設定

### メイン設定ファイル

| ファイル | 説明 |
|---------|------|
| AGENTS.md | ワークスペース設定 |
| SOUL.md | パーソナリティ設定 |
| TOOLS.md | ツール設定 |
| USER.md | ユーザー設定 |

### 環境変数

```bash
# オプション：カスタムワークスペースディレクトリ
OPENCLAW_WORKSPACE=/path/to/workspace

# オプション：Gatewayポート
OPENCLAW_PORT=3000
```

---

## 確認

### インストールをテスト

```bash
# バージョン情報を表示
openclaw --version

# Gatewayステータスを確認
openclaw gateway status

# 使用可能なコマンドを表示
openclaw --help
```

### クイックスタート

```bash
# インタラクティブセッションを開始
openclaw chat

# 単一のタスクを実行
openclaw exec "helloと返す"
```

## トラブルシューティング

### Gatewayが起動しない？

**解決策**：
1. Node.jsが正しくインストールされていることを確認
2. 権限を確認
3. ログを確認：`openclaw gateway logs`

### ポートが既に使用されている？

**解決策**：
```bash
# ポートを変更
openclaw gateway start --port 3001
```

### モジュールが見つからない？

**解決策**：
```bash
# 再インストール
npm uninstall -g openclaw
npm install -g openclaw
```

---

## アンインストール

```bash
npm uninstall -g openclaw

# Gatewayサービスを削除
openclaw gateway stop
rm -rf ~/.openclaw
```
