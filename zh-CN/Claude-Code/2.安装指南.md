# Claude Code 安装指南

本指南将帮助你完成 Claude Code 的安装和初始配置。无论你使用哪种操作系统，都可以按照以下步骤顺利进行。

## 系统要求

在安装 Claude Code 之前，请确保你的系统满足以下要求：

### 基础要求
- **操作系统**：macOS、Windows（WSL）、Linux
- **Node.js**：18.x 或更高版本
- **Git**：已安装并配置好
- **网络**：能够访问 Anthropic API（需要代理请自行配置）

### 推荐配置
- 至少 8GB 内存
- 稳定的网络连接
- 已配置 SSH 密钥（用于 Git 操作）

## 安装步骤

### 第一步：安装 CLI 工具

Claude Code 提供命令行工具，你可以选择以下安装方式：

#### 使用 npm 安装（推荐）

```bash
npm install -g @anthropic-ai/claude-code
```

#### 使用 Homebrew 安装（macOS）

```bash
brew install claude-code
```

#### 手动下载安装

1. 访问 [GitHub Releases](https://github.com/anthropic/claude-code/releases) 页面
2. 下载对应你操作系统的最新版本
3. 解压并将可执行文件移动到系统路径中

```bash
# 例如在 Linux/macOS 上
sudo mv claude-code /usr/local/bin/
```

### 第二步：配置认证

1. **获取 API Key**
   - 访问 [Anthropic Console](https://console.anthropic.com/)
   - 创建或登录你的账户
   - 在 API Keys 页面生成新的密钥

2. **配置环境变量**

   ```bash
   # 临时设置（当前终端有效）
   export ANTHROPIC_API_KEY="your-api-key-here"

   # 永久设置（添加到 ~/.bashrc 或 ~/.zshrc）
   echo 'export ANTHROPIC_API_KEY="your-api-key-here"' >> ~/.bashrc
   ```

### 第三步：验证安装

运行以下命令验证 Claude Code 是否正常工作：

```bash
claude-code --version
```

你应该能看到版本号输出，表示安装成功。

## IDE 集成安装

如果你希望在 IDE 中使用 Claude Code，可以选择以下集成方式：

### VS Code 集成

1. 打开 VS Code
2. 进入扩展市场（左侧栏 Extensions 图标）
3. 搜索 "Claude Code"
4. 点击安装

安装完成后：
1. 点击左侧的 Claude Code 图标
2. 登录你的 Anthropic 账户
3. 开始使用

### JetBrains 系列集成

支持的 IDE：
- IntelliJ IDEA
- WebStorm
- PyCharm
- GoLand
- 等其他 JetBrains 产品

安装步骤：
1. 打开 IDE
2. 进入 Settings → Plugins
3. 搜索 "Claude Code"
4. 安装并重启 IDE

## 代理配置

如果你在中国大陆等需要代理的地区使用，需要配置代理：

### 方式一：环境变量

```bash
export HTTP_PROXY="http://127.0.0.1:7890"
export HTTPS_PROXY="http://127.0.0.1:7890"
```

### 方式二：Claude Code 配置

创建配置文件 `~/.claude/settings.json`：

```json
{
  "proxy": {
    "http": "http://127.0.0.1:7890",
    "https": "http://127.0.0.1:7890"
  }
}
```

## 常见问题

### 安装失败怎么办

1. **检查 Node.js 版本**
   ```bash
   node --version
   ```
   如果版本低于 18，请先升级 Node.js。

2. **清除 npm 缓存**
   ```bash
   npm cache clean --force
   ```

3. **查看详细错误信息**
   ```bash
   npm install -g @anthropic-ai/claude-code --verbose
   ```

### API Key 无效

1. 确认 API Key 已正确复制，没有多余空格
2. 检查账户是否已激活
3. 确认 API Key 有足够的使用额度

### 连接超时

1. 检查网络连接
2. 确认代理配置正确
3. 尝试更换网络环境

## 升级 Claude Code

定期升级以获得最新功能：

```bash
# npm 升级
npm update -g @anthropic-ai/claude-code

# Homebrew 升级
brew upgrade claude-code
```

## 卸载

如果需要卸载 Claude Code：

```bash
# npm 卸载
npm uninstall -g @anthropic-ai/claude-code

# 删除配置
rm -rf ~/.claude
```

完成以上步骤后，你就可以开始使用 Claude Code 了。下一章将介绍基本使用方法。
