# Flutter

Flutter는 크로스 플랫폼 애플리케이션 구축을 위해 Google이 개발한 오픈소스 UI 툴킷입니다.

## 설명

Flutter를 사용하면 단일 코드베이스로 iOS, Android, 웹 및 데스크톱용으로 아름다운 애플리케이션을 구축할 수 있습니다.

## 주요 기능

- **크로스 플랫폼**: 하나의 코드, 여러 플랫폼
- **핫 리로드**: 빠른 개발 반복
- **풍부한 컴포넌트**: Material Design 및 Cupertino 컴포넌트
- **Dart 언어**: 배우기 쉽고 우수한 성능
- **선언형 UI**: 아름다운 인터페이스 구축

## 버전 정보

| 버전 | 상태 |
|------|------|
| Flutter 3.x | ✅ 현재 |

## 빠른 시작

### 설치

```bash
# macOS
brew install flutter

# Linux
git clone https://github.com/flutter/flutter.git -b stable
```

### 프로젝트 생성

```bash
flutter create my_app
cd my_app
flutter run
```
