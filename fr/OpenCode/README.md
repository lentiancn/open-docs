# OpenCode

Agent de codage IA open source.

## Aperçu

OpenCode est un agent de codage IA open source disponible comme interface terminal, application bureau ou extension IDE. Avec LSP activé, multi-sessions et support pour 75+ fournisseurs LLM.

### Fonctionnalités principales

- **LSP activé** - Charge automatiquement les bons LSP
- **Multi-sessions** - Démarrez plusieurs agents en parallèle
- **N'importe quel modèle** - 75+ fournisseurs LLM
- **Confidentialité d'abord** - Pas de code ou données de contexte stockés

### Statistiques

- GitHub Stars : 100 000+
- Contributeurs : 700+
- Développeurs actifs mensuels : 2 500 000+

## Démarrage rapide

### Installer

```bash
curl -fsSL https://opencode.ai/install | bash
```

### Configurer

```bash
opencode
/connect
```

Visitez opencode.ai/auth pour obtenir la clé API.

### Initialiser

```bash
/init
```

### Utilisation

```
Explique ce code
Aide-moi à ajouter une fonction de connexion

```
