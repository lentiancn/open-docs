# MultiTalk Installationsanleitung

MultiTalk ist ein Multi-Sprecher-Audiogenerierungswerkzeug, das natürlich klingende Multi-Voice-Audioinhalte erstellt.

## Versionsinformationen

| Version | Status | Beschreibung |
|---------|--------|-------------|
| MultiTalk v1.0 | ✅ Aktuell | Erste Version |

## Systemanforderungen

### Hardwareanforderungen

| Anforderung | Minimum | Empfohlen |
|-------------|---------|-------------|
| GPU | NVIDIA 4GB VRAM | NVIDIA 8GB+ VRAM |
| RAM | 8GB | 16GB+ |
| Speicher | 5GB | 10GB+ SSD |

### Softwareanforderungen

| Software | Version |
|----------|---------|
| Python | 3.8+ |
| CUDA | 11.0+ |

## Installationsmethoden

### Methode 1: Aus dem Quellcode

#### 1. Repository klonen

```bash
git clone https://github.com/MultiTalk/MultiTalk.git
cd MultiTalk
```

#### 2. Virtuelle Umgebung erstellen

```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
# oder
venv\Scripts\activate  # Windows
```

#### 3. Abhängigkeiten installieren

```bash
pip install -r requirements.txt
```

#### 4. Modelle herunterladen

```bash
bash scripts/download_models.sh
```

### Methode 2: Docker

```bash
docker pull multitalk/multitalk:latest
docker run --gpus all -v /path/to/data:/data multitalk/multitalk:latest
```

## Konfiguration

### Umgebungsvariablen

```bash
export CUDA_VISIBLE_DEVICES=0
export PYTHONPATH=/path/to/MultiTalk:$PYTHONPATH
```

### Modellpfade

Bearbeite `config.yaml`:

```yaml
model:
  checkpoint_path: "checkpoints/multitalk.pth"
```

## Schnellstart

### Multi-Sprecher-Audio generieren

```bash
python inference.py \
  --text "Hallo, dies ist eine Multi-Sprecher-Demo." \
  --speakers speaker1,speaker2 \
  --output output.wav
```

### Parameter

```bash
--text         # Eingabetext
--speakers     # Sprecher-IDs (kommagetrennt)
--output       # Ausgabedateipfad
--speed        # Sprechgeschwindigkeit (Standard: 1.0)
```

## Häufige Probleme

### GPU nicht verfügbar?

```bash
nvidia-smi
python -c "import torch; print(torch.cuda.is_available())"
```

### Modell-Download fehlgeschlagen?

Überprüfe die Internetverbindung oder lade die Modelle manuell herunter.
