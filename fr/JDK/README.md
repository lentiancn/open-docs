# JDK

JDK (Java Development Kit) est le kit de développement Java officiel.

## Introduction

JDK fournit des outils et bibliothèques pour le développement Java, y compris le compilateur (javac), le runtime (java) et divers utilitaires.

## Fonctionnalités

- **javac**: Compilateur Java
- **java**: Lanceur runtime Java
- **javadoc**: Générateur de documentation
- **jar**: Outil d'archive Java
- **jdb**: Débogueur
- **Outils de sécurité**: keytool, jarsigner

## Démarrage rapide

### Installer

```bash
# Utiliser Homebrew
brew install openjdk@21

# Utiliser SDKMAN
sdk install java 21.0.2-tem
```

### Compiler et exécuter

```bash
javac HelloWorld.java
java HelloWorld
```
