# PlantUML Manual de Usuario

## Sintaxis Básica

### Diagrama de Secuencia

```plantuml
@startuml
Alice -> Bob: Hola
Bob --> Alice: ¡Hola!
@enduml
```

### Diagrama de Clases

```plantuml
@startuml
class Usuario {
    +id: int
    +nombre: string
    +login()
}

class Administrador extends Usuario {
    +aprobar()
}
@enduml
```

---

## Tipos de Diagramas

- Diagramas de secuencia
- Diagramas de clases
- Diagramas de casos de uso
- Diagramas de actividad
- Diagramas de estado

---

## Exportar

```bash
java -jar plantuml.jar -tpng diagrama.wsd
```
