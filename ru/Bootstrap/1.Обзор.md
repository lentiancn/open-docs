# Обзор Bootstrap

> Популярный CSS-фреймворк для фронтенда

---

## Описание проекта

Bootstrap — это CSS-фреймворк с открытым исходным кодом, разработанный и поддерживаемый Twitter, впервые выпущенный в 2011 году. Bootstrap предоставляет богатые готовые компоненты и адаптивную сетку для быстрого создания современных адаптивных веб-сайтов и веб-приложений.

---

## Информация о версиях

| Версия | Дата выпуска | Статус |
|--------|--------------|--------|
| Bootstrap 5.3 | Май 2023 | Текущая стабильная |
| Bootstrap 5.2 | Сентябрь 2022 | Стабильная |
| Bootstrap 5.1 | Август 2021 | Стабильная |
| Bootstrap 4.x | Январь 2018 | Не поддерживается |

---

## Основные функции

- **Адаптивный дизайн**: Встроенные адаптивные макеты
- **Готовые компоненты**: Кнопки, навигация, формы, карточки и т.д.
- **Мощная система сетки**: 12-колоночная адаптивная сетка
- **Параметры настройки**: Переменные Sass для легкой настройки
- **JavaScript-плагины**: Выпадающие списки, модальные окна и т.д.
- **Библиотека иконок**: Встроенные Bootstrap Icons

---

## Сценарии использования

- Разработка адаптивных веб-сайтов
- Административные панели
- Быстрое прототипирование
- Проекты с приоритетом для мобильных
