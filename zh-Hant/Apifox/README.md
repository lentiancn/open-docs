# Apifox

API 一體化協作平台。

## 簡介

Apifox 是 API 一體化協作平台，集 API 文檔、調試、Mock、自動化測試於一體。

### 主要特性

- **API 文檔** - 自動生成 API 文檔
- **API 調試** - HTTP 請求模擬
- **Mock 服務** - 快速創建 Mock 服務器
- **自動化測試** - API 自動化測試
- **團隊協作** - 項目管理和權限控制
