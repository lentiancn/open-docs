# OpenCode Usage Guide

## Quick Start

### Create New File
- File → New File or Ctrl+N

### Open Folder
- File → Open Folder or Ctrl+K Ctrl+O

### Save File
- File → Save or Ctrl+S

## Editor Features

### Syntax Highlighting
Supported languages include:
- JavaScript, TypeScript, Python, Java
- C, C++, C#, Go, Rust
- HTML, CSS, JSON, YAML
- Markdown, and more

### Multi-Cursor Editing
- Alt+Click to add cursor
- Ctrl+Alt+Down to add line below
- Ctrl+Alt+Up to add line above

### Search and Replace
- Ctrl+F: Find
- Ctrl+H: Find and Replace
- Ctrl+Shift+F: Find in files
- Ctrl+Shift+H: Replace in files

## Terminal

### Open Terminal
- View → Terminal or Ctrl+`

### Terminal Commands
```bash
# Run commands
npm run dev

# Git commands
git status
git commit -m "message"
```

## Git Integration

### Stage Changes
- Click on the file in Source Control
- Or use Stage All button

### Commit
1. Enter commit message
2. Click Commit button
3. Push to remote

### Branch Management
- Click branch name in bottom bar
- Create new branch
- Switch branches

## Extensions

### Install Extension
1. Extensions view (Ctrl+Shift+X)
2. Search for extension
3. Click Install

### Popular Extensions
- Prettier - Code formatter
- ESLint - Linting
- GitLens - Git features
- Live Server - Dev server

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Ctrl+S | Save |
| Ctrl+C | Copy |
| Ctrl+V | Paste |
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
| Ctrl+/ | Toggle comment |
| Alt+Up/Down | Move line |
| Ctrl+D | Find next |

## Settings

### Open Settings
- File → Preferences → Settings
- Or Ctrl+,

### Common Settings
```json
{
  "editor.fontSize": 14,
  "editor.tabSize": 2,
  "files.autoSave": "afterDelay",
  "terminal.integrated.fontSize": 14
}
```
