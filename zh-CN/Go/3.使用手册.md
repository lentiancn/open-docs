# Go 使用手册

## 基本语法

### 变量声明

Go 语言有多种变量声明方式。基本的声明方式使用 var 关键字：

```go
// 声明变量，指定类型
var name string = "Go"

// 类型推断声明
var age = 10

// 批量声明
var (
    firstName string
    lastName  string
    score     float64
)

// 简短声明（函数内部）
message := "Hello, World!"
```

Go 语言的变量声明后必须使用，否则会编译错误。如果需要声明暂时不使用的变量，可以使用下划线 _ 接收：

```go
result, _ := someFunction()  // 忽略第一个返回值
```

### 常量声明

常量使用 const 关键字声明，常量可以是字符、字符串、布尔值或数值：

```go
const Pi = 3.14159
const (
    StatusOK = 200
    StatusNotFound = 404
)
```

### 基本数据类型

Go 语言的基本数据类型包括：

- 布尔类型：bool（true 或 false）
- 整数：int、int8、int16、int32、int64，以及对应的无符号类型 uint 等
- 浮点数：float32、float64
- 复数：complex64、complex128
- 字符串：string

类型转换需要显式进行：

```go
var i int = 42
var f float64 = float64(i)
var s string = strconv.Itoa(i)  // 整数转字符串
```

## 函数

### 函数定义

Go 函数的定义使用 func 关键字：

```go
func greet(name string) string {
    return "Hello, " + name
}

// 多返回值
func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}

// 命名返回值
func split(sum int) (x, y int) {
    x = sum * 4 / 9
    y = sum - x
    return  // 不需要指定返回值
}
```

### 变长参数

函数可以接受可变数量的参数：

```go
func sum(nums ...int) int {
    total := 0
    for _, n := range nums {
        total += n
    }
    return total
}

// 使用
result := sum(1, 2, 3, 4, 5)
```

### 闭包

Go 支持闭包，即函数可以捕获并修改外部作用域的变量：

```go
func counter() func() int {
    count := 0
    return func() int {
        count++
        return count
    }
}

// 使用
c := counter()
fmt.Println(c())  // 1
fmt.Println(c())  // 2
```

## 并发编程

### Goroutine

Goroutine 是 Go 语言中的轻量级执行单元，创建成本极低：

```go
// 启动 goroutine
go func() {
    fmt.Println("Hello from goroutine!")
}()

// 带参数的 goroutine
go func(msg string) {
    fmt.Println(msg)
}("Hello!")
```

### Channel

Channel 是 goroutine 之间的通信机制：

```go
// 创建 channel
ch := make(chan string)

// 发送数据
ch <- "message"

// 接收数据
msg := <-ch

// 带缓冲的 channel
bufferedCh := make(chan int, 10)
```

### Select

Select 语句用于同时等待多个 channel 操作：

```go
select {
case msg := <-ch1:
    fmt.Println("Received:", msg)
case msg := <-ch2:
    fmt.Println("Received:", msg)
case <-time.After(time.Second):
    fmt.Println("Timeout")
}
```

### 并发安全

使用 sync 包进行并发控制：

```go
var (
    mu    sync.Mutex
    count int
)

func increment() {
    mu.Lock()
    defer mu.Unlock()
    count++
}
```

## 错误处理

### 错误返回

Go 使用返回值进行错误处理，这是 Go 的惯用方式：

```go
result, err := doSomething()
if err != nil {
    return err
}
```

### 自定义错误

可以使用 errors 包创建自定义错误：

```go
import "errors"

func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}
```

### 错误包装

Go 1.13 引入了错误包装，可以使用 fmt.Errorf 和 %w 动词：

```go
if err != nil {
    return fmt.Errorf("failed to do something: %w", err)
}
```

## 接口

### 接口定义

Go 的接口是隐式实现的，不需要显式声明：

```go
type Writer interface {
    Write([]byte) (int, error)
}

type Reader interface {
    Read([]byte) (int, error)
}

// 常用组合接口
type ReadWriter interface {
    Reader
    Writer
}
```

### 空接口

空接口 interface{} 可以存储任意类型的值：

```go
var i interface{} = 42
var s interface{} = "hello"
var m interface{} = map[string]int{"a": 1}
```

Go 1.18 引入了泛型，可以使用 any 替代 interface{}：

```go
var i any = 42
```

## 包和模块

### 创建模块

使用 go mod init 创建新模块：

```bash
go mod init github.com/username/project
```

### 依赖管理

添加依赖：

```bash
go get github.com/pkg/errors
```

更新依赖：

```bash
go get -u all
```

清理未使用的依赖：

```bash
go mod tidy
```

## 测试

### 单元测试

Go 内置了测试框架，测试文件以 _test.go 结尾：

```go
import "testing"

func TestAdd(t *testing.T) {
    result := Add(2, 3)
    if result != 5 {
        t.Errorf("Add(2, 3) = %d; want 5", result)
    }
}
```

运行测试：

```bash
go test ./...
go test -v ./...
go test -cover ./...
```

### 基准测试

测试函数以 Benchmark 开头：

```go
func BenchmarkHello(b *testing.B) {
    for i := 0; i < b.N; i++ {
        Hello()
    }
}
```

运行基准测试：

```bash
go test -bench=.
```

## 常用标准库

### fmt - 格式化 I/O

```go
fmt.Println("Hello")
fmt.Printf("Name: %s, Age: %d\n", name, age)
fmt.Sprintf("Result: %d", result)
```

### strings - 字符串操作

```go
strings.Contains(s, "sub")
strings.Split(s, ",")
strings.TrimSpace(s)
strings.ToUpper(s)
```

### net/http - HTTP 服务

```go
http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
    fmt.Fprint(w, "Hello, HTTP!")
})
http.ListenAndServe(":8080", nil)
```

### json - JSON 处理

```go
// 编码
data, _ := json.Marshal(person)

// 解码
var p Person
json.Unmarshal(data, &p)
```

## 常用工具

### gofmt - 代码格式化

```bash
gofmt -w main.go  # 格式化并写入文件
gofmt -d main.go  # 显示格式化差异
```

### go vet - 静态分析

```bash
go vet ./...  # 检查代码问题
```

### go doc - 文档查看

```bash
go doc fmt.Println  # 查看函数文档
go doc fmt          # 查看包文档
```
