# JDK

Java 開発ツールキット

## はじめに

JDK（Java Development Kit）は、Javaプログラミング言語用のソフトウェア開発キットであり、Java開発の中核を成します。JDKにはJava実行環境（JRE）、Javaコンパイラ（javac）、Java仮想マシン（JVM）、および大量のJavaクラスライブラリやツールが含まれています。

## メジャーバージョン

- **JDK 25** - 最新バージョン、新機能とパフォーマンス改善
- **JDK 21** - 最新の長期サポートバージョン（LTS）
- **JDK 17** - 成熟したLTSバージョン、広く使用
- **JDK 11** - LTSバージョン、Java有料機能の境目

## コアコンポーネント

- **javac** - Javaコンパイラ、ソースコードをバイトコードにコンパイル
- **java** - Javaプログラム実行コマンド
- **jvm** - Java仮想マシン、バイトコードを実行
- **jar** - Javaアーカイブツール
- **javadoc** - APIドキュメント生成ツール
- **jdb** - Javaデバッガー

## バージョンによる違い

異なるJDKメジャーバージョンには、インストール、使用、機能において大きな違いがあります：

- **JDK 8**: 商用に完全に無料のOracle JDKの最後のバージョン
- **JDK 11**: var、新しいHTTPクライアント、ZGC（実験的）を導入
- **JDK 17**: NFTCの下で完全に無料のOracle JDKの最後のバージョン
- **JDK 21**: 仮想スレッド、パターンマッチング、レコード（安定）
- **JDK 25**: 最新機能、サポートサイクルが短い
