# Oracle Часто задаваемые вопросы

## Общие вопросы

### Что такое Oracle Database?

Oracle Database — это собственная система управления реляционными базами данных, разработанная Oracle Corporation.

### Что такое PL/SQL?

PL/SQL — это процедурное расширение языка SQL от Oracle.

---

## Установка

### Системные требования?

- Минимум 2GB RAM
- Минимум 8GB места на диске

### ORA-12541: TNS:no listener?

Запустите listener:
```bash
lsnrctl start
```

## Производительность

### Как проверить производительность запросов?

```sql
EXPLAIN PLAN FOR your_query;
SELECT * FROM TABLE(DBMS_XPLAN.DISPLAY);
```
