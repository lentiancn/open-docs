# MongoDB Installationsanleitung

## Versionsauswahl und Plattformanforderungen

### Versionsauswahl

MongoDB hat derzeit zwei Hauptversionsserien:

- **MongoDB 8.2**: Neueste stabile Version, empfohlen für neue Projekte
- **MongoDB 7.0**: Langzeitunterstützungsversion, geeignet für Produktionsumgebungen, die Stabilität erfordern

Ab MongoDB 8.2 sind Minor Releases für bestimmte Anwendungsfälle bei lokalen Bereitstellungen verfügbar.

### Plattformanforderungen

- MongoDB unterstützt keine 32-Bit-Plattformen mehr. Stellen Sie sicher, dass Ihr Betriebssystem 64-Bit ist
- MongoDB 6.0 hat am 31. Juli 2025 das Ende des Lebenszyklus erreicht

### Unterstützte Plattformen

MongoDB Community Edition unterstützt die folgenden Plattformen und Architekturen:

**Ubuntu**

- Ubuntu 24.04 (x86_64, arm64)
- Ubuntu 22.04 (x86_64, arm64)
- Ubuntu 20.04 (x86_64, arm64)

**Debian**

- Debian 12 (x86_64)
- Debian 11 (x86_64)

**RHEL/CentOS/Rocky Linux**

- RHEL/Rocky/Alma/Oracle Linux 9.0+ (x86_64, arm64)
- RHEL/Rocky/Alma/Oracle Linux 8.0+ (x86_64, arm64, ppc64le, s390x)
- RHEL/Oracle Linux 7.0+ (x86_64)

**macOS**

- macOS 14 (x86_64, arm64)
- macOS 13 (x86_64, arm64)
- macOS 12 (x86_64, arm64)
- macOS 11 (x86_64, arm64)

**Windows**

- Windows 11 (x86_64)

**Andere Linux**

- Amazon Linux 2023 (x86_64, arm64)

## MongoDB Community Edition auf Ubuntu installieren

### Schritt 1: Öffentlichen Schlüssel importieren

```bash
curl -fsSL https://www.mongodb.org/static/pgp/server-8.0.asc | sudo gpg -o /usr/share/keyrings/mongodb-server-8.0.gpg --dearmor
```

### Schritt 2: MongoDB-Repository-Datei erstellen

**Ubuntu 24.04 (Noble)**:

```bash
echo "deb [ signed-by=/usr/share/keyrings/mongodb-server-8.0.gpg ] http://repo.mongodb.org/apt/debian noble/mongodb-org/8.0 main" | sudo tee /etc/apt/sources.list.d/mongodb-org-8.0.list
```

**Ubuntu 22.04 (Jammy)**:

```bash
echo "deb [ signed-by=/usr/share/keyrings/mongodb-server-8.0.gpg ] http://repo.mongodb.org/apt/debian jammy/mongodb-org/8.0 main" | sudo tee /etc/apt/sources.list.d/mongodb-org-8.0.list
```

**Ubuntu 20.04 (Focal)**:

```bash
echo "deb [ signed-by=/usr/share/keyrings/mongodb-server-8.0.gpg ] http://repo.mongodb.org/apt/debian focal/mongodb-org/8.0 main" | sudo tee /etc/apt/sources.list.d/mongodb-org-8.0.list
```

### Schritt 3: Paketliste aktualisieren

```bash
sudo apt update
```

### Schritt 4: MongoDB installieren

```bash
sudo apt install -y mongodb-org
```

### Schritt 5: MongoDB starten

```bash
sudo systemctl start mongod
sudo systemctl enable mongod
```

## Auf RHEL/CentOS/Rocky Linux installieren

### Mit YUM/DNF installieren

#### Schritt 1: Repository-Datei erstellen

```bash
[mongodb-org-8.0]
name=MongoDB Repository
baseurl=https://repo.mongodb.org/yum/redhat/$releasever/mongodb-org/8.0/x86_64/
gpgcheck=1
enabled=1
gpgkey=https://www.mongodb.org/static/pgp/server-8.0.asc
```

#### Schritt 2: MongoDB installieren

```bash
sudo dnf install -y mongodb-org
```

#### Schritt 3: MongoDB starten

```bash
sudo systemctl start mongod
sudo systemctl enable mongod
```

## Auf macOS installieren

```bash
brew update

```

## Auf Windows installieren

### MSI-Installer verwenden

1. MSI-Installer von MongoDB Download Center herunterladen
2. Installer ausführen und den Assistenten-Schritten folgen
3. "Complete" auswählen, um alle Komponenten zu installieren

### ZIP-Paket verwenden

1. Windows-ZIP-Paket herunterladen
2. Dateien nach `C:\Program Files\MongoDB` extrahieren
3. Datenverzeichnis erstellen: `mkdir C:\data\db`
4. MongoDB starten: `"C:\Program Files\MongoDB\Server\8.0\bin\mongod.exe" --dbpath C:\data\db`

## Mit Docker installieren

### Schritt 1: Image pullen

```bash
docker pull mongodb/mongodb-community-server
```

### Schritt 2: Container ausführen

```bash
docker run --name mongodb -d -p 27017:27017 mongodb/mongodb-community-server
```

### Datenpersistenz

```bash
docker run --name mongodb -d -p 27017:27017 -v mongodb_data:/data/db mongodb/mongodb-community-server
```

## Installation überprüfen

### Mit MongoDB verbinden

```bash
mongosh
```

### Version überprüfen

```bash
db.version()
```

### Grundlegende Befehle

```bash
show dbs
use test
db.help()
```

## Nach der Installation

### Administratorbenutzer erstellen

```bash
use admin
db.createUser({
  user: "admin",
  pwd: "ihr_sicheres_passwort",
  roles: [
    { role: "userAdminAnyDatabase", db: "admin" },
    { role: "readWriteAnyDatabase", db: "admin" }
  ]
})
```

### Authentifizierung aktivieren

1. MongoDB-Dienst stoppen
2. `/etc/mongod.conf` bearbeiten:

```yaml
security:
  authorization: enabled
```

3. Dienst neu starten:

```bash
sudo systemctl restart mongod
```

## MongoDB deinstallieren

### Ubuntu/Debian

```bash
sudo apt remove --purge mongodb-org
sudo rm -rf /var/lib/mongodb
sudo rm -rf /var/log/mongodb
```

### macOS

```bash
brew uninstall mongodb-community
brew services stop mongodb-community
```

## Häufig gestellte Fragen

### Was ist der Standardport von MongoDB?

Der Standardport ist 27017. Sie können einen anderen Port mit dem Parameter `--port` angeben.

### Wo speichert MongoDB Daten?

Standardmäßig:
- Linux/macOS: `/data/db`
- Windows: `C:\data\db`

### Wie überprüfe ich den MongoDB-Status?

```bash
sudo systemctl status mongod
```
