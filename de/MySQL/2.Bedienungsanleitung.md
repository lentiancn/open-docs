# MySQL Nutzungsanleitung

Grundlegende MySQL-Operationen.

---

## Verbinden

```bash
mysql -u root -p
```

## Grundbefehle

```sql
SHOW DATABASES;
CREATE DATABASE mydb;
USE mydb;
SHOW TABLES;
```

## CRUD

```sql
INSERT INTO users (name) VALUES ('John');
SELECT * FROM users;
UPDATE users SET name='Jane' WHERE id=1;
DELETE FROM users WHERE id=1;
```
