# Angular Обзор

Современная веб-платформа разработки.

## Что такое Angular?

Angular — это платформа разработки для создания мобильных и десктопных веб-приложений, разработанная Google. Использует TypeScript/JavaScript и другие языки.

### Основные функции

- **Компонентный** - Архитектура на основе компонентов
- **TypeScript** - Поддержка строгой типизации
- **Двустороннее связывание** - Упрощённая синхронизация данных
- **Внедрение зависимостей** - Мощная система управления зависимостями
- **Роутинг** - Полное решение для маршрутизации
- **Формы** - Реактивные и шаблонные формы
- **HTTP-клиент** - Полные HTTP-сервисы
- **Серверный рендеринг** - Поддержка SSR
