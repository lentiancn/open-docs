# Element Plus Guía de Uso

Esta guía explica cómo usar los componentes de Element Plus.

## Inicio Rápido

```vue
<template>
  <div>
    <el-button type="primary">Botón Primary</el-button>
    <el-input v-model="input" placeholder="Texto" />
  </div>
</template>

<script setup>
import { ref } from 'vue'
const input = ref('')
</script>
```

## Componentes Comunes

### Botones

```vue
<el-button>Default</el-button>
<el-button type="primary">Primary</el-button>
<el-button type="success">Success</el-button>
<el-button type="danger">Danger</el-button>
```

### Formulario

```vue
<el-form :model="form" :rules="rules" ref="formRef">
  <el-form-item label="Nombre" prop="name">
    <el-input v-model="form.name"></el-input>
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="onSubmit">Enviar</el-button>
  </el-form-item>
</el-form>

<script setup>
import { reactive, ref } from 'vue'
const formRef = ref(null)
const form = reactive({ name: '' })
const rules = { name: [{ required: true, message: 'Required', trigger: 'blur' }] }
const onSubmit = () => formRef.value.validate()
</script>
```

### Tabla

```vue
<el-table :data="tableData">
  <el-table-column prop="name" label="Nombre"></el-table-column>
  <el-table-column prop="age" label="Edad"></el-table-column>
</el-table>

<script setup>
const tableData = [
  { name: 'Tom', age: 20 },
  { name: 'Jerry', age: 18 }
]
</script>
```

### Diálogo

```vue
<el-button @click="visible = true">Abrir</el-button>
<el-dialog v-model="visible" title="Título">
  <span>Contenido</span>
</el-dialog>

<script setup>
import { ref } from 'vue'
const visible = ref(false)
</script>
```

## Iconos

```vue
<el-icon><Plus /></el-icon>
<el-button :icon="Search">Buscar</el-button>

<script setup>
import { Plus, Search } from '@element-plus/icons-vue'
</script>
```
