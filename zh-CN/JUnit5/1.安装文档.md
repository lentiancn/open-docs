# JUnit 5 安装文档

在所有主流平台上安装 JUnit 5 的完整指南。

---

## 什么是 JUnit 5？

JUnit 5 是 JUnit 的下一代产品，由三个子项目组成：JUnit Platform、JUnit Jupiter 和 JUnit Vintage。

**注意**：JUnit 5 需要 Java 8 或更高版本。

---

## Maven

### 添加依赖

```xml
<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter</artifactId>
    <version>5.10.1</version>
    <scope>test</scope>
</dependency>
```

### pom.xml 配置

```xml
<build>
    <plugins>
        <plugin>
            <groupId>org.apache.maven.plugins</groupId>
            <artifactId>maven-surefire-plugin</artifactId>
            <version>3.2.3</version>
        </plugin>
    </plugins>
</build>
```

---

## Gradle

### 添加依赖

```groovy
testImplementation 'org.junit.jupiter:junit-jupiter:5.10.1'
```

### build.gradle 配置

```groovy
test {
    useJUnitPlatform()
}
```

---

## Eclipse

### 使用 Eclipse Marketplace

1. 进入 **Help > Eclipse Marketplace**
2. 搜索 **JUnit**
3. 安装 **JUnit**

---

## IntelliJ IDEA

### 使用 Maven/Gradle

添加 Maven 或 Gradle 依赖后，JUnit 5 自动可用。
