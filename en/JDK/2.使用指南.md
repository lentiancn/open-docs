# JDK Usage Guide

This guide covers how to use JDK for Java development.

## Quick Start

### Compile and Run

```bash
# Compile
javac HelloWorld.java

# Run
java HelloWorld
```

### Single File Execution (JDK 11+)

```bash
# Run directly
java HelloWorld.java
```

## Basic Commands

### javac - Java Compiler

```bash
# Compile single file
javac MyClass.java

# Compile multiple files
javac *.java

# Compile with specific encoding
javac -encoding UTF-8 MyClass.java

# Compile to specific directory
javac -d bin MyClass.java

# Show all warnings
javac -Xlint MyClass.java
```

### java - Java Launcher

```bash
# Run class
java MyClass

# Run with arguments
java MyClass arg1 arg2

# Set classpath
java -classpath "lib/*:." MyClass

# Set memory
java -Xmx2g -Xms512m MyClass
```

### jar - Java Archive

```bash
# Create JAR
jar cf myapp.jar -C bin .

# Create JAR with manifest
jar cfm myapp.jar manifest.txt -C bin .

# Extract JAR
jar xf myapp.jar

# List JAR contents
jar tf myapp.jar
```

### javadoc - Documentation Generator

```bash
# Generate documentation
javadoc -d docs -sourcepath src *.java

# Generate documentation with custom title
javadoc -d docs -header "My App" *.java
```

### jdb - Java Debugger

```bash
# Start debugger
jdb MyClass

# Set breakpoints
stop at MyClass:10

# Run
run

# Step through
next
step
```

## JVM Options

### Memory Options

```bash
# Heap size
-Xmx2g        # Maximum heap
-Xms512m      # Initial heap
-Xss256k      # Stack size

# Garbage collection
-XX:+UseG1GC  # G1 garbage collector
-XX:+UseParallelGC # Parallel GC
```

### Performance Options

```bash
# JIT compiler
-XX:+TieredCompilation

# Class data sharing
-Xshare:on

# String deduplication
-XX:+UseStringDeduplication
```

## IDE Integration

### IntelliJ IDEA

1. File → Project Structure → SDKs
2. Add JDK installation path
3. Set Project SDK

### VS Code

1. Install "Extension Pack for Java"
2. Configure JAVA_HOME in settings

### Eclipse

1. Window → Preferences → Java → Installed JREs
2. Add Standard VM
3. Set as default

## Modules (Java 9+)

### List Modules

```bash
java --list-modules
```

### Module Options

```bash
# Export package
--export-module java.base/java.lang

# Open module
--open-module java.base
```

## Security

### Keytool

```bash
# Generate key pair
keytool -genkeypair -alias mykey -keyalg RSA -keystore keystore.jks

# List certificates
keytool -list -keystore keystore.jks

# Export certificate
keytool -export -alias mykey -file cert.crt -keystore keystore.jks
```

## Profiling

### jcmd

```bash
# List running Java processes
jcmd

# Get process info
jcmd <pid> VM.flags

# Trigger GC
jcmd <pid> GC.run
```

### jvisualvm

```bash
# Launch visual VM
jvisualvm
```

## Best Practices

### 1. Use LTS for Production

```bash
# Recommended: JDK 17 or 21 for production
```

### 2. Set Appropriate Memory

```bash
# For web applications
java -Xmx4g -Xms2g -XX:+UseG1GC MyApp
```

### 3. Enable GC Logs

```bash
# Enable GC logging
-Xlog:gc*:file=gc.log:time,uptime,level,tags
```
