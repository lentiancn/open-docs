# Руководство пользователя OpenClaw

Это руководство охватывает ежедневное использование и основные функции OpenClaw.

## Основные понятия

### Gateway

Gateway — это основной компонент OpenClaw, отвечающий за управление сессиями, маршрутизацию сообщений и подключения каналов. Вам нужно запустить только один процесс Gateway, и он автоматически обрабатывает все запросы на подключение каналов.

### Каналы

Каналы — это приложения для чата, которые вы используете для общения с ИИ-помощником. OpenClaw поддерживает одновременное подключение нескольких каналов, включая WhatsApp, Telegram, Discord, iMessage и другие.

### Агенты

Агенты — это экземпляры ИИ-помощников. У каждого агента есть собственные рабочая область и сессии. Вы можете настроить несколько агентов для независимой обработки разных задач.

## Повседневные операции

### Запуск и остановка Gateway

```bash
# Запустить Gateway (фоновый сервис)
openclaw gateway start

# Остановить Gateway
openclaw gateway stop

# Перезапустить Gateway
openclaw gateway restart

# Проверить состояние
openclaw gateway status

# Запустить в foreground (для отладки)
openclaw gateway --port 18789
```

### Отправка сообщений

```bash
# Отправить личное сообщение
openclaw message send --target +15555550123 --message "Привет"

# Отправить в группу
openclaw message send --target "Название группы" --message "Привет всем"
```

### Управление конфигурацией

```bash
# Просмотреть текущую конфигурацию
openclaw config get

# Установить значение конфигурации
openclaw config set agents.defaults.model "anthropic/claude-sonnet-4-5"

# Удалить значение конфигурации
openclaw config unset channels.whatsapp.enabled
```

## Управление конфигурацией

### Расположение файла конфигурации

Файл конфигурации находится в `~/.openclaw/openclaw.json`.

### Горячая перезагрузка

OpenClaw поддерживает горячую перезагрузку конфигурации — можно изменять файл конфигурации без ручного перезапуска Gateway. Большинство изменений конфигурации применяются автоматически.

### Примеры часто используемой конфигурации

#### Настройка разрешённых источников сообщений

```json5
{
  channels: {
    whatsapp: {
      allowFrom: ["+15555550123"]
    }
  }
}
```

#### Настройка правил упоминания в группах

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

#### Настройка нескольких агентов

```json5
{
  agents: {
    list: [
      { id: "home", default: true, workspace: "~/.openclaw/workspace-home" },
      { id: "work", workspace: "~/.openclaw/workspace-work" }
    ]
  }
}
```

## Управление каналами

### Добавление новых каналов

```bash
# Войти в WhatsApp
openclaw channels login whatsapp

# Войти в Telegram
openclaw channels login telegram
```

### Просмотр подключённых каналов

```bash
openclaw channels list
```

## Расширенные функции

### Cron-задачи

Настройка запланированных задач:

```json5
{
  cron: {
    enabled: true,
    maxConcurrentRuns: 2
  }
}
```

### Heartbeat

Настройка периодических проверок:

```json5
{
  agents: {
    defaults: {
      heartbeat: {
        every: "30m",
        target: "last"
      }
    }
  }
}
```

### Режим песочницы

Запуск агентов в изолированных контейнерах Docker:

```json5
{
  agents: {
    defaults: {
      sandbox: {
        mode: "non-main",
        scope: "agent"
      }
    }
  }
}
```

### Webhooks

Настройка HTTP webhook-эндпоинтов:

```json5
{
  hooks: {
    enabled: true,
    token: "ваш-секретный-токен",
    path: "/hooks"
  }
}
```

## Устранение неполадок

### Просмотр логов

```bash
# Просмотреть логи Gateway
openclaw logs

# Следить за логами в реальном времени
openclaw logs -f
```

### Проверка здоровья

```bash
openclaw health
```

### Диагностика проблем

```bash
openclaw doctor
```

## Интерфейс управления

OpenClaw предоставляет веб-интерфейс управления, где вы можете:

- Отправлять и получать сообщения
- Управлять конфигурацией
- Просматривать историю сессий
- Управлять мобильными узлами
- Отслеживать состояние Gateway

Доступ по адресу: `http://127.0.0.1:18789/`
