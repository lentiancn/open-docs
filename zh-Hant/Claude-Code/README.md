# Claude Code

下一代編程助手。

## 簡介

Claude Code 是 Anthropic 公司開發的 AI 編程助手，基於 Claude 大語言模型，專為代碼生成、調試和開發任務優化。

### 主要特性

- 代碼生成 - 根據描述生成高質量代碼
- 智能補全 - 上下文感知的代碼補全
- 調試輔助 - 識別並修復代碼問題
- 解釋代碼 - 幫助理解複雜代碼邏輯
