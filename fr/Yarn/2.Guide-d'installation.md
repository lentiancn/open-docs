# Guide d'Installation

## Configuration Requise

- Node.js: 16.0+

---

## Installation

### Corepack (Recommandé)

```bash
corepack enable
corepack prepare yarn@stable --activate
```

### npm

```bash
npm install -g yarn
```

---

## Vérifier

```bash
yarn --version
```
