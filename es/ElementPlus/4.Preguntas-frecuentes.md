# Preguntas Frecuentes de Element Plus

Esta página enumera los problemas comunes y sus soluciones al usar Element Plus.

## Problemas de Instalación

### Instalación Fallida

Síntomas: npm install o yarn add fallan

Soluciones:

1. Verifica la versión de Node.js, asegúrate de que sea mayor o igual a 14.0.0
2. Limpia la caché de npm:
   npm cache clean --force
3. Elimina node_modules y package-lock.json, luego reinstala

### Errores de Tipo TypeScript

Síntomas: Errores de tipo al importar componentes

Soluciones:

1. Asegúrate de que el plugin Volar esté instalado
2. Añade en tsconfig.json:
   {
     "compilerOptions": {
       "types": ["element-plus/global"]
     }
   }

## Problemas de Uso

### Estilos de Componentes No se Aplican

Síntomas: Los estilos de componentes no coinciden con las expectativas

Soluciones:

1. Asegúrate de que el CSS esté correctamente importado:
   import 'element-plus/dist/index.css'
2. Verifica si hay sobrescrituras de estilos globales

### Iconos No se Muestran

Síntomas: Los iconos se muestran en blanco o como cuadrados

Soluciones:

1. Instala la biblioteca de iconos:
   npm install @element-plus/icons-vue

2. Registra los iconos globalmente:
   import * as icons from '@element-plus/icons-vue'
   for (const [key, component] of Object.entries(icons)) {
     app.component(key, component)
   }

### Importación bajo Demanda No Funciona

Síntomas: El tamaño del bundle sigue siendo grande

Soluciones:

1. Verifica que la configuración del plugin sea correcta
2. Asegúrate de usar el resolver correcto

### Internacionalización No Funciona

Síntomas: El texto del componente no cambia después de cambiar el idioma

Soluciones:

1. Importa correctamente el paquete de idioma:
   import es from 'element-plus/es/locale/lang/es'
2. Configura el locale:
   app.use(ElementPlus, { locale: es })

## Problemas de Componentes

### Problemas de Tabla

#### Ordenamiento de Tabla No Funciona

Asegúrate de que el atributo prop esté configurado:

<el-table-column prop="date" sortable />

#### Columnas Fijas de Tabla Desalineadas

Asegúrate de que la tabla tenga una altura fija:

<el-table :data="data" height="300" />

### Problemas de Formulario

#### Validación de Formulario No se Activa

Asegúrate de que el-form-item tenga el atributo prop:

<el-form-item label="Nombre" prop="name">
  <el-input v-model="ruleForm.name" />
</el-form-item>

### Problemas de Diálogo

#### Problemas de Desplazamiento del Diálogo

Cuando el contenido del diálogo es demasiado largo, asegúrate de agregar desplazamiento:

<el-dialog>
  <div style="max-height: 60vh; overflow-y: auto;">
    Contenido
  </div>
</el-dialog>

## Problemas de Estilos

### Conflictos de Estilos

Síntomas: Los estilos personalizados se sobrescriben

Soluciones:

1. Usa selectores más específicos
2. Usa ::v-deep o :deep()

### Color de Tema No se Aplica

Verifica si las variables CSS están correctamente sobrescritas:

:root {
  --el-color-primary: #409eff;
}

Asegúrate de que se cargue después de los estilos de element-plus.

## Problemas de Rendimiento

### Lentitud en la Página

Para grandes cantidades de datos:

1. Usa el-table-v2 (tabla virtual)
2. Usa paginación para cargar datos
3. Evita componentes complejos en las tablas

### Tamaño del Bundle Demasiado Grande

1. Usa importación bajo demanda
2. Elimina plugins no usados

## Problemas de Versión de Vue

### No Compatible con Vue 2

Element Plus solo soporta Vue 3. Si necesitas Vue 2, usa Element UI.

### Composition API No Soportada

Asegúrate de usar Vue 3.2 o superior para mejor soporte de Composition API.
