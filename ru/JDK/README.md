# JDK

Комплект разработки Java

## Введение

JDK (Java Development Kit) — это комплект разработки программного обеспечения для языка программирования Java, образующий основу любой Java-разработки. JDK включает среду выполнения Java (JRE), компилятор Java (javac), виртуальную машину Java (JVM) и обширные библиотеки классов и инструменты Java.

## Основные версии

- **JDK 25** — Последняя версия с новыми функциями и улучшениями производительности
- **JDK 21** — Последняя версия с долгосрочной поддержкой (LTS)
- **JDK 17** — Зрелая версия LTS, широко используемая
- **JDK 11** — Версия LTS, ознаменовавшая разделение платных функций Java

## Основные компоненты

- **javac** — Компилятор Java, компилирует исходный код в байт-код
- **java** — Команда запуска программ Java
- **jvm** — Виртуальная машина Java, выполняет байт-код
- **jar** — Инструмент архива Java
- **javadoc** — Инструмент генерации документации API
- **jdb** — Отладчик Java

## Различия между версиями

Различные основные версии JDK имеют значительные различия в установке, использовании и функциональности:

- **JDK 8**: Последняя полностью бесплатная версия Oracle JDK для коммерческого использования
- **JDK 11**: Ввела var, новый HTTP-клиент, ZGC (экспериментальный)
- **JDK 17**: Последняя бесплатная версия Oracle JDK по NFTC
- **JDK 21**: Виртуальные потоки, сопоставление с образцом, записи (стабильные)
- **JDK 25**: Новейшие функции, более короткий цикл поддержки
