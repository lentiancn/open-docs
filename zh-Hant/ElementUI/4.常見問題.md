# Element UI 常見問題

## 安裝問題

### Q: 安裝失敗？

**解決方案：**
1. 檢查 Node.js 版本
2. 清理 npm 快取
```bash
npm cache clean --force
```

### Q: Vue 版本不相容？

**解決方案：**
Element UI 2.x 需要 Vue 2.6.0+

## 使用問題

### Q: 元件不顯示？

**解決方案：**
確認已正確引入元件：
```javascript
import { Button } from 'element-ui'
Vue.use(Button)
```

### Q: 樣式不生效？

**解決方案：**
確認已引入樣式文件：
```javascript
import 'element-ui/lib/theme-chalk/index.css'
```

### Q: 表單驗證不生效？

**解決方案：**
1. 引入 Form 和 FormItem
2. 設置 prop 屬性
