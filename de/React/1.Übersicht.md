# React Übersicht

> Bibliothek für Web- und native Benutzeroberflächen

---

## Projektübersicht

React ist eine JavaScript-Bibliothek zum Erstellen von Benutzeroberflächen, entwickelt von Meta. Sie verwendet einen komponentenbasierten Ansatz und Virtual DOM.

---

## Hauptmerkmale

- **Komponentenbasiert**: Wiederverwendbare UI-Komponenten
- **Virtual DOM**: Effiziente DOM-Aktualisierungen
- **JSX**: HTML-ähnliche Syntax in JavaScript
- **Einweg-Datenfluss**: Vorhersehbare Zustandsverwaltung
