# Spring Boot 安装指南

本指南涵盖 Spring Boot 开发的环境设置，包括 Java、Maven 和 Gradle。

## 前提条件

### Java 安装

Spring Boot 3.x 需要 Java 17 或更高版本。Spring Boot 2.x 需要 Java 8 或更高版本。

**Windows：**
1. 从 [Oracle](https://www.oracle.com/java/technologies/downloads/) 或 [Adoptium](https://adoptium.net/) 下载 JDK
2. 运行安装程序
3. 设置环境变量：
   ```
   JAVA_HOME=C:\Program Files\Java\jdk-17
   PATH=%JAVA_HOME%\bin;%PATH%
   ```

**macOS：**
```bash
# 使用 Homebrew
brew install openjdk@17

# 添加到 PATH
echo 'export JAVA_HOME=$(/usr/libexec/java_home)' >> ~/.zshrc
source ~/.zshrc
```

**Linux：**
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install openjdk-17-jdk

# 设置 JAVA_HOME
echo 'export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64' >> ~/.bashrc
source ~/.bashrc
```

验证安装：
```bash
java -version
```

### Maven 安装

**Windows：**
1. 从 [Apache Maven](https://maven.apache.org/download.cgi) 下载
2. 解压到文件夹（例如 C:\apache-maven）
3. 设置环境变量：
   ```
   MAVEN_HOME=C:\apache-maven
   PATH=%MAVEN_HOME%\bin;%PATH%
   ```

**macOS：**
```bash
brew install maven
```

**Linux：**
```bash
sudo apt install maven
```

验证安装：
```bash
mvn -version
```

### Gradle 安装

**Windows：**
1. 从 [Gradle](https://gradle.org/install/) 下载
2. 解压并添加到 PATH

**macOS：**
```bash
brew install gradle
```

**Linux：**
```bash
sudo install gradle
```

验证安装：
```bash
gradle -version
```

## IDE 设置

### IntelliJ IDEA（推荐）
- 从 [JetBrains](https://www.jetbrains.com/idea/download/) 下载
- 确保在项目结构中配置了 JDK
- 安装 Spring Boot Assistant 插件

### VS Code
- 安装 Java 扩展包
- 安装 Spring Boot 扩展

## 使用 Spring Initializr 快速开始

1. 访问 [start.spring.io](https://start.spring.io/)
2. 选择：
   - 项目：Maven 或 Gradle
   - 语言：Java
   - Spring Boot 版本
   - 项目元数据
3. 添加依赖（Spring Web、Spring Data JPA 等）
4. 点击"Generate"下载项目

## 验证环境配置

```bash
# 检查 Java
java -version

# 检查 Maven
mvn -version

# 检查 Gradle（如果使用 Gradle）
gradle -version

# 创建简单的 Spring Boot 项目并运行
curl https://start.spring.io/starter.zip -o demo.zip
unzip demo.zip -d demo
cd demo
./mvnw spring-boot:run  # 或 ./gradlew bootRun
```
