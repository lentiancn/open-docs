# Element UI 安装指南

> 完整的环境配置和安装教程

---

## 环境要求

### 开发环境要求

| 环境 | 最低版本 | 推荐版本 |
|------|----------|----------|
| Node.js | 8.x | 10.x 或 12.x |
| npm | 5.x | 6.x |
| Vue | 2.6.0 | 2.6.14 |
| 构建工具 | Webpack 4.x | Webpack 4.x 或 5.x |

### 浏览器兼容性

- Chrome >= 49
- Firefox >= 45
- Safari >= 10
- Edge >= 14
- IE 11（有限支持）

## 安装步骤

### 使用 npm 安装（推荐）

```bash
# 进入 Vue 2 项目目录
cd your-vue-project

# 安装 Element UI
npm install element-ui --save
```

### 使用 yarn 安装

```bash
yarn add element-ui
```

### 指定版本安装

```bash
# 安装最新 2.x 版本
npm install element-ui@latest

# 安装特定版本
npm install element-ui@2.15.13
```

## 在 Vue 2 项目中引入

### 完整引入（适合小型项目）

```javascript
// main.js
import Vue from 'vue'
import App from './App.vue'

// 引入 Element UI
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)

new Vue({
  render: h => h(App)
}).$mount('#app')
```

### 按需引入（推荐）

#### 1. 安装 babel-plugin-component

```bash
npm install babel-plugin-component -D
```

#### 2. 配置 babel

```json
{
  "presets": [
    ["env", {
      "modules": false,
      "targets": {
        "browsers": ["> 1%", "last 2 versions", "not ie <= 8"]
      }
    }]
  ],
  "plugins": [
    ["component", {
      "libraryName": "element-ui",
      "styleLibraryName": "theme-chalk"
    }]
  ]
}
```

#### 3. 按需引入组件

```javascript
import { Button, Form, FormItem, Input, Table, TableColumn } from 'element-ui'

Vue.use(Button)
Vue.use(Form)
Vue.use(FormItem)
Vue.use(Input)
Vue.use(Table)
Vue.use(TableColumn)
```

## 国际化配置

### 全局中文配置

```javascript
import ElementUI from 'element-ui'
import locale from 'element-ui/lib/locale/lang/zh-CN'

Vue.use(ElementUI, {
  locale
})
```
