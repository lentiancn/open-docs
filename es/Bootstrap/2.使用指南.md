# Bootstrap 5 Guía de Uso

Esta guía detalla cómo usar Bootstrap 5 para construir sitios web responsivos.

## Inicio Rápido

### Plantilla Básica

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Página Bootstrap 5</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <!-- Área de contenido -->
    <div class="container mt-5">
        <h1>¡Hola, Bootstrap 5!</h1>
        <p class="lead">Esta es una página responsiva.</p>
    </div>

    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## Sistema de Diseño

### Contenedores

Bootstrap 5 proporciona dos clases de contenedor:

| Clase | Descripción |
|-------|-------------|
| `.container` | Contenedor de ancho fijo |
| `.container-fluid` | Contenedor de ancho completo |

```html
<!-- Contenedor de ancho fijo -->
<div class="container">
    <p>Contenido centrado con márgenes</p>
</div>

<!-- Contenedor de ancho completo -->
<div class="container-fluid">
    <p>Ocupar ancho completo</p>
</div>
```

### Sistema de Grid

Bootstrap usa un sistema de grid de 12 columnas:

```html
<div class="container">
    <div class="row">
        <div class="col-12 col-md-6 col-lg-4">
            <p>Columna responsiva</p>
        </div>
        <div class="col-12 col-md-6 col-lg-4">
            <p>Columna responsiva</p>
        </div>
        <div class="col-12 col-md-6 col-lg-4">
            <p>Columna responsiva</p>
        </div>
    </div>
</div>
```

#### Puntos de Rotura del Grid

| Punto de Rotura | Prefijo de Clase | Ancho |
|-----------------|------------------|-------|
| Extra pequeño | col- | < 576px |
| Pequeño | col-sm- | ≥ 576px |
| Mediano | col-md- | ≥ 768px |
| Grande | col-lg- | ≥ 992px |
| Extra grande | col-xl- | ≥ 1200px |
| XXL | col-xxl- | ≥ 1400px |

### Diseño Flex

```html
<div class="d-flex justify-content-between">
    <div>Izquierda</div>
    <div>Derecha</div>
</div>

<div class="d-flex align-items-center">
    <div>Centrado vertical</div>
</div>
```

```html

    Abrir modal
</button>

        </div>

</div>
```

### Tooltip

```html
<button type="button" class="btn btn-secondary" data-bs-toggle="tooltip" data-bs-title="Contenido del tooltip">
    Hover para ver
</button>

<!-- Requiere inicialización -->
<script>
    var tooltipTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="tooltip"]'))
    var tooltipList = tooltipTriggerList.map(function (tooltipTriggerEl) {
        return new bootstrap.Tooltip(tooltipTriggerEl)
    })
</script>
```

### Popover

```html
<button type="button" class="btn btn-secondary" data-bs-toggle="popover" data-bs-title="Título" data-bs-content="Contenido del popover">
    Click para mostrar
</button>

<script>
    var popoverTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="popover"]'))
    var popoverList = popoverTriggerList.map(function (popoverTriggerEl) {
        return new bootstrap.Popover(popoverTriggerEl)
    })
</script>
```

## Componentes JavaScript

### Inicialización

```javascript
// Habilitar todos los tooltips
var tooltipList = [].slice.call(document.querySelectorAll('[data-bs-toggle="tooltip"]'))
var tooltipList = tooltipList.map(function (tooltipTriggerEl) {
    return new bootstrap.Tooltip(tooltipTriggerEl)
})

// Habilitar todos los popovers
var popoverTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="popover"]'))
var popoverList = popoverTriggerList.map(function (popoverTriggerEl) {
    return new bootstrap.Popover(popoverTriggerEl)
})

// Control de modal
var myModal = new bootstrap.Modal(document.getElementById('myModal'))
myModal.show()     // Mostrar
myModal.hide()     // Ocultar

// Control de carrusel
var carousel = new bootstrap.Carousel(document.getElementById('myCarousel'))
carousel.next()     // Siguiente
carousel.prev()    // Anterior
carousel.cycle()   // Iniciar autoplay
carousel.pause()   // Pausar
```

## Mejores Prácticas

### 1. Usar HTML Semántico

```html
<!-- No recomendado -->
<div class="container">
    <div class="row">
        <div class="col">
            <div class="btn btn-primary">Botón</div>
        </div>
    </div>
</div>

<!-- Recomendado -->
<main class="container">
    <section class="row">
        <article class="col">
            <button class="btn btn-primary">Botón</button>
        </article>
    </section>
</main>
```

### 2. Mantener el Código Simple

```html
<!-- No recomendado -->
<div class="container-fluid">
    <div class="row">
        <div class="col-12 col-sm-6 col-md-4 col-lg-3 col-xl-2">
            <div class="p-3 mb-2 bg-primary text-white rounded shadow-lg">
                Contenido
            </div>
        </div>
    </div>
</div>

<!-- Recomendado: usar combinación de clases de utilidad -->
<div class="container">
    <div class="row g-3">
        <!-- g-3 agrega espacio de grid -->
    </div>
</div>
```

### 3. Optimización de Rendimiento

```html

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
```

## Preguntas Frecuentes

### ¿Cómo centrar elementos?

```html

<div class="d-flex justify-content-center align-items-center" style="height: 100px;">
```
