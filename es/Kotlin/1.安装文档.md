# Guía de Instalación de Kotlin

Kotlin es un lenguaje de programación moderno desarrollado por JetBrains. Esta guía cubre los métodos de instalación en varios entornos.

## Información de Versión

| Versión | Estado | Descripción |
|---------|--------|-------------|
| Kotlin 2.x | ✅ Actual | Última versión |
| Kotlin 1.9.x | ✅ Soportado | Versión estable |

## Requisitos del Sistema

### Requisitos de Hardware

| Requisito | Configuración |
|------------|---------------|
| CPU | 1+ núcleos |
| Memoria | 2+ GB |
| Disco | 500+ MB |

### Requisitos de Software

| Software | Versión |
|----------|---------|
| Java | JDK 8 o superior |

## Métodos de Instalación

### Método 1: IntelliJ IDEA (Recomendado)

1. Descarga IntelliJ IDEA: https://www.jetbrains.com/idea/download/
2. Instala la versión Ultimate o Community
3. Kotlin ya viene incluido

### Método 2: Línea de Comandos

#### Windows

```powershell
# Usando Chocolatey
choco install kotlin

# O descarga manual
# Descarga desde https://github.com/JetBrains/kotlin/releases
```

#### macOS

```bash
# Usando Homebrew
brew install kotlin

# O instalación manual
curl -s https://get.sdkman.io | bash
sdk install kotlin
```

#### Linux

```bash
# Ubuntu/Debian
sudo apt install kotlin

# O usando SDKMAN
curl -s https://get.sdkman.io | bash
sdk install kotlin
```

### Método 3: SDKMAN

```bash
# Instalar SDKMAN
curl -s "https://get.sdkman.io" | bash

# Instalar Kotlin
sdk install kotlin

# Verificar
kotlin -version
```

### Método 4: Maven

```xml
<dependencies>
    <dependency>
        <groupId>org.jetbrains.kotlin</groupId>
        <artifactId>kotlin-stdlib</artifactId>
        <version>2.0.0</version>
    </dependency>
</dependencies>
```

### Método 5: Gradle

```groovy
plugins {
    id 'org.jetbrains.kotlin.jvm' version '2.0.0'
}

repositories {
    mavenCentral()
}

dependencies {
    implementation "org.jetbrains.kotlin:kotlin-stdlib"
}
```

### Método 6: Docker

```dockerfile
# Usar imagen del compilador Kotlin
FROM gradle:8.5-jdk-21 AS builder
WORKDIR /app
COPY . .
RUN gradle build

# Imagen de ejecución
FROM openjdk:21
COPY --from=builder /app/build/libs/*.jar /app.jar
ENTRYPOINT ["java", "-jar", "/app.jar"]
```

## Configuración del Entorno

### Configuración de Java

```bash
# Verificar Java
java -version

# Configurar JAVA_HOME
export JAVA_HOME=/path/to/java
```

### Compilación y Ejecución

```bash
# Compilar
kotlinc hello.kt -include-runtime -d hello.jar

# Ejecutar
java -jar hello.jar

# Usar script kotlin
kotlin hello.kts
```

## Configuración del IDE

### IntelliJ IDEA

1. Instalar el complemento Kotlin (generalmente viene preinstalado)
2. Crear un proyecto Kotlin
3. Configurar el SDK de Kotlin

### VS Code

1. Instalar la extensión Kotlin Language Server
2. Instalar el compilador de Kotlin

### Eclipse

1. Instalar el complemento Kotlin (Kotlin IDE for Eclipse)
2. Crear un proyecto Kotlin

## Problemas Comunes

### ¿Errores de compilación?

Verifica la compatibilidad de versiones entre Kotlin y Java.

### ¿Memoria insuficiente?

Aumenta la memoria del heap de JVM:

```bash
kotlinc -J-Xmx2g
```
