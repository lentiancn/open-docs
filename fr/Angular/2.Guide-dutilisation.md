# Guide d'utilisation d'Angular

Ce guide explique comment utiliser Angular pour créer des applications web.

## Démarrage rapide

### Serveur de développement

```bash
# Démarrer le serveur de développement
ng serve

# Démarrer avec un port spécifique
ng serve --port 4201

# Ouvrir automatiquement le navigateur
ng serve --open

# Activer le hot reload
ng serve --hmr
```

### Build

```bash
# Build de développement
ng build

# Build de production
ng build --configuration production

# Build avec statistiques
ng build --stats-json
```

## Composants

### Créer un composant

```bash
# Générer un composant
ng generate component components/user-list

# Forme courte
ng g c components/user-list
```

### Exemple de composant

```typescript
import { Component } from '@angular/core';

@Component({
  selector: 'app-user-list',
  template: `
    <h1>Liste des utilisateurs</h1>
    <ul>
      <li *ngFor="let user of users">{{ user.name }}</li>
    </ul>
  `,
  standalone: true
})
export class UserListComponent {
  users = [
    { name: 'Jean' },
    { name: 'Marie' }
  ];
}
```

## Services

### Créer un service

```bash
ng generate service services/user
# ou
ng g s services/user
```

### Exemple de service

```typescript
import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class UserService {
  constructor(private http: HttpClient) {}

  getUsers(): Observable<User[]> {
    return this.http.get<User[]>('/api/users');
  }
}
```

## Routage

### Configurer les routes

```typescript
// app.routes.ts
import { Routes } from '@angular/router';

export const routes: Routes = [
  { path: '', redirectTo: '/home', pathMatch: 'full' },
  { path: 'home', component: HomeComponent },
  { path: 'users', component: UserListComponent },
  { path: 'users/:id', component: UserDetailComponent },
  { path: '**', component: NotFoundComponent }
];
```

### Navigation

```typescript
import { RouterLink, RouterOutlet } from '@angular/router';

@Component({
  imports: [RouterLink, RouterOutlet],
  template: `
    <nav>
      <a routerLink="/home">Accueil</a>
      <a routerLink="/users">Utilisateurs</a>
    </nav>
    <router-outlet></router-outlet>
  `
})
export class AppComponent {}
```

## Formulaires

### Formulaires pilotés par le modèle

```typescript
import { FormsModule } from '@angular/forms';

@Component({
  imports: [FormsModule],
  template: `
    <input [(ngModel)]="name" name="name">
    <button (click)="submit()">Soumettre</button>
  `
})
export class LoginComponent {
  name = '';
}
```

### Formulaires réactifs

```typescript
import { ReactiveFormsModule, FormBuilder, FormGroup, Validators } from '@angular/forms';

@Component({
  imports: [ReactiveFormsModule],
  template: `
    <form [formGroup]="form" (ngSubmit)="submit()">
      <input formControlName="email">
      <input formControlName="password" type="password">
      <button type="submit">Soumettre</button>
    </form>
  `
})
export class LoginComponent {
  form: FormGroup;

  constructor(private fb: FormBuilder) {
    this.form = this.fb.group({
      email: ['', [Validators.required, Validators.email]],
      password: ['', [Validators.required, Validators.minLength(6)]]
    });
  }
}
```

## Client HTTP

### Configuration

```typescript
// app.config.ts
import { provideHttpClient } from '@angular/common/http';

export const appConfig: ApplicationConfig = {
  providers: [
    provideHttpClient()
  ]
};
```

### Appel API

```typescript
import { HttpClient } from '@angular/common/http';

@Component({})
export class UserComponent {
  constructor(private http: HttpClient) {}

  getUsers() {
    this.http.get('/api/users').subscribe(data => {
      console.log(data);
    });
  }
}
```

## Gestion d'état

### Signals (Angular 16+)

```typescript
import { signal, computed } from '@angular/core';

@Component({})
export class CounterComponent {
  count = signal(0);
  double = computed(() => this.count() * 2);

  increment() {
    this.count.update(c => c + 1);
  }
}
```

## Bonnes pratiques

### 1. Utiliser les composants autonomes

```typescript
@Component({
  standalone: true,
  imports: [CommonModule],
  ...
})
export class MyComponent {}
```

### 2. Utiliser Signals pour l'état

```typescript
# Préféré
count = signal(0);

# Éviter
count = 0;
```

### 3. Chargement lazy

```typescript
{
  path: 'admin',
  loadComponent: () => import('./admin/admin.component')
}
```
