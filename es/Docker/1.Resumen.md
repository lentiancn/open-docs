# Docker Resumen

La plataforma de contenedorización más popular del mundo.

## ¿Qué es Docker?

Docker es una plataforma de código abierto para desarrollar, desplegar y ejecutar aplicaciones. A través de la tecnología de contenedorización, Docker puede empaquetar aplicaciones y sus dependencias en contenedores ligeros y portátiles.

### Características Principales

- Ligero - Los contenedores comparten el kernel, inicio rápido
- Portátil - Construye una vez, ejecuta en cualquier lugar
- Aislamiento - Los contenedores están aislados entre sí
- Eficiente - Más eficiente en recursos que las máquinas virtuales

## Conceptos Clave

- Imagen - Plantilla de solo lectura para crear contenedores
- Contenedor - Instancia en ejecución de una imagen
- Repositorio - Lugar para almacenar imágenes
- Dockerfile - Script que define cómo construir una imagen
