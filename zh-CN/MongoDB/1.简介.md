# MongoDB 简介

## 什么是 MongoDB？

MongoDB 是一个基于文档的 NoSQL 数据库，使用 JSON 风格的 BSON（二进制 JSON）格式存储数据。与传统的关系型数据库不同，MongoDB 采用灵活的 schema 设计，允许文档拥有不同的字段和结构，非常适合快速迭代的开发模式和大规模数据存储需求。

## 核心特性

### 文档存储
MongoDB 以文档为基本单位存储数据，每个文档可以包含嵌套的数组和对象。这种灵活的文档模型使得开发者可以直接存储复杂的数据结构，无需事先定义严格的表结构。

### 高性能
通过支持嵌入式文档和索引，MongoDB 提供了极高的读写性能。内置的 WiredTiger 存储引擎支持压缩和并发控制，确保在高频访问场景下依然保持流畅。

### 水平扩展
MongoDB 原生支持分片（Sharding）功能，可以将数据分散存储在多台服务器上，轻松实现 PB 级别的数据存储和横向扩展。

### 高可用
副本集（Replica Set）机制提供了自动故障转移和数据冗余，确保数据库的高可用性。即使某个节点发生故障，系统也能自动切换到备用节点，保证服务不中断。

### 丰富的查询语言
MongoDB 提供强大的查询语言，支持复杂的聚合管道、地理空间查询和全文搜索，满足各种业务场景的需求。

## 应用场景

MongoDB 广泛应用于以下场景：内容管理系统、移动应用后端、实时分析、物联网数据存储、电商平台订单管理等需要灵活数据模型和高扩展性的应用。

## 版本与发行版

MongoDB 分为社区版（Community Edition）和企业版（Enterprise Edition）。社区版免费开源，适合大多数应用场景；企业版提供高级安全功能、技术支持和管理工具，适合对可靠性要求更高的企业环境。
