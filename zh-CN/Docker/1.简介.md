# Docker 简介

容器化平台。

## 什么是 Docker？

Docker 是一个开源的容器化平台，用于开发、部署和运行应用程序。Docker 使开发者能够将应用程序及其依赖项打包到轻量级、可移植的容器中，从而实现快速部署和一致的环境。

### 核心特性

- **容器化**：将应用程序及其依赖打包为容器
- **轻量级**：共享操作系统内核，启动快速
- **可移植**：一次构建，随处运行
- **版本控制**：支持容器版本管理
- **镜像仓库**：Docker Hub 公共仓库
- **网络**：容器网络配置
- **数据卷**：持久化存储
- **Compose**：多容器编排

### 核心概念

- **镜像（Image）**：只读模板，用于创建容器
- **容器（Container）**：镜像的运行实例
- **仓库（Repository）**：存储镜像的地方
- **Dockerfile**：定义镜像构建脚本

- **应用容器化**：将传统应用容器化
