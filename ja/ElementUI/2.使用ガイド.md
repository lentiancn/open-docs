# Element UI 使用ガイド

Element UI を使用するガイド。

---

## ボタン

```vue
<el-button>ボタン</el-button>
<el-button type="primary">プライマリ</el-button>
```

---

## 入力

```vue
<el-input v-model="input"></el-input>
```

---

## テーブル

```vue
<el-table :data="tableData">
  <el-table-column prop="name" label="名前"></el-table-column>
</el-table>
```

---

## アイコン

```vue
<i class="el-icon-edit"></i>
```
