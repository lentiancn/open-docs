# FantasyTalking - 常見問題

常見問題解答。

## 安裝問題

### Q：安裝依賴失敗？

**解決方案：**
1. 升級 pip：`pip install --upgrade pip`
2. 使用國內鏡像：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```
3. 單獨安裝失敗的包

### Q：CUDA 不可用？

**解決方案：**
1. 確認 NVIDIA 驅動已安裝：`nvidia-smi`
2. 安裝匹配版本的 PyTorch：
```bash
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
```
3. 驗證 CUDA 可用：
```python
import torch
print(torch.cuda.is_available())
```

### Q：記憶體不足？

**解決方案：**
1. 使用更小的輸入圖片
2. 減少 batch size
3. 使用 CPU 模式（不推薦，速度慢）

## 使用問題

### Q：生成失敗，提示找不到人臉？

**解決方案：**
1. 使用包含清晰人臉的圖片
2. 確保臉部朝前，無遮擋
3. 檢查圖片格式是否正確

### Q：影片生成很慢？

**解決方案：**
1. 使用 GPU 加速
2. 降低輸出解析度
3. 減少輸出幀數

### Q：口型不同步？

**解決方案：**
1. 確保音訊清晰無噪音
2. 檢查音訊採樣率是否正確
3. 嘗試不同的同步參數

### Q：輸出影片模糊？

**解決方案：**
1. 使用人臉增強器：`--face_enhancer realusf`
2. 提高輸出解析度
3. 使用更清晰的源圖片

### Q：人臉變形/扭曲？

**解決方案：**
1. 使用正面免冠照片
2. 確保圖片中只有一個人臉
3. 調整人臉檢測參數

## 模型問題

### Q：模型下載失敗？

**解決方案：**
1. 使用代理或鏡像
2. 手動下載模型檔案
3. 檢查網路連接

### Q：模型載入失敗？

**解決方案：**
1. 確認模型檔案完整
2. 檢查模型路徑是否正確
3. 重新下載模型

## 效能問題

### Q：如何提升生成速度？

**解決方案：**
1. 使用 GPU
2. 啟用混合精度：`--precision fp16`
3. 使用更小的解析度

### Q：記憶體占用過高？

**解決方案：**
1. 減少 batch size
2. 使用更小的模型
3. 關閉不必要的功能

## 輸出問題

### Q：輸出影片無法播放？

**解決方案：**
1. 安裝解碼器：`pip install opencv-python`
2. 使用支持的格式：MP4

### Q：影片檔案過大？

**解決方案：**
1. 降低解析度
2. 降低幀率
3. 使用更高壓縮率
