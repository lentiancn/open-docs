# APISIX 概要

クラウドネイティブ API ゲートウェイ。

## APISIX とは？

APISIX は Apache ソフトウェア財団旗下的クラウドネイティブ API ゲートウェイであり、企業に高性能・高可用性の API 管理サービスを提供します。動的・リアルタイム・スケーラブルな API ゲートウェイとして、APISIX は数百万のコンテナ化デプロイメントの首选ソリューションとなっています。

### コア機能

- **高性能**: Nginx + Lua ベース、単一コアで 10k+ QPS
- **動的ルーティング**: 再起動なしでルートルールを動的に更新
- **プラグインシステム**: 70+ プラグイン、ホットロード対応
- **サービス発見**: Consul、Eureka、Nacos などをサポート
- **負荷分散**: ラウンドロビン、ハッシュ、コンシステントハッシュ
- **セキュリティ**: レート制限、認証、JWT、ファイアウォール

### 技術アーキテク

APISIX は Nginx を底層の HTTP リバースプロキシサーバーとしてを採用し、Lua スクリプトで動的オーケストレーションを実現します。

## ユースケース

- **API 統一管理**: 企業内すべての API を集中管理
- **マイクロサービスゲートウェイ**: 認証、レート制限、ルーティング
- **サービスメッシュイングレス**: Istio、Linkerd との連携
- **API トラフィック制御**: レート制限、サーキットブレーカー
