# Введение в MySQL

## Что такое MySQL?

MySQL — самая популярная в мире база данных с открытым исходным кодом. Благодаря проверенной производительности, надёжности и простоте использования MySQL стал основным выбором для веб-приложений, используется такими известными сайтами, как Facebook, Twitter, YouTube, Yahoo! и многими другими.

Oracle продолжает развивать MySQL, предоставляя новые возможности для веб-приложений, облачных, мобильных и встроенных систем нового поколения.

## Основные характеристики MySQL

### Открытый исходный код и бесплатность
MySQL Community Edition — бесплатная версия с открытым исходным кодом, лицензированная под GNU General Public License (GPL), поддерживается большим активным сообществом разработчиков.

### Кроссплатформенность
MySQL работает на различных операционных системах, включая Windows, Linux, macOS, FreeBSD, Solaris и других. Может работать на различных аппаратных конфигурациях и в программных средах.

### Высокая производительность
MySQL использует оптимизированную архитектуру движка хранения и поддерживает многопоточное выполнение, способное обрабатывать большие объёмы данных и высокопараллельные запросы.

### Надёжность
MySQL обеспечивает полную поддержку транзакций (соответствие ACID), включая фиксацию, откат, восстановление после сбоев и блокировку на уровне строк, обеспечивая целостность данных.

### Простота использования
MySQL предлагает простые инструменты управления и полную документацию с плавной кривой обучения, подходящую для разработчиков любого уровня.

## Продукты и редакции MySQL

### MySQL Community Edition
Бесплатная версия с открытым исходным кодом, подходящая для сред разработки и производства, поддерживается активным сообществом.

### MySQL Enterprise Edition
Корпоративная версия с расширенными функциями, инструментами управления и технической поддержкой для критически важных бизнес-приложений.

### MySQL HeatWave
Облачный сервис со встроенными возможностями генеративного ИИ и машинного обучения, поддерживающий как обработку транзакций, так и аналитику в масштабе озера данных.

### MySQL NDB Cluster
Встроенная база данных реального времени с открытым исходным кодом, разработанная для высокой доступности и быстрого непрерывного доступа к данным в условиях высокой пропускной способности.

## Области применения MySQL

MySQL широко используется в:

- Веб-сайтах и серверной части веб-приложений
- Платформах электронной коммерции
- Хранилищах данных и аналитике
- Системах журналирования
- Системах управления контентом
- Серверной части мобильных приложений
- Встроенных системах

## Экосистема MySQL

Экосистема MySQL включает множество инструментов и продуктов:

- **MySQL Server**: основной сервер базы данных
- **MySQL Workbench**: визуальный инструмент проектирования и управления базами данных
- **MySQL Shell**: интерактивный инструмент командной строки
- **MySQL Router**: промежуточное ПО для маршрутизации баз данных
- **MySQL Connectors**: драйверы баз данных для различных языков программирования

## Информация о лицензии

MySQL Community Edition имеет открытый исходный код и бесплатна по лицензии GPL. Для интеграции MySQL в проприетарные приложения можно приобрести коммерческую лицензию MySQL Enterprise Edition.
