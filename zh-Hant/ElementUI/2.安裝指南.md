# Element UI 安裝指南

> 環境設定

---

## 需求

| 環境 | 最小版本 |
|------|---------|
| Node.js | 8.x |
| npm | 5.x |
| Vue | 2.6.0 |

---

## 安裝

```bash
npm install element-ui --save
```

---

## 引入

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```
