# Hadoop

Фреймворк распределённого хранения и вычислений

## Введение

Apache Hadoop — это фреймворк с открытым исходным кодом для распределённого хранения и вычислений, предназначенный для обработки больших наборов данных. Hadoop предоставляет надёжные и масштабируемые возможности распределённого хранения данных (HDFS) и управления ресурсами (YARN), способный обрабатывать данные от масштаба терабайт до петабайт на обычных серверных кластерах.

## Основные функции

- **HDFS**: Распределённая файловая система с поддержкой массивного хранения данных
- **YARN**: Единое планирование и управление ресурсами
- **MapReduce**: Модель программирования распределённых вычислений
- **Высокая отказоустойчивость**: Автоматическая репликация данных и восстановление после сбоев
- **Высокая масштабируемость**: Линейное масштабирование путём добавления узлов
- **Низкая стоимость**: Работает на обычном оборудовании

## Быстрый старт

### Установка

```bash
# Загрузить Hadoop
wget https://dlcdn.apache.org/hadoop/common/hadoop-3.3.6/hadoop-3.3.6.tar.gz

# Распаковать
tar -xzf hadoop-3.3.6.tar.gz -C /usr/local/

# Настроить переменные среды
export HADOOP_HOME=/usr/local/hadoop
export PATH=$PATH:$HADOOP_HOME/bin
```

### Запуск

```bash
# Отформатировать NameNode (при первом запуске)
hdfs namenode -format

# Запустить HDFS
start-dfs.sh

# Запустить YARN
start-yarn.sh
```

### Запуск примера

```bash
# Запустить пример WordCount
hadoop jar $HADOOP_HOME/share/hadoop/mapreduce/hadoop-mapreduce-examples-3.3.6.jar \
    wordcount /input /output
```
