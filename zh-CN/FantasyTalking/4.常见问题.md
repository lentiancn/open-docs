# FantasyTalking 常见问题

常见问题解答。

## 安装问题

### Q: 安装依赖失败？

**解决方案：**
1. 升级 pip：`pip install --upgrade pip`
2. 使用国内镜像：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```
3. 单独安装失败的包

### Q: CUDA 不可用？

**解决方案：**
1. 确认 NVIDIA 驱动已安装：`nvidia-smi`
2. 安装匹配 CUDA 版本的 PyTorch：
```bash
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
```
3. 验证 CUDA 可用：
```python
import torch
print(torch.cuda.is_available())
```

### Q: 显存不足？

**解决方案：**
1. 减少输入图片尺寸
2. 使用更小的 batch size
3. 使用 CPU 模式（不推荐，速度慢）

## 使用问题

### Q: 生成失败，提示找不到人脸？

**解决方案：**
1. 使用包含清晰人脸的图片
2. 确保脸部朝前，无遮挡
3. 检查图片格式是否正确

### Q: 视频生成很慢？

**解决方案：**
1. 使用 GPU 加速
2. 减少输出分辨率
3. 减少输出帧数

### Q: 口型不同步？

**解决方案：**
1. 确保音频清晰无噪音
2. 检查音频采样率是否正确
3. 尝试不同的同步参数

### Q: 输出视频模糊？

**解决方案：**
1. 使用人脸增强器：`--face_enhancer realusf`
2. 提高输出分辨率
3. 使用更清晰的源图片

### Q: 人脸变形/扭曲？

**解决方案：**
1. 使用正面免冠照片
2. 确保图片中只有一个人脸
3. 调整人脸检测参数

## 模型问题

### Q: 模型下载失败？

**解决方案：**
1. 使用代理或镜像
2. 手动下载模型文件
3. 检查网络连接

### Q: 模型加载失败？

**解决方案：**
1. 确认模型文件完整
2. 检查模型路径是否正确
3. 重新下载模型

## 性能问题

### Q: 如何提升生成速度？

**解决方案：**
1. 使用 GPU
2. 启用混合精度：`--precision fp16`
3. 使用更小的分辨率

### Q: 内存占用过高？

**解决方案：**
1. 减少 batch size
2. 使用更小的模型
3. 关闭不必要的功能

## 输出问题

### Q: 输出视频无法播放？

**解决方案：**
1. 安装解码器：`pip install opencv-python`
2. 使用支持的格式：MP4

### Q: 视频文件过大？

**解决方案：**
1. 降低分辨率
2. 降低帧率
3. 使用更高压缩率
