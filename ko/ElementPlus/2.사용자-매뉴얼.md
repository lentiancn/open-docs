## 1. 컴포넌트 기초

### 1.1 기본 버튼

버튼은 가장 많이 사용되는 컴포넌트 중 하나입니다. Element Plus는 다양한 유형의 버튼을 제공합니다.

```vue
<template>
  <!-- 주요 버튼 -->
  <el-button type="primary">주요 버튼</el-button>
  
  <!-- 성공 버튼 -->
  <el-button type="success">성공 버튼</el-button>
  
  <!-- 정보 버튼 -->
  <el-button type="info">정보 버튼</el-button>
  
  <!-- 경고 버튼 -->
  <el-button type="warning">경고 버튼</el-button>
  
  <!-- 위험 버튼 -->
  <el-button type="danger">위험 버튼</el-button>
  
  <!-- 텍스트 버튼 -->
  <el-button type="text">텍스트 버튼</el-button>
  
  <!-- 평범한 버튼 -->
  <el-button plain>평범한 버튼</el-button>
  
  <!-- 둥근 버튼 -->
  <el-button round>둥근 버튼</el-button>
  
  <!-- 아이콘 버튼 -->
  <el-button icon="el-icon-search">검색</el-button>
</template>
```

### 1.2 버튼 속성

| 속성 | 설명 | 유형 | 선택값 | 기본값 |
|------|------|------|--------|--------|
| type | 버튼 유형 | string | primary / success / warning / danger / info / text | — |
| plain | 평범한 버튼 여부 | boolean | — | false |
| round | 둥근 버튼 여부 | boolean | — | false |
| circle | 원형 버튼 여부 | boolean | — | false |
| loading | 로딩 중 여부 | boolean | — | false |
| disabled | 비활성화 여부 | boolean | — | false |
| icon | 아이콘 클래스 이름 | string | — | — |
| size | 크기 | string | large / default / small | — |

### 1.3 버튼 이벤트

```vue
<template>
  <el-button @click="handleClick">클릭 이벤트</el-button>
</template>

<script setup>
const handleClick = () => {
  console.log('버튼이 클릭되었습니다')
}
</script>
```

## 2. 폼 컴포넌트

### 2.1 기본 폼

```vue
<template>
  <el-form :model="form" :rules="rules" ref="formRef" label-width="100px">
    <el-form-item label="사용자 이름" prop="username">
      <el-input v-model="form.username" placeholder="사용자 이름 입력" />
    </el-form-item>
    
    <el-form-item label="비밀번호" prop="password">
      <el-input v-model="form.password" type="password" placeholder="비밀번호 입력" show-password />
    </el-form-item>
    
    <el-form-item label="이메일" prop="email">
      <el-input v-model="form.email" placeholder="이메일 입력" />
    </el-form-item>
    
    <el-form-item label="성별" prop="gender">
      <el-radio-group v-model="form.gender">
        <el-radio label="male">남성</el-radio>
        <el-radio label="female">여성</el-radio>
      </el-radio-group>
    </el-form-item>
    
    <el-form-item label="취미" prop="hobby">
      <el-checkbox-group v-model="form.hobby">
        <el-checkbox label="농구" />
        <el-checkbox label="축구" />
        <el-checkbox label="수영" />
      </el-checkbox-group>
    </el-form-item>
    
    <el-form-item label="자기소개" prop="bio">
      <el-input v-model="form.bio" type="textarea" :rows="3" placeholder="자기소개 입력" />
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm">제출</el-button>
      <el-button @click="resetForm">초기화</el-button>
    </el-form-item>
  </el-form>
</template>

<script setup>
import { reactive, ref } from 'vue'

const formRef = ref(null)

const form = reactive({
  username: '',
  password: '',
  email: '',
  gender: '',
  hobby: [],
  bio: ''
})

const rules = {
  username: [
    { required: true, message: '사용자 이름을 입력하세요', trigger: 'blur' },
    { min: 3, max: 20, message: '사용자 이름은 3~20자', trigger: 'blur' }
  ],
  password: [
    { required: true, message: '비밀번호를 입력하세요', trigger: 'blur' },
    { min: 6, message: '비밀번호는 6자 이상', trigger: 'blur' }
  ],
  email: [
    { required: true, message: '이메일을 입력하세요', trigger: 'blur' },
    { type: 'email', message: '유효한 이메일을 입력하세요', trigger: 'blur' }
  ],
  gender: [
    { required: true, message: '성별을 선택하세요', trigger: 'change' }
  ]
}

const submitForm = async () => {
  if (!formRef.value) return
  
  await formRef.value.validate((valid) => {
    if (valid) {
      console.log('폼 데이터:', form)
      ElMessage.success('제출 성공')
    } else {
      ElMessage.error('폼을 확인하세요')
    }
  })
}

const resetForm = () => {
  formRef.value?.resetFields()
}
</script>
```

### 3.2 페이지네이션이 있는 테이블

```vue
<template>
  <div>
    <el-table :data="tableData" v-loading="loading">
      <el-table-column prop="id" label="ID" width="80" />
      <el-table-column prop="title" label="제목" />
      <el-table-column prop="author" label="작성자" width="120" />
      <el-table-column prop="date" label="게시일" width="180" />
    </el-table>
    
    <el-pagination
      v-model:current-page="currentPage"
      v-model:page-size="pageSize"
      :total="total"
      :page-sizes="[10, 20, 50, 100]"
      layout="total, sizes, prev, pager, next, jumper"
      @size-change="handleSizeChange"
      @current-change="handleCurrentChange"
    />
  </div>
</template>

<script setup>
import { ref, onMounted } from 'vue'

const tableData = ref([])
const loading = ref(false)
const currentPage = ref(1)
const pageSize = ref(10)
const total = ref(0)

const loadData = async () => {
  loading.value = true
  // API 호출 시뮬레이션
  setTimeout(() => {
    tableData.value = [
      { id: 1, title: '게시물 제목 1', author: '작성자 1', date: '2024-01-01' },
      { id: 2, title: '게시물 제목 2', author: '작성자 2', date: '2024-01-02' }
    ]
    total.value = 100
    loading.value = false
  }, 500)
}

const handleSizeChange = (val) => {
  pageSize.value = val
  loadData()
}

const handleCurrentChange = (val) => {
  currentPage.value = val
  loadData()
}

onMounted(() => {
  loadData()
})
</script>
```

## 4. 다이얼로그와 드로어

### 4.1 다이얼로그

```vue
<template>
  <el-button type="primary" @click="dialogVisible = true">다이얼로그 열기</el-button>
  
  <el-dialog
    v-model="dialogVisible"
    title="정보 편집"
    width="500px"
    :close-on-click-modal="false"
    @close="handleClose"
  >
    <el-form :model="form" label-width="80px">
      <el-form-item label="이름">
        <el-input v-model="form.name" />
      </el-form-item>
      <el-form-item label="설명">
        <el-input v-model="form.desc" type="textarea" :rows="3" />
      </el-form-item>
    </el-form>
    
    <template #footer>
      <el-button @click="dialogVisible = false">취소</el-button>
      <el-button type="primary" @click="handleConfirm">확인</el-button>
    </template>
  </el-dialog>
</template>

<script setup>
import { ref, reactive } from 'vue'

const dialogVisible = ref(false)

const form = reactive({
  name: '',
  desc: ''
})

const handleClose = () => {
  form.name = ''
  form.desc = ''
}

const handleConfirm = () => {
  console.log('제출:', form)
  dialogVisible.value = false
  ElMessage.success('성공')
}
</script>
```

## 5. 메시지 알림

### 5.1 전역 메시지

```vue
<script setup>
import { ElMessage } from 'element-plus'

// 성공 메시지
const showSuccess = () => {
  ElMessage.success('작업 성공')
}

// 경고 메시지
const showWarning = () => {
  ElMessage.warning('경고 메시지')
}

// 오류 메시지
const showError = () => {
  ElMessage.error('작업 실패')
}

// 정보 메시지
const showInfo = () => {
  ElMessage.info('정보 메시지')
}
</script>
```

### 5.2 알림

```vue
<script setup>
import { ElNotification } from 'element-plus'

// 성공 알림
const notifySuccess = () => {

    title: '성공',

  ElNotification({
    title: '오류',
```
