# FantasyTalking - FAQ

Häufige Fragen und Antworten.

## Installationsprobleme

### Q: Abhängigkeiten konnten nicht installiert werden?

**Lösung:**
1. pip aktualisieren: `pip install --upgrade pip`
2. Chinesischen Spiegelserver verwenden:
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```
3. Fehlende Pakete einzeln installieren

### Q: CUDA nicht verfügbar?

**Lösung:**
1. NVIDIA-Treiber installiert: `nvidia-smi`
2. Passende PyTorch-Version installieren:
```bash
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
```
3. CUDA-Verfügbarkeit überprüfen:
```python
import torch
print(torch.cuda.is_available())
```

### Q: Zu wenig Speicher?

**Lösung:**
1. Eingabebilder verkleinern
2. Kleineren Batchsize verwenden
3. CPU-Modus verwenden (nicht empfohlen, langsam)

## Nutzungsprobleme

### Q: Generierung fehlgeschlagen, Gesicht nicht gefunden?

**Lösung:**
1. Bild mit klarem Gesicht verwenden
2. Gesicht nach vorne, ohne Verdeckung
3. Bildformat überprüfen

### Q: Video-Generierung sehr langsam?

**Lösung:**
1. GPU-Beschleunigung verwenden
2. Ausgabeauflösung verringern
3. Ausgabeframes reduzieren

### Q: Lippensynchronisation nicht synchron?

**Lösung:**
1. Audio klar und rauschfrei
2. Audio-Abtastrate überprüfen
3. Verschiedene Synchronisationsparameter ausprobieren

### Q: Ausgabevideo unscharf?

**Lösung:**
1. Gesichtserweiterung verwenden: `--face_enhancer realusf`
2. Ausgabeauflösung erhöhen
3. Klareres Quellbild verwenden

### Q: Gesicht verformt/verzerrt?

**Lösung:**
1. Frontales Foto ohne Bedeckung verwenden
2. Nur ein Gesicht im Bild
3. Parameter für Gesichtserkennung anpassen

## Modellprobleme

### Q: Modell-Download fehlgeschlagen?

**Lösung:**
1. Proxy oder Spiegelserver verwenden
2. Modelldateien manuell herunterladen
3. Netzwerkverbindung überprüfen

### Q: Modell konnte nicht geladen werden?

**Lösung:**
1. Modelldatei vollständig
2. Modellpfad korrekt
3. Modell erneut herunterladen

## Leistungsprobleme

### Q: Wie kann die Generierungsgeschwindigkeit verbessert werden?

**Lösung:**
1. GPU verwenden
2. Gemischte Präzision aktivieren: `--precision fp16`
3. Kleinere Auflösung verwenden

### Q: Speicherbelegung zu hoch?

**Lösung:**
1. Batchsize reduzieren
2. Kleineres Modell verwenden
3. Unnötige Funktionen deaktivieren

## Ausgabeprobleme

### Q: Ausgabevideo kann nicht abgespielt werden?

**Lösung:**
1. Decoder installieren: `pip install opencv-python`
2. Unterstütztes Format verwenden: MP4

### Q: Videodatei zu groß?

**Lösung:**
1. Auflösung reduzieren
2. Framerate reduzieren
3. Höhere Komprimierung verwenden
