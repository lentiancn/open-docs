# PyTorch Übersicht

> Open-Source Machine Learning Framework

---

## Projektübersicht

PyTorch ist ein Open-Source-Framework für maschinelles Lernen von Meta. Es bietet flexible Plattformen für Deep Learning mit dynamischen Berechnungsgraphen.

---

## Hauptmerkmale

- **Dynamische Berechnungsgraphen**: Flexible Netzwerkänderungen
- **GPU-Beschleunigung**: CUDA-Unterstützung
- **Pythonic**: Native Python-API
- **Autograd**: Automatische Differentiation
