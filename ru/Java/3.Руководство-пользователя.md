## Hello World

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

## Классы

```java
public class Person {
    private String name;
    
    public Person(String name) {
        this.name = name;
    }
}

Person p = new Person("Tom");
```
