# Element Plus 简介

为 Vue 3 打造的桌面端组件库。

## 什么是 Element Plus？

Element Plus 是由饿了么团队开发的 Vue 3 组件库，为现代 Web 应用提供了一套完整的桌面端 UI 组件。作为 Element UI 的 Vue 3 版本，Element Plus 继承了前代产品的优秀特性，并针对 Vue 3 进行了全面优化。

### 核心特性

- **Vue 3 驱动**：完全基于 Vue 3 Composition API 构建
- **TypeScript 支持**：原生 TypeScript 类型定义
- **组件丰富**：70+ 高质量组件
- **按需加载**：支持 tree shaking，优化打包体积
- **主题定制**：支持 CSS 变量自定义主题
- **国际化**：内置 20+ 语言包
- **无障碍支持**：完善的 ARIA 支持
- **响应式设计**：适配各种屏幕尺寸
