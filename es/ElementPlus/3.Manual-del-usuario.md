# Manual de Usuario de Element Plus

Este manual te ayudará a comenzar rápidamente con Element Plus y dominar los componentes y funciones más utilizados.

## Inicio Rápido

Antes de usar Element Plus, asegúrate de que la instalación esté completa.

### Uso Básico

Usa los componentes directamente en componentes Vue:

<template>
  <el-button>Botón Predeterminado</el-button>
  <el-button type="primary">Botón Principal</el-button>
  <el-button type="success">Botón de Éxito</el-button>
  <el-button type="warning">Botón de Advertencia</el-button>
  <el-button type="danger">Botón de Peligro</el-button>
</template>

## Componentes Comunes

### Botón

Uso básico:

<el-button>Predeterminado</el-button>
<el-button type="primary">Principal</el-button>
<el-button type="success">Éxito</el-button>
<el-button type="warning">Advertencia</el-button>
<el-button type="danger">Peligro</el-button>

Diferentes tamaños:

<el-button size="large">Grande</el-button>
<el-button>Predeterminado</el-button>
<el-button size="small">Pequeño</el-button>

Estado deshabilitado:

<el-button disabled>Deshabilitado</el-button>

Botones con icono:

<el-button icon="Search">Buscar</el-button>
<el-button icon="Edit">Editar</el-button>
<el-button icon="Delete">Eliminar</el-button>

### Entrada

Uso básico:

<el-input v-model="input" placeholder="Por favor ingrese contenido" />

Con icono:

<el-input v-model="input" prefix-icon="Search" />
<el-input v-model="input" suffix-icon="Calendar" />

Con botón de limpiar:

<el-input v-model="input" clearable />

Entrada de contraseña:

<el-input v-model="input" type="password" show-password />

Texto multilínea:

<el-input v-model="textarea" type="textarea" :rows="3" />

### Selector

Selector básico:

<el-select v-model="value" placeholder="Por favor seleccione">
  <el-option label="Opción 1" value="1" />
  <el-option label="Opción 2" value="2" />
</el-select>

Selección múltiple:

<el-select v-model="value" multiple placeholder="Por favor seleccione">
  <el-option label="Opción 1" value="1" />
  <el-option label="Opción 2" value="2" />
</el-select>

### Tabla

Tabla básica:

<el-table :data="tableData" stripe border>
  <el-table-column prop="date" label="Fecha" width="180" />
  <el-table-column prop="name" label="Nombre" width="180" />
  <el-table-column prop="address" label="Dirección" />
</el-table>

Con índice:

<el-table-column type="index" label="Número" width="60" />

Columnas fijas:

<el-table-column prop="date" label="Fecha" width="150" fixed />

### Formulario

Formulario básico:

<el-form :model="form" label-width="80px">
  <el-form-item label="Nombre de Actividad">
    <el-input v-model="form.name" />
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="onSubmit">Enviar</el-button>
  </el-form-item>
</el-form>

Validación de formulario:

<el-form :model="ruleForm" :rules="rules" ref="ruleFormRef">
  <el-form-item label="Nombre" prop="name">
    <el-input v-model="ruleForm.name" />
  </el-form-item>
</el-form>

### Diálogo

<el-dialog v-model="visible" title="Título" width="30%">
  <span>Contenido</span>
</el-dialog>

### Mensaje

import { ElMessage } from 'element-plus'

ElMessage.success('Mensaje de éxito')
ElMessage.warning('Mensaje de advertencia')

## Sistema de Diseño

### Sistema de Cuadrícula

Element Plus usa un sistema de cuadrícula de 24 columnas:

<el-row :gutter="20">
  <el-col :span="12"><div>12 columnas</div></el-col>
  <el-col :span="12"><div>12 columnas</div></el-col>
</el-row>

### Diseño de Contenedor

<el-container>
  <el-header>Encabezado</el-header>
  <el-main>Principal</el-main>
  <el-footer>Pie de página</el-footer>
</el-container>

## Problemas Comunes

### Estilos de Componentes No se Aplican

Asegúrate de importar el CSS:

import 'element-plus/dist/index.css'

### Iconos No se Muestran

Necesitas instalar la biblioteca de iconos:

npm install @element-plus/icons-vue

Luego regístralos globalmente:

import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}

### Importación bajo Demanda No Funciona

Verifica que la configuración del plugin sea correcta y asegúrate de usar el resolver correcto.
