# JDK

Java Development Kit

## Einführung

JDK (Java Development Kit) ist das Software Development Kit für die Java-Programmiersprache und bildet den Kern aller Java-Entwicklung. JDK enthält die Java-Laufzeitumgebung (JRE), den Java-Compiler (javac), die Java Virtual Machine (JVM) und umfangreiche Java-Klassenbibliotheken und Werkzeuge.

## Hauptversionen

- **JDK 25** - Neueste Version mit neuen Funktionen und Leistungsverbesserungen
- **JDK 21** - Neueste Langzeit-Support-Version (LTS)
- **JDK 17** - Reife LTS-Version, weit verbreitet
- **JDK 11** - LTS-Version, markiert die Teilung der kostenpflichtigen Java-Funktionen

## Hauptkomponenten

- **javac** - Java-Compiler, kompiliert Quellcode zu Bytecode
- **java** - Java-Programmausführungsbefehl
- **jvm** - Java Virtual Machine, führt Bytecode aus
- **jar** - Java-Archivwerkzeug
- **javadoc** - API-Dokumentationsgenerierungswerkzeug
- **jdb** - Java-Debugger

## Versionsunterschiede

Verschiedene JDK-Hauptversionen unterscheiden sich erheblich in Installation, Verwendung und Funktionen:

- **JDK 8**: Letzte vollständig kostenlose Oracle JDK-Version für kommerzielle Nutzung
- **JDK 11**: Führte var, neuen HTTP-Client, ZGC (experimentell) ein
- **JDK 17**: Letzte vollständig kostenlose Oracle JDK-Version unter NFTC
- **JDK 21**: Virtuelle Threads, Pattern Matching, Records (stabil)
- **JDK 25**: Neueste Funktionen, kürzerer Supportzyklus
