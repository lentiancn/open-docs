# Руководство пользователя

## Установка и запуск приложений Android

Waydroid может выполнять несколько различных операций, найденных с помощью команды `waydroid app -h`:

```bash
usage: waydroid app [-h] {install,remove,launch,list} ...

optional arguments:
  -h, --help            показать это справочное сообщение и выйти

subaction:
  {install,remove,launch,list}
    install             отправить один пакет в контейнер и установить его
    remove              удалить один пакет приложения из контейнера
    launch              запустить одно приложение
    list                список установленных приложений
```

### Установка приложений Android

Вы можете установить приложения Android с помощью:

```bash
waydroid app install xyz.apk
```

Затем вы можете запустить приложение через меню приложений на рабочем столе.

Если вы знаете имя пакета Android приложения (`com.xxx.yyy`), вы также можете использовать командную строку:

```bash
waydroid app launch com.foo.bar
```

### Примечания

Файлы apk, которые вы иногда найдете в Интернете, обычно поддерживают только arm и поэтому не будут работать на x86_64.

Вы можете установить [F-Droid](https://f-droid.org), чтобы получать приложения графически. Или используйте Play Маркет, если вы установили образ GAPPS.

## Общие команды

### Опции командной строки Waydroid

Waydroid предоставляет богатые опции командной строки для управления контейнерами и сессиями:

- `waydroid show-full-ui`: Показать полный пользовательский интерфейс Android
- `waydroid session start`: Запустить сессию Waydroid
- `waydroid session stop`: Остановить сессию Waydroid
- `waydroid container start`: Запустить контейнер Waydroid
- `waydroid container stop`: Остановить контейнер Waydroid
- `waydroid logcat`: Просмотреть журналы Android
- `waydroid shell`: Войти в оболочку Android
- `waydroid prop set <property> <value>`: Установить системные свойства
- `waydroid bugreport`: Сгенерировать журналы отчета об ошибке

### Конфигурация системных свойств

Различные системные свойства можно настроить с помощью команды `waydroid prop`:

- `persist.waydroid.multi_windows`: Включить многоконечный режим
- Другие свойства см. в официальной документации

## Расширенные функции

### Использование пользовательских образов

Чтобы использовать пользовательские образы, выполните следующие шаги:

1. Скачайте образы с [SourceForge](https://sourceforge.net/projects/waydroid/files/images/) или используйте пользовательские собранные `system.img` и `vendor.img`
2. Скопируйте образы в каталог `/etc/waydroid-extra/images/`
3. Реинициализируйте Waydroid: `sudo waydroid init -f`

### Отладка ADB

Waydroid поддерживает отладку ADB. По умолчанию включен безопасный режим ADB, который не подключается автоматически при запуске сессии. Используйте следующую команду для подключения:

```bash
waydroid adb connect
```

Затем предоставьте авторизацию на стороне Android.

### Пересылка уведомлений

Waydroid поддерживает пересылку уведомлений Android в среду рабочего стола, что требует обновления образа системы Android до совместимой версии.
