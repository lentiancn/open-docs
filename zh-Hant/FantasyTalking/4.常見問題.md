# FantasyTalking 常见问题

## 安装问题

### 安装失败

确保 Node.js 版本 >= 14.0.0，清除缓存后重试。

### 依赖缺失

检查是否安装了必要的依赖。

## 使用问题

### 语音不播放

1. 检查浏览器是否支持 Web Audio API
2. 检查网络连接
3. 验证 API 密钥

### 语音质量差

调整 speed、pitch、volume 参数。

### 事件不触发

确保正确绑定事件监听器。

## 性能问题

### 卡顿

1. 减少同时播放的语音数量
2. 使用批量合成替代单个合成

## 获取帮助

查看项目 GitHub 页面获取更多帮助。
