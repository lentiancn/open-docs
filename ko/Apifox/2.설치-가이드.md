# 설치 가이드

이 문서에서는 Apifox를 설치하는 다양한 방법을 설명하며, 빠르게 시작할 수 있도록 도와줍니다.

## 시스템 요구사항

Apifox는 다음 운영 체제를 지원합니다:

- Windows 7 이상 (64비트)
- macOS 10.13 이상
- Linux (Debian/Ubuntu 20.04+, Fedora 36+)

## 다운로드

### 공식 웹사이트에서 다운로드

Apifox 공식 웹사이트(https://apifox.com/)를 방문하여 "다운로드" 버튼을 클릭하고, 사용 중인 운영 체제에 맞는 버전을 선택하세요.

### 버전 선택

Apifox는 두 가지 버전을 제공합니다:

- **Apifox** (표준): 모든 핵심 기능이 포함되어 있어 대부분의 사용자에게 적합합니다
- **Apifox** (유료): 팀 관리 및 고급 자동화 같은 더 고급 기능이 포함되어 있습니다

표준 버전에는 필요한 모든 것이 기본적으로 충분합니다.

## 설치 단계

### Windows

1. Windows 설치 파일(.exe 파일)을 다운로드합니다
2. 더블 클릭하여 설치 프로그램을 실행합니다
3. 설치 경로를 선택합니다 (기본값 권장)
4. 설치가 완료될 때까지 기다립니다
5. Apifox를 실행하고 로그인하거나 계정을 생성합니다

### macOS

1. macOS 설치 파일(.dmg 파일)을 다운로드합니다
2. 설치 프로그램을 열고 Apifox를 Applications 폴더로 드래그합니다
3. Apifox를 실행하고 로그인하거나 계정을 생성합니다

최초 실행 시 시스템 환경 설정에서 식별되지 않은 개발자의 앱을 허용해야 할 수 있습니다.

### Linux

**Debian/Ubuntu:**

```bash
# deb 패키지 다운로드
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.deb

# 설치
sudo dpkg -i Apifox-latest-linux.x64.deb

# 의존성이 누락된 경우 설치 수정
sudo apt-get install -f
```

**Fedora:**

```bash
# rpm 패키지 다운로드
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.rpm

# 설치
sudo rpm -i Apifox-latest-linux.x64.rpm
```

## 계정 및 로그인

### 계정 생성

시작하려면 계정을 생성해야 합니다:

1. Apifox를 실행합니다
2. "회원가입"을 클릭합니다
3. 이메일을 입력하고 회원가입을 완료합니다
4. 이메일을 확인합니다 (필요한 경우)

### 로그인

다양한 로그인 방법이 지원됩니다:

- 이메일과 비밀번호 로그인
-第三方 계정 (GitHub, Google 등)

### 클라우드 동기화

로그인하면 프로젝트 데이터가 자동으로 클라우드에 동기화됩니다. 장치 간을 원활하게 전환할 수 있으며 데이터는 실시간으로 동기화 상태를 유지합니다.

## 초기 설정

최초 실행 후 다음을 설정하는 것이 권장됩니다:

### 테마 설정

"설정"에서 라이트 및 다크 테마 중 하나를 선택할 수 있습니다 — 마음에 드는 것을 선택하세요.

### 프록시 설정

프록시를 통해 네트워크에 접근해야 하는 경우 "설정" -> "네트워크"에서 프록시 서버를 구성하세요.

### 가져오기 및 내보내기

- **가져오기**: Postman, Swagger, HAR 및 기타 형식의 프로젝트 가져오기 지원
- **내보내기**: OpenAPI, Markdown, HTML 및 기타 형식으로 내보내기 지원

## 빠른 시작

### 프로젝트 생성

1. 왼쪽 사이드바의 "프로젝트" 옆에 있는 "+"를 클릭합니다
2. 프로젝트 이름을 입력합니다
3. 프로젝트 유형을 선택합니다 (API 프로젝트 또는 Mock 프로젝트)

### 인터페이스 생성

1. 프로젝트에서 "+"를 클릭하고 "새 인터페이스"를 선택합니다
2. 기본 인터페이스 정보를 입력합니다:
   - 인터페이스 이름
   - 요청 경로
   - 요청 메서드 (GET/POST/PUT/DELETE 등)
3. 요청 매개변수, 본문, 응답 등을 입력합니다
4. 인터페이스를 저장합니다

### 인터페이스 디버깅

1. 생성한 인터페이스를 엽니다
2. "전송" 버튼을 클릭합니다
3. 응답을 확인합니다

### Mock 사용

1. 인터페이스에서 데이터 구조를 정의합니다
2. 시스템이 자동으로 Mock URL을 생성합니다
3. Mock URL에 접근하여 시뮬레이션된 데이터를 가져옵니다

## 제거

### Windows

- "제어판" -> "프로그램 및 기능"을 엽니다
- Apifox를 찾아 "제거"를 클릭합니다

### macOS

- Applications 폴더에서 Apifox를 휴지통으로 드래그합니다
- 휴지통을 비웁니다

### Linux

```bash
# Debian/Ubuntu
sudo apt-get remove apifox

# Fedora
sudo rpm -e apifox
```
