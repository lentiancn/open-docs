# Bootstrap 5

Bootstrap — самый популярный открытый инструментарий для создания адаптивных веб-сайтов.

## Введение

Bootstrap 5 — последняя версия, полностью переписанная на нативном JavaScript без зависимости от jQuery. Предоставляет богатые CSS и JavaScript компоненты.

## Основные функции

- **Адаптивный дизайн**: встроенная сетка
- **Компоненты**: кнопки, навигация, формы, карточки
- **Утилиты**: отступы, цвета, текст
- **Sass поддержка**: настройка темы
- **Иконки**: Bootstrap Icons

## Быстрый старт

### CDN

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
```

### npm

```bash
npm install bootstrap@5.3.3
```
