# OpenClaw 🦞

OpenClaw est une **passerelle auto-hébergée** qui connecte vos applications de messagerie préférées — WhatsApp, Telegram, Discord, iMessage et plus — à des assistants de codage IA comme Pi. Vous exécutez un seul processus Gateway sur votre propre machine (ou serveur), et il devient le pont entre vos applications de messagerie et un assistant IA toujours disponible.

## Pour qui est-ce?

OpenClaw est conçu pour les développeurs et les utilisateurs avancés qui souhaitent un assistant IA personnel qu'ils peuvent contacter de n'importe où — sans renoncer au contrôle de leurs données ni dépendre d'un service hébergé.

## Caractéristiques Principales

Ce qui rend OpenClaw différent:

- **Auto-hébergé**: s'exécute sur votre matériel, vos règles
- **Multi-canal**: un Gateway sert WhatsApp, Telegram, Discord et plus simultanément
- **Natif pour les agents**: construit pour les agents de codage avec utilisation d'outils, sessions, mémoire et routage multi-agents
- **Open source**: licence MIT, piloté par la communauté

## De quoi avez-vous besoin?

- Node.js 22 ou supérieur
- Une clé API de votre fournisseur IA choisi
- 5 minutes

Pour la meilleure qualité et sécurité, utilisez le modèle le plus puissant de dernière génération disponible.

## Démarrage Rapide

```bash
# Installer OpenClaw
npm install -g openclaw@latest

# Exécuter l'assistant d'intégration
openclaw onboard --install-daemon

# Se connecter aux canaux et démarrer le Gateway
openclaw channels login
openclaw gateway --port 18789
```

## Comment Ça Marche

Le Gateway est la source unique de vérité pour les sessions, le routage et les connexions de canal. Les applications de messagerie s'y connectent aux agents IA, CLI, UI de contrôle Web ou nœuds mobiles.

## Fonctionnalités Principales

- **Passerelle multi-canal**: WhatsApp, Telegram, Discord et iMessage avec un seul processus Gateway
- **Canaux插件**: Ajoutez Mattermost et plus avec des packages d'extension
- **Routage multi-agents**: sessions isolées par agent, espace de travail ou expéditeur
- **Support multimédia**: envoyer et recevoir des images, audio et documents
- **UI de contrôle Web**: tableau de bord navigateur pour le chat, la configuration, les sessions et les nœuds
- **Nœuds mobiles**: couplez les nœuds iOS et Android pour Canvas, caméra/écran et flux de travail vocaux
