# Vue Benutzerhandbuch

Dieses Handbuch erklärt, wie Sie Vue.js effektiv verwenden.

## Grundkonzepte

### Vue-Instanz

```javascript
import { createApp } from 'vue'

const app = createApp({
  data() {
    return { message: 'Hello Vue!' }
  }
})

app.mount('#app')
```

### Templatesyntax

```html
<!-- Textinterpolation -->
<h1>{{ message }}</h1>

<!-- Direktiven -->
<div v-if="show">Inhalt</div>
<button @click="handleClick">Klick</button>
<input v-model="input">
```

## Reaktivität

### Vue 3 Composition API (empfohlen)

```javascript
import { ref, reactive, computed } from 'vue'

export default {
  setup() {
    const count = ref(0)
    const state = reactive({ name: 'Vue' })
    const double = computed(() => count.value * 2)
    return { count, state, double }
  }
}
```

## Komponenten

### Komponente erstellen

```vue
<template>
  <div>
    <h2>{{ title }}</h2>
    <button @click="emit">Klick</button>
  </div>
</template>

<script setup>
import { defineProps, defineEmits } from 'vue'

const props = defineProps({
  title: { type: String, required: true }
})

const emit = defineEmits(['click'])
</script>
```

## Komponentenkommunikation

### Props (Eltern zu Kind)

```vue
<!-- Kind -->
<script setup>
defineProps({ message: String })
</script>

<template>
  <p>{{ message }}</p>
</template>

<!-- Eltern -->
<template>
  <Child message="Hallo" />
</template>
```

### Events (Kind zu Eltern)

```vue
<!-- Kind -->
<script setup>
const emit = defineEmits(['update'])
const send = () => emit('update', { name: 'Vue' })
</script>

<template>
  <button @click="send">Senden</button>
</template>

<!-- Eltern -->
<template>
  <Child @update="handle" />
</template>
```


## List Rendering

```html
<li v-for="item in items" :key="item.id">{{ item.name }}</li>
```

## Ereignisbehandlung

```html
<button @click="handle">Klick</button>
<button @click.stop>Stop Propagierung</button>
<input @keyup.enter="submit">
```

## Formulare

```vue
<input v-model="form.name">
<el-checkbox v-model="checked">Label</el-switch v-model="switch">
```

## Berechnete Eigenschaften

```javascript
const count = ref(0)
const double = computed(() => count.value * 2)
```

## Watchers

```javascript
import { ref, watch } from 'vue'

const count = ref(0)
watch(count, (newVal, oldVal) => {
  console.log(`${oldVal} -> ${newVal}`)
})
```

## Lebenszyklus-Hooks

```javascript
import { onMounted, onUpdated } from 'vue'

onMounted(() => console.log('Mounted'))
onUpdated(() => console.log('Updated'))
```

## Vue Router

```javascript
const router = createRouter({
  history: createWebHistory(),
  routes: [
    { path: '/', component: Home },
    { path: '/about', component: About }
  ]
})
```

```vue
<router-link to="/">Start</router-link>
<button @click="$router.push('/about')">Über</button>
```

## Pinia Store

```javascript
import { defineStore } from 'pinia'

export const useUserStore = defineStore('user', {
  state: () => ({ name: 'John' }),
  actions: {
    updateName(name) { this.name = name }
  }
})
```

```vue
<script setup>
import { useUserStore } from './stores/user'
const userStore = useUserStore()
</script>
```
