## Installation Methods

### Method 1: From Source

#### 1. Clone Repository

```bash
git clone https://github.com/MultiTalk/MultiTalk.git
cd MultiTalk
```

#### 2. Create Virtual Environment

```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
# or
venv\Scripts\activate  # Windows
```

#### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

#### 4. Download Models

```bash
bash scripts/download_models.sh
```

### Method 2: Docker

```bash
docker pull multitalk/multitalk:latest
docker run --gpus all -v /path/to/data:/data multitalk/multitalk:latest
```

## Configuration

### Environment Variables

```bash
export CUDA_VISIBLE_DEVICES=0
export PYTHONPATH=/path/to/MultiTalk:$PYTHONPATH
```

### Model Paths

Edit `config.yaml`:

```yaml
model:
  checkpoint_path: "checkpoints/multitalk.pth"
```

## Quick Start

### Generate Multi-Speaker Audio

```bash
python inference.py \
  --text "Hello, this is a multi-speaker demo." \
  --speakers speaker1,speaker2 \
  --output output.wav
```

### Parameters

```bash
--text         # Input text
--speakers     # Speaker IDs (comma-separated)
--output       # Output file path
--speed        # Speech speed (default: 1.0)
```
