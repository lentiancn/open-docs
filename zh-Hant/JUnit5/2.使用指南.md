# JUnit 5 使用指南

---

## 測試類

```java
@Test
void test() {
    assertEquals(5, 2 + 3);
}
```

---

## 斷言

```java
assertEquals(expected, actual);
assertTrue(condition);
assertNull(object);
```

---

## 註解

```java
@BeforeEach
void setUp() { }

@AfterEach
void tearDown() { }

@Disabled
@Test
void testSkip() { }

@DisplayName("名稱")
@Test
void testName() { }
```

---

## 參數化測試

```java
@ParameterizedTest
@ValueSource(ints = { 1, 2, 3 })
void test(int number) {
    assertTrue(number > 0);
}
```
