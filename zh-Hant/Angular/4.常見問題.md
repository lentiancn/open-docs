# Angular 常見問題

常見問題解答。

## 安裝問題

### Q: Angular CLI 安裝失敗？

**解決方案**：
1. 清除 npm 緩存：`npm cache clean --force`
2. 使用管理員權限安裝
3. 檢查 Node.js 版本是否滿足要求

### Q: ng 命令找不到？

**解決方案**：
1. 確認 Angular CLI 已全局安裝
2. 檢查 PATH 環境變量
3. 嘗試重新安裝

### Q: Node.js 版本不相容？

**解決方案**：
1. 使用 nvm 管理 Node.js 版本
2. 或升級 Node.js 到推薦版本

## 使用問題

### Q: 組件樣式不生效？

**解決方案**：
1. 檢查 styleUrls 路徑是否正確
2. 確認 CSS 選擇器正確
3. 檢查 ViewEncapsulation 設置

### Q: 路由跳轉失敗？

**解決方案**：
1. 檢查路由配置是否正確
2. 確認 RouterModule 已正確導入
3. 檢查 routerLink 屬性寫法

### Q: HTTP 請求失敗？

**解決方案**：
1. 確認 HttpClientModule 已導入
2. 檢查 CORS 配置
3. 驗證 API URL 是否正確

### Q: 表單驗證不工作？

**解決方案**：
1. 確認導入 ReactiveFormsModule
2. 檢查 Validators 配置
3. 正確使用 formControlName

## 性能問題

### Q: 應用加載慢？

**解決方案**：
1. 啟用懶加載
2. 使用 OnPush 變更檢測
3. 壓縮圖片和資源
4. 啟用生產模式構建

### Q: 內存洩漏？

**解決方案**：
1. 及時取消訂閱
2. 清理定時器
3. 銷毀事件監聽器

## 版本問題

### Q: 如何升級 Angular？

```bash
ng update @angular/core @angular/cli
```

### Q: AngularJS 遷移到 Angular？

使用 angular-upgrade 工具逐步遷移。
