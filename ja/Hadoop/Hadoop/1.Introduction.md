# Hadoop Introduction

> Hadoop is an open-source distributed computing platform for storing and processing large-scale data

## Overview

Hadoop, developed by Apache Software Foundation, provides distributed storage (HDFS) and distributed computing (MapReduce) framework.

## Core Components

- **HDFS**: Hadoop Distributed File System, stores massive data
- **MapReduce**: Distributed computing framework
- **YARN**: Resource manager
- **Common**: Utilities supporting other modules

## Features

- High reliability: Automatic data backup
- High scalability: Scale to thousands of nodes
- High efficiency: Parallel data processing
- Low cost: Runs on commodity hardware
