## 简介

### 什么是 Bootstrap？

Bootstrap 是世界上最流行的前端开源工具包，由 Twitter 开发并维护。它是一个用于快速开发响应式网站和 Web 应用程序的 CSS、JavaScript 框架。

### Bootstrap 版本

| 版本 | 发布日期 | 状态 |
|------|----------|------|
| Bootstrap 5.3 | 2023年5月 | 当前稳定版 |
| Bootstrap 5.2 | 2022年9月 | 支持 |
| Bootstrap 5.1 | 2021年8月 | 支持 |

### Bootstrap 5 新特性

- 全新配色方案和样式改进
- 改进的表单控件
- 新的实用工具 API
- 改进的网格系统
- 更好的自定义方式
- 不再依赖 jQuery
- 放弃 Internet Explorer 支持

## 安装方法

### 方法一：使用 CDN（最简单）

CDN（内容分发网络）是最快的开始方式，不需要下载任何文件。

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 入门</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>Hello, Bootstrap!</h1>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### 方法二：使用 npm

```bash
# 创建项目
mkdir my-bootstrap-project
cd my-bootstrap-project
npm init -y

# 安装 Bootstrap
npm install bootstrap
```

### 方法三：下载源码

访问 https://getbootstrap.com/ 下载预编译的文件。

## 在 Windows 上安装

### 使用 CDN（推荐）

1. 创建新的 HTML 文件
2. 添加 Bootstrap CSS 链接到 `<head>`
3. 添加 Bootstrap JS 脚本到 `<body>` 末尾

### 使用 npm

1. 安装 Node.js：https://nodejs.org/
2. 打开命令提示符
3. 运行：
   ```cmd
   npm install bootstrap
   ```

## 在 macOS 上安装

### 使用 Homebrew（推荐）

```bash
# 安装 Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装 Node.js
brew install node

# 创建项目并安装 Bootstrap
mkdir bootstrap-project
cd bootstrap-project
npm init -y
npm install bootstrap
```

## 在 Linux 上安装

### Ubuntu/Debian

```bash
sudo apt update
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs

mkdir bootstrap-project
cd bootstrap-project
npm init -y
npm install bootstrap
```

### Fedora

```bash
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -
sudo dnf install nodejs

mkdir bootstrap-project
cd bootstrap-project
npm init -y
npm install bootstrap
```

## 验证安装

创建测试 HTML 文件：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <div class="container mt-5">
        <div class="alert alert-success">Bootstrap 安装成功！</div>
        <button class="btn btn-primary">主要按钮</button>
    </div>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

用浏览器打开，如果看到绿色提示框和蓝色按钮，说明安装成功。

## 常见问题

### 1. 样式不生效

- 检查 CSS 链接是否正确
- 清除浏览器缓存

### 2. JavaScript 不工作

- 确保引入的是 bootstrap.bundle.min.js（包含 Popper）
- 检查浏览器控制台是否有错误

### 3. 下拉菜单不工作

- 检查 data-bs-toggle 属性
- 确保引入了 Popper.js
