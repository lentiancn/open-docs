# 사용 매뉴얼

## 기본 개념

### Gateway

시스템 핵심. 메시지, AI 요청, 채널 연결을全部 관리.

### 채널

채팅 앱: WhatsApp, Telegram, Discord 등.

### 에이전트

독립적인 AI 인스턴스. 여러 개 실행 가능.

## 명령어

```bash
# 실행/중지
openclaw gateway start
openclaw gateway stop
openclaw gateway status

# 메시지 보내기
openclaw message send --target +123456789 --message "테스트"
```

## 설정

파일: `~/.openclaw/openclaw.json`

변경 방법:
- `openclaw configure`
- `openclaw config set xxx`
- 웹 인터페이스: http://127.0.0.1:18789/
- 파일 직접 편집

## 설정 예

### 접근 제한

```json5
{
  channels: {
    whatsapp: {
      allowFrom: ["+123456789"]
    }
  }
}
```

### 여러 에이전트

```json5
{
  agents: {
    list: [
      { id: "home", default: true },
      { id: "work" }
    ]
  }
}
```

## 문제 해결

```bash
openclaw logs

openclaw doctor
```
