# Flutter - Guía de Instalación

Guía completa para configurar el entorno de desarrollo de Flutter.

## Requisitos

### Windows

- **SO**: Windows 10 o superior (64 bits)
- **Espacio en disco**: Mínimo 2.8 GB
- **Herramientas**: PowerShell 5.0+, Git para Windows

### macOS

- **SO**: macOS 10.14 (Mojave) o superior
- **Espacio en disco**: Mínimo 2.8 GB
- **Herramientas**: Command Line Tools, Git

### Linux

- **SO**: Ubuntu 18.04 o superior
- **Espacio en disco**: Mínimo 2.8 GB
- **Dependencias**: bash, curl, git, unzip, xz-utils, zip, libglu1-mesa

## Pasos de Instalación

### Instalación en Windows

1. **Descargar Flutter SDK**

Visite el sitio web de Flutter para descargar la última versión:
```bash
# O clonar vía git
git clone https://github.com/flutter/flutter.git -b stable
```

2. **Configurar variables de entorno**

Agregar Flutter al PATH del sistema:
```powershell
# Ejecutar en PowerShell
[Environment]::SetEnvironmentVariable(
    "Path",
    $env:Path + ";C:\src\flutter\bin",
    "User"
)
```

3. **Verificar instalación**

```bash
flutter doctor
```

### Instalación en macOS

1. **Descargar Flutter SDK**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

2. **Configurar variables de entorno**

```bash
# Agregar a ~/.bash_profile o ~/.zshrc
export PATH="$PATH:/Users/nombre_usuario/flutter/bin"
```

3. **Ejecutar flutter doctor**

```bash
flutter doctor
```

### Instalación en Linux

1. **Instalar dependencias**

```bash
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa
```

2. **Descargar Flutter SDK**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

3. **Configurar PATH**

```bash
export PATH="$PATH:/home/nombre_usuario/flutter/bin"
```

## Configuración del Entorno de Desarrollo Android

### Instalar Android Studio

1. Descargar Android Studio: https://developer.android.com/studio
2. Instalar y lanzar
3. Configurar SDK:

```bash
flutter config --android-sdk /ruta/a/android/sdk
```

### Aceptar licencias

```bash
flutter doctor --android-licenses
```

### Configurar emulador

```bash
# Crear emulador
flutter emulator --create --name mi-emulador

# Lanzar emulador
flutter emulator --launch mi-emulador
```

## Configuración del Entorno de Desarrollo iOS (macOS)

### Instalar Xcode

```bash
# Instalar vía App Store
xcode-select --install
```

### Configurar Simulator de iOS

```bash
# Listar simuladores disponibles
xcrun simctl list devices available

# Lanzar simulador
open -a Simulator
```

## Verificación

```bash
# Verificar versión de Flutter
flutter --version

# Ejecutar flutter doctor
flutter doctor

# Buscar actualizaciones
flutter upgrade
```

## Problemas Comunes

### ¿Descarga fallida?

Usar espejo:
```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
export PUB_CACHE=/ruta/al/cache
```

### ¿Android SDK no encontrado?

```bash
flutter config --android-sdk /ruta/a/android/sdk

# O establecer variable de entorno
export ANDROID_SDK_ROOT=/ruta/a/android-sdk
```

### ¿Comando no encontrado?

Asegurarse que Flutter está en PATH:
```bash
echo $PATH
# debe contener flutter/bin
```

## Desinstalación

```bash
# Eliminar directorio de Flutter
rm -rf /ruta/a/flutter

# Eliminar del PATH
```
