# Flutter

Google 推出的开源 UI 开发工具包。

## 简介

Flutter 是一个用于构建跨平台应用的 UI 工具包，使用 Dart 语言，可以让开发者使用一套代码同时构建 iOS、Android、Web 和桌面应用。

## 功能特点

- **跨平台**：一套代码支持多平台
- **热重载**：即时预览修改
- **丰富组件**：Material Design + Cupertino
- **高性能**：原生渲染体验
- **开源免费**

## 快速开始

### 安装

参考安装指南：./2.安装指南.md

### 创建项目

```bash
flutter create my_app
cd my_app
flutter run
```

### 项目结构

```
lib/
└── main.dart        # 入口文件
pubspec.yaml         # 配置文件
```

## 核心技术

- **Dart**：Flutter 开发语言
- **Widget**：UI 构建基础
- **State**：状态管理
- **Platform Channel**：平台通道
