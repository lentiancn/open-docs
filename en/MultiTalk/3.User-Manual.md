# MultiTalk User Manual

Detailed guide to using MultiTalk.

## Creating a Room

### Step 1: Create New Room

1. Click "Create Room" button
2. Enter room name
3. Set room password (optional)
4. Configure room settings

### Step 2: Invite Participants

- Share room link
- Share room code
- Invite from contact list

### Step 3: Manage Room

- Toggle mute/unmute
- Adjust participant volumes
- Kick or mute specific participants

## Joining a Room

### Method 1: Room Link

1. Click on room link
2. MultiTalk will open automatically
3. Enter password if required

### Method 2: Room Code

1. Click "Join Room"
2. Enter room code
3. Enter password if required

## Audio Settings

### Voice Activation

1. Go to Settings > Audio
2. Enable voice activation
3. Adjust threshold level
4. Test sensitivity

### Audio Noise Filters

- suppression
- Echo cancellation
- Automatic gain control

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Ctrl+Shift+T | Push-to-talk |
| Ctrl+M | Mute/unmute |
| Ctrl+Space | Deafen |
| Ctrl+Up | Increase volume |
| Ctrl+Down | Decrease volume |

## Best Practices

1. Use headphones to prevent echo
2. Configure push-to-talk for noisy environments
3. Test audio before important meetings
4. Keep microphone close to mouth

## Advanced Features

### Channels

Create sub-channels within a room:

1. Right-click on room
2. Select "Create Channel"
3. Name the channel
4. Set participant limits

### Permissions

Configure participant permissions:

- Can speak
- Can share screen
- Can invite others
