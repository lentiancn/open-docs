# APISIX 개요

클라우드 네이티브 API 게이트웨이.

## APISIX란?

APISIX는 Apache Software Foundation이 개발한 클라우드 네이티브 API 게이트웨이로, 고성능・高 가용성 API 관리 서비스를 제공합니다. 동적이고 실시간이며 확장 가능한 API 게이트웨이로, APISIX는 수백만 개의 컨테이너화된 배포의 선호 솔루션이 되었습니다.

### 주요 기능

- **고성능**: Nginx + Lua 기반, 코어당 10k+ QPS
- **동적 라우팅**: 재시작 없이 라우팅 규칙 업데이트
- **플러그인 시스템**: 70+ 플러그인, 핫로딩 지원
- **서비스 디스커버리**: Consul, Eureka, Nacos 지원
- **부하 분산**: 라운드로빈, 해시, 일관된 해시
- **보안**: 레이트 리밋, 인증, JWT, 방화벽

### 통계

- GitHub 별표: 13,000+
- 프로덕션 컨테이너: 수백만
- 기여자: 500+
