# Vue インストールガイド

Vueはユーザーインターフェースを構築するためのプログレッシブJavaScriptフレームワークです。

## バージョンの概要

Vueには2つの主要なバージョンが使用されています：

| バージョン | 公開日 | ステータス |
|------------|--------|------------|
| Vue 3 | 2020年9月 | 推奨バージョン |
| Vue 2 | 2016年10月 | メンテナンス終了 (2023年) |

**推奨：** すべての新規プロジェクトにはVue 3を使用してください。Vue 2は2023年にメンテナンスが終了しました。

### Vue 2とVue 3の主な違い

| 機能 | Vue 2 | Vue 3 |
|------|-------|-------|
| アーキテクチャ | Options API | Composition API + Options API |
| リアクティブ | Object.defineProperty | Proxy |
| TypeScript | 一部サポート | 完全サポート |
| パフォーマンス | 良い | より良い |
| バンドルサイズ | ~20KB gzip | ~10KB gzip |
| ライフサイクル | created, mounted | setup + onMounted |
| Teleport | なし | あり |
| Fragments | なし | あり |
| マルチルート | なし | あり |

## 前提条件

### システム要件

| 要件 | Vue 2 | Vue 3 |
|------|-------|-------|
| Node.js | 8.9+ | 16+ (18+推奨) |
| npm | 5.6+ | 8+ |
| ブラウザサポート | IE9+ | IE11非サポート |

### 必要なソフトウェア

- **Node.js** - JavaScriptランタイム
- **npm** - パッケージマネージャー（Node.jsに付属）
- **Git** - バージョン管理（オプション但し推奨）

## インストール方法

### 方法1：npmを使用（推奨）

#### Vue 3（推奨）

```bash
# 新しいVueプロジェクトを作成
npm create vue@latest

# またはVue CLIを使用
npm install -g @vue/cli

# プロジェクトを作成
vue create my-vue-app
```

#### Vue 2（旧バージョン）

```bash
# Vue CLIをインストール
npm install -g @vue/cli

# Vue 2プロジェクトを作成
vue create my-vue2-app --default
```

### 方法2：Viteを使用（Vue 3に推奨）

Viteは、非常に高速な開発サーバーを提供するモダンなビルドツールです。

```bash
# ViteでVue 3プロジェクトを作成
npm create vite@latest my-vue-app -- --template vue

# プロジェクトに移動
cd my-vue-app

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

### 方法3：CDNを使用

#### Vue 3 CDN

```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
```

#### Vue 2 CDN

```html
<script src="https://unpkg.com/vue@2/dist/vue.js"></script>
```

### 方法4：Vue CLIを使用

```bash
# Vue CLIをグローバルインストール
npm install -g @vue/cli

# インストール確認
vue --version

# プロジェクトを作成
vue create my-project

# またはGUIを使用
vue ui
```

### 方法5：Nuxt.jsを使用

NuxtはVueアプリケーションを作成するためのフレームワークです。

```bash
# Nuxt 3プロジェクトを作成
npx nuxi@latest init my-nuxt-app

# プロジェクトに移動
cd my-nuxt-app

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

## 環境設定

### Node.jsのインストール

#### Windows

1. https://nodejs.org からNode.jsをダウンロード
2. インストーラーを実行（LTSバージョンを推奨）
3. 確認：`node --version`

#### macOS

```bash
# Homebrewを使用
brew install node
```

#### Linux (Ubuntu/Debian)

```bash
# NodeSourceを使用
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# またはnvmを使用
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 18
```

### npmの設定

```bash
# npmバージョンを確認
npm --version

# npmレジストリを設定（中国向けオプション）
npm config set registry https://registry.npmmirror.com
```

## Vue Routerの設定

### インストール

```bash
npm install vue-router@4
```

### 基本設定（Vue 3）

```javascript
// src/router/index.js
import { createRouter, createWebHistory } from 'vue-router'
import Home from '../views/Home.vue'
import About from '../views/About.vue'

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```

## Pinia（状態管理）

### インストール

```bash
npm install pinia
```

### 基本設定（Vue 3）

```javascript
// src/stores/counter.js
import { defineStore } from 'pinia'

export const useCounterStore = defineStore('counter', {
  state: () => ({
    count: 0
  }),
  getters: {
    doubleCount: (state) => state.count * 2
  },
  actions: {
    increment() {
      this.count++
    }
  }
})
```

## Vue 2 Options API（旧バージョン）

```javascript
new Vue({
  el: '#app',
  data() {
    return {
      message: 'Hello Vue 2!',
      items: []
    }
  },
  computed: {
    reversedMessage() {
      return this.message.split('').reverse().join('')
    }
  },
  methods: {
    handleClick() {
      console.log('Clicked!')
    }
  },
  created() {
    console.log('Component created')
  }
})
```

## Vue 3 Composition API

```javascript
import { ref, computed, onMounted } from 'vue'

export default {
  setup() {
    // リアクティブ状態
    const message = ref('Hello Vue 3!')
    const count = ref(0)
    
    // コンピューテッド
    const reversedMessage = computed(() => {
      return message.value.split('').reverse().join('')
    })
    
    // メソッド
    const increment = () => {
      count.value++
    }
    
    // ライフサイクル
    onMounted(() => {
      console.log('Component mounted')
    })
    
    return {
      message,
      count,
      reversedMessage,
      increment
    }
  }
}
```

## トラブルシューティング

### 常见问题

#### npm installが失敗する

```bash
# npmキャッシュをクリア
npm cache clean --force

# node_modulesを削除して再インストール
rm -rf node_modules
npm install
```

#### ポートが使用中

```bash
# 別のポートを使用
npm run dev -- --port 3000
```

#### Nodeバージョン問題

```bash
# Nodeバージョンを確認
node --version

# nvmでバージョンを切り替え
nvm use 18
```

## ベストプラクティス

1. **Vue 3を使用** - 特定のVue 2要件がない限り
2. **Viteを使用** - より高速な開発とビルド
3. **Composition APIを使用** - より良いTypeScriptサポートとコード整理
4. **TypeScriptを使用** - より良い開発体験
5. **Piniaを使用** - 新規プロジェクトではVuexの代わりにPinia
