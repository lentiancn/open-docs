# JUnit 5 FAQ

## 일반적인 질문

### JUnit이란?
Java를 위한 오픈소스 단위 테스트 프레임워크입니다.

### JUnit 4 vs 5의 차이점은?
JUnit 5: 모듈식 아키텍처, Java 8+, 내장된 매개변수화 테스트

### JUnit 4 테스트를 다시 작성해야 하나요?
아니요, JUnit Vintage로 실행할 수 있습니다.

## 설치

### Maven
```xml
<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter</artifactId>
    <version>5.10.0</version>
</dependency>
```
