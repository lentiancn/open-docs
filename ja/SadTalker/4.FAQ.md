# SadTalker FAQ

## 一般的な質問

### SadTalkerとは何ですか？
SadTalkerは、深層学習プロジェクトで、単一の静的な画像と音声入力からリアリスティックな話し顔ビデオを生成します。

### SadTalkerは免费ですか？
はい、SadTalkerは個人および研究用に免费のオープンソースです。詳細についてはLICENSEファイルを参照してください。

### システム要件は何ですか？
- **GPU**：8GB+ VRAM付きNVIDIA GPU（推奨）
- **RAM**：最小8GB、推奨16GB
- **ストレージ**：モデルと出力用に20GB+

## インストールに関する質問

### SadTalkerをインストールするには？
```bash
# リポジトリをクローン
git clone https://github.com/WinfredSadTalker/SadTalker.git
cd SadTalker

# Conda環境を作成
conda create -n sadtalker python=3.8
conda activate sadtalker

# 依存関係をインストール
pip install -r requirements.txt
```

### どのPythonバージョンを使用すべきですか？
SadTalkerはPython 3.8または3.9で最適に動作します。

### GPUが必要ですか？
合理的なパフォーマンスにはGPUを強く推奨します。CPU推論も可能ですが非常に遅くなります。

## 使用に関する質問

### 話し顔ビデオを生成するには？
```bash
python inference.py --source_image path/to/image.jpg --audio path/to/audio.wav
```

### どの画像形式がサポートされていますか？
- JPG/JPEG
- PNG
- 一般的な画像形式のほとんど

### どの音声形式がサポートされていますか？
- WAV（推奨）
- MP3
- FFmpegがサポートするその他の形式

### 生成にはどのくらいの時間がかかりますか？
処理時間は以下によって異なります：
- ビデオの長さ
- GPUパフォーマンス
- モデル設定
- 解像度

## トラブルシューティング

### モデルのダウンロードに失敗する
リポジトリまたはドキュメントに記載されている代替ソースから手動でモデルをダウンロードしてください。

### メモリ不足エラー
- バッチサイズを縮小
- 出力解像度を下げる
- モデルスワッピングを使用
- 他のGPUアプリケーションを閉じる

### 結果の品質が悪い
- 高解像度入力画像を使用
- 背景噪音のないクリーンな音声を使用
- 強調パラメータを調整

### 顔検出が失敗する
- 顔が明確に見える画像を使用
- 正面顔を確保
- 顔が隠れていない画像を使用

## 上級者向け質問

### SadTalkerを商用にできますか？
ライセンス条項を確認してください。一部のコンポーネントは商用利用に制限がある場合があります。

### SadTalkerはどのように動作しますか？
SadTalkerは以下を使用します：
1. 顔検出で顔ランドマークを検出
2. 音声特徴抽出
3. 3D顔モデル生成（3DMM）
4. 音声からの運動係数予測
5. 顔レンダリングネットワークでビデオをレンダリング

### 独自のモデルをトレーニングできますか？
はい、カスタムデータセットでSadTalkerを微調整できます。トレーニングドキュメントを参照してください。
