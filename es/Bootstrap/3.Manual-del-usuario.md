# Manual del Usuario

Este manual te explica en detalle las funciones principales de Bootstrap 5 y cómo usarlas, incluyendo el sistema de cuadrícula, componentes y clases utilitarias.

## Sistema de Cuadrícula

El sistema de cuadrícula es el corazón de Bootstrap, proporcionando un diseño de 12 columnas.

### Uso Básico

```html
<div class="container">
    <div class="row">
        <div class="col">Columna 1</div>
        <div class="col">Columna 2</div>
        <div class="col">Columna 3</div>
    </div>
</div>
```

### Puntos de Rotura Responsivos

| Clase | Ancho de Pantalla | Descripción |
|--------|------------------|-------------|
| `.col-xs` | < 576px | Pantalla muy pequeña |
| `.col-sm` | ≥ 576px | Pantalla pequeña |
| `.col-md` | ≥ 768px | Pantalla mediana |
| `.col-lg` | ≥ 992px | Pantalla grande |
| `.col-xl` | ≥ 1200px | Pantalla muy grande |
| `.col-xxl` | ≥ 1400px | Pantalla extra grande |

### Desplazamiento de Columnas

```html
<div class="row">
    <div class="col-md-4">.col-md-4</div>
    <div class="col-md-4 offset-md-4">.col-md-4 .offset-md-4</div>
</div>
```

### Columnas Responsivas

```html
<div class="row">
    <div class="col-12 col-md-6 col-lg-4">
        Ocupa todo el ancho en móvil, mitad en tablet, un tercio en pantalla grande
    </div>
</div>
```

## Navegación

### Navegación Básica

```html
<ul class="nav">
    <li class="nav-item">
        <a class="nav-link active" href="#">Inicio</a>
    </li>
    <li class="nav-item">
        <a class="nav-link" href="#">Acerca de</a>
    </li>
    <li class="nav-item">
        <a class="nav-link" href="#">Contacto</a>
    </li>
</ul>
```

### Barra de Navegación

```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">Marca</a>
        <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav">
            <span class="navbar-toggler-icon"></span>
        </button>
        <div class="collapse navbar-collapse" id="navbarNav">
            <ul class="navbar-nav">
                <li class="nav-item">
                    <a class="nav-link active" href="#">Inicio</a>
                </li>
                <li class="nav-item">
                    <a class="nav-link" href="#">Acerca de</a>
                </li>
            </ul>
        </div>
    </div>
</nav>
```

## Ventana Modal

```html
<!-- Botón que abre el modal -->
<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#exampleModal">
    Abrir Modal
</button>

<!-- Modal -->
<div class="modal fade" id="exampleModal" tabindex="-1">
    <div class="modal-dialog">
        <div class="modal-content">
            <div class="modal-header">
                <h5 class="modal-title">Título del Modal</h5>
                <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
            </div>
            <div class="modal-body">
                Contenido...
            </div>
            <div class="modal-footer">
                <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Cerrar</button>
                <button type="button" class="btn btn-primary">Guardar</button>
            </div>
        </div>
    </div>
</div>
```

## Mejores Prácticas

1. **Usa HTML semántico**: Emplea etiquetas como header, nav, main, footer de forma adecuada
2. **Mantén la simplicidad**: Evita el anidamiento excesivo
3. **Diseña primero para móviles**: Enfócate en el diseño responsivo desde el inicio
4. **Aprovecha las clases utilitarias**: Ajusta estilos rápidamente
5. **Personaliza con variables Sass**: Mantén la consistencia en tu tema
