# FantasyTalking Часто задаваемые вопросы

## Проблемы с установкой

### Ошибка установки?

Сначала убедитесь, что среда соответствует требованиям. Проверьте версию Node.js (должна быть >= 14.0). Если используете Python, убедитесь, что версия >= 3.7.

Если установка не работает, попробуйте: Очистить кэш npm; Использовать зеркало; Или установить из исходного кода.

### Проблемы с FFmpeg

На некоторых системах FFmpeg может быть неправильно установлен. Можно скачать бинарные файлы с официального сайта FFmpeg.

## Использование

### Синтез речи медленный?

Скорость зависит от многих факторов. Проверьте сетевое подключение. Используйте самостоятельный хостинг для уменьшения задержек.

### Как выбрать стиль голоса?

FantasyTalking предлагает разные стили: neutral, happy, sad, angry, excited. Используйте neutral или professional для客服.

### Как реализовать английский синтез?

Укажите английский в конфигурации:

```javascript
const audio = await client.speak('Hello, world!', {
  voice: { language: 'en-US', style: 'neutral' }
});
```

## Аккаунт и оплата

### Как получить API-ключ?

Зарегистрируйтесь на официальном сайте FantasyTalking и создайте ключ в консоли.

### Как посмотреть использование?

После входа в консоль можно увидеть количество вызовов API.

## Развертывание

### Docker не работает?

Проверьте, что Docker установлен и запущен. Проверьте, не занят ли порт 8080.

## Другие вопросы

### Какие платформы поддерживаются?

- Веб: все современные браузеры
- Мобильные: iOS и Android
- Десктоп: Windows, macOS и Linux
- Сервер: Node.js и Python
