# HeyGen Installationsanleitung

HeyGen ist eine AI-Videogenerierungsplattform.

## Überblick

HeyGen ist ein Cloud-basiertes SaaS - keine lokale Installation erforderlich.

## Versionen

| Komponente | Version |
|------------|---------|
| HeyGen Web | Neueste |
| HeyGen API | v1/v2 |

## Zugriff

### Schritt 1: Konto erstellen

1. Besuchen Sie https://www.heygen.com
2. Klicken Sie auf "Registrieren"
3. E-Mail verifizieren

### Schritt 2: Plan wählen

| Plan | Funktionen | Preis |
|------|------------|-------|
| Kostenlos | Begrenzte Minuten | Kostenlos |
| Pro | Mehr Minuten | $29/Monat |
| Enterprise | API | Kontakt |

### Schritt 3: API-Schlüssel

1. Dashboard anmelden
2. Settings → API
3. Schlüssel generieren

## Installation

### Python SDK

```bash
pip install heygen-python
```

### Node.js SDK

```bash
npm install heygen-node
```

## Konfiguration

```python
from heygen import HeyGen

heygen = HeyGen(api_key="your_api_key")
account = heygen.get_account_info()
```

## Umgebungsvariablen

```bash
export HEYGEN_API_KEY="your_api_key"
```
