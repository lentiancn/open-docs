# Python Installationsanleitung

> Vollständige Einrichtung

---

## Installation

### Windows

```powershell
choco install python
```

### macOS

```bash
brew install python@3.12
```

### Linux

```bash
sudo apt install python3 python3-pip
```

---

## Paketverwaltung

```bash
pip install requests
python -m venv myenv
```

---

## Überprüfen

```bash
python --version
```
