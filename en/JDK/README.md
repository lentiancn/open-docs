# JDK

Java Development Kit

## Introduction

JDK (Java Development Kit) is the software development kit for the Java programming language, forming the core of all Java development. JDK includes Java Runtime Environment (JRE), Java compiler (javac), Java Virtual Machine (JVM), and extensive Java class libraries and tools.

## Major Versions

- **JDK 25** - Latest version with new features and performance improvements
- **JDK 21** - Latest Long-Term Support version (LTS)
- **JDK 17** - Mature LTS version, widely adopted
- **JDK 11** - LTS version, marking the divide for Java paid features

## Core Components

- **javac** - Java compiler, compiles source code to bytecode
- **java** - Java program execution command
- **jvm** - Java Virtual Machine, executes bytecode
- **jar** - Java archive tool
- **javadoc** - API documentation generation tool
- **jdb** - Java debugger

## Version Differences

Different JDK major versions have significant differences in installation, usage, and features:

- **JDK 8**: Last version with completely free Oracle JDK for commercial use
- **JDK 11**: Introduced var, new HTTP client, ZGC (experimental)
- **JDK 17**: Last free Oracle JDK under NFTC
- **JDK 21**: Virtual Threads, pattern matching, records (stable)
- **JDK 25**: Latest features, shorter support cycle
