# Nacos 安装文档

在所有平台上安装 Nacos 的详细指南。

---

## 什么是 Nacos？

Nacos 提供：
- **配置管理**：动态配置
- **服务发现**：注册和发现服务

---

## 安装方式

| 方式 | 描述 |
|------|------|
| **单机** | 单服务器 |
| **Docker** | 容器化部署 |
| **集群** | 多服务器 |

---

## Docker 安装

```bash
# 运行 Nacos
docker run -d --name nacos -p 8848:8848 -e MODE=standalone nacos/nacos-server:v2.3.0
```

---

## 访问

- URL: http://localhost:8848/nacos
- 用户名: nacos
- 密码: nacos
