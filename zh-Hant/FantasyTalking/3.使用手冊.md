# FantasyTalking 使用手册

## 基本用法

### 语音合成

```javascript
// 播放语音
ft.speak('你好，世界')

// 停止播放
ft.stop()

// 暂停
ft.pause()

// 继续
ft.resume()
```

### 事件监听

```javascript
// 开始播放
ft.on('start', () => {
  console.log('开始播放')
})

// 播放结束
ft.on('end', () => {
  console.log('播放结束')
})

// 错误处理
ft.on('error', (err) => {
  console.error('错误:', err)
})
```

### 语音配置

```javascript
// 设置语速
ft.setSpeed(1.5)

// 设置音调
ft.setPitch(1.2)

// 设置音量
ft.setVolume(0.8)

// 切换语音
ft.setVoice('female')
```

### 对话功能

```javascript
// 添加对话
ft.addDialogue([
  { text: '你好', voice: 'male' },
  { text: '有什么可以帮助你？', voice: 'female' }
])

// 播放对话
ft.playDialogue()

// 停止对话
ft.stopDialogue()
```
