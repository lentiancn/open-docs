# OpenClaw 安裝指南

> 設定和安裝 OpenClaw 的完整指南

---

## 系統需求

| 需求 | 規格 |
|------|------|
| 作業系統 | Windows 10+、macOS、Linux |
| Node.js | 版本 18+ |
| 記憶體 | 至少 4 GB RAM |
| 儲存空間 | 約 500 MB |

---

## 安裝

### 前置條件

確保已安裝 Node.js：

```bash
node --version
npm --version
```

### 逐步安裝

#### 1. 透過 npm 安裝

```bash
npm install -g openclaw
```

#### 2. 啟動 Gateway 服務

Gateway 服務是 OpenClaw 的核心：

```bash
# 檢查狀態
openclaw gateway status

# 啟動 Gateway
openclaw gateway start

# 重新啟動 Gateway
openclaw gateway restart

# 停止 Gateway
openclaw gateway stop
```

#### 3. 初始配置

安裝後，設定配置檔案：

```bash
# 建立配置目錄
mkdir -p ~/.openclaw

# 複製預設配置
cp /path/to/openclaw/config/* ~/.openclaw/
```

## 配置

### 主要配置檔案

| 檔案 | 說明 |
|------|------|
| AGENTS.md | 工作區配置 |
| SOUL.md | 個性設定 |
| TOOLS.md | 工具配置 |
| USER.md | 使用者偏好 |

### 環境變數

```bash
# 可選：自訂工作區目錄
OPENCLAW_WORKSPACE=/path/to/workspace

# 可選：Gateway 連接埠
OPENCLAW_PORT=3000
```

## 驗證

### 測試安裝

```bash
# 顯示版本資訊
openclaw --version

# 檢查 Gateway 狀態
openclaw gateway status

# 顯示可用命令
openclaw --help
```

### 快速入門

```bash
# 啟動互動式工作階段
openclaw chat

# 執行單一任務
openclaw exec "打招呼"
```

## 故障排除

### Gateway 無法啟動？

**解決方案**：
1. 確認 Node.js 已正確安裝
2. 檢查權限
3. 查看日誌：`openclaw gateway logs`

### 連接埠已被佔用？

**解決方案**：
```bash
# 變更連接埠
openclaw gateway start --port 3001
```

### 找不到模組？

**解決方案**：
```bash
# 重新安裝
npm uninstall -g openclaw
npm install -g openclaw
```

---

## 解除安裝

```bash
npm uninstall -g openclaw

# 移除 Gateway 服務
openclaw gateway stop
rm -rf ~/.openclaw
```
