# Element Plus 설치 가이드

이 가이드에서는 다양한 프로젝트에서 Element Plus를 설치하고 사용하는 방법에 대한 자세한 지침을 제공합니다.

## 환경 요구사항

Element Plus는 다음 요구사항이 있습니다:

- Vue 3.0.0+
- Node.js 14.0.0+
- 최신 브라우저 (Chrome, Firefox, Safari, Edge)

## 설치 방법

### 패키지 관리자 사용

npm, yarn 또는 pnpm을 사용하여 설치하는 것이 권장됩니다:

```bash
# npm
npm install element-plus

# yarn
yarn add element-plus

# pnpm
pnpm add element-plus
```

## 전체 가져오기

번들 크기가 그렇게 중요하지 않다면 전체 가져오기가 더 편리합니다.

main.ts에서:

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')
```

### Volar 지원

Volar를 사용하는 경우 tsconfig.json의 compilerOptions.types에 전역 컴포넌트 타입 정의를 추가하세요:

```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

## 필요 시 가져오기

사용하는 컴포넌트를 가져오려면 추가 플러그인이 필요합니다.

### 자동 가져오기 (권장)

먼저 unplugin-vue-components와 unplugin-auto-import를 설치해야 합니다:

```bash
npm install -D unplugin-vue-components unplugin-auto-import
# 또는
yarn add -D unplugin-vue-components unplugin-auto-import
# 또는
pnpm install -D unplugin-vue-components unplugin-auto-import
```

#### Vite 구성

vite.config.ts에서:

```typescript
import { defineConfig } from 'vite'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    AutoImport({ resolvers: [ElementPlusResolver()] }),
    Components({ resolvers: [ElementPlusResolver()] }),
  ],
})
```

#### Webpack 구성

webpack.config.js에서:

```javascript
const AutoImport = require('unplugin-auto-import/webpack')
const Components = require('unplugin-vue-components/webpack')
const { ElementPlusResolver } = require('unplugin-vue-components/resolvers')

module.exports = {
  plugins: [
    AutoImport({ resolvers: [ElementPlusResolver()] }),
    Components({ resolvers: [ElementPlusResolver()] }),
  ],
}
```

### Nuxt와 함께 사용

Nuxt 사용자의 경우 @element-plus/nuxt만 설치하면 됩니다:

```bash
npm install -D @element-plus/nuxt
# 또는
yarn add -D @element-plus/nuxt
# 또는
pnpm install -D @element-plus/nuxt
```

그런 다음 구성 파일에 다음 코드를 추가하세요:

```typescript
export default defineNuxtConfig({
  modules: ['@element-plus/nuxt'],
})
```

### 수동 가져오기

Element Plus는 ES 모듈 기반의 즉시 사용 가능한 Tree Shaking 기능을 제공합니다.

하지만 스타일 가져오기에 unplugin-element-plus를 설치해야 합니다:

```bash
npm install -D unplugin-element-plus
```

컴포넌트에서:

```vue
<template>
  <el-button>I am ElButton</el-button>
</template>

<script>
import { ElButton } from 'element-plus'

export default {
  components: { ElButton },
}
</script>
```

Vite 구성:

```typescript
import { defineConfig } from 'vite'
import ElementPlus from 'unplugin-element-plus/vite'

export default defineConfig({
  plugins: [ElementPlus()],
})
```

## 전역 구성

Element Plus를 등록할 때 size와 zIndex가 포함된 전역 구성 객체를 전달할 수 있습니다:

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import App from './App.vue'

const app = createApp(App)
app.use(ElementPlus, { size: 'small', zIndex: 3000 })
```

구성 옵션:
- size: 폼 컴포넌트의 기본 크기 설정 (large, default, small)
- zIndex: 팝업 컴포넌트의 z-index 설정, 기본값은 2000

### 필요 시 가져오기 구성

el-config-provider 컴포넌트 사용:

```vue
<template>
  <el-config-provider :size="size" :z-index="zIndex">
    <app />
  </el-config-provider>
</template>

<script>
import { defineComponent } from 'vue'
import { ElConfigProvider } from 'element-plus'

export default defineComponent({
  components: { ElConfigProvider },
  setup() {
    return { zIndex: 3000, size: 'small' }
  },
})
</script>
```
