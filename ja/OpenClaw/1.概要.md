# OpenClaw 🦞

OpenClawは、自分のデバイス上で動かせるツールです。WhatsApp、Telegram、Discord、iMessageなどのよくあるメッセージングアプリをAIサービスにつなぎます。よく使うチャットアプリから直接AIを使えるようになります。

## 誰向け？

第三方にデータを渡したくないけど、自分のAIアシスタントを持ちたい人にぴったりです。コマンドラインを使う開発者やかで効率的な作業を好む人に特に向いています。

## 主な特徴

- **データの自主性**：全部自分のハードディスクに保存
- **複数プラットフォーム**：一つのプロセスで複数のチャットアプリを扱える
- **コーディング助手而生**：ツール呼び出し、会話履歴、メモリ、複数アシスタント対応
- **オープンソース**：MITライセンス、社区で開発

## 必要なもの

- Node.js 22 以上
- AIプロバイダーからのAPIキー
- 最初の設定に数分

最新世代の高性能モデルを使うと一番いです。

## クイックスタート

```bash
# インストール
npm install -g openclaw@latest

# 初期設定
openclaw onboard --install-daemon

# 起動
openclaw gateway --port 18789
```

ブラウザで http://127.0.0.1:18789/ を開くと使えます。

## 仕組み

Gatewayという一つのプロセスが全部 обработкуします。メッセージの転送、AIリクエスト、チャンネル管理，全部このプロセスが担当します。

## できること

- **一元管理**：全部のチャット 채널を一つのシステムで
- **プラグイン**：拡張パックで更多平台に対応
- **複数アシスタント**：異なる目的で别々のAI实例を運行
- **ファイル対応**：画像、音声、ドキュメントの送受信
- **Web管理画面**：ブラウザで設定変更など
- **スマホ連携**：iOS/Androidをペアリングして遠隔操作
