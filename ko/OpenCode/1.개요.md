# OpenCode 개요

오픈소스 AI 코딩 에이전트.

## OpenCode란?

OpenCode는 터미널 인터페이스, 데스크톱 앱 또는 IDE 확장으로 사용 가능한 오픈소스 AI 코딩 에이전트입니다.

### 주요 기능

- **LSP 활성화** - 올바른 LSP 자동 로드
- **멀티 세션** -同一个 프로젝트에서 여러 에이전트 병렬 실행
- **링크 공유** - 모든 세션 링크 공유 가능
- **GitHub Copilot** - GitHub 로그인으로 Copilot 사용
- **ChatGPT Plus/Pro** - OpenAI 로그인으로 ChatGPT 사용
- **모든 모델** - 로컬 모델을 포함한 75+ LLM 제공자 지원
- **모든 편집기** - 터미널, 데스크톱 앱, IDE 확장 지원
- **개인정보 보호 우선** - 코드나 컨텍스트 데이터 저장 안 함
