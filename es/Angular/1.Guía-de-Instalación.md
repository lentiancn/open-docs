## Requisitos del Sistema

### Requisitos de Hardware
- **Procesador**: Procesador de doble núcleo o mejor
- **RAM**: Mínimo 4GB (8GB recomendados)
- **Espacio en Disco**: Al menos 3GB de espacio libre

### Requisitos de Software

#### Para Windows
- Windows 10 o posterior
- Node.js 18.19 o posterior (versión LTS recomendada)
- npm 9.x o posterior (incluido con Node.js)
- Git para control de versiones

#### Para macOS
- macOS 10.15 (Catalina) o posterior
- Node.js 18.19 o posterior (versión LTS recomendada)
- npm 9.x o posterior
- Git (incluido con Xcode Command Line Tools)

#### Para Linux
- Ubuntu 20.04 LTS o posterior / Distribuciones basadas en Debian
- Fedora 36 o posterior / Distribuciones basadas en RHEL
- Node.js 18.19 o posterior (versión LTS recomendada)
- npm 9.x o posterior
- Git

## Prerrequisitos

### 1. Instalar Node.js

Node.js es el entorno de tiempo de ejecución necesario para ejecutar Angular.

#### Instalación en Windows

**Método A: Usando el Instalador de Node.js**
1. Visite https://nodejs.org/
2. Descargue la versión **LTS**
3. Ejecute el instalador (.msi)
4. Siga el asistente de instalación
5. Verifique en Command Prompt:
   ```
   node --version
   npm --version
   ```

**Método B: Usando Chocolatey**
```powershell
# Instalar Node.js
choco install nodejs-lts -y
```

**Método C: Usando nvm-windows (Recomendado)**
```powershell
# Instalar y usar Node.js LTS
nvm install 18.19.0
nvm use 18.19.0
```

#### Instalación en macOS

**Método A: Usando Homebrew (Recomendado)**
```bash
# Instalar Homebrew si no está instalado
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Instalar Node.js LTS
brew install node@18
```

**Método B: Usando nvm**
```bash
# Instalar nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Instalar Node.js LTS
nvm install 18.19.0
nvm use 18.19.0
```

#### Instalación en Linux

**Ubuntu/Debian**
```bash
# Actualizar lista de paquetes
sudo apt update

# Instalar curl y build-essential
sudo apt install curl build-essential

# Añadir repositorio de Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -

# Instalar Node.js
sudo apt install nodejs
```

**Fedora/RHEL**
```bash
# Añadir repositorio de Node.js
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -

# Instalar Node.js
sudo dnf install nodejs
```

### 2. Instalar Git

Git es requerido para las funciones de control de versiones de Angular CLI.

#### Windows
- Descargue desde https://git-scm.com/
- Seleccione "Use Git from the Windows Command Prompt"

#### macOS
```bash
# Usando Homebrew
brew install git

# O instalar Xcode Command Line Tools
xcode-select --install
```

#### Linux
```bash
# Ubuntu/Debian
sudo apt install git

# Fedora
sudo dnf install git
```

### Guía Paso a Paso

1. **Instalar Node.js**
   - Descargue Node.js LTS desde https://nodejs.org/
   - Ejecute el instalador

2. **Instalar Git** (si no está instalado)
   - Descargue desde https://git-scm.com/

   ```bash
   npm install -g @angular/cli
   ```

4. **Verificar Instalación**
   ```bash
   ng version
   ```

   ```bash
   ng new my-first-app

   ng serve
   ```

## Instalación en Linux

### Ubuntu/Debian

```bash
# Actualizar sistema
sudo apt update && sudo apt upgrade -y

# Instalar prerrequisitos
sudo apt install curl build-essential git

# Añadir repositorio de Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -

# Instalar Node.js
sudo apt install nodejs

# Instalar Angular CLI
sudo npm install -g @angular/cli
```

### Fedora

```bash
# Instalar prerrequisitos
sudo dnf install curl git

# Añadir repositorio de Node.js
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -

# Instalar Node.js
sudo dnf install nodejs

# Instalar Angular CLI
sudo npm install -g @angular/cli
```

## Creando tu Primer Proyecto Angular

### 1. Crear Nuevo Proyecto
```bash
ng new my-angular-app
```

### 2. Navegar al Proyecto
```bash
cd my-angular-app
```

### 3. Iniciar Servidor de Desarrollo
```bash
ng serve
```

### 4. Abrir en Navegador
Navegue a: http://localhost:4200

### 5. Generar Componentes
```bash
ng generate component my-component
# o
ng g c my-component
```

## Solución de Problemas

### Problemas Comunes y Soluciones

#### 1. Errores de permisos npm (EACCES)

**Solución:**
```bash
# Usar nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
```

#### 2. Versión de Node.js incompatible

**Solución:**
```bash
# Usar nvm para cambiar versiones
nvm install 18.19.0
nvm use 18.19.0
```

#### 3. Angular CLI no encontrado

**Solución:**
```bash
# Reinstalar Angular CLI
npm uninstall -g @angular/cli
npm install -g @angular/cli
```

#### 4. Puerto 4200 en uso

**Solución:**
```bash
# Usar puerto diferente
ng serve --port 4201
```

#### 5. Compatibilidad de versiones de Angular

| Versión de Angular | Versión de Node.js | Versión de TypeScript |
|-------------------|-------------------|---------------------|
| Angular 17        | 18.x, 20.x        | ~5.2.0              |
| Angular 16        | 16.x, 18.x        | ~5.0.0              |
| Angular 15        | 16.x, 18.x        | ~4.9.0              |
| Angular 14        | 14.x, 16.x        | ~4.7.0              |
