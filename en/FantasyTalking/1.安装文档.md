## Installation Methods

### Method 1: From Source

#### 1. Clone Repository

```bash
git clone https://github.com/FantasyTalking/FantasyTalking.git
cd FantasyTalking
```

#### 2. Create Virtual Environment

```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
# or
venv\Scripts\activate  # Windows
```

#### 3. Install Dependencies

```bash
pip install torch torchvision
pip install -r requirements.txt
```

#### 4. Download Models

```bash
# Download pretrained models
bash scripts/download_models.sh
```

### Method 2: Docker

```bash
# Pull image
docker pull fantasytalking/fantasytalking:latest

# Run
docker run --gpus all -v /path/to/data:/data fantasytalking/fantasytalking:latest
```

## Configuration

### Environment Variables

```bash
export CUDA_VISIBLE_DEVICES=0
export PYTHONPATH=/path/to/FantasyTalking:$PYTHONPATH
```

### Model Paths

Edit `config.yaml` to set model paths:

```yaml
model:
  checkpoint_path: "checkpoints/fantasytalking.pth"
  face_parser: "models/face_parser.pth"
```

## Quick Start

### Generate Talking Head

```bash
python inference.py \
  --source_image path/to/image.jpg \
  --audio path/to/audio.wav \
  --output output.mp4
```

### Parameters

```bash
--source_image    # Source face image
--audio          # Input audio file
--output         # Output video path
--face enhancer  # Enable face enhancement
--fps            # Output FPS (default: 25)
```

## Common Issues

### GPU Not Available?

Check CUDA installation:

```bash
nvidia-smi
python -c "import torch; print(torch.cuda.is_available())"
```

### Out of Memory?

Reduce batch size or use lower resolution.
