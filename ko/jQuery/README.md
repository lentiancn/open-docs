# jQuery

## 개요

jQuery는 빠르고 작으며 기능이 풍부한 JavaScript 라이브러리입니다. HTML 문서 탐색 및 조작, 이벤트 처리, 애니메이션, Ajax 등을 다양한 브라우저에서 작동하는 사용하기 쉬운 API로 단순화합니다.

## 특징

- **간결한 구문**: 체인 가능한 API, 코드가 간결하고 읽기 쉬움
- **강력한 선택자**: 전체 CSS 선택자 지원
- **크로스 브라우저 호환성**: 브라우저 API 차이 해결
- **풍부한 플러그인 생태계**: 방대한 플러그인 라이브러리
