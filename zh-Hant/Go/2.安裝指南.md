| 環境 | 要求 |
|------|-----|

## 安裝

### Windows

#### MSI 安裝程式

1. 下載：https://go.dev/dl/
2. 執行 MSI 安裝程式
3. 驗證：`go version`

#### Chocolatey

```powershell
choco install golang
```

#### 手動

1. 下載 Windows ZIP
2. 解壓縮到 `C:\Go`
3. 將 `C:\Go\bin` 加入 PATH
4. 驗證：`go version`

### macOS

#### brew（推薦）

```bash
brew install go
```

### Linux

#### apt（Ubuntu/Debian）

```bash
sudo apt update
sudo apt install golang-go
```

#### 手動

```bash
wget https://go.dev/dl/go1.22.0.linux-amd64.tar.gz
sudo tar -C /usr/local -xzf go1.22.0.linux-amd64.tar.gz
export PATH=$PATH:/usr/local/go/bin
```

## 常見問題

### Windows：找不到 go 命令？

確保 `C:\Go\bin` 在您的 PATH 中。
