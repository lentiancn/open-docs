# HeyGen 설치 가이드

HeyGen은 사용자가 현실적인 AI 아바타를 만들고 텍스트 또는 오디오 입력에서 말하는 헤드 비디오를 생성할 수 있는 AI 비디오 생성 플랫폼입니다.

## 플랫폼 개요

HeyGen은 주로 클라우드 기반 SaaS 플랫폼입니다. 오픈 소스 프로젝트와 달리 HeyGen은 로컬 설치가 필요하지 않습니다 - 사용자는 웹 인터페이스 또는 API를 통해 액세스합니다.

### 액세스 방법

| 방법 | 설명 | 요구사항 |
|------|------|----------|
| 웹 인터페이스 | 브라우저 기반 비디오 제작 | 계정 + 인터넷 |
| API | 프로그래밍 방식 액세스 | API 키 |
| SDK | 로컬 개발 | 개발자 계정 |

## 버전 개요

HeyGen은 클라우드 서비스이며 로컬 설치가 아닙니다:

| 구성요소 | 버전 | 참고 |
|----------|------|------|
| HeyGen 웹 | 최신 | 항상 최신 상태 유지 |
| HeyGen API | v1/v2 | REST API |
| HeyGen SDK | 최신 | 개발자용 |

**참고:** HeyGen은 상용 제품입니다. 일부 기능은 유료 구독이 필요합니다.

## HeyGen 액세스

### 1단계: 계정 생성

1. https://www.heygen.com 방문
2. "가입" 또는 "시작하기" 클릭
3. 이메일 주소 인증

### 2단계: 요금제 선택

| 요금제 | 기능 | 가격 |
|--------|------|------|
| 무료 | 제한된 시간, 워터마크 | 무료 |
| Pro | 더 많은 시간, 워터마크 없음 | $29/월 |
| Enterprise | 사용자 정의 제한, API 액세스 | 영업팀 문의 |

### 3단계: API 키 가져오기(개발자용)

1. HeyGen 대시보드에 로그인
2. 설정 → API로 이동
3. API 키 생성

## API 설치(개발자용)

### Python SDK

```bash
# HeyGen SDK 설치
pip install heygen-python

# 또는 소스에서 설치
pip install git+https://github.com/heygen-ai/heygen-python.git
```

### Node.js SDK

```bash
# HeyGen SDK 설치
npm install heygen-node

# 또는 소스에서 설치
npm install github:heygen-ai/heygen-node
```

### 설치 확인

```python
# Python
import heygen
print(heygen.__version__)
```

```javascript
// Node.js
const heygen = require('heygen-node');
console.log(heygen.version);
```

## API 구성

### Python 예제

```python
from heygen import HeyGen

# API 키로 초기화
heygen = HeyGen(api_key="your_api_key")

# 계정 정보 가져오기
account = heygen.get_account_info()
print(account)
```

### Node.js 예제

```javascript
const { HeyGen } = require('heygen-node');

const heygen = new HeyGen({ apiKey: 'your_api_key' });

// 계정 정보 가져오기
const account = await heygen.getAccountInfo();
console.log(account);
```

## 환경 변수

### API 키 설정

```bash
# Linux/macOS
export HEYGEN_API_KEY="your_api_key"

# Windows (PowerShell)
$env:HEYGEN_API_KEY="your_api_key"

# .env 파일에서
HEYGEN_API_KEY=your_api_key
```

### dotenv가 포함된 Python

```python
from dotenv import load_dotenv
load_dotenv()

import heygen
heygen = HeyGen(api_key=os.getenv("HEYGEN_API_KEY"))
```

## 일반적인 문제

### 인증 오류

**문제:** "잘못된 API 키"

**해결책:**
1. API 키가 정확한지 확인
2. API 키가 활성 상태인지 확인
3. 계정에 API 액세스 권한이 있는지 확인

### 속도 제한

**문제:** "너무 많은 요청"

**해결책:**
1. API 사용 제한 확인
2 더 높은 제한을 위해 요금제 업그레이드
3. 요청 대기열 구현

### 비디오 생성 실패

**문제:** "생성 실패"

**해결책:**
1. 입력 매개변수 확인
2. 크레딧 잔액 확인
3. 나중에 다시 시도
