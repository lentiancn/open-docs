# Element Plus 安装文档

在所有主流平台上安装 Element Plus 的完整指南。

---

## 什么是 Element Plus？

Element Plus 是一个基于 Vue 3 的组件库，为开发者、设计师和产品经理提供高质量的 UI 组件。

**注意**：Element Plus 适用于 Vue 3。如果使用 Vue 2，请使用 Element UI。

---

## CDN

### 快速开始

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Element Plus CDN</title>
  <link rel="stylesheet" href="https://unpkg.com/element-plus/dist/index.css">
</head>
<body>
  <div id="app">
    <el-button>按钮</el-button>
  </div>

  <script src="https://unpkg.com/vue@3"></script>
  <script src="https://unpkg.com/element-plus"></script>
</body>
</html>
```

### CDN 提供商

| 提供商 | CSS | JS |
|--------|-----|-----|
| unpkg | https://unpkg.com/element-plus/dist/index.css | https://unpkg.com/element-plus |
| jsDelivr | https://cdn.jsdelivr.net/npm/element-plus/dist/index.css | https://cdn.jsdelivr.net/npm/element-plus |

## NPM

### 安装 Element Plus

```bash
npm install element-plus
```

### 安装图标

```bash
npm install @element-plus/icons-vue
```

---

## Vite

### 创建项目

```bash
npm create vite@latest my-element-app -- --template vue
cd my-element-app
npm install
npm install element-plus
npm install @element-plus/icons-vue
```

### main.js

```javascript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)
app.use(ElementPlus)
app.mount('#app')
```
