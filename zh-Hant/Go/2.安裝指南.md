# Go 安裝指南

> Go 語言安裝和配置的完整指南

---

## 系統需求

| 資源 | 要求 |
|------|------|
| 作業系統 | Windows、macOS、Linux、FreeBSD |
| 架構 | amd64、arm64、386 等 |
| 磁碟空間 | 約 100 MB |

---

## 安裝方式

### Windows

#### 使用 MSI
1. 下載 Go：https://go.dev/dl/
2. 執行安裝程式
3. 遵循安裝精靈

#### Chocolatey
```powershell
choco install golang
```

### macOS

#### Homebrew
```bash
brew install go
```

### Linux

#### Ubuntu/Debian
```bash
sudo apt-get update
sudo apt-get install golang-go
```

#### 手動安裝
```bash
wget https://go.dev/dl/go1.22.0.linux-amd64.tar.gz
sudo tar -C /usr/local -xzf go1.22.0.linux-amd64.tar.gz
export PATH=$PATH:/usr/local/go/bin
```

---

## 環境設定

```bash
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin
go env -w GOPROXY=https://goproxy.cn,direct
```

## 常用指令

```bash
go mod init <模組>
go mod tidy
go build
go run main.go
go test ./...
```
