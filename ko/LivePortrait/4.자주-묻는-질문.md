# LivePortrait 자주 묻는 질문

이 문서에서는 LivePortrait 사용 중에 발생할 수 있는 일반적인 질문과 해결 방안을 모았습니다.

## 설치 문제

### CUDA 버전 호환성 문제

**문제**: PyTorch 설치 시 CUDA 버전 호환성 경고가 표시됩니다.

**해결책**: 먼저 nvcc -V를 사용하여 시스템 CUDA 버전을 확인한 다음 해당 버전의 PyTorch를 설치하세요. CUDA 버전이 확실하지 않으면 테스트용 CPU 버전을 설치할 수 있지만 추론 속도가 매우 느립니다.

### Windows 설치 실패

**문제**: Windows 시스템에서 설치 또는 실행 중 오류가 발생합니다.

**해결책**: Windows 사용자는 원클릭 인스톨러를 다운로드하여 압축을 푼 후 직접 실행하는 것이 좋습니다. CUDA 문제가 발생하면 CUDA를 버전 11.8로 다운그레이드해 보세요. 일부 안티바이러스 프로그램이 종속성 설치를 차단할 수 있습니다. 일시적으로 비활성화한 후 다시 시도하세요.

### 사전 학습된 모델 다운로드 실패

**문제**: HuggingFace에서 모델 가중치를 다운로드할 수 없습니다.

**해결책**: HF_ENDPOINT 환경 변수를 https://hf-mirror.com으로 설정하여 국내 미러 사이트를 사용해 보세요. Google Drive 또는 Baidu Netdisk에서 모델 파일을 다운로드할 수도 있습니다.

### VRAM 부족

**문제**: 추론 중 메모리 부족 오류가 발생합니다.

**해결책**: 입력 이미지 해상도를 낮춰 보세요. --scale 매개변수를 사용하여 출력 크기를 조정하세요. 많은 양의 VRAM을 소비하는 다른 프로그램을 닫으세요. 배치 처리를 사용하는 경우 배치 크기를 줄이세요.

## 실행 문제

### 추론 속도가 너무 느림

**문제**: 추론 속도가 예상보다 훨씬 느립니다.

**해결책**: NVIDIA GPU而不是 CPU를 사용하고 있는지 확인하세요. CUDA가 제대로 작동하는지 확인하세요(python -c "import torch; print(torch.cuda.is_available())" 실행). torch.compile 가속을 활성화하세요. 그래픽 드라이버를 업그레이드하거나 더 강력한 GPU 사용을 고려하세요.

### 출력 비디오 이상

**문제**: 생성된 비디오에 이미지 왜곡이나 색상 이상이 있습니다.

**해결책**: 구동 비디오가 사양을 충족하는지 확인하세요. 구동 비디오의 첫 번째 프레임이 정면 중립적 표정인지 확인하세요. 자동 자르기 기능(--flag_crop_driving_video)을 사용해 보세요.

### 동물 모드를 실행할 수 없음

**문제**: 동물 모드를 사용할 수 없습니다.

**해결책**: 시스템이 NVIDIA GPU가 있는 Linux 또는 Windows인지 확인하세요. 설치 가이드에 설명된 대로 X-Pose 종속성을 설치하세요. HuggingFace에서 동물 모델 가중치를 다운로드하세요. macOS에서는 동물 모드가 지원되지 않습니다.

### Gradio 인터페이스를 시작할 수 없음

**문제**: app.py를 실행한 후 웹 인터페이스에 액세스할 수 없습니다.

**해결책**: 포트가 사용 중인지 확인하세요; --server_port를 사용하여 다른 포트를 지정하세요. 방화벽이 해당 포트에 대한 액세스를 허용하는지 확인하세요. 터미널의 오류 메시지를 확인하세요.

## 효과 문제

### 표정이 자연스럽지 않음

**문제**: 생성된 애니메이션의 표정이僵硬하거나 자연스럽지 않습니다.

**해결책**: 표정이 더 풍부한 구동 비디오를 선택하세요. --driving_multiplier 매개변수를 조정하세요. 소스 이미지와 구동 비디오 간의 얼굴 각도 차이가 너무 크지 않은지 확인하세요.

### 눈 방향이 일치하지 않음

**문제**: 구동 비디오에서 인물의 시선 방향이 대상 초상화에 올바르게 전송되지 않습니다.

**해결책**: 이것은 현재 기술적 제한입니다. 시선 방향이 더 명확한 구동 비디오를 선택하거나 향후 버전 업데이트를 기다려 보세요.

### 머리 회전 범위 제한

**문제**: 머리를 원하는 각도로 회전할 수 없습니다.

**해결책**: --flag_stitching 매개변수를 사용하여 연결 설정을 조정하세요. 머리 회전 범위는 원래 초상화에서 사용 가능한 얼굴 정보에 의해 제한됩니다. 소스로 정면에서 찍은 사진을 사용해 보세요.

### 입술 움직임이 동기화되지 않음

**문제**: 입술 움직임이 구동 비디오와 동기화되지 않습니다.

**해결책**: 구동 비디오의 오디오와 비디오가 동기화되어 있는지 확인하세요. 소스 이미지의 입이 자연스럽게 열려 있는지 확인하세요. 다른 구동 비디오를 사용해 보세요.

## 기술 문제

### 모델 가중치 디렉터리 구조가 잘못됨

**문제**: 모델 파일을 찾을 수 없습니다.

**해결책**: pretrained_weights 디렉터리 구조가 정확한지 확인하세요. GitHub 저장소의 디렉터리 구조 설명을 참조하세요. 디렉터리를 삭제하고 모델을 다시 다운로드하세요.

### 종속성 버전 충돌

**문제**: 종속성 설치 시 버전 충돌 경고가 표시됩니다.

**해결책**: 설치를 위해 새 Python 환경을 만드세요. 충돌하는 패키지를 제거하고 지정된 버전을 다시 설치하세요. requirements.txt에 지정된 버전을 사용하세요.

### macOS 실행 오류

**문제**: macOS에서 실행 시 오류가 발생합니다.

**해결책**: Apple Silicon macOS의 경우 환경 변수 PYTORCH_ENABLE_MPS_FALLBACK=1을 설정하세요. requirements_macOS.txt의 종속성을 사용하고 있는지 확인하세요. Intel 기반 Mac은 제대로 작동하지 않을 수 있습니다.

## 보안 및 윤리

### 딥페이크 콘텐츠 식별 방법

**문제**: LivePortrait에서 생성된 딥페이크 콘텐츠를 식별하는 방법.

**해결책**: 현재 기술은 식별 가능한 시각 아티팩트가 포함된 결과를 생성합니다. 전문 딥페이크 탐지 도구는 대부분의 합성 콘텐츠를 식별할 수 있습니다. 이 기술을 책임감 있게 사용해 주세요.

### 사용 제한

**문제**: LivePortrait의 사용 제한은 무엇입니까?

**해결책**: 현지 법률과 규정을 준수해 주세요. 가짜 뉴스, 사기 또는 기타 악의적인 목적으로 사용하지 마세요. 프로젝트는 생성된 결과에 대한 법적 책임을 지지 않습니다. 이 기술을 사용할 때 관련 당사자에게 알려야 합니다.

## 성능 최적화

### 추론 속도를 개선하는 방법

**제안**: 더 강력한 GPU 하드웨어를 사용하세요. torch.compile 가속을 활성화하세요. 모션 템플릿을 사용하여 반복 계산을 피하세요. 출력 해상도를 낮추세요. 불필요한 기능을 비활성화하세요.

### 배치 처리

**문제**: 한 번에 여러 비디오를 처리하는 방법.

**제안**: inference.py 호출을 반복하는 스크립트를 작성하세요. 추론 코드를 수정하여 배치 입력을 지원하세요. 여러 작업을 관리하기 위해 큐를 사용하세요.
