# OpenCode

오픈소스 AI 코딩 에이전트.

## 개요

OpenCode는 터미널 인터페이스, 데스크톱 앱 또는 IDE 확장으로 사용 가능한 오픈소스 AI 코딩 에이전트입니다. LSP 활성화, 멀티 세션, 75+ LLM 제공자 지원等特点이 있습니다.

### 주요 기능

- **LSP 활성화** - 올바른 LSP 자동 로드
- **멀티 세션** - 여러 에이전트 병렬 실행
- **모든 모델** - 75+ LLM 제공자
- **개인정보 보호 우선** - 코드나 컨텍스트 데이터 저장 안 함

### 통계

- GitHub Stars: 100,000+
- 기여자: 700+
- 월간 활성 개발자: 2,500,000+

## 빠른 시작

### 설치

```bash
curl -fsSL https://opencode.ai/install | bash
```

### 설정

```bash
opencode
/connect
```

opencode.ai/auth에서 API 키를 얻으세요.

### 초기화

```bash
/init
```

### 사용

