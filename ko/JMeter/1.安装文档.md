# JMeter 설치 가이드

JMeter는 Apache 조직에서 개발한 오픈소스 성능 테스트 도구입니다. 이 가이드에서는 다양한 환경에서의 설치 방법을 설명합니다.

## 버전 정보

| 버전 | 상태 | 설명 |
|------|------|------|
| JMeter 5.x | ✅ 최신 | 최신 버전 |

## 시스템 요구사항

### 하드웨어 요구사항

| 요구사항 | 구성 |
|------|------|
| CPU | 2코어 이상 |
| 메모리 | 4GB 이상 |
| 디스크 | 5GB 이상 |

### 소프트웨어 요구사항

| 소프트웨어 | 버전 |
|------|------|
| Java | JDK 8 이상 |

## 설치 방법

### 방법 1: Windows 설치

#### 1. 다운로드

1. https://jmeter.apache.org/download.cgi 방문
2. Binaries 버전 다운로드
3. 원하는 디렉토리에 압축 해제

#### 2. 실행

```powershell
# bin\jmeter.bat 더블 클릭
# 또는 명령줄에서 실행
jmeter.bat
```

### 방법 2: macOS 설치

#### 1. Homebrew 사용

```bash
brew install jmeter
```

#### 2. 수동 설치

```bash
# 다운로드 및 압축 해제
wget https://archive.apache.org/dist/jmeter/binaries/apache-jmeter-5.6.3.tgz
tar -xzf apache-jmeter-5.6.3.tgz

# 실행
./apache-jmeter-5.6.3/bin/jmeter
```

### 방법 3: Linux 설치

```bash
# Ubuntu/Debian
sudo apt-get install jmeter

# 또는 다운로드 설치
wget https://archive.apache.org/dist/jmeter/binaries/apache-jmeter-5.6.3.tgz
tar -xzf apache-jmeter-5.6.3.tgz
cd apache-jmeter-5.6.3/bin
./jmeter
```

### 방법 4: Docker

```bash
# JMeter 컨테이너 실행
docker run -v $(pwd):/workspace justb4/jmeter -n -t test.jmx -l result.jtl
```

## 환경 구성

### Java 구성

```bash
# Java 확인
java -version

# 환경 변수 설정
export JAVA_HOME=/path/to/java
export PATH=$JAVA_HOME/bin:$PATH
```

## 플러그인 설치

### 방법 1: 플러그인 관리자 사용

1. Plugins Manager 다운로드
2. lib/ext 디렉토리에 배치
3. JMeter 재시작

### 방법 2: 수동 설치

```bash
# 플러그인 다운로드
# lib/ext 디렉토리에 배치
```

## 자주 묻는 질문

### 메모리 부족 시

jmeter.bat 또는 jmeter 수정:

```bash
# 힙 크기 설정
HEAP="-Xms1g -Xmx2g"
```

### GUI 시작이 느린 경우

명령줄 모드를 사용하거나 메모리를 증가시키세요.
