# Руководство по установке SadTalker

SadTalker — это проект ИИ для генерации реалистичных видео говорящих голов из одного изображения и аудиовхода.

## Системные требования

### Требования к аппаратному обеспечению

| Компонент | Минимум | Рекомендуется |
|-----------|---------|---------------|
| GPU | NVIDIA 6GB VRAM | NVIDIA 16GB VRAM |
| RAM | 8GB | 32GB |
| Накопитель | 20GB свободно | 50GB свободно |
| ОС | Ubuntu 18.04+, Windows 10+, macOS 11+ | Ubuntu 20.04+ |

### Требования к программному обеспечению

| Программа | Версия | Примечания |
|-----------|--------|------------|
| Python | 3.8 - 3.10 | Рекомендуется 3.9 |
| CUDA | 11.7+ | Для ускорения GPU |
| cuDNN | 8.5+ | Требуется для CUDA |
| ffmpeg | Последняя версия | Для обработки видео |

## Установка на Linux/Ubuntu

### Шаг 1: Установка системных зависимостей

```bash
# Обновление системы
sudo apt update && sudo apt upgrade -y

# Установка Python и инструментов разработки
sudo apt install python3.9 python3-pip python3-venv git wget curl

# Установка системных библиотек
sudo apt install libgl1-mesa-glx libglib2.0-0 libsm6 libxext6 libxrender-dev libgomp1

# Установка ffmpeg
sudo apt install ffmpeg

# Установка драйверов NVIDIA (если не установлены)
sudo apt install nvidia-driver-525
```

### Шаг 2: Установка CUDA (при необходимости)

```bash
# Загрузка и установка CUDA 11.8
wget https://developer.download.nvidia.com/compute/cuda/11.8.0/local_installers/cuda_11.8.0_520.61.05_linux.run
sudo sh cuda_11.8.0_520.61.05_linux.run
```

### Шаг 3: Клонирование репозитория

```bash
git clone https://github.com/OpenTalker/SadTalker.git
cd SadTalker
```

### Шаг 4: Создание виртуальной среды

```bash
python3.9 -m venv venv
source venv/bin/activate

# Проверка версии Python
python --version
```

### Шаг 5: Установка PyTorch

```bash
# Для CUDA 11.8
pip install torch==2.0.1 torchvision==0.15.2 --index-url https://download.pytorch.org/whl/cu118

# Проверка установки
python -c "import torch; print(torch.cuda.is_available())"
```

### Шаг 6: Установка зависимостей

```bash
pip install -r requirements.txt

# Установка дополнительных зависимостей
pip install numpy opencv-python librosa soundfile
```

### Шаг 7: Загрузка моделей

```bash
# Создание директории для моделей
mkdir -p checkpoints

# Загрузка моделей (автоматически)
bash scripts/download_models.sh

# Или загрузить вручную с релиза
# Поместить модели в директорию checkpoints/
```

## Установка на Windows

### Шаг 1: Установка Python

1. Скачать Python 3.9 с python.org
2. При установке отметить "Add Python to PATH"
3. Проверить: `python --version`

### Шаг 2: Установка CUDA

1. Скачать CUDA 11.8 с сайта NVIDIA
2. Установить с настройками по умолчанию
3. Скачать и установить cuDNN 8.9

### Шаг 3: Клонирование и настройка

```powershell
git clone https://github.com/OpenTalker/SadTalker.git
cd SadTalker

python -m venv venv
venv\Scripts\activate
```

### Шаг 4: Установка зависимостей

```powershell
# Установка PyTorch с поддержкой CUDA
pip install torch==2.0.1 torchvision==0.15.2 --index-url https://download.pytorch.org/whl/cu118

pip install -r requirements.txt
```

### Шаг 5: Загрузка моделей

Скачать модели со страницы релизов GitHub и поместить в папку `checkpoints/`.

## Установка на macOS

### Шаг 1: Установка зависимостей

```bash
# Установка Homebrew (если не установлен)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Установка Python и ffmpeg
brew install python@3.9 ffmpeg git
```

### Шаг 2: Настройка

```bash
git clone https://github.com/OpenTalker/SadTalker.git
cd SadTalker

python3.9 -m venv venv
source venv/bin/activate

# Установка PyTorch (только CPU для macOS)
pip install torch==2.0.1 torchvision==0.15.2

pip install -r requirements.txt
```

Примечание: macOS работает в режиме только CPU. Ускорение GPU недоступно.

## Установка через Docker

### Использование Docker

```bash
# Загрузка образа
docker pull ghcr.io/opentalker/sadtalker:latest

# Запуск с поддержкой GPU
docker run --gpus all -v $(pwd):/workspace -p 8888:8888 ghcr.io/opentalker/sadtalker:latest
```

### Использование Docker Compose

```yaml
version: '3.8'
services:
  sadtalker:
    image: ghcr.io/opentalker/sadtalker:latest
    volumes:
      - ./workspace:/workspace
    ports:
      - "8888:8888"
    deploy:
      resources:
        reservations:
          devices:
            - driver: nvidia
              count: 1
              capabilities: [gpu]
```

## Распространённые проблемы

### Недостаточно памяти CUDA

**Решение:**
- Уменьшить размер батча: `--batch_size 1`
- Использовать меньшее разрешение: `--size 256`
- Включить выгрузку моделей
- Закрыть другие приложения GPU

### Отсутствуют библиотеки (Windows)

**Решение:**
- Установить Visual C++ Redistributable 2015-2022
- Добавить CUDA в системный PATH

### Ошибка загрузки моделей

**Решение:**
- Использовать VPN или прокси
- Загрузить вручную с релизов GitHub
- Проверить подключение к сети

### ffmpeg не найден

**Решение:**
- Linux: `sudo apt install ffmpeg`
- Windows: Добавить ffmpeg в PATH
- macOS: `brew install ffmpeg`
