# H2 설치 가이드

## Maven 의존성

```xml
<dependency>
    <groupId>com.h2database</groupId>
    <artifactId>h2</artifactId>
    <version>2.2.224</version>
</dependency>
```

## 실행

```bash
java -jar h2.jar
```

웹 콘솔: http://localhost:8082

## 연결 URL

| 모드 | URL |
|------|-----|
| 인메모리 | jdbc:h2:mem:testdb |
| 파일 | jdbc:h2:file:./data/testdb |
| TCP | jdbc:h2:tcp://localhost:9092/testdb |
