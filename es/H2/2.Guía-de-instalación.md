# Guía de instalación de H2

## Requisitos

- Java 11 o superior
- Cualquier sistema operativo que soporte Java (Windows, Linux, macOS, etc.)

## Métodos de instalación

### Método 1: Descargar ZIP

1. Visita la página de descarga: https://www.h2database.com/html/download.html
2. Descarga la versión "Platform-Independent Zip"
3. Extrae el archivo ZIP a cualquier directorio

### Método 2: Dependencia Maven

```xml
<dependency>
    <groupId>com.h2database</groupId>
    <artifactId>h2</artifactId>
    <version>2.2.224</version>
</dependency>
```

### Método 3: Instalador de Windows

Descarga el instalador y ejecútalo.

## Verificar instalación

```bash
java -jar h2.jar
```

Esto inicia la consola web de H2 en http://localhost:8082

## Formatos de URL de conexión

| Modo | URL |
|------|-----|
| Memoria | jdbc:h2:mem:testdb |
| Archivo | jdbc:h2:file:./data/testdb |
| TCP | jdbc:h2:tcp://localhost:9092/testdb |
