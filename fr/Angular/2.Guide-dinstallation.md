# Guide d'installation d'Angular

> Configuration

---

## Configuration requise

| Environnement | Requirement |
|---------------|-------------|
| Node.js | 18.10.0+ |
| npm | 9.0.0+ |

---

## Installation

```bash
npm install -g @angular/cli
ng new my-app
ng serve
```

---

## Commandes

| Commande | Description |
|--------|-------------|
| ng new | Nouveau projet |
| ng serve | Serveur dev |
| ng build | Compiler |
