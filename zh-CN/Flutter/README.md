## 什么是 Flutter？

Google 推出的**跨平台 UI 框架**，用**一套代码**同时开发 iOS、Android、Web、Windows、macOS、Linux 应用。

### 核心优势

- **省时** - 一套代码，多端运行
- **快速** - 热重载，开发效率高
- **漂亮** - Material + Cupertino 组件库
- **流畅** - 原生编译，60fps

```bash

flutter create my_app
cd my_app

# 2. 运行
flutter run
```
