# HeyGen Руководство по установке

HeyGen — это платформа генерации AI-видео, которая позволяет пользователям создавать реалистичные аватары и генерировать видео с говорящей головой из текста или аудио.

## Версия

HeyGen — это облачная платформа SaaS:

| Компонент | Версия | Примечания |
|-----------|--------|-----------|
| HeyGen Web | Последняя | Всегда актуальна |
| HeyGen API | v1/v2 | REST API |
| HeyGen SDK | Последняя | Для разработчиков |

## Системные требования

### Для веб-доступа

| Требование | Минимум | Рекомендуется |
|------------|---------|---------------|
| Браузер | Chrome 90+, Firefox 88+, Safari 14+, Edge 90+ | Последний Chrome |
| Интернет | 10 Мбит/с | 25 Мбит/с+ |

### Для разработки API/SDK

| Требование | Версия |
|------------|--------|
| Node.js | 16+ |
| Python | 3.8+ |
| API ключ | Из панели HeyGen |

## Доступ к HeyGen

### Шаг 1: Создание аккаунта

1. Посетите https://www.heygen.com
2. Нажмите "Sign Up" или "Get Started"
3. Подтвердите адрес электронной почты

### Шаг 2: Выбор тарифа

| Тариф | Функции | Цена |
|-------|----------|------|
| Free | Ограниченные минуты, водяной знак | Бесплатно |
| Pro | Больше минут, без водяного знака | $29/мес |
| Enterprise | Пользовательские лимиты, API | Связаться с отделом продаж |

### Шаг 3: Получение API ключа

1. Войдите в панель управления HeyGen
2. Перейдите в Settings → API
3. Сгенерируйте API ключ

## Установка API

### Python SDK

```bash
pip install heygen-python
```

### Node.js SDK

```bash
npm install heygen-node
```

## Конфигурация

```python
from heygen import HeyGen

heygen = HeyGen(api_key="your_api_key")
```

## Частые проблемы

### Ошибки аутентификации

**Решение:** Проверьте API ключ

### Ограничение скорости

**Решение:** Проверьте лимиты тарифа
