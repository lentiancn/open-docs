## What is Bootstrap 5?

Bootstrap 5 is the latest version of the popular CSS framework for building responsive, mobile-first sites. It features new components, faster JavaScript, and no jQuery dependency.

### Key Features

- Mobile-first approach
- Responsive grid system
- Pre-built components
- Powerful JavaScript plugins
- Customizable via Sass
- No jQuery dependency (new in v5)

## CDN

### Quick Start (CDN)

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Bootstrap 5 Page</title>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <h1>Hello, Bootstrap 5!</h1>
  
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### CDN Providers

| Provider | CSS | JS |
|----------|-----|-----|
| jsDelivr | https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css | https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js |
| unpkg | https://unpkg.com/bootstrap@5.3.2/dist/css/bootstrap.min.css | https://unpkg.com/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js |
| cdnjs | https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.2/css/bootstrap.min.css | https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.2/js/bootstrap.bundle.min.js |

### Icons CDN

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.1/font/bootstrap-icons.css">
```

## NPM

### Install Bootstrap 5

```bash
# Install Bootstrap
npm install bootstrap

# Install Bootstrap icons
npm install bootstrap-icons
```

### Package.json

```json
{
  "dependencies": {
    "bootstrap": "^5.3.2",
    "bootstrap-icons": "^1.11.1"
  }
}
```

## Bundler

### With Vite

```bash
npm create vite@latest my-app -- --template vanilla
cd my-app
npm install bootstrap@5 sass
npm run dev
```

### vite.config.js

```javascript
import { defineConfig } from 'vite'
import sass from 'sass'

export default defineConfig({
  css: {
    preprocessorOptions: {
      scss: {
        api: 'modern-compiler'
      }
    }
  }
})
```

### main.js

```javascript
import 'bootstrap/dist/css/bootstrap.min.css'
import 'bootstrap/dist/js/bootstrap.bundle.min.js'
```

### With Webpack

```bash
npm install bootstrap sass sass-loader css-loader style-loader
```

### webpack.config.js

```javascript
module.exports = {
  module: {
    rules: [
      {
        test: /\.scss$/,
        use: ['style-loader', 'css-loader', 'sass-loader']
      }
    ]
  }
}
```

## Basic Setup

### HTML Template

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Bootstrap 5 Site</title>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.1/font/bootstrap-icons.css">
</head>
<body>
  <nav class="navbar navbar-expand-lg navbar-dark bg-primary">
    <div class="container">
      <a class="navbar-brand" href="#">My App</a>
    </div>
  </nav>

  <main class="container my-5">
    <h1>Welcome to Bootstrap 5!</h1>
  </main>

  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## Customization

### Using Sass

```scss
// Custom Bootstrap
$primary: #ff6b6b;
$secondary: #4ecdc4;
$border-radius: 0.5rem;

@import "bootstrap/scss/bootstrap";
```

### CSS Variables

```css
:root {
  --bs-primary: #ff6b6b;
  --bs-body-bg: #f8f9fa;
}

body {
  background-color: var(--bs-body-bg);
}
```

### Theme Configurator

1. Visit https://bootstrap.build/
2. Customize colors, components
3. Download custom CSS
