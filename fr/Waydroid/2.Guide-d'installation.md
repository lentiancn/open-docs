# Guide d'installation

## Instructions d'installation spécifiques à chaque distribution

### Arch Linux

Les utilisateurs d'Arch peuvent consulter les instructions complètes d'installation sur Arch Wiki :
[https://wiki.archlinux.org/title/Waydroid](https://wiki.archlinux.org/title/Waydroid)

### Postmarket OS

La communauté Postmarket fournit des instructions d'installation et des conseils de dépannage :
[https://wiki.postmarketos.org/wiki/Waydroid](https://wiki.postmarketos.org/wiki/Waydroid)

### Fedora

Waydroid peut être installé depuis le dépôt officiel de paquets :

```bash
sudo dnf install waydroid
```

Après l'installation, lancez Waydroid depuis le menu des applications et procédez à l'initialisation en collant ces URL dans les champs OTA :

OTA système : `https://ota.waydro.id/system`

OTA fournisseur : `https://ota.waydro.id/vendor`

#### Silverblue/Kinoite/...

Les mêmes instructions s'appliquent aux variantes immuables de Fedora, mais vous devez utiliser `rpm-ostree` au lieu de `dnf` :

```bash
rpm-ostree install waydroid
```

### Ubuntu/Debian et dérivés

Assurez-vous d'avoir activé la session Wayland (Ubuntu 22.04+)

* Installer les prérequis

```bash
sudo apt install curl ca-certificates -y
```

* Ajouter le dépôt officiel

```bash
curl -s https://repo.waydro.id | sudo bash
```

Si le script ne détecte pas votre distribution, vous pouvez fournir une option valide en ajoutant `-s <DISTRO>`. Les valeurs actuellement prises en charge sont : **mantic**, **focal**, **jammy**, **kinetic**, **lunar**, **noble**, **plucky**, **questing**, **bookworm**, **bullseye**, **trixie**, **sid**

* Installer waydroid

```bash
sudo apt install waydroid -y
```

Ensuite, lancez Waydroid depuis le menu des applications.

## Dépannage

### Démarrer Waydroid manuellement

Pour démarrer Waydroid sans systemctl, vous devez suivre quelques étapes simples :

**Démarrer d'abord le conteneur :**

```bash
sudo waydroid container start
```

**Et dans un nouvel onglet de terminal, démarrer la session waydroid (sans ***sudo*****):**

```bash
waydroid session start
```

Une fois démarré et que vous voyez "Android with user 0 is ready", il est sûr de lancer une application depuis le menu des applications.

### Lancer Waydroid en mode plein écran

```bash
waydroid show-full-ui
```

### Lancer Waydroid en mode multi-fenêtres

Nous devons d'abord définir la propriété pendant qu'une session Waydroid est en cours d'exécution :

```bash
waydroid prop set persist.waydroid.multi_windows true
```

Après cela, nous pouvons redémarrer la session :

```bash
waydroid session stop
```

Nous sommes alors prêts à lancer une application, qui démarrera en mode multi-fenêtres.

### Réinstaller Waydroid

Si vous avez besoin de réinstaller complètement :

```bash
waydroid session stop
sudo waydroid container stop
sudo apt remove waydroid  # ou commande de désinstallation correspondante de la distribution
# Après le redémarrage, effectuer le nettoyage
sudo rm -rf /var/lib/waydroid /home/.waydroid ~/waydroid ~/.share/waydroid ~/.local/share/applications/*aydroid* ~/.local/share/waydroid
```
