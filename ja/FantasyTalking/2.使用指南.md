# FantasyTalking 使用ガイド

このガイドでは、FantasyTalkingを使用してトッキングヘッド動画を生成する方法を説明します。

## クイックスタート

### 基本的な使用法

```bash
python inference.py \
  --source_image images/face.jpg \
  --audio audio/speech.wav \
  --output results/output.mp4
```

## 入力要件

### 画像要件

- 形式：JPG、PNG
- 解像度：512x512以上
- 顔：正面向き、明確な特徴
- 背景：シンプルな背景が推奨

### 音声要件

- 形式：WAV、MP3
- サンプルレート：16000 Hz以上
- 長さ：制限なし
- 言語：多言語対応

## 高度な使用法

### 顔強調

```bash
python inference.py \
  --source_image images/face.jpg \
  --audio audio/speech.wav \
  --output results/output.mp4 \
  --face_enhancer
```

### バッチ処理

```bash
python batch_inference.py \
  --image_dir images/ \
  --audio_dir audio/ \
  --output_dir results/
```

### カスタムFPS

```bash
python inference.py \
  --source_image images/face.jpg \
  --audio audio/speech.wav \
  --output results/output.mp4 \
  --fps 30
```

## 設定

### 設定ファイル

`config.yaml`を編集：

```yaml
inference:
  face_enhancer: true
  fps: 25
  output_resolution: 512
  
model:
  checkpoint: "checkpoints/fantasytalking.pth"
```

## モデルトレーニング

### データセットの準備

```bash
python prepare_dataset.py \
  --video_dir videos/ \
  --output_dir dataset/
```

### トレーニング

```bash
python train.py \
  --config config/train.yaml \
  --epochs 100
```

## ベストプラクティス

### 1. 高品質な画像を使用

- 清晰で正面からの写真を使用
- 良好な照明条件
- 高解像度推奨

### 2. 音声品質を一致させる

- 背景噪音のない明確な音声
- 適切なサンプルレート
- 顔の人種と言語を一致させる

### 3. 後処理

- より良い品質のために顔強調を使用
- 必要に応じてFPSを調整

## トラブルシューティング

### 動画が暗すぎる？

ソース画像の照明を確認。

### 音声が同期していない？

音声のサンプルレートが正しいことを確認。

### 品質が悪い？

顔強調オプションを試してください。
