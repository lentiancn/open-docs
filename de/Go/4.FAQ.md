# Go FAQ

> Häufig gestellte Fragen

## Installation

### Windows: go Befehl nicht gefunden?

**Lösung**: `C:\Go\bin` zum PATH hinzufügen.

### Proxy-Konfiguration?

```bash
go env -w GOPROXY=https://goproxy.cn,direct
```

---

## Entwicklung

### "cannot find package"?

**Lösung**:
1. `go mod init` ausführen
2. `go mod tidy` ausführen

### := vs var?

- `var`: Explizit, außerhalb von Funktionen
- `:=`: Kurz, nur innerhalb von Funktionen

## Concurrent

### Goroutine Leak?

```go
func worker(ctx context.Context) {
    for {
        select {
        case <-ctx.Done():
            return
        }
    }
}
```
