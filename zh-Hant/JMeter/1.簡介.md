# Apache JMeter - 簡介

## 什麼是 Apache JMeter？

Apache JMeter™ 是一個開源軟體，是一個 100% 純 Java 應用程式，旨在進行負載測試和效能測量。它最初是為測試 Web 應用程式而設計的，但後來已擴展到其他測試功能。

## JMeter 可以做什麼？

Apache JMeter 可用於測試靜態和動態資源、動態 Web 應用程式的效能。它可以模擬伺服器、伺服器組、網路或物件上的重負載，以測試其強度或分析在不同負載類型下的整體效能。

### 主要功能

- **多協定支援**：Web (HTTP、HTTPS)、SOAP/REST Web 服務、FTP、透過 JDBC 的資料庫、LDAP、JMS、郵件、TCP、Java 物件
- **完整測試 IDE**：從瀏覽器或原生應用程式錄製測試計劃
- **CLI 模式**：從任何 Java 相容作業系統執行負載測試
- **動態 HTML 報告**：完整且可直接展示
- **資料擷取**：支援 HTML、JSON、XML 和正規表示式
- **100% Java**：完全可攜
- **多執行緒**：透過多個執行緒同時取樣
- **高度可擴展**：可外掛的取樣器和可指令碼化的取樣器（Groovy、BeanShell）
- **CI 整合**：支援 Maven、Gradle 和 Jenkins

## JMeter 不是什麼

JMeter 不是瀏覽器 - 它在通訊協定層級運作。它不會執行 HTML 頁面中發現的 JavaScript，也不會像真正的瀏覽器一樣呈現 HTML 頁面。

## 開始使用

1. 從官方網站下載 JMeter
2. 執行 JMeter 指令碼（Windows 為 jmeter.bat，Linux/Mac 為 jmeter.sh）
3. 使用 GUI 建立您的測試計劃
4. 執行測試並分析結果
