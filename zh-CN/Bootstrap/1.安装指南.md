## 简介

### 什么是 Bootstrap？

Bootstrap 是世界上最流行的前端开源工具包，由 Twitter 开发并维护。它是一个用于快速开发响应式网站和 Web 应用程序的 CSS、JavaScript 框架。

### Bootstrap 版本

| 版本 | 发布日期 | 状态 |
|------|----------|------|
| Bootstrap 5.3 | 2023年5月 | 当前稳定版 |
| Bootstrap 5.2 | 2022年9月 | 支持 |
| Bootstrap 5.1 | 2021年8月 | 支持 |
| Bootstrap 5.0 | 2021年7月 | 支持 |

### Bootstrap 5 新特性

- 全新配色方案和样式改进
- 更好的表单控件
- 新的 utilitis API
- 改进的网格系统
- 更好的自定义方式
- 不再依赖 jQuery（但仍兼容）
- 放弃对 Internet Explorer 的支持

## 系统要求

### 浏览器支持

Bootstrap 5 支持所有主流浏览器的最新版本：

| 浏览器 | 支持版本 |
|--------|----------|
| Chrome | 90+ |
| Firefox | 88+ |
| Safari | 14+ |
| Edge | 90+ |
| Opera | 76+ |

**不再支持：**
- Internet Explorer 10 和 11
- Microsoft Edge Legacy

#### 开发环境
- 文本编辑器（VS Code、Sublime Text、WebStorm 等）

#### 可选工具
- Node.js（用于使用 npm 包）
- Git（用于版本控制）
- Sass 编译器（用于自定义样式）

## 安装方法

### 方法一：使用 CDN（最简单）

CDN（内容分发网络）是最快的开始方式，不需要下载任何文件。

#### CSS only

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 入门</title>
    <!-- 引入 Bootstrap CSS -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>Hello, Bootstrap!</h1>
    
    <!-- 引入 Bootstrap JS -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

#### CDN 提供商

**jsDelivr（推荐）**
```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

**unpkg**
```html
<link href="https://unpkg.com/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://unpkg.com/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

**cdnjs**
```html
<link href="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

### 方法二：使用 npm 安装

#### 1. 安装 Node.js

参考 Angular 安装指南安装 Node.js。

#### 2. 创建项目并安装 Bootstrap

```bash
# 创建新项目目录
mkdir my-bootstrap-project
cd my-bootstrap-project

# 初始化 npm
npm init -y

# 安装 Bootstrap
npm install bootstrap

# 安装 Popper（用于下拉菜单、弹出框等）
npm install @popperjs/core
```

#### 3. 在项目中使用

```javascript
// 在 JavaScript 中引入
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.min.js';
```

或者在 HTML 中引用：

```html
<link rel="stylesheet" href="node_modules/bootstrap/dist/css/bootstrap.min.css">
<script src="node_modules/bootstrap/dist/js/bootstrap.bundle.min.js"></script>
```

### 方法三：下载源码

#### 1. 下载 Bootstrap

访问 Bootstrap 官网 https://getbootstrap.com/ 点击 Download 按钮下载预编译的 CSS 和 JS 文件。

#### 2. 解压文件

```
bootstrap-5.3.0/
├── css/
│   ├── bootstrap.css
│   ├── bootstrap.css.map
│   ├── bootstrap.min.css
│   ├── bootstrap.min.css.map
│   ├── bootstrap.rtl.css
│   ├── bootstrap.rtl.min.css
│   └── ...
└── js/
    ├── bootstrap.bundle.js
    ├── bootstrap.bundle.js.map
    ├── bootstrap.bundle.min.js
    └── ...
```

#### 3. 在 HTML 中引用

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 项目</title>
    <link href="css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <!-- 内容 -->
    <script src="js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### 方法四：使用 Sass 源码（高级用户）

```bash
# 安装 Bootstrap 源码
npm install bootstrap

# 或者使用 Sass
npm install bootstrap-sass
```

## 在 Windows 上安装

### 方法一：使用 CDN（推荐）

1. 创建一个新的 HTML 文件
2. 在 `<head>` 中添加 Bootstrap CSS 链接
3. 在 `<body>` 末尾添加 Bootstrap JS 脚本

### 方法二：使用 npm

1. **安装 Node.js**
   - 访问 https://nodejs.org/
   - 下载 LTS 版本
   - 运行安装程序

2. **打开命令提示符**

3. **创建项目**
   ```cmd
   mkdir bootstrap-project
   cd bootstrap-project
   npm init -y
   npm install bootstrap
   ```

4. **创建 HTML 文件**
   ```html
   <!DOCTYPE html>
   <html lang="zh-CN">
   <head>
       <meta charset="UTF-8">
       <meta name="viewport" content="width=device-width, initial-scale=1.0">
       <link href="node_modules/bootstrap/dist/css/bootstrap.min.css" rel="stylesheet">
   </head>
   <body>
       <div class="container mt-5">
           <h1>Hello, Bootstrap!</h1>
       </div>
       <script src="node_modules/bootstrap/dist/js/bootstrap.bundle.min.js"></script>
   </body>
   </html>
   ```

### 方法三：使用 Visual Studio Code

1. 下载并安装 VS Code：https://code.visualstudio.com/
2. 安装 "Bootstrap 5 Quick Snippets" 扩展
3. 创建新 HTML 文件，输入 `bs5` 即可快速生成 Bootstrap 模板

## 在 macOS 上安装

### 方法一：使用 Homebrew 安装 Node.js（推荐）

1. **安装 Homebrew**
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```

2. **安装 Node.js**
   ```bash
   brew install node
   ```

3. **创建项目**
   ```bash
   mkdir bootstrap-project
   cd bootstrap-project
   npm init -y
   npm install bootstrap
   ```

### 方法二：使用 CDN

直接下载或创建 HTML 文件使用 CDN 链接。

### 方法三：使用下载包

1. 访问 https://getbootstrap.com/
2. 下载预编译的压缩包
3. 解压并添加到项目

## 在 Linux 上安装

### Ubuntu/Debian

```bash
# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装 Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs

# 创建项目
mkdir bootstrap-project
cd bootstrap-project
npm init -y
npm install bootstrap
```

### Fedora

```bash
# 添加 Node.js 仓库
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -

# 安装 Node.js
sudo dnf install nodejs

# 创建项目
mkdir bootstrap-project
cd bootstrap-project
npm init -y
npm install bootstrap
```

## 创建第一个项目

### 项目结构

```

├── index.html
├── css/
```

## 故障排除

### 常见问题

#### 1. 样式不生效

**检查清单：**
- [ ] CSS 文件链接是否正确
- [ ] CDN 链接是否正确
- [ ] 文件路径是否正确
- [ ] 浏览器缓存是否需要清除

**解决方案：**
```html
<!-- 添加版本号强制刷新 -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css?v=1" rel="stylesheet">
```

#### 2. JavaScript 功能不工作

**检查清单：**
- [ ] 是否引入了 bootstrap.bundle.min.js（不是 bootstrap.min.js）
- [ ] JS 文件是否在 body 结束前引入
- [ ] 是否有 JavaScript 控制台错误

#### 3. 下拉菜单不工作

**解决方案：**
- 确保引入了 Popper.js
- 使用 bootstrap.bundle.min.js
- 检查 data-bs-toggle 属性

```html
<!-- 正确的下拉菜单 -->
<div class="dropdown">
    <button class="btn btn-secondary dropdown-toggle" type="button" data-bs-toggle="dropdown">
        下拉菜单
    </button>
    <ul class="dropdown-menu">
        <li><a class="dropdown-item" href="#">操作一</a></li>
        <li><a class="dropdown-item" href="#">操作二</a></li>
    </ul>
</div>
```

#### 4. 模态框不工作

**检查清单：**
- [ ] 模态框是否在页面加载完成前初始化
- [ ] 是否有 z-index 冲突

#### 5. 移动端响应式问题

**确保添加视口 meta 标签：**
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### 使用 Sass 自定义

```scss
// 自定义 Bootstrap 变量

$secondary: #4ecdc4;

// 引入 Bootstrap
@import "~bootstrap/scss/bootstrap";
```
