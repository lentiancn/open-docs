# Руководство по установке Element UI

Element UI — это библиотека компонентов пользовательского интерфейса для приложений Vue 2.x.

## Обзор версий

| Версия | Версия Vue | Статус | Дата выпуска |
|--------|------------|--------|---------------|
| Element UI 2.x | Vue 2.6+ | Текущая | 2020 |
| Element UI 1.x | Vue 2.x | Устаревшая | 2016 |

**Примечание:** Element UI предназначен только для Vue 2. Для Vue 3 используйте Element Plus.

### Element UI и Element Plus

| Функция | Element UI (Vue 2) | Element Plus (Vue 3) |
|---------|--------------------|---------------------|
| Версия Vue | 2.x | 3.x |
| TypeScript | Ограниченная | Полная поддержка |
| Статус | Режим обслуживания | Активная разработка |

## Предварительные требования

### Системные требования

| Требование | Версия |

## Способы установки

### Способ 1: npm (Рекомендуется)

```bash
# Установить Element UI
npm install element-ui --save
```

### Способ 2: Использование Vue CLI

```bash
# Установить Vue CLI
npm install -g @vue/cli

# Создать проект с шаблоном Element UI
vue init webpack my-element-app
```

### Способ 3: CDN

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Демо Element UI CDN</title>
  <!-- Импорт Vue 2 -->
  <script src="https://unpkg.com/vue@2.6.14/dist/vue.js"></script>
  <!-- Импорт стилей Element UI -->
  <link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
  <!-- Импорт Element UI -->
  <script src="https://unpkg.com/element-ui/lib/index.js"></script>
</head>
<body>
  <div id="app">
    <el-button @click="visible = true">Открыть диалог</el-button>
    <el-dialog :visible.sync="visible" title="Hello Element UI">
      <p>Добро пожаловать в Element UI!</p>
    </el-dialog>
  </div>

  <script>
    new Vue({
      el: '#app',
      data: {
        visible: false
      }
    })
  </script>
</body>
</html>
```

## Настройка проекта

### Шаг 1: Создание проекта Vue

```bash
# Использование Vue CLI
npm install -g @vue/cli
vue create my-element-app
cd my-element-app
```

### Шаг 2: Установка Element UI

```bash
npm install element-ui --save
```

### Шаг 3: Импорт в main.js

#### Полный импорт

```javascript
import Vue from 'vue'
import App from './App.vue'

// Импорт Element UI
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

// Использование Element UI
Vue.use(ElementUI)

Vue.config.productionTip = false

new Vue({
  render: h => h(App)
}).$mount('#app')
```

#### Импорт по требованию

Сначала установите плагин:

```bash
npm install babel-plugin-component -D
```

Настройте babel.config.js:

```javascript
module.exports = {
  presets: ['@vue/cli-plugin-babel/preset'],
  plugins: [
    [
      'component',
      {
        libraryName: 'element-ui',
        styleLibraryName: 'theme-chalk'
      }
    ]
  ]
}
```

Затем импортируйте компоненты:

```javascript
import { Button, Dialog, Input } from 'element-ui'

Vue.component(Button.name, Button)
Vue.component(Dialog.name, Dialog)
Vue.component(Input.name, Input)

// Или использовать локально в компонентах
export default {
  components: {
    [Button.name]: Button
  }
}
```

## Устранение неполадок

### Компонент не найден

**Проблема:** Компоненты не отображаются

**Решения:**
1. Убедитесь, что установлено: `npm list element-ui`
2. Проверьте импорт в main.js
3. Убедитесь, что Vue.use() вызван

### Стили не применяются

**Проблема:** Стили отсутствуют

**Решения:**
1. Импортировать CSS: `import 'element-ui/lib/theme-chalk/index.css'`
2. Проверьте конфигурацию CSS-загрузчика webpack

### Совместимость с IE10

```bash
# Установить полифилы
npm install babel-polyfill --save
```

```javascript
// main.js
import 'babel-polyfill'
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```

## Миграция на Element Plus (Vue 3)

При миграции на Vue 3 используйте Element Plus:

```bash
npm install element-plus --save
```

Основные изменения:
1. Имя пакета: `element-ui` → `element-plus`
2. Путь импорта: `element-ui/lib/index` → `element-plus`
3. Некоторые свойства компонентов могут измениться

## Рекомендации

1. **Используйте импорт по требованию** - Уменьшает размер пакета
2. **Настраивайте тему** - Используйте переменные SCSS
3. **Используйте Vue 2.6+** - Лучшая совместимость
4. **Настройте babel** - Для продакшен-сборок
