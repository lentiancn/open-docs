## What is Mockito?

Mockito is a popular mock framework for Java unit testing. It allows you to create and configure mock objects to simplify testing.

---

## Maven

### Add Dependency

```xml
<dependencies>
    <!-- Mockito Core -->
    <dependency>
        <groupId>org.mockito</groupId>
        <artifactId>mockito-core</artifactId>
        <version>5.8.0</version>
        <scope>test</scope>
    </dependency>
</dependencies>
```

### With JUnit 4

```xml
<dependency>
    <groupId>org.mockito</groupId>
    <artifactId>mockito-inline</artifactId>
    <version>5.8.0</version>
    <scope>test</scope>
</dependency>
```

### With JUnit 5 (Jupiter)

```xml
<dependency>
    <groupId>org.mockito</groupId>
    <artifactId>mockito-junit-jupiter</artifactId>
    <version>5.8.0</version>
    <scope>test</scope>
</dependency>
```

## Gradle

### Add Dependency

```groovy
// Groovy DSL
testImplementation 'org.mockito:mockito-core:5.8.0'
```

### With JUnit 5

```groovy
testImplementation 'org.mockito:mockito-junit-jupiter:5.8.0'
```

### Using Mock Maker

```groovy
// For mocking final classes and methods
testImplementation 'org.mockito:mockito-inline:5.8.0'
```

### Kotlin DSL

```kotlin
testImplementation("org.mockito:mockito-core:5.8.0")
testImplementation("org.mockito:mockito-junit-jupiter:5.8.0")
```

## Gradle (Kotlin)

```kotlin
plugins {
    id("java-test-fixtures")
}

dependencies {
    testImplementation("org.mockito:mockito-core:5.8.0")
    testImplementation("org.mockito:mockito-junit-jupiter:5.8.0")

```

## Standalone JAR

### Download JAR

Download from Maven Central:

- [mockito-core-5.8.0.jar](https://repo1.maven.org/maven2/org/mockito/mockito-core/5.8.0/mockito-core-5.8.0.jar)

### Add to Classpath

```bash
# Compile with Mockito
javac -cp mockito-core-5.8.0.jar:junittest.jar MyTest.java

# Run tests
java -cp .:mockito-core-5.8.0.jar:junittest.jar:junit.jar:hamcrest.jar org.junit.runner.JUnitCore MyTest
```

## Version Compatibility

### Mockito 5.x

| Java Version | Mockito Version |
|--------------|----------------|
| Java 8+ | 5.8.0 |
| Java 11+ | 5.8.0 |
| Java 17+ | 5.8.0 |
| Java 21+ | 5.8.0 |

### Mockito 4.x (Legacy)

```xml
<dependency>
    <groupId>org.mockito</groupId>
    <artifactId>mockito-core</artifactId>
    <version>4.11.0</version>
</dependency>
```

## IDE Support

### IntelliJ IDEA

- Built-in support for Mockito
- Install Mockit-o plugin for enhanced features

### Eclipse

- Install Eclipse IDE for Java Developers
- Use Mockito plugin from Eclipse Marketplace

### VS Code

- Install Java Extension Pack
- Use Java Test Runner

## Dependencies

### Required Dependencies

Mockito requires:
- Java 8 or higher
- Byte Buddy (automatically included)

### Optional Dependencies

```xml
<!-- For JUnit 4 -->
<dependency>
    <groupId>junit</groupId>
    <artifactId>junit</artifactId>
    <version>4.13.2</version>
    <scope>test</scope>
</dependency>

<!-- For JUnit 5 -->
<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter</artifactId>
    <version>5.10.0</version>
    <scope>test</scope>
</dependency>

<!-- For TestNG -->
<dependency>
    <groupId>org.testng</groupId>
    <artifactId>testng</artifactId>
    <version>7.8.0</version>
    <scope>test</scope>
</dependency>
```
