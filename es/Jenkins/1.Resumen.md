# Jenkins - Resumen

## Descripción

Jenkins es un servidor de automatización de código abierto. Ayuda a automatizar procesos de desarrollo de software como construcción, pruebas y despliegue, facilitando la integración continua y entrega continua (CI/CD).

## Características

- Integración y entrega continua
- ecosistema de Plugins (1800+)
- Builds distribuidos
- Configuración web
- Soporte para Pipelines
