# Java Introduction

> Java is an object-oriented programming language

---

## Overview

Java is an object-oriented programming language released by Sun Microsystems in 1995, now maintained by Oracle. Known for "write once, run anywhere" cross-platform capability, widely used in enterprise applications, Android development, and big data processing.

---

## Features

- **Object-oriented**: Pure OOP design
- **Cross-platform**: JVM enables "write once, run anywhere"
- **Automatic memory management**: Garbage collection
- **Multithreading**: Built-in multithreading support
- **Security**: Security sandbox mechanism
- **Rich ecosystem**: Large open-source libraries

---

## Use Cases

- Enterprise applications
- Android development
- Web applications
- Big data processing
- Distributed systems
