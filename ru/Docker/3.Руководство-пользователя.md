# Docker Руководство пользователя

Основные концепции, использование и руководство по разработке.

## Операции с образами

### Загрузка образов

```
docker pull nginx
docker pull ubuntu:22.04
```

### Сборка образов

```
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
EXPOSE 3000
CMD ["npm", "start"]
```

```
docker build -t my-app:tag .
```

### Управление образами

```
docker images
docker rmi nginx
```

## Операции с контейнерами

### Создание и запуск контейнеров

```
# Интерактивный контейнер
docker run -it ubuntu /bin/bash

# Фоновый контейнер
docker run -d nginx

# Проброс портов
docker run -d -p 8080:80 nginx

# Переменные окружения
docker run -e MYSQL_ROOT_PASSWORD=123456 mysql
```

### Управление контейнерами

```
docker ps              # Работающие контейнеры
docker ps -a          # Все контейнеры
docker start container_id
docker stop container_id
docker rm container_id
```

### Вход в контейнер

```
docker exec -it container_id /bin/bash
```

### Просмотр логов

```
docker logs container_id
docker logs -f container_id
```

## Написание Dockerfile

### Базовые инструкции

```
FROM node:18-alpine
WORKDIR /app
COPY . .
RUN npm install
EXPOSE 3000
CMD ["npm", "start"]
```

### Многоэтапная сборка

```
# Этап сборки
FROM node:18 AS builder
WORKDIR /app
COPY . .
RUN npm run build

# Продакшен-этап
FROM nginx:alpine
COPY --from=builder /app/dist /usr/share/nginx/html
```

## Docker Compose

### docker-compose.yml

```
version: '3.8'

services:
  web:
    build: .
    ports:
      - "3000:3000"
  db:
    image: postgres:15
    environment:
      POSTGRES_PASSWORD: secret
```

### Частые команды

```
docker-compose up -d
docker-compose down
docker-compose logs -f
```

## Управление сетью

### Проброс портов

```
docker run -p 8080:80 nginx
```

### Пользовательские сети

```
docker network create my-network
docker run --network my-network nginx
```

## Управление данными

### Тома

```
docker volume create my-data
docker run -v my-data:/data nginx
```

### Bind mount

```
docker run -v /host/path:/container/path nginx
```

## Ограничение ресурсов

```
# Ограничение памяти
docker run -m 512m nginx

# Ограничение CPU
docker run --cpus=0.5 nginx
```

## Мониторинг и отладка

```
docker stats
docker inspect container_id
docker logs container_id
```

## Очистка

```
docker system prune -a
docker volume prune
```
