# Руководство пользователя

Это руководство подробно описывает основные возможности и способы использования Bootstrap 5: сетку, компоненты, служебные классы и другое.

## Сетка

Сетка — это основа Bootstrap. Она обеспечивает 12-колоночный макет.

### Базовое использование

```html
<div class="container">
    <div class="row">
        <div class="col">Колонка 1</div>
        <div class="col">Колонка 2</div>
        <div class="col">Колонка 3</div>
    </div>
</div>
```

### Адаптивные контрольные точки

| Класс | Ширина экрана | Описание |
|------|----------------|----------|
| `.col-xs` | < 576px | Сверхмалые экраны |
| `.col-sm` | ≥ 576px | Малые экраны |
| `.col-md` | ≥ 768px | Средние экраны |
| `.col-lg` | ≥ 992px | Большие экраны |
| `.col-xl` | ≥ 1200px | Очень большие экраны |
| `.col-xxl` | ≥ 1400px | Экстрабольшие экраны |

### Смещение колонок

```html
<div class="row">
    <div class="col-md-4">.col-md-4</div>
    <div class="col-md-4 offset-md-4">.col-md-4 .offset-md-4</div>
</div>
```

### Адаптивные колонки

```html
<div class="row">
    <div class="col-12 col-md-6 col-lg-4">
        На телефоне — полная ширина, на планшете — половина, на большом экране — треть
    </div>
</div>
```

## Навигация

### Базовая навигация

```html
<ul class="nav">
    <li class="nav-item">
        <a class="nav-link active" href="#">Главная</a>
    </li>
    <li class="nav-item">
        <a class="nav-link" href="#">О нас</a>
    </li>
    <li class="nav-item">
        <a class="nav-link" href="#">Контакты</a>
    </li>
</ul>
```

### Навигационная панель

```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">Бренд</a>
        <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav">
            <span class="navbar-toggler-icon"></span>
        </button>
        <div class="collapse navbar-collapse" id="navbarNav">
            <ul class="navbar-nav">
                <li class="nav-item">
                    <a class="nav-link active" href="#">Главная</a>
                </li>
                <li class="nav-item">
                    <a class="nav-link" href="#">О нас</a>
                </li>
            </ul>
        </div>
    </div>
</nav>
```

</button>

## Рекомендации

1. **Используйте семантический HTML**: применяйте теги header, nav, main, footer
2. **Не переусердствуйте**: избегайте глубокой вложенности
3. **Сначала мобильные**: проектируйте с учётом мобильных устройств
4. **Используйте служебные классы**: для быстрой настройки стилей
5. **Настраивайте тему**: используйте Sass-переменные для единообразия
