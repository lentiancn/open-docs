# React

用于构建用户界面的 JavaScript 库。

## 简介

React 是 Facebook 维护的用于构建用户界面的 JavaScript 库。

### 主要特性

- **组件化**：基于组件的开发
- **虚拟 DOM**：高效的 DOM 更新
- **Hooks**：函数组件状态管理
- **单向数据流**：数据流向清晰

## 快速开始

### 创建项目

```bash
npm create vite@latest my-app -- --template react
cd my-app
npm install
npm run dev
```

### 访问应用

打开浏览器：http://localhost:5173

## 常用命令

```bash
npm run dev      # 开发服务器
npm run build    # 构建生产版本
npm run preview  # 预览构建结果
```

## 核心概念

- **组件**：可复用的 UI 片段
- **JSX**：JavaScript 语法扩展
- **Hooks**：函数组件增强
- **虚拟 DOM**：性能优化
