# Angular 安裝指南

Angular 環境配置詳細指南。

## 環境要求

### Node.js

- 推薦 Node.js 18+
- 需要 npm 或 yarn

### 檢查 Node.js 版本

```bash
node --version
npm --version
```

## 安裝 Angular CLI

### 安裝

```bash
npm install -g @angular/cli
```

### 驗證安裝

```bash
ng version
```

### 升級

```bash
npm install -g @angular/cli@latest
```

## 創建新項目

### 使用 Angular CLI 創建

```bash
ng new my-app
cd my-app
```

### 互動式選項

- 是否添加路由？Yes/No
- 使用哪種樣式表格式？（SCSS/CSS/Sass/Less）

## 運行應用

### 開發服務器

```bash
ng serve
# 或
npm start
```

訪問：http://localhost:4200

### 熱重載

Angular CLI 支持熱重載，修改代碼後自動刷新。

## 構建生產版本

```bash
ng build --configuration production
```

構建產物位於 `dist/` 目錄。

## 常用命令

| 命令 | 功能 |
|------|------|
| ng new | 創建新項目 |
| ng serve | 啟動開發服務器 |
| ng build | 構建項目 |
| ng generate | 生成組件/服務/模塊等 |
| ng test | 運行單元測試 |
| ng e2e | 運行端到端測試 |

## 項目結構

```
my-app/
├── src/
│   ├── app/
│   │   ├── components/
│   │   ├── services/
│   │   ├── app.component.ts
│   │   └── app.module.ts
│   ├── assets/
│   └── styles.css
├── angular.json
└── package.json
```

## 卸載 Angular CLI

```bash
npm uninstall -g @angular/cli
```
