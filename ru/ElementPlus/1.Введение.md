# ElementPlus Введение

ElementPlus - это библиотека компонентов UI на основе Vue 3.

## Что такое ElementPlus

ElementPlus - это версия Element UI для Vue 3.

## Основные функции

- Поддержка Vue 3
- 70+ компонентов
- Поддержка TypeScript
- Tree Shaking
- Интернационализация
- Настройка темы

## Компоненты

- Базовые: Кнопка, Иконка, Макет
- Форма: Ввод, Выбор, Дата
- Отображение данных: Таблица, Дерево, Пагинация
- Обратная связь: Диалог, Сообщение

## Установка

```bash
npm install element-plus
```
