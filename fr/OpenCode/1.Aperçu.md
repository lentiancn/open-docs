# OpenCode Aperçu

Agent de codage IA open source.

## Qu'est-ce qu'OpenCode ?

OpenCode est un agent de codage IA open source disponible comme interface terminal, application bureau ou extension IDE.

### Fonctionnalités principales

- **LSP activé** - Charge automatiquement les bons LSP
- **Multi-sessions** - Démarrez plusieurs agents en parallèle
- **Partager des liens** - Partagez n'importe quel lien de session
- **GitHub Copilot** - Connectez-vous avec GitHub pour utiliser Copilot
- **ChatGPT Plus/Pro** - Connectez-vous avec OpenAI pour utiliser ChatGPT
- **N'importe quel modèle** - 75+ fournisseurs LLM incluant les modèles locaux
- **N'importe quel éditeur** - Terminal, application bureau et extension IDE
- **Confidentialité d'abord** - Ne stocke pas le code ou les données de contexte
