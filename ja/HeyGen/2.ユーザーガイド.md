# HeyGen ユーザーガイド

このガイドでは、HeyGen用于AIビデオ生成方法について説明します。

## はじめに

### Webインターフェースを使用

1. HeyGenにログイン：https://www.heygen.com
2. ライブラリからアバターを選択またはカスタムをアップロード
3. 入力タイプを選択：テキスト、オーディオ、またはビデオ
4. ビデオを生成してダウンロード

### APIを使用

```python
from heygen import HeyGen

heygen = HeyGen(api_key="your_api_key")

# テキストからビデオを生成
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="Hello, this is my first AI video!",
    voice_id="voice_id"
)

print(f"ビデオURL: {video.url}")
```

## コア機能

### 1. アバター選択

```python
# 利用可能なアバターを一覧表示
avatars = heygen.list_avatars()
for avatar in avatars:
    print(f"{avatar.id}: {avatar.name}")

# 特定のアバターを取得
avatar = heygen.get_avatar(avatar_id="avatar_id")
```

### 2. 音声選択

```python
# 利用可能な音声を一覧表示
voices = heygen.list_voices()
for voice in voices:
    print(f"{voice.id}: {voice.name} ({voice.language})")

# 音声の詳細を取得
voice = heygen.get_voice(voice_id="voice_id")
```

### 3. ビデオ生成

#### テキストから

```python
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="あなたのスクリプトテキスト",
    voice_id="voice_id",
    background="office"  # オプション：背景シーン
)
```

#### オーディオから

```python
video = heygen.generate_video_from_audio(
    avatar_id="avatar_id",
    audio_url="https://example.com/audio.wav",
)
```

#### 写真から (TalkingPhoto)

```python
video = heygen.generate_talking_photo(
    photo_url="https://example.com/photo.jpg",
    script="あなたのスクリプト",
    voice_id="voice_id"
)
```

### 4. バッチ生成

```python
scripts = [
    "最初のビデオスクリプト",
    "2番目のビデオスクリプト",
    "3番目のビデオスクリプト"
]

# 複数のビデオを生成
jobs = heygen.batch_generate(
    avatar_id="avatar_id",
    scripts=scripts,
    voice_id="voice_id"
)

# ジョブのステータスを確認
for job in jobs:
    status = heygen.get_job_status(job.id)
    print(f"ジョブ {job.id}: {status}")
```

### ビデオ編集

```python

# ビデオ設定を更新
updated = heygen.update_video(
    video_id="video_id",
    title="マイ更新ビデオ",
    description="更新された説明"

# ビデオを削除

```

## パラメータリファレンス

### generate_videoパラメータ

| パラメータ | タイプ | 必須 | 説明 |
|------------|--------|------|------|
| avatar_id | string | はい | 使用するアバターID |
| script | string | はい | 話すテキスト |
| voice_id | string | はい | 音声ID |
| background | string | いいえ | 背景シーン |
| speed | float | いいえ |  Speech速度 (0.5-2.0) |
| emotion | string | いいえ |感情スタイル |
| caption | boolean | いいえ | 字幕を有効にする |

### 利用可能な感情

| 感情 | 説明 |
|------|------|
| Neutral | デフォルト表情 |
| Happy | 幸福な表情 |
| Sad | 悲しい表情 |
| Angry | 怒った表情 |
| Surprised | 驚いた表情 |

## Webインターフェース機能

### ダッシュボード

- **プロジェクト**: ビデオプロジェクトを管理
- **テンプレート**: 事前作成テンプレートを使用
- **メディアライブラリ**: アセットを保存

### ビデオエディター

- **タイムライン**: ビデオセグメントを編集
- **字幕**: 字幕を追加/編集

## ベストプラクティス

### 1. スクリプト作成

- 文を短く明確に
- 自然言語を使用
- 複雑な用語を避ける

### 2. アバター選択

- ユースケースにアバターをマッチ
- 対象者の好みを考慮
- 複数のアバターをテスト

### 3. 音声選択

- ブランドに音声をマッチ
- 言語要件を考慮
- サンプルスクリプトでテスト

### 4. ビデオ最適化

- 適切な解像度を使用
- プラットフォーム用に最適化（ソーシャルメディア、Webサイト）
- アクセシビリティのために字幕を追加

## トラブルシューティング

### ビデオ品質の問題

**問題：** ビデオがピクセル化する

**解決策：**
- より高い解像度設定を使用
- ソース画像が高品質であることを確認
- インターネット接続を確認

### オーディオの問題

**問題：** オーディオがビデオと一致しない

**解決策：**
- 提供された音声を使用
- オーディオが正しくフォーマットされていることを確認
- リップsync設定を確認

### 処理時間

**問題：** ビデオの処理に時間がかかる

**解決策：**
- クレジット残高を確認
- バッチ処理にキューを使用
- 優先処理を検討
