# OpenClaw よくある質問

## 基本

### OpenClaw何？

自分が持ってるデバイスで動くツールです。チャットアプリをAIにつながるようにするやつ。

### お金かかる？

ソフト自体は無料（オープンソース）。かかるのは：
- 動かすサーバーやパソコン
- AIサービスのAPIキー利用料

### どこで動く？

- デスクトップ：macOS、Linux、Windows
- スマホ：iOS、Android
- チャット：WhatsApp、Telegram、Discord、iMessage、Signal、Slackなど20以上

## インストール

### 失敗したら？

1. Node.jsバージョン確認：`node --version`（22以上必要）
2. 診断実行：`openclaw doctor`
3. ログ確認：`openclaw logs`

### Raspberry Piで動く？

OK。Raspberry Pi 4以上、Raspberry Pi OSかUbuntuで動く。

### Mac必要？

不要。Node.jsが入るなら何でもok。iMessageチャンネルだけAppleの制限でMacが必要。

## 設定

### 設定ファイルはどこ？

`~/.openclaw/openclaw.json`

### アクセス制御？

`dmPolicy`で：
- `pairing`：初めての人にはペアリングコードが必要
- `allowlist`：白名单の人のみ
- `open`：全员OK
- `disabled`：オフ

## 使い道

### チャンネル追加？

```bash
openclaw channels login whatsapp
```

### 複数のチャンネル同時？

OK。一つのGatewayで複数のチャットアプリにつながる。

### AIが前の会話を覚えてる？

自動保存。再来の設定で履歴期間を调整できる。

## 安全

### データ安全？

OK。全部自分のデバイスに保存。第三方には行かない（AIプロバイダーのAPI呼び出しを除く）。

## トラブル

### Gateway动かない

1. ポート確認：`openclaw gateway status`
2. ログ確認：`openclaw logs`
3. 診断：`openclaw doctor`

### AIが返事しない

1. APIキー確認
2. アカウント残高確認
3. ログのエラー確認
