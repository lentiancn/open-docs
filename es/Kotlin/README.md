# Kotlin

Kotlin es un lenguaje de programación moderno desarrollado por JetBrains que se ejecuta en la JVM.

## Introducción

Kotlin es un lenguaje de programación de tipo estático que puede ejecutarse en la máquina virtual de Java y también puede compilarse a código JavaScript.

## Características Principales

- **Conciso**: Reduce el código repetitivo
- **Seguro**: Diseño con seguridad nula
- **Interoperable**: 100% interoperable con Java
- **Herramientas amigables**: Excelente soporte de IDE
- **Corrutinas**: Potente soporte para programación asíncrona

## Inicio Rápido

### Instalación

```bash
# macOS
brew install kotlin

# Linux
sudo apt install kotlin

# SDKMAN
sdk install kotlin
```

### Compilación y Ejecución

```bash
kotlinc hello.kt -include-runtime -d hello.jar
java -jar hello.jar
```
