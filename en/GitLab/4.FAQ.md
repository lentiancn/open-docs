# GitLab FAQ

> Frequently Asked Questions about GitLab

---

## General

### What is GitLab?

GitLab is a complete DevOps platform that provides Git repository management, issue tracking, CI/CD pipelines, and more. It can be used as a SaaS service (gitlab.com) or self-hosted.

### GitLab vs GitHub?

| Feature | GitLab | GitHub |
|---------|--------|--------|
| CI/CD | Built-in | GitHub Actions |
| Self-hosted | Yes | Enterprise only |
| Container Registry | Built-in | GitHub Packages |
| Wiki | Built-in | Built-in |
| Issue Tracking | Advanced | Basic |

## Installation

### What are the minimum requirements?

| Resource | Minimum |
|----------|---------|
| CPU | 2 cores |
| RAM | 4 GB |
| Storage | 10 GB |

### Installation fails with dependency error?

**Solution**:
1. Update system packages
2. Install missing dependencies manually
3. Check for conflicting packages

### How to change external URL after installation?

```bash
# Edit configuration
sudo vim /etc/gitlab/gitlab.rb
external_url 'https://new-url.com'

# Reconfigure
sudo gitlab-ctl reconfigure
```

## Usage

### How to reset root password?

```bash
# Open Rails console
sudo gitlab-rails console -e production

# Find and reset password
user = User.where(id: 1).first
user.password = 'new_password'
user.password_confirmation = 'new_password'
user.save!
```

### How to backup GitLab?

```bash
# Create backup
sudo gitlab-backup create

# Backup location: /var/opt/gitlab/backups
```

### How to restore from backup?

```bash
# Stop GitLab
sudo gitlab-ctl stop

# Restore backup
sudo gitlab-backup restore BACKUP=timestamp

# Restart GitLab
sudo gitlab-ctl start
```

## CI/CD

### Pipeline not triggering?

**Solution**:
1. Check .gitlab-ci.yml syntax
2. Verify GitLab Runner is registered
3. Check runner status in project settings

### How to use Docker in CI?

```yaml
image: docker:latest

services:
  - docker:dind

build:
  script:
    - docker build -t my-image .
```

### How to cache dependencies?

```yaml
cache:
  key: ${CI_COMMIT_REF_SLUG}
  paths:
    - node_modules/
    - .npm/
```

## Security

### How to enable 2FA?

1. Go to User Settings > Account
2. Click "Enable Two-Factor Authentication"
3. Scan QR code with authenticator app
4. Save backup codes

### How to scan for vulnerabilities?

Enable security dashboards in project settings, or use templates:
```yaml
include:
  - template: Security/SAST.gitlab-ci.yml
```

## Performance

### GitLab is slow?

**Solutions**:
1. Increase server resources
2. Enable Prometheus monitoring
3. Optimize PostgreSQL settings
4. Use separate runners for CI

### How to check GitLab health?

```bash
sudo gitlab-rake gitlab:env:info
sudo gitlab-ctl status
```

## Troubleshooting

### 502 Bad Gateway error?

**Solution**:
1. Check unicorn/worker processes
2. Increase worker memory
3. Review logs: `sudo gitlab-ctl tail`

### Git operations failing?

**Solution**:
1. Check SSH keys are properly configured
2. Verify HTTPS certificates
3. Check repository permissions
