# Vue Installationsanleitung

Vue ist ein progressives JavaScript-Framework zum Erstellen von Benutzeroberflächen.

## Version

| Version | Erscheinungsdatum | Status |
|---------|-------------------|--------|
| Vue 3 | September 2020 | Empfohlen |
| Vue 2 | Oktober 2016 | Veraltet |

**Empfehlung:** Verwenden Sie Vue 3 für alle neuen Projekte.

## Voraussetzungen

| Anforderung | Vue 2 | Vue 3 |
|-------------|--------|-------|
| Node.js | 8.9+ | 16+ (18+ empfohlen) |
| npm | 5.6+ | 8+ |

## Installationsmethoden

### Methode 1: npm (Empfohlen)

```bash
Neues Vue-Projekt erstellen
npm create vue@latest
```

### Methode 2: Vite (Empfohlen für Vue 3)

```bash
npm create vite@latest my-vue-app -- --template vue
cd my-vue-app
npm install
npm run dev
```

### Methode 3: CDN

```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
```

## Projektkonfiguration

```bash
npm install vue-router@4
npm install pinia
```

## Entwicklungsserver

```bash
# Server starten
npm run dev

# Für Produktion bauen
npm run build
```

## Problembehebung

### npm install fehlschlägt

```bash
npm cache clean --force
rm -rf node_modules
npm install
```

## Best Practices

1. **Vue 3 verwenden** - Für neue Projekte
2. **Vite verwenden** - Schnellere Entwicklung
3. **Composition API verwenden** - Bessere TypeScript-Unterstützung
