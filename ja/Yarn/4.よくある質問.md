# Yarn よくある質問

## 基本的な質問

### Yarnとは？

YarnはJavaScriptパッケージマネージャーで、依存関係のインストール、更新、削除を支援します。

### Yarnとnpmの違いは？

- 速度：Yarnはより速い
- 決定論性：yarn.lockが一貫性を保証
- オフラインサポート：強力なキャッシュ機能

### Yarnは可能ですか？

はい、MITライセンスで無料です。

---

## インストール

### インストールに失敗した場合は？

1. Node.jsバージョンを確認
2. Corepackを使用

## 使用

### 依存関係を追加するには？

```bash
yarn add lodash
yarn add -D typescript
```

## ワークスペース

### ワークスペースとは？

単一のリポジトリで複数のパッケージを管理できます。
