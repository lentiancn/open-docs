# GitLab 개요

> 소프트웨어 개발 및 운영을 위한 완전한 DevOps 플랫폼

---

## 개요

GitLab은 위키, 이슈 추적 및 CI/CD 파이프라인 기능을 갖춘 웹 기반 Git 저장소 관리 도구입니다. 단일 인터페이스에서 팀이 애플리케이션을 계획, 개발, 테스트, 배포 및 모니터링할 수 있는 완전한 DevOps 플랫폼입니다.

---

## 주요 기능

- **Git 저장소 호스팅**: 브랜칭, 태깅 및 병합 요청을 갖춘 완전한 Git 저장소 관리
- **이슈 추적**: 레이블, 마일스톤 및 보드 뷰를 갖춘 종합적인 이슈 추적
- **CI/CD 파이프라인**: 기본 제공 연속 통합 및 배포 파이프라인
- **위키**: 문서화를 위한 기본 제공 위키
- **코드 리뷰**: 병합 요청 워크플로우를 갖춘 기본 제공 코드 리뷰
- **보안**: 엔터프라이즈급 보안 스캐닝 및 규정 준수 기능
- **컨테이너 레지스트리**: 기본 제공 Docker 컨테이너 레지스트리
- **GitOps**: Kubernetes 배포 통합

---

## 아키텍처

| 구성요소 | 설명 |
|---------|------|
| GitLab Core | 저장소 관리 및 이슈 추적 |
| GitLab CI/CD | 연속 통합 및 배포 |
| GitLab Registry | Docker 컨테이너 레지스트리 |
| GitLab Pages | 정적 사이트 호스팅 |
| GitLab Mattermost | 기본 제공 채팅 통합 |

---

## 사용 사례

- 소스 코드 관리
- 연속 통합/연속 배포
- 프로젝트 관리 및 이슈 추적
- 위키로 문서화
- 보안 스캐닝 및 규정 준수
- 컨테이너 관리
