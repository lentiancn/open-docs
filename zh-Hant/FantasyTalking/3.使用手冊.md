# FantasyTalking 使用手冊

詳細使用方法和示例。

## 基本用法

### 生成影片

```bash
python inference.py \
  --source_image face.jpg \
  --audio speech.wav \
  --output output.mp4
```

### 參數

| 參數 | 說明 |
|------|------|
| --source_image | 輸入人物圖片 |
| --audio | 輸入音訊檔案 |
| --output | 輸出影片路徑 |
| --face_enhancer | 人臉增強器 |
| --fps | 輸出幀率 |

## Python API

```python
from fantasytalking import FantasyTalking

ft = FantasyTalking()
ft.generate(
    source_image="face.jpg",
    audio="speech.wav",
    output="output.mp4"
)
```

## 批次處理

```python
from fantasytalking import FantasyTalking

ft = FantasyTalking()

# 批次生成
ft.batch_generate(image_list, audio_list, output_dir)
```

## Web API

```bash
python api.py --port 8080
```

## 輸入要求

### 圖片

- 格式：JPG、PNG
- 解析度：512x512 或更高
- 人臉：正面、無遮擋、光線充足

### 音訊

- 格式：WAV、MP3
- 時長：1-60 秒

## 輸出設置

### 解析度

```bash
python inference.py --size 512
```

### 幀率

```bash
python inference.py --fps 30
```

## 最佳實踐

1. 使用清晰、正面的照片
2. 準備清晰的音訊
3. 使用人臉增強器提升質量
