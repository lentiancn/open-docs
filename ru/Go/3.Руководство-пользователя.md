## Базовый синтаксис

### Hello World

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

## Функции

```go
func add(a, b int) int {
    return a + b
}

func divide(a, b int) (int, error) {
    if b == 0 {
        return 0, errors.New("деление на ноль")
    }
    return a / b, nil
}
```

## Параллелизм

### Goroutine

```go
func say(s string) {
    for i := 0; i < 3; i++ {
        fmt.Println(s)
    }
}

func main() {
    go say("Hello")
    say("World")
    time.Sleep(time.Second)
}
```

### Канал

```go
ch := make(chan string)
go func() { ch <- "message" }()
msg := <-ch
```

## Обработка ошибок

```go
result, err := divide(10, 0)
if err != nil {
    fmt.Println("Ошибка:", err)
}
```
