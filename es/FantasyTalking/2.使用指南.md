# Guía de Uso de FantasyTalking

Esta guía cubre cómo usar FantasyTalking para generar videos de talking head.

## Inicio Rápido

### Uso Básico

```bash
python inference.py \
  --source_image images/face.jpg \
  --audio audio/speech.wav \
  --output results/output.mp4
```

## Requisitos de Entrada

### Requisitos de Imagen

- Formato: JPG, PNG
- Resolución: 512x512 o superior
- Rostro: Frontal, rasgos claros
- Fondo: Fondo simple preferido

### Requisitos de Audio

- Formato: WAV, MP3
- Frecuencia de Muestreo: 16000 Hz o superior
- Duración: Sin límite
- Idioma: Soporte multilingüe

## Uso Avanzado

### Mejora de Rostro

```bash
python inference.py \
  --source_image images/face.jpg \
  --audio audio/speech.wav \
  --output results/output.mp4 \
  --face_enhancer
```

### Procesamiento por Lotes

```bash
python batch_inference.py \
  --image_dir images/ \
  --audio_dir audio/ \
  --output_dir results/
```

### FPS Personalizado

```bash
python inference.py \
  --source_image images/face.jpg \
  --audio audio/speech.wav \
  --output results/output.mp4 \
  --fps 30
```

## Configuración

### Archivo de Configuración

Editar `config.yaml`:

```yaml
inference:
  face_enhancer: true
  fps: 25
  output_resolution: 512
  
model:
  checkpoint: "checkpoints/fantasytalking.pth"
```

## Entrenamiento del Modelo

### Preparar Conjunto de Datos

```bash
python prepare_dataset.py \
  --video_dir videos/ \
  --output_dir dataset/
```

### Entrenamiento

```bash
python train.py \
  --config config/train.yaml \
  --epochs 100
```

## Mejores Prácticas

### 1. Usar Imágenes de Alta Calidad

- Usar fotos claras y frontales
- Buenas condiciones de iluminación
- Alta resolución preferida

### 2. Hacer Coincidir la Calidad del Audio

- Audio claro sin ruido de fondo
- Frecuencia de muestreo adecuada
- Idioma que coincida con la etnia del rostro

### 3. Post-Procesamiento

- Usar mejora de rostro para mejor calidad
- Ajustar FPS según sea necesario

## Solución de Problemas

### ¿Video Muy Oscuro?

Verificar iluminación en imagen de origen.

### ¿Audio No Sincronizado?

Asegurarse de que la frecuencia de muestreo del audio sea correcta.

### ¿Mala Calidad?

Probar la opción de mejora de rostro.
