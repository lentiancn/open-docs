# Manual de Usuario de Element UI

> Uso de componentes

---

## Botones

```html
<el-button>Predeterminado</el-button>
<el-button type="primary">Primario</el-button>
```

## Formulario

```html
<el-form :model="form">
  <el-form-item>
    <el-input v-model="form.name"></el-input>
  </el-form-item>
</el-form>
```

## Tabla

```html
<el-table :data="tableData">
  <el-table-column prop="name" label="Nombre"></el-table-column>
</el-table>
```

## Diálogo

```html
<el-dialog :visible.sync="dialogVisible" title="Título">
  <span>Contenido</span>
</el-dialog>
```
