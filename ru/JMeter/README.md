# Apache JMeter

Apache JMeter — инструмент нагрузочного тестирования с открытым исходным кодом от Apache Software Foundation. Это приложение на 100% чистом Java, предназначенное для нагрузочного тестирования и измерения производительности.

## Обзор

JMeter изначально был разработан для тестирования веб-приложений, но с тех пор расширился для тестирования других функций. Он может использоваться для тестирования производительности статических и динамических ресурсов, имитации высоких нагрузок на серверы, сети или объекты для проверки их прочности или анализа общей производительности при различных типах нагрузки.

## Основные характеристики

- **Поддержка множества протоколов**: HTTP, HTTPS, SOAP, REST, FTP, JDBC, LDAP, JMS, SMTP, POP3, IMAP, TCP, Java-объекты
- **Полнофункциональная среда тестирования IDE**: GUI для записи, создания и отладки планов тестирования
- **Режим командной строки**: Headless-режим для нагрузочного тестирования на любой ОС, совместимой с Java
- **Динамические отчёты**: Встроенная генерация динамических HTML-отчётов
- **Извлечение данных**: Извлечение данных из форматов HTML, JSON, XML
- **100% Java**: Полная переносимость
- **Многопоточная среда**: Параллельная выборка и распределённое тестирование
- **Высокая расширяемость**: Поддержка плагинов и расширений сценариев

## Быстрый старт

### Основное использование

1. Запустите JMeter:
   ```bash
   jmeter
   ```

2. Создайте план тестирования, добавьте группы потоков и HTTP-запросы

3. Запустите тест и просмотрите результаты

### Режим командной строки

```bash
jmeter -n -t test_plan.jmx -l results.jtl -e -o report
```
