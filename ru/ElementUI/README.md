# Element UI

> Библиотека компонентов для настольных компьютеров на Vue 2

---

## Описание проекта

Element UI — это библиотека компонентов для настольных компьютеров на Vue 2, разработанная и поддерживаемая командой Ele.me. Она предоставляет богатый набор базовых компонентов, включая кнопки, формы, таблицы, диалоги, навигационные меню и многое другое.

**Примечание**: Для проектов Vue 3 используйте [Element Plus](https://github.com/element-plus/element-plus)

---

## Основные функции

- На базе Vue 2
- Богатая библиотека компонентов для ПК
- Поддержка частичного импорта
- Поддержка интернационализации
- Поддержка тем

## Быстрый старт

### Установка

```bash
npm install element-ui --save
```

### Импорт

```javascript
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```
