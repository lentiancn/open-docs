# MongoDB Часто задаваемые вопросы

## Основные понятия

### Что такое MongoDB?

MongoDB — это документоориентированная база данных, созданная для помощи разработчикам быстрее создавать современные приложения.

### MongoDB — это реляционная база данных?

Нет. MongoDB — это документоориентированная база данных NoSQL.

### Что такое документ в MongoDB?

Документ — это основная единица данных в MongoDB, состоящая из пар поле-значение.

### В чем разница между MongoDB и MySQL?

| Характеристика | MongoDB | MySQL |
|----------------|---------|-------|
| Модель данных | Документы | Реляционная |
| Схема | Гибкая | Фиксированная |
| Масштабирование | Горизонтальное | Вертикальное |

## Проблемы установки

### Сколько памяти нужно MongoDB?

MongoDB использует доступную память как кэш. Рекомендуется минимум 4 ГБ для разработки.

### MongoDB поддерживает 32-битные системы?

Нет. Требуется 64-битная ОС.

## Вопросы использования

### Как сделать резервную копию MongoDB?

```bash
mongodump --db=mydatabase --out=/backup
```

### Как MongoDB обеспечивает высокую доступность?

С помощью Replica Sets.

### Что такое шардинг?

Шардинг — это методика MongoDB для горизонтального масштабирования.

### MongoDB поддерживает транзакции?

Да. MongoDB 4.0+ поддерживает мультидокументные транзакции.

### Как оптимизировать производительность запросов?

1. Создавайте соответствующие индексы
2. Используйте проекции
3. Избегайте полных сканирований коллекций

### Как защитить данные в MongoDB?

1. Включите аутентификацию
2. Используйте контроль доступа на основе ролей (RBAC)
3. Включите TLS/SSL
4. Регулярно делайте резервные копии

### В чем разница между Community и Enterprise?

Enterprise Edition включает:
- Аутентификацию LDAP
- Аудит
- Шифруемое хранилище
- Техническую поддержку

## Разработка

### Какие языки программирования поддерживает MongoDB?

MongoDB предоставляет драйверы для основных языков:
- JavaScript/Node.js
- Python
- Java
- C#
- Go
- Ruby
- PHP

### Как использовать MongoDB в приложениях?

1. Установите драйвер для вашего языка
2. Установите соединение
3. Выполните операции CRUD

Пример на Node.js:

```javascript
const { MongoClient } = require('mongodb');
const client = new MongoClient('mongodb://localhost:27017');

async function main() {
  await client.connect();
  const database = client.db('myDB');
  const collection = database.collection('users');
  
  await collection.insertOne({ name: 'Иван', age: 30 });
  const user = await collection.findOne({ name: 'Иван' });
  
  await client.close();
}
```

### Что такое MongoDB Atlas?

MongoDB Atlas — это облачная служба баз данных от MongoDB.
