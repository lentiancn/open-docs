## 1. Install Docker

### Windows / macOS

1. Download Docker Desktop: https://www.docker.com/products/docker-desktop
2. Install and launch
3. Verify:
```bash
docker --version
```

### Linux (Ubuntu)

```bash
# One-line install script
curl -fsSL https://get.docker.com | sh

# Start Docker
sudo systemctl start docker

# Enable on boot
sudo systemctl enable docker
```

## 2. First Container

### 2.1 Run Hello World

```bash
docker run hello-world
```

### 2.2 Run Nginx

```bash
# Run Nginx in background
docker run -d -p 8080:80 nginx

# Visit http://localhost:8080
```

### 2.3 Run Ubuntu Interactive

```bash
docker run -it ubuntu /bin/bash
```

## 3. Common Commands

### 3.1 Image Commands

```bash
# List images
docker images

# Pull image
docker pull nginx

# Remove image
docker rmi nginx
```

### 3.2 Container Commands

```bash
# List running containers
docker ps

# List all containers
docker ps -a

# Stop container
docker stop container_id

# Remove container
docker rm container_id
```

### 3.3 Logs and Debugging

```bash
# View logs
docker logs container_id

# Enter container
docker exec -it container_id /bin/bash

# View resource usage
docker stats
```

## 4. First Dockerfile

Create `Dockerfile`:

```dockerfile
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
EXPOSE 3000
CMD ["npm", "start"]
```

Build image:

```bash
docker build -t my-app .
```

Run:

```bash
docker run -p 3000:3000 my-app
```

---

## 5. Docker Compose

Create `docker-compose.yml`:

```yaml
version: '3.8'
services:
  web:
    build: .
    ports:
      - "3000:3000"
    environment:
      - NODE_ENV=production
  redis:
    image: redis:alpine
    ports:
      - "6379:6379"
```

Start:

```bash
docker-compose up -d
```

Stop:

```bash
docker-compose down
```
