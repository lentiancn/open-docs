# Element Plus 簡介

Vue 3 桌面端 UI 元件庫。

## 什麼是 Element Plus？

Element Plus 是由餓了麼團隊開發的 Vue 3 元件庫，為現代 Web 應用提供了一套完整的桌面端 UI 元件。作為 Element UI 的 Vue 3 版本，Element Plus 繼承了前代產品的優秀特性，並針對 Vue 3 進行了全面優化。

### 核心特性

- **Vue 3 驅動**：完全基於 Vue 3 Composition API 構建
- **TypeScript 支持**：原生 TypeScript 類型定義
- **元件豐富**：70+ 高質量元件
- **按需加載**：支持 tree shaking，優化打包體積
- **主題定製**：支持 CSS 變量自定義主題
- **國際化**：內置 20+ 語言包
- **無障礙支持**：完善的 ARIA 支持
- **響應式設計**：適配各種螢幕尺寸

**基礎元件：**
- Button 按鈕

**表單元件：**

**數據展示：**

- Progress 進度條
- Tree 樹形控件
- Pagination 分頁
- Badge 標記
- Avatar 頭像

**反饋元件：**
- Message 消息提示

- Dialog 對話框

- MessageBox 消息框
- Loading 加載

**導航元件：**
