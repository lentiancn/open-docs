# Guía de Uso de MultiTalk

Esta guía cubre cómo usar MultiTalk para generar contenido de audio multiorador.

## Inicio Rápido

### Uso Básico

```bash
python inference.py \
  --text "Hola a todos, bienvenidos a MultiTalk." \
  --speakers speaker1,speaker2 \
  --output result.wav
```

## Requisitos de Entrada

### Requisitos de Texto

- Idioma: Soporte multilingüe
- Formato: Texto plano
- Longitud: Sin límite estricto

### Configuración de Oradores

Definir oradores en `speakers.yaml`:

```yaml
speakers:
  speaker1:
    voice_id: "voice_001"
    language: "es"
  speaker2:
    voice_id: "voice_002"
    language: "es"
```

## Uso Avanzado

### IDs de Voz Personalizados

```bash
python inference.py \
  --text "Esta es una demostración de voz personalizada." \
  --voice_ids voice001,voice002 \
  --output output.wav
```

### Ajustar Velocidad

```bash
python inference.py \
  --text "Ejemplo de habla lenta." \
  --speakers speaker1 \
  --speed 0.8 \
  --output slow.wav
```

### Procesamiento por Lotes

```bash
python batch_inference.py \
  --text_file texts.txt \
  --output_dir output/
```

## Configuración

### Archivo de Configuración

Editar `config.yaml`:

```yaml
inference:
  sample_rate: 22050
  speed: 1.0
  
model:
  checkpoint: "checkpoints/multitalk.pth"
```

## Mejores Prácticas

### 1. Usar IDs de Voz Apropiados

Elige IDs de voz que coincidan con las características del orador.

### 2. Coincidencia de Idioma

Asegúrate de que el idioma del texto coincida con el idioma de la voz.

### 3. Optimizar Velocidad

Ajusta la velocidad para una salida de sonido natural.

## Solución de Problemas

### ¿Mala Calidad de Audio?

Prueba diferentes IDs de voz.

### ¿Problemas de Velocidad?

Ajusta el parámetro de velocidad gradualmente.
