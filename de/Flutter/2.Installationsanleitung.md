# Flutter Installationsanleitung

## Anforderungen

Bevor Sie Flutter installieren, stellen Sie sicher, dass Ihre Entwicklungsumgebung die folgenden Anforderungen erfüllt. Bei Betriebssystemen unterstützt Flutter Windows, macOS und Linux. Windows erfordert Windows 10 oder höher, macOS erfordert macOS 10.14 oder höher, und Linux erfordert eine Desktop-Umgebung, die GNOME oder KDE unterstützt.

Bei Festplattenspeicher nimmt das Flutter SDK selbst etwa 1 GB ein, aber angesichts von Entwicklungsprojekten und Build-Cache wird empfohlen, mindestens 10 GB verfügbaren Speicherplatz einzuplanen. Bei Arbeitsspeicher bieten 8 GB oder mehr ein flüssigeres Entwicklungserlebnis.

Für mobile Entwicklung erfordert iOS-Entwicklung macOS und Xcode, Android-Entwicklung erfordert die Installation von Android Studio und Android SDK. Web- und Desktop-Entwicklung haben keine besonderen Anforderungen, Sie müssen nur das Flutter SDK installieren.

## Installationsschritte

### Windows-Installation

Es gibt zwei Hauptmethoden zur Installation von Flutter unter Windows. Die erste Methode besteht darin, das vorkompilierte Archiv von der Flutter-Website herunterzuladen, es an einem geeigneten Ort zu entpacken und dann das flutter/bin-Verzeichnis zur System-PATH-Umgebungsvariable hinzuzufügen.

Laden Sie die neueste Stable-Version von der Flutter-Website flutter.dev herunter. Entpacken Sie es nach C:\src\flutter oder an einem anderen geeigneten Ort. Öffnen Sie Systemeigenschaften → Erweitert → Umgebungsvariablen, und fügen Sie in den Benutzervariablen unter Path den Pfad zum Flutter bin-Verzeichnis hinzu.

Öffnen Sie nach dem Abschluss die Eingabeaufforderung und führen Sie den Befehl flutter doctor aus, um zu überprüfen, ob die Installation erfolgreich war. Flutter doctor zeigt den Status des Flutter SDK an und gibt Konfigurationsvorschläge.

### macOS-Installation

Unter macOS wird die Git-basierte Installation empfohlen. Stellen Sie zunächst sicher, dass Xcode installiert ist, öffnen Sie dann das Terminal und führen Sie den folgenden Befehl aus, um Flutter zu klonen:

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

Flutter zum PATH hinzufügen:

```bash
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.bash_profile
```

Für zsh-Benutzer:

```bash
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.zshrc
```

Führen Sie dann source ~/.bash_profile oder source ~/.zshrc aus, um die Umgebungsvariablen zu aktualisieren.

### Linux-Installation

Die Installation unter Linux ist ähnlich wie unter macOS. Zuerst die Abhängigkeiten installieren:

```bash
sudo apt-get update
sudo apt-get install curl git unzip xz-utils zip libglu1-mesa clang cmake ninja-build pkg-config libgtk-3-dev
```

Dann das Flutter-Repository klonen und PATH konfigurieren, wie bei macOS.

## Installation überprüfen

Führen Sie nach der Installation die folgenden Befehle zur Überprüfung aus:

```bash
flutter --version
flutter doctor
```

Flutter doctor zeigt den Installationsstatus und fehlende Abhängigkeiten an. Vervollständigen Sie die erforderlichen Konfigurationen gemäß den Anweisungen, z. B. die Installation von Android Studio und die Konfiguration des Android SDK.

## IDE-Konfiguration

### VS Code

VS Code ist einer der empfohlenen Editoren für Flutter. Nach der Installation von VS Code öffnen Sie den Erweiterungsmarkt und suchen nach Flutter- und Dart-Erweiterungen und installieren Sie diese. Nach der Installation starten Sie VS Code neu, um Flutter-bezogene Funktionen zu nutzen.

### Android Studio

Nach der Installation von Android Studio öffnen Sie die Seite Plugins, suchen nach dem Flutter-Plugin und installieren Sie es. Konfigurieren Sie nach der Installation den Flutter SDK-Pfad: Öffnen Sie Preferences → Language & Frameworks → Flutter und legen Sie den Flutter SDK-Pfad fest.

## Schnellstart

Erstellen Sie Ihr erstes Flutter-Projekt:

```bash
flutter create my_first_app
cd my_first_app
flutter run
```

Oder verwenden Sie in VS Code die Befehlspalette (Strg+Umschalt+P) und geben Sie Flutter: New Project ein, um ein neues Projekt zu erstellen.

Führen Sie flutter devices aus, um verbundene Geräte anzuzeigen, und flutter run -d Geräte-ID, um auf einem bestimmten Gerät auszuführen.
