# OpenClaw 설치 가이드

OpenClaw는 macOS, Linux 또는 Windows를 사용하는지 여부에 따라 다양한 설치 방법을 지원합니다. 이 가이드에서는 각 설치 옵션과 전제 조건을 안내합니다.

---

## 시스템 요구사항

설치하기 전에 시스템이 다음 요구사항을 충족하는지 확인하세요:

- **Node.js**: 22.0.0 이상
- **운영 체제**:
  - macOS 10.15 (Catalina) 이상
  - Linux (Ubuntu 20.04+, Debian 11+, Fedora 36+ 등)
  - Windows 10/11 (WSL2 강력 권장)
- **네트워크**: npm 레지스트리 및 OpenClaw 서비스에 대한 액세스
- **저장공간**: 최소 500MB의 여유 공간

Node.js 버전을 확인하세요:

```bash
node --version
```

버전이 22.0.0 미만이면 먼저 Node.js를 업그레이드하세요. nvm(Node Version Manager)을 사용하여 여러 Node.js 버전을 관리할 것을 권장합니다.

## 설치 방법

### 방법 1: 공식 설치 스크립트 (권장)

가장 간단하고 빠른 방법입니다. 공식 설치 스크립트가 운영 체제를 자동으로 감지하고 모든 구성을 완료합니다.

**macOS / Linux:**

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

**Windows (PowerShell):**

```powershell
iwr -useb https://openclaw.ai/install.ps1 | iex
```

### 방법 2: npm 전역 설치

이미 Node.js와 npm이 설치되어 있는 경우:

```bash
npm install -g openclaw@latest
```

특정 버전을 지정할 수도 있습니다:

```bash
npm install -g openclaw@v2026.2.28
```

사용 가능한 버전 태그:
- `latest`: 최신 안정 버전
- `beta`: 베타 버전
- `dev`: 개발 버전

### 방법 3: pnpm 설치

OpenClaw는 pnpm 사용을 권장합니다:

```bash
pnpm add -g openclaw@latest
```

### 방법 4: Bun 설치

Bun을 JavaScript 런타임으로 사용하는 경우:

```bash
bun add -g openclaw@latest
```

### 방법 5: 소스에서 빌드

```bash
# 저장소 복제
git clone https://github.com/openclaw/openclaw.git
cd openclaw

# 종속성 설치
pnpm install

# 프로젝트 빌드
pnpm ui:build
pnpm build

# 초기화 및 서비스 설치
pnpm openclaw onboard --install-daemon

# 개발 모드 (자동 재로드)
pnpm gateway:watch
```

## 설치 후 구성

### 온보딩 마법사

설치 후 마법사를 실행하세요:

```bash
openclaw onboard --install-daemon
```

### 설치 확인

```bash
# 버전 확인
openclaw --version

# Gateway 상태 확인
openclaw gateway status

# 제어 UI 열기
openclaw dashboard
```

## 일반적인 설치 문제

### 문제 1: npm 권한 오류

nvm을 사용하세요:

```bash
# nvm 설치
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# nvm으로 Node.js 설치
nvm install 22
nvm use 22

# 이제 OpenClaw 설치
npm install -g openclaw@latest
```

### 문제 2: Gateway가 시작되지 않음

포트 사용 여부 확인:

```bash
lsof -i :18789
```

### 문제 3: 오래된 버전

업데이트하세요:

```bash
npm update -g openclaw@latest
```

## OpenClaw 업그레이드

```bash
npm update -g openclaw@latest
openclaw doctor
```

## OpenClaw 제거

```bash
# 서비스 중지
openclaw gateway stop

# 전역 패키지 제거
npm uninstall -g openclaw

# 구성 디렉토리 제거 (선택사항)
rm -rf ~/.openclaw
```
