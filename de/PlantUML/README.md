# PlantUML

Open-Source-Tool zum Zeichnen von UML-Diagrammen

## Übersicht

PlantUML ermöglicht das Erstellen verschiedener Diagramme aus einfachem Text.

## Schnellstart

```plantuml
@startuml
Alice -> Bob: Hallo
@enduml
```
