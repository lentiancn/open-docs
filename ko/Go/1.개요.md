# Go 개요

> Go는 Google이 개발한 컴파일형 프로그래밍 언어입니다

---

## 설명

Go는 2009년 Google의 Robert Griesemer, Rob Pike, Ken Thompson이 만들었습니다.シンプルさ、효율성、강력한 동시성 지원으로 유명합니다.

---

## 특징

- **정적 타이핑**: 컴파일 시 타입 안전성
- **가비지 컬렉션**: 자동 메모리 관리
- **네이티브 동시성**: goroutine과 channel로 가벼운 동시성
- **간단하고 효율적**: 깔끔한 문법, 빠른 컴파일
- **풍부한 표준 라이브러리**

---

## 사용 사례

- 서버 프로그래밍
- 클라우드 네이티브 애플리케이션 (Kubernetes, Docker)
- 네트워크 프로그래밍
-命令行 도구
- 분산 시스템
