# Kotlin 설치 가이드

Kotlin은 JetBrains가 개발한 최신 프로그래밍 언어입니다. 이 가이드에서는 다양한 환경에서의 설치 방법을 설명합니다.

## 버전 정보

| 버전 | 상태 | 설명 |
|------|------|------|
| Kotlin 2.x | ✅ 현재 | 최신 버전 |
| Kotlin 1.9.x | ✅ 지원 | 안정 버전 |

## 시스템 요구사항

### 하드웨어 요구사항

| 요구사항 | 구성 |
|----------|------|
| CPU | 1코어 이상 |
| 메모리 | 2GB 이상 |
| 디스크 | 500MB 이상 |

### 소프트웨어 요구사항

| 소프트웨어 | 버전 |
|-----------|------|
| Java | JDK 8 이상 |

## 설치 방법

### 방법 1: IntelliJ IDEA (권장)

1. IntelliJ IDEA 다운로드: https://www.jetbrains.com/idea/download/
2. Ultimate 또는 Community 버전 설치
3. Kotlin은 이미 포함되어 있습니다

### 방법 2:命令行

#### Windows

```powershell
# Chocolatey 사용
choco install kotlin

# 수동 다운로드
# https://github.com/JetBrains/kotlin/releases 에서 다운로드
```

#### macOS

```bash
# Homebrew 사용
brew install kotlin

# 수동 설치
curl -s https://get.sdkman.io | bash
sdk install kotlin
```

#### Linux

```bash
# Ubuntu/Debian
sudo apt install kotlin

# 또는 SDKMAN 사용
curl -s https://get.sdkman.io | bash
sdk install kotlin
```

### 방법 3: SDKMAN

```bash
# SDKMAN 설치
curl -s "https://get.sdkman.io" | bash

# Kotlin 설치
sdk install kotlin

# 확인
kotlin -version
```

### 방법 4: Maven

```xml
<dependencies>
    <dependency>
        <groupId>org.jetbrains.kotlin</groupId>
        <artifactId>kotlin-stdlib</artifactId>
        <version>2.0.0</version>
    </dependency>
</dependencies>
```

### 방법 5: Gradle

```groovy
plugins {
    id 'org.jetbrains.kotlin.jvm' version '2.0.0'
}

repositories {
    mavenCentral()
}

dependencies {
    implementation "org.jetbrains.kotlin:kotlin-stdlib"
}
```

### 방법 6: Docker

```dockerfile
# Kotlin 컴파일러 이미지 사용
FROM gradle:8.5-jdk-21 AS builder
WORKDIR /app
COPY . .
RUN gradle build

# 실행 이미지
FROM openjdk:21
COPY --from=builder /app/build/libs/*.jar /app.jar
ENTRYPOINT ["java", "-jar", "/app.jar"]
```

## 환경 설정

### Java 설정

```bash
# Java 확인
java -version

# JAVA_HOME 설정
export JAVA_HOME=/path/to/java
```

### 컴파일 및 실행

```bash
# 컴파일
kotlinc hello.kt -include-runtime -d hello.jar

# 실행
java -jar hello.jar

# kotlin 스크립트 사용
kotlin hello.kts
```

## IDE 설정

### IntelliJ IDEA

1. Kotlin 플러그인 설치 (일반적으로 사전 설치됨)
2. Kotlin 프로젝트 생성
3. Kotlin SDK 구성

### VS Code

1. Kotlin Language Server 확장 프로그램 설치
2. Kotlin 컴파일러 설치

### Eclipse

1. Kotlin 플러그인 설치 (Kotlin IDE for Eclipse)
2. Kotlin 프로젝트 생성

## 자주 묻는 질문

### 컴파일 오류?

Kotlin과 Java 버전 호환성을 확인하세요.

### 메모리 부족?

JVM 힙 메모리 증가:

```bash
kotlinc -J-Xmx2g
```
