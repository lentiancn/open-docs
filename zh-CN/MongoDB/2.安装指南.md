# MongoDB 安装指南

## 环境要求

在安装 MongoDB 之前，请确保系统满足以下要求：64 位处理器、至少 2GB 可用内存、足够的磁盘空间（建议 10GB 以上）。不同操作系统有特定的硬件和软件依赖要求，请查阅官方文档获取详细信息。

## 在 Windows 上安装

### 使用 MSI 安装包

从 MongoDB 官方下载中心获取 Windows 版本的 MSI 安装包。运行安装程序，选择 "Complete" 完整安装类型。在安装过程中，可以选择将 MongoDB 配置为 Windows 服务，这样系统启动时会自动运行数据库。

安装完成后，需要创建数据目录。默认路径为 `C:\data\db`，可以通过命令行创建：

```cmd
md C:\data\db
```

启动 MongoDB 服务：

```cmd
"C:\Program Files\MongoDB\Server\版本号\bin\mongod.exe"
```

### 验证安装

打开另一个命令行窗口，连接 MongoDB：

```cmd
"C:\Program Files\MongoDB\Server\版本号\bin\mongosh.exe"
```

如果看到 MongoDB shell 提示符，说明安装成功。

## 在 macOS 上安装

### 使用 Homebrew

macOS 用户推荐使用 Homebrew 包管理器安装 MongoDB：

```bash
# 更新 Homebrew
brew update

# 安装 MongoDB 社区版
brew tap mongodb/brew
brew install mongodb-community
```

### 启动 MongoDB

作为后台服务启动：

```bash
brew services start mongodb-community
```

或者直接运行：

```bash
mongod --config /usr/local/etc/mongod.conf
```

## 在 Linux 上安装

### Debian 和 Ubuntu

```bash
# 导入 MongoDB 公钥
wget -qO - https://www.mongodb.org/static/pgp/server-7.0.asc | sudo apt-key add -

# 添加 MongoDB 仓库
echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu jammy/mongodb-org/7.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-7.0.list

# 更新并安装
sudo apt update
sudo apt install -y mongodb-org
```

### CentOS 和 RHEL

```bash
# 创建 yum 仓库文件
sudo tee /etc/yum.repos.d/mongodb-org-7.0.repo << EOF
[mongodb-org-7.0]
name=MongoDB Repository
baseurl=https://repo.mongodb.org/yum/redhat/\$releasever/mongodb-org/7.0/x86_64/
gpgcheck=1
enabled=1
gpgkey=https://www.mongodb.org/static/pgp/server-7.0.asc
EOF

# 安装
sudo yum install -y mongodb-org
```

## 配置 MongoDB

### 配置文件

MongoDB 的默认配置文件位于 `/etc/mongod.conf`（Linux/macOS）或安装目录下的 `bin\mongod.cfg`（Windows）。

常用配置项包括：

```yaml
storage:
  dbPath: /var/lib/mongo
  journal:
    enabled: true

systemLog:
  destination: file
  path: /var/log/mongodb/mongod.log

net:
  port: 27017
  bindIp: 127.0.0.1
```

### 启动与停止

```bash
# 启动
sudo systemctl start mongod

# 停止
sudo systemctl stop mongod

# 设置开机自启
sudo systemctl enable mongod
```

## 安装 mongosh

mongosh 是 MongoDB 的交互式 shell，用于连接和操作数据库。从 MongoDB 6.0 开始，mongosh 已取代传统的 mongo shell。

下载对应平台的 mongosh 包，解压后将可执行文件移动到系统 PATH 中即可使用。

## 远程连接配置

如果需要从其他机器连接 MongoDB，需要修改配置文件中的 `bindIp` 选项：

```yaml
net:
  port: 27017
  bindIp: 0.0.0.0  # 允许所有 IP 访问，生产环境请谨慎使用
```

建议同时配置防火墙规则，限制可访问的 IP 范围。

## 验证安装

使用 mongosh 连接数据库并验证：

```bash
mongosh
> db.version()
> show dbs
```

如果能看到数据库版本和列表，说明安装成功。
