# OpenClaw 常見問題

## 基礎問題

### 1. 什麼是 OpenClaw？

OpenClaw 是連接聊天應用到 AI 編程助手的自托管閘道軟體。

### 2. OpenClaw 免費嗎？

軟體本身免費，但需要：
- 運行 Gateway 的電腦/伺服器
- AI 服務 API 密鑰

### 3. OpenClaw 安全嗎？

是的：
- 數據本地存儲
- 白名單支持
- 開源（MIT 許可證）

### 4. 支持哪些平台？

- 系統：macOS、Linux、Windows
- 聊天：WhatsApp、Telegram、Discord、iMessage
- 行動：iOS、Android

## 安裝問題

### 5. 安裝失敗？

- 檢查 Node 版本：`node --version`（需要 22+）
- 權限問題：嘗試 sudo
- 網路問題：檢查網路連接

### 6. 端口被佔用？

```bash
lsof -i :18789
openclaw gateway --port 18889
```

## 配置問題

### 7. 如何配置 API 密鑰？

```bash
openclaw config set providers.anthropic.key <your-key>
```

### 8. 配置文件位置？

`~/.openclaw/openclaw.json`

## 使用問題

### 9. Gateway 無法啟動？

```bash
openclaw status
openclaw logs
```

### 10. 如何查看聊天記錄？

```bash
openclaw sessions list
```
