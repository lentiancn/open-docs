# PlantUML Guía de Instalación

## Prerrequisitos

- **Java Runtime**: Java 8 o superior requerido
- **Memoria**: Mínimo 512MB RAM
- **Espacio en Disco**: 100MB para instalación

---

## Instalación Independiente

### Windows

1. Descargar el archivo JAR de PlantUML
2. Guardar `plantuml.jar` en una ubicación conveniente
3. (Opcional) Agregar la ubicación al PATH

### macOS

```bash
brew install plantuml
```

### Linux

```bash
sudo apt-get install plantuml
```

---

## Integración con IDE

### VS Code

1. Abrir VS Code
2. Ir a Extensiones
3. Buscar "PlantUML"
4. Instalar
5. Reiniciar VS Code

---

## Verificar Instalación

```bash
java -jar plantuml.jar -version
```
