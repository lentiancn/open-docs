# Docker

全球最受歡迎的容器化平台。

## 簡介

Docker 是開源的容器化平台，用於開發、部署和運行應用程序。通過容器化技術，Docker 可以將應用程序及其依賴打包成輕量級，可移植的容器。

### 主要特性

- 輕量級 - 容器共享內核，啟動快速
- 可移植 - 一次構建，到處運行
- 隔離性 - 容器間相互隔離
- 高效 - 比虛擬機更節省資源

## 快速開始

### 安裝

**Windows/macOS**：
下載 Docker Desktop：https://www.docker.com/products/docker-desktop

**Linux (Ubuntu)**：
```
curl -fsSL https://get.docker.com | sh
```

### 運行第一個容器

```
docker run hello-world
```
