# Yarn インストールガイド

## Yarn v4+ (モダンバージョン) インストール

### 推奨方法: Corepack を使用

Corepack は、プロジェクトごとにパッケージマネージャーのバージョンを設定できる中間ツールです。

```bash
# 1. Corepack をインストール
npm install -g corepack

# 2. 新しいプロジェクトを初期化
yarn init -2
```

### Yarn の更新

Yarn を最新バージョンに更新したい場合は、以下のコマンドを実行するだけです:

```bash
yarn set version stable
yarn install
```

Yarn はプロジェクトを最新の安定版バイナリを使用するように設定します。

### Canary ビルドの使用

Yarn は頻繁にリリース候補ビルドも提供しています。安定チャンネルでまだリリースされていない機能が必要な場合は、以下を使用してください:

```bash
yarn set version canary
```

これらのビルドは非常に安定しており、通常のチャンネルとの唯一の違いは、新しい破壊的変更を実装する際にメジャーバージョン間でより段階的な移行を行うことです。

### master から最新ビルドをインストール

まだリリース候補でもリリースされていない、あるいはマージされていない非常に新しい Yarn のバージョンをテストしたい場合があります。以下のコマンドは、リポジトリから直接 Yarn をクローン、ビルド、インストールします:

```bash
yarn set version from sources
```

`--branch` フラグを受け入れるため、特定の PR をテストするために使用できます:

```bash
yarn set version from sources --branch 1211
```

> 警告: 安定チャンネルや canary チャンネルとは異なり、`yarn set version from sources` コマンドは Corepack を活用できず、Yarn バイナリを `.yarn/releases` フォルダ内に保存し、プロジェクトの `.yarnrc.yml` ファイルから参照する必要があります。

## Yarn v1 (Classic) インストール

### npm 経由でのインストール (推奨)

Yarn は [npm パッケージマネージャー](http://npmjs.org/) を通じてインストールすることを推奨します。npm はシステムに [Node.js](https://nodejs.org/) をインストールするとバンドルされています。

npm がインストールされている場合は、以下のコマンドを実行して Yarn をインストールおよびアップグレードできます:

```bash
npm install --global yarn
```

### インストールの確認

Yarn がインストールされているか確認するには、以下を実行します:

```bash
yarn --version
```

## バージョン選択の推奨事項

- **新規プロジェクト**: モダンな機能とより良いパフォーマンスを享受するために Yarn v4+ を使用することを推奨
- **既存プロジェクト**: Yarn v1 をすでに使用している場合は、引き続き使用するか、モダンバージョンへの移行を検討
- **エンタープライズ環境**: Yarn v1 はより安定した長期サポートを提供し、v4+ はより革新的な機能を提供
