# Guide d'installation

Ce guide présente les différentes méthodes pour installer Bootstrap et commencer rapidement à l'utiliser dans vos projets.

## Prérequis

Bootstrap 5 nécessite les environnements suivants :

- **HTML5** avec un navigateur moderne
- **CSS3** supporté
- **JavaScript** ES6+ (si vous utilisez les plugins JavaScript)

## Méthodes d'installation

### Méthode 1 : CDN (recommandé)

La méthode la plus simple est d'utiliser un CDN pour inclure directement les fichiers CSS et JavaScript de Bootstrap :

```html
<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Bootstrap</title>
    <!-- Inclusion du CSS Bootstrap -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>Bonjour, Bootstrap !</h1>

    <!-- Inclusion du JavaScript Bootstrap (avec Popper) -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### Méthode 2 : Installation npm

Si vous utilisez npm pour gérer les dépendances de votre projet :

```bash
# Installer Bootstrap
npm install bootstrap@5

# Installer Popper (pour les menus déroulants, info-bulles, etc.)
npm install @popperjs/core
```

Puis importez dans votre projet :

```javascript
// Importer le CSS de Bootstrap
import 'bootstrap/dist/css/bootstrap.min.css';

// Importer le JS de Bootstrap
import 'bootstrap/dist/js/bootstrap.bundle.min.js';
```

### Méthode 3 : Téléchargement des sources

Téléchargez les fichiers précompilés depuis le site officiel :

1. Rendez-vous sur https://getbootstrap.com/
2. Cliquez sur le bouton "Download"
3. Choisissez "Compiled CSS and JS" pour télécharger
4. Décompressez et copiez les dossiers css et js dans votre projet

Structure du répertoire :

```
votre-projet/
├── css/
│   └── bootstrap.min.css
├── js/
│   └── bootstrap.bundle.min.js
└── index.html
```

### Méthode 4 : Sources Sass

Si vous avez besoin de personnaliser les styles de Bootstrap, vous pouvez télécharger les sources Sass :

```bash
npm install bootstrap
```

Puis importez dans votre fichier Sass :

```scss
// Variables personnalisées (à définir avant l'import de Bootstrap)
$primary: #007bff;
$body-bg: #f8f9fa;

// Import de Bootstrap
@import "~bootstrap/scss/bootstrap";
```

### Éditeurs en ligne

- **Bootstrap Builder** : Construction visuelle de pages Bootstrap
- **Bootstrap Studio** : Outil professionnel de conception Bootstrap

## Démarrage rapide

### Créer votre première page

```html
<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Démarrage rapide Bootstrap</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <!-- Barre de navigation -->
    <nav class="navbar navbar-expand-lg navbar-dark bg-primary">
        <div class="container">
            <a class="navbar-brand" href="#">Mon site</a>
        </div>
    </nav>

    <!-- Contenu principal -->
    <main class="container mt-5">
        <div class="row">
            <div class="col-md-6">
                <h1>Bienvenue sur Bootstrap</h1>
                <p class="lead">Ceci est une page responsive construite avec Bootstrap 5.</p>
                <button class="btn btn-primary">En savoir plus</button>
            </div>
            <div class="col-md-6">
                <div class="card">
                    <div class="card-body">
                        <h5 class="card-title">Fonctionnalités</h5>
                        <ul>
                            <li>Mise en page responsive</li>
                            <li>Composants riches</li>
                            <li>Plugins JavaScript puissants</li>
                        </ul>
                    </div>
                </div>
            </div>
        </div>
    </main>

    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## Choix de la version

### Environnement de production

Utilisez une version stable :

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### Environnement de développement

Utilisez la dernière version de développement (avec les sources) :

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.js"></script>
```

### Miroir chinois

En Chine, vous pouvez utiliser un miroir local pour accélérer :

```html
<link href="https://cdn.bootcss.com/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.bootcss.com/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

## Désinstallation

Si vous souhaitez désinstaller (il suffit de supprimer les fichiers concernés) :

- Supprimez les références CSS et JS dans votre HTML
- Supprimez le dossier bootstrap dans node_modules (si vous utilisez npm)
- Supprimez les fichiers CSS et JS locaux téléchargés
