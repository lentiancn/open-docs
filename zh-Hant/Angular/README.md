# Angular

Angular 是由 Google 維護的現代 Web 應用框架。

## 簡介

Angular 是一個 TypeScript 開源 Web 應用框架，專為建構高效、複雜的單頁應用而設計。

## 主要特性

- **元件化架構**：基於元件的設計模式
- **TypeScript 支援**：原生 TypeScript 支援
- **雙向資料綁定**：自動同步模型和視圖
- **依賴注入**：強大的依賴注入系統
- **路由系統**：完整的路由解決方案
- **HTTP 客戶端**：完整的 HTTP 客戶端
- **表單處理**：反應式表單和範本驅動表單
- **測試支援**：完整的單元測試和端到端測試

## 快速開始

### 安裝 Angular CLI

```bash
npm install -g @angular/cli
```

### 建立新專案

```bash
ng new my-app
cd my-app
ng serve
```
