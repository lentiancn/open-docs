# 설치 가이드

이 가이드에서는 Angular를 설치하고 개발 환경을 설정하는 방법을 설명합니다. 이를 통해 신속하게 개발을 시작할 수 있습니다.

## 사전 요구사항

Angular를 설치하기 전에 시스템이 다음 요구사항을 충족하는지 확인하세요:

- **Node.js**: 버전 18.10.0 이상 권장
- **npm**: Node.js에 포함되어 있으며, 버전 6.x 이상
- **운영 체제**: Windows, macOS 또는 Linux

## Angular CLI 설치

Angular CLI는 Angular 프로젝트를 생성하고 관리하기 위한 공식 명령줄 도구입니다.

### 전역 설치

```bash
npm install -g @angular/cli
```

### 설치 확인

```bash
ng version
```

성공하면 Angular CLI 버전 정보가 표시됩니다.

### 특정 버전 설치

필요한 특정 버전이 있는 경우:

```bash
npm install -g @angular/cli@17
```

## 새 프로젝트 생성

### CLI 사용

```bash
ng new my-app
```

"my-app"이라는 새 프로젝트가 생성되고 의존성이 자동으로 설치됩니다.

### 옵션과 함께 생성

```bash
# 의존성 설치 없이 생성
ng new my-app --skip-install

# SCSS 스타일 사용
ng new my-app --style=scss

# 라우팅 활성화
ng new my-app --routing

# 독립형 컴포넌트 생성 (Angular 14+)
ng new my-app --standalone
```

## 개발 환경 설정

### 중국 미러 사용

네트워크 문제로 인해 중국 미러를 설정하는 것이 권장됩니다:

```bash
# npm 레지스트리 설정
npm config set registry https://registry.npmmirror.com

# 또는 nrm을 사용하여 레지스트리 관리
npm install -g nrm
nrm use taobao
```

### Node.js 설치

**nvm 사용 (권장):**

```bash
# nvm 설치
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Node.js LTS 버전 설치
nvm install --lts
nvm use --lts
```

**Windows에서 nvm-windows 사용:**

https://github.com/coreybutler/nvm-windows에서 인스톨러를 다운로드하세요

## 개발 서버 시작

### 프로젝트로 이동

```bash
cd my-app
```

### 개발 서버 시작

```bash
ng serve
```

기본 주소: http://localhost:4200

### 라이브 리로드

Angular CLI는 라이브 리로드를 지원합니다—코드 변경 시 브라우저가 자동으로 새로고침됩니다.

## 프로덕션 빌드

### 빌드 명령

```bash
ng build```

결과물은 `dist` 디렉터리에 저장됩니다.

### 프로덕션 최적화

```bash
ng build --configuration=production
```

이 명령은 압축, 트리 쉐이킹 및 기타 최적화를 활성화합니다.

## 일반적인 CLI 명령

| 명령 | 설명 |
|---------|-------------|
| ng generate component xxx | 컴포넌트 생성 |
| ng generate service xxx | 서비스 생성 |
| ng generate module xxx | 모듈 생성 |
| ng generate directive xxx | 디렉티브 생성 |
| ng generate pipe xxx | 파이프 생성 |
| ng generate guard xxx | 라우트 가드 생성 |

## 개발 도구

### VS Code 확장

- **Angular Language Service**: 템플릿 타입 검사 및 자동완성 제공
- **Angular Snippets**: 일반적인 코드 스니펫
- **Prettier**: 코드 포맷팅

### 디버깅

Angular는 디버깅 및 성능 프로파일을 위한 Angular DevTools 브라우저 확장을 제공합니다.

## Angular CLI 제거

제거가 필요한 경우:

```bash
npm uninstall -g @angular/cli
```
