# HeyGen Guía de Uso

Esta guía cubre cómo usar HeyGen para la generación de video con IA.

## Comenzar

### Usar la interfaz web

1. **Inicia sesión** en HeyGen en https://www.heygen.com
2. **Selecciona un avatar** de la biblioteca o carga uno personalizado
3. **Elige el tipo de entrada**: Texto, Audio o Video
4. **Genera el video** y descárgalo

### Usar la API

```python
from heygen import HeyGen

heygen = HeyGen(api_key="your_api_key")

# Generar video desde texto
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="Hello, this is my first AI video!",
    voice_id="voice_id"
)

print(f"URL del video: {video.url}")
```

## Funciones principales

### 1. Selección de avatar

```python
# Listar avatares disponibles
avatars = heygen.list_avatars()
for avatar in avatars:
    print(f"{avatar.id}: {avatar.name}")

# Obtener un avatar específico
avatar = heygen.get_avatar(avatar_id="avatar_id")
```

### 2. Selección de voz

```python
# Listar voces disponibles
voices = heygen.list_voices()
for voice in voices:
    print(f"{voice.id}: {voice.name} ({voice.language})")

# Obtener detalles de la voz
voice = heygen.get_voice(voice_id="voice_id")
```

### 3. Generación de video

#### Desde texto

```python
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="Tu texto de guión aquí",
    voice_id="voice_id",
    background="office"  # Opcional: escena de fondo
)
```

#### Desde audio

```python
video = heygen.generate_video_from_audio(
    avatar_id="avatar_id",
    audio_url="https://example.com/audio.wav"
)
```

#### Desde video (TalkingPhoto)

```python
video = heygen.generate_talking_photo(
    photo_url="https://example.com/photo.jpg",
    script="Tu guión aquí",
    voice_id="voice_id"
)
```

### 4. Generación por lotes

```python
scripts = [
    "Guión del primer video",
    "Guión del segundo video",
    "Guión del tercer video"
]

# Generar múltiples videos
jobs = heygen.batch_generate(
    avatar_id="avatar_id",
    scripts=scripts,
    voice_id="voice_id"
)

# Verificar estado del trabajo
for job in jobs:
    status = heygen.get_job_status(job.id)
    print(f"Trabajo {job.id}: {status}")
```

## Funciones avanzadas

### Avatar personalizado

```python

    name="Mi avatar personalizado",
    video_url="https://example.com/reference.mp4"

```

### Clonación instantánea de voz

```python
# Clonar voz desde muestra de audio
voice = heygen.clone_voice(
    name="Mi clon de voz",
    audio_url="https://example.com/voice_sample.wav"
)
print(f"ID de voz clonada: {voice.id}")
```

### Edición de video

```python
# Obtener detalles del video
video = heygen.get_video(video_id="video_id")

# Actualizar configuración del video
updated = heygen.update_video(
    video_id="video_id",
    title="Mi video actualizado",
    description="Descripción actualizada"
)

# Eliminar video
heygen.delete_video(video_id="video_id")
```

## Referencia de parámetros

### Parámetros de generate_video

| Parámetro | Tipo | Requerido | Descripción |
|-----------|------|-----------|-------------|
| avatar_id | string | Sí | ID del avatar a usar |
| script | string | Sí | Texto a decir |
| voice_id | string | Sí | ID de la voz |
| background | string | No | Escena de fondo |
| speed | float | No | Velocidad del habla (0.5-2.0) |
| emotion | string | No | Estilo de emoción |
| caption | boolean | No | Habilitar subtítulos |

### Emociones disponibles

| Emoción | Descripción |
|---------|-------------|
| Neutral | Expresión predeterminada |
| Happy | Expresión feliz |
| Sad | Expresión triste |
| Angry | Expresión enojada |
| Surprised | Expresión sorprendida |

## Mejores prácticas

### 1. Escritura de guión

- Mantén las oraciones cortas y claras
- Usa lenguaje natural
- Evita terminología compleja

### 2. Selección de avatar

- Combina el avatar con el caso de uso
- Considera las preferencias de la audiencia
- Prueba múltiples avatares

### 3. Selección de voz

- Combina la voz con la marca
- Considera los requisitos de idioma
- Prueba con guiones de muestra

### 4. Optimización de video

- Usa la resolución apropiada
- Optimiza para la plataforma (redes sociales, sitio web)
- Agrega subtítulos para accesibilidad

## Solución de problemas

### Problemas de calidad de video

**Problema:** El video se ve pixelado

**Soluciones:**
- Usa una configuración de mayor resolución
- Asegúrate de que la imagen de origen sea de alta calidad
- Verifica la conexión a internet

### Problemas de audio

**Problema:** El audio no coincide con el video

**Soluciones:**
- Usa las voces proporcionadas
- Asegúrate de que el audio esté correctamente formateado
- Verifica la configuración de sincronización de labios

### Tiempo de procesamiento

**Problema:** El video tarda demasiado

**Soluciones:**
- Verifica el saldo de créditos
- Usa cola para trabajos por lotes
- Considera el procesamiento prioritario
