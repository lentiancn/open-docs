# Apifox

Apifox 是國內領先的 API 協作平台，一站式解決 API 設計、Mock、測試、檔案等問題。它整合了 API 設計、API 檔案管理、API 調試、API Mock、API 自動化測試等功能，是國內最受歡迎的 API 管理工具之一。

## 快速開始

### 下載安裝

造訪 https://apifox.com/ 下載對應系統的安裝套件。

### 建立專案

1. 啟動 Apifox，登入帳號
2. 點擊「新建專案」
3. 輸入專案名稱

### 建立介面

1. 在專案中點擊「新建介面」
2. 填寫介面資訊
3. 儲存

### 調試介面

1. 開啟介面
2. 點擊「傳送」
3. 查看回應

## 核心功能

- **API 設計**：視覺化定義 API 介面
- **API 檔案**：自動產生，支援多種格式匯出
- **API 調試**：強大的線上調試功能
- **API Mock**：智慧 Mock，自動產生模擬資料
- **自動化測試**：無需編寫程式碼即可完成 API 測試
- **團隊協作**：即時同步，權限管理
