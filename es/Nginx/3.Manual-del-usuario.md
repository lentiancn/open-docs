# Manual de usuario de Nginx

## Operaciones básicas

### Iniciar Nginx

```bash
# Iniciar Nginx
nginx

# O usando systemctl (para systemd)
systemctl start nginx
```

### Detener Nginx

```bash
# Apagado rápido (terminar inmediatamente)
nginx -s stop

# Apagado graceful (esperar a que terminen las solicitudes)
nginx -s quit
```

### Recargar configuración

```bash
# Recargar archivo de configuración
nginx -s reload
```

### Reabrir logs

```bash
# Reabrir archivos de log
nginx -s reopen
```

### Verificar configuración

```bash
# Probar sintaxis de configuración
nginx -t

# Ver configuración completa
nginx -T
```

## Estructura del archivo de configuración

### Estructura de configuración principal

```
nginx.conf
├── main (bloque global)
│   ├── user
│   ├── worker_processes
│   ├── error_log
│   └── pid
│
├── events (bloque events)
│   └── worker_connections
│
└── http (bloque http)
    ├── include
    ├── default_type
    ├── log_format
    ├── access_log
    ├── sendfile
    ├── keepalive_timeout
    ├── gzip
    │
    └── server (bloque server)
        ├── listen
        ├── server_name
        ├── location
        │   ├── root
        │   ├── index
        │   └── proxy_pass
        │
        └── error_page
```

### Directivas comunes

#### Bloque main

```nginx
# Usuario que ejecuta Nginx
user nginx;

# Número de procesos worker (auto detecta núcleos CPU)
worker_processes auto;

# Ruta de log de errores
error_log /var/log/nginx/error.log warn;

# Ruta de archivo PID
pid /var/run/nginx.pid;
```

#### Bloque events

```nginx
events {
    # Máximo de conexiones por proceso worker
    worker_connections 1024;
    
    # Usar modelo epoll (Linux)
    use epoll;
    
    # Aceptar múltiples conexiones simultáneamente
    multi_accept on;
}
```

#### Bloque http

```nginx
http {
    # Incluir tipos MIME
    include /etc/nginx/mime.types;
    
    # Tipo de archivo predeterminado
    default_type application/octet-stream;
    
    # Log de acceso
    access_log /var/log/nginx/access.log;
    
    # Enviar archivo
    sendfile on;
    
    # TCP_NOPUSH y TCP_NODELAY
    tcp_nopush on;
    tcp_nodelay on;
    
    # Tiempo de espera keep-alive
    keepalive_timeout 65;
    
    # Habilitar compresión gzip
    gzip on;
}
```

## Servir archivos estáticos

### Configuración básica

```nginx
server {
    listen 80;
    server_name example.com www.example.com;
    
    # Directorio raíz
    root /var/www/html;
    
    # Archivos de índice predeterminados
    index index.html index.htm;
    
    location / {
        try_files $uri $uri/ =404;
    }
    
    # Logs de acceso
    access_log /var/log/nginx/example.com.access.log;
    error_log /var/log/nginx/example.com.error.log;
}
```

### Listado de directorios

```nginx
location / {
    # Habilitar listado de directorios
    autoindex on;
    
    # Mostrar tamaños de archivos
    autoindex_exact_size on;
    
    # Mostrar hora local
    autoindex_localtime on;
}
```

### Caché de recursos estáticos

```nginx
location ~* \.(jpg|jpeg|png|gif|ico|css|js|woff|woff2|ttf|svg)$ {
    expires 30d;
    add_header Cache-Control "public, no-transform";
}
```

## Proxy inverso

### Proxy inverso básico

```nginx
server {
    listen 80;
    server_name example.com;
    
    location / {
        proxy_pass http://backend_server;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

### Proxy inverso con caché

```nginx
proxy_cache_path /var/cache/nginx levels=1:2 keys_zone=my_cache:10m max_size=100m inactive=60m use_temp_path=off;

server {
    listen 80;
    server_name example.com;
    
    location / {
        proxy_cache my_cache;
        proxy_cache_valid 200 60m;
        proxy_cache_use_stale error timeout http_500 http_502 http_503 http_504;
        proxy_cache_lock on;
        
        proxy_pass http://backend_server;
        proxy_set_header Host $host;
    }
}
```

### Proxy de WebSocket

```nginx
location /ws/ {
    proxy_pass http://backend_server;
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection "upgrade";
    proxy_set_header Host $host;
    proxy_read_timeout 86400;
}
```

## Balanceo de carga

### Balanceo round-robin

```nginx
upstream backend {
    server backend1.example.com;
    server backend2.example.com;
    server backend3.example.com;
}

server {
    listen 80;
    server_name example.com;
    
    location / {
        proxy_pass http://backend;
    }
}
```

### Balanceo ponderado

```nginx
upstream backend {
    server backend1.example.com weight=5;
    server backend2.example.com weight=2;
    server backend3.example.com;
}
```

### Balanceo IP hash

```nginx
upstream backend {
    ip_hash;
    server backend1.example.com;
    server backend2.example.com;
    server backend3.example.com down;
}
```

### Least connections

```nginx
upstream backend {
    least_conn;
    server backend1.example.com;
    server backend2.example.com;
}
```

## Configuración SSL/TLS

### Configuración HTTPS básica

```nginx
server {
    listen 443 ssl http2;
    server_name example.com;
    
    ssl_certificate /etc/nginx/ssl/example.com.crt;
    ssl_certificate_key /etc/nginx/ssl/example.com.key;
    
    # Versiones de protocolo SSL
    ssl_protocols TLSv1.2 TLSv1.3;
    
    # Conjuntos de cifrado SSL
    ssl_ciphers ECDHE-ECDSA-AES128-GCM-SHA256:ECDHE-RSA-AES128-GCM-SHA256;
    ssl_prefer_server_ciphers off;
    
    # HSTS
    add_header Strict-Transport-Security "max-age=63072000" always;
    
    location / {
        root /var/www/html;
        index index.html;
    }
}
```

### Redirección de HTTP a HTTPS

```nginx
server {
    listen 80;
    server_name example.com;
    return 301 https://$server_name$request_uri;
}
```

## Ejemplos de configuración común

### Configuración PHP-FPM

```nginx
server {
    listen 80;
    server_name example.com;
    root /var/www/html;
    index index.php index.html;
    
    location ~ \.php$ {
        fastcgi_pass unix:/var/run/php/php-fpm.sock;
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        include fastcgi_params;
    }
    
    location ~ /\.ht {
        deny all;
    }
}
```

### Proxy Node.js

```nginx
server {
    listen 80;
    server_name example.com;
    
    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

### Configuración de limitación de tasa

```nginx
# Limitación basada en conexiones
limit_conn_zone $binary_remote_addr zone=addr:10m;

server {
    location / {
        limit_conn addr 10;
    }
}

# Limitación basada en tasa de solicitudes
limit_req_zone $binary_remote_addr zone=one:10m rate=10r/s;

server {
    location / {
        limit_req zone=one burst=20 nodelay;
    }
}
```

### Configuración de protección contra hotlinking

```nginx
location ~* \.(jpg|jpeg|png|gif|ico|css|js)$ {
    valid_referers none blocked example.com *.example.com;
    
    if ($invalid_referer) {
        return 403;
    }
}
```

## Gestión de logs

### Formato de log personalizado

```nginx
log_format main '$remote_addr - $remote_user [$time_local] "$request" '
                '$status $body_bytes_sent "$http_referer" '
                '"$http_user_agent" "$http_x_forwarded_for"';

access_log /var/log/nginx/access.log main;
```

### Logs condicionales

```nginx
map $request_uri $loggable {
    ~^/health-check 0;
    default 1;
}

access_log /var/log/nginx/access.log combined if=$loggable;
```

### Rotación de logs

Usando logrotate para rotación de logs:

```bash
# /etc/logrotate.d/nginx
/var/log/nginx/*.log {
    daily
    missingok
    rotate 14
    compress
    delaycompress
    notifempty
    create 0640 www-data adm
    sharedscripts
    postrotate
        [ -f /var/run/nginx.pid ] && kill -USR1 `cat /var/run/nginx.pid`
    endscript
}
```
