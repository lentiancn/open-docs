# Guide d'installation de Python

> Configuration complète

---

## Installation

### Windows

```powershell
choco install python
```

### macOS

```bash
brew install python@3.12
```

### Linux

```bash
sudo apt install python3 python3-pip
```

---

## Paquets

```bash
pip install requests
python -m venv mon_env
```
