# Bootstrap 5 安装指南

本指南详细介绍 Bootstrap 5 在各种环境下的安装和配置方法。

## 版本说明

| 版本 | 状态 | 说明 |
|------|------|------|
| Bootstrap 5.3.x | ✅ 当前 | 最新版本，包含所有新功能 |
| Bootstrap 5.2.x | ✅ 支持 | 稳定版本 |
| Bootstrap 5.1.x | ⚠️ 维护 | 安全更新 |
| Bootstrap 4.x | ❌ 停止维护 | 不推荐新项目使用 |

> **重要提示**：本指南主要针对 **Bootstrap 5.3.x**。Bootstrap 4 已停止维护，不建议用于新项目。

## 安装方法

### 方法一：CDN 引入（最简单）

最快速的方式是通过 CDN 引入 Bootstrap 的 CSS 和 JavaScript 文件。

#### CSS only

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
```

#### CSS + JavaScript

```html
<!-- Bootstrap CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- Bootstrap JS (包含 Popper) -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
```

**CDN 提供商**：
- [jsDelivr](https://www.jsdelivr.com/) - 推荐
- [cdnjs](https://cdnjs.com/libraries/bootstrap)
- [unpkg](https://unpkg.com/browse/bootstrap/)

#### 完整模板

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 5 页面</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>你好，Bootstrap 5！</h1>
    
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### 方法二：npm 安装

#### 使用 npm

```bash
# 创建新项目
npm init -y

# 安装 Bootstrap
npm install bootstrap@5.3.3

# 安装 Popper.js (如果需要)
npm install @popperjs/core
```

#### 在项目中引入

```javascript
// main.js 或 app.js
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

### 方法三：使用 Vite

#### 1. 创建 Vite 项目

```bash
npm create vite@latest my-bootstrap-app -- --template vanilla
cd my-bootstrap-app
npm install
```

#### 2. 安装 Bootstrap

```bash
npm install bootstrap@5.3.3
```

#### 3. 引入 Bootstrap

```javascript
// main.js
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

```javascript
// vite.config.js
export default {
  plugins: [],
  resolve: {
    alias: {
      '@': '/src'
    }
  }
}
```

### 方法四：使用 Webpack

#### 1. 安装依赖

```bash
npm install bootstrap@5.3.3
npm install --save-dev webpack webpack-cli css-loader style-loader
```

#### 2. 配置 Webpack

```javascript
// webpack.config.js
module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js'
  },
  module: {
    rules: [
      {
        test: /\.css$/,
        use: ['style-loader', 'css-loader']
      }
    ]
  }
};
```

#### 3. 引入 Bootstrap

```javascript
// src/index.js
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

### 方法五：使用 Vue CLI / Nuxt

#### Vue CLI

```bash

npm install -g @vue/cli

# 安装 Bootstrap

npm install bootstrap@5.3.3

# 在 main.js 中引入
import 'bootstrap/dist/css/bootstrap.min.css';

```

#### Nuxt 3

```bash
# 创建 Nuxt 项目
npx nuxi@latest init my-nuxt-app
cd my-nuxt-app
npm install

# 安装 Bootstrap
npm install bootstrap@5.3.3

export default defineNuxtConfig({
  css: ['bootstrap/dist/css/bootstrap.min.css'],
  build: {
    transpile: ['bootstrap']
  }
})
```

### 方法七：下载源码

#### 从 GitHub 下载

```bash

git clone https://github.com/twbs/bootstrap.git
cd bootstrap

# 安装依赖
npm install

# 构建
npm run dist
```

#### 直接下载

1. 访问 [Bootstrap 官方发布页](https://github.com/twbs/bootstrap/releases)
2. 下载 Source code (zip)
3. 解压后引入 `dist/css/bootstrap.min.css` 和 `dist/js/bootstrap.bundle.min.js`

### 使用 Sass 自定义

```bash
npm install bootstrap@5.3.3
npm install --save-dev sass
```

```scss
// custom.scss
// 引入 Bootstrap 函数
@import "bootstrap/scss/functions";

// 自定义变量
$primary: #ff6b35;
$font-family-sans-serif: "Noto Sans SC", sans-serif;

// 引入 Bootstrap 其他部分
@import "bootstrap/scss/variables";
@import "bootstrap/scss/mixins";
@import "bootstrap/scss/utilities";
@import "bootstrap/scss/root";
@import "bootstrap/scss/reboot";
@import "bootstrap/scss/type";
@import "bootstrap/scss/containers";
@import "bootstrap/scss/grid";
@import "bootstrap/scss/utilities/api";
// 添加其他需要的组件...
```

```javascript
// vite.config.js
export default {
  css: {
    preprocessorOptions: {
      scss: {
        additionalData: `@import "bootstrap/scss/functions"; @import "custom-variables";`
      }
    }
  }
}
```

## 常见问题

### Popper 是什么？

Popper 是一个定位引擎，用于工具提示（Tooltip）和弹出框（Popover）的定位。`bootstrap.bundle.min.js` 已包含 Popper，无需单独安装。

### jQuery 是否还需要？

**不需要！** Bootstrap 5 完全使用原生 JavaScript 重写，不再依赖 jQuery。

### 如何使用图标？

Bootstrap 5 使用 Bootstrap Icons：

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css">

<i class="bi bi-github"></i>
<i class="bi bi-bootstrap-fill"></i>
```

### 主题颜色如何修改？

可以通过 CSS 变量覆盖：

```css
:root {
  --bs-primary: #ff6b35;
  --bs-secondary: #6c757d;
  --bs-success: #198754;
}
```
