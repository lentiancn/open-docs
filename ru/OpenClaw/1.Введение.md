# OpenClaw 🦞

OpenClaw — это **шлюз с самостоятельным хостингом**, который соединяет ваши любимые приложения для обмена сообщениями — WhatsApp, Telegram, Discord, iMessage и другие — с ИИ-помощниками для программирования, такими как Pi. Вы запускаете один процесс Gateway на своём компьютере (или сервере), и он становится мостом между вашими приложениями для обмена сообщениями и всегда доступным ИИ-помощником.

## Для кого это?

OpenClaw предназначен для разработчиков и опытных пользователей, которые хотят иметь личного ИИ-помощника, с которым можно связаться из любого места, не отказываясь от контроля над своими данными и не полагаясь на хостинговый сервис.

## Ключевые особенности

Что отличает OpenClaw:

- **Самостоятельный хостинг**: работает на вашем оборудовании, ваши правила
- **Множество каналов**: один Gateway обслуживает WhatsApp, Telegram, Discord и другие одновременно
- **Нативный для агентов**: создан для агентов программирования с использованием инструментов, сессий, памяти и маршрутизацией нескольких агентов
- **Открытый исходный код**: лицензия MIT, сообщество

## Что вам нужно

- Node.js 22 или выше
- API-ключ от выбранного вами ИИ-провайдера
- 5 минут

Для лучшего качества и безопасности используйте самую мощную модель последнего поколения.

## Быстрый старт

```bash
# Установить OpenClaw
npm install -g openclaw@latest

# Запустить мастер настройки
openclaw onboard --install-daemon

# Войти в каналы и запустить Gateway
openclaw channels login
openclaw gateway --port 18789
```

## Как это работает

Gateway — единственный источник истины для сессий, маршрутизации и подключений каналов. Приложения для обмена сообщениями подключаются через него к ИИ-агентам, CLI, веб-интерфейсу управления или мобильным узлам.

## Основные возможности

- **Многоканальный шлюз**: WhatsApp, Telegram, Discord и iMessage с одним процессом Gateway
- **Плагинные каналы**: добавьте Mattermost и другое через пакеты расширений
- **Маршрутизация нескольких агентов**: изолированные сессии для каждого агента, рабочей области или отправителя
- **Поддержка мультимедиа**: отправка и получение изображений, аудио и документов
- **Веб-интерфейс управления**: панель управления для чата, настроек, сессий и узлов
- **Мобильные узлы**: подключение узлов iOS и Android для Canvas, камеры/экрана и голосовых рабочих процессов
