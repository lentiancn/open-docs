# Rukovodstvo po ispolzovaniyu Vue 2

Rukovodstvo po effektivnomu ispolzovaniyu Vue 2.

---

## Bazovye ponyatiya

Vue 2 yavlyaetsya JavaScript-freymworkom dlya sozdaniya polzovatelskikh intereysov.

---

## Instanciya

```javascript
new Vue({
  el: '#app',
  data: {
    message: 'Hello Vue!'
  }
})
```

---

## Direktivy

| Direktiva | Opisaniye |
|-----------|------------|
| `v-if` | Uslovnaya vizualizaciya |
| `v-for` | Vizualizaciya spiskov |
| `v-bind` | Svyaz attributov |
| `v-model` | Dvukhnapravlennaya svyaz |

---

## Komponenty

```vue
<template>
  <div>{{ title }}</div>
</template>

<script>
export default {
  props: {
    title: String
  }
}
</script>
```
