# OpenClaw FAQ

## Basics

### 1. What is OpenClaw?

OpenClaw is a self-hosted gateway connecting chat apps (WhatsApp, Telegram, Discord, iMessage) to AI coding assistants.

### 2. Is OpenClaw free?

OpenClaw itself is free, but you need:
- A computer/server to run Gateway
- AI service API key (Anthropic, OpenAI)

### 3. Is OpenClaw secure?

Yes:
- Data stored locally
- Whitelist support
- Group mention rules
- Open source (MIT license)

### 4. What platforms are supported?

- OS: macOS, Linux, Windows
- Chat: WhatsApp, Telegram, Discord, iMessage
- Mobile: iOS, Android (nodes)

## Installation

### 5. Installation failed?

- Check Node version: `node --version` (need 22+)
- Permission issues: Try sudo
- Network issues: Check internet

### 6. Port in use?

```bash
lsof -i :18789
openclaw gateway --port 18889
```

### 7. How to upgrade?

```bash
npm install -g openclaw@latest
```

## Configuration

### 8. How to configure API key?

```bash
openclaw config set providers.anthropic.key <your-key>
```

### 9. How to restrict users?

```bash
openclaw security allow +15555550123
```

### 10. Config file location?

`~/.openclaw/openclaw.json`

## Usage

### 11. Gateway won't start?

```bash
openclaw status
openclaw logs
```

### 12. WhatsApp login failed?

```bash
openclaw channels login
```

### 13. How to view chat history?

```bash
openclaw sessions list
openclaw sessions history --session-id <id>
```

### 14. Multiple AI agents?

Configure in config file:
```json
{
  "agents": {
    "coding": {"provider": "anthropic"},
    "general": {"provider": "openai"}
  }
}
```

## Mobile

### 15. Pair mobile device?

```bash
openclaw nodes pair
```

### 16. Mobile not pairing?

- Same network
- Check firewall
- Regenerate code
