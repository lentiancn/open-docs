# Claude Code Guía de instalación

Instrucciones detalladas de instalación para Claude Code.

## Instalación en Windows

### Método uno: Extensión de VS Code

1. Abre VS Code
2. Busca la extensión "Claude Code"
3. Haz clic en Instalar
4. Inicia sesión en tu cuenta de Anthropic

### Método dos: Cliente independiente

1. Visita https://claude.com/code
2. Descarga el cliente para Windows
3. Ejecuta el instalador
4. Inicia sesión

## Instalación en macOS

### Método uno: Extensión de VS Code

1. Abre VS Code
2. Busca la extensión "Claude Code"
3. Haz clic en Instalar
4. Inicia sesión

### Método dos: Homebrew

```
brew install claude-code
```

### Método tres: Cliente independiente

1. Visita https://claude.com/code
2. Descarga el cliente para macOS
3. Ejecuta el instalador
4. Inicia sesión

## Instalación en Linux

### Método uno: Extensión de VS Code

1. Abre VS Code
2. Busca la extensión "Claude Code"
3. Haz clic en Instalar
4. Inicia sesión

### Método dos: Gestor de paquetes

**Ubuntu/Debian**:
```
curl -fsSL https://downloads.anthropic.com/claude-code/install.sh | sh
```

**Arch Linux**:
```
yay -S claude-code
```

## Verificar instalación

```
claude --version
```

## Configuración

### Clave API

1. Visita https://console.anthropic.com
2. Crea una clave API
3. Configura en Claude Code

### Variables de entorno

```
export ANTHROPIC_API_KEY=tu-clave-api
```

## Desinstalar

### macOS

```
brew uninstall claude-code
```

### Linux

```
sudo rm -rf /usr/local/claude-code
```
