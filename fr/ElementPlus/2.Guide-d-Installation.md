# Element Plus Guide d'installation

Plusieurs méthodes d'installation pour Element Plus.

## Configuration requise

### Configuration matérielle

- CPU: 2 cœurs ou plus
- Mémoire: 4 Go ou plus

### Configuration logicielle

- Node.js: 18.0+
- npm: 9.0+ / yarn: 1.22+ / pnpm: 8.0+
- Vue: 3.2+
- Vite: 4.0+ (recommandé)

## Installation via gestionnaire de packages (recommandé)

### npm

```bash
npm install element-plus
```

### yarn

```bash
yarn add element-plus
```

### pnpm

```bash
pnpm add element-plus
```

## Import complet

### Configuration main.ts

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')
```

## Import à la demande (recommandé)

### Import automatique (recommandé)

**1. Installer les dépendances**

```bash
npm install -D unplugin-vue-components unplugin-auto-import
```

**2. Configurer Vite**

```typescript
// vite.config.ts
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

**3. Utiliser les composants**

```vue
<template>
  <el-button type="primary">Bonjour</el-button>
</template>
```

### Import manuel à la demande

```typescript
import { ElButton, ElInput } from 'element-plus'

export default {
  components: {
    ElButton,
    ElInput,
  },
}
```

## Installation avec Vue CLI

### Installer

```bash
npm install -D sass sass-loader
npm install element-plus
```

### Configurer

```javascript
// vue.config.js
module.exports = {
  configureWebpack: {
    plugins: [
      AutoImport({
        resolvers: [ElementPlusResolver()],
      }),
      Components({
        resolvers: [ElementPlusResolver()],
      }),
    ],
  },
}
```

## Import via CDN

### HTML

```html
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>Element Plus CDN</title>
  <link rel="stylesheet" href="//unpkg.com/element-plus/dist/index.css" />
  <script src="//unpkg.com/vue@3"></script>
  <script src="//unpkg.com/element-plus"></script>
</head>
<body>
  <div id="app">
    <el-button type="primary">Bouton primaire</el-button>
  </div>

  <script>
    const { createApp, ref } = Vue
    const App = {
      setup() {
        const visible = ref(false)
        return { visible }
      },
      template: `
        <el-button @click="visible = true">Ouvrir le dialogue</el-button>
        <el-dialog v-model="visible" title="Titre">Contenu</el-dialog>
      `
    }
    const app = createApp(App)
    app.use(ElementPlus)
    app.mount('#app')
  </script>
</body>
</html>
```

## Problèmes courants

### Les icônes ne s'affichent pas?

Installer le package d'icônes:

```bash
npm install @element-plus/icons-vue
```

Enregistrer:

```typescript
import * as icons from '@element-plus/icons-vue'

for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```

### Conflits de styles?

Utiliser CSS Modules ou scoped:

```vue
<style scoped>
:deep(.el-button) {
  margin: 10px;
}
</style>
```

### Erreurs de type TypeScript?

S'assurer que les définitions de types sont installées:

```bash
npm install -D @types/node
```

Configurer dans `tsconfig.json`:

```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

### Configuration de l'internationalisation

```typescript
import fr from 'element-plus/es/locale/lang/fr'

app.use(ElementPlus, { locale: fr })
```

## Désinstaller

### Désinstaller via npm

```bash
npm uninstall element-plus
```

### Nettoyer la configuration

Supprimer:
- `element-variables.scss` (si utilisé)
- Configurations Vite/Webpack associées
