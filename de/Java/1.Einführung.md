# Java Einführung

## Was ist Java

Java ist eine objektorientierte Programmiersprache, die 1995 von Sun Microsystems entwickelt wurde und jetzt nach der Übernahme von Oracle Corporation gepflegt wird. Ursprünglich bekannt als Oak, wurde sie 1995 offiziell in Java umbenannt.

Java ist eine weit verbreitete Programmiersprache für die Unternehmensentwicklung. Ihr Designprinzip ist "Einmal schreiben, überall ausführen" (Write Once, Run Anywhere). Das bedeutet, dass in Java geschriebene Programme auf verschiedenen Betriebssystemen ausgeführt werden können, ohne neu kompiliert werden zu müssen.

## Kernmerkmale von Java

### Objektorientiert

Java ist eine rein objektorientierte Sprache, bei der aller Code in Klassen enthalten sein muss. Sie unterstützt Kapselung, Vererbung und Polymorphie und hilft Entwicklern, modulare und wartbare Anwendungen zu erstellen.

### Plattformunabhängigkeit

Java-Programme werden über die Java Virtual Machine (JVM) ausgeführt, die Implementierungen für verschiedene Betriebssysteme hat. Derselbe Java-Bytecode kann auf jedem System mit installierter JVM ausgeführt werden. Dies ist eines der wichtigsten Merkmale von Java.

### Automatische Speicherverwaltung

Java bietet Garbage Collection, um den Speicher für Objekte während der Programmausführung automatisch zu verwalten. Entwickler müssen Speicher nicht manuell freigeben, was Speicherlecks und Dangling-Pointer-Probleme erheblich reduziert.

### Sicherheit

Das Design von Java umfasst mehrere Sicherheitsebenen, einschließlich Bytecode-Verifizierung, Sicherheitsmanager und Zugriffskontrollen, die die Ausführung von schädlichem Code effektiv verhindern.

### Multithreading-Unterstützung

Java hat integrierte Multithreading-Unterstützung, die die gleichzeitige Ausführung mehrerer Aufgaben ermöglicht und die Parallelitätsleistung des Programms verbessert.

### Umfangreiche Klassenbibliothek

Java wird mit einer umfassenden Standard-Klassenbibliothek (Java API) geliefert, die Dateioperationen, Netzwerkprogrammierung, Datenbankkonnektivität, grafische Benutzeroberflächen, Webentwicklung und mehr abdeckt.

## Java-Versionsentwicklung

### Long Term Support (LTS)-Versionen

- Java 8: Veröffentlicht 2014, die am weitesten verbreitete LTS-Version, führte Lambda-Ausdrücke und Stream API ein
- Java 11: Veröffentlicht 2018, fügte lokale Variablentyp-Inferenz (var) und neue Garbage Collectors hinzu
- Java 17: Veröffentlicht 2021, die neueste kostenlose LTS-Version mit zusätzlichen Sprachverbesserungen
- Java 21: Veröffentlicht 2023, führte wichtige Funktionen wie virtuelle Threads ein
- Java 25: Veröffentlicht 2025, die aktuell neueste LTS-Version

### Versionsauswahl-Empfehlungen

Für neue Projekte werden Java 21 oder Java 25 empfohlen, da sie die neuesten Sprachfunktionen und Leistungsverbesserungen bieten. Für die Wartung bestehender Systeme wählen Sie eine geeignete Version basierend auf Projektabhängigkeiten und Teamvertrautheit.

- Unternehmensanwendungen: Große verteilte Systeme, ERP-Systeme, CRM-Systeme
- Web-Anwendungen: E-Commerce-Plattformen, Content-Management-Systeme, Unternehmensportale

## Java-Ökosystem

Java hat ein vollständiges Ökosystem, das umfasst:

- Entwicklungstools: IntelliJ IDEA, Eclipse, VS Code
- Build-Tools: Maven, Gradle
- Frameworks: Spring, Hibernate, MyBatis, Struts
- Anwendungsserver: Tomcat, Jetty, WildFly, WebLogic

## Zusammenfassung

Als eine der beliebtesten Programmiersprachen nimmt Java eine wichtige Position in der Unternehmensentwicklung aufgrund seiner Stabilität, Skalierbarkeit und reichhaltigen Ökosystems ein. Sowohl für Anfänger als auch für erfahrene Entwickler ist Java eine wertvolle Fähigkeit zu lernen und zu beherrschen.
