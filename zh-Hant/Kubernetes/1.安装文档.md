# Kubernetes 安裝文檔

在所有主流平台上安裝 Kubernetes 的詳細指南。

---

## 安裝方式

| 方式 | 描述 |
|------|------|
| **minikube** | 單節點本地叢集 |
| **kind** | Docker 中運行 K8s |
| **kubeadm** | 生產級叢集 |

---

## minikube（推薦新手）

```bash
# macOS
brew install minikube

# 啟動叢集
minikube start
```

---

## kind

```bash
brew install kind
kind create cluster
```

---

## kubeadm

```bash
sudo apt-get update
sudo apt-get install -y kubelet kubeadm kubectl
sudo kubeadm init
kubectl apply -f https://docs.projectcalico.org/manifests/calico.yaml
```

---

## 安裝 kubectl

```bash
brew install kubectl
```

---

## 驗證

```bash
kubectl get nodes
```
