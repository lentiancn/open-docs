# Angular 설치 가이드

Angular는 Google이 개발한 강력한 프론트엔드 프레임워크입니다. 이 가이드에서는 다양한 환경의 설치 방법을 설명합니다.

## 버전 정보

| 버전 | 상태 | 설명 |
|---------|--------|-------------|
| Angular 18.x | ✅ 현재 | 최신 버전 |
| Angular 17.x | ✅ 지원 | 이전 버전 |
| Angular 16.x | ✅ 지원 | 이전 버전 |
| Angular 15.x | ✅ 지원 | 레거시 버전 |

## 시스템 요구사항

### 하드웨어 요구사항

| 요구사항 | 최소 | 권장 |
|-------------|---------|-------------|
| CPU | 2코어 | 4코어 이상 |
| RAM | 4GB | 8GB 이상 |
| 디스크 | 3GB | 10GB 이상 SSD |

### 소프트웨어 요구사항

| 소프트웨어 | 버전 |
|----------|---------|
| Node.js | 18.19.1+ 또는 20.10.0+ |
| npm | 10.2.3+ |

## 설치 방법

### 방법 1: Angular CLI (권장)

#### Angular CLI 설치

```bash
# 전역 설치
npm install -g @angular/cli

# 설치 확인
ng version
```

#### 새 프로젝트 생성

```bash
# 새 프로젝트 생성
ng new my-app

# 옵션과 함께 생성
ng new my-app --routing --style=scss --skip-tests
```

#### 옵션

```bash
# 설치 건너뛰기
ng new my-app --skip-install

# 독립 컴포넌트 (Angular 14+)
ng new my-app --standalone

# SSR 활성화
ng new my-app --ssr
```

### 방법 2: npx 사용

```bash
# CLI 설치 없이 프로젝트 생성
npx @angular/cli new my-app
```

### 방법 3: 수동 설정

```bash
# 디렉토리 생성
mkdir my-app
cd my-app

# npm 초기화
npm init -y

# Angular 설치
npm install @angular/core @angular/common @angular/platform-browser @angular/platform-browser-dynamic @angular/compiler @angular/forms @angular/router @angular/animations rxjs zone.js

# 개발 의존성 설치
npm install --save-dev @angular/compiler-cli @angular-devkit/build-angular typescript
```

### 방법 4: Docker

```dockerfile
# Dockerfile
FROM node:20-alpine

WORKDIR /app

# Angular CLI 설치
RUN npm install -g @angular/cli

# 프로젝트 생성
RUN ng new my-app --directory .

# 개발 서버 시작
CMD ["ng", "serve", "--host", "0.0.0.0"]
```

## 환경 설정

### Node.js 버전 관리

#### nvm 사용 (권장)

```bash
# nvm 설치
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Node.js 설치
nvm install 20
nvm use 20

# 기본 설정
nvm alias default 20
```

### npm 설정

```bash
# 레지스트리 설정
npm config set registry https://registry.npmjs.org/

# 엄격한 SSL 활성화
npm config set strict-ssl true

# 캐시 지우기
npm cache clean --force
```

## 확인

```bash
# Angular CLI 확인
ng version

# Node.js 확인
node --version

# npm 확인
npm --version
```

## 일반적인 문제

### 권한 거부?

```bash
# npm 권한 수정
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
export PATH=~/.npm-global/bin:$PATH
```

### 설치가 느림?

```bash
# 더 빠른 설치를 위해 npm ci 사용
npm ci

# 또는 yarn 사용
npm install -g yarn
```

## Angular 업데이트

```bash
# Angular CLI 업데이트
npm install -g @angular/cli@latest

# 프로젝트 업데이트
ng update @angular/core @angular/cli
```
