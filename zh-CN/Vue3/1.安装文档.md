# Vue 3 安装文档

在所有主流平台上安装 Vue 3 的完整指南。

---

## 什么是 Vue 3？

Vue 3 是构建用户界面的渐进式 JavaScript 框架的最新版本。它具有更好的性能、Composition API 和 TypeScript 支持。

---

## CDN

### 基本 CDN 使用

```html
<!-- 开发版 -->
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<!-- 生产版 -->
<script src="https://unpkg.com/vue@3/dist/vue.global.prod.js"></script>
```

### CDN 提供商

| 提供商 | URL |
|--------|-----|
| unpkg | https://unpkg.com/vue@3 |
| jsDelivr | https://cdn.jsdelivr.net/npm/vue@3 |

---

## NPM

### 安装 Vue 3

```bash
# 安装 Vue 3
npm install vue@3
```

---

## Vite

### 创建新项目（推荐）

```bash
# 使用 Vite 创建项目
npm create vite@latest my-vue3-app -- --template vue

# 进入项目
cd my-vue3-app

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### Vite 选项

```bash
# 使用 TypeScript
npm create vite@latest my-app -- --template vue-ts

# 安装 Vue Router
npm install vue-router@4

# 安装 Pinia（状态管理）
npm install pinia
```

## 项目结构

```
my-vue3-app/
├── src/
│   ├── components/
│   ├── App.vue
│   └── main.js
├── index.html
└── package.json
```

### main.js

```javascript
import { createApp } from 'vue'
import App from './App.vue'

createApp(App).mount('#app')
```
