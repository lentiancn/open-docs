# PyTorch

An Open Source Machine Learning Framework

## Overview

PyTorch is a powerful open-source machine learning framework with dynamic computation graphs, widely used for deep learning research and production.

## Quick Start

```python
import torch

x = torch.randn(3, 3)
print(x)
```

## Features

- Dynamic computation graphs
- GPU acceleration
- Rich ecosystem (torchvision, torchaudio)
- Easy to debug and prototype
- ONNX export support
