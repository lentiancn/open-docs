# Flutter - FAQ

Häufige Fragen.

## Installation

### Download fehlgeschlagen?

```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
```

### Flutter doctor Fehler?

PATH überprüfen.

### Android SDK nicht gefunden?

```bash
flutter config --android-sdk /pfad/zur/android-sdk
```

## Entwicklung

### Hot Reload funktioniert nicht?

Speichern und warten.

### APK Build fehlgeschlagen?

```bash
flutter clean
```
