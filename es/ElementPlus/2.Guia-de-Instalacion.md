# Element Plus Guía de instalación

Múltiples métodos de instalación para Element Plus.

## Requisitos del entorno

### Requisitos de hardware

- CPU: 2 núcleos o más
- Memoria: 4GB o más

### Requisitos de software

- Node.js: 18.0+
- npm: 9.0+ / yarn: 1.22+ / pnpm: 8.0+
- Vue: 3.2+
- Vite: 4.0+ (recomendado)

## Instalación con gestor de paquetes (recomendado)

### npm

```bash
npm install element-plus
```

### yarn

```bash
yarn add element-plus
```

### pnpm

```bash
pnpm add element-plus
```

## Importación completa

### Configuración main.ts

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')
```

## Importación bajo demanda (recomendado)

### Importación automática (recomendado)

**1. Instalar dependencias**

```bash
npm install -D unplugin-vue-components unplugin-auto-import
```

**2. Configurar Vite**

```typescript
// vite.config.ts
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

**3. Usar componentes**

```vue
<template>
  <el-button type="primary">Hola</el-button>
</template>
```

### Importación manual bajo demanda

```typescript
import { ElButton, ElInput } from 'element-plus'

export default {
  components: {
    ElButton,
    ElInput,
  },
}
```

## Instalación con Vue CLI

### Instalar

```bash
npm install -D sass sass-loader
npm install element-plus
```

### Configurar

```javascript
// vue.config.js
module.exports = {
  configureWebpack: {
    plugins: [
      AutoImport({
        resolvers: [ElementPlusResolver()],
      }),
      Components({
        resolvers: [ElementPlusResolver()],
      }),
    ],
  },
}
```

## Importación desde CDN

### HTML

```html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Element Plus CDN</title>
  <link rel="stylesheet" href="//unpkg.com/element-plus/dist/index.css" />
  <script src="//unpkg.com/vue@3"></script>
  <script src="//unpkg.com/element-plus"></script>
</head>
<body>
  <div id="app">
    <el-button type="primary">Botón primario</el-button>
  </div>

  <script>
    const { createApp, ref } = Vue
    const App = {
      setup() {
        const visible = ref(false)
        return { visible }
      },
      template: `
        <el-button @click="visible = true">Abrir diálogo</el-button>
        <el-dialog v-model="visible" title="Título">Contenido</el-dialog>
      `
    }
    const app = createApp(App)
    app.use(ElementPlus)
    app.mount('#app')
  </script>
</body>
</html>
```

## Personalización del tema

### Usar variables CSS

```css
:root {
  --el-color-primary: #409eff;
  --el-color-success: #67c23a;
  --el-border-radius-base: 4px;
}
```

### Personalización profunda

**1. Instalar generador de temas**

```bash
npm install -D element-theme-chalk
```

**2. Inicializar archivos de tema**

```bash
et -i
```

**3. Modificar variables**

Editar `element-variables.scss`, luego:

```bash
et
```

### Usar tema oscuro

```typescript
import 'element-plus/theme-chalk/dark/css-vars.css'
```

## Problemas comunes

### ¿Los iconos no se muestran?

Instalar paquete de iconos:

```bash
npm install @element-plus/icons-vue
```

Registrar:

```typescript
import * as icons from '@element-plus/icons-vue'

for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```

### ¿Conflictos de estilos?

Usar CSS Modules o scoped:

```vue
<style scoped>
:deep(.el-button) {
  margin: 10px;
}
</style>
```

### ¿Errores de tipo TypeScript?

Asegurarse de que las definiciones de tipos estén instaladas:

```bash
npm install -D @types/node
```

Configurar en `tsconfig.json`:

```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

### Configuración de internacionalización

```typescript
import es from 'element-plus/es/locale/lang/es'

app.use(ElementPlus, { locale: es })
```

## Desinstalar

### Desinstalar con npm

```bash
npm uninstall element-plus
```

### Limpiar configuración

Eliminar:
- `element-variables.scss` (si se usó)
- Configuraciones Vite/Webpack relacionadas
