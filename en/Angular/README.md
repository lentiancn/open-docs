# Angular

Angular is an open-source web application framework developed and maintained by Google for building modern web applications. It uses TypeScript as its primary development language and provides powerful features like component-based architecture, two-way data binding, and dependency injection.

## Quick Start

### Install Angular CLI

```bash
npm install -g @angular/cli
```

### Create a New Project

```bash
ng new my-app
cd my-app
```

### Start Development Server

```bash
ng serve
```

Visit http://localhost:4200

## Key Features

- **Component-based Architecture**: Modular, reusable UI components
- **TypeScript**: Strong typing for better code quality
- **Dependency Injection**: Loose coupling, easy to test
- **Two-Way Data Binding**: Data syncs automatically
- **Routing**: Complete routing solution
- **HTTP Client**: Powerful network request capabilities
- **Form Handling**: Reactive and template-driven forms
- **RxJS**: Reactive programming support
