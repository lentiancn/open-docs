# Angular

> Angular - モダンWebアプリケーション開発フレームワーク

---

## 简介

Angularは、GoogleがメンテナンスするオープンソースのWebアプリケーションフレームワークで、主にシングルページアプリケーション（SPA）の構築に使用されます。TypeScriptを主要な開発言語として使用し、完全な開発ツールチェーンと丰富的機能セットを提供します。

### 主な機能

- 📦 **コンポーネントアーキテクチャ** - コンポーネントベースの開発モデル
- 🔄 **双方向データバインディング** - ビューとデータを自動同期
- 💉 **依存性注入** - 強力な依存性管理システム
- 🌐 **ルーティングシステム** - 完全なクライアントサイドルーティングソリューション
- 📝 **フォーム処理** - 強力なフォーム検証機能
- 🔌 **HTTPクライアント** - シンプルなHTTP通信
- 🧪 **テストツール** - 完全なテストサポート
- ⚡ **高性能** - 最適化された変更検出とレンダリング

## クイックスタート

### 1. Angular CLIのインストール

```bash
npm install -g @angular/cli
```

### 2. 新規プロジェクトの作成

```bash
ng new my-app
cd my-app
```

### 3. 開発サーバーの起動

```bash
ng serve
```

ブラウザで http://localhost:4200 にアクセス

## プロジェクト構造

```
my-app/
├── src/
│   ├── app/
│   │   ├── app.component.ts    # ルートコンポーネント
│   │   ├── app.config.ts      # アプリケーション設定
│   │   └── app.routes.ts      # ルーティング設定
│   ├── assets/                 # 静的アセット
│   └── styles.css             # グローバルスタイル
├── angular.json                # CLI設定
├── package.json                # 依存関係設定
└── tsconfig.json              # TypeScript設定
```

## 常用コマンド

### コード生成

```bash
# コンポーネントの作成
ng generate component my-component

# サービスの作成
ng generate service services/my-service

# モジュールの作成
ng generate module my-module

# ルーティングの作成
ng generate routing
```

### 開発

```bash
# 開発サーバーの起動
ng serve

# 本番ビルド
ng build --configuration production

# テストの実行
ng test

# E2Eテストの実行
ng e2e
```
