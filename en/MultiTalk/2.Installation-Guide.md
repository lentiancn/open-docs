# MultiTalk Installation Guide

Guide to installing and configuring MultiTalk.

## Prerequisites

- Windows 10/11, macOS 10.15+, or Linux
- Microphone and speakers/headset
- Internet connection

## Installation

### Windows

1. Download the installer from the official website
2. Run the installer
3. Follow the installation wizard
4. Launch MultiTalk

### macOS

1. Download the .dmg file
2. Open the installer
3. Drag MultiTalk to Applications
4. Launch MultiTalk

### Linux

#### Debian/Ubuntu

```bash
sudo dpkg -i multitalk_5.x_amd64.deb
sudo apt-get install -f
```

#### Fedora

```bash
sudo rpm -i multitalk-5.x.fc34.x86_64.rpm
```

## Configuration

### Audio Setup

1. Open Settings > Audio
2. Select your input device (microphone)
3. Select your output device (speakers/headset)
4. Test audio levels

### Push-to-Talk

1. Go to Settings > Hotkeys
2. Enable push-to-talk
3. Press the key combination you want to use
4. Test the configuration

## Troubleshooting

### No audio input

**Solution:**
- Check microphone permissions
- Verify input device selection
- Test microphone in system settings

### Echo issues

**Solution:**
- Lower speaker volume
- Use headphones
- Adjust input gain

### Connection problems

**Solution:**
- Check internet connection
- Disable firewall temporarily
- Use a different server region
