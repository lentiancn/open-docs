# 安裝指南

本文檔介紹 Bootstrap 的多種安裝方式，幫助您快速開始在專案中使用 Bootstrap。

## 環境需求

Bootstrap 5 需要以下環境：

- **HTML5** 現代瀏覽器
- **CSS3** 支援
- **JavaScript** ES6+（如使用 JavaScript 外掛程式）

## 安裝方式

### 方式一：CDN 引入（推薦）

最簡單的方式是透過 CDN 直接引入 Bootstrap 的 CSS 和 JavaScript 檔案：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 頁面</title>
    <!-- 引入 Bootstrap CSS -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>你好，Bootstrap！</h1>

    <!-- 引入 Bootstrap JavaScript（包含 Popper） -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### 方式二：npm 安裝

如果使用 npm 管理專案依賴：

```bash
# 安裝 Bootstrap
npm install bootstrap@5

# 安裝 Popper（用於下拉選單、工具提示等）
npm install @popperjs/core
```

然後在專案中引入：

```javascript
// 引入 Bootstrap 的 CSS
import 'bootstrap/dist/css/bootstrap.min.css';

// 引入 Bootstrap 的 JS
import 'bootstrap/dist/js/bootstrap.bundle.min.js';
```

### 方式三：下載原始碼

從官網下載預先編譯好的檔案：

1. 前往 https://getbootstrap.com/
2. 點擊「Download」按鈕
3. 選擇「Compiled CSS and JS」下載
4. 解壓後將 css 和 js 資料夾複製到專案

目錄結構：

```
your-project/
├── css/
│   └── bootstrap.min.css
├── js/
│   └── bootstrap.bundle.min.js
└── index.html
```

### 方式四：Sass 原始碼

如果需要自訂 Bootstrap 樣式，可以下載 Sass 原始碼：

```bash
npm install bootstrap
```

然後在 Sass 檔案中引入：

```scss
// 自訂變數（在引入 Bootstrap 之前定義）
$primary: #007bff;
$body-bg: #f8f9fa;

// 引入 Bootstrap
@import "~bootstrap/scss/bootstrap";
```

## 開發工具

### VS Code 擴充套件

推薦安裝以下擴充套件：

- **Bootstrap 5 Snippets**：提供 Bootstrap 元件程式碼片段
- **HTML CSS Support**：CSS 類別名稱自動完成
- **Live Server**：即時預覽頁面

### 線上編輯器

- **Bootstrap Builder**：視覺化建構 Bootstrap 頁面
- **Bootstrap Studio**：專業的 Bootstrap 設計工具

## 快速開始

### 建立第一個頁面

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 快速開始</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <!-- 導航欄 -->
    <nav class="navbar navbar-expand-lg navbar-dark bg-primary">
        <div class="container">
            <a class="navbar-brand" href="#">我的網站</a>
        </div>
    </nav>

    <!-- 主要內容 -->
    <main class="container mt-5">
        <div class="row">
            <div class="col-md-6">
                <h1>歡迎使用 Bootstrap</h1>
                <p class="lead">這是一個使用 Bootstrap 5 建構的響應式頁面。</p>
                <button class="btn btn-primary">了解更多</button>
            </div>
            <div class="col-md-6">
                <div class="card">
                    <div class="card-body">
                        <h5 class="card-title">特色</h5>
                        <ul>
                            <li>響應式布局</li>
                            <li>豐富的元件</li>
                            <li>強大的 JavaScript 外掛程式</li>
                        </ul>
                    </div>
                </div>
            </div>
        </div>
    </main>

    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## 版本選擇

### 生產環境

使用穩定版本：

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### 開發環境

使用最新開發版本（包含原始碼）：

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.js"></script>
```

### 中文鏡像

國內可以使用中文鏡像加速：

```html
<link href="https://cdn.bootcss.com/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.bootcss.com/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

## 解除安裝

如果需要解除安裝（刪除相關檔案即可）：

- 刪除 HTML 中的 CSS 和 JS 引用
- 刪除 node_modules 中的 bootstrap 目錄（如果使用 npm）
- 刪除本地下載的 CSS 和 JS 檔案
