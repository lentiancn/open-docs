# ElementPlus 소개

ElementPlus는 Vue 3 기반 UI 컴포넌트 라이브러리입니다.

## ElementPlus란

ElementPlus는 Element UI의 Vue 3 버전입니다.

## 주요 기능

- Vue 3 지원
- 70+ 컴포넌트
- TypeScript 지원
- Tree Shaking
-国际化
- 테마 커스터마이징

## 컴포넌트

- 기본: 버튼, 아이콘, 레이아웃
- 폼: 입력, 선택, 날짜
- 데이터 표시: 테이블, 트리, 페이지네이션
- 피드백: 다이얼로그, 메시지

## 설치

```bash
npm install element-plus
```
