# Angular Overview

Modern web development platform.

## What is Angular?

Angular is a development platform for building mobile and desktop web applications, developed by Google. It uses TypeScript/JavaScript and other languages.

### Key Features

- **Component-based** - Component-driven architecture
- **TypeScript** - Strong typing support
- **Two-way binding** - Simplified data synchronization
- **Dependency injection** - Powerful dependency management
- **Routing** - Complete routing solution
- **Forms** - Reactive and template-driven forms
- **HTTP client** - Full HTTP services
- **Server-side rendering** - SSR support
