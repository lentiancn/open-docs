# OpenClaw 사용 매뉴얼

이 매뉴얼은 OpenClaw의 일상적인 작업과 기능을 설명합니다.

## 기본 개념

### Gateway

Gateway는 OpenClaw의 핵심입니다. 모든 메시지 라우팅, AI 요청 및 채널 관리는 이 단일 프로세스를 통해 이루어집니다. Gateway 하나만 실행하면 모든 채팅 플랫폼의 연결을 자동으로 처리합니다.

### 채널

채널은 AI 어시스턴트와 통신하는 데 사용하는 채팅 앱입니다. WhatsApp, Telegram, Discord, iMessage 등 여러 채널을 동시에 연결할 수 있습니다.

### 에이전트

에이전트는 AI 어시스턴트의 독립적인 인스턴스입니다. 각 에이전트에는 자체 작업 공간과 대화 기록이 있습니다. 다양한 목적으로 여러 에이전트를 실행할 수 있습니다.

## 일상적인 작업

### 시작 및 중지

```bash
# 시작 (백그라운드에서 실행)
openclaw gateway start

# 중지
openclaw gateway stop

# 다시 시작
openclaw gateway restart

# 상태 확인
openclaw gateway status

# 포그라운드에서 실행 (디버깅용)
openclaw gateway --port 18789
```

### 메시지 보내기

```bash
# 직접 메시지 보내기
openclaw message send --target +15555550123 --message "테스트 메시지"

# 그룹으로 보내기
openclaw message send --target "그룹 이름" --message "모두 안녕하세요"
```

### 구성 관리

OpenClaw는 JSON5 형식 구성 파일을 사용합니다. 위치: `~/.openclaw/openclaw.json`

구성 변경 방법:

1. **마법사 모드**:
   ```bash
   openclaw onboard
   openclaw configure
   ```

2. **명령줄**:
   ```bash
   openclaw config get agents.defaults.workspace
   openclaw config set agents.defaults.heartbeat.every "2h"
   openclaw config unset tools.web.search.apiKey
   ```

3. **웹 인터페이스**:
   http://127.0.0.1:18789/를 열고 Config 탭으로 이동

4. **직접 편집**:
   ```bash
   nano ~/.openclaw/openclaw.json
   ```

### 핫 리로드

대부분의 구성 변경은 재시작 없이 자동으로 적용됩니다.

## 일반적인 구성 예

### 액세스 제한

```json5
{
  channels: {
    whatsapp: {
      allowFrom: ["+15555550123"]
    }
  }
}
```

### 그룹 채팅 설정

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

### 여러 에이전트

```json5
{
  agents: {
    list: [
      { id: "home", default: true, workspace: "~/.openclaw/workspace-home" },
      { id: "work", workspace: "~/.openclaw/workspace-work" }
    ]
  }
}
```

### 예약 작업

```json5
{
  cron: {
    enabled: true,
    maxConcurrentRuns: 2
  }
}
```

### 주기적 체크인

```json5
{
  agents: {
    defaults: {
      heartbeat: {
        every: "30m",
        target: "last"
      }
    }
  }
}
```

### 샌드박스 격리

격리된 컨테이너에서 AI 작업을 실행하려면:

```json5
{
  agents: {
    defaults: {
      sandbox: {
        mode: "non-main",
        scope: "agent"
      }
    }
  }
}
```

### 웹훅

```json5
{
  hooks: {
    enabled: true,
    token: "비밀 토큰",
    path: "/hooks"
  }
}
```

## 채널 관리

### 채널 추가

```bash
# WhatsApp에 로그인
openclaw channels login whatsapp

# Telegram에 로그인
openclaw channels login telegram
```

### 연결된 채널 나열

```bash
openclaw channels list
```

## 문제 해결

### 로그 보기

```bash
# 서비스 로그
openclaw logs

# 실시간 로그 추적
openclaw logs -f
```

### 상태 확인

```bash
openclaw health
```

### 자동 진단

```bash
openclaw doctor
```

## 웹 콘솔

OpenClaw는 웹 인터페이스로 관리할 수 있습니다:

- 메시지 보내기 및 받기
- 설정 수정
- 대화 기록 보기
- 모바일 노드 관리
- 서비스 상태 모니터링

방문: http://127.0.0.1:18789/
