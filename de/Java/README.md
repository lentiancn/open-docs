# Java

## Übersicht

Java ist eine objektorientierte Programmiersprache, die von Oracle Corporation gepflegt wird. Es ist eine der beliebtesten Unternehmensentwicklungssprachen der Welt. Java ist bekannt für seine "Einmal schreiben, überall ausführen"-Eigenschaft und hat ein ausgereiftes Ökosystem mit breiten Anwendungsbereichen.

## Hauptmerkmale

- Objektorientiert: Vollständig objektorientierte Programmiersprache
- Plattformunabhängig: Plattformübergreifende Ausführung durch JVM
- Automatische Speicherverwaltung: Garbage-Collection-Mechanismus
- Reichhaltige Klassenbibliothek: Umfassende Standard-API
- Multithreading-Unterstützung: Integrierte Unterstützung für parallele Programmierung
- Sicherheit: Mehrschichtige Sicherheitsmechanismen

## Schnellstart

### JDK installieren

Besuchen Sie die offizielle Oracle-Download-Seite, um das JDK zu erhalten: https://www.oracle.com/java/technologies/downloads/

### Ihr erstes Programm schreiben

Erstellen Sie HelloWorld.java:

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

### Kompilieren und Ausführen

```bash
# Kompilieren
javac HelloWorld.java

# Ausführen
java HelloWorld
```
