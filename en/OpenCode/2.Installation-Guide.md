# OpenCode Installation Guide

OpenCode supports multiple installation methods.

## Linux/macOS Installation

### Recommended: Install Script

```bash
curl -fsSL https://opencode.ai/install | bash
```

### Node.js

```bash
npm install -g opencode-ai
```

### Bun

```bash
bun install -g opencode-ai
```

### pnpm

```bash
pnpm install -g opencode-ai
```

### Yarn

```bash
yarn global add opencode-ai
```

### Homebrew (macOS/Linux)

```bash
brew install anomalyco/tap/opencode
```

### Arch Linux

```bash
# Stable
sudo pacman -S opencode

# Latest (AUR)
paru -S opencode-bin
```

## Windows Installation

### Chocolatey

```bash
choco install opencode
```

### Scoop

```bash
scoop install opencode
```

### NPM

```bash
npm install -g opencode-ai
```

### Mise

```bash
mise use -g github:anomalyco/opencode
```

### Docker

```bash
docker run -it --rm ghcr.io/anomalyco/opencode
```

### Manual Download

You can also download the binary from [GitHub Releases](https://github.com/anomalyco/opencode/releases).

## Verify Installation

```bash
opencode --version
```

## Configuration

### Connect LLM Provider

Run the following to connect:

```bash
opencode
/connect
```

Select `opencode`, then visit [opencode.ai/auth](https://opencode.ai/auth) to log in and get your API key.

### Other Providers

You can also select other LLM providers. See [official docs](https://opencode.ai/docs/providers#directory) for details.

## Initialize Project

```bash
cd /path/to/project
opencode
/init
```

This will have OpenCode analyze your project and create an AGENTS.md file in the project root.
