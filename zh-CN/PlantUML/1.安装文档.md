# PlantUML 安装指南

PlantUML 是一个用于快速创建 UML 图的开源工具。本指南涵盖在各种环境中的安装方法。

## 版本信息

| 版本 | 状态 | 说明 |
|------|------|------|
| PlantUML | ✅ 当前 | 最新稳定版本 |

## 安装方法

### 方法一：Windows 安装

#### 1. 下载 JAR 文件

1. 下载 PlantUML JAR：https://sourceforge.net/projects/plantuml/files/plantuml.jar/download
2. 保存到本地目录

#### 2. 安装 Graphviz（可选但推荐）

```powershell
# 使用 Chocolatey
choco install graphviz
```

### 方法二：macOS 安装

```bash
# 使用 Homebrew
brew install plantuml

# 或安装 Graphviz
brew install graphviz
```

### 方法三：Linux 安装

```bash
# Ubuntu/Debian
sudo apt-get install -y plantuml graphviz

# CentOS/RHEL
sudo yum install -y plantuml graphviz
```

### 方法四：作为 VS Code 插件

1. 打开 VS Code
2. 搜索 "PlantUML"
3. 安装 "PlantUML" 插件
4. 安装 Graphviz

### 方法五：作为 IntelliJ 插件

1. 打开 IntelliJ IDEA
2. Settings → Plugins
3. 搜索 "PlantUML"
4. 安装插件

### 方法六：使用 Docker

```bash
# 运行 PlantUML 服务器
docker run -d -p 8080:8080 plantuml/plantuml-server:jetty

# 访问 http://localhost:8080
```

## 环境配置

### Java 要求

```bash
# 检查 Java
java -version

# 需要 Java 8 或更高版本
```

### Graphviz 配置

```bash
# 设置环境变量（Linux/macOS）
export GRAPHVIZ_DOT=/usr/bin/dot

# Windows 添加到 PATH
# C:\Program Files\Graphviz\bin
```

## 常见问题

### 图片不显示？

确保已安装 Graphviz：

```bash
dot -V
```

### 中文乱码？

在图表中添加：

```plantuml
skinparam defaultFontName SimSun
```
