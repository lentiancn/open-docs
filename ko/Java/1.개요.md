# Java 개요

> Java는 객체지향 프로그래밍 언어입니다

---

## 설명

Java는 1995년 Sun Microsystems에서 만들었습니다. 플랫폼 독립성으로 유명합니다.

---

## 특징

- 객체지향
- 플랫폼 독립 (JVM)
- 자동 메모리 관리
- 멀티스레딩
- 풍부한 생태계
