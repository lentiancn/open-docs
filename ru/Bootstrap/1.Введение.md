# Введение

Bootstrap — самый популярный в мире открытый фреймворк для разработки интерфейсов на HTML, CSS и JavaScript. Он был создан дизайнерами Twitter — Марком Отто и Джейкобом Торнтоном. Bootstrap включает в себя множество готовых компонентов и мощных JavaScript-плагинов, которые позволяют разработчикам быстро создавать адаптивные веб-приложения.

Главный принцип Bootstrap — «сначала мобильные» (Mobile First). Фреймворк с самого начала проектировался с учётом адаптации под разные устройства. С помощью Bootstrap можно легко создавать сайты, которые отлично выглядят на компьютерах, планшетах и смартфонах.

## История развития

Изначально Bootstrap назывался Twitter Blueprint и был открыт в 2011 году. В 2012 году вышла версия Bootstrap 2 с поддержкой адаптивного дизайна. В 2013 году появился Bootstrap 3 — полностью переписанный с учётом плоского дизайна. В 2015 году вышел Bootstrap 4, где впервые начали использовать Sass для CSS. В 2021 году был представлен Bootstrap 5: отказались от jQuery и полностью перешли на чистый JavaScript.

## Основные возможности

### Адаптивная вёрстка

Сетка — одна из самых сильных сторон Bootstrap. Благодаря 12-колоночной системе разработчики могут легко создавать макеты под любые размеры экранов. В Bootstrap 5 есть шесть контрольных точек: xs, sm, md, lg, xl и xxl — каждая для устройств определённой ширины.

### Готовые компоненты

Bootstrap предлагает огромный набор готовых UI-компонентов: навигационные панели, карточки, кнопки, формы, модальные окна, карусели, выпадающие списки и многое другое. Все компоненты тщательно продуманы и обеспечивают приятный визуальный опыт.

### JavaScript-плагины

В Bootstrap входит более 20 JavaScript-плагинов: модальные окна, карусели, всплывающие подсказки, уведомления, аккордеоны и прочее. Каждый плагин подключается отдельно, что позволяет не увеличивать размер проекта лишним кодом.

### Поддержка Sass

Bootstrap 5 использует Sass как препроцессор CSS. Предоставляется множество Sass-переменных и миксинов. Это позволяет легко настраивать темы: цвета, размеры шрифтов и другие стили.

### Создание тем

Благодаря системе тем Bootstrap разработчики могут без труда создавать собственные дизайн-системы. На официальном маркетплейсе Bootstrap Themes доступны готовые профессиональные темы.

### Доступность

Все компоненты Bootstrap изначально поддерживают доступность: клавиатурную навигацию, совместимость с читалками экрана и многое другое. Это помогает создавать сайты, которыми могут пользоваться все.

## Что нового в Bootstrap 5

По сравнению с Bootstrap 4, в пятой версии появилось много важных изменений:

- Убрали зависимость от jQuery, полностью перешли на нативный JavaScript
- Улучшили сетку, добавили контрольную точку XXL
- Новые элементы форм вместо старой валидации
- Лучшая поддержка пользовательских атрибутов
- Иконки теперь из Bootstrap Icons вместо Font Awesome
- Поддержка RTL-направления текста (справа налево)

## Когда это нужно

- Быстрая разработка прототипов
- Внутренние системы управления предприятием
- Интернет-магазины
- Блоги и контентные сайты
- Административные панели
- Адаптивные веб-сайты

Как один из самых популярных CSS-фреймворков в мире, Bootstrap имеет огромное сообщество и богатые учебные материалы. Это обязательный навык для любого frontend-разработчика.
