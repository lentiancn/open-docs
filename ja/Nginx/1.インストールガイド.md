# Nginx インストールガイド

すべての主要プラットフォームに Nginx をインストールするためのガイド。

---

## Linux

### Ubuntu / Debian

```bash
# パッケージ更新
sudo apt update

# Nginx インストール
sudo apt install nginx

# 起動
sudo systemctl start nginx

# 自動起動
sudo systemctl enable nginx
```

### CentOS / RHEL

```bash
# EPEL リポジトリインストール
sudo yum install epel-release

# Nginx インストール
sudo yum install nginx

# 起動
sudo systemctl start nginx
```

### Fedora

```bash
sudo dnf install nginx
sudo systemctl start nginx
```

### Arch Linux

```bash
sudo pacman -S nginx
sudo systemctl start nginx
```

---

## macOS

### Homebrew

```bash
brew install nginx
brew services start nginx
```

---

## Windows

### バイナリパッケージ

1. [nginx.org](http://nginx.org/en/download.html) からダウンロード
2. `C:\nginx` に展開

```powershell
cd C:\nginx
start nginx
```

### Docker

```bash
docker run -d -p 80:80 --name nginx nginx:latest
```

---

## 検証

```bash
nginx -v
nginx -t
curl http://localhost
```

---

## 設定

### 主な設定ファイル

| OS | パス |
|----|------|
| Ubuntu/Debian | `/etc/nginx/nginx.conf` |
| CentOS/RHEL | `/etc/nginx/nginx.conf` |

### 主なコマンド

```bash
nginx -t
sudo systemctl reload nginx
```

---

## アンインストール

### Ubuntu/Debian

```bash
sudo apt remove nginx
```

### CentOS/RHEL

```bash
sudo yum remove nginx
```

### macOS

```bash
brew uninstall nginx
```
