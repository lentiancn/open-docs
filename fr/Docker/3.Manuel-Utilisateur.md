# Docker Manuel utilisateur

Concepts de base, utilisation et guide de développement.

## Opérations sur les images

### Tirer des images

```
docker pull nginx
docker pull ubuntu:22.04
```

### Construire des images

```
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
EXPOSE 3000
CMD ["npm", "start"]
```

```
docker build -t my-app:tag .
```

### Gérer les images

```
docker images
docker rmi nginx
```

## Opérations sur les conteneurs

### Créer et démarrer des conteneurs

```
# Conteneur interactif
docker run -it ubuntu /bin/bash

# Conteneur détaché
docker run -d nginx

# Mapping de ports
docker run -d -p 8080:80 nginx

# Variables d'environnement
docker run -e MYSQL_ROOT_PASSWORD=123456 mysql
```

### Gérer les conteneurs

```
docker ps              # Conteneurs en cours d'exécution
docker ps -a         # Tous les conteneurs
docker start container_id
docker stop container_id
docker rm container_id
```

### Entrer dans un conteneur

```
docker exec -it container_id /bin/bash
```

### Voir les logs

```
docker logs container_id
docker logs -f container_id
```

## Écrire des Dockerfiles

### Instructions de base

```
FROM node:18-alpine
WORKDIR /app
COPY . .
RUN npm install
EXPOSE 3000
CMD ["npm", "start"]
```

### Builds multi-étapes

```
# Étape de construction
FROM node:18 AS builder
WORKDIR /app
COPY . .
RUN npm run build

# Étape de production
FROM nginx:alpine
COPY --from=builder /app/dist /usr/share/nginx/html
```

## Docker Compose

### docker-compose.yml

```
version: '3.8'

services:
  web:
    build: .
    ports:
      - "3000:3000"
  db:
    image: postgres:15
    environment:
      POSTGRES_PASSWORD: secret
```

### Commandes courantes

```
docker-compose up -d
docker-compose down
docker-compose logs -f
```

## Gestion réseau

### Mapping de ports

```
docker run -p 8080:80 nginx
```

### Réseaux personnalisés

```
docker network create my-network
docker run --network my-network nginx
```

## Gestion des données

### Volumes

```
docker volume create my-data
docker run -v my-data:/data nginx
```

### Bind mounts

```
docker run -v /host/path:/container/path nginx
```

## Limites de ressources

```
# Limite mémoire
docker run -m 512m nginx

# Limite CPU
docker run --cpus=0.5 nginx
```

## Surveillance et débogage

```
docker stats
docker inspect container_id
docker logs container_id
```

## Nettoyage

```
docker system prune -a
docker volume prune
```
