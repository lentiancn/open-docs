# Webpack Installationsanleitung

Webpack ist ein statischer Modul-Bundler für moderne JavaScript-Anwendungen. Diese Anleitung deckt die Installationsmethoden in allen Umgebungen ab.

## Versionsinformationen

| Version | Status | Beschreibung |
|---------|--------|--------------|
| Webpack 5.x | ✅ Aktuell | Neueste stabile Version |
| Webpack 4.x | ❌ Veraltet | Nicht für neue Projekte empfohlen |

> **Wichtig**: Diese Anleitung richtet sich an **Webpack 5.x**. Für neue Projekte wird die neueste Version empfohlen.

## Installationsmethoden

### Methode 1: Lokale Installation (Empfohlen)

```bash
# Projektverzeichnis erstellen
mkdir my-webpack-project
cd my-webpack-project

# npm initialisieren
npm init -y

# webpack installieren
npm install webpack --save-dev

# webpack-cli installieren
npm install webpack-cli --save-dev
```

### Methode 2: Globale Installation

```bash
npm install -g webpack webpack-cli
```

### Methode 3: npx verwenden

```bash
npx webpack --version
```

### Methode 4: Projektvorlagen erstellen

```bash
# create-react-app verwenden (hat webpack eingebaut)
npx create-react-app my-app

# Vue CLI verwenden
npm install -g @vue/cli
vue create my-app

# Vite verwenden
npm create vite@latest my-app -- --template vanilla
```

## webpack.config.js Konfiguration

### Grundkonfiguration

```javascript
const path = require('path');

module.exports = {
  mode: 'development', // oder 'production'
  
  // Eingabedatei
  entry: './src/index.js',
  
  // Ausgabekonfiguration
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
    clean: true, // dist-Verzeichnis bereinigen
  },
  
  // Loader
  module: {
    rules: [
      {
        test: /\.css$/i,
        use: ['style-loader', 'css-loader'],
      },
      {
        test: /\.(png|svg|jpg|jpeg|gif)$/i,
        type: 'asset/resource',
      },
    ],
  },
};
```

### Produktionskonfiguration

```javascript
const path = require('path');
const HtmlWebpackPlugin = require('html-webpack-plugin');

module.exports = {
  mode: 'production',
  entry: './src/index.js',
  output: {
    filename: '[name].[contenthash].js',
    path: path.resolve(__dirname, 'dist'),
    clean: true,
  },
  module: {
    rules: [
      {
        test: /\.css$/i,
        use: ['style-loader', 'css-loader'],
      },
      {
        test: /\.js$/i,
        exclude: /node_modules/,
        use: {
          loader: 'babel-loader',
          options: {
            presets: ['@babel/preset-env'],
          },
        },
      },
    ],
  },
  plugins: [
    new HtmlWebpackPlugin({
      template: './src/index.html',
    }),
  ],
  optimization: {
    splitChunks: {
      chunks: 'all',
    },
  },
};
```

### Entwicklungs konfiguration

```javascript
const path = require('path');
const HtmlWebpackPlugin = require('html-webpack-plugin');

module.exports = {
  mode: 'development',
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
  },
  devServer: {
    static: './dist',
    hot: true,
    port: 3000,
    open: true,
  },
  plugins: [
    new HtmlWebpackPlugin({
      template: './src/index.html',
    }),
  ],
};
```

## Loader-Konfiguration

### CSS-Loader

```bash
npm install --save-dev style-loader css-loader
```

```javascript
module: {
  rules: [
    {
      test: /\.css$/i,
      use: ['style-loader', 'css-loader'],
    },
  ],
}
```

### Sass-Loader

```bash
npm install --save-dev sass sass-loader style-loader css-loader
```

```javascript
{
  test: /\.scss$/i,
  use: ['style-loader', 'css-loader', 'sass-loader'],
}
```

```javascript
{

  exclude: /node_modules/,
```

## Plugin-Konfiguration

### HtmlWebpackPlugin

```bash
npm install --save-dev html-webpack-plugin
```

```javascript
const HtmlWebpackPlugin = require('html-webpack-plugin');

plugins: [
  new HtmlWebpackPlugin({
    title: 'My App',
    template: './src/index.html',
    filename: 'index.html',
  }),
];
```

### MiniCssExtractPlugin

```bash
npm install --save-dev mini-css-extract-plugin
```

```javascript
const MiniCssExtractPlugin = require('mini-css-extract-plugin');

module.exports = {
  plugins: [
    new MiniCssExtractPlugin({
      filename: '[name].[contenthash].css',
    }),
  ],
  module: {
    rules: [
      {
        test: /\.css$/i,
        use: [MiniCssExtractPlugin.loader, 'css-loader'],
      },
    ],
  },
};
```

### DefinePlugin

```javascript
const webpack = require('webpack');

plugins: [
  new webpack.DefinePlugin({
    'process.env.NODE_ENV': JSON.stringify('production'),
  }),
];
```

## Entwicklungsserver

### Installation

```bash
npm install --save-dev webpack-dev-server
```

### Konfiguration

```javascript
module.exports = {
  devServer: {
    static: {
      directory: path.join(__dirname, 'dist'),
    },
    compress: true,
    port: 3000,
    hot: true,
    open: true,
    historyApiFallback: true,
    proxy: {
      '/api': 'http://localhost:8080',
    },
  },
};
```

### Ausführung

```bash
# package.json
{
  "scripts": {
    "start": "webpack serve --mode development",
    "build": "webpack --mode production"
  }
}

npm start
```

## Häufige Probleme

### Build schlägt fehl?

Überprüfe die Fehlermeldung und stelle sicher, dass alle Abhängigkeiten installiert sind:

```bash
npm install
npm audit fix
```

### Modul nicht gefunden?

Stelle sicher, dass resolve.extensions korrekt konfiguriert ist:

```javascript
resolve: {
  extensions: ['.js', '.jsx', '.ts', '.tsx'],
}
```

### Hot Reload funktioniert nicht?

Stelle sicher, dass die devServer-Konfiguration korrekt ist:

```javascript
devServer: {
  hot: true,
  liveReload: true,
}
```
