# Angular Обзор

Современный фреймворк для веб-разработки.

## Что такое Angular?

Angular — это открытый веб-фреймворк, разработанный и поддерживаемый Google. Это одно из предпочтительных решений для современной веб-разработки.

### Основные функции

- **Нативный TypeScript**: Полностью на TypeScript
- **Компонентная архитектура**: Модульная разработка для простого сопровождения
- **Внедрение зависимостей**: Мощная система DI для лучшей тестируемости
- **Система директив**: Директивы для расширения возможностей HTML
- **Маршрутизация**: Полное клиентское решение для маршрутизации
- **Формы**: Реактивные формы и формы на основе шаблонов
- **HTTP-клиент**: Полный HTTP-клиент
- **Интернационализация**: Встроенная поддержка i18n
- **Анимации**: Мощный API анимаций
- **Тестирование**: Полная поддержка юнит- и E2E-тестирования

### Техническая архитектура

Angular использует модульную архитектуру с ключевыми концепциями:

- **Модуль**: Базовая единица для организации кода
- **Компонент**: Сочетание представления и бизнес-логики
- **Сервис**: Инкапсулированная повторно используемая бизнес-логика
- **Директива**: Расширяет поведение HTML-элементов
- **ПаЙп**: Утилита для преобразования данных

## Области применения

- **Корпоративные приложения**: Крупные сложные бизнес-системы
- **Одностраничные приложения (SPA)**: Приложения с плавным пользовательским интерфейсом
- **Прогрессивные веб-приложения (PWA)**: Приложения с офлайн-возможностями
- **Гибридные мобильные приложения**: Комбинировать с Ionic для мобильных приложений
- **Дашборды**: Визуализация данных и панели администрирования
