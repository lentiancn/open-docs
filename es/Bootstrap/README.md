# Bootstrap

Bootstrap es el framework de desarrollo front-end de código abierto más popular del mundo para HTML, CSS y JavaScript, creado y mantenido por Twitter. Ofrece una amplia variedad de componentes y potentes plugins de JavaScript que ayudan a los desarrolladores a crear aplicaciones web responsivas de forma rápida.

## Inicio Rápido

### Incluir via CDN

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### Instalar con npm

```bash
npm install bootstrap@5
```

## Funciones Principales

- **Diseño Responsivo**: Sistema de cuadrícula de 12 columnas que se adapta a cualquier pantalla
- **Componentes Variados**: Barras de navegación, tarjetas, formularios, ventanas modales y más
- **Plugins de JavaScript**: Menús desplegables, carruseles, paneles colapsables y otros
- **Soporte Sass**: Personaliza fácilmente el tema
