## Introducción a Bootstrap

Bootstrap es el marco HTML, CSS y JS más popular para desarrollar diseños responsivos, sitios web y aplicaciones web adaptados para dispositivos móviles.

### Características de Bootstrap

- **Diseño Responsivo**: Sistema de cuadrícula responsivo integrado
- **Estilos Predefinidos**: Rica gama de componentes y clases de estilo
- **Componentes JavaScript**: Componentes interactivos (modales, carruseles, etc.)
- **Personalización**: Personalización de temas a través de variables Sass
- **Compatibilidad con Navegadores**: Soporta todos los navegadores modernos

### Historial de Versiones de Bootstrap

| Versión | Fecha de Lanzamiento | Características Principales |
|---------|----------------------|----------------------------|
| Bootstrap 5 | Mayo 2021 | Eliminada dependencia de jQuery, componentes de formulario mejorados, nueva API de Utilidades |
| Bootstrap 4 | Enero 2018 | Soporte Flexbox, Sass en lugar de Less, nuevos componentes |
| Bootstrap 3 | Agosto 2013 | Diseño primero para dispositivos móviles |
| Bootstrap 2 | Enero 2012 | Mejoras en diseño responsivo |
| Bootstrap 1 | Agosto 2011 | Versión inicial |

> **Nota**: Bootstrap 5 es la versión principal actual y se recomienda para nuevos proyectos. Bootstrap 4 aún recibe actualizaciones de mantenimiento, pero el desarrollo de nuevas funciones ha cesado.

## Instalación por CDN

### Solo CSS

Agregue el siguiente código a la sección `<head>` de su HTML:

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH" crossorigin="anonymous">
```

### CSS + JavaScript

Después de agregar CSS, agregue JS antes de `</body>`:

```html
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-YvpcrYf0tY3lHB60NNkmXc5s9fDVZLESaAA55NDzOxhy9GkcIdslK1eN7N6jIeHz" crossorigin="anonymous"></script>
```

> **Nota**: Bootstrap 5 usa `bootstrap.bundle.min.js` (incluye Popper), ya no necesita incluir Popper por separado.

### Uso de CDN Nacional (Recomendado para Usuarios de China)

```html
<!-- CSS -->
<link href="https://cdn.staticfile.org/twitter-bootstrap/5.3.3/css/bootstrap.min.css" rel="stylesheet">

<!-- JS -->
<script src="https://cdn.staticfile.org/twitter-bootstrap/5.3.3/js/bootstrap.bundle.min.js"></script>
```

## Instalación por npm

### Instalar Bootstrap

```bash
# Instalar Bootstrap usando npm
npm install bootstrap

# Instalar versión específica
npm install bootstrap@5.3.3

# Instalar Bootstrap 4
npm install bootstrap@4.6.2
```

### Referenciar en el Proyecto

**CSS**:
```javascript
// webpack o bundler
import 'bootstrap/dist/css/bootstrap.min.css';
```

**JavaScript**:
```javascript
import * as bootstrap from 'bootstrap';

// O importar solo los módulos necesarios
import { Modal } from 'bootstrap';
```

## Instalación con Gestor de Paquetes

### yarn

```bash
yarn add bootstrap
```

### pnpm

```bash
pnpm add bootstrap
```

### Ruby on Rails

Agregar al Gemfile:

```ruby
gem 'bootstrap', '~> 5.3'
```

Luego ejecute `bundle install`.

### Python/Django

```bash
pip install bootstrap5
```

O agregar a requirements.txt:

```
django-bootstrap5
```

## Instalación por Descarga

### Descargar Versión Precompilada

1. Visite el [Sitio Web Oficial de Bootstrap](https://getbootstrap.com/)
2. Haga clic en el botón "Download"
3. Descargue los archivos CSS y JS precompilados

### Descargar Versión con Código Fuente

Descargue la versión que contiene archivos fuente Sass:

1. Visite [Código Fuente de Bootstrap](https://github.com/twbs/bootstrap/releases)
2. Descargue el archivo comprimido del código fuente
3. Después de extraer, contiene:
   - `dist/css/` - CSS compilado
   - `dist/js/` - JS compilado
   - `scss/` - Archivos fuente Sass

### Estructura de Archivos

```
bootstrap/
├── css/
│   ├── bootstrap.css
│   ├── bootstrap.css.map
│   ├── bootstrap.min.css
│   ├── bootstrap.min.css.map
│   ├── bootstrap.rtl.css
│   └── bootstrap.rtl.min.css
├── js/
│   ├── bootstrap.bundle.js
│   ├── bootstrap.bundle.min.js
│   ├── bootstrap.esm.js
│   └── bootstrap.esm.min.js
└── scss/
    └── ...
```

## Verificar Instalación

### Plantilla HTML Básica

```html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Prueba de Bootstrap</title>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <div class="container mt-5">
    <h1>¡Hola, Bootstrap!</h1>
    <button class="btn btn-primary">Botón Primario</button>
  </div>
  
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### Pasos de Verificación

1. Guarde el código anterior como un archivo `.html`
2. Ábralo en un navegador
3. Si ve una página con estilos de Bootstrap, la instalación fue exitosa

## Inicio Rápido

### Usando el Sistema de Cuadrícula

```html
<div class="container">
  <div class="row">
    <div class="col-md-4">Columna 1</div>
    <div class="col-md-4">Columna 2</div>
    <div class="col-md-4">Columna 3</div>
  </div>
</div>
```

### Usando Componentes

**Botones**:
```html
<button class="btn btn-primary">Primario</button>
<button class="btn btn-secondary">Secundario</button>
<button class="btn btn-success">Éxito</button>
<button class="btn btn-danger">Peligro</button>
<button class="btn btn-warning">Advertencia</button>
<button class="btn btn-info">Información</button>
```

**Tarjetas**:
```html
<div class="card" style="width: 18rem;">
  <img src="..." class="card-img-top" alt="...">
  <div class="card-body">
    <h5 class="card-title">Título de Tarjeta</h5>
    <p class="card-text">Contenido de Tarjeta</p>
    <a href="#" class="btn btn-primary">Enlace</a>
  </div>
</div>
```

**Modales**:
```html
<!-- Botón Activador -->
<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#exampleModal">
  Abrir Modal
</button>

<!-- Modal -->
<div class="modal fade" id="exampleModal" tabindex="-1">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title">Título del Modal</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
      </div>
      <div class="modal-body">
        Contenido del Modal
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Cerrar</button>
        <button type="button" class="btn btn-primary">Guardar</button>
      </div>
    </div>
  </div>
</div>
```

## Preguntas Frecuentes

### Q1: ¿Cuál es la Diferencia entre Bootstrap 5 y Bootstrap 4?

Diferencias principales:
- Bootstrap 5 eliminó la dependencia de jQuery
- Usa JavaScript nativo
- Componentes de formulario mejorados
- Nueva API de Utilidades
- Ya no soporta IE

### Q2: ¿Cómo Personalizar el Tema de Bootstrap?

**Método 1: Usando Sass**
```scss
// Variables personalizadas
$primary: #ff6600;
$secondary: #cccccc;

// Importar Bootstrap
@import "bootstrap";
```

**Método 2: Usando la Herramienta de Personalización de Bootstrap**
Visite https://getbootstrap.com/docs/5.3/customize/options/

### Q3: ¿Bootstrap es Responsivo?

Sí, Bootstrap es diseño primero para dispositivos móviles:
- `col-` Pantalla extra pequeña (<576px)
- `col-sm-` Pantalla pequeña (≥576px)
- `col-md-` Pantalla mediana (≥768px)
- `col-lg-` Pantalla grande (≥992px)
- `col-xl-` Pantalla extra grande (≥1200px)
- `col-xxl-` Pantalla extra extra grande (≥1400px)

### Q4: ¿Cómo Usar Iconos de Bootstrap?

```bash
npm install bootstrap-icons
```

O use CDN:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css">
```

Uso:
```html
<i class="bi bi-github"></i>
```

### Q5: ¿Bootstrap 5 Soporta IE11?

No. Bootstrap 5 requiere navegadores modernos. Si necesita soporte para IE11, use Bootstrap 4.6.

### Q6: ¿Cómo Depurar Estilos de Bootstrap?

1. Use las herramientas de desarrollo del navegador (F12)
2. Verifique los nombres de clase del elemento
3. Vea las fuentes de estilo en el panel Computed

### Q7: ¿Qué Hacer si Bootstrap Conflicta con Otros Marcos?

- Use selectores específicos
- Utilice el espacio de nombres de Bootstrap
- Use las API proporcionadas por Bootstrap

### Q8: ¿Cómo Actualizar la Versión de Bootstrap?

1. Haga copia de seguridad del proyecto actual
2. Actualice el enlace CDN o la versión del paquete npm
3. Verifique los cambios importantes
4. Pruebe toda la funcionalidad
