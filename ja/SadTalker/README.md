# SadTalker ドキュメント

## 概要

SadTalkerは、単一の静的な画像と音声入力からリアリスティックな話し顔ビデオを生成する高度なAIプロジェクトです。深層学習技術を使用して、唇同期のフォトリアリスティックな顔アニメーションを作成します。

## クイックスタート

### 話し顔ビデオを生成

```bash
python inference.py \
    --source_image path/to/portrait.jpg \
    --audio path/to/speech.wav \
    --result_dir ./results
```

### パラメータ

| パラメータ | 説明 | デフォルト |
|-----------|------|----------|
| `--source_image` | 入力ポートレート画像 | 必須 |
| `--audio` | 入力音声ファイル | 必須 |
| `--result_dir` | 出力ディレクトリ | ./results |
| `--style` | アニメーションスタイル | reference |
| `--preprocess` | 前処理 | crop |

## 特徴

- **フォトリアリスティック生成**：高品質な顔合成
- **音声駆動**：自然な唇同期
- **アイデンティティ維持**：ソースアイデンティティを維持
- **複数のスタイル**：さまざまなアニメーションスタイル
- **使いやすい**：シンプルなコマンドラインインターフェース

## 要件

- Python 3.8+
- PyTorch 1.12+
- CUDA 11.3+
- 8GB+ GPUメモリ
