# Webpack 安装指南

Webpack 是现代 JavaScript 应用程序的静态模块打包器。本指南涵盖在所有环境中的安装方法。

## 版本信息

| 版本 | 状态 | 说明 |
|------|------|------|
| Webpack 5.x | ✅ 当前 | 最新稳定版本 |
| Webpack 4.x | ❌ 旧版 | 不建议新项目使用 |

> **重要提示**：本指南主要针对 **Webpack 5.x**。建议新项目使用最新版本。

## 前提条件

### 系统要求

| 要求 | 版本 |

| Node.js | 12.0.0+ |

## 安装方法

### 方法一：本地安装（推荐）

```bash
# 创建项目目录
mkdir my-webpack-project
cd my-webpack-project

# 初始化 npm
npm init -y

# 安装 webpack
npm install webpack --save-dev

# 安装 webpack-cli
npm install webpack-cli --save-dev
```

### 方法二：全局安装

```bash
npm install -g webpack webpack-cli
```

### 方法三：使用 npx

```bash
npx webpack --version
```

### 方法四：创建项目模板

```bash
# 使用 create-react-app（已内置 webpack）
npx create-react-app my-app

# 使用 Vue CLI
npm install -g @vue/cli
vue create my-app

# 使用 Vite
npm create vite@latest my-app -- --template vanilla
```

## 配置 webpack.config.js

### 基础配置

```javascript
const path = require('path');

module.exports = {
  mode: 'development', // 或 'production'
  
  // 入口文件
  entry: './src/index.js',
  
  // 输出配置
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
    clean: true, // 清理 dist 目录
  },
  
  // 加载器
  module: {
    rules: [
      {
        test: /\.css$/i,
        use: ['style-loader', 'css-loader'],
      },
      {
        test: /\.(png|svg|jpg|jpeg|gif)$/i,
        type: 'asset/resource',
      },
    ],
  },
};
```

### 生产环境配置

```javascript
const path = require('path');
const HtmlWebpackPlugin = require('html-webpack-plugin');

module.exports = {
  mode: 'production',
  entry: './src/index.js',
  output: {
    filename: '[name].[contenthash].js',
    path: path.resolve(__dirname, 'dist'),
    clean: true,
  },
  module: {
    rules: [
      {
        test: /\.css$/i,
        use: ['style-loader', 'css-loader'],
      },
      {
        test: /\.js$/i,
        exclude: /node_modules/,
        use: {
          loader: 'babel-loader',
          options: {
            presets: ['@babel/preset-env'],
          },
        },
      },
    ],
  },
  plugins: [
    new HtmlWebpackPlugin({
      template: './src/index.html',
    }),
  ],
  optimization: {
    splitChunks: {
      chunks: 'all',
    },
  },
};
```

### 开发环境配置

```javascript
const path = require('path');
const HtmlWebpackPlugin = require('html-webpack-plugin');

module.exports = {
  mode: 'development',
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
  },
  devServer: {
    static: './dist',
    hot: true,
    port: 3000,
    open: true,
  },
  plugins: [
    new HtmlWebpackPlugin({
      template: './src/index.html',
    }),
  ],
};
```

## 加载器配置

### CSS 加载器

```bash
npm install --save-dev style-loader css-loader
```

```javascript
module: {
  rules: [
    {
      test: /\.css$/i,
      use: ['style-loader', 'css-loader'],
    },
  ],
}
```

### Sass 加载器

```bash
npm install --save-dev sass sass-loader style-loader css-loader
```

```javascript
{
  test: /\.scss$/i,
  use: ['style-loader', 'css-loader', 'sass-loader'],
}
```

```bash
npm install --save-dev @babel/core @babel/preset-env babel-loader
```

```javascript
{
  test: /\.js$/i,
  exclude: /node_modules/,

    loader: 'babel-loader',
```

### TypeScript 配置

```bash
npm install --save-dev typescript ts-loader
```

```javascript

  exclude: /node_modules/,
```

## 插件配置

### HtmlWebpackPlugin

```bash
npm install --save-dev html-webpack-plugin
```

```javascript
const HtmlWebpackPlugin = require('html-webpack-plugin');

plugins: [
  new HtmlWebpackPlugin({
    title: 'My App',
    template: './src/index.html',
    filename: 'index.html',
  }),
];
```

### MiniCssExtractPlugin

```bash
npm install --save-dev mini-css-extract-plugin
```

```javascript
const MiniCssExtractPlugin = require('mini-css-extract-plugin');

module.exports = {
  plugins: [
    new MiniCssExtractPlugin({
      filename: '[name].[contenthash].css',
    }),
  ],
  module: {
    rules: [
      {
        test: /\.css$/i,
        use: [MiniCssExtractPlugin.loader, 'css-loader'],
      },
    ],
  },
};
```

### DefinePlugin

```javascript
const webpack = require('webpack');

plugins: [
  new webpack.DefinePlugin({
    'process.env.NODE_ENV': JSON.stringify('production'),
  }),
];
```

## 开发服务器

### 安装

```bash
npm install --save-dev webpack-dev-server
```

### 配置

```javascript
module.exports = {
  devServer: {
    static: {
      directory: path.join(__dirname, 'dist'),
    },
    compress: true,
    port: 3000,
    hot: true,
    open: true,
    historyApiFallback: true,
    proxy: {
      '/api': 'http://localhost:8080',
    },
  },
};
```

### 运行

```bash
# package.json
{
  "scripts": {
    "start": "webpack serve --mode development",
    "build": "webpack --mode production"
  }
}

npm start
```

## 常见问题

### 构建失败？

检查错误信息并确保所有依赖已安装：

```bash
npm install
npm audit fix
```

### 模块找不到？

确保正确配置了 resolve.extensions：

```javascript
resolve: {
  extensions: ['.js', '.jsx', '.ts', '.tsx'],
}
```

### 热更新不工作？

确保 devServer 配置正确：

```javascript
devServer: {
  hot: true,
  liveReload: true,
}
```
