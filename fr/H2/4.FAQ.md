# FAQ H2

## Installation

### Comment installer?
Maven: ajouter la dépendance com.h2database:h2

### Version Java?
Java 11+

## Connexion

### Utilisateur?
sa (mot de passe vide)

### URL?
jdbc:h2:mem:testdb

## Données

### Données perdues?
Les données en mémoire sont perdues à la fermeture.

### Chiffrement?
jdbc:h2:file:./encrypted;CIPHER=AES

## Outils

### Console?
java -jar h2.jar -web

http://localhost:8082
