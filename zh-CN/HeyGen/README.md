# HeyGen

AI 数字人视频生成平台

## 简介

HeyGen 是一个专注于 AI 视频生成的创新平台，利用先进的人工智能技术创建逼真的数字人视频。用户可以通过输入文字或上传音频，快速生成具有自然表情和口型同步的 AI 主播视频。

## 主要功能

- **AI 数字人**：生成逼真的虚拟人物形象
- **语音合成**：支持多种语言和音色选择
- **视频模板**：丰富的预设模板和场景
- **多语言支持**：中文、英文、日文等多语种
- **自定义形象**：基于照片创建专属数字人

## 快速开始

1. 访问 HeyGen 官网注册账号
2. 选择数字人形象或上传照片创建自定义形象
3. 输入视频文案或上传音频
4. 选择语音和背景
5. 生成并下载视频
