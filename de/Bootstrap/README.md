# Bootstrap 5

Bootstrap ist das beliebteste Open-Source-Front-End-Toolkit für den Aufbau von responsiven, mobilen Websites.

## Einführung

Bootstrap 5 ist die neueste Version, vollständig in nativem JavaScript neu geschrieben, ohne Abhängigkeit von jQuery. Es bietet reichhaltige CSS- und JavaScript-Komponenten, um Entwicklern zu helfen, schnell moderne Websites zu erstellen.

## Hauptfunktionen

- **Responsives Design**: Integriertes responsives Grid-System
- **Reichhaltige Komponenten**: Buttons, Navigation, Formulare, Karten, Modals
- **Starke Hilfsklassen**: Abstände, Farben, Text, Anzeige
- **Sass-Unterstützung**: Theme-Anpassung mit Sass
- **Symbol-Bibliothek**: Bootstrap Icons integriert
- **Kein jQuery**: Vollständig in nativem JavaScript

## Schnellstart

### CDN

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
```

### npm

```bash
npm install bootstrap@5.3.3
```
