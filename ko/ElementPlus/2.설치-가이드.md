# Element Plus Installation Guide

This guide describes how to install and use Element Plus in various projects.

## Environment Requirements

Element Plus has the following requirements:

- Vue 3.0.0+
- Node.js 14.0.0+
- Modern browsers (Chrome, Firefox, Safari, Edge)

## Installation

### Using Package Manager

Install using npm, yarn, or pnpm:

```bash
# npm
npm install element-plus

# yarn
yarn add element-plus

# pnpm
pnpm add element-plus
```

## Full Import

If you don't mind the bundle size, full import is more convenient.

### Vue 3 Project

In main.ts:

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)
app.use(ElementPlus)
app.mount('#app')
```

### Volar Support

If using Volar plugin, configure in tsconfig.json:

```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

## On-Demand Import

On-demand import can significantly reduce project size. You need additional plugins.

### Auto Import (Recommended)

Install plugins first:

```bash
npm install -D unplugin-vue-components unplugin-auto-import
yarn add -D unplugin-vue-components unplugin-auto-import
pnpm install -D unplugin-vue-components unplugin-auto-import
```

#### Vite Configuration

In vite.config.ts:

```typescript
import { defineConfig } from 'vite'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    AutoImport({ resolvers: [ElementPlusResolver()] }),
    Components({ resolvers: [ElementPlusResolver()] }),
  ],
})
```

#### Webpack Configuration

In webpack.config.js:

```javascript
const AutoImport = require('unplugin-auto-import/webpack')
const Components = require('unplugin-vue-components/webpack')
const { ElementPlusResolver } = require('unplugin-vue-components/resolvers')

module.exports = {
  plugins: [
    AutoImport({ resolvers: [ElementPlusResolver()] }),
    Components({ resolvers: [ElementPlusResolver()] }),
  ],
}
```

### Nuxt

For Nuxt users:

```bash
npm install -D @element-plus/nuxt
```

Configure in nuxt.config.ts:

```typescript
export default defineNuxtConfig({
  modules: ['@element-plus/nuxt'],
})
```

## Global Configuration

Configure size and zIndex:

```typescript
app.use(ElementPlus, { size: 'small', zIndex: 3000 })
```

## Quick Start

After installation, use components directly:

```vue
<template>
  <el-button type="primary">Primary Button</el-button>
  <el-input v-model="input" placeholder="Enter content" />
</template>
```
