# Element Plus Manuel utilisateur

Fonctionnalités principales et conseils d'utilisation pour Element Plus.

## Démarrage rapide

### Premier exemple

```vue
<template>
  <el-button type="primary" @click="handleClick">
    Cliquez-moi
  </el-button>
</template>

<script setup>
import { ref } from 'vue'

const handleClick = () => {
  ElMessage.success('Bonjour Element Plus!')
}
</script>
```

### Form (Formulaire)

```vue
<template>
  <el-form :model="form" :rules="rules" ref="formRef" label-width="100px">
    <el-form-item label="Nom d'utilisateur" prop="username">
      <el-input v-model="form.username" />
    </el-form-item>
    
    <el-form-item label="Email" prop="email">
      <el-input v-model="form.email" />
    </el-form-item>
    
    <el-form-item label="Rôle" prop="role">
      <el-select v-model="form.role" placeholder="Sélectionner un rôle">
        <el-option label="Administrateur" value="admin" />
        <el-option label="Utilisateur" value="user" />
      </el-select>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm">Soumettre</el-button>
      <el-button @click="resetForm">Réinitialiser</el-button>
    </el-form-item>
  </el-form>
</template>

<script setup>
import { ref, reactive } from 'vue'

const formRef = ref()
const form = reactive({
  username: '',
  email: '',
  role: '',
})

const rules = {
  username: [{ required: true, message: 'Veuillez entrer le nom', trigger: 'blur' }],
  email: [
    { required: true, message: 'Veuillez entrer l\'email', trigger: 'blur' },
    { type: 'email', message: 'Veuillez entrer un email valide', trigger: 'blur' }
  ],
  role: [{ required: true, message: 'Veuillez sélectionner un rôle', trigger: 'change' }],
}

const submitForm = async () => {
  const valid = await formRef.value.validate()
  if (valid) {
    console.log('Soumis avec succès', form)
  }
}

const resetForm = () => {
  formRef.value.resetFields()
}
</script>
```

## Bonnes pratiques

### Validation de formulaire

```typescript
const rules = {
  name: [
    { required: true, message: 'Veuillez entrer le nom', trigger: 'blur' },
    { min: 2, max: 20, message: 'Longueur entre 2 et 20 caractères', trigger: 'blur' }
  ],
  email: [
    { required: true, message: 'Veuillez entrer l\'email', trigger: 'blur' },
    { type: 'email', message: 'Veuillez entrer un email valide', trigger: 'blur' }
  ],
  phone: [
    { pattern: /^1[3-9]\d{9}$/, message: 'Veuillez entrer un numéro valide', trigger: 'blur' }
  ]
}
```

### Pagination de tableau

```vue
<template>
  <el-table :data="paginatedData">
    <el-table-column prop="name" label="Nom" />
  </el-table>
  
  <el-pagination
    v-model:current-page="currentPage"
    v-model:page-size="pageSize"
    :total="total"
    :page-sizes="[10, 20, 50, 100]"
    layout="total, sizes, prev, pager, next, jumper"
    @size-change="handleSizeChange"
    @current-change="handleCurrentChange"
  />
</template>

<script setup>
import { ref, computed } from 'vue'

const currentPage = ref(1)
const pageSize = ref(10)
const tableData = ref([/* beaucoup de données */])
const total = computed(() => tableData.value.length)

const paginatedData = computed(() => {
  const start = (currentPage.value - 1) * pageSize.value
  return tableData.value.slice(start, start + pageSize.value)
})
</script>
```

### Internationalisation

```typescript
import { ref } from 'vue'
import fr from 'element-plus/es/locale/lang/fr'
import en from 'element-plus/es/locale/lang/en'

const locale = ref(fr)

const switchLanguage = (lang) => {
  locale.value = lang === 'fr' ? fr : en
}
```

## Conseils de débogage

### Vérifier les problèmes de composants

```bash
# Voir la version d'Element Plus
npm list element-plus
```

### Erreurs courantes

**Composant non enregistré**: Vérifier l'import ou la configuration d'import automatique

**Styles non appliqués**: Vérifier les conflits CSS, essayer `:deep()`

**Erreurs de type**: Installer les types `element-plus/global`
