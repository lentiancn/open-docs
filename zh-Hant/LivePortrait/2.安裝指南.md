# LivePortrait 安裝指南

本文檔詳細介紹 LivePortrait 在不同作業系統和環境下的安裝步驟。

## 環境要求

在開始安裝之前，請確保您的系統滿足以下基本要求：

**作業系統**：Linux（推薦）、Windows 10/11、macOS（需要Apple Silicon）

**Python 版本**：Python 3.10

**硬體要求**：NVIDIA GPU（推薦RTX 4090或其他支援CUDA的顯示卡），至少8GB顯示記憶體

**儲存空間**：建議預留至少20GB空間用於模型權重和執行環境

## 基礎環境配置

### 檢查CUDA版本

首先確認您的系統上安裝的CUDA版本，以便安裝對應版本的PyTorch：

```bash
nvcc -V
```

常見的CUDA版本包括11.1、11.8、12.1等。根據顯示的版本號，選擇對應的PyTorch安裝命令。

### 安裝PyTorch

根據您的CUDA版本，選擇合適的PyTorch安裝命令：

**CUDA 11.1**：

```bash
pip install torch==1.10.1+cu111 torchvision==0.11.2 torchaudio==0.10.1 -f https://download.pytorch.org/whl/cu111/torch_stable.html
```

**CUDA 11.8**：

```bash
pip install torch==2.3.0 torchvision==0.18.0 torchaudio==2.3.0 --index-url https://download.pytorch.org/whl/cu118
```

**CUDA 12.1**：

```bash
pip install torch==2.3.0 torchvision==0.18.0 torchaudio==2.3.0 --index-url https://download.pytorch.org/whl/cu121
```

如果您只需要執行人類模式（Humans mode），可以跳過CUDA版本檢查，直接安裝最新版本的PyTorch。

## 下載專案程式碼

使用Git複製LivePortrait倉庫：

```bash
git clone https://github.com/KlingTeam/LivePortrait
cd LivePortrait
```

## 建立Python環境

建議使用conda建立獨立的Python環境：

```bash
conda create -n LivePortrait python=3.10
conda activate LivePortrait
```

或者使用venv：

```bash
python -m venv LivePortrait
# Linux/macOS
source LivePortrait/bin/activate
# Windows
LivePortrait\Scripts\activate
```

## 安裝依賴

在專案根目錄下執行：

```bash
pip install -r requirements.txt
```

## 下載預訓練模型

預訓練模型可以從HuggingFace下載。如果您的網路可以存取HuggingFace：

```bash
pip install -U "huggingface_hub[cli]"
huggingface-cli download KlingTeam/LivePortrait --local-dir pretrained_weights --exclude "*.git*" "README.md" "docs"
```

如果無法存取HuggingFace，可以使用鏡像站點：

```bash
export HF_ENDPOINT=https://hf-mirror.com
huggingface-cli download KlingTeam/LivePortrait --local-dir pretrained_weights --exclude "*.git*" "README.md" "docs"
```

或者從Google Drive或百度網盤下載完整模型檔案，解壓後放入./pretrained_weights目錄。

## 驗證安裝

安裝完成後，執行測試腳本驗證環境配置是否正確：

```bash
python inference.py
```

如果一切正常，您將在animations目錄下看到生成的輸出檔案。

## 動物模型額外安裝步驟

動物模式（Animals mode）僅支援Linux和Windows系統，且需要NVIDIA GPU。

### 安裝X-Pose依賴

動物模式需要X-Pose專案作為關鍵點檢測框架。首先進入X-Pose目錄：

```bash
cd src/utils/dependencies/XPose/models/UniPose/ops
python setup.py build install
cd -
```

### 使用HuggingFace下載動物模型

```bash
huggingface-cli download KlingTeam/LivePortrait --include "animal*" --local-dir pretrained_weights
```

## Windows特別說明

在Windows系統上，某些較高版本的CUDA（如12.4、12.6）可能導致相容性問題。建議將CUDA降級到11.8版本以獲得更好的穩定性。

Windows使用者也可以選擇下載一鍵安裝包，解壓後雙擊run_windows.bat即可執行：

```bash
# 從HuggingFace下載Windows一鍵安裝包
# 下載位址：https://huggingface.co/cleardusk/LivePortrait-Windows
```

## macOS特別說明

### Apple Silicon macOS

對於搭載Apple Silicon的macOS使用者，需要使用特殊的依賴檔案：

```bash
pip install -r requirements_macOS.txt
```

執行推理時需要設定環境變數：

```bash
PYTORCH_ENABLE_MPS_FALLBACK=1 python inference.py
```

注意：在Apple Silicon macOS上執行推理可能比RTX 4090慢約20倍。

### Intel Mac

Intel晶片的Mac未經官方測試，可能無法正常執行。

### 動物模式限制

由於X-Pose依賴不支援macOS系統，動物模式在macOS上不可用，只能使用人類模式。

## 目錄結構驗證

安裝完成後，請確保目錄結構符合以下要求：

```
LivePortrait/
├── pretrained_weights/     # 預訓練模型權重目錄
├── assets/                 # 範例素材目錄
│   ├── examples/
│   │   ├── source/        # 來源圖像/影片
│   │   └── driving/       # 驅動影片
├── src/                    # 原始碼目錄
├── inference.py           # 推理腳本
├── app.py                 # Gradio介面
└── requirements.txt       # 依賴列表
```

## 常見問題

如果安裝過程中遇到問題，請檢查以下幾點：

**CUDA不可用**：確認NVIDIA驅動程式已正確安裝，執行nvidia-smi檢查驅動程式狀態

**顯示記憶體不足**：嘗試使用更小的輸入圖像或降低批次處理大小

**模型下載失敗**：檢查網路連線，或使用國內鏡像源

**依賴衝突**：建議使用全新的Python環境安裝，避免與其他專案產生衝突
