# Guide dinstallation Redis

Guide pour installer Redis sur toutes les plateformes principales.

---

## Linux

```bash
# Ubuntu/Debian
sudo apt update
sudo apt install redis-server

# CentOS/RHEL
sudo yum install epel-release
sudo yum install redis
```

---

## macOS

```bash
brew install redis
```

---

## Docker

```bash
docker run -d --name redis -p 6379:6379 redis:latest
```
