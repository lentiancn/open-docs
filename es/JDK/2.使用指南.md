# Guía de Uso de JDK

Esta guía cubre cómo usar JDK para el desarrollo de Java.

## Inicio Rápido

### Compilar y Ejecutar

```bash
# Compilar
javac HelloWorld.java

# Ejecutar
java HelloWorld
```

### Ejecución de Archivo Único (JDK 11+)

```bash
# Ejecutar directamente
java HelloWorld.java
```

## Comandos Básicos

### javac - Compilador Java

```bash
# Compilar un archivo
javac MyClass.java

# Compilar múltiples archivos
javac *.java

# Compilar con codificación específica
javac -encoding UTF-8 MyClass.java

# Compilar en directorio específico
javac -d bin MyClass.java

# Mostrar todas las advertencias
javac -Xlint MyClass.java
```

### java - Lanzador Java

```bash
# Ejecutar clase
java MyClass

# Ejecutar con argumentos
java MyClass arg1 arg2

# Establecer classpath
java -classpath "lib/*:." MyClass

# Establecer memoria
java -Xmx2g -Xms512m MyClass
```

### jar - Archivo Java

```bash
# Crear JAR
jar cf myapp.jar -C bin .

# Crear JAR con manifiesto
jar cfm myapp.jar manifest.txt -C bin .

# Extraer JAR
jar xf myapp.jar

# Listar contenido del JAR
jar tf myapp.jar
```

### javadoc - Generador de Documentación

```bash
# Generar documentación
javadoc -d docs -sourcepath src *.java

# Generar documentación con título personalizado
javadoc -d docs -header "My App" *.java
```

### jdb - Depurador Java

```bash
# Iniciar depurador
jdb MyClass

# Establecer puntos de interrupción
stop at MyClass:10

# Ejecutar
run

# Ejecutar paso a paso
next
step
```

## Opciones de JVM

### Opciones de Memoria

```bash
# Tamaño del heap
-Xmx2g        # Heap máximo
-Xms512m      # Heap inicial
-Xss256k      # Tamaño de stack

# Recolección de basura
-XX:+UseG1GC  # Recolector G1
-XX:+UseParallelGC # GC Paralelo
```

### Opciones de Rendimiento

```bash
# Compilador JIT
-XX:+TieredCompilation

# Compartir datos de clase
-Xshare:on

# Deduplicación de cadenas
-XX:+UseStringDeduplication
```

## Integración con IDE

### IntelliJ IDEA

1. File → Project Structure → SDKs
2. Añadir ruta de instalación del JDK
3. Establecer Project SDK

### VS Code

1. Instalar "Extension Pack for Java"
2. Configurar JAVA_HOME en configuración

### Eclipse

1. Window → Preferences → Java → Installed JREs
2. Añadir Standard VM
3. Establecer como predeterminado

## Módulos (Java 9+)

### Listar Módulos

```bash
java --list-modules
```

### Opciones de Módulo

```bash
# Exportar paquete
--export-module java.base/java.lang

# Abrir módulo
--open-module java.base
```

## Seguridad

### Keytool

```bash
# Generar par de claves
keytool -genkeypair -alias mykey -keyalg RSA -keystore keystore.jks

# Listar certificados
keytool -list -keystore keystore.jks

# Exportar certificado
keytool -export -alias mykey -file cert.crt -keystore keystore.jks
```

## Perfilado

### jcmd

```bash
# Listar procesos Java en ejecución
jcmd

# Obtener información del proceso
jcmd <pid> VM.flags

# Ejecutar GC
jcmd <pid> GC.run
```

### jvisualvm

```bash
# Iniciar VM visual
jvisualvm
```
