# Manuel utilisateur de Nginx

## Opérations de base

```bash
nginx              # Démarrer
nginx -s stop     # Arrêter
nginx -s reload    # Recharger la configuration
nginx -t            # Tester la configuration
```

## Configuration de base

```nginx
worker_processes auto;
events {
    worker_connections 1024;
}
http {
    server {
        listen 80;
        server_name exemple.fr;
        root /var/www/html;
        location / {
            index index.html;
        }
    }
}
```

## Fichiers statiques

```nginx
server {
    listen 80;
    root /var/www/html;
    location / {
        try_files $uri $uri/ =404;
    }
}
```

## Proxy inverse

```nginx
location / {
    proxy_pass http://backend;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
}
```

## Équilibrage de charge

```nginx
upstream backend {
    server backend1.exemple.fr;
    server backend2.exemple.fr;
}
```

## SSL/TLS

```nginx
server {
    listen 443 ssl;
    ssl_certificate certificat.crt;
    ssl_certificate_key certificat.key;
    ssl_protocols TLSv1.2 TLSv1.3;
}
```

## Directives courantes

- `root` : Racine des documents
- `index` : Fichiers d'index par défaut
- `proxy_pass` : Backend du proxy
- `upstream` : Groupe de serveurs backend
- `gzip` : Activer la compression
- `expires` : Cache du navigateur
