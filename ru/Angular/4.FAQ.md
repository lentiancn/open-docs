# FAQ Angular

> Часто задаваемые вопросы

---

## Установка

### Ошибка установки CLI?

```bash
npm cache clean --force
npm install -g @angular/cli
```

### Команда ng не найдена?

```bash
npm install -g @angular/cli
```

---

## Разработка

### Проект не запускается?

```bash
rm -rf node_modules
npm install
```
