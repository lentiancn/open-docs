# Waydroid

Waydroid — это контейнерный подход к запуску полноценной системы Android на обычной системе GNU/Linux, такой как Ubuntu.

## Обзор

Waydroid использует пространства имен Linux (user, pid, uts, net, mount, ipc) для запуска полноценной системы Android в контейнере и предоставления приложений Android на любой платформе на базе GNU/Linux.

Android внутри контейнера имеет прямой доступ к необходимому оборудованию.

Среда выполнения Android поставляется с минимальным настроенным образом системы Android на основе [LineageOS](https://lineageos.org). Используемый образ в настоящее время основан на Android 11.
