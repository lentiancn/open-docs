# Guide d'installation H2

## Installation Maven

```xml
<dependency>
    <groupId>com.h2database</groupId>
    <artifactId>h2</artifactId>
    <version>2.2.224</version>
</dependency>
```

## Démarrage rapide

```bash
java -jar h2.jar
```

Console web: http://localhost:8082

## URLs de connexion

| Mode | URL |
|------|-----|
| Mémoire | jdbc:h2:mem:testdb |
| Fichier | jdbc:h2:file:./data/testdb |
| TCP | jdbc:h2:tcp://localhost:9092/testdb |
