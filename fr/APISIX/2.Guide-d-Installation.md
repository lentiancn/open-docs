# APISIX Guide d'Installation

Plusieurs méthodes d'installation pour APISIX.

## Exigences

### Matériel

- CPU: 2 cœurs ou plus
- Mémoire: 4GB ou plus
- Disque: 40GB ou plus

### Logiciel

- OS: Linux (Ubuntu 20.04+, CentOS 7+)
- Nginx ou OpenResty
- Lua 5.1+
- etcd 3.4+

## Installation Docker (Recommandée)

### Démarrage Rapide

```bash
# Démarrer etcd
docker run -d --name etcd -p 2379:2379 apache/apisix:latest etcd

# Démarrer APISIX
docker run -d --name apisix -p 9080:9080 -p 9090:9090 apache/apisix:latest
```

### docker-compose

```yaml
version: '3'
services:
  etcd:
    image: apache/apisix:3.5.0-etcd
    ports:
      - "2379:2379"
  apisix:
    image: apache/apisix:3.5.0
    ports:
      - "9080:9080"
    depends_on:
      - etcd
```

```bash
docker-compose up -d
```

## Installation Helm (Kubernetes)

```bash
helm repo add apisix https://apache.github.io/apisix-helm-chart
helm install apisix apisix/apisix
```

## Installation depuis les Sources

### Installer les dépendances

**Ubuntu/Debian:**

```bash
wget -qO - https://openresty.org/package/pubkey.gpg | sudo apt-key add -
sudo apt-get update
sudo apt-get install -y openresty etcd
```

### Cloner les sources

```bash
git clone https://github.com/apache/apisix.git
cd apisix
git checkout 3.5.0
```

### Installer les dépendances

```bash
make deps
make init
```

### Démarrer

```bash
make run
```

## Configuration

```yaml
apisix:
  node_listen: 9080
  enable_admin: true
  admin_key:
    - name: admin
      key: 123456

etcd:
  host:
    - "http://127.0.0.1:2379"
```

## Désinstallation

```bash
docker stop apisix etcd
docker rm apisix etcd
```
