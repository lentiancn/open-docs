# Guía de Instalación de Element UI

Esta guía detalla cómo instalar y usar Element UI en varios proyectos.

## Requisitos del Entorno

Element UI tiene los siguientes requisitos:

- Vue 2.0.0 o superior
- Node.js 8.0.0 o superior
- Navegadores modernos (Chrome, Firefox, Safari, Edge 10+)

## Métodos de Instalación

### Usando el Gestor de Paquetes

Se recomienda instalar usando npm:

npm install element-ui --save

O usando yarn:

yarn add element-ui

## Importación Completa

En main.js:

import Vue from 'vue'
import Element from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(Element)

new Vue({
  el: '#app',
  render: h => h(App)
})

### Importación bajo Demanda

La importación bajo demanda puede reducir significativamente el tamaño del proyecto.

Primero instale babel-plugin-component:

npm install babel-plugin-component -D

Luego configure en .babelrc:

{
  "presets": [["env", {
    "modules": false,
    "targets": {
      "browsers": ["> 1%", "last 2 versions", "not ie <= 8"]
    }
  }]],
  "plugins": [
    ["component", {
      "libraryName": "element-ui",
      "styleLibraryName": "theme-chalk"
    }]
  ]
}

Importe componentes bajo demanda:

import { Button, Select } from 'element-ui'

Vue.component(Button.name, Button)
Vue.component(Select.name, Select)

## Configuración Global

Al registrar Element UI, puede pasar un objeto de configuración global:

import Vue from 'vue'
import Element from 'element-ui'

Vue.use(Element, {
  size: 'small',
  zIndex: 2000
})

Opciones de configuración:

- size: Establecer tamaño predeterminado para componentes de formulario (medium, small, mini)
- z-index: Establecer z-index para componentes emergentes. El valor predeterminado es 2000

Para más información, visite el sitio web oficial de Element UI.
