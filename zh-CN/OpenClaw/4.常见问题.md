# OpenClaw 常见问题

这里汇总了关于 OpenClaw 的常见问题及解答。

## 基础问题

### OpenClaw 是什么？

OpenClaw 是一个可以部署在自有设备上的工具，它能把微信、Telegram、Discord 等聊天软件和 AI 服务连接起来。简单说就是让你在常用的聊天软件里直接用上 AI 助手。

### OpenClaw 需要付费吗？

基础使用是免费的，因为它是开源项目。但运行它需要：
- 一台电脑或服务器
- AI 服务的 API 密钥（按用量收费）

### 需要购买 AI 服务商的订阅吗？

不一定。你可以选择按调用次数付费的 API 密钥，也可以使用订阅式的账号。OpenClaw 支持 Anthropic、OpenAI、Google Gemini 等多种服务商。

### 可以在哪些平台上运行？

- 电脑系统：macOS、Linux、Windows
- 手机端：iOS、Android（作为配套节点）
- 聊天软件：微信国际版、Telegram、Discord、iMessage、Signal、Slack 等 20 多种

## 安装问题

### 安装失败了怎么办？

1. 检查 Node.js 版本：`node --version`（需要 22 或更高）
2. 运行诊断：`openclaw doctor`
3. 查看日志：`openclaw logs`

### 能在树莓派上用吗？

可以。树莓派 4 或更新型号配合 Raspberry Pi OS 或 Ubuntu 都可以运行 OpenClaw。

### 需要用 Mac 电脑吗？

不需要。OpenClaw 可以在任何安装 了 Node.js 的系统上运行。只有 iMessage 这个渠道需要 Mac（因为 Apple 的限制）。

## 配置问题

### 配置文件在哪里？

默认位置是 `~/.openclaw/openclaw.json`。

### 如何设置谁能发消息给我？

通过 `dmPolicy` 参数来控制：

- `pairing`：陌生人需要配对码才能对话
- `allowlist`：只有白名单里的用户可以发消息
- `open`：所有人都可以发消息
- `disabled`：关闭私信功能

### 群聊怎么设置？

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

## 使用问题

### 如何添加新的聊天渠道？

```bash
openclaw channels login whatsapp
```

### 可以同时用好几个渠道吗？

可以。一个 Gateway 进程可以同时连接多个聊天渠道。

### AI 怎么能记住之前的对话？

OpenClaw 会自动保存对话上下文。你可以在配置里调整记忆的保留策略。

### 能同时运行多个 AI 代理吗？

可以。配置多个代理并设置不同的路由规则：

```json5
{
  agents: {
    list: [
      { id: "home", default: true },
      { id: "work" }
    ]
  }
}
```

## 安全相关

### 我的数据安全吗？

是的。OpenClaw 是部署在本地的，所有数据都保存在你的设备上，不会发送到第三方服务器（AI 服务商的 API 调用除外）。

### 怎么保护我的服务安全？

1. 通过 `allowFrom` 设置可以访问的来源
2. 使用强密码和安全的 API 密钥
3. 保持 OpenClaw 版本更新
4. 用防火墙限制开放端口

## 问题排查

### 服务启动不了

1. 检查端口是否被占用：`openclaw gateway status`
2. 查看错误日志：`openclaw logs`
3. 运行诊断：`openclaw doctor`

### AI 不回复

1. 确认 API 密钥配置正确
2. 检查 AI 服务商账号余额是否充足
3. 查看日志里的错误信息

### 消息发不出去

1. 检查渠道是否已经登录成功
2. 确认对方账号或群组是否存在
3. 查看详细的错误日志
