# JUnit Installationsanleitung

JUnit kann auf verschiedene Weise zu Ihrem Projekt hinzugefügt werden, einschließlich Maven, Gradle und direkter JAR-Download. Im Folgenden werden die Installationsmethoden für jede Version erläutert.

## JUnit 5 Installation

### Maven-Konfiguration

Fügen Sie die folgenden Abhängigkeiten in Ihrer pom.xml hinzu:

```xml
<!-- JUnit Jupiter Aggregator-Modul -->
<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter</artifactId>
    <version>5.14.3</version>
    <scope>test</scope>
</dependency>
```

Wenn Sie das Maven Surefire Plugin zum Ausführen von Tests verwenden, fügen Sie die folgende Konfiguration hinzu:

```xml
<plugin>
    <groupId>org.apache.maven.plugins</groupId>
    <artifactId>maven-surefire-plugin</artifactId>
    <version>3.5.2</version>
</plugin>
```

### Gradle-Konfiguration

Fügen Sie die Abhängigkeit in Ihrer build.gradle hinzu:

```groovy
testImplementation 'org.junit.jupiter:junit-jupiter:5.14.3'
```

So führen Sie Tests aus:

```groovy
test {
    useJUnitPlatform()
}
```

### Direkter JAR-Download

Laden Sie die erforderlichen JAR-Dateien von Maven Central herunter:

- junit-jupiter-api: JUnit Jupiter API
- junit-jupiter-engine: JUnit Jupiter Engine
- junit-platform-console-standalone: Eigenständiger Konsolen-Launcher

## JUnit 4 Installation

### Maven-Konfiguration

```xml
<dependency>
    <groupId>junit</groupId>
    <artifactId>junit</artifactId>
    <version>4.13.2</version>
    <scope>test</scope>
</dependency>
```

### Gradle-Konfiguration

```groovy
testImplementation 'junit:junit:4.13.2'
```

### Direkter JAR-Download

Laden Sie junit-4.13.2.jar von Maven Central herunter:

https://repo1.maven.org/maven2/junit/junit/4.13.2/junit-4.13.2.jar

## JUnit Vintage Installation

Um JUnit 4 Tests auf der JUnit 5 Plattform auszuführen, fügen Sie die Vintage-Engine-Abhängigkeit hinzu:

### Maven

```xml
<dependency>
    <groupId>org.junit.vintage</groupId>
    <artifactId>junit-vintage-engine</artifactId>
    <version>5.14.3</version>
    <scope>test</scope>
</dependency>
```

### Gradle

```groovy
testImplementation 'org.junit.vintage:junit-vintage-engine:5.14.3'
```

## Versionskompatibilität

- JUnit 5 erfordert Java 8 oder höher
- JUnit 4 erfordert Java 5 oder höher
- JUnit Vintage 5.14.3 erfordert JUnit 4.12 oder höher

## Installation verifizieren

Erstellen Sie eine Testklasse zur Überprüfung der Installation:

```java
import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;

class SampleTest {
    @Test
    void testExample() {
        assertEquals(2, 1 + 1);
    }
}
```

Tests ausführen:

```bash
mvn test
# oder
gradle test
```
