# MultiTalk FAQ

MultiTalk에 대한 자주 묻는 질문.

## 일반적인 질문

### MultiTalk란?

MultiTalk는 게이머와 커뮤니티를 위한 다중 파티 음성 채팅 데스크톱 애플리케이션입니다.

### MultiTalk는 무료인가요?

네, MultiTalk는 개인 및 상업적 사용에 무료입니다.

### 지원되는 플랫폼은?

- Windows 10/11
- macOS 10.15+
- Linux (Ubuntu, Fedora, Debian)

## 기술적인 질문

### 에코를 해결하는 방법은?

**해결:**
- 스피커 대신 헤드셋 사용
- 스피커 볼륨 낮추기
- 마이크를 스피커에서 멀리 놓기

### 음성이 전송되지 않는 이유는?

**해결:**
- 푸시 투 토크가 활성화되어 있는지 확인
- 마이크 선택 확인
- 음소거되어 있는지 확인

## 방 질문

### 방에 몇 명이나 참여할 수 있나요?

무료 버전: 최대 25명
Pro 버전: 최대 100명

### 방을 비밀번호로 보호할 수 있나요?

네, 방을 만들 때 비밀번호를 설정할 수 있습니다.

## 문제 해결

### 연결 오류

**해결:**
- 인터넷 연결 확인
- 다른 서버 지역 시도
- 애플리케이션 재시작
