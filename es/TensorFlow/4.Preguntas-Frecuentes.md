# TensorFlow Preguntas Frecuentes

## Preguntas Generales

### ¿Qué es TensorFlow?
TensorFlow es un framework de aprendizaje automático de código abierto desarrollado por Google. Proporciona herramientas para construir y entrenar redes neuronales, con soporte para investigación y despliegue en producción.

### ¿Qué lenguajes de programación soporta TensorFlow?
- Python (principal)
- JavaScript
- Java
- C++
- Swift
- Go

### ¿Cuál es la diferencia entre PyTorch y TensorFlow?
| Aspecto | TensorFlow | PyTorch |
|---------|------------|---------|
| Grafos Dinámicos | Limitado (TF 2.x) | Nativo |
| Depuración | Más difícil | Más fácil |
| Producción | TensorFlow Serving | TorchServe |
| Comunidad | Mayor | Creciendo rápido |

## Preguntas de Uso

### ¿Cómo construyo una red neuronal simple?
```python
import tensorflow as tf
from tensorflow import keras

# Cargar datos
mnist = keras.datasets.mnist
(x_train, y_train), (x_test, y_test) = mnist.load_data()

# Normalizar
x_train, x_test = x_train / 255.0, x_test / 255.0

# Construir modelo
model = keras.Sequential([
    keras.layers.Flatten(input_shape=(28, 28)),
    keras.layers.Dense(128, activation='relu'),
    keras.layers.Dense(10)
])

# Compilar y entrenar
model.compile(optimizer='adam',
              loss=tf.keras.losses.SparseCategoricalCrossentropy(from_logits=True),
              metrics=['accuracy'])

model.fit(x_train, y_train, epochs=5)
```

### ¿Cómo uso TensorBoard?
```python
tensorboard_callback = keras.callbacks.TensorBoard(log_dir='./logs')
model.fit(x_train, y_train, epochs=5, callbacks=[tensorboard_callback])
```

Ejecutar en terminal:
```bash
tensorboard --logdir=./logs
```

### ¿Cómo guardo y cargo modelos?
```python
# Guardar
model.save('mi_modelo.keras')

# Cargar
model = keras.models.load_model('mi_modelo.keras')
```

## Preguntas de GPU

### ¿Cómo habilito soporte GPU?
```python
gpus = tf.config.list_physical_devices('GPU')
if gpus:
    for gpu in gpus:
        tf.config.experimental.set_memory_growth(gpu, True)
```

### ¿Qué pasa si no tengo GPU?
Usa la versión CPU de TensorFlow. El entrenamiento será más lento pero funciona en cualquier máquina.

### ¿Qué versión de CUDA necesito?
TensorFlow 2.x requiere CUDA 11.0+ y cuDNN 8.0+.

## Solución de Problemas

### Errores de memoria insuficiente
- Reducir tamaño de batch
- Usar `tf.config.experimental.set_memory_growth`
- Limpiar sesión Keras con `tf.keras.backend.clear_session()`

### Errores de importación
- Verificar compatibilidad de versión de Python
- Reinstalar TensorFlow
- Verificar paquetes en conflicto

### Entrenamiento lento
- Habilitar aceleración GPU
- Usar entrenamiento de precisión mixta
- Habilitar compilación XLA
- Optimizar pipeline de datos

## Preguntas Avanzadas

### ¿Qué es Keras?
Keras es la API de alto nivel de TensorFlow para construir redes neuronales. Está incluida en TensorFlow 2.x como `tf.keras`.

### ¿Cómo uso capas personalizadas?
```python
class MiCapa(keras.layers.Layer):
    def __init__(self, units, **kwargs):
        super(MiCapa, self).__init__(**kwargs)
        self.units = units
    
    def build(self, input_shape):
        self.kernel = self.add_weight(
            name='kernel',
            shape=(input_shape[-1], self.units),
            initializer='glorot_uniform',
            trainable=True
        )
    
    def call(self, inputs):
        return tf.matmul(inputs, self.kernel)
```

### ¿Cómo uso TensorFlow Extended (TFX)?
TFX proporciona un pipeline ML de producción:
- ExampleGen: Ingesta de datos
- StatisticsGen: Estadísticas de datos
- SchemaGen: Definición de esquema
- ExampleValidator: Validación de datos
- Trainer: Entrenamiento de modelo
- Evaluator: Evaluación de modelo
- Pusher: Despliegue de modelo
