# Apifox 簡介

API 一體化協作平台。

## 什麼是 Apifox？

Apifox 是 API 一體化協作平台，集 API 文檔、API 調試、API Mock、API 自動化測試於一體。

### 主要特性

- **API 文檔** - 自動生成 API 文檔，支持團隊協作
- **API 調試** - 完整的 HTTP 請求模擬工具
- **Mock 服務** - 快速創建 Mock 服務器
- **自動化測試** - API 自動化測試功能
- **團隊協作** - 團隊項目管理、權限控制

### 統計數據

- 用戶數：數百萬開發者
