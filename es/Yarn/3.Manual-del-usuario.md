# Manual del Usuario de Yarn

## Uso Diario

### Inicializar Proyecto

```bash
yarn init -2
```

### Instalar Dependencias

```bash
yarn install
```

### Agregar Dependencias

```bash
yarn add lodash
yarn add -D typescript
```

### Remover Dependencias

```bash
yarn remove lodash
```

### Actualizar

```bash
yarn up lodash
yarn up --all
```

### Ejecutar Scripts

```bash
yarn dev
yarn build
yarn test
```
