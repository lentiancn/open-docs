# Element UI 简介

> 基于 Vue 2 的桌面端组件库

---

## 项目简介

Element UI 是一套基于 Vue 2.0 的桌面端组件库，由饿了么团队开发和维护。它提供了丰富的 PC 端常用组件，涵盖了表单、表格、对话框、导航菜单、按钮、输入框等常用 UI 元素。

Element UI 的设计风格简洁大方，文档详细完善，是国内最流行的 Vue 2 UI 组件库之一。虽然官方已经停止更新（推荐迁移到 Element Plus for Vue 3），但由于国内大量存量项目仍在使用，Element UI 仍然具有很高的参考价值和学习意义。

---

## 版本信息

| 版本 | Vue 版本 | 状态 | 说明 |
|------|----------|------|------|
| Element UI 2.x | Vue 2 | 稳定 | 当前文档主要版本 |
| Element UI 1.x | Vue 2 | 停止维护 | 已不再推荐使用 |

---

## 主要特性

- 基于 Vue 2 开发
- 提供丰富的 PC 端组件
- 支持按需引入
- 支持国际化
- 支持主题定制
- 完善的文档和示例

---

## 与 Element Plus 的区别

| 特性 | Element UI | Element Plus |
|------|------------|--------------|
| Vue 版本 | Vue 2 | Vue 3 |
| TypeScript | 部分支持 | 完全支持 |
| Composition API | 不支持 | 支持 |
| 维护状态 | 已停止 | 活跃维护 |
| 发布年份 | 2016-2021 | 2022-至今 |

---

## 适用场景

- 企业级 PC 端管理系统
- 后台管理界面
- 数据展示平台
- 表单驱动的应用

## ⚠️ 重要提示

2. **存量项目**：计划迁移到 Element Plus
