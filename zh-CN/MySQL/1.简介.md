# MySQL - 简介

MySQL 是最流行的开源关系型数据库管理系统，由瑞典 MySQL AB 公司开发，后被 Oracle 收购。MySQL 以其高性能、稳定性和易用性，广泛应用于各类 Web 应用和企业系统中。

## MySQL 的核心特性

### 1. 开源免费

MySQL 采用双授权策略，提供开源免费版本和商业版本，降低了使用成本。

### 2. 跨平台

MySQL 支持 Windows、Linux、macOS、FreeBSD 等多种操作系统。

### 3. 高性能

MySQL 采用优化查询引擎，提供高性能的数据处理能力：

```sql
-- 创建索引提升查询性能
CREATE INDEX idx_user_name ON users(name);

-- 查询优化
EXPLAIN SELECT * FROM users WHERE name = 'Tom';
```

### 4. 存储引擎

MySQL 支持多种存储引擎，每种引擎有不同的特性：

| 引擎 | 特性 |
|------|------|
| InnoDB | 支持事务、行级锁、外键（默认） |
| MyISAM | 全文索引、压缩表 |
| Memory | 内存存储、快速访问 |
| Archive | 归档存储、压缩 |

```sql
-- 指定存储引擎
CREATE TABLE t1 (id INT) ENGINE = InnoDB;
CREATE TABLE t2 (id INT) ENGINE = MyISAM;
```

### 5. 事务支持

InnoDB 存储引擎支持完整的事务特性（ACID）：

```sql
START TRANSACTION;

INSERT INTO accounts (id, balance) VALUES (1, 1000);
INSERT INTO accounts (id, balance) VALUES (2, 500);

COMMIT; -- 或 ROLLBACK;
```

### 6. 复制和高可用

MySQL 支持主从复制、组复制、MySQL Router 等高可用方案。

## MySQL 的应用场景

### 1. Web 应用

MySQL 是 LAMP/LNMP 架构的核心组件，广泛用于各类网站：

```sql
-- 典型的用户表
CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) NOT NULL UNIQUE,
    email VARCHAR(100) NOT NULL UNIQUE,
    password VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 2. 数据仓库

MySQL 可以作为数据仓库，存储和分析大量业务数据。

### 3. 内容管理系统

WordPress、Drupal 等 CMS 都使用 MySQL 作为后端数据库。

### 4. 电子商务

MySQL 用于存储商品、订单、用户等电商数据。

| MySQL 3.23 | 2000 | 初始开源版本 |
| MySQL 4.0 | 2004 | UNION、JOIN 改进 |

## MySQL 与其他数据库对比

### MySQL vs PostgreSQL

| 特性 | MySQL | PostgreSQL |
|------|-------|------------|
| 性能 | 读性能优秀 | 复杂查询更强 |
| 功能 | 较简单 | 功能丰富 |
| 事务 | 支持 | 完全支持 |
| 复制 | 主从 | 逻辑复制 |

### MySQL vs MongoDB

| 特性 | MySQL | MongoDB |
|------|-------|---------|
| 类型 | 关系型 | 文档型 |
| 架构 | 表 | 集合、文档 |
| 事务 | 支持 | 有限支持 |
