# OpenClaw FAQ

Common questions and answers about OpenClaw.

## Basics

### What is OpenClaw?

OpenClaw is a self-hosted tool that connects messaging apps (WhatsApp, Telegram, Discord, etc.) to AI assistants. It lets you use AI through chat apps you already have installed, while keeping full control over your data.

### Is OpenClaw free?

The software itself is free since it's open source. However, you need to cover:
- A computer or server to run it
- API key costs for AI services

### Do I need an AI service subscription?

Not necessarily. You can use pay-per-use API keys or subscription accounts. OpenClaw supports Anthropic, OpenAI, Google Gemini, and others.

### What platforms does it run on?

- **Desktop**: macOS, Linux, Windows
- **Mobile**: iOS, Android (as companion nodes)
- **Chat apps**: WhatsApp, Telegram, Discord, iMessage, Signal, Slack, and 20+ more

## Installation

### Installation failed. What now?

1. Check Node.js version: `node --version` (needs 22+)
2. Run diagnostics: `openclaw doctor`
3. Check logs: `openclaw logs`

### Can I run it on a Raspberry Pi?

Yes. Raspberry Pi 4 or newer with Raspberry Pi OS or Ubuntu works fine.

### Do I need a Mac?

No. OpenClaw runs on any system with Node.js. Only the iMessage channel requires a Mac due to Apple's restrictions.

## Configuration

### Where is the config file?

Default location: `~/.openclaw/openclaw.json`

### How do I control who can message me?

Use the `dmPolicy` setting:

- `pairing`: Strangers need a pairing code
- `allowlist`: Only whitelisted users can message
- `open`: Anyone can message
- `disabled`: Turn off direct messages

### How to set up group chats?

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

## Usage

### How do I add a new chat channel?

```bash
openclaw channels login whatsapp
```

### Can I use multiple channels at once?

Yes. One Gateway process connects to multiple chat platforms simultaneously.

### How does the AI remember past conversations?

OpenClaw automatically saves conversation context. You can adjust retention policies in config.

### Can I run multiple AI agents?

Yes. Configure multiple agents with different routing rules:

```json5
{
  agents: {
    list: [
      { id: "home", default: true },
      { id: "work" }
    ]
  }
}
```

## Security

### Is my data safe?

Yes. OpenClaw runs locally — all data stays on your device and isn't sent to third parties (except for AI provider API calls).

### How do I secure my installation?

1. Set `allowFrom` to restrict access sources
2. Use strong passwords and API keys
3. Keep OpenClaw updated
4. Use firewall to limit open ports

## Troubleshooting

### Gateway won't start

1. Check if port is in use: `openclaw gateway status`
2. Check error logs: `openclaw logs`
3. Run diagnostics: `openclaw doctor`

### AI doesn't respond

1. Verify API key is correct
2. Check your AI provider account has credit
3. Look for errors in logs

### Messages won't send

1. Check if channel is logged in
2. Verify recipient exists
3. Check detailed logs
