# Введение в Element UI

Element UI — это библиотека компонентов пользовательского интерфейса на Vue 2.0, предоставляющая разработчикам полные решения для настольных приложений.

## Что такое Element UI

Element UI — это библиотека компонентов Vue 2.0, разработанная и поддерживаемая командой Ele.me.

## Основные характеристики

### Полная система компонентов

Element UI предлагает более 70 высококачественных компонентов:

- Базовые компоненты: Кнопки, Иконки, Макет, Контейнеры
- Компоненты форм: Ввод, Выбор, Дата, Переключатель
- Отображение данных: Таблица, Дерево, Пагинация, Теги
- Компоненты обратной связи: Диалог, Сообщение, Загрузка
- Навигационные компоненты: Меню, Вкладки, Хлебные крошки

### Полная поддержка Vue 2.0

Element UI полностью построен на Vue 2.0.

### Импорт по требованию

С использованием babel-plugin-component.

### Поддержка интернационализации

Встроенная многоязычная поддержка.

### Настройка тем

Поддержка глобальной настройки тем.

## Принципы проектирования

- Согласованность: С реальной жизнью и интерфейсом
- Обратная связь: Управление и обратная связь страницы
- Эффективность: Упрощенные процессы
- Управляемость: Решения пользователя

## Сценарии применения

- Системы управления предприятием
- Системы электронной коммерции
- Любой проект Vue 2
