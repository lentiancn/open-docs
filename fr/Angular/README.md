# Angular

Framework moderne de développement d'applications web.

## Aperçu

Angular est un framework d'application web open source développé et maintenu par Google, offrant une solution complète pour le développement moderne d'applications web.

### Caractéristiques principales

- **TypeScript natif**: Construit avec TypeScript
- **Basé sur composants**: Maintenance et tests faciles
- **Injection de dépendances**: Système DI puissant
- **Routage**: Routage complet côté client
- **Formulaires**: Support des formulaires réactifs
- **Client HTTP**: Services HTTP complets

## Démarrage rapide

### Installer Angular CLI

```bash
npm install -g @angular/cli
```

### Créer un nouveau projet

```bash
ng new my-app
cd my-app
ng serve
```

### Accéder à l'application

Ouvrir le navigateur: http://localhost:4200

## Commandes courantes

```bash
# Créer un composant
ng g c my-component

# Créer un service
ng g s my-service

# Compiler pour la production
ng build

# Exécuter les tests
ng test
```

## Concepts clés

- **Composant**: Combinaison de vue et logique
- **Module**: Organiser le code de l'application
- **Service**: Encapsuler la logique métier
- **Routage**: Navigation entre pages
- **Directive**: Étendre HTML
- **Pipe**: Transformation de données
