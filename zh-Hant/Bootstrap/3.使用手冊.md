# Bootstrap 使用手冊

Bootstrap 核心功能和使用技巧。

## 網格系統

### 基本用法

```html
<div class="container">
  <div class="row">
    <div class="col">列</div>
  </div>
</div>
```

### 列寬設置

```html
<div class="row">
  <div class="col-4">col-4</div>
  <div class="col-4">col-4</div>
  <div class="col-4">col-4</div>
</div>
```

### 響應式斷點

| 類名 | 小螢幕 | 中螢幕 | 大螢幕 |
|------|--------|--------|--------|
| col- | <576px | ≥576px | ≥768px |
| col-sm- | - | ≥576px | ≥768px |
| col-md- | - | - | ≥768px |

## 組件

### 導航欄

```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <a class="navbar-brand" href="#">品牌</a>
  <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav">
    <span class="navbar-toggler-icon"></span>
  </button>
  <div class="collapse navbar-collapse" id="navbarNav">
    <ul class="navbar-nav">
      <li class="nav-item">
        <a class="nav-link active" href="#">首頁</a>
      </li>
    </ul>
  </div>
</nav>
```

### 卡片

```html
<div class="card" style="width: 18rem;">
  <div class="card-body">
    <h5 class="card-title">卡片標題</h5>
    <p class="card-text">卡片內容</p>
    <a href="#" class="btn btn-primary">按鈕</a>
  </div>
</div>
```

### 模態框

```html
<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#exampleModal">
  打開模態框
</button>

<div class="modal fade" id="exampleModal">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title">標題</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
      </div>
      <div class="modal-body">內容</div>
    </div>
  </div>
</div>
```

## 工具類

### 間距

```html
<div class="m-4">間距</div>
<div class="p-3">內邊距</div>
```

### 顏色

```html
<div class="text-primary">主要文字</div>
<div class="bg-dark text-white">深色背景</div>
```

## 表單

### 基本表單

```html
<form>
  <div class="mb-3">
    <label class="form-label">郵箱</label>
    <input type="email" class="form-control">
  </div>
  <button type="submit" class="btn btn-primary">提交</button>
</form>
```

2. 合理使用網格系統
3. 自定義 SASS 變量
