# Guía de Instalación de Angular

Angular es un potente framework de frontend desarrollado por Google. Esta guía cubre los métodos de instalación para diversos entornos.

## Información de Versión

| Versión | Estado | Descripción |
|---------|--------|-------------|
| Angular 18.x | ✅ Actual | Última versión |
| Angular 17.x | ✅ Soportada | Versión anterior |
| Angular 16.x | ✅ Soportada | Versión anterior |
| Angular 15.x | ✅ Soportada | Versión legacy |

## Requisitos del Sistema

### Requisitos de Hardware

| Requisito | Mínimo | Recomendado |
|-------------|---------|-------------|
| CPU | 2 núcleos | 4+ núcleos |
| RAM | 4 GB | 8+ GB |

| Software | Versión |
|----------|---------|
| Node.js | 18.19.1+ o 20.10.0+ |
| npm | 10.2.3+ |

## Métodos de Instalación

### Método 1: Angular CLI (Recomendado)

#### Instalar Angular CLI

```bash
# Instalar globalmente
npm install -g @angular/cli

# Verificar instalación
ng version
```

#### Crear Nuevo Proyecto

```bash
# Crear nuevo proyecto
ng new my-app

# Crear con opciones
ng new my-app --routing --style=scss --skip-tests
```

#### Opciones

```bash
# Omitir instalación
ng new my-app --skip-install

# Componentes standalone (Angular 14+)
ng new my-app --standalone

# Habilitar SSR
ng new my-app --ssr
```

### Método 2: Usando npx

```bash
# Crear proyecto sin instalar CLI
npx @angular/cli new my-app
```

### Método 3: Configuración Manual

```bash
# Crear directorio
mkdir my-app
cd my-app

# Inicializar npm
npm init -y

# Instalar Angular
npm install @angular/core @angular/common @angular/platform-browser @angular/platform-browser-dynamic @angular/compiler @angular/forms @angular/router @angular/animations rxjs zone.js

# Instalar dependencias de desarrollo
npm install --save-dev @angular/compiler-cli @angular-devkit/build-angular typescript
```

### Método 4: Docker

```dockerfile
# Dockerfile
FROM node:20-alpine

WORKDIR /app

# Instalar Angular CLI
RUN npm install -g @angular/cli

# Crear proyecto
RUN ng new my-app --directory .

# Iniciar servidor de desarrollo
CMD ["ng", "serve", "--host", "0.0.0.0"]
```

## Configuración del Entorno

### Gestión de Versiones de Node.js

#### Usando nvm (Recomendado)

```bash
# Instalar nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Instalar Node.js
nvm install 20
nvm use 20

# Establecer predeterminado
nvm alias default 20
```

### Configuración de npm

```bash
# Establecer registry
npm config set registry https://registry.npmjs.org/

# Habilitar SSL estricto
npm config set strict-ssl true

# Limpiar caché
npm cache clean --force
```

## Verificación

```bash
# Verificar Angular CLI
ng version

# Verificar Node.js
node --version

# Verificar npm
npm --version
```

## Problemas Comunes

### ¿Permiso Denegado?

```bash
# Arreglar permisos de npm
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
export PATH=~/.npm-global/bin:$PATH
```

### ¿Instalación Lenta?

```bash
# Usar npm ci para instalación más rápida
npm ci

# O usar yarn
npm install -g yarn
```

## Actualizando Angular

```bash
# Actualizar Angular CLI
npm install -g @angular/cli@latest

# Actualizar tu proyecto
ng update @angular/core @angular/cli
```
