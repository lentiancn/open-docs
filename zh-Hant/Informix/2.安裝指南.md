# Informix 安裝指南

## Linux 安裝步驟

以下步驟在 CentOS 7/8 環境下測試有效，適用於 Informix 11.5 和 12.1 版本。

### 步驟 1：建立使用者群組

必須在 root 權限下執行以下命令：

```bash
su root

# 建立群組
groupadd -g 1000 informix

# 建立使用者
useradd -u 1000 -g 1000 -d /home/informix -m informix

# 設定密碼
passwd informix
```

### 步驟 2：建立安裝目錄

```bash
mkdir /opt/informix
chown informix.informix /opt/informix
```

### 步驟 3：上傳並解壓縮安裝包

將 Informix 安裝包上傳到伺服器，然後解壓縮：

```bash
cd /tmp/informix
tar -xvf iif.11.50.UC4E.Linux-RHEL4.tar
```

### 步驟 4：執行安裝

```bash
./ids_install
```

安裝過程中：
- 輸入 `1` 表示同意授權合約
- 選擇安裝路徑為 `/opt/informix`
- 根據需要選擇安裝元件

### 步驟 5：設定環境變數

以 informix 使用者身份登入，編輯 `~/.bash_profile`：

```bash
su - informix
vim ~/.bash_profile
```

新增以下環境變數：

```bash
export INFORMIXDIR=/opt/informix
export INFORMIXSERVER=ol_informix1210
export ONCONFIG=onconfig.ol_informix1210
export INFORMIXSQLHOSTS=/opt/informix/etc/sqlhosts.ol_informix1210
export PATH=$PATH:$INFORMIXDIR/bin
```

使設定生效：

```bash
source ~/.bash_profile
```

### 步驟 6：設定 sqlhosts 檔案

```bash
cd /opt/informix/etc
cp sqlhosts.std sqlhosts
vim sqlhosts
```

新增以下內容：

```
ol_informix1210 onsoctcp localhost ol_informix1210
dr_informix1210 drsoctcp localhost dr_informix1210
lo_informix1210 onsoctcp localhost lo_informix1210
```

**注意**：如果需要遠端存取，將 `localhost` 改為 `*`。

### 步驟 7：設定 onconfig 檔案

```bash
cd /opt/informix/etc
cp onconfig.std onconfig.ol_informix1210
vim onconfig.ol_informix1210
```

修改以下關鍵參數：

```
SERVERNUM 49
FULL_DISK_INIT 1
NETTYPE soctcp,1,50,CPU
```

### 步驟 8：初始化資料庫

首次啟動需要使用 `-ivy` 參數進行初始化：

```bash
oninit -ivy
```

驗證初始化是否成功：

```bash
# 查看資料檔案大小
ls -la /opt/informix/storage/
# 如果 rootdbs 檔案大小不為 0，表示初始化成功

# 查看資料庫狀態
onstat -
```

### 步驟 9：常用管理命令

```bash
# 關閉資料庫
onmode -ky

# 查看服務狀態
onstat -

# 查看邏輯日誌狀態
onstat -l

# 連線資料庫（互動模式）
onstat -i

# 查看連接埠
netstat -apn | grep oninit
```

## Windows 安裝概述

Windows 平台的安裝相對簡單：

1. 從 IBM 官網下載 Informix 安裝包（.exe 或 .zip 格式）
2. 執行安裝程式 `ids_install.exe`
3. 按照安裝精靈選擇安裝路徑、元件等
4. 設定環境變數（透過系統屬性 → 進階 → 環境變數）
5. 使用圖形化工具 `oninit` 或 `DB-Access` 進行初始化

## 安裝後驗證

安裝完成後，可以透過以下方式驗證：

```bash
# 檢查 Informix 程式是否執行
ps -ef | grep oninit

# 查看版本資訊
onstat -

# 連線測試
dbaccess -version
```

## 注意事項

- 確保系統滿足硬體和軟體需求
- 防火牆需要開放 Informix 使用的連接埠（預設 9088）
- 建議使用專用使用者執行 Informix，避免使用 root 使用者
- 生產環境請仔細規劃儲存佈局
