# Введение в Nginx

Nginx - высокопроизводительный HTTP веб-сервер, обратный прокси, балансировщик нагрузки и почтовый прокси.

## Основные функции
- HTTP веб-сервер
- Обратный прокси
- Балансировка нагрузки
- Кэширование
- SSL/TLS

## Преимущества
- Высокая производительность
- Низкое потребление ресурсов
- Высокая стабильность
