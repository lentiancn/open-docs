# JDK インストールガイド

JDK（Java Development Kit）は、Oracle公式のJava開発キットです。本ガイドでは、さまざまな環境でのインストール方法を説明します。

## バージョン情報

| バージョン | ステータス | 説明 |
|---------|--------|-------------|
| JDK 21 (LTS) | ✅ 現行 | 最新のLTSバージョン |
| JDK 17 (LTS) | ✅ サポート済み | 前のLTSバージョン |
| JDK 11 (LTS) | ✅ サポート済み | レガシーLTS |
| JDK 8 | ✅ サポート済み | レガシーバージョン |
| JDK 22 | ✅ 最新 | 最新の機能リリース |

### LTSと機能リリース

- **LTS（長期サポート）**：2〜3年の更新があり、本番環境におすすめ
- **機能リリース**：6ヶ月ごとに新機能リリース、サポート期間は短い

## システム要件

## インストール方法

### 方法1：Oracle JDK

#### Windows

1. 以下からダウンロード：https://www.oracle.com/java/technologies/downloads/
2. インストーラー（.exe）を実行
3. ウィザードに従う

#### macOS

```bash
# OracleウェブサイトからDMGをダウンロード
# またはHomebrewを使用
brew install openjdk@21
```

#### Linux

```bash
# tar.gzをダウンロード
wget https://download.oracle.com/java/21/latest/jdk-21_linux-x64_bin.tar.gz
tar -xzf jdk-21_linux-x64_bin.tar.gz
sudo mv jdk-21 /opt/jdk-21
```

### 方法2：Eclipse Temurin（Adoptium）

#### Windows

1. 以下からダウンロード：https://adoptium.net/temurin/releases/
2. インストーラーを実行

#### macOS

```bash
# Homebrewを使用
brew install --cask temurin

# またはウェブサイトからpkgをダウンロード
```

#### Linux

```bash
# Ubuntu/Debian
wget https://github.com/adoptium/temurin21-binaries/releases/download/jdk-21.0.2%2B13/OpenJDK21U-jdk_x64_linux_hotspot_21.0.2_13.tar.gz
tar -xzf OpenJDK21U-jdk_x64_linux_hotspot_21.0.2_13.tar.gz

# 環境変数を設定
export JAVA_HOME=/path/to/jdk-21
export PATH=$JAVA_HOME/bin:$PATH
```

### 方法3：SDKMAN

```bash
# SDKMANをインストール
curl -s "https://get.sdkman.io" | bash

# JDKをインストール
sdk install java 21.0.2-tem

# 利用可能なバージョンを一覧表示
sdk list java
```

### 方法4：Docker

```dockerfile
FROM eclipse-temurin:21-jdk
WORKDIR /app
COPY . .
RUN javac Main.java
CMD ["java", "Main"]
```

### 方法5：Windowsパッケージマネージャー

#### Chocolatey

```powershell
choco install openjdk21
```

#### Scoop

```powershell
scoop install openjdk
```

## 環境設定

### Windows

```powershell
# JAVA_HOMEを設定
[System.Environment]::SetEnvironmentVariable("JAVA_HOME", "C:\Program Files\Java\jdk-21", "User")

# PATHに追加
[System.Environment]::SetEnvironmentVariable("PATH", "$env:PATH;C:\Program Files\Java\jdk-21\bin", "User")
```

### macOS/Linux

```bash
# ~/.bashrc または ~/.zshrc に追加
export JAVA_HOME=/opt/jdk-21
export PATH=$JAVA_HOME/bin:$PATH

# 適用
source ~/.bashrc
```

## 確認

```bash
# Javaバージョンを確認
java -version

# Javacバージョンを確認
javac -version

# JAVA_HOMEを確認
echo $JAVA_HOME
```

## 複数のJDKバージョン

### SDKMANを使用

```bash
# 複数のバージョンをインストール
sdk install java 17.0.8-tem
sdk install java 21.0.2-tem

# バージョンを切り替え
sdk use java 17.0.8-tem
```

## よくある問題

### "java" が見つからない？

PATHにJavaのbinディレクトリが含まれているか確認してください。

### Javaのバージョンが違う？

使用中のJavaを確認：

```bash
which java
java -version
```

### JAVA_HOMEが設定されていない？

JAVA_HOMEをJDKインストールディレクトリに設定してください。
