# OpenCode

OpenCode is a modern, powerful code editor for developers.

## Overview

OpenCode provides a fast, feature-rich editing experience with:
- Syntax highlighting
- Integrated terminal
- Git integration
- Extensions support

## Documentation

- [Installation Guide](./1.安装文档.md) - Install on all platforms
- [Usage Guide](./2.使用指南.md) - Learn editor features

## Quick Start

### Install
```bash
# Windows
winget install OpenCode

# macOS
brew install opencode

# Linux
sudo apt install opencode
```

### Create Project
```bash
mkdir myproject
cd myproject
code .
```

## Features

- Multi-language support
- Intelligent code completion
- Integrated debugging
- Customizable themes
- Extensive extension marketplace
