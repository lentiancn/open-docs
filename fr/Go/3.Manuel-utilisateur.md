## Syntaxe de base

### Hello World

```go
package main

import "fmt"

func main() {
    fmt.Println("Bonjour, le monde!")
}
```

## Fonctions

```go
func ajouter(a, b int) int {
    return a + b
}

func diviser(a, b int) (int, error) {
    if b == 0 {
        return 0, errors.New("division par zéro")
    }
    return a / b, nil
}
```

## Concurrence

### Goroutine

```go
func dire(s string) {
    for i := 0; i < 3; i++ {
        fmt.Println(s)
    }
}

func main() {
    go dire("Bonjour")
    dire("Monde")
    time.Sleep(time.Second)
}
```

### Channel

```go
ch := make(chan string)
go func() { ch <- "message" }()
msg := <-ch
```

## Gestion des erreurs

```go
resultat, err := diviser(10, 0)
if err != nil {
    fmt.Println("Erreur:", err)
}
```
