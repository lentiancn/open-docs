# HeyGen Руководство по установке

HeyGen — это платформа для создания видео с использованием искусственного интеллекта, которая позволяет пользователям создавать реалистичных ИИ-аватаров и генерировать видео с говорящей головой из текста или аудио.

## Обзор платформы

HeyGen — это в основном облачная SaaS-платформа. В отличие от проектов с открытым исходным кодом, HeyGen не требует локальной установки — пользователи получают доступ через веб-интерфейс или API.

### Способы доступа

| Способ | Описание | Требования |
|--------|----------|------------|
| Веб-интерфейс | Создание видео в браузере | Учетная запись + Интернет |
| API | Программный доступ | API-ключ |
| SDK | Локальная разработка | Учетная запись разработчика |

## Обзор версий

HeyGen — это облачный сервис, а не локальная установка:

| Компонент | Версия | Примечания |
|-----------|--------|------------|
| HeyGen Web | Последняя | Всегда актуальна |
| HeyGen API | v1/v2 | REST API |
| HeyGen SDK | Последняя | Для разработчиков |

**Примечание:** HeyGen — коммерческий продукт. Некоторые функции требуют платной подписки.

## Доступ к HeyGen

### Шаг 1: Создание учетной записи

1. Посетите https://www.heygen.com
2. Нажмите «Регистрация» или «Начать»
3. Подтвердите адрес электронной почты

### Шаг 2: Выбор тарифа

| План | Функции | Цена |
|------|---------|------|
| Бесплатный | Ограниченные минуты, водяной знак | Бесплатно |
| Pro | Больше минут, без водяного знака | $29/мес |
| Enterprise | Пользовательские лимиты, доступ к API | Связаться с отделом продаж |

### Шаг 3: Получение API-ключа (для разработчиков)

1. Войдите в панель управления HeyGen
2. Перейдите в Настройки → API
3. Сгенерируйте API-ключ

## Установка API (для разработчиков)

### Python SDK

```bash
# Установить HeyGen SDK
pip install heygen-python

# Или установить из исходного кода
pip install git+https://github.com/heygen-ai/heygen-python.git
```

### Node.js SDK

```bash
# Установить HeyGen SDK
npm install heygen-node

# Или установить из исходного кода
npm install github:heygen-ai/heygen-node
```

### Проверка установки

```python
# Python
import heygen
print(heygen.__version__)
```

```javascript
// Node.js
const heygen = require('heygen-node');
console.log(heygen.version);
```

## Настройка API

### Пример Python

```python
from heygen import HeyGen

# Инициализация с API-ключом
heygen = HeyGen(api_key="your_api_key")

# Получить информацию об учетной записи
account = heygen.get_account_info()
print(account)
```

### Пример Node.js

```javascript
const { HeyGen } = require('heygen-node');

const heygen = new HeyGen({ apiKey: 'your_api_key' });

// Получить информацию об учетной записи
const account = await heygen.getAccountInfo();
console.log(account);
```

## Переменные окружения

### Установка API-ключа

```bash
# Linux/macOS
export HEYGEN_API_KEY="your_api_key"

# Windows (PowerShell)
$env:HEYGEN_API_KEY="your_api_key"

# В файле .env
HEYGEN_API_KEY=your_api_key
```

### Python с dotenv

```python
from dotenv import load_dotenv
load_dotenv()

import heygen
heygen = HeyGen(api_key=os.getenv("HEYGEN_API_KEY"))
```

## Частые проблемы

### Ошибки аутентификации

**Проблема:** «Неверный API-ключ»

**Решения:**
1. Проверьте правильность API-ключа
2. Убедитесь, что API-ключ активен
3. Проверьте наличие доступа к API у учетной записи

### Ограничения скорости

**Проблема:** «Слишком много запросов»

**Решения:**
1. Проверьте лимиты использования API
2. Перейдите на более высокий тариф
3. Реализуйте очередь запросов

### Ошибка генерации видео

**Проблема:** «Ошибка генерации»

**Решения:**
1. Проверьте входные параметры
2. Проверьте баланс кредитов
3. Попробуйте позже

## Рекомендации по безопасности

1. **Никогда не раскрывайте API-ключи** — используйте переменные окружения
2. **Регулярно меняйте ключи** — периодически генерируйте новые ключи
3. **Используйте минимальные разрешения** — запрашивайте только необходимые области
4. **Отслеживайте использование** — проверяйте панель управления на наличие аномалий
