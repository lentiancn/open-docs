# Oracle Database FAQ

## General Questions

### What is Oracle Database?

Oracle Database is a proprietary relational database management system (RDBMS) developed by Oracle Corporation. It is known for its enterprise-grade features, reliability, and scalability.

### What is the difference between Oracle DB and MySQL?

Oracle Database is an enterprise-grade RDBMS with advanced features, while MySQL is an open-source database more suited for web applications. Oracle offers more advanced security, partitioning, and high availability features.

### What is PL/SQL?

PL/SQL (Procedural Language/SQL) is Oracle's procedural language extension to SQL. It allows developers to write complex business logic using stored procedures, functions, and triggers.

## Installation Questions

### What are the minimum system requirements?

- **Memory**: 2GB RAM minimum
- **Disk Space**: 8GB for software, more for data
- **OS**: Linux, Windows Server, Solaris, AIX
- **CPU**: 2+ cores recommended

### How to install Oracle Database on Linux?

1. Create oracle user and groups
2. Configure kernel parameters
3. Set up environment variables
4. Run the Oracle installer
5. Create a database using DBCA

### Oracle Installer hangs at 73%?

This is often due to memory or resource issues. Try:
- Increase system memory
- Disable antivirus software
- Run installer with `-ignoreSysPrereqs` flag

## Connection Questions

### ORA-12541: TNS:no listener?

Start the Oracle listener:
```bash
lsnrctl start
```

### ORA-01017: invalid username/password?

Verify credentials and ensure the user exists in the database:
```sql
SELECT username FROM dba_users;
```

### How to connect to Oracle from remote?

1. Configure TNS listener on server
2. Open port 1521 in firewall
3. Add entry in tnsnames.ora on client

## Performance Questions

### How to check query performance?

Use `EXPLAIN PLAN` or SQL Tuning Advisor:
```sql
EXPLAIN PLAN FOR your_query;
SELECT * FROM TABLE(DBMS_XPLAN.DISPLAY);
```

### When to create indexes?

Create indexes when:
- Columns are frequently queried
- High selectivity (few rows returned)
- JOIN operations are common

Avoid indexes for:
- Small tables
- Frequently updated columns
- Low selectivity columns

### What is SGA and PGA?

- **SGA (System Global Area)**: Shared memory area for database buffer cache, shared pool, redo log buffer
- **PGA (Program Global Area)**: Private memory for each server process

## Backup & Recovery Questions

### How to backup Oracle Database?

Use RMAN (Recovery Manager):
```bash
rman target /
BACKUP DATABASE;
```

### How to recover from data loss?

```bash
# Restore from backup
RESTORE DATABASE;

# Apply redo logs
RECOVER DATABASE;
```

## Common Error Codes

| Error Code | Description | Solution |
|------------|-------------|----------|
| ORA-00001 | Unique constraint violated | Check for duplicate values |
| ORA-01000 | Maximum open cursors exceeded | Close unused cursors |
| ORA-01017 | Invalid username/password | Verify credentials |
| ORA-12541 | TNS:no listener | Start listener |
| ORA-12560 | TNS:protocol adapter error | Check Oracle service |
| ORA-30036 | Snapshot too old | Increase undo retention |
