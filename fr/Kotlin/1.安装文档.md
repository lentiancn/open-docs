# Guide d'installation de Kotlin

Kotlin est un langage de programmation moderne développé par JetBrains. Ce guide couvre les méthodes d'installation dans divers environnements.

## Informations sur les versions

| Version | Statut | Description |
|---------|--------|-------------|
| Kotlin 2.x | ✅ Actuelle | Dernière version |
| Kotlin 1.9.x | ✅ Supportée | Version stable |

## Configuration système

### Configuration matérielle

| Exigence | Configuration |
|----------|---------------|
| CPU | 1+ cœurs |
| Mémoire | 2+ Go |
| Disque | 500+ Mo |

### Configuration logicielle

| Logiciel | Version |
|----------|---------|
| Java | JDK 8 ou supérieur |

## Méthodes d'installation

### Méthode 1 : IntelliJ IDEA (Recommandé)

1. Téléchargez IntelliJ IDEA : https://www.jetbrains.com/idea/download/
2. Installez la version Ultimate ou Community
3. Kotlin est intégré par défaut

### Méthode 2 : Ligne de commande

#### Windows

```powershell
# Utiliser Chocolatey
choco install kotlin

# Ou télécharger manuellement
# Téléchargez depuis https://github.com/JetBrains/kotlin/releases
```

#### macOS

```bash
# Utiliser Homebrew
brew install kotlin

# Ou installation manuelle
curl -s https://get.sdkman.io | bash
sdk install kotlin
```

#### Linux

```bash
# Ubuntu/Debian
sudo apt install kotlin

# Ou utiliser SDKMAN
curl -s https://get.sdkman.io | bash
sdk install kotlin
```

### Méthode 3 : SDKMAN

```bash
# Installer SDKMAN
curl -s "https://get.sdkman.io" | bash

# Installer Kotlin
sdk install kotlin

# Vérifier
kotlin -version
```

### Méthode 4 : Maven

```xml
<dependencies>
    <dependency>
        <groupId>org.jetbrains.kotlin</groupId>
        <artifactId>kotlin-stdlib</artifactId>
        <version>2.0.0</version>
    </dependency>
</dependencies>
```

### Méthode 5 : Gradle

```groovy
plugins {
    id 'org.jetbrains.kotlin.jvm' version '2.0.0'
}

repositories {
    mavenCentral()
}

dependencies {
    implementation "org.jetbrains.kotlin:kotlin-stdlib"
}
```

### Méthode 6 : Docker

```dockerfile
# Utiliser l'image du compilateur Kotlin
FROM gradle:8.5-jdk-21 AS builder
WORKDIR /app
COPY . .
RUN gradle build

# Image d'exécution
FROM openjdk:21
COPY --from=builder /app/build/libs/*.jar /app.jar
ENTRYPOINT ["java", "-jar", "/app.jar"]
```

## Configuration de l'environnement

### Configuration Java

```bash
# Vérifier Java
java -version

# Configurer JAVA_HOME
export JAVA_HOME=/path/to/java
```

### Compilation et exécution

```bash
# Compiler
kotlinc hello.kt -include-runtime -d hello.jar

# Exécuter
java -jar hello.jar

# Utiliser le script kotlin
kotlin hello.kts
```

## Configuration IDE

### IntelliJ IDEA

1. Installer le plugin Kotlin (généralement préinstallé)
2. Créer un projet Kotlin
3. Configurer le SDK Kotlin

### VS Code

1. Installer l'extension Kotlin Language Server
2. Installer le compilateur Kotlin

### Eclipse

1. Installer le plugin Kotlin (Kotlin IDE for Eclipse)
2. Créer un projet Kotlin

## Problèmes courants

### Erreurs de compilation ?

Vérifiez la compatibilité des versions de Kotlin et Java.

### Mémoire insuffisante ?

Augmentez la mémoire heap JVM :

```bash
kotlinc -J-Xmx2g
```
