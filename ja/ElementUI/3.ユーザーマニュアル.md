# Element UI ユーザーマニュアル

> コンポーネントの使用

---

## ボタン

```html
<el-button>デフォルト</el-button>
<el-button type="primary">プライマリ</el-button>
```

## フォーム

```html
<el-form :model="form">
  <el-form-item>
    <el-input v-model="form.name"></el-input>
  </el-form-item>
</el-form>
```

## テーブル

```html
<el-table :data="tableData">
  <el-table-column prop="name" label="名前"></el-table-column>
</el-table>
```

## ダイアログ

```html
<el-dialog :visible.sync="dialogVisible" title="タイトル">
  <span>内容</span>
</el-dialog>
```
