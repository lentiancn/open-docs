# Guía de Instalación de MultiTalk

MultiTalk es una herramienta de generación de audio multiorador que crea contenido de audio multivocal natural.

## Información de Versión

| Versión | Estado | Descripción |
|---------|--------|-------------|
| MultiTalk v1.0 | ✅ Actual | Versión inicial |

## Requisitos del Sistema

### Requisitos de Hardware

| Requisito | Mínimo | Recomendado |
|-------------|---------|-------------|
| GPU | NVIDIA 4GB VRAM | NVIDIA 8GB+ VRAM |
| RAM | 8GB | 16GB+ |
| Almacenamiento | 5GB | 10GB+ SSD |

### Requisitos de Software

| Software | Versión |
|----------|---------|
| Python | 3.8+ |
| CUDA | 11.0+ |

## Métodos de Instalación

### Método 1: Desde el Código Fuente

#### 1. Clonar el Repositorio

```bash
git clone https://github.com/MultiTalk/MultiTalk.git
cd MultiTalk
```

#### 2. Crear Entorno Virtual

```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
# o
venv\Scripts\activate  # Windows
```

#### 3. Instalar Dependencias

```bash
pip install -r requirements.txt
```

#### 4. Descargar Modelos

```bash
bash scripts/download_models.sh
```

### Método 2: Docker

```bash
docker pull multitalk/multitalk:latest
docker run --gpus all -v /path/to/data:/data multitalk/multitalk:latest
```

## Configuración

### Variables de Entorno

```bash
export CUDA_VISIBLE_DEVICES=0
export PYTHONPATH=/path/to/MultiTalk:$PYTHONPATH
```

### Rutas de Modelos

Editar `config.yaml`:

```yaml
model:
  checkpoint_path: "checkpoints/multitalk.pth"
```

## Inicio Rápido

### Generar Audio Multiorador

```bash
python inference.py \
  --text "Hola, esto es una demostración multiorador." \
  --speakers speaker1,speaker2 \
  --output output.wav
```

### Parámetros

```bash
--text         # Texto de entrada
--speakers     # IDs de orador (separados por comas)
--output       # Ruta del archivo de salida
--speed        # Velocidad del habla (predeterminado: 1.0)
```

## Problemas Comunes

### ¿GPU No Disponible?

```bash
nvidia-smi
python -c "import torch; print(torch.cuda.is_available())"
```

### ¿Error al Descargar el Modelo?

Verifica la conexión a Internet o descarga los modelos manualmente.
