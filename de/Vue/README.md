# Vue

Vue ist ein progressives JavaScript-Framework zum Erstellen von Benutzeroberflächen.

## Überblick

Vue ist inkrementell adoptierbar - von einfachen Komponenten bis zu komplexen SPAs.

### Hauptversionen

| Version | Status | Veröffentlichung |
|---------|--------|------------------|
| Vue 3 | Empfohlen | September 2020 |
| Vue 2 | Veraltet | Oktober 2016 |

## Schnellstart

```bash
npm create vue@latest
# Oder
npm create vite@latest my-app -- --template vue
cd my-app
npm install
npm run dev
```

## Funktionen

- **Reaktiv** - Automatische Abhängigkeitsverfolgung
- **Komponentenbasiert** - Wiederverwendbare UI-Blöcke
- **Direktiven** - Template-Syntax-Erweiterungen
- **Übergänge** - Eingebaute Animation
- **Router** - SPA-Routing
- **State Management** - Pinia

## Vue 2 vs Vue 3

| Feature | Vue 2 | Vue 3 |
|---------|-------|-------|
| API | Options | Composition + Options |
| Reaktivität | defineProperty | Proxy |
| TypeScript | Teilweise | Voll |
| Leistung | Gut | Besser |
