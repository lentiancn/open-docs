# Bootstrap 5

Bootstrap es el kit de herramientas de código abierto más popular del mundo para construir sitios web responsivos y móviles.

## Introducción

Bootstrap 5 es la última versión, completamente reescrita en JavaScript nativo sin dependencia de jQuery. Proporciona ricos componentes CSS y JavaScript para ayudar a los desarrolladores a construir rápidamente sitios web modernos.

## Características Principales

- **Diseño Responsivo**: Sistema de grid responsivo incorporado
- **Ricos Componentes**: Botones, navegación, formularios, tarjetas, modales, etc.
- **Potentes Clases de Utilidad**: Espaciado, colores, texto, visualización
- **Soporte Sass**: Personalización de temas con Sass
- **Biblioteca de Iconos**: Integración con Bootstrap Icons
- **Sin jQuery**: Completamente escrito en JavaScript nativo

## Inicio Rápido

### Importar desde CDN

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
```

### Instalar con npm

```bash
npm install bootstrap@5.3.3
```
