# Häufig gestellte Fragen zu OpenClaw

Diese Seite fasst die häufigsten Fragen und Antworten zur Verwendung von OpenClaw zusammen.

---

## Grundlegende Fragen

### Was ist OpenClaw?

OpenClaw ist ein selbst gehostetes KI-Assistenten-Gateway, das mehrere Chat-Plattformen wie WhatsApp, Telegram, Discord und iMessage mit einem KI-Coding-Assistenten (wie Pi) verbindet.

### Ist OpenClaw kostenlos?

OpenClaw selbst ist Open Source und kostenlos (MIT-Lizenz). Die Verwendung von KI-Modellen kann jedoch Kosten verursachen.

### Welche Hardware wird benötigt?

Mindestanforderungen:
- Node.js 22+
- 2GB RAM
- 500MB Speicherplatz

Empfohlen:
- 4GB+ RAM
- Stabile Netzwerkverbindung
- Immer eingeschalteter PC oder Server

## Installationsfragen

### Installation fehlgeschlagen — was tun?

1. Node.js-Version prüfen:
   ```bash
   node --version
   ```

2. Offizielles Installationsskript verwenden:
   ```bash
   curl -fsSL https://openclaw.ai/install.sh | bash
   ```

3. Diagnose ausführen:
   ```bash
   openclaw doctor
   ```

### Probleme unter Windows?

**WSL2 wird stark empfohlen.** OpenClaw funktioniert am besten in einer Linux-Umgebung.

### OpenClaw aktualisieren?

```bash
npm update -g openclaw@latest
openclaw doctor
```

## Konfigurationsfragen

### Wo ist die Konfigurationsdatei?

Standardpfad: `~/.openclaw/openclaw.json`

### Mehrere Kanäle konfigurieren?

```json5
{
  channels: {
    whatsapp: { enabled: true },
    telegram: { enabled: true, botToken: "xxx" },
    discord: { enabled: true, token: "xxx" },
  },
}
```

### Konfiguration wird nicht angewendet?

1. Format prüfen (JSON5)
2. Gateway neu starten:
   ```bash
   openclaw gateway restart
   ```

## Verbindungsfragen

### Gateway startet nicht?

Mögliche Gründe:
- Port 18789 belegt
- Konfigurationsfehler
- Unzureichende Berechtigungen

Lösung:
```bash
lsof -i :18789
openclaw doctor
```

### Kanalverbindung fehlgeschlagen?

1. Anmeldedaten prüfen
2. Kanalstatus prüfen:
   ```bash
   openclaw channels status --probe
   ```
3. Logs prüfen:
   ```bash
   openclaw logs --follow
   ```

## Nutzungsfragen

### KI antwortet nicht?

Prüfen Sie:
1. Kanal verbunden?
2. In Whitelist?
3. In Gruppen @-Erwähnung erforderlich?
4. Logs auf Fehler prüfen

### Session zurücksetzen?

```bash
openclaw session reset
```

### KI-Modell ändern?

```json5
{
  agents: {
    defaults: {
      model: {
        primary: "anthropic/claude-sonnet-4-5",
      },
    },
  },
}
```

## Sicherheitsfragen

### Ist OpenClaw sicher?

OpenClaw prioriert Sicherheit:
- Alle Daten lokal gespeichert
- Anmeldedaten verschlüsselt
- Whitelist und Pairing-Mechanismen
- Sandbox-Isolation möglich

Empfehlungen:
- Whitelist zur Zugriffskontrolle
- Regelmäßige Updates
- `openclaw doctor` ausführen

### Was ist Pairing?

Pairing ist ein Sicherheitsmechanismus, bei dem unbekannte Benutzer beim ersten Nachricht blockiert werden und Ihre Genehmigung benötigen.
