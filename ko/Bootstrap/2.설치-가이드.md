# 설치 가이드

이 문서에서는 Bootstrap의 다양한 설치 방법을 소개하여, 빠르게 프로젝트에서 Bootstrap을 사용하기 시작할 수 있도록 도와줍니다.

## 개발 환경 요구사항

Bootstrap 5는 다음の開発 환경이 필요합니다:

- **HTML5** 모던 브라우저
- **CSS3** 지원
- **JavaScript** ES6+(JavaScript 플러그인을 사용하는 경우)

## 설치 방법

### 방법 1: CDN 불러오기(추천)

가장 간단한 방법은 CDN을 통해 직접 Bootstrap의 CSS와 JavaScript 파일을 불러오는 것입니다:

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 페이지</title>
    <!-- Bootstrap CSS 불러오기 -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>안녕하세요, Bootstrap!</h1>

    <!-- Bootstrap JavaScript 불러오기(Popper 포함) -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### 방법 2: npm 설치

npm으로 프로젝트 의존성을 관리하는 경우:

```bash
# Bootstrap 설치
npm install bootstrap@5

# Popper 설치(드롭다운 메뉴, 툴팁 등에 사용)
npm install @popperjs/core
```

그 다음 프로젝트에서 불러옵니다:

```javascript
// Bootstrap의 CSS 불러오기
import 'bootstrap/dist/css/bootstrap.min.css';

// Bootstrap의 JS 불러오기
import 'bootstrap/dist/js/bootstrap.bundle.min.js';
```

### 방법 3: 소스 코드 다운로드

공식 웹사이트에서 미리 컴파일된 파일을 다운로드합니다:

1. https://getbootstrap.com/ 방문
2. "Download" 버튼 클릭
3. "Compiled CSS and JS" 선택하여 다운로드
4. 압축 해제 후 css와 js 폴더를 프로젝트에 복사

디렉터리 구조:

```
your-project/
├── css/
│   └── bootstrap.min.css
├── js/
│   └── bootstrap.bundle.min.js
└── index.html
```

### 방법 4: Sass 소스 코드

Bootstrap 스타일을 직접 커스텀해야 하는 경우 Sass 소스 코드를 다운로드할 수 있습니다:

```bash
npm install bootstrap
```

그 다음 Sass 파일에서 불러옵니다:

```scss
// 커스텀 변수(Bootstrap을 불러오기 전에 정의)
$primary: #007bff;
$body-bg: #f8f9fa;

// Bootstrap 불러오기
@import "~bootstrap/scss/bootstrap";
```

## 빠르게 시작하기

### 첫 번째 페이지 만들기

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 빠르게 시작하기</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <!-- 네비게이션 바 -->
    <nav class="navbar navbar-expand-lg navbar-dark bg-primary">
        <div class="container">
            <a class="navbar-brand" href="#">내 웹사이트</a>
        </div>
    </nav>

    <!-- 주요 콘텐츠 -->
    <main class="container mt-5">
        <div class="row">
            <div class="col-md-6">
                <h1>Bootstrap에 오신 것을 환영합니다</h1>
                <p class="lead">이것은 Bootstrap 5로 만든 반응형 페이지입니다.</p>
                <button class="btn btn-primary">더 알아보기</button>
            </div>
            <div class="col-md-6">
                <div class="card">
                    <div class="card-body">
                        <h5 class="card-title">특징</h5>
                        <ul>
                            <li>반응형 레이아웃</li>
                            <li>다양한 컴포넌트</li>
                            <li>강력한 JavaScript 플러그인</li>
                        </ul>
                    </div>
                </div>
            </div>
        </div>
    </main>

    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## 버전 선택

### 운영 환경

안정적인 버전을 사용합니다:

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### 개발 환경

최신 개발 버전을 사용합니다(소스 코드 포함):

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.js"></script>
```

### 중국 미러

국내에서 사용할 수 있는 중국 미러로 속도 향상:

```html
<link href="https://cdn.bootcss.com/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.bootcss.com/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

## 제거

제거해야 하는 경우(관련 파일만 삭제하면 됩니다):

- HTML에서 CSS와 JS 참조 삭제
- node_modules에서 bootstrap 디렉터리 삭제(npm을 사용하는 경우)
- 로컬에 다운로드한 CSS와 JS 파일 삭제
