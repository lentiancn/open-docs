# H2 FAQ

## Installation

### Wie H2 herunterladen?
Von https://www.h2database.com/html/download.html

### Java-Anforderungen?
Java 11 oder höher.

### Maven-Abhängigkeit?
```xml
<dependency>
    <groupId>com.h2database</groupId>
    <artifactId>h2</artifactId>
    <version>2.2.224</version>
</dependency>
```

## Verbindung

### Standard-Benutzer?
Benutzer: sa, Passwort: (leer)

### Wie verbinden?
```java
Class.forName("org.h2.Driver");
Connection conn = DriverManager.getConnection("jdbc:h2:mem:testdb", "sa", "");
```

## Daten

### Datenverlust bei In-Memory?
Ja, Daten gehen verloren, wenn alle Verbindungen geschlossen werden.

### Datenbank verschlüsseln?
```java
jdbc:h2:file:./encrypted;CIPHER=AES
```

## Leistung

### Batch-Operationen?
```java
conn.setAutoCommit(false);
// ... batch operations ...
conn.commit();
```

## Sicherheit

### SQL-Injection verhindern?
PreparedStatement verwenden:
```java
PreparedStatement pstmt = conn.prepareStatement("SELECT * FROM users WHERE name = ?");
pstmt.setString(1, name);
```

## Werkzeuge

### Konsole starten?
```bash
java -jar h2.jar -web
```
Dann http://localhost:8082

### Version prüfen?
```bash
java -jar h2.jar -version
```
