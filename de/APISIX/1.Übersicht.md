# APISIX Übersicht

Cloud-nativer API-Gateway.

## Was ist APISIX?

APISIX ist ein cloud-nativer API-Gateway der Apache Software Foundation, der hochleistungsfähige und hochverfügbare API-Verwaltungsdienste bietet. Als dynamischer, echtzeitfähiger und skalierbarer API-Gateway ist APISIX die bevorzugte Lösung für Millionen von Container-Deployments.

### Hauptfunktionen

- **Hohe Leistung**: Basiert auf Nginx + Lua, 10k+ QPS pro Kern
- **Dynamisches Routing**: Routing-Regeln ohne Neustart aktualisieren
- **Plugin-System**: 70+ Plugins mit Hot-Loading-Unterstützung
- **Service-Discovery**: Unterstützung für Consul, Eureka, Nacos
- **Load Balancing**: Roundrobin, Hash, Konsistenter Hash
- **Sicherheit**: Rate Limiting, Authentifizierung, JWT, Firewall

### Technische Architektur

APISIX verwendet Nginx als zugrundeliegenden HTTP-Reverse-Proxy-Server, kombiniert mit Lua-Skripten für dynamische Orchestrierung.

## Anwendungsfälle

- **Einheitliche API-Verwaltung**: Alle internen APIs zentral verwalten
- **Microservices-Gateway**: Als Eingangspunkt für Auth, Rate Limiting
- **Service-Mesh-Ingress**: Integration mit Istio, Linkerd
- **API-Verkehrskontrolle**: Rate Limiting, Circuit Breaking
