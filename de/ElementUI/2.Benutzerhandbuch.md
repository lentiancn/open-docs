# Element UI Benutzerhandbuch

Dieses Handbuch erklärt die Verwendung von Element UI Komponenten.

## Grundlegende Verwendung

```vue
<el-button>Standard</el-button>
<el-button type="primary">Primär</el-button>
```

## Komponenten

### Button

```vue
<el-button type="primary">Primär</el-button>
<el-button type="success">Erfolg</el-button>
<el-button type="warning">Warnung</el-button>
<el-button type="danger">Gefahr</el-button>
```

### Input

```vue
<el-input v-model="input" placeholder="Text eingeben"></el-input>
```

### Table

```vue
<el-table :data="tableData">
  <el-table-column prop="name" label="Name"></el-table-column>
</el-table>
```

### Form

```vue
<el-form :model="form">
  <el-form-item label="Name">
    <el-input v-model="form.name"></el-input>
  </el-form-item>
  <el-button type="primary" @click="submit">Absenden</el-button>
</el-form>
```

### Dialog

```vue
<el-button @click="visible = true">Öffnen</el-button>
<el-dialog :visible.sync="visible" title="Titel">
  <span>Inhalt</span>
</el-dialog>
```

## Nachrichten

```javascript
this.$message.success('Erfolg')
this.$message.error('Fehler')
```
