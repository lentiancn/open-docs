# Element Plus 사용자 매뉴얼

사용 가이드.

## 빠른 시작

```vue
<template>
  <el-button type="primary" @click="handleClick">
    클릭
  </el-button>
</template>

<script setup>
import { ref } from 'vue'

const handleClick = () => {
  ElMessage.success('안녕하세요!')
}
</script>
```

## 컴포넌트

### Button

```vue
<el-button>기본</el-button>
<el-button type="primary">프라이머리</el-button>
<el-button disabled>비활성화</el-button>
<el-button loading>로딩</el-button>
```

### Input

```vue
<el-input v-model="value" placeholder="입력" />
<el-input v-model="value" disabled placeholder="비활성화" />
<el-input v-model="value" clearable placeholder="지우기 가능" />
```

### Table

```vue
<el-table :data="tableData" stripe border>
  <el-table-column prop="name" label="이름" />
  <el-table-column prop="email" label="이메일" />
</el-table>
```

### Form

```vue
<el-form :model="form" :rules="rules" ref="formRef">
  <el-form-item label="이름" prop="name">
    <el-input v-model="form.name" />
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="submit">제출</el-button>
  </el-form-item>
</el-form>
```

### Dialog

```vue
<el-button @click="visible = true">열기</el-button>
<el-dialog v-model="visible" title="제목">
  <span>내용</span>
</el-dialog>
```

### Message

```typescript
import { ElMessage } from 'element-plus'

ElMessage.success('성공')
ElMessage.error('오류')
```

## 레이아웃

```vue
<el-row :gutter="20">
  <el-col :span="12"><div>12</div></el-col>
  <el-col :span="12"><div>12</div></el-col>
</el-row>
```
