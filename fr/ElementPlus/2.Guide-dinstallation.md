# Guide d'installation Element Plus

Ce guide détaille comment installer et utiliser Element Plus dans différents projets.

## Exigences environnementales

Element Plus nécessite:

- Vue 3.0.0 ou supérieur
- Node.js 14.0.0 ou supérieur
- Navigateurs modernes (Chrome, Firefox, Safari, Edge)

## Méthodes d'installation

### Utilisation du gestionnaire de paquets

Installation recommandée avec npm, yarn ou pnpm:

Avec npm:
npm install element-plus

Avec yarn:
yarn add element-plus

Avec pnpm:
pnpm add element-plus

## Importation complète

Si la taille du bundle ne vous préoccupe pas, l'importation complète est plus pratique.

Dans main.ts:

import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')

### Support Volar

Si vous utilisez Volar, ajoutez la définition de type de composant global dans compilerOptions.types:

{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}

## Importation à la demande

L'importation à la demande peut réduire considérablement la taille du projet. Vous devez utiliser des plugins supplémentaires pour importer les composants que vous utilisez.

### Importation automatique (Recommandée)

Installer les plugins:

Pour npm:
npm install -D unplugin-vue-components unplugin-auto-import

Pour yarn:
yarn add -D unplugin-vue-components unplugin-auto-import

Pour pnpm:
pnpm add -D unplugin-vue-components unplugin-auto-import

#### Configuration Vite

Dans vite.config.ts:

import { defineConfig } from 'vite'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    AutoImport({ resolvers: [ElementPlusResolver()] }),
    Components({ resolvers: [ElementPlusResolver()] }),
  ],
})

#### Configuration Webpack

Dans webpack.config.js:

const AutoImport = require('unplugin-auto-import/webpack')
const Components = require('unplugin-vue-components/webpack')
const { ElementPlusResolver } = require('unplugin-vue-components/resolvers')

module.exports = {
  plugins: [
    AutoImport({ resolvers: [ElementPlusResolver()] }),
    Components({ resolvers: [ElementPlusResolver()] }),
  ],
}

### Utilisation avec Nuxt

Pour les utilisateurs Nuxt, installez simplement @element-plus/nuxt:

npm install -D @element-plus/nuxt

Puis ajouter dans nuxt.config.ts:

export default defineNuxtConfig({
  modules: ['@element-plus/nuxt'],
})

### Importation manuelle

Element Plus fournit des fonctionnalités Tree Shaking prêtes à l'emploi basées sur ES Module.

Vous devez installer unplugin-element-plus pour importer les styles:

npm install -D unplugin-element-plus

Dans les composants:

<template>
  <el-button>Bouton</el-button>
</template>

<script>
import { ElButton } from 'element-plus'

export default {
  components: { ElButton },
}
</script>

Configuration Vite:

import { defineConfig } from 'vite'
import ElementPlus from 'unplugin-element-plus/vite'

export default defineConfig({
  plugins: [ElementPlus()],
})

## Configuration globale

Lors de l'enregistrement de Element Plus, vous pouvez passer un objet de configuration global avec size et zIndex:

import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import App from './App.vue'

const app = createApp(App)
app.use(ElementPlus, { size: 'small', zIndex: 3000 })

Options de configuration:

finir la taille par défaut pour- size: Dé les composants de formulaire. Les valeurs possibles sont large, default et small
- zIndex: Définir le z-index pour les composants popup. La valeur par défaut est 2000

### Configuration d'importation à la demande

Utilisez le composant el-config-provider:

<template>
  <el-config-provider :size="size" :z-index="zIndex">
    <app />
  </el-config-provider>
</template>

<script>
import { defineComponent } from 'vue'
import { ElConfigProvider } from 'element-plus'

export default defineComponent({
  components: { ElConfigProvider },
  setup() {
    return { zIndex: 3000, size: 'small' }
  },
})
</script>
