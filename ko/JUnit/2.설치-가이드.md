# JUnit 5 설치 가이드

## 요구 사항

- JDK 8 이상
- Maven 또는 Gradle

## Maven 설치

pom.xml에 추가:
```xml
<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter</artifactId>
    <version>5.10.0</version>
    <scope>test</scope>
</dependency>
```

실행: `mvn test`

## Gradle 설치

build.gradle:
```groovy
dependencies {
    testImplementation 'org.junit.jupiter:junit-jupiter:5.10.0'
}
test { useJUnitPlatform() }
```

실행: `./gradlew test`
