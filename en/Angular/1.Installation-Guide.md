## Introduction

### What is Angular?

Angular is a TypeScript-based open-source web application framework developed and maintained by Google for building modern single-page applications (SPAs).

## Installation Methods

### Method 1: Using Angular CLI (Recommended)

```bash
# Install Angular CLI
npm install -g @angular/cli

# Create new project
ng new my-app

# Start development server
cd my-app
ng serve
```

### Method 2: Using npm

```bash
mkdir angular-project
cd angular-project
npm init -y
npm install @angular/core @angular/common @angular/platform-browser
```

---

## Installation on Windows

### Step 1: Install Node.js

1. Visit https://nodejs.org/
2. Download LTS version
3. Run installer

### Step 2: Install Angular CLI

```cmd
npm install -g @angular/cli
```

### Step 3: Create Project

```cmd
ng new my-angular-app
cd my-angular-app
ng serve
```

Access http://localhost:4200

## Installation on macOS

### Using Homebrew (Recommended)

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install node@18
npm install -g @angular/cli
ng new my-angular-app
cd my-angular-app
ng serve
```

---

## Installation on Linux

### Ubuntu/Debian

```bash
sudo apt update && sudo apt upgrade -y
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs
sudo npm install -g @angular/cli
ng new my-angular-app
cd my-angular-app
ng serve
```

## Troubleshooting

### npm Permission Errors

```bash
# Install nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 18
nvm use 18
```

### Port Already in Use

```bash
ng serve --port 4201
```
