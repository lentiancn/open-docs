> Actualizado: Febrero 2026

## 1. Introducción

DragonTalker es una herramienta de generación de cabezas parlantes basada en aprendizaje profundo que crea videos realistas de cabezas parlantes a partir de una imagen estática y un archivo de audio. Esta tecnología se usa ampliamente enPresentadores virtuales, servicio al cliente digital humano, educación IA y otros escenarios.

### 1.1 Tecnología Central

DragonTalker utiliza una arquitectura de pipeline multifase: primero extrae características de mel-espectrograma mediante análisis de audio; luego usa detección de puntos clave faciales 3D para obtener parámetros de movimiento facial; después genera marcos de video continuos usando tecnología de renderizado de imagen; finalmente mejora la calidad del video a través de redes de restauración facial. Todo el proceso no requiere grandes conjuntos de datos de entrenamiento y es fácil de implementar.

### 1.2 Características Principales

- Animación impulsada por audio: Genera videos parlantes sincronizados a partir de una imagen estática y archivo de audio
- Estimación de movimiento 3D: Posturas de cabeza naturales y expresiones faciales basadas en reconstrucción de landmarks faciales 3D
- Soporte múltiples poses: Más de 45 plantillas de poses de cabeza predefinidas
- Mejora facial: Integra GFPGAN, RestoreFormer, CodeFormer para restauración facial avanzada
- Interfaz web: Operación basada en navegador sin configuración compleja
- Python API: Interfaz de programación completa para desarrollo secundario y procesamiento por lotes

## 2. Requisitos del Sistema

### 2.1 Configuración de Hardware

| Componente | Mínimo | Recomendado |
|------------|--------|-------------|
| GPU | NVIDIA GTX 1060 (6GB VRAM) | NVIDIA RTX 3080+ (16GB VRAM) |
| RAM | 8GB | 32GB |
| Almacenamiento | 20GB | 50GB SSD |

### 2.2 Entorno de Software

| Software | Versión | Notas |
|----------|---------|-------|
| SO | Ubuntu 18.04/20.04/22.04, Windows 10/11, macOS 11+ | 64 bits recomendado |
| Python | 3.8 - 3.10 | Versiones mayores pueden tener problemas |
| CUDA | 11.7+ | Requerido para inferencia GPU |
| ffmpeg | Cualquier versión estable | Codificador de video requerido |

## 3. Pasos de Instalación

### 3.1 Instalación en Ubuntu/Linux

#### 3.1.1 Instalar Dependencias del Sistema

```bash
sudo apt-get update
sudo apt-get install -y build-essential git wget curl
sudo apt-get install -y python3.8 python3-pip python3-venv
sudo apt-get install -y ffmpeg libsm6 libxext6 libxrender-dev libgl1-mesa-glx
```

#### 3.1.2 Crear Entorno Virtual

```bash
cd DragonTalker
python3 -m venv venv
source venv/bin/activate
python --version
```

#### 3.1.3 Instalar Dependencias Python

```bash
pip install --upgrade pip
pip install torch==1.12.1+cu113 torchvision==0.13.1+cu113 -f https://download.pytorch.org/whl/cu113
pip install -r requirements.txt
```

#### 3.1.4 Descargar Modelos Pre-entrenados

```bash
bash scripts/download_models.sh
```

### 3.2 Instalación en Windows

1. **Instalar Python**: Descargar Python 3.8-3.10 desde python.org, marcar "Add Python to PATH"

2. **Instalar Git**: Descargar desde git-scm.com

3. **Instalar CUDA**: Descargar CUDA Toolkit 11.7+ desde NVIDIA

4. **Instalar ffmpeg**:
```powershell
choco install ffmpeg
```

#### Clonar y Configurar

```powershell
git clone https://github.com/your-repo/DragonTalker.git
cd DragonTalker
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 3.3 Instalación en macOS

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install ffmpeg python@3.10 git wget
git clone https://github.com/your-repo/DragonTalker.git
cd DragonTalker
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## 4. Verificar Instalación

```bash
source venv/bin/activate
python inference.py \
  --source_image examples/source_image.jpg \
  --driven_audio examples/driven_audio.wav \
  --result_dir ./results \
  --enhancer gfpgan
```

---

## 5. Instalación con Docker (Recomendado)

```bash
docker build -t dragon-talker:latest .
docker run --gpus all -v $(pwd)/examples:/app/examples \
  -v $(pwd)/results:/app/results \
  dragon-talker:latest \
  python inference.py \
    --source_image /app/examples/source_image.jpg \
    --driven_audio /app/examples/driven_audio.wav \
    --result_dir /app/results
```

---

## 6. Desinstalación

```bash
deactivate
rm -rf venv
docker rmi dragon-talker:latest
```
