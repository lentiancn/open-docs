# Element Plus FAQ

## Installationsprobleme

### Installation fehlgeschlagen

**Lösungen**:

1. Node.js-Version prüfen (>= 14.0.0)
2. npm-Cache leeren: npm cache clean --force

### TypeScript-Fehler

In tsconfig.json hinzufügen:

```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

## Styling-Probleme

### Stile werden nicht angewendet

CSS importieren:

```javascript
import 'element-plus/dist/index.css'
```

### Symbole werden nicht angezeigt

Icon-Bibliothek installieren:

```bash
npm install @element-plus/icons-vue
```

## Leistungsprobleme

### Große Datenmengen

Verwenden Sie Paginierung oder virtuelle Tabellen.
