# H2 数据库简介

## 什么是 H2 数据库

H2 是一个用 Java 编写的开源关系型数据库管理系统。它可以用作 Java 应用程序的嵌入式数据库，也可以运行在客户端-服务器模式下。该软件以 Mozilla 公共许可证 2.0（MPL 2.0）或原始的 Eclipse 公共许可证（EPL 1.0）开源提供。

## H2 的主要特点

### 嵌入式数据库
H2 可以嵌入到 Java 应用程序中直接运行，无需单独的数据库服务器。这使得它在开发和测试环境中非常受欢迎。

### 客户端-服务器模式
除了嵌入式模式，H2 还支持客户端-服务器模式，可以作为独立的数据库服务器运行，供多个应用程序同时访问。

### 内存数据库
H2 支持内存数据库模式，数据存储在内存中，重启后数据会丢失。这对于快速测试和原型开发非常有用。

### 兼容多种数据库
H2 兼容多种 SQL 标准和语法，可以模拟 MySQL、PostgreSQL、Oracle、DB2、MS SQL Server 等数据库的行为，便于数据库迁移和测试。

### 加密数据库
H2 支持数据库加密，可以保护敏感数据的安全。

### 全文搜索
H2 内置全文搜索功能，无需额外配置。

### 零管理
作为嵌入式数据库，H2 不需要专门的数据库管理员进行维护。

## H2 应用场景

H2 广泛应用于以下场景：

- Java 应用程序的嵌入式数据库
- 开发与测试环境
- 原型快速开发
- 单元测试和集成测试
- 轻量级桌面应用
- 学习 SQL 语法

## H2 版本历史

H2 数据库引擎的开发始于 2004 年 5 月，首次发布于 2005 年 12 月。原始开发者 Thomas Mueller 也是 Java 数据库引擎 HSQLDB（原名 Hypersonic SQL）的创始人。

## 许可证

H2 采用双许可证模式：
- Mozilla 公共许可证 2.0（MPL 2.0）
- Eclipse 公共许可证 1.0（EPL 1.0）

## 系统要求

- Java 运行时环境（JRE）11 或更高版本
- 可以在任何支持 Java 的操作系统上运行，包括 Windows、Linux、macOS 等
