# MySQL インストールガイド

## Windows へのインストール

### MySQL Installer を使用したインストール

MySQL Installer は、Windows プラットフォームで MySQL をインストールする最も簡単な方法でウィザードベースのインストール体験を提供します。

1. MySQL Installer をダウンロード
   https://dev.mysql.com/downloads/installer/ からインストーラーをダウンロードします。

2. インストーラーを実行
   ダウンロードした msi ファイルをダブルクリックしてインストールウィザードを起動します。

3. インストールタイプを選択
   - **Full**: すべての MySQL 製品と機能をインストール
   - **Developer Default**: 開発環境に適切
   - **Server only**: MySQL Server のみをインストール
   - **Custom**: コンポーネントをカスタマイズ

4. MySQL Server を設定
   - MySQL Server バージョンを選択
   - ポート番号を設定（デフォルト 3306）
   - 起動タイプを選択

5. root パスワードを設定
   MySQL 管理者アカウントに強力なパスワードを設定します。

6. インストールを完了
   ウィザードに従ってインストールプロセスを完了します。

### ZIP アーカイブを使用した手動インストール

1. MySQL ZIP アーカイブをダウンロード
   https://dev.mysql.com/downloads/mysql/ から ZIP 格式のインストールアーカイブをダウンロードします。

2. ファイルを展開
   ZIP アーカイブを目的のディレクトリ（例：C:\mysql）に展開します。

3. データディレクトリを初期化
   ```
   mysqld --initialize-insecure
   ```

4. MySQL サービスを起動
   ```
   mysqld --install MySQL
   net start MySQL
   ```

5. ログインしてパスワードを変更
   ```
   mysql -u root -p
   ALTER USER 'root'@'localhost' IDENTIFIED BY '新しいパスワード';
   ```

## Linux へのインストール

### Ubuntu/Debian システム

1. パッケージインデックスを更新
   ```bash
   sudo apt update
   ```

2. MySQL Server をインストール
   ```bash
   sudo apt install mysql-server
   ```

3. MySQL サービスを起動
   ```bash
   sudo systemctl start mysql
   sudo systemctl enable mysql
   ```

4. セキュアな設定
   ```bash
   sudo mysql_secure_installation
   ```

### CentOS/RHEL システム

1. MySQL Yum リポジトリを追加
   ```bash
   sudo yum install https://dev.mysql.com/get/mysql80-community-release-el8-4.noarch.rpm
   ```

2. MySQL Server をインストール
   ```bash
   sudo yum install mysql-community-server
   ```

3. サービスを起動
   ```bash
   sudo systemctl start mysqld
   sudo systemctl enable mysqld
   ```

4. 一時パスワードを取得
   ```bash
   sudo grep 'temporary password' /var/log/mysqld.log
   ```

## macOS へのインストール

### Homebrew を使用

1. Homebrew をインストール（未インストールの場合）
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```

2. MySQL をインストール
   ```bash
   brew install mysql
   ```

3. MySQL サービスを起動
   ```bash
   brew services start mysql
   ```

4. セキュアな設定
   ```bash
   mysql_secure_installation
   ```

### DMG パッケージを使用

1. DMG パッケージをダウンロード
   https://dev.mysql.com/downloads/mysql/ から macOS DMG パッケージをダウンロードします。

2. インストーラーを実行
   PKG ファイルをダブルクリックし、ウィザードに従ってインストールを完了します。

3. MySQL を起動
   システム環境設定から MySQL サービスを起動します。

## Docker への MySQL のインストール

### MySQL イメージをプル
```bash
docker pull mysql:latest
```

### MySQL コンテナを実行
```bash
docker run -d \
  --name mysql-container \
  -e MYSQL_ROOT_PASSWORD=your_password \
  -e MYSQL_DATABASE=testdb \
  -p 3306:3306 \
  mysql:latest
```

### 一般的な Docker パラメータの説明
- `-e MYSQL_ROOT_PASSWORD`: root パスワードを設定
- `-e MYSQL_DATABASE`: 初期データベースを作成
- `-p 3306:3306`: ポートマッピング
- `-v /data/mysql:/var/lib/mysql`: データボリュームの永続化

## インストール後の確認

### MySQL バージョンを確認
```bash
mysql --version
```

### サービスの状態を確認
```bash
systemctl status mysql   # Linux
brew services list      # macOS
```

### MySQL にログイン
```bash
mysql -u root -p
```

## 一般的なインストール問題

### ポートの競合
3306 ポートが使用中の場合は、設定ファイルでポートを変更するか、そのポートを使用しているサービスを停止してください。

### 権限の問題
Linux システムでは、sudo を使用して権限を昇格し、データディレクトリに正しい読み取り/書き込み権限があることを確認してください。

### ファイアウォールの設定
ファイアウォールが 3306 ポートへのアクセスを許可していることを確認してください。
