# Apache JMeter - 개요

## Apache JMeter란?

Apache JMeter™는 성능 테스트 및 부하 테스트를 위해 설계된 100% 순수 Java 애플리케이션인 오픈소스 소프트웨어입니다. Originally 웹 애플리케이션 테스트를 위해 설계되었지만, 이후 다른 테스트 기능으로 확장되었습니다.

## JMeter로 무엇을 할 수 있나요?

Apache JMeter는 정적 및 동적 리소스, 동적 웹 애플리케이션의 성능 테스트에 사용될 수 있습니다. 서버, 서버 그룹, 네트워크 또는 개체에 무거운 부하를 시뮬레이션하여 강도를 테스트하거나 다양한 부하 유형에서 전반적인 성능을 분석할 수 있습니다.

### 주요 기능

- **멀티 프로토콜 지원**: 웹(HTTP, HTTPS), SOAP/REST 웹 서비스, FTP, JDBC를 통한 데이터베이스, LDAP, JMS, 메일, TCP, Java 객체
- **전체 기능 테스트 IDE**: 브라우저 또는 네이티브 애플리케이션에서 테스트 플랜 기록
- **CLI 모드**: 모든 Java 호환 OS에서 부하 테스트 실행
- **동적 HTML 보고서**: 완전히 제공 가능한 상태
- **데이터 추출**: HTML, JSON, XML 및 정규식 지원
- **100% Java**: 완전한 이식성
- **멀티 스레딩**: 여러 스레드による동시 샘플링
- **높은 확장성**: 플러그형 샘플러 및 스크립트 가능한 샘플러(Groovy, BeanShell)
- **CI 통합**: Maven, Gradle, Jenkins 지원

## JMeter가 아닌 것

JMeter는 브라우저가 아닙니다 - 프로토콜 수준에서 작동합니다. HTML 페이지에서 발견된 JavaScript를 실행하거나 실제 브라우저처럼 HTML 페이지를 렌더링하지 않습니다.

## 시작하기

1. 공식 웹사이트에서 JMeter 다운로드
2. JMeter 스크립트 실행(Windows의 경우 jmeter.bat, Linux/Mac의 경우 jmeter.sh)
3. GUI를 사용하여 테스트 플랜 생성
4. 테스트 실행 및 결과 분석
