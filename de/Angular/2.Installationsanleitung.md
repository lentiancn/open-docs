# Angular Installationsanleitung

Detaillierte Anleitung zur Einrichtung der Angular-Umgebung.

## Anforderungen

### Node.js

- Empfohlen: Node.js 18+
- Benötigt npm oder yarn

### Node.js Version prüfen

```bash
node --version
npm --version
```

## Angular CLI installieren

### Installieren

```bash
npm install -g @angular/cli
```

### Installation überprüfen

```bash
ng version
```

### Aktualisieren

```bash
npm install -g @angular/cli@latest
```

## Neues Projekt erstellen

### Angular CLI verwenden

```bash
ng new my-app
cd my-app
```

### Interaktive Optionen

- Routing hinzufügen? Ja/Nein
- Welches Stylesheet-Format? (SCSS/CSS/Sass/Less)

## Anwendung ausführen

### Entwicklungsserver

```bash
ng serve
# oder
npm start
```

Besuchen: http://localhost:4200

### Hot Reload

Angular CLI unterstützt Hot Reload - Änderungen werden automatisch aktualisiert.

## Für Produktion bauen

```bash
ng build --configuration production
```

Ausgabe im `dist/` Verzeichnis.

## Häufige Befehle

| Befehl | Funktion |
|--------|----------|
| ng new | Neues Projekt erstellen |
| ng serve | Entwicklungsserver starten |
| ng build | Projekt bauen |
| ng generate | Komponente/Service/Modul generieren |
| ng test | Unit-Tests ausführen |
| ng e2e | E2E-Tests ausführen |

## Projektstruktur

```
my-app/
├── src/
│   ├── app/
│   │   ├── components/
│   │   ├── services/
│   │   ├── app.component.ts
│   │   └── app.module.ts
│   ├── assets/
│   └── styles.css
├── angular.json
└── package.json
```

## Angular CLI deinstallieren

```bash
npm uninstall -g @angular/cli
```
