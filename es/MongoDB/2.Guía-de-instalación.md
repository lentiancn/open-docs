# MongoDB Guía de Instalación

## Selección de Versión y Requisitos de Plataforma

### Selección de Versión

MongoDB actualmente tiene dos series de versiones principales:

- **MongoDB 8.2**: Versión estable más reciente, recomendada para nuevos proyectos
- **MongoDB 7.0**: Versión de soporte a largo plazo, adecuada para entornos de producción que requieren estabilidad

A partir de MongoDB 8.2, las versiones menores están disponibles para casos de uso específicos en implementaciones locales.

### Requisitos de Plataforma

- MongoDB ya no es compatible con plataformas de 32 bits. Asegúrese de que su sistema operativo sea de 64 bits
- MongoDB 6.0 llegó al final de su vida útil el 31 de julio de 2025

### Plataformas Soportadas

**Ubuntu**

- Ubuntu 24.04 (x86_64, arm64)

**RHEL/CentOS/Rocky Linux**

- RHEL/Rocky/Alma/Oracle Linux 9.0+ (x86_64, arm64)
- RHEL/Rocky/Alma/Oracle Linux 8.0+ (x86_64, arm64, ppc64le, s390x)

## Instalar MongoDB Edición Comunidad en Ubuntu

### Paso 1: Importar la Clave Pública

```bash
curl -fsSL https://www.mongodb.org/static/pgp/server-8.0.asc | sudo gpg -o /usr/share/keyrings/mongodb-server-8.0.gpg --dearmor
```

### Paso 2: Crear el Archivo de Lista de Repositorios

**Ubuntu 24.04 (Noble)**:

```bash
echo "deb [ signed-by=/usr/share/keyrings/mongodb-server-8.0.gpg ] http://repo.mongodb.org/apt/debian noble/mongodb-org/8.0 main" | sudo tee /etc/apt/sources.list.d/mongodb-org-8.0.list
```

**Ubuntu 22.04 (Jammy)**:

```bash
echo "deb [ signed-by=/usr/share/keyrings/mongodb-server-8.0.gpg ] http://repo.mongodb.org/apt/debian jammy/mongodb-org/8.0 main" | sudo tee /etc/apt/sources.list.d/mongodb-org-8.0.list
```

**Ubuntu 20.04 (Focal)**:

```bash
echo "deb [ signed-by=/usr/share/keyrings/mongodb-server-8.0.gpg ] http://repo.mongodb.org/apt/debian focal/mongodb-org/8.0 main" | sudo tee /etc/apt/sources.list.d/mongodb-org-8.0.list
```

### Paso 3: Actualizar la Lista de Paquetes

```bash
sudo apt update
```

### Paso 4: Instalar MongoDB

```bash
sudo apt install -y mongodb-org
```

### Paso 5: Iniciar MongoDB

```bash
sudo systemctl start mongod
sudo systemctl enable mongod
```

## Instalar en RHEL/CentOS/Rocky Linux

### Usando YUM/DNF

#### Paso 1: Crear el Archivo de Repositorio

```bash
[mongodb-org-8.0]
name=MongoDB Repository
baseurl=https://repo.mongodb.org/yum/redhat/$releasever/mongodb-org/8.0/x86_64/
gpgcheck=1
enabled=1
gpgkey=https://www.mongodb.org/static/pgp/server-8.0.asc
```

#### Paso 2: Instalar MongoDB

```bash
sudo dnf install -y mongodb-org
```

#### Paso 3: Iniciar MongoDB

```bash
sudo systemctl start mongod
sudo systemctl enable mongod
```


## Instalar en Windows

### Usando el Instalador MSI

1. Descargue el instalador MSI de MongoDB Download Center
2. Ejecute el instalador y siga los pasos del asistente
3. Seleccione "Complete" para instalar todos los componentes

### Usando el Paquete ZIP

1. Descargue el paquete ZIP de Windows
2. Extraiga los archivos a `C:\Program Files\MongoDB`
3. Cree el directorio de datos: `mkdir C:\data\db`
4. Inicie MongoDB: `"C:\Program Files\MongoDB\Server\8.0\bin\mongod.exe" --dbpath C:\data\db`

## Instalar Usando Docker

### Paso 1: Extraer la Imagen

```bash
docker pull mongodb/mongodb-community-server
```

### Paso 2: Ejecutar el Contenedor

```bash
docker run --name mongodb -d -p 27017:27017 mongodb/mongodb-community-server
```

### Persistencia de Datos

```bash
docker run --name mongodb -d -p 27017:27017 -v mongodb_data:/data/db mongodb/mongodb-community-server
```


```bash
show dbs

```

## Configuración Post-Instalación

### Crear un Usuario Administrador

```bash
use admin
db.createUser({
  user: "admin",
  pwd: "your_secure_password",
  roles: [
    { role: "userAdminAnyDatabase", db: "admin" },
    { role: "readWriteAnyDatabase", db: "admin" }
  ]
})
```

### Habilitar Autenticación

1. Detenga el servicio MongoDB
2. Modifique `/etc/mongod.conf`:

```yaml
security:
  authorization: enabled
```

3. Reinicie el servicio:

```bash
sudo systemctl restart mongod
```

## Desinstalar MongoDB

### Ubuntu/Debian

```bash
sudo apt remove --purge mongodb-org
sudo rm -rf /var/lib/mongodb
sudo rm -rf /var/log/mongodb
```

### macOS

```bash
brew uninstall mongodb-community
brew services stop mongodb-community
```

## Preguntas Frecuentes

### ¿Cuál es el puerto predeterminado de MongoDB?

El puerto predeterminado es 27017. Puede especificar otro puerto usando el parámetro `--port`.

### ¿Dónde almacena MongoDB los datos?

Por defecto:
- Linux/macOS: `/data/db`
- Windows: `C:\data\db`

### ¿Cómo verificar el estado de MongoDB?

```bash
sudo systemctl status mongod
```
