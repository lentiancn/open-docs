# Flutter - Руководство по установке

Настройка среды разработки Flutter.

## Требования

- Windows 10+ / macOS 10.14+ / Ubuntu 18.04+
- 2.8 GB места
- Git

## Установка

### Windows

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

Добавить в PATH.

### macOS

```bash
git clone https://github.com/flutter/flutter.git -b stable
export PATH="$PATH:/Users/пользователь/flutter/bin"
```

### Linux

```bash
sudo apt-get install -y curl git unzip xz-utils
git clone https://github.com/flutter/flutter.git -b stable
export PATH="$PATH:$HOME/flutter/bin"
```

## Проверка

```bash
flutter doctor
flutter --version
```
