# FantasyTalking 사용자 매뉴얼

## 음성 합성

```javascript
const audio = await client.speak('안녕하세요, FantasyTalking입니다');
```

## 이벤트

```javascript
client.on('speech-start', () => console.log('재생 시작'));
client.on('speech-end', () => console.log('재생 종료'));
```

## 대화

```javascript
const session = client.createSession({ language: 'ko-KR' });
const response = await session.chat('오늘 날씨가 어때요?');
```

## 스트리밍

```javascript
const stream = await client.speakStream('긴 텍스트입니다');
stream.on('data', chunk => player.play(chunk));
```
