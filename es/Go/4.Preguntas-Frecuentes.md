# Preguntas Frecuentes de Go

> Preguntas frecuentes

---

## Instalación

### Windows: ¿comando go no encontrado?

**Solución**: Agregar `C:\Go\bin` al PATH.

### ¿Proxy?

```bash
go env -w GOPROXY=https://goproxy.cn,direct
```

---

## Desarrollo

### Error "cannot find package"?

**Solución**:
1. Ejecutar `go mod init`
2. Ejecutar `go mod tidy`

### := vs var?

- `var`: Explícito, fuera de funciones
- `:=`: Corto, solo dentro de funciones

---

## Concurrencia

### ¿Goroutine leak?

```go
func worker(ctx context.Context) {
    for {
        select {
        case <-ctx.Done():
            return
        }
    }
}
```
