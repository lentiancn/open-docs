# FantasyTalking - Manuel Utilisateur

Guide détaillé.

## Utilisation de base

### Générer une vidéo

```bash
python inference.py --source_image face.jpg --audio speech.wav --output output.mp4
```

### Paramètres

| Paramètre | Description |
|-----------|-------------|
| --source_image | Image d'entrée |
| --audio | Fichier audio |
| --output | Chemin de sortie |
| --face_enhancer | Amélioration du visage |
| --fps | Images par seconde |

## API Python

```python
from fantasytalking import FantasyTalking

ft = FantasyTalking()
ft.generate(
    source_image="face.jpg",
    audio="speech.wav",
    output="output.mp4"
)
```

## Traitement par lots

```python
ft.batch_generate(image_list, audio_list, output_dir)
```

## Exigences d'entrée

### Image

- Format: JPG, PNG
- Résolution: 512x512+
- Visage: Frontal, clair

### Audio

- Format: WAV, MP3
- Durée: 1-60 secondes
