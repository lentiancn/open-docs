# MongoDB Manuel d'utilisation

## Concepts de base

### Base de données

Une base de données est le conteneur de plus haut niveau pour les données dans MongoDB. Chaque base de données a ses propres collections et paramètres de permission.

### Collection

Une collection est un groupe de documents MongoDB, similaire à une table dans une base de données relationnelle.

### Document

Un document est l'unité de données de base dans MongoDB, consisting de paires champ-valeur. Les documents sont similaires aux objets JSON et sont stockés au format BSON.

## Connexion à MongoDB

### Se connecter à la base de données locale

```bash
mongosh
```

### Se connecter à une base de données distante

```bash
mongosh "mongodb://hostname:port"
```

## Opérations de base de données

### Afficher les bases de données

```bash
show dbs
```

### Basculer/créer une base de données

```bash
use mabasededonnees
```

### Supprimer la base de données

```bash
db.dropDatabase()
```

## Opérations de collection

### Afficher les collections

```bash
show collections
```

### Créer une collection

```bash
db.createCollection("macollection")
```

### Supprimer une collection

```bash
db.macollection.drop()
```

## Insertion de documents

### Insérer un document

```bash
db.utilisateurs.insertOne({
  nom: "Jean",
  age: 30,
  email: "jean@exemple.fr"
})
```

### Insérer plusieurs documents

```bash
db.utilisateurs.insertMany([
  { nom: "Marie", age: 25 },
  { nom: "Pierre", age: 35 }
])
```

## Requête de documents

### Requêter tous les documents

```bash
db.utilisateurs.find()
```

### Requêter avec conditions

```bash
db.utilisateurs.find({ age: 30 })
db.utilisateurs.find({ age: { $gte: 25 } })
```

### Trier les résultats

```bash
db.utilisateurs.find().sort({ age: -1 })
```

### Limiter les résultats

```bash
db.utilisateurs.find().limit(5)
```

## Mise à jour des documents

### Mettre à jour un document

```bash
db.utilisateurs.updateOne(
  { nom: "Jean" },
  { $set: { age: 31 } }
)
```

### Mettre à jour plusieurs documents

```bash
db.utilisateurs.updateMany(
  { age: { $lt: 30 } },
  { $set: { statut: "jeune" } }
)
```

## Suppression de documents

### Supprimer un document

```bash
db.utilisateurs.deleteOne({ nom: "Jean" })
```

### Supprimer plusieurs documents

```bash
db.utilisateurs.deleteMany({ age: { $lt: 20 } })
```

## Agrégation

### Pipeline d'agrégation

```bash
db.utilisateurs.aggregate([
  { $match: { age: { $gte: 25 } } },
  { $group: { _id: "$ville", total: { $sum: 1 } } },
  { $sort: { total: -1 } }
])
```

## Index

### Créer un index

```bash
db.utilisateurs.createIndex({ nom: 1 })
db.utilisateurs.createIndex({ age: 1, nom: 1 })
db.utilisateurs.createIndex({ email: 1 }, { unique: true })
```

### Afficher les index

```bash
db.utilisateurs.getIndexes()
```

### Supprimer un index

```bash
db.utilisateurs.dropIndex("nom_1")
```

### Activer le sharding

```bash

```


## Importation et exportation de données

### Exporter avec mongodump

```bash
mongodump --db=mabasededonnees --out=/sauvegarde
```

### Importer avec mongorestore

```bash
mongorestore --db=mabasededonnees /sauvegarde/mabasededonnees
```

### Exporter avec mongoexport

```bash
mongoexport --db=mabasededonnees --collection=utilisateurs --out=utilisateurs.json
```

### Importer avec mongoimport

```bash
mongoimport --db=mabasededonnees --collection=utilisateurs --file=utilisateurs.json
```

## Transactions

MongoDB 4.0+ prend en charge les transactions multi-documents.

```bash
session = db.getMongo().startSession()
session.startTransaction()
try {
  session.getDatabase("maDB").comptes.updateOne(
    { compteId: 1 },
    { $inc: { solde: -100 } }
  )
  session.commitTransaction()
} catch (erreur) {
  session.abortTransaction()
} finally {
  session.endSession()
}
```

## Change Streams

Les change streams permettent aux applications d'écouter les changements de données en temps réel.

```bash
var changeStream = db.utilisateurs.watch()
while (changeStream.hasNext()) {
  printjson(changeStream.next())
}
```

## GridFS

GridFS est utilisé pour stocker les gros fichiers dépassant la limite de 16 Mo.

```bash
mongofiles put monfichier.txt
mongofiles list
mongofiles get monfichier.txt
mongofiles delete monfichier.txt
```
