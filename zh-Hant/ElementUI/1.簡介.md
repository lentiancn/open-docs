# Element UI 簡介

Element UI 是餓了麼團隊開發的 Vue 2 UI 組件庫。

## 什麼是 Element UI

Element UI 是一套為 Vue 2 設計的桌面端 UI 組件庫，用於快速構建企業級應用後台。

## 主要特性

- Vue 2 深度集成
- 70+ 組件
- 按需加載
- 國際化支持
- 主題定製

## 快速開始

### 安裝

```bash
npm install element-ui -S
```

### 使用

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```
