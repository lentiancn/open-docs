# jQuery 사용 가이드

jQuery를 사용하는 가이드입니다.

---

## 기본

```javascript
$('#myId')
$(document).ready(function() {})
```

---

## 선택자

```javascript
$('#myId')
$('.myClass')
$('div')
```

---

## 이벤트

```javascript
$('#myButton').click(function() {})
```

---

## AJAX

```javascript
$.get('url', function(data) {})
$.post('url', data, function(data) {})
```
