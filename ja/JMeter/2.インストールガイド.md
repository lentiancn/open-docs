# JMeter インストールガイド

このガイドでは、不同のオペレーティングシステムにApache JMeterをインストールして設定する方法を詳しく説明します。

## 環境要件

- **Java実行環境**：JMeterはJava 8以上が必要
- **オペレーティングシステム**：Linux、Windows、Mac OSXなどのJava対応OSをサポート
- **メモリ**：少なくとも4GBの空きメモリを推奨（テスト規模により更多が必要な場合あり）
- **ディスク容量**：少なくとも1GBの空き容量を推奨

## JMeterのダウンロード

1. JMeter公式ダウンロードページにアクセス：https://jmeter.apache.org/download_jmeter.cgi
2. 最新バージョンのJMeterバイナリ（Binaries）をダウンロード
3. ダウンロードの整合性を検証するためにPGPまたはSHA-512署名ファイルを同時にダウンロードすることを推奨

## Windows インストール手順

### 解凍インストール

1. ダウンロードしたApache JMeterアーカイブ（apache-jmeter-x.x.zip）を見つける
2. 適切なインストールディレクトリを選択（スペースや特殊文字を含むパスは避ける）
3. 選択したディレクトリにアーカイブを解凍
4. 解凍完了後、JMeterのインストールが完了

### インストール確認

1. JMeterインストールディレクトリのbinフォルダに移動
2. jmeter.bat（Windowsバッチファイル）をダブルクリック
3. 正常であれば、JMeter GUIインターフェースが起動

## LinuxとMac OSX インストール手順

### 解凍インストール

1. ダウンロードしたApache JMeterアーカイブ（apache-jmeter-x.x.tar.gz）を見つける
2. 次のコマンドで解凍：
   ```bash
   tar -xf apache-jmeter-x.x.tar.gz
   ```
3. 解凍完了後、JMeterのインストールが完了

### 環境変数の設定（オプション）

1. ターミナル設定ファイルを開く（~/.bashrcまたは~/.profile）
2. 次の環境変数を追加：
   ```bash
   export JMETER_HOME=/path/to/apache-jmeter-x.x
   export PATH=$PATH:$JMETER_HOME/bin:$JMETER_HOME/lib
   ```
3. 保存して設定ファイルをリロード：
   ```bash
   source ~/.bashrc
   ```

### インストール確認

1. ターミナルで次を実行：
   ```bash
   jmeter --version
   ```
2. バージョン情報が表示されれば、インストール成功

## JMeterの起動

### GUIモード

- **Windows**：binディレクトリのjmeter.batをダブルクリック
- **Linux/Mac**：ターミナルでjmeterコマンドを実行

### コマンドラインモード

負荷テストの場合、リソース使用量を減らすためにコマンドラインモードを推奨：

```bash
jmeter -n -t test_plan.jmx -l results.jtl -e -o /path/to/output
```

パラメータ説明：
- -n：非GUIモードで実行
- -t：テスト計画ファイルを指定
- -l：結果ファイルを指定
- -e：テスト終了後にレポートを生成
- -o：レポート出力ディレクトリを指定

## ディレクトリ構造の説明

- **bin**：実行ファイルと起動スクリプトを含む
- **lib**：JMeter依存のライブラリファイルを含む
- **docs**：JMeter公式ドキュメントを含む
- **extras**：追加のヘルパーツールを含む
- **printable_docs**：印刷可能な形式のマニュアルを含む

## 依存関係の管理

JMeterのlibディレクトリにはコア依存関係が含まれています。追加のプラグインやライブラリを追加する場合：

1. JARファイルをlibディレクトリにコピー
2. 一部のプラグインはlib/extディレクトリに配置する必要がある
3. 新しいプラグインを適用するにはJMeterを再起動

## 一般的なインストール問題

- **Javaが見つからない**：Javaが正しくインストールされ、JAVA_HOME環境変数が設定されていることを確認
- **メモリ不足**：bin/jmeter.shまたはjmeter.batのHeapサイズパラメータを変更して調整可能
- **起動失敗**：正しいバージョンのJava（Java 8以上）がインストールされているか確認
