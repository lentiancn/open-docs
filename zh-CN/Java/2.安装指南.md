# Java 安装指南

## 获取 Java 开发工具包

Oracle 官方提供 Java 开发工具包（JDK）的免费下载。当前主要的 LTS 版本包括 JDK 25、JDK 21、JDK 17、JDK 11 和 JDK 8。

官方下载页面：https://www.oracle.com/java/technologies/downloads/

## Windows 系统安装

### 使用安装程序

1. 访问 Oracle 官方下载页面，选择 Windows 版本的 JDK
2. 下载 x64 Installer（exe 文件）或 MSI Installer
3. 运行下载的安装程序
4. 按照安装向导的提示完成安装，默认安装路径为 C:\Program Files\Java\jdk-version
5. 安装完成后，验证安装是否成功

### 使用压缩包

1. 下载 x64 Compressed Archive（zip 文件）
2. 解压到指定目录，如 C:\Java\jdk-25
3. 配置环境变量

### 配置环境变量

1. 右键点击"此电脑"，选择"属性"
2. 点击"高级系统设置"
3. 点击"环境变量"
4. 在系统变量中，点击"新建"
   - 变量名：JAVA_HOME
   - 变量值：JDK 安装路径（如 C:\Program Files\Java\jdk-25）
5. 找到 Path 变量，点击"编辑"
6. 添加新变量：%JAVA_HOME%\bin
7. 点击确定保存

### 验证安装

打开命令提示符，输入以下命令：

```cmd
java -version
javac -version
```

如果显示版本信息，说明安装成功。

## Linux 系统安装

### 使用压缩包安装

1. 下载 Linux 版本的 JDK（tar.gz 格式）
2. 打开终端，解压到指定目录：

```bash
sudo mkdir -p /usr/local/java
sudo tar -zxf jdk-25_linux-x64_bin.tar.gz -C /usr/local/java
```

3. 配置环境变量，编辑 ~/.bashrc 或 /etc/profile：

```bash
export JAVA_HOME=/usr/local/java/jdk-25
export PATH=$JAVA_HOME/bin:$PATH
```

4. 使配置生效：

```bash
source ~/.bashrc
```

5. 验证安装：

```bash
java -version
```

### 使用包管理器安装

#### Ubuntu/Debian

```bash
sudo apt update
sudo apt install openjdk-21-jdk
```

#### CentOS/RHEL

```bash
sudo yum install java-21-openjdk
```

#### Fedora

```bash
sudo dnf install java-21-openjdk
```

## macOS 系统安装

### 使用 DMG 安装程序

1. 下载 macOS 版本的 JDK（dmg 文件）
2. 打开下载的 dmg 文件
3. 将 JDK 拖入 Applications 文件夹
4. 打开终端，配置环境变量：

```bash
export JAVA_HOME=$(/usr/libexec/java_home)
export PATH=$JAVA_HOME/bin:$PATH
```

5. 将以上内容添加到 ~/.zshrc 或 ~/.bash_profile
6. 验证安装：

```bash
java -version
```

### 使用 Homebrew

```bash
brew install openjdk@21
sudo ln -sfn $(brew --prefix)/opt/openjdk@21/libexec/openjdk /Library/Java/JavaVirtualMachines/openjdk-21.jdk
```

## 多个 JDK 版本共存

### Linux 下使用 update-alternatives

```bash
sudo update-alternatives --install "/usr/bin/java" "java" "/usr/local/java/jdk-21/bin/java" 1
sudo update-alternatives --install "/usr/bin/java" "java" "/usr/local/java/jdk-17/bin/java" 2
sudo update-alternatives --config java
```

### Windows 下切换版本

修改 JAVA_HOME 环境变量指向不同的 JDK 目录即可切换版本。

### 使用 SDKMAN 管理多版本

```bash
curl -s "https://get.sdkman.io" | bash
sdk install java 21.0.5-tem
sdk install java 17.0.5-tem
sdk install java 11.0.21-tem
```

```bash

```

## 常见安装问题

### 问题一：java 命令不可用

**解决方法**：检查 Path 环境变量是否包含 JDK 的 bin 目录。

### 问题二：权限不足

**解决方法**：Linux/macOS 上使用 sudo 提权，或检查文件权限。

### 问题三：端口被占用

**解决方法**：Java 应用默认使用 8080 端口，确保端口未被其他程序占用。

### 问题四：JDK 和 JRE 的区别

- **JDK**（Java Development Kit）：Java 开发工具包，包含开发工具和 JRE
- **JRE**（Java Runtime Environment）：Java 运行时环境，仅包含运行 Java 程序所需的组件

开发 Java 程序需要安装 JDK，运行已编译的 Java 程序只需 JRE。

## 安装后配置建议

1. 配置 MAVEN_HOME 或 GRADLE_HOME：如果使用 Maven 或 Gradle
2. 配置 M2_HOME：设置 Maven 本地仓库路径
3. 配置 HTTP 代理：如果在企业环境中，需要配置代理才能访问外网
4. 设置时区：确保 Java 应用程序使用正确的时区

## 卸载 Java

### Windows 控制面板卸载

1. 打开控制面板
2. 选择"程序和功能"
3. 找到 Java 相关程序
4. 右键选择"卸载"

### Linux 卸载

```bash
sudo apt remove openjdk-*  # Debian/Ubuntu
sudo yum remove java-*    # CentOS/RHEL
```

### macOS 卸载

```bash
sudo rm -rf /Library/Java/JavaVirtualMachines/jdk-*
```
