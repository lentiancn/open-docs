# APISIX

Cloud-native API gateway.

## Overview

Apache API gateway with high performance.

### Features

- High performance
- Dynamic routing
- Plugin system

## Quick Start

### Docker

```bash
docker run -d -p 9080:9080 apache/apisix:latest
```
