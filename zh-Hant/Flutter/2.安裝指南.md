# Flutter 安裝指南

完整的 Flutter 開發環境安裝和配置指南。

## 環境要求

### Windows

- **操作系統**：Windows 10 或更高版本（64位）
- **磁盤空間**：至少 2.8 GB
- **工具**：PowerShell 5.0+、Git

### macOS

- **操作系統**：macOS 10.14+ 
- **磁盤空間**：至少 2.8 GB
- **工具**：Command Line Tools、Git

### Linux

- **操作系統**：Ubuntu 18.04+
- **磁盤空間**：至少 2.8 GB
- **依賴**：bash, curl, git, unzip, xz-utils, zip, libglu1-mesa

## 安裝步驟

### Windows 安裝

1. **下載 Flutter SDK**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

2. **配置環境變量**

```powershell
[Environment]::SetEnvironmentVariable(
    "Path",
    $env:Path + ";C:\src\flutter\bin",
    "User"
)
```

3. **驗證安裝**

```bash
flutter doctor
```

### macOS 安裝

1. **下載 Flutter SDK**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

2. **配置環境變量**

```bash
export PATH="$PATH:/Users/用戶名/flutter/bin"
```

3. **運行 flutter doctor**

```bash
flutter doctor
```

### Linux 安裝

1. **安裝依賴**

```bash
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa
```

2. **下載 Flutter SDK**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

3. **配置 PATH**

```bash
export PATH="$PATH:/home/用戶名/flutter/bin"
```

## Android 開發環境配置

### 安裝 Android Studio

1. 下載 Android Studio
2. 安裝並啟動
3. 配置 SDK

### 接受許可證

```bash
flutter doctor --android-licenses
```

## IDE 配置

### VS Code

1. 安裝 Flutter 擴展
2. 安裝 Dart 擴展

### Android Studio

1. 安裝 Flutter 插件
2. 安裝 Dart 插件

## 驗證安裝

```bash
flutter --version
flutter doctor
flutter upgrade
```

## 常見問題

### 下載失敗？

使用鏡像：
```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
```

### Android SDK 找不到？

```bash
flutter config --android-sdk /path/to/android-sdk
```
