# Element UI FAQ

## Установка

### Q: Установка не удалась?

**Решения:**
1. Проверить версию Node.js
2. Очистить кэш npm
```bash
npm cache clean --force
```

### Q: Версия Vue несовместима?

**Решения:**
Element UI 2.x требует Vue 2.6.0+

## Использование

### Q: Компоненты не отображаются?

**Решения:**
Проверить импорт:
```javascript
import { Button } from 'element-ui'
Vue.use(Button)
```

### Q: Стили не применяются?

**Решения:**
Импортировать файл стилей:
```javascript
import 'element-ui/lib/theme-chalk/index.css'
```
