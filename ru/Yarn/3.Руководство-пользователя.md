# Руководство пользователя

## Повседневное использование

### Инициализация проекта

```bash
yarn init -2
```

### Установка зависимостей

```bash
yarn install
```

### Добавление зависимостей

```bash
yarn add lodash
yarn add -D typescript
```

### Удаление

```bash
yarn remove lodash
```

### Обновление

```bash
yarn up lodash
yarn up --all
```

## Команды

| Команда | Описание |
|---------|----------|
| yarn init | Инициализировать |
| yarn install | Установить |
| yarn add | Добавить |
| yarn remove | Удалить |
| yarn up | Обновить |
