# OpenCode 설치 가이드

OpenCode는 다양한 설치 방법을 지원합니다.

## Linux/macOS 설치

### 권장: 설치 스크립트

```bash
curl -fsSL https://opencode.ai/install | bash
```

### Node.js

```bash
npm install -g opencode-ai
```

### Bun

```bash
bun install -g opencode-ai
```

### pnpm

```bash
pnpm install -g opencode-ai
```

### Yarn

```bash
yarn global add opencode-ai
```

### Homebrew (macOS/Linux)

```bash
brew install anomalyco/tap/opencode
```

### Arch Linux

```bash
# 안정판
sudo pacman -S opencode

# 최신판 (AUR)
paru -S opencode-bin
```

## Windows 설치

### Chocolatey

```bash
choco install opencode
```

### Scoop

```bash
scoop install opencode
```

### NPM

```bash
npm install -g opencode-ai
```

### Mise

```bash
mise use -g github:anomalyco/opencode
```

### Docker

```bash
docker run -it --rm ghcr.io/anomalyco/opencode
```

### 수동 다운로드

[GitHub Releases](https://github.com/anomalyco/opencode/releases)에서 바이너리를 다운로드할 수도 있습니다.

## 설치 확인

```bash
opencode --version
```

## 설정

### LLM 제공자 연결

연결하려면 다음을 실행:

```bash
opencode
/connect
```

`opencode`를 선택하고 [opencode.ai/auth](https://opencode.ai/auth)를 방문하여 로그인하고 API 키를 얻으세요.

### 다른 제공자

다른 LLM 제공자를 선택할 수도 있습니다. 자세한 내용은 [공식 문서](https://opencode.ai/docs/providers#directory)를 참조하세요.

## 프로젝트 초기화

```bash
cd /path/to/project
opencode
/init
```

이렇게 하면 OpenCode가 프로젝트를 분석하고 프로젝트 루트에 AGENTS.md 파일을 생성합니다.
