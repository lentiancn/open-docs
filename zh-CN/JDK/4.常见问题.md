# JDK 常见问题

## 环境配置

### 问题：java 命令不可用

**解决方案**：
1. 确认 JDK 已正确安装
2. 配置环境变量 JAVA_HOME 和 PATH
3. 重新打开终端

### 问题：JAVA_HOME 指向 JRE 而非 JDK

**解决方案**：
确保 JAVA_HOME 指向 JDK 安装目录（包含 bin、lib 等文件夹），而非 JRE 目录。

## 版本问题

### 问题：运行 Java 程序提示版本不兼容

**解决方案**：
```bash
# 查看当前 Java 版本
java -version

# 检查编译时版本
javap -v MyClass | findstr "major"
```

### 问题：多个 JDK 版本切换

```bash
# Linux
update-alternatives --config java

# macOS
# 修改 JAVA_HOME 指向不同版本
```

## 性能问题

### 问题：内存溢出

**解决方案**：
调整堆内存大小：
```bash
java -Xms256m -Xmx1024m -XX:+HeapDumpOnOutOfMemoryError MyApp
```

### 问题：GC 频繁

**解决方案**：
选择合适的垃圾收集器：
```bash
# G1GC
java -XX:+UseG1GC MyApp

# ZGC（低延迟）
java -XX:+UseZGC MyApp
```

## 兼容性问题

### 问题：程序在旧 JDK 运行正常，新 JDK 报错

检查是否使用了已废弃的 API，查看官方迁移指南。
