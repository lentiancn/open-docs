# OpenClaw Manual de Usuario

> Guía completa para usar OpenClaw

---

## Primeros Pasos

### Uso Básico

OpenClaw puede usarse de varias maneras:

#### Sesión Interactiva

```bash
# Iniciar una conversación interactiva
openclaw chat
```

#### Ejecución de Comandos

```bash
# Ejecutar un solo comando
openclaw exec "ayúdame a crear un archivo"

# Ejecutar tareas más complejas
openclaw exec --task "investigar noticias actuales"
```

### Gestión del Gateway

El servicio Gateway debe estar activo para la mayoría de operaciones:

```bash
# Iniciar Gateway
openclaw gateway start

# Verificar estado
openclaw gateway status

# Detener Gateway
openclaw gateway stop
```

## Gestión de Archivos

### Leer Archivos

```bash
# Usar la herramienta de lectura
read /ruta/al/archivo.txt
```

### Escribir Archivos

```bash
# Crear nuevo archivo o sobrescribir
write --content "Contenido del archivo" --path /ruta/al/archivo.txt
```

### Editar Archivos

```ambio de texto exacto en un archivo
edit --path /ruta/al/archivo.txt --oldText "texto antiguo" --newText "texto nuevo"
```

## Control del Navegador

OpenClaw puede automatizar navegadores web:

### Iniciar Navegador

```bash
# Iniciar navegador aislado
browser start --profile openclaw

# Usar extensión de Chrome
browser start --profile chrome
```

### Acciones del Navegador

```bash
# Navegar a URL
browser navigate --url https://ejemplo.com

# Tomar captura de pantalla
browser screenshot --path /salida/captura.png

# Hacer clic en elemento
browser click --selector "#boton-id"
```

## Acceso Web

### Búsqueda Web

```bash
# Buscar en internet
web_search --query "noticias actuales" --count 5
```

### Obtener Página Web

```bash
# Recuperar contenido de una página web
web_fetch --url https://ejemplo.com --maxChars 5000
```

## Mensajería

### Enviar Mensaje

```bash
# Enviar mensaje a través de plataforma de mensajería
message send --target usuario --message "¡Hola!"
```

## Subagentes

### Crear Subagentes

```bash
# Iniciar nuevo subagente para tareas específicas
subagents create --task "recopilación de documentación"

# Dirigir subagente
subagents steer --id agent-123 --message "siguiente tarea"

# Finalizar subagente
subagents kill --id agent-123
```

## Configuración y Personalización

### Personalizar Personalidad

Edita `SOUL.md` para personalizar la personalidad del asistente:

```markdown
# SOUL.md - Quién eres

## Verdades Fundamentales

- Sé genuinamente útil
- Ten opiniones
- Sé ingenioso
```

### Configurar Herramientas

Edita `TOOLS.md` para configuraciones específicas de herramientas:

```markdown
# TOOLS.md - Notas Locales

## Conexiones SSH
- servidor1 -> 192.168.1.100
```

## Mejores Prácticas

1. **Espacio de Trabajo**: Mantén tus archivos en el workspace definido
2. **Seguridad**: Ten cuidado con acciones externas
3. **Memoria**: Usa archivos de memoria para información importante
4. **Gateway**: Mantén el servicio Gateway activo para tiempos de respuesta rápidos

---

## Características Avanzadas

### Automatización

Crea scripts para automatizar tareas recurrentes:

```bash
# Recolección automatizada de datos
openclaw exec --task "recopilar información sobre tema X"
```

### Trabajos Cron

Configura tareas periódicas:

```bash
# Crear tarea programada
openclaw cron --schedule "0 9 * * *" --task "informe matutino"
```
