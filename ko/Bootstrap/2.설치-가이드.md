# 설치 가이드

이 가이드에서는 Bootstrap을 설치하는 다양한 방법을 설명합니다. 이를 통해 신속하게 개발을 시작할 수 있습니다.

## 필요 사항

Bootstrap 5에는 다음이 필요합니다:

- **HTML5**를 지원하는 최신 브라우저
- **CSS3** 지원
- **JavaScript** ES6+ (JavaScript 플러그인을 사용하는 경우)

## 설치 옵션

### 옵션 1: CDN (권장)

가장 빠른 방법은 CDN에서 Bootstrap의 CSS와 JavaScript를 직접 포함하는 것입니다:

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 페이지</title>
    <!-- Bootstrap CSS -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>안녕하세요, Bootstrap!</h1>

    <!-- Bootstrap JavaScript (Popper 포함) -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### 옵션 2: npm 설치

npm을 사용하여 프로젝트 의존성을 관리하는 경우:

```bash
# Bootstrap 설치
npm install bootstrap@5

# Popper 설치 (드롭다운, 툴팁 등에 필요)
npm install @popperjs/core
```

그런 다음 프로젝트에서 가져오기:

```javascript
// Bootstrap의 CSS 가져오기
import 'bootstrap/dist/css/bootstrap.min.css';

// Bootstrap의 JS 가져오기
import 'bootstrap/dist/js/bootstrap.bundle.min.js';
```

### 옵션 3: 소스 파일 다운로드

공식 웹사이트에서 사전 컴파일된 파일을 가져오세요:

1. https://getbootstrap.com/으로 이동
2. "Download" 버튼 클릭
3. "Compiled CSS and JS" 선택
4. 압축을 풀고 css와 js 폴더를 프로젝트에 복사

디렉터리 구조는 다음과 같습니다:

```
your-project/
├── css/
│   └── bootstrap.min.css
├── js/
│   └── bootstrap.bundle.min.js
└── index.html
```

### 옵션 4: Sass 소스

Bootstrap의 스타일을 사용자 정의해야 하는 경우 Sass 소스로 작업할 수 있습니다:

```bash
npm install bootstrap
```

그런 다음 Sass 파일에서 가져오기:

```scss
// 사용자 정의 변수 (Bootstrap 가져오기 전에 정의)
$primary: #007bff;
$body-bg: #f8f9fa;

// Bootstrap 가져오기
@import "~bootstrap/scss/bootstrap";
```

## 개발 도구

### VS Code 확장

유용한 확장 프로그램:

- **Bootstrap 5 Snippets** — 즉시 사용 가능한 컴포넌트 코드 스니펫 제공
- **HTML CSS Support** — CSS 클래스 이름 자동 완성
- **Live Server** — 실시간으로 페이지 미리보기

### 온라인 편집기

- **Bootstrap Builder** — 시각적 드래그 앤 드롭 Bootstrap 페이지 빌더
- **Bootstrap Studio** — 전문 Bootstrap 디자인 도구

                    <div class="card-body">

## 어떤 버전을 사용해야 하나요

### 프로덕션

안정 릴리스 사용:

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### 개발

최신 개발 버전 사용 (소스 맵 포함):

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.js"></script>
```

### 중국 CDN 미러

중국에서 더 빠르게 로드하려면:

```html
<link href="https://cdn.bootcss.com/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.bootcss.com/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

## 제거

Bootstrap을 제거해야 하는 경우 (파일만 삭제):

- HTML에서 CSS 및 JS 링크 제거
- npm 사용 시 node_modules에서 bootstrap 폴더 삭제
- 로컬로 다운로드한 CSS 및 JS 파일 삭제
