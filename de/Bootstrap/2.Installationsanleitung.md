# Installationsanleitung

Hier erfährst du, wie du Bootstrap auf verschiedenen Wegen installierst – damit du schnell loslegen kannst.

## Voraussetzungen

Bootstrap 5 braucht folgende Umgebung:

- **HTML5** in einem modernen Browser
- **CSS3** Unterstützung
- **JavaScript** ES6+ (falls du JavaScript-Plugins nutzt)

## Installationsmethoden

### Methode 1: CDN einbinden (empfohlen)

Der einfachste Weg: Bindet Bootstrap CSS und JavaScript direkt über ein CDN ein:

```html
<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap Seite</title>
    <!-- Bootstrap CSS einbinden -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>Hallo, Bootstrap!</h1>

    <!-- Bootstrap JavaScript (inkl. Popper) einbinden -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### Methode 2: npm Installation

Falls du npm für deine Abhängigkeiten nutzt:

```bash
# Bootstrap installieren
npm install bootstrap@5

# Popper installieren (für Dropdowns, Tooltips etc.)
npm install @popperjs/core
```

Dann im Projekt einbinden:

```javascript
# Bootstrap CSS einbinden
import 'bootstrap/dist/css/bootstrap.min.css';

# Bootstrap JS einbinden
import 'bootstrap/dist/js/bootstrap.bundle.min.js';
```

### Methode 3: Quellcode herunterladen

Lade die vorkompilierten Dateien von der offiziellen Website:

1. Gehe auf https://getbootstrap.com/
2. Klicke auf „Download"
3. Wähle „Compiled CSS and JS" zum Herunterladen
4. Entpacken und die css- und js-Ordner ins Projekt kopieren

Verzeichnisstruktur:

```
dein-projekt/
├── css/
│   └── bootstrap.min.css
├── js/
│   └── bootstrap.bundle.min.js
└── index.html
```

### Methode 4: Sass-Quellcode

Falls du Bootstraps Styles anpassen möchtest, lade den Sass-Quellcode:

```bash
npm install bootstrap
```

Dann in der Sass-Datei einbinden:

```scss
# Eigene Variablen (vor Bootstrap einbinden definieren)
$primary: #007bff;
$body-bg: #f8f9fa;

# Bootstrap einbinden
@import "~bootstrap/scss/bootstrap";
```

## Schnellstart

### Erste Seite erstellen

```html
<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap Schnellstart</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <!-- Navigationsbereich -->
    <nav class="navbar navbar-expand-lg navbar-dark bg-primary">
        <div class="container">
            <a class="navbar-brand" href="#">Meine Website</a>
        </div>
    </nav>

    <!-- Hauptinhalt -->
    <main class="container mt-5">
        <div class="row">
            <div class="col-md-6">
                <h1>Willkommen bei Bootstrap</h1>
                <p class="lead">Eine responsive Seite mit Bootstrap 5.</p>
                <button class="btn btn-primary">Mehr erfahren</button>
            </div>
            <div class="col-md-6">
                <div class="card">
                    <div class="card-body">
                        <h5 class="card-title">Features</h5>
                        <ul>
                            <li>Responsives Layout</li>
                            <li>Umfangreiche Komponenten</li>
                            <li>Leistungsstarke JavaScript-Plugins</li>
                        </ul>
                    </div>
                </div>
            </div>
        </div>
    </main>

    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## Versionswahl

### Produktionsumgebung

Nutze die stabile Version:

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### Entwicklungsumgebung

Nutze die neueste Entwicklerversion (mit Quellcode):

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.js"></script>
```

### Chinesischer Spiegel

In China kannst du einen chinesischen Spiegel für schnellere Ladezeiten nutzen:

```html
<link href="https://cdn.bootcss.com/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.bootcss.com/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

## Deinstallieren

Falls du Bootstrap entfernen möchtest (einfach die Dateien löschen):

- Entferne die CSS- und JS-Einbindungen aus dem HTML
- Lösche das bootstrap-Verzeichnis aus node_modules (falls per npm installiert)
- Lösche die lokalen CSS- und JS-Dateien
