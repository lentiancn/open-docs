# Angular FAQ

Часто задаваемые вопросы.

## Установка

### Q: Установка Angular CLI не удалась?

**Решения**:
1. Очистить кэш npm: `npm cache clean --force`
2. Использовать права администратора
3. Проверить версию Node.js

### Q: Команда ng не найдена?

**Решения**:
1. Angular CLI установлен глобально?
2. Проверить переменную PATH
3. Попробовать переустановить

### Q: Несовместимая версия Node.js?

**Решения**:
1. Использовать nvm для управления версиями Node.js
2. Обновить Node.js до рекомендуемой версии

## Использование

### Q: Стили компонента не работают?

**Решения**:
1. Путь styleUrls правильный?
2. CSS-селекторы правильные?
3. Проверить настройки ViewEncapsulation

### Q: Роутинг не работает?

**Решения**:
1. Конфигурация роутов правильная?
2. RouterModule импортирован?
3. Проверить синтаксис routerLink

### Q: HTTP-запросы не работают?

**Решения**:
1. HttpClientModule импортирован?
2. Проверить настройки CORS
3. API URL правильный?

### Q: Валидация формы не работает?

**Решения**:
1. ReactiveFormsModule импортирован?
2. Проверить конфигурацию Validators
3. Правильно использовать formControlName

## Производительность

### Q: Приложение загружается медленно?

**Решения**:
1. Включить ленивую загрузку
2. Использовать обнаружение изменений OnPush
3. Сжать изображения и ресурсы
4. Собирать в продакшен-режиме

### Q: Утечки памяти?

**Решения**:
1. Отписываться от Observable
2. Очищать интервалы
3. Удалять обработчики событий

## Версия

### Q: Как обновить Angular?

```bash
ng update @angular/core @angular/cli
```

### Q: Миграция с AngularJS на Angular?

Использовать инструмент angular-upgrade для постепенной миграции.
