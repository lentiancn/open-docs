# JUnit Einführung

JUnit ist das beliebteste Unit-Test-Framework für Java, das zum Schreiben und Ausführen von wiederholbaren Tests verwendet wird. Es ist die Grundlage für die Entwicklung von hochwertigem Java-Code und wird weltweit von Millionen von Entwicklern verwendet.

## JUnit 4 vs JUnit 5

### Versionsübersicht

- **JUnit 4**: Veröffentlicht 2010, die letzte Hauptversion des Test-Frameworks, das lange Zeit von der Java-Community verwendet wurde
- **JUnit 5**: Veröffentlicht 2017, eine vollständige Überarbeitung des JUnit-Frameworks, bestehend aus drei Teilprojekten: JUnit Platform, JUnit Jupiter und JUnit Vintage

### Architektonische Unterschiede

JUnit 4 verwendet eine monolithische Architektur mit allen Funktionen in einer einzigen Bibliothek. JUnit 5 verwendet eine modulare Architektur mit drei separaten Projekten:

- **JUnit Platform**: Die Grundlage für das Starten von Test-Frameworks, bietet die TestEngine-API und den Konsolen-Launcher
- **JUnit Jupiter**: Das neue Programmiermodell und Erweiterungsmodell zum Schreiben von Tests und Erweiterungen
- **JUnit Vintage**: Die Kompatibilitäts-Engine, die das Ausführen von JUnit 3 und JUnit 4 Tests auf der JUnit 5 Plattform ermöglicht

### Hauptunterschiede

| Funktion | JUnit 4 | JUnit 5 |
|----------|---------|---------|
| Minimale Java-Version | Java 5+ | Java 8+ |
| Annotations-Paket | org.junit | org.junit.jupiter.api |
| Testklasse ohne Vererbung | Nein | Ja |
| Assertions-Stil | Assert statische Methoden | Assertions statische Methoden |
| Parametrisierte Tests | @Parameters | @ParameterizedTest |
| Verschachtelte Tests | Nicht unterstützt | Unterstützt |
| Dynamische Tests | Nicht unterstützt | Unterstützt |
| Parallele Ausführung | Begrenzte Unterstützung | Native Unterstützung |
| Erweiterungsmodell | @RunWith | @ExtendWith |

### Versionsempfehlung

- **Neue Projekte**: Verwenden Sie JUnit 5 für moderne Funktionen und bessere Erweiterbarkeit
- **Legacy-Projekte**: Verwenden Sie weiterhin JUnit 4 oder JUnit Vintage für schrittweise Migration
- **Langzeitpflege**: Migrieren Sie schrittweise zu JUnit 5 für bessere Test-Erfahrung
