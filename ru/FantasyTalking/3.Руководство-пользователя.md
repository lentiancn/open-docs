# FantasyTalking Руководство пользователя

## Основные функции

### Синтез речи

```javascript
const audio = await client.speak('Привет, добро пожаловать в FantasyTalking');

// Указать стиль
const audio = await client.speak('Сегодня хорошая погода', {
  voice: { style: 'happy', language: 'ru-RU' }
});

// Указать скорость и высоту тона
const audio = await client.speak('Пожалуйста, подождите', {
  voice: { speed: 1.2, pitch: 0.9 }
});
```

### События

```javascript
client.on('speech-start', () => console.log('Начало'));
client.on('speech-end', () => console.log('Конец'));
client.on('error', (e) => console.error('Ошибка:', e));
```

### Пакетный синтез

```javascript
const texts = ['Первый голос', 'Второй голос', 'Третий голос'];
const results = await Promise.all(texts.map(t => client.speak(t)));
```

## Функции диалога

### Базовый диалог

```javascript
const session = client.createSession({
  systemPrompt: 'Вы дружелюбный ассистент',
  language: 'ru-RU'
});

const response = await session.chat('Какая погода сегодня?');
console.log(response.text);
```

## Продвинутые функции

### Потоковая передача

```javascript
const stream = await client.speakStream('Это длинный текст');
stream.on('data', chunk => player.play(chunk));
```

## Рекомендации

1. Повторное использование подключений
2. Пакетная обработка
3. Кэширование
4. Потоковая передача для длинных текстов

### Обработка ошибок

```javascript
try {
  const audio = await client.speak('Тест');
} catch (error) {
  if (error.code === 'RATE_LIMIT') {
    console.log('Слишком много запросов');
  }
}
```
