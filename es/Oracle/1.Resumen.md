# Oracle Base de Datos

> Sistema de gestión de bases de datos relacional de clase empresarial

---

## Descripción

Oracle Database es un sistema de gestión de bases de datos relacional patentado desarrollado por Oracle Corporation. Es uno de los sistemas de bases de datos empresariales más utilizados del mundo.

---

## Características

- **Alto Rendimiento**: Procesamiento de consultas optimizado
- **Escalabilidad**: Soporte para grandes volúmenes de datos
- **Seguridad**: Funciones de seguridad avanzadas
- **Alta Disponibilidad**: Real Application Clusters (RAC)
- **PL/SQL**: Extensión de lenguaje procedimental
