# IBM Informix

IBM Informix is a family of relational database management systems (RDBMS) and multi-model database offerings developed by IBM.

## Documentation Overview

This directory contains IBM Informix technical documentation:

- **Introduction**: Product overview, features, and editions
- **Installation Guide**: Linux and Windows installation steps
- **User Manual**: Basic commands, SQL statements, and configuration
- **FAQ**: Common questions and solutions
