# OpenClaw 使用手冊

## 快速開始

```bash
# 啟動 Gateway
openclaw gateway --port 18789

# 開啟控制台
openclaw dashboard
```

## 通道管理

```bash
# 列出通道
openclaw channels list

# 新增通道
openclaw channels add whatsapp

# 登入
openclaw channels login
```

## 訊息管理

```bash
# 發送訊息
openclaw message send --target +15555550123 --message "你好"

# 列出會話
openclaw sessions list
```

## 配置管理

```bash
# 查看配置
openclaw config get

# 設定配置
openclaw config set channels.whatsapp.allowFrom "+15555550123"
```

## 故障排除

```bash
# 查看日誌
openclaw logs

# 檢查狀態
openclaw status
```

## 常用命令

| 命令 | 說明 |
|------|------|
| `openclaw gateway start` | 啟動 Gateway |
| `openclaw dashboard` | 開啟控制台 |
| `openclaw channels list` | 列出通道 |
| `openclaw sessions list` | 列出會話 |
| `openclaw message send` | 發送訊息 |
