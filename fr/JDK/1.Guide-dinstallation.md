# Guide d'installation JDK

JDK (Java Development Kit) est le kit de développement Java officiel. Ce guide couvre les méthodes d'installation pour différents environnements.

## Informations sur les versions

| Version | Statut | Description |
|---------|--------|-------------|
| JDK 21 (LTS) | ✅ Actuelle | Dernière version LTS |
| JDK 17 (LTS) | ✅ Supportée | LTS précédente |
| JDK 11 (LTS) | ✅ Supportée | LTS héritée |
| JDK 8 | ✅ Supportée | Version héritée |
| JDK 22 | ✅ Dernière | Dernière version avec nouvelles fonctionnalités |

### Versions LTS vs Versions fonctionnalités

- **LTS (Long Term Support)**: 2-3 ans de mises à jour, recommandé pour la production
- **Versions fonctionnalités**: Nouvelles fonctionnalités tous les 6 mois, support plus court

## Méthodes d'installation

### Méthode 1: Oracle JDK

#### Windows

1. Téléchargez depuis: https://www.oracle.com/java/technologies/downloads/
2. Exécutez l'installateur (.exe)
3. Suivez l'assistant

#### macOS

```bash
# Utiliser Homebrew
brew install openjdk@21
```

#### Linux

```bash
# Télécharger tar.gz
wget https://download.oracle.com/java/21/latest/jdk-21_linux-x64_bin.tar.gz
tar -xzf jdk-21_linux-x64_bin.tar.gz
sudo mv jdk-21 /opt/jdk-21
```

### Méthode 2: Eclipse Temurin (Adoptium)

#### Windows

1. Téléchargez depuis: https://adoptium.net/temurin/releases/
2. Exécutez l'installateur

#### macOS

```bash
brew install --cask temurin
```

### Méthode 3: SDKMAN

```bash
# Installer SDKMAN
curl -s "https://get.sdkman.io" | bash

# Installer JDK
sdk install java 21.0.2-tem
```

### Méthode 4: Docker

```dockerfile
FROM eclipse-temurin:21-jdk
WORKDIR /app
COPY . .
RUN javac Main.java
CMD ["java", "Main"]
```

### Méthode 5: Gestionnaires de packages Windows

```powershell
choco install openjdk21
```

## Configuration des variables d'environnement

### Windows

```powershell
[System.Environment]::SetEnvironmentVariable("JAVA_HOME", "C:\Program Files\Java\jdk-21", "User")
[System.Environment]::SetEnvironmentVariable("PATH", "$env:PATH;C:\Program Files\Java\jdk-21\bin", "User")
```

### macOS/Linux

```bash
export JAVA_HOME=/opt/jdk-21
export PATH=$JAVA_HOME/bin:$PATH
source ~/.bashrc
```

## Vérification

```bash
java -version
javac -version
echo $JAVA_HOME
```

## Versions multiples JDK

```bash
sdk install java 17.0.8-tem
sdk install java 21.0.2-tem
sdk use java 17.0.8-tem
```

## Problèmes courants

### "java" non trouvé?

Vérifiez que PATH inclut le répertoire bin Java.

### Mauvaise version Java?

```bash
which java
java -version
```
