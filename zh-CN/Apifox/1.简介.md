# Apifox 简介

API 一体化协作平台。

## 什么是 Apifox？

Apifox 是 API 一体化协作平台，集 API 文档、API 调试、API Mock、API 自动化测试于一体。

### 主要特性

- **API 文档** - 自动生成 API 文档，支持团队协作
- **API 调试** - 完整的 HTTP 请求模拟工具
- **Mock 服务** - 快速创建 Mock 服务器
- **自动化测试** - API 自动化测试功能
- **团队协作** - 团队项目管理、权限控制
- **数据同步** - 实时同步 API 变更
- **导入/导出** - 支持多种格式导入导出

### 统计数据

- 用户数：数百万开发者
- API 管理：数亿个 API
