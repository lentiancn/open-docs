# JMeter Installationsanleitung

## Voraussetzungen

- **Java Development Kit (JDK)**: JMeter erfordert Java 8 oder höher
- **Betriebssystem**: Windows, Linux oder macOS
- **Speicher**: Mindestens 4GB RAM für Lasttests empfohlen

## Installationsschritte

### 1. JMeter herunterladen

Besuchen Sie die offizielle Apache JMeter Website: https://jmeter.apache.org/

Laden Sie die neueste Version herunter (normalerweise eine .zip oder .tar.gz Datei).

### 2. Archiv entpacken

Entpacken Sie die heruntergeladene Datei an Ihren gewünschten Speicherort:

```bash
# Linux/Mac
tar -xvf apache-jmeter-x.x.x.tgz

# Windows
# Verwenden Sie 7-Zip oder WinRAR zum Entpacken
```

### 3. Umgebungsvariablen einrichten (Optional)

Der Einfachheit halber können Sie JMeter zu Ihrem System-PFAD hinzufügen:

```bash
# Linux/Mac (zu ~/.bashrc oder ~/.zshrc hinzufügen)
export JMETER_HOME=/path/to/apache-jmeter-x.x.x
export PATH=$PATH:$JMETER_HOME/bin

# Windows
# Zu Systemeigenschaften -> Umgebungsvariablen -> PFAD hinzufügen
```

### 4. Installation überprüfen

Führen Sie den folgenden Befehl aus, um zu überprüfen, ob JMeter korrekt installiert ist:

```bash
# Linux/Mac
./jmeter.sh --version

# Windows
jmeter.bat --version
```

### 5. JMeter GUI starten

```bash
# Linux/Mac
./jmeter.sh

# Windows
jmeter.bat
```

## Verzeichnisstruktur

```
apache-jmeter/
├── bin/          # Ausführbare Skripte und JMeter-Programm
├── lib/          # JMeter-Bibliotheken
├── lib/ext/      # Plugins und Erweiterungen
├── docs/         # Dokumentation
├── printable_docs/  # Benutzerhandbuch
└── LICENSE       # Lizenzdatei
```

## Ausführung in verschiedenen Modi

### GUI-Modus (Interaktiv)
```bash
jmeter.sh          # Linux/Mac
jmeter.bat         # Windows
```

### CLI-Modus (Headless)
```bash
jmeter -n -t testplan.jmx -l results.jtl -e -o html-report
```

Parameter:
- `-n`: Im Non-GUI-Modus ausführen
- `-t`: Testplandatei
- `-l`: Ergebnistdatei
- `-e`: HTML-Bericht nach dem Test generieren
- `-o`: Ausgabeverzeichnis für HTML-Bericht

## Installationsprobleme

### Java nicht gefunden
Stellen Sie sicher, dass JAVA_HOME korrekt eingestellt ist:
```bash
export JAVA_HOME=/path/to/jdk
```

### Speicherprobleme
Bearbeiten Sie das `jmeter` Skript, um die Heap-Größe zu erhöhen:
```bash
HEAP="-Xms1g -Xmx4g"
```

## Nächste Schritte

Nach der Installation fahren Sie mit dem Benutzerhandbuch fort, um zu lernen, wie Sie Testpläne erstellen und ausführen.
