# Flutter - Guide d'Installation

Guide complet de configuration de l'environnement de développement Flutter.

## Configuration Requise

### Windows

- **Système d'exploitation**: Windows 10 ou supérieur (64 bits)
- **Espace disque**: Minimum 2.8 Go
- **Outils**: PowerShell 5.0+, Git pour Windows

### macOS

- **Système d'exploitation**: macOS 10.14 (Mojave) ou supérieur
- **Espace disque**: Minimum 2.8 Go
- **Outils**: Command Line Tools, Git

### Linux

- **Système d'exploitation**: Ubuntu 18.04 ou supérieur
- **Espace disque**: Minimum 2.8 Go
- **Dépendances**: bash, curl, git, unzip, xz-utils, zip, libglu1-mesa

## Étapes d'Installation

### Installation Windows

1. **Télécharger Flutter SDK**

Visitez le site web Flutter pour télécharger la dernière version :
```bash
# Ou cloner via git
git clone https://github.com/flutter/flutter.git -b stable
```

2. **Configurer les variables d'environnement**

Ajoutez Flutter au PATH système :
```powershell
# Exécuter dans PowerShell
[Environment]::SetEnvironmentVariable(
    "Path",
    $env:Path + ";C:\src\flutter\bin",
    "User"
)
```

3. **Vérifier l'installation**

```bash
flutter doctor
```

### Installation macOS

1. **Télécharger Flutter SDK**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

2. **Configurer les variables d'environnement**

```bash
# Ajouter à ~/.bash_profile ou ~/.zshrc
export PATH="$PATH:/Users/nom_utilisateur/flutter/bin"
```

3. **Exécuter flutter doctor**

```bash
flutter doctor
```

### Installation Linux

1. **Installer les dépendances**

```bash
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa
```

2. **Télécharger Flutter SDK**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

3. **Configurer PATH**

```bash
export PATH="$PATH:/home/nom_utilisateur/flutter/bin"
```

## Configuration de l'Environnement de Développement Android

### Installer Android Studio

1. Télécharger Android Studio : https://developer.android.com/studio
2. Installer et lancer
3. Configurer SDK :

```bash
flutter config --android-sdk /chemin/vers/android/sdk
```

### Accepter les licences

```bash
flutter doctor --android-licenses
```

### Configurer l'émulateur

```bash
# Créer un émulateur
flutter emulator --create --name mon-emulateur

# Lancer l'émulateur
flutter emulator --launch mon-emulateur
```

## Configuration de l'Environnement de Développement iOS (macOS)

### Installer Xcode

```bash
# Installer via App Store
xcode-select --install
```

### Configurer le Simulateur iOS

```bash
# Lister les simulateurs disponibles
xcrun simctl list devices available

# Lancer le simulateur
open -a Simulator
```

## Vérification

```bash
# Vérifier la version Flutter
flutter --version

# Exécuter flutter doctor
flutter doctor

# Rechercher les mises à jour
flutter upgrade
```

## Problèmes Courants

### Échec du téléchargement ?

Utiliser un miroir :
```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
export PUB_CACHE=/chemin/vers/cache
```

### Android SDK non trouvé ?

```bash
flutter config --android-sdk /chemin/vers/android-sdk

# Ou définir une variable d'environnement
export ANDROID_SDK_ROOT=/chemin/vers/android-sdk
```

### Commande non trouvée ?

S'assurer que Flutter est dans PATH :
```bash
echo $PATH
# doit contenir flutter/bin
```

## Désinstallation

```bash
# Supprimer le répertoire Flutter
rm -rf /chemin/vers/flutter

# Retirer du PATH
```
