# Apifox Introduction

API integrated collaboration platform.

## What is Apifox?

Apifox is an API integrated collaboration platform that combines API documentation, API debugging, API Mock, and API automated testing.

### Main Features

- **API Documentation** - Auto-generate API docs, support team collaboration
- **API Debugging** - Complete HTTP request simulation tool
- **Mock Service** - Quickly create Mock servers
- **Automated Testing** - API automated testing features
- **Team Collaboration** - Team project management, permission control
- **Data Sync** - Real-time sync of API changes
- **Import/Export** - Support multiple format import/export

### Statistics

- Users: Millions of developers
- APIs Managed: Hundreds of millions of APIs
