# OpenCode 安装指南

详细介绍 OpenCode 的安装方法。

## Windows 安装

### 方法一：下载安装包

1. 访问 https://opencode.dev
2. 下载 Windows 安装包
3. 运行安装程序

### 方法二：Chocolatey

```
choco install opencode
```

## macOS 安装

### 方法一：下载安装包

1. 访问 https://opencode.dev
2. 下载 macOS 安装包
3. 拖入 Applications

### 方法二：Homebrew

```
brew install opencode
```

## Linux 安装

### Ubuntu/Debian

```
sudo apt install opencode
```

### Arch Linux

```
sudo pacman -S opencode
```

## 验证安装

```
opencode --version
```

## 卸载

### macOS

```
brew uninstall opencode
```

### Linux

```
sudo apt remove opencode
```
