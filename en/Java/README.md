# Java

## Overview

Java is an object-oriented programming language maintained by Oracle Corporation. It is one of the most popular enterprise development languages globally. Java is known for its "Write Once, Run Anywhere" characteristic and has a mature ecosystem with wide application areas.

## Key Features

- Object-Oriented: Fully object-oriented programming language
- Platform Independent: Cross-platform execution through JVM
- Automatic Memory Management: Garbage collection mechanism
- Rich Class Library: Comprehensive standard API
- Multi-threading Support: Built-in concurrent programming support
- Security: Multi-layer security mechanisms

## Quick Start

### Install JDK

Visit Oracle official download page to get JDK: https://www.oracle.com/java/technologies/downloads/

### Write Your First Program

Create HelloWorld.java:

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

### Compile and Run

```bash
# Compile
javac HelloWorld.java

# Run
java HelloWorld
```
