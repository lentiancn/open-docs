# PlantUML 설치 가이드

PlantUML은 UML 다이어그램을 빠르게 생성하기 위한 오픈소스 도구입니다. 이 가이드에서는 다양한 환경에서의 설치 방법을 설명합니다.

## 버전 정보

| 버전 | 상태 | 설명 |
|------|------|------|
| PlantUML | ✅ 현재 | 최신 안정 버전 |

## 설치 방법

### 방법 1: Windows 설치

#### 1. JAR 파일 다운로드

1. PlantUML JAR 다운로드: https://sourceforge.net/projects/plantuml/files/plantuml.jar/download
2. 로컬 디렉토리에 저장

#### 2. Graphviz 설치 (선택 사항이지만 권장)

```powershell
# Chocolatey 사용
choco install graphviz
```

### 방법 2: macOS 설치

```bash
# Homebrew 사용
brew install plantuml

# 또는 Graphviz 설치
brew install graphviz
```

### 방법 3: Linux 설치

```bash
# Ubuntu/Debian
sudo apt-get install -y plantuml graphviz

# CentOS/RHEL
sudo yum install -y plantuml graphviz
```

### 방법 4: VS Code 플러그인

1. VS Code 열기
2. "PlantUML" 검색
3. "PlantUML" 플러그인 설치
4. Graphviz 설치

### 방법 5: IntelliJ 플러그인

1. IntelliJ IDEA 열기
2. Settings → Plugins
3. "PlantUML" 검색
4. 플러그인 설치

### 방법 6: Docker 사용

```bash
# PlantUML 서버 실행
docker run -d -p 8080:8080 plantuml/plantuml-server:jetty

# http://localhost:8080 접근
```

## 환경 설정

### Java 요구사항

```bash
# Java 확인
java -version

# Java 8 이상 필요
```

### Graphviz 설정

```bash
# 환경 변수 설정 (Linux/macOS)
export GRAPHVIZ_DOT=/usr/bin/dot

# Windows의 경우 PATH에 추가
# C:\Program Files\Graphviz\bin
```

## 자주 묻는 질문

### 이미지가 표시되지 않음?

Graphviz가 설치되어 있는지 확인:

```bash
dot -V
```

### 한글 깨짐?

다이어그램에 추가:

```plantuml
skinparam defaultFontName MalgunGothic
```
