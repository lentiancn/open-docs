# OpenClaw よくある質問

## 基礎

### OpenClawとは？

チャットアプリをAIアシスタントに接続する自己ホスト型ゲートウェイ。

### 無料？

ソフト自体は無料。APIキー必要。

### 対応プラットフォーム

- OS: macOS, Linux, Windows
- チャット: WhatsApp, Telegram, Discord, iMessage
- モバイル: iOS, Android

## インストール

### 失敗する場合

- Nodeバージョン確認：`node --version`（22+必要）
- 権限問題：sudoを試す

### ポート競合

```bash
openclaw gateway --port 18889
```

## 設定

### APIキー設定

```bash
openclaw config set providers.anthropic.key <キー>
```

### 設定ファイル

`~/.openclaw/openclaw.json`
