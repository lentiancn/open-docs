# Apache JMeter - Обзор

## Что такое Apache JMeter?

Apache JMeter™ — программное обеспечение с открытым исходным кодом, 100% чистое Java-приложение, предназначенное для нагрузочного тестирования и измерения производительности. Изначально оно было разработано для тестирования веб-приложений, но с тех пор расширилось до других тестовых функций.

## Что может делать JMeter?

Apache JMeter можно использовать для тестирования производительности как статических, так и динамических ресурсов и динамических веб-приложений. Он может имитировать высокую нагрузку на сервер, группу серверов, сеть или объект для проверки его прочности или анализа общей производительности при различных типах нагрузки.

### Ключевые функции

- **Поддержка множества протоколов**: Веб (HTTP, HTTPS), веб-службы SOAP/REST, FTP, база данных через JDBC, LDAP, JMS, почта, TCP, объекты Java
- **Полнофункциональная тестовая IDE**: Запись тестовых планов из браузеров или нативных приложений
- **Режим CLI**: Выполнение нагрузочных тестов с любой совместимой с Java ОС
- **Динамические HTML-отчёты**: Полные и готовые к представлению
- **Извлечение данных**: Поддержка HTML, JSON, XML и регулярных выражений
- **100% Java**: Полная переносимость
- **Многопоточность**: Параллельная выборка несколькими потоками
- **Высокая расширяемость**: подключаемые и скриптуемые сэмплеры (Groovy, BeanShell)
- **Интеграция CI**: Поддержка Maven, Gradle и Jenkins

## Чем JMeter НЕ является

JMeter не является браузером — он работает на уровне протокола. Он не выполняет JavaScript, найденный на HTML-страницах, и не отображает HTML-страницы как настоящий браузер.

## Начало работы

1. Скачайте JMeter с официального сайта
2. Запустите скрипт JMeter (jmeter.bat для Windows, jmeter.sh для Linux/Mac)
3. Создайте тестовый план с помощью графического интерфейса
4. Запустите тесты и проанализируйте результаты
