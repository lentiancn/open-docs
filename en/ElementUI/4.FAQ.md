# Element UI FAQ

## Installation Issues

### Installation Failed

Ensure Node.js version >= 12.0.0, clear cache and retry.

### Styles Not Applied

Make sure to import CSS:

```javascript
import 'element-ui/lib/theme-chalk/index.css'
```

## Usage Issues

### Components Not Displaying

Ensure components are globally registered or use full import.

### Table Issues

Ensure data property is correctly bound.

### Form Validation

Ensure form has model property, form-item has prop property.

## Vue Version

Element UI only supports Vue 2. For Vue 3, use ElementPlus.
