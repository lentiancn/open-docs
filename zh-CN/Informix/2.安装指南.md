# Informix 安装指南

## Linux 安装步骤

以下步骤在 CentOS 7/8 环境下测试有效，适用于 Informix 11.5 和 12.1 版本。

### 步骤 1：创建用户和用户组

必须在 root 权限下执行以下命令：

```bash
su root

# 创建组
groupadd -g 1000 informix

# 创建用户
useradd -u 1000 -g 1000 -d /home/informix -m informix

# 设置密码
passwd informix
```

### 步骤 2：创建安装目录

```bash
mkdir /opt/informix
chown informix.informix /opt/informix
```

### 步骤 3：上传并解压安装包

将 Informix 安装包上传到服务器，然后解压：

```bash
cd /tmp/informix
tar -xvf iif.11.50.UC4E.Linux-RHEL4.tar
```

### 步骤 4：执行安装

```bash
./ids_install
```

安装过程中：
- 输入 `1` 表示同意许可协议
- 选择安装路径为 `/opt/informix`
- 根据需要选择安装组件

### 步骤 5：配置环境变量

以 informix 用户身份登录，编辑 `~/.bash_profile`：

```bash
su - informix
vim ~/.bash_profile
```

添加以下环境变量：

```bash
export INFORMIXDIR=/opt/informix
export INFORMIXSERVER=ol_informix1210
export ONCONFIG=onconfig.ol_informix1210
export INFORMIXSQLHOSTS=/opt/informix/etc/sqlhosts.ol_informix1210
export PATH=$PATH:$INFORMIXDIR/bin
```

使配置生效：

```bash
source ~/.bash_profile
```

### 步骤 6：配置 sqlhosts 文件

```bash
cd /opt/informix/etc
cp sqlhosts.std sqlhosts
vim sqlhosts
```

添加以下内容：

```
ol_informix1210 onsoctcp localhost ol_informix1210
dr_informix1210 drsoctcp localhost dr_informix1210
lo_informix1210 onsoctcp localhost lo_informix1210
```

**注意**：如果需要远程访问，将 `localhost` 改为 `*`。

### 步骤 7：配置 onconfig 文件

```bash
cd /opt/informix/etc
cp onconfig.std onconfig.ol_informix1210
vim onconfig.ol_informix1210
```

修改以下关键参数：

```
SERVERNUM 49
FULL_DISK_INIT 1
NETTYPE soctcp,1,50,CPU
```

### 步骤 8：初始化数据库

首次启动需要使用 `-ivy` 参数进行初始化：

```bash
oninit -ivy
```

验证初始化是否成功：

```bash
# 查看数据文件大小
ls -la /opt/informix/storage/
# 如果 rootdbs 文件大小不为 0，说明初始化成功

# 查看数据库状态
onstat -
```

### 步骤 9：常用管理命令

```bash
# 关闭数据库
onmode -ky

# 查看服务状态
onstat -

# 查看逻辑日志状态
onstat -l

# 连接数据库（交互模式）
onstat -i

# 查看端口
netstat -apn | grep oninit
```

## Windows 安装概述

Windows 平台的安装相对简单：

1. 从 IBM 官网下载 Informix 安装包（.exe 或 .zip 格式）
2. 运行安装程序 `ids_install.exe`
3. 按照安装向导选择安装路径、组件等
4. 配置环境变量（通过系统属性 → 高级 → 环境变量）
5. 使用图形化工具 `oninit` 或 `DB-Access` 进行初始化

## 安装后验证

安装完成后，可以通过以下方式验证：

```bash
# 检查 Informix 进程是否运行
ps -ef | grep oninit

# 查看版本信息
onstat -

# 连接测试
dbaccess -version
```

## 注意事项

- 确保系统满足硬件和软件要求
- 防火墙需要开放 Informix 使用的端口（默认 9088）
- 建议使用专用用户运行 Informix，避免使用 root 用户
- 生产环境请仔细规划存储布局
