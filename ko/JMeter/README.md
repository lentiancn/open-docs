# Apache JMeter 문서

Apache JMeter 문서에 오신 것을 환영합니다. 이 폴더에는 성능 테스트 및 부하 테스트를 위한 강력한 오픈소스 도구인 Apache JMeter에 대한 종합 문서가 포함되어 있습니다.

## Apache JMeter란?

Apache JMeter™는 기능 동작을 테스트하고 성능을 측정하기 위해 설계된 100% 순수 Java 애플리케이션입니다. Originally 웹 애플리케이션 테스트용으로 설계되었지만 다양한 테스트 기능을 포함하도록 확장되었습니다.

## 빠른 시작

1. **다운로드**: https://jmeter.apache.org/에서 JMeter를 가져오세요
2. **설치**: 압축을 풀고 `./jmeter.sh`(Linux/Mac) 또는 `jmeter.bat`(Windows) 실행
3. **테스트 생성**: GUI를 사용하여 테스트 플랜 구축
4. **실행**: 테스트 실행 및 결과 분석

## 주요 기능

- **멀티 프로토콜 지원**: HTTP, FTP, JDBC, LDAP, JMS 등
- **테스트 IDE**: 테스트 플랜 기록 및 구축
- **CLI 모드**: 명령줄에서 테스트 실행
- **보고서**: 동적 HTML 보고서
- **확장성**: 플러그인 및 스크립트 지원
- **분산 테스트**: 부하를 여러 시스템에 분산

## 요구 사항

- Java 8 이상
- 4GB+ RAM 권장
- Windows / Linux / macOS
