# Guía de Instalación de Kubernetes

Instalación de Kubernetes.

---

## Métodos

| Método | Descripción |
|--------|-------------|
| **minikube** | Clúster local |
| **kind** | K8s en Docker |
| **kubeadm** | Producción |

---

## minikube

```bash
brew install minikube
minikube start
```

---

## kubectl

```bash
brew install kubectl
```

---

## Verificar

```bash
kubectl get nodes
```
