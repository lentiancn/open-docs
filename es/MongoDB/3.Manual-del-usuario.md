# MongoDB Manual del Usuario

## Conceptos Básicos

### Base de Datos

Una base de datos es el contenedor de nivel más alto para datos en MongoDB. Cada base de datos tiene sus propias colecciones y configuraciones de permisos.

### Colección

Una colección es un grupo de documentos de MongoDB, similar a una tabla en una base de datos relacional. Los documentos en una colección pueden tener diferentes campos.

### Documento

Un documento es la unidad básica de datos en MongoDB, consiste en pares de campo y valor. Los documentos son similares a objetos JSON y se almacenan en formato BSON.

## Conectar a MongoDB

### Conectar a la Base de Datos Local

```bash
mongosh
```

### Conectar a una Base de Datos Remota

```bash
mongosh "mongodb://hostname:port"
```

## Operaciones de Base de Datos

### Ver Bases de Datos

```bash
show dbs
```

### Cambiar/Crear Base de Datos

```bash
use midatabase
```

### Eliminar Base de Datos

```bash
db.dropDatabase()
```

## Operaciones de Colección

### Ver Colecciones

```bash
show collections
```

### Crear Colección

```bash
db.createCollection("micoleccion")
```

### Eliminar Colección

```bash
db.micoleccion.drop()
```

## Insertar Documentos

### Insertar un Documento

```bash
db.usuarios.insertOne({
  nombre: "Juan",
  edad: 30,
  email: "juan@ejemplo.com"
})
```

### Insertar Múltiples Documentos

```bash
db.usuarios.insertMany([
  { nombre: "María", edad: 25 },
  { nombre: "Pedro", edad: 35 }
])
```

## Consultar Documentos

### Consultar Todos los Documentos

```bash
db.usuarios.find()
```

### Consultar con Condiciones

```bash
db.usuarios.find({ edad: 30 })
db.usuarios.find({ edad: { $gte: 25 } })
```

### Ordenar Resultados

```bash
db.usuarios.find().sort({ edad: -1 })
```

### Limitar Resultados

```bash
db.usuarios.find().limit(5)
```

## Actualizar Documentos

### Actualizar un Documento

```bash
db.usuarios.updateOne(
  { nombre: "Juan" },
  { $set: { edad: 31 } }
)
```

### Actualizar Múltiples Documentos

```bash
db.usuarios.updateMany(
  { edad: { $lt: 30 } },
  { $set: { estado: "joven" } }
)
```

## Eliminar Documentos

### Eliminar un Documento

```bash
db.usuarios.deleteOne({ nombre: "Juan" })
```

### Eliminar Múltiples Documentos

```bash
db.usuarios.deleteMany({ edad: { $lt: 20 } })
```

## Agregación

### Pipeline de Agregación

```bash
db.usuarios.aggregate([
  { $match: { edad: { $gte: 25 } } },
  { $group: { _id: "$ciudad", total: { $sum: 1 } } },
  { $sort: { total: -1 } }
])
```

## Índices

### Crear Índice

```bash
db.usuarios.createIndex({ nombre: 1 })
db.usuarios.createIndex({ edad: 1, nombre: 1 })
db.usuarios.createIndex({ email: 1 }, { unique: true })
```

### Ver Índices

```bash
db.usuarios.getIndexes()
```

### Eliminar Índice

```bash
db.usuarios.dropIndex("nombre_1")
```

## Importar y Exportar Datos

### Exportar con mongodump


```bash
mongorestore --db=midatabase /respaldo/midatabase
```

### Exportar con mongoexport

```bash
mongoexport --db=midatabase --collection=usuarios --out=usuarios.json
```

```bash
mongoimport --db=midatabase --collection=usuarios --file=usuarios.json
```

## Transacciones

MongoDB 4.0+ soporta transacciones de múltiples documentos.

```bash
sesion = db.getMongo().startSession()
sesion.startTransaction()
try {
  sesion.getDatabase("miDB").cuentas.updateOne(
    { cuentaId: 1 },
    { $inc: { saldo: -100 } }
  )
  sesion.commitTransaction()
} catch (error) {
  sesion.abortTransaction()
} finally {
  sesion.endSession()
}
```

## Change Streams

Los change streams permiten a las aplicaciones escuchar cambios de datos en tiempo real.

```bash
var changeStream = db.usuarios.watch()
while (changeStream.hasNext()) {
  printjson(changeStream.next())
}
```

## GridFS

GridFS se usa para almacenar archivos grandes que exceden el límite de 16 MB.

```bash
mongofiles put miarchivo.txt
mongofiles list
mongofiles get miarchivo.txt
mongofiles delete miarchivo.txt
```
