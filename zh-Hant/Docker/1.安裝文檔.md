# Docker 安裝文檔

## Linux

```bash
# Ubuntu/Debian
curl -fsSL https://get.docker.com | sh
sudo systemctl start docker
sudo systemctl enable docker
```

## macOS

```bash
brew install --cask docker
```

## Windows

下載 [Docker Desktop](https://www.docker.com/products/docker-desktop) 安裝。

## 驗證

```bash
docker --version
```
