# JUnit Введение

JUnit — самый популярный фреймворк для модульного тестирования на Java, используемый для написания и выполнения повторяемых тестов. Это основа для разработки высококачественного кода на Java, которую используют миллионы разработчиков по всему миру.

## JUnit 4 и JUnit 5

### Обзор версий

- **JUnit 4**: Выпущен в 2010 году, последняя основная версия фреймворка тестирования, которую долгое время использовало Java-сообщество
- **JUnit 5**: Выпущен в 2017 году, полная переработка фреймворка JUnit, состоящая из трёх подпроектов: JUnit Platform, JUnit Jupiter и JUnit Vintage

### Архитектурные отличия

JUnit 4 использует монолитную архитектуру со всеми функциями в одной библиотеке. JUnit 5 использует модульную архитектуру с тремя отдельными проектами:

- **JUnit Platform**: Основа для запуска фреймворков тестирования, предоставляет API TestEngine и консольный запускатель
- **JUnit Jupiter**: Новая модель программирования и модель расширения для написания тестов и расширений
- **JUnit Vintage**: движок совместимости, позволяющий запускать тесты JUnit 3 и JUnit 4 на платформе JUnit 5

### Основные отличия

| Функция | JUnit 4 | JUnit 5 |
|---------|---------|---------|
| Минимальная версия Java | Java 5+ | Java 8+ |
| Пакет аннотаций | org.junit | org.junit.jupiter.api |
| Тестовый класс без наследования | Нет | Да |
| Стиль проверок | Статические методы Assert | Статические методы Assertions |
| Параметризованные тесты | @Parameters | @ParameterizedTest |
| Вложенные тесты | Не поддерживаются | Поддерживаются |
| Динамические тесты | Не поддерживаются | Поддерживаются |
| Параллельное выполнение | Ограниченная поддержка | Нативная поддержка |
| Модель расширения | @RunWith | @ExtendWith |

### Рекомендации по выбору версии

- **Новые проекты**: Используйте JUnit 5 для современных функций и лучшей расширяемости
- **Существующие проекты**: Продолжайте использовать JUnit 4 или используйте JUnit Vintage для постепенной миграции
- **Долгосрочная поддержка**: Постепенно мигрируйте на JUnit 5 для лучшего опыта тестирования
