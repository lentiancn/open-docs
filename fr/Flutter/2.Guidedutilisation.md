# Guide d'utilisation de Flutter

Ce guide explique comment utiliser Flutter pour créer des applications multiplateformes.

## Démarrage rapide

### Créer un projet

```bash
# Créer un nouveau projet
flutter create my_app

# Aller dans le répertoire du projet
cd my_app

# Lancer l'application
flutter run
```

### Structure du projet

```
my_app/
├── lib/
│   ├── main.dart
│   └── ...
├── test/
├── android/
├── ios/
├── web/
├── linux/
├── windows/
├── macos/
├── pubspec.yaml
└── README.md
```

## Widgets de mise en page

### Colonne et ligne

```dart
// Colonne (verticale)
Column(
  mainAxisAlignment: MainAxisAlignment.center,
  children: [
    Text('Première ligne'),

Row(
  mainAxisAlignment: MainAxisAlignment.spaceEvenly,
  children: [
    Icon(Icons.home),
    Icon(Icons.search),
    Icon(Icons.settings),
  ],

```

### Liste

```dart
// Liste verticale
ListView.builder(
  itemCount: items.length,
  itemBuilder: (context, index) {
    return ListTile(
      title: Text(items[index].title),
    );

// Liste horizontale

  scrollDirection: Axis.horizontal,
```

## Gestion d'état

### setState

```dart
class MyWidget extends StatefulWidget {
  @override
  _MyWidgetState createState() => _MyWidgetState();
}

class _MyWidgetState extends State<MyWidget> {
  int _counter = 0;

  void _increment() {
    setState(() {
      _counter++;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        Text('Compteur: $_counter'),
        ElevatedButton(
          onPressed: _increment,
          child: Text('Incrémenter'),
        ),
      ],
    );
  }
}
```

### Provider

```bash
# Ajouter une dépendance
flutter pub add provider
```

```dart
// Créer un Provider
class CounterModel extends ChangeNotifier {
  int _count = 0;
  int get count => _count;

  void increment() {
    _count++;
    notifyListeners();
  }
}

// Utiliser Provider
Consumer<CounterModel>(
  builder: (context, counter, child) {
    return Text('${counter.count}');
  },
)
```

### GetX

```bash
# Ajouter une dépendance
flutter pub add get
```

```dart
// Utiliser GetX
Get.put(CounterController());

// Utiliser dans un Widget
Get.find<CounterController>().increment();
```

## Navigation par routes

### Navigation de base

```dart
// Naviguer vers une nouvelle page
Navigator.push(
  context,
  MaterialPageRoute(builder: (context) => NextPage()),
);

// Retour
Navigator.pop(context);
```

### Routes nommées

```dart
// Configurer les routes
MaterialApp(
  routes: {
    '/': (context) => HomePage(),
    '/about': (context) => AboutPage(),
  },
);

// Naviguer
Navigator.pushNamed(context, '/about');
```

### Passage de paramètres

```dart
// Passer des paramètres
Navigator.pushNamed(
  context,
  '/detail',
  arguments: {'id': 123},
);

// Recevoir des paramètres
final args = ModalRoute.of(context).settings.arguments;
```

## Requêtes réseau

### HTTP

```bash
flutter pub add http
```

```dart
import 'package:http/http.dart' as http;

Future<void> fetchData() async {
  final response = await http.get(Uri.parse('https://api.example.com/data'));
  if (response.statusCode == 200) {
    print(response.body);
  }
}
```

### Dio

```bash
flutter pub add dio
```

```dart
import 'package:dio/dio.dart';

final dio = Dio();

```

## Stockage local

### SharedPreferences

```bash
flutter pub add shared_preferences
```

```dart
import 'package:shared_preferences/shared_preferences.dart';

// Sauvegarder des données
SharedPreferences prefs = await SharedPreferences.getInstance();
await prefs.setString('name', 'John');
await prefs.setInt('age', 30);

// Lire des données
String name = prefs.getString('name') ?? '';
int age = prefs.getInt('age') ?? 0;
```

### SQLite

```bash
flutter pub add sqflite
```

```dart
import 'package:sqflite/sqflite.dart';

final db = await openDatabase('my.db');
await db.insert('users', {'name': 'John', 'age': 30});
final users = await db.query('users');
```
