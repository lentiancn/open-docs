# Apifox

Apifox ist eine der führenden API-Kollaborationsplattformen in China und bietet eine ganzheitliche Lösung für API-Design, Mocking, Tests und Dokumentation. Es integriert API-Design, API-Dokumentationsverwaltung, API-Debugging, API-Mocking und automatisierte API-Tests – das beliebteste API-Verwaltungstool in China.

## Schnellstart

### Herunterladen und installieren

Besuchen Sie https://apifox.com/, um das Installationspaket für Ihr System herunterzuladen.

### Projekt erstellen

1. Starten Sie Apifox und melden Sie sich an
2. Klicken Sie auf "Neues Projekt"
3. Geben Sie einen Projektnamen ein

### Schnittstelle erstellen

1. Klicken Sie im Projekt auf "Neue Schnittstelle"
2. Füllen Sie die Schnittstelleninformationen aus
3. Speichern Sie

### Schnittstelle testen

1. Öffnen Sie die Schnittstelle
2. Klicken Sie auf "Senden"
3. Sehen Sie sich die Antwort an

## Kernfunktionen

- **API-Design:** Visuelle Definition von API-Schnittstellen
- **API-Dokumentation:** Automatisch generiert, Export in verschiedenen Formaten möglich
- **API-Debugging:** Leistungsstarke Online-Debugging-Funktion
- **API-Mocking:** Intelligentes Mocking, automatische Generierung von Simulationsdaten
- **Automatisierte Tests:** API-Tests ohne Codierung möglich
- **Teamzusammenarbeit:** Echtzeitsynchronisierung, Berechtigungsverwaltung
