# Angular FAQ

## В: ng не распознаётся?
**О:** Проверить Node.js: `node --version`. Перезапустить терминал.

## В: Медленная установка?
**О:** Зеркало: `npm config set registry https://registry.npmmirror.com`

## В: Ошибка CORS?
**О:** Настроить прокси или добавить CORS заголовки на сервере.

## В: Обновить Angular?
**О:** `npm install -g @angular/cli@latest`
