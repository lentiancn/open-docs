# Java 설치 가이드

> 환경 설정

---

## 요구 사항

| 환경 | 요구사항 |
|------|---------|
| OS | Windows, macOS, Linux |
| RAM | 2GB 이상 |

---

## 설치

### Windows

```powershell
choco install openjdk21
```

### macOS

```bash
brew install openjdk@21
```

### Linux

```bash
sudo apt install openjdk-21-jdk
```

---

## 환경 변수

```
JAVA_HOME = /usr/lib/jvm/java-21-openjdk
```

---

## 확인

```bash
java -version
```
