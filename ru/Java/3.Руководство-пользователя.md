# Руководство пользователя Java

## Ваша первая программа на Java

### Создание Hello World

Создайте новый текстовый файл с именем HelloWorld.java. Обратите внимание, что имя файла должно точно совпадать с именем класса.

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

### Компиляция программы

Откройте терминал или командную строку, перейдите в каталог с файлом и выполните команду компиляции:

```bash
javac HelloWorld.java
```

После успешной компиляции будет создан файл HelloWorld.class.

### Запуск программы

```bash
java HelloWorld
```

Вывод:

```
Hello, World!
```

## Основной синтаксис Java

### Переменные и типы данных

Java — это язык со статической типизацией. Переменные должны быть объявлены перед использованием.

```java
// Примитивные типы данных
int age = 25;
double salary = 5000.50;
boolean isActive = true;
char grade = 'A';
String name = "Иван";

// Ссылочные типы
Integer count = 10;
Double price = 99.99;
```

### Операторы

```java
// Арифметические операторы
int a = 10, b = 3;
int sum = a + b;      // Сложение
int diff = a - b;     // Вычитание
int prod = a * b;     // Умножение
int quot = a / b;     // Деление
int rem = a % b;      // Остаток

// Операторы сравнения
boolean result = (a > b);

// Логические операторы
boolean flag = (a > 5) && (b < 10);
```

### Условные операторы

```java
// Оператор if-else
if (score >= 90) {
    System.out.println("Отлично");
} else if (score >= 60) {
    System.out.println("Хорошо");
} else {
    System.out.println("Плохо");
}

// Оператор switch
switch (day) {
    case 1:
        System.out.println("Понедельник");
        break;
    case 2:
        System.out.println("Вторник");
        break;
    default:
        System.out.println("Неизвестно");
}
```

### Операторы цикла

```java
// Цикл for
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}

// Цикл while
int i = 0;
while (i < 10) {
    System.out.println(i);
    i++;
}

// Цикл do-while
do {
    System.out.println(i);
    i++;
} while (i < 10);

// for-each для массивов
int[] numbers = {1, 2, 3, 4, 5};
for (int num : numbers) {
    System.out.println(num);
}
```

## Объектно-ориентированное программирование

### Классы и объекты

```java
// Определение класса
public class Person {
    // Поля
    private String name;
    private int age;
    
    // Конструктор
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // Геттеры и сеттеры
    public String getName() {
        return name;
    }
    
    public void setName(String name) {
        this.name = name;
    }
    
    // Метод
    public void sayHello() {
        System.out.println("Привет, я " + name);
    }
}

// Создание объекта
Person person = new Person("Иван", 25);
person.sayHello();
```

## Stream API

```java
import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);

// Фильтрация
List<Integer> evenNumbers = numbers.stream()
    .filter(n -> n % 2 == 0)
    .collect(Collectors.toList());

// Отображение
List<Integer> squared = numbers.stream()
    .map(n -> n * n)
    .collect(Collectors.toList());

// Сумма
int sum = numbers.stream()
    .mapToInt(Integer::intValue)
    .sum();
```

## Многопоточное программирование

### Создание потоков

```java

class MyThread extends Thread {
    @Override
    public void run() {
        System.out.println("Поток выполняется");
    }
}

// Реализация интерфейса Runnable
class MyRunnable implements Runnable {
    @Override
    public void run() {
        System.out.println("Поток выполняется");
    }
}

// Запуск потока
MyThread thread = new MyThread();
thread.start();

// Использование Runnable
Thread runnableThread = new Thread(new MyRunnable());
runnableThread.start();
```

## Сетевое программирование

### TCP-соединение

```java
import java.net.*;
import java.io.*;

ServerSocket serverSocket = new ServerSocket(8080);
Socket clientSocket = serverSocket.accept();

    new InputStreamReader(clientSocket.getInputStream()));
```

```java
HttpClient client = HttpClient.newHttpClient();
```

## Операции с базами данных

### JDBC-подключение

```java
import java.sql.*;

String url = "jdbc:mysql://localhost:3306/mydb";
String user = "root";
String password = "password";

try (Connection conn = DriverManager.getConnection(url, user, password)) {
    String sql = "SELECT * FROM users WHERE id = ?";
    PreparedStatement stmt = conn.prepareStatement(sql);
    stmt.setInt(1, 1);
    
    ResultSet rs = stmt.executeQuery();
    while (rs.next()) {
        System.out.println(rs.getString("name"));
    }
} catch (SQLException e) {
    e.printStackTrace();
}
```
