# Kotlin 사용 가이드

이 가이드에서는 Kotlin의 기본 문법과 고급 기능을 설명합니다.

## 빠른 시작

### Hello World

```kotlin
fun main() {
    println("Hello, World!")
}
```

### 변수

```kotlin
// 불변

// 가변
var age: Int = 10
age = 11
```

## 함수

### 기본 함수

```kotlin

    return a + b

```

### 기본 매개변수


### 가변 매개변수

```kotlin
}
```

## 컬렉션

### 리스트


### 맵


### 컬렉션 연산

```kotlin
val list = listOf(1, 2, 3, 4, 5)

list.filter { it > 2 }           // [3, 4, 5]
list.map { it * 2 }              // [2, 4, 6, 8, 10]
list.reduce { acc, i -> acc + i } // 15
```

## Null 안전

### Nullable 타입

```kotlin
var name: String? = null
name = "Kotlin"
```

### 안전 연산자

```kotlin
val length = name?.length
val len = name?.length ?: 0
```

### Non-null 단언

```kotlin
val len = name!!.length
```

## 코루틴

### 기본 사용법

```kotlin
import kotlinx.coroutines.*

suspend fun fetchData(): String {
    delay(1000)
    return "Data"
}

fun main() = runBlocking {
    val result = fetchData()
    println(result)
}
```

### 병렬 처리

```kotlin
suspend fun fetchAll() = coroutineScope {
    val a = async { fetchData1() }
    val b = async { fetchData2() }
    "${a.await()} ${b.await()}"
}
```
