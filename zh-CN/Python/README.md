# Python

简洁优雅的编程语言

## 概述

Python 是一种广泛使用的高级编程语言，语法简洁、可读性高，拥有丰富的标准库和强大的第三方生态系统。适用于 Web 开发、数据科学、人工智能、自动化等各个领域。

## 快速开始

### 安装 Python

```bash
# Linux
sudo apt install python3 python3-pip

# macOS
brew install python3

# Windows
# 下载安装包：https://www.python.org/downloads/
```

### 第一个程序

```python
print("Hello, World!")
```

### 运行程序

```bash
python hello.py
```

## 核心特性

- **简洁易学**：语法简洁，门槛低
- **动态类型**：无需声明变量类型
- **丰富库**：标准库+第三方库
- **跨平台**：Windows、macOS、Linux
- **多范式**：面向对象、函数式、过程式
- **应用广泛**：Web、数据、AI、自动化

## 常用命令

```bash
python --version     # 查看版本
python -m venv venv  # 创建虚拟环境
pip install package  # 安装包
pip list             # 列出已安装包
python -c "code"     # 执行单行代码
```
