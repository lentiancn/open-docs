# Claude Code FAQ

Questions fréquemment posées.

## Installation

### Q: L'installation a échoué?

**Solutions**:
1. Vérifiez les exigences système
2. Mettez à jour VS Code vers la dernière version
3. Vérifiez la connexion réseau
4. Essayez de réinstaller

### Q: Impossible de se connecter?

**Solutions**:
1. Vérifiez si la clé API est valide
2. Vérifiez la connexion réseau
3. Essayez d'effacer le cache

### Q: L'extension ne s'affiche pas?

**Solutions**:
1. Redémarrez VS Code
2. Vérifiez si l'extension est activée
3. Vérifiez les notifications VS Code

## Utilisation

### Q: Réponse lente?

**Solutions**:
1. Vérifiez la connexion réseau
2. Réduisez la longueur du contexte
3. Attendez la réponse du serveur

### Q: Mauvaise qualité de génération de code?

**Solutions**:
1. Soyez plus précis dans les descriptions
2. Fournissez plus de contexte
3. Itérez et améliorez

### Q: Comment changer de modèle?

Sélectionnez la version du modèle dans les paramètres.

### Q: Quelles langues sont prises en charge?

Prend en charge les langages principaux: Python, JavaScript, TypeScript, Go, Rust, Java, C++ et plus.

## Compte

### Q: Comment obtenir une clé API?

Visitez https://console.anthropic.com pour en créer une.

### Q: Quota épuisé?

Achetez un quota supplémentaire dans la console.

## Sécurité

### Q: Le code sera-t-il divulgué?

Claude Code ne sauvegarde ni ne partage votre code.

### Q: Comment protéger la vie privée?

- N'entrez pas d'informations sensibles dans les conversations
- Utilisez le mode local
- Effacez régulièrement les sessions
