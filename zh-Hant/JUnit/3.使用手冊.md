# JUnit 5 使用手冊

## 基本測試

```java
@Test
void test() {
    assertEquals(4, 2 + 2);
}
```

## 斷言

```java
assertEquals(4, 2 + 2);
assertTrue(5 > 3);
assertThrows(Exception.class, () -> { });
```

## 參數化測試

```java
@ParameterizedTest
@ValueSource(ints = {1, 2, 3})
void test(int n) {
    assertTrue(n > 0);
}
```

## 巢狀測試

```java
@Nested
class InnerTests {
    @Test
    void test() { }
}
```
