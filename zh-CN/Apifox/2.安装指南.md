# Apifox 安装指南

Apifox 多种安装方式详解。

## 下载安装

### 桌面应用

1. 访问 https://apifox.com
2. 下载 Windows/macOS/Linux 版本
3. 运行安装程序

### Windows

- 下载 .exe 安装包
- 运行安装向导

### macOS

- 下载 .dmg 安装包
- 拖入 Applications

### Linux

- 下载 AppImage 或 deb 包
- 赋予执行权限

## Web 版

直接访问 https://web.apifox.com 使用。

## 账号注册

1. 打开 Apifox
2. 选择注册方式（邮箱/手机/GitHub）
3. 完成注册并登录

## 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl + Enter | 发送请求 |
| Ctrl + S | 保存 |
| Ctrl + / | 注释 |
| Ctrl + Space | 自动完成 |

## 首次使用

### 创建项目

1. 点击"新建项目"
2. 输入项目名称
3. 选择项目类型

### 添加 API

1. 在项目中点击"新建接口"
2. 填写接口信息
3. 发送请求测试

## 配置代理

### 设置代理

1. 打开设置
2. 选择网络代理
3. 配置代理地址和端口

## 插件扩展

支持安装插件扩展功能。
