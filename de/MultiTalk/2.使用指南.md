# MultiTalk Benutzeranleitung

Diese Anleitung beschreibt, wie Sie MultiTalk zur Generierung von Multi-Sprecher-Audioinhalten verwenden.

## Schnellstart

### Grundlegende Verwendung

```bash
python inference.py \
  --text "Hallo zusammen, willkommen bei MultiTalk." \
  --speakers speaker1,speaker2 \
  --output result.wav
```

## Eingabeanforderungen

### Textanforderungen

- Sprache: Mehrsprachige Unterstützung
- Format: Klartext
- Länge: Keine strenge Begrenzung

### Sprecherkonfiguration

Sprecher in `speakers.yaml` definieren:

```yaml
speakers:
  speaker1:
    voice_id: "voice_001"
    language: "de"
  speaker2:
    voice_id: "voice_002"
    language: "de"
```

## Erweiterte Verwendung

### Benutzerdefinierte Sprach-IDs

```bash
python inference.py \
  --text "Dies ist eine Demo mit benutzerdefinierter Stimme." \
  --voice_ids voice001,voice002 \
  --output output.wav
```

### Geschwindigkeit anpassen

```bash
python inference.py \
  --text "Beispiel für langsames Sprechen." \
  --speakers speaker1 \
  --speed 0.8 \
  --output slow.wav
```

### Stapelverarbeitung

```bash
python batch_inference.py \
  --text_file texts.txt \
  --output_dir output/
```

## Konfiguration

### Konfigurationsdatei

Bearbeite `config.yaml`:

```yaml
inference:
  sample_rate: 22050
  speed: 1.0
  
model:
  checkpoint: "checkpoints/multitalk.pth"
```

## Best Practices

### 1. Verwenden Sie geeignete Sprach-IDs

Wählen Sie Sprach-IDs, die zu den Sprechermerkmalen passen.

### 2. Sprachübereinstimmung

Stellen Sie sicher, dass die Textsprache mit der Sprachsprache übereinstimmt.

### 3. Geschwindigkeit optimieren

Passen Sie die Geschwindigkeit für natürlich klingende Ausgabe an.

## Fehlerbehebung

### Schlechte Audioqualität?

Versuchen Sie verschiedene Sprach-IDs.

### Geschwindigkeitsprobleme?

Passen Sie den Geschwindigkeitsparameter schrittweise an.
