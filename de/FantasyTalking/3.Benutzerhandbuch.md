# FantasyTalking - Benutzerhandbuch

Detaillierte Anleitung.

## Grundlegende Nutzung

### Video generieren

```bash
python inference.py --source_image face.jpg --audio speech.wav --output output.mp4
```

### Parameter

| Parameter | Beschreibung |
|-----------|--------------|
| --source_image | Eingabebild |
| --audio | Audiodatei |
| --output | Ausgabepfad |
| --face_enhancer | Gesichtserweiterung |
| --fps | Bilder pro Sekunde |

## Python API

```python
from fantasytalking import FantasyTalking

ft = FantasyTalking()
ft.generate(
    source_image="face.jpg",
    audio="speech.wav",
    output="output.mp4"
)
```

## Stapelverarbeitung

```python
ft.batch_generate(image_list, audio_list, output_dir)
```
