# Введение в Element Plus

Element Plus — это библиотека компонентов пользовательского интерфейса на Vue 3.

## Что такое Element Plus

Element Plus — это версия Vue 3 библиотеки Element UI, разработанная и поддерживаемая командой Ele.me. Она предоставляет полный набор спецификаций дизайна и библиотек компонентов.

## Основные характеристики

- Более 70 компонентов
- Полная поддержка TypeScript
- Импорт по требованию
- Интернационализация
- Настраиваемые темы

## Применение

- Системы управления предприятием
- Платформы электронной коммерции
- CRM системы
