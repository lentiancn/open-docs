# Preguntas frecuentes de JUnit 5

## Preguntas generales

### ¿Qué es JUnit?
JUnit es un marco de código abierto para pruebas unitarias en Java. Es el estándar de facto para probar aplicaciones Java.

### ¿Cuál es la diferencia entre JUnit 4 y JUnit 5?

| Característica | JUnit 4 | JUnit 5 |
|----------------|----------|----------|
| Arquitectura | Monolítica | Modular |
| Java | 5+ | 8+ |
| Pruebas anidadas | No | Sí |
| Parametrizadas | Externo | Integrado |

### ¿Necesito reescribir mis pruebas de JUnit 4?
No, JUnit Vintage permite ejecutar pruebas JUnit 3/4 en JUnit 5.

## Instalación

### Maven
```xml
<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter</artifactId>
    <version>5.10.0</version>
</dependency>
```

### Gradle
```groovy
testImplementation 'org.junit.jupiter:junit-jupiter:5.10.0'
```

## Uso

### Prueba básica
```java
@Test
void prueba() {
    assertEquals(4, 2 + 2);
}
```

### Probar excepciones
```java
assertThrows(IllegalArgumentException.class, 
    () -> { throw new IllegalArgumentException(); });
```
