# OpenClaw 🦞

OpenClaw es una **puerta de enlace autohospedada** que conecta tus aplicaciones de mensajería favoritas — WhatsApp, Telegram, Discord, iMessage y más — con asistentes de codificación de IA como Pi. Ejecutas un solo proceso de Gateway en tu propia máquina (o en un servidor), y se convierte en el puente entre tus aplicaciones de mensajería y un asistente de IA siempre disponible.

## ¿Para quién es?

OpenClaw está diseñado para desarrolladores y usuarios avanzados que desean un asistente de IA personal con el que puedan contactar desde cualquier lugar, sin renunciar al control de sus datos ni depender de un servicio alojado.

## Características Principales

Lo que hace diferente a OpenClaw:

- **Autohospedado**: se ejecuta en tu hardware, tus reglas
- **Multicanal**: un Gateway sirve WhatsApp, Telegram, Discord y más simultáneamente
- **Nativo para agentes**: construido para agentes de codificación con uso de herramientas, sesiones, memoria y enrutamiento multiagente
- **Código abierto**: licencia MIT, impulsado por la comunidad

## ¿Qué necesitas?

- Node.js 22 o superior
- Una clave API de tu proveedor de IA elegido
- 5 minutos

Para la mejor calidad y seguridad, usa el modelo más potente de última generación disponible.

## Inicio Rápido

```bash
# Instalar OpenClaw
npm install -g openclaw@latest

# Ejecutar el asistente de incorporación
openclaw onboard --install-daemon

# Iniciar sesión en canales y comenzar el Gateway
openclaw channels login
openclaw gateway --port 18789
```

## Cómo Funciona

El Gateway es la fuente única de verdad para sesiones, enrutamiento y conexiones de canal. Las aplicaciones de mensajería se conectan a través de él a agentes de IA, CLI, UI de Control Web o nodos móviles.

## Capacidades Principales

- **Puerta de enlace multicanal**: WhatsApp, Telegram, Discord e iMessage con un solo proceso de Gateway
- **Canales de plugin**: Añade Mattermost y más con paquetes de extensión
- **Enrutamiento multiagente**: sesiones aisladas por agente, espacio de trabajo o remitente
- **Soporte multimedia**: enviar y recibir imágenes, audio y documentos
- **UI de Control Web**: panel de control para chat, configuración, sesiones y nodos
- **Nodos móviles**: empareja nodos iOS y Android para Canvas, cámara/pantalla y flujos de trabajo habilitados para voz
