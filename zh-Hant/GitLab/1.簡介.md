# GitLab 簡介

> 軟體開發和營運的完整 DevOps 平台

---

## 概述

GitLab 是一個基於網頁的 Git 儲存庫管理工具，具有 wiki、問題追蹤和 CI/CD 管道功能。它是一個完整的 DevOps 平台，讓團隊能夠在單一介面中規劃、開發、測試、部署和監控應用程式。

---

## 主要功能

- **Git 儲存庫託管**：具有分支、標籤和合併請求的完整 Git 儲存庫管理
- **問題追蹤**：具有標籤、里程碑和看板視圖的綜合問題追蹤
- **CI/CD 管道**：內建的持續整合和部署管道
- **Wiki**：用於文檔的內建 wiki
- **程式碼審查**：具有合併請求工作流程的內建程式碼審查
- **安全性**：企業級安全掃描和合規功能
- **容器登錄檔**：內建的 Docker 容器登錄檔
- **GitOps**：Kubernetes 部署整合
