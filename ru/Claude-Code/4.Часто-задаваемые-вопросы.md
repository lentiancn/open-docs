# Claude Code Часто Задаваемые Вопросы

Распространённые проблемы и их решения.

## Проблемы с Установкой

### Команда Не Найдена

Ошибка: `command not found: claude`

**Решение:** Проверьте PATH.

macOS/Linux:
```bash
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

Windows PowerShell:
```powershell
$currentPath = [Environment]::GetEnvironmentVariable('PATH', 'User')
[Environment]::SetEnvironmentVariable('PATH', "$currentPath;$env:USERPROFILE\.local\bin", 'User')
```

Проверка:
```bash
claude --version
```

### Скрипт Возвращает HTML

**Решение:** Используйте альтернативный метод:
- Homebrew: `brew install --cask claude-code`
- WinGet: `winget install Anthropic.ClaudeCode`

### Ошибки TLS/SSL

**Решение:**
1. Обновите сертификаты CA
2. Проверьте настройки прокси

## Настройка Сети

Установщик загружает с `storage.googleapis.com`:

```bash
curl -sI https://storage.googleapis.com
```

## Проверка Конфликтов

```bash
which -a claude
```

## Диагностика

Используйте `claude doctor`:

```bash
claude doctor
```
