# Guía de Instalación de Docker

Esta guía proporciona instrucciones detalladas para instalar Docker en varios sistemas operativos.

## Métodos de Instalación

Puedes instalar Docker de varias maneras:

1. **Docker Desktop**: Incluye Docker Engine en una aplicación de escritorio, recomendado para la mayoría de usuarios
2. **Solo Docker Engine**: Instala solo los componentes principales de Docker, adecuado para servidores Linux
3. **Docker Engine a través del repositorio apt**: Instalar desde el repositorio apt oficial de Docker
4. **Instalación Manual**: Descargar paquetes deb e instalar manualmente

## Instalación de Docker Desktop

Docker Desktop es la forma más fácil y rápida de comenzar. Incluye un entorno de desarrollo completo de Docker, incluyendo Docker Engine, Docker CLI, Docker Compose, Docker Build y Kubernetes.

### Requisitos del Sistema

**macOS:**

- macOS 10.15 (Catalina) o superior
- Al menos 4 GB de RAM
- Al menos 4 GB de espacio disponible en disco

**Windows:**

- Windows 10 Pro, Enterprise o Education (64-bit)
- Al menos 4 GB de RAM
- Hyper-V o WSL 2 activado

**Linux:**

- Ubuntu 20.04 o superior
- Debian 11 o superior
- Fedora 33 o superior
- Al menos 4 GB de RAM

### Instalación en macOS

1. Descarga Docker Desktop para Mac desde el sitio web oficial
2. Haz doble clic en el archivo .dmg
3. Arrastra el icono de Docker a la carpeta "Aplicaciones"
4. Inicia Docker Desktop

### Instalación en Windows

1. Descarga Docker Desktop para Windows
2. Ejecuta el instalador
3. Sigue las instrucciones del asistente de instalación
4. Reinicia Docker Desktop

### Instalación en Linux (Ubuntu)

```bash
# Actualiza la lista de paquetes
sudo apt-get update

# Instala las dependencias requeridas
sudo apt-get install ca-certificates curl gnupg lsb-release

# Añade la clave GPG de Docker
sudo mkdir -p /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg

# Configura el repositorio
echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

# Instala Docker
sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

## Instalación de Docker Engine

Si deseas instalar solo Docker Engine sin el escritorio, sigue estos pasos.

### Ubuntu y Debian

```bash
sudo apt-get update
sudo apt-get install docker.io
sudo systemctl start docker
sudo systemctl enable docker
```

### Fedora

```bash
sudo dnf install docker-ce
sudo systemctl start docker
sudo systemctl enable docker
```

## Verificar la Instalación

Después de la instalación, puedes verificar la instalación con el siguiente comando:

```bash
docker --version
docker run hello-world
```

Si todo funciona, verás un mensaje de bienvenida de Docker.
