# Apache APISIX 소개

## Apache APISIX란?

Apache APISIX는 API7.ai에서 개발하고 Apache Software Foundation에 기증한 클라우드 네이티브 API 게이트웨이입니다.

APISIX는 클라이언트와 백엔드 서비스 사이의 "트래픽 관리자" 역할을 합니다.

## 주요 기능

### 1. 동적 기능

- 동적 라우팅
- 동적 업스트림
- 동적 인증서
- 핫 리로드 플러그인

### 2. 트래픽 관리

- 로드밸런싱 (round-robin, 가중치, 일관된 해싱)
- 카나리 배포
- A/B 테스트
- 트래픽 분할
- 서킷 브레이커

### 3. 보안

- API 키 인증
- JWT 인증
- OAuth 2.0
- IP 화이트리스트/블랙리스트
- 레이트 리미팅

### 4. 관찰 가능성

- Prometheus 메트릭
- 분산 추적 (Zipkin, SkyWalking)
- 로그 내보내기 (Kafka, Elasticsearch)

### 5. 다중 프로토콜

- HTTP/HTTPS
- gRPC
- Dubbo
- MQTT
- WebSocket

## 아키텍처

- 데이터 플레인: Nginx + Lua
- 컨트롤 플레인: etcd
- 관리: REST API + Dashboard
