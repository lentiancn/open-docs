# OpenClaw 常见问题

本页面汇总了 OpenClaw 使用过程中最常见的问题和解答。如果你在这里没有找到答案，建议查看官方文档或加入社区寻求帮助。

---

## 基础问题

### 什么是 OpenClaw？

OpenClaw 是一个自托管的 AI 助手网关，它可以连接 WhatsApp、Telegram、Discord、iMessage 等多种聊天平台到一个 AI 编程助手（如 Pi）。你可以在自己的电脑上运行它，完全掌控自己的数据。

### OpenClaw 是免费的吗？

OpenClaw 本身是开源且免费的（MIT 许可证）。但使用 AI 模型（如 Anthropic Claude 或 OpenAI GPT）可能需要付费。你只需要支付 AI API 的使用费用。

### 我需要什么样的设备来运行 OpenClaw？

最低配置要求：
- Node.js 22+
- 2GB RAM
- 500MB 磁盘空间

推荐配置：
- 4GB+ RAM
- 稳定的网络连接
- 长期开机的电脑或服务器

## 安装问题

### 安装失败怎么办？

1. 检查 Node.js 版本：
   ```bash
   node --version
   ```
   确保版本 ≥ 22.0.0

2. 使用官方安装脚本：
   ```bash
   curl -fsSL https://openclaw.ai/install.sh | bash
   ```

3. 查看详细错误信息：
   ```bash
   npm install -g openclaw@latest --verbose
   ```

4. 运行诊断：
   ```bash
   openclaw doctor
   ```

### Windows 上安装有问题？

**强烈建议使用 WSL2**。OpenClaw 在 Linux 环境下运行最佳。

安装 WSL2：
```powershell
wsl --install
```

然后在 WSL2 中安装 OpenClaw。

### 如何升级 OpenClaw？

```bash
npm update -g openclaw@latest
```

或使用官方更新命令：
```bash
openclaw update
```

升级后运行诊断：
```bash
openclaw doctor
```

## 配置问题

### 配置文件在哪里？

默认位置：`~/.openclaw/openclaw.json`

你也可以指定自定义路径：
```bash
openclaw gateway --config /path/to/config.json
```

### 如何配置多个渠道？

在配置文件中添加多个渠道配置：

```json5
{
  channels: {
    whatsapp: { enabled: true },
    telegram: { enabled: true, botToken: "xxx" },
    discord: { enabled: true, token: "xxx" },
  },
}
```

### 配置不生效？

1. 确认配置文件格式正确（JSON5）
2. 检查 Gateway 是否在运行
3. 尝试重启 Gateway：
   ```bash
   openclaw gateway restart
   ```

## 连接问题

### Gateway 无法启动？

可能的原因：
- 端口 18789 被占用
- 配置文件格式错误
- 权限不足

解决方法：
```bash
# 查看端口占用
lsof -i :18789

# 检查配置
openclaw doctor

# 以管理员权限运行
sudo openclaw gateway
```

### 渠道连接失败？

1. 检查凭证是否有效
2. 查看渠道状态：
   ```bash
   openclaw channels status --probe
   ```
3. 查看日志：
   ```bash
   openclaw logs --follow
   ```

### 无法访问控制界面？

1. 确认 Gateway 正在运行
2. 检查 URL 是否正确：http://127.0.0.1:18789/
3. 检查防火墙设置
4. 尝试重新启动：
   ```bash
   openclaw gateway restart
   ```

## 使用问题

### AI 不回复消息？

检查以下几点：
1. 渠道是否已连接
2. 是否在白名单中
3. 群组中是否需要 @ 提及
4. 查看日志中的错误信息

### 如何重置会话？

使用 `/new` 或 `/reset` 命令，或在 CLI 中：
```bash
openclaw session reset
```

### 如何更改 AI 模型？

在配置文件中设置：

```json5
{
  agents: {
    defaults: {
      model: {
        primary: "anthropic/claude-sonnet-4-5",
      },
    },
  },
}
```

## 安全问题

### OpenClaw 安全吗？

OpenClaw 注重安全性：
- 所有数据存储在本地
- 凭证加密存储
- 支持白名单和配对机制
- 可启用沙盒隔离

建议：
- 使用白名单控制访问
- 定期更新版本
- 运行 `openclaw doctor` 检查安全

### 如何设置白名单？

```json5
{
  channels: {
    whatsapp: {
      allowFrom: ["+15555550123", "+15555550124"],
    },
  },
}
```

### 什么是配对（Pairing）？

配对是一种安全机制，未知用户首次发送消息时会被拦截，需要你批准后才能使用 AI。

批准用户：
```bash
openclaw pairing approve --channel whatsapp --sender +15555550123
```

## 功能问题

### 如何使用语音功能？

1. 配置 Talk Mode：
   ```json5
   {
     talk: {
       enabled: true,
       voice: "elevenlabs",
     },
   }
   ```

2. 在支持的设备上启用 Voice Wake

### 定时任务不执行？

1. 检查 Cron 是否启用：
   ```bash
   openclaw cron status
   ```

2. 查看任务列表：
   ```bash
   openclaw cron list
   ```

3. 查看运行日志：
   ```bash
   openclaw cron runs --id <jobId>
   ```

### 移动端节点无法连接？

1. 确保手机和 Gateway 在同一网络
2. 检查节点状态：
   ```bash
   openclaw nodes status
   ```

3. 重新配对节点：
   ```bash
   openclaw nodes pair
   ```

## 性能问题

### 运行缓慢怎么办？

1. 检查系统资源：
   ```bash
   openclaw status
   ```

2. 压缩会话：
   ```bash
   /compact
   ```

3. 清理旧日志：
   ```bash
   openclaw logs clean
   ```

### 如何限制资源使用？

```json5
{
  agents: {
    defaults: {
      maxTokens: 100000,
    },
  },
  session: {
    maxAgeHours: 24,
  },
}
```
