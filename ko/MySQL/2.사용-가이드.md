# MySQL 사용 가이드

기본 MySQL 작업입니다.

---

## 연결

```bash
mysql -u root -p
```

## 기본 명령어

```sql
SHOW DATABASES;
CREATE DATABASE mydb;
USE mydb;
```

## CRUD

```sql
INSERT INTO users (name) VALUES ('John');
SELECT * FROM users;
UPDATE users SET name='Jane' WHERE id=1;
DELETE FROM users WHERE id=1;
```
