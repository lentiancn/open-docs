# FantasyTalking - FAQ

Questions fréquemment posées et réponses.

## Problèmes d'installation

### Q : Échec de l'installation des dépendances ?

**Solution :**
1. Mettre à jour pip : `pip install --upgrade pip`
2. Utiliser un miroir chinois :
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```
3. Installer les paquets échoués individuellement

### Q : CUDA non disponible ?

**Solution :**
1. Confirmer que le pilote NVIDIA est installé : `nvidia-smi`
2. Installer la version compatible de PyTorch :
```bash
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
```
3. Vérifier la disponibilité de CUDA :
```python
import torch
print(torch.cuda.is_available())
```

### Q : Mémoire insuffisante ?

**Solution :**
1. Utiliser des images d'entrée plus petites
2. Réduire la taille du lot
3. Utiliser le mode CPU (non recommandé, lent)

## Problèmes d'utilisation

### Q : Échec de la génération, visage non trouvé ?

**Solution :**
1. Utiliser une image avec un visage clair
2. S'assurer que le visage est de face, sans occlusion
3. Vérifier le format de l'image

### Q : Génération vidéo très lente ?

**Solution :**
1. Utiliser l'accélération GPU
2. Réduire la résolution de sortie
3. Réduire les images de sortie

### Q : Lèvres non synchronisées ?

**Solution :**
1. S'assurer que l'audio est clair et sans bruit
2. Vérifier la fréquence d'échantillonnage audio
3. Essayer différents paramètres de synchronisation

### Q : Vidéo de sortie floue ?

**Solution :**
1. Utiliser l'amélioration du visage : `--face_enhancer realusf`
2. Augmenter la résolution de sortie
3. Utiliser une image source plus claire

### Q : Visage déformé/tordu ?

**Solution :**
1. Utiliser une photo frontale sans occlusion
2. S'assurer qu'il n'y a qu'un seul visage dans l'image
3. Ajuster les paramètres de détection de visage

## Problèmes de modèle

### Q : Téléchargement du modèle échoué ?

**Solution :**
1. Utiliser un proxy ou un miroir
2. Télécharger les fichiers du modèle manuellement
3. Vérifier la connexion réseau

### Q : Échec du chargement du modèle ?

**Solution :**
1. Confirmer que le fichier du modèle est complet
2. Vérifier que le chemin du modèle est correct
3. Re-télécharger le modèle

## Problèmes de performance

### Q : Comment améliorer la vitesse de génération ?

**Solution :**
1. Utiliser le GPU
2. Activer la précision mixte : `--precision fp16`
3. Utiliser une résolution plus petite

### Q : Utilisation de mémoire trop élevée ?

**Solution :**
1. Réduire la taille du lot
2. Utiliser un modèle plus petit
3. Désactiver les fonctionnalités inutiles

## Problèmes de sortie

### Q : La vidéo de sortie ne peut pas être lue ?

**Solution :**
1. Installer le décodeur : `pip install opencv-python`
2. Utiliser un format pris en charge : MP4

### Q : Fichier vidéo trop volumineux ?

**Solution :**
1. Réduire la résolution
2. Réduire le nombre d'images par seconde
3. Utiliser une compression plus élevée
