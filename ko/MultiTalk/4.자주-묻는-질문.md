# MultiTalk 자주 묻는 질문

이 페이지는 MultiTalk 사용자가 자주 묻는 질문과 답변을 모은 것입니다.

## 기본 질문

### MultiTalk란?

MultiTalk는 MeiGen-AI 팀에서 개발한 오디오 기반 다인물 대화 영상 생성 프레임워크입니다. 다중 스트림 오디오 입력, 참조 이미지 및 텍스트 프롬프트에 따라 여러 인물이 상호작용하는 영상을 생성할 수 있으며, 오디오와 입술 동기화를 지원합니다.

### MultiTalk는 어떤 기능을 지원합니까?

MultiTalk는 다양한 기능을 지원합니다: 단일 인물 및 다인물 대화 영상 생성, 만화 캐릭터 영상 생성, 노래 영상 생성, 명령 Following 영상 생성. 480P 및 720P 해상도 출력을 지원하며 최대 15초의 비디오 클립을 생성할 수 있습니다.

### MultiTalk는 무료입니까?

MultiTalk는 오픈소스 프로젝트이며 무료로 사용할 수 있습니다. 프로젝트는 Apache 2.0 라이선스를 사용하며 학술 및 상업적 사용이 허용됩니다.

## 설치 문제

### 설치 중 CUDA 버전이 일치하지 않으면 어떻게 합니까?

MultiTalk에는 CUDA 12.1 이상이 필요합니다. 먼저 시스템 CUDA 버전을 확인하세요: nvcc --version. 버전이 너무 낮으면 CUDA 드라이버를 업그레이드해야 합니다. 해당 PyTorch CUDA 버전을 사용해 볼 수도 있습니다.

### VRAM이 부족하면 어떻게 합니까?

VRAM이 부족한 경우 다음 방법을 시도해 볼 수 있습니다: --num_persistent_param_in_dit 0 매개변수를 사용하여 메모리 사용량 줄이기; TeaCache 가속을 사용하여 메모리 사용량 줄이기; LoRA 가속을 사용하여 단계 줄이기; 출력 해상도 낮추기.

### flash_attn 설치가 실패하면 어떻게 합니까?

flash_attn 설치에는 특정 컴파일 환경이 필요합니다. ninja와 올바른 CUDA 툴킷이 설치되어 있는지 확인하세요. 계속 실패하면 사전 컴파일된 버전을 설치하거나 이 종속성을 건너뛰세요.

### 모델 다운로드가 너무 느리면 어떻게 합니까?

미러 다운로드 도구를 사용하여 모델 파일을 미리 다운로드한 다음 지정된 디렉토리로 이동할 수 있습니다. 국내 미러 원본을 사용해 볼 수도 있습니다.

## 사용 문제

### 생성된 영상의 입술이 동기화되지 않으면 어떻게 합니까?

Audio CFG 매개변수 값을 높여 보세요, 권장 설정은 3~5 사이입니다. 동기화를 개선하기 위해 샘플링 단계 늘릴 수도 있습니다.

### 비디오 생성이 너무 느리면 어떻게 합니까?

다음 가속 방법을 활성화할 수 있습니다: TeaCache 가속 사용, 속도를 2~3배 향상시킬 수 있음; LoRA 가속 사용, FusioniX는 8단계, lightx2v는 4단계만 필요; 계산량을 줄이기 위해 양자화 버전 사용.

### 더 긴 인물 대화 영상을 생성하려면 어떻게 합니까?

streaming 모드를 사용하여 긴 영상을 생성합니다. 색조 일관성을 유지하려면 Audio CFG가 3으로 설정되어 있는지 확인하세요. 생성된 영상은 최대 15초까지 지속될 수 있습니다.

### 다인물 영상에서 인물 바인딩이 잘못되면 어떻게 합니까?

이는 다인물 영상 생성의 일반적인 문제입니다. MultiTalk는 L-RoPE 기술을 통해 이를 해결합니다. 입력 JSON에서 각 인물 영역의 좌표가 올바르게 지정되어 있는지 확인하세요.

### TTS 텍스트 음성 변환 기능을 사용하려면 어떻게 합니까?

텍스트가 포함된 입력 JSON 파일을 준비하고, --audio_mode tts 매개변수를 설정하면 텍스트가 자동으로 음성으로 변환되어 영상이 생성됩니다.

### 만화 캐릭터 영상을 생성할 수 있습니까?

네, MultiTalk는 만화 캐릭터 생성을 지원합니다. 참조 이미지에 만화 인물 이미지를 해당 설명 텍스트와 함께 사용하세요.

## 기술 문제

### 어떤 입력 오디오 형식이 지원됩니다?

WAV, MP3와 같은 일반적인 오디오 형식이 지원됩니다. 오디오에는 명확한 인간 음성이 포함되어야 합니다.

### 비디오 해상도를 어떻게 선택합니까?

현재 버전은 기본적으로 480P를 지원합니다. 720P는 다중 GPU 구성이 필요합니다. 480P는 대부분의 사용 시나리오에 충분합니다.

### 저 VRAM 기기에서 실행하려면 어떻게 합니까?

--num_persistent_param_in_dit 0 매개변수를 사용하면 VRAM 요구 사항을 크게 줄일 수 있어 단일 RTX 4090에서도 작동합니다.

### 다른 플랫폼과 통합하려면 어떻게 합니까?

MultiTalk는 여러 사용 방법을 지원합니다: Replicate 클라우드 플랫폼을 통해 직접 호출 가능; ComfyUI 워크플로우에 통합 가능; 프로젝트에서 제공하는 Gradio 인터페이스도 사용 가능.

## 라이선스 문제

### 생성된 영상을 상업적으로 사용할 수 있습니까?

프로젝트는 Apache 2.0 라이선스를 사용하며 생성된 영상 콘텐츠는 상업적 목적으로 사용할 수 있습니다. 그러나 사용이 라이선스 약관 및 관련 법률 및 규정을 준수하는지 확인하세요.

### 사용할 때 무엇을 인용해야 합니까?

MultiTalk가 연구에 도움이 되면 관련 논문을 인용해 주세요. 논문 정보는 프로젝트 GitHub 페이지에서 확인할 수 있습니다.
