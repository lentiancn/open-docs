# Документация Node.js

## Обзор

Node.js® — это бесплатная, с открытым исходным кодом, кроссплатформенная среда выполнения JavaScript, которая выполняет код JavaScript вне веб-браузера. Она позволяет разработчикам использовать JavaScript для серверного скриптинга, позволяя создавать динамический веб-контент до того, как страницы будут отправлены в браузер пользователя.

Построенный на движке JavaScript V8 от Chrome, Node.js использует модель ввода-вывода, ориентированную на события и неблокирующую, что делает его легким и эффективным для приложений реального времени, интенсивно использующих данные, которые работают на распределенных устройствах.

## Основные возможности

- **Асинхронный и ориентированный на события**: Все API являются асинхронными, что означает, что сервер не ждет, пока API вернет данные, прежде чем перейти к следующей задаче.
- **Однопоточный, но высоко масштабируемый**: Использует однопоточную модель с циклом событий, что делает его высоко масштабируемым по сравнению с традиционными многопоточными серверами.
- **Без буферизации**: Приложения никогда не буферизуют никакие данные; они просто выводят данные частями.
- **Кроссплатформенный**: Работает на Windows, macOS, Linux и других операционных системах.

## Первые шаги

### Установка
Посетите [официальный сайт Node.js](https://nodejs.org/en/download), чтобы скачать и установить Node.js для вашей операционной системы.

### Первое приложение
Создайте простой HTTP-сервер:

```javascript
const http = require('http');

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello World\n');
});

server.listen(3000, '127.0.0.1', () => {
  console.log('Server running at http://127.0.0.1:3000/');
});
```

Сохраните как `server.js` и запустите с помощью `node server.js`.
