# OpenClaw 🦞

Инструмент для подключения мессенджеров к ИИ. WhatsApp, Telegram, Discord, iMessage — всё работает с ИИ-сервисами.

## Для кого?

Для тех, кто хочет своего ИИ-помощника без передачи данных третьим лицам. Особенно для разработчиков.

## Возможности

- **Ваши данные на вашем устройстве**
- **Несколько платформ**: один процесс для многих чатов
- **Для программирования**: инструменты, история, память
- **Открытый код**: MIT лицензия

## Что нужно

- Node.js 22+
- API-ключ от ИИ-провайдера

## Быстрый старт

```bash
npm install -g openclaw@latest
openclaw onboard --install-daemon
openclaw gateway --port 18789
```

Откройте http://127.0.0.1:18789/
