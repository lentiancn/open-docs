# Angular

現代 Web 應用開發框架。

## 簡介

Angular 是由 Google 開發維護的開源 Web 應用框架。

### 主要特性

- **TypeScript 原生**
- **組件化架構**
- **依賴注入**
- **路由系統**
- **表單處理**

### 統計

- GitHub 星標：93,000+

## 快速開始

### 安裝 Angular CLI

```bash
npm install -g @angular/cli
```

### 創建新項目

```bash
ng new my-app
ng serve
```

## 常用命令

```bash
ng g c my-component
ng g s my-service
ng build
```

## 核心概念

- **組件**
- **服務**
- **路由**
- **指令**
- **管道**
