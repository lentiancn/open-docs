# Element UI 安装指南

> Element UI 是基于 Vue 2 的桌面端组件库
> 
> 更新日期：2026年2月
> 
> 当前版本：2.x（稳定版）
> 
> 注意：如果是 Vue 3 项目，请使用 Element Plus

---

## 一、项目简介

Element UI 是一套基于 Vue 2.0 的桌面端组件库，由饿了么团队开发和维护。 它提供了丰富的 PC 端常用组件，涵盖了表单、表格、对话框、导航菜单、按钮、输入框等常用 UI 元素。

Element UI 的设计风格简洁大方，文档详细完善，是国内最流行的 Vue 2 UI 组件库之一。虽然官方已经停止更新（推荐迁移到 Element Plus for Vue 3），但由于国内大量存量项目仍在使用，Element UI 仍然具有很高的参考价值和学习意义。

### 1.1 版本说明

| 版本 | Vue 版本 | 状态 | 说明 |
|------|----------|------|------|
| Element UI 2.x | Vue 2 | 稳定 | 当前文档主要版本 |
| Element UI 1.x | Vue 2 | 停止维护 | 已不再推荐使用 |

### 1.2 与 Element Plus 的区别

| 特性 | Element UI | Element Plus |
|------|------------|--------------|
| Vue 版本 | Vue 2 | Vue 3 |
| TypeScript | 部分支持 | 完全支持 |
| Composition API | 不支持 | 支持 |
| 维护状态 | 已停止 | 活跃维护 |
| 发布年份 | 2016-2021 | 2022-至今 |

## 三、安装步骤

### 3.1 使用 npm 安装（推荐）

```bash
# 进入 Vue 2 项目目录
cd your-vue-project

# 安装 Element UI
npm install element-ui --save
```

### 3.2 使用 yarn 安装

```bash
yarn add element-ui
```

### 3.3 指定版本安装

```bash
# 安装最新 2.x 版本
npm install element-ui@latest

# 安装特定版本
npm install element-ui@2.15.13
```

## 四、在 Vue 2 项目中引入

### 4.1 完整引入（适合小型项目）

在项目的入口文件（通常是 main.js）中引入 Element UI 和它的样式文件：

```javascript
// main.js
import Vue from 'vue'
import App from './App.vue'

// 引入 Element UI
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)

new Vue({
  render: h => h(App)
}).$mount('#app')
```

### 4.2 按需引入（推荐，适合生产环境）

为了减少项目体积，建议按需引入只需要的组件。

#### 4.2.1 安装 babel-plugin-component

```bash
npm install babel-plugin-component -D
```

#### 4.2.2 配置 babel

在 .babelrc 或 babel.config.js 中添加配置：

```json
{
  "presets": [
    ["env", {
      "modules": false,
      "targets": {
        "browsers": ["> 1%", "last 2 versions", "not ie <= 8"]
      }
    }]
  ],
  "plugins": [
    ["component", {
      "libraryName": "element-ui",
      "styleLibraryName": "theme-chalk"
    }]
  ]
}
```

#### 4.2.3 按需引入组件

```javascript
// main.js
import Vue from 'vue'
import { Button, Form, FormItem, Input, Table, TableColumn } from 'element-ui'
import App from './App.vue'

// 注册需要的组件
Vue.use(Button)
Vue.use(Form)
Vue.use(FormItem)
Vue.use(Input)
Vue.use(Table)
Vue.use(TableColumn)

new Vue({
  render: h => h(App)
}).$mount('#app')
```

### 4.3 使用 CDN 引入

如果你不想使用构建工具，可以通过 CDN 方式引入 Element UI：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Element UI CDN 示例</title>
  <!-- 引入 Vue 2 -->
  <script src="https://cdn.jsdelivr.net/npm/vue@2.6.14/dist/vue.js"></script>
  <!-- 引入 Element UI -->
  <link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
  <script src="https://unpkg.com/element-ui"></script>
</head>
<body>
  <div id="app">
    <el-button type="primary" @click="visible = true">打开对话框</el-button>
    <el-dialog title="提示" :visible.sync="visible" width="30%">
      <span>这是一段信息</span>
      <span slot="footer" class="dialog-footer">
        <el-button @click="visible = false">取消</el-button>
        <el-button type="primary" @click="visible = false">确定</el-button>
      </span>
    </el-dialog>
  </div>

  <script>
    new Vue({
      el: '#app',
      data: {
        visible: false
      }
    })
  </script>
</body>
</html>
```

## 五、快速开始示例

### 5.1 基础按钮

```html
<template>
  <div>
    <el-button>默认按钮</el-button>
    <el-button type="primary">主要按钮</el-button>
    <el-button type="success">成功按钮</el-button>
    <el-button type="info">信息按钮</el-button>
    <el-button type="warning">警告按钮</el-button>
    <el-button type="danger">危险按钮</el-button>
  </div>
</template>
```

### 5.2 基础表单

```html
<template>
  <el-form :model="form" :rules="rules" ref="loginForm" label-width="80px">
    <el-form-item label="用户名" prop="username">
      <el-input v-model="form.username" placeholder="请输入用户名"></el-input>
    </el-form-item>
    <el-form-item label="密码" prop="password">
      <el-input v-model="form.password" type="password" placeholder="请输入密码"></el-input>
    </el-form-item>
    <el-form-item>
      <el-button type="primary" @click="submitForm">登录</el-button>
      <el-button @click="resetForm">重置</el-button>
    </el-form-item>
  </el-form>
</template>

<script>
export default {
  data() {
    return {
      form: {
        username: '',
        password: ''
      },
      rules: {
        username: [
          { required: true, message: '请输入用户名', trigger: 'blur' }
        ],
        password: [
          { required: true, message: '请输入密码', trigger: 'blur' }
        ]
      }
    }
  },
  methods: {
    submitForm() {
      this.$refs.loginForm.validate(valid => {
        if (valid) {
          console.log('提交登录表单：', this.form)
        } else {
          this.$message.error('表单验证失败')
        }
      })
    },
    resetForm() {
      this.$refs.loginForm.resetFields()
    }
  }
}
</script>
```

### 5.3 基础表格

```html
<template>
  <el-table :data="tableData" stripe border>
    <el-table-column prop="date" label="日期" width="180"></el-table-column>
    <el-table-column prop="name" label="姓名" width="180"></el-table-column>
    <el-table-column prop="address" label="地址"></el-table-column>
    <el-table-column label="操作" width="180">
      <template slot-scope="scope">
        <el-button size="mini" @click="handleEdit(scope.$index, scope.row)">编辑</el-button>
        <el-button size="mini" type="danger" @click="handleDelete(scope.$index, scope.row)">删除</el-button>
      </template>
    </el-table-column>
  </el-table>
</template>

<script>
export default {
  data() {
    return {
      tableData: [{
        date: '2024-01-01',
        name: '张三',
        address: '北京市朝阳区'
      }, {
        date: '2024-01-02',
        name: '李四',
        address: '上海市浦东新区'
      }]
    }
  },
  methods: {
    handleEdit(index, row) {
      console.log('编辑：', index, row)
    },
    handleDelete(index, row) {
      console.log('删除：', index, row)
    }
  }
}
</script>
```

### 5.4 对话框

```html
<template>
  <el-button type="text" @click="dialogVisible = true">打开对话框</el-button>

  <el-dialog
    title="编辑信息"
    :visible.sync="dialogVisible"
    width="30%"
    :before-close="handleClose">
    <span>这是一段对话框内容</span>
    <span slot="footer" class="dialog-footer">
      <el-button @click="dialogVisible = false">取 消</el-button>
      <el-button type="primary" @click="dialogVisible = false">确 定</el-button>
    </span>
  </el-dialog>
</template>

<script>
export default {
  data() {
    return {
      dialogVisible: false
    }
  },
  methods: {
    handleClose(done) {
      this.$confirm('确认关闭？')
        .then(_ => {
          done()
        })
        .catch(_ => {})
    }
  }
}
</script>
```

## 六、国际化配置

Element UI 默认使用英文，要使用中文需要配置国际化。

### 6.1 全局中文配置

```javascript
// main.js
import Vue from 'vue'
import ElementUI from 'element-ui'
import locale from 'element-ui/lib/locale/lang/zh-CN'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI, {
  locale
})
```

## 七、主题定制

### 7.1 使用在线主题工具

访问 Element UI 主题定制页面：https://elementui.github.io/theme-chalk/

可以选择主题色、边框颜色、圆角等参数，下载定制后的样式文件。

### 7.2 手动覆盖变量

在项目中创建自定义样式文件：

```scss
// styles/element-variables.scss
$--color-primary: #409EFF;
$--color-success: #67C23A;
$--color-warning: #E6A23C;
$--color-danger: #F56C6C;
$--color-info: #909399;

// 覆盖组件变量
@import 'element-ui/packages/theme-chalk/src/common/var.scss';
```

然后在 main.js 中引入：

```javascript
import 'element-ui/lib/theme-chalk/index.css'
import './styles/element-variables.scss'
```

## 八、常见问题

### 8.1 安装失败

**问题**：npm install 安装失败

**解决方案**：

1. 清除 npm 缓存：`npm cache clean --force`
2. 检查网络，可以使用国内镜像源

### 8.2 样式不生效

**问题**：组件样式显示异常

**解决方案**：

1. 确保已引入样式文件：`import 'element-ui/lib/theme-chalk/index.css'`
2. 检查 webpack 配置是否正确处理了 CSS
3. 确保没有 CSS 冲突

### 8.3 组件不显示

**问题**：组件引入后不显示

**解决方案**：

1. 检查是否正确使用了 Vue.use(ElementUI)
2. 确认 Vue 版本是否在 2.6.0 以上
3. 检查浏览器控制台是否有错误信息

## 九、版本迁移建议

Element UI 已停止维护，强烈建议新项目使用 Element Plus (Vue 3)。如果现有项目计划迁移，可以参考以下步骤：

### 9.1 主要变化

| 变化项 | Element UI | Element Plus |
|--------|------------|--------------|
| Vue 版本 | Vue 2 | Vue 3 |
| 事件绑定 | v-model (Vue 2 语法) | v-model (Vue 3 语法) |
| 组件注册 | Vue.component | app.component |

### 9.2 迁移工具

官方提供了迁移工具 gogocode，可以帮助自动化迁移：

```bash
# 全局安装
npm install -g gogocode

# 执行迁移
gogocode -s your-project-path -t element-plus -o
```
