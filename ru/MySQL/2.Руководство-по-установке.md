# Руководство по установке MySQL

## Установка в Windows

### Установка с помощью MySQL Installer

MySQL Installer — самый простой способ установки MySQL в Windows, предоставляющий установку через мастер.

1. Скачать MySQL Installer
   Посетите https://dev.mysql.com/downloads/installer/ для загрузки установщика.

2. Запустить установщик
   Дважды щёлкните по загруженному файлу MSI, чтобы запустить мастер установки.

3. Выбрать тип установки
   - **Full**: установить все продукты и функции MySQL
   - **Developer Default**: подходит для среды разработки
   - **Server Only**: установить только MySQL Server
   - **Custom**: выбрать конкретные компоненты

4. Настроить MySQL Server
   - Выбрать версию MySQL Server
   - Установить номер порта (по умолчанию 3306)
   - Выбрать тип запуска

5. Установить пароль root
   Установите надёжный пароль для учётной записи администратора MySQL.

6. Завершить установку
   Следуйте мастеру для завершения процесса установки.

### Ручная установка с ZIP-архивом

1. Скачать ZIP-архив MySQL
   Посетите https://dev.mysql.com/downloads/mysql/ для загрузки ZIP-архива.

2. Распаковать файлы
   Распакуйте ZIP-архив в нужный каталог, например C:\mysql.

3. Инициализировать каталог данных
   ```
   mysqld --initialize-insecure
   ```

4. Запустить службу MySQL
   ```
   mysqld --install MySQL
   net start MySQL
   ```

5. Войти и изменить пароль
   ```
   mysql -u root -p
   ALTER USER 'root'@'localhost' IDENTIFIED BY 'новый_пароль';
   ```

## Установка в Linux

### Системы Ubuntu/Debian

1. Обновить индекс пакетов
   ```bash
   sudo apt update
   ```

2. Установить MySQL Server
   ```bash
   sudo apt install mysql-server
   ```

3. Запустить службу MySQL
   ```bash
   sudo systemctl start mysql
   sudo systemctl enable mysql
   ```

4. Безопасная настройка
   ```bash
   sudo mysql_secure_installation
   ```

### Системы CentOS/RHEL

1. Добавить репозиторий MySQL Yum
   ```bash
   sudo yum install https://dev.mysql.com/get/mysql80-community-release-el8-4.noarch.rpm
   ```

2. Установить MySQL Server
   ```bash
   sudo yum install mysql-community-server
   ```

3. Запустить службу
   ```bash
   sudo systemctl start mysqld
   sudo systemctl enable mysqld
   ```

4. Получить временный пароль
   ```bash
   sudo grep 'temporary password' /var/log/mysqld.log
   ```

## Установка в macOS

### Использование Homebrew

1. Установить Homebrew (если не установлен)
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```

2. Установить MySQL
   ```bash
   brew install mysql
   ```

3. Запустить службу MySQL
   ```bash
   brew services start mysql
   ```

4. Безопасная настройка
   ```bash
   mysql_secure_installation
   ```

### Использование DMG-пакета

1. Скачать DMG-пакет
   Посетите https://dev.mysql.com/downloads/mysql/ для загрузки DMG-пакета для macOS.

2. Запустить установщик
   Дважды щёлкните по файлу PKG и следуйте мастеру для завершения установки.

3. Запустить MySQL
   Запустите службу MySQL через Системные настройки.

## Установка MySQL с Docker

### Загрузить образ MySQL
```bash
docker pull mysql:latest
```

### Запустить контейнер MySQL
```bash
docker run -d \
  --name mysql-container \
  -e MYSQL_ROOT_PASSWORD=ваш_пароль \
  -e MYSQL_DATABASE=testdb \
  -p 3306:3306 \
  mysql:latest
```

### Общие параметры Docker
- `-e MYSQL_ROOT_PASSWORD`: установить пароль root
- `-e MYSQL_DATABASE`: создать начальную базу данных
- `-p 3306:3306`: сопоставление портов
- `-v /data/mysql:/var/lib/mysql`: том для сохранения данных

## Проверка после установки

### Проверить версию MySQL
```bash
mysql --version
```

### Проверить статус службы
```bash
systemctl status mysql   # Linux
brew services list      # macOS
```

### Войти в MySQL
```bash
mysql -u root -p
```

## Распространённые проблемы установки

### Конфликты портов
Если порт 3306 занят, измените порт в файле конфигурации или остановите службу, использующую этот порт.

### Проблемы с разрешениями
В Linux используйте sudo для повышения прав и убедитесь, что каталог данных имеет правильные разрешения на чтение/запись.

### Настройки брандмауэра
Убедитесь, что брандмауэр разрешает доступ к порту 3306.
