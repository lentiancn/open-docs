# Java FAQ

> Häufig gestellte Fragen

---

## Installation

### javac nicht gefunden?

**Lösung**: JAVA_HOME und PATH setzen.

### Java Version wechseln?

```bash
export JAVA_HOME=/path/to/jdk
```

---

## Entwicklung

### JDK vs JRE?

- **JRE**: Laufzeitumgebung
- **JDK**: Entwicklungsumgebung

### == vs equals()?

- **==**: Referenzvergleich
- **equals()**: Inhaltsvergleich

---

## Sammlungen

### ArrayList vs LinkedList?

- **ArrayList**: Schneller Zugriff
- **LinkedList**: Schnelle Einfügung
