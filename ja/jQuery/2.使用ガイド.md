# jQuery 使用ガイド

jQuery を使用するガイド。

---

## 基本

```javascript
$('#myId')
$(document).ready(function() {})
```

---

## セレクター

```javascript
$('#myId')
$('.myClass')
$('div')
```

---

## イベント

```javascript
$('#myButton').click(function() {})
```

---

## AJAX

```javascript
$.get('url', function(data) {})
$.post('url', data, function(data) {})
```
