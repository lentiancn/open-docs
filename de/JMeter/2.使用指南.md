# JMeter Benutzeranleitung

Diese Anleitung beschreibt, wie Sie JMeter für Leistungstests verwenden.

## Schnellstart

### Testplan erstellen

1. JMeter öffnen
2. Testplan erstellen
3. Thread-Gruppe hinzufügen
4. Sampler hinzufügen
5. Listener hinzufügen
6. Test ausführen

## Thread-Gruppen

### Thread-Gruppe konfigurieren

```
Anzahl der Threads：10
 Ramp-Up Zeit：5 Sekunden
 Schleifenanzahl：1
```

### Thread-Eigenschaften

- Anzahl der Threads: Gleichzeitige Benutzer
- Ramp-Up Zeit: Startzeit
- Schleifenanzahl: Ausführungen pro Benutzer

## Verteilte Tests

### Master/Slave-Konfiguration

```bash
# Slave-Konfiguration
remote_hosts=192.168.1.2,192.168.1.3

# Starten
jmeter -R 192.168.1.2
```

### Ausführung

```bash
# Verteilte Ausführung
jmeter -n -t test.jmx -R server1,server2
```

## Befehlszeilenausführung

### Grundbefehle

```bash
# Test ausführen
jmeter -n -t test.jmx -l result.jtl

# Bericht generieren
jmeter -g result.jtl -o report
```

### Parameter

```bash
-n Nicht-GUI-Modus
-t Testplandatei
-l Ergebnissdatei
-r Verteilte Ausführung
```
