# Guía de Uso de Element Plus

Esta guía cubre cómo usar los componentes de Element Plus efectivamente en su aplicación Vue 3.

## Uso Básico

### Ejemplo Hello World

```vue
<template>
  <div>
    <el-button type="primary">Hello Element Plus</el-button>
  </div>
</template>

<script setup>
import { ElButton } from 'element-plus'
</script>
```

Con auto-importación (recomendado):

```vue
<template>
  <div>
    <el-button type="primary">Haz Clic</el-button>
    <el-input v-model="input" placeholder="Ingrese texto"></el-input>
  </div>
</template>

<script setup>
import { ref } from 'vue'

const input = ref('')
</script>
```

## Categorías de Componentes

Element Plus proporciona estas categorías de componentes:

| Categoría | Componentes |
|----------|------------|
| Básico | Button, Link, Typography, Divider |
| Formulario | Input, Select, Radio, Checkbox, Switch, Slider, TimePicker, DatePicker |
| Datos | Table, Tree, Pagination, Tag |
| Notificación | Alert, Message, Notification, Progress |
| Navegación | Menu, Tabs, Breadcrumb, Steps |
| Otros | Dialog, Drawer, Card, Collapse |

### Formulario

```vue
<template>
  <el-form
    ref="ruleFormRef"
    :model="form"
    :rules="rules"
    label-width="120px"
  >
    <el-form-item label="Nombre" prop="name">
      <el-input v-model="form.name"></el-input>
    </el-form-item>
    
    <el-form-item label="Email" prop="email">
      <el-input v-model="form.email"></el-input>
    </el-form-item>
    
    <el-form-item label="Género" prop="gender">
      <el-radio-group v-model="form.gender">
        <el-radio label="male">Masculino</el-radio>
        <el-radio label="female">Femenino</el-radio>
      </el-radio-group>
    </el-form-item>
    
    <el-form-item label="Pasatiempo" prop="hobby">
      <el-checkbox-group v-model="form.hobby">
        <el-checkbox label="Lectura">Lectura</el-checkbox>
        <el-checkbox label="Deportes">Deportes</el-checkbox>
        <el-checkbox label="Música">Música</el-checkbox>
      </el-checkbox-group>
    </el-form-item>
    
    <el-form-item label="País" prop="country">
      <el-select v-model="form.country" placeholder="Seleccionar país">
        <el-option label="España" value="spain"></el-option>
        <el-option label="México" value="mexico"></el-option>
        <el-option label="Argentina" value="argentina"></el-option>
      </el-select>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm(ruleFormRef)">
        Enviar
      </el-button>
      <el-button @click="resetForm(ruleFormRef)">Reiniciar</el-button>
    </el-form-item>
  </el-form>
</template>

<script setup>
import { reactive, ref } from 'vue'

const ruleFormRef = ref()

const form = reactive({
  name: '',
  email: '',
  gender: '',
  hobby: [],
  country: '',
})

const rules = reactive({
  name: [
    { required: true, message: 'Por favor ingrese el nombre', trigger: 'blur' },
  ],
  email: [
    { required: true, message: 'Por favor ingrese el email', trigger: 'blur' },
    { type: 'email', message: 'Formato de email inválido', trigger: 'blur' },
  ],
  gender: [
    { required: true, message: 'Por favor seleccione el género', trigger: 'change' },
  ],
  country: [
    { required: true, message: 'Por favor seleccione el país', trigger: 'change' },
  ],
})

const submitForm = async (formEl) => {
  if (!formEl) return
  await formEl.validate((valid) => {
    if (valid) {
      console.log('Enviar:', form)
    }
  })
}

const resetForm = (formEl) => {
  if (!formEl) return
  formEl.resetFields()
}
</script>
```

## Slots

Los componentes de Element Plus soportan slots de Vue:

```vue
<template>
  <!-- Slot predeterminado -->
  <el-button>Texto del Botón</el-button>
  
  <!-- Slot nombrado -->
  <el-dialog>
    <template #header>
      <h3>Encabezado Personalizado</h3>
    </template>
    <p>Contenido del diálogo</p>
    <template #footer>
      <el-button>Cancelar</el-button>
      <el-button type="primary">Aceptar</el-button>
    </template>

  <!-- Slot vacío -->
  <el-table-column>
    <template #empty>
      <div>No Hay Datos Disponibles</div>
    </template>
  </el-table-column>
</template>
```

## Eventos

```vue
<template>
  <el-button @click="handleClick">Haz Clic</el-button>
  
  <el-input
    v-model="input"
    @focus="handleFocus"
    @blur="handleBlur"
    @change="handleChange"
    @input="handleInput"
  ></el-input>
  
  <el-table
    :data="tableData"
    @select="handleSelect"
    @selection-change="handleSelectionChange"
    @row-click="handleRowClick"
  ></el-table>
</template>

<script setup>
const handleClick = () => {
  console.log('Botón presionado')
}

const handleFocus = () => {
  console.log('Entrada enfocada')
}

const handleBlur = () => {
  console.log('Entrada desenfocada')
}

const handleChange = (value) => {
  console.log('Valor cambiado:', value)
}

const handleInput = (value) => {
  console.log('Entrada:', value)
}

const handleSelect = (selection, row) => {
  console.log('Seleccionado:', selection, row)
}

const handleSelectionChange = (selection) => {
  console.log('Selección cambiada:', selection)
}

const handleRowClick = (row) => {
  console.log('Fila presionada:', row)
}
</script>
```

## v-model

Element Plus soporta v-model para enlace bidireccional:

```vue
<template>
  <!-- Entrada -->
  <el-input v-model="inputValue"></el-input>
  
  <!-- Casilla de verificación -->
  <el-checkbox v-model="checked">Etiqueta</el-checkbox>
  <el-checkbox-group v-model="checkboxGroup">
    <el-checkbox label="A">A</el-checkbox>
    <el-checkbox label="B">B</el-checkbox>
  </el-checkbox-group>
  
  <!-- Radio -->
  <el-radio-group v-model="radio">
    <el-radio label="A">A</el-radio>
    <el-radio label="B">B</el-radio>
  </el-radio-group>
  
  <!-- Interruptor -->
  <el-switch v-model="switchValue"></el-switch>
  
  <!-- Selector -->
  <el-select v-model="selectValue">
    <el-option label="Opción 1" value="1"></el-option>
    <el-option label="Opción 2" value="2"></el-option>
  </el-select>
  
  <!-- Deslizador -->
  <el-slider v-model="sliderValue"></el-slider>
</template>

<script setup>
import { ref } from 'vue'

const inputValue = ref('')
const checked = ref(true)
const checkboxGroup = ref(['A'])
const radio = ref('A')
const switchValue = ref(false)
const selectValue = ref('')
const sliderValue = ref(50)
</script>
```

## Mejores Prácticas

### 1. Usar Importación Bajo Demanda

```typescript
// Malo - Importación completa
import ElementPlus from 'element-plus'

// Bueno - Importación bajo demanda vía plugin de auto-importación
// Componentes auto-importados cuando se usan
```

### 2. Usar Composition API

```vue
<script setup>
<!-- Recomendado -->
import { ref, computed, onMounted } from 'vue'

const count = ref(0)
const doubled = computed(() => count.value * 2)

onMounted(() => {
  console.log('Montado')
})
</script>
```

### 3. Manejar Formularios Correctamente

```vue
<script setup>
const formRef = ref()
const form = reactive({ ... })
const rules = reactive({ ... })

const handleSubmit = async () => {
  if (!formRef.value) return
  await formRef.value.validate((valid) => {
    if (valid) {
      // Enviar
    }
  })
}
</script>
```

### 4. Usar Slots para Personalización

```vue
<el-table-column>
  <template #default="{ row }">
    <span>{{ row.name }}</span>
  </template>
</el-table-column>
```

Los componentes de Element Plus incluyen atributos ARIA:

```vue
<!-- Automático -->

<!-- Con aria-label -->

```

## Consejos de Rendimiento

1. **Usar importación bajo demanda** - Reduce el tamaño del bundle
2. **Cargar diálogos lazy** - Para diálogos grandes
3. **Scroll virtual** - Para tablas grandes
4. **Valores computados en caché** - Evitar cálculos redundantes

## Problemas Comunes

### Componente No Registrado

```typescript
// Solución: Verificar configuración de auto-importación
// vite.config.ts
Components({
  resolvers: [ElementPlusResolver()],
})
```

### Conflictos de Estilos

```css
/* Solución: Usar selectores más específicos */
.el-button.my-button {
  /* estilos personalizados */
}
```

### Evento No Se Dispara

```vue
<!-- Solución: Usar modificador .native para eventos nativos -->
<el-input @keyup.enter.native="handleEnter"></el-input>

<!-- O usar evento @key en Vue 3 -->
<el-input @keyup.enter="handleEnter"></el-input>
```
