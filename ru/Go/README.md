# Go

Go - Простой, Эффективный, Параллельный

## Обзор

Go (также известный как Golang) — язык программирования с открытым исходным кодом, разработанный Google, представленный в 2009 году и особенно подходящий для создания современных распределённых систем и облачных приложений.

## Возможности

- **Параллелизм**: Лёгкие горутины и каналы
- **Сборка мусора**: Встроенное автоматическое управление памятью
- **Быстрая компиляция**: Инкрементная компиляция с чрезвычайно быстрым временем сборки
- **Статическая компоновка**: Компилируется в один исполняемый файл
- **Богатая стандартная библиотека**: Комплексная стандартная библиотека
- **Кроссплатформенность**: Поддерживает Linux, macOS, Windows и другое

## Быстрый старт

### Установка

```bash
# Linux/macOS
wget https://go.dev/dl/go1.21.5.linux-amd64.tar.gz
sudo tar -C /usr/local -xzf go1.21.5.linux-amd64.tar.gz

# macOS (Homebrew)
brew install go
```

### Проверка

```bash
go version
```

### Первая программа

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, Go!")
}
```

Запуск:

```bash
go run main.go
```
