# Manuel d'utilisation

Ce manuel détaille les fonctionnalités principales et les méthodes d'utilisation de Bootstrap 5, incluant le système de grille, les composants et les classes utilitaires.

## Système de grille

Le système de grille est la fonctionnalité centrale de Bootstrap, offrant une mise en page sur 12 colonnes.

### Utilisation de base

```html
<div class="container">
    <div class="row">
        <div class="col">Colonne 1</div>
        <div class="col">Colonne 2</div>
        <div class="col">Colonne 3</div>
    </div>
</div>
```

### Points de rupture responsifs

| Classe | Largeur d'écran | Description |
|--------|----------------|-------------|
| `.col-xs` | < 576px | Écran très petit |
| `.col-sm` | ≥ 576px | Petit écran |
| `.col-md` | ≥ 768px | Écran moyen |
| `.col-lg` | ≥ 992px | Grand écran |
| `.col-xl` | ≥ 1200px | Très grand écran |
| `.col-xxl` | ≥ 1400px | Écran extra-large |

### Décalage des colonnes

```html
<div class="row">
    <div class="col-md-4">.col-md-4</div>
    <div class="col-md-4 offset-md-4">.col-md-4 .offset-md-4</div>
</div>
```

### Colonnes responsives

```html
<div class="row">
    <div class="col-12 col-md-6 col-lg-4">
        Plein écran sur mobile, moitié sur tablette, tiers sur grand écran
    </div>
</div>
```

## Navigation

### Navigation de base

```html
<ul class="nav">
    <li class="nav-item">
        <a class="nav-link active" href="#">Accueil</a>
    </li>
    <li class="nav-item">
        <a class="nav-link" href="#">À propos</a>
    </li>
    <li class="nav-item">
        <a class="nav-link" href="#">Contact</a>
    </li>
</ul>
```

### Barre de navigation

```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">Marque</a>
        <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav">
            <span class="navbar-toggler-icon"></span>
        </button>
        <div class="collapse navbar-collapse" id="navbarNav">
            <ul class="navbar-nav">
                <li class="nav-item">
                    <a class="nav-link active" href="#">Accueil</a>
                </li>
                <li class="nav-item">
                    <a class="nav-link" href="#">À propos</a>
                </li>
            </ul>
        </div>
    </div>
</nav>
```

```html
</button>

<div class="modal fade" id="exampleModal" tabindex="-1">
    <div class="modal-dialog">
        <div class="modal-content">
            <div class="modal-header">

                <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
            </div>

            </div>
            <div class="modal-footer">
                <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Fermer</button>

    </div>
```
```html
</div>
```

## Bonnes pratiques

1. **Utilisez du HTML sémantique** : Utilisez correctement les balises header, nav, main, footer, etc.
2. **Gardez simple** : Évitez les imbrications excessives
3. **Priorisez le responsive** : Commencez par la version mobile
4. **Utilisez les classes utilitaires** : Ajustez rapidement les styles
5. **Personnalisez le thème** : Utilisez les variables Sass pour maintenir la cohérence
