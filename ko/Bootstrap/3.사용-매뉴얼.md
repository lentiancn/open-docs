# 사용 매뉴얼

이 가이드에서는 Bootstrap 5의 핵심 기능 — 그리드 시스템, 컴포넌트, CSS 유틸리티 등을 안내합니다.

## 그리드 시스템

그리드 시스템은 12열 레이아웃을 제공하는 Bootstrap의 핵심입니다.

### 기본 사용법

```html
<div class="container">
    <div class="row">
        <div class="col">열 1</div>
        <div class="col">열 2</div>
        <div class="col">열 3</div>
    </div>
</div>
```

### 반응형 중단점

| 클래스 | 화면 너비 | 설명 |
|-------|--------------|-------------|
| `.col-xs` | < 576px | 극소 |
| `.col-sm` | ≥ 576px | 소 |
| `.col-md` | ≥ 768px | 중 |
| `.col-lg` | ≥ 992px | 대 |
| `.col-xl` | ≥ 1200px |特大 |
| `.col-xxl` | ≥ 1400px | 超特大 |

### 열 오프셋

```html
<div class="row">
    <div class="col-md-4">.col-md-4</div>
    <div class="col-md-4 offset-md-4">.col-md-4 .offset-md-4</div>
</div>
```

### 반응형 열

```html
<div class="row">
    <div class="col-12 col-md-6 col-lg-4">
        모바일에서는 전체 너비, 태블릿에서는 절반, 큰 화면에서는 1/3
    </div>
</div>
```

## 모범 사례

1. **시맨틱 HTML 사용** — header, nav, main, footer 등 유사한 태그 활용
2. **단순하게 유지** — 과도한 중첩 피하기
3. **모바일 우선으로 시작** — 먼저 모바일용으로 디자인한 후 확장
4. **유틸리티 클래스 활용** — 스타일 빠르게 조정
5. **Sass 변수로 사용자 정의** — 프로젝트 전체에서 일관성 유지
