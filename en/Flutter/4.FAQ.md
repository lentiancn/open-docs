# Flutter FAQ

Common Flutter development questions and answers.

## Installation Issues

### Q: Flutter download failed?

**Solution:**
1. Use mirror:
```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
export PUB_CACHE=/path/to/cache
```
2. Use proxy
3. Manually download SDK archive

### Q: flutter doctor error?

**Solution:**
1. Check PATH configuration
2. Ensure Git is installed
3. Run `flutter doctor -v` for detailed information

### Q: Android SDK not found?

**Solution:**
```bash
flutter config --android-sdk /path/to/android/sdk

# Or set environment variable
export ANDROID_SDK_ROOT=/path/to/android-sdk
```

### Q: macOS Xcode Command Line Tools not installed?

**Solution:**
```bash
xcode-select --install
```

## Development Issues

### Q: Hot reload not working?

**Solution:**
1. Save file and wait for hot reload
2. Check console for errors
3. Try completely restarting the app

### Q: APK build failed?

**Solution:**
1. Run `flutter clean`
2. Update Gradle
3. Check network connection

### Q: Emulator startup failed?

**Solution:**
1. Enable CPU virtualization
2. Check HAXM installation
3. Use physical device for debugging

### Q: Widget not displaying?

**Solution:**
1. Check build() method return value
2. Confirm added to MaterialApp
3. Check layout constraints

### Q: State not updating?

**Solution:**
1. Ensure setState() is called
2. Check StatefulWidget is correct
3. Confirm using correct Context

### Q: Memory leak?

**Solution:**
1. Properly dispose controllers
2. Clean up Provider properly
3. Avoid creating objects in build

## Performance Issues

### Q: App lagging?

**Solution:**
1. Use const constructors
2. Avoid unnecessary setState
3. Use ListView.builder
4. Optimize image loading

### Q: APK too large?

**Solution:**
1. Use minification
2. Remove debug symbols
3. Use split-per-abi
4. Clean unused resources

### Q: Slow startup?

**Solution:**
1. Reduce init() operations
2. Lazy load components
3. Use Deferred Components

## Platform Issues

### Q: iOS build failed?

**Solution:**

### Q: Web build failed?

**Solution:**

### Q: Platform channel not working?

**Solution:**
