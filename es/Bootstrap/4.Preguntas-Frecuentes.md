# Preguntas Frecuentes de Bootstrap

> Respuestas a preguntas comunes

---

## Problemas de Instalación

### ¿CDN no carga?

**Soluciones**:

1. Verificar conexión de red
2. Usar CDN alternativo:
```html
<link href="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

### ¿Estilos no funcionan después de instalar npm?

Asegúrate de importar correctamente:

```javascript
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap';
```

---

## Problemas de Estilos

### ¿Estilos de componentes no funcionan?

**Causas Comunes**:

1. Bootstrap CSS no importado
2. JavaScript no cargado correctamente (necesario para algunos componentes)
3. Prioridad CSS sobrescrita

### ¿Responsive no funciona?

Asegúrate de agregar la meta etiqueta viewport:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

---

## Problemas de Componentes

### ¿Menú desplegable no funciona?

**Lista de Verificación**:

1. Importar Bootstrap JS (no solo CSS)
2. Usar estructura HTML correcta
3. Verificar errores de JavaScript

### ¿Modal no abre?

Asegúrate de que:
1. Bootstrap JS está cargado
2. Los atributos data-bs-toggle y data-bs-target son correctos
3. El modal existe cuando la página carga

---

## Problemas de Personalización

### ¿Cómo personalizar colores de tema?

**Método 1: Usar Sass**

```scss
$primary: #ff6600;
@import "bootstrap";
```

**Método 2: Usar Variables CSS**

```css
:root {
  --bs-primary: #ff6600;
}
```

---

## Problemas de Compatibilidad

### ¿IE 11 no soportado?

Bootstrap 5 ya no soporta IE 11. Si necesitas soporte IE 11, usa Bootstrap 4.

---

## Problemas de Rendimiento

### ¿Cómo reducir tamaño de archivo?

1. Importar solo los componentes necesarios (Sass)
2. Usar versiones de producción (.min.css/.min.js)
3. Habilitar compresión gzip

---

## Conflictos con Otros Frameworks

### ¿Usar con Vue/React?

Bootstrap puede usarse con cualquier framework de JavaScript, pero ten en cuenta:

1. Algunos componentes dependen de jQuery (Bootstrap 5 ya no depende)
2. Usa librerías wrapper de Bootstrap específicas del framework (ej. react-bootstrap)
