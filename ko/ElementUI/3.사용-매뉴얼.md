# Element UI 사용 매뉴얼

이 매뉴얼은 Element UI를 빠르게 시작하고 일반적으로 사용되는 컴포넌트를 마스터하는 데 도움이 됩니다.

## 빠른 시작

Element UI를 사용하기 전에 설치가 완료되었는지 확인하세요.

### 기본 사용법

Vue 컴포넌트에서 컴포넌트를 직접 사용:

```vue
<template>
  <el-button>기본 버튼</el-button>
  <el-button type="primary">주요 버튼</el-button>
  <el-button type="success">성공 버튼</el-button>
  <el-button type="warning">경고 버튼</el-button>
  <el-button type="danger">위험 버튼</el-button>
</template>
```

## 일반적인 컴포넌트

### 버튼

```vue
<el-button>기본</el-button>
<el-button type="primary">주요</el-button>
<el-button size="small">작은</el-button>
<el-button disabled>비활성화</el-button>
<el-button icon="el-icon-search">검색</el-button>
```

### 입력

```vue
<el-input v-model="input" placeholder="입력하세요" />
<el-input v-model="input" prefix-icon="el-icon-search" />
<el-input v-model="input" clearable />
```

### 선택

```vue
<el-select v-model="value" placeholder="선택">
  <el-option label="옵션 1" value="1" />
  <el-option label="옵션 2" value="2" />
</el-select>
```

### 테이블

```vue
<el-table :data="tableData" stripe border>
  <el-table-column prop="date" label="날짜" width="180" />
  <el-table-column prop="name" label="이름" width="180" />
  <el-table-column prop="address" label="주소" />
</el-table>
```

### 폼

```vue
<el-form :model="form" label-width="80px">
  <el-form-item label="이름">
    <el-input v-model="form.name" />
  </el-form-item>
</el-form>
```

### 대화상자

```vue
<el-dialog :visible.sync="dialogVisible" title="제목" width="30%">
  <span>내용</span>
</el-dialog>
```

### 메시지

```javascript
import { Message } from 'element-ui'

Message.success('성공 메시지')
Message.warning('경고 메시지')
Message.error('오류 메시지')
```

## 레이아웃

### 그리드 시스템

Element UI는 24열 그리드 시스템을 사용합니다:

```vue
<el-row :gutter="20">
  <el-col :span="12">열</el-col>
  <el-col :span="12">열</el-col>
</el-row>
```

### 컨테이너

```vue
<el-container>
  <el-header>헤더</el-header>
  <el-main>메인</el-main>
  <el-footer>푸터</el-footer>
</el-container>
```

## 테마 사용자 정의

SCSS 변수 사용:

```scss
$--color-primary: #409eff;
```

온라인 테마 생성기: https://elementui.github.io/theme-chalk-preview/

## 모범 사례

### 반응형 디자인

```vue
<el-row>
  <el-col :xs="24" :sm="12" :md="8" :lg="6">반응형</el-col>
</el-row>
```
