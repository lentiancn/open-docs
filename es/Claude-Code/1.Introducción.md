# Claude Code Introducción

Claude Code es un asistente de programación inteligente desarrollado por Anthropic que puede leer bases de código, editar archivos, ejecutar comandos e integrarse perfectamente con herramientas de desarrollo. Está disponible en terminal, IDE, aplicación de escritorio y navegador.

## ¿Qué es Claude Code?

Claude Code es un asistente de programación impulsado por IA que te ayuda a construir funciones, corregir errores y automatizar tareas de desarrollo. Entiende toda tu base de código y puede trabajar en múltiples archivos y herramientas para completar diversas tareas de desarrollo.

## Características Principales

Claude Code proporciona las siguientes capacidades principales:

**Automatización de Tareas de Desarrollo**
- Escribir pruebas para código sin pruebas
- Corregir errores de lint en todo el proyecto
- Resolver conflictos de fusión
- Actualizar dependencias
- Escribir notas de lanzamiento

**Desarrollo de Funciones y Corrección de Errores**
- Describe lo que quieres lograr en lenguaje natural
- Claude Code planificará el enfoque, escribirá código en múltiples archivos y verificará la implementación

**Integración con Git**
- Realiza operaciones Git directamente en Claude Code
- Prepara cambios, escribe mensajes de commit, crea ramas, abre Pull Requests

**Integración MCP**
- Model Context Protocol (MCP) es un estándar abierto para conectar herramientas de IA con fuentes de datos externas
- Con MCP, Claude Code puede leer documentos de diseño desde Google Drive, actualizar tickets de Jira, obtener datos de Slack o usar herramientas personalizadas

**Capacidades de Personalización**
- Usa archivos CLAUDE.md para establecer estándares de codificación, decisiones de arquitectura, bibliotecas preferidas y listas de verificación de revisión
- Crea comandos personalizados para encapsular flujos de trabajo repetibles
- Usa Hooks para auto-formatear después de editar archivos o ejecutar lint antes de commits

**Colaboración Multi-Agente**
- Activa múltiples agentes de Claude Code para trabajar en paralelo en tareas complejas
- Un agente líder coordina el trabajo, asigna subtareas y fusiona resultados

**Flexibilidad CLI**
- Soporte para piping, scripting y automatización
- Pipea logs, ejecútalo en CI o combínalo con otras herramientas

## Escenarios de Uso

Claude Code puede usarse en varios escenarios:

| Escenario | Método Recomendado |
|-----------|-------------------|
| Desarrollo local | Terminal CLI |
| Codificación en IDE | VS Code o JetBrains |
| Continuar entre dispositivos | Control Remoto |
| Operaciones móviles | Web o aplicación iOS |
| Automatización CI/CD | GitHub Actions o GitLab CI/CD |
| Colaboración en equipo | Integración Slack |
| Depuración en navegador | Extensión Chrome |

## Requisitos de Cuenta

Claude Code requiere una de las siguientes cuentas:
- Suscripción Claude Pro, Max, Teams o Enterprise
- Cuenta de Anthropic Console
- Proveedores de nube de terceros soportados (Amazon Bedrock, Google Vertex AI, Microsoft Foundry)

El plan gratuito de Claude.ai no incluye acceso a Claude Code.
