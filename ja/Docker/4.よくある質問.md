# Docker よくある質問

## Q: Docker Desktop が起動しない？

BIOSで仮想化が有効になっていることを確認し、WSL 2を正しくインストールしてください。

---

## Q: イメージのダウンロードが遅い？

国内ミラーを設定：

```json
{
  "registry-mirrors": [
    "https://docker.mirrors.ustc.edu.cn"
  ]
}
```

---

## Q: 権限がない？

```bash
sudo usermod -aG docker $USER
# 再ログイン
```

---

## Q: コンテナが起動しない？

```bash
docker logs container_id
docker inspect container_id
```

---

## Q: 未使用のリソースを清理するには？

```bash
docker system prune -a
```
