# Kotlin

Kotlin — это современный язык программирования, разработанный компанией JetBrains, который работает на JVM.

## Введение

Kotlin — это статически типизированный язык программирования, который может выполняться на виртуальной машине Java, а также компилироваться в код JavaScript.

## Основные особенности

- **Краткость**: Сокращение шаблонного кода
- **Безопасность**: Проект с защитой от null
- **Взаимодействие**: 100% совместимость с Java
- **Инструменты**: Отличная поддержка IDE
- **Корутины**: Мощная поддержка асинхронного программирования

## Быстрый старт

### Установка

```bash
# macOS
brew install kotlin

# Linux
sudo apt install kotlin

# SDKMAN
sdk install kotlin
```

### Компиляция и запуск

```bash
kotlinc hello.kt -include-runtime -d hello.jar
java -jar hello.jar
```
