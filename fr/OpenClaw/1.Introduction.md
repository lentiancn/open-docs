# OpenClaw 🦞

OpenClaw est un outil que vous installez sur votre propre appareil et qui connecte vos applications de messagerie préférées aux assistants IA. WhatsApp, Telegram, Discord, iMessage... tout fonctionne directement avec les services IA comme un pont entre vos chats et le cerveau artificiel.

## Pour qui?

Si vous voulez votre propre assistant IA sans confier vos données à des tiers, OpenClaw est un excellent choix. Particulièrement adapté aux développeurs et utilisateurs avancés qui préfèrent les outils en ligne de commande et veulent un contrôle total.

## Caractéristiques principales

- **Vos données, votre machine**: Tout reste sur votre propre matériel, rien ne va vers des serveurs externes
- **Multi-plateformes**: Un seul processus gère plusieurs applications de chat simultanément
- **Pour la programmation**: Supporte les outils, l'historique, la mémoire et les multi-assistants
- **Code ouvert**: Licence MIT, communauté active

## Ce qu'il faut

- Node.js version 22 ou supérieure
- Une clé API de votre fournisseur IA préféré
- Quelques minutes pour la configuration initiale

Pour la meilleure expérience, utilisez les modèles de dernière génération les plus puissants.

## Démarrage rapide

```bash
# Installer OpenClaw
npm install -g openclaw@latest

# Exécuter l'assistant de configuration
openclaw onboard --install-daemon

# Démarrer le service
openclaw gateway --port 18789
```

Une fois en cours d'exécution, ouvrez votre navigateur sur http://127.0.0.1:18789/ pour commencer.

| Plateforme | Statut | Notes |
|------------|--------|-------|
| WhatsApp | ✅ | Nécessite le couplage QR |
