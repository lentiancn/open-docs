# OpenClaw

> 用於進階自動化的 AI 助理框架

## 簡介

OpenClaw 是一款多功能 AI 助理框架，透過先進的代理程式技術自動執行任務。

## 快速入門

### 安裝

```bash
npm install -g openclaw
```

### 啟動 Gateway

```bash
openclaw gateway start
```

### 開始使用

```bash
openclaw chat
```

## 功能

- 智慧代理程式
- 瀏覽器自動化
- 檔案管理
- 訊息傳送
- 工具擴展性
