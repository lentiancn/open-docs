Angular is a TypeScript-based framework developed and maintained by Google. Provides a complete solution for building modern web applications.

## Quick Start

```bash
# Install Angular CLI
npm install -g @angular/cli

# Create new project
ng new my-app

# Start dev server
cd my-app
ng serve
```
