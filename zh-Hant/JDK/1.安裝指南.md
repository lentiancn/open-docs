# JDK 安裝指南

JDK（Java Development Kit）是官方的 Java 開發工具包。本指南涵蓋各種環境的安裝方法。

## 版本資訊

| 版本 | 狀態 | 說明 |
|---------|--------|-------------|
| JDK 21 (LTS) | ✅ 目前版本 | 最新的 LTS 版本 |
| JDK 17 (LTS) | ✅ 受支援 | 上一版 LTS |
| JDK 11 (LTS) | ✅ 受支援 | 傳統 LTS |
| JDK 8 | ✅ 受支援 | 傳統版本 |
| JDK 22 | ✅ 最新版本 | 最新的功能版本 |

### LTS 與功能版本

- **LTS（長期支援）**：2-3 年更新，推薦用於生產環境
- **功能版本**：每 6 個月發布新功能，支援期較短

## 系統需求

### 硬體需求

| 需求 | 最低配置 | 推薦配置 |
|-------------|---------|-------------|

| 軟體 | 版本 |
|----------|---------|

## 安裝方法

### 方法 1：Oracle JDK

#### Windows

1. 從以下地址下載：https://www.oracle.com/java/technologies/downloads/
2. 執行安裝程式（.exe）
3. 依照精靈操作

#### macOS

```bash
# 從 Oracle 網站下載 DMG
# 或使用 Homebrew
brew install openjdk@21
```

#### Linux

```bash
# 下載 tar.gz
wget https://download.oracle.com/java/21/latest/jdk-21_linux-x64_bin.tar.gz
tar -xzf jdk-21_linux-x64_bin.tar.gz
sudo mv jdk-21 /opt/jdk-21
```

### 方法 2：Eclipse Temurin（Adoptium）

#### Windows

1. 從以下地址下載：https://adoptium.net/temurin/releases/
2. 執行安裝程式

#### macOS

```bash
# 使用 Homebrew
brew install --cask temurin

# 或從網站下載 pkg
```

#### Linux

```bash
# Ubuntu/Debian
wget https://github.com/adoptium/temurin21-binaries/releases/download/jdk-21.0.2%2B13/OpenJDK21U-jdk_x64_linux_hotspot_21.0.2_13.tar.gz
tar -xzf OpenJDK21U-jdk_x64_linux_hotspot_21.0.2_13.tar.gz

# 設定環境變數
export JAVA_HOME=/path/to/jdk-21
export PATH=$JAVA_HOME/bin:$PATH
```

### 方法 3：SDKMAN

```bash
# 安裝 SDKMAN
curl -s "https://get.sdkman.io" | bash

# 安裝 JDK
sdk install java 21.0.2-tem

# 列出可用版本
sdk list java
```

### 方法 4：Docker

```dockerfile
FROM eclipse-temurin:21-jdk
WORKDIR /app
COPY . .
RUN javac Main.java
CMD ["java", "Main"]
```

### 方法 5：Windows 套件管理員

#### Chocolatey

```powershell
choco install openjdk21
```

#### Scoop

```powershell
scoop install openjdk
```

## 環境配置

### Windows

```powershell
# 設定 JAVA_HOME
[System.Environment]::SetEnvironmentVariable("JAVA_HOME", "C:\Program Files\Java\jdk-21", "User")

# 新增到 PATH
[System.Environment]::SetEnvironmentVariable("PATH", "$env:PATH;C:\Program Files\Java\jdk-21\bin", "User")
```

### macOS/Linux

```bash
# 新增到 ~/.bashrc 或 ~/.zshrc
export JAVA_HOME=/opt/jdk-21
export PATH=$JAVA_HOME/bin:$PATH

# 生效
source ~/.bashrc
```

## 驗證

```bash
# 檢查 Java 版本
java -version

# 檢查 Javac 版本
javac -version

# 檢查 JAVA_HOME
echo $JAVA_HOME
```

## 多個 JDK 版本

### 使用 SDKMAN

```bash
# 安裝多個版本
sdk install java 17.0.8-tem
sdk install java 21.0.2-tem

# 切換版本
sdk use java 17.0.8-tem
```

## 常見問題

### 找不到 "java"？

請驗證 PATH 是否包含 Java 的 bin 目錄。

### Java 版本錯誤？

檢查正在使用的 Java：

```bash
which java
java -version
```

### JAVA_HOME 未設定？

將 JAVA_HOME 設定為 JDK 安裝目錄。
