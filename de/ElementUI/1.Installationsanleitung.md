# Element UI Installationsanleitung

> Element UI ist eine Desktop-UI-Komponentenbibliothek basierend auf Vue 2
> 
> Aktualisiert: Februar 2026
> 
> Aktuelle Version: 2.x (Stabil)
> 
> Hinweis: Für Vue 3-Projekte verwenden Sie bitte Element Plus

---

## 1. Projektübersicht

Element UI ist eine Desktop-UI-Komponentenbibliothek basierend auf Vue 2.0, entwickelt und gepflegt vom Ele.me-Team. Es bietet einen reichhaltigen Satz häufig verwendeter PC-Komponenten, einschließlich Formulare, Tabellen, Dialoge, Navigationsmenüs, Buttons, Eingabefelder und andere UI-Elemente.

Element UI hat einen sauberen und eleganten Designstil mit detaillierter Dokumentation. Es ist eine der beliebtesten Vue 2 UI-Komponentenbibliotheken in China. Obwohl die offizielle Wartung eingestellt wurde (Empfehlung zur Migration zu Element Plus für Vue 3), hat Element UI aufgrund der großen Anzahl bestehender Projekte immer noch einen hohen Referenz- und Lernwert.

### 1.1 Versionsinformationen

| Version | Vue Version | Status | Beschreibung |
|---------|-------------|--------|--------------|
| Element UI 2.x | Vue 2 | Stabil | Hauptversions-Dokumentation |
| Element UI 1.x | Vue 2 | Eingestellt | Frühversion |

### 1.2 Unterschied zu Element Plus

| Funktion | Element UI | Element Plus |
|----------|------------|--------------|
| Vue Version | Vue 2 | Vue 3 |
| TypeScript | Teilweise Unterstützung | Volle Unterstützung |
| Composition API | Nicht unterstützt | Unterstützt |
| Wartungsstatus | Eingestellt | Aktiv gepflegt |

## 3. Installationsschritte

### 3.1 Mit npm (Empfohlen)

```bash
# In Ihr Vue 2-Projektverzeichnis wechseln
cd ihr-vue-projekt

# Element UI installieren
npm install element-ui --save
```

### 3.2 Mit yarn

```bash
yarn add element-ui
```

---

## 4. Import in Vue 2-Projekt

### 4.1 Vollständiger Import

Importieren Sie Element UI und seine Stile in der Hauptdatei (normalerweise main.js):

```javascript
// main.js
import Vue from 'vue'
import App from './App.vue'

import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)

new Vue({
  render: h => h(App)
}).$mount('#app')
```

### 4.2 Bedarfsweiser Import

Um die Bundle-Größe zu reduzieren, importieren Sie nur benötigte Komponenten:

```javascript
import { Button, Form, FormItem, Input } from 'element-ui'

Vue.use(Button)
Vue.use(Form)
Vue.use(FormItem)
Vue.use(Input)
```

## 5. Schnellstart Beispiele

### 5.1 Basis-Button

```html
<el-button>Standard</el-button>
<el-button type="primary">Primär</el-button>
<el-button type="success">Erfolg</el-button>
<el-button type="warning">Warnung</el-button>
<el-button type="danger">Gefahr</el-button>
```

### 5.2 Basis-Formular

```html
<el-form :model="form" label-width="80px">
  <el-form-item label="Benutzername">
    <el-input v-model="form.username"></el-input>
  </el-form-item>
  <el-form-item label="Passwort">
    <el-input v-model="form.password" type="password"></el-input>
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="submit">Absenden</el-button>
  </el-form-item>
</el-form>
```

### 5.3 Basis-Tabelle

```html
<el-table :data="tableData" stripe border>
  <el-table-column prop="name" label="Name"></el-table-column>
  <el-table-column prop="email" label="E-Mail"></el-table-column>
</el-table>
```

### 5.4 Dialog

```html
<el-dialog title="Titel" :visible.sync="visible">
  <span>Inhalt</span>
</el-dialog>
```
