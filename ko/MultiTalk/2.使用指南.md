# MultiTalk 사용 가이드

이 가이드에서는 MultiTalk를 사용하여 다중 화자 오디오 콘텐츠를 생성하는 방법을 설명합니다.

## 빠른 시작

### 기본 사용법

```bash
python inference.py \
  --text "안녕하세요, MultiTalk에 오신 것을 환영합니다." \
  --speakers speaker1,speaker2 \
  --output result.wav
```

## 입력 요구사항

### 텍스트 요구사항

- 언어: 다국어 지원
- 형식: 일반 텍스트
- 길이: 엄격한 제한 없음

### 화자 구성

`speakers.yaml`에서 화자 정의:

```yaml
speakers:
  speaker1:
    voice_id: "voice_001"
    language: "ko"
  speaker2:
    voice_id: "voice_002"
    language: "ko"
```

## 고급 사용법

### 사용자 지정 음성 ID

```bash
python inference.py \
  --text "사용자 지정 음성 데모입니다." \
  --voice_ids voice001,voice002 \
  --output output.wav
```

### 속도 조정

```bash
python inference.py \
  --text "느린 음성 예제입니다." \
  --speakers speaker1 \
  --speed 0.8 \
  --output slow.wav
```

### 배치 처리

```bash
python batch_inference.py \
  --text_file texts.txt \
  --output_dir output/
```

## 구성

### 구성 파일

`config.yaml` 편집:

```yaml
inference:
  sample_rate: 22050
  speed: 1.0
  
model:
  checkpoint: "checkpoints/multitalk.pth"
```

## 모범 사례

### 1. 적절한 음성 ID 사용

화자 특성과 일치하는 음성 ID를 선택하세요.

### 2. 언어 일치

텍스트 언어가 음성 언어와 일치하는지 확인하세요.

### 3. 속도 최적화

자연스러운 출력을 위해 속도를 조정하세요.

## 문제 해결

### 오디오 품질이 나쁩니다?

다른 음성 ID를 사용해 보세요.

### 속도 문제?

속도 매개변수를 점진적으로 조정하세요.
