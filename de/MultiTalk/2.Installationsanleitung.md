# Installationsanleitung

Der Installationsprozess von MultiTalk erfordert die Konfiguration der Python-Umgebung und das Herunterladen von Modellgewichten. Im Folgenden finden Sie detaillierte Installationsanweisungen.

## Umgebungsanforderungen

### Hardware-Anforderungen

MultiTalk ist ein grosses Videogenerierungsmodell mit hohen Hardware-Anforderungen. Die Ausfuehrung mit NVIDIA-GPU wird empfohlen. Die VRAM-Anforderungen variieren je nach Ausgabeauflösung. Die Generierung von 480P-Videos erfordert mindestens 16GB VRAM, wahrend die Generierung von 720P-Videos mehrere GPU-Konfigurationen erfordert. Das Projekt unterstützt auch die Ausfuehrung in Umgebungen mit geringem VRAM und ermoglicht 480P-Videogenerierung auf einer einzelnen RTX 4090 durch spezifische Optimierungen.

### Software-Anforderungen

Linux-Betriebssystem wird unterstützt. Python-Version 3.10 oder hoher ist erforderlich. Auch CUDA 12.1 oder hoher ist erforderlich. Das Projekt hangt von Deep-Learning-Frameworks wie PyTorch 2.4.1 und xformers ab.

## Erstellen der virtuellen Umgebung

Erstellen Sie zunaechst eine neue Python virtuelle Umgebung, um Projekt-Abhangigkeiten zu isolieren. Oeffnen Sie das Terminal und fuehren Sie die folgenden Befehle aus:

```bash
conda create -n multitalk python=3.10
conda activate multitalk
```

## Installieren von PyTorch und xformers

Installieren Sie PyTorch und zugehoerige Pakete in der aktivierten Umgebung. Fuehren Sie den folgenden Befehl aus, um PyTorch 2.4.1 zu installieren:

```bash
pip install torch==2.4.1 torchvision==0.19.1 torchaudio==2.4.1 --index-url https://download.pytorch.org/whl/cu121
```

Installieren Sie anschliessend xformers:

```bash
pip install -U xformers==0.0.28 --index-url https://download.pytorch.org/whl/cu121
```

## Installieren von Projektabhaengigkeiten

Installieren Sie andere erforderliche Abhangigkeitspakete:

```bash
pip install misaki[en]
pip install ninja
pip install psutil
pip install packaging
pip install flash_attn==2.7.4.post1
```

Klonen Sie dann das Projekt-Repository und installieren Sie die Projektabhaengigkeiten:

```bash
git clone https://github.com/MeiGen-AI/MultiTalk.git
cd MultiTalk
pip install -r requirements.txt
```

## Installieren von Audio-Verarbeitungsbibliotheken

Das Projekt erfordert Audio-Verarbeitungsbibliotheken. Fuehren Sie die folgenden Befehle aus, um sie zu installieren:

```bash
conda install -c conda-forge librosa
conda install -c conda-forge ffmpeg
```

Fuer CentOS- oder RHEL-Systeme verwenden Sie:

```bash
sudo yum install ffmpeg ffmpeg-devel
```

## Herunterladen von Modellgewichten

MultiTalk erfordert das Herunterladen mehrerer Modelldateien, um ordnungsgemass zu funktionieren. Die hauptsachlichen Modelle umfassen:

Wan2.1-I2V-14B-480P ist das Basis-Diffusionsmodell, erhaeltlich bei Hugging Face. chinese-wav2vec2-base ist der Audio-Encoder zur Verarbeitung von Eingabeaudio. Kokoro-82M sind die TTS-Gewichte für Text-zu-Sprache-Funktionalitaet. MeiGen-MultiTalk sind die vom Projekt bereitgestellten Audio-Bedingungsgewichte.

Laden Sie Modelle mit huggingface-cli herunter:

```bash
huggingface-cli download Wan-AI/Wan2.1-I2V-14B-480P --local-dir ./weights/Wan2.1-I2V-14B-480P
huggingface-cli download TencentGameMate/chinese-wav2vec2-base --local-dir ./weights/chinese-wav2vec2-base
huggingface-cli download hexgrad/Kokoro-82M --local-dir ./weights/Kokoro-82M
huggingface-cli download MeiGen-AI/MeiGen-MultiTalk --local-dir ./weights/MeiGen-MultiTalk
```

## Konfigurieren von Modellenlinks

Nach dem Herunterladen der Modelle muessen Sie die Links konfigurieren. Es gibt zwei Optionen.

Option eins ist die Verwendung symbolischer Links. Wechseln Sie in das weights-Verzeichnis und fuehren Sie aus:

```bash
cd weights
mv Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json_old
ln -s /path/to/weights/MeiGen-MultiTalk/diffusion_pytorch_model.safetensors.index.json Wan2.1-I2V-14B-480P/
ln -s /path/to/weights/MeiGen-MultiTalk/multitalk.safetensors Wan2.1-I2V-14B-480P/
```

Option zwei ist das direkte Kopieren von Dateien:

```bash
cd weights
mv Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json_old
cp MeiGen-MultiTalk/diffusion_pytorch_model.safetensors_index.json Wan2.1-I2V-14B-480P/
cp MeiGen-MultiTalk/multitalk.safetensors Wan2.1-I2V-14B-480P/
```
