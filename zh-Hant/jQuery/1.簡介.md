# jQuery 簡介

## 什麼是 jQuery？

jQuery 是一個快速、小巧、功能豐富的 JavaScript 庫。它讓 HTML 文件遍歷和操作、事件處理、動畫以及 Ajax 等操作變得更加簡單，透過一個易於使用的 API 在多種瀏覽器中都能正常工作。憑藉其多功能性和可擴展性，jQuery 改變了數百萬人編寫 JavaScript 的方式。

## jQuery 的核心特點

### 簡潔的語法
jQuery 採用了鏈式呼叫的設計模式，使得程式碼簡潔易讀。例如：
```javascript
$("#myElement").addClass("highlight").fadeIn();
```

### 強大的選擇器
jQuery 實現了 CSS 選擇器的全部功能，並在此基礎上擴展了自訂選擇器，讓你能夠快速定位頁面元素。

### 跨瀏覽器相容
jQuery 解決了不同瀏覽器之間的 API 差異，開發者無需關心瀏覽器相容性問題。

### 豐富的插件生態
jQuery 擁有龐大的插件庫，涵蓋了從 UI 元件到表單驗證的各個方面。

## jQuery 能做什麼？

### DOM 操作
取得和修改 HTML 元素的內容、屬性、樣式變得輕而易舉：
```javascript
$("button.continue").html("下一步...");
$("#header").css("background-color", "blue");
$(".item").addClass("selected");
```

### 事件處理
為頁面元素綁定事件處理器，處理使用者互動：
```javascript
$("#button-container button").on("click", function(event) {
    $("#banner-message").show();
});
```

### 動畫效果
建立流暢的動畫效果：
```javascript
$("#element").animate({
    opacity: 0.5,
    height: "200px"
}, 300);
```

### Ajax 異步通訊
與伺服器進行資料交換，無需刷新頁面：
```javascript
$.ajax({
    url: "/api/getData",
    data: { id: 123 },
    success: function(result) {
        $("#content").html(result);
    }
});
```

## 相關專案

### jQuery UI
jQuery UI 是一個建立在 jQuery 之上的使用者介面互動庫，提供了豐富的 UI 元件和互動效果。該專案目前處於維護模式。

### jQuery Mobile
jQuery Mobile 是一個用於建立行動端響應式網站的框架，已被標記為廢棄。

## 許可證

jQuery 採用 MIT 許可證開源，可以免費用於商業和個人專案。

## 適用場景

jQuery 特別適合以下場景：
- 快速開發簡單的網頁互動功能
- 需要支援較舊瀏覽器的專案
- 快速原型開發
- 學習和理解 JavaScript DOM 操作

## 版本歷史

目前穩定版本為 4.0.0。jQuery 從 2006 年發布至今，已經成為最流行的 JavaScript 庫之一，被廣泛應用於各類網站和 Web 應用中。
