# Element UI Installation Guide

Complete Element UI installation and configuration guide.

## Environment Requirements

### Software Requirements

- Node.js: 12.0.0+
- npm: 6.0.0+ or yarn: 1.22.0+
- Vue: 2.6.0+

## Installation Steps

### Method 1: npm install (Recommended)

```bash
# Using npm
npm install element-ui --save

# Or using yarn
yarn add element-ui
```

### Method 2: CDN Import

```html
<!-- Import styles -->
<link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">

<!-- Import Vue -->
<script src="https://unpkg.com/vue"></script>

<!-- Import Element UI -->
<script src="https://unpkg.com/element-ui/lib/index.js"></script>
```

## Quick Start

### Full Import

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(ElementUI)

new Vue({
  el: '#app',
  render: h => h(App)
})
```

### On-Demand Import (Recommended)

First install babel-plugin-component:

```bash
npm install babel-plugin-component -D
```

Configure babel.config.js:

```javascript
module.exports = {
  presets: [
    '@vue/cli-plugin-babel/preset'
  ],
  plugins: [
    [
      'component',
      {
        libraryName: 'element-ui',
        styleLibraryName: 'theme-chalk'
      }
    ]
  ]
}
```

Import components on-demand:

```javascript
import { Button, Select, Option, Table, TableColumn } from 'element-ui'

Vue.use(Button)
Vue.use(Select)
Vue.use(Option)
Vue.use(Table)
Vue.use(TableColumn)
```

## Theme Customization

### Using SCSS Variables

Create variables file:

```scss
// styles/element-variables.scss

$--color-primary: #409EFF;
$--color-success: #67C23A;
$--color-warning: #E6A23C;
$--color-danger: #F56C6C;

@import 'element-ui/packages/theme-chalk/src/common/var.scss';
```

Import in main.js:

```javascript
import './styles/element-variables.scss'
```

### Online Theme Editor

Visit: https://element.eleme.io/#/en/theme

1. Select or customize theme colors
2. Click "Download" button
3. Replace Element UI theme files

## Internationalization

### Configure Multi-language

Install vue-i18n:

```bash
npm install vue-i18n -S
```

Configure:

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import VueI18n from 'vue-i18n'
import enLocale from 'element-ui/lib/locale/lang/en'

Vue.use(ElementUI, {
  i18n: (key, value) => i18n.t(key, value)
})

const messages = {
  'en': { message: 'hello' }
}

const i18n = new VueI18n({
  locale: 'en',
  messages
})

new Vue({ i18n }).$mount('#app')
```

## Common Issues

### Components not displaying?

1. Confirm components are properly imported
2. Check if Vue.use() is called
3. Check console for errors

### Styles not applied?

1. Confirm style file is imported
2. Check for global style overrides
3. Check z-index levels

### Icons not displaying?

Install element-ui-icons:

```bash
npm install element-ui-icons
```

Import icons:

```javascript
import 'element-ui-icons'
```

Usage:

```html
<i class="el-icon-search"></i>
```
