# Bootstrap FAQ

Часто задаваемые вопросы.

## Установка

### Q: CDN не загружается?

**Решения**:
1. Проверить подключение
2. Проверить URL
3. Использовать альтернативный CDN

### Q: npm установка не удалась?

**Решения**:
1. Очистить кэш npm
2. Проверить версию Node.js

### Q: Нужен jQuery?

Нет. Bootstrap 5 не использует jQuery.

## Использование

### Q: Стили не работают?

**Решения**:
1. CSS файл подключён
2. Очистить кэш браузера

### Q: Адаптивность не работает?

Добавить viewport meta:
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

## Совместимость

### Q: IE не поддерживается?

Bootstrap 5 не поддерживает IE.

## Производительность

### Q: Файлы слишком большие?

**Решения**:
1. Использовать Sass
2. Использовать .min версии
