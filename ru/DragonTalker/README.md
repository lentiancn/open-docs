# DragonTalker

DragonTalker — это проект искусственного интеллекта для создания реалистичных видео с говорящей головой из одного изображения и аудио.

## Обзор

DragonTalker создаёт высококачественные видео с говорящей головой путём:
- Извлечения лицевых ориентиров из аудио
- Генерации коэффициентов трёхмерного движения лица
- Рендеринга реалистичной синхронизации губ и выражений лица
- Улучшения качества видео с помощью восстановления лица

## Быстрый старт

### Установка

```bash
git clone https://github.com/your-repo/DragonTalker.git
cd DragonTalker

# Создание виртуальной среды
python -m venv venv
source venv/bin/activate  # Linux/macOS
# venv\Scripts\activate  # Windows

# Установка зависимостей
pip install -r requirements.txt

# Загрузка моделей
bash scripts/download_models.sh
```

### Создание видео

```bash
python inference.py \
  --source_image examples/source_image.jpg \
  --driven_audio examples/driven_audio.wav \
  --result_dir ./results \
  --preprocess full \
  --enhancer gfpgan
```

## Функции

- **Аудио-управляемая анимация** - Создание говорящей головы из аудио
- **Трёхмерная оценка движения** - Реалистичное движение лица
- **Множественные стили позы** - Более 45 вариантов поз
- **Улучшение лица** - Интеграция GFP-GAN, RestoreFormer, CodeFormer
- **Веб-интерфейс** - Простая в использовании демонстрация
- **Python API** - Программный доступ

## Т- macOS ребования к входным данным

### Изображение
- Формат: JPG, PNG
- Разрешение: Рекомендуется 512x512 или выше
- Лицо: Фронтальное, чёткое

### Аудио
- Формат: WAV, MP3
- Длительность: 1-60 секунд
- Качество: Чёткая речь

## Выходные данные

- Формат: MP4 (H.264)
- Разрешение: 256x256 или 512x512
- Частота кадров: 25
