# Flutter 설치 가이드

## 요구 사항

Flutter를 설치하기 전에 개발 환경이 다음 요구 사항을 충족하는지 확인하세요. 운영 체제의 경우, Flutter는 Windows, macOS 및 Linux를 지원합니다. Windows는 Windows 10 이상이, macOS는 macOS 10.14 이상이, Linux는 GNOME 또는 KDE 데스크톱 환경을 지원하는 것이 필요합니다.

디스크 공간의 경우, Flutter SDK 자체는 약 1GB를 차지하지만 개발 프로젝트와 빌드 캐시를 고려하면 최소 10GB의 여유 공간을 확보하는 것이 좋습니다. 메모리의 경우, 8GB 이상이면 더 부드러운 개발 경험을 얻을 수 있습니다.

모바일 개발의 경우, iOS 개발은 macOS와 Xcode가, Android 개발은 Android Studio와 Android SDK 설치가 필요합니다. 웹 및 데스크톱 개발은 특별한 요구 사항이 없으며 Flutter SDK만 설치하면 됩니다.

## 설치 단계

### Windows 설치

Windows에서 Flutter를 설치하는 두 가지 주요 방법이 있습니다. 첫 번째 방법은 Flutter 웹사이트에서 사전 컴파일된 아카이브를 다운로드하여 적절한 위치에 추출한 다음 flutter/bin 디렉터리를 시스템 PATH 환경 변수에 추가하는 것입니다.

다운로드 방법: Flutter 웹사이트 flutter.dev를 방문하여 최신 stable 버전을 다운로드합니다. C:\src\flutter 또는 다른 적절한 위치에 추출합니다. 시스템 속성 → 고급 → 환경 변수를 열고, 사용자 변수의 Path에 Flutter bin 디렉터리 경로를 추가합니다.

완료 후, 명령 프롬프트를 열고 flutter doctor 명령을 실행하여 설치가 성공했는지 확인합니다. Flutter doctor는 Flutter SDK 상태를 검사하고 구성 제안을 제공합니다.

### macOS 설치

macOS에서는 Git 기반 설치를 권장합니다. 먼저 Xcode가 설치되어 있는지 확인한 다음 터미널을 열고 Flutter를克隆하려면 다음 명령을 실행합니다:

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

PATH에 Flutter 추가:

```bash
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.bash_profile
```

zsh 사용자의 경우:

```bash
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.zshrc
```

그런 다음 source ~/.bash_profile 또는 source ~/.zshrc를 실행하여 환경 변수를 새로 고칩니다.

### Linux 설치

Linux 설치는 macOS와 유사합니다. 먼저 종속성을 설치합니다:

```bash
sudo apt-get update
sudo apt-get install curl git unzip xz-utils zip libglu1-mesa clang cmake ninja-build pkg-config libgtk-3-dev
```

그런 다음 Flutter 저장소를克隆하고 PATH를 구성합니다(macOS 단계와 동일).

## 설치 확인

설치 후 다음 명령을 실행하여 확인합니다:

```bash
flutter --version
flutter doctor
```

Flutter doctor는 설치 상태와 부족한 종속성을 표시합니다. 지침에 따라 Android Studio 설치 및 Android SDK 구성과 같은 필요한 구성을 완료합니다.

## IDE 구성

### VS Code

VS Code는 Flutter 권장 편집기 중 하나입니다. VS Code를 설치한 후 확장 마켓플레이스를 열고 Flutter 및 Dart 확장을 검색하여 설치합니다. 설치 후 VS Code를 다시 시작하면 Flutter 관련 기능을 사용할 수 있습니다.

### Android Studio

Android Studio를 설치한 후 Plugins 페이지를 열고 Flutter 플러그인을 검색하여 설치합니다. 설치 후 Flutter SDK 경로를 구성합니다: Preferences → Language & Frameworks → Flutter를 열고 Flutter SDK 경로를 설정합니다.

## 빠른 시작

첫 번째 Flutter 프로젝트 생성:

```bash
flutter create my_first_app
cd my_first_app
flutter run
```

또는 VS Code에서 명령 팔레트(Ctrl+Shift+P)를 열고 Flutter: New Project를 입력하여 새 프로젝트를 만듭니다.

flutter devices를 실행하면 연결된 장치가 표시되고, flutter run -d device_id로 특정 장치에서 실행할 수 있습니다.
