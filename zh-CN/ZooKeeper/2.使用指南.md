# ZooKeeper 使用指南

ZooKeeper 基本操作。

---

## 基本命令

```bash
# 连接
zkCli.sh -server localhost:2181

# 列出节点
ls /

# 创建节点
create /test "hello"

# 获取数据
get /test

# 更新数据
set /test "world"

# 删除节点
delete /test
```

## 节点类型

- **持久节点**: 永久存在
- **临时节点**: 客户端断开时删除
- **顺序节点**: 自动递增
