# MySQL 설치 가이드

모든 플랫폼에 MySQL을 설치하는 방법입니다.

---

## Docker (권장)

```bash
docker run -d \
  --name mysql \
  -e MYSQL_ROOT_PASSWORD=secret \
  -e MYSQL_DATABASE=mydb \
  -p 3306:3306 \
  mysql:8
```

---

## Linux

```bash
sudo apt update
sudo apt install -y mysql-server
sudo systemctl start mysql
```

---

## 확인

```bash
mysql --version
```
