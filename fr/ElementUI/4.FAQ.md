# Element UI FAQ

Questions et réponses courantes.

## Problèmes d'installation

### Q: L'installation a échoué?

**Solutions:**
1. Vérifier la version de Node.js
```bash
node --version  # Requiert 12.0.0+
```
2. Vider le cache npm
```bash
npm cache clean --force
```
3. Utiliser le miroir Taobao
```bash
npm config set registry https://registry.npmmirror.com
```
4. Supprimer node_modules et réinstaller
```bash
rm -rf node_modules
npm install
```

### Q: Module non trouvé?

**Solutions:**
1. Confirmer qu'element-ui est installé
```bash
npm list element-ui
```
2. Vérifier package.json pour les dépendances

### Q: Version de Vue incompatible?

**Solutions:**
Element UI 2.x requiert Vue 2.6.0+:
```bash
npm install vue@2
```

## Problèmes d'utilisation

### Q: Les composants ne s'affichent pas?

**Solutions:**
1. Confirmer que les composants sont correctement importés
```javascript
import { Button } from 'element-ui'
Vue.use(Button)
```
2. Vérifier la console pour les erreurs
3. Confirmer que les noms de composants sont corrects

### Q: Les styles ne s'appliquent pas?

**Solutions:**
1. Confirmer que le fichier de styles est importé
```javascript
import 'element-ui/lib/theme-chalk/index.css'
```
2. Vérifier les remplacements de styles globaux
3. Utiliser les outils de développement pour inspecter

### Q: Le tableau n'affiche pas les données?

**Solutions:**
1. Vérifier le format des données
2. Confirmer les définitions de colonnes
3. Vérifier si les données sont réactives

### Q: La validation de formulaire ne fonctionne pas?

**Solutions:**
1. Importer Form et FormItem
2. Définir l'attribut prop
3. Définir les règles dans data

### Q: Les options de la liste déroulante ne s'affichent pas?

**Solutions:**
1. Vérifier les données d'options
2. Confirmer le mappage de value et label

### Q: Les données du dialogue ne se réinitialisent pas après la fermeture?

**Solutions:**
1. Utiliser v-if pour contrôler le dialogue
2. Réinitialiser manuellement le formulaire à la fermeture

## Problèmes de style

### Q: Conflits de styles?

**Solutions:**
1. Utiliser des sélecteurs plus spécifiques
2. Vérifier les niveaux z-index
3. Utiliser !important pour écraser (non recommandé)

### Q: Les icônes ne s'affichent pas?

**Solutions:**
1. Confirmer qu'element-ui-icons est installé
```bash
npm install element-ui-icons
```
2. Importer correctement

### Q: Couleur du thème incorrecte?

**Solutions:**
1. Vérifier les remplacements de styles globaux
2. Confirmer l'ordre d'importation des variables SCSS
3. Vider le cache du navigateur

## Problèmes d'internationalisation

### Q: Le changement de langue ne fonctionne pas?

**Solutions:**
1. Configurer i18n correctement
2. Confirmer que le pack de langue est importé

## Problèmes de performance

### Q: La page charge lentement?

**Solutions:**
1. Utiliser l'importation à la demande
2. Utiliser l'importation CDN
3. Activer la compression Gzip

### Q: Tableau avec grandes données lent?

**Solutions:**
1. Utiliser le défilement virtuel
2. Charger les données par pages
3. Désactiver les fonctionnalités inutiles

## Problèmes de build

### Q: Les styles perdus après le build?

**Solutions:**
1. Vérifier la configuration de vue.config.js
2. Confirmer la position d'importation des styles

### Q: Les composants ne s'affichent pas en production?

**Solutions:**
1. Vérifier que Vue.use() est appelé
2. Confirmer l'importation à la demande complète

## Problèmes de mise à niveau

### Q: Comment mettre à jour de 1.x vers 2.x?

**Solutions:**
1. Lire le guide de migration officiel
2. Remplacer les API incompatibles progressivement
3. Exécuter les tests

### Q: Que utiliser pour un projet Vue 3?

**Solutions:**
Element UI est pour Vue 2. Pour les projets Vue 3, utiliser Element Plus:
```bash
npm install element-plus
```
