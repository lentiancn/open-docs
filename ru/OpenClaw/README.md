# OpenClaw

[English](./README.md) | [简体中文](./README.md) | [日本語](./README.md)

🦞 *Any OS gateway for AI agents across WhatsApp, Telegram, Discord, iMessage, and more.*

## Возможности

- 🤖 **Шлюз ИИ-помощника** — Подключите несколько приложений чата к ИИ-помощникам для программирования
- 📱 **Поддержка множества платформ** — WhatsApp, Telegram, Discord, iMessage, Signal, Slack и другие
- 🔒 **Самостоятельный хостинг** — Все данные хранятся на вашем устройстве
- 🧠 **Поддержка нескольких агентов** — Изолированные рабочие области и сессии
- 🌐 **Веб-интерфейс управления** — Управление и чат через браузер
- 📷 **Мобильные узлы** — Поддержка подключения устройств iOS и Android

## Быстрый старт

```bash
# Установить
npm install -g openclaw@latest

# Настроить
openclaw onboard --install-daemon

# Запустить
openclaw gateway --port 18789
```

Откройте браузер по адресу http://127.0.0.1:18789/, чтобы начать.

## Поддерживаемые платформы

| Платформа | Статус | Примечания |
|-----------|--------|------------|
| WhatsApp | ✅ | Использует Baileys, требуется QR-подключение |
| Telegram | ✅ | Bot API, поддерживает группы |
| Discord | ✅ | Bot API + Gateway |
| iMessage | ✅ | Требуется Mac или BlueBubbles |
| Signal | ✅ | Использует signal-cli |
| Slack | ✅ | Bolt SDK |
| Другие | ➕ | Через плагины |

## Требования

- Node.js 22+
- Любая операционная система с поддержкой Node.js
