# MultiTalk Resumen

Una potente aplicación de chat de voz multiparte.

## ¿Qué es MultiTalk?

MultiTalk es una aplicación de escritorio para chat de voz multiparte.

### Características Principales

- **Voz multiparte** - Soporte para múltiples participantes
- **Baja latencia** - Comunicación en tiempo real
- **Fuentes de audio** - Múltiples dispositivos
- **Push-to-talk** - Tecla configurable
- **Reducción de ruido** - Filtrado integrado

## Conceptos Clave

- **Sala** - Un espacio virtual para comunicarse
- **Push-to-talk** - Presiona una tecla para transmitir
- **Activación de voz** - Transmisión automática al hablar
