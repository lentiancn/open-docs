# Claude Code 安装指南

本指南详细介绍 Claude Code 在各平台上的安装、配置和更新方法。

## 系统要求

Claude Code 支持以下操作系统和配置：

| 组件 | 要求 |
|------|------|
| 操作系统 | macOS 13.0+、Windows 10 1809+ 或 Windows Server 2019+、Ubuntu 20.04+、Debian 10+、Alpine Linux 3.19+ |
| 内存 | 4 GB 以上 |
| 网络 | 需要互联网连接 |
| Shell | Bash、Zsh、PowerShell 或 CMD |
| 地区 | Anthropic 支持的国家和地区 |

### 额外依赖

- **ripgrep**：通常随 Claude Code 一起提供，如果搜索失败请参阅故障排除部分。

## 安装方法

### 方法一：原生安装（推荐）

**macOS、Linux、WSL：**

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell：**

```powershell
irm https://claude.ai/install.ps1 | iex
```

**Windows CMD：**

```batch
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
```

**注意：Windows 需要先安装 [Git for Windows](https://git-scm.com/downloads/win)。**

原生安装会自动在后台更新到最新版本。

### 方法二：Homebrew（macOS/Linux）

```bash
brew install --cask claude-code
```

Homebrew 安装不会自动更新，需要定期运行 `brew upgrade claude-code` 获取最新功能和安全修复。

### 方法三：WinGet（Windows）

```powershell
winget install Anthropic.ClaudeCode
```

WinGet 安装不会自动更新，需要定期运行 `winget upgrade Anthropic.ClaudeCode` 进行手动更新。

## 验证安装

安装完成后，确认 Claude Code 正常工作：

```bash
claude --version
```

如需更详细的安装和配置检查，可以运行：

```bash
claude doctor
```

## 账户认证

Claude Code 需要 Pro、Max、Teams、Enterprise 或 Console 账户。免费 Claude.ai 计划不包含 Claude Code 访问权限。

安装完成后，运行 `claude` 命令并按照浏览器提示登录即可。

## 更新 Claude Code

### 自动更新

原生安装的 Claude Code 会在启动时和运行期间定期检查更新。更新会在后台下载安装，下次启动 Claude Code 时生效。

可以通过配置发布频道来控制更新时机：
- `latest`（默认）：立即接收新功能
- `stable`：使用大约一周前的版本，跳过有重大问题的版本

在设置文件中配置：

```json
{
  "autoUpdatesChannel": "stable"
}
```

### 禁用自动更新

在设置文件中设置环境变量：

```json
{
  "env": {
    "DISABLE_AUTOUPDATER": "1"
  }
}
```

### 手动更新

立即应用更新而不等待后台检查：

```bash
claude update
```

Homebrew 和 WinGet 安装需要手动更新：
- Homebrew：`brew upgrade claude-code`
- WinGet：`winget upgrade Anthropic.ClaudeCode`

## 高级安装选项

### 安装指定版本

安装最新版本：
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

安装稳定版本：
```bash
curl -fsSL https://claude.ai/install.sh | bash -s stable
```

安装特定版本号（如 1.0.58）：
```bash
curl -fsSL https://claude.ai/install.sh | bash -s 1.0.58
```

### 从 npm 迁移到原生安装

如果你之前用 npm 安装过 Claude Code，可以迁移到原生安装：

```bash
# 安装原生二进制
curl -fsSL https://claude.ai/install.sh | bash

# 移除旧的 npm 安装
npm uninstall -g @anthropic-ai/claude-code
```

### 废弃的 npm 安装方式

npm 安装方式已废弃。如果确实需要使用 npm 安装，必须先安装 Node.js 18+：

```bash
npm install -g @anthropic-ai/claude-code
```

**注意：不要使用 `sudo npm install -g`，这会导致权限问题和安全风险。**

## 卸载 Claude Code

### 原生安装卸载

**macOS、Linux、WSL：**

```bash
rm -f ~/.local/bin/claude
rm -rf ~/.local/share/claude
```

**Windows PowerShell：**

```powershell
Remove-Item -Path "$env:USERPROFILE\.local\bin\claude.exe" -Force
Remove-Item -Path "$env:USERPROFILE\.local\share\claude" -Recurse -Force
```

### Homebrew 卸载

```bash
brew uninstall --cask claude-code
```

### WinGet 卸载

```powershell
winget uninstall Anthropic.ClaudeCode
```

### npm 卸载

```bash
npm uninstall -g @anthropic-ai/claude-code
```

### 移除配置文件

**警告：移除配置文件将删除所有设置、允许的工具、MCP 服务器配置和会话历史。**

**macOS、Linux、WSL：**

```bash
# 移除用户设置和状态
rm -rf ~/.claude
rm ~/.claude.json

# 移除项目特定设置（从项目目录运行）
rm -rf .claude
rm -f .mcp.json
```

**Windows PowerShell：**

```powershell
# 移除用户设置和状态
Remove-Item -Path "$env:USERPROFILE\.claude" -Recurse -Force
Remove-Item -Path "$env:USERPROFILE\.claude.json" -Force

# 移除项目特定设置
Remove-Item -Path ".claude" -Recurse -Force
Remove-Item -Path ".mcp.json" -Force
```

## Windows 特殊配置

Claude Code 在 Windows 上需要 Git for Windows 或 WSL。可以从 PowerShell、CMD 或 Git Bash 启动 `claude`。

如果 Claude Code 找不到 Git Bash 安装位置，可以在设置文件中配置路径：

```json
{
  "env": {
    "CLAUDE_CODE_GIT_BASH_PATH": "C:\\Program Files\\Git\\bin\\bash.exe"
  }
}
```

## Alpine Linux 配置

在 Alpine 和其他 musl/uClibc 发行版上，原生安装程序需要 `libgcc`、`libstdc++` 和 `ripgrep`。先使用包管理器安装这些依赖，然后设置 `USE_BUILTIN_RIPGREP=0`。

在 Alpine 上安装所需包：

```bash
apk add libgcc libstdc++ ripgrep
```

然后在设置文件中配置：

```json
{
  "env": {
    "USE_BUILTIN_RIPGREP": "0"
  }
}
```

## 二进制完整性验证

可以通过 SHA256 校验和验证 Claude Code 二进制的完整性。校验和发布在发布清单中，URL 格式为：
`https://storage.googleapis.com/claude-code-dist-86c565f3-f756-42ad-8dfa-d59b1c096819/claude-code-releases/{VERSION}/manifest.json`

签名二进制发布在以下平台：
- **macOS**：由 "Anthropic PBC" 签名并经过 Apple 公证
- **Windows**：由 "Anthropic, PBC" 签名
