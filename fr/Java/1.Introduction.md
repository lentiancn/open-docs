# Introduction à Java

> Java est un langage orienté objet

---

## Description

Java a été créé en 1995 par Sun Microsystems. Connu pour sa portabilité.

---

## Caractéristiques

- Orienté objet
- Multiplateforme (JVM)
- Gestion automatique de la mémoire
- Multithreading
- Écosystème riche
