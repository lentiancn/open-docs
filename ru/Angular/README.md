# Angular

Angular — мощный интерфейсный фреймворк, разработанный Google для создания веб-приложений.

## Введение

Angular — это фреймворк на основе TypeScript, который предоставляет полное решение для создания современных веб-приложений с такими функциями, как двустороннее связывание данных, внедрение зависимостей и мощные инструменты CLI.

## Функции

- **На основе компонентов**: создавайте многократно используемые UI-компоненты
- **TypeScript**: типобезопасная разработка
- **CLI**: мощные инструменты командной строки
- **Маршрутизация**: встроенная система маршрутизации
- **Формы**: шаблонные и реактивные формы
- **HTTP**: HTTP-клиент для API-вызовов
- **Тестирование**: встроенная поддержка тестирования

## Быстрый старт

### Установка

```bash
npm install -g @angular/cli
```

### Создание проекта

```bash
ng new my-app
cd my-app
ng serve
```

Откройте http://localhost:4200
