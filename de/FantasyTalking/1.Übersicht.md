# FantasyTalking - Übersicht

KI-gesteertes Tool zur Talking-Head-Videogenerierung.

## Was ist Talking Head?

Talking Head ist eine KI-Technologie, die realistische Sprechvideos aus einem Standbild und Audio erzeugt.

## Funktionen

- Einzelbild-Generierung
- Mehrsprachig
- Hochwertige Ausgabe
- Gesichtserweiterung
- Einfach zu verwenden
- Open Source
