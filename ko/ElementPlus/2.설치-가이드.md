# Element Plus 설치 가이드

Element Plus의 다양한 설치 방법.

## 환경 요구사항

### 하드웨어 요구사항

- CPU: 2코어 이상
- 메모리: 4GB 이상

### 소프트웨어 요구사항

- Node.js: 18.0+
- npm: 9.0+ / yarn: 1.22+ / pnpm: 8.0+
- Vue: 3.2+
- Vite: 4.0+ (권장)

## 패키지 매니저 설치 (권장)

### npm

```bash
npm install element-plus
```

### yarn

```bash
yarn add element-plus
```

### pnpm

```bash
pnpm add element-plus
```

## 전체 임포트

### main.ts 설정

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')
```

## 필요시 임포트 (권장)

### 자동 임포트 (권장)

**1. 의존성 설치**

```bash
npm install -D unplugin-vue-components unplugin-auto-import
```

**2. Vite 설정**

```typescript
// vite.config.ts
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

**3. 컴포넌트 사용**

```vue
<template>
  <el-button type="primary">안녕하세요</el-button>
</template>
```

### 수동 필요시 임포트

```typescript
import { ElButton, ElInput } from 'element-plus'

export default {
  components: {
    ElButton,
    ElInput,
  },
}
```

```javascript

module.exports = {
  configureWebpack: {
    plugins: [
      AutoImport({

      }),
      Components({
        resolvers: [ElementPlusResolver()],
      }),
    ],
  },

```

## CDN 임포트

### HTML

```html
<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <title>Element Plus CDN</title>
  <link rel="stylesheet" href="//unpkg.com/element-plus/dist/index.css" />
  <script src="//unpkg.com/vue@3"></script>
  <script src="//unpkg.com/element-plus"></script>
</head>
<body>
  <div id="app">
    <el-button type="primary">기본 버튼</el-button>
  </div>

  <script>
    const { createApp, ref } = Vue
    const App = {
      setup() {
        const visible = ref(false)
        return { visible }
      },
      template: `
        <el-button @click="visible = true">다이얼로그 열기</el-button>
        <el-dialog v-model="visible" title="제목">내용</el-dialog>
      `
    }
    const app = createApp(App)
    app.use(ElementPlus)
    app.mount('#app')
  </script>
</body>
</html>
```

## 일반적인 문제

### 아이콘이 표시되지 않음?

아이콘 패키지 설치:

```bash
npm install @element-plus/icons-vue
```

등록:

```typescript
import * as icons from '@element-plus/icons-vue'

for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```

### 스타일 충돌?

CSS Modules 또는 scoped 사용:

```vue
<style scoped>
:deep(.el-button) {
  margin: 10px;
}
</style>
```

### TypeScript 타입 오류?

타입 정의가 설치되어 있는지 확인:

```bash
npm install -D @types/node
```

`tsconfig.json`에 설정:

```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

###国际化 설정

```typescript
import ko from 'element-plus/es/locale/lang/ko'

app.use(ElementPlus, { locale: ko })
```
