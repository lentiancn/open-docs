# Element UI インストールガイド

Element UIはVue 2.xアプリケーション用のUIコンポーネントライブラリです。

## バージョンの概要

| バージョン | Vueバージョン | ステータス |
|------------|-------------|-----------|
| Element UI 2.x | Vue 2.6+ | urrent |
| Element UI 1.x | Vue 2.x | レガシー |

**注意：** Element UIはVue 2専用です。Vue 3にはElement Plusを使用してください。

## 前提条件

### システムの要件

| 要件 | バージョン |
|------|-----------|
| Node.js | 8.0+ |
| npm | 5.0+ |
| Vue | 2.6+ |
| ブラウザ | IE 10+, Chrome, Firefox, Safari |

## インストール方法

### 方法1: npm（推奨）

```bash
# Element UIをインストール
npm install element-ui --save
```

### 方法2: CDN

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Element UI CDNデモ</title>
  <!-- Vue 2を読み込み -->
  <script src="https://unpkg.com/vue@2.6.14/dist/vue.js"></script>
  <!-- Element UIスタイルを読み込み -->
  <link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
  <!-- Element UIを読み込み -->
  <script src="https://unpkg.com/element-ui/lib/index.js"></script>
</head>
```

## プロジェクト設定

### ステップ1: Vueプロジェクトを作成

```bash
npm install -g @vue/cli
vue create my-element-app
cd my-element-app
```

### ステップ2: Element UIをインストール

```bash
npm install element-ui --save
```

### ステップ3: main.jsでインポート

```javascript
import Vue from 'vue'
import App from './App.vue'

import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)

Vue.config.productionTip = false

new Vue({
  render: h => h(App)
}).$mount('#app')
```

## オンデマンドインポート

まず、プラグインをインストール：

```bash
npm install babel-plugin-component -D
```

babel.config.jsを設定：

```javascript
module.exports = {
  presets: ['@vue/cli-plugin-babel/preset'],
  plugins: [
    [
      'component',
      {
        libraryName: 'element-ui',
        styleLibraryName: 'theme-chalk'
      }
    ]
  ]
}
```

## トラブルシューティング

### コンポーネントが見つからない

**解決策：**
1. インストール確認：`npm list element-ui`
2. main.jsでのインポートを確認
3. Vue.use()が呼び出されているか確認

### スタイルが適用されない

**解決策：**
1. CSSをインポート：`import 'element-ui/lib/theme-chalk/index.css'`
