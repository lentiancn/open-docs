# Element Plus 常见问题

常见问题解答。

## 安装问题

### Q: 安装失败？

**解决方案**：
1. 检查 Node.js 版本
```bash
node --version  # 需要 18.0+
```
2. 清理 npm 缓存
```bash
npm cache clean --force
```
3. 删除 node_modules 和 package-lock.json，重新安装
```bash
rm -rf node_modules package-lock.json
npm install
```

### Q: 提示模块找不到？

**解决方案**：
1. 确认包已安装
```bash
npm list element-plus
```
2. 检查 npm/yarn/pnpm 配置
3. 尝试删除 lock 文件重新安装

### Q: TypeScript 类型错误？

**解决方案**：
1. 安装类型定义
```bash
npm install -D @types/node
```
2. 在 tsconfig.json 中添加
```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

## 使用问题

### Q: 组件不显示？

**解决方案**：
1. 确认已正确引入
2. 检查控制台错误信息
3. 确认 Vue 版本（需要 3.2+）

### Q: 样式不生效？

**解决方案**：
1. 确认已引入 CSS
```typescript
import 'element-plus/dist/index.css'
```
2. 检查 CSS 优先级，使用 `:deep()` 或 `!important`
3. 检查是否有其他样式覆盖

### Q: 图标不显示？

**解决方案**：
1. 安装图标包
```bash
npm install @element-plus/icons-vue
```
2. 全局注册
```typescript
import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```
3. 或者按需引入

### Q: Table 表格不渲染？

**解决方案**：
1. 确认 data 属性正确绑定
2. 检查 prop 是否正确
3. 确认 table 在 el-table-column 外面

### Q: Form 表单验证不生效？

**解决方案**：
1. 确保 prop 与 v-model 对应
2. 确保有 ref 引用
3. 正确调用 validate 方法
```typescript
const formRef = ref()
await formRef.value.validate()
```

### Q: Dialog 对话框不显示？

**解决方案**：
1. 检查 v-model 绑定
2. 确认 visible 为 true
3. 检查 z-index 是否有冲突

## 主题问题

### Q: 如何修改主题颜色？

**解决方案**：
使用 CSS 变量：
```css
:root {
  --el-color-primary: #409eff;
}
```

### Q: 如何使用暗色主题？

**解决方案**：
引入暗色 CSS：
```typescript
import 'element-plus/theme-chalk/dark/css-vars.css'
```

### Q: 组件样式怎么覆盖？

**解决方案**：
使用 CSS 选择器：
```css
.el-button--primary {
  background-color: #your-color;
}
```

## 国际化问题

### Q: 如何切换语言？

**解决方案**：
```typescript
import zhCn from 'element-plus/es/locale/lang/zh-cn'
import en from 'element-plus/es/locale/lang/en'

app.use(ElementPlus, { locale: zhCn })
```

### Q: 自定义文案？

**解决方案**：
```typescript
import {ElMessage} from 'element-plus'
ElMessage('自定义信息')
```

## 性能问题

### Q: 首屏加载慢？

**解决方案**：
1. 使用按需引入
2. 使用自动导入插件
3. 配置 Vite 预构建

### Q: 打包体积太大？

**解决方案**：
1. 确保使用按需引入
2. 使用 tree shaking
3. 检查是否引入了完整包

## 兼容性问题

### Q: 不支持 IE 浏览器？

Element Plus 不支持 IE 浏览器，建议使用以下浏览器：
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### Q: 与其他库冲突？

**解决方案**：
1. 检查 z-index 层级
2. 使用 CSS Modules 避免样式冲突
3. 检查 Vue 版本兼容性
