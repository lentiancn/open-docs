# PlantUML Installationsanleitung

PlantUML ist ein Open-Source-Tool zum schnellen Erstellen von UML-Diagrammen. Diese Anleitung behandelt die Installationsmethoden in verschiedenen Umgebungen.

## Versionsinformationen

| Version | Status | Beschreibung |
|------|------|------|
| PlantUML | ✅ Aktuell | Neueste stabile Version |

## Installationsmethoden

### Methode Eins: Installation unter Windows

#### 1. JAR-Datei herunterladen

1. PlantUML JAR herunterladen: https://sourceforge.net/projects/plantuml/files/plantuml.jar/download
2. In einem lokalen Verzeichnis speichern

#### 2. Graphviz installieren (Optional aber empfohlen)

```powershell
# Chocolatey verwenden
choco install graphviz
```

### Methode Zwei: Installation unter macOS

```bash
# Homebrew verwenden
brew install plantuml

# Oder Graphviz installieren
brew install graphviz
```

### Methode Drei: Installation unter Linux

```bash
# Ubuntu/Debian
sudo apt-get install -y plantuml graphviz

# CentOS/RHEL
sudo yum install -y plantuml graphviz
```

### Methode Vier: Als VS Code-Plugin

1. VS Code öffnen
2. Nach "PlantUML" suchen
3. Das "PlantUML"-Plugin installieren
4. Graphviz installieren

### Methode Fünf: Als IntelliJ-Plugin

1. IntelliJ IDEA öffnen
2. Settings → Plugins
3. Nach "PlantUML" suchen
4. Das Plugin installieren

### Methode Sechs: Docker verwenden

```bash
# PlantUML-Server ausführen
docker run -d -p 8080:8080 plantuml/plantuml-server:jetty

# Auf http://localhost:8080 zugreifen
```

## Umgebungskonfiguration

### Java-Anforderungen

```bash
# Java überprüfen
java -version

# Java 8 oder höher erforderlich
```

### Graphviz-Konfiguration

```bash
# Umgebungsvariable festlegen (Linux/macOS)
export GRAPHVIZ_DOT=/usr/bin/dot

# Windows zum PATH hinzufügen
# C:\Program Files\Graphviz\bin
```

## Häufig gestellte Fragen

### Bilder werden nicht angezeigt?

Stellen Sie sicher, dass Graphviz installiert ist:

```bash
dot -V
```

### Chinesische Zeichen werden falsch angezeigt?

Fügen Sie zum Diagramm hinzu:

```plantuml
skinparam defaultFontName SimSun
```
