# Element Plus 사용 매뉴얼

이 매뉴얼은 Element Plus를 빠르게 시작하고 일반적으로 사용되는 컴포넌트와 기능을 마스터하는 데 도움이 됩니다.

## 빠른 시작

Element Plus를 사용하기 전에 설치가 완료되었는지 확인하세요.

### 기본 사용법

Vue 컴포넌트에서 컴포넌트를 직접 사용:

```vue
<template>
  <el-button>기본 버튼</el-button>
  <el-button type="primary">주요 버튼</el-button>
  <el-button type="success">성공 버튼</el-button>
  <el-button type="warning">경고 버튼</el-button>
  <el-button type="danger">위험 버튼</el-button>
</template>
```

## 일반적인 컴포넌트

### 버튼

기본 사용법:

```vue
<el-button>기본</el-button>
<el-button type="primary">주요</el-button>
<el-button type="success">성공</el-button>
<el-button type="warning">경고</el-button>
<el-button type="danger">위험</el-button>
```

다른 크기:

```vue
<el-button size="large">큰</el-button>
<el-button>기본</el-button>
<el-button size="small">작은</el-button>
```

비활성화 상태:

```vue
<el-button disabled>비활성화</el-button>
```

아이콘 버튼:

```vue
<el-button icon="Search">검색</el-button>
<el-button icon="Edit">편집</el-button>
<el-button icon="Delete">삭제</el-button>
```

### 입력

기본 사용법:

```vue
<el-input v-model="input" placeholder="내용을 입력하세요" />
```

아이콘 포함:

```vue
<el-input v-model="input" prefix-icon="Search" />
<el-input v-model="input" suffix-icon="Calendar" />
```

지우기 가능:

```vue
<el-input v-model="input" clearable />
```

비밀번호 입력:

```vue
<el-input v-model="input" type="password" show-password />
```

텍스트 영역:

```vue
<el-input v-model="textarea" type="textarea" :rows="3" />
```

### 선택

기본 선택:

```vue
<el-select v-model="value" placeholder="선택하세요">
  <el-option label="옵션 1" value="1" />
  <el-option label="옵션 2" value="2" />
</el-select>
```

다중 선택:

```vue
<el-select v-model="value" multiple placeholder="선택하세요">
  <el-option label="옵션 1" value="1" />
  <el-option label="옵션 2" value="2" />
</el-select>
```

### 테이블

기본 테이블:

```vue
<el-table :data="tableData" stripe border>
  <el-table-column prop="date" label="날짜" width="180" />
  <el-table-column prop="name" label="이름" width="180" />
  <el-table-column prop="address" label="주소" />
</el-table>
```

인덱스 포함:

```vue
<el-table-column type="index" label="번호" width="60" />
```

고정 열:

```vue
<el-table-column prop="date" label="날짜" width="150" fixed />
```

### 폼

기본 폼:

```vue
<el-form :model="form" label-width="80px">
  <el-form-item label="활동 이름">
    <el-input v-model="form.name" />
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="onSubmit">제출</el-button>
  </el-form-item>
</el-form>
```

폼 유효성 검사:

```vue
<el-form :model="ruleForm" :rules="rules" ref="ruleFormRef">
  <el-form-item label="이름" prop="name">
    <el-input v-model="ruleForm.name" />
  </el-form-item>
</el-form>
```

### 대화상자

```vue
<el-dialog v-model="visible" title="제목" width="30%">
  <span>내용</span>
</el-dialog>
```

### 메시지

```javascript
import { ElMessage } from 'element-plus'

ElMessage.success('성공 메시지')
ElMessage.warning('경고 메시지')
ElMessage.error('오류 메시지')
```

### 메뉴

수평 메뉴:

```vue
<el-menu mode="horizontal">
  <el-menu-item index="1">처리 센터</el-menu-item>
  <el-menu-item index="2">주문 관리</el-menu-item>
</el-menu>
```

수직 메뉴:

```vue
<el-menu default-active="1">
  <el-menu-item index="1">네비게이션 1</el-menu-item>
  <el-menu-item index="2">네비게이션 2</el-menu-item>
</el-menu>
```

### 탭

```vue
<el-tabs v-model="activeName">
  <el-tab-pane label="사용자 관리" name="first">사용자 관리</el-tab-pane>
  <el-tab-pane label="설정 관리" name="second">설정 관리</el-tab-pane>
</el-tabs>
```

### 페이지네이션

```vue
<el-pagination
  v-model:current-page="currentPage"
  :page-size="100"
  :total="1000"
  layout="total, prev, pager, next"
/>
```

## 레이아웃 시스템

### 그리드 시스템

Element Plus는 24열 그리드 시스템을 사용합니다:

```vue
<el-row :gutter="20">
  <el-col :span="12"><div>12열</div></el-col>
  <el-col :span="12"><div>12열</div></el-col>
</el-row>
```

### 컨테이너 레이아웃

```vue
<el-container>
  <el-header>헤더</el-header>
  <el-main>메인</el-main>
  <el-footer>푸터</el-footer>
</el-container>
```

## 일반적인 문제

### 컴포넌트 스타일이 적용되지 않음

CSS가 올바르게 가져와졌는지 확인:

```javascript
import 'element-plus/dist/index.css'
```

### 아이콘이 표시되지 않음

아이콘 라이브러리를 설치해야 합니다:

```bash
npm install @element-plus/icons-vue
```

그런 다음 전역으로 등록:

```javascript
import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```

### 필요 시 가져오기가 작동하지 않음

플러그인 구성이 올바른지 확인하고 올바른 resolver가 사용되고 있는지 확인하세요.
