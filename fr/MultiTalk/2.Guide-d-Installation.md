# MultiTalk Guide d'Installation

Guide d'installation et de configuration de MultiTalk.

## Prérequis

- Windows 10/11, macOS 10.15+, ou Linux
- Microphone et casque/haut-parleurs
- Connexion Internet

## Installation

### Windows

1. Téléchargez l'installateur
2. Exécutez l'installateur
3. Suivez l'assistant d'installation
4. Lancez MultiTalk

### macOS

1. Téléchargez le fichier .dmg
2. Ouvrez l'installateur
3. Faites glisser MultiTalk vers Applications
4. Lancez MultiTalk

## Configuration

### Configuration Audio

1. Ouvrez Paramètres > Audio
2. Sélectionnez votre périphérique d'entrée
3. Sélectionnez votre périphérique de sortie
4. Testez les niveaux audio

### Push-to-talk

1. Allez dans Paramètres > Raccourcis
2. Activez push-to-talk
3. Appuyez sur la combinaison de touches souhaitée

## Dépannage

### Pas d'entrée audio

**Solution:**
- Vérifiez les permissions du microphone
- Vérifiez la sélection du périphérique d'entrée

### Problèmes d'écho

**Solution:**
- Baissez le volume des haut-parleurs
- Utilisez un casque
