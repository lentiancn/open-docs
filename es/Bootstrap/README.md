# Bootstrap

> Marco CSS frontend popular

---

## Descripción del Proyecto

Bootstrap es un marco CSS de código abierto desarrollado y mantenido por Twitter, lanzado por primera vez en 2011. Bootstrap proporciona ricos componentes preconstruidos y un sistema de cuadrícula receptiva para construir rápidamente sitios web y aplicaciones web modernas y receptivas.

## Inicio Rápido

### Incluir CDN

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### Crear una Página

```html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Página Bootstrap</title>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <div class="container">
    <h1>¡Hola Bootstrap!</h1>
  </div>
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```
