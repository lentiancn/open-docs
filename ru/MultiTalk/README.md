# MultiTalk

Мощное приложение для многопользовательного голосового чата для игр и общения.

## Что такое MultiTalk?

MultiTalk — это приложение для многопользовательского голосового чата для геймеров и сообществ.

### Основные функции

- **Многопользовательский голос** - Поддержка нескольких участников
- **Низкая задержка** - Общение в реальном времени
- **Аудиоисточники** - Несколько аудиоустройств
- **Push-to-talk** - Настраиваемая клавиша
- **Шумоподавление** - Встроенная фильтрация
