# Bootstrap 常见问题

> 常见问题解答

---

## 安装问题

### CDN 加载失败？

**解决方案**：

1. 检查网络连接
2. 使用备选 CDN：
```html
<link href="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

### npm 安装后样式不生效？

确保正确引入：

```javascript
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap';
```

---

## 样式问题

### 组件样式不生效？

**常见原因**：

1. 未引入 Bootstrap CSS
2. JavaScript 未正确加载（部分组件需要）
3. CSS 优先级被覆盖

### 响应式不工作？

确保添加 viewport meta 标签：

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

---

## 组件问题

### 下拉菜单不工作？

**检查清单**：

1. 引入 Bootstrap JS（不是仅 CSS）
2. 使用正确的 HTML 结构
3. 检查是否有 JavaScript 错误

### 模态框无法打开？

确保：
1. Bootstrap JS 已加载
2. data-bs-toggle 和 data-bs-target 属性正确
3. 模态框在页面加载时已存在

---

## 定制问题

### 如何自定义主题色？

**方法一：使用 Sass**

```scss
$primary: #ff6600;
@import "bootstrap";
```

**方法二：使用 CSS 变量**

```css
:root {
  --bs-primary: #ff6600;
}
```

---

## 兼容性问题

### 不支持 IE 11？

Bootstrap 5 不再支持 IE 11。如果需要 IE 11 支持，请使用 Bootstrap 4。

---

## 性能问题

### 如何减少文件大小？

1. 仅引入需要的组件（Sass）
2. 使用生产版本（.min.css/.min.js）
3. 启用 gzip 压缩

---

## 与其他框架冲突

### 与 Vue/React 一起使用？

Bootstrap 可以与任何 JavaScript 框架一起使用，但需要注意：

1. 部分组件依赖 jQuery（Bootstrap 5 不再依赖）
2. 使用框架特定的 Bootstrap 包装库（如 react-bootstrap）
