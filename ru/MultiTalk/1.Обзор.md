# MultiTalk Обзор

Мощное приложение для многопользовательского голосового чата.

## Что такое MultiTalk?

MultiTalk — это приложение для многопользовательского голосового чата.

### Основные функции

- **Многопользовательский голос** - Поддержка нескольких участников
- **Низкая задержка** - Общение в реальном времени
- **Аудиоисточники** - Несколько устройств
- **Push-to-talk** - Настраиваемая клавиша
- **Шумоподавление** - Встроенная фильтрация

## Ключевые концепции

- **Комната** - Виртуальное пространство для общения
- **Push-to-talk** - Нажмите клавишу для передачи
- **Голосовая активация** - Автоматическая передача при разговоре
