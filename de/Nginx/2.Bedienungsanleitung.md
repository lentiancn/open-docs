# Nginx Bedienungsanleitung

Anleitung zur effektiven Nutzung von Nginx.

---

## Grundkonfiguration

### Server Block

```nginx
server {
    listen 80;
    server_name beispiel.de;

    root /var/www/html;
    index index.html;

    location / {
        try_files $uri $uri/ =404;
    }
}
```

---

## Reverse Proxy

```nginx
server {
    listen 80;
    server_name api.beispiel.de;

    location / {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
    }
}
```

---

## Load Balancing

```nginx
upstream backend {
    server backend1.beispiel.de;
    server backend2.beispiel.de;
}

server {
    location / {
        proxy_pass http://backend;
    }
}
```

---

## SSL/TLS

```nginx
server {
    listen 443 ssl;
    server_name beispiel.de;

    ssl_certificate /etc/nginx/ssl/cert.pem;
    ssl_certificate_key /etc/nginx/ssl/key.pem;
}
```

---

## Fehlerbehebung

```bash
nginx -t
tail -f /var/log/nginx/error.log
```
