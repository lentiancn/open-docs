# Kotlin - Übersicht

## Überblick

Kotlin ist eine moderne, statisch typisierte Programmiersprache von JetBrains. Sie läuft auf der JVM.

## Was ist Kotlin?

Kotlin ist eine plattformübergreifende, statisch typisierte Programmiersprache mit Typinferenz. Sie ist vollständig mit Java interoperabel.

## Hauptmerkmale

- **Prägnant**: Reduziert Boilerplate-Code
- **Sicher**: Eingebaute Null-Sicherheit
- **Interoperabel**: 100% kompatibel mit Java
- **Coroutines**: Eingebaute Unterstützung für asynchrone Programmierung
