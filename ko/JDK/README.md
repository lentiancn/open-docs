# JDK

Java 개발 키트

## 소개

JDK(Java Development Kit)는 Java 프로그래밍 언어를 위한 소프트웨어 개발 키트이며 모든 Java 개발의 핵심입니다. JDK에는 Java 런타임 환경(JRE), Java 컴파일러(javac), Java 가상 머신(JVM) 및 광범위한 Java 클래스 라이브러리와 도구가 포함되어 있습니다.

## 주요 버전

- **JDK 25** - 새로운 기능과 성능 개선이 포함된 최신 버전
- **JDK 21** - 최신 장기 지원 버전(LTS)
- **JDK 17** - 성숙한 LTS 버전, 널리 사용됨
- **JDK 11** - LTS 버전, Java 유료 기능 분할을 나타냄

## 주요 구성 요소

- **javac** - Java 컴파일러, 소스 코드를 바이트코드로 컴파일
- **java** - Java 프로그램 실행 명령
- **jvm** - Java 가상 머신, 바이트코드 실행
- **jar** - Java 아카이브 도구
- **javadoc** - API 문서 생성 도구
- **jdb** - Java 디버거

## 버전별 차이점

JDK의 서로 다른 주요 버전은 설치, 사용 및 기능에서 중요한 차이가 있습니다:

- **JDK 8**: 상업적 사용을 위한 마지막 무료 Oracle JDK 버전
- **JDK 11**: var, 새로운 HTTP 클라이언트, ZGC(실험적) 도입
- **JDK 17**: NFTC에 따른 마지막 무료 Oracle JDK 버전
- **JDK 21**: 가상 스레드, 패턴 매칭, 레코드(안정)
- **JDK 25**: 최신 기능, 더 짧은 지원 주기
