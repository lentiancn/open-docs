# jQuery

## 概述

jQuery 是一個快速、小巧、功能豐富的 JavaScript 庫。它讓 HTML 文件遍歷和操作、事件處理、動畫以及 Ajax 等操作變得更加簡單，透過一個易於使用的 API 在多種瀏覽器中都能正常工作。

## 特性

- **簡潔的語法**：鏈式呼叫，程式碼簡潔易讀
- **強大的選擇器**：支援 CSS 選擇器的全部功能
- **跨瀏覽器相容**：解決不同瀏覽器之間的 API 差異
- **豐富的插件生態**：龐大的插件庫支援
