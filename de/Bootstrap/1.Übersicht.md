# Bootstrap Übersicht

Das beliebteste CSS-Framework der Welt.

## Was ist Bootstrap?

Bootstrap ist das beliebteste CSS-Framework, entwickelt von Twitter, für responsive und mobile-first Webseiten.

### Hauptfunktionen

- **Responsives Gridsystem** - 12-Spalten responsives Grid
- **Vorgefertigte Komponenten** - Navbars, Karten, Modals
- **CSS-Hilfsklassen** - Schnelle Stilanpassungen
- **JavaScript-Komponenten** - Interaktive Komponenten

### Statistiken

- GitHub Sterne: 160.000+
- Websites: Millionen
