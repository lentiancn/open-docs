# Python 설치 가이드

> 환경 설정

---

## 설치

### Windows

```powershell
choco install python
```

### macOS

```bash
brew install python@3.12
```

### Linux

```bash
sudo apt install python3 python3-pip
```

---

## 패키지

```bash
pip install requests
python -m venv myenv
```

## 확인

```bash
python --version
```
