# V-Express Häufig Gestellte Fragen

## Allgemeine Fragen

### Was ist V-Express?
V-Express ist ein KI-Modell, das Sprechkopf-Videos aus einem einzelnen Porträtbild und Audio-Eingabe generiert. Es erstellt realistische Gesichtsanimationen mit Lippensynchronisation.

### Ist V-Express kostenlos nutzbar?
V-Express ist Open-Source für Forschungszwecke. Überprüfen Sie die Lizenz für kommerzielle Nutzung.

### Was unterscheidet V-Express von anderen Sprechgeneratoren?
V-Express verwendet:
- Progressive conditional diffusion decoding
- Weakness-aware VQ-Module
- Multi-level conditioning
- Bessere Handhabung schwacher Ausdrücke

## Installationsfragen

### Was sind die Anforderungen?
```bash
# Python-Umgebung
conda create -n vexpress python=3.8
conda activate vexpress

# Abhängigkeiten installieren
pip install torch torchvision
pip install opencv-python
pip install librosa
pip install -r requirements.txt
```

### Wie lade ich die Modelle herunter?
```bash
bash scripts/download_models.sh
```

### Brauche ich eine leistungsstarke GPU?
Ja, eine GPU mit 8GB+ VRAM wird empfohlen. Mehr VRAM ermöglicht bessere Qualität und schnelleres Processing.

## Nutzungsfragen

### Wie generiere ich ein Sprechvideo?
```bash
python inference.py \
    --reference_image path/to/image.jpg \
    --audio path/to/audio.wav \
    --output output.mp4
```

### Welche Bildformate werden unterstützt?
- JPG/JPEG
- PNG
- BMP
- WebP

### Welche Audioformate werden unterstützt?
- WAV (empfohlen)
- MP3
- FLAC

## Fehlerbehebung

### Schlechte Lippensynchronisation
- Verwenden Sie hochwertiges Audio ohne Hintergrundgeräusche
- Stellen Sie sicher, dass die Sprache im Audio klar ist

### Identitätsverzerrung
- Verwenden Sie ein hochauflösendes, frontales Bild
- Vermeiden Sie stark bearbeitete Fotos

### Artefakte in der Ausgabe
- Verringern Sie die Ausgabeauflösung
- Überprüfen Sie die ordnungsgemäße Modellinstallation

### Out-of-Memory-Fehler
- Verringern Sie die Batch-Größe
- Senken Sie die Ausgabeauflösung
- Schließen Sie andere GPU-Anwendungen
