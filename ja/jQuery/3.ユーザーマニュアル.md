# jQuery ユーザーマニュアル

## クイックスタート

### jQueryの読み込み

HTMLファイルの `<head>` または `<body>` の最後にjQueryを読み込みます：

```html
<!DOCTYPE html>
<html>
<head>
    <title>jQuery デモ</title>
    <script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
</head>
<body>
    <!-- ページコンテンツ -->
</body>
</html>
```

### DOM準備完了を待つ

jQueryコードはDOMが完全にロードされた後に実行する必要があります：

```javascript
$(document).ready(function() {
    // ここにjQueryコードを記述
});

// 省略形
$(function() {
    // ここにjQueryコードを記述
});
```

## セレクター

jQueryセレクターはjQueryのコア機能の一つで、CSSセレクターと同じ構文です。

### 基本的なセレクター

```javascript
$("#myId")          // IDセレクター
$(".myClass")       // クラスセレクター
$("div")            // 要素セレクター
$("*")              // オールセレクター
$("div, p, span")   // 複数セレクター
```

### 属性セレクター

```javascript
$('[href]')                    // href属性を持つ要素
$('[href="value"]')           // hrefが指定値と等しい要素
$('[href!="value"]')          // hrefが指定値と等しくない要素
$('[href^="http"]')           // hrefが指定値で始まる要素
$('[href$=".pdf"]')           // hrefが指定値で終わる要素
$('[href*="google"]')         // hrefが指定値を含む要素
```

### フィルタセレクター

```javascript
$("div:first")        // 最初のdiv要素
$("div:last")         // 最後のdiv要素
$("div:even")         // 偶数位置的div要素
$("div:odd")          // 奇数位置的div要素
$("div:eq(n)")        // インデックスnのdiv要素
$("div:gt(n)")        // インデックスnより大きいdiv要素
$("div:lt(n)")        // インデックスnより小さいdiv要素
$("div:not(.exclude") // 指定クラスを持たないdiv要素
```

## DOM操作

### コンテンツの取得と設定

```javascript
// テキストコンテンツの取得/設定
var text = $("#myElement").text();
$("#myElement").text("新しいテキストコンテンツ");

// HTMLコンテンツの取得/設定
var html = $("#myElement").html();
$("#myElement").html("<p>新しいHTMLコンテンツ</p>");

// フォーム値の取得/設定
var value = $("#inputField").val();
$("#inputField").val("新しい値");
```

### 属性の取得と設定

```javascript
// 属性の取得
var src = $("#myImage").attr("src");

// 単一属性の設定
$("#myImage").attr("src", "image.jpg");

// 複数の属性の設定
$("#myImage").attr({
    src: "image.jpg",
    alt: "画像の説明",
    title: "画像のタイトル"
});

// 属性の削除
$("#myImage").removeAttr("title");
```

### CSSの取得と設定

```javascript
// 単一CSSプロパティの取得
var color = $("#myElement").css("color");

// 単一CSSプロパティの設定
$("#myElement").css("color", "red");

// 複数のCSSプロパティの設定
$("#myElement").css({
    color: "red",
    backgroundColor: "blue",
    fontSize: "16px"
});

// クラスの追加/削除
$("#myElement").addClass("active highlight");
$("#myElement").removeClass("highlight");
$("#myElement").toggleClass("active");  // クラス切り替え
```

### DOMトラバーサル

```javascript
// 親要素
$("#myElement").parent();
$("#myElement").parents();
$("#myElement").closest(".container");

// 子要素
$("#myElement").children();
$("#myElement").find(".child");

// 兄弟要素
$("#myElement").siblings();
$("#myElement").next();
$("#myElement").prev();
$("#myElement").nextAll();
```

### 要素の作成と削除

```javascript
// 新規要素の作成
var newDiv = $("<div>", {
    class: "new-element",
    text: "新しい要素のコンテンツ"
});

// ページに追加
$("#container").append(newDiv);      // 内側の末尾
$("#container").prepend(newDiv);    // 内側の先頭
$("#container").before(newDiv);      // 外側の前
$("#container").after(newDiv);       // 外側の後

// 要素の削除
$("#myElement").remove();             // 要素と子要素を削除
$("#myElement").empty();              // 子要素のみ削除
```

## イベント処理

### イベントのバインド

```javascript
// クリックイベントのバインド
$("#myButton").click(function() {
    alert("ボタンがクリックされました");
});

// onメソッドを使用
$("#myButton").on("click", function() {
    alert("ボタンがクリックされました");
});

// 複数イベントのバインド
$("#myElement").on({
    mouseenter: function() {
        $(this).css("background", "yellow");
    },
    mouseleave: function() {
        $(this).css("background", "");
    }
});
```

### イベント委譲

```javascript
// 動的要素のイベントバインド
$("#container").on("click", ".dynamic-item", function() {
    alert("動的要素がクリックされました");
});
```

### 一般的なイベント

| イベント | 説明 |
|----------|------|
| click | クリックイベント |
| dblclick | ダブルクリックイベント |
| mouseenter | マウス 진입 |
| mouseleave | マウス退出 |
| mousemove | マウス移動 |
| keydown | キー押下 |
| keyup | キー開放 |
| focus | フォーカス獲得 |
| blur | フォーカス喪失 |
| submit | フォーム送信 |
| change | 値変更 |
| load | ロード完了 |

## アニメーション効果

### javascript
// 要素を表示
$("#myElement").基本的なアニメーション

```show();

// 要素を非表示
$("#myElement").hide();

// 表示状態を切り替え
$("#myElement").toggle();

// フェードイン
$("#myElement").fadeIn();

// フェードアウト
$("#myElement").fadeOut();

// フェード切り替え
$("#myElement").fadeToggle();

// スライドダウン
$("#myElement").slideDown();

// スライドアップ
$("#myElement").slideUp();

// スライド切り替え
$("#myElement").slideToggle();
```

### カスタムアニメーション

```javascript
$("#myElement").animate({
    opacity: 0.5,
    left: "+=50",
    height: "200px"
}, 300, function() {
    // アニメーション後のコールバック
    console.log("アニメーション完了");
});
```

### アニメーションの停止

```javascript
// 現在のアニメーションを停止
$("#myElement").stop();

// すべてのアニメーションを停止し、最終状態に達する
$("#myElement").stop(true, true);
```

## Ajaxリクエスト

### 基本的な使用法

```javascript
$.ajax({
    url: "/api/data",
    method: "GET",
    data: { id: 1, name: "test" },
    success: function(response) {
        console.log("成功:", response);
    },
    error: function(xhr, status, error) {
        console.log("エラー:", error);
    }
});
```

### ショートカットメソッド

```javascript
// GETリクエスト
$.get("/api/data", { id: 1 }, function(response) {
    console.log(response);
});

// POSTリクエスト
$.post("/api/submit", { name: "test" }, function(response) {
    console.log(response);
});

// HTMLの読み込み
$("#content").load("fragment.html");
```

### 完全な例

```javascript
$.ajax({
    url: "https://api.example.com/users",
    type: "GET",
    dataType: "json",
    timeout: 5000,
    headers: {
        "Authorization": "Bearer token"
    },
    beforeSend: function(xhr) {
        // リクエスト送信前に実行
        console.log("リクエスト送信準備中");
    },
    success: function(data) {
        // リクエスト成功
        $("#user-list").html("");
        data.forEach(function(user) {
            $("#user-list").append("<li>" + user.name + "</li>");
        });
    },
    error: function(xhr, status, error) {
        // リクエスト失敗
        alert("読み込み失敗: " + error);
    },
    complete: function(xhr, status) {
        // リクエスト完了（成功または失敗）
        console.log("リクエスト完了, ステータス: " + status);
    }
});
```

## ユーティリティ関数

### 配列とオブジェクト操作

```javascript
// 配列の反復
$.each([1, 2, 3], function(index, value) {
    console.log(index + ": " + value);
});

// オブジェクトの反復
$.each({ name: "John", age: 30 }, function(key, value) {
    console.log(key + ": " + value);
});

// 配列のフィルタリング
var filtered = $.grep([1, 2, 3, 4, 5], function(value) {
    return value > 3;
});

// 配列のマッピング
var mapped = $.map([1, 2, 3], function(value) {
    return value * 2;
});

// オブジェクトのマージ
var merged = $.extend({}, obj1, obj2);
```

### 文字列操作

```javascript
// 空白の削除
$.trim("  hello  ");  // "hello"

// 特殊文字のエスケープ
$.escapeSelector(".myClass");  // エスケープ
```

### 型チェック

```javascript
$.isArray([]);        // true
$.isFunction(fn);     // true
$.isEmptyObject({});  // true
$.isPlainObject({});  // true
```

## チェーン呼び出し

jQueryはチェーンをサポートしており、同じ要素に対して連続して操作できます：

```javascript
$("#myElement")
    .addClass("highlight")
    .css("color", "red")
    .fadeIn()
    .animate({ height: "100px" }, function() {
        console.log("完了");
    });
```

`end()`メソッドを使用すると前のセレクターに戻れます：

```javascript
$("#myElement")
    .find(".child")
    .addClass("active")
    .end()    // #myElementに戻る
    .removeClass("highlight");
```
