# OpenCode インストールガイド

OpenCode は複数のインストール方法に対応しています。

## Linux/macOS インストール

### 推奨：インストールスクリプト

```bash
curl -fsSL https://opencode.ai/install | bash
```

### Node.js

```bash
npm install -g opencode-ai
```

### Bun

```bash
bun install -g opencode-ai
```

### pnpm

```bash
pnpm install -g opencode-ai
```

### Yarn

```bash
yarn global add opencode-ai
```

### Homebrew (macOS/Linux)

```bash
brew install anomalyco/tap/opencode
```

### Arch Linux

```bash
# 安定版
sudo pacman -S opencode

# 最新版（AUR）
paru -S opencode-bin
```

## Windows インストール

### Chocolatey

```bash
choco install opencode
```

### Scoop

```bash
scoop install opencode
```

### NPM

```bash
npm install -g opencode-ai
```

### Mise

```bash
mise use -g github:anomalyco/opencode
```

### Docker

```bash
docker run -it --rm ghcr.io/anomalyco/opencode
```

### 手動ダウンロード

[GitHub Releases](https://github.com/anomalyco/opencode/releases) からバイナリをダウンロードも可能。

## インストール確認

```bash
opencode --version
```

## 設定

### LLM プロバイダへの接続

以下を実行して接続：

```bash
opencode
/connect
```

`opencode` を選択し、[opencode.ai/auth](https://opencode.ai/auth) にアクセスしてログインし、API キーを取得。

### 他のプロバイダ

他の LLM プロバイダも選択可能。詳細は[公式ドキュメント](https://opencode.ai/docs/providers#directory)を参照。

## プロジェクト初期化

 /path/to/project```bash
cd
opencode
/init
```

これにより、OpenCode がプロジェクトを分析し、ルートディレクトリに AGENTS.md ファイルを作成します。
