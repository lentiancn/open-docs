# Element UI 安装指南

本指南详细介绍在各种项目中安装和使用 Element UI 的方法。

## 环境要求

Element UI 对运行环境有以下要求：

- Vue 2.0.0+
- Node.js 8.0.0+
- 现代浏览器（Chrome、Firefox、Safari、Edge 10+）

## 安装方式

### 使用包管理器

推荐使用 npm 安装：

```bash
npm install element-ui --save
```

或者使用 yarn：

```bash
yarn add element-ui
```

## 完整引入

在 main.js 中引入：

```javascript
import Vue from 'vue'
import Element from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(Element)

new Vue({
  el: '#app',
  render: h => h(App)
})
```

### 按需导入

按需导入可以显著减少项目体积。

首先安装 babel-plugin-component：

```bash
npm install babel-plugin-component -D
```

然后在 .babelrc 中配置：

```json
{
  "presets": [["env", {
    "modules": false,
    "targets": {
      "browsers": ["> 1%", "last 2 versions", "not ie <= 8"]
    }
  }]],
  "plugins": [
    ["component", {
      "libraryName": "element-ui",
      "styleLibraryName": "theme-chalk"
    }]
  ]
}
```

按需引入组件：

```javascript
import { Button, Select } from 'element-ui'

Vue.component(Button.name, Button)
Vue.component(Select.name, Select)
```

## 快捷搭建项目模板

Element UI 提供了官方项目模板：

- **Vue CLI 模板**：https://github.com/ElementUI/element-starter
- **Laravel 模板**：https://github.com/ElementUI/element-in-laravel-starter

## 全局配置

在引入 Element UI 时，可以传入全局配置对象：

```javascript
import Vue from 'vue'
import Element from 'element-ui'

Vue.use(Element, {
  size: 'small',
  zIndex: 2000
})
```

配置项说明：
- size：设置表单组件的默认尺寸，可选值：medium / small / mini
- zIndex：设置弹出组件的层级，默认值为 2000

## 使用 Nuxt.js

对于 Nuxt 项目，可以使用 nuxt-element：

```bash
npm install nuxt-element
```

在 nuxt.config.js 中配置：

```javascript
module.exports = {
  modules: ['nuxt-element']
}
```

更多信息请访问 [Element UI 官网](https://element.eleme.io/)。
