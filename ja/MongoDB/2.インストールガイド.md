# MongoDB インストールガイド

## バージョン選択とプラットフォーム要件

### バージョン選択

MongoDB には現在2つの主要なバージョンシリーズがあります。

- **MongoDB 8.2**: 最新安定バージョン、新規プロジェクトに推奨
- **MongoDB 7.0**: 長期サポートバージョン、安定性が必要な本番環境に適しています

MongoDB 8.2 以降では、特定のユースケースに対応したオンプレミス展開でマイnorリリースを利用できます。

### プラットフォーム要件

- MongoDB は 32 ビットプラットフォームをサポートしていません。オペレーティングシステムが 64 ビットであることを確認してください
- MongoDB 6.0 は2025年7月31日にサポートが終了しました

MongoDB コミュニティ版は次のプラットフォームとアーキテクチャをサポートしています。

## Ubuntu に MongoDB コミュニティ版をインストールする

### ステップ1: パッケージマネージャの公開鍵をインポートする

```bash
curl -fsSL https://www.mongodb.org/static/pgp/server-8.0.asc | sudo gpg -o /usr/share/keyrings/mongodb-server-8.0.gpg --dearmor
```

### ステップ2: MongoDB リポジトリリストファイルを作成する

Ubuntu のバージョンに応じて、適切な設定を選択します。

**Ubuntu 24.04 (Noble)**:

```bash
echo "deb [ signed-by=/usr/share/keyrings/mongodb-server-8.0.gpg ] http://repo.mongodb.org/apt/debian noble/mongodb-org/8.0 main" | sudo tee /etc/apt/sources.list.d/mongodb-org-8.0.list
```

**Ubuntu 22.04 (Jammy)**:

```bash
echo "deb [ signed-by=/usr/share/keyrings/mongodb-server-8.0.gpg ] http://repo.mongodb.org/apt/debian jammy/mongodb-org/8.0 main" | sudo tee /etc/apt/sources.list.d/mongodb-org-8.0.list
```

**Ubuntu 20.04 (Focal)**:

```bash
echo "deb [ signed-by=/usr/share/keyrings/mongodb-server-8.0.gpg ] http://repo.mongodb.org/apt/debian focal/mongodb-org/8.0 main" | sudo tee /etc/apt/sources.list.d/mongodb-org-8.0.list
```

### ステップ3: パッケージリストを更新する

```bash
sudo apt update
```

### ステップ4: MongoDB をインストールする

最新バージョンをインストール:

```bash
sudo apt install -y mongodb-org
```

または特定のバージョンをインストール:

```bash
sudo apt install -y mongodb-org=8.0.0 mongodb-org-server=8.0.0 mongodb-org-shell=8.0.0 mongodb-org-mongos=8.0.0 mongodb-org-tools=8.0.0
```

### ステップ5: MongoDB を起動する

mongod サービスを起動:

```bash
sudo systemctl start mongod
```

自動起動を設定:

```bash
sudo systemctl enable mongod
```

## RHEL/CentOS/Rocky Linux にインストールする

### YUM/DNF を使用してインストールする

#### ステップ1: リポジトリファイルを作成する

`/etc/yum.repos.d/mongodb-org.repo` ファイルを作成:

```bash
[mongodb-org-8.0]
name=MongoDB Repository
baseurl=https://repo.mongodb.org/yum/redhat/$releasever/mongodb-org/8.0/x86_64/
gpgcheck=1
enabled=1
gpgkey=https://www.mongodb.org/static/pgp/server-8.0.asc
```

#### ステップ2: MongoDB をインストールする

DNF を使用（推奨）:

```bash
sudo dnf install -y mongodb-org
```

または YUM を使用:

```bash
sudo yum install -y mongodb-org
```

#### ステップ3: MongoDB を起動する

```bash
sudo systemctl start mongod
sudo systemctl enable mongod
```

## macOS にインストールする

### Homebrew を使用してインストールする

#### ステップ1: Homebrew を更新する

```bash
brew update
```

#### ステップ2: MongoDB をインストールする

```bash
brew install mongodb-community
```

#### ステップ3: MongoDB を起動する

バックグラウンドサービスとして起動（推奨）:

```bash
brew services start mongodb-community
```

または手動で実行:

```bash
mongod --config /usr/local/etc/mongod.conf
```

#### ステップ4: MongoDB を停止する

```bash
brew services stop mongodb-community
```

### 手動インストール

#### ステップ1: MongoDB をダウンロードする

```bash
curl -O https://fastdl.mongodb.org/osx/mongodb-macos-x86_64-8.0.0.tgz
```

ARM64 アーキテクチャの場合:

```bash
curl -O https://fastdl.mongodb.org/osx/mongodb-macos-arm64-8.0.0.tgz
```

#### ステップ2: 解凍する

```bash
tar -zxvf mongodb-macos-*.tgz
```

#### ステップ3: バイナリファイルを移動する

```bash
sudo mkdir -p /usr/local/bin
sudo cp mongodb-macos-*/bin/* /usr/local/bin/
```

#### ステップ4: データディレクトリを作成する

```bash
sudo mkdir -p /data/db
sudo chown -R $(whoami) /data/db
```

#### ステップ5: MongoDB を起動する

```bash
mongod --dbpath /data/db
```

## Windows にインストールする

### MSI インストーラーを使用する

1. MongoDB ダウンロードセンターから Windows 用 MSI インストーラーをダウンロードします
2. インストーラーを実行し、ウィザードの手順に従います
3. 「Complete」インストールタイプを選択してすべてのコンポーネントをインストールします
4. インストール完了後、MongoDB は自動的に Windows サービスとしてインストールされます

### ZIP パッケージを使用する

#### ステップ1: MongoDB をダウンロードする

MongoDB ダウンロードセンターから Windows ZIP パッケージをダウンロードします。

#### ステップ2: 解凍する

ターゲットディレクトリ（例: `C:\Program Files\MongoDB`）にファイルを解凍します。

#### ステップ3: データディレクトリを作成する

```cmd
mkdir C:\data\db
```

#### ステップ4: MongoDB を起動する

```cmd
"C:\Program Files\MongoDB\Server\8.0\bin\mongod.exe" --dbpath C:\data\db
```

## Docker を使用してインストールする

### MongoDB コミュニティ版をインストールする

#### ステップ1: イメージをプルする

```bash
docker pull mongodb/mongodb-community-server
```

#### ステップ2: コンテナを実行する

```bash
docker run --name mongodb -d -p 27017:27017 mongodb/mongodb-community-server
```

#### データの永続化

データを永続化するには、ボリュームマウントを使用:

```bash
docker run --name mongodb -d -p 27017:27017 -v mongodb_data:/data/db mongodb/mongodb-community-server
```

### Docker Compose を使用する

`docker-compose.yml` ファイルを作成:

```yaml
version: '3.8'
services:
  mongodb:
    image: mongodb/mongodb-community-server
    container_name: mongodb
    ports:
      - "27017:27017"
    volumes:
      - mongodb_data:/data/db
    environment:
      - MONGO_INITDB_ROOT_USERNAME=admin
      - MONGO_INITDB_ROOT_PASSWORD=yourpassword

volumes:
  mongodb_data:
```

起動:

```bash
docker-compose up -d
```

## インストールを確認する

### MongoDB に接続する

インストール後、mongosh（MongoDB Shell）を使用して接続:

```bash
mongosh
```

接続に成功すると、MongoDB のインタラクティブシェルプロンプトが表示されます。

### バージョンを確認する

mongosh でバージョンを確認:

```bash
db.version()
```

### 基本的なコマンド

```bash
# データベースを表示
show dbs

# データベースを切り替え
use test

# 現在のデータベースを表示
db

# ヘルプを表示
db.help()
```

## インストール後の設定

### 管理者ユーザーを作成する

mongosh で実行:

```bash
use admin
db.createUser({
  user: "admin",
  pwd: "your_secure_password",
  roles: [
    { role: "userAdminAnyDatabase", db: "admin" },
    { role: "readWriteAnyDatabase", db: "admin" }
  ]
})
```

### 認証を有効にする

1. MongoDB サービスを停止
2. `/etc/mongod.conf` を編集して認証を有効にする:

```yaml
security:
  authorization: enabled
```

3. MongoDB サービスを再起動:

```bash
sudo systemctl restart mongod
```

### リモート接続を設定する

1. `/etc/mongod.conf` のネットワーク設定を変更:

```yaml
net:
  port: 27017
  bindIp: 0.0.0.0
```

2. MongoDB サービスを再起動

3. ファイアウォールでポートを開く:

```bash
sudo ufw allow 27017
```

## MongoDB をアンインストールする

### Ubuntu/Debian

```bash
sudo apt remove --purge mongodb-org
sudo rm -rf /var/lib/mongodb
sudo rm -rf /var/log/mongodb
sudo rm -rf /etc/mongod.conf
```

### macOS (Homebrew)

```bash
brew uninstall mongodb-community
brew services stop mongodb-community
rm -rf /usr/local/etc/mongod.conf
rm -rf /data/db
```

### RHEL/CentOS

```bash
sudo yum remove -y mongodb-org
sudo rm -rf /var/lib/mongo
sudo rm -rf /var/log/mongodb
```

### Windows

コントロールパネルの「プログラムと機能」から MongoDB をアンインストールし、データディレクトリを手動で削除します。

## よくある質問

### MongoDB のデフォルトポートは？

デフォルトポートは 27017 です。`--port` パラメータで別のポートを指定することもできます。

### MongoDB はどこにデータを保存しますか？

デフォルトでは:

- Linux/macOS: `/data/db`
- Windows: `C:\data\db`

設定ファイルの `storage.dbPath` を変更して指定できます。

### MongoDB の実行状態を確認するには？

```bash
sudo systemctl status mongod
```

または mongosh で:

```bash
db.adminCommand({ ping: 1 })
```
