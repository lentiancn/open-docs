# Vue 3 사용 가이드

Vue 3를 효과적으로 사용하는 가이드입니다.

---

## Composition API

```javascript
import { ref } from 'vue'

const count = ref(0)
count.value++
```

---

## 템플릿

```html
<h1>{{ title }}</h1>
<button @click="handleClick">클릭</button>
```

---

## 컴포넌트

```vue
<script setup>
defineProps({
  title: String
})
</script>
```
