# JMeter 설치 가이드

## 전제 조건

- **Java Development Kit (JDK)**: JMeter에는 Java 8 이상이 필요
- **운영 체제**: Windows, Linux 또는 macOS
- **메모리**: 부하 테스트에는 최소 4GB RAM 권장

## 설치 단계

### 1. JMeter 다운로드

공식 Apache JMeter 웹사이트 방문: https://jmeter.apache.org/

최신 버전 다운로드(일반적으로 .zip 또는 .tar.gz 파일).

### 2. 아카이브 추출

다운로드한 파일을 원하는 위치에 추출:

```bash
# Linux/Mac
tar -xvf apache-jmeter-x.x.x.tgz

# Windows
# 7-Zip 또는 WinRAR를 사용하여 추출
```

### 3. 환경 변수 설정(선택 사항)

편의상 시스템 PATH에 JMeter를 추가할 수 있습니다:

```bash
# Linux/Mac(~/.bashrc 또는 ~/.zshrc에 추가)
export JMETER_HOME=/path/to/apache-jmeter-x.x.x
export PATH=$PATH:$JMETER_HOME/bin

# Windows
# 시스템 속성 -> 환경 변수 -> PATH에 추가
```

### 4. 설치 확인

JMeter가 올바르게 설치되었는지 확인하려면 다음 명령을 실행:

```bash
# Linux/Mac
./jmeter.sh --version

# Windows
jmeter.bat --version
```

### 5. JMeter GUI 시작

```bash
# Linux/Mac
./jmeter.sh

# Windows
jmeter.bat
```

## 디렉토리 구조

```
apache-jmeter/
├── bin/          # 실행 스크립트 및 JMeter 실행 파일
├── lib/          # JMeter 라이브러리
├── lib/ext/      # 플러그인 및 확장
├── docs/         # 문서
├── printable_docs/  # 사용자 매뉴얼
└── LICENSE       # 라이선스 파일
```

## 다양한 모드로 실행

### GUI 모드(대화형)
```bash
jmeter.sh          # Linux/Mac
jmeter.bat         # Windows
```

### CLI 모드(헤드리스)
```bash
jmeter -n -t testplan.jmx -l results.jtl -e -o html-report
```

매개변수:
- `-n`: 비 GUI 모드로 실행
- `-t`: 테스트 플랜 파일
- `-l`: 결과 파일
- `-e`: 테스트 후 HTML 보고서 생성
- `-o`: HTML 보고서 출력 디렉토리

## 설치 문제 해결

### Java를 찾을 수 없음
JAVA_HOME이 올바르게 설정되었는지 확인:
```bash
export JAVA_HOME=/path/to/jdk
```

### 메모리 문제
jmeter 시작 스크립트를 편집하여 힙 크기 증가:
```bash
HEAP="-Xms1g -Xmx4g"
```
