# Element Plus FAQ

This page lists common issues and solutions when using Element Plus.

## Installation Issues

### Installation Failed

**Symptoms**: npm install or yarn add fails

**Solutions**:

1. Check Node.js version, ensure >= 14.0.0
2. Clear npm cache:
   ```bash
   npm cache clean --force
   ```
3. Delete node_modules and package-lock.json, then reinstall

### TypeScript Type Errors

**Symptoms**: Type errors when importing components

**Solutions**:

1. Make sure Volar plugin is installed
2. Add to tsconfig.json:
   ```json
   {
     "compilerOptions": {
       "types": ["element-plus/global"]
     }
   }
   ```

## Usage Issues

### Component Styles Not Applied

**Symptoms**: Component styles don't match expectations

**Solutions**:

1. Make sure CSS is correctly imported:
   ```javascript
   import 'element-plus/dist/index.css'
   ```

### Icons Not Displayed

**Symptoms**: Icon components show blank or squares

**Solutions**:

1. Install icon library:
   ```bash
   npm install @element-plus/icons-vue
   ```

2. Register icons globally:
   ```javascript
   import * as icons from '@element-plus/icons-vue'
   for (const [key, component] of Object.entries(icons)) {
     app.component(key, component)
   }
   ```

### On-Demand Import Not Working

**Symptoms**: Bundle size still large

**Solutions**:

1. Check if plugin configuration is correct
2. Ensure correct resolver is used

### Internationalization Not Working

**Solutions**:

1. Import language pack:
   ```javascript
   import en from 'element-plus/es/locale/lang/en'
   ```
2. Configure locale:
   ```javascript
   app.use(ElementPlus, { locale: en })
   ```

## Component Issues

### Table Sorting Not Working

Ensure prop attribute is set:

```vue
<el-table-column prop="date" sortable />
```

### Form Validation Not Triggering

Ensure el-form-item has prop attribute:

```vue
<el-form-item label="Name" prop="name">
  <el-input v-model="ruleForm.name" />
</el-form-item>
```

## Style Issues

### Style Conflicts

**Solutions**:

1. Use more specific selectors
2. Use ::v-deep or :deep()

### Theme Color Not Applied

Check CSS variables:

```css
:root {
  --el-color-primary: #409eff;
}
```

## Performance Issues

### Page Lag

For large datasets:

1. Use el-table-v2 (virtual table)
2. Use pagination to load data

### Bundle Size Too Large

1. Use on-demand import
2. Remove unused plugins

## Vue Version Issues

### Not Compatible with Vue 2

Element Plus only supports Vue 3. If you need Vue 2, use Element UI.
