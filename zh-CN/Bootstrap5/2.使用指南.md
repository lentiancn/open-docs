# Bootstrap 5 使用指南

高效使用 Bootstrap 5 的完整指南。

---

## 栅格系统

### 基本栅格

```html
<div class="container">
  <div class="row">
    <div class="col">列1</div>
    <div class="col">列2</div>
    <div class="col">列3</div>
  </div>
</div>
```

### 响应式列

```html
<div class="row">
  <div class="col-12 col-md-6 col-lg-4">响应式</div>
</div>
```

---

## 组件

### 按钮

```html
<button class="btn btn-primary">主要</button>
<button class="btn btn-secondary">次要</button>
<button class="btn btn-success">成功</button>
<button class="btn btn-danger">危险</button>
<button class="btn btn-warning">警告</button>
<button class="btn btn-info">信息</button>
```

### 卡片

```html
<div class="card" style="width: 18rem;">
  <div class="card-body">
    <h5 class="card-title">卡片标题</h5>
    <p class="card-text">一些示例文本。</p>
    <a href="#" class="btn btn-primary">按钮</a>
  </div>
</div>
```

### 导航栏

```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">品牌</a>
  </div>
</nav>
```

### 模态框

```html
<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#myModal">
  打开模态框
</button>

<div class="modal fade" id="myModal">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title">标题</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
      </div>
      <div class="modal-body">
        内容
      </div>
    </div>
  </div>
</div>
```

## 表单

### 基本表单

```html
<form>
  <div class="mb-3">
    <label for="email" class="form-label">邮箱</label>
    <input type="email" class="form-control" id="email">
  </div>
  <button type="submit" class="btn btn-primary">提交</button>
</form>
```

### 选择框

```html
<select class="form-select">
  <option selected>打开选择菜单</option>
  <option value="1">选项1</option>
  <option value="2">选项2</option>
</select>
```

## 实用工具

### 间距

```html
<div class="m-0">外边距: 0</div>
<div class="mt-3">上边距: 1rem</div>
<div class="p-4">内边距: 1.5rem</div>
```

### 文本

```html
<p class="text-start">左对齐</p>
<p class="text-center">居中</p>
<p class="text-end">右对齐</p>
<p class="text-primary">主要颜色</p>
```

### 颜色

| 类名 | 颜色 |
|------|------|
| `text-primary` | 蓝色 |
| `text-success` | 绿色 |
| `text-danger` | 红色 |
| `text-warning` | 黄色 |
| `text-info` | 青色 |

## JavaScript 组件

### 下拉菜单

```html
<div class="dropdown">
  <button class="btn btn-secondary dropdown-toggle" type="button" data-bs-toggle="dropdown">
    下拉菜单
  </button>
  <ul class="dropdown-menu">
    <li><a class="dropdown-item" href="#">操作</a></li>
  </ul>
</div>
```

### 标签页

```html
<ul class="nav nav-tabs" role="tablist">
  <li class="nav-item">
    <button class="nav-link active" data-bs-toggle="tab" data-bs-target="#home">首页</button>
  </li>
</ul>
```

### 工具提示

```html
<button type="button" class="btn btn-secondary" data-bs-toggle="tooltip" title="提示文本">
  提示按钮
</button>
```
