# Element UI よくある質問

## インストール問題

Node.js >= 8.0.0を確認してください。

npm cache clean --force

## 使用問題

### スタイルが適用されない

CSSをインポートしていることを確認してください：

import 'element-ui/lib/theme-chalk/index.css'

### アイコンが表示されない

アイコンフォントをインポート：

import 'element-ui/lib/theme-chalk/fonts/element-icons.ttf'

## Vueバージョン問題

Element UIはVue 2のみサポートします。
