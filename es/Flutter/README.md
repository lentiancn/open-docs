# Flutter

Flutter - Marco de Interfaz de Usuario Multiplataforma

## Descripción

Flutter es un kit de desarrollo de software de código abierto creado por Google, utilizado para crear aplicaciones bellas, compiladas de forma nativa para móvil, web y escritorio desde una única base de código.

## Características

- **Multiplataforma**: Una base de código para iOS, Android, Web, Windows, macOS, Linux
- **Recarga en Caliente**: Iteración de desarrollo rápida, preserva el estado de la aplicación
- **Componentes Ricos**: Bibliotecas de componentes Material Design y Cupertino
- **Rendimiento Nativo**: Compila a código ARM nativo
- **Interfaz Declarativa**: Paradigma moderno de desarrollo de UI
- **Código Abierto**: Licencia BSD

## Inicio Rápido

### Instalación

```bash
# Clonar Flutter
git clone https://github.com/flutter/flutter.git -b stable

# Agregar a PATH
export PATH="$PATH:$HOME/flutter/bin"

# Verificar instalación
flutter doctor
```

### Crear Proyecto

```bash
flutter create my_app
cd my_app
flutter run
```
