# Claude Code Benutzerhandbuch

Grundlegende Nutzung und Techniken.

## Grundlegende Nutzung

### Starten

- VS Code: Shortcut Cmd/Ctrl + Shift + P
- CLI: `claude` direkt im Terminal ausführen

### Gespräch

Beschreibe deine Bedürfnisse direkt im Chat-Fenster:
```
Schreibe eine Sortierfunktion für mich
Erkläre diesen Code
Wie behebe ich diesen Bug
```

## Codegenerierung

### Neuen Code generieren

Beschreibe die gewünschte Funktionalität:
```
Eine Benutzer-Login-API erstellen
Paginierungsabfrage implementieren
Unit-Tests schreiben
```

### Bestehenden Code ändern

1. Code auswählen
2. Änderungsbedarf beschreiben
3. Claude generiert den geänderten Code

## Debugging-Unterstützung

### Fehler erklären

Fehlermeldung direkt einfügen:
```
Was bedeutet dieser Fehler?
```

### Korrekturvorschläge

Claude das Problem analysieren lassen:
```
Was ist falsch an dieser Funktion?
```

## Code-Review

### Code-Review-Anfragen

```
Überprüfe diesen Code
Sicherheitsprobleme?
Leistungsoptimierungsvorschläge?
```

## Tastenkombinationen

### VS Code

| Shortcut | Funktion |
|---------|----------|
| Cmd/Ctrl + L | Chat öffnen |
| Cmd/Ctrl + Enter | Eingabe senden |

## Konfigurationsoptionen

### claude.json

Erstelle `claude.json` im Projektverzeichnis:

```json
{
  "preamble": "Du bist ein Python-Experte",
  "maxTokens": 4000
}
```

## Best Practices

1. Anforderungen klar beschreiben
2. Kontextinformationen bereitstellen
3. Code iterativ verbessern
4. Mit menschlicher Überprüfung kombinieren
