# Element Plus Руководство пользователя

Основные функции и советы по использованию Element Plus.

## Быстрый старт

### Первый пример

```vue
<template>
  <el-button type="primary" @click="handleClick">
    Нажми меня
  </el-button>
</template>

<script setup>
import { ref } from 'vue'

const handleClick = () => {
  ElMessage.success('Привет Element Plus!')
}
</script>
```

### Form (Форма)

```vue
<template>
  <el-form :model="form" :rules="rules" ref="formRef" label-width="120px">
    <el-form-item label="Имя пользователя" prop="username">
      <el-input v-model="form.username" />
    </el-form-item>
    
    <el-form-item label="Эл. почта" prop="email">
      <el-input v-model="form.email" />
    </el-form-item>
    
    <el-form-item label="Роль" prop="role">
      <el-select v-model="form.role" placeholder="Выберите роль">
        <el-option label="Администратор" value="admin" />
        <el-option label="Пользователь" value="user" />
      </el-select>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm">Отправить</el-button>
      <el-button @click="resetForm">Сбросить</el-button>
    </el-form-item>
  </el-form>
</template>

<script setup>
import { ref, reactive } from 'vue'

const formRef = ref()
const form = reactive({
  username: '',
  email: '',
  role: '',
})

const rules = {
  username: [{ required: true, message: 'Введите имя', trigger: 'blur' }],
  email: [
    { required: true, message: 'Введите email', trigger: 'blur' },
    { type: 'email', message: 'Введите корректный email', trigger: 'blur' }
  ],
  role: [{ required: true, message: 'Выберите роль', trigger: 'change' }],
}

const submitForm = async () => {
  const valid = await formRef.value.validate()
  if (valid) {
    console.log('Отправлено успешно', form)
  }
}

const resetForm = () => {
  formRef.value.resetFields()
}
</script>
```

### Message (Сообщение)

```typescript
import { ElMessage } from 'element-plus'

// Сообщение об успехе
ElMessage.success('Операция успешна')

// Предупреждение
ElMessage.warning('Предупреждение')

// Сообщение об ошибке
ElMessage.error('Операция не удалась')

// Информационное сообщение
ElMessage.info('Информация')

// Длительное отображение
ElMessage({
  message: 'Пользовательская длительность',
  duration: 10000,
})
```

```vue
<script setup>

  ElMessage.success('Загружено успешно')

const handleError = () => {
  ElMessage.error('Ошибка загрузки')
```


### Пагинация таблицы

```vue
<template>
  <el-table :data="paginatedData">
    <el-table-column prop="name" label="Имя" />
  </el-table>
  
  <el-pagination
    v-model:current-page="currentPage"
    v-model:page-size="pageSize"
    :total="total"
    :page-sizes="[10, 20, 50, 100]"
    layout="total, sizes, prev, pager, next, jumper"
    @size-change="handleSizeChange"
    @current-change="handleCurrentChange"
  />
</template>

<script setup>
import { ref, computed } from 'vue'

const currentPage = ref(1)
const pageSize = ref(10)
const tableData = ref([/* много данных */])
const total = computed(() => tableData.value.length)

const paginatedData = computed(() => {
  const start = (currentPage.value - 1) * pageSize.value
  return tableData.value.slice(start, start + pageSize.value)
})
</script>
```

## Советы по отладке

### Проверка проблем с компонентами

```bash
# Посмотреть версию Element Plus
npm list element-plus
```

### Распространённые ошибки

**Компонент не зарегистрирован**: Проверить импорт или настройку автоматического импорта

**Стили не применяются**: Проверить конфликты CSS, попробовать `:deep()`

**Ошибки типов**: Установить типы `element-plus/global`
