# Guide d'installation de jQuery

## Obtenir jQuery

### Dernière version

La version actuelle la plus récente est **jQuery 4.0.0**.

#### Version compressée pour la production (Recommandée)
- Téléchargement : [jquery-4.0.0.min.js](https://code.jquery.com/jquery-4.0.0.min.js)
- Adaptée pour la production, petite taille de fichier, chargement rapide

#### Version de développement
- Téléchargement : [jquery-4.0.0.js](https://code.jquery.com/jquery-4.0.0.js)
- Contient le code complet, facile à lire et à déboguer

#### Fichier Source Map
- Téléchargement : [jquery-4.0.0.min.map](https://code.jquery.com/jquery-4.0.0.min.map)
- Utilisé pour afficher le code source original dans les outils de développement du navigateur

#### Version Slim
La version Slim est une version plus petite qui exclut les modules Ajax et effets :
- [jquery-4.0.0.slim.min.js](https://code.jquery.com/jquery-4.0.0.slim.min.js) - compressée
- [jquery-4.0.0.slim.js](https://code.jquery.com/jquery-4.0.0.slim.js) - non compressée

Si vous n'avez pas besoin des fonctionnalités Ajax et d'animation, utiliser la version Slim peut réduire la taille du fichier d'environ 30%.

## Méthodes d'installation

### Méthode 1 : Inclusion via CDN (Recommandé)

Utiliser le CDN jQuery est la méthode la plus simple :

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
```

Version avec validation Subresource Integrity (SRI) (recommande, prévient la falsification du CDN) :

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js" 
        integrity="sha384-KJ3o2DKtIkvYIK3UENzmM7KCkRr/rE9/Qpg6aAZGJwFDMVNA/GpGFF93hXpG5KkN" 
        crossorigin="anonymous"></script>
```

Visitez [https://releases.jquery.com](https://releases.jquery.com) pour obtenir la liste complète des versions et des hashes SRI.

### Méthode 2 : Installation npm

Pour les projets Node.js, installez via npm :

```bash
npm install jquery

Après l'installation, vous trouverez dans node_modules/jquery/dist/ :
- jquery-4.0.0.min.js - compressée
- jquery-4.0.0.js - non compressée
- jquery-4.0.0.min.map - source map
```

### Méthode 3 : Installation Yarn

```bash
yarn add jquery
```

### Méthode 4 : Téléchargement local

Visitez la [page de téléchargement de jQuery](https://jquery.com/download/), faites un clic droit sur la version souhaitée et sélectionnez "Enregistrer sous..." pour enregistrer le fichier dans votre répertoire de projet local.

## Mise à jour de jQuery

### Utilisation du plugin jQuery Migrate

Lors de la mise à jour de jQuery, il est recommandé d'utiliser le plugin jQuery Migrate pour identifier les problèmes de compatibilité :

1. Mise à niveau depuis des versions antérieures vers 1.9+ ou 3.x : Utilisez [jQuery Migrate 1.4.1](https://code.jquery.com/jquery-migrate-1.4.1.min.js)

2. Mise à niveau depuis jQuery 3.x vers une version plus récente : Utilisez [jQuery Migrate 3.6.0](https://code.jquery.com/jquery-migrate-3.6.0.min.js)

Utilisation : Incluez le plugin Migrate après jQuery

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
<script src="https://code.jquery.com/jquery-migrate-3.6.0.min.js"></script>
```

Migrate affichera des avertissements de compatibilité dans la console pour vous aider à résoudre les problèmes étape par étape.
