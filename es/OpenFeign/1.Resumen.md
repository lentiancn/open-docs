# OpenFeign Resumen

> Cliente HTTP declarativo para Java

---

## Descripción del Proyecto

OpenFeign es una biblioteca Java que proporciona capacidades de cliente REST declarativo. Simplifica las llamadas a la API HTTP permitiendo a los desarrolladores definir interfaces de servicio HTTP usando anotaciones.

---

## Características Principales

- **API Declarativa**: Definir interfaces HTTP con anotaciones
- **Balanceo de Carga**: Soporte integrado para balanceo de carga del cliente
- **Descubrimiento de Servicios**: Integración con Eureka
- **Compresión**: Soporte GZIP para transferencia eficiente de datos
- **Registro**: Registro detallado de solicitudes y respuestas HTTP
- **Manejo de Errores**: Manejo de errores personalizado con mecanismos de fallback
- **Interceptores**: Soporte para interceptores personalizados

---

## Casos de Uso

- Comunicación de microservicios
- Desarrollo de cliente REST API
- Integración de API externa
- Llamadas servicio a servicio
