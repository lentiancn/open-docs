# OpenFeign 安装指南

## 项目依赖配置

### Maven 依赖

```xml
<!-- Spring Cloud OpenFeign Starter -->
<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-openfeign</artifactId>
</dependency>

<!-- 如果需要负载均衡支持 -->
<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-loadbalancer</artifactId>
</dependency>
```

### Gradle 依赖

```gradle
implementation 'org.springframework.cloud:spring-cloud-starter-openfeign'
implementation 'org.springframework.cloud:spring-cloud-starter-loadbalancer'
```

### 版本管理

在 `pom.xml` 中使用 Spring Cloud BOM 管理版本：

```xml
<dependencyManagement>
    <dependencies>
        <dependency>
            <groupId>org.springframework.cloud</groupId>
            <artifactId>spring-cloud-dependencies</artifactId>
            <version>2022.0.4</version>
            <type>pom</type>
            <scope>import</scope>
        </dependency>
    </dependencies>
</dependencyManagement>
```

## 基础配置

### 启用 Feign 客户端

在主应用类上添加 `@EnableFeignClients` 注解：

```java
@SpringBootApplication
@EnableFeignClients
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
```

### 指定客户端扫描包

如果 Feign 客户端接口不在主应用类的包路径下，需要指定扫描包：

```java
@SpringBootApplication
@EnableFeignClients(basePackages = "com.example.clients")
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
```

或者显式列出客户端：

```java
@SpringBootApplication
@EnableFeignClients(clients = {UserServiceClient.class, OrderServiceClient.class})
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
```

## 创建 Feign 客户端

### 基本用法

```java
@FeignClient(name = "user-service")
public interface UserService {
    
    @GetMapping("/users/{id}")
    User getUser(@PathVariable("id") Long id);
    
    @GetMapping("/users")
    List<User> getAllUsers();
    
    @PostMapping("/users")
    User createUser(@RequestBody User user);
    
    @PutMapping("/users/{id}")
    User updateUser(@PathVariable("id") Long id, @RequestBody User user);
    
    @DeleteMapping("/users/{id}")
    void deleteUser(@PathVariable("id") Long id);
}
```

### 使用 URL 属性

如果不需要服务发现，可以直接指定 URL：

```java
@FeignClient(name = "external-api", url = "https://api.example.com")
public interface ExternalApiClient {
    @GetMapping("/data")
    ApiResponse getData();
}
```

### 支持的注解

OpenFeign 支持以下 Spring MVC 注解：

- **HTTP 方法注解**: `@GetMapping`, `@PostMapping`, `@PutMapping`, `@DeleteMapping`, `@PatchMapping`
- **路径变量**: `@PathVariable`
- **请求参数**: `@RequestParam`
- **请求体**: `@RequestBody`
- **请求头**: `@RequestHeader`
- **Cookie**: `@CookieValue`

### 复杂参数示例

```java
@FeignClient(name = "order-service")
public interface OrderService {
    
    // 路径变量 + 请求参数
    @GetMapping("/orders/{userId}")
    List<Order> getOrdersByUser(
        @PathVariable("userId") Long userId,
        @RequestParam("status") String status,
        @RequestParam(value = "page", defaultValue = "0") int page
    );
    
    // 请求体 + 请求头
    @PostMapping("/orders")
    @Headers("X-API-Version: 2.0")
    Order createOrder(
        @RequestBody CreateOrderRequest request,
        @RequestHeader("Authorization") String token
    );
    
    // 查询参数 Map
    @GetMapping("/orders/search")
    List<Order> searchOrders(@SpringQueryMap SearchCriteria criteria);
}
```

## 使用 Feign 客户端

### 在 Service 中注入

```java
@Service
public class OrderProcessingService {
    
    private final UserService userService;
    private final OrderService orderService;
    
    public OrderProcessingService(UserService userService, OrderService orderService) {
        this.userService = userService;
        this.orderService = orderService;
    }
    
    public Order processOrder(Long userId, CreateOrderRequest request) {
        // 验证用户
        User user = userService.getUser(userId);
        if (user == null) {
            throw new UserNotFoundException("User not found: " + userId);
        }
        
        // 创建订单
        return orderService.createOrder(request, "Bearer " + generateToken());
    }
    
    private String generateToken() {
        // 生成认证令牌
        return "token";
    }
}
```

### 在 Controller 中使用

```java
@RestController
@RequestMapping("/api/v1")
public class OrderController {
    
    private final OrderService orderService;
    
    public OrderController(OrderService orderService) {
        this.orderService = orderService;
    }
    
    @GetMapping("/orders/{id}")
    public ResponseEntity<Order> getOrder(@PathVariable Long id) {
        Order order = orderService.getOrder(id);
        return ResponseEntity.ok(order);
    }
    
    @PostMapping("/orders")
    public ResponseEntity<Order> createOrder(@RequestBody CreateOrderRequest request) {
        Order order = orderService.createOrder(request);
        return ResponseEntity.status(HttpStatus.CREATED).body(order);
    }
}
```

## 配置属性

### 基本配置

在 `application.yml` 中配置：

```yaml
spring:
  cloud:
    openfeign:
      client:
        config:
          default:
            connect-timeout: 5000
            read-timeout: 5000
            logger-level: basic
          user-service:
            connect-timeout: 3000
            read-timeout: 10000
            logger-level: full
```

### 日志级别配置

```yaml
logging:
  level:
    com.example.clients.UserService: DEBUG
```

支持的日志级别：
- **NONE**: 不记录日志（默认）
- **BASIC**: 只记录请求方法、URL 和响应状态码、执行时间
- **HEADERS**: 记录基本信息以及请求和响应头
- **FULL**: 记录头、正文和元数据（请求和响应）

## 错误处理

### 自定义错误解码器

```java
@Component
public class CustomErrorDecoder implements ErrorDecoder {
    
    private final ErrorDecoder defaultErrorDecoder = new Default();
    
    @Override
    public Exception decode(String methodKey, Response response) {
        if (response.status() >= 400 && response.status() < 500) {
            return new ClientException("Client error: " + response.status());
        } else if (response.status() >= 500) {
            return new ServerException("Server error: " + response.status());
        }
        return defaultErrorDecoder.decode(methodKey, response);
    }
}
```

### 在配置中注册

```java
@Configuration
public class FeignConfig {
    
    @Bean
    public ErrorDecoder errorDecoder() {
        return new CustomErrorDecoder();
    }
}
```

## 测试 Feign 客户端

### 单元测试

```java
@ExtendWith(MockitoExtension.class)
class OrderProcessingServiceTest {
    
    @Mock
    private UserService userService;
    
    @Mock
    private OrderService orderService;
    
    @InjectMocks
    private OrderProcessingService orderProcessingService;
    
    @Test
    void shouldProcessOrderSuccessfully() {
        // Given
        Long userId = 1L;
        User user = new User(userId, "John Doe");
        CreateOrderRequest request = new CreateOrderRequest();
        Order expectedOrder = new Order(1L, userId);
        
        when(userService.getUser(userId)).thenReturn(user);
        when(orderService.createOrder(any(), anyString())).thenReturn(expectedOrder);
        
        // When
        Order result = orderProcessingService.processOrder(userId, request);
        
        // Then
        assertThat(result).isEqualTo(expectedOrder);
        verify(userService).getUser(userId);
        verify(orderService).createOrder(request, anyString());
    }
}
```

### 集成测试

```java
@SpringBootTest
@AutoConfigureTestDatabase
class FeignIntegrationTest {
    
    @Autowired
    private UserService userService;
    
    @Test
    void shouldGetUserFromRemoteService() {
        // When
        User user = userService.getUser(1L);
        
        // Then
        assertThat(user).isNotNull();
        assertThat(user.getId()).isEqualTo(1L);
    }
}
```
