# OpenClaw Guide d'Installation

> Guide complet pour configurer et installer OpenClaw

---

## Configuration Requise

| Exigence | Spécification |
|-----------|---------------|
| Système d'Exploitation | Windows 10+, macOS, Linux |
| Node.js | Version 18+ |
| Mémoire | Au moins 4 Go RAM |
| Stockage | ~500 Mo |

---

## Installation

### Prérequis

Assurez-vous que Node.js est installé :

```bash
node --version
npm --version
```

### Installation Étape par Étape

#### 1. Installer via npm

```bash
npm install -g openclaw
```

#### 2. Démarrer le Service Gateway

Le service Gateway est le cœur d'OpenClaw :

```bash
# Vérifier le statut
openclaw gateway status

# Démarrer Gateway
openclaw gateway start

# Redémarrer Gateway
openclaw gateway restart

# Arrêter Gateway
openclaw gateway stop
```

#### 3. Configuration Initiale

Après l'installation, configurez les fichiers :

```bash
# Créer le répertoire de configuration
mkdir -p ~/.openclaw

# Copier la configuration par défaut
cp /chemin/vers/openclaw/config/* ~/.openclaw/
```

## Configuration

### Fichiers de Configuration Principaux

| Fichier | Description |
|---------|-------------|
| AGENTS.md | Configuration de l'espace de travail |
| SOUL.md | Paramètres de personnalité |
| TOOLS.md | Configuration des outils |
| USER.md | Préférences utilisateur |

### Variables d'Environnement

```bash
# Optionnel : Répertoire de travail personnalisé
OPENCLAW_WORKSPACE=/chemin/vers/workspace

# Optionnel : Port Gateway
OPENCLAW_PORT=3000
```

---

## Vérification

### Tester l'Installation

```bash
# Afficher les informations de version
openclaw --version

# Vérifier le statut du Gateway
openclaw gateway status

# Afficher les commandes disponibles
openclaw --help
```

### Démarrage Rapide

```bash
# Démarrer une session interactive
openclaw chat

# Exécuter une seule tâche
openclaw exec "dire bonjour"
```

---

## Dépannage

### Le Gateway ne démarre pas ?

**Solution** :
1. Vérifiez que Node.js est correctement installé
2. Vérifiez les permissions
3. Consultez les logs : `openclaw gateway logs`

### Le port est déjà utilisé ?

**Solution** :
```bash
# Changer le port
openclaw gateway start --port 3001
```

### Module non trouvé ?

**Solution** :
```bash
# Réinstaller
npm uninstall -g openclaw
npm install -g openclaw
```

---

## Désinstallation

```bash
npm uninstall -g openclaw

# Supprimer le service Gateway
openclaw gateway stop
rm -rf ~/.openclaw
```
