# Element Plus Manual de usuario

Funciones principales y consejos de uso para Element Plus.

## Inicio rápido

### Primer ejemplo

```vue
<template>
  <el-button type="primary" @click="handleClick">
    Haz clic en mí
  </el-button>
</template>

<script setup>
import { ref } from 'vue'

const handleClick = () => {
  ElMessage.success('¡Hola Element Plus!')
}
</script>
```

### Form (Formulario)

```vue
<template>
  <el-form :model="form" :rules="rules" ref="formRef" label-width="120px">
    <el-form-item label="Nombre de usuario" prop="username">
      <el-input v-model="form.username" />
    </el-form-item>
    
    <el-form-item label="Correo electrónico" prop="email">
      <el-input v-model="form.email" />
    </el-form-item>
    
    <el-form-item label="Rol" prop="role">
      <el-select v-model="form.role" placeholder="Seleccionar rol">
        <el-option label="Administrador" value="admin" />
        <el-option label="Usuario" value="user" />
      </el-select>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm">Enviar</el-button>
      <el-button @click="resetForm">Restablecer</el-button>
    </el-form-item>
  </el-form>
</template>

<script setup>
import { ref, reactive } from 'vue'

const formRef = ref()
const form = reactive({
  username: '',
  email: '',
  role: '',
})

const rules = {
  username: [{ required: true, message: 'Por favor ingrese el nombre', trigger: 'blur' }],
  email: [
    { required: true, message: 'Por favor ingrese el correo', trigger: 'blur' },
    { type: 'email', message: 'Por favor ingrese un correo válido', trigger: 'blur' }
  ],
  role: [{ required: true, message: 'Por favor seleccione un rol', trigger: 'change' }],
}

const submitForm = async () => {
  const valid = await formRef.value.validate()
  if (valid) {
    console.log('Enviado con éxito', form)
  }
}

const resetForm = () => {
  formRef.value.resetFields()
}
</script>
```

### Message (Mensaje)

```typescript
import { ElMessage } from 'element-plus'

// Mensaje de éxito
ElMessage.success('Operación exitosa')

// Mensaje de advertencia
ElMessage.warning('Advertencia')

// Mensaje de error
ElMessage.error('Operación fallida')

// Mensaje de información
ElMessage.info('Información')

// Mostrar por más tiempo
ElMessage({
  message: 'Duración personalizada',
  duration: 10000,

```

```vue
<script setup>

  ElMessage.success('Subido con éxito')

const handleError = () => {

}
</script>
```

## Mejores prácticas

### Validación de formulario

```typescript
const rules = {
  name: [
    { required: true, message: 'Por favor ingrese el nombre', trigger: 'blur' },
    { min: 2, max: 20, message: 'Longitud entre 2 y 20 caracteres', trigger: 'blur' }
  ],
  email: [
    { required: true, message: 'Por favor ingrese el correo', trigger: 'blur' },
    { type: 'email', message: 'Por favor ingrese un correo válido', trigger: 'blur' }
  ],
  phone: [
    { pattern: /^1[3-9]\d{9}$/, message: 'Por favor ingrese un número válido', trigger: 'blur' }
  ]
}
```

### Paginación de tabla

```vue
<template>
  <el-table :data="paginatedData">
    <el-table-column prop="name" label="Nombre" />
  </el-table>
  
  <el-pagination
    v-model:current-page="currentPage"
    v-model:page-size="pageSize"
    :total="total"
    :page-sizes="[10, 20, 50, 100]"
    layout="total, sizes, prev, pager, next, jumper"
    @size-change="handleSizeChange"
    @current-change="handleCurrentChange"
  />
</template>

<script setup>
import { ref, computed } from 'vue'

const currentPage = ref(1)
const pageSize = ref(10)
const tableData = ref([/* muchos datos */])
const total = computed(() => tableData.value.length)

const paginatedData = computed(() => {
  const start = (currentPage.value - 1) * pageSize.value
  return tableData.value.slice(start, start + pageSize.value)
})
</script>
```

### Internacionalización

```typescript
import { ref } from 'vue'
import es from 'element-plus/es/locale/lang/es'
import en from 'element-plus/es/locale/lang/en'

const locale = ref(es)

const switchLanguage = (lang) => {
  locale.value = lang === 'es' ? es : en
}
```

## Consejos de depuración

### Verificar problemas de componentes

```bash
# Ver versión de Element Plus
npm list element-plus
```

### Errores comunes

**Componente no registrado**: Verificar importación o configuración de importación automática

**Estilos no aplicados**: Verificar conflictos CSS, intentar `:deep()`

**Errores de tipo**: Instalar tipos `element-plus/global`
