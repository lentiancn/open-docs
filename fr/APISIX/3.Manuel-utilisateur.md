# APISIX Manuel Utilisateur

Fonctionnalités principales et conseils d'utilisation d'APISIX.

## Concepts de Base

### Terminologie

- **Route**: Définit les règles de correspondance des requêtes
- **Upstream**: Services backend réels
- **Service**: Collection de routes
- **Plugin**: Logique de traitement des requêtes
- **Consumer**: Consommateur d'API

### Flux des Requêtes

1. Client envoie une requête à APISIX
2. APISIX fait correspondre avec les règles de routage
3. Exécute la logique des plugins
4. Transmet au service upstream
5. Retourne la réponse au client

## Gestion des Routes

### Créer une Route

```bash
curl -X PUT http://127.0.0.1:9180/apisix/admin/routes/1 \
  -H "X-API-KEY: 123456" \
  -d '{
    "id": "1",
    "uri": "/api/users/*",
    "name": "user-api",
    "upstream": {
      "type": "roundrobin",
      "nodes": {"backend1.example.com:8080": 1}
    }
  }'
```

## Gestion Upstream

### Créer un Upstream

```bash
curl -X PUT http://127.0.0.1:9180/apisix/admin/upstreams/1 \
  -H "X-API-KEY: 123456" \
  -d '{
    "type": "roundrobin",
    "nodes": {"host1:8080": 1, "host2:8080": 1}
  }'
```

## Plugins

### Plugins Courants

**Contrôle de Trafic:**
- limit-req: Limitation de débit
- limit-conn: Limite de connexions

**Authentification:**
- jwt-auth: Authentification JWT
- key-auth: Authentification par clé
- basic-auth: Authentification basique

### Utiliser le Plugin limit-req

```bash
curl -X PUT http://127.0.0.1:9180/apisix/admin/routes/1 \
  -H "X-API-KEY: 123456" \
  -d '{
    "uri": "/api/*",
    "plugins": {
      "limit-req": {
        "rate": 100,
        "burst": 50,
        "rejected_code": 503
      }
    }
  }'
```

### Utiliser JWT

**1. Créer un Consumer:**

```bash
curl -X PUT http://127.0.0.1:9180/apisix/admin/consumers \
  -H "X-API-KEY: 123456" \
  -d '{
    "username": "john",
    "plugins": {
      "jwt-auth": {"key": "user-key", "secret": "my-secret"}
    }
  }'
```

**2. Activer sur la Route:**

```bash
curl -X PUT http://127.0.0.1:9180/apisix/admin/routes/1 \
  -H "X-API-KEY: 123456" \
  -d '{"uri":"/api/*","plugins":{"jwt-auth":{}}}'
```
