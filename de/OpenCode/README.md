# OpenCode

Open-Source-KI-Coding-Agent.

## Übersicht

OpenCode ist ein Open-Source-KI-Coding-Agent, der als Terminal-Interface, Desktop-App oder IDE-Erweiterung verfügbar ist. Mit LSP-Aktivierung, Multi-Sitzung und Unterstützung für 75+ LLM-Anbieter.

### Hauptfunktionen

- **LSP aktiviert** - Lädt automatisch die richtigen LSPs
- **Multi-Sitzung** - Starte mehrere Agenten parallel
- **Beliebiges Modell** - 75+ LLM-Anbieter
- **Datenschutz zuerst** - Kein Code oder Kontextdaten gespeichert

### Statistiken

- GitHub Sterne: 100.000+
- Mitwirkende: 700+
- Monatlich aktive Entwickler: 2.500.000+

## Schnellstart

### Installieren

```bash
curl -fsSL https://opencode.ai/install | bash
```

### Konfigurieren

```bash
opencode
/connect
```

Besuche opencode.ai/auth um API-Schlüssel zu erhalten.

### Initialisieren

```bash
/init
```
