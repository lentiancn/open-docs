# Docker 사용 가이드

기본 Docker 명령어입니다.

---

## 기본 개념

| 개념 | 설명 |
|------|------|
| 이미지 | 컨테이너를 만들기 위한 템플릿 |
| 컨테이너 | 실행 중인 이미지의 인스턴스 |
| 레지스트리 | 이미지를 저장하는 곳 |

---

## 이미지 명령어

```bash
docker pull nginx      # 다운로드
docker images          # 목록
docker rmi nginx       # 삭제
docker build -t 이름 . # 빌드
```

---

## 컨테이너 명령어

```bash
docker run -d nginx              # 백그라운드
docker run -d -p 8080:80 nginx   # 포트
docker run -d -v /data:/data nginx # 볼륨

docker ps             # 목록
docker stop <id>     # 중지
docker start <id>    # 시작
docker rm <id>       # 삭제
docker logs <id>     # 로그
docker exec -it <id> bash # 셸
```

---

## Docker Compose

```bash
docker compose up -d    # 시작
docker compose down    # 중지
```
