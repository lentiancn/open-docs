# Yarn ユーザーマニュアル

## 基本的な使用

### プロジェクト初期化

```bash
yarn init -2
```

### 依存関係インストール

```bash
yarn install
```

### 依存関係追加

```bash
yarn add lodash
yarn add -D typescript
```

### 依存関係削除

```bash
yarn remove lodash
```

### 依存関係更新

```bash
yarn up lodash
yarn up --all
```

### スクリプト実行

```bash
yarn dev
yarn build
yarn test
```

---

## ワークスペース

### 設定

```json
{
  "workspaces": ["packages/*"]
}
```

### コマンド

```bash
yarn workspace @my-org/core add lodash
yarn workspaces foreach -pt run build
```

---

## キャッシュ管理

```bash
yarn cache dir
yarn cache clean
```

---

## 一般的なコマンド

| コマンド | 説明 |
|----------|------|
| yarn init | プロジェクト初期化 |
| yarn install | 依存関係インストール |
| yarn add | 依存関係追加 |
| yarn remove | 依存関係削除 |
| yarn up | 依存関係更新 |
| yarn list | 依存関係一覧 |

---

## ベストプラクティス

1. yarn.lockを常にコミット
2. CIで--immutableを使用
3. 依存関係を定期的に更新
