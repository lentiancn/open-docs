# JDK

Kit de Desarrollo Java

## Introducción

JDK (Java Development Kit) es el kit de desarrollo de software para el lenguaje de programación Java, formando el núcleo de todo el desarrollo en Java. JDK incluye el entorno de ejecución de Java (JRE), el compilador de Java (javac), la máquina virtual de Java (JVM) y extensas bibliotecas de clases y herramientas de Java.

## Versiones principales

- **JDK 25** - Versión más reciente con nuevas características y mejoras de rendimiento
- **JDK 21** - Versión de soporte a largo plazo más reciente (LTS)
- **JDK 17** - Versión LTS madura, ampliamente adoptada
- **JDK 11** - Versión LTS, marcando la división de características pagadas de Java

## Componentes principales

- **javac** - Compilador de Java, compila código fuente a bytecode
- **java** - Comando de ejecución de programas Java
- **jvm** - Máquina virtual de Java, ejecuta bytecode
- **jar** - Herramienta de archivos Java
- **javadoc** - Herramienta de generación de documentación API
- **jdb** - Depurador de Java

## Diferencias entre versiones

Las diferentes versiones principales de JDK tienen diferencias significativas en instalación, uso y características:

- **JDK 8**: Última versión de Oracle JDK completamente gratuita para uso comercial
- **JDK 11**: Introdujo var, nuevo cliente HTTP, ZGC (experimental)
- **JDK 17**: Última versión de Oracle JDK gratuita bajo NFTC
- **JDK 21**: Hilos virtuales, coincidencia de patrones, registros (estable)
- **JDK 25**: Características más recientes, ciclo de soporte más corto
