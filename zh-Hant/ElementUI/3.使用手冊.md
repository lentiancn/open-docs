# Element UI 使用手冊

## 快速開始

### 基礎使用

```vue
<el-button>按鈕</el-button>
<el-button type="primary">主要</el-button>
```

## 常用組件

### 按鈕

```vue
<el-button>默認</el-button>
<el-button type="primary">主要</el-button>
<el-button size="small">小型</el-button>
```

### 輸入框

```vue
<el-input v-model="input" placeholder="輸入" />
```

### 選擇器

```vue
<el-select v-model="value" placeholder="選擇">
  <el-option label="選項1" value="1" />
</el-select>
```

### 表格

```vue
<el-table :data="tableData">
  <el-table-column prop="name" label="姓名" />
</el-table>
```

### 對話框

```vue
<el-dialog :visible.sync="visible" title="標題">
  <span>內容</span>
</el-dialog>
```

更多信息請訪問 Element UI 官網。
