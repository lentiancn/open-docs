# FantasyTalking Manuel

## Synthèse Vocale

La synthèse vocale est la fonction principale de FantasyTalking, convertissant le texte en parole naturelle et fluide:

```javascript
// Texte simple vers parole
const audio = await client.speak('Bonjour, bienvenue chez FantasyTalking');

// Spécifier le style vocal
const audio = await client.speak('Le temps est magnifique aujourd\'hui', {
  voice: {
    style: 'happy',
    language: 'fr-FR'
  }
});

// Spécifier la vitesse et la hauteur
const audio = await client.speak('Veuillez patienter un instant', {
  voice: {
    speed: 1.2,
    pitch: 0.9
  }
});
```

## Événements Vocaux

Pendant la lecture vocale, vous pouvez écouter différents événements:

```javascript
client.on('speech-start', (event) => {
  console.log('Lecture vocale démarrée');
});

client.on('speech-end', (event) => {
  console.log('Lecture vocale terminée');
});

client.on('error', (error) => {
  console.error('Erreur:', error);
});
```

## Dialogue

FantasyTalking prend en charge les conversations multi-tours:

```javascript
const session = client.createSession({
  systemPrompt: 'Vous êtes un assistant amical',
  language: 'fr-FR'
});

const response1 = await session.chat('Comment ça va?');
console.log(response1.text);
```

## Fonctionnalités Avancées

### Synthèse en Streaming

Pour les scénarios nécessitant un retour en temps réel:

```javascript
const stream = await client.speakStream('Ceci est un long texte');
stream.on('data', (chunk) => {
  audioPlayer.play(chunk);
});
```

## Meilleures Pratiques

1. **Réutilisation des connexions**: Évitez de créer des instances fréquemment
2. **Traitement par lots**: Utilisez la synthèse par lots pour gros volumes
3. **Mise en cache**: Utilisez le cache pour éviter les synthèses doublons
