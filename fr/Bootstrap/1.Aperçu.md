# Aperçu de Bootstrap

> Framework CSS frontend populaire

---

## Introduction au projet

Bootstrap est un framework CSS open-source développé et maintenu par Twitter, publié pour la première fois en 2011. Bootstrap fournit des composants préconçus riches et un système de grille responsive pour construire rapidement des sites web et applications web modernes et responsives.

---

## Informations de version

| Version | Date de sortie | Statut |
|---------|----------------|--------|
| Bootstrap 5.3 | Mai 2023 | Stable actuelle |
| Bootstrap 5.2 | Septembre 2022 | Stable |
| Bootstrap 5.1 | Août 2021 | Stable |
| Bootstrap 4.x | Janvier 2018 | Non supporté |

---

## Caractéristiques principales

- **Design responsive**: Mises en page responsives intégrées
- **Composants préconçus**: Boutons, navigation, formulaires, cartes, etc.
- **Système de grille puissant**: Grille responsive à 12 colonnes
- **Options de personnalisation**: Variables Sass pour une personnalisation facile
- **Plugins JavaScript**: Menus déroulants, modales, etc.
- **Bibliothèque d'icônes**: Bootstrap Icons intégré

---

## Cas d'utilisation

- Développement de sites web responsives
- Tableaux de bord administrateur
- Prototypage rapide
- Projets mobile-first
