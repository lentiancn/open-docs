# Guia de Uso de jQuery

Guia para usar jQuery.

---

## Basico

```javascript
$('#myId')
$(document).ready(function() {})
```

---

## Selectores

```javascript
$('#myId')
$('.myClass')
$('div')
```

---

## Eventos

```javascript
$('#myButton').click(function() {})
```

---

## AJAX

```javascript
$.get('url', function(data) {})
$.post('url', data, function(data) {})
```
