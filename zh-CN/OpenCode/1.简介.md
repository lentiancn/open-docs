# OpenCode 简介

开源 AI 编程助手。

## 什么是 OpenCode？

OpenCode 是一个开源 AI 编程助手，可作为终端界面、桌面应用或 IDE 扩展使用。

### 主要特性

- **LSP 自动启用** - 自动加载正确的 LSP
- **多会话** - 可在同一个项目中并行启动多个代理
- **分享链接** - 可分享任何会话链接供他人参考或调试
- **GitHub Copilot** - 登录 GitHub 使用 Copilot 账户
- **ChatGPT Plus/Pro** - 登录 OpenAI 使用 ChatGPT 账户
- **任意模型** - 支持 75+ LLM 提供商，包括本地模型
- **任意编辑器** - 支持终端、桌面应用和 IDE 扩展
- **隐私优先** - 不存储任何代码或上下文数据
