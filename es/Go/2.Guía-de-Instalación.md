# Guía de Instalación de Go

> Configuración completa del entorno

---

## Requisitos

| Entorno | Requisito |
|---------|-----------|
| SO | Linux, macOS, Windows |
| Arquitectura | amd64, arm64, 386 |

---

## Instalación

### Windows

```powershell
choco install golang
```

### macOS

```bash
brew install go
```

### Linux

```bash
sudo apt install golang-go
```

---

## Configuración

### Proxy (China)

```bash
go env -w GOPROXY=https://goproxy.cn,direct
```

### Verificar

```bash
go version
```
