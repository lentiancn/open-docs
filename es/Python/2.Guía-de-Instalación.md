# Guía de Instalación de Python

> Configuración completa

---

## Instalación

### Windows

```powershell
choco install python
```

### macOS

```bash
brew install python@3.12
```

### Linux

```bash
sudo apt install python3 python3-pip
```

---

## Paquetes

```bash
pip install requests
python -m venv mi_entorno
```

---

## Verificar

```bash
python --version
```
