## システム要件

### ハードウェア要件
- **プロセッサ**：デュアルファコア以上
- **メモリ**：最低 4GB（8GB を推奨）
- **ディスク容量**：少なくとも 3GB の空き容量

### ソフトウェア要件

#### Windows
- Windows 10 以降
- Node.js 18.19 以降（LTS バージョンを推奨）
- npm 9.x 以降（Node.js に付属）
- Git バージョン管理ツール

#### macOS
- macOS 10.15（Catalina）以降
- Node.js 18.19 以降（LTS バージョンを推奨）
- npm 9.x 以降
- Git（Xcode コマンドラインツールに付属）

#### Linux
- Ubuntu 20.04 LTS 以降 / Debian 系ディストリビューション
- Fedora 36 以降 / RHEL 系ディストリビューション
- Node.js 18.19 以降（LTS バージョンを推奨）
- npm 9.x 以降
- Git

## 前提条件

### 1. Node.js のインストール

Node.js は Angular を実行するために必要なランタイム環境です。

#### Windows へのインストール

**方法 A：Node.js インストーラー使用**
1. https://nodejs.org/ にアクセス
2. **LTS** バージョンをダウンロード
3. インストーラー（.msi ファイル）を実行
4. インストールウィザードの手順に従う
5. コマンドプロンプトで確認：
   ```
   node --version
   npm --version
   ```

**方法 B：Chocolatey 使用**
```powershell
# Node.js のインストール
choco install nodejs-lts -y
```

**方法 C：nvm-windows 使用（推奨）**
```powershell
# Node.js LTS のインストールと使用
nvm install 18.19.0
nvm use 18.19.0
```

#### macOS へのインストール

**方法 A：Homebrew 使用（推奨）**
```bash
# Homebrew のインストール（未安装の場合）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Node.js LTS のインストール
brew install node@18
```

**方法 B：nvm 使用**
```bash
# nvm のインストール
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Node.js LTS のインストール
nvm install 18.19.0
nvm use 18.19.0
```

#### Linux へのインストール

**Ubuntu/Debian**
```bash
# パッケージリストの更新
sudo apt update

# curl と build-essential のインストール
sudo apt install curl build-essential

# Node.js リポジトリの追加
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -

# Node.js のインストール
sudo apt install nodejs
```

**Fedora/RHEL**
```bash
# Node.js リポジトリの追加
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -

# Node.js のインストール
sudo dnf install nodejs
```

### 2. Git のインストール

Git は Angular CLI のバージョン管理機能に必要です。

#### Windows
- https://git-scm.com/ からダウンロード
- インストール時に「Use Git from the Windows Command Prompt」を選択

#### macOS
```bash
# Homebrew 使用
brew install git

# または Xcode コマンドラインツールのインストール
xcode-select --install
```

#### Linux
```bash
# Ubuntu/Debian
sudo apt install git

# Fedora
sudo dnf install git
```

## Windows へのインストール

### ステップバイステップガイド

1. **Node.js のインストール**
   - https://nodejs.org/ から Node.js LTS をダウンロード
   - インストーラーを実行

2. **Git のインストール**（未安装の場合）
   - https://git-scm.com/ からダウンロード

3. **Angular CLI のインストール**
   ```bash
   npm install -g @angular/cli
   ```

4. **インストール確認**
   ```bash
   ng version
   ```

5. **最初のプロジェクトの作成**
   ```bash
   ng new my-first-app
   cd my-first-app
   ng serve
   ```

### Visual Studio Code の使用（推奨）

1. https://code.visualstudio.com/ から VS Code をダウンロード
2. Angular 拡張機能をインストール：
   - Angular Language Service
   - Angular Snippets

## Linux へのインストール

### Ubuntu/Debian

```bash
# システムの更新
sudo apt update && sudo apt upgrade -y

# 前提条件のインストール
sudo apt install curl build-essential git

# Node.js リポジトリの追加
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -

# Node.js のインストール
sudo apt install nodejs

# Angular CLI のインストール
sudo npm install -g @angular/cli
```

### Fedora

```bash
# 前提条件のインストール
sudo dnf install curl git

# Node.js リポジトリの追加
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -

# Node.js のインストール
sudo dnf install nodejs

# Angular CLI のインストール
sudo npm install -g @angular/cli
```

## トラブルシューティング

### 一般的な問題と解決策

#### 1. npm 権限エラー (EACCES)

**解決策：**
```bash
# nvm を使用
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
```

#### 2. Node.js バージョンの不一致

**解決策：**
```bash
# nvm でバージョンを切り替え
nvm install 18.19.0
nvm use 18.19.0
```

#### 3. Angular CLI が見つからない

**解決策：**
```bash
# Angular CLI の再インストール
npm uninstall -g @angular/cli
npm install -g @angular/cli
```

#### 4. ポート 4200 が使用中

**解決策：**
```bash
# 別のポートを使用
ng serve --port 4201
```

#### 5. Angular バージョンの互換性

| Angular バージョン | Node.js バージョン | TypeScript バージョン |
|-------------|-------------|----------------|
| Angular 17  | 18.x, 20.x  | ~5.2.0         |
| Angular 16  | 16.x, 18.x  | ~5.0.0         |
| Angular 15  | 16.x, 18.x  | ~4.9.0         |
| Angular 14  | 14.x, 16.x  | ~4.7.0         |
