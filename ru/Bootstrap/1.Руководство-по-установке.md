# Руководство по установке Bootstrap

Это руководство описывает установку Bootstrap на всех основных операционных системах.

## Методы установки

### Использование CDN (Рекомендуется)

`html
<!DOCTYPE html>
<html lang=""ru"">
<head>
    <link href=""https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css"" rel=""stylesheet"">
</head>
<body>
    <h1>Hello, Bootstrap!</h1>
    <script src=""https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js""></script>
</body>
</html>
`

### Использование npm

`ash
npm install bootstrap
`

### Версии

| Версия | Статус |
|--------|--------|
| Bootstrap 5.3 | Текущая |
| Bootstrap 5.2 | Поддерживается |
