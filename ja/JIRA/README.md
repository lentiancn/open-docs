# Jira

JiraはAtlassian社が開発したプロジェクト管理および課題追跡ソフトウェアであり，全球30万社以上の企業にプロジェクト管理ソリューションを提供しています。

## クイックスタート

Atlassian公式ウェブサイトでアカウントを登録し，チームに最適なプランを選択すると，すぐにJiraサイトを作成できます。

## 主要機能

プロジェクト管理と課題追跡，アジャイル開発サポート（Scrum，カンバン），チームコラボレーションとコミュニケーション，AIアシスタントRovo統合，丰富的なサードパーティ統合，強力な自動化ルール。
