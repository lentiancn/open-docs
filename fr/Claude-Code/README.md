# Claude Code

[Claude Code](https://code.claude.com/) est un assistant de programmation intelligent développé par Anthropic.

## Vue d'Ensemble

Claude Code est un assistant de programmation alimenté par l'IA disponible dans le terminal, IDE, application de bureau et navigateur.

## Fonctionnalités

- **Automatiser les tâches**: Tests, corrections lint, conflits de fusion
- **Développement**: Décrivez vos besoins; Claude Code implémente
- **Intégration Git**: Commits, branches, Pull Requests
- **Intégration MCP**: Connexion aux outils externes
- **Personnalisation**: CLAUDE.md, commandes personnalisées

## Démarrage Rapide

### Installation

**macOS, Linux, WSL:**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell:**
```powershell
irm https://claude.ai/install.ps1 | iex
```

### Lancement

```bash
cd your-project
claude
```
