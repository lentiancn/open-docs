# Angular 설치 가이드

> 설정

---

## 필요 환경

| 환경 | 요구사항 |
|------|---------|
| Node.js | 18.10.0+ |
| npm | 9.0.0+ |

---

## 설치

```bash
npm install -g @angular/cli
ng new my-app
ng serve
```

---

## 명령어

| 명령어 | 설명 |
|--------|------|
| ng new | 새 프로젝트 |
| ng serve | 개발 서버 |
| ng build | 빌드 |
