# HeyGen 安裝指南

HeyGen 是一個 AI 視頻生成平台，允許用戶從文本或音頻輸入創建逼真的 AI 頭像並生成說話頭像視頻。

## 平台概述

HeyGen 主要是一個基於雲的 SaaS 平台。與開源項目不同，HeyGen 不需要本地安裝 - 用戶通過網頁界面或 API 訪問。

### 訪問方式

| 方法 | 描述 | 要求 |
|------|------|------|
| 網頁界面 | 基於瀏覽器的視頻創建 | 帳戶 + 互聯網 |
| API | 編程訪問 | API 密鑰 |
| SDK | 本地開發 | 開發者帳戶 |

## 版本概述

HeyGen 是雲服務，非本地安裝：

| 組件 | 版本 | 說明 |
|------|------|------|
| HeyGen 網頁 | 最新 | 始終保持最新 |
| HeyGen API | v1/v2 | REST API |
| HeyGen SDK | 最新 | 開發者使用 |

**注意：** HeyGen 是商業產品，某些功能需要付費訂閱。

## 系統要求

### 網頁訪問要求

| 要求 | 最低 | 推薦 |
|------|------|------|
| 瀏覽器 | Chrome 90+, Firefox 88+, Safari 14+, Edge 90+ | 最新版 Chrome |
| 互聯網 | 10 Mbps | 25 Mbps+ |

### API/SDK 開發要求

| 要求 | 版本 |
|------|------|
| Node.js | 16+ |
| Python | 3.8+ |
| API 密鑰 | 來自 HeyGen 儀表板 |

## 訪問 HeyGen

### 步驟 1：創建帳戶

1. 訪問 https://www.heygen.com
2. 點擊"註冊"或"開始使用"
3. 驗證電子郵件地址

### 步驟 2：選擇套餐

| 套餐 | 功能 | 價格 |
|------|------|------|
| 免費 | 有限分鐘數，有水印 | 免費 |
| 專業版 | 更多分鐘數，無水印 | $29/月 |
| 企業版 | 自定義限額，API 訪問 | 聯繫銷售 |

### 步驟 3：獲取 API 密鑰（開發者）

1. 登錄 HeyGen 儀表板
2. 導航到設置 → API
3. 生成 API 密鑰

## API 安裝（開發者）

### Python SDK

```bash
# 安裝 HeyGen SDK
pip install heygen-python
```

### Node.js SDK

```bash
# 安裝 HeyGen SDK
npm install heygen-node
```

## 驗證安裝

```python
# Python
import heygen
print(heygen.__version__)
```

## 環境變量

### 設置 API 密鑰

```bash
# Linux/macOS
export HEYGEN_API_KEY="your_api_key"

# Windows (PowerShell)
$env:HEYGEN_API_KEY="your_api_key"
```

## 常見問題

### 認證錯誤

**問題：** "API 密鑰無效"

**解決方案：**
1. 檢查 API 密鑰是否正確
2. 確保 API 密鑰處於活動狀態
3. 驗證帳戶是否有 API 訪問權限
