# Yarn 包管理器

Yarn 是一个成熟的开源包管理器，用于管理 JavaScript 项目中的依赖关系。

## 主版本信息

### Yarn v4+ (现代版本)
- **官网**: https://yarnpkg.com/
- **状态**: 活跃开发中
- **特点**: 插件化架构、工作区原生支持、PnP、离线缓存

### Yarn v1 (Classic)
- **官网**: https://classic.yarnpkg.com/
- **状态**: 维护中
- **特点**: 传统架构、简单直接、广泛兼容
