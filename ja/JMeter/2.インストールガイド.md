# JMeter インストールガイド

## 前提条件

- **Java Development Kit (JDK)**: JMeterはJava 8以上が必要
- **オペレーティングシステム**: Windows、Linux、またはmacOS
- **メモリ**: 負荷テストには少なくとも4GBのRAMを推奨

## インストール手順

### 1. JMeterをダウンロード

Apache JMeterの公式サイトにアクセス: https://jmeter.apache.org/

最新バージョンをダウンロード（通常は.zipまたは.tar.gzファイル）。

### 2. アーカイブを展開

ダウンロードしたファイルを 원하는場所に展開:

```bash
# Linux/Mac
tar -xvf apache-jmeter-x.x.x.tgz

# Windows
# 7-ZipまたはWinRARを使用して展開
```

### 3. 環境変数の設定（オプション）

便利のため、システムPATHにJMeterを追加できます:

```bash
 # Linux/Mac（~/.bashrcまたは~/.zshrcに追加）
export JMETER_HOME=/path/to/apache-jmeter-x.x.x
export PATH=$PATH:$JMETER_HOME/bin

# Windows
# システムプロパティ -> 環境変数 -> PATHに追加
```

### 4. インストールを確認

JMeterが正しくインストールされていることを確認するには следующую командуを実行:

```bash
# Linux/Mac
./jmeter.sh --version

# Windows
jmeter.bat --version
```

### 5. JMeter GUIを起動

```bash
# Linux/Mac
./jmeter.sh

# Windows
jmeter.bat
```

## ディレクトリ構造

```
apache-jmeter/
├── bin/          # 実行スクリプトとJMeter実行ファイル
├── lib/          # JMeterライブラリ
├── lib/ext/      # プラグインと拡張機能
├── docs/         # ドキュメント
├── printable_docs/  # ユーザーマニュアル
└── LICENSE       # ライセンスファイル
```

## さまざまなモードでの実行

### GUIモード（インタラクティブ）
```bash
jmeter.sh          # Linux/Mac
jmeter.bat         # Windows
```

### CLIモード（ヘッドレス）
```bash
jmeter -n -t testplan.jmx -l results.jtl -e -o html-report
```

パラメータ:
- `-n`: 非GUIモードで実行
- `-t`: テストplanファイル
- `-l`: 結果ファイル
- `-e`: テストを生成
-後にHTMLレポート `-o`: HTMLレポートの インスト出力ディレクトリ
