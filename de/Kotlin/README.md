# Kotlin

Kotlin ist eine von JetBrains entwickelte moderne Programmiersprache, die auf der JVM läuft.

## Einführung

Kotlin ist eine statisch typisierte Programmiersprache, die auf der Java Virtual Machine ausgeführt oder auch zu JavaScript-Code kompiliert werden kann.

## Hauptmerkmale

- **Prägnant**: Reduziert Boilerplate-Code
- **Sicher**: Null-Sicherheit-Design
- **Interoperabel**: 100% interoperabel mit Java
- **Werkzeugfreundlich**: Ausgezeichnete IDE-Unterstützung
- **Koroutinen**: Starke Unterstützung für asynchrone Programmierung

## Schnellstart

### Installation

```bash
# macOS
brew install kotlin

# Linux
sudo apt install kotlin

# SDKMAN
sdk install kotlin
```

### Kompilieren und Ausführen

```bash
kotlinc hello.kt -include-runtime -d hello.jar
java -jar hello.jar
```
