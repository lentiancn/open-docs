# HeyGen Guía de Instalación

HeyGen es una plataforma de generación de video AI que permite crear avatares AI realistas y generar videos de cabeza parlante.

## Descripción

HeyGen es principalmente una plataforma SaaS basada en la nube. No requiere instalación local.

### Métodos de acceso

| Método | Descripción |
|--------|-------------|
| Interfaz web | Creación de video basada en navegador |
| API | Acceso programático |
| SDK | Desarrollo local |

## Versiones

HeyGen es un servicio en la nube:

| Componente | Versión |
|------------|---------|
| HeyGen Web | Latest |
| HeyGen API | v1/v2 |
| HeyGen SDK | Latest |

**Nota:** HeyGen es un producto comercial.

## Requisitos

### Acceso web

| Requisito | Mínimo | Recomendado |
|------------|--------|--------------|
| Navegador | Chrome 90+ | Latest Chrome |
| Internet | 10 Mbps | 25 Mbps+ |

### Desarrollo API/SDK

| Requisito | Versión |
|------------|---------|
| Node.js | 16+ |
| Python | 3.8+ |
| API Key | Del dashboard HeyGen |

## Acceso a HeyGen

### Paso 1: Crear cuenta

1. Visite https://www.heygen.com
2. Haga clic en "Registrarse"
3. Verifique su email

### Paso 2: Elegir plan

| Plan | Características | Precio |
|------|-----------------|--------|
| Gratis | Minutos limitados | Gratis |
| Pro | Más minutos | $29/mes |
| Enterprise | Límites personalizados | Contacto |

### Paso 3: Obtener API Key

1. Inicie sesión en el dashboard
2. Vaya a Settings → API
3. Genere API key

## Instalación API

### Python SDK

```bash
pip install heygen-python
```

### Node.js SDK

```bash
npm install heygen-node
```

### Verificar instalación

```python
import heygen
print(heygen.__version__)
```

## Configuración

### Python

```python
from heygen import HeyGen

heygen = HeyGen(api_key="your_api_key")
account = heygen.get_account_info()
print(account)
```

### Variables de entorno

```bash
export HEYGEN_API_KEY="your_api_key"
```

## Problemas comunes

### Errores de autenticación

**Solución:**
1. Verificar API key
2. Verificar que la cuenta tiene acceso API

### Límites de velocidad

**Solución:**
1. Verificar límites de uso
2. Actualizar plan

## Mejores prácticas

1. **Nunca exponga API keys** - Usar variables de entorno
2. **Rotar claves regularmente**
3. **Usar permisos mínimos**
