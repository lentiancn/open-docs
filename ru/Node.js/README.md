# Node.js

Node.js — это среда выполнения JavaScript, основанная на движке Chrome V8.

## Введение

Node.js — это мощная среда выполнения JavaScript для создания быстрых и масштабируемых сетевых приложений. Она использует событийную неблокирующую модель ввода-вывода, что делает её идеальной для приложений реального времени.

## Основные особенности

- **Высокая производительность**: Основана на движке Chrome V8
- **Событийная модель**: Неблокирующая модель ввода-вывода
- **Кроссплатформенность**: Поддерживает Windows, macOS и Linux
- **Богатая экосистема**: npm содержит огромное количество пакетов
- **Fullstack-разработка**: JavaScript можно использовать как для фронтенда, так и для бэкенда

## Быстрый старт

### Установка

```bash
# Windows
choco install nodejs

# macOS
brew install node@20

# Linux
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
```

### Проверка установки

```bash
node --version
npm --version
```

### Запуск первой программы

```javascript
// hello.js
console.log('Hello, Node.js!');
```

Запуск:

```bash
node hello.js
```
