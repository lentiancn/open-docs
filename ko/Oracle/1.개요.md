# Oracle 데이터베이스

> 엔터프라이즈급 관계형 데이터베이스 관리 시스템

---

## 개요

Oracle Database는 Oracle Corporation에서 개발한 독점적인 관계형 데이터베이스 관리 시스템입니다. 전 세계적으로 가장 널리 사용되는 엔터프라이즈 데이터베이스 시스템 중 하나입니다.

---

## 주요 기능

- **고성능**: 최적화된 쿼리 처리
- **확장성**: 대용량 데이터 지원
- **보안**: 고급 보안 기능
- **고가용성**: Real Application Clusters (RAC)
- **PL/SQL**: 절차적 언어 확장
