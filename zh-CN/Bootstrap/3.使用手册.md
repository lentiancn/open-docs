## 1. 网格系统

Bootstrap 的网格系统是其核心功能之一，基于 Flexbox 构建，可以轻松创建响应式布局。

### 基本结构

网格系统由三个主要元素组成：

```html
<!-- 容器：网格的包装元素 -->
<div class="container">
    <!-- 行：包含列的容器 -->
    <div class="row">
        <!-- 列：实际放置内容的区域 -->
        <div class="col">
            内容
        </div>
    </div>
</div>
```

### 十二列系统

Bootstrap 使用十二列系统，每行可以放置最多 12 个列：

```html
<div class="container">
    <div class="row">
        <div class="col-12">12 列 - 整行</div>
    </div>
    <div class="row">
        <div class="col-6">6 列</div>
        <div class="col-6">6 列</div>
    </div>
    <div class="row">
        <div class="col-4">4 列</div>
        <div class="col-4">4 列</div>
        <div class="col-4">4 列</div>
    </div>
    <div class="row">
        <div class="col-3">3 列</div>
        <div class="col-3">3 列</div>
        <div class="col-3">3 列</div>
        <div class="col-3">3 列</div>
    </div>
</div>
```

### 响应式断点

Bootstrap 提供了六个响应式断点：

| 断点 | 类名前缀 | 屏幕宽度 |
|------|----------|----------|
| 超小 | col- | < 576px |
| 小 | col-sm- | ≥ 576px |
| 中 | col-md- | ≥ 768px |
| 大 | col-lg- | ≥ 992px |
| 超大 | col-xl- | ≥ 1200px |
| 超超大 | col-xxl- | ≥ 1400px |

### 响应式布局示例

```html
<div class="container">
    <div class="row">
        <!-- 超小屏幕上占满12列，中等及以上屏幕占6列 -->
        <div class="col-12 col-md-6">列 1</div>
        <div class="col-12 col-md-6">列 2</div>
    </div>
    <div class="row">
        <!-- 超小屏幕上占满12列，大屏幕占4列 -->
        <div class="col-12 col-lg-4">列 A</div>
        <div class="col-12 col-lg-4">列 B</div>
        <div class="col-12 col-lg-4">列 C</div>
    </div>
</div>
```

### 列偏移

使用偏移类将列向右移动：

```html
<div class="row">
    <!-- 偏移4列，剩下8列 -->
    <div class="col-8 offset-4">偏移列</div>
</div>
```

### 列排序

使用排序类改变列的显示顺序：

```html
<div class="row">
    <div class="col order-last">最后</div>
    <div class="col order-first">最前</div>
    <div class="col order-5">第5位</div>
</div>
```

### 嵌套网格

可以在列内嵌套新的网格：

```html
<div class="container">
    <div class="row">
        <div class="col-sm-9">
            外层列
            <div class="row">
                <div class="col-8 col-sm-6">嵌套列 1</div>
                <div class="col-4 col-sm-6">嵌套列 2</div>
            </div>
        </div>
    </div>
</div>
```

### gutter（间距）

调整列之间的间距：

```html
<!-- 水平间距 -->
<div class="row gx-5">
    <div class="col-6">列</div>
    <div class="col-6">列</div>
</div>

<!-- 垂直间距 -->
<div class="row gy-3">
    <div class="col-6">列</div>
    <div class="col-6">列</div>
</div>

<!-- 水平和垂直间距 -->
<div class="row g-2">
    <div class="col-6">列</div>
    <div class="col-6">列</div>
</div>
```

## 2. 容器和布局

### 容器类型

```html
<!-- 响应式固定宽度容器 -->
<div class="container">
    在不同断点有最大宽度
</div>

<!-- 100% 宽度容器 -->
<div class="container-fluid">
    全宽度，贯穿整个屏幕
</div>

<!-- 响应式容器 -->
<div class="container-md">
    移动端全宽度，中等及以上屏幕固定宽度
</div>
```

### 容器最大宽度对照

| 类名 | 超小 | 小 | 中 | 大 | 超大 | 超超大 |
|------|------|-----|-----|------|------|--------|
| .container | 100% | 540px | 720px | 960px | 1140px | 1320px |
| .container-fluid | 100% | 100% | 100% | 100% | 100% | 100% |

## 6. 自定义主题

### 使用 CSS 变量

Bootstrap 5 使用 CSS 变量，可以轻松自定义主题：

```css
:root {
    /* 修改主色调 */
    --bs-primary: #0d6efd;
    --bs-secondary: #6c757d;
    --bs-success: #198754;
    --bs-danger: #dc3545;
    --bs-warning: #ffc107;
    --bs-info: #0dcaf0;
    --bs-light: #f8f9fa;
    --bs-dark: #212529;

    /* 修改字体 */
    --bs-font-sans-serif: system-ui, -apple-system, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif;
}
```

### 使用 Sass 自定义

```scss
// 引入 Bootstrap 的 Sass 源文件
@import "bootstrap/scss/functions";

// 自定义变量
$primary: #712cf9;
$enable-rounded: false;

// 引入 Bootstrap 其他部分
@import "bootstrap/scss/bootstrap";
```
