# SadTalker 安装指南

SadTalker 是一个 AI 项目，用于从单张图像和音频生成逼真的说话头像视频。

## 系统要求

### 硬件
- **GPU**: NVIDIA GPU 8GB+ 显存（推荐 16GB）
- **内存**: 16GB+ 系统内存
- **存储**: 30GB+ 可用空间

### 软件
- **系统**: Ubuntu 20.04/22.04, Windows 10/11, macOS
- **Python**: 3.8 - 3.10
- **CUDA**: 11.7+

## Linux/Ubuntu 安装

### 1. 安装依赖

```bash
sudo apt update && sudo apt upgrade
sudo apt install python3.10 python3-pip libgl1-mesa-glx libglib2.0-0 ffmpeg
```

### 2. 克隆项目

```bash
git clone https://github.com/OpenTalker/SadTalker.git
cd SadTalker
```

### 3. 创建虚拟环境

```bash
python3 -m venv venv
source venv/bin/activate
```

### 4. 安装依赖

```bash
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu117
pip install -r requirements.txt
```

### 5. 下载模型

```bash
bash scripts/download_models.sh
```

## Windows 安装

```bash
git clone https://github.com/OpenTalker/SadTalker.git
cd SadTalker

python -m venv venv
venv\Scripts\activate

pip install torch torchvision --index-url https://download.pytorch.org/whl/cu117
pip install -r requirements.txt
```

## Docker 安装

```bash
docker pull ghcr.io/opentalker/sadtalker:latest
docker run --gpus all -v /path/to/data:/data ghcr.io/opentalker/sadtalker:latest
```
