# Guide d'installation du JDK

JDK (Java Development Kit) est le kit de développement Java officiel. Ce guide couvre les méthodes d'installation pour différents environnements.

## Informations sur les versions

| Version | Statut | Description |
|---------|--------|-------------|
| JDK 21 (LTS) | ✅ Actuelle | Dernière version LTS |
| JDK 17 (LTS) | ✅ Supportée | Version LTS précédente |
| JDK 11 (LTS) | ✅ Supportée | Version LTS historique |
| JDK 8 | ✅ Supportée | Version historique |
| JDK 22 | ✅ Dernière | Dernière version fonctionnalités |

### Différence entre LTS et versions fonctionnalités

- **LTS (support à long terme)**: 2-3 ans de mises à jour, recommandé pour la production
- **Versions fonctionnalités**: nouvelles fonctionnalités tous les 6 mois, support plus court

## Méthodes d'installation

### Méthode 1: Oracle JDK

#### Windows

1. Télécharger depuis: https://www.oracle.com/java/technologies/downloads/
2. Exécuter l'installateur (.exe)
3. Suivre l'assistant

#### macOS

```bash
# Télécharger le DMG depuis le site Oracle
# Ou utiliser Homebrew
brew install openjdk@21
```

#### Linux

```bash
# Télécharger tar.gz
wget https://download.oracle.com/java/21/latest/jdk-21_linux-x64_bin.tar.gz
tar -xzf jdk-21_linux-x64_bin.tar.gz
sudo mv jdk-21 /opt/jdk-21
```

### Méthode 2: Eclipse Temurin (Adoptium)

#### Windows

1. Télécharger depuis: https://adoptium.net/temurin/releases/
2. Exécuter l'installateur

#### macOS

```bash
# Utiliser Homebrew
brew install --cask temurin

# Ou télécharger le pkg depuis le site
```

#### Linux

```bash
# Ubuntu/Debian
wget https://github.com/adoptium/temurin21-binaries/releases/download/jdk-21.0.2%2B13/OpenJDK21U-jdk_x64_linux_hotspot_21.0.2_13.tar.gz
tar -xzf OpenJDK21U-jdk_x64_linux_hotspot_21.0.2_13.tar.gz

# Configurer l'environnement
export JAVA_HOME=/path/to/jdk-21
export PATH=$JAVA_HOME/bin:$PATH
```

### Méthode 3: SDKMAN

```bash
# Installer SDKMAN
curl -s "https://get.sdkman.io" | bash

# Installer JDK
sdk install java 21.0.2-tem

# Lister les versions disponibles
sdk list java
```

### Méthode 4: Docker

```dockerfile
FROM eclipse-temurin:21-jdk
WORKDIR /app
COPY . .
RUN javac Main.java
CMD ["java", "Main"]
```

### Méthode 5: Gestionnaires de paquets Windows

#### Chocolatey

```powershell
choco install openjdk21
```

#### Scoop

```powershell
scoop install openjdk
```

## Configuration de l'environnement

### Windows

```powershell
# Définir JAVA_HOME
[System.Environment]::SetEnvironmentVariable("JAVA_HOME", "C:\Program Files\Java\jdk-21", "User")

# Ajouter au PATH
[System.Environment]::SetEnvironmentVariable("PATH", "$env:PATH;C:\Program Files\Java\jdk-21\bin", "User")
```

### macOS/Linux

```bash
# Ajouter à ~/.bashrc ou ~/.zshrc
export JAVA_HOME=/opt/jdk-21
export PATH=$JAVA_HOME/bin:$PATH

# Appliquer
source ~/.bashrc
```

## Vérification

```bash
# Vérifier la version de Java
java -version

# Vérifier la version de Javac
javac -version

# Vérifier JAVA_HOME
echo $JAVA_HOME
```

## Plusieurs versions de JDK

### Utiliser SDKMAN

```bash
# Installer plusieurs versions
sdk install java 17.0.8-tem
sdk install java 21.0.2-tem

# Changer de version
sdk use java 17.0.8-tem
```

## Problèmes courants

### "java" non trouvé ?

Vérifiez que le PATH inclut le répertoire bin de Java.

### Mauvaise version de Java ?

Vérifiez quel java est utilisé:

```bash
which java
java -version
```

### JAVA_HOME non défini ?

Définissez JAVA_HOME vers le répertoire d'installation du JDK.
