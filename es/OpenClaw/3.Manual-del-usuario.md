# Manual del Usuario de OpenClaw

Este manual cubre el uso diario y las características principales de OpenClaw.

## Conceptos Básicos

### Gateway

El Gateway es el componente central de OpenClaw, responsable de la gestión de sesiones, enrutamiento de mensajes y conexiones de canal. Solo necesitas ejecutar un proceso de Gateway, y maneja automáticamente todas las solicitudes de conexión de canales.

### Canales

Los canales son las aplicaciones de chat que usas para comunicarte con tu asistente de IA. OpenClaw supports connecting multiple channels simultaneously, including WhatsApp, Telegram, Discord, iMessage, and more.

### Agentes

Los agentes son instancias de asistentes de IA. Cada agente tiene su propio espacio de trabajo y sesiones. Puedes configurar múltiples agentes para manejar diferentes tareas independientemente.

## Operaciones Diarias

### Iniciar y Detener Gateway

```bash
# Iniciar Gateway (servicio en segundo plano)
openclaw gateway start

# Detener Gateway
openclaw gateway stop

# Reiniciar Gateway
openclaw gateway restart

# Ver estado
openclaw gateway status

# Ejecutar en primer plano (para depuración)
openclaw gateway --port 18789
```

### Enviar Mensajes

```bash
# Enviar un mensaje directo
openclaw message send --target +15555550123 --message "Hola"

# Enviar a un grupo
openclaw message send --target "Nombre del Grupo" --message "Hola a todos"
```

### Gestionar Configuración

```bash
# Ver configuración actual
openclaw config get

# Establecer un valor de configuración
openclaw config set agents.defaults.model "anthropic/claude-sonnet-4-5"

# Eliminar un valor de configuración
openclaw config unset channels.whatsapp.enabled
```

## Gestión de Configuración

### Ubicación del Archivo de Configuración

El archivo de configuración está ubicado en `~/.openclaw/openclaw.json`.

### Recarga en Caliente

OpenClaw soporta recarga de configuración en caliente: puedes modificar el archivo de configuración sin reiniciar manualmente el Gateway. La mayoría de los cambios de configuración se aplican automáticamente.

### Ejemplos de Configuración Común

#### Establecer Fuentes de Mensaje Permitidas

```json5
{
  channels: {
    whatsapp: {
      allowFrom: ["+15555550123"]
    }
  }
}
```

#### Establecer Reglas de Mención en Grupos

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

#### Configurar Múltiples Agentes

```json5
{
  agents: {
    list: [
      { id: "home", default: true, workspace: "~/.openclaw/workspace-home" },
      { id: "work", workspace: "~/.openclaw/workspace-work" }
    ]
  }
}
```

## Gestión de Canales

### Agregar Nuevos Canales

```bash
# Iniciar sesión en WhatsApp
openclaw channels login whatsapp

# Iniciar sesión en Telegram
openclaw channels login telegram
```

### Ver Canales Conectados

```bash
openclaw channels list
```

## Funciones Avanzadas

### Trabajos Cron

Configurar tareas programadas:

```json5
{
  cron: {
    enabled: true,
    maxConcurrentRuns: 2
  }
}
```

### Heartbeat

Configurar verificaciones periódicas:

```json5
{
  agents: {
    defaults: {
      heartbeat: {
        every: "30m",
        target: "last"
      }
    }
  }
}
```

### Modo Sandbox

Ejecutar agentes en contenedores Docker aislados:

```json5
{
  agents: {
    defaults: {
      sandbox: {
        mode: "non-main",
        scope: "agent"
      }
    }
  }
}
```

### Webhooks

Configurar endpoints de webhook HTTP:

```json5
{
  hooks: {
    enabled: true,
    token: "tu-token-secreto",
    path: "/hooks"
  }
}
```

## Solución de Problemas

### Ver Registros

```bash
# Ver registros del Gateway
openclaw logs

# Seguir registros en tiempo real
openclaw logs -f
```

### Verificación de Salud

```bash
openclaw health
```

### Diagnosticar Problemas

```bash
openclaw doctor
```

## Interfaz de Control

OpenClaw proporciona una Interfaz de Control web donde puedes:

- Enviar y recibir mensajes
- Gestionar configuración
- Ver historial de sesiones
- Gestionar nodos móviles
- Monitorear estado del Gateway

Accede en: `http://127.0.0.1:18789/`
