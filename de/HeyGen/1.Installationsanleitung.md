# HeyGen Installationsanleitung

HeyGen ist eine KI-Videogenerierungsplattform, die es Benutzern ermöglicht, realistische KI-Avatare zu erstellen und Sprechkopfvideos aus Text- oder Audioeingaben zu generieren.

## Plattformübersicht

HeyGen ist hauptsächlich eine cloudbasierte SaaS-Plattform. Im Gegensatz zu Open-Source-Projekten erfordert HeyGen keine lokale Installation – Benutzer greifen über die Weboberfläche oder API zu.

### Zugriffsmethoden

| Methode | Beschreibung | Anforderungen |
|---------|--------------|---------------|
| Web-Oberfläche | Browserbasierte Videoerstellung | Konto + Internet |
| API | Programmatischer Zugriff | API-Schlüssel |
| SDK | Lokale Entwicklung | Entwicklerkonto |

## Versionsübersicht

HeyGen ist ein Cloud-Dienst, keine lokale Installation:

| Komponente | Version | Hinweise |
|------------|---------|----------|
| HeyGen Web | Aktuellste | Immer aktuell |
| HeyGen API | v1/v2 | REST-API |
| HeyGen SDK | Aktuellste | Für Entwickler |

**Hinweis:** HeyGen ist ein kommerzielles Produkt. Einige Funktionen erfordern ein kostenpflichtiges Abonnement.

## Zugriff auf HeyGen

### Schritt 1: Konto erstellen

1. Besuchen Sie https://www.heygen.com
2. Klicken Sie auf "Registrieren" oder "Loslegen"
3. E-Mail-Adresse verifizieren

### Schritt 2: Plan wählen

| Plan | Funktionen | Preis |
|------|------------|-------|
| Kostenlos | Begrenzte Minuten, Wasserzeichen | Kostenlos |
| Pro | Mehr Minuten, kein Wasserzeichen | $29/Monat |
| Enterprise | Benutzerdefinierte Limits, API-Zugriff | Vertrieb kontaktieren |

### Schritt 3: API-Schlüssel erhalten (für Entwickler)

1. Bei HeyGen-Dashboard anmelden
2. Zu Einstellungen → API navigieren
3. API-Schlüssel generieren

## API-Installation (Für Entwickler)

### Python SDK

```bash
# HeyGen SDK installieren
pip install heygen-python

# Oder von Quelle installieren
pip install git+https://github.com/heygen-ai/heygen-python.git
```

### Node.js SDK

```bash
# HeyGen SDK installieren
npm install heygen-node

# Oder von Quelle installieren
npm install github:heygen-ai/heygen-node
```

### Installation überprüfen

```python
# Python
import heygen
print(heygen.__version__)
```

```javascript
// Node.js
const heygen = require('heygen-node');
console.log(heygen.version);
```

## API-Konfiguration

### Python-Beispiel

```python
from heygen import HeyGen

# Mit API-Schlüssel initialisieren
heygen = HeyGen(api_key="your_api_key")

# Kontoinformationen abrufen
account = heygen.get_account_info()
print(account)
```

### Node.js-Beispiel

```javascript
const { HeyGen } = require('heygen-node');

const heygen = new HeyGen({ apiKey: 'your_api_key' });

// Kontoinformationen abrufen
const account = await heygen.getAccountInfo();
console.log(account);
```

## Umgebungsvariablen

### API-Schlüssel festlegen

```bash
# Linux/macOS
export HEYGEN_API_KEY="your_api_key"

# Windows (PowerShell)
$env:HEYGEN_API_KEY="your_api_key"

# In .env-Datei
HEYGEN_API_KEY=your_api_key
```

### Python mit dotenv

```python
from dotenv import load_dotenv
load_dotenv()

import heygen
heygen = HeyGen(api_key=os.getenv("HEYGEN_API_KEY"))
```

## Häufige Probleme

### Authentifizierungsfehler

**Problem:** "Ungültiger API-Schlüssel"

**Lösungen:**
1. Überprüfen Sie, ob der API-Schlüssel korrekt ist
2. Stellen Sie sicher, dass der API-Schlüssel aktiv ist
3. Verifizieren Sie, ob das Konto API-Zugriff hat

### Ratenbegrenzung

**Problem:** "Zu viele Anfragen"

**Lösungen:**
1. API-Nutzungslimits überprüfen
2. Plan für höhere Limits upgraden
3. Anfrage-Warteschlange implementieren

### Videogenerierung fehlgeschlagen

**Problem:** "Generierung fehlgeschlagen"

**Lösungen:**
1. Eingabeparameter überprüfen
2. Guthaben überprüfen
3. Später erneut versuchen

## Best Practices für Sicherheit

1. **API-Schlüssel niemals offenlegen** - Umgebungsvariablen verwenden
2. **Schlüssel regelmäßig rotieren** - Regelmäßig neue Schlüssel generieren
3. **Minimale Berechtigungen verwenden** - Nur benötigte Bereiche anfordern
4. **Nutzung überwachen** - Dashboard auf Anomalien prüfen
