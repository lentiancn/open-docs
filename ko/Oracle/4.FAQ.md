# Oracle FAQ

## 일반적인 질문

### Oracle Database란?

Oracle Database는 Oracle Corporation에서 개발한 독점적인 관계형 데이터베이스 관리 시스템입니다.

### PL/SQL이란?

PL/SQL은 Oracle의 SQL에 대한 절차적 언어 확장입니다.

---

## 설치

### 시스템 요구 사항은?

- 최소 2GB RAM
- 최소 8GB 디스크 공간

### ORA-12541: TNS:no listener?

리스너 시작:
```bash
lsnrctl start
```
