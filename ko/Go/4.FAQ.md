## 설치

### Windows: go 명령어를 찾을 수 없음?

**해결책**: `C:\Go\bin`을 PATH에 추가.

### 프록시 설정?

```bash
go env -w GOPROXY=https://goproxy.cn,direct
```

---

## 개발

### "cannot find package" 에러?

**해결책**:
1. `go mod init` 실행
2. `go mod tidy` 실행

### :=와 var의 차이?

- `var`: 명시적, 함수 외부 사용 가능
- `:=`: 짧은 선언, 함수 내부만

## 동시성

### Goroutine 누수?

```go
func worker(ctx context.Context) {
    for {
        select {
        case <-ctx.Done():
            return
        }
    }
}
```
