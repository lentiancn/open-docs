# Preguntas Frecuentes de OpenClaw

Esta página compila las preguntas y respuestas más comunes sobre el uso de OpenClaw. Si no encuentras la respuesta aquí, consulta la documentación oficial o únete a la comunidad para obtener ayuda.

---

## Preguntas Básicas

### ¿Qué es OpenClaw?

OpenClaw es una puerta de enlace de asistente de IA autohospedada que conecta múltiples plataformas de chat como WhatsApp, Telegram, Discord e iMessage a un asistente de codificación de IA (como Pi). Lo ejecutas en tu propia computadora y tienes control total sobre tus datos.

### ¿Es OpenClaw gratis?

OpenClaw en sí es de código abierto y gratis (licencia MIT). Sin embargo, usar modelos de IA (como Anthropic Claude u OpenAI GPT) puede requerir pago. Solo necesitas pagar por el uso de la API de IA.

### ¿Qué tipo de dispositivo necesito para ejecutar OpenClaw?

Requisitos mínimos:
- Node.js 22+
- 2GB de RAM
- 500MB de espacio en disco

Recomendado:
- 4GB+ de RAM
- Conexión de red estable
- Computadora o servidor siempre encendido

## Preguntas de Instalación

### La instalación falló. ¿Qué debo hacer?

1. Verifica la versión de Node.js:
   ```bash
   node --version
   ```
   Asegúrate de que sea ≥ 22.0.0

2. Usa el script de instalación oficial:
   ```bash
   curl -fsSL https://openclaw.ai/install.sh | bash
   ```

3. Ver mensajes de error detallados:
   ```bash
   npm install -g openclaw@latest --verbose
   ```

4. Ejecuta diagnósticos:
   ```bash
   openclaw doctor
   ```

### ¿Tengo problemas en Windows?

**WSL2 es fuertemente recomendado**. OpenClaw funciona mejor en un entorno Linux.

Instala WSL2:
```powershell
wsl --install
```

Luego instala OpenClaw en WSL2.

### ¿Cómo actualizo OpenClaw?

```bash
npm update -g openclaw@latest
```

O usa el comando de actualización oficial:
```bash
openclaw update
```

Después de actualizar, ejecuta diagnósticos:
```bash
openclaw doctor
```

## Preguntas de Configuración

### ¿Dónde está el archivo de configuración?

Ubicación predeterminada: `~/.openclaw/openclaw.json`

También puedes especificar una ruta personalizada:
```bash
openclaw gateway --config /path/to/config.json
```

### ¿Cómo configuro múltiples canales?

Agrega múltiples configuraciones de canal en el archivo de configuración:

```json5
{
  channels: {
    whatsapp: { enabled: true },
    telegram: { enabled: true, botToken: "xxx" },
    discord: { enabled: true, token: "xxx" },
  },
}
```

### ¿La configuración no surte efecto?

1. Asegúrate de que el formato del archivo de configuración sea correcto (JSON5)
2. Verifica si el Gateway está ejecutándose
3. Intenta reiniciar el Gateway:
   ```bash
   openclaw gateway restart
   ```

## Preguntas de Conexión

### El Gateway no inicia?

Posibles razones:
- El puerto 18789 ya está en uso
- Error de formato del archivo de configuración
- Permisos insuficientes

Soluciones:
```bash
# Ver uso del puerto
lsof -i :18789

# Verificar configuración
openclaw doctor

# Ejecutar como administrador
sudo openclaw gateway
```

### ¿La conexión del canal falló?

1. Verifica si las credenciales son válidas
2. Verifica el estado del canal:
   ```bash
   openclaw channels status --probe
   ```
3. Ver los registros:
   ```bash
   openclaw logs --follow
   ```

### ¿No puedo acceder a la interfaz de control?

1. Asegúrate de que el Gateway esté ejecutándose
2. Verifica si la URL es correcta: http://127.0.0.1:18789/
3. Verifica la configuración del firewall
4. Intenta reiniciar:
   ```bash
   openclaw gateway restart
   ```

## Preguntas de Uso

### ¿La IA no responde a mensajes?

Verifica lo siguiente:
1. ¿El canal está conectado?
2. ¿Estás en la lista blanca?
3. En grupos, ¿se requiere @mención?
4. Verifica los mensajes de error en los registros

### ¿Cómo reinicio una sesión?

Usa el comando `/new` o `/reset`, o en CLI:
```bash
openclaw session reset
```

### ¿Cómo cambio el modelo de IA?

Establece en el archivo de configuración:

```json5
{
  agents: {
    defaults: {
      model: {
        primary: "anthropic/claude-sonnet-4-5",
      },
    },
  },
}
```

## Preguntas de Seguridad

### ¿Es OpenClaw seguro?

OpenClaw prioriza la seguridad:
- Todos los datos almacenados localmente
- Credenciales almacenadas cifradas
- Soporta mecanismos de lista blanca y emparejamiento
- Puede habilitar aislamiento de sandbox

Recomendaciones:
- Usa lista blanca para controlar el acceso
- Actualiza las versiones regularmente
- Ejecuta `openclaw doctor` para verificar seguridad

### ¿Cómo configuro una lista blanca?

```json5
{
  channels: {
    whatsapp: {
      allowFrom: ["+15555550123", "+15555550124"],
    },
  },
}
```

### ¿Qué es el Emparejamiento (Pairing)?

El emparejamiento es un mecanismo de seguridad donde usuarios desconocidos son bloqueados cuando envían su primer mensaje. Necesitan tu aprobación antes de usar la IA.

Aprobar un usuario:
```bash
openclaw pairing approve --channel whatsapp --sender +15555550123
```

## Preguntas de Funcionalidades

### ¿Cómo uso las funciones de voz?

1. Configura Talk Mode:
   ```json5
   {
     talk: {
       enabled: true,
       voice: "elevenlabs",
     },
   }
   ```

2. Habilita Voice Wake en dispositivos soportados

### ¿Las tareas programadas no se ejecutan?

1. Verifica si Cron está habilitado:
   ```bash
   openclaw cron status
   ```

2. Ver la lista de tareas:
   ```bash
   openclaw cron list
   ```

3. Ver registros de ejecución:
   ```bash
   openclaw cron runs --id <jobId>
   ```

### ¿El nodo móvil no puede conectar?

1. Asegúrate de que el teléfono y el Gateway estén en la misma red
2. Verifica el estado del nodo:
   ```bash
   openclaw nodes status
   ```

3. Volver a emparejar el nodo:
   ```bash
   openclaw nodes pair
   ```

## Preguntas de Rendimiento

### ¿Está funcionando lentamente?

1. Verifica los recursos del sistema:
   ```bash
   openclaw status
   ```

2. Compactar sesiones:
   ```bash
   /compact
   ```

3. Limpiar registros antiguos:
   ```bash
   openclaw logs clean
   ```

### ¿Cómo limito el uso de recursos?

```json5
{
  agents: {
    defaults: {
      maxTokens: 100000,
    },
  },
  session: {
    maxAgeHours: 24,
  },
}
```
