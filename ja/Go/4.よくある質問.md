# Go よくある質問

> Go に関する一般的な質問への回答

## インストール

### Q: インストール後コマンドが見つからない？
**A**: PATH に Go が追加されているか確認

### Q: インストール確認方法は？
```bash
go version
go env GOPATH
```

---

## 構文

### Q: Go に例外処理はある？
**A**: いいえ、エラーは戻り値で処理:
```go
result, err := doSomething()
if err != nil {
    // エラー処理
}
```

### Q: ジェネリックはサポートされてる？
**A**: Go 1.18+ でサポート:
```go
func Print[T any](s []T) {
    for _, v := range s {
        fmt.Println(v)
    }
}
```

## 並发

### Q: Goroutine とスレッドの違いは？
- Goroutine は Go ランタイムで管理
- スレッドより軽量

### Q: Goroutine の終了を待つ方法は？
```go
var wg sync.WaitGroup
wg.Add(1)
go func() {
    defer wg.Done()
}()
wg.Wait()
```
