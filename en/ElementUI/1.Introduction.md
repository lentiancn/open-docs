# Element UI Introduction

> Element UI is a Vue 2-based desktop component library

---

## Overview

Element UI is a Vue 2.0-based desktop component library developed by Eleme team. Provides rich PC-side components including forms, tables, dialogs, navigation menus, buttons, and inputs.

Element UI is known for its clean design style and comprehensive documentation. Although officially discontinued (recommended to migrate to Element Plus for Vue 3), it's still widely used in many existing projects.

---

## Version

| Version | Vue Version | Status |
|---------|-------------|--------|
| Element UI 2.x | Vue 2 | Stable |
| Element UI 1.x | Vue 2 | Deprecated |

---

## Features

- Based on Vue 2
- Rich PC-side components
- Support on-demand import
- Internationalization support
- Theme customization
- Complete documentation

---

## Element UI vs Element Plus

| Feature | Element UI | Element Plus |
|---------|------------|--------------|
| Vue Version | Vue 2 | Vue 3 |
| TypeScript | Partial | Full |
| Composition API | Not supported | Supported |
| Maintenance | Discontinued | Active |
