# HeyGen Руководство по использованию

## Начало работы

### Использование веб-интерфейса

1. Войдите в HeyGen на https://www.heygen.com
2. Выберите аватар из библиотеки или загрузите свой
3. Выберите тип ввода: текст, аудио или видео
4. Сгенерируйте видео и скачайте

### Использование API

```python
from heygen import HeyGen

heygen = HeyGen(api_key="your_api_key")

video = heygen.generate_video(
    avatar_id="avatar_id",
    script="Hello, this is my first AI video!",
    voice_id="voice_id"
)
```

## Основные функции

### 1. Выбор аватара

```python
avatars = heygen.list_avatars()
```

### 2. Выбор голоса

```python
voices = heygen.list_voices()
```

### 3. Генерация видео

```python
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="Ваш текст",
    voice_id="voice_id"
)
```

## Параметры

| Параметр | Тип | Описание |
|-----------|------|----------|
| avatar_id | string | ID аватара |
| script | string | Текст для озвучивания |
| voice_id | string | ID голоса |

## Примеры

### Маркетинговые видео

```python
video = heygen.generate_video(
```
