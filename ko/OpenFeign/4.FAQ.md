# OpenFeign FAQ

## 일반적인 질문

### OpenFeign이란?

OpenFeign은 선언적 REST 클라이언트 기능을 제공하는 Java 라이브러리입니다. 개발자가 어노테이션을 사용하여 HTTP 서비스 인터페이스를 정의할 수 있게 합니다.

### Feign과 OpenFeign의 차이점은?

Feign은 원래 Netflix에서 만들었습니다. OpenFeign은 커뮤니티에서 유지 관리하는 오픈소스 버전입니다.

---

## 설치 및 설정

### 어떤 버전을 사용해야 하나요?

새 프로젝트의 경우 최신 안정화 버전(13.x)을 사용하세요.

### Spring Boot에서 OpenFeign을 활성화하려면?

메인 애플리케이션 클래스에 `@EnableFeignClients` 어노테이션을 추가하세요.

---

## 문제 해결

### 연결 시간 초과 오류?

구성에서 시간 초과 값을 늘리세요:

```yaml
feign:
  client:
    config:
      default:
        connectTimeout: 10000
        readTimeout: 10000
```

### HTTP 요청을 디버깅하려면?

구성에서 로깅을 활성화하세요:

```yaml
feign:
  client:
    config:
      default:
        loggerLevel: full
```
