# Kotlin 使用手冊

## 變數與基本類型

Kotlin 使用 var 和 val 兩個關鍵字來宣告變數。var 宣告的是可變變數，類似於 Java 中的普通變數；val 宣告的是不可變變數，類似於 Java 中的 final 變數。在 Kotlin 中，優先使用 val 是最佳實踐，這可以讓程式碼更加安全且易於理解。

宣告變數時，可以顯式指定類型，也可以讓編譯器自動推斷類型：

val name: String = "Kotlin"
val age = 10 // 編譯器自動推斷為 Int 類型
var count = 0 // 可變變數

Kotlin 的基本類型包括：整數型（Int、Long、Short、Byte）、浮點型（Double、Float）、字元型（Char）、布林型（Boolean）和字串（String）。與 Java 不同，Kotlin 中所有基本類型都是物件，它們都有對應的成員方法。

Kotlin 的整數型預設是 Int 類型，如果需要表示更大的數值，可以使用 Long 後綴：

val longNumber = 100L // Long 類型

Kotlin 的字串非常強大，支援字串範本功能，可以在字串中直接嵌入變數或表達式：

val name = "World"
println("Hello, $name!") // 輸出 Hello, World!
println("Length: ${name.length}") // 輸出 Length: 5

## 集合

Kotlin 標準庫提供了豐富的集合類型，包括 List（列表）、Set（集合）和 Map（映射）。每種集合都有可變和不可變兩個版本。

不可變集合建立後不能新增、刪除或修改元素：

val list = listOf(1, 2, 3, 4, 5)
val set = setOf("a", "b", "c")
val map = mapOf("key1" to 1, "key2" to 2)

可變集合允許修改元素：

val mutableList = mutableListOf(1, 2, 3)
mutableList.add(4)
mutableList.removeAt(0)

Kotlin 的集合提供了豐富的函數式操作，如 map、filter、reduce 等：

val numbers = listOf(1, 2, 3, 4, 5)
val doubled = numbers.map { it * 2 } // [2, 4, 6, 8, 10]
val evens = numbers.filter { it % 2 == 0 } // [2, 4]
val sum = numbers.reduce { acc, i -> acc + i } // 15

## 控制流

Kotlin 的控制流包括條件表達式、迴圈和跳轉表達式。

if 表達式在 Kotlin 中有回傳值，可以直接賦值給變數：

val max = if (a > b) a else b

when 表達式是 Kotlin 強大的多分支匹配工具，可以匹配各種條件：

when (x) {
    1 -> println("x is 1")
    2 -> println("x is 2")
    in 3..10 -> println("x is between 3 and 10")
    else -> println("x is something else")
}

Kotlin 的迴圈包括 for 迴圈和 while 迴圈。for 迴圈可以遍歷範圍、陣列和集合：

for (i in 1..10) {
    println(i)
}

for (item in list) {
    println(item)
}

for (i in list.indices) {
    println("Index: $i, Value: ${list[i]}")
}

## 函數

Kotlin 使用 fun 關鍵字宣告函數。函數可以指定參數類型和回傳類型：

fun greet(name: String): String {
    return "Hello, $name!"
}

如果函數體只有一個表達式，可以使用單一表達式函數語法：

fun greet(name: String) = "Hello, $name!"

Kotlin 支援預設參數和命名參數，這大大減少了重載函數的需求：

fun greet(name: String, greeting: String = "Hello") = "$greeting, $name!"

greet("World") // Hello, World!
greet("World", "Hi") // Hi, World!
greet(name = "World", greeting = "Hi") // Hi, World!

Kotlin 還支援可變參數，使用 vararg 關鍵字：

fun sum(vararg numbers: Int): Int {
    return numbers.sum()
}

sum(1, 2, 3, 4, 5) // 15

## 類別與物件

Kotlin 的類別宣告比 Java 更加簡潔。預設情況下，Kotlin 的類別是 final 的，不能被繼承。如果希望類別可以被繼承，需要使用 open 關鍵字：

open class Animal(val name: String) {
    open fun speak() = println("$name makes a sound")
}

class Dog(name: String) : Animal(name) {
    override fun speak() = println("$name barks")
}

Kotlin 的建構函數分為主建構函數和次建構函數。主建構函數直接在類別名稱後面宣告：

class Person(val name: String, val age: Int)

如果需要在建構函數中執行邏輯，可以使用 init 區塊：

class Person(name: String, age: Int) {
    val name: String
    val age: Int

    init {
        this.name = name
        this.age = age
        require(age >= 0) { "Age cannot be negative" }
    }
}

Kotlin 的資料類別（data class）自動生成 equals、hashCode、toString 和 copy 方法，非常適合用於儲存資料的類別：

data class User(val name: String, val email: String)

val user1 = User("Alice", "alice@example.com")
val user2 = user1.copy(email = "newemail@example.com")

## 空值安全

Kotlin 的空值安全是其最重要的特性之一。在 Kotlin 中，類型系統區分可空類型和非可空類型，這從根本上減少了空指標異常。

預設情況下，類型是不可空的：

val name: String = "Kotlin" // 不能為 null
val name: String = null // 編譯錯誤

如果要允許變數為 null，需要使用可空類型（?）：

val name: String? = null // 可以為 null

存取可空類型的屬性或方法時，需要使用安全呼叫操作符（?.）或非空斷言（!!）：

val length = name?.length // 如果 name 為 null，回傳 null
val length = name!!.length // 如果 name 為 null，拋出異常

Elvis 操作符（?:）提供了一種簡潔的預設值處理方式：

val length = name?.length ?: 0 // 如果 name 為 null，回傳 0

## 擴展函數與屬性

Kotlin 允許為現有類別新增新的函數和屬性，而不需要繼承或修改原類別。這就是擴展函數和擴展屬性。

擴展函數的宣告方式是在函數名前加上要擴展的類別名：

fun String.addExclamation() = this + "!"

"Hello".addExclamation() // "Hello!"

擴展屬性同樣可以新增新的屬性到現有類別：

val String.wordCount: Int
    get() = this.split(" ").size

"Hello World".wordCount // 2

## 高階函數與 Lambda

高階函數是指接受函數作為參數或回傳函數的函數。Lambda 表達式是實現這一特性的主要方式。

Lambda 表達式的語法使用大括號：

val sum = { a: Int, b: Int -> a + b }
sum(1, 2) // 3

如果 Lambda 表達式是函數的最後一個參數，可以將 Lambda 移到括號外：

list.filter { it > 0 }

Lambda 表達式中的 it 代表單個參數的隱式名稱，當參數只有一個時可以省略參數宣告。

## 協程

協程是 Kotlin 非同步編程的核心特性。相比執行緒，協程更加輕量級，可以在單個執行緒中執行多個協程。

launch 是啟動協程的常用方式，它會在背景啟動一個新的協程：

GlobalScope.launch {
    val data = async { fetchData() }
    updateUI(data.await())
}

suspend 函數是只能在協程中呼叫的函數：

suspend fun fetchData(): Data {
    return withContext(Dispatchers.IO) {
        // 執行耗時操作
    }
}

Kotlin 協程提供了多種調度器來控制協程執行所在的執行緒：Dispatchers.Main 用於 UI 操作，Dispatchers.IO 用於 I/O 操作，Dispatchers.Default 用於 CPU 密集型任務。

## 密封類別與枚舉

密封類別是一種受限的類別階層結構，限制了可以繼承它的子類別數量。所有子類別必須在同一个檔案中宣告。這在表示有限狀態或結果類型時非常有用：

sealed class Result<out T> {
    data class Success<out T>(val data: T) : Result<T>()
    data class Error(val message: String) : Result<Nothing>()
    object Loading : Result<Nothing>()
}

when 表達式可以充分利用密封類別的特性，編譯器會檢查是否處理了所有可能的情況：

fun handleResult(result: Result<Any>) = when (result) {
    is Result.Success -> println(result.data)
    is Result.Error -> println(result.message)
    is Result.Loading -> println("Loading...")
}

枚舉類別用於表示有限數量的常數值：

enum class Direction {
    NORTH, SOUTH, EAST, WEST
}

枚舉類別可以有屬性和方法：

enum class Color(val rgb: Int) {
    RED(0xFF0000),
    GREEN(0x00FF00),
    BLUE(0x0000FF);

    fun containsRed() = (rgb and 0xFF0000) != 0
}

## 介面與抽象類別

介面在 Kotlin 中可以包含預設實作，這使得介面更加靈活：

interface Clickable {
    fun click()
    fun showOff() = println("I'm clickable!")
}

類別可以實作多個介面：

class Button : Clickable, Focusable {
    override fun click() = println("Button clicked")
    override fun showOff() {
        super<Clickable>.showOff()
        println("I'm focusable too!")
    }
}

## 物件表達式與物件宣告

物件表達式用於建立匿名類別的實例，類似於 Java 的匿名內部類別：

window.addMouseListener(object : MouseAdapter() {
    override fun mouseClicked(e: MouseEvent) {
        // 處理點擊事件
    }
})

物件宣告用於建立單例：

object Config {
    val apiUrl = "https://api.example.com"
    val timeout = 30
}

## 伴隨物件

每個類別可以包含一個伴隨物件，類似於 Java 中的靜態成員：

class MyClass {
    companion object {
        const val TAG = "MyClass"
        fun create() = MyClass()
    }
}

伴隨物件的成員可以透過類別名稱直接存取：

MyClass.TAG
MyClass.create()

## 類型檢查與轉換

is 運算子用於類型檢查，類似於 Java 的 instanceof：

if (obj is String) {
    // 在這個分支中，obj 被自動轉換為 String
    println(obj.length)
}

智慧轉換會自動轉換已經檢查過的類型。如果編譯器能夠推斷出變數在特定分支中的類型，就會自動進行轉換。

如果智慧轉換不適用，可以使用顯式轉換：

val str = obj as String

安全的類型轉換使用 as?，當轉換失敗時回傳 null：

val str = obj as? String // 如果轉換失敗，回傳 null
