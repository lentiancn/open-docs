# Element Plus 概要

Vue 3 用 UI コンポーネントライブラリ。

## Element Plus とは？

Element Plus は、Vele.me チーム開発された Vue 3 コンポーネントライブラリで сучасным Web アプリケーションのための完全なデスクトップ UI コンポーネントを提供します。

### コア機能

- **Vue 3 ベース**: Composition API で構築
- **TypeScript サポート**: ネイティブ型定義
- **70+ コンポーネント**: 高品質コンポーネント
- **ツリーシェイキング**: 必要に応じた読み込み
- **テーマカスタマイズ**: CSS 変数サポート
- **国際化**: 20+ 言語パック

### 技術スタック

- Vue 3.2+
- TypeScript 4.5+
- Vite
- SCSS
