# Bootstrap Обзор

Самая популярная CSS-фреймворк в мире.

## Что такое Bootstrap?

Bootstrap — самый популярный CSS-фреймворк от Twitter для создания адаптивных веб-сайтов.

### Основные функции

- **Адаптивная сетка** - 12-колоночная адаптивная сетка
- **Готовые компоненты** - Навбары, карточки, модальные окна
- **CSS утилиты** - Быстрые настройки стилей
