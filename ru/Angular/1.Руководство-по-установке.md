## Системные требования

### Аппаратные требования
- **Процессор**: Двухъядерный или лучше
- **ОЗУ**: Минимум 4 ГБ (рекомендуется 8 ГБ)
- **Место на диске**: Не менее 3 ГБ свободного места

### Программные требования

#### Для Windows
- Windows 10 или выше
- Node.js 18.19 или выше (рекомендуется LTS)
- npm 9.x или выше
- Git

#### Для macOS
- macOS 10.15 или выше
- Node.js 18.19 или выше
- npm 9.x или выше
- Git

#### Для Linux
- Ubuntu 20.04 LTS или выше
- Node.js 18.19 или выше
- npm 9.x или выше
- Git

## Предварительные требования

### 1. Установка Node.js

#### Windows
Загрузите с https://nodejs.org/ и установите LTS-версию.

#### macOS
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install node@18
```

#### Linux
```bash
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs
```

### 2. Установка Git

```bash
# Ubuntu/Debian
sudo apt install git

# macOS
brew install git
```

## Установка в Windows

1. Установите Node.js с https://nodejs.org/
2. Установите Git с https://git-scm.com/
3. Установите Angular CLI: `npm install -g @angular/cli`
4. Создайте проект:
   ```bash
   ng new my-first-app
   cd my-first-app
   ng serve
   ```

---

## Установка в macOS

1. Установите Homebrew
2. Установите Node.js: `brew install node@18`
3. Установите Git: `brew install git`
4. Установите Angular CLI: `npm install -g @angular/cli`

---

## Установка в Linux

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install curl build-essential git
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs
sudo npm install -g @angular/cli
```

## Устранение проблем

### Ошибки прав npm
```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
```

### Порт 4200 занят
```bash
ng serve --port 4201
```

### Совместимость версий

| Angular | Node.js |
|---------|---------|
| Angular 17 | 18.x, 20.x |
| Angular 16 | 16.x, 18.x |
