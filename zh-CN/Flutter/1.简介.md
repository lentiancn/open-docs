# Flutter 简介

Flutter 是 Google 推出的开源 UI 开发工具包。

## 什么是 Flutter？

Flutter 是一个用于构建跨平台应用的 UI 工具包，可以让开发者使用一套代码同时构建 iOS、Android、Web 和桌面应用。Flutter 使用 Dart 语言编写，以其丰富的组件库和高性能渲染而闻名。

## 主要特性

- **跨平台开发**：一套代码同时支持 iOS、Android、Web、桌面
- **热重载**：修改代码即时预览，无需重启应用
- **丰富组件**：Material Design 和 Cupertino 风格组件
- **高性能**：使用 Skia 图形引擎，原生性能体验
- ** expressive UI**：灵活的声明式 UI 构建方式
- **开源免费**：完全开源，免费使用

## 核心技术

### Dart 语言

Flutter 使用 Dart 语言开发，Dart 是一种面向对象的编程语言，具有以下特点：

- **强类型**：静态类型检查，减少运行时错误
- **JIT & AOT**：支持 JIT 即时编译和 AOT 预编译
- **垃圾回收**：自动内存管理
- **协程**：异步编程支持

### Widget 系统

Flutter 的核心是 Widget，所有 UI 元素都是 Widget：

- **StatelessWidget**：无状态组件
- **StatefulWidget**：有状态组件
- **InheritedWidget**：数据传递
- **RenderObjectWidget**：渲染对象

## 应用场景

- **移动应用**：iOS 和 Android 原生体验
- **Web 应用**：响应式 Web 应用
- **桌面应用**：Windows、macOS、Linux 桌面应用
- **嵌入式**：IoT 设备界面
