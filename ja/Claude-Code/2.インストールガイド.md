# Claude Code インストールガイド

Claude Codeのインストール方法を詳しく紹介。

## Windows インストール

### 方法1：VS Code 拡張機能

1. VS Codeを開く
2. "Claude Code"拡張機能を検索
3. インストールをクリック
4. Anthropicアカウントにログイン

### 方法2：スタンドアロンクライアント

1. https://claude.com/codeにアクセス
2. Windowsクライアントをダウンロード
3. インストーラーを実行
4. ログイン

## macOS インストール

### 方法1：VS Code 拡張機能

1. VS Codeを開く
2. "Claude Code"拡張機能を検索
3. インストールをクリック
4. ログイン

### 方法2：Homebrew

```
brew install claude-code
```

### 方法3：スタンドアロンクライアント

1. https://claude.com/codeにアクセス
2. macOSクライアントをダウンロード
3. インストーラーを実行
4. ログイン

## Linux インストール

### 方法1：VS Code 拡張機能

1. VS Codeを開く
2. "Claude Code"拡張機能を検索
3. インストールをクリック
4. ログイン

### 方法2：パッケージマネージャー

**Ubuntu/Debian**:
```
curl -fsSL https://downloads.anthropic.com/claude-code/install.sh | sh
```

**Arch Linux**:
```
yay -S claude-code
```

## インストールを確認

```
claude --version
```

## 設定

### APIキー

1. https://console.anthropic.comにアクセス
2. APIキーを作成
3. Claude Codeで設定

### 環境変数

```
export ANTHROPIC_API_KEY=your-api-key
```

## アンインストール

### macOS

```
brew uninstall claude-code
```

### Linux

```
sudo rm -rf /usr/local/claude-code
```
