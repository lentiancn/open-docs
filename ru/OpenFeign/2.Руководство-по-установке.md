# OpenFeign Руководство по установке

## Предварительные требования

- Java Development Kit (JDK) 8 или выше
- Инструмент сборки Maven или Gradle
- IDE (IntelliJ IDEA, Eclipse или VS Code)

---

## Использование Maven

Добавьте зависимость OpenFeign в ваш `pom.xml`:

```xml
<dependencies>
    <!-- OpenFeign Core -->
    <dependency>
        <groupId>io.github.openfeign</groupId>
        <artifactId>feign-core</artifactId>
        <version>13.2.1</version>
    </dependency>
    
    <!-- OkHTTP Client (опционально, но рекомендуется) -->
    <dependency>
        <groupId>io.github.openfeign</groupId>
        <artifactId>feign-okhttp</artifactId>
        <version>13.2.1</version>
    </dependency>
</dependencies>
```

---

## Использование Gradle

Добавьте зависимости в ваш `build.gradle`:

```groovy
dependencies {
    implementation 'io.github.openfeign:feign-core:13.2.1'
    implementation 'io.github.openfeign:feign-okhttp:13.2.1'
}
```

---

## Интеграция со Spring Cloud

Для приложений Spring Boot:

```xml
<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-openfeign</artifactId>
    <version>4.1.0</version>
</dependency>
```

Активируйте Feign-клиенты в вашем приложении:

```java
@SpringBootApplication
@EnableFeignClients
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
```

---

## Проверка установки

Создайте простой Feign-клиент для проверки:

```java
@FeignClient(name = "example-service", url = "https://api.example.com")
public interface ExampleClient {
    
    @RequestMapping(method = RequestMethod.GET, value = "/users/{id}")
    User getUser(@PathVariable("id") Long id);
}
```
