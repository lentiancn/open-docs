# Vue.js

Vue (произносится как /vjuː/, похоже на "view") — это JavaScript-фреймворк для создания пользовательских интерфейсов. Он основан на стандартных HTML, CSS и JavaScript и предоставляет декларативную, компонентную модель программирования, которая помогает эффективно разрабатывать пользовательские интерфейсы любой сложности.

## Основные функции

- **Доступность**: Основан на стандартных веб-технологиях с интуитивным API и документацией мирового класса
- **Производительность**: Действительно реактивная, оптимизированная компилятором система рендеринга
- **Универсальность**: Богатая, постепенно внедряемая экосистема

## Прогрессивный фреймворк

Vue спроектирован так, чтобы быть гибким и постепенно внедряемым, подходящим для различных сценариев:

- Улучшение статического HTML без этапа сборки
- Встраивание как Web Components на любой странице
- Одностраничное приложение (SPA)
- Fullstack/Серверный рендеринг (SSR)
- Jamstack/Генерация статических сайтов (SSG)
- Приложения для десктопа, мобильных устройств, WebGL и терминала

## Ключевые концепции

### Декларативный рендеринг
Расширяет стандартный HTML с помощью синтаксиса шаблонов для декларативного описания HTML-вывода на основе состояния JavaScript.

### Реактивность
Автоматически отслеживает изменения состояния JavaScript и эффективно обновляет DOM при возникновении изменений.

### Компоненты в одном файле (SFC)
Инкапсулирует логику компонента (JavaScript), шаблон (HTML) и стили (CSS) в одном файле *.vue.

### Стили API
- **Options API**: Использует объекты с опциями data, methods, mounted для описания компонентов
- **Composition API**: Использует импортированные функции API для описания логики компонента, обычно с `<script setup>`

## Быстрый старт

### Подход через CDN (сборка не требуется)
```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<div id="app">{{ message }}</div>

<script>
const { createApp } = Vue

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

### Официальный каркас (рекомендуется)
```bash
npm create vue@latest
cd <project-name>
npm install
npm run dev
```
