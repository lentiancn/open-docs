> 组件库详细使用说明

## 基础组件

### 1.1 按钮组件

```html
<el-button>默认按钮</el-button>
<el-button type="primary">主要按钮</el-button>
<el-button type="success">成功按钮</el-button>
<el-button type="warning">警告按钮</el-button>
<el-button type="danger">危险按钮</el-button>
```

#### 按钮属性

| 属性 | 说明 | 类型 | 可选值 | 默认值 |
|------|------|------|--------|--------|
| type | 按钮类型 | string | primary/success/warning/danger/info | - |
| plain | 朴素按钮 | boolean | - | false |
| round | 圆角按钮 | boolean | - | false |
| disabled | 禁用 | boolean | - | false |
| size | 尺寸 | string | medium/small/mini | - |

### 1.2 输入框

```html
<el-input v-model="input" placeholder="请输入内容"></el-input>
```

#### 输入框属性

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| v-model | 绑定值 | string | - |
| placeholder | 占位符 | string | - |
| disabled | 禁用 | boolean | false |
| clearable | 可清空 | boolean | false |
| show-password | 密码切换 | boolean | false |

## 表单组件

### 2.1 表单验证

```html
<el-form :model="form" :rules="rules" ref="ruleForm">
  <el-form-item label="用户名" prop="username">
    <el-input v-model="form.username"></el-input>
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="submitForm">提交</el-button>
  </el-form-item>
</el-form>
```

```javascript
export default {
  data() {
    return {
      form: { username: '' },
      rules: {
        username: [
          { required: true, message: '请输入用户名', trigger: 'blur' }
        ]
      }
    }
  },
  methods: {
    submitForm() {
      this.$refs.ruleForm.validate(valid => {
        if (valid) {
          console.log('提交成功')
        }
      })
    }
  }
}
```

## 表格组件

### 3.1 基础表格

```html
<el-table :data="tableData" stripe border>
  <el-table-column prop="date" label="日期" width="180"></el-table-column>
  <el-table-column prop="name" label="姓名" width="180"></el-table-column>
  <el-table-column prop="address" label="地址"></el-table-column>
</el-table>
```

```javascript
data() {
  return {
    tableData: [
      { date: '2024-01-01', name: '张三', address: '北京市' }
    ]
  }
}
```

### 3.2 带操作列

```html
<el-table-column label="操作" width="180">
  <template slot-scope="scope">
    <el-button size="mini" @click="handleEdit(scope.row)">编辑</el-button>
    <el-button size="mini" type="danger" @click="handleDelete(scope.row)">删除</el-button>
  </template>
</el-table-column>
```

## 对话框和通知

### 4.1 对话框

```html
<el-button type="text" @click="dialogVisible = true">打开对话框</el-button>

<el-dialog title="提示" :visible.sync="dialogVisible" width="30%">
  <span>这是一段信息</span>
  <span slot="footer">
    <el-button @click="dialogVisible = false">取消</el-button>
    <el-button type="primary" @click="dialogVisible = false">确定</el-button>
  </span>
</el-dialog>
```

### 4.2 消息提示

```javascript
this.$message.success('操作成功')
this.$message.warning('警告信息')
this.$message.error('操作失败')
this.$message.info('提示信息')
```
