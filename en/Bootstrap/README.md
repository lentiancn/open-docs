# Bootstrap

Bootstrap is the world's most popular open-source front-end framework for HTML, CSS, and JavaScript. Originally developed at Twitter, it provides a rich set of components and powerful JavaScript plugins that help developers quickly build responsive web applications.

## Quick Start

### CDN Include

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### npm Install

```bash
npm install bootstrap@5
```

## Core Features

- **Responsive layout** — 12-column grid system that adapts to any screen
- **Rich components** — navbars, cards, forms, modals, and more
- **JavaScript plugins** — dropdowns, carousels, accordions, and more
- **Sass support** — easy theme customization
