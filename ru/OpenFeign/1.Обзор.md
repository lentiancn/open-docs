# OpenFeign Обзор

> Декларативный HTTP-клиент для Java

---

## Описание проекта

OpenFeign — это библиотека Java, предоставляющая возможности декларативного REST-клиента. Она упрощает вызовы HTTP API, позволяя разработчикам определять интерфейсы HTTP-сервисов с помощью аннотаций.

---

## Основные функции

- **Декларативный API**: Определение HTTP-интерфейсов с помощью аннотаций
- **Балансировка нагрузки**: Встроенная поддержка балансировки нагрузки на стороне клиента
- **Обнаружение сервисов**: Интеграция с Eureka
- **Сжатие**: Поддержка GZIP для эффективной передачи данных
- **Логирование**: Детальное логирование HTTP-запросов и ответов
- **Обработка ошибок**: Пользовательская обработка ошибок с механизмами fallback
- **Перехватчики**: Поддержка пользовательских перехватчиков запросов/ответов

---

## Варианты использования

- Коммуникация микросервисов
- Разработка REST API-клиентов
- Интеграция с внешними API
- Вызовы между сервисами в распределённых системах
