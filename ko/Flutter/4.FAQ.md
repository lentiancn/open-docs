# Flutter - FAQ

자주 묻는 질문.

## 설치

### 다운로드 실패?

```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
```

### flutter doctor 오류?

PATH 확인.

### Android SDK 없음?

```bash
flutter config --android-sdk /경로/-android-sdk
```

## 개발

### 핫 리로드 작동 안 함?

저장하고 기다리기.

### APK 빌드 실패?

```bash
flutter clean
```
