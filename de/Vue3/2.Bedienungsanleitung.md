# Vue 3 Bedienungsanleitung

Anleitung zur effektiven Nutzung von Vue 3.

---

## Composition API

```javascript
import { ref } from 'vue'

const count = ref(0)
count.value++
```

---

## Template

```html
<h1>{{ title }}</h1>
<button @click="handleClick">Klicken</button>
```

---

## Komponenten

```vue
<script setup>
defineProps({
  title: String
})
</script>
```
