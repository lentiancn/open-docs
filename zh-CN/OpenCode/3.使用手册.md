# OpenCode 使用手册

基本使用方法和技巧。

## 基本操作

### 打开文件

```
opencode filename
opencode .
```

### 保存文件

```
Ctrl + S
```

### 退出

```
Ctrl + Q
```

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl + O | 打开文件 |
| Ctrl + S | 保存 |
| Ctrl + F | 搜索 |
| Ctrl + H | 替换 |
| Ctrl + G | 跳转到行 |
| Ctrl + Q | 退出 |

## 搜索和替换

### 搜索

```
Ctrl + F
```

### 替换

```
Ctrl + H
```

## 多文件操作

### 标签页

- 新建标签：Ctrl + T
- 关闭标签：Ctrl + W
- 切换标签：Ctrl + Tab

## 插件管理

### 安装插件

```
opencode --install-plugin plugin-name
```

### 列出插件

```
opencode --list-plugins
```

## 设置

### 配置文件

主配置文件：`~/.opencode/config.json`

### 主题设置

```json
{
  "theme": "monokai"
}
```
