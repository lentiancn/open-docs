# Element UI Обзор

Библиотека компонентов Vue.js 2.0 для десктопа.

## Что такое Element UI?

Element UI — это библиотека компонентов Vue.js 2.0 с открытым исходным кодом, разработанная командой Ele.me, предоставляющая полный набор элегантных UI-компонентов для корпоративных систем управления.

### Основные функции

- **Богатые компоненты**: 60+ часто используемых компонентов
- **Красивый дизайн**: Соответствует рекомендациям Material Design
- **Полная документация**: Подробная документация и примеры
- **Поддержка TypeScript**: Полные определения типов TypeScript
- **Tree Shaking**: Поддержка импорта по требованию
- **Настройка тем**: Поддержка переменных SCSS
- **Интернационализация**: Поддержка нескольких языков
- **Адаптивный дизайн**: Поддержка различных размеров экрана

### Категории компонентов

- **Базовые**: Button, Layout, Container
- **Формы**: Input, Select, Cascader, Form
- **Данные**: Table, Tree, Pagination
- **Уведомления**: Alert, Message, Notification
- **Навигация**: Menu, Tabs, Breadcrumb
- **Другие**: Dialog, Drawer, Tooltip, Upload
