# MongoDB Руководство пользователя

## Основные понятия

### База данных

База данных — это контейнер верхнего уровня для данных в MongoDB.

### Коллекция

Коллекция — это группа документов MongoDB, аналогичная таблице в реляционной базе данных.

### Документ

Документ — это основная единица данных в MongoDB, состоящая из пар поле-значение.

## Подключение к MongoDB

### Подключение к локальной базе данных

```bash
mongosh
```

## Операции с базой данных

### Просмотр баз данных

```bash
show dbs
```

### Переключение/создание базы данных

```bash
use mydatabase
```

## Операции с коллекциями

### Просмотр коллекций

```bash
show collections
```

### Создание коллекции

```bash
db.createCollection("mycollection")
```

## Вставка документов

### Вставка одного документа

```bash
db.users.insertOne({
  name: "Иван",
  age: 30,
  email: "ivan@example.ru"
})
```

### Вставка нескольких документов

```bash
db.users.insertMany([
  { name: "Мария", age: 25 },
  { name: "Петр", age: 35 }
])
```

## Запрос документов

### Запрос всех документов

```bash
db.users.find()
```

### Запрос с условиями

```bash
db.users.find({ age: 30 })
db.users.find({ age: { $gte: 25 } })
```

## Обновление документов

```bash
db.users.updateOne(
  { name: "Иван" },
  { $set: { age: 31 } }
)
```

## Удаление документов

```bash
db.users.deleteOne({ name: "Иван" })
```

## Агрегация

```bash
db.users.aggregate([
  { $match: { age: { $gte: 25 } } },
  { $group: { _id: "$city", total: { $sum: 1 } } }
])
```

## Индексы

### Создание индекса

```bash
db.users.createIndex({ name: 1 })
db.users.createIndex({ age: 1, name: 1 })
```

## Операции Replica Set

### Инициализация

```bash
rs.initiate()
```

## Операции шардинга

### Включение шардинга

```bash
sh.enableSharding("mydatabase")
```

## Импорт и экспорт данных

### Экспорт с mongodump

```bash
mongodump --db=mydatabase --out=/backup
```

### Импорт с mongorestore

```bash
mongorestore --db=mydatabase /backup/mydatabase
```

## Административные команды

### Статус сервера

```bash
db.serverStatus()
```

## Транзакции

MongoDB 4.0+ поддерживает мультидокументные транзакции.

## Change Streams

Change Streams позволяют приложениям отслеживать изменения данных в реальном времени.

## GridFS

GridFS используется для хранения файлов, превышающих ограничение в 16 МБ.
