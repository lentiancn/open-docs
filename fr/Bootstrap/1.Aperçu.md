# Bootstrap Aperçu

Le framework CSS frontend le plus populaire au monde.

## Qu'est-ce que Bootstrap?

Bootstrap est le framework CSS le plus populaire, développé par Twitter, pour créer des sites web responsives.

### Fonctionnalités principales

- **Grille responsive** - Grille 12 colonnes
- **Composants prêts** - Navbars, cartes, modales
- **Utilitaires CSS** - Ajustements rapides
- **Composants JavaScript** - Composants interactifs
