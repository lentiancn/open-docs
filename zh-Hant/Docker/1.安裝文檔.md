# Docker 安裝文檔

## Linux

```bash
# Ubuntu/Debian
curl -fsSL https://get.docker.com | sh
sudo systemctl start docker
sudo systemctl enable docker

# CentOS/RHEL
sudo yum install -y docker-ce
sudo systemctl start docker
sudo systemctl enable docker
```

## macOS

```bash
# Intel
brew install --cask docker

# Apple Silicon
brew install --cask docker
```

## Windows

下載 [Docker Desktop](https://www.docker.com/products/docker-desktop) 安裝。

## 驗證

```bash
docker --version
docker-compose --version
```
