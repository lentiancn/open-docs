# Java 安裝指南

> 環境設定

---

## 需求

| 環境 | 需求 |
|------|------|
| 作業系統 | Windows, macOS, Linux |
| 記憶體 | 至少 2GB |

---

## 安裝

### Windows

```powershell
choco install openjdk21
```

### macOS

```bash
brew install openjdk@21
```

### Linux

```bash
sudo apt install openjdk-21-jdk
```

---

## 環境變數

```
JAVA_HOME = /usr/lib/jvm/java-21-openjdk
```

---

## 驗證

```bash
java -version
```
