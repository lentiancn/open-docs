# Часто задаваемые вопросы о Nginx

## Установка

### Как проверить версию
```bash
nginx -v
```

### Как проверить конфигурацию
```bash
nginx -t
```

## Конфигурация

### Перезагрузка
```bash
nginx -s reload
```

## Производительность

### Оптимизация
```nginx
worker_processes auto;
worker_connections 4096;
gzip on;
```

## Безопасность

### Включить HTTPS
```nginx
server {
    listen 443 ssl;
    ssl_certificate cert.crt;
    ssl_certificate_key cert.key;
}
```

### Ограничение частоты
```nginx
limit_req_zone $binary_remote_addr zone=one:10m rate=10r/s;
```

## Частые ошибки

- **400**: Проверить размер запроса
- **403**: Проверить права доступа
- **404**: Проверить путь
- **502**: Бэкенд-сервис не запущен
- **504**: Таймаут, увеличить proxy_read_timeout

## Прокси

### Получить реальный IP
```nginx
proxy_set_header X-Real-IP $remote_addr;
```
