# Element UI インストールガイド

Element UIはVue 2.xアプリケーション用のUIコンポーネントライブラリです。

## バージョンの概要

| バージョン | Vueバージョン | ステータス | リリース日 |
|------------|---------------|------------|------------|
| Element UI 2.x | Vue 2.6+ | 現行 | 2020 |
| Element UI 1.x | Vue 2.x | レガシー | 2016 |

**注意：** Element UIはVue 2専用です。Vue 3の場合はElement Plusを使用してください。

### Element UI と Element Plus の比較

| 機能 | Element UI (Vue 2) | Element Plus (Vue 3) |
|------|--------------------|----------------------|
| Vueバージョン | 2.x | 3.x |
| TypeScript | 一部対応 | 完全サポート |
| ステータス | メンテナンスモード | アクティブな開発 |

## 前提条件

### システム要件

| 要件 | バージョン |
|------|------------|
| Node.js | 8.0以上 |
| npm | 5.0以上 |
| Vue | 2.6以上 |
| ブラウザ | IE 10+, Chrome, Firefox, Safari |

### 必要なソフトウェア

- **Node.js** - JavaScriptランタイム
- **npm** - パッケージマネージャー
- **Vue CLI** - プロジェクトスキャフォールド（オプション）

## インストール方法

### 方法1: npm（推奨）

```bash
# Element UIをインストール
npm install element-ui --save
```

### 方法2: Vue CLIを使用

```bash
# Vue CLIをインストール
npm install -g @vue/cli

# Element UIテンプレートでプロジェクトを作成
vue init webpack my-element-app
```

### 方法3: CDN

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Element UI CDNデモ</title>
  <!-- Vue 2をインポート -->
  <script src="https://unpkg.com/vue@2.6.14/dist/vue.js"></script>
  <!-- Element UIスタイルをインポート -->
  <link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
  <!-- Element UIをインポート -->
  <script src="https://unpkg.com/element-ui/lib/index.js"></script>
</head>
<body>
  <div id="app">
    <el-button @click="visible = true">ダイアログを開く</el-button>
    <el-dialog :visible.sync="visible" title="Hello Element UI">
      <p>Element UIへようこそ！</p>
    </el-dialog>
  </div>

  <script>
    new Vue({
      el: '#app',
      data: {
        visible: false
      }
    })
  </script>
</body>
</html>
```

## プロジェクト設定

### ステップ1: Vueプロジェクトを作成

```bash
# Vue CLIを使用
npm install -g @vue/cli
vue create my-element-app
cd my-element-app
```

### ステップ2: Element UIをインストール

```bash
npm install element-ui --save
```

### ステップ3: main.jsでインポート

#### 完全インポート

```javascript
import Vue from 'vue'
import App from './App.vue'

// Element UIをインポート
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

// Element UIを使用
Vue.use(ElementUI)

Vue.config.productionTip = false

new Vue({
  render: h => h(App)
}).$mount('#app')
```

#### 必要に応じてインポート

まず、プラグインをインストールします：

```bash
npm install babel-plugin-component -D
```

babel.config.jsを設定：

```javascript
module.exports = {
  presets: ['@vue/cli-plugin-babel/preset'],
  plugins: [
    [
      'component',
      {
        libraryName: 'element-ui',
        styleLibraryName: 'theme-chalk'
      }
    ]
  ]
}
```

次にコンポーネントをインポート：

```javascript
import { Button, Dialog, Input } from 'element-ui'

Vue.component(Button.name, Button)
Vue.component(Dialog.name, Dialog)
Vue.component(Input.name, Input)

// またはコンポーネント内でローカルに使用
export default {
  components: {
    [Button.name]: Button
  }
}
```

## トラブルシューティング

### コンポーネントが見つからない

**問題：** コンポーネントがレンダリングされない

**解決策：**
1. インストール確認：`npm list element-ui`
2. main.jsでのインポートを確認
3. Vue.use()が呼び出されていることを確認

### スタイルが適用されない

**問題：** スタイルが欠落している

**解決策：**
1. CSSをインポート：`import 'element-ui/lib/theme-chalk/index.css'`
2. webpack CSSローダー設定を確認

### IE10互換性

```bash
# polyfillをインストール
npm install babel-polyfill --save
```

```javascript
// main.js
import 'babel-polyfill'
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```

## Element Plus (Vue 3)への移行

Vue 3に移行する場合は、Element Plusを使用してください：

```bash
npm install element-plus --save

主な変更点：
1. パッケージ名：`element-ui` → `element-plus`
2. インポートパス：`element-ui/lib/index` → `element-plus`
3. 一部のコンポーネントプロパティが変更されている場合があります
```

## ベストプラクティス

1. **必要に応じたインポートを使用** - バンドルサイズを削減
2. **テーマのカスタマイズ** - SCSS変数を使用
3. **Vue 2.6+を使用** - より良い互換性
4. **babelを設定** - 本番ビルド用
