# OpenClaw 🦞

OpenClaw 是一款能够将常用聊天软件与 AI 编程助手连接的工具。它部署在你的自有设备上运行，通过一个核心进程把你的消息应用和 AI 服务串联起来，让你可以直接在微信、Telegram、Discord 等平台上与 AI 对话。

## 适合谁用

如果你想拥有自己的 AI 助手，但又不想把数据交给第三方服务，OpenClaw 是个不错的选择。它特别适合习惯使用命令行的开发者和对效率有要求的用户。

## 主要特点

- **数据自主**：所有内容保存在你的设备上，不经过第三方服务器
- **多平台接入**：一个程序同时支持多个聊天渠道
- **代理原生设计**：专门为编程类 AI 助手打造，支持工具调用、对话管理、记忆功能和多代理调度
- **开源透明**：代码公开，社区共建

## 使用条件

- Node.js 22 或更高版本
- 一个 AI 服务商提供的 API 密钥
- 几分钟的初始配置时间

建议选用最新一代的高性能模型，以获得最佳体验。

## 快速上手

```bash
# 安装
npm install -g openclaw@latest

# 初始化设置
openclaw onboard --install-daemon

# 启动服务
openclaw gateway --port 18789
```

服务启动后，打开浏览器访问本地地址即可开始使用。

## 工作方式

OpenClaw 的核心是一个 Gateway 进程，它负责处理所有渠道的连接、AI 调用的转发、以及对话状态的维护。你只需要维护这一个进程，就能同时在多个平台上使用 AI 助手。

## 核心能力

- **统一入口**：一套系统管理所有聊天渠道
- **插件扩展**：通过扩展包还能支持更多平台
- **多代理支持**：可以为不同用途配置独立的 AI 实例
- **文件交互**：支持收发图片、音频、文档
- **网页管理界面**：在浏览器中管理配置、查看对话历史
- **移动设备联动**：手机安装配套应用后可远程控制、拍照、截屏
