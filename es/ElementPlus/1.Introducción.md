# Element Plus Introducción

Element Plus es una biblioteca de componentes UI basada en Vue 3.

## Características

- Componentes Vue 3
- 70+ componentes
- TypeScript
- Importación bajo demanda
- Internacionalización
- Temas personalizables

## Escenarios

- Sistemas de gestión empresarial
- Plataformas de comercio electrónico
- Sistemas CRM

Para más información, visite el [sitio web oficial](https://element-plus.org/).
