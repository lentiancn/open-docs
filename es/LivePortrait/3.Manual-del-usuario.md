# Manual del Usuario de LivePortrait

Este documento proporciona instrucciones detalladas de uso para LivePortrait, incluyendo inferencia básica, interfaz Gradio, modo animales, edición de video y otras funciones.

## Inferencia Básica

### Control de Imagen

El uso más básico implica usar una imagen estática como fuente y un video de control como referencia de animación:

```bash
python inference.py
```

De manera predeterminada, el script usa imágenes de ejemplo del directorio assets/examples/source y videos de control de ejemplo del directorio assets/examples/driving. Si se ejecuta correctamente, los resultados se guardan en el directorio de animaciones con el nombre "archivo_fuente--archivo_control_concat.mp4".

El video de salida contiene tres partes: el video de control, la imagen/video fuente y el resultado de animación generado, unidos en secuencia.

### Entradas Personalizadas

Puede especificar sus propios archivos de fuente y control usando parámetros de línea de comandos:

```bash
# El archivo fuente es una imagen
python inference.py -s assets/examples/source/s9.jpg -d assets/examples/driving/d0.mp4

# El archivo fuente es un video
python inference.py -s assets/examples/source/s13.mp4 -d assets/examples/driving/d0.mp4
```

El parámetro -s especifica la ruta de la imagen o video fuente, mientras que el parámetro -d especifica la ruta del video de control.

### Ver Más Opciones

Vea todos los parámetros disponibles de línea de comandos:

```bash
python inference.py -h
```

Los parámetros incluyen ruta de salida, tamaño de imagen, velocidad de animación, método de ensamblaje y otras configuraciones detalladas.

## Usando la Interfaz Web Gradio

Para usuarios no familiarizados con interfaces de línea de comandos, LivePortrait ofrece una interfaz gráfica basada en Gradio.

### Lanzar Interfaz de Modo Humano

```bash
python app.py
```

Después de lanzar, abra su navegador a la URL mostrada (típicamente http://localhost:7860). En la interfaz, cargue una imagen o video fuente, luego seleccione un video de control y haga clic en el botón generar para obtener el resultado de animación.

Puede especificar el puerto usando el parámetro --server_port, generar un enlace público usando --share, y especificar la dirección del servidor usando --server_name.

### Lanzar Interfaz de Modo Animales

En sistemas Linux con GPUs NVIDIA, puede lanzar la interfaz de modo animales:

```bash
python app_animals.py
```

## Especificaciones del Video de Control

Para resultados óptimos, los videos de control deben cumplir con los siguientes requisitos:

**Relación de Aspecto**: Se recomienda recortarlo a cuadrado 1:1 (como 512x512 o 256x256 píxeles). También puede habilitar el recorte automático (--flag_crop_driving_video).

**Requisitos de Contenido**: El video de control debe enfocarse en el área de la cabeza, similar a los videos de ejemplo. Los movimientos de hombros deben minimizarse.

**Requisito del Primer Cuadro**: El primer cuadro del video de control debe ser una cara frontal con expresión neutral. Esto es crucial para asegurar efectos de animación precisos.

### Recorte Automático

Si su video de control no cumple con los requisitos anteriores, puede usar la función de recorte automático:

```bash
python inference.py -s assets/examples/source/s9.jpg -d assets/examples/driving/d13.mp4 --flag_crop_driving_video
```

Si los resultados del recorte automático no son satisfactorios, puede ajustar los parámetros --scale_crop_driving_video y --vy_ratio_crop_driving_video para afinar la escala y el desplazamiento.

## Usando Plantillas de Movimiento

Para acelerar la inferencia y proteger la privacidad, LivePortrait soporta el uso de archivos de plantilla de movimiento pregenerados (.pkl). Estos archivos de plantilla registran información de movimiento de los videos de control y pueden reutilizarse sin procesar el video original cada vez.

### Uso

```bash
# Animación de retrato
python inference.py -s assets/examples/source/s9.jpg -d assets/examples/driving/d5.pkl

# Edición de video de retrato
python inference.py -s assets/examples/source/s13.mp4 -d assets/examples/driving/d5.pkl
```

Las plantillas de plantilla no solo aceleran la inferencia sino que también protegen la privacidad—usar plantillas elimina la necesidad de subir videos de control originales.

## Edición de Video de Retrato (Video-to-Video)

LivePortrait soporta la edición y modificación de expresiones y movimientos de cabeza de sujetos en videos existentes.

### Uso Básico

```bash
# La fuente es imagen, el control es video
python inference.py -s assets/examples/source/s9.jpg -d assets/examples/driving/d0.mp4
```

### Ajustando Parámetros de Animación

**Intensidad de Expresión**: Ajuste la fuerza de los cambios de expresión usando el parámetro --driving_multiplier. Valores mayores a 1 mejoran los cambios de expresión, mientras que valores menores a 1 los reducen.

**Control de Ensamblaje**: Use el parámetro --flag_stitching para controlar si el ensamblaje está habilitado. Desactivar el ensamblaje puede alterar cómo se presentan las regiones faciales.

## Aceleración de Inferencia

### Aceleración Torch Compile

LivePortrait soporta el uso de la función torch.compile de PyTorch para acelerar la inferencia. La primera inferencia dispara un proceso de optimización (aproximadamente un minuto), después del cual la velocidad de inferencia mejora entre 20-30%. Los efectos de aceleración pueden variar según la versión de CUDA.

```bash
python app.py --flag_do_torch_compile
```

Nota: torch.compile no es compatible con Windows y macOS.

### Prueba de Velocidad

El proyecto proporciona un script de prueba de velocidad de inferencia:

```bash
python speed.py
```

Los resultados de la prueba muestran el tiempo de procesamiento de cada módulo, ayudándole a identificar cuellos de botella de rendimiento.

## Modo Animales

El modo animales soporta la animación de retratos de mascotas como gatos y perros.

### Condiciones de Uso

El modo animales ha sido probado solo en los siguientes entornos: Sistema Linux con GPU NVIDIA, Sistema Windows con GPU NVIDIA.

### Requisitos de Instalación

Antes de usar el modo animales, instale la dependencia X-Pose como se describe en la guía de instalación y descargue los pesos del modelo de animales.

### Ejecutando Inferencia en Modo Animales

```bash
python inference_animals.py -s assets/examples/source/s39.jpg -d assets/examples/driving/wink.pkl --driving_multiplier 1.75 --no_flag_stitching
```

### Interfaz de Modo Animales

```bash
python app_animals.py
```

## Control Regional

LivePortrait soporta el control de animación para regiones específicas de imágenes, permitiendo ajustes de efectos más refinados.

### Habilitando Control Regional

En la interfaz Gradio, puede encontrar opciones de control regional para seleccionar qué regiones faciales animar (como ojos, boca, pose de cabeza, etc.).

## Edición de Retratos

En la interfaz Gradio, los usuarios pueden editar con precisión varios atributos del retrato fuente, incluyendo expresión, dirección de la mirada, ángulo de cabeza, y más.

## Protección de Privacidad

LivePortrait proporciona múltiples funciones de protección de privacidad:

**Plantillas de Movimiento**: Use archivos de plantilla .pkl en lugar de videos de control originales para evitar subir contenido sensible

**Recorte Automático**: Recorte automáticamente los videos de control para enfocarse en la cabeza, reduciendo información de fondo

**Protección de Video**: Función de creación de plantillas convierte videos a plantillas de movimiento antes de usar

## Formato de Salida

El formato de salida predeterminado es un archivo de video MP4 que contiene tres partes: video de control, contenido fuente y resultado generado. Use el parámetro --no_flag_concat para salida solo la porción de animación generada.

## Integración de Herramientas Comunitarias

LivePortrait está integrado por múltiples proyectos comunitarios:

**ComfyUI**: Múltiples nodos ComfyUI soportan LivePortrait con características de vista previa en tiempo real

**Stable Diffusion WebUI**: La extensión sd-webui-live-portrait integra LivePortrait en SD WebUI

**FaceFusion**: La versión 3.0 integra LivePortrait como restaurador de expresiones y procesador de editor de cara

**HuggingFace Spaces**: Experimente LivePortrait en línea sin instalación local

## Mejores Prácticas

Para efectos de animación óptimos, siga estas mejores prácticas:

Seleccione imágenes fuente de alta calidad: Los retratos frontales claros producen los mejores resultados

Use videos de control profesionales: Videos con expresiones frontales y naturales conducen a mejores efectos

Controle la longitud del video de control: Videos de control excesivamente largos aumentan el tiempo de procesamiento y pueden causar degradación del efecto

Establezca parámetros razonablemente: Ajuste la intensidad de expresión y otros parámetros según los resultados reales

Mantenga iluminación consistente: Intente mantener las condiciones de iluminación consistentes entre las imágenes fuente y los videos de control
