# PlantUML Руководство по установке

## Предварительные требования

- Требуется Java 8 или выше
- Минимум 512MB RAM

---

## Установка

### Windows
Скачать plantuml.jar

### macOS
```bash
brew install plantuml
```

### Linux
```bash
sudo apt-get install plantuml
```

---

## Проверка
```bash
java -jar plantuml.jar -version
```
