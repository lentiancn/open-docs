# MultiTalk 소개

MultiTalk은 MeiGen-AI 팀에서 개발한 오디오 기반 다인물 대화 영상 생성 프레임워크입니다. 이 프로젝트는 기존 디지털 휴먼 생성 기술이 단일 인물의 애니메이션만 지원한다는 제약을 해결하고, 다중 스트림 오디오 입력, 참조 이미지 및 텍스트 프롬프트에 따라 여러 인물이 상호작용하는 영상을 생성할 수 있습니다.

## 프로젝트 배경

 talking head 및 talking body 생성과 같은 기존 오디오 기반 인물 애니메이션 방법은 동기화된 얼굴 움직임과 고품질 영상 생성에서 상당한 진전을 이루었습니다. 그러나 기존 방법은 주로 단일 인물 애니메이션에 중점을 두고 있으며 다중 스트림 오디오 입력 처리에 어려움이 있고, 오디오와 인물 간의 잘못된 바인딩 문제에 직면합니다. 또한 이러한 방법은 명령 Following 능력 측면에서 한계가 있습니다. MultiTalk 프로젝트는 이러한 과제를 해결하기 위해 만들어졌습니다.

## 핵심 기능

MultiTalk의 핵심 기능은 다인물 대화 영상의 자동 생성입니다. 사용자는 여러 인물의 대화가 포함된 오디오, 여러 인물이 포함된 참조 이미지, 영상 장면을 설명하는 텍스트 프롬프트만 제공하면 됩니다. 시스템은 자동으로 해당 대화 영상을 생성하며, 오디오와 동기화된 높은 입술 움직임과 부드럽고 자연스러운 시각 효과를 제공합니다.

## 기술적 특징

MultiTalk는 다인물 영상 생성의 과제를 해결하기 위해 여러 혁신적인 기술을 채택하고 있습니다. 첫째, 프로젝트는 다중 스트림 오디오 주입 시 오디오와 인물의 잘못된 바인딩 문제를 해결하기 위해 레이블 회전 위치 임베딩(L-RoPE) 방법을 도입합니다. 둘째, 부분 매개변수 훈련 및 다중 작업 훈련 전략을 통해 기본 모델의 명령 Following 능력을 효과적으로 보존합니다. 또한 프로젝트는 참조 이미지에서 인물 영역의 특성을 기반으로 생성 영상에서 해당 인물을 정확하게 위치시키는 적응형 인물Localisation 기능을 지원합니다.

## 적용 시나리오

MultiTalk는 다양한 적용 시나리오를 가지고 있습니다. 콘텐츠 창작에서 창작자는 교육 설명, 창작적 표현 또는 오락 콘텐츠 제작을 위해 이 도구를 사용하여 디지털 휴먼 대화 영상을 신속하게 생성할 수 있습니다. 연구에서 이 프로젝트는 다인물 영상 생성을 위한 새로운 연구 방향과 기술 벤치마크를 제공합니다. 비즈니스에서 기업은 가상 호스트, 다인물 고객 서비스, 디지털 휴먼 회의와 같은 시나리오에 적용할 수 있습니다.

## 성능

MultiTalk는 여러 벤치마크에서 우수한 성능을 보여줍니다. 이 프로젝트는 talking head, talking body 및 다인물 데이터 세트에서 최고 수준의 생성 결과를 달성하며, 입술 동기화 정확성이 업계 최고 수준에 도달합니다. 프로젝트는 480P 및 720P 해상도에서의 영상 생성을 지원하며 최대 15초의 비디오 클립을 생성할 수 있습니다.

## 프로젝트 정보

MultiTalk는 오픈소스 프로젝트이며, 코드와 모델 가중치는 GitHub 및 Hugging Face에서 구할 수 있습니다. 관련 논문은 arXiv에 발표되어 기술 방법과 실험 결과를상세하게 설명하고 있습니다. 프로젝트는 Apache 2.0 라이선스를 사용하며 학술 및 상업적 사용이 허용됩니다.
