# Vue インストールガイド

Vueは、ユーザインターフェースを構築するためのプログレッシブJavaScriptフレームワークです。

## バージョンの概要

Vueには2つのメジャーバージョンがあります：

| バージョン | 名前 | JavaScript バージョン | ステータス |
|------------|------|---------------------|-----------|
| Vue 3 | - | ES2015+ | 推奨 |
| Vue 2 | - | ES5 | レガシー |

**推奨：** すべての新規プロジェクトにVue 3を使用してください。

## 前提条件

### システムの要件

| 要件 | Vue 2 | Vue 3 |
|------|-------|-------|
| Node.js | 8.9+ | 16+ (18+推奨) |
| npm | 5.6+ | 8+ |

### 必要なソフトウェア

- **Node.js** - JavaScriptランタイム
- **npm** - パッケージマネージャー

## インストール方法

### 方法1: npmを使用（推奨）

```bash
# Vue 3プロジェクトを作成
npm create vue@latest
```

### 方法2: Viteを使用（Vue 3推奨）

```bash
# ViteでVue 3プロジェクトを作成
npm create vite@latest my-vue-app -- --template vue

cd my-vue-app
npm install
npm run dev
```

### 方法3: CDN

```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
```

## プロジェクト設定

### Vue 3 + Viteプロジェクト

my-vue-app/

## 依存関係のインストール

```bash
npm install
npm install vue-router@4
npm install pinia
```

## Vue Routerの設定

```javascript
import { createRouter, createWebHistory } from 'vue-router'
import Home from './views/Home.vue'

const routes = [
  { path: '/', component: Home }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```

## Pinia（状態管理）

```javascript
import { defineStore } from 'pinia'

export const useCounterStore = defineStore('counter', {
  state: () => ({
    count: 0
  }),
  actions: {
    increment() {
      this.count++
    }
  }
})
```

## 開発サーバー

```bash
# 開発サーバーを起動
npm run dev

# 本番用にビルド
npm run build
```

## トラブルシューティング

### npm installが失敗する場合

```bash
# npmキャッシュをクリア
npm cache clean --force

# node_modulesを削除して再インストール
rm -rf node_modules
npm install
```

## ベストプラクティス

1. **Vue 3を使用** - 新規プロジェクトにはVue 3を
2. **Viteを使用** - より速い開発とビルド
3. **Composition APIを使用** - より良いTypeScriptサポート
