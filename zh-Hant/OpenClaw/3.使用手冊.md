# OpenClaw 使用手冊

本文檔介紹 OpenClaw 的日常使用方法和各項功能。

## 核心概念

### Gateway

Gateway 是 OpenClaw 的核心組件，所有的訊息轉發、AI 調用和渠道管理都通過它完成。你只需要運行這一個進程，它會自動處理來自各個聊天平台的請求。

### 渠道

渠道指的是你用來和 AI 交流的聊天應用。OpenClaw 支援同時連接多個渠道，包括 WhatsApp、Telegram、Discord 等。

### 代理

代理是 AI 助手的獨立實例。你可以配置多個代理，分別處理不同的工作任務，每個代理有獨立的對話歷史和工作區。

## 日常操作

### 啟動和停止服務

```bash
# 啟動服務（後台運行）
openclaw gateway start

# 停止服務
openclaw gateway stop

# 重啟服務
openclaw gateway restart

# 查看運行狀態
openclaw gateway status

# 前台運行（調試時使用）
openclaw gateway --port 18789
```

### 發送訊息

```bash
# 發送私人訊息
openclaw message send --target +15555550123 --message "測試訊息"

# 發送群組訊息
openclaw message send --target "群名稱" --message "大家好"
```

### 配置管理

OpenClaw 使用 JSON5 格式的配置文件，位於 `~/.openclaw/openclaw.json`。

修改配置的方式：

1. **精靈模式**：
   ```bash
   openclaw onboard
   openclaw configure
   ```

2. **命令列模式**：
   ```bash
   openclaw config get agents.defaults.workspace
   openclaw config set agents.defaults.heartbeat.every "2h"
   ```

3. **網頁介面**：
   開啟 http://127.0.0.1:18789/ ，在 Config 標籤頁中修改

4. **直接編輯**：
   ```bash
   nano ~/.openclaw/openclaw.json
   ```

### 熱重載

修改配置文件後無需手動重啟服務，大部分配置會自動生效。

## 常用配置示例

### 限制可訪問的用戶

```json5
{
  channels: {
    whatsapp: {
      allowFrom: ["+15555550123"]
    }
  }
}
```

### 設置群組訊息必須@

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

### 配置多個獨立代理

```json5
{
  agents: {
    list: [
      { id: "home", default: true, workspace: "~/.openclaw/workspace-home" },
      { id: "work", workspace: "~/.openclaw/workspace-work" }
    ]
  }
}
```

### 定時任務

```json5
{
  cron: {
    enabled: true,
    maxConcurrentRuns: 2
  }
}
```

### 定期心跳檢測

```json5
{
  agents: {
    defaults: {
      heartbeat: {
        every: "30m",
        target: "last"
      }
    }
  }
}
```

### 沙箱隔離

```json5
{
  agents: {
    defaults: {
      sandbox: {
        mode: "non-main",
        scope: "agent"
      }
    }
  }
}
```

### Webhook 設置

```json5
{
  hooks: {
    enabled: true,
    token: "你的密鑰",
    path: "/hooks"
  }
}
```

## 渠道管理

### 添加新渠道

```bash
# 登入 WhatsApp
openclaw channels login whatsapp

# 登入 Telegram
openclaw channels login telegram
```

### 查看已連接的渠道

```bash
openclaw channels list
```

## 問題排查

### 查看日誌

```bash
# 查看服務日誌
openclaw logs

# 即時查看日誌
openclaw logs -f
```

### 健康檢查

```bash
openclaw health
```

### 自動診斷

```bash
openclaw doctor
```

## 網頁控制台

你可以通過網頁介面管理 OpenClaw：

- 發送和接收訊息
- 修改各項配置
- 查看對話歷史
- 管理手機節點
- 監控服務狀態

訪問位址：http://127.0.0.1:18789/
