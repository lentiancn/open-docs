# Angular 安裝指南

## 環境要求

- Node.js：18.10.0+ 或 20.0.0+
- npm：9.0.0+

## 安裝步驟

### 1. 安裝 Node.js

從 https://nodejs.org 下載安裝包，或使用 nvm：

```bash
nvm install --lts
nvm use --lts
```

### 2. 安裝 Angular CLI

```bash
npm install -g @angular/cli
```

### 3. 創建新項目

```bash
ng new my-app
cd my-app
ng serve
```

## 常用命令

```bash
# 創建組件
ng g c my-component

# 創建服務
ng g s my-service

# 構建生產版本
ng build
```

## 項目結構

```
my-app/
├── src/
│   ├── app/
│   │   ├── app.component.ts
│   │   └── pages/
│   ├── assets/
│   ├── environments/
│   └── styles.scss
├── angular.json
└── package.json
```

## 常見問題

### npm 安裝慢？

```bash
npm config set registry https://registry.npmmirror.com
```

## 卸載

```bash
npm uninstall -g @angular/cli
```
