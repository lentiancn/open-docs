# Element UI Preguntas Frecuentes

Preguntas y respuestas comunes.

## Problemas de instalación

### Q: ¿La instalación falló?

**Soluciones:**
1. Verificar versión de Node.js
```bash
node --version  # Requiere 12.0.0+
```
2. Limpiar caché de npm
```bash
npm cache clean --force
```
3. Usar espejo de Taobao
```bash
npm config set registry https://registry.npmmirror.com
```
4. Eliminar node_modules y reinstallar
```bash
rm -rf node_modules
npm install
```

### Q: ¿Módulo no encontrado?

**Soluciones:**
1. Confirmar que element-ui está instalado
```bash
npm list element-ui
```
2. Verificar package.json para dependencias

### Q: ¿Versión de Vue incompatible?

**Soluciones:**
Element UI 2.x requiere Vue 2.6.0+:
```bash
npm install vue@2
```

## Problemas de uso

### Q: ¿Los componentes no se muestran?

**Soluciones:**
1. Confirmar que los componentes están correctamente importados
```javascript
import { Button } from 'element-ui'
Vue.use(Button)
```
2. Verificar errores en consola
3. Confirmar que los nombres de componentes son correctos

### Q: ¿Los estilos no se aplican?

**Soluciones:**
1. Confirmar que el archivo de estilos está importado
```javascript
import 'element-ui/lib/theme-chalk/index.css'
```
2. Verificar anulaciones de estilos globales
3. Usar herramientas de desarrollo para inspeccionar

### Q: ¿La tabla no muestra datos?

**Soluciones:**
1. Verificar formato de datos
2. Confirmar definiciones de columnas
```html
<el-table-column prop="name" label="Nombre"></el-table-column>
```
3. Verificar si los datos son reactivos

### Q: ¿La validación de formulario no funciona?

**Soluciones:**
1. Importar Form y FormItem
2. Establecer atributo prop
```html
<el-form-item label="Nombre" prop="name">
  <el-input v-model="form.name"></el-input>
</el-form-item>
```
3. Definir reglas en data

### Q: ¿Las opciones del dropdown no se muestran?

**Soluciones:**
1. Verificar datos de opciones
2. Confirmar mapeo de value y label
```html
<el-option
  :value="item.value"
  :label="item.label">
</el-option>
```

### Q: ¿Los datos del diálogo no se restablecen después de cerrar?

**Soluciones:**
1. Usar v-if para controlar el diálogo
```html
<el-dialog :visible.sync="dialogVisible" v-if="dialogVisible">
```
2. Restablecer manualmente el formulario al cerrar
```javascript
this.$refs.form.resetFields()
```

## Problemas de estilo

### Q: ¿Conflictos de estilos?

**Soluciones:**
1. Usar selectores más específicos
2. Verificar niveles de z-index
3. Usar !important para anular (no recomendado)

### Q: ¿Los iconos no se muestran?

**Soluciones:**
1. Confirmar que element-ui-icons está instalado
```bash
npm install element-ui-icons
```
2. Importar correctamente
```javascript
import 'element-ui-icons'
```

### Q: ¿Color de tema incorrecto?

**Soluciones:**
1. Verificar anulaciones de estilos globales
2. Confirmar orden de importación de variables SCSS
3. Limpiar caché del navegador

## Problemas de internacionalización

### Q: ¿El cambio de idioma no funciona?

**Soluciones:**
1. Configurar i18n correctamente
```javascript
Vue.use(ElementUI, {
  i18n: (key, value) => i18n.t(key, value)
})
```
2. Confirmar que el paquete de idioma está importado

### Q: ¿Algunos textos no están traducidos?

**Soluciones:**
1. Verificar paquete de idioma completo
2. Usar configuración de locales de ElementUI

## Problemas de rendimiento

### Q: ¿La página carga lentamente?

**Soluciones:**
1. Usar importación bajo demanda
2. Usar importación CDN
3. Habilitar compresión Gzip

### Q: ¿Tabla con muchos datos va lenta?

**Soluciones:**
1. Usar scroll virtual
2. Cargar datos paginados
3. Desactivar características innecesarias

## Problemas de compilación

### Q: ¿Los estilos se pierden después de compilar?

**Soluciones:**
1. Verificar configuración de vue.config.js
2. Confirmar posición de importación de estilos

### Q: ¿Los componentes no se muestran en producción?

**Soluciones:**
1. Verificar que Vue.use() está llamado
2. Confirmar importación bajo demanda completa

## Problemas de actualización

### Q: ¿Cómo actualizar de 1.x a 2.x?

**Soluciones:**
1. Leer guía oficial de migración
2. Reemplazar APIs incompatibles gradualmente
3. Ejecutar pruebas

### Q: ¿Qué usar para proyectos Vue 3?

**Soluciones:**
Element UI es para Vue 2. Para proyectos Vue 3, usar Element Plus:
```bash
npm install element-plus
```
