# Angular 簡介

現代 Web 應用開發框架。

## 什麼是 Angular？

Angular 是由 Google 開發維護的開源 Web 應用框架，是現代 Web 應用開發的首選方案之一。

### 核心特性

- **TypeScript 原生**：完全基於 TypeScript 開發
- **組件化架構**：基於組件的開發模式
- **依賴注入**：強大的依賴注入系統
- **路由系統**：完整的客戶端路由解決方案
- **表單處理**：響應式表單和模板驅動表單
- **HTTP 客戶端**：完整的 HTTP 客戶端服務

### 技術架構

- **模組（Module）**：組織應用代碼的基本單元
- **組件（Component）**：視圖和業務邏輯的組合
- **服務（Service）**：封裝可複用的業務邏輯
- **指令（Directive）**：擴展 HTML 元素行為
- **管道（Pipe）**：數據轉換工具

### 統計數據

- GitHub 星標：93,000+
- 週下載量：3,000,000+

## 版本

- Angular 17（最新）：新控制流、信號、Standalone API

## 適用場景

- **企業級應用**：大型複雜業務系統
- **單頁應用（SPA）**：需要流暢用戶體驗的應用
- **儀表盤**：數據可視化管理後台
