# Guía de instalación de Element UI

Element UI es una biblioteca de componentes de UI para aplicaciones Vue 2.x.

## Descripción general de versiones

| Versión | Versión de Vue | Estado | Fecha de lanzamiento |
|---------|---------------|--------|---------------------|
| Element UI 2.x | Vue 2.6+ | Actual | 2020 |
| Element UI 1.x | Vue 2.x | Heredado | 2016 |

**Nota:** Element UI es solo para Vue 2. Para Vue 3, usa Element Plus.

### Element UI vs Element Plus

| Característica | Element UI (Vue 2) | Element Plus (Vue 3) |
|----------------|--------------------|----------------------|
| Versión de Vue | 2.x | 3.x |
| TypeScript | Limitado | Soporte completo |
| Estado | Modo mantenimiento | Desarrollo activo |

## Métodos de instalación

### Método 1: npm (Recomendado)

```bash
# Instalar Element UI
npm install element-ui --save
```

### Método 2: Usando Vue CLI

```bash
# Instalar Vue CLI
npm install -g @vue/cli

# Crear proyecto con plantilla Element UI
vue init webpack my-element-app
```

### Método 3: CDN

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Demo CDN de Element UI</title>
  <!-- Importar Vue 2 -->
  <script src="https://unpkg.com/vue@2.6.14/dist/vue.js"></script>
  <!-- Importar estilos de Element UI -->
  <link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
  <!-- Importar Element UI -->
  <script src="https://unpkg.com/element-ui/lib/index.js"></script>
</head>
<body>
  <div id="app">
    <el-button @click="visible = true">Abrir diálogo</el-button>
    <el-dialog :visible.sync="visible" title="Hello Element UI">
      <p>¡Bienvenido a Element UI!</p>
    </el-dialog>
  </div>

  <script>
    new Vue({
      el: '#app',
      data: {
        visible: false
      }
    })
  </script>
</body>
</html>
```

## Configuración del proyecto

### Paso 1: Crear proyecto Vue

```bash
# Usando Vue CLI
npm install -g @vue/cli
vue create my-element-app
cd my-element-app
```

### Paso 2: Instalar Element UI

```bash
npm install element-ui --save
```

### Paso 3: Importar en main.js

#### Importación completa

```javascript
import Vue from 'vue'
import App from './App.vue'

// Importar Element UI
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

// Usar Element UI
Vue.use(ElementUI)

Vue.config.productionTip = false

new Vue({
  render: h => h(App)
}).$mount('#app')
```

#### Importación bajo demanda

Primero, instalar el plugin:

```bash
npm install babel-plugin-component -D
```

Configurar babel.config.js:

```javascript
module.exports = {
  presets: ['@vue/cli-plugin-babel/preset'],
  plugins: [
    [
      'component',
      {
        libraryName: 'element-ui',
        styleLibraryName: 'theme-chalk'
      }
    ]
  ]
}
```

Luego importar componentes:

```javascript
import { Button, Dialog, Input } from 'element-ui'

Vue.component(Button.name, Button)
Vue.component(Dialog.name, Dialog)
Vue.component(Input.name, Input)

// O usar localmente en componentes
export default {
  components: {
    [Button.name]: Button
  }
}
```

## Usar con Webpack

### Configuración básica

```javascript

  module: {
    rules: [
      {
        test: /\.js$/,

  }
```

```javascript

  module: {
```

## Solución de problemas

### Componente no encontrado

**Problema:** Los componentes no se renderizan

**Soluciones:**
1. Asegurarse de que está instalado: `npm list element-ui`
2. Verificar la importación en main.js
3. Comprobar que Vue.use() está llamado

### Estilos no aplicados

**Problema:** Estilos faltantes

**Soluciones:**
1. Importar CSS: `import 'element-ui/lib/theme-chalk/index.css'`
2. Verificar la configuración del loader CSS de webpack

### Compatibilidad con IE10

```bash
# Instalar polyfills
npm install babel-polyfill --save
```

```javascript
// main.js
import 'babel-polyfill'
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```

## Migración a Element Plus (Vue 3)

Si migras a Vue 3, usa Element Plus en su lugar:

```bash
npm install element-plus --save
```

Cambios principales:
1. Nombre del paquete: `element-ui` → `element-plus`
2. Ruta de importación: `element-ui/lib/index` → `element-plus`
3. Algunas props de componentes pueden haber cambiado

## Mejores prácticas

1. **Usar importación bajo demanda** - Reduce el tamaño del bundle
2. **Personalizar tema** - Usar variables SCSS
3. **Usar Vue 2.6+** - Mejor compatibilidad
4. **Configurar babel** - Para builds de producción
