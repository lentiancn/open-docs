# Руководство по установке Angular

> Настройка

---

## Требования

| Среда | Требование |
|--------|------------|
| Node.js | 18.10.0+ |
| npm | 9.0.0+ |

---

## Установка

```bash
npm install -g @angular/cli
ng new my-app
ng serve
```

---

## Команды

| Команда | Описание |
|--------|----------|
| ng new | Новый проект |
| ng serve | Сервер разработки |
| ng build | Сборка |
