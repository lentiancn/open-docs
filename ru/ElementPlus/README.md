# Element Plus

UI-компонентная библиотека для Vue 3.

## Обзор

Element Plus — библиотека компонентов Vue 3 от Ele.me, более 70 высококачественных компонентов.

### Основные функции

- **Vue 3**: Composition API
- **TypeScript**: Полная поддержка типов
- **Tree Shaking**: Оптимизация
- **Темы**: CSS переменные

### Статистика

- GitHub звёзд: 20,000+

## Быстрый старт

### Установка

```bash
npm install element-plus
```

### Использование

```vue
<el-button type="primary">Кнопка</el-button>
```
