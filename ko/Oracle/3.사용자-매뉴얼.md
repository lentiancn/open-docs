# Oracle 사용자 매뉴얼

## 데이터베이스 연결

### SQL*Plus 사용

```bash
# SYSDBA로 연결
sqlplus / as sysdba

# 사용자 이름과 비밀번호로 연결
sqlplus 사용자/비밀번호@데이터베이스
```

---

## 기본 SQL 작업

### 테이블 생성

```sql
CREATE TABLE 직원 (
    직원ID NUMBER(10) PRIMARY KEY,
    이름 VARCHAR2(50) NOT NULL,
    성 VARCHAR2(50) NOT NULL,
    이메일 VARCHAR2(100),
    급여 NUMBER(10, 2)
);
```

### 데이터 삽입

```sql
INSERT INTO 직원 (직원ID, 이름, 성, 이메일, 급여)
VALUES (1001, '김', '철수', 'cheolsu@company.com', 75000);
```

### 데이터 조회

```sql
SELECT * FROM 직원;
SELECT 이름, 성, 급여 FROM 직원 WHERE 부서ID = 10;
```

## PL/SQL

### 저장 프로시저

```sql
CREATE OR REPLACE PROCEDURE 급여수정 (
    p_직원ID IN NUMBER,
    p_새급여 IN NUMBER
) AS
BEGIN
    UPDATE 직원
    SET 급여 = p_새급여
    WHERE 직원ID = p_직원ID;
    COMMIT;
END;
/
```
