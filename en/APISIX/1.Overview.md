# APISIX Overview

Cloud-native API gateway.

## What is APISIX?

APISIX is a cloud-native API gateway from Apache, providing high-performance, high-availability API management.

### Key Features

- **High performance** - Based on Nginx + Lua
- **Dynamic routing** - Dynamic route updates
- **Plugin system** - Rich plugin ecosystem
- **Service discovery** - Multiple mechanisms supported
- **Load balancing** - Multiple strategies
- **Security** - Rate limiting, auth, firewall
