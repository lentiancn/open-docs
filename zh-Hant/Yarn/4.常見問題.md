# Yarn 常見問題

## 基本問題

### 什麼是 Yarn？

Yarn 是一個 JavaScript 套件管理器，幫助你安裝、更新、配置和刪除依賴。

### Yarn 和 npm 的區別？

- **速度**：Yarn 使用並行安裝更快
- **確定性**：yarn.lock 確保一致的安裝
- **離線支持**：Yarn 有出色的離線緩存
- **工作區**：Yarn 原生支持工作區

### Yarn 免費嗎？

是的，Yarn 完全免費，開源（MIT 許可證）。

---

## 安裝問題

### 安裝失敗？

1. 檢查 Node.js 版本：`node --version`
2. 清理 npm 緩存：`npm cache clean --force`
3. 使用 corepack：`corepack enable`

### 如何升級？

```bash
corepack prepare yarn@stable --activate
```

---

## 使用問題

### 如何添加依賴？

```bash
yarn add lodash
yarn add -D typescript
```

### 依賴安裝失敗？

```bash
yarn cache clean
rm -rf node_modules yarn.lock
yarn install
```

## 工作區

### 什麼是工作區？

工作區允許你在單個存儲庫中管理多個包。
