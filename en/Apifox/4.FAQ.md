# Apifox FAQ

Frequently asked questions and solutions.

## Installation Issues

### Q: Download failed?

**Solution**:
1. Check network connection
2. Use proxy
3. Try other download sources

### Q: Startup failed?

**Solution**:
1. Check system requirements
2. Reinstall
3. Check error logs

## Usage Issues

### Q: Request sending failed?

**Solution**:
1. Check network connection
2. Confirm URL is correct
3. Check request parameters
4. Check console errors

### Q: Response garbled?

**Solution**:
1. Check encoding settings
2. Try manual decoding

### Q: Mock not working?

**Solution**:
1. Confirm Mock service is started
2. Check Mock rules
3. Confirm request path is correct

### Q: Environment variables not working?

**Solution**:
1. Confirm environment is selected
2. Check variable syntax
3. Verify variable values

## Account Issues

### Q: Forgot password?

**Solution**:
1. Click forgot password
2. Recover via email/phone

### Q: Cannot log in?

**Solution**:
1. Check network
2. Confirm account and password are correct
3. Try registering again

## Feature Issues

### Q: Postman import failed?

**Solution**:
1. Confirm file format is correct
2. Check file size
3. Import in batches

### Q: How to set up automated testing?

**Solution**:
1. Create test suite
2. Add test steps
3. Configure assertion conditions

### Q: How to collaborate with team members?

**Solution**:
1. Invite members to join
2. Set member permissions
3. Use branch management

## Performance Issues

### Q: Response is slow?

**Solution**:
1. Check network
2. Close unnecessary interfaces
3. Clear cache
