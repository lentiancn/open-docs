# FantasyTalking

## Résumé

FantasyTalking est un projet open-source de synthèse vocale et systèmes de dialogue.

## Caractéristiques

- Synthèse vocale de haute qualité
- Support multilingue
- Système de dialogue flexible
- Multiplateforme

## Installation

```bash
npm install fantasy-talking
# ou
pip install fantasy-talking
```
