# Bootstrap 5

Bootstrap est la boîte à outils open source la plus populaire pour créer des sites web responsives.

## Introduction

Bootstrap 5 est la dernière version, complètement réécrite en JavaScript natif sans dépendance jQuery.

## Fonctionnalités

- Design responsive
- Composants riches
- Utilitaires puissants
- Support Sass
- Icônes intégrées

## Versions

| Version | Statut |
|---------|--------|
| 5.3.3 | ✅ Actuelle |
| 4.x | ❌ Obsolète |

## Démarrage rapide

### CDN

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="delivr.net/npmhttps://cdn.js/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
```

### npm

```bash
npm install bootstrap@5.3.3
```
