# Introduction à Element Plus

Element Plus est une bibliothèque de composants d'interface utilisateur basée sur Vue 3 qui fournit aux développeurs des solutions complètes de composants pour les applications de bureau.

## Qu'est-ce que Element Plus

Element Plus est la version Vue 3 de Element UI, développée et maintenue par l'équipe Ele.me. Elle fournit un ensemble complet de spécifications de conception et de bibliothèques de composants pour aider les développeurs à construire rapidement des applications de bureau.

En tant que framework UI complémentaire Vue 3, Element Plus exploite pleinement la Composition API de Vue 3, offrant un meilleur support TypeScript et une encapsulation de composants plus flexible.

## Caractéristiques principales

### Système de composants complet

Element Plus propose plus de 70 composants de haute qualité couvrant tous les aspects du développement d'applications de bureau:

- Composants de base: Boutons, Icônes, Mise en page, Conteneurs
- Composants de formulaire: Entrée, Sélecteur, Sélecteur de date, Interrupteur, Curseur
- Affichage de données: Tableau, Arbre, Pagination, Étiquettes
- Composants de retour: Dialogue, Tiroir, Message, Chargement
- Composants de navigation: Menu, Onglets, Fil d'Ariane, Étapes

### Support complet Vue 3

Element Plus est entièrement construit sur Vue 3, exploitant pleinement ses nouvelles fonctionnalités:

- Composants construits avec Composition API
- Support complet des types TypeScript
- Meilleure optimisation des performances
- Support du système réactif de Vue 3

### Importation à la demande

Element Plus supporte Tree Shaking, permettant d'importer uniquement les composants utilisés, réduisant efficacement la taille du projet:

- Support des plugins d'importation automatique
- Importation manuelle à la demande
- Support complet ES Module

### Support de l'internationalisation

Support intégré pour plus de 40 paquets linguistiques, répondant aux besoins du développement d'applications internationalisées.

### Personnalisation des thèmes

Support de la personnalisation globale des thèmes, ajustant facilement le style global des composants:

- Configuration des couleurs de thème
- Substitution des styles de composants
- Support des variables CSS

## Principes de conception

Element Plus suit quatre principes de conception majeurs:

### Cohérence

- Cohérence avec la vie réelle: Maintenir la cohérence avec les processus et la logique de la vie réelle, en suivant les habitudes de l'utilisateur en matière de langage et de concepts
- Cohérence dans l'interface: Tous les éléments et structures doivent être cohérents, y compris les styles de conception, les icônes et le texte, les positions des éléments, etc.

### Retour d'information

- Retour de contrôle: Permettre aux utilisateurs de percevoir clairement leurs opérations à travers les styles d'interface et les effets interactifs
- Retour de page: Après les opérations, afficher clairement l'état actuel à travers les changements des éléments de la page

### Efficacité

- Simplifier les processus: Concevoir des processus d'opération simples et intuitifs
- Clair et explicite: Expressions linguistiques claires permettant aux utilisateurs de comprendre rapidement et de prendre des décisions
- Aider les utilisateurs à reconnaître: Interfaces simples et directes permettant aux utilisateurs de reconnaître rapidement plutôt que de se souvenir, réduisant la charge de mémoire

### Contrôlabilité

- Décisions utilisateur: Donner des suggestions d'opération ou des avis de sécurité selon le scénario, mais ne pas décider à leur place
- Résultats contrôlables: Les utilisateurs peuvent effectuer des opérations librement, y compris annuler, revenir en arrière et terminer l'opération actuelle

## Scénarios d'application

Element Plus convient aux scénarios suivants:

- Systèmes de gestion interne d'entreprise
- Systèmes de gestion backend de commerce électronique
- Plateformes de visualisation de données
- Systèmes de gestion client CRM
- Tout projet Vue 3 nécessitant une interface utilisateur de bureau

## Relation avec Element UI

Element Plus est le successeur spirituel de Element UI (version Vue 2):

- Element UI: Conçu pour Vue 2
- Element Plus: Redessiné pour Vue 3

Si votre projet utilise encore Vue 2, envisagez une mise à niveau vers Vue 3 et une migration vers Element Plus.
