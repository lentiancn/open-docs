# Руководство по использованию Element UI

В этом руководстве описано, как использовать компоненты Element UI в вашем приложении Vue 2.

## Основное использование

### Hello World

```vue
<template>
  <div>
    <el-button>Кнопка по умолчанию</el-button>
    <el-button type="primary">Основная кнопка</el-button>
  </div>
</template>

<script>
export default {
  name: 'App'
}
</script>
```

## Категории компонентов

Element UI предоставляет следующие категории компонентов:

| Категория | Компоненты |
|-----------|------------|
| Базовые | Layout, Button, Icon, Link |
| Формы | Input, Select, Radio, Checkbox, Switch, Slider, TimePicker, DatePicker |
| Данные | Table, Tree, Pagination, Tag |
| Уведомления | Alert, Message, Notification, Progress |
| Навигация | Menu, Tabs, Breadcrumb, Steps, Dropdown |
| Другие | Dialog, Drawer, Card, Collapse, Timeline |

### Ввод Input

```vue
<template>
  <div>
    <!-- Базовый -->
    <el-input v-model="input" placeholder="Введите текст"></el-input>
    
    <!-- С иконкой -->
    <el-input
      v-model="input1"
      placeholder="Поиск"
      prefix-icon="el-icon-search">
    </el-input>
    
    <!-- Пароль -->
    <el-input
      v-model="input2"
      placeholder="Пароль"
      show-password>
    </el-input>
    
    <!-- Текстовая область -->
    <el-input
      v-model="textarea"
      type="textarea"
      :rows="3"
      placeholder="Введите описание">
    </el-input>
    
    <!-- Отключенный -->
    <el-input disabled placeholder="Отключен"></el-input>
  </div>
</template>

<script>
export default {
  data() {
    return {
      input: '',
      input1: '',
      input2: '',
      textarea: ''
    }
  }
}
</script>
```

### Форма Form

```vue
<template>
  <el-form
    :model="ruleForm"
    :rules="rules"
    ref="ruleForm"
    label-width="120px"
  >
    <el-form-item label="Имя" prop="name">
      <el-input v-model="ruleForm.name"></el-input>
    </el-form-item>
    
    <el-form-item label="Email" prop="email">
      <el-input v-model="ruleForm.email"></el-input>
    </el-form-item>
    
    <el-form-item label="Пол" prop="gender">
      <el-radio-group v-model="ruleForm.gender">
        <el-radio label="male">Мужской</el-radio>
        <el-radio label="female">Женский</el-radio>
      </el-radio-group>
    </el-form-item>
    
    <el-form-item label="Хобби" prop="hobby">
      <el-checkbox-group v-model="ruleForm.hobby">
        <el-checkbox label="Чтение" name="hobby"></el-checkbox>
        <el-checkbox label="Спорт" name="hobby"></el-checkbox>
        <el-checkbox label="Музыка" name="hobby"></el-checkbox>
      </el-checkbox-group>
    </el-form-item>
    
    <el-form-item label="Страна" prop="country">
      <el-select v-model="ruleForm.country" placeholder="Выберите страну">
        <el-option label="Россия" value="russia"></el-option>
        <el-option label="Украина" value="ukraine"></el-option>
        <el-option label="Беларусь" value="belarus"></el-option>
      </el-select>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm('ruleForm')">Отправить</el-button>
      <el-button @click="resetForm('ruleForm')">Сбросить</el-button>
    </el-form-item>
  </el-form>
</template>

<script>
export default {
  data() {
    return {
      ruleForm: {
        name: '',
        email: '',
        gender: '',
        hobby: [],
        country: ''
      },
      rules: {
        name: [
          { required: true, message: 'Пожалуйста, введите имя', trigger: 'blur' }
        ],
        email: [
          { required: true, message: 'Пожалуйста, введите email', trigger: 'blur' },
          { type: 'email', message: 'Неверный формат email', trigger: 'blur' }
        ],
        gender: [
          { required: true, message: 'Пожалуйста, выберите пол', trigger: 'change' }
        ],
        country: [
          { required: true, message: 'Пожалуйста, выберите страну', trigger: 'change' }
        ]
      }
    };
  },
  methods: {
    submitForm(formName) {
      this.$refs[formName].validate((valid) => {
        if (valid) {
          alert('Отправлено!');
        } else {
          console.log('Ошибка отправки!!');
          return false;
        }
      });
    },
    resetForm(formName) {
      this.$refs[formName].resetFields();
    }
  }
}
</script>
```

### Сообщение Message

```javascript
// Успех
this.$message.success('Сообщение об успехе')

// Ошибка
this.$message.error('Сообщение об ошибке')

// Предупреждение
this.$message.warning('Предупреждающее сообщение')

// Информация
this.$message.info('Информационное сообщение')

// С указанием длительности
this.$message({
  message: 'Пользовательское сообщение',
  type: 'success',
  duration: 3000
})
```

## События

```vue
<template>
  <el-button @click="handleClick">Нажмите меня</el-button>
  
  <el-input
    @focus="handleFocus"
    @blur="handleBlur"
    @change="handleChange"
    @input="handleInput">
  </el-input>
</template>

<script>
export default {
  methods: {
    handleClick() {
      console.log('Нажато!')
    },
    handleFocus() {
      console.log('Фокус получен!')
    },
    handleBlur() {
      console.log('Фокус потерян!')
    },
    handleChange(value) {
      console.log('Изменено:', value)
    },
    handleInput(value) {
      console.log('Ввод:', value)
    }
  }
}
</script>
```

## Рекомендации

### 1. Использование компонентов локально

```javascript
// Хорошо - использовать локально
export default {
  components: {
    [Button.name]: Button,
    [Input.name]: Input
  }
}
```

### 2. Валидация формы

```javascript
const validatePass = (rule, value, callback) => {
  if (value === '') {
    callback(new Error('Пожалуйста, введите пароль'));
  } else {
    if (this.ruleForm.checkPass !== '') {
      this.$refs.ruleForm.validateField('checkPass');
    }
    callback();
  }
};
```

### 3. Производительность таблицы

```vue
<el-table
  :data="tableData"
  :default-sort = "{prop: 'date', order: 'descending'}"
  >
</el-table>
```

## Частые проблемы

### Модификаторы событий

```vue
<!-- Используйте .native для нативных событий -->
<el-input @keyup.enter.native="handleEnter"></el-input>
```

### Ленивая загрузка

```vue
<el-select
  v-model="value"
  filterable
  remote
  :remote-method="remoteMethod"
  :loading="loading">
  <el-option
    v-for="item in options"
    :key="item.value"
    :label="item.label"
    :value="item.value">
  </el-option>
</el-select>
```
