# Element UI 설치 가이드

> Element UI는 Vue 2 기반 데스크톱 UI 컴포넌트 라이브러리입니다
> 
> 업데이트: 2026년 2월
> 
> 현재 버전: 2.x (안정판)
> 
> 참고: Vue 3 프로젝트는 Element Plus를 사용하세요

---

## 1. 프로젝트 개요

Element UI는 Vue 2.0 기반 데스크톱 UI 컴포넌트 라이브러리로, Ele.me 팀에서 개발하고 유지관리합니다. 폼, 테이블, 다이얼로그, 탐색 메뉴, 버튼, 입력 필드 등 자주 사용되는 PC 컴포넌트를 풍부하게 제공합니다.

Element UI는 세련된 디자인 스타일과 상세한 문서를 가지고 있습니다. 중국에서 가장 인기 있는 Vue 2 UI 컴포넌트 라이브러리 중 하나입니다.

### 1.1 버전 정보

| 버전 | Vue 버전 | 상태 | 설명 |
|------|----------|------|------|
| Element UI 2.x | Vue 2 | 안정 | 주요 문서 버전 |
| Element UI 1.x | Vue 2 | 종료 | 초기 버전 |

### 1.2 Element Plus와의 차이

| 기능 | Element UI | Element Plus |
|------|------------|--------------|
| Vue 버전 | Vue 2 | Vue 3 |
| TypeScript | 부분 지원 | 전체 지원 |
| Composition API | 미지원 | 지원 |
| 유지관리 상태 | 종료됨 | 활발히 유지 |

## 3. 설치 단계

### 3.1 npm 사용 (권장)

```bash
# Vue 2 프로젝트 디렉토리로 이동
cd your-vue-project

# Element UI 설치
npm install element-ui --save
```

### 3.2 yarn 사용

```bash
yarn add element-ui
```

## 4. Vue 2 프로젝트에서 가져오기

### 4.1 전체 가져오기

메인 파일(일반적으로 main.js)에서 Element UI와 스타일을 가져옵니다:

```javascript
import Vue from 'vue'
import App from './App.vue'

import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)

new Vue({
  render: h => h(App)
}).$mount('#app')
```

### 4.2 필요시 가져오기 (권장)

번들 크기를 줄이려면 필요한 컴포넌트만 가져옵니다:

```javascript
import { Button, Form, FormItem, Input } from 'element-ui'

Vue.use(Button)
Vue.use(Form)
Vue.use(FormItem)
Vue.use(Input)
```

## 5. 빠른 시작 예제

### 5.1 기본 버튼

```html
<el-button>기본</el-button>
<el-button type="primary">주요</el-button>
<el-button type="success">성공</el-button>
<el-button type="warning">경고</el-button>
<el-button type="danger">위험</el-button>
```

### 5.2 기본 폼

```html
<el-form :model="form" label-width="80px">
  <el-form-item label="사용자 이름">
    <el-input v-model="form.username"></el-input>
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="submit">제출</el-button>
  </el-form-item>
</el-form>
```

### 5.3 기본 테이블

```html
<el-table :data="tableData" stripe border>
  <el-table-column prop="name" label="이름"></el-table-column>
  <el-table-column prop="email" label="이메일"></el-table-column>
</el-table>
```
