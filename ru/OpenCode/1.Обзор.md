# OpenCode Обзор

AI-агент для программирования с открытым исходным кодом.

## Что такое OpenCode?

OpenCode — это AI-агент для программирования с открытым исходным кодом, доступный как терминальный интерфейс, десктопное приложение или расширение IDE.

### Основные функции

- **LSP включен** - Автоматически загружает правильные LSP
- **Множественные сессии** - Запускайте несколько агентов параллельно
- **Обмен ссылками** - Делитесь любой ссылкой на сессию
- **GitHub Copilot** - Войдите через GitHub для использования Copilot
- **ChatGPT Plus/Pro** - Войдите через OpenAI для использования ChatGPT
- **Любая модель** - 75+ LLM-провайдеров, включая локальные модели
- **Любой редактор** - Терминал, десктопное приложение и расширение IDE
- **Конфиденциальность прежде всего** - Не сохраняет код или контекстные данные
