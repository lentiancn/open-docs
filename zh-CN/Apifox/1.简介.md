# Apifox - 简介

Apifox 是一款 API 文档、API 调试、API Mock、API 自动化测试一体化协作平台。由腾讯公司于 2020 年推出，旨在解决 API 开发团队在文档管理、调试、测试等环节的协作问题。

## Apifox 的核心特性

### 1. API 文档管理

Apifox 提供强大的 API 文档管理功能：

- 支持 Swagger、OpenAPI、RAP2、YApi 等格式导入
- 自动生成 API 文档
- 文档在线分享
- 版本管理

### 2. API 调试

可视化的 API 调试界面：

```http
### 获取用户列表
GET https://api.example.com/users
Content-Type: application/json

### 创建用户
POST https://api.example.com/users
Content-Type: application/json

{
  "name": "Tom",
  "email": "tom@example.com"
}
```

### 3. API Mock

内置 Mock 功能，快速生成模拟数据：

```javascript
// Mock 规则
{
  "id": "@integer",
  "name": "@cname",
  "email": "@email",
  "avatar": "@image",
  "createTime": "@datetime"
}
```

### 4. 自动化测试

可视化配置自动化测试：

```javascript
// 测试脚本
pm.test("状态码为200", function() {
    pm.response.to.have.status(200);
});

pm.test("响应时间小于500ms", function() {
    pm.expect(pm.response.responseTime).to.be.below(500);
});
```

## Apifox 的应用场景

### 1. 前端开发

前端开发者可以使用 Mock 功能，在后端 API 未完成时进行开发：

- 自动生成 Mock 数据
- 自定义 Mock 规则
- 实时预览效果

### 2. 后端开发

后端开发者可以快速调试 API：

- 调试 HTTP 请求
- 查看响应详情
- 生成代码片段

### 3. 测试团队

测试团队可以进行自动化测试：

- 可视化测试用例配置
- 批量执行测试
- 生成测试报告

## Apifox 与其他工具对比

### Apifox vs Postman

| 特性 | Apifox | Postman |
|------|---------|---------|
| 中文界面 | 支持 | 英文 |
| API 文档 | 强大 | 一般 |
| Mock | 内置 | 需配置 |
| 价格 | 免费 | 部分收费 |
| 团队协作 | 方便 | 方便 |

### Apifox vs Swagger

| 特性 | Apifox | Swagger |
|------|---------|---------|
| 调试功能 | 强大 | 一般 |
| 可视化 | 好 | 一般 |
| 导入导出 | 多格式 | OpenAPI |
| 本地部署 | 支持 | 支持 |
