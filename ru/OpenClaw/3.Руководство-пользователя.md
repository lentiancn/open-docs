# Руководство пользователя OpenClaw

Это руководство содержит подробное описание функций и способов использования OpenClaw, помогая вам максимально эффективно использовать этот мощный шлюз ИИ-ассистента.

---

## Основные понятия

### Gateway

Gateway — это основной компонент OpenClaw. Он работает на вашем локальном компьютере как плоскость управления для всех подключений каналов, ИИ-агентов и клиентов. Один процесс Gateway может одновременно обрабатывать несколько каналов чата и несколько ИИ-агентов.

### Каналы

Каналы — это платформы чата, которые вы используете для взаимодействия с ИИ-ассистентом, включая WhatsApp, Telegram, Discord, iMessage и другие. Каждый канал может быть настроен и управляться независимо.

### Агенты

Агенты — это экземпляры ИИ-ассистента. У каждого агента есть собственное рабочее пространство, сессии и конфигурация. Вы можете настроить несколько агентов для разных целей — личного использования, работы и т.д.

### Сессии

Сессии управляют контекстом разговора. OpenClaw поддерживает несколько режимов сессий:
- **main**: Общая сессия
- **per-peer**: Отдельная сессия для каждого контакта
- **per-channel-peer**: Отдельная сессия для каждой комбинации канал-контакт
- **per-account-channel-peer**: Наиболее детальная — отдельная сессия для каждой комбинации аккаунт-канал

## Повседневное использование

### Запуск и остановка Gateway

**Запуск:**

```bash
# Запуск на переднем плане (для отладки)
openclaw gateway --port 18789

# Запуск в фоновом режиме
openclaw gateway start
```

**Остановка:**

```bash
openclaw gateway stop
```

**Проверка статуса:**

```bash
openclaw gateway status
```

### Использование интерфейса управления

Посетите http://127.0.0.1:18789/ в браузере, чтобы открыть интерфейс управления.

### Команды CLI

**Отправка сообщения:**

```bash
openclaw message send --target +15555550123 --message "Привет, OpenClaw"
```

**Общение с ИИ:**

```bash
openclaw agent --message "Помоги написать программу Hello World" --thinking high
```

## Управление каналами

### Настройка WhatsApp

```bash
openclaw channels login
```

### Настройка Telegram

```json5
{
  channels: {
    telegram: {
      botToken: "ВашТокенБота",
    },
  },
}
```

### Настройка Discord

```json5
{
  channels: {
    discord: {
      token: "ВашТокенБота",
    },
  },
}
```

## Продвинутые функции

### Маршрутизация нескольких агентов

```json5
{
  agents: {
    list: [
      { id: "home", default: true, workspace: "~/.openclaw/workspace-home" },
      { id: "work", workspace: "~/.openclaw/workspace-work" },
    ],
  },
}
```

### Cron-задачи

```json5
{
  cron: {
    enabled: true,
    maxConcurrentRuns: 2,
  },
}
```

### Режим песочницы

```json5
{
  agents: {
    defaults: {
      sandbox: {
        mode: "non-main",
        scope: "agent",
      },
    },
  },
}
```

## Частые команды

| Команда | Описание |
|---------|----------|
| `openclaw --version` | Показать версию |
| `openclaw status` | Показать статус |
| `openclaw gateway start` | Запустить Gateway |
| `openclaw gateway stop` | Остановить Gateway |
| `openclaw dashboard` | Открыть интерфейс управления |
| `openclaw channels login` | Войти в каналы |
| `openclaw config get` | Получить конфигурацию |
| `openclaw config set` | Установить конфигурацию |
| `openclaw doctor` | Диагностировать проблемы |

---

## Команды чата

- `/status` — Показать статус сессии
- `/new` или `/reset` — Сбросить сессию
- `/compact` — Сжать контекст сессии
- `/think` — Установить уровень мышления
- `/verbose on|off` — Переключить подробный вывод
