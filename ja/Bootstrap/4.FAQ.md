# Bootstrap FAQ

> よくある質問

---

## インストール問題

### CDNの読み込みに失敗する？

**解決策**:

1. ネットワーク接続を確認
2. 代替CDNを使用:
```html
<link href="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

### npmインストール後にスタイルが効かない？

正しくインポートしていることを確認:

```javascript
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap';
```

## スタイル問題

### コンポーネントのスタイルが効かない？

**一般的な原因**:

1. Bootstrap CSSがインポートされていない
2. JavaScriptが正しく読み込まれていない（一部のコンポーネントで必要）
3. CSSの優先度が上書きされている

### レスポンシブが動作しない？

viewportメタタグを追加していることを確認:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

---

## コンポーネント問題

### ドロップダウンが動作しない？

**チェックリスト**:

1. Bootstrap JSをインポートしている（CSSのみではない）
2. 正しいHTML構造を使用している
3. JavaScriptエラーがないか確認

### モーダルが開けない？

確認:
1. Bootstrap JSが読み込まれている
2. data-bs-toggleとdata-bs-target属性が正しい
3. ページ読み込み時にモーダルが存在する

---

## カスタマイズ問題

### テーマ色をカスタマイズするには？

**方法1: Sassを使用**

```scss
$primary: #ff6600;
@import "bootstrap";
```

**方法2: CSS変数を使用**

```css
:root {
  --bs-primary: #ff6600;
}
```

## 互換性問題

### IE 11はサポートされていない？

Bootstrap 5はIE 11をサポートしなくなりました。IE 11サポートが必要な場合はBootstrap 4を使用してください。

---

## パフォーマンス問題

### ファイルサイズを削減するには？

1. 必要なコンポーネントのみをインポート（Sass）
2. 本番バージョン（.min.css/.min.js）を使用
3. gzip圧縮を有効にする

---

## 他のフレームワークとの競合

### Vue/Reactと一緒に使用するには？

BootstrapはどのJavaScriptフレームワークでも使用できますが、注意点:

1. 一部のコンポーネントはjQueryに依存（Bootstrap 5は依存なし）
2. フレームワーク固有のBootstrapラッパーを使用（例: react-bootstrap）
