# Angular

Framework de desarrollo de aplicaciones web modernas.

## Resumen

Angular es un framework de código abierto desarrollado por Google.

### Características

- **TypeScript nativo**
- **Componentes**
- **Inyección de dependencias**
- **Enrutamiento**
- **Formularios**

### Estadísticas

- GitHub Stars: 93,000+

## Inicio rápido

```bash
npm install -g @angular/cli
ng new my-app
ng serve
```

## Comandos

```bash
ng g c my-component
ng g s my-service
ng build
```
