# Foire Aux Questions

Ce document recense les questions techniques fréquentes rencontrées lors de l'utilisation de Bootstrap.

## Questions de base

### Qu'est-ce que Bootstrap ?

Bootstrap est un framework de développement frontend open source, développé et maintenu par Twitter. Il fournit de nombreux styles CSS et composants JavaScript pour aider les développeurs à créer rapidement des applications web responsives.

### Quelle est la différence entre Bootstrap 5 et Bootstrap 4 ?

Les principales différences :

- Bootstrap 5 a supprimé la dépendance à jQuery et utilise le JavaScript natif
- Ajout du point de rupture XXL
- Amélioration des contrôles de formulaire
- Remplacement de Font Awesome par Bootstrap Icons
- Plus de support pour Internet Explorer

### De quels environnements Bootstrap a-t-il besoin ?

Un navigateur moderne suffit. Bootstrap 5 nécessite un navigateur supportant ES6.

## Problèmes d'installation

### Le chargement CDN échoue ?

1. Vérifiez votre connexion réseau
2. Essayez un autre CDN (comme bootcdn, staticfile)
3. Téléchargez les fichiers en local

### L'installation npm échoue ?

```bash
# Vider le cache
npm cache clean --force

# Utiliser le miroir taobao
npm config set registry https://registry.npmmirror.com

# Réinstaller
npm install bootstrap@5
```

### Les styles ne s'appliquent pas ?

1. Vérifiez que le fichier CSS est bien inclus
2. Videz le cache du navigateur (forcez l'actualisation Ctrl+Shift+R)
3. Assurez-vous qu'il n'y a pas de conflits CSS

## Problèmes d'utilisation

### Le système de grille ne fonctionne pas ?

1. Assurez-vous que row est à l'intérieur de container
2. Assurez-vous que col est à l'intérieur de row
3. Vérifiez que les classes de point de rupture sont bien utilisées

### Le responsive ne fonctionne pas ?

Ajoutez bien la balise viewport :

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### La barre de navigation ne se replie pas sur mobile ?

1. Assurez-vous d'inclure bootstrap.bundle.min.js (qui contient Popper)
2. Vérifiez les attributs data-bs-toggle et data-bs-target

### La fenêtre modale ne s'ouvre pas ?

1. Assurez-vous d'inclure le JavaScript de Bootstrap
2. Vérifiez les attributs data-bs-toggle et data-bs-target du bouton
3. Vérifiez que l'ID de la modale est correct

### Le menu déroulant ne fonctionne pas ?

1. Incluez bootstrap.bundle.min.js
2. Vérifiez l'attribut data-bs-toggle="dropdown"

## Problèmes de style

### Comment modifier les couleurs du thème ?

Utilisez Sass :

```scss
$primary: #007bff;
@import "bootstrap/scss/bootstrap";
```

### Comment écraser les styles par défaut ?

Utilisez un sélecteur avec une priorité plus haute :

```css
.navbar {
    background-color: #f8f9fa !important;
}
```

### Comment modifier la police ?

```scss
$font-family-sans-serif: "Helvetica Neue", Helvetica, Arial, sans-serif;
@import "bootstrap/scss/bootstrap";
```

### Les icônes ne s'affichent pas ?

Bootstrap 5 utilise Bootstrap Icons :

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.10.0/font/bootstrap-icons.css">

<i class="bi bi-alarm"></i>
```

## Problèmes de composants

### Comment centrer une carte ?

```html
<div class="card">
    <div class="card-body">
        Contenu
    </div>
</div>
```

Utilisez flex :

```html
<div class="card">
    <div class="card-body d-flex justify-content-center align-items-center">
        Contenu centré
    </div>
</div>
```

### La validation de formulaire ne fonctionne pas ?

Bootstrap 5 utilise un nouveau système de validation :

```html
<form class="row g-3 needs-validation" novalidate>
    <div class="col-md-4">
        <label for="validationCustom01" class="form-label">Prénom</label>
        <input type="text" class="form-control" id="validationCustom01" required>
        <div class="valid-feedback">Correct !</div>
        <div class="invalid-feedback">Veuillez entrer un prénom</div>
    </div>
    <div class="col-12">
        <button class="btn btn-primary" type="submit">Soumettre</button>
    </div>
</form>

<script>
    (function () {
        'use strict'
        var forms = document.querySelectorAll('.needs-validation')
        Array.prototype.slice.call(forms).forEach(function (form) {
            form.addEventListener('submit', function (event) {
                if (!form.checkValidity()) {
                    event.preventDefault()
                    event.stopPropagation()
                }
                form.classList.add('was-validated')
            }, false)
        })
    })()
</script>
```

### Comment fermer la modale en cliquant à l'extérieur ?

Définissez l'attribut :

```html
<div class="modal" data-bs-backdrop="static">
```

`static` - Cliquer sur le fond ne ferme pas
`true` - Cliquer ferme
`false` - Cliquer ne ferme pas

## Problèmes de performance

### Comment réduire la taille des fichiers ?

1. Importez uniquement les composants nécessaires (avec Sass)
2. Utilisez la version de production
3. Activez la compression Gzip

### Comment importer uniquement les styles requis ?

Utilisez Sass pour n'importer que les composants nécessaires :

```scss
// Importer les fonctions et variables nécessaires
@import "bootstrap/scss/functions";
@import "bootstrap/scss/variables";

// Importer uniquement les composants voulus
@import "bootstrap/scss/root";
@import "bootstrap/scss/reboot";
@import "bootstrap/scss/containers";
@import "bootstrap/scss/grid";
@import "bootstrap/scss/buttons";
@import "bootstrap/scss/forms";
```

## Problèmes de compatibilité

### Le navigateur IE n'est pas supporté ?

Bootstrap 5 ne supporte plus Internet Explorer. Pour le support d'IE11, utilisez Bootstrap 4.

### Le retard de clic sur mobile ?

Bootstrap 5 utilise touch-action: manipulation pour éliminer le retard de 300ms sur le clic.
