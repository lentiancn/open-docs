# LivePortrait 安装指南

本文档详细介绍 LivePortrait 在不同操作系统和环境下的安装步骤。

## 环境要求

在开始安装之前，请确保您的系统满足以下基本要求：

**操作系统**：Linux（推荐）、Windows 10/11、macOS（需要Apple Silicon）

**Python 版本**：Python 3.10

**硬件要求**：NVIDIA GPU（推荐RTX 4090或其他支持CUDA的显卡），至少8GB显存

**存储空间**：建议预留至少20GB空间用于模型权重和运行环境

## 基础环境配置

### 检查CUDA版本

首先确认您的系统上安装的CUDA版本，以便安装对应版本的PyTorch：

```bash
nvcc -V
```

常见的CUDA版本包括11.1、11.8、12.1等。根据显示的版本号，选择对应的PyTorch安装命令。

### 安装PyTorch

根据您的CUDA版本，选择合适的PyTorch安装命令：

**CUDA 11.1**：

```bash
pip install torch==1.10.1+cu111 torchvision==0.11.2 torchaudio==0.10.1 -f https://download.pytorch.org/whl/cu111/torch_stable.html
```

**CUDA 11.8**：

```bash
pip install torch==2.3.0 torchvision==0.18.0 torchaudio==2.3.0 --index-url https://download.pytorch.org/whl/cu118
```

**CUDA 12.1**：

```bash
pip install torch==2.3.0 torchvision==0.18.0 torchaudio==2.3.0 --index-url https://download.pytorch.org/whl/cu121
```

如果您只需要运行人类模式（Humans mode），可以跳过CUDA版本检查，直接安装最新版本的PyTorch。

## 下载项目代码

使用Git克隆LivePortrait仓库：

```bash
git clone https://github.com/KlingTeam/LivePortrait
cd LivePortrait
```

## 创建Python环境

建议使用conda创建独立的Python环境：

```bash
conda create -n LivePortrait python=3.10
conda activate LivePortrait
```

或者使用venv：

```bash
python -m venv LivePortrait
# Linux/macOS
source LivePortrait/bin/activate
# Windows
LivePortrait\Scripts\activate
```

## 安装依赖

在项目根目录下执行：

```bash
pip install -r requirements.txt
```

## 下载预训练模型

预训练模型可以从HuggingFace下载。如果您的网络可以访问HuggingFace：

```bash
pip install -U "huggingface_hub[cli]"
huggingface-cli download KlingTeam/LivePortrait --local-dir pretrained_weights --exclude "*.git*" "README.md" "docs"
```

如果无法访问HuggingFace，可以使用镜像站点：

```bash
export HF_ENDPOINT=https://hf-mirror.com
huggingface-cli download KlingTeam/LivePortrait --local-dir pretrained_weights --exclude "*.git*" "README.md" "docs"
```

或者从Google Drive或百度网盘下载完整模型文件，解压后放入./pretrained_weights目录。

## 验证安装

安装完成后，运行测试脚本验证环境配置是否正确：

```bash
python inference.py
```

如果一切正常，您将在animations目录下看到生成的输出文件。

## 动物模型额外安装步骤

动物模式（Animals mode）仅支持Linux和Windows系统，且需要NVIDIA GPU。

### 安装X-Pose依赖

动物模式需要X-Pose项目作为关键点检测框架。首先进入X-Pose目录：

```bash
cd src/utils/dependencies/XPose/models/UniPose/ops
python setup.py build install
cd -
```

### 使用HuggingFace下载动物模型

```bash
huggingface-cli download KlingTeam/LivePortrait --include "animal*" --local-dir pretrained_weights
```

## Windows特殊说明

在Windows系统上，某些较高版本的CUDA（如12.4、12.6）可能导致兼容性问题。建议将CUDA降级到11.8版本以获得更好的稳定性。

Windows用户也可以选择下载一键安装包，解压后双击run_windows.bat即可运行：

```bash
# 从HuggingFace下载Windows一键安装包
# 下载地址：https://huggingface.co/cleardusk/LivePortrait-Windows
```

## macOS特殊说明

### Apple Silicon macOS

对于搭载Apple Silicon的macOS用户，需要使用特殊的依赖文件：

```bash
pip install -r requirements_macOS.txt
```

运行推理时需要设置环境变量：

```bash
PYTORCH_ENABLE_MPS_FALLBACK=1 python inference.py
```

注意：在Apple Silicon macOS上运行推理可能比RTX 4090慢约20倍。

### Intel Mac

Intel芯片的Mac未经官方测试，可能无法正常运行。

### 动物模式限制

由于X-Pose依赖不支持macOS系统，动物模式在macOS上不可用，只能使用人类模式。

## 目录结构验证

安装完成后，请确保目录结构符合以下要求：

```
LivePortrait/
├── pretrained_weights/     # 预训练模型权重目录
├── assets/                 # 示例素材目录
│   ├── examples/
│   │   ├── source/        # 源图像/视频
│   │   └── driving/       # 驱动视频
├── src/                    # 源代码目录
├── inference.py           # 推理脚本
├── app.py                 # Gradio界面
└── requirements.txt       # 依赖列表
```

## 常见问题

如果安装过程中遇到问题，请检查以下几点：

**CUDA不可用**：确认NVIDIA驱动已正确安装，运行nvidia-smi检查驱动状态

**显存不足**：尝试使用更小的输入图像或降低批处理大小

**模型下载失败**：检查网络连接，或使用国内镜像源

**依赖冲突**：建议使用全新的Python环境安装，避免与其他项目产生冲突
