# GitLab

> 軟體開發和營運的完整 DevOps 平台

---

## 概述

GitLab 是一個具有完整 DevP 功能（包括原始碼管理、問題追蹤、CI/CD 管道和安全掃描）的基於網頁的 Git 儲存庫管理工具。

## 文檔

本目錄包含繁體中文的完整文檔：

| 檔案 | 說明 |
|------|------|
| 1.簡介.md | 簡介和概述 |
| 2.安裝指南.md | 安裝和設定指南 |
| 3.使用手冊.md | 詳細使用指南 |
| 4.常見問題.md | 常見問題 |
| README.md | 本概述檔案 |

## 快速開始

### 主要功能

- 具有分支和標籤的 Git 儲存庫託管
- 問題追蹤和專案管理
- 內建 CI/CD 管道
- 程式碼審查和合併請求
- 用於文檔的 wiki
- 安全掃描
- 容器登錄檔

### 基本工作流程

1. 建立專案
2. 複製儲存庫
3. 建立分支並進行更改
4. 推送並建立合併請求
5. 審查並合併
6. CI/CD 自動部署
