# Claude Code 簡介

Claude Code 是由 Anthropic 公司開發的智慧程式設計助手，能夠讀取程式碼庫、編輯檔案、執行命令，並與開發工具無縫整合。它支援在終端、IDE、桌面應用程式和瀏覽器等多種環境中使用。

## 什麼是 Claude Code

Claude Code 是一款 AI 驅動的程式設計助手，能夠幫助你建構功能、修復錯誤和自動化開發任務。它能夠理解整個程式碼庫，可以跨多個檔案和工具協同工作來完成各種開發任務。

## 主要特性

Claude Code 提供以下核心能力：

**自動化開發任務**
- 為未測試的程式碼編寫測試用例
- 修復整個專案的 lint 錯誤
- 解決合併衝突
- 更新依賴版本
- 編寫發布說明

**功能開發與錯誤修復**
- 用自然語言描述你想要實現的功能
- Claude Code 會規劃方案、跨檔案編寫程式碼並驗證功能

**Git 整合**
- 直接在 Claude Code 中進行 Git 操作
- 暫存更改、編寫提交資訊、建立分支、開啟 Pull Request

**MCP 整合**
- Model Context Protocol（MCP）是一個開放標準，用於將 AI 工具連接到外部資料來源
- 透過 MCP，Claude Code 可以讀取 Google Drive 中的設計文件、更新 Jira 工單、從 Slack 取得資料或使用自訂工具

**自訂能力**
- 使用 CLAUDE.md 檔案為專案設定編碼標準、架構決策、首選函式庫和審查清單
- 建立自訂命令來封裝可重複的工作流
- 使用 Hooks 在檔案編輯後自動格式化或在提交前執行 lint

**多代理協作**
- 可以同時啟動多個 Claude Code 代理並行工作
- 牵头代理協調工作、分配子任務並合併結果

**CLI 靈活性**
- 支援管道操作、腳本化和自動化
- 可以將日誌透過管道傳輸給它、在 CI 中執行或與其他工具鏈式組合

## 使用場景

Claude Code 可以在多種場景下使用：

| 使用場景 | 推薦方式 |
|---------|---------|
| 本地開發 | 終端 CLI |
| IDE 內編碼 | VS Code 或 JetBrains |
| 跨裝置繼續工作 | 遠端控制功能 |
| 行動端操作 | Web 或 iOS 應用 |
| CI/CD 自動化 | GitHub Actions 或 GitLab CI/CD |
| 團隊協作 | Slack 整合 |
| 瀏覽器偵錯 | Chrome 擴充功能 |

## 帳戶要求

Claude Code 需要以下帳戶之一：
- Claude Pro、Max、Teams 或 Enterprise 訂閱
- Anthropic Console 帳戶
- 支援的第三方雲端提供商（Amazon Bedrock、Google Vertex AI、Microsoft Foundry）

免費 Claude.ai 方案不包含 Claude Code 存取權限。
