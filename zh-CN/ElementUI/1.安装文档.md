# Element UI 安装指南

Element UI 是一个用于 Vue 2.x 应用程序的 UI 组件库。

## 版本概述

| 版本 | Vue 版本 | 状态 | 发布日期 |
|------|----------|------|----------|
| Element UI 2.x | Vue 2.6+ | 当前 | 2020 |
| Element UI 1.x | Vue 2.x | 旧版 | 2016 |

**注意：** Element UI 仅适用于 Vue 2。对于 Vue 3，请使用 Element Plus。

### Element UI vs Element Plus

| 特性 | Element UI (Vue 2) | Element Plus (Vue 3) |
|------|-------------------|---------------------|
| Vue 版本 | 2.x | 3.x |
| TypeScript | 有限 | 完全支持 |
| 状态 | 维护模式 | 活跃开发 |

## 前置条件

### 系统要求

| 要求 | 版本 |

## 安装方法

### 方法 1：npm（推荐）

```bash
# 安装 Element UI
npm install element-ui --save
```

### 方法 2：CDN

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Element UI CDN 示例</title>
  <!-- 引入 Vue 2 -->
  <script src="https://unpkg.com/vue@2.6.14/dist/vue.js"></script>
  <!-- 引入 Element UI 样式 -->
  <link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
  <!-- 引入 Element UI -->
  <script src="https://unpkg.com/element-ui/lib/index.js"></script>
</head>
<body>
  <div id="app">
    <el-button @click="visible = true">打开对话框</el-button>
    <el-dialog :visible.sync="visible" title="Hello Element UI">
      <p>欢迎使用 Element UI！</p>
    </el-dialog>
  </div>

  <script>
    new Vue({
      el: '#app',
      data: {
        visible: false
      }
    })
  </script>
</body>
</html>
```

## 项目设置

### 步骤 1：创建 Vue 项目

```bash
# 使用 Vue CLI
npm install -g @vue/cli
vue create my-element-app
cd my-element-app
```

### 步骤 2：安装 Element UI

```bash
npm install element-ui --save
```

### 步骤 3：在 main.js 中引入

#### 全量引入

```javascript
import Vue from 'vue'
import App from './App.vue'

// 引入 Element UI
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

// 使用 Element UI
Vue.use(ElementUI)

Vue.config.productionTip = false

new Vue({
  render: h => h(App)
}).$mount('#app')
```

#### 按需引入

首先，安装插件：

```bash
npm install babel-plugin-component -D
```

配置 babel.config.js：

```javascript
module.exports = {
  presets: ['@vue/cli-plugin-babel/preset'],
  plugins: [
    [
      'component',
      {
        libraryName: 'element-ui',
        styleLibraryName: 'theme-chalk'
      }
    ]
  ]
}
```

然后引入组件：

```javascript
import { Button, Dialog, Input } from 'element-ui'

Vue.component(Button.name, Button)
Vue.component(Dialog.name, Dialog)
Vue.component(Input.name, Input)
```

## 故障排除

### 组件找不到

**问题：** 组件不显示

**解决方案：**
1. 确保已安装：`npm list element-ui`
2. 检查 main.js 中的引入
3. 确认 Vue.use() 已调用

### 样式不生效

**问题：** 样式缺失

**解决方案：**
1. 引入 CSS：`import 'element-ui/lib/theme-chalk/index.css'`
2. 检查 webpack CSS loader 配置

### IE10 兼容性

```javascript
// main.js
import 'babel-polyfill'
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```

## 迁移到 Element Plus (Vue 3)

如果迁移到 Vue 3，请使用 Element Plus：

```bash
npm install element-plus --save
```

主要变化：
1. 包名：`element-ui` → `element-plus`
2. 引入路径：`element-ui/lib/index` → `element-plus`

## 最佳实践

1. **使用按需引入** - 减小打包体积
2. **自定义主题** - 使用 SCSS 变量
3. **使用 Vue 2.6+** - 更好的兼容性
