# Element Plus Benutzerhandbuch

Dieses Handbuch hilft Ihnen, schnell mit Element Plus zu beginnen und die am häufigsten verwendeten Komponenten und Funktionen zu meistern.

## Schnellstart

Stellen Sie vor der Verwendung von Element Plus sicher, dass die Installation abgeschlossen ist.

### Grundlegende Verwendung

Verwenden Sie Komponenten direkt in Vue-Komponenten:

<template>
  <el-button>Standardschaltfläche</el-button>
  <el-button type="primary">Primäre Schaltfläche</el-button>
  <el-button type="success">Erfolgschaltfläche</el-button>
  <el-button type="warning">Warnschaltfläche</el-button>
  <el-button type="danger">Gefahrenschaltfläche</el-button>
</template>

## Häufige Komponenten

### Schaltfläche

Grundlegende Verwendung:

<el-button>Standard</el-button>
<el-button type="primary">Primär</el-button>
<el-button type="success">Erfolg</el-button>
<el-button type="warning">Warnung</el-button>
<el-button type="danger">Gefahr</el-button>

Unterschiedliche Größen:

<el-button size="large">Groß</el-button>
<el-button>Standard</el-button>
<el-button size="small">Klein</el-button>

Deaktivierter Zustand:

<el-button disabled>Deaktiviert</el-button>

Schaltflächen mit Symbol:

<el-button icon="Search">Suchen</el-button>
<el-button icon="Edit">Bearbeiten</el-button>
<el-button icon="Delete">Löschen</el-button>

### Eingabe

Grundlegende Verwendung:

<el-input v-model="input" placeholder="Bitte Inhalt eingeben" />

Mit Symbol:

<el-input v-model="input" prefix-icon="Search" />
<el-input v-model="input" suffix-icon="Calendar" />

Mit Löschschaltfläche:

<el-input v-model="input" clearable />

Passworteingabe:

<el-input v-model="input" type="password" show-password />

Mehrzeiliger Text:

<el-input v-model="textarea" type="textarea" :rows="3" />

### Auswahl

Grundauswahl:

  <el-option label="Option 1" value="1" />
  <el-option label="Option 2" value="2" />
</el-select>

Mehrfachauswahl:

<el-select v-model="value" multiple placeholder="Bitte auswählen">
  <el-option label="Option 1" value="1" />
  <el-option label="Option 2" value="2" />
</el-select>

### Tabelle

Grundlegende Tabelle:

<el-table :data="tableData" stripe border>
  <el-table-column prop="date" label="Datum" width="180" />
  <el-table-column prop="name" label="Name" width="180" />
  <el-table-column prop="address" label="Adresse" />
</el-table>

Mit Index:

<el-table-column type="index" label="Nr." width="60" />

Fixierte Spalten:

<el-table-column prop="date" label="Datum" width="150" fixed />

### Formular

Grundlegendes Formular:

<el-form :model="form" label-width="80px">
  <el-form-item label="Aktivitätsname">
    <el-input v-model="form.name" />
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="onSubmit">Absenden</el-button>
  </el-form-item>

<el-form :model="ruleForm" :rules="rules" ref="ruleFormRef">
  <el-form-item label="Name" prop="name">
    <el-input v-model="ruleForm.name" />
  </el-form-item>
</el-form>

## Layout-System

### Rastersystem

Element Plus verwendet ein 24-Spalten-Rastersystem:

<el-row :gutter="20">
  <el-col :span="12"><div>12 Spalten</div></el-col>
  <el-col :span="12"><div>12 Spalten</div></el-col>
</el-row>

### Container-Layout

<el-container>
  <el-header>Kopfzeile</el-header>
  <el-main>Hauptbereich</el-footer>
  <el-footer>Fußzeile</el-footer>
</el-container>

## Häufige Probleme

### Komponentenstile werden nicht angewendet

Stellen Sie sicher, dass CSS importiert ist:

import 'element-plus/dist/index.css'

### Symbole werden nicht angezeigt

Sie müssen die Symbolbibliothek installieren:

npm install @element-plus/icons-vue

Dann global registrieren:

import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}

### Import bei Bedarf funktioniert nicht

Überprüfen Sie, ob die Plugin-Konfiguration korrekt ist, und stellen Sie sicher, dass der richtige Resolver verwendet wird.
