# Python FAQ

> Frequently Asked Questions

---

## Installation

### Python command not found?

**Windows**: Ensure "Add Python to PATH" was checked during installation

**macOS/Linux**: Use `python3` instead of `python`

### pip install fails?

**Solution**:

1. Upgrade pip:
```bash
python -m pip install --upgrade pip
```

2. Use China mirror:
```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple package
```

## Environment

### Python 2 and 3 coexistence?

Use `python3` for Python 3, or use pyenv to manage multiple versions.

### Virtual environment?

```bash
python -m venv myenv
source myenv/bin/activate  # Linux/macOS
myenv\Scripts\activate     # Windows
```

## Syntax

### list vs tuple?

- **list**: Mutable, can be modified
- **tuple**: Immutable, cannot be modified after creation

### dict vs set?

- **dict**: Key-value pairs
- **set**: Unique elements, no duplicates

### == vs is?

- `==`: Value equality
- `is`: Identity (same object)

```python
a = [1, 2, 3]
b = [1, 2, 3]

a == b  # True
a is b  # False
```

## Performance

### Python is slow?

1. Use built-in functions and libraries (C implementations)
2. Use list comprehensions
3. Use NumPy, Pandas for optimization

### Profile code?

```bash
python -m cProfile script.py
```

## Common Errors

### IndentationError?

Python uses indentation for code blocks. Use consistent indentation (usually 4 spaces).

### ModuleNotFoundError?

Install missing module:
```bash
pip install modulename
```
