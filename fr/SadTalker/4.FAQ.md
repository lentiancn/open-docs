# SadTalker FAQ

## Questions Générales

### Qu'est-ce que SadTalker?
SadTalker est un projet d'apprentissage profond qui génère des vidéos réalistes de visages parlants à partir d'une image statique et d'une entrée audio.

### SadTalker est-il gratuit?
Oui, SadTalker est open source et gratuit pour un usage personnel et de recherche.

### Quels sont les prérequis système?
- **GPU**: NVIDIA GPU avec 8GB+ VRAM (recommandé)
- **RAM**: 8GB minimum, 16GB recommandé
- **Stockage**: 20GB+ pour les modèles et sorties

## Questions d'Installation

### Comment installer SadTalker?
```bash
# Cloner le dépôt
git clone https://github.com/WinfredSadTalker/SadTalker.git
cd SadTalker

# Créer environnement conda
conda create -n sadtalker python=3.8
conda activate sadtalker

# Installer les dépendances
pip install -r requirements.sh
```

### Quelle version de Python utiliser?
SadTalker fonctionne mieux avec Python 3.8 ou 3.9.

## Questions d'Utilisation

### Comment générer une vidéo parlante?
```bash
python inference.py --source_image path/to/image.jpg --audio path/to/audio.wav
```

### Quels formats d'image sont supports?
- JPG/JPEG
- PNG
- La plupart des formats courants

### Quels formats audio sont supports?
- WAV (recommandé)
- MP3
- Autres formats supportés par FFmpeg

## Dépannage

### Le modèle ne se télécharge pas
Téléchargez manuellement les modèles depuis le dépôt officiel.

### Erreurs de mémoire
- Réduire la taille du batch
- Baisser la résolution de sortie
- Fermer d'autres applications GPU

### Mauvaise qualité
- Utiliser des images d'entrée haute résolution
- Utiliser un audio propre sans bruit de fond
