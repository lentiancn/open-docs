# MultiTalk 簡介

MultiTalk 是由梅津人工智慧團隊開發的一個音訊驅動多人對話影片生成框架。該專案旨在解決現有數字人生成技術僅支援單人物動畫的局限性，能夠根據多流音訊輸入、參考圖像和文字提示生成包含多人互動的影片內容。

## 專案背景

傳統的音訊驅動人類動畫方法，如 talking head 和 talking body 生成技術，在生成同步面部動作和高質量影片方面已取得顯著進展。然而，現有方法主要關注單人物動畫，難以處理多流音訊輸入，存在音訊與人物綁定錯誤的問題。此外，這些方法在指令跟隨能力方面也存在局限。MultiTalk 專案的誕生正是為了解決這些挑戰。

## 核心功能

MultiTalk 的核心功能是實現多人對話影片的自動生成。使用者只需提供一段包含多人對話的音訊、一張包含多人的參考圖像，以及描述影片場景的文字提示，系統即可自動生成對應的對話影片。生成的人物口型與音訊高度同步，畫面流暢自然。

## 技術特點

MultiTalk 採用多項創新技術來解決多人影片生成的難題。首先，專案提出了標籤旋轉位置嵌入方法，用於解決多流音訊注入時音訊與人物的錯誤綁定問題。其次，透過部分參數訓練和多任務訓練策略，有效保留了基礎模型的指令跟隨能力。此外，專案還支援自適應人物定位功能，能夠根據參考圖像中人物區域的特徵在生成影片中準確定位對應人物。

## 應用場景

MultiTalk 的應用場景十分廣泛。在內容創作領域，創作者可以使用該工具快速生成數字人對話影片，用於教育講解、創意表達或娛樂內容製作。在研究領域，該專案為多人影片生成提供了新的研究方向和技術基準。在商業領域，企業可以將其應用於虛擬主播、多人客服、數字人會議等場景。

## 效能表現

MultiTalk 在多項基準測試中展現出優異效能。專案在 talking head、talking body 以及多人物資料集上均取得了領先的生成效果，口型同步準確率達到業界領先水平。專案支援生成 480P 和 720P 解析度的影片，最長可生成 15 秒的影片片段。

## 專案資訊

MultiTalk 是開源專案，程式碼和模型權重可在 GitHub 和 Hugging Face 上獲取。該專案相關論文已發表於學術平台 arXiv，詳細介紹了技術方法和實驗結果。專案採用 Apache 2.0 許可證，允許學術和商業使用。
