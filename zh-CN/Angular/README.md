# Angular

Angular 是由 Google 开发维护的开源 Web 应用框架，用于构建现代 Web 应用程序。采用 TypeScript 作为主要开发语言，提供组件化架构、双向数据绑定、依赖注入等强大功能。

## 快速开始

### 安装 Angular CLI

```bash
npm install -g @angular/cli
```

### 创建新项目

```bash
ng new my-app
cd my-app
```

### 启动开发服务器

```bash
ng serve
```

访问 http://localhost:4200

## 核心特性

- **组件化架构**：模块化、可复用的 UI 组件
- **TypeScript**：强类型支持，提高代码质量
- **依赖注入**：松耦合，易于测试
- **双向数据绑定**：数据自动同步
- **路由**：完整的路由解决方案
- **HTTP 客户端**：强大的网络请求能力
- **表单处理**：响应式表单和模板驱动表单
- **RxJS**：响应式编程支持

## 常用命令

| 命令 | 说明 |
|------|------|
| ng new xxx | 创建新项目 |
| ng serve | 启动开发服务器 |
| ng build | 构建生产版本 |
| ng generate component xxx | 生成组件 |
| ng generate service xxx | 生成服务 |
| ng generate module xxx | 生成模块 |
