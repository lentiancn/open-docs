## 安裝方式

### Windows 安裝

#### 使用 MSI 安裝包

1. 下載 Go：https://go.dev/dl/
2. 執行 MSI 安裝包
3. 遵循安裝精靈
4. 打開命令提示字元，輸入 `go version` 驗證

#### 使用 Chocolatey

```powershell
choco install golang
```

### macOS 安裝

#### 使用 Homebrew

```bash
brew install go
```

#### 使用 DMG 安裝包

1. 下載 macOS 安裝包：https://go.dev/dl/
2. 打開包檔案並執行
3. 將 Go 目錄移動到 /usr/local/go：`sudo mv go /usr/local/`

### Linux 安裝

#### 使用套件管理器

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install golang-go

# CentOS/RHEL
sudo yum install golang
```

#### 手動安裝

```bash
# 下載 Go 二進制包
wget https://go.dev/dl/go1.22.0.linux-amd64.tar.gz

# 解壓到 /usr/local
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go1.22.0.linux-amd64.tar.gz

# 添加到 PATH
export PATH=$PATH:/usr/local/go/bin

# 驗證安裝
go version
```

## 環境設定

### GOPATH 設定

```bash
# Linux/macOS
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin

# Windows (PowerShell)
$env:GOPATH = "$env:USERPROFILE\go"
$env:PATH += ";$env:GOPATH\bin"
```

### Go Module 代理設定

```bash
# 使用國內鏡像
go env -w GOPROXY=https://goproxy.cn,direct

# 設定代理
go env -w GOPROXY=https://proxy.golang.org,direct
```

### 驗證安裝

```bash
# 查看版本
go version

# 查看環境資訊
go env

# 執行測試
go version
```

## 常用指令

```bash
# 初始化模組
go mod init <模組名>

# 下載依賴
go mod tidy

# 建構專案
go build

# 執行程式
go run main.go

# 執行測試
go test ./...

# 安裝工具
go install
```

## 故障排除

### 安裝失敗？

- 檢查系統相容性
- 確認下載的版本與系統架構匹配
- 檢查權限是否足夠

### 指令找不到？

- 確認 Go 已添加到 PATH 環境變數
- 重新打開終端機
- 檢查安裝路徑

### 依賴下載慢？

- 設定國內鏡像代理
- 使用 VPN
