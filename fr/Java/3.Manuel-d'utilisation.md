# Manuel d'utilisation Java

## Votre premier programme Java

### Créer Hello World

Créez un nouveau fichier texte nommé HelloWorld.java. Notez que le nom du fichier doit correspondre exactement au nom de la classe.

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

### Compiler le programme

Ouvrez le terminal ou l'invite de commande, accédez au répertoire du fichier et exécutez la commande de compilation :

```bash
javac HelloWorld.java
```

Après une compilation réussie, un fichier HelloWorld.class sera généré.

### Exécuter le programme

```bash
java HelloWorld
```

Sortie :

```
Hello, World!
```

## Syntaxe de base Java

### Variables et types de données

Java est un langage à typage statique. Les variables doivent être déclarées avant utilisation.

```java
// Types de données primitifs
int age = 25;
double salary = 5000.50;
boolean isActive = true;
char grade = 'A';
String name = "Jean";

// Types de référence
Integer count = 10;
Double price = 99.99;
```

### Opérateurs

```java
// Opérateurs arithmétiques
int a = 10, b = 3;
int sum = a + b;      // Addition
int diff = a - b;     // Soustraction
int prod = a * b;     // Multiplication
int quot = a / b;     // Division
int rem = a % b;      // Reste

// Opérateurs de comparaison
boolean result = (a > b);

// Opérateurs logiques
boolean flag = (a > 5) && (b < 10);
```

### Instructions conditionnelles

```java
// Instruction if-else
if (score >= 90) {
    System.out.println("Excellent");
} else if (score >= 60) {
    System.out.println("Réussi");
} else {
    System.out.println("Échoué");
}

// Instruction switch
switch (day) {
    case 1:
        System.out.println("Lundi");
        break;
    case 2:
        System.out.println("Mardi");
        break;
    default:
        System.out.println("Inconnu");
}
```

### Instructions de boucle

```java
// Boucle for
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}

// Boucle while
int i = 0;
while (i < 10) {
    System.out.println(i);
    i++;
}

// Boucle do-while
do {
    System.out.println(i);
    i++;
} while (i < 10);

// for-each pour les tableaux
int[] numbers = {1, 2, 3, 4, 5};
for (int num : numbers) {
    System.out.println(num);
}
```

## Programmation orientée objet

### Classes et objets

```java
// Définir une classe
public class Person {
    // Champs
    private String name;
    private int age;
    
    // Constructeur
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // Getter et Setter
    public String getName() {
        return name;
    }
    
    public void setName(String name) {
        this.name = name;
    }
    
    // Méthode
    public void sayHello() {
        System.out.println("Bonjour, je suis " + name);
    }
}

// Créer un objet
Person person = new Person("Jean", 25);
person.sayHello();
```

## Expressions Lambda

```java
// Classe interne anonyme
Runnable r1 = new Runnable() {
    @Override
    public void run() {
        System.out.println("En cours d'exécution");
    }
};

// Expression lambda
Runnable r2 = () -> System.out.println("En cours d'exécution");

// Lambda avec paramètres
Comparator<String> c = (s1, s2) -> s1.compareTo(s2);
```

## API Stream

```java
import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);

// Filtrer
List<Integer> evenNumbers = numbers.stream()
    .filter(n -> n % 2 == 0)
    .collect(Collectors.toList());

// Mapper
List<Integer> squared = numbers.stream()
    .map(n -> n * n)
    .collect(Collectors.toList());

// Sommer
int sum = numbers.stream()
    .mapToInt(Integer::intValue)
    .sum();
```

## Opérations sur les fichiers

## Programmation multithread

### Créer des threads

```java
// Étendre la classe Thread
class MyThread extends Thread {
    @Override
    public void run() {
        System.out.println("Thread en cours");
    }
}

// Implémenter l'interface Runnable
class MyRunnable implements Runnable {
    @Override
    public void run() {
        System.out.println("Thread en cours");
    }
}

// Démarrer le thread
MyThread thread = new MyThread();
thread.start();

// Utiliser Runnable
Thread runnableThread = new Thread(new MyRunnable());
runnableThread.start();
```

### Synchronisation des threads

```java
public class Counter {
    private int count = 0;
    
    public synchronized void increment() {
        count++;
    }
    
    public synchronized int getCount() {
        return count;
    }
}
```

## Programmation réseau

### Communication TCP

```java
import java.net.*;
import java.io.*;

ServerSocket serverSocket = new ServerSocket(8080);
Socket clientSocket = serverSocket.accept();
BufferedReader in = new BufferedReader(
    new InputStreamReader(clientSocket.getInputStream()));
PrintWriter out = new PrintWriter(
    clientSocket.getOutputStream(), true);

String message = in.readLine();
out.println("Serveur a reçu: " + message);
```

### Requêtes HTTP

```java
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

HttpClient client = HttpClient.newHttpClient();
HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create("https://api.example.com/data"))
    .GET()
    .build();

HttpResponse<String> response = client.send(request, 
    HttpResponse.BodyHandlers.ofString());
System.out.println(response.body());
```

## Opérations de base de données

### Connexion JDBC

```java
import java.sql.*;

String url = "jdbc:mysql://localhost:3306/mydb";
String user = "root";
String password = "password";

try (Connection conn = DriverManager.getConnection(url, user, password)) {
    String sql = "SELECT * FROM users WHERE id = ?";
    PreparedStatement stmt = conn.prepareStatement(sql);
    stmt.setInt(1, 1);
    
    ResultSet rs = stmt.executeQuery();
    while (rs.next()) {
        System.out.println(rs.getString("name"));
    }
} catch (SQLException e) {
    e.printStackTrace();
}
```
