# Guide d'installation Docker

Instructions pour installer Docker sur toutes les plateformes.

---

## Docker Desktop (Recommandé)

La solution la plus simple pour macOS, Windows et Linux.

### Configuration requise

**macOS:** Apple Silicon/Intel, macOS 10.15+, 4Go RAM
**Windows:** Windows 10/11 64-bit, WSL2 (recommandé)
**Linux:** Ubuntu, Debian, Fedora, Arch Linux

### Télécharger

- [Docker Desktop for Mac](https://desktop.docker.com/mac/main/amd64/Docker.dmg)
- [Docker Desktop for Windows](https://desktop.docker.com/win/main/amd64/Docker%20Desktop%20Installer.exe)

## Linux

### Ubuntu

```bash
sudo apt update
sudo apt install -y ca-certificates curl gnupg l install -m 0755 -d /etc/apt/keysb-release

sudorings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
sudo chmod a+r /etc/apt/keyrings/docker.gpg

echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

sudo apt update
sudo apt install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin

sudo systemctl start docker
sudo systemctl enable docker
sudo usermod -aG docker $USER
```

### CentOS/RHEL

```bash
sudo yum install -y yum-utils
sudo yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
sudo yum install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
sudo systemctl start docker
sudo systemctl enable docker
```

## Windows

### WSL2 (Recommandé)

```powershell
wsl --install
# Redémarrer
sudo apt update
sudo apt install -y docker.io
```
