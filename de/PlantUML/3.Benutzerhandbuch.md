# PlantUML Benutzerhandbuch

## Grundlegende Syntax

### Sequenzdiagramm

```plantuml
@startuml
Alice -> Bob: Hallo
Bob --> Alice: Hi!
@enduml
```

### Klassendiagramm

```plantuml
@startuml
class User {
    +id: int
    +name: string
    +login()
}

class Admin extends User {
    +approve()
}
@enduml
```
