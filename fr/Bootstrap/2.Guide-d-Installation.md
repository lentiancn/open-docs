# Guide d'installation de Bootstrap

> Configuration complète de l'environnement et didacticiel d'installation

---

## Méthodes d'installation

### Méthode 1: CDN (Recommandé pour les projets simples)

```html
<!-- CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- JavaScript -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### Méthode 2: Installation npm

```bash
npm install bootstrap
```

### Méthode 3: Télécharger le code source

Télécharger depuis le site officiel: https://getbootstrap.com/

---

## Initialisation du projet

### Créer un projet avec npm

```bash
mkdir my-bootstrap-project
cd my-bootstrap-project
npm init -y
npm install bootstrap
```

### Créer un fichier HTML

```html
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Projet Bootstrap</title>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <h1>Bonjour Bootstrap!</h1>
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## Personnalisation Sass (Utilisateurs avancés)

