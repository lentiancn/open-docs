# Docker Installationsanleitung

Diese Anleitung zeigt verschiedene Methoden zur Installation von Docker, damit Sie schnell eine Docker-Umgebung einrichten können.

## Installationsmethoden im Überblick

Sie können Docker auf verschiedene Weise installieren:

1. **Docker Desktop**: Enthält Docker Engine in einer Desktop-Anwendung, empfohlen für die meisten Benutzer
2. **Nur Docker Engine**: Installiert nur die Kern-Docker-Komponenten, geeignet für Linux-Server
3. **Docker Engine über apt-Repository**: Installation aus dem offiziellen Docker-apt-Repository
4. **Manuelle Installation**: DEB-Pakete herunterladen und manuell installieren

## Docker Desktop Installation

Docker Desktop ist der einfachste und schnellste Weg, um zu beginnen. Es enthält eine vollständige Docker-Entwicklungsumgebung, einschließlich Docker Engine, Docker CLI, Docker Compose, Docker Build und Kubernetes.

### Systemanforderungen

**macOS:**

- macOS 10.15 (Catalina) oder höher
- Mindestens 4 GB RAM
- Mindestens 4 GB verfügbarer Speicherplatz

**Windows:**

- Windows 10 Pro, Enterprise oder Education (64-Bit)
- Mindestens 4 GB RAM
- Hyper-V oder WSL 2 aktiviert

**Linux:**

- Ubuntu 20.04 oder höher
- Debian 11 oder höher
- Fedora 33 oder höher
- Mindestens 4 GB RAM

### Installation unter macOS

1. Laden Sie Docker Desktop für Mac von der offiziellen Website herunter
2. Doppelklicken Sie auf die .dmg-Datei
3. Ziehen Sie das Docker-Symbol in den Ordner "Programme"
4. Starten Sie Docker Desktop

### Installation unter Windows

1. Laden Sie Docker Desktop für Windows herunter
2. Führen Sie den Installer aus
3. Folgen Sie den Anweisungen des Installationsassistenten
4. Starten Sie Docker Desktop neu

### Installation unter Linux (Ubuntu)

```bash
# Aktualisieren Sie die Paketliste
sudo apt-get update

# Installieren Sie erforderliche Abhängigkeiten
sudo apt-get install ca-certificates curl gnupg lsb-release

# Fügen Sie den Docker GPG-Schlüssel hinzu
sudo mkdir -p /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg

# Richten Sie das Repository ein
echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

# Docker installieren
sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

## Docker Engine Installation

Wenn Sie nur Docker Engine ohne Desktop installieren möchten, folgen Sie diesen Schritten.

### Ubuntu und Debian

```bash
sudo apt-get update
sudo apt-get install docker.io
sudo systemctl start docker
sudo systemctl enable docker
```

### Fedora

```bash
sudo dnf install docker-ce
sudo systemctl start docker
sudo systemctl enable docker
```

## Installation überprüfen

Nach der Installation können Sie die Installation mit folgendem Befehl überprüfen:

```bash
docker --version
docker run hello-world
```

Wenn alles funktioniert, sehen Sie eine Willkommensnachricht von Docker.
