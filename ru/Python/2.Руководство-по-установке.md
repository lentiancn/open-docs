# Руководство по установке Python

> Полная настройка

---

## Установка

### Windows

```powershell
choco install python
```

### macOS

```bash
brew install python@3.12
```

### Linux

```bash
sudo apt install python3 python3-pip
```

---

## Пакеты

```bash
pip install requests
python -m venv myenv
```

---

## Проверка

```bash
python --version
```
