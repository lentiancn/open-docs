# Nginx Benutzerhandbuch

## Grundlegende Operationen

```bash
nginx              # Starten
nginx -s stop     # Stoppen
nginx -s reload    # Konfiguration neu laden
nginx -t            # Konfiguration testen
```

## Grundkonfiguration

```nginx
worker_processes auto;
events {
    worker_connections 1024;
}
http {
    server {
        listen 80;
        server_name beispiel.de;
        root /var/www/html;
        location / {
            index index.html;
        }
    }
}
```

## Statische Dateien

```nginx
server {
    listen 80;
    root /var/www/html;
    location / {
        try_files $uri $uri/ =404;
    }
}
```

## Reverse-Proxy

```nginx
location / {
    proxy_pass http://backend;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
}
```

## Load Balancing

```nginx
upstream backend {
    server backend1.beispiel.de;
    server backend2.beispiel.de;
}
```

## SSL/TLS

```nginx
server {
    listen 443 ssl;
    ssl_certificate zertifikat.crt;
    ssl_certificate_key zertifikat.key;
    ssl_protocols TLSv1.2 TLSv1.3;
}
```

## Häufige Direktiven

- `root`: Dokumentenstammverzeichnis
- `index`: Standard-Indexdateien
- `proxy_pass`: Proxy-Backend
- `upstream`: Backend-Servergruppe
- `gzip`: Komprimierung aktivieren
- `expires`: Browser-Cache
