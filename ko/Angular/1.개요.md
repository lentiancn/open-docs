# Angular 개요

현대 웹 애플리케이션 개발 프레임워크.

## Angular이란?

Angular는 Google이 개발하고 유지 관리하는 오픈소스 웹 애플리케이션 프레임워크입니다. 현대 웹 애플리케이션 개발을 위한 선호되는 솔루션 중 하나입니다.

### 주요 기능

- **네이티브 TypeScript**: 완전한 TypeScript 기반
- **컴포넌트 기반 아키텍처**: 유지보수가 쉬운 모듈식 개발
- **의존성 주입**: 더 나은 테스트 가능성을 위한 강력한 DI 시스템
- **디렉티브 시스템**: HTML 기능을 확장하는 디렉티브
- **라우팅**: 완전한 클라이언트 사이드 라우팅 솔루션
- **폼**: 반응형 폼과 템플릿 기반 폼
- **HTTP 클라이언트**: 완전한 HTTP 클라이언트 서비스
- **국제화**: 내장된 i18n 지원
- **애니메이션**: 강력한 애니메이션 API
- **테스트**: 완전한 단위 및 E2E 테스트 지원

### 기술 아키텍처

Angular는 핵심 개념과 모듈식 아키텍처를 사용합니다:

- **모듈**: 애플리케이션 코드를 구성하는 기본 단위
- **컴포넌트**: 뷰와 비즈니스 로직의 조합
- **서비스**: 캡슐화된 재사용 가능한 비즈니스 로직
- **디렉티브**: HTML 요소의 동작을 확장
- **파이프**: 데이터 변환 유틸리티

## 사용 사례

- **엔터프라이즈 애플리케이션**: 대규모 복잡한 비즈니스 시스템
- **싱글 페이지 애플리케이션(SPA)**: 부드러운 사용자 경험이 필요한 앱
- **프로그레시브 웹 앱(PWA)**: 오프라인 기능이 필요한 앱
- **하이브리드 모바일 앱**: Ionic과 결합하여 모바일 앱 제작
- **대시보드**: 데이터 시각화 및 관리 패널
