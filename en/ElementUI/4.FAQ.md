# Element UI FAQ

This page lists common issues and solutions when using Element UI.

## Installation Issues

### Installation Failed

**Symptoms**: npm install fails

**Solutions**:

1. Check Node.js version, ensure >= 8.0.0
2. Clear npm cache:
   ```bash
   npm cache clean --force
   ```
3. Delete node_modules and package-lock.json, then reinstall

### On-Demand Import Not Working

**Symptoms**: Bundle size still large

**Solutions**:

1. Ensure .babelrc configuration is correct
2. Check if babel-plugin-component is properly installed

## Usage Issues

### Component Styles Not Applied

**Solutions**:

1. Make sure CSS is imported:
   ```javascript
   import 'element-ui/lib/theme-chalk/index.css'
   ```

### Icons Not Displayed

**Solutions**:

1. Import icon fonts:
   ```javascript
   import 'element-ui/lib/theme-chalk/fonts/element-icons.ttf'
   ```

## Vue Version Issues

### Not Compatible with Vue 3

Element UI only supports Vue 2. Use Element Plus for Vue 3.

## Style Issues

### Style Conflicts

Use more specific selectors or ::deep()

## Performance Issues

### Page Lag

For large datasets:
- Use pagination
- Avoid complex components in tables

### Bundle Size Too Large

- Use on-demand import
- Remove unused components
