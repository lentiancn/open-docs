# MultiTalk Manuel Utilisateur

Guide détaillé de MultiTalk.

## Créer une Salle

### Étape 1: Créer une Nouvelle Salle

1. Cliquez sur "Créer une Salle"
2. Entrez le nom de la salle
3. Configurez le mot de passe (optionnel)
4. Configurez les paramètres de la salle

### Étape 2: Inviter des Participants

- Partagez le lien de la salle
- Partagez le code de la salle

### Étape 3: Gérer la Salle

- Activer/désactiver le son
- Ajuster les volumes des participants
- Expulser ou rendre muet des participants

## Joindre une Salle

### Méthode 1: Lien de Salle

1. Cliquez sur le lien de la salle
2. MultiTalk s'ouvrira automatiquement
3. Entrez le mot de passe si nécessaire

### Méthode 2: Code de Salle

1. Cliquez sur "Rejoindre une Salle"
2. Entrez le code de la salle
3. Entrez le mot de passe si nécessaire

## Paramètres Audio

### Activation Vocale

1. Allez dans Paramètres > Audio
2. Activez l'activation vocale
3. Ajustez le niveau de seuil

## Raccourcis Clavier

| Raccourci | Action |
|-----------|--------|
| Ctrl+Shift+T | Push-to-talk |
| Ctrl+M | Muet/son |
