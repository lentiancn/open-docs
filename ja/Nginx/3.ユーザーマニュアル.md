# Nginx ユーザーマニュアル

## 基本操作

### Nginx の開始

```bash
nginx
systemctl start nginx
```

### 停止

```bash
nginx -s stop   # 高速停止
nginx -s quit   # 優雅停止
```

### 設定再読み込み

```bash
nginx -s reload
```

### 設定テスト

```bash
nginx -t
```

## 設定ファイル構造

```nginx
worker_processes auto;
events {
    worker_connections 1024;
}
http {
    server {
        listen 80;
        server_name example.com;
        root /var/www/html;
        location / {
            index index.html;
        }
    }
}
```

## 静的ファイルサービス

```nginx
server {
    listen 80;
    root /var/www/html;
    location / {
        try_files $uri $uri/ =404;
    }
}
```

## リバースプロキシ

```nginx
location / {
    proxy_pass http://backend;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
}
```

## ロードバランシング

```nginx
upstream backend {
    server backend1.example.com;
    server backend2.example.com;
}
location / {
    proxy_pass http://backend;
}
```

## SSL/TLS

```nginx
server {
    listen 443 ssl http2;
    ssl_certificate /path/to/cert.crt;
    ssl_certificate_key /path/to/cert.key;
    ssl_protocols TLSv1.2 TLSv1.3;
}
```

## 一般的な指令

- `root`: ドキュメントルート
- `index`: デフォルトインデックスファイル
- `proxy_pass`: プロキシバックエンド
- `upstream`: バックエンドサーバーグループ
- `gzip`: 圧縮を有効化
- `expires`: ブラウザキャッシュ
