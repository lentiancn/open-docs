# Mockito

Javaユニットテスト向けの主要なモックフレームワーク。

## Mockitoとは？

Mockitoは、Javaユニットテスト向けの主要なモックフレームワークです。開発者が分離したテストを作成し、外部コンポーネントへの依存関係を排除するためのモックオブジェクトを作成できます。

### 主な機能

- **直感的なAPI** - 学びやすく使いやすい
- **柔軟なモック作成** - クラスやインターフェースのモックを作成
- **検証** - メソッド呼び出しの検証
- **スタブ設定** - 定義済み戻り値の設定
