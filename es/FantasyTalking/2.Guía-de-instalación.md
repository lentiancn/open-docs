# FantasyTalking Guía de Instalación

## Requisitos

Antes de instalar FantasyTalking, asegúrate de que tu entorno de desarrollo cumpla los siguientes requisitos. En cuanto a sistemas operativos, el proyecto soporta Windows, macOS y Linux. Se recomienda usar un sistema de 64 bits para un rendimiento óptimo. Se requiere Node.js versión 14.0 o superior, y Python versión 3.7 o superior. Si usas el backend de Python, también necesitas instalar la herramienta de procesamiento de audio FFmpeg.

En cuanto al hardware, para una experiencia de síntesis de voz fluida, se recomienda que el sistema tenga al menos 4GB de RAM y un procesador de doble núcleo. Para aplicaciones que requieren síntesis de voz en tiempo real, se recomienda usar almacenamiento SSD para reducir la latencia de E/S. Si planeas entrenar modelos o desarrollar voces personalizadas, necesitas una tarjeta gráfica con soporte CUDA.

## Métodos de Instalación

FantasyTalking soporta múltiples métodos de instalación. Los desarrolladores pueden elegir el más adecuado según sus necesidades y stack tecnológico.

### Usando npm (Node.js)

```bash
# Instalar librería core
npm install fantasy-talking

# Instalar componentes opcionales
npm install fantasy-talking-ui  # Componentes UI web
npm install fantasy-talking-node  # Soporte backend Node.js
```

### Usando pip (Python)

```bash
# Instalar librería core
pip install fantasy-talking

# Instalar componentes opcionales
pip install fantasy-talking[ui]  # Componentes UI web
pip install fantasy-talking[all]  # Todos los componentes opcionales
```

### Compilando desde el Código Fuente

```bash
# Clonar repositorio
git clone https://github.com/fantasytalking/fantasy-talking.git
cd fantasy-talking

# Instalar dependencias Node.js
npm install

# Instalar dependencias Python (opcional)
pip install -r requirements.txt

# Compilar proyecto
npm run build
```

## Inicio Rápido

### Ejemplo JavaScript/TypeScript

```javascript
const FantasyTalking = require('fantasy-talking');

// Inicializar cliente
const client = new FantasyTalking({
  apiKey: 'your-api-key',
  voice: {
    style: 'neutral',
    language: 'es-ES',
    speed: 1.0,
    pitch: 1.0
  }
});

// Sintetizar voz
async function synthesize() {
  const audio = await client.speak('Hola, esta es una prueba de FantasyTalking');
  console.log('Síntesis de voz exitosa');
  
  // Guardar en archivo
  const fs = require('fs');
  fs.writeFileSync('output.wav', audio);
  
  // O reproducir directamente
  client.play(audio);
}

synthesize();
```

## Verificación de Instalación

Después de la instalación, puedes verificar que FantasyTalking funciona correctamente:

```bash
# Probar con herramienta de línea de comandos
fantasy-talking --text "Texto de prueba" --output test.wav

# O ejecutar ejemplo
npm run example
```

Si todo está bien, deberías ver el archivo de voz generado exitosamente. Para problemas comunes de instalación, consulta la página de preguntas frecuentes.
