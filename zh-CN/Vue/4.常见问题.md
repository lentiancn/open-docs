# Vue.js 常见问题

常见问题解答。

## 安装问题

### Q: npm 安装失败？

**解决方案**：
1. 检查 Node.js 版本
2. 清理 npm 缓存
3. 使用 --legacy-peer-deps

### Q: 创建项目失败？

**解决方案**：
1. 检查网络连接
2. 使用 npm create vue@latest

## 开发问题

### Q: 响应式数据不更新？

**解决方案**：
1. 使用 ref 或 reactive
2. 不要直接修改数组索引
3. 使用浅层 ref

### Q: 组件不显示？

**解决方案**：
1. 检查组件导入
2. 检查模板语法
3. 检查控制台错误

### Q: 样式不生效？

**解决方案**：
1. 检查 scoped
2. 检查 CSS 选择器
3. 使用深度选择器

## 性能问题

### Q: 应用太慢？

**解决方案**：
1. 使用 v-show 代替 v-if
2. 使用 keep-alive
3. 懒加载路由

### Q: 内存泄漏？

**解决方案**：
1. 清理定时器
2. 清理事件监听器

## 路由问题

### Q: 路由不生效？

**解决方案**：
1. 检查路由配置
2. 检查 router-view 位置

## 状态管理问题

### Q: 状态不更新？

**解决方案**：
1. 检查 store 修改方式
2. 使用 reactive 时保持引用
