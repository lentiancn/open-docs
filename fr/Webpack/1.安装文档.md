# Webpack Guide d'installation

Webpack est un bundler de modules pour applications JavaScript modernes.

## Installation

```bash
npm install webpack webpack-cli --save-dev
```

## Configuration

```javascript
module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
  },
};
```
