# Element UI 자주 묻는 질문

이 페이지는 Element UI 사용 시 일반적인 문제와 해결 방법을 나열합니다.

## 설치 문제

### 설치 실패

**증상**: npm install 실패

**해결 방법**:

1. Node.js 버전 확인, 8.0.0 이상인지
2. npm 캐시 지우기:
   ```bash
   npm cache clean --force
   ```
3. node_modules와 package-lock.json을 삭제하고 다시 설치

### 필요 시 가져오기가 작동하지 않음

**증상**: 번들 크기가 여전히 큼

**해결 방법**:

1. .babelrc 구성이 올바른지 확인
2. babel-plugin-component가 제대로 설치되어 있는지 확인

## 사용 문제

### 컴포넌트 스타일이 적용되지 않음

**해결 방법**:

1. CSS가 가져와졌는지 확인:
   ```javascript
   import 'element-ui/lib/theme-chalk/index.css'
   ```

### 아이콘이 표시되지 않음

**해결 방법**:

1. 아이콘 폰트 가져오기:
   ```javascript
   import 'element-ui/lib/theme-chalk/fonts/element-icons.ttf'
   ```

## Vue 버전 문제

### Vue 3와 호환되지 않음

Element UI는 Vue 2만 지원합니다. Vue 3는 Element Plus를 사용하세요.

## 스타일 문제

### 스타일 충돌

더 구체적인 선택자 또는 ::deep() 사용

## 성능 문제

### 페이지 버벅임

대용량 데이터셋의 경우:
- 페이지네이션 사용
- 테이블에서 복잡한 컴포넌트 피하기

### 번들 크기가 너무 큼

- 필요 시 가져오기 사용
- 사용하지 않는 컴포넌트 제거
