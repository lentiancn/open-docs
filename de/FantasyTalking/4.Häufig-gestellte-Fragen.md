# FantasyTalking Häufig Gestellte Fragen

## Installationsprobleme

### Was tun, wenn die Installation fehlschlägt?

Bei der Installation können verschiedene Probleme auftreten. Zuerst sollten Sie sicherstellen, dass die Umgebung die Anforderungen erfüllt. Überprüfen Sie die Node.js-Version mit `node --version` (muss >= 14.0 sein). Wenn Sie Python verwenden, stellen Sie sicher, dass Python 3.7 oder höher ist.

Wenn die Installation weiterhin fehlschlägt, versuchen Sie Folgendes: npm-Cache leeren mit `npm cache clean --force`; Spiegelserver verwenden mit `npm config set registry https://registry.npmmirror.com`; Oder aus dem Quellcode installieren.

### FFmpeg Installationsprobleme

Auf einigen Systemen ist FFmpeg möglicherweise nicht richtig installiert oder konfiguriert. Sie können vorkompilierte Binärdateien von der FFmpeg-Website herunterladen oder mit Paketmanagern installieren: Verwenden Sie `brew install ffmpeg` auf macOS, `apt-get install ffmpeg` oder `yum install ffmpeg` auf Linux.

## Nutzungsprobleme

### Was tun, wenn die Sprachsynthese langsam ist?

Die Geschwindigkeit der Sprachsynthese hängt von mehreren Faktoren ab. Überprüfen Sie zuerst die Netzwerkverbindung. Wenn Sie Cloud-Dienste verwenden, beeinflusst die Netzwerklatenz die Antwortzeit. Erwägen Sie die Nutzung selbst gehosteter Dienste, um den Overhead zu reduzieren. Für große Syntheseaufgaben empfiehlt sich die Stapelverarbeitungsschnittstelle.

### Wie wähle ich den richtigen Sprachstil?

FantasyTalking bietet verschiedene Sprachstile: neutral, happy, sad, angry, excited. Wählen Sie basierend auf dem Anwendungsszenario: Verwenden Sie neutral oder professional für Kundenservice; Wählen Sie verschiedene Stile für Bildungsanwendungen basierend auf dem Inhalt.

### Wie implementiere ich englische Sprachsynthese?

Sie müssen Englisch in der Standardkonfiguration angeben:

```javascript
const audio = await client.speak('Hello, world!', {
  voice: {
    language: 'en-US',
    style: 'neutral'
  }
});
```

### SSML-Markup funktioniert nicht?

Stellen Sie sicher, dass Sie den `ssml: true` Parameter beim Aufruf der speak-Funktion übergeben. Überprüfen Sie, ob das SSML-Markup korrekt ist - Tagnamen sollten Kleinbuchstaben sein.

## Konten und Abrechnung

### Wie erhalte ich einen API-Schlüssel?

Registrieren Sie sich auf der FantasyTalking-Website, um ein Konto zu erstellen, und erstellen Sie dann einen API-Schlüssel in der Konsole. Neue Benutzer erhalten normalerweise kostenlose Credits zum Testen.

### Wie kann ich die Nutzung anzeigen?

Nach der Anmeldung bei der Konsole können Sie API-Aufrufe und -Verbrauch im Dashboard anzeigen.

## Bereitstellungsprobleme

### Docker-Bereitstellung fehlgeschlagen?

Stellen Sie zuerst sicher, dass Docker ordnungsgemäß installiert ist und läuft. Überprüfen Sie, ob Port 8080 belegt ist. Wenn Sie Cloud-Dienste verwenden, stellen Sie sicher, dass Sicherheitsgruppen die entsprechenden Ports geöffnet haben.

## Sonstiges

### Welche Plattformen unterstützt FantasyTalking?

FantasyTalking unterstützt gängige Plattformen und Frameworks: Web unterstützt alle modernen Browser, Mobile unterstützt iOS und Android, Desktop unterstützt Windows, macOS und Linux, Server unterstützt Node.js und Python.

### Wie kann ich zum Projekt beitragen?

Beiträge zu FantasyTalking sind willkommen! Sie können teilnehmen durch: Probleme auf GitHub melden; Pull-Requests einreichen; Dokumentation verbessern; An Community-Diskussionen teilnehmen.
