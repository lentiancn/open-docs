# OpenClaw FAQ

> Frequently Asked Questions about OpenClaw

---

## General

### What is OpenClaw?

OpenClaw is an AI assistant framework that leverages advanced agent technology to automate tasks. It combines various tools for file management, browser control, and more.

### Who is OpenClaw for?

OpenClaw is for developers, power users, and anyone looking to increase productivity through AI-powered automation.

---

## Installation

### What are the system requirements?

| Requirement | Minimum |
|-------------|---------|
| Operating System | Windows 10+, macOS, Linux |
| Node.js | Version 18+ |
| RAM | 4 GB |
| Storage | 500 MB |

### Installation fails?

**Solution**:
1. Check Node.js version: `node --version`
2. Clear npm cache: `npm cache clean --force`
3. Try with admin/root privileges
4. Document error message

### Gateway won't start?

**Solution**:
1. Check port: `netstat -ano | findstr 3000`
2. Use different port: `openclaw gateway start --port 3001`
3. Review logs: `openclaw gateway logs`

## Usage

### How do I start a conversation?

```bash
openclaw chat
```

### How do I execute a single command?

```bash
openclaw exec "my command"
```

### Can I use OpenClaw headless?

Yes, OpenClaw can operate in headless mode. Start the Gateway service and use the API or CLI.

---

## File Management

### How do I read a file?

Use the read tool:
```
read /path/to/file.txt
```

### How do I create a new file?

Use the write tool:
```
write --content "content" --path /new/file.txt
```

### How do I edit a file?

Use the edit tool for precise changes:
```
edit --path /file.txt --oldText "old" --newText "new"
```

## Browser Control

### Which browsers are supported?

- Chrome (via extension)
- Isolated OpenClaw browser
- Others via browser automation

### How do I control the browser?

```bash
browser start --profile openclaw
browser navigate --url https://example.com
browser screenshot --path /output.png
```

### Browser won't start?

**Solution**:
1. Install Chrome extension
2. Check firewall settings
3. Clear browser cache

## Subagents

### What are subagents?

Subagents are independent instances that can execute specific tasks in parallel or sequentially.

### How do I create a subagent?

```bash
subagents create --task "description"
```

### How do I kill a subagent?

```bash
subagents kill --id agent-id
```

## Security

### Is OpenClaw safe?

OpenClaw implements multiple security layers:
- File permissions
- Confirmation for external actions
- Sandbox options

### How do I protect sensitive data?

1. Use encrypted configuration
2. Don't share passwords in conversations
3. Use environment variables for secrets

## Troubleshooting

### "Command not found"?

Make sure:
1. OpenClaw is correctly installed
2. PATH variable is set correctly
3. Gateway is running

### File operations fail?

1. Check path
2. Verify permissions
3. Check filename for special characters

### Slow response times?

1. Check Gateway status
2. Verify network connection
3. Analyze logs
