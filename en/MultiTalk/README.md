# MultiTalk

MultiTalk is a multi-speaker audio generation tool.

## Introduction

MultiTalk creates natural-sounding multi-voice audio content using deep learning technology.

## Features

- **Multi-Speaker**: Support multiple speakers
- **High Quality**: Natural-sounding output
- **Multi-Language**: Multiple language support
- **Easy to Use**: Simple command-line interface

## Version

| Version | Status |
|---------|--------|
| v1.0 | ✅ Current |

## Quick Start

### Installation

```bash
git clone https://github.com/MultiTalk/MultiTalk.git
cd MultiTalk
pip install -r requirements.txt
```

### Generate Audio

```bash
python inference.py --text "Hello" --speakers speaker1,speaker2 --output output.wav
```
