## Einführung

### Was ist Bootstrap?

Bootstrap ist das weltweit beliebteste Open-Source-Front-End-Toolkit, das von Twitter entwickelt wurde. Es ist ein CSS- und JavaScript-Framework für die schnelle Entwicklung von responsiven Websites und Webanwendungen.

### Bootstrap Versionen

| Version | Veröffentlichungsdatum | Status |
|---------|--------------------------|--------|
| Bootstrap 5.3 | Mai 2023 | Aktuell |
| Bootstrap 5.2 | September 2022 | Unterstützt |
| Bootstrap 5.1 | August 2021 | Unterstützt |

### Bootstrap 5 Funktionen

- Neue Farbschemata
- Verbesserte Formularsteuerelemente
- Neue Utilities API
- Keine jQuery-Abhängigkeit

## Installationsmethoden

### Methode 1: CDN verwenden (Empfohlen)

CDN ist der schnellste Weg zum Starten.

```html
<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap Start</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>Hello, Bootstrap!</h1>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### Methode 2: npm verwenden

```bash
mkdir my-bootstrap-project
cd my-bootstrap-project
npm init -y
npm install bootstrap
```

### Methode 3: Download

Laden Sie die kompilierten Dateien von https://getbootstrap.com/ herunter.

## Installation unter Windows

### CDN verwenden (Empfohlen)

1. Erstellen Sie eine neue HTML-Datei
2. Fügen Sie den Bootstrap CSS-Link in `<head>` hinzu
3. Fügen Sie das Bootstrap JS-Skript am Ende von `<body>` hinzu

### npm verwenden

1. Node.js von https://nodejs.org/ installieren
2. Befehl öffnen:
   ```cmd
   npm install bootstrap
   ```

## Installation unter macOS

### Homebrew verwenden (Empfohlen)

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install node
npm install bootstrap
```

## Installation unter Linux

### Ubuntu/Debian

```bash
sudo apt update
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs

mkdir bootstrap-project
cd bootstrap-project
npm install bootstrap
```

### Fedora

```bash
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -
sudo dnf install nodejs
npm install bootstrap
```

## Verifizierung

Erstellen Sie eine Test-HTML-Datei:

```html
<!DOCTYPE html>
<html lang="de">
<head>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <div class="container mt-5">
        <div class="alert alert-success">Bootstrap installiert!</div>
        <button class="btn btn-primary">Primärer Button</button>
    </div>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

Öffnen Sie die Datei im Browser - Sie sollten grüne Warnungen und blaue Buttons sehen.

## Fehlerbehebung

### Styles funktionieren nicht

- Überprüfen Sie, ob der CSS-Link korrekt ist
- Browser-Cache leeren

### JavaScript funktioniert nicht

- Stellen Sie sicher, dass bootstrap.bundle.min.js verwendet wird (enthält Popper)
- Browser-Konsole auf Fehler prüfen
