# JMeter Installationsanleitung

Dieses Handbuch bietet detaillierte Anweisungen zur Installation und Konfiguration von Apache JMeter auf verschiedenen Betriebssystemen.

## Anforderungen

- **Java-Laufzeitumgebung**: JMeter erfordert Java 8 oder höher
- **Betriebssystem**: Unterstützt jedes Java-kompatible Betriebssystem, einschließlich Linux, Windows und Mac OSX
- **Speicher**: Mindestens 4GB verfügbarer Speicher empfohlen (je nach Testumfang kann mehr erforderlich sein)
- **Festplattenplatz**: Mindestens 1GB freier Speicherplatz empfohlen

## JMeter herunterladen

1. Besuchen Sie die offizielle JMeter-Downloadseite: https://jmeter.apache.org/download_jmeter.cgi
2. Laden Sie die neueste binäre Version von JMeter (Binaries) herunter
3. Es wird empfohlen, auch PGP- oder SHA-512-Signaturdateien herunterzuladen, um die Download-Integrität zu überprüfen

## Windows-Installation

### Entpacken

1. Suchen Sie das heruntergeladene Apache JMeter-Archiv (apache-jmeter-x.x.zip)
2. Wählen Sie ein geeignetes Installationsverzeichnis (vermeiden Sie Pfade mit Leerzeichen und Sonderzeichen)
3. Entpacken Sie das Archiv in das ausgewählte Verzeichnis
4. JMeter ist nun installiert

### Überprüfung

1. Navigieren Sie zum bin-Ordner im JMeter-Installationsverzeichnis
2. Doppelklicken Sie auf jmeter.bat (Windows-Batchdatei)
3. Bei Erfolg wird die JMeter-GUI gestartet

## Linux- und Mac OSX-Installation

### Entpacken

1. Suchen Sie das heruntergeladene Apache JMeter-Archiv (apache-jmeter-x.x.tar.gz)
2. Entpacken Sie mit dem folgenden Befehl:
   ```bash
   tar -xf apache-jmeter-x.x.tar.gz
   ```
3. JMeter ist nun installiert

### Umgebungsvariablen (Optional)

1. Öffnen Sie Ihre Terminal-Konfigurationsdatei (~/.bashrc oder ~/.profile)
2. Fügen Sie die folgenden Umgebungsvariablen hinzu:
   ```bash
   export JMETER_HOME=/path/to/apache-jmeter-x.x
   export PATH=$PATH:$JMETER_HOME/bin:$JMETER_HOME/lib
   ```
3. Speichern und aktualisieren Sie die Konfiguration:
   ```bash
   source ~/.bashrc
   ```

### Überprüfung

1. Führen Sie Folgendes im Terminal aus:
   ```bash
   jmeter --version
   ```
2. Wenn Versionsinformationen angezeigt werden, war die Installation erfolgreich

## JMeter starten

### GUI-Modus

- **Windows**: Doppelklicken Sie auf jmeter.bat im bin-Verzeichnis
- **Linux/Mac**: Führen Sie den Befehl jmeter im Terminal aus

### Befehlszeilenmodus

Für Lasttests wird der Befehlszeilenmodus empfohlen, um den Ressourcenverbrauch zu reduzieren:

```bash
jmeter -n -t test_plan.jmx -l results.jtl -e -o /path/to/output
```

Parameter:
- -n: Im Nicht-GUI-Modus ausführen
- -t: Testplandatei angeben
- -l: Ergebnistatei angeben
- -e: Bericht nach Testende generieren
- -o: Berichtsausgabeverzeichnis angeben

## Verzeichnisstruktur

- **bin**: Enthält ausführbare Dateien und Startskripte
- **lib**: Enthält JMeter-Abhängigkeitsbibliotheken
- **docs**: Enthält die offizielle JMeter-Dokumentation
- **extras**: Enthält zusätzliche Hilfsprogramme
- **printable_docs**: Enthält druckbares Benutzerhandbuch

## Abhängigkeitsverwaltung

Das lib-Verzeichnis von JMeter enthält Kernabhängigkeiten. So fügen Sie zusätzliche Plugins oder Bibliotheken hinzu:

1. Kopieren Sie JAR-Dateien in das lib-Verzeichnis
2. Einige Plugins müssen in das lib/ext-Verzeichnis eingefügt werden
3. Starten Sie JMeter neu, damit neue Plugins wirksam werden

## Häufige Installationsprobleme

- **Java nicht gefunden**: Stellen Sie sicher, dass Java ordnungsgemäß installiert und die JAVA_HOME-Umgebungsvariable gesetzt ist
- **Unzureichender Speicher**: Passen Sie die Heap-Größenparameter in bin/jmeter.sh oder jmeter.bat an
- **Startfehler**: Überprüfen Sie, ob die richtige Java-Version (Java 8 oder höher) installiert ist
