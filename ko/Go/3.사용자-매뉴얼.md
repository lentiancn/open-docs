# Go 사용자 매뉴얼

## 기본 문법

### 변수 선언

Go에는 변수를 선언하는 여러 방법이 있습니다. 기본 선언은 var 키워드를 사용합니다:

```go
// 타입과 함께 변수 선언
var name string = "Go"

// 타입 추론 선언
var age = 10

// 일괄 선언
var (
    firstName string
    lastName  string
    score     float64
)

// 짧은 선언 (함수 내부)
message := "Hello, World!"
```

Go 변수는 선언 후 사용해야 하며, 그렇지 않으면 컴파일 오류가 발생합니다. 일시적으로 사용하지 않을 변수를 선언해야 하는 경우 _를 사용하여 받을 수 있습니다:

```go
result, _ := someFunction()  // 첫 번째 반환 값 무시
```

### 상수 선언

상수는 const 키워드로 선언하며 문자, 문자열, 부울 또는 숫자일 수 있습니다:

```go
const Pi = 3.14159
const (
    StatusOK = 200
    StatusNotFound = 404
)
```

### 기본 데이터 타입

Go의 기본 데이터 타입은 다음과 같습니다:

- 부울: bool (true 또는 false)
- 정수: int, int8, int16, int32, int64, 해당하는 부호 없는 타입 uint 등
- 부동소수점: float32, float64
- 복소수: complex64, complex128
- 문자열: string

타입 변환은 명시적으로 수행해야 합니다:

```go
var i int = 42
var f float64 = float64(i)
var s string = strconv.Itoa(i)  // 정수를 문자열로
```

## 함수

### 함수 정의

Go 함수는 func 키워드를 사용하여 정의합니다:

```go
func greet(name string) string {
    return "Hello, " + name
}

// 여러 반환 값
func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}

// 명명된 반환 값
func split(sum int) (x, y int) {
    x = sum * 4 / 9
    y = sum - x
    return  // 반환 값 지정 불필요
}
```

### 가변 인자

함수는 가변数量的 매개변수를 받을 수 있습니다:

```go
func sum(nums ...int) int {
    total := 0
    for _, n := range nums {
        total += n
    }
    return total
}

// 사용
result := sum(1, 2, 3, 4, 5)
```

### 클로저

Go는 함수가 외부 범위의 변수를 포착하고 수정할 수 있는 클로저를 지원합니다:

```go
func counter() func() int {
    count := 0
    return func() int {
        count++
        return count
    }
}

// 사용
c := counter()
fmt.Println(c())  // 1
fmt.Println(c())  // 2
```

## 동시 프로그래밍

### Goroutine

Goroutine은 생성 비용이 매우 낮은 Go의 가벼운 실행 단위입니다:

```go
// goroutine 시작
go func() {
    fmt.Println("Hello from goroutine!")
}()

// 매개변수가 있는 goroutine
go func(msg string) {
    fmt.Println(msg)
}("Hello!")
```

### 채널

채널은 goroutine 간의 통신 메커니즘입니다:

```go
// 채널 생성
ch := make(chan string)

// 데이터 전송
ch <- "message"

// 데이터 수신
msg := <-ch

// 버퍼 있는 채널
bufferedCh := make(chan int, 10)
```

### Select

select 문은 여러 채널 작업을 동시에 기다리는 데 사용됩니다:

```go
select {
case msg := <-ch1:
    fmt.Println("Received:", msg)
case msg := <-ch2:
    fmt.Println("Received:", msg)
case <-time.After(time.Second):
    fmt.Println("Timeout")
}
```

### 동시성 안전

동시성 제어를 위해 sync 패키지를 사용합니다:

```go
var (
    mu    sync.Mutex
    count int
)

func increment() {
    mu.Lock()
    defer mu.Unlock()
    count++
}
```

## 오류 처리

### 오류 반환

Go는 반환 값을 사용하여 오류 처리하며, 이것이 Go의 관용적인 방식입니다:

```go
result, err := doSomething()
if err != nil {
    return err
}
```

### 사용자 정의 오류

errors 패키지를 사용하여 사용자 정의 오류를 생성할 수 있습니다:

```go
import "errors"

func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}
```

### 오류 래핑

Go 1.13은 fmt.Errorf와 %w 동사를 사용하여 오류 래핑을 도입했습니다:

```go
if err != nil {
    return fmt.Errorf("failed to do something: %w", err)
}
```

## 인터페이스

### 인터페이스 정의

Go 인터페이스는 명시적인 선언 없이 암시적으로 구현됩니다:

```go
type Writer interface {
    Write([]byte) (int, error)
}

type Reader interface {
    Read([]byte) (int, error)
}

// 일반 복합 인터페이스
type ReadWriter interface {
    Reader
    Writer
}
```

### 빈 인터페이스

빈 인터페이스 interface{}는 모든 타입의 값을 저장할 수 있습니다:

```go
var i interface{} = 42
var s interface{} = "hello"
var m interface{} = map[string]int{"a": 1}
```

Go 1.18은 제네릭을 도입하여 interface{} 대신 any를 사용할 수 있습니다:

```go
var i any = 42
```

## 패키지 및 모듈

### 모듈 만들기

go mod init을 사용하여 새 모듈을 만듭니다:

```bash
go mod init github.com/username/project
```

### 종속성 관리

종속성 추가:

```bash
go get github.com/pkg/errors
```

종속성 업데이트:

```bash
go get -u all
```

사용하지 않는 종속성 정리:

```bash
go mod tidy
```

## 테스트

### 단위 테스트

Go에는 내장된 테스트 프레임워크가 있습니다. 테스트 파일은 _test.go로 끝납니다:

```go
import "testing"

func TestAdd(t *testing.T) {
    result := Add(2, 3)
    if result != 5 {
        t.Errorf("Add(2, 3) = %d; want 5", result)
    }
}
```

테스트 실행:

```bash
go test ./...
go test -v ./...
go test -cover ./...
```

### 벤치마크 테스트

벤치마크 함수는 Benchmark로 시작합니다:

```go
func BenchmarkHello(b *testing.B) {
    for i := 0; i < b.N; i++ {
        Hello()
    }
}
```

벤치마크 테스트 실행:

```bash
go test -bench=.
```

## 일반적인 표준 라이브러리

### fmt - 서식화된 I/O

```go
fmt.Println("Hello")
fmt.Printf("Name: %s, Age: %d\n", name, age)
fmt.Sprintf("Result: %d", result)
```

### strings - 문자열 작업

```go
strings.Contains(s, "sub")
strings.Split(s, ",")
strings.TrimSpace(s)
strings.ToUpper(s)
```

### net/http - HTTP 서버

```go
http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
    fmt.Fprint(w, "Hello, HTTP!")
})
http.ListenAndServe(":8080", nil)
```

### json - JSON 처리

```go
// 인코딩
data, _ := json.Marshal(person)

// 디코딩
var p Person
json.Unmarshal(data, &p)
```

## 일반적인 도구

### gofmt - 코드 서식 지정

```bash
gofmt -w main.go  # 서식 지정하고 파일에 쓰기
gofmt -d main.go  # 서식 지정 차이점 표시
```

### go vet - 정적 분석

```bash
go vet ./...  # 코드 문제 확인
```

### go doc - 문서 뷰어

```go
go doc fmt.Println  # 함수 문서 보기
go doc fmt          # 패키지 문서 보기
```
