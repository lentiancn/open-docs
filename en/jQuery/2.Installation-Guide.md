# jQuery Installation Guide

## Getting jQuery

### Latest Version

The current latest version is **jQuery 4.0.0**.

#### Production Minified Version (Recommended)
- Download: [jquery-4.0.0.min.js](https://code.jquery.com/jquery-4.0.0.min.js)
- Suitable for production, small file size, fast loading

#### Development Version
- Download: [jquery-4.0.0.js](https://code.jquery.com/jquery-4.0.0.js)
- Contains full code, easy to read and debug

#### Source Map File
- Download: [jquery-4.0.0.min.map](https://code.jquery.com/jquery-4.0.0.min.map)
- Used to display original source code in browser devtools

#### Slim Version
The Slim version is a smaller version that excludes Ajax and effects modules:
- [jquery-4.0.0.slim.min.js](https://code.jquery.com/jquery-4.0.0.slim.min.js) - minified
- [jquery-4.0.0.slim.js](https://code.jquery.com/jquery-4.0.0.slim.js) - uncompressed

If you don't need Ajax and animation functionality, using the Slim version can reduce file size by about 30%.

## Installation Methods

### Method 1: CDN Include (Recommended)

Using jQuery CDN is the simplest way - no need to download files, just reference directly in HTML:

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
```

Version with Subresource Integrity (SRI) validation (recommended, prevents CDN tampering):

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js" 
        integrity="sha384-KJ3o2DKtIkvYIK3UENzmM7KCkRr/rE9/Qpg6aAZGJwFDMVNA/GpGFF93hXpG5KkN" 
        crossorigin="anonymous"></script>
```

Visit [https://releases.jquery.com](https://releases.jquery.com) to get the complete version list and SRI hashes.

### Method 2: npm Installation

For Node.js projects, install via npm:

```bash
npm install jquery
```

After installation, you can find in node_modules/jquery/dist/:
- jquery-4.0.0.min.js - minified
- jquery-4.0.0.js - uncompressed
- jquery-4.0.0.min.map - source map

### Method 3: Yarn Installation

```bash
yarn add jquery
```

### Method 4: Download Locally

Visit the [jQuery download page](https://jquery.com/download/), right-click the version you need and select "Save as..." to save the file to your local project directory.

## Upgrading jQuery

### Using jQuery Migrate Plugin

When upgrading jQuery, it's recommended to use the jQuery Migrate plugin to identify compatibility issues:

1. Upgrading from older versions to 1.9+ or 3.x: Use [jQuery Migrate 1.4.1](https://code.jquery.com/jquery-migrate-1.4.1.min.js)

2. Upgrading from jQuery 3.x to a newer version: Use [jQuery Migrate 3.6.0](https://code.jquery.com/jquery-migrate-3.6.0.min.js)

Usage: Include the Migrate plugin after jQuery

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
<script src="https://code.jquery.com/jquery-migrate-3.6.0.min.js"></script>
```

Migrate will output compatibility warnings in the console to help you fix issues step by step.

### Upgrade Guides
