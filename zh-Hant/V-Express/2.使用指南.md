# V-Express 使用指南

V-Express 在參考圖像、音頻和 V-Kps 序列的控制下生成說話頭像視頻。

## 快速開始

### 基本命令

```bash
python inference.py \
  --reference_image_path "./test_samples/ref.jpg" \
  --audio_path "./test_samples/audio.mp3" \
  --output_path "./output/result.mp4" \
  --retarget_strategy "no_retarget" \
  --num_inference_steps 25
```

## 命令參數

### 輸入參數

| 參數 | 說明 |
|------|------|
| `--reference_image_path` | 參考人像圖像路徑 |
| `--audio_path` | 輸入音頻文件路徑 |
| `--output_path` | 輸出視頻路徑 |

### 重定向策略

| 策略 | 說明 |
|------|------|
| `no_retarget` | 同一個人（最佳效果）|
| `fix_face` | 任意圖像和音頻 |
| `offset_retarget` | 不同人帶運動 |

## 使用示例

### 場景 1：同一個人

```bash
python inference.py \
  --reference_image_path "./test_samples/ref.jpg" \
  --audio_path "./test_samples/audio.mp3" \
  --output_path "./output/result.mp4" \
  --retarget_strategy "no_retarget"
```

## 輸入要求

### 參考圖像

- 格式：JPG、PNG
- 分辨率：512x512 或更高

### 音頻

- 格式：MP3、WAV
- 時長：1-60 秒
