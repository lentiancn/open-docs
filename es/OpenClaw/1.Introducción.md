# OpenClaw 🦞

OpenClaw es una herramienta que instalas en tu propio dispositivo y que conecta tus aplicaciones de mensajería favoritas con asistentes de IA. WhatsApp, Telegram, Discord, iMessage... todos funcionan directamente con servicios de IA como si fuera un puente entre tus chats y el cerebro artificial.

## ¿Para quién es?

Si quieres tener tu propio asistente de IA pero no quieres confiar tus datos a servicios de terceros, OpenClaw es una excelente opción. Está especialmente diseñado para desarrolladores y usuarios avanzados que prefieren herramientas de línea de comandos y quieren control total sobre su configuración.

## Características principales

- **Tus datos, tu máquina**: Todo permanece en tu propio hardware, nada va a servidores externos
- **Múltiples plataformas**: Un solo proceso maneja varias aplicaciones de chat al mismo tiempo
- **Diseñado para programación**: Soporta herramientas, historial de conversaciones, memoria y múltiples asistentes
- **Código abierto**: Licencia MIT, comunidad activa contribuyendo

## Qué necesitas

- Node.js versión 22 o superior
- Una clave API de tu proveedor de IA favorito
- Algunos minutos para la configuración inicial

Para la mejor experiencia, usa los modelos de última generación más potentes disponibles.

## Inicio rápido

```bash
# Instalar OpenClaw
npm install -g openclaw@latest

# Ejecutar el asistente de configuración
openclaw onboard --install-daemon

# Iniciar el servicio
openclaw gateway --port 18789
```

Una vez ejecutándose, abre tu navegador en http://127.0.0.1:18789/ para comenzar.

## Cómo funciona

OpenClaw funciona con un proceso central llamado Gateway. Este proceso es el responsable de enrutar mensajes, gestionar las llamadas a la IA y mantener las conexiones con todos tus canales de chat. Solo necesitas mantener este proceso en ejecución y automáticamente tendrás capacidades de IA en todas tus plataformas de mensajería simultáneamente.

## Capacidades principales

- **Gestión centralizada**: Un sistema para controlar todos tus canales de chat
- **Arquitectura de plugins**: Añade más plataformas a través de paquetes de extensión
- **Múltiples asistentes**: Ejecuta instancias de IA separadas para diferentes propósitos
- **Manejo de archivos**: Envía y recibe imágenes, audio y documentos
- **Interfaz web**: Gestiona todo a través de un panel en el navegador
- **Integración móvil**: Empareja dispositivos iOS o Android para control remoto, acceso a cámara y capturas de pantalla
