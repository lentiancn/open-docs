# Apifox

API 一体化协作平台。

## 简介

Apifox 是 API 一体化协作平台，集 API 文档、API 调试、API Mock、API 自动化测试于一体。

### 主要特性

- **API 文档** - 自动生成 API 文档
- **API 调试** - HTTP 请求模拟
- **Mock 服务** - 快速创建 Mock 服务器
- **自动化测试** - API 自动化测试
- **团队协作** - 项目管理和权限控制

### 统计数据

- 数百万开发者使用
