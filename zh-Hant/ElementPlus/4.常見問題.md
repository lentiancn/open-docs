# Element Plus 常見問題

常見問題解答。

## 安裝問題

### Q: 安裝失敗？

**解決方案**：
1. 檢查 Node.js 版本
```bash
node --version  # 需要 18.0+
```
2. 清理 npm 緩存
```bash
npm cache clean --force
```
3. 刪除 node_modules 重新安裝

### Q: TypeScript 類型錯誤？

**解決方案**：
1. 安裝類型定義
```bash
npm install -D @types/node
```
2. 在 tsconfig.json 中添加
```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

## 使用問題

### Q: 組件不顯示？

**解決方案**：
1. 確認已正確引入
2. 檢查控制台錯誤信息
3. 確認 Vue 版本（需要 3.2+）

### Q: 樣式不生效？

**解決方案**：
1. 確認已引入 CSS
2. 檢查 CSS 優先級，使用 `:deep()`

### Q: 圖標不顯示？

**解決方案**：
1. 安裝圖標包
```bash
npm install @element-plus/icons-vue
```
2. 全局註冊
```typescript
import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```

## 主題問題

### Q: 如何修改主題顏色？

**解決方案**：
使用 CSS 變量：
```css
:root {
  --el-color-primary: #409eff;
}
```

### Q: 如何使用暗色主題？

**解決方案**：
引入暗色 CSS：
```typescript
import 'element-plus/theme-chalk/dark/css-vars.css'
```
