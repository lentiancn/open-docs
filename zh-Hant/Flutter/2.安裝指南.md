# Flutter 安裝指南

## 環境要求

在安裝 Flutter 之前，需要確保開發環境滿足以下要求。作業系統方面，Flutter 支援 Windows、macOS 和 Linux。Windows 需要 Windows 10 或更高版本，macOS 需要 macOS 10.14 或更高版本，Linux 需要支援 GNOME 或 KDE 桌面環境。

磁碟空間方面，Flutter SDK 本身佔用約 1GB，但考慮到開發專案和編譯快取，建議預留至少 10GB 的可用空間。記憶體方面，8GB 或更多記憶體可以獲得更流暢的開發體驗。

對於行動開發，iOS 開發需要 macOS 和 Xcode，Android 開發需要安裝 Android Studio 和 Android SDK。Web 和桌面開發沒有特殊要求，只需安裝 Flutter SDK 即可。

## 安裝步驟

### Windows 安裝

在 Windows 上安裝 Flutter 有兩種主要方式。第一種是從 Flutter 官網下載預編譯的壓縮包，解壓到合適的位置，然後將 flutter/bin 目錄加入到系統 PATH 環境變數中。

下載方式：訪問 Flutter 官網 flutter.dev，下載最新的 stable 版本壓縮包。解壓到 C:\src\flutter 或其他合適的位置。開啟「系統內容」→「進階」→「環境變數」，在使用者變數的 Path 中加入 Flutter 的 bin 目錄路徑。

完成後，開啟命令提示字元，執行 flutter doctor 命令檢查安裝是否成功。Flutter doctor 會檢測 Flutter SDK 的狀態，並給出配置建議。

### macOS 安裝

在 macOS 上，推薦使用 Git 方式安裝。首先確保已安裝 Xcode，然後開啟終端，執行以下命令克隆 Flutter：

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

將 Flutter 加入 PATH：

```bash
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.bash_profile
```

或者使用 zsh 的使用者：

```bash
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.zshrc
```

然後執行 source ~/.bash_profile 或 source ~/.zshrc 重新整理環境變數。

### Linux 安裝

Linux 上的安裝與 macOS 類似。首先安裝依賴：

```bash
sudo apt-get update
sudo apt-get install curl git unzip xz-utils zip libglu1-mesa clang cmake ninja-build pkg-config libgtk-3-dev
```

然後克隆 Flutter 儲存庫並配置 PATH，與 macOS 步驟相同。

## 驗證安裝

安裝完成後，執行以下命令驗證：

```bash
flutter --version
flutter doctor
```

flutter doctor 會顯示安裝狀態和任何缺失的依賴。按照提示完成必要的配置，如安裝 Android Studio、配置 Android SDK 等。

## IDE 配置

### VS Code

VS Code 是 Flutter 推薦的編輯器之一。安裝 VS Code 後，開啟擴充功能市場，搜尋 Flutter 和 Dart 擴充功能並安裝。安裝完成後，重新啟動 VS Code，即可使用 Flutter 相關功能。

### Android Studio

安裝 Android Studio 後，開啟 Plugins 頁面，搜尋 Flutter 外掛並安裝。安裝完成後，配置 Flutter SDK 路徑：開啟 Preferences → Language & Frameworks → Flutter，設定 Flutter SDK 路徑。

## 快速開始

建立第一個 Flutter 專案：

```bash
flutter create my_first_app
cd my_first_app
flutter run
```

或者在 VS Code 中使用命令面板（Ctrl+Shift+P），輸入 Flutter: New Project 來建立新專案。

執行 flutter devices 可以查看已連接的裝置，flutter run -d device_id 可以指定裝置執行。
