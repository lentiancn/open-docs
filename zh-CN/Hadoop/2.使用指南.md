# Hadoop 使用指南

Hadoop 基本操作指南。

---

## HDFS 操作

### 文件操作

```bash
# 列出文件
hdfs dfs -ls /

# 创建目录
hdfs dfs -mkdir -p /user/hadoop

# 上传文件
hdfs dfs -put localfile.txt /user/hadoop/

# 下载文件
hdfs dfs -get /user/hadoop/file.txt ./

# 查看文件
hdfs dfs -cat /user/hadoop/file.txt

# 删除文件
hdfs dfs -rm /user/hadoop/file.txt
```

---

## YARN 操作

### 应用管理

```bash
# 列出应用
yarn application -list

# 杀死应用
yarn application -kill application_id
```

---

## MapReduce

### 运行示例任务

```bash
# WordCount 示例
hadoop jar $HADOOP_HOME/share/hadoop/mapreduce/hadoop-mapreduce-examples-3.4.3.jar wordcount /input /output
```
