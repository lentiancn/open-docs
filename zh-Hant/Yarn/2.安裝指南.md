# Yarn 安裝指南

## 系統要求

- Node.js：16.0+（推薦 18+）
- 操作系統：macOS、Linux、Windows 10+
- 磁盤空間：至少 200MB

---

## 安裝方法

### 方法一：使用 corepack（推薦）

```bash
corepack enable
corepack prepare yarn@stable --activate
```

### 方法二：使用 npm

```bash
npm install -g yarn
```

### 方法三：腳本安裝（macOS/Linux）

```bash
curl -o- -L https://yarnpkg.com/install.sh | bash
```

### 方法四：PowerShell（Windows）

```powershell
iwr -useb https://yarnpkg.com/install.ps1 | iex
```

---

## 驗證安裝

```bash
yarn --version
```

---

## 升級

```bash
corepack prepare yarn@stable --activate
```
