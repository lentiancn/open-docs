# Claude Code 使用手册

基本使用方法和技巧。

## 基本使用

### 启动

- VS Code：使用快捷键 Cmd/Ctrl + Shift + P
- CLI：直接在终端运行 `claude`

### 对话

直接在聊天窗口描述需求：
```
帮我写一个排序函数
解释这段代码的含义
这个 bug 怎么修复
```

## 代码生成

### 生成新代码

描述你想要的功能：
```
创建一个用户登录 API
实现分页查询功能
编写单元测试
```

### 修改现有代码

1. 选中代码
2. 描述修改需求
3. Claude 会生成修改后的代码

## 调试辅助

### 解释错误

直接粘贴错误信息：
```
这个报错是什么意思？
```

### 修复建议

让 Claude 分析问题：
```
帮我看看这个函数有什么问题
```

## 代码审查

### 代码审查请求

```
审查这段代码
有什么安全问题
性能优化建议
```

## 快捷键

### VS Code

| 快捷键 | 功能 |
|--------|------|
| Cmd/Ctrl + L | 打开对话 |
| Cmd/Ctrl + Enter | 提交输入 |

## 配置选项

### claude.json

在项目根目录创建 `claude.json`：

```json
{
  "preamble": "你是一个 Python 专家",
  "maxTokens": 4000
}
```

## 最佳实践

1. 清晰描述需求
2. 提供上下文信息
3. 迭代改进代码
4. 结合人工审核
