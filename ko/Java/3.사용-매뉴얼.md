# Java 사용 매뉴얼

## 첫 번째 Java 프로그램

### Hello World 만들기

HelloWorld.java라는 이름의 새 텍스트 파일을 만드세요. 파일 이름이 클래스 이름과 정확히 일치해야 합니다.

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

### 프로그램 컴파일

터미널 또는 명령 프롬프트를 열고 파일이 있는 디렉터리로 이동하여 컴파일 명령을 실행하세요:

```bash
javac HelloWorld.java
```

컴파일 성공 시 HelloWorld.class 파일이 생성됩니다.

### 프로그램 실행

```bash
java HelloWorld
```

출력:

```
Hello, World!
```

## Java 기본 문법

### 변수와 데이터 유형

Java는 정적 유형 언어입니다. 변수는 사용 전에 선언해야 합니다.

```java
// 기본 데이터 유형
int age = 25;
double salary = 5000.50;
boolean isActive = true;
char grade = 'A';
String name = "홍길동";

// 참조 유형
Integer count = 10;
Double price = 99.99;
```

### 연산자

```java
// 산술 연산자
int a = 10, b = 3;
int sum = a + b;      // 더하기
int diff = a - b;     // 빼기
int prod = a * b;     // 곱하기
int quot = a / b;     // 나누기
int rem = a % b;      // 나머지

// 비교 연산자
boolean result = (a > b);

// 논리 연산자
boolean flag = (a > 5) && (b < 10);
```

### 조건문

```java
// if-else 문
if (score >= 90) {
    System.out.println("우수");
} else if (score >= 60) {
    System.out.println("통과");
} else {
    System.out.println("불통과");
}

// switch 문
switch (day) {
    case 1:
        System.out.println("월요일");
        break;
    case 2:
        System.out.println("화요일");
        break;
    default:
        System.out.println("알 수 없음");
}
```

### 반복문

```java
// for 문
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}

// while 문
int i = 0;
while (i < 10) {
    System.out.println(i);
    i++;
}

// do-while 문
do {
    System.out.println(i);
    i++;
} while (i < 10);

// for-each로 배열 반복
int[] numbers = {1, 2, 3, 4, 5};
for (int num : numbers) {
    System.out.println(num);
}
```

## 객체 지향 프로그래밍

### 클래스와 객체

```java
// 클래스 정의
public class Person {
    // 필드
    private String name;
    private int age;
    
    // 생성자
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // Getter와 Setter
    public String getName() {
        return name;
    }
    
    public void setName(String name) {
        this.name = name;
    }
    
    // 메서드
    public void sayHello() {
        System.out.println("안녕하세요, 저는 " + name + "입니다");
    }
}

// 객체 생성
Person person = new Person("홍길동", 25);
person.sayHello();
```

## Stream API

```java
import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);

// 필터링
List<Integer> evenNumbers = numbers.stream()
    .filter(n -> n % 2 == 0)
    .collect(Collectors.toList());

// 매핑
List<Integer> squared = numbers.stream()
    .map(n -> n * n)
    .collect(Collectors.toList());

// 합계
int sum = numbers.stream()
    .mapToInt(Integer::intValue)
    .sum();
```

## 네트워크 프로그래밍

### TCP 통신

```java
import java.net.*;
import java.io.*;

ServerSocket serverSocket = new ServerSocket(8080);
Socket clientSocket = serverSocket.accept();
BufferedReader in = new BufferedReader(
    new InputStreamReader(clientSocket.getInputStream()));
PrintWriter out = new PrintWriter(
    clientSocket.getOutputStream(), true);

String message = in.readLine();
out.println("서버 수신: " + message);
```

### HTTP 요청

```java
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

HttpClient client = HttpClient.newHttpClient();
HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create("https://api.example.com/data"))
    .GET()
    .build();

HttpResponse<String> response = client.send(request, 
    HttpResponse.BodyHandlers.ofString());
System.out.println(response.body());
```

## 데이터베이스 작업

### JDBC 연결

```java
import java.sql.*;

String url = "jdbc:mysql://localhost:3306/mydb";
String user = "root";
String password = "password";

try (Connection conn = DriverManager.getConnection(url, user, password)) {
    String sql = "SELECT * FROM users WHERE id = ?";
    PreparedStatement stmt = conn.prepareStatement(sql);
    stmt.setInt(1, 1);
    
    ResultSet rs = stmt.executeQuery();
    while (rs.next()) {
        System.out.println(rs.getString("name"));
    }
} catch (SQLException e) {
    e.printStackTrace();
}
```
