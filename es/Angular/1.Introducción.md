# Introducción

Angular es un framework de código abierto desarrollado y mantenido por Google, diseñado para crear aplicaciones web modernas. Es el sucesor de AngularJS (Angular 1.x), fue reescrito completamente y utiliza TypeScript como lenguaje principal de desarrollo.

Angular ofrece una solución completa para el desarrollo frontend, incluyendo arquitectura basada en componentes, enlace de datos bidireccional, inyección de dependencias, enrutamiento, manejo de formularios, cliente HTTP y más. Gracias a estas características, los desarrolladores pueden crear rápidamente aplicaciones de página única (SPA) de alto rendimiento y fáciles de mantener.

## Conceptos Fundamentales

### Componente

Los componentes son los bloques básicos de construcción de una aplicación Angular. Cada componente consta de tres partes:

- **Plantilla (Template)**: Define la estructura de la vista
- **Estilos (Styles)**: Define la apariencia de la vista
- **Lógica (Logic)**: Define el comportamiento del componente

### Módulo

Los módulos se utilizan para organizar el código, agrupando componentes, servicios, directivas y otros elementos relacionados. Toda aplicación Angular tiene al menos un módulo raíz (AppModule).

### Servicio

Los servicios encapsulan lógica de negocio reutilizable y se proporcionan a los componentes mediante el mecanismo de inyección de dependencias.

### Directiva

Las directivas extienden la funcionalidad de los elementos HTML. Se dividen en directivas estructurales (como *ngIf, *ngFor) y directivas de atributo (como ngClass, ngStyle).

## Características Principales

### Arquitectura de Componentes

Angular utiliza un diseño basado en componentes, donde cada uno es una unidad funcional independiente que puede reutilizarse y combinarse con otros.

### TypeScript

Angular usa TypeScript como lenguaje principal de desarrollo, proporcionando soporte de tipos fuertes, programación orientada a objetos y características modernas de JavaScript.

### Inyección de Dependencias

Angular cuenta con un sistema integrado de inyección de dependencias que hace que el código sea más modular, testeable y mantenible.

### Enlace de Datos Bidireccional

Los datos entre la plantilla y el componente se sincronizan automáticamente, reduciendo la necesidad de manipular el DOM manualmente.

### Enrutamiento

Un sistema de enrutamiento potente que soporta carga diferida (lazy loading), guardianes, paso de parámetros y más funciones.

### Cliente HTTP

Cliente HTTP integrado con soporte para interceptores, transformaciones de solicitud/respuesta y otras funciones avanzadas.

### Manejo de Formularios

Proporciona dos enfoques: formularios reactivos y formularios basados en plantillas, para manejar validaciones complejas.

### Programación Reactiva

Soporte completo para RxJS, proporcionando potentes capacidades de programación asíncrona.

## Casos de Uso

- Aplicaciones web empresariales
- Aplicaciones de página única (SPA)
- Aplicaciones web progresivas (PWA)
- Aplicaciones móviles híbridas (combinado con Ionic)
- Aplicaciones de escritorio (combinado con Electron)

## Historial de Versiones

- **AngularJS (1.x)**: Lanzado en 2010
- **Angular 2**: Lanzado en 2016, reescrito completamente
- **Angular 17**: Última versión estable, lanzada en 2023

Angular cuenta con una comunidad activa y documentación completa, siendo una de las opciones más populares para crear aplicaciones web modernas.
