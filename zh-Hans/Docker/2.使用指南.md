# Docker 使用指南

## 常用命令

```bash
# 镜像
docker pull <image>    # 拉取镜像
docker images          # 查看镜像

# 容器
docker run -d <image> # 后台运行
docker ps              # 运行中容器
docker ps -a           # 所有容器
docker stop <id>       # 停止容器
docker rm <id>         # 删除容器
docker logs -f <id>    # 查看日志

# Docker Compose
docker-compose up -d   # 启动
docker-compose down    # 停止
```

## 示例

```bash
docker run -d -p 80:80 nginx
```
