## Creando una Aplicación Vue (Recomendado)

### Requisitos Previos
- Familiaridad con la línea de comandos
- Instalar Node.js versión ^20.19.0 || >=22.12.0

### Usando la Herramienta de Scaffolding Oficial
Ejecuta uno de los siguientes comandos en tu terminal:

```bash
# npm
npm create vue@latest

# pnpm  
pnpm create vue@latest

# yarn (v1+)
yarn create vue

# yarn (Modern v2+)
yarn create vue@latest

# yarn (^v4.11)
yarn dlx create-vue@latest

# bun
bun create vue@latest
```

### Opciones de Configuración del Proyecto
Se te preguntará por varias características opcionales:
- Project name: Nombre de tu proyecto
- Add TypeScript?: Si deseas agregar soporte para TypeScript
- Add JSX Support?: Si deseas agregar soporte para JSX
- Add Vue Router: Para desarrollo de Aplicaciones de Página Única
- Add Pinia: Para gestión de estado
- Add Vitest: Para pruebas unitarias
- Add End-to-End Testing: Cypress/Nightwatch/Playwright
- Add ESLint: Para calidad de código
- Add Prettier: Para formateo de código
- Add Vue DevTools 7: Para depuración (experimental)

### Iniciando el Servidor de Desarrollo
```bash
cd <your-project-name>
npm install
npm run dev
```

### Compilación para Producción
```bash
npm run build
```

## Usando Vue desde CDN

### Construcción Global
```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<div id="app">{{ message }}</div>

<script>
const { createApp } = Vue

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

### Construcción de Módulos ES
```html
<div id="app">{{ message }}</div>

<script type="module">
import { createApp } from 'https://unpkg.com/vue@3/dist/vue.esm-browser.js'

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

### Habilitando Mapas de Importación
```html
<script type="importmap">
{
  "imports": {
    "vue": "https://unpkg.com/vue@3/dist/vue.esm-browser.js"
  }
}
</script>

<div id="app">{{ message }}</div>

<script type="module">
import { createApp } from 'vue'

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

## Recomendaciones de Entorno de Desarrollo

- **IDE**: Visual Studio Code + [extensión oficial de Vue](https://marketplace.visualstudio.com/items?itemName=Vue.volar)
- **Herramienta de Compilación**: Vite (incluida en el scaffolding oficial)
- **Servidor Local**: Para módulos ES, usa `npx serve` para iniciar un servidor HTTP local
- **Alternativa para Casos Simples**: Considera [petite-vue](https://github.com/vuejs/petite-vue) para escenarios ligeros
