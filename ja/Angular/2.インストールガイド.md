# Angular インストールガイド

## 環境要件

- Node.js：18.10.0+ または 20.0.0+
- npm：9.0.0+

## インストール

### 1. Node.jsをインストール

```bash
nvm install --lts
nvm use --lts
```

### 2. Angular CLIをインストール

```bash
npm install -g @angular/cli
```

### 3. 新規プロジェクト作成

```bash
ng new my-app
cd my-app
ng serve
```

## 主なコマンド

```bash
# コンポーネント作成
ng g c my-component

# サービス作成
ng g s my-service

# 本番ビルド
ng build
```

## よくある問題

### npmが遅い？

```bash
npm config set registry https://registry.npmmirror.com
```
