# Bootstrap 개요

세계에서 가장 인기 있는 프론트엔드 CSS 프레임워크.

## Bootstrap이란?

Bootstrap은 반응형 모바일 우선 웹사이트를 구축하기 위해 Twitter가 개발한 가장 인기 있는 CSS 프레임워크입니다.

### 주요 기능

- **반응형 그리드 시스템** - 12열 반응형 그리드
- **프리빌드 컴포넌트** - 네비게이션 바, 카드, 모달 등
- **CSS 유틸리티 클래스** - 빠른 스타일 조정
- **JavaScript 컴포넌트** - 인터랙티브 컴포넌트

### 통계

- GitHub 별표: 160,000+
- 웹사이트 사용: 수백만
