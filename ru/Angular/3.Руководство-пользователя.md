# Руководство пользователя

Это руководство подробно описывает основные возможности и способы использования Angular: компоненты, сервисы, маршрутизацию, директивы, каналы, формы и другие.

## Разработка компонентов

### Создание компонента

```bash
ng generate component user-list
```

Или вручную:

```typescript
import { Component } from '@angular/core';

@Component({
  selector: 'app-user-list',
  templateUrl: './user-list.component.html',
  styleUrls: ['./user-list.component.scss']
})
export class UserListComponent {
  users = ['Алексей', 'Мария', 'Иван'];
}
```

### Синтаксис шаблонов

**Интерполяция:**


**Привязка свойств:**


## Сервисы и внедрение зависимостей

### Создание сервиса

```bash
ng generate service user
```

Код сервиса:

```typescript
import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class UserService {
  constructor(private http: HttpClient) {}

  getUsers(): Observable<User[]> {
    return this.http.get<User[]>('/api/users');
  }
}
```

### Использование сервиса

```typescript
import { Component, OnInit } from '@angular/core';
import { UserService } from './user.service';

@Component({
  selector: 'app-user-list',
  templateUrl: './user-list.component.html'
})
export class UserListComponent implements OnInit {
  users: User[] = [];

  constructor(private userService: UserService) {}

  ngOnInit() {
    this.userService.getUsers().subscribe(data => {
      this.users = data;
    });
  }
}
```

### Создание собственного канала

```bash
ng generate pipe capitalize
```

```typescript
import { Pipe, PipeTransform } from '@angular/core';

@Pipe({
  name: 'capitalize'
})
export class CapitalizePipe implements PipeTransform {
  transform(value: string): string {
    if (!value) return value;
    return value.charAt(0).toUpperCase() + value.slice(1);
  }
}
```

## Обработка форм

### Реактивные формы

```typescript
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

export class LoginComponent implements OnInit {
  loginForm: FormGroup;

  constructor(private fb: FormBuilder) {}

  ngOnInit() {
    this.loginForm = this.fb.group({
      username: ['', Validators.required],
      password: ['', [Validators.required, Validators.minLength(6)]],
      remember: [false]
    });
  }

  onSubmit() {
    if (this.loginForm.valid) {
      console.log(this.loginForm.value);
    }
  }
}
```

Шаблон:

```html
<form [formGroup]="loginForm" (ngSubmit)="onSubmit()">
  <input formControlName="username" placeholder="Имя пользователя">
  <div *ngIf="loginForm.get('username').touched && loginForm.get('username').errors?.required">
    Имя пользователя обязательно
  </div>
  
  <input type="password" formControlName="password" placeholder="Пароль">
  
  <label>
    <input type="checkbox" formControlName="remember"> Запомнить меня
  </label>
  
  <button type="submit" [disabled]="loginForm.invalid">Войти</button>
</form>
```

### Формы на основе шаблона

```html
<form #form="ngForm" (ngSubmit)="onSubmit(form)">
  <input name="username" [(ngModel)]="username" required #name="ngModel" placeholder="Имя пользователя">
  <div *ngIf="name.touched && name.errors?.required">Имя пользователя обязательно</div>
  
  <button type="submit" [disabled]="form.invalid">Отправить</button>
</form>
```

## HTTP-клиент

### Настройка

В app.module.ts добавьте импорт:

```typescript
import { HttpClientModule } from '@angular/common/http';

@NgModule({
  imports: [HttpClientModule]
})
export class AppModule {}
```

### Отправка запросов

```typescript
import { HttpClient } from '@angular/common/http';

constructor(private http: HttpClient) {}

getData() {
  this.http.get('/api/data').subscribe(data => {
    console.log(data);
  });
}

postData() {
  this.http.post('/api/data', { name: 'test' }).subscribe();
}
```

### Перехватчики

```typescript
import { HttpInterceptorFn } from '@angular/common/http';

export const authInterceptor: HttpInterceptorFn = (req, next) => {
  const token = localStorage.getItem('token');
  const cloned = req.clone({
    setHeaders: { Authorization: `Bearer ${token}` }
  });
  return next(cloned);
};
```

## Лучшие практики

1. **Используйте строгий режим TypeScript** — включите опцию strict при компиляции
2. **Компоненты должны быть маленькими и сфокусированными** — каждый компонент делает одно дело
3. **Применяйте стратегию OnPush** — это улучшает производительность проверки изменений
4. **Используйте каналы разумно** — избегайте сложных вычислений в каналах
5. **Применяйте ленивую загрузку** — это уменьшает время первоначальной загрузки
6. **Пишите тесты** — это обеспечивает качество кода
