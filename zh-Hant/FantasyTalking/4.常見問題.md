# FantasyTalking 常見問題

常見問題解答。

## 安裝問題

### Q: 安裝失敗？

**解決方案：**
1. 升級 pip
2. 使用鏡像：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q: CUDA 不可用？

**解決方案：**
安裝匹配的 PyTorch 版本。

### Q: 記憶體不足？

**解決方案：**
使用更小的輸入圖片。

## 使用問題

### Q: 生成失敗，找不到人臉？

**解決方案：**
使用包含清晰人臉的圖片。

### Q: 影片生成太慢？

**解決方案：**
使用 GPU 加速。

### Q: 口型不同步？

**解決方案：**
確保音訊清晰。
