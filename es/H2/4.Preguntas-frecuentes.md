# Preguntas frecuentes sobre H2

## Instalación

### ¿Cómo descargar H2?
Visita https://www.h2database.com/html/download.html

### ¿Qué versión de Java requiere H2?
Java 11 o superior.

### ¿Cómo usar H2 en Maven?
Añade la dependencia en pom.xml:
```xml
<dependency>
    <groupId>com.h2database</groupId>
    <artifactId>h2</artifactId>
    <version>2.2.224</version>
</dependency>
```

## Conexión

### ¿Cuál es el usuario y contraseña por defecto?
Usuario: sa, contraseña: (vacía)

### ¿Cómo conectar a H2?
```java
Class.forName("org.h2.Driver");
Connection conn = DriverManager.getConnection("jdbc:h2:mem:testdb", "sa", "");
```

## Datos

### ¿Los datos en memoria se pierden?
Sí, los datos en memoria (jdbc:h2:mem:) se pierden cuando todas las conexiones se cierran.

### ¿Cómo cifrar la base de datos?
```java
jdbc:h2:file:./encrypted;CIPHER=AES
```

## Rendimiento

### ¿Cómo mejorar el rendimiento?
Usa operaciones por lotes y desactiva auto-commit:
```java
conn.setAutoCommit(false);
// ... operaciones por lotes ...
conn.commit();
```

## Seguridad

### ¿Cómo prevenir inyección SQL?
Usa PreparedStatement en lugar de concatenación de cadenas:
```java
// Correcto
PreparedStatement pstmt = conn.prepareStatement("SELECT * FROM users WHERE name = ?");
pstmt.setString(1, name);
```

## Migración

### ¿Cómo migrar desde MySQL a H2?
1. Configura el modo compatible: jdbc:h2:file:db;MODE=MySQL
2. Exporta los datos de MySQL como script SQL
3. Ejecuta el script en H2

## Herramientas

### ¿Cómo usar la consola H2?
Ejecuta: java -jar h2.jar -web
Luego visita: http://localhost:8082

### ¿Cómo verificar la versión de H2?
```bash
java -jar h2.jar -version
```
O en SQL: SELECT H2VERSION();
