# Spring Boot Guía de Instalación

Esta guía te lleva a través de la instalación y configuración de Spring Boot.

## Requisitos Previos

- Java Development Kit (JDK) 8 o superior
- Maven 3.3+ o Gradle 5.x+
- Tu IDE favorito (IntelliJ IDEA, Eclipse, VS Code)

## Instalación en Windows

### 1. Instalar JDK

Descarga e instala JDK 17 (LTS) desde:
- Oracle JDK: https://www.oracle.com/java/technologies/downloads/
- Open JDK: https://adoptium.net/

### 2. Configurar Variables de Entorno

```powershell
# Agregar JAVA_HOME
[System.Environment]::SetEnvironmentVariable("JAVA_HOME", "C:\Program Files\Java\jdk-17", "User")

# Agregar al PATH
$env:PATH = "$env:PATH;$env:JAVA_HOME\bin"
```

### 3. Instalar Maven

```powershell
# Descargar Maven
# https://maven.apache.org/download.cgi

# Extraer y configurar
[System.Environment]::SetEnvironmentVariable("MAVEN_HOME", "C:\apache-maven", "User")
$env:PATH = "$env:PATH;$env:MAVEN_HOME\bin"
```

## Instalación en macOS

### Usando Homebrew

```bash
# Instalar JDK
brew install openjdk@17

# Instalar Maven
brew install maven
```

## Crear un Proyecto Spring Boot

### Usando Spring Initializr

1. Ve a https://start.spring.io
2. Selecciona:
   - Project: Maven Project
   - Language: Java
   - Spring Boot: 3.2.x
   - Group: com.example
   - Artifact: demo
3. Agrega dependencias (Web, DevTools, etc.)
4. Click en "Generate"
5. Extrae el archivo zip

### Usando IntelliJ

1. File → New → Project
2. Selecciona "Spring Initializr"
3. Configura el proyecto
4. Agrega dependencias
5. Click en Next → Finish

### Usando Línea de Comandos

```bash
# Usando curl
curl https://start.spring.io/starter.zip \
    -d name=demo \
    -d dependencies=web \
    -d javaVersion=17 \
    -o demo.zip

# Extraer
unzip demo.zip
cd demo
```

## Ejecutar la Aplicación

### Usando Maven

```bash
# En Windows
.\mvnw.cmd spring-boot:run

# En macOS/Linux
./mvnw spring-boot:run
```

### Usando Gradle

```bash
./gradlew bootRun
```

### Desde tu IDE

1. Abre el proyecto en tu IDE
2. Ejecuta `DemoApplication.main()`

## Configuración Básica

### application.properties

```properties
# Puerto del servidor
server.port=8080

# Nombre de la aplicación
spring.application.name=demo

# Configuración de logging
logging.level.com.example.demo=DEBUG
```

### application.yml

```yaml
server:
  port: 8080

spring:
  application:
    name: demo

logging:
  level:
    com.example.demo: DEBUG
```

## Dependencias Comunes

### Web
```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-web</artifactId>
</dependency>
```

### Data JPA
```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-data-jpa</artifactId>
</dependency>
```

### Security
```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-security</artifactId>
</dependency>
```

## Construcción para Producción

### Crear JAR ejecutable

```bash
# Maven
./mvnw clean package

# Gradle
./gradlew clean build
```

### Ejecutar JAR

```bash
java -jar target/demo-0.0.1-SNAPSHOT.jar
```
