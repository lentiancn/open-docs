# Flutter

Flutter - Кроссплатформенный фреймворк пользовательского интерфейса

## Обзор

Flutter — это набор инструментов для разработки программного обеспечения с открытым исходным кодом, созданный Google, используемый для создания красивых нативно скомпилированных мобильных, веб- и настольных приложений из единой кодовой базы.

## Функции

- **Кроссплатформенность**: Единая кодовая база для iOS, Android, Web, Windows, macOS, Linux
- **Горячая перезагрузка**: Быстрая итерация разработки, сохраняет состояние приложения
- **Богатые компоненты**: Библиотеки компонентов Material Design и Cupertino
- **Нативная производительность**: Компилируется в нативный код ARM
- **Декларативный UI**: Современная парадигма разработки пользовательского интерфейса
- **Открытый код**: Лицензия BSD

## Быстрый старт

### Установка

```bash
# Клонировать Flutter
git clone https://github.com/flutter/flutter.git -b stable

# Добавить в PATH
export PATH="$PATH:$HOME/flutter/bin"

# Проверить установку
flutter doctor
```

### Создать проект

```bash
flutter create my_app
cd my_app
flutter run
```
