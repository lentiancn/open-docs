# APISIX Installationsanleitung

Verschiedene Installationsmethoden für APISIX.

## Umgebungsanforderungen

### Hardware-Anforderungen

- CPU: 2 Kerne oder mehr
- Speicher: 4GB oder mehr
- Festplatte: 40GB oder mehr

### Software-Anforderungen

- Betriebssystem: Linux (Ubuntu 20.04+, CentOS 7+ empfohlen)
- Nginx oder OpenResty
- Lua 5.1+
- etcd 3.4+

## Docker-Installation (Empfohlen)

### Schnellstart

```bash
# etcd starten
docker run -d --name etcd \
  -p 2379:2379 \
  apache/apisix:latest etcd

# APISIX starten
docker run -d --name apisix \
  -p 9080:9080 \
  -p 9090:9090 \
  apache/apisix:latest
```

### docker-compose

```yaml
version: '3'
services:
  etcd:
    image: apache/apisix:3.5.0-etcd
    ports:
      - "2379:2379"
  apisix:
    image: apache/apisix:3.5.0
    ports:
      - "9080:9080"
    depends_on:
      - etcd
```

```bash
docker-compose up -d
```

## Helm-Installation (Kubernetes)

```bash
helm repo add apisix https://apache.github.io/apisix-helm-chart
helm install apisix apisix/apisix
```

## Quellcode-Installation

### Abhängigkeiten installieren

**Ubuntu/Debian:**

```bash
wget -qO - https://openresty.org/package/pubkey.gpg | sudo apt-key add -
sudo apt-get update
sudo apt-get install -y openresty etcd
```

### Quellcode klonen

```bash
git clone https://github.com/apache/apisix.git
cd apisix
git checkout 3.5.0
```

### Abhängigkeiten installieren

```bash
make deps
make init
```

### Dienst starten

```bash
make run
```

## Konfiguration

### Hauptkonfigurationsdatei

```yaml
apisix:
  node_listen: 9080
  enable_admin: true
  admin_key:
    - name: admin
      key: 123456

etcd:
  host:
    - "http://127.0.0.1:2379"
```

## Häufige Probleme

### Port bereits belegt

```bash
netstat -tlnp | grep 9080
```

### etcd-Verbindungsfehler

```bash
etcdctl endpoint health
```

## Deinstallation

```bash
docker stop apisix etcd
docker rm apisix etcd
```
