# Guía de uso de Flutter

Esta guía explica cómo usar Flutter para crear aplicaciones multiplataforma.

## Inicio rápido

### Crear proyecto

```bash
# Crear nuevo proyecto
flutter create my_app

# Entrar al directorio del proyecto
cd my_app

# Ejecutar la aplicación
flutter run
```

### Estructura del proyecto

```
my_app/
├── lib/
│   ├── main.dart
│   └── ...
├── test/
├── android/
├── ios/
├── web/
├── linux/
├── windows/
├── macos/
├── pubspec.yaml
└── README.md
```

## Widgets básicos

### Texto

```dart
import 'package:flutter/material.dart';

Text(
  'Hello Flutter',
  style: TextStyle(
    fontSize: 24,
    color: Colors.blue,

// Botón elevado

TextButton(
  onPressed: () {},

```

### Campo de texto

```dart
TextField(
  decoration: InputDecoration(
    hintText: 'Introduce el contenido',

  },

```

## Widgets de diseño

### Columna y fila

```dart
// Columna (vertical)
Column(
  mainAxisAlignment: MainAxisAlignment.center,
  children: [
    Text('Primera línea'),
    Text('Segunda línea'),
  ],
)

// Fila (horizontal)
Row(
  mainAxisAlignment: MainAxisAlignment.spaceEvenly,
  children: [
    Icon(Icons.home),
    Icon(Icons.search),
    Icon(Icons.settings),
  ],
)
```

### Lista

```dart
// Lista vertical
ListView.builder(
  itemCount: items.length,
  itemBuilder: (context, index) {
    return ListTile(
      title: Text(items[index].title),
    );
  },
)

// Lista horizontal
ListView.builder(
  scrollDirection: Axis.horizontal,
  itemCount: items.length,
  itemBuilder: (context, index) {
    return Container(
      width: 150,
      child: Card(
        child: Text(items[index].name),
      ),
    );
  },

```

### Cuadrícula

## Gestión de estado

### setState

```dart
class MyWidget extends StatefulWidget {
  @override
  _MyWidgetState createState() => _MyWidgetState();
}

class _MyWidgetState extends State<MyWidget> {
  int _counter = 0;

  void _increment() {
    setState(() {
      _counter++;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        Text('Contador: $_counter'),
        ElevatedButton(
          onPressed: _increment,
          child: Text('Incrementar'),
        ),
      ],
    );
  }
}
```

### Provider

```bash
# Añadir dependencia
flutter pub add provider
```

```dart
// Crear Provider
class CounterModel extends ChangeNotifier {
  int _count = 0;
  int get count => _count;

  void increment() {
    _count++;
    notifyListeners();
  }
}

// Usar Provider
Consumer<CounterModel>(
  builder: (context, counter, child) {
    return Text('${counter.count}');
  },
)
```

### GetX

```bash
# Añadir dependencia
flutter pub add get
```

```dart
// Usar GetX
Get.put(CounterController());

// Usar en Widget
Get.find<CounterController>().increment();
```

## Navegación de rutas

### Navegación básica

```dart
// Navegar a nueva página
Navigator.push(
  context,
  MaterialPageRoute(builder: (context) => NextPage()),
);

// Volver
Navigator.pop(context);
```

### Rutas nombradas

```dart
// Configurar rutas
MaterialApp(
  routes: {
    '/': (context) => HomePage(),
    '/about': (context) => AboutPage(),
  },
);

// Navegar
Navigator.pushNamed(context, '/about');
```

### Pasar parámetros

```dart
// Pasar parámetros
Navigator.pushNamed(
  context,
  '/detail',
  arguments: {'id': 123},
);

// Recibir parámetros
final args = ModalRoute.of(context).settings.arguments;
```

## Solicitudes de red

### HTTP

```bash
flutter pub add http
```

```dart
import 'package:http/http.dart' as http;

Future<void> fetchData() async {
  final response = await http.get(Uri.parse('https://api.example.com/data'));
  if (response.statusCode == 200) {
    print(response.body);
  }
}
```

### Dio

```bash
flutter pub add dio
```

```dart
import 'package:dio/dio.dart';

final dio = Dio();
final response = await dio.get('https://api.example.com/data');
```

## Almacenamiento local

### SharedPreferences

```bash
flutter pub add shared_preferences
```

```dart
import 'package:shared_preferences/shared_preferences.dart';

// Guardar datos
SharedPreferences prefs = await SharedPreferences.getInstance();
await prefs.setString('name', 'John');
await prefs.setInt('age', 30);

// Leer datos
String name = prefs.getString('name') ?? '';
int age = prefs.getInt('age') ?? 0;
```

### SQLite

```bash
flutter pub add sqflite
```

```dart
import 'package:sqflite/sqflite.dart';

final db = await openDatabase('my.db');
await db.insert('users', {'name': 'John', 'age': 30});
final users = await db.query('users');
```
