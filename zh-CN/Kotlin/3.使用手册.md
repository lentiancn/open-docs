# Kotlin 使用手册

## Hello World

```kotlin
fun main() {
    println("Hello, World!")
}
```

## 变量

```kotlin
val name = "Kotlin"  // 不可变
var age = 10           // 可变
```

## 函数

```kotlin
fun add(a: Int, b: Int): Int = a + b

fun greet(name: String) = println("Hello, $name")
```

## 类

```kotlin
class Person(val name: String, var age: Int)

data class User(val id: Int, val name: String)
```
