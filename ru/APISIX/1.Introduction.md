# APISIX 简介

云原生 API 网关。

## 什么是 APISIX？

Apache APISIX 是 Apache 软件基金会旗下的开源云原生 API 网关，由 API7.ai 开发并捐赠。它是一个具有动态、实时、高性能等特点的 API 网关，可作为所有业务的流量入口。

APISIX 提供了动态路由、动态上游、动态证书、A/B 测试、灰度发布（金丝雀发布）、蓝绿部署、限速、防攻击、收集指标、监控报警、可观测性、服务治理等功能。

### 核心特性

- **高性能**：基于 Nginx + Lua 实现，单核可达 18,000+ QPS，平均延迟仅 0.2ms
- **完全动态**：无需重启即可动态更新路由、插件、上游等配置
- **热加载**：支持插件热加载，修改插件无需重启
- **多协议支持**：HTTP、HTTPS、HTTP/2、Dubbo、gRPC、WebSocket、MQTT
- **插件生态丰富**：内置 80+ 插件，支持自定义插件开发
- **服务发现**：支持 Consul、Eureka、Nacos 等多种服务发现机制
- **负载均衡**：轮询、哈希、一致性哈希等多种策略
- **安全防护**：限流、认证、JWT、API 密钥、防火墙
- **可视化 Dashboard**：直观的管理界面
- **AI 网关**：支持 LLM 和 AI 工作负载
- **平台无关**：可运行在裸机、Kubernetes、Apple Silicon 等多种环境

### 技术架构

APISIX 采用 Nginx 作为底层的 HTTP 反向代理服务器，结合 Lua 脚本实现动态编排。配置存储使用 etcd，支持实时配置同步。

- **数据面**：处理实际请求转发，零依赖
- **控制面**：通过 Admin API 和 Dashboard 进行配置管理

## 适用场景

- **API 统一管理**：集中管理企业内部所有 API
- **微服务网关**：作为微服务入口，处理认证、限流、路由
- **服务网格入口**：对接 Istio、Linkerd 等服务网格
- **API 流量控制**：限流、熔断、降级
- **混合云架构**：统一管理多云、混合云 API
- **API 统计分析**：实时监控 API 调用情况
- **AI 网关**：LLM 和 AI 应用的流量入口
- **灰度发布**：金丝雀发布、A/B 测试
