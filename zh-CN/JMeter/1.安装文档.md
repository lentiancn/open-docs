# JMeter 安装指南

JMeter 是 Apache 组织开发的开源性能测试工具。本指南涵盖在各种环境中的安装方法。

## 版本信息

| 版本 | 状态 | 说明 |
|------|------|------|
| JMeter 5.x | ✅ 当前 | 最新版本 |

## 系统要求

### 硬件要求

| 要求 | 配置 |
|------|------|
| CPU | 2+ 核 |
| 内存 | 4+ GB |
| 磁盘 | 5+ GB |

### 软件要求

| 软件 | 版本 |
|------|------|
| Java | JDK 8 或更高 |

## 安装方法

### 方法一：Windows 安装

#### 1. 下载

1. 访问 https://jmeter.apache.org/download.cgi
2. 下载 Binaries 版本
3. 解压到指定目录

#### 2. 运行

```powershell
# 双击 bin\jmeter.bat
# 或命令行运行
jmeter.bat
```

### 方法二：macOS 安装

#### 1. 使用 Homebrew

```bash
brew install jmeter
```

#### 2. 手动安装

```bash
# 下载并解压
wget https://archive.apache.org/dist/jmeter/binaries/apache-jmeter-5.6.3.tgz
tar -xzf apache-jmeter-5.6.3.tgz

# 运行
./apache-jmeter-5.6.3/bin/jmeter
```

### 方法三：Linux 安装

```bash
# Ubuntu/Debian
sudo apt-get install jmeter

# 或下载安装
wget https://archive.apache.org/dist/jmeter/binaries/apache-jmeter-5.6.3.tgz
tar -xzf apache-jmeter-5.6.3.tgz
cd apache-jmeter-5.6.3/bin
./jmeter
```

### 方法四：Docker

```bash
# 运行 JMeter 容器
docker run -v $(pwd):/workspace justb4/jmeter -n -t test.jmx -l result.jtl
```

## 环境配置

### Java 配置

```bash
# 检查 Java
java -version

# 设置环境变量
export JAVA_HOME=/path/to/java
export PATH=$JAVA_HOME/bin:$PATH
```

## 常见问题

### 内存不足？

修改 jmeter.bat 或 jmeter：

```bash
# 设置堆大小
HEAP="-Xms1g -Xmx2g"
```

### GUI 启动慢？

使用命令行模式或增加内存。
