# Yarn インストールガイド

## システム要件

- Node.js：16.0.0以上（推奨18+）
- OS：macOS、Linux、Windows 10+

---

## インストール方法

### Corepackを使用（推奨）

```bash
corepack enable
corepack prepare yarn@stable --activate
```

### npmを使用

```bash
npm install -g yarn
```

### スクリプトを使用

```bash
curl -o- -L https://yarnpkg.com/install.sh | bash
```

---

## 確認

```bash
yarn --version
```
