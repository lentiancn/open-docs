# Nginx よくある質問

## インストール

### バージョンを確認
```bash
nginx -v
```

### 設定テスト
```bash
nginx -t
```

## 設定

### 再読み込み
```bash
nginx -s reload
```

## パフォーマンス

### 最適化
```nginx
worker_processes auto;
worker_connections 4096;
gzip on;
```

## セキュリティ

### HTTPS を有効化
```nginx
server {
    listen 443 ssl;
    ssl_certificate cert.crt;
    ssl_certificate_key cert.key;
}
```

### レート制限
```nginx
limit_req_zone $binary_remote_addr zone=one:10m rate=10r/s;
```

## エラー

- **400**: リクエストサイズを確認
- **403**: 権限を確認
- **404**: パスを確認
- **502**: バックエンドサービスが実行中か確認
- **504**: タイムアウト увеличьте proxy_read_timeout

## プロキシ

### 実際の IP を取得
```nginx
proxy_set_header X-Real-IP $remote_addr;
```
