# FantasyTalking ユーザーマニュアル

詳細な使用方法と例。

## 基本的な使い方

### 動画生成

```bash
python inference.py \
  --source_image face.jpg \
  --audio speech.wav \
  --output output.mp4
```

### パラメータ

| パラメータ | 説明 |
|------------|------|
| --source_image | 入力人物画像 |
| --audio | 入力音声ファイル |
| --output | 出力動画パス |
| --face_enhancer | 顔強調モデル |
| --fps | フレームレート |

## Python API

```python
from fantasytalking import FantasyTalking

ft = FantasyTalking()
ft.generate(
    source_image="face.jpg",
    audio="speech.wav",
    output="output.mp4"
)
```

## バッチ処理

```python
ft.batch_generate(image_list, audio_list, output_dir)
```

## Web API

```bash
python api.py --port 8080
```

## 入力要件

### 画像

- 形式：JPG、PNG
- 解像度：512x512以上
- 顔：正面、クリア、照明良好

### 音声

- 形式：WAV、MP3
- 長さ：1-60秒

## 出力設定

### 解像度

```bash
python inference.py --size 512
```

### フレームレート

```bash
python inference.py --fps 30
```

## ベストプラクティス

1. 清晰で正面の写真を使用
2. ノイズのない音声を準備
3. 顔強調を使用
