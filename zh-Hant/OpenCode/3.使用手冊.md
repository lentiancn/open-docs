# OpenCode 使用手冊

OpenCode 基本使用方法和技巧。

## 快速開始

### 提問

你可以讓 OpenCode 解釋代碼庫：

```
How is authentication handled in @packages/functions/src/api/index.ts
```

### 添加功能

讓 OpenCode 添加新功能前，建議先讓它創建計劃：

1. 按 **Tab** 切換到 Plan 模式
2. 描述你想要的功能
3. 查看計劃後可提供反饋或補充細節
4. 滿意後按 **Tab** 切換回 Build 模式執行

### 直接修改

對於更直接的修改，可以直接讓 OpenCode 執行：

```
We need to add authentication to the /settings route. Take a look at how this is handled in the /notes route and implement the same logic.
```

### 撤銷更改

如果對修改不滿意，可以使用撤銷命令：

```bash
/undo
```

OpenCode 會恢復更改並重新顯示你的原始消息，你可以在此基礎上調整提示。

### 重做

可以使用 `/redo` 命令重做更改。

## 分享對話

```bash
/share
```

這會創建一個指向當前對話的連結並複製到剪貼板。
