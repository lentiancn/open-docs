## 系统要求

### 硬件要求
- **处理器**：双核处理器或更好
- **内存**：最低 4GB（建议 8GB）
- **磁盘空间**：至少 3GB 可用空间

### 软件要求

#### Windows
- Windows 10 或更高版本
- Node.js 18.19 或更高版本（建议使用 LTS 版本）
- npm 9.x 或更高版本（随 Node.js 一起提供）
- Git 版本控制工具

#### macOS
- macOS 10.15（Catalina）或更高版本
- Node.js 18.19 或更高版本（建议使用 LTS 版本）
- npm 9.x 或更高版本
- Git（随 Xcode 命令行工具一起提供）

#### Linux
- Ubuntu 20.04 LTS 或更高版本 / Debian 系列发行版
- Fedora 36 或更高版本 / RHEL 系列发行版
- Node.js 18.19 或更高版本（建议使用 LTS 版本）
- npm 9.x 或更高版本
- Git

## 前置条件

### 1. 安装 Node.js

Node.js 是运行 Angular 所需的运行时环境。

#### Windows 安装

**方法 A：使用 Node.js 安装程序**
1. 访问 https://nodejs.org/
2. 下载 **LTS** 版本
3. 运行安装程序（.msi 文件）
4. 按照安装向导操作
5. 打开命令提示符验证：
   ```
   node --version
   npm --version
   ```

**方法 B：使用 Chocolatey**
```powershell
# 安装 Node.js
choco install nodejs-lts -y
```

**方法 C：使用 nvm-windows（推荐）**
```powershell
# 安装并使用 Node.js LTS
nvm install 18.19.0
nvm use 18.19.0
```

#### macOS 安装

**方法 A：使用 Homebrew（推荐）**
```bash
# 安装 Homebrew（如未安装）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装 Node.js LTS
brew install node@18
```

**方法 B：使用 nvm**
```bash
# 安装 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 安装 Node.js LTS
nvm install 18.19.0
nvm use 18.19.0
```

#### Linux 安装

**Ubuntu/Debian**
```bash
# 更新软件包列表
sudo apt update

# 安装 curl 和 build-essential
sudo apt install curl build-essential

# 添加 Node.js 仓库
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -

# 安装 Node.js
sudo apt install nodejs
```

**Fedora/RHEL**
```bash
# 添加 Node.js 仓库
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -

# 安装 Node.js
sudo dnf install nodejs
```

### 2. 安装 Git

Git 是 Angular CLI 版本控制功能必需的。

#### Windows
- 从 https://git-scm.com/ 下载
- 安装时选择 "Use Git from the Windows Command Prompt"

#### macOS
```bash
# 使用 Homebrew
brew install git

# 或安装 Xcode 命令行工具
xcode-select --install
```

#### Linux
```bash
# Ubuntu/Debian
sudo apt install git

# Fedora
sudo dnf install git
```

## 在 Windows 上安装

### 步骤指南

1. **安装 Node.js**
   - 从 https://nodejs.org/ 下载 Node.js LTS
   - 运行安装程序

2. **安装 Git**（如未安装）
   - 从 https://git-scm.com/ 下载

3. **安装 Angular CLI**
   ```bash
   npm install -g @angular/cli
   ```

4. **验证安装**
   ```bash
   ng version
   ```

5. **创建第一个项目**
   ```bash
   ng new my-first-app
   cd my-first-app
   ng serve
   ```

### 使用 Visual Studio Code（推荐）

1. 从 https://code.visualstudio.com/ 下载 VS Code
2. 安装 Angular 扩展：
   - Angular Language Service
   - Angular Snippets

## 在 Linux 上安装

### Ubuntu/Debian

```bash
# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装依赖
sudo apt install curl build-essential git

# 添加 Node.js 仓库
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -

# 安装 Node.js
sudo apt install nodejs

# 安装 Angular CLI
sudo npm install -g @angular/cli
```

### Fedora

```bash
# 安装依赖
sudo dnf install curl git

# 添加 Node.js 仓库
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -

# 安装 Node.js
sudo dnf install nodejs

# 安装 Angular CLI
sudo npm install -g @angular/cli
```

## 故障排除

### 常见问题及解决方案

#### 1. npm 权限错误 (EACCES)

**解决方案：**
```bash
# 使用 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
```

#### 2. Node.js 版本不匹配

**解决方案：**
```bash
# 使用 nvm 切换版本
nvm install 18.19.0
nvm use 18.19.0
```

#### 3. 找不到 Angular CLI

**解决方案：**
```bash
# 重新安装 Angular CLI
npm uninstall -g @angular/cli
npm install -g @angular/cli
```

#### 4. 端口 4200 被占用

**解决方案：**
```bash
# 使用不同端口
ng serve --port 4201
```

#### 5. Angular 版本兼容性

| Angular 版本 | Node.js 版本 | TypeScript 版本 |
|-------------|-------------|----------------|
| Angular 17  | 18.x, 20.x  | ~5.2.0         |
| Angular 16  | 16.x, 18.x  | ~5.0.0         |
| Angular 15  | 16.x, 18.x  | ~4.9.0         |
| Angular 14  | 14.x, 16.x  | ~4.7.0         |
