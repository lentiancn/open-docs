# Java

> 面向对象的编程语言

---

## 项目简介

Java 是由 Sun Microsystems 于 1995 年发布的面向对象编程语言，以其"一次编写，到处运行"的跨平台特性著称。广泛应用于企业级应用、Android 开发、大数据处理等领域。

## 快速开始

### Hello World

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

### 编译运行

```bash
# 编译
javac HelloWorld.java

# 运行
java HelloWorld
```
