# Oracle Руководство по установке

## Предварительные требования

- **Операционная система**: Oracle Linux, RHEL, Windows Server
- **Память**: Минимум 2GB RAM (рекомендуется 8GB+)
- **Место на диске**: Минимум 8GB для ПО

---

## Установка на Linux

### 1. Создание пользователя oracle

```bash
groupadd -g 54321 oinstall
groupadd -g 54322 dba
useradd -u 54321 -g oinstall -G dba oracle
passwd oracle
```

### 2. Настройка параметров ядра

```bash
kernel.shmmax = 4294967296
kernel.shmall = 2097152
```

### 3. Запуск установщика

```bash
./runInstaller
```

---

## После установки

```sql
sqlplus / as sysdba
SELECT name, open_mode FROM v$database;
```
