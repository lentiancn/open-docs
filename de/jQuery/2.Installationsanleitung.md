# jQuery Installationsanleitung

## jQuery erhalten

### Neueste Version

Die aktuell neueste Version ist **jQuery 4.0.0**.

#### Komprimierte Version für Produktion (Empfohlen)
- Download: [jquery-4.0.0.min.js](https://code.jquery.com/jquery-4.0.0.min.js)
- Geeignet für Produktion, kleine Dateigröße, schnelles Laden

#### Entwicklerversion
- Download: [jquery-4.0.0.js](https://code.jquery.com/jquery-4.0.0.js)
- Enthält vollständigen Code, einfach zu lesen und zu debuggen

#### Source Map-Datei
- Download: [jquery-4.0.0.min.map](https://code.jquery.com/jquery-4.0.0.min.map)
- Wird verwendet, um den Originalquellcode in Browser-Entwicklungstools anzuzeigen

#### Slim-Version
Die Slim-Version ist eine kleinere Version, die die Ajax- und Effects-Module ausschließt:
- [jquery-4.0.0.slim.min.js](https://code.jquery.com/jquery-4.0.0.slim.min.js) - komprimiert
- [jquery-4.0.0.slim.js](https://code.jquery.com/jquery-4.0.0.slim.js) - unkomprimiert

Wenn Sie Ajax und Animationsfunktionalität nicht benötigen, kann die Verwendung der Slim-Version die Dateigröße um etwa 30% reduzieren.

## Installationsmethoden

### Methode 1: CDN einbinden (Empfohlen)

Die Verwendung des jQuery CDN ist der einfachste Weg:

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
```

Version mit Subresource Integrity (SRI)-Validierung (empfohlen, verhindert CDN-Manipulation):

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js" 
        integrity="sha384-KJ3o2DKtIkvYIK3UENzmM7KCkRr/rE9/Qpg6aAZGJwFDMVNA/GpGFF93hXpG5KkN" 
        crossorigin="anonymous"></script>
```

Besuchen Sie [https://releases.jquery.com](https://releases.jquery.com) für die vollständige Versionsliste und SRI-Hashes.

### Methode 2: npm-Installation

Für Node.js-Projekte über npm installieren:

```bash
npm install jquery
```

Nach der Installation finden Sie in node_modules/jquery/dist/:
- jquery-4.0.0.min.js - komprimiert
- jquery-4.0.0.js - unkomprimiert
- jquery-4.0.0.min.map - Source Map

### Methode 3: Yarn-Installation

```bash
yarn add jquery
```

### Methode 4: Lokal herunterladen

Besuchen Sie die [jQuery-Downloadseite](https://jquery.com/download/), klicken Sie mit der rechten Maustaste auf die gewünschte Version und wählen Sie "Speichern unter...".

## jQuery aktualisieren

### jQuery Migrate-Plugin verwenden

Beim Aktualisieren von jQuery wird empfohlen, das jQuery Migrate-Plugin zu verwenden, um Kompatibilitätsprobleme zu identifizieren:

1. Aktualisierung von älteren Versionen auf 1.9+ oder 3.x: [jQuery Migrate 1.4.1](https://code.jquery.com/jquery-migrate-1.4.1.min.js) verwenden

2. Aktualisierung von jQuery 3.x auf eine neuere Version: [jQuery Migrate 3.6.0](https://code.jquery.com/jquery-migrate-3.6.0.min.js) verwenden

Verwendung: Migrate-Plugin nach jQuery einbinden

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
<script src="https://code.jquery.com/jquery-migrate-3.6.0.min.js"></script>
```

Migrate gibt Kompatibilitätswarnungen in der Konsole aus, um Ihnen zu helfen, Probleme Schritt für Schritt zu beheben.

## Vorabversionen

Das jQuery-Entwicklungsteam arbeitet kontinuierlich am Code. Jeder Commit im GitHub-Repository erzeugt eine Arbeitsversion. Diese Versionen können instabil sein und werden nicht für die Produktion empfohlen.

Vorabversionen ansehen: [https://releases.jquery.com/jquery/](https://releases.jquery.com/jquery/)
