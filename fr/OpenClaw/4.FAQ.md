# OpenClaw FAQ

> Foire Aux Questions sur OpenClaw

---

## Général

### Qu'est-ce qu'OpenClaw ?

OpenClaw est un cadre d'assistant IA qui utilise une technologie d'agent avancée pour automatiser les tâches. Il combine divers outils pour la gestion de fichiers, le contrôle du navigateur et plus encore.

### Pour qui est OpenClaw ?

OpenClaw est pour les développeurs, les utilisateurs avancés et toute personne cherchant à augmenter sa productivité grâce à l'automatisation alimentée par l'IA.

## Installation

### Quelles sont les exigences système ?

| Exigence | Minimum |
|----------|---------|
| Système d'Exploitation | Windows 10+, macOS, Linux |
| Node.js | Version 18+ |
| RAM | 4 Go |
| Stockage | 500 Mo |

### L'installation échoue ?

**Solution** :
1. Vérifiez la version de Node.js : `node --version`
2. Videz le cache npm : `npm cache clean --force`
3. Essayez avec les privilèges administrateur/root
4. Documentez le message d'erreur

### Le Gateway ne démarre pas ?

**Solution** :
1. Vérifiez le port : `netstat -ano | findstr 3000`
2. Utilisez un autre port : `openclaw gateway start --port 3001`
3. Consultez les logs : `openclaw gateway logs`

## Utilisation

### Comment démarrer une conversation ?

```bash
openclaw chat
```

### Comment exécuter une seule commande ?

```bash
openclaw exec "ma commande"
```

### Puis-je utiliser OpenClaw en mode headless ?

Oui, OpenClaw peut fonctionner en mode headless. Démarrez le service Gateway et utilisez l'API ou CLI.

## Gestion de Fichiers

### Comment lire un fichier ?

Utilisez l'outil de lecture :
```
read /chemin/vers/fichier.txt
```

### Comment créer un nouveau fichier ?

Utilisez l'outil d'écriture :
```
write --content "contenu" --path /nouveau/fichier.txt
```

### Comment modifier un fichier ?

Utilisez l'outil d'édition pour des modifications précises :
```
edit --path /fichier.txt --oldText "ancien" --newText "nouveau"
```

## Contrôle du Navigateur

### Quels navigateurs sont pris en charge ?

- Chrome (via extension)
- Navigateur OpenClaw isolé
- Autres via automatisation du navigateur

### Comment contrôler le navigateur ?

```bash
browser start --profile openclaw
browser navigate --url https://exemple.com
browser screenshot --path /sortie.png
```

### Le navigateur ne démarre pas ?

**Solution** :
1. Installez l'extension Chrome
2. Vérifiez les paramètres du pare-feu
3. Videz le cache du navigateur

## Sous-Agents

### Que sont les sous-agents ?

Les sous-agents sont des instances indépendantes qui peuvent exécuter des tâches spécifiques en parallèle ou séquentiellement.

### Comment créer un sous-agent ?

```bash
subagents create --task "description"
```

### Comment terminer un sous-agent ?

```bash
subagents kill --id agent-id
```

## Sécurité

### OpenClaw est-il sûr ?

OpenClaw implémente plusieurs couches de sécurité :
- Permissions de fichiers
- Confirmation pour les actions externes
- Options de sandbox

### Comment protéger les données sensibles ?

1. Utilisez une configuration chiffrée
2. Ne partagez pas les mots de passe dans les conversations
3. Utilisez des variables d'environnement pour les secrets

## Dépannage

### "Commande non trouvée" ?

Assurez-vous que :
1. OpenClaw est correctement installé
2. La variable PATH est correctement définie
3. Le Gateway est en cours d'exécution

### Les opérations de fichier échouent ?

1. Vérifiez le chemin
2. Confirmez les permissions
3. Vérifiez le nom du fichier pour les caractères spéciaux

### Temps de réponse lents ?

1. Vérifiez le statut du Gateway
2. Confirmez la connexion réseau
3. Analysez les logs
