# Руководство по использованию Docker

Основные команды Docker.

---

## Основные понятия

| Понятие | Описание |
|---------|----------|
| Образ | Шаблон для создания контейнеров |
| Контейнер | Запущенный экземпляр образа |
| Реестр | Место хранения образов |

---

## Команды для образов

```bash
docker pull nginx      # Скачать
docker images          # Список
docker rmi nginx       # Удалить
docker build -t name . # Собрать
```

---

## Команды для контейнеров

```bash
docker run -d nginx              # Фон
docker run -d -p 8080:80 nginx   # Порт
docker run -d -v /data:/data nginx # Том

docker ps             # Список
docker stop <id>     # Остановить
docker start <id>    # Запустить
docker rm <id>       # Удалить
docker logs <id>     # Логи
docker exec -it <id> bash # Shell
```

---

## Docker Compose

```bash
docker compose up -d    # Запустить
docker compose down    # Остановить
```
