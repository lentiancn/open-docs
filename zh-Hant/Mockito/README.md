# Mockito

領先的 Java 單元測試模擬框架。

## 什麼是 Mockito？

Mockito 是領先的 Java 單元測試模擬框架。它使開發人員能夠建立和驗證模擬物件，以編寫隔離的測試並消除對外部元件的依賴。

### 主要特點

- **直觀的 API** - 易於學習和使用
- **靈活的模擬** - 為類別和介面建立模擬
- **驗證** - 驗證方法調用
- **Stubbing** - 定義預定義的返回值

## 快速入門

```xml
<dependency>
    <groupId>org.mockito</groupId>
    <artifactId>mockito-core</artifactId>
    <version>5.8.0</version>
    <scope>test</scope>
</dependency>
```
