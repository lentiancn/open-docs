# Claude Code Installation Guide

Detailed installation instructions for Claude Code.

## Windows Installation

### Method One: VS Code Extension

1. Open VS Code
2. Search for "Claude Code" extension
3. Click Install
4. Sign in to your Anthropic account

### Method Two: Standalone Client

1. Visit https://claude.com/code
2. Download Windows client
3. Run installer
4. Sign in

## macOS Installation

### Method One: VS Code Extension

1. Open VS Code
2. Search for "Claude Code" extension
3. Click Install
4. Sign in

### Method Two: Homebrew

```
brew install claude-code
```

### Method Three: Standalone Client

1. Visit https://claude.com/code
2. Download macOS client
3. Run installer
4. Sign in

## Linux Installation

### Method One: VS Code Extension

1. Open VS Code
2. Search for "Claude Code" extension
3. Click Install
4. Sign in

### Method Two: Package Manager

**Ubuntu/Debian**:
```
curl -fsSL https://downloads.anthropic.com/claude-code/install.sh | sh
```

**Arch Linux**:
```
yay -S claude-code
```

## Verify Installation

```
claude --version
```

## Configuration

### API Key

1. Visit https://console.anthropic.com
2. Create API key
3. Configure in Claude Code

### Environment Variables

```
export ANTHROPIC_API_KEY=your-api-key
```

## Uninstall

### macOS

```
brew uninstall claude-code
```

### Linux

```
sudo rm -rf /usr/local/claude-code
```
