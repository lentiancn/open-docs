# Angular 使用手册

## 组件基础

组件是 Angular 应用的基本构建块。每个组件由三部分组成：

### 1. 组件定义

```typescript
import { Component } from '@angular/core';

@Component({
  selector: 'app-root',          // 组件在 HTML 中使用的标签名
  templateUrl: './app.component.html',  // 模板文件路径
  styleUrls: ['./app.component.css']    // 样式文件路径
})
export class AppComponent {
  title = '我的第一个 Angular 应用';
}
```

### 2. 组件模板

```html
<h1>{{ title }}</h1>
<button (click)="sayHello()">点击我</button>
```

### 3. 组件样式

```css
h1 {
  color: #1976d2;
}
```

## 数据绑定

数据绑定是 Angular 的核心特性之一。

### 1. 插值绑定

将组件的属性显示到模板中：

```typescript
export class AppComponent {
  name = '张三';
}
```

```html
<p>你好，{{ name }}！</p>
```

### 2. 属性绑定

将组件的属性绑定到 HTML 元素的属性：

```typescript
export class AppComponent {
  imageUrl = 'https://angular.io/assets/images/logos/angular/angular.svg';
}
```

```html
<img [src]="imageUrl" alt="Angular Logo">
```

### 3. 事件绑定

监听用户的操作：

```typescript
export class AppComponent {
  count = 0;

  increment() {
    this.count++;
  }
}
```

```html
<button (click)="increment()">计数: {{ count }}</button>
```

### 4. 双向数据绑定

使用 `ngModel` 实现双向绑定（需要导入 `FormsModule`）：

```typescript
export class AppComponent {
  username = '';
}
```

```html
<input [(ngModel)]="username" placeholder="请输入用户名">
<p>你输入的是：{{ username }}</p>
```

要使用双向绑定，需要在模块中导入 `FormsModule`：

```typescript
import { FormsModule } from '@angular/forms';

@NgModule({
  imports: [
    BrowserModule,
    FormsModule  // 添加这一行
  ],
  ...
})
export class AppModule { }
```

## 指令

指令是 Angular 用来操作 DOM 的特殊标记。

### 1. 内置指令

#### *ngIf - 条件渲染

```html
<div *ngIf="isLoggedIn">
  欢迎回来！
</div>
```

#### *ngFor - 循环渲染

```typescript
export class AppComponent {
  users = ['张三', '李四', '王五'];
}
```

```html
<ul>
  <li *ngFor="let user of users">{{ user }}</li>
</ul>
```

#### [ngClass] - 动态样式类

```typescript
export class AppComponent {
  isActive = true;
}
```

```html
<div [ngClass]="{'active': isActive}">
  内容
</div>
```

#### [ngStyle] - 动态内联样式

```html
<div [ngStyle]="{'font-size': '16px', 'color': 'red'}">
  样式内容
</div>
```

### 2. 自定义指令

创建自定义指令：

```bash
ng g directive highlight
```

```typescript
import { Directive, ElementRef, Renderer2, OnInit } from '@angular/core';

@Directive({
  selector: '[appHighlight]'
})
export class HighlightDirective implements OnInit {
  constructor(private el: ElementRef, private renderer: Renderer2) {}

  ngOnInit() {
    this.renderer.setStyle(this.el.nativeElement, 'background-color', 'yellow');
  }
}
```

使用：

```html
<p appHighlight>这段文字会高亮显示</p>
```

## 服务

服务用于封装可重用的业务逻辑和数据访问代码。

### 1. 创建服务

```bash
ng g s user
```

```typescript
import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'  // 表示服务是全局单例
})
export class UserService {
  private apiUrl = 'https://api.example.com/users';

  constructor(private http: HttpClient) {}

  getUsers(): Observable<any[]> {
    return this.http.get<any[]>(this.apiUrl);
  }

  getUser(id: number): Observable<any> {
    return this.http.get<any>(`${this.apiUrl}/${id}`);
  }
}
```

### 2. 在组件中使用服务

```typescript
import { Component, OnInit } from '@angular/core';
import { UserService } from './user.service';

@Component({
  selector: 'app-user-list',
  templateUrl: './user-list.component.html'
})
export class UserListComponent implements OnInit {
  users: any[] = [];

  constructor(private userService: UserService) {}

  ngOnInit() {
    this.userService.getUsers().subscribe(data => {
      this.users = data;
    });
  }
}
```

## 路由

### 1. 配置路由

在 `app.routes.ts` 中定义路由：

```typescript
import { Routes } from '@angular/router';
import { HomeComponent } from './home/home.component';
import { AboutComponent } from './about/about.component';
import { UserComponent } from './user/user.component';

export const routes: Routes = [
  { path: '', component: HomeComponent },
  { path: 'about', component: AboutComponent },
  { path: 'user/:id', component: UserComponent },  // :id 是路由参数
  { path: '**', redirectTo: '' }  // 404 重定向
];
```

### 2. 在模板中使用路由

```html
<nav>
  <a routerLink="/">首页</a>
  <a routerLink="/about">关于</a>
  <a [routerLink]="['/user', 1]">用户详情</a>
</nav>

<router-outlet></router-outlet>
```

### 3. 获取路由参数

```typescript
import { ActivatedRoute } from '@angular/router';

@Component({...})
export class UserComponent implements OnInit {
  userId: number = 0;

  constructor(private route: ActivatedRoute) {}

  ngOnInit() {
    this.route.paramMap.subscribe(params => {
      this.userId = Number(params.get('id'));
    });
  }
}
```

### 4. 路由守卫

#### CanActivate - 控制路由访问权限

```bash
ng g guard auth
```

```typescript
import { inject } from '@angular/core';
import { CanActivateFn, Router } from '@angular/router';

export const authGuard: CanActivateFn = (route, state) => {
  const router = inject(Router);
  const isLoggedIn = false; // 这里应该检查真实的登录状态

  if (isLoggedIn) {
    return true;
  } else {
    router.navigate(['/login']);
    return false;
  }
};
```

在路由中使用：

```typescript
export const routes: Routes = [
  { path: 'dashboard', component: DashboardComponent, canActivate: [authGuard] }
];
```

## HTTP 请求

### 1. 配置 HttpClient

在 `app.config.ts` 中：

```typescript
import { ApplicationConfig, provideZoneChangeDetection } from '@angular/core';
import { provideHttpClient } from '@angular/common/http';
import { provideRouter } from '@angular/router';

import { routes } from './app.routes';

export const appConfig: ApplicationConfig = {
  providers: [
    provideZoneChangeDetection({ eventCoalescing: true }),
    provideHttpClient(),
    provideRouter(routes)
  ]
};
```

### 2. 发送请求

```typescript
import { HttpClient } from '@angular/common/http';

@Component({...})
export class UserService {
  constructor(private http: HttpClient) {}

  getUsers() {
    return this.http.get('/api/users');
  }

  createUser(user: any) {
    return this.http.post('/api/users', user);
  }

  updateUser(id: number, user: any) {
    return this.http.put(`/api/users/${id}`, user);
  }

  deleteUser(id: number) {
    return this.http.delete(`/api/users/${id}`);
  }
}
```

### 3. 请求拦截器

创建拦截器：

```bash
ng g interceptor logging
```

```typescript
import { HttpInterceptorFn } from '@angular/common/http';

export const loggingInterceptor: HttpInterceptorFn = (req, next) => {
  console.log(`请求URL: ${req.url}`);
  return next(req);
};
```

注册拦截器：

```typescript
export const appConfig: ApplicationConfig = {
  providers: [
    provideHttpClient(withInterceptors([loggingInterceptor]))
  ]
};
```

## 表单处理

### 1. 响应式表单

```typescript
import { Component } from '@angular/core';
import { FormBuilder, FormGroup, Validators, ReactiveFormsModule } from '@angular/forms';

@Component({
  selector: 'app-login',
  templateUrl: './login.component.html',
  standalone: true,
  imports: [ReactiveFormsModule]
})
export class LoginComponent {
  loginForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.loginForm = this.fb.group({
      username: ['', [Validators.required, Validators.minLength(3)]],
      password: ['', [Validators.required, Validators.minLength(6)]]
    });
  }

  onSubmit() {
    if (this.loginForm.valid) {
      console.log('表单提交:', this.loginForm.value);
    }
  }
}
```

模板：

```html
<form [formGroup]="loginForm" (ngSubmit)="onSubmit()">
  <div>
    <label>用户名</label>
    <input formControlName="username">
    <span *ngIf="loginForm.get('username')?.errors?.['required']">用户名必填</span>
  </div>
  <div>
    <label>密码</label>
    <input type="password" formControlName="password">
  </div>
  <button type="submit" [disabled]="!loginForm.valid">登录</button>
</form>
```

### 2. 表单验证器

```typescript
import { AbstractControl, ValidationErrors, ValidatorFn } from '@angular/forms';

export function customValidator(): ValidatorFn {
  return (control: AbstractControl): ValidationErrors | null => {
    const value = control.value;
    if (value && value.includes('admin')) {
      return { customError: '不能包含 admin' };
    }
    return null;
  };
}
```

## 生命周期钩子

Angular 组件有多个生命周期钩子：

```typescript
import { Component, OnInit, OnDestroy, OnChanges, AfterViewInit } from '@angular/core';

@Component({...})
export class MyComponent implements OnInit, OnDestroy, OnChanges, AfterViewInit {
  ngOnInit() {
    // 组件初始化时调用
  }

  ngOnChanges() {
    // 输入属性变化时调用
  }

  ngAfterViewInit() {
    // 视图初始化后调用
  }

  ngOnDestroy() {
    // 组件销毁时调用，用于清理资源
  }
}
```

## 信号（Signals）

Angular 16 引入了信号，这是一种新的响应式原语。

### 1. 基本用法

```typescript
import { Component, signal } from '@angular/core';

@Component({...})
export class CounterComponent {
  count = signal(0);

  increment() {
    this.count.update(c => c + 1);
  }
}
```

```html
<p>计数: {{ count() }}</p>
<button (click)="increment()">+1</button>
```

### 2. 计算信号

```typescript
import { Component, signal, computed } from '@angular/core';

@Component({...})
export class AppComponent {
  count = signal(0);
  doubleCount = computed(() => this.count() * 2);
}
```

### 3. 效果（Effect）

```typescript
import { Component, signal, effect } from '@angular/core';

@Component({...})
export class AppComponent {
  count = signal(0);

  constructor() {
    effect(() => {
      console.log(`计数变化: ${this.count()}`);
    });
  }
}
```

## 总结

1. **组件**：构建 UI 的基本单元
2. **数据绑定**：实现数据与视图的同步
3. **指令**：操作 DOM
4. **服务**：封装业务逻辑
5. **路由**：实现页面导航
6. **HTTP**：与后端通信
7. **表单**：处理用户输入
8. **生命周期**：管理组件状态
9. **信号**：新的响应式编程方式

这些知识足以让你开始构建 Angular 应用。更多高级特性请参考官方文档。
