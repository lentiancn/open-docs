# MultiTalk Introduction

A powerful multi-party voice chat application for gaming and communication.

## What is MultiTalk?

MultiTalk is a desktop application that enables multi-party voice chat for gamers and communities. It supports multiple audio sources, low-latency communication, and customizable audio settings.

### Key Features

- **Multi-party voice** - Support for multiple participants
- **Low latency** - Real-time communication with minimal delay
- **Audio sources** - Multiple audio input/output devices
- **Push-to-talk** - Configurable hotkey for transmission
- **Noise reduction** - Built-in audio filtering

## Core Concepts

- **Room** - A virtual space for participants to communicate
- **Push-to-talk** - Press a key to transmit audio
- **Voice activation** - Automatically transmit when speaking
- **Audio devices** - Input and output devices configuration
