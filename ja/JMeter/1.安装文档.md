# JMeter インストールガイド

JMeterはApache組織開発したオープンソースのパフォーマンステストツールです。本ガイドでは、様々な環境でのインストール方法を説明します。

## バージョン情報

| バージョン | 状態 | 説明 |
|------|------|------|
| JMeter 5.x | ✅ 現行 | 最新バージョン |

## システム要件

### ハードウェア要件

| 要件 | 設定 |
|------|------|
| CPU | 2コア以上 |
| メモリ | 4GB以上 |
| ディスク | 5GB以上 |

### ソフトウェア要件

| ソフトウェア | バージョン |
|------|------|
| Java | JDK 8以上 |

## インストール方法

### 方法1：Windowsへのインストール

#### 1. ダウンロード

1. https://jmeter.apache.org/download.cgi にアクセス
2. Binariesバージョンをダウンロード
3. 指定ディレクトリに展開

#### 2. 実行

```powershell
# bin\jmeter.batをダブルクリック
# またはコマンドラインで実行
jmeter.bat
```

### 方法2：macOSへのインストール

#### 1. Homebrewを使用

```bash
brew install jmeter
```

#### 2. 手動インストール

```bash
# ダウンロードして展開
wget https://archive.apache.org/dist/jmeter/binaries/apache-jmeter-5.6.3.tgz
tar -xzf apache-jmeter-5.6.3.tgz

# 実行
./apache-jmeter-5.6.3/bin/jmeter
```

### 方法3：Linuxへのインストール

```bash
# Ubuntu/Debian
sudo apt-get install jmeter

# またはダウンロードインストール
wget https://archive.apache.org/dist/jmeter/binaries/apache-jmeter-5.6.3.tgz
tar -xzf apache-jmeter-5.6.3.tgz
cd apache-jmeter-5.6.3/bin
./jmeter
```

### 方法4：Docker

```bash
# JMeterコンテナを実行
docker run -v $(pwd):/workspace justb4/jmeter -n -t test.jmx -l result.jtl
```

## 環境設定

### Java設定

```bash
# Javaを確認
java -version

# 環境変数を設定
export JAVA_HOME=/path/to/java
export PATH=$JAVA_HOME/bin:$PATH
```

## プラグインのインストール

### 方法1：プラグインマネージャを使用

1. Plugins Managerをダウンロード
2. lib/extディレクトリに配置
3. JMeterを再起動

### 方法2：手動インストール

```bash
# プラグインをダウンロード
# lib/extディレクトリに配置
```

## よくある問題

### メモリ不足の場合

jmeter.batまたはjmeterを変更：

```bash
# ヒープサイズを設定
HEAP="-Xms1g -Xmx2g"
```

### GUIの起動が遅い場合

コマンドラインモードを使用するか、メモリを増設してください。
