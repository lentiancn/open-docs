# Vue Benutzungsanleitung

## Grundkonzepte

### Vue-Instanz

```javascript
import { createApp } from 'vue'

const app = createApp({
  data() {
    return {
      message: 'Hallo Vue!'
    }
  }
})

app.mount('#app')
```

### Vorlagensyntax

```html
<!-- Textinterpolation -->
<h1>{{ message }}</h1>

<!-- Direktiven -->
<div v-if="show">Inhalt</div>
<div v-for="item in items">{{ item }}</div>
<button @click="handleClick">Klick</button>
<input v-model="inputValue">
```

### Vue 3 Composition API (Empfohlen)

```javascript
import { ref, reactive, computed } from 'vue'

export default {
  setup() {
    const count = ref(0)
    const state = reactive({ name: 'Vue' })
    
    const doubleCount = computed(() => count.value * 2)
    
    return { count, state, doubleCount }
  }
}
```

## Komponenten

### Komponente erstellen

```vue
<template>
  <div>
    <h2>{{ title }}</h2>
    <button @click="emitEvent">Klick</button>
  </div>
</template>

<script setup>
import { defineProps, defineEmits } from 'vue'

const props = defineProps({ title: String })
const emit = defineEmits(['click'])
</script>
```

## Bedingtes Rendern

```html
<div v-if="show">Inhalt</div>
<div v-else>Standard</div>
```
