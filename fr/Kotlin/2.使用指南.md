# Guide d'utilisation de Kotlin

Ce guide présente la syntaxe de base et les fonctionnalités avancées de Kotlin.

## Démarrage rapide

### Hello World

```kotlin
fun main() {
    println("Hello, World!")
}
```

### Variables

```kotlin
// Immuable
val name: String = "Kotlin"

// Mutable
var age: Int = 10
age = 11
```

## Types de base

### Nombres

```kotlin
val int: Int = 100
val long: Long = 100L
val double: Double = 100.0
val float: Float = 100.0f
```

### Chaînes

```kotlin
val str: String = "Hello"
val template = "Name: $name, Age: $age"
```

### Tableaux

```kotlin
val arr = arrayOf(1, 2, 3)
val intArray = intArrayOf(1, 2, 3)
```

## Contrôle de flux

### Conditions

```kotlin
val max = if (a > b) a else b

// Expression When
val result = when (x) {
    1 -> "one"
    2 -> "two"
    else -> "other"
}
```

### Boucles

```kotlin
// Boucle for
for (i in 1..10) {
    println(i)
}

// While
while (x > 0) {
    x--
}
```

## Fonctions

### Fonctions de base

```kotlin
fun sum(a: Int, b: Int): Int {
    return a + b
}

// Forme abrégée
fun sum(a: Int, b: Int) = a + b
```

### Paramètres par défaut

```kotlin
fun greet(name: String = "World") {
    println("Hello, $name!")
}
```

### Paramètres vararg

```kotlin
fun sum(vararg numbers: Int): Int {
    return numbers.sum()
}
```

## Orienté objet

### Classes

```kotlin
class Person(val name: String, var age: Int) {
    
    fun speak() {
        println("My name is $name")
    }
}
```

### Héritage

```kotlin
open class Animal(val name: String) {
    open fun sound() = ""
}

class Dog(name: String) : Animal(name) {
    override fun sound() = "Woof!"
}
```

### Classes de données

```kotlin
data class User(val name: String, val email: String)
```

### Singleton

```kotlin
object Database {
    fun connect() { }
}
```

## Collections

### Listes

```kotlin
val list = listOf(1, 2, 3)
val mutableList = mutableListOf(1, 2, 3)
```

### Maps

```kotlin
val map = mapOf("a" to 1, "b" to 2)
```

### Opérations sur les collections

```kotlin
val list = listOf(1, 2, 3, 4, 5)

list.filter { it > 2 }           // [3, 4, 5]
list.map { it * 2 }              // [2, 4, 6, 8, 10]
list.reduce { acc, i -> acc + i } // 15
```

## Sécurité Null

### Types Nullable

```kotlin
var name: String? = null
name = "Kotlin"
```

### Opérateurs de sécurité

```kotlin
val length = name?.length
val len = name?.length ?: 0
```

### Assertion non-null

```kotlin
val len = name!!.length
```

## Coroutines

### Utilisation de base

```kotlin
import kotlinx.coroutines.*

suspend fun fetchData(): String {
    delay(1000)
    return "Data"
}

fun main() = runBlocking {
    val result = fetchData()
    println(result)
}
```

### Parallélisme

```kotlin
suspend fun fetchAll() = coroutineScope {
    val a = async { fetchData1() }
    val b = async { fetchData2() }
    "${a.await()} ${b.await()}"
}
```

## Extensions

### Fonctions d'extension

```kotlin
fun String.addExclamation() = this + "!"

val str = "Hello".addExclamation() // "Hello!"
```

### Propriétés d'extension

```kotlin
val String.lastChar: Char
    get() = this[length - 1]
```

## Génériques

```kotlin
class Box<T>(val value: T)

fun <T> printValue(value: T) {
    println(value)
}
```

## Bonnes pratiques

### 1. Préférer val

```kotlin
// Recommandé
val immutable = "value"

// À éviter
var mutable = "value"
```

### 2. Utiliser les classes de données

```kotlin
// Recommandé
data class User(val name: String, val age: Int)
```

### 3. Éviter null

```kotlin
// Utiliser les valeurs par défaut
val name = nullableName ?: "default"
```

### 4. Utiliser les coroutines pour l'async

```kotlin
suspend fun loadData() = withContext(Dispatchers.IO) {
    // Opérations asynchrones
}
```
