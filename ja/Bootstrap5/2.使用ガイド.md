# Bootstrap 5 使用ガイド

Bootstrap 5 を使用するガイド。

---

## グリッド

```html
<div class="container">
  <div class="row">
    <div class="col">列</div>
  </div>
</div>
```

---

## ボタン

```html
<button class="btn btn-primary">プライマリ</button>
<button class="btn btn-secondary">セカンダリ</button>
```

---

## カード

```html
<div class="card">
  <div class="card-body">コンテンツ</div>
</div>
```

---

## アイコン

```html
<i class="bi bi-github"></i>
```
