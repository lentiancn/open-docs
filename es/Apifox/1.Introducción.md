# Introducción

Apifox es una plataforma de colaboración para APIs líder en China, que resolve de manera integral el diseño de APIs, Mock, pruebas y documentación. Fue desarrollada y mantenida por Hangzhou Xiong Weiguo Network Technology Co., y es una de las herramientas de gestión de APIs más populares en China.

Apifox integra diseño de APIs, gestión de documentación, depuración, Mock y pruebas automatizadas. Los miembros del equipo pueden realizar todo el trabajo relacionado con APIs en una sola plataforma, lo cual mejora enormemente la eficiencia del desarrollo.

## Características Principales

### Diseño de APIs

Apifox ofrece funciones visuales de diseño de APIs, compatibles con las especificaciones de APIs RESTful. Los usuarios pueden definir la ruta de la interfaz, método de solicitud, parámetros de solicitud y respuesta a través de una interfaz gráfica, sin necesidad de escribir documentos en Markdown o YAML manualmente. El sistema genera automáticamente documentación de APIs profesional y permite compartirla con los miembros del equipo con un solo clic.

### Gestión de Documentación de APIs

Genera documentación de APIs automáticamente y permite explorarla en línea y exportarla. Una vez definida la API, la documentación se sincroniza automáticamente, asegurando que los miembros del equipo siempre vean la información más reciente de las interfaces. La documentación soporta múltiples formatos de exportación, incluyendo Markdown, HTML, OpenAPI (Swagger), entre otros, facilitando la integración con sistemas externos.

### Depuración de APIs

Potentes funciones de depuración en línea que permiten enviar varios tipos de solicitudes HTTP, incluyendo GET, POST, PUT, DELETE, PATCH, entre otras. Puedes personalizar encabezados de solicitud, parámetros y cuerpo, soportando múltiples formatos de datos como formulario, JSON y XML. El contenido de la respuesta se muestra formateado, incluyendo JSON, XML, HTML, con resaltado de sintaxis y funciones de plegado.

### Mock de APIs

La función inteligente de Mock es una de las características destacadas de Apifox. Solo necesitas definir la estructura de datos y el sistema generará datos Mock automáticamente según las especificaciones, sin necesidad de escribir reglas de Mock complejas manualmente. Soporta generación automática de datos según el tipo de campo, como cadenas, números, booleanos, arrays y objetos. También puedes personalizar reglas de Mock para escenarios más complejos.

### Pruebas Automatizadas

Ofrece funciones visuales de pruebas automatizadas donde puedes configurar casos de prueba mediante la interfaz gráfica, establecer condiciones de aserciones y repeticiones. No necesitas escribir código para completar las pruebas automatizadas de APIs, lo cual reduce enormemente la barrera de entrada. Soporta ejecución masiva de casos de prueba y genera informes detallados para ayudar al equipo a descubrir y localizar problemas rápidamente.

### Colaboración en Equipo

Funciones completas de colaboración que permiten invitar miembros al proyecto con diferentes roles y permisos. El contenido del proyecto se sincroniza en tiempo real, asegurando que los miembros del equipo siempre vean la última versión. Soporta funciones de comentarios y anotaciones para facilitar la comunicación directa entre los miembros del equipo sobre las interfaces.

## Casos de Uso

- Diseño de APIs y gestión de documentación
- Depuración de APIs y solución de problemas
- Datos Mock para desarrollo frontend
- Pruebas automatizadas de APIs
- Colaboración en equipo y compartición de conocimientos
- Integración con pipelines CI/CD

## Comparación con Otras Herramientas

En comparación con Postman, Apifox es más popular entre los usuarios chinos debido a: mejor soporte en chino, funciones de Mock más inteligentes, experiencia de colaboración en equipo más completa y todas las funciones básicas son gratuitas. En comparación con Swagger, Apifox ofrece una interfaz visual más intuitiva, es más fácil de usar y no requiere escribir código para gestionar y probar APIs.

Como herramienta líder de APIs en China, Apifox continúa迭代更新, ofreciendo soporte técnico y servicios de calidad para los desarrolladores.

# Guía de Instalación

Este documento presenta las diferentes formas de instalar Apifox para que puedas comenzar a usarlo rápidamente en tu máquina local.

## Requisitos del Sistema

Apifox es compatible con los siguientes sistemas operativos:

- Windows 7 o superior (64 bits)
- macOS 10.13 o superior
- Linux (Debian/Ubuntu 20.04+, Fedora 36+)

## Cómo Descargar

### Descarga desde la Web Oficial

Visita el sitio web oficial de Apifox en https://apifox.com/, haz clic en el botón "Descargar" y selecciona la versión correspondiente a tu sistema operativo.

### Selección de Versión

Apifox ofrece dos versiones:

- **Apifox** (versión estándar): Incluye todas las funciones principales, ideal para la mayoría de usuarios
- **Apifox** (versión de pago): Incluye funciones avanzadas adicionales, como gestión de equipos y automatización avanzada

Las funciones básicas son suficientes en la versión estándar.

## Pasos de Instalación

### Instalación en Windows

1. Descarga el paquete de instalación para Windows (archivo .exe)
2. Haz doble clic para ejecutar el instalador
3. Selecciona la ruta de instalación (se recomienda usar la ruta predeterminada)
4. Espera a que termine la instalación
5. Inicia Apifox e inicia sesión o regístrate

### Instalación en macOS

1. Descarga el paquete de instalación para macOS (archivo .dmg)
2. Abre el paquete y arrastra Apifox a la carpeta Aplicaciones
3. Inicia Apifox e inicia sesión o regístrate

En el primer inicio, puede que necesites permitir la ejecución de aplicaciones de desarrolladores desconocidos en las preferencias del sistema.

### Instalación en Linux

**Debian/Ubuntu:**

```bash
# Descargar el paquete deb
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.deb

# Instalar
sudo dpkg -i Apifox-latest-linux.x64.deb

# Si faltan dependencias, reparar la instalación
sudo apt-get install -f
```

**Fedora:**

```bash
# Descargar el paquete rpm
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.rpm

# Instalar
sudo rpm -i Apifox-latest-linux.x64.rpm
```

## Cuenta e Inicio de Sesión

### Registrarse

La primera vez que uses la aplicación, necesitas registrarte:

1. Inicia Apifox
2. Haz clic en el botón "Registrarse"
3. Ingresa tu correo electrónico y contraseña para completar el registro
4. Verifica tu correo electrónico (si es necesario)

### Inicio de Sesión

Soporta múltiples métodos de inicio de sesión:

- Inicio de sesión con correo y contraseña
- Inicio de sesión con cuentas de terceros (GitHub, Google, etc.)

### Sincronización en la Nube

Después de iniciar sesión, los datos del proyecto se sincronizan automáticamente con la nube. Puedes cambiar entre diferentes dispositivos sin problemas y los datos se mantienen actualizados en tiempo real.

## Configuración Inicial

Después del primer inicio, se recomienda realizar las siguientes configuraciones:

### Configurar el Tema

En "Configuración" puedes elegir entre tema claro u oscuro, según tus preferencias.

### Configurar el Proxy

Si necesitas acceder a internet a través de un proxy, configura el servidor proxy en "Configuración" -> "Red".

### Importar y Exportar

- **Importar**: Soporta importar proyectos en formatos como Postman, Swagger, Har, etc.
- **Exportar**: Soporta exportar en formatos como OpenAPI, Markdown, HTML, etc.

## Primeros Pasos

### Crear un Proyecto

1. Haz clic en el "+" junto a "Proyectos" en el panel izquierdo
2. Ingresa el nombre del proyecto
3.Selecciona el tipo de proyecto (Proyecto API o Proyecto Mock)

### Crear una Interfaz

1. En el proyecto, haz clic en "+" y selecciona "Nueva Interfaz"
2. Completa la información básica de la interfaz:
   - Nombre de la interfaz
   - Ruta de solicitud
   - Método de solicitud (GET/POST/PUT/DELETE, etc.)
3. Completa los parámetros de solicitud, cuerpo y respuesta
4. Guarda la interfaz

### Depurar una Interfaz

1. Abre la interfaz creada
2. Haz clic en el botón "Enviar"
3. Verifica los resultados de la respuesta

### Usar Mock

1. Define la estructura de datos en la interfaz
2. El sistema generará automáticamente una dirección Mock
3. Accede a la dirección Mock para obtener datos simulados

## Desinstalación

### Windows

- Abre "Panel de control" -> "Programas y características"
- Busca Apifox y haz clic en "Desinstalar"

### macOS

- Arrastra Apifox desde la carpeta Aplicaciones a la papelera
- Vacía la papelera

### Linux

```bash
# Debian/Ubuntu
sudo apt-get remove apifox

# Fedora
sudo rpm -e apifox
```
