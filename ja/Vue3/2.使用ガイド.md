# Vue 3 使用ガイド

Vue 3 を効果的に使用するガイド。

---

## Composition API

```javascript
import { ref } from 'vue'

const count = ref(0)
count.value++
```

---

## テンプレート

```html
<h1>{{ title }}</h1>
<button @click="handleClick">クリック</button>
```

---

## コンポーネント

```vue
<script setup>
defineProps({
  title: String
})
</script>
```
