# MultiTalk Installationsanleitung

Anleitung zur Installation und Konfiguration von MultiTalk.

## Voraussetzungen

- Windows 10/11, macOS 10.15+ oder Linux
- Mikrofon und Lautsprecher/Headset
- Internetverbindung

## Installation

### Windows

1. Installer von der offiziellen Website herunterladen
2. Installer ausführen
3. Installationsassistent folgen
4. MultiTalk starten

### macOS

1. .dmg-Datei herunterladen
2. Installer öffnen
3. MultiTalk in Anwendungen ziehen
4. MultiTalk starten

## Konfiguration

### Audio-Setup

1. Einstellungen > Audio öffnen
2. Eingabegerät auswählen
3. Ausgabegerät auswenden
4. Pegel testen

### Push-to-talk

1. Einstellungen > Tastenkürzel gehen
2. Push-to-talk aktivieren
3. Gewünschte Tastenkombination drücken

## Fehlerbehebung

### Kein Audio-Eingang

**Lösung:**
- Mikrofonberechtigungen prüfen
- Eingabegeräteauswahl überprüfen

### Echo-Probleme

**Lösung:**
- Lautsprecher lauter stellen
- Kopfhörer verwenden
