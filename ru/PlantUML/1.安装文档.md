# Руководство по установке PlantUML

PlantUML — это инструмент с открытым исходным кодом для быстрого создания UML-диаграмм. В этом руководстве описаны методы установки в различных средах.

## Информация о версии

| Версия | Статус | Описание |
|------|------|------|
| PlantUML | ✅ Текущая | Последняя стабильная версия |

## Методы установки

### Способ первый: Установка в Windows

#### 1. Загрузка JAR-файла

1. Загрузите PlantUML JAR: https://sourceforge.net/projects/plantuml/files/plantuml.jar/download
2. Сохраните в локальном каталоге

#### 2. Установка Graphviz (Необязательно, но рекомендуется)

```powershell
# Использование Chocolatey
choco install graphviz
```

### Способ второй: Установка в macOS

```bash
# Использование Homebrew
brew install plantuml

# Или установить Graphviz
brew install graphviz
```

### Способ третий: Установка в Linux

```bash
# Ubuntu/Debian
sudo apt-get install -y plantuml graphviz

# CentOS/RHEL
sudo yum install -y plantuml graphviz
```

### Способ четвёртый: Как плагин VS Code

1. Откройте VS Code
2. Найдите "PlantUML"
3. Установите плагин "PlantUML"
4. Установите Graphviz

### Способ пятый: Как плагин IntelliJ

1. Откройте IntelliJ IDEA
2. Settings → Plugins
3. Найдите "PlantUML"
4. Установите плагин

### Способ шестой: Использование Docker

```bash
# Запуск PlantUML-сервера
docker run -d -p 8080:8080 plantuml/plantuml-server:jetty

# Доступ по адресу http://localhost:8080
```

## Настройка окружения

### Требования к Java

```bash
# Проверить Java
java -version

# Требуется Java 8 или выше
```

### Настройка Graphviz

```bash
# Установка переменной окружения (Linux/macOS)
export GRAPHVIZ_DOT=/usr/bin/dot

# Windows добавить в PATH
# C:\Program Files\Graphviz\bin
```

## Часто задаваемые вопросы

### Изображения не отображаются?

Убедитесь, что Graphviz установлен:

```bash
dot -V
```

### Китайские иероглифы отображаются некорректно?

Добавьте в диаграмму:

```plantuml
skinparam defaultFontName SimSun
```
