# Jenkins 使用手册

## 创建任务

1. 点击"新建任务"
2. 输入任务名称
3. 选择任务类型
4. 配置构建步骤

## 配置构建

```groovy
pipeline {
    agent any
    stages {
        stage('Build') {
            steps {
                echo 'Building...'
                sh 'make'
            }
        }
        stage('Test') {
            steps {
                echo 'Testing...'
                sh 'make test'
            }
        }
        stage('Deploy') {
            steps {
                echo 'Deploying...'
            }
        }
    }
}
```
