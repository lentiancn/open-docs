# Guía de Instalación de Go

> Guía completa para instalar y configurar Go

---

## Requisitos del Sistema

| Recurso | Requisito |
|---------|-----------|
| Sistema Operativo | Windows, macOS, Linux, FreeBSD |
| Arquitectura | amd64, arm64, 386, etc. |
| Espacio en Disco | Aprox. 100 MB |

---

## Métodos de Instalación

### Instalación en Windows

#### Usando Instalador MSI

1. Descarga el instalador de Go: https://go.dev/dl/
2. Ejecuta el instalador MSI
3. Sigue el asistente de instalación
4. Abre la línea de comandos, escribe `go version` para verificar

#### Usando Chocolatey

```powershell
choco install golang
```

### Instalación en macOS

#### Usando Homebrew

```bash
brew install go
```

#### Usando Paquete DMG

1. Descarga el paquete para macOS: https://go.dev/dl/
2. Abre el archivo del paquete y ejecútalo
3. Mueve el directorio de Go a /usr/local/go: `sudo mv go /usr/local/`

### Instalación en Linux

#### Usando Gestor de Paquetes

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install golang-go

# CentOS/RHEL
sudo yum install golang
```

#### Instalación Manual

```bash
# Descargar binario de Go
wget https://go.dev/dl/go1.22.0.linux-amd64.tar.gz

# Extraer a /usr/local
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go1.22.0.linux-amd64.tar.gz

# Agregar a PATH
export PATH=$PATH:/usr/local/go/bin

# Verificar instalación
go version
```

## Configuración del Entorno

### Configurar GOPATH

```bash
# Linux/macOS
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin

# Windows (PowerShell)
$env:GOPATH = "$env:USERPROFILE\go"
$env:PATH += ";$env:GOPATH\bin"
```

### Configurar Proxy de Módulos Go

```bash
# Usar espejo de China
go env -w GOPROXY=https://goproxy.cn,direct

# Configurar proxy
go env -w GOPROXY=https://proxy.golang.org,direct
```

### Verificar Instalación

```bash
# Ver versión
go version

# Ver información del entorno
go env

# Ejecutar prueba
go version
```

## Comandos Comunes

```bash
# Inicializar módulo
go mod init <nombre-módulo>

# Descargar dependencias
go mod tidy

# Construir proyecto
go build

# Ejecutar programa
go run main.go

# Ejecutar pruebas
go test ./...

# Instalar herramientas
go install
```

## Solución de Problemas

### ¿La instalación falló?

- Verificar compatibilidad del sistema
- Confirmar que la versión descargada coincide con la arquitectura del sistema
- Verificar permisos

### ¿Comando no encontrado?

- Verificar que Go esté agregado a la variable PATH
- Cerrar y reopen la terminal
- Verificar ruta de instalación

### ¿Dependencias descargan lentamente?

- Configurar espejo proxy de China
- Usar VPN
