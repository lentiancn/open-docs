# JDK FAQ

> Frequently Asked Questions about Java Development Kit

---

## General

### What is JDK?

JDK (Java Development Kit) is the software development environment used for developing Java applications. It includes the JRE, compiler, and development tools.

### What is the difference between JDK, JRE, and JVM?

- **JVM**: Java Virtual Machine - executes bytecode
- **JRE**: Java Runtime Environment - includes JVM and libraries
- **JDK**: Java Development Kit - includes JRE + development tools

## Installation

### Which JDK version should I use?

- **JDK 21**: Latest, with latest features
- **JDK 17**: Recommended LTS version for production
- **JDK 11**: Previous LTS, still widely used
- **JDK 8**: Legacy, but still common in enterprise

### How to install JDK?

```bash
# macOS with Homebrew
brew install openjdk@17

# Ubuntu/Debian
sudo apt install openjdk-17-jdk

# Windows
# Download from Oracle or use chocolatey
choco install openjdk17
```

### How to set JAVA_HOME?

```bash
# Linux/macOS (add to ~/.bashrc or ~/.zshrc)
export JAVA_HOME=/usr/lib/jvm/java-17-openjdk
export PATH=$JAVA_HOME/bin:$PATH

# Windows (System Properties > Environment Variables)
JAVA_HOME=C:\Program Files\Java\jdk-17
```

## Development

### How to compile a Java file?

```bash
javac MyApp.java
```

### How to run a Java program?

```bash
java MyApp
```

### How to create a JAR file?

```bash
jar cf myapp.jar -C out .
```

## Troubleshooting

### "java: command not found"?

**Solution**: Set JAVA_HOME and add it to PATH.

### Which IDE should I use?

Popular Java IDEs:
- **IntelliJ IDEA**: Full-featured, paid/free
- **Eclipse**: Open source
- **VS Code**: Lightweight, with extensions
- **NetBeans**: Open source
