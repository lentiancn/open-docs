# Spring Cloud 安裝指南

本指南介紹如何安裝和設定 Spring Cloud 開發環境。

## 系統需求

### Java 和 Spring Boot 版本相容性

| 發布版本 | Spring Boot 版本 |
|---------|----------------|
| 2025.1.x (Oakwood) | 4.0.x |
| 2024.0.x (Moorgate) | 3.4.x |
| 2023.0.x (Leyton) | 3.3.x, 3.2.x |
| 2022.0.x (Kilburn) | 3.0.x, 3.1.x |

## 前置條件

### 安裝 Java

```bash
# macOS
brew install openjdk@21

# Linux
sudo apt install openjdk-21-jdk
```

## 建立專案

### 使用 Spring Initializr

1. 訪問 [start.spring.io](https://start.spring.io/)
2. 選擇：
   - 專案：Maven / Gradle
   - 語言：Java
   - Spring Boot：3.3.x
   - Group：com.example
3. 新增依賴：
   - Spring Cloud Config Client
   - Spring Cloud Eureka
   - Spring Cloud Gateway
4. 點擊「Generate」

### 命令列

```bash
curl https://start.spring.io/starter.zip \
  -d type=maven-project \
  -d language=java \
  -d bootVersion=3.3.0 \
  -d baseDir=myapp \
  -d dependencies=cloud-config-client,cloud-eureka,cloud-gateway \
  -o myapp.zip
```

## Maven 配置

```xml
<properties>
    <spring-cloud.version>2023.0.3</spring-cloud.version>
</properties>

<dependencyManagement>
    <dependencies>
        <dependency>
            <groupId>org.springframework.cloud</groupId>
            <artifactId>spring-cloud-dependencies</artifactId>
            <version>${spring-cloud.version}</version>
            <type>pom</type>
            <scope>import</scope>
        </dependency>
    </dependencies>
</dependencyManagement>
```

## 主要專案

| 專案 | 說明 |
|------|------|
| Spring Cloud Config | 集中配置管理 |
| Spring Cloud Gateway | API 閘道器 |
| Spring Cloud Eureka | 服務發現 |
| Spring Cloud OpenFeign | REST 客戶端 |
