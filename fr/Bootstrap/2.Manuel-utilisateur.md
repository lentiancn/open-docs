# Manuel utilisateur Bootstrap

Ce manuel fournit des informations détaillées sur les composants principaux de Bootstrap.

## Composants

### Boutons

`html
<button class=""btn btn-primary"">Primaire</button>
<button class=""btn btn-secondary"">Secondaire</button>
`

### Cartes

`html
<div class=""card"" style=""width: 18rem;"">
    <div class=""card-body"">
        <h5 class=""card-title"">Titre</h5>
    </div>
</div>
`
