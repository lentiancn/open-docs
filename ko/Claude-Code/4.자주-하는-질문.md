# Claude Code 자주 묻는 질문

자주 묻는 질문과 답변.

## 설치

### Q: 설치 실패?

**해결책**:
1. 시스템 요구 사항 확인
2. VS Code를 최신 버전으로 업데이트
3. 네트워크 연결 확인
4. 재설치 시도

### Q: 로그인 불가?

**해결책**:
1. API 키가 유효한지 확인
2. 네트워크 연결 확인
3. 캐시 지우기 시도

### Q: 확장이 표시되지 않음?

**해결책**:
1. VS Code 재시작
2. 확장이 활성화되어 있는지 확인
3. VS Code 알림 확인

## 사용

### Q: 응답 느림?

**해결책**:
1. 네트워크 연결 확인
2. 컨텍스트 길이 줄이기
3. 서버 응답 대기

### Q: 코드 생성 품질 나쁨?

**해결책**:
1. 설명을 더 구체적으로
2. 더 많은 컨텍스트 제공
3. 반복 개선

### Q: 모델切换하는 방법은?

설정에서 모델 버전 선택.

### Q: 어떤 언어를 지원합니까?

주요 언어 지원: Python, JavaScript, TypeScript, Go, Rust, Java, C++ 등.

## 계정

### Q: API 키를 어떻게 얻습니까?

https://console.anthropic.com을 방문하여 생성.

### Q: 할당량 소진?

콘솔에서 추가 할당량 구매.

## 보안

### Q: 코드가 유출됩니다?

Claude Code는 코드를 저장하거나 공유하지 않습니다.

### Q: 프라이버시를 보호하는 방법은?

- 대화에서 민감한 정보 입력하지 않음
- 로컬 모드 사용
- 세션 정기적으로 지우기
