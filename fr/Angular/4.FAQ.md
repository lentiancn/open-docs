# Angular FAQ

## Q: ng non reconnu?
**R:** Vérifier Node.js: `node --version`. Redémarrer terminal.

## Q: Installation lente?
**R:** Miroir: `npm config set registry https://registry.npmmirror.com`

## Q: Erreur CORS?
**R:** Configurer proxy ou ajouter en-têtes CORS.

## Q: Mettre à jour Angular?
**R:** `npm install -g @angular/cli@latest`
