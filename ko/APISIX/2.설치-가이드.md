# APISIX 설치 가이드

APISIX 다양한 설치 방법.

## 환경 요구사항

### 하드웨어 요구사항

- CPU: 2코어 이상
- 메모리: 4GB 이상
- 디스크: 40GB 이상

### 소프트웨어 요구사항

- OS: Linux (Ubuntu 20.04+, CentOS 7+ 권장)
- Nginx 또는 OpenResty
- Lua 5.1+
- etcd 3.4+

## Docker 설치 (권장)

### 빠른 시작

```bash
# etcd 시작
docker run -d --name etcd -p 2379:2379 apache/apisix:latest etcd

# APISIX 시작
docker run -d --name apisix -p 9080:9080 -p 9090:9090 apache/apisix:latest
```

### docker-compose

```yaml
version: '3'
services:
  etcd:
    image: apache/apisix:3.5.0-etcd
    ports:
      - "2379:2379"
  apisix:
    image: apache/apisix:3.5.0
    ports:
      - "9080:9080"
    depends_on:
      - etcd
```

```bash
docker-compose up -d
```

## Helm 설치 (Kubernetes)

```bash
helm repo add apisix https://apache.github.io/apisix-helm-chart
helm install apisix apisix/apisix
```

## 소스 설치

### 의존성 설치

**Ubuntu/Debian:**

```bash
wget -qO - https://openresty.org/package/pubkey.gpg | sudo apt-key add -
sudo apt-get update
sudo apt-get install -y openresty etcd
```

### 소스 클론

```bash
git clone https://github.com/apache/apisix.git
cd apisix
git checkout 3.5.0
```

### 의존성 설치

```bash
make deps
make init
```

### 시작

```bash
make run
```

## 설정

```yaml
apisix:
  node_listen: 9080
  enable_admin: true
  admin_key:
    - name: admin
      key: 123456

etcd:
  host:
    - "http://127.0.0.1:2379"
```

## 제거

```bash
docker stop apisix etcd
docker rm apisix etcd
```
