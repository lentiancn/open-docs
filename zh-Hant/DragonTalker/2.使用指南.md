# DragonTalker 使用指南

DragonTalker 使用深度學習從單張圖像和音訊輸入生成逼真的說話頭像影片。

## 快速開始

### 基本命令

```bash
python inference.py \
  --source_image examples/source_image.jpg \
  --driven_audio examples/driven_audio.wav \
  --result_dir ./results
```

### 帶增強功能

```bash
python inference.py \
  --source_image examples/source_image.jpg \
  --driven_audio examples/driven_audio.wav \
  --result_dir ./results \
  --preprocess full \
  --enhancer gfpgan
```

## 命令參數

### 輸入參數

| 參數 | 說明 | 預設值 |
|------|------|--------|
| `--source_image` | 輸入肖像圖片路徑 | 必需 |
| `--driven_audio` | 輸入音訊檔案路徑 (WAV/MP3) | 必需 |
| `--result_dir` | 輸出目錄 | ./results |

### 處理參數

| 參數 | 說明 | 預設值 |
|------|------|--------|
| `--preprocess` | 圖像預處理：crop, resize, full | full |
| `--size` | 圖像大小：256, 512 | 256 |
| `--pose_style` | 姿勢風格 0-45 | 0 |
| `--expression_scale` | 表情強度 0.5-1.5 | 1.0 |

### 增強參數

| 參數 | 說明 | 預設值 |
|------|------|--------|
| `--enhancer` | 人臉增強器：gfpgan, RestoreFormer, CodeFormer | None |

### 效能參數

| 參數 | 說明 | 預設值 |
|------|------|--------|
| `--batch_size` | 批處理大小 | 1 |
| `--fps` | 輸出影片幀率 | 25 |

## Python API

### 基本用法

```python
from inference import DragonTalker

# 初始化 DragonTalker
dragon = DragonTalker()

# 產生影片
video_path = dragon.generate(
    source_image="image.jpg",
    driven_audio="audio.wav",
    preprocess="full",
    enhancer="gfpgan"
)

print(f"影片儲存至: {video_path}")
```

### 高級用法

```python
from inference import DragonTalker
import torch

# 使用自訂設定初始化
dragon = DragonTalker(
    checkpoint_path="checkpoints/DragonTalker.pth",
    config_path="config/DragonTalker.yaml",
    device="cuda" if torch.cuda.is_available() else "cpu"
)

# 產生帶高級選項的影片
video_path = dragon.generate(
    source_image="image.jpg",
    driven_audio="audio.wav",
    preprocess="full",
    pose_style=10,
    expression_scale=1.2,
    enhancer="gfpgan",
    batch_size=1,
    output_video="output.mp4"
)
```

### 批量處理

```python
from inference import DragonTalker
import os

dragon = DragonTalker()

# 處理多張圖片（同一音訊）
audio_file = "audio.wav"
image_dir = "images/"

for image_file in os.listdir(image_dir):
    if image_file.endswith(('.jpg', '.png')):
        output_path = f"results/{image_file}"
        dragon.generate(
            source_image=os.path.join(image_dir, image_file),
            driven_audio=audio_file,
            preprocess="full"
        )
```

## Web 介面

### 執行 Web 演示

```bash
python app.py
```

然後在瀏覽器中開啟 http://localhost:8888

### Web 介面功能

1. **圖像上傳** - 拖放或選擇肖像圖片
2. **音訊上傳** - 選擇音訊檔案（WAV/MP3）
3. **預覽** - 預覽來源圖像和音訊
4. **參數調整** - 調整姿勢、表情、增強器
5. **產生** - 一鍵產生影片
6. **下載** - 儲存產生的影片

## 人臉增強器

### 可用增強器

| 增強器 | 品質 | 速度 | 顯示記憶體 |
|--------|------|------|------------|
| 無 | 基礎 | 快 | 低 |
| gfpgan | 良好 | 中等 | 中等 |
| RestoreFormer | 非常好 | 慢 | 高 |
| CodeFormer | 非常好 | 慢 | 高 |

### 建議

- **低顯示記憶體**：無增強器或 gfpgan
- **平衡**：gfpgan
- **最佳品質**：RestoreFormer 或 CodeFormer

## 預處理模式

| 模式 | 說明 | 適用場景 |
|------|------|----------|
| crop | 居中裁剪人臉 | 快速處理 |
| resize | 調整為目標大小 | 簡單背景 |
| full | 完整處理流程 | 最佳效果 |

## 輸入需求

### 圖像需求

- 格式：JPG、PNG
- 解析度：建議 512x512 或更大
- 人臉：正面、無遮擋
- 背景：簡單最好

### 音訊需求

- 格式：WAV、MP3
- 時長：1-60 秒
- 取樣率：16000-48000 Hz
- 語音：清晰、最小噪音

## 輸出

### 影片格式

- 格式：MP4
- 編解碼器：H.264
- 解析度：256x256 或 512x512
- 幀率：25
- 位元率：2-5 Mbps

### 輸出目錄

結果儲存在指定的輸出目錄：
```
results/
├── video.mp4          # 產生的影片
└── (暫時檔案)
```

## 故障排除

### 影片過暗/過亮

**原因：** 訓練和輸入之間的光照不匹配

**解決方案：**
- 調整 expression_scale 參數 (0.8-1.2)
- 使用更高品質的來源圖像
- 嘗試不同的增強器

### 嘴唇不同步

**原因：** 音訊品質或對齊問題

**解決方案：**
- 使用高品質音訊
- 確保音訊有清晰的語音
- 檢查音影片同步

### 臉部變形

**原因：** 輸入品質低或不尋常

**解決方案：**
- 使用更高解析度圖像 (512)
- 嘗試不同的預處理模式
- 使用人臉增強器
- 確保臉部清晰可見

### 處理緩慢

**解決方案：**
- 使用 `--preprocess crop`
- 設定 `--size 256`
- 停用增強器
- 減小批次大小

### GPU 記憶體不足

**解決方案：**
- 將 batch_size 減小到 1
- 使用更小的圖像大小
- 關閉其他 GPU 應用程式

## 效能提示

3. 停用增強器
