# frp 使用手册

## 基础使用

### TCP 穿透

```ini
[common]
server_addr = 123.45.67.89
server_port = 7000
token = your_token

[ssh]
type = tcp
local_ip = 127.0.0.1
local_port = 22
remote_port = 6000
```

访问：`ssh -p 6000 user@123.45.67.89`

### HTTP 服务穿透

```ini
[web]
type = http
local_port = 8080
custom_domains = yourdomain.com
```

### UDP 穿透

```ini
[dns]
type = udp
local_ip = 127.0.0.1
local_port = 53
remote_port = 6001
```

### 端口复用

```ini
[common]
bind_port = 7000
vhost_http_port = 80
vhost_https_port = 443
```

## 高级功能

### 负载均衡

```ini
[web1]
type = http
local_port = 8080
custom_domains = web.yourdomain.com
group = web_group
group_key = group_key

[web2]
type = http
local_port = 8081
custom_domains = web.yourdomain.com
group = web_group
group_key = group_key
```

### 身份验证

```ini
[common]
token = your_token

# 或使用 OAuth2
authentication_method = oidc
oidc_client_id = your_client_id
oidc_client_secret = your_secret
```

### 加密与压缩

```ini
[common]
use_encryption = true
use_compression = true
```

### 热更新配置

```bash
# 客户端热更新
./frpc reload -c frpc.ini
```

## 命令行参数

```bash
# 服务端
frps -h
frps -c frps.ini

# 客户端  
frpc -h
frpc -c frpc.ini
```

## 监控面板

访问服务端 dashboard（默认 7500 端口）查看连接状态和流量统计。
