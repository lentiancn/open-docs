# Claude Code 常見問題

常見問題解答。

## 安裝問題

### Q: 安裝失敗？

**解決方案**：
1. 檢查系統要求
2. 更新 VS Code 到最新版本
3. 檢查網絡連接
4. 嘗試重新安裝

### Q: 無法登錄？

**解決方案**：
1. 檢查 API 密鑰是否有效
2. 檢查網絡是否正常
3. 嘗試清除緩存

### Q: 擴展不顯示？

**解決方案**：
1. 重啟 VS Code
2. 檢查擴展是否啟用
3. 查看 VS Code 通知

## 使用問題

### Q: 響應慢？

**解決方案**：
1. 檢查網絡連接
2. 減少上下文長度
3. 等待服務器響應

### Q: 代碼生成質量差？

**解決方案**：
1. 描述更具體
2. 提供更多上下文
3. 迭代改進

### Q: 如何切換模型？

在設置中選擇模型版本。

### Q: 支持哪些語言？

支持主流編程語言：Python、JavaScript、TypeScript、Go、Rust、Java、C++ 等。

## 賬戶問題

### Q: 如何獲取 API 密鑰？

訪問 https://console.anthropic.com 創建。

### Q: 額度用完了？

在控制台購買額外額度。

## 安全問題

### Q: 代碼會洩露嗎？

Claude Code 不會保存或分享你的代碼。

### Q: 如何保護隱私？

- 不在對話中輸入敏感信息
- 使用本地模式
- 定期清除會話
