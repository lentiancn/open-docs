# Element Plus Manuel utilisateur

Guide d'utilisation.

## Démarrage rapide

```vue
<template>
  <el-button type="primary" @click="handleClick">
    Cliquez
  </el-button>
</template>

<script setup>
import { ref } from 'vue'

const handleClick = () => {
  ElMessage.success('Bonjour!')
}
</script>
```

## Composants

### Button

```vue
<el-button>Défaut</el-button>
<el-button type="primary">Primaire</el-button>
<el-button disabled>Désactivé</el-button>
<el-button loading>Chargement</el-button>
```

### Input

```vue
<el-input v-model="value" placeholder="Entrez" />
<el-input v-model="value" disabled placeholder="Désactivé" />
<el-input v-model="value" clearable placeholder="Effaçable" />
```

### Table

```vue
<el-table :data="tableData" stripe border>
  <el-table-column prop="name" label="Nom" />
  <el-table-column prop="email" label="Email" />
</el-table>
```

### Form

```vue
<el-form :model="form" :rules="rules" ref="formRef">
  <el-form-item label="Nom" prop="name">
    <el-input v-model="form.name" />
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="submit">Envoyer</el-button>
  </el-form-item>
</el-form>
```

### Dialog

```vue
<el-button @click="visible = true">Ouvrir</el-button>
<el-dialog v-model="visible" title="Titre">
  <span>Contenu</span>
</el-dialog>
```

### Message

```typescript
import { ElMessage } from 'element-plus'

ElMessage.success('Succès')
ElMessage.error('Erreur')
```

## Mise en page

```vue
<el-row :gutter="20">
  <el-col :span="12"><div>12</div></el-col>
  <el-col :span="12"><div>12</div></el-col>
</el-row>
```
