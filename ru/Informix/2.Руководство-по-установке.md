# Руководство по установке IBM Informix

## Системные требования

**Поддерживаемые платформы:**
- Linux (Red Hat, Ubuntu, SUSE и др.)
- Windows Server
- AIX
- HP-UX
- Solaris
- macOS
- ARM (для пограничных устройств)

**Минимальные требования:**
- ОЗУ: минимум 1 ГБ (рекомендуется 8 ГБ)
- Диск: минимум 2 ГБ для установки
- CPU: совместимость с архитектурами x86, PowerPC, ARM

## Шаги установки в Linux

### Шаг 1: Загрузка установщика

Посетите IBM Passport Advantage или IBM Fix Central и загрузите соответствующий пакет установщика Informix.

### Шаг 2: Подготовка системы

```bash
# Создание пользователя и группы informix
groupadd informix
useradd -g informix -d /home/informix informix

# Создание каталога установки
mkdir -p /opt/IBM/informix
chown informix:informix /opt/IBM/informix
```

### Шаг 3: Распаковка установщика

```bash
tar -xf informix.tar
cd informix
```

### Шаг 4: Запуск установки

```bash
# От имени root или с sudo
./install
```

Для более новых версий (14.10+) используйте IBM Installation Manager:

```bash
chmod +x ids_install
./ids_install
```

Следуйте инструкциям:
- Примите лицензионное соглашение
- Выберите путь установки (/opt/IBM/informix)
- Выберите компоненты для установки

### Шаг 5: Настройка переменных окружения

Добавьте в `~/.bashrc` или `/etc/profile`:

```bash
export INFORMIXDIR=/opt/IBM/informix
export PATH=$INFORMIXDIR/bin:$PATH
export ONCONFIG=onconfig
export INFORMIXSERVER=имя_вашего_сервера
```

Примените изменения:

```bash
source ~/.bashrc
```

### Шаг 6: Настройка файла sqlhosts

```bash
cd $INFORMIXDIR/etc
cp sqlhosts.std sqlhosts
vi sqlhosts
```

Добавьте запись сервера:

```
ol_informix onsoctcp localhost ol_informix
```

### Шаг 7: Инициализация сервера базы данных

```bash
# Инициализация Informix с параметрами по умолчанию
oninit -ivy
```

### Шаг 8: Проверка установки

```bash
# Проверка статуса сервера
onstat -

# Проверка инициализации
ls -la $INFORMIXDIR/storage/
```

## Шаги установки в Windows

### Шаг 1: Загрузка установщика для Windows

Загрузите с IBM Passport Advantage.

### Шаг 2: Запуск установщика

- Дважды щёлкните файл `.exe`
- Следуйте мастеру установки

### Шаг 3: Настройка

- Используйте мастер настройки Informix
- Настройте службу Windows

### Шаг 4: Настройка переменных окружения

Настройте `INFORMIXDIR` в свойствах системы:
- Щёлкните правой кнопкой «Мой компьютер» → Свойства
- Дополнительные параметры системы → Переменные среды
- Добавьте `INFORMIXDIR` и `INFORMIXSERVER`

## Проверка после установки

```bash
# Проверка статуса сервера
onstat -

# Запуск сервера (если не запущен)
oninit

# Остановка сервера
onmode -ky

# Подключение к базе данных
dbaccess
```

## Типичные задачи после установки

### Создание дополнительных хранилищ

```bash
onspaces -c -d datadbs -p /путь/к/data_chunk -s 500000
```

### Настройка логических журналов

```bash
onparams -a -d datadbs -s 10000
```

## Устранение неполадок

### Сервер не запускается

- Проверьте правильность настройки переменных окружения
- Проверьте файлы `onconfig` и `sqlhosts`
- Просмотрите журналы ошибок в `$INFORMIXDIR/tmp/`
- Убедитесь в достаточном месте на диске и правах доступа

### Проблемы с подключением

- Проверьте настройки брандмауэра (порт по умолчанию 9088)
- Проверьте соответствие `INFORMIXSERVER` конфигурации sqlhosts
- Убедитесь в правильности сетевых настроек в sqlhosts
