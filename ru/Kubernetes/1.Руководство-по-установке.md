# Руководство по установке Kubernetes

Установка Kubernetes.

---

## Методы

| Метод | Описание |
|-------|----------|
| **minikube** | Локальный кластер |
| **kind** | K8s в Docker |
| **kubeadm** | Продакшен |

## minikube

```bash
brew install minikube
minikube start
```

---

## kubectl

```bash
brew install kubectl
```

---

## Проверка

```bash
kubectl get nodes
```
