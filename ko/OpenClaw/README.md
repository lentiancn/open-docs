# OpenClaw

[English](./README.md) | [简体中文](./README.md) | [日本語](./README.md)

🦞 어떤 OS든 가능한 AI 어시스턴트 게이트웨이. WhatsApp, Telegram, Discord, iMessage 등 지원.

## 기능

- 🤖 AI 어시스턴트 게이트웨이
- 📱 멀티플랫폼
- 🔒 로컬 배포
- 🧠 여러 에이전트
- 🌐 웹 관리
- 📷 모바일 노드

## 빠른 시작

```bash
npm install -g openclaw@latest
openclaw onboard --install-daemon
openclaw gateway --port 18789
```

열기: http://127.0.0.1:18789/
