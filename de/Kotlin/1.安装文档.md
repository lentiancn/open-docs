# Kotlin Installationsanleitung

Kotlin ist eine von JetBrains entwickelte moderne Programmiersprache. Diese Anleitung behandelt die Installation in verschiedenen Umgebungen.

## Versionsinformationen

| Version | Status | Beschreibung |
|---------|--------|--------------|
| Kotlin 2.x | ✅ Aktuell | Neueste Version |
| Kotlin 1.9.x | ✅ Unterstützt | Stabile Version |

## Systemanforderungen

### Hardwareanforderungen

| Anforderung | Konfiguration |
|-------------|---------------|
| CPU | 1+ Kerne |
| Speicher | 2+ GB |
| Festplatte | 500+ MB |

### Softwareanforderungen

| Software | Version |
|----------|---------|
| Java | JDK 8 oder höher |

## Installationsmethoden

### Methode 1: IntelliJ IDEA (Empfohlen)

1. IntelliJ IDEA herunterladen: https://www.jetbrains.com/idea/download/
2. Ultimate oder Community Edition installieren
3. Kotlin ist bereits integriert

### Methode 2: Kommandozeile

#### Windows

```powershell
# Mit Chocolatey
choco install kotlin

# Oder manuell herunterladen
# Von https://github.com/JetBrains/kotlin/releases herunterladen
```

#### macOS

```bash
# Mit Homebrew
brew install kotlin

# Oder manuelle Installation
curl -s https://get.sdkman.io | bash
sdk install kotlin
```

#### Linux

```bash
# Ubuntu/Debian
sudo apt install kotlin

# Oder mit SDKMAN
curl -s https://get.sdkman.io | bash
sdk install kotlin
```

### Methode 3: SDKMAN

```bash
# SDKMAN installieren
curl -s "https://get.sdkman.io" | bash

# Kotlin installieren
sdk install kotlin

# Verifizieren
kotlin -version
```

### Methode 4: Maven

```xml
<dependencies>
    <dependency>
        <groupId>org.jetbrains.kotlin</groupId>
        <artifactId>kotlin-stdlib</artifactId>
        <version>2.0.0</version>
    </dependency>
</dependencies>
```

### Methode 5: Gradle

```groovy
plugins {
    id 'org.jetbrains.kotlin.jvm' version '2.0.0'
}

repositories {
    mavenCentral()
}

dependencies {
    implementation "org.jetbrains.kotlin:kotlin-stdlib"
}
```

### Methode 6: Docker

```dockerfile
# Kotlin-Compiler-Image verwenden
FROM gradle:8.5-jdk-21 AS builder
WORKDIR /app
COPY . .
RUN gradle build

# Laufzeit-Image
FROM openjdk:21
COPY --from=builder /app/build/libs/*.jar /app.jar
ENTRYPOINT ["java", "-jar", "/app.jar"]
```

## Umgebungskonfiguration

### Java-Konfiguration

```bash
# Java prüfen
java -version

# JAVA_HOME setzen
export JAVA_HOME=/path/to/java
```

### Kompilieren und Ausführen

```bash
# Kompilieren
kotlinc hello.kt -include-runtime -d hello.jar

# Ausführen
java -jar hello.jar

# Kotlin-Skript verwenden
kotlin hello.kts
```

## IDE-Konfiguration

### IntelliJ IDEA

1. Kotlin-Plugin installieren (in der Regel vorinstalliert)
2. Kotlin-Projekt erstellen
3. Kotlin SDK konfigurieren

### VS Code

1. Kotlin Language Server Erweiterung installieren
2. Kotlin-Compiler installieren

### Eclipse

1. Kotlin-Plugin installieren (Kotlin IDE for Eclipse)
2. Kotlin-Projekt erstellen

## Häufige Probleme

### Kompilierungsfehler?

Prüfe die Kompatibilität von Kotlin und Java Versionen.

### Nicht genügend Speicher?

Erhöhe den JVM-Heap-Speicher:

```bash
kotlinc -J-Xmx2g
```
