# Angular Guía de Instalación

Configuración completa del entorno de desarrollo Angular.

## Requisitos del entorno

### Requisitos de hardware

- CPU: 2 núcleos o más
- Memoria: 4GB o más
- Disco: 10GB o más

### Requisitos de software

- Sistema operativo: Windows/macOS/Linux
- Node.js: 18.10.0+ o 20.0.0+
- npm: 9.0.0+ (incluido con Node.js)
- Git: Cualquier versión

## Pasos de instalación

### 1. Instalar Node.js

**Windows/macOS:**

Descargar desde: https://nodejs.org

O usar nvm:

```bash
# Instalar nvm (macOS/Linux)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Instalar Node.js LTS
nvm install --lts
nvm use --lts
```

**Verificar instalación:**

```bash
node --version  # Debe mostrar v18.x.x o v20.x.x
npm --version    # Debe mostrar 9.x.x o superior
```

### 2. Instalar Angular CLI

```bash
# Instalar globalmente vía npm
npm install -g @angular/cli

# Verificar instalación
ng version
```

### 3. Crear nuevo proyecto

```bash
# Crear nueva aplicación Angular
ng new my-app

# Navegar al proyecto
cd my-app

# Iniciar servidor de desarrollo
ng serve
```

### 4. Acceder a la aplicación

Abrir navegador: http://localhost:4200

## Comandos comunes

### Comandos de proyecto

```bash
# Crear nuevo proyecto con routing y SCSS
ng new my-app --routing --style=scss

# Crear app standalone (Angular 14+)
ng new my-app --standalone

# Iniciar servidor de desarrollo
ng serve

# Construir para producción
ng build

# Ejecutar pruebas unitarias
ng test

# Ejecutar pruebas E2E
ng e2e
```

### Comandos de generación

```bash
# Generar componente
ng generate component my-component
ng g c my-component

# Generar servicio
ng generate service my-service
ng g s my-service

# Generar directiva
ng generate directive my-directive
ng g d my-directive

# Generar pipe
ng generate pipe my-pipe
ng g p my-pipe

# Generar módulo
ng generate module my-module
ng g m my-module

# Generar clase
ng generate class my-class
ng g cl my-class

# Generar interfaz
ng generate interface my-interface
ng g i my-interface

# Generar guard
ng generate guard my-guard
ng g g my-guard

# Generar interceptor
ng generate interceptor my-interceptor
ng g in my-interceptor
```

## Configuración de entorno

### Entorno de desarrollo

```typescript
// src/environments/environment.ts
export const environment = {
  production: false,
  apiUrl: 'http://localhost:3000/api'
};
```

### Entorno de producción

```typescript
// src/environments/environment.prod.ts
export const environment = {
  production: true,
  apiUrl: 'https://api.example.com'
};
```

### Usar entorno

```typescript
import { Component } from '@angular/core';
import { environment } from '../environments/environment';

@Component({
  selector: 'app-root',
  template: `<p>API: {{ apiUrl }}</p>`
})
export class AppComponent {
  apiUrl = environment.apiUrl;
}
```

## Problemas comunes

### ¿npm lento?

```bash
# Usar mirror de Taobao
npm config set registry https://registry.npmmirror.com

# O usar nrm para gestionar mirrors
npm install -g nrm
nrm use taobao
```

### ¿Versión de Node no coincide?

```bash
# Usar nvm para cambiar versión
nvm install 20
nvm use 20
```

### ¿Problemas de permisos (Linux/macOS)?

```bash
# Corregir permisos de npm
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc
```

## Desinstalar

```bash
# Desinstalar Angular CLI
npm uninstall -g @angular/cli

# Eliminar proyecto
rm -rf my-app
```
