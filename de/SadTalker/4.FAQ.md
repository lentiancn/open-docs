# SadTalker Häufig Gestellte Fragen

## Allgemeine Fragen

### Was ist SadTalker?
SadTalker ist ein Deep-Learning-Projekt, das realistische Sprechgesicht-Videos aus einem einzelnen statischen Bild und Audio-Eingabe generiert.

### Ist SadTalker kostenlos nutzbar?
Ja, SadTalker ist Open-Source und kostenlos für persönliche und Forschungszwecke.

### Was sind die Systemanforderungen?
- **GPU**: NVIDIA GPU mit 8GB+ VRAM (empfohlen)
- **RAM**: 8GB Minimum, 16GB empfohlen
- **Speicher**: 20GB+ für Modelle und Ausgaben
- **OS**: Linux (Ubuntu 20.04+), Windows 10/11, macOS

## Installationsfragen

### Wie installiere ich SadTalker?
```bash
# Repository klonen
git clone https://github.com/WinfredSadTalker/SadTalker.git
cd SadTalker

# Conda-Umgebung erstellen
conda create -n sadtalker python=3.8
conda activate sadtalker

# Abhängigkeiten installieren
pip install -r requirements.txt

# Vortrainierte Modelle herunterladen
bash scripts/download_models.sh
```

### Welche Python-Version sollte ich verwenden?
SadTalker funktioniert am besten mit Python 3.8 oder 3.9.

## Nutzungsfragen

### Wie generiere ich ein Sprechvideo?
```bash
python inference.py --source_image path/to/image.jpg --audio path/to/audio.wav
```

### Welche Bildformate werden unterstützt?
- JPG/JPEG
- PNG
- Die meisten gängigen Bildformate

### Welche Audioformate werden unterstützt?
- WAV (empfohlen)
- MP3
- Andere von FFmpeg unterstützte Formate

## Fehlerbehebung

### Modell-Download fehlschlägt
Laden Sie Modelle manuell vom offiziellen Repository herunter.

### Out-of-Memory-Fehler
- Verringern Sie die Batch-Größe
- Senken Sie die Ausgabeauflösung
- Schließen Sie andere GPU-Anwendungen

### Schlechte Qualität der Ergebnisse
- Verwenden Sie hochauflösende Eingabebilder
- Stellen Sie sicher, dass das Audio ohne Hintergrundgeräusche ist
