# Angular

现代 Web 开发平台。

## 简介

Angular 是由 Google 开发的设计平台，用于构建移动端和桌面端 Web 应用。使用 TypeScript/JavaScript，具有组件化、双向绑定、依赖注入等特点。

### 主要特性

- **组件化** - 基于组件的架构
- **TypeScript** - 强类型支持
- **双向绑定** - 简化数据同步
- **依赖注入** - 强大的依赖管理系统
- **路由** - 完整的路由解决方案

- GitHub 星标：90,000+
- 全球数百万开发者

## 快速开始

### 安装

```bash
npm install -g @angular/cli
```

### 创建项目

```bash
ng new my-app
cd my-app
ng serve
```

访问：http://localhost:4200
