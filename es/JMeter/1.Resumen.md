# Apache JMeter - Resumen

## ¿Qué es Apache JMeter?

Apache JMeter™ es software de código abierto, una aplicación 100% pura Java diseñada para pruebas de carga y medición de rendimiento. Fue diseñado originalmente para probar aplicaciones web pero desde entonces se ha expandido a otras funciones de prueba.

## ¿Qué puede hacer JMeter?

Apache JMeter puede probar el rendimiento de recursos estáticos y dinámicos, aplicaciones web dinámicas. Puede simular una carga pesada en un servidor, grupo de servidores, red u objeto para probar su resistencia o analizar el rendimiento general bajo diferentes tipos de carga.

### Características principales

- **Soporte multi-protocolo**: Web (HTTP, HTTPS), Servicios Web SOAP/REST, FTP, Base de datos via JDBC, LDAP, JMS, Mail, TCP, Objetos Java
- **IDE de pruebas completo**: Grabar planes de prueba desde navegadores o aplicaciones nativas
- **Modo CLI**: Ejecutar pruebas de carga desde cualquier sistema operativo compatible con Java
- **Informes HTML dinámicos**: Completos y listos para presentar
- **Extracción de datos**: Soporte para HTML, JSON, XML y expresiones regulares
- **100% Java**: Total portabilidad
- **Multi-threading**: Muestreo simultáneo por muchos hilos
- **Altamente extensible**: Samplers conectables y samplers scriptables (Groovy, BeanShell)
- **Integración CI**: Soporte para Maven, Gradle y Jenkins

## Lo que JMeter NO es

JMeter no es un navegador - funciona a nivel de protocolo. No ejecuta JavaScript encontrado en páginas HTML ni renderiza páginas HTML como un navegador real.

## Primeros pasos

1. Descargar JMeter desde el sitio web oficial
2. Ejecutar el script de JMeter (jmeter.bat para Windows, jmeter.sh para Linux/Mac)
3. Crear su plan de prueba usando la GUI
4. Ejecutar pruebas y analizar resultados
