# MultiTalk FAQ

Frequently asked questions about MultiTalk.

## General Questions

### What is MultiTalk?

MultiTalk is a desktop application for multi-party voice chat, designed for gamers and communities.

### Is MultiTalk free?

Yes, MultiTalk is free for personal and commercial use.

### What platforms are supported?

- Windows 10/11
- macOS 10.15+
- Linux (Ubuntu, Fedora, Debian)

## Technical Questions

### How do I fix echo?

**Solutions:**
- Use headphones instead of speakers
- Lower your speaker volume
- Move microphone further from speakers
- Enable echo cancellation in settings

### Why is my voice not being transmitted?

**Solutions:**
- Check if push-to-talk is enabled
- Verify microphone is selected in settings
- Check if you're muted
- Test microphone in system settings

### How to reduce latency?

**Solutions:**
- Use wired internet connection
- Select closest server region
- Close bandwidth-intensive applications
- Update MultiTalk to latest version

## Account Questions

### How do I create an account?

1. Open MultiTalk
2. Click "Register"
3. Enter email and password
4. Verify email address

### How do I change my username?

1. Go to Settings > Account
2. Click on username
3. Enter new username
4. Save changes

## Room Questions

### How many people can join a room?

Free version: Up to 25 participants
Pro version: Up to 100 participants

### Can I password protect my room?

Yes, you can set a password when creating a room.

### How do I transfer room ownership?

1. Right-click on participant
2. Select "Transfer Ownership"
3. Confirm transfer

## Troubleshooting

### Connection errors

**Solution:**
- Check internet connection
- Try different server region
- Restart application
- Check firewall settings

### Audio quality issues

**Solution:**
- Update audio drivers
- Use better microphone
- Adjust noise suppression settings
- Check system audio processing
