# Element UI Guide d'utilisation

Ce guide couvre comment utiliser les composants Element UI dans votre application Vue 2.

## Utilisation de Base

### Hello World

```vue
<template>
  <div>
    <el-button>Bouton par défaut</el-button>
    <el-button type="primary">Bouton primaire</el-button>
  </div>
</template>

<script>
export default {
  name: 'App'
}
</script>
```

## Composants Courants

### Bouton

```vue
<template>
  <div>
    <!-- Types -->
    <el-button>Par défaut</el-button>
    <el-button type="primary">Primaire</el-button>
    <el-button type="success">Succès</el-button>
    <el-button type="warning">Avertissement</el-button>
    <el-button type="danger">Danger</el-button>
    
    <!-- Simple -->
    <el-button plain>Simple</el-button>
    
    <!-- Arrondi -->
    <el-button round>Arrondi</el-button>
    
    <!-- Cercle -->
    <el-button circle icon="el-icon-search"></el-button>
    
    <!-- Désactivé -->
    <el-button disabled>Désactivé</el-button>
    
    <!-- Chargement -->
    <el-button loading>Chargement</el-button>
  </div>
</template>
```

### Entrée

```vue
<template>
  <div>
    <!-- Entrée basique -->
    <el-input v-model="input" placeholder="Entrez le texte"></el-input>
    
    <!-- Avec icône -->
    <el-input
      v-model="input1"
      placeholder="Rechercher"
      prefix-icon="el-icon-search">
    </el-input>
    
    <!-- Mot de passe -->
    <el-input
      v-model="input2"
      placeholder="Mot de passe"
      show-password>
    </el-input>
    
    <!-- Zone de texte -->
    <el-input
      v-model="textarea"
      type="textarea"
      :rows="3"
      placeholder="Entrez la description">
    </el-input>
  </div>
</template>

<script>
export default {
  data() {
    return {
      input: '',
      input1: '',
      input2: '',
      textarea: ''
    }
  }
}
</script>
```

### Tableau

```vue
<template>
  <el-table :data="tableData" stripe border>
    <el-table-column prop="date" label="Date" width="180"></el-table-column>
    <el-table-column prop="name" label="Nom" width="180"></el-table-column>
    <el-table-column prop="address" label="Adresse"></el-table-column>
    <el-table-column label="Opérations">
      <template slot-scope="scope">
        <el-button size="mini" @click="handleEdit(scope.$index, scope.row)">Modifier</el-button>
        <el-button size="mini" type="danger" @click="handleDelete(scope.$index, scope.row)">Supprimer</el-button>
      </template>
    </el-table-column>
  </el-table>
</template>

<script>
export default {
  data() {
    return {
      tableData: [{
        date: '2024-01-01',
        name: 'Jean',
        address: 'Paris'
      }]
    }
  },
  methods: {
    handleEdit(index, row) {
      console.log(index, row);
    },
    handleDelete(index, row) {
      console.log(index, row);
    }
  }
}
</script>
```

### Formulaire

```vue
<template>
  <el-form
    :model="ruleForm"
    :rules="rules"
    ref="ruleForm"
    label-width="120px"
  >
    <el-form-item label="Nom" prop="name">
      <el-input v-model="ruleForm.name"></el-input>
    </el-form-item>
    
    <el-form-item label="Email" prop="email">
      <el-input v-model="ruleForm.email"></el-input>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm('ruleForm')">Soumettre</el-button>
      <el-button @click="resetForm('ruleForm')">Réinitialiser</el-button>
    </el-form-item>
  </el-form>
</template>

<script>
export default {
  data() {
    return {
      ruleForm: {
        name: '',
        email: ''
      },
      rules: {
        name: [
          { required: true, message: 'Veuillez entrer le nom', trigger: 'blur' }
        ],
        email: [
          { required: true, message: 'Veuillez entrer lemail', trigger: 'blur' },
          { type: 'email', message: 'Format email invalide', trigger: 'blur' }
        ]
      }
    };
  },
  methods: {
    submitForm(formName) {
      this.$refs[formName].validate((valid) => {
        if (valid) {
          alert('Soumis!');
        }
      });
    },
    resetForm(formName) {
      this.$refs[formName].resetFields();
    }
  }
}
</script>
```

### Dialogue

```vue
<template>
  <div>
    <el-button type="text" @click="dialogVisible = true">Ouvrir le dialogue</el-button>
    
    <el-dialog
      title="Titre du dialogue"
      :visible.sync="dialogVisible"
      width="30%">
      <span>Contenu du dialogue</span>
      <span slot="footer" class="dialog-footer">
        <el-button @click="dialogVisible = false">Annuler</el-button>
        <el-button type="primary" @click="dialogVisible = false">Confirmer</el-button>
      </span>
    </el-dialog>
  </div>
</template>

<script>
export default {
  data() {
    return {
      dialogVisible: false
    }
  }
}
</script>
```

```

  }

</script>
```
