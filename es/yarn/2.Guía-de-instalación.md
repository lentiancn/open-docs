# Guía de Instalación de Yarn

## Instalación de Yarn v4+ (Versión Moderna)

### Método Recomendado: Usando Corepack

Corepack es una herramienta intermedia que te permitirá configurar la versión de tu gestor de paquetes por proyecto.

```bash
# 1. Instalar Corepack
npm install -g corepack

# 2. Inicializar un nuevo proyecto
yarn init -2
```

### Actualizando Yarn

Cada vez que desees actualizar Yarn a la última versión, simplemente ejecuta:

```bash
yarn set version stable
yarn install
```

Yarn configurará tu proyecto para usar el binario estable más reciente.

### Usando Versiones Canary

Yarn también lanza frecuentemente versiones candidatas a lanzamiento (Release Candidate). Usa el siguiente comando si necesitas una función que aún no se ha lanzado en el canal estable:

```bash
yarn set version canary
```

Estas versiones son muy estables, la única diferencia con el canal regular es una migración más escalonada entre versiones mayores a medida que implementamos nuevos cambios rotos.

### Instalando la Última Versión desde Master

Es posible que desees probar una versión de Yarn tan reciente que aún no se ha lanzado en una versión candidata, o incluso no se ha fusionado. El siguiente comando clonará, construirá e instalará Yarn en tu proyecto directamente desde nuestro repositorio:

```bash
yarn set version from sources
```

Acepta una bandera `--branch` que puedes usar para probar PRs específicas:

```bash
yarn set version from sources --branch 1211
```

> Advertencia: A diferencia de los canales estable y canary, el comando `yarn set version from sources` no puede aprovechar Corepack y necesitará almacenar el binario de Yarn dentro de la carpeta `.yarn/releases` y hacer referencia a él desde el archivo `.yarnrc.yml` de tu proyecto.

## Instalación de Yarn v1 (Classic)

### Mediante npm (Recomendado)

Se recomienda instalar Yarn a través del [gestor de paquetes npm](http://npmjs.org/), que viene incluido con [Node.js](https://nodejs.org/) cuando lo instalas en tu sistema.

Una vez que tengas npm instalado, puedes ejecutar lo siguiente tanto para instalar como para actualizar Yarn:

```bash
npm install --global yarn
```

### Verificar Instalación

Verifica que Yarn esté instalado ejecutando:

```bash
yarn --version
```

## Recomendaciones de Selección de Versión

- **Nuevos Proyectos**: Se recomienda usar Yarn v4+ para disfrutar de funciones modernas y mejor rendimiento
- **Proyectos Existentes**: Si ya estás usando Yarn v1, puedes continuar usándolo o considerar migrar a la versión moderna
- **Entornos Empresariales**: Yarn v1 proporciona soporte a largo plazo más estable, mientras que v4+ ofrece funciones más innovadoras
