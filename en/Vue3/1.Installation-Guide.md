## What is Vue 3?

Vue 3 is the latest version of the progressive JavaScript framework for building user interfaces. It features improved performance, Composition API, TypeScript support, and better TypeScript integration.

## CDN

### Basic CDN Usage

```html
<!-- Development -->
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<!-- Production -->
<script src="https://unpkg.com/vue@3/dist/vue.global.prod.js"></script>
```

### CDN with Options API

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Vue 3 App</title>
</head>
<body>
  <div id="app">
    {{ message }}
  </div>

  <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
  <script>
    const { createApp } = Vue

    createApp({
      data() {
        return {
          message: 'Hello Vue 3!'
        }
      }
    }).mount('#app')
  </script>
</body>
</html>
```

## NPM

### Install Vue 3

```bash
# Install Vue 3
npm install vue@3

# Install specific version
npm install vue@3.4.0
```

## Vite

### Create New Project (Recommended)

```bash
# Create project with Vite
npm create vite@latest my-vue3-app -- --template vue

# Or
npm create vue@latest my-vue3-app

# Navigate to project
cd my-vue3-app

# Install dependencies
npm install

# Start development server
npm run dev
```

### Vite Options

```bash
# With TypeScript
npm create vite@latest my-app -- --template vue-ts

# With Vue Router
npm install vue-router@4

# With Pinia (state management)
npm install pinia

# With VueUse
npm install @vueuse/core
```

## Vue CLI

### Install Vue CLI

```bash
# Install Vue CLI globally
npm install -g @vue/cli

# Upgrade Vue CLI
npm install -g @vue/cli@latest
```

### Create New Project

```bash
# Create project
vue create my-vue3-app

# Select Vue 3 when prompted
```

### Add Vue 3 to Existing Project

```bash
# Upgrade to Vue 3
vue add @vue/cli-plugin-vue-next
```

## Project Setup

### Basic Project Structure

```
my-vue3-app/
├── public/
│   └── index.html
├── src/
│   ├── assets/
│   ├── components/
│   │   └── HelloWorld.vue
│   ├── App.vue
│   └── main.js
├── index.html
├── package.json
└── vite.config.js
```

### main.js

```javascript
import { createApp } from 'vue'
import App from './App.vue'

createApp(App).mount('#app')
```

### App.vue

```vue
<script setup>
import { ref } from 'vue'
import HelloWorld from './components/HelloWorld.vue'

const msg = ref('Hello Vue 3!')
</script>

<template>
  <HelloWorld :msg="msg" />
</template>
```

## TypeScript Support

### Install TypeScript

```bash
# Install TypeScript
npm install -D typescript vue-tsc

# Create tsconfig.json
npx tsc --init
```

### tsconfig.json

```json
{
  "compilerOptions": {
    "target": "ESNext",
    "module": "ESNext",
    "moduleResolution": "Node",
    "strict": true,
    "jsx": "preserve",
    "resolveJsonModule": true,
    "isolatedModules": true,
    "esModuleInterop": true,
    "lib": ["ESNext", "DOM"],
    "skipLibCheck": true,
    "noEmit": true
  },
  "include": ["src/**/*.ts", "src/**/*.d.ts", "src/**/*.tsx", "src/**/*.vue"],
  "references": [{ "path": "./tsconfig.node.json" }]
}
```

### Vue with TypeScript

```vue
<script setup lang="ts">
import { ref, onMounted } from 'vue'

interface User {
  name: string
  age: number
}

const user = ref<User>({
  name: 'John',
  age: 30
})

onMounted(() => {
  console.log('User:', user.value)
})
</script>

<template>
  <div>{{ user.name }}</div>
</template>
```
