# FantasyTalking Manual del Usuario

## Funciones Básicas

### Síntesis de Voz

La síntesis de voz es la función principal de FantasyTalking, convirtiendo texto en voz natural y fluida:

```javascript
// Texto simple a voz
const audio = await client.speak('Hola, bienvenido a FantasyTalking');

// Especificar estilo de voz
const audio = await client.speak('El clima está很好 hoy', {
  voice: {
    style: 'happy',
    language: 'es-ES'
  }
});
```

## Funciones Avanzadas

### Síntesis de Voz en Streaming

Para escenarios que requieren retroalimentación en tiempo real:

```javascript
const stream = await client.speakStream('Este es un texto largo');

stream.on('data', (chunk) => {
  audioPlayer.play(chunk);
});

stream.on('end', () => {
  console.log('Síntesis completada');
});
```

## Mejores Prácticas

### Optimización de Rendimiento

1. **Reutilización de conexiones**: Evita crear instancias de cliente frecuentemente
2. **Procesamiento por lotes**: Usa síntesis por lotes para grandes cantidades de texto
3. **Caché**: Usa caché para evitar síntesis duplicada
