# FantasyTalking Руководство по установке

## Требования

Перед установкой FantasyTalking убедитесь, что среда соответствует требованиям. Для ОС проект поддерживает Windows, macOS и Linux. Рекомендуется 64-битная система. Требуется Node.js 14.0+ или Python 3.7+.

Для оборудования рекомендуется минимум 4 ГБ ОЗУ и двухъядерный процессор.

## Установка

### Использование npm (Node.js)

```bash
npm install fantasy-talking
npm install fantasy-talking-ui
npm install fantasy-talking-node
```

### Использование pip (Python)

```bash
pip install fantasy-talking
pip install fantasy-talking[all]
```

### Сборка из исходного кода

```bash
git clone https://github.com/fantasytalking/fantasy-talking.git
cd fantasy-talking
npm install
pip install -r requirements.txt
npm run build
```

## Быстрый старт

```javascript
const client = new FantasyTalking({
  apiKey: 'your-api-key',
  voice: { style: 'neutral', language: 'ru-RU' }
});

const audio = await client.speak('Привет, это тест');
```

## Параметры конфигурации

| Параметр | Тип | По умолчанию | Описание |
|----------|------|--------------|----------|
| apiKey | string | - | API-ключ (обязательно) |
| apiUrl | string | Default | Адрес сервера API |
| timeout | number | 30000 | Таймаут (мс) |
| voice.style | string | neutral | Стиль голоса |
| voice.language | string | ru-RU | Код языка |
| voice.speed | number | 1.0 | Скорость |

## Развертывание

### Docker

```bash
docker pull fantasytalking/server:latest
docker run -d -p 8080:8080 -e API_KEY=your-api-key fantasytalking/server:latest
```

## Проверка установки

```bash
fantasy-talking --text "Тест" --output test.wav
```
