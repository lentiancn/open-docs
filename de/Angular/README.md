# Angular

Moderne Web-Entwicklungsplattform.

## Übersicht

Angular ist eine Entwicklungsplattform zum Erstellen von mobilen und Desktop-Webanwendungen, entwickelt von Google. Verwendet TypeScript/JavaScript mit komponentenbasierter Architektur, bidirektionaler Bindung und Abhängigkeitsinjektion.

### Hauptfunktionen

- **Komponentenbasiert** - Komponentengetriebene Architektur
- **TypeScript** - Starke Typisierungsunterstützung
- **Bidirektionale Bindung** - Vereinfachte Datensynchronisation
- **Abhängigkeitsinjektion** - Leistungsstarkes Abhängigkeitsmanagement
- **Routing** - Vollständige Routing-Lösung

### Statistiken

- GitHub Sterne: 90.000+
- Millionen Entwickler weltweit

## Schnellstart

### Installieren

```bash
npm install -g @angular/cli
```

### Projekt erstellen

```bash
ng new my-app
cd my-app
ng serve
```

Besuchen: http://localhost:4200
