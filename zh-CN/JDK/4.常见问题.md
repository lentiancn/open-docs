# JDK 常见问题

## 安装问题

### Q: java 命令找不到？
**A**: 
- 确认 JAVA_HOME 和 PATH 配置正确
- 重新打开终端

### Q: 多个 JDK 版本如何切换？
**A**: 
- Windows：修改 JAVA_HOME 环境变量
- macOS：使用 jenv 管理多版本

## 使用问题

### Q: 如何查看 JDK 路径？
**A**: 
```bash
which java
echo $JAVA_HOME
```

### Q: 内存不足怎么办？
**A**: 
```bash
java -Xmx4g -Xms2g Hello
```

## 版本问题

### Q: JDK 和 JRE 区别？
**A**: 
- JDK = JRE + 开发工具
- JRE = JVM + 类库

### Q: 哪个版本最适合生产环境？
**A**: 推荐 LTS 版本：JDK 21、JDK 17、JDK 11
