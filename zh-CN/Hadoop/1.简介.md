# Hadoop - 简介

Apache Hadoop 是开源的大数据处理框架，提供分布式存储（HDFS）和分布式计算（MapReduce）能力。

## 核心组件

- **HDFS**：Hadoop Distributed File System，分布式文件系统
- **MapReduce**：分布式计算框架
- **YARN**：资源调度器
- **Common**：基础工具库

## 特性

- **分布式存储**：PB 级数据存储
- **高容错**：数据自动复制
- **高吞吐量**：适合大数据处理
- **横向扩展**：增加节点即可扩展
- **成本低**：普通硬件即可运行

## 生态系统

- **Hive**：数据仓库
- **HBase**：NoSQL 数据库
- **Spark**：内存计算引擎
- **Flink**：流处理引擎
- **Kafka**：消息队列
- **ZooKeeper**：协调服务
