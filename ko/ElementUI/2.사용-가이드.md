# Element UI 사용 가이드

Element UI를 사용하는 가이드입니다.

---

## 버튼

```vue
<el-button>버튼</el-button>
<el-button type="primary">기본</el-button>
```

---

## 입력

```vue
<el-input v-model="input"></el-input>
```

---

## 테이블

```vue
<el-table :data="tableData">
  <el-table-column prop="name" label="이름"></el-table-column>
</el-table>
```

---

## 아이콘

```vue
<i class="el-icon-edit"></i>
```
