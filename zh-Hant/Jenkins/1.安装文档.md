# Jenkins 安裝指南

Jenkins 是一個開源自動化伺服器，用於實現持續整合和持續部署（CI/CD）。本指南涵蓋在各種環境中的安裝方法。

## 版本資訊

| 版本 | 狀態 | 說明 |
|------|------|------|
| Jenkins 2.x | ✅ 當前 | 最新穩定版本 |
| Jenkins 1.x | ❌ 舊版 | 已停止維護 |

## 系統需求

### 硬體需求

| 需求 | 最低配置 | 推薦配置 |
|------|----------|----------|
| CPU | 1 核心 | 2+ 核心 |
| 記憶體 | 1 GB | 4+ GB |
| 磁碟 | 10 GB | 50+ GB |

### 軟體需求

| 軟體 | 版本 |
|------|------|
| Java | JDK 8 或更高 |
| 作業系統 | Windows/macOS/Linux |

## 安裝方法

### 方法一：Windows 安裝

#### 1. 使用安裝程式

1. 下載 Jenkins：https://www.jenkins.io/download/
2. 執行 jenkins.msi 安裝程式
3. 按照安裝精靈完成安裝

#### 2. 使用 Chocolatey

```powershell
choco install jenkins
```

#### 3. 手動安裝

```powershell
# 下載 WAR 檔案
# 解壓到指定目錄
java -jar jenkins.war
```

### 方法二：macOS 安裝

#### 1. 使用 Homebrew

```bash
# 安裝 Jenkins
brew install jenkins

# 啟動服務
brew services start jenkins

# 停止服務
brew services stop jenkins
```

#### 2. 手動安裝

```bash
# 下載 WAR 檔案
wget https://get.jenkins.io/jenkins.war

# 執行
java -jar jenkins.war
```

### 方法三：Linux 安裝

#### Ubuntu/Debian

```bash
# 新增金鑰
wget -q -O - https://pkg.jenkins.io/debian/jenkins.io.key | sudo apt-key add -

# 新增倉庫
sudo sh -c 'echo deb https://pkg.jenkins.io/debian binary/ > /etc/apt/sources.list.d/jenkins.list'

# 更新並安裝
sudo apt-get update
sudo apt-get install jenkins

# 啟動服務
sudo systemctl start jenkins
sudo systemctl enable jenkins
```

#### CentOS/RHEL

```bash
# 新增倉庫
sudo wget -O /etc/yum.repos.d/jenkins.repo https://pkg.jenkins.io/redhat/jenkins.repo

# 匯入金鑰
sudo rpm --import https://pkg.jenkins.io/redhat/jenkins.io.key

# 安裝
sudo yum install jenkins

# 啟動服務
sudo systemctl start jenkins
sudo systemctl enable jenkins
```

### 方法四：Docker 安裝

```bash
# 執行 Jenkins 容器
docker run -d \
  --name jenkins \
  -p 8080:8080 \
  -v jenkins_home:/var/jenkins_home \
  jenkins/jenkins:lts
```

#### Docker Compose

```yaml
version: '3'
services:
  jenkins:
    image: jenkins/jenkins:lts
    ports:
      - "8080:8080"
    volumes:
      - jenkins_home:/var/jenkins_home
```

### 方法五：WAR 檔案部署

```bash
# 下載 WAR 檔案
wget https://get.jenkins.io/war-stable/latest/jenkins.war

# 執行
java -jar jenkins.war --httpPort=8080
```

## 安裝後配置

### 首次訪問

1. 打開瀏覽器存取 http://localhost:8080
2. 查看初始管理員密碼：
   ```bash
   # Linux/macOS
   sudo cat /var/lib/jenkins/secrets/initialAdminPassword
   
   # Docker
   docker exec jenkins cat /var/jenkins_home/secrets/initialAdminPassword
   ```
3. 安裝推薦外掛程式
4. 建立管理員使用者

### 配置中文介面

1. 進入 Manage Jenkins → Manage Plugins
2. 安裝 "Localization: Chinese (Traditional)"
3. 重新啟動 Jenkins

## 常見問題

### 啟動失敗？

檢查 Java 是否安裝：

```bash
java -version
```

### 連接埠被佔用？

修改連接埠：

```bash
java -jar jenkins.war --httpPort=8090
```

### 外掛程式安裝失敗？

檢查網路連線或使用 Proxy。
