# MultiTalk 설치 가이드

MultiTalk 설치 과정에는 Python 환경 구성과 모델 가중치 다운로드가 필요합니다. 자세한 설치 지침은 다음과 같습니다.

## 환경 요구 사항

### 하드웨어 요구 사항

MultiTalk는 하드웨어 요구 사항이 높은大型 비디오 생성 모델입니다. NVIDIA GPU 사용을 권장합니다. VRAM 요구 사항은 출력 해상도에 따라 다릅니다. 480P 영상 생성에는 최소 16GB의 VRAM이 필요하며, 720P 영상 생성에는 다중 GPU 구성이 필요합니다. 프로젝트는 특정 최적화를 통해 단일 RTX 4090에서 480P 영상 생성을 가능하게 하는 저 VRAM 환경에서의 실행도 지원합니다.

### 소프트웨어 요구 사항

Linux 운영 체제가 지원됩니다. Python 버전 3.10 이상이 필요합니다. CUDA 12.1 이상 지원도 필요합니다. 프로젝트는 PyTorch 2.4.1, xformers와 같은 딥러닝 프레임워크에 의존합니다.

## 가상 환경 만들기

먼저 프로젝트 종속성을 격리하기 위해 새 Python 가상 환경을 만듭니다. 터미널을 열고 다음 명령을 실행합니다:

```bash
conda create -n multitalk python=3.10
conda activate multitalk
```

## PyTorch 및 xformers 설치

활성화된 환경에서 PyTorch 및 관련 패키지를 설치합니다. PyTorch 2.4.1을 설치하려면 다음 명령을 실행합니다:

```bash
pip install torch==2.4.1 torchvision==0.19.1 torchaudio==2.4.1 --index-url https://download.pytorch.org/whl/cu121
```

그런 다음 xformers를 설치합니다:

```bash
pip install -U xformers==0.0.28 --index-url https://download.pytorch.org/whl/cu121
```

## 프로젝트 종속성 설치

다른 필요한 종속성 패키지를 설치합니다:

```bash
pip install misaki[en]
pip install ninja
pip install psutil
pip install packaging
pip install flash_attn==2.7.4.post1
```

그런 다음 프로젝트 저장소를克隆하고 프로젝트 종속성을 설치합니다:

```bash
git clone https://github.com/MeiGen-AI/MultiTalk.git
cd MultiTalk
pip install -r requirements.txt
```

## 오디오 처리 라이브러리 설치

프로젝트에는 오디오 처리 관련 라이브러리가 필요합니다. 설치하려면 다음 명령을 실행합니다:

```bash
conda install -c conda-forge librosa
conda install -c conda-forge ffmpeg
```

CentOS 또는 RHEL 시스템의 경우 다음을 사용합니다:

```bash
sudo yum install ffmpeg ffmpeg-devel
```

## 모델 가중치 다운로드

MultiTalk가 올바르게 작동하려면 여러 모델 파일을 다운로드해야 합니다. 주요 모델은 다음과 같습니다:

Wan2.1-I2V-14B-480P는 Hugging Face에서 다운로드할 수 있는 기본 확산 모델입니다. chinese-wav2vec2-base는 입력 오디오를 처리하기 위한 오디오 인코더입니다. Kokoro-82M는 텍스트 음성 변환 기능용 TTS 가중치입니다. MeiGen-MultiTalk는 프로젝트에서 제공하는 오디오 조건 가중치입니다.

huggingface-cli를 사용하여 모델을 다운로드합니다:

```bash
huggingface-cli download Wan-AI/Wan2.1-I2V-14B-480P --local-dir ./weights/Wan2.1-I2V-14B-480P
huggingface-cli download TencentGameMate/chinese-wav2vec2-base --local-dir ./weights/chinese-wav2vec2-base
huggingface-cli download hexgrad/Kokoro-82M --local-dir ./weights/Kokoro-82M
huggingface-cli download MeiGen-AI/MeiGen-MultiTalk --local-dir ./weights/MeiGen-MultiTalk
```

## 모델 링크 구성

모델을 다운로드한 후 링크를 구성해야 합니다. 두 가지 옵션이 있습니다.

옵션 하나는 심볼릭 링크를 사용하는 것입니다. weights 디렉토리로 이동하여 다음을 실행합니다:

```bash
cd weights
mv Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json_old
ln -s /path/to/weights/MeiGen-MultiTalk/diffusion_pytorch_model.safetensors.index.json Wan2.1-I2V-14B-480P/
ln -s /path/to/weights/MeiGen-MultiTalk/multitalk.safetensors Wan2.1-I2V-14B-480P/
```

옵션 둘은 파일을 직접 복사하는 것입니다:

```bash
cd weights
mv Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json_old
cp MeiGen-MultiTalk/diffusion_pytorch_model.safetensors_index.json Wan2.1-I2V-14B-480P/
cp MeiGen-MultiTalk/multitalk.safetensors Wan2.1-I2V-14B-480P/
```

## 설치 확인

설치가 완료되면 예제 스크립트를 실행하여 설치를 확인할 수 있습니다. 예제 파일은 examples 디렉토리에 있습니다.
