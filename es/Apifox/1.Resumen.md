# Apifox Resumen

Plataforma de colaboración API todo en uno.

## ¿Qué es Apifox?

Apifox es una plataforma de colaboración API que integra documentación, depuración, Mock y pruebas automatizadas.

### Características

- **Documentación API** - Generación automática
- **Depuración API** - Simulador HTTP
- **Servicios Mock** - Crear servidores Mock
- **Pruebas automatizadas** - Pruebas API
- **Colaboración** - Gestión de proyecto

### Estadísticas

- Usuarios: Millones
