# Claude Code

[Claude Code](https://code.claude.com/) — интеллектуальный помощник для программирования от Anthropic.

## Обзор

Claude Code — помощник программирования на основе ИИ, доступный в терминале, IDE, десктопном приложении и браузере.

## Функции

- **Автоматизация**: Тесты, исправление ошибок, конфликты слияния
- **Разработка**: Опишите需求; Claude Code реализует
- **Интеграция Git**: Коммиты, ветки, Pull Requests
- **Интеграция MCP**: Подключение к внешним инструментам
- **Настройка**: CLAUDE.md, пользовательские команды

## Быстрый Старт

### Установка

**macOS, Linux, WSL:**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell:**
```powershell
irm https://claude.ai/install.ps1 | iex
```

### Запуск

```bash
cd your-project
claude
```
