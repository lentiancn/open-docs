## What is Element UI?

Element UI is a Vue.js 2.0 based component library for developers, designers and product managers. It provides high quality UI components for building responsive websites.

### Note on Element Plus

**Element UI is for Vue 2.** For Vue 3, use **Element Plus** instead.

## CDN

### Quick Start (CDN)

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Element UI CDN</title>
  <link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
</head>
<body>
  <div id="app">
    <el-button>Button</el-button>
  </div>

  <script src="https://unpkg.com/vue@2"></script>
  <script src="https://unpkg.com/element-ui/lib/index.js"></script>
  <script>
    new Vue({
      el: '#app'
    })
  </script>
</body>
</html>
```

### CDN Providers

| Provider | CSS | JS |
|----------|-----|-----|
| unpkg | https://unpkg.com/element-ui/lib/theme-chalk/index.css | https://unpkg.com/element-ui/lib/index.js |
| jsDelivr | https://cdn.jsdelivr.net/npm/element-ui/lib/theme-chalk/index.css | https://cdn.jsdelivr.net/npm/element-ui/lib/index.js |

## NPM

### Install Element UI

```bash
# Install Element UI for Vue 2
npm install element-ui

# For Vue 3, use Element Plus instead
npm install element-plus
```

### Install Specific Version

```bash
npm install element-ui@2.15.13
```

---

## Vue CLI

### With Vue CLI 3/4

```bash
# Create Vue project
vue create my-element-app

# Add Element UI
cd my-element-app
npm install element-ui
```

### Manual Setup

```bash
npm install element-ui --save
```

## Vite

### With Vite + Vue 2

```bash
npm create vite@latest my-element-app -- --template vue
cd my-element-app
npm install element-ui
npm install @vitejs/plugin-vue2
```

### vite.config.js

```javascript
import { defineConfig } from 'vite'
import vue2 from '@vitejs/plugin-vue2'

export default defineConfig({
  plugins: [vue2()],
  optimizeDeps: {
    include: ['element-ui']
  }
})
```

## Basic Setup

### main.js

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(ElementUI)

new Vue({
  render: h => h(App)
}).$mount('#app')
```

### App.vue

```vue
<template>
  <div id="app">
    <el-button>Hello Element</el-button>
    <el-button type="primary">Primary</el-button>
  </div>
</template>

<script>
export default {
  name: 'App'
}
</script>
```
