# Руководство пользователя

## Основные понятия

### Gateway

Ядро системы. Управляет сообщениями, запросами к ИИ и подключениями каналов.

### Каналы

Чат-приложения: WhatsApp, Telegram, Discord и т.д.

### Агенты

Независимые экземпляры ИИ. Можно запустить несколько для разных целей.

## Команды

```bash
# Запуск/остановка
openclaw gateway start
openclaw gateway stop
openclaw gateway status

# Отправить сообщение
openclaw message send --target +123456789 --message "Привет"
```

## Настройка

Файл: `~/.openclaw/openclaw.json`

Способы изменения:
- `openclaw configure`
- `openclaw config set xxx`
- Веб-интерфейс: http://127.0.0.1:18789/
- Редактирование файла напрямую

## Примеры настроек

### Ограничение доступа

```json5
{
  channels: {
    whatsapp: {
      allowFrom: ["+123456789"]
    }
  }
}
```

### Несколько агентов

```json5
{
  agents: {
    list: [
      { id: "home", default: true },
      { id: "work" }
    ]
  }
}
```

## Решение проблем

```bash
openclaw logs
openclaw logs -f
openclaw doctor
```
