# インストールガイド

このドキュメントでは、Angularのインストールと環境設定について説明します。スムーズに開発環境を構築するために役立ちます。

## 環境要件

Angularをインストールする前に、以下の環境要件を満たしているか確認してください：

- **Node.js**：推奨バージョン 18.10.0 以上
- **npm**：Node.jsに付属しています、バージョン 6.x 以上
- **オペレーティングシステム**：Windows、macOS、Linux

## Angular CLIのインストール

Angular CLIはAngular公式のコマンドラインツールで、Angularプロジェクトの作成や管理に使います。

###グローバルインストール

```bash
npm install -g @angular/cli
```

### インストール確認

```bash
ng version
```

インストールに成功하면、Angular CLIのバージョン情報が表示されます。

### 特定のバージョンをインストール

特定のバージョンをインストールする必要がある場合は：

```bash
npm install -g @angular/cli@17
```

## 新規プロジェクトの作成

### CLIを使って作成

```bash
ng new my-app
```

これで「my-app」という名前の新しいプロジェクトが作成され、依存関係も自動的にインストールされます。

### オプションを指定して作成

```bash
# インストールをスキップして作成
ng new my-app --skip-install

# SCSSを使用
ng new my-app --style=scss

# ルーティングを有効化
ng new my-app --routing

# スタンドアロンコンポーネントを使用（Angular 14以降）
ng new my-app --standalone
```

## 開発環境の設定

### 国内ミラーの使用

ネットワークの都合で、国内ミラーを設定することをお勧めします：

```bash
# npmレジストリを設定
npm config set registry https://registry.npmmirror.com

# またはnrmツールを使ってミラーを管理
npm install -g nrm
nrm use taobao
```

### Node.jsのインストール

**nvmを使ってインストール（推奨）：**

```bash
# nvmをインストール
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Node.js LTSバージョンをインストール
nvm install --lts
nvm use --lts
```

**Windowsではnvm-windowsを使用：**

https://github.com/coreybutler/nvm-windows からインストーラーをダウンロードしてください。

## 開発サーバーを起動

### プロジェクトディレクトリに入る

```bash
cd my-app
```

### 開発サーバーを起動

```bash
ng serve
```

デフォルトのアクセス先：http://localhost:4200

### ライブリロード

Angular CLIはライブリロードをサポートしているので、コードを修正するとブラウザが自動的に更新されます。

## 本番用ビルド

### ビルドコマンド

```bash
ng build```

ビルド結果はdistディレクトリに出力されます。

### 本番環境用に最適化

```bash
ng build --configuration=production
```

これで圧縮、tree-shakingなどの最適化が適用されます。

## よく使うCLIコマンド

| コマンド | 説明 |
|------|------|
| ng generate component xxx | コンポーネントを生成 |
| ng generate service xxx | サービスを生成 |
| ng generate module xxx | モジュールを生成 |
| ng generate directive xxx | Directiveを生成 |
| ng generate pipe xxx |  pipeを生成 |
| ng generate guard xxx | 路由ガードを生成 |

## 開発ツール

### VS Code拡張

- Angular Language Service：テンプレートの型チェックと自動補完
- Angular Snippets：よく使うコードスニペット
- Prettier：コード整形

### デバッグ

Angular DevToolsブラウザ拡張を使えば、デバッグやパフォーマンス分析ができます。

## Angular CLIのアンインストール

もしアンインストールする必要がある場合は：

```bash
npm uninstall -g @angular/cli
```
