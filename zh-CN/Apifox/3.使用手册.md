# Apifox 使用手册

本手册介绍 Apifox 的核心功能和使用方法。

## 快速开始

### 创建项目

1. 点击新建项目
2. 输入项目名称
3. 选择团队或私有

### 创建接口

1. 选择接口分类
2. 点击新建接口
3. 填写接口信息

### 发送请求

1. 选择请求方法（GET/POST/PUT/DELETE）
2. 输入请求 URL
3. 设置请求头
4. 设置请求体
5. 点击发送

## 接口调试

### 请求设置

```http
### 获取列表
GET https://api.example.com/users
Content-Type: application/json
Authorization: Bearer token

### 创建数据
POST https://api.example.com/users
Content-Type: application/json

{
  "name": "Tom",
  "email": "tom@example.com"
}
```

### 断言

```javascript
// 断言状态码
pm.test("状态码为200", function() {
    pm.response.to.have.status(200);
});

// 断言响应体
pm.test("返回正确数据", function() {
    var jsonData = pm.response.json();
    pm.expect(jsonData.code).to.eql(0);
});

// 断言响应时间
pm.test("响应时间", function() {
    pm.expect(pm.response.responseTime).to.be.below(200);
});
```

## Mock 服务

### 内置 Mock

```javascript
// 自动生成 Mock 数据
{
  "id": "@integer",
  "name": "@cname",
  "email": "@email",
  "avatar": "@image"
}
```

### 自定义 Mock

1. 在响应中设置 Mock
2. 使用 Mock 规则
3. 测试 Mock 效果

## 自动化测试

### 创建测试用例

1. 打开接口
2. 点击测试
3. 编写测试脚本

### 测试脚本

```javascript
// 设置变量
pm.collectionVariables.set("token", "test-token");

// 获取响应数据
var jsonData = pm.response.json();
pm.globals.set("userId", jsonData.data.id);

// 断言
pm.test("成功", function() {
    pm.expect(pm.response.json().code).to.eql(0);
});
```

### 运行测试

1. 选择测试集合
2. 点击运行
3. 查看报告

## 团队协作

### 邀请成员

1. 进入项目设置
2. 点击成员管理
3. 邀请成员

### 权限设置
