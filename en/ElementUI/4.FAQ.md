# Element UI FAQ

> Frequently Asked Questions

---

## Installation

### npm install fails?

**Solution**:
1. Clear npm cache: `npm cache clean --force`
2. Check network or use mirror

### Styles not working?

**Solution**:
1. Ensure CSS is imported: `import 'element-ui/lib/theme-chalk/index.css'`
2. Check webpack config

---

## Component Issues

### Component not displaying?

**Solution**:
1. Check if `Vue.use(ElementUI)` is called
2. Verify Vue version >= 2.6.0

## i18n

### How to use Chinese?

```javascript
import ElementUI from 'element-ui'
import locale from 'element-ui/lib/locale/lang/zh-CN'

Vue.use(ElementUI, { locale })
```

---

## Migration

### How to migrate to Element Plus?

1. Upgrade Vue 2 → Vue 3
2. Upgrade Element UI → Element Plus
3. Update v-model syntax
4. Update component registration

Migration tool:
```bash
npm install -g gogocode
gogocode -s your-project-path -t element-plus -o
```

## Security

### Is Element UI safe to use?

**Note**:
1. Element UI is discontinued, may have security vulnerabilities
2. Vue 2 is EOL, recommend upgrading to Vue 3
3. For new projects, use Element Plus (Vue 3)
