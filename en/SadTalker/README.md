# SadTalker

AI project for generating talking head videos from image and audio.

## Docs

- [Installation](./1.安装文档.md)
- [Usage](./2.使用指南.md)

## Quick Start

```bash
git clone https://github.com/OpenTalker/SadTalker.git
cd SadTalker
pip install -r requirements.txt
python inference.py --source_image img.jpg --driven_audio audio.wav
```

## Requirements

- GPU 8- Python 3GB+
.8+
