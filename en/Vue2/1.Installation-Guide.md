## What is Vue 2?

Vue 2 is a progressive JavaScript framework for building user interfaces. It was released in 2016 and became one of the most popular frontend frameworks.

**Note**: Vue 2 reached end of life on December 31st, 2023. Consider migrating to Vue 3 for new projects.

## CDN

### Basic CDN Usage

```html
<!-- Development -->
<script src="https://cdn.jsdelivr.net/npm/vue@2/dist/vue.js"></script>

<!-- Production -->
<script src="https://cdn.jsdelivr.net/npm/vue@2/dist/vue.min.js"></script>
```

### CDN with Options

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Vue 2 App</title>
</head>
<body>
  <div id="app">
    {{ message }}
  </div>

  <script src="https://cdn.jsdelivr.net/npm/vue@2/dist/vue.js"></script>
  <script>
    new Vue({
      el: '#app',
      data: {
        message: 'Hello Vue 2!'
      }
    })
  </script>
</body>
</html>
```

## NPM

### Install Vue 2

```bash
# Install Vue 2
npm install vue@2

# Install Vue 2 with specific version
npm install vue@2.6.14
```

### Install Vue CLI 4 (for Vue 2)

```bash
# Install Vue CLI
npm install -g @vue/cli-service-global

# Or use Vue CLI 4
npm install -g @vue/cli@4
```

## Vue CLI

### Create New Project

```bash
# Install Vue CLI 4
npm install -g @vue/cli@4

# Create new project
vue create my-vue2-app

# Select Vue 2 when prompted
```

### Manual Setup with Vite

```bash
# Create project with Vite
npm create vite@latest my-vue2-app -- --template vue

# Install Vue 2
cd my-vue2-app
npm install vue@2

# Configure vite.config.js
```

### vite.config.js for Vue 2

```javascript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue2'

export default defineConfig({
  plugins: [vue()],
  resolve: {
    alias: {
      vue: 'vue/dist/vue.esm.js'
    }
  }
})
```

## Vue Loader

### Install Vue Loader

```bash
npm install --save-dev vue-loader@15 vue-template-compiler
```

### Webpack Configuration

```javascript
const VueLoaderPlugin = require('vue-loader/lib/plugin')

module.exports = {
  module: {
    rules: [
      {
        test: /\.vue$/,
        loader: 'vue-loader'
      }
    ]
  },
  plugins: [
    new VueLoaderPlugin()
  ]
}
```

## Project Setup

### Basic Project Structure

```
my-vue2-app/
├── public/
│   └── index.html
├── src/
│   ├── assets/
│   ├── components/
│   │   └── HelloWorld.vue
│   ├── App.vue
│   └── main.js
├── package.json
└── vue.config.js
```

### main.js

```javascript
import Vue from 'vue'
import App from './App.vue'

Vue.config.productionTip = false

new Vue({
  render: h => h(App)
}).$mount('#app')
```

### App.vue

```vue
<template>
  <div id="app">
    <img alt="Vue logo" src="./assets/logo.png">
    <HelloWorld msg="Welcome to Vue 2"/>
  </div>
</template>

<script>
import HelloWorld from './components/HelloWorld.vue'

export default {
  name: 'App',
  components: {
    HelloWorld
  }
}
</script>

<style>
#app {
  font-family: Avenir, Helvetica, Arial, sans-serif;
}
</style>
```

## Version Migration

### From Vue 2 to Vue 3

Vue 2 reached EOL on December 31st, 2023. Consider migrating to Vue 3.

**Migration Tools:**

```bash
# Install migration build
npm install vue@3

# Use Vue Migration Build
npm install @vue/compat
```

### Vue 2 to Vue 3 Changes

| Feature | Vue 2 | Vue 3 |
|---------|-------|-------|
| Lifecycle | `created` | `onMounted` |
| Data | `data()` | `setup()` |
| Components | Options API | Composition API |
| v-model | `v-model` | `v-model:modelValue` |
