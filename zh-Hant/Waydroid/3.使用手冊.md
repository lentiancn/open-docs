# 使用手冊

## Android 應用程式安裝與執行

Waydroid 可以透過 `waydroid app -h` 指令執行多種操作：

```bash
usage: waydroid app [-h] {install,remove,launch,list} ...

optional arguments:
  -h, --help            顯示此說明資訊並退出

子指令:
  {install,remove,launch,list}
    install             將單個軟體包推送到容器並安裝
    remove              從容器中移除單個應用程式軟體包
    launch              啟動單個應用程式
    list                列出已安裝的應用程式
```

### 安裝 Android 應用程式

使用以下指令安裝 Android 應用程式：

```bash
waydroid app install xyz.apk
```

安裝後，您可以從桌面應用程式選單啟動該應用程式。

如果您知道應用程式的 Android 套件名稱（`com.xxx.yyy`），也可以透過指令列啟動：

```bash
waydroid app launch com.foo.bar
```

### 注意事項

在網際網路上找到的 apk 檔案通常只支援 arm 架構，因此在 x86_64 系統上無法正常運作。

您可以安裝 [F-Droid](https://f-droid.org) 來圖形化取得應用程式，或者如果您安裝了 GAPPS 映像，也可以使用 Play 商店。

## 常用指令

### Waydroid 指令列選項

Waydroid 提供了豐富的指令列選項來控制容器和工作階段：

- `waydroid show-full-ui`：顯示完整的 Android UI
- `waydroid session start`：啟動 Waydroid 工作階段
- `waydroid session stop`：停止 Waydroid 工作階段
- `waydroid container start`：啟動 Waydroid 容器
- `waydroid container stop`：停止 Waydroid 容器
- `waydroid logcat`：檢視 Android 日誌
- `waydroid shell`：進入 Android shell
- `waydroid prop set <property> <value>`：設定系統屬性
- `waydroid bugreport`：產生錯誤報告日誌

### 系統屬性設定

可以透過 `waydroid prop` 指令設定各種系統屬性：

- `persist.waydroid.multi_windows`：啟用多視窗模式
- 其他屬性請參考官方文件

## 進階功能

### 自訂映像使用

要使用自訂映像，需要執行以下步驟：

1. 從 [SourceForge](https://sourceforge.net/projects/waydroid/files/images/) 下載映像或使用自訂建置的 `system.img` 和 `vendor.img`
2. 將映像複製到 `/etc/waydroid-extra/images/` 目錄
3. 重新初始化 Waydroid：`sudo waydroid init -f`

### ADB 偵錯

Waydroid 支援 ADB 偵錯，預設情況下 ADB 安全模式已啟用，不會在工作階段啟動時自動連接。可以使用 `waydroid adb connect` 指令連接 ADB，並在 Android 端提供授權。

### 通知轉發

Waydroid 支援將 Android 通知轉發到桌面環境，需要更新 Android 系統映像到相容版本。
