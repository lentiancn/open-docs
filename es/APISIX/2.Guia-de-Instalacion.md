# APISIX Guía de Instalación

Múltiples métodos de instalación de APISIX.

## Requisitos del Entorno

### Requisitos de Hardware

- CPU: 2 núcleos o más
- Memoria: 4GB o más
- Disco: 40GB o más

### Requisitos de Software

- SO: Linux (Ubuntu 20.04+, CentOS 7+ recomendado)
- Nginx u OpenResty
- Lua 5.1+
- etcd 3.4+

## Instalación Docker (Recomendado)

### Inicio Rápido

```bash
# Iniciar etcd
docker run -d --name etcd \
  -p 2379:2379 \
  apache/apisix:latest etcd

# Iniciar APISIX
docker run -d --name apisix \
  -p 9080:9080 \
  -p 9090:9090 \
  apache/apisix:latest
```

### docker-compose

```yaml
version: '3'
services:
  etcd:
    image: apache/apisix:3.5.0-etcd
    ports:
      - "2379:2379"
  apisix:
    image: apache/apisix:3.5.0
    ports:
      - "9080:9080"
    depends_on:
      - etcd
```

```bash
docker-compose up -d
```

## Instalación Helm (Kubernetes)

```bash
helm repo add apisix https://apache.github.io/apisix-helm-chart
helm install apisix apisix/apisix
```

## Instalación desde Código Fuente

### Instalar Dependencias

**Ubuntu/Debian:**

```bash
wget -qO - https://openresty.org/package/pubkey.gpg | sudo apt-key add -
sudo apt-get update
sudo apt-get install -y openresty etcd
```

### Clonar Código

```bash
git clone https://github.com/apache/apisix.git
cd apisix
git checkout 3.5.0
```

### Instalar Dependencias

```bash
make deps
make init
```

### Iniciar Servicio

```bash
make run
```

## Configuración

### Archivo de Configuración Principal

```yaml
apisix:
  node_listen: 9080
  enable_admin: true
  admin_key:
    - name: admin
      key: 123456

etcd:
  host:
    - "http://127.0.0.1:2379"
```

## Problemas Comunes

### Puerto en Uso

```bash
netstat -tlnp | grep 9080
```

### Error de Conexión etcd

```bash
etcdctl endpoint health
```

## Desinstalación

```bash
docker stop apisix etcd
docker rm apisix etcd
```
