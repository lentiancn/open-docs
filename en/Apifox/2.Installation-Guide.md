# Installation Guide

This document covers the various ways to install Apifox, helping you get started quickly.

## System Requirements

Apifox supports the following operating systems:

- Windows 7 and above (64-bit)
- macOS 10.13 and above
- Linux (Debian/Ubuntu 20.04+, Fedora 36+)

## Downloading

### Download from the Official Website

Visit the Apifox official website at https://apifox.com/, click the "Download" button, and choose the version for your operating system.

### Choosing Your Version

Apifox offers two versions:

- **Apifox** (Standard): Includes all core features, suitable for most users
- **Apifox Plus** (Paid): Includes more advanced features like team management and advanced automation

The standard version has everything you need—the basic features are more than enough.

## Installation Steps

### Windows

1. Download the Windows installer (.exe file)
2. Double-click to run the installer
3. Choose an installation path (the default is recommended)
4. Wait for installation to complete
5. Launch Apifox and log in or create an account

### macOS

1. Download the macOS installer (.dmg file)
2. Open the installer and drag Apifox into the Applications folder
3. Launch Apifox and log in or create an account

On first launch, you may need to allow apps from unidentified developers in System Preferences.

### Linux

**Debian/Ubuntu:**

```bash
# Download the deb package
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.deb

# Install
sudo dpkg -i Apifox-latest-linux.x64.deb

# If dependencies are missing, fix the installation
sudo apt-get install -f
```

**Fedora:**

```bash
# Download the rpm package
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.rpm

# Install
sudo rpm -i Apifox-latest-linux.x64.rpm
```

## Account and Login

### Creating an Account

You'll need to create an account to get started:

1. Launch Apifox
2. Click "Register"
3. Enter your email and complete registration
4. Verify your email (if required)

### Logging In

Multiple login methods are supported:

- Email and password login
- Third-party accounts (GitHub, Google, etc.)

### Cloud Sync

Once logged in, your project data syncs to the cloud automatically. You can switch between devices seamlessly, with data staying in real-time sync.

## Initial Setup

After first launch, it's recommended to configure the following:

### Setting the Theme

In "Settings," you can choose between light and dark themes—pick whichever you prefer.

### Setting Up a Proxy

If you need to access the network through a proxy, configure your proxy server in "Settings" -> "Network."

### Import and Export

- **Import**: Supports importing projects from Postman, Swagger, HAR, and other formats
- **Export**: Supports exporting to OpenAPI, Markdown, HTML, and other formats

## Quick Start

### Creating a Project

1. Click the "+" next to "Project" on the left sidebar
2. Enter a project name
3. Choose the project type (API project or Mock project)

### Creating an Interface

1. In your project, click "+" and select "New Interface"
2. Fill in the basic interface information:
   - Interface name
   - Request path
   - Request method (GET/POST/PUT/DELETE, etc.)
3. Fill in request parameters, body, response, etc.
4. Save the interface

### Debugging an Interface

1. Open the interface you created
2. Click the "Send" button
3. View the response

### Using Mock

1. Define your data structure in the interface
2. The system automatically generates a Mock URL
3. Access the Mock URL to get simulated data

## Uninstalling

### Windows

- Open "Control Panel" -> "Programs and Features"
- Find Apifox and click "Uninstall"

### macOS

- Drag Apifox from the Applications folder to the Trash
- Empty the Trash

### Linux

```bash
# Debian/Ubuntu
sudo apt-get remove apifox

# Fedora
sudo rpm -e apifox
```
