# Apifox 使用手册

Apifox 核心功能和使用技巧。

## API 管理

### 创建接口

1. 选择项目
2. 点击"新建接口"
3. 填写接口信息：
   - 接口名称
   - 请求方法（GET/POST/PUT/DELETE）
   - 请求路径
   - 请求参数
   - 请求体

### 调试接口

1. 选择接口
2. 填写请求参数
3. 点击发送
4. 查看响应

## 请求配置

### 请求头

```http
Content-Type: application/json
Authorization: Bearer token
```

### 请求参数

- **Query 参数**：URL 查询参数
- **Header 参数**：请求头参数
- **Body 参数**：请求体参数

### 认证

- **Bearer Token**：JWT 认证
- **Basic Auth**：基本认证
- **OAuth 2.0**：OAuth 认证

## Mock 服务

### 创建 Mock

1. 点击"新建 Mock"
2. 定义响应规则
3. 获取 Mock URL

### Mock 规则

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "id": 1,
    "name": "{{name}}"
  }
}
```
