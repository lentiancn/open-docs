# MySQL Introduction

MySQL is the world's most popular open-source relational database management system (RDBMS). It is widely used for web applications and is a core component of the LAMP (Linux, Apache, MySQL, PHP/Python/Perl) stack.

## What is MySQL?

MySQL is a powerful, fast, and reliable database system that stores and manages data for applications. It uses SQL (Structured Query Language) as its query language and supports multiple storage engines.

### Key Features

- **Open Source** - Free to use under GPL license
- **High Performance** - Optimized for speed and efficiency
- **Scalability** - Handles large databases with ease
- **Security** - Robust user management and encryption
- **Replication** - Master-slave and master-master support
- **Storage Engines** - InnoDB, MyISAM, Memory, and more
- **Cross-Platform** - Runs on Windows, Linux, macOS, and more

## Why Use MySQL?

## MySQL Architecture

### Components

```
┌─────────────────────────────────┐
│       Client Applications       │
├─────────────────────────────────┤
│     Connection Pool (API)       │
├─────────────────────────────────┤
│      Query Cache & Parser       │
├─────────────────────────────────┤
│     Optimizer (Engine)          │
├─────────────────────────────────┤
│      Storage Engines            │
│   ┌─────────┬──────────────┐   │
│   │ InnoDB  │   MyISAM     │   │
│   └─────────┴──────────────┘   │
├─────────────────────────────────┤
│         System Files            │
└─────────────────────────────────┘
```

### Storage Engines

| Engine | Description | Use Case |
|--------|-------------|----------|
| **InnoDB** | Default, ACID compliant | Transactions, row-level locking |
| **MyISAM** | Fast, full-text search | Read-heavy workloads |
| **Memory** | In-memory tables | Fast, temporary data |
| **CSV** | CSV file storage | Data import/export |
| **Archive** | Compressed storage | Logging, archiving |
