# Claude Code Руководство по Установке

Подробная инструкция по установке, настройке и обновлению Claude Code.

## Системные Требования

Claude Code работает на:
- macOS 13.0+, Windows 10 1809+, Ubuntu 20.04+, Debian 10+, Alpine Linux 3.19+
- Оперативная память: 4 ГБ+
- Интернет-соединение

## Методы Установки

### Нативная Установка (Рекомендуется)

**macOS, Linux, WSL:**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell:**
```powershell
irm https://claude.ai/install.ps1 | iex
```

**Windows CMD:**
```batch
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
```

### Homebrew (macOS/Linux)
```bash
brew install --cask claude-code
```

### WinGet (Windows)
```powershell
winget install Anthropic.ClaudeCode
```

## Проверка Установки

```bash
claude --version
```

Для диагностики:
```bash
claude doctor
```

## Обновление

### Автоматическое (нативная установка)
Обновления устанавливаются автоматически в фоновом режиме.

### Ручное
```bash
claude update
```

## Настройка

### Windows
Настройте путь к Git Bash:
```json
{
  "env": {
    "CLAUDE_CODE_GIT_BASH_PATH": "C:\\Program Files\\Git\\bin\\bash.exe"
  }
}
```

### Alpine Linux
```json
{
  "env": {
    "USE_BUILTIN_RIPGREP": "0"
  }
}
```

## Удаление

**Нативная:**
```bash
rm -f ~/.local/bin/claude
rm -rf ~/.local/share/claude
```

**Homebrew:**
```bash
brew uninstall --cask claude-code
```

**WinGet:**
```powershell
winget uninstall Anthropic.ClaudeCode
```
