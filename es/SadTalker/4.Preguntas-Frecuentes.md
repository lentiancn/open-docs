# SadTalker Preguntas Frecuentes

## Preguntas Generales

### ¿Qué es SadTalker?
SadTalker es un proyecto de aprendizaje profundo que genera videos realistas de caras parlantes a partir de una imagen estática y entrada de audio. Crea animaciones faciales con sincronización de labios.

### ¿SadTalker es gratuito?
Sí, SadTalker es de código abierto y gratuito para uso personal e investigación. Consulta el archivo LICENSE para detalles.

### ¿Cuáles son los requisitos del sistema?
- **GPU**: NVIDIA GPU con 8GB+ VRAM (recomendado)
- **RAM**: 8GB mínimo, 16GB recomendado
- **Almacenamiento**: 20GB+ para modelos y salidas
- **OS**: Linux (Ubuntu 20.04+), Windows 10/11, macOS

## Preguntas de Instalación

### ¿Cómo instalo SadTalker?
```bash
# Clonar el repositorio
git clone https://github.com/WinfredSadTalker/SadTalker.git
cd SadTalker

# Crear entorno conda
conda create -n sadtalker python=3.8
conda activate sadtalker

# Instalar dependencias
pip install -r requirements.txt

# Descargar modelos pre-entrenados
bash scripts/download_models.sh
```

### ¿Qué versión de Python debo usar?
SadTalker funciona mejor con Python 3.8 o 3.9. Python 3.10+ puede tener problemas de compatibilidad.

### ¿Necesito una GPU?
Una GPU es muy recomendable para un rendimiento razonable. La inferencia por CPU es posible pero muy lenta.

## Preguntas de Uso

### ¿Cómo genero un video parlante?
```bash
python inference.py --source_image path/to/image.jpg --audio path/to/audio.wav
```

### ¿Qué formatos de imagen son soportados?
- JPG/JPEG
- PNG
- La mayoría de formatos comunes

### ¿Qué formatos de audio son soportados?
- WAV (recomendado)
- MP3
- Otros formatos soportados por FFmpeg

### ¿Cuánto tiempo toma la generación?
El tiempo de procesamiento depende de:
- Longitud del video (típicamente 1-2 segundos por segundo de audio)
- Rendimiento GPU
- Configuración del modelo
- Resolución

## Solución de Problemas

### El modelo no se descarga
Descarga manualmente los modelos desde el repositorio oficial o usa fuentes alternativas mencionadas en la documentación.

### Errores de memoria insuficiente
- Reducir tamaño de batch
- Bajar resolución de salida
- Usar model swapping
- Cerrar otras aplicaciones GPU

### Resultados de mala calidad
- Usar imágenes de entrada de alta resolución
- Asegurar audio limpio sin ruido de fondo
- Ajustar parámetros de mejora
- Probar diferentes opciones de preprocesamiento

### Detección de cara falla
- Usar imágenes con caras claramente visibles
- Asegurar pose de frente
- Evitar caras muy ocultadas
- Verificar archivos del modelo de detección de cara

## Preguntas Avanzadas

### ¿Puedo usar SadTalker comercialmente?
Consulta los términos de la licencia. Algunos componentes pueden tener restricciones para uso comercial.

### ¿Cómo funciona SadTalker?
SadTalker usa:
1. Detección de cara para localizar landmarks faciales
2. Extracción de características de audio
3. Generación de modelo de cara 3D (3DMM)
4. Predicción de coeficientes de movimiento desde audio
5. Renderizado de video con red de renderizado de cara

### ¿Puedo entrenar mi propio modelo?
Sí, puedes afinar SadTalker en conjuntos de datos personalizados. Consulta la documentación de entrenamiento para detalles.
