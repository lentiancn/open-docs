# Claude Code 安装指南

详细介绍 Claude Code 的安装方法。

## Windows 安装

### 方法一：VS Code 扩展

1. 打开 VS Code
2. 搜索 "Claude Code" 扩展
3. 点击安装
4. 登录 Anthropic 账户

### 方法二：独立客户端

1. 访问 https://claude.com/code
2. 下载 Windows 客户端
3. 运行安装程序
4. 登录账户

## macOS 安装

### 方法一：VS Code 扩展

1. 打开 VS Code
2. 搜索 "Claude Code" 扩展
3. 点击安装
4. 登录账户

### 方法二：Homebrew

```
brew install claude-code
```

### 方法三：独立客户端

1. 访问 https://claude.com/code
2. 下载 macOS 客户端
3. 运行安装程序
4. 登录账户

## Linux 安装

### 方法一：VS Code 扩展

1. 打开 VS Code
2. 搜索 "Claude Code" 扩展
3. 点击安装
4. 登录账户

### 方法二：包管理器

**Ubuntu/Debian**:
```
curl -fsSL https://downloads.anthropic.com/claude-code/install.sh | sh
```

**Arch Linux**:
```
yay -S claude-code
```

## 验证安装

```
claude --version
```

## 配置

### API 密钥

1. 访问 https://console.anthropic.com
2. 创建 API 密钥
3. 在 Claude Code 中配置

### 环境变量

```
export ANTHROPIC_API_KEY=your-api-key
```

## 卸载

### macOS

```
brew uninstall claude-code
```

### Linux

```
sudo rm -rf /usr/local/claude-code
```
