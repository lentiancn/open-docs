# Element Plus FAQ

자주 묻는 질문.

## 설치

### Q: 설치 실패?

**해결책:**
1. Node.js 버전 확인
```bash
node --version  # 18.0+ 필요
```
2. 캐시 비우기
```bash
npm cache clean --force
```

### Q: TypeScript 오류?

**해결책:**
타입 설치:
```bash
npm install -D @types/node
```

## 사용

### Q: 컴포넌트가 표시되지 않음?

**해결책:**
1. 임포트 확인
2. Vue 버전 확인 (3.2+)

### Q: 스타일이 적용되지 않음?

**해결책:**
1. CSS 임포트 확인
2. `:deep()` 사용

### Q: 아이콘이 표시되지 않음?

**해결책:**
아이콘 패키지 설치:
```bash
npm install @element-plus/icons-vue
```

등록:
```typescript
import * as icons from '@element-plus/icons-vue'
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```
