# jQuery 安装指南

## 获取 jQuery

### 最新版本

当前最新版本为 **jQuery 4.0.0**。

#### 生产环境压缩版（推荐）
- 下载地址：[jquery-4.0.0.min.js](https://code.jquery.com/jquery-4.0.0.min.js)
- 适用于生产环境，文件体积小，加载速度快

#### 开发调试版
- 下载地址：[jquery-4.0.0.js](https://code.jquery.com/jquery-4.0.0.js)
- 包含完整代码，便于阅读和调试

#### Source Map 文件
- 下载地址：[jquery-4.0.0.min.map](https://code.jquery.com/jquery-4.0.0.min.map)
- 用于在浏览器调试器中显示原始源代码

#### Slim 版本
Slim 版本是精简版，移除了 Ajax 和动画模块，文件更小：
- [jquery-4.0.0.slim.min.js](https://code.jquery.com/jquery-4.0.0.slim.min.js) - 压缩版
- [jquery-4.0.0.slim.js](https://code.jquery.com/jquery-4.0.0.slim.js) - 未压缩版

如果不需要 Ajax 和动画功能，使用 Slim 版本可以减少约 30% 的文件大小。

## 安装方式

### 方式一：CDN 引入（推荐）

使用 jQuery CDN 引入是最简单的方式，无需下载文件，直接在 HTML 中引用即可：

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
```

带 Subresource Integrity（SRI）验证的版本（推荐，可防止 CDN 被篡改）：

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js" 
        integrity="sha384-KJ3o2DKtIkvYIK3UENzmM7KCkRr/rE9/Qpg6aAZGJwFDMVNA/GpGFF93hXpG5KkN" 
        crossorigin="anonymous"></script>
```

可以通过访问 [https://releases.jquery.com](https://releases.jquery.com) 获取完整版本列表和 SRI 哈希值。

### 方式二：npm 安装

如果使用 Node.js 项目，可以通过 npm 安装：

```bash
npm install jquery
```

安装后，在 node_modules/jquery/dist/ 目录下可以找到：
- jquery-4.0.0.min.js - 压缩版
- jquery-4.0.0.js - 未压缩版
- jquery-4.0.0.min.map - Source Map

### 方式三：Yarn 安装

```bash
yarn add jquery
```

### 方式四：下载到本地

访问 [jQuery 官网下载页面](https://jquery.com/download/)，右键点击所需版本，选择"另存为"将文件保存到本地项目目录中。

## 其他 CDN 服务

除官方 CDN 外，以下 CDN 也提供 jQuery：

### Google CDN
```
https://ajax.googleapis.com/ajax/libs/jquery/4.0.0/jquery.min.js
```

### Microsoft CDN
```
https://ajax.aspnetcdn.com/ajax/jquery/jquery-4.0.0.min.js
```

### CDNJS
```
https://cdnjs.cloudflare.com/ajax/libs/jquery/4.0.0/jquery.min.js
```

### jsDelivr
```
https://cdn.jsdelivr.net/npm/jquery@4.0.0/dist/jquery.min.js
```

注意：这些第三方 CDN 可能存在一定的延迟，新版本发布后可能需要等待一段时间才能同步。

## 升级 jQuery

### 使用 jQuery Migrate 插件

升级 jQuery 时，建议使用 jQuery Migrate 插件来识别兼容性问题：

1. 从旧版本升级到 1.9+ 或 3.x：使用 [jQuery Migrate 1.4.1](https://code.jquery.com/jquery-migrate-1.4.1.min.js)

2. 从 jQuery 3.x 升级到更新版本：使用 [jQuery Migrate 3.6.0](https://code.jquery.com/jquery-migrate-3.6.0.min.js)

使用方法：在 jQuery 之后引入 Migrate 插件

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
<script src="https://code.jquery.com/jquery-migrate-3.6.0.min.js"></script>
```

Migrat 会在控制台输出兼容性问题警告，帮助你逐步修复代码。

### 升级指南

针对不同版本升级，请查阅官方升级指南：
- [jQuery 4.0 升级指南](https://jquery.com/upgrade-guide/4.0/)
- [jQuery 3.0 升级指南](https://jquery.com/upgrade-guide/3.0/)

## 历史版本

需要使用旧版本 jQuery 时，可以访问 [jQuery CDN](https://releases.jquery.com) 查看所有历史版本。

## 预发布版本

jQuery 开发团队会在 GitHub 上持续开发，每隔提交都会生成预发布版本。这些版本可能不稳定，不建议在生产环境中使用。

查看预发布版本：[https://releases.jquery.com/jquery/](https://releases.jquery.com/jquery/)
