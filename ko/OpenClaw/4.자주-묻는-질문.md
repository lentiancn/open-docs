# 자주 묻는 질문

OpenClaw에 대한 자주 묻는 질문과 답변입니다.

## 기본

### OpenClaw란?

OpenClaw는 메시징 앱을 AI 어시스턴트에 연결하는 도구입니다. 데이터를 완전히 관리하면서 이미 설치된 앱을 통해 AI를 사용할 수 있습니다.

### OpenClaw가 무료인가요?

소프트웨어 자체는 무료입니다 (오픈 소스). 그러나 다음을 지불해야 합니다:
- 실행할 컴퓨터 또는 서버
- AI 서비스용 API 키 비용

### AI 서비스 구독이 필요한가요?

필수하지 않습니다. 사용량에 따라 지불하는 API 키 또는 구독 계정을 사용할 수 있습니다. OpenClaw는 Anthropic, OpenAI, Google Gemini 등을 지원합니다.

### 어떤 플랫폼에서 작동하나요?

- **데스크톱**: macOS, Linux, Windows
- **모바일**: iOS, Android (노드로)
- **메시징 앱**: WhatsApp, Telegram, Discord, iMessage, Signal, Slack 등 20개 이상

## 설치

### 설치가 실패했습니다. 어떻게 해야 하나요?

1. Node.js 버전 확인: `node --version` (22 이상 필요)
2. 진단 실행: `openclaw doctor`
3. 로그 확인: `openclaw logs`

### Raspberry Pi에서 실행할 수 있나요?

네. Raspberry Pi 4 이상과 Raspberry Pi OS 또는 Ubuntu에서 작동합니다.

### Mac이 필요한가요?

아니요. Node.js가 있는 모든 시스템에서 작동합니다. iMessage 채널만 Apple 제한으로 Mac이 필요합니다.

## 설정

### 구성 파일은 어디에 있나요?

기본 위치: `~/.openclaw/openclaw.json`

### 누가 메시지를 보낼 수 있는지 어떻게 통제하나요?

`dmPolicy` 설정을 사용합니다:
- `pairing`: 모르는 사람은 연결 코드 필요
- `allowlist`: 화이트리스트의 사용자만
- `open`: 누구나 가능
- `disabled`: 비활성화

### 그룹 채팅을 어떻게 설정하나요?

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

## 사용

### 새 채팅 채널을 어떻게 추가하나요?

```bash
openclaw channels login whatsapp
```

### 여러 채널을 동시에 사용할 수 있나요?

네. 하나의 Gateway가 여러 플랫폼에 연결됩니다.

### AI가 이전 대화를 기억하나요?

OpenClaw는 대화 컨텍스트를 자동으로 저장합니다. 구성에서 보존 정책을 조정할 수 있습니다.

### 여러 AI 에이전트를 실행할 수 있나요?

네. 다양한 규칙으로 여러 에이전트를 구성하세요:

```json5
{
  agents: {
    list: [
      { id: "home", default: true },
      { id: "work" }
    ]
  }
}
```

## 보안

### 제 데이터가 안전한가요?

네. OpenClaw는 로컬에서 실행됩니다 — 모든 데이터가 기기에 남아 있습니다.

### 설치를 어떻게 보호하나요?

1. `allowFrom`으로 소스 제한
2. 강력한 비밀번호와 API 키 사용
3. OpenClaw를 최신 상태로 유지
4. 방화벽 사용

## 문제 해결

### Gateway가 시작되지 않습니다

1. 포트 사용 여부 확인: `openclaw gateway status`
2. 오류 로그 확인: `openclaw logs`
3. 진단 실행: `openclaw doctor`

### AI가 응답하지 않습니다

1. API 키 확인
2. 계정 잔액 확인
3. 로그에서 오류 확인
