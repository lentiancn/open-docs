# Element UI インストールガイド

## 環境要件

- Vue 2.0.0+
- Node.js 8.0.0+
- 最新のブラウザ

## インストール

npm install element-ui --save

## 使用方法

import Vue from 'vue'
import Element from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(Element)

## 必要に応じたインポート

まずbabel-plugin-componentをインストール：

npm install babel-plugin-component -D

.babelrcで設定：

{
  "plugins": [
    ["component", {
      "libraryName": "element-ui",
      "styleLibraryName": "theme-chalk"
    }]
  ]
}

## グローバル設定

Vue.use(Element, {
  size: 'small',
  zIndex: 2000
})

詳細については、Element UI公式ウェブサイトを参照してください。
