# Bootstrap 5 Руководство по установке

Это руководство подробно описывает установку и настройку Bootstrap 5 в различных средах.

## Информация о версиях

| Версия | Статус | Описание |
|---------|--------|----------|
| Bootstrap 5.3.x | ✅ Текущая | Последняя версия со всеми новыми функциями |
| Bootstrap 5.2.x | ✅ Поддерживается | Стабильная версия |
| Bootstrap 4.x | ❌ Архивная | Не рекомендуется для новых проектов |

> **Важно**: Это руководство ориентировано на **Bootstrap 5.3.x**.

## Методы установки

### Метод 1: CDN (самый простой)

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
```

### Метод 2: npm

```bash
npm install bootstrap@5.3.3
```

### Метод 3: Vite

```bash
npm create vite@latest my-app -- --template vanilla
cd my-app
npm install bootstrap@5.3.3
```

```javascript
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

### Метод 4: Webpack

```bash
npm install bootstrap@5.3.3
npm install --save-dev webpack webpack-cli css-loader style-loader
```

### Метод 5: Vue CLI / Nuxt

```bash
npm install -g @vue/cli
vue create my-app
npm install bootstrap@5.3.3
```

### Метод 6: React

```bash
npx create-react-app my-app
npm install bootstrap@5.3.3
```

### Метод 7: Скачать исходники

```bash
git clone https://github.com/twbs/bootstrap.git
cd bootstrap
npm install
npm run dist
```

## Особые настройки

### RTL (справа налево)

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.rtl.min.css" rel="stylesheet">
```

### Настройка с Sass

```bash
npm install bootstrap@5.3.3
npm install --save-dev sass
```

## Частые вопросы

### Что такое Popper?

Popper — это движок позиционирования для всплывающих подсказок и popover. `bootstrap.bundle.min.js` уже включает Popper.

### Нужен ли jQuery?

**Нет!** Bootstrap 5 полностью переписан на нативном JavaScript без зависимости от jQuery.

### Как использовать иконки?

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css">
```
