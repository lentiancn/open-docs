# Bootstrap

Le framework CSS frontend le plus populaire au monde.

## Aperçu

Bootstrap est le framework CSS le plus populaire de Twitter. Avec grille responsive, composants prêts et utilitaires CSS.

### Fonctionnalités principales

- **Grille responsive** - Grille 12 colonnes
- **Composants prêts** - Navbars, cartes, modales
- **Utilitaires CSS** - Ajustements rapides

## Démarrage rapide

### CDN

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```
