# Claude Code Häufig Gestellte Fragen

Diese Seite listet häufige Probleme bei der Installation und Nutzung von Claude Code auf.

## Installationsprobleme

### Befehl nicht gefunden

Nach der Installation treten `command not found: claude`-Fehler auf.

**Lösung:** Überprüfen und konfigurieren Sie Ihre PATH-Umgebungsvariable.

macOS/Linux:
```bash
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

Windows PowerShell:
```powershell
$currentPath = [Environment]::GetEnvironmentVariable('PATH', 'User')
[Environment]::SetEnvironmentVariable('PATH', "$currentPath;$env:USERPROFILE\.local\bin", 'User')
```

Dann Terminal neu starten und verifizieren:
```bash
claude --version
```

### Installationsskript gibt HTML zurück

**Lösung:** Verwenden Sie eine alternative Installationsmethode:
- macOS/Linux: Homebrew
  ```bash
  brew install --cask claude-code
  ```
- Windows: WinGet
  ```powershell
  winget install Anthropic.ClaudeCode
  ```

### TLS/SSL-Verbindungsfehler

**Lösung:**
1. CA-Zertifikate aktualisieren:
   ```bash
   sudo apt-get update && sudo apt-get install ca-certificates
   ```
2. Proxy-Einstellungen überprüfen

## Netzwerkkonfiguration

Der Installer lädt von `storage.googleapis.com` herunter. Überprüfen Sie die Konnektivität:

```bash
curl -sI https://storage.googleapis.com
```

## Verifizierung der Installation

Prüfen Sie auf mehrere Installationen:

```bash
which -a claude
```

## Diagnose

Verwenden Sie `claude doctor` für detaillierte Informationen:

```bash
claude doctor
```
