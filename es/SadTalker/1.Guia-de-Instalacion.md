# SadTalker Guía de Instalación

Esta guía te lleva a través de la instalación y configuración de SadTalker en tu sistema.

## Requisitos del Sistema

### Hardware
- **GPU**: NVIDIA GPU con 8GB+ VRAM (recomendado)
- **RAM**: 8GB mínimo, 16GB recomendado
- **Almacenamiento**: 20GB+ para modelos y salidas

### Software
- Python 3.8+
- CUDA 11.3+
- cuDNN 8.0+
- FFmpeg

## Instalación

### 1. Clonar el Repositorio

```bash
git clone https://github.com/WinfredSadTalker/SadTalker.git
cd SadTalker
```

### 2. Crear Entorno Conda

```bash
conda create -n sadtalker python=3.8
conda activate sadtalker
```

### 3. Instalar Dependencias

```bash
# Instalar PyTorch con CUDA
pip install torch==1.12.1 torchvision==0.13.1 --extra-index-url https://download.pytorch.org/whl/cu113

# Instalar dependencias base
pip install matplotlib scipy tqdm
pip install opencv-python
pip install librosa soundfile

# Instalar face alignment
pip install face-alignment

# Instalar otras dependencias
pip install -r requirements.txt
```

### 4. Descargar Modelos Pre-entrenados

```bash
bash scripts/download_models.sh
```

O descarga manualmente desde el releases page.

### 5. Verificar Instalación

```bash
python inference.py --help
```

## Configuración

### Estructura de Directorios

```
SadTalker/
├── configs/
│   └── sadtalker.yaml
├── scripts/
│   └── download_models.sh
├── inference.py
├── requirements.txt
└── README.md
```

### Configuración de Rutas

Edita `inference.py` o usa argumentos de línea de comandos para especificar rutas.

## Solución de Problemas

### Error de Memoria GPU
- Reducir tamaño de batch
- Bajar resolución de salida
- Cerrar otras aplicaciones GPU

### Error de Descarga de Modelos
Descarga manualmente desde fuentes alternativas mencionadas en la documentación.

### Face Detection Falla
- Usar imágenes con caras claramente visibles
- Asegurar pose de frente
- Evitar caras muy ocultadas
