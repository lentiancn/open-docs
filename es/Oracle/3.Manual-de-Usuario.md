# Oracle Manual de Usuario

## Conexión a la Base de Datos

### Usando SQL*Plus

```bash
# Conectar como SYSDBA
sqlplus / as sysdba

# Con usuario y contraseña
sqlplus usuario/contraseña@database
```

---

## Operaciones SQL Básicas

### Crear Tabla

```sql
CREATE TABLE empleados (
    empleado_id NUMBER(10) PRIMARY KEY,
    nombre VARCHAR2(50) NOT NULL,
    apellido VARCHAR2(50) NOT NULL,
    email VARCHAR2(100),
    salario NUMBER(10, 2)
);
```

### Insertar Datos

```sql
INSERT INTO empleados (empleado_id, nombre, apellido, email, salario)
VALUES (1001, 'Juan', 'Pérez', 'juan@company.com', 75000);
```

### Consultar Datos

```sql
SELECT * FROM empleados;
SELECT nombre, apellido, salario FROM empleados WHERE departamento_id = 10;
```

## PL/SQL

### Procedimiento Almacenado

```sql
CREATE OR REPLACE PROCEDURE actualizar_salario (
    p_empleado_id IN NUMBER,
    p_nuevo_salario IN NUMBER
) AS
BEGIN
    UPDATE empleados
    SET salario = p_nuevo_salario
    WHERE empleado_id = p_empleado_id;
    COMMIT;
END;
/
```
