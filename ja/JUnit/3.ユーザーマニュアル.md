# JUnit 5 ユーザーマニュアル

## 基本的なテスト

```java
@Test
void test() {
    assertEquals(4, 2 + 2);
}
```

## アサーション

```java
assertEquals(4, 2 + 2);
assertTrue(5 > 3);
assertThrows(Exception.class, () -> { });
```

## パラメータ化テスト

```java
@ParameterizedTest
@ValueSource(ints = {1, 2, 3})
void test(int n) {
    assertTrue(n > 0);
}
```

## ネストされたテスト

```java
@Nested
class InnerTests {
    @Test
    void test() { }
}
```
