# Flutter - Aperçu

Flutter est un outil UI open source par Google.

## Qu'est-ce que Flutter?

Flutter est un outil UI pour créer des applications multiplateformes pour iOS, Android, Web et Desktop à partir d'une seule base de code. Utilise Dart.

## Fonctionnalités

- **Multiplateforme**: Une base de code
- **Hot Reload**: Aperçu instantané
- **Widgets riches**: Material Design + Cupertino
- **Haute performance**: Moteur graphique Skia
- **Open Source**: Gratuit
