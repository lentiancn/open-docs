# Angular FAQ

## インストール問題

### Q: Angular CLI インストール失敗？

**解決策:**
1. Node.jsバージョンを確認
```bash
node --version  # 18+ または 20+ 必要
```
2. npmキャッシュをクリア
```bash
npm cache clean --force
```

## 開発問題

### Q: データが更新されない？

**解決策:**
Signalの場合：
```typescript
count.set(1);
count.update(c => c + 1);
```

## パフォーマンス問題

### Q: 初期ロードが遅い？

**解決策:**
1. 遅延ロードを使用
```typescript
{
  path: 'admin',
  loadComponent: () => import('./admin/admin.component')
    .then(m => m.AdminComponent)
}
```
2. 本番モードを有効化
```bash
ng build --configuration=production
```
