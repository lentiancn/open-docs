# Element UI Resumen

Biblioteca de componentes UI de escritorio Vue.js 2.0.

## ¿Qué es Element UI?

Element UI es una biblioteca de componentes de código abierto desarrollada por Ele.me.

### Características principales

- **Rich Components**: 60+ componentes
- **Beautiful Design**: Directrices de Material Design
- **TypeScript Support**: Definiciones de tipo completas
- **On-demand Import**: Importación bajo demanda

- GitHub Stars: 54,000+
- Descargas semanales: 180,000+
