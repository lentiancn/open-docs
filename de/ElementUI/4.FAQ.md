# Element UI FAQ

## Installation

### Q: Installation fehlgeschlagen?

**Lösungen:**
1. Node.js Version prüfen
2. npm Cache leeren
```bash
npm cache clean --force
```

### Q: Vue Version inkompatibel?

**Lösungen:**
Element UI 2.x erfordert Vue 2.6.0+

## Verwendung

### Q: Komponenten werden nicht angezeigt?

**Lösungen:**
Import prüfen:
```javascript
import { Button } from 'element-ui'
Vue.use(Button)
```

### Q: Styles nicht angewendet?

**Lösungen:**
Style-Datei importieren:
```javascript
import 'element-ui/lib/theme-chalk/index.css'
```
