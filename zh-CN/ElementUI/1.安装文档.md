# Element UI 安装文档

在所有主流平台上安装 Element UI 的完整指南。

---

## 什么是 Element UI？

Element UI 是一个基于 Vue 2.0 的组件库，为开发者、设计师和产品经理提供高质量的 UI 组件。

### 注意

**Element UI 适用于 Vue 2。** 如果使用 Vue 3，请使用 **Element Plus**。

---

## CDN

### 快速开始

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Element UI CDN</title>
  <link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
</head>
<body>
  <div id="app">
    <el-button>按钮</el-button>
  </div>

  <script src="https://unpkg.com/vue@2"></script>
  <script src="https://unpkg.com/element-ui/lib/index.js"></script>
  <script>
    new Vue({
      el: '#app'
    })
  </script>
</body>
</html>
```

### CDN 提供商

| 提供商 | CSS | JS |
|--------|-----|-----|
| unpkg | https://unpkg.com/element-ui/lib/theme-chalk/index.css | https://unpkg.com/element-ui/lib/index.js |
| jsDelivr | https://cdn.jsdelivr.net/npm/element-ui/lib/theme-chalk/index.css | https://cdn.jsdelivr.net/npm/element-ui/lib/index.js |

## NPM

### 安装 Element UI

```bash
# 安装 Vue 2 版本的 Element UI
npm install element-ui

# 对于 Vue 3，请使用 Element Plus
npm install element-plus
```

---

## Vue CLI

### 使用 Vue CLI 3/4

```bash
# 创建 Vue 项目
vue create my-element-app

# 添加 Element UI
cd my-element-app
npm install element-ui
```

---

## Vite

### 使用 Vite + Vue 2

```bash
npm create vite@latest my-element-app -- --template vue
cd my-element-app
npm install element-ui
npm install @vitejs/plugin-vue2
```

---

## 基础配置

### main.js

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(ElementUI)

new Vue({
  render: h => h(App)
}).$mount('#app')
```
