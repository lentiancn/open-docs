# Angular

Angular ist ein Open-Source-Webframework, das von Google entwickelt und gepflegt wird und zur Erstellung moderner Webanwendungen dient. TypeScript ist die hauptsächliche Programmiersprache. Es bietet leistungsstarke Funktionen wie komponentenbasierte Architektur, Two-Way-Databinding und Dependency Injection.

## Schnellstart

### Angular CLI installieren

```bash
npm install -g @angular/cli
```

### Neues Projekt erstellen

```bash
ng new my-app
cd my-app
```

### Entwicklungsserver starten

```bash
ng serve
```

Öffne http://localhost:4200

## Kernfunktionen

- **Komponentenbasierte Architektur:** Modulare, wiederverwendbare UI-Komponenten
- **TypeScript:** Starke Typunterstützung für bessere Codequalität
- **Dependency Injection:** Lose Kopplung, leicht zu testen
- **Two-Way-Databinding:** Automatische Datensynchronisation
- **Routing:** Vollständige Routing-Lösung
- **HTTP-Client:** Leistungsstarke Netzwerkanfragen
- **Formularverarbeitung:** Reaktive und templategetriebene Formulare
- **RxJS:** Unterstützung für reaktive Programmierung
