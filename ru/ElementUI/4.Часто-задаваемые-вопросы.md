# Часто задаваемые вопросы Element UI

## Проблемы с установкой

Проверьте Node.js >= 8.0.0

npm cache clean --force

## Проблемы использования

### Стили не применяются

Импортировать CSS:

import 'element-ui/lib/theme-chalk/index.css'

### Иконки не отображаются

Импортировать шрифты иконок:

import 'element-ui/lib/theme-chalk/fonts/element-icons.ttf'

## Проблемы с версией Vue

Element UI поддерживает только Vue 2.
