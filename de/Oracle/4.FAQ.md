# Oracle Datenbank FAQ

## Allgemeine Fragen

### Was ist Oracle Database?

Oracle Database ist ein proprietäres relationales Datenbankmanagementsystem von Oracle Corporation.

### Was ist PL/SQL?

PL/SQL ist die prozedurale Spracherweiterung von Oracle für SQL.

---

## Installation

### Systemanforderungen?

- Mindestens 2GB RAM
- Mindestens 8GB Festplatte

### ORA-12541: TNS:no listener?

Starten Sie den Listener:
```bash
lsnrctl start
```

## Leistung

### Abfrageleistung prüfen?

```sql
EXPLAIN PLAN FOR your_query;
SELECT * FROM TABLE(DBMS_XPLAN.DISPLAY);
```
