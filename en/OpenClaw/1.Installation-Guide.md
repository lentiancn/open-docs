# OpenClaw Installation Guide

Comprehensive guide for installing OpenClaw on all major platforms.

---

## What is OpenClaw?

OpenClaw is a **self-hosted gateway** that connects chat apps (WhatsApp, Telegram, Discord, iMessage) to AI coding agents. It runs on your own machine or server.

**Requirements:**
- Node.js 22+
- API key (Anthropic recommended)
- 5 minutes

---

## npm Installation (Recommended)

### Quick Install

```bash
npm install -g openclaw@latest
```

### Verify Installation

```bash
openclaw --version
openclaw --help
```

---

## Onboard and Setup

### Initialize OpenClaw

```bash
# Onboard and install the service
openclaw onboard --install-daemon
```

### Start Gateway

```bash
# Start the gateway
openclaw gateway --port 18789

# Or run in background
openclaw gateway --port 18789 &
```

## Platform-Specific Installation

### macOS

```bash
# Using npm
npm install -g openclaw@latest

# Using Homebrew (if available)
brew install openclaw
```

### Linux

```bash
# Using npm
npm install -g openclaw@latest

# For Ubuntu/Debian
sudo apt-get install nodejs npm
npm install -g openclaw@latest
```

### Windows

```bash
# Using npm (PowerShell or CMD)
npm install -g openclaw@latest

# Using winget
winget install OpenClaw.OpenClaw
```

## Docker Installation

### Pull Image

```bash
docker pull openclaw/openclaw:latest
```

### Run Container

```bash
docker run -d \
  --name openclaw \
  -p 18789:18789 \
  -v ~/.openclaw:/root/.openclaw \
  openclaw/openclaw:latest
```

### Docker Compose

```yaml
version: '3.8'
services:
  openclaw:
    image: openclaw/openclaw:latest
    ports:
      - "18789:18789"
    volumes:
      - ~/.openclaw:/root/.openclaw
    environment:
      - ANTHROPIC_API_KEY=your-api-key
```

## Configuration

### Basic Configuration

Config file location: `~/.openclaw/openclaw.json`

```json
{
  "channels": {
    "whatsapp": {
      "allowFrom": ["+15555550123"]
    }
  },
  "messages": {
    "groupChat": {
      "mentionPatterns": ["@openclaw"]
    }
  }
}
```

### Environment Variables

```bash
# Set API key
export ANTHROPIC_API_KEY=sk-ant-your-key

# Or in config
{
  "providers": {
    "anthropic": {
      "apiKey": "your-api-key"
    }
  }
}
```

## Update OpenClaw

```bash
# Update to latest version
npm update -g openclaw

# Or reinstall
npm install -g openclaw@latest
```

---

## Uninstall

```bash
npm uninstall -g openclaw

# Remove config (optional)
rm -rf ~/.openclaw
```
