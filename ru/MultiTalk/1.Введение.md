# Введение

MultiTalk — это инновационный фреймворк для генерации видео с многопользовательскими разговорами, управляемый аудио, разработанный командой MeiGen-AI. Этот проект направлен на преодоление ограничений существующих технологий создания цифровых людей, которые поддерживают только анимацию одного человека. MultiTalk способен генерировать видео, содержащие взаимодействие нескольких человек, на основе многопотокового аудиоввода, референсных изображений и текстовых подсказок.

## Предыстория проекта

Традиционные методы анимации человека, управляемые аудио, такие как генерация говорящей головы и говорящего тела, достигли значительного прогресса в создании синхронизированных движений лица и видео высокого качества. Однако существующие методы в основном сосредоточены на анимации одного человека и испытывают трудности с многопотоковыми аудиовводами, сталкиваясь с проблемами неправильного связывания аудио и людей. Кроме того, эти методы имеют ограничения в возможностях следования инструкциям. MultiTalk был создан для решения этих проблем.

## Основные функции

Основная функция MultiTalk — автоматическая генерация видео с многопользовательскими разговорами. Пользователям нужно только предоставить аудио, содержащее диалог нескольких человек, референсное изображение с несколькими людьми и текстовую подсказку, описывающую сцену видео. Система автоматически генерирует соответствующие разговорные видео с высокой синхронизацией движений губ с аудио и плавным, естественным изображением.

## Технические характеристики

MultiTalk использует несколько инновационных технологий для решения проблем генерации многопользовательского видео. Во-первых, проект представляет метод маркировки ротационного позиционного вложения (Label Rotary Position Embedding, L-RoPE) для решения проблемы связывания аудио и людей при многопотоковом вводе аудио. Во-вторых, частичное параметрическое обучение и многозадачное обучение эффективно сохраняют способность базовой модели следовать инструкциям. Кроме того, проект поддерживает адаптивную локализацию людей, точно определяя соответствующих людей в генерируемых видео на основе характеристик областей людей на референсных изображениях.

## Области применения

MultiTalk имеет широкие области применения. В области создания контента создатели могут использовать этот инструмент для быстрой генерации разговорных видео с цифровыми людьми в образовательных целях, для творческого выражения или создания развлекательного контента. В исследовательской области проект предоставляет новые направления исследований и технические ориентиры для генерации многопользовательского видео. В бизнесе предприятия могут применять его в таких сценариях, как виртуальные ведущие, многопользовательская поддержка клиентов и совещания с цифровыми людьми.

## Производительность

MultiTalk демонстрирует отличную производительность на нескольких бенчмарках. Проект достигает лидирующих результатов генерации на наборах данных говорящей головы, говорящего тела и многопользовательских наборах данных с точностью синхронизации губ, достигающей лидирующего уровня в индустрии. Проект поддерживает генерацию видео с разрешением 480P и 720P, с максимальной длиной клипа до 15 секунд.

## Информация о проекте

MultiTalk — это проект с открытым исходным кодом, с кодом и весами моделей, доступными на GitHub и Hugging Face. Связанная статья опубликована на arXiv, подробно описывающая технические методы и экспериментальные результаты.
