# Java

## 개요

Java는 Oracle Corporation이 유지보수하는 객체 지향 프로그래밍 언어입니다. 세계에서 가장 인기 있는 기업 개발 언어 중 하나입니다. Java는 "한 번 작성하면 어디서나 실행"이라는 특성으로 유명하며, 성숙한 생태계와 광범위한 적용 분야를 가지고 있습니다.

## 주요 특징

- 객체 지향: 완전한 객체 지향 프로그래밍 언어
- 플랫폼 독립: JVM을 통한 플랫폼 간 실행
- 자동 메모리 관리: 가비지 컬렉션 메커니즘
- 풍부한 클래스 라이브러리: 포괄적인 표준 API
- 멀티스레딩 지원: 기본 제공 병렬 프로그래밍 지원
- 보안: 다층 보안 메커니즘

## 빠른 시작

### JDK 설치

JDK를 얻으려면 Oracle 공식 다운로드 페이지를 방문하세요: https://www.oracle.com/java/technologies/downloads/

### 첫 번째 프로그램 작성

HelloWorld.java를 만드세요:

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

### 컴파일 및 실행

```bash
# 컴파일
javac HelloWorld.java

# 실행
java HelloWorld
```
