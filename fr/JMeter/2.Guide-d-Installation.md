# Guide d'Installation de JMeter

## Prérequis

- **Java Development Kit (JDK)** : JMeter nécessite Java 8 ou supérieur
- **Système d'exploitation** : Windows, Linux ou macOS
- **Mémoire** : Au moins 4 Go de RAM recommandés pour les tests de charge

## Étapes d'installation

### 1. Télécharger JMeter

Visitez le site officiel d'Apache JMeter : https://jmeter.apache.org/

Téléchargez la dernière version (généralement un fichier .zip ou .tar.gz).

### 2. Extraire l'archive

Extrayez le fichier téléchargé à l'emplacement souhaité :

```bash
# Linux/Mac
tar -xvf apache-jmeter-x.x.x.tgz

# Windows
# Utilisez 7-Zip ou WinRAR pour extraire
```

### 3. Configurer les variables d'environnement (Optionnel)

Pour plus de commodité, vous pouvez ajouter JMeter à votre PATH système :

```bash
# Linux/Mac (ajouter à ~/.bashrc ou ~/.zshrc)
export JMETER_HOME=/path/to/apache-jmeter-x.x.x
export PATH=$PATH:$JMETER_HOME/bin

# Windows
# Ajouter à Propriétés système -> Variables d'environnement -> PATH
```

### 4. Vérifier l'installation

Exécutez la commande suivante pour vérifier que JMeter est correctement installé :

```bash
# Linux/Mac
./jmeter.sh --version

# Windows
jmeter.bat --version
```

### 5. Lancer l'interface GUI JMeter

```bash
# Linux/Mac
./jmeter.sh

# Windows
jmeter.bat
```

## Structure des répertoires

```
apache-jmeter/
├── bin/          # Scripts exécutables et exécutable JMeter
├── lib/          # Bibliothèques JMeter
├── lib/ext/      # Plugins et extensions
├── docs/         # Documentation
├── printable_docs/  # Manuel utilisateur
└── LICENSE       # Fichier de licence
```

## Exécution en différents modes

### Mode GUI (Interactif)
```bash
jmeter.sh          # Linux/Mac
jmeter.bat         # Windows
```

### Mode CLI (Headless)
```bash
jmeter -n -t testplan.jmx -l results.jtl -e -o html-report
```

Paramètres :
- `-n` : Exécuter en mode non-GUI
- `-t` : Fichier du plan de test
- `-l` : Fichier des résultats
- `-e` : Générer un rapport HTML après le test
- `-o` : Répertoire de sortie pour le rapport HTML

## Dépannage d'installation

### Java non trouvé
Assurez-vous que JAVA_HOME est correctement défini :
```bash
export JAVA_HOME=/path/to/jdk
```

### Problèmes de mémoire
Modifiez le script de démarrage de jmeter pour augmenter la taille du heap :
```bash
HEAP="-Xms1g -Xmx4g"
```
