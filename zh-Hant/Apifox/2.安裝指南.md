# 安裝指南

本文說明 Apifox 的多種安裝方式，幫助你快速在本地開始使用 Apifox。

## 環境需求

Apifox 支援以下作業系統：

- Windows 7 及以上版本（64位元）
- macOS 10.13 及以上版本
- Linux（Debian/Ubuntu 20.04+、Fedora 36+）

## 下載方式

### 官網下載

造訪 Apifox 官網 https://apifox.com/，點擊「下載」按鈕，選擇對應作業系統的版本進行下載。

### 版本選擇

Apifox 提供兩個版本：

- **Apifox**（標準版）：包含所有核心功能，適合大多數使用者
- **Apifox**（付費版）：包含更多進階功能，如團隊管理、高階自動化等

標準版就已經足夠使用，基礎功能都很齊全。

## 安裝步驟

### Windows 安裝

1. 下載 Windows 安裝套件（.exe 檔案）
2. 雙擊執行安裝程式
3. 選擇安裝路徑（建議使用預設路徑）
4. 等待安裝完成
5. 啟動 Apifox，登入或註冊帳號

### macOS 安裝

1. 下載 macOS 安裝套件（.dmg 檔案）
2. 開啟安裝套件，將 Apifox 拖入 Applications 資料夾
3. 啟動 Apifox，登入或註冊帳號

首次啟動時，可能需要在系統偏好設定中允許執行來自未知開發者的應用程式。

### Linux 安裝

**Debian/Ubuntu：**

```bash
# 下載 deb 套件
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.deb

# 安裝
sudo dpkg -i Apifox-latest-linux.x64.deb

# 如果缺少依賴，修復安裝
sudo apt-get install -f
```

**Fedora：**

```bash
# 下載 rpm 套件
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.rpm

# 安裝
sudo rpm -i Apifox-latest-linux.x64.rpm
```

## 帳號與登入

### 註冊帳號

首次使用需要註冊帳號：

1. 啟動 Apifox
2. 點擊「註冊」按鈕
3. 輸入電子郵件和密碼完成註冊
4. 電子郵件驗證（如果需要）

### 登入

支援多種登入方式：

- 電子郵件密碼登入
- 第三方帳號登入（GitHub、Google 等）

### 雲端同步

登入帳號後，專案資料會自動同步到雲端。可以在不同裝置間無縫切換，資料即時保持一致。

## 初始設定

首次啟動後，建議進行以下設定：

### 設定主題

在「設定」中可以選擇淺色或深色主題，依個人喜好選擇。

### 設定代理

如果需要透過代理存取網路，在「設定」→「網路」中設定代理伺服器。

### 匯入匯出

- **匯入**：支援匯入 Postman、Swagger、Har 等格式的專案
- **匯出**：支援匯出為 OpenAPI、Markdown、HTML 等格式

## 快速開始

### 建立專案

1. 點擊左側「專案」旁邊的「+」
2. 輸入專案名稱
3. 選擇專案類型（API 專案或 Mock 專案）

### 建立介面

1. 在專案中點擊「+」選擇「新建介面」
2. 填寫介面基本資訊：
   - 介面名稱
   - 請求路徑
   - 請求方法（GET/POST/PUT/DELETE 等）
3. 填寫請求參數、主體、回應等資訊
4. 儲存介面

### 調試介面

1. 開啟已建立的介面
2. 點擊「傳送」按鈕
3. 查看回應結果

### 使用 Mock

1. 在介面中定義資料結構
2. 系統會自動產生 Mock 位址
3. 存取 Mock 位址即可取得模擬資料

## 解除安裝

### Windows 解除安裝

- 開啟「控制台」→「程式和功能」
- 找到 Apifox，點擊「解除安裝」

### macOS 解除安裝

- 將 Applications 資料夾中的 Apifox 拖到垃圾桶
- 清空垃圾桶

### Linux 解除安裝

```bash
# Debian/Ubuntu
sudo apt-get remove apifox

# Fedora
sudo rpm -e apifox
```
