# Häufig gestellte Fragen

## Grundlagen

### Was ist OpenClaw?

Tool das Messaging-Apps mit KI verbindet. Nutze KI über WhatsApp, Telegram, etc.

### Kosten?

Software ist kostenlos (Open Source). Kosten: Server + KI-API.

### Wo läuft es?

- Desktop: macOS, Linux, Windows
- Mobil: iOS, Android
- Chat: WhatsApp, Telegram, Discord, +20 mehr

## Installation

### Installation fehlgeschlagen?

1. `node --version` (braucht 22+)
2. `openclaw doctor`
3. `openclaw logs`

## Konfiguration

### Wo ist Config?

`~/.openclaw/openclaw.json`

### Zugriff kontrollieren?

`dmPolicy`: pairing, allowlist, open, disabled

## Nutzung

### Kanal hinzufügen?

```bash
openclaw channels login whatsapp
```

### Mehrere Kanäle?

Ja, ein Gateway serves mehrere.
