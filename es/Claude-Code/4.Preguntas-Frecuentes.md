# Claude Code FAQ

Preguntas frecuentes.

## Instalación

### Q: ¿La instalación falló?

**Soluciones**:
1. Verifica los requisitos del sistema
2. Actualiza VS Code a la última versión
3. Verifica la conexión de red
4. Intenta reinstallar

### Q: ¿No puedo iniciar sesión?

**Soluciones**:
1. Verifica si la clave API es válida
2. Verifica la conexión de red
3. Intenta borrar la caché

### Q: ¿La extensión no aparece?

**Soluciones**:
1. Reinicia VS Code
2. Verifica si la extensión está habilitada
3. Revisa las notificaciones de VS Code

## Uso

### Q: ¿Respuesta lenta?

**Soluciones**:
1. Verifica la conexión de red
2. Reduce la longitud del contexto
3. Espera la respuesta del servidor

### Q: ¿Mala calidad de generación de código?

**Soluciones**:
1. Sé más específico en las descripciones
2. Proporciona más contexto
3. Itera y mejora

### Q: ¿Cómo cambiar de modelo?

Selecciona la versión del modelo en configuración.

### Q: ¿Qué idiomas soporta?

Soporta lenguajes principales: Python, JavaScript, TypeScript, Go, Rust, Java, C++ y más.

## Cuenta

### Q: ¿Cómo obtener una clave API?

Visita https://console.anthropic.com para crear una.

### Q: ¿Se agotó la cuota?

Compra cuota adicional en la consola.

## Seguridad

### Q: ¿El código se filtrará?

Claude Code no guarda ni comparte tu código.

### Q: ¿Cómo proteger la privacidad?

- No ingreses información sensible en conversaciones
- Usa el modo local
- Borra las sesiones regularmente
