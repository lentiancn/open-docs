# 常見問題

本文彙整了 Apifox 使用過程中常見的技術問題和解答。

## 基礎問題

### 什麼是 Apifox？

Apifox 是一款 API 協作平台，整合 API 設計、文件管理、調試、Mock、自動化測試等功能。它可以幫助開發團隊更有效率地進行 API 開發、測試和文件維護。

### Apifox 是免費的嗎？

Apifox 提供免費版和付費版。免費版已經包含所有核心功能，足以滿足大多數開發需求。付費版提供更多進階功能，如更大的團隊規模、高階自動化等。

### Apifox 和 Postman 有什麼差別？

Apifox 和 Postman 都是 API 工具，但 Apifox 更適合國內使用者：更完善的中文支援、更智慧的 Mock 功能、更順暢的團隊協作體驗。Apifox 基礎功能完全免費，而 Postman 很多功能需要付費。

## 安裝問題

### 下載速度慢怎麼辦？

Apifox 官方提供多個下載節點，可以嘗試造訪 https://apifox.com/ 選擇離你最近的節點下載。

### 無法啟動怎麼辦？

1. 檢查系統版本是否滿足需求
2. 嘗試以管理員身份執行
3. 檢查是否安裝了必要的執行庫
4. 查看錯誤日誌取得更多資訊

### 登入失敗怎麼辦？

1. 檢查網路連線是否正常
2. 確認帳號密碼是否正確
3. 嘗試使用第三方帳號登入
4. 聯繫官方客服取得協助

## 使用問題

### 介面資料不同步？

1. 確認已登入帳號
2. 檢查網路連線
3. 點擊「同步」按鈕手動同步
4. 檢查帳號權限

### Mock 資料不生效？

1. 確認 Mock 功能已開啟
2. 檢查 Mock 位址是否正確
3. 查看介面是否有語法錯誤
4. 嘗試重新整理頁面

### 請求傳送失敗？

1. 檢查網路連線
2. 確認請求位址正確
3. 檢查請求參數格式
4. 查看是否被代理攔截

### 回應顯示亂碼？

1. 嘗試變更回應編碼
2. 檢查回應頭中的編碼設定
3. 使用「格式化」功能重新解析

## 功能問題

### 如何匯入 Postman 資料？

1. 點擊「匯入」
2. 選擇「Postman」
3. 選擇要匯入的檔案
4. 確認匯入

### 如何匯出為 Swagger？

1. 進入專案設定
2. 選擇「匯出」
3. 選擇「Swagger/OpenAPI」
4. 選擇版本
5. 點擊匯出

### 如何使用環境變數？

1. 在專案設定中定義環境變數
2. 在請求中使用 {{變數名}} 引用
3. 切換頂部環境選擇器切換環境

### 如何批次操作介面？

1. 按住 Ctrl/Cmd 選擇多個介面
2. 右鍵選擇批次操作
3. 選擇操作類型（刪除、匯出、移動等）

## 測試問題

### 自動化測試如何執行？

1. 開啟測試套件
2. 點擊「執行」按鈕
3. 查看測試結果

### 判斷失敗怎麼辦？

1. 查看失敗原因
2. 檢查預期值和實際值
3. 修改判斷條件
4. 重新執行測試

### 測試報告如何匯出？

1. 執行測試完成後
2. 點擊「匯出報告」
3. 選擇格式（HTML、PDF、JSON）

## 團隊協作問題

### 如何邀請團隊成員？

1. 進入專案設定
2. 點擊「成員管理」
3. 點擊「邀請成員」
4. 輸入成員電子郵件或使用者名稱

### 成員權限如何設定？

邀請成員時，可以設定不同角色：

- 管理員：完全控制
- 編輯者：可以編輯和執行
- 檢視者：唯讀權限

### 如何解決衝突？

當多人同時編輯同一介面時：

1. 系統會提示衝突
2. 選擇保留本地版本或遠端版本
3. 或者手動合併修改

## 資料問題

### 資料如何備份？

1. 定期匯出專案為 Apifox 格式
2. 或者使用雲端同步功能

### 資料如何還原？

1. 如果有備份檔案，點擊「匯入」
2. 選擇備份檔案進行還原

### 刪除的資料能復原嗎？

刪除的資料會在回收筒中保留 30 天，可以從回收筒還原。

## 效能問題

### 執行卡頓怎麼辦？

1. 關閉不需要的專案
2. 清理瀏覽器快取
3. 升級硬體配備
4. 嘗試最新版本

### 大量資料匯入慢？

1. 分批匯入
2. 關閉自動同步
3. 耐心等待匯入完成
