# HeyGen 使用指南

本指南介紹如何使用 HeyGen 進行 AI 視頻生成。

## 入門

### 使用網頁界面

1. 登錄 HeyGen：https://www.heygen.com
2. 從庫中選擇頭像或上傳自定義頭像
3. 選擇輸入類型：文本、音頻或視頻
4. 生成視頻並下載

### 使用 API

```python
from heygen import HeyGen

heygen = HeyGen(api_key="your_api_key")

# 從文本生成視頻
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="Hello, this is my first AI video!",
    voice_id="voice_id"
)

print(f"視頻 URL: {video.url}")
```

## 核心功能

### 1. 頭像選擇

```python
# 列出可用頭像
avatars = heygen.list_avatars()
for avatar in avatars:
    print(f"{avatar.id}: {avatar.name}")
```

### 2. 語音選擇

```python
# 列出可用語音
voices = heygen.list_voices()
for voice in voices:
    print(f"{voice.id}: {voice.name} ({voice.language})")
```

### 3. 視頻生成

#### 從文本

```python
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="您的腳本文本",
    voice_id="voice_id",
    background="office"
)
```

## 參數參考

### generate_video 參數

| 參數 | 類型 | 必需 | 描述 |
|------|------|------|------|
| avatar_id | string | 是 | 要使用的頭像 ID |
| script | string | 是 | 要說的文本 |
| voice_id | string | 是 | 語音 ID |
| background | string | 否 | 背景場景 |
| speed | float | 否 | 語速 (0.5-2.0) |
| emotion | string | 否 | 情感風格 |

## 用例

### 1. 營銷視頻

```python
# 創建產品公告
video = heygen.generate_video(
    avatar_id="professional_avatar",
    script="介紹我們的新產品...",
    voice_id="professional_voice"
)
```

### 2. 培訓視頻

```python
# 創建培訓內容
video = heygen.generate_video(
    avatar_id="instructor_avatar",
    script="歡迎參加培訓模塊...",
    voice_id="instructor_voice"
)
```
