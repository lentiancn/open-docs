# OpenFeign Guía de Instalación

## Prerrequisitos

- Java Development Kit (JDK) 8 o superior
- Herramienta de construcción Maven o Gradle
- IDE (IntelliJ IDEA, Eclipse o VS Code)

---

## Usando Maven

Agregue la dependencia OpenFeign a su `pom.xml`:

```xml
<dependencies>
    <!-- OpenFeign Core -->
    <dependency>
        <groupId>io.github.openfeign</groupId>
        <artifactId>feign-core</artifactId>
        <version>13.2.1</version>
    </dependency>
    
    <!-- OkHttp Client (opcional pero recomendado) -->
    <dependency>
        <groupId>io.github.openfeign</groupId>
        <artifactId>feign-okhttp</artifactId>
        <version>13.2.1</version>
    </dependency>
</dependencies>
```

---

## Usando Gradle

Agregue dependencias a su `build.gradle`:

```groovy
dependencies {
    implementation 'io.github.openfeign:feign-core:13.2.1'
    implementation 'io.github.openfeign:feign-okhttp:13.2.1'
}
```

---

## Integración con Spring Cloud

Para aplicaciones Spring Boot:

```xml
<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-openfeign</artifactId>
    <version>4.1.0</version>
</dependency>
```

Habilite los clientes Feign en su aplicación:

```java
@SpringBootApplication
@EnableFeignClients
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
```

---

## Verificar Instalación

Cree un cliente Feign simple para verificar:

```java
@FeignClient(name = "example-service", url = "https://api.example.com")
public interface ExampleClient {
    
    @RequestMapping(method = RequestMethod.GET, value = "/users/{id}")
    User getUser(@PathVariable("id") Long id);
}
```
