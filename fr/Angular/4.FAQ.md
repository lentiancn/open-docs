# FAQ Angular

> Questions fréquentes

---

## Installation

### Échec de l'installation CLI?

```bash
npm cache clean --force
npm install -g @angular/cli
```

### ng non trouvé?

```bash
npm install -g @angular/cli
```

---

## Développement

### Le projet ne démarre pas?

```bash
rm -rf node_modules
npm install
```
