# OpenCode

Agente de codificación AI de código abierto.

## Resumen

OpenCode es un agente de codificación AI de código abierto disponible como interfaz de terminal, aplicación de escritorio o extensión de IDE. Cuenta con LSP habilitado, multisesión y soporte para 75+ proveedores LLM.

### Características principales

- **LSP habilitado** - Carga automáticamente los LSP correctos
- **Multisesión** - Inicia múltiples agentes en paralelo
- **Cualquier modelo** - 75+ proveedores LLM
- **Privacidad primero** - No almacena código ni datos de contexto

### Estadísticas

- GitHub Stars: 100,000+
- Contribuidores: 700+
- Desarrolladores activos mensuales: 2,500,000+

## Inicio rápido

### Instalar

```bash
curl -fsSL https://opencode.ai/install | bash
```

### Configurar

```bash
opencode
/connect
```

Visita opencode.ai/auth para obtener la clave API.

### Inicializar

```bash
/init
```
