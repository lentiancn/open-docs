# Preguntas Frecuentes

Aquí encontrarás respuestas a las dudas técnicas más comunes que surgen al usar Bootstrap.

## Preguntas Básicas

### ¿Qué es Bootstrap?

Bootstrap es un framework de desarrollo front-end de código abierto, creado y mantenido por Twitter. Proporciona una amplia variedad de estilos CSS y componentes de JavaScript que ayudan a los desarrolladores a crear aplicaciones web responsivas de forma rápida.

### ¿Cuál es la diferencia entre Bootstrap 5 y Bootstrap 4?

Las diferencias principales son:

- Bootstrap 5 eliminó la dependencia de jQuery y usa JavaScript nativo
- Se añadió el punto de rotura XXL
- Se mejoraron los controles de formulario
- Se eliminó Font Awesome en favor de Bootstrap Icons
- Ya no es compatible con Internet Explorer

### ¿Qué entorno necesito para Bootstrap?

Solo necesitas un navegador moderno. Bootstrap 5 requiere un navegador que soporte ES6.

## Problemas de Instalación

### ¿Qué hago si el CDN no carga?

1. Verifica tu conexión a internet
2. Prueba otro CDN (como bootcdn o staticfile)
3. Descárgalo y úsalo localmente

### ¿La instalación con npm falla?

```bash
# Limpiar la caché
npm cache clean --force

# Usar el mirror de Taobao
npm config set registry https://registry.npmmirror.com

# Reinstalar
npm install bootstrap@5
```

### ¿Los estilos no se aplican?

1. Verifica que el archivo CSS esté correctamente incluido
2. Limpia la caché del navegador (recarga forzada con Ctrl+Shift+R)
3. Asegúrate de que no haya conflictos con otros CSS

## Problemas de Uso

### ¿El sistema de cuadrícula no funciona?

1. Asegúrate de que row esté dentro de container
2. Verifica que col esté dentro de row
3. Confirma que estás usando las clases de punto de rotura correctas

### ¿El diseño responsivo no funciona?

Asegúrate de incluir la etiqueta viewport:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### ¿La barra de navegación no se colapsa en móvil?

1. Verifica que hayas incluido bootstrap.bundle.min.js (que incluye Popper)
2. Revisa los atributos data-bs-toggle y data-bs-target

### ¿El modal no se abre?

1. Asegúrate de incluir el JS de Bootstrap
2. Verifica los atributos data-bs-toggle y data-bs-target del botón
3. Confirma que el ID del modal sea correcto

### ¿El menú desplegable no funciona?

1. Incluye bootstrap.bundle.min.js
2. Revisa que tengas el atributo data-bs-toggle="dropdown"

## Problemas de Estilos

### ¿Cómo cambio los colores del tema?

Usa Sass:

```scss
$primary: #007bff;
@import "bootstrap/scss/bootstrap";
```

### ¿Cómo sobrescribo estilos predeterminados?

Usa selectores con mayor prioridad:

```css
.navbar {
    background-color: #f8f9fa !important;
}
```

### ¿Cómo cambio la fuente?

```scss
$font-family-sans-serif: "Helvetica Neue", Helvetica, Arial, sans-serif;
@import "bootstrap/scss/bootstrap";
```

### ¿Los iconos no se muestran?

Bootstrap 5 usa Bootstrap Icons:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.10.0/font/bootstrap-icons.css">

<i class="bi bi-alarm"></i>
```

## Problemas con Componentes

### ¿Cómo centrar una tarjeta?

```html
<div class="card">
    <div class="card-body">
        Contenido
    </div>
</div>
```

Usa flex:

```html
<div class="card">
    <div class="card-body d-flex justify-content-center align-items-center">
        Contenido centrado
    </div>
</div>
```

### ¿La validación de formularios no funciona?

Bootstrap 5 usa el nuevo sistema de validación de formularios:

```html
<form class="row g-3 needs-validation" novalidate>
    <div class="col-md-4">
        <label for="validationCustom01" class="form-label">Nombre</label>
        <input type="text" class="form-control" id="validationCustom01" required>
        <div class="valid-feedback">¡Correcto!</div>
        <div class="invalid-feedback">Por favor ingresa el nombre</div>
    </div>
    <div class="col-12">
        <button class="btn btn-primary" type="submit">Enviar</button>
    </div>
</form>

<script>
    (function () {
        'use strict'
        var forms = document.querySelectorAll('.needs-validation')
        Array.prototype.slice.call(forms).forEach(function (form) {
            form.addEventListener('submit', function (event) {
                if (!form.checkValidity()) {
                    event.preventDefault()
                    event.stopPropagation()
                }
                form.classList.add('was-validated')
            }, false)
        })
    })()
</script>
```

### ¿Cómo cerrar el modal al hacer clic en el fondo?

Establece el atributo:

```html
<div class="modal" data-bs-backdrop="static">
```

`static` - hacer clic en el fondo no cierra
`true` - hacer clic cierra
`false` - hacer clic no cierra

## Problemas de Rendimiento

### ¿Cómo reduzco el tamaño de los archivos?

1. Importa solo los componentes que necesitas (usando Sass)
2. Usa la versión de producción
3. Activa la compresión Gzip

### ¿Cómo importo solo lo que necesito?

Usa Sass para importar solo los componentes requeridos:

```scss
// Importar funciones y variables necesarias
@import "bootstrap/scss/functions";
@import "bootstrap/scss/variables";

// Importar solo los componentes que usas
@import "bootstrap/scss/root";
@import "bootstrap/scss/reboot";
@import "bootstrap/scss/containers";
@import "bootstrap/scss/grid";
@import "bootstrap/scss/buttons";
@import "bootstrap/scss/forms";
```

## Problemas de Compatibilidad

### ¿No es compatible con IE?

Bootstrap 5 ya no es compatible con Internet Explorer. Si necesitas soporte para IE11, usa Bootstrap 4.

### ¿Retraso al hacer clic en móvil?

Bootstrap 5 usa touch-action: manipulation para eliminar el retraso de 300ms en clics.
