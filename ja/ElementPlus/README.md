# Element Plus

Element Plusは、開発者、デザイナー、プロダクトマネージャーのためのVue 3コンポーネントライブラリです。

## はじめに

Element Plusは、Vue 3向けに構築されたUIコンポーネントライブラリで、最新のWebアプリケーションに必要なコンポーネントのセットを提供します。全面的に設計され、機能が豊富なコンポーネントのセットを提供します。

## 特徴

- **豊富なコンポーネント**: ボタン、フォーム、テーブル、ダイアログ、ナビゲーション
- **TypeScriptサポート**: 完全なTypeScript型定義
- **ツリーシェイキング**: 必要なものだけを導入
- **テーマカスタマイズ**: CSS変数とSCSSサポート
- **国際化**: 10以上の言語をサポート
- **アクセシビリティ**: WCAG準拠

## バージョン情報

| バージョン | 状態 | Vueバージョン |
|---------|--------|-------------|
| 2.x | ✅ 現行 | Vue 3 |
| 1.x | ⚠️ メンテナンス中 | Vue 3 |
| Element UI | ❌ レガシー | Vue 2 |

> **注意**: Element PlusはVue 3のみです。Vue 2プロジェクトにはElement UIを使用してください。

## クイックスタート

### npm

```bash
npm install element-plus
```

### CDN

```html
<link rel="stylesheet" href="https://unpkg.com/element-plus/dist/index.css">
<script src="https://unpkg.com/element-plus/dist/index.full.js"></script>
```
