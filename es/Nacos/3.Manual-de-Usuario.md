# Manual de Usuario de Nacos

## Descubrimiento de servicios

### Registrar servicio

```java
NamingService namingService = NacosFactory.createNamingService("127.0.0.1:8848");
namingService.registerInstance("mi-servicio", "192.168.1.10", 8080);
```

### Descubrir servicios

```java
List<Instance> instances = namingService.selectInstances("mi-servicio", true);
```

## Gestión de configuración

### Crear configuración

```bash
curl -X POST 'http://127.0.0.1:8848/nacos/v1/cs/configs' \
  -d 'dataId=app-config' \
  -d 'group=DEFAULT_GROUP' \
  -d 'content=clave=valor'
```

### Obtener configuración

```bash
curl 'http://127.0.0.1:8848/nacos/v1/cs/configs?dataId=app-config&group=DEFAULT_GROUP'
```

## Namespace

### Crear namespace

```bash
curl -X POST 'http://127.0.0.1:/v1/console8848/nacos/namespaces' \
  -d 'customNamespaceId=dev' \
  -d 'namespaceName=Desarrollo'
```

## Mejores prácticas

1. Use nombres de servicio significativos
2. Organice configuraciones con grupos
3. Habilite autenticación en producción
4. Monitoree la salud del clúster
