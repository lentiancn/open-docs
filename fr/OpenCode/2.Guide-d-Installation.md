# OpenCode Guide d'installation

OpenCode supporte plusieurs méthodes d'installation.

## Installation Linux/macOS

### Recommandé : Script d'installation

```bash
curl -fsSL https://opencode.ai/install | bash
```

### Node.js

```bash
npm install -g opencode-ai
```

### Bun

```bash
bun install -g opencode-ai
```

### pnpm

```bash
pnpm install -g opencode-ai
```

### Yarn

```bash
yarn global add opencode-ai
```

### Homebrew (macOS/Linux)

```bash
brew install anomalyco/tap/opencode
```

### Arch Linux

```bash
# Stable
sudo pacman -S opencode

# Dernière (AUR)
paru -S opencode-bin
```

## Installation Windows

### Chocolatey

```bash
choco install opencode
```

### Scoop

```bash
scoop install opencode
```

### NPM

```bash
npm install -g opencode-ai
```

### Mise

```bash
mise use -g github:anomalyco/opencode
```

### Docker

```bash
docker run -it --rm ghcr.io/anomalyco/opencode
```

### Téléchargement manuel

Vous pouvez aussi télécharger le binaire depuis [GitHub Releases](https://github.com/anomalyco/opencode/releases).

## Vérifier l'installation

```bash
opencode --version
```

## Configuration

### Connecter un fournisseur LLM

Exécutez pour connecter :

```bash
opencode
/connect
```

Sélectionnez `opencode`, puis visitez [opencode.ai/auth](https://opencode.ai/auth) pour vous connecter et obtenir votre clé API.

### Autres fournisseurs

Vous pouvez aussi sélectionner d'autres fournisseurs LLM. Consultez les [docs officiels](https://opencode.ai/docs/providers#directory) pour les détails.

## Initialiser le projet

```bash
cd /path/to/project
opencode
/init
```

Ceci permettra à OpenCode d'analyser votre projet et de créer un fichier AGENTS.md à la racine du projet.
