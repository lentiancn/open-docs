# OpenClaw 使用手冊

> 使用 OpenClaw 的詳細指南

---

## 開始使用

### 基本用法

OpenClaw 可以有多種使用方式：

#### 互動式工作階段

```bash
# 啟動互動式對話
openclaw chat
```

#### 命令執行

```bash
# 執行單一命令
openclaw exec "幫我建立檔案"

# 執行更複雜的任務
openclaw exec --task "研究最新新聞"
```

### Gateway 管理

大多數操作需要 Gateway 服務處於啟用狀態：

```bash
# 啟動 Gateway
openclaw gateway start

# 檢查狀態
openclaw gateway status

# 停止 Gateway
openclaw gateway stop
```

## 檔案管理

### 讀取檔案

```bash
# 使用讀取工具
read /path/to/file.txt
```

### 寫入檔案

```bash
# 建立新檔案或覆寫
write --content "檔案內容" --path /path/to/file.txt
```

### 編輯檔案

```bash
# 檔案中的精確文字替換
edit --path /path/to/file.txt --oldText "舊文字" --newText "新文字"
```

## 瀏覽器控制

OpenClaw 可以自動化網頁瀏覽器：

### 啟動瀏覽器

```bash
# 啟動隔離的瀏覽器
browser start --profile openclaw

# 使用 Chrome 擴充功能
browser start --profile chrome
```

### 瀏覽器操作

```bash
# 導航至 URL
browser navigate --url https://example.com

# 截圖
browser screenshot --path /output/screenshot.png

# 點擊元素
browser click --selector "#button-id"
```

## 網路存取

### 網路搜尋

```bash
# 搜尋網路
web_search --query "最新新聞" --count 5
```

### 取得網頁

```bash
# 擷取網頁內容
web_fetch --url https://example.com --maxChars 5000
```

## 子代理程式

### 建立子代理程式

```bash
# 為特定任務啟動新子代理程式
subagents create --task "文件收集"

# 引導子代理程式
subagents steer --id agent-123 --message "下一個任務"

# 終止子代理程式
subagents kill --id agent-123

## 配置與個人化

### 自訂個性

編輯 `SOOL.md` 來自訂助理的個性：

```markdown
# SOUL.md - 你是誰

## 核心真理

- 真誠地提供幫助
- 有自己的觀點
- 善於利用資源
```

### 配置工具

編輯 `TOOLS.md` 進行特定工具設定：

```markdown
# TOOLS.md - 本地筆記

## SSH 連線
- server1 -> 192.168.1.100
```

## 最佳實踐

1. **工作區**：將檔案保持在定義的工作區中
2. **安全**：小心外部操作
3. **記憶體**：使用記憶體檔案儲存重要資訊
4. **Gateway**：保持 Gateway 服務啟用以獲得快速回應時間

## 進階功能

### 自動化

建立腳本以自動化重複任務：

```bash
# 自動化資料收集
openclaw exec --task "收集主題 X 的資訊"
```

### Cron 工作

設定定期任務：

```bash
# 建立排程任務
openclaw cron --schedule "0 9 * * *" --task "早晨簡報"
```
