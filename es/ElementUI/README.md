# Element UI

Element UI es una biblioteca de componentes de interfaz de usuario basada en Vue 2.0 que proporciona a los desarrolladores soluciones completas de componentes para aplicaciones de escritorio.

## Descripción General

Element UI es una biblioteca de componentes Vue 2.0 desarrollada y mantenida por el equipo de Ele.me.

## Características Principales

- Soporte Completo para Vue 2.0: Construido sobre Vue 2.0
- Más de 70 Componentes: Cubre todos los aspectos del desarrollo de escritorio
- Importación bajo Demanda: Usando babel-plugin-component
- Internacionalización: Soporte de múltiples idiomas
- Personalización de Temas: Configuración de temas flexible

## Inicio Rápido

### Instalación

npm install element-ui --save

### Uso

Importación completa:

import Vue from 'vue'
import Element from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(Element)

Importación bajo demanda (recomendado):

npm install -D babel-plugin-component
