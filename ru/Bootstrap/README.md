# Bootstrap

Bootstrap — самый популярный в мире открытый фреймворк для разработки интерфейсов на HTML, CSS и JavaScript. Создан и поддерживается компанией Twitter. Включает богатый набор компонентов и мощных JavaScript-плагинов для быстрого создания адаптивных веб-приложений.

## Быстрый старт

### Подключение через CDN

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### Установка через npm

```bash
npm install bootstrap@5
```

## Основные возможности

- **Адаптивная вёрстка**: 12-колоночная сетка для любых экранов
- **Готовые компоненты**: навигационные панели, карточки, формы, модальные окна
- **JavaScript-плагины**: выпадающие списки, карусели, аккордеоны
- **Поддержка Sass**: лёгкая настройка темы

## Основные классы

| Класс | Назначение |
|------|------------|
| container | Контейнер фиксированной ширины |
| container-fluid | Контейнер на всю ширину |
| row | Строка |
| col | Колонка |
| btn | Кнопка |
| card | Карточка |
| navbar | Навигационная панель |
| modal | Модальное окно |
