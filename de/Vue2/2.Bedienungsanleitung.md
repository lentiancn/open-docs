# Vue 2 Bedienungsanleitung

Anleitung zur effektiven Nutzung von Vue 2.

---

## Grundkonzepte

Vue 2 ist ein JavaScript-Framework zum Erstellen von Benutzeroberflachen.

---

## Instanz

```javascript
new Vue({
  el: '#app',
  data: {
    message: 'Hello Vue!'
  }
})
```

---

## Direktiven

| Direktive | Beschreibung |
|-----------|--------------|
| `v-if` | Bedingtes Rendern |
| `v-for` | Listen-Rendern |
| `v-bind` | Attributbindung |
| `v-model` | Zwei-Wege-Bindung |

---

## Komponenten

```vue
<template>
  <div>{{ title }}</div>
</template>

<script>
export default {
  props: {
    title: String
  }
}
</script>
```
