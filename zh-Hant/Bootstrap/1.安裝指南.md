# Bootstrap 安裝指南

本指南詳細介紹 Bootstrap 在所有主要作業系統上的安裝方法。

## 簡介

Bootstrap 是全球最受歡迎的前端開源工具包，用於快速開發響應式網站和 Web 應用程序。

## 系統需求

### 瀏覽器支援
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## 安裝方法

### 使用 CDN（推薦）

```html
<!DOCTYPE html>
<html lang="zh-Hant">
<head>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>Hello, Bootstrap!</h1>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### 使用 npm

```bash
npm install bootstrap
```

## 驗證

創建測試 HTML 文件，如果看到藍色按鈕，說明安裝成功。
