# Руководство по установке Angular

Angular — мощный интерфейсный фреймворк, разработанный Google. В этом руководстве описаны способы установки для различных окружений.

## Способы установки

### Способ 1: Angular CLI (Рекомендуется)

#### Установка Angular CLI

```bash
# Глобальная установка
npm install -g @angular/cli

# Проверка установки
ng version
```

#### Создание нового проекта

```bash
# Создать новый проект
ng new my-app

# Создать с параметрами
ng new my-app --routing --style=scss --skip-tests
```

#### Параметры

```bash
# Пропустить установку
ng new my-app --skip-install

# Автономные компоненты (Angular 14+)
ng new my-app --standalone

# Включить SSR
ng new my-app --ssr
```

### Способ 2: Использование npx

```bash
# Создать проект без установки CLI
npx @angular/cli new my-app
```

### Способ 3: Ручная настройка

```bash
# Создать директорию
mkdir my-app
cd my-app

# Инициализировать npm
npm init -y

# Установить Angular
npm install @angular/core @angular/common @angular/platform-browser @angular/platform-browser-dynamic @angular/compiler @angular/forms @angular/router @angular/animations rxjs zone.js

# Установить зависимости для разработки
npm install --save-dev @angular/compiler-cli @angular-devkit/build-angular typescript
```

### Способ 4: Docker

```dockerfile
# Dockerfile
FROM node:20-alpine

WORKDIR /app

# Установить Angular CLI
RUN npm install -g @angular/cli

# Создать проект
RUN ng new my-app --directory .

# Запустить сервер разработки
CMD ["ng", "serve", "--host", "0.0.0.0"]
```

## Настройка окружения

### Управление версиями Node.js

#### Использование nvm (Рекомендуется)

```bash
# Установить nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Установить Node.js
nvm install 20
nvm use 20

# Установить по умолчанию
nvm alias default 20
```

### Настройка npm

```bash
# Установить registry
npm config set registry https://registry.npmjs.org/

# Включить строгий SSL
npm config set strict-ssl true

# Очистить кэш
npm cache clean --force
```

## Проверка

```bash
# Проверить Angular CLI
ng version

# Проверить Node.js
node --version

# Проверить npm
npm --version
```

## Частые проблемы

### Отказано в доступе?

```bash
# Исправить права npm
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
export PATH=~/.npm-global/bin:$PATH
```

### Медленная установка?

```bash
# Использовать npm ci для более быстрой установки
npm ci

# Или использовать yarn
npm install -g yarn
```

## Обновление Angular

```bash
# Обновить Angular CLI
npm install -g @angular/cli@latest

# Обновить проект
ng update @angular/core @angular/cli
```
