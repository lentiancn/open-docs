## 简介

### 什么是 Bootstrap？

Bootstrap 是一个功能强大的前端框架，用于使用 HTML、CSS 和 JavaScript 快速开发响应式网站。

### 核心特性

- **响应式网格系统**：12 列响应式网格
- **预编译 CSS**：丰富的实用工具类
- **组件库**：超过 20 个 UI 组件
- **JavaScript 插件**：模态框、轮播、工具提示等
- **自定义选项**：使用 Sass 变量深度定制

## 布局

### 容器

```html
<!-- 固定宽度容器 -->
<div class="container">
    内容
</div>

<!-- 满宽容器 -->
<div class="container-fluid">
    内容
</div>
```

### 网格系统

```html
<!-- 基本行 -->
<div class="row">
    <div class="col">列</div>
    <div class="col">列</div>
    <div class="col">列</div>
</div>

<!-- 12 列网格 -->
<div class="row">
    <div class="col-12">col-12</div>
</div>
<div class="row">
    <div class="col-6">col-6</div>
    <div class="col-6">col-6</div>
</div>
<div class="row">
    <div class="col-4">col-4</div>
    <div class="col-4">col-4</div>
    <div class="col-4">col-4</div>
</div>
```

### 响应式断点

| 断点 | 类名前缀 | 宽度 |
|------|----------|------|
| 超小 | none | < 576px |
| 小 | sm | ≥ 576px |
| 中 | md | ≥ 768px |
| 大 | lg | ≥ 992px |
| 超大 | xl | ≥ 1200px |
| 特大 | xxl | ≥ 1400px |

### 响应式列

```html
<div class="row">
    <!-- 移动端12列，平板6列，桌面4列 -->
    <div class="col-12 col-sm-6 col-lg-4">响应式列</div>
    <div class="col-12 col-sm-6 col-lg-4">响应式列</div>
    <div class="col-12 col-sm-6 col-lg-4">响应式列</div>
</div>
```

### 列偏移

```html
<div class="row">
    <div class="col-md-4">col-md-4</div>
    <div class="col-md-4 offset-md-4">col-md-4 offset-md-4</div>
</div>
```

### Gutter（间距）

```html
<!-- 默认间距 -->
<div class="row">
    <div class="col-4">列</div>
    <div class="col-4">列</div>
    <div class="col-4">列</div>
</div>

<!-- 无间距 -->
<div class="row g-0">
    <div class="col-4">列</div>
    <div class="col-4">列</div>
    <div class="col-4">列</div>
</div>
```

## 最佳实践

### 项目结构

```
project/
├── index.html
├── css/
│   ├── custom.css
│   └── bootstrap.min.css (可选)
├── js/
│   ├── custom.js
│   └── bootstrap.bundle.min.js
├── images/
│   └── logo.png
└── fonts/
```

### 性能优化

1. **使用 CDN**：提高加载速度
2. **压缩文件**：生产环境使用 .min 文件
3. **按需加载**：只引入需要的组件
4. **图片优化**：使用适当格式和尺寸

### 响应式设计

1. **移动优先**：从最小屏幕开始设计
2. **测试工具**：使用浏览器开发者工具
3. **断点使用**：合理使用响应式类
