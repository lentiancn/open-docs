# GitLab

> 소프트웨어 개발 및 운영을 위한 완전한 DevOps 플랫폼

---

## 개요

GitLab은 소스 코드 관리, 이슈 추적, CI/CD 파이프라인 및 보안 스캐닝을 갖춘 완전한 DevOps 기능을 갖춘 웹 기반 Git 저장소 관리 도구입니다.

## 문서

이 디렉토리에는 한국어로 된 완전한 문서가 포함되어 있습니다:

| 파일 | 설명 |
|------|------|
| 1.개요.md | 소개 및 개요 |
| 2.설치-가이드.md | 설치 및 설정 가이드 |
| 3.사용자-매뉴얼.md | 상세 사용 가이드 |
| 4.FAQ.md | 자주 묻는 질문 |
| README.md | 이 개요 파일 |

## 빠른 시작

### 주요 기능

- 브랜칭 및 태깅을 갖춘 Git 저장소 호스팅
- 이슈 추적 및 프로젝트 관리
- 기본 제공 CI/CD 파이프라인
- 코드 리뷰 및 병합 요청
- 문서화를 위한 위키
- 보안 스캐닝
- 컨테이너 레지스트리

### 기본 워크플로우

1. 프로젝트 만들기
2. 저장소 클론
3. 브랜치 만들고 변경
4. 푸시하고 병합 요청 만들기
5. 리뷰하고 병합
6. CI/CD 자동 배포
