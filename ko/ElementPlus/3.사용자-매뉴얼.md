# Element Plus 사용자 매뉴얼

Element Plus의 주요 기능과 사용 팁.

## 빠른 시작

### 첫 번째 예

```vue
<template>
  <el-button type="primary" @click="handleClick">
    클릭하세요
  </el-button>
</template>

<script setup>
import { ref } from 'vue'

const handleClick = () => {
  ElMessage.success('안녕하세요 Element Plus!')
}
</script>
```

### Form (폼)

```vue
<template>
  <el-form :model="form" :rules="rules" ref="formRef" label-width="120px">
    <el-form-item label="사용자 이름" prop="username">
      <el-input v-model="form.username" />
    </el-form-item>
    
    <el-form-item label="이메일" prop="email">
      <el-input v-model="form.email" />
    </el-form-item>
    
    <el-form-item label="역할" prop="role">
      <el-select v-model="form.role" placeholder="역할 선택">
        <el-option label="관리자" value="admin" />
        <el-option label="사용자" value="user" />
      </el-select>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm">제출</el-button>
      <el-button @click="resetForm">초기화</el-button>
    </el-form-item>
  </el-form>
</template>

<script setup>
import { ref, reactive } from 'vue'

const formRef = ref()
const form = reactive({
  username: '',
  email: '',
  role: '',
})

const rules = {
  username: [{ required: true, message: '이름을 입력하세요', trigger: 'blur' }],
  email: [
    { required: true, message: '이메일을 입력하세요', trigger: 'blur' },
    { type: 'email', message: '올바른 이메일 형식을 입력하세요', trigger: 'blur' }
  ],
  role: [{ required: true, message: '역할을 선택하세요', trigger: 'change' }],
}

const submitForm = async () => {
  const valid = await formRef.value.validate()
  if (valid) {
    console.log('제출 성공', form)
  }
}

const resetForm = () => {
  formRef.value.resetFields()
}
</script>
```

### 테이블 페이지네이션

```vue
<template>
  <el-table :data="paginatedData">
    <el-table-column prop="name" label="이름" />
  </el-table>
  
  <el-pagination
    v-model:current-page="currentPage"
    v-model:page-size="pageSize"
    :total="total"
    :page-sizes="[10, 20, 50, 100]"
    layout="total, sizes, prev, pager, next, jumper"
    @size-change="handleSizeChange"
    @current-change="handleCurrentChange"
  />
</template>

<script setup>
import { ref, computed } from 'vue'

const currentPage = ref(1)
const pageSize = ref(10)
const tableData = ref([/* 많은 데이터 */])
const total = computed(() => tableData.value.length)

const paginatedData = computed(() => {
  const start = (currentPage.value - 1) * pageSize.value
  return tableData.value.slice(start, start + pageSize.value)
})
</script>
```

## 디버깅 팁

### 컴포넌트 문제 확인

```bash
# Element Plus 버전 확인
npm list element-plus
```

### 일반적인 오류

**컴포넌트가 등록되지 않음**: 임포트 또는 자동 임포트 설정 확인

**스타일이 적용되지 않음**: CSS 충돌 확인, `:deep()` 시도

**타입 오류**: `element-plus/global` 타입 설치
