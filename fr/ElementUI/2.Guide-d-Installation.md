# Element UI Guide d'installation

## Exigences

- Node.js: 12.0.0+
- npm: 6.0.0+
- Vue: 2.6.0+

## Installation

```bash
npm install element-ui --save
```

## Démarrage rapide

### Import complet

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```

### Import à la demande

```bash
npm install babel-plugin-component -D
```

Configuration babel.config.js:

```javascript
module.exports = {
  plugins: [
    ['component', {
      libraryName: 'element-ui',
      styleLibraryName: 'theme-chalk'
    }]
  ]
}
```

Importer composants:

```javascript
import { Button, Select } from 'element-ui'
Vue.use(Button)
```
