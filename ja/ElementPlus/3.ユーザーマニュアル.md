# Element Plus ユーザーマニュアル

Element Plus の使い方。

## クイックスタート

```vue
<template>
  <el-button type="primary" @click="handleClick">
    クリック
  </el-button>
</template>

<script setup>
import { ref } from 'vue'

const handleClick = () => {
  ElMessage.success('Hello!')
}
</script>
```

## コンポーネント

### Button

```vue
<el-button>デフォルト</el-button>
<el-button type="primary">プライマリ</el-button>
<el-button disabled>無効</el-button>
<el-button loading>ローディング</el-button>
```

### Input

```vue
<el-input v-model="value" placeholder="入力" />
<el-input v-model="value" disabled placeholder="無効" />
<el-input v-model="value" clearable placeholder="クリア可能" />
```

### Table

```vue
<el-table :data="tableData" stripe border>
  <el-table-column prop="name" label="名前" />
  <el-table-column prop="email" label="メール" />
</el-table>
```

### Form

```vue
<el-form :model="form" :rules="rules" ref="formRef">
  <el-form-item label="名前" prop="name">
    <el-input v-model="form.name" />
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="submit">送信</el-button>
  </el-form-item>
</el-form>
```

### Dialog

```vue
<el-button @click="visible = true">開く</el-button>
<el-dialog v-model="visible" title="タイトル">
  <span>内容</span>
</el-dialog>
```

### Message

```typescript
import { ElMessage } from 'element-plus'

ElMessage.success('成功('エラー')
```
