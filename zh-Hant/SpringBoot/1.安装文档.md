# Spring Boot 安裝指南

本指南說明如何在 Windows、macOS 和 Linux 上安裝和設定 Spring Boot 開發環境。

## 系統需求

### Java 版本需求

| Spring Boot 版本 | 最低 Java 版本 |
|---------------------|---------------------|
| 3.x                 | Java 17+            |
| 2.7.x               | Java 8+             |
| 2.5.x               | Java 8+             |

**注意：** Spring Boot 3.x 需要 Java 17 或更高版本。建議使用 Java 21 (LTS) 以獲得最佳相容性。

## 安裝 Java

### Windows

**選項 1：從 Oracle/OpenJDK 安裝**
1. 從 [Oracle](https://www.oracle.com/java/technologies/downloads/) 或 [Adoptium](https://adoptium.net/) 下載 JDK
2. 執行安裝程式
3. 設定環境變數：
   ```
   JAVA_HOME=C:\Program Files\Java\jdk-21
   PATH=%JAVA_HOME%\bin;%PATH%
   ```

**選項 2：使用 Chocolatey**
```powershell
choco install openjdk21
```

**選項 3：使用 Winget**
```powershell
winget install Oracle.JDK.21
```

驗證：
```cmd
java -version
```

### macOS

**選項 1：使用 Homebrew**
```bash
brew install openjdk@21
sudo ln -sfn $(brew --prefix)/opt/openjdk@21/libexec/openjdk.jdk /Library/Java/JavaVirtualMachines/openjdk-21.jdk
echo 'export JAVA_HOME=$(/usr/libexec/java_home -v 21)' >> ~/.zshrc
source ~/.zshrc
```

**選項 2：使用 SDKMAN**
```bash
curl -s "https://get.sdkman.io" | bash
sdk install java 21.0.2-tem
```

驗證：
```bash
java -version
```

### Linux

**Ubuntu/Debian：**
```bash
sudo apt update
sudo apt install openjdk-21-jdk
export JAVA_HOME=/usr/lib/jvm/java-21-openjdk-amd64
export PATH=$JAVA_HOME/bin:$PATH
```

**RHEL/CentOS/Fedora：**
```bash
sudo dnf install java-21-openjdk-devel
export JAVA_HOME=/usr/lib/jvm/java-21-openjdk
export PATH=$JAVA_HOME/bin:$PATH
```

**使用 SDKMAN：**
```bash
curl -s "https://get.sdkman.io" | bash
source ~/.sdkman/bin/sdkman-init.sh
sdk install java 21.0.2-tem
```

驗證：
```bash
java -version
```

## 安裝 Maven

### Windows

**選項 1：手動安裝**
1. 從 [Apache Maven](https://maven.apache.org/download.cgi) 下載
2. 解壓縮至 `C:\apache-maven`
3. 設定環境變數：
   ```
   MAVEN_HOME=C:\apache-maven
   PATH=%MAVEN_HOME%\bin;%PATH%
   ```

**選項 2：使用 Chocolatey**
```powershell
choco install maven
```

驗證：
```cmd
mvn -version
```

### macOS

```bash
brew install maven
```

驗證：
```bash
mvn -version
```

### Linux

**Ubuntu/Debian：**
```bash
sudo apt install maven
```

**RHEL/CentOS：**
```bash
sudo dnf install maven
```

驗證：
```bash
mvn -version
```

## 安裝 Spring Boot CLI

### 所有平台

**使用 SDKMAN：**
```bash
sdk install springboot
```

**使用 Homebrew (macOS)：**
```bash
brew install spring-boot-cli
```

**手動下載：**
```bash
wget https://repo.spring.io/release/org/springframework/boot/spring-boot-cli/3.2.0/spring-boot-cli-3.2.0-bin.zip
unzip spring-boot-cli-3.2.0-bin.zip
export PATH=$PATH:/path/to/spring-boot-cli-3.2.0/bin
```

驗證：
```bash
spring --version
```

## IDE 設定

### IntelliJ IDEA (推薦)

1. 從 [JetBrains](https://www.jetbrains.com/idea/download/) 下載
2. 安裝並開啟
3. 設定 JDK：`File` → `Project Structure` → `Project SDK`
4. 安裝 Spring Boot Assistant 外掛 (選用)

### VS Code

1. 安裝 [VS Code](https://code.visualstudio.com/)
2. 安裝擴充功能：
   - Extension Pack for Java
   - Spring Boot Extension Pack
3. 設定 Java：`Ctrl+Shift+P` → `Java: Configure Java Runtime`

### Eclipse

1. 下載 [Spring Tools 4 (STS)](https://spring.io/tools)
2. 解壓縮並執行
3. File → New → Spring Starter Project

## 快速開始

### 使用 Spring Initializr

1. 訪問 [start.spring.io](https://start.spring.io/)
2. 選擇選項：
   - Project: Maven / Gradle
   - Language: Java / Kotlin / Groovy
   - Spring Boot Version: 3.2.x (最新)
   - Group: com.example
   - Artifact: myapp
   - Packaging: Jar / War
   - Java Version: 17 / 21
3. 新增依賴：
   - Spring Web
   - Spring Data JPA
   - H2 Database (開發用)
   - Spring Boot DevTools
4. 點擊「Generate」下載

### 使用命令列

```bash
# 建立 Maven 專案
curl https://start.spring.io/starter.zip \
  -d type=maven-project \
  -d language=java \
  -d bootVersion=3.2.5 \
  -d baseDir=myapp \
  -d dependencies=web,jpa,h2,devtools \
  -o myapp.zip

unzip myapp.zip
cd myapp
./mvnw spring-boot:run
```

## 驗證安裝

```bash
# 檢查 Java
java -version

# 檢查 Maven
mvn -version

# 檢查 Gradle (若已安裝)
gradle -version

# 檢查 Spring Boot CLI
spring --version

# 執行範例專案
curl https://start.spring.io/starter.zip -o demo.zip
unzip demo.zip -d demo
cd demo
./mvnw spring-boot:run
```

```bash
docker run -p 8080:8080 springio/gs-spring-boot-docker
```

## 常見問題

### 找不到 Java
- 請確認 JAVA_HOME 設定正確
- 設定環境變數後請重新開啟終端機
- 使用 `echo $JAVA_HOME` (Linux/macOS) 或 `echo %JAVA_HOME%` (Windows) 驗證

### 連接埠 8080 已被佔用
```bash
# Windows
netstat -ano | findstr :8080
taskkill /PID <pid> /F

# Linux/macOS
lsof -i :8080
kill -9 <pid>
```

### Maven 建置失敗
- 清除 Maven 快取：`mvn clean`
- 更新依賴：`mvn dependency:resolve`
- 使用 `./mvnw` (Maven Wrapper) 而非系統 Maven
