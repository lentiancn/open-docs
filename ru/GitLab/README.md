# GitLab

GitLab — полная платформа жизненного цикла DevOps

## Обзор

GitLab — это платформа для разработки программного обеспечения с открытым исходным кодом, предоставляющая полный набор инструментов DevOps от управления кодом до непрерывного развёртывания.

## Функции

- **Репозиторий кода**: Хостинг репозиториев Git, просмотр кода, управление ветками
- **Проверка кода**: Запросы на слияние, комментарии кода, рабочие процессы утверждения
- **CI/CD**: Автоматизированные конвейеры сборки, тестирования и развёртывания
- **Управление проектами**: Отслеживание проблем, доски Канбан, вики
- **Анализ безопасности**: Сканирование образов контейнеров, сканирование зависимостей, статический анализ
- **Инструменты совместной работы**: Группы, шаблоны проектов, управление зависимостями

## Быстрый старт

### Установка

```bash
# Ubuntu/Debian
curl -fsSL https://packages.gitlab.com/install/repositories/gitlab/gitlab-ce/script.deb.sh | sudo bash
sudo EXTERNAL_URL="https://gitlab.example.com" apt-get install gitlab-ce
```

### Настройка SSH

```bash
ssh-keygen -t ed25519 -C "your_email@example.com"
# Добавить открытый ключ в настройки SSH-ключей GitLab
```

### Клонирование проекта

```bash
git clone git@gitlab.example.com:username/project.git
```
