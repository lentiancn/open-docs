# Nacos 使用指南

## 配置管理

```bash
# 发布配置
curl -X POST "http://localhost:8848/nacos/v1/cs/configs?dataId=example.yaml&group=DEFAULT_GROUP&content=key=value"

# 获取配置
curl "http://localhost:8848/nacos/v1/cs/configs?dataId=example.yaml&group=DEFAULT_GROUP"
```

## 服务发现

```bash
# 注册服务
curl -X POST "http://localhost:8848/nacos/v1/ns/instance?serviceName=my-service&ip=192.168.1.100&port=8080"

# 查询服务
curl "http://localhost:8848/nacos/v1/ns/instance/list?serviceName=my-service"
```
