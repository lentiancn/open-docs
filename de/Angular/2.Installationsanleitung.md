# Angular Installationsanleitung

> Einrichtung

---

## Anforderungen

| Umgebung | Anforderung |
|----------|-------------|
| Node.js | 18.10.0+ |
| npm | 9.0.0+ |

---

## Installation

```bash
npm install -g @angular/cli
ng new my-app
ng serve
```

---

## Befehle

| Befehl | Beschreibung |
|--------|--------------|
| ng new | Neues Projekt |
| ng serve | Entwicklungsserver |
| ng build | Build |
