# Element UI Übersicht

Vue.js 2.0 Desktop-UI-Komponentenbibliothek.

## Was ist Element UI?

Element UI ist eine Open-Source-Komponentenbibliothek von Ele.me.

### Hauptmerkmale

- **Reichhaltige Komponenten**: 60+ Komponenten
- **Schönes Design**: Material Design Richtlinien
- **TypeScript Support**: Vollständige Typdefinitionen
- **On-demand Import**: Import bei Bedarf
