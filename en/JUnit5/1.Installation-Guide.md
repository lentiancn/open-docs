# JUnit 5 Installation Guide

Comprehensive guide for installing JUnit 5 on all major platforms.

---

## What is JUnit 5?

JUnit 5 is the next generation of JUnit. It consists of three sub-projects: JUnit Platform, JUnit Jupiter, and JUnit Vintage.

**Note**: JUnit 5 is for Java 8 and above.

---

## Maven

### Add Dependency

```xml
<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter</artifactId>
    <version>5.10.1</version>
    <scope>test</scope>
</dependency>
```

### Full pom.xml

```xml
<project xmlns="http://maven.apache.org/POM/4.0.0"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 http://maven.apache.org/xsd/maven-4.0.0.xsd">
    <modelVersion>4.0.0</modelVersion>

    <groupId>com.example</groupId>
    <artifactId>my-project</artifactId>
    <version>1.0-SNAPSHOT</version>

    <properties>
        <maven.compiler.source>11</maven.compiler.source>
        <maven.compiler.target>11</maven.compiler.target>
        <junit.jupiter.version>5.10.1</junit.jupiter.version>
    </properties>

    <dependencies>
        <dependency>
            <groupId>org.junit.jupiter</groupId>
            <artifactId>junit-jupiter</artifactId>
            <version>${junit.jupiter.version}</version>
            <scope>test</scope>
        </dependency>
    </dependencies>

    <build>
        <plugins>
            <plugin>
                <groupId>org.apache.maven.plugins</groupId>
                <artifactId>maven-surefire-plugin</artifactId>
                <version>3.2.3</version>
            </plugin>
        </plugins>
    </build>
</project>
```

## Gradle

### Add Dependency

```groovy
testImplementation 'org.junit.jupiter:junit-jupiter:5.10.1'
```

### Full build.gradle

```groovy
plugins {
    id 'java'
}

group = 'com.example'
version = '1.0-SNAPSHOT'

repositories {
    mavenCentral()
}

dependencies {
    testImplementation 'org.junit.jupiter:junit-jupiter:5.10.1'
}

test {
    useJUnitPlatform()
}
```

## Eclipse

### Using Eclipse Marketplace

1. Go to **Help > Eclipse Marketplace**
2. Search for **JUnit**
3. Install **JUnit**

JUnit 5 is included in recent Eclipse versions.

---

## IntelliJ IDEA

### Using Maven/Gradle

JUnit 5 is automatically available when you add Maven or Gradle dependency.

### Manual Setup

1. Go to **File > Project Structure > Modules**
2. Add JUnit library to **Dependencies** tab

## Standalone

### Download JAR

```bash
# Download JUnit Jupiter
curl -L -o junit-jupiter-5.10.1.jar https://repo1.maven.org/maven2/org/junit/jupiter/junit-jupiter/5.10.1/junit-jupiter-5.10.1.jar
```

### Compile and Run

```bash
# Compile
javac -cp .:junit-jupiter-5.10.1.jar Test.java

# Run
java -jar junit-platform-console-standalone-1.10.1.jar -cp .
```
