# OpenCode FAQ

Questions fréquemment posées.

## Installation

### Q : L'installation a échoué ?

**Solutions** :
1. Vérifiez les exigences du système (nécessite un émulateur de terminal moderne)
2. Vérifiez la connexion réseau
3. Essayez le script d'installation : `curl -fsSL https://opencode.ai/install | bash`
4. Vérifiez le message d'erreur et réessayez

### Q : Impossible d'installer sur Windows ?

**Solutions** :
1. Essayez Chocolatey : `choco install opencode`
2. Ou utilisez Scoop : `scoop install opencode`
3. Ou utilisez NPM : `npm install -g opencode-ai`

### Q : La vérification de l'installation a échoué ?

**Solutions** :
1. Vérifiez que opencode est dans le PATH
2. Redémarrez le terminal
3. Vérifiez si l'installation a réussi

## Utilisation

### Q : Impossible de se connecter au LLM ?

**Solutions** :
1. Exécutez la commande `/connect`
2. Visitez opencode.ai/auth pour vous connecter
3. Ajoutez vos informations de facturation et obtenez la clé API
4. Collez la clé API dans le terminal

### Q : La réponse est lente ?

**Solutions** :
1. Vérifiez la connexion réseau
2. Essayez de basculer vers un autre fournisseur LLM
3. Réduisez la longueur du contexte de conversation

### Q : Comment changer de modèle ?

Utilisez la commande `/connect` pour sélectionner un autre fournisseur LLM.

### Q : Quels modèles sont supportés ?

Supporte 75+ fournisseurs LLM :
- OpenAI (GPT-4, Claude, etc.)
- Anthropic (Claude)
- Google (Gemini)
- Modèles locaux
- Voir [Models.dev](https://models.dev)

## Fonctionnalités

### Q : Qu'est-ce que AGENTS.md ?

AGENTS.md est un fichier créé lors de l'initialisation d'OpenCode, contenant la structure du projet et les conventions de codage pour aider OpenCode à comprendre le projet.

### Q : Quelle est la différence entre le mode Plan et Build ?

- **Mode Plan** : Génère seulement des plans, ne modifie pas le code
- **Mode Build** : Exécute directement les changements

Appuyez sur **Tab** pour basculer entre les modes.

### Q : Comment partager des sessions ?

Utilisez la commande `/share` pour créer un lien de partage.

## Confidentialité

### Q : OpenCode enregistre-t-il mon code ?

Non. OpenCode n'enregistre pas le code ou les données de contexte, sûr pour les environnements sensibles.

### Q : Comment protéger la confidentialité ?

- N'entrez pas d'informations sensibles dans les conversations
- Utilisez des modèles locaux
- Effacez régulièrement les sessions
