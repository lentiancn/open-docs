# OpenClaw ユーザーマニュアル

このマニュアルではOpenClawの日常的な操作と機能説明します。

## 基本概念

### Gateway

GatewayはOpenClawの核心部分です。すべてのメッセージ転送、AIリクエスト、チャンネル管理はこの一つのプロセスを通じて行われます。Gatewayを一つ実行すれば、自動的にすべてのチャットプラットフォームからの接続を処理します。

### チャンネル

チャンネルはAIアシスタントと通信するために使用するチャットアプリです。WhatsApp、Telegram、Discord、iMessageなど、複数のチャンネルを同時に接続できます。

### エージェント

エージェントはAIアシスタントの独立したインスタンスです。各エージェントは独自のワークスペースと会話履歴を持ちます。異なる目的に応じて複数のエージェントを実行できます。

## 日常操作

### 起動と停止

```bash
# 起動（バックグラウンドで実行）
openclaw gateway start

# 停止
openclaw gateway stop

# 再起動
openclaw gateway restart

# 状態確認
openclaw gateway status

# フォアグラウンドで実行（デバッグ用）
openclaw gateway --port 18789
```

### メッセージ送信

```bash
# ダイレクトメッセージを送信
openclaw message send --target +15555550123 --message "テストメッセージ"

# グループに送信
openclaw message send --target "グループ名" --message "皆さんこんにちは"
```

### 設定管理

OpenClawはJSON5形式の設定ファイルを使用します。保存場所：`~/.openclaw/openclaw.json`

設定を変更する方法：

1. **ウィザードモード**：
   ```bash
   openclaw onboard
   openclaw configure
   ```

2. **コマンドライン**：
   ```bash
   openclaw config get agents.defaults.workspace
   openclaw config set agents.defaults.heartbeat.every "2h"
   openclaw config unset tools.web.search.apiKey
   ```

3. **Webインターフェース**：
   http://127.0.0.1:18789/ を開き、Configタブで設定

4. **直接編集**：
   ```bash
   nano ~/.openclaw/openclaw.json
   ```

### ホットリロード

設定ファイルの変更後，大多数の設定は自動的に適用され、再起動は不要です。

## よく使う設定例

### アクセス制限

```json5
{
  channels: {
    whatsapp: {
      allowFrom: ["+15555550123"]
    }
  }
}
```

### グループチャットの設定

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

### 複数のエージェント

```json5
{
  agents: {
    list: [
      { id: "home", default: true, workspace: "~/.openclaw/workspace-home" },
      { id: "work", workspace: "~/.openclaw/workspace-work" }
    ]
  }
}
```

### 定时任務

```json5
{
  cron: {
    enabled: true,
    maxConcurrentRuns: 2
  }
}
```

### ハートビート（定期チェック）

```json5
{
  agents: {
    defaults: {
      heartbeat: {
        every: "30m",
        target: "last"
      }
    }
  }
}
```

### サンドボックス分離

分離されたコンテナでAI操作を実行する場合：

```json5
{
  agents: {
    defaults: {
      sandbox: {
        mode: "non-main",
        scope: "agent"
      }
    }
  }
}
```

### ウェブフック

```json5
{
  hooks: {
    enabled: true,
    token: "あなたのシークレットトークン",
    path: "/hooks"
  }
}
```

## チャンネル管理

### チャンネルを追加

```bash
# WhatsAppにログイン
openclaw channels login whatsapp

# Telegramにログイン
openclaw channels login telegram
```

### 接続されたチャンネルを表示

```bash
openclaw channels list
```

## トラブルシューティング

### ログを表示

```bash
# サービスのログ
openclaw logs

# リアルタイムでログを監視
openclaw logs -f
```

### ヘルスチェック

```bash
openclaw health
```

### 自動診断

```bash
openclaw doctor
```

## Webコンソール

OpenClawはWebインターフェースで管理できます：

- メッセージの送受信
- 設定の変更
- 会話履歴の表示
- モバイルノードの管理
- サービスの状態監視

アクセス先：http://127.0.0.1:18789/
