# JUnit Manual del usuario

Este manual cubre el uso de JUnit 4 y JUnit 5, incluyendo escritura de pruebas, aserciones, suites de pruebas y otras características principales.

## Uso básico de JUnit 5

### Escribir clases de prueba

```java
import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;

class CalculatorTest {
    
    @Test
    void testAddition() {
        int result = 2 + 3;
        assertEquals(5, result);
    }
}
```

### Aserciones comunes

```java
import static org.junit.jupiter.api.Assertions.*;

// Aserciones de igualdad
assertEquals(expected, actual);
assertEquals(expected, actual, "Mensaje de error personalizado");

// Aserciones booleanas
assertTrue(condition);
assertFalse(condition);

// Aserciones de nulo
assertNull(object);
assertNotNull(object);

// Aserciones de arrays
assertArrayEquals(expectedArray, actualArray);

// Aserciones de excepciones
assertThrows(Exception.class, () -> {
    // Código que puede lanzar excepción
});

// Aserciones combinadas
assertAll("Propiedades de persona",
    () -> assertEquals("John", person.getFirstName()),
    () -> assertEquals("Doe", person.getLastName())
);
```

### Operaciones Before y After

```java
import org.junit.jupiter.api.BeforeEach;
import org.junit.jupiter.api.AfterEach;

class MyTest {
    
    @BeforeEach
    void setUp() {
        // Se ejecuta antes de cada método de prueba
    }
    
    @AfterEach
    void tearDown() {
        // Se ejecuta después de cada método de prueba
    }
}
```

### Pruebas anidadas

```java
import org.junit.jupiter.api.Nested;

@Nested
class NestedTests {
    
    @Test
    void testInNestedClass() {
        // Pruebas en clase de prueba anidada
    }
}
```

### Pruebas parametrizadas

```java
import org.junit.jupiter.params.ParameterizedTest;
import org.junit.jupiter.params.provider.ValueSource;

@ParameterizedTest
@ValueSource(ints = {1, 2, 3, 4, 5})
void testWithMultipleValues(int number) {
    assertTrue(number > 0);
}
```

### Pruebas dinámicas

```java
import org.junit.jupiter.api.TestFactory;
import org.junit.jupiter.api.DynamicTest;
import java.util.Arrays;

@TestFactory
Collection<DynamicTest> dynamicTestsFromIterator() {
    return Arrays.asList(
        DynamicTest.dynamicTest("Primera prueba dinámica", () -> assertTrue(true)),
        DynamicTest.dynamicTest("Segunda prueba dinámica", () -> assertEquals(4, 2 + 2))
    );
}
```

## Uso básico de JUnit 4

### Escribir clases de prueba

```java
import org.junit.Test;
import static org.junit.Assert.*;

public class CalculatorTest {
    
    @Test
    public void testAddition() {
        assertEquals(5, 2 + 3);
    }
}
```

### Aserciones comunes

```java
// Aserciones de igualdad
assertEquals(expected, actual);
assertEquals("mensaje", expected, actual);

// Aserciones booleanas
assertTrue(condition);
assertFalse(condition);

// Aserciones de nulo
assertNull(object);
assertNotNull(object);

// Aserciones de excepciones
@Test(expected = NullPointerException.class)
public void testException() {
    Object obj = null;
    obj.toString();
}
```

### Before y After

```java
import org.junit.Before;
import org.junit.After;

public class MyTest {
    
    @Before
    public void setUp() {
        // Se ejecuta antes de cada método de prueba
    }
    
    @After
    public void tearDown() {
        // Se ejecuta después de cada método de prueba
    }
    
    @BeforeClass
    public static void setUpClass() {
        // Se ejecuta una vez antes de todas las pruebas
    }
    
    @AfterClass
    public static void tearDownClass() {
        // Se ejecuta una vez después de todas las pruebas
    }
}
```

### Suites de pruebas

```java
import org.junit.runner.RunWith;
import org.junit.runners.Suite;
import org.junit.runners.Suite.SuiteClasses;

@RunWith(Suite.class)
@SuiteClasses({Test1.class, Test2.class, Test3.class})
public class AllTests {
    // Ejecuta todas las pruebas en Test1, Test2 y Test3
}
```

## Migración de JUnit 4 a JUnit 5

### Cambios en anotaciones

| JUnit 4 | JUnit 5 |
|---------|---------|
| @Test | @Test |
| @Before | @BeforeEach |
| @After | @AfterEach |
| @BeforeClass | @BeforeAll |
| @AfterClass | @AfterAll |
| @Ignore | @Disabled |
| @SuiteClasses | @SelectClasses |

### Cambios en aserciones

```java
// JUnit 4
Assert.assertEquals(expected, actual);

// JUnit 5
Assertions.assertEquals(expected, actual);
```

### Cambios en pruebas de excepciones

```java
// JUnit 4
@Test(expected = NullPointerException.class)
public void testException() {
    Object obj = null;
    obj.toString();
}

// JUnit 5
@Test
public void testException() {
    assertThrows(NullPointerException.class, () -> {
        Object obj = null;
        obj.toString();
    });
}
```

## Ejecutar pruebas

### Línea de comandos

```bash
# Maven
mvn test

# Gradle
gradle test

# Consola independiente
java -jar junit-platform-console-standalone-5.14.3.jar -c MyTestClass
```

### Ejecución en IDE

- IntelliJ IDEA: Haz clic derecho en la clase o método de prueba, selecciona Run
- Eclipse: Haz clic derecho en la clase o método de prueba, selecciona Run As → JUnit Test
- VS Code: Usa la extensión JUnit para ejecutar pruebas
