# Vue.js 安装指南

本指南将帮助你搭建 Vue.js 开发环境。

## 环境要求

| 软件 | 要求 |
|------|------|
| Node.js | 16.0+ |
| npm | 8.0+ |
| 代码编辑器 | VS Code（推荐） |

## 安装步骤

### 方式一：使用 Vite（推荐）

Vite 是 Vue 官方推荐的新一代构建工具，启动速度快：

```bash
# 创建项目
npm create vue@latest my-vue-app

# 或指定选项
npm create vue@latest my-vue-app -- --typescript --router --pinia

# 进入项目目录
cd my-vue-app

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 方式二：使用 Vue CLI

Vue CLI 是官方提供的脚手架工具：

```bash
# 全局安装 CLI
npm install -g @vue/cli

# 创建项目
vue create my-vue-app

# 启动 UI 向导
vue ui
```

### 方式三：CDN 引入

适合简单页面或学习：

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Vue 入门</title>
  <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
</head>
<body>
  <div id="app">
    {{ message }}
  </div>

  <script>
    const { createApp } = Vue
    
    createApp({
      data() {
        return {
          message: 'Hello Vue!'
        }
      }
    }).mount('#app')
  </script>
</body>
</html>
```

## Vue 项目结构

```
my-vue-app/
├── public/              # 静态资源
├── src/
│   ├── assets/         # 资源文件
│   ├── components/     # 组件
│   ├── views/          # 页面视图
│   ├── router/         # 路由配置
│   ├── stores/         # 状态管理（Pinia）
│   ├── App.vue         # 根组件
│   └── main.js         # 入口文件
├── index.html
├── package.json
└── vite.config.js      # Vite 配置
```

## Vue 插件安装

### Vue Router

```bash
npm install vue-router@4
```

### Pinia（状态管理）

```bash
npm install pinia
```

### Axios（HTTP 请求）

```bash
npm install axios
```

### UI 组件库

```bash
# Element Plus
npm install element-plus

# Ant Design Vue
npm install ant-design-vue

# Naive UI
npm install naive-ui
```

## 配置国内镜像

如果 npm 下载慢，可以配置国内镜像：

```bash
# 设置镜像
npm config set registry https://registry.npmmirror.com

# 恢复官方镜像
npm config set registry https://registry.npmjs.org
```

```bash
# 创建 TypeScript 项目

```

## VS Code 配置

### 推荐扩展

1. **Vue - Official**：Vue 语言支持
2. **Volar**：Vue 3 语言支持
3. **ESLint**：代码检查
4. **Prettier**：代码格式化
5. **Auto Rename Tag**：自动重命名标签

### 配置 ESLint

```javascript
// .eslintrc.js
module.exports = {
  extends: [
    'plugin:vue/vue3-essential',
    'eslint:recommended'
  ]
}
```

## 常见问题

### Q1: 启动失败

**解决方案：**
```bash
# 删除 node_modules 和 package-lock.json
rm -rf node_modules package-lock.json

# 重新安装
npm install
```

### Q2: 端口被占用

**解决方案：**
```javascript
// vite.config.js
export default {
  server: {
    port: 3000
  }
}
```

### Q3: 无法解析 .vue 文件

**解决方案：**
1. 安装 Volar 扩展
2. 配置 TypeScript：
   ```json
   // tsconfig.json
   {
     "compilerOptions": {
       "moduleResolution": "bundler"
     }
   }
   ```
