# H2 소개

H2는 Java로 작성된 오픈 소스 관계형 데이터베이스管理系统입니다.

## 주요 기능

- 임베디드 데이터베이스 (별도 서버 불필요)
- 클라이언트-서버 모드
- 인메모리 데이터베이스
- MySQL, PostgreSQL, Oracle 호환
- 데이터베이스 암호화
- 풀텍스트 검색

## 적용 분야

- Java 개발
- 단위 테스트
- 빠른 프로토타이핑

## 라이선스

이중 라이선스: MPL 2.0 또는 EPL 1.0

## 요구사항

Java 11 이상
