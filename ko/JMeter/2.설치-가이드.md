# JMeter 설치 가이드

이 가이드는 다양한 운영 체제에서 Apache JMeter를 설치하고 구성하는 지침을 제공합니다.

## 요구사항

- **Java 런타임 환경**: JMeter에는 Java 8 이상이 필요합니다
- **운영 체제**: Linux, Windows, Mac OSX를 포함한 모든 Java 호환 OS 지원
- **메모리**: 최소 4GB 사용 가능 메모리 권장(테스트 규모에 따라 더 필요할 수 있음)
- **디스크 공간**: 최소 1GB 사용 가능 공간 권장

## JMeter 다운로드

1. 공식 JMeter 다운로드 페이지 방문: https://jmeter.apache.org/download_jmeter.cgi
2. 최신 바이너리 배포판(Binaries) 다운로드
3. 다운로드 무결성을 확인하기 위해 PGP 또는 SHA-512 서명 파일도 다운로드 권장

## Windows 설치

### 압축 해제

1. 다운로드한 Apache JMeter 아카이브(apache-jmeter-x.x.zip) 찾기
2. 적절한 설치 디렉토리 선택(공백과 특수 문자가 있는 경로 피하기)
3. 선택한 디렉토리에 아카이브 압축 해제
4. JMeter 설치 완료

### 확인

1. JMeter 설치 디렉토리의 bin 폴더로 이동
2. jmeter.bat 더블 클릭(Windows 배치 파일)
3. 성공하면 JMeter GUI가 실행됩니다

## Linux 및 Mac OSX 설치

### 압축 해제

1. 다운로드한 Apache JMeter 아카이브(apache-jmeter-x.x.tar.gz) 찾기
2. 다음 명령으로 압축 해제:
   ```bash
   tar -xf apache-jmeter-x.x.tar.gz
   ```
3. JMeter 설치 완료

### 환경 변수(선택 사항)

1. 터미널 구성 파일 열기(~/.bashrc 또는 ~/.profile)
2. 다음 환경 변수 추가:
   ```bash
   export JMETER_HOME=/path/to/apache-jmeter-x.x
   export PATH=$PATH:$JMETER_HOME/bin:$JMETER_HOME/lib
   ```
3. 구성 저장 및 새로고침:
   ```bash
   source ~/.bashrc
   ```

### 확인

1. 터미널에서 다음 실행:
   ```bash
   jmeter --version
   ```
2. 버전 정보가 표시되면 설치 성공

## JMeter 시작

### GUI 모드

- **Windows**: bin 디렉토리의 jmeter.bat 더블 클릭
- **Linux/Mac**: 터미널에서 jmeter 명령 실행

### 명령줄 모드

부하 테스트의 경우 리소스 사용을 줄이기 위해 명령줄 모드 권장:

```bash
jmeter -n -t test_plan.jmx -l results.jtl -e -o /path/to/output
```

매개변수:
- -n: non-GUI 모드로 실행
- -t: 테스트 플랜 파일 지정
- -l: 결과 파일 지정
- -e: 테스트 종료 후 보고서 생성
- -o: 보고서 출력 디렉토리 지정

## 디렉토리 구조

- **bin**: 실행 파일 및 시작 스크립트 포함
- **lib**: JMeter 종속성 라이브러리 포함
- **docs**: 공식 JMeter 문서 포함
- **extras**: 추가 보조 도구 포함
- **printable_docs**: 인쇄 가능한 사용자 매뉴얼 포함

## 종속성 관리

JMeter의 lib 디렉토리에는 핵심 종속성이 있습니다. 추가 플러그인 또는 라이브러리를 추가하려면:

1. JAR 파일을 lib 디렉토리에 복사
2. 일부 플러그인은 lib/ext 디렉토리에 배치해야 함
3. 새 플러그인을 적용하려면 JMeter 재시작

## 일반적인 설치 문제

- **Java를 찾을 수 없음**: Java가 제대로 설치되어 있고 JAVA_HOME 환경 변수가 설정되어 있는지 확인
- **메모리 부족**: bin/jmeter.sh 또는 jmeter.bat의 힙 크기 매개변수 수정
- **시작 실패**: 올바른 Java 버전(Java 8 이상)이 설치되어 있는지 확인
