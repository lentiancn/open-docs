# インストールガイド

このドキュメントでは、Apifox の 다양한インストール方法について説明します。スムーズにローカル環境で Apifox を使い始められるようにナビゲートします。

## 動作環境

Apifox は以下のオペレーティングシステムに対応しています：

- Windows 7 以降（64ビット版）
- macOS 10.13 以降
- Linux（Debian/Ubuntu 20.04+、Fedora 36+）

## ダウンロード方法

### 官网からダウンロード

Apifox 官网 https://apifox.com/ にアクセスして、「ダウンロード」ボタンをクリックし、お使いの OS 合ったバージョンを選択してください。

### バージョンの選択

Apifox は2つのバージョンを用意しています：

- **Apifox**（標準版）：全てのコア機能が含まれており、一般的な用途ならこれで十分です
- **Apifox**（有料版）：チーム管理、高度な自動化など、より 많은 高度な機能が使えます

基本的な機能であれば、標準版で全く問題ありません。

## インストール手順

### Windows の場合

1. Windows 向けインストーラー（.exe ファイル）をダウンロード
2. インストーラーをダブルクリックして実行
3. インストール先を指定（デフォルトのままでもOK）
4. インストールが完了するまで待つ
5. Apifox を起動して、ログインまたは新規登録

### macOS の場合

1. macOS 向けインストーラー（.dmg ファイル）をダウンロード
2. インストーラーを開き、Apifox を Applications フォルダにドラッグ
3. Apifox を起動して、ログインまたは新規登録

初めて起動する際は、システム環境設定で「未知の开发者からのアプリを許可」する必要がある場合があります。

### Linux の場合

**Debian/Ubuntu：**

```bash
# deb パッケージをダウンロード
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.deb

# インストール
sudo dpkg -i Apifox-latest-linux.x64.deb

# 依存関係のエラーが出たら、修復インストールを実行
sudo apt-get install -f
```

**Fedora：**

```bash
# rpm パッケージをダウンロード
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.rpm

# インストール
sudo rpm -i Apifox-latest-linux.x64.rpm
```

## アカウントとログイン

### アカウント登録

初めて使う方はアカウント登録が必要です：

1. Apifox を起動
2. 「登録」ボタンをクリック
3. メールアドレスとパスワードを入力して登録完了
4. メール認証（必要な場合のみ）

### ログイン

いくつかのログイン方法に対応しています：

- メールアドレスとパスワードでログイン
- 第三方アカウントでログイン（GitHub、Google など）

### クラウド同期

ログインすると、プロジェクトデータが自動的にクラウドに同期されます。異なるデバイス間でもシームレスに切り替えられ、データはリアルタイムで最新の状態を保ちます。

## 初期設定

初めて起動したら、以下の設定しておくことをおすすめします：

### テーマの設定

「設定」で、ライトテーマかダークテーマか選べます。お好みのスタイルを選んでください。

### プロキシの設定

プロキシ経由でインターネットに接続する必要がある場合は、「設定」→「ネットワーク」でプロキシサーバーを設定してください。

### インポート・エクスポート

- **インポート**：Postman、Swagger、Har などの形式のプロジェクトをインポート可能
- **エクスポート**：OpenAPI、Markdown、HTML などの形式でエクスポート可能

## クイックスタート

### プロジェクト作成

1. 左側の「プロジェクト」の横にある「+」をクリック
2. プロジェクト名を入力
3. プロジェクトタイプを選択（API プロジェクトまたは Mock プロジェクト）

### 接口作成

1. プロジェクト内で「+」→「新規接口作成」をクリック
2. 接口の基本情報を入力：
   - 接口の名前
   - リクエストパス
   - リクエストメソッド（GET/POST/PUT/DELETE など）
3. リクエストパラメータ、本文、レスポンスなどを入力
4. 接口を保存

### 接口デバッグ

1. 作成した接口を開く
2. 「送信」ボタンをクリック
3. レスポンス結果を確認

### Mock を使う

1. 接口にデータ構造を定義
2. システムが自動的に Mock URL を生成
3. Mock URL にアクセスするとモックデータが取得できる

## アンインストール

### Windows の場合

- 「コントロールパネル」→「プログラムと機能」を開く
- Apifox を探して「アンインストール」をクリック

### macOS の場合

- Applications フォルダ内の Apifox をゴミ箱にドラッグ
- ゴミ箱を空にする

### Linux の場合

```bash
# Debian/Ubuntu
sudo apt-get remove apifox

# Fedora
sudo rpm -e apifox
```
