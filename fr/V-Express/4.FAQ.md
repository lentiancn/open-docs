# V-Express FAQ

## Questions Générales

### Qu'est-ce que V-Express?
V-Express est un modèle IA qui génère des vidéos de tête parlante à partir d'une image de portrait et d'une entrée audio. Il crée des animations faciales réalistes avec synchronisation labiale.

### V-Express est-il gratuit?
V-Express est open source pour la recherche. Consultez la LICENSE pour un usage commercial.

### Quelle différence avec d'autres générateurs?
V-Express utilise:
- Décodage progressif conditionnel diffusion
- Module VQ-aware de faiblesse
- Conditionnement multi-niveaux

## Questions d'Installation

### Quels sont les prérequis?
```bash
conda create -n vexpress python=3.8
conda activate vexpress
pip install torch torchvision
pip install opencv-python
pip install librosa
```

### Comment télécharger les modèles?
```bash
bash scripts/download_models.sh
```

### Ai-je besoin d'un GPU puissant?
Oui, un GPU avec 8GB+ VRAM est recommandé.

## Questions d'Utilisation

### Comment générer une vidéo parlante?
```bash
python inference.py --reference_image path/to/image.jpg --audio path/to/audio.wav --output output.mp4
```

### Quels formats d'image sont supports?
- JPG/JPEG
- PNG
- BMP
- WebP

### Quels formats audio sont supports?
- WAV (recommandé)
- MP3
- FLAC

## Dépannage

### Mauvaise synchronisation labiale
- Utiliser audio de haute qualité sans bruit
- Assurer une parole claire

### Distorsion d'identité
- Utiliser image haute résolution
- Éviter les photos trop éditées

### Artefacts dans la sortie
- Réduire la résolution de sortie
- Vérifier l'installation des modèles

### Erreurs de mémoire
- Réduire la taille du batch
- Baisser la résolution de sortie
- Fermer autres applications GPU
