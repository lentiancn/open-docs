## Introduction

### What is Bootstrap?

Bootstrap is the world's most popular front-end open source toolkit, developed by Twitter. It's a CSS and JavaScript framework for building responsive websites and web applications.

### Bootstrap Versions

| Version | Release Date | Status |
|---------|--------------|--------|
| Bootstrap 5.3 | May 2023 | Current Stable |
| Bootstrap 5.2 | September 2022 | Supported |
| Bootstrap 5.1 | August 2021 | Supported |

### Bootstrap 5 New Features

- New color schemes
- Improved form controls
- New utilities API
- Better customization
- No jQuery dependency

## Installation Methods

### Method 1: Using CDN (Easiest)

CDN is the fastest way to start - no downloads required.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Getting Started</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>Hello, Bootstrap!</h1>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### Method 2: Using npm

```bash
mkdir my-bootstrap-project
cd my-bootstrap-project
npm init -y
npm install bootstrap
```

### Method 3: Download Source

Visit https://getbootstrap.com/ to download compiled files.

## Installation on Windows

### Using CDN (Recommended)

1. Create a new HTML file
2. Add Bootstrap CSS link in `<head>`
3. Add Bootstrap JS script at end of `<body>`

### Using npm

1. Install Node.js from https://nodejs.org/
2. Open command prompt
3. Run: `npm install bootstrap`

---

## Installation on macOS

### Using Homebrew (Recommended)

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install node
mkdir bootstrap-project
cd bootstrap-project
npm init -y
npm install bootstrap
```

## Installation on Linux

### Ubuntu/Debian

```bash
sudo apt update
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs

mkdir bootstrap-project
cd bootstrap-project
npm init -y
npm install bootstrap
```

## Verification

Create test HTML:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <div class="container mt-5">
        <div class="alert alert-success">Bootstrap installed!</div>
        <button class="btn btn-primary">Primary Button</button>
    </div>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

Open in browser - you should see green alert and blue button.

## Troubleshooting

### Styles Not Working

- Check CSS link is correct
- Clear browser cache

### JavaScript Not Working

- Make sure bootstrap.bundle.min.js is used (includes Popper)
- Check browser console for errors
