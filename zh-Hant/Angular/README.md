## 快速開始

### 安裝 Angular CLI

```bash
npm install -g @angular/cli
```

### 建立新專案

```bash
ng new my-app
cd my-app
ng serve
```

### 產生元件

```bash
ng generate component my-component
# 或
ng g c my-component
```
