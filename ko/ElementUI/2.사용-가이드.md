# Element UI 사용 가이드

이 가이드는 Vue 2 애플리케이션에서 Element UI 컴포넌트를 사용하는 방법을 설명합니다.

## 기본 사용

### Hello World

```vue

  <div>
    <el-button>기본 버튼</el-button>
    <el-button type="primary">주요 버튼</el-button>
  </div>
</template>

<script>
export default {
  name: 'App'
}
</script>
```

## 일반적인 컴포넌트

### 버튼

```vue
<template>
  <div>
    <!-- 유형 -->
    <el-button>기본</el-button>
    <el-button type="primary">주요</el-button>
    <el-button type="success">성공</el-button>
    <el-button type="warning">경고</el-button>
    <el-button type="danger">위험</el-button>
    
    <!-- 플레인 -->
    <el-button plain>플레인</el-button>

    <el-button round>라운드</el-button>
    
    <!-- 서클 -->
    <el-button circle icon="el-icon-search"></el-button>
    
    <!-- 비활성화 -->
    <el-button disabled>비활성화</el-button>

```

### 입력

```vue
<template>
  <div>
    <!-- 기본 -->
    <el-input v-model="input" placeholder="텍스트 입력"></el-input>
    
    <!-- 아이콘 있음 -->
    <el-input
      v-model="input1"
      placeholder="검색"
      prefix-icon="el-icon-search">
    </el-input>
    
    <!-- 비밀번호 -->
    <el-input
      v-model="input2"
      placeholder="비밀번호"
      show-password>
    </el-input>
    
    <!-- 텍스트 영역 -->
    <el-input
      v-model="textarea"
      type="textarea"
      :rows="3"
      placeholder="설명 입력">
    </el-input>
  </div>
</template>

<script>
export default {
  data() {
    return {
      input: '',
      input1: '',
      input2: '',
      textarea: ''
    }
  }
}
</script>
```

### 테이블

```vue
<template>
  <el-table :data="tableData" stripe border>
    <el-table-column prop="date" label="날짜" width="180"></el-table-column>
    <el-table-column prop="name" label="이름" width="180"></el-table-column>
    <el-table-column prop="address" label="주소"></el-table-column>
    <el-table-column label="작업">
      <template slot-scope="scope">
        <el-button size="mini" @click="handleEdit(scope.$index, scope.row)">편집</el-button>
        <el-button size="mini" type="danger" @click="handleDelete(scope.$index, scope.row)">삭제</el-button>
      </template>
    </el-table-column>
  </el-table>
</template>

<script>
export default {
  data() {
    return {
      tableData: [{
        date: '2024-01-01',
        name: '김철수',
        address: '서울시 강남구'
      }]
    }
  },
  methods: {
    handleEdit(index, row) {
      console.log(index, row);
    },
    handleDelete(index, row) {
      console.log(index, row);
    }
  }
}
</script>
```

### 폼

```vue
<template>
  <el-form
    :model="ruleForm"
    :rules="rules"
    ref="ruleForm"
    label-width="120px"
  >
    <el-form-item label="이름" prop="name">
      <el-input v-model="ruleForm.name"></el-input>
    </el-form-item>
    
    <el-form-item label="이메일" prop="email">
      <el-input v-model="ruleForm.email"></el-input>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm('ruleForm')">제출</el-button>
      <el-button @click="resetForm('ruleForm')">초기화</el-button>
    </el-form-item>
  </el-form>
</template>

<script>
export default {
  data() {
    return {
      ruleForm: {
        name: '',
        email: ''
      },
      rules: {
        name: [
          { required: true, message: '이름을 입력하세요', trigger: 'blur' }
        ],
        email: [
          { required: true, message: '이메일을 입력하세요', trigger: 'blur' },
          { type: 'email', message: '유효하지 않은 이메일 형식입니다', trigger: 'blur' }
        ]
      }
    };
  },
  methods: {
    submitForm(formName) {
      this.$refs[formName].validate((valid) => {
        if (valid) {
          alert('제출 성공!');
        }
      });
    },
    resetForm(formName) {
      this.$refs[formName].resetFields();
    }
  }
}
</script>
```

### 대화상자

```vue
<template>
  <div>
    <el-button type="text" @click="dialogVisible = true">대화상자 열기</el-button>
    
    <el-dialog
      title="대화상제 제목"
      :visible.sync="dialogVisible"
      width="30%">
      <span>대화상자 콘텐츠</span>
      <span slot="footer" class="dialog-footer">
        <el-button @click="dialogVisible = false">취소</el-button>
        <el-button type="primary" @click="dialogVisible = false">확인</el-button>
      </span>
    </el-dialog>
  </div>
</template>

<script>
export default {
  data() {
    return {
      dialogVisible: false
    }
  }
}
</script>
```
