# Go インストールガイド

## インストール
### Windows
```powershell
choco install golang
```

### macOS
```bash
brew install go
```

### Linux
```bash
sudo apt-get install golang-go
```

## 設定
```bash
export GOPATH=$HOME/go
go env -w GOPROXY=https://goproxy.cn,direct
```

## コマンド
```bash
go mod init
go mod tidy
go build
go run
```
