# FantasyTalking よくある質問

## インストール関連

### インストールに失敗した場合は？

インストール中に問題が発生する可能性があります。まず、環境が要件を満たしていることを確認してください。Node.jsのバージョンが14.0以上であることを確認します `node --version` で確認できます。Pythonバージョンを使用する場合は、Pythonバージョンが3.7以上であることを確認してください。

インストールが失敗する場合は、以下を試してください：npmキャッシュをクリア `npm cache clean --force`；国内ミラーを使用 `npm config set registry https://registry.npmmirror.com`；またはソースからインストールします。

### FFmpegのインストール問題

一部のシステムでは、FFmpegが正しくインストールまたは設定されていない可能性があります。FFmpegの公式Web sitioから事前コンパイルされたバイaryをダウンロードするか、パッケージマネージャーでインストールできます：macOSでは `brew install ffmpeg`、Linuxでは `apt-get install ffmpeg` または `yum install ffmpeg`。

## 使用関連

### 音声合成が遅い場合は？

音声合成速度は複数の要因に依存します。まずネットワーク接続を確認し、クラウドサービスを使用している場合、ネットワークレイテンシが応答時間に影響します。オーバーヘッドを減らすためにセルフホストサービスを検討してください。大量の合成タスクにはバッチ合成インターフェースを使用すると効率が大幅に向上します。

### 適切な声のスタイルを選択方法は？

FantasyTalkingは neutral、happy、sad、angry、excited などさまざまな声のスタイルを提供しています。アプリケーションシナリオに基づいて選択してください：カスタマーサービスにはneutralまたはprofessional、教育アプリケーションではコンテンツに基づいて異なるスタイルを選択できます。

### 英文音声合成を実現方法は？

デフォルト設定では英文を指定する必要があります：

```javascript
const audio = await client.speak('Hello, world!', {
  voice: {
    language: 'en-US',
    style: 'neutral'
  }
});
```

## アカウントと請求

### APIキーを取得方法は？

FantasyTalking公式Web sitioでアカウントを登録し、コンソールでAPIキーを作成します。新規ユーザーには通常、テスト用の無料クレジットが提供されます。APIキーは安全に保管し、クライアントコードにハードコードせず、環境変数を使用してください。

### 使用量を確認方法は？

コンソールにログイン後、ダッシュボードでAPI呼び出し回数と消費配额を確認できます。SDK提供の方法を使用してクエリすることもできます：

```javascript
const usage = await client.getUsage();
console.log(`今月の呼び出し: ${usage.count}`);
console.log(`残り無料回数: ${usage.remaining}`);
```

## デプロイ関連

### Dockerデプロイが失敗した場合は？

まずDockerが正しくインストール되어あり実行されていることを確認してください。ポート8080が使用されているかどうかを確認し、マッピングポートを変更できます。クラウドサービスを使用している場合、セキュリティグループが開放されているポートを確認してください。

## その他

### FantasyTalkingがサポートするプラットフォームは？

FantasyTalkingは主流のプラットフォームとフレームワークをサポートしています：Webはすべての最新ブラウザをサポート、モバイルはiOSとAndroidをサポート、デスクトップはWindows、macOSとLinuxをサポート、サーバーはNode.jsとPythonをサポートします。
