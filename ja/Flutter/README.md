# Flutter

Flutter - クロスプラットフォームUIフレームワーク

## 概要

FlutterはGoogleが開発したオープンソースのUIソフトウェア開発キットで、単一のコードベースから美しくネイティブにコンパイルされたモバイル、Web、デスクトップアプリケーションを構築するために使用されます。

## 機能

- **クロスプラットフォーム**：1つのコードベースでiOS、Android、Web、Windows、macOS、Linuxをサポート
- **ホットリロード**：快速開発反復、アプリの状態を維持
- **豊富なコンポーネント**：Material DesignとCupertinoコンポーネントライブラリ
- **ネイティブパフォーマンス**：ネイティブARMコードにコンパイル
- **宣言的UI**：モダンUI開発パラダイム
- **オープンソース**：BSDライセンス

## クイックスタート

### インストール

```bash
# Flutterをクローン
git clone https://github.com/flutter/flutter.git -b stable

# PATHに追加
export PATH="$PATH:$HOME/flutter/bin"

# インストール確認
flutter doctor
```

### プロジェクト作成

```bash
flutter create my_app
cd my_app
flutter run
```
