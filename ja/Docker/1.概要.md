# Docker 概要

世界で最も人気のコンテナ化プラットフォーム。

## Dockerとは？

Dockerは、アプリケーションの開発、デプロイ、実行のためのオープンソースプラットフォームです。コンテナ化技術により、アプリケーションとその依存関係を軽量で移植可能なコンテナにパッケージ化できます。

### 主な機能

- 軽量 - カーネルを共有し、迅速に起動
- 移植性 - 一度ビルドすればどこでも実行
- 分離 - コンテナは互いに分離
- 効率的 - 仮想マシンよりリソースを節約

## コアコンセプト

- イメージ - コンテナを作成するための読み取り専用テンプレート
- コンテナ - イメージの実行インスタンス
- リポジトリ - イメージを保存する場所
- Dockerfile - イメージビルドスクリプトを定義
