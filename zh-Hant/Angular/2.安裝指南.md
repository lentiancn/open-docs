# Angular 安裝指南

完整的 Angular 開發環境設置。

## 環境要求

### 硬件要求

- CPU：2 核心或以上
- 記憶體：4GB 或以上
- 磁盤：10GB 或以上

### 軟件要求

- 操作系統：Windows/macOS/Linux
- Node.js：18.10.0+ 或 20.0.0+
- npm：9.0.0+（隨 Node.js 附帶）
- Git：任何版本

## 安裝步驟

### 1. 安裝 Node.js

**Windows/macOS：**

從 https://nodejs.org 下載安裝包

或使用 nvm：

```bash
# 安裝 nvm（macOS/Linux）
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 安裝 Node.js LTS
nvm install --lts
nvm use --lts
```

**驗證安裝：**

```bash
node --version  # 應顯示 v18.x.x 或 v20.x.x
npm --version   # 應顯示 9.x.x 或更高
```

### 2. 安裝 Angular CLI

```bash
# 通過 npm 全域安裝
npm install -g @angular/cli

# 驗證安裝
ng version
```

### 3. 創建新項目

```bash
# 創建新的 Angular 應用
ng new my-app

# 進入項目目錄
cd my-app

# 啟動開發伺服器
ng serve
```

### 4. 訪問應用

打開瀏覽器：http://localhost:4200

## 常用命令

### 項目命令

```bash
# 使用路由和 SCSS 創建新項目
ng new my-app --routing --style=scss

# 創建獨立應用（Angular 14+）
ng new my-app --standalone

# 啟動開發伺服器
ng serve

# 建構生產版本
ng build

# 運行單元測試
ng test

# 運行 E2E 測試
ng e2e
```

### 生成命令

```bash
# 生成組件
ng generate component my-component
ng g c my-component

# 生成服務
ng generate service my-service
ng g s my-service

# 生成指令
ng generate directive my-directive
ng g d my-directive

# 生成管道
ng generate pipe my-pipe
ng g p my-pipe

# 生成模組
ng generate module my-module
ng g m my-module

# 生成類
ng generate class my-class
ng g cl my-class

# 生成介面
ng generate interface my-interface
ng g i my-interface

# 生成守衛
ng generate guard my-guard
ng g g my-guard

# 生成攔截器
ng generate interceptor my-interceptor
ng g in my-interceptor
```

│   │       ├── home/

## 環境配置

### 開發環境

```typescript
// src/environments/environment.ts
export const environment = {
  production: false,
  apiUrl: 'http://localhost:3000/api'
};
```

### 生產環境

```typescript
// src/environments/environment.prod.ts
export const environment = {
  production: true,
  apiUrl: 'https://api.example.com'
};
```

### 使用環境

```typescript
import { Component } from '@angular/core';
import { environment } from '../environments/environment';

@Component({
  selector: 'app-root',
  template: `<p>API: {{ apiUrl }}</p>`
})
export class AppComponent {
  apiUrl = environment.apiUrl;
}
```

## 常見問題

### npm 安裝慢？

```bash
# 使用淘寶鏡像
npm config set registry https://registry.npmmirror.com

# 或使用 nrm 管理鏡像
npm install -g nrm
nrm use taobao
```

### Node 版本不符？

```bash
# 使用 nvm 切換版本
nvm install 20
nvm use 20
```

### 權限問題（Linux/macOS）？

```bash
# 修復 npm 權限
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc
```

## 卸載

```bash
# 卸載 Angular CLI
npm uninstall -g @angular/cli

# 刪除項目
rm -rf my-app
```
