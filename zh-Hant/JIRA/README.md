# Jira

Jira 是由 Atlassian 公司開發的專案管理與事務追蹤軟體，為全球超過30萬家企業提供專案管理解決方案。

## 快速開始

訪問 Atlassian 官網註冊帳號，選擇適合您團隊的方案，即可快速建立 Jira 站點。

## 主要功能

專案管理與事務追蹤、敏捷開發支援（Scrum、看板）、團隊協作與溝通、人工智慧助手 Rovo 整合、豐富的第三方整合、強大的自動化規則。
