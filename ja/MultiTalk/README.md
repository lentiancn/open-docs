# MultiTalk

MultiTalkは、梅津AIチームによって開発されたオーディオ駆動型の多人対話動画生成フレームワークです。このプロジェクトは、マルチストリームオーディオ入力、参照画像、テキストプロンプトに基づいて、複数の人物が相互作用する高品質な動画コンテンツを生成できます。

## 主な機能

- オーディオ駆動型の多人対話動画生成
- 単一人物と多人物のシナリオのサポート
- 業界トップの口唇同期精度
- アニメーションキャラクターと歌の動画生成のサポート
- 出力解像度は480Pと720Pをサポート
- 最大15秒の動画クリップを生成可能
