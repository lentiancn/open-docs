# Flutter - Installationsanleitung

Flutter Entwicklungsumgebung einrichten.

## Anforderungen

- Windows 10+ / macOS 10.14+ / Ubuntu 18.04+
- 2.8 GB Speicherplatz
- Git

## Installation

### Windows

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

Zum PATH hinzuf√ºgen.

### macOS

```bash
git clone https://github.com/flutter/flutter.git -b stable
export PATH="$PATH:/Users/benutzer/flutter/bin"
```

### Linux

```bash
sudo apt-get install -y curl git unzip xz-utils
git clone https://github.com/flutter/flutter.git -b stable
export PATH="$PATH:$HOME/flutter/bin"
```

## Verifizieren

```bash
flutter doctor
flutter --version
```
