# Manuel utilisateur JDK

Ce guide couvre l'utilisation de JDK pour le développement Java.

## Démarrage rapide

### Compilation et exécution

```bash
# Compiler
javac HelloWorld.java

# Exécuter
java HelloWorld
```

### Exécution d'un seul fichier (JDK 11+)

```bash
java HelloWorld.java
```

## Commandes de base

### javac - Compilateur Java

```bash
javac MyClass.java
javac *.java
javac -encoding UTF-8 MyClass.java
javac -d bin MyClass.java
javac -Xlint MyClass.java
```

### java - Lanceur Java

```bash
java MyClass
java MyClass arg1 arg2
java -classpath "lib/*:." MyClass
java -Xmx2g -Xms512m MyClass
```

### jar - Archive Java

```bash
jar cf myapp.jar -C bin .
jar cfm myapp.jar manifest.txt -C bin .
jar xf myapp.jar
jar tf myapp.jar
```

### javadoc - Générateur de documentation

```bash
javadoc -d docs -sourcepath src *.java
```

### jdb - Débogueur Java

```bash
jdb MyClass
stop at MyClass:10
run
next
step
```

## Options JVM

### Options mémoire

```bash
-Xmx2g        # Heap maximum
-Xms512m      # Heap initial
-Xss256k      # Taille de pile

# Garbage collection
-XX:+UseG1GC
-XX:+UseParallelGC
```

### Options de performance

```bash
-XX:+TieredCompilation
-Xshare:on
-XX:+UseStringDeduplication
```

## Intégration IDE

### IntelliJ IDEA

1. File → Project Structure → SDKs
2. Ajouter le chemin d'installation JDK

### VS Code

1. Installer "Extension Pack for Java"
2. Configurer JAVA_HOME

### Eclipse

1. Window → Preferences → Java → Installed JREs

## Outils de build

### Maven

```xml
<properties>
    <maven.compiler.source>21</maven.compiler.source>
    <maven.compiler.target>21</maven.compiler.target>
</properties>
```

### Gradle

```groovy
sourceCompatibility = '21'
targetCompatibility = '21'
```

## Modules (Java 9+)

```bash
java --list-modules
```

## Sécurité

### Keytool

```bash
keytool -genkeypair -alias mykey -keyalg RSA -keystore keystore.jks
keytool -list -keystore keystore.jks
```

## Profilage

### jcmd

```bash
jcmd
jcmd <pid> VM.flags
jcmd <pid> GC.run
```

## Bonnes pratiques

### 1. Utiliser LTS pour la production

```bash
# Recommandé: JDK 17 ou 21
```

### 2. Allouer suffisamment de mémoire

```bash
java -Xmx4g -Xms2g -XX:+UseG1GC MyApp
```

### 3. Activer les logs GC

```bash
-Xlog:gc*:file=gc.log:time,uptime,level,tags
```
