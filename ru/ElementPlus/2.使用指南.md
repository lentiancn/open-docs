# Element Plus Руководство по использованию

## Быстрый старт

```vue
<template>
  <el-button type="primary">Кнопка</el-button>
  <el-input v-model="input" />
</template>

<script setup>
import { ref } from 'vue'
const input = ref('')
</script>
```

## Компоненты

### Кнопки

```vue
<el-button>По умолчанию</el-button>
<el-button type="primary">Основная</el-button>
```

### Форма

```vue
<el-form :model="form">
  <el-form-item label="Имя">
    <el-input v-model="form.name" />
  </el-form-item>
</el-form>
```
