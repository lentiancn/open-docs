# Vue.js 安装指南

Vue.js 多种安装方式详解。

## 环境要求

### 硬件要求

- CPU：2 核及以上
- 内存：4GB 及以上

### 软件要求

- 操作系统：Windows/macOS/Linux
- Node.js：18.0.0+

## 安装步骤

### 1. 使用 npm

```bash
# 创建 Vue 项目
npm create vue@latest

# 进入项目目录
cd my-vue-project

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 2. 使用 Vite

```bash
# 创建 Vue 项目
npm create vite@latest my-vue-app -- --template vue

# 进入项目目录
cd my-vue-app

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 3. 使用 CDN

```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
```

### 4. 使用 Vue CLI（不推荐用于新项目）

```bash
# 全局安装 Vue CLI
npm install -g @vue/cli

# 创建项目
vue create my-project
```

## 项目结构

```
my-vue-app/
├── src/
│   ├── components/          # 组件
│   ├── views/             # 页面视图
│   ├── router/            # 路由配置
│   ├── stores/            # 状态管理
│   ├── assets/            # 静态资源
│   ├── App.vue            # 根组件
│   └── main.ts            # 入口文件
├── public/                 # 公共资源
├── index.html             # HTML 模板
├── package.json           # 依赖配置
├── vite.config.ts         # Vite 配置
└── tsconfig.json         # TypeScript 配置
```

## 常用命令

```bash
# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview

# 代码检查
npm run lint
```
