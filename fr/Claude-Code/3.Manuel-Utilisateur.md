# Claude Code Manuel utilisateur

Utilisation de base et techniques.

## Utilisation de base

### Démarrer

- VS Code: Utilisez le raccourci Cmd/Ctrl + Shift + P
- CLI: Exécutez `claude` directement dans le terminal

### Conversation

Décrivez vos besoins directement dans la fenêtre de chat:
```
Écris-moi une fonction de tri
Explique ce code
Comment corriger ce bug
```

## Génération de code

### Générer nouveau code

Décrivez la fonctionnalité souhaitée:
```
Créer une API de connexion utilisateur
Implémenter une requête avec pagination
Écrire des tests unitaires
```

### Modifier code existant

1. Sélectionnez le code
2. Décrivez les besoins de modification
3. Claude générera le code modifié

## Assistance au débogage

### Expliquer les erreurs

Collez le message d'erreur directement:
```
Que signifie cette erreur?
```

### Suggestions de correction

Demandez à Claude d'analyser le problème:
```
Qu'est-ce qui ne va pas avec cette fonction?
```

## Revue de code

### Demandes de revue de code

```
Revois ce code
Des problèmes de sécurité?
Suggestions d'optimisation des performances
```

## Raccourcis clavier

### VS Code

| Raccourci | Fonction |
|-----------|----------|
| Cmd/Ctrl + L | Ouvrir le chat |
| Cmd/Ctrl + Enter | Soumettre la saisie |

## Options de configuration

### claude.json

Créez `claude.json` à la racine du projet:

```json
{
  "preamble": "Tu es un expert en Python",
  "maxTokens": 4000
}
```

## Meilleures pratiques

1. Décrivez clairement les exigences
2. Fournissez des informations de contexte
3. Itérez et améliorez le code
4. Combinez avec une revue humaine
