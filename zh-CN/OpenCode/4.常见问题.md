# OpenCode 常见问题

常见问题解答。

## 安装问题

### Q: 安装失败？

**解决方案**：
1. 检查系统要求（需要现代终端模拟器）
2. 确认网络连接正常
3. 尝试使用安装脚本：`curl -fsSL https://opencode.ai/install | bash`
4. 查看错误信息并重试

### Q: Windows 上无法安装？

**解决方案**：
1. 尝试使用 Chocolatey：`choco install opencode`
2. 或使用 Scoop：`scoop install opencode`
3. 或使用 NPM：`npm install -g opencode-ai`

### Q: 验证安装失败？

**解决方案**：
1. 确认 opencode 已添加到 PATH
2. 重启终端
3. 检查安装是否成功

## 使用问题

### Q: 无法连接 LLM？

**解决方案**：
1. 运行 `/connect` 命令
2. 访问 opencode.ai/auth 登录
3. 添加账单信息并获取 API 密钥
4. 将 API 密钥粘贴到终端

### Q: 响应很慢？

**解决方案**：
1. 检查网络连接
2. 尝试切换到其他 LLM 提供商
3. 减少对话上下文长度

### Q: 如何切换模型？

使用 `/connect` 命令选择不同的 LLM 提供商。

### Q: 支持哪些模型？

支持 75+ LLM 提供商，包括：
- OpenAI (GPT-4, Claude 等)
- Anthropic (Claude)
- Google (Gemini)
- 本地模型
- 详见 [Models.dev](https://models.dev)

## 功能问题

### Q: AGENTS.md 是什么？

AGENTS.md 是 OpenCode 初始化时创建的文件，包含项目结构和编码规范，帮助 OpenCode 更好地理解项目。

### Q: Plan 模式和 Build 模式区别？

- **Plan 模式**：只生成计划，不实际修改代码
- **Build 模式**：直接执行修改

按 **Tab** 键切换模式。

### Q: 如何分享会话？

使用 `/share` 命令创建分享链接。

## 隐私问题

### Q: OpenCode 会保存我的代码吗？

不会。OpenCode 不会存储任何代码或上下文数据，可以在敏感环境中安全使用。

### Q: 如何保护隐私？

- 不在对话中输入敏感信息
- 使用本地模型
- 定期清理会话
