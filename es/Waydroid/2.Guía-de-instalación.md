# Guía de instalación

## Instrucciones de instalación específicas por distribución

### Arch Linux

Los usuarios de Arch pueden consultar las instrucciones completas de instalación en Arch Wiki:
[https://wiki.archlinux.org/title/Waydroid](https://wiki.archlinux.org/title/Waydroid)

### Postmarket OS

La comunidad de Postmarket proporciona instrucciones de instalación y consejos para solucionar problemas:
[https://wiki.postmarketos.org/wiki/Waydroid](https://wiki.postmarketos.org/wiki/Waydroid)

### Fedora

Waydroid se puede instalar desde el repositorio oficial de paquetes:

```bash
sudo dnf install waydroid
```

Después de instalar, inicie Waydroid desde el menú de aplicaciones y proceda con la inicialización pegando estas URL en los campos OTA:

OTA del sistema: `https://ota.waydro.id/system`

OTA del proveedor: `https://ota.waydro.id/vendor`

#### Silverblue/Kinoite/...

Las mismas instrucciones se aplican a las variantes inmutables de Fedora, pero debe usar `rpm-ostree` en lugar de `dnf`:

```bash
rpm-ostree install waydroid
```

### Ubuntu/Debian y derivados

Asegúrese de tener habilitada la sesión Wayland (Ubuntu 22.04+)

* Instale los requisitos previos

```bash
sudo apt install curl ca-certificates -y
```

* Agregue el repositorio oficial

```bash
curl -s https://repo.waydro.id | sudo bash
```

Si el script no detecta su distribución, puede proporcionar una opción válida agregando `-s <DISTRO>`. Los valores actualmente admitidos son: **mantic**, **focal**, **jammy**, **kinetic**, **lunar**, **noble**, **plucky**, **questing**, **bookworm**, **bullseye**, **trixie**, **sid**

* Instale waydroid

```bash
sudo apt install waydroid -y
```

Luego inicie Waydroid desde el menú de aplicaciones.

## Solución de problemas

### Iniciar Waydroid manualmente

Para iniciar Waydroid sin systemctl, debe seguir algunos pasos simples:

**Primero inicie el contenedor:**

```bash
sudo waydroid container start
```

**Y en una nueva pestaña de terminal, inicie la sesión waydroid (sin ***sudo*****):**

```bash
waydroid session start
```

Después de que se inicie y vea "Android with user 0 is ready", es seguro lanzar una aplicación desde el menú de aplicaciones.

### Iniciar Waydroid en modo de pantalla completa

```bash
waydroid show-full-ui
```

### Iniciar Waydroid en modo de múltiples ventanas

Primero necesitamos establecer la propiedad mientras se ejecuta una sesión de Waydroid:

```bash
waydroid prop set persist.waydroid.multi_windows true
```

Después de eso, podemos reiniciar la sesión:

```bash
waydroid session stop
```

Luego estamos listos para lanzar una aplicación, y se iniciará en modo de múltiples ventanas.

### Reinstalar Waydroid

Si necesita reinstalar completamente:

```bash
waydroid session stop
sudo waydroid container stop
sudo apt remove waydroid  # o el comando de desinstalación correspondiente de la distribución
# Después del reinicio, realice la limpieza
sudo rm -rf /var/lib/waydroid /home/.waydroid ~/waydroid ~/.share/waydroid ~/.local/share/applications/*aydroid* ~/.local/share/waydroid
```
