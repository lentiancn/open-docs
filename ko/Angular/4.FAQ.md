# Angular 자주 묻는 질문

자주 묻는 질문과 답변.

## 설치

### Q: Angular CLI 설치 실패?

**해결책**:
1. npm 캐시 지우기: `npm cache clean --force`
2. 관리자 권한으로 설치
3. Node.js 버전 확인

### Q: ng 명령어를 찾을 수 없음?

**해결책**:
1. Angular CLI가 전역으로 설치되었는지 확인
2. PATH 환경 변수 확인
3. 재설치 시도

### Q: Node.js 버전 호환성 없음?

**해결책**:
1. nvm으로 Node.js 버전 관리
2. 권장 버전으로 업그레이드

## 사용

### Q: 컴포넌트 스타일이 작동하지 않음?

**해결책**:
1. styleUrls 경로가 정확한지 확인
2. CSS 선택자가 정확한지 확인
3. ViewEncapsulation 설정 확인

### Q: 라우팅 실패?

**해결책**:
1. 라우트 설정이 정확한지 확인
2. RouterModule이 임포트되었는지 확인
3. routerLink 구문 확인

### Q: HTTP 요청 실패?

**해결책**:
1. HttpClientModule이 임포트되었는지 확인
2. CORS 설정 확인
3. API URL이 정확한지 확인

### Q: 폼 검증이 작동하지 않음?

**해결책**:
1. ReactiveFormsModule이 임포트되었는지 확인
2. Validators 설정 확인
3. formControlName을 정확히 사용

## 성능

### Q: 애플리케이션 로딩 느림?

**해결책**:
1. 레이지 로딩 활성화
2. OnPush 변경 감지 사용
3. 이미지 및 리소스 압축
4. 프로덕션 모드로 빌드

### Q: 메모리 누수?

**해결책**:
1. Observable 구독 취소
2. 타이머 정리
3 이벤트 리스너 파괴

## 버전

### Q: Angular 업그레이드 방법은?

```bash
ng update @angular/core @angular/cli
```

### Q: AngularJS에서 Angular로 마이그레이션?

angular-upgrade 도구로 점진적 마이그레이션.
