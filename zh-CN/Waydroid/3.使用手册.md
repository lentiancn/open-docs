# 使用手册

## Android 应用程序安装与运行

Waydroid 可以通过 `waydroid app -h` 命令执行多种操作：

```bash
usage: waydroid app [-h] {install,remove,launch,list} ...

optional arguments:
  -h, --help            显示此帮助信息并退出

子命令:
  {install,remove,launch,list}
    install             将单个软件包推送到容器并安装
    remove              从容器中移除单个应用软件包
    launch              启动单个应用程序
    list                列出已安装的应用程序
```

### 安装 Android 应用程序

使用以下命令安装 Android 应用程序：

```bash
waydroid app install xyz.apk
```

安装后，您可以从桌面应用程序菜单启动该应用程序。

如果您知道应用程序的 Android 包名（`com.xxx.yyy`），也可以通过命令行启动：

```bash
waydroid app launch com.foo.bar
```

### 注意事项

在互联网上找到的 apk 文件通常只支持 arm 架构，因此在 x86_64 系统上无法正常工作。

您可以安装 [F-Droid](https://f-droid.org) 来图形化获取应用程序，或者如果您安装了 GAPPS 镜像，也可以使用 Play 商店。

## 常用命令

### Waydroid 命令行选项

Waydroid 提供了丰富的命令行选项来控制容器和会话：

- `waydroid show-full-ui`：显示完整的 Android UI
- `waydroid session start`：启动 Waydroid 会话
- `waydroid session stop`：停止 Waydroid 会话
- `waydroid container start`：启动 Waydroid 容器
- `waydroid container stop`：停止 Waydroid 容器
- `waydroid logcat`：查看 Android 日志
- `waydroid shell`：进入 Android shell
- `waydroid prop set <property> <value>`：设置系统属性
- `waydroid bugreport`：生成错误报告日志

### 系统属性配置

可以通过 `waydroid prop` 命令配置各种系统属性：

- `persist.waydroid.multi_windows`：启用多窗口模式
- 其他属性请参考官方文档

## 高级功能

### 自定义镜像使用

要使用自定义镜像，需要执行以下步骤：

1. 从 [SourceForge](https://sourceforge.net/projects/waydroid/files/images/) 下载镜像或使用自定义构建的 `system.img` 和 `vendor.img`
2. 将镜像复制到 `/etc/waydroid-extra/images/` 目录
3. 重新初始化 Waydroid：`sudo waydroid init -f`

### ADB 调试

Waydroid 支持 ADB 调试，默认情况下 ADB 安全模式已启用，不会在会话启动时自动连接。可以使用 `waydroid adb connect` 命令连接 ADB，并在 Android 端提供授权。

### 通知转发

Waydroid 支持将 Android 通知转发到桌面环境，需要更新 Android 系统镜像到兼容版本。
