# Ant Design Vue 3 Usage Guide

Comprehensive guide for using Ant Design Vue 3 effectively.

---

## Basic Components

### Button

```vue
<a-button>Default</a-button>
<a-button type="primary">Primary</a-button>
<a-button type="dashed">Dashed</a-button>
<a-button type="text">Text</a-button>
<a-button type="link">Link</a-button>

<!-- Sizes -->
<a-button size="large">Large</a-button>
<a-button size="small">Small</a-button>

<!-- Icons -->
<a-button icon="search">Search</a-button>
```

### Layout

```vue
<a-layout>
  <a-layout-header>Header</a-layout-header>
  <a-layout-content>Content</a-layout-content>
  <a-layout-footer>Footer</a-layout-footer>
</a-layout>

<!-- Grid -->
<a-row :gutter="16">
  <a-col :span="12">Col 12</a-col>
  <a-col :span="12">Col 12</a-col>
</a-row>
```

## Form Components

### Input

```vue
<a-input v-model:value="value" placeholder="Please input" />

<!-- With prefix/suffix -->
<a-input v-model:value="value">
  <template #prefix>
    <UserOutlined />
  </template>
</a-input>
```

### Select

```vue
<a-select v-model:value="value" placeholder="Select">
  <a-select-option value="jack">Jack</a-select-option>
  <a-select-option value="lucy">Lucy</a-select-option>
</a-select>
```

### Form

```vue
<a-form :model="form" layout="vertical">
  <a-form-item label="Name">
    <a-input v-model:value="form.name" />
  </a-form-item>
  <a-form-item>
    <a-button type="primary" @click="onSubmit">Submit</a-button>
  </a-form-item>
</a-form>
```

## Data Display

### Table

```vue
<a-table :columns="columns" :data-source="data" :pagination="false" />
```

### Tag

```vue
<a-tag>Tag</a-tag>
<a-tag color="green">Green</a-tag>
<a-tag color="blue">Blue</a-tag>
<a-tag color="red">Red</a-tag>
<a-tag color="orange">Orange</a-tag>
```

### Card

```vue
<a-card title="Card Title">
  <p>Card content</p>
</a-card>
```

## Navigation

### Menu

```vue
<a-menu mode="horizontal">
  <a-menu-item key="mail">
    <MailOutlined />
    Navigation One
  </a-menu-item>
  <a-menu-item key="app">
    <AppstoreOutlined />
    Navigation Two
  </a-menu-item>
</a-menu>
```

### Tabs

```vue
<a-tabs v-model:activeKey="activeKey">
  <a-tab-pane key="1" tab="Tab 1">Tab 1</a-tab-pane>
  <a-tab-pane key="2" tab="Tab 2">Tab 2</a-tab-pane>
</a-tabs>
```

## Feedback

### Message

```vue
<script setup>
import { message } from 'ant-design-vue'

message.success('Success')
message.error('Error')
message.warning('Warning')
message.info('Info')
</script>
```

### Modal

```vue
<a-button @click="showModal = true">Open Modal</a-button>

<a-modal v-model:open="showModal" title="Title">
  <p>Content</p>
</a-modal>
```

### Notification

```vue
<script setup>
import { notification } from 'ant-design-vue'

notification.success({
  message: 'Title',
  description: 'Description'
})
</script>
```
