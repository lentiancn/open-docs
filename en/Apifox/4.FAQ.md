# Frequently Asked Questions

This document compiles common technical questions and answers encountered while using Apifox.

## Basic Questions

### What is Apifox?

Apifox is an API collaboration platform that brings together API design, documentation management, debugging, mocking, and automated testing. It helps development teams work more efficiently on API development, testing, and documentation maintenance.

### Is Apifox free?

Apifox offers both free and paid versions. The free version includes all core features, which are enough to meet most development needs. The paid version provides more advanced features like larger team sizes and advanced automation.

### What's the difference between Apifox and Postman?

Both Apifox and Postman are API tools, but Apifox is better suited for Chinese users: better Chinese language support, smarter mocking features, and a more polished team collaboration experience. Apifox's basic features are completely free, while many Postman features require payment.

## Installation Issues

### What if the download is slow?

Apifox provides multiple download servers. Try visiting https://apifox.com/ to select the server closest to you.

### What if it won't start?

1. Check if your system version meets the requirements
2. Try running as administrator
3. Check if necessary runtime libraries are installed
4. Check error logs for more information

### What if login fails?

1. Check if your network connection is working
2. Verify your account credentials are correct
3. Try logging in with a third-party account
4. Contact official support for help

## Usage Issues

### Interface data not syncing?

1. Make sure you're logged in
2. Check your network connection
3. Click the "Sync" button to manually sync
4. Check your account permissions

### Mock data not working?

1. Make sure Mock is enabled
2. Check if the Mock URL is correct
3. Check if there are any syntax errors in the interface
4. Try refreshing the page

### Request sending failing?

1. Check your network connection
2. Verify the request URL is correct
3. Check if request parameters are formatted correctly
4. Check if it's being blocked by a proxy

### Response showing garbled text?

1. Try changing the response encoding
2. Check the encoding settings in response headers
3. Use the "Format" function to re-parse

## Feature Questions

### How do I import Postman data?

1. Click "Import"
2. Select "Postman"
3. Choose the file to import
4. Confirm the import

### How do I export to Swagger?

1. Go to project settings
2. Select "Export"
3. Choose "Swagger/OpenAPI"
4. Select the version
5. Click Export

### How do I use environment variables?

1. Define environment variables in project settings
2. Use {{variableName}} in requests to reference them
3. Switch environments using the top environment selector

### How do I batch operate on interfaces?

1. Hold Ctrl/Cmd to select multiple interfaces
2. Right-click to select batch operations
3. Choose the operation type (delete, export, move, etc.)

## Testing Issues

### How do I run automated tests?

1. Open the test suite
2. Click the "Run" button
3. View the test results

### What do I do when an assertion fails?

1. Check the failure reason
2. Compare expected vs actual values
3. Modify the assertion condition
4. Run the test again

### How do I export test reports?

1. After running tests completes
2. Click "Export Report"
3. Choose the format (HTML, PDF, JSON)

## Team Collaboration Issues

### How do I invite team members?

1. Go to project settings
2. Click "Member Management"
3. Click "Invite Member"
4. Enter the member's email or username

### How do I set member permissions?

When inviting members, you can set different roles:

- Admin: Full control
- Editor: Can edit and run
- Viewer: Read-only access

### How do I resolve conflicts?

When multiple people edit the same interface simultaneously:

1. The system will alert you to the conflict
2. Choose to keep the local or remote version
3. Or manually merge the changes

## Data Issues

### How do I back up data?

1. Regularly export the project in Apifox format
2. Or use the cloud sync feature

### How do I restore data?

1. If you have a backup file, click "Import"
2. Select the backup file to restore

### Can I recover deleted data?

Deleted data stays in the trash for 30 days—you can restore it from the trash.

## Performance Issues

### What if it runs laggy?

1. Close projects you don't need
2. Clear browser cache
3. Upgrade your hardware
4. Try the latest version

### What if importing large amounts of data is slow?

1. Import in batches
2. Turn off auto-sync
3. Be patient and wait for the import to complete
