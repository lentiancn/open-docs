# APISIX Resumen

API Gateway nativo para la nube.

## ¿Qué es APISIX?

APISIX es un API gateway nativo para la nube desarrollado por la Apache Software Foundation, que proporciona servicios de gestión de API de alto rendimiento y alta disponibilidad. Como un API gateway dinámico, en tiempo real y escalable, APISIX se ha convertido en la solución preferida para millones de despliegues en contenedores.

### Características Principales

- **Alto Rendimiento**: Basado en Nginx + Lua, 10k+ QPS por núcleo
- **Enrutamiento Dinámico**: Actualizar reglas de ruta sin reiniciar
- **Sistema de Plugins**: 70+ plugins con soporte hot-loading
- **Descubrimiento de Servicios**: Soporte para Consul, Eureka, Nacos
- **Balanceo de Carga**: Roundrobin, hash, hash consistente
- **Seguridad**: Rate limiting, autenticación, JWT, firewall

### Arquitectura Técnica

APISIX utiliza Nginx como servidor proxy inverso HTTP subyacente, combinado con scripts Lua para orquestación dinámica.

### Estadísticas

- GitHub Stars: 13,000+
- Contenedores en producción: Millones

## Casos de Uso

- **Gestión Unificada de API**: Gestionar todas las API internas
- **Gateway de Microservicios**: Punto de entrada para auth, rate limiting
- **Ingress de Service Mesh**: Integración con Istio, Linkerd
- **Control de Tráfico API**: Rate limiting, circuit breaking
