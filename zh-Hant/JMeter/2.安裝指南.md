# JMeter 安裝指南

本指南將詳細介紹如何在不同作業系統上安裝和設定 Apache JMeter。

## 環境要求

- **Java 執行環境**：JMeter 需要 Java 8 或更高版本執行
- **作業系統**：支援 Linux、Windows、Mac OSX 等任何 Java 相容作業系統
- **記憶體**：建議至少 4GB 可用記憶體（根據測試規模可能需要更多）
- **磁碟空間**：建議至少 1GB 可用空間

## 下載 JMeter

1. 造訪 JMeter 官方下載頁面：https://jmeter.apache.org/download_jmeter.cgi
2. 下載最新版本的 JMeter 二進位檔案（Binaries）
3. 建議同時下載 PGP 或 SHA-512 簽章檔案用於驗證下載完整性

## Windows 安裝步驟

### 解壓安裝

1. 找到下載的 Apache JMeter 壓縮包（apache-jmeter-x.x.zip）
2. 選擇一個合適的安裝目錄，建議路徑不包含空格和特殊字元
3. 解壓縮壓縮包到選定目錄
4. 解壓完成後，JMeter 已安裝完畢

### 驗證安裝

1. 進入 JMeter 安裝目錄的 bin 資料夾
2. 點擊 jmeter.bat（Windows 批次檔）
3. 如果一切正常，JMeter GUI 介面將會啟動

## Linux 和 Mac OSX 安裝步驟

### 解壓安裝

1. 找到下載的 Apache JMeter 壓縮包（apache-jmeter-x.x.tar.gz）
2. 使用以下命令解壓：
   ```bash
   tar -xf apache-jmeter-x.x.tar.gz
   ```
3. 解壓完成後，JMeter 已安裝完畢

### 設定環境變數（可選）

1. 開啟終端設定檔（~/.bashrc 或 ~/.profile）
2. 新增以下環境變數：
   ```bash
   export JMETER_HOME=/path/to/apache-jmeter-x.x
   export PATH=$PATH:$JMETER_HOME/bin:$JMETER_HOME/lib
   ```
3. 儲存並重新整理設定檔：
   ```bash
   source ~/.bashrc
   ```

### 驗證安裝

1. 在終端中執行：
   ```bash
   jmeter --version
   ```
2. 如果顯示版本資訊，表示安裝成功

## 啟動 JMeter

### 圖形介面模式

- **Windows**：點擊 bin 目錄下的 jmeter.bat
- **Linux/Mac**：在終端執行 jmeter 命令

### 命令列模式

對於負載測試，建議使用命令列模式以減少資源占用：

```bash
jmeter -n -t test_plan.jmx -l results.jtl -e -o /path/to/output
```

參數說明：
- -n：以非 GUI 模式執行
- -t：指定測試計畫檔案
- -l：指定結果檔案
- -e：在測試結束後生成報告
- -o：指定報告輸出目錄

## 目錄結構說明

- **bin**：包含可執行檔案和啟動指令稿
- **lib**：包含 JMeter 依賴的庫檔案
- **docs**：包含 JMeter 官方檔案
- **extras**：包含額外的輔助工具
- **printable_docs**：包含可列印格式的使用者手冊

## 依賴管理

JMeter 的 lib 目錄包含核心依賴，如需新增額外的外掛程式或庫：

1. 將 JAR 檔案複製到 lib 目錄
2. 部分外掛程式需要放入 lib/ext 目錄
3. 重新啟動 JMeter 使新外掛程式生效

## 常見安裝問題

- **Java 未找到**：確保已正確安裝 Java 並設定 JAVA_HOME 環境變數
- **記憶體不足**：可以透過修改 bin/jmeter.sh 或 jmeter.bat 中的 Heap size 參數來解決
- **啟動失敗**：檢查是否安裝了正確版本的 Java（Java 8 或更高版本）
