# Angular 安裝指南

> 環境設定

---

## 需求

| 環境 | 要求 |
|------|------|
| Node.js | 18.10.0+ |
| npm | 9.0.0+ |

---

## 安裝

```bash
npm install -g @angular/cli
ng new my-app
ng serve
```

---

## 命令

| 命令 | 說明 |
|------|------|
| ng new | 新專案 |
| ng serve | 開發伺服器 |
| ng build | 編譯 |
