# Element UI Installationsanleitung

Vollständige Element UI Installations- und Konfigurationsanleitung.

## Umgebungsanforderungen

### Software-Anforderungen

- Node.js: 12.0.0+
- npm: 6.0.0+ oder yarn: 1.22.0+
- Vue: 2.6.0+

## Installationsschritte

### Methode 1: npm-Installation (Empfohlen)

```bash
# Mit npm installieren
npm install element-ui --save

# Oder mit yarn
yarn add element-ui
```

### Methode 2: CDN-Einbindung

```html
<!-- Styles einbinden -->
<link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">

<!-- Vue einbinden -->
<script src="https://unpkg.com/vue"></script>

<!-- Element UI einbinden -->
<script src="https://unpkg.com/element-ui/lib/index.js"></script>
```

## Schnellstart

### Vollständiger Import

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(ElementUI)

new Vue({
  el: '#app',
  render: h => h(App)
})
```

### On-Demand-Import (Empfohlen)

Zunächst babel-plugin-component installieren:

```bash
npm install babel-plugin-component -D
```

babel.config.js konfigurieren:

```javascript
module.exports = {
  presets: [
    '@vue/cli-plugin-babel/preset'
  ],
  plugins: [
    [
      'component',
      {
        libraryName: 'element-ui',
        styleLibraryName: 'theme-chalk'
      }
    ]
  ]
}
```

Komponenten importieren:

```javascript
import { Button, Select, Option, Table, TableColumn } from 'element-ui'

Vue.use(Button)
Vue.use(Select)
Vue.use(Option)
Vue.use(Table)
Vue.use(TableColumn)
```

## Themenanpassung

### SCSS-Variablen verwenden

Variablendatei erstellen:

```scss
// styles/element-variables.scss

$--color-primary: #409EFF;
$--color-success: #67C23A;
$--color-warning: #E6A23C;
$--color-danger: #F56C6C;

@import 'element-ui/packages/theme-chalk/src/common/var.scss';
```

In main.js einbinden:

```javascript
import './styles/element-variables.scss'
```

## Internationalisierung

### Mehrsprachigkeit konfigurieren

vue-i18n installieren:

```bash
npm install vue-i18n -S
```

Konfigurieren:

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import VueI18n from 'vue-i18n'
import deLocale from 'element-ui/lib/locale/lang/de'

Vue.use(ElementUI, {
  i18n: (key, value) => i18n.t(key, value)
})

const messages = {
  'de': { message: 'Hallo' }
}

const i18n = new VueI18n({
  locale: 'de',
  messages
})

new Vue({ i18n }).$mount('#app')
```

## Häufige Probleme

### Komponenten werden nicht angezeigt?

1. Komponenten korrekt importieren
2. Vue.use() aufrufen
3. Browser-Konsole auf Fehler prüfen

### Styles werden nicht angewendet?

1. Stylesheet korrekt importieren
2. Auf globale Style-Überschreibungen prüfen
3. z-index-Ebenen überprüfen

### Icons werden nicht angezeigt?

```bash
# Icon-Bibliothek installieren
npm install element-ui-icons
```

Icons importieren:

```javascript
import 'element-ui-icons'
```

Verwendung:

```html
<i class="el-icon-search"></i>
```

## Deinstallieren

```bash
npm uninstall element-ui
```
