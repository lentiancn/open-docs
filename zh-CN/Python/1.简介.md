# Python 简介

> 简洁优雅的编程语言

---

## 项目简介

Python 是由 Guido van Rossum 于 1991 年发布的解释型编程语言。Python 以其简洁易读的语法、强大的标准库和丰富的第三方包著称，广泛用于 Web 开发、数据分析、人工智能、自动化脚本等领域。

---

## 语言特性

- **简洁易读**：语法简洁，接近自然语言
- **动态类型**：无需声明变量类型
- **解释型**：无需编译，直接运行
- **丰富的库**：强大的标准库和第三方包
- **跨平台**：支持 Windows、macOS、Linux
- **多范式**：支持面向对象、函数式、过程式编程

---

## 适用场景

- Web 开发（Django、Flask）
- 数据分析（Pandas、NumPy）
- 人工智能（TensorFlow、PyTorch）
- 自动化脚本
- 网络爬虫
- 科学计算
