# Apifox Обзор

Платформа API- всёв-одном.

## Что такое Apifox?

Apifox — платформа для API документации, отладки, Mock и тестирования.

### Функции

- **API документация** - Автогенерация
- **API отладка** - HTTP симулятор
- **Mock сервисы** - Создание Mock серверов
- **Автотесты** - API тестирование
- **Команда** - Управление проектом
