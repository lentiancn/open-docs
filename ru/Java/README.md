# Java

## Обзор

Java — это объектно-ориентированный язык программирования, поддерживаемый Oracle Corporation. Это один из самых популярных языков корпоративной разработки в мире. Java известна своей характеристикой «Напиши один раз, запускай везде» и имеет зрелую экосистему с широкими областями применения.

## Основные характеристики

- Объектно-ориентированный: Полностью объектно-ориентированный язык программирования
- Независимость от платформы: Кроссплатформенное выполнение через JVM
- Автоматическое управление памятью: Механизм сборки мусора
- Обширная библиотека классов: Комплексная стандартная API
- Поддержка многопоточности: Встроенная поддержка параллельного программирования
- Безопасность: Многоуровневые механизмы безопасности

## Быстрый старт

### Установка JDK

Посетите официальную страницу загрузки Oracle для получения JDK: https://www.oracle.com/java/technologies/downloads/

### Написание первой программы

Создайте HelloWorld.java:

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

### Компиляция и запуск

```bash
# Компиляция
javac HelloWorld.java

# Запуск
java HelloWorld
```
