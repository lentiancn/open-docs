## Введение

Angular — это TypeScript-фреймворк с открытым исходным кодом от Google для создания современных одностраничных приложений (SPA).

### Основные возможности

- Компонентная архитектура
- Двустороннее связывание данных
- Внедрение зависимостей
- Маршрутизация
- Формы с валидацией
- HTTP-клиент
- Встроенное тестирование
- Кроссплатформенность

## Компоненты

### Создание

```bash
ng generate component components/user-list
# или
ng g c components/user-list
```

### Пример компонента

```typescript
import { Component, Input, Output, EventEmitter } from '@angular/core';

@Component({
  selector: 'app-user-card',
  templateUrl: './user-card.component.html'
})
export class UserCardComponent {
  @Input() user: User | null = null;
  @Output() delete = new EventEmitter<number>();

  onDelete() {
    if (this.user) {
      this.delete.emit(this.user.id);
    }
  }
}
```

## Шаблоны

### Синтаксис

```html
<!-- Интерполяция -->
<h1>Привет, {{ name }}!</h1>

<!-- Привязка свойств -->
<img [src]="imageUrl">

<!-- Привязка событий -->
<button (click)="onClick()">Клик</button>

<!-- Двусторонняя привязка -->

```

### Структурные директивы

```html
<div *ngIf="isLoggedIn">Добро пожаловать!</div>

<li *ngFor="let user of users">{{ user.name }}</li>
```

### Pipes

```html
<p>{{ today | date:'fullDate' }}</p>
<p>{{ price | currency:'RUB' }}</p>
```

## Сервисы

### Создание

```bash
ng generate service services/user
```

### Пример сервиса

```typescript
import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';

@Injectable({ providedIn: 'root' })
export class UserService {
  constructor(private http: HttpClient) {}

  getUsers(): Observable<User[]> {
    return this.http.get<User[]>('https://api.example.com/users');
  }
}
```

## Маршрутизация

### Определение маршрутов

```typescript
import { Routes } from '@angular/router';

export const routes: Routes = [
  { path: '', redirectTo: '/home', pathMatch: 'full' },
  { path: 'home', loadComponent: () => import('./home/home.component').then(m => m.HomeComponent) },
  { path: '**', loadComponent: () => import('./not-found/not-found.component').then(m => m.NotFoundComponent) }
];
```

### Параметры маршрута

```typescript
constructor(private route: ActivatedRoute) {}

ngOnInit() {
  this.route.paramMap.subscribe(params => {
    const id = params.get('id');
  });
}
```

```typescript
  }
}
```

## Формы

### Template-driven

```typescript
@Component({
  selector: 'app-login',

    <form #form="ngForm" (ngSubmit)="onSubmit(form)">
```

## HTTP-клиент

### Настройка

```typescript
import { provideHttpClient } from '@angular/common/http';

export const appConfig: ApplicationConfig = {
  providers: [provideHttpClient()]
};
```

### Методы

```typescript
this.http.get<User[]>(url).subscribe();
this.http.post<User>(url, data).subscribe();
this.http.put<User>(`${url}/${id}`, data).subscribe();
this.http.delete(`${url}/${id}`).subscribe();
```

## Управление состоянием

### BehaviorSubject

```typescript
@Injectable({ providedIn: 'root' })
export class CartService {
  private items$ = new BehaviorSubject<CartItem[]>([]);
  
  getItems() { return this.items$.asObservable(); }
  addItem(item: CartItem) {
    this.items$.next([...this.items$.value, item]);
  }
}
```

### Signals (Angular 16+)

```typescript
count = signal(0);
doubled = computed(() => this.count() * 2);
increment() { this.count.update(c => c + 1); }
```

## Сборка и деплой

### Разработка
```bash
ng build
ng serve
```

### Продакшен
```bash
ng build --configuration production
```

### Деплой

**Netlify:**
```bash
npm install -g netlify-cli
netlify deploy --prod --dir=dist/your-app
```

**Docker:**
```dockerfile
FROM node:18-alpine as build
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build -- --configuration production

FROM nginx:alpine
COPY --from=build /app/dist/your-app/browser /usr/share/nginx/html
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
```
