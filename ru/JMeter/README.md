# Документация Apache JMeter

Добро пожаловать в документацию Apache JMeter. Эта папка содержит полную документацию по Apache JMeter — мощному инструменту с открытым исходным кодом для тестирования производительности и нагрузочного тестирования.

## Что такое Apache JMeter?

Apache JMeter™ — это 100% чистое Java-приложение, предназначенное для тестирования функционального поведения и измерения производительности. Изначально разработанное для тестирования веб-приложений, оно расширилось для охвата различных тестовых функций.

## Быстрый старт

1. **Скачать**: Получите JMeter по адресу https://jmeter.apache.org/
2. **Установить**: Распаковать и запустить `./jmeter.sh` (Linux/Mac) или `jmeter.bat` (Windows)
3. **Создать тест**: Используйте GUI для создания тестового плана
4. **Запустить**: Выполните тест и проанализируйте результаты

## Ключевые функции

- **Поддержка множества протоколов**: HTTP, FTP, JDBC, LDAP, JMS и другое
- **Тестовая IDE**: Запись и создание тестовых планов
- **Режим CLI**: Запуск тестов из командной строки
- **Отчёты**: Динамические HTML-отчёты
- **Расширяемость**: Поддержка плагинов и скриптов
- **Распределённое тестирование**: Распределение нагрузки между несколькими машинами

## Требования

- Java 8 или выше
- 4 ГБ+ ОЗУ рекомендуется
- Windows / Linux / macOS
