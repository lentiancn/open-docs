# Claude Code FAQ

Common questions and answers.

## Installation

### Q: Installation failed?

**Solutions**:
1. Check system requirements
2. Update VS Code to latest version
3. Check network connection
4. Try reinstalling

### Q: Cannot sign in?

**Solutions**:
1. Check if API key is valid
2. Check network connection
3. Try clearing cache

### Q: Extension not showing?

**Solutions**:
1. Restart VS Code
2. Check if extension is enabled
3. Check VS Code notifications

## Usage

### Q: Slow response?

**Solutions**:
1. Check network connection
2. Reduce context length
3. Wait for server response

### Q: Poor code generation quality?

**Solutions**:
1. Be more specific in descriptions
2. Provide more context
3. Iterate and improve

### Q: How to switch models?

Select model version in settings.

### Q: What languages are supported?

Supports mainstream languages: Python, JavaScript, TypeScript, Go, Rust, Java, C++ and more.

## Account

### Q: How to get API key?

Visit https://console.anthropic.com to create one.

### Q: Ran out of quota?

Purchase additional quota in console.

## Security

### Q: Will code be leaked?

Claude Code does not save or share your code.

### Q: How to protect privacy?

- Don't enter sensitive info in conversations
- Use local mode
- Clear sessions regularly
