# Vue 3 安裝文檔

在所有主流平台上安裝 Vue 3 的指南。

---

## CDN

```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
```

---

## NPM

```bash
npm install vue@3
```

---

## Vite（推薦）

```bash
npm create vite@latest my-vue3-app -- --template vue
cd my-vue3-app
npm install
npm run dev
```
