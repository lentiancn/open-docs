# FAQ Nginx

## Installation

### Vérifier la version
```bash
nginx -v
```

### Tester la configuration
```bash
nginx -t
```

## Configuration

### Recharger
```bash
nginx -s reload
```

## Performance

### Optimisation
```nginx
worker_processes auto;
worker_connections 4096;
gzip on;
```

## Sécurité

### Activer HTTPS
```nginx
server {
    listen 443 ssl;
    ssl_certificate cert.crt;
    ssl_certificate_key cert.key;
}
```

### Limitation de débit
```nginx
limit_req_zone $binary_remote_addr zone=one:10m rate=10r/s;
```

## Erreurs courantes

- **400** : Vérifier la taille de la requête
- **403** : Vérifier les permissions
- **404** : Vérifier le chemin
- **502** : Service backend non exécuté
- **504** : Timeout, augmenter proxy_read_timeout

## Proxy

### Obtenir l'IP réelle
```nginx
proxy_set_header X-Real-IP $remote_addr;
```
