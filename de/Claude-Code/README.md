# Claude Code

[Claude Code](https://code.claude.com/) ist ein intelligenter Programmierassistent von Anthropic.

## Übersicht

Claude Code ist ein KI-gesteuerter Programmierassistent in Terminal, IDE, Desktop-App und Browser.

## Funktionen

- **Aufgaben automatisieren**: Tests schreiben, Lint-Fehler beheben, Merge-Konflikte lösen
- **Funktionsentwicklung**: Beschreiben Sie Anforderungen; Claude Code plant und implementiert
- **Git-Integration**: Änderungen bereitstellen, Commits erstellen, Branches anlegen
- **MCP-Integration**: Externe Datenquellen und Tools verbinden
- **Anpassung**: CLAUDE.md, benutzerdefinierte Befehle, Hooks

## Schnellstart

### Installation

**macOS, Linux, WSL:**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell:**
```powershell
irm https://claude.ai/install.ps1 | iex
```

### Starten

```bash
cd your-project
claude
```
