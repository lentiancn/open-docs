# JDK Installation Guide

This guide covers how to install JDK (Java Development Kit) on Windows, macOS, and Linux.

## Version Overview

| Version | Type | Release Date | Support Until |
|---------|------|--------------|---------------|
| JDK 25 | Latest | 2026 | 6 months |
| JDK 21 | LTS | Sep 2023 | September 2029 |
| JDK 17 | LTS | Sep 2021 | September 2029 |
| JDK 11 | LTS | Sep 2018 | September 2031 |
| JDK 8 | Legacy | Mar 2014 | December 2030 |

**Recommendation:** Use JDK 21 (LTS) or JDK 17 (LTS) for production. JDK 25 for latest features.

## Install on Windows

### Option 1: Installer (.exe)

1. Download JDK from Oracle or Adoptium
2. Run the installer
3. Click "Next" through the wizard
4. Note the installation path (default: `C:\Program Files\Java\jdk-21`)

### Option 2: ZIP Archive

1. Download ZIP from Oracle or Adoptium
2. Extract to desired location (e.g., `C:\Java\jdk-21`)
3. No installation required

### Set Environment Variables

```cmd
# Open System Properties → Advanced → Environment Variables

# System Variables → New
JAVA_HOME = C:\Program Files\Java\jdk-21

# Edit Path variable, add:
%JAVA_HOME%\bin
```

### Verify Installation

```cmd
java -version
javac -version
```

## Install on macOS

### Option 1: DMG Installer

1. Download DMG from Oracle or Adoptium
2. Open the DMG file
3. Double-click the package installer
4. Follow the wizard

### Option 2: Homebrew

```bash
# Install Homebrew if not installed
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install JDK 21
brew install openjdk@21

# Link to system Java
sudo ln -sfn $(brew --prefix)/opt/openjdk@21/libexec/openjdk.jdk /Library/Java/JavaVirtualMachines/openjdk-21.jdk

# Add to PATH
echo 'export JAVA_HOME=$(/usr/libexec/java_home -v 21)' >> ~/.zshrc
source ~/.zshrc
```

### Option 3: SDKMAN

```bash
# Install SDKMAN
curl -s "https://get.sdkman.io" | bash
source ~/.sdkman/bin/sdkman-init.sh

# Install JDK 21
sdk install java 21.0.5-tem
```

### Verify Installation

```bash
java -version
javac -version
```

## Install on Linux

### Ubuntu/Debian

```bash
# Update package list
sudo apt update

# Install OpenJDK 21
sudo apt install openjdk-21-jdk

# Set JAVA_HOME
echo 'export JAVA_HOME=/usr/lib/jvm/java-21-openjdk-amd64' >> ~/.bashrc
source ~/.bashrc
```

### RHEL/CentOS/Fedora

```bash
# Install JDK 21
sudo dnf install java-21-openjdk-devel

# Set JAVA_HOME
echo 'export JAVA_HOME=/usr/lib/jvm/java-21-openjdk' >> ~/.bashrc
source ~/.bashrc
```

### Manual Installation (All Linux)

```bash
# Download JDK
wget https://download.oracle.com/java/21/latest/jdk-21_linux-x64_bin.tar.gz

# Extract
sudo tar -xzf jdk-21_linux-x64_bin.tar.gz -C /usr/local/

# Set JAVA_HOME
echo 'export JAVA_HOME=/usr/local/jdk-21' >> ~/.bashrc
source ~/.bashrc
```

### Verify Installation

```bash
java -version
javac -version
```

## Multiple JDK Versions

### Switch JDK on macOS

```bash
# List installed JDKs
/usr/libexec/java_home -V

# Switch to specific version
export JAVA_HOME=$(/usr/libexec/java_home -v 17)
```

### Switch JDK on Linux

```bash
# Using update-alternatives (Ubuntu/Debian)
sudo update-alternatives --config java

# Manual switch
export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64
export PATH=$JAVA_HOME/bin:$PATH
```

## IDE Configuration

### IntelliJ IDEA

1. File → Project Structure → Project
2. Select SDK: Add JDK → choose installed JDK
3. Apply → OK

### VS Code

1. Install "Extension Pack for Java"
2. Ctrl+Shift+P → Java: Configure Java Runtime
3. Select installed JDK

### Eclipse

1. Window → Preferences → Java → Installed JREs
2. Add → Standard VM
3. Browse to JDK installation folder
4. Set as default

## Common Issues

### "java" not recognized

- Verify JAVA_HOME is set correctly
- Check PATH includes `%JAVA_HOME%\bin`
- Restart terminal/command prompt

### Port 8080 already in use

```bash
# Windows
netstat -ano | findstr :8080

# Linux/macOS
lsof -i :8080
```

### JAVA_HOME points to JRE instead of JDK

- Ensure JAVA_HOME points to JDK (contains `bin/javac`)
- Not JRE (only contains `bin/java`)
