# Mockito Resumen

El principal framework de mocking para pruebas unitarias en Java.

## ¿Qué es Mockito?

Mockito es el principal framework de mocking para pruebas unitarias en Java. Permite a los desarrolladores crear y verificar objetos mock para escribir pruebas aisladas y eliminar dependencias de componentes externos.

### Características Principales

- **API intuitiva** - Fácil de aprender y usar
- **Mocking flexible** - Crear mocks para clases e interfaces
- **Verificación** - Verificar llamadas a métodos
- **Stubbing** - Definir valores de retorno predefinidos
- **Argument Matchers** - Verificación flexible de argumentos

## Conceptos Clave

- **Mock** - Un objeto sustituto que simula el comportamiento de un objeto real
- **Stub** - Respuestas predefinidas a llamadas de métodos
- **Verificación** - Comprobar si se llamaron métodos específicos
- **Argument Matcher** - Verificación flexible de argumentos

## Beneficios

- Mejora la calidad de las pruebas
- Permite pruebas aisladas
- Reduce la fragilidad de las pruebas
- Se integra bien con JUnit

## Casos de Uso

- Pruebas unitarias con componentes aislados
- Pruebas de capa de servicio
- Simulación de dependencias externas
- Pruebas de capa de acceso a datos
