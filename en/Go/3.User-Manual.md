# Go User Manual

> Complete guide to Go programming

## Quick Start

### First Go Program

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

Run the program:

```bash
go run main.go
```

## Basic Syntax

### Variable Declaration

```go
// Method 1: Type inference
var name = "Go"

// Method 2: Explicit type
var age int = 10

// Method 3: Short variable declaration (inside functions)
message := "Hello"
```

### Data Types

```go
// Integers
var i int = 10
var ui uint = 10

// Floating point numbers
var f float64 = 3.14

// Strings
var s string = "Hello"

// Booleans
var b bool = true

// Arrays
var arr [5]int = [5]int{1, 2, 3, 4, 5}

// Slices
slice := []int{1, 2, 3}

// Maps
m := map[string]int{"a": 1, "b": 2}
```

### Control Flow

```go
// if statement
if x > 10 {
    fmt.Println("Greater than 10")
} else {
    fmt.Println("Less than or equal to 10")
}

// for loop
for i := 0; i < 10; i++ {
    fmt.Println(i)
}

// switch statement
switch day {
case 1:
    fmt.Println("Monday")
default:
    fmt.Println("Other")
}
```

## Functions

### Function Definition

```go
// Basic function
func add(a, b int) int {
    return a + b
}

// Multiple return values
func divide(a, b int) (int, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}

// Named return values
func split(sum int) (x, y int) {
    x = sum * 4 / 9
    y = sum - x
    return
}
```

### Variadic Functions

```go
func sum(nums ...int) int {
    total := 0
    for _, n := range nums {
        total += n
    }
    return total
}

// Usage
sum(1, 2, 3, 4, 5)
```

## Concurrency

### Goroutines

```go
// Start a goroutine
go func() {
    fmt.Println("Hello from goroutine")
}()

// With arguments
go func(msg string) {
    fmt.Println(msg)
}("Hello")
```

### Channels

```go
// Create channel
ch := make(chan int)

// Send data
ch <- 10

// Receive data
value := <-ch

// Buffered channel
buffered := make(chan int, 10)
```

### Select

```go
select {
case msg := <-ch1:
    fmt.Println("Received:", msg)
case msg := <-ch2:
    fmt.Println("Received:", msg)
case <-time.After(time.Second):
    fmt.Println("Timeout")
}
```

## Error Handling

### Error Definition

```go
import "errors"

func divide(a, b int) (int, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}
```

### Error Checking

```go
result, err := divide(10, 0)
if err != nil {
    fmt.Println("Error:", err)
    return
}
fmt.Println("Result:", result)
```


## Testing

### Unit Tests

```go
package main

import "testing"

func TestAdd(t *testing.T) {
    result := add(2, 3)
    if result != 5 {
        t.Errorf("Expected 5, got %d", result)
    }
}
```

Run tests:

```bash
go test ./...
```

### Benchmark Tests

```go
func BenchmarkHello(b *testing.B) {
    for i := 0; i < b.N; i++ {
        Hello()
    }
}
```

Run benchmarks:

```bash
go test -bench=.
```

## Best Practices

1. Use Go Modules for dependency management
2. Follow Go code conventions (run `go fmt`)
3. Prefer channels for concurrency communication
4. Avoid global variables
5. Write unit tests
6. Handle errors promptly
7. Use `context` for request lifecycle management
