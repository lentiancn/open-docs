# JUnit 5 - Resumen

## Descripción general

JUnit 5 es la próxima generación de JUnit, un marco de pruebas fácil de programar para Java y JVM. Proporciona una base sólida para escribir pruebas en Java.

## ¿Qué es JUnit?

JUnit es un marco de código abierto para escribir y ejecutar pruebas unitarias. Es parte de la familia xUnit de marcos de pruebas y es el estándar de facto para probar aplicaciones Java.

## Características principales de JUnit 5

### Arquitectura modular
JUnit 5 se compone de tres módulos distintos:
- **JUnit Platform**: Base para lanzar marcos de pruebas en la JVM
- **JUnit Jupiter**: Nuevo modelo de programación para escribir pruebas
- **JUnit Vintage**: Motor para ejecutar pruebas JUnit 3 y 4

### Características modernas
- Nombres de prueba descriptivos
- Pruebas anidadas
- Pruebas parametrizadas integradas
- Pruebas repetidas
- Pruebas dinámicas
- Etiquetado y filtrado
