# Vue 2 使用ガイド

Vue 2 を効果的に使用するガイド。

---

## 基本的な概念

Vue 2 はユーザーインターフェースを構築するための JavaScript フレームワークです。

---

## インスタンス

```javascript
new Vue({
  el: '#app',
  data: {
    message: 'Hello Vue!'
  }
})
```

---

## テンプレート指令

| 指令 | 説明 |
|------|------|
| `v-if` | 条件レンダリング |
| `v-for` | リストレンダリング |
| `v-bind` | 属性バインディング |
| `v-model` | 双方向バインディング |

---

## コンポーネント

```vue
<template>
  <div>{{ title }}</div>
</template>

<script>
export default {
  props: {
    title: String
  }
}
</script>
```
