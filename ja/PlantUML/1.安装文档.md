# PlantUML インストールガイド

PlantUMLは、UML図を素早く作成するためのオープンソースツールです。このガイドでは、さまざまな環境でのインストール方法について説明します。

## バージョン情報

| バージョン | 状態 | 説明 |
|------|------|------|
| PlantUML | ✅ 現行 | 最新安定版 |

## インストール方法

### 方法一：Windows へのインストール

#### 1. JAR ファイルをダウンロード

1. PlantUML JARをダウンロード：https://sourceforge.net/projects/plantuml/files/plantuml.jar/download
2. ローカルディレクトリに保存

#### 2. Graphvizのインストール（オプション但推奨）

```powershell
# Chocolatey を使用
choco install graphviz
```

### 方法二：macOS へのインストール

```bash
# Homebrew を使用
brew install plantuml

# または Graphviz をインストール
brew install graphviz
```

### 方法三：Linux へのインストール

```bash
# Ubuntu/Debian
sudo apt-get install -y plantuml graphviz

# CentOS/RHEL
sudo yum install -y plantuml graphviz
```

### 方法四：VS Code プラグインとして

1. VS Codeを開く
2. "PlantUML" を検索
3. "PlantUML" プラグインをインストール
4. Graphvizをインストール

### 方法五：IntelliJ プラグインとして

1. IntelliJ IDEAを開く
2. Settings → Plugins
3. "PlantUML" を検索
4. プラグインをインストール

### 方法六：Docker を使用

```bash
# PlantUML サーバーを実行
docker run -d -p 8080:8080 plantuml/plantuml-server:jetty

# http://localhost:8080 にアクセス
```

## 環境設定

### Java の要件

```bash
# Java を確認
java -version

# Java 8 以上が必要
```

### Graphviz の設定

```bash
# 環境変数を設定（Linux/macOS）
export GRAPHVIZ_DOT=/usr/bin/dot

# Windows の場合は PATH に追加
# C:\Program Files\Graphviz\bin
```

## よくある質問

### 画像が表示されない？

Graphvizがインストールされていることを確認：

```bash
dot -V
```

### 日本語が文字化けする？

図に追加：

```plantuml
skinparam defaultFontName Meiryo
```
