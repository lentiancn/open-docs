# Apifox

Apifox는 중국 국내 최고 수준의 API 협업 플랫폼으로, API 설계, Mock, 테스트, 문서화 등의 문제를 하나의 플랫폼에서 해결할 수 있습니다. API 설계, API 문서 관리, API 디버깅, API Mock, API 자동화 테스트等功能을 통합하여 중국 국내에서 가장 사랑받는 API 관리 도구입니다.

## 빠른 시작

### 다운로드 및 설치

https://apifox.com/에서 사용하는 시스템에 맞는 설치 패키지를 다운로드하세요.

### 프로젝트 생성

1. Apifox 실행 후 로그인
2. "새 프로젝트" 클릭
3. 프로젝트 이름 입력

### 인터페이스 생성

1. 프로젝트에서 "새 인터페이스" 클릭
2. 인터페이스 정보 입력
3. 저장

### 인터페이스 디버깅

1. 인터페이스 열기
2. "보내기" 클릭
3. 응답 확인

## 핵심 기능

- **API 설계**: 시각적으로 API 인터페이스 정의
- **API 문서**: 자동 생성 및 다양한 형식 내보내기 지원
- **API 디버깅**: 강력한 온라인 디버깅 기능
- **API Mock**: 지능형 Mock, 자동 모의 데이터 생성
- **자동화 테스트**: 코드 작성 없이 API 테스트 완료
- **팀 협업**: 실시간 동기화, 권한 관리
