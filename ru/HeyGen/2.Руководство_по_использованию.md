# HeyGen Руководство по использованию

В этом руководстве описано, как использовать HeyGen для создания видео с искусственным интеллектом.

## Начало работы

### Использование веб-интерфейса

1. **Войдите** в HeyGen по адресу https://www.heygen.com
2. **Выберите аватар** из библиотеки или загрузите собственный
3. **Выберите тип ввода**: Текст, Аудио или Видео
4. **Сгенерируйте видео** и скачайте его

### Использование API

```python
from heygen import HeyGen

heygen = HeyGen(api_key="your_api_key")

# Генерировать видео из текста
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="Hello, this is my first AI video!",
    voice_id="voice_id"
)

print(f"URL видео: {video.url}")
```

## Основные функции

### 1. Выбор аватара

```python
# Список доступных аватаров
avatars = heygen.list_avatars()
for avatar in avatars:
    print(f"{avatar.id}: {avatar.name}")

# Получить конкретный аватар
avatar = heygen.get_avatar(avatar_id="avatar_id")
```

### 2. Выбор голоса

```python
# Список доступных голосов
voices = heygen.list_voices()
for voice in voices:
    print(f"{voice.id}: {voice.name} ({voice.language})")

# Получить детали голоса
voice = heygen.get_voice(voice_id="voice_id")
```

### 3. Генерация видео

#### Из текста

```python
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="Ваш текст сценария здесь",
    voice_id="voice_id",
    background="office"  # Опционально: фоновая сцена
)
```

#### Из аудио

```python
video = heygen.generate_video_from_audio(
    avatar_id="avatar_id",
    audio_url="https://example.com/audio.wav"
)
```

#### Из фотографии (TalkingPhoto)

```python
video = heygen.generate_talking_photo(
    photo_url="https://example.com/photo.jpg",
    script="Ваш сценарий здесь",
    voice_id="voice_id"
)
```

### 4. Пакетная генерация

```python
scripts = [
    "Первый сценарий видео",
    "Второй сценарий видео",
    "Третий сценарий видео"
]

# Генерировать несколько видео
jobs = heygen.batch_generate(
    avatar_id="avatar_id",
    scripts=scripts,
    voice_id="voice_id"
)

# Проверить статус задачи
for job in jobs:
    status = heygen.get_job_status(job.id)
    print(f"Задача {job.id}: {status}")
```

## Расширенные функции

### Пользовательский аватар

```python
# Загрузить пользовательский аватар
avatar = heygen.upload_avatar(
    name="Мой пользовательский аватар",
    video_url="https://example.com/reference.mp4"
)
print(f"ID пользовательского аватара: {avatar.id}")
```

### Мгновенное клонирование голоса

```python
# Клонировать голос из аудиообразца
voice = heygen.clone_voice(
    name="Мой клон голоса",
    audio_url="https://example.com/voice_sample.wav"
)
print(f"ID клонированного голоса: {voice.id}")
```

### Редактирование видео

```python
# Получить детали видео
video = heygen.get_video(video_id="video_id")

# Обновить настройки видео
updated = heygen.update_video(
    video_id="video_id",
    title="Мое обновленное видео",
    description="Обновленное описание"
)

# Удалить видео
heygen.delete_video(video_id="video_id")
```

## Справочник параметров

### Параметры generate_video

| Параметр | Тип | Обязательно | Описание |
|----------|-----|-------------|----------|
| avatar_id | string | Да | ID аватара для использования |
| script | string | Да | Текст для произнесения |
| voice_id | string | Да | ID голоса |
| background | string | Нет | Фоновая сцена |
| speed | float | Нет | Скорость речи (0.5-2.0) |
| emotion | string | Нет | Стиль эмоции |
| caption | boolean | Нет | Включить субтитры |

### Доступные эмоции

| Эмоция | Описание |
|--------|----------|
| Neutral | Выражение по умолчанию |
| Happy | Радостное выражение |
| Sad | Грустное выражение |
| Angry | Злое выражение |
| Surprised | Удивленное выражение |

## Функции веб-интерфейса

### Панель управления

- **Проекты**: Управление видеопроектами
- **Шаблоны**: Использование готовых шаблонов
- **Медиатека**: Хранение материалов
- **Аналитика**: Просмотр показателей производительности

### Редактор видео

- **Временная шкала**: Редактирование видеосегментов
- **Субтитры**: Добавление/редактирование субтитров
- **Брендинг**: Добавление логотипов и водяных знаков
- **Экспорт**: Несколько форматов на выбор

## Устранение неполадок

### Проблемы с качеством видео

**Проблема:** Видео выглядит пиксельным

**Решения:**
- Используйте настройку более высокого разрешения
- Убедитесь, что исходное изображение высокого качества
- Проверьте подключение к интернету

### Проблемы с аудио

**Проблема:** Аудио не соответствует видео

**Решения:**
- Используйте предоставленные голоса
- Убедитесь, что аудио правильно отформатировано
- Проверьте настройки синхронизации губ

### Время обработки

**Проблема:** Видео обрабатывается слишком долго

**Решения:**
- Проверьте баланс кредитов
- Используйте очередь для пакетных заданий
- Рассмотрите приоритетную обработку
