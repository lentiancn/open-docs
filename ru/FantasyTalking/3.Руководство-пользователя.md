# FantasyTalking Руководство пользователя

## Основные функции

### Синтез речи

```javascript
const audio = await client.speak('Привет, это FantasyTalking');
```

### События

```javascript
client.on('speech-start', () => console.log('Начало'));
client.on('speech-end', () => console.log('Конец'));
```

### Диалог

```javascript
const session = client.createSession({ language: 'ru-RU' });
const response = await session.chat('Какая погода?');
```

### Потоковая передача

```javascript
const stream = await client.speakStream('Длинный текст');
stream.on('data', chunk => player.play(chunk));
```
