# Flutter

Flutter — это набор инструментов пользовательского интерфейса с открытым исходным кодом, разработанный Google для создания кроссплатформенных приложений.

## Описание

Flutter позволяет создавать красивые приложения для iOS, Android, Web и настольных компьютеров с использованием единой кодовой базы.

## Основные функции

- **Кроссплатформенность**: один код, несколько платформ
- **Горячая перезагрузка**: быстрая разработка и итерации
- **Богатые компоненты**: компоненты Material Design и Cupertino
- **Язык Dart**: легко изучить, отличная производительность
- **Декларативный UI**: создание красивых интерфейсов

## Быстрый старт

### Установка

```bash
# macOS
brew install flutter

# Linux
git clone https://github.com/flutter/flutter.git -b stable
```

### Создание проекта

```bash
flutter create my_app
cd my_app
flutter run
```
