# Flutter

Google의 오픈소스 UI 툴킷.

## 개요

Flutter는 Dart를 사용하는 크로스플랫폼 UI 툴킷.

## 기능

- 크로스플랫폼
- 핫 리로드
- 풍부한 위젯
- 고성능
