# Nginx 설치 가이드

모든 주요 플랫폼에 Nginx를 설치하는 가이드입니다.

---

## Linux

### Ubuntu / Debian

```bash
# 패키지 업데이트
sudo apt update

# Nginx 설치
sudo apt install nginx

# Nginx 시작
sudo systemctl start nginx

# 부팅 시 자동 시작
sudo systemctl enable nginx
```

### CentOS / RHEL

```bash
# EPEL 설치
sudo yum install epel-release

# Nginx 설치
sudo yum install nginx

# 시작
sudo systemctl start nginx
```

### Fedora

```bash
sudo dnf install nginx
sudo systemctl start nginx
```

---

## macOS

```bash
brew install nginx
brew services start nginx
```

---

## Windows

1. [nginx.org](http://nginx.org/en/download.html)에서 다운로드
2. `C:\nginx`에 압축 해제

```powershell
cd C:\nginx
start nginx
```

---

## Docker

```bash
docker run -d -p 80:80 --name nginx nginx:latest
```

---

## 확인

```bash
nginx -v
nginx -t
curl http://localhost
```

---

## 설정

| OS | 경로 |
|----|------|
| Ubuntu/Debian | `/etc/nginx/nginx.conf` |
| CentOS/RHEL | `/etc/nginx/nginx.conf` |

---

## 제거

```bash
sudo apt remove nginx
```
