# Häufig gestellte Fragen

Hier findest du Antworten auf die häufigsten technischen Fragen zu Bootstrap.

## Grundlagen

### Was ist Bootstrap?

Bootstrap ist ein Open-Source-Framework für die Frontend-Entwicklung, entwickelt und gepflegt von Twitter. Es bietet eine Fülle von CSS-Stilen und JavaScript-Komponenten, mit denen Entwickler schnell responsive Web-Anwendungen erstellen können.

### Was ist der Unterschied zwischen Bootstrap 5 und Bootstrap 4?

Die wichtigsten Unterschiede:

- Bootstrap 5 nutzt natives JavaScript, jQuery ist nicht mehr nötig
- Neuer XXL-Breakpoint hinzugefügt
- Verbesserte Formularsteuerelemente
- Font Awesome wurde durch Bootstrap Icons ersetzt
- Internet Explorer wird nicht mehr unterstützt

### Welche Umgebung braucht Bootstrap?

Ein moderner Browser genügt. Bootstrap 5 benötigt einen Browser mit ES6-Unterstützung.

## Installationsprobleme

### CDN lädt nicht?

1. Netzwerkverbindung prüfen
2. Andere CDNs ausprobieren (z.B. bootcdn, staticfile)
3. Dateien lokal herunterladen und einbinden

### npm Installation schlägt fehl?

```bash
# Cache leeren
npm cache clean --force

# China-Mirror nutzen
npm config set registry https://registry.npmmirror.com

# Erneut installieren
npm install bootstrap@5
```

### Styles werden nicht angezeigt?

1. CSS-Datei korrekt eingebunden?
2. Browser-Cache leeren (Strg+Shift+R erzwingen)
3. CSS-Konflikte prüfen

## Nutzungsprobleme

### Grid-System funktioniert nicht?

1. row muss innerhalb von container sein
2. col muss innerhalb von row sein
3. Breakpoint-Klassen richtig verwendet?

### Responsives Layout funktioniert nicht?

Viewport-Meta-Tag muss gesetzt sein:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### Navbar klappt auf mobilen Geräten nicht zusammen?

1. bootstrap.bundle.min.js (inkl. Popper) eingebunden?
2. data-bs-toggle und data-bs-target Attribute korrekt?

### Modal öffnet sich nicht?

1. Bootstrap JS eingebunden?
2. data-bs-toggle und data-bs-target am Button korrekt?
3. Modal-ID korrekt?

### Dropdown funktioniert nicht?

1. bootstrap.bundle.min.js eingebunden?
2. data-bs-toggle="dropdown" Attribut vorhanden?

## Style-Probleme

### Theme-Farben anpassen?

Mit Sass:

```scss
$primary: #007bff;
@import "bootstrap/scss/bootstrap";
```

### Standard-Styles überschreiben?

Mit Selektoren höherer Priorität:

```css
.navbar {
    background-color: #f8f9fa !important;
}
```

### Schriftart ändern?

```scss
$font-family-sans-serif: "Helvetica Neue", Helvetica, Arial, sans-serif;
@import "bootstrap/scss/bootstrap";
```

### Icons werden nicht angezeigt?

Bootstrap 5 nutzt Bootstrap Icons:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.10.0/font/bootstrap-icons.css">

<i class="bi bi-alarm"></i>
```

## Komponenten-Probleme

### Card zentrieren?

```html
<div class="card">
    <div class="card-body">
        Inhalt
    </div>
</div>
```

Mit Flex:

```html
<div class="card">
    <div class="card-body d-flex justify-content-center align-items-center">
        Zentrierter Inhalt
    </div>
</div>
```

### Formular-Validierung funktioniert nicht?

Bootstrap 5 nutzt die neue Formular-Validierung:

```html
<form class="row g-3 needs-validation" novalidate>
    <div class="col-md-4">
        <label for="validationCustom01" class="form-label">Vorname</label>
        <input type="text" class="form-control" id="validationCustom01" required>
        <div class="valid-feedback">Richtig!</div>
        <div class="invalid-feedback">Bitte Vornamen eingeben</div>
    </div>
    <div class="col-12">
        <button class="btn btn-primary" type="submit">Absenden</button>
    </div>
</form>

<script>
    (function () {
        'use strict'
        var forms = document.querySelectorAll('.needs-validation')
        Array.prototype.slice.call(forms).forEach(function (form) {
            form.addEventListener('submit', function (event) {
                if (!form.checkValidity()) {
                    event.preventDefault()
                    event.stopPropagation()
                }
                form.classList.add('was-validated')
            }, false)
        })
    })()
</script>
```

### Modal im Hintergrund schließen?

Attribut setzen:

```html
<div class="modal" data-bs-backdrop="static">
```

`static` – Klick auf Hintergrund schließt nicht
`true` – Klick schließt
`false` – Klick schließt nicht

## Performance-Probleme

### Dateigröße reduzieren?

1. Nur benötigte Komponenten einbinden (per Sass)
2. Produktionsversion nutzen
3. Gzip-Kompression aktivieren

### Styles nur bei Bedarf einbinden?

Mit Sass nur gewünschte Komponenten importieren:

```scss
# Notwendige Funktionen und Variablen einbinden
@import "bootstrap/scss/functions";
@import "bootstrap/scss/variables";

# Nur benötigte Komponenten einbinden
@import "bootstrap/scss/root";
@import "bootstrap/scss/reboot";
@import "bootstrap/scss/containers";
@import "bootstrap/scss/grid";
@import "bootstrap/scss/buttons";
@import/forms";
```

### Internet Explorer wird nicht unterstützt?

Bootstrap 5 unterstützt den Internet Explorer nicht mehr. Für IE11-Unterstützung: Bootstrap 4 nutzen.

### Verzögerung bei Klick auf Mobile?

Bootstrap 5 nutzt touch-action: manipulation, um die 300ms Klickverzögerung zu entfernen.
