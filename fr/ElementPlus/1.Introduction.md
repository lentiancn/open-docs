# Introduction à Element Plus

Element Plus est une bibliothèque de composants UI basée sur Vue 3.

## Caractéristiques

- 70+ composants
- Support TypeScript complet
- Importation à la demande
- Internationalisation
- Thèmes personnalisables

## Pour Commencer

```bash
npm install element-plus
```

Voir la [documentation officielle](https://element-plus.org/).
