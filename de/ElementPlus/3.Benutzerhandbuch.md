# Element Plus Benutzerhandbuch

Kernfunktionen und Anwendungstipps für Element Plus.

## Schnellstart

### Erstes Beispiel

```vue
<template>
  <el-button type="primary" @click="handleClick">
    Klick mich
  </el-button>
</template>

<script setup>
import { ref } from 'vue'

const handleClick = () => {
  ElMessage.success('Hallo Element Plus!')
}
</script>
```

### Form

```vue
<template>
  <el-form :model="form" :rules="rules" ref="formRef" label-width="100px">
    <el-form-item label="Benutzername" prop="username">
      <el-input v-model="form.username" />
    </el-form-item>
    
    <el-form-item label="E-Mail" prop="email">
      <el-input v-model="form.email" />
    </el-form-item>
    
    <el-form-item label="Rolle" prop="role">
      <el-select v-model="form.role" placeholder="Rolle auswählen">
        <el-option label="Administrator" value="admin" />
        <el-option label="Benutzer" value="user" />
      </el-select>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm">Absenden</el-button>
      <el-button @click="resetForm">Zurücksetzen</el-button>
    </el-form-item>
  </el-form>
</template>

<script setup>
import { ref, reactive } from 'vue'

const formRef = ref()
const form = reactive({
  username: '',
  email: '',
  role: '',
})

const rules = {
  username: [{ required: true, message: 'Bitte Benutzername eingeben', trigger: 'blur' }],
  email: [
    { required: true, message: 'Bitte E-Mail eingeben', trigger: 'blur' },
    { type: 'email', message: 'Bitte gültige E-Mail eingeben', trigger: 'blur' }
  ],
  role: [{ required: true, message: 'Bitte Rolle auswählen', trigger: 'change' }],
}

const submitForm = async () => {
  const valid = await formRef.value.validate()
  if (valid) {
    console.log('Erfolgreich', form)
  }
}

const resetForm = () => {
  formRef.value.resetFields()
}
</script>
```

## Best Practices

### Formularvalidierung

```typescript
const rules = {
  name: [
    { required: true, message: 'Bitte Name eingeben', trigger: 'blur' },
    { min: 2, max: 20, message: '2-20 Zeichen', trigger: 'blur' }
  ],
  email: [
    { required: true, message: 'Bitte E-Mail eingeben', trigger: 'blur' },
    { type: 'email', message: 'Gültige E-Mail erforderlich', trigger: 'blur' }
  ],
}
```

### Tabellen-Pagination

```vue
<template>
  <el-table :data="paginatedData">
    <el-table-column prop="name" label="Name" />
  </el-table>
  
  <el-pagination
    v-model:current-page="currentPage"
    v-model:page-size="pageSize"
    :total="total"
    :page-sizes="[10, 20, 50, 100]"
    layout="total, sizes, prev, pager, next, jumper"
  />
</template>

<script setup>
import { ref, computed } from 'vue'

const currentPage = ref(1)
const pageSize = ref(10)
const tableData = ref([/* viele Daten */])
const total = computed(() => tableData.value.length)

const paginatedData = computed(() => {
  const start = (currentPage.value - 1) * pageSize.value
  return tableData.value.slice(start, start + pageSize.value)
})
</script>
```

## Debugging-Tipps

### Komponentenprobleme überprüfen

```bash
npm list element-plus
```

### Häufige Fehler

**Komponente nicht registriert**: Import oder Auto-Import-Konfiguration prüfen

**Stile nicht angewendet**: CSS-Konflikte prüfen, `:deep()` versuchen

**Typfehler**: `element-plus/global` Types installieren
