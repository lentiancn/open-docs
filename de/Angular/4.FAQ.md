# Angular FAQ

## Installationsprobleme

### Q: CLI Installation fehlgeschlagen?

**Lösungen:**
1. Node.js Version prüfen
```bash
node --version
```
2. npm Cache leeren
```bash
npm cache clean --force
```

## Leistung

### Q: Langsame Ladezeit?

**Lösungen:**
1. Lazy Loading verwenden
2. Produktionsmodus
```bash
ng build --configuration=production
```
