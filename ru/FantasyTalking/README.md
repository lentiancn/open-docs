# FantasyTalking

ИИ-инструмент для генерации видео Talking Head.

## Обзор

FantasyTalking — это ИИ-инструмент для генерации видео Talking Head, который использует технологию глубокого обучения для создания реалистичных видео говорящих голов из одного изображения и аудиовхода.

## Функции

- **Одно изображение**: Создавайте видео из одной фотографии
- **Многоязычность**: Поддержка китайского, английского, японского, корейского
- **Высокое качество**: Продвинутые модели глубокого обучения
- **Улучшение лица**: Встроенный модуль улучшения
- **Простота использования**: Простой CLI интерфейс
- **Открытый исходный код**: Бесплатно и открыто

## Быстрый старт

### Установка

```bash
git clone https://github.com/FantasyTalking/FantasyTalking.git
cd FantasyTalking
pip install -r requirements.txt
python download_models.py
```

### Генерация видео

```bash
python inference.py --source_image face.jpg --audio speech.wav --output output.mp4
```

## Технические характеристики

- **Вход**: Изображение человека + аудиофайл
- **Выход**: Видео говорящей головы
- **Поддерживаемые языки**: Китайский, английский, японский, корейский и др.
- **Формат вывода**: MP4, AVI
- **Разрешение вывода**: 256-1024px
