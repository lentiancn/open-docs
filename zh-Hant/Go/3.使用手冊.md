# Go 使用手冊

> Go 程式設計完整指南

## 變數和類型
```go
var name = "Go"
age := 10

// 類型
var i int = 10
var f float64 = 3.14
var s string = "Hello"
var b bool = true
```

## 函數
```go
func add(a, b int) int {
    return a + b
}

func divide(a, b int) (int, error) {
    if b == 0 {
        return 0, errors.New("除數為零")
    }
    return a / b, nil
}
```

## Goroutine 和 Channel
```go
go func() {
    fmt.Println("Hello from goroutine")
}()

ch := make(chan int)
ch <- 10
value := <-ch
```

## 測試
```go
import "testing"

func TestAdd(t *testing.T) {
    result := add(2, 3)
    if result != 5 {
        t.Errorf("Expected 5, got %d", result)
    }
}
```
