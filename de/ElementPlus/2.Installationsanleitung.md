# Element Plus Installationsanleitung

Verschiedene Installationsmethoden für Element Plus.

## Umgebungsanforderungen

### Hardware-Anforderungen

- CPU: 2 Kerne oder mehr
- Speicher: 4GB oder mehr

### Software-Anforderungen

- Node.js: 18.0+
- npm: 9.0+ / yarn: 1.22+ / pnpm: 8.0+
- Vue: 3.2+
- Vite: 4.0+ (empfohlen)

## Paketmanager-Installation (empfohlen)

### npm

```bash
npm install element-plus
```

### yarn

```bash
yarn add element-plus
```

### pnpm

```bash
pnpm add element-plus
```

## Vollständiger Import

### main.ts Konfiguration

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')
```

## Import bei Bedarf (empfohlen)

### Automatischer Import

**1. Abhängigkeiten installieren**

```bash
npm install -D unplugin-vue-components unplugin-auto-import
```

**2. Vite konfigurieren**

```typescript
// vite.config.ts
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

**3. Komponenten verwenden**

```vue
<template>
  <el-button type="primary">Hallo</el-button>
</template>
```

### Manuelle Import bei Bedarf

```typescript
import { ElButton, ElInput } from 'element-plus'

export default {
  components: {
    ElButton,
    ElInput,
  },
}
```

## Vue CLI Installation

### Installieren

```bash
npm install -D sass sass-loader
npm install element-plus
```

### Konfigurieren

```javascript
// vue.config.js
module.exports = {
  configureWebpack: {
    plugins: [
      AutoImport({
        resolvers: [ElementPlusResolver()],
      }),
      Components({
        resolvers: [ElementPlusResolver()],
      }),
    ],
  },
}
```

## CDN-Import

### HTML

```html
<!DOCTYPE html>
<html lang="de">
<head>
  <meta charset="UTF-8">
  <title>Element Plus CDN</title>
  <link rel="stylesheet" href="//unpkg.com/element-plus/dist/index.css" />
  <script src="//unpkg.com/vue@3"></script>
  <script src="//unpkg.com/element-plus"></script>
</head>
<body>
  <div id="app">
    <el-button type="primary">Primärer Button</el-button>
  </div>

  <script>
    const { createApp, ref } = Vue
    const App = {
      setup() {
        const visible = ref(false)
        return { visible }
      },
      template: `
        <el-button @click="visible = true">Dialog öffnen</el-button>
        <el-dialog v-model="visible" title="Titel">Inhalt</el-dialog>
      `
    }
    const app = createApp(App)
    app.use(ElementPlus)
    app.mount('#app')
  </script>
</body>
</html>
```

```typescript
import 'element-plus/theme-chalk/dark/css-vars.css'
```

## Häufige Probleme

### Icons werden nicht angezeigt?

Icon-Paket installieren:

```bash
npm install @element-plus/icons-vue
```

Registrieren:

```typescript
import * as icons from '@element-plus/icons-vue'

for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```

### Style-Konflikte?

CSS Modules oder scoped verwenden:

```vue
<style scoped>
:deep(.el-button) {
  margin: 10px;
}
</style>
```

### TypeScript-Typfehler?

Typdefinitionen installieren:

```bash
npm install -D @types/node
```

In `tsconfig.json` konfigurieren:

```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

### Internationalisierung

```typescript
import de from 'element-plus/es/locale/lang/de'

app.use(ElementPlus, { locale: de })
```

## Deinstallieren

### npm Deinstallieren

```bash
npm uninstall element-plus
```

### Konfiguration bereinigen

Entfernen Sie:
- `element-variables.scss` (falls verwendet)
- zugehörige Vite/Webpack-Konfigurationen
