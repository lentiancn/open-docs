# FantasyTalking User Manual

## Basic Usage

```javascript
// Play speech
ft.speak('Hello world')

// Stop
ft.stop()

// Pause
ft.pause()

// Resume
ft.resume()
```

## Events

```javascript
ft.on('start', () => {
  console.log('Started')
})

ft.on('end', () => {
  console.log('Finished')
})

ft.on('error', (err) => {
  console.error('Error:', err)
})
```

## Voice Settings

```javascript
ft.setSpeed(1.5)
ft.setPitch(1.2)
ft.setVolume(0.8)
```
