# Webpack Руководство по установке

Webpack - это сборщик модулей для современных JavaScript-приложений.

## Установка

```bash
npm install webpack webpack-cli --save-dev
```

## Настройка

```javascript
module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
  },
};
```
