# Python インストールガイド

> 環境のセットアップ

---

## インストール

### Windows

```powershell
choco install python
```

### macOS

```bash
brew install python@3.12
```

### Linux

```bash
sudo apt install python3 python3-pip
```

---

## パッケージ

```bash
pip install requests
python -m venv myenv
```

---

## 確認

```bash
python --version
```
