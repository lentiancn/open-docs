# Manuel d'utilisation Element Plus

Ce manuel vous aide à démarrer rapidement avec Element Plus.

## Démarrage rapide

Avant d'utiliser Element Plus, assurezvous que l'installation est complète.

### Utilisation de base

Utilisez les composants directement dans les composants Vue:

<template>
  <el-button>Bouton par défaut</el-button>
  <el-button type="primary">Bouton principal</el-button>
  <el-button type="success">Bouton succès</el-button>
  <el-button type="warning">Bouton avertissement</el-button>
  <el-button type="danger">Bouton danger</el-button>
</template>

## Composants courants

### Bouton

<el-button>Par défaut</el-button>
<el-button type="primary">Principal</el-button>
<el-button size="small">Petit</el-button>
<el-button disabled>Désactivé</el-button>
<el-button icon="Search">Rechercher</el-button>

### Entrée

<el-input v-model="input" placeholder="Veuillez saisir" />
<el-input v-model="input" prefix-icon="Search" />
<el-input v-model="input" clearable />

### Sélecteur

<el-select v-model="value" placeholder="Sélectionnez">
  <el-option label="Option 1" value="1" />
  <el-option label="Option 2" value="2" />
</el-select>

### Tableau

<el-table :data="tableData">
  <el-table-column prop="name" label="Nom" />
</el-table>

### Formulaire

<el-form :model="form">
  <el-form-item label="Nom">
    <el-input v-model="form.name" />
  </el-form-item>
</el-form>

### Dialogue

<el-dialog v-model="visible" title="Titre">
  <span>Contenu</span>
</el-dialog>

## Personnalisation des thèmes

:root {
  --el-color-primary: #409eff;
}

Pour plus d'informations, consultez le site officiel Element Plus.
