# Redis 概要

Redis（Remote Dictionary Server）は、オープンソースのメモリ内データ構造ストアです。データベース、キャッシュ、メッセージブローカーとして、さまざまなデータ構造（文字列、ハッシュ、リスト、セット、ソート済みセット、ビットマップ、ハイパーログログ、地理空間インデックス）をサポートします。

## Redisとは？

RedisはNoSQLデータベースで、データをキーと値のペアとして保存します。優れたパフォーマンス、柔軟性、丰富的機能で知られています。

## 主な特徴

### メモリ内ストレージ
Redisはすべてのデータをメモリ（RAM）に保存し、非常に高速な読み書き操作を提供します。

### データ構造
Redisは複数のデータ型をサポートしています：
- **文字列**：基本的なキーバリューストレージ
- **リスト**：文字列の順序付きコレクション
- **セット**：一意の文字列の順序なしコレクション
- **ソート済みセット**：ランキング用のスコア付きセット
- **ハッシュ**：文字列フィールドと値の間のマップ

### 永続性
Redisはオプションで永続化を提供します：
- RDB（Redis Database）スナップショット
- AOF（Append-Only File）ログ

### レプリケーション
Redisはデータ冗長性と読み取りスケーラビリティのためにマスター-slaveレプリケーションをサポートします。

## ユースケース

1. **キャッシング**：頻繁にアクセされるデータをメモリに保存
2. **セッション管理**：Webアプリケーションでユーザーセッションを管理
3. **リアルタイム分析**：高速データストリームを処理
4. **メッセージキュー**：pub/subパターンを実装
5. **レート制限**：APIリクエストレートを制御

## 利点

- **パフォーマンス**：読み書き操作のサブミリ秒レイテンシ
- **柔軟性**：さまざまなデータ構造をサポート
- **スケーラビリティ**：クラスタリングによる水平スケーリング
- **コミュニティ**：豊富なドキュメントとアクティブなコミュニティ
