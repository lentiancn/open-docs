# Jira

Jira는 300,000개 이상의 기업에 프로젝트 관리 솔루션을 제공하는 Atlassian이 개발한 프로젝트 관리 및 문제 추적 소프트웨어입니다.

## 빠른 시작

Atlassian 웹사이트를 방문하여 가입하고 팀에 적합한 플랜을 선택하세요. Jira 사이트를 빠르게 만들 수 있습니다.

## 주요 기능

프로젝트 관리 및 문제 추적, 애자일 개발 지원(Scrum, Kanban), 팀 협업 및 커뮤니케이션, Rovo AI 어시스턴트 통합, 다양한 타사 통합, 강력한 자동화 규칙.

## 가격

Free: 10명 영구 무료

Standard: 사용자당 월 $7.50

Premium: 사용자당 월 $14.50

Enterprise: 가격은 영업팀에 문의하세요
