# 사용 매뉴얼

## 일상적인 사용

### 프로젝트 초기화

```bash
yarn init -2
```

### 의존성 설치

```bash
yarn install
```

### 추가

```bash
yarn add lodash
yarn add -D typescript
```

### 제거

```bash
yarn remove lodash
```

### 업데이트

```bash
yarn up lodash
yarn up --all
```
