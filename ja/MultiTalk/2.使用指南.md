# MultiTalk ユーザーガイド

このガイドでは、MultiTalkを使用してマルチスピーカー音声コンテンツを生成する方法を説明します。

## クイックスタート

### 基本的な使用法

```bash
python inference.py \
  --text "みなさま、MultiTalkへようこそ。" \
  --speakers speaker1,speaker2 \
  --output result.wav
```

## 入力要件

### テキスト要件

- 言語：多言語サポート
- 形式：プレーンテキスト
- 長さ：厳密な制限なし

### スピーカー設定

`speakers.yaml`でスピーカーを定義:

```yaml
speakers:
  speaker1:
    voice_id: "voice_001"
    language: "ja"
  speaker2:
    voice_id: "voice_002"
    language: "ja"
```

## 上級用法

### カスタム音声ID

```bash
python inference.py \
  --text "これはカスタム音声デモです。" \
  --voice_ids voice001,voice002 \
  --output output.wav
```

### 速度の調整

```bash
python inference.py \
  --text "遅い話し方の例。" \
  --speakers speaker1 \
  --speed 0.8 \
  --output slow.wav
```

### バッチ処理

```bash
python batch_inference.py \
  --text_file texts.txt \
  --output_dir output/
```

## 設定

### 設定ファイル

`config.yaml`を編集:

```yaml
inference:
  sample_rate: 22050
  speed: 1.0
  
model:
  checkpoint: "checkpoints/multitalk.pth"
```

## ベストプラクティス

### 1. 適切な音声IDを使用する

スピーカーの特性に一致する音声IDを選択してください。

### 2. 言語の一致

テキストの言語が音声の言語と一致していることを確認してください。

### 3. 速度の最適化

自然な出力になるように速度を調整してください。

## トラブルシューティング

### 音声品質が悪い？

別の音声IDを試してください。

### 速度の問題？

速度パラメータを徐々に調整してください。
