# OpenFeign FAQ

## Questions Générales

### Qu'est-ce que OpenFeign?

OpenFeign est une bibliothèque Java qui offre des capacités de client REST déclaratif. Elle permet aux développeurs de définir des interfaces de service HTTP à l'aide dannotations.

### Quelle est la différence entre Feign et OpenFeign?

Feign a été initialement créé par Netflix. OpenFeign est la version open-source maintenue par la communauté.

---

## Installation

### Quelle version dois-je utiliser?

Pour les nouveaux projets, utilisez la dernière version stable (13.x).

### Comment activer OpenFeign dans Spring Boot?

Ajoutez l'annotation `@EnableFeignClients` à votre classe d'application principale.

---

## Dépannage

### Erreurs de timeout de connexion?

Augmentez les valeurs de timeout dans votre configuration:

```yaml
feign:
  client:
    config:
      default:
        connectTimeout: 10000
        readTimeout: 10000
```

### Comment déboguer les requêtes HTTP?

Activez la journalisation dans votre configuration:

```yaml
feign:
  client:
    config:
      default:
        loggerLevel: full
```
