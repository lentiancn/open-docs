# HeyGen Guide d'utilisation

Ce guide explique comment utiliser HeyGen pour la génération vidéo IA.

## Commencer

### Utiliser l'interface web

1. **Connectez-vous** à HeyGen sur https://www.heygen.com
2. **Sélectionnez un avatar** dans la bibliothèque ou téléchargez un personnalisé
3. **Choisissez le type d'entrée** : Texte, Audio ou Vidéo
4. **Générez la vidéo** et téléchargez-la

### Utiliser l'API

```python
from heygen import HeyGen

heygen = HeyGen(api_key="your_api_key")

# Générer une vidéo à partir du texte
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="Hello, this is my first AI video!",
    voice_id="voice_id"
)

print(f"URL de la vidéo: {video.url}")
```

## Fonctionnalités principales

### 1. Sélection d'avatar

```python
# Lister les avatars disponibles
avatars = heygen.list_avatars()
for avatar in avatars:
    print(f"{avatar.id}: {avatar.name}")

# Obtenir un avatar spécifique
avatar = heygen.get_avatar(avatar_id="avatar_id")
```

### 2. Sélection de voix

```python
# Lister les voix disponibles
voices = heygen.list_voices()
for voice in voices:
    print(f"{voice.id}: {voice.name} ({voice.language})")

# Obtenir les détails de la voix
voice = heygen.get_voice(voice_id="voice_id")
```

### 3. Génération vidéo

#### À partir du texte

```python
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="Votre texte de script ici",
    voice_id="voice_id",
    background="office"  # Optionnel : scène de fond
)
```

#### À partir de l'audio

```video = heygen.generate_video_from_audio(
    avatar_id="avatar_id",
    audio_url="https://example.com/audio.wav"
)
```

#### À partir d'une photo (TalkingPhoto)

```python
video = heygen.generate_talking_photo(
    photo_url="https://example.com/photo.jpg",
    script="Votre script ici",
    voice_id="voice_id"
)
```

### 4. Génération par lots

```python
scripts = [
    "Premier script vidéo",
    "Deuxième script vidéo",
    "Troisième script vidéo"
]

# Générer plusieurs vidéos
jobs = heygen.batch_generate(
    avatar_id="avatar_id",
    scripts=scripts,
    voice_id="voice_id"
)

# Vérifier le statut des tâches
for job in jobs:
    status = heygen.get_job_status(job.id)
    print(f"Tâche {job.id}: {status}")
```

## Fonctionnalités avancées

### Avatar personnalisé

```python
# Téléverser un avatar personnalisé
avatar = heygen.upload_avatar(
    name="Mon avatar personnalisé",
    video_url="https://example.com/reference.mp4"
)
print(f"ID de l'avatar personnalisé: {avatar.id}")
```

### Clonage vocal instantané

```python
# Cloner la voix à partir d'un échantillon audio
voice = heygen.clone_voice(
    name="Mon clone vocal",
    audio_url="https://example.com/voice_sample.wav"
)
print(f"ID de la voix clonée: {voice.id}")
```

### Édition vidéo

```python
# Obtenir les détails de la vidéo
video = heygen.get_video(video_id="video_id")

# Mettre à jour les paramètres vidéo
updated = heygen.update_video(
    video_id="video_id",
    title="Ma vidéo mise à jour",
    description="Description mise à jour"
)

# Supprimer la vidéo
heygen.delete_video(video_id="video_id")
```

## Référence des paramètres

### Paramètres generate_video

| Paramètre | Type | Requis | Description |
|-----------|------|--------|-------------|
| avatar_id | string | Oui | ID de l'avatar à utiliser |
| script | string | Oui | Texte à dire |
| voice_id | string | Oui | ID de la voix |
| background | string | Non | Scène de fond |
| speed | float | Non | Vitesse de parole (0.5-2.0) |
| emotion | string | Non | Style d'émotion |
| caption | boolean | Non | Activer les sous-titres |

### Émotions disponibles

| Émotion | Description |
|---------|-------------|
| Neutral | Expression par défaut |
| Happy | Expression joyeuse |
| Sad | Expression triste |
| Angry | Expression en colère |
| Surprised | Expression surprise |

## Bonnes pratiques

- Évitez la terminologie complexe

### 2. Sélection d'avatar

- Correspondance avatar/cas d'utilisation
- Tenez compte des préférences du public
- Testez plusieurs avatars

### 3. Sélection de voix

- Correspondance voix/marque

- Testez avec des exemples de scripts

### 4. Optimisation vidéo

- Utilisez une résolution appropriée
- Optimisez pour la plateforme (réseaux sociaux, site web)
- Ajoutez des sous-titres pour l'accessibilité

## Dépannage

### Problèmes de qualité vidéo

**Problème :** La vidéo semble pixelisée

**Solutions :**
- Utilisez un paramètre de résolution plus élevé
- Assurez-vous que l'image source est de haute qualité
- Vérifiez la connexion Internet

### Problèmes audio

**Problème :** L'audio ne correspond pas à la vidéo

**Solutions :**
- Utilisez les voix fournies
- Assurez-vous que l'audio est correctement formaté
- Vérifiez les paramètres de synchronisation labiale

### Temps de traitement

**Problème :** La vidéo prend trop de temps

**Solutions :**
- Vérifiez le solde de crédits
- Utilisez une file d'attente pour les tâches par lots
- Envisagez le traitement prioritaire
