# FantasyTalking Benutzerhandbuch

Dieses Handbuch beschreibt, wie Sie FantasyTalking zur Generierung von Talking-Head-Videos verwenden.

## Schnellstart

### Grundlegende Verwendung

```bash
python inference.py \
  --source_image images/face.jpg \
  --audio audio/speech.wav \
  --output results/output.mp4
```

## Eingabeanforderungen

### Bildanforderungen

- Format: JPG, PNG
- Auflösung: 512x512 oder höher
- Gesicht: Frontale Aufnahme, klare Merkmale
- Hintergrund: Einfacher Hintergrund bevorzugt

### Audioanforderungen

- Format: WAV, MP3
- Abtastrate: 16000 Hz oder höher
- Dauer: Kein Limit
- Sprache: Mehrsprachige Unterstützung

## Erweiterte Verwendung

### Gesichtsverbesserung

```bash
python inference.py \
  --source_image images/face.jpg \
  --audio audio/speech.wav \
  --output results/output.mp4 \
  --face_enhancer
```

### Stapelverarbeitung

```bash
python batch_inference.py \
  --image_dir images/ \
  --audio_dir audio/ \
  --output_dir results/
```

### Benutzerdefinierte FPS

```bash
python inference.py \
  --source_image images/face.jpg \
  --audio audio/speech.wav \
  --output results/output.mp4 \
  --fps 30
```

## Konfiguration

### Konfigurationsdatei

Bearbeiten Sie `config.yaml`:

```yaml
inference:
  face_enhancer: true
  fps: 25
  output_resolution: 512
  
model:
  checkpoint: "checkpoints/fantasytalking.pth"
```

## Modelltraining

### Datensatz vorbereiten

```bash
python prepare_dataset.py \
  --video_dir videos/ \
  --output_dir dataset/
```

### Training

```bash
python train.py \
  --config config/train.yaml \
  --epochs 100
```

## Best Practices

### 1. Hochwertige Bilder verwenden

- Verwenden Sie klare, frontale Fotos
- Gute Beleuchtungsbedingungen
- Hohe Auflösung bevorzugt

### 2. Audioqualität abstimmen

- Klares Audio ohne Hintergrundgeräusche
- Angemessene Abtastrate
- Übereinstimmung von Sprache und Gesichtsethnie

### 3. Nachbearbeitung

- Gesichtsverbesserung für bessere Qualität verwenden
- FPS nach Bedarf anpassen

## Fehlerbehebung

### Video zu dunkel?

Beleuchtung im Quellbild überprüfen.

### Audio nicht synchron?

Stellen Sie sicher, dass die Audio-Abtastrate korrekt ist.

### Schlechte Qualität?

Versuchen Sie die Gesichtsverbesserungsoption.
