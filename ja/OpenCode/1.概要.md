# OpenCode 概要

オープンソース AI コーディングエージェント。

## OpenCode とは？

OpenCode は、ターミナルインターフェース、デスクトップアプリ、または IDE 拡張として可以使用するオープンソース AI コーディングエージェントです。

### 主な機能

- **LSP 自動有効化** - 適切な LSP を自動ロード
- **マルチセッション** - 同じプロジェクトで複数のエージェントを並行起動
- **リンク共有** - 会話リンクを共有可能
- **GitHub Copilot** - GitHub ログインで Copilot を使用
- **ChatGPT Plus/Pro** - OpenAI ログインで ChatGPT を使用
- **任意のモデル** - ローカルモデル含め 75+ LLM プロバイダをサポート
- **任意のエディタ** - ターミナル、デスクトップ、IDE 拡張に対応
- **プライバシー優先** - コードやコンテキストデータを保存しない
