# Element Plus 使用指南

## 快速开始

```vue
<template>
  <el-button type="primary">按钮</el-button>
  <el-input v-model="input" />
</template>

<script setup>
import { ref } from 'vue'
const input = ref('')
</script>
```

## 组件

### 按钮

```vue
<el-button>默认</el-button>
<el-button type="primary">主要</el-button>
```

### 表单

```vue
<el-form :model="form">
  <el-form-item label="名称">
    <el-input v-model="form.name" />
  </el-form-item>
</el-form>
```
