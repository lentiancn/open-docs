# FantasyTalking Руководство по установке

## Требования

Перед установкой FantasyTalking убедитесь, что ваша среда разработки соответствует следующим требованиям. Для операционных систем проект поддерживает Windows, macOS и Linux. Рекомендуется использовать 64-битную систему для оптимальной производительности. Требуется Node.js версии 14.0 или выше и Python версии 3.7 или выше. При использовании Python-бэкенда также необходимо установить инструмент обработки аудио FFmpeg.

Для оборудования рекомендуется минимум 4 ГБ оперативной памяти и двухъядерный процессор для плавной работы синтеза речи.

## Способы установки

### Использование npm (Node.js)

```bash
npm install fantasy-talking
npm install fantasy-talking-ui
```

### Использование pip (Python)

```bash
pip install fantasy-talking
pip install fantasy-talking[all]
```

## Быстрый старт

```javascript
const client = new FantasyTalking({
  apiKey: 'your-api-key',
  voice: { style: 'neutral', language: 'ru-RU' }
});

const audio = await client.speak('Привет, это тест');
```

## Настройки

| Параметр | Тип | По умолчанию | Описание |
|----------|------|--------------|----------|
| apiKey | string | - | API-ключ (обязательно) |
| voice.style | string | neutral | Стиль голоса |
| voice.language | string | ru-RU | Код языка |
| voice.speed | number | 1.0 | Скорость |

## Развертывание

```bash
docker pull fantasytalking/server:latest
docker run -d -p 8080:8080 -e API_KEY=your-api-key fantasytalking/server:latest
```
