# Flutter - FAQ

Часто задаваемые вопросы и ответы по разработке на Flutter.

## Проблемы установки

### Q: Загрузка Flutter не удалась?

**Решение:**
1. Использовать зеркало:
```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
export PUB_CACHE=/путь/к/кэшу
```
2. Использовать прокси
3. Скачать SDK архив вручную

### Q: Ошибка flutter doctor?

**Решение:**
1. Проверить настройку PATH
2. Убедиться, что Git установлен
3. Запустить `flutter doctor -v` для детальной информации

### Q: Android SDK не найден?

**Решение:**
```bash
flutter config --android-sdk /путь/к/android-sdk

# Или установить переменную среды
export ANDROID_SDK_ROOT=/путь/к/android-sdk
```

### Q: Инструменты командной строки Xcode не установлены на macOS?

**Решение:**
```bash
xcode-select --install
```

## Проблемы разработки

### Q: Горячая перезагрузка не работает?

**Решение:**
1. Сохранить файл и подождать горячей перезагрузки
2. Проверить консоль на ошибки
3. Попробовать полностью перезапустить приложение

### Q: Сборка APK не удалась?

**Решение:**
1. Запустить `flutter clean`
2. Обновить Gradle
3. Проверить сетевое подключение

### Q: Запуск эмулятора не удался?

**Решение:**
1. Включить виртуализацию CPU
2. Проверить установку HAXM
3. Использовать физическое устройство для отладки

### Q: Виджет не отображается?

**Решение:**
1. Проверить возвращаемое значение метода build()
2. Подтвердить добавление в MaterialApp
3. Проверить ограничения макета

### Q: Состояние не обновляется?

**Решение:**
1. Убедиться, что setState() вызывается
2. Проверить StatefulWidget
3. Подтвердить использование правильного Context

### Q: Утечка памяти?

**Решение:**
1. Правильно утилизировать контроллеры
2. Очищать Provider правильно
3. Избегать создания объектов в build

## Проблемы производительности

### Q: Приложение тормозит?

**Решение:**
1. Использовать const конструкторы
2. Избегать ненужного setState
3. Использовать ListView.builder
4. Оптимизировать загрузку изображений

### Q: APK слишком большой?

**Решение:**
1. Использовать минификацию
2. Удалить символы отладки
3. Использовать split-per-abi
4. Очистить неиспользуемые ресурсы

### Q: Медленный запуск?

**Решение:**
1. Уменьшить операции init()
2. Ленивая загрузка компонентов
3. Использовать Deferred Components

## Проблемы платформы

### Q: Сборка iOS не удалась?

**Решение:**
1. Проверить версию Xcode
2. Обновить CocoaPods
3. Проверить сертификаты подписи

### Q: Сборка веба не удалась?

**Решение:**
1. Подтвердить поддержку Flutter Web
2. Проверить совместимость браузера
3. Обновить Chrome

### Q: Канал платформы не работает?

**Решение:**
1. Проверить соответствие имён методов
2. Подтвердить типы возвращаемых значений
3. Проверить ошибки в консоли
