# Angular

Angular est un framework Web moderne maintenu par Google.

## Introduction

Angular est un framework Web open-source TypeScript, conçu pour construire des applications monopages efficaces et complexes.

## Fonctionnalités principales

- **Architecture basée sur les composants**: Modèle de conception basé sur les composants
- **Support TypeScript**: Support natif TypeScript
- **Liaison de données bidirectionnelle**: Synchronisation automatique entre le modèle et la vue
- **Injection de dépendances**: Système puissant d'injection de dépendances
- **Système de routage**: Solution de routage complète
- **Client HTTP**: Client HTTP complet
- **Gestion des formulaires**: Formulaires réactifs et pilotés par modèle
- **Support de test**: Tests unitaires et end-to-end complets

## Démarrage rapide

### Installer Angular CLI

```bash
npm install -g @angular/cli
```

### Créer un nouveau projet

```bash
ng new my-app
cd my-app
ng serve
```
