# Apache JMeter - Aperçu

## Qu'est-ce qu'Apache JMeter ?

Apache JMeter™ est un logiciel open source, une application 100% pure Java conçue pour les tests de charge et la mesure de performance. Il a été initialement conçu pour tester les applications Web mais s'est depuis étendu à d'autres fonctions de test.

## Que peut faire JMeter ?

Apache JMeter peut être utilisé pour tester les performances sur les ressources statiques et dynamiques, les applications Web dynamiques. Il peut simuler une charge lourde sur un serveur, un groupe de serveurs, un réseau ou un objet pour tester sa force ou analyser les performances globales sous différents types de charge.

### Caractéristiques principales

- **Support multi-protocole** : Web (HTTP, HTTPS), Services Web SOAP/REST, FTP, Base de données via JDBC, LDAP, JMS, Mail, TCP, Objets Java
- **IDE de test complet** : Enregistrer des plans de test à partir de navigateurs ou d'applications natives
- **Mode CLI** : Tester la charge depuis n'importe quel OS compatible Java
- **Rapports HTML dynamiques** : Complets et prêts à présenter
- **Extraction de données** : Support pour HTML, JSON, XML et expressions régulières
- **100% Java** : Portabilité complète
- **Multi-threading** : Échantillonnage simultané par plusieurs threads
- **Highly Extensible** : Samplers enfichables et samplers scriptables (Groovy, BeanShell)
- **Intégration CI** : Support pour Maven, Gradle et Jenkins

## Ce que JMeter n'est pas

JMeter n'est pas un navigateur - il fonctionne au niveau du protocole. Il n'exécute pas le JavaScript trouvé dans les pages HTML et ne rend pas les pages HTML comme un vrai navigateur.

## Pour commencer

1. Téléchargez JMeter depuis le site officiel
2. Exécutez le script JMeter (jmeter.bat pour Windows, jmeter.sh pour Linux/Mac)
3. Créez votre plan de test en utilisant l'interface GUI
4. Exécutez les tests et analysez les résultats
