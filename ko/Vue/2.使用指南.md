# Vue 사용 가이드

Vue.js를 효과적으로 사용하는 방법을 설명합니다.

## 기본 개념

### Vue 인스턴스

```javascript
import { createApp } from 'vue'

const app = createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
})

app.mount('#app')
```

## 반응성

### Vue 3 Composition API (권장)

```javascript
import { ref, reactive, computed } from 'vue'

export default {
  setup() {
    // 기본값에는 ref
    const count = ref(0)
    
    // 객체에는 reactive
    const state = reactive({
      name: 'Vue',
      version: 3
    })
    
    // 계산된 속성
    const doubleCount = computed(() => count.value * 2)
    
    return { count, state, doubleCount }
  }
}
```

## 컴포넌트

### 컴포넌트 생성

```vue
<template>
  <div class="my-component">
    <h2>{{ title }}</h2>
    <button @click="emitEvent">클릭</button>
  </div>
</template>

<script setup>
import { defineProps, defineEmits } from 'vue'

const props = defineProps({
  title: String
})

const emit = defineEmits(['click'])

const emitEvent = () => {
  emit('click', '컴포넌트에서 왔습니다!')
}
</script>
```

## 이벤트 처리

```html
<!-- 클릭 이벤트 -->
<button @click="handleClick">클릭</button>

<!-- 수식어 -->
<button @click.stop="handleClick">전파 중지</button>
<button @click.prevent="handleSubmit">기본 방지</button>

<!-- 키보드 수식어 -->
<input @keyup.enter="submit" />
```

## 계산된 속성

```javascript
import { ref, computed } from 'vue'

const firstName = ref('John')
const lastName = ref('Doe')

// computed - 캐시됨, 종속성이 변경될 때만 다시 평가
const fullName = computed(() => {
  return `${firstName.value} ${lastName.value}`
})
```

## 감시자

```javascript
import { ref, watch } from 'vue'

const count = ref(0)

// 단순 감시자
watch(count, (newValue, oldValue) => {
  console.log(`${oldValue}에서 ${newValue}로 변경됨`)
})
```

##生命周期 후크

### Vue 3 Composition API

| 후크 | 설명 |
|------|------|
| `setup()` | 다른 후크之前 호출 |
| `onMounted()` | 컴포넌트가 마운트된 후 |
| `onUpdated()` | 컴포넌트가 업데이트된 후 |
| `onUnmounted()` | 컴포넌트가 언마운트된 후 |

```javascript
import { onMounted, onUpdated, onUnmounted } from 'vue'

export default {
  setup() {
    onMounted(() => {
      console.log('마운트됨!')
    })
  }
}
```

## Vue Router

### 기본 설정

```javascript
import { createRouter, createWebHistory } from 'vue-router'
import Home from '../views/Home.vue'

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```

### 네비게이션

```vue
<template>
  <button @click="$router.push('/about'>정보</button>
  <router-link to="/">홈</router-link>
</template>
```

## Pinia 상태 관리

### 스토어 정의

```javascript
import { defineStore } from 'pinia'

export const useUserStore = defineStore('user', {
  state: () => ({
    name: 'John',
    email: 'john@example.com'
  }),
  getters: {
    displayName: (state) => `${state.name} (${state.email})`
  },
  actions: {
    updateName(newName) {
      this.name = newName
    }
  }
})
```
