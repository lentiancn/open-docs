# Element UI Installation Guide

> Complete environment setup

---

## Requirements

| Environment | Minimum | Recommended |
|-------------|---------|-------------|
| Node.js | 8.x | 10.x or 12.x |
| npm | 5.x | 6.x |
| Vue | 2.6.0 | 2.6.14 |

---

## Installation

### Using npm (Recommended)

```bash
npm install element-ui --save
```

### Using yarn

```bash
yarn add element-ui
```

### Specific version

```bash
npm install element-ui@2.15.13
```

---

## Importing

### Full import

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```

### On-demand import

1. Install babel-plugin-component:

```bash
npm install babel-plugin-component -D
```

2. Configure babel:

```json
{
  "plugins": [
    ["component", {
      "libraryName": "element-ui",
      "styleLibraryName": "theme-chalk"
    }]
  ]
}
```

3. Import components:

```javascript
import { Button, Form, Input } from 'element-ui'

Vue.use(Button)
Vue.use(Form)
Vue.use(Input)
```

---

## i18n Configuration

```javascript
import ElementUI from 'element-ui'
import locale from 'element-ui/lib/locale/lang/en'

Vue.use(ElementUI, { locale })
```
