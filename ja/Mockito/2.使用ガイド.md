# Mockito 使用ガイド

Mockito を使用した効果的な单元テストのガイド。

---

## 基本的な使い方

### Mock の作成

```java
List<String> mockList = Mockito.mock(List.class);
```

### Stubbing

```java
when(mockList.get(0)).thenReturn("first");
```

### 検証

```java
verify(mockList).add("element");
```

---

## @Mock アノテーション

```java
@Mock
private UserService userService;

@BeforeEach
void setUp() {
    MockitoAnnotations.openMocks(this);
}
```
