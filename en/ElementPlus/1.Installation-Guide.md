# Element Plus Installation Guide

Element Plus is a Vue 3 based component library for developers, designers and product managers.

## Version Overview

| Version | Vue Version | Status |
|---------|-------------|--------|
| Element Plus 2.x | Vue 3.2+ | Current - Recommended |
| Element Plus 1.x | Vue 3.0+ | Legacy |
| Element UI 2.x | Vue 2.x | Legacy (Vue 2) |

**Recommendation:** Use Element Plus 2.x for all new Vue 3 projects. Element UI is for Vue 2 projects only.

### Version Compatibility

| Element Plus | Vue Version | Node.js | Release Date |
|--------------|-------------|---------|--------------|
| 2.5.x | Vue 3.3+ | 16+ | 2024 |
| 2.4.x | Vue 3.2+ | 14+ | 2023 |
| 2.3.x | Vue 3.2+ | 14+ | 2023 |
| 2.2.x | Vue 3.2+ | 14+ | 2023 |

## Prerequisites

### System Requirements

| Requirement | Minimum | Recommended |
|-------------|---------|-------------|
| Node.js | 16.x | 18.x or 20.x |
| npm | 8.x | 9.x or 10.x |
| Vue | 3.2+ | 3.4+ |
| Vite | 4.x | 5.x |

### Browser Compatibility

Element Plus supports browsers that support the last 2 versions:

| Browser | Minimum Version |
|---------|-----------------|
| Chrome | 64+ (85+ for 2.5.0+) |
| Edge | 79+ (85+ for 2.5.0+) |
| Firefox | 78+ (79+ for 2.5.0+) |
| Safari | 12+ (14.1+ for 2.5.0+) |

**Note:** Element Plus does NOT support IE11. If you need IE11 support, use Element UI with Vue 2.

## Installation Methods

### Method 1: Package Manager (Recommended)

#### Using npm

```bash
npm install element-plus --save
```

#### Using yarn

```bash
yarn add element-plus
```

#### Using pnpm

```bash
pnpm install element-plus
```

#### Using cnpm (China mirror)

```bash
# Set mirror
npm config set registry https://registry.npmmirror.com

# Install
npm install element-plus --save
```

### Method 2: CDN Import

#### Using unpkg

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Element Plus CDN Demo</title>
  <!-- Import Vue 3 -->
  <script src="//unpkg.com/vue@3"></script>
  <!-- Import Element Plus styles -->
  <link rel="stylesheet" href="//unpkg.com/element-plus/dist/index.css">
  <!-- Import Element Plus -->
  <script src="//unpkg.com/element-plus"></script>
</head>
<body>
  <div id="app">
    <el-button>{{ message }}</el-button>
  </div>

  <script>
    const { createApp, ref } = Vue
    const app = createApp({
      setup() {
        const message = ref('Hello Element Plus!')
        return { message }
      }
    })
    app.use(ElementPlus)
    app.mount('#app')
  </script>
</body>
</html>
```

## Project Setup

### Step 1: Create Vue Project

#### Using Vite (Recommended)

```bash
# Create project
npm create vite@latest my-element-app -- --template vue-ts

# Navigate to project
cd my-element-app

# Install dependencies
npm install
```

#### Using Vue CLI

```bash
# Install Vue CLI
npm install -g @vue/cli

# Create project
vue create my-element-app

# Select Vue 3 when prompted
```

### Step 2: Install Element Plus

```bash
npm install element-plus
```

### Step 3: Configure in main.ts/main.js

#### Full Import (Simple but larger bundle)

```typescript
// main.ts
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')
```

#### On-Demand Import (Recommended for production)

First, install the auto-import plugins:

```bash
npm install -D unplugin-vue-components unplugin-auto-import
```

Configure Vite:

```typescript
// vite.config.ts
import { defineConfig } from 'vite'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

Configure TypeScript:

```json
// tsconfig.json
{
  "compilerOptions": {
    // ... other options
    "types": ["element-plus/global"]
  }
}
```

Configure Webpack:

```javascript
// webpack.config.js
const AutoImport = require('unplugin-auto-import/webpack')
const Components = require('unplugin-vue-components/webpack')
const { ElementPlusResolver } = require('unplugin-vue-components/resolvers')

module.exports = {
  plugins: [
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
}
```

### Step 4: Use Components

```vue
<!-- App.vue -->
<template>
  <div>
    <h1>{{ message }}</h1>
    <el-button type="primary" @click="handleClick">
      Click Me
    </el-button>
    
    <el-input v-model="input" placeholder="Enter text"></el-input>
    
    <el-table :data="tableData" style="width: 100%">
      <el-table-column prop="name" label="Name"></el-table-column>
      <el-table-column prop="age" label="Age"></el-table-column>
    </el-table>
  </div>
</template>

<script setup>
import { ref } from 'vue'

const message = ref('Hello Element Plus!')
const input = ref('')
const tableData = ref([
  { name: 'John', age: 25 },
  { name: 'Jane', age: 30 },
])
</script>
```

## Global Configuration

### Setting Default Size and zIndex

```typescript
// Full import configuration
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import App from './App.vue'

const app = createApp(App)
app.use(ElementPlus, { size: 'small', zIndex: 3000 })
```

### Using el-config-provider

```vue
<template>
  <el-config-provider :size="size" :z-index="zIndex">
    <app />
  </el-config-provider>
</template>

<script setup>
import { ref } from 'vue'
import { ElConfigProvider } from 'element-plus'

const size = ref('default')
const zIndex = ref(3000)
</script>
```

## Troubleshooting

### Components Not Found

**Problem:** Components not showing or getting errors

**Solutions:**
1. Ensure Element Plus is installed: `npm list element-plus`
2. Verify import in main.ts
3. For on-demand import, check vite.config.ts plugins configuration
4. Restart development server

### Styles Not Applied

**Problem:** Components styled incorrectly

**Solutions:**
1. Import CSS: `import 'element-plus/dist/index.css'`
2. Check for CSS conflicts in your project
3. Verify build tool configuration

### TypeScript Errors

**Problem:** TypeScript complains about Element Plus types

**Solutions:**
1. Add to tsconfig.json: `"types": ["element-plus/global"]`
2. Run `npm install -D @types/node`
3. Update TypeScript: `npm install -D typescript@latest`

### Memory Issues

**Problem:** Build running out of memory

**Solutions:**
1. Increase Node.js memory: `NODE_OPTIONS=--max_old_space_size=4096 npm run build`
2. Use on-demand import to reduce bundle size
3. Upgrade Node.js to latest LTS version

## Migration from Element UI (Vue 2)

### Key Differences

| Element UI (Vue 2) | Element Plus (Vue 3) |
|-------------------|---------------------|
| `el-button` | Same |
| `el-table` | Same |
| `el-form` | Same |
| `v-model` | Same syntax |
| `slot` | Same syntax |
| Events | Same |

### Package Name Changes

```javascript
// Vue 2 + Element UI
import { Button, Table } from 'element-ui'

// Vue 3 + Element Plus
import { Button, Table } from 'element-plus'
```

### Breaking Changes

1. **No longer supports IE11**
2. **Vue 3 Composition API support**
3. **TypeScript support improved**
4. **Some component props renamed**

## Best Practices

1. **Use on-demand import** - Reduces bundle size significantly
2. **Use Vite** - Faster development and build times
3. **Lock versions** - Use specific versions in production
4. **Use TypeScript** - Better type safety and IDE support
5. **Check browser support** - Ensure target browsers are supported
