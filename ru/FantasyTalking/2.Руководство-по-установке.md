# FantasyTalking - Руководство по установке

Полное руководство по установке.

## Требования

### Оборудование

- GPU: NVIDIA 8GB+
- Место: 20GB+
- Оперативная память: 16GB+

### Программное обеспечение

- ОС: Linux, Windows 10+, macOS
- Python: 3.8 - 3.10
- CUDA: 11.8+

## Установка

### Способ 1: Из исходников

```bash
git clone https://github.com/FantasyTalking/FantasyTalking.git
cd FantasyTalking
pip install -r requirements.txt
python download_models.py
```

### Способ 2: Docker

```bash
docker pull fantasytalking/fantasytalking:latest
docker run -it --gpus all -v /your/data:/data fantasytalking/fantasytalking:latest
```

### Способ 3: pip

```bash
pip install fantasytalking
```

## Проверка

```bash
python -c "import fantasytalking; print(fantasytalking.__version__)"
```

## Проблемы

### CUDA недоступен?

Установите совместимую версию PyTorch.

### Недостаточно памяти?

Используйте меньшие изображения.

## Удаление

```bash
pip uninstall fantasytalking
```
