# Angular

Angular est un framework open-source développé et maintenu par Google, conçu pour créer des applications web modernes. Il utilise TypeScript comme langage principal et offre des fonctionnalités puissantes comme l'architecture par composants, la liaison de données bidirectionnelle et l'injection de dépendances.

## Démarrage rapide

### Installer Angular CLI

```bash
npm install -g @angular/cli
```

### Créer un nouveau projet

```bash
ng new my-app
cd my-app
```

### Lancer le serveur de développement

```bash
ng serve
```

Accédez à http://localhost:4200

## Fonctionnalités principales

- **Architecture par composants** : composants UI modulaires et réutilisables
- **TypeScript** : support du typage fort pour une meilleure qualité de code
- **Injection de dépendances** : couplage faible, facile à tester
- **Liaison de données bidirectionnelle** : synchronisation automatique des données
- **Routage** : solution complète de navigation
- **Client HTTP** : puissantes capacités de requêtes réseau
- **Gestion des formulaires** : formulaires réactifs et pilotés par le modèle
- **RxJS** : support de la programmation réactive
