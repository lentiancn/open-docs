# Bootstrap 使用手冊

## 組件

### 按鈕

```html
<button class="btn btn-primary">主要</button>
<button class="btn btn-secondary">次要</button>
```

### 卡片

```html
<div class="card">
    <div class="card-body">
        <h5 class="card-title">標題</h5>
    </div>
</div>
```

### 表單

```html
<form>
    <input class="form-control">
    <button class="btn btn-primary">提交</button>
</form>
```
