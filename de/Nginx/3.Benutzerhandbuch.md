# Nginx Benutzerhandbuch

## Grundlegende Operationen
```bash
nginx              # Starten
nginx -s reload   # Konfiguration neu laden
nginx -t          # Konfiguration testen
```

## Reverse-Proxy
```nginx
location / {
    proxy_pass http://backend;
}
```

## Load Balancing
```nginx
upstream backend {
    server backend1;
    server backend2;
}
```
