# jQuery 使用手册

## 选择器

```javascript
$('#id').hide()
$('.class').show()
$('div').css('color', 'red')
```

## 事件

```javascript
$('#btn').click(function() {
    alert('Clicked!')
})
```

## Ajax

```javascript
$.ajax({
    url: '/api/data',
    success: function(data) {
        console.log(data)
    }
})
```
