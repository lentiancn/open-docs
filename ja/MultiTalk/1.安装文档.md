# MultiTalk インストールガイド

MultiTalkはマルチスピーカー音声生成ツールで、自然なマルチボイス音声コンテンツを作成します。

## バージョン情報

| バージョン | ステータス | 説明 |
|---------|--------|-------------|
| MultiTalk v1.0 | ✅ 現行バージョン | 初期リリース |

## システム要件

### ハードウェア要件

| 要件 | 最小要件 | 推奨要件 |
|-------------|---------|-------------|
| GPU | NVIDIA 4GB VRAM | NVIDIA 8GB+ VRAM |
| RAM | 8GB | 16GB+ |
| ストレージ | 5GB | 10GB+ SSD |

### ソフトウェア要件

| ソフトウェア | バージョン |
|----------|---------|
| Python | 3.8+ |
| CUDA | 11.0+ |

## インストール方法

### 方法1：ソースからインストール

#### 1. リポジトリをクローン

```bash
git clone https://github.com/MultiTalk/MultiTalk.git
cd MultiTalk
```

#### 2. 仮想環境を作成

```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
# または
venv\Scripts\activate  # Windows
```

#### 3. 依存関係をインストール

```bash
pip install -r requirements.txt
```

#### 4. モデルをダウンロード

```bash
bash scripts/download_models.sh
```

### 方法2：Dockerを使用

```bash
docker pull multitalk/multitalk:latest
docker run --gpus all -v /path/to/data:/data multitalk/multitalk:latest
```

## 設定

### 環境変数

```bash
export CUDA_VISIBLE_DEVICES=0
export PYTHONPATH=/path/to/MultiTalk:$PYTHONPATH
```

### モデルパス

`config.yaml`を編集:

```yaml
model:
  checkpoint_path: "checkpoints/multitalk.pth"
```

## クイックスタート

### マルチスピーカー音声を生成

```bash
python inference.py \
  --text "こんにちは、これはマルチスピーカーデモです。" \
  --speakers speaker1,speaker2 \
  --output output.wav
```

### パラメータ

```bash
--text         # 入力テキスト
--speakers     # スピーカーID（カンマ区切り）
--output       # 出力ファイルパス
--speed        # peech速度（デフォルト：1.0）
```

## よくある問題

### GPUが利用できない？

```bash
nvidia-smi
python -c "import torch; print(torch.cuda.is_available())"
```

### モデルのダウンロードに失敗した？

インターネット接続を確認するか、モデルを手動でダウンロードしてください。
