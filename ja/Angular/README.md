# Angular

モダウェブアプリケーション開発フレームワーク。

## 概要

Angularは、Googleが開発・メンテナンスするオープンソースのWebアプリケーションフレームワークで、モダウェブアプリケーション開発のための完全なソリューションを提供します。

### 主な機能

- **TypeScriptネイティブ**: TypeScriptで構築
- **コンポーネント指向**: メンテナンスとテストが容易
- **依存性注入**: 強力なDIシステム
- **ルーティング**: 完全なクライアントサイドルーティング
- **フォーム**: リアクティブフォームサポート
- **HTTPクライアント**: 完全なHTTPサービス

### 統計

- GitHubスター: 93,000以上
- 週間ダウンロード: 3,000,000以上

## クイックスタート

### Angular CLIをインストール

```bash
npm install -g @angular/cli
```

### 新規プロジェクト作成

```bash
ng new my-app
cd my-app
ng serve
```

### アプリケーションにアクセス

ブラウザで開く: http://localhost:4200

## 一般的なコマンド

```bash
# コンポーネント作成
ng g c my-component

# サービス作成
ng g s my-service

# 本番ビルド
ng build

# テスト実行
ng test
```

## コアコンセプト

- **コンポーネント**: ビューとロジックの組み合わせ
- **モジュール**: アプリケーションコードを整理
- **サービス**: ビジネスロジックをカプセル化
- **ルーティング**: ページナビゲーション
- **ディレクティブ**: HTMLを拡張
- **パイプ**: データ変換
