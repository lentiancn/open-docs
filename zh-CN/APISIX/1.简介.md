# APISIX 简介

云原生 API 网关。

## 什么是 APISIX？

APISIX 是 Apache 旗下的云原生 API 网关，提供高性能、高可用的 API 管理服务。

### 主要特性

- **高性能** - 基于 Nginx + Lua 实现
- **动态路由** - 动态更新路由规则
- **插件系统** - 丰富的插件生态
- **服务发现** - 支持多种服务发现机制
- **负载均衡** - 多种负载均衡策略
- **安全防护** - 限流、认证、防火墙
- **可视化仪表盘** - 直观的管理界面

### 统计数据

- GitHub 星标：13,000+
- 生产部署：数百万容器
