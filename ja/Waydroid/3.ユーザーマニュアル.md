# ユーザーマニュアル

## Android アプリケーションのインストールと実行

Waydroid は `waydroid app -h` コマンドを使用していくつかの操作を実行できます：

```bash
usage: waydroid app [-h] {install,remove,launch,list} ...

optional arguments:
  -h, --help            このヘルプメッセージを表示して終了

subaction:
  {install,remove,launch,list}
    install             単一のパッケージをコンテナにプッシュしてインストール
    remove              コンテナから単一のアプリパッケージを削除
    launch              単一のアプリケーションを起動
    list                インストール済みアプリケーションをリスト表示
```

### Android アプリケーションのインストール

以下のコマンドで Android アプリケーションをインストールできます：

```bash
waydroid app install xyz.apk
```

その後、デスクトップのアプリケーションメニューからアプリケーションを起動できます。

アプリの Android パッケージ名（`com.xxx.yyy`）がわかっている場合は、コマンドラインでも使用できます：

```bash
waydroid app launch com.foo.bar
```

### 注意事項

インターネット上で見つかる apk ファイルは通常 arm サポートのみを持ち、そのため x86_64 では動作しません。

グラフィカルにアプリケーションを取得するために [F-Droid](https://f-droid.org) をインストールすることをお勧めします。または GAPPS イメージをインストールした場合は Play ストアを使用できます。

## 共通コマンド

### Waydroid コマンドラインオプション

Waydroid はコンテナとセッションを制御するための豊富なコマンドラインオプションを提供します：

- `waydroid show-full-ui`：完全な Android UI を表示
- `waydroid session start`：Waydroid セッションを開始
- `waydroid session stop`：Waydroid セッションを停止
- `waydroid container start`：Waydroid コンテナを開始
- `waydroid container stop`：Waydroid コンテナを停止
- `waydroid logcat`：Android ログを表示
- `waydroid shell`：Android シェルに入る
- `waydroid prop set <property> <value>`：システムプロパティを設定
- `waydroid bugreport`：バグレポートログを生成

### システムプロパティ設定

`waydroid prop` コマンドでさまざまなシステムプロパティを設定できます：

- `persist.waydroid.multi_windows`：マルチウィンドウモードを有効化
- その他のプロパティは公式ドキュメントを参照

## 高度な機能

### カスタムイメージの使用

カスタムイメージを使用するには、以下の手順に従います：

1. [SourceForge](https://sourceforge.net/projects/waydroid/files/images/) からイメージをダウンロードするか、カスタムビルドした `system.img` と `vendor.img` を使用
2. イメージを `/etc/waydroid-extra/images/` ディレクトリにコピー
3. Waydroid を再初期化：`sudo waydroid init -f`

### ADB デバッグ

Waydroid は ADB デバッグをサポートしています。デフォルトでは ADB セキュアモードが有効になっており、セッション開始時に自動接続されません。接続するには以下のコマンドを使用します：

```bash
waydroid adb connect
```

その後、Android 側で認証を提供します。

### 通知転送

Waydroid は Android 通知をデスクトップ環境に転送することをサポートしており、互換性のあるバージョンに Android システムイメージを更新する必要があります。
