# Docker 安装指南

## Linux

```bash
# Ubuntu/Debian
curl -fsSL https://get.docker.com | sh
sudo systemctl start docker
sudo systemctl enable docker
```

## macOS

```bash
brew install --cask docker
```

## Windows

下载 [Docker Desktop](https://www.docker.com/products/docker-desktop) 安装。

## 验证

```bash
docker --version
docker-compose --version
```
