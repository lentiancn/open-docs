# Element Plus 常見問題

本頁面列出 Element Plus 使用過程中最常見的問題及解決方案。

## 安裝問題

### 安裝失敗

**症狀**：npm install 或 yarn add 失敗

**解決方案**：

1. 檢查 Node.js 版本，確保 >= 14.0.0
2. 清除 npm 緩存：
   ```bash
   npm cache clean --force
   ```
3. 刪除 node_modules 和 package-lock.json 後重新安裝

### TypeScript 類型錯誤

**症狀**：引入組件時提示類型錯誤

**解決方案**：

1. 確保安裝了 Volar 插件
2. 在 tsconfig.json 中添加：
   ```json
   {
     "compilerOptions": {
       "types": ["element-plus/global"]
     }
   }
   ```

## 使用問題

### 組件樣式不生效

**症狀**：組件樣式與預期不符

**解決方案**：

1. 確保已正確引入 CSS：
   ```javascript
   import 'element-plus/dist/index.css'
   ```
2. 檢查是否有全局樣式覆蓋

### 圖標不顯示

**症狀**：圖標組件顯示為空白或方塊

**解決方案**：

1. 安裝圖標庫：
   ```bash
   npm install @element-plus/icons-vue
   ```

2. 全局註冊圖標：
   ```javascript
   import * as icons from '@element-plus/icons-vue'
   for (const [key, component] of Object.entries(icons)) {
     app.component(key, component)
   }
   ```

### 按需導入不生效

**症狀**：打包後體積仍然很大

**解決方案**：

1. 檢查插件配置是否正確
2. 確保使用了正確的 resolver

### 國際化不生效

**症狀**：語言切換後組件文字未變化

**解決方案**：

1. 正確引入語言包：
   ```javascript
   import zhCn from 'element-plus/es/locale/lang/zh-cn'
   ```
2. 配置 locale：
   ```javascript
   app.use(ElementPlus, { locale: zhCn })
   ```

## 組件問題

### 表格問題

#### 表格排序不生效

確保設置了 prop 屬性：

```vue
<el-table-column prop="date" sortable />
```

#### 表格固定列錯位

確保表格有固定高度：

```vue
<el-table :data="data" height="300" />
```

### 表單問題

#### 表單驗證不觸發

確保 el-form-item 有 prop 屬性：

```vue
<el-form-item label="名稱" prop="name">
  <el-input v-model="ruleForm.name" />
</el-form-item>
```

### 對話框問題

#### 對話框滾動問題

在對話框內容過多時，確保添加滾動：

```vue
<el-dialog>
  <div style="max-height: 60vh; overflow-y: auto;">
    <!-- 內容 -->
  </div>
</el-dialog>
```

## 樣式問題

### 樣式衝突

**症狀**：自定義樣式被覆蓋

**解決方案**：

1. 使用更具體的選擇器
2. 使用 ::v-deep 或 :deep()

### 主題色不生效

檢查 CSS 變量是否被正確覆蓋：

```css
:root {
  --el-color-primary: #409eff;
}
```

確保在 element-plus 樣式之後加載。

## 效能問題

### 頁面卡頓

對於大數據量：

1. 使用 el-table-v2（虛擬表格）
2. 使用分頁加載數據
3. 避免在表格中使用複雜組件

### 打包體積過大

1. 使用按需導入
2. 移除未使用的插件

## Vue 版本問題

### 與 Vue 2 不兼容

Element Plus 僅支援 Vue 3。如果需要 Vue 2，請使用 Element UI。

### Composition API 不支援

確保使用 Vue 3.2+ 以獲得更好的 Composition API 支援。
