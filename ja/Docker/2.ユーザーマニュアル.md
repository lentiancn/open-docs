# Docker ユーザーマニュアル

## 基本概念

### イメージ
`ash
docker images
docker pull ubuntu:20.04
`

### コンテナ
`ash
docker run ubuntu:20.04
docker ps
`

## コマンド

`ash
docker run -it ubuntu /bin/bash
docker run -d nginx
docker run -p 8080:80 nginx
docker exec -it <container> /bin/bash
docker logs <container>
`

## Dockerfile

`dockerfile
FROM node:18-alpine
WORKDIR /app
COPY . .
RUN npm install
CMD ["node", "server.js"]
`
