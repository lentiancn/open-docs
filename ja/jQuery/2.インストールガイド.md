# jQuery インストールガイド

## jQuery の取得

### 最新バージョン

現在の最新バージョンは **jQuery 4.0.0** です。

#### 本番環境向け圧縮版（推奨）
- ダウンロード：[jquery-4.0.0.min.js](https://code.jquery.com/jquery-4.0.0.min.js)
- 本番環境向け、ファイルサイズ小、ロード速い

#### 開発・デバッグ版
- ダウンロード：[jquery-4.0.0.js](https://code.jquery.com/jquery-4.0.0.js)
- 完全なコード含まれ、読みやすくデバッグしやすい

#### Source Map ファイル
- ダウンロード：[jquery-4.0.0.min.map](https://code.jquery.com/jquery-4.0.0.min.map)
- ブラウザデバッガで元のソースコードを表示するために使用

#### Slim バージョン
SlimバージョンはAjaxとエフェクトモジュールを除いた小型バージョンです：
- [jquery-4.0.0.slim.min.js](https://code.jquery.com/jquery-4.0.0.slim.min.js) - 圧縮版
- [jquery-4.0.0.slim.js](https://code.jquery.com/jquery-4.0.0.slim.js) - 非圧縮版

Ajaxとアニメーション機能が必要ない場合、Slimバージョンを使用すると約30%ファイルサイズを削減できます。

## インストール方法

### 方法1：CDN読み込み（推奨）

jQuery CDNを使用するのが最も簡単な方法です：

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
```

Subresource Integrity（SRI）検証付きのバージョン（推奨、CDN改ざん防止）：

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js" 
        integrity="sha384-KJ3o2DKtIkvYIK3UENzmM7KCkRr/rE9/Qpg6aAZGJwFDMVNA/GpGFF93hXpG5KkN" 
        crossorigin="anonymous"></script>
```

完全なバージョンリストとSRIハッシュは [https://releases.jquery.com](https://releases.jquery.com) で取得できます。

### 方法2：npmインストール

Node.jsプロジェクトの場合、npmでインストール：

```bash
npm install jquery

インストール後、node_modules/jquery/dist/ に以下が含まれます：
- jquery-4.0.0.min.js - 圧縮版
- jquery-4.0.0.js - 非圧縮版
- jquery-4.0.0.min.map - Source Map
```

### 方法3：Yarnインストール

```bash
yarn add jquery
```

### 方法4：ローカルダウンロード

[jQueryダウンロードページ](https://jquery.com/download/)にアクセスし、必要なバージョンを右クリックして「名前を付けて保存」を選択します。

## jQueryのアップグレード

### jQuery Migrateプラグインの使用

jQueryをアップグレードする際は、jQuery Migrateプラグインを使用して互換性問題を特定することを推奨します：

1. 旧バージョンから1.9+または3.xへのアップグレード：[jQuery Migrate 1.4.1](https://code.jquery.com/jquery-migrate-1.4.1.min.js)を使用

2. jQuery 3.xから新しいバージョンへのアップグレード：[jQuery Migrate 3.6.0](https://code.jquery.com/jquery-migrate-3.6.0.min.js)を使用

使用方法：jQueryの後にMigrateプラグインを読み込み

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
<script src="https://code.jquery.com/jquery-migrate-3.6.0.min.js"></script>
```

Migrateはコンソールに互換性警告を出力し、問題点を段階的に修正できます。

### アップグレードガイド

バージョン間のアップグレードについては、公式ガイドを参照してください：
- [jQuery 4.0 アップグレードガイド](https://jquery.com/upgrade-guide/4.0/)
- [jQuery 3.0 アップグレードガイド](https://jquery.com/upgrade-guide/3.0/)
