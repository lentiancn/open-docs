# Element Plus 安裝指南

本指南詳細介紹在各種項目中安裝和使用 Element Plus 的方法。

## 環境要求

Element Plus 對運行環境有以下要求：

- Vue 3.0.0+
- Node.js 14.0.0+
- 現代瀏覽器（Chrome、Firefox、Safari、Edge）

## 安裝方式

### 使用包管理器

推薦使用 npm、yarn 或 pnpm 安裝：

```bash
# npm
npm install element-plus

# yarn
yarn add element-plus

# pnpm
pnpm add element-plus
```

## 完整引入

如果你對打包後的文件大小不是很在乎，那麼使用完整引入會更方便。

### Vue 3 項目

在 main.ts 中引入：

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')
```

### Volar 類型支持

如果使用 Volar 插件，需要在 tsconfig.json 中配置：

```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

## 按需導入

按需導入可以顯著減少項目體積。你需要使用額外的插件來導入要使用的組件。

### 自動導入（推薦）

首先安裝插件：

```bash
# npm
npm install -D unplugin-vue-components unplugin-auto-import

# yarn
yarn add -D unplugin-vue-components unplugin-auto-import

# pnpm
pnpm add -D unplugin-vue-components unplugin-auto-import
```

#### Vite 配置

在 vite.config.ts 中配置：

```typescript
import { defineConfig } from 'vite'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

#### Webpack 配置

在 webpack.config.js 中配置：

```javascript
const AutoImport = require('unplugin-auto-import/webpack')
const Components = require('unplugin-vue-components/webpack')
const { ElementPlusResolver } = require('unplugin-vue-components/resolvers')

module.exports = {
  plugins: [
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
}
```

### Nuxt 使用

對於 Nuxt 用戶，只需要安裝 @element-plus/nuxt 即可：

```bash
npm install -D @element-plus/nuxt
yarn add -D @element-plus/nuxt
pnpm add -D @element-plus/nuxt
```

然後將配置寫入 nuxt.config.ts：

```typescript
export default defineNuxtConfig({
  modules: ['@element-plus/nuxt'],
})
```

### 手動導入

Element Plus 提供了基於 ES Module 的開箱即用的 Tree Shaking 功能。

需要安裝 unplugin-element-plus 來導入樣式：

```bash
npm install -D unplugin-element-plus
```

在組件中手動導入<template>
 ：

```vue
 <el-button>I am ElButton</el-button>
</template>

<script>
import { ElButton } from 'element-plus'

export default {
  components: { ElButton },
}
</script>
```

Vite 配置：

```typescript
import { defineConfig } from 'vite'
import ElementPlus from 'unplugin-element-plus/vite'

export default defineConfig({
  plugins: [ElementPlus()],
})
```

## 全局配置

在引入 Element Plus 時，可以傳入全局配置對象。

### 完整引入配置

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import App from './App.vue'

const app = createApp(App)
app.use(ElementPlus, { size: 'small', zIndex: 3000 })
```

配置項說明：
- size：設置表單組件的默認尺寸，可選值：large、default、small
- zIndex：設置彈出組件的層級，默認值為 2000

### 按需引入配置

使用 el-config-provider 組件進行配置：

```vue
<template>
  <el-config-provider :size="size" :z-index="zIndex">
    <app />
  </el-config-provider>
</template>

<script>
import { defineComponent } from 'vue'
import { ElConfigProvider } from 'element-plus'

export default defineComponent({
  components: {
    ElConfigProvider,
  },
  setup() {
    return {
      zIndex: 3000,
      size: 'small',
    }
  },
})
</script>
```

## 開始使用

安裝完成後，你可以在項目中使用任何 Element Plus 組件：

```vue
<template>
  <el-button type="primary">主要按鈕</el-button>
  <el-input v-model="input" placeholder="請輸入內容" />
  <el-table :data="tableData" style="width: 100%">
    <el-table-column prop="date" label="日期" width="180" />
    <el-table-column prop="name" label="姓名" width="180" />
    <el-table-column prop="address" label="地址" />
  </el-table>
</template>

<script setup>
import { ref } from 'vue'

const input = ref('')
const tableData = ref([
  {
    date: '2016-05-02',
    name: '王小虎',
    address: '上海市普陀區金沙江路 1518 弄',
  },
])
</script>
```

## 升級指南

從舊版本升級時，請注意：

1. 查看官方遷移指南
2. 注意 breaking changes
3. 測試所有使用的組件

## 卸載

如果需要卸載 Element Plus：

```bash
npm uninstall element-plus
yarn remove element-plus
pnpm remove element-plus
```

如果使用了按需導入插件，也需要卸載：

```bash
npm uninstall unplugin-vue-components unplugin-auto-import
```
