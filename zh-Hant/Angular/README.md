# Angular

現代 Web 開發平台。

## 簡介

Angular 是由 Google 開發的設計平台，用於構建移動端和桌面端 Web 應用。使用 TypeScript/JavaScript，具有組件化、雙向綁定、依賴注入等特點。

### 主要特性

- **組件化** - 基於組件的架構
- **TypeScript** - 強類型支持
- **雙向綁定** - 簡化數據同步
- **依賴注入** - 強大的依賴管理系統
- **路由** - 完整的路由解決方案

## 快速開始

### 安裝

```bash
npm install -g @angular/cli
```

### 創建項目

```bash
ng new my-app
cd my-app
ng serve
```

訪問：http://localhost:4200
