# Guide d'Installation de Go

> Guide complet pour installer et configurer Go

---

## Configuration Requise

| Ressource | Exigence |
|-----------|----------|
| Système d'exploitation | Windows, macOS, Linux, FreeBSD |
| Architecture | amd64, arm64, 386, etc. |
| Espace disque | Environ 100 Mo |

---

## Méthodes d'Installation

### Installation Windows

#### Utiliser le MSI

1. Télécharger Go : https://go.dev/dl/
2. Exécuter le programme d'installation MSI
3. Suivre l'assistant d'installation
4. Ouvrir l'invite de commandes, taper `go version`

#### Utiliser Chocolatey

```powershell
choco install golang
```

### Installation macOS

#### Utiliser Homebrew

```bash
brew install go
```

#### Utiliser le DMG

1. Télécharger le package macOS : https://go.dev/dl/
2. Ouvrir le fichier package et l'exécuter
3. Déplacer le répertoire Go vers /usr/local/go : `sudo mv go /usr/local/`

### Installation Linux

#### Utiliser le Gestionnaire de Paquets

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install golang-go

# CentOS/RHEL
sudo yum install golang
```

#### Installation Manuelle

```bash
# Télécharger le binaire Go
wget https://go.dev/dl/go1.22.0.linux-amd64.tar.gz

# Extraire vers /usr/local
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go1.22.0.linux-amd64.tar.gz

# Ajouter à PATH
export PATH=$PATH:/usr/local/go/bin

# Vérifier l'installation
go version
```

## Configuration de l'Environnement

### Configurer GOPATH

```bash
# Linux/macOS
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin

# Windows (PowerShell)
$env:GOPATH = "$env:USERPROFILE\go"
$env:PATH += ";$env:GOPATH\bin"
```

### Configurer le Proxy Go Module

```bash
# Utiliser le miroir chinois
go env -w GOPROXY=https://goproxy.cn,direct

# Configurer le proxy
go env -w GOPROXY=https://proxy.golang.org,direct
```

### Vérifier l'Installation

```bash
# Voir la version
go version

# Voir les infos d'environnement
go env

# Exécuter le test
go version
```

## Commandes Courantes

```bash
# Initialiser le module
go mod init <nom-module>

# Télécharger les dépendances
go mod tidy

# Construire le projet
go build

# Exécuter le programme
go run main.go

# Exécuter les tests
go test ./...

# Installer les outils
go install
```

## Dépannage

### L'installation a échoué?

- Vérifier la compatibilité système
- Confirmer que la version téléchargée correspond à l'architecture
- Vérifier les permissions

### Commande non trouvée?

- Vérifier que Go est ajouté à la variable PATH
- Rouvrir le terminal
- Vérifier le chemin d'installation

### Téléchargement des dépendances lent?

- Configurer le miroir proxy chinois
- Utiliser un VPN
