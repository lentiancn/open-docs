# Claude Code 소개

Claude Code는 Anthropic에서 개발한 지능형 프로그래밍 어시스턴트로, 코드베이스를 읽고, 파일을 편집하고, 명령을 실행하며 개발 도구와 원활하게 통합될 수 있습니다. 터미널, IDE, 데스크톱 애플리케이션, 브라우저에서 사용할 수 있습니다.

## Claude Code란?

Claude Code는 AI 기반 코딩 어시스턴트로, 기능을 구축하고, 버그를 수정하고, 개발 작업을 자동화하는 데 도움이 됩니다. 전체 코드베이스를 이해하며 여러 파일과 도구에서 다양한 개발 작업을 완료할 수 있습니다.

## 주요 기능

Claude Code는 다음과 같은 핵심 기능을 제공합니다:

**개발 작업 자동화**
- 테스트되지 않은 코드에 대한 테스트 작성
- 전체 프로젝트의 린트 오류 수정
- 병합 충돌 해결
- 종속성 버전 업데이트
- 릴리스 노트 작성

**기능 개발 및 버그 수정**
- 자연어로 구현하려는 내용을 설명하세요
- Claude Code가 접근 방식을 계획하고, 파일에 걸쳐 코드를 작성하며, 구현을 확인합니다

**Git 통합**
- Claude Code에서 직접 Git 작업 수행
- 변경 사항 스테이징, 커밋 메시지 작성, 브랜치 생성, 풀 리퀘스트 열기

**MCP 통합**
- Model Context Protocol (MCP)는 AI 도구를 외부 데이터 소스에 연결하기 위한 공개 표준입니다
- MCP를 통해 Claude Code는 Google Drive에서 설계 문서를 읽고, Jira 티켓을 업데이트하고, Slack에서 데이터를 가져오거나 사용자 정의 도구를 사용할 수 있습니다

**사용자 정의 기능**
- CLAUDE.md를 사용하여 프로젝트의 코딩 표준, 아키텍처 결정, 선호 라이브러리, 검토 체크리스트를 설정하세요
- 반복 가능한 워크플로를 캡슐화하기 위한 사용자 정의 명령을 만드세요
- Hooks를 사용하여 파일 편집 후 자동 포맷하거나 커밋 전에 린트를 실행하세요

**멀티 에이전트 협업**
- 복잡한 작업에서 병렬로 작업할 여러 Claude Code 에이전트를 생성하세요
- 리드 에이전트가 작업을 조정하고, 하위 작업을 할당하고, 결과를 병합합니다

**CLI 유연성**
- 파이핑, 스크립팅, 자동화 지원
- 로그를 파이프하고, CI에서 실행하거나 다른 도구와 연결

## 사용 시나리오

Claude Code는 다양한 시나리오에서 사용할 수 있습니다:

| 시나리오 | 권장 방법 |
|----------|----------|
| 로컬 개발 | 터미널 CLI |
| IDE 내 코딩 | VS Code 또는 JetBrains |
| 장치 간 계속하기 | 원격 제어 |
| 모바일 작업 | 웹 또는 iOS 앱 |
| CI/CD 자동화 | GitHub Actions 또는 GitLab CI/CD |
| 팀 협업 | Slack 통합 |
| 브라우저 디버깅 | Chrome 확장 프로그램 |

## 계정 요구사항

Claude Code는 다음 계정 중 하나가 필요합니다:
- Claude Pro, Max, Teams 또는 Enterprise 구독
- Anthropic Console 계정
- 지원되는 서드파티 클라우드 제공자 (Amazon Bedrock, Google Vertex AI, Microsoft Foundry)

무료 Claude.ai 플랜에는 Claude Code 접근 권한이 포함되지 않습니다.
