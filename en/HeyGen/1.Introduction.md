# HeyGen Introduction

> HeyGen is an AI video generation platform for creating digital human videos

## Overview

HeyGen is an AI-powered video generation tool that converts text into digital human videos, supporting multiple languages and characters.

## Key Features

- **Digital Human Creation**: Create AI digital avatars
- **Text to Video**: Input text and generate videos
- **Voice Synthesis**: Multiple voice options
- **Multi-language Support**: Supports various languages
- **Template Library**: Rich video templates
