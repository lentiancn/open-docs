# Element UI Guía de Instalación

Guía completa de instalación y configuración de Element UI.

## Requisitos del entorno

### Requisitos de software

- Node.js: 12.0.0+
- npm: 6.0.0+ o yarn: 1.22.0+
- Vue: 2.6.0+

## Pasos de instalación

### Método 1: instalación npm (Recomendado)

```bash
# Usando npm
npm install element-ui --save

# O usando yarn
yarn add element-ui
```

### Método 2: Importación CDN

```html
<!-- Importar estilos -->
<link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">

<!-- Importar Vue -->
<script src="https://unpkg.com/vue"></script>

<!-- Importar Element UI -->
<script src="https://unpkg.com/element-ui/lib/index.js"></script>
```

## Inicio rápido

### Importación completa

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(ElementUI)

new Vue({
  el: '#app',
  render: h => h(App)
})
```

### Importación bajo demanda (Recomendado)

Primero instala babel-plugin-component:

```bash
npm install babel-plugin-component -D
```

Configura babel.config.js:

```javascript
module.exports = {
  presets: [
    '@vue/cli-plugin-babel/preset'
  ],
  plugins: [
    [
      'component',
      {
        libraryName: 'element-ui',
        styleLibraryName: 'theme-chalk'
      }
    ]
  ]
}
```

Importa componentes bajo demanda:

```javascript
import { Button, Select, Option, Table, TableColumn } from 'element-ui'

Vue.use(Button)
Vue.use(Select)
Vue.use(Option)
Vue.use(Table)
Vue.use(TableColumn)
```

## Personalización de temas

### Usando variables SCSS

Crear archivo de variables:

```scss
// styles/element-variables.scss

$--color-primary: #409EFF;
$--color-success: #67C23A;
$--color-warning: #E6A23C;
$--color-danger: #F56C6C;

@import 'element-ui/packages/theme-chalk/src/common/var.scss';
```

Importar en main.js:

```javascript
import './styles/element-variables.scss'
```

### Editor de temas en línea

Visita: https://element.eleme.io/#/es/theme

1. Selecciona o personaliza colores del tema
2. Haz clic en "Descargar"
3. Reemplaza los archivos del tema

## Internacionalización

### Configurar multilingüe

Instala vue-i18n:

```bash
npm install vue-i18n -S
```

Configurar:

```javascript
import Vue from 'vue'
import ElementUI from 'element-ui'
import VueI18n from 'vue-i18n'
import esLocale from 'element-ui/lib/locale/lang/es'

Vue.use(ElementUI, {
  i18n: (key, value) => i18n.t(key, value)
})

const messages = {
  'es': { message: 'hola' }
}

const i18n = new VueI18n({
  locale: 'es',
  messages
})

new Vue({ i18n }).$mount('#app')
```

## Problemas comunes

### ¿Los componentes no se muestran?

1. Confirma que los componentes están correctamente importados
2. Verifica si se llama a Vue.use()
3. Revisa la consola para errores

### ¿Los estilos no se aplican?

1. Confirma que el archivo de estilos está importado
2. Verifica si hay anulaciones de estilos globales
3. Verifica los niveles de z-index

### ¿Los iconos no se muestran?

```bash
# Instalar biblioteca de iconos
npm install element-ui-icons
```

Importar iconos:

```javascript
import 'element-ui-icons'
```

Uso:

```html
<i class="el-icon-search"></i>
```

## Desinstalar

```bash
npm uninstall element-ui
```
