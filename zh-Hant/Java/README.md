# Java

## 概述

Java 是由 Oracle 公司維護的物件導向程式語言，是全球最流行的企業級開發語言之一。Java 以「一次編寫，到處執行」的特性著稱，擁有成熟的生態系統和廣泛的應用領域。

## 主要特性

- 物件導向：完全物件導向的程式語言
- 平台無關：透過 JVM 實現跨平台執行
- 自動記憶體管理：垃圾回收機制
- 豐富的類別庫：龐大的標準 API
- 多執行緒支援：內建並發程式設計支援
- 安全性：多層安全機制

## 快速開始

### 安裝 JDK

造訪 Oracle 官方下載頁面取得 JDK：https://www.oracle.com/java/technologies/downloads/

### 撰寫第一個程式

建立 HelloWorld.java：

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

### 編譯與執行

```bash
# 編譯
javac HelloWorld.java

# 執行
java HelloWorld
```
