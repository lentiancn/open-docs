# OpenCode Guía de instalación

OpenCode soporta múltiples métodos de instalación.

## Instalación en Linux/macOS

### Recomendado: Script de instalación

```bash
curl -fsSL https://opencode.ai/install | bash
```

### Node.js

```bash
npm install -g opencode-ai
```

### Bun

```bash
bun install -g opencode-ai
```

### pnpm

```bash
pnpm install -g opencode-ai
```

### Yarn

```bash
yarn global add opencode-ai
```

### Homebrew (macOS/Linux)

```bash
brew install anomalyco/tap/opencode
```

### Arch Linux

```bash
# Estable
sudo pacman -S opencode

# Última (AUR)
paru -S opencode-bin
```

## Instalación en Windows

### Chocolatey

```bash
choco install opencode
```

### Scoop

```bash
scoop install opencode
```

### NPM

```bash
npm install -g opencode-ai
```

### Mise

```bash
mise use -g github:anomalyco/opencode
```

### Docker

```bash
docker run -it --rm ghcr.io/anomalyco/opencode
```

### Descarga manual

También puedes descargar el binario desde [GitHub Releases](https://github.com/anomalyco/opencode/releases).

## Verificar instalación

```bash
opencode --version
```

## Configuración

### Conectar proveedor LLM

Ejecuta lo siguiente para conectar:

```bash
opencode
/connect
```

Selecciona `opencode`, luego visita [opencode.ai/auth](https://opencode.ai/auth) para iniciar sesión y obtener tu clave API.

### Otros proveedores

También puedes seleccionar otros proveedores LLM. Consulta los [docs oficiales](https://opencode.ai/docs/providers#directory) para detalles.

## Inicializar proyecto

```bash
cd /path/to/project
opencode
/init
```

Esto hará que OpenCode analice tu proyecto y cree un archivo AGENTS.md en la raíz del proyecto.
