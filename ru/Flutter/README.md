# Flutter

Инструмент UI с открытым исходным кодом от Google.

## Обзор

Flutter — это инструмент UI с открытым исходным кодом для создания кроссплатформенных приложений из единой кодовой базы с использованием Dart.

## Функции

- **Кроссплатформенность**: Единая кодовая база для всех платформ
- **Горячая перезагрузка**: Мгновенный предпросмотр изменений
- **Богатые виджеты**: Material Design + Cupertino
- **Высокая производительность**: Нативный рендеринг
- **Открытый исходный код**: Бесплатно

## Быстрый старт

### Установка

См. руководство по установке: ./2.Руководство-по-установке.md

### Создать проект

```bash
flutter create my_app
cd my_app
flutter run
```

### Структура проекта

```
lib/
└── main.dart        # Точка входа
pubspec.yaml         # Файл конфигурации
```

## Ключевые концепции

- **Dart**: Язык программирования Flutter
- **Widget**: Строительный блок UI
- **State**: Управление состоянием
- **Platform Channel**: Каналы платформы
