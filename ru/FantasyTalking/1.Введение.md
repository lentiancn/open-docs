# FantasyTalking Введение

## Что такое FantasyTalking?

FantasyTalking — проект с открытым исходным кодом для синтеза речи и диалоговых систем. Проект направлен на предоставление разработчикам полного решения для ИИ, способного генерировать естественные голосовые взаимодействия.

Как проект с открытым исходным кодом, FantasyTalking поощряет участие сообщества. Разработчики могут настраивать и расширять проект.

## Основные функции

- Высококачественный синтез речи
- Гибкая диалоговая система
- Кроссплатформенные SDK

## Области применения

- Интеллектуальная поддержка клиентов
- Производство аудиокниг
- Голосовые помощники

## Архитектура

3 уровня: движок синтеза речи, управление диалогами, интеграция приложений
