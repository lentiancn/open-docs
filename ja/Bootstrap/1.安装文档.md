# Bootstrap 5 インストールガイド

このガイドでは、Bootstrap 5 のインストールと設定方法について詳しく説明します。

## バージョン情報

| バージョン | 状態 | 説明 |
|------------|------|------|
| Bootstrap 5.3.x | ✅ 現行 | 最新バージョン、すべての新機能を含む |
| Bootstrap 5.2.x | ✅ サポート | 安定バージョン |
| Bootstrap 5.1.x | ⚠️ メンテナンス | セキュリティ更新のみ |
| Bootstrap 4.x | ❌ メンテナンス終了 | 新規プロジェクトには非推奨 |

> **重要**：このガイドは **Bootstrap 5.3.x** を対象としています。Bootstrap 4 のメンテナンスは終了しており、新規プロジェクトには使用しないことを推奨します。

## インストール方法

### 方法1：CDN読み込み（最も簡単）

最も簡単な方法は、CDNを通じてBootstrapのCSSとJavaScriptファイルを読み込むことです。

#### CSSのみ

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
```

#### CSS + JavaScript

```html
<!-- Bootstrap CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- Bootstrap JS (Popperを含む) -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
```

**CDNプロバイダー**：
- [jsDelivr](https://www.jsdelivr.com/) - 推奨
- [cdnjs](https://cdnjs.com/libraries/bootstrap)
- [unpkg](https://unpkg.com/browse/bootstrap/)

#### テンプレート

```html
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 5 ページ</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>こんにちは、Bootstrap 5！</h1>
    
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### 方法2：npmインストール

#### npmを使用

```bash
# 新規プロジェクト作成
npm init -y

# Bootstrapをインストール
npm install bootstrap@5.3.3

# Popper.jsをインストール（必要な場合）
npm install @popperjs/core
```

#### プロジェクトで読み込み

```javascript
// main.js または app.js
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

### 方法3：Viteを使用

#### 1. Viteプロジェクトを作成

```bash
npm create vite@latest my-bootstrap-app -- --template vanilla
cd my-bootstrap-app
npm install
```

#### 2. Bootstrapをインストール

```bash
npm install bootstrap@5.3.3
```

#### 3. Bootstrapを読み込み

```javascript
// main.js
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

```javascript
// vite.config.js
export default {
  plugins: [],
  resolve: {
    alias: {
      '@': '/src'
    }
  }
}
```

### 方法4：Webpackを使用

#### 1. 依存関係をインストール

```bash
npm install bootstrap@5.3.3
npm install --save-dev webpack webpack-cli css-loader style-loader
```

#### 2. Webpackを設定

```javascript
// webpack.config.js
module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js'
  },
  module: {
    rules: [
      {
        test: /\.css$/,
        use: ['style-loader', 'css-loader']
      }
    ]
  }
};
```

#### 3. Bootstrapを読み込み

```javascript
// src/index.js
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.js';
```

### 方法5：Vue CLI / Nuxtを使用

#### Vue CLI

```bash
# Vueプロジェクトを作成
npm install -g @vue/cli

# Bootstrapをインストール
cd my-app
npm install bootstrap@5.3.3

# main.jsで読み込み
import 'bootstrap/dist/css/bootstrap.min.css';

```

#### Nuxt 3

```bash
# Nuxtプロジェクトを作成
npx nuxi@latest init my-nuxt-app
cd my-nuxt-app
npm install

# Bootstrapをインストール
npm install bootstrap@5.3.3

# nuxt.config.tsで設定
export default defineNuxtConfig({
  css: ['bootstrap/dist/css/bootstrap.min.css'],
  build: {
    transpile: ['bootstrap']
  }
})
```

### 方法7：ソースコードをダウンロード

#### GitHubからダウンロード

```bash
# リポジトリをクローン
git clone https://github.com/twbs/bootstrap.git
cd bootstrap

# 依存関係をインストール
npm install

# ビルド
npm run dist
```

#### 直接ダウンロード

1. [Bootstrap 公式リリースページ](https://github.com/twbs/bootstrap/releases)にアクセス
2. Source code (zip) をダウンロード
3. 解凍後、`dist/css/bootstrap.min.css` と `dist/js/bootstrap.bundle.min.js` を読み込み

## 特別な設定

### RTL（右から左）有効化

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.rtl.min.css" rel="stylesheet">
```

### Sassでカスタマイズ

```bash
npm install bootstrap@5.3.3
npm install --save-dev sass
```

```scss
// custom.scss
// Bootstrap関数をインポート
@import "bootstrap/scss/functions";

// カスタム変数
$primary: #ff6b35;
$font-family-sans-serif: "Noto Sans JP", sans-serif;

// Bootstrapの他の部分をインポート
@import "bootstrap/scss/variables";
@import "bootstrap/scss/mixins";
@import "bootstrap/scss/utilities";
@import "bootstrap/scss/root";
@import "bootstrap/scss/reboot";
@import "bootstrap/scss/type";
@import "bootstrap/scss/containers";
@import "bootstrap/scss/grid";
@import "bootstrap/scss/utilities/api";
// 他の必要なコンポーネントを追加...
```

```javascript
// vite.config.js
export default {
  css: {
    preprocessorOptions: {
      scss: {
        additionalData: `@import "bootstrap/scss/functions"; @import "custom-variables";`
      }
    }
  }
}
```

## よくある質問

### Popperとは何ですか？

Popperは位置決エンジンで、ツールチップ（Tooltip）とポップオーバー（Popover）の位置決めに使用されます。`bootstrap.bundle.min.js` にはPopperが含まれているため、別途インストールは不要です。

### jQueryはまだ必要ですか？

**不要です！** Bootstrap 5は完全にネイティブJavaScriptで書き直されており、jQueryへの依存はありません。

### アイコンを使用するには？

Bootstrap 5ではBootstrap Iconsを使用します：

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css">

<i class="bi bi-github"></i>
<i class="bi bi-bootstrap-fill"></i>
```

### テーマカラーを変更するには？

CSS変数でオーバーライドできます：

```css
:root {
  --bs-primary: #ff6b35;
  --bs-secondary: #6c757d;
  --bs-success: #198754;
}
```
