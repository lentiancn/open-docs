# OpenClaw Preguntas Frecuentes

> Preguntas frecuentes sobre OpenClaw

---

## General

### ¿Qué es OpenClaw?

OpenClaw es un marco de asistente de IA que utiliza tecnología de agentes avanzados para automatizar tareas. Combina varias herramientas para gestión de archivos, control de navegador y más.

### ¿Para quién es OpenClaw?

OpenClaw es para desarrolladores, usuarios avanzados y cualquiera que quiera aumentar la productividad mediante automatización impulsada por IA.

---

## Instalación

### ¿Cuáles son los requisitos del sistema?

| Requisito | Mínimo |
|-----------|--------|
| Sistema Operativo | Windows 10+, macOS, Linux |
| Node.js | Versión 18+ |
| RAM | 4 GB |
| Almacenamiento | 500 MB |

### ¿La instalación falla?

**Solución**:
1. Verifica la versión de Node.js: `node --version`
2. Limpia la caché de npm: `npm cache clean --force`
3. Intenta con privilegios de administrador/root
4. Documenta el mensaje de error

### ¿El Gateway no inicia?

**Solución**:
1. Verifica el puerto: `netstat -ano | findstr 3000`
2. Usa un puerto diferente: `openclaw gateway start --port 3001`
3. Revisa los logs: `openclaw gateway logs`

## Uso

### ¿Cómo inicio una conversación?

```bash
openclaw chat
```

### ¿Cómo ejecuto un solo comando?

```bash
openclaw exec "mi comando"
```

### ¿Puedo usar OpenClaw sin interfaz?

Sí, OpenClaw puede operar en modo headless. Inicia el servicio Gateway y usa la API o CLI.

## Gestión de Archivos

### ¿Cómo leo un archivo?

Usa la herramienta de lectura:
```
read /ruta/al/archivo.txt
```

### ¿Cómo creo un nuevo archivo?

Usa la herramienta de escritura:
```
write --content "contenido" --ruta /nuevo/archivo.txt
```

### ¿Cómo edito un archivo?

Usa la herramienta de edición para cambios precisos:
```
edit --path /archivo.txt --oldText "viejo" --newText "nuevo"
```

## Control del Navegador

### ¿Qué navegadores son compatibles?

- Chrome (vía extensión)
- Navegador OpenClaw aislado
- Otros mediante automatización de navegador

### ¿Cómo controlo el navegador?

```bash
browser start --profile openclaw
browser navigate --url https://ejemplo.com
browser screenshot --path /salida.png
```

### ¿El navegador no inicia?

**Solución**:
1. Instala la extensión de Chrome
2. Revisa la configuración del firewall
3. Limpia la caché del navegador

## Subagentes

### ¿Qué son los subagentes?

Los subagentes son instancias independientes que pueden ejecutar tareas específicas en paralelo o secuencialmente.

### ¿Cómo creo un subagente?

```bash
subagents create --task "descripción"
```

### ¿Cómo finalizo un subagente?

```bash
subagents kill --id agent-id
```

## Seguridad

### ¿Es seguro OpenClaw?

OpenClaw implementa múltiples capas de seguridad:
- Permisos de archivos
- Confirmación para acciones externas
- Opciones de sandbox

### ¿Cómo protejo datos sensibles?

1. Usa configuración encriptada
2. No compartas contraseñas en conversaciones
3. Usa variables de entorno para secretos

## Solución de Problemas

### ¿"Comando no encontrado"?

Asegúrate de que:
1. OpenClaw esté correctamente instalado
2. La variable PATH esté configurada correctamente
3. El Gateway esté ejecutándose

### ¿Las operaciones de archivo fallan?

1. Verifica la ruta
2. Confirma los permisos
3. Revisa el nombre del archivo por caracteres especiales

### ¿Tiempos de respuesta lentos?

1. Verifica el estado del Gateway
2. Confirma la conexión de red
3. Analiza los logs
