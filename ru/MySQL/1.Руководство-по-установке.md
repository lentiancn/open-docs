# Руководство по установке MySQL

Установка MySQL на всех платформах.

---

## Docker (Рекомендуется)

```bash
docker run -d \
  --name mysql \
  -e MYSQL_ROOT_PASSWORD=secret \
  -e MYSQL_DATABASE=mydb \
  -p 3306:3306 \
  mysql:8
```

---

## Linux

```bash
sudo apt update
sudo apt install -y mysql-server
sudo systemctl start mysql
```

---

## macOS

```bash
brew install mysql
```

---

## Проверка

```bash
mysql --version
```
