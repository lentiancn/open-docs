# OpenClaw 🦞

OpenClaw 是一款可以部署在自有設備上的工具，能把常用的通訊軟體和 AI 服務連接起來。它在你的設備上運行，成為通訊軟體和 AI 之間的橋樑，讓你能夠在 WhatsApp、Telegram、Discord 等平台上直接使用 AI 助手。

## 適合誰用

如果你想擁有自己的 AI 助手，但又不想把數據交給第三方服務，OpenClaw 是個不錯的選擇。它特別適合習慣使用命令列的開發者和對效率有要求的用戶。

## 主要特點

- **資料自主**：所有內容保存在你的設備上，不經過第三方伺服器
- **多平台接入**：一個程式同時支援多個聊天渠道
- **專為程式設計打造**：支援工具調用、對話管理、記憶功能和多代理調度
- **開源透明**：程式碼公開、社區共建

## 使用條件

- Node.js 22 或更高版本
- 一個 AI 服務商提供的 API 密鑰
- 几分鐘的初始配置時間

建議選用最新一代的高性能模型，以獲得最佳體驗。

## 快速上手

```bash
# 安裝
npm install -g openclaw@latest

# 初始化設置
openclaw onboard --install-daemon

# 啟動服務
openclaw gateway --port 18789
```

服務啟動後，開啟瀏覽器訪問本地位址即可開始使用。

## 工作方式

OpenClaw 的核心是一個 Gateway 進程，它負責處理所有渠道的連接、AI 調用的轉發、以及對話狀態的維護。你只需要維護這一個進程，就能同時在多個平台上使用 AI 助手。

## 核心能力

- **統一入口**：一套系統管理所有聊天渠道
- **插件擴展**：通過擴展包還能支援更多平台
- **多代理支援**：可以為不同用途配置獨立的 AI 實例
- **檔案互動**：支援收發圖片、音訊、文檔
- **網頁管理介面**：在瀏覽器中管理配置、查看對話歷史
- **移動設備聯動**：手機安裝配套應用後可遠端控制、拍照、截圖
