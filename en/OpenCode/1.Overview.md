# OpenCode Overview

Open source AI coding agent.

## What is OpenCode?

OpenCode is an open source AI coding agent available as a terminal interface, desktop app, or IDE extension.

### Key Features

- **LSP Enabled** - Automatically loads the right LSPs
- **Multi-session** - Start multiple agents in parallel on the same project
- **Share Links** - Share any session link for reference or debugging
- **GitHub Copilot** - Log in with GitHub to use your Copilot account
- **ChatGPT Plus/Pro** - Log in with OpenAI to use ChatGPT account
- **Any Model** - 75+ LLM providers including local models
- **Any Editor** - Terminal, desktop app, and IDE extension
- **Privacy First** - No code or context data is stored
