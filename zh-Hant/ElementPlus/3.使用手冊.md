# Element Plus 使用手冊

本手冊將幫助你快速上手 Element Plus，掌握日常使用中常用的組件和功能。

## 快速開始

在使用 Element Plus 之前，確保已完成安裝配置。

### 基礎使用

在 Vue 組件中直接使用組件：

```vue
<template>
  <el-button>默認按鈕</el-button>
  <el-button type="primary">主要按鈕</el-button>
  <el-button type="success">成功按鈕</el-button>
  <el-button type="warning">警告按鈕</el-button>
  <el-button type="danger">危險按鈕</el-button>
  <el-button type="info">信息按鈕</el-button>
</template>
```

## 常用組件

### 按鈕（Button）

基礎用法：

```vue
<el-button>默認</el-button>
<el-button type="primary">主要</el-button>
<el-button type="success">成功</el-button>
<el-button type="warning">警告</el-button>
<el-button type="danger">危險</el-button>
<el-button type="info">信息</el-button>
```

不同尺寸：

```vue
<el-button size="large">大型</el-button>
<el-button>默認</el-button>
<el-button size="small">小型</el-button>
```

禁用狀態：

```vue
<el-button disabled>禁用</el-button>
```

圖標按鈕：

```vue

<el-button icon="Edit">編輯</el-button>
<el-button icon="Delete">刪除</el-button>
```

### 輸入框（Input）

基礎用法：

```vue
<el-input v-model="input" placeholder="請輸入內容" />
```

帶圖標的輸入框：

```vue
<el-input v-model="input" placeholder="請輸入內容" prefix-icon="Search" />
<el-input v-model="input" placeholder="請輸入內容" suffix-icon="Calendar" />
```

帶清空按鈕：

```vue
<el-input v-model="input" placeholder="請輸入內容" clearable />
```

密碼輸入框：

```vue
<el-input v-model="input" type="password" placeholder="請輸入密碼" show-password />
```

多行文本：

```vue
<el-input
  v-model="textarea"
  type="textarea"
  :rows="3"
  placeholder="請輸入內容"
/>
```

    :key="item.value"
    :label="item.label"
    :value="item.value"
  />

<script setup>
import { ref } from 'vue'

const options = [

  { value: '選項2', label: '雙皮奶' },

多選：

```vue

  <el-option
    v-for="item in options"
    :key="item.value"
    :label="item.label"
    :value="item.value"

```

### 表單（Form）

基礎表單：

```vue
<el-form :model="form" label-width="80px">
  <el-form-item label="活動名稱">
    <el-input v-model="form.name" />
  </el-form-item>
  <el-form-item label="活動區域">
    <el-select v-model="form.region" placeholder="請選擇活動區域">
      <el-option label="區域一" value="shanghai" />
      <el-option label="區域二" value="beijing" />
    </el-select>
  </el-form-item>
  <el-form-item label="活動時間">
    <el-date-picker
      v-model="form.date"
      type="date"
      placeholder="選擇日期"
    />
  </el-form-item>
  <el-form-item label="即時配送">
    <el-switch v-model="form.delivery" />
  </el-form-item>
  <el-form-item label="活動性質">
    <el-checkbox-group v-model="form.type">
      <el-checkbox label="美食/餐廳線上活動" name="type" />
      <el-checkbox label="線下推廣活動" name="type" />
      <el-checkbox label="線下主題活動" name="type" />
    </el-checkbox-group>
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="onSubmit">立即創建</el-button>
    <el-button>取消</el-button>
  </el-form-item>
</el-form>

<script setup>
import { reactive } from 'vue'

const form = reactive({
  name: '',
  region: '',
  date: '',
  delivery: false,
  type: [],
})

const onSubmit = () => {
  console.log('submit!', form)
}
</script>
```

表單驗證：

```vue
<el-form :model="ruleForm" :rules="rules" ref="ruleFormRef" label-width="100px">
  <el-form-item label="活動名稱" prop="name">
    <el-input v-model="ruleForm.name" />
  </el-form-item>
  <el-form-item label="活動區域" prop="region">
    <el-select v-model="ruleForm.region" placeholder="請選擇活動區域">
      <el-option label="區域一" value="shanghai" />
      <el-option label="區域二" value="beijing" />
    </el-select>
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="submitForm(ruleFormRef)">提交</el-button>
    <el-button @click="resetForm(ruleFormRef)">重置</el-button>
  </el-form-item>
</el-form>

<script setup>
import { reactive, ref } from 'vue'

const ruleFormRef = ref()
const ruleForm = reactive({
  name: '',
  region: '',
})

const rules = reactive({
  name: [
    { required: true, message: '請輸入活動名稱', trigger: 'blur' },
    { min: 3, max: 5, message: '長度在 3 到 5 個字符', trigger: 'blur' },
  ],
  region: [
    { required: true, message: '請選擇活動區域', trigger: 'change' },
  ],
})

const submitForm = async (formEl) => {
  if (!formEl) return
  await formEl.validate((valid, fields) => {
    if (valid) {
      console.log('submit!')
    } else {
      console.log('error submit!', fields)
    }
  })
}

const resetForm = (formEl) => {
  if (!formEl) return
  formEl.resetFields()
}
</script>
```

## 常見問題

### 組件樣式不生效

確保已正確引入 Element Plus 的 CSS：

```javascript
import 'element-plus/dist/index.css'
```

### 圖標不顯示

需要額外安裝圖標庫：

```bash
npm install @element-plus/icons-vue
```

然後全局註冊：

```javascript
import * as icons from '@element-plus/icons-vue'
app.use(ElementPlus)
for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```

### 表格性能問題

對於大數據量的表格，使用虛擬滾動：

```vue
<el-table-v2
  :columns="columns"
  :data="data"
  :height="400"
/>
```
