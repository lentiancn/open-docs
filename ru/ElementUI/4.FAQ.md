# FAQ Element UI

## Установка

### Ошибка npm install?

**Решение**: Очистить кэш или использовать зеркало.

### Стили не отображаются?

**Решение**: Импортировать CSS.

## Компоненты

### Компонент не отображается?

**Решение**: Вызвать Vue.use(ElementUI).
