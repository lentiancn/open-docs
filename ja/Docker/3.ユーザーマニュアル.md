# Docker ユーザーマニュアル

コアコンセプト、基本的な使い方、開発ガイド。

## イメージ操作

### イメージをプル

```
docker pull nginx
docker pull ubuntu:22.04
```

### イメージをビルド

```
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
EXPOSE 3000
CMD ["npm", "start"]
```

```
docker build -t my-app:tag .
```

### イメージを管理

```
docker images
docker rmi nginx
```

## コンテナ操作

### コンテナを作成して起動

```
# インタラクティブコンテナ
docker run -it ubuntu /bin/bash

# デタッチドコンテナ
docker run -d nginx

# ポートマッピング
docker run -d -p 8080:80 nginx

# 環境変数
docker run -e MYSQL_ROOT_PASSWORD=123456 mysql
```

### コンテナを管理

```
docker ps              # 実行中のコンテナ
docker ps -a          # 全てのコンテナ
docker start container_id
docker stop container_id
docker rm container_id
```

### コンテナに入る

```
docker exec -it container_id /bin/bash
```

### ログを表示

```
docker logs container_id
docker logs -f container_id
```

## Dockerfileの記述

### 基本的な命令

```
FROM node:18-alpine
WORKDIR /app
COPY . .
RUN npm install
EXPOSE 3000
CMD ["npm", "start"]
```

### マルチステージビルド

```
# ビルドステージ
FROM node:18 AS builder
WORKDIR /app
COPY . .
RUN npm run build

# プロダクションステージ
FROM nginx:alpine
COPY --from=builder /app/dist /usr/share/nginx/html
```

## Docker Compose

### docker-compose.yml

```
version: '3.8'

services:
  web:
    build: .
    ports:
      - "3000:3000"
  db:
    image: postgres:15
    environment:
      POSTGRES_PASSWORD: secret
```

### 一般的なコマンド

```
docker-compose up -d
docker-compose down
docker-compose logs -f
```

## ネットワーク管理

### ポートマッピング

```
docker run -p 8080:80 nginx
```

### カスタムネットワーク

```
docker network create my-network
docker run --network my-network nginx
```

## データ管理

### ボリューム

```
docker volume create my-data
docker run -v my-data:/data nginx
```

### バインドマウント

```
docker run -v /host/path:/container/path nginx
```

## リソース制限

```
# メモリ制限
docker run -m 512m nginx

# CPU制限
docker run --cpus=0.5 nginx
```

## 監視とデバッグ

```
docker stats
docker inspect container_id
docker logs container_id
```

## クリーンアップ

```
docker system prune -a
docker volume prune
```
