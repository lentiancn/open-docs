# Element UI Manuel Utilisateur

## Composants de base

### Button

```html
<el-button>Bouton</el-button>
<el-button type="primary">Primaire</el-button>
<el-button type="success">Succès</el-button>
<el-button type="warning">Avertissement</el-button>
<el-button type="danger">Danger</el-button>
```

### Layout

```html
<el-row>
  <el-col :span="12"><div></div></el-col>
  <el-col :span="12"><div></div></el-col>
</el-row>
```

## Formulaires

### Input

```html
<el-input v-model="input" placeholder="Entrée"></el-input>
```

### Select

```html
<el-select v-model="value" placeholder="Sélectionner">
  <el-option label="Option 1" value="1"></el-option>
</el-select>
```

## Données

### Table

```el-table :data="tableData">
  <el-table-column prop="date" label="Date"></el-table-column>
  <el-table-column prop="name" label="Nom"></el-table-column>
</el-table>
```

### Pagination

```html
<el-pagination :total="1000"></el-pagination>
```

## Notifications

### Message

```javascript
this.$message({ message: 'Succès', type: 'success' })
```

### Alert

```html
<el-alert title="Succès" type="success"></el-alert>
```

## Navigation

### Menu

```html
<el-menu>
  <el-menu-item index="1">Menu 1</el-menu-item>
</el-menu>
```

### Tabs

```html
<el-tabs v-model="activeName">
  <el-tab-pane label="Onglet 1" name="first">Contenu</el-tab-pane>
</el-tabs>
```

## Autres

### Dialog

```html
<el-button @click="visible = true">Ouvrir</el-button>
<el-dialog :visible.sync="visible" title="Titre">
  <span>Contenu</span>
</el-dialog>
```
