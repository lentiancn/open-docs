# 使用手冊

本手冊詳細介紹 Angular 的核心功能和使用方法，包括元件、服務、路由、指令、管道、表單等。

## 元件開發

### 建立元件

```bash
ng generate component user-list
```

或者手動建立：

```typescript
import { Component } from '@angular/core';

@Component({
  selector: 'app-user-list',
  templateUrl: './user-list.component.html',
  styleUrls: ['./user-list.component.scss']
})
export class UserListComponent {
  users = ['張三', '李四', '王五'];
}
```

**插值：**


**屬性綁定：**

## 服務與依賴注入

### 建立服務

```bash
ng generate service user
```

服務程式碼：

```typescript
import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class UserService {
  constructor(private http: HttpClient) {}

  getUsers(): Observable<User[]> {
    return this.http.get<User[]>('/api/users');
  }
}
```

### 使用服務

```typescript
import { Component, OnInit } from '@angular/core';
import { UserService } from './user.service';

@Component({
  selector: 'app-user-list',
  templateUrl: './user-list.component.html'
})
export class UserListComponent implements OnInit {
  users: User[] = [];

  constructor(private userService: UserService) {}

  ngOnInit() {
    this.userService.getUsers().subscribe(data => {
      this.users = data;
    });
  }
}
```

## 路由

### 配置路由

```typescript
import { Routes } from '@angular/router';

const routes: Routes = [
  { path: '', redirectTo: '/home', pathMatch: 'full' },
  { path: 'home', component: HomeComponent },
  { path: 'users', component: UserListComponent },
  { path: 'user/:id', component: UserDetailComponent },
  { path: '**', component: NotFoundComponent }
];
```

### 路由連結

```html
<nav>
  <a routerLink="/home">首頁</a>
  <a routerLink="/users">使用者列表</a>
  <a [routerLink]="['/user', userId]">使用者詳情</a>
</nav>
```

### 自訂管道

```bash
ng generate pipe capitalize
```

```typescript
import { Pipe, PipeTransform } from '@angular/core';

@Pipe({
  name: 'capitalize'
})
export class CapitalizePipe implements PipeTransform {
  transform(value: string): string {
    if (!value) return value;
    return value.charAt(0).toUpperCase() + value.slice(1);
  }
}
```

## 表單處理

### 響應式表單

```typescript
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

export class LoginComponent implements OnInit {
  loginForm: FormGroup;

  constructor(private fb: FormBuilder) {}

  ngOnInit() {
    this.loginForm = this.fb.group({
      username: ['', Validators.required],
      password: ['', [Validators.required, Validators.minLength(6)]],
      remember: [false]
    });
  }

  onSubmit() {
    if (this.loginForm.valid) {
      console.log(this.loginForm.value);
    }
  }
}
```

模板：

```html
<form [formGroup]="loginForm" (ngSubmit)="onSubmit()">
  <input formControlName="username" placeholder="使用者名稱">
  <div *ngIf="loginForm.get('username').touched && loginForm.get('username').errors?.required">
    使用者名稱必填
  </div>
  
  <input type="password" formControlName="password" placeholder="密碼">
  
  <label>
    <input type="checkbox" formControlName="remember"> 記住我
  </label>
  
  <button type="submit" [disabled]="loginForm.invalid">登入</button>
</form>
```

### 模板驅動表單

```html
<form #form="ngForm" (ngSubmit)="onSubmit(form)">
  <input name="username" [(ngModel)]="username" required #name="ngModel" placeholder="使用者名稱">
  <div *ngIf="name.touched && name.errors?.required">使用者名稱必填</div>
  
  <button type="submit" [disabled]="form.invalid">提交</button>
</form>
```

## HTTP 客戶端

### 配置

在 app.module.ts 中匯入：

```typescript
import { HttpClientModule } from '@angular/common/http';

@NgModule({
  imports: [HttpClientModule]
})
export class AppModule {}
```

### 發送請求

```typescript
import { HttpClient } from '@angular/common/http';

constructor(private http: HttpClient) {}

getData() {
  this.http.get('/api/data').subscribe(data => {
    console.log(data);
  });
}

postData() {
  this.http.post('/api/data', { name: 'test' }).subscribe();
}
```

### 攔截器

```typescript
import { HttpInterceptorFn } from '@angular/common/http';

export const authInterceptor: HttpInterceptorFn = (req, next) => {
  const token = localStorage.getItem('token');
  const cloned = req.clone({
    setHeaders: { Authorization: `Bearer ${token}` }
  });
  return next(cloned);
};
```

## 最佳實踐

1. **使用 TypeScript 嚴格模式**：啟用 strict 編譯選項
2. **元件要小而專注**：每個元件只做一件事
3. **使用 OnPush 策略**：提高變更檢測效能
4. **合理使用管道**：避免在管道中進行複雜計算
5. **使用懶載入**：減少首屏載入時間
6. **撰寫測試**：確保程式碼品質
