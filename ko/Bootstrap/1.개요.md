# Bootstrap 개요

> 인기 있는 프론트엔드 CSS 프레임워크

---

## 프로젝트 소개

Bootstrap은 Twitter에서 개발하고 유지 관리하는 오픈 소스 CSS 프레임워크로, 2011년에 처음 출시되었습니다. Bootstrap은 풍부한 사전 구축된 컴포넌트와 반응형 그리드 시스템을 제공하여 현대적인 반응형 웹사이트와 웹 애플리케이션을 빠르게 구축할 수 있습니다.

---

## 주요 기능

- **반응형 디자인**: 내장된 반응형 레이아웃
- **사전 구축된 컴포넌트**: 버튼, 네비게이션, 폼, 카드 등
- **강력한 그리드 시스템**: 12열 반응형 그리드
- **사용자 정의 옵션**: 손쉬운 사용자화를 위한 Sass 변수
- **JavaScript 플러그인**: 드롭다운, 모달 등
- **아이콘 라이브러리**: 내장된 Bootstrap Icons
