# Element UI 常见问题

## 安装问题

### 安装失败

确保 Node.js 版本 >= 12.0.0，清除缓存后重试。

### 样式不生效

确保引入 CSS：

```javascript
import 'element-ui/lib/theme-chalk/index.css'
```

## 使用问题

### 组件不显示

确保已全局注册组件或使用完整引入。

### 表格问题

确保 data 属性正确绑定。

### 表单验证

确保 form 有 model 属性，form-item 有 prop 属性。

## Vue 版本

Element UI 仅支持 Vue 2。如需 Vue 3，请使用 Element Plus。
