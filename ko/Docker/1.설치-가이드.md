# Docker 설치 가이드

## 시스템 요구사항

- Windows 10/11 64비트
- macOS 10.15+
- Ubuntu 20.04+

## 설치

### Windows/macOS
https://www.docker.com/products/docker-desktop/에서 다운로드

### Linux (Ubuntu)

```bash
sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io docker-compose-plugin
sudo systemctl start docker
```

## 확인

```bash
docker --version
docker run hello-world
```
