# Nginx FAQ

## Installation

### Version prüfen
```bash
nginx -v
```

### Konfiguration testen
```bash
nginx -t
```

## Konfiguration

### Neu laden
```bash
nginx -s reload
```

## Leistung

### Optimierung
```nginx
worker_processes auto;
worker_connections 4096;
gzip on;
```

## Sicherheit

### HTTPS aktivieren
```nginx
server {
    listen 443 ssl;
    ssl_certificate cert.crt;
    ssl_certificate_key cert.key;
}
```

### Ratenbegrenzung
```nginx
limit_req_zone $binary_remote_addr zone=one:10m rate=10r/s;
```

## Fehler

- **400**: Anfragegröße prüfen
- **403**: Berechtigungen prüfen
- **404**: Pfad prüfen
- **502**: Backend-Dienst läuft nicht
- **504**: Timeout, proxy_read_timeout erhöhen

## Proxy

### Echte IP erhalten
```nginx
proxy_set_header X-Real-IP $remote_addr;
```
