# Kotlin インストールガイド

KotlinはJetBrainsが開発した現代的なプログラミング言語です。このガイドでは、さまざまな環境でのインストール方法を説明します。

## バージョン情報

| バージョン | ステータス | 説明 |
|------------|------------|------|
| Kotlin 2.x | ✅ 最新 | 最新バージョン |
| Kotlin 1.9.x | ✅ サポート | 安定バージョン |

## システム要件

### ハードウェア要件

| 要件 | 設定 |
|------|------|
| CPU | 1コア以上 |
| メモリ | 2GB以上 |
| ディスク | 500MB以上 |

### ソフトウェア要件

| ソフトウェア | バージョン |
|--------------|------------|
| Java | JDK 8以上 |

## インストール方法

### 方法1：IntelliJ IDEA（推奨）

1. IntelliJ IDEAをダウンロード：https://www.jetbrains.com/idea/download/
2. UltimateまたはCommunityエディションをインストール
3. Kotlinは最初からサポートされています

### 方法2：コマンドライン

#### Windows

```powershell
# Chocolateyを使用
choco install kotlin

# または手動ダウンロード
# https://github.com/JetBrains/kotlin/releases からダウンロード
```

#### macOS

```bash
# Homebrewを使用
brew install kotlin

# または手動インストール
curl -s https://get.sdkman.io | bash
sdk install kotlin
```

#### Linux

```bash
# Ubuntu/Debian
sudo apt install kotlin

# またはSDKMANを使用
curl -s https://get.sdkman.io | bash
sdk install kotlin
```

### 方法3：SDKMAN

```bash
# SDKMANをインストール
curl -s "https://get.sdkman.io" | bash

# Kotlinをインストール
sdk install kotlin

# 確認
kotlin -version
```

### 方法4：Maven

```xml
<dependencies>
    <dependency>
        <groupId>org.jetbrains.kotlin</groupId>
        <artifactId>kotlin-stdlib</artifactId>
        <version>2.0.0</version>
    </dependency>
</dependencies>
```

### 方法5：Gradle

```groovy
plugins {
    id 'org.jetbrains.kotlin.jvm' version '2.0.0'
}

repositories {
    mavenCentral()
}

dependencies {
    implementation "org.jetbrains.kotlin:kotlin-stdlib"
}
```

### 方法6：Docker

```dockerfile
# Kotlinコンパイラーイメージを使用
FROM gradle:8.5-jdk-21 AS builder
WORKDIR /app
COPY . .
RUN gradle build

# 実行イメージ
FROM openjdk:21
COPY --from=builder /app/build/libs/*.jar /app.jar
ENTRYPOINT ["java", "-jar", "/app.jar"]
```

## 環境設定

### Java設定

```bash
# Javaを確認
java -version

# JAVA_HOMEを設定
export JAVA_HOME=/path/to/java
```

### コンパイルと実行

```bash
# コンパイル
kotlinc hello.kt -include-runtime -d hello.jar

# 実行
java -jar hello.jar

# Kotlinスクリプトを使用
kotlin hello.kts
```

## IDE設定

### IntelliJ IDEA

1. Kotlinプラグインをインストール（通常はプレインストール）
2. Kotlinプロジェクトを作成
3. Kotlin SDKを設定

### VS Code

1. Kotlin Language Server拡張機能をインストール
2. Kotlinコンパイラーをインストール

### Eclipse

1. Kotlinプラグインをインストール（Kotlin IDE for Eclipse）
2. Kotlinプロジェクトを作成

## よくある問題

### コンパイルエラー？

KotlinとJavaのバージョンの互換性を確認してください。

### メモリ不足？

JVMヒープメモリを増量：

```bash
kotlinc -J-Xmx2g
```
