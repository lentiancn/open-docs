# Element Plus Installation Guide

This guide provides detailed instructions on how to install and use Element Plus in various projects.

## Environment Requirements

Element Plus has the following requirements:

- Vue 3.0.0+
- Node.js 14.0.0+
- Modern browsers (Chrome, Firefox, Safari, Edge)

## Installation Methods

### Using Package Manager

It is recommended to install using npm, yarn, or pnpm:

```bash
# npm
npm install element-plus

# yarn
yarn add element-plus

# pnpm
pnpm add element-plus
```

## Full Import

If you don't mind the bundle size, full import is more convenient.

### Vue 3 Project

In main.ts:

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')
```

### Volar Type Support

If using Volar plugin, configure in tsconfig.json:

```json
{
  "compilerOptions": {
    "types": ["element-plus/global"]
  }
}
```

## On-Demand Import

On-demand import can significantly reduce project size. You need additional plugins to import the components you use.

### Auto Import (Recommended)

First install the plugins:

```bash
npm install -D unplugin-vue-components unplugin-auto-import
yarn add -D unplugin-vue-components unplugin-auto-import
pnpm add -D unplugin-vue-components unplugin-auto-import
```

#### Vite Configuration

In vite.config.ts:

```typescript
import { defineConfig } from 'vite'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    AutoImport({ resolvers: [ElementPlusResolver()] }),
    Components({ resolvers: [ElementPlusResolver()] }),
  ],
})
```

#### Webpack Configuration

In webpack.config.js:

```javascript
const AutoImport = require('unplugin-auto-import/webpack')
const Components = require('unplugin-vue-components/webpack')
const { ElementPlusResolver } = require('unplugin-vue-components/resolvers')

module.exports = {
  plugins: [
    AutoImport({ resolvers: [ElementPlusResolver()] }),
    Components({ resolvers: [ElementPlusResolver()] }),
  ],
}
```

### Using with Nuxt

For Nuxt users:

```bash
npm install -D @element-plus/nuxt
yarn add -D @element-plus/nuxt
pnpm add -D @element-plus/nuxt
```

Configure in nuxt.config.ts:

```typescript
export default defineNuxtConfig({
  modules: ['@element-plus/nuxt'],
})
```

### Manual Import

Element Plus provides ES Module-based Tree Shaking functionality.

You need to install unplugin-element-plus to import styles:

```bash
npm install -D unplugin-element-plus
```

Import manually in components:

```vue
<template>
  <el-button>I am ElButton</el-button>
</template>

<script>
import { ElButton } from 'element-plus'

export default {
  components: { ElButton },
}
</script>
```

Vite configuration:

```typescript
import { defineConfig } from 'vite'
import ElementPlus from 'unplugin-element-plus/vite'

export default defineConfig({
  plugins: [ElementPlus()],
})
```

## Global Configuration

When importing Element Plus, you can pass a global configuration object.

### Full Import Configuration

```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import App from './App.vue'

const app = createApp(App)
app.use(ElementPlus, { size: 'small', zIndex: 3000 })
```

Configuration options:
- size: Set default size for form components (large, default, small)
- zIndex: Set z-index for popup components, default is 2000

### On-Demand Import Configuration

Use el-config-provider component:

```vue
<template>
  <el-config-provider :size="size" :z-index="zIndex">
    <app />
  </el-config-provider>
</template>

<script>
import { defineComponent } from 'vue'
import { ElConfigProvider } from 'element-plus'

export default defineComponent({
  components: { ElConfigProvider },
  setup() {
    return { zIndex: 3000, size: 'small' }
  },
})
</script>
```

## Getting Started

After installation, you can use any Element Plus component:

```vue
<template>
  <el-button type="primary">Primary Button</el-button>
  <el-input v-model="input" placeholder="Please input" />
  <el-table :data="tableData" style="width: 100%">
    <el-table-column prop="date" label="Date" width="180" />
    <el-table-column prop="name" label="Name" width="180" />
    <el-table-column prop="address" label="Address" />
  </el-table>
</template>

<script setup>
import { ref } from 'vue'

const input = ref('')
const tableData = ref([
  { date: '2016-05-02', name: 'Tom', address: 'No. 189, Grove St' },
])
</script>
```
