# MultiTalk インストールガイド

MultiTalkのインストールには、Python環境の構成とモデルウェイトのダウンロードが必要です。以下に詳細なインストール手順を説明します。

## 環境要件

### ハードウェア要件

MultiTalkは大型の動画生成モデルであり、ハードウェア要件が高くなっています。NVIDIA GPUでの実行を推奨します。VRAMの要件は出力解像度によって異なります。480P動画の生成には少なくとも16GBのVRAMが必要で、720P動画の生成には複数のGPU設定が必要です。プロジェクトは特定の最適化により、単一のRTX 4090で480P動画の生成を可能にする低VRAM環境での実行もサポートしています。

### ソフトウェア要件

Linuxオペレーティングシステムがサポートされています。Pythonバージョンは3.10以上が必要です。CUDA 12.1以上のサポートも必要です。プロジェクトはPyTorch 2.4.1、xformersなどの深層学習フレームワークに依存しています。

## 仮想環境の作成

まず、プロジェクトの依存関係を分離するために、新しいPython仮想環境を作成します。ターミナルを開いて、以下のコマンドを実行します：

```bash
conda create -n multitalk python=3.10
conda activate multitalk
```

## PyTorchとxformersのインストール

 활성화된環境 でPyTorch和相关 패키지를インストールします。PyTorch 2.4.1をインストールするには、以下のコマンドを実行します：

```bash
pip install torch==2.4.1 torchvision==0.19.1 torchaudio==2.4.1 --index-url https://download.pytorch.org/whl/cu121
```

次に、xformersをインストールします：

```bash
pip install -U xformers==0.0.28 --index-url https://download.pytorch.org/whl/cu121
```

## プロジェクトの依存関係のインストール

その他の必要な依存パッケージをインストールします：

```bash
pip install misaki[en]
pip install ninja
pip install psutil
pip install packaging
pip install flash_attn==2.7.4.post1
```

次に、プロジェクトリポジトリをクローンし、プロジェクトの依存関係をインストールします：

```bash
git clone https://github.com/MeiGen-AI/MultiTalk.git
cd MultiTalk
pip install -r requirements.txt
```

## オーディオ処理ライブラリのインストール

プロジェクトにはオーディオ処理関連のライブラリが必要です。以下のコマンドを実行してインストールします：

```bash
conda install -c conda-forge librosa
conda install -c conda-forge ffmpeg
```

CentOSまたはRHELシステムの場合は、以下を使用します：

```bash
sudo yum install ffmpeg ffmpeg-devel
```

## モデルウェイトのダウンロード

MultiTalkは正常に動作するために複数のモデルファイルをダウンロードする必要があります。主要なモデルは次のとおりです：

Wan2.1-I2V-14B-480Pはベースの拡散モデルで、Hugging Faceからダウンロードできます。chinese-wav2vec2-baseは入力オーディオを処理するためのオーディオエンコーダーです。Kokoro-82Mはテキスト読み上げ機能のためのTTSウェイトです。MeiGen-MultiTalkはプロジェクトが提供するオーディオ条件ウェイトです。

huggingface-cliを使用してモデルをダウンロードします：

```bash
huggingface-cli download Wan-AI/Wan2.1-I2V-14B-480P --local-dir ./weights/Wan2.1-I2V-14B-480P
huggingface-cli download TencentGameMate/chinese-wav2vec2-base --local-dir ./weights/chinese-wav2vec2-base
huggingface-cli download hexgrad/Kokoro-82M --local-dir ./weights/Kokoro-82M
huggingface-cli download MeiGen-AI/MeiGen-MultiTalk --local-dir ./weights/MeiGen-MultiTalk
```

## モデルリンクの設定

モデルのダウンロードが完了したら、リンクを設定する必要があります。2つのオプションがあります。

オプション1はシンボリックリンクを使用することです。weightsディレクトリに入り、以下のコマンドを実行します：

```bash
cd weights
mv Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json_old
ln -s /path/to/weights/MeiGen-MultiTalk/diffusion_pytorch_model.safetensors.index.json Wan2.1-I2V-14B-480P/
ln -s /path/to/weights/MeiGen-MultiTalk/multitalk.safetensors Wan2.1-I2V-14B-480P/
```

オプション2はファイルを直接コピーすることです：

```bash
cd weights
mv Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json_old
cp MeiGen-MultiTalk/diffusion_pytorch_model.safetensors_index.json Wan2.1-I2V-14B-480P/
cp MeiGen-MultiTalk/multitalk.safetensors Wan2.1-I2V-14B-480P/
```

## インストールの確認

インストールが完了したら、サンプルスクリプトを実行してインストールが成功したかどうかを確認できます。サンプルファイルはexamplesディレクトリに配置されています。
