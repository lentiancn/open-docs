# Apifox

Apifox は、国内でトップクラスの API  協業プラットフォームです。API 設計、Mock、テスト、ドキュメント作成などを一貫して行うことができます。API 設計からドキュメント管理、デバッグ、Mock、自動化テストといった全ての機能を網羅しており、国内で最も人気の API 管理ツールの一つです。

## クイックスタート

### ダウンロードとインストール

https://apifox.com/ にアクセスして、お使いの OS 合ったインストールパッケージをダウンロードしてください。

### プロジェクト作成

1. Apifox を起動してログイン
2. 「新規プロジェクト」をクリック
3. プロジェクト名を入力

### 接口作成

1. プロジェクト内で「新規接口作成」をクリック
2. 接口の情報を入力
3. 保存

### 接口デバッグ

1. 接口を開く
2. 「送信」をクリック
3. レスポンスを確認

## 主な機能

- **API 設計**：ビジュアル的に API 接口を定義
- **API ドキュメント**：自動生成、複数形式でのエクスポートに対応
- **API デバッグ**：強力なオンライン デバッグ機能
- **API Mock**：スマート Mock、自动生成のモックデータ
- **自動化テスト**：コードを書かずに API テストが完了
- **チームコラボレーション**：リアルタイム同期、権限管理
