# Docker Benutzerhandbuch

Kernkonzepte, grundlegende Nutzung und Entwicklungsleitfaden.

## Image-Operationen

### Images pullen

```
docker pull nginx
docker pull ubuntu:22.04
```

### Images bauen

```
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
EXPOSE 3000
CMD ["npm", "start"]
```

```
docker build -t my-app:tag .
```

### Images verwalten

```
docker images
docker rmi nginx
```

## Container-Operationen

### Container erstellen und starten

```
# Interaktiver Container
docker run -it ubuntu /bin/bash

# Detachierter Container
docker run -d nginx

# Port-Mapping
docker run -d -p 8080:80 nginx

# Umgebungsvariablen
docker run -e MYSQL_ROOT_PASSWORD=123456 mysql
```

### Container verwalten

```
docker ps              # Laufende Container
docker ps -a         # Alle Container
docker start container_id
docker stop container_id
docker rm container_id
```

### In Container einsteigen

```
docker exec -it container_id /bin/bash
```

### Logs anzeigen

```
docker logs container_id
docker logs -f container_id
```

## Dockerfiles schreiben

### Grundlegende Anweisungen

```
FROM node:18-alpine
WORKDIR /app
COPY . .
RUN npm install
EXPOSE 3000
CMD ["npm", "start"]
```

### Mehrstufige Builds

```
# Build-Stufe
FROM node:18 AS builder
WORKDIR /app
COPY . .
RUN npm run build

# Produktions-Stufe
FROM nginx:alpine
COPY --from=builder /app/dist /usr/share/nginx/html
```

## Docker Compose

### docker-compose.yml

```
version: '3.8'

services:
  web:
    build: .
    ports:
      - "3000:3000"
  db:
    image: postgres:15
    environment:
      POSTGRES_PASSWORD: secret
```

### Häufige Befehle

```
docker-compose up -d
docker-compose down
docker-compose logs -f
```

## Netzwerkverwaltung

### Port-Mapping

```
docker run -p 8080:80 nginx
```

### Eigene Netzwerke

```
docker network create my-network
docker run --network my-network nginx
```

## Datenverwaltung

### Volumes

```
docker volume create my-data
docker run -v my-data:/data nginx
```

### Bind-Mounts

```
docker run -v /host/path:/container/path nginx
```

## Ressourcenlimits

```
# Speicherlimit
docker run -m 512m nginx

# CPU-Limit
docker run --cpus=0.5 nginx
```

## Überwachung und Fehlersuche

```
docker stats
docker inspect container_id
docker logs container_id
```

## Aufräumen

```
docker system prune -a
docker volume prune
```
