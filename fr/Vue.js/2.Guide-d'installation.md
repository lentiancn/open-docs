# Vue.js Guide d'Installation

## Essayez Vue en Ligne

- **Playground**: Essayez directement dans le [Playground Vue](https://play.vuejs.org)
- **JSFiddle**: Utilisez ce [template JSFiddle](https://jsfiddle.net/yyx990803/2ke1ab0z/) pour une configuration HTML simple
- **StackBlitz**: Essayez une configuration de compilation complète directement dans votre navigateur sur [StackBlitz](https://vite.new/vue)
- **Tutoriel Interactif**: Regardez ce tutoriel interactif [Scrimba](http://scrimba.com/links/vue-quickstart) qui vous montre comment exécuter, modifier et déployer votre première application Vue

## Création d'une Application Vue (Recommandé)

### Prérequis
- Familiarité avec la ligne de commande
- Installer Node.js version ^20.19.0 || >=22.12.0

### Utilisation de l'Outil de Scaffolding Officiel
Exécutez l'une des commandes suivantes dans votre terminal :

```bash
# npm
npm create vue@latest

# pnpm  
pnpm create vue@latest

# yarn (v1+)
yarn create vue

# yarn (Modern v2+)
yarn create vue@latest

# yarn (^v4.11)
yarn dlx create-vue@latest

# bun
bun create vue@latest
```

### Options de Configuration du Projet
Vous serez invité à plusieurs fonctionnalités optionnelles :
- Project name: Nom de votre projet
- Add TypeScript?: Ajouter le support TypeScript
- Add JSX Support?: Ajouter le support JSX
- Add Vue Router: Pour le développement d'applications à page unique
- Add Pinia: Pour la gestion d'état
- Add Vitest: Pour les tests unitaires
- Add End-to-End Testing: Cypress/Nightwatch/Playwright
- Add ESLint: Pour la qualité du code
- Add Prettier: Pour le formatage du code
- Add Vue DevTools 7: Pour le débogage (expérimental)

### Démarrage du Serveur de Développement
```bash
cd <your-project-name>
npm install
npm run dev
```

### Build de Production
```bash
npm run build
```

## Utilisation de Vue depuis un CDN

### Build Global
```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<div id="app">{{ message }}</div>

<script>
const { createApp } = Vue

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

### Build en Module ES
```html
<div id="app">{{ message }}</div>

<script type="module">
import { createApp } from 'https://unpkg.com/vue@3/dist/vue.esm-browser.js'

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

### Activation des Maps d'Importation
```html
<script type="importmap">
{
  "imports": {
    "vue": "https://unpkg.com/vue@3/dist/vue.esm-browser.js"
  }
}
</script>

<div id="app">{{ message }}</div>

<script type="module">
import { createApp } from 'vue'

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')
</script>
```

## Recommandations pour l'Environnement de Développement

- **IDE**: Visual Studio Code + [extension officielle Vue](https://marketplace.visualstudio.com/items?itemName=Vue.volar)
- **Outil de Compilation**: Vite (inclus dans le scaffolding officiel)
- **Serveur Local**: Pour les modules ES, utilisez `npx serve` pour démarrer un serveur HTTP local
- **Alternative pour les Cas Simples**: Considérez [petite-vue](https://github.com/vuejs/petite-vue) pour les scénarios légers
