# Jira

Jira est un logiciel de gestion de projet et de suivi des problèmes développé par Atlassian, offrant des solutions de gestion de projet à plus de 300 000 entreprises dans le monde.

## Démarrage rapide

Visitez le site web d'Atlassian pour vous inscrire et choisissez un plan adapté à votre équipe. Vous pouvez créer rapidement un site Jira.

## Caractéristiques principales

Gestion de projet et suivi des problèmes, Support du développement agile (Scrum, Kanban), Collaboration et communication d'équipe, Intégration avec l'assistant IA Rovo, Intégrations tierces riches, Puissantes règles d'automatisation.
