# Flutter - インストールガイド

完全なFlutter開発環境のセットアップガイド。

## 必要環境

### Windows

- **OS**：Windows 10以上（64ビット）
- **ディスク容量**：最低2.8GB
- **ツール**：PowerShell 5.0+、Git for Windows

### macOS

- **OS**：macOS 10.14（Mojave）以上
- **ディスク容量**：最低2.8GB
- **ツール**：Command Line Tools、Git

### Linux

- **OS**：Ubuntu 18.04以上
- **ディスク容量**：最低2.8GB
- **依存関係**：bash、curl、git、unzip、xz-utils、zip、libglu1-mesa

## インストール手順

### Windowsインストール

1. **Flutter SDKをダウンロード**

Flutterウェブサイトから最新バージョンをダウンロード：
```bash
# またはgitでクローン
git clone https://github.com/flutter/flutter.git -b stable
```

2. **環境変数を設定**

FlutterをシステムPATHに追加：
```powershell
# PowerShellで実行
[Environment]::SetEnvironmentVariable(
    "Path",
    $env:Path + ";C:\src\flutter\bin",
    "User"
)
```

3. **インストールを確認**

```bash
flutter doctor
```

### macOSインストール

1. **Flutter SDKをダウンロード**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

2. **環境変数を設定**

```bash
# ~/.bash_profile または ~/.zshrc に追加
export PATH="$PATH:/Users/ユーザー名/flutter/bin"
```

3. **flutter doctorを実行**

```bash
flutter doctor
```

### Linuxインストール

1. **依存関係をインストール**

```bash
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa
```

2. **Flutter SDKをダウンロード**

```bash
git clone https://github.com/flutter/flutter.git -b stable
```

3. **PATHを設定**

```bash
export PATH="$PATH:/home/ユーザー名/flutter/bin"
```

## Android開発環境の設定

### Android Studioをインストール

1. Android Studioをダウンロード：https://developer.android.com/studio
2. インストールして起動
3. SDKを設定：

```bash
flutter config --android-sdk /path/to/android/sdk
```

### ライセンスに同意

```bash
flutter doctor --android-licenses
```

### エミュレータを設定

```bash
# エミュレータを作成
flutter emulator --create --name my-emulator

# エミュレータを起動
flutter emulator --launch my-emulator
```

## iOS開発環境の設定（macOS）

### Xcodeをインストール

```bash
# App Storeからインストール
xcode-select --install
```

### iOSシミュレータを設定

```bash
# 利用可能なシミュレータを一覧表示
xcrun simctl list devices available

# シミュレータを起動
open -a Simulator
```

## IDE設定

### VS Code

1. Flutter拡張機能をインストール
2. Dart拡張機能をインストール
3. VS Codeを再起動

### Android Studio

1. Flutterプラグインをインストール
2. Dartプラグインをインストール
3. Android Studioを再起動

## 確認

```bash
# Flutterバージョンを確認
flutter --version

# flutter doctorを実行
flutter doctor

# アップデートを確認
flutter upgrade
```

## 一般的な問題

### ダウンロードに失敗？

ミラーを使う：
```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
export PUB_CACHE=/path/to/cache
```

### Android SDKが見つからない？

```bash
flutter config --android-sdk /path/to/android-sdk

# または環境変数を設定
export ANDROID_SDK_ROOT=/path/to/android-sdk
```

### コマンドが見つからない？

FlutterがPATHにあることを確認：
```bash
echo $PATH
# flutter/binが含まれている必要がある
```

## アンインストール

```bash
# Flutterディレクトリを削除
rm -rf /path/to/flutter

# PATHから削除
```
