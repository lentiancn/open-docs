# PlantUML 安裝指南

PlantUML 是一個用於快速建立 UML 圖的開源工具。本指南涵蓋在各種環境中的安裝方法。

## 版本資訊

| 版本 | 狀態 | 說明 |
|------|------|------|
| PlantUML | ✅ 目前 | 最新穩定版本 |

## 安裝方法

### 方法一：Windows 安裝

#### 1. 下載 JAR 檔案

1. 下載 PlantUML JAR：https://sourceforge.net/projects/plantuml/files/plantuml.jar/download
2. 保存到本機目錄

#### 2. 安裝 Graphviz（可選但推薦）

```powershell
# 使用 Chocolatey
choco install graphviz
```

### 方法二：macOS 安裝

```bash
# 使用 Homebrew
brew install plantuml

# 或安裝 Graphviz
brew install graphviz
```

### 方法三：Linux 安裝

```bash
# Ubuntu/Debian
sudo apt-get install -y plantuml graphviz

# CentOS/RHEL
sudo yum install -y plantuml graphviz
```

### 方法四：作為 VS Code 外掛

1. 開啟 VS Code
2. 搜尋 "PlantUML"
3. 安裝 "PlantUML" 外掛
4. 安裝 Graphviz

### 方法五：作為 IntelliJ 外掛

1. 開啟 IntelliJ IDEA
2. Settings → Plugins
3. 搜尋 "PlantUML"
4. 安裝外掛

### 方法六：使用 Docker

```bash
# 執行 PlantUML 伺服器
docker run -d -p 8080:8080 plantuml/plantuml-server:jetty

# 存取 http://localhost:8080
```

## 環境設定

### Java 需求

```bash
# 檢查 Java
java -version

# 需要 Java 8 或更高版本
```

### Graphviz 設定

```bash
# 設定環境變數（Linux/macOS）
export GRAPHVIZ_DOT=/usr/bin/dot

# Windows 新增到 PATH
# C:\Program Files\Graphviz\bin
```

## 常見問題

### 圖片不顯示？

確保已安裝 Graphviz：

```bash
dot -V
```

### 中文亂碼？

在圖表中加入：

```plantuml
skinparam defaultFontName SimSun
```
