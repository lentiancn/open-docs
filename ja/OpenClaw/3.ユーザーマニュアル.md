# OpenClaw ユーザーマニュアル

日常的な使い方と機能的介绍です。

## 基本概念

### Gateway

OpenClawの核心部分です。全部のメッセージ転送、AIリクエスト、チャンネルを管理します。一つだけ動かせばok。

### チャンネル

AIと話すためのチャットアプリです。WhatsApp、Telegram、Discordなどを同時につながります。

### エージェント

AIアシスタントの独立インスタンスです。複数動かして別々のことができます。

## 日常操作

### 起動・停止

```bash
# 起動
openclaw gateway start

# 停止
openclaw gateway stop

# 状態確認
openclaw gateway status
```

### メッセージ送信

```bash
openclaw message send --target +15555550123 --message "テスト"
```

### 設定管理

設定ファイル：`~/.openclaw/openclaw.json`

変更方法：
- ウィザード：`openclaw configure`
- コマンド：`openclaw config set xxx`
- Web画面：http://127.0.0.1:18789/
- 直接編集：`nano ~/.openclaw/openclaw.json`

## よく使う設定例

### アクセス制限

```json5
{
  channels: {
    whatsapp: {
      allowFrom: ["+15555550123"]
    }
  }
}
```

### グループチャット

```json5
{
  channels: {
    whatsapp: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

### 複数エージェント

```json5
{
  agents: {
    list: [
      { id: "home", default: true },
      { id: "work" }
    ]
  }
}
```
