# Apifox Installation Guide

Multiple Apifox installation methods.

## Download and Install

### Desktop App

1. Visit https://apifox.com
2. Download Windows/macOS/Linux version
3. Run installer

### Windows

- Download .exe installer
- Run installation wizard

### macOS

- Download .dmg package
- Drag to Applications

### Linux

- Download AppImage or deb package
- Grant execute permissions

## Web Version

Visit https://web.apifox.com directly.

## Account Registration

1. Open Apifox
2. Choose registration (email/phone/GitHub)
3. Complete registration and login

## Common Shortcuts

| Shortcut | Function |
|---------|----------|
| Ctrl + Enter | Send request |
| Ctrl + S | Save |
| Ctrl + / | Comment |
| Ctrl + Space | Autocomplete |

## First Use

### Create Project

1. Click "New Project"
2. Enter project name
3. Select project type

### Add API

1. Click "New Interface" in project
2. Fill in interface details
3. Send request to test
