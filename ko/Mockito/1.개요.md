# Mockito 개요

Java 단위 테스트를 위한 주요 모킹 프레임워크.

## Mockito란?

Mockito는 Java 단위 테스트를 위한 주요 모킹 프레임워크입니다. 개발자가 격리된 테스트를 작성하고 외부 컴포넌트에 대한 종속성을 제거하기 위한 모크 객체를 생성하고 검증할 수 있습니다.

### 주요 기능

- **직관적인 API** - 배우기 쉽고 사용하기 쉽습니다
- **유연한 모킹** - 클래스 및 인터페이스에 대한 모크 생성
- **검증** - 메서드 호출 검증
- **스텁 설정** - 사전 정의된 반환 값 설정
- **인수 매처** - 유연한 인수 검증

## 핵심 개념

- **모크** - 실제 객체의 동작을 시뮬레이션하는 대리 객체
- **스텁** - 메서드 호출에 대한 사전 정의된 응답
- **검증** - 특정 메서드가 호출되었는지 확인
