# FantasyTalking Guide d'installation

## Configuration requise

- Node.js 14.0+ ou Python 3.7+
- FFmpeg pour le traitement audio
- 4GB RAM recommandé

## Installation

```bash
# Node.js
npm install fantasy-talking

# Python
pip install fantasy-talking
```

## Démarrage rapide

```javascript
const client = new FantasyTalking({
  apiKey: 'votre-api-key',
  voice: { style: 'neutral', language: 'fr-FR' }
});

const audio = await client.speak('Bonjour, ceci est un test');
```
