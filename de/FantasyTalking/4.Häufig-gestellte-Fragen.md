# FantasyTalking FAQ

## Installation

### Installation fehlgeschlagen?

- Node.js Version prüfen (>= 14.0)
- npm Cache leeren
- Aus Quellcode installieren

## Nutzung

### Sprachsynthese langsam?

- Netzwerkverbindung prüfen
- Selbst gehosteten Dienst nutzen
- Stapelverarbeitung verwenden

### API-Schlüssel erhalten?

Auf der offiziellen Website registrieren und in der Konsole erstellen.
