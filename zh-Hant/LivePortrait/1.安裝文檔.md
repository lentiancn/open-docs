# LivePortrait 安裝指南

LivePortrait 是由快手 VGI 開發的高效人像動畫框架。它使用驅動影片中的運動來為靜態肖像製作動畫，支援拼接和重定向控制。

## 版本概述

| 版本 | 狀態 | 描述 |
|------|------|------|
| LivePortrait 1.x | 目前 | 主要版本 |

## 系統要求

### 硬體要求

| 元件 | 最低配置 | 推薦配置 |
|------|----------|----------|
| GPU | NVIDIA 8GB 顯示記憶體 | NVIDIA 16GB 顯示記憶體 |
| 記憶體 | 8GB | 16GB |
| 儲存空間 | 30GB | 50GB |
| CUDA | 11.8+ | 12.1+ |

### 軟體要求

| 軟體 | 版本 | 說明 |
|------|------|------|
| Python | 3.8 - 3.11 | 推薦 3.10 |
| CUDA | 11.8+ | GPU 加速必需 |
| cuDNN | 8.5+ | CUDA 必需 |
| ffmpeg | 最新版 | 影片處理必需 |

## Linux/Ubuntu 安裝

### 步驟 1：安裝系統依賴

```bash
# 更新系統
sudo apt update && sudo apt upgrade -y

# 安裝 Python 和開發工具
sudo apt install python3.10 python3-pip python3-venv git wget curl

# 安裝系統庫
sudo apt install libgl1-mesa-glx libglib2.0-0 libsm6 libxext6 libxrender-dev libgomp1

# 安裝 ffmpeg
sudo apt install ffmpeg
```

### 步驟 2：安裝 CUDA

```bash
# 下載並安裝 CUDA 12.1
wget https://developer.download.nvidia.com/compute/cuda/12.1.0/local_installers/cuda_12.1.0_520.53_linux.run
sudo sh cuda_12.1.0_520.53_linux.run
```

### 步驟 3：複製專案

```bash
git clone https://github.com/KwaiVGI/LivePortrait.git
cd LivePortrait
```

### 步驟 4：建立虛擬環境

```bash
python3.10 -m venv venv
source venv/bin/activate

# 驗證 Python 版本
python --version
```

### 步驟 5：安裝 PyTorch

```bash
# CUDA 12.1 版本
pip install torch==2.1.0 torchvision==0.16.0 --index-url https://download.pytorch.org/whl/cu121

# 驗證安裝
python -c "import torch; print(torch.cuda.is_available())"
```

### 步驟 6：安裝依賴

```bash
# 安裝基礎依賴
pip install -r requirements.txt

# 安裝額外依賴
pip install numpy opencv-python librosa soundfile
```

### 步驟 7：下載模型

```bash
# 建立模型目錄
mkdir -p pretrained_models

# 從發布頁面下載或使用下載腳本
bash scripts/download_weights.sh
```

## Windows 安裝

### 步驟 1：安裝 Python

1. 從 python.org 下載 Python 3.10
2. 安裝時勾選「Add Python to PATH」
3. 驗證：`python --version`

### 步驟 2：安裝 CUDA

1. 從 NVIDIA 網站下載 CUDA 12.1
2. 預設安裝
3. 下載並安裝 cuDNN 8.9

### 步驟 3：複製和設定

```powershell
git clone https://github.com/KwaiVGI/LivePortrait.git
cd LivePortrait

python -m venv venv
venv\Scripts\activate
```

### 步驟 4：安裝依賴

```powershell
# 安裝 PyTorch（CUDA 支援）
pip install torch==2.1.0 torchvision==0.16.0 --index-url https://download.pytorch.org/whl/cu121

pip install -r requirements.txt
```

### 步驟 5：下載模型

從 GitHub releases 頁面下載模型，放入 `pretrained_models/` 目錄。

## macOS 安裝

### 步驟 1：安裝依賴

```bash
# 安裝 Homebrew（如未安裝）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安裝 Python 和 ffmpeg
brew install python@3.10 ffmpeg git
```

### 步驟 2：設定

```bash
git clone https://github.com/KwaiVGI/LivePortrait.git
cd LivePortrait

python3.10 -m venv venv
source venv/bin/activate

# 安裝 PyTorch（macOS 僅 CPU）
pip install torch==2.1.0 torchvision==0.16.0

pip install -r requirements.txt
```

注意：macOS 僅支援 CPU 模式，無法使用 GPU 加速。

## Docker 安裝

### 使用 Docker

```bash
# 拉取映像檔
docker pull liveportrait:latest

# 執行（需要 GPU）
docker run --gpus all -v $(pwd):/workspace -p 7860:7860 liveportrait:latest
```

### 使用 Docker Compose

```yaml
version: '3.8'
services:
  liveportrait:
    image: liveportrait:latest
    volumes:
      - ./workspace:/workspace
    ports:
      - "7860:7860"
    deploy:
      resources:
        reservations:
          devices:
            - driver: nvidia
              count: 1
              capabilities: [gpu]
```

## 常見問題

### CUDA 記憶體不足

**解決方案：**
- 減小批次大小：`--batch_size 1`
- 使用較低解析度
- 啟用模型卸載

### 缺少庫（Windows）

**解決方案：**
- 安裝 Visual C++ Redistributable 2015-2022
- 將 CUDA 新增到系統 PATH

### 模型下載失敗

**解決方案：**
- 使用 VPN 或代理
- 從 GitHub releases 手動下載
- 檢查網路連線

### 找不到 ffmpeg

**解決方案：**
- Linux：`sudo apt install ffmpeg`
- Windows：將 ffmpeg 新增到 PATH
- macOS：`brew install ffmpeg`
