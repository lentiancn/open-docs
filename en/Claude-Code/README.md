# Claude Code

[Claude Code](https://code.claude.com/) is an intelligent programming assistant developed by Anthropic that can read codebases, edit files, execute commands, and integrate seamlessly with development tools.

## Overview

Claude Code is an AI-powered coding assistant available in terminals, IDEs, desktop applications, and browsers. It helps you build features, fix bugs, and automate development tasks.

## Features

- **Automate Tasks**: Write tests, fix lint errors, resolve merge conflicts, update dependencies
- **Feature Development**: Describe requirements in natural language; Claude Code plans and implements solutions
- **Git Integration**: Stage changes, write commit messages, create branches, open Pull Requests
- **MCP Integration**: Connect to external data sources and tools
- **Customization**: CLAUDE.md, custom commands, Hooks
- **Multi-Agent Collaboration**: Parallel processing for complex tasks

## Quick Start

### Install

**macOS, Linux, WSL:**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell:**
```powershell
irm https://claude.ai/install.ps1 | iex
```

### Launch

```bash
cd your-project
claude
```
