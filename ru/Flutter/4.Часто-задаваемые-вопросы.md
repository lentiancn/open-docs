# Flutter - FAQ

Частые вопросы.

## Установка

### Ошибка загрузки?

```bash
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
```

### Ошибка flutter doctor?

Проверить PATH.

### Android SDK не найден?

```bash
flutter config --android-sdk /путь/к/android-sdk
```

## Разработка

### Hot Reload не работает?

Сохранить и подождать.

### Сборка APK не удалась?

```bash
flutter clean
```
