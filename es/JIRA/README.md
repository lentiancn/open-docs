# JIRA

JIRA es una herramienta de seguimiento de problemas y gestión de proyectos desarrollada por Atlassian.

## Descripción

JIRA es una herramienta líder en la industria para la gestión de proyectos y problemas, ampliamente utilizada en desarrollo ágil y gestión de servicios de TI.

## Características Principales

- **Seguimiento de Problemas**: Gestión completa de defectos y tareas
- **Kanban Ágil**: Gestión visual de tareas
- **Soporte para Scrum**: Planificación y seguimiento de sprints
- **Automatización de Flujos de Trabajo**: Automatización de procesos de negocio
- **Análisis e Informes**: Funciones ricas de generación de informes
- **Integraciones**: Integración con múltiples herramientas

## Inicio Rápido

### Instalación

```bash
# Docker
docker run -v jira-data:/var/atlassian -p 8080:8080 atlassian/jira-software
```

### Acceso

Abra su navegador y acceda a http://localhost:8080
