## Installation Methods

### Windows Installation

#### Using MSI Installer

1. Download Go installer: https://go.dev/dl/
2. Run the MSI installer
3. Follow the installation wizard
4. Open command prompt, type `go version` to verify

#### Using Chocolatey

```powershell
choco install golang
```

### macOS Installation

#### Using Homebrew

```bash
brew install go
```

#### Using DMG Package

1. Download macOS package: https://go.dev/dl/
2. Open the package file and run it
3. Move Go directory to /usr/local/go: `sudo mv go /usr/local/`

### Linux Installation

#### Using Package Manager

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install golang-go

# CentOS/RHEL
sudo yum install golang
```

#### Manual Installation

```bash
# Download Go binary
wget https://go.dev/dl/go1.22.0.linux-amd64.tar.gz

# Extract to /usr/local
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go1.22.0.linux-amd64.tar.gz

# Add to PATH
export PATH=$PATH:/usr/local/go/bin

# Verify installation
go version
```

## Environment Configuration

### Set GOPATH

```bash
# Linux/macOS
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin

# Windows (PowerShell)
$env:GOPATH = "$env:USERPROFILE\go"
$env:PATH += ";$env:GOPATH\bin"
```

### Configure Go Module Proxy

```bash
# Use China mirror
go env -w GOPROXY=https://goproxy.cn,direct

# Set proxy
go env -w GOPROXY=https://proxy.golang.org,direct
```

### Verify Installation

```bash
# Check version
go version

# View environment info
go env

# Run test
go version
```

## Common Commands

```bash
# Initialize module

# Download dependencies
go mod tidy

# Build project
go build

# Run program
go run main.go

# Run tests
go test ./...

# Install tools
go install
```

## Troubleshooting

### Installation failed?

- Check system compatibility
- Confirm the downloaded version matches your system architecture
- Check permissions

### Command not found?

- Verify Go is added to PATH environment variable
- Reopen terminal
- Check installation path

### Slow dependency downloads?

- Configure China mirror proxy
- Use VPN
