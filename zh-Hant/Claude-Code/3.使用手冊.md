# Claude Code 使用手冊

基本使用方法和技巧。

## 基本使用

### 啟動

- VS Code：使用快捷鍵 Cmd/Ctrl + Shift + P
- CLI：直接在終端運行 `claude`

### 對話

直接在聊天窗口描述需求：
```
幫我寫一個排序函數
解釋這段代碼的含義
這個 bug 怎麼修復
```

## 代碼生成

### 生成新代碼

描述你想要的功能：
```
創建一個用戶登錄 API
實現分頁查詢功能
編寫單元測試
```

### 修改現有代碼

1. 選中代碼
2. 描述修改需求
3. Claude 會生成修改後的代碼

## 調試輔助

### 解釋錯誤

直接粘貼錯誤信息：
```
這個報錯是什麼意思？
```

### 修復建議

讓 Claude 分析問題：
```
幫我看看這個函數有什麼問題
```

## 代碼審查

### 代碼審查請求

```
審查這段代碼
有什麼安全問題
性能優化建議
```

## 快捷鍵

### VS Code

| 快捷鍵 | 功能 |
|--------|------|
| Cmd/Ctrl + L | 打開對話 |
| Cmd/Ctrl + Enter | 提交輸入 |

## 配置選項

### claude.json

在項目根目錄創建 `claude.json`：

```json
{
  "preamble": "你是一個 Python 專家",
  "maxTokens": 4000
}
```

## 最佳實踐

1. 清晰描述需求
2. 提供上下文信息
3. 迭代改進代碼
4. 結合人工審核
