# Angular User Manual

Angular core concepts and basic usage.

## Components

### Create Component

```bash
ng generate component my-component
# or
ng g c my-component
```

### Component Structure

```typescript
import { Component } from '@angular/core';

@Component({
  selector: 'app-my-component',
  templateUrl: './my-component.component.html',
  styleUrls: ['./my-component.component.css']
})
export class MyComponentComponent {
  title = 'Hello Angular';
}
```

## Data Binding

### Property Binding

```html
<img [src]="imageUrl">
```

### Event Binding

```html
<button (click)="onClick()">Click</button>
```

### Two-way Binding

```html
<input [(ngModel)]="name">
```

## Services

### Create Service

```bash
ng generate service my-service
# or
ng g s my-service
```

### Using Services

```typescript
import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root'
})
export class MyServiceService {
  getData() {
    return 'data';
  }
}
```

## Routing

### Configure Routes

```typescript
const routes: Routes = [
  { path: '', component: HomeComponent },
  { path: 'about', component: AboutComponent },
  { path: '**', component: NotFoundComponent }
];
```

### Route Navigation

```html
<a routerLink="/about">About</a>
<router-outlet></router-outlet>
```

## HTTP Requests

### Import HttpClient

```typescript
import { HttpClientModule } from '@angular/common/http';
```

### Make Requests

```typescript
import { HttpClient } from '@angular/common/http';

constructor(private http: HttpClient) {}

getData() {
  return this.http.get('https://api.example.com/data');
}
```

## Forms

### Reactive Forms

```typescript
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

form: FormGroup;

constructor(private fb: FormBuilder) {
  this.form = this.fb.group({
    name: ['', Validators.required],
    email: ['', [Validators.required, Validators.email]]
  });
}
```

## Lifecycle Hooks

| Hook | Description |
|------|-------------|
| ngOnInit | Component initialization |
| ngOnChanges | Input properties change |
| ngOnDestroy | Component destruction |

## Directives

### Structural Directives

```html
<div *ngIf="show">Show</div>
<div *ngFor="let item of items">{{item}}</div>
```

### Attribute Directives

```html
<div [ngClass]="{'active': isActive}"></div>
```

## Best Practices

1. Use TypeScript strong typing
2. Keep components simple
3. Use services for shared logic
4. Unsubscribe from observables
5. Use OnPush change detection strategy
