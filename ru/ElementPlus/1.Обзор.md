# Element Plus Обзор

UI-компонентная библиотека для Vue 3.

## Что такое Element Plus?

Element Plus — это библиотека компонентов Vue 3, разработанная командой Ele.me, предоставляющая более 70 высококачественных компонентов.

### Основные функции

- **На Vue 3**: Построена на Composition API
- **TypeScript**: Полная поддержка типов
- **Tree Shaking**: Оптимизация размера бандла
- **Темы**: Настройка через CSS переменные
- **Интернационализация**: 20+ языков
