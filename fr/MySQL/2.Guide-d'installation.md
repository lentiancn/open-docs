# Guide d'installation de MySQL

## Installation sous Windows

### Installation avec MySQL Installer

MySQL Installer est le moyen le plus simple d'installer MySQL sous Windows, offrant une expérience d'installation basée sur un assistant.

1. Télécharger MySQL Installer
   Visitez https://dev.mysql.com/downloads/installer/ pour télécharger l'installateur.

2. Exécuter l'installateur
   Double-cliquez sur le fichier MSI téléchargé pour lancer l'assistant d'installation.

3. Choisir le type d'installation
   - **Full** : Installer tous les produits et fonctionnalités MySQL
   - **Developer Default** : Adapté à l'environnement de développement
   - **Server Only** : Installer uniquement MySQL Server
   - **Custom** : Choisir les composants spécifiques

4. Configurer MySQL Server
   - Sélectionner la version de MySQL Server
   - Définir le numéro de port (par défaut 3306)
   - Choisir le type de démarrage

5. Définir le mot de passe root
   Définissez un mot de passe fort pour le compte administrateur MySQL.

6. Terminer l'installation
   Suivez l'assistant pour terminer le processus d'installation.

### Installation manuelle avec ZIP

1. Télécharger l'archive ZIP MySQL
   Visitez https://dev.mysql.com/downloads/mysql/ pour télécharger l'archive ZIP.

2. Extraire les fichiers
   Extrayez l'archive ZIP dans le répertoire souhaité, comme C:\mysql.

3. Initialiser le répertoire de données
   ```
   mysqld --initialize-insecure
   ```

4. Démarrer le service MySQL
   ```
   mysqld --install MySQL
   net start MySQL
   ```

5. Se connecter et changer le mot de passe
   ```
   mysql -u root -p
   ALTER USER 'root'@'localhost' IDENTIFIED BY 'nouveau_mot_de_passe';
   ```

## Installation sous Linux

### Systèmes Ubuntu/Debian

1. Mettre à jour l'index des paquets
   ```bash
   sudo apt update
   ```

2. Installer MySQL Server
   ```bash
   sudo apt install mysql-server
   ```

3. Démarrer le service MySQL
   ```bash
   sudo systemctl start mysql
   sudo systemctl enable mysql
   ```

4. Configuration sécurisée
   ```bash
   sudo mysql_secure_installation
   ```

### Systèmes CentOS/RHEL

1. Ajouter le dépôt MySQL Yum
   ```bash
   sudo yum install https://dev.mysql.com/get/mysql80-community-release-el8-4.noarch.rpm
   ```

2. Installer MySQL Server
   ```bash
   sudo yum install mysql-community-server
   ```

3. Démarrer le service
   ```bash
   sudo systemctl start mysqld
   sudo systemctl enable mysqld
   ```

4. Obtenir le mot de passe temporaire
   ```bash
   sudo grep 'temporary password' /var/log/mysqld.log
   ```

## Installation sous macOS

### Utilisation de Homebrew

1. Installer Homebrew (si non installé)
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```

2. Installer MySQL
   ```bash
   brew install mysql
   ```

3. Démarrer le service MySQL
   ```bash
   brew services start mysql
   ```

4. Configuration sécurisée
   ```bash
   mysql_secure_installation
   ```

### Utilisation du paquet DMG

1. Télécharger le paquet DMG
   Visitez https://dev.mysql.com/downloads/mysql/ pour télécharger le paquet DMG macOS.

2. Exécuter l'installateur
   Double-cliquez sur le fichier PKG et suivez l'assistant pour compléter l'installation.

3. Démarrer MySQL
   Démarrez le service MySQL via Préférences Système.

## Installation de MySQL avec Docker

### Récupérer l'image MySQL
```bash
docker pull mysql:latest
```

### Exécuter le conteneur MySQL
```bash
docker run -d \
  --name mysql-container \
  -e MYSQL_ROOT_PASSWORD=votre_mot_de_passe \
  -e MYSQL_DATABASE=testdb \
  -p 3306:3306 \
  mysql:latest
```

### Paramètres Docker courants
- `-e MYSQL_ROOT_PASSWORD` : Définir le mot de passe root
- `-e MYSQL_DATABASE` : Créer une base de données initiale
- `-p 3306:3306` : Mappage de ports
- `-v /data/mysql:/var/lib/mysql` : Volume pour la persistance des données

## Vérification après l'installation

### Vérifier la version MySQL
```bash
mysql --version
```

### Vérifier l'état du service
```bash
systemctl status mysql   # Linux
brew services list      # macOS
```

### Se connecter à MySQL
```bash
mysql -u root -p
```

## Problèmes d'installation courants

### Conflits de ports
Si le port 3306 est occupé, modifiez le port dans le fichier de configuration ou arrêtez le service utilisant ce port.

### Problèmes de permissions
Sous Linux, utilisez sudo pour élever les permissions et assurezvous que le répertoire de données a les bonnes permissions de lecture/écriture.

### Paramètres du pare-feu
Assurez-vous que le pare-feu permet l'accès au port 3306.
