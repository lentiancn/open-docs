# Base de données H2 - Introduction

## Qu'est-ce que H2 ?

H2 est un système de gestion de base de données relationnelles open source écrit en Java. Il peut être utilisé comme base de données intégrée dans les applications Java ou fonctionner en mode client-serveur. Le logiciel est disponible en open source sous la Licence Publique Mozilla 2.0 (MPL 2.0) ou la Licence Publique Eclipse originale (EPL 1.0).

## Caractéristiques principales de H2

### Base de données intégrée
H2 peut être intégré directement dans les applications Java pour fonctionner sans serveur de base de données distinct. Cela le rend très populaire dans les environnements de développement et de test.

### Mode client-serveur
En plus du mode intégré, H2 prend également en charge le mode client-serveur, fonctionnant comme un serveur de base de données indépendant auquel plusieurs applications peuvent accéder simultanément.

### Base de données en mémoire
H2 prend en charge le mode base de données en mémoire, où les données sont stockées en mémoire et perdues après un redémarrage. Ceci est très utile pour les tests rapides et le développement de prototypes.

### Compatibilité avec plusieurs bases de données
H2 est compatible avec diverses normes et syntaxes SQL et peut simuler le comportement de MySQL, PostgreSQL, Oracle, DB2, MS SQL Server, facilitant la migration et les tests de bases de données.

### Base de données chiffrée
H2 prend en charge le chiffrement de la base de données pour protéger les données sensibles.

### Recherche en texte intégral
H2 dispose de fonctions de recherche en texte intégré sans configuration supplémentaire.

### Zéro administration
En tant que base de données intégrée, H2 ne nécessite pas d'administrateur de base de données dédié pour sa maintenance.

## Cas d'utilisation de H2

H2 est largement utilisé dans les scénarios suivants :

- Base de données intégrée pour applications Java
- Environnements de développement et de test
- Développement rapide de prototypes
- Tests unitaires et d'intégration
- Applications de bureau légères
- Apprentissage de la syntaxe SQL

## Historique des versions de H2

Le développement du moteur de base de données H2 a commencé en mai 2004, avec la première publication en décembre 2005. Le développeur d'origine Thomas Mueller est également le fondateur du moteur de base de données Java HSQLDB (anciennement Hypersonic SQL).
