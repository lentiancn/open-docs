## 一、元件使用基礎

### 1.1 基礎按鈕

按鈕是最常用的元件之一，Element Plus 提供了多種類型的按鈕。

```vue
<template>
  <!-- 主要按鈕 -->
  <el-button type="primary">主要按鈕</el-button>
  
  <!-- 成功按鈕 -->
  <el-button type="success">成功按鈕</el-button>
  
  <!-- 資訊按鈕 -->
  <el-button type="info">資訊按鈕</el-button>
  
  <!-- 警告按鈕 -->
  <el-button type="warning">警告按鈕</el-button>
  
  <!-- 危險按鈕 -->
  <el-button type="danger">危險按鈕</el-button>
  
  <!-- 文字按鈕 -->
  <el-button type="text">文字按鈕</el-button>
  
  <!-- 樸素按鈕 -->
  <el-button plain>樸素按鈕</el-button>
  
  <!-- 圓角按鈕 -->
  <el-button round>圓角按鈕</el-button>
  
  <!-- 圖標按鈕 -->
  <el-button icon="el-icon-search">搜尋</el-button>
</template>
```

### 1.2 按鈕屬性

| 屬性 | 說明 | 類型 | 可選值 | 預設值 |
|------|------|------|--------|--------|
| type | 按鈕類型 | string | primary / success / warning / danger / info / text | — |
| plain | 是否樸素按鈕 | boolean | — | false |
| round | 是否圓角按鈕 | boolean | — | false |
| circle | 是否圓形按鈕 | boolean | — | false |
| loading | 是否載入中 | boolean | — | false |
| disabled | 是否禁用 | boolean | — | false |
| icon | 圖標類名 | string | — | — |
| size | 尺寸 | string | large / default / small | — |

### 1.3 按鈕事件

```vue
<template>
  <el-button @click="handleClick">點擊事件</el-button>
</template>

<script setup>
const handleClick = () => {
  console.log('按鈕被點擊了')
}
</script>
```

## 二、表單元件

### 2.1 基礎表單

```vue
<template>
  <el-form :model="form" :rules="rules" ref="formRef" label-width="100px">
    <el-form-item label="使用者名稱" prop="username">
      <el-input v-model="form.username" placeholder="請輸入使用者名稱" />
    </el-form-item>
    
    <el-form-item label="密碼" prop="password">
      <el-input v-model="form.password" type="password" placeholder="請輸入密碼" show-password />
    </el-form-item>
    
    <el-form-item label="郵箱" prop="email">
      <el-input v-model="form.email" placeholder="請輸入郵箱" />
    </el-form-item>
    
    <el-form-item label="性別" prop="gender">
      <el-radio-group v-model="form.gender">
        <el-radio label="male">男</el-radio>
        <el-radio label="female">女</el-radio>
      </el-radio-group>
    </el-form-item>
    
    <el-form-item label="愛好" prop="hobby">
      <el-checkbox-group v-model="form.hobby">
        <el-checkbox label="籃球" />
        <el-checkbox label="足球" />
        <el-checkbox label="游泳" />
      </el-checkbox-group>
    </el-form-item>
    
    <el-form-item label="簡介" prop="bio">
      <el-input v-model="form.bio" type="textarea" :rows="3" placeholder="請輸入個人簡介" />
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm">提交</el-button>
      <el-button @click="resetForm">重設</el-button>
    </el-form-item>
  </el-form>
</template>

<script setup>
import { reactive, ref } from 'vue'

const formRef = ref(null)

const form = reactive({
  username: '',
  password: '',
  email: '',
  gender: '',
  hobby: [],
  bio: ''
})

const rules = {
  username: [
    { required: true, message: '請輸入使用者名稱', trigger: 'blur' },
    { min: 3, max: 20, message: '使用者名稱長度在 3 到 20 個字元', trigger: 'blur' }
  ],
  password: [
    { required: true, message: '請輸入密碼', trigger: 'blur' },
    { min: 6, message: '密碼長度至少 6 位', trigger: 'blur' }
  ],
  email: [
    { required: true, message: '請輸入郵箱', trigger: 'blur' },
    { type: 'email', message: '請輸入正確的郵箱格式', trigger: 'blur' }
  ],
  gender: [
    { required: true, message: '請選擇性別', trigger: 'change' }
  ]
}

const submitForm = async () => {
  if (!formRef.value) return
  
  await formRef.value.validate((valid) => {
    if (valid) {
      console.log('表單資料：', form)
      ElMessage.success('提交成功')
    } else {
      ElMessage.error('請檢查表單填寫')
    }
  })
}

const resetForm = () => {
  formRef.value?.resetFields()
}
</script>
```

### 2.2 表單驗證規則

| 規則類型 | 說明 | 示例 |
|----------|------|------|
| required | 必填項 | { required: true, message: '必填', trigger: 'blur' } |
| type | 資料類型 | type: 'email', type: 'number', type: 'date' |
| min/max | 長度限制 | { min: 3, max: 20, message: '長度不對', trigger: 'blur' } |
| pattern | 正則表達式 | { pattern: /^1[3-9]\d{9}$/, message: '手機號格式不對' } |
| validator | 自定義驗證 | { validator: validatePass, trigger: 'blur' } |

## 三、表格元件

### 3.1 基礎表格

```vue
<template>
  <el-table :data="tableData" stripe border>
    <el-table-column prop="date" label="日期" width="180" />
    <el-table-column prop="name" label="姓名" width="180" />
    <el-table-column prop="address" label="地址" />
    <el-table-column label="操作" width="200">
      <template #default="{ row }">
        <el-button type="primary" size="small" @click="handleEdit(row)">編輯</el-button>
        <el-button type="danger" size="small" @click="handleDelete(row)">刪除</el-button>
      </template>
    </el-table-column>
  </el-table>
</template>

<script setup>
import { ref } from 'vue'

const tableData = ref([
  { date: '2024-01-01', name: '張三', address: '北京市朝陽區' },
  { date: '2024-01-02', name: '李四', address: '上海市浦東新區' },
  { date: '2024-01-03', name: '王五', address: '廣州市天河區' }
])

const handleEdit = (row) => {
  console.log('編輯：', row)
}

const handleDelete = (row) => {
  console.log('刪除：', row)
}
</script>
```

### 3.2 帶分頁的表格

```vue
<template>
  <div>
    <el-table :data="tableData" v-loading="loading">
      <el-table-column prop="id" label="ID" width="80" />
      <el-table-column prop="title" label="標題" />
      <el-table-column prop="author" label="作者" width="120" />
      <el-table-column prop="date" label="發布日期" width="180" />
    </el-table>
    
    <el-pagination
      v-model:current-page="currentPage"
      v-model:page-size="pageSize"
      :total="total"
      :page-sizes="[10, 20, 50, 100]"
      layout="total, sizes, prev, pager, next, jumper"
      @size-change="handleSizeChange"
      @current-change="handleCurrentChange"
    />
  </div>
</template>

<script setup>
import { ref, onMounted } from 'vue'

const tableData = ref([])
const loading = ref(false)
const currentPage = ref(1)
const pageSize = ref(10)
const total = ref(0)

const loadData = async () => {
  loading.value = true
  // 模擬接口調用
  setTimeout(() => {
    tableData.value = [
      { id: 1, title: '文章標題1', author: '作者1', date: '2024-01-01' },
      { id: 2, title: '文章標題2', author: '作者2', date: '2024-01-02' }
    ]
    total.value = 100
    loading.value = false
  }, 500)
}

const handleSizeChange = (val) => {
  pageSize.value = val
  loadData()
}

const handleCurrentChange = (val) => {
  currentPage.value = val
  loadData()
}

onMounted(() => {
  loadData()
})
</script>
```

## 四、對話框和抽屜

### 4.1 對話框

```vue
<template>
  <el-button type="primary" @click="dialogVisible = true">打開對話框</el-button>
  
  <el-dialog
    v-model="dialogVisible"
    title="編輯資訊"
    width="500px"
    :close-on-click-modal="false"
    @close="handleClose"
  >
    <el-form :model="form" label-width="80px">
      <el-form-item label="名稱">
        <el-input v-model="form.name" />
      </el-form-item>
      <el-form-item label="描述">
        <el-input v-model="form.desc" type="textarea" :rows="3" />
      </el-form-item>
    </el-form>
    
    <template #footer>
      <el-button @click="dialogVisible = false">取消</el-button>
      <el-button type="primary" @click="handleConfirm">確定</el-button>
    </template>
  </el-dialog>
</template>

<script setup>
import { ref, reactive } from 'vue'

const dialogVisible = ref(false)

const form = reactive({
  name: '',
  desc: ''
})

const handleClose = () => {
  form.name = ''
  form.desc = ''
}

const handleConfirm = () => {
  console.log('提交：', form)
  dialogVisible.value = false
  ElMessage.success('操作成功')
}
</script>
```

## 五、訊息提示

### 5.1 全域訊息

```vue
<script setup>
import { ElMessage } from 'element-plus'

// 成功提示
const showSuccess = () => {
  ElMessage.success('操作成功')
}

// 警告提示
const showWarning = () => {
  ElMessage.warning('警告資訊')
}

// 錯誤提示
const showError = () => {
  ElMessage.error('操作失敗')
}

// 資訊提示
const showInfo = () => {
  ElMessage.info('提示資訊')
}

// 可關閉的訊息
const showClosable = () => {
  ElMessage({
    message: '這是一條可關閉的訊息',
    showClose: true,
  })
}
</script>
```

### 5.2 通知提醒

```vue
<script setup>
import { ElNotification } from 'element-plus'

// 成功通知
const notifySuccess = () => {
  ElNotification({
    title: '成功',
    message: '操作已成功完成',
    type: 'success',
  })
}

// 錯誤通知
const notifyError = () => {
  ElNotification({
    title: '錯誤',
    message: '操作執行失敗',
    type: 'error',
  })
}
</script>
```
