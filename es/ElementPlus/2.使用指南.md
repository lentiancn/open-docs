# Guía de Uso de Element Plus

Esta guía te enseña cómo usar los componentes de Element Plus en aplicaciones Vue 3.

## Inicio Rápido

### Plantilla Básica

```vue
<template>
  <div>
    <el-button type="primary">Botón Primario</el-button>
    <el-input v-model="input" placeholder="Ingrese texto" />
    <el-date-picker v-model="date" type="date" placeholder="Seleccione fecha" />
  </div>
</template>

<script setup>
import { ref } from 'vue'

const input = ref('')
const date = ref('')
</script>
```

### Formulario

```vue
<template>
  <el-form :model="form" :rules="rules" ref="formRef" label-width="120px">
    <el-form-item label="Nombre del Evento" prop="name">
      <el-input v-model="form.name"></el-input>
    </el-form-item>
    
    <el-form-item label="Área del Evento" prop="region">
      <el-select v-model="form.region" placeholder="Por favor seleccione">
        <el-option label="Área 1" value="shanghai"></el-option>
        <el-option label="Área 2" value="beijing"></el-option>
      </el-select>
    </el-form-item>
    
    <el-form-item label="Fecha del Evento">
      <el-date-picker
        v-model="form.date"
        type="datetimerange"
        range-separator="a"
        start-placeholder="Fecha de inicio"
        end-placeholder="Fecha de fin"
      ></el-date-picker>
    </el-form-item>
    
    <el-form-item label="Entrega Inmediata">
      <el-switch v-model="form.delivery"></el-switch>
    </el-form-item>
    
    <el-form-item label="Recursos del Evento">
      <el-radio-group v-model="form.resource">
        <el-radio label="Patrocinio de Marca Online">Patrocinio de Marca Online</el-button>
        <el-radio label="Lugar Presencial Gratis">Lugar Presencial Gratis</el-radio>
      </el-radio-group>
    </el-form-item>
    
    <el-form-item label="Forma del Evento">
      <el-input v-model="form.desc" type="textarea"></el-input>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="onSubmit">Crear</el-button>
      <el-button @click="onReset">Restablecer</el-button>
    </el-form-item>
  </el-form>
</template>

<script setup>
import { reactive, ref } from 'vue'

const formRef = ref(null)

const form = reactive({
  name: '',
  region: '',
  date: [],
  delivery: false,
  resource: '',
  desc: ''
})

const rules = {
  name: [
    { required: true, message: 'Por favor ingrese el nombre del evento', trigger: 'blur' },
    { min: 3, max: 5, message: 'Longitud entre 3 y 5 caracteres', trigger: 'blur' }
  ],
  region: [
    { required: true, message: 'Por favor seleccione el área del evento', trigger: 'change' }
  ]
}

const onSubmit = () => {
  formRef.value.validate((valid) => {
    if (valid) {
      console.log('submit!')
    }
  })
}

const onReset = () => {
  formRef.value.resetFields()
}
</script>
```

## Preguntas Frecuentes

### ¿Los componentes no se renderizan?

Asegúrate de que los componentes estén correctamente importados o registrados globalmente.

### ¿Los estilos no se aplican?

Verifica que los estilos de Element Plus estén importados:

```javascript
import 'element-plus/dist/index.css'
```

### ¿Los iconos no se muestran?

Registra los iconos globalmente:

```javascript
import * as icons from '@element-plus/icons-vue'
for (const [key, icon] of Object.entries(icons)) {
    app.component(key, icon)
}
```
