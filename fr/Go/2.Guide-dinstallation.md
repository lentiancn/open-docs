# Guide d'installation de Go

> Configuration complète de l'environnement

---

## Configuration requise

| Environnement | Requirement |
|---------------|-------------|
| SE | Linux, macOS, Windows |
| Architecture | amd64, arm64, 386 |

---

## Installation

### Windows

```powershell
choco install golang
```

### macOS

```bash
brew install go
```

### Linux

```bash
sudo apt install golang-go
```

---

## Configuration

### Proxy (Chine)

```bash
go env -w GOPROXY=https://goproxy.cn,direct
```

### Vérifier

```bash
go version
```
