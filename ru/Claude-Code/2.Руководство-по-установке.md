# Claude Code Руководство по установке

Подробные инструкции по установке Claude Code.

## Установка на Windows

### Способ один: Расширение VS Code

1. Откройте VS Code
2. Найдите расширение "Claude Code"
3. Нажмите "Установить"
4. Войдите в аккаунт Anthropic

### Способ два: Автономный клиент

1. Посетите https://claude.com/code
2. Скачайте клиент для Windows
3. Запустите установщик
4. Войдите

## Установка на macOS

### Способ один: Расширение VS Code

1. Откройте VS Code
2. Найдите расширение "Claude Code"
3. Нажмите "Установить"
4. Войдите

### Способ два: Homebrew

```
brew install claude-code
```

### Способ три: Автономный клиент

1. Посетите https://claude.com/code
2. Скачайте клиент для macOS
3. Запустите установщик
4. Войдите

## Установка на Linux

### Способ один: Расширение VS Code

1. Откройте VS Code
2. Найдите расширение "Claude Code"
3. Нажмите "Установить"
4. Войдите

### Способ два: Менеджер пакетов

**Ubuntu/Debian**:
```
curl -fsSL https://downloads.anthropic.com/claude-code/install.sh | sh
```

**Arch Linux**:
```
yay -S claude-code
```

## Проверка установки

```
claude --version
```

## Настройка

### API-ключ

1. Посетите https://console.anthropic.com
2. Создайте API-ключ
3. Настройте в Claude Code

### Переменные окружения

```
export ANTHROPIC_API_KEY=ваш-api-ключ
```

## Удаление

### macOS

```
brew uninstall claude-code
```

### Linux

```
sudo rm -rf /usr/local/claude-code
```
