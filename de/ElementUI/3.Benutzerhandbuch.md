# Element UI Benutzerhandbuch

## Basiskomponenten

### Button

```html
<el-button>Standard</el-button>
<el-button type="primary">Primär</el-button>
<el-button type="success">Erfolg</el-button>
<el-button type="warning">Warnung</el-button>
<el-button type="danger">Gefahr</el-button>
```

### Layout

```html
<el-row>
  <el-col :span="12"><div></div></el-col>
  <el-col :span="12"><div></div></el-col>
</el-row>
```

## Formularkomponenten

### Input

```html
<el-input v-model="input" placeholder="Eingabe"></el-input>
```

### Select

```html
<el-select v-model="value" placeholder="Auswählen">
  <el-option label="Option 1" value="1"></el-option>
</el-select>
```

### Form

```html
<el-form :model="form" label-width="100px">
  <el-form-item label="Name">
    <el-input v-model="form.name"></el-input>
  </el-form-item>
</el-form>
```

## Datenkomponenten

### Table

```html
<el-table :data="tableData">
  <el-table-column prop="date" label="Datum"></el-table-column>
  <el-table-column prop="name" label="Name"></el-table-column>
</el-table>
```

### Pagination

```html
<el-pagination :total="1000" layout="prev, pager, next"></el-pagination>
```

## Benachrichtigungen

### Message

```javascript
this.$message({ message: 'Erfolg', type: 'success' })
```

### Alert

```html
<el-alert title="Erfolg" type="success"></el-alert>
```

## Navigation

### Menu

```html
<el-menu>
  <el-menu-item index="1">Nav 1</el-menu-item>
</el-menu>
```

### Tabs

```html
<el-tabs v-model="activeName">
  <el-tab-pane label="Tab 1" name="first">Inhalt</el-tab-pane>
</el-tabs>
```

## Andere

### Dialog

```html
<el-button @click="visible = true">Öffnen</el-button>
<el-dialog :visible.sync="visible" title="Titel">
  <span>Inhalt</span>
</el-dialog>
```
