# Element UI 使用手冊

> 元件使用

---

## 按鈕

```html
<el-button>預設</el-button>
<el-button type="primary">主要</el-button>
```

## 表單

```html
<el-form :model="form">
  <el-form-item>
    <el-input v-model="form.name"></el-input>
  </el-form-item>
</el-form>
```

## 表格

```html
<el-table :data="tableData">
  <el-table-column prop="name" label="名稱"></el-table-column>
</el-table>
```

## 對話框

```html
<el-dialog :visible.sync="dialogVisible" title="標題">
  <span>內容</span>
</el-dialog>
```
