# Руководство по установке

В этом документе описаны различные способы установки Bootstrap, которые помогут вам быстро начать использовать фреймворк в проекте.

## Требования к окружению

Для Bootstrap 5 нужно следующее окружение:

- **HTML5** — современный браузер
- **CSS3** — поддержка стилей
- **JavaScript** — ES6 и выше (если используются JavaScript-плагины)

## Способы установки

### Способ 1: Подключение через CDN (рекомендуется)

Самый простой вариант — подключить CSS и JavaScript файлы Bootstrap напрямую через CDN:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Страница на Bootstrap</title>
    <!-- Подключаем Bootstrap CSS -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>Привет, Bootstrap!</h1>

    <!-- Подключаем Bootstrap JavaScript (включает Popper) -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### Способ 2: Установка через npm

Если вы используете npm для управления зависимостями:

```bash
# Установка Bootstrap
npm install bootstrap@5

# Установка Popper (для выпадающих списков, подсказок и т.д.)
npm install @popperjs/core
```

Затем подключите в проекте:

```javascript
// Подключаем CSS Bootstrap
import 'bootstrap/dist/css/bootstrap.min.css';

// Подключаем JS Bootstrap
import 'bootstrap/dist/js/bootstrap.bundle.min.js';
```

### Способ 3: Скачивание исходников

Скачайте готовые файлы с официального сайта:

1. Перейдите на https://getbootstrap.com/
2. Нажмите кнопку "Download"
3. Выберите "Compiled CSS and JS" для скачивания
4. Распакуйте архив и скопируйте папки css и js в проект

Структура проекта:

```
your-project/
├── css/
│   └── bootstrap.min.css
├── js/
│   └── bootstrap.bundle.min.js
└── index.html
```

### Способ 4: Исходники на Sass

Если нужно настроить Bootstrap под себя, скачайте исходники на Sass:

```bash
npm install bootstrap
```

Затем подключите в Sass-файле:

```scss
// Свои переменные (до подключения Bootstrap)
$primary: #007bff;
$body-bg: #f8f9fa;

// Подключаем Bootstrap
@import "~bootstrap/scss/bootstrap";
```

## Инструменты разработки

### Расширения для VS Code

Рекомендуем установить следующие расширения:

- **Bootstrap 5 Snippets** — готовые сниппеты компонентов Bootstrap
- **HTML CSS Support** — автодополнение CSS-классов
- **Live Server** — live-предпросмотр страницы

### Онлайн-редакторы

- **Bootstrap Builder** — визуальный конструктор страниц на Bootstrap
- **Bootstrap Studio** — профессиональный инструмент для дизайна на Bootstrap

## Быстрый старт

### Создание первой страницы

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Быстрый старт с Bootstrap</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <!-- Навигационная панель -->
    <nav class="navbar navbar-expand-lg navbar-dark bg-primary">
        <div class="container">
            <a class="navbar-brand" href="#">Мой сайт</a>
        </div>
    </nav>

    <!-- Основной контент -->
    <main class="container mt-5">
        <div class="row">
            <div class="col-md-6">
                <h1>Добро пожаловать в Bootstrap</h1>
                <p class="lead">Это адаптивная страница, построенная на Bootstrap 5.</p>
                <button class="btn btn-primary">Узнать больше</button>
            </div>
            <div class="col-md-6">
                <div class="card">
                    <div class="card-body">
                        <h5 class="card-title">Возможности</h5>
                        <ul>
                            <li>Адаптивная вёрстка</li>
                            <li>Готовые компоненты</li>
                            <li>Мощные JavaScript-плагины</li>
                        </ul>
                    </div>
                </div>
            </div>
        </div>
    </main>

    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## Выбор версии

### Для продакшена

Используйте стабильную версию:

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### Для разработки

Используйте последнюю версию для разработки (включает исходный код):

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.js"></script>
```

### Китайское зеркало

В Китае можно использовать локальное зеркало для ускорения загрузки:

```html
<link href="https://cdn.bootcss.com/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.bootcss.com/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

## Удаление

Если нужно удалить (просто удалите связанные файлы):

- Удалите ссылки на CSS и JS из HTML
- Удалите папку bootstrap из node_modules (если использовался npm)
- Удалите локальные CSS и JS файлы
