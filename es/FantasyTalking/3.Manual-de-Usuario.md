# FantasyTalking - Manual de Usuario

Guía detallada de uso.

## Uso Básico

### Generar video

```bash
python inference.py --source_image face.jpg --audio speech.wav --output output.mp4
```

### Parámetros

| Parámetro | Descripción |
|-----------|-------------|
| --source_image | Imagen de entrada |
| --audio | Archivo de audio |
| --output | Ruta de salida |
| --face_enhancer | Mejorador de rostro |
| --fps | Cuadros por segundo |

## Python API

```python
from fantasytalking import FantasyTalking

ft = FantasyTalking()
ft.generate(
    source_image="face.jpg",
    audio="speech.wav",
    output="output.mp4"
)
```

## Procesamiento por lotes

```python
ft.batch_generate(image_list, audio_list, output_dir)
```

## API Web

```bash
python api.py --port 8080
```

## Requisitos de entrada

### Imagen

- Formato: JPG, PNG
- Resolución: 512x512+
- Rostro: Frontal, claro

### Audio

- Formato: WAV, MP3
- Duración: 1-60 segundos

## Mejores prácticas

1. Usa fotos claras y frontales
2. Prepara audio sin ruido
3. Usa mejorador de rostros
