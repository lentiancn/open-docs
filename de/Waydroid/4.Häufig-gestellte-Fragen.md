# Häufig gestellte Fragen

## Google Play-Zertifizierung

Waydroid verwendet ein angepasstes Android-Image basierend auf LineageOS, das standardmäßig keine Google Play-Dienste enthält. Wenn Sie eine Google Play-Zertifizierung benötigen, können Sie ein GAPPS-Image installieren.

## Bildschirmtastatur deaktivieren

In einigen Fällen möchten Sie möglicherweise die Bildschirmtastatur deaktivieren. Dies kann über Systemeinstellungen oder Waydroid-Konfiguration erreicht werden.

## Waydroid in einer VM betreiben

Um Waydroid in einer virtuellen Maschine zu betreiben, stellen Sie sicher, dass die VM verschachtelte Virtualisierung unterstützt und die Hardware-Beschleunigung korrekt konfiguriert ist.

## ADB mit Waydroid verwenden

Waydroid unterstützt ADB-Debugging. Standardmäßig ist der ADB-Sicherheitsmodus aktiviert und verbindet sich nicht automatisch. Verwenden Sie den folgenden Befehl zum Verbinden:

```bash
waydroid adb connect
```

Geben Sie dann die Autorisierung auf der Android-Seite an.

## Waydroid-only-Sitzungen einrichten

Sie können dedizierte Sitzungen konfigurieren, die nur Waydroid ausführen, geeignet für spezifische Anwendungsfälle.

## Gemeinsamen Ordner einrichten

Waydroid unterstützt die Einrichtung gemeinsamer Ordner für einen einfachen Dateiaustausch zwischen dem Linux-Host und dem Android-Container.

## Farbkorrektur in Mutter

Bei Verwendung des Mutter-Fenstermanagers können Farbanzeigeprobleme auftreten, die mit spezifischen Konfigurationen korrigiert werden können.

## Apps und Daten sichern und wiederherstellen

Waydroid unterstützt die Sicherung und Wiederherstellung von Anwendungen und Daten, um sicherzustellen, dass wichtige Daten nicht verloren gehen.

## Benutzerdefinierte Waydroid-Images verwenden

Um benutzerdefinierte Images zu verwenden:

1. Laden Sie benutzerdefinierte `system.img` und `vendor.img` herunter oder erstellen Sie sie
2. Kopieren Sie sie in das Verzeichnis `/etc/waydroid-extra/images/`
3. Reinitialisieren Sie: `sudo waydroid init -f`

## System-Apps installieren

System-Level-Anwendungen können installiert werden, erfordern jedoch spezielle Berechtigungen und Konfigurationsschritte.

## Hardware-Kompatibilität

Waydroid hat bestimmte Hardware-Anforderungen:
- Erfordert Unterstützung des Wayland-Display-Servers
- Erfordert Linux-Kernel-Unterstützung für Namespaces und Container-Funktionen
- Grafiktreiber müssen OpenGL ES 3.2 unterstützen

## Leistungsoptimierung

Für optimale Leistung:
- Stellen Sie sicher, dass Hardware-Beschleunigung aktiviert ist
- Verwenden Sie geeignete Grafiktreiber
- Führen Sie auf unterstützten CPUs aus (vermeiden Sie reine ARM-Emulation)
