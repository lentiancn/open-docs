# 소개

Yarn은 Facebook/Meta에서 개발한 **패키지 관리자**입니다.

---

## 특징

- **속도**: 병렬 다운로드
- **결정론**: yarn.lock이 일관된 설치 보장
- **보안**: 체크섬 검증

---

## 워크스페이스

워크스페이스를 사용하면 하나의 저장소에서 여러 패키지를 관리할 수 있습니다.

---

## 빠른 시작

```bash
corepack enable
corepack prepare yarn@stable --activate
yarn --version
```
