## 安装步骤

### Windows 安装

#### 方法一：MSI 安装包

1. 下载 Go 安装包：https://go.dev/dl/
2. 运行 MSI 安装包
3. 按照提示完成安装
4. 验证安装：`go version`

#### 方法二：Chocolatey 安装

```powershell
choco install golang
```

#### 方法三：手动安装

1. 下载 Windows ZIP 包
2. 解压到 `C:\Go`
3. 添加环境变量：`C:\Go\bin` 到 PATH
4. 验证安装：`go version`

### macOS 安装

#### 方法一：brew 安装（推荐）

```bash
brew install go
```

#### 方法二：PKG 安装包

1. 下载 macOS 安装包
2. 运行 PKG 文件
3. 按照提示完成安装

### Linux 安装

#### 方法一：apt 安装（Ubuntu/Debian）

```bash
sudo apt update
sudo apt install golang-go
```

#### 方法二：yum 安装（CentOS/RHEL）

```bash
sudo yum install golang
```

#### 方法三：手动安装

```bash
# 下载
wget https://go.dev/dl/go1.22.0.linux-amd64.tar.gz

# 解压
sudo tar -C /usr/local -xzf go1.22.0.linux-amd64.tar.gz

# 添加环境变量
export PATH=$PATH:/usr/local/go/bin

# 验证
go version
```

## 环境配置

### GOPATH 配置

```bash
# 设置 GOPATH
export GOPATH=$HOME/go

# 添加 bin 到 PATH
export PATH=$PATH:$GOPATH/bin
```

### Go Modules 配置

Go 1.11+ 默认启用 Modules，无需额外配置。

```bash
# 初始化模块
go mod init module_name

# 下载依赖
go mod tidy

# 构建项目
go build
```

## 常见问题

### Windows 下找不到 go 命令？

确保 `C:\Go\bin` 已添加到系统 PATH 环境变量。

### 代理配置

国内访问 Go 模块可能需要配置代理：

```bash
go env -w GOPROXY=https://goproxy.cn,direct
```
