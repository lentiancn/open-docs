# Jenkins - Aperçu

## Présentation

Jenkins est un serveur d'automatisation open source. Il aide à automatiser les processus de développement logiciel comme la construction, les tests et le déploiement.

## Caractéristiques

- Intégration et livraison continues
- Écosystème de plugins (1800+)
- Builds distribués
- Configuration web
