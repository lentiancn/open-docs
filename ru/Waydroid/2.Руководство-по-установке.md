### Fedora

Waydroid можно установить из официального репозитория пакетов:

```bash
sudo dnf install waydroid
```

После установки запустите Waydroid из меню приложений и выполните инициализацию, вставив эти URL-адреса в поля OTA:

OTA системы: `https://ota.waydro.id/system`

OTA поставщика: `https://ota.waydro.id/vendor`

#### Silverblue/Kinoite/...

Те же инструкции применяются к неизменяемым вариантам Fedora, но вместо `dnf` следует использовать `rpm-ostree`:

```bash
rpm-ostree install waydroid
```

### Ubuntu/Debian и производные

Убедитесь, что у вас включена сессия Wayland (Ubuntu 22.04+)

* Установите предварительные требования

```bash
sudo apt install curl ca-certificates -y
```

* Добавьте официальный репозиторий

```bash
curl -s https://repo.waydro.id | sudo bash
```

Если скрипт не определяет ваш дистрибутив, вы можете указать допустимый параметр, добавив `-s <DISTRO>`. В настоящее время поддерживаются следующие значения: **mantic**, **focal**, **jammy**, **kinetic**, **lunar**, **noble**, **plucky**, **questing**, **bookworm**, **bullseye**, **trixie**, **sid**

* Установите waydroid

```bash
sudo apt install waydroid -y
```

Затем запустите Waydroid из меню приложений.

## Устранение неполадок

### Запуск Waydroid вручную

Чтобы запустить Waydroid без systemctl, необходимо выполнить несколько простых шагов:

**Сначала запустите контейнер:**

```bash
sudo waydroid container start
```

**И в новой вкладке терминала запустите сессию waydroid (без ***sudo*****):**

```bash
waydroid session start
```

После запуска и появления сообщения "Android with user 0 is ready" можно безопасно запускать приложение из меню приложений.

### Запуск Waydroid в полноэкранном режиме

```bash
waydroid show-full-ui
```

### Запуск Waydroid в многоконечном режиме

Сначала нам нужно установить свойство во время работы сессии Waydroid:

```bash
waydroid prop set persist.waydroid.multi_windows true
```

После этого мы можем перезапустить сессию:

```bash
waydroid session stop
```

Теперь мы готовы запустить приложение, и оно запустится в многоконечном режиме.

### Переустановка Waydroid

Если вам нужно полностью переустановить:

```bash
waydroid session stop
sudo waydroid container stop
sudo apt remove waydroid  # или соответствующая команда удаления дистрибутива
# После перезагрузки выполните очистку
sudo rm -rf /var/lib/waydroid /home/.waydroid ~/waydroid ~/.share/waydroid ~/.local/share/applications/*aydroid* ~/.local/share/waydroid
```
