[https://wiki.postmarketos.org/wiki/Waydroid](https://wiki.postmarketos.org/wiki/Waydroid)

### Fedora

Waydroid は公式パッケージリポジトリからインストールできます：

```bash
sudo dnf install waydroid
```

インストール後、アプリケーションメニューから Waydroid を起動し、OTA フィールドに以下の URL を貼り付けて初期化を進めてください：

システム OTA：`https://ota.waydro.id/system`

ベンダー OTA：`https://ota.waydro.id/vendor`

#### Silverblue/Kinoite/...

Fedora Immutable バリアントにも同じ手順が適用されますが、`dnf` の代わりに `rpm-ostree` を使用する必要があります：

```bash
rpm-ostree install waydroid
```

### Ubuntu/Debian および派生版

Wayland セッションが有効になっていることを確認してください（Ubuntu 22.04+）

* 前提条件のインストール

```bash
sudo apt install curl ca-certificates -y
```

* 公式リポジトリの追加

```bash
curl -s https://repo.waydro.id | sudo bash
```

スクリプトがディストリビューションを検出できない場合、`-s <DISTRO>` を追加して有効なオプションを指定できます。現在サポートされている値は次のとおりです：**mantic**、**focal**、**jammy**、**kinetic**、**lunar**、**noble**、**plucky**、**questing**、**bookworm**、**bullseye**、**trixie**、**sid**

* waydroid のインストール

```bash
sudo apt install waydroid -y
```

その後、アプリケーションメニューから Waydroid を起動します。

## トラブルシューティング

### Waydroid の手動起動

systemctl を使わずに Waydroid を起動するには、いくつかの簡単な手順に従う必要があります：

**まずコンテナを起動します：**

```bash
sudo waydroid container start
```

**新しいターミナルタブで waydroid セッションを起動します（***sudo*** なし）：**

```bash
waydroid session start
```

起動後、「Android with user 0 is ready」と表示されたら、アプリケーションメニューからアプリを起動できます。

### Waydroid のフルスクリーンモード起動

```bash
waydroid show-full-ui
```

### Waydroid のマルチウィンドウモード起動

まず Waydroid セッション実行中にプロパティを設定する必要があります：

```bash
waydroid prop set persist.waydroid.multi_windows true
```

その後、セッションを再起動します：

```bash
waydroid session stop
```

これでアプリを起動すると、マルチウィンドウモードで開始されます。

### Waydroid の再インストール

完全に再インストールする必要がある場合：

```bash
waydroid session stop
sudo waydroid container stop
sudo apt remove waydroid  # または対応するディストリビューションのアンインストールコマンド
# 再起動後、クリーンアップを実行
sudo rm -rf /var/lib/waydroid /home/.waydroid ~/waydroid ~/.share/waydroid ~/.local/share/applications/*aydroid* ~/.local/share/waydroid
```
