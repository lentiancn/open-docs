# Apifox 安装指南

## 系统要求

在安装 Apifox 之前，我们需要确认你的电脑满足以下基本要求：

Apifox 支持 Windows、macOS 和 Linux 三大操作系统。Windows 用户需要 Windows 10 或更高版本；macOS 用户需要 macOS 10.15 或更高版本；Linux 用户需要 Ubuntu 20.04 或更高版本，或其他主流 Linux 发行版。

硬件方面，建议电脑至少有 4GB 内存可用，硬盘空间至少保留 2GB 用于安装和缓存数据。显示器分辨率建议 1920x1080 或更高，以获得最佳的使用体验。

## 下载 Apifox

Apifox 提供多种下载方式，你可以根据自己的操作系统选择合适的版本。

### 官网下载

访问 Apifox 官网 https://apifox.com ，点击首页的「下载」按钮，进入下载页面。页面会自动识别你的操作系统，显示对应的下载选项。点击「立即下载」即可开始下载安装包。

如果你需要下载其他平台的版本，可以点击页面中的「其他平台」链接进行选择。官网提供的版本都是正式版，稳定性有保障。

### 版本选择

Apifox 提供两种版本：

**桌面客户端**：这是最常用的版本，提供了完整的图形界面功能。桌面客户端支持 Windows、macOS 和 Linux 系统，可以直接从官网下载安装包进行安装。

**Web 版**：如果你不想在本地安装软件，也可以使用 Apifox 的 Web 版。Web 版功能与桌面客户端基本一致，但需要保持浏览器标签页打开。Web 版适合临时使用或在公共电脑上工作。

### 版本更新

Apifox 会定期推送更新，桌面客户端会在启动时自动检查更新。你也可以手动点击「帮助」→「检查更新」来查看是否有新版本可用。建议保持版本更新，以获得最新功能和修复的 bug。

## Windows 系统安装

Windows 系统的安装非常简单，按照以下步骤操作即可：

首先，下载 Windows 版本的安装包（.exe 文件）。下载完成后，双击安装包启动安装程序。

安装向导会引导你完成安装过程。第一步是选择安装位置，默认安装在 C 盘的用户目录下，你可以点击「浏览」按钮选择其他位置。建议保持默认路径，除非你有特殊的安装需求。

接下来是创建快捷方式的选项，你可以选择是否在桌面创建快捷方式、是否在开始菜单创建文件夹。根据个人喜好勾选即可。

最后点击「安装」按钮，等待安装完成。安装过程通常只需要几分钟，安装完成后点击「完成」启动 Apifox。

如果你下载的是便携版（.zip 格式），只需解压到指定目录，然后双击「Apifox.exe」即可运行，无需安装过程。

## macOS 系统安装

macOS 系统的安装同样简单：

首先，下载 macOS 版本的安装包（.dmg 文件）。下载完成后，在 Finder 中找到下载的文件，双击打开。

打开后会看到一个「Apifox」的应用程序图标，直接拖拽到「Applications」文件夹即可完成安装。拖拽完成后，你可以在 Launchpad 或 Applications 文件夹中找到 Apifox。

首次打开 Apifox 时，系统可能会提示「无法打开应用程序，因为无法验证开发者」。这是 macOS 的安全机制导致的。解决方法是将 Apifox 从 Applications 文件夹中拖到「废纸篓」，然后重新从 Launchpad 中打开；或者在系统偏好设置中点击「隐私与安全性」，在「安全性」区域点击「仍要打开」。

安装完成后，你可以将 Apifox 固定到 Dock 栏，方便后续使用。

## Linux 系统安装

Linux 系统有多种安装方式，你可以根据自己的发行版和偏好选择：

### AppImage 方式（推荐）

AppImage 是一种通用的 Linux 打包格式，可以在大多数发行版上运行。下载 AppImage 文件后，需要添加执行权限：

```bash
chmod +x Apifox-x.x.x.AppImage
```

然后直接运行即可：

```bash
./Apifox-x.x.x.AppImage
```

你可以将 AppImage 文件移动到 /opt 目录下方便管理，并创建桌面快捷方式。

### Debian/Ubuntu 安装

如果你使用的是 Debian 或 Ubuntu 系统，可以下载 .deb 安装包，然后使用以下命令安装：

```bash
sudo dpkg -i apifox_x.x.x_amd64.deb
```

如果安装过程中出现依赖问题，运行以下命令修复：

```bash
sudo apt-get install -f
```

### Fedora/RHEL 安装

如果你使用的是 Fedora 或 RHEL 系统，可以下载 .rpm 安装包，然后使用以下命令安装：

```bash
sudo rpm -i apifox_x.x.x_amd64.rpm
```

## 首次启动与账号

安装完成后，双击桌面图标启动 Apifox。首次启动时会显示登录界面。

### 注册账号

如果你还没有 Apifox 账号，可以点击「注册账号」进行注册。注册方式支持邮箱注册和手机号注册。按照提示填写信息，完成验证后即可注册成功。

### 登录账号

已有账号的用户直接输入账号密码登录即可。Apifox 支持「记住密码」功能，方便后续快速登录。

### 跳过登录

如果你暂时不想注册或登录，可以点击「暂不登录，先试试」跳过这一步。跳过登录后，你可以正常使用 Apifox 的大部分功能，但无法使用云端同步和团队协作功能。建议后续还是登录账号，以获得完整的功能体验。

## 基本设置

首次登录后，建议进行一些基本设置，以获得更好的使用体验：

### 主题设置

Apifox 支持浅色和深色主题。你可以根据个人喜好在「设置」→「外观」中选择。深色主题适合长时间在暗光环境下工作，可以减少眼睛疲劳。

### 语言设置

Apifox 默认使用简体中文界面。你可以在「设置」→「通用」→「语言」中切换到其他语言，如英文、繁体中文、日文等。

### 快捷键设置

Apifox 提供了丰富的快捷键，可以大大提高操作效率。你可以在「设置」→「快捷键」中查看和自定义快捷键。建议先熟悉常用的快捷键，如发送请求（Ctrl+Enter）、新建接口（Ctrl+N）等。

### 代理设置

如果你的网络需要代理才能访问外网，需要在「设置」→「网络」中配置代理。填入代理服务器的地址和端口即可。

## 常见安装问题

### 下载速度慢

如果官网下载速度较慢，可以尝试使用国内镜像站点。或者如果你的网络环境允许，可以使用下载工具进行多线程下载。

### 安装后无法启动

如果安装后无法启动，可能是由于缺少运行库导致。Windows 用户可以尝试安装 Visual C++ 运行库；Linux 用户可以检查是否缺少必要的 GTK 库。

### 卸载问题

如果需要卸载 Apifox，Windows 用户可以在「设置」→「应用」→「应用和功能」中找到 Apifox 进行卸载；macOS 用户直接将 Apifox 从 Applications 中拖到废纸篓即可；Linux 用户使用系统自带的软件卸载工具或执行安装时使用的命令进行卸载。

卸载前建议先导出你的数据，以免丢失重要的接口信息。
