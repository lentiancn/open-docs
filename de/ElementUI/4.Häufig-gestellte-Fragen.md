# Element UI FAQ

Diese Seite listet häufige Probleme und Lösungen bei der Verwendung von Element UI auf.

## Installationsprobleme

### Installation fehlgeschlagen

Symptome: npm install schlägt fehl

Lösungen:

1. Überprüfen Sie die Node.js-Version, sie muss 8.0.0 oder höher sein
2. Leeren Sie den npm-Cache:
   npm cache clean --force
3. Löschen Sie node_modules und package-lock.json, dann installieren Sie erneut

### Import bei Bedarf funktioniert nicht

Symptome: Paketgröße bleibt groß

Lösungen:

1. Stellen Sie sicher, dass die .babelrc-Konfiguration korrekt ist
2. Überprüfen Sie, ob babel-plugin-component korrekt installiert ist

## Nutzungsprobleme

### Komponentenstile werden nicht angewendet

Symptome: Komponentenstile entsprechen nicht den Erwartungen

Lösungen:

1. Stellen Sie sicher, dass CSS korrekt importiert ist:
   import 'element-ui/lib/theme-chalk/index.css'

### Symbole werden nicht angezeigt

Symptome: Symbolkomponenten werden leer oder als Quadrate angezeigt

Lösungen:

1. Symbolschriftarten importieren:
   import 'element-ui/lib/theme-chalk/fonts/element-icons.ttf'

### Dialogprobleme

#### Dialog-Scroll-Probleme

Wenn der Dialoginhalt zu lang ist, stellen Sie sicher, dass Scrollen hinzugefügt wird:

<el-dialog>
  <div style="height: 400px; overflow-y: scroll;">
    Inhalt
  </div>
</el-dialog>

## Vue-Versionsprobleme

### Nicht kompatibel mit Vue 3

Element UI unterstützt nur Vue 2. Wenn Sie Vue 3 benötigen, verwenden Sie Element Plus.

### Probleme mit babel6

Wenn Sie babel6 verwenden, stellen Sie sicher, dass die Konfiguration korrekt ist:

{
  "presets": ["es2015", "stage-2"],
  "plugins": ["transform-runtime"]
}

## Stilprobleme

### Stilkonflikte

Symptome: Benutzerdefinierte Stile werden überschrieben

Lösungen:

1. Verwenden Sie spezifischere Selektoren
2. Verwenden Sie ::deep()

### Theme-Farbe wird nicht angewendet

Überprüfen Sie, ob SCSS-Variablen korrekt überschrieben werden:

$--color-primary: #409ungsproblemeeff;

## Leist

### Seitenverzögerung

Bei großen Datenmengen:

1. Verwenden Sie Paginierung zum Laden von Daten
2. Vermeiden Sie komplexe Komponenten in Tabellen

### Paketgröße zu groß

1. Verwenden Sie Import bei Bedarf
2. Entfernen Sie nicht verwendete Komponenten
