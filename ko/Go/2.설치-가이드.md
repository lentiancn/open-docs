## 설치 방법

### Windows 설치

#### MSI 인스톨러 사용

1. Go 다운로드: https://go.dev/dl/
2. MSI 인스톨러 실행
3. 설치 마법사 따르기
4. 명령 프롬프트 열고 `go version` 입력하여 확인

#### Chocolatey 사용

```powershell
choco install golang
```

### macOS 설치

#### Homebrew 사용

```bash
brew install go
```

#### DMG 패키지 사용

1. macOS 패키지 다운로드: https://go.dev/dl/
2. 패키지 파일 열고 실행
3. Go 디렉토리를 /usr/local/go로 이동: `sudo mv go /usr/local/`

### Linux 설치

#### 패키지 매니저 사용

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install golang-go

# CentOS/RHEL
sudo yum install golang
```

#### 수동 설치

```bash
# Go 바이너리 다운로드
wget https://go.dev/dl/go1.22.0.linux-amd64.tar.gz

# /usr/local에 추출
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go1.22.0.linux-amd64.tar.gz

# PATH에 추가
export PATH=$PATH:/usr/local/go/bin

# 설치 확인
go version
```

## 환경 설정

### GOPATH 설정

```bash
# Linux/macOS
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin

# Windows (PowerShell)
$env:GOPATH = "$env:USERPROFILE\go"
$env:PATH += ";$env:GOPATH\bin"
```

### Go 모듈 프록시 구성

```bash
# 중국 미러 사용
go env -w GOPROXY=https://goproxy.cn,direct

# 프록시 설정
go env -w GOPROXY=https://proxy.golang.org,direct
```

### 설치 확인

```bash
# 버전 확인
go version

# 환경 정보 보기
go env

# 테스트 실행
go version
```

## 일반적인 명령

```bash
# 모듈 초기화
go mod init <모듈이름>

# 의존성 다운로드
go mod tidy

# 프로젝트 빌드
go build

# 프로그램 실행
go run main.go

# 테스트 실행
go test ./...

# 도구 설치
go install
```

## 문제 해결

### 설치 실패?

- 시스템 호환성 확인
- 다운로드한 버전이 아키텍처와 일치하는지 확인
- 권한 확인

### 명령을 찾을 수 없음?

- Go가 PATH 환경 변수에 추가되었는지 확인
- 터미널 다시 열기
- 설치 경로 확인

### 의존성 다운로드 느림?

- 중국 미러 프록시 구성
- VPN 사용
