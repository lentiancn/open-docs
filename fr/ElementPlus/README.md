# Element Plus

Bibliothèque de composants UI pour Vue 3.

## Aperçu

Element Plus est une bibliothèque de composants Vue 3 par Ele.me, plus de 70 composants de haute qualité.

### Fonctionnalités

- **Vue 3**: API Composition
- **TypeScript**: Support complet des types
- **Tree Shaking**: Optimisation
- **Thèmes**: Variables CSS

### Statistiques

- GitHub Stars: 20 000+

## Démarrage rapide

### Installation

```bash
npm install element-plus
```

### Utilisation

```vue
<el-button type="primary">Bouton</el-button>
```
