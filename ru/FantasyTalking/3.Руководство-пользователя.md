# FantasyTalking Руководство пользователя

## Синтез речи

```javascript
const audio = await client.speak('Текст для синтеза');
```

## Потоковая передача

```javascript
const stream = await client.speakStream('Длинный текст');
stream.on('data', chunk => player.play(chunk));
```

## Рекомендации

- Повторное использование подключений
- Пакетная обработка
- Кэширование результатов
