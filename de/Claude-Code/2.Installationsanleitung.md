# Claude Code Installationsanleitung

Diese Anleitung bietet detaillierte Anweisungen zur Installation, Konfiguration und Aktualisierung von Claude Code auf verschiedenen Plattformen.

## Systemanforderungen

Claude Code unterstützt die folgenden Betriebssysteme und Konfigurationen:

| Komponente | Anforderungen |
|------------|---------------|
| Betriebssysteme | macOS 13.0+, Windows 10 1809+ oder Windows Server 2019+, Ubuntu 20.04+, Debian 10+, Alpine Linux 3.19+ |
| Speicher | 4 GB oder mehr |
| Netzwerk | Internetverbindung erforderlich |
| Shell | Bash, Zsh, PowerShell oder CMD |
| Region | Von Anthropic unterstützte Länder und Regionen |

### Zusätzliche Abhängigkeiten

- **ripgrep**: In der Regel in Claude Code enthalten; wenn die Suche fehlschlägt, lesen Sie den Abschnitt zur Fehlerbehebung.

## Installationsmethoden

### Methode Eins: Native Installation (Empfohlen)

**macOS, Linux, WSL:**

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell:**

```powershell
irm https://claude.ai/install.ps1 | iex
```

**Windows CMD:**

```batch
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
```

**Hinweis: Windows erfordert [Git for Windows](https://git-scm.com/downloads/win).**

Native Installationen werden automatisch im Hintergrund aktualisiert.

### Methode Zwei: Homebrew (macOS/Linux)

```bash
brew install --cask claude-code
```

Homebrew-Installationen werden nicht automatisch aktualisiert. Führen Sie regelmäßig `brew upgrade claude-code` aus.

### Methode Drei: WinGet (Windows)

```powershell
winget install Anthropic.ClaudeCode
```

WinGet-Installationen werden nicht automatisch aktualisiert. Führen Sie regelmäßig `winget upgrade Anthropic.ClaudeCode` aus.

## Installation überprüfen

Nach der Installation bestätigen Sie, dass Claude Code funktioniert:

```bash
claude --version
```

Für eine detailliertere Überprüfung führen Sie aus:

```bash
claude doctor
```

## Konto-Authentifizierung

Claude Code erfordert Pro, Max, Teams, Enterprise oder Console-Konten. Der kostenlose Claude.ai-Plan beinhaltet keinen Claude Code-Zugang.

Führen Sie nach der Installation den Befehl `claude` aus und folgen Sie den Browser-Anweisungen zur Anmeldung.

## Claude Code aktualisieren

### Automatische Updates

Native Installationen prüfen beim Start und periodisch während des Betriebs auf Updates. Updates werden im Hintergrund heruntergeladen und installiert und beim nächsten Start von Claude Code wirksam.

Sie können das Update-Timing durch Konfiguration des Release-Kanals steuern:
- `latest` (Standard): Neue Funktionen sofort erhalten
- `stable`: Verwenden Sie Versionen, die etwa eine Woche alt sind

Konfigurieren Sie in Ihrer Einstellungsdatei:

```json
{
  "autoUpdatesChannel": "stable"
}
```

### Automatische Updates deaktivieren

Setzen Sie die Umgebungsvariable in Ihrer Einstellungsdatei:

```json
{
  "env": {
    "DISABLE_AUTOUPDATER": "1"
  }
}
```

### Manuelle Updates

Wenden Sie ein Update sofort an:

```bash
claude update
```

Homebrew und WinGet erfordern manuelle Updates:
- Homebrew: `brew upgrade claude-code`
- WinGet: `winget upgrade Anthropic.ClaudeCode`

## Erweiterte Installationsoptionen

### Bestimmte Version installieren

Neueste Version installieren:
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

Stabile Version installieren:
```bash
curl -fsSL https://claude.ai/install.sh | bash -s stable
```

Bestimmte Versionsnummer installieren (z.B. 1.0.58):
```bash
curl -fsSL https://claude.ai/install.sh | bash -s 1.0.58
```

### Von npm auf native Installation migrieren

Wenn Sie Claude Code zuvor mit npm installiert haben, migrieren Sie zur nativen Installation:

```bash
# Native Binary installieren
curl -fsSL https://claude.ai/install.sh | bash

# Alte npm-Installation entfernen
npm uninstall -g @anthropic-ai/claude-code
```

## Claude Code deinstallieren

### Native Deinstallation

**macOS, Linux, WSL:**

```bash
rm -f ~/.local/bin/claude
rm -rf ~/.local/share/claude
```

**Windows PowerShell:**

```powershell
Remove-Item -Path "$env:USERPROFILE\.local\bin\claude.exe" -Force
Remove-Item -Path "$env:USERPROFILE\.local\share\claude" -Recurse -Force
```

### Homebrew-Deinstallation

```bash
brew uninstall --cask claude-code
```

### WinGet-Deinstallation

```powershell
winget uninstall Anthropic.ClaudeCode
```

### npm-Deinstallation

```bash
npm uninstall -g @anthropic-ai/claude-code
```

### Konfigurationsdateien entfernen

**Warnung: Das Entfernen von Konfigurationsdateien löscht alle Einstellungen, erlaubte Tools, MCP-Serverkonfigurationen und Sitzungsverlauf.**

**macOS, Linux, WSL:**

```bash
# Benutzereinstellungen entfernen
rm -rf ~/.claude
rm ~/.claude.json

# Projektspezifische Einstellungen entfernen
rm -rf .claude
rm -f .mcp.json
```

## Windows-Besonderheiten

Claude Code unter Windows erfordert Git for Windows oder WSL. Sie können `claude` von PowerShell, CMD oder Git Bash starten.

Konfigurieren Sie den Pfad in Ihrer Einstellungsdatei:

```json
{
  "env": {
    "CLAUDE_CODE_GIT_BASH_PATH": "C:\\Program Files\\Git\\bin\\bash.exe"
  }
}
```

## Alpine Linux-Konfiguration

Auf Alpine und anderen musl/uClibc-Distributionen erfordert der native Installer `libgcc`, `libstdc++` und `ripgrep`. Installieren Sie diese Abhängigkeiten und setzen Sie `USE_BUILTIN_RIPGREP=0`.

```json
{
  "env": {
    "USE_BUILTIN_RIPGREP": "0"
  }
}
```
