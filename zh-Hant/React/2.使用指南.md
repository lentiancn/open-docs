# React 使用指南

---

## 組件

```jsx
function Welcome() {
  return <h1>Hello World</h1>;
}
```

---

## 狀態 (useState)

```jsx
import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);
  return <button onClick={() => setCount(count + 1)}>{count}</button>;
}
```

---

## 副作用 (useEffect)

```jsx
useEffect(() => {
  // 副作用代碼
}, []);
```
