# Kotlin 使用手册

## 变量与基本类型

Kotlin 使用 var 和 val 两个关键字来声明变量。var 声明的是可变变量，类似于 Java 中的普通变量；val 声明的是不可变变量，类似于 Java 中的 final 变量。在 Kotlin 中，优先使用 val 是最佳实践，这可以让代码更加安全和易于理解。

声明变量时，可以显式指定类型，也可以让编译器自动推断类型：

val name: String = "Kotlin"
val age = 10 // 编译器自动推断为 Int 类型
var count = 0 // 可变变量

Kotlin 的基本类型包括：整型（Int、Long、Short、Byte）、浮点型（Double、Float）、字符型（Char）、布尔型（Boolean）和字符串（String）。与 Java 不同，Kotlin 中所有基本类型都是对象，它们都有对应的成员方法。

Kotlin 的整型默认是 Int 类型，如果需要表示更大的数值，可以使用 Long 后缀：

val longNumber = 100L // Long 类型

Kotlin 的字符串非常强大，支持字符串模板功能，可以在字符串中直接嵌入变量或表达式：

val name = "World"
println("Hello, $name!") // 输出 Hello, World!
println("Length: ${name.length}") // 输出 Length: 5

## 集合

Kotlin 标准库提供了丰富的集合类型，包括 List（列表）、Set（集合）和 Map（映射）。每种集合都有可变和不可变两个版本。

不可变集合创建后不能添加、删除或修改元素：

val list = listOf(1, 2, 3, 4, 5)
val set = setOf("a", "b", "c")
val map = mapOf("key1" to 1, "key2" to 2)

可变集合允许修改元素：

val mutableList = mutableListOf(1, 2, 3)
mutableList.add(4)
mutableList.removeAt(0)

Kotlin 的集合提供了丰富的函数式操作，如 map、filter、reduce 等：

val numbers = listOf(1, 2, 3, 4, 5)
val doubled = numbers.map { it * 2 } // [2, 4, 6, 8, 10]
val evens = numbers.filter { it % 2 == 0 } // [2, 4]
val sum = numbers.reduce { acc, i -> acc + i } // 15

## 控制流

Kotlin 的控制流包括条件表达式、循环和跳转表达式。

if 表达式在 Kotlin 中有返回值，可以直接赋值给变量：

val max = if (a > b) a else b

when 表达式是 Kotlin 强大的多分支匹配工具，可以匹配各种条件：

when (x) {
    1 -> println("x is 1")
    2 -> println("x is 2")
    in 3..10 -> println("x is between 3 and 10")
    else -> println("x is something else")
}

Kotlin 的循环包括 for 循环和 while 循环。for 循环可以遍历范围、数组和集合：

for (i in 1..10) {
    println(i)
}

for (item in list) {
    println(item)
}

for (i in list.indices) {
    println("Index: $i, Value: ${list[i]}")
}

## 函数

Kotlin 使用 fun 关键字声明函数。函数可以指定参数类型和返回值类型：

fun greet(name: String): String {
    return "Hello, $name!"
}

如果函数体只有一个表达式，可以使用单表达式函数语法：

fun greet(name: String) = "Hello, $name!"

Kotlin 支持默认参数和命名参数，这大大减少了重载函数的需求：

fun greet(name: String, greeting: String = "Hello") = "$greeting, $name!"

greet("World") // Hello, World!
greet("World", "Hi") // Hi, World!
greet(name = "World", greeting = "Hi") // Hi, World!

Kotlin 还支持可变参数，使用 vararg 关键字：

fun sum(vararg numbers: Int): Int {
    return numbers.sum()
}

sum(1, 2, 3, 4, 5) // 15

## 类与对象

Kotlin 的类声明比 Java 更加简洁。默认情况下，Kotlin 的类是 final 的，不能被继承。如果希望类可以被继承，需要使用 open 关键字：

open class Animal(val name: String) {
    open fun speak() = println("$name makes a sound")
}

class Dog(name: String) : Animal(name) {
    override fun speak() = println("$name barks")
}

Kotlin 的构造函数分为主构造函数和次构造函数。主构造函数直接在类名后面声明：

class Person(val name: String, val age: Int)

如果需要在构造函数中执行逻辑，可以使用 init 块：

class Person(name: String, age: Int) {
    val name: String
    val age: Int

    init {
        this.name = name
        this.age = age
        require(age >= 0) { "Age cannot be negative" }
    }
}

Kotlin 的数据类（data class）自动生成 equals、hashCode、toString 和 copy 方法，非常适合用于存储数据的类：

data class User(val name: String, val email: String)

val user1 = User("Alice", "alice@example.com")
val user2 = user1.copy(email = "newemail@example.com")

## 空安全

Kotlin 的空安全是其最重要的特性之一。在 Kotlin 中，类型系统区分可空类型和非可空类型，这从根本上减少了空指针异常。

默认情况下，类型是不可空的：

val name: String = "Kotlin" // 不能为 null
val name: String = null // 编译错误

如果要允许变量为 null，需要使用可空类型（?）：

val name: String? = null // 可以为 null

访问可空类型的属性或方法时，需要使用安全调用操作符（?.）或非空断言（!!）：

val length = name?.length // 如果 name 为 null，返回 null
val length = name!!.length // 如果 name 为 null，抛出异常

Elvis 操作符（?:）提供了一种简洁的默认值处理方式：

val length = name?.length ?: 0 // 如果 name 为 null，返回 0

## 扩展函数与属性

Kotlin 允许为现有类添加新的函数和属性，而不需要继承或修改原类。这就是扩展函数和扩展属性。

扩展函数的声明方式是在函数名前加上要扩展的类名：

fun String.addExclamation() = this + "!"

"Hello".addExclamation() // "Hello!"

扩展属性同样可以添加新的属性到现有类：

val String.wordCount: Int
    get() = this.split(" ").size

"Hello World".wordCount // 2

## 高阶函数与 Lambda

高阶函数是指接受函数作为参数或返回函数的函数。Lambda 表达式是实现这一特性的主要方式。

Lambda 表达式的语法使用大括号：

val sum = { a: Int, b: Int -> a + b }
sum(1, 2) // 3

如果 Lambda 表达式是函数的最后一个参数，可以将 Lambda 移到括号外：

list.filter { it > 0 }

Lambda 表达式中的 it 代表单个参数的隐式名称，当参数只有一个时可以省略参数声明。

## 协程

协程是 Kotlin 异步编程的核心特性。相比线程，协程更加轻量级，可以在单个线程中运行多个协程。

launch 是启动协程的常用方式，它会在后台启动一个新的协程：

GlobalScope.launch {
    val data = async { fetchData() }
    updateUI(data.await())
}

suspend 函数是只能在协程中调用的函数：

suspend fun fetchData(): Data {
    return withContext(Dispatchers.IO) {
        // 执行耗时操作
    }
}

Kotlin 协程提供了多种调度器来控制协程运行所在的线程：Dispatchers.Main 用于 UI 操作，Dispatchers.IO 用于 I/O 操作，Dispatchers.Default 用于 CPU 密集型任务。

## 密封类与枚举

密封类是一种受限的类层次结构，限制了可以继承它的子类数量。所有子类必须在同一个文件中声明。这在表示有限状态或结果类型时非常有用：

sealed class Result<out T> {
    data class Success<out T>(val data: T) : Result<T>()
    data class Error(val message: String) : Result<Nothing>()
    object Loading : Result<Nothing>()
}

when 表达式可以充分利用密封类的特性，编译器会检查是否处理了所有可能的情况：

fun handleResult(result: Result<Any>) = when (result) {
    is Result.Success -> println(result.data)
    is Result.Error -> println(result.message)
    is Result.Loading -> println("Loading...")
}

枚举类用于表示有限数量的常量值：

enum class Direction {
    NORTH, SOUTH, EAST, WEST
}

枚举类可以有属性和方法：

enum class Color(val rgb: Int) {
    RED(0xFF0000),
    GREEN(0x00FF00),
    BLUE(0x0000FF);

    fun containsRed() = (rgb and 0xFF0000) != 0
}

## 接口与抽象类

接口在 Kotlin 中可以包含默认实现，这使得接口更加灵活：

interface Clickable {
    fun click()
    fun showOff() = println("I'm clickable!")
}

类可以实现多个接口：

class Button : Clickable, Focusable {
    override fun click() = println("Button clicked")
    override fun showOff() {
        super<Clickable>.showOff()
        println("I'm focusable too!")
    }
}

## 对象表达式与对象声明

对象表达式用于创建一个匿名类的实例，类似于 Java 的匿名内部类：

window.addMouseListener(object : MouseAdapter() {
    override fun mouseClicked(e: MouseEvent) {
        // 处理点击事件
    }
})

对象声明用于创建单例：

object Config {
    val apiUrl = "https://api.example.com"
    val timeout = 30
}

## 伴生对象

每个类可以包含一个伴生对象，类似于 Java 中的静态成员：

class MyClass {
    companion object {
        const val TAG = "MyClass"
        fun create() = MyClass()
    }
}

伴生对象的成员可以通过类名直接访问：

MyClass.TAG
MyClass.create()

## 类型检查与转换

is 运算符用于类型检查，类似于 Java 的 instanceof：

if (obj is String) {
    // 在这个分支中，obj 被自动转换为 String
    println(obj.length)
}

智能转换会自动转换已经检查过的类型。如果编译器能够推断出变量在特定分支中的类型，就会自动进行转换。

如果智能转换不适用，可以使用显式转换：

val str = obj as String

安全的类型转换使用 as?，当转换失败时返回 null：

val str = obj as? String // 如果转换失败，返回 null
