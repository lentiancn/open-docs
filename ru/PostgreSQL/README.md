# PostgreSQL

PostgreSQL — мощная система управления объектно-реляционными базами данных с открытым исходным кодом.

## Введение

PostgreSQL — это продвинутая, многофункциональная система управления объектно-реляционными базами данных с открытым исходным кодом, известная своей надежностью, широким набором функций и производительностью.

## Основные функции

- **ACID-совместимость**: Полная поддержка транзакций
- **Многоверсионное управление параллелизмом**: MVCC
- **Триггеры и хранимые процедуры**: Поддержка нескольких языков программирования
- **Ограничения внешнего ключа**: Полная ссылочная целостность
- **Представления**: Поддержка виртуальных таблиц
- **Поддержка JSON**: Функции NoSQL
- **Полнотекстовый поиск**: Встроенная поисковая система
- **Репликация**: Поддержка мастер-слейв репликации
- **Таблицы с разделами**: Разделение больших таблиц

## Быстрый старт

### Установка

```bash
# Ubuntu
sudo apt install postgresql-16

# macOS
brew install postgresql@16

# Docker
docker run -d -e POSTGRES_PASSWORD=secret postgres:16
```

### Подключение

```bash
psql -U postgres
```
