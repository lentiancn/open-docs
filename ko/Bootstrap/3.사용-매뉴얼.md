# 사용 매뉴얼

이 매뉴얼에서는 Bootstrap 5의 핵심 기능과 사용 방법을 자세히介绍합니다. 그리드 시스템, 컴포넌트, CSS 유틸리티 클래스 등을 포함합니다.

## 그리드 시스템

그리드 시스템은 Bootstrap의 핵심 기능으로, 12열 레이아웃을 제공합니다.

### 기본 사용법

```html
<div class="container">
    <div class="row">
        <div class="col">열 1</div>
        <div class="col">열 2</div>
        <div class="col">열 3</div>
    </div>
</div>
```

### 반응형 중단점

| 클래스명 | 화면 너비 | 설명 |
|------|---------|------|
| `.col-xs` | < 576px | 초소형 화면 |
| `.col-sm` | ≥ 576px | 소형 화면 |
| `.col-md` | ≥ 768px | 중형 화면 |
| `.col-lg` | ≥ 992px | 대형 화면 |
| `.col-xl` | ≥ 1200px | 초대형 화면 |
| `.col-xxl` | ≥ 1400px |特大형 화면 |

### 열 오프셋

```html
<div class="row">
    <div class="col-md-4">.col-md-4</div>
    <div class="col-md-4 offset-md-4">.col-md-4 .offset-md-4</div>
</div>
```

### 반응형 열

```html
<div class="row">
    <div class="col-12 col-md-6 col-lg-4">
       手机上는 전체 너비, 태블릿은 절반 너비, 큰 화면은 1/3 너비
    </div>
</div>
```

## 유틸리티 클래스

```html
<!-- 마진 -->
<div class="m-1">m-1부터 m-5까지</div>
<div class="mx-auto">수평 중앙 정렬</div>

<!-- 패딩 -->
<div class="p-1">p-1부터 p-5까지</div>
```

### 디스플레이

```html
<div class="d-none">숨기기</div>
<div class="d-block">보이기</div>
<div class="d-flex">플랙스박스</div>
```

### 텍스트

```html

<p class="fw-bold">굵은 글씨</p>
```
