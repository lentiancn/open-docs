# Angular

> Веб-фреймворк от Google

---

## Описание

Angular — это фреймворк на TypeScript, разработанный Google. Полное решение для современных веб-приложений.

## Версия

| Angular 17 | Ноя 2023 | Текущая |
| Angular 16 | Май 2023 | Стабильная |
