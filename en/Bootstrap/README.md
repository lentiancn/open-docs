# Bootstrap

The world's most popular front-end responsive CSS framework.

## Overview

Bootstrap is the most popular front-end CSS framework developed by Twitter for building responsive, mobile-first websites. Features responsive grid system, pre-built components, and CSS utility classes.

### Key Features

- **Responsive grid system** - 12-column responsive grid
- **Pre-built components** - Navbars, cards, modals, and more
- **CSS utility classes** - Quick style adjustments
- **JavaScript components** - Interactive components

## Quick Start

### CDN

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### npm

```bash
npm install bootstrap
```
