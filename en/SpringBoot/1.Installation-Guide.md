# Spring Boot Installation Guide

This guide covers how to install and set up Spring Boot development environment on Windows, macOS, and Linux.

## System Requirements

### Java Version Requirements

| Spring Boot Version | Minimum Java Version |
|---------------------|---------------------|
| 3.x                 | Java 17+            |
| 2.7.x               | Java 8+             |
| 2.5.x               | Java 8+             |

**Note:** Spring Boot 3.x requires Java 17 or later. It is recommended to use Java 21 (LTS) for best compatibility.

## Install Java

### Windows

**Option 1: Install from Oracle/OpenJDK**
1. Download JDK from [Oracle](https://www.oracle.com/java/technologies/downloads/) or [Adoptium](https://adoptium.net/)
2. Run the installer
3. Set environment variables:
   ```
   JAVA_HOME=C:\Program Files\Java\jdk-21
   PATH=%JAVA_HOME%\bin;%PATH%
   ```

**Option 2: Using Chocolatey**
```powershell
choco install openjdk21
```

**Option 3: Using Winget**
```powershell
winget install Oracle.JDK.21
```

Verify:
```cmd
java -version
```

### macOS

**Option 1: Using Homebrew**
```bash
brew install openjdk@21
sudo ln -sfn $(brew --prefix)/opt/openjdk@21/libexec/openjdk.jdk /Library/Java/JavaVirtualMachines/openjdk-21.jdk
echo 'export JAVA_HOME=$(/usr/libexec/java_home -v 21)' >> ~/.zshrc
source ~/.zshrc
```

**Option 2: Using SDKMAN**
```bash
curl -s "https://get.sdkman.io" | bash
sdk install java 21.0.2-tem
```

Verify:
```bash
java -version
```

### Linux

**Ubuntu/Debian:**
```bash
sudo apt update
sudo apt install openjdk-21-jdk
export JAVA_HOME=/usr/lib/jvm/java-21-openjdk-amd64
export PATH=$JAVA_HOME/bin:$PATH
```

**RHEL/CentOS/Fedora:**
```bash
sudo dnf install java-21-openjdk-devel
export JAVA_HOME=/usr/lib/jvm/java-21-openjdk
export PATH=$JAVA_HOME/bin:$PATH
```

**Using SDKMAN:**
```bash
curl -s "https://get.sdkman.io" | bash
source ~/.sdkman/bin/sdkman-init.sh
sdk install java 21.0.2-tem
```

Verify:
```bash
java -version
```

## Install Maven

### Windows

**Option 1: Manual Installation**
1. Download from [Apache Maven](https://maven.apache.org/download.cgi)
2. Extract to `C:\apache-maven`
3. Set environment variables:
   ```
   MAVEN_HOME=C:\apache-maven
   PATH=%MAVEN_HOME%\bin;%PATH%
   ```

**Option 2: Using Chocolatey**
```powershell
choco install maven
```

Verify:
```cmd
mvn -version
```

### macOS

```bash
brew install maven
```

Verify:
```bash
mvn -version
```

### Linux

**Ubuntu/Debian:**
```bash
sudo apt install maven
```

**RHEL/CentOS:**
```bash
sudo dnf install maven
```

Verify:
```bash
mvn -version
```

### All Platforms

**Using SDKMAN:**

```bash

```

**Manual Download:**

## IDE Setup

### IntelliJ IDEA (Recommended)

1. Download from [JetBrains](https://www.jetbrains.com/idea/download/)
2. Install and open
3. Configure JDK: `File` → `Project Structure` → `Project SDK`
4. Install Spring Boot Assistant plugin (optional)

### VS Code

1. Install [VS Code](https://code.visualstudio.com/)
2. Install extensions:
   - Extension Pack for Java
   - Spring Boot Extension Pack
3. Configure Java: `Ctrl+Shift+P` → `Java: Configure Java Runtime`

### Eclipse

1. Download [Spring Tools 4 (STS)](https://spring.io/tools)
2. Extract and run
3. File → New → Spring Starter Project

## Quick Start

### Using Spring Initializr

1. Visit [start.spring.io](https://start.spring.io/)
2. Select options:
   - Project: Maven / Gradle
   - Language: Java / Kotlin / Groovy
   - Spring Boot Version: 3.2.x (latest)
   - Group: com.example
   - Artifact: myapp
   - Packaging: Jar / War
   - Java Version: 17 / 21
3. Add dependencies:
   - Spring Web
   - Spring Data JPA
   - H2 Database (dev)
   - Spring Boot DevTools
4. Click "Generate" to download

### Using Command Line

```bash
# Create Maven project
curl https://start.spring.io/starter.zip \
  -d type=maven-project \
  -d language=java \
  -d bootVersion=3.2.5 \
  -d baseDir=myapp \
  -d dependencies=web,jpa,h2,devtools \
  -o myapp.zip

unzip myapp.zip
cd myapp
./mvnw spring-boot:run
```

## Verify Installation

```bash
# Check Java
java -version

# Check Gradle (if installed)
gradle -version

# Check Spring Boot CLI

# Run demo project
curl https://start.spring.io/starter.zip -o demo.zip
unzip demo.zip -d demo

./mvnw spring-boot:run
```

## Docker Installation (Optional)

### Install Docker

**Windows/macOS:**
Download and install [Docker Desktop](https://www.docker.com/products/docker-desktop)

**Linux:**
```bash
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER
```

### Run Spring Boot in Docker

```bash
docker run -p 8080:8080 springio/gs-spring-boot-docker
```

## Common Issues

### Java Not Found
- Ensure JAVA_HOME is set correctly
- Restart terminal after setting environment variables
- Verify with `echo $JAVA_HOME` (Linux/macOS) or `echo %JAVA_HOME%` (Windows)

### Port 8080 Already in Use
```bash
# Windows
netstat -ano | findstr :8080
taskkill /PID <pid> /F

# Linux/macOS
lsof -i :8080
kill -9 <pid>
```

### Maven Build Failed
- Clear Maven cache: `mvn clean`
- Update dependencies: `mvn dependency:resolve`
- Use `./mvnw` (Maven Wrapper) instead of system Maven
