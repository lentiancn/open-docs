# Element Plus 使用ガイド

このガイドでは、Vue 3アプリケーションでElement Plusコンポーネントを使用する方法について説明します。

## クイックスタート

### 基本的なテンプレート

```vue
<template>
  <div>
    <el-button type="primary">プライマリボタン</el-button>
    <el-input v-model="input" placeholder="テキストを入力" />
    <el-date-picker v-model="date" type="date" placeholder="日付を選択" />
  </div>
</template>

<script setup>
import { ref } from 'vue'

const input = ref('')
const date = ref('')
</script>
```

### フォーム

```vue
<template>
  <el-form :model="form" :rules="rules" ref="formRef" label-width="120px">
    <el-form-item label="アクティビティ名" prop="name">
      <el-input v-model="form.name"></el-input>
    </el-form-item>
    
    <el-form-item label="アクティビティゾーン" prop="region">
      <el-select v-model="form.region" placeholder="選択">
        <el-option label="ゾーン1" value="shanghai"></el-option>
        <el-option label="ゾーン2" value="beijing"></el-option>
      </el-select>
    </el-form-item>
    
    <el-form-item label="アクティビティ時間">
      <el-date-picker
        v-model="form.date"
        type="datetimerange"
        range-separator="至"
        start-placeholder="開始日付"
        end-placeholder="終了日付"
      ></el-date-picker>
    </el-form-item>
    
    <el-form-item label="即時配送">
      <el-switch v-model="form.delivery"></el-switch>
    </el-form-item>
    
    <el-form-item label="リソース">
      <el-radio-group v-model="form.resource">
        <el-radio label="Sponsor">スポンサー</el-radio>
        <el-radio label="Venue">会場</el-radio>
      </el-radio-group>
    </el-form-item>
    
    <el-form-item label="説明">
      <el-input v-model="form.desc" type="textarea"></el-input>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="onSubmit">作成</el-button>
      <el-button @click="onReset">リセット</el-button>
    </el-form-item>
  </el-form>
</template>

<script setup>
import { reactive, ref } from 'vue'

const formRef = ref(null)

const form = reactive({
  name: '',
  region: '',
  date: [],
  delivery: false,
  resource: '',
  desc: ''
})

const rules = {
  name: [
    { required: true, message: 'アクティビティ名を入力してください', trigger: 'blur' },
    { min: 3, max: 5, message: '長さは3〜5文字', trigger: 'blur' }
  ],
  region: [
    { required: true, message: 'アクティビティゾーンを選択してください', trigger: 'change' }
  ]
}

const onSubmit = () => {
  formRef.value.validate((valid) => {
    if (valid) {
      console.log('送信!')
    }
  })
}

const onReset = () => {
  formRef.value.resetFields()
}
</script>
```

 // オブジェクトにはreactiveを使用

## よくある問題

### コンポーネントがレンダリングされない？

コンポーネントが正しくインポートまたはグローバル登録されていることを確認してください。

### スタイルが適用されない？

Element Plusスタイルがインポートされていることを確認してください：

```javascript
import 'element-plus/dist/index.css'
```

### アイコンが表示されない？

アイコンをグローバルに登録：

```javascript
import * as icons from '@element-plus/icons-vue'
for (const [key, icon] of Object.entries(icons)) {
    app.component(key, icon)
}
```
