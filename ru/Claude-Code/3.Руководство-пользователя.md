# Claude Code Руководство пользователя

Основное использование и техники.

## Основное использование

### Запуск

- VS Code: Используйте горячую клавишу Cmd/Ctrl + Shift + P
- CLI: Запустите `claude` напрямую в терминале

### Беседа

Опишите свои потребности напрямую в окне чата:
```
Напиши для меня функцию сортировки
Объясни этот код
Как исправить этот баг
```

## Генерация кода

### Генерировать новый код

Опишите нужный функционал:
```
Создай API входа пользователя
Реализуй запрос с пагинацией
Напиши юнит-тесты
```

### Изменить существующий код

1. Выделите код
2. Опишите потребности в изменении
3. Claude сгенерирует изменённый код

## Помощь в отладке

### Объяснить ошибки

Вставьте сообщение об ошибке напрямую:
```
Что означает эта ошибка?
```

### Предложения по исправлению

Попросите Claude проанализировать проблему:
```
Что не так с этой функцией?
```

## Обзор кода

### Запросы на обзор кода

```
Просмотри этот код
Есть проблемы с безопасностью?
Предложения по оптимизации производительности
```

## Горячие клавиши

### VS Code

| Горячая клавиша | Функция |
|----------------|----------|
| Cmd/Ctrl + L | Открыть чат |
| Cmd/Ctrl + Enter | Отправить ввод |

## Параметры настройки

### claude.json

Создайте `claude.json` в корне проекта:

```json
{
  "preamble": "Ты эксперт по Python",
  "maxTokens": 4000
}
```
