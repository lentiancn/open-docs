# OpenCode

AI-агент для программирования с открытым исходным кодом.

## Обзор

OpenCode — это AI-агент для программирования с открытым исходным кодом, доступный как терминальный интерфейс, десктопное приложение или расширение IDE. С функциями LSP, множественные сессии и поддержка 75+ LLM-провайдеров.

### Основные функции

- **LSP включен** - Автоматически загружает правильные LSP
- **Множественные сессии** - Запускайте несколько агентов параллельно
- **Любая модель** - 75+ LLM-провайдеров
- **Конфиденциальность прежде всего** - Код и данные не сохраняются

## Быстрый старт

### Установка

```bash
curl -fsSL https://opencode.ai/install | bash
```

### Настройка

```bash
opencode
/connect
```

Посетите opencode.ai/auth для получения API-ключа.

### Инициализация

```bash
/init
```

### Использование

```
Объясни этот код
Помоги добавить функцию входа
Исправь этот баг
```
