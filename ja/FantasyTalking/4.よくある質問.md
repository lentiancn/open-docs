# FantasyTalking よくある質問

## インストール

### インストールに失敗した場合は？

まず、環境が要件を満たしていることを確認してください。Node.jsバージョンが14.0以上であることを確認します。Pythonを使用する場合は、3.7以上であることを確認してください。

インストールが失敗する場合は、以下を試してください：npmキャッシュをクリア; 国内ミラーを使用; またはソースからインストールします。

## 使用

### 音声合成が遅い場合は？

速度は複数の要因に依存します。まずネットワーク接続を確認します。オーバーヘッドを減らすためにセルフホストを検討してください。

### 適切な声のスタイルを選択方法は？

FantasyTalkingはさまざまな声のスタイルを提供しています。カスタマーサービスにはneutralまたはprofessionalを使用してください。

### 英文音声合成を実現方法は？

デフォルト設定では英文を指定する必要があります：

```javascript
const audio = await client.speak('Hello, world!', {
  voice: { language: 'en-US', style: 'neutral' }
});
```

## アカウントと請求

### APIキーを取得方法は？

FantasyTalking公式Web sitioでアカウントを登録し、コンソールでAPIキーを作成します。

## デプロイ

### Dockerデプロイが失敗した場合は？

まずDockerが正しくインストールされてあり実行されていることを確認してください。

## その他

### サポートするプラットフォームは？

Web、モバイル、デスクトップ、サーバーをサポートしています。
