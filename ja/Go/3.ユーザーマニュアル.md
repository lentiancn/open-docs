## 関数

```go
func add(a, b int) int {
    return a + b
}

func divide(a, b int) (int, error) {
    if b == 0 {
        return 0, errors.New("ゼロ除算")
    }
    return a / b, nil
}
```

## 並行処理

### Goroutine

```go
func say(s string) {
    for i := 0; i < 3; i++ {
        fmt.Println(s)
    }
}

func main() {
    go say("Hello")
    say("World")
    time.Sleep(time.Second)
}
```

### Channel

```go
ch := make(chan string)
go func() { ch <- "message" }()
msg := <-ch
```
