# JIRA 使用指南

本指南介绍如何使用 JIRA 进行项目管理和问题跟踪。

## 快速开始

### 创建项目

1. 点击"创建项目"
2. 选择项目模板
3. 输入项目名称和密钥
4. 点击"创建"

## 项目类型

### 敏捷项目

```markdown
- Scrum 项目
- 看板项目
- 规模化敏捷
```

### 经典项目

```markdown
- 业务项目
- 项目管理
- 任务跟踪
```

## 问题类型

### 创建问题

```markdown
1. 点击"创建问题"
2. 选择项目
3. 选择问题类型
4. 填写标题和描述
5. 设置优先级和组件
6. 分配给用户
```

### 问题类型

- **Bug** - 软件缺陷
- **Story** - 用户故事
- **Task** - 任务
- **Epic** - 史诗
- **Sub-task** - 子任务

## 工作流

### 默认工作流

```
创建 → 分配 → 解决 → 关闭
```

### 敏捷工作流

```
待办 → 进行中 → 完成 → 审核
```

## 自动化

### 创建规则

1. 进入项目设置 → 自动化
2. 创建规则
3. 配置触发器
4. 设置操作

### 自动化示例

```
触发器: 问题创建
条件: 类型 = Bug
操作: 分配给 = 负责人
```
