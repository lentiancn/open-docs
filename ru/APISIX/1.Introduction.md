# APISIX Introduction

Cloud-Native API Gateway.

## What is APISIX?

Apache APISIX is a cloud-native API gateway under the Apache Software Foundation, developed and donated by API7.ai. It is a dynamic, real-time, high-performance API gateway that serves as the traffic entrance for all your business.

APISIX provides features including dynamic routing, dynamic upstream, dynamic certificates, A/B testing, canary release, blue-green deployment, rate limiting, defense against malicious attacks, metrics collection, monitoring alarms, observability, and service governance.

### Core Features

- **High Performance**: Built on Nginx + Lua, capable of 10k+ QPS per core
- **Dynamic Routing**: Dynamically update routing rules without restart
- **Plugin System**: Rich plugin ecosystem with hot-reloading support
- **Service Discovery**: Supports Consul, Eureka, Nacos, and other service discovery mechanisms
- **Load Balancing**: Round-robin, hash, consistent hash, and other strategies
- **Security**: Rate limiting, authentication, JWT, firewall
- **Visual Dashboard**: Intuitive management interface
- **Multi-Protocol Support**: HTTP, HTTPS, HTTP/2, Dubbo, gRPC, WebSocket
- **AI Gateway**: Supports LLM and AI workload traffic

### Technical Architecture

APISIX uses Nginx as the underlying HTTP reverse proxy server, combined with Lua scripts for dynamic orchestration. The architecture is divided into data plane and control plane:

- **Data Plane**: Handles actual request forwarding, zero dependency
- **Control Plane**: Configuration management through Admin API

## Use Cases

- **Unified API Management**: Centrally manage all internal APIs
- **Microservices Gateway**: Act as microservices entry point for authentication, rate limiting, routing
- **Service Mesh Ingress**: Integrate with Istio, Linkerd, and other service meshes
- **API Traffic Control**: Rate limiting, circuit breaking, degradation
- **Hybrid Cloud Architecture**: Unified management of multi-cloud and hybrid cloud APIs
- **API Analytics**: Real-time monitoring of API calls
- **AI Gateway**: Traffic entrance for LLM and AI applications
