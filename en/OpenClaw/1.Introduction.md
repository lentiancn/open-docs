# OpenClaw 🦞

OpenClaw is a tool you run on your own device that connects popular messaging apps to AI assistants. It acts as a bridge between apps like WhatsApp, Telegram, Discord, and iMessage on one side, and AI services on the other. This lets you interact with an AI through the chat apps you already use every day.

## Who is it for?

If you want your own AI assistant but don't want to trust your data with third-party services, OpenClaw is worth considering. It's particularly well-suited for developers and power users who prefer command-line tools and want fine-grained control.

## Key Features

- **Data independence**: Everything stays on your own hardware
- **Multiple platforms**: One process handles several chat apps at once
- **Built for coding assistants**: Supports tool calls, conversation history, memory, and multi-assistant routing
- **Open source**: MIT licensed, contributions welcome

## What you need

- Node.js 22 or higher
- An API key from your chosen AI provider
- A few minutes for initial setup

For the best experience, use the latest generation models available.

## Quick Start

```bash
# Install
npm install -g openclaw@latest

# Initial setup
openclaw onboard --install-daemon

# Run
openclaw gateway --port 18789
```

Once running, open your browser to http://127.0.0.1:18789/ to get started.

## How it works

OpenClaw runs a single Gateway process that handles message routing, AI requests, and channel connections. You maintain just this one process to have AI capabilities across all your messaging platforms simultaneously.

## Core Capabilities

- **Centralized management**: One system to rule all your chat channels
- **Plugin architecture**: Add more platforms through extension packages
- **Multiple assistants**: Run separate AI instances for different purposes
- **File handling**: Send and receive images, audio, and documents
- **Web interface**: Manage everything through a browser-based dashboard
- **Mobile integration**: Pair iOS or Android devices for remote control, camera access, and screenshots
