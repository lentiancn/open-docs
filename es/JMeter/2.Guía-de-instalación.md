# Guía de instalación de JMeter

Esta guía proporciona instrucciones detalladas para instalar y configurar Apache JMeter en diferentes sistemas operativos.

## Requisitos del entorno

- **Entorno de ejecución Java**: JMeter requiere Java 8 o superior
- **Sistema operativo**: Soporta cualquier sistema operativo compatible con Java, incluyendo Linux, Windows y Mac OSX
- **Memoria**: Se recomienda al menos 4GB de memoria disponible (puede ser más según la escala de las pruebas)
- **Espacio en disco**: Se recomienda al menos 1GB de espacio disponible

## Descargando JMeter

1. Visite la página oficial de descarga de JMeter: https://jmeter.apache.org/download_jmeter.cgi
2. Descargue la última versión binaria de JMeter (Binaries)
3. Se recomienda descargar también los archivos de firma PGP o SHA-512 para verificar la integridad de la descarga

## Instalación en Windows

### Extracción

1. Localice el archivo descargado de Apache JMeter (apache-jmeter-x.x.zip)
2. Elija un directorio de instalación apropiado (evite rutas con espacios y caracteres especiales)
3. Extraiga el archivo al directorio seleccionado
4. JMeter ya está instalado

### Verificación

1. Navegue a la carpeta bin en el directorio de instalación de JMeter
2. Haga doble clic en jmeter.bat (archivo por lotes de Windows)
3. Si tiene éxito, se lanzará la GUI de JMeter

## Instalación en Linux y Mac OSX

### Extracción

1. Localice el archivo descargado de Apache JMeter (apache-jmeter-x.x.tar.gz)
2. Extraiga usando el siguiente comando:
   ```bash
   tar -xf apache-jmeter-x.x.tar.gz
   ```
3. JMeter ya está instalado

### Variables de entorno (Opcional)

1. Abra el archivo de configuración de su terminal (~/.bashrc o ~/.profile)
2. Agregue las siguientes variables de entorno:
   ```bash
   export JMETER_HOME=/path/to/apache-jmeter-x.x
   export PATH=$PATH:$JMETER_HOME/bin:$JMETER_HOME/lib
   ```
3. Guarde y actualice la configuración:
   ```bash
   source ~/.bashrc
   ```

### Verificación

1. Ejecute lo siguiente en la terminal:
   ```bash
   jmeter --version
   ```
2. Si se muestra información de la versión, la instalación fue exitosa

## Iniciando JMeter

### Modo GUI

- **Windows**: Haga doble clic en jmeter.bat en el directorio bin
- **Linux/Mac**: Ejecute el comando jmeter en la terminal

### Modo de línea de comandos

Para pruebas de carga, se recomienda el modo de línea de comandos para reducir el uso de recursos:

```bash
jmeter -n -t test_plan.jmx -l results.jtl -e -o /path/to/output
```

Parámetros:
- -n: Ejecutar en modo no-GUI
- -t: Especificar archivo de plan de prueba
- -l: Especificar archivo de resultados
- -e: Generar informe después de que termine la prueba
- -o: Especificar directorio de salida del informe

## Estructura de directorios

- **bin**: Contiene archivos ejecutables y scripts de inicio
- **lib**: Contiene bibliotecas de dependencias de JMeter
- **docs**: Contiene documentación oficial de JMeter
- **extras**: Contiene herramientas auxiliares adicionales
- **printable_docs**: Contiene manual de usuario en formato imprimible

## Gestión de dependencias

El directorio lib de JMeter contiene dependencias centrales. Para agregar plugins o bibliotecas adicionales:

1. Copie archivos JAR al directorio lib
2. Algunos plugins necesitan colocarse en el directorio lib/ext
3. Reinicie JMeter para que los nuevos plugins tengan efecto

## Problemas comunes de instalación

- **Java no encontrado**: Asegúrese de que Java esté correctamente instalado y la variable de entorno JAVA_HOME esté configurada
- **Memoria insuficiente**: Modifique los parámetros de tamaño de Heap en bin/jmeter.sh o jmeter.bat
- **Fallo de inicio**: Verifique si está instalada la versión correcta de Java (Java 8 o superior)
