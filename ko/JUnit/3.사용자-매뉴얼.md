# JUnit 5 사용자 매뉴얼

## 기본 테스트

```java
@Test
void test() {
    assertEquals(4, 2 + 2);
}
```

## 어설션

```java
assertEquals(4, 2 + 2);
assertTrue(5 > 3);
assertThrows(Exception.class, () -> { });
```

## 매개변수화 테스트

```java
@ParameterizedTest
@ValueSource(ints = {1, 2, 3})
void test(int n) {
    assertTrue(n > 0);
}
```

## 중첩 테스트

```java
@Nested
class InnerTests {
    @Test
    void test() { }
}
```
