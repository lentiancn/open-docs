## はじめに

### Bootstrapとは？

Bootstrapは、Twitterによって開発・維持されている、世界で最も普及しているフロントエンドオープンソースツールキットです。レスポンシブウェブサイトやWebアプリケーションを素早く構築するためのCSS・JavaScriptフレームワークです。

### Bootstrapバージョン

| バージョン | リリース日 | ステータス |
|---------|----------|----------|
| Bootstrap 5.3 | 2023年5月 | 現行安定版 |
| Bootstrap 5.2 | 2022年9月 | サポート |

## インストール方法

### 方法1：CDNを使用（最も簡単）

```html
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap入門</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>Hello, Bootstrap!</h1>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

### 方法2：npmを使用

```bash
mkdir bootstrap-project
cd bootstrap-project
npm init -y
npm install bootstrap
```

### 方法3：ソースをダウンロード

https://getbootstrap.com/ からダウンロード

## Windowsへのインストール

### CDNを使用（推奨）

1. 新しいHTMLファイルを作成
2. <head>にBootstrap CSSリンクを追加
3. <body>の末尾にBootstrap JSスクリプトを追加

---

## macOSへのインストール

1. Homebrewをインストール
2. Node.jsをインストール
3. プロジェクトを作成してBootstrapをインストール

---

## Linuxへのインストール

```bash
sudo apt update
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install nodejs
npm install bootstrap
```

---

## 検証

テストHTMLファイルを作成し、青いボタンと緑のアラートが表示されれば成功。

## 最初のプロジェクト作成

```html
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <div class="container mt-5">
        <h1>マイBootstrapサイト</h1>
        <button class="btn btn-primary">プライマリーボタン</button>
    </div>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## トラブルシューティング

### 一般的な問題

1. **スタイルが効かない**
   - CSSリンクを確認
   - ブラウザキャッシュをクリア

2. **JavaScriptが動かない**
   - bootstrap.bundle.min.jsを使用確認
