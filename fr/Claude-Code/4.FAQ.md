# Claude Code FAQ

Cette page liste les problèmes courants et leurs solutions.

## Problèmes d'Installation

### Commande Non Trouvée

Erreur: `command not found: claude`

**Solution:** Vérifiez votre PATH.

macOS/Linux:
```bash
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

Windows PowerShell:
```powershell
$currentPath = [Environment]::GetEnvironmentVariable('PATH', 'User')
[Environment]::SetEnvironmentVariable('PATH', "$currentPath;$env:USERPROFILE\.local\bin", 'User')
```

Vérifiez:
```bash
claude --version
```

### Le Script Renvoie HTML

**Solution:** Utilisez une méthode alternative:
- Homebrew: `brew install --cask claude-code`
- WinGet: `winget install Anthropic.ClaudeCode`

### Erreurs TLS/SSL

**Solution:**
1. Mettez à jour les certificats CA
2. Vérifiez les paramètres proxy

## Configuration Réseau

L'installateur télécharge depuis `storage.googleapis.com`:

```bash
curl -sI https://storage.googleapis.com
```

## Vérifier les Installations en Conflit

```bash
which -a claude
```

## Diagnostic

Utilisez `claude doctor` pour plus d'informations:

```bash
claude doctor
```
