## Sistema de Cuadrícula

### Principios Básicos

El sistema de cuadrícula de Bootstrap está construido usando Flexbox con soporte hasta 12 columnas.

### Contenedores

```html
<!-- Contenedor de ancho fijo -->
<div class="container">
  Contenido
</div>

<!-- Contenedor de ancho completo -->
<div class="container-fluid">
  Contenido
</div>

<!-- Contenedores de punto de rotura responsivos (Bootstrap 5.2+) -->
<div class="container-sm">100% ancho < 576px</div>
<div class="container-md">100% ancho >= 768px</div>
<div class="container-lg">100% ancho >= 992px</div>
<div class="container-xl">100% ancho >= 1200px</div>
<div class="container-xxl">100% ancho >= 1400px</div>
```

### Filas y Columnas

```html
<div class="container">
  <div class="row">
    <div class="col">Columna</div>
    <div class="col">Columna</div>
    <div class="col">Columna</div>
  </div>
</div>
```

### Configuración de Ancho de Columna

```html
<div class="row">
  <div class="col-4">4 columnas</div>
  <div class="col-8">8 columnas</div>
</div>
```

### Puntos de Rotura Responsivos

```html
<!-- Ancho de columna responsivo -->
<div class="row">
  <div class="col-12 col-md-6 col-lg-4">
    Ancho completo en móvil, 6 columnas en tablet, 4 columnas en escritorio
  </div>
</div>
```

### Offset de Columnas

```html
<!-- Usando offset -->
<div class="row">
  <div class="col-md-4">4 columnas</div>
  <div class="col-md-4 offset-md-4">Offset 4 columnas</div>
</div>

<!-- Usando margen automático -->
<div class="row">
  <div class="col-md-3 ms-auto">Offset derecho</div>
</div>
```

### Anidación de Cuadrícula

```html
<div class="row">
  <div class="col-sm-8">
    8 columnas externas
    <div class="row">
      <div class="col-6">6 columnas internas</div>
      <div class="col-6">6 columnas internas</div>
    </div>
  </div>
</div>
```

### Diseño Flex

```html
<!-- Alineación horizontal -->
<div class="row justify-content-start">...</div>
<div class="row justify-content-center">...</div>
<div class="row justify-content-end">...</div>
<div class="row justify-content-around">...</div>
<div class="row justify-content-between">...</div>
<div class="row justify-content-evenly">...</div>

<!-- Alineación vertical -->
<div class="row align-items-start">...</div>
<div class="row align-items-center">...</div>
<div class="row align-items-end">...</div>
```

## Mejores Prácticas

### 1. Usar HTML Semántico

```html
<!-- Bien -->
<nav class="navbar">...</nav>
<main class="container">...</main>
<article class="card">...</article>

<!-- Evitar -->
<div class="navbar">...</div>
<div class="container">...</div>
<div class="card">...</div>
```

### 2. Mantener el Código Limpio

```html
<!-- Bien -->
<div class="container">
  <div class="row">
    <div class="col-md-6">...</div>
  </div>
</div>

<!-- Evitar anidación excesiva -->
<div class="container">
  <div class="row">
    <div class="col-md-6">
      <div class="card">
        <div class="card-body">...</div>
      </div>
    </div>
  </div>
</div>
```

### 3. Diseño Responsivo

```html
<!-- Usar clases de punto de rotura correctas -->
<div class="col-12 col-md-6 col-lg-4">
  Contenido
</div>
```

### 4. Optimización del Rendimiento

- Usar CDN para aceleración
- Habilitar compresión CSS y JS
- Cargar JavaScript bajo demanda
