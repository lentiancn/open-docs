# Jenkins

Jenkins — это сервер автоматизации с открытым исходным кодом для реализации непрерывной интеграции и непрерывного развёртывания (CI/CD).

## Введение

Jenkins — один из самых популярных инструментов CI/CD с открытым исходным кодом, который позволяет автоматизировать процессы сборки, тестирования и развёртывания.

## Основные функции

- **Непрерывная интеграция**: Автоматическая сборка и тестирование кода
- **Непрерывное развёртывание**: Автоматическое развёртывание в производственную среду
- **Богатый набор плагинов**: Поддержка различных инструментов и платформ
- **Распределённые сборки**: Поддержка многоузловых сборок
- **Pipeline**: Workflow на основе кода

## Быстрый старт

### Установка

```bash
# Docker
docker run -d -p 8080:8080 jenkins/jenkins:lts

# macOS
brew install jenkins

# Linux
sudo apt install jenkins
```

### Доступ

Открыть браузер по адресу http://localhost:8080
