# Element UI Benutzerhandbuch

Diese Anleitung beschreibt, wie Sie Element UI-Komponenten in Ihrer Vue 2-Anwendung verwenden.

## Grundlegende Verwendung

### Hello World

```vue
<template>
  <div>
    <el-button>Standard-Schaltfläche</el-button>
    <el-button type="primary">Primäre Schaltfläche</el-button>
  </div>
</template>

<script>
export default {
  name: 'App'
}
</script>
```

## Komponentenkategorien

Element UI bietet diese Komponentenkategorien:

| Kategorie | Komponenten |
|-----------|-------------|
| Basis | Layout, Button, Icon, Link |
| Formular | Input, Select, Radio, Checkbox, Switch, Slider, TimePicker, DatePicker |
| Daten | Table, Tree, Pagination, Tag |
| Benachrichtigung | Alert, Message, Notification, Progress |
| Navigation | Menu, Tabs, Breadcrumb, Steps, Dropdown |
| Sonstige | Dialog, Drawer, Card, Collapse, Timeline |

### Formular Form

```vue
<template>
  <el-form
    :model="ruleForm"
    :rules="rules"
    ref="ruleForm"
    label-width="120px"
  >
    <el-form-item label="Name" prop="name">
      <el-input v-model="ruleForm.name"></el-input>
    </el-form-item>
    
    <el-form-item label="E-Mail" prop="email">
      <el-input v-model="ruleForm.email"></el-input>
    </el-form-item>
    
    <el-form-item label="Geschlecht" prop="gender">
      <el-radio-group v-model="ruleForm.gender">
        <el-radio label="male">Männlich</el-radio>
        <el-radio label="female">Weiblich</el-radio>
      </el-radio-group>
    </el-form-item>
    
    <el-form-item label="Hobby" prop="hobby">
      <el-checkbox-group v-model="ruleForm.hobby">
        <el-checkbox label="Lesen" name="hobby"></el-checkbox>
        <el-checkbox label="Sport" name="hobby"></el-checkbox>
        <el-checkbox label="Musik" name="hobby"></el-checkbox>
      </el-checkbox-group>
    </el-form-item>
    
    <el-form-item label="Land" prop="country">
      <el-select v-model="ruleForm.country" placeholder="Land auswählen">
        <el-option label="Deutschland" value="germany"></el-option>
        <el-option label="Österreich" value="austria"></el-option>
        <el-option label="Schweiz" value="switzerland"></el-option>
      </el-select>
    </el-form-item>
    
    <el-form-item>
      <el-button type="primary" @click="submitForm('ruleForm')">Absenden</el-button>
      <el-button @click="resetForm('ruleForm')">Zurücksetzen</el-button>
    </el-form-item>
  </el-form>
</template>

<script>
export default {
  data() {
    return {
      ruleForm: {
        name: '',
        email: '',
        gender: '',
        hobby: [],
        country: ''
      },
      rules: {
        name: [
          { required: true, message: 'Bitte Namen eingeben', trigger: 'blur' }
        ],
        email: [
          { required: true, message: 'Bitte E-Mail eingeben', trigger: 'blur' },
          { type: 'email', message: 'Ungültiges E-Mail-Format', trigger: 'blur' }
        ],
        gender: [
          { required: true, message: 'Bitte Geschlecht auswählen', trigger: 'change' }
        ],
        country: [
          { required: true, message: 'Bitte Land auswählen', trigger: 'change' }
        ]
      }
    };
  },
  methods: {
    submitForm(formName) {
      this.$refs[formName].validate((valid) => {
        if (valid) {
          alert('Erfolgreich gesendet!');
        } else {
          console.log('Fehler beim Senden!!');
          return false;
        }
      });
    },
    resetForm(formName) {
      this.$refs[formName].resetFields();
    }
  }
}
</script>
```

## Best Practices

### 1. Komponenten lokal verwenden

```javascript
// Gut - lokal verwenden
export default {
  components: {
    [Button.name]: Button,
    [Input.name]: Input
  }
}
```

### 2. Formularvalidierung

```javascript
const validatePass = (rule, value, callback) => {
  if (value === '') {
    callback(new Error('Bitte Passwort eingeben'));
  } else {
    if (this.ruleForm.checkPass !== '') {
      this.$refs.ruleForm.validateField('checkPass');
    }
    callback();
  }
};
```

### 3. Tabellenleistung

```vue
<el-table
  :data="tableData"
  :default-sort = "{prop: 'date', order: 'descending'}"
  >
</el-table>
```

## Häufige Probleme

### Ereignismodifikatoren

```vue
<!-- .native für native Ereignisse verwenden -->
<el-input @keyup.enter.native="handleEnter"></el-input>
```

### Lazy Loading

```vue
<el-select
  v-model="value"
  filterable
  remote
  :remote-method="remoteMethod"
  :loading="loading">
  <el-option
    v-for="item in options"
    :key="item.value"
    :label="item.label"
    :value="item.value">
  </el-option>
</el-select>
```
