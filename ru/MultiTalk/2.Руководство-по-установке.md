# Руководство по установке

В этом разделе описаны требования к системе и процесс установки MultiTalk.

## Требования к системе

### Аппаратные требования

Для запуска MultiTalk рекомендуется использовать мощный GPU с достаточным объёмом видеопамяти:

- **Минимальные требования**: GPU с 8 ГБ видеопамяти (например, RTX 3060)
- **Рекомендуемые требования**: GPU с 16-24 ГБ видеопамяти (например, RTX 4090, A100)
- **Для генерации 720P**: Требуется несколько GPU или высокопроизводительный GPU с большим объёмом памяти

### Программные требования

- Операционная система: Linux (рекомендуется)
- Python 3.10
- CUDA 12.1 или выше
- Conda (для управления окружением)

## Установка

### Шаг 1: Создание окружения Conda

Создайте новое окружение Python и активируйте его:

```bash
conda create -n multitalk python=3.10
conda activate multitalk
```

### Шаг 2: Установка PyTorch

Установите PyTorch с поддержкой CUDA:

```bash
pip install torch==2.4.1 torchvision==0.19.1 torchaudio==2.4.1 --index-url https://download.pytorch.org/whl/cu121
```

### Шаг 3: Установка xFormers

Установите xFormers для оптимизации:

```bash
pip install -U xformers==0.0.28 --index-url https://download.pytorch.org/whl/cu121
```

### Шаг 4: Установка дополнительных зависимостей

Установите необходимые библиотеки:

```bash
pip install misaki[en]
pip install ninja
pip install psutil
pip install packaging
pip install flash_attn==2.7.4.post1
pip install -r requirements.txt
```

### Шаг 5: Установка аудио библиотек

Установите библиотеки для обработки аудио:

```bash
conda install -c conda-forge librosa
conda install -c conda-forge ffmpeg
```

Для систем на базе yum:

```bash
sudo yum install ffmpeg ffmpeg-devel
```

## Загрузка моделей

### Список моделей

Для работы MultiTalk необходимо загрузить следующие модели:

| Модель | Ссылка для скачивания | Примечания |
|--------|----------------------|-----------|
| Wan2.1-I2V-14B-480P | Huggingface | Базовая модель |
| chinese-wav2vec2-base | Huggingface | Аудио энкодер |
| Kokoro-82M | Huggingface | Веса TTS |
| MeiGen-MultiTalk | Huggingface | Веса аудио условия |

### Загрузка моделей с помощью huggingface-cli

```bash
huggingface-cli download Wan-AI/Wan2.1-I2V-14B-480P --local-dir ./weights/Wan2.1-I2V-14B-480P
huggingface-cli download TencentGameMate/chinese-wav2vec2-base --local-dir ./weights/chinese-wav2vec2-base
huggingface-cli download TencentGameMate/chinese-wav2vec2-base model.safetensors --revision refs/pr/1 --local-dir ./weights/chinese-wav2vec2-base
huggingface-cli download hexgrad/Kokoro-82M --local-dir ./weights/Kokoro-82M
huggingface-cli download MeiGen-AI/MeiGen-MultiTalk --local-dir ./weights/MeiGen-MultiTalk
```

### Настройка ссылок на модели

Способ 1: Создание символических ссылок

```bash
mv weights/Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json weights/Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json_old
sudo ln -s {Absolute path}/weights/MeiGen-MultiTalk/diffusion_pytorch_model.safetensors.index.json weights/Wan2.1-I2V-14B-480P/
sudo ln -s {Absolute path}/weights/MeiGen-MultiTalk/multitalk.safetensors weights/Wan2.1-I2V-14B-480P/
```

Способ 2: Копирование файлов

```bash
mv weights/Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json weights/Wan2.1-I2V-14B-480P/diffusion_pytorch_model.safetensors.index.json_old
cp weights/MeiGen-MultiTalk/diffusion_pytorch_model.safetensors.index.json weights/Wan2.1-I2V-14B-480P/
cp weights/MeiGen-MultiTalk/multitalk.safetensors weights/Wan2.1-I2V-14B-480P/
```

## Проверка установки

После установки всех зависимостей и загрузки моделей вы можете запустить базовый пример для проверки работоспособности системы:

```bash
python generate_multitalk.py \
 --ckpt_dir weights/Wan2.1-I2V-14B-480P \
 --wav2vec_dir 'weights/chinese-wav2vec2-base' \
 --input_json examples/single_example_1.json \
 --sample_steps 40 \
 --mode streaming \
 --use_teacache \
 --save_file test_run
```

## Примечания

- Модель совместима с разрешениями 480P и 720P
- Текущий код поддерживает только вывод 480P
- Для генерации 720P требуется несколько GPU
