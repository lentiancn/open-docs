# Manuel d'utilisation d'IBM Informix

## Commandes de Base

### Administration du Serveur

| Commande | Description |
|----------|-------------|
| `oninit` | Initialiser et démarrer le serveur Informix |
| `onmode` | Changer le mode du serveur (quiescent, online, etc.) |
| `onstat` | Afficher le statut et les statistiques du serveur |
| `ontape` | Utilitaire de sauvegarde et restauration |
| `onbar` | Sauvegarde et restauration avec IBM Storage Manager |
| `onspaces` | Gérer les espaces de stockage |
| `onparams` | Modifier les paramètres du serveur |

### Commandes Serveur Courantes

```bash
# Initialiser et démarrer le serveur
oninit

# Démarrer en mode production complet
onmode -vy

# Arrêter le serveur
onstat -

# Voir l'espace disque
onstat -d

# Voir les journaux logiques
onstat -l

# Voir les utilisateurs actifs
onstat -u

# Voir les verrous
onstat -k

# Voir les transactions
onstat -x
```

## Connexion à la Base de Données

### Utiliser dbaccess

```bash
# Mode interactif
dbaccess

# Exécuter un script SQL
dbaccess nom_base_de_données < script_sql.sql

# Se connecter avec un utilisateur spécifique
dbaccess -user monutilisateur mabase
```

### Paramètres de Connexion

```
Hôte : localhost (ou IP du serveur)
Port : 9088 (par défaut)
Base de données : votre_base_de_données
Nom du serveur : votre_serveur_informix
```

## Instructions SQL

### Créer une Base de Données

```sql
-- Créer une base de données simple
CREATE DATABASE mabase;

-- Créer avec journalisation tamponnée
CREATE DATABASE mabase WITH BUFFERED LOG;

-- Créer avec journalisation
CREATE DATABASE mabase WITH LOG;

-- Créer une base de données compatible ANSI
CREATE DATABASE mabase WITH LOG MODE ANSI;
```

### Créer des Tables

```sql
-- Création de table basique
CREATE TABLE clients (
    client_id SERIAL PRIMARY KEY,
    nom CHAR(50),
    email CHAR(100),
    date_creation DATE
);

-- Table avec contraintes
CREATE TABLE commandes (
    commande_id SERIAL PRIMARY KEY,
    client_id INTEGER REFERENCES clients(client_id),
    date_commande DATE,
    montant_total DECIMAL(10,2)
);

-- Table temporaire
CREATE TEMP TABLE donnees_session (
    id INTEGER,
    donnees VARCHAR(255)
);
```

### Insérer des Données

```sql
-- Insérer une ligne
INSERT INTO clients (nom, email, date_creation)
VALUES ('Jean Dupont', 'jean@exemple.com', TODAY);

-- Insérer plusieurs lignes
INSERT INTO clients (nom, email, date_creation)
VALUES 
    ('Marie Martin', 'marie@exemple.com', TODAY),
    ('Pierre Durand', 'pierre@exemple.com', TODAY);
```

### Interroger les Données

```sql
-- Sélectionner toutes les colonnes
SELECT * FROM clients;

-- Sélectionner des colonnes spécifiques
SELECT nom, email FROM clients;

-- Clause WHERE
SELECT * FROM clients WHERE client_id > 100;

-- Trier les résultats
SELECT nom, email FROM clients ORDER BY nom;

-- Regrouper avec fonctions d'agrégation
SELECT client_id, COUNT(*) as nb_commandes
FROM commandes GROUP BY client_id;

-- Joindre des tables
SELECT c.nom, com.date_commande, com.montant_total
FROM clients c, commandes com
WHERE c.client_id = com.client_id;
```

### Mettre à Jour les Données

```sql
-- Mettre à jour une ligne
UPDATE clients 
SET email = 'nouveau@exemple.com' 
WHERE client_id = 101;
```

### Supprimer les Données

```sql
-- Supprimer une ligne
DELETE FROM clients WHERE client_id = 101;

-- Supprimer basé sur une condition
DELETE FROM commandes WHERE date_commande < '2024-01-01';
```

## Fichiers de Configuration

### Fichiers de Configuration Clés

| Fichier | Purpose |
|---------|---------|
| `$INFORMIXDIR/etc/onconfig` | Configuration principale du serveur |
| `$INFORMIXDIR/etc/sqlhosts` | Configuration réseau |
| `$INFORMIXDIR/etc/$ONCONFIG` | Configuration root dbspace |

## Sauvegarde et Restauration

### Utiliser ontape

```bash
# Sauvegarde complète
ontape -s -L 0

# Restauration
ontape -r
```

### Utiliser onbar

```bash
# Sauvegarde
onbar -b -L 0

# Restauration
onbar -r
```

## Optimisation des Performances

### Paramètres Mémoire Clés

- `SHMVIRTSIZE` - Taille de mémoire partagée virtuelle
- `SHMADD` - Segments de mémoire partagée supplémentaires
- `BUFFERPOOL` - Configuration du cache de données

### Statistiques

```sql
-- Mettre à jour les statistiques de table
UPDATE STATISTICS FOR TABLE clients;

-- Mettre à jour les statistiques en détail
UPDATE STATISTICS HIGH FOR TABLE clients;
```
