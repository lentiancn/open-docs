# OpenClaw Manuel Utilisateur

> Guide complet pour utiliser OpenClaw

---

## Pour Commencer

### Utilisation Basique

OpenClaw peut être utilisé de plusieurs manières :

#### Session Interactive

```bash
# Démarrer une conversation interactive
openclaw chat
```

#### Exécution de Commandes

```bash
# Exécuter une seule commande
openclaw exec "aide-moi à créer un fichier"

# Exécuter des tâches plus complexes
openclaw exec --task "rechercher les actualités"
```

### Gestion du Gateway

Le service Gateway doit être actif pour la plupart des opérations :

```bash
# Démarrer Gateway
openclaw gateway start

# Vérifier le statut
openclaw gateway status

# Arrêter Gateway
openclaw gateway stop
```

## Gestion de Fichiers

### Lire des Fichiers

```bash
# Utiliser l'outil de lecture
read /chemin/vers/fichier.txt
```

### Écrire des Fichiers

```bash
# Créer un nouveau fichier ou écraser
write --content "Contenu du fichier" --path /chemin/vers/fichier.txt
```

### Modifier des Fichiers

```bash
# Remplacement de texte exact dans un fichier
edit --path /chemin/vers/fichier.txt --oldText "ancien texte" --newText "nouveau texte"
```

## Contrôle du Navigateur

OpenClaw peut automatiser les navigateurs Web :

### Démarrer le Navigateur

```bash
# Démarrer un navigateur isolé
browser start --profile openclaw

# Utiliser l'extension Chrome
browser start --profile chrome
```

### Actions du Navigateur

```bash
# Naviguer vers une URL
browser navigate --url https://exemple.com

# Prendre une capture d'écran
browser screenshot --path /sortie/capture.png

# Cliquer sur un élément
browser click --selector "#bouton-id"
```

## Accès Web

### Recherche Web

```bash
# Rechercher sur Internet
web_search --query "actualités" --count 5
```

### Récupérer une Page Web

```bash
# Récupérer le contenu d'une page web
web_fetch --url https://exemple.com --maxChars 5000
```

## Sous-Agents

### Créer des Sous-Agents

```bash
# Démarrer un nouveau sous-agent pour des tâches spécifiques
subagents create --task "collecte de documentation"

# Diriger le sous-agent
subagents steer --id agent-123 --message "tâche suivante"

# Terminer le sous-agent
subagents kill --id agent-123
```

## Configuration et Personnalisation

### Personnaliser la Personnalité

Modifiez `SOUL.md` pour personnaliser la personnalité de l'assistant :

```markdown
# SOUL.md - Qui vous êtes

## Vérités Fondamentales

- Soyez vraiment utile
- Ayez des opinions
- Soyez ingénieux
```

### Configurer les Outils

Modifiez `TOOLS.md` pour les paramètres d'outils spécifiques :

```markdown
# TOOLS.md - Notes Locales

## Connexions SSH
- serveur1 -> 192.168.1.100
```

## Bonnes Pratiques

1. **Espace de Travail** : Gardez vos fichiers dans le workspace défini
2. **Sécurité** : Soyez prudent avec les actions externes
3. **Mémoire** : Utilisez les fichiers de mémoire pour les informations importantes
4. **Gateway** : Gardez le service Gateway actif pour des temps de réponse rapides

## Fonctionnalités Avancées

### Automatisation

Créez des scripts pour automatiser les tâches récurrentes :

```bash
# Collecte automatisée de données
openclaw exec --task "collecter des informations sur le sujet X"
```

### Tâches Cron

Configurez des tâches périodiques :

```bash
# Créer une tâche planifiée
openclaw cron --schedule "0 9 * * *" --task "briefing matinal"
```
