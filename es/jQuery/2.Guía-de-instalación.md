# Guía de instalación de jQuery

## Obtener jQuery

### Última versión

La versión actual más reciente es **jQuery 4.0.0**.

#### Versión comprimida para producción (Recomendada)
- Descarga: [jquery-4.0.0.min.js](https://code.jquery.com/jquery-4.0.0.min.js)
- Apta para producción, tamaño de archivo pequeño, carga rápida

#### Versión de desarrollo
- Descarga: [jquery-4.0.0.js](https://code.jquery.com/jquery-4.0.0.js)
- Contiene código completo, fácil de leer y depurar

#### Archivo Source Map
- Descarga: [jquery-4.0.0.min.map](https://code.jquery.com/jquery-4.0.0.min.map)
- Se utiliza para mostrar el código fuente original en las herramientas de desarrollo del navegador

#### Versión Slim
La versión Slim es una versión más pequeña que excluye los módulos de Ajax y efectos:
- [jquery-4.0.0.slim.min.js](https://code.jquery.com/jquery-4.0.0.slim.min.js) - comprimida
- [jquery-4.0.0.slim.js](https://code.jquery.com/jquery-4.0.0.slim.js) - sin comprimir

Si no necesita las funcionalidades de Ajax y animación, usar la versión Slim puede reducir el tamaño del archivo en aproximadamente un 30%.

## Métodos de instalación

### Método 1: Incluir desde CDN (Recomendado)

Usar el CDN de jQuery es la forma más sencilla:

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
```

Versión con validación de Subresource Integrity (SRI) (recomendado, previene la manipulación del CDN):

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js" 
        integrity="sha384-KJ3o2DKtIkvYIK3UENzmM7KCkRr/rE9/Qpg6aAZGJwFDMVNA/GpGFF93hXpG5KkN" 
        crossorigin="anonymous"></script>
```

Visite [https://releases.jquery.com](https://releases.jquery.com) para obtener la lista completa de versiones y hashes SRI.

### Método 2: Instalación con npm

Para proyectos Node.js, instale vía npm:

```bash
npm install jquery

Después de la instalación, encontrará en node_modules/jquery/dist/:
- jquery-4.0.0.min.js - comprimida
- jquery-4.0.0.js - sin comprimir
- jquery-4.0.0.min.map - source map
```

### Método 3: Instalación con Yarn

```bash
yarn add jquery
```

### Método 4: Descargar localmente

Visite la [página de descarga de jQuery](https://jquery.com/download/), haga clic derecho en la versión que necesita y seleccione "Guardar como..." para guardar el archivo en su directorio de proyecto local.

## Otros servicios CDN

Además del CDN oficial, los siguientes CDN también proporcionan jQuery:

### Google CDN
```
https://ajax.googleapis.com/ajax/libs/jquery/4.0.0/jquery.min.js
```

### Microsoft CDN
```
https://ajax.aspnetcdn.com/ajax/jquery/jquery-4.0.0.min.js
```

### CDNJS
```
https://cdnjs.cloudflare.com/ajax/libs/jquery/4.0.0/jquery.min.js
```

### jsDelivr
```
https://cdn.jsdelivr.net/npm/jquery@4.0.0/dist/jquery.min.js
```

Nota: Estos CDN de terceros pueden tener cierto retraso, y los nuevos lanzamientos pueden tardar un tiempo en sincronizarse.

## Actualizando jQuery

### Usando el plugin jQuery Migrate

Al actualizar jQuery, se recomienda usar el plugin jQuery Migrate para identificar problemas de compatibilidad:

1. Actualizando desde versiones anteriores a 1.9+ o 3.x: Use [jQuery Migrate 1.4.1](https://code.jquery.com/jquery-migrate-1.4.1.min.js)

2. Actualizando desde jQuery 3.x a una versión más reciente: Use [jQuery Migrate 3.6.0](https://code.jquery.com/jquery-migrate-3.6.0.min.js)

Uso: Incluya el plugin Migrate después de jQuery

```html
<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
<script src="https://code.jquery.com/jquery-migrate-3.6.0.min.js"></script>
```

Migrate mostrará advertencias de compatibilidad en la consola para ayudarle a solucionar los problemas paso a paso.

## Versiones anteriores

Para usar versiones anteriores de jQuery, visite [jQuery CDN](https://releases.jquery.com) para ver todas las versiones históricas.

## Versiones pre-lanzamiento

El equipo de desarrollo de jQuery trabaja continuamente en el código. Cada commit en el repositorio de GitHub genera una versión en progreso. Estas versiones pueden ser inestables y no se recomiendan para uso en producción.

Ver versiones pre-lanzamiento: [https://releases.jquery.com/jquery/](https://releases.jquery.com/jquery/)
