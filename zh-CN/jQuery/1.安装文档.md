# jQuery 安装文档

在所有主流平台上安装 jQuery 的完整指南。

---

## 什么是 jQuery？

jQuery 是一个快速、小巧、功能丰富的 JavaScript 库。它简化了 HTML 文档遍历、事件处理、动画和 Ajax 交互。

---

## CDN

### CDN 提供商

| 提供商 | URL |
|--------|-----|
| cdnjs | https://cdnjs.cloudflare.com/ajax/libs/jquery/3.7.1/jquery.min.js |
| jsDelivr | https://cdn.jsdelivr.net/npm/jquery@3.7.1/dist/jquery.min.js |
| Google | https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js |

### 基本 CDN 使用

```html
<!-- jQuery 核心（压缩版） -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script>
```

### 完整示例

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>jQuery App</title>
</head>
<body>
  <div id="content">
    <p>Hello, jQuery!</p>
  </div>

  <script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script>
  <script>
    $(document).ready(function() {
      $('#content p').text('Hello, jQuery!')
    })
  </script>
</body>
</html>
```

## NPM

### 安装 jQuery

```bash
# 安装最新版本
npm install jquery

# 安装指定版本
npm install jquery@3.6.0
```

### 模块打包器使用

```javascript
// ES Modules
import $ from 'jquery'

// CommonJS
const $ = require('jquery')
```
