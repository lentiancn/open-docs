# OpenCode

オープンソース AI コーディングエージェント。

## 概要

OpenCode は、ターミナルインターフェース、デスクトップアプリ、または IDE 拡張として可以使用するオープンソース AI コーディングエージェントです。LSP 自動有効化、マルチセッション、75+ LLM プロバイダサポート等特点があります。

### 主な機能

- **LSP 自動有効化** - 適切な LSP を自動ロード
- **マルチセッション** - 複数のエージェントを並行起動
- **任意のモデル** - 75+ LLM プロバイダをサポート
- **プライバシー優先** - コードやコンテキストデータを保存しない

### 統計

- GitHub スター：100,000 以上
- コントリビューター：700 以上
- 月間アクティブ開発者：2,500,000 以上


```bash

```
