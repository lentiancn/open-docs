# Webpack 사용 가이드

## 빠른 시작

```javascript
// webpack.config.js
module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
  },
};
```

## 로더

```javascript
module: {
  rules: [
    { test: /\.css$/, use: ['style-loader', 'css-loader'] },
  ],
}
```

## 플러그인

```javascript
const HtmlWebpackPlugin = require('html-webpack-plugin');
plugins: [new HtmlWebpackPlugin()];
```
