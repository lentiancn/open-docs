# Rukovodstvo po ispolzovaniyu Nginx

Rukovodstvo po effektivnomu ispolzovaniyu Nginx.

---

## Basovaya konfiguraciya

### Server Block

```nginx
server {
    listen 80;
    server_name primer.ru;

    root /var/www/html;
    index index.html;

    location / {
        try_files $uri $uri/ =404;
    }
}
```

---

## Obratny proxy

```nginx
server {
    listen 80;
    server_name api.primer.ru;

    location / {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
    }
}
```

---

## Balansirovka nagruzki

```nginx
upstream backend {
    server backend1.primer.ru;
    server backend2.primer.ru;
}

server {
    location / {
        proxy_pass http://backend;
    }
}
```

---

## SSL/TLS

```nginx
server {
    listen 443 ssl;
    server_name primer.ru;

    ssl_certificate /etc/nginx/ssl/cert.pem;
    ssl_certificate_key /etc/nginx/ssl/key.pem;
}
```

---

## Optimizaciya

```nginx
http {
    gzip on;
    gzip_types text/plain text/css application/json application/javascript;
}
```
