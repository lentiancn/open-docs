# Введение в Java

> Java — объектно-ориентированный язык программирования

---

## Описание

Java был создан в 1995 году Sun Microsystems. Известен кроссплатформенностью.

---

## Особенности

- Объектно-ориентированный
- Кроссплатформенный (JVM)
- Автоматическое управление памятью
- Многопоточность
- Богатая экосистема
