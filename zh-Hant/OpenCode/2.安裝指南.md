# OpenCode 安裝指南

OpenCode 支持多種安裝方式。

## Linux/macOS 安裝

### 推薦：安裝腳本

```bash
curl -fsSL https://opencode.ai/install | bash
```

### Node.js

```bash
npm install -g opencode-ai
```

### Bun

```bash
bun install -g opencode-ai
```

### pnpm

```bash
pnpm install -g opencode-ai
```

### Yarn

```bash
yarn global add opencode-ai
```

### Homebrew (macOS/Linux)

```bash
brew install anomalyco/tap/opencode
```

### Arch Linux

```bash
# 穩定版
sudo pacman -S opencode

# 最新版（AUR）
paru -S opencode-bin
```

## Windows 安裝

### Chocolatey

```bash
choco install opencode
```

### Scoop

```bash
scoop install opencode
```

### NPM

```bash
npm install -g opencode-ai
```

### Mise

```bash
mise use -g github:anomalyco/opencode
```

### Docker

```bash
docker run -it --rm ghcr.io/anomalyco/opencode
```

### 手動下載

也可以從 [GitHub Releases](https://github.com/anomalyco/opencode/releases) 下載二進制文件。

## 驗證安裝

```bash
opencode --version
```

## 配置

### 連接 LLM 提供商

運行以下命令連接：

```bash
opencode
/connect
```

選擇 `opencode`，然後訪問 [opencode.ai/auth](https://opencode.ai/auth) 登入並獲取 API 金鑰。

### 其他提供商

也可以選擇其他 LLM 提供商，詳細配置請參閱[官方文檔](https://opencode.ai/docs/providers#directory)。

## 初始化項目

```bash
cd /path/to/project
opencode
/init
```

這會讓 OpenCode 分析項目並在根目錄建立 AGENTS.md 文件，幫助它理解項目結構和編碼規範。
