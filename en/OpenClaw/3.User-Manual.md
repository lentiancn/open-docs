# OpenClaw User Manual

> Comprehensive guide to using OpenClaw

---

## Getting Started

### Basic Usage

OpenClaw can be used in several ways:

#### Interactive Session

```bash
# Start an interactive conversation
openclaw chat
```

#### Command Execution

```bash
# Execute a single command
openclaw exec "help me create a file"

# Execute more complex tasks
openclaw exec --task "research current news"
```

### Gateway Management

The Gateway service must be active for most operations:

```bash
# Start Gateway
openclaw gateway start

# Check status
openclaw gateway status

# Stop Gateway
openclaw gateway stop
```

## File Management

### Reading Files

```bash
# Use the read tool
read /path/to/file.txt
```

### Writing Files

```bash
# Create new file or overwrite
write --content "File content" --path /path/to/file.txt
```

### Editing Files

```bash
# Exact text replacement in a file
edit --path /path/to/file.txt --oldText "old text" --newText "new text"
```

## Browser Control

OpenClaw can automate web browsers:

### Starting Browser

```bash
# Start isolated browser
browser start --profile openclaw

# Use Chrome extension
browser start --profile chrome
```

### Browser Actions

```bash
# Navigate to URL
browser navigate --url https://example.com

# Take screenshot
browser screenshot --path /output/screenshot.png

# Click element
browser click --selector "#button-id"
```

## Web Access

### Web Search

```bash
# Search the internet
web_search --query "current news" --count 5
```

### Fetch Web Page

```bash
# Retrieve content from a webpage
web_fetch --url https://example.com --maxChars 5000

## Messaging

### Send Message

```bash
# Send message via messaging platform
message send --target user --message "Hello!"
```

## Subagents

### Creating Subagents

```bash
# Start new subagent for specific tasks
subagents create --task "documentation collection"

# Steer subagent
subagents steer --id agent-123 --message "next task"

# Kill subagent
subagents kill --id agent-123
```

## Configuration and Personalization

### Customizing Personality

Edit `SOUL.md` to customize the assistant's personality:

```markdown
# SOUL.md - Who You Are

## Core Truths

- Be genuinely helpful
- Have opinions
- Be resourceful
```

### Configuring Tools

Edit `TOOLS.md` for specific tool settings:

```markdown
# TOOLS.md - Local Notes

## SSH Connections
- server1 -> 192.168.1.100
```

## Best Practices

1. **Workspace**: Keep your files in the defined workspace
2. **Security**: Be careful with external actions
3. **Memory**: Use memory files for important information
4. **Gateway**: Keep the Gateway service active for fast response times

---

## Advanced Features

### Automation

Create scripts to automate recurring tasks:

```bash
# Automated data collection
openclaw exec --task "collect information about topic X"
```

### Cron Jobs

Set up periodic tasks:

```bash
# Create scheduled task
openclaw cron --schedule "0 9 * * *" --task "morning briefing"
```
