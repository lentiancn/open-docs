# FantasyTalking よくある質問

## インストール関連

### インストールに失敗した場合は？

まず、環境が要件を満たしていることを確認してください。Node.jsバージョンが14.0以上であることを確認します。

インストールが失敗する場合は、以下を試してください：npmキャッシュをクリア; 国内ミラーを使用; またはソースからインストールします。

### FFmpegのインストール問題

一部システムではFFmpegが正しくインストールされていない可能性があります。FFmpeg公式サイトからバイaryをダウンロードするか、パッケージマネージャーでインストールできます。

## 使用関連

### 音声合成が遅い場合は？

速度は複数の要因に依存します。まずネットワーク接続を確認します。オーバーヘッドを減らすためにセルフホストを検討してください。

### 適切な声のスタイルを選択方法は？

FantasyTalkingはさまざまな声のスタイルを提供しています：neutral、happy、sad、angry、excited。カスタマーサービスにはneutralまたはprofessionalを使用してください。

### 英文音声合成を実現方法は？

デフォルト設定では英文を指定する必要があります：

```javascript
const audio = await client.speak('Hello, world!', {
  voice: { language: 'en-US', style: 'neutral' }
});
```

### SSMLマークアップが動作しない場合は？

speak関数呼び出し時にssml: trueパラメータを渡していることを確認してください。

## アカウントと請求

### APIキーを取得方法は？

FantasyTalking公式Web日でアカウントを登録し、コンソールでAPIキーを作成します。新規ユーザーは通常、テスト用の無料クレジットが提供されます。

### 使用量を確認方法は？

コンソールにログイン後、ダッシュボードでAPI呼び出し回数と消耗を確認できます。

## デプロイ関連

### Dockerデプロイが失敗した場合は？

まずDockerが正しくインストールされてあり実行されていることを確認してください。
