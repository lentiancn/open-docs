# JUnit

JUnit 是 Java 最流行的单元测试框架。

## 文档

- [安装指南](./1.安装文档.md)
- [使用指南](./2.使用指南.md)

## 快速开始

```xml
<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter</artifactId>
    <version>5.11.0</version>
</dependency>
```

## 版本

- **JUnit 5** - 现代版本
- **JUnit 4** - 旧版本

## 运行测试

```bash
mvn test
```
