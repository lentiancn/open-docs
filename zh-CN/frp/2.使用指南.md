# frp 使用指南

本指南介绍如何使用 frp 进行内网穿透和端口映射。

## 快速开始

### 服务端启动

```bash
# 启动 frps
./frps -c frps.ini
```

### 客户端启动

```bash
# 启动 frpc
./frpc -c frpc.ini
```

## 配置详解

### 服务端配置

#### 基本设置

```ini
[common]
bind_port = 7000           # frp 服务端口
dashboard_port = 7500      # 控制面板端口
token = password           # 认证令牌
```

#### 端口映射

```ini
[common]
vhost_http_port = 80       # HTTP 虚拟主机端口
vhost_https_port = 443    # HTTPS 虚拟主机端口
```

#### 负载均衡

```ini
[common]
max_pool_count = 10        # 连接池大小
max_ports_per_client = 0   # 每客户端端口数限制
```

### 客户端配置

#### TCP 映射

```ini
# 映射本地 SSH
[ssh]
type = tcp
local_ip = 127.0.0.1
local_port = 22
remote_port = 6000
```

#### HTTP 映射

```ini
# 映射 Web 服务
[web]
type = http
local_port = 8080
custom_domains = example.com
```

#### HTTPS 映射

```ini
[web_https]
type = https
local_port = 8443
custom_domains = example.com
```

## 高级功能

### 负载均衡

```ini[test_http]
type = http
local_port = 8080
custom_domains = web.example.com
group = web_group
group_key = 123
```

### 插件

#### HTTP 代理

```ini
[http_proxy]
type = tcp
plugin = http_proxy
plugin_http_user = user
plugin_http_password = password
local_port = 8080
remote_port = 8000
```

#### 静态文件

```ini
[static_file]
type = tcp
plugin = static_file
plugin_local_path = /var/www
pluginstrip_prefix = static
remote_port = 8000
```

## 安全配置

### 加密

```ini
[common]
use_encryption = true
```

### 压缩

```ini
[common]
use_compression = true
```

### 身份验证

```ini
[common]
authentication_method = token
token = your_secure_token
```

## 监控

### Dashboard

```ini
[common]
dashboard_port = 7500
dashboard_user = admin
dashboard_pwd = admin
```

访问 http://your_server:7500 查看状态。

### 端口统计

```ini
[common]
plugin = ports_stats
```

## 性能优化

### 连接池

```ini
[common]
max_pool_count = 50
max_conn_per_proxy = 10
```

### 心跳

```ini
[common]
heartbeat_interval = 30
heartbeat_timeout = 90
```

### 日志级别

```ini
[common]
log_level = warn
log_max_days = 3
```

## 使用场景

### 1. 远程访问内网 SSH

```ini
[ssh]
type = tcp
local_ip = 192.168.1.100
local_port = 22
remote_port = 6000
```

### 2. 暴露内网 Web 服务

```ini
[web]
type = http
local_port = 8080
custom_domains = yourdomain.com
```

### 3. 访问内网数据库

```ini
[mysql]
type = tcp
local_ip = 192.168.1.100
local_port = 3306
remote_port = 13306
```

### 4. 微信/支付回调

```ini
[wechat]
type = http
local_port = 8080
custom_domains = callback.yourdomain.com
```

## 故障排查

### 连接问题

1. 检查服务端日志
2. 验证端口是否开放
3. 检查防火墙

### 性能问题

1. 调整连接池大小
2. 启用压缩
3. 检查网络带宽
