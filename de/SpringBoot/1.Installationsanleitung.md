# Spring Boot Installationsanleitung

Dieses Handbuch behandelt die Einrichtung der Entwicklungsumgebung für Spring Boot, einschließlich Java, Maven und Gradle.

## Voraussetzungen

### Java-Installation

Spring Boot 3.x erfordert Java 17 oder höher. Spring Boot 2.x erfordert Java 8 oder höher.

**Windows:**
1. JDK von [Oracle](https://www.oracle.com/java/technologies/downloads/) oder [Adoptium](https://adoptium.net/) herunterladen
2. Installer ausführen
3. Umgebungsvariablen festlegen:
   ```
   JAVA_HOME=C:\Program Files\Java\jdk-17
   PATH=%JAVA_HOME%\bin;%PATH%
   ```

**macOS:**
```bash
# Mit Homebrew
brew install openjdk@17

# Zum PATH hinzufügen
echo 'export JAVA_HOME=$(/usr/libexec/java_home)' >> ~/.zshrc
source ~/.zshrc
```

**Linux:**
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install openjdk-17-jdk

# JAVA_HOME festlegen
echo 'export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64' >> ~/.bashrc
source ~/.bashrc
```

Installation überprüfen:
```bash
java -version
```

### Maven-Installation

**Windows:**
1. Von [Apache Maven](https://maven.apache.org/download.cgi) herunterladen
2. In einen Ordner entpacken (z.B. C:\apache-maven)
3. Umgebungsvariablen festlegen:
   ```
   MAVEN_HOME=C:\apache-maven
   PATH=%MAVEN_HOME%\bin;%PATH%
   ```

**macOS:**
```bash
brew install maven
```

**Linux:**
```bash
sudo apt install maven
```

Installation überprüfen:
```bash
mvn -version
```

### Gradle-Installation

**Windows:**
1. Von [Gradle](https://gradle.org/install/) herunterladen
2. Entpacken und zum PATH hinzufügen

**macOS:**
```bash
brew install gradle
```

**Linux:**
```bash
sudo apt install gradle
```

Installation überprüfen:
```bash
gradle -version
```

## IDE-Einrichtung

### IntelliJ IDEA (Empfohlen)
- Von [JetBrains](https://www.jetbrains.com/idea/download/) herunterladen
- JDK in der Projektstruktur konfigurieren
- Spring Boot Assistant Plugin installieren

### VS Code
- Extension Pack for Java installieren
- Spring Boot Extension installieren

## Schnellstart mit Spring Initializr

1. Besuche [start.spring.io](https://start.spring.io/)
2. Wähle:
   - Projekt: Maven oder Gradle
   - Sprache: Java
   - Spring Boot-Version
   - Projektmetadaten
3. Abhängigkeiten hinzufügen (Spring Web, Spring Data JPA, etc.)
4. Klicke auf "Generate" um das Projekt herunterzuladen

## Einrichtung überprüfen

```bash
# Java prüfen
java -version

# Maven prüfen
mvn -version

# Gradle prüfen (falls verwendet)
gradle -version

# Einfaches Spring Boot Projekt erstellen und ausführen
curl https://start.spring.io/starter.zip -o demo.zip
unzip demo.zip -d demo
cd demo
./mvnw spring-boot:run  # oder ./gradlew bootRun
```
