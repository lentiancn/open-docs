# Vue Installation Guide

Vue is a progressive JavaScript framework for building user interfaces.

## Version Overview

Vue has two major versions in active use:

| Version | Release Date | Status | Vue.js |
|---------|-------------|--------|--------|
| Vue 3 | September 2020 | Current - Recommended | 3.x |
| Vue 2 | October 2016 | Legacy (End of Life in 2023) | 2.x |

**Recommendation:** Use Vue 3 for all new projects. Vue 2 reached end of life in 2023.

### Key Differences: Vue 2 vs Vue 3

| Feature | Vue 2 | Vue 3 |
|---------|-------|-------|
| Architecture | Options API | Composition API + Options API |
| Reactivity | Object.defineProperty | Proxy |
| TypeScript | Partial support | Full TypeScript support |
| Performance | Good | Significantly better |
| Bundle Size | ~20KB gzipped | ~10KB gzipped |
| Lifecycle | created, mounted | setup + onMounted |
| Teleport | No | Yes |
| Fragments | No | Yes |
| Multiple Root | No | Yes |

## Prerequisites

### System Requirements

| Requirement | Vue 2 | Vue 3 |
|-------------|-------|-------|
| Node.js | 8.9+ | 16+ (18+ recommended) |
| npm | 5.6+ | 8+ |
| Browser Support | IE9+ | IE11 not supported |

### Required Software

- **Node.js** - JavaScript runtime
- **npm** - Package manager (comes with Node.js)
- **Git** - Version control (optional but recommended)

## Installation Methods

### Method 1: Using npm (Recommended)

#### Vue 3 (Recommended)

```bash
# Create a new Vue project
npm create vue@latest

# Or using Vue CLI (for Vue 2 compatibility)
npm install -g @vue/cli

# Create project
vue create my-vue-app
```

#### Vue 2 (Legacy)

```bash
# Install Vue CLI
npm install -g @vue/cli

# Create Vue 2 project
vue create my-vue2-app --default

# Or specify Vue 2 template
vue init webpack my-vue2-app
```

### Method 2: Using Vite (Recommended for Vue 3)

Vite is a modern build tool that provides extremely fast development server.

```bash
# Create Vue 3 project with Vite
npm create vite@latest my-vue-app -- --template vue

# Navigate to project
cd my-vue-app

# Install dependencies
npm install

# Start development server
npm run dev
```

### Method 3: Using CDN

#### Vue 3 CDN

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Vue 3 CDN Demo</title>
</head>
<body>
  <div id="app">
    <h1>{{ message }}</h1>
    <button @click="reverseMessage">Reverse</button>
  </div>

  <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
  <script>
    const { createApp, ref } = Vue
    
    createApp({
      setup() {
        const message = ref('Hello Vue 3!')
        
        const reverseMessage = () => {
          message.value = message.value.split('').reverse().join('')
        }
        
        return { message, reverseMessage }
      }
    }).mount('#app')
  </script>
</body>
</html>
```

#### Vue 2 CDN

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Vue 2 CDN Demo</title>
</head>
<body>
  <div id="app">
    <h1>{{ message }}</h1>
    <button @click="reverseMessage">Reverse</button>
  </div>

  <script src="https://unpkg.com/vue@2/dist/vue.js"></script>

      data: {
        message: 'Hello Vue 2!'
      },
      methods: {
        reverseMessage() {
          this.message = this.message.split('').reverse().join('')
        }
      }
    })
  </script>
</body>
</html>
```

## Vue Router Setup

### Installation

```bash
npm install vue-router@4
```

### Basic Setup (Vue 3)

```javascript
// src/router/index.js
import { createRouter, createWebHistory } from 'vue-router'
import Home from '../views/Home.vue'
import About from '../views/About.vue'

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```

```javascript
// src/main.js
import { createApp } from 'vue'
import App from './App.vue'
import router from './router'

createApp(App)
  .use(router)
  .mount('#app')
```

## Pinia (State Management)

### Installation

```bash
npm install pinia
```

### Basic Setup (Vue 3)

```javascript
// src/stores/counter.js
import { defineStore } from 'pinia'

export const useCounterStore = defineStore('counter', {
  state: () => ({
    count: 0
  }),
  getters: {
    doubleCount: (state) => state.count * 2
  },
  actions: {
    increment() {
      this.count++
    }
  }
})
```

```javascript
// src/main.js
import { createApp } from 'vue'
import { createPinia } from 'pinia'
import App from './App.vue'

const app = createApp(App)
app.use(createPinia())
app.mount('#app')
```

## Vue 3 Composition API

```javascript
import { ref, computed, onMounted } from 'vue'

export default {
  setup() {
    // Reactive state
    const message = ref('Hello Vue 3!')
    const count = ref(0)
    
    // Computed
    const reversedMessage = computed(() => {
      return message.value.split('').reverse().join('')
    })
    
    // Methods
    const increment = () => {
      count.value++
    }
    
    // Lifecycle
    onMounted(() => {
      console.log('Component mounted')
    })
    
    return {
      message,
      count,
      reversedMessage,
      increment
    }
  }
}
```

## Troubleshooting

### Common Issues

#### npm install fails

```bash
# Clear npm cache
npm cache clean --force

# Delete node_modules and reinstall
rm -rf node_modules
npm install
```

#### Port already in use

```bash
# Use different port
npm run dev -- --port 3000

# Or kill process on port
fkill -k 8080
```

#### Node version issues

```bash
# Check Node version
node --version

# Use nvm to switch version
nvm use 18

# Or install specific version
nvm install 18
```

#### Memory issues during build

```bash
# Increase Node memory
NODE_OPTIONS=--max_old_space_size=4096 npm run build
```

## Best Practices

1. **Use Vue 3** - Unless you have specific Vue 2 requirements
2. **Use Vite** - Faster development and building
3. **Use Composition API** - Better TypeScript support and code organization
4. **Use TypeScript** - Better developer experience
5. **Use Pinia** - For state management (not Vuex for new projects)
6. **Use Vue Router** - For navigation
