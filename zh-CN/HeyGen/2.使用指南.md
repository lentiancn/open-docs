# HeyGen 使用指南

本指南介绍如何使用 HeyGen 进行 AI 视频生成。

## 入门

### 使用网页界面

1. 登录 HeyGen：https://www.heygen.com
2. 从库中选择头像或上传自定义头像
3. 选择输入类型：文本、音频或视频
4. 生成视频并下载

### 使用 API

```python
from heygen import HeyGen

heygen = HeyGen(api_key="your_api_key")

# 从文本生成视频
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="Hello, this is my first AI video!",
    voice_id="voice_id"
)

print(f"视频 URL: {video.url}")
```

### 4. 批量生成

```python

    "第一个视频脚本",
    "第二个视频脚本",
    "第三个视频脚本"
]

# 生成多个视频

    avatar_id="avatar_id",
    scripts=scripts,
    voice_id="voice_id"
)

# 检查任务状态

    status = heygen.get_job_status(job.id)
    print(f"任务 {job.id}: {status}")
```

## 参数参考

### generate_video 参数

| 参数 | 类型 | 必需 | 描述 |
|------|------|------|------|
| avatar_id | string | 是 | 使用的头像 ID |
| script | string | 是 | 要说的文本 |
| voice_id | string | 是 | 语音 ID |
| background | string | 否 | 背景场景 |
| speed | float | 否 | 语速 (0.5-2.0) |
| emotion | string | 否 | 情感风格 |
| caption | boolean | 否 | 启用字幕 |

### 可用情感

| 情感 | 描述 |
|------|------|
| Neutral | 默认表情 |
| Happy | 开心表情 |
| Sad | 悲伤表情 |
| Angry | 愤怒表情 |
| Surprised | 惊讶表情 |

## 最佳实践

### 1. 脚本编写

- 保持句子简短清晰
- 使用自然语言
- 避免复杂术语

### 2. 头像选择

- 匹配头像与用例
- 考虑受众偏好
- 测试多个头像

### 3. 语音选择

- 匹配语音与品牌
- 考虑语言要求
- 使用示例脚本测试

### 4. 视频优化

- 使用适当的分辨率
- 针对平台优化（社交媒体、网站）
- 添加字幕以提高可访问性

## 故障排除

### 视频质量问题

**问题：** 视频看起来像素化

**解决方案：**
- 使用更高分辨率设置
- 确保源图像高质量
- 检查网络连接

### 音频问题

**问题：** 音频与视频不匹配

**解决方案：**
- 使用提供的语音
- 确保音频格式正确
- 检查唇同步设置

### 处理时间

**问题：** 视频处理时间过长

**解决方案：**
- 检查积分余额
- 使用队列进行批量处理
- 考虑优先处理
