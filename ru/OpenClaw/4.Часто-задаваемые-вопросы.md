# OpenClaw FAQ

> Часто задаваемые вопросы о OpenClaw

---

## Общие вопросы

### Что такое OpenClaw?

OpenClaw — это фреймворк ИИ-ассистента, который использует передовые технологии агентов для автоматизации задач. Он объединяет различные инструменты для управления файлами, управления браузером и многого другого.

### Для кого предназначен OpenClaw?

OpenClaw предназначен для разработчиков, продвинутых пользователей и всех, кто хочет повысить производительность с помощью ИИ-автоматизации.

## Установка

### Каковы системные требования?

| Требование | Минимум |
|------------|---------|
| Операционная система | Windows 10+, macOS, Linux |
| Node.js | Версия 18+ |
| ОЗУ | 4 ГБ |
| Место на диске | 500 МБ |

### Установка не удается?

**Решение**:
1. Проверьте версию Node.js: `node --version`
2. Очистите кэш npm: `npm cache clean --force`
3. Попробуйте с правами администратора/root
4. Запишите сообщение об ошибке

### Gateway не запускается?

**Решение**:
1. Проверьте порт: `netstat -ano | findstr 3000`
2. Используйте другой порт: `openclaw gateway start --port 3001`
3. Проверьте логи: `openclaw gateway logs`

## Использование

### Как начать разговор?

```bash
openclaw chat
```

### Как выполнить одну команду?

```bash
openclaw exec "моя команда"
```

### Можно ли использовать OpenClaw в headless-режиме?

Да, OpenClaw может работать в headless-режиме. Запустите службу Gateway и используйте API или CLI.

## Управление файлами

### Как прочитать файл?

Используйте инструмент чтения:
```
read /path/to/file.txt
```

### Как создать новый файл?

Используйте инструмент записи:
```
write --content "content" --path /new/file.txt
```

### Как отредактировать файл?

Используйте инструмент редактирования для точных изменений:
```
edit --path /file.txt --oldText "old" --newText "new"
```

## Управление браузером

### Какие браузеры поддерживаются?

- Chrome (через расширение)
- Изолированный браузер OpenClaw
- Другие через автоматизацию браузера

### Как управлять браузером?

```bash
browser start --profile openclaw
browser navigate --url https://example.com
browser screenshot --path /output.png
```

### Браузер не запускается?

**Решение**:
1. Установите расширение Chrome
2. Проверьте настройки брандмауэра
3. Очистите кэш браузера

## Под-агенты

### Что такое под-агенты?

Под-агенты — это независимые экземпляры, которые могут выполнять конкретные задачи параллельно или последовательно.

### Как создать под-агента?

```bash
subagents create --task "description"
```

### Как завершить под-агента?

```bash
subagents kill --id agent-id
```

## Безопасность

### OpenClaw безопасен?

OpenClaw реализует несколько уровней безопасности:
- Разрешения файлов
- Подтверждение внешних действий
- Параметры песочницы

### Как защитить конфиденциальные данные?

1. Используйте зашифрованную конфигурацию
2. Не делитесь паролями в разговорах
3. Используйте переменные окружения для секретов

## Устранение неполадок

### "Команда не найдена"?

Убедитесь, что:
1. OpenClaw правильно установлен
2. Переменная PATH настроена правильно
3. Gateway запущен

### Операции с файлами не работают?

1. Проверьте путь
2. Подтвердите разрешения
3. Проверьте имя файла на специальные символы

### Медленные ответы?

1. Проверьте статус Gateway
2. Подтвердите сетевое подключение
3. Проанализируйте логи
