# Element UI

[Element UI](https://element.eleme.io/)는 개발자에게 풍부한 데스크톱 컴포넌트 솔루션을 제공하는 Vue 2.0 기반 UI 컴포넌트 라이브러리입니다.

## 개요

Element UI는 Ele.me 팀에서 개발하고 관리하는 Vue 2.0 컴포넌트 라이브러리입니다.

## 주요 기능

- **Vue 2.0 지원**: Vue 2.0 위에 구축
- **70+ 컴포넌트**: 데스크톱 개발의 모든 측면을 다루는
- **필요 시 가져오기**: babel-plugin-component 지원
- **국제화**: 다중 언어 지원
- **테마 사용자 정의**: 유연한 테마

## 빠른 시작

### 설치

```bash
npm install element-ui --save
```

### 사용법

전체 가져오기:

```javascript
import Vue from 'vue'
import Element from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import App from './App.vue'

Vue.use(Element)
```
