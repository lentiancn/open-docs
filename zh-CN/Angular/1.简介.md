# Angular 简介

现代 Web 开发平台。

## 什么是 Angular？

Angular 是由 Google 开发的设计平台，用于构建移动端和桌面端 Web 应用。它使用 TypeScript/JavaScript 及其他语言。

### 主要特性

- **组件化** - 基于组件的架构
- **TypeScript** - 强类型支持
- **双向绑定** - 简化数据同步
- **依赖注入** - 强大的依赖管理系统
- **路由** - 完整的路由解决方案
- **表单处理** - 响应式表单和模板驱动表单
- **HTTP 客户端** - 完整的 HTTP 服务
- **服务端渲染** - 支持 SSR
