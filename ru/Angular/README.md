# Angular

Современная веб-платформа разработки.

## Обзор

Angular — это платформа разработки для создания мобильных и десктопных веб-приложений, разработанная Google. Использует TypeScript/JavaScript с компонентной архитектурой, двусторонним связыванием и внедрением зависимостей.

### Основные функции

- **Компонентный** - Архитектура на основе компонентов
- **TypeScript** - Поддержка строгой типизации
- **Двустороннее связывание** - Упрощённая синхронизация данных
- **Внедрение зависимостей** - Мощная система управления зависимостями
- **Роутинг** - Полное решение для маршрутизации

## Быстрый старт

### Установить

```bash
npm install -g @angular/cli
```

### Создать проект

```bash
ng new my-app
cd my-app
ng serve
```

Открыть: http://localhost:4200
