# HeyGen 使用指南

本指南介紹如何使用 HeyGen 進行 AI 影片生成。

## 入門

### 使用網頁介面

1. 登入 HeyGen：https://www.heygen.com
2. 從庫中選擇虛擬人像或上傳自訂虛擬人像
3. 選擇輸入類型：文字、音訊或影片
4. 產生影片並下載

### 使用 API

```python
from heygen import HeyGen

heygen = HeyGen(api_key="your_api_key")

# 從文字產生影片
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="Hello, this is my first AI video!",
    voice_id="voice_id"
)

print(f"影片 URL: {video.url}")
```

## 核心功能

### 1. 虛擬人像選擇

```python
# 列出可用虛擬人像
avatars = heygen.list_avatars()
for avatar in avatars:
    print(f"{avatar.id}: {avatar.name}")

# 取得特定虛擬人像
avatar = heygen.get_avatar(avatar_id="avatar_id")
```

### 2. 語音選擇

```python
# 列出可用語音
voices = heygen.list_voices()
for voice in voices:
    print(f"{voice.id}: {voice.name} ({voice.language})")

# 取得語音詳情
voice = heygen.get_voice(voice_id="voice_id")
```

### 3. 影片產生

#### 從文字產生

```python
video = heygen.generate_video(
    avatar_id="avatar_id",
    script="您的腳本文本",
    voice_id="voice_id",
    background="office"  # 可選：背景場景
)
```

#### 從音訊產生

```python
video = heygen.generate_video_from_audio(
    avatar_id="avatar_id",
    audio_url="https://example.com/audio.wav"
)
```

#### 從照片產生 (TalkingPhoto)

```python
video = heygen.generate_talking_photo(
    photo_url="https://example.com/photo.jpg",
    script="您的腳本",
    voice_id="voice_id"
)
```

### 4. 批次產生

```python
scripts = [
    "第一個影片腳本",
    "第二個影片腳本",
    "第三個影片腳本"
]

# 產生多個影片
jobs = heygen.batch_generate(
    avatar_id="avatar_id",
    scripts=scripts,
    voice_id="voice_id"
)

# 檢查任務狀態
for job in jobs:
    status = heygen.get_job_status(job.id)
    print(f"任務 {job.id}: {status}")
```

## 進階功能

### 自訂虛擬人像

```python
# 上傳自訂虛擬人像
avatar = heygen.upload_avatar(
    name="我的自訂虛擬人像",
    video_url="https://example.com/reference.mp4"
)
print(f"自訂虛擬人像 ID: {avatar.id}")
```

### 即時語音克隆

```python
# 從音訊樣本克隆語音
voice = heygen.clone_voice(
    name="我的語音克隆",
    audio_url="https://example.com/voice_sample.wav"
)
print(f"克隆語音 ID: {voice.id}")
```

### 影片編輯

```python
# 取得影片詳情
video = heygen.get_video(video_id="video_id")

# 更新影片設定
updated = heygen.update_video(
    video_id="video_id",
    title="我的更新影片",
    description="更新的說明"
)

# 刪除影片
heygen.delete_video(video_id="video_id")
```

## 參數參考

### generate_video 參數

| 參數 | 類型 | 必需 | 說明 |
|------|------|------|------|
| avatar_id | string | 是 | 使用的虛擬人像 ID |
| script | string | 是 | 要說的文字 |
| voice_id | string | 是 | 語音 ID |
| background | string | 否 | 背景場景 |
| speed | float | 否 | 語速 (0.5-2.0) |
| emotion | string | 否 | 情感風格 |
| caption | boolean | 否 | 啟用字幕 |

### 可用情感

| 情感 | 說明 |
|------|------|
| Neutral | 預設表情 |
| Happy | 開心表情 |
| Sad | 悲傷表情 |
| Angry | 憤怒表情 |
| Surprised | 驚訝表情 |

## 最佳實踐

### 1. 腳本編寫

- 保持句子簡短清晰
- 使用自然語言
- 避免複雜術語

### 2. 虛擬人像選擇

- 匹配虛擬人像與使用場景
- 考慮受眾偏好
- 測試多個虛擬人像

### 3. 語音選擇

- 匹配語音與品牌
- 考慮語言要求
- 使用範例腳本測試

### 4. 影片優化

- 使用適當的解析度
- 針對平台優化（社群媒體、網站）
- 新增字幕以提高可訪問性

## 故障排除

### 影片品質問題

**問題：** 影片看起來像素化

**解決方案：**
- 使用更高解析度設定
- 確保來源影像高品質
- 檢查網路連線

### 音訊問題

**問題：** 音訊與影片不匹配

**解決方案：**
- 使用提供的語音
- 確保音訊格式正確
- 檢查唇同步設定

### 處理時間

**問題：** 影片處理時間過長

**解決方案：**
- 檢查點數餘額
- 使用佇列進行批次處理
- 考慮優先處理
