# Node.js

Node.js® 是一个免费、开源、跨平台的 JavaScript 运行时环境，它让开发人员能够创建服务器、Web 应用、命令行工具和脚本。

## 快速开始

### 安装 Node.js

访问 [Node.js 官方网站](https://nodejs.org/zh-cn/download) 下载并安装最新版本。

### 验证安装

```bash
node --version
npm --version
```

### 创建第一个应用

创建 `hello.js` 文件：
```javascript
console.log('Hello World!');
```

运行应用：
```bash
node hello.js
```
