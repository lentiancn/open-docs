# Element UI

> Vue 2 기반 데스크톱 UI 컴포넌트 라이브러리

---

## 개요

Element UI는 Vue 2.0 기반 데스크톱 컴포넌트 라이브러리입니다. Eleme 팀이 개발했습니다.

**참고**: Vue 3 프로젝트는 [Element Plus](https://github.com/element-plus/element-plus)를 사용하세요

---

## 버전

| 버전 | Vue | 상태 |
|------|-----|------|
| Element UI 2.x | Vue 2 | 안정판 |
| Element UI 1.x | Vue 2 | 중단 |
