# Ant Design Vue 3 Installation Guide

Comprehensive guide for installing Ant Design Vue 3 on all major platforms.

---

## What is Ant Design Vue 3?

Ant Design Vue 3 is a Vue 3 based component library that follows Ant Design specifications. It provides high-quality UI components for building enterprise-level applications.

**Note**: This is for Vue 3. For Vue 2, use Ant Design Vue 2.x.

---

## CDN

### Quick Start

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Ant Design Vue CDN</title>
  <link rel="stylesheet" href="https://unpkg.com/ant-design-vue@4/dist/reset.min.css">
</head>
<body>
  <div id="app">
    <a-button>Button</a-button>
  </div>

  <script src="https://unpkg.com/vue@3"></script>
  <script src="https://unpkg.com/ant-design-vue@4"></script>
  <script>
    const { createApp } = Vue
    const App = {
      components: { AButton: AntDesignVue.AButton }
    }
    createApp(App).use(AntDesignVue).mount('#app')
  </script>
</body>
</html>
```

### CDN Providers

| Provider | CSS | JS |
|----------|-----|-----|
| unpkg | https://unpkg.com/ant-design-vue/dist/reset.min.css | https://unpkg.com/ant-design-vue |
| jsDelivr | https://cdn.jsdelivr.net/npm/ant-design-vue/dist/reset.min.css | https://cdn.jsdelivr.net/npm/ant-design-vue |

## NPM

### Install Ant Design Vue

```bash
npm install ant-design-vue
```

### Install Specific Version

```bash
npm install ant-design-vue@4
```

---

## Vite

### Create Project

```bash
npm create vite@latest my-ant-app -- --template vue
cd my-ant-app
npm install
npm install ant-design-vue
```

### vite.config.js

```javascript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [vue()],
})
```

---

## main.js

```javascript
import { createApp } from 'vue'
import Antd from 'ant-design-vue'
import 'ant-design-vue/dist/reset.css'
import App from './App.vue'

const app = createApp(App)
app.use(Antd)
app.mount('#app')
```
