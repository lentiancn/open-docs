# DragonTalker 설치 가이드

DragonTalker은 단일 이미지 및 오디오에서 사실적인 talking head 비디오를 생성하는 AI 프로젝트입니다.

## 시스템 요구 사항

### 하드웨어 요구 사항

| 구성 요소 | 최소 | 권장 |
|----------|------|------|
| GPU | NVIDIA 6GB VRAM | NVIDIA 16GB VRAM |
| RAM | 8GB | 32GB |
| 스토리지 | 20GB | 50GB |
| CUDA | 11.7+ | 11.8+ |

### 소프트웨어 요구 사항

| 소프트웨어 | 버전 | 참고 |
|----------|------|------|
| Python | 3.8 - 3.10 | 권장 3.9 |
| CUDA | 11.7+ | GPU 가속 필요 |
| ffmpeg | 최신 | 비디오 처리 필요 |

## Linux/Ubuntu 설치

### 1단계: 시스템 종속성 설치

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install python3.9 python3-pip python3-venv git wget curl
sudo apt install libgl1-mesa-glx libglib2.0-0 libsm6 libxext6 libxrender-dev libgomp1
sudo apt install ffmpeg
```

### 2단계: 프로젝트 클론

```bash
git clone https://github.com/your-repo/DragonTalker.git
cd DragonTalker
```

### 3단계: 가상 환경 생성

```bash
python3.9 -m venv venv
source venv/bin/activate
```

### 4단계: PyTorch 설치

```bash
pip install torch==2.0.1 torchvision==0.15.2 --index-url https://download.pytorch.org/whl/cu118
```

### 5단계: 종속성 설치

```bash
pip install -r requirements.txt
```

### 6단계: 모델 다운로드

```bash
mkdir -p checkpoints
bash scripts/download_models.sh
```

## Windows 설치

### 1단계: Python 설치

1. python.org에서 Python 3.9 다운로드
2. 설치 시 "Add Python to PATH" 선택

### 2단계: CUDA 설치

1. NVIDIA에서 CUDA 11.8 다운로드
2. 기본 설정으로 설치

### 3단계: 클론 및 설정

```powershell
git clone https://github.com/your-repo/DragonTalker.git
cd DragonTalker
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## 확인

```bash
python inference.py --help
```

## 일반적인 문제

### CUDA 메모리 부족

**해결책:**
- 배치 크기 줄이기: `--batch_size 1`
- 더 낮은 해상도 사용: `--size 256`
