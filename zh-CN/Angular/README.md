# Angular

> Google 开发的前端框架

---

## 项目简介

Angular 是由 Google 开发和维护的前端框架，采用 TypeScript 作为主要开发语言。提供完整的应用开发解决方案。

## 快速开始

```bash
# 安装 Angular CLI
npm install -g @angular/cli

# 创建新项目
ng new my-app

# 启动开发服务器
cd my-app
ng serve
```
