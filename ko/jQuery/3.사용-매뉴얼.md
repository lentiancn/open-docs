# jQuery 사용자 매뉴얼

## 빠른 시작

### jQuery 포함

HTML 파일의 `<head>` 또는 `<body>` 끝에 jQuery를 포함하세요:

```html
<!DOCTYPE html>
<html>
<head>
    <title>jQuery 데모</title>
    <script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
</head>
<body>
    <!-- 페이지 콘텐츠 -->
</body>
</html>
```

### DOM 준비 완료를 기다리기

jQuery 코드는 DOM이 완전히 로드된 후에 실행되어야 합니다:

```javascript
$(document).ready(function() {
    // 여기에 jQuery 코드 작성
});

// 단축형
$(function() {
    // 여기에 jQuery 코드 작성
});
```

## 선택자

jQuery 선택자는 jQuery의 핵심 기능 중 하나로 CSS 선택자와 거의 동일한 구문을 갖습니다.

### 기본 선택자

```javascript
$("#myId")          // ID 선택자
$(".myClass")       // 클래스 선택자
$("div")            // 요소 선택자
$("*")              // 범용 선택자
$("div, p, span")   // 다중 선택자
```

### 속성 선택자

```javascript
$('[href]')                    // href 속성을 가진 요소
$('[href="value"]')           // href가 지정 값과 같은 요소
$('[href!="value"]')          // href가 지정 값과 다른 요소
$('[href^="http"]')           // href가 지정 값으로 시작하는 요소
$('[href$=".pdf"]')           // href가 지정 값으로 끝나는 요소
$('[href*="google"]')         // href가 지정 값을 포함하는 요소
```

### 필터 선택자

```javascript
$("div:first")        // 첫 번째 div 요소
$("div:last")         // 마지막 div 요소
$("div:even")         // 짝수 위치의 div 요소
$("div:odd")          // 홀수 위치의 div 요소
$("div:eq(n)")        // 인덱스 n의 div 요소
$("div:gt(n)")        // 인덱스 n보다 큰 div 요소
$("div:lt(n)")        // 인덱스 n보다 작은 div 요소
$("div:not(.exclude") // 지정 클래스가 없는 div 요소
```

## DOM 조작

### 콘텐츠 가져오기 및 설정

```javascript
// 텍스트 콘텐츠 가져오기/설정
var text = $("#myElement").text();
$("#myElement").text("새 텍스트 콘텐츠");

// HTML 콘텐츠 가져오기/설정
var html = $("#myElement").html();
$("#myElement").html("<p>새 HTML 콘텐츠</p>");

// 폼 값 가져오기/설정
var value = $("#inputField").val();
$("#inputField").val("새 값");
```

### 속성 가져오기 및 설정

```javascript
// 속성 가져오기
var src = $("#myImage").attr("src");

// 단일 속성 설정
$("#myImage").attr("src", "image.jpg");

// 여러 속성 설정
$("#myImage").attr({
    src: "image.jpg",
    alt: "이미지 설명",
    title: "이미지 제목"
});

// 속성 제거
$("#myImage").removeAttr("title");
```

### CSS 가져오기 및 설정

```javascript
// 단일 CSS 속성 가져오기
var color = $("#myElement").css("color");

// 단일 CSS 속성 설정
$("#myElement").css("color", "red");

// 여러 CSS 속성 설정
$("#myElement").css({
    color: "red",
    backgroundColor: "blue",
    fontSize: "16px"
});

// 클래스 추가/제거
$("#myElement").addClass("active highlight");
$("#myElement").removeClass("highlight");
$("#myElement").toggleClass("active");  // 클래스 전환
```

### DOM 탐색

```javascript
// 부모 요소
$("#myElement").parent();
$("#myElement").parents();
$("#myElement").closest(".container");

// 자식 요소
$("#myElement").children();
$("#myElement").find(".child");

// 형제 요소
$("#myElement").siblings();
$("#myElement").next();
$("#myElement").prev();
$("#myElement").nextAll();
```

### 요소 생성 및 제거

```javascript
// 새 요소 생성
var newDiv = $("<div>", {
    class: "new-element",
    text: "새 요소 콘텐츠"
});

// 페이지에 추가
$("#container").append(newDiv);      // 내부, 끝
$("#container").prepend(newDiv);   // 내부, 처음
$("#container").before(newDiv);      // 외부, 이전
$("#container").after(newDiv);       // 외부, 이후

// 요소 제거
$("#myElement").remove();             // 요소 및 자식 제거
$("#myElement").empty();            // 자식만 제거
```

## 이벤트 처리

### 이벤트 바인딩

```javascript
// 클릭 이벤트 바인딩
$("#myButton").click(function() {
    alert("버튼 클릭됨");
});

// on 메서드 사용
$("#myButton").on("click", function() {
    alert("버튼 클릭됨");
});

// 여러 이벤트 바인딩
$("#myElement").on({
    mouseenter: function() {
        $(this).css("background", "yellow");
    },
    mouseleave: function() {
        $(this).css("background", "");
    }
});
```

### 이벤트 위임

```javascript
// 동적 요소의 이벤트 바인딩
$("#container").on("click", ".dynamic-item", function() {
    alert("동적 요소 클릭됨");
});
```

### 일반적인 이벤트

| 이벤트 | 설명 |
|--------|------|
| click | 클릭 이벤트 |
| dblclick | 더블 클릭 이벤트 |
| mouseenter | 마우스 들어옴 |
| mouseleave | 마우스 나감 |
| mousemove | 마우스 이동 |
| keydown | 키 눌림 |
| keyup | 키 해제 |
| focus | 포커스 얻음 |
| blur | 포커스 잃음 |
| submit | 폼 제출 |
| change | 값 변경 |
| load | 로드 완료 |

## 애니메이션 효과

### 기본 애니메이션

```javascript
// 요소 표시
$("#myElement").show();

// 요소 숨기기
$("#myElement").hide();

// 표시 상태 전환
$("#myElement").toggle();

// 페이드 인
$("#myElement").fadeIn();

// 페이드 아웃
$("#myElement").fadeOut();

// 페이드 전환
$("#myElement").fadeToggle();

// 슬라이드 다운
$("#myElement").slideDown();

// 슬라이드 업
$("#myElement").slideUp();

// 슬라이드 전환
$("#myElement").slideToggle();
```

### 사용자 정의 애니메이션

```javascript
$("#myElement").animate({
    opacity: 0.5,
    left: "+=50",
    height: "200px"
}, 300, function() {
    // 애니메이션 후 콜백
    console.log("애니메이션 완료");
});
```

### 애니메이션 중지

```javascript
// 현재 애니메이션 중지
$("#myElement").stop();

// 모든 애니메이션 중지 및 최종 상태 도달
$("#myElement").stop(true, true);
```

## Ajax 요청

### 기본 사용법

```javascript
$.ajax({
    url: "/api/data",
    method: "GET",
    data: { id: 1, name: "test" },
    success: function(response) {
        console.log("성공:", response);
    },
    error: function(xhr, status, error) {
        console.log("오류:", error);
    }
});
```

### 바로가기 메서드

```javascript
// GET 요청
$.get("/api/data", { id: 1 }, function(response) {
    console.log(response);
});

// POST 요청
$.post("/api/submit", { name: "test" }, function(response) {
    console.log(response);
});

// HTML 로드
$("#content").load("fragment.html");
```

### 전체 예제

```javascript
$.ajax({
    url: "https://api.example.com/users",
    type: "GET",
    dataType: "json",
    timeout: 5000,
    headers: {
        "Authorization": "Bearer token"
    },
    beforeSend: function(xhr) {
        // 요청 전송 전 실행
        console.log("요청 전송 예정");
    },
    success: function(data) {
        // 요청 성공
        $("#user-list").html("");
        data.forEach(function(user) {
            $("#user-list").append("<li>" + user.name + "</li>");
        });
    },
    error: function(xhr, status, error) {
        // 요청 실패
        alert("로드 실패: " + error);
    },
    complete: function(xhr, status) {
        // 요청 완료 (성공 또는 실패)
        console.log("요청 완료, 상태: " + status);
    }
});
```

## 유틸리티 함수

### 배열 및 객체 작업

```javascript
// 배열 반복
$.each([1, 2, 3], function(index, value) {
    console.log(index + ": " + value);
});

// 객체 반복
$.each({ name: "John", age: 30 }, function(key, value) {
    console.log(key + ": " + value);
});

// 배열 필터링
var filtered = $.grep([1, 2, 3, 4, 5], function(value) {
    return value > 3;
});

// 배열 매핑
var mapped = $.map([1, 2, 3], function(value) {
    return value * 2;
});

// 객체 병합
var merged = $.extend({}, obj1, obj2);
```

### 문자열 작업

```javascript
// 공백 제거
$.trim("  hello  ");  // "hello"

// 특수문자 이스케이프
$.escapeSelector(".myClass");  // "\.myClass"
```

### 타입 확인

```javascript
$.isArray([]);        // true
$.isFunction(fn);     // true
$.isEmptyObject({});  // true
$.isPlainObject({});  // true
```

## 체인 호출

jQuery는 동일한 요소에서 연속적으로 작업할 수 있는 체인을 지원합니다:

```javascript
$("#myElement")
    .addClass("highlight")
    .css("color", "red")
    .fadeIn()
    .animate({ height: "100px" }, function() {
        console.log("완료");
    });
```

`end()` 메서드를 사용하여 이전 선택자로 돌아갈 수 있습니다:

```javascript
$("#myElement")
    .find(".child")
    .addClass("active")
    .end()    // #myElement로 돌아가기
    .removeClass("highlight");
```
