# Bootstrap 사용자 매뉴얼

> 핵심 컴포넌트 및 사용법

---

## 그리드 시스템

### 기본 구조

```html
<div class="container">
  <div class="row">
    <div class="col">열 1</div>
    <div class="col">열 2</div>
    <div class="col">열 3</div>
  </div>
</div>
```

### 반응형 열

```html
<!-- 모바일에서는 전체, 중간 화면에서는 6열 -->
<div class="col-12 col-md-6">콘텐츠</div>

<!-- 큰 화면에서는 4열 -->
<div class="col-lg-4">콘텐츠</div>
```

## 버튼

### 기본 버튼

```html
<button class="btn btn-primary">주요</button>
<button class="btn btn-secondary">보조</button>
<button class="btn btn-success">성공</button>
<button class="btn btn-danger">위험</button>
<button class="btn btn-warning">경고</button>
<button class="btn btn-info">정보</button>
<button class="btn btn-light">밝은</button>
<button class="btn btn-dark">어두운</button>
```

### 버튼 크기

```html
<button class="btn btn-primary btn-lg">큰</button>
<button class="btn btn-primary">기본</button>
<button class="btn btn-primary btn-sm">작은</button>
```

## 폼

### 기본 폼

```html
<form>
  <div class="mb-3">
    <label class="form-label">이메일</label>
    <input type="email" class="form-control" placeholder="name@example.com">
  </div>
  <div class="mb-3">
    <label class="form-label">비밀번호</label>
    <input type="password" class="form-control">
  </div>
  <button type="submit" class="btn btn-primary">제출</button>
</form>
```

### 선택 상자

```html
<select class="form-select">
  <option selected>선택...</option>
  <option value="1">옵션 1</option>
  <option value="2">옵션 2</option>
</select>
```

## 네비게이션

### 네비게이션 바

```html
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">브랜드</a>
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav">
      <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="navbarNav">
      <ul class="navbar-nav">
        <li class="nav-item">
          <a class="nav-link active" href="#">홈</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">정보</a>
        </li>
      </ul>
    </div>
  </div>
</nav>
```

## 모달

```html
<!-- 트리거 버튼 -->
<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#exampleModal">
  모달 열기
</button>

<!-- 모달 -->
<div class="modal fade" id="exampleModal" tabindex="-1">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title">제목</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
      </div>
      <div class="modal-body">
        콘텐츠
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">닫기</button>
        <button type="button" class="btn btn-primary">저장</button>
      </div>
    </div>
  </div>
</div>
```

## 유틸리티 클래스

### 간격

```html

<div class="mt-3">상단 여백</div>
<div class="mb-3">하단 여백</div>
```

```html

<p class="text-danger">위험 텍스트</p>

```
