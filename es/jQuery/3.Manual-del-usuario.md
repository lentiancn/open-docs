# Manual del usuario de jQuery

## Inicio rápido

### Incluir jQuery

Incluya jQuery en el `<head>` o al final del `<body>` en su archivo HTML:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Demo de jQuery</title>
    <script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
</head>
<body>
    <!-- Contenido de la página -->
</body>
</html>
```

### Esperar a que el DOM esté listo

El código jQuery debe ejecutarse después de que el DOM esté completamente cargado:

```javascript
$(document).ready(function() {
    // Escriba el código jQuery aquí
});

// Forma abreviada
$(function() {
    // Escriba el código jQuery aquí
});
```

## Selectores

Los selectores de jQuery son una de las características centrales de jQuery, con una sintaxis casi idéntica a los selectores CSS.

### Selectores básicos

```javascript
$("#myId")          // Selector de ID
$(".myClass")       // Selector de clase
$("div")            // Selector de elemento
$("*")              // Selector universal
$("div, p, span")   // Múltiples selectores
```

### Selectores de atributo

```javascript
$('[href]')                    // Elementos con atributo href
$('[href="value"]')           // href igual al valor especificado
$('[href!="value"]')          // href diferente al valor especificado
$('[href^="http"]')           // href comienza con el valor especificado
$('[href$=".pdf"]')           // href termina con el valor especificado
$('[href*="google"]')         // href contiene el valor especificado
```

### Selectores de filtro

```javascript
$("div:first")        // Primer elemento div
$("div:last")         // Último elemento div
$("div:even")         // Elementos div en posición par
$("div:odd")          // Elementos div en posición impar
$("div:eq(n)")        // Div en índice n
$("div:gt(n)")        // Divs con índice mayor que n
$("div:lt(n)")        // Divs con índice menor que n
$("div:not(.exclude") // Divs sin la clase especificada
```

## Manipulación del DOM

### Obtener y establecer contenido

```javascript
// Obtener/Establecer contenido de texto
var text = $("#myElement").text();
$("#myElement").text("Nuevo contenido de texto");

// Obtener/Establecer contenido HTML
var html = $("#myElement").html();
$("#myElement").html("<p>Nuevo contenido HTML</p>");

// Obtener/Establecer valor de formulario
var value = $("#inputField").val();
$("#inputField").val("Nuevo valor");
```

### Obtener y establecer atributos

```javascript
// Obtener atributo
var src = $("#myImage").attr("src");

// Establecer atributo único
$("#myImage").attr("src", "image.jpg");

// Establecer múltiples atributos
$("#myImage").attr({
    src: "image.jpg",
    alt: "Descripción de imagen",
    title: "Título de imagen"
});

// Eliminar atributo
$("#myImage").removeAttr("title");
```

### Obtener y establecer CSS

```javascript
// Obtener propiedad CSS individual
var color = $("#myElement").css("color");

// Establecer propiedad CSS individual
$("#myElement").css("color", "red");

// Establecer múltiples propiedades CSS
$("#myElement").css({
    color: "red",
    backgroundColor: "blue",
    fontSize: "16px"
});

// Añadir/Eliminar clases
$("#myElement").addClass("active highlight");
$("#myElement").removeClass("highlight");
$("#myElement").toggleClass("active");  // Alternar clase
```

### Recorrido del DOM

```javascript
// Elementos padre
$("#myElement").parent();
$("#myElement").parents();
$("#myElement").closest(".container");

// Elementos hijo
$("#myElement").children();
$("#myElement").find(".child");

// Elementos hermanos
$("#myElement").siblings();
$("#myElement").next();
$("#myElement").prev();
$("#myElement").nextAll();
```

### Crear y eliminar elementos

```javascript
// Crear nuevo elemento
var newDiv = $("<div>", {
    class: "new-element",
    text: "Contenido del nuevo elemento"
});

// Añadir a la página
$("#container").append(newDiv);      // Dentro, al final
$("#container").prepend(newDiv);    // Dentro, al principio
$("#container").before(newDiv);      // Fuera, antes
$("#container").after(newDiv);       // Fuera, después

// Eliminar elementos
$("#myElement").remove();             // Eliminar elemento e hijos
$("#myElement").empty();              // Eliminar solo hijos
```

## Manejo de eventos

### Vincular eventos

```javascript
// Vincular evento de clic
$("#myButton").click(function() {
    alert("Botón cliqueado");
});

// Vincular usando el método on
$("#myButton").on("click", function() {
    alert("Botón cliqueado");
});

// Vincular múltiples eventos
$("#myElement").on({
    mouseenter: function() {
        $(this).css("background", "yellow");
    },
    mouseleave: function() {
        $(this).css("background", "");
    }
});
```

### Delegación de eventos

```javascript
// Vinculación de eventos para elementos dinámicos
$("#container").on("click", ".dynamic-item", function() {
    alert("Elemento dinámico cliqueado");
});
```

### Eventos comunes

| Evento | Descripción |
|--------|-------------|
| click | Evento de clic |
| dblclick | Evento de doble clic |
| mouseenter | Ratón entra |
| mouseleave | Ratón sale |
| mousemove | Ratón se mueve |
| keydown | Tecla presionada |
| keyup | Tecla liberada |
| focus | Gana foco |
| blur | Pierde foco |
| submit | Formulario enviado |
| change | Valor cambiado |
| load | Cargado |

## Efectos de animación

### Animaciones básicas

```javascript
// Mostrar elemento
$("#myElement").show();

// Ocultar elemento
$("#myElement").hide();

// Alternar visibilidad
$("#myElement").toggle();

// Desvanecer entrada
$("#myElement").fadeIn();

// Desvanecer salida
$("#myElement").fadeOut();

// Alternar desvanecimiento
$("#myElement").fadeToggle();

// Deslizar hacia abajo
$("#myElement").slideDown();

// Deslizar hacia arriba
$("#myElement").slideUp();

// Alternar deslizamiento
$("#myElement").slideToggle();
```

### Animaciones personalizadas

```javascript
$("#myElement").animate({
    opacity: 0.5,
    left: "+=50",
    height: "200px"
}, 300, function() {
    // Callback después de la animación
    console.log("Animación completada");
});
```

### Detener animaciones

```javascript
// Detener animación actual
$("#myElement").stop();

// Detener todas las animaciones y llegar al estado final
$("#myElement").stop(true, true);
```

## Solicitudes Ajax

### Uso básico

```javascript
$.ajax({
    url: "/api/data",
    method: "GET",
    data: { id: 1, name: "test" },
    success: function(response) {
        console.log("Éxito:", response);
    },
    error: function(xhr, status, error) {
        console.log("Error:", error);
    }
});
```

### Métodos abreviados

```javascript
// Solicitud GET
$.get("/api/data", { id: 1 }, function(response) {
    console.log(response);
});

// Solicitud POST
$.post("/api/submit", { name: "test" }, function(response) {
    console.log(response);
});

// Cargar HTML
$("#content").load("fragment.html");
```

### Ejemplo completo

```javascript
$.ajax({
    url: "https://api.example.com/users",
    type: "GET",
    dataType: "json",
    timeout: 5000,
    headers: {
        "Authorization": "Bearer token"
    },
    beforeSend: function(xhr) {
        // Antes de enviar la solicitud
        console.log("Solicitud a punto de enviarse");
    },
    success: function(data) {
        // Solicitud exitosa
        $("#user-list").html("");
        data.forEach(function(user) {
            $("#user-list").append("<li>" + user.name + "</li>");
        });
    },
    error: function(xhr, status, error) {
        // Solicitud fallida
        alert("Error al cargar: " + error);
    },
    complete: function(xhr, status) {
        // Solicitud completada (éxito o fallo)
        console.log("Solicitud completada, estado: " + status);
    }
});
```

## Funciones de utilidad

### Operaciones con arrays y objetos

```javascript
// Iterar array
$.each([1, 2, 3], function(index, value) {
    console.log(index + ": " + value);
});

// Iterar objeto
$.each({ name: "John", age: 30 }, function(key, value) {
    console.log(key + ": " + value);
});

// Filtrar array
var filtered = $.grep([1, 2, 3, 4, 5], function(value) {
    return value > 3;
});

// Mapear array
var mapped = $.map([1, 2, 3], function(value) {
    return value * 2;
});

// Fusionar objetos
var merged = $.extend({}, obj1, obj2);
```

### Operaciones con cadenas

```javascript
// Recortar espacios
$.trim("  hello  ");  // "hello"

// Escapar caracteres especiales
$.escapeSelector(".myClass");  // "\.myClass"
```

### Verificación de tipos

```javascript
$.isArray([]);        // true
$.isFunction(fn);     // true
$.isEmptyObject({});  // true
$.isPlainObject({});  // true
```

## Encadenamiento

jQuery soporta encadenamiento, permitiendo operar continuamente en el mismo elemento:

```javascript
$("#myElement")
    .addClass("highlight")
    .css("color", "red")
    .fadeIn()
    .animate({ height: "100px" }, function() {
        console.log("Hecho");
    });
```

Use el método `end()` para volver al selector anterior:

```javascript
$("#myElement")
    .find(".child")
    .addClass("active")
    .end()    // Volver a #myElement
    .removeClass("highlight");
```
