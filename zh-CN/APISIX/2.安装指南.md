# APISIX 安装指南

APISIX 多种安装方式。

## Docker 安装（推荐）

### 启动 APISIX

```bash
docker run -d -p 9080:9080 -p 9090:9090 apache/apisix:latest
```

### docker-compose

```yaml
version: '3'
services:
  apisix:
    image: apache/apisix:latest
    ports:
      - "9080:9080"
      - "9090:9090"
```

## Helm 安装

### 添加仓库

```bash
helm repo add apisix https://apache.github.io/apisix-helm-chart
helm repo update
```

### 安装

```bash
helm install apisix apisix/apisix
```

## 源码安装

### 依赖

- Lua 5.1+
- OpenResty
- etcd

### 安装步骤

```bash
# 克隆源码
git clone https://github.com/apache/apisix.git
cd apisix

# 安装依赖
make deps

# 启动
make run
```

## 配置

### 配置文件

主配置文件：`conf/config.yaml`

### 基本配置

```yaml
apisix:
  node_listen: 9080
  admin_key:
    - name: admin
      key: 123456
      role: admin
etcd:
  host:
    - "http://127.0.0.1:2379"
  prefix: /apisix
```

## 验证安装

```bash
curl http://127.0.0.1:9080
```

## 卸载

```bash
# Docker
docker rm -f apisix

# Helm
helm uninstall apisix
```
