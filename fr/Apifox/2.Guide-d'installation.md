# Guide d'installation

Ce document présente les différentes méthodes d'installation d'Apifox pour vous aider à commencer rapidement à utiliser Apifox en local.

## Configuration requise

Apifox est compatible avec les systèmes d'exploitation suivants :

- Windows 7 et versions supérieures (64 bits)
- macOS 10.13 et versions supérieures
- Linux (Debian/Ubuntu 20.04+, Fedora 36+)

## Méthodes de téléchargement

### Téléchargement depuis le site officiel

Rendez-vous sur le site officiel d'Apifox à l'adresse https://apifox.com/, cliquez sur le bouton "Télécharger" et，选择nez la version correspondant à votre système d'exploitation.

### Choix de la version

Apifox propose deux versions :

- **Apifox** (version standard) : contient toutes les fonctionnalités principales, adaptée à la plupart des utilisateurs
- **Apifox** (version payante) : contient des fonctionnalités avancées supplémentaires, comme la gestion d'équipe et l'automatisation avancée

Les fonctionnalités de base sont suffisantes dans la version standard.

## Étapes d'installation

### Installation sur Windows

1. Téléchargez le fichier d'installation pour Windows (.exe)
2. Double-cliquez pour exécuter le programme d'installation
3. Choisissez le chemin d'installation (il est recommandé de conserver le chemin par défaut)
4. Attendez la fin de l'installation
5. Lancez Apifox et connectez-vous ou créez un compte

### Installation sur macOS

1. Téléchargez le fichier d'installation pour macOS (.dmg)
2. Ouvrez le fichier d'installation et faites glisser Apifox vers le dossier Applications
3. Lancez Apifox et connectez-vous ou créez un compte

Lors du premier lancement, vous devrez peut-être autoriser l'exécution d'applications de développeurs inconnus dans les préférences système.

### Installation sur Linux

**Debian/Ubuntu :**

```bash
# Télécharger le paquet deb
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.deb

# Installer
sudo dpkg -i Apifox-latest-linux.x64.deb

# Si des dépendances manquent, réparer l'installation
sudo apt-get install -f
```

**Fedora :**

```bash
# Télécharger le paquet rpm
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.rpm

# Installer
sudo rpm -i Apifox-latest-linux.x64.rpm
```

## Compte et connexion

### Créer un compte

Pour une première utilisation, vous devez créer un compte :

1. Lancez Apifox
2. Cliquez sur le bouton "S'inscrire"
3. Entrez votre adresse e-mail et votre mot de passe pour compléter l'inscription
4. Vérifiez votre e-mail (si nécessaire)

### Connexion

Plusieurs méthodes de connexion sont prises en charge :

- Connexion par e-mail et mot de passe
- Connexion via un compte tiers (GitHub, Google, etc.)

### Synchronisation cloud

Une fois connecté, les données du projet sont automatiquement synchronisées avec le cloud. Vous pouvez basculer de manière transparente entre différents appareils, et les données restent toujours en temps réel.

## Configuration initiale

Après le premier lancement, il est recommandé d'effectuer les configurations suivantes :

### Choisir le thème

Dans "Paramètres", vous pouvez choisir un thème clair ou sombre selon vos préférences.

### Configurer le proxy

Si vous avez besoin d'accéder à Internet via un proxy, configurez le serveur proxy dans "Paramètres" -> "Réseau".

### Importation et exportation

- **Importation** : prend en charge l'importation de projets aux formats Postman, Swagger, Har, etc.
- **Exportation** : prend en charge l'exportation aux formats OpenAPI, Markdown, HTML, etc.

## Démarrage rapide

### Créer un projet

1. Cliquez sur le "+" à côté de "Projet" dans le panneau de gauche
2. Entrez le nom du projet
3. Choisissez le type de projet (projet API ou projet Mock)

### Créer une interface

1. Dans le projet, cliquez sur "+" et sélectionnez "Nouvelle interface"
2. Renseignez les informations de base de l'interface :
   - Nom de l'interface
   - Chemin de la requête
   - Méthode de requête (GET/POST/PUT/DELETE, etc.)
3. Renseignez les paramètres de requête, le corps de requête, les informations de réponse, etc.
4. Enregistrez l'interface

### Déboguer une interface

1. Ouvrez l'interface créée
2. Cliquez sur le bouton "Envoyer"
3. Consultez les résultats de la réponse

### Utiliser le Mock

1. Définissez la structure des données dans l'interface
2. Le système génère automatiquement une adresse Mock
3. Accédez à l'adresse Mock pour obtenir des données simulées

## Désinstallation

### Désinstaller sur Windows

- Ouvrez le "Panneau de configuration" -> "Programmes et fonctionnalités"
- Trouvez Apifox et cliquez sur "Désinstaller"

### Désinstaller sur macOS

- Faites glisser Apifox du dossier Applications vers la corbeille
- Videz la corbeille

### Désinstaller sur Linux

```bash
# Debian/Ubuntu
sudo apt-get remove apifox

# Fedora
sudo rpm -e apifox
```
