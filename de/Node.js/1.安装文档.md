# Node.js Installationsanleitung

Node.js ist ein JavaScript-Laufzeitumgebung, die auf der Chrome V8-Engine basiert. Diese Anleitung deckt die Installationsmethoden für alle Umgebungen ab.

## Versionsinformationen

| Version | Status | Beschreibung |
|------|------|------|
| Node.js 22.x | ✅ Aktuell | Neueste LTS-Version |
| Node.js 20.x | ✅ LTS | Langzeitunterstützungsversion |
| Node.js 18.x | ✅ LTS | Langzeitunterstützungsversion |
| Node.js 16.x | ⚠️ Wartung | Nur Sicherheitsupdates |
| Node.js 14.x | ❌ Veraltet | Unterstützung eingestellt |

> **Wichtig**: Diese Anleitung richtet sich hauptsächlich an **Node.js 20.x LTS**. Für neue Projekte wird die LTS-Version empfohlen.

## Installationsmethoden

### Methode 1: Windows-Installation

#### 1. Installationsprogramm verwenden (Empfohlen)

1. Node.js Installationsprogramm herunterladen: https://nodejs.org/
2. Das .msi-Installationsprogramm ausführen
3. Den Installationsassistenten durchlaufen

#### 2. Chocolatey verwenden

```powershell
# Node.js installieren
choco install nodejs

# Bestimmte Version installieren
choco install nodejs --version=20.10.0
```

#### 3. winget verwenden

```powershell
# Node.js LTS installieren
winget install OpenJS.NodeJS.LTS

# Bestimmte Version installieren
winget install OpenJS.NodeJS --version=20.10.0
```

### Methode 2: macOS-Installation

#### 1. Homebrew verwenden (Empfohlen)

```bash
# Node.js LTS installieren
brew install node@20

# Dienst starten
brew services start node@20
```

#### 2. nvm verwenden

```bash
# nvm installieren
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Node.js installieren
nvm install 20
nvm use 20
nvm alias default 20
```

#### 3. Installationsprogramm verwenden

macOS-Installationsprogramm herunterladen: https://nodejs.org/

### Methode 3: Linux-Installation

#### 1. Paketmanager verwenden

**Ubuntu/Debian:**

```bash
# Node.js-Repository hinzufügen
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -

# Node.js installieren
sudo apt-get install -y nodejs
```

**CentOS/RHEL:**

```bash
# Node.js-Repository hinzufügen
curl -fsSL https://rpm.nodesource.com/setup_20.x | sudo bash -

# Node.js installieren
sudo yum install -y nodejs
```

**Fedora:**

```bash
# Node.js-Repository hinzufügen
curl -fsSL https://rpm.nodesource.com/setup_20.x | sudo bash -

# Node.js installieren
sudo dnf install -y nodejs
```

#### 2. nvm verwenden (Empfohlen)

```bash
# nvm installieren
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Node.js installieren
nvm install 20
nvm use 20
```

### Methode 4: fnm verwenden (Schneller Node-Versionsmanager)

```bash
# fnm installieren
curl -fsSL https://fnm.vercel.app/install | bash

# Node.js installieren
fnm install 20
fnm use 20
fnm default 20
```

### Methode 5: Docker

```bash
# Node.js-Offizielles Image verwenden
docker pull node:20

# Container ausführen
docker run -it node:20 node --version
```

#### Dockerfile-Beispiel

```dockerfile
FROM node:20-alpine

WORKDIR /app

COPY package*.json ./
RUN npm ci --only=production

COPY . .

EXPOSE 3000
CMD ["node", "server.js"]
```

## Installation überprüfen

### Versionen prüfen

```bash
# Node.js-Version prüfen
node --version

# npm-Version prüfen
npm --version
```

### npm aktualisieren

```bash
# npm auf die neueste Version aktualisieren
npm install -g npm
```

## Paketverwaltung

### npm-Befehle

```bash
# Projekt initialisieren
npm init

# Abhängigkeiten installieren
npm install

# Produktionsabhängigkeiten installieren
npm install <paket>

# Entwicklungsabhängigkeiten installieren
npm install <paket> --save-dev

# Global installieren
npm install -g <paket>

# Abhängigkeiten aktualisieren
npm update

# Abhängigkeiten entfernen
npm uninstall <paket>
```

### yarn verwenden

```bash
# yarn installieren
npm install -g yarn

# Abhängigkeiten installieren
yarn install

# Abhängigkeit hinzufügen
yarn add <paket>

# Global installieren
yarn global add <paket>
```

### pnpm verwenden

```bash
# pnpm installieren
npm install -g pnpm

# Abhängigkeiten installieren
pnpm install

# Abhängigkeit hinzufügen
pnpm add <paket>
```

## Umgebungskonfiguration

### nvm für mehrere Versionen verwenden

```bash
# Installierte Versionen anzeigen
nvm list

# Neue Version installieren
nvm install 21

# Version wechseln
nvm use 21

# Standardversion festlegen
nvm alias default 20
```

### Spiegel einrichten

```bash
# npm-Spiegel einrichten
npm config set registry https://registry.npmmirror.com

# yarn-Spiegel einrichten
yarn config set registry https://registry.npmmirror.com
```

## Häufige Probleme

### Node.js und npm stimmen nicht überein?

Verwenden Sie nvm für einheitliche Versionsverwaltung:

```bash
nvm use 20
```

### Installationsfehler (Berechtigungsprobleme)?

```bash
# Linux/macOS verwendet sudo
sudo npm install -g <paket>

# Oder npm-Berechtigungen korrigieren
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc
```

### Build-Fehler?

Cache leeren:

```bash
npm cache clean --force
```
