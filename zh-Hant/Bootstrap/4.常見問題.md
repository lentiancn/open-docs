# Bootstrap 常見問題

> 常見問題解答

---

## 安裝問題

### CDN 加載失敗？

**解決方案**：

1. 檢查網絡連接
2. 使用備用 CDN：
```html
<link href="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

### npm 安裝後樣式不生效？

確保正確引入：

```javascript
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap';
```

---

## 樣式問題

### 組件樣式不生效？

**常見原因**：

1. 未引入 Bootstrap CSS
2. JavaScript 未正確加載（部分組件需要）
3. CSS 優先級被覆蓋

### 響應式不工作？

確保添加 viewport meta 標籤：

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

---

## 組件問題

### 下拉菜單不工作？

**檢查清單**：

1. 引入 Bootstrap JS（不是僅 CSS）
2. 使用正確的 HTML 結構
3. 檢查是否有 JavaScript 錯誤

### 模態框無法打開？

確保：
1. Bootstrap JS 已加載
2. data-bs-toggle 和 data-bs-target 屬性正確
3. 模態框在頁面加載時已存在

---

## 定製問題

### 如何自定義主題色？

**方法一：使用 Sass**

```scss
$primary: #ff6600;
@import "bootstrap";
```

**方法二：使用 CSS 變量**

```css
:root {
  --bs-primary: #ff6600;
}
```

---

## 兼容性問題

### 不支持 IE 11？

Bootstrap 5 不再支持 IE 11。如果需要 IE 11 支持，請使用 Bootstrap 4。

---

## 性能問題

### 如何減少文件大小？

1. 僅引入需要的組件（Sass）
2. 使用生產版本（.min.css/.min.js）
3. 啟用 gzip 壓縮

---

## 與其他框架衝突

### 與 Vue/React 一起使用？

Bootstrap 可以與任何 JavaScript 框架一起使用，但需要注意：

1. 部分組件依賴 jQuery（Bootstrap 5 不再依賴）
2. 使用框架特定的 Bootstrap 包裝庫（如 react-bootstrap）
