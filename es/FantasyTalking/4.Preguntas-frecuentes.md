# FantasyTalking Preguntas Frecuentes

## Instalación

### La instalación falla, qué hacer?

Primero confirma que el entorno cumple los requisitos. Verifica la versión de Node.js con `node --version` (debe ser >= 14.0). Si usas Python, asegúrate de que sea >= 3.7.

Si la instalación sigue fallando, intenta: Limpiar caché npm con `npm cache clean --force`; Usar espejo nacional; O instalar desde código fuente.

## Uso

### La síntesis de voz es lenta?

La velocidad depende de múltiples factores. Primero verifica la conexión de red. Considera autoalojar servicios para reducir latencia. Para grandes tareas, usa la interfaz de síntesis por lotes.

### Cómo obtener clave API?

Visita el sitio web oficial de FantasyTalking para registrarte, luego crea una clave API en la consola.

## Despliegue

### Docker despliegue falla?

Primero confirma que Docker está correctamente instalado y ejecutándose. Verifica si el puerto 8080 está en uso.
