# Vue 2 安装文档

在所有主流平台上安装 Vue 2 的完整指南。

---

## 什么是 Vue 2？

Vue 2 是一个用于构建用户界面的渐进式 JavaScript 框架。2016年发布，成为最流行的前端框架之一。

**注意**：Vue 2 于 2023 年 12 月 31 日停止维护。新项目建议使用 Vue 3。

---

## CDN

### 基本 CDN 使用

```html
<!-- 开发版 -->
<script src="https://cdn.jsdelivr.net/npm/vue@2/dist/vue.js"></script>

<!-- 生产版 -->
<script src="https://cdn.jsdelivr.net/npm/vue@2/dist/vue.min.js"></script>
```

### 完整示例

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Vue 2 App</title>
</head>
<body>
  <div id="app">
    {{ message }}
  </div>

  <script src="https://cdn.jsdelivr.net/npm/vue@2/dist/vue.js"></script>
  <script>
    new Vue({
      el: '#app',
      data: {
        message: 'Hello Vue 2!'
      }
    })
  </script>
</body>
</html>
```

### CDN 提供商

| 提供商 | URL |

## NPM

### 安装 Vue 2

```bash
# 安装 Vue 2
npm install vue@2

# 安装指定版本
npm install vue@2.6.14
```

### 安装 Vue CLI

```bash
# 安装 Vue CLI
npm install -g @vue/cli@4
```

---

## Vue CLI

### 创建新项目

```bash
# 安装 Vue CLI 4
npm install -g @vue/cli@4

# 创建新项目
vue create my-vue2-app

# 出现提示时选择 Vue 2
```

### 手动配置

```bash
# 使用 Vite 创建项目
npm create vite@latest my-vue2-app -- --template vue

# 安装 Vue 2
cd my-vue2-app
npm install vue@2
```

## 项目结构

```
my-vue2-app/
├── public/
│   └── index.html
├── src/
│   ├── components/
│   │   └── HelloWorld.vue
│   ├── App.vue
│   └── main.js
├── package.json
└── vue.config.js
```

### main.js

```javascript
import Vue from 'vue'
import App from './App.vue'

Vue.config.productionTip = false

new Vue({
  render: h => h(App)
}).$mount('#app')
```

## 版本迁移

### 从 Vue 2 迁移到 Vue 3

Vue 2 于 2023 年 12 月 31 日停止维护。建议迁移到 Vue 3。

**迁移工具：**

```bash
# 安装 Vue 3
npm install vue@3

# 使用兼容版本
npm install @vue/compat
```
