# Vue インストールガイド

Vueはユーザーインターフェースを構築するためのプログレッシブJavaScriptフレームワークです。

## バージョンの概要

Vueには現在使用中の2つのメジャーバージョンがあります：

| バージョン | リリース日 | ステータス | Vue.js |
|---------|-------------|--------|--------|
| Vue 3 | 2020年9月 | 現行 - 推奨 | 3.x |
| Vue 2 | 2016年10月 | レガシー（2023年終了）| 2.x |

**推奨：** すべての新規プロジェクトにはVue 3を使用してください。Vue 2は2023年にサポート終了しました。

### 主要な違い：Vue 2 と Vue 3

| 機能 | Vue 2 | Vue 3 |
|---------|-------|-------|
| アーキテクチャ | Options API | Composition API + Options API |
| リアクティブ | Object.defineProperty | Proxy |
| TypeScript | 一部サポート | 完全なTypeScriptサポート |
| パフォーマンス | 良好 | 大幅に向上 |
| バンドルサイズ | 約20KB（圧縮後）| 約10KB（圧縮後）|
| ライフサイクル | created, mounted | setup + onMounted |
| Teleport | なし | あり |
| Fragments | なし | あり |
| 複数ルート | なし | あり |

## 前提条件

### システム要件

| 要件 | Vue 2 | Vue 3 |
|-------------|-------|-------|
| Node.js | 8.9+ | 16+（18+を推奨）|
| npm | 5.6+ | 8+ |
| ブラウザサポート | IE9+ | IE11非サポート |

### 必要なソフトウェア

- **Node.js** - JavaScriptランタイム
- **npm** - パッケージマネージャー（Node.jsに付属）
- **Git** - バージョン管理（オプションだが推奨）

## インストール方法

### 方法1：npmを使用（推奨）

#### Vue 3（推奨）

```bash
# 新しいVueプロジェクトを作成
npm create vue@latest

# またはVue CLIを使用（Vue 2互換性のため）
npm install -g @vue/cli

# プロジェクト作成
vue create my-vue-app
```

#### Vue 2（レガシー）

```bash
# Vue CLIをインストール
npm install -g @vue/cli

# Vue 2プロジェクトを作成
vue create my-vue2-app --default

# またはVue 2テンプレートを指定
vue init webpack my-vue2-app
```

### 方法2：Viteを使用（Vue 3に推奨）

Viteは極めて高速な開発サーバーを提供する最新のビルドツールです。

```bash
# ViteでVue 3プロジェクトを作成
npm create vite@latest my-vue-app -- --template vue

# プロジェクトに移動
cd my-vue-app

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

### 方法3：CDNを使用

#### Vue 3 CDN

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>Vue 3 CDN デモ</title>
</head>
<body>
  <div id="app">
    <h1>{{ message }}</h1>
    <button @click="reverseMessage">反転</button>
  </div>

  <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
  <script>
    const { createApp, ref } = Vue
    
    createApp({
      setup() {
        const message = ref('Hello Vue 3!')
        
        const reverseMessage = () => {
          message.value = message.value.split('').reverse().join('')
        }
        
        return { message, reverseMessage }
      }
    }).mount('#app')
  </script>
</body>
</html>
```

#### Vue 2 CDN

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>Vue 2 CDN デモ</title>
</head>
<body>
  <div id="app">
    <h1>{{ message }}</h1>
    <button @click="reverseMessage">反転</button>
  </div>

  <script src="https://unpkg.com/vue@2/dist/vue.js"></script>
  <script>
    new Vue({
      el: '#app',
      data: {
        message: 'Hello Vue 2!'
      },
      methods: {
        reverseMessage() {
          this.message = this.message.split('').reverse().join('')
        }
      }
    })
  </script>
</body>
</html>
```

## 環境セットアップ

1. https://nodejs.org からNode.jsをダウンロード
2. インスール 프로그램을実行（LTSバージョンを推奨）

## Vue Routerのセットアップ

### インストール

```bash
npm install vue-router@4
```

### 基本セットアップ（Vue 3）

```javascript
// src/router/index.js
import { createRouter, createWebHistory } from 'vue-router'
import Home from '../views/Home.vue'
import About from '../views/About.vue'

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```

```javascript
// src/main.js
import { createApp } from 'vue'
import App from './App.vue'
import router from './router'

createApp(App)
  .use(router)
  .mount('#app')
```

## Pinia（状態管理）

### インストール

```bash
npm install pinia
```

### 基本セットアップ（Vue 3）

```javascript
// src/stores/counter.js
import { defineStore } from 'pinia'

export const useCounterStore = defineStore('counter', {
  state: () => ({
    count: 0
  }),
  getters: {
    doubleCount: (state) => state.count * 2
  },
  actions: {
    increment() {
      this.count++
    }
  }
})
```

```javascript
// src/main.js
import { createApp } from 'vue'
import { createPinia } from 'pinia'
import App from './App.vue'

const app = createApp(App)
app.use(createPinia())
app.mount('#app')
```

## Vue 2 Options API（レガシー）

```javascript
new Vue({
  el: '#app',
  data() {
    return {
      message: 'Hello Vue 2!',
      items: []
    }
  },
  computed: {
    reversedMessage() {
      return this.message.split('').reverse().join('')
    }
  },
  methods: {
    handleClick() {
      console.log('Clicked!')
    }
  },
  created() {
    console.log('Component created')
  },
  mounted() {
    console.log('Component mounted')
  }
})
```

## Vue 3 Composition API

```javascript
import { ref, computed, onMounted } from 'vue'

export default {
  setup() {
    // リアクティブな状態
    const message = ref('Hello Vue 3!')
    const count = ref(0)
    
    // 算出プロパティ
    const reversedMessage = computed(() => {
      return message.value.split('').reverse().join('')
    })
    
    // メソッド
    const increment = () => {
      count.value++
    }
    
    // ライフサイクル
    onMounted(() => {
      console.log('Component mounted')
    })
    
    return {
      message,
      count,
      reversedMessage,
      increment
    }
  }
}
```

## トラブルシューティング

### 一般的な問題

#### npm installが失敗する

```bash
# npmキャッシュをクリア
npm cache clean --force

# node_modulesを削除して再インストール
rm -rf node_modules
npm install
```

#### ポートが使用中

```bash
# 別のポートを使用
npm run dev -- --port 3000

# またはポート上のプロセスを終了
fkill -k 8080
```

#### Nodeバージョン問題

```bash
# Nodeバージョンを確認
node --version

# nvmでバージョンを切り替え
nvm use 18

# または特定のバージョンをインストール
nvm install 18
```

#### ビルド中のメモリ問題

```bash
# Nodeのメモリを増量
NODE_OPTIONS=--max_old_space_size=4096 npm run build
```

## ベストプラクティス

1. **Vue 3を使用** - 特定のVue 2要件がない限り
2. **Viteを使用** - より高速な開発とビルド
3. **Composition APIを使用** - よりよいTypeScriptサポートとコード整理
4. **TypeScriptを使用** - よりよい開発者体験
5. **Piniaを使用** - 状態管理のため（新規プロジェクトではVuexを避ける）
6. **Vue Routerを使用** - ナビゲーションのため

## バージョンの互換性

|-------------|-------------|------|-------|
| 3.x | 4.x | 4.x | 2.x+ |
| 2.x | 3.x | 3.x/4.x | なし |
