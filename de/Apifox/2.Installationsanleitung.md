# Installationsanleitung

Dieses Dokument beschreibt die verschiedenen Installationsmethoden von Apifox und hilft Ihnen, schnell mit Apifox auf Ihrem lokalen Gerät zu beginnen.

## Systemanforderungen

Apifox unterstützt folgende Betriebssysteme:

- Windows 7 und höher (64-Bit)
- macOS 10.13 und höher
- Linux (Debian/Ubuntu 20.04+, Fedora 36+)

## Download-Methoden

### Download von der offiziellen Website

Besuchen Sie die Apifox-Website unter https://apifox.com/, klicken Sie auf die Schaltfläche "Herunterladen" und wählen Sie die Version für Ihr Betriebssystem.

### Versionsauswahl

Apifox bietet zwei Versionen:

- **Apifox** (Standardversion): Enthält alle Kernfunktionen, geeignet für die meisten Nutzer
- **Apifox Plus** (kostenpflichtige Version): Enthält zusätzliche erweiterte Funktionen wie Teamverwaltung und erweiterte Automatisierung

Die Grundfunktionen in der Standardversion sind bereits völlig ausreichend.

## Installationsschritte

### Windows-Installation

1. Laden Sie das Windows-Installationspaket herunter (.exe-Datei)
2. Doppelklicken Sie auf das Installationsprogramm
3. Wählen Sie den Installationspfad (Standardpfad wird empfohlen)
4. Warten Sie, bis die Installation abgeschlossen ist
5. Starten Sie Apifox und melden Sie sich an oder registrieren Sie ein Konto

### macOS-Installation

1. Laden Sie das macOS-Installationspaket herunter (.dmg-Datei)
2. Öffnen Sie das Installationspaket und ziehen Sie Apifox in den Ordner "Programme"
3. Starten Sie Apifox und melden Sie sich an oder registrieren Sie ein Konto

Beim ersten Start müssen Sie möglicherweise in den Systemeinstellungen erlauben, Apps von unbekannten Entwicklern auszuführen.

### Linux-Installation

**Debian/Ubuntu:**

```bash
# Debian-Paket herunterladen
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.deb

# Installieren
sudo dpkg -i Apifox-latest-linux.x64.deb

# Fehlende Abhängigkeiten reparieren
sudo apt-get install -f
```

**Fedora:**

```bash
# RPM-Paket herunterladen
wget https://cdn.apifox.cn/latest/Apifox-latest-linux.x64.rpm

# Installieren
sudo rpm -i Apifox-latest-linux.x64.rpm
```

## Konto und Anmeldung

### Konto registrieren

Bei der ersten Nutzung müssen Sie ein Konto registrieren:

1. Starten Sie Apifox
2. Klicken Sie auf die Schaltfläche "Registrieren"
3. Geben Sie Ihre E-Mail-Adresse und Ihr Passwort ein
4. E-Mail-Verifizierung (falls erforderlich)

### Anmeldung

Verschiedene Anmeldemethoden werden unterstützt:

- Anmeldung mit E-Mail und Passwort
- Anmeldung über Drittanbieterkonten (GitHub, Google usw.)

### Cloud-Synchronisierung

Nach der Anmeldung werden Projektdaten automatisch mit der Cloud synchronisiert. Sie können nahtlos zwischen verschiedenen Geräten wechseln, wobei die Daten in Echtzeit synchron bleiben.

## Ersteinrichtung

Nach dem ersten Start sollten Sie folgende Konfigurationen vornehmen:

### Design festlegen

In den "Einstellungen" können Sie zwischen hellem und dunklem Design wählen.

### Proxy einrichten

Falls Sie einen Proxy für den Netzwerkzugriff benötigen, konfigurieren Sie den Proxyserver unter "Einstellungen" -> "Netzwerk".

### Import und Export

- **Import:** Unterstützt den Import von Projekten in Formaten wie Postman, Swagger und HAR
- **Export:** Unterstützt den Export in Formaten wie OpenAPI, Markdown und HTML

## Schnellstart

### Projekt erstellen

1. Klicken Sie auf das "+" neben "Projekt" auf der linken Seite
2. Geben Sie einen Projektnamen ein
3. Wählen Sie den Projekttyp (API-Projekt oder Mock-Projekt)

### Schnittstelle erstellen

1. Klicken Sie im Projekt auf "+" und wählen Sie "Neue Schnittstelle"
2. Füllen Sie die grundlegenden Schnittstelleninformationen aus:
   - Schnittstellenname
   - Anfragepfad
   - Anfragemethode (GET/POST/PUT/DELETE usw.)
3. Füllen Sie Anfrageparameter, Anfragekörper, Antwort usw. aus
4. Speichern Sie die Schnittstelle

### Schnittstelle testen

1. Öffnen Sie die erstellte Schnittstelle
2. Klicken Sie auf die Schaltfläche "Senden"
3. Sehen Sie sich das Antwortergebnis an

### Mock verwenden

1. Definieren Sie die Datenstruktur in der Schnittstelle
2. Das System generiert automatisch eine Mock-Adresse
3. Rufen Sie die Mock-Adresse auf, um simulierte Daten zu erhalten

## Deinstallation

### Windows deinstallieren

- Öffnen Sie die "Systemsteuerung" -> "Programme und Funktionen"
- Suchen Sie Apifox und klicken Sie auf "Deinstallieren"

### macOS deinstallieren

- Ziehen Sie Apifox aus dem Programmordner in den Papierkorb
- Leeren Sie den Papierkorb

### Linux deinstallieren

```bash
# Debian/Ubuntu
sudo apt-get remove apifox

# Fedora
sudo rpm -e apifox
```
