# Element Plus 사용 가이드

## 빠른 시작

```vue
<template>
  <el-button type="primary">버튼</el-button>
  <el-input v-model="input" />
</template>

<script setup>
import { ref } from 'vue'
const input = ref('')
</script>
```

## 컴포넌트

### 버튼

```vue
<el-button>기본</el-button>
<el-button type="primary">주요</el-button>
```

### 폼

```vue
<el-form :model="form">
  <el-form-item label="이름">
    <el-input v-model="form.name" />
  </el-form-item>
</el-form>
```

### 테이블

```vue
<el-table :data="data">
  <el-table-column prop="name" label="이름" />
</el-table>
```
