# Angular Руководство по установке

Подробное руководство по настройке среды Angular.

## Требования

### Node.js

- Рекомендуется: Node.js 18+
- Требуется npm или yarn

### Проверить версию Node.js

```bash
node --version
npm --version
```

## Установка Angular CLI

### Установить

```bash
npm install -g @angular/cli
```

### Проверить установку

```bash
ng version
```

### Обновить

```bash
npm install -g @angular/cli@latest
```

## Создание нового проекта

### Использование Angular CLI

```bash
ng new my-app
cd my-app
```

### Интерактивные параметры

- Добавить роутинг? Да/Нет
- Какой формат стилей? (SCSS/CSS/Sass/Less)

## Запуск приложения

### Сервер разработки

```bash
ng serve
# или
npm start
```

Открыть: http://localhost:4200

### Горячая перезагрузка

Angular CLI поддерживает горячую перезагрузку — изменения автоматически обновляются.

## Сборка для продакшена

```bash
ng build --configuration production
```

Результат в директории `dist/`.

## Частые команды

| Команда | Функция |
|---------|----------|
| ng new | Создать новый проект |
| ng serve | Запустить сервер разработки |
| ng build | Собрать проект |
| ng generate | Создать компонент/сервис/модуль |
| ng test | Запустить юнит-тесты |
| ng e2e | Запустить E2E-тесты |

## Удаление Angular CLI

```bash
npm uninstall -g @angular/cli
```
