# Введение в IBM Informix

## Что такое IBM Informix?

IBM Informix — это семейство систем управления реляционными базами данных (СУРБД) и многомодельных баз данных, первоначально разработанных корпорацией Informix и приобретённых IBM в 2001 году. В 2017 году IBM передала активную разработку и поддержку компании HCLSoftware, продолжая при этом продавать продукт под маркой IBM Informix.

Informix оптимизирован для сред с минимальным или отсутствующим администрированием баз данных, включая использование в качестве встроенной базы данных. Он имеет долгую историю поддержки очень высоких транзакционных нагрузок и обеспечивает характеристики доступности, необходимые для критически важных приложений.

## Основные характеристики

- **Многомодельная база данных**: Поддержка реляционных и объектно-ориентированных возможностей для моделирования сложных структур данных
- **Высокая производительность**: Оптимизация для транзакций и аналитики в реальном времени
- **Компактность**: Может работать на устройствах объёмом около 100 МБ (например, Raspberry Pi)
- **Пограничные вычисления**: Идеально для IoT-решений со встроенными возможностями базы данных
- **Поддержка временных рядов**: Встроенная поддержка данных временных рядов
- **Совместимость с JSON/MongoDB**: Интеграция с коллекциями JSON и драйверами MongoDB
- **Высокая доступность**: Функции HDR (репликация данных с высокой доступностью), RSS и SDS

## Редакции продукта

На 2025-2026 год IBM Informix предлагает следующие редакции:

| Редакция | Целевые пользователи | Ограничения | Ключевые функции |
|----------|---------------------|-------------|-----------------|
| **Developer Edition** | Индивидуальные разработчики | 1 CPU VP/Core, 1 ГБ ОЗУ, 8 ГБ хранилища | Бесплатно для разработки/тестирования |
| **Innovator-C Edition** | Малые предприятия | 2 CPU Cores, 8 ГБ ОЗУ, 50 ГБ хранилища | Бесплатно для разработки |
| **Express Edition** | Сторонние разработчики | 4 CPU Cores, 8 ГБ ОЗУ | Встраиваемая база данных |
| **Workgroup Edition** | Средние предприятия | Макс. 24 CPU Cores, 32 ГБ ОЗУ | Все опции репликации |
| **Enterprise Edition** | Крупные предприятия | Без ограничений CPU/ОЗУ | Все функции, кроме IWA |
| **Advanced Enterprise Edition** | Enterprise + BI | Без ограничений | Полный набор с IWA |
