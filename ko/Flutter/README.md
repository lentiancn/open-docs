# Flutter

Google의 오픈소스 UI 툴킷.

## 개요

Flutter는 Dart를 사용하여 단일 코드베이스에서 크로스플랫폼 애플리케이션을 구축하기 위한 오픈소스 UI 툴킷.

## 기능

- **크로스플랫폼**: 모든 플랫폼용 단일 코드베이스
- **핫 리로드**: 변경 사항 즉시 미리보기
- **풍부한 위젯**: Material Design + Cupertino
- **높은 성능**: 네이티브 렌더링 경험
- **오픈소스**: 무료 사용
