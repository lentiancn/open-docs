# Element UI 개요

Vue.js 2.0 데스크톱 UI 컴포넌트 라이브러리.

## Element UI란?

Element UI는 Ele.me 팀이 개발한 Vue.js 2.0용 오픈소스 UI 컴포넌트 라이브러리로, 기업 관리 시스템에서 널리 사용되는cola UI 컴포넌트 세트를 제공합니다.

### 주요 기능

- **풍부한 컴포넌트**: 60개 이상의 일반적인 컴포넌트
- **아름다운 디자인**: Material Design 가이드라인 준수
- **완전한 문서**: 상세한 문서와 예제
- **TypeScript 지원**: 완전한 TypeScript 정의
- **트리 셰이킹**: 온디맨드 임포트 지원
- **테마 커스터마이징**: SCSS 변수 지원
- **국제화**: 다국어 지원
- **반응형 디자인**: 다양한 화면 크기 지원

### 컴포넌트 카테고리

- **기본**: Button, Layout, Container
- **폼**: Input, Select, Cascader, Form
- **데이터**: Table, Tree, Pagination
- **알림**: Alert, Message, Notification
- **내비게이션**: Menu, Tabs, Breadcrumb
- **기타**: Dialog, Drawer, Tooltip, Upload

### 통계

- GitHub 별표: 54,000+
- 주간 npm 다운로드: 180,000+
- Vue 버전: 2.x

## 사용 사례

- **기업 관리 시스템**: ERP, CRM, OA 시스템
- **관리 대시보드**: 데이터 시각화
- **내부 비즈니스 시스템**: 워크플로 승인, 자산 관리
