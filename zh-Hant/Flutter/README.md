# Flutter

Google 推出的開源 UI 工具包。

## 簡介

Flutter 是一個用於構建跨平台應用的 UI 工具包，使用 Dart 語言。

## 功能特點

- **跨平台**：一套代碼支持多平台
- **熱重載**：即時預覽修改
- **豐富組件**：Material Design + Cupertino
- **高性能**：原生渲染體驗
- **開源免費**
