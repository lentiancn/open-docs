## Introduction

### What is Bootstrap?

Bootstrap is the world's most popular front-end open source toolkit, developed and maintained by Twitter. It's a CSS and JavaScript framework for quickly building responsive websites and web applications.

### Bootstrap Versions

| Version | Release Date | Status |
|---------|--------------|--------|
| Bootstrap 5.3 | May 2023 | Current Stable |
| Bootstrap 5.2 | September 2022 | Supported |
| Bootstrap 5.1 | August 2021 | Supported |
| Bootstrap 5.0 | July 2021 | Supported |

### Bootstrap 5 New Features

- New color schemes and style improvements
- Better form controls
- New utilities API
- Improved grid system
- Better customization options
- No jQuery dependency (but still compatible)
- Dropped Internet Explorer support

## Installation Methods

### Method 1: Using CDN (Easiest)

CDN is the fastest way to get started - no downloads required.

#### CSS Only

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Getting Started</title>
    <!-- Bootstrap CSS -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1>Hello, Bootstrap!</h1>
    
    <!-- Bootstrap JS -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

#### CDN Providers

**jsDelivr (Recommended)**
```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

**unpkg**
```html
<link href="https://unpkg.com/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://unpkg.com/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

**cdnjs**
```html
<link href="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>
```

### Method 2: Using npm

#### 1. Install Node.js

#### 2. Create Project and Install Bootstrap

```bash
mkdir my-bootstrap-project
cd my-bootstrap-project
npm init -y
npm install bootstrap
npm install @popperjs/core
```

#### 3. Use in Project

```javascript
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.min.js';
```

### Method 3: Download Source

1. Visit https://getbootstrap.com/ and click Download
2. Extract the files
3. Reference in HTML:

```html
<link href="css/bootstrap.min.css" rel="stylesheet">
<script src="js/bootstrap.bundle.min.js"></script>
```

### Method 4: Using Sass Source (Advanced)

```bash
npm install bootstrap
# or
npm install bootstrap-sass
```

## Installation on Windows

### Method 1: Using CDN (Recommended)

1. Create a new HTML file
2. Add Bootstrap CSS link in `<head>`
3. Add Bootstrap JS script at end of `<body>`

### Method 2: Using npm

1. **Install Node.js**
   - Visit https://nodejs.org/
   - Download LTS version

2. **Open Command Prompt**

3. **Create Project**
   ```cmd
   mkdir bootstrap-project
   cd bootstrap-project
   npm init -y
   npm install bootstrap
   ```

## Installation on macOS

### Using Homebrew (Recommended)

1. **Install Homebrew**
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```

2. **Install Node.js**
   ```bash
   brew install node
   ```

3. **Create Project**
   ```bash
   mkdir bootstrap-project
   cd bootstrap-project
   npm init -y
   npm install bootstrap
   ```

## Troubleshooting

### Common Issues

#### 1. Styles Not Working

- Check CSS link is correct
- Clear browser cache

#### 2. JavaScript Not Working

- Ensure bootstrap.bundle.min.js is used (includes Popper)
- Check for JavaScript console errors

#### 3. Dropdown Not Working

- Ensure Popper.js is included
- Use bootstrap.bundle.min.js
- Check data-bs-toggle attribute
