# Angular

现代 Web 应用开发框架。

## 简介

Angular 是由 Google 开发维护的开源 Web 应用框架，为现代 Web 应用开发提供完整解决方案。

### 主要特性

- **TypeScript 原生**：完全基于 TypeScript 开发
- **组件化架构**：基于组件的开发模式
- **依赖注入**：强大的依赖注入系统
- **路由系统**：完整的客户端路由
- **表单处理**：响应式表单支持
- **HTTP 客户端**：完整的 HTTP 服务

## 快速开始

### 安装 Angular CLI

```bash
npm install -g @angular/cli
```

### 创建新项目

```bash
ng new my-app
cd my-app
ng serve
```

### 访问应用

打开浏览器：http://localhost:4200

## 常用命令

```bash
# 创建组件
ng g c my-component

# 创建服务
ng g s my-service

# 构建生产版本
ng build

# 运行测试
ng test
```

## 核心概念

- **组件**：视图和逻辑的组合
- **模块**：组织应用代码
- **服务**：封装业务逻辑
- **路由**：页面导航
- **指令**：扩展 HTML
