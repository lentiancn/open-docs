# Guide d'installation d'Apache APISIX

## Configuration Requise

- **Système**: Linux (Ubuntu, CentOS, Debian), macOS
- **Mémoire**: 2GB minimum (4GB+ pour production)
- **CPU**: 2 cœurs minimum

## Méthodes d'Installation

### Méthode 1: Docker Quickstart (Recommandé)

```bash
curl -sL https://run.api7.ai/apisix/quickstart | sh
```

Après installation:
```
✔ APISIX is ready!
```

Ports par défaut:
- 9080: HTTP
- 9443: HTTPS
- 9180: Admin API

### Méthode 2: Docker Compose

```yaml
version: '3'
services:
  apisix:
    image: apache/apisix:3.3.0
    ports:
      - "9080:9080"
      - "9443:9443"
      - "9180:9180"
  etcd:
    image: apache/apisix/etcdserve:v3.5.0
```

```bash
docker-compose up -d
```

### Méthode 3: Helm (Kubernetes)

```bash
helm repo add apisix https://apache.github.io/apisix-helm-chart
helm install apisix apisix/apisix --namespace apisix
```

## Vérifier l'Installation

```bash
curl "http://127.0.0.1:9080" --head
# Réponse: Server: APISIX/3.3.0
```

## Dashboard

```
http://127.0.0.1:9180/ui
```

## Désinstallation

### Docker
```bash
docker stop apisix-quickstart etcd
docker rm apisix-quickstart etcd
```

### Kubernetes
```bash
helm uninstall apisix --namespace apisix
```
