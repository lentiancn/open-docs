# Element Plus Usage Guide

Comprehensive guide for using Element Plus effectively.

---

## Basic Components

### Button

```vue
<el-button>Default</el-button>
<el-button type="primary">Primary</el-button>
<el-button type="success">Success</el-button>
<el-button type="danger">Danger</el-button>
<el-button type="warning">Warning</el-button>

<!-- Sizes -->
<el-button size="large">Large</el-button>
<el-button size="small">Small</el-button>

<!-- Icons -->
<el-button icon="Edit">Edit</el-button>
```

### Layout

```vue
<el-row>
  <el-col :span="24"><div class="grid-content"></div></el-col>
</el-row>

<el-row :gutter="20">
  <el-col :span="12"><div class="grid-content"></div></el-col>
  <el-col :span="12"><div class="grid-content"></div></el-col>
</el-row>
```

## Form Components

### Input

```vue
<el-input v-model="input" placeholder="Please input"></el-input>

<!-- With prefix -->
<el-input v-model="input2">
  <template #prefix>
    <el-icon><Search /></el-icon>
  </template>
</el-input>
```

### Select

```vue
<el-select v-model="value" placeholder="Select">
  <el-option
    v-for="item in options"
    :key="item.value"
    :label="item.label"
    :value="item.value"
  />
</el-select>
```

### Form

```vue
<el-form :model="form" label-width="120px">
  <el-form-item label="Activity name">
    <el-input v-model="form.name"></el-input>
  </el-form-item>
  <el-form-item>
    <el-button type="primary" @click="onSubmit">Submit</el-button>
  </el-form-item>
</el-form>
```

## Data Display

### Table

```vue
<el-table :data="tableData" stripe>
  <el-table-column prop="date" label="Date" width="180"></el-table-column>
  <el-table-column prop="name" label="Name" width="180"></el-table-column>
  <el-table-column prop="address" label="Address"></el-table-column>
</el-table>
```

### Tag

```vue
<el-tag>Tag</el-tag>
<el-tag type="success">Success</el-tag>
<el-tag type="info">Info</el-tag>
<el-tag type="warning">Warning</el-tag>
<el-tag type="danger">Danger</el-tag>
```

### Pagination

```vue
<el-pagination
  v-model:current-page="currentPage"
  :page-size="100"
  :total="1000"
  layout="prev, pager, next"
/>
```

## Navigation

### Menu

```vue
<el-menu default-active="1">
  <el-menu-item index="1">
    <el-icon><House /></el-icon>
    <span>Menu</span>
  </el-menu-item>
</el-menu>
```

## Feedback

### Message

```vue
<script setup>
import { ElMessage } from 'element-plus'

const open = () => {
  ElMessage.success('Success')
}
</script>
```

### Dialog

```vue
<el-button @click="visible = true">Open</el-button>

<el-dialog v-model="visible" title="Title">
  <span>Content</span>
  <template #footer>
    <el-button @click="visible = false">Cancel</el-button>
    <el-button type="primary" @click="visible = false">Confirm</el-button>
  </template>
</el-dialog>
```

### Notification

```vue
<script setup>
import { ElNotification } from 'element-plus'

ElNotification({
  title: 'Title',
  message: 'Message',
  type: 'success',
})
</script>
```

## Icons

### Usage

```vue
<el-icon><Edit /></el-icon>
<el-button :icon="Search">Search</el-button>
```

### Global Registration

```javascript
import * as icons from '@element-plus/icons-vue'

for (const [key, component] of Object.entries(icons)) {
  app.component(key, component)
}
```
