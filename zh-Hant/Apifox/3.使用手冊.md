# Apifox 使用手冊

Apifox 核心功能和使用技巧。

## API 管理

### 創建接口

1. 選擇項目
2. 點擊"新建接口"
3. 填寫接口信息

### 調試接口

1. 選擇接口
2. 填寫請求參數
3. 點擊發送
4. 查看響應

## 請求配置

### 請求頭

```http
Content-Type: application/json
Authorization: Bearer token
```

### 認證

- Bearer Token
- Basic Auth
- OAuth 2.0

## Mock 服務

### 創建 Mock

1. 點擊"新建 Mock"
2. 定義響應規則
3. 獲取 Mock URL

## 自動化測試

### 創建測試用例

1. 點擊"新建測試"
2. 添加測試步驟
3. 配置斷言
