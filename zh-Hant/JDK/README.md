# JDK

Java 開發工具包

## 簡介

JDK（Java Development Kit）是 Java 語言的軟體開發工具包，是整個 Java 開發的核心。JDK 包含了 Java 執行環境（JRE）、Java 編譯器（javac）、Java 虛擬機（JVM）以及大量的 Java 類別庫和工具。

## 主要版本

- **JDK 25** - 最新版本，帶來新特性和效能改進
- **JDK 21** - 最新的長期支援版本（LTS）
- **JDK 17** - 成熟的 LTS 版本，廣泛應用
- **JDK 11** - LTS 版本，標誌 Java 付費特性的分水嶺

## 核心組件

- **javac** - Java 編譯器，將原始碼編譯為位元組碼
- **java** - Java 程式執行命令
- **jvm** - Java 虛擬機，執行位元組碼
- **jar** - Java 歸檔工具
- **javadoc** - API 文檔生成工具
- **jdb** - Java 偵錯器

- 雲端運算和微服務

## 版本差異

不同的 JDK 主要版本在安裝、使用和功能上存在顯著差異：

- **JDK 8**：最後一個免費用於商業使用的 Oracle JDK 版本
- **JDK 11**：引入了 var、新的 HTTP 用戶端、ZGC（實驗性）
- **JDK 17**：最後一個在 NFTC 下完全免費的 Oracle JDK 版本
- **JDK 21**：虛擬線程、模式匹配、記錄類（穩定）
- **JDK 25**：最新特性，較短的支援週期
