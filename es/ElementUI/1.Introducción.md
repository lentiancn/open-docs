# Introducción a Element UI

Element UI es una biblioteca de componentes UI basada en Vue 2.0 que proporciona a los desarrolladores soluciones completas de componentes de escritorio.

## Qué es Element UI

Element UI es una biblioteca de componentes Vue 2.0 desarrollada y mantenida por el equipo Ele.me.

## Características Principales

### Sistema de Componentes Completo

Element UI proporciona más de 70 componentes de alta calidad:

- Componentes Básicos: Botones, Iconos, Diseño, Contenedores
- Componentes de Formulario: Entrada, Selector, Selector de fecha
- Visualización de Datos: Tabla, Árbol, Paginación, Etiquetas
- Componentes de Retroalimentación: Diálogo, Mensaje
- Componentes de Navegación: Menú, Tabs, Breadcrumb

### Soporte Vue 2.0

- Desarrollo basado en componentes
- Enlace de datos reactivo
- Soporte TypeScript

### Importación bajo Demanda

Usando babel-plugin-component.

### Soporte de Internacionalización

Múltiples paquetes de idiomas incorporados.

### Personalización de Temas

Soporta personalización global de temas.

## Principios de Diseño

- Consistencia: Consistencia con la vida real y la interfaz
- Retroalimentación: Control y retroalimentación de página
- Eficiencia: Procesos simples e intuitivos
- Controlabilidad: Decisiones de usuario y resultados controlables

## Escenarios de Aplicación

- Sistemas de gestión empresarial
- Sistemas de gestión de comercio electrónico
- Cualquier proyecto Vue 2
