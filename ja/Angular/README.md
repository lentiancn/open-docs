## クイックスタート

### Angular CLI のインストール

```bash
npm install -g @angular/cli
```

### 新規プロジェクトの作成

```bash
ng new my-app
cd my-app
ng serve
```

### コンポーネントの生成

```bash
ng generate component my-component
# または
ng g c my-component
```
