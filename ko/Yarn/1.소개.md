# Yarn 패키지 매니저

Yarn은 JavaScript 프로젝트의 의존성을 관리하는 데 사용되는 확립된 오픈소스 패키지 매니저입니다. Yarn은 패키지 의존성의 설치, 업데이트, 구성 및 제거 과정을 지원하여 궁극적으로 방해 없이 목표를 더 빠르게 달성할 수 있도록 도와줍니다.

Yarn의 주요 관심사는 속도, 정확성, 보안 및 개발자 경험이며, 워크스페이스, 오프라인 캐싱, 병렬 설치, 강화 모드, 대화형 명령어 등의 혁신적인 기능을 활용하여 모든 측면에서 개선하고 있습니다.

견고한 기능과 탁월한 실적을 바탕으로 Yarn은 모든 규모의 프로젝트에서 효율성, 보안 및 긍정적인 개발자 경험을 주도하며 JavaScript 생태계에 없어서는 안 될 자산임이 입증되었습니다.

## 주요 버전 비교

### Yarn v4+ (모던 버전)
- **아키텍처**: 플러그인 기반 아키텍처 (Berry)
- **기능**: 네이티브 워크스페이스 지원, PnP (Plug'n'Play), 오프라인 캐싱, 병렬 설치
- **설치 방법**: Corepack을 통한 설치 권장
- **상태**: 활발히 개발 중

### Yarn v1 (Classic)
- **아키텍처**: 전통적인 아키텍처
- **기능**: 단순하고 직관적이며 광범위한 호환성
- **설치 방법**: npm을 통한 글로벌 설치
- **상태**: 유지보수 중

> 참고: 다른 대부분의 패키지 매니저와 달리, Yarn은 설치와 관련되지 않은 명령어에 대해서도 npm에 의존하지 않고 모든 명령어를 자체적으로 재구현하여 개발자 경험과 안정성을 완전히 제어합니다.
