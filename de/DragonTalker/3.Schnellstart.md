# DragonTalker Schnellstart

> Starte mit DragonTalker in 10 Minuten

---

## 1. Installation

```bash
git clone https://github.com/your-repo/DragonTalker.git
cd DragonTalker
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
bash scripts/download_models.sh
```

---

## 2. Video generieren

```bash
python inference.py \
  --source_image examples/source_image.jpg \
  --driven_audio examples/driven_audio.wav \
  --result_dir ./results \
  --enhancer gfpgan
```

---

## 3. Web-Interface verwenden

```bash
python app.py
```

Browser: http://localhost:7860

---

## 4. Anforderungen

### Bild
- Format: JPG, PNG
- Auflösung: 512×512 oder höher

### Audio
- Format: WAV, MP3
- Dauer: 1-60 Sekunden

---

## 5. Ausgabe

- Format: MP4 (H.264)
- Auflösung: 256×256 oder 512×512
- Bildrate: 25 FPS
